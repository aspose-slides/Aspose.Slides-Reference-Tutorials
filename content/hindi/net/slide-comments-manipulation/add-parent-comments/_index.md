## नामस्थान आयात करना

सबसे पहले, आपको अपने प्रोजेक्ट में प्रासंगिक नामस्थान आयात करने की आवश्यकता होगी। ये नेमस्पेस .NET के लिए Aspose.Slides के साथ काम करने के लिए आवश्यक कक्षाएं और विधियां प्रदान करते हैं।

```csharp
using Aspose.Slides;
using Aspose.Slides.SlideComments;
```

पूर्वापेक्षाओं और नामस्थानों के साथ, आइए एक स्लाइड में मूल टिप्पणियाँ जोड़ने की प्रक्रिया को कई चरणों में विभाजित करें।

## चरण 1: एक प्रेजेंटेशन बनाएं

आरंभ करने के लिए, आपको .NET के लिए Aspose.Slides का उपयोग करके एक नई प्रस्तुति बनानी होगी। यह प्रस्तुति वह कैनवास होगी जिस पर आप अपनी टिप्पणियाँ जोड़ेंगे।

```csharp
// आउटपुट निर्देशिका का पथ.
string outPptxFile = "Output Path";

using (Presentation pres = new Presentation())
{
    // टिप्पणियाँ जोड़ने के लिए आपका कोड यहां जाएगा।
    
    pres.Save(outPptxFile + "parent_comment.pptx", SaveFormat.Pptx);
}
```

 उपरोक्त कोड में, बदलें`"Output Path"` आपकी आउटपुट प्रस्तुति के लिए वांछित पथ के साथ।

## चरण 2: टिप्पणी लेखक जोड़ें

टिप्पणियाँ जोड़ने से पहले, आपको इन टिप्पणियों के लेखकों को परिभाषित करना होगा। इस उदाहरण में, हमारे पास दो लेखक हैं, "लेखक_1" और "लेखक_2", प्रत्येक को एक उदाहरण द्वारा दर्शाया गया है`ICommentAuthor`.

```csharp
// टिप्पणी जोड़ना
ICommentAuthor author1 = pres.CommentAuthors.AddAuthor("Author_1", "A.A.");
IComment comment1 = author1.Comments.AddComment("comment1", pres.Slides[0], new PointF(10, 10), DateTime.Now);

// टिप्पणी1 के लिए उत्तर जोड़ें
ICommentAuthor author2 = pres.CommentAuthors.AddAuthor("Autror_2", "B.B.");
IComment reply1 = author2.Comments.AddComment("reply 1 for comment 1", pres.Slides[0], new PointF(10, 10), DateTime.Now);
reply1.ParentComment = comment1;
```

इस चरण में, हम दो टिप्पणी लेखक बनाते हैं और प्रारंभिक टिप्पणी और टिप्पणी में एक उत्तर जोड़ते हैं।

## चरण 3: अधिक उत्तर जोड़ें

टिप्पणियों की एक पदानुक्रमित संरचना बनाने के लिए, आप मौजूदा टिप्पणियों में अधिक उत्तर जोड़ सकते हैं। यहां, हम "टिप्पणी1" में दूसरा उत्तर जोड़ते हैं।

```csharp
// टिप्पणी1 के लिए उत्तर जोड़ें
IComment reply2 = author2.Comments.AddComment("reply 2 for comment 1", pres.Slides[0], new PointF(10, 10), DateTime.Now);
reply2.ParentComment = comment1;
```

यह आपकी प्रस्तुति में वार्तालाप प्रवाह स्थापित करता है।

## चरण 4: नेस्टेड उत्तर जोड़ें

टिप्पणियों में नेस्टेड उत्तर भी हो सकते हैं। इसे प्रदर्शित करने के लिए, हम "टिप्पणी 1 के लिए उत्तर 2" में एक उप-उत्तर बनाते हुए एक उत्तर जोड़ते हैं।

```csharp
// उत्तर में उत्तर जोड़ें
IComment subReply = author1.Comments.AddComment("subreply 3 for reply 2", pres.Slides[0], new PointF(10, 10), DateTime.Now);
subReply.ParentComment = reply2;
```

यह चरण टिप्पणी पदानुक्रमों को प्रबंधित करने में .NET के लिए Aspose.Slides की बहुमुखी प्रतिभा पर प्रकाश डालता है।

## चरण 5: अधिक टिप्पणियाँ और उत्तर

आप आवश्यकतानुसार अधिक टिप्पणियाँ और उत्तर जोड़ना जारी रख सकते हैं। इस उदाहरण में, हम दो और टिप्पणियाँ और उनमें से एक का उत्तर जोड़ते हैं।

```csharp
IComment comment2 = author2.Comments.AddComment("comment 2", pres.Slides[0], new PointF(10, 10), DateTime.Now);
IComment comment3 = author2.Comments.AddComment("comment 3", pres.Slides[0], new PointF(10, 10), DateTime.Now);

IComment reply3 = author1.Comments.AddComment("reply 4 for comment 3", pres.Slides[0], new PointF(10, 10), DateTime.Now);
reply3.ParentComment = comment3;
```

यह चरण दर्शाता है कि आप अपनी प्रस्तुतियों के लिए आकर्षक और इंटरैक्टिव सामग्री कैसे बना सकते हैं।

## चरण 6: पदानुक्रम प्रदर्शित करें

टिप्पणी पदानुक्रम की कल्पना करने के लिए, आप इसे कंसोल पर प्रदर्शित कर सकते हैं। यह चरण वैकल्पिक है लेकिन डिबगिंग और संरचना को समझने में सहायक हो सकता है।

```csharp
ISlide slide = pres.Slides[0];
var comments = slide.GetSlideComments(null);
for (int i = 0; i < comments.Length; i++)
{
    IComment comment = comments[i];
    while (comment.ParentComment != null)
    {
        Console.Write("\t");
        comment = comment.ParentComment;
    }

    Console.Write("{0} : {1}", comments[i].Author.Name, comments[i].Text);
    Console.WriteLine();
}
```

## चरण 7: टिप्पणियाँ हटाएँ

कुछ मामलों में, आपको टिप्पणियाँ और उनके उत्तर हटाने की आवश्यकता हो सकती है। नीचे दिया गया कोड स्निपेट दर्शाता है कि "टिप्पणी1" और उसके सभी उत्तरों को कैसे हटाया जाए।

```csharp
comment1.Remove();
pres.Save(outPptxFile + "remove_comment.pptx", SaveFormat.Pptx);
```

यह चरण आपकी प्रस्तुति सामग्री को प्रबंधित और अद्यतन करने के लिए उपयोगी है।

इन चरणों के साथ, आप .NET के लिए Aspose.Slides का उपयोग करके इंटरैक्टिव टिप्पणियों और उत्तरों के साथ प्रस्तुतियाँ बना सकते हैं। चाहे आप अपने दर्शकों को शामिल करना चाह रहे हों या टीम के सदस्यों के साथ सहयोग करना चाह रहे हों, यह सुविधा संभावनाओं की एक विस्तृत श्रृंखला प्रदान करती है।

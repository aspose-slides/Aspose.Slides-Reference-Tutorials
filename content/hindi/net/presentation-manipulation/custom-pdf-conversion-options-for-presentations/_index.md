अपने C# कोड में, आपको Aspose.Slides लाइब्रेरी को शामिल करना होगा। यहां बताया गया है कि आप यह कैसे कर सकते हैं:

## चरण 3: रूपांतरण विकल्प प्रारंभ करें

 कस्टम पीडीएफ रूपांतरण प्राप्त करने के लिए, आप अपनी आवश्यकताओं के अनुसार विभिन्न विकल्प सेट कर सकते हैं। यहां एक उदाहरण दिया गया है कि आप कैसे आरंभ कर सकते हैं`PdfOptions` ऑब्जेक्ट करें और कुछ सेटिंग्स कस्टमाइज़ करें:

```csharp
string dataDir = "Your Document Directory";
string outPath = "Your Output Directory";

using (Presentation pres = new Presentation(dataDir + "ConvertToPDF.pptx"))
{
    PdfOptions pdfOptions = new PdfOptions();

    // जेपीईजी गुणवत्ता सेट करें
    pdfOptions.JpegQuality = 90;

    // मेटाफ़ाइल्स के लिए व्यवहार को परिभाषित करें
    pdfOptions.SaveMetafilesAsPng = true;

    // पाठ संपीड़न स्तर सेट करें
    pdfOptions.TextCompression = PdfTextCompression.Flate;

    // पीडीएफ मानक को परिभाषित करें
    pdfOptions.Compliance = PdfCompliance.Pdf15;

    INotesCommentsLayoutingOptions options = pdfOptions.NotesCommentsLayouting;
    options.NotesPosition = NotesPositions.BottomFull;

    // निर्दिष्ट विकल्पों के साथ प्रेजेंटेशन को पीडीएफ में सहेजें
    pres.Save(outPath + "Custom_Option_Pdf_Conversion_out.pdf", SaveFormat.Pdf, pdfOptions);
}
```

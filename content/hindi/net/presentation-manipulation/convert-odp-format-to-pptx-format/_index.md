## पीपीटीएक्स से ओडीपी में कनवर्ट करना

आइए पीपीटीएक्स से ओडीपी में कनवर्ट करने के लिए कोड से शुरुआत करें। यहां चरण-दर-चरण मार्गदर्शिका दी गई है:

```csharp
// एक प्रेजेंटेशन ऑब्जेक्ट को इंस्टेंट करें जो एक प्रेजेंटेशन फ़ाइल का प्रतिनिधित्व करता है
using (Presentation pres = new Presentation("ConversionFromPresentation.pptx"))
{
    // पीपीटीएक्स प्रस्तुति को ओडीपी प्रारूप में सहेजा जा रहा है
    pres.Save("ConvertedToOdp", Aspose.Slides.Export.SaveFormat.Odp);
}
```

 इस कोड स्निपेट में, हम एक बनाते हैं`Presentation` ऑब्जेक्ट, इनपुट PPTX फ़ाइल निर्दिष्ट करता है। फिर हम इसका उपयोग करते हैं`Save` प्रेजेंटेशन को ओडीपी फॉर्मेट में सेव करने की विधि।

## ओडीपी से पीपीटीएक्स में कनवर्ट करना

अब, आइए ओडीपी से पीपीटीएक्स तक रिवर्स रूपांतरण का पता लगाएं:

```csharp
// एक प्रेजेंटेशन ऑब्जेक्ट को इंस्टेंट करें जो एक प्रेजेंटेशन फ़ाइल का प्रतिनिधित्व करता है
using (Presentation pres = new Presentation("OpenOfficePresentation.odp"))
{
    // ODP प्रस्तुति को PPTX प्रारूप में सहेजा जा रहा है
    pres.Save("ConvertedFromOdp", Aspose.Slides.Export.SaveFormat.Pptx);
}
```

 यह कोड पिछले उदाहरण से काफी मिलता-जुलता है। हम एक बनाते हैं`Presentation`ऑब्जेक्ट, इनपुट ओडीपी फ़ाइल निर्दिष्ट करें, और इसका उपयोग करें`Save` इसे PPTX फॉर्मेट में सेव करने की विधि।

इस व्यापक गाइड में, हम आपको .NET के लिए Aspose.Slides का उपयोग करके किसी प्रेजेंटेशन को HTML में परिवर्तित करते समय मूल फ़ॉन्ट को संरक्षित करने की प्रक्रिया के बारे में बताएंगे। हम आपको आवश्यक C# स्रोत कोड प्रदान करेंगे और प्रत्येक चरण को विस्तार से समझाएंगे। इस ट्यूटोरियल के अंत तक, आप यह सुनिश्चित करने में सक्षम होंगे कि आपके परिवर्तित HTML दस्तावेज़ में फ़ॉन्ट मूल प्रस्तुति के प्रति वफादार रहें।

## 1 परिचय

PowerPoint प्रस्तुतियों को HTML में परिवर्तित करते समय, आपकी सामग्री की दृश्य स्थिरता सुनिश्चित करने के लिए मूल फ़ॉन्ट को बनाए रखना महत्वपूर्ण है। .NET के लिए Aspose.Slides इसे प्राप्त करने के लिए एक शक्तिशाली समाधान प्रदान करता है। इस ट्यूटोरियल में, हम रूपांतरण प्रक्रिया के दौरान मूल फ़ॉन्ट को संरक्षित करने के लिए आवश्यक चरणों के बारे में आपका मार्गदर्शन करेंगे।

## 2. पूर्वापेक्षाएँ

शुरू करने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित आवश्यक शर्तें हैं:

- आपकी मशीन पर विज़ुअल स्टूडियो स्थापित है।
- .NET लाइब्रेरी के लिए Aspose.Slides आपके प्रोजेक्ट में जोड़ा गया।

## 3. अपना प्रोजेक्ट स्थापित करना

आरंभ करने के लिए, विज़ुअल स्टूडियो में एक नया प्रोजेक्ट बनाएं और संदर्भ के रूप में .NET लाइब्रेरी के लिए Aspose.Slides जोड़ें।

## 4. प्रेजेंटेशन लोड हो रहा है

अपनी PowerPoint प्रस्तुति को लोड करने के लिए निम्नलिखित कोड का उपयोग करें:

```csharp
string dataDir = "Your Document Directory";

using (Presentation pres = new Presentation("input.pptx"))
{
    // आपका कोड यहाँ
}
```

 प्रतिस्थापित करें`"Your Document Directory"` आपकी प्रस्तुति फ़ाइल के पथ के साथ।

## 5. डिफ़ॉल्ट फ़ॉन्ट्स को छोड़कर

कैलिबरी और एरियल जैसे डिफ़ॉल्ट फ़ॉन्ट को बाहर करने के लिए, निम्नलिखित कोड का उपयोग करें:

```csharp
string[] fontNameExcludeList = { "Calibri", "Arial" };
```

आप इस सूची को आवश्यकतानुसार अनुकूलित कर सकते हैं.

## 6. सभी फ़ॉन्ट एम्बेड करना

इसके बाद, हम HTML दस्तावेज़ में सभी फ़ॉन्ट एम्बेड करेंगे। यह सुनिश्चित करता है कि मूल फ़ॉन्ट संरक्षित हैं। निम्नलिखित कोड का प्रयोग करें:

```csharp
EmbedAllFontsHtmlController embedFontsController = new EmbedAllFontsHtmlController(fontNameExcludeList);

HtmlOptions htmlOptionsEmbed = new HtmlOptions
{
    HtmlFormatter = HtmlFormatter.CreateCustomFormatter(embedFontsController)
};
```

## 7. HTML के रूप में सहेजा जा रहा है

अब, प्रेजेंटेशन को एम्बेडेड फ़ॉन्ट्स के साथ HTML दस्तावेज़ के रूप में सहेजें:

```csharp
pres.Save("output.html", SaveFormat.Html, htmlOptionsEmbed);
```

 प्रतिस्थापित करें`"output.html"` आपके इच्छित आउटपुट फ़ाइल नाम के साथ।

## 8. निष्कर्ष

इस ट्यूटोरियल में, हमने दिखाया है कि .NET के लिए Aspose.Slides का उपयोग करके PowerPoint प्रस्तुति को HTML में परिवर्तित करते समय मूल फ़ॉन्ट को कैसे संरक्षित किया जाए। इन चरणों का पालन करके, आप यह सुनिश्चित कर सकते हैं कि आपका परिवर्तित HTML दस्तावेज़ मूल प्रस्तुति की दृश्य अखंडता को बनाए रखता है।

## 9. अक्सर पूछे जाने वाले प्रश्न

### Q1: क्या मैं बहिष्कृत फ़ॉन्ट की सूची को अनुकूलित कर सकता हूँ?

 हाँ तुम कर सकते हो। संशोधित करें`fontNameExcludeList` आपकी आवश्यकताओं के अनुसार विशिष्ट फ़ॉन्ट को शामिल करने या बाहर करने के लिए सरणी।

### Q2: यदि मैं सभी फ़ॉन्ट एम्बेड नहीं करना चाहता तो क्या होगा?

यदि आप केवल विशिष्ट फ़ॉन्ट एम्बेड करना चाहते हैं, तो आप तदनुसार कोड को संशोधित कर सकते हैं। अधिक विवरण के लिए .NET दस्तावेज़ के लिए Aspose.Slides से परामर्श लें।

### Q3: क्या .NET के लिए Aspose.Slides का उपयोग करने के लिए कोई लाइसेंसिंग आवश्यकताएं हैं?

हां, आपको अपने प्रोजेक्ट में .NET के लिए Aspose.Slides का उपयोग करने के लिए एक वैध लाइसेंस की आवश्यकता हो सकती है। लाइसेंस संबंधी जानकारी के लिए Aspose वेबसाइट देखें।

### Q4: क्या मैं .NET के लिए Aspose.Slides का उपयोग करके अन्य फ़ाइल स्वरूपों को HTML में परिवर्तित कर सकता हूँ?

.NET के लिए Aspose.Slides मुख्य रूप से PowerPoint प्रस्तुतियों पर केंद्रित है। अन्य फ़ाइल स्वरूपों को HTML में परिवर्तित करने के लिए, आपको उन प्रारूपों के लिए तैयार किए गए अन्य Aspose उत्पादों का पता लगाने की आवश्यकता हो सकती है।

### Q5: मैं अतिरिक्त संसाधनों और सहायता तक कहां पहुंच सकता हूं?

 आप Aspose वेबसाइट पर अधिक दस्तावेज़, ट्यूटोरियल और समर्थन पा सकते हैं। मिलने जाना[.NET दस्तावेज़ीकरण के लिए Aspose.Slides](https://reference.aspose.com/slides/net/) विस्तृत जानकारी के लिए.

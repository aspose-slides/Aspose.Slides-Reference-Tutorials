आज के डिजिटल युग में, प्रस्तुतियों को पीडीएफ में परिवर्तित करना एक आम आवश्यकता है, खासकर व्यवसाय और शैक्षिक क्षेत्रों में। .NET के लिए Aspose.Slides इस कार्य को आसानी से पूरा करने के लिए एक शक्तिशाली समाधान प्रदान करता है। इस चरण-दर-चरण ट्यूटोरियल में, हम रूपांतरण प्रगति पर नज़र रखते हुए एक प्रस्तुति को पीडीएफ में परिवर्तित करने की प्रक्रिया में आपका मार्गदर्शन करेंगे।

## परिचय

इस ट्यूटोरियल में, हम PowerPoint प्रेजेंटेशन को PDF दस्तावेज़ में बदलने के लिए .NET के लिए Aspose.Slides का उपयोग करेंगे। हम आपको रूपांतरण की स्थिति के बारे में सूचित रखने के लिए एक प्रगति अद्यतन सुविधा भी लागू करेंगे।

## आवश्यक शर्तें

शुरू करने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित आवश्यक शर्तें हैं:

1. विजुअल स्टूडियो या कोई पसंदीदा कोड संपादक।
2. .NET लाइब्रेरी के लिए Aspose.Slides स्थापित।
3. कनवर्ट करने के लिए एक PowerPoint प्रेजेंटेशन फ़ाइल (उदाहरण के लिए, "ConvertToPDF.pptx")।

## चरण 1: पर्यावरण की स्थापना

सबसे पहले, विज़ुअल स्टूडियो या अपने पसंदीदा कोड संपादक में एक नया C# प्रोजेक्ट बनाएं। सुनिश्चित करें कि आपने अपने प्रोजेक्ट में .NET लाइब्रेरी के लिए Aspose.Slides का एक संदर्भ जोड़ा है।

## चरण 2: कोड लिखना

अब, आइए उस कोड के बारे में जानें जो प्रगति अपडेट के साथ प्रेजेंटेशन-टू-पीडीएफ रूपांतरण करेगा। निम्नलिखित स्रोत कोड का उपयोग करें:

```csharp
using (Presentation presentation = new Presentation(dataDir + "ConvertToPDF.pptx"))
{
    ISaveOptions saveOptions = new PdfOptions();
    saveOptions.ProgressCallback = new ExportProgressHandler();
    presentation.Save(dataDir + "ConvertToPDF.pdf", SaveFormat.Pdf, saveOptions);
}
```

 इस कोड स्निपेट में, हम Aspose.Slides का उपयोग करके PowerPoint प्रेजेंटेशन खोलते हैं और सहेजने के लिए पीडीएफ प्रारूप निर्दिष्ट करते हैं। हमने यह भी सेट किया है`ProgressCallback` के एक उदाहरण के लिए संपत्ति`ExportProgressHandler` कक्षा।

## चरण 3: प्रगति कॉलबैक लागू करना

 अब हमें इसे लागू करने की जरूरत है`ExportProgressHandler` रूपांतरण प्रक्रिया के दौरान प्रगति अपडेट को संभालने के लिए क्लास। यहाँ के लिए कोड है`ExportProgressHandler` कक्षा:

```csharp
class ExportProgressHandler : IProgressCallback
{
    public void Reporting(double progressValue)
    {
        // यहां प्रगति प्रतिशत मान का उपयोग करें
        int progress = Convert.ToInt32(progressValue);
        Console.WriteLine(progress + "% file converted");
    }
}
```

 यह वर्ग कार्यान्वित करता है`IProgressCallback` इंटरफ़ेस और परिभाषित करता है`Reporting` प्रगति अद्यतनों को संभालने की विधि। यह वर्तमान प्रगति प्रतिशत को कंसोल पर प्रिंट करता है।

## चरण 4: कोड चलाना

अपना प्रोजेक्ट संकलित करें और चलाएँ। प्रेजेंटेशन पीडीएफ में परिवर्तित होते ही आप कंसोल में प्रगति अपडेट देखेंगे।

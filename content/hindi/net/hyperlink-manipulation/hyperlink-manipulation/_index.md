प्रस्तुतियों में हाइपरलिंक आवश्यक तत्व हैं, क्योंकि वे स्लाइड के बीच नेविगेट करने या बाहरी संसाधनों तक पहुंचने का एक सुविधाजनक तरीका प्रदान करते हैं। .NET के लिए Aspose.Slides आपकी प्रेजेंटेशन स्लाइड्स में हाइपरलिंक जोड़ने और हटाने के लिए शक्तिशाली सुविधाएँ प्रदान करता है। इस ट्यूटोरियल में, हम .NET के लिए Aspose.Slides का उपयोग करके हाइपरलिंक हेरफेर की प्रक्रिया के माध्यम से आपका मार्गदर्शन करेंगे। हम स्लाइड में हाइपरलिंक जोड़ने और स्लाइड से हाइपरलिंक हटाने पर चर्चा करेंगे। तो, आइए गोता लगाएँ!

## आवश्यक शर्तें

शुरू करने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित आवश्यक शर्तें हैं:

1.  .NET के लिए Aspose.Slides: आपके पास .NET लाइब्रेरी के लिए Aspose.Slides इंस्टॉल और सेटअप होना चाहिए। आप दस्तावेज़ पा सकते हैं[यहाँ](https://reference.aspose.com/slides/net/) और इसे यहां से डाउनलोड करें[इस लिंक](https://releases.aspose.com/slides/net/).

2. आपकी दस्तावेज़ निर्देशिका: आपको एक निर्देशिका की आवश्यकता है जहाँ आप अपनी प्रस्तुति फ़ाइलें संग्रहीत करेंगे। अपने कोड में इस निर्देशिका का पथ निर्दिष्ट करना सुनिश्चित करें।

3. C# का बुनियादी ज्ञान: यह ट्यूटोरियल मानता है कि आपको C# प्रोग्रामिंग की बुनियादी समझ है।

अब जब आपके पास अपनी पूर्वापेक्षाएँ हैं, तो आइए .NET के लिए Aspose.Slides का उपयोग करके हाइपरलिंक हेरफेर के लिए चरण-दर-चरण मार्गदर्शिका पर आगे बढ़ें।

## स्लाइड में हाइपरलिंक जोड़ना

### चरण 1: प्रस्तुति आरंभ करें

आरंभ करने के लिए, आपको Aspose.Slides का उपयोग करके एक प्रेजेंटेशन प्रारंभ करना होगा। आप इसे निम्नलिखित कोड के साथ कर सकते हैं:

```csharp
using (Presentation presentation = new Presentation())
{
    // आपका कोड यहाँ
}
```

### चरण 2: टेक्स्ट फ़्रेम जोड़ें

अब, एक स्लाइड में एक टेक्स्ट फ़्रेम जोड़ें। यह कोड टेक्स्ट के साथ एक आयताकार आकार बनाता है:

```csharp
IAutoShape shape1 = presentation.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 100, 100, 600, 50, false);
shape1.AddTextFrame("Aspose: File Format APIs");
```

### चरण 3: हाइपरलिंक जोड़ें

इसके बाद, आप अपने द्वारा बनाए गए आकार में टेक्स्ट में एक हाइपरलिंक जोड़ देंगे। यहां बताया गया है कि आप यह कैसे कर सकते हैं:

```csharp
shape1.TextFrame.Paragraphs[0].Portions[0].PortionFormat.HyperlinkClick = new Hyperlink("https://www.aspose.com/");
shape1.TextFrame.Paragraphs[0].Portions[0].PortionFormat.HyperlinkClick.Tooltip = "More than 70% Fortune 100 companies trust Aspose APIs";
shape1.TextFrame.Paragraphs[0].Portions[0].PortionFormat.FontHeight = 32;
```

### चरण 4: प्रस्तुति सहेजें

अंत में, अपनी प्रस्तुति को जोड़े गए हाइपरलिंक के साथ सहेजें:

```csharp
presentation.Save("presentation-out.pptx", SaveFormat.Pptx);
```

बधाई हो! आपने .NET के लिए Aspose.Slides का उपयोग करके स्लाइड में हाइपरलिंक सफलतापूर्वक जोड़ लिया है।

## स्लाइड से हाइपरलिंक हटाना

### चरण 1: प्रस्तुति आरंभ करें

किसी स्लाइड से हाइपरलिंक हटाने के लिए, आपको एक मौजूदा प्रस्तुतिकरण खोलना होगा:

```csharp
string dataDir = "Your Document Directory";
Presentation presentation = new Presentation(dataDir + "Hyperlink.pptx");
```

### चरण 2: हाइपरलिंक हटाएँ

अब, निम्नलिखित कोड का उपयोग करके प्रेजेंटेशन से सभी हाइपरलिंक हटा दें:

```csharp
presentation.HyperlinkQueries.RemoveAllHyperlinks();
```

### चरण 3: प्रस्तुति सहेजें

हाइपरलिंक हटाने के बाद, प्रेजेंटेशन सहेजें:

```csharp
presentation.Save(dataDir + "RemovedHyperlink_out.pptx", SaveFormat.Pptx);
```

और बस! आपने .NET के लिए Aspose.Slides का उपयोग करके स्लाइड से हाइपरलिंक को सफलतापूर्वक हटा दिया है।

अंत में, .NET के लिए Aspose.Slides आपकी प्रस्तुतियों में हाइपरलिंक में हेरफेर करने का एक कुशल तरीका प्रदान करता है, जिससे आप इंटरैक्टिव और आकर्षक स्लाइड बना सकते हैं। चाहे आप बाहरी संसाधनों में हाइपरलिंक जोड़ना चाहते हों या उन्हें हटाना चाहते हों, Aspose.Slides प्रक्रिया को सरल बनाता है और आपकी प्रस्तुति-निर्माण क्षमताओं को बढ़ाता है।

 .NET के लिए Aspose.Slides में हाइपरलिंक हेरफेर पर इस ट्यूटोरियल में हमारे साथ शामिल होने के लिए धन्यवाद। यदि आपके कोई प्रश्न हैं या अतिरिक्त सहायता की आवश्यकता है, तो बेझिझक पता लगाएं[Aspose.Slides दस्तावेज़ीकरण](https://reference.aspose.com/slides/net/) या Aspose समुदाय तक पहुंचें[सहयता मंच](https://forum.aspose.com/).

## निष्कर्ष

इस ट्यूटोरियल में, हमने सीखा है कि .NET के लिए Aspose.Slides का उपयोग करके प्रस्तुतियों में हाइपरलिंक्स में हेरफेर कैसे किया जाता है। हमने हाइपरलिंक्स को जोड़ने और हटाने दोनों को कवर किया है, जिससे आप गतिशील और इंटरैक्टिव प्रस्तुतियाँ बना सकते हैं। Aspose.Slides प्रक्रिया को सरल बनाता है, जिससे बाहरी संसाधनों के हाइपरलिंक के साथ आपकी स्लाइड्स को बढ़ाना आसान हो जाता है।

क्या आपके पास Aspose.Slides के साथ काम करने या प्रेजेंटेशन डिज़ाइन के अन्य पहलुओं के बारे में कोई और प्रश्न हैं? अधिक जानकारी के लिए नीचे दिए गए अक्सर पूछे जाने वाले प्रश्न देखें।

## अक्सर पूछे जाने वाले प्रश्न (अक्सर पूछे जाने वाले प्रश्न)

### .NET के लिए Aspose.Slides का उपयोग करने के प्रमुख लाभ क्या हैं?
.NET के लिए Aspose.Slides प्रस्तुतियों को बनाने, हेरफेर करने और परिवर्तित करने के लिए सुविधाओं की एक विस्तृत श्रृंखला प्रदान करता है। यह आपकी स्लाइड में सामग्री, एनिमेशन और इंटरैक्शन जोड़ने के लिए टूल का एक व्यापक सेट प्रदान करता है।

### क्या मैं Aspose.Slides में टेक्स्ट के अलावा अन्य ऑब्जेक्ट में हाइपरलिंक जोड़ सकता हूँ?
हाँ, Aspose.Slides आपको आकृतियों, छवियों और पाठ सहित विभिन्न वस्तुओं में हाइपरलिंक जोड़ने की अनुमति देता है, जिससे आपको इंटरैक्टिव प्रस्तुतियाँ बनाने में लचीलापन मिलता है।

### क्या Aspose.Slides विभिन्न PowerPoint फ़ाइल स्वरूपों के साथ संगत है?
बिल्कुल। Aspose.Slides पीपीटी, पीपीटीएक्स, पीपीएस और अन्य सहित विभिन्न पावरपॉइंट प्रारूपों का समर्थन करता है। यह Microsoft PowerPoint के विभिन्न संस्करणों के साथ संगतता सुनिश्चित करता है।

### मुझे Aspose.Slides के लिए अतिरिक्त संसाधन और समर्थन कहां मिल सकता है?
गहन दस्तावेज़ीकरण और सामुदायिक सहायता के लिए, पर जाएँ[Aspose.Slides दस्तावेज़ीकरण](https://reference.aspose.com/slides/net/) और यह[Aspose समर्थन मंच](https://forum.aspose.com/).

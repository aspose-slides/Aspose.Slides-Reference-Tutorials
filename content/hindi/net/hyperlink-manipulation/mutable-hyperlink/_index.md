## परिवर्तनीय हाइपरलिंक निर्माण

### चरण 1: अपना प्रोजेक्ट स्थापित करना
सबसे पहले, एक नया प्रोजेक्ट बनाएं या अपनी IDE में कोई मौजूदा प्रोजेक्ट खोलें। सुनिश्चित करें कि आपके प्रोजेक्ट में .NET के लिए Aspose.Slides सही ढंग से संदर्भित है।

### चरण 2: नामस्थान आयात करें
अपनी कोड फ़ाइल में, Aspose.Slides के साथ काम करने के लिए आवश्यक नामस्थान आयात करें:

```csharp
using Aspose.Slides;
using Aspose.Slides.Export;
using Aspose.Slides.Shape;
```

### चरण 3: एक नई प्रस्तुति बनाएं
एक नया पावरपॉइंट प्रेजेंटेशन बनाने के लिए, निम्नलिखित कोड का उपयोग करें:

```csharp
string dataDir = "Your Document Directory";

using (Presentation presentation = new Presentation())
{
    // प्रेजेंटेशन बनाने और उसमें हेरफेर करने के लिए आपका कोड यहां दिया गया है
    presentation.Save(dataDir + "presentation-out.pptx", SaveFormat.Pptx);
}
```

### चरण 4: हाइपरलिंक्ड आकृति जोड़ना
अब, आइए हाइपरलिंक के साथ अपनी प्रस्तुति में एक आकृति जोड़ें। इस उदाहरण में, हम Aspose वेबसाइट के हाइपरलिंक के साथ एक आयताकार आकार बनाएंगे:

```csharp
IAutoShape shape1 = presentation.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 100, 100, 600, 50, false);
shape1.AddTextFrame("Aspose: File Format APIs");
shape1.TextFrame.Paragraphs[0].Portions[0].PortionFormat.HyperlinkClick = new Hyperlink("https://www.aspose.com/");
shape1.TextFrame.Paragraphs[0].Portions[0].PortionFormat.HyperlinkClick.Tooltip = "More than 70% Fortune 100 companies trust Aspose APIs";
shape1.TextFrame.Paragraphs[0].Portions[0].PortionFormat.FontHeight = 32;
```

इस चरण में, हमने "एस्पोज़: फ़ाइल फ़ॉर्मेट एपीआई" टेक्स्ट और एक क्लिक करने योग्य हाइपरलिंक के साथ एक आयताकार आकार जोड़ा है। आप अपनी आवश्यकताओं के अनुसार आकार, पाठ और हाइपरलिंक को अनुकूलित कर सकते हैं।

### चरण 5: प्रस्तुति को सहेजना
अंत में, निम्नलिखित कोड का उपयोग करके अपनी प्रस्तुति को एक फ़ाइल में सहेजें:

```csharp
presentation.Save(dataDir + "presentation-out.pptx", SaveFormat.Pptx);
```

आपकी परिवर्तनीय हाइपरलिंक प्रस्तुति अब तैयार है!

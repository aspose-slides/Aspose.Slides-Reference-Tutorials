```csharp

```

## चरण 4: पहली स्लाइड प्राप्त करें
```csharp
ISlide sld = pres.Slides[0];
```
अपनी प्रस्तुति की पहली स्लाइड तक पहुंचें।
## चरण 5: एलिप्से ऑटोशेप जोड़ें
```csharp
IShape shp = sld.Shapes.AddAutoShape(ShapeType.Ellipse, 50, 150, 150, 50);
```
स्लाइड पर एक दीर्घवृत्त ऑटोशेप डालें, उसकी स्थिति और आयाम निर्दिष्ट करें।
## चरण 6: दीर्घवृत्त आकार को प्रारूपित करें
```csharp
shp.FillFormat.FillType = FillType.Solid;
shp.FillFormat.SolidFillColor.Color = Color.Chocolate;
shp.LineFormat.FillFormat.FillType = FillType.Solid;
shp.LineFormat.FillFormat.SolidFillColor.Color = Color.Black;
shp.LineFormat.Width = 5;
```
दीर्घवृत्त आकार में फ़ॉर्मेटिंग लागू करें, भरण रंग और रेखा गुण सेट करें।

संशोधित प्रस्तुति को डिस्क पर सहेजें।

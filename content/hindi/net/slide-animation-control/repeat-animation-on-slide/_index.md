## परिचय
प्रस्तुतियों की गतिशील दुनिया में, एनिमेशन को नियंत्रित करने की क्षमता दर्शकों का ध्यान आकर्षित करने और आकर्षित करने में महत्वपूर्ण भूमिका निभाती है। .NET के लिए Aspose.Slides डेवलपर्स को स्लाइड के भीतर एनीमेशन प्रकारों का प्रभार लेने का अधिकार देता है, जिससे अधिक इंटरैक्टिव और दृश्यमान रूप से आकर्षक प्रस्तुति की अनुमति मिलती है। इस ट्यूटोरियल में, हम चरण दर चरण .NET के लिए Aspose.Slides का उपयोग करके स्लाइड पर एनीमेशन प्रकारों को नियंत्रित करने का तरीका जानेंगे।
## आवश्यक शर्तें
इससे पहले कि हम ट्यूटोरियल में उतरें, सुनिश्चित करें कि आपके पास निम्नलिखित आवश्यक शर्तें हैं:
1.  .NET लाइब्रेरी के लिए Aspose.Slides: लाइब्रेरी को यहां से डाउनलोड और इंस्टॉल करें[यहाँ](https://releases.aspose.com/slides/net/).
2. .NET विकास वातावरण: अपनी मशीन पर एक .NET विकास वातावरण स्थापित करें।
## नामस्थान आयात करें
अपने .NET प्रोजेक्ट में, Aspose.Slides द्वारा प्रदान की गई कार्यक्षमताओं का लाभ उठाने के लिए आवश्यक नामस्थान आयात करके शुरुआत करें:
```csharp
using Aspose.Slides.Animation;
using Aspose.Slides.SlideShow;
using Aspose.Slides.Export;
```
## चरण 1: प्रोजेक्ट सेट करें
अपने प्रोजेक्ट के लिए एक नई निर्देशिका बनाएं और प्रेजेंटेशन फ़ाइल का प्रतिनिधित्व करने के लिए प्रेजेंटेशन क्लास को इंस्टेंट करें।
```csharp
string dataDir = "Your Document Directory";
bool IsExists = System.IO.Directory.Exists(dataDir);
if (!IsExists)
    System.IO.Directory.CreateDirectory(dataDir);
using (Presentation pres = new Presentation(dataDir + "AnimationOnSlide.pptx"))
{
    // आपका कोड यहां जाता है
}
```
## चरण 2: पहुँच प्रभाव अनुक्रम
MainSequence प्रॉपर्टी का उपयोग करके पहली स्लाइड के लिए प्रभाव अनुक्रम पुनर्प्राप्त करें।
```csharp
ISequence effectsSequence = pres.Slides[0].Timeline.MainSequence;
```
## चरण 3: पहले प्रभाव तक पहुंचें
इसके गुणों में हेरफेर करने के लिए मुख्य अनुक्रम का पहला प्रभाव प्राप्त करें।
```csharp
IEffect effect = effectsSequence[0];
```
## चरण 4: रिपीट सेटिंग्स को संशोधित करें
प्रभाव के समय/दोहराएँ गुण को "स्लाइड के अंत तक" में बदलें।
```csharp
effect.Timing.RepeatUntilEndSlide = true;
```
## चरण 5: प्रस्तुति सहेजें
परिवर्तनों को देखने के लिए संशोधित प्रस्तुति को सहेजें।
```csharp
pres.Save(RunExamples.OutPath + "AnimationOnSlide-out.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
```
अतिरिक्त प्रभावों के लिए इन चरणों को दोहराएं या अपनी प्रस्तुति आवश्यकताओं के अनुसार उन्हें अनुकूलित करें।
## निष्कर्ष
.NET के लिए Aspose.Slides के साथ अपने PowerPoint प्रस्तुतियों में गतिशील एनिमेशन को शामिल करना इतना आसान कभी नहीं रहा। यह चरण-दर-चरण मार्गदर्शिका आपको एनीमेशन प्रकारों को नियंत्रित करने के ज्ञान से सुसज्जित करती है, जिससे यह सुनिश्चित होता है कि आपकी स्लाइड आपके दर्शकों पर स्थायी प्रभाव छोड़ें।

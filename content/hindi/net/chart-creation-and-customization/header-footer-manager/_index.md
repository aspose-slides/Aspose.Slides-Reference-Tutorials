# .NET के लिए Aspose.Slides में डायनामिक हेडर और फ़ुटर बनाना

## नामस्थान आयात करें

आरंभ करने के लिए, आपको अपने प्रोजेक्ट में आवश्यक नामस्थान आयात करना होगा। ये नेमस्पेस Aspose.Slides के साथ काम करने के लिए आवश्यक उपकरण प्रदान करते हैं।

### चरण 1: नामस्थान आयात करें

अपने C# प्रोजेक्ट में, अपनी कोड फ़ाइल के शीर्ष पर निम्नलिखित नामस्थान जोड़ें:

```csharp
using Aspose.Slides;
using Aspose.Slides.Export;
```

## डायनामिक हेडर और फ़ुटर जोड़ना

अब, आइए आपके पावरपॉइंट प्रेजेंटेशन में डायनामिक हेडर और फ़ुटर जोड़ने की प्रक्रिया को चरण दर चरण तोड़ें।

### चरण 2: अपनी प्रस्तुति लोड करें

इस चरण में, आपको अपनी PowerPoint प्रस्तुति को अपने C# प्रोजेक्ट में लोड करना होगा।

```csharp
string dataDir = "Your Document Directory";

using (Presentation presentation = new Presentation(dataDir + "presentation.ppt"))
{
    // शीर्षलेख और पादलेख प्रबंधन के लिए आपका कोड यहां जाएगा।
    // ...
}
```

### चरण 3: हेडर और फ़ुटर मैनेजर तक पहुंचें

.NET के लिए Aspose.Slides हेडर और फ़ुटर को प्रबंधित करने का एक सुविधाजनक तरीका प्रदान करता है। हम आपकी प्रस्तुति में पहली स्लाइड के लिए हेडर और फ़ुटर मैनेजर तक पहुँचते हैं।

```csharp
IBaseSlideHeaderFooterManager headerFooterManager = presentation.Slides[0].HeaderFooterManager;
```

### चरण 4: फ़ुटर दृश्यता सेट करें

 पाद लेख प्लेसहोल्डर की दृश्यता को नियंत्रित करने के लिए, आप इसका उपयोग कर सकते हैं`SetFooterVisibility` तरीका।

```csharp
if (!headerFooterManager.IsFooterVisible)
{
    headerFooterManager.SetFooterVisibility(true);
}
```

### चरण 5: स्लाइड संख्या दृश्यता सेट करें

 इसी प्रकार, आप इसका उपयोग करके स्लाइड पेज नंबर प्लेसहोल्डर की दृश्यता को नियंत्रित कर सकते हैं`SetSlideNumberVisibility` तरीका।

```csharp
if (!headerFooterManager.IsSlideNumberVisible)
{
    headerFooterManager.SetSlideNumberVisibility(true);
}
```

### चरण 6: दिनांक और समय दृश्यता निर्धारित करें

 यह निर्धारित करने के लिए कि दिनांक-समय प्लेसहोल्डर दृश्यमान है या नहीं, इसका उपयोग करें`IsDateTimeVisible`संपत्ति। यदि यह दृश्यमान नहीं है, तो आप इसका उपयोग करके इसे दृश्यमान बना सकते हैं`SetDateTimeVisibility` तरीका।

```csharp
if (!headerFooterManager.IsDateTimeVisible)
{
    headerFooterManager.SetDateTimeVisibility(true);
}
```

### चरण 7: पादलेख और दिनांक-समय टेक्स्ट सेट करें

अंत में, आप अपने पादलेख और दिनांक-समय प्लेसहोल्डर्स के लिए टेक्स्ट सेट कर सकते हैं।

```csharp
headerFooterManager.SetFooterText("Footer text");
headerFooterManager.SetDateTimeText("Date and time text");
```

### चरण 8: अपनी प्रस्तुति सहेजें

सभी आवश्यक परिवर्तन करने के बाद, अपनी अद्यतन प्रस्तुति को सहेजें।

```csharp
presentation.Save(dataDir + "Presentation.ppt", SaveFormat.Ppt);
```

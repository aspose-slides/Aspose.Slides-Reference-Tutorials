## नामस्थान आयात करें
अपने .NET एप्लिकेशन में, Aspose.Slides कार्यात्मकताओं तक पहुंचने के लिए आवश्यक नामस्थान आयात करके प्रारंभ करें:
```csharp
using System.Drawing;
using System.Drawing.Imaging;
using Aspose.Slides;
```
## चरण 1: प्रेजेंटेशन सेट करें
एक प्रेजेंटेशन क्लास को इंस्टेंट करके प्रारंभ करें जो उस पावरपॉइंट प्रेजेंटेशन फ़ाइल का प्रतिनिधित्व करता है जिसके साथ आप काम करना चाहते हैं:
```csharp
string dataDir = "Your Documents Directory";
using (Presentation presentation = new Presentation(dataDir + "HelloWorld.pptx"))
{
    // थंबनेल बनाने के लिए आपका कोड यहां दिया गया है
}
```
## चरण 2: एक पूर्ण-स्तरीय छवि बनाएं

```csharp
using (Bitmap bitmap = presentation.Slides[0].Shapes[0].GetThumbnail(ShapeThumbnailBounds.Shape, 1, 1))
{
    //छवि को सहेजने के लिए आपका कोड यहां दिया गया है

```
## चरण 3: छवि को डिस्क पर सहेजें
प्रारूप निर्दिष्ट करते हुए उत्पन्न छवि को डिस्क पर सहेजें (इस मामले में, पीएनजी):

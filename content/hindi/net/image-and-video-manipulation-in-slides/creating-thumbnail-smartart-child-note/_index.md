## नामस्थान आयात करें
अपने C# प्रोजेक्ट में आवश्यक नामस्थान आयात करके प्रारंभ करें। ये नेमस्पेस Aspose.Slides के साथ काम करने के लिए आवश्यक कक्षाओं और विधियों तक पहुंच प्रदान करते हैं।
```csharp
using System.Drawing;
using System.Drawing.Imaging;
using Aspose.Slides.SmartArt;
using Aspose.Slides;
```
## चरण 1: त्वरित प्रस्तुति कक्षा
 को इंस्टेंट करके प्रारंभ करें`Presentation` क्लास, उस पीपीटीएक्स फ़ाइल का प्रतिनिधित्व करती है जिसके साथ आप काम करेंगे।
```csharp
string dataDir = "Your Documents Directory";
Presentation pres = new Presentation();
```
## चरण 2: स्मार्टआर्ट जोड़ें
 अब, प्रेजेंटेशन के भीतर एक स्लाइड में स्मार्टआर्ट जोड़ें। इस उदाहरण में, हम इसका उपयोग कर रहे हैं`BasicCycle` लेआउट।
```csharp
ISmartArt smart = pres.Slides[0].Shapes.AddSmartArt(10, 10, 400, 300, SmartArtLayoutType.BasicCycle);
```
## चरण 3: नोड संदर्भ प्राप्त करें
स्मार्टआर्ट में एक विशिष्ट नोड के साथ काम करने के लिए, इसके सूचकांक का उपयोग करके इसका संदर्भ प्राप्त करें।
```csharp
ISmartArtNode node = smart.Nodes[1];
```
## चरण 4: थंबनेल प्राप्त करें
स्मार्टआर्ट नोड के भीतर चाइल्ड नोट की थंबनेल छवि पुनर्प्राप्त करें।
```csharp
Bitmap bmp = node.Shapes[0].GetThumbnail();
```
## चरण 5: थंबनेल सहेजें
उत्पन्न थंबनेल छवि को एक निर्दिष्ट निर्देशिका में सहेजें।
```csharp
bmp.Save(dataDir + "SmartArt_ChildNote_Thumbnail_out.jpeg", ImageFormat.Jpeg);
```
आवश्यकतानुसार लेआउट और शैलियों को अनुकूलित करते हुए, अपनी प्रस्तुति में प्रत्येक स्मार्टआर्ट नोड के लिए इन चरणों को दोहराएं।

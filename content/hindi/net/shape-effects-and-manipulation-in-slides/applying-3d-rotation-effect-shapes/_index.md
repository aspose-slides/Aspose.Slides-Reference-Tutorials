## परिचय
आकर्षक और गतिशील प्रस्तुति स्लाइड बनाना प्रभावी संचार का एक प्रमुख पहलू है। .NET के लिए Aspose.Slides आपकी प्रस्तुतियों को बेहतर बनाने के लिए टूल का एक शक्तिशाली सेट प्रदान करता है, जिसमें आकृतियों पर 3D रोटेशन प्रभाव लागू करने की क्षमता भी शामिल है। इस ट्यूटोरियल में, हम .NET के लिए Aspose.Slides का उपयोग करके प्रेजेंटेशन स्लाइड्स में आकृतियों पर 3D रोटेशन प्रभाव लागू करने की प्रक्रिया के बारे में जानेंगे।

इससे पहले कि हम ट्यूटोरियल में उतरें, सुनिश्चित करें कि आपके पास निम्नलिखित आवश्यक शर्तें हैं:

- विकास परिवेश: अपना कोड लिखने और चलाने के लिए विज़ुअल स्टूडियो जैसा एक .NET विकास परिवेश स्थापित करें।
## नामस्थान आयात करें
अपने .NET प्रोजेक्ट में, Aspose.Slides की कार्यक्षमता का लाभ उठाने के लिए आवश्यक नेमस्पेस आयात करें। अपने कोड की शुरुआत में निम्नलिखित नामस्थान शामिल करें:
```csharp
using System.Drawing;
using Aspose.Slides.Export;
using Aspose.Slides;
```
## चरण 1: अपना प्रोजेक्ट सेट करें
अपने पसंदीदा .NET विकास परिवेश में एक नया प्रोजेक्ट बनाएं। सुनिश्चित करें कि आपने अपने प्रोजेक्ट में Aspose.Slides संदर्भ जोड़ा है।
## चरण 2: प्रस्तुति आरंभ करें
स्लाइड के साथ काम शुरू करने के लिए प्रेजेंटेशन क्लास को इंस्टेंट करें:
```csharp
Presentation pres = new Presentation();
```
## चरण 3: ऑटोशेप जोड़ें
स्लाइड में एक ऑटोशेप जोड़ें, उसका प्रकार, स्थिति और आयाम निर्दिष्ट करें:
```csharp
IShape autoShape = pres.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 30, 30, 200, 200);
```
## चरण 4: 3डी रोटेशन प्रभाव सेट करें
ऑटोशेप के लिए 3डी रोटेशन प्रभाव कॉन्फ़िगर करें:
```csharp
autoShape.ThreeDFormat.Depth = 6;
autoShape.ThreeDFormat.Camera.SetRotation(40, 35, 20);
autoShape.ThreeDFormat.Camera.CameraType = CameraPresetType.IsometricLeftUp;
autoShape.ThreeDFormat.LightRig.LightType = LightRigPresetType.Balanced;
```
## चरण 5: प्रस्तुति सहेजें
लागू 3डी रोटेशन प्रभाव के साथ संशोधित प्रस्तुति को सहेजें:
```csharp
pres.Save("Your Document Directory" + "Rotation_out.pptx", SaveFormat.Pptx);
```
## चरण 6: अन्य आकृतियों के लिए दोहराएँ
यदि आपके पास अतिरिक्त आकृतियाँ हैं, तो प्रत्येक आकृति के लिए चरण 3 से 5 दोहराएँ।

अपनी प्रस्तुति स्लाइड में आकृतियों में 3डी रोटेशन प्रभाव जोड़ने से उनकी दृश्य अपील में उल्लेखनीय वृद्धि हो सकती है। .NET के लिए Aspose.Slides के साथ, यह प्रक्रिया सरल हो जाती है, जिससे आप मनोरम प्रस्तुतियाँ बना सकते हैं।

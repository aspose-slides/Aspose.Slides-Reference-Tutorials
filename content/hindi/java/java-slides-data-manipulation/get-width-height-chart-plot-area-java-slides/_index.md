## परिचय

पावरपॉइंट प्रस्तुतियों में डेटा को विज़ुअलाइज़ करने के लिए चार्ट एक शक्तिशाली तरीका है। कभी-कभी, आपको विभिन्न कारणों से चार्ट के प्लॉट क्षेत्र के आयामों को जानने की आवश्यकता हो सकती है, जैसे कि चार्ट के भीतर तत्वों का आकार बदलना या उनका स्थान बदलना। यह मार्गदर्शिका प्रदर्शित करेगी कि Java और Aspose.Slides for Java का उपयोग करके प्लॉट क्षेत्र की चौड़ाई और ऊंचाई कैसे प्राप्त की जाए।

## आवश्यक शर्तें

 इससे पहले कि हम कोड में उतरें, सुनिश्चित करें कि आपके जावा प्रोजेक्ट में Aspose.Slides for Java लाइब्रेरी स्थापित और सेटअप है। आप लाइब्रेरी को Aspose वेबसाइट से डाउनलोड कर सकते हैं[यहाँ](https://releases.aspose.com/slides/java/).

## चरण 1: पर्यावरण की स्थापना

सुनिश्चित करें कि आपके जावा प्रोजेक्ट में Aspose.Slides for Java लाइब्रेरी जोड़ी गई है। आप अपने प्रोजेक्ट की निर्भरता में लाइब्रेरी को शामिल करके या JAR फ़ाइल को मैन्युअल रूप से जोड़कर ऐसा कर सकते हैं।

## चरण 2: एक पावरपॉइंट प्रेजेंटेशन बनाना

आइए एक पावरपॉइंट प्रेजेंटेशन बनाकर और उसमें एक स्लाइड जोड़कर शुरुआत करें। यह हमारे चार्ट के लिए कंटेनर के रूप में काम करेगा।

```java
String dataDir = "Your Document Directory";
Presentation pres = new Presentation(dataDir + "test.Pptx");
```

 प्रतिस्थापित करें`"Your Document Directory"` आपकी दस्तावेज़ निर्देशिका के पथ के साथ।

## चरण 3: एक चार्ट जोड़ना

अब, स्लाइड में एक क्लस्टर्ड कॉलम चार्ट जोड़ें। हम चार्ट लेआउट को भी मान्य करेंगे।

```java
Chart chart = (Chart) pres.getSlides().get_Item(0).getShapes().addChart(ChartType.ClusteredColumn, 100, 100, 500, 350);
chart.validateChartLayout();
```

यह कोड आयाम (500, 350) के साथ स्थिति (100, 100) पर एक क्लस्टर्ड कॉलम चार्ट बनाता है।

## चरण 4: प्लॉट क्षेत्र के आयाम प्राप्त करना

चार्ट के प्लॉट क्षेत्र की चौड़ाई और ऊंचाई प्राप्त करने के लिए, हम निम्नलिखित कोड का उपयोग कर सकते हैं:

```java
double x = chart.getPlotArea().getActualX();
double y = chart.getPlotArea().getActualY();
double w = chart.getPlotArea().getActualWidth();
double h = chart.getPlotArea().getActualHeight();
```

 अब, चर`x`, `y`, `w` , और`h` इसमें प्लॉट क्षेत्र के एक्स-निर्देशांक, वाई-निर्देशांक, चौड़ाई और ऊंचाई के लिए संबंधित मान शामिल हैं।

## चरण 5: प्रस्तुति को सहेजना

अंत में, प्रस्तुतिकरण को चार्ट के साथ सहेजें।

```java
pres.save(dataDir + "Chart_out.pptx", SaveFormat.Pptx);
```

 प्रतिस्थापित करना सुनिश्चित करें`"Chart_out.pptx"` आपके इच्छित आउटपुट फ़ाइल नाम के साथ।

## जावा स्लाइड में चार्ट प्लॉट क्षेत्र से चौड़ाई और ऊंचाई प्राप्त करने के लिए संपूर्ण स्रोत कोड

```java
// दस्तावेज़ निर्देशिका का पथ.
String dataDir = "Your Document Directory";
Presentation pres = new Presentation(dataDir + "test.Pptx");
try
{
	Chart chart = (Chart) pres.getSlides().get_Item(0).getShapes().addChart(ChartType.ClusteredColumn, 100, 100, 500, 350);
	chart.validateChartLayout();
	double x = chart.getPlotArea().getActualX();
	double y = chart.getPlotArea().getActualY();
	double w = chart.getPlotArea().getActualWidth();
	double h = chart.getPlotArea().getActualHeight();
	// प्रस्तुतिकरण को चार्ट के साथ सहेजें
	pres.save(dataDir + "Chart_out.pptx", SaveFormat.Pptx);
}
finally
{
	if (pres != null) pres.dispose();
}
```

## निष्कर्ष

इस लेख में, हमने जावा एपीआई के लिए Aspose.Slides का उपयोग करके जावा स्लाइड्स में चार्ट के प्लॉट क्षेत्र की चौड़ाई और ऊंचाई कैसे प्राप्त की जाए, इसके बारे में बताया है। यह जानकारी तब मूल्यवान हो सकती है जब आपको PowerPoint प्रस्तुतियों के भीतर अपने चार्ट के लेआउट को गतिशील रूप से समायोजित करने की आवश्यकता होती है।

## अक्सर पूछे जाने वाले प्रश्न

### मैं चार्ट प्रकार को क्लस्टर्ड कॉलम के अलावा किसी अन्य चीज़ में कैसे बदल सकता हूँ?

 आप चार्ट प्रकार को प्रतिस्थापित करके बदल सकते हैं`ChartType.ClusteredColumn` वांछित चार्ट प्रकार की गणना के साथ, जैसे`ChartType.Line` या`ChartType.Pie`.

### क्या मैं चार्ट के अन्य गुणों को संशोधित कर सकता हूँ?

हां, आप जावा एपीआई के लिए Aspose.Slides का उपयोग करके चार्ट के विभिन्न गुणों, जैसे डेटा, लेबल और फ़ॉर्मेटिंग को संशोधित कर सकते हैं। अधिक विवरण के लिए दस्तावेज़ देखें।

### क्या जावा के लिए Aspose.Slides पेशेवर PowerPoint स्वचालन के लिए उपयुक्त है?

हां, जावा के लिए Aspose.Slides जावा अनुप्रयोगों में PowerPoint कार्यों को स्वचालित करने के लिए एक शक्तिशाली लाइब्रेरी है। यह प्रस्तुतियों, स्लाइडों, आकृतियों, चार्टों और बहुत कुछ के साथ काम करने के लिए व्यापक सुविधाएँ प्रदान करता है।

### मैं जावा के लिए Aspose.Slides के बारे में और अधिक कैसे जान सकता हूँ?

 आप जावा दस्तावेज़ीकरण पृष्ठ के लिए Aspose.Slides पर व्यापक दस्तावेज़ और उदाहरण पा सकते हैं[यहाँ](https://reference.aspose.com/slides/java/).

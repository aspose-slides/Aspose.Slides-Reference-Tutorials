## जावा स्लाइड्स में पूर्वनिर्धारित दृश्य प्रकार के रूप में सहेजने का परिचय

इस चरण-दर-चरण मार्गदर्शिका में, हम यह पता लगाएंगे कि जावा के लिए Aspose.Slides का उपयोग करके पूर्वनिर्धारित दृश्य प्रकार के साथ एक प्रस्तुति को कैसे सहेजा जाए। इस कार्य को सफलतापूर्वक पूरा करने के लिए हम आपको आवश्यक कोड और स्पष्टीकरण प्रदान करेंगे।

## आवश्यक शर्तें

शुरू करने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित हैं:

- जावा प्रोग्रामिंग का बुनियादी ज्ञान।
- जावा लाइब्रेरी के लिए Aspose.Slides स्थापित।
- आपकी पसंद का एकीकृत विकास वातावरण (आईडीई)।

## अपना वातावरण स्थापित करना

आरंभ करने के लिए, अपना विकास परिवेश स्थापित करने के लिए इन चरणों का पालन करें:

1. अपनी आईडीई में एक नया जावा प्रोजेक्ट बनाएं।
2. निर्भरता के रूप में अपने प्रोजेक्ट में जावा लाइब्रेरी के लिए Aspose.Slides जोड़ें।

अब जब आपका वातावरण तैयार हो गया है, तो कोड के साथ आगे बढ़ें।

## चरण 1: एक प्रस्तुति बनाना

किसी प्रस्तुति को पूर्वनिर्धारित दृश्य प्रकार के साथ सहेजने का प्रदर्शन करने के लिए, हम पहले एक नई प्रस्तुति बनाएंगे। प्रेजेंटेशन बनाने के लिए कोड यहां दिया गया है:

```java
// दस्तावेज़ निर्देशिका का पथ.
String dataDir = "Your Document Directory";
// प्रेजेंटेशन फ़ाइल खुल रही है
Presentation presentation = new Presentation();
```

 इस कोड में, हम एक नया बनाते हैं`Presentation` ऑब्जेक्ट, जो हमारी PowerPoint प्रस्तुति का प्रतिनिधित्व करता है।

## चरण 2: दृश्य प्रकार सेट करना

इसके बाद, हम अपनी प्रस्तुति के लिए दृश्य प्रकार सेट करेंगे। दृश्य प्रकार परिभाषित करते हैं कि खोले जाने पर प्रस्तुतिकरण कैसे प्रदर्शित होगा। इस उदाहरण में, हम इसे "स्लाइड मास्टर व्यू" पर सेट करेंगे। यहाँ कोड है:

```java
// दृश्य प्रकार सेट करना
presentation.getViewProperties().setLastView(ViewType.SlideMasterView);
```

 उपरोक्त कोड में, हम इसका उपयोग करते हैं`setLastView` की विधि`ViewProperties` दृश्य प्रकार को सेट करने के लिए क्लास`SlideMasterView`. आप आवश्यकतानुसार अन्य दृश्य प्रकार चुन सकते हैं।

## चरण 3: प्रस्तुति को सहेजना

अब जब हमने अपना प्रेजेंटेशन बना लिया है और दृश्य प्रकार सेट कर लिया है, तो प्रेजेंटेशन को सहेजने का समय आ गया है। हम इसे PPTX फॉर्मेट में सेव करेंगे. यहाँ कोड है:

```java
// प्रस्तुतिकरण सहेजा जा रहा है
presentation.save(dataDir + "SetViewType_out.pptx", SaveFormat.Pptx);
```

 इस कोड में, हम इसका उपयोग करते हैं`save` की विधि`Presentation` प्रेजेंटेशन को निर्दिष्ट फ़ाइल नाम और प्रारूप के साथ सहेजने के लिए क्लास।

## जावा स्लाइड्स में पूर्वनिर्धारित दृश्य प्रकार के रूप में सहेजने के लिए संपूर्ण स्रोत कोड

```java
// दस्तावेज़ निर्देशिका का पथ.
String dataDir = "Your Document Directory";
// प्रेजेंटेशन फ़ाइल खुल रही है
Presentation presentation = new Presentation();
try
{
	// दृश्य प्रकार सेट करना
	presentation.getViewProperties().setLastView(ViewType.SlideMasterView);
	// प्रस्तुतिकरण सहेजा जा रहा है
	presentation.save(dataDir + "SetViewType_out.pptx", SaveFormat.Pptx);
}
finally
{
	if (presentation != null) presentation.dispose();
}
```

## निष्कर्ष

इस ट्यूटोरियल में, हमने सीखा है कि जावा के लिए Aspose.Slides का उपयोग करके जावा में पूर्वनिर्धारित दृश्य प्रकार के साथ एक प्रस्तुति को कैसे सहेजा जाए। दिए गए कोड और चरणों का पालन करके, आप आसानी से अपनी प्रस्तुतियों का दृश्य प्रकार सेट कर सकते हैं और उन्हें वांछित प्रारूप में सहेज सकते हैं।

## अक्सर पूछे जाने वाले प्रश्न

### मैं दृश्य प्रकार को "स्लाइड मास्टर व्यू" के अलावा किसी अन्य चीज़ में कैसे बदलूँ?

 दृश्य प्रकार को "स्लाइड मास्टर व्यू" के अलावा किसी अन्य चीज़ में बदलने के लिए, बस बदलें`ViewType.SlideMasterView` वांछित दृश्य प्रकार के साथ, जैसे`ViewType.NormalView` या`ViewType.SlideSorterView`, उस कोड में जहां हम दृश्य प्रकार सेट करते हैं।

### क्या मैं प्रस्तुतिकरण में अलग-अलग स्लाइडों के लिए दृश्य गुण सेट कर सकता हूँ?

हाँ, आप Java के लिए Aspose.Slides का उपयोग करके अलग-अलग स्लाइड के लिए दृश्य गुण सेट कर सकते हैं। आप प्रेजेंटेशन में स्लाइडों के माध्यम से पुनरावृत्ति करके प्रत्येक स्लाइड के गुणों तक अलग-अलग पहुंच और हेरफेर कर सकते हैं।

### मैं अपनी प्रस्तुति को अन्य किन प्रारूपों में सहेज सकता हूँ?

Java के लिए Aspose.Slides PPTX, PDF, TIFF, HTML और अन्य सहित विभिन्न आउटपुट स्वरूपों का समर्थन करता है। आप उपयुक्त का उपयोग करके अपनी प्रस्तुति को सहेजते समय वांछित प्रारूप निर्दिष्ट कर सकते हैं`SaveFormat` एनम मान.

### क्या जावा के लिए Aspose.Slides प्रस्तुतियों की बैच प्रोसेसिंग के लिए उपयुक्त है?

हां, जावा के लिए Aspose.Slides बैच प्रोसेसिंग कार्यों के लिए उपयुक्त है। आप जावा कोड का उपयोग करके एकाधिक प्रस्तुतियों के प्रसंस्करण को स्वचालित कर सकते हैं, परिवर्तन लागू कर सकते हैं और उन्हें थोक में सहेज सकते हैं।

### जावा के लिए Aspose.Slides के लिए मुझे अधिक जानकारी और दस्तावेज़ कहां मिल सकते हैं?

 जावा के लिए Aspose.Slides से संबंधित व्यापक दस्तावेज़ीकरण और संदर्भों के लिए, कृपया दस्तावेज़ीकरण वेबसाइट पर जाएँ:[जावा दस्तावेज़ीकरण के लिए Aspose.Slides](https://reference.aspose.com/slides/java/).

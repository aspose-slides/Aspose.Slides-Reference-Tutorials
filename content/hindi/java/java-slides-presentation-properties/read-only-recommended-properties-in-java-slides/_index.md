## जावा स्लाइड्स में केवल-पढ़ने के लिए अनुशंसित गुणों को सक्षम करने का परिचय

इस ट्यूटोरियल में, हम यह पता लगाएंगे कि जावा के लिए Aspose.Slides का उपयोग करके PowerPoint प्रस्तुतियों के लिए केवल-पढ़ने के लिए अनुशंसित गुणों को कैसे सक्षम किया जाए। केवल-पढ़ने के लिए अनुशंसित गुण तब उपयोगी हो सकते हैं जब आप उपयोगकर्ताओं को बिना कोई बदलाव किए प्रस्तुति देखने के लिए प्रोत्साहित करना चाहते हैं। ये गुण सुझाव देते हैं कि प्रस्तुतिकरण को केवल पढ़ने योग्य मोड में खोला जाना चाहिए। इसे प्राप्त करने के लिए हम आपको जावा स्रोत कोड के साथ चरण-दर-चरण मार्गदर्शिका प्रदान करेंगे।

## आवश्यक शर्तें

 शुरू करने से पहले, सुनिश्चित करें कि आपके प्रोजेक्ट में जावा लाइब्रेरी के लिए Aspose.Slides सेटअप है। आप इसे यहां से डाउनलोड कर सकते हैं[जावा वेबसाइट के लिए Aspose.Slides](https://products.aspose.com/slides/java/).

## चरण 1: एक नया पावरपॉइंट प्रेजेंटेशन बनाएं

हम Java के लिए Aspose.Slides का उपयोग करके एक नई PowerPoint प्रस्तुति बनाकर शुरुआत करेंगे। यदि आपके पास पहले से ही कोई प्रेजेंटेशन है, तो आप इस चरण को छोड़ सकते हैं।

```java
String outPptxPath = RunExamples.getOutPath() + "ReadOnlyRecommended.pptx";
Presentation pres = new Presentation();
```

उपरोक्त कोड में, हमने आउटपुट पावरपॉइंट फ़ाइल के लिए पथ परिभाषित किया है और एक नया प्रेजेंटेशन ऑब्जेक्ट बनाया है।

## चरण 2: केवल-पढ़ने के लिए अनुशंसित संपत्ति सक्षम करें

अब, प्रेजेंटेशन के लिए केवल-पढ़ने के लिए अनुशंसित प्रॉपर्टी को सक्षम करें।

```java
try
{
    pres.getProtectionManager().setReadOnlyRecommended(true);
    pres.save(outPptxPath, SaveFormat.Pptx);
}
finally
{
    if (pres != null) pres.dispose();
}
```

 इस कोड स्निपेट में, हम इसका उपयोग करते हैं`getProtectionManager().setReadOnlyRecommended(true)` केवल-पढ़ने के लिए अनुशंसित संपत्ति को सेट करने की विधि`true`. यह सुनिश्चित करता है कि जब कोई प्रेजेंटेशन खोलेगा, तो उन्हें इसे केवल-पढ़ने के लिए मोड में खोलने के लिए प्रेरित किया जाएगा।

## चरण 3: प्रस्तुति सहेजें

अंत में, हम प्रेजेंटेशन को केवल-पढ़ने के लिए अनुशंसित प्रॉपर्टी सक्षम करके सहेजते हैं।

## जावा स्लाइड्स में केवल-पढ़ने के लिए अनुशंसित गुणों के लिए संपूर्ण स्रोत कोड

```java
String outPptxPath = RunExamples.getOutPath() + "ReadOnlyRecommended.pptx";
Presentation pres = new Presentation();
try
{
	pres.getProtectionManager().setReadOnlyRecommended(true);
	pres.save(outPptxPath, SaveFormat.Pptx);
}
finally
{
	if (pres != null) pres.dispose();
}
```

## निष्कर्ष

इस ट्यूटोरियल में, आपने सीखा है कि जावा के लिए Aspose.Slides का उपयोग करके पावरपॉइंट प्रेजेंटेशन के लिए केवल-पढ़ने के लिए अनुशंसित प्रॉपर्टी को कैसे सक्षम किया जाए। यह सुविधा तब सहायक हो सकती है जब आप संपादन को प्रतिबंधित करना चाहते हैं और दर्शकों को प्रेजेंटेशन को केवल-पढ़ने के लिए मोड में उपयोग करने के लिए प्रोत्साहित करना चाहते हैं। आप प्रेजेंटेशन के लिए पासवर्ड सेट करके सुरक्षा को और बढ़ा सकते हैं।

## अक्सर पूछे जाने वाले प्रश्न

### मैं केवल-पढ़ने के लिए अनुशंसित संपत्ति को कैसे अक्षम कर सकता हूं?

केवल-पढ़ने के लिए अनुशंसित संपत्ति को अक्षम करने के लिए, बस निम्नलिखित कोड का उपयोग करें:

```java
pres.getProtectionManager().setReadOnlyRecommended(false);
```

### क्या मैं केवल पढ़ने योग्य अनुशंसित प्रस्तुतिकरण के लिए पासवर्ड सेट कर सकता हूँ?

हां, आप जावा के लिए Aspose.Slides का उपयोग करके केवल-पढ़ने के लिए अनुशंसित प्रस्तुति के लिए पासवर्ड सेट कर सकते हैं। आप इसका उपयोग कर सकते हैं`setPassword` प्रेजेंटेशन के लिए पासवर्ड सेट करने की विधि। यदि कोई पासवर्ड सेट है, तो उपयोगकर्ताओं को प्रेजेंटेशन खोलने के लिए इसे दर्ज करना होगा, यहां तक कि केवल-पढ़ने के लिए मोड में भी।

```java
pres.getProtectionManager().setPassword("YourPassword");
```

 प्रतिस्थापित करना याद रखें`"YourPassword"` अपने इच्छित पासवर्ड के साथ.

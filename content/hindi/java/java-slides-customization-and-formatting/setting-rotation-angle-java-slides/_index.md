## जावा स्लाइड्स में रोटेशन एंगल सेट करने का परिचय

इस ट्यूटोरियल में, हम जावा लाइब्रेरी के लिए Aspose.Slides का उपयोग करके चार्ट अक्ष शीर्षक में टेक्स्ट के लिए रोटेशन कोण कैसे सेट करें, इसका पता लगाएंगे। रोटेशन कोण को समायोजित करके, आप अपनी प्रस्तुति आवश्यकताओं के अनुरूप बेहतर ढंग से अपने चार्ट के अक्ष शीर्षकों की उपस्थिति को अनुकूलित कर सकते हैं।

## आवश्यक शर्तें

शुरू करने से पहले, सुनिश्चित करें कि आपके जावा प्रोजेक्ट में Aspose.Slides for Java लाइब्रेरी स्थापित और सेटअप है। आप Aspose वेबसाइट से लाइब्रेरी डाउनलोड कर सकते हैं और उनके दस्तावेज़ में दिए गए इंस्टॉलेशन निर्देशों का पालन कर सकते हैं।

## चरण 1: एक प्रेजेंटेशन बनाएं

सबसे पहले, आपको एक नया प्रेजेंटेशन बनाना होगा या किसी मौजूदा को लोड करना होगा। इस उदाहरण में, हम एक नई प्रस्तुति बनाएंगे:

```java
// दस्तावेज़ निर्देशिका का पथ.
String dataDir = "Your Document Directory";
Presentation pres = new Presentation();
```

## चरण 2: स्लाइड में एक चार्ट जोड़ें

इसके बाद, हम स्लाइड में एक चार्ट जोड़ेंगे। इस उदाहरण में, हम एक संकुल स्तंभ चार्ट जोड़ रहे हैं:

```java
try
{
    IChart chart = pres.getSlides().get_Item(0).getShapes().addChart(ChartType.ClusteredColumn, 50, 50, 450, 300);
```

## चरण 3: अक्ष शीर्षक के लिए रोटेशन कोण सेट करें

अक्ष शीर्षक के लिए रोटेशन कोण सेट करने के लिए, आपको चार्ट के ऊर्ध्वाधर अक्ष शीर्षक तक पहुंचने और उसके रोटेशन कोण को समायोजित करने की आवश्यकता होगी। यहां बताया गया है कि आप यह कैसे कर सकते हैं:

```java
    chart.getAxes().getVerticalAxis().setTitle(true);
    chart.getAxes().getVerticalAxis().getTitle().getTextFormat().getTextBlockFormat().setRotationAngle(90);
```

इस कोड स्निपेट में, हम रोटेशन कोण को 90 डिग्री पर सेट कर रहे हैं, जो टेक्स्ट को लंबवत रूप से घुमाएगा। आप कोण को अपने इच्छित मूल्य पर समायोजित कर सकते हैं।

## चरण 4: प्रस्तुति सहेजें

अंत में, प्रेजेंटेशन को PowerPoint फ़ाइल में सहेजें:

```java
    pres.save(dataDir + "test.pptx", SaveFormat.Pptx);
}
finally
{
    if (pres != null) pres.dispose();
}
```

## जावा स्लाइड्स में रोटेशन कोण सेट करने के लिए संपूर्ण स्रोत कोड

```java
// दस्तावेज़ निर्देशिका का पथ.
String dataDir = "Your Document Directory";
Presentation pres = new Presentation();
try
{
	IChart chart = pres.getSlides().get_Item(0).getShapes().addChart(ChartType.ClusteredColumn, 50, 50, 450, 300);
	chart.getAxes().getVerticalAxis().setTitle(true);
	chart.getAxes().getVerticalAxis().getTitle().getTextFormat().getTextBlockFormat().setRotationAngle(90);
	pres.save(dataDir + "test.pptx", SaveFormat.Pptx);
}
finally
{
	if (pres != null) pres.dispose();
}
```

## निष्कर्ष

इस ट्यूटोरियल में, आपने सीखा कि जावा के लिए Aspose.Slides का उपयोग करके चार्ट अक्ष शीर्षक में टेक्स्ट के लिए रोटेशन कोण कैसे सेट किया जाए। यह सुविधा आपको आकर्षक प्रस्तुतियाँ बनाने के लिए अपने चार्ट के स्वरूप को अनुकूलित करने की अनुमति देती है। अपने चार्ट के लिए वांछित स्वरूप प्राप्त करने के लिए विभिन्न घूर्णन कोणों के साथ प्रयोग करें।

## अक्सर पूछे जाने वाले प्रश्न

### मैं स्लाइड में अन्य टेक्स्ट तत्वों के लिए रोटेशन कोण कैसे बदल सकता हूं?

आप समान दृष्टिकोण का उपयोग करके अन्य टेक्स्ट तत्वों, जैसे आकार या टेक्स्ट बॉक्स के लिए रोटेशन कोण को बदल सकते हैं। तत्व के टेक्स्ट प्रारूप तक पहुंचें और आवश्यकतानुसार रोटेशन कोण सेट करें।

### क्या मैं क्षैतिज अक्ष शीर्षक में भी पाठ को घुमा सकता हूँ?

हाँ, आप घूर्णन कोण को समायोजित करके क्षैतिज अक्ष शीर्षक में पाठ को घुमा सकते हैं। बस रोटेशन कोण को अपने इच्छित मान पर सेट करें, जैसे ऊर्ध्वाधर पाठ के लिए 90 डिग्री या क्षैतिज पाठ के लिए 0 डिग्री।

### चार्ट शीर्षकों के लिए अन्य कौन से फ़ॉर्मेटिंग विकल्प उपलब्ध हैं?

जावा के लिए Aspose.Slides फ़ॉन्ट शैलियों, रंगों और संरेखण सहित चार्ट शीर्षकों के लिए विभिन्न स्वरूपण विकल्प प्रदान करता है। आप चार्ट शीर्षकों को अनुकूलित करने के बारे में अधिक जानकारी के लिए दस्तावेज़ का पता लगा सकते हैं।

### क्या चार्ट अक्ष शीर्षक में पाठ के घूर्णन को एनिमेट करना संभव है?

हां, आप जावा के लिए Aspose.Slides का उपयोग करके, चार्ट अक्ष शीर्षक सहित पाठ तत्वों में एनीमेशन प्रभाव जोड़ सकते हैं। अपनी प्रस्तुतियों में एनिमेशन जोड़ने के बारे में जानकारी के लिए दस्तावेज़ देखें।

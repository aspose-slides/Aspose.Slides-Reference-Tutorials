## जावा के लिए Aspose.Slides में स्थिति अक्ष सेट करने का परिचय

इस ट्यूटोरियल में, हम सीखेंगे कि जावा के लिए Aspose.Slides का उपयोग करके चार्ट में स्थिति अक्ष कैसे सेट करें। जब आप अपने चार्ट के स्वरूप और लेआउट को अनुकूलित करना चाहते हैं तो अक्ष की स्थिति बनाना उपयोगी हो सकता है। हम एक क्लस्टर्ड कॉलम चार्ट बनाएंगे और श्रेणियों के बीच क्षैतिज अक्ष की स्थिति को समायोजित करेंगे।

## आवश्यक शर्तें

 शुरू करने से पहले, सुनिश्चित करें कि आपके जावा प्रोजेक्ट में Aspose.Slides for Java लाइब्रेरी स्थापित और सेटअप है। आप यहां से लाइब्रेरी डाउनलोड कर सकते हैं[यहाँ](https://releases.aspose.com/slides/java/).

## चरण 1: एक प्रस्तुति बनाना

सबसे पहले, आइए काम करने के लिए एक नई प्रस्तुति बनाएं:

```java
String dataDir = "Your Document Directory";
Presentation pres = new Presentation();
```

 प्रतिस्थापित करना सुनिश्चित करें`"Your Document Directory"` आपकी दस्तावेज़ निर्देशिका के वास्तविक पथ के साथ।

## चरण 2: एक चार्ट जोड़ना

इसके बाद, हम स्लाइड में एक क्लस्टर्ड कॉलम चार्ट जोड़ेंगे। हम चार्ट का प्रकार, स्थिति (x, y निर्देशांक), और चार्ट के आयाम (चौड़ाई और ऊंचाई) निर्दिष्ट करते हैं:

```java
IChart chart = pres.getSlides().get_Item(0).getShapes().addChart(ChartType.ClusteredColumn, 50, 50, 450, 300);
```

यहां, हमने 450 की चौड़ाई और 300 की ऊंचाई के साथ स्थिति (50, 50) पर एक क्लस्टर्ड कॉलम चार्ट जोड़ा है। आप इन मानों को आवश्यकतानुसार समायोजित कर सकते हैं।

## चरण 3: स्थिति अक्ष निर्धारित करना

श्रेणियों के बीच स्थिति अक्ष सेट करने के लिए, आप निम्नलिखित कोड का उपयोग कर सकते हैं:

```java
chart.getAxes().getHorizontalAxis().setAxisBetweenCategories(true);
```

यह कोड श्रेणियों के बीच प्रदर्शित करने के लिए क्षैतिज अक्ष सेट करता है, जो कुछ चार्ट लेआउट के लिए उपयोगी हो सकता है।

## चरण 4: प्रस्तुति को सहेजना

अंत में, आइए प्रस्तुतीकरण को चार्ट के साथ सहेजें:

```java
pres.save(dataDir + "AsposeClusteredColumnChart.pptx", SaveFormat.Pptx);
```

 प्रतिस्थापित करें`"AsposeClusteredColumnChart.pptx"` आपके इच्छित फ़ाइल नाम के साथ.

इतना ही! आपने सफलतापूर्वक क्लस्टर्ड कॉलम चार्ट बना लिया है और जावा के लिए Aspose.Slides का उपयोग करके श्रेणियों के बीच स्थिति अक्ष सेट कर दिया है।

## संपूर्ण स्रोत कोड
```java
String dataDir = "Your Document Directory";
Presentation pres = new Presentation();
try
{
	IChart chart = pres.getSlides().get_Item(0).getShapes().addChart(ChartType.ClusteredColumn, 50, 50, 450, 300);
	chart.getAxes().getHorizontalAxis().setAxisBetweenCategories(true);
	pres.save(dataDir + "AsposeScatterChart.pptx", SaveFormat.Pptx);
}
finally
{
	if (pres != null) pres.dispose();
}
```

## निष्कर्ष

इस ट्यूटोरियल में, हमने पता लगाया है कि जावा के लिए Aspose.Slides का उपयोग करके चार्ट में स्थिति अक्ष कैसे सेट करें। इस गाइड में उल्लिखित चरणों का पालन करके, आपने सीखा है कि क्लस्टर्ड कॉलम चार्ट कैसे बनाया जाए और श्रेणियों के बीच क्षैतिज अक्ष की स्थिति बनाकर उसके स्वरूप को कैसे अनुकूलित किया जाए। जावा के लिए Aspose.Slides चार्ट और प्रस्तुतियों के साथ काम करने के लिए शक्तिशाली सुविधाएँ प्रदान करता है, जिससे यह जावा डेवलपर्स के लिए एक मूल्यवान टूल बन जाता है।

## अक्सर पूछे जाने वाले प्रश्न

### मैं चार्ट को और अधिक अनुकूलित कैसे करूँ?

आप चार्ट के विभिन्न पहलुओं को अनुकूलित कर सकते हैं, जिसमें डेटा श्रृंखला, चार्ट शीर्षक, किंवदंतियाँ और बहुत कुछ शामिल हैं। को देखें[जावा दस्तावेज़ीकरण के लिए Aspose.Slides](https://reference.aspose.com/slides/java/) विस्तृत निर्देशों और उदाहरणों के लिए।

### क्या मैं चार्ट प्रकार बदल सकता हूँ?

 हां, आप चार्ट प्रकार को संशोधित करके बदल सकते हैं`ChartType` चार्ट जोड़ते समय पैरामीटर. जावा के लिए Aspose.Slides विभिन्न चार्ट प्रकारों जैसे बार चार्ट, लाइन चार्ट और बहुत कुछ का समर्थन करता है।

### मुझे और अधिक उदाहरण और दस्तावेज़ कहां मिल सकते हैं?

 आप व्यापक दस्तावेज़ीकरण और अधिक उदाहरण पा सकते हैं[जावा दस्तावेज़ीकरण के लिए Aspose.Slides](https://reference.aspose.com/slides/java/) पृष्ठ।

जब आप सिस्टम संसाधनों को जारी करने का काम पूरा कर लें तो प्रेजेंटेशन ऑब्जेक्ट का निपटान करना याद रखें:

```java
if (pres != null) pres.dispose();
```

इस ट्यूटोरियल के लिए बस इतना ही। आपने जावा के लिए Aspose.Slides का उपयोग करके चार्ट में स्थिति अक्ष सेट करना सीख लिया है।

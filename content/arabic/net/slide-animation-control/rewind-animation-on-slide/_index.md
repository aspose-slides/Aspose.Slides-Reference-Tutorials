## مقدمة للرسوم المتحركة مع Aspose.Slides

يمكن للرسوم المتحركة أن تبث الحياة في عروضك التقديمية، مما يجعلها أكثر جاذبية وجاذبية بصريًا. Aspose.Slides for .NET هي مكتبة قوية تمكن المطورين من العمل مع عروض PowerPoint التقديمية برمجياً، بما في ذلك إضافة الرسوم المتحركة وتعديلها وإدارتها.

## المتطلبات الأساسية

قبل أن نبدأ، تأكد من توفر ما يلي:

- Visual Studio: قم بتثبيت Visual Studio أو أي بيئة تطوير .NET أخرى.
-  Aspose.Slides: قم بتنزيل وتثبيت Aspose.Slides لمكتبة .NET من[هنا](https://releases.aspose.com/slides/net/).

## الخطوة 1: تحميل ملف العرض التقديمي

أولاً، لنبدأ بتحميل ملف PowerPoint التقديمي الذي يحتوي على الشريحة مع الرسوم المتحركة. إليك مقتطف الشفرة لتحقيق ذلك:

```csharp
using Aspose.Slides;

// قم بتحميل العرض التقديمي
string presentationPath = "path_to_your_presentation.pptx";
using (Presentation presentation = new Presentation(presentationPath))
{
    // الرمز الخاص بك هنا
}
```

## الخطوة 2: الوصول إلى الشرائح والرسوم المتحركة

بعد ذلك، نحتاج إلى الوصول إلى الشريحة المحددة والرسوم المتحركة الخاصة بها. في هذه الخطوة، سنستهدف الشريحة التي تحتوي على الرسوم المتحركة التي تريد إرجاعها. إليك الطريقة:

```csharp
// افترض أن فهرس الشريحة هو 0 (الشريحة الأولى)
ISlide slide = presentation.Slides[0];

// الوصول إلى الرسوم المتحركة للشريحة
ISlideAnimation slideAnimation = slide.SlideShowTransition;
```

## الخطوة 3: إعادة لف الرسوم المتحركة

الآن يأتي الجزء المثير – إعادة عرض الرسوم المتحركة. يسمح لك Aspose.Slides بإعادة تعيين الرسوم المتحركة على الشريحة، مما يؤدي بشكل فعال إلى إعادة الشريحة إلى حالتها الأولية. إليك مقتطف الشفرة لتحقيق ذلك:

```csharp
// إرجاع الرسوم المتحركة على الشريحة
slideAnimation.StopAfterRepeats = 0; // اضبط عدد التكرارات على 0
```

## الخطوة 4: حفظ العرض التقديمي المعدل

بعد إرجاع الرسوم المتحركة، حان الوقت لحفظ العرض التقديمي المعدل. يمكنك حفظه باسم جديد أو الكتابة فوق الملف الموجود. إليك كيفية حفظ العرض التقديمي:

```csharp
// احفظ العرض التقديمي المعدل
string outputPath = "path_to_save_modified_presentation.pptx";
presentation.Save(outputPath, SaveFormat.Pptx);
```

## مقدمة إلى Aspose.Slides لـ .NET

Aspose.Slides for .NET هي مكتبة قوية تمكن المطورين من إنشاء عروض PowerPoint التقديمية ومعالجتها وتحويلها باستخدام إطار عمل .NET. فهو يوفر مجموعة واسعة من الميزات للعمل مع الشرائح والأشكال والنصوص والصور والرسوم المتحركة والمزيد.

## إعداد بيئة التطوير الخاصة بك

قبل أن نبدأ، تحتاج إلى إعداد بيئة التطوير الخاصة بك. اتبع الخطوات التالية:

1. قم بتنزيل وتثبيت Visual Studio من[تحميل فيجوال ستوديو](https://visualstudio.microsoft.com/downloads/).
2. قم بإنشاء مشروع .NET جديد (تطبيق وحدة التحكم، على سبيل المثال) في Visual Studio.

## تحميل عرض تقديمي ل PowerPoint

للبدء، ستحتاج إلى عرض تقديمي من PowerPoint للعمل عليه. تأكد من أن لديك ملف PowerPoint جاهزًا.

```csharp
using Aspose.Slides;

// قم بتحميل عرض PowerPoint التقديمي
using var presentation = new Presentation("presentation.pptx");
```

## الوصول إلى الرسوم المتحركة وتعديلها

الآن بعد أن قمنا بتحميل العرض التقديمي الخاص بنا، دعنا نصل إلى الرسوم المتحركة وتعديلها على شريحة معينة. في هذا المثال، لنفترض أننا نريد تكرار الرسوم المتحركة على الشريحة رقم 2.

```csharp
// الوصول إلى الشريحة حسب الفهرس (يعتمد على 0)
var slideIndex = 1;
var slide = presentation.Slides[slideIndex];

// الوصول إلى الرسوم المتحركة للشريحة
var animations = slide.Timeline.MainSequence;
```

## تكرار الرسوم المتحركة على الشريحة

لتكرار الرسوم المتحركة، سنقوم باستنساخ الرسوم المتحركة وإضافتها إلى الشريحة مرة أخرى. سيؤدي هذا إلى إنشاء تأثير يحلق. وإليك كيف يمكنك تحقيق ذلك:

```csharp
// استنساخ الرسوم المتحركة وإضافتها مرة أخرى
var clonedAnimations = animations.CloneSequence();
animations.AddSequence(clonedAnimations);
```

## اختبار وتصدير العرض التقديمي المعدل

بعد تعديل الرسوم المتحركة، حان الوقت لاختبار العرض التقديمي وتصديره. يمكنك تصديره إلى تنسيقات مختلفة مثل PPTX أو PDF أو الصور.

```csharp
// احفظ العرض التقديمي المعدل
presentation.Save("modified_presentation.pptx", SaveFormat.Pptx);
```

في هذا الدليل، اكتشفنا كيفية تكرار الرسوم المتحركة على شريحة باستخدام Aspose.Slides لـ .NET. لقد بدأنا بالتعريف بالمكتبة وتهيئة بيئة التطوير. بعد ذلك، قمنا بتحميل عرض PowerPoint التقديمي، والوصول إلى الرسوم المتحركة وتعديلها، وأخيرًا، قمنا بتنفيذ ميزة تكرار الرسوم المتحركة. يعمل Aspose.Slides for .NET على تمكين المطورين من إنشاء عروض تقديمية ديناميكية وجذابة برمجيًا.

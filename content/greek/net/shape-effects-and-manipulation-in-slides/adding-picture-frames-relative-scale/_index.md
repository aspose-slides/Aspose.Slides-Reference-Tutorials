## Εισαγωγή
Το Aspose.Slides for .NET είναι μια ισχυρή βιβλιοθήκη που επιτρέπει στους προγραμματιστές να δημιουργούν, να χειρίζονται και να μετατρέπουν παρουσιάσεις PowerPoint στις εφαρμογές τους .NET χωρίς κόπο. Σε αυτό το σεμινάριο, θα βουτήξουμε στη διαδικασία προσθήκης πλαισίων εικόνων με σχετικό ύψος κλίμακας χρησιμοποιώντας το Aspose.Slides για .NET. Ακολουθήστε αυτόν τον οδηγό βήμα προς βήμα για να βελτιώσετε τις δεξιότητές σας στη δημιουργία παρουσιάσεων.

Πριν ξεκινήσουμε, βεβαιωθείτε ότι έχετε τα εξής:
- Βασικές γνώσεις γλώσσας προγραμματισμού C#.
- Το Visual Studio ή οποιοδήποτε άλλο προτιμώμενο περιβάλλον ανάπτυξης C# έχει εγκατασταθεί.
- Το Aspose.Slides για τη βιβλιοθήκη .NET προστέθηκαν στο έργο σας.
## Εισαγωγή χώρων ονομάτων
Ξεκινήστε εισάγοντας τους απαραίτητους χώρους ονομάτων στον κώδικα C#. Αυτό το βήμα διασφαλίζει ότι έχετε πρόσβαση στις κλάσεις και τις λειτουργίες που παρέχονται από τη βιβλιοθήκη Aspose.Slides.
```csharp
using System.Drawing;
using Aspose.Slides.Export;

```
## Βήμα 1: Ρύθμιση του έργου σας
Ξεκινήστε δημιουργώντας ένα νέο έργο C# στο περιβάλλον ανάπτυξης που προτιμάτε. Βεβαιωθείτε ότι έχετε προσθέσει τη βιβλιοθήκη Aspose.Slides for .NET στο έργο σας κάνοντας αναφορά σε αυτήν.
## Βήμα 2: Φόρτωση παρουσίασης και εικόνας
```csharp
string dataDir = "Your Document Directory";
using (Presentation presentation = new Presentation())
{
    // Φόρτωση εικόνας για προσθήκη στη συλλογή εικόνων παρουσίασης
    Image img = new Bitmap(dataDir + "aspose-logo.jpg");
    IPPImage image = presentation.Images.AddImage(img);
    // ...
}
```
Σε αυτό το βήμα, δημιουργούμε ένα νέο αντικείμενο παρουσίασης και φορτώνουμε την εικόνα που θέλουμε να προσθέσουμε στην παρουσίαση.
## Βήμα 3: Προσθέστε το πλαίσιο εικόνας στη διαφάνεια
```csharp
IPictureFrame pf = presentation.Slides[0].Shapes.AddPictureFrame(ShapeType.Rectangle, 50, 50, 100, 100, image);
```
Τώρα, προσθέστε μια κορνίζα στην πρώτη διαφάνεια της παρουσίασης. Προσαρμόστε τις παραμέτρους όπως ο τύπος σχήματος, η θέση και οι διαστάσεις σύμφωνα με τις απαιτήσεις σας.
## Βήμα 4: Ορίστε το πλάτος και το ύψος της σχετικής κλίμακας
```csharp
pf.RelativeScaleHeight = 0.8f;
pf.RelativeScaleWidth = 1.35f;
```
Ρυθμίστε το σχετικό ύψος και πλάτος κλίμακας για την κορνίζα για να επιτύχετε το επιθυμητό εφέ κλιμάκωσης.
## Βήμα 5: Αποθήκευση παρουσίασης
```csharp
presentation.Save(dataDir + "Adding Picture Frame with Relative Scale_out.pptx", SaveFormat.Pptx);
```
Τέλος, αποθηκεύστε την παρουσίαση με την προστιθέμενη κορνίζα στην καθορισμένη μορφή εξόδου.

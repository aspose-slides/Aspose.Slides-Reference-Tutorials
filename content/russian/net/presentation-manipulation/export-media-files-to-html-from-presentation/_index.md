## Загрузка презентации

Для начала давайте загрузим презентацию PowerPoint с помощью библиотеки Aspose.Slides. В качестве ссылки вы можете использовать следующий фрагмент кода:

```csharp
using Aspose.Slides;

// Загрузите презентацию
using (Presentation presentation = new Presentation("your-presentation.pptx"))
{
    // Здесь будет ваш код для извлечения и экспорта медиафайлов.
}
```

## Извлечение медиафайлов

Далее нам нужно извлечь медиафайлы (изображения, видео, аудио) из презентации. Aspose.Slides предоставляет простой способ добиться этого. Вот пример:

```csharp
// Перебирать каждый слайд презентации.
foreach (ISlide slide in presentation.Slides)
{
    // Перебирать каждую фигуру на слайде
    foreach (IShape shape in slide.Shapes)
    {
        // Проверьте, является ли фигура медиа-фреймом
        if (shape is IMediaFrame)
        {
            IMediaFrame mediaFrame = (IMediaFrame)shape;

            // Извлечь медиафайл из кадра
            byte[] mediaBytes = mediaFrame.MediaData.BinaryData;
            
            // Здесь будет находиться ваш код для экспорта медиабайтов.
        }
    }
}
```

## Экспорт медиафайлов в HTML

После извлечения медиафайлов мы можем приступить к их экспорту в HTML. Для этого мы воспользуемся возможностями Aspose.Slides для создания HTML-представлений медиафайлов. Вот как:

```csharp
using Aspose.Slides.Export;

// Предположим, что mediaBytes содержит байты медиафайла.
using (MemoryStream stream = new MemoryStream(mediaBytes))
{
    // Сохранение мультимедиа в формате HTML
    using (HtmlOptions htmlOptions = new HtmlOptions())
    {
        presentation.MediaEncoder.EncodeToHtml(stream, htmlOptions);
    }
}
```

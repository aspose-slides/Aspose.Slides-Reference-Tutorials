## Введение в Aspose.Slides для .NET

Aspose.Slides for .NET — это комплексная библиотека, которая позволяет .NET-разработчикам работать с презентациями PowerPoint без необходимости использования Microsoft Office. Он предоставляет широкий спектр функций, включая создание, редактирование и управление слайдами, фигурами, текстом, изображениями и многим другим.

## Предварительные условия

Прежде чем мы начнем, убедитесь, что у вас есть следующие предварительные условия:

1. Visual Studio установлена на вашем компьютере.
2. Базовое понимание C# и .NET framework.
3.  Aspose.Slides для библиотеки .NET. Вы можете скачать его с[здесь](https://releases.aspose.com/slides/net/).

## Настройка проекта

Начнем с настройки нового проекта C# в Visual Studio:

1. Откройте Visual Studio.
2. Нажмите «Создать новый проект».
3. Выберите «Консольное приложение (.NET Framework/Core)».
4. Выберите подходящее имя и место для вашего проекта.
5. Нажмите «Создать».

Затем добавьте ссылку на библиотеку Aspose.Slides for .NET в свой проект. Щелкните правой кнопкой мыши «Ссылки» в обозревателе решений, выберите «Управление пакетами NuGet», найдите «Aspose.Slides» и установите пакет.

## Добавление смещения растяжения слева для рамки изображения

Чтобы добавить смещение растяжения влево для рамки изображения с помощью Aspose.Slides for .NET, выполните следующие действия:

1.  Загрузите файл презентации, используя`Presentation` сорт.
2. Найдите слайд, содержащий рамку изображения, которую вы хотите изменить.
3. Получите доступ к форме рамки изображения, перебирая фигуры на слайде.
4.  Примените смещение растяжения влево, используя`PictureFrame` сорт.

## Пример кода

```csharp
using Aspose.Slides;
using Aspose.Slides.ShapeManagers;

namespace PictureFrameStretchOffsetExample
{
    class Program
    {
        static void Main(string[] args)
        {
            // Загрузите презентацию
            using (Presentation presentation = new Presentation("sample.pptx"))
            {
                // Получить первый слайд
                ISlide slide = presentation.Slides[0];

                // Перебирать фигуры на слайде
                foreach (IShape shape in slide.Shapes)
                {
                    if (shape is IPictureFrame)
                    {
                        IPictureFrame pictureFrame = (IPictureFrame)shape;

                        // Применить смещение растяжения влево
                        pictureFrame.PictureFormat.StretchOffsetX = -10;
                    }
                }

                // Сохраните измененную презентацию
                presentation.Save("output.pptx", SaveFormat.Pptx);
            }
        }
    }
}
```

В этом примере мы загружаем презентацию, перебираем фигуры на первом слайде и, если находим форму рамки изображения, применяем смещение растяжения -10 влево.

## Введение в Aspose.Slides для .NET

Aspose.Slides for .NET — это мощный API, который позволяет разработчикам создавать, изменять и манипулировать презентациями PowerPoint, не требуя установки Microsoft Office или PowerPoint. Он предоставляет широкий спектр функций для работы с различными элементами презентаций, включая слайды, фигуры, текст, изображения и рамки объектов OLE.

## Предварительные условия

Прежде чем мы начнем, убедитесь, что у вас есть следующее:

- Установлена Visual Studio или любая совместимая среда разработки .NET.
-  Aspose.Slides для библиотеки .NET. Вы можете скачать его с[здесь](https://releases.aspose.com/slides/net/).

## Загрузка презентации

Начнем с загрузки существующей презентации PowerPoint с помощью Aspose.Slides для .NET. Если у вас нет презентации для тестирования, вы можете создать новую или скачать образец презентации.

```csharp
using Aspose.Slides;

// Загрузите презентацию
using var presentation = new Presentation("sample.pptx");
```

 Фреймы объектов OLE (связывание и внедрение объектов) позволяют встраивать такие объекты, как изображения, документы и другие файлы, в слайд PowerPoint. Чтобы получить доступ к фреймам объектов OLE на слайде, вы можете перебирать фигуры и проверять наличие экземпляров`OleObjectFrameEx`.

```csharp
// Перелистывание слайдов
foreach (var slide in presentation.Slides)
{
    // Перебор фигур на слайде
    foreach (var shape in slide.Shapes)
    {
        if (shape is OleObjectFrameEx oleObject)
        {
            //Доступ к свойствам объекта OLE
            var title = oleObject.Title;
            var data = oleObject.ObjectData;
            
            // Выполните дальнейшие действия
        }
    }
}
```

## Замена названия изображения

 Чтобы заменить заголовок изображения кадра объекта OLE, вы можете просто обновить`Title` собственность`OleObjectFrameEx` пример.

```csharp
foreach (var slide in presentation.Slides)
{
    foreach (var shape in slide.Shapes)
    {
        if (shape is OleObjectFrameEx oleObject)
        {
            // Обновить заголовок
            oleObject.Title = "New Picture Title";
        }
    }
}
```

## Сохранение измененной презентации

После внесения необходимых изменений необходимо сохранить измененную презентацию. Вы можете сохранить его в различных форматах, таких как PPTX, PDF или изображения.

```csharp
// Сохранить презентацию
presentation.Save("modified.pptx", SaveFormat.Pptx);
```

## Создание миниатюр из слайдов

Теперь давайте сгенерируем миниатюры слайдов, используя класс SlideUtil:

```csharp
using Aspose.Slides.Util;

// Создать миниатюру слайда
var thumbnail = SlideUtil.GetSlideThumbnail(slide, 1.0f);
```

## Сохранение миниатюр на диск

Создав миниатюры, вы можете сохранить их на локальный диск:

```csharp
// Сохранить миниатюру на диск
thumbnail.Save("slide-thumbnail.png", ImageFormat.Png);
```

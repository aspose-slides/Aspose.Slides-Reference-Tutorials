## Введение в масштабирование разделов

Масштабирование разделов — это фантастический способ упорядочить различные части презентации и перемещаться по ней без необходимости переключаться между слайдами вручную. Они обеспечивают структурированный поток вашего контента и позволяют вам глубже вникать в конкретные темы, сохраняя при этом четкий обзор. С помощью Aspose.Slides для .NET вы можете легко масштабировать разделы своей презентации, добавляя нотку профессионализма и интерактивности.

## Добавление разделов в презентацию

В этом разделе мы узнаем, как организовать презентацию по разделам, которые послужат основой для создания масштабов разделов.

Чтобы добавить разделы в презентацию, выполните следующие действия:

1.  Создайте новый экземпляр`Presentation` класс из Aspose.Slides.

```csharp
using Aspose.Slides;
// ...
Presentation presentation = new Presentation();
```

2. Добавляйте слайды в презентацию и группируйте их по разделам.

```csharp
// Добавление слайдов
ISlide slide1 = presentation.Slides.AddEmptySlide(presentation.LayoutSlides[0]);
ISlide slide2 = presentation.Slides.AddEmptySlide(presentation.LayoutSlides[0]);

// Добавление разделов
presentation.SectionSlides.AddSection(slide1, "Introduction");
presentation.SectionSlides.AddSection(slide2, "Main Content");
```

## Создание масштабов разделов

Теперь, когда вы организовали презентацию по разделам, давайте приступим к созданию масштабов разделов, которые обеспечат плавную навигацию между этими разделами.

1. Создайте новый слайд, который будет служить слайдом «Содержание», содержащим гиперссылки на ваши разделы.

```csharp
ISlide tocSlide = presentation.Slides.AddEmptySlide(presentation.LayoutSlides[0]);
```

2. Добавьте на слайд «Оглавление» интерактивные фигуры, каждая из которых ссылается на определенный раздел.

```csharp
// Добавление кликабельных фигур
IShape introShape = tocSlide.Shapes.AddAutoShape(ShapeType.Rectangle, 100, 100, 200, 50);
introShape.TextFrame.Text = "Introduction";
introShape.ActionSettings.HyperlinkClick = new HyperlinkClick(presentation.SectionSlides[0]);

IShape contentShape = tocSlide.Shapes.AddAutoShape(ShapeType.Rectangle, 100, 200, 200, 50);
contentShape.TextFrame.Text = "Main Content";
contentShape.ActionSettings.HyperlinkClick = new HyperlinkClick(presentation.SectionSlides[1]);
```

## Настройка поведения масштабирования раздела

Вы можете настроить масштабирование разделов в соответствии с потребностями вашей презентации. Например, вы можете определить, будет ли увеличенный раздел запускаться автоматически или по щелчку пользователя.

Чтобы автоматически запустить масштабирование раздела:

```csharp
presentation.SlideShowSettings.ShowType = SlideShowType.SectionZoom;
presentation.SlideShowSettings.StartingSlide = presentation.SectionSlides[0];
```

Чтобы начать масштабирование раздела по клику пользователя:

```csharp
presentation.SlideShowSettings.ShowType = SlideShowType.SectionZoom;
presentation.SlideShowSettings.StartingSlide = presentation.Slides[0];
```

## Часто задаваемые вопросы

### Как загрузить Aspose.Slides для .NET?

 Вы можете скачать библиотеку Aspose.Slides для .NET с веб-сайта Aspose:[Загрузите Aspose.Slides для .NET](https://releases.aspose.com/slides/net/).

### Могу ли я настроить внешний вид кликабельных фигур?

Да, вы можете настроить внешний вид интерактивных фигур, настроив их свойства, такие как цвет, размер и шрифт.

### Доступно ли масштабирование раздела во всех макетах слайдов?

Да, вы можете реализовать масштабирование разделов на слайдах с разными макетами. Процесс остается тем же, независимо от макета слайда.

### Могу ли я увеличить масштаб разделов между непоследовательными слайдами?

Да, Aspose.Slides позволяет создавать масштабирование разделов между непоследовательными слайдами, обеспечивая гибкость в планировании потока презентации.

### Как добавить анимацию при масштабировании раздела?

Масштабирование разделов само по себе не поддерживает анимацию. Однако вы можете комбинировать масштабирование разделов с другими анимациями и переходами, чтобы создать динамичную презентацию.

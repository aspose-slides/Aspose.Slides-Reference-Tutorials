# Marqueurs d'image personnalisés dans les graphiques en courbes PowerPoint avec Aspose.Slides

Aspose.Slides pour .NET offre des fonctionnalités complètes pour créer des présentations PowerPoint riches en fonctionnalités par programmation. Cet exemple montre comment créer un graphique en courbes avec des marqueurs d'image personnalisés, ce qui vous permet d'améliorer la visualisation des données avec des images de marque ou significatives au lieu des marqueurs standard. La bibliothèque offre un contrôle total sur les éléments du graphique, permettant aux développeurs de créer des présentations professionnelles avec un minimum d'effort.

## Instructions d'installation

### Prérequis
- Visual Studio 2010 ou supérieur
- .NET Framework 4.0 ou supérieur ou .NET Standard 2.0
- .NET Core 2.0 ou supérieur

### Installer via le gestionnaire de packages NuGet
1. Ouvrez votre projet dans Visual Studio
2. Accédez à Outils → Gestionnaire de packages NuGet → Gérer les packages NuGet pour la solution
3. Rechercher « Aspose.Slides »
4. Cliquez sur Installer pour ajouter le package à votre projet

### Installer via la console du gestionnaire de packages
```csharp
PM> Install-Package Aspose.Slides.NET
```

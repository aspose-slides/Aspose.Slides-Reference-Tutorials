# Modification de graphiques dans PowerPoint avec Aspose.Slides

Aspose.Slides pour .NET fournit de puissantes API pour créer et modifier des présentations PowerPoint par programmation. Cet exemple montre comment utiliser des graphiques dans des présentations existantes en mettant à jour les données, en ajoutant de nouvelles séries et en modifiant les types de graphiques, le tout sans avoir besoin d'installer Microsoft PowerPoint.

## Instructions d'installation

1. Installer via le gestionnaire de packages NuGet :
   ```
   Install-Package Aspose.Slides
   ```

2. Ou en utilisant .NET CLI :
   ```
   dotnet add package Aspose.Slides
   ```

3. Ajoutez les directives using suivantes :
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Charts;
   ```

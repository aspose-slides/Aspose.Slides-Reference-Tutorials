Aspose.Slides pour .NET offre de puissantes fonctionnalités pour créer et manipuler des présentations PowerPoint par programmation. Cet exemple montre comment créer un histogramme groupé avec des catégories et des séries de données personnalisées.

## Installation

```csharp
// Installer via le gestionnaire de packages NuGet
Install-Package Aspose.Slides
```

Ou ajoutez la référence suivante à votre fichier de projet :

```xml
<PackageReference Include="Aspose.Slides" Version="23.2.0" />
```

# Créer un graphique arborescent dans PowerPoint à l'aide d'Aspose.Slides
  
Créer un graphique Treemap dans PowerPoint est un jeu d'enfant avec Aspose.Slides pour .NET. Cette puissante API vous permet de concevoir des présentations visuelles époustouflantes grâce au code. ## Instructions d'installation :  
Pour démarrer avec Aspose.Slides pour .NET, suivez ces étapes :  

1. Téléchargez Aspose.Slides pour .NET depuis le site officiel [Page de publication d'Aspose](https://releases.aspose.com/slides/net/).  
2. Installez le package à l’aide du gestionnaire de packages NuGet dans Visual Studio :  
   ```shell
   Install-Package Aspose.Slides
   ```  
3. Ajoutez les espaces de noms requis dans votre code :  
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Charts;
   ```

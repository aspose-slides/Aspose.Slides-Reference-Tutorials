## Instructions d'installation
Pour démarrer avec Aspose.Slides pour .NET, suivez ces étapes :

1. Télécharger Aspose.Slides pour .NET  
   - Téléchargez la dernière version à partir du [Page de téléchargement d'Aspose.Slides pour .NET](https://releases.aspose.com/slides/net/).

2. Installer via le gestionnaire de packages NuGet  
   Exécutez la commande suivante dans la console du gestionnaire de packages NuGet :
   ```shell
   Install-Package Aspose.Slides
   ```

3. Ajouter les références requises  
   Dans votre projet Visual Studio, ajoutez une référence à `Aspose.Slides.dll`.

4. Importer des espaces de noms  
   Ajoutez les espaces de noms nécessaires au début de votre code :
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Charts;
   using System.Drawing;
   ```

5. Appliquer une licence (facultatif mais recommandé)  
   Pour éviter les limitations d’évaluation, vous pouvez [appliquer une licence](https://purchase.aspose.com/temporary-license/).

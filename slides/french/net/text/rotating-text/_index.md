# Créer du texte vertical dans PowerPoint avec Aspose.Slides

Aspose.Slides pour .NET offre de puissantes fonctionnalités pour créer et manipuler des présentations PowerPoint par programmation. Cet exemple montre comment créer une présentation contenant un rectangle avec du texte pivoté verticalement. Le texte est orienté à 270 degrés, ce qui permet une lecture de bas en haut. Cette technique est utile pour créer des étiquettes latérales, des mises en page de texte uniques ou des éléments de design spécifiques dans vos présentations.

## Instructions d'installation

1. Installer Aspose.Slides pour .NET
   
   Installez le package Aspose.Slides à l'aide du gestionnaire de packages NuGet :
   
   ```csharp
   Install-Package Aspose.Slides
   ```
   
   Ou en utilisant la CLI .NET :
   
   ```csharp
   dotnet add package Aspose.Slides
   ```

2. Ajouter des références
   
   Ajoutez l’instruction using suivante à votre fichier C# :
   
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

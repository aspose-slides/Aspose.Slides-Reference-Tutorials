# Créer des puces à plusieurs niveaux dans PowerPoint avec Aspose.Slides

Aspose.Slides pour .NET fournit une API puissante pour créer et manipuler des présentations PowerPoint par programmation. Cet exemple montre comment créer une présentation avec des puces à plusieurs niveaux, ajouter une forme automatique avec un cadre de texte et formater des paragraphes avec différents styles de puces et niveaux de retrait. Vous pouvez facilement créer du contenu hiérarchique avec des symboles de puces personnalisés pour chaque niveau de retrait, idéal pour créer des présentations structurées et professionnelles sans avoir besoin d'installer Microsoft PowerPoint.

## Instructions d'installation

1. Installer via le gestionnaire de packages NuGet :
   ```
   Install-Package Aspose.Slides
   ```

2. Installation via .NET CLI :
   ```
   dotnet add package Aspose.Slides
   ```

3. Installation via la console du gestionnaire de packages :
   ```
   PM> Install-Package Aspose.Slides
   ```

4. Installation manuelle :
   - Téléchargez la dernière version depuis [Aspose.Slides pour .NET](https://releases.aspose.com/slides/net/)
   - Ajoutez une référence à Aspose.Slides.dll dans votre projet

# Créer des présentations PowerPoint avec du texte formaté

Aspose.Slides pour .NET est une API puissante qui permet aux développeurs de créer, manipuler et convertir des présentations PowerPoint sans utiliser Microsoft PowerPoint. Cet exemple montre comment créer une présentation simple contenant une forme rectangulaire avec des paragraphes de texte formatés.

Le code crée une nouvelle présentation, ajoute une forme rectangulaire à la première diapositive, puis ajoute plusieurs paragraphes avec différentes options de mise en forme du texte. Il montre comment définir les propriétés de police, comme la taille et la famille, pour des portions de texte spécifiques.

## Instructions d'installation

1. Installer via le gestionnaire de packages NuGet :
   ```
   Install-Package Aspose.Slides
   ```

2. Installation via .NET CLI :
   ```
   dotnet add package Aspose.Slides
   ```

3. Installation via la console du gestionnaire de packages :
   ```
   PM> Install-Package Aspose.Slides
   ```

4. Ajoutez une référence à votre projet :
   Après l'installation, ajoutez une référence à votre projet :
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

# Créer des présentations PowerPoint avec TextBox dans Aspose.Slides pour .NET

Aspose.Slides pour .NET est une API puissante qui permet aux développeurs de créer, modifier et convertir des présentations PowerPoint sans utiliser Microsoft PowerPoint. Cet exemple montre comment créer une présentation avec une zone de texte personnalisée, appliquer une mise en forme et l'enregistrer au format PPTX.

## Instructions d'installation

1. Installer via le gestionnaire de packages NuGet :
   ```
   Install-Package Aspose.Slides
   ```

2. Installation via .NET CLI :
   ```
   dotnet add package Aspose.Slides
   ```

3. Installer manuellement :
   - Téléchargez la dernière version à partir du [Page des versions d'Aspose.Slides pour .NET](https://releases.aspose.com/slides/net/)
   - Ajoutez la référence DLL à votre projet

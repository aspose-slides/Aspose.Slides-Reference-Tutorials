# Création de puces personnalisées dans les présentations PowerPoint

Aspose.Slides pour .NET offre des fonctionnalités robustes pour créer et manipuler des présentations PowerPoint par programmation. Cet exemple se concentre spécifiquement sur l'implémentation de puces personnalisées dans des blocs de texte, montrant comment contrôler le type, la couleur, le retrait et la taille des puces. Le code crée une présentation rectangulaire contenant deux paragraphes : l'un avec une puce symbolique et l'autre avec une puce numérotée.

## Instructions d'installation

1. Installez Aspose.Slides pour .NET à l'aide du gestionnaire de packages NuGet :
   ```
   Install-Package Aspose.Slides
   ```

2. Alternativement, téléchargez directement le package :
   - Visitez le [Page de téléchargement d'Aspose.Slides](https://downloads.aspose.com/slides/net)
   - Téléchargez la dernière version
   - Extraire et référencer Aspose.Slides.dll dans votre projet

3. Ajoutez l'instruction using nécessaire :
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

# Rotation des titres et étiquettes de graphiques dans PowerPoint avec Aspose.Slides

Aspose.Slides pour .NET offre de puissantes fonctionnalités pour créer et manipuler des présentations PowerPoint par programmation. Cet exemple montre comment créer une présentation avec un histogramme groupé intégrant des éléments de texte pivotés, notamment le titre et les libellés de données du graphique pour une meilleure lisibilité et un meilleur attrait visuel.

## Instructions d'installation

1. Installer via le gestionnaire de packages NuGet :
   ```
   Install-Package Aspose.Slides
   ```

2. Ou installez via .NET CLI :
   ```
   dotnet add package Aspose.Slides
   ```

3. Ajoutez l’espace de noms à votre fichier C# :
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Charts;
   ```

# Créer des zones de texte formatées dans PowerPoint avec Aspose.Slides

Aspose.Slides pour .NET est une API puissante qui permet aux développeurs de créer, modifier et convertir des présentations Microsoft PowerPoint sans avoir à installer Microsoft Office. Cet exemple montre comment créer une présentation avec une zone de texte formatée, en appliquant des styles, tailles et couleurs de police personnalisés pour améliorer l'attrait visuel.

## Instructions d'installation

1. Installer via le gestionnaire de packages NuGet :
   ```
   Install-Package Aspose.Slides
   ```

2. Installation via .NET CLI :
   ```
   dotnet add package Aspose.Slides
   ```

3. Ajoutez une référence à votre projet manuellement :
   - Téléchargez la dernière version depuis [Aspose.Slides pour .NET](https://releases.aspose.com/slides/net/)
   - Ajoutez la DLL comme référence dans votre projet

4. Initialiser une licence :
   ```csharp
   // Initialiser une licence
   License license = new License();
   license.SetLicense("Aspose.Slides.lic");
   ```

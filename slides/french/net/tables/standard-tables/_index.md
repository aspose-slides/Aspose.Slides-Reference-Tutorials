# Créez des tableaux standardisés dans PowerPoint avec Aspose.Slides

Aspose.Slides pour .NET est une API puissante qui permet aux développeurs de créer, lire, écrire, modifier et convertir des présentations PowerPoint par programmation, sans recourir à Microsoft PowerPoint. Cet exemple montre comment créer un tableau standardisé avec des bordures personnalisées dans une présentation PowerPoint.

## Instructions d'installation

1. Installer via le gestionnaire de packages NuGet :
   ```
   Install-Package Aspose.Slides
   ```

2. Ou via .NET CLI :
   ```
   dotnet add package Aspose.Slides
   ```

3. Ajoutez l’espace de noms suivant à votre fichier C# :
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Tables;
   using System.Drawing;
   ```

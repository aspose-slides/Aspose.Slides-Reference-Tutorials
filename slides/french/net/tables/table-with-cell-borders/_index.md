# Créer des tableaux sans bordures dans les présentations PowerPoint

Aspose.Slides pour .NET est une bibliothèque puissante qui permet aux développeurs de créer, manipuler et convertir des présentations PowerPoint par programmation. Cet exemple montre comment créer un tableau dans une présentation et personnaliser son apparence en supprimant toutes les bordures des cellules.

Le code crée une nouvelle présentation avec un tableau sur la première diapositive, puis parcourt chaque cellule pour définir toutes les bordures (haut, bas, gauche et droite) pour qu'elles n'aient aucun remplissage, créant ainsi un tableau sans bordure.

## Instructions d'installation

1. Installer via le gestionnaire de packages NuGet :
   ```
   Install-Package Aspose.Slides
   ```

2. Ou installez via .NET CLI :
   ```
   dotnet add package Aspose.Slides
   ```

3. Ajoutez un espace de noms à votre code :
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

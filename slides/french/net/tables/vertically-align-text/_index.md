# Créer des tableaux avec du texte vertical dans les présentations PowerPoint

Aspose.Slides pour .NET offre de puissantes fonctionnalités pour la création et la manipulation de présentations PowerPoint par programmation. Cet exemple montre comment créer un tableau avec une mise en forme personnalisée, incluant l'alignement vertical et la rotation du texte, une exigence courante pour les présentations professionnelles spécialisées et la visualisation de données.

## Instructions d'installation

1. Installez Aspose.Slides pour .NET via le gestionnaire de packages NuGet :
   ```
   Install-Package Aspose.Slides
   ```
   Ou via .NET CLI :
   ```
   dotnet add package Aspose.Slides
   ```

2. Ajoutez des références à votre projet :
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

3. Télécharger les ressources de documentation :
   - [Référence de l'API](https://reference.aspose.com/slides/net/)
   - [Guide du développeur](https://docs.aspose.com/slides/net/)

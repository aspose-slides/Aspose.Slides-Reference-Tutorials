# Ajout d'images aux cellules d'un tableau dans PowerPoint avec Aspose.Slides

Ce guide explique comment ajouter des images aux cellules d'un tableau PowerPoint par programmation avec Aspose.Slides pour .NET. Cet exemple montre comment créer une présentation, ajouter un tableau avec des dimensions personnalisées, charger une image et l'insérer dans une cellule de tableau avec la mise en forme appropriée. Cette technique est utile pour créer des présentations professionnelles avec un contenu visuel riche et des données tabulaires organisées.

## Instructions d'installation

1. Installez Aspose.Slides pour .NET en utilisant l’une de ces méthodes :
   - Gestionnaire de paquets NuGet : `Install-Package Aspose.Slides.NET`
   - .NET CLI : `dotnet add package Aspose.Slides.NET`
   - Ajoutez une référence à Aspose.Slides.dll directement dans votre projet

2. Ajoutez une référence d’espace de noms dans votre fichier C# :
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

3. Télécharger la documentation et la référence API :
   - [Documentation Aspose.Slides pour .NET](https://docs.aspose.com/slides/net/)
   - [Référence de l'API Aspose.Slides pour .NET](https://reference.aspose.com/slides/net/)

# Créer des ellipses remplies de dégradés dans PowerPoint avec Aspose.Slides

Cet exemple montre comment créer par programmation une présentation PowerPoint contenant une ellipse avec un effet de remplissage dégradé personnalisé. Le code crée une nouvelle présentation, ajoute une ellipse à la première diapositive et applique un remplissage dégradé linéaire avec des points de couleur rouge et violet. Aspose.Slides pour .NET fournit une API complète pour créer et manipuler des présentations PowerPoint avec un contrôle précis des formes, de la mise en forme et des effets.

## Instructions d'installation

1. Installez Aspose.Slides pour .NET en utilisant l’une de ces méthodes :
   - Gestionnaire de paquets NuGet : `Install-Package Aspose.Slides.NET`
   - .NET CLI : `dotnet add package Aspose.Slides.NET`
   - Console du gestionnaire de paquets : `Install-Package Aspose.Slides.NET`

2. Ajoutez l’instruction using suivante à votre fichier C# :
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

# Modifier les chemins géométriques dans les formes PowerPoint avec Aspose.Slides

Aspose.Slides pour .NET offre de puissantes fonctionnalités de modification programmatique des présentations PowerPoint, notamment la manipulation avancée des formes. Cet exemple montre comment créer des formes personnalisées en modifiant le chemin géométrique d'une forme existante, notamment en supprimant un segment d'une forme de cœur pour créer une variante unique.

Le chemin géométrique définit le contour d'une forme à l'aide d'une série de points et de segments. En accédant à ces chemins et en les modifiant, les développeurs peuvent créer des formes hautement personnalisées, non disponibles dans la bibliothèque de formes standard de PowerPoint.

## Instructions d'installation

1. Installer via le gestionnaire de packages NuGet :
   ```
   Install-Package Aspose.Slides
   ```

2. Installation via .NET CLI :
   ```
   dotnet add package Aspose.Slides
   ```

3. Ajoutez la directive using dans votre code :
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

4. Ajoutez l'initialisation de la licence au démarrage de votre application :
   ```csharp
   // Si vous avez un permis
   License license = new License();
   license.SetLicense("path/to/Aspose.Slides.lic");
   ```

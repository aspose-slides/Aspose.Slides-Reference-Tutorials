# Créer des formes de groupe dans PowerPoint avec Aspose.Slides

Aspose.Slides pour .NET fournit des API complètes pour créer et manipuler des présentations PowerPoint par programmation. Cet exemple montre comment créer un groupe de formes, y ajouter plusieurs rectangles, les positionner correctement, ajouter un encadrement et enregistrer la présentation. Les groupes de formes sont essentiels pour organiser les formes apparentées devant être manipulées comme une seule unité, offrant ainsi davantage de structure et de contrôle sur les éléments de votre présentation.

## Instructions d'installation

1. Installez Aspose.Slides pour .NET en utilisant l’une de ces méthodes :

   - Gestionnaire de paquets NuGet :
     ```
     Install-Package Aspose.Slides.NET
     ```

   - Console du gestionnaire de paquets :
     ```
     PM> Install-Package Aspose.Slides.NET
     ```

   - .NET CLI :
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. Ajoutez des références à votre projet :
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

3. Configurer une licence (facultatif pour une utilisation en production) :
   ```csharp
   // Définir la licence pour supprimer les restrictions d'évaluation
   License license = new License();
   license.SetLicense("Aspose.Slides.lic");
   ```

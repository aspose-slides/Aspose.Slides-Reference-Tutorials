# Création et rotation de formes rectangulaires dans PowerPoint

Aspose.Slides pour .NET est une API puissante qui permet aux développeurs de créer, modifier et convertir des présentations PowerPoint sans avoir à installer Microsoft PowerPoint. Cet exemple montre comment créer une présentation avec un rectangle pivoté, illustrant les capacités de manipulation de formes de la bibliothèque.

Avec Aspose.Slides, vous pouvez ajouter par programmation diverses formes à vos diapositives et appliquer des transformations comme la rotation. Cette fonctionnalité est particulièrement utile pour créer des présentations dynamiques avec un contrôle précis des éléments visuels.

## Instructions d'installation

1. Installer via le gestionnaire de packages NuGet :
   ```
   Install-Package Aspose.Slides
   ```

2. Installation via .NET CLI :
   ```
   dotnet add package Aspose.Slides
   ```

3. Ajoutez la directive using dans votre code :
   ```csharp
   using Aspose.Slides;
   ```

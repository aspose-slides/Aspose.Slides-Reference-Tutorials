# Créer des formes elliptiques dans PowerPoint avec Aspose.Slides

Aspose.Slides pour .NET est une API puissante qui permet aux développeurs de créer, lire, écrire, modifier et convertir des présentations PowerPoint sans utiliser Microsoft PowerPoint. Cet exemple montre comment créer une forme elliptique dans une présentation, appliquer une mise en forme, notamment la couleur de remplissage et les propriétés de ligne, et enregistrer le résultat au format PPTX. Le code illustre la capacité d'Aspose.Slides à manipuler les formes avec un contrôle précis de leur apparence et de leur positionnement.

## Instructions d'installation

1. Installer via le gestionnaire de packages NuGet :
   ```
   Install-Package Aspose.Slides
   ```

   Ou en utilisant la CLI .NET :
   ```
   dotnet add package Aspose.Slides
   ```

2. Ajouter des références :
   Ajoutez l’espace de noms suivant à votre fichier C# :
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

3. Configuration de la licence (facultatif) :
   Si vous avez une licence, appliquez-la en utilisant :
   ```csharp
   License license = new License();
   license.SetLicense("Aspose.Slides.lic");
   ```

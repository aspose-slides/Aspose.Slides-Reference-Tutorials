# Ajouter des images aux diapositives PowerPoint avec Aspose.Slides

Aspose.Slides pour .NET offre de puissantes fonctionnalités pour créer et modifier des présentations PowerPoint par programmation. Cet exemple montre comment charger une image externe et l'ajouter à une diapositive de présentation sous forme de cadre, une exigence courante lors de la génération de rapports, de la création de modèles ou de la création dynamique de présentations à partir de sources de données.

Le code ci-dessous gère tout, de la création d'une nouvelle présentation à l'ajout de l'image et à l'enregistrement du fichier PPTX final, le tout avec seulement quelques lignes de code C#.

## Instructions d'installation

1. Installez Aspose.Slides pour .NET en utilisant l’une de ces méthodes :
   - Gestionnaire de paquets NuGet : `Install-Package Aspose.Slides.NET`
   - .NET CLI : `dotnet add package Aspose.Slides.NET`
   - Ou téléchargez directement depuis [Site Web d'Aspose](https://releases.aspose.com/slides/net/)

2. Ajoutez l’espace de noms à votre fichier C# :
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

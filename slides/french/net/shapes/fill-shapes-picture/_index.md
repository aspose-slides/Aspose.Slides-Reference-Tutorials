# Créer des formes avec des images en mosaïque dans PowerPoint à l'aide d'Aspose.Slides

Aspose.Slides pour .NET offre de puissantes fonctionnalités pour créer et manipuler des présentations PowerPoint par programmation. Cet exemple montre comment créer un rectangle et le remplir avec un motif d'image en mosaïque. Ce motif est particulièrement utile pour créer des arrière-plans texturés, des filigranes ou des éléments décoratifs dans vos présentations.

## Instructions d'installation

1. Installer via le gestionnaire de packages NuGet :
   ```
   Install-Package Aspose.Slides
   ```

2. Installation via .NET CLI :
   ```
   dotnet add package Aspose.Slides
   ```

3. Installation manuelle :
   - Téléchargez la dernière version depuis [Aspose.Slides pour .NET](https://releases.aspose.com/slides/net/)
   - Ajoutez une référence à Aspose.Slides.dll dans votre projet

# Créer des images d'arrière-plan en mosaïque dans PowerPoint avec Aspose.Slides

Aspose.Slides pour .NET offre de puissantes fonctionnalités pour créer et manipuler des présentations PowerPoint par programmation. Cet exemple montre comment créer une présentation avec un rectangle rempli d'un motif d'image en mosaïque. Le code montre comment charger une image, l'ajouter à une diapositive comme arrière-plan d'un rectangle et appliquer diverses propriétés de mosaïque telles que le décalage, l'échelle, l'alignement et les options de retournement. Cette technique est utile pour créer des arrière-plans, des filigranes ou des éléments décoratifs personnalisés dans vos présentations, avec un contrôle précis du comportement de mosaïque.

## Instructions d'installation

1. Installez Aspose.Slides pour .NET à l'aide du gestionnaire de packages NuGet :
   ```
   Install-Package Aspose.Slides.NET
   ```

2. Ou installez via .NET CLI :
   ```
   dotnet add package Aspose.Slides.NET
   ```

3. Ajoutez les directives using suivantes à votre fichier C# :
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.IO;
   ```

# Ajout de segments à GeometryShape dans les présentations PowerPoint

Cet exemple montre comment accéder au chemin géométrique des formes dans les présentations PowerPoint et le modifier avec Aspose.Slides pour .NET. En ajoutant des segments de ligne personnalisés à une forme rectangulaire, vous pouvez créer des figures géométriques plus complexes, inaccessibles aux formes standard. Cette technique est utile pour créer des diagrammes, des organigrammes et d'autres éléments visuels personnalisés dans vos présentations.

## Instructions d'installation

1. Installez Aspose.Slides pour .NET en utilisant l’une des méthodes suivantes :

   - Gestionnaire de paquets NuGet :
     ```
     Install-Package Aspose.Slides.NET
     ```

   - Console du gestionnaire de paquets :
     ```
     PM> Install-Package Aspose.Slides.NET
     ```

   - .NET CLI :
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. Ajoutez l’espace de noms à votre fichier C# :
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

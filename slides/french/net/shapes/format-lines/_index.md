# Création d'une forme rectangulaire avec un formatage de ligne personnalisé dans PowerPoint

Aspose.Slides pour .NET offre de puissantes fonctionnalités pour la création et la modification programmatiques de présentations PowerPoint. Cet exemple montre comment créer un rectangle avec une mise en forme de ligne personnalisée, notamment en définissant le style, la largeur, le motif de tirets et la couleur des lignes. L'exemple de code crée une nouvelle présentation, ajoute un rectangle à la première diapositive et applique une mise en forme spécifique au remplissage et au contour de la forme, ce qui crée un élément visuellement attrayant avec un remplissage blanc et un contour en pointillés bleus.

## Instructions d'installation

1. Installez Aspose.Slides pour .NET en utilisant l’une des méthodes suivantes :

   - Gestionnaire de paquets NuGet :
     ```
     Install-Package Aspose.Slides
     ```

   - Console du gestionnaire de paquets :
     ```
     PM> Install-Package Aspose.Slides
     ```

   - .NET CLI :
     ```
     dotnet add package Aspose.Slides
     ```

2. Ajoutez l’espace de noms nécessaire à votre fichier C# :
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

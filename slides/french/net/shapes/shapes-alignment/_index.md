# Alignement et regroupement de formes dans les présentations PowerPoint

Aspose.Slides pour .NET offre de puissantes fonctionnalités pour créer et manipuler des présentations PowerPoint par programmation. Cet exemple montre comment aligner des formes dans des diapositives et utiliser des formes groupées dans des présentations PowerPoint. Vous apprendrez à positionner des rectangles à des coordonnées spécifiques, à les aligner par rapport à la diapositive ou entre eux, et à aligner sélectivement des formes au sein de groupes.

## Instructions d'installation

1. Installez Aspose.Slides pour .NET en utilisant l’une de ces méthodes :
   
   - Gestionnaire de paquets NuGet :
     ```
     Install-Package Aspose.Slides.NET
     ```
   
   - Console du gestionnaire de paquets :
     ```
     PM> Install-Package Aspose.Slides.NET
     ```
   
   - .NET CLI :
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. Ajoutez des références à votre projet :
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.IO;
   ```

3. Téléchargez l'exemple de code ci-dessous et intégrez-le dans votre projet.

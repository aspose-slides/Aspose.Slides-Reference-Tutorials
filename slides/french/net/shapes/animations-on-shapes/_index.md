# Créer des effets de texte animés dans PowerPoint avec Aspose.Slides

Aspose.Slides pour .NET offre de puissantes fonctionnalités pour créer des effets de texte animés dans les présentations PowerPoint par programmation. Cet exemple montre comment créer une zone de texte avec des effets d'animation, notamment une trajectoire de mouvement personnalisée déclenchée par un clic sur un bouton. Le code crée une présentation avec un rectangle contenant du texte, applique une animation PathFootball et ajoute une trajectoire de mouvement personnalisée, définie par l'utilisateur et déclenchable de manière interactive.

## Instructions d'installation

1. Installez Aspose.Slides pour .NET en utilisant l’une des méthodes suivantes :

   Gestionnaire de paquets NuGet :
   ```csharp
   Install-Package Aspose.Slides
   ```

   Console du gestionnaire de paquets :
   ```csharp
   PM> Install-Package Aspose.Slides
   ```

   .NET CLI :
   ```csharp
   dotnet add package Aspose.Slides
   ```

2. Ajoutez les espaces de noms requis à votre code C# :
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Animation;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

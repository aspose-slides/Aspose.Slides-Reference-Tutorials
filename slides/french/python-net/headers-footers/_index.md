# Tutoriels PowerPoint sur les en-têtes et pieds de page pour Aspose.Slides Python

Ces guides complets expliquent comment ajouter des logos d'entreprise, des numéros de page, des informations de date/heure et des métadonnées personnalisées à vos présentations par programmation.

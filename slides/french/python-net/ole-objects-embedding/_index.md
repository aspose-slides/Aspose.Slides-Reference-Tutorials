# Tutoriels PowerPoint sur les objets OLE et le contenu intégré pour Aspose.Slides Python

Nos tutoriels sur les objets OLE expliquent comment utiliser du contenu incorporé dans des présentations PowerPoint avec Aspose.Slides pour Python. Ces guides complets abordent l'insertion de feuilles de calcul Excel, de documents Word et d'autres types de fichiers sous forme d'objets incorporés, la gestion des fichiers liés et le contrôle programmatique du comportement des objets. Chaque tutoriel inclut des exemples pratiques de code Python pour des scénarios d'incorporation courants, vous aidant à créer des présentations riches intégrant parfaitement du contenu provenant de sources multiples.

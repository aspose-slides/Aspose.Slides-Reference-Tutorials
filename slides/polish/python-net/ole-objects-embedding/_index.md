# Samouczki dotyczące obiektów OLE i osadzonej zawartości programu PowerPoint dla języka Aspose.Slides Python

Te kompleksowe przewodniki obejmują wstawianie arkuszy kalkulacyjnych Excela, dokumentów Worda i innych typów plików jako osadzonych obiektów, zarządzanie połączonymi plikami i programowe sterowanie zachowaniem obiektów. Każdy samouczek zawiera praktyczne przykłady kodu Pythona dla typowych scenariuszy osadzania, pomagając Ci tworzyć bogate prezentacje, które płynnie integrują zawartość z wielu źródeł.

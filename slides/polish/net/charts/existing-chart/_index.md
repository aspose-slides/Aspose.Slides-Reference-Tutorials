# Modyfikowanie wykresów w programie PowerPoint za pomocą Aspose.Slides

Aspose.Slides for .NET udostępnia potężne API do tworzenia i modyfikowania prezentacji PowerPoint programowo. Ten przykład pokazuje, jak pracować z wykresami w istniejących prezentacjach, aktualizując dane wykresu, dodając nowe serie i zmieniając typy wykresów — wszystko to bez konieczności instalowania programu Microsoft PowerPoint.

## Instrukcje instalacji

1. Instalacja za pomocą Menedżera pakietów NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Lub używając .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Dodaj następujące dyrektywy using:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Charts;
   ```

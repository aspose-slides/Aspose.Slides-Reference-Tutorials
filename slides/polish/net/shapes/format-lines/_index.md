# Tworzenie kształtu prostokąta z niestandardowym formatowaniem linii w programie PowerPoint

Aspose.Slides for .NET zapewnia potężne możliwości programowego tworzenia i modyfikowania prezentacji PowerPoint. Ten przykład pokazuje, jak utworzyć kształt prostokąta z niestandardowym formatowaniem linii, w tym ustawieniem stylu linii, szerokości, wzoru kreski i koloru. Przykładowy kod tworzy nową prezentację, dodaje prostokąt do pierwszego slajdu i stosuje określone formatowanie zarówno do wypełnienia, jak i konturu kształtu, co skutkuje wizualnie atrakcyjnym elementem z białym wypełnieniem i niebieskim przerywanym konturem.

## Instrukcje instalacji

1. Zainstaluj Aspose.Slides dla platformy .NET, korzystając z jednej z następujących metod:

   - Menedżer pakietów NuGet:
     ```
     Install-Package Aspose.Slides
     ```

   - Konsola Menedżera Pakietów:
     ```
     PM> Install-Package Aspose.Slides
     ```

   - Interfejs wiersza poleceń .NET:
     ```
     dotnet add package Aspose.Slides
     ```

2. Dodaj potrzebną przestrzeń nazw do pliku C#:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

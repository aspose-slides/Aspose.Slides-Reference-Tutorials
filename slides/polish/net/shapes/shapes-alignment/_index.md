# Wyrównywanie i grupowanie kształtów w prezentacjach programu PowerPoint

Aspose.Slides for .NET zapewnia potężne możliwości tworzenia i manipulowania prezentacjami PowerPoint programowo. Ten przykład pokazuje, jak wyrównywać kształty w slajdach i pracować z pogrupowanymi kształtami w prezentacjach PowerPoint. Dowiesz się, jak pozycjonować prostokąty w określonych współrzędnych, wyrównywać je względem slajdu lub względem siebie oraz selektywnie wyrównywać kształty w grupach.

## Instrukcje instalacji

1. Zainstaluj Aspose.Slides dla .NET, korzystając z jednej z poniższych metod:
   
   - Menedżer pakietów NuGet:
     ```
     Install-Package Aspose.Slides.NET
     ```
   
   - Konsola Menedżera Pakietów:
     ```
     PM> Install-Package Aspose.Slides.NET
     ```
   
   - Interfejs wiersza poleceń .NET:
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. Dodaj odniesienia do swojego projektu:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.IO;
   ```

3. Pobierz przykładowy kod poniżej i zintegruj go ze swoim projektem.

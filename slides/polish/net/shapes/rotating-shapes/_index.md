# Tworzenie i obracanie kształtów prostokątnych w programie PowerPoint

Aspose.Slides for .NET to potężne API, które pozwala deweloperom tworzyć, modyfikować i konwertować prezentacje PowerPoint bez konieczności instalowania programu Microsoft PowerPoint. Ten przykład pokazuje, jak utworzyć prezentację z obróconym prostokątnym kształtem, prezentując możliwości manipulacji kształtem biblioteki.

Dzięki Aspose.Slides możesz programowo dodawać różne kształty do slajdów i stosować transformacje, takie jak obrót. Ta funkcjonalność jest szczególnie przydatna do tworzenia dynamicznych prezentacji z precyzyjną kontrolą nad elementami wizualnymi.

## Instrukcje instalacji

1. Instalacja za pomocą Menedżera pakietów NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Instalacja za pomocą .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Dodaj dyrektywę using w swoim kodzie:
   ```csharp
   using Aspose.Slides;
   ```

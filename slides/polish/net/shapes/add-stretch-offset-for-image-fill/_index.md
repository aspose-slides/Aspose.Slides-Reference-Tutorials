# Dodawanie obrazów do slajdów programu PowerPoint za pomocą Aspose.Slides

Aspose.Slides for .NET zapewnia potężne możliwości programowego tworzenia i modyfikowania prezentacji PowerPoint. Ten przykład pokazuje, jak załadować zewnętrzny obraz i dodać go do slajdu prezentacji jako ramkę obrazu — jest to powszechny wymóg podczas generowania raportów, tworzenia szablonów lub dynamicznego budowania prezentacji ze źródeł danych.

Poniższy kod obsługuje wszystko, od utworzenia nowej prezentacji po dodanie obrazu i zapisanie końcowego pliku PPTX. Wszystko to przy użyciu zaledwie kilku linijek kodu C#.

## Instrukcje instalacji

1. Zainstaluj Aspose.Slides dla .NET, korzystając z jednej z poniższych metod:
   - Menedżer pakietów NuGet: `Install-Package Aspose.Slides.NET`
   - Interfejs wiersza poleceń .NET: `dotnet add package Aspose.Slides.NET`
   - Lub pobierz bezpośrednio z [Strona internetowa Aspose](https://releases.aspose.com/slides/net/)

2. Dodaj przestrzeń nazw do pliku C#:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

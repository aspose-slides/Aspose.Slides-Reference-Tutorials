# Dodawanie segmentów do GeometryShape w prezentacjach PowerPoint

Ten przykład pokazuje, jak uzyskać dostęp i modyfikować ścieżkę geometrii kształtów w prezentacjach PowerPoint przy użyciu Aspose.Slides dla .NET. Dodając niestandardowe segmenty linii do kształtu prostokąta, możesz tworzyć bardziej złożone figury geometryczne, które nie są dostępne za pośrednictwem standardowych typów kształtów. Ta technika jest przydatna do tworzenia niestandardowych diagramów, schematów blokowych i innych elementów wizualnych w prezentacjach.

## Instrukcje instalacji

1. Zainstaluj Aspose.Slides dla platformy .NET, korzystając z jednej z następujących metod:

   - Menedżer pakietów NuGet:
     ```
     Install-Package Aspose.Slides.NET
     ```

   - Konsola Menedżera Pakietów:
     ```
     PM> Install-Package Aspose.Slides.NET
     ```

   - Interfejs wiersza poleceń .NET:
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. Dodaj przestrzeń nazw do pliku C#:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

# Tworzenie kształtów grupowych w programie PowerPoint za pomocą Aspose.Slides

Aspose.Slides for .NET udostępnia kompleksowe interfejsy API do tworzenia i manipulowania prezentacjami PowerPoint programowo. Ten przykład pokazuje, jak utworzyć kształt grupy, dodać wiele kształtów prostokątnych w jej obrębie, prawidłowo ustawić kształty, dodać ramkę i zapisać prezentację. Kształty grup są niezbędne do organizowania powiązanych kształtów, które muszą być manipulowane jako pojedyncza jednostka, zapewniając większą strukturę i kontrolę nad elementami prezentacji.

## Instrukcje instalacji

1. Zainstaluj Aspose.Slides dla .NET, korzystając z jednej z poniższych metod:

   - Menedżer pakietów NuGet:
     ```
     Install-Package Aspose.Slides.NET
     ```

   - Konsola Menedżera Pakietów:
     ```
     PM> Install-Package Aspose.Slides.NET
     ```

   - Interfejs wiersza poleceń .NET:
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. Dodaj odniesienia do swojego projektu:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

3. Skonfiguruj licencję (opcjonalne do użytku produkcyjnego):
   ```csharp
   // Ustaw licencję, aby usunąć ograniczenia dotyczące oceny
   License license = new License();
   license.SetLicense("Aspose.Slides.lic");
   ```

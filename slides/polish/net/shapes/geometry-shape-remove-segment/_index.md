# Modyfikuj ścieżki geometryczne w kształtach programu PowerPoint za pomocą Aspose.Slides

Aspose.Slides for .NET zapewnia potężne możliwości programowego modyfikowania prezentacji PowerPoint, w tym zaawansowaną manipulację kształtami. Ten przykład pokazuje, jak tworzyć niestandardowe kształty, modyfikując ścieżkę geometrii istniejącego kształtu — konkretnie poprzez usunięcie segmentu z kształtu serca, aby utworzyć unikalną odmianę.

Ścieżka geometrii definiuje zarys kształtu poprzez serię punktów i segmentów. Uzyskując dostęp do tych ścieżek i je modyfikując, programiści mogą tworzyć wysoce dostosowane kształty, które nie są dostępne w standardowej bibliotece kształtów programu PowerPoint.

## Instrukcje instalacji

1. Instalacja za pomocą Menedżera pakietów NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Instalacja za pomocą .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Dodaj dyrektywę using w swoim kodzie:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

4. Dodaj inicjalizację licencji podczas uruchamiania aplikacji:
   ```csharp
   // Jeśli masz licencję
   License license = new License();
   license.SetLicense("path/to/Aspose.Slides.lic");
   ```

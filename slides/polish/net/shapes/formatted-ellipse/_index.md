# Tworzenie kształtów eliptycznych w programie PowerPoint za pomocą Aspose.Slides

Aspose.Slides for .NET to potężne API, które umożliwia programistom tworzenie, czytanie, pisanie, modyfikowanie i konwertowanie prezentacji PowerPoint bez konieczności korzystania z programu Microsoft PowerPoint. Ten przykład pokazuje, jak utworzyć kształt eliptyczny w prezentacji, zastosować formatowanie, w tym kolor wypełnienia i właściwości linii, i zapisać wynik jako plik PPTX. Kod pokazuje zdolność Aspose.Slides do manipulowania kształtami z precyzyjną kontrolą nad ich wyglądem i pozycjonowaniem.

## Instrukcje instalacji

1. Instalacja za pomocą Menedżera pakietów NuGet:
   ```
   Install-Package Aspose.Slides
   ```

   Lub używając .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

2. Dodaj odniesienia:
   Dodaj następującą przestrzeń nazw do pliku C#:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

3. Konfiguracja licencji (opcjonalnie):
   Jeżeli posiadasz licencję, zastosuj ją za pomocą:
   ```csharp
   License license = new License();
   license.SetLicense("Aspose.Slides.lic");
   ```

# Tworzenie kształtów eliptycznych wypełnionych gradientem w programie PowerPoint za pomocą Aspose.Slides

Ten przykład pokazuje, jak programowo utworzyć prezentację PowerPoint zawierającą kształt elipsy z niestandardowym efektem wypełnienia gradientowego. Kod tworzy nową prezentację, dodaje elipsę do pierwszego slajdu i stosuje liniowe wypełnienie gradientowe z czerwonymi i fioletowymi punktami koloru. Aspose.Slides dla .NET zapewnia kompleksowe API do tworzenia i manipulowania prezentacjami PowerPoint z precyzyjną kontrolą nad kształtami, formatowaniem i efektami.

## Instrukcje instalacji

1. Zainstaluj Aspose.Slides dla .NET, korzystając z jednej z poniższych metod:
   - Menedżer pakietów NuGet: `Install-Package Aspose.Slides.NET`
   - Interfejs wiersza poleceń .NET: `dotnet add package Aspose.Slides.NET`
   - Konsola Menedżera Pakietów: `Install-Package Aspose.Slides.NET`

2. Dodaj następującą instrukcję using do pliku C#:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

# Tworzenie kształtów z wypełnieniem obrazem kafelkowym w programie PowerPoint przy użyciu Aspose.Slides

Aspose.Slides for .NET zapewnia potężne możliwości tworzenia i manipulowania prezentacjami PowerPoint programowo. Ten przykład pokazuje, jak utworzyć kształt prostokąta i wypełnić go kafelkowym wzorem obrazu. Kafelkowe wypełnienie obrazu jest szczególnie przydatne do tworzenia teksturowanych teł, znaków wodnych lub elementów dekoracyjnych w prezentacjach.

## Instrukcje instalacji

1. Instalacja za pomocą Menedżera pakietów NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Instalacja za pomocą .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Instalacja ręczna:
   - Pobierz najnowszą wersję z [Aspose.Slides dla .NET](https://releases.aspose.com/slides/net/)
   - Dodaj odwołanie do Aspose.Slides.dll w swoim projekcie

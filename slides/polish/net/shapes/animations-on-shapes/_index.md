# Tworzenie animowanych efektów tekstowych w programie PowerPoint za pomocą Aspose.Slides

Aspose.Slides for .NET zapewnia potężne możliwości tworzenia animowanych efektów tekstowych w prezentacjach PowerPoint programowo. Ten przykład pokazuje, jak utworzyć pole tekstowe z efektami animacji, w tym niestandardową ścieżkę ruchu wyzwalaną kliknięciem przycisku. Kod tworzy prezentację z prostokątnym kształtem zawierającym tekst, stosuje animację PathFootball i dodaje niestandardową ścieżkę ruchu zdefiniowaną przez użytkownika, którą można wyzwalać interaktywnie.

## Instrukcje instalacji

1. Zainstaluj Aspose.Slides dla platformy .NET, korzystając z jednej z następujących metod:

   Menedżer pakietów NuGet:
   ```csharp
   Install-Package Aspose.Slides
   ```

   Konsola Menedżera Pakietów:
   ```csharp
   PM> Install-Package Aspose.Slides
   ```

   Interfejs wiersza poleceń .NET:
   ```csharp
   dotnet add package Aspose.Slides
   ```

2. Dodaj wymagane przestrzenie nazw do swojego kodu C#:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Animation;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

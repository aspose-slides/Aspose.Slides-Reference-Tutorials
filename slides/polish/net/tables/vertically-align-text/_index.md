# Tworzenie tabel z tekstem pionowym w prezentacjach PowerPoint

Aspose.Slides for .NET zapewnia potężne możliwości programowego tworzenia i manipulowania prezentacjami PowerPoint. Ten przykład pokazuje, jak utworzyć tabelę z niestandardowym formatowaniem, w tym pionowym wyrównaniem tekstu i obrotem — powszechnym wymogiem specjalistycznych prezentacji biznesowych i wizualizacji danych.

## Instrukcje instalacji

1. Zainstaluj Aspose.Slides dla .NET za pomocą Menedżera pakietów NuGet:
   ```
   Install-Package Aspose.Slides
   ```
   Lub poprzez .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

2. Dodaj odniesienia do swojego projektu:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

3. Pobierz zasoby dokumentacji:
   - [Odniesienie do API](https://reference.aspose.com/slides/net/)
   - [Podręcznik dla programistów](https://docs.aspose.com/slides/net/)

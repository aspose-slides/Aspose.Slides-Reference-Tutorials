# Tworzenie standardowych tabel w programie PowerPoint za pomocą Aspose.Slides

Aspose.Slides for .NET to potężne API, które pozwala programistom programowo tworzyć, czytać, pisać, modyfikować i konwertować prezentacje PowerPoint bez konieczności korzystania z programu Microsoft PowerPoint. Ten przykład pokazuje, jak utworzyć standardową tabelę z obramowaniami o niestandardowym formacie w prezentacji PowerPoint.

## Instrukcje instalacji

1. Instalacja za pomocą Menedżera pakietów NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Lub poprzez .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Dodaj następującą przestrzeń nazw do pliku C#:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Tables;
   using System.Drawing;
   ```

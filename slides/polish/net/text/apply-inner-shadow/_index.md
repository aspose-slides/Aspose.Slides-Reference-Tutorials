# Tworzenie prezentacji PowerPoint z TextBox w Aspose.Slides dla .NET

Aspose.Slides dla .NET to potężne API, które pozwala deweloperom tworzyć, modyfikować i konwertować prezentacje PowerPoint bez konieczności korzystania z programu Microsoft PowerPoint. Ten przykład pokazuje, jak utworzyć prezentację z niestandardowym polem tekstowym, zastosować formatowanie i zapisać ją jako plik PPTX.

## Instrukcje instalacji

1. Instalacja za pomocą Menedżera pakietów NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Instalacja za pomocą .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Zainstaluj ręcznie:
   - Pobierz najnowszą wersję z [Strona wydań Aspose.Slides dla .NET](https://releases.aspose.com/slides/net/)
   - Dodaj odwołanie do biblioteki DLL do swojego projektu

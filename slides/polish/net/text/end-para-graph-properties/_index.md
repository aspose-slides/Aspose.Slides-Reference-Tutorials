# Tworzenie prezentacji PowerPoint z sformatowanym tekstem

Aspose.Slides dla .NET to potężne API, które pozwala deweloperom tworzyć, manipulować i konwertować prezentacje PowerPoint bez konieczności korzystania z Microsoft PowerPoint. Ten przykład pokazuje, jak utworzyć prostą prezentację zawierającą prostokątny kształt z sformatowanymi akapitami tekstowymi.

Kod tworzy nową prezentację, dodaje prostokątny kształt do pierwszego slajdu, a następnie dodaje wiele akapitów z różnymi opcjami formatowania tekstu. Pokazuje, jak ustawić właściwości czcionki, takie jak rozmiar i rodzina dla określonych części tekstu.

## Instrukcje instalacji

1. Instalacja za pomocą Menedżera pakietów NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Instalacja za pomocą .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Instalacja za pomocą konsoli Menedżera pakietów:
   ```
   PM> Install-Package Aspose.Slides
   ```

4. Dodaj odniesienie do swojego projektu:
   Po instalacji dodaj odniesienie do swojego projektu:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

# Obracanie tytułów i etykiet wykresów w programie PowerPoint za pomocą Aspose.Slides

Aspose.Slides for .NET zapewnia potężne możliwości tworzenia i manipulowania prezentacjami PowerPoint programowo. Ten przykład pokazuje, jak utworzyć prezentację z wykresem kolumnowym klastrowanym, który zawiera obrócone elementy tekstowe - konkretnie, obracając zarówno tytuł wykresu, jak i etykiety danych, aby poprawić czytelność i atrakcyjność wizualną.

## Instrukcje instalacji

1. Instalacja za pomocą Menedżera pakietów NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Lub zainstaluj poprzez .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Dodaj przestrzeń nazw do pliku C#:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Charts;
   ```

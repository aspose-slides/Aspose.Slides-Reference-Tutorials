# Tworzenie wielopoziomowych punktów wypunktowania w programie PowerPoint za pomocą Aspose.Slides

Aspose.Slides for .NET udostępnia potężne API do tworzenia i manipulowania prezentacjami PowerPoint programowo. Ten przykład pokazuje, jak utworzyć prezentację z wielopoziomowymi punktami wypunktowania, pokazując, jak dodać automatyczny kształt z ramką tekstową i sformatować akapity z różnymi stylami punktowania i poziomami wcięcia. Możesz łatwo tworzyć hierarchiczną zawartość z dostosowanymi symbolami punktowania dla każdego poziomu wcięcia, co jest idealne do tworzenia uporządkowanych i profesjonalnych prezentacji bez konieczności instalowania programu Microsoft PowerPoint.

## Instrukcje instalacji

1. Instalacja za pomocą Menedżera pakietów NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Instalacja za pomocą .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Instalacja za pomocą konsoli Menedżera pakietów:
   ```
   PM> Install-Package Aspose.Slides
   ```

4. Instalacja ręczna:
   - Pobierz najnowszą wersję z [Aspose.Slides dla .NET](https://releases.aspose.com/slides/net/)
   - Dodaj odwołanie do Aspose.Slides.dll w swoim projekcie

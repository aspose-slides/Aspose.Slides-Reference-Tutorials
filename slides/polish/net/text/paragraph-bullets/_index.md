# Tworzenie niestandardowych punktów wypunktowanych w prezentacjach programu PowerPoint

Aspose.Slides for .NET zapewnia solidne możliwości tworzenia i manipulowania prezentacjami PowerPoint programowo. Ten przykład koncentruje się konkretnie na implementacji niestandardowych punktów wypunktowania w ramkach tekstowych, pokazując, jak kontrolować typ wypunktowania, kolor, wcięcie i rozmiar. Kod tworzy prezentację o kształcie prostokąta zawierającą dwa akapity — jeden z symbolem wypunktowania i drugi ze stylem wypunktowania numerowanego.

## Instrukcje instalacji

1. Zainstaluj Aspose.Slides dla .NET przy użyciu Menedżera pakietów NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Można również pobrać pakiet bezpośrednio:
   - Odwiedź [Strona pobierania Aspose.Slides](https://downloads.aspose.com/slides/net)
   - Pobierz najnowszą wersję
   - Wyodrębnij i odwołaj się do pliku Aspose.Slides.dll w swoim projekcie

3. Dodaj niezbędne polecenie using:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

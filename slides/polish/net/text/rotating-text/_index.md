# Tworzenie pionowego tekstu w programie PowerPoint za pomocą Aspose.Slides

Aspose.Slides for .NET zapewnia potężne możliwości tworzenia i manipulowania prezentacjami PowerPoint programowo. Ten przykład pokazuje, jak utworzyć prezentację zawierającą kształt prostokąta z tekstem obróconym pionowo. Tekst jest zorientowany pod kątem 270 stopni, dzięki czemu można go czytać od dołu do góry. Ta technika jest przydatna do tworzenia etykiet bocznych, unikalnych układów tekstu lub specjalistycznych elementów projektowych w prezentacjach.

## Instrukcje instalacji

1. Zainstaluj Aspose.Slides dla .NET
   
   Zainstaluj pakiet Aspose.Slides przy użyciu Menedżera pakietów NuGet:
   
   ```csharp
   Install-Package Aspose.Slides
   ```
   
   Lub używając .NET CLI:
   
   ```csharp
   dotnet add package Aspose.Slides
   ```

2. Dodaj odniesienia
   
   Dodaj następującą instrukcję using do pliku C#:
   
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

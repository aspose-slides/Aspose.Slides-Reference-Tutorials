# Ändra diagram i PowerPoint med Aspose.Slides

Aspose.Slides för .NET tillhandahåller kraftfulla API:er för att skapa och modifiera PowerPoint-presentationer programmatiskt. Det här exemplet visar hur man arbetar med diagram i befintliga presentationer genom att uppdatera diagramdata, lägga till nya serier och ändra diagramtyper – allt utan att Microsoft PowerPoint behöver installeras.

## Installationsanvisningar

1. Installera via NuGet-pakethanteraren:
   ```
   Install-Package Aspose.Slides
   ```

2. Eller med hjälp av .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Lägg till följande med hjälp av direktiv:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Charts;
   ```

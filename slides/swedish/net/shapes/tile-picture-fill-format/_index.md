# Skapa kaklade bildbakgrunder i PowerPoint med Aspose.Slides

Aspose.Slides för .NET erbjuder kraftfulla funktioner för att skapa och manipulera PowerPoint-presentationer programmatiskt. Det här exemplet visar hur man skapar en presentation med en rektangelform fylld med ett sida vid sida-bildmönster. Koden visar hur man laddar en bild, lägger till den på en bild som en rektangels bakgrund och tillämpar olika sida-vid-sida-egenskaper som offset, skala, justering och vändningsalternativ. Den här tekniken är användbar för att skapa anpassade bakgrunder, vattenstämplar eller dekorativa element i dina presentationer med exakt kontroll över sida-vid-sida-beteendet.

## Installationsanvisningar

1. Installera Aspose.Slides för .NET med hjälp av NuGet Package Manager:
   ```
   Install-Package Aspose.Slides.NET
   ```

2. Eller installera via .NET CLI:
   ```
   dotnet add package Aspose.Slides.NET
   ```

3. Lägg till följande using-direktiv i din C#-fil:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.IO;
   ```

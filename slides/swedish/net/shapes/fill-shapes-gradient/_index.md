# Skapa gradientfyllda ellipsformer i PowerPoint med Aspose.Slides

Det här exemplet visar hur man programmatiskt skapar en PowerPoint-presentation som innehåller en ellipsform med en anpassad gradientfyllningseffekt. Koden skapar en ny presentation, lägger till en ellips på den första bilden och tillämpar en linjär gradientfyllning med röda och lila färgstopp. Aspose.Slides för .NET tillhandahåller ett omfattande API för att skapa och manipulera PowerPoint-presentationer med exakt kontroll över former, formatering och effekter.

## Installationsanvisningar

1. Installera Aspose.Slides för .NET med någon av dessa metoder:
   - NuGet-pakethanterare: `Install-Package Aspose.Slides.NET`
   - .NET CLI: `dotnet add package Aspose.Slides.NET`
   - Pakethanterarkonsol: `Install-Package Aspose.Slides.NET`

2. Lägg till följande using-sats i din C#-fil:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

# Skapa en rektangelform med anpassad linjeformatering i PowerPoint

Aspose.Slides för .NET erbjuder kraftfulla funktioner för att programmatiskt skapa och modifiera PowerPoint-presentationer. Det här exemplet visar hur man skapar en rektangelform med anpassad linjeformatering, inklusive att ställa in linjestil, bredd, streckmönster och färg. Exempelkoden skapar en ny presentation, lägger till en rektangel på den första bilden och tillämpar specifik formatering på både fyllningen och konturen av formen, vilket resulterar i ett visuellt tilltalande element med en vit fyllning och en blå streckad kontur.

## Installationsanvisningar

1. Installera Aspose.Slides för .NET med någon av följande metoder:

   - NuGet-pakethanterare:
     ```
     Install-Package Aspose.Slides
     ```

   - Pakethanterarkonsol:
     ```
     PM> Install-Package Aspose.Slides
     ```

   - .NET CLI:
     ```
     dotnet add package Aspose.Slides
     ```

2. Lägg till nödvändigt namnutrymme i din C#-fil:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

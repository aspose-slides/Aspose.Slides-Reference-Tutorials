# Ändra geometriska banor i PowerPoint-former med Aspose.Slides

Aspose.Slides för .NET erbjuder kraftfulla funktioner för att programmatiskt modifiera PowerPoint-presentationer, inklusive avancerad formmanipulation. Det här exemplet visar hur man skapar anpassade former genom att modifiera geometrin för en befintlig form – specifikt genom att ta bort ett segment från en hjärtform för att skapa en unik variant.

Geometribanan definierar konturen av en form genom en serie punkter och segment. Genom att komma åt och modifiera dessa banor kan utvecklare skapa mycket anpassade former som inte är tillgängliga i PowerPoints standardformbibliotek.

## Installationsanvisningar

1. Installera via NuGet-pakethanteraren:
   ```
   Install-Package Aspose.Slides
   ```

2. Installera via .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Lägg till using-direktivet i din kod:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

4. Lägg till licensinitiering i din programstart:
   ```csharp
   // Om du har en licens
   License license = new License();
   license.SetLicense("path/to/Aspose.Slides.lic");
   ```

# Justera och gruppera former i PowerPoint-presentationer

Aspose.Slides för .NET erbjuder kraftfulla funktioner för att skapa och manipulera PowerPoint-presentationer programmatiskt. Det här exemplet visar hur man justerar former i bilder och arbetar med grupperade former i PowerPoint-presentationer. Du lär dig hur du placerar rektanglar vid specifika koordinater, justerar dem i förhållande till bilden eller i förhållande till varandra, och selektivt justerar former inom grupper.

## Installationsanvisningar

1. Installera Aspose.Slides för .NET med någon av dessa metoder:
   
   - NuGet-pakethanterare:
     ```
     Install-Package Aspose.Slides.NET
     ```
   
   - Pakethanterarkonsol:
     ```
     PM> Install-Package Aspose.Slides.NET
     ```
   
   - .NET CLI:
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. Lägg till referenser till ditt projekt:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.IO;
   ```

3. Ladda ner exempelkoden nedan och integrera den i ditt projekt.

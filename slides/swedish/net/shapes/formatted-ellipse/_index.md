# Skapa elliptiska former i PowerPoint med Aspose.Slides

Aspose.Slides för .NET är ett kraftfullt API som gör det möjligt för utvecklare att skapa, läsa, skriva, modifiera och konvertera PowerPoint-presentationer utan att behöva Microsoft PowerPoint. Det här exemplet visar hur man skapar en elliptisk form i en presentation, tillämpar formatering inklusive fyllningsfärg och linjeegenskaper, och sparar resultatet som en PPTX-fil. Koden visar Aspose.Slides förmåga att manipulera former med exakt kontroll över deras utseende och positionering.

## Installationsanvisningar

1. Installera via NuGet-pakethanteraren:
   ```
   Install-Package Aspose.Slides
   ```

   Eller med hjälp av .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

2. Lägg till referenser:
   Lägg till följande namnrymd i din C#-fil:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

3. Licensinställningar (valfritt):
   Om du har en licens, ansök om den med hjälp av:
   ```csharp
   License license = new License();
   license.SetLicense("Aspose.Slides.lic");
   ```

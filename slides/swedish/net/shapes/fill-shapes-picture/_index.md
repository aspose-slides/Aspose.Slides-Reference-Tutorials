# Skapa former med kaklad bildfyllning i PowerPoint med Aspose.Slides

Aspose.Slides för .NET erbjuder kraftfulla funktioner för att skapa och manipulera PowerPoint-presentationer programmatiskt. Det här exemplet visar hur man skapar en rektangelform och fyller den med ett kaklat bildmönster. Den kaklade bildfyllningen är särskilt användbar för att skapa texturerade bakgrunder, vattenstämplar eller dekorativa element i dina presentationer.

## Installationsanvisningar

1. Installera via NuGet-pakethanteraren:
   ```
   Install-Package Aspose.Slides
   ```

2. Installera via .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Manuell installation:
   - Ladda ner den senaste versionen från [Aspose.Slides för .NET](https://releases.aspose.com/slides/net/)
   - Lägg till referens till Aspose.Slides.dll i ditt projekt

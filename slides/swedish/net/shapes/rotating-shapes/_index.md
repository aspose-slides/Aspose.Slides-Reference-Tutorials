# Skapa och rotera rektangulära former i PowerPoint

Aspose.Slides för .NET är ett kraftfullt API som låter utvecklare skapa, modifiera och konvertera PowerPoint-presentationer utan att behöva installera Microsoft PowerPoint. Det här exemplet visar hur man skapar en presentation med en roterad rektangelform och visar bibliotekets möjligheter att manipulera former.

Med Aspose.Slides kan du programmatiskt lägga till olika former till dina bilder och tillämpa transformationer som rotation. Denna funktion är särskilt användbar för att skapa dynamiska presentationer med exakt kontroll över visuella element.

## Installationsanvisningar

1. Installera via NuGet-pakethanteraren:
   ```
   Install-Package Aspose.Slides
   ```

2. Installera via .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Lägg till using-direktivet i din kod:
   ```csharp
   using Aspose.Slides;
   ```

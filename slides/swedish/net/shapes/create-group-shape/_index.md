# Skapa gruppformer i PowerPoint med Aspose.Slides

Aspose.Slides för .NET tillhandahåller omfattande API:er för att skapa och manipulera PowerPoint-presentationer programmatiskt. Det här exemplet visar hur man skapar en gruppform, lägger till flera rektangelformer i den, placerar formerna korrekt, lägger till inramning och sparar presentationen. Gruppformer är viktiga för att organisera relaterade former som behöver manipuleras som en enda enhet, vilket ger mer struktur och kontroll över dina presentationselement.

## Installationsanvisningar

1. Installera Aspose.Slides för .NET med någon av dessa metoder:

   - NuGet-pakethanterare:
     ```
     Install-Package Aspose.Slides.NET
     ```

   - Pakethanterarkonsol:
     ```
     PM> Install-Package Aspose.Slides.NET
     ```

   - .NET CLI:
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. Lägg till referenser till ditt projekt:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

3. Konfigurera en licens (valfritt för produktionsanvändning):
   ```csharp
   // Ställ in licens för att ta bort utvärderingsbegränsningar
   License license = new License();
   license.SetLicense("Aspose.Slides.lic");
   ```

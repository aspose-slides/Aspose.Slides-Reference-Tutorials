# Rotera diagramtitlar och etiketter i PowerPoint med Aspose.Slides

Aspose.Slides för .NET erbjuder kraftfulla funktioner för att skapa och manipulera PowerPoint-presentationer programmatiskt. Det här exemplet visar hur man skapar en presentation med ett klustrat kolumndiagram som innehåller roterade textelement – specifikt roterar både diagramrubriken och dataetiketterna för förbättrad läsbarhet och visuellt tilltalande.

## Installationsanvisningar

1. Installera via NuGet-pakethanteraren:
   ```
   Install-Package Aspose.Slides
   ```

2. Eller installera via .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Lägg till namnrymden i din C#-fil:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Charts;
   ```

# Skapa vertikal text i PowerPoint med hjälp av Aspose.Slides

Aspose.Slides för .NET erbjuder kraftfulla funktioner för att skapa och manipulera PowerPoint-presentationer programmatiskt. Det här exemplet visar hur man skapar en presentation som innehåller en rektangelform med vertikalt roterad text. Texten är orienterad i en 270-graders vinkel, vilket gör att den läses nerifrån och upp. Den här tekniken är användbar för att skapa sidoetiketter, unika textlayouter eller specialiserade designelement i dina presentationer.

## Installationsanvisningar

1. Installera Aspose.Slides för .NET
   
   Installera Aspose.Slides-paketet med NuGet Package Manager:
   
   ```csharp
   Install-Package Aspose.Slides
   ```
   
   Eller med hjälp av .NET CLI:
   
   ```csharp
   dotnet add package Aspose.Slides
   ```

2. Lägg till referenser
   
   Lägg till följande using-sats i din C#-fil:
   
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

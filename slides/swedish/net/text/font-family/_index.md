# Skapa formaterade textrutor i PowerPoint med Aspose.Slides

Aspose.Slides för .NET är ett kraftfullt API som låter utvecklare skapa, modifiera och konvertera Microsoft PowerPoint-presentationer utan att kräva installation av Microsoft Office. Det här exemplet visar hur man skapar en presentation med en formaterad textruta, och använder anpassade teckensnitt, storlekar och färger för att förbättra den visuella attraktionskraften.

## Installationsanvisningar

1. Installera via NuGet-pakethanteraren:
   ```
   Install-Package Aspose.Slides
   ```

2. Installera via .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Lägg till referens till ditt projekt manuellt:
   - Ladda ner den senaste versionen från [Aspose.Slides för .NET](https://releases.aspose.com/slides/net/)
   - Lägg till DLL-filen som referens i ditt projekt

4. Initiera en licens:
   ```csharp
   // Initiera en licens
   License license = new License();
   license.SetLicense("Aspose.Slides.lic");
   ```

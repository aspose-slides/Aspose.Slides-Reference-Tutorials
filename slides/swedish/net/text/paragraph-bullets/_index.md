# Skapa anpassade punktlistor i PowerPoint-presentationer

Aspose.Slides för .NET erbjuder robusta funktioner för att skapa och manipulera PowerPoint-presentationer programmatiskt. Det här exemplet fokuserar specifikt på att implementera anpassade punktlistor i textramar och visar hur man styr punkttyp, färg, indrag och storlek. Koden skapar en presentation med en rektangulär form som innehåller två stycken - ett med en symbolpunkt och ett annat med en numrerad punktstil.

## Installationsanvisningar

1. Installera Aspose.Slides för .NET med hjälp av NuGet Package Manager:
   ```
   Install-Package Aspose.Slides
   ```

2. Alternativt kan du ladda ner paketet direkt:
   - Besök [Aspose.Slides nedladdningssida](https://downloads.aspose.com/slides/net)
   - Ladda ner den senaste versionen
   - Extrahera och referera till Aspose.Slides.dll i ditt projekt

3. Lägg till det nödvändiga using-uttrycket:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

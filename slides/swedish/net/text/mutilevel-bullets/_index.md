# Skapa flernivåpunkter i PowerPoint med Aspose.Slides

Aspose.Slides för .NET tillhandahåller ett kraftfullt API för att skapa och manipulera PowerPoint-presentationer programmatiskt. Det här exemplet visar hur man skapar en presentation med punktlistor i flera nivåer, hur man lägger till en automatisk form med en textram och formaterar stycken med olika punktformat och indenteringsnivåer. Du kan enkelt skapa hierarkiskt innehåll med anpassade punktsymboler för varje indenteringsnivå, perfekt för att skapa strukturerade och professionella presentationer utan att Microsoft PowerPoint behöver installeras.

## Installationsanvisningar

1. Installera via NuGet-pakethanteraren:
   ```
   Install-Package Aspose.Slides
   ```

2. Installera via .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Installera via pakethanterarkonsolen:
   ```
   PM> Install-Package Aspose.Slides
   ```

4. Manuell installation:
   - Ladda ner den senaste versionen från [Aspose.Slides för .NET](https://releases.aspose.com/slides/net/)
   - Lägg till referens till Aspose.Slides.dll i ditt projekt

# Skapa PowerPoint-presentationer med formaterad text

Aspose.Slides för .NET är ett kraftfullt API som låter utvecklare skapa, manipulera och konvertera PowerPoint-presentationer utan att behöva Microsoft PowerPoint. Det här exemplet visar hur man skapar en enkel presentation som innehåller en rektangulär form med formaterade textstycken.

Koden skapar en ny presentation, lägger till en rektangulär form på den första bilden och lägger sedan till flera stycken med olika textformateringsalternativ. Den visar hur man ställer in teckensnittsegenskaper som storlek och teckensnittsfamilj för specifika textdelar.

## Installationsanvisningar

1. Installera via NuGet-pakethanteraren:
   ```
   Install-Package Aspose.Slides
   ```

2. Installera via .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Installera via pakethanterarkonsolen:
   ```
   PM> Install-Package Aspose.Slides
   ```

4. Lägg till referens till ditt projekt:
   Efter installationen, lägg till en referens till ditt projekt:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

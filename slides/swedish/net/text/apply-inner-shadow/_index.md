# Skapa PowerPoint-presentationer med TextBox i Aspose.Slides för .NET

Aspose.Slides för .NET är ett kraftfullt API som låter utvecklare skapa, modifiera och konvertera PowerPoint-presentationer utan att behöva Microsoft PowerPoint. Det här exemplet visar hur man skapar en presentation med en anpassad textruta, tillämpar formatering och sparar den som en PPTX-fil.

## Installationsanvisningar

1. Installera via NuGet-pakethanteraren:
   ```
   Install-Package Aspose.Slides
   ```

2. Installera via .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Installera manuellt:
   - Ladda ner den senaste versionen från [Aspose.Slides för .NET-versionssida](https://releases.aspose.com/slides/net/)
   - Lägg till DLL-referensen i ditt projekt

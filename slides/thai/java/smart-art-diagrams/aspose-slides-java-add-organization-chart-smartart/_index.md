## คู่มือการใช้งาน
### การเพิ่ม SmartArt ลงในงานนำเสนอ
**ภาพรวม**
ส่วนนี้จะแสดงวิธีแทรกแผนภูมิองค์กรประเภท SmartArt ลงในสไลด์แรกของการนำเสนอของคุณ

**ขั้นตอนที่ 1: สร้างอินสแตนซ์การนำเสนอใหม่**
```java
Presentation presentation = new Presentation();
```
- **ทำไม:** นี่เป็นการเริ่มต้นวัตถุการนำเสนอใหม่ ซึ่งเราจะปรับเปลี่ยนโดยการเพิ่มรูปร่างและเนื้อหา

**ขั้นตอนที่ 2: เข้าถึงสไลด์แรก**
```java
ISlide slide = presentation.getSlides().get_Item(0);
```
- **ทำไม:** สไลด์แรกมักจะเป็นจุดเริ่มต้นด้วยเนื้อหาหลักของคุณ รวมทั้งกราฟิก SmartArt ด้วย

**ขั้นตอนที่ 3: เพิ่มแผนภูมิองค์กรด้วยกราฟิก SmartArt**
```java
ISmartArt smart = slide.getShapes().addSmartArt(10, 10, 400, 300, SmartArtLayoutType.OrganizationChart);
```
- **ทำไม:** การเรียกใช้เมธอดนี้จะเพิ่มกราฟิก SmartArt ใหม่ลงในสไลด์โดยระบุขนาดและประเภทเค้าโครง พารามิเตอร์ (x, y, ความกว้าง, ความสูง) จะกำหนดตำแหน่งและขนาดของกราฟิก

### การกำหนดรูปแบบผังองค์กร
**ภาพรวม**
ที่นี่คุณจะได้เรียนรู้วิธีการปรับเปลี่ยนเค้าโครงของแผนผังองค์กรที่มีอยู่ในกราฟิก SmartArt ของคุณ

**ขั้นตอนที่ 4: ปรับเปลี่ยนเค้าโครงของโหนดแรก**
```java
smart.getNodes().get_Item(0).setOrganizationChartLayout(OrganizationChartLayoutType.LeftHanging);
```
- **ทำไม:** ขั้นตอนนี้จะปรับแต่งเค้าโครงเพื่อให้การแสดงภาพข้อมูลแบบลำดับชั้นมีความเหมาะสมมากขึ้น 

### การบันทึกการนำเสนอลงในไฟล์
**ภาพรวม**
ในฟีเจอร์สุดท้ายนี้ คุณจะบันทึกการนำเสนอของคุณด้วยกราฟิก SmartArt ที่เพิ่มเข้ามา

**ขั้นตอนที่ 5: บันทึกงานของคุณ**
```java
String outputDir = "YOUR_OUTPUT_DIRECTORY";
presentation.save(outputDir + "OrganizeChartLayoutType_out.pptx", SaveFormat.Pptx);
```
- **ทำไม:** วิธีนี้จะช่วยให้มั่นใจว่าการเปลี่ยนแปลงทั้งหมดจะถูกบันทึกลงในไฟล์ ซึ่งสามารถแชร์หรือแสดงได้

## คู่มือการใช้งาน

ตอนนี้คุณได้ตั้งค่า Aspose.Slides แล้ว มาแบ่งการใช้งานออกเป็นส่วนที่จัดการได้:

### สร้างคุณลักษณะไดเรกทอรี

#### ภาพรวม

ฟีเจอร์นี้จะตรวจสอบว่ามีไดเรกทอรีอยู่ในเส้นทางที่ระบุหรือไม่ หากไม่มี ฟีเจอร์นี้จะสร้างไดเรกทอรีขึ้นมาโดยอัตโนมัติ

#### ขั้นตอนการดำเนินการ

**1. กำหนดเส้นทางไดเร็กทอรี**
```java
import java.io.File;

public class DirectoryCreator {
    public static void main(String[] args) {
        // ระบุไดเรกทอรีเอกสารของคุณที่นี่
        String dataDir = "YOUR_DOCUMENT_DIRECTORY";

        // ตรวจสอบการมีอยู่ของไดเร็กทอรี
        boolean isExists = new File(dataDir).exists();
        
        // สร้างมันขึ้นมาถ้ามันไม่มีอยู่
        if (!isExists) {
            new File(dataDir).mkdirs();
        }
    }
}
```

- **คำอธิบาย**: เดอะ `File` คลาสตรวจสอบและสร้างไดเร็กทอรี ใช้ `exists()` เพื่อตรวจสอบการมีอยู่และ `mkdirs()` เพื่อสร้างโครงสร้างไดเร็กทอรี

**2. เคล็ดลับการแก้ไขปัญหา**
ตรวจสอบให้แน่ใจว่าได้ระบุเส้นทางอย่างถูกต้องและตรวจสอบสิทธิ์การเข้าถึงระบบไฟล์ของแอปพลิเคชันของคุณ

### สร้างตัวอย่างคุณสมบัติการนำเสนอ

#### ภาพรวม

ฟีเจอร์นี้สาธิตวิธีการสร้างอินสแตนซ์การนำเสนอใหม่โดยใช้ Aspose.Slides

#### ขั้นตอนการดำเนินการ
```java
import com.aspose.slides.Presentation;

public class CreatePresentation {
    public static void main(String[] args) {
        // เริ่มต้นวัตถุการนำเสนอ
        Presentation pres = new Presentation();
        
        try {
            // โค้ดเพิ่มเติมสำหรับการทำงานกับการนำเสนออยู่ที่นี่
        } finally {
            if (pres != null) pres.dispose();  // ทำความสะอาดทรัพยากร
        }
    }
}
```

- **คำอธิบาย**: สร้างอินสแตนซ์ `Presentation` ชั้นเรียนเพื่อเริ่มสร้างสไลด์ ควรกำจัดวัตถุทิ้งเสมอเพื่อเพิ่มหน่วยความจำ

### เพิ่มและจัดรูปแบบฟีเจอร์รูปวงรี

#### ภาพรวม

เพิ่มรูปร่างวงรีลงในสไลด์ จัดรูปแบบด้วยสีทึบ และบันทึกการนำเสนอ

#### ขั้นตอนการดำเนินการ
```java
import com.aspose.slides.IAutoShape;
import com.aspose.slides.IShapeCollection;
import com.aspose.slides.Presentation;
import com.aspose.slides.SaveFormat;
import com.aspose.slides.ShapeType;
import java.awt.Color;

public class AddAndFormatEllipse {
    public static void main(String[] args) {
        // สร้างอินสแตนซ์การนำเสนอใหม่
        Presentation pres = new Presentation();
        
        try {
            // เข้าถึงคอลเลกชันรูปร่างของสไลด์แรก
            IShapeCollection shapes = pres.getSlides().get_Item(0).getShapes();

            // เพิ่มวงรีให้กับสไลด์
            IAutoShape shp = (IAutoShape) shapes.addAutoShape(ShapeType.Ellipse, 50, 150, 150, 50);

            // จัดรูปแบบการเติมวงรีด้วยสีทึบ
            shp.getFillFormat().setFillType(com.aspose.slides.FillType.Solid);
            shp.getFillFormat().getSolidFillColor().setColor(new Color(210, 105, 30)); // ช็อคโกแลต

            // ตั้งค่ารูปแบบเส้นให้กับวงรี
            shp.getLineFormat().getFillFormat().setFillType(com.aspose.slides.FillType.Solid);
            shp.getLineFormat().getFillFormat().getSolidFillColor().setColor(Color.BLACK);
            shp.getLineFormat().setWidth(5);

            // บันทึกการนำเสนอของคุณลงในไฟล์
            pres.save("YOUR_OUTPUT_DIRECTORY/EllipseShp2_out.pptx", SaveFormat.Pptx);
        } finally {
            if (pres != null) pres.dispose();  // ให้แน่ใจว่าทรัพยากรได้รับการปลดปล่อย
        }
    }
}
```

- **คำอธิบาย**: เดอะ `addAutoShape` วิธีการเพิ่มวงรีลงในสไลด์ ใช้รูปแบบการเติมและเส้นเพื่อปรับแต่งลักษณะที่ปรากฏ

**เคล็ดลับการแก้ไขปัญหา**
- ตรวจสอบพิกัดรูปร่างและขนาดอีกครั้ง
- ตรวจสอบการเข้าถึงไดเรกทอรีเอาท์พุตสำหรับการบันทึกไฟล์

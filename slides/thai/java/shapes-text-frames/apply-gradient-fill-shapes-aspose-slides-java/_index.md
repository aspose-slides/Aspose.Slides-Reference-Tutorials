## คู่มือการใช้งาน
ตอนนี้เราจะแนะนำขั้นตอนต่างๆ ที่จำเป็นในการใช้การเติมไล่ระดับให้กับรูปร่างวงรีในการนำเสนอ PowerPoint โดยใช้ Aspose.Slides สำหรับ Java

### การเพิ่มรูปร่างวงรีด้วยการเติมแบบไล่เฉดสี
#### ขั้นตอนที่ 1: สร้างและกำหนดค่าการนำเสนอ
ขั้นแรกให้เริ่มต้นใหม่ `Presentation` วัตถุที่แสดงไฟล์ PowerPoint ของคุณ นี่คือที่ที่คุณจะเพิ่มรูปร่างและใช้การจัดรูปแบบ

```java
import com.aspose.slides.*;

public class FillShapesGradient {
    public static void main(String[] args) {
        String dataDir = "YOUR_DOCUMENT_DIRECTORY";
        
        Presentation pres = new Presentation();
        try {
            ISlide sld = pres.getSlides().get_Item(0);
```

#### ขั้นตอนที่ 2: เพิ่มรูปทรงวงรี
เพิ่มรูปวงรีให้กับสไลด์ คุณสามารถระบุขนาดและตำแหน่งบนสไลด์ได้

```java
            // เพิ่มรูปวงรีโดยกำหนดขนาดและตำแหน่ง
            IShape shp = sld.getShapes().addAutoShape(ShapeType.Ellipse, 50, 150, 75, 150);
```

#### ขั้นตอนที่ 3: ใช้การเติมแบบไล่เฉดสี
กำหนดค่าการเติมแบบไล่ระดับสำหรับรูปร่างของคุณ คุณสามารถเลือกระหว่างรูปร่างและทิศทางการไล่ระดับที่แตกต่างกันได้

```java
            // ตั้งค่าประเภทการเติมแบบไล่ระดับสี
            shp.getFillFormat().setFillType(FillType.Gradient);

            // เลือกรูปทรงไล่ระดับสีแบบเส้นตรง
            shp.getFillFormat().getGradientFormat().setGradientShape(GradientShape.Linear);

            // กำหนดทิศทางการไล่ระดับ
            shp.getFillFormat().getGradientFormat().setGradientDirection(GradientDirection.FromCorner2);
```

#### ขั้นตอนที่ 4: ปรับแต่งสีแบบไล่ระดับ
กำหนดสีและตำแหน่งสำหรับจุดหยุดการไล่ระดับสี ซึ่งจะควบคุมการเปลี่ยนผ่านระหว่างสีต่างๆ ของการไล่ระดับสี

```java
            // เพิ่มจุดหยุดสีเพื่อกำหนดการเปลี่ยนผ่านแบบไล่ระดับ
            shp.getFillFormat().getGradientFormat().getGradientStops().add((float) 1.0, new Color(PresetColor.Purple));
            shp.getFillFormat().getGradientFormat().getGradientStops().add((float) 0, Color.RED);
```

#### ขั้นตอนที่ 5: บันทึกการนำเสนอ
สุดท้าย ให้บันทึกการนำเสนอของคุณลงในไฟล์โดยใช้รูปร่างที่มีการไล่ระดับสี

```java
            // บันทึกการนำเสนอด้วยสไลด์ที่อัปเดต
            pres.save(dataDir + "EllipseShpGrad_out.pptx", SaveFormat.Pptx);
        } finally {
            if (pres != null) pres.dispose();
        }
    }
}
```

**เคล็ดลับการแก้ไขปัญหา:**

- ตรวจสอบว่าเวอร์ชันของ JDK ตรงกับตัวจำแนกประเภท Aspose.Slides ที่ใช้ในการอ้างอิง

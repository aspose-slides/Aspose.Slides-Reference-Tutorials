## การตั้งค่า Aspose.Slides สำหรับ Java

รวม Aspose.Slides เข้ากับโปรเจ็กต์ของคุณโดยเพิ่มการอ้างอิงโดยใช้ Maven หรือ Gradle:

**เมเวน:**
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

**เกรเดิ้ล:**
```gradle
implementation group: 'com.aspose', name: 'aspose-slides', version: '25.4', classifier: 'jdk16'
```

## คู่มือการใช้งาน

### ภาพรวมของการดึงข้อมูลพิกัดส่วน

คุณลักษณะนี้ช่วยให้สามารถแยกและจัดการพิกัดส่วนข้อความในรูปร่าง PowerPoint ได้อย่างแม่นยำ

#### ขั้นตอนที่ 1: โหลดงานนำเสนอของคุณ

โหลดไฟล์ PowerPoint ของคุณลงใน Aspose.Slides `Presentation` วัตถุ:
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY" + "/Shapes.pptx";
Presentation presentation = new Presentation(dataDir);
```

#### ขั้นตอนที่ 2: เข้าถึงรูปร่างที่ต้องการ

ดึงรูปร่างที่คุณต้องการแยกพิกัดข้อความออกมา ที่นี่ เราจะเข้าถึงรูปร่างแรกในสไลด์แรก:
```java
IAutoShape shape = (IAutoShape)presentation.getSlides().get_Item(0).getShapes().get_Item(0);
```

#### ขั้นตอนที่ 3: นำทางผ่านส่วนข้อความ

เข้าถึงกรอบข้อความและทำซ้ำผ่านย่อหน้าและส่วนต่างๆ เพื่อดึงพิกัด:
```java
ITextFrame textFrame = shape.getTextFrame();
for (IParagraph paragraph : textFrame.getParagraphs()) {
    for (IPortion portion : paragraph.getPortions()) {
        Point2D.Float point = portion.getCoordinates();
        // ทางเลือก พิมพ์พิกัดที่นี่เพื่อการตรวจสอบ
    }
}
```

#### ขั้นตอนที่ 4: ทำความสะอาดทรัพยากร

กำจัดของคุณ `Presentation` คัดค้านการปล่อยทรัพยากร:
```java
if (presentation != null) presentation.dispose();
```

## คู่มือการใช้งาน
### การปรับระยะห่างระหว่างบรรทัดใน TextFrames
ฟีเจอร์นี้จะแสดงวิธีการปรับระยะห่างระหว่างบรรทัดภายในย่อหน้าของกรอบข้อความ นี่คือคำแนะนำทีละขั้นตอน:
#### โหลดการนำเสนอที่มีอยู่
เริ่มต้นด้วยการโหลดไฟล์นำเสนอของคุณ
```java
Presentation presentation = new Presentation("YOUR_DOCUMENT_DIRECTORY/Foo.pptx");
```
**คำอธิบาย**: นี่คือการเริ่มต้น `Presentation` วัตถุซึ่งให้คุณสามารถจัดการสไลด์ภายในเอกสาร PowerPoint ได้
#### เข้าถึงสไลด์
ขั้นตอนต่อไปคือเข้าถึงสไลด์เฉพาะที่มีกรอบข้อความที่คุณต้องการปรับเปลี่ยน:
```java
ISlide sld = presentation.getSlides().get_Item(0);
```
**คำอธิบาย**สไลด์แรกจะถูกดึงมาโดยใช้การจัดทำดัชนีแบบฐานศูนย์ ปรับดัชนีตามสไลด์ที่คุณต้องการ
#### รับกรอบข้อความจากรูปร่าง
สมมติว่ารูปร่างของคุณเป็น AutoShape ให้ดึงกรอบข้อความของรูปร่างนั้น:
```java
ITextFrame tf1 = ((IAutoShape) sld.getShapes().get_Item(0)).getTextFrame();
```
**คำอธิบาย**: นี่จะเข้าถึง `TextFrame` ของรูปร่างแรกบนสไลด์ ให้แน่ใจว่ารูปร่างนี้เป็น AutoShape ที่เก็บข้อความไว้
#### การเข้าถึงและแก้ไขย่อหน้า
ดึงข้อมูลย่อหน้าที่คุณต้องการปรับเปลี่ยน:
```java
IParagraph para1 = tf1.getParagraphs().get_Item(0);
```
**คำอธิบาย**:นี่คือการดึงย่อหน้าแรกภายในกรอบข้อความ ซึ่งเราจะปรับเปลี่ยนระยะห่างระหว่างบรรทัด
กำหนดค่าพารามิเตอร์ระยะห่างบรรทัดที่คุณต้องการ:
```java
para1.getParagraphFormat().setSpaceWithin(80);  // ช่องว่างระหว่างบรรทัดของย่อหน้าเดียวกัน
para1.getParagraphFormat().setSpaceBefore(40);  // ช่องว่างก่อนเริ่มย่อหน้า
para1.getParagraphFormat().setSpaceAfter(40);   // ช่องว่างหลังจบย่อหน้า
```
**คำอธิบาย**วิธีการเหล่านี้จะปรับระยะห่างเพื่อให้อ่านง่ายและน่ามองมากขึ้น
#### บันทึกการนำเสนอของคุณ
สุดท้ายให้บันทึกการเปลี่ยนแปลงของคุณ:
```java
presentation.save("YOUR_OUTPUT_DIRECTORY/LineSpacing_out.pptx", SaveFormat.Pptx);
```
**คำอธิบาย**:นี่จะเขียนการปรับเปลี่ยนทั้งหมดกลับไปยังไฟล์การนำเสนอใหม่ในไดเร็กทอรีที่ระบุ
### เคล็ดลับการแก้ไขปัญหา
- **ตรวจสอบประเภทรูปร่าง**:ให้แน่ใจว่ารูปร่างที่เข้าถึงนั้นเป็นประเภท `IAutoShape` เพื่อหลีกเลี่ยงข้อยกเว้นรันไทม์
- **เส้นทางไฟล์**:ตรวจสอบว่าเอกสารและเส้นทางเอาต์พุตของคุณได้รับการตั้งค่าอย่างถูกต้องและโปรแกรมสามารถเข้าถึงได้
- **การพึ่งพาอาศัย**:ยืนยันว่า Aspose.Slides ได้รับการเพิ่มไปยังโครงการของคุณอย่างถูกต้องแล้ว

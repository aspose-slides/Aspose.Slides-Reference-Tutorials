# สร้างแผนภูมิวงกลมใน Java ด้วย Aspose.Slides: คู่มือฉบับสมบูรณ์

## แผนภูมิและกราฟ

### การแนะนำ

ในการแสดงภาพข้อมูล แผนภูมิวงกลมเป็นวิธีที่ใช้งานง่ายในการแสดงสัดส่วนภายในชุดข้อมูล อย่างไรก็ตาม เมื่อต้องจัดการกับชุดข้อมูลที่ซับซ้อนซึ่งบางส่วนมีขนาดเล็กกว่าส่วนอื่นๆ อย่างมาก แผนภูมิวงกลมแบบเดิมอาจดูยุ่งเหยิงและตีความได้ยาก แผนภูมิวงกลมของวงกลมช่วยแก้ปัญหานี้โดยแยกส่วนเล็กๆ ออกเป็นแผนภูมิรอง ซึ่งจะทำให้อ่านได้ง่ายขึ้น

ในบทช่วยสอนนี้ คุณจะได้เรียนรู้วิธีสร้างและจัดการแผนภูมิวงกลมโดยใช้ Aspose.Slides สำหรับ Java คุณจะครอบคลุมถึงการตั้งค่าสภาพแวดล้อม การสร้างแผนภูมิ การปรับแต่งคุณสมบัติ เช่น ป้ายข้อมูลและตำแหน่งแยก และการบันทึกการนำเสนอของคุณในรูปแบบ PPTX เมื่อสิ้นสุดบทช่วยสอน คุณจะเชี่ยวชาญคุณสมบัติเหล่านี้ด้วยแอปพลิเคชันในทางปฏิบัติและเคล็ดลับประสิทธิภาพ

**สิ่งที่คุณจะได้เรียนรู้:**
- การตั้งค่า Aspose.Slides สำหรับ Java
- การสร้างแผนภูมิวงกลมจากวงกลม
- การปรับแต่งคุณสมบัติของแผนภูมิ เช่น ป้ายข้อมูลและการกำหนดค่าการแยก
- บันทึกการนำเสนอของคุณลงในดิสก์

พร้อมที่จะเริ่มต้นหรือยัง? มาดูข้อกำหนดเบื้องต้นกันก่อน!

## ข้อกำหนดเบื้องต้น

ก่อนที่จะสร้างแผนภูมิวงกลมของวงกลม ให้แน่ใจว่าคุณมี:

### ไลบรารี เวอร์ชัน และการอ้างอิงที่จำเป็น:
- **Aspose.Slides สำหรับ Java**: จำเป็นสำหรับการจัดการการนำเสนอ PowerPoint ด้วยโปรแกรม

### ข้อกำหนดการตั้งค่าสภาพแวดล้อม:
- ติดตั้ง Java Development Kit (JDK) บนเครื่องของคุณ เราขอแนะนำให้ใช้ JDK 16 ขึ้นไป
- สภาพแวดล้อมการพัฒนาแบบบูรณาการ (IDE) เช่น IntelliJ IDEA, Eclipse หรือ NetBeans

### ข้อกำหนดความรู้เบื้องต้น:
- ความเข้าใจพื้นฐานเกี่ยวกับการเขียนโปรแกรมภาษา Java
- ความคุ้นเคยกับ Maven หรือ Gradle สำหรับการจัดการการอ้างอิง

## การตั้งค่า Aspose.Slides สำหรับ Java

### ข้อมูลการติดตั้ง:

**เมเวน:**
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

**เกรเดิ้ล:**
```gradle
implementation group: 'com.aspose', name: 'aspose-slides', version: '25.4', classifier: 'jdk16'
```

**ดาวน์โหลดโดยตรง**:คุณสามารถดาวน์โหลดเวอร์ชันล่าสุดได้จาก [Aspose.Slides สำหรับการเปิดตัว Java](https://releases-aspose.com/slides/java/).

### ขั้นตอนการรับใบอนุญาต:
- **ทดลองใช้งานฟรี**:เริ่มด้วยการทดลองใช้ฟรี 30 วันเพื่อสำรวจคุณสมบัติทั้งหมด
- **ใบอนุญาตชั่วคราว**:ขอใบอนุญาตชั่วคราวเพื่อการประเมินผลขยายเวลา
- **ซื้อ**:ควรพิจารณาซื้อใบอนุญาตหาก Aspose.Slides ตรงตามความต้องการของคุณ

### การเริ่มต้นและการตั้งค่าเบื้องต้น

เมื่อคุณตั้งค่าไลบรารีในโครงการของคุณแล้ว ให้เริ่มต้นโดยการสร้างอินสแตนซ์ของ `Presentation` ระดับ:

```java
Presentation presentation = new Presentation();
```

ขั้นตอนนี้จะช่วยให้คุณเพิ่มแผนภูมิต่างๆ ลงในสไลด์ได้ ต่อไปเราจะมาดูการใช้แผนภูมิวงกลมของวงกลมกัน

## คู่มือการใช้งาน

### การสร้างแผนภูมิ 'วงกลมของวงกลม'

#### ภาพรวม
เราจะเริ่มต้นด้วยการสร้างอินสแตนซ์ของ `Presentation` และเพิ่มแผนภูมิวงกลมในสไลด์แรก แผนภูมินี้จะแสดงข้อมูลได้อย่างมีประสิทธิภาพโดยแบ่งส่วนย่อยๆ ออกเป็นวงกลมรอง ทำให้อ่านได้ง่ายขึ้น

#### ขั้นตอนที่ 1: สร้างอินสแตนซ์ของคลาสการนำเสนอ
```java
// สร้างการนำเสนอใหม่
ePresentation presentation = new Presentation();
```
โค้ดนี้จะเริ่มต้นการนำเสนอของคุณซึ่งเราจะเพิ่มแผนภูมิของเรา

#### ขั้นตอนที่ 2: เพิ่มแผนภูมิ 'วงกลมของวงกลม' ในสไลด์แรก
```java
// เพิ่มแผนภูมิวงกลมของวงกลมในสไลด์แรกที่ตำแหน่ง (50, 50) ด้วยขนาด (500x400)
eIChart chart = presentation.getSlides().get_Item(0).getShapes().addChart(
    ChartType.PieOfPie, 50, 50, 500, 400);
```
ที่นี่เราจะระบุประเภทของแผนภูมิ (`PieOfPie`) และตำแหน่งและขนาดบนสไลด์

#### ขั้นตอนที่ 3: ตั้งค่าป้ายข้อมูลเพื่อแสดงค่าสำหรับชุดข้อมูล
```java
// กำหนดค่าป้ายข้อมูลเพื่อแสดงค่า
echart.getChartData().getSeries().get_Item(0)
    .getLabels()
    .getDefaultDataLabelFormat()
    .setShowValue(true);
```
ขั้นตอนนี้จะช่วยให้แน่ใจว่าแต่ละส่วนของแผนภูมิวงกลมของเราจะแสดงค่าที่สอดคล้องกัน ช่วยในการตีความข้อมูลอย่างรวดเร็ว

#### ขั้นตอนที่ 4: กำหนดค่าขนาดพายที่สองและแบ่งตามเปอร์เซ็นต์
```java
// กำหนดขนาดของพายรอง
echart.getChartData().getSeries().get_Item(0)
    .getParentSeriesGroup()
    .setSecondPieSize(149);

// แบ่งพายตามเปอร์เซ็นต์
echart.getChartData().getSeries().get_Item(0)
    .getParentSeriesGroup()
    .setPieSplitBy(PieSplitType.ByPercentage);

// ตั้งค่าตำแหน่งการแยก
echart.getChartData().getSeries().get_Item(0)
    .getParentSeriesGroup()
    .setPieSplitPosition(53);
```
การกำหนดค่าเหล่านี้ช่วยให้คุณกำหนดวิธีการแยกแผนภูมิและแสดงส่วนที่เล็กลงได้ ซึ่งจะช่วยเพิ่มความชัดเจนให้กับผู้ชม

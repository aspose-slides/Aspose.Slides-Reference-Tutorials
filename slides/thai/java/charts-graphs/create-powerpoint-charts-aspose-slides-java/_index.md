## คู่มือการใช้งาน

ให้เราแบ่งการใช้งานออกเป็นการสร้างแผนภูมิและการดึงช่วงข้อมูล

### การสร้างแผนภูมิในงานนำเสนอ PowerPoint

#### ภาพรวม
ฟีเจอร์นี้ช่วยให้คุณสร้างแผนภูมิคอลัมน์แบบคลัสเตอร์ภายในสไลด์การนำเสนอของคุณได้โดยใช้ Aspose.Slides สำหรับ Java แผนภูมิจะวางอยู่บนสไลด์แรกด้วยขนาดที่กำหนด

#### ขั้นตอนการดำเนินการ

**ขั้นตอนที่ 1: สร้างงานนำเสนอ**
เริ่มต้นด้วยการสร้างอินสแตนซ์ของ `Presentation`-

```java
Presentation pres = new Presentation();
```

**ขั้นตอนที่ 2: เพิ่มแผนภูมิคอลัมน์แบบคลัสเตอร์**
ใช้ `addChart` วิธีการแทรกแผนภูมิลงในงานนำเสนอของคุณ ระบุประเภท ตำแหน่ง (พิกัด x และ y) และขนาด

```java
IChart chart = pres.getSlides().get_Item(0).getShapes().addChart(
    ChartType.ClusteredColumn, 10, 10, 400, 300);
```
- **คำอธิบายพารามิเตอร์**-
  - `ChartType.ClusteredColumn`: กำหนดประเภทของแผนภูมิ
  - `(10, 10)`:พิกัด X และ Y สำหรับการวางตำแหน่งแผนภูมิบนสไลด์
  - `(400, 300)`: ความกว้างและความสูงของแผนภูมิ

### การดึงข้อมูลช่วงแผนภูมิ

#### ภาพรวม
การเข้าถึงและการดึงช่วงข้อมูลจากแผนภูมิเป็นสิ่งสำคัญสำหรับการตรวจสอบข้อมูลหรือการจัดการเพิ่มเติมภายในแอปพลิเคชันของคุณ

#### ขั้นตอนการดำเนินการ

**ขั้นตอนที่ 1: เพิ่มแผนภูมิคอลัมน์แบบกลุ่ม**
ขั้นแรก ให้เพิ่มแผนภูมิคอลัมน์แบบกลุ่มตามที่อธิบายไว้ก่อนหน้านี้

```java
IChart chart = pres.getSlides().get_Item(0).getShapes().addChart(
    ChartType.ClusteredColumn, 10, 10, 400, 300);
```

**ขั้นตอนที่ 2: ดึงช่วงข้อมูล**
ใช้ `getChartData().getRange()` เพื่อรับการแสดงสตริงของช่วงข้อมูล

```java
String result = chart.getChartData().getRange();
// เอาต์พุตถูกละเว้นเพื่อความชัดเจน
```
- **การดึงข้อมูล**:วิธีนี้จะให้ภาพรวมข้อมูลแผนภูมิของคุณซึ่งมีประโยชน์สำหรับการแก้จุดบกพร่องหรือการแสดงผล

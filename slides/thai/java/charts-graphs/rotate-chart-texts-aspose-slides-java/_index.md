## คู่มือการใช้งาน
เราจะแบ่งขั้นตอนการสร้างและปรับแต่งแผนภูมิด้วยข้อความที่หมุนเป็นขั้นตอนง่าย ๆ

### ขั้นตอนที่ 1: เริ่มต้นการนำเสนอ
สร้างอินสแตนซ์ของ `Presentation` ชั้นเรียนเพื่อทำงานกับไฟล์นำเสนอของคุณ

```java
import com.aspose.slides.Presentation;
import com.aspose.slides.IChart;
import com.aspose.slides.ChartType;
import com.aspose.slides.ISlide;
import com.aspose.slides.SaveFormat;

// กำหนดเส้นทางไดเรกทอรีเอกสารของคุณ
String dataDir = "YOUR_DOCUMENT_DIRECTORY"; // แทนที่ด้วยไดเร็กทอรีจริงของคุณ

Presentation presentation = new Presentation();
```

### ขั้นตอนที่ 2: เพิ่มแผนภูมิลงในสไลด์
เข้าถึงสไลด์แรกและเพิ่ม `ClusteredColumn` แผนภูมิ.

```java
ISlide slide = presentation.getSlides().get_Item(0);
IChart chart = slide.getShapes().addChart(ChartType.ClusteredColumn, 50, 50, 500, 300);
```

### ขั้นตอนที่ 3: ปรับแต่งการหมุนฉลาก
เข้าถึงชุดแรกเพื่อปรับเปลี่ยนป้ายข้อมูล

```java
import com.aspose.slides.IChartData;

IChartSeries series = chart.getChartData().getSeries().get_Item(0);
series.getLabels().getDefaultDataLabelFormat().setShowValue(true);

// ตั้งค่ามุมหมุนข้อความฉลากเป็น 65 องศา
series.getLabels().getDefaultDataLabelFormat().getTextFormat()
        .getTextBlockFormat().setRotationAngle(65);
```

### ขั้นตอนที่ 4: ปรับแต่งการหมุนชื่อแผนภูมิ
เพิ่มชื่อเรื่องและกำหนดมุมการหมุน

```java
chart.setTitle(true);
chart.getChartTitle().addTextFrameForOverriding("Custom title")
        .getTextFrameFormat().setRotationAngle(-30); // หมุนข้อความชื่อเรื่อง -30 องศา
```

### ขั้นตอนที่ 5: บันทึกการนำเสนอของคุณ
บันทึกการนำเสนอของคุณพร้อมการปรับแต่งทั้งหมด

```java
presentation.save(dataDir + "/textframe-rotation_out.pptx", SaveFormat.Pptx);

// ตรวจสอบให้แน่ใจว่าคุณปิดวัตถุการนำเสนอเพื่อปล่อยทรัพยากร
disposePresentation(presentation);
```

#### วิธีช่วยกำจัด
เพื่อบริหารจัดการทรัพยากรอย่างมีประสิทธิภาพ กำจัด `Presentation` วัตถุอย่างถูกต้อง

```java
private static void disposePresentation(Presentation pres) {
    if (pres != null) {
        pres.dispose();
    }
}
```

สำรวจสถานการณ์ในโลกแห่งความเป็นจริงที่ข้อความแผนภูมิที่หมุนสามารถเพิ่มประสิทธิภาพการนำเสนอของคุณได้:
1. **การแสดงภาพข้อมูล**:ปรับปรุงการอ่านได้และความสวยงามในภาพข้อมูลที่ซับซ้อน
2. **การนำเสนอการตลาด**:เน้นตัวชี้วัดหรือข้อความสำคัญด้วยข้อความที่หมุนวางไว้ในตำแหน่งที่เหมาะสม
3. **สื่อการเรียนรู้**:ใช้การหมุนเพื่อเน้นย้ำบางส่วนของบทเรียนหรือการบรรยาย

## การพิจารณาประสิทธิภาพ
เพื่อประสิทธิภาพสูงสุดเมื่อใช้ Aspose.Slides:
- **การจัดการหน่วยความจำ**: กำจัดทิ้งเสมอ `Presentation` วัตถุหลังการใช้งานเพื่อปลดปล่อยทรัพยากร
- **การจัดการข้อมูลอย่างมีประสิทธิภาพ**:ลดปริมาณข้อมูลที่ประมวลผลในครั้งเดียวเพื่อลดการใช้หน่วยความจำ

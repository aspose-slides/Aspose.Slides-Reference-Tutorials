# กู้คืนข้อมูลเวิร์กบุ๊กจากแผนภูมิ PowerPoint โดยใช้ Aspose.Slides Java

## การแนะนำ
การนำทางในงานนำเสนอ โดยเฉพาะงานนำเสนอที่มีข้อมูลที่ซับซ้อนภายในแผนภูมิ อาจเป็นเรื่องท้าทาย บทช่วยสอนนี้จะแนะนำคุณเกี่ยวกับการใช้ Aspose.Slides สำหรับ Java เพื่อกู้คืนข้อมูลเวิร์กบุ๊กที่ฝังอยู่ในแคชแผนภูมิภายในงานนำเสนอ PowerPoint ได้อย่างราบรื่น

**สิ่งที่คุณจะได้เรียนรู้:**
- การตั้งค่า LoadOptions เพื่อกู้คืนเวิร์กบุ๊กจากแคชแผนภูมิ
- การนำไปใช้ทีละขั้นตอนในการกู้คืนข้อมูลเวิร์กบุ๊กโดยใช้ Aspose.Slides สำหรับ Java
- แนวทางปฏิบัติที่ดีที่สุดสำหรับการเพิ่มประสิทธิภาพเมื่อจัดการสเปรดชีตที่ฝังไว้ในงานนำเสนอ PowerPoint

เมื่อสิ้นสุดหลักสูตร คุณจะได้รับทักษะที่จำเป็นในการจัดการการกู้คืนข้อมูลอย่างมีประสิทธิภาพ เริ่มต้นด้วยการครอบคลุมข้อกำหนดเบื้องต้น!

## คู่มือการใช้งาน
เราจะแบ่งการใช้งานออกเป็นสองส่วนหลัก: การกู้คืนข้อมูลเวิร์กบุ๊กจากแคชแผนภูมิและการกำหนดค่า LoadOptions

### กู้คืนสมุดงานจากแคชแผนภูมิ
#### ภาพรวม
คุณลักษณะนี้ช่วยให้สามารถเข้าถึงและกู้คืนข้อมูลเวิร์กบุ๊กที่ฝังอยู่ในแผนภูมิในงานนำเสนอ PowerPoint ได้ โดยรับประกันว่าไม่มีข้อมูลสูญหายในระหว่างกระบวนการแปลงหรือการแก้ไข

#### การดำเนินการแบบทีละขั้นตอน
##### ตั้งค่า LoadOptions สำหรับการกู้คืน
กำหนดค่า `LoadOptions` เพื่อเปิดใช้งานการกู้คืนสมุดงาน:
```java
import com.aspose.slides.*;

String pptxFile = "YOUR_DOCUMENT_DIRECTORY/ExternalWB.pptx";
String outPptxFile = "YOUR_OUTPUT_DIRECTORY/ExternalWB_out.pptx";

// ขั้นตอนที่ 1: ตั้งค่า LoadOptions เพื่อกู้คืนเวิร์กบุ๊กจากแคชแผนภูมิ
LoadOptions lo = new LoadOptions();
lo.getSpreadsheetOptions().setRecoverWorkbookFromChartCache(true);
```
ที่นี่, `setRecoverWorkbookFromChartCache(true)` มีความสำคัญ เนื่องจากเป็นการสั่งให้ Aspose.Slides ดึงข้อมูลเวิร์กบุ๊กที่ฝังอยู่ในแผนภูมิ

##### โหลดงานนำเสนอพร้อมตัวเลือก
โหลดไฟล์ PowerPoint ของคุณโดยใช้ตัวเลือกเหล่านี้:
```java
// ขั้นตอนที่ 2: โหลดงานนำเสนอด้วย LoadOptions ที่ระบุ
Presentation pres = new Presentation(pptxFile, lo);
```
ขั้นตอนนี้จะช่วยให้แน่ใจว่าข้อมูลที่จำเป็นทั้งหมดได้รับการเตรียมพร้อมสำหรับการกู้คืน

##### การเข้าถึงและดึงข้อมูล
ขั้นตอนต่อไป เข้าถึงแผนภูมิและดึงข้อมูลสมุดงานที่เกี่ยวข้อง:
```java
try {
    // ขั้นตอนที่ 3: เข้าถึงแผนภูมิแรกในสไลด์แรก
    IChart chart = (IChart) pres.getSlides().get_Item(0).getShapes().get_Item(0);

    // ขั้นตอนที่ 4: ดึงข้อมูลเวิร์กบุ๊กที่เชื่อมโยงกับแผนภูมิ
    IChartDataWorkbook wb = chart.getChartData().getChartDataWorkbook();
    
    // ขั้นตอนที่ 5: บันทึกการนำเสนอไปยังไฟล์ใหม่
    pres.save(outPptxFile, SaveFormat.Pptx);
} finally {
    if (pres != null) pres.dispose();
}
```
ในสคริปท์นี้:
- เราเข้าถึงแผนภูมิแรกและสมุดงานข้อมูลของมัน
- สุดท้ายเราบันทึกการนำเสนอที่ปรับเปลี่ยนแล้ว

### การกำหนดค่า LoadOptions
#### ภาพรวม
การกำหนดค่า `LoadOptions` ช่วยให้คุณควบคุมวิธีการจัดการเวิร์กบุ๊กที่ฝังไว้ในระหว่างการดำเนินการโหลดได้อย่างมีประสิทธิภาพ

#### คำอธิบายโดยละเอียด
```java
// คุณสมบัติ: การกำหนดค่า LoadOptions
import com.aspose.slides.*;

ตัวเลือกการโหลด lo = new LoadOptions();
lo.getSpreadsheetOptions().setRecoverWorkbookFromChartCache(true);
```
- **LoadOptions**: ตั้งค่าการกำหนดค่าสำหรับการโหลดการนำเสนอ
- **รับตัวเลือกสเปรดชีต()**: ให้การเข้าถึงการตั้งค่าที่เกี่ยวข้องกับสเปรดชีตที่ฝังอยู่
- **ตั้งค่าการกู้คืนเวิร์กบุ๊กจากแคชแผนภูมิ (จริง)**: เปิดใช้งานการกู้คืนข้อมูลเวิร์กบุ๊กจากแคชแผนภูมิ

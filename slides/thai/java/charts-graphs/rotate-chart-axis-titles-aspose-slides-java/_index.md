## คู่มือการใช้งาน
ในส่วนนี้เราจะแนะนำคุณเกี่ยวกับการตั้งค่าแผนภูมิที่มีชื่อแกนหมุนทีละขั้นตอน
### การเพิ่มแผนภูมิคอลัมน์แบบคลัสเตอร์
**ภาพรวม**เริ่มต้นด้วยการเพิ่มแผนภูมิคอลัมน์แบบกลุ่มลงในสไลด์ของคุณ
#### ขั้นตอนที่ 1: สร้างงานนำเสนอ
เริ่มต้นอินสแตนซ์การนำเสนอใหม่:
```java
Presentation pres = new Presentation();
```
บรรทัดโค้ดนี้จะตั้งค่าไฟล์ PowerPoint เปล่าไว้สำหรับการจัดการ
#### ขั้นตอนที่ 2: เพิ่มแผนภูมิคอลัมน์แบบคลัสเตอร์
เพิ่มแผนภูมิลงในสไลด์แรกที่ตำแหน่ง (50, 50) พร้อมด้วยมิติ (450, 300):
```java
IChart chart = pres.getSlides().get_Item(0).getShapes().addChart(ChartType.ClusteredColumn, 50, 50, 450, 300);
```
ที่นี่, `ChartType.ClusteredColumn` ระบุประเภทของแผนภูมิ คุณสามารถเปลี่ยนเป็นประเภทอื่นได้ เช่น `Pie`- `Bar`ฯลฯ ขึ้นอยู่กับความต้องการของคุณ
#### ขั้นตอนที่ 3: เปิดใช้งานและหมุนชื่อแกนแนวตั้ง
ถัดไป เปิดใช้งานชื่อเรื่องสำหรับแกนแนวตั้งและตั้งค่ามุมการหมุน:
```java
// เปิดใช้งานชื่อแกนแนวตั้ง
chart.getAxes().getVerticalAxis().setTitle(true);

// ตั้งค่ามุมการหมุนเป็น 90 องศา
chart.getAxes().getVerticalAxis().getTitle().getTextFormat().getTextBlockFormat().setRotationAngle(90);
```
การ `setRotationAngle` วิธีการนี้ช่วยให้คุณปรับการวางแนวของข้อความได้ ซึ่งจะช่วยเพิ่มความสามารถในการอ่านเมื่อมีพื้นที่จำกัด
#### ขั้นตอนที่ 4: บันทึกการนำเสนอ
สุดท้ายให้บันทึกการเปลี่ยนแปลงของคุณ:
```java
pres.save("YOUR_DOCUMENT_DIRECTORY/test.pptx", SaveFormat.Pptx);
```
แทนที่ "YOUR_DOCUMENT_DIRECTORY" ด้วยเส้นทางจริงที่คุณต้องการจัดเก็บงานนำเสนอของคุณ

# วิธีการยกเว้นฟอนต์เริ่มต้นจากการแปลง HTML โดยใช้ Aspose.Slides สำหรับ Java
## การแนะนำ
เมื่อแปลงงานนำเสนอเป็น HTML การรักษาแบบอักษรที่กำหนดเองของคุณถือเป็นสิ่งสำคัญเนื่องจากการตั้งค่าแบบอักษรเริ่มต้น คู่มือนี้สาธิตว่า Aspose.Slides สำหรับ Java ช่วยให้คุณหลีกเลี่ยงค่าเริ่มต้นเหล่านี้และรับรองการพิมพ์ที่สอดคล้องกันบนแพลตฟอร์มต่างๆ ได้อย่างไร
**สิ่งที่คุณจะได้เรียนรู้:**
- การตั้งค่าสภาพแวดล้อมด้วย Aspose.Slides สำหรับ Java
- เทคนิคในการยกเว้นแบบอักษรเริ่มต้นในระหว่างการแปลง HTML
- ตัวเลือกการกำหนดค่าที่สำคัญและผลกระทบต่อผลลัพธ์
- การประยุกต์ใช้งานจริงในสถานการณ์โลกแห่งความเป็นจริง
มาเริ่มต้นด้วยการหารือเกี่ยวกับข้อกำหนดเบื้องต้นก่อนที่จะเจาะลึกคู่มือการใช้งาน

## คู่มือการใช้งาน
### ภาพรวมคุณลักษณะ: การแยกแบบอักษรเริ่มต้นออกจากการแปลง HTML
ฟีเจอร์นี้ช่วยปรับแต่งการจัดการแบบอักษรระหว่างการแปลงไฟล์ PowerPoint เป็น HTML เพื่อเพิ่มการสร้างแบรนด์และความสอดคล้องกัน
#### ขั้นตอนที่ 1: เตรียมสภาพแวดล้อมของคุณ
ตรวจสอบให้แน่ใจว่า Aspose.Slides ได้รับการตั้งค่าอย่างถูกต้องตามคำแนะนำด้านบน ซึ่งเกี่ยวข้องกับการเพิ่มการอ้างอิงหรือดาวน์โหลด JAR ลงในโปรเจ็กต์ของคุณโดยตรง
#### ขั้นตอนที่ 2: โหลดงานนำเสนอ
โหลดการนำเสนอของคุณโดยใช้ `Presentation` ระดับ:
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY/presentation.pptx";
try {
    Presentation pres = new Presentation(dataDir);
```
#### ขั้นตอนที่ 3: กำหนดการยกเว้นแบบอักษร
สร้างอาร์เรย์เพื่อระบุแบบอักษรที่คุณต้องการยกเว้น ในตัวอย่างนี้ เราจะเริ่มต้นด้วยรายการว่างเป็นตำแหน่งว่าง:
```java
String[] fontNameExcludeList = {};
```
#### ขั้นตอนที่ 4: เริ่มต้นตัวควบคุม HTML แบบกำหนดเอง
การ `LinkAllFontsHtmlController` คลาสนี้ใช้สำหรับการจัดการฟอนต์แบบกำหนดเองในระหว่างกระบวนการแปลง
```java
LinkAllFontsHtmlController linkcont = new LinkAllFontsHtmlController(fontNameExcludeList, "YOUR_DOCUMENT_DIRECTORY");
```
#### ขั้นตอนที่ 5: กำหนดค่าตัวเลือก HTML
ตั้งค่าของคุณ `HtmlOptions` ในการใช้ตัวจัดรูปแบบแบบกำหนดเอง:
```java
HtmlOptions htmlOptionsEmbed = new HtmlOptions();
htmlOptionsEmbed.setHtmlFormatter(HtmlFormatter.createCustomFormatter(linkcont));
```
#### ขั้นตอนที่ 6: บันทึกเป็น HTML
สุดท้ายให้บันทึกงานนำเสนอที่แปลงแล้วเป็นรูปแบบ HTML:
```java
pres.save("YOUR_OUTPUT_DIRECTORY/pres.html", SaveFormat.Html, htmlOptionsEmbed);
} catch (Exception e) {
    e.printStackTrace();
}
```
**คำอธิบาย:** ตัวอย่างโค้ดนี้สาธิตวิธีการยกเว้นแบบอักษรเริ่มต้นโดยการกำหนดค่าตัวจัดรูปแบบแบบกำหนดเองในระหว่างการแปลง HTML

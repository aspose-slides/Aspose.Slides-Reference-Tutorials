## คู่มือการใช้งาน

### การสร้างและการกำหนดค่าสไลด์

#### ภาพรวม

หัวข้อนี้จะแนะนำคุณเกี่ยวกับการสร้างสไลด์ การเพิ่มรูปร่าง และการบันทึกการนำเสนอ

1. **การเริ่มต้นการนำเสนอ**
   เริ่มต้นด้วยการตั้งค่าไดเรกทอรีการทำงานของคุณและเริ่มต้นใช้งาน `Presentation` ระดับ:
    
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";

if (!Directory.Exists(dataDir))
    Directory.CreateDirectory(dataDir);
    
Presentation pres = new Presentation();
```

2. **เพิ่มรูปทรงสี่เหลี่ยมผืนผ้า**
   เพิ่มรูปร่างลงในสไลด์ของคุณซึ่งคุณสามารถวางข้อความได้ภายหลัง
    
```csharp
ISlide sld = pres.Slides[0];
IAutoShape rect = sld.Shapes.AddAutoShape(ShapeType.Rectangle, 100, 100, 500, 150);
```

3. **บันทึกการนำเสนอ**
   บันทึกงานของคุณลงในดิสก์:
    
```csharp
pres.Save(dataDir + "/CreatedSlide.pptx", SaveFormat.Pptx);
```

### การเพิ่มและการจัดรูปแบบข้อความในรูปทรง

#### ภาพรวม
ที่นี่เราจะเพิ่มข้อความลงในรูปร่างและกำหนดลักษณะที่ปรากฏของมัน

1. **เพิ่ม TextFrame**
   ฝัง `TextFrame` ภายในสี่เหลี่ยมที่คุณสร้างขึ้น:
    
```csharp
ITextFrame tf = rect.AddTextFrame("This is first line \rThis is second line \rThis is third line");
```

2. **ตั้งค่าประเภทการปรับพอดีอัตโนมัติ**
   ตรวจสอบให้แน่ใจว่าข้อความพอดีกับขอบเขตรูปร่าง:
    
```csharp
tf.TextFrameFormat.AutofitType = TextAutofitType.Shape;
```

3. **ซ่อนเส้นรูปร่าง**
   ทางเลือกอื่นคือซ่อนเส้นสี่เหลี่ยมผืนผ้าเพื่อให้ดูสะอาดขึ้น:
    
```csharp
rect.LineFormat.FillFormat.FillType = FillType.NoFill; // เปลี่ยนเป็น NoFill เพื่อไม่ให้มีเส้นที่มองเห็นได้
```

4. **บันทึกการนำเสนอ**
   บันทึกการเปลี่ยนแปลงของคุณ:
    
```csharp
pres.Save(dataDir + "/TextFormattedSlide.pptx", SaveFormat.Pptx);
```

### การกำหนดค่าการเยื้องย่อหน้าและรูปแบบหัวข้อย่อย

#### ภาพรวม
ตอนนี้ เรามาจัดรูปแบบย่อหน้าด้วยจุดหัวข้อและการเยื้องกัน

1. **ตั้งค่าหัวข้อย่อยและการจัดตำแหน่งสำหรับย่อหน้า**
   กำหนดค่าให้แต่ละย่อหน้าแสดงจุดหัวข้อย่อย:
    
```csharp
foreach (IParagraph para in tf.Paragraphs)
{
    para.ParagraphFormat.Bullet.Type = BulletType.Symbol;
    para.ParagraphFormat.Bullet.Char = Convert.ToChar(8226);
    para.ParagraphFormat.Alignment = TextAlignment.Left;

    // ตั้งค่าความลึกและการเยื้องตามดัชนีย่อหน้า
    para.ParagraphFormat.Depth = 2; 
    para.ParagraphFormat.Indent = 30 + (tf.Paragraphs.IndexOf(para) * 10);
}
```

2. **บันทึกการนำเสนอ**
   สรุปการเปลี่ยนแปลงของคุณ:
    
```csharp
pres.Save(dataDir + "/IndentedTextSlide.pptx", SaveFormat.Pptx);
```

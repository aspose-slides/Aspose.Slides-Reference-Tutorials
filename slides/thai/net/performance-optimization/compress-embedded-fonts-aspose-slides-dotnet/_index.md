## คู่มือการใช้งาน: การบีบอัดแบบอักษรที่ฝังไว้ในงานนำเสนอ
### ภาพรวม
คุณสมบัตินี้ช่วยลดขนาดไฟล์โดยการบีบอัดฟอนต์ที่ฝังไว้ ทำให้สามารถแชร์การนำเสนอได้ง่ายขึ้น

#### การดำเนินการแบบทีละขั้นตอน
##### 1. กำหนดเส้นทางสำหรับเอกสารอินพุตและเอาต์พุต
ตั้งค่าเส้นทางสำหรับไฟล์ของคุณ:
```csharp
string presentationName = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "presWithEmbeddedFonts.pptx");
string outPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "presWithEmbeddedFonts-out.pptx");
```
##### 2. โหลดงานนำเสนอ
โหลดไฟล์ PowerPoint ของคุณโดยใช้ Aspose.Slides:
```csharp
using (Presentation pres = new Presentation(presentationName))
{
    // จะดำเนินการเพิ่มเติมกับวัตถุนี้
}
```
##### 3. บีบอัดฟอนต์ที่ฝังไว้
เรียก `CompressEmbeddedFonts` เพื่อเพิ่มประสิทธิภาพการจัดเก็บแบบอักษรภายในไฟล์:
```csharp
pres.FontsManager.CompressEmbeddedFonts();
```
*ทำไม*:วิธีนี้จะลดขนาดข้อมูลแบบอักษรที่ฝังไว้โดยไม่สูญเสียคุณภาพ
##### 4. บันทึกการนำเสนอที่แก้ไขแล้ว
บันทึกการนำเสนอของคุณด้วยการตั้งค่าใหม่:
```csharp
pres.Save(outPath, Aspose.Slides.Export.SaveFormat.Pptx);
```
##### การตรวจสอบผลลัพธ์การบีบอัด
เปรียบเทียบขนาดไฟล์ก่อนและหลังการบีบอัด:
```csharp
FileInfo fi = new FileInfo(presentationName);
Console.WriteLine("Source file size = {0:N0} bytes", fi.Length);

fi = new FileInfo(outPath);
Console.WriteLine("Result file size = {0:N0} bytes", fi.Length);
```
### เคล็ดลับการแก้ไขปัญหา
- ตรวจสอบให้แน่ใจว่าเส้นทางไฟล์อินพุตถูกต้องและสามารถเข้าถึงได้
- ตรวจสอบการอัปเดต Aspose.Slides ที่อาจรวมถึงการแก้ไขจุดบกพร่องหรือการปรับปรุง

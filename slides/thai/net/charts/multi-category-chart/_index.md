# การสร้างแผนภูมิคอลัมน์แบบคลัสเตอร์ด้วย Aspose.Slides สำหรับ .NET

Aspose.Slides สำหรับ .NET มอบความสามารถอันทรงพลังสำหรับการสร้างและจัดการการนำเสนอ PowerPoint ด้วยโปรแกรม ตัวอย่างนี้แสดงวิธีการสร้างแผนภูมิคอลัมน์แบบคลัสเตอร์พร้อมหมวดหมู่และชุดข้อมูลที่กำหนดเอง

## การติดตั้ง

```csharp
// ติดตั้งผ่านตัวจัดการแพ็คเกจ NuGet
Install-Package Aspose.Slides
```

หรือเพิ่มการอ้างอิงต่อไปนี้ในไฟล์โครงการของคุณ:

```xml
<PackageReference Include="Aspose.Slides" Version="23.2.0" />
```

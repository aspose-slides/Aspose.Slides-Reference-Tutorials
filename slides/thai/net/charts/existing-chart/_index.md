# การปรับเปลี่ยนแผนภูมิใน PowerPoint ด้วย Aspose.Slides

Aspose.Slides สำหรับ .NET มอบ API ที่มีประสิทธิภาพในการสร้างและปรับเปลี่ยนการนำเสนอ PowerPoint ด้วยโปรแกรม ตัวอย่างนี้แสดงวิธีการทำงานกับแผนภูมิในการนำเสนอที่มีอยู่โดยการอัปเดตข้อมูลแผนภูมิ เพิ่มชุดข้อมูลใหม่ และเปลี่ยนประเภทแผนภูมิ ทั้งหมดนี้ไม่จำเป็นต้องติดตั้ง Microsoft PowerPoint

## คำแนะนำในการติดตั้ง

1. ติดตั้งผ่านตัวจัดการแพ็กเกจ NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. หรือใช้ .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. เพิ่มการใช้คำสั่งดังต่อไปนี้:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Charts;
   ```

## คู่มือการใช้งาน

เมื่อตั้งค่าสภาพแวดล้อมเสร็จเรียบร้อยแล้ว มาปรับเปลี่ยนคุณสมบัติการนำเสนอ PowerPoint โดยใช้ Aspose.Slides สำหรับ .NET กัน

### การเข้าถึงคุณสมบัติการนำเสนอ

#### ภาพรวม
การเข้าถึงและแก้ไขคุณสมบัติในตัวของไฟล์ PowerPoint:

```csharp
using System;
using Aspose.Slides;

// กำหนดไดเรกทอรีเอกสารของคุณ
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// สร้างอินสแตนซ์คลาสการนำเสนอ
Presentation presentation = new Presentation(dataDir + "/ModifyBuiltinProperties.pptx");

// เข้าถึงคุณสมบัติในตัว
IDocumentProperties documentProperties = presentation.DocumentProperties;
```

#### คำอธิบาย
- **`dataDir`**: เส้นทางไปยังไฟล์ PowerPoint อินพุตของคุณ
- **`outputDir`**:ไดเรกทอรีที่จะบันทึกงานนำเสนอที่ปรับเปลี่ยนแล้ว

### การปรับเปลี่ยนคุณสมบัติในตัว
ตั้งค่าคุณสมบัติต่างๆ ดังต่อไปนี้:

**ผู้เขียน:**
```csharp
documentProperties.Author = "Aspose.Slides for .NET";
```
- กำหนดผู้เขียนงานนำเสนอ

**ชื่อ:**
```csharp
documentProperties.Title = "Modifying Presentation Properties with Aspose.Slides";
```
- อัปเดตหัวข้อการนำเสนอของคุณ

**หัวเรื่อง, ความคิดเห็น และผู้จัดการ:**
```csharp
documentProperties.Subject = "Aspose Subject";
documentProperties.Comments = "Aspose Description";
documentProperties.Manager = "Aspose Manager";
```
- คุณสมบัติเหล่านี้ให้ข้อมูลเมตาเพิ่มเติมเกี่ยวกับเอกสาร

### การบันทึกการเปลี่ยนแปลง
บันทึกการปรับเปลี่ยนของคุณด้วย:

```csharp
presentation.Save(outputDir + "/DocumentProperties_out.pptx", SaveFormat.Pptx);
```

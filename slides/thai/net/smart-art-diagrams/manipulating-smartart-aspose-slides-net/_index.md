## คู่มือการใช้งาน
เราจะครอบคลุมคุณลักษณะเฉพาะโดยใช้ Aspose.Slides สำหรับ .NET

### การโหลดงานนำเสนอ
เริ่มต้นด้วยการโหลดไฟล์งานนำเสนอที่มีอยู่เพื่อเพิ่ม SmartArt หรือทำการปรับเปลี่ยน

**โค้ดตัวอย่าง:**
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
Presentation pres = new Presentation(dataDir + "/AccessChildNodes.pptx");
```
*คำอธิบาย:* โค้ดด้านบนโหลดไฟล์ PowerPoint จากไดเร็กทอรีที่คุณระบุ เพื่อเตรียมพร้อมสำหรับการจัดการเพิ่มเติม

### การเพิ่มและการวางตำแหน่งรูปร่าง SmartArt
ปรับปรุงสไลด์ของคุณด้วยการเพิ่มรูปร่าง SmartArt หัวข้อนี้จะแนะนำคุณเกี่ยวกับการวางตำแหน่ง SmartArt บนสไลด์ของคุณอย่างแม่นยำ

**ภาพรวม:**
เพิ่มเค้าโครง SmartArt ลงในสไลด์แรกตามพิกัดที่กำหนดพร้อมมิติที่กำหนดไว้

**โค้ดตัวอย่าง:**
```csharp
ISmartArt smart = pres.Slides[0].Shapes.AddSmartArt(20, 20, 600, 500, SmartArtLayoutType.OrganizationChart);
```
*คำอธิบาย:* การ `AddSmartArt` วิธีการนี้จะวางรูปร่าง SmartArt ใหม่บนสไลด์ พารามิเตอร์จะกำหนดตำแหน่งและขนาดของรูปร่างดังกล่าว

**การย้ายรูปร่างของโหนดย่อย:**
```csharp
ISmartArtNode node = smart.AllNodes[1];
ISmartArtShape shape = node.Shapes[1];
shape.X += (shape.Width * 2); // ขยับไปทางขวาเป็นสองเท่าของความกว้าง
shape.Y -= (shape.Height / 2); // ขยับขึ้นครึ่งหนึ่งของความสูง
```
*คำอธิบาย:* ปรับตำแหน่งรูปร่างโหนดย่อยเฉพาะภายใน SmartArt

### การปรับความกว้างและความสูงของรูปร่าง
ปรับเปลี่ยนขนาดรูปร่างให้เหมาะกับความต้องการในการออกแบบงานนำเสนอของคุณมากขึ้น

**โค้ดตัวอย่าง:**
```csharp
node = smart.AllNodes[2];
shape = node.Shapes[1];
shape.Width += (shape.Width / 2); // เพิ่มความกว้างขึ้นครึ่งหนึ่งจากขนาดเดิม

node = smart.AllNodes[3];
shape = node.Shapes[1];
shape.Height += (shape.Height / 2); // เพิ่มความสูงได้ครึ่งหนึ่ง
```
*คำอธิบาย:* บรรทัดโค้ดเหล่านี้ปรับขนาดรูปร่างเพื่อเพิ่มความน่าสนใจทางสายตา

### การหมุนรูปร่าง SmartArt
หมุนรูปทรงเพื่อสร้างเค้าโครงที่น่าสนใจและเป็นแบบไดนามิก

**โค้ดตัวอย่าง:**
```csharp
node = smart.AllNodes[4];
shape = node.Shapes[1];
shape.Rotation = 90; // หมุนได้ 90 องศา
```
*คำอธิบาย:* บรรทัดโค้ดที่เรียบง่ายนี้จะหมุนรูปร่างที่เลือกภายใน SmartArt เพิ่มความคิดสร้างสรรค์ให้กับสไลด์ของคุณ

### การบันทึกการนำเสนอ
หลังจากทำการเปลี่ยนแปลงทั้งหมดแล้ว ให้บันทึกการนำเสนอในไดเร็กทอรีเอาต์พุตที่คุณต้องการ

**โค้ดตัวอย่าง:**
```csharp
pres.Save("YOUR_OUTPUT_DIRECTORY/SmartArt.pptx");
```
*คำอธิบาย:* การ `Save` วิธีการนี้จะยืนยันการแก้ไขทั้งหมดที่ทำในระหว่างเซสชันไปยังไฟล์ใหม่

# สร้างแผนผังองค์กรโดยใช้ Aspose.Slides สำหรับ .NET: คู่มือที่ครอบคลุม

## คู่มือการใช้งาน

### การเริ่มต้นการนำเสนอ
เริ่มต้นด้วยการสร้างอินสแตนซ์ใหม่ของ `Presentation` คลาส นี่คือไฟล์ PowerPoint เปล่าที่เราจะเพิ่มแผนภูมิองค์กร SmartArt ของเรา

**ขั้นตอนที่ 1: สร้างวัตถุการนำเสนอใหม่**
```csharp
using Aspose.Slides;
using Aspose.Slides.Export;

// เริ่มต้นวัตถุการนำเสนอใหม่
using (Presentation presentation = new Presentation()) {
    // โค้ดสำหรับเพิ่ม SmartArt จะอยู่ที่นี่
}
```

### การเพิ่ม SmartArt
ตอนนี้เพิ่มแผนผังองค์กรลงในสไลด์แรกของคุณโดยใช้ `AddSmartArt`-

**ขั้นตอนที่ 2: เพิ่ม SmartArt**
```csharp
// เพิ่ม SmartArt ด้วยพิกัด ขนาด และประเภทเค้าโครงที่ระบุ
ISmartArt smart = presentation.Slides[0].Shapes.AddSmartArt(10, 10, 400, 300, SmartArtLayoutType.OrganizationChart);
```
ขั้นตอนนี้เกี่ยวข้องกับการระบุตำแหน่ง (`x`- `y`), ขนาด (ความกว้าง ความสูง) และประเภทของเค้าโครงสำหรับ SmartArt ของคุณ

### การกำหนดค่าเค้าโครงโหนด
แต่ละโหนดในแผนผังองค์กรสามารถกำหนดรูปแบบได้เป็นรายบุคคล ต่อไปนี้เป็นวิธีตั้งค่าเค้าโครงแบบกำหนดเองสำหรับโหนดแรก

**ขั้นตอนที่ 3: ตั้งค่าเค้าโครงแผนผังองค์กร**
```csharp
// กำหนดเค้าโครงแผนผังองค์กรสำหรับโหนดแรก
smart.Nodes[0].OrganizationChartLayout = OrganizationChartLayoutType.LeftHanging;
```

### การบันทึกการนำเสนอของคุณ
สุดท้าย ให้บันทึกงานนำเสนอของคุณลงในไฟล์ ตรวจสอบให้แน่ใจว่าคุณได้ระบุไดเรกทอรีเอาต์พุตของคุณอย่างถูกต้อง

**ขั้นตอนที่ 4: บันทึกการนำเสนอ**
```csharp
// บันทึกการนำเสนอไปที่ไดเร็กทอรีเอาท์พุตที่ระบุ
presentation.Save(outputDir + "OrganizeChartLayoutType_out.pptx", SaveFormat.Pptx);
```

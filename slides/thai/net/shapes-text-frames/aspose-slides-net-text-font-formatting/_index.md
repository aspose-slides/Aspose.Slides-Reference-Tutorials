# เรียนรู้การจัดรูปแบบข้อความและแบบอักษรในงานนำเสนอโดยใช้ Aspose.Slides สำหรับ .NET

ในยุคดิจิทัลทุกวันนี้ การสร้างงานนำเสนอที่ดึงดูดสายตาถือเป็นสิ่งสำคัญ ไม่ว่าจะเป็นการประชุมทางธุรกิจ การบรรยายเชิงวิชาการ หรือโปรเจ็กต์ส่วนตัว การออกแบบงานนำเสนอที่มีประสิทธิภาพมักขึ้นอยู่กับความสามารถในการจัดรูปแบบข้อความภายในรูปทรงต่างๆ เช่น สี่เหลี่ยมผืนผ้าหรือวงกลม บทช่วยสอนนี้จะแนะนำคุณเกี่ยวกับการใช้ **Aspose.Slides สำหรับ .NET** เพื่อยกระดับสไลด์ของคุณด้วยข้อความและรูปแบบอักษรที่กำหนดเอง

## สิ่งที่คุณจะได้เรียนรู้
- วิธีการเพิ่มข้อความลงในรูปร่างอัตโนมัติในงานนำเสนอ
- การตั้งค่าความสูงของฟอนต์เริ่มต้นสำหรับการนำเสนอทั้งหมด
- การปรับแต่งความสูงของแบบอักษรสำหรับแต่ละย่อหน้าและส่วนต่างๆ
- บันทึกการนำเสนอที่จัดรูปแบบของคุณอย่างมีประสิทธิภาพ

เราจะสำรวจข้อกำหนดเบื้องต้น ขั้นตอนการตั้งค่า แอปพลิเคชันในทางปฏิบัติ ข้อควรพิจารณาด้านประสิทธิภาพ และสรุปด้วยส่วนคำถามที่พบบ่อย มาดำดิ่งสู่โลกแห่ง **Aspose.Slides สำหรับ .NET**-

## คู่มือการใช้งาน

เราจะแบ่งการใช้งานออกเป็นส่วนต่างๆ ตามฟังก์ชันการทำงาน

### การเพิ่มข้อความลงในรูปร่าง

#### ภาพรวม
ฟีเจอร์นี้ช่วยให้คุณสามารถเพิ่มข้อความที่กำหนดเองภายในรูปร่างอัตโนมัติ เช่น รูปสี่เหลี่ยมผืนผ้าในสไลด์ของคุณ ซึ่งเป็นสิ่งสำคัญสำหรับการนำเสนอเนื้อหาที่ปรับแต่งให้เหมาะสมโดยตรงบนรูปร่างสไลด์

#### ขั้นตอนการดำเนินการ

**1. สร้างและเพิ่มรูปร่างอัตโนมัติ**

```csharp
using (Presentation pres = new Presentation())
{
    IAutoShape newShape = pres.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 100, 100, 400, 75, false);
```
- **พารามิเตอร์**- 
  - `ShapeType.Rectangle`: กำหนดประเภทรูปร่าง
  - พิกัด (x=100, y=100) และขนาด (ความกว้าง=400, ความสูง=75): ตำแหน่งและขนาดของรูปร่าง

**2. เพิ่มกรอบข้อความ**

```csharp
    newShape.AddTextFrame("");
```
- **วัตถุประสงค์**:สร้างกรอบข้อความว่างเพื่อเก็บข้อความที่คุณกำหนดเอง

**3. แทรกส่วนข้อความ**

```csharp
    newShape.TextFrame.Paragraphs[0].Portions.Clear();
    
    IPortion portion0 = new Portion("Sample text with first portion");
    IPortion portion1 = new Portion(" and second portion.");
    
    newShape.TextFrame.Paragraphs[0].Portions.Add(portion0);
    newShape.TextFrame.Paragraphs[0].Portions.Add(portion1);
}
```
- **คำอธิบาย**:ล้างส่วนที่มีอยู่ จากนั้นสร้างและเพิ่มส่วนข้อความใหม่ วิธีนี้ช่วยให้สามารถแบ่งเนื้อหาออกเป็นย่อหน้าเดียวได้

### การตั้งค่าความสูงของฟอนต์เริ่มต้นสำหรับการนำเสนอ

#### ภาพรวม
การตั้งค่าความสูงของแบบอักษรที่สม่ำเสมอตลอดงานนำเสนอทั้งหมดจะช่วยให้การออกแบบและการอ่านมีความสอดคล้องกัน

#### ขั้นตอนการดำเนินการ

**1. เพิ่มส่วนข้อความ**
นำโค้ดกลับมาใช้ใหม่เพื่อเพิ่มส่วนข้อความดังที่แสดงไว้ด้านบน

**2. ตั้งค่าความสูงของฟอนต์เริ่มต้น**

```csharp
    pres.DefaultTextStyle.GetLevel(0).DefaultPortionFormat.FontHeight = 24;
```
- **วัตถุประสงค์**:ใช้ความสูงของฟอนต์ที่สม่ำเสมอ 24 จุดกับส่วนข้อความทั้งหมดในงานนำเสนอ

### การตั้งค่าความสูงของฟอนต์เริ่มต้นสำหรับย่อหน้า

#### ภาพรวม
คุณสามารถปรับแต่งย่อหน้าแต่ละย่อหน้าภายในสไลด์ของคุณได้ ซึ่งจะทำให้เนื้อหาที่เจาะจงโดดเด่นขึ้นมา

#### ขั้นตอนการดำเนินการ

**1. เพิ่มส่วนข้อความ**
ตามที่ได้ระบุไว้ก่อนหน้านี้

**2. ปรับแต่งความสูงของแบบอักษรสำหรับย่อหน้าเฉพาะ**

```csharp
    newShape.TextFrame.Paragraphs[0].ParagraphFormat.DefaultPortionFormat.FontHeight = 40;
```
- **คำอธิบาย**กำหนดความสูงของแบบอักษรของส่วนต่าง ๆ ทั้งหมดภายในย่อหน้านี้ไว้ที่ 40 จุด เพื่อให้ดูโดดเด่นยิ่งขึ้น

### การตั้งค่าความสูงของฟอนต์สำหรับส่วนต่างๆ

#### ภาพรวม
หากต้องการควบคุมการพิมพ์ของงานนำเสนอของคุณอย่างแม่นยำ ให้ปรับขนาดตัวอักษรของส่วนข้อความที่ต้องการทีละส่วน

#### ขั้นตอนการดำเนินการ

**1. เพิ่มส่วนข้อความ**
ย้อนกลับไปดูขั้นตอนเริ่มต้นในการเพิ่มส่วนข้อความ

**2. ตั้งค่าความสูงของฟอนต์โดยเฉพาะ**

```csharp
    newShape.TextFrame.Paragraphs[0].Portions[0].PortionFormat.FontHeight = 55;
    newShape.TextFrame.Paragraphs[0].Portions[1].PortionFormat.FontHeight = 18;
```
- **คำอธิบาย**การปรับแต่งนี้กำหนดความสูงของตัวอักษรให้แต่ละส่วนไม่เหมือนกัน ช่วยให้เน้นรายละเอียดได้ตามต้องการ

### การบันทึกการนำเสนอ

#### ภาพรวม
เมื่อนำเสนอของคุณได้รับการออกแบบอย่างสมบูรณ์แบบแล้ว ให้บันทึกเป็นรูปแบบไฟล์ที่คุณต้องการ

```csharp
using (Presentation pres = new Presentation())
{
    // เพิ่มรูปทรงและข้อความตามที่อธิบายไว้ข้างต้น...

    // บันทึกการนำเสนอ
    pres.Save("YOUR_OUTPUT_DIRECTORY\SetLocalFontHeightValues.pptx", SaveFormat.Pptx);
}
```
- **รายละเอียด**:การดำเนินการนี้จะบันทึกสไลด์ที่จัดรูปแบบของคุณลงในไฟล์ PPTX เพื่อพร้อมสำหรับการแจกจ่ายหรือแก้ไขเพิ่มเติม

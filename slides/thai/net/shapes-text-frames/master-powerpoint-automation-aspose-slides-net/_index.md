## คู่มือการใช้งาน
ตอนนี้ มาแบ่งการใช้งานของเราออกเป็นขั้นตอนที่สามารถจัดการได้

### คุณสมบัติ 1: การสร้างไดเรกทอรี
**ภาพรวม:** คุณลักษณะนี้ช่วยให้แน่ใจว่าแอปพลิเคชันของคุณมีโครงสร้างไดเร็กทอรีที่จำเป็นก่อนที่จะพยายามดำเนินการกับไฟล์

#### ทีละขั้นตอน:
1. **ตรวจสอบการมีอยู่ของไดเรกทอรี**
   ```csharp
   using System.IO;

   string dataDir = "YOUR_DOCUMENT_DIRECTORY";
   bool isExists = Directory.Exists(dataDir);
   ```
2. **สร้างไดเรกทอรีหากไม่มีอยู่**
   ```csharp
   if (!isExists)
   {
       Directory.CreateDirectory(dataDir); // สร้างไดเร็กทอรีตามเส้นทางที่ระบุ
   }
   ```
   
#### คำอธิบาย:
- `Directory.Exists`: ตรวจสอบว่าไดเร็กทอรีมีอยู่ที่เส้นทางที่ระบุหรือไม่
- `Directory.CreateDirectory`: สร้างไดเร็กทอรีใหม่

### คุณลักษณะที่ 2: การสร้างอินสแตนซ์ของวัตถุการนำเสนอ
**ภาพรวม:** ฟีเจอร์นี้สาธิตวิธีการสร้างงานนำเสนอ PowerPoint ที่ว่างเปล่าโดยใช้ Aspose.Slides
```csharp
using (Presentation pres = new Presentation())
{
    // วัตถุ 'pres' แสดงถึงการนำเสนอ PowerPoint ของคุณ
}
```
#### คำอธิบาย:
- `new Presentation()`: เริ่มต้นวัตถุการนำเสนอใหม่ที่ว่างเปล่า

### คุณสมบัติที่ 3: การเพิ่ม AutoShape ด้วย TextFrame และเอฟเฟกต์เงา
**ภาพรวม:** เรียนรู้วิธีการเพิ่มรูปสี่เหลี่ยมผืนผ้าพร้อมข้อความและการใช้เอฟเฟ็กต์เงาเพื่อปรับปรุงภาพ

#### ทีละขั้นตอน:
1. **เพิ่มรูปร่างอัตโนมัติ**
   ```csharp
   ISlide slide = pres.Slides[0]; // รับข้อมูลอ้างอิงของสไลด์แรก
   IAutoShape autoShape = slide.Shapes.AddAutoShape(ShapeType.Rectangle, 150, 75, 150, 50); // เพิ่มรูปทรงสี่เหลี่ยมผืนผ้า
   ```
2. **เพิ่ม TextFrame**
   ```csharp
   autoShape.AddTextFrame("Aspose TextBox"); // แทรกข้อความลงในรูปร่าง
   autoShape.FillFormat.FillType = FillType.NoFill; // ปิดใช้งานการเติมเพื่อให้สามารถมองเห็นเอฟเฟกต์เงาได้
   ```
3. **ใช้เอฟเฟกต์เงา**
   ```csharp
   autoShape.EffectFormat.EnableOuterShadowEffect(); 
   IOuterShadow shadow = autoShape.EffectFormat.OuterShadowEffect;

   // กำหนดค่าคุณสมบัติเงา:
   shadow.BlurRadius = 4.0; // ตั้งค่ารัศมีการเบลอ
   shadow.Direction = 45; // กำหนดมุมทิศทาง
   shadow.Distance = 3; // ระบุระยะห่างจากข้อความ
   shadow.RectangleAlign = RectangleAlignment.TopLeft; // จัดตำแหน่งเงาสี่เหลี่ยม
   shadow.ShadowColor.PresetColor = PresetColor.Black; // เลือกสีดำสำหรับเงา
   ```

#### คำอธิบาย:
- **ออโต้เชพ**:รูปทรงอเนกประสงค์ที่สามารถปรับแต่งด้วยคุณสมบัติต่างๆ รวมถึงข้อความและเอฟเฟกต์
- **เอฟเฟกต์เงาภายนอก**:ใช้เงาที่สมจริงเพื่อเพิ่มความลึกให้กับภาพ

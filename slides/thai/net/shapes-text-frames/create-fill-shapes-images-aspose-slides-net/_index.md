## คู่มือการใช้งาน

ในส่วนนี้จะแบ่งกระบวนการออกเป็นคุณลักษณะที่สามารถจัดการได้

### การสร้างไดเรกทอรี

เพื่อให้แน่ใจว่าไฟล์การนำเสนอของเราได้รับการบันทึกอย่างถูกต้อง เราต้องตรวจสอบก่อนว่าไดเร็กทอรีเป้าหมายมีอยู่หรือไม่ หากไม่มี เราจะสร้างไดเร็กทอรีนั้นขึ้นมา:
```csharp
using System.IO;

string dataDir = "YOUR_DOCUMENT_DIRECTORY";
bool isExists = Directory.Exists(dataDir);
if (!isExists)
{
    // สร้างไดเรกทอรีหากไม่มีอยู่
    Directory.CreateDirectory(dataDir);
}
```

### การทำงานกับการนำเสนอ

เราเริ่มต้นด้วยการสร้างอินสแตนซ์ของการนำเสนอ จากนั้นจึงจัดการสไลด์ของมัน:
```csharp
using Aspose.Slides;

// สร้างอินสแตนซ์คลาสการนำเสนอที่แสดงไฟล์ PPTX
using (Presentation pres = new Presentation())
{
    // รับสไลด์แรกจากการนำเสนอ
    ISlide sld = pres.Slides[0];

    // เพิ่มรูปร่างอัตโนมัติของชนิดสี่เหลี่ยมผืนผ้าลงในสไลด์
    IShape shp = sld.Shapes.AddAutoShape(ShapeType.Rectangle, 50, 150, 75, 150);
}
```

### การตั้งค่ารูปร่างเติมด้วยรูปภาพ

ต่อไปเราจะเติมรูปร่างด้วยรูปภาพโดยตั้งค่าประเภทการเติม:
```csharp
using Aspose.Slides;
using System.Drawing;

// ตั้งค่าประเภทการเติมของรูปร่างเป็นรูปภาพ
shp.FillFormat.FillType = FillType.Picture;
// กำหนดค่าโหมดเติมภาพเป็นไทล์
shp.FillFormat.PictureFillFormat.PictureFillMode = PictureFillMode.Tile;

// โหลดรูปภาพจากไดเร็กทอรีที่ระบุและตั้งค่าในรูปแบบการเติมของรูปร่าง
IImage img = Images.FromFile("YOUR_DOCUMENT_DIRECTORY/Tulips.jpg");
IPPImage imgx = pres.Images.AddImage(img);
shp.FillFormat.PictureFillFormat.Picture.Image = imgx;
```

### การบันทึกการนำเสนอ

สุดท้ายให้บันทึกการนำเสนอของคุณพร้อมการเปลี่ยนแปลงทั้งหมด:
```csharp
using Aspose.Slides.Export;

// บันทึกการนำเสนอที่แก้ไขแล้วกลับลงดิสก์
pres.Save("YOUR_OUTPUT_DIRECTORY/RectShpPic_out.pptx", SaveFormat.Pptx);
```

ต่อไปนี้คือกรณีการใช้งานจริงสำหรับฟีเจอร์เหล่านี้:
- **การสร้างรายงานอัตโนมัติ:** สร้างสไลด์โดยอัตโนมัติด้วยรูปร่างที่เติมข้อมูล
- **การสร้างเนื้อหาทางการศึกษา:** สร้างเนื้อหาการนำเสนอสำหรับหลักสูตรหรือบทช่วยสอนออนไลน์
- **การผลิตสื่อการตลาด:** สร้างสไลด์โชว์ที่น่าสนใจได้อย่างรวดเร็วและมีประสิทธิภาพ

ความสามารถเหล่านี้ช่วยให้สามารถบูรณาการเข้ากับระบบต่างๆ เช่น แพลตฟอร์มการจัดการเอกสาร โมดูลการเรียนรู้ทางอิเล็กทรอนิกส์ หรือเครื่องมือการตลาดอัตโนมัติได้อย่างราบรื่น

## คู่มือการใช้งาน

ตอนนี้คุณได้ตั้งค่าเรียบร้อยแล้ว มาดูกระบวนการเน้นข้อความโดยใช้รูปแบบ regex กัน

### การเน้นข้อความโดยใช้ Regex

ฟีเจอร์นี้ช่วยให้คุณเน้นข้อความเฉพาะในสไลด์โดยอัตโนมัติตามรูปแบบ regex โดยมีวิธีการทำงานดังนี้:

#### ภาพรวม

เราจะใช้นิพจน์ทั่วไปเพื่อค้นหาคำทั้งหมดที่มีอักขระตั้งแต่ 5 ตัวขึ้นไป และเน้นคำเหล่านั้นภายใน AutoShape

#### การดำเนินการแบบทีละขั้นตอน

1. **เข้าถึงสไลด์และรูปร่าง**
   เข้าถึงสไลด์แรกและรูปร่างแรก โดยถือว่าเป็น AutoShape:
   ```csharp
   using Aspose.Slides;
   
   string dataDir = "YOUR_DOCUMENT_DIRECTORY";
   Presentation presentation = new Presentation(dataDir + "SomePresentation.pptx");
   AutoShape shape = (AutoShape)presentation.Slides[0].Shapes[0];
   ```

2. **กำหนดและใช้รูปแบบ Regex**
   ใช้รูปแบบ regex เพื่อระบุข้อความที่คุณต้องการเน้น:
   ```csharp
   using System.Text.RegularExpressions;
   using System.Drawing;

   // กำหนดรูปแบบ regex สำหรับคำที่มี 5 อักขระหรือมากกว่า
   string pattern = @"\b[^\s]{5,}\b";

   // เน้นข้อความที่ตรงกันในรูปร่าง
   shape.TextFrame.HighlightRegex(pattern);
   ```

3. **บันทึกการนำเสนอ**
   เมื่อคุณเน้นข้อความที่ต้องการแล้ว ให้บันทึกการนำเสนอ:
   ```csharp
   presentation.Save(dataDir + "HighlightedPresentation.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
   ```

#### เคล็ดลับการแก้ไขปัญหา
- ตรวจสอบให้แน่ใจว่ารูปร่างนั้นเป็น AutoShape เพื่อหลีกเลี่ยงข้อผิดพลาดในการแปลง
- ตรวจสอบว่ารูปแบบ regex ตรงกับเกณฑ์ของคุณอย่างถูกต้อง

## การประยุกต์ใช้งานจริง

การเน้นข้อความโดยใช้ regex ไม่ใช่แค่สำหรับการนำเสนอเท่านั้น แต่ยังมีการใช้งานจริงหลายประการ:
1. **เนื้อหาการศึกษา:** เน้นย้ำคำหลักในสื่อการศึกษาเพื่อให้ชัดเจน
2. **การนำเสนอทางธุรกิจ:** เน้นย้ำสถิติหรือจุดข้อมูลที่สำคัญ
3. **การสาธิตผลิตภัณฑ์:** ดึงดูดความสนใจไปที่คุณลักษณะของผลิตภัณฑ์ด้วยการเน้นให้โดดเด่น

## คู่มือการใช้งาน

มาสำรวจวิธีการใช้ฟีเจอร์การจัดเรียงรูปร่างโดยใช้ Aspose.Slides สำหรับ .NET กัน

### การจัดตำแหน่งรูปร่างในสไลด์ (H2)
ฟีเจอร์นี้จะแสดงการจัดตำแหน่งรูปร่างภายในสไลด์ทั้งหมด โดยคุณสามารถทำได้ดังนี้:

#### ขั้นตอนที่ 1: สร้างและเพิ่มรูปร่าง
เพิ่มสี่เหลี่ยมผืนผ้าสองสามอันลงในสไลด์ของคุณเพื่อใช้เป็นพื้นที่ว่าง:

```csharp
ISlide slide = pres.Slides[0];
slide.Shapes.AddAutoShape(ShapeType.Rectangle, 100, 100, 100, 100);
slide.Shapes.AddAutoShape(ShapeType.Rectangle, 200, 200, 100, 100);
slide.Shapes.AddAutoShape(ShapeType.Rectangle, 300, 300, 100, 100);
```

#### ขั้นตอนที่ 2: จัดตำแหน่งรูปร่าง
ใช้ `AlignShapes` วิธีการจัดตำแหน่งรูปร่างเหล่านี้ที่ด้านล่าง:

```csharp
SlideUtil.AlignShapes(ShapesAlignmentType.AlignBottom, true, pres.Slides[0]);
```
**คำอธิบาย:** พารามิเตอร์กำหนดประเภทการจัดตำแหน่ง (`AlignBottom`) ว่าจะรวมข้อความหรือไม่ (`true`) และสไลด์เป้าหมาย

#### ขั้นตอนที่ 3: บันทึกการนำเสนอ
บันทึกการเปลี่ยนแปลงของคุณไปยังไฟล์ใหม่:

```csharp
pres.Save("ShapesAlignment_out.pptx", SaveFormat.Pptx);
```

### จัดเรียงรูปร่างใน GroupShape (H2)
หัวข้อนี้จะแสดงวิธีการจัดตำแหน่งรูปร่างภายในกลุ่มรูปร่างเพื่อให้แน่ใจว่ามีการจัดตำแหน่งที่สอดประสานกัน

#### ขั้นตอนที่ 1: สร้างรูปร่างกลุ่มและเพิ่มรูปร่าง
เพิ่มรูปร่างของคุณไปยังกลุ่มใหม่:

```csharp
ISlide slide = pres.Slides.AddEmptySlide(pres.Slides[0].LayoutSlide);
IGroupShape groupShape = slide.Shapes.AddGroupShape();
groupShape.Shapes.AddAutoShape(ShapeType.Rectangle, 350, 50, 50, 50);
// เพิ่มรูปทรงเพิ่มเติมตามต้องการ
```

#### ขั้นตอนที่ 2: จัดตำแหน่งรูปร่างภายในกลุ่ม
จัดตำแหน่งรูปร่างทั้งหมดเหล่านี้ไปทางซ้ายภายในกลุ่มของพวกเขา:

```csharp
SlideUtil.AlignShapes(ShapesAlignmentType.AlignLeft, false, groupShape);
```

### การจัดตำแหน่งรูปร่างเฉพาะใน GroupShape (H2)
คุณยังสามารถกำหนดเป้าหมายรูปร่างเฉพาะสำหรับการจัดตำแหน่งโดยใช้ดัชนีได้

#### ขั้นตอนที่ 1: ตั้งค่ารูปร่างกลุ่มของคุณ
คล้ายกับส่วนก่อนหน้า สร้างกลุ่มของคุณและเพิ่มรูปร่าง:

```csharp
IGroupShape groupShape = slide.Shapes.AddGroupShape();
groupShape.Shapes.AddAutoShape(ShapeType.Rectangle, 350, 50, 50, 50);
// รูปทรงเพิ่มเติม...
```

#### ขั้นตอนที่ 2: จัดตำแหน่งรูปร่างเฉพาะ
ใช้ดัชนีเพื่อระบุว่าจะจัดตำแหน่งรูปร่างใด:

```csharp
SlideUtil.AlignShapes(ShapesAlignmentType.AlignLeft, false, groupShape, new int[] { 0, 2 });
```
**คำอธิบาย:** วิธีนี้จะจัดตำแหน่งเฉพาะรูปร่างแรกและสามภายในกลุ่มเท่านั้น

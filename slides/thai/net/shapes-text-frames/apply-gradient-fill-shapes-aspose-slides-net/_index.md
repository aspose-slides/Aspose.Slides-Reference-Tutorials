### คู่มือการใช้งาน

#### ใช้การเติมแบบไล่ระดับกับรูปร่าง

ฟีเจอร์นี้ช่วยให้คุณเพิ่มความน่าสนใจให้กับรูปร่างในสไลด์ PowerPoint ของคุณได้โดยการเพิ่มการเติมแบบไล่ระดับ มาลองดูวิธีการใช้งานฟีเจอร์นี้กัน:

##### ขั้นตอนที่ 1: สร้างรูปทรงวงรี

```csharp
// โหลดหรือสร้างการนำเสนอโดยใช้ (Presentation pres = new Presentation())
{
    // การเข้าถึงสไลด์แรก
    ISlide sld = pres.Slides[0];
    
    // เพิ่มรูปร่างอัตโนมัติของชนิดวงรี
    IAutoShape ashp = sld.Shapes.AddAutoShape(ShapeType.Ellipse, 50, 150, 150, 50);
}
```

ในขั้นตอนนี้ เราจะสร้างวงรีบนสไลด์แรก พารามิเตอร์จะกำหนดตำแหน่งและขนาดของวงรี

##### ขั้นตอนที่ 2: ใช้การเติมแบบไล่เฉดสี

```csharp
// ตั้งค่าประเภทการเติมเป็นแบบไล่ระดับสี
ashp.FillFormat.FillType = FillType.Gradient;

// กำหนดสีและรูปแบบการไล่ระดับสี
ashp.FillFormat.GradientFormat.StartColor = Color.Red;
ashp.FillFormat.GradientFormat.EndColor = Color.Blue;
ashp.FillFormat.GradientFormat.TileFlip = TileFlip.FlipBoth;
```

ที่นี่ เราตั้งค่าวงรีเพื่อให้มีการเติมแบบไล่ระดับจากสีแดงไปเป็นสีน้ำเงิน

##### ขั้นตอนที่ 3: บันทึกการนำเสนอ

```csharp
// กำหนดเส้นทางเอาท์พุต
string dataDir = "YOUR_DOCUMENT_DIRECTORY";

// ตรวจสอบให้แน่ใจว่ามีไดเร็กทอรีอยู่
if (!Directory.Exists(dataDir))
{
    Directory.CreateDirectory(dataDir);
}

// บันทึกการนำเสนอ
pres.Save(Path.Combine(dataDir, "GradientEllipse.pptx"), SaveFormat.Pptx);
```

สไนปเป็ตนี้จะช่วยให้แน่ใจว่าการนำเสนอถูกบันทึกไว้ในไดเร็กทอรีที่คุณระบุ

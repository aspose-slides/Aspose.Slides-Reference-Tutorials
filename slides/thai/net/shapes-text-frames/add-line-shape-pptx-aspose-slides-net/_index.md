## คู่มือการใช้งาน

### เพิ่มรูปร่างเส้นลงในสไลด์
หัวข้อนี้จะแนะนำคุณเกี่ยวกับการเพิ่มรูปร่างเส้นลงในสไลด์ PowerPoint ของคุณโดยใช้ Aspose.Slides สำหรับ .NET

#### ภาพรวม
การเพิ่มเส้นเป็นเรื่องง่ายด้วย Aspose.Slides ฟีเจอร์นี้ช่วยในการแบ่งส่วนหรือเน้นเนื้อหาภายในสไลด์

#### ขั้นตอนการดำเนินการ:

##### ขั้นตอนที่ 1: สร้างอินสแตนซ์คลาสการนำเสนอ
เริ่มต้นด้วยการสร้างอินสแตนซ์ของ `Presentation` คลาสที่แสดงไฟล์ PowerPoint ของคุณ

```csharp
using (Presentation pres = new Presentation())
{
    // โค้ดสำหรับจัดการการนำเสนออยู่ที่นี่
}
```

##### ขั้นตอนที่ 2: เข้าถึงสไลด์แรก
เข้าถึงสไลด์แรกในงานนำเสนอของคุณ นี่คือจุดที่เราจะเพิ่มรูปร่างเส้น

```csharp
ISlide sld = pres.Slides[0];
```

##### ขั้นตอนที่ 3: เพิ่มรูปร่างเส้น
ใช้ `AddAutoShape` วิธีการเพิ่มเส้นที่ตำแหน่งที่ระบุพร้อมทั้งมีมิติที่กำหนดไว้

```csharp
sld.Shapes.AddAutoShape(ShapeType.Line, 50, 150, 300, 0);
```
- **พารามิเตอร์**-
  - `ShapeType.Line`: ระบุว่าเราจะเพิ่มรูปร่างเส้น
  - `(50, 150)`:ตำแหน่งเริ่มต้นบนสไลด์ (พิกัด x, y)
  - `300`: ความกว้างของเส้น.
  - `0`: ความสูงของเส้น (ตั้งเป็นศูนย์สำหรับความสูงหนึ่งพิกเซล)

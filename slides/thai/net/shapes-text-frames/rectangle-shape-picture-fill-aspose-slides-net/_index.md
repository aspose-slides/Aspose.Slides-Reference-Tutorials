## คู่มือการใช้งาน: เพิ่มรูปสี่เหลี่ยมผืนผ้าด้วยการเติมรูปภาพ
ตอนนี้สภาพแวดล้อมของเราพร้อมแล้ว มาสร้างฟีเจอร์ในการเพิ่มรูปสี่เหลี่ยมผืนผ้าที่เต็มไปด้วยรูปภาพกัน

### ภาพรวมของคุณสมบัติ
ฟีเจอร์นี้สาธิตวิธีการสร้างรูปสี่เหลี่ยมผืนผ้าบนสไลด์และเติมรูปภาพโดยใช้ Aspose.Slides เทคนิคนี้ใช้เพื่อปรับปรุงสไลด์ของคุณได้โดยการเพิ่มโลโก้ พื้นหลัง หรือองค์ประกอบกราฟิกใดๆ ที่ทำให้การนำเสนอของคุณน่าสนใจยิ่งขึ้น

### การดำเนินการแบบทีละขั้นตอน
#### 1. เริ่มต้นวัตถุการนำเสนอ
เริ่มต้นด้วยการสร้างวัตถุการนำเสนอใหม่ ซึ่งจะทำหน้าที่เป็นเอกสารการทำงานของเรา โดยเราจะเพิ่มรูปร่างและองค์ประกอบอื่นๆ
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // ตั้งค่าเส้นทางไดเร็กทอรีเอกสารของคุณ
total slides count: pres.Slides.Count;
using (Presentation pres = new Presentation())
{
    ISlide firstSlide = pres.Slides[0]; // เข้าถึงสไลด์แรก

    // โหลดรูปภาพเพื่อใช้เป็นวัสดุเติม
    IPPImage ppImage;
    using (IImage newImage = Aspose.Slides.Images.FromFile(Path.Combine(dataDir, "image.png")))
        ppImage = pres.Images.AddImage(newImage); // เพิ่มรูปภาพลงในคอลเลคชันรูปภาพของงานนำเสนอ

    // เพิ่มรูปทรงสี่เหลี่ยมผืนผ้าที่มีขนาดตามที่กำหนด
    var newShape = firstSlide.Shapes.AddAutoShape(ShapeType.Rectangle, 0, 0, 350, 350);

    // ตั้งค่าประเภทการเติมของรูปร่างเป็นรูปภาพ
    newShape.FillFormat.FillType = FillType.Picture;
    IPictureFillFormat pictureFillFormat = newShape.FillFormat.PictureFillFormat;
    pictureFillFormat.Picture.Image = ppImage; // กำหนดรูปภาพที่โหลดให้เป็นสีเติมสำหรับสี่เหลี่ยมผืนผ้า

    // บันทึกการนำเสนอ
    pres.Save(Path.Combine("YOUR_OUTPUT_DIRECTORY", "RectangleWithPictureFill.pptx"), SaveFormat.Pptx);
}
```
#### คำอธิบายขั้นตอนสำคัญ:
- **กำลังโหลดรูปภาพ**: เดอะ `FromFile` วิธีการโหลดรูปภาพจากไดเร็กทอรีที่คุณระบุ ซึ่งจะถูกเพิ่มลงในคอลเล็กชันรูปภาพของการนำเสนอ
  
- **การเพิ่มรูปสี่เหลี่ยมผืนผ้า**: เราใช้ `AddAutoShape` กับ `ShapeType.Rectangle` และกำหนดขนาดของมัน ซึ่งจะทำให้สไลด์กลายเป็นรูปสี่เหลี่ยมผืนผ้า

- **การตั้งค่าการเติมรูปภาพ**: โดยการมอบหมาย `FillType.Picture` เป็นรูปแบบการเติมของรูปร่าง เราแปลงรูปสี่เหลี่ยมผืนผ้าให้กลายเป็นคอนเทนเนอร์รูปภาพ จากนั้นรูปภาพที่โหลดจะถูกตั้งค่าเป็นการเติมโดยใช้ `Picture.Image` คุณสมบัติ.

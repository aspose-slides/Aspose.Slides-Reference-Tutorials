## คู่มือการใช้งาน

มาแบ่งกระบวนการออกเป็นขั้นตอนที่ชัดเจนในการทำซ้ำรูปร่างและระบุกล่องข้อความ

### คุณสมบัติ: ทำซ้ำรูปร่างการนำเสนอ

ฟีเจอร์นี้มุ่งเน้นที่การวนซ้ำผ่านรูปร่างทั้งหมดที่มีอยู่ในสไลด์ โดยตรวจสอบว่าแต่ละรูปร่างเป็นกล่องข้อความหรือไม่ คุณสามารถนำไปใช้งานฟีเจอร์นี้ได้ดังนี้:

#### ขั้นตอนที่ 1: โหลดงานนำเสนอของคุณ

ก่อนอื่น ให้แน่ใจว่าเส้นทางไฟล์การนำเสนอของคุณถูกตั้งค่าอย่างถูกต้อง:

```csharp
string presentationPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "CheckTextShapes.pptx");
```

เปิดการนำเสนอโดยใช้ Aspose.Slides:

```csharp
using (Presentation presentation = new Presentation(presentationPath))
{
    // โค้ดสำหรับการวนซ้ำผ่านรูปร่างจะอยู่ที่นี่
}
```

#### ขั้นตอนที่ 2: ทำซ้ำรูปร่างต่างๆ

นำทางผ่านแต่ละรูปร่างในสไลด์ที่เฉพาะเจาะจง ในตัวอย่างนี้ เราจะดูที่สไลด์แรก:

```csharp
foreach (IShape shape in presentation.Slides[0].Shapes)
{
    // ตรวจสอบว่ารูปร่างนั้นเป็น AutoShape หรือไม่ และระบุด้วยว่าเป็นกล่องข้อความหรือไม่
}
```

#### ขั้นตอนที่ 3: ระบุกล่องข้อความ

ตรวจสอบว่าแต่ละรูปร่างเป็น `AutoShape` แล้วตรวจสอบว่ามีข้อความหรือไม่:

```csharp
if (shape is AutoShape autoShape)
{
    bool isTextBox = autoShape.IsTextBox;
    // ใช้ 'isTextBox' เพื่อกำหนดว่ารูปร่างเป็นกล่องข้อความหรือไม่
}
```

### เคล็ดลับการแก้ไขปัญหา

- ตรวจสอบให้แน่ใจว่าเส้นทางไฟล์การนำเสนอของคุณถูกต้องและสามารถเข้าถึงได้
- ตรวจสอบว่า Aspose.Slides มีการอ้างอิงอย่างถูกต้องในโครงการของคุณ
- หากคุณพบข้อผิดพลาด ให้ตรวจสอบความเข้ากันได้ของเวอร์ชันระหว่าง Aspose.Slides และ .NET

1. **การสร้างรายงานอัตโนมัติ**:ดึงข้อความจากงานนำเสนอโดยอัตโนมัติเพื่อสร้างรายงานหรือสรุป

## คู่มือการใช้งาน

ในส่วนนี้จะแนะนำคุณเกี่ยวกับการเปลี่ยนสีเส้นผู้นำโดยใช้ Aspose.Slides

### การเข้าถึงการนำเสนอ PowerPoint

โหลดการนำเสนอ PowerPoint ที่คุณต้องการเปลี่ยนสีเส้นผู้นำ

#### โหลดงานนำเสนอ

```csharp
string presentationName = "YOUR_DOCUMENT_DIRECTORY/LeaderLinesColor.pptx";
using (Presentation pres = new Presentation(presentationName))
{
    // ขั้นตอนต่อไปจะตามมาที่นี่...
}
```

### การเข้าถึงข้อมูลแผนภูมิ

ระบุตำแหน่งและเข้าถึงข้อมูลแผนภูมิที่เส้นผู้นำต้องมีการปรับสี

#### รับแผนภูมิสไลด์แรก

```csharp
IChart chart = (IChart)pres.Slides[0].Shapes[0];
```

### การปรับเปลี่ยนสีเส้นผู้นำ

ตอนนี้ เปลี่ยนสีของเส้นผู้นำในซีรีย์ที่คุณระบุ

#### เปลี่ยนเส้นผู้นำเป็นสีแดง

```csharp
IChartSeriesCollection series = chart.ChartData.Series;
IDataLabelCollection labels = series[0].Labels;
labels.LeaderLinesFormat.Line.FillFormat.SolidFillColor.Color = Color.FromArgb(255, 255, 0, 0);
```

### การบันทึกการนำเสนอ

สุดท้ายให้บันทึกการเปลี่ยนแปลงของคุณลงในไฟล์ใหม่

#### บันทึกการนำเสนอที่ปรับเปลี่ยน

```csharp
string outPath = "YOUR_OUTPUT_DIRECTORY/LeaderLinesColor-out.pptx";
pres.Save(outPath, SaveFormat.Pptx);
```

## คู่มือการใช้งาน

เมื่อจัดเตรียมทุกอย่างเสร็จเรียบร้อยแล้ว เรามาใช้งานฟีเจอร์รูปร่างแบบร่างกัน

### การเพิ่มและปรับเปลี่ยนรูปทรง

#### ภาพรวม

ในส่วนนี้เราจะเพิ่ม AutoShape ของชนิดสี่เหลี่ยมผืนผ้าลงในสไลด์ และกำหนดค่าคุณสมบัติเพื่อสร้างเอฟเฟ็กต์แบบร่าง

**การเพิ่มรูปทรงสี่เหลี่ยมผืนผ้า**

เริ่มต้นด้วยการสร้างอินสแตนซ์การนำเสนอใหม่และเพิ่มรูปร่างสี่เหลี่ยมผืนผ้า:
```csharp
using Aspose.Slides;
using Aspose.Slides.Export;

string outPptxFile = Path.Combine("YOUR_OUTPUT_DIRECTORY", "SketchedShapes_out.pptx");
string outPngFile = Path.Combine("YOUR_OUTPUT_DIRECTORY", "SketchedShapes_out.png");

using (Presentation pres = new Presentation())
{
    // เพิ่ม AutoShape ของชนิดสี่เหลี่ยมผืนผ้าในสไลด์แรก
    IAutoShape shape = pres.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 20, 20, 300, 150);
}
```

#### การตั้งค่ารูปแบบการเติม

หากต้องการให้มีลักษณะเหมือนร่าง ให้ลบสีเติมออกจากรูปร่าง:
```csharp
shape.FillFormat.FillType = FillType.NoFill;
```

### การใช้เอฟเฟ็กต์ภาพร่างกับรูปทรง

#### ภาพรวม

จากนั้นแปลงรูปสี่เหลี่ยมผืนผ้าให้เป็นภาพร่างแบบวาดด้วยมือ

**การแปลงรูปร่างเป็นภาพร่าง**

ใช้ `SketchFormat` คุณสมบัติที่จะใช้เอฟเฟ็กต์ลายเส้น:
```csharp
// แปลงรูปร่างเป็นภาพร่างแบบฟรีแฮนด์ (Scribble)
shape.LineFormat.SketchFormat.SketchType = LineSketchType.Scribble;
```

### การบันทึกการนำเสนอและรูปภาพ

สุดท้าย ให้บันทึกงานของคุณเป็นทั้งไฟล์งานนำเสนอและรูปภาพ

**บันทึกเป็น PPTX**
```csharp
// บันทึกการนำเสนอไปยังไฟล์ PPTX
pres.Save(outPptxFile, SaveFormat.Pptx);
```

**บันทึกเป็นรูปภาพ PNG**
```csharp
// บันทึกสไลด์เป็นไฟล์ภาพในรูปแบบ PNG
pres.Slides[0].GetThumbnail(4/3f, 4/3f).Save(outPngFile, System.Drawing.Imaging.ImageFormat.Png);
```

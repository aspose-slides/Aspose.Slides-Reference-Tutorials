# วิธีตั้งค่าภาษาใน PowerPoint Shapes โดยใช้ Aspose.Slides สำหรับ .NET

ในโลกของการนำเสนอแบบดิจิทัล การทำให้แน่ใจว่าเนื้อหาของคุณสามารถเข้าถึงได้และมีรูปแบบที่ถูกต้องในภาษาต่างๆ อาจเป็นเรื่องท้าทาย ด้วย Aspose.Slides สำหรับ .NET คุณสามารถตั้งค่าแอตทริบิวต์ภาษาสำหรับข้อความภายในรูปร่างในสไลด์ PowerPoint ได้อย่างง่ายดาย คุณสมบัตินี้มีประโยชน์อย่างยิ่งเมื่อเตรียมเอกสารหลายภาษาหรือทำให้แน่ใจว่าการสื่อสารทั่วโลกมีความสอดคล้องกัน

**สิ่งที่คุณจะได้เรียนรู้:**
- การเพิ่มรูปร่างอัตโนมัติและแทรกข้อความลงไป
- การตั้งค่า ID ภาษาสำหรับส่วนข้อความโดยใช้ Aspose.Slides
- บันทึกการนำเสนอด้วยการกำหนดค่าแบบกำหนดเอง

มาเจาะลึกกันว่าคุณสามารถนำฟีเจอร์นี้ไปใช้ได้อย่างราบรื่นอย่างไร

## การตั้งค่า Aspose.Slides สำหรับ .NET

ในการเริ่มต้น คุณต้องติดตั้งไลบรารี Aspose.Slides คุณสามารถทำได้โดยใช้หนึ่งในวิธีต่อไปนี้:

### ภาพรวมคุณลักษณะ: การตั้งค่าภาษาข้อความรูปร่าง

ฟีเจอร์นี้ช่วยให้คุณระบุภาษาของข้อความในรูปทรงของ PowerPoint ได้ การตั้งค่า ID ภาษาจะช่วยให้คุณมั่นใจได้ว่าการตรวจสอบการสะกดคำและฟีเจอร์เฉพาะภาษาอื่นๆ จะถูกนำไปใช้ได้อย่างถูกต้อง

#### ขั้นตอนที่ 1: เริ่มต้นการนำเสนอ

เริ่มต้นด้วยการสร้างอินสแตนซ์ของ `Presentation` ระดับ.

```csharp
using (Presentation pres = new Presentation())
{
    // รหัสของคุณที่นี่
}
```

นี่เป็นการเริ่มต้นวัตถุการนำเสนอ PowerPoint ใหม่ซึ่งเราจะจัดการ

#### ขั้นตอนที่ 2: เพิ่มรูปร่างอัตโนมัติและกรอบข้อความ

เพิ่มรูปทรงสี่เหลี่ยมผืนผ้าลงในสไลด์ของคุณและแทรกข้อความลงไป:

```csharp
IAutoShape shape = pres.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 50, 50, 200, 50);
shape.AddTextFrame("Text to apply spellcheck language");
```

ที่นี่, `AddAutoShape` เพิ่มรูปสี่เหลี่ยมผืนผ้าลงในสไลด์แรก พารามิเตอร์จะกำหนดตำแหน่งและขนาดของรูปสี่เหลี่ยมผืนผ้า

#### ขั้นตอนที่ 3: ตั้งค่ารหัสภาษา

ตั้งค่าภาษาสำหรับส่วนข้อความภายในรูปร่าง:

```csharp

```

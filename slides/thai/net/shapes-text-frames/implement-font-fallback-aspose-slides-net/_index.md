# วิธีตั้งค่ากฎการสำรองแบบอักษรใน Aspose.Slides สำหรับ .NET: คู่มือฉบับสมบูรณ์

## การแนะนำ

การสร้างงานนำเสนอด้วย Aspose.Slides สำหรับ .NET บางครั้งต้องจัดการกับอักขระที่แบบอักษรเฉพาะบางแบบรองรับไม่ได้ เช่น อักษรทมิฬหรืออักษรฮิรางานะภาษาญี่ปุ่น การกำหนดกฎสำรองแบบอักษรถือเป็นสิ่งสำคัญเพื่อให้แน่ใจว่างานนำเสนอของคุณแสดงข้อความได้อย่างถูกต้องในภาษาและสัญลักษณ์ต่างๆ

ในบทช่วยสอนนี้ เราจะแนะนำคุณเกี่ยวกับการใช้กฎสำรองแบบอักษรโดยใช้ Aspose.Slides สำหรับ .NET ตั้งแต่การติดตั้งไปจนถึงการใช้งานจริง คู่มือนี้จะช่วยให้มั่นใจว่าการนำเสนอของคุณจะมีความสอดคล้องกันทางภาพไม่ว่าเนื้อหาจะเป็นอย่างไรก็ตาม

**สิ่งที่คุณจะได้เรียนรู้:**
- กำหนดช่วง Unicode สำหรับสคริปต์ที่แตกต่างกัน
- ตั้งค่าแบบอักษรสำรองสำหรับอักขระที่ไม่ได้รับการสนับสนุน
- ใช้แบบอักษรสำรองในสถานการณ์การนำเสนอในโลกแห่งความเป็นจริง
- เคล็ดลับสำหรับการเพิ่มประสิทธิภาพและการบูรณาการกับระบบอื่นๆ

มาเริ่มต้นด้วยการทบทวนข้อกำหนดเบื้องต้นกันก่อน

## คู่มือการใช้งาน

ในหัวข้อนี้ เราจะสรุปขั้นตอนการตั้งค่ากฎการสำรองแบบอักษรเป็นขั้นตอนที่ชัดเจน

### 1. กำหนดช่วง Unicode และแบบอักษรสำรอง

แต่ละสคริปต์หรือชุดสัญลักษณ์ต้องมีช่วง Unicode เฉพาะและแบบอักษรสำรองที่สอดคล้องกันเพื่อให้แน่ใจว่ามีการแสดงผลอย่างถูกต้อง

#### อักษรทมิฬ

- **ภาพรวม**:ใช้ "Vijaya" สำหรับอักขระทมิฬเมื่อแบบอักษรหลักขาดการรองรับ

**ขั้นตอนการดำเนินการ:**

##### ขั้นตอนที่ 1: กำหนดช่วง Unicode
```csharp
uint startUnicodeIndexTamil = 0x0B80; // จุดเริ่มต้นของยุคทมิฬ
uint endUnicodeIndexTamil = 0x0BFF;   // ปลายยุคทมิฬ
```
สไนปเป็ตนี้จะกำหนดช่วง Unicode สำหรับอักขระทมิฬ

##### ขั้นตอนที่ 2: สร้างกฎสำรอง
```csharp
IFontFallBackRule tamilFallbackRule = new FontFallBackRule(startUnicodeIndexTamil, endUnicodeIndexTamil, "Vijaya");
```
ที่นี่ เราสร้างกฎสำรองโดยใช้ "Vijaya" เป็นแบบอักษรทางเลือก

#### ฮิรางานะภาษาญี่ปุ่น

- **ภาพรวม**:ใช้ "MS Mincho" หรือ "MS Gothic" สำหรับอักขระฮิรางานะที่ไม่รองรับ

**ขั้นตอนการดำเนินการ:**

##### ขั้นตอนที่ 1: กำหนดช่วง Unicode
```csharp
uint startUnicodeIndexHiragana = 0x3040; // จุดเริ่มต้นของช่วงฮิรางานะ
uint endUnicodeIndexHiragana = 0x309F;   // ปลายช่วงฮิรางานะ
```
ตัวอย่างนี้จะกำหนดขอบเขต Unicode สำหรับอักษรฮิรางานะ

##### ขั้นตอนที่ 2: สร้างกฎสำรอง
```csharp
IFontFallBackRule hiraganaFallbackRule = new FontFallBackRule(startUnicodeIndexHiragana, endUnicodeIndexHiragana, "MS Mincho, MS Gothic");
```
กฎนี้ระบุแบบอักษรสำรองหลายแบบสำหรับอักขระฮิรางานะ

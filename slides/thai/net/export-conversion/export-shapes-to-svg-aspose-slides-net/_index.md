## คู่มือการใช้งาน

เน้นการส่งออกรูปร่างไปยังรูปแบบ SVG ด้วยคำแนะนำโดยละเอียดนี้:

### ส่งออกรูปร่างไปยัง SVG

#### ภาพรวม
ส่งออกรูปร่างจากสไลด์ PowerPoint ใดๆ ไปยังไฟล์ SVG ซึ่งมีประโยชน์สำหรับการผสานกราฟิกแบบเวกเตอร์ในแอปพลิเคชันเว็บหรือระบบซอฟต์แวร์ที่ต้องการรูปแบบที่ปรับขนาดได้

#### คำแนะนำทีละขั้นตอน
**1. ตั้งค่าเส้นทางสำหรับไฟล์อินพุตและเอาต์พุต**
กำหนดไดเรกทอรีสำหรับไฟล์อินพุตและเอาต์พุต:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // ไดเรกทอรีที่มีไฟล์ PowerPoint
string outSvgFileName = "YOUR_OUTPUT_DIRECTORY/SingleShape.svg"; // เส้นทางเอาท์พุตไฟล์ SVG
```

**2. โหลดงานนำเสนอของคุณ**
โหลดการนำเสนอโดยใช้ Aspose.Slides:

```csharp
using (Presentation pres = new Presentation(dataDir + "/TestExportShapeToSvg.pptx"))
{
    // เข้าถึงสไลด์แรกและรูปร่างแรก
    var slide = pres.Slides[0];
    var shape = slide.Shapes[0];

    // สร้าง FileStream สำหรับเอาท์พุตไฟล์ SVG
    using (Stream stream = new FileStream(outSvgFileName, FileMode.Create, FileAccess.Write))
    {
        // ส่งออกรูปร่างเป็นรูปแบบ SVG
        shape.WriteAsSvg(stream);
    }
}
```

**คำอธิบาย:**
- `dataDir`:ไดเร็กทอรีที่มีไฟล์ PowerPoint ของคุณ
- `outSvgFileName`: เส้นทางที่ SVG ที่ส่งออกจะถูกบันทึก
- **`Presentation` วัตถุ**: หมายถึงเอกสาร PowerPoint
- **`Slide.Shapes[0]`**:เข้าถึงรูปร่างแรกของสไลด์แรกเพื่อส่งออก

### เคล็ดลับการแก้ไขปัญหา
- ตรวจสอบให้แน่ใจว่าเส้นทางไฟล์อินพุตของคุณถูกต้องและสามารถเข้าถึงได้
- ตรวจสอบสิทธิ์ของไฟล์เพื่อยืนยันการเข้าถึงการเขียนไปยังไดเร็กทอรีเอาต์พุต
- ตรวจสอบว่าไฟล์ PowerPoint ไม่เสียหายโดยการเปิดใน Microsoft PowerPoint

# วิธีการส่งออกการนำเสนอ PowerPoint ไปยัง PDF ด้วยข้อมูล OLE ที่ฝังไว้โดยใช้ Aspose.Slides สำหรับ .NET

## การแนะนำ

คุณจำเป็นต้องแบ่งปันงานนำเสนอ PowerPoint แบบโต้ตอบที่สมบูรณ์ในรูปแบบ PDF พร้อมรักษาฟังก์ชันการใช้งานไว้หรือไม่ ด้วย **Aspose.Slides สำหรับ .NET**การส่งออกงานนำเสนอที่มีข้อมูลการเชื่อมโยงและฝังวัตถุ (OLE) เป็นเรื่องง่าย บทช่วยสอนนี้จะแนะนำคุณเกี่ยวกับการใช้งานฟีเจอร์นี้อย่างง่ายดาย เพื่อเพิ่มความสามารถในการจัดการเอกสารของคุณ

**ประเด็นสำคัญ:**
- เชี่ยวชาญกระบวนการส่งออกงานนำเสนอ PowerPoint เป็น PDF
- ทำความเข้าใจว่าข้อมูล OLE รักษาการโต้ตอบภายในเอกสารได้อย่างไร
- ค้นพบว่า Aspose.Slides สำหรับ .NET ทำให้การดำเนินการที่ซับซ้อนง่ายขึ้นอย่างไร
- สำรวจการใช้งานจริงและการเพิ่มประสิทธิภาพการทำงาน

ให้เราดำเนินการตามข้อกำหนดเบื้องต้นที่จำเป็นก่อนจะดูรายละเอียดคู่มือการใช้งาน

### ส่งออก PPT เป็น PDF ด้วยข้อมูล OLE ที่ฝังไว้

**ภาพรวม:**
คุณลักษณะนี้ช่วยให้คุณสามารถส่งออกงานนำเสนอเป็นรูปแบบ PDF โดยรักษาวัตถุ OLE ที่ฝังไว้และรักษาการทำงานและรูปลักษณ์ของวัตถุเหล่านั้นไว้

#### ขั้นตอนที่ 1: เริ่มต้นวัตถุการนำเสนอ

```csharp
// โหลดไฟล์ PowerPoint ของคุณโดยใช้ Aspose.Slides
Presentation presentation = new Presentation(dataDir + "YourPresentation.pptx");
```
- **คำอธิบาย:** ที่นี่เราสร้าง `Presentation` วัตถุโดยโหลดไฟล์ PPTX จากไดเร็กทอรีที่ระบุ

#### ขั้นตอนที่ 2: กำหนดค่าตัวเลือก PDF

```csharp
// ตั้งค่าตัวเลือก PDF เพื่อรวมวัตถุ OLE
PdfOptions pdfOptions = new PdfOptions();
pdfOptions.EmbedFullFonts = true; // รับประกันว่าแบบอักษรถูกฝังอยู่ใน PDF
```
- **พารามิเตอร์:** `EmbedFullFonts` ทำให้แน่ใจว่ามีการรวมแบบอักษรทั้งหมดไว้และรักษาลักษณะที่ปรากฏของข้อความ

## คู่มือการใช้งาน
มาเจาะลึกการแปลงงานนำเสนอ PowerPoint โดยคงแบบอักษรไว้โดยใช้ Aspose.Slides สำหรับ .NET กัน เราจะอธิบายทีละขั้นตอน:

### ภาพรวมคุณสมบัติ
คุณลักษณะนี้ช่วยให้สามารถแปลงไฟล์ PPTX เป็นเอกสาร HTML โดยยังคงรูปแบบอักษรดั้งเดิมตามที่ปรากฏในงานนำเสนอ

#### ขั้นตอนที่ 1: โหลดงานนำเสนอของคุณ
เริ่มต้นด้วยการโหลดไฟล์ PowerPoint ของคุณลงใน `Presentation` วัตถุ นี่เป็นสิ่งสำคัญสำหรับการเข้าถึงและการจัดการสไลด์

```csharp
using Aspose.Slides;
using Aspose.Slides.Export;

string dataDir = @"YOUR_DOCUMENT_DIRECTORY";
using (Presentation pres = new Presentation(dataDir + "input.pptx"))
{
    // ดำเนินการเพิ่มเติมที่นี่
}
```

**คำอธิบาย:** เราเริ่มต้นด้วยการสร้าง `Presentation` วัตถุที่ช่วยให้เราโต้ตอบกับสไลด์ในไฟล์ PowerPoint ของคุณได้

#### ขั้นตอนที่ 2: กำหนดค่าการตั้งค่าแบบอักษร
คุณสามารถระบุแบบอักษรที่คุณต้องการไม่ให้ฝังไว้ใน HTML ได้ ซึ่งจะช่วยเพิ่มประสิทธิภาพเวลาในการโหลดและลดขนาดไฟล์

```csharp
string[] fontNameExcludeList = { "Calibri" };
```

**คำอธิบาย:** การ `fontNameExcludeList` อาร์เรย์จะกำหนดแบบอักษรที่ไม่ควรฝังลงในเอกสาร HTML ขั้นสุดท้าย ช่วยให้จัดการการใช้ทรัพยากรได้อย่างมีประสิทธิภาพ

#### ขั้นตอนที่ 3: แปลงเป็น HTML
ขั้นตอนต่อไปคือการแปลงสไลด์การนำเสนอของคุณเป็นรูปแบบ HTML คุณสามารถปรับแต่งกระบวนการนี้เพิ่มเติมได้โดยระบุการตั้งค่าเพิ่มเติมหากจำเป็น

```csharp
pres.Save(outputDir + "output.html", SaveFormat.Html5);
```

**คำอธิบาย:** การ `Save` วิธีการส่งออกการนำเสนอเป็นเอกสาร HTML ด้วย `Html5` เพื่อให้แน่ใจว่ามีความเข้ากันได้ทั่วทั้งเว็บเบราว์เซอร์สมัยใหม่

### เคล็ดลับการแก้ไขปัญหา:
- รับรองเส้นทางใน `dataDir` และ `outputDir` ถูกต้องครับ.
- ตรวจสอบว่ามีแบบอักษรที่ถูกยกเว้นอยู่ในอุปกรณ์เป้าหมายหรือไม่เพื่อหลีกเลี่ยงรูปแบบที่ขาดหายไป

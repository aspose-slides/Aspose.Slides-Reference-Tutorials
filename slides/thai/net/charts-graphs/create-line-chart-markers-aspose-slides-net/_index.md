# วิธีการสร้างแผนภูมิเส้นด้วยเครื่องหมายใน C# โดยใช้ Aspose.Slides สำหรับ .NET

## การแนะนำ
การสร้างแผนภูมิเส้นที่มีภาพดึงดูดใจและให้ข้อมูลถือเป็นสิ่งสำคัญสำหรับการนำเสนอข้อมูลอย่างมีประสิทธิภาพใน C# **Aspose.Slides สำหรับ .NET** ทำให้กระบวนการเพิ่มแผนภูมิที่ดูเป็นมืออาชีพง่ายขึ้น รวมถึงแผนภูมิที่มีเครื่องหมายด้วย บทช่วยสอนนี้จะแนะนำคุณตลอดกระบวนการสร้างแผนภูมิเส้นที่มีเครื่องหมายเริ่มต้นโดยใช้ Aspose.Slides สำหรับ .NET

ในบทช่วยสอนนี้ คุณจะได้เรียนรู้:
- การตั้งค่าสภาพแวดล้อมของคุณเพื่อใช้ Aspose.Slides สำหรับ .NET
- การสร้างและปรับแต่งการนำเสนอด้วยแผนภูมิเส้นที่มีเครื่องหมาย
- การกำหนดค่าคุณสมบัติของแผนภูมิ เช่น หมวดหมู่ ชุด และจุดข้อมูล
- บันทึกไฟล์การนำเสนอขั้นสุดท้าย

เริ่มต้นด้วยการทบทวนข้อกำหนดเบื้องต้นที่จำเป็นก่อนนำโซลูชันของเราไปใช้

## คู่มือการใช้งาน
### การสร้างแผนภูมิเส้นด้วยเครื่องหมาย
ในส่วนนี้ คุณจะเรียนรู้แต่ละขั้นตอนที่จำเป็นในการสร้างและกำหนดค่าแผนภูมิเส้นด้วยเครื่องหมายเริ่มต้นในการนำเสนอของคุณโดยใช้ Aspose.Slides สำหรับ .NET

#### ขั้นตอนที่ 1: สร้างวัตถุการนำเสนอ
เริ่มต้นด้วยการสร้างอินสแตนซ์ของ `Presentation` ระดับ:
```csharp
using (Presentation pres = new Presentation())
{
    ISlide slide = pres.Slides[0];
```
ที่นี่เราเข้าถึงสไลด์แรกในงานนำเสนอที่สร้างขึ้นใหม่

#### ขั้นตอนที่ 2: เพิ่มแผนภูมิเส้นพร้อมเครื่องหมาย
ขั้นตอนต่อไป เพิ่มแผนภูมิเส้นพร้อมเครื่องหมายลงในสไลด์ของคุณ:
```csharp
IChart chart = slide.Shapes.AddChart(ChartType.LineWithMarkers, 10, 10, 400, 400);
```
โค้ดนี้จะเพิ่มแผนภูมิชนิดใหม่ `LineWithMarkers` ที่พิกัด `(10, 10)` มีมิติ `400x400`-

#### ขั้นตอนที่ 3: ล้างซีรีย์และหมวดหมู่ที่มีอยู่
ก่อนที่จะเพิ่มข้อมูล ให้ล้างชุดข้อมูลหรือหมวดหมู่ที่มีอยู่:
```csharp
chart.ChartData.Series.Clear();
chart.ChartData.Categories.Clear();
```
วิธีนี้ช่วยให้แน่ใจว่าแผนภูมิของเราเริ่มต้นด้วยความสะอาด

#### ขั้นตอนที่ 4: กำหนดค่าสมุดงานข้อมูลแผนภูมิ
เข้าถึง `ChartDataWorkbook` ในการจัดการข้อมูลแผนภูมิของคุณ:
```csharp
IChartDataWorkbook fact = chart.ChartData.ChartDataWorkbook;
```
วัตถุนี้มีความสำคัญต่อการจัดการเซลล์ที่ประกอบด้วยข้อมูลชุดและหมวดหมู่

#### ขั้นตอนที่ 5: เพิ่มซีรี่ส์และหมวดหมู่
เพิ่มชุดข้อมูลใหม่ลงในแผนภูมิและเติมจุดข้อมูล:
```csharp
chart.ChartData.Series.Add(fact.GetCell(0, 0, 1, "Series 1"), chart.Type);
IChartSeries series = chart.ChartData.Series[0];

// กำหนดหมวดหมู่และจุดข้อมูลที่สอดคล้องกัน
chart.ChartData.Categories.Add(fact.GetCell(0, 1, 0, "C1"));
series.DataPoints.AddDataPointForLineSeries(fact.GetCell(0, 1, 1, 24));
chart.ChartData.Categories.Add(fact.GetCell(0, 2, 0, "C2"));
series.DataPoints.AddDataPointForLineSeries(fact.GetCell(0, 2, 1, 23));
chart.ChartData.Categories.Add(fact.GetCell(0, 3, 0, "C3"));
series.DataPoints.AddDataPointForLineSeries(fact.GetCell(0, 3, 1, -10));
chart.ChartData.Categories.Add(fact.GetCell(0, 4, 0, "C4"));

// เพิ่มจุดข้อมูลว่างเพื่อแสดงการจัดการค่าที่หายไป
series.DataPoints.AddDataPointForLineSeries(fact.GetCell(0, 4, 1, (double?)null));
```
ที่นี่ เราเติมข้อมูลลงในแผนภูมิด้วยหมวดหมู่และข้อมูลชุดที่เกี่ยวข้อง สังเกตว่า `null` มูลค่าได้รับการจัดการเป็นการสาธิต

#### ขั้นตอนที่ 6: เพิ่มซีรีส์อื่น
ทำซ้ำขั้นตอนนี้เพื่อเพิ่มซีรีส์อื่น:
```csharp
chart.ChartData.Series.Add(fact.GetCell(0, 0, 2, "Series 2"), chart.Type);
IChartSeries series2 = chart.ChartData.Series[1];

series2.DataPoints.AddDataPointForLineSeries(fact.GetCell(0, 1, 2, 30));
series2.DataPoints.AddDataPointForLineSeries(fact.GetCell(0, 2, 2, 10));
series2.DataPoints.AddDataPointForLineSeries(fact.GetCell(0, 3, 2, 60));
series2.DataPoints.AddDataPointForLineSeries(fact.GetCell(0, 4, 2, 40));
```

#### ขั้นตอนที่ 7: เปิดใช้งานและกำหนดค่าตำนาน
เปิดใช้งานคำอธิบายแผนภูมิเพื่อปรับปรุงการอ่าน:
```csharp
chart.HasLegend = true;
chart.Legend.Overlay = false;
```
วิธีนี้จะช่วยให้แน่ใจว่าตำนานจะมองเห็นได้และไม่ทับซ้อนบนแผนภูมิ

#### ขั้นตอนที่ 8: บันทึกการนำเสนอ
สุดท้าย ให้บันทึกการนำเสนอของคุณด้วยแผนภูมิที่เพิ่มเข้ามาใหม่:
```csharp
pres.Save("DefaultMarkersInChart.pptx");
}
```
### เคล็ดลับการแก้ไขปัญหา
- **ข้อผิดพลาดในการผูกข้อมูล:** ตรวจสอบให้แน่ใจว่าจุดข้อมูลสอดคล้องกับหมวดหมู่อย่างถูกต้อง
- **แผนภูมิไม่แสดง:** ตรวจสอบว่า `chart.HasLegend` และคุณสมบัติอื่น ๆ ได้ถูกตั้งค่าให้เหมาะสม

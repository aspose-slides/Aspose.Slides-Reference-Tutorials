## คู่มือการใช้งาน

เราจะแบ่งการใช้งานออกเป็นส่วนที่จัดการได้ตามคุณลักษณะ แต่ละส่วนจะอธิบายอย่างชัดเจนว่ากำลังทำอะไรและทำอย่างไร

### คุณสมบัติ 1: สร้างงานนำเสนอ

**ภาพรวม:** ขั้นตอนเริ่มต้นนี้สาธิตการสร้างงานนำเสนอ PowerPoint ใหม่โดยใช้ Aspose.Slides

#### ขั้นตอนที่ 1: กำหนดเส้นทางเอาต์พุต

ตั้งค่าตำแหน่งที่จะบันทึกการนำเสนอของคุณ:

```csharp
string outPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "RadarChart_Out.pptx");
```

#### ขั้นตอนที่ 2: เริ่มต้นการนำเสนอ

สร้างใหม่ `Presentation` วัตถุและบันทึกไว้:

```csharp
using (Presentation pres = new Presentation())
{
    pres.Save(outPath, SaveFormat.Pptx);
}
```

### คุณสมบัติที่ 2: เข้าถึงสไลด์และเพิ่มแผนภูมิ

**ภาพรวม:** เรียนรู้วิธีการเข้าถึงสไลด์ที่มีอยู่และเพิ่มแผนภูมิเรดาร์

#### ขั้นตอนที่ 1: เข้าถึงสไลด์แรก

เข้าถึงสไลด์แรกในการนำเสนอของคุณ:

```csharp
ISlide sld = pres.Slides[0];
```

#### ขั้นตอนที่ 2: เพิ่มแผนภูมิเรดาร์

เพิ่มแผนภูมิเรดาร์ลงในสไลด์ที่เลือก:

```csharp
IChart ichart = sld.Shapes.AddChart(ChartType.Radar, 0, 0, 400, 400);
pres.Save(outPath, SaveFormat.Pptx);
```

### คุณสมบัติที่ 3: กำหนดค่าข้อมูลแผนภูมิและชุดข้อมูล

**ภาพรวม:** ปรับแต่งแผนภูมิเรดาร์ของคุณด้วยการกำหนดค่าหมวดหมู่และชุดข้อมูล

#### ขั้นตอนที่ 1: ล้างหมวดหมู่และซีรีส์ที่มีอยู่

ลบการกำหนดค่าที่มีอยู่ก่อนหน้านี้ทั้งหมด:

```csharp
ichart.ChartData.Categories.Clear();
ichart.ChartData.Series.Clear();
```

#### ขั้นตอนที่ 2: เพิ่มหมวดหมู่และซีรีส์ใหม่

กำหนดค่าจุดข้อมูลใหม่สำหรับแผนภูมิ:

```csharp
int defaultWorksheetIndex = 0;
IChartDataWorkbook fact = ichart.ChartData.ChartDataWorkbook;

// การเพิ่มหมวดหมู่
ichart.ChartData.Categories.Add(fact.GetCell(defaultWorksheetIndex, 1, 0, "Category 1"));
// ดำเนินการเพิ่มหมวดหมู่เพิ่มเติมต่อไป...

// การเพิ่มซีรีย์
ichart.ChartData.Series.Add(fact.GetCell(defaultWorksheetIndex, 0, 1, "Series 1"), ichart.Type);
```

### คุณสมบัติที่ 4: เติมข้อมูลชุดข้อมูล

**ภาพรวม:** กรอกจุดข้อมูลสำหรับแต่ละชุดเพื่อทำให้แผนภูมิของคุณเสร็จสมบูรณ์

#### ขั้นตอนที่ 1: เพิ่มจุดข้อมูล

เติมข้อมูลตามลำดับลงในชุดแรกและชุดที่สอง:

```csharp
IChartSeries series = ichart.ChartData.Series[0];
series.DataPoints.AddDataPointForRadarSeries(fact.GetCell(defaultWorksheetIndex, 1, 1, 2.7));
// ดำเนินการเพิ่มจุดข้อมูลเพิ่มเติม...
```

### คุณสมบัติ 5: ปรับแต่งลักษณะแผนภูมิ

**ภาพรวม:** ปรับปรุงความน่าสนใจทางภาพของแผนภูมิเรดาร์ของคุณด้วยการปรับแต่งชื่อ คำอธิบาย และคุณสมบัติของแกน

#### ขั้นตอนที่ 1: ตั้งชื่อเรื่องและตำแหน่งคำอธิบาย

```csharp
ichart.ChartTitle.AddTextFrameForOverriding("Radar Chart");
ichart.Legend.Position = LegendPositionType.Bottom;
```

#### ขั้นตอนที่ 2: ปรับแต่งคุณสมบัติข้อความแกน

ใช้รูปแบบกับองค์ประกอบข้อความของแผนภูมิ:

```csharp
IChartPortionFormat txtCat = ichart.Axes.HorizontalAxis.TextFormat.PortionFormat;
txtCat.FontBold = NullableBool.True;
// ดำเนินการปรับแต่งต่อ...
```

## คู่มือการใช้งาน
ตอนนี้คุณได้ตั้งค่าเรียบร้อยแล้ว มาดูคุณลักษณะแต่ละอย่างทีละขั้นตอนกัน
### สร้างและเพิ่มแผนภูมิลงในงานนำเสนอ
#### ภาพรวม
เราจะเริ่มต้นด้วยการสร้างงานนำเสนอและเพิ่มแผนภูมิวงกลมลงในสไลด์แรก
#### ขั้นตอน:
1. **การเริ่มต้นการนำเสนอ**
   เริ่มต้นด้วยการสร้างอินสแตนซ์ของ `Presentation` คลาสซึ่งแสดงถึงไฟล์ PowerPoint ของคุณ
   ```csharp
   using Aspose.Slides;
   
   string dataDir = "YOUR_DOCUMENT_DIRECTORY";
   
   using (Presentation pres = new Presentation())
   {
       // นี่คือจุดที่เราจะเพิ่มแผนภูมิของเรา
   }
   ```
2. **เพิ่มแผนภูมิวงกลม**
   ใช้ `Shapes.AddChart` วิธีการแทรกแผนภูมิวงกลมตามพิกัดที่เจาะจงบนสไลด์ของคุณ
   ```csharp
   IChart chart = pres.Slides[0].Shapes.AddChart(
       ChartType.Pie, 50, 50, 400, 600, true);
   ```
### ตั้งค่าเวิร์กบุ๊กภายนอกสำหรับข้อมูลแผนภูมิ
#### ภาพรวม
ต่อไปเราจะมาตั้งค่าแผนภูมิวงกลมให้ใช้ข้อมูลจากเวิร์กบุ๊กภายนอกกัน
#### ขั้นตอน:
1. **การเข้าถึงข้อมูลแผนภูมิ**
   ดึงข้อมูลอินเทอร์เฟซแผนภูมิซึ่งคุณจะระบุ URL แหล่งข้อมูลภายนอกของคุณ
   ```csharp
   IChartData chartData = chart.ChartData;
   ```
2. **ตั้งค่า URL ของสมุดงานภายนอก**
   ตั้งค่า URL สำหรับแหล่งข้อมูลของคุณโดยใช้ `SetExternalWorkbook`ตัวอย่างนี้ใช้ URL ตัวแทน ซึ่งควรแทนที่ด้วยเส้นทางแหล่งข้อมูลจริงของคุณ
   ```csharp
   (chartData as ChartData).SetExternalWorkbook("http://เส้นทาง/ไม่ได้/มีอยู่", false);
   ```
### บันทึกการนำเสนอลงในไฟล์
#### ภาพรวม
สุดท้ายให้บันทึกการนำเสนอในรูปแบบ PPTX ลงในตำแหน่งที่คุณต้องการ
#### ขั้นตอน:
1. **บันทึกการนำเสนอ**
   ใช้ `Save` วิธีการของ `Presentation` คลาสที่จะเขียนไฟล์ลงดิสก์
   ```csharp
   pres.Save(dataDir + "SetExternalWorkbookWithUpdateChartData.pptx", SaveFormat.Pptx);
   ```

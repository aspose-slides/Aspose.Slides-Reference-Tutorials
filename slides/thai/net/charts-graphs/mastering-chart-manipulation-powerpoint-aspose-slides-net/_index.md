# เรียนรู้การจัดการแผนภูมิใน PowerPoint โดยใช้ Aspose.Slides สำหรับ .NET

## การแนะนำ
ในโลกปัจจุบันที่ขับเคลื่อนด้วยข้อมูล การสร้างภาพข้อมูลอย่างมีประสิทธิผลผ่านแผนภูมิถือเป็นสิ่งสำคัญสำหรับการสื่อสารและการตัดสินใจ การแยกภาพแผนภูมิจากการนำเสนอหรือการเพิ่มภาพใหม่อาจมีความซับซ้อนหากไม่มีเครื่องมือที่เหมาะสม **Aspose.Slides สำหรับ .NET** ทำให้ภารกิจเหล่านี้ง่ายขึ้น บทช่วยสอนนี้จะแนะนำคุณเกี่ยวกับวิธีการดึงภาพแผนภูมิและเพิ่มแผนภูมิประเภทต่างๆ ลงในงานนำเสนอ PowerPoint โดยใช้ Aspose.Slides

## คู่มือการใช้งาน

### การแยกภาพแผนภูมิจากงานนำเสนอ PowerPoint

#### ภาพรวม
การแยกภาพแผนภูมิมีประโยชน์เมื่อต้องแชร์หรือเก็บถาวรการแสดงภาพข้อมูลเฉพาะอย่างเป็นอิสระจากการนำเสนอแหล่งที่มา 

**ขั้นตอนที่ 1: โหลดงานนำเสนอของคุณ**
เริ่มต้นด้วยการโหลดไฟล์ PowerPoint ที่มีอยู่ของคุณ:
```csharp
using (Presentation pres = new Presentation("YOUR_DOCUMENT_DIRECTORY/test.pptx"))
{
    // ดำเนินการประมวลผลต่อไป...
}
```
แทนที่ `"YOUR_DOCUMENT_DIRECTORY"` พร้อมกับเส้นทางที่คุณเก็บเอกสารไว้

**ขั้นตอนที่ 2: เข้าถึงสไลด์และแผนภูมิที่ต้องการ**
เข้าถึงสไลด์และแผนภูมิเฉพาะโดยใช้ดัชนี:
```csharp
ISlide slide = pres.Slides[0]; // สไลด์แรก
IChart chart = (IChart)slide.Shapes[1]; // ถือว่าแผนภูมิเป็นรูปร่างที่สอง
```

**ขั้นตอนที่ 3: ดึงภาพแผนภูมิ**
ใช้ `GetImage` วิธีการสกัดภาพแทน:
```csharp
IImage img = chart.GetImage();
img.Save("YOUR_OUTPUT_DIRECTORY/image.png", Aspose.Slides.Export.ImageFormat.Png);
```
การดำเนินการนี้จะบันทึกแผนภูมิที่แยกออกมาเป็นไฟล์ PNG ปรับเส้นทางเอาต์พุตและรูปแบบตามต้องการ

### การเพิ่มแผนภูมิประเภทต่างๆ ลงใน PowerPoint

#### ภาพรวม
การเพิ่มแผนภูมิที่หลากหลายจะช่วยเสริมการนำเสนอของคุณให้สมบูรณ์ยิ่งขึ้น และให้มุมมองที่หลากหลายเกี่ยวกับข้อมูล

**ขั้นตอนที่ 1: สร้างงานนำเสนอใหม่**
เริ่มต้นด้วยการนำเสนอที่ว่างเปล่าหรือที่มีอยู่แล้ว:
```csharp
using (Presentation pres = new Presentation())
{
    ISlide slide = pres.Slides[0]; // เข้าถึงสไลด์แรก
```

**ขั้นตอนที่ 2: เพิ่มแผนภูมิประเภทต่างๆ**
เพิ่มแผนภูมิประเภทต่างๆ เช่น แผนภูมิคอลัมน์แบบกลุ่มและแผนภูมิวงกลม:
```csharp
IChart chart1 = slide.Shapes.AddChart(ChartType.ClusteredColumn, 50, 50, 300, 200);
IChart chart2 = slide.Shapes.AddChart(ChartType.Pie, 400, 50, 300, 200);
```

**ขั้นตอนที่ 3: บันทึกการนำเสนอที่อัปเดต**
บันทึกการนำเสนอหลังจากเพิ่มแผนภูมิของคุณ:
```csharp
pres.Save("YOUR_DOCUMENT_DIRECTORY/ChartsPresentation.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
```

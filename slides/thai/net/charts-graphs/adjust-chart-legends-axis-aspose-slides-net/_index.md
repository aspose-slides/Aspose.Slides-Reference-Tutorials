# วิธีการปรับคำอธิบายแผนภูมิและค่าแกนโดยใช้ Aspose.Slides .NET

คุณกำลังมองหาวิธีเพิ่มความน่าสนใจให้กับงานนำเสนอ PowerPoint ของคุณโดยการปรับคำอธิบายแผนภูมิและค่าแกนหรือไม่ ไม่ว่าคุณจะเป็นนักพัฒนาที่ต้องการสร้างรายงานแบบไดนามิกหรือเป็นผู้ที่ได้รับมอบหมายให้ปรับปรุงสุนทรียศาสตร์ของงานนำเสนอ การเชี่ยวชาญฟีเจอร์เหล่านี้ใน Aspose.Slides สำหรับ .NET จะช่วยเปลี่ยนแปลงชีวิตคุณได้ บทช่วยสอนนี้จะแนะนำคุณเกี่ยวกับการใช้ Aspose.Slides .NET เพื่อปรับขนาดแบบอักษรของคำอธิบายแผนภูมิและกำหนดค่าค่าต่ำสุดและสูงสุดของแกนแนวตั้งในแผนภูมิของคุณ

**สิ่งที่คุณจะได้เรียนรู้:**

## คู่มือการใช้งาน
หัวข้อนี้จะแนะนำคุณทีละขั้นตอนเกี่ยวกับคุณลักษณะต่างๆ

### ปรับขนาดตัวอักษรคำอธิบาย
การปรับขนาดตัวอักษรคำอธิบายจะช่วยให้อ่านได้ง่ายขึ้น โดยทำได้ดังนี้:

#### ภาพรวม
เราจะปรับขนาดตัวอักษรคำอธิบายแผนภูมิของแผนภูมิโดยใช้ Aspose.Slides สำหรับ .NET

#### ขั้นตอน
**1. โหลดงานนำเสนอของคุณ:**
เริ่มต้นด้วยการโหลดไฟล์ PowerPoint ของคุณในตำแหน่งที่คุณต้องการปรับคำอธิบายแผนภูมิ
```csharp
using (Presentation pres = new Presentation(dataDir))
{
    // เข้าถึงสไลด์แรกและเพิ่มแผนภูมิคอลัมน์แบบกลุ่ม
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 50, 50, 600, 400);
```

**2. ตั้งค่าขนาดตัวอักษรคำอธิบาย:**
ระบุความสูงของตัวอักษรที่ต้องการเพื่อการมองเห็นที่ชัดเจนยิ่งขึ้น
```csharp
    // ปรับขนาดตัวอักษรของข้อความคำอธิบายเป็น 20
    chart.Legend.TextFormat.PortionFormat.FontHeight = 20;
}
```
- **คำอธิบาย:** `FontHeight` กำหนดขนาดเป็นจุดเพื่อเพิ่มการอ่านง่าย

**3. บันทึกการนำเสนอของคุณ:**
หลังจากทำการเปลี่ยนแปลงแล้ว ให้บันทึกการนำเสนอของคุณเพื่อเก็บรักษาไว้
```csharp
pres.Save(outputDir, Aspose.Slides.Export.SaveFormat.Pptx);
```

### กำหนดค่าค่าต่ำสุดและสูงสุดของแกนแนวตั้ง
การกำหนดค่าแกนเองช่วยให้แสดงข้อมูลได้แม่นยำ

#### ภาพรวม
เรียนรู้วิธีตั้งค่าต่ำสุดและสูงสุดที่เฉพาะเจาะจงสำหรับแกนตั้งของแผนภูมิของคุณ

#### ขั้นตอน
**1. โหลดงานนำเสนอของคุณ:**
เช่นเดียวกับก่อนหน้านี้ ให้เปิดการนำเสนอที่มีแผนภูมิของคุณ
```csharp
using (Presentation pres = new Presentation(dataDir))
{
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 50, 50, 600, 400);
```

**2. ตั้งค่าแกนที่กำหนดเอง:**
ปิดใช้งานการตั้งค่าค่าแกนอัตโนมัติและกำหนดค่าของคุณเอง
```csharp
    // ปิดใช้งานค่าต่ำสุดอัตโนมัติสำหรับแกนตั้ง
    chart.Axes.VerticalAxis.IsAutomaticMinValue = false;
    // ตั้งค่าขั้นต่ำที่กำหนดเองเป็น -5
    chart.Axes.VerticalAxis.MinValue = -5;

    // ในทำนองเดียวกัน ให้ปิดการใช้งานค่าสูงสุดอัตโนมัติและตั้งค่าเป็น 10
    chart.Axes.VerticalAxis.IsAutomaticMaxValue = false;
    chart.Axes.VerticalAxis.MaxValue = 10;
}
```
- **คำอธิบาย:** การกำหนดค่าเองเหล่านี้ช่วยให้สามารถปรับขนาดข้อมูลได้ตามต้องการ

**3. บันทึกการนำเสนอของคุณ:**
ตรวจสอบให้แน่ใจว่าการเปลี่ยนแปลงของคุณได้รับการบันทึกโดยการเขียนกลับไปยังไฟล์
```csharp
pres.Save(outputDir, Aspose.Slides.Export.SaveFormat.Pptx);
```

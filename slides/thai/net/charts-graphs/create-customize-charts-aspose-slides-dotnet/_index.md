## คู่มือการใช้งาน

### การสร้างแผนภูมิคอลัมน์แบบซ้อน

เป้าหมายของเราคือการสร้างแผนภูมิคอลัมน์แบบเรียงซ้อนและปรับแต่งโดยแสดงเปอร์เซ็นต์เป็นป้ายข้อมูล ดังต่อไปนี้:

#### การเริ่มต้นการนำเสนอ

เริ่มต้นด้วยการสร้างอินสแตนซ์ของ `Presentation`-
```csharp
using Aspose.Slides;

// เริ่มต้นอินสแตนซ์คลาสการนำเสนอ
tPresentation presentation = new Presentation();
ISlide slide = presentation.Slides[0];
```

#### เพิ่มแผนภูมิลงในสไลด์

เพิ่มแผนภูมิคอลัมน์แบบเรียงซ้อนในสไลด์แรกของคุณตามพิกัดและมิติที่ระบุ:
```csharp
IChart chart = slide.Shapes.AddChart(ChartType.StackedColumn, 20, 20, 400, 400);
```
เส้นนี้สร้าง `StackedColumn` แผนภูมิที่ตำแหน่ง (20, 20) มีความกว้างและความสูง 400

#### คำนวณค่ารวมสำหรับการคำนวณเปอร์เซ็นต์

เพื่อแสดงเปอร์เซ็นต์ ให้คำนวณมูลค่ารวมสำหรับแต่ละหมวดหมู่ในชุดทั้งหมด:
```csharp
IChartSeries series;
double[] total_for_Cat = new double[chart.ChartData.Categories.Count];

for (int k = 0; k < chart.ChartData.Categories.Count; k++)
{
    IChartCategory cat = chart.ChartData.Categories[k];
    // สรุปค่าของทุกซีรี่ส์ในแต่ละหมวดหมู่
    for (int i = 0; i < chart.ChartData.Series.Count; i++)
    {
        total_for_Cat[k] += Convert.ToDouble(chart.ChartData.Series[i].DataPoints[k].Value.Data);
    }
}
```

#### ปรับแต่งป้ายข้อมูลเพื่อแสดงค่าเปอร์เซ็นต์

ถัดไป ให้ทำซ้ำผ่านแต่ละชุดและปรับแต่งป้ายข้อมูล:
```csharp
for (int x = 0; x < chart.ChartData.Series.Count; x++)
{
    series = chart.ChartData.Series[x];
    series.Labels.DefaultDataLabelFormat.ShowLegendKey = false;

    for (int j = 0; j < series.DataPoints.Count; j++)
    {
        IDataLabel lbl = series.DataPoints[j].Label;
        
        // คำนวณเปอร์เซ็นต์
        double dataPontPercent = (Convert.ToDouble(series.DataPoints[j].Value.Data) / total_for_Cat[j]) * 100;
        IPortion port = new Portion();
        port.Text = String.Format("{0:F2} %", dataPontPercent);
        port.PortionFormat.FontHeight = 8f;

        lbl.TextFrameForOverriding.Text = ""; // ล้างข้อความเพื่อหลีกเลี่ยงการทับซ้อน
        IParagraph para = lbl.TextFrameForOverriding.Paragraphs[0];
        para.Portions.Add(port);

        // กำหนดค่ารูปแบบฉลากเพื่อซ่อนฉลากข้อมูลเริ่มต้น
        lbl.DataLabelFormat.ShowSeriesName = false;
        lbl.DataLabelFormat.ShowPercentage = false; 
        lbl.DataLabelFormat.ShowLegendKey = false;
        lbl.DataLabelFormat.ShowCategoryName = false;
        lbl.DataLabelFormat.ShowBubbleSize = false;
    }
}
```

ส่วนนี้จะคำนวณเปอร์เซ็นต์สำหรับจุดข้อมูลแต่ละจุดและตั้งค่าเป็นป้ายที่กำหนดเอง เพื่อให้แน่ใจว่าไม่มีการทับซ้อนกับป้ายเริ่มต้น

#### บันทึกการนำเสนอ

สุดท้ายให้บันทึกการนำเสนอของคุณเพื่อดูผลลัพธ์:
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
presentation.Save(outputDir + "/DisplayPercentageAsLabels_out.pptx", SaveFormat.Pptx);
```

## การประยุกต์ใช้งานจริง

การแสดงเปอร์เซ็นต์ในแผนภูมิอาจมีประโยชน์อย่างยิ่งในสถานการณ์เช่น:
1. **การรายงานทางการเงิน:** แสดงการกระจายพอร์ตโฟลิโอหรือผลตอบแทนการลงทุนเป็นเปอร์เซ็นต์
2. **การวิเคราะห์การขาย:** แสดงข้อมูลส่วนแบ่งการตลาดตามเปอร์เซ็นต์เพื่อเน้นย้ำถึงประสิทธิภาพในแต่ละภูมิภาค
3. **ผลการสำรวจ:** แสดงการตอบแบบสำรวจเป็นเปอร์เซ็นต์ เพื่อการเปรียบเทียบภาพที่ดีขึ้น
4. **การจัดการโครงการ:** ใช้แผนภูมิวงกลมพร้อมเปอร์เซ็นต์เพื่อแสดงการจัดสรรทรัพยากร
5. **การศึกษา:** อธิบายแนวคิดทางสถิติโดยใช้ภาพประกอบเปอร์เซ็นต์ที่ชัดเจน

การรวมแผนภูมิที่ปรับแต่งเองเหล่านี้เข้าในระบบ เช่น CRM หรือ ERP จะช่วยปรับปรุงแดชบอร์ดและรายงาน ช่วยในการตัดสินใจ

#### การปรับแต่งคุณสมบัติแบบอักษรสำหรับแผนภูมิ
ฟีเจอร์นี้ช่วยให้คุณปรับปรุงความสวยงามของแผนภูมิได้โดยการปรับคุณสมบัติของแบบอักษร วิธีใช้งานมีดังนี้

**ขั้นตอนที่ 1: กำหนดเส้นทางไดเร็กทอรี**
เริ่มต้นโดยระบุตำแหน่งที่ไฟล์อินพุตและเอาต์พุตของคุณ:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputPath = Path.Combine(dataDir, "FontPropertiesForChart.pptx");
```

**ขั้นตอนที่ 2: สร้างอินสแตนซ์การนำเสนอใหม่**
เริ่มต้นวัตถุการนำเสนอใหม่เพื่อโฮสต์แผนภูมิของคุณ:
```csharp
using (Presentation pres = new Presentation()) {
    // ขั้นตอนต่อไปจะดำเนินการที่นี่
}
```

**ขั้นตอนที่ 3: เพิ่มแผนภูมิคอลัมน์แบบกลุ่ม**
แทรกแผนภูมิลงในสไลด์แรกตามพิกัดและมิติที่ระบุ:
```csharp
IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 100, 100, 500, 400);
```

**ขั้นตอนที่ 4: ตั้งค่าความสูงของแบบอักษรสำหรับข้อความในแผนภูมิ**
ปรับแต่งขนาดตัวอักษรเพื่อปรับปรุงการอ่าน:
```csharp
chart.TextFormat.PortionFormat.FontHeight = 20;
```

**ขั้นตอนที่ 5: เปิดใช้งานการแสดงค่าบนป้ายข้อมูล**
ตรวจสอบให้แน่ใจว่าค่าข้อมูลสามารถมองเห็นได้ โดยเพิ่มบริบทให้กับแผนภูมิของคุณ:
```csharp
chart.ChartData.Series[0].Labels.DefaultDataLabelFormat.ShowValue = true;
```

**ขั้นตอนที่ 6: บันทึกการนำเสนอ**
บันทึกการนำเสนอของคุณพร้อมการปรับแต่งทั้งหมดที่ใช้:
```csharp
pres.Save(outputPath, SaveFormat.Pptx);
```

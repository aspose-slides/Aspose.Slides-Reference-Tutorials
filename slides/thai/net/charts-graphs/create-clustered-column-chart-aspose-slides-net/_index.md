## คู่มือการใช้งาน

ในส่วนนี้เราจะแนะนำการสร้างงานนำเสนอและการเพิ่มแผนภูมิคอลัมน์แบบกลุ่ม

### การสร้างการนำเสนอที่ว่างเปล่า

เริ่มต้นด้วยการตั้งค่าเส้นทางไดเรกทอรีเอกสารของคุณ นี่คือที่ที่งานนำเสนอที่สร้างขึ้นจะถูกบันทึก:
```csharp

Presentation pres = new Presentation();
```

### การเพิ่มแผนภูมิคอลัมน์แบบคลัสเตอร์ลงในสไลด์

ขั้นตอนต่อไป เพิ่มแผนภูมิคอลัมน์แบบคลัสเตอร์ลงในสไลด์แรกที่ตำแหน่งและขนาดที่ระบุ:
```csharp
// เพิ่มแผนภูมิคอลัมน์แบบคลัสเตอร์ที่ (20, 20) พร้อมมิติ (500x400)
IChart chart = pres.Slides[0].Shapes.AddChart(
    ChartType.ClusteredColumn,
    20, 20, 500, 400);
```
**คำอธิบาย:** สไนปเป็ตนี้จะสร้างการนำเสนอแบบว่างเปล่าและเพิ่มแผนภูมิคอลัมน์แบบคลัสเตอร์ `AddChart` วิธีการระบุชนิดของแผนภูมิ (`ClusteredColumn`) และตำแหน่ง/ขนาด (x: 20, y: 20, กว้าง: 500, สูง: 400)

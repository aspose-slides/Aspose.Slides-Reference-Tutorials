## คู่มือการใช้งาน
มาใช้ฟีเจอร์การจัดการชุดแผนภูมิของเรากันแบบทีละขั้นตอน

### การเพิ่มและการกำหนดค่าชุดแผนภูมิ
#### ภาพรวม
การเพิ่มแผนภูมิคอลัมน์แบบคลัสเตอร์เกี่ยวข้องกับการเริ่มต้นแผนภูมิ การกำหนดค่าคุณสมบัติ และการเติมข้อมูล ทำตามขั้นตอนเหล่านี้:

##### ขั้นตอนที่ 1: เริ่มต้นเอกสารการนำเสนอของคุณ
สร้างวัตถุการนำเสนอเพื่อเริ่มเพิ่มแผนภูมิของคุณ:
```csharp
string yourDocumentDirectory = "YOUR_DOCUMENT_DIRECTORY";
using (Presentation pres = new Presentation())
{
    // โค้ดสำหรับการเพิ่มแผนภูมิอยู่ที่นี่
}
```
**ทำไม**:โค้ดนี้จะตั้งค่าสภาพแวดล้อมการทำงานเพื่อให้แน่ใจว่าทุกอย่างถูกรวมไว้ในวัตถุการนำเสนอ

##### ขั้นตอนที่ 2: เพิ่มแผนภูมิคอลัมน์แบบคลัสเตอร์
เพิ่มแผนภูมิคอลัมน์แบบกลุ่มลงในสไลด์แรกของคุณ:
```csharp
IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 50, 50, 600, 400, true);
```
**ทำไม**การเรียกวิธีการนี้จะเพิ่มวัตถุแผนภูมิใหม่ตามพิกัดที่ระบุโดยมีมิติที่กำหนดไว้ล่วงหน้า

##### ขั้นตอนที่ 3: กำหนดค่าชุดแผนภูมิ
ล้างซีรีย์ที่มีอยู่และเพิ่มซีรีย์ของคุณเอง:
```csharp
IChartSeriesCollection series = chart.ChartData.Series;
series.Clear();
series.Add(chart.ChartData.ChartDataWorkbook.GetCell(0, "B1"), chart.Type);
```
**ทำไม**การล้างข้อมูลจะทำให้ไม่มีข้อมูลเหลือมาขัดขวางการกำหนดค่าใหม่ การเพิ่มชุดข้อมูลจะเริ่มต้นการแทรกจุดข้อมูล

##### ขั้นตอนที่ 4: เพิ่มจุดข้อมูล
เติมข้อมูลในแผนภูมิของคุณ รวมถึงค่าลบ:
```csharp
series[0].DataPoints.AddDataPointForBarSeries(chart.ChartData.ChartDataWorkbook.GetCell(0, "B2"), -50);
```
**ทำไม**:การเพิ่มจุดข้อมูลถือเป็นสิ่งสำคัญสำหรับการแสดงภาพชุดข้อมูล ค่าลบได้รับการสนับสนุนเพื่อแสดงการขาดดุลหรือการสูญเสีย

### เคล็ดลับการแก้ไขปัญหา
- ตรวจสอบให้แน่ใจว่าเนมสเปซทั้งหมดได้รับการนำเข้าอย่างถูกต้อง
- ตรวจสอบประเภทแผนภูมิและตัวระบุชุดอีกครั้งเพื่อความแม่นยำ
- ตรวจสอบแหล่งข้อมูลของคุณเพื่อดูความไม่สอดคล้องกันซึ่งอาจทำให้เกิดข้อผิดพลาดในการรันไทม์

# เรียนรู้การเติมสีแบบอัตโนมัติในแผนภูมิ .NET ด้วย Aspose.Slides

## การแนะนำ
คุณกำลังประสบปัญหาในการตั้งค่าสีสำหรับแต่ละชุดแผนภูมิด้วยตนเองใช่หรือไม่ ปรับปรุงการนำเสนอของคุณได้อย่างง่ายดายด้วยการใช้ Aspose.Slides สำหรับ .NET เพื่อทำให้กระบวนการเป็นแบบอัตโนมัติ บทช่วยสอนนี้จะแนะนำคุณเกี่ยวกับการใช้สีเติมอัตโนมัติ การปรับกระบวนการทำงานให้มีประสิทธิภาพ และการรับประกันความสอดคล้องของภาพในแต่ละสไลด์

### สิ่งที่คุณจะได้เรียนรู้:
- การนำการเติมสีแบบอัตโนมัติไปใช้กับแผนภูมิด้วย Aspose.Slides
- คุณสมบัติหลักและประโยชน์ของฟังก์ชันนี้
- การประยุกต์ใช้งานจริงและความเป็นไปได้ในการบูรณาการ

ก่อนจะดำเนินการตามขั้นตอนการใช้งาน ให้แน่ใจว่าคุณมีทุกสิ่งที่จำเป็นสำหรับประสบการณ์ที่ราบรื่น

## คู่มือการใช้งาน
ในส่วนนี้มีรายละเอียดเกี่ยวกับการนำการเติมสีแบบซีรีส์อัตโนมัติไปใช้งานด้วย Aspose.Slides สำหรับ .NET เพื่อให้แน่ใจว่ามีความชัดเจนและเข้าใจง่าย

### การเพิ่มแผนภูมิคอลัมน์แบบคลัสเตอร์พร้อมสีเติมชุดข้อมูลอัตโนมัติ
#### ภาพรวม
สร้างแผนภูมิคอลัมน์แบบกลุ่มในงานนำเสนอของคุณ โดยกำหนดค่าให้กำหนดสีชุดโดยอัตโนมัติเพื่อความสวยงามและประสิทธิภาพที่เพิ่มขึ้น

#### ขั้นตอนที่ 1: สร้างงานนำเสนอใหม่
เริ่มต้นใหม่ `Presentation` วัตถุ:
```csharp
using Aspose.Slides;
using Aspose.Slides.Charts;
// ระบุเส้นทางไดเร็กทอรีเอกสารของคุณ
cstring dataDir = "YOUR_DOCUMENT_DIRECTORY";
using (Presentation presentation = new Presentation()) {
    // ดำเนินการเพิ่มแผนภูมิในขั้นตอนถัดไป...
}
```

#### ขั้นตอนที่ 2: เพิ่มแผนภูมิคอลัมน์แบบคลัสเตอร์
เพิ่มแผนภูมิคอลัมน์แบบคลัสเตอร์ที่ตำแหน่ง (100, 50) พร้อมมิติ (600x400):
```csharp
// เพิ่มแผนภูมิคอลัมน์แบบคลัสเตอร์\IChart chart = presentation.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 100, 50, 600, 400);
```

#### ขั้นตอนที่ 3: กำหนดค่าสีซีรีย์อัตโนมัติ
ทำซ้ำผ่านแต่ละซีรีส์เพื่อเปิดใช้งานการเติมสีอัตโนมัติ:
```csharp
// วนซ้ำในแต่ละซีรี่ส์เพื่อตั้งค่าสีอัตโนมัติ
type IChartSeries series;
for (int i = 0; i < chart.ChartData.Series.Count; i++) {
    series = chart.ChartData.Series[i];
    // ตั้งค่าสีซีรีย์โดยอัตโนมัติ
    series.Format.Fill.FillType = FillType.Solid;
    series.Format.Fill.SolidFillColor.Color = Color.FromArgb(255, GetRandomColor());
}
```

## การแปลงจาก PPTX เป็น ODP

เริ่มต้นด้วยโค้ดสำหรับแปลงจาก PPTX เป็น ODP คำแนะนำทีละขั้นตอนมีดังนี้

```csharp
// สร้างอินสแตนซ์วัตถุการนำเสนอที่แสดงถึงไฟล์การนำเสนอ
using (Presentation pres = new Presentation("ConversionFromPresentation.pptx"))
{
    // บันทึกงานนำเสนอ PPTX เป็นรูปแบบ ODP
    pres.Save("ConvertedToOdp", Aspose.Slides.Export.SaveFormat.Odp);
}
```

 ในข้อมูลโค้ดนี้ เราสร้าง`Presentation` วัตถุ ระบุไฟล์ PPTX อินพุต จากนั้นเราก็ใช้`Save` วิธีการบันทึกการนำเสนอในรูปแบบ ODP

## การแปลงจาก ODP เป็น PPTX

ตอนนี้ เรามาสำรวจการแปลงแบบย้อนกลับจาก ODP เป็น PPTX:

```csharp
// สร้างอินสแตนซ์วัตถุการนำเสนอที่แสดงถึงไฟล์การนำเสนอ
using (Presentation pres = new Presentation("OpenOfficePresentation.odp"))
{
    // บันทึกงานนำเสนอ ODP เป็นรูปแบบ PPTX
    pres.Save("ConvertedFromOdp", Aspose.Slides.Export.SaveFormat.Pptx);
}
```

 รหัสนี้ค่อนข้างคล้ายกับตัวอย่างก่อนหน้า เราสร้างก`Presentation`วัตถุระบุไฟล์ ODP อินพุตและใช้ไฟล์`Save` วิธีการบันทึกในรูปแบบ PPTX

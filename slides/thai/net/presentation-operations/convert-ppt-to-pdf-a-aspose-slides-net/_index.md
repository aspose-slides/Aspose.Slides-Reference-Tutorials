## คู่มือการใช้งาน
ตอนนี้เราจะมาเจาะลึกการใช้งานฟีเจอร์แต่ละอย่าง แต่ละส่วนจะให้ข้อมูลสรุปสั้นๆ ตามด้วยขั้นตอนโดยละเอียด

### บันทึกการนำเสนอเป็นเอกสารที่สอดคล้องกับ PDF/A-1a
**ภาพรวม:**
รูปแบบ PDF/A-1a เหมาะอย่างยิ่งสำหรับการเก็บถาวรในระยะยาว โดยรับรองว่าเอกสารจะไม่มีการเปลี่ยนแปลงและสามารถเข้าถึงได้ตลอดเวลา

#### การดำเนินการทีละขั้นตอน:
##### 1. กำหนดเส้นทางไฟล์
```csharp
string pptxFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "tagged-pdf-demo.pptx");
string outPdf1aFile = Path.Combine("YOUR_OUTPUT_DIRECTORY", "tagged-pdf-demo_1a.pdf");
```
##### 2. โหลดงานนำเสนอ
```csharp
using (Presentation presentation = new Presentation(pptxFile))
{
    // ดำเนินการบันทึกเอกสาร
}
```
##### 3. บันทึกเป็น PDF/A-1a
```csharp
presentation.Save(outPdf1aFile, SaveFormat.Pdf,
    new PdfOptions { Compliance = PdfCompliance.PdfA1a });
```
**คำอธิบาย:** การ `PdfOptions` วัตถุระบุว่าเอาท์พุตควรเป็นไปตามมาตรฐาน PDF/A-1a

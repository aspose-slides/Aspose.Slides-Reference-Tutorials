### คู่มือการใช้งาน

ให้เราแบ่งการใช้งานออกเป็นสองคุณสมบัติหลัก: การใช้การเปลี่ยนสไลด์และการบันทึกการนำเสนอ

#### การใช้การเปลี่ยนสไลด์

ฟีเจอร์นี้สาธิตวิธีการเพิ่มเอฟเฟ็กต์การเปลี่ยนผ่านระหว่างสไลด์ในงานนำเสนอ PowerPoint โดยใช้ Aspose.Slides สำหรับ .NET

##### ภาพรวมทีละขั้นตอน:
1. **โหลดการนำเสนอของคุณ**:เริ่มต้นด้วยการโหลดไฟล์ PowerPoint ที่มีอยู่ลงใน `Presentation` ระดับ.
2. **ใช้ประเภทการเปลี่ยนแปลง**กำหนดประเภทการเปลี่ยนผ่านที่แตกต่างกันเช่น วงกลมหรือหวี ให้กับสไลด์เฉพาะ
3. **กำหนดเวลาการเปลี่ยนแปลง**กำหนดว่าการเปลี่ยนแปลงแต่ละครั้งจะใช้เวลานานเท่าใด โดยพิจารณาจากจำนวนคลิกหรือช่วงเวลา

##### การนำโค้ดไปใช้

```csharp
// ระบุไดเรกทอรีเอกสาร
string dataDir = "YOUR_DOCUMENT_DIRECTORY";

using (Presentation presentation = new Presentation(dataDir + "/AccessSlides.pptx"))
{
    // ใช้การเปลี่ยนรูปแบบวงกลมกับสไลด์ที่ 1
    presentation.Slides[0].SlideShowTransition.Type = TransitionType.Circle;
    
    // ตั้งเวลาเปลี่ยนภาพเป็น 3 วินาที
    presentation.Slides[0].SlideShowTransition.AdvanceOnClick = true;
    presentation.Slides[0].SlideShowTransition.AdvanceAfterTime = 3000; // เวลาเป็นมิลลิวินาที
    
    // ใช้การเปลี่ยนประเภทหวีกับสไลด์ที่ 2
    presentation.Slides[1].SlideShowTransition.Type = TransitionType.Comb;
    
    // ตั้งเวลาเปลี่ยนภาพเป็น 5 วินาที
    presentation.Slides[1].SlideShowTransition.AdvanceOnClick = true;
    presentation.Slides[1].SlideShowTransition.AdvanceAfterTime = 5000; // เวลาเป็นมิลลิวินาที
    
    // บันทึกการนำเสนอด้วยชื่อใหม่เพื่อระบุการเปลี่ยนแปลงที่ใช้
    presentation.Save(dataDir + "/BetterTransitions_out.pptx", SaveFormat.Pptx);
}
```

**คำอธิบาย-**
- **`TransitionType`:** ระบุชนิดของเอฟเฟ็กต์การเปลี่ยนแปลง
- **`AdvanceOnClick` และ `AdvanceAfterTime`-** ควบคุมวิธีการเลื่อนสไลด์ ไม่ว่าจะด้วยการคลิกหรือหลังจากระยะเวลาที่ตั้งไว้

##### เคล็ดลับการแก้ไขปัญหา:
- ตรวจสอบให้แน่ใจว่าเส้นทางเอกสารของคุณถูกต้องเพื่อหลีกเลี่ยงข้อผิดพลาดไม่พบไฟล์
- ตรวจสอบข้อยกเว้นใด ๆ เมื่อบันทึกไฟล์เนื่องจากสิทธิ์ไม่เพียงพอ

#### การบันทึกการนำเสนอ

เมื่อใช้การเปลี่ยนแปลงแล้ว คุณสามารถบันทึกการนำเสนอของคุณในรูปแบบต่างๆ โดยใช้ Aspose.Slides

##### การนำโค้ดไปใช้

```csharp
// ระบุไดเรกทอรีเอาท์พุต
string dataDir = "YOUR_DOCUMENT_DIRECTORY";

using (Presentation presentation = new Presentation(dataDir + "/AccessSlides.pptx"))
{
    // บันทึกการนำเสนอที่อัปเดตในรูปแบบ PPTX
    string outputFilePath = dataDir + "/SampleTransition_out.pptx";
    presentation.Save(outputFilePath, SaveFormat.Pptx);
}
```

**คำอธิบาย-**
- **`Save`:** บันทึกการนำเสนอของคุณไปยังตำแหน่งและรูปแบบที่ระบุ

2. **สื่อการเรียนรู้**:ปรับปรุงการมีส่วนร่วมในสถานศึกษาโดยทำให้เนื้อหามีความมีชีวิตชีวามากขึ้น

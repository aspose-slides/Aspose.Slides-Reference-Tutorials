## คู่มือการใช้งาน
ตอนนี้ทุกอย่างตั้งค่าเสร็จเรียบร้อยแล้ว มาดูวิธีการล็อกหรือปลดล็อกอัตราส่วนภาพของตารางใน PowerPoint โดยใช้ Aspose.Slides กัน
### การล็อก/ปลดล็อกอัตราส่วนภาพ
ฟีเจอร์นี้ช่วยให้คุณรักษาขนาดของตารางได้แม้ในขณะที่ปรับขนาดองค์ประกอบอื่นๆ บนสไลด์ของคุณ โดยวิธีการใช้งานมีดังนี้
#### ขั้นตอนที่ 1: โหลดงานนำเสนอของคุณ
ขั้นแรกโหลดไฟล์นำเสนอที่มีตาราง:
```csharp
using (Presentation pres = new Presentation(dataDir + "/pres.pptx"))
{
    // โค้ดสำหรับจัดการตารางจะอยู่ที่นี่
}
```
#### ขั้นตอนที่ 2: เข้าถึงรูปร่างตาราง
ระบุและเข้าถึงรูปร่างแรกบนสไลด์ของคุณ โดยให้แน่ใจว่าเป็นตาราง:
```csharp
ITable table = (ITable)pres.Slides[0].Shapes[0];
```
#### ขั้นตอนที่ 3: สลับการล็อกอัตราส่วนภาพ
ตรวจสอบว่าอัตราส่วนภาพถูกล็อกอยู่หรือไม่ จากนั้นสลับสถานะเป็นล็อกหรือปลดล็อก:
```csharp
bool originalLockState = table.ShapeLock.AspectRatioLocked;
table.ShapeLock.AspectRatioLocked = !originalLockState; // ย้อนกลับสถานะปัจจุบัน
```
#### ขั้นตอนที่ 4: บันทึกการเปลี่ยนแปลงของคุณ
สุดท้าย ให้บันทึกงานนำเสนอที่คุณแก้ไขลงในไฟล์ใหม่:
```csharp
pres.Save(outputPath + "/pres-out.pptx", SaveFormat.Pptx);
```
### เคล็ดลับการแก้ไขปัญหา
- ตรวจสอบให้แน่ใจว่ารูปร่างที่คุณกำลังเข้าถึงนั้นเป็นตารางจริง ๆ
- ตรวจสอบว่าเส้นทางสำหรับไฟล์อินพุตและเอาต์พุตได้รับการตั้งค่าอย่างถูกต้อง
- หากการเปลี่ยนแปลงอัตราส่วนภาพไม่สะท้อน ให้ตรวจสอบว่าองค์ประกอบสไลด์อื่น ๆ อาจมีอิทธิพลต่อมิติหรือไม่

2. **เค้าโครงที่ตอบสนอง**:ปรับขนาดตารางโดยไม่ทำให้การนำเสนอข้อมูลผิดเพี้ยนเมื่อปรับขนาดการนำเสนอสำหรับขนาดหน้าจอที่แตกต่างกัน

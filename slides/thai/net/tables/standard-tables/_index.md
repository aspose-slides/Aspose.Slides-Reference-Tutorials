# สร้างตารางมาตรฐานใน PowerPoint ด้วย Aspose.Slides

Aspose.Slides สำหรับ .NET เป็น API ที่มีประสิทธิภาพที่ช่วยให้ผู้พัฒนาสามารถสร้าง อ่าน เขียน แก้ไข และแปลงงานนำเสนอ PowerPoint ได้โดยผ่านโปรแกรมโดยไม่ต้องใช้ Microsoft PowerPoint ตัวอย่างนี้แสดงวิธีการสร้างตารางมาตรฐานพร้อมเส้นขอบที่จัดรูปแบบเองในงานนำเสนอ PowerPoint

## คำแนะนำในการติดตั้ง

1. ติดตั้งผ่านตัวจัดการแพ็กเกจ NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. หรือผ่านทาง .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. เพิ่มเนมสเปซต่อไปนี้ลงในไฟล์ C# ของคุณ:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Tables;
   using System.Drawing;
   ```

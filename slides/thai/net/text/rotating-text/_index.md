# การสร้างข้อความแนวตั้งใน PowerPoint โดยใช้ Aspose.Slides

Aspose.Slides สำหรับ .NET มอบความสามารถอันทรงพลังสำหรับการสร้างและจัดการการนำเสนอ PowerPoint ด้วยโปรแกรม ตัวอย่างนี้แสดงวิธีการสร้างการนำเสนอที่มีรูปสี่เหลี่ยมผืนผ้าพร้อมข้อความที่หมุนในแนวตั้ง ข้อความจะถูกวางในมุม 270 องศา ทำให้สามารถอ่านจากล่างขึ้นบนได้ เทคนิคนี้มีประโยชน์สำหรับการสร้างป้ายด้านข้าง เค้าโครงข้อความเฉพาะ หรือองค์ประกอบการออกแบบเฉพาะในการนำเสนอของคุณ

## คำแนะนำในการติดตั้ง

1. ติดตั้ง Aspose.Slides สำหรับ .NET
   
   ติดตั้งแพ็กเกจ Aspose.Slides โดยใช้ตัวจัดการแพ็กเกจ NuGet:
   
   ```csharp
   Install-Package Aspose.Slides
   ```
   
   หรือใช้ .NET CLI:
   
   ```csharp
   dotnet add package Aspose.Slides
   ```

2. เพิ่มการอ้างอิง
   
   เพิ่มคำสั่ง using ต่อไปนี้ลงในไฟล์ C# ของคุณ:
   
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

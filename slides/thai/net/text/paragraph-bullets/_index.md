# การสร้างจุดหัวข้อแบบกำหนดเองในงานนำเสนอ PowerPoint

Aspose.Slides สำหรับ .NET มอบความสามารถที่แข็งแกร่งสำหรับการสร้างและจัดการการนำเสนอ PowerPoint ด้วยโปรแกรม ตัวอย่างนี้เน้นที่การนำจุดหัวข้อย่อยแบบกำหนดเองไปใช้ในกรอบข้อความโดยเฉพาะ โดยแสดงวิธีการควบคุมประเภท สี การเยื้อง และขนาด โค้ดนี้จะสร้างการนำเสนอที่มีรูปร่างสี่เหลี่ยมผืนผ้าที่มีย่อหน้าสองย่อหน้า โดยย่อหน้าหนึ่งมีสัญลักษณ์หัวข้อย่อย และอีกย่อหน้าหนึ่งมีรูปแบบหัวข้อย่อยแบบมีหมายเลข

## คำแนะนำในการติดตั้ง

1. ติดตั้ง Aspose.Slides สำหรับ .NET โดยใช้ตัวจัดการแพ็กเกจ NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. หรือดาวน์โหลดแพ็คเกจโดยตรง:
   - เยี่ยมชม [หน้าดาวน์โหลด Aspose.Slides](https://downloads.aspose.com/slides/net)
   - ดาวน์โหลดเวอร์ชั่นล่าสุด
   - แยกและอ้างอิง Aspose.Slides.dll ในโครงการของคุณ

3. เพิ่มคำสั่งใช้ที่จำเป็น:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

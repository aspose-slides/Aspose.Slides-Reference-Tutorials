# สร้างการนำเสนอ PowerPoint ด้วย TextBox ใน Aspose.Slides สำหรับ .NET

Aspose.Slides สำหรับ .NET เป็น API ที่มีประสิทธิภาพที่ช่วยให้นักพัฒนาสามารถสร้าง แก้ไข และแปลงงานนำเสนอ PowerPoint ได้โดยไม่ต้องใช้ Microsoft PowerPoint ตัวอย่างนี้แสดงวิธีการสร้างงานนำเสนอโดยใช้กล่องข้อความที่กำหนดเอง ใช้การจัดรูปแบบ และบันทึกเป็นไฟล์ PPTX

## คำแนะนำในการติดตั้ง

1. ติดตั้งผ่านตัวจัดการแพ็กเกจ NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. ติดตั้งผ่าน .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. ติดตั้งด้วยตนเอง:
   - ดาวน์โหลดเวอร์ชันล่าสุดได้จาก [หน้าเผยแพร่ Aspose.Slides สำหรับ .NET](https://releases.aspose.com/slides/net/)
   - เพิ่มการอ้างอิง DLL ลงในโครงการของคุณ

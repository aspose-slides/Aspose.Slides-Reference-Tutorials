# การหมุนชื่อและป้ายกำกับของแผนภูมิใน PowerPoint ด้วย Aspose.Slides

Aspose.Slides สำหรับ .NET มอบความสามารถอันทรงพลังสำหรับการสร้างและจัดการการนำเสนอ PowerPoint ด้วยโปรแกรม ตัวอย่างนี้แสดงวิธีการสร้างการนำเสนอด้วยแผนภูมิคอลัมน์แบบคลัสเตอร์ที่มีองค์ประกอบข้อความที่หมุนได้ โดยเฉพาะการหมุนทั้งชื่อแผนภูมิและป้ายข้อมูลเพื่อให้อ่านง่ายและดึงดูดสายตามากขึ้น

## คำแนะนำในการติดตั้ง

1. ติดตั้งผ่านตัวจัดการแพ็กเกจ NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. หรือติดตั้งผ่าน .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. เพิ่มเนมสเปซลงในไฟล์ C# ของคุณ:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Charts;
   ```

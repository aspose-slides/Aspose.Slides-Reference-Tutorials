# การสร้างจุดหัวข้อหลายระดับใน PowerPoint ด้วย Aspose.Slides

Aspose.Slides สำหรับ .NET มอบ API ที่ทรงพลังสำหรับการสร้างและจัดการการนำเสนอ PowerPoint ด้วยโปรแกรม ตัวอย่างนี้แสดงวิธีการสร้างการนำเสนอด้วยจุดหัวข้อย่อยหลายระดับ โดยแสดงวิธีการเพิ่มรูปร่างอัตโนมัติด้วยกรอบข้อความ และจัดรูปแบบย่อหน้าด้วยรูปแบบจุดหัวข้อย่อยและระดับการย่อหน้าที่แตกต่างกัน คุณสามารถสร้างเนื้อหาแบบลำดับชั้นได้อย่างง่ายดายด้วยสัญลักษณ์จุดหัวข้อย่อยที่กำหนดเองสำหรับแต่ละระดับการย่อหน้า เหมาะอย่างยิ่งสำหรับการสร้างการนำเสนอที่มีโครงสร้างและเป็นมืออาชีพโดยไม่ต้องติดตั้ง Microsoft PowerPoint

## คำแนะนำในการติดตั้ง

1. ติดตั้งผ่านตัวจัดการแพ็กเกจ NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. ติดตั้งผ่าน .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. ติดตั้งผ่านคอนโซล Package Manager:
   ```
   PM> Install-Package Aspose.Slides
   ```

4. การติดตั้งด้วยตนเอง:
   - ดาวน์โหลดเวอร์ชันล่าสุดได้จาก [Aspose.Slides สำหรับ .NET](https://releases.aspose.com/slides/net/)
   - เพิ่มการอ้างอิงถึง Aspose.Slides.dll ในโครงการของคุณ

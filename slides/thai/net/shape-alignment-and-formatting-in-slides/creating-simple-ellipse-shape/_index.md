# สร้างรูปทรงวงรีได้อย่างง่ายดายด้วย Aspose.Slides .NET

## การแนะนำ
ในโลกที่เปลี่ยนแปลงตลอดเวลาของการออกแบบงานนำเสนอ การใช้รูปทรงต่างๆ เช่น วงรีสามารถเพิ่มความคิดสร้างสรรค์และความเป็นมืออาชีพได้ Aspose.Slides สำหรับ .NET นำเสนอโซลูชันอันทรงพลังสำหรับการจัดการไฟล์งานนำเสนอด้วยโปรแกรม บทช่วยสอนนี้จะแนะนำคุณตลอดขั้นตอนการสร้างรูปทรงวงรีที่เรียบง่ายในสไลด์งานนำเสนอโดยใช้ Aspose.Slides สำหรับ .NET
## ข้อกำหนดเบื้องต้น
ก่อนจะเริ่มบทช่วยสอนนี้ โปรดแน่ใจว่าคุณมีข้อกำหนดเบื้องต้นดังต่อไปนี้:
- Aspose.Slides สำหรับ .NET: ตรวจสอบให้แน่ใจว่าคุณได้ติดตั้งไลบรารี Aspose.Slides สำหรับ .NET แล้ว คุณสามารถดาวน์โหลดได้จาก [หน้าวางจำหน่าย](https://releases-aspose.com/slides/net/).
- สภาพแวดล้อมการพัฒนา: ตั้งค่าสภาพแวดล้อมการพัฒนา .NET บนเครื่องของคุณ
## นำเข้าเนมสเปซ
ในโครงการ .NET ของคุณ เริ่มต้นด้วยการนำเข้าเนมสเปซที่จำเป็น:
```csharp
using System.IO;
using Aspose.Slides;
using Aspose.Slides.Export;
```
เนมสเปซเหล่านี้ให้คลาสและวิธีการที่จำเป็นสำหรับการทำงานกับสไลด์และรูปร่างการนำเสนอ
## ขั้นตอนที่ 1: ตั้งค่าการนำเสนอ
เริ่มต้นด้วยการสร้างงานนำเสนอใหม่และเข้าถึงสไลด์แรก เพิ่มโค้ดต่อไปนี้เพื่อให้บรรลุสิ่งนี้:
```csharp
// เส้นทางไปยังไดเร็กทอรีเอกสาร
string dataDir = "Your Document Directory";
// สร้างไดเร็กทอรีหากยังไม่มีอยู่
bool IsExists = System.IO.Directory.Exists(dataDir);
if (!IsExists)
    System.IO.Directory.CreateDirectory(dataDir);
// คลาสการสร้างตัวอย่างการนำเสนอ
using (Presentation pres = new Presentation())
{
    // รับสไลด์แรก
    ISlide sld = pres.Slides[0];
```
โค้ดนี้จะเริ่มต้นการนำเสนอใหม่และเลือกสไลด์แรกเพื่อการจัดการเพิ่มเติม
## ขั้นตอนที่ 2: เพิ่มรูปทรงวงรี
ตอนนี้เรามาเพิ่มรูปวงรีลงในสไลด์โดยใช้ `AddAutoShape` วิธี:
```csharp
// เพิ่มรูปร่างอัตโนมัติของชนิดวงรี
sld.Shapes.AddAutoShape(ShapeType.Ellipse, 50, 150, 150, 50);
```
บรรทัดโค้ดนี้จะสร้างรูปวงรีที่พิกัด (50, 150) โดยมีความกว้าง 150 หน่วยและมีความสูง 50 หน่วย
## ขั้นตอนที่ 3: บันทึกการนำเสนอ
สุดท้าย ให้บันทึกงานนำเสนอที่แก้ไขแล้วลงในดิสก์โดยใช้ชื่อไฟล์ที่ระบุโดยใช้รหัสดังต่อไปนี้:
```csharp
// เขียนไฟล์ PPTX ลงดิสก์
pres.Save(dataDir + "EllipseShp1_out.pptx", SaveFormat.Pptx);
```
ขั้นตอนนี้จะช่วยให้แน่ใจว่าการเปลี่ยนแปลงของคุณยังคงอยู่ และคุณสามารถดูการนำเสนอที่ได้ผลลัพธ์ด้วยรูปร่างวงรีที่เพิ่มเข้ามาใหม่ได้
## บทสรุป
ขอแสดงความยินดี! คุณได้สร้างรูปทรงวงรีที่เรียบง่ายในสไลด์การนำเสนอสำเร็จแล้วโดยใช้ Aspose.Slides สำหรับ .NET บทช่วยสอนนี้ให้ความเข้าใจพื้นฐานเกี่ยวกับการทำงานกับรูปทรง การตั้งค่าการนำเสนอ และการบันทึกไฟล์ที่แก้ไข

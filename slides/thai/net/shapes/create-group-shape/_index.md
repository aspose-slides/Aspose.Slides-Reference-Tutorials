# สร้างรูปร่างกลุ่มใน PowerPoint ด้วย Aspose.Slides

Aspose.Slides สำหรับ .NET มี API ที่ครอบคลุมสำหรับการสร้างและจัดการการนำเสนอ PowerPoint ด้วยโปรแกรม ตัวอย่างนี้แสดงวิธีการสร้างรูปร่างกลุ่ม เพิ่มรูปร่างสี่เหลี่ยมผืนผ้าหลาย ๆ รูปภายในรูปร่างนั้น จัดตำแหน่งรูปร่างให้เหมาะสม เพิ่มกรอบ และบันทึกการนำเสนอ รูปร่างกลุ่มมีความจำเป็นสำหรับการจัดระเบียบรูปร่างที่เกี่ยวข้องซึ่งจำเป็นต้องจัดการเป็นหน่วยเดียว ซึ่งจะทำให้มีโครงสร้างและการควบคุมองค์ประกอบการนำเสนอของคุณมากขึ้น

## คำแนะนำในการติดตั้ง

1. ติดตั้ง Aspose.Slides สำหรับ .NET โดยใช้หนึ่งในวิธีต่อไปนี้:

   - ตัวจัดการแพ็กเกจ NuGet:
     ```
     Install-Package Aspose.Slides.NET
     ```

   - คอนโซลตัวจัดการแพ็คเกจ:
     ```
     PM> Install-Package Aspose.Slides.NET
     ```

   - .NET CLI:
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. เพิ่มการอ้างอิงไปยังโครงการของคุณ:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

3. ตั้งค่าใบอนุญาต (ทางเลือกสำหรับการใช้งานการผลิต):
   ```csharp
   // กำหนดใบอนุญาตเพื่อลบข้อจำกัดการประเมิน
   License license = new License();
   license.SetLicense("Aspose.Slides.lic");
   ```

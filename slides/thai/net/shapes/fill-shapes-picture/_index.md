# การสร้างรูปทรงด้วยการเติมภาพแบบเรียงต่อกันใน PowerPoint โดยใช้ Aspose.Slides

Aspose.Slides สำหรับ .NET มอบความสามารถอันทรงพลังสำหรับการสร้างและจัดการการนำเสนอ PowerPoint ด้วยโปรแกรม ตัวอย่างนี้แสดงวิธีการสร้างรูปสี่เหลี่ยมผืนผ้าและเติมด้วยรูปแบบภาพแบบเรียงต่อกัน การเติมภาพแบบเรียงต่อกันนั้นมีประโยชน์อย่างยิ่งสำหรับการสร้างพื้นหลังที่มีพื้นผิว ลายน้ำ หรือองค์ประกอบตกแต่งภายในงานนำเสนอของคุณ

## คำแนะนำในการติดตั้ง

1. ติดตั้งผ่านตัวจัดการแพ็กเกจ NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. ติดตั้งผ่าน .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. การติดตั้งด้วยตนเอง:
   - ดาวน์โหลดเวอร์ชันล่าสุดได้จาก [Aspose.Slides สำหรับ .NET](https://releases.aspose.com/slides/net/)
   - เพิ่มการอ้างอิงถึง Aspose.Slides.dll ในโครงการของคุณ

# การสร้างรูปทรงสี่เหลี่ยมผืนผ้าด้วยการจัดรูปแบบเส้นแบบกำหนดเองใน PowerPoint

Aspose.Slides สำหรับ .NET มอบความสามารถอันทรงพลังสำหรับการสร้างและปรับเปลี่ยนการนำเสนอ PowerPoint ด้วยโปรแกรม ตัวอย่างนี้แสดงวิธีการสร้างรูปสี่เหลี่ยมผืนผ้าพร้อมการจัดรูปแบบเส้นแบบกำหนดเอง รวมถึงการกำหนดสไตล์เส้น ความกว้าง รูปแบบเส้นประ และสี โค้ดตัวอย่างจะสร้างการนำเสนอใหม่ เพิ่มรูปสี่เหลี่ยมผืนผ้าในสไลด์แรก และใช้การจัดรูปแบบเฉพาะกับทั้งการเติมและโครงร่างของรูปร่าง ส่งผลให้ได้องค์ประกอบที่ดึงดูดสายตาด้วยการเติมสีขาวและโครงร่างเส้นประสีน้ำเงิน

## คำแนะนำในการติดตั้ง

1. ติดตั้ง Aspose.Slides สำหรับ .NET โดยใช้หนึ่งในวิธีต่อไปนี้:

   - ตัวจัดการแพ็กเกจ NuGet:
     ```
     Install-Package Aspose.Slides
     ```

   - คอนโซลตัวจัดการแพ็คเกจ:
     ```
     PM> Install-Package Aspose.Slides
     ```

   - .NET CLI:
     ```
     dotnet add package Aspose.Slides
     ```

2. เพิ่มเนมสเปซที่จำเป็นลงในไฟล์ C# ของคุณ:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

# การเพิ่มส่วนต่างๆ ลงใน GeometryShape ในงานนำเสนอ PowerPoint

ตัวอย่างนี้สาธิตวิธีการเข้าถึงและปรับเปลี่ยนเส้นทางเรขาคณิตของรูปร่างในงานนำเสนอ PowerPoint โดยใช้ Aspose.Slides สำหรับ .NET การเพิ่มเส้นส่วนที่กำหนดเองลงในรูปร่างสี่เหลี่ยมผืนผ้าจะช่วยให้คุณสร้างรูปทรงเรขาคณิตที่ซับซ้อนมากขึ้นซึ่งไม่สามารถใช้ได้ในประเภทรูปร่างมาตรฐาน เทคนิคนี้มีประโยชน์สำหรับการสร้างไดอะแกรม แผนผังกระแสข้อมูล และองค์ประกอบภาพอื่นๆ ที่กำหนดเองในงานนำเสนอของคุณ

## คำแนะนำในการติดตั้ง

1. ติดตั้ง Aspose.Slides สำหรับ .NET โดยใช้หนึ่งในวิธีต่อไปนี้:

   - ตัวจัดการแพ็กเกจ NuGet:
     ```
     Install-Package Aspose.Slides.NET
     ```

   - คอนโซลตัวจัดการแพ็คเกจ:
     ```
     PM> Install-Package Aspose.Slides.NET
     ```

   - .NET CLI:
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. เพิ่มเนมสเปซลงในไฟล์ C# ของคุณ:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

# การจัดตำแหน่งและการจัดกลุ่มรูปร่างในงานนำเสนอ PowerPoint

Aspose.Slides สำหรับ .NET มอบความสามารถอันทรงพลังสำหรับการสร้างและจัดการการนำเสนอ PowerPoint ด้วยโปรแกรม ตัวอย่างนี้แสดงวิธีการจัดตำแหน่งรูปร่างภายในสไลด์และการทำงานกับรูปร่างที่จัดกลุ่มในการนำเสนอ PowerPoint คุณจะได้เรียนรู้วิธีการจัดตำแหน่งรูปสี่เหลี่ยมผืนผ้าในพิกัดที่กำหนด การจัดตำแหน่งรูปสี่เหลี่ยมผืนผ้าให้สัมพันธ์กับสไลด์หรือระหว่างกัน และการจัดตำแหน่งรูปร่างภายในกลุ่มอย่างเลือกสรร

## คำแนะนำในการติดตั้ง

1. ติดตั้ง Aspose.Slides สำหรับ .NET โดยใช้หนึ่งในวิธีต่อไปนี้:
   
   - ตัวจัดการแพ็กเกจ NuGet:
     ```
     Install-Package Aspose.Slides.NET
     ```
   
   - คอนโซลตัวจัดการแพ็คเกจ:
     ```
     PM> Install-Package Aspose.Slides.NET
     ```
   
   - .NET CLI:
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. เพิ่มการอ้างอิงไปยังโครงการของคุณ:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.IO;
   ```

3. ดาวน์โหลดโค้ดตัวอย่างด้านล่างและรวมเข้ากับโครงการของคุณ

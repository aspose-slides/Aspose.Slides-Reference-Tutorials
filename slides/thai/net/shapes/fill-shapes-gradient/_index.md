# การสร้างรูปทรงวงรีแบบเติมไล่ระดับสีใน PowerPoint ด้วย Aspose.Slides

ตัวอย่างนี้สาธิตวิธีการสร้างงานนำเสนอ PowerPoint ที่มีรูปร่างวงรีพร้อมเอฟเฟกต์การเติมแบบไล่ระดับแบบกำหนดเองด้วยโปรแกรม โค้ดนี้จะสร้างงานนำเสนอใหม่ เพิ่มวงรีในสไลด์แรก และใช้การเติมแบบไล่ระดับเชิงเส้นพร้อมจุดหยุดสีแดงและม่วง Aspose.Slides สำหรับ .NET มอบ API ที่ครอบคลุมสำหรับการสร้างและจัดการงานนำเสนอ PowerPoint พร้อมการควบคุมที่แม่นยำสำหรับรูปร่าง การจัดรูปแบบ และเอฟเฟกต์

## คำแนะนำในการติดตั้ง

1. ติดตั้ง Aspose.Slides สำหรับ .NET โดยใช้หนึ่งในวิธีต่อไปนี้:
   - ตัวจัดการแพ็กเกจ NuGet: `Install-Package Aspose.Slides.NET`
   - .NET CLI: `dotnet add package Aspose.Slides.NET`
   - คอนโซลตัวจัดการแพ็คเกจ: `Install-Package Aspose.Slides.NET`

2. เพิ่มคำสั่ง using ต่อไปนี้ลงในไฟล์ C# ของคุณ:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

# การสร้างรูปทรงวงรีใน PowerPoint ด้วย Aspose.Slides

Aspose.Slides สำหรับ .NET เป็น API ที่มีประสิทธิภาพที่ช่วยให้นักพัฒนาสามารถสร้าง อ่าน เขียน แก้ไข และแปลงงานนำเสนอ PowerPoint ได้โดยไม่ต้องใช้ Microsoft PowerPoint ตัวอย่างนี้สาธิตวิธีการสร้างรูปร่างวงรีในงานนำเสนอ ใช้การจัดรูปแบบรวมทั้งการเติมสีและคุณสมบัติของเส้น และบันทึกผลลัพธ์เป็นไฟล์ PPTX โค้ดนี้แสดงให้เห็นความสามารถของ Aspose.Slides ในการจัดการรูปร่างด้วยการควบคุมที่แม่นยำสำหรับลักษณะที่ปรากฏและตำแหน่งของรูปร่าง

## คำแนะนำในการติดตั้ง

1. ติดตั้งผ่านตัวจัดการแพ็กเกจ NuGet:
   ```
   Install-Package Aspose.Slides
   ```

   หรือใช้ .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

2. เพิ่มการอ้างอิง:
   เพิ่มเนมสเปซต่อไปนี้ลงในไฟล์ C# ของคุณ:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

3. การตั้งค่าใบอนุญาต (ทางเลือก):
   หากคุณมีใบอนุญาต ให้ใช้ดังนี้:
   ```csharp
   License license = new License();
   license.SetLicense("Aspose.Slides.lic");
   ```

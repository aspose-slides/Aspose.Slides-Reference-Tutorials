# สร้างภาพขนาดย่อจากสไลด์ใน Notes


ในโลกของการนำเสนอแบบสมัยใหม่ เนื้อหาภาพถือเป็นราชา การสร้างสไลด์ที่น่าสนใจถือเป็นสิ่งสำคัญสำหรับการสื่อสารที่มีประสิทธิภาพ วิธีหนึ่งในการปรับปรุงการนำเสนอของคุณคือการสร้างภาพขนาดย่อจากสไลด์ โดยเฉพาะอย่างยิ่งเมื่อคุณต้องการเน้นรายละเอียดเฉพาะหรือแชร์ภาพรวม Aspose.Slides สำหรับ .NET เป็นเครื่องมืออันทรงพลังที่จะช่วยให้คุณทำสิ่งนี้ได้อย่างราบรื่น ในคู่มือทีละขั้นตอนนี้ เราจะแนะนำคุณเกี่ยวกับกระบวนการสร้างภาพขนาดย่อจากสไลด์ในส่วนบันทึกของการนำเสนอโดยใช้ Aspose.Slides สำหรับ .NET

## ข้อกำหนดเบื้องต้น

ก่อนที่เราจะเจาะลึกรายละเอียด คุณควรมีข้อกำหนดเบื้องต้นดังต่อไปนี้:

### 1. Aspose.Slides สำหรับ .NET

ตรวจสอบให้แน่ใจว่าคุณได้ติดตั้งและตั้งค่า Aspose.Slides สำหรับ .NET แล้ว คุณสามารถดาวน์โหลดได้จาก [ที่นี่](https://releases-aspose.com/slides/net/).

### 2. สภาพแวดล้อม .NET

คุณควรมีสภาพแวดล้อมการพัฒนา .NET ที่พร้อมใช้งานบนระบบของคุณ

### 3. ไฟล์นำเสนอ

มีไฟล์นำเสนอ (เช่น `ThumbnailFromSlideInNotes.pptx`) ที่คุณต้องการสร้างภาพขนาดย่อ

ตอนนี้เรามาแบ่งกระบวนการออกเป็นขั้นตอน:

## ขั้นตอนที่ 1: นำเข้าเนมสเปซ

ขั้นแรก คุณต้องนำเข้าเนมสเปซที่จำเป็นสำหรับการใช้งาน Aspose.Slides เพิ่มโค้ดต่อไปนี้ที่จุดเริ่มต้นของสคริปต์ C# ของคุณ:

```csharp
using Aspose.Slides;
using System.Drawing;
```

## ขั้นตอนที่ 2: โหลดงานนำเสนอ

ต่อไป คุณจะต้องโหลดไฟล์การนำเสนอที่มีสไลด์พร้อมหมายเหตุ ใช้โค้ดต่อไปนี้เพื่อสร้างอินสแตนซ์ `Presentation` ระดับ:

```csharp
string dataDir = "Your Document Directory";

using (Presentation pres = new Presentation(dataDir + "ThumbnailFromSlideInNotes.pptx"))
{
    // รหัสของคุณอยู่ที่นี่
}
```

## ขั้นตอนที่ 3: เข้าถึงสไลด์

คุณสามารถเลือกสไลด์ที่ต้องการสร้างภาพขนาดย่อในงานนำเสนอได้ ในตัวอย่างนี้ เราจะเข้าถึงสไลด์แรก:

```csharp
ISlide sld = pres.Slides[0];
```

## ขั้นตอนที่ 4: กำหนดมิติที่ต้องการ

ระบุขนาด (ความกว้างและความสูง) สำหรับภาพขนาดย่อที่คุณต้องการสร้าง ตัวอย่างเช่น:

```csharp
int desiredX = 1200; // ความกว้าง
int desiredY = 800;  // ความสูง
```

## ขั้นตอนที่ 5: คำนวณปัจจัยการปรับขนาด

เพื่อให้แน่ใจว่าภาพขนาดย่อพอดีกับขนาดที่ต้องการ ให้คำนวณปัจจัยการปรับขนาดดังต่อไปนี้:

```csharp
float ScaleX = (float)(1.0 / pres.SlideSize.Size.Width) * desiredX;
float ScaleY = (float)(1.0 / pres.SlideSize.Size.Height) * desiredY;
```

## ขั้นตอนที่ 6: สร้างภาพขนาดย่อ

ตอนนี้ ให้สร้างภาพขนาดย่อขนาดเต็มโดยใช้ปัจจัยการปรับขนาดที่คำนวณแล้ว:

```csharp
Bitmap bmp = sld.GetThumbnail(ScaleX, ScaleY);
```

## ขั้นตอนที่ 7: บันทึกภาพขนาดย่อ

สุดท้ายให้บันทึกภาพขนาดย่อที่สร้างขึ้นเป็นภาพ JPEG:

```csharp
bmp.Save(dataDir + "Notes_tnail_out.jpg", System.Drawing.Imaging.ImageFormat.Jpeg);
```

เสร็จเรียบร้อย! คุณได้สร้างภาพขนาดย่อจากสไลด์ในส่วนบันทึกของงานนำเสนอของคุณสำเร็จแล้วโดยใช้ Aspose.Slides สำหรับ .NET

## บทสรุป

การนำภาพขนาดย่อมาใช้กับงานนำเสนอของคุณจะช่วยปรับปรุงความสวยงามและประสิทธิภาพของงานนำเสนอได้อย่างมาก Aspose.Slides สำหรับ .NET ช่วยให้กระบวนการนี้ง่ายขึ้น ช่วยให้คุณสามารถสร้างภาพขนาดย่อที่กำหนดเองจากสไลด์ของคุณได้อย่างง่ายดาย

ใช่ คุณสามารถวนซ้ำสไลด์ในงานนำเสนอของคุณและสร้างภาพขนาดย่อสำหรับแต่ละสไลด์ได้

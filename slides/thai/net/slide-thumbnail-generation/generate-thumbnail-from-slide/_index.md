## สร้างภาพขนาดย่อจากสไลด์

ในส่วนนี้ เราจะแบ่งกระบวนการสร้างภาพขนาดย่อจากสไลด์ออกเป็นหลายขั้นตอน

### ขั้นตอนที่ 1: กำหนดไดเร็กทอรีเอกสาร

 คุณควรระบุไดเร็กทอรีซึ่งเป็นที่ตั้งของไฟล์งานนำเสนอของคุณ แทนที่`"Your Document Directory"` กับเส้นทางที่แท้จริง

```csharp
string dataDir = "Your Document Directory";
```

### ขั้นตอนที่ 2: เปิดการนำเสนอ

 ใช้`Presentation` คลาสเพื่อเปิดงานนำเสนอ PowerPoint ของคุณ ตรวจสอบให้แน่ใจว่าคุณมีเส้นทางไฟล์ที่ถูกต้อง

```csharp
using (Presentation pres = new Presentation(dataDir + "ThumbnailFromSlide.pptx"))
{
    // เข้าถึงสไลด์แรก
    ISlide sld = pres.Slides[0];

    // สร้างภาพขนาดเต็ม
    Bitmap bmp = sld.GetThumbnail(1f, 1f);

    // บันทึกภาพลงดิสก์ในรูปแบบ JPEG
    bmp.Save(dataDir + "Thumbnail_out.jpg", System.Drawing.Imaging.ImageFormat.Jpeg);
}
```

ต่อไปนี้เป็นคำอธิบายสั้นๆ ว่าแต่ละขั้นตอนทำหน้าที่อะไรบ้าง:

1.  คุณเปิดงานนำเสนอ PowerPoint ของคุณโดยใช้ไฟล์`Presentation` ระดับ.
2.  คุณเข้าถึงสไลด์แรกโดยใช้`ISlide` อินเตอร์เฟซ.
3.  คุณสร้างภาพขนาดเต็มของสไลด์โดยใช้`GetThumbnail` วิธี.
4. คุณบันทึกรูปภาพที่สร้างขึ้นไปยังไดเร็กทอรีที่คุณระบุในรูปแบบ JPEG

แค่นั้นแหละ! คุณสร้างภาพขนาดย่อจากสไลด์ได้สำเร็จโดยใช้ Aspose.Slides สำหรับ .NET

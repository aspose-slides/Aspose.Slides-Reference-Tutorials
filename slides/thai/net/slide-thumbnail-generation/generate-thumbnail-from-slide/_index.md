เปิดโครงการ .NET ของคุณใน Visual Studio

### ขั้นตอนที่ 2: เพิ่มการใช้คำสั่ง

ในไฟล์โค้ดที่คุณวางแผนจะทำงานกับ Aspose.Slides ให้เพิ่ม using directives ดังต่อไปนี้:

```csharp
using Aspose.Slides;
```

## สร้างภาพขนาดย่อจากสไลด์

ในส่วนนี้ เราจะแบ่งขั้นตอนการสร้างภาพขนาดย่อจากสไลด์ออกเป็นหลายขั้นตอน

### ขั้นตอนที่ 1: กำหนดไดเรกทอรีเอกสาร

คุณควรระบุไดเรกทอรีที่ไฟล์การนำเสนอของคุณตั้งอยู่ แทนที่ `"Your Document Directory"` ด้วยเส้นทางที่แท้จริง

```csharp
string dataDir = "Your Document Directory";
```

### ขั้นตอนที่ 2: เปิดการนำเสนอ

ใช้ `Presentation` เพื่อเปิดงานนำเสนอ PowerPoint ของคุณ ตรวจสอบให้แน่ใจว่าคุณมีเส้นทางไฟล์ที่ถูกต้อง

```csharp
using (Presentation pres = new Presentation(dataDir + "ThumbnailFromSlide.pptx"))
{
    // เข้าถึงสไลด์แรก
    ISlide sld = pres.Slides[0];

    // สร้างภาพขนาดเต็ม
    Bitmap bmp = sld.GetThumbnail(1f, 1f);

    // บันทึกภาพลงในดิสก์ในรูปแบบ JPEG
    bmp.Save(dataDir + "Thumbnail_out.jpg", System.Drawing.Imaging.ImageFormat.Jpeg);
}
```

นี่คือคำอธิบายสั้นๆ เกี่ยวกับสิ่งที่แต่ละขั้นตอนทำ:

1. คุณเปิดการนำเสนอ PowerPoint ของคุณโดยใช้ `Presentation` ระดับ.
2. คุณเข้าถึงสไลด์แรกโดยใช้ `ISlide` อินเทอร์เฟซ
3. คุณสามารถสร้างภาพสไลด์ขนาดเต็มได้โดยใช้ `GetThumbnail` วิธี.
4. คุณบันทึกภาพที่สร้างขึ้นไปยังไดเร็กทอรีที่คุณระบุในรูปแบบ JPEG

เสร็จเรียบร้อย! คุณได้สร้างภาพขนาดย่อจากสไลด์โดยใช้ Aspose.Slides สำหรับ .NET สำเร็จแล้ว

## คู่มือการใช้งาน

### การแยกวัตถุ Flash จากสไลด์ PowerPoint

สำรวจวิธีการแยกวัตถุแฟลชที่ชื่อ `ShockwaveFlash1` จากสไลด์แรกของการนำเสนอ

#### การโหลดไฟล์นำเสนอ

เริ่มต้นด้วยการโหลดไฟล์ PowerPoint ของคุณ:

```csharp
using Aspose.Slides;

string dataDir = "YOUR_DOCUMENT_DIRECTORY/withFlash.pptm";

// โหลดงานนำเสนอ
class Program
{
    static void Main(string[] args)
    {
        using (Presentation pres = new Presentation(dataDir))
        {
            // การควบคุมการเข้าถึงบนสไลด์แรก
            IControlCollection controls = pres.Slides[0].Controls;
            
            Control flashControl = null; // ตัวแปรสำหรับเก็บการควบคุมแฟลช
            
            foreach (IControl control in controls)
            {
                if (control.Name == "ShockwaveFlash1")
                {
                    // แคสต์และจัดเก็บการควบคุมแฟลช
                    flashControl = (Control)control;
                }
            }
        }
    }
}
```

**จุดสำคัญ:**
- **การเข้าถึงการควบคุม:** `pres.Slides[0].Controls` ให้การเข้าถึงการควบคุมทั้งหมดในสไลด์แรก
- **การวนซ้ำผ่านการควบคุม:** ทำซ้ำผ่านการควบคุมแต่ละรายการและตรวจสอบชื่อโดยใช้คำสั่ง if

#### เคล็ดลับการแก้ไขปัญหา

- ตรวจสอบให้แน่ใจว่าไฟล์ PowerPoint ของคุณมีชื่อถูกต้องและอยู่ในไดเร็กทอรีที่ระบุ
- ตรวจสอบว่าชื่อของวัตถุแฟลชตรงกันทุกประการ (`ShockwaveFlash1`-

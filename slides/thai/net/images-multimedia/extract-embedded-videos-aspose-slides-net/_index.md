## คู่มือการใช้งาน
### การแยกวิดีโอที่ฝังไว้จาก PowerPoint
ฟีเจอร์นี้ช่วยให้คุณแยกวิดีโอที่ฝังอยู่ในสไลด์ PowerPoint ออกมาได้ มาดูขั้นตอนต่างๆ กัน:
#### ภาพรวมของคุณสมบัติ
เราจะทำซ้ำผ่านแต่ละสไลด์และรูปร่าง ตรวจสอบเฟรมวิดีโอ จากนั้นแยกและบันทึกวิดีโอ
#### การดำเนินการแบบทีละขั้นตอน
##### 1. โหลดงานนำเสนอ
เริ่มต้นด้วยการโหลดไฟล์งานนำเสนอโดยใช้ Aspose.Slides
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
Presentation presentation = new Presentation(dataDir + "/Video.pptx");
```
##### 2. ทำซ้ำสไลด์และรูปร่าง
วนซ้ำผ่านแต่ละสไลด์ จากนั้นผ่านแต่ละรูปร่างภายในสไลด์เพื่อค้นหาเฟรมวิดีโอ
```csharp
foreach (ISlide slide in presentation.Slides) {
    foreach (IShape shape in slide.Shapes) {
        if (shape is VideoFrame) {
            // ประมวลผลเฟรมวิดีโอ
        }
    }
}
```
##### 3. ระบุและแยกวิดีโอ
ตรวจสอบว่ารูปร่างเป็น `VideoFrame`ดึงเนื้อหาออกมาแล้วบันทึกไว้
```csharp
if (shape is VideoFrame vf) {
    String type = vf.EmbeddedVideo.ContentType;
    int ss = type.LastIndexOf('/');
    type = type.Remove(0, ss + 1);
    Byte[] buffer = vf.EmbeddedVideo.BinaryData;

    using (FileStream stream = new FileStream("YOUR_OUTPUT_DIRECTORY/NewVideo_out." + type, FileMode.Create, FileAccess.Write, FileShare.Read)) {
        stream.Write(buffer, 0, buffer.Length);
    }
}
```
**คำอธิบาย:**
- **ประเภทเนื้อหา**: กำหนดนามสกุลไฟล์ของวิดีโอ
- **ข้อมูลไบนารี**: ประกอบด้วยข้อมูลวิดีโอแบบดิบสำหรับการแยกข้อมูล
##### เคล็ดลับการแก้ไขปัญหา
- ตรวจสอบให้แน่ใจว่าเส้นทางไดเร็กทอรีของคุณได้รับการตั้งค่าอย่างถูกต้องเพื่อหลีกเลี่ยง `FileNotFoundException`-
- หากไม่สามารถแยกวิดีโอออกมาได้ ให้ตรวจสอบว่ามีรูปร่างจริง `VideoFrame` อินสแตนซ์

3. **การรายงานอัตโนมัติ**:สร้างรายงานที่มีบทสรุปเป็นวิดีโอ

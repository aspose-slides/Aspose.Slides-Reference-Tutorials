## คู่มือการใช้งาน
หัวข้อนี้จะแนะนำคุณเกี่ยวกับการเพิ่มคำบรรยายลงในเฟรมวิดีโอและการแยกหรือการลบคำบรรยาย ฟีเจอร์แต่ละอย่างจะอธิบายโดยละเอียดด้านล่าง

### คุณสมบัติ 1: เพิ่มคำบรรยายลงในวิดีโอเฟรม

#### ภาพรวม
ฟีเจอร์นี้ช่วยแทรกคำบรรยายจากไฟล์ภายนอก (เช่น VTT) ลงในเฟรมวิดีโอ เพิ่มการเข้าถึงให้กับผู้ชมของคุณ

#### ขั้นตอนการดำเนินการ
**ขั้นตอนที่ 1: เตรียมไฟล์ของคุณ**
ให้แน่ใจว่าคุณมีวิดีโอ (`sample_bunny.mp4`) และไฟล์แทร็กคำบรรยาย (`bunny.vtt`-

```csharp
string mediaFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample_bunny.mp4");
string trackFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "bunny.vtt");
```

**ขั้นตอนที่ 2: เพิ่มวิดีโอลงในงานนำเสนอ**
สร้าง `Presentation` วัตถุและเพิ่มวิดีโอของคุณ

```csharp
using (Presentation pres = new Presentation())
{
    IVideo video = pres.Videos.AddVideo(File.ReadAllBytes(mediaFile));
    var videoFrame = pres.Slides[0].Shapes.AddVideoFrame(0, 0, 100, 100, video);
```

**ขั้นตอนที่ 3: เพิ่มแทร็กคำบรรยาย**
แนบไฟล์แทร็กคำบรรยายของคุณเข้ากับเฟรมวิดีโอ

```csharp
videoFrame.CaptionTracks.Add("New track", trackFile);
pres.Save(Path.Combine("YOUR_OUTPUT_DIRECTORY", "VideoCaptionAdd_out.pptx"), SaveFormat.Pptx);
}
```

#### พารามิเตอร์และวัตถุประสงค์ของวิธีการ
- `Presentation`: หมายถึงการนำเสนอ PowerPoint
- `IVideo` และ `IVideoFrame`:แสดงเนื้อหาวิดีโอและเฟรมภายในสไลด์ตามลำดับ
- `captionTracks.Add()`: เพิ่มคำบรรยายให้กับแทร็กที่ระบุ

### คุณสมบัติ 2: แยกและลบคำบรรยายจาก VideoFrame

#### ภาพรวม
เมื่อคุณเพิ่มคำบรรยายแล้ว อาจมีสถานการณ์ที่คุณต้องแยกหรือลบคำบรรยายออก คุณลักษณะนี้เน้นที่วิธีการบรรลุทั้งสองงานอย่างมีประสิทธิภาพ

#### ขั้นตอนการดำเนินการ
**ขั้นตอนที่ 1: โหลดการนำเสนอ**
เปิดการนำเสนอที่มีวิดีโอของคุณพร้อมคำบรรยาย

```csharp
string outAddPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "VideoCaptionAdd_out.pptx");
using (Presentation pres = new Presentation(outAddPath))
{
    IVideoFrame videoFrame = pres.Slides[0].Shapes[0] as VideoFrame;
```

**ขั้นตอนที่ 2: แยกคำอธิบายภาพ**
แยกข้อมูลไบนารีของคำบรรยายและบันทึกลงในไฟล์

```csharp
if (videoFrame != null)
{
    foreach (var captionTrack in videoFrame.CaptionTracks) 
    {
        File.WriteAllBytes(Path.Combine("YOUR_OUTPUT_DIRECTORY", "Caption_out.vtt"), captionTrack.BinaryData);
    }
```

**ขั้นตอนที่ 3: ลบคำบรรยาย**
ล้างคำบรรยายทั้งหมดจาก VideoFrame

```csharp
videoFrame.CaptionTracks.Clear();
pres.Save(Path.Combine("YOUR_OUTPUT_DIRECTORY", "VideoCaptionRemove_out.pptx"), SaveFormat.Pptx);
}
```

#### พารามิเตอร์และวัตถุประสงค์ของวิธีการ
- `BinaryData`: แสดงข้อมูลคำบรรยายในรูปแบบไบนารี
- `CaptionTracks.Clear()`: ลบคำบรรยายทั้งหมดจากเฟรมวิดีโอ

## การประยุกต์ใช้งานจริง
การใส่คำบรรยายวิดีโอสามารถเพิ่มประสิทธิภาพการนำเสนอของคุณได้อย่างมาก ต่อไปนี้คือการใช้งานจริงบางส่วน:

1. **เนื้อหาการศึกษา**:ปรับปรุงความเข้าใจสำหรับนักเรียนที่มีความบกพร่องทางการได้ยินหรือผู้ที่กำลังเรียนภาษาที่สอง
2. **การฝึกอบรมองค์กร**:รับรองความชัดเจนและการเก็บรักษาข้อมูลในทีมที่มีความหลากหลาย
3. **การประชุมนานาชาติ**:รองรับผู้ที่ไม่ใช่เจ้าของภาษาโดยให้คำบรรยายเป็นภาษาท้องถิ่น
4. **การกระจายเสียงสาธารณะ**:เพิ่มการเข้าถึงให้กับผู้คนจำนวนมากขึ้น รวมถึงผู้พิการทางการได้ยิน

## การพิจารณาประสิทธิภาพ
เมื่อทำงานกับข้อมูลวิดีโอใน .NET โดยใช้ Aspose.Slides:
- **เพิ่มประสิทธิภาพการใช้หน่วยความจำ**:จัดการหน่วยความจำอย่างมีประสิทธิภาพด้วยการกำจัดทรัพยากรทันทีหลังการใช้งาน
- **ปรับปรุงการดำเนินการ I/O**:ลดการดำเนินการอ่าน/เขียนไฟล์ให้เหลือน้อยที่สุดเพื่อเพิ่มประสิทธิภาพ
- **แนวทางปฏิบัติที่ดีที่สุดสำหรับการจัดการหน่วยความจำ .NET**: ใช้ประโยชน์ `using` คำสั่งและให้แน่ใจว่าวัตถุจะถูกยกเลิกการอ้างอิงเมื่อไม่จำเป็นอีกต่อไป

## คู่มือการใช้งาน
### แยกเสียงจากไทม์ไลน์ของสไลด์
ฟีเจอร์นี้ช่วยให้คุณค้นหาไฟล์เสียงที่ฝังอยู่ในแอนิเมชั่นสไลด์ของงานนำเสนอ PowerPoint ได้ โดยคุณสามารถนำไปใช้งานดังนี้:
#### ขั้นตอนที่ 1: กำหนดเส้นทางไฟล์
เริ่มต้นด้วยการกำหนดเส้นทางสำหรับไฟล์อินพุตและเอาต์พุตของคุณโดยใช้ตัวแทน
```csharp
string pptxFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "AnimationAudio.pptx");
string outMediaPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "MediaTimeline.mpg");
```
#### ขั้นตอนที่ 2: โหลดงานนำเสนอ
โหลดไฟล์ PowerPoint ของคุณเพื่อเข้าถึงเนื้อหา
```csharp
using (Presentation pres = new Presentation(pptxFile))
{
    // โค้ดยังคงดำเนินต่อไป...
}
```
#### ขั้นตอนที่ 3: เข้าถึงสไลด์และไทม์ไลน์
เข้าถึงสไลด์แรกและดึงลำดับแอนิเมชั่นหลัก
```csharp
ISlide slide = pres.Slides[0];
ISequence effectsSequence = slide.Timeline.MainSequence;
```
#### ขั้นตอนที่ 4: แยกข้อมูลเสียง
แยกข้อมูลไบนารีของเอฟเฟกต์เสียงที่เชื่อมโยงกับเอฟเฟกต์แอนิเมชันแรก
```csharp
byte[] audio = effectsSequence[0].Sound.BinaryData;
```
#### ขั้นตอนที่ 5: บันทึกเสียงลงในไฟล์
เขียนข้อมูลเสียงที่แยกออกมาไปยังไฟล์ที่เส้นทางเอาต์พุตที่คุณระบุ
```csharp
File.WriteAllBytes(outMediaPath, audio);
```

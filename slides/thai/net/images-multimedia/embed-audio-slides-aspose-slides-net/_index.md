## คู่มือการใช้งาน

หัวข้อนี้จะอธิบายวิธีการเพิ่มเฟรมเสียงพร้อมเสียงที่ฝังไว้ในงานนำเสนอ

### การเพิ่มเฟรมเสียง

#### ภาพรวม

การฝังเสียงสามารถเพิ่มการโต้ตอบของการนำเสนอของคุณ ทำให้น่าสนใจยิ่งขึ้น เราจะแนะนำวิธีสร้างและฝังไฟล์เสียงในสไลด์โดยใช้ Aspose.Slides สำหรับ .NET

#### การดำเนินการแบบทีละขั้นตอน

##### 1. โหลดหรือสร้างงานนำเสนอ

เริ่มต้นด้วยการโหลดงานนำเสนอที่มีอยู่หรือสร้างงานนำเสนอใหม่:

```csharp
// สร้างการนำเสนอใหม่หรือโหลดการนำเสนอที่มีอยู่
Presentation pres = new Presentation();
```

##### 2. เข้าถึงสไลด์

เลือกสไลด์ที่คุณต้องการฝังเสียง:

```csharp
ISlide slide = pres.Slides[0]; // เข้าถึงสไลด์แรก
```

##### 3. เพิ่มเฟรมเสียง

วิธีการเพิ่มเฟรมเสียงพร้อมเสียงที่ฝังไว้มีดังนี้:

```csharp
// กำหนดเส้นทางสำหรับสื่ออินพุตและไฟล์เอาท์พุต
string mediaFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "audio.mp3");

// โหลดไฟล์เสียงลงใน FileStream
using (FileStream fs = new FileStream(mediaFile, FileMode.Open))
{
    // เพิ่มเฟรมเสียงลงในสไลด์
    IAudioFrame audioFrame = slide.Shapes.AddAudioFrameEmbedded(50, 150, 100, 100, fs);
    
    // กำหนดค่าคุณสมบัติเสียงหากจำเป็น
    audioFrame.PlayMode = AudioPlayModePreset.OnClick;
}
```

**คำอธิบาย:**
- **เพิ่ม AudioFrameEmbedded**วิธีการนี้จะเพิ่มเฟรมเสียงลงในสไลด์ พารามิเตอร์จะกำหนดตำแหน่งและขนาดของเฟรมบนสไลด์
- **โหมดการเล่น**: กำหนดค่าวิธีการเล่นเสียง เช่น เริ่มอัตโนมัติหรือเมื่อคลิก

#### เคล็ดลับการแก้ไขปัญหา

- ตรวจสอบให้แน่ใจว่าเส้นทางไฟล์สื่อถูกต้องและสามารถเข้าถึงได้
- ตรวจสอบข้อยกเว้นใดๆ ที่เกี่ยวข้องกับการดำเนินการ I/O ของไฟล์และจัดการอย่างเหมาะสม

## คู่มือการใช้งาน

ตอนนี้คุณได้ตั้งค่าสภาพแวดล้อมของคุณเรียบร้อยแล้ว ให้เราลองใช้งานฟังก์ชันการโหลดและการแสดงภาพดู

### คุณสมบัติ: การโหลดและการแสดงรูปภาพในงานนำเสนอ

ฟีเจอร์นี้สาธิตวิธีโหลดรูปภาพจากระบบไฟล์และเพิ่มเป็นกรอบรูปในสไลด์แรกของการนำเสนอโดยใช้ Aspose.Slides สำหรับ .NET

#### ภาพรวม:
ในส่วนนี้เราจะดูขั้นตอนในการโหลดรูปภาพ แทรกรูปภาพในสไลด์ และบันทึกงานนำเสนอของคุณ

**ขั้นตอนที่ 1: สร้างไดเรกทอรี**
กำหนดเส้นทางสำหรับไดเรกทอรีเอกสารและไดเรกทอรีเอาต์พุต หากไม่มี ให้สร้างเส้นทางโดยใช้:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // กำหนดเส้นทางไดเรกทอรีเอกสารของคุณที่นี่
string outputDir = "YOUR_OUTPUT_DIRECTORY"; // กำหนดเส้นทางไดเร็กทอรีเอาท์พุตของคุณที่นี่

// สร้างไดเร็กทอรีข้อมูลถ้ายังไม่มีอยู่
if (!Directory.Exists(dataDir))
{
    Directory.CreateDirectory(dataDir);
}
```

**ขั้นตอนที่ 2: โหลดและแทรกภาพ**
สร้างอินสแตนซ์การนำเสนอใหม่และเข้าถึงสไลด์แรก จากนั้นโหลดรูปภาพจากระบบไฟล์:
```csharp
using (Presentation pres = new Presentation())
{
    // เข้าถึงสไลด์แรกในการนำเสนอ
    ISlide sld = pres.Slides[0];

    // โหลดรูปภาพจากระบบไฟล์และเพิ่มลงในคอลเล็กชั่นรูปภาพของงานนำเสนอ
    IImage img = Images.FromFile(Path.Combine(dataDir, "aspose-logo.jpg"));
    IPPImage imgx = pres.Images.AddImage(img);

    // เพิ่มกรอบรูปให้มีขนาดตรงกับภาพที่โหลดมา
    sld.Shapes.AddPictureFrame(ShapeType.Rectangle, 50, 150, imgx.Width, imgx.Height, imgx);
}
```

**ขั้นตอนที่ 3: บันทึกการนำเสนอ**
สุดท้ายให้บันทึกงานนำเสนอที่คุณแก้ไขแล้วลงในดิสก์ในรูปแบบ PPTX:
```csharp
pres.Save(Path.Combine(outputDir, "AddStretchOffsetForImageFill_out.pptx"), SaveFormat.Pptx);
```

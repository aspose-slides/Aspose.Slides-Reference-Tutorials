## คู่มือการใช้งาน

มาดูการตั้งค่าไฮเปอร์ลิงก์มาโครบนรูปร่างกัน

### ภาพรวมคุณลักษณะ: การตั้งค่าไฮเปอร์ลิงก์มาโคร

ฟีเจอร์นี้ช่วยให้คุณสามารถแนบฟังก์ชันแมโครกับรูปร่างใน PowerPoint โดยใช้ Aspose.Slides สำหรับ .NET ซึ่งเหมาะอย่างยิ่งสำหรับการสร้างงานนำเสนอแบบโต้ตอบที่ตอบสนองต่ออินพุตของผู้ใช้

#### ขั้นตอนที่ 1: สร้างรูปร่าง

เพิ่มรูปร่างอัตโนมัติลงในสไลด์ของคุณ:

```csharp
using Aspose.Slides;

string macroName = "TestMacro";
using (Presentation presentation = new Presentation())
{
    // เพิ่มรูปร่างปุ่มว่างที่ตำแหน่ง (20, 20) พร้อมขนาด (80x30)
    IAutoShape shape = presentation.Slides[0].Shapes.AddAutoShape(ShapeType.BlankButton, 20, 20, 80, 30);
```

#### ขั้นตอนที่ 2: ตั้งค่าไฮเปอร์ลิงก์มาโคร

แนบแมโครกับรูปร่างนี้:

```csharp
    // เชื่อมโยงรูปร่างกับเหตุการณ์การคลิกไฮเปอร์ลิงก์มาโคร
    shape.HyperlinkManager.SetMacroHyperlinkClick(macroName);

    // บันทึกการนำเสนอ
    presentation.Save("output.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
}
```
**คำอธิบาย:**
- `AddAutoShape(ShapeType.BlankButton, 20, 20, 80, 30)`: เพิ่มรูปร่างปุ่มว่างเปล่าตามพิกัดและขนาดที่ระบุ
- `SetMacroHyperlinkClick(macroName)`:เชื่อมโยงแมโครกับเหตุการณ์คลิกรูปร่าง

#### เคล็ดลับการแก้ไขปัญหา

- **มาโครไม่ทำงาน:** ตรวจสอบให้แน่ใจว่ามีแมโครอยู่ในเทมเพลต PowerPoint ของคุณ
- **ปัญหาการวางตำแหน่งรูปร่าง:** ตรวจสอบค่าพิกัดซ้ำอีกครั้งเพื่อวางบนสไลด์อย่างแม่นยำ

## การประยุกต์ใช้งานจริง

การรวมแมโครกับรูปร่างสามารถใช้เพื่อวัตถุประสงค์ต่างๆ ได้ดังนี้:
1. **การป้อนข้อมูลอัตโนมัติ**:แมโครที่ทริกเกอร์ด้วยการคลิกปุ่มสามารถทำให้การทำงานซ้ำ ๆ เช่น การป้อนข้อมูลหรือการจัดรูปแบบเป็นไปโดยอัตโนมัติ
2. **แบบทดสอบแบบโต้ตอบ**:ใช้แมโครเพื่อนำทางระหว่างสไลด์ตามคำตอบแบบทดสอบ เพิ่มการมีส่วนร่วมของผู้ใช้
3. **การนำทางแบบกำหนดเอง**:สร้างปุ่มแบบกำหนดเองที่เรียกใช้การนำเสนอหรือส่วนต่างๆ ที่เฉพาะเจาะจงภายในสไลด์

## คู่มือการใช้งาน

### ภาพรวมคุณลักษณะ: สร้างแผนภูมิองค์กร

คุณลักษณะนี้ช่วยให้คุณสามารถสร้างงานนำเสนอด้วยแผนผังองค์กรโดยใช้เค้าโครงแผนผังองค์กรแบบรูปภาพใน Aspose.Slides

#### ขั้นตอนที่ 1: เริ่มต้นวัตถุการนำเสนอ

สร้างใหม่ `Presentation` วัตถุที่จะทำหน้าที่เป็นผืนผ้าใบของคุณในการเพิ่มรูปทรงและเนื้อหา:

```python
import aspose.slides as slides

def create_organization_chart():
    with slides.Presentation() as pres:
        # ขั้นตอนเพิ่มเติมจะเพิ่มไว้ที่นี่
```

#### ขั้นตอนที่ 2: เพิ่มรูปร่าง SmartArt ลงในสไลด์

ใช้ `PICTURE_ORGANIZATION_CHART` เค้าโครงสำหรับโครงสร้างองค์กรของคุณ:

```python
smart_art = pres.slides[0].shapes.add_smart_art(
    0,   # ตำแหน่ง x
    0,   # ตำแหน่ง y
    400, # ความกว้าง
    400, # ความสูง
    slides.smartart.SmartArtLayoutType.PICTURE_ORGANIZATION_CHART
)
```

**คำอธิบาย**:โค้ดนี้จะเพิ่มรูปร่าง SmartArt ลงในสไลด์แรกตามพิกัดที่กำหนดพร้อมขนาดที่กำหนดไว้ล่วงหน้า `SmartArtLayoutType` ถูกกำหนดไว้สำหรับการแสดงภาพข้อมูลแบบลำดับชั้น

#### ขั้นตอนที่ 3: บันทึกการนำเสนอ

บันทึกแผนผังองค์กรของคุณในรูปแบบ PPTX:

```python
pres.save("YOUR_OUTPUT_DIRECTORY/charts_organization_chart_out.pptx", slides.export.SaveFormat.PPTX)
```

**คำอธิบาย**: เดอะ `save` วิธีการเขียนการนำเสนอลงในไฟล์ แทนที่ `"YOUR_OUTPUT_DIRECTORY"` ตามเส้นทางที่คุณต้องการ

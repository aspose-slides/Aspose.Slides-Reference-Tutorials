## คู่มือการใช้งาน

มาแบ่งกระบวนการใช้งานออกเป็นขั้นตอนที่สามารถจัดการได้

### การสร้าง SmartArt ด้วยการเติมภาพแบบ Bullet Fill

#### ภาพรวม

ในส่วนนี้ คุณจะได้เรียนรู้วิธีการเพิ่มรูปร่าง SmartArt ลงในสไลด์และตั้งค่ารูปภาพเป็นรูปแบบการเติมหัวข้อย่อย

#### ขั้นตอนที่ 1: สร้างวัตถุการนำเสนอ

เริ่มต้นด้วยการสร้างวัตถุสำหรับการนำเสนอ ซึ่งจะเป็นผืนผ้าใบของคุณ:

```python
with slides.Presentation() as document:
    # โค้ดสำหรับเพิ่ม SmartArt อยู่ที่นี่
```

#### ขั้นตอนที่ 2: เพิ่มรูปร่าง SmartArt

เพิ่มรูปร่าง SmartArt ลงในสไลด์แรกของคุณในตำแหน่งและขนาดที่ต้องการ:

```python
smart = document.slides[0].shapes.add_smart_art(
    10, 10, 500, 400,
    slides.smartart.SmartArtLayoutType.VERTICAL_PICTURE_LIST
)
```

#### ขั้นตอนที่ 3: เข้าถึงโหนดแรก

เข้าถึงโหนดแรกเพื่อใช้การจัดรูปแบบภาพกระสุน:

```python
node = smart.all_nodes[0]
```

#### ขั้นตอนที่ 4: ตั้งค่ารูปแบบการเติมหัวข้อย่อย

ตรวจสอบว่ามีรูปแบบการเติมหัวข้อย่อยหรือไม่ และตั้งค่ารูปภาพเป็นหัวข้อย่อย:

```python
if node.bullet_fill_format is not None:
    img = slides.Images.from_file("YOUR_DOCUMENT_DIRECTORY/image1.jpg")
    image = document.images.add_image(img)

    node.bullet_fill_format.fill_type = slides.FillType.PICTURE
    node.bullet_fill_format.picture_fill_format.picture.image = image
    node.bullet_fill_format.picture_fill_format.picture_fill_mode = slides.PictureFillMode.STRETCH
```

#### ขั้นตอนที่ 5: บันทึกการนำเสนอ

สุดท้ายให้บันทึกการนำเสนอของคุณพร้อมกับการเปลี่ยนแปลง:

```python
document.save("YOUR_OUTPUT_DIRECTORY/smart_art_bullet_fill_format_out.pptx", slides.export.SaveFormat.PPTX)
```

## การประยุกต์ใช้งานจริง

ความสามารถในการตั้งค่าภาพเป็นจุดหัวข้อสามารถใช้ได้ในสถานการณ์ต่างๆ ดังนี้:

1. **การนำเสนอด้านการศึกษา**:ใช้ไอคอนหรือสัญลักษณ์เพื่อช่วยในการเรียนรู้ภาพที่ดีขึ้น
2. **สื่อการตลาด**:เพิ่มการรับรู้แบรนด์ด้วยการใช้โลโก้หรือภาพผลิตภัณฑ์เป็นหัวข้อย่อย
3. **อินโฟกราฟิก**:สร้างอินโฟกราฟิกที่น่าสนใจยิ่งขึ้นด้วยรายการแบบรูปภาพ

# วิธีการเปลี่ยนเค้าโครง SmartArt ใน PowerPoint โดยใช้ Python และ Aspose.Slides

## การแนะนำ

ปรับปรุงงานนำเสนอ PowerPoint ของคุณโดยปรับเปลี่ยนเค้าโครงของกราฟิก SmartArt ด้วย Python และ Aspose.Slides บทช่วยสอนนี้จะแนะนำคุณเกี่ยวกับการเปลี่ยนการออกแบบกราฟิก SmartArt จาก 'Basic Block List' เป็น 'Basic Process' เพื่อปรับปรุงทั้งความสวยงามและความชัดเจนของภาพ

**สิ่งที่คุณจะได้เรียนรู้:**
- การติดตั้งและตั้งค่า Aspose.Slides สำหรับ Python
- การสร้างการนำเสนอ PowerPoint ใหม่ด้วย Python
- การเพิ่มและแก้ไขกราฟิก SmartArt ในสไลด์
- บันทึกการนำเสนอที่อัปเดต

## ข้อกำหนดเบื้องต้น

ตรวจสอบให้แน่ใจว่าสภาพแวดล้อมการพัฒนาของคุณพร้อมแล้ว คุณจะต้องมี:
- **ติดตั้ง Python แล้ว** (แนะนำเวอร์ชัน 3.x)

- ความรู้พื้นฐานเกี่ยวกับแนวคิดการเขียนโปรแกรม Python

ความคุ้นเคยกับการนำเสนอ PowerPoint และกราฟิก SmartArt จะเป็นประโยชน์

## คู่มือการใช้งาน

ทำตามขั้นตอนเหล่านี้เพื่อเปลี่ยนเค้าโครง SmartArt ใน PowerPoint โดยใช้ Python

### การสร้างและปรับเปลี่ยนเค้าโครง SmartArt

#### ภาพรวม:
เพิ่มกราฟิก SmartArt ลงในสไลด์ของคุณและเปลี่ยนประเภทเค้าโครงตามโปรแกรม

#### ขั้นตอนที่ 1: เริ่มต้นการนำเสนอ
สร้างวัตถุการนำเสนอเพื่อให้มั่นใจถึงการจัดการทรัพยากรที่มีประสิทธิภาพด้วยการจัดการบริบท:

```python
with slides.Presentation() as presentation:
    # เข้าถึงสไลด์แรกในการนำเสนอ
slide = presentation.slides[0]
```

#### ขั้นตอนที่ 2: เพิ่มกราฟิก SmartArt
เพิ่มกราฟิก SmartArt 'BasicBlockList' ในตำแหน่งและขนาดที่ระบุโดยใช้:

```python
smart_art = slide.shapes.add_smart_art(
    10, 
    10, 
    400, 
    300,
    slides.smartart.SmartArtLayoutType.BASIC_BLOCK_LIST
)
```

พารามิเตอร์ระบุตำแหน่ง x และ y ความกว้าง ความสูง และประเภทเค้าโครงเริ่มต้น

#### ขั้นตอนที่ 3: เปลี่ยนเค้าโครง SmartArt
ปรับเปลี่ยนเค้าโครงให้เป็น 'BasicProcess':

```python
smart_art.layout = slides.smartart.SmartArtLayoutType.BASIC_PROCESS
```

การดำเนินการนี้จะอัพเดตการออกแบบกราฟิก SmartArt ของคุณเพื่อให้แสดงขั้นตอนลำดับขั้นตอนต่างๆ ได้ดีขึ้น

#### ขั้นตอนที่ 4: บันทึกการนำเสนอ
บันทึกการนำเสนอที่แก้ไข:

```python
output_path = 'YOUR_OUTPUT_DIRECTORY/smart_art_change_layout_out.pptx'
presentation.save(output_path, slides.export.SaveFormat.PPTX)
```

2. **เนื้อหาการศึกษา**:สร้างสรรค์สื่อการเรียนรู้ที่น่าสนใจด้วยการแสดงแนวคิดผ่านแผนภาพกระบวนการในสไลด์

## คู่มือการใช้งาน

### คุณสมบัติ 1: อัปเดตข้อความส่วนหัวในบันทึกย่อหลัก

ฟีเจอร์นี้มุ่งเน้นที่การอัปเดตข้อความส่วนหัวของตัวแทนภายในบันทึกย่อหลักของสไลด์ คุณสามารถทำได้ดังนี้:

#### ภาพรวม
คุณจะทำซ้ำผ่านรูปร่างในบันทึกย่อหลักและอัปเดตส่วนหัวที่พบ

#### ขั้นตอนการดำเนินการ
**ขั้นตอนที่ 1: กำหนดฟังก์ชันเพื่ออัปเดตส่วนหัว**
```python
import aspose.slides as slides

def update_header_footer_text(master):
    """
    Iterate through shapes in the master and update header text if applicable.
    
    Args:
        master (slides.MasterSlide): The master slide containing the shapes to be updated.
    """
    for shape in master.shapes:
        # ตรวจสอบว่ารูปร่างเป็นตัวแทนและเฉพาะประเภท HEADER หรือไม่
        if shape.placeholder is not None and shape.placeholder.type == slides.PlaceholderType.HEADER:
            shape.text_frame.text = "HI there new header"
```
**ขั้นตอนที่ 2: เข้าถึงสไลด์บันทึกย่อหลัก**
โหลดงานนำเสนอของคุณ เข้าถึงสไลด์บันทึกหลัก และใช้การอัปเดตส่วนหัว
```python
def manage_header_footer_text():
    data_dir = "/path/to/your/document/directory/"
    out_dir = "/path/to/your/output/directory/"

    with slides.Presentation(data_dir + "layout_presentation.ppt") as pres:
        # การเข้าถึงสไลด์บันทึกย่อหลักเพื่ออัปเดตข้อความส่วนหัว
        master_notes_slide = pres.master_notes_slide_manager.master_notes_slide
        if master_notes_slide is not None:
            update_header_footer_text(master_notes_slide)

        # บันทึกการนำเสนอด้วยส่วนหัวที่อัปเดต
        pres.save(out_dir + "layout_update_header_footer_text_out.pptx", slides.export.SaveFormat.PPTX)
```
### คุณสมบัติ 2: จัดการข้อความส่วนหัวและส่วนท้าย

ที่นี่เราจะตั้งค่าข้อความส่วนท้ายสำหรับสไลด์ทั้งหมดและบันทึกการปรับเปลี่ยน

#### ภาพรวม
คุณลักษณะนี้ช่วยให้คุณตั้งค่าและแสดงส่วนท้ายของสไลด์ทั้งหมดภายในงานนำเสนอได้

**ขั้นตอนที่ 1: ตั้งค่าข้อความส่วนท้าย**
ใช้ตัวจัดการส่วนหัว-ส่วนท้ายเพื่ออัปเดตส่วนท้ายของสไลด์ทั้งหมด:
```python
def manage_header_footer_text():
    data_dir = "/path/to/your/document/directory/"
    out_dir = "/path/to/your/output/directory/"

    with slides.Presentation(data_dir + "layout_presentation.ppt") as pres:
        # อัปเดตข้อความส่วนท้ายและทำให้มองเห็นได้บนสไลด์ทั้งหมด
        pres.header_footer_manager.set_all_footers_text("My Footer Text")
        pres.header_footer_manager.set_all_footers_visibility(True)
        
        # บันทึกการนำเสนอที่อัปเดต
        pres.save(out_dir + "layout_update_header_footer_text_out.pptx", slides.export.SaveFormat.PPTX)
```

1. **การนำเสนอขององค์กร:** อัปเดตโลโก้หรือวันที่บริษัทโดยอัตโนมัติในส่วนหัวและส่วนท้ายของสไลด์ทั้งหมด
2. **สื่อการเรียนรู้:** การทำให้แน่ใจว่าข้อมูลสอดคล้องกัน เช่น ชื่อหลักสูตรหรือชื่ออาจารย์ผู้สอนปรากฏบนทุกสไลด์

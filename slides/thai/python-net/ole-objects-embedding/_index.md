# บทช่วยสอนเกี่ยวกับวัตถุ PowerPoint OLE และเนื้อหาที่ฝังตัวสำหรับ Aspose.Slides Python

บทช่วยสอนเกี่ยวกับอ็อบเจกต์ OLE ของเราสาธิตวิธีการทำงานกับเนื้อหาที่ฝังไว้ในงานนำเสนอ PowerPoint โดยใช้ Aspose.Slides สำหรับ Python คำแนะนำที่ครอบคลุมเหล่านี้ครอบคลุมถึงการแทรกสเปรดชีต Excel เอกสาร Word และประเภทไฟล์อื่นๆ เป็นอ็อบเจกต์ที่ฝังไว้ การจัดการไฟล์ที่เชื่อมโยง และการควบคุมพฤติกรรมของอ็อบเจกต์ด้วยโปรแกรม บทช่วยสอนแต่ละบทประกอบด้วยตัวอย่างโค้ด Python ในทางปฏิบัติสำหรับสถานการณ์การฝังทั่วไป ช่วยให้คุณสร้างงานนำเสนอที่มีเนื้อหาสมบูรณ์ซึ่งผสานรวมเนื้อหาจากหลายแหล่งได้อย่างราบรื่น

## บทช่วยสอนที่พร้อมใช้งาน

### [ฝัง Excel เป็นวัตถุ OLE ใน PowerPoint โดยใช้ Python: คู่มือที่ครอบคลุม](./embed-excel-ole-object-powerpoint-python/)

### [ส่งออกข้อมูล OLE ไปยัง PDF โดยใช้ Aspose.Slides ใน Python: คำแนะนำทีละขั้นตอน](./export-ole-data-to-pdf-aspose-slides-python/)

### [วิธีการแทนที่ OLE Object Frame Title ด้วยรูปภาพใน PowerPoint โดยใช้ Aspose.Slides สำหรับ Python](./substitute-ole-object-frame-title-with-image-aspose-slides/)

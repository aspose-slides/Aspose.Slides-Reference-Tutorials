## คู่มือการใช้งาน
มาแบ่งกระบวนการควบคุมการรีเฟรชภาพขนาดย่อออกเป็นขั้นตอนกัน

### คุณสมบัติ: การควบคุมการรีเฟรชภาพขนาดย่อที่มีประสิทธิภาพ
ฟีเจอร์นี้สาธิตวิธีการจัดการว่าภาพขนาดย่อของ PowerPoint จะได้รับการรีเฟรชหรือไม่เมื่อแก้ไขสไลด์ ซึ่งจะช่วยเพิ่มประสิทธิภาพการทำงานสำหรับการนำเสนอขนาดใหญ่

#### ภาพรวม
โดยการตั้งค่า `refresh_thumbnail` ถึง `False`คุณสามารถป้องกันการสร้างภาพขนาดย่อที่ไม่จำเป็นได้ ช่วยประหยัดเวลาและทรัพยากร

#### ขั้นตอนการดำเนินการ
**ขั้นตอนที่ 1: เปิดการนำเสนอ**
เปิดไฟล์ PowerPoint ที่มีอยู่โดยใช้ Aspose.Slides:

```python
import aspose.slides as slides

def refresh_thumbnail_presentation():
    # โหลดการนำเสนอจากไดเร็กทอรีของคุณ
    with slides.Presentation("YOUR_DOCUMENT_DIRECTORY/Image.pptx") as pres:
```

**ขั้นตอนที่ 2: แก้ไขเนื้อหาสไลด์**
ลบรูปร่างทั้งหมดออกจากสไลด์เพื่อแสดงการเปลี่ยนแปลงโดยไม่ต้องรีเฟรชภาพขนาดย่อ:

```python
        # ล้างรูปร่างทั้งหมดตั้งแต่สไลด์แรก
        pres.slides[0].shapes.clear()
```

**ขั้นตอนที่ 3: กำหนดค่าตัวเลือกภาพขนาดย่อ**
ตั้งค่าตัวเลือกสำหรับการบันทึกการนำเสนอ การกำหนดค่าว่าจะรีเฟรชภาพขนาดย่อหรือไม่:

```python
        # ตั้งค่า PptxOptions เพื่อควบคุมพฤติกรรมของภาพขนาดย่อ
        pptx_options = slides.export.PptxOptions()
        pptx_options.refresh_thumbnail = False  # ป้องกันการรีเฟรชรูปขนาดย่อ
```

**ขั้นตอนที่ 4: บันทึกการนำเสนอ**
บันทึกการนำเสนอที่คุณแก้ไขโดยใช้ตัวเลือกที่กำหนดค่าไว้:

```python
        # บันทึกด้วย PptxOptions ที่กำหนดเอง
        pres.save("YOUR_OUTPUT_DIRECTORY/result_with_old_thumbnail.pptx",
                  slides.export.SaveFormat.PPTX,
                  pptx_options)
```

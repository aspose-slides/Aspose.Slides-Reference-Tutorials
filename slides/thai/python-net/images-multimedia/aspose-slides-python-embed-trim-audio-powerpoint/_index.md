## คู่มือการใช้งาน
หัวข้อนี้จะแนะนำคุณเกี่ยวกับการฝังและตัดแต่งเสียงโดยใช้ Aspose.Slides

### เพิ่มเฟรมเสียงให้กับงานนำเสนอ
**ภาพรวม**:ปรับปรุงการโต้ตอบในการนำเสนอของคุณด้วยการเพิ่มไฟล์เสียงเป็นเฟรมที่ฝังอยู่ในสไลด์ PowerPoint

#### ขั้นตอนที่ 1: เปิดการนำเสนอเพื่อแก้ไข
```python
# เปิดหรือสร้างการนำเสนอใหม่
current_pres = slides.Presentation()
```

#### ขั้นตอนที่ 2: อ่านและเพิ่มไฟล์เสียง
```python
    # เปิดไฟล์เสียงจากไดเร็กทอรีของคุณในโหมดไบนารี
    with open('YOUR_DOCUMENT_DIRECTORY/audio.m4a', 'rb') as audio_file:
        # เพิ่มเสียงลงในคอลเลคชันการนำเสนอ
        current_audio = current_pres.audios.add_audio(audio_file)
```

#### ขั้นตอนที่ 3: ฝังเฟรมเสียงลงในสไลด์
```python
    # เพิ่มเฟรมเสียงฝังตัวตามพิกัดที่ระบุ (50, 50) ด้วยขนาด (100, 100)
    audio_frame = current_pres.slides[0].shapes.add_audio_frame_embedded(50, 50, 100, 100, current_audio)
```

### การตัดแต่งเฟรมเสียงในงานนำเสนอ
**ภาพรวม**:การตัดส่วนเริ่มต้นและส่วนสิ้นสุดของเฟรมเสียงอาจมีความสำคัญต่อจังหวะที่แม่นยำในการนำเสนอของคุณ

#### ขั้นตอนที่ 1: ตั้งค่าการตัดแต่งเริ่มต้น
```python
    # ตัดช่วงต้นของเสียงลง 500 มิลลิวินาที (0.5 วินาที)
    audio_frame.trim_from_start = 500
```

#### ขั้นตอนที่ 2: ตั้งค่าการตัดแต่งปลาย
```python
    # ตัดตอนท้ายของเสียงลง 1,000 มิลลิวินาที (1 วินาที)
    audio_frame.trim_from_end = 1000
```

### การบันทึกการนำเสนอ
บันทึกการนำเสนอที่คุณแก้ไขแล้วไปยังไดเร็กทอรีเอาท์พุต:
```python
    current_pres.save('YOUR_OUTPUT_DIRECTORY/AudioFrameTrim_out.pptx', slides.export.SaveFormat.PPTX)
```

## คู่มือการใช้งาน

### แยกเสียงจากไฮเปอร์ลิงก์

#### ภาพรวม

ฟีเจอร์นี้ช่วยให้คุณเข้าถึงและแยกข้อมูลเสียงที่เชื่อมโยงผ่านไฮเปอร์ลิงก์ในรูปร่างแรกของสไลด์แรกในงานนำเสนอ PowerPoint ซึ่งมีประโยชน์โดยเฉพาะอย่างยิ่งสำหรับงานนำเสนอที่สไลด์เสริมเสียงโดยไม่ต้องฝังเสียงโดยตรงลงไป

#### คำแนะนำทีละขั้นตอน

##### 1. กำหนดไดเรกทอรีอินพุตและเอาต์พุต

ระบุไดเรกทอรีสำหรับไฟล์ PowerPoint ของคุณ (`input_directory`) และไดเรกทอรีที่จะบันทึกเสียงที่แยกออกมา (`output_directory`-

```python
import aspose.slides as slides

def extract_audio_from_hyperlink():
    input_directory = 'YOUR_DOCUMENT_DIRECTORY/'
    output_directory = 'YOUR_OUTPUT_DIRECTORY/'
```

##### 2. เปิดไฟล์ PowerPoint

ใช้ Aspose.Slides เพื่อเปิดไฟล์งานนำเสนอของคุณ โดยให้แน่ใจว่ามีไฮเปอร์ลิงก์ที่มีข้อมูลเสียง

```python
with slides.Presentation(input_directory + 'HyperlinkSound.pptx') as pres:
    # โค้ดเพิ่มเติมที่นี่
```

##### 3. การเข้าถึงการดำเนินการคลิกไฮเปอร์ลิงก์

เข้าถึงการดำเนินการคลิกไฮเปอร์ลิงก์จากรูปร่างแรกบนสไลด์แรกเพื่อตรวจสอบเสียงที่เกี่ยวข้อง

```python
    link = pres.slides[0].shapes[0].hyperlink_click
```

##### 4. แยกและบันทึกข้อมูลเสียง

หากเสียงเชื่อมโยงกัน ให้แยกเสียงออกมาเป็นอาร์เรย์ไบต์ และบันทึกเป็นรูปแบบ MP3

```python
    if link.sound is not None:
        audio_data = link.sound.binary_data
        with open(output_directory + 'HyperlinkSound.mp3', 'wb') as audio_file:
            audio_file.write(audio_data)
```

- **ไม่สามารถแยกเสียงออกได้**:ให้แน่ใจว่าไฮเปอร์ลิงก์ในสไลด์ของคุณมีข้อมูลที่ถูกต้อง
- **ข้อผิดพลาดเส้นทางไฟล์**ตรวจสอบอีกครั้งว่าไดเร็กทอรีอินพุตและเอาต์พุตของคุณได้รับการระบุอย่างถูกต้อง

1. **การสกัดเนื้อหาอัตโนมัติ**:แยกเนื้อหาสื่อโดยอัตโนมัติเพื่อเก็บถาวรหรือนำไปใช้ใหม่
2. **การปรับปรุงการนำเสนอระยะไกล**:จัดเตรียมไฟล์เสียงแบบสแตนด์อโลนเพื่อใช้ประกอบการนำเสนอระยะไกล
3. **สื่อการเรียนรู้แบบโต้ตอบ**:ใช้เสียงที่แยกออกมาเป็นส่วนหนึ่งของทรัพยากรการศึกษาแบบมัลติมีเดียเชิงโต้ตอบ

### แปลงงานนำเสนอเป็น PDF โดยใช้รูปแบบฟอนต์ที่ไม่รองรับการแรสเตอร์ไรเซชัน
#### ภาพรวม
คุณลักษณะนี้จะช่วยให้แน่ใจว่าแม้ว่ารูปแบบอักษรบางรูปในงานนำเสนอของคุณจะไม่รองรับรูปแบบ PDF แต่แบบอักษรเหล่านั้นก็จะถูกแรสเตอร์ไรซ์และยังคงรูปลักษณ์เดิมไว้

#### ขั้นตอนการดำเนินการ
1. **เริ่มต้นวัตถุการนำเสนอ**-
   เริ่มต้นด้วยการสร้างวัตถุการนำเสนอใหม่หรือโหลดวัตถุที่มีอยู่แล้ว ที่นี่เราจะเริ่มต้นการนำเสนอเปล่าเพื่อความเรียบง่าย
2. **กำหนดค่า PdfOptions**-
   สร้างและกำหนดค่า `PdfOptions` เพื่อระบุว่าควรแรสเตอร์แบบอักษรที่ไม่รองรับ
3. **บันทึกไฟล์ PDF**-
   บันทึกการนำเสนอของคุณเป็นไฟล์ PDF พร้อมตัวเลือกที่กำหนดค่าไว้

คุณสามารถนำฟีเจอร์นี้ไปใช้งานได้ดังนี้:

```python
import aspose.slides as slides

def convert_to_pdf_unsupported_font_styles():
    # เริ่มต้นวัตถุการนำเสนอด้วยการนำเสนอที่ว่างเปล่า
    with slides.Presentation() as presentation:
        # สร้าง PdfOptions เพื่อระบุวิธีการสร้าง PDF
        pdf_options = slides.export.PdfOptions()
        
        # เปิดใช้งานการแรสเตอร์ของรูปแบบอักษรที่ไม่รองรับ
        pdf_options.rasterize_unsupported_font_styles = True
        
        # บันทึกการนำเสนอเป็นไฟล์ PDF
        output_path = 'YOUR_OUTPUT_DIRECTORY/UnsupportedFontStyles.pdf'
        presentation.save(output_path, slides.export.SaveFormat.PDF, pdf_options)
```

**คำอธิบาย**- 
- `PdfOptions` ช่วยให้ปรับแต่งวิธีการสร้าง PDF ได้ การตั้งค่า `rasterize_unsupported_font_styles` ถึง `True` ช่วยให้แน่ใจว่าแบบอักษรที่ไม่รองรับจะถูกแรสเตอร์ไรซ์
- การ `presentation.save()` วิธีการเขียนการนำเสนอของคุณไปยังไฟล์ที่ระบุโดย `output_path`-

#### เคล็ดลับการแก้ไขปัญหา
- ตรวจสอบให้แน่ใจว่าคุณมีสิทธิ์การเขียนสำหรับไดเร็กทอรีที่คุณกำลังบันทึก PDF
- หากปัญหาแบบอักษรยังคงเกิดขึ้น ให้ตรวจสอบว่าไฟล์แบบอักษรได้รับการติดตั้งบนระบบของคุณอย่างถูกต้อง

### การสร้างและการบันทึกการนำเสนอขั้นพื้นฐาน
#### ภาพรวม
คุณลักษณะนี้ช่วยให้คุณสามารถสร้างการนำเสนอ PowerPoint ง่ายๆ ตั้งแต่เริ่มต้นและบันทึกเป็นไฟล์ PPTX ได้

#### ขั้นตอนการดำเนินการ
1. **สร้างการนำเสนอที่ว่างเปล่า**-
   สร้างวัตถุการนำเสนอใหม่โดยเริ่มต้นด้วยกระดานเปล่า
2. **ตรวจสอบให้แน่ใจว่ามีไดเร็กทอรีเอาท์พุตอยู่**-
   ก่อนที่จะบันทึก โปรดตรวจสอบให้แน่ใจว่าไดเร็กทอรีที่คุณต้องการจัดเก็บไฟล์นั้นมีอยู่ หรือสร้างขึ้นใหม่หากจำเป็น
3. **บันทึกการนำเสนอเป็น PPTX**-
   สุดท้ายให้บันทึกงานนำเสนอที่คุณสร้างขึ้นใหม่ในรูปแบบที่ต้องการ

คุณสามารถทำสิ่งนี้ได้ดังนี้:

```python
import os
from pathlib import Path
import aspose.slides as slides

def create_and_save_presentation():
    # สร้างวัตถุการนำเสนอที่ว่างเปล่า
    with slides.Presentation() as presentation:
        # ตรวจสอบว่าไดเรกทอรีเอาท์พุตมีอยู่หรือสร้างขึ้นใหม่
        output_dir = Path('YOUR_OUTPUT_DIRECTORY/')
        os.makedirs(output_dir, exist_ok=True)
        
        # กำหนดเส้นทางที่จะบันทึกการนำเสนอ
        output_path = output_dir / 'SimplePresentation.pptx'
        
        # บันทึกการนำเสนอที่ว่างเปล่าเป็นไฟล์ PPTX
        presentation.save(str(output_path), slides.export.SaveFormat.PPTX)
```

**คำอธิบาย**- 
- โดยใช้ `os.makedirs()` ช่วยให้แน่ใจว่าไดเร็กทอรีที่คุณระบุพร้อมสำหรับการบันทึกไฟล์
- การ `presentation.save()` วิธีการเขียนการนำเสนอของคุณในรูปแบบ .pptx

#### เคล็ดลับการแก้ไขปัญหา
- ตรวจสอบพื้นที่ดิสก์ให้เพียงพอสำหรับบันทึกการนำเสนอ
- ตรวจสอบรูปแบบเส้นทางไฟล์ โดยเฉพาะอย่างยิ่งถ้าใช้ระบบปฏิบัติการที่แตกต่างกัน

2. **สื่อการเรียนรู้**:สร้างและแบ่งปันแผนการสอนหรือสไลด์ในรูปแบบ PDF โดยไม่สูญเสียความชัดเจนของข้อความ
3. **โบรชัวร์การตลาด**:ออกแบบโบรชัวร์ใน PowerPoint และแปลงเป็น PDF โดยให้แน่ใจว่าแบบอักษรของแบรนด์ได้รับการรักษาไว้
4. **การวางแผนกิจกรรม**:แบ่งปันรายละเอียดกิจกรรมกับผู้เข้าร่วมผ่านไฟล์ PDF ที่สะท้อนถึงการออกแบบการนำเสนอดั้งเดิม

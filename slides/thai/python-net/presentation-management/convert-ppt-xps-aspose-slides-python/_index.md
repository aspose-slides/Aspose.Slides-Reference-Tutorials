## คู่มือการใช้งาน

ในส่วนนี้เราจะแนะนำการแปลงไฟล์ PowerPoint เป็นรูปแบบ XPS โดยใช้ Aspose.Slides สำหรับ Python

### ภาพรวม: แปลงงานนำเสนอเป็น XPS

ฟังก์ชันหลักของบทช่วยสอนนี้คือการสาธิตให้เห็นว่าคุณสามารถแปลงไฟล์ PPT เป็นรูปแบบ XPS ที่พกพาได้และอเนกประสงค์ยิ่งขึ้นได้อย่างไร

#### ขั้นตอนที่ 1: กำหนดไดเรกทอรี
เริ่มต้นด้วยการกำหนดไดเร็กทอรีอินพุตและเอาต์พุตที่ไฟล์ PowerPoint ของคุณอยู่และที่คุณต้องการบันทึกไฟล์ XPS ที่แปลงแล้ว:

```python
input_directory = "YOUR_DOCUMENT_DIRECTORY/"
output_directory = "YOUR_OUTPUT_DIRECTORY/"
```

เส้นทางเหล่านี้จะถูกใช้ในภายหลังในฟังก์ชันการแปลงของเรา

#### ขั้นตอนที่ 2: โหลดงานนำเสนอ
สร้าง `Presentation` วัตถุที่แสดงไฟล์ PowerPoint กำหนดเส้นทางไปยังไฟล์ของคุณ `.pptx` ไฟล์:

```python
demo_presentation_path = input_directory + "welcome-to-powerpoint.pptx"
```

โดยใช้ตัวจัดการบริบท (`with slides.Presentation(demo_presentation_path) as pres:`) เรามั่นใจว่าทรัพยากรได้รับการจัดการอย่างเหมาะสม

#### ขั้นตอนที่ 3: บันทึกในรูปแบบ XPS
เมื่อโหลดการนำเสนอแล้ว ให้ระบุว่าคุณต้องการบันทึกเอาต์พุตและใช้ที่ใด `save` วิธีการแปลง:

```python
dxps_output_path = output_directory + "converted_to_xps_out.xps"
pres.save(dxps_output_path, slides.export.SaveFormat.XPS)
```

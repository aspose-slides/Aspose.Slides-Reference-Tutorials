## คู่มือการใช้งาน

### การตรวจจับรูปแบบ PPT95

หากต้องการตรวจสอบว่าการนำเสนออยู่ในรูปแบบ PPT เก่า (PPT95) หรือไม่ ให้ทำตามขั้นตอนเหล่านี้:

#### การดำเนินการแบบทีละขั้นตอน

**1. รับข้อมูลการนำเสนอ**

โหลดข้อมูลการนำเสนอโดยใช้ Aspose.Slides:

```python
import aspose.slides as slides

def check_presentation_format():
    # แทนที่ 'YOUR_DOCUMENT_DIRECTORY/' ด้วยเส้นทางไดเร็กทอรีของคุณ
    load_info = slides.PresentationFactory.instance.get_presentation_info(
        "YOUR_DOCUMENT_DIRECTORY/open_presentation.ppt")
```

*คำอธิบาย:* เราใช้ `PresentationFactory` เพื่อดึงรายละเอียดการนำเสนอ วิธีการ `get_presentation_info` อ่านข้อมูลเมตาของไฟล์ รวมถึงรูปแบบด้วย

**2. กำหนดรูปแบบ**

ตรวจสอบว่ารูปแบบที่โหลดเป็น PPT95 หรือไม่:

```python
    # ตรวจสอบว่ารูปแบบการนำเสนอเป็น PPT95 หรือไม่
is_old_format = load_info.load_format == slides.LoadFormat.PPT95

return is_old_format
```

*คำอธิบาย:* โดยการเปรียบเทียบ `load_info.load_format` กับ `slides.LoadFormat.PPT95`เราจะตรวจสอบว่าไฟล์อยู่ในรูปแบบ PPT เก่าหรือไม่

### เคล็ดลับการแก้ไขปัญหา

- **ข้อผิดพลาดเส้นทางไฟล์:** ตรวจสอบให้แน่ใจว่าเส้นทางไดเร็กทอรีและชื่อไฟล์ของคุณถูกต้อง
- **ปัญหาการติดตั้ง:** ตรวจสอบเวอร์ชัน pip และ Python ใช้ `pip --version` เพื่อตรวจสอบว่าติดตั้ง PIP อย่างถูกต้องหรือไม่
- **ปัญหาเรื่องใบอนุญาต:** ตรวจสอบเส้นทางใบอนุญาตของคุณอีกครั้งและให้แน่ใจว่าได้ใช้ก่อนที่จะเรียกใช้สคริปต์

## การประยุกต์ใช้งานจริง

การตรวจจับรูปแบบ PPT95 อาจมีความสำคัญในหลายสถานการณ์:
1. **การรวมระบบเก่า:** รับรองความเข้ากันได้กับระบบรุ่นเก่าที่รองรับเฉพาะรูปแบบ PPT
2. **โครงการย้ายข้อมูล:** ระบุไฟล์ที่จำเป็นต้องแปลงในระหว่างการย้ายข้อมูลไปยังรูปแบบใหม่กว่า เช่น PPTX
3. **การจัดการเอกสาร:** ติดตามการนำเสนอที่เก็บถาวรและวางแผนสำหรับการอัปเดตหรือการแปลงรูปแบบ

ความเป็นไปได้ของการบูรณาการได้แก่ การทำให้การตรวจสอบนี้เป็นแบบอัตโนมัติภายในเวิร์กโฟลว์ที่ใหญ่กว่า เช่น ระบบการจัดการเอกสาร หรือกระบวนการสร้างรายงานอัตโนมัติ

## คู่มือการใช้งาน
ในส่วนนี้ เราจะแนะนำคุณเกี่ยวกับการแปลงการนำเสนอ PPT เป็นรูปแบบ HTML5 พร้อมเปิดใช้งานแอนิเมชัน

### ขั้นตอนที่ 1: กำหนดไดเรกทอรีอินพุตและเอาต์พุต
ตั้งค่าไดเรกทอรีอินพุตและเอาต์พุตของคุณโดยใช้ Python `pathlib` ห้องสมุด:
```python
from pathlib import Path

data_dir = Path("YOUR_DOCUMENT_DIRECTORY/") / "welcome-to-powerpoint.pptx"
out_dir = Path("YOUR_OUTPUT_DIRECTORY/")
output_file = out_dir / "convert_to_html5_out.html"
# ให้แน่ใจว่ามีไดเร็กทอรีอยู่
Path("YOUR_DOCUMENT_DIRECTORY/").mkdir(parents=True, exist_ok=True)
Path("YOUR_OUTPUT_DIRECTORY/").mkdir(parents=True, exist_ok=True)
```
### ขั้นตอนที่ 2: เปิดการนำเสนอ
เปิดไฟล์การนำเสนอของคุณโดยใช้ Aspose.Slides:
```python
with slides.Presentation(data_dir) as pres:
    # ดำเนินการตามขั้นตอนการแปลงที่นี่
```
### ขั้นตอนที่ 3: กำหนดค่าตัวเลือกการส่งออก HTML5
หากต้องการรวมแอนิเมชันไว้ในผลลัพธ์ HTML5 ให้กำหนดค่าตัวเลือกการส่งออก:
```python
html5_options = slides.export.Html5Options()
html5_options.animate_shapes = True     # เปิดใช้งานแอนิเมชั่นรูปร่าง
click to enable transition animations
html5_options.animate_transitions = True
```
### ขั้นตอนที่ 4: บันทึกการนำเสนอเป็น HTML5
สุดท้ายให้บันทึกการนำเสนอของคุณด้วยตัวเลือกที่ระบุ:
```python
pres.save(output_file, slides.export.SaveFormat.HTML5, html5_options)
```
วิธีนี้จะช่วยให้แน่ใจว่าการเปลี่ยนสไลด์และแอนิเมชั่นรูปร่างทั้งหมดจะยังคงอยู่ในเอาต์พุต HTML5

## การประยุกต์ใช้งานจริง
การแปลงงานนำเสนอเป็น HTML5 มีการใช้งานจริงหลายประการ:
1. **แพลตฟอร์มการเรียนรู้แบบออนไลน์:** แจกจ่ายสื่อการสอนแบบโต้ตอบ
2. **เว็บสัมมนาและการประชุมเสมือนจริง:** เพิ่มการมีส่วนร่วมด้วยสไลด์เคลื่อนไหว
3. **เว็บไซต์ขององค์กร:** จัดแสดงตัวอย่างผลิตภัณฑ์หรือเนื้อหาการตลาดแบบโต้ตอบ
4. **ระบบจัดการเนื้อหา:** บูรณาการการนำเสนอเข้ากับแพลตฟอร์มอย่างราบรื่น เช่น WordPress
5. **แอปพลิเคชันมือถือ:** ให้การเข้าถึงสื่อการนำเสนอแบบออฟไลน์บนอุปกรณ์เคลื่อนที่

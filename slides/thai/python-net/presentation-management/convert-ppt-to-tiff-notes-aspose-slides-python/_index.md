## คู่มือการใช้งาน
### แปลงงานนำเสนอเป็น TIFF ด้วย Notes
คุณลักษณะนี้ช่วยให้คุณสามารถส่งออกการนำเสนอ PowerPoint เป็นรูปแบบ TIFF โดยมั่นใจได้ว่ามีบันทึกย่ออยู่ที่ด้านล่างของแต่ละสไลด์

#### ภาพรวม
กระบวนการนี้เกี่ยวข้องกับการตั้งค่าตัวเลือกเฉพาะสำหรับการเรนเดอร์สไลด์เป็นไฟล์ TIFF และกำหนดค่าวิธีการแสดงบันทึก

#### การดำเนินการแบบทีละขั้นตอน
**1. นำเข้า Aspose.Slides**
เริ่มต้นด้วยการนำเข้าโมดูลที่จำเป็น:

```python
import aspose.slides as slides
```

**2. ตั้งค่าตัวเลือกการส่งออก**
กำหนดค่า `TiffOptions` รวมถึงการตั้งค่าเค้าโครงสำหรับบันทึกสไลด์:

```python
# สร้างวัตถุ TiffOptions
 tiff_options = slides.export.TiffOptions()

# กำหนดค่าตัวเลือกเค้าโครงบันทึก
slides_layout_options = slides.export.NotesCommentsLayoutingOptions()
slides_layout_options.notes_position = slides.export.NotesPositions.BOTTOM_FULL

# กำหนดตัวเลือกเค้าโครงเหล่านี้ให้กับตัวเลือก TIFF
tiff_options.slides_layout_options = slides_layout_options
```

**3. โหลดและแปลงงานนำเสนอ**
โหลดไฟล์ PowerPoint ของคุณและแปลงเป็นภาพ TIFF โดยใช้ตัวเลือกที่กำหนดค่าไว้:

```python
with slides.Presentation('YOUR_DOCUMENT_DIRECTORY/presentation_with_notes.pptx') as pres:
    # บันทึกการนำเสนอในรูปแบบ TIFF พร้อมหมายเหตุที่ด้านล่าง
    pres.save('YOUR_OUTPUT_DIRECTORY/convert_to_tiff_with_notes_out.tiff',
              slides.export.SaveFormat.TIFF, tiff_options)
```

**คำอธิบาย**
- `tiff_options`:กำหนดค่าวิธีการเรนเดอร์แต่ละสไลด์เป็นภาพ TIFF
- `slides_layout_options.notes_position`:ทำให้แน่ใจว่าบันทึกจะถูกวางไว้อย่างครบถ้วนที่ด้านล่างของสไลด์แต่ละสไลด์

#### เคล็ดลับการแก้ไขปัญหา
- **ไม่พบไฟล์**: ตรวจสอบให้แน่ใจว่าเส้นทางไฟล์ของคุณถูกต้องและสามารถเข้าถึงได้
- **ปัญหาการอนุญาต**: ตรวจสอบว่าคุณมีสิทธิ์อ่าน/เขียนสำหรับไดเร็กทอรีที่ระบุหรือไม่

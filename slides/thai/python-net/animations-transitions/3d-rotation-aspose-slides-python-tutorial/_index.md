## คู่มือการใช้งาน

ในส่วนนี้จะให้รายละเอียดเกี่ยวกับวิธีการใช้เอฟเฟ็กต์การหมุน 3 มิติ

### การใช้การหมุน 3 มิติกับรูปทรงสี่เหลี่ยมผืนผ้า

#### ภาพรวม

เพิ่มความลึกและมุมมองให้กับรูปสี่เหลี่ยมผืนผ้าโดยใช้การหมุน 3 มิติ

#### การดำเนินการแบบทีละขั้นตอน

**1. เพิ่มรูปทรงสี่เหลี่ยมผืนผ้า:**

```python
auto_shape = pres.slides[0].shapes.add_auto_shape(
    slides.ShapeType.RECTANGLE, 30, 30, 200, 200)
```

*คำอธิบาย*:โค้ดนี้จะเพิ่มสี่เหลี่ยมผืนผ้าที่ตำแหน่ง (30, 30) โดยมีขนาด 200x200

**2. ใช้การหมุน 3D:**

```python
auto_shape.three_d_format.depth = 6
auto_shape.three_d_format.camera.set_rotation(40, 35, 20)
auto_shape.three_d_format.camera.camera_type = slides.CameraPresetType.ISOMETRIC_LEFT_UP
auto_shape.three_d_format.light_rig.light_type = slides.LightRigPresetType.BALANCED
```

*คำอธิบาย*- 
- `depth`: ตั้งค่าความลึกของเอฟเฟกต์ 3 มิติ
- `camera.set_rotation()`: กำหนดมุมการหมุนสำหรับแกน X, Y และ Z
- `camera_type`: กำหนดมุมมองของกล้อง
- `light_rig.light_type`: ปรับแสงเพื่อเพิ่มความสวยงามแบบสามมิติ

**3. บันทึกการนำเสนอของคุณ:**

```python
pres.save("shapes_apply_3d_rotation_to_rectangle_out.pptx", slides.export.SaveFormat.PPTX)
```

### การใช้การหมุน 3 มิติกับรูปร่างเส้น

#### ภาพรวม

สร้างองค์ประกอบภาพที่น่าสนใจด้วยการเพิ่มเอฟเฟกต์ 3 มิติให้กับรูปร่างเส้น

#### การดำเนินการแบบทีละขั้นตอน

**1. เพิ่มรูปร่างเส้น:**

```python
auto_shape = pres.slides[0].shapes.add_auto_shape(
    slides.ShapeType.LINE, 30, 300, 200, 200)
```

*คำอธิบาย*:โค้ดนี้จะเพิ่มบรรทัดที่ตำแหน่ง (30, 300) และมีขนาด 200x200

**2. ใช้การหมุน 3D:**

```python
auto_shape.three_d_format.depth = 6
auto_shape.three_d_format.camera.set_rotation(0, 35, 20)
auto_shape.three_d_format.camera.camera_type = slides.CameraPresetType.ISOMETRIC_LEFT_UP
auto_shape.three_d_format.light_rig.light_type = slides.LightRigPresetType.BALANCED
```

*คำอธิบาย*:คล้ายกับรูปทรงสี่เหลี่ยมผืนผ้า แต่มีมุมการหมุนที่แตกต่างกันเพื่อสร้างเอฟเฟกต์ที่เป็นเอกลักษณ์

**3. บันทึกการนำเสนอของคุณ:**

```python
pres.save("shapes_apply_3d_rotation_to_line_out.pptx", slides.export.SaveFormat.PPTX)
```

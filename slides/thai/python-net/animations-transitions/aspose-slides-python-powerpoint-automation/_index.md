## คู่มือการใช้งาน

เราจะแบ่งกระบวนการออกเป็นส่วนๆ ตามคุณลักษณะแต่ละอย่าง

### คุณสมบัติ 1: โหลดและทำซ้ำผ่านการนำเสนอ

#### ภาพรวม:
ฟีเจอร์นี้ช่วยให้คุณโหลดไฟล์งานนำเสนอ PowerPoint และทำซ้ำผ่านสไลด์ ซึ่งมีประโยชน์สำหรับการประมวลผลสไลด์อัตโนมัติหรือการดึงข้อมูลเฉพาะ

#### การดำเนินการทีละขั้นตอน:
**ขั้นตอนที่ 1: กำหนดฟังก์ชัน**
กำหนดฟังก์ชั่น `load_presentation` ซึ่งนำเส้นทางไปยังไฟล์การนำเสนอของคุณเป็นอาร์กิวเมนต์

```python
def load_presentation(presentation_path):
    with slides.Presentation(presentation_path) as pres:
        for slide in pres.slides:
            print(f"Slide #โหลด {slide.slide_number} แล้ว)
```
**คำอธิบาย:**
- `slides.Presentation(presentation_path)` เปิดไฟล์ PowerPoint ของคุณ
- ตัวจัดการบริบทจะตรวจสอบให้แน่ใจว่าการนำเสนอปิดอย่างถูกต้องหลังจากประมวลผลแล้ว

**ขั้นตอนที่ 2: ตัวอย่างการใช้งาน**
แทนที่ `'YOUR_DOCUMENT_DIRECTORY/'` พร้อมด้วยเส้นทางไดเร็กทอรีจริงที่เอกสารของคุณถูกเก็บอยู่:

```python
load_presentation('YOUR_DOCUMENT_DIRECTORY/shapes_animation_example.pptx')
```

### คุณสมบัติ 2: สกัดเอฟเฟกต์แอนิเมชันจากสไลด์

#### ภาพรวม:
แยกและพิมพ์รายละเอียดเกี่ยวกับเอฟเฟกต์แอนิเมชันที่ใช้กับรูปร่างในแต่ละสไลด์ ซึ่งจะช่วยวิเคราะห์การตั้งค่าแอนิเมชันในงานนำเสนอของคุณ

#### การดำเนินการทีละขั้นตอน:
**ขั้นตอนที่ 1: กำหนดฟังก์ชัน**
สร้างฟังก์ชั่น `extract_animation_effects` ที่โหลดงานนำเสนอและทำซ้ำผ่านแอนิเมชัน

```python
def extract_animation_effects(presentation_path):
    with slides.Presentation(presentation_path) as pres:
        for slide in pres.slides:
            for effect in slide.timeline.main_sequence:
                print(f"{effect.type} animation effect is set to shape#{effect.target_shape.unique_id} บนสไลด์#{slide.slide_number}")
```
**คำอธิบาย:**
- `slide.timeline.main_sequence` ให้การเข้าถึงแอนิเมชั่นทั้งหมดที่นำมาใช้บนสไลด์
- แต่ละ `effect` วัตถุประกอบด้วยรายละเอียดเกี่ยวกับประเภทของแอนิเมชั่นและรูปร่างเป้าหมาย

**ขั้นตอนที่ 2: ตัวอย่างการใช้งาน**
ใช้ฟังก์ชันกับเส้นทางการนำเสนอของคุณ:

```python
extract_animation_effects('YOUR_DOCUMENT_DIRECTORY/shapes_animation_example.pptx')
```

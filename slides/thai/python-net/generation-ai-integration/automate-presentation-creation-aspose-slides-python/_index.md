## คู่มือการใช้งาน

หัวข้อนี้จะแนะนำคุณเกี่ยวกับการสร้างงานนำเสนอและการกำหนดค่าเพื่อรวมรูปภาพในรูปแบบไทล์

### การสร้างและกำหนดค่าการนำเสนอ

#### ภาพรวม
เราจะสร้างงานนำเสนอใหม่ เพิ่มสไลด์ แทรกภาพ และกำหนดค่ารูปร่างด้วยรูปแบบการเติมรูปภาพแบบกระเบื้อง

#### การเข้าถึงสไลด์แรก

เริ่มต้นโดยเข้าถึงสไลด์แรก:

```python
# สร้างค่าเริ่มต้นของวัตถุการนำเสนอด้วย slides.Presentation() เป็น pres:
    # เข้าถึงสไลด์แรกในการนำเสนอ
    first_slide = pres.slides[0]
```

#### การเพิ่มรูปภาพลงในงานนำเสนอ

โหลดและเพิ่มรูปภาพที่คุณต้องการจากไดเร็กทอรี:

```python
# โหลดรูปภาพจากไดเร็กทอรีที่ระบุและเพิ่มลงในคอลเล็กชันรูปภาพของการนำเสนอด้วย slides.Images.from_file("YOUR_DOCUMENT_DIRECTORY/image.png") เป็นรูปภาพใหม่:
    pp_image = pres.images.add_image(new_image)
```

#### การเพิ่มรูปทรงด้วยการเติมรูปภาพแบบกระเบื้อง

เพิ่มรูปทรงสี่เหลี่ยมผืนผ้าลงในสไลด์ของคุณ:

```python
# เพิ่มรูปร่างสี่เหลี่ยมผืนผ้าลงในสไลด์แรก
ew_shape = first_slide.shapes.add_auto_shape(
    slides.ShapeType.RECTANGLE, 0, 0, 350, 350
)

# ตั้งค่าประเภทการเติมของรูปร่างเป็นรูปภาพและกำหนดค่าสำหรับการปูกระเบื้อง
new_shape.fill_format.fill_type = slides.FillType.PICTURE
picture_fill_format = new_shape.fill_format.picture_fill_format

# กำหนดรูปภาพที่โหลดให้เป็นรูปแบบการเติมรูปภาพของรูปร่าง\ppicture_fill_format.picture.image = pp_image

# กำหนดค่าคุณสมบัติการเติมกระเบื้อง\ppicture_fill_format.picture_fill_mode = slides.PictureFillMode.TILE
picture_fill_format.tile_offset_x = -275
picture_fill_format.tile_offset_y = -247
picture_fill_format.tile_scale_x = 120
picture_fill_format.tile_scale_y = 120
picture_fill_format.tile_alignment = slides.RectangleAlignment.BOTTOM_RIGHT
picture_fill_format.tile_flip = slides.TileFlip.FLIP_BOTH
```

#### การบันทึกการนำเสนอ

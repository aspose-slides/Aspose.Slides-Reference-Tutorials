## การแนะนำ

## คู่มือการใช้งาน
### การแก้ไขคุณสมบัติของตารางสไลด์
การปรับคุณสมบัติของกริด โดยเฉพาะระยะห่างระหว่างแถวและคอลัมน์ อาจมีความสำคัญต่อการบรรลุเค้าโครงที่สวยงาม
#### การตั้งค่าวัตถุการนำเสนอ
เริ่มต้นด้วยการสร้างวัตถุการนำเสนอใหม่ซึ่งคุณจะใช้การตั้งค่ากริด:
```python
import aspose.slides as slides

def set_grid_properties():
    # สร้างวัตถุการนำเสนอใหม่
    with slides.Presentation() as pres:
        # กำหนดระยะห่างระหว่างแถวและคอลัมน์ (เป็นหน่วยจุด)
        pres.view_properties.grid_spacing = 72
        
        # บันทึกการนำเสนอที่แก้ไขแล้วลงในไดเร็กทอรีผลลัพธ์ของคุณ
        pres.save("YOUR_OUTPUT_DIRECTORY/GridProperties-out.pptx", slides.export.SaveFormat.PPTX)
# หากต้องการดำเนินการให้เรียกใช้ฟังก์ชัน
def main():
    set_grid_properties()

if __name__ == "__main__":
    main()
```
#### ทำความเข้าใจพารามิเตอร์หลัก
- **`grid_spacing`**:พารามิเตอร์นี้จะกำหนดระยะห่างระหว่างแถวและคอลัมน์เป็นหน่วยจุด การปรับค่านี้จะช่วยให้มีพื้นที่ว่างมากขึ้นหรือกริดจะแคบลงตามต้องการ

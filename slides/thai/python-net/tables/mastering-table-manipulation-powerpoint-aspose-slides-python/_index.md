## คู่มือการใช้งาน

### การเข้าถึงและแก้ไขตารางใน PowerPoint

#### ภาพรวม
เราจะเปิดไฟล์ PPTX ที่มีอยู่ ค้นหาตารางที่ต้องการ อัปเดตเนื้อหา และบันทึกการเปลี่ยนแปลง กระบวนการนี้เหมาะอย่างยิ่งสำหรับการอัปเดตข้อมูลการนำเสนอแบบแบตช์

#### ขั้นตอน
1. **เปิดการนำเสนอของคุณ**
   โหลดไฟล์ PowerPoint ของคุณ:
   ```python
   with slides.Presentation("YOUR_DOCUMENT_DIRECTORY/tables_update.pptx") as presentation:
       slide = presentation.slides[0]
   ```
   โค้ดนี้จะเปิดไฟล์และเข้าถึงสไลด์แรก

2. **ค้นหาและอัปเดตตาราง**
   ระบุและอัปเดตเซลล์ตาราง:
   ```python
   for shape in slide.shapes:
       if isinstance(shape, slides.Table):
           # อัปเดตข้อความในเซลล์เฉพาะ
           shape.rows[0][1].text_frame.text = "New"
   ```
   สไนปเป็ตนี้จะอัปเดตเซลล์ที่ต้องการภายในแถวแรก

3. **บันทึกการเปลี่ยนแปลงของคุณ**
   บันทึกการนำเสนอที่อัปเดตของคุณ:
   ```python
   presentation.save("YOUR_OUTPUT_DIRECTORY/tables_update_table_out.pptx", slides.export.SaveFormat.PPTX)
   ```
   คำสั่งนี้จะเขียนการเปลี่ยนแปลงลงในดิสก์ในรูปแบบ PPTX

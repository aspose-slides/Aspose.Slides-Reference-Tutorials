## คู่มือการใช้งาน

มาดูขั้นตอนการจัดรูปแบบข้อความภายในแถวตารางใน PowerPoint โดยใช้ Aspose.Slides กัน

### การเข้าถึงและการจัดรูปแบบแถวตาราง (H2)

#### ภาพรวม
เราจะเริ่มต้นด้วยการโหลดงานนำเสนอที่มีอยู่ เข้าถึงตารางเฉพาะภายในนั้น และนำตัวเลือกการจัดรูปแบบต่างๆ ไปใช้กับแถวต่างๆ

#### ขั้นตอนที่ 1: โหลดงานนำเสนอของคุณ

ขั้นแรก ให้สร้างหรือเปิดไฟล์ PowerPoint ที่มีตาราง:

```python
input_presentation = 'YOUR_DOCUMENT_DIRECTORY/tables.pptx'
output_presentation = 'YOUR_OUTPUT_DIRECTORY/tables_text_format_inside_row_out.pptx'

with slides.Presentation(input_presentation) as presentation:
    # เข้าถึงรูปร่างแรกในสไลด์แรก ถือว่าเป็นตาราง
    table = presentation.slides[0].shapes[0]
```

#### ขั้นตอนที่ 2: ตั้งค่าความสูงของแบบอักษรสำหรับเซลล์ในแถวแรก

ปรับขนาดตัวอักษรโดยใช้ `PortionFormat`-

```python
# ตั้งค่าความสูงของแบบอักษรสำหรับเซลล์ในแถวแรก
portion_format = slides.PortionFormat()
portion_format.font_height = 25  # เปลี่ยนความสูงของตัวอักษรตามต้องการ
table.rows[0].set_text_format(portion_format)
```

**คำอธิบาย:** การ `font_height` พารามิเตอร์ควบคุมขนาดของข้อความในแต่ละเซลล์เพื่อเพิ่มการมองเห็น

#### ขั้นตอนที่ 3: จัดแนวข้อความและตั้งค่าระยะขอบ

ในการจัดตำแหน่งข้อความในเซลล์ของแถวแรกให้ชิดขวา ให้ทำดังนี้:

```python
# ตั้งค่าการจัดตำแหน่งข้อความและระยะขอบขวาสำหรับเซลล์ในแถวแรก
paragraph_format = slides.ParagraphFormat()
paragraph_format.alignment = slides.TextAlignment.RIGHT
paragraph_format.margin_right = 20  # พื้นที่จากขอบด้านขวา
table.rows[0].set_text_format(paragraph_format)
```

**คำอธิบาย:** `ParagraphFormat` ช่วยให้คุณจัดแนวข้อความและกำหนดระยะขอบ ให้รูปลักษณ์สวยงาม

#### ขั้นตอนที่ 4: ตั้งค่าประเภทข้อความแนวตั้งสำหรับเซลล์ในแถวที่สอง

สำหรับการวางแนวข้อความแนวตั้ง:

```python
# ตั้งค่าชนิดข้อความแนวตั้งสำหรับเซลล์ในแถวที่สอง
text_frame_format = slides.TextFrameFormat()
text_frame_format.text_vertical_type = slides.TextVerticalType.VERTICAL
table.rows[1].set_text_format(text_frame_format)
```

**คำอธิบาย:** `TextFrameFormat` เปลี่ยนแปลงวิธีการแสดงข้อความ ซึ่งอาจเป็นประโยชน์กับภาษาต่างๆ เช่น ญี่ปุ่นหรือจีน

#### ขั้นตอนที่ 5: บันทึกการนำเสนอของคุณ

สุดท้ายให้บันทึกการเปลี่ยนแปลงไปยังไฟล์ใหม่:

```python
# บันทึกการนำเสนอที่แก้ไขไปยังไฟล์ใหม่ในไดเร็กทอรีเอาท์พุต
table.save(output_presentation, slides.export.SaveFormat.PPTX)
```

## การประยุกต์ใช้งานจริง (H2)

ต่อไปนี้คือสถานการณ์จริงบางส่วนที่ฟังก์ชันนี้มีประโยชน์:

1. **รายงานทางธุรกิจ**การปรับแต่งตารางเพื่อเน้นตัวเลขสำคัญหรือจุดข้อมูลในการนำเสนอขององค์กร
2. **สื่อการเรียนรู้**:เพิ่มความสามารถในการอ่านด้วยข้อความแนวตั้งสำหรับสไลด์การเรียนรู้ภาษา
3. **โบรชัวร์การตลาด**:การจัดเรียงและปรับแต่งเนื้อหาตารางให้เหมาะกับมาตรฐานความสวยงามของวัสดุของแบรนด์

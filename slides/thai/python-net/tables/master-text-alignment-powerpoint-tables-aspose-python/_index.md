## คู่มือการใช้งาน

เราจะแบ่งกระบวนการจัดตำแหน่งข้อความแนวตั้งภายในเซลล์ตารางออกเป็นขั้นตอนต่างๆ ที่สามารถจัดการได้

### การเข้าถึงสไลด์และการเพิ่มตาราง

ขั้นแรก เราต้องเข้าถึงสไลด์และกำหนดขนาดของตารางของเรา:

```python
with slides.Presentation() as presentation:
    slide = presentation.slides[0]
    dbl_cols = [120, 120, 120, 120]
    dbl_rows = [100, 100, 100, 100]

    # เพิ่มตารางลงในสไลด์
    tbl = slide.shapes.add_table(100, 50, dbl_cols, dbl_rows)
```

### การแทรกและการจัดตำแหน่งข้อความ

ขั้นตอนต่อไปคือการแทรกข้อความลงในเซลล์และจัดแนวแนวตั้ง:

```python
# แทรกข้อความลงในเซลล์ที่เฉพาะเจาะจง
tbl.rows[1][0].text_frame.text = "10"
tbl.rows[2][0].text_frame.text = "20"
tbl.rows[3][0].text_frame.text = "30"

# เข้าถึงกรอบข้อความของเซลล์แรกเพื่อแก้ไขคุณสมบัติ
text_frame = tbl.rows[0][0].text_frame
paragraph = text_frame.paragraphs[0]
portion = paragraph.portions[0]

# ตั้งค่าข้อความและรูปแบบสำหรับส่วนนี้
portion.text = "Text here"
portion.portion_format.fill_format.fill_type = slides.FillType.SOLID
portion.portion_format.fill_format.solid_fill_color.color = drawing.Color.black

# จัดตำแหน่งข้อความตามแนวตั้ง
cell = tbl.rows[0][0]
cell.text_anchor_type = slides.TextAnchorType.CENTER
cell.text_vertical_type = slides.TextVerticalType.VERTICAL270
```

### การบันทึกการนำเสนอของคุณ

สุดท้ายให้บันทึกการนำเสนอที่แก้ไขของคุณ:

```python
presentation.save("YOUR_OUTPUT_DIRECTORY/tables_vertical_align_text_out.pptx", slides.export.SaveFormat.PPTX)
```

2. **การออกแบบเชิงสร้างสรรค์**:ใช้การจัดตำแหน่งแนวตั้งในส่วนหัวหรือส่วนพิเศษเพื่อสร้างองค์ประกอบที่โดดเด่นทางสายตา
3. **ข้อความเฉพาะภาษา**:จัดตำแหน่งข้อความหลายภาษาในแนวตั้งเพื่อรองรับทิศทางการเขียนที่แตกต่างกัน

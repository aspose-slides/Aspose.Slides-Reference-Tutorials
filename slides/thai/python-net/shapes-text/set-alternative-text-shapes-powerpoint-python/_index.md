## คู่มือการใช้งาน
ตอนนี้เรามาดูขั้นตอนการตั้งค่าข้อความทางเลือกสำหรับรูปร่างในงานนำเสนอ PowerPoint กัน

### การตั้งค่าสภาพแวดล้อมการนำเสนอของคุณ
ขั้นแรก เราต้องตั้งค่าเส้นทางเอกสารและสร้างอินสแตนซ์คลาสการนำเสนอ ขั้นตอนนี้เกี่ยวข้องกับการสร้างหรือโหลดไฟล์ PPTX ที่มีอยู่ ซึ่งคุณสามารถจัดการรูปร่างได้

#### เริ่มต้นเส้นทางและการนำเสนอคลาส

```python
import aspose.slides as slides
import os

document_directory = "YOUR_DOCUMENT_DIRECTORY/"
output_directory = "YOUR_OUTPUT_DIRECTORY/"

# ตรวจสอบให้แน่ใจว่ามีไดเร็กทอรีเอาท์พุตอยู่
if not os.path.exists(output_directory):
    os.makedirs(output_directory)

with slides.Presentation() as pres:
    # รหัสของคุณอยู่ที่นี่
```

### การเพิ่มรูปร่างลงในสไลด์
ต่อไปเรามาเพิ่มรูปทรงต่างๆ ลงในสไลด์กัน ตัวอย่างนี้รวมถึงการเพิ่มรูปสี่เหลี่ยมผืนผ้าและวัตถุรูปพระจันทร์

#### เพิ่มรูปสี่เหลี่ยมผืนผ้า

```python
# รับสไลด์แรกจากการนำเสนอ
slide = pres.slides[0]

# เพิ่มรูปสี่เหลี่ยมผืนผ้า
shape1 = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 50, 40, 150, 50)
```

#### เพิ่มวัตถุรูปพระจันทร์ด้วยการเติมสี

```python
# เพิ่มวัตถุที่มีรูปร่างเหมือนดวงจันทร์และตั้งค่าสีเติมเป็นสีเทา
define shape2 = slide.shapes.add_auto_shape(slides.ShapeType.MOON, 160, 40, 150, 50)
shape2.fill_format.fill_type = slides.FillType.SOLID
shape2.fill_format.solid_fill_color.color = drawing.Color.gray
```

### การตั้งค่าข้อความทางเลือกสำหรับรูปร่าง
สุดท้าย ให้ทำซ้ำในแต่ละรูปร่างในสไลด์และกำหนดข้อความทางเลือก ขั้นตอนนี้มีความสำคัญต่อการเข้าถึง

```python
# ทำซ้ำในแต่ละรูปร่างในสไลด์และตั้งค่าข้อความทางเลือกสำหรับรูปร่างอัตโนมัติ
define for shape in slide.shapes:
    if isinstance(shape, slides.AutoShape):
        shape.alternative_text = "User Defined"
```

### การบันทึกการนำเสนอของคุณ
ตรวจสอบให้แน่ใจว่าคุณบันทึกการนำเสนอของคุณหลังจากทำการเปลี่ยนแปลง:

```python
pres.save(os.path.join(output_directory, "shapes_set_alternative_text_out.pptx"), slides.export.SaveFormat.PPTX)
```

# วิธีการใช้เอฟเฟ็กต์เอียงกับรูปร่างใน PowerPoint โดยใช้ Aspose.Slides และ Python

## การแนะนำ
การสร้างงานนำเสนอที่ดึงดูดสายตาเป็นสิ่งสำคัญในการดึงดูดความสนใจของผู้ชม บทช่วยสอนนี้จะแนะนำคุณเกี่ยวกับการปรับปรุงรูปร่างในสไลด์ PowerPoint โดยใช้ไลบรารี Aspose.Slides ที่ทรงพลังพร้อม Python โดยเน้นที่การใช้เอฟเฟกต์เอียงเพื่อเพิ่มความลึกและความซับซ้อน

**การเริ่มต้นขั้นพื้นฐาน:**
หากต้องการเริ่มใช้ Aspose.Slides ในสคริปต์ Python ของคุณ โปรดนำเข้าโมดูลที่จำเป็นและสร้างอินสแตนซ์ของคลาส Presentation:
```python
import aspose.slides as slides
from aspose.pydrawing import Color

class Presentation:
    def __enter__(self):
        self.pres = slides.Presentation()
        return self.pres

    def __exit__(self, exc_type, exc_value, traceback):
        self.pres.dispose()

with Presentation() as pres:
    # รหัสของคุณอยู่ที่นี่
```
การตั้งค่านี้ช่วยให้เราเตรียมเราในการนำเอฟเฟ็กต์เอียงไปใช้กับรูปร่างใน PowerPoint

## คู่มือการใช้งาน
### การเพิ่มรูปร่างและการกำหนดค่าคุณสมบัติ
#### ภาพรวม
เราจะเพิ่มรูปร่างวงรีลงในสไลด์ของเรา กำหนดค่าคุณสมบัติการเติมและเส้น และใช้เอฟเฟกต์เอียง 3 มิติเพื่อให้ดูสวยงาม

#### เพิ่มรูปทรงวงรี
ขั้นแรก ให้เพิ่มรูปร่างวงรีพื้นฐาน:
```python
# เข้าถึงสไลด์แรกในการนำเสนอ
slide = pres.slides[0]

# เพิ่มรูปวงรีให้กับสไลด์
shape = slide.shapes.add_auto_shape(
    slides.ShapeType.ELLIPSE, 30, 30, 100, 100
)
```
โค้ดนี้จะสร้างวงรีเรียบง่ายในตำแหน่ง (30,30) และมีขนาด 100x100

#### ตั้งค่าคุณสมบัติการเติมและเส้น
ถัดไป กำหนดสีเติมและคุณสมบัติเส้นสำหรับรูปร่างของเรา:
```python
# ตั้งค่าประเภทการเติมเป็นแบบทึบและเลือกสีเขียว
drawing.Color.green
shape.fill_format.fill_type = slides.FillType.SOLID
shape.fill_format.solid_fill_color.color = Color.green

# กำหนดรูปแบบเส้นด้วยการเติมสีทึบสีส้มและกำหนดความกว้าง
type: solid
fill_format = shape.line_format.fill_format
fill_format.fill_type = slides.FillType.SOLID
fill_format.solid_fill_color.color = Color.orange
shape.line_format.width = 2.0
```
การตั้งค่าเหล่านี้ทำให้รูปวงรีของเราโดดเด่นบนสไลด์

#### ใช้เอฟเฟ็กต์ 3D Bevel
ขั้นตอนสุดท้ายคือการใช้เอฟเฟกต์เอียงเพื่อเพิ่มความลึก:
```python
# กำหนดค่ารูปแบบ 3 มิติของรูปร่างและใช้เอฟเฟกต์เอียงแบบวงกลม
type: circle
shape.three_d_format.depth = 4
shape.three_d_format.bevel_top.bevel_type = slides.BevelPresetType.CIRCLE
shape.three_d_format.bevel_top.height = 6
shape.three_d_format.bevel_top.width = 6

# ตั้งค่ากล้องและแสงเพื่อให้ได้เอฟเฟกต์ที่สมจริง
type: orthographic_front
camera = shape.three_d_format.camera
camera.camera_type = slides.CameraPresetType.ORTHOGRAPHIC_FRONT
light_rig = shape.three_d_format.light_rig
light_rig.light_type = slides.LightRigPresetType.THREE_PT
light_rig.direction = slides.LightingDirection.TOP
```
การกำหนดค่าเหล่านี้สร้างเอฟเฟกต์ 3 มิติที่ดึงดูดสายตา ช่วยให้การนำเสนอมีความสวยงามมากขึ้น

#### บันทึกการนำเสนอของคุณ
สุดท้ายให้บันทึกการเปลี่ยนแปลงของคุณ:
```python
# ระบุไดเรกทอรีและชื่อไฟล์สำหรับบันทึกการนำเสนอ
directory = "YOUR_OUTPUT_DIRECTORY"

```

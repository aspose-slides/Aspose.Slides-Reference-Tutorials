# วิธีการเพิ่มรูปร่างวงรีลงในสไลด์ PowerPoint โดยใช้ Aspose.Slides ใน Python

## การแนะนำ

เพิ่มประสิทธิภาพการนำเสนอ PowerPoint ของคุณด้วยการเพิ่มรูปร่างที่กำหนดเอง เช่น วงรี ไม่ว่าคุณจะกำลังสร้างรายงานอัตโนมัติหรือสร้างสไลด์ที่ดึงดูดสายตา การรวมรูปร่างเหล่านี้เข้าด้วยกันก็สามารถสร้างการเปลี่ยนแปลงได้ บทช่วยสอนนี้จะแนะนำคุณเกี่ยวกับการใช้ Aspose.Slides สำหรับ Python เพื่อเพิ่มรูปร่างวงรีในสไลด์แรกของการนำเสนอ PowerPoint ใหม่

เมื่ออ่านคู่มือนี้จบ คุณจะรู้วิธีผสานรูปทรงต่างๆ เข้ากับงานนำเสนอของคุณได้อย่างราบรื่นและง่ายดาย

- การทำงาน `pip` การติดตั้งเพื่อการบริหารจัดการห้องสมุด
- IDE หรือโปรแกรมแก้ไขข้อความสำหรับเขียนและรันสคริปต์ Python

### การเพิ่มรูปร่างวงรีลงในสไลด์ (H3)

#### ขั้นตอนที่ 1: สร้างอินสแตนซ์การนำเสนอ (H4)
สร้างอินสแตนซ์ของ `Presentation` คลาสที่แสดงไฟล์ PowerPoint ของคุณ
```python
import aspose.slides as slides

    with slides.Presentation() as pres:
```

#### ขั้นตอนที่ 2: เข้าถึงสไลด์แรก (H4)
แก้ไขสไลด์แรกเพื่อแทรกวงรีของคุณ
```python
        # เข้าถึงสไลด์แรก
        slide = pres.slides[0]
```

#### ขั้นตอนที่ 3: เพิ่มรูปร่างวงรี (H4)
แทรกวงรีที่ตำแหน่งที่ระบุด้วยมิติที่กำหนดโดยใช้ `add_auto_shape` วิธี.
```python
        # แทรกรูปวงรีเข้าไปในสไลด์
        slide.shapes.add_auto_shape(slides.ShapeType.ELLIPSE, 50, 150, 150, 50)
```
ที่นี่:
- **ShapeType.รูปวงรี**: ระบุรูปร่างเป็นรูปวงรี
- **50, 150**:พิกัด x และ y สำหรับการวางตำแหน่งบนสไลด์
- **150, 50**:ความกว้างและความสูงของวงรี

#### ขั้นตอนที่ 4: บันทึกการนำเสนอ (H4)
บันทึกการนำเสนอของคุณไปยังตำแหน่งที่ต้องการในรูปแบบ PPTX:
```python
        # บันทึกการนำเสนอที่ปรับเปลี่ยนแล้ว
        pres.save("YOUR_OUTPUT_DIRECTORY/shapes_add_ellipse_out.pptx", slides.export.SaveFormat.PPTX)
```

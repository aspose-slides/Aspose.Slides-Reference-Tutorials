## คู่มือการใช้งาน

มาแบ่งกระบวนการในการเพิ่มการเยื้องหัวข้อย่อยและการจัดรูปแบบย่อหน้าให้เป็นส่วนที่จัดการได้

### การเพิ่มรูปร่างลงในสไลด์

#### ภาพรวม

ขั้นแรก เราต้องเพิ่มรูปร่างลงในสไลด์เพื่อใส่ข้อความ ซึ่งจะช่วยให้จัดระเบียบเนื้อหาได้เป็นระเบียบเรียบร้อย

#### ขั้นตอน:

1. **รับสไลด์แรก**:เข้าถึงสไลด์แรกของการนำเสนอของคุณ
2. **เพิ่มรูปสี่เหลี่ยมผืนผ้า**: ใช้ `add_auto_shape` เพื่อสร้างสี่เหลี่ยมผืนผ้าสำหรับเก็บข้อความ

```python
# รับสไลด์แรก
slide = pres.slides[0]

# เพิ่มรูปทรงสี่เหลี่ยมผืนผ้าลงในสไลด์
rect = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 100, 100, 500, 150)
```

### การแทรกและการจัดรูปแบบข้อความ

#### ภาพรวม

เมื่อเรามีรูปร่างแล้ว ก็ถึงเวลาที่จะแทรกข้อความและจัดรูปแบบเพื่อความชัดเจนและผลกระทบ

#### ขั้นตอน:

1. **เพิ่มกรอบข้อความ**: สร้าง `TextFrame` ที่จะถือข้อความของคุณ
2. **ประเภทปรับพอดีอัตโนมัติ**:ทำให้แน่ใจว่าข้อความจะพอดีกับสี่เหลี่ยมโดยอัตโนมัติ
3. **ลบเส้นขอบ**:เพื่อความชัดเจนทางภาพ ให้ลบเส้นขอบของรูปร่างออก

```python
# เพิ่ม TextFrame ลงในสี่เหลี่ยมผืนผ้า
tf = rect.add_text_frame("This is first line \r\nThis is second line \r\nThis is third line")

# ตั้งค่าข้อความให้พอดีกับรูปร่างโดยอัตโนมัติ
tf.text_frame_format.autofit_type = slides.TextAutofitType.SHAPE

# ลบเส้นขอบของสี่เหลี่ยมผืนผ้าเพื่อความชัดเจนทางสายตา
rect.line_format.fill_format.fill_type = slides.FillType.NONE
```

### การปรับแต่งรูปแบบหัวข้อย่อยและการเยื้อง

#### ภาพรวม

พลังที่แท้จริงอยู่ที่การปรับแต่งรูปแบบของหัวข้อย่อยและการปรับการเยื้องย่อหน้าเพื่อให้เนื้อหาของคุณน่าดึงดูดสายตา

#### ขั้นตอน:

1. **ตั้งค่ารูปแบบกระสุน**: กำหนดประเภทและลักษณะของหัวข้อย่อยสำหรับแต่ละย่อหน้า
2. **ปรับตำแหน่งและความลึก**: จัดตำแหน่งข้อความและกำหนดระดับความลึกสำหรับลำดับชั้น
3. **กำหนดการเยื้อง**: ระบุค่าการเยื้องที่แตกต่างกันสำหรับระยะห่างที่หลากหลาย

```python
# รูปแบบย่อหน้าแรก: ตั้งค่ารูปแบบหัวข้อย่อย สัญลักษณ์ การจัดตำแหน่ง และการเยื้อง
def format_paragraph(para, char, align, depth, indent):
    para.paragraph_format.bullet.type = slides.BulletType.SYMBOL
    para.paragraph_format.bullet.char = char
    para.paragraph_format.alignment = align
    para.paragraph_format.depth = depth
    para.paragraph_format.indent = indent

para1 = tf.paragraphs[0]
format_paragraph(para1, chr(8226), slides.TextAlignment.LEFT, 2, 30)

# ทำซ้ำสำหรับย่อหน้าที่สองและสามด้วยค่าการเยื้องที่ต่างกัน
def format_multiple_paragraphs(paragraphs):
    for i, para in enumerate(paragraphs[1:], start=1):
        format_paragraph(para, chr(8226), slides.TextAlignment.LEFT, 4, 40 + i * 10)

format_multiple_paragraphs(tf.paragraphs)
```

### การบันทึกการนำเสนอของคุณ

หลังจากปรับแต่งทั้งหมดแล้ว ให้บันทึกการนำเสนอของคุณเพื่อเก็บรักษาการเปลี่ยนแปลง:

```python

```

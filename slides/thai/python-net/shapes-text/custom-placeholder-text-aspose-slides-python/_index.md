## คู่มือการใช้งาน
มาดูกระบวนการเพิ่มข้อความตัวแทนแบบกำหนดเองลงในงานนำเสนอ PowerPoint กัน

### การเพิ่มข้อความตัวแทนแบบกำหนดเอง
ปรับเปลี่ยนตัวแทน เช่น ชื่อเรื่องและคำบรรยายใต้ภาพด้วยคำแนะนำหรือข้อความที่กำหนดเองโดยใช้ Aspose.Slides สำหรับ Python

#### คำแนะนำทีละขั้นตอน
**ขั้นตอนที่ 1: กำหนดเส้นทางของคุณ**
ตั้งค่าเส้นทางไปยังไฟล์อินพุตและเอาต์พุตของคุณ แทนที่ `'YOUR_DOCUMENT_DIRECTORY'` และ `'YOUR_OUTPUT_DIRECTORY'` ด้วยไดเร็กทอรีจริงบนระบบของคุณ

```python
document_path = 'YOUR_DOCUMENT_DIRECTORY/text_add_custom_placeholder_text.pptx'
output_path = 'YOUR_OUTPUT_DIRECTORY/text_add_custom_placeholder_text_out.pptx'
```

**ขั้นตอนที่ 2: เปิดการนำเสนอ**
เปิดไฟล์ PowerPoint ของคุณโดยใช้ Aspose.Slides โดยเริ่มต้น `Presentation` วัตถุ.

```python
def add_custom_prompt_text():
    with slides.Presentation(document_path) as pres:
        slide = pres.slides[0]
```

**ขั้นตอนที่ 3: ทำซ้ำผ่านรูปร่างสไลด์**
วนซ้ำผ่านรูปร่างต่างๆ บนสไลด์แรกของคุณและตรวจสอบช่องว่าง

```python
for shape in slide.shapes:
    if isinstance(shape, slides.AutoShape) and shape.placeholder is not None:
        text = ''
        # ตรวจสอบประเภทตัวแทนและตั้งค่าข้อความกำหนดเองตามนั้น
```

**ขั้นตอนที่ 4: ตั้งค่าข้อความตัวแทนแบบกำหนดเอง**
กำหนดประเภทตัวแทนและกำหนดข้อความแบบกำหนดเองที่เหมาะสม

```python
if shape.placeholder.type == slides.PlaceholderType.CENTERED_TITLE:
    text = 'Click to add a custom title'
elif shape.placeholder.type == slides.PlaceholderType.SUBTITLE:
    text = 'Click to add a custom subtitle'

shape.text_frame.text = text
```

**ขั้นตอนที่ 5: บันทึกการนำเสนอที่แก้ไขแล้ว**
หลังจากแก้ไขตัวแทนแล้ว ให้บันทึกการนำเสนอของคุณ

```python
pres.save(output_path, slides.export.SaveFormat.PPTX)
```

1. **การนำเสนอแบบโต้ตอบ**:ส่งเสริมการมีส่วนร่วมของผู้ชมโดยให้คำแนะนำที่ชัดเจนโดยตรงบนสไลด์

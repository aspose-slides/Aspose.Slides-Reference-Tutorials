# รายการหัวข้อย่อยที่มีหมายเลขกำหนดเองใน PowerPoint โดยใช้ Aspose.Slides สำหรับ Python

## การแนะนำ
คุณกำลังมองหาวิธีเพิ่มความสวยงามให้กับงานนำเสนอ PowerPoint ของคุณนอกเหนือจากการใช้จุดหัวข้อเริ่มต้นอยู่หรือไม่ ไม่ว่าจะเป็นรายงานขององค์กร การบรรยายทางวิชาการ หรือการประชุมทางธุรกิจ การปรับแต่งรายการหัวข้อสามารถดึงดูดและรักษาความสนใจของผู้ชมของคุณได้อย่างมีประสิทธิภาพมากขึ้น **Aspose.Slides สำหรับ Python**คุณมีความยืดหยุ่นในการปรับแต่งรายการหัวข้อย่อยที่มีหมายเลขตามความต้องการในการจัดรูปแบบเฉพาะของคุณได้

ในคู่มือฉบับสมบูรณ์นี้ เราจะสาธิตวิธีตั้งค่าสัญลักษณ์หัวข้อย่อยที่มีหมายเลขแบบกำหนดเองโดยใช้ Aspose.Slides ใน PowerPoint ด้วย Python การรวมฟีเจอร์นี้เข้ากับงานนำเสนอของคุณ จะช่วยให้คุณได้รูปลักษณ์ที่ดูเป็นมืออาชีพและสวยงาม

ก่อนที่จะใช้การกำหนดหมายเลขหัวข้อย่อยแบบกำหนดเองกับ Aspose.Slides สำหรับ Python โปรดตรวจสอบให้แน่ใจว่าคุณมี:

### ห้องสมุดที่จำเป็น:
- **Aspose.Slides สำหรับ Python**:ไลบรารีที่แข็งแกร่งสำหรับการสร้างและจัดการการนำเสนอ PowerPoint

### การตั้งค่าสภาพแวดล้อม:
- Python 3.x ติดตั้งอยู่บนระบบของคุณแล้ว
- ความเข้าใจพื้นฐานเกี่ยวกับแนวคิดการเขียนโปรแกรม Python เป็นสิ่งที่มีประโยชน์แต่ไม่ใช่สิ่งบังคับ

## การตั้งค่า Aspose.Slides สำหรับ Python
ในการเริ่มต้น ให้ติดตั้ง `aspose.slides` ไลบรารีที่ใช้ pip:

```bash
pip install aspose.slides
```

### การได้มาซึ่งใบอนุญาต:
Aspose.Slides เป็นผลิตภัณฑ์เชิงพาณิชย์ที่ให้ทดลองใช้งานฟรีเพื่อทดสอบความสามารถต่างๆ คุณสามารถขอรับใบอนุญาตชั่วคราวหรือซื้อใบอนุญาตเพื่อใช้งานต่อได้

- **ทดลองใช้งานฟรี**:เข้าถึงฟังก์ชันพื้นฐานโดยไม่มีข้อจำกัด
- **ใบอนุญาตชั่วคราว**:ขอเข้าถึงแบบเต็มเป็นการชั่วคราวได้ที่เว็บไซต์ Aspose
- **ซื้อ**:ควรพิจารณาซื้อใบอนุญาตสำหรับโครงการระยะยาว

### การเริ่มต้นขั้นพื้นฐาน:
เมื่อติดตั้งแล้ว ให้เริ่มการนำเสนอของคุณดังนี้:

```python
import aspose.slides as slides

with slides.Presentation() as presentation:
    # รหัสของคุณที่นี่...
```

การตั้งค่านี้จะเตรียมสภาพแวดล้อมสำหรับการเพิ่มหัวข้อย่อยที่มีหมายเลขแบบกำหนดเองลงในสไลด์ PowerPoint ของคุณ

## คู่มือการใช้งาน
มาเริ่มสร้างรายการหัวข้อย่อยแบบมีหมายเลขแบบกำหนดเองกัน แต่ละขั้นตอนจะถูกแบ่งย่อยเพื่อความชัดเจนและง่ายต่อการนำไปใช้

### การเพิ่มรูปสี่เหลี่ยมผืนผ้าด้วยกรอบข้อความ
#### ภาพรวม:
ขั้นแรก ให้เพิ่มรูปร่างที่จะประกอบด้วยกรอบข้อความสำหรับจุดหัวข้อย่อย

```python
# เพิ่มรูปสี่เหลี่ยมผืนผ้าลงในสไลด์แรก
shape = presentation.slides[0].shapes.add_auto_shape(
    slides.ShapeType.RECTANGLE, 200, 200, 400, 200)
```
- **คำอธิบายพารามิเตอร์**: เดอะ `add_auto_shape` วิธีนี้ใช้พารามิเตอร์สำหรับประเภทรูปร่าง (สี่เหลี่ยมผืนผ้า) ตำแหน่ง (พิกัด x และ y) และขนาด (ความกว้างและความสูง)

### การกำหนดค่าเฟรมข้อความ
#### ภาพรวม:
เข้าถึงกรอบข้อความของสี่เหลี่ยมผืนผ้าเพื่อเพิ่มจุดหัวข้อย่อย

```python
# เข้าถึงกรอบข้อความของรูปร่างอัตโนมัติที่สร้างขึ้น
text_frame = shape.text_frame

# ลบย่อหน้าเริ่มต้นที่มีอยู่ทั้งหมดออกหากมีอยู่
text_frame.paragraphs.clear()
```
- **วัตถุประสงค์**:ช่วยให้แน่ใจว่าไม่มีข้อผิดพลาดใดๆ ก่อนที่จะเพิ่มจุดหัวข้อแบบกำหนดเอง

### การเพิ่มหัวข้อย่อยที่มีหมายเลขกำหนดเอง
#### ภาพรวม:
เพิ่มย่อหน้าด้วยการตั้งค่าหัวข้อย่อยเฉพาะ:

```python
# เพิ่มย่อหน้าด้วยเครื่องหมายหัวข้อย่อยที่มีหมายเลขกำหนดเอง
for start_number, bullet_text in [(2, "bullet 2"), (3, "bullet 3"), (7, "bullet 7")]:
    paragraph = slides.Paragraph()
    paragraph.text = bullet_text
    paragraph.paragraph_format.depth = 4
    paragraph.paragraph_format.bullet.numbered_bullet_start_with = start_number
    paragraph.paragraph_format.bullet.type = slides.BulletType.NUMBERED
    text_frame.paragraphs.add(paragraph)
```
- **การกำหนดค่า**:แต่ละย่อหน้าจะเริ่มต้นด้วยตัวเลขที่เฉพาะเจาะจง ซึ่งให้ความยืดหยุ่นและการควบคุมในการจัดรูปแบบการนำเสนอ

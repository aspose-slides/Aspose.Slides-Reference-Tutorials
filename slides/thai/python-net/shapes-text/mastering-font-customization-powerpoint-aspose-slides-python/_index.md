```python
import aspose.slides as slides

# สร้างอินสแตนซ์ของคลาส Presentation ซึ่งแสดงไฟล์ PowerPoint
class FontCustomizationTutorial:
    def __init__(self):
        self.pres = slides.Presentation()
    
    def add_rectangle_shape(self):
        slide = self.pres.slides[0]
        auto_shape = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 50, 50, 200, 50)
        return auto_shape
```

## คู่มือการใช้งาน

### การเพิ่มรูปร่างและการตั้งค่าคุณสมบัติแบบอักษรในสไลด์ PowerPoint

#### ภาพรวม
หัวข้อนี้จะแนะนำคุณเกี่ยวกับการเพิ่มรูปทรงสี่เหลี่ยมผืนผ้าในสไลด์ของคุณและปรับแต่งคุณสมบัติแบบอักษรโดยใช้ Aspose.Slides สำหรับ Python

**1. การสร้างคลาสการนำเสนอ**
เริ่มต้นด้วยการสร้างอินสแตนซ์ของ `Presentation` คลาสซึ่งทำหน้าที่เป็นจุดเข้าสู่การจัดการไฟล์ PowerPoint ของคุณ

```python
class FontCustomizationTutorial:
    def __init__(self):
        self.pres = slides.Presentation()

# เพิ่มรูปทรงสี่เหลี่ยมผืนผ้าและตั้งค่าคุณสมบัติแบบอักษร
def customize_font(self):
    auto_shape = self.add_rectangle_shape()
    tf = auto_shape.text_frame
    tf.text = "Aspose TextBox"
    port = tf.paragraphs[0].portions[0]
```

**2. ปรับแต่งคุณสมบัติแบบอักษร**
กำหนดค่าคุณสมบัติแบบอักษรต่างๆ เช่น แบบอักษร ตัวหนา ตัวเอียง ขีดเส้นใต้ ขนาด และสี สำหรับข้อความภายในรูปร่าง
- **ตั้งค่าแบบอักษร:**
  
  ```python
  port.portion_format.latin_font = slides.FontData("Times New Roman")
  ```

- **คุณสมบัติตัวหนาและตัวเอียง:**

  ```python
  port.portion_format.font_bold = slides.NullableBool.TRUE
  port.portion_format.font_italic = slides.NullableBool.TRUE
  ```

- **ขีดเส้นใต้ข้อความ:**

  ```python
  port.portion_format.font_underline = slides.TextUnderlineType.SINGLE
  ```

- **ตั้งค่าขนาดและสีของตัวอักษร:**

  ```python
  port.portion_format.font_height = 25
  port.portion_format.fill_format.fill_type = slides.FillType.SOLID
  port.portion_format.fill_format.solid_fill_color.color = drawing.Color.blue
  ```

**3. บันทึกการนำเสนอ**
สุดท้ายให้บันทึกงานนำเสนอที่คุณแก้ไขลงในไดเร็กทอรีที่ต้องการ

```python
self.pres.save("YOUR_OUTPUT_DIRECTORY/text_font_family_out.pptx", slides.export.SaveFormat.PPTX)
```

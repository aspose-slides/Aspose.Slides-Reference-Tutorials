# วิธีการใช้ Gradient Fill กับรูปร่างใน PowerPoint โดยใช้ Aspose.Slides สำหรับ Python

## การแนะนำ

โดยทำตามคู่มือนี้ คุณจะเรียนรู้วิธีการ:
- ตั้งค่าและติดตั้ง Aspose.Slides สำหรับ Python
- สร้างสไลด์ที่มีรูปร่างเป็นวงรี
- ใช้เอฟเฟกต์การเติมแบบไล่ระดับโดยใช้โค้ดสั้นๆ
- เพิ่มประสิทธิภาพการนำเสนอของคุณ

### การเริ่มต้นและการตั้งค่าเบื้องต้น

นำเข้า Aspose.Slides ในสคริปต์ Python ของคุณ:

```python
import aspose.slides as slides
```

เมื่อตั้งค่านี้แล้ว คุณก็พร้อมที่จะใช้การเติมแบบไล่ระดับสีได้แล้ว

## คู่มือการใช้งาน

ในส่วนนี้จะอธิบายขั้นตอนในการเพิ่มการเติมแบบไล่ระดับให้กับรูปร่างวงรี

### ขั้นตอนที่ 1: สร้างตัวอย่างคลาสการนำเสนอ

สร้างอินสแตนซ์ของ `Presentation` ระดับ:

```python
with slides.Presentation() as pres:
    # การดำเนินการสไลด์ไปที่นี่
```

ซึ่งจะทำให้การบริหารจัดการทรัพยากรมีประสิทธิภาพ

### ขั้นตอนที่ 2: เข้าถึงหรือสร้างสไลด์

เข้าถึงสไลด์แรกและสร้างใหม่หากจำเป็น:

```python
slide = pres.slides[0]
```

### ขั้นตอนที่ 3: เพิ่มรูปวงรี

เพิ่มรูปวงรีลงในสไลด์ของคุณ:

```python
shape = slide.shapes.add_auto_shape(slides.ShapeType.ELLIPSE, 50, 150, 75, 150)
```

- `ShapeType.ELLIPSE` ระบุประเภทรูปร่าง
- พารามิเตอร์ (50, 150, 75, 150) กำหนดตำแหน่งและขนาดของวงรี

### ขั้นตอนที่ 4: ใช้การเติมแบบไล่ระดับกับรูปร่าง

กำหนดค่าการเติมไล่ระดับ:

```python
shape.fill_format.fill_type = slides.FillType.GRADIENT
shape.fill_format.gradient_format.gradient_shape = slides.GradientShape.LINEAR
shape.fill_format.gradient_format.gradient_direction = slides.GradientDirection.FROM_CORNER2
```

- **ประเภทการเติม**: ตั้งค่าเป็น `GRADIENT`-
- **รูปร่างและทิศทางการไล่ระดับสี**:สิ่งเหล่านี้จะกำหนดรูปแบบและทิศทางของการเติมไล่ระดับสีของคุณ

### ขั้นตอนที่ 5: เพิ่มจุดหยุดการไล่ระดับสี

กำหนดจุดหยุดการไล่ระดับสีสองจุดสำหรับการเปลี่ยนสี:

```python
shape.fill_format.gradient_format.gradient_stops.add(1.0, slides.PresetColor.PURPLE)
shape.fill_format.gradient_format.gradient_stops.add(0, slides.PresetColor.RED)
```

- `1.0` และ `0` คือตำแหน่งของจุดหยุดความชัน
- `PresetColor.PURPLE` และ `PresetColor.RED` กำหนดสี

### ขั้นตอนที่ 6: บันทึกการนำเสนอของคุณ

บันทึกการนำเสนอที่แก้ไขของคุณ:

```python
pres.save(global_opts.out_dir + "shapes_fill_gradient_out.pptx", slides.export.SaveFormat.PPTX)
```

นี่จะเขียนการเปลี่ยนแปลงของคุณลงในไฟล์ใหม่ที่ชื่อ `shapes_fill_gradient_out-pptx`.

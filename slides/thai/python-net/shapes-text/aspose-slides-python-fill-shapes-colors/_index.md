## คู่มือการใช้งาน
มาแยกรายละเอียดขั้นตอนการเติมรูปทรงด้วยสีทึบกัน

### ภาพรวม: การเติมรูปทรงด้วยสีทึบ
คุณสมบัตินี้ช่วยให้คุณปรับปรุงสไลด์ของคุณได้โดยการเพิ่มรูปทรงสีสัน ทำให้สไลด์น่าสนใจและติดตามได้ง่ายขึ้น

#### ขั้นตอนที่ 1: สร้างอินสแตนซ์การนำเสนอ
เริ่มต้นด้วยการสร้างอินสแตนซ์ของ `Presentation` คลาสนี้จัดการทรัพยากรโดยอัตโนมัติ:
```python
import aspose.slides as slides

with slides.Presentation() as presentation:
    # รหัสของคุณที่นี่
```

#### ขั้นตอนที่ 2: เข้าถึงสไลด์
เข้าถึงสไลด์แรกเพื่อเพิ่มรูปร่าง:
```python
slide = presentation.slides[0]
```

#### ขั้นตอนที่ 3: เพิ่มรูปร่างลงในสไลด์
เพิ่มรูปสี่เหลี่ยมผืนผ้าในตำแหน่งและขนาดที่ระบุ:
```python
shape = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 50, 150, 75, 150)
```

#### ขั้นตอนที่ 4: ตั้งค่าประเภทการเติมเป็นแบบทึบ
ตั้งค่าประเภทการเติมของรูปร่างให้เป็นแบบทึบ:
```python
shape.fill_format.fill_type = slides.FillType.SOLID
```

#### ขั้นตอนที่ 5: กำหนดและใช้สี
กำหนดสี (เช่น สีเหลือง) สำหรับรูปแบบการเติม:
```python
import aspose.pydrawing as drawing

shape.fill_format.solid_fill_color.color = drawing.Color.yellow
```

#### ขั้นตอนที่ 6: บันทึกการนำเสนอของคุณ
บันทึกการนำเสนอที่คุณแก้ไขแล้วไปยังไดเร็กทอรีเอาท์พุต:
```python
directory = "YOUR_OUTPUT_DIRECTORY"
presentation.save(f"{directory}/shapes_filltype_solid_out.pptx", slides.export.SaveFormat.PPTX)
```

### เคล็ดลับการแก้ไขปัญหา
- ให้แน่ใจว่าคุณมีเส้นทางไฟล์ที่ถูกต้องใน `presentation-save()`.
- หากสีไม่ปรากฏตามที่คาดหวัง ให้ตรวจสอบว่าประเภทการเติมและการตั้งค่าสีของคุณถูกนำไปใช้ถูกต้อง

## คู่มือการใช้งาน

### การเพิ่มแผนภูมิคอลัมน์แบบคลัสเตอร์และการเปิดใช้งานตารางข้อมูล

#### ภาพรวม

ประการแรกเราจะเพิ่มแผนภูมิคอลัมน์แบบคลัสเตอร์ลงในการนำเสนอของเราและเปิดใช้งานคุณลักษณะตารางข้อมูล

#### การดำเนินการแบบทีละขั้นตอน

1. **เพิ่มแผนภูมิคอลัมน์แบบคลัสเตอร์:**
   
   เพิ่มโค้ดสั้นๆ ต่อไปนี้เพื่อสร้างแผนภูมิคอลัมน์แบบคลัสเตอร์พื้นฐานในสไลด์แรกของคุณ:

    ```python
    chart = pres.slides[0].shapes.add_chart(
        slides.charts.ChartType.CLUSTERED_COLUMN, 50, 50, 600, 400)
    ```
   
2. **เปิดใช้งานการแสดงตารางข้อมูล:**
   
   ถัดไป เปิดใช้งานตารางข้อมูลสำหรับแผนภูมิเพื่อให้สามารถกำหนดแบบอักษรได้:

    ```python
    chart.has_data_table = True
    ```

### การปรับแต่งคุณสมบัติของแบบอักษร

#### ภาพรวม

เมื่อเปิดใช้งานตารางข้อมูลแล้ว เราสามารถปรับแต่งคุณสมบัติแบบอักษรเพื่อปรับปรุงการอ่านและรูปแบบได้

#### การดำเนินการแบบทีละขั้นตอน

1. **ตั้งค่าแบบอักษรตัวหนา:**
   
   ใช้ชิ้นส่วนนี้เพื่อทำให้ข้อความตารางข้อมูลของคุณเป็นตัวหนา:

    ```python
    chart.chart_data_table.text_format.portion_format.font_bold = slides.NullableBool.TRUE
    ```

2. **ปรับความสูงของตัวอักษร:**
   
   ปรับขนาดตัวอักษรเพื่อให้มองเห็นได้ชัดเจนยิ่งขึ้น:

    ```python
    chart.chart_data_table.text_format.portion_format.font_height = 20
    ```

## การประยุกต์ใช้งานจริง

การปรับแต่งคุณสมบัติแบบอักษรสามารถปรับปรุงการแสดงข้อมูลได้อย่างมีนัยสำคัญในสถานการณ์ต่างๆ:

1. **รายงานทางธุรกิจ:** การแสดงข้อมูลทางการเงินอย่างชัดเจนด้วยแบบอักษรตัวหนาที่อ่านง่าย ช่วยให้ผู้ถือผลประโยชน์สามารถตีความมาตรวัดสำคัญได้อย่างง่ายดาย
2. **การนำเสนอผลงานทางวิชาการ:** เพิ่มความสามารถในการอ่านสำหรับชุดข้อมูลหรือสูตรที่ซับซ้อนโดยการปรับขนาดและรูปแบบของแบบอักษร
3. **สไลด์โชว์การตลาด:** ใช้แบบอักษรที่กำหนดเองเพื่อเน้นคุณสมบัติหรือสถิติผลิตภัณฑ์ที่สำคัญ

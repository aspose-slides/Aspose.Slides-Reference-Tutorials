## คู่มือการใช้งาน

ตอนนี้เรามาดูการใช้งานการจัดรูปแบบบรรทัดในสไลด์กัน

### การเปิดและการเตรียมการนำเสนอ

#### ภาพรวม:
เริ่มต้นด้วยการเปิดงานนำเสนอที่มีอยู่หรือสร้างงานนำเสนอใหม่เพื่อใช้การจัดรูปแบบบรรทัด

```python
import aspose.slides as slides
class PresentationDemo:
    def format_lines(self):
        # เปิดหรือสร้างการนำเสนอ
        with self.presentation as pres:
            ...
```

**คำอธิบาย:**
- การ `slides.Presentation()` ตัวจัดการบริบทช่วยให้แน่ใจว่าทรัพยากรได้รับการจัดการโดยอัตโนมัติ ซึ่งเป็นสิ่งสำคัญสำหรับประสิทธิภาพการทำงานและการจัดการหน่วยความจำ

### การเพิ่มรูปร่างอัตโนมัติลงในสไลด์

#### ภาพรวม:
เพิ่มรูปทรงสี่เหลี่ยมผืนผ้าลงในสไลด์ของคุณซึ่งคุณสามารถใช้การจัดรูปแบบบรรทัดแบบกำหนดเองได้

```python
# รับสไลด์แรกจากการนำเสนอ
class PresentationDemo:
    def format_lines(self):
        with self.presentation as pres:
            slide = pres.slides[0]

            # เพิ่มรูปร่างอัตโนมัติของประเภทสี่เหลี่ยมผืนผ้าลงในสไลด์
            shape = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 50, 150, 150, 75)
```

**คำอธิบาย:**
- `add_auto_shape()` วิธีนี้ใช้เพื่อแทรกรูปร่างใหม่ โดยในที่นี้ เราจะระบุรูปร่างใหม่เป็นรูปสี่เหลี่ยมผืนผ้า และระบุพารามิเตอร์ตำแหน่งและขนาด

### การจัดรูปแบบรูปแบบเส้นของรูปร่าง

#### ภาพรวม:
ใช้รูปแบบเส้นหนา-บางพร้อมความกว้างที่กำหนดเองและรูปแบบเส้นประเพื่อเสริมรูปลักษณ์ของรูปร่างของคุณ

```python
class PresentationDemo:
    def format_lines(self):
        with self.presentation as pres:
            slide = pres.slides[0]
            shape = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 50, 150, 150, 75)

            # ตั้งค่าสีเติมของสี่เหลี่ยมเป็นสีขาว
            shape.fill_format.fill_type = slides.FillType.SOLID
            shape.fill_format.solid_fill_color.color = drawing.Color.white

            # ใช้รูปแบบเส้นหนา-บางที่มีความกว้างและเส้นประที่กำหนด
            shape.line_format.style = slides.LineStyle.THICK_THIN
            shape.line_format.width = 7
            shape.line_format.dash_style = slides.LineDashStyle.DASH

            # ตั้งค่าสีของเส้นขอบสี่เหลี่ยมเป็นสีน้ำเงิน
            shape.line_format.fill_format.fill_type = slides.FillType.SOLID
            shape.line_format.fill_format.solid_fill_color.color = drawing.Color.blue
```

**คำอธิบาย:**
- การ `fill_format` และ `line_format` คุณสมบัติช่วยให้คุณปรับแต่งรูปแบบการเติมและโครงร่างของรูปร่างได้
- การกำหนดค่า `LineStyle`- `width`, และ `dash_style` ช่วยให้คุณสามารถสร้างเอฟเฟ็กต์ภาพที่เฉพาะเจาะจงได้

### การบันทึกการนำเสนอของคุณ

#### ภาพรวม:
บันทึกการนำเสนอที่จัดรูปแบบของคุณลงในไฟล์เพื่อใช้หรือแชร์ในภายหลัง

```python
class PresentationDemo:
    def save_presentation(self, output_path):
        # บันทึกการนำเสนอด้วยรูปทรงที่จัดรูปแบบลงในดิสก์
        self.presentation.save(output_path, slides.export.SaveFormat.PPTX)
```

**คำอธิบาย:**
- `save()` วิธีการนี้คงการเปลี่ยนแปลงไว้ เพื่อให้แน่ใจว่าการปรับเปลี่ยนทั้งหมดจะถูกเก็บไว้ในไฟล์ใหม่

## คู่มือการใช้งาน: ลบส่วนออกจากรูปร่าง

ตอนนี้เรามาเน้นที่การลบส่วนต่างๆ ออกจากรูปร่างกัน คุณลักษณะนี้มีประโยชน์โดยเฉพาะสำหรับการปรับแต่งรูปร่างที่ซับซ้อน เช่น รูปหัวใจ

### ภาพรวมของคุณสมบัติ
คู่มือนี้จะแนะนำคุณเกี่ยวกับวิธีการลบส่วนที่ต้องการ (เช่น ส่วนที่สาม) ออกจากเส้นทางรูปหัวใจในงานนำเสนอของคุณ

#### ขั้นตอนที่ 1: เริ่มต้นการนำเสนอ
```python
# สร้างหรือโหลดการนำเสนอที่มีอยู่
with slides.Presentation() as pres:
    # เพิ่มรูปร่างอัตโนมัติของประเภทหัวใจลงในสไลด์แรก
    shape = pres.slides[0].shapes.add_auto_shape(slides.ShapeType.HEART, 100, 100, 300, 300)
```

#### ขั้นตอนที่ 2: เข้าถึงและแก้ไขเส้นทางเรขาคณิต
```python
# เข้าถึงเส้นทางเรขาคณิตจากรูปหัวใจ
path = shape.get_geometry_paths()[0]

# ลบส่วนเฉพาะ (ดัชนี 2) ออกจากเส้นทาง
del path.s_segments[2]

# อัปเดตรูปร่างด้วยเส้นทางที่แก้ไขแล้ว
shape.set_geometry_path(path)
```

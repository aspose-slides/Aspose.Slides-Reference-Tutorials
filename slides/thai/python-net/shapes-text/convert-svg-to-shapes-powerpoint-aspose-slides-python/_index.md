# วิธีการแปลงรูปภาพ SVG เป็นรูปร่างใน PowerPoint ด้วย Aspose.Slides สำหรับ Python

**สิ่งที่คุณจะได้เรียนรู้:**

- วิธีการติดตั้งและตั้งค่า Aspose.Slides สำหรับ Python

## คู่มือการใช้งาน

ในส่วนนี้จะให้รายละเอียดเกี่ยวกับขั้นตอนการแปลงภาพ SVG เป็นกลุ่มรูปร่างภายในงานนำเสนอ PowerPoint

### การแปลงภาพ SVG เป็นกลุ่มรูปร่าง

นี่คือวิธีที่คุณสามารถแปลงภาพ SVG ที่ฝังไว้ในสไลด์เป็นกลุ่มรูปร่างที่สามารถแก้ไขได้:

#### ภาพรวม

โหลดงานนำเสนอ ค้นหาภาพ SVG ภายในนั้น และแปลงภาพนี้ให้เป็นกลุ่มรูปร่างเพื่อให้มีตัวเลือกการแก้ไขที่ปรับปรุงดีขึ้น

#### ขั้นตอนที่ 1: โหลดงานนำเสนอ

เปิดไฟล์ PowerPoint ของคุณโดยใช้ Aspose.Slides:

```python
with slides.Presentation('YOUR_DOCUMENT_DIRECTORY/save_convert_svg_to_group_of_shapes.pptx') as pres:
    picture_frame = pres.slides[0].shapes[0]
```

#### ขั้นตอนที่ 2: ตรวจสอบรูปภาพ SVG

ตรวจสอบว่ารูปร่างแรกในสไลด์ของคุณมีรูปภาพ SVG หรือไม่:

```python
svg_image = picture_frame.picture_format.picture.image.svg_image
if svg_image is not None:
    # ดำเนินการแปลง
```

การ `picture_format` วัตถุระบุว่าเฟรมมี SVG หรือไม่

#### ขั้นตอนที่ 3: แปลงเป็นกลุ่มรูปร่าง

แปลง SVG ให้เป็นกลุ่มรูปทรงที่ตำแหน่งเดิม:

```python
group_shape = pres.slides[0].shapes.add_group_shape(
    svg_image,
    picture_frame.frame.x,
    picture_frame.frame.y,
    picture_frame.frame.width,
    picture_frame.frame.height
)
```

การ `add_group_shape` วิธีการนี้มีความสำคัญสำหรับการรักษาความสม่ำเสมอของโครงร่าง

#### ขั้นตอนที่ 4: ถอดกรอบเดิมออก

หลังจากการแปลงแล้ว ให้ลบภาพ SVG ต้นฉบับ:

```python
pres.slides[0].shapes.remove(picture_frame)
```

ขั้นตอนนี้จะช่วยให้แน่ใจว่าไม่มีเนื้อหาซ้ำซ้อนภายในสไลด์ของคุณ

#### ขั้นตอนที่ 5: บันทึกการนำเสนอ

สุดท้าย ให้บันทึกงานนำเสนอที่คุณแก้ไขลงในไฟล์ใหม่:

```python
pres.save('YOUR_OUTPUT_DIRECTORY/save_convert_svg_to_group_of_shapes_out.pptx', slides.export.SaveFormat.PPTX)
```

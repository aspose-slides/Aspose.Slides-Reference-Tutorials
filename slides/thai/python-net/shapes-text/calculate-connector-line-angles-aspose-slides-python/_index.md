## คู่มือการใช้งาน
เราจะนำฟีเจอร์นี้ไปใช้ในสองส่วนหลัก: การคำนวณทิศทางของเส้นและนำไปใช้กับตัวเชื่อมต่อ PowerPoint
### คุณสมบัติ 1: การคำนวณทิศทาง
#### ภาพรวม
ฟังก์ชันนี้จะคำนวณมุมโดยอิงตามขนาดและคุณสมบัติการพลิกของเส้น ช่วยให้ควบคุมการวางแนวได้อย่างแม่นยำ
#### การดำเนินการแบบทีละขั้นตอน
**นำเข้าไลบรารีที่จำเป็น**
```python
import math
```
**กำหนดความหมาย `get_direction` การทำงาน**
คำนวณมุมโดยคำนึงถึงความกว้าง (`w`), ความสูง (`h`), พลิกแนวนอน (`flip_h`) และการพลิกแนวตั้ง (`flip_v`-
```python
def get_direction(w, h, flip_h, flip_v):
    # คำนวณพิกัดปลายด้วยการพลิก
    end_line_x = w * (-1 if flip_h else 1)
    end_line_y = h * (-1 if flip_v else 1)

    # พิกัดสำหรับเส้นแนวตั้งอ้างอิง (แกน y)
    end_y_axis_x = 0
    end_y_axis_y = h

    # คำนวณมุมระหว่างแกน y และเส้นที่กำหนด
    angle = math.atan2(end_y_axis_y, end_y_axis_x) - math.atan2(end_line_y, end_line_x)

    if angle < 0:
        angle += 2 * math.pi
    
    # แปลงเรเดียนเป็นองศาเพื่อให้สามารถอ่านได้
    return angle * 180.0 / math.pi
```
**คำอธิบาย**
- **พารามิเตอร์**- `w` และ `h` กำหนดขนาดของเส้น; `flip_h` และ `flip_v` ตรวจสอบว่ามีการใช้การพลิกหรือไม่
- **ค่าส่งคืน**:ฟังก์ชันนี้จะคืนค่ามุมเป็นองศา ซึ่งบ่งชี้ทิศทางของเส้น
#### เคล็ดลับการแก้ไขปัญหา
- ตรวจสอบให้แน่ใจว่าพารามิเตอร์ทั้งหมดเป็นจำนวนเต็มที่ไม่เป็นลบเพื่อหลีกเลี่ยงผลลัพธ์ที่ไม่คาดคิด
- ตรวจสอบว่าการดำเนินการทางคณิตศาสตร์สามารถจัดการกรณีขอบเช่นมิติศูนย์ได้อย่างสวยงาม
### คุณสมบัติ 2: การคำนวณมุมเส้นเชื่อมต่อ
#### ภาพรวม
ฟีเจอร์นี้จะคำนวณมุมทิศทางของเส้นเชื่อมต่อในการนำเสนอ PowerPoint พร้อมดำเนินการกำหนดมุมโดยอัตโนมัติด้วย Aspose.Slides
**นำเข้าห้องสมุด**
```python
import aspose.slides as slides
```
**กำหนดความหมาย `connector_line_angle` การทำงาน**
โหลดและประมวลผลไฟล์ PowerPoint เพื่อคำนวณมุม:
```python
def connector_line_angle():
    # โหลดไฟล์นำเสนอ
    with slides.Presentation("YOUR_DOCUMENT_DIRECTORY/shapes_connector_line_angle.pptx") as pres:
        # เข้าถึงสไลด์แรก
        slide = pres.slides[0]

        for shape in slide.shapes:
            direction = 0.0

            if isinstance(shape, slides.AutoShape):
                # ตรวจสอบว่าเป็นเส้นชนิด AutoShape หรือไม่
                if shape.shape_type == slides.ShapeType.LINE:
                    direction = get_direction(
                        shape.width,
                        shape.height,
                        shape.frame.flip_h,
                        shape.frame.flip_v
                    )
            elif isinstance(shape, slides.Connector):
                # คำนวณทิศทางสำหรับขั้วต่อ
                direction = get_direction(
                    shape.width,
                    shape.height,
                    shape.frame.flip_h,
                    shape.frame.flip_v
                )

            # ส่งออกมุมทิศทางที่คำนวณได้
            print(f"Shape Direction: {direction} degrees")
```
**คำอธิบาย**
- **การเข้าถึงรูปทรง**:ทำซ้ำผ่านรูปร่างแต่ละรูปร่างเพื่อระบุประเภทและคุณสมบัติของรูปร่างนั้น
- **การคำนวณทิศทาง**: นำมาใช้ `get_direction` สำหรับทั้ง AutoShapes (เส้น) และตัวเชื่อมต่อ
- **เอาท์พุต**: พิมพ์มุมทิศทางที่คำนวณได้เป็นองศา

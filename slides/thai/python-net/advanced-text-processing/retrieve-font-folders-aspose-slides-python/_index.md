## คู่มือการใช้งาน

### ดึงข้อมูลโฟลเดอร์ฟอนต์คุณสมบัติ

เราจะสำรวจวิธีการแสดงรายการไดเรกทอรีที่เก็บไฟล์แบบอักษร รวมถึงไดเรกทอรีแบบกำหนดเองที่เพิ่มผ่าน `LoadExternalFonts` วิธี.

#### ขั้นตอนการดำเนินการ

**ขั้นตอนที่ 1: นำเข้า Aspose.Slides**

เริ่มต้นด้วยการนำเข้าโมดูลที่จำเป็น:

```python
import aspose.slides as slides
```

**ขั้นตอนที่ 2: กำหนดฟังก์ชันในการรับโฟลเดอร์ฟอนต์**

สร้างฟังก์ชันโดยใช้ Aspose.Slides API เพื่อดึงไดเรกทอรีฟอนต์

```python
def get_fonts_folder():
    # ดึงรายการโฟลเดอร์ฟอนต์โดยใช้ Aspose.Slides
    font_folders = slides.FontsLoader.get_font_folders()
    
    # ทำซ้ำและพิมพ์เส้นทางโฟลเดอร์แต่ละอัน
    for font_folder in font_folders:
        print(font_folder)
```

**คำอธิบาย**- 
- `get_font_folders()` ดึงไดเร็กทอรีทั้งหมดที่มีแบบอักษรให้ใช้งาน รวมถึงแบบอักษรของระบบและแบบอักษรที่เพิ่มด้วยตนเอง
- ฟังก์ชันนี้จะดำเนินการซ้ำในรายการเพื่อแสดงไดเรกทอรีแต่ละรายการ

### เคล็ดลับการแก้ไขปัญหา

# สร้างแอนิเมชั่นแผนภูมิ PowerPoint ด้วย Python

## การแนะนำ

เปลี่ยนแปลงการนำเสนอ PowerPoint ของคุณด้วยการสร้างภาพเคลื่อนไหวให้กับชุดแผนภูมิด้วย **Aspose.Slides สำหรับ Python**บทช่วยสอนนี้ให้คำแนะนำที่ครอบคลุมเกี่ยวกับการสร้างแผนภูมิแบบไดนามิก เพื่อเพิ่มการมีส่วนร่วมในการนำเสนอของคุณ เมื่ออ่านคู่มือนี้จบ คุณจะเชี่ยวชาญเทคนิคในการสร้างภาพเคลื่อนไหวให้กับองค์ประกอบแผนภูมิอย่างราบรื่นโดยใช้ Python

### คู่มือการใช้งาน
ปฏิบัติตามขั้นตอนเหล่านี้เพื่อสร้างภาพเคลื่อนไหวให้กับองค์ประกอบชุดแผนภูมิอย่างมีประสิทธิภาพ:

#### การโหลดและการเข้าถึงข้อมูลแผนภูมิ
เข้าถึงแผนภูมิที่ต้องการภายในสไลด์ของคุณ:

```python
# โหลดงานนำเสนอ
with slides.Presentation("charts_existing_chart.pptx") as presentation:
    # เข้าถึงสไลด์แรก
    slide = presentation.slides[0]
    
    # รับการรวบรวมรูปทรงและดึงรูปร่างแรก (แผนภูมิ)
    shapes = slide.shapes
    chart = shapes[0]
```

#### แผนภูมิภาพเคลื่อนไหวองค์ประกอบชุด
สร้างภาพเคลื่อนไหวให้กับแต่ละองค์ประกอบภายในซีรีส์:

```python
# เพิ่มเอฟเฟกต์การจางลงให้กับแผนภูมิทั้งหมดในตอนแรก
slide.timeline.main_sequence.add_effect(chart, slides.animation.EffectType.FADE, 
                                        slides.animation.EffectSubtype.NONE, 
                                        slides.animation.EffectTriggerType.AFTER_PREVIOUS)

# สร้างภาพเคลื่อนไหวให้กับแต่ละองค์ประกอบในซีรีส์ 0
for i in range(4):
    slide.timeline.main_sequence.add_effect(chart, 
                                            slides.animation.EffectChartMinorGroupingType.BY_ELEMENT_IN_SERIES,
                                            0, i, 
                                            slides.animation.EffectType.APPEAR,
                                            slides.animation.EffectSubtype.NONE,
                                            slides.animation.EffectTriggerType.AFTER_PREVIOUS)

# ทำซ้ำสำหรับซีรีย์อื่น ๆ
for j in range(1, 3):
    for i in range(4):
        slide.timeline.main_sequence.add_effect(chart, 
                                                slides.animation.EffectChartMinorGroupingType.BY_ELEMENT_IN_SERIES,
                                                j, i, 
                                                slides.animation.EffectType.APPEAR,
                                                slides.animation.EffectSubtype.NONE,
                                                slides.animation.EffectTriggerType.AFTER_PREVIOUS)
```

**คำอธิบาย:**
- **ประเภทเอฟเฟกต์.FADE:** เริ่มเอฟเฟกต์การเฟดอินให้กับแผนภูมิ
- **องค์ประกอบในซีรี่ส์:** กำหนดเป้าหมายองค์ประกอบแต่ละองค์ประกอบภายในแต่ละซีรีส์สำหรับแอนิเมชัน
- **สไลด์.ภาพเคลื่อนไหว.EffectTriggerType.AFTER_PREVIOUS:** รับประกันการเคลื่อนไหวแบบต่อเนื่องขององค์ประกอบ

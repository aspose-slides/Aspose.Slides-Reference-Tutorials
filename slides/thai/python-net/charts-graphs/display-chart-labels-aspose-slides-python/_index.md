# วิธีการแสดงป้ายชื่อแผนภูมิในงานนำเสนอ PowerPoint โดยใช้ Aspose.Slides สำหรับ Python

## การแนะนำ

เพิ่มประสิทธิภาพให้กับงานนำเสนอ PowerPoint ของคุณด้วยการเพิ่มป้ายชื่อแผนภูมิที่ให้ข้อมูลและปรับแต่งได้โดยใช้ Aspose.Slides สำหรับ Python บทช่วยสอนนี้จะแนะนำคุณตลอดกระบวนการผสานป้ายชื่อแผนภูมิเข้ากับสไลด์ของคุณ ทำให้ข้อมูลเข้าถึงได้ง่ายขึ้นและดึงดูดสายตามากขึ้น

**สิ่งที่คุณจะได้เรียนรู้:**
- การตั้งค่า Aspose.Slides สำหรับ Python ในสภาพแวดล้อมของคุณ
- การสร้างงานนำเสนอด้วยแผนภูมิวงกลม
- การกำหนดค่าและปรับแต่งคุณสมบัติฉลากบนชุดแผนภูมิ
- บันทึกการนำเสนอที่ได้รับการปรับปรุง

## คู่มือการใช้งาน
ปฏิบัติตามขั้นตอนเหล่านี้เพื่อแสดงป้ายแผนภูมิในงานนำเสนอ PowerPoint

### ขั้นตอนที่ 1: สร้างการนำเสนอและสไลด์ใหม่
สร้างการนำเสนอใหม่และเพิ่มสไลด์:

```python
def display_chart_labels():
    with slides.Presentation() as presentation:
        # เข้าถึงสไลด์แรก (โดยค่าเริ่มต้นจะสร้างไว้หนึ่งอัน)
        slide = presentation.slides[0]
```

### ขั้นตอนที่ 2: เพิ่มแผนภูมิวงกลมลงในสไลด์
เพิ่มแผนภูมิวงกลมที่ตำแหน่ง `(50, 50)` มีมิติ `500x400`-

```python
        # การเพิ่มแผนภูมิวงกลมลงในสไลด์แรก
        chart = slide.shapes.add_chart(
            slides.charts.ChartType.PIE, 50, 50, 500, 400)
```

### ขั้นตอนที่ 3: กำหนดค่าตัวเลือกการแสดงฉลาก
กำหนดค่าคุณสมบัติฉลากเพื่อการแสดงข้อมูลที่ดีขึ้น:
- **แสดงป้ายค่า**: แสดงค่าตัวเลขบนแต่ละชิ้น
- **การเรียกข้อมูล**:ใช้เส้นคำอธิบายเพื่อเชื่อมต่อฉลากด้วยส่วนต่างๆ

```python
        # กำหนดค่าตัวเลือกการแสดงป้ายชุดแผนภูมิ
        series_labels = chart.chart_data.series[0].labels.default_data_label_format
        series_labels.show_value = True  # แสดงป้ายค่าตามค่าเริ่มต้น
        series_labels.show_label_as_data_callout = True  # ใช้คำอธิบายข้อมูล
```

### ขั้นตอนที่ 4: ปรับแต่งฉลากเฉพาะ
ปิดใช้งานการเรียกข้อมูลสำหรับป้ายกำกับเฉพาะ เช่น ป้ายกำกับที่สาม:

```python
        # การแทนที่การตั้งค่าคำอธิบายข้อมูลสำหรับป้ายกำกับที่เฉพาะเจาะจง
        chart.chart_data.series[0].labels[2].data_label_format.show_label_as_data_callout = False
```

### ขั้นตอนที่ 5: บันทึกการนำเสนอ
บันทึกการนำเสนอของคุณไปยังไดเร็กทอรีเอาท์พุตด้วยชื่อไฟล์ที่ต้องการ:

```python
        # บันทึกการนำเสนอที่ได้รับการปรับปรุง
        presentation.save("YOUR_OUTPUT_DIRECTORY/charts_display_chart_labels_out.pptx")
```

1. **รายงานทางธุรกิจ**:ปรับปรุงรายงานด้วยแผนภูมิวงกลมโดยละเอียดที่แสดงข้อมูลทางการเงิน
2. **การนำเสนอผลงานทางวิชาการ**:ใช้แผนภูมิที่มีป้ายกำกับเพื่อนำเสนอผลการวิจัยได้อย่างมีประสิทธิภาพ
3. **ข้อเสนอการตลาด**:ปรับปรุงการนำเสนอลูกค้าด้วยการนำเสนอข้อมูลที่น่าสนใจและน่ามอง

การบูรณาการกับระบบอื่นๆ เช่น ฐานข้อมูลหรือเครื่องมือวิเคราะห์ จะช่วยปรับปรุงการสร้างแผนภูมิแบบไดนามิกเหล่านี้โดยอิงตามข้อมูลเรียลไทม์ได้

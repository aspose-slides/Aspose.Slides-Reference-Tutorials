## คู่มือการใช้งาน

มาเจาะลึกการตั้งค่าขนาดฟองแบบไดนามิกในแผนภูมิกัน

### การสร้างและแก้ไขแผนภูมิฟองสบู่

#### ภาพรวม

เราจะสร้างงานนำเสนอ PowerPoint เพิ่มแผนภูมิฟอง และปรับเปลี่ยนขนาดฟองตามมิติข้อมูลเฉพาะโดยใช้ Aspose.Slides

#### การดำเนินการแบบทีละขั้นตอน

**1. เริ่มต้นการนำเสนอ**

เริ่มต้นด้วยการสร้างอินสแตนซ์ของ `Presentation` ภายในตัวจัดการบริบท:

```python
import aspose.slides as slides

def charts_bubble_size_representation():
    with slides.Presentation() as pres:
        # โค้ดยังคงดำเนินต่อไป...
```

**2. เพิ่มแผนภูมิฟองสบู่**

เพิ่มแผนภูมิฟองสบู่ในตำแหน่ง `(50, 50)` มีมิติ `600x400` ในสไลด์แรก

```python
chart = pres.slides[0].shapes.add_chart(
    slides.charts.ChartType.BUBBLE,
    50, 50, 600, 400, True
)
```

**3. ตั้งค่าการแสดงขนาดฟองอากาศ**

กำหนดค่าการแสดงขนาดฟองอากาศให้ `WIDTH` สำหรับกลุ่มซีรีย์แรก:

```python
chart.chart_data.series_groups[0].bubble_size_representation = \\
    slides.charts.BubbleSizeRepresentationType.WIDTH
```

**4. บันทึกการนำเสนอ**

สุดท้าย ให้บันทึกการนำเสนอของคุณไปยังไดเร็กทอรีที่ระบุ:

```python
pres.save(
    "YOUR_OUTPUT_DIRECTORY/charts_bubble_size_representation_out.pptx"
)
```

ต่อไปนี้คือสถานการณ์จริงบางสถานการณ์ที่การปรับขนาดฟองอากาศอาจเป็นประโยชน์ได้:

1. **การวิเคราะห์ทางธุรกิจ**:แสดงข้อมูลการขายตามขนาดผลิตภัณฑ์หรือรายได้ในรายงานรายไตรมาส
2. **การนำเสนอด้านการศึกษา**:แสดงภาพมาตรวัดผลการปฏิบัติงานของนักศึกษาในแต่ละวิชา
3. **การจัดการโครงการ**: แสดงอัตราความสำเร็จของงานในไทม์ไลน์ของโครงการ
4. **การวิจัยการตลาด**:เปรียบเทียบส่วนแบ่งการตลาดของบริษัทที่ใช้ขนาดฟองอากาศเพื่อสร้างผลกระทบทางภาพ

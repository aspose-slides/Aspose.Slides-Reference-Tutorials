# สร้าง PowerPoint ด้วยแผนภูมิ Excel ภายนอกโดยใช้ Aspose.Slides สำหรับ Python

## วิธีการผสานรวมแผนภูมิ Excel เข้ากับการนำเสนอ PowerPoint โดยใช้ Aspose.Slides สำหรับ Python

### การแนะนำ
การสร้างงานนำเสนอแบบไดนามิกเป็นสิ่งสำคัญสำหรับการประชุมทางธุรกิจ การบรรยายเชิงวิชาการ และโปรเจ็กต์ส่วนตัว ความท้าทายทั่วไปที่นักพัฒนามักเผชิญคือการรวมแหล่งข้อมูลภายนอก เช่น ไฟล์ Excel ลงในงานนำเสนออย่างราบรื่น บทช่วยสอนนี้จะกล่าวถึงปัญหานี้โดยสาธิตวิธีใช้ **Aspose.Slides สำหรับ Python** เพื่อสร้างการนำเสนอ PowerPoint ด้วยแผนภูมิจากสมุดงานภายนอก

เมื่ออ่านคู่มือนี้จบ คุณจะเรียนรู้สิ่งต่อไปนี้:
- วิธีการคัดลอกไฟล์สมุดงานภายนอกโดยใช้ Python
- วิธีการสร้างและกำหนดค่าการนำเสนอใน Aspose.Slides
- วิธีตั้งค่าแผนภูมิที่ดึงข้อมูลโดยตรงจากสมุดงาน Excel

มาเจาะลึกถึงข้อกำหนดเบื้องต้นกันก่อน!

## คู่มือการใช้งาน

### คุณสมบัติ: คัดลอกสมุดงานภายนอก
**ภาพรวม:**
ก่อนอื่นเราจะสาธิตวิธีการคัดลอกไฟล์เวิร์กบุ๊กภายนอกจากไดเร็กทอรีแหล่งที่มาไปยังไดเร็กทอรีเอาท์พุตเป้าหมายโดยใช้ Python `shutil` โมดูล ซึ่งจะทำให้มั่นใจได้ว่าการนำเสนอของคุณสามารถเข้าถึงข้อมูลที่จำเป็นได้

#### ขั้นตอนที่ 1: นำเข้าไลบรารีที่จำเป็น
```python
import shutil
```

#### ขั้นตอนที่ 2: กำหนดเส้นทางไฟล์และคัดลอกเวิร์กบุ๊ก
```python
external_workbook_file_name = "charts_external_workbook.xlsx"
source_path = "YOUR_DOCUMENT_DIRECTORY/" + external_workbook_file_name
output_path = "YOUR_OUTPUT_DIRECTORY/" + external_workbook_file_name
shutil.copyfile(source_path, output_path)
```
ตัวอย่างนี้คัดลอก `charts_external_workbook.xlsx` จากไดเร็กทอรีเอกสารของคุณไปยังไดเร็กทอรีผลลัพธ์

### คุณสมบัติ: สร้างการนำเสนอและตั้งค่าเวิร์กบุ๊กภายนอกสำหรับข้อมูลแผนภูมิ
**ภาพรวม:**
ต่อไปเราจะสร้างงานนำเสนอและกำหนดเวิร์กบุ๊กภายนอกเป็นแหล่งข้อมูลสำหรับแผนภูมิโดยใช้ Aspose.Slides วิธีนี้ช่วยให้คุณแสดงข้อมูล Excel ในสไลด์ PowerPoint ได้โดยตรง

#### ขั้นตอนที่ 1: นำเข้า Aspose.Slides
```python
import aspose.slides as slides
```

#### ขั้นตอนที่ 2: กำหนดฟังก์ชันการสร้างงานนำเสนอ
```python
def create_presentation_with_external_chart():
    with slides.Presentation() as pres:
        chart = pres.slides[0].shapes.add_chart(
            slides.charts.ChartType.PIE, 50, 50, 400, 600, False)
        
        chart_data = chart.chart_data
        chart_data.set_external_workbook("YOUR_OUTPUT_DIRECTORY/charts_external_workbook.xlsx")
        
        series = chart_data.series.add(chart_data.chart_data_workbook.get_cell(0, "B1"), slides.charts.ChartType.PIE)
        
        # เพิ่มจุดข้อมูลสำหรับชุดวงกลมจากเซลล์เวิร์กบุ๊กภายนอก
        series.data_points.add_data_point_for_pie_series(chart_data.chart_data_workbook.get_cell(0, "B2"))
        series.data_points.add_data_point_for_pie_series(chart_data.chart_data_workbook.get_cell(0, "B3"))
        series.data_points.add_data_point_for_pie_series(chart_data.chart_data_workbook.get_cell(0, "B4"))

        chart_data.categories.add(chart_data.chart_data_workbook.get_cell(0, "A2"))
        chart_data.categories.add(chart_data.chart_data_workbook.get_cell(0, "A3"))
        chart_data.categories.add(chart_data.chart_data_workbook.get_cell(0, "A4"))
        
        pres.save("YOUR_OUTPUT_DIRECTORY/charts_set_external_workbook_out.pptx", slides.export.SaveFormat.PPTX)
```

#### คำอธิบาย:
- **สร้างงานนำเสนอ**:เราเริ่มต้นด้วยการเปิดวัตถุการนำเสนอใหม่
- **เพิ่มแผนภูมิ**:แผนภูมิวงกลมจะถูกเพิ่มลงในสไลด์แรกตามพิกัดและมิติที่ระบุ
- **ตั้งค่าสมุดงานภายนอก**เส้นทางของเวิร์กบุ๊กจะถูกกำหนดเพื่อให้ Aspose.Slides รู้ว่าจะดึงข้อมูลจากที่ใด
- **เพิ่มซีรีส์และจุดข้อมูล**:เราตั้งค่าชุดข้อมูลด้วยเซลล์เฉพาะจากเวิร์กบุ๊กภายนอก ทำให้สามารถอัปเดตแบบไดนามิกได้

#### เคล็ดลับการแก้ไขปัญหา:
- ตรวจสอบให้แน่ใจว่าเส้นทางไฟล์ถูกต้อง มิฉะนั้นคุณจะพบข้อผิดพลาดไม่พบไฟล์
- ตรวจสอบว่าการอ้างอิงเซลล์ในไฟล์ Excel ของคุณตรงกับการอ้างอิงในโค้ดของคุณเพื่อหลีกเลี่ยงปัญหาการจัดตำแหน่งข้อมูลที่ไม่ถูกต้อง

## การประยุกต์ใช้งานจริง
ต่อไปนี้เป็นการใช้งานจริงบางส่วนของการบูรณาการ Aspose.Slides เข้ากับเวิร์กบุ๊กภายนอก:
1. **รายงานทางการเงิน**อัปเดตแผนภูมิโดยอัตโนมัติในการนำเสนอรายไตรมาสตามสเปรดชีตทางการเงินล่าสุด
2. **การนำเสนอที่ขับเคลื่อนด้วยข้อมูล**:บูรณาการการวิเคราะห์แบบเรียลไทม์เข้ากับข้อมูลขายหรือการอัปเดตโครงการได้อย่างราบรื่น
3. **สื่อการเรียนรู้**:ครูสามารถใช้ข้อมูลผลการปฏิบัติงานของนักเรียนที่อัปเดตเพื่อสร้างรายงานส่วนบุคคลได้
4. **ระบบการรายงานอัตโนมัติ**:นำระบบอัตโนมัติที่สร้างและแจกจ่ายงานนำเสนอโดยอิงจากข้อมูลรายการใหม่

# วิธีการสร้างแผนภูมิวงกลมใน PowerPoint โดยใช้ Aspose.Slides สำหรับ Python

การสร้างแผนภูมิที่ดึงดูดสายตา เช่น แผนภูมิวงกลมของวงกลม จะช่วยปรับปรุงการนำเสนอ PowerPoint ของคุณได้อย่างมาก โดยทำให้ข้อมูลที่ซับซ้อนเข้าใจง่ายขึ้น บทช่วยสอนนี้จะแนะนำคุณเกี่ยวกับการสร้างแผนภูมิวงกลมของวงกลมโดยใช้ Aspose.Slides สำหรับ Python

### คู่มือการใช้งาน

#### คุณสมบัติ 1: สร้างการนำเสนอด้วยแผนภูมิ

ฟีเจอร์นี้จะสาธิตวิธีการสร้างงานนำเสนอ PowerPoint และเพิ่มแผนภูมิวงกลมลงในสไลด์แรก

##### การเพิ่มแผนภูมิ

เริ่มต้นด้วยการสร้างงานนำเสนอใหม่และเพิ่มแผนภูมิวงกลมของวงกลมที่ตำแหน่ง (50, 50) บนสไลด์แรก:

```python
import aspose.slides as slides

with slides.Presentation() as presentation:
    # เพิ่มแผนภูมิ 'วงกลมของวงกลม' พร้อมมิติที่ระบุ
    chart = presentation.slides[0].shapes.add_chart(
        slides.charts.ChartType.PIE_OF_PIE, 50, 50, 500, 400)
```

##### การกำหนดค่าป้ายข้อมูล

เพื่อปรับปรุงการอ่าน ให้กำหนดค่าป้ายข้อมูลเพื่อแสดงค่า:

```python
# เปิดใช้งานการแสดงค่าในป้ายข้อมูลเพื่อความชัดเจนยิ่งขึ้น
chart.chart_data.series[0].labels.default_data_label_format.show_value = True
```

##### การตั้งค่าตัวเลือกพายของพาย

กำหนดค่าคุณสมบัติเฉพาะสำหรับแผนภูมิวงกลมของวงกลม เช่น ขนาดของวงกลมที่สองและตำแหน่งแยก:

```python
# ตั้งค่าขนาดพายที่สองและคุณสมบัติการแยก
chart.chart_data.series[0].parent_series_group.second_pie_size = 149
chart.chart_data.series[0].parent_series_group.pie_split_by = slides.charts.PieSplitType.BY_PERCENTAGE
chart.chart_data.series[0].parent_series_group.pie_split_position = 53
```

##### การบันทึกการนำเสนอ

สุดท้ายให้บันทึกการนำเสนอของคุณไปยังไดเร็กทอรีที่ต้องการ:

```python
# บันทึกการนำเสนอด้วยแผนภูมิ
presentation.save("YOUR_OUTPUT_DIRECTORY/charts_second_plot_options_out.pptx", slides.export.SaveFormat.PPTX)
```

2. **โครงการวิชาการ**:นำเสนอผลการสำรวจที่แสดงให้เห็นประเด็นหลักร่วมกับการค้นพบที่สำคัญน้อยกว่า
3. **การวิเคราะห์ทางการเงิน**:เปรียบเทียบค่าใช้จ่ายหลักกับค่าใช้จ่ายรองในรายงานงบประมาณ

# วิธีการสร้างแผนภูมิโดนัทใน PowerPoint พร้อมขนาดรูที่กำหนดเองโดยใช้ Aspose.Slides สำหรับ Python

## การแนะนำ
การสร้างแผนภูมิที่ดึงดูดสายตาใน PowerPoint จะทำให้ข้อมูลของคุณน่าสนใจและเข้าใจง่ายขึ้น ปัญหาทั่วไปคือการขาดตัวเลือกการปรับแต่งเมื่อสร้างแผนภูมิเหล่านี้ด้วยโปรแกรม บทช่วยสอนนี้จะแก้ปัญหานี้โดยสาธิตวิธีการสร้างแผนภูมิโดนัทที่มีขนาดรูที่กำหนดเองโดยใช้ Aspose.Slides สำหรับ Python

**คำสำคัญ:** Aspose.Slides Python, แผนภูมิโดนัท, ขนาดรูที่กำหนดเอง

### สิ่งที่คุณจะได้เรียนรู้:
- การตั้งค่าและการใช้งาน Aspose.Slides สำหรับ Python
- การสร้างแผนภูมิโดนัทใน PowerPoint
- การปรับแต่งขนาดรูของแผนภูมิโดนัทของคุณ
- แนวทางปฏิบัติที่ดีที่สุดสำหรับการบันทึกและส่งออกงานนำเสนอ

- ความรู้พื้นฐานเกี่ยวกับแนวคิดการเขียนโปรแกรม Python
- การ `aspose.slides` ห้องสมุด (มีคำแนะนำในการติดตั้งอยู่ด้านล่าง)

ในการเริ่มต้น ให้ติดตั้ง Aspose.Slides สำหรับ Python โดยใช้ pip:

```bash
pip install aspose.slides
```

## คู่มือการใช้งาน
ในส่วนนี้จะแบ่งขั้นตอนที่จำเป็นในการสร้างและปรับแต่งแผนภูมิโดนัทใน PowerPoint โดยใช้ Aspose.Slides

### ขั้นตอนที่ 1: การเข้าถึงและแก้ไขสไลด์
ในการเริ่มต้น ให้เข้าถึงสไลด์แรกจากงานนำเสนอของคุณ นี่คือที่ที่คุณจะเพิ่มแผนภูมิโดนัทแบบกำหนดเองของคุณ

```python
# เข้าถึงสไลด์แรก
class SlideModifier:
    def modify_slide(self, presentation):
        first_slide = presentation.slides[0]
```

### ขั้นตอนที่ 2: การเพิ่มแผนภูมิโดนัท
คุณสามารถเพิ่มแผนภูมิโดนัทลงในสไลด์ใดๆ ได้โดยระบุตำแหน่งและขนาดของแผนภูมิ ในที่นี้ เราจะวางไว้ที่พิกัด (50, 50) โดยมีขนาด 400x400

```python
class ChartAdder:
    def add_doughnut_chart(self, first_slide):
        # เพิ่มแผนภูมิโดนัท
        chart = first_slide.shapes.add_chart(
            slides.charts.ChartType.DOUGHNUT,
            50, 50, 400, 400
        )
```

### ขั้นตอนที่ 3: ปรับแต่งขนาดรู
การปรับขนาดรูของแผนภูมิโดนัทเป็นเรื่องง่ายมาก ตั้งค่าเป็น 90% เพื่อให้ได้ผลลัพธ์ที่ชัดเจน

```python
class ChartCustomizer:
    def customize_hole_size(self, chart):
        # ตั้งค่าขนาดรูที่กำหนดเอง
        chart.chart_data.series_groups[0].doughnut_hole_size = 90
```

### ขั้นตอนที่ 4: บันทึกการนำเสนอของคุณ
สุดท้ายให้บันทึกการนำเสนอของคุณไปยังตำแหน่งที่ต้องการด้วยชื่อไฟล์ที่เลือก

```python
class PresentationSaver:
    def save_presentation(self, presentation):
        # บันทึกการนำเสนอ
        presentation.save(
            "charts_doughnut_chart_hole_out.pptx",
            slides.export.SaveFormat.PPTX
        )
```

- **สื่อการตลาด:** การจัดแสดงรายละเอียดของผลิตภัณฑ์หรือข้อมูลประชากรของลูกค้า

## คู่มือการใช้งาน

### หน่วยแสดงผลบนแกนแผนภูมิ
#### ภาพรวม
คุณลักษณะนี้ช่วยให้คุณสามารถกำหนดหน่วยที่กำหนดเองบนแกนแผนภูมิได้ เช่น ล้านหรือพันล้าน ซึ่งจะช่วยให้ข้อมูลอ่านได้ง่ายขึ้นในงานนำเสนอ

#### การดำเนินการแบบทีละขั้นตอน
1. **การเริ่มต้นการนำเสนอ**
   เริ่มต้นด้วยการสร้างอินสแตนซ์การนำเสนอใหม่ซึ่งคุณจะเพิ่มแผนภูมิ:

   ```python
   with slides.Presentation() as pres:
       # โค้ดของคุณสำหรับการจัดการสไลด์และแผนภูมิอยู่ที่นี่
   ```

2. **เพิ่มแผนภูมิคอลัมน์แบบคลัสเตอร์**
   เพิ่มแผนภูมิคอลัมน์แบบคลัสเตอร์ตามพิกัดที่ระบุในสไลด์แรก:

   ```python
   chart = pres.slides[0].shapes.add_chart(
       slides.charts.ChartType.CLUSTERED_COLUMN, 50, 50, 450, 300
   )
   ```

3. **ตั้งค่าหน่วยแสดงผลแกนแนวตั้ง**
   กำหนดค่าแกนตั้งให้แสดงค่าเป็นล้าน:

   ```python
   chart.axes.vertical_axis.display_unit = slides.charts.DisplayUnitType.MILLIONS
   ```

4. **บันทึกการนำเสนอ**
   บันทึกการนำเสนอของคุณด้วยแผนภูมิที่กำหนดค่า:

   ```python
   pres.save("YOUR_OUTPUT_DIRECTORY/charts_showing_display_unit_label_out.pptx", slides.export.SaveFormat.PPTX)
   ```

#### พารามิเตอร์และวิธีการ
- `add_chart`เพิ่มวัตถุแผนภูมิใหม่ลงในสไลด์
- `display_unit`:ตั้งค่าหน่วยแสดงผลสำหรับค่าตัวเลขบนแกนตั้ง

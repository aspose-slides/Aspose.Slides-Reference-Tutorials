## คู่มือการใช้งาน

### ภาพรวมของการปรับแต่งตำนานแผนภูมิ
การปรับแต่งคำอธิบายแผนภูมิเกี่ยวข้องกับการตั้งค่าคุณสมบัติ เช่น ตำแหน่ง ขนาด และการจัดตำแหน่งที่สัมพันธ์กับมิติของแผนภูมิ หัวข้อนี้จะแนะนำคุณเกี่ยวกับการเพิ่มแผนภูมิคอลัมน์แบบคลัสเตอร์และการแก้ไขคำอธิบายแผนภูมิ

#### ขั้นตอนที่ 1: สร้างงานนำเสนอใหม่
```python
import aspose.slides as slides

def charts_set_legend_custom_options():
    with slides.Presentation() as presentation:
        slide = presentation.slides[0]
```
โค้ดนี้จะเริ่มต้นการนำเสนอใหม่และเข้าถึงสไลด์แรกเพื่อทำการปรับเปลี่ยน

#### ขั้นตอนที่ 2: เพิ่มแผนภูมิคอลัมน์แบบคลัสเตอร์
```python
chart = slide.shapes.add_chart(
    slides.charts.ChartType.CLUSTERED_COLUMN,
    50, 50, 500, 500
)
```
เพิ่มแผนภูมิคอลัมน์แบบคลัสเตอร์ลงในสไลด์ พารามิเตอร์จะระบุประเภทแผนภูมิ ตำแหน่ง และมิติบนสไลด์

#### ขั้นตอนที่ 3: ตั้งค่าคุณสมบัติตำนาน
การปรับคุณสมบัติของตำนานเกี่ยวข้องกับการคำนวณตำแหน่งเป็นเศษส่วนของความกว้างและความสูงของแผนภูมิ:
```python
chart.legend.x = 50 / chart.width
chart.legend.y = 50 / chart.height
chart.legend.width = 100 / chart.width
chart.legend.height = 100 / chart.height
```
ที่นี่- `x`- `y`, `width`, และ `height` จะถูกปรับเป็นเศษส่วนเพื่อรักษาความสามารถในการตอบสนอง

#### ขั้นตอนที่ 4: บันทึกการนำเสนอ
```python
presentation.save("YOUR_OUTPUT_DIRECTORY/charts_set_legend_custom_options_out.pptx")
```
แทนที่ `"YOUR_OUTPUT_DIRECTORY"` ตามตำแหน่งที่คุณต้องการบันทึก ขั้นตอนนี้จะบันทึกการนำเสนอที่คุณปรับแต่งเอง

## คู่มือการใช้งาน: การตั้งค่าโหมดเค้าโครงแผนภูมิ

มาดูวิธีการตั้งค่าโหมดเค้าโครงของแผนภูมิภายในงานนำเสนอ PowerPoint กัน

### สร้างและเข้าถึงสไลด์

เริ่มต้นด้วยการสร้างการนำเสนอ PowerPoint ใหม่และเข้าถึงสไลด์แรก:

```python
with slides.Presentation() as presentation:
    slide = presentation.slides[0]
```

นี่เป็นการตั้งค่าสภาพแวดล้อมของคุณสำหรับการเพิ่มแผนภูมิ

### เพิ่มแผนภูมิคอลัมน์แบบคลัสเตอร์

เพิ่มแผนภูมิคอลัมน์แบบคลัสเตอร์ไปยังตำแหน่งที่ระบุบนสไลด์:

```python
chart = slide.shapes.add_chart(
    slides.charts.ChartType.CLUSTERED_COLUMN, 20, 100, 600, 400
)
```

พารามิเตอร์:
- `ChartType.CLUSTERED_COLUMN`: กำหนดประเภทของแผนภูมิ
- `(20, 100)`:พิกัด x และ y คือตำแหน่งที่จะวางแผนภูมิบนสไลด์
- `(600, 400)`: ความกว้างและความสูงของแผนภูมิเป็นจุด

### ปรับแต่งคุณสมบัติเค้าโครง

ตอนนี้ปรับคุณสมบัติเค้าโครงของพื้นที่พล็อตเพื่อกำหนดตำแหน่งและขนาดของมัน:

```python
chart.plot_area.as_i_layoutable.x = 0.2
chart.plot_area.as_i_layoutable.y = 0.2
chart.plot_area.as_i_layoutable.width = 0.7
chart.plot_area.as_i_layoutable.height = 0.7
```

ค่าเหล่านี้เป็นหน่วยสัมพันธ์ ซึ่งรับรองว่าแผนภูมิจะปรับเปลี่ยนไปตามขนาดสไลด์ที่แตกต่างกันอย่างไดนามิก

### ระบุประเภทเป้าหมายเค้าโครง

ตั้งค่าประเภทเป้าหมายเค้าโครงเพื่อควบคุมอย่างแม่นยำว่าพื้นที่พล็อตจะทำงานอย่างไร:

```python
chart.plot_area.layout_target_type = slides.charts.LayoutTargetType.INNER
```

การกำหนดค่านี้จะช่วยให้แน่ใจว่าพื้นที่แปลงจะอยู่ตรงกลางคอนเทนเนอร์ และยังคงดูสะอาดตา

### บันทึกการนำเสนอของคุณ

สุดท้าย ให้บันทึกการนำเสนอของคุณไปยังไดเร็กทอรีเอาต์พุตที่ระบุ:

```python
output_directory = 'YOUR_OUTPUT_DIRECTORY/'
presentation.save(output_directory + 'charts_set_layout_mode_out.pptx', slides.export.SaveFormat.PPTX)
```

ต่อไปนี้เป็นการใช้งานจริงบางส่วนของการตั้งค่าโหมดเค้าโครงแผนภูมิในการนำเสนอ:

1. **รายงานทางธุรกิจ**:เพิ่มความสามารถในการอ่านและความเป็นมืออาชีพของรายงานทางการเงินโดยการทำให้แน่ใจว่าแผนภูมิต่างๆ อยู่ในตำแหน่งที่เหมาะสม
2. **เนื้อหาการศึกษา**:สร้างสื่อการเรียนรู้ที่น่าสนใจและดึงดูดสายตาด้วยแผนภูมิที่ดึงความสนใจไปที่จุดข้อมูลสำคัญ
3. **การนำเสนอการตลาด**:ใช้เค้าโครงแผนภูมิแบบกำหนดเองเพื่อเน้นย้ำตัวชี้วัดการตลาดอย่างมีประสิทธิผลในระหว่างการนำเสนอต่อลูกค้า
4. **การจัดการโครงการ**:แสดงกำหนดเวลาและความคืบหน้าของโครงการอย่างชัดเจนโดยใช้แผนภูมิแกนต์ที่จัดอย่างเป็นระบบ

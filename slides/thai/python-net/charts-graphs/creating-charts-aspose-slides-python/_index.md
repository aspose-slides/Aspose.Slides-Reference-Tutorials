# การสร้างแผนภูมิใน Python ด้วย Aspose.Slides: คู่มือที่ครอบคลุม

## การแนะนำ
การสร้างแผนภูมิที่ดึงดูดสายตาในงานนำเสนอของคุณสามารถทำให้ข้อมูลเข้าใจง่ายขึ้น ช่วยให้คุณสามารถถ่ายทอดข้อมูลที่ซับซ้อนได้อย่างง่ายดาย บทช่วยสอนนี้จะแนะนำคุณเกี่ยวกับการสร้างและกำหนดค่าแผนภูมิโดยใช้ Aspose.Slides สำหรับ Python ซึ่งเป็นไลบรารีที่มีประสิทธิภาพที่เปลี่ยนวิธีการออกแบบงานนำเสนอของคุณโดยนำเสนอคุณลักษณะอันทรงพลังสำหรับการจัดการแผนภูมิ

**สิ่งที่คุณจะได้เรียนรู้:**
- วิธีการสร้างแผนภูมิคอลัมน์แบบซ้อนกันในงานนำเสนอ
- การเพิ่มและการจัดรูปแบบชุดข้อมูลด้วยป้ายกำกับแบบกำหนดเอง
- บันทึกการนำเสนอที่คุณกำหนดค่าไว้

เมื่อสิ้นสุดบทช่วยสอนนี้ คุณจะได้รับประสบการณ์จริงในการใช้ Aspose.Slides Python เพื่อเพิ่มประสิทธิภาพการนำเสนอของคุณ มาเริ่มตั้งค่าสภาพแวดล้อมของคุณก่อนที่เราจะเริ่มสร้างแผนภูมิที่สวยงามกัน!

## ข้อกำหนดเบื้องต้น
ก่อนที่เราจะเริ่มต้น โปรดตรวจสอบให้แน่ใจว่าคุณมีคุณสมบัติตามข้อกำหนดเบื้องต้นต่อไปนี้:

1. **สภาพแวดล้อม Python:** คุณควรติดตั้ง Python ไว้ในระบบของคุณ (แนะนำเวอร์ชัน 3.x)
2. **Aspose.Slides สำหรับ Python:** สามารถติดตั้งได้ผ่าน pip
3. **การได้มาซึ่งใบอนุญาต:** แม้ว่าจะมีรุ่นทดลองใช้งานฟรี แต่ควรพิจารณาซื้อใบอนุญาตชั่วคราวหรือเต็มรูปแบบเพื่อปลดล็อกฟีเจอร์ทั้งหมด

## การตั้งค่า Aspose.Slides สำหรับ Python
หากต้องการเริ่มใช้ Aspose.Slides ในโปรเจ็กต์ของคุณ คุณต้องติดตั้งไลบรารีและทำความเข้าใจวิธีตั้งค่าสภาพแวดล้อมของคุณ:

**การติดตั้ง:**
```bash
pip install aspose.slides
```

หลังจากติดตั้งแล้ว คุณสามารถเริ่มต้นใช้งาน Aspose.Slides ได้โดยนำเข้าไปในสคริปต์ของคุณ หากต้องการใช้คุณสมบัติต่างๆ ของ Aspose.Slides อย่างเต็มที่ ให้ซื้อใบอนุญาต มีรุ่นทดลองใช้งานฟรีให้ใช้งาน หรือหากต้องการใช้งานแบบขยายเวลา ให้พิจารณาซื้อหรือสมัครใบอนุญาตชั่วคราว

## คู่มือการใช้งาน

### คุณลักษณะที่ 1: สร้างและกำหนดค่าการนำเสนอด้วยแผนภูมิ
**ภาพรวม:** หัวข้อนี้จะแนะนำคุณเกี่ยวกับการตั้งค่าสไลด์การนำเสนอและการเพิ่มแผนภูมิโดยใช้ Aspose.Slides ใน Python

#### ขั้นตอนที่ 1: เริ่มต้นการนำเสนอ
เริ่มต้นด้วยการสร้างวัตถุการนำเสนอใหม่ ใช้ `with` คำชี้แจงสำหรับการจัดการทรัพยากรอัตโนมัติ:
```python
import aspose.slides as slides

with slides.Presentation() as presentation:
    # เข้าถึงสไลด์แรกในการนำเสนอ
    slide = presentation.slides[0]
```

#### ขั้นตอนที่ 2: เพิ่มแผนภูมิลงในสไลด์
ที่นี่ เราจะเพิ่มแผนภูมิคอลัมน์แบบเรียงซ้อนที่ตำแหน่งที่ระบุ พร้อมด้วยมิติที่กำหนดไว้:
```python
# เพิ่มแผนภูมิคอลัมน์แบบเรียงซ้อนลงในสไลด์
chart = slide.shapes.add_chart(slides.charts.ChartType.PERCENTS_STACKED_COLUMN, 20, 20, 500, 400)
```

#### ขั้นตอนที่ 3: กำหนดค่าแกนแผนภูมิ
ตั้งค่ารูปแบบตัวเลขแกนแนวตั้งเพื่อให้แสดงข้อมูลได้ดีขึ้น:
```python
# กำหนดค่ารูปแบบตัวเลขแกนแนวตั้ง
chart.axes.vertical_axis.is_number_format_linked_to_source = False
chart.axes.vertical_axis.number_format = "0.00%"
```

### คุณสมบัติ 2: เพิ่มและจัดรูปแบบชุดข้อมูลลงในแผนภูมิ
**ภาพรวม:** หัวข้อนี้มุ่งเน้นที่การเพิ่มชุดข้อมูล การเติมค่า และการปรับแต่งลักษณะที่ปรากฏ

#### ขั้นตอนที่ 1: กำหนดสมุดงานข้อมูล
เริ่มต้นสมุดงานข้อมูลแผนภูมิของคุณ:
```python
default_worksheet_index = 0
workbook = chart.chart_data.chart_data_workbook
```

#### ขั้นตอนที่ 2: เพิ่มและเติมชุดข้อมูล
เพิ่มชุดข้อมูลใหม่ที่ชื่อ "สีแดง" ลงในแผนภูมิของคุณ จากนั้นเติมจุดข้อมูลลงไป:
```python
# เพิ่มซีรีส์ใหม่และเติมด้วยจุดข้อมูล
series = chart.chart_data.series.add(workbook.get_cell(default_worksheet_index, 0, 1, "Reds"), chart.type)

for i in range(1, 5):
    series.data_points.add_data_point_for_bar_series(
        workbook.get_cell(default_worksheet_index, i, 1, [0.30, 0.50, 0.80, 0.65][i-1])
    )
```

#### ขั้นตอนที่ 3: จัดรูปแบบรูปลักษณ์ของซีรีส์
ปรับแต่งสีเติมและรูปแบบป้ายข้อมูล:
```python
# ตั้งค่าซีรีย์เติมเป็นสีแดง
series.format.fill.fill_type = slides.FillType.SOLID
series.format.fill.solid_fill_color.color = drawing.Color.red

# กำหนดค่าป้ายข้อมูลสำหรับการแสดงเปอร์เซ็นต์
series.labels.default_data_label_format.show_value = True
series.labels.default_data_label_format.number_format = "0.0%"
```

### คุณสมบัติที่ 3: เพิ่มและจัดรูปแบบชุดข้อมูลที่สองลงในแผนภูมิ
**ภาพรวม:** หัวข้อนี้ขยายความเกี่ยวกับการเพิ่มชุดข้อมูลชุดที่สองซึ่งมีรูปแบบของตัวเอง

#### ขั้นตอนที่ 1: เพิ่มซีรีย์ที่สอง
เพิ่มซีรีย์อีกเรื่องชื่อ "บลูส์":
```python
# เพิ่มซีรีย์ที่ 2 ชื่อ "บลูส์"
series2 = chart.chart_data.series.add(workbook.get_cell(default_worksheet_index, 0, 2, "Blues"), chart.type)
```

#### ขั้นตอนที่ 2: เติมข้อมูลและจัดรูปแบบซีรีส์
เติมด้วยจุดข้อมูลและใช้การจัดรูปแบบ:
```python
# เติมชุดที่ 2
for i in range(1, 5):
    series2.data_points.add_data_point_for_bar_series(
        workbook.get_cell(default_worksheet_index, i, 2, [0.70, 0.50, 0.20, 0.35][i-1])
    )

# ตั้งค่าการเติมเป็นสีน้ำเงินและกำหนดค่าป้ายกำกับ
series2.format.fill.fill_type = slides.FillType.SOLID
series2.format.fill.solid_fill_color.color = drawing.Color.blue

series2.labels.default_data_label_format.show_value = True
```

### คุณสมบัติที่ 4: บันทึกการนำเสนอลงในดิสก์
**ภาพรวม:** เมื่อกำหนดค่าแผนภูมิของคุณแล้ว ให้บันทึกการนำเสนอ

#### ขั้นตอนที่ 1: บันทึกงานของคุณ
ใช้ `save` วิธีการจัดเก็บไฟล์ของคุณ:
```python
# บันทึกการนำเสนอลงในดิสก์
directory = "YOUR_OUTPUT_DIRECTORY"
presentation.save(f"{directory}/charts_set_data_labels_percentage_sign_out.pptx", slides.export.SaveFormat.PPTX)
```

## การประยุกต์ใช้งานจริง
การใช้ Aspose.Slides สำหรับ Python ช่วยให้คุณสามารถปรับปรุงการนำเสนอในโดเมนต่างๆ ได้:
1. **รายงานทางธุรกิจ:** สร้างรายงานรายไตรมาสโดยละเอียดด้วยแผนภูมิแบบไดนามิก
2. **เนื้อหาการศึกษา:** ออกแบบสื่อการเรียนรู้ที่น่าสนใจด้วยการนำเสนอข้อมูลทางภาพ
3. **การนำเสนอการขาย:** แสดงแนวโน้มและคาดการณ์การขายได้อย่างมีประสิทธิภาพ

ตัวอย่างเหล่านี้แสดงให้เห็นว่า Aspose.Slides สามารถผสานรวมเข้ากับเวิร์กโฟลว์ที่มีอยู่เพื่อนำเสนองานนำเสนอที่สวยงามได้อย่างไร

## คู่มือการใช้งาน

เรามาแบ่งการใช้งานออกเป็นส่วนๆ ที่สามารถจัดการได้

### การตั้งค่าสูตรในเซลล์ข้อมูลแผนภูมิ

#### ภาพรวม
ฟีเจอร์นี้ช่วยให้คุณคำนวณข้อมูลภายในแผนภูมิของคุณแบบไดนามิกได้โดยการตั้งค่าสูตรโดยตรงในเซลล์ข้อมูล ฟีเจอร์นี้มีประโยชน์อย่างยิ่งสำหรับการอัปเดตอัตโนมัติและการรับรองความถูกต้องในงานนำเสนอต่างๆ

#### ขั้นตอนการดำเนินการ

1. **สร้างวัตถุการนำเสนอ:**
   เริ่มต้นด้วยการสร้างการเริ่มต้นวัตถุการนำเสนอซึ่งเราจะเพิ่มแผนภูมิของเรา
   
   ```python
   import aspose.slides as slides
   
   def set_formula_in_chart_cell():
       with slides.Presentation() as presentation:
           # ขั้นตอนต่อไปมีดังนี้...
   ```

2. **เพิ่มแผนภูมิคอลัมน์แบบคลัสเตอร์:**
   แทรกแผนภูมิคอลัมน์แบบกลุ่มลงในสไลด์แรกของการนำเสนอของคุณ
   
   ```python
   chart = presentation.slides[0].shapes.add_chart(
       slides.charts.ChartType.CLUSTERED_COLUMN, 150, 150, 500, 300)
   ```

3. **สมุดงานข้อมูลแผนภูมิการเข้าถึง:**
   ดึงข้อมูลวัตถุเวิร์กบุ๊กที่เชื่อมโยงกับแผนภูมิเพื่อจัดการเซลล์ข้อมูล
   
   ```python
   workbook = chart.chart_data.chart_data_workbook
   ```

4. **ตั้งค่าสูตรในเซลล์ B2:**
   กำหนดสูตรสำหรับเซลล์ B2 โดยใช้รูปแบบสเปรดชีตมาตรฐาน
   
   ```python
   cell1 = workbook.get_cell(0, "B2")
   cell1.formula = "1 + SUM(F2:H5)"
   ```

5. **ใช้สัญลักษณ์ R1C1 ในเซลล์ C2:**
   ทางเลือกอื่นคือใช้สัญลักษณ์ R1C1 สำหรับสูตรที่ซับซ้อนยิ่งขึ้น
   
   ```python
   cell2 = workbook.get_cell(0, "C2")
   cell2.r1c1_formula = "MAX(R2C6:R5C8) / 3"
   ```

6. **คำนวณสูตร:**
   คำนวณผลลัพธ์ของสูตรเหล่านี้ภายในแผนภูมิของคุณ
   
   ```python
   workbook.calculate_formulas()
   ```

7. **บันทึกการนำเสนอของคุณ:**
   บันทึกการนำเสนอของคุณไปยังไดเร็กทอรีเอาต์พุตที่ระบุ
   
   ```python
   presentation.save("YOUR_OUTPUT_DIRECTORY/charts_data_cell_formulas_out.pptx")
   ```

## คู่มือการใช้งาน

มาสำรวจวิธีการแยกแมโคร VBA จากงานนำเสนอ PowerPoint กัน

### คุณสมบัติ: การแยกแมโคร VBA

#### ภาพรวม

ฟีเจอร์นี้ช่วยให้คุณเข้าถึงและพิมพ์แมโคร VBA ที่ฝังอยู่ในงานนำเสนอ PowerPoint ของคุณได้ โดยใช้ Aspose.Slides คุณสามารถเปิดงานนำเสนอและโต้ตอบกับโปรเจ็กต์ VBA ของงานนำเสนอได้โดยอัตโนมัติ

#### การดำเนินการแบบทีละขั้นตอน

##### โหลดงานนำเสนอ

เริ่มต้นโดยระบุเส้นทางไปยังไดเร็กทอรีเอกสารของคุณและโหลดไฟล์งานนำเสนอ:

```python
document_directory = 'YOUR_DOCUMENT_DIRECTORY/'
presentation_file_path = document_directory + 'VBA.pptm'

with slides.Presentation(presentation_file_path) as pres:
    # โค้ดสำหรับการเข้าถึงโครงการ VBA จะตามมาที่นี่
```

##### ตรวจสอบโครงการ VBA

ตรวจสอบให้แน่ใจว่าการนำเสนอมีโครงการ VBA:

```python
if pres.vba_project is not None:
    print("VBA Project found.")
else:
    print("No VBA Project in this presentation.")
```

##### การแยกและพิมพ์แมโคร

ทำซ้ำในแต่ละโมดูลภายในโครงการ VBA เพื่อแยกชื่อแมโครและโค้ดต้นฉบับ:

```python
for module in pres.vba_project.modules:
    print(f"Module Name: {module.name}")
    print(f"Source Code:\n{module.source_code}\n")
```

### คำอธิบายพารามิเตอร์และวิธีการ

- **`slides.Presentation()`**: เปิดไฟล์ PowerPoint เพื่อการโต้ตอบ
- **`pres.vba_project`**: ตรวจสอบว่าการนำเสนอมีโครงการ VBA หรือไม่ ส่งคืน `None` ถ้าไม่มี
- **`pres.vba_project.modules`**: ให้การเข้าถึงโมดูลทั้งหมดภายในโครงการ VBA

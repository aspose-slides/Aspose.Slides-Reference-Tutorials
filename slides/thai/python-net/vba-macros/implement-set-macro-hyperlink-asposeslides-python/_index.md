## คู่มือการใช้งาน

ตอนนี้คุณได้ตั้งค่าสภาพแวดล้อมเรียบร้อยแล้ว ต่อไปมาดูการใช้งานฟีเจอร์หลักของเรากัน

### การเพิ่มรูปทรงด้วยไฮเปอร์ลิงก์มาโคร

#### ภาพรวม
หัวข้อนี้จะแนะนำคุณเกี่ยวกับการเพิ่มรูปร่างปุ่มลงในสไลด์ PowerPoint และการกำหนดเหตุการณ์คลิกไฮเปอร์ลิงก์มาโคร ซึ่งเป็นสิ่งสำคัญสำหรับการทำงานอัตโนมัติภายในการนำเสนอ

#### การดำเนินการแบบทีละขั้นตอน

##### เพิ่มรูปทรงปุ่ม

ก่อนอื่นเราจะเพิ่มรูปร่างปุ่มว่างเปล่าลงในสไลด์แรกตามพิกัดที่เฉพาะเจาะจง:

```python
import aspose.slides as slides

macro_name = "TestMacro"
with slides.Presentation() as presentation:
    # การเพิ่มรูปร่างปุ่มว่างลงในสไลด์แรก
    shape = presentation.slides[0].shapes.add_auto_shape(
        slides.ShapeType.BLANK_BUTTON, 20, 20, 80, 30
    )
```
- **พารามิเตอร์:**
  - `ShapeType.BLANK_BUTTON`: ระบุว่าเราจะเพิ่มปุ่มว่าง
  - `(20, 20, 80, 30)`:พิกัด x, y และความกว้าง ความสูงของรูปร่าง

##### ตั้งค่าการคลิกไฮเปอร์ลิงก์มาโคร

ถัดไป ให้ตั้งค่าไฮเปอร์ลิงก์มาโครโดยคลิกที่รูปร่างที่เพิ่ม:

```python
    # การกำหนดไฮเปอร์ลิงก์มาโครให้กับรูปร่าง
    shape.hyperlink_manager.set_macro_hyperlink_click(macro_name)
```
- **พารามิเตอร์:**
  - `macro_name`: ชื่อของแมโครที่จะถูกเรียกใช้งานเมื่อคลิกปุ่ม

## การพิจารณาประสิทธิภาพ

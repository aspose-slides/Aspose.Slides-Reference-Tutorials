## คู่มือการใช้งาน

ต่อไปเรามาดูการตั้งค่าแบบอักษรเริ่มต้นสำหรับข้อความปกติและแบบเอเชียกัน

### การตั้งค่าแบบอักษรเริ่มต้น

คุณลักษณะนี้ช่วยให้คุณกำหนดแบบอักษรที่จะใช้เมื่อไม่ได้ระบุไว้ในเนื้อหาการนำเสนอ

#### ขั้นตอนที่ 1: สร้าง LoadOptions

เริ่มต้นโดยการกำหนด `LoadOptions` เพื่อระบุพารามิเตอร์การโหลดของคุณ:

```python
load_options = slides.LoadOptions()
load_options.load_format = slides.LoadFormat.AUTO
```

นี่แจ้งให้ Aspose.Slides ทราบถึงวิธีการตีความรูปแบบไฟล์โดยอัตโนมัติ

#### ขั้นตอนที่ 2: ระบุแบบอักษรเริ่มต้น

ต่อไป ให้ตั้งค่าแบบอักษรทั้งแบบปกติและแบบเอเชีย ในตัวอย่างนี้ เราใช้ "Wingdings" เพื่อความเรียบง่าย:

```python
load_options.default_regular_font = "Wingdings"
load_options.default_asian_font = "Wingdings"
```

ซึ่งจะทำให้แน่ใจถึงความสอดคล้องกันระหว่างข้อความทั้งหมดภายในงานนำเสนอของคุณ

#### ขั้นตอนที่ 3: โหลดงานนำเสนอ

เมื่อคุณตั้งค่าตัวเลือกแล้ว ให้โหลดไฟล์ PowerPoint โดยใช้พารามิเตอร์เหล่านี้:

```python
with slides.Presentation("YOUR_DOCUMENT_DIRECTORY/text_default_fonts.pptx", load_options) as pptx:
    # สร้างภาพขนาดย่อของสไลด์และบันทึกเป็น PNG
    pptx.slides[0].get_image(1, 1).save("YOUR_OUTPUT_DIRECTORY/text_default_fonts_out.png", slides.ImageFormat.PNG)
    
    # บันทึกการนำเสนอในรูปแบบ PDF
    pptx.save("YOUR_OUTPUT_DIRECTORY/text_default_fonts_out.pdf", slides.export.SaveFormat.PDF)
    
    # นอกจากนี้ ให้บันทึกเป็นไฟล์ XPS
    pptx.save("YOUR_OUTPUT_DIRECTORY/text_default_fonts_out.xps", slides.export.SaveFormat.XPS)
```

### การประยุกต์ใช้งานจริง

การใช้แบบอักษรเริ่มต้นอาจเป็นประโยชน์ในสถานการณ์ต่างๆ:

1. **การสร้างแบรนด์องค์กร**:ให้แน่ใจว่าการนำเสนอทั้งหมดสอดคล้องกับแนวทางของแบรนด์
2. **การนำเสนอหลายภาษา**จัดการหลายภาษาได้อย่างราบรื่นด้วยการตั้งค่าแบบอักษรเอเชีย
3. **ความสม่ำเสมอระหว่างทีม**:ปรับแบบอักษรให้เป็นมาตรฐานสำหรับการมีส่วนร่วมของสมาชิกในทีมที่แตกต่างกัน

## การพิจารณาประสิทธิภาพ

- **เพิ่มประสิทธิภาพการใช้ทรัพยากร**:โหลดเฉพาะสไลด์ที่จำเป็นเพื่อประหยัดหน่วยความจำ
- **การจัดการหน่วยความจำที่มีประสิทธิภาพ**:กำจัดสิ่งของทันทีเพื่อปลดปล่อยทรัพยากร

การยึดมั่นตามหลักปฏิบัติที่ดีที่สุดจะช่วยให้มั่นใจได้ว่าแอปพลิเคชันของคุณทำงานได้อย่างราบรื่นโดยไม่มีค่าใช้จ่ายที่ไม่จำเป็น

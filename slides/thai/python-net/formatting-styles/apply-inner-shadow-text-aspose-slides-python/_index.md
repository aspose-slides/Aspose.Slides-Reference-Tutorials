## คู่มือการใช้งาน
ตอนนี้เรามาดูการใช้เอฟเฟกต์เงาภายในกับข้อความในสไลด์ PowerPoint กัน
### การเพิ่มข้อความด้วยเอฟเฟกต์เงาภายใน
#### ภาพรวม
เราจะสร้างรูปสี่เหลี่ยมผืนผ้า เพิ่มข้อความลงไป แล้วจึงใช้เอฟเฟกต์เงาภายใน วิธีนี้จะทำให้สไลด์ของคุณดูสวยงามขึ้นโดยเพิ่มความลึกให้กับข้อความ
#### คำแนะนำทีละขั้นตอน
**1. การเข้าถึงสไลด์**
ขั้นแรก ให้รับการอ้างอิงถึงสไลด์แรกในงานนำเสนอของคุณ:

```python
slide = total_presentation.slides[0]
```
**2. การเพิ่มรูปร่างอัตโนมัติ**
เพิ่มรูปสี่เหลี่ยมผืนผ้าเพื่อเก็บข้อความของเรา:

```python
auto_shape = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 150, 75, 400, 300)
auto_shape.fill_format.fill_type = slides.FillType.NO_FILL
```
**3. การแทรกข้อความ**
แทรกกรอบข้อความและกำหนดเนื้อหาสำหรับสี่เหลี่ยมของคุณ:

```python
auto_shape.add_text_frame("Aspose TextBox")
port = auto_shape.text_frame.paragraphs[0].portions[0]
pf = port.portion_format
pf.font_height = 50  # ตั้งค่าขนาดตัวอักษรเพื่อเพิ่มการมองเห็น
```
**4. การใช้เอฟเฟกต์เงาภายใน**
เปิดใช้งานและกำหนดค่าเอฟเฟกต์เงาภายในบนข้อความ:

```python
ef = pf.effect_format
ef.enable_inner_shadow_effect()
# กำหนดค่าพารามิเตอร์เงาภายใน
ef.inner_shadow_effect.blur_radius = 8.0  # รัศมีเบลอสำหรับเงาที่นุ่มนวลขึ้น
ef.inner_shadow_effect.direction = 90.0  # ทิศทางเงาเป็นองศา
ef.inner_shadow_effect.distance = 6.0    # ระยะห่างของเงาจากข้อความ
ef.inner_shadow_effect.shadow_color.b = 189  # ส่วนประกอบสีน้ำเงินของสีเงา
# กำหนดรูปแบบที่สอดคล้องกันโดยใช้สีรูปแบบ
ef.inner_shadow_effect.shadow_color.color_type = slides.ColorType.SCHEME
ef.inner_shadow_effect.shadow_color.scheme_color = slides.SchemeColor.ACCENT1
```
**5. การบันทึกการนำเสนอของคุณ**
สุดท้ายให้บันทึกการนำเสนอของคุณลงในไฟล์:

```python
presentation.save("YOUR_OUTPUT_DIRECTORY/text_apply_inner_shadow_out.pptx")
```

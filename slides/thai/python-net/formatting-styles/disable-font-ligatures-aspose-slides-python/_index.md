# วิธีปิดใช้งานฟอนต์ลิเกเจอร์ในการส่งออก PPTX โดยใช้ Aspose.Slides สำหรับ Python

## การแนะนำ

เมื่อคุณส่งออกงานนำเสนอ PowerPoint เป็น HTML การรักษาความสม่ำเสมอของตัวอักษรถือเป็นสิ่งสำคัญ ปัจจัยหนึ่งที่สามารถส่งผลต่อการอ่านและการออกแบบได้คือตัวอักษรควบคู่กัน ในบทช่วยสอนนี้ เราจะแนะนำคุณเกี่ยวกับการปิดใช้งานตัวอักษรควบคู่กันเหล่านี้โดยใช้ **Aspose.Slides สำหรับ Python**กระบวนการนี้เหมาะสำหรับนักพัฒนาที่ต้องการนำเสนอข้อความที่สม่ำเสมอบนแพลตฟอร์มที่แตกต่างกันหรือผู้ที่ต้องการควบคุมการส่งออกมากขึ้น

**สิ่งที่คุณจะได้เรียนรู้:**

## การตั้งค่า Aspose.Slides สำหรับ Python

ในการเริ่มใช้ Aspose.Slides ให้ติดตั้งไลบรารีดังต่อไปนี้:

**การติดตั้ง PIP:**

```bash
pip install aspose.slides
```

หลังจากติดตั้งแล้ว คุณสามารถสำรวจฟีเจอร์ต่างๆ ได้ พิจารณาขอใบอนุญาตทดลองใช้งานฟรีหากจำเป็น

### การเริ่มต้นขั้นพื้นฐาน

วิธีการเริ่มต้น Aspose.Slides ในสคริปต์ Python ของคุณมีดังนี้:

```python
import aspose.slides as slides

# เริ่มต้นวัตถุการนำเสนอ
pres = slides.Presentation()
```

การตั้งค่านี้ช่วยให้คุณสามารถดำเนินการต่างๆ กับไฟล์ PowerPoint ได้ รวมถึงการปิดใช้งานการผูกฟอนต์ด้วย

## คู่มือการใช้งาน

### ปิดใช้งานฟอนต์ลิเกเจอร์ระหว่างการส่งออก

ในส่วนนี้ เราจะมุ่งเน้นโดยเฉพาะถึงวิธีปิดใช้งานการผนวกฟอนต์เมื่อส่งออกงานนำเสนอจาก PPTX เป็น HTML โดยใช้ Aspose.Slides

#### โหลดการนำเสนอของคุณ

ขั้นแรก โหลดไฟล์ PowerPoint ที่คุณต้องการส่งออก ใช้ `Presentation` ชั้นเรียนสำหรับสิ่งนี้:

```python
with slides.Presentation("YOUR_DOCUMENT_DIRECTORY/TextLigatures.pptx") as pres:
    # ดำเนินการตามขั้นตอนต่อไป...
```

แทนที่ `"YOUR_DOCUMENT_DIRECTORY/TextLigatures.pptx"` พร้อมเส้นทางไฟล์การนำเสนอของคุณ

#### บันทึกด้วยการตั้งค่าเริ่มต้น

ก่อนที่จะปิดการใช้งานลิเกเจอร์ เรามาทำความเข้าใจกระบวนการส่งออกเริ่มต้นกันก่อน ซึ่งจะช่วยให้คุณเห็นการเปลี่ยนแปลงได้:

```python
pres.save("YOUR_OUTPUT_DIRECTORY/EnableLigatures-out.html", slides.export.SaveFormat.HTML)
```

การดำเนินการนี้จะบันทึกการนำเสนอในรูปแบบ HTML พร้อมเปิดใช้งานฟอนต์ควบคู่กัน

#### กำหนดค่าตัวเลือกการส่งออก

ถัดไป กำหนดค่าตัวเลือกเพื่อปิดใช้งานการผูกแบบอักษร:

```python
options = slides.export.HtmlOptions()
options.disable_font_ligatures = True
```

การ `HtmlOptions` คลาสช่วยให้คุณสามารถระบุการตั้งค่าต่างๆ สำหรับผลลัพธ์ HTML การตั้งค่า `disable_font_ligatures` ถึง `True` ป้องกันไม่ให้ Aspose.Slides ใช้ ligatures

#### การส่งออกด้วยลิเกเจอร์ที่ปิดใช้งาน

สุดท้าย ให้ใช้ตัวเลือกเหล่านี้เมื่อบันทึกการนำเสนอ:

```python
pres.save("YOUR_OUTPUT_DIRECTORY/DisableLigatures-out.html", slides.export.SaveFormat.HTML, options)
```

การดำเนินการนี้จะช่วยให้แน่ใจว่าไฟล์ HTML ที่ส่งออกมีการปิดใช้งานการเชื่อมโยงฟอนต์ ทำให้ข้อความปรากฏสม่ำเสมอ

### เคล็ดลับการแก้ไขปัญหา

- **ปัญหาเส้นทางไฟล์**:ตรวจสอบเส้นทางทั้งหมดอีกครั้งเพื่อความถูกต้องและสามารถเข้าถึงได้

# Vytváření prezentací v PowerPointu s formátovaným textem

Aspose.Slides pro .NET je výkonné API, které umožňuje vývojářům vytvářet, manipulovat a převádět prezentace v PowerPointu bez nutnosti používat Microsoft PowerPoint. Tento příklad ukazuje, jak vytvořit jednoduchou prezentaci obsahující obdélníkový tvar s formátovanými textovými odstavci.

Kód vytvoří novou prezentaci, přidá obdélníkový tvar na první snímek a poté přidá několik odstavců s různými možnostmi formátování textu. Ukazuje, jak nastavit vlastnosti písma, jako je velikost a rodina písma, pro konkrétní části textu.

## Pokyny k instalaci

1. Instalace přes Správce balíčků NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Instalace přes .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Instalace přes konzoli Správce balíčků:
   ```
   PM> Install-Package Aspose.Slides
   ```

4. Přidejte odkaz do svého projektu:
   Po instalaci přidejte do projektu odkaz:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

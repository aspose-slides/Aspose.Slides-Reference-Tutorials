# Vytváření svislého textu v PowerPointu pomocí Aspose.Slides

Aspose.Slides pro .NET poskytuje výkonné funkce pro programovou tvorbu a manipulaci s prezentacemi v PowerPointu. Tento příklad ukazuje, jak vytvořit prezentaci obsahující obdélníkový tvar se svisle otočeným textem. Text je orientován pod úhlem 270 stupňů, takže se čte zdola nahoru. Tato technika je užitečná pro vytváření bočních popisků, jedinečných rozvržení textu nebo specializovaných designových prvků ve vašich prezentacích.

## Pokyny k instalaci

1. Instalace Aspose.Slides pro .NET
   
   Nainstalujte balíček Aspose.Slides pomocí Správce balíčků NuGet:
   
   ```csharp
   Install-Package Aspose.Slides
   ```
   
   Nebo pomocí .NET CLI:
   
   ```csharp
   dotnet add package Aspose.Slides
   ```

2. Přidat reference
   
   Přidejte do souboru C# následující příkaz using:
   
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

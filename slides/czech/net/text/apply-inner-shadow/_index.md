# Vytvářejte prezentace v PowerPointu s textovým polem v Aspose.Slides pro .NET

Aspose.Slides pro .NET je výkonné API, které umožňuje vývojářům vytvářet, upravovat a převádět prezentace v PowerPointu bez nutnosti používat Microsoft PowerPoint. Tento příklad ukazuje, jak vytvořit prezentaci s přizpůsobeným textovým polem, použít formátování a uložit ji jako soubor PPTX.

## Pokyny k instalaci

1. Instalace přes Správce balíčků NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Instalace přes .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Ruční instalace:
   - Stáhněte si nejnovější verzi z [Stránka s vydáním Aspose.Slides pro .NET](https://releases.aspose.com/slides/net/)
   - Přidejte odkaz na DLL do svého projektu

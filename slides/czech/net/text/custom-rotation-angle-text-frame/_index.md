# Otáčení názvů a popisků grafů v PowerPointu s Aspose.Slides

Aspose.Slides pro .NET poskytuje výkonné funkce pro programovou tvorbu a manipulaci s prezentacemi v PowerPointu. Tento příklad ukazuje, jak vytvořit prezentaci s klastrovaným sloupcovým grafem, který obsahuje otočené textové prvky – konkrétně otáčení názvu grafu i popisků dat pro lepší čitelnost a vizuální přitažlivost.

## Pokyny k instalaci

1. Instalace přes Správce balíčků NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Nebo nainstalujte pomocí .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Přidejte jmenný prostor do souboru C#:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Charts;
   ```

# Vytváření vlastních odrážek v prezentacích PowerPointu

Aspose.Slides pro .NET poskytuje robustní funkce pro programovou tvorbu a manipulaci s prezentacemi v PowerPointu. Tento příklad se konkrétně zaměřuje na implementaci vlastních odrážek v textových rámech a ukazuje, jak ovládat typ odrážek, barvu, odsazení a velikost. Kód vytvoří prezentaci s obdélníkovým tvarem obsahujícím dva odstavce - jeden s odrážkou se symboly a druhý s číslovaným stylem odrážek.

## Pokyny k instalaci

1. Nainstalujte Aspose.Slides pro .NET pomocí Správce balíčků NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Nebo si balíček stáhněte přímo:
   - Navštivte [Stránka pro stažení Aspose.Slides](https://downloads.aspose.com/slides/net)
   - Stáhněte si nejnovější verzi
   - Extrahujte a odkazujte na soubor Aspose.Slides.dll ve vašem projektu

3. Přidejte potřebný příkaz using:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

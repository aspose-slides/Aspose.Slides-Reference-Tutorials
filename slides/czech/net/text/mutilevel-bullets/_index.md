# Vytváření víceúrovňových odrážek v PowerPointu pomocí Aspose.Slides

Aspose.Slides pro .NET poskytuje výkonné API pro programovou tvorbu a manipulaci s prezentacemi v PowerPointu. Tento příklad ukazuje, jak vytvořit prezentaci s víceúrovňovými odrážkami, jak přidat automatický tvar s textovým rámečkem a jak formátovat odstavce s různými styly odrážek a úrovněmi odsazení. Můžete snadno vytvářet hierarchický obsah s přizpůsobenými symboly odrážek pro každou úroveň odsazení, což je ideální pro vytváření strukturovaných a profesionálních prezentací bez nutnosti instalace aplikace Microsoft PowerPoint.

## Pokyny k instalaci

1. Instalace přes Správce balíčků NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Instalace přes .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Instalace přes konzoli Správce balíčků:
   ```
   PM> Install-Package Aspose.Slides
   ```

4. Ruční instalace:
   - Stáhněte si nejnovější verzi z [Aspose.Slides pro .NET](https://releases.aspose.com/slides/net/)
   - Přidejte do projektu odkaz na Aspose.Slides.dll

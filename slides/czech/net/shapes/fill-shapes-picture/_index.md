# Vytváření tvarů s dlaždicovou výplní obrázků v PowerPointu pomocí Aspose.Slides

Aspose.Slides pro .NET poskytuje výkonné funkce pro programovou tvorbu a manipulaci s prezentacemi v PowerPointu. Tento příklad ukazuje, jak vytvořit obdélníkový tvar a vyplnit jej dlaždicovým obrázkovým vzorem. Dlaždicová obrázková výplň je obzvláště užitečná pro vytváření texturovaných pozadí, vodoznaků nebo dekorativních prvků v prezentacích.

## Pokyny k instalaci

1. Instalace přes Správce balíčků NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Instalace přes .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Ruční instalace:
   - Stáhněte si nejnovější verzi z [Aspose.Slides pro .NET](https://releases.aspose.com/slides/net/)
   - Přidejte do projektu odkaz na Aspose.Slides.dll

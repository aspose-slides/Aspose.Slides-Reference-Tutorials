# Vytváření vlastních tvarů čar v PowerPointu pomocí Aspose.Slides

Aspose.Slides pro .NET poskytuje výkonné funkce pro programovou tvorbu a manipulaci s prezentacemi v PowerPointu. Tento příklad ukazuje, jak vytvořit vlastní tvar čáry s různými možnostmi formátování, včetně stylu čáry, šířky, čárkovaného vzoru, šipek a barvy.

S Aspose.Slides můžete snadno přidávat do svých prezentací profesionálně vypadající čáry, aniž byste museli mít na svém počítači nainstalovanou aplikaci Microsoft PowerPoint. To je obzvláště užitečné pro automatické generování sestav nebo vytváření prezentací na straně serveru.

## Pokyny k instalaci

1. Instalace přes Správce balíčků NuGet:
   ```
   Install-Package Aspose.Slides.NET
   ```

2. Instalace přes .NET CLI:
   ```
   dotnet add package Aspose.Slides.NET
   ```

3. Ruční přidání reference:
   - Stáhněte si nejnovější verzi z [Aspose Releases](https://releases.aspose.com/slides/net/)
   - Přidejte do projektu odkaz na Aspose.Slides.dll

# Přidávání obrázků do slidů PowerPointu pomocí Aspose.Slides

Aspose.Slides pro .NET poskytuje výkonné funkce pro programově vytvářet a upravovat prezentace v PowerPointu. Tento příklad ukazuje, jak načíst externí obrázek a přidat ho do snímku prezentace jako rámeček obrázku – což je běžný požadavek při generování sestav, vytváření šablon nebo dynamickém vytváření prezentací ze zdrojů dat.

Níže uvedený kód zvládne vše od vytvoření nové prezentace přes přidání obrázku až po uložení finálního souboru PPTX, a to vše jen s několika řádky kódu v jazyce C#.

## Pokyny k instalaci

1. Nainstalujte Aspose.Slides pro .NET pomocí jedné z těchto metod:
   - Správce balíčků NuGet: `Install-Package Aspose.Slides.NET`
   - Rozhraní příkazového řádku .NET: `dotnet add package Aspose.Slides.NET`
   - Nebo si stáhněte přímo z [Webové stránky Aspose](https://releases.aspose.com/slides/net/)

2. Přidejte jmenný prostor do souboru C#:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

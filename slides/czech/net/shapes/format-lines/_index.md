# Vytvoření obdélníkového tvaru s vlastním formátováním čar v PowerPointu

Aspose.Slides pro .NET poskytuje výkonné funkce pro programově vytvářet a upravovat prezentace v PowerPointu. Tento příklad ukazuje, jak vytvořit obdélníkový tvar s přizpůsobeným formátováním čar, včetně nastavení stylu čáry, šířky, čárkovaného vzoru a barvy. Ukázkový kód vytvoří novou prezentaci, přidá obdélník na první snímek a použije specifické formátování na výplň i obrys tvaru, což vede k vizuálně atraktivnímu prvku s bílou výplní a modrým čárkovaným obrysem.

## Pokyny k instalaci

1. Nainstalujte Aspose.Slides pro .NET pomocí jedné z následujících metod:

   - Správce balíčků NuGet:
     ```
     Install-Package Aspose.Slides
     ```

   - Konzola Správce balíčků:
     ```
     PM> Install-Package Aspose.Slides
     ```

   - Rozhraní příkazového řádku .NET:
     ```
     dotnet add package Aspose.Slides
     ```

2. Přidejte potřebný jmenný prostor do souboru C#:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

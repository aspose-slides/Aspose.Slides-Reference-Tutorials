# Vytváření eliptických tvarů v PowerPointu pomocí Aspose.Slides

Aspose.Slides pro .NET je výkonné API, které umožňuje vývojářům vytvářet, číst, psát, upravovat a převádět prezentace v PowerPointu bez nutnosti používat Microsoft PowerPoint. Tento příklad ukazuje, jak vytvořit eliptický tvar v prezentaci, použít formátování včetně barvy výplně a vlastností čáry a uložit výsledek jako soubor PPTX. Kód ukazuje schopnost Aspose.Slides manipulovat s tvary s přesnou kontrolou nad jejich vzhledem a umístěním.

## Pokyny k instalaci

1. Instalace přes Správce balíčků NuGet:
   ```
   Install-Package Aspose.Slides
   ```

   Nebo pomocí .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

2. Přidat reference:
   Přidejte do souboru C# následující jmenný prostor:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

3. Nastavení licence (volitelné):
   Pokud máte licenci, použijte ji:
   ```csharp
   License license = new License();
   license.SetLicense("Aspose.Slides.lic");
   ```

# Zarovnání a seskupení tvarů v prezentacích PowerPointu

Aspose.Slides pro .NET poskytuje výkonné funkce pro programovou tvorbu a manipulaci s prezentacemi v PowerPointu. Tento příklad ukazuje, jak zarovnávat tvary v rámci snímků a pracovat se seskupenými tvary v prezentacích v PowerPointu. Naučíte se, jak umisťovat obdélníky na určité souřadnice, zarovnávat je vzhledem ke snímku nebo vůči sobě navzájem a selektivně zarovnávat tvary v rámci skupin.

## Pokyny k instalaci

1. Nainstalujte Aspose.Slides pro .NET pomocí jedné z těchto metod:
   
   - Správce balíčků NuGet:
     ```
     Install-Package Aspose.Slides.NET
     ```
   
   - Konzola Správce balíčků:
     ```
     PM> Install-Package Aspose.Slides.NET
     ```
   
   - Rozhraní příkazového řádku .NET:
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. Přidejte do svého projektu reference:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.IO;
   ```

3. Stáhněte si níže uvedený ukázkový kód a integrujte ho do svého projektu.

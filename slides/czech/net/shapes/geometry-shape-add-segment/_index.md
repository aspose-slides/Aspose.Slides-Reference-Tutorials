# Přidávání segmentů do GeometryShape v prezentacích PowerPointu

Tento příklad ukazuje, jak přistupovat k geometrickým cestám tvarů v prezentacích PowerPointu a jak je upravovat pomocí Aspose.Slides pro .NET. Přidáním vlastních úseček k obdélníkovému tvaru můžete vytvářet složitější geometrické obrazce, které nejsou k dispozici prostřednictvím standardních typů tvarů. Tato technika je užitečná pro vytváření vlastních diagramů, vývojových diagramů a dalších vizuálních prvků ve vašich prezentacích.

## Pokyny k instalaci

1. Nainstalujte Aspose.Slides pro .NET pomocí jedné z následujících metod:

   - Správce balíčků NuGet:
     ```
     Install-Package Aspose.Slides.NET
     ```

   - Konzola Správce balíčků:
     ```
     PM> Install-Package Aspose.Slides.NET
     ```

   - Rozhraní příkazového řádku .NET:
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. Přidejte jmenný prostor do souboru C#:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

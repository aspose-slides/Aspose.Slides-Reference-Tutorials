# Úprava geometrických cest v obrazcích PowerPointu pomocí Aspose.Slides

Aspose.Slides pro .NET poskytuje výkonné funkce pro programovou úpravu prezentací v PowerPointu, včetně pokročilé manipulace s tvary. Tento příklad ukazuje, jak vytvářet vlastní tvary úpravou geometrické cesty existujícího tvaru – konkrétně odstraněním segmentu z tvaru srdce a vytvořením jedinečné varianty.

Geometrická cesta definuje obrys tvaru pomocí řady bodů a úseček. Přístupem k těmto cestám a jejich úpravou mohou vývojáři vytvářet vysoce přizpůsobené tvary, které nejsou k dispozici ve standardní knihovně tvarů PowerPointu.

## Pokyny k instalaci

1. Instalace přes Správce balíčků NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Instalace přes .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Přidejte do kódu direktivu using:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

4. Přidejte inicializaci licence při spouštění aplikace:
   ```csharp
   // Pokud máte licenci
   License license = new License();
   license.SetLicense("path/to/Aspose.Slides.lic");
   ```

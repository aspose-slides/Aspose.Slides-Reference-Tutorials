# Vytváření a otáčení obdélníkových tvarů v PowerPointu

Aspose.Slides pro .NET je výkonné API, které umožňuje vývojářům vytvářet, upravovat a převádět prezentace v PowerPointu bez nutnosti instalace Microsoft PowerPointu. Tento příklad ukazuje, jak vytvořit prezentaci s otočeným obdélníkovým tvarem a ukazuje možnosti manipulace s tvary, které knihovna nabízí.

Aspose.Slides můžete programově přidávat do snímků různé tvary a aplikovat transformace, jako je rotace. Tato funkce je obzvláště užitečná pro vytváření dynamických prezentací s přesnou kontrolou nad vizuálními prvky.

## Pokyny k instalaci

1. Instalace přes Správce balíčků NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Instalace přes .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Přidejte do kódu direktivu using:
   ```csharp
   using Aspose.Slides;
   ```

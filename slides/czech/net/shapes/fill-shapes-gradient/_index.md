# Vytváření eliptických tvarů s přechodovou výplní v PowerPointu pomocí Aspose.Slides

Tento příklad ukazuje, jak programově vytvořit prezentaci v PowerPointu obsahující elipsu s vlastním efektem přechodové výplně. Kód vytvoří novou prezentaci, přidá elipsu na první snímek a aplikuje lineární přechodovou výplň s červenými a fialovými barevnými zarážkami. Aspose.Slides pro .NET poskytuje komplexní API pro vytváření a manipulaci s prezentacemi v PowerPointu s přesnou kontrolou nad tvary, formátováním a efekty.

## Pokyny k instalaci

1. Nainstalujte Aspose.Slides pro .NET pomocí jedné z těchto metod:
   - Správce balíčků NuGet: `Install-Package Aspose.Slides.NET`
   - Rozhraní příkazového řádku .NET: `dotnet add package Aspose.Slides.NET`
   - Konzola Správce balíčků: `Install-Package Aspose.Slides.NET`

2. Přidejte do souboru C# následující příkaz using:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

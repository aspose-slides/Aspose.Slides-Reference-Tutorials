# Vytváření tabulek se svislým textem v prezentacích v PowerPointu

Aspose.Slides pro .NET poskytuje výkonné funkce pro programově vytvářet a manipulovat s prezentacemi v PowerPointu. Tento příklad ukazuje, jak vytvořit tabulku s přizpůsobeným formátováním, včetně svislého zarovnání textu a rotace – což je běžný požadavek pro specializované obchodní prezentace a vizualizaci dat.

## Pokyny k instalaci

1. Nainstalujte Aspose.Slides pro .NET pomocí Správce balíčků NuGet:
   ```
   Install-Package Aspose.Slides
   ```
   Nebo přes .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

2. Přidejte do svého projektu odkazy:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

3. Stáhnout dokumentaci:
   - [Referenční informace k API](https://reference.aspose.com/slides/net/)
   - [Průvodce pro vývojáře](https://docs.aspose.com/slides/net/)

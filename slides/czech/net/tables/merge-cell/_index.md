# Vytvářejte a formátujte tabulky v PowerPointu pomocí Aspose.Slides

Aspose.Slides pro .NET poskytuje výkonné funkce pro vytváření sofistikovaných tabulkových struktur v prezentacích PowerPointu. S touto knihovnou můžete vytvářet tabulky s vlastními rozměry, aplikovat detailní formátování na ohraničení buněk a strategicky slučovat buňky pro vytvoření složitých rozvržení. Tento příklad ukazuje, jak vytvořit tabulku 4x4, aplikovat jednotné červené ohraničení o šířce 5px na všechny buňky a implementovat specifický vzor slučování buněk pro vytvoření vlastního rozvržení.

## Pokyny k instalaci

1. Instalace přes Správce balíčků NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Instalace přes .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Přidejte do kódu příkazy using:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

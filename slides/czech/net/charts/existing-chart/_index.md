# Úprava grafů v PowerPointu pomocí Aspose.Slides

Aspose.Slides pro .NET poskytuje výkonná API pro programovou tvorbu a úpravu prezentací v PowerPointu. Tento příklad ukazuje, jak pracovat s grafy v existujících prezentacích aktualizací dat grafu, přidáním nových řad a změnou typů grafů – to vše bez nutnosti instalace aplikace Microsoft PowerPoint.

## Pokyny k instalaci

1. Instalace přes Správce balíčků NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Nebo pomocí .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Přidejte následující pomocí direktiv:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Charts;
   ```

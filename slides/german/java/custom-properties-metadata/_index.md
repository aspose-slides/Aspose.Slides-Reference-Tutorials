Jedes Tutorial enthält funktionierende Java-Codebeispiele für gängige Metadatenszenarien und unterstützt Sie beim Erstellen von Anwendungen, die umfassende Präsentationsinformationen für eine bessere Organisation und Durchsuchbarkeit bereitstellen.

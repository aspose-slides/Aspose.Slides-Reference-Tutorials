# Erstellen von PowerPoint-Präsentationen mit formatiertem Text

Aspose.Slides für .NET ist eine leistungsstarke API, mit der Entwickler PowerPoint-Präsentationen erstellen, bearbeiten und konvertieren können, ohne Microsoft PowerPoint zu benötigen. Dieses Beispiel zeigt, wie Sie eine einfache Präsentation mit einer rechteckigen Form und formatierten Textabsätzen erstellen.

Der Code erstellt eine neue Präsentation, fügt der ersten Folie eine rechteckige Form hinzu und fügt anschließend mehrere Absätze mit unterschiedlichen Textformatierungsoptionen hinzu. Er zeigt, wie Sie Schrifteigenschaften wie Größe und Familie für bestimmte Textabschnitte festlegen.

## Installationsanweisungen

1. Installieren Sie über den NuGet-Paketmanager:
   ```
   Install-Package Aspose.Slides
   ```

2. Installation über .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Installation über die Paketmanager-Konsole:
   ```
   PM> Install-Package Aspose.Slides
   ```

4. Fügen Sie Ihrem Projekt einen Verweis hinzu:
   Fügen Sie nach der Installation einen Verweis auf Ihr Projekt hinzu:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

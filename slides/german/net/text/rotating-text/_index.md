# Erstellen von vertikalem Text in PowerPoint mit Aspose.Slides

Aspose.Slides für .NET bietet leistungsstarke Funktionen zum programmgesteuerten Erstellen und Bearbeiten von PowerPoint-Präsentationen. Dieses Beispiel zeigt, wie Sie eine rechteckige Präsentation mit vertikal gedrehtem Text erstellen. Der Text ist in einem 270-Grad-Winkel ausgerichtet und somit von unten nach oben lesbar. Diese Technik eignet sich für die Erstellung von Seitenbeschriftungen, individuellen Textlayouts oder speziellen Designelementen in Ihren Präsentationen.

## Installationsanweisungen

1. Installieren Sie Aspose.Slides für .NET
   
   Installieren Sie das Aspose.Slides-Paket mit dem NuGet-Paket-Manager:
   
   ```csharp
   Install-Package Aspose.Slides
   ```
   
   Oder mithilfe der .NET-CLI:
   
   ```csharp
   dotnet add package Aspose.Slides
   ```

2. Referenzen hinzufügen
   
   Fügen Sie Ihrer C#-Datei die folgende using-Anweisung hinzu:
   
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

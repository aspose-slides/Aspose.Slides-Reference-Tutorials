# Erstellen benutzerdefinierter Aufzählungspunkte in PowerPoint-Präsentationen

Aspose.Slides für .NET bietet leistungsstarke Funktionen zum programmgesteuerten Erstellen und Bearbeiten von PowerPoint-Präsentationen. Dieses Beispiel konzentriert sich speziell auf die Implementierung benutzerdefinierter Aufzählungspunkte in Textrahmen und zeigt, wie Aufzählungstyp, Farbe, Einzug und Größe gesteuert werden. Der Code erstellt eine rechteckige Präsentation mit zwei Absätzen – einem mit einem symbolischen und einem nummerierten Aufzählungszeichen.

## Installationsanweisungen

1. Installieren Sie Aspose.Slides für .NET mit dem NuGet Package Manager:
   ```
   Install-Package Aspose.Slides
   ```

2. Alternativ können Sie das Paket direkt herunterladen:
   - Besuchen Sie die [Aspose.Slides Download-Seite](https://downloads.aspose.com/slides/net)
   - Laden Sie die neueste Version herunter
   - Extrahieren und referenzieren Sie die Aspose.Slides.dll in Ihrem Projekt

3. Fügen Sie die erforderliche Using-Anweisung hinzu:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

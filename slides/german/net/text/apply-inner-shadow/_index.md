# Erstellen Sie PowerPoint-Präsentationen mit TextBox in Aspose.Slides für .NET

Aspose.Slides für .NET ist eine leistungsstarke API, mit der Entwickler PowerPoint-Präsentationen erstellen, bearbeiten und konvertieren können, ohne Microsoft PowerPoint zu benötigen. Dieses Beispiel zeigt, wie Sie eine Präsentation mit einem benutzerdefinierten Textfeld erstellen, formatieren und als PPTX-Datei speichern.

## Installationsanweisungen

1. Installieren Sie über den NuGet-Paketmanager:
   ```
   Install-Package Aspose.Slides
   ```

2. Installation über .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Manuell installieren:
   - Laden Sie die neueste Version herunter von der [Aspose.Slides für .NET-Releases-Seite](https://releases.aspose.com/slides/net/)
   - Fügen Sie Ihrem Projekt den DLL-Verweis hinzu

# Rotieren von Diagrammtiteln und Beschriftungen in PowerPoint mit Aspose.Slides

Aspose.Slides für .NET bietet leistungsstarke Funktionen zum programmgesteuerten Erstellen und Bearbeiten von PowerPoint-Präsentationen. Dieses Beispiel zeigt, wie Sie eine Präsentation mit einem gruppierten Säulendiagramm erstellen, das gedrehte Textelemente enthält – insbesondere die Drehung des Diagrammtitels und der Datenbeschriftungen für bessere Lesbarkeit und Optik.

## Installationsanweisungen

1. Installieren Sie über den NuGet-Paketmanager:
   ```
   Install-Package Aspose.Slides
   ```

2. Oder installieren Sie über die .NET-CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Fügen Sie den Namespace zu Ihrer C#-Datei hinzu:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Charts;
   ```

# Erstellen mehrstufiger Aufzählungspunkte in PowerPoint mit Aspose.Slides

Aspose.Slides für .NET bietet eine leistungsstarke API zum programmgesteuerten Erstellen und Bearbeiten von PowerPoint-Präsentationen. Dieses Beispiel zeigt, wie Sie eine Präsentation mit mehrstufigen Aufzählungspunkten erstellen, eine automatische Form mit Textrahmen hinzufügen und Absätze mit unterschiedlichen Aufzählungsstilen und Einrückungsebenen formatieren. Sie können ganz einfach hierarchische Inhalte mit individuellen Aufzählungssymbolen für jede Einrückungsebene erstellen – ideal für strukturierte und professionelle Präsentationen, ohne dass Microsoft PowerPoint installiert sein muss.

## Installationsanweisungen

1. Installieren Sie über den NuGet-Paketmanager:
   ```
   Install-Package Aspose.Slides
   ```

2. Installation über .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Installation über die Paketmanager-Konsole:
   ```
   PM> Install-Package Aspose.Slides
   ```

4. Manuelle Installation:
   - Laden Sie die neueste Version herunter von [Aspose.Slides für .NET](https://releases.aspose.com/slides/net/)
   - Fügen Sie in Ihrem Projekt einen Verweis auf Aspose.Slides.dll hinzu

# Erstellen von Tabellen mit vertikalem Text in PowerPoint-Präsentationen

Aspose.Slides für .NET bietet leistungsstarke Funktionen zum programmgesteuerten Erstellen und Bearbeiten von PowerPoint-Präsentationen. Dieses Beispiel zeigt, wie Sie eine Tabelle mit individueller Formatierung erstellen, einschließlich vertikaler Textausrichtung und -drehung – eine häufige Anforderung für spezielle Geschäftspräsentationen und Datenvisualisierungen.

## Installationsanweisungen

1. Installieren Sie Aspose.Slides für .NET über den NuGet-Paket-Manager:
   ```
   Install-Package Aspose.Slides
   ```
   Oder über die .NET-CLI:
   ```
   dotnet add package Aspose.Slides
   ```

2. Fügen Sie Ihrem Projekt Referenzen hinzu:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

3. Dokumentationsressourcen herunterladen:
   - [API-Referenz](https://reference.aspose.com/slides/net/)
   - [Entwicklerhandbuch](https://docs.aspose.com/slides/net/)

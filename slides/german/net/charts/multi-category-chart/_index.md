# Erstellen von gruppierten Säulendiagrammen mit Aspose.Slides für .NET

Aspose.Slides für .NET bietet leistungsstarke Funktionen zum programmgesteuerten Erstellen und Bearbeiten von PowerPoint-Präsentationen. Dieses Beispiel zeigt, wie Sie ein gruppiertes Säulendiagramm mit benutzerdefinierten Kategorien und Datenreihen erstellen.

## Installation

```csharp
// Installation über den NuGet-Paketmanager
Install-Package Aspose.Slides
```

Oder fügen Sie Ihrer Projektdatei den folgenden Verweis hinzu:

```xml
<PackageReference Include="Aspose.Slides" Version="23.2.0" />
```

# Ändern von Diagrammen in PowerPoint mit Aspose.Slides

Aspose.Slides für .NET bietet leistungsstarke APIs zum programmgesteuerten Erstellen und Ändern von PowerPoint-Präsentationen. Dieses Beispiel zeigt, wie Sie mit Diagrammen in bestehenden Präsentationen arbeiten, indem Sie Diagrammdaten aktualisieren, neue Reihen hinzufügen und Diagrammtypen ändern – alles ohne die Installation von Microsoft PowerPoint.

## Installationsanweisungen

1. Installieren Sie über den NuGet-Paketmanager:
   ```
   Install-Package Aspose.Slides
   ```

2. Oder mithilfe der .NET-CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Fügen Sie die folgenden Using-Direktiven hinzu:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Charts;
   ```

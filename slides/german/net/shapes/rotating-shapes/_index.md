# Erstellen und Drehen von Rechteckformen in PowerPoint

Aspose.Slides für .NET ist eine leistungsstarke API, mit der Entwickler PowerPoint-Präsentationen erstellen, bearbeiten und konvertieren können, ohne Microsoft PowerPoint installieren zu müssen. Dieses Beispiel zeigt, wie Sie eine Präsentation mit einem gedrehten Rechteck erstellen und die Möglichkeiten der Formbearbeitung der Bibliothek demonstrieren.

Mit Aspose.Slides können Sie Ihren Folien programmgesteuert verschiedene Formen hinzufügen und Transformationen wie Rotationen anwenden. Diese Funktionalität ist besonders nützlich für die Erstellung dynamischer Präsentationen mit präziser Kontrolle über visuelle Elemente.

## Installationsanweisungen

1. Installieren Sie über den NuGet-Paketmanager:
   ```
   Install-Package Aspose.Slides
   ```

2. Installation über .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Fügen Sie Ihrem Code die Using-Direktive hinzu:
   ```csharp
   using Aspose.Slides;
   ```

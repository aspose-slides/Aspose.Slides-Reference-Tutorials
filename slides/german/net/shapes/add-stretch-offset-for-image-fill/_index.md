# Hinzufügen von Bildern zu PowerPoint-Folien mit Aspose.Slides

Aspose.Slides für .NET bietet leistungsstarke Funktionen zum programmgesteuerten Erstellen und Bearbeiten von PowerPoint-Präsentationen. Dieses Beispiel zeigt, wie Sie ein externes Bild laden und als Bilderrahmen in eine Präsentationsfolie einfügen – eine häufige Anforderung beim Generieren von Berichten, Erstellen von Vorlagen oder beim dynamischen Erstellen von Präsentationen aus Datenquellen.

Der folgende Code erledigt alles, vom Erstellen einer neuen Präsentation über das Hinzufügen des Bildes bis hin zum Speichern der endgültigen PPTX-Datei, und das alles mit nur wenigen Zeilen C#-Code.

## Installationsanweisungen

1. Installieren Sie Aspose.Slides für .NET mit einer der folgenden Methoden:
   - NuGet-Paket-Manager: `Install-Package Aspose.Slides.NET`
   - .NET-CLI: `dotnet add package Aspose.Slides.NET`
   - Oder direkt herunterladen von [Aspose-Website](https://releases.aspose.com/slides/net/)

2. Fügen Sie den Namespace zu Ihrer C#-Datei hinzu:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

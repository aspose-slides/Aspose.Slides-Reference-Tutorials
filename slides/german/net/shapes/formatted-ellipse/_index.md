# Erstellen elliptischer Formen in PowerPoint mit Aspose.Slides

Aspose.Slides für .NET ist eine leistungsstarke API, mit der Entwickler PowerPoint-Präsentationen erstellen, lesen, schreiben, ändern und konvertieren können, ohne Microsoft PowerPoint zu benötigen. Dieses Beispiel zeigt, wie Sie eine elliptische Form in einer Präsentation erstellen, Formatierungen wie Füllfarbe und Linieneigenschaften anwenden und das Ergebnis als PPTX-Datei speichern. Der Code demonstriert die Fähigkeit von Aspose.Slides, Formen mit präziser Kontrolle über Aussehen und Positionierung zu bearbeiten.

## Installationsanweisungen

1. Installieren Sie über den NuGet-Paketmanager:
   ```
   Install-Package Aspose.Slides
   ```

   Oder mithilfe der .NET-CLI:
   ```
   dotnet add package Aspose.Slides
   ```

2. Referenzen hinzufügen:
   Fügen Sie Ihrer C#-Datei den folgenden Namespace hinzu:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

3. Lizenz-Setup (optional):
   Wenn Sie eine Lizenz haben, wenden Sie diese wie folgt an:
   ```csharp
   License license = new License();
   license.SetLicense("Aspose.Slides.lic");
   ```

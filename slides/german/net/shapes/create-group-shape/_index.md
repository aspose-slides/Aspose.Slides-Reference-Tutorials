# Erstellen Sie Gruppenformen in PowerPoint mit Aspose.Slides

Aspose.Slides für .NET bietet umfassende APIs zum programmgesteuerten Erstellen und Bearbeiten von PowerPoint-Präsentationen. Dieses Beispiel zeigt, wie Sie eine Gruppenform erstellen, mehrere Rechtecke hinzufügen, die Formen korrekt positionieren, Rahmen hinzufügen und die Präsentation speichern. Gruppenformen sind unerlässlich, um zusammengehörige Formen zu organisieren, die als Einheit bearbeitet werden müssen. Dies bietet mehr Struktur und Kontrolle über Ihre Präsentationselemente.

## Installationsanweisungen

1. Installieren Sie Aspose.Slides für .NET mit einer der folgenden Methoden:

   - NuGet-Paket-Manager:
     ```
     Install-Package Aspose.Slides.NET
     ```

   - Paketmanager-Konsole:
     ```
     PM> Install-Package Aspose.Slides.NET
     ```

   - .NET-CLI:
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. Fügen Sie Ihrem Projekt Referenzen hinzu:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

3. Richten Sie eine Lizenz ein (optional für den Produktionseinsatz):
   ```csharp
   // Legen Sie eine Lizenz fest, um Evaluierungsbeschränkungen aufzuheben
   License license = new License();
   license.SetLicense("Aspose.Slides.lic");
   ```

# Erstellen einer Rechteckform mit benutzerdefinierter Linienformatierung in PowerPoint

Aspose.Slides für .NET bietet leistungsstarke Funktionen zum programmgesteuerten Erstellen und Bearbeiten von PowerPoint-Präsentationen. Dieses Beispiel zeigt, wie Sie eine Rechteckform mit individueller Linienformatierung erstellen, einschließlich der Festlegung von Linienstil, Breite, Strichmuster und Farbe. Der Beispielcode erstellt eine neue Präsentation, fügt der ersten Folie ein Rechteck hinzu und wendet spezifische Formatierungen auf Füllung und Umriss der Form an. Das Ergebnis ist ein optisch ansprechendes Element mit weißer Füllung und blau gestricheltem Umriss.

## Installationsanweisungen

1. Installieren Sie Aspose.Slides für .NET mit einer der folgenden Methoden:

   - NuGet-Paket-Manager:
     ```
     Install-Package Aspose.Slides
     ```

   - Paketmanager-Konsole:
     ```
     PM> Install-Package Aspose.Slides
     ```

   - .NET-CLI:
     ```
     dotnet add package Aspose.Slides
     ```

2. Fügen Sie Ihrer C#-Datei den erforderlichen Namespace hinzu:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

# Erstellen von Ellipsenformen mit Farbverlaufsfüllung in PowerPoint mit Aspose.Slides

Dieses Beispiel zeigt, wie Sie programmgesteuert eine PowerPoint-Präsentation mit einer Ellipse und einem benutzerdefinierten Farbverlaufseffekt erstellen. Der Code erstellt eine neue Präsentation, fügt der ersten Folie eine Ellipse hinzu und wendet einen linearen Farbverlauf mit roten und violetten Farbstopps an. Aspose.Slides für .NET bietet eine umfassende API zum Erstellen und Bearbeiten von PowerPoint-Präsentationen mit präziser Kontrolle über Formen, Formatierungen und Effekte.

## Installationsanweisungen

1. Installieren Sie Aspose.Slides für .NET mit einer der folgenden Methoden:
   - NuGet-Paket-Manager: `Install-Package Aspose.Slides.NET`
   - .NET-CLI: `dotnet add package Aspose.Slides.NET`
   - Paketmanager-Konsole: `Install-Package Aspose.Slides.NET`

2. Fügen Sie Ihrer C#-Datei die folgende using-Anweisung hinzu:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

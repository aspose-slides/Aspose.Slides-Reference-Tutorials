# Ausrichten und Gruppieren von Formen in PowerPoint-Präsentationen

Aspose.Slides für .NET bietet leistungsstarke Funktionen zum programmgesteuerten Erstellen und Bearbeiten von PowerPoint-Präsentationen. Dieses Beispiel zeigt, wie Sie Formen innerhalb von Folien ausrichten und mit gruppierten Formen in PowerPoint-Präsentationen arbeiten. Sie lernen, wie Sie Rechtecke an bestimmten Koordinaten positionieren, sie relativ zur Folie oder zueinander ausrichten und Formen innerhalb von Gruppen selektiv ausrichten.

## Installationsanweisungen

1. Installieren Sie Aspose.Slides für .NET mit einer der folgenden Methoden:
   
   - NuGet-Paket-Manager:
     ```
     Install-Package Aspose.Slides.NET
     ```
   
   - Paketmanager-Konsole:
     ```
     PM> Install-Package Aspose.Slides.NET
     ```
   
   - .NET-CLI:
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. Fügen Sie Ihrem Projekt Referenzen hinzu:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.IO;
   ```

3. Laden Sie den Beispielcode unten herunter und integrieren Sie ihn in Ihr Projekt.

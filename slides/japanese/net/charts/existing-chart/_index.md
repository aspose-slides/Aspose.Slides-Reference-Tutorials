# Aspose.Slides で PowerPoint のグラフを変更する

Aspose.Slides for .NET は、PowerPoint プレゼンテーションをプログラムで作成および変更するための強力な API を提供します。この例では、Microsoft PowerPoint をインストールすることなく、既存のプレゼンテーション内のグラフデータの更新、新しい系列の追加、グラフの種類の変更など、グラフを操作する方法を示します。

## インストール手順

1. NuGet パッケージ マネージャー経由でインストールします。
   ```
   Install-Package Aspose.Slides
   ```

2. または .NET CLI を使用します:
   ```
   dotnet add package Aspose.Slides
   ```

3. 次の using ディレクティブを追加します。
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Charts;
   ```

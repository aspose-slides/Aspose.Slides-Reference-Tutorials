## インストール

```csharp
// NuGet パッケージ マネージャー経由でインストールする
Install-Package Aspose.Slides
```

または、プロジェクト ファイルに次の参照を追加します。

```xml
<PackageReference Include="Aspose.Slides" Version="23.2.0" />
```

# Aspose.Slides を使用して PowerPoint のグラフのタイトルとラベルを回転する

Aspose.Slides for .NET は、PowerPoint プレゼンテーションをプログラムで作成および操作するための強力な機能を提供します。この例では、回転したテキスト要素を備えた集合縦棒グラフを含むプレゼンテーションを作成する方法を示します。具体的には、グラフのタイトルとデータラベルの両方を回転させることにより、読みやすさと視覚的な魅力を向上させます。

## インストール手順

1. NuGet パッケージ マネージャー経由でインストールします。
   ```
   Install-Package Aspose.Slides
   ```

2. または、.NET CLI 経由でインストールします。
   ```
   dotnet add package Aspose.Slides
   ```

3. C# ファイルに名前空間を追加します。
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Charts;
   ```

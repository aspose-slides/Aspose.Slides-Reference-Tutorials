# Aspose.Slides for .NET で TextBox を使った PowerPoint プレゼンテーションを作成する

## インストール手順

1. NuGet パッケージ マネージャー経由でインストールします。
   ```
   Install-Package Aspose.Slides
   ```

2. .NET CLI 経由でインストール:
   ```
   dotnet add package Aspose.Slides
   ```

3. 手動でインストール:
   - 最新バージョンをダウンロードするには、 [Aspose.Slides for .NET リリース ページ](https://releases.aspose.com/slides/net/)
   - プロジェクトにDLL参照を追加する

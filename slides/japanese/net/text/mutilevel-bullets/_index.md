# Aspose.Slides を使用して PowerPoint で多段階の箇条書きを作成する

Aspose.Slides for .NET は、PowerPoint プレゼンテーションをプログラムで作成および操作するための強力な API を提供します。この例では、複数レベルの箇条書きを含むプレゼンテーションを作成する方法を紹介します。テキストフレーム付きのオートシェイプを追加し、段落ごとに異なる箇条書きスタイルとインデントレベルを設定する方法を示します。インデントレベルごとにカスタマイズされた箇条書き記号を使用することで、階層化されたコンテンツを簡単に作成できます。Microsoft PowerPoint をインストールすることなく、構造化されたプロフェッショナルなプレゼンテーションを作成するのに最適です。

## インストール手順

1. NuGet パッケージ マネージャー経由でインストールします。
   ```
   Install-Package Aspose.Slides
   ```

2. .NET CLI 経由でインストール:
   ```
   dotnet add package Aspose.Slides
   ```

3. パッケージ マネージャー コンソール経由でインストールします。
   ```
   PM> Install-Package Aspose.Slides
   ```

4. 手動インストール:
   - 最新バージョンをダウンロードするには [Aspose.Slides .NET 版](https://releases.aspose.com/slides/net/)
   - プロジェクトにAspose.Slides.dllへの参照を追加します

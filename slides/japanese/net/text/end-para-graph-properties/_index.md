# 書式付きテキストを使用したPowerPointプレゼンテーションの作成

Aspose.Slides for .NETは、Microsoft PowerPointを必要とせずにPowerPointプレゼンテーションを作成、操作、変換できる強力なAPIです。この例では、書式設定されたテキスト段落を含む長方形の図形を含むシンプルなプレゼンテーションを作成する方法を説明します。

このコードは新しいプレゼンテーションを作成し、最初のスライドに長方形の図形を追加し、異なるテキスト書式設定オプションを持つ複数の段落を追加します。テキストの特定の部分に、サイズやフォントファミリーなどのフォントプロパティを設定する方法を示しています。

## インストール手順

1. NuGet パッケージ マネージャー経由でインストールします。
   ```
   Install-Package Aspose.Slides
   ```

2. .NET CLI 経由でインストール:
   ```
   dotnet add package Aspose.Slides
   ```

3. パッケージ マネージャー コンソール経由でインストールします。
   ```
   PM> Install-Package Aspose.Slides
   ```

4. プロジェクトへの参照を追加する:
   インストール後、プロジェクトへの参照を追加します。
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

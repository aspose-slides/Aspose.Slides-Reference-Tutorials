# Aspose.Slides を使用して PowerPoint で縦書きテキストを作成する

Aspose.Slides for .NET は、PowerPoint プレゼンテーションをプログラムで作成および操作するための強力な機能を提供します。この例では、縦向きに回転したテキストを含む長方形の図形を含むプレゼンテーションを作成する方法を示します。テキストは 270 度の角度で配置され、下から上に向かって読み取られます。この手法は、プレゼンテーションにサイドラベル、独自のテキストレイアウト、または特別なデザイン要素を作成するのに役立ちます。

## インストール手順

1. Aspose.Slides for .NET をインストールする
   
   NuGet パッケージ マネージャーを使用して Aspose.Slides パッケージをインストールします。
   
   ```csharp
   Install-Package Aspose.Slides
   ```
   
   または、.NET CLI を使用します。
   
   ```csharp
   dotnet add package Aspose.Slides
   ```

2. 参照を追加する
   
   次の using ステートメントを C# ファイルに追加します。
   
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

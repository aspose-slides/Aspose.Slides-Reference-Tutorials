# PowerPoint プレゼンテーションでカスタム箇条書きを作成する

Aspose.Slides for .NET は、PowerPoint プレゼンテーションをプログラムで作成および操作するための強力な機能を提供します。この例では、テキストフレームにカスタム箇条書きを実装することに特に焦点を当て、箇条書きの種類、色、インデント、サイズを制御する方法を示します。このコードは、記号付きの箇条書きと番号付きの箇条書きスタイルを使用した2つの段落を含む長方形のプレゼンテーションを作成します。

## インストール手順

1. NuGet パッケージ マネージャーを使用して Aspose.Slides for .NET をインストールします。
   ```
   Install-Package Aspose.Slides
   ```

2. または、パッケージを直接ダウンロードします。
   - 訪問 [Aspose.Slides ダウンロードページ](https://downloads.aspose.com/slides/net)
   - 最新バージョンをダウンロード
   - プロジェクトで Aspose.Slides.dll を抽出して参照します。

3. 必要な using ステートメントを追加します。
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

### プレゼンテーションファイルの保存
**概要**
プレゼンテーションを保存するには、変更したスライドや新しく作成したスライドをディスクに書き込む必要があります。これは、プログラムによる変更を永続化するために不可欠です。

#### ステップ1: プレゼンテーションを読み込む
まず、既存の PowerPoint ファイルを読み込みます。
```csharp
Presentation presentation = new Presentation("input.pptx");
```
これにより、プレゼンテーションがメモリに読み込まれ、変更または保存できるようになります。

#### ステップ2: プレゼンテーションを保存する
次に、指定した場所に保存します。
```csharp
presentation.Save("output.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
```
確実に `YOUR_OUTPUT_DIRECTORY` 希望のパスに置き換えられます。この手順により、すべての変更がディスクに書き込まれます。

### プレゼンテーションから画像を抽出する
**概要**
スライド内に埋め込まれた画像を抽出し、アプリケーションの他の場所で使用したり、分析したりします。

#### ステップ1：スライドにアクセスする
各スライドを繰り返します。
```csharp
foreach (ISlide slide in presentation.Slides)
{
    // 各スライドを処理する
}
```
このループを使用すると、個々のスライドとそのコンポーネントにアクセスできます。

#### ステップ2: 画像を抽出する
各スライド内で画像を抽出します。
```csharp
int imageIndex = 0;
foreach (IPPImage img in slide.Images)
{
    using (FileStream fileStream = new FileStream($"image{imageIndex++}.png", FileMode.Create))
    {
        img.SystemImage.Save(fileStream, ImageFormat.Png);
    }
}
```
このコードは各画像をディスクに保存します。 `imageIndex` 抽出された画像のファイル名が一意であることを保証します。

# Aspose.Slides を使用して PowerPoint の表のセルに画像を追加する

## インストール手順

1. 次のいずれかの方法で Aspose.Slides for .NET をインストールします。
   - NuGet パッケージ マネージャー: `Install-Package Aspose.Slides.NET`
   - .NET CLI: `dotnet add package Aspose.Slides.NET`
   - プロジェクトに Aspose.Slides.dll への参照を直接追加します。

2. C# ファイルに名前空間参照を追加します。
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

3. ドキュメントと API リファレンスをダウンロードしてください:
   - [Aspose.Slides for .NET ドキュメント](https://docs.aspose.com/slides/net/)
   - [Aspose.Slides for .NET API リファレンス](https://reference.aspose.com/slides/net/)

## インストール手順

1. Aspose.Slides for .NET をインストールします。
   
   NuGet パッケージ マネージャー コンソールの使用:
   ```
   Install-Package Aspose.Slides.NET
   ```

   .NET CLI の使用:
   ```
   dotnet add package Aspose.Slides.NET
   ```

2. 必要な using ディレクティブをコードに追加します。
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Tables;
   using System.IO;
   ```

# PowerPointで長方形を作成および回転する

Aspose.Slides for .NETは、Microsoft PowerPointをインストールすることなく、開発者がPowerPointプレゼンテーションを作成、変更、変換できる強力なAPIです。この例では、回転した四角形を含むプレゼンテーションを作成する方法を示し、ライブラリの図形操作機能を紹介します。

Aspose.Slides を使用すると、プログラムからスライドに様々な図形を追加したり、回転などの変形を適用したりできます。この機能は、視覚要素を正確に制御してダイナミックなプレゼンテーションを作成する際に特に役立ちます。

## インストール手順

1. NuGet パッケージ マネージャー経由でインストールします。
   ```
   Install-Package Aspose.Slides
   ```

2. .NET CLI 経由でインストール:
   ```
   dotnet add package Aspose.Slides
   ```

3. コードに using ディレクティブを追加します。
   ```csharp
   using Aspose.Slides;
   ```

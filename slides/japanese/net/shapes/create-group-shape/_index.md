# Aspose.Slides を使用して PowerPoint でグループ図形を作成する

Aspose.Slides for .NET は、PowerPoint プレゼンテーションをプログラムで作成および操作するための包括的な API を提供します。この例では、グループ図形を作成し、その中に複数の四角形図形を追加し、図形を適切に配置し、フレームを追加して、プレゼンテーションを保存する方法を示します。グループ図形は、関連する図形を単一のユニットとして操作するために不可欠であり、プレゼンテーション要素をより構造化して制御できます。

## インストール手順

1. 次のいずれかの方法で Aspose.Slides for .NET をインストールします。

   - NuGet パッケージ マネージャー:
     ```
     Install-Package Aspose.Slides.NET
     ```

   - パッケージ マネージャー コンソール:
     ```
     PM> Install-Package Aspose.Slides.NET
     ```

   - .NET CLI:
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. プロジェクトに参照を追加します。
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

3. ライセンスを設定します (本番環境での使用の場合はオプション)。
   ```csharp
   // 評価制限を解除するためのライセンスを設定する
   License license = new License();
   license.SetLicense("Aspose.Slides.lic");
   ```

# Aspose.Slides を使用して PowerPoint で楕円形を作成する

Aspose.Slides for .NETは、Microsoft PowerPointを必要とせずにPowerPointプレゼンテーションを作成、読み込み、書き込み、変更、変換できる強力なAPIです。この例では、プレゼンテーションに楕円形を作成し、塗りつぶしの色や線のプロパティなどの書式設定を適用し、結果をPPTXファイルとして保存する方法を紹介します。このコードは、Aspose.Slidesが図形の外観と位置を正確に制御して操作できることを示しています。

## インストール手順

1. NuGet パッケージ マネージャー経由でインストールします。
   ```
   Install-Package Aspose.Slides
   ```

   または、.NET CLI を使用します。
   ```
   dotnet add package Aspose.Slides
   ```

2. 参照を追加:
   次の名前空間を C# ファイルに追加します。
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

3. ライセンス設定（オプション）:
   ライセンスをお持ちの場合は、以下を使用して適用します。
   ```csharp
   License license = new License();
   license.SetLicense("Aspose.Slides.lic");
   ```

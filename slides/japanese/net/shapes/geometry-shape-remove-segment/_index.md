# Aspose.Slides を使用して PowerPoint 図形のジオメトリ パスを変更する

Aspose.Slides for .NET は、高度な図形操作を含む、PowerPoint プレゼンテーションをプログラムで変更するための強力な機能を提供します。この例では、既存の図形のジオメトリパスを変更することでカスタム図形を作成する方法を示します。具体的には、ハート型からセグメントを削除して独自のバリエーションを作成します。

ジオメトリパスは、一連の点と線分によって図形のアウトラインを定義します。開発者はこれらのパスにアクセスして変更することで、標準のPowerPoint図形ライブラリにはない、高度にカスタマイズされた図形を作成できます。

## インストール手順

1. NuGet パッケージ マネージャー経由でインストールします。
   ```
   Install-Package Aspose.Slides
   ```

2. .NET CLI 経由でインストール:
   ```
   dotnet add package Aspose.Slides
   ```

3. コードに using ディレクティブを追加します。
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

4. アプリケーションの起動時にライセンス初期化を追加します。
   ```csharp
   // 免許をお持ちの場合
   License license = new License();
   license.SetLicense("path/to/Aspose.Slides.lic");
   ```

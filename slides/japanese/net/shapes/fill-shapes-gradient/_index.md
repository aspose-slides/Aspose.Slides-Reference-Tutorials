# Aspose.Slides を使用して PowerPoint でグラデーション塗りつぶしの楕円形を作成する

この例では、楕円とカスタムグラデーション塗りつぶし効果を含むPowerPointプレゼンテーションをプログラムで作成する方法を示します。このコードは、新しいプレゼンテーションを作成し、最初のスライドに楕円を追加し、赤と紫のカラーストップを含む線形グラデーション塗りつぶしを適用します。Aspose.Slides for .NETは、図形、書式、効果を詳細に制御しながらPowerPointプレゼンテーションを作成および操作するための包括的なAPIを提供します。

## インストール手順

1. 次のいずれかの方法で Aspose.Slides for .NET をインストールします。
   - NuGet パッケージ マネージャー: `Install-Package Aspose.Slides.NET`
   - .NET CLI: `dotnet add package Aspose.Slides.NET`
   - パッケージ マネージャー コンソール: `Install-Package Aspose.Slides.NET`

2. 次の using ステートメントを C# ファイルに追加します。
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

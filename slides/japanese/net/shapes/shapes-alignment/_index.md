# PowerPoint プレゼンテーションでの図形の配置とグループ化

Aspose.Slides for .NET は、PowerPoint プレゼンテーションをプログラムで作成および操作するための強力な機能を提供します。この例では、スライド内の図形を整列させる方法と、PowerPoint プレゼンテーション内のグループ化された図形を操作する方法を紹介します。四角形を特定の座標に配置する方法、四角形をスライド全体または他の四角形に対して整列させる方法、そしてグループ内の図形を個別に整列させる方法を学習します。

## インストール手順

1. 次のいずれかの方法で Aspose.Slides for .NET をインストールします。
   
   - NuGet パッケージ マネージャー:
     ```
     Install-Package Aspose.Slides.NET
     ```
   
   - パッケージ マネージャー コンソール:
     ```
     PM> Install-Package Aspose.Slides.NET
     ```
   
   - .NET CLI:
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. プロジェクトに参照を追加します。
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.IO;
   ```

3. 以下のサンプルコードをダウンロードして、プロジェクトに統合します。

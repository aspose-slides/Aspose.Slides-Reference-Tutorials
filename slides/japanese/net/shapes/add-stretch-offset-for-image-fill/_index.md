# Aspose.Slides を使用して PowerPoint スライドに画像を追加する

Aspose.Slides for .NET は、PowerPoint プレゼンテーションをプログラムで作成および変更するための強力な機能を提供します。この例では、外部画像を読み込み、それをピクチャフレームとしてプレゼンテーションスライドに追加する方法を示します。これは、レポートの生成、テンプレートの作成、データソースからのプレゼンテーションの動的な構築など、さまざまな場面でよく使用されます。

以下のコードは、新しいプレゼンテーションの作成から画像の追加、最終的な PPTX ファイルの保存まで、すべてを数行の C# コードで処理します。

## インストール手順

1. 次のいずれかの方法で Aspose.Slides for .NET をインストールします。
   - NuGet パッケージ マネージャー: `Install-Package Aspose.Slides.NET`
   - .NET CLI: `dotnet add package Aspose.Slides.NET`
   - または直接ダウンロードしてください [Aspose ウェブサイト](https://releases.aspose.com/slides/net/)

2. C# ファイルに名前空間を追加します。
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

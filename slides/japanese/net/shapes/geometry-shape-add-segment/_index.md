# PowerPoint プレゼンテーションの GeometryShape にセグメントを追加する

この例では、Aspose.Slides for .NET を使用して、PowerPoint プレゼンテーション内の図形のジオメトリパスにアクセスし、変更する方法を示します。長方形の図形にカスタム線分を追加することで、標準的な図形の種類では表現できない複雑な幾何学図形を作成できます。この手法は、プレゼンテーションでカスタム図、フローチャート、その他の視覚要素を作成する際に役立ちます。

## インストール手順

1. 次のいずれかの方法で Aspose.Slides for .NET をインストールします。

   - NuGet パッケージ マネージャー:
     ```
     Install-Package Aspose.Slides.NET
     ```

   - パッケージ マネージャー コンソール:
     ```
     PM> Install-Package Aspose.Slides.NET
     ```

   - .NET CLI:
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. C# ファイルに名前空間を追加します。
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

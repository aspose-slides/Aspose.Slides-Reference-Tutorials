## 実装ガイド

### ダイナミックバブルチャートの作成

Aspose.Slides を使って、PowerPoint で動的なバブルチャートを作成してみましょう。この機能を使うと、さまざまなサイズのデータポイントを視覚化できるため、データセットの複数の次元を比較するのに最適です。

#### チャートの追加

**ステップ1: プレゼンテーションの初期化**

まず、チャートを追加するプレゼンテーションを作成するか開きます。

```python
with slides.Presentation() as pres:
    slide = pres.slides[0]  # 最初のスライドにアクセス
```

**ステップ2: ダイナミックバブルチャートを追加する**

定義された寸法を持つ特定の座標で、選択したスライドに動的なバブル チャートを追加します。

```python
chart = slide.shapes.add_chart(
    slides.charts.ChartType.BUBBLE, 100, 100, 400, 300
)
```

このコード スニペットは、スライド上の (100, 100) に配置され、幅 400、高さ 300 の動的なバブル チャートを作成します。

#### バブルサイズのスケールを調整する

**ステップ3：バブルのサイズを設定する**

最初の系列グループのバブルのサイズスケールを調整して、データの視覚化を微調整します。

```python
chart.chart_data.series_groups[0].bubble_size_scale = 150
```

この調整により、バブルのサイズが調整され、明瞭度と視覚的なインパクトが向上します。

#### プレゼンテーションを保存する

**ステップ4: ファイルを保存する**

調整を行った後、変更を保持するためにプレゼンテーションを保存します。

```python
pres.save('dynamic_bubble_chart_scaling_out.pptx', slides.export.SaveFormat.PPTX)
```

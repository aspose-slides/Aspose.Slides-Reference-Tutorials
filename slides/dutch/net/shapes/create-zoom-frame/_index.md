## Installatie-instructies

1. Installeren via NuGet Package Manager:
   ```
   Install-Package Aspose.Slides
   ```

2. Installeren via .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Handmatige installatie:
   - Download de nieuwste versie van [Aspose.Slides voor .NET](https://docs.aspose.com/slides/net/)
   - Voeg een verwijzing naar Aspose.Slides.dll toe in uw project

# Een rechthoekige vorm maken met aangepaste lijnopmaak in PowerPoint

Aspose.Slides voor .NET biedt krachtige mogelijkheden voor het programmatisch maken en aanpassen van PowerPoint-presentaties. Dit voorbeeld laat zien hoe u een rechthoekige vorm maakt met aangepaste lijnopmaak, inclusief het instellen van de lijnstijl, breedte, streepjespatroon en kleur. De voorbeeldcode maakt een nieuwe presentatie, voegt een rechthoek toe aan de eerste dia en past specifieke opmaak toe op zowel de vulling als de omtrek van de vorm. Dit resulteert in een visueel aantrekkelijk element met een witte vulling en een blauwe stippellijn.

## Installatie-instructies

1. Installeer Aspose.Slides voor .NET met behulp van een van de volgende methoden:

   - NuGet-pakketbeheerder:
     ```
     Install-Package Aspose.Slides
     ```

   - Pakketbeheerconsole:
     ```
     PM> Install-Package Aspose.Slides
     ```

   - .NET CLI:
     ```
     dotnet add package Aspose.Slides
     ```

2. Voeg de benodigde naamruimte toe aan uw C#-bestand:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

## Installatie-instructies

1. Installeren via NuGet Package Manager:
   ```
   Install-Package Aspose.Slides
   ```

2. Installeren via .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Voeg de using-richtlijn toe aan uw code:
   ```csharp
   using Aspose.Slides;
   ```

# Vormen maken met betegelde afbeeldingen in PowerPoint met Aspose.Slides

Aspose.Slides voor .NET biedt krachtige mogelijkheden voor het programmatisch maken en bewerken van PowerPoint-presentaties. Dit voorbeeld laat zien hoe u een rechthoekige vorm maakt en deze vult met een tegelpatroon. De tegelvulling is vooral handig voor het maken van textuurachtergronden, watermerken of decoratieve elementen in uw presentaties.

## Installatie-instructies

1. Installeren via NuGet Package Manager:
   ```
   Install-Package Aspose.Slides
   ```

2. Installeren via .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Handmatige installatie:
   - Download de nieuwste versie van [Aspose.Slides voor .NET](https://releases.aspose.com/slides/net/)
   - Voeg een verwijzing naar Aspose.Slides.dll toe in uw project

# Elliptische vormen maken in PowerPoint met Aspose.Slides

Aspose.Slides voor .NET is een krachtige API waarmee ontwikkelaars PowerPoint-presentaties kunnen maken, lezen, schrijven, wijzigen en converteren zonder Microsoft PowerPoint. Dit voorbeeld laat zien hoe u een ellipsvormige vorm in een presentatie kunt maken, opmaak kunt toepassen, inclusief vulkleur en lijneigenschappen, en het resultaat kunt opslaan als een PPTX-bestand. De code laat zien hoe Aspose.Slides vormen kan manipuleren met nauwkeurige controle over hun uiterlijk en positionering.

## Installatie-instructies

1. Installeren via NuGet Package Manager:
   ```
   Install-Package Aspose.Slides
   ```

   Of gebruik de .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

2. Referenties toevoegen:
   Voeg de volgende naamruimte toe aan uw C#-bestand:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

3. Licentie-instelling (optioneel):
   Als u een licentie heeft, kunt u deze als volgt aanvragen:
   ```csharp
   License license = new License();
   license.SetLicense("Aspose.Slides.lic");
   ```

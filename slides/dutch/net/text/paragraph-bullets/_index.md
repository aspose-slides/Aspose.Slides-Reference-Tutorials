# Aangepaste opsommingstekens maken in PowerPoint-presentaties

Aspose.Slides voor .NET biedt robuuste mogelijkheden voor het programmatisch maken en bewerken van PowerPoint-presentaties. Dit voorbeeld richt zich specifiek op de implementatie van aangepaste opsommingstekens in tekstkaders en laat zien hoe u het type, de kleur, de inspringing en de grootte van opsommingstekens kunt bepalen. De code creëert een presentatie met een rechthoekige vorm met twee alinea's: één met een symboolopsommingsteken en één met een genummerde opsommingstekenstijl.

## Installatie-instructies

1. Installeer Aspose.Slides voor .NET met behulp van NuGet Package Manager:
   ```
   Install-Package Aspose.Slides
   ```

2. U kunt het pakket ook rechtstreeks downloaden:
   - Bezoek de [Aspose.Slides downloadpagina](https://downloads.aspose.com/slides/net)
   - Download de nieuwste versie
   - Pak de Aspose.Slides.dll uit en verwijs ernaar in uw project

3. Voeg de benodigde using-instructie toe:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

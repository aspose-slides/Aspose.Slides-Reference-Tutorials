## Installatie-instructies

1. Installeer Aspose.Slides voor .NET met behulp van een van de volgende methoden:

   - Via NuGet Package Manager:
     ```
     Install-Package Aspose.Slides
     ```
   
   - Via .NET CLI:
     ```
     dotnet add package Aspose.Slides
     ```

2. Voeg de benodigde using-instructie toe aan uw C#-bestand:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

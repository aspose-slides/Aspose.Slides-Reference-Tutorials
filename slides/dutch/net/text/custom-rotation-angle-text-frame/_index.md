# Roterende grafiektitels en labels in PowerPoint met Aspose.Slides

Aspose.Slides voor .NET biedt krachtige mogelijkheden voor het programmatisch maken en bewerken van PowerPoint-presentaties. Dit voorbeeld laat zien hoe u een presentatie maakt met een geclusterde kolomgrafiek met gedraaide tekstelementen. Dit omvat het roteren van zowel de grafiektitel als de gegevenslabels voor een betere leesbaarheid en visuele aantrekkingskracht.

## Installatie-instructies

1. Installeren via NuGet Package Manager:
   ```
   Install-Package Aspose.Slides
   ```

2. Of installeer via .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Voeg de naamruimte toe aan uw C#-bestand:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Charts;
   ```

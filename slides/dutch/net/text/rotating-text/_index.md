# Verticale tekst maken in PowerPoint met Aspose.Slides

Aspose.Slides voor .NET biedt krachtige mogelijkheden voor het programmatisch maken en bewerken van PowerPoint-presentaties. Dit voorbeeld laat zien hoe u een presentatie maakt met een rechthoekige vorm met verticaal gedraaide tekst. De tekst is georiënteerd in een hoek van 270 graden, waardoor deze van onder naar boven leesbaar is. Deze techniek is handig voor het maken van zijlabels, unieke tekstindelingen of gespecialiseerde ontwerpelementen in uw presentaties.

## Installatie-instructies

1. Aspose.Slides voor .NET installeren
   
   Installeer het Aspose.Slides-pakket met NuGet Package Manager:
   
   ```csharp
   Install-Package Aspose.Slides
   ```
   
   Of gebruik de .NET CLI:
   
   ```csharp
   dotnet add package Aspose.Slides
   ```

2. Referenties toevoegen
   
   Voeg de volgende using-instructie toe aan uw C#-bestand:
   
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

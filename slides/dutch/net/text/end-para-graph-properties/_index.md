## Installatie-instructies

1. Installeren via NuGet Package Manager:
   ```
   Install-Package Aspose.Slides
   ```

2. Installeren via .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Installeren via Package Manager Console:
   ```
   PM> Install-Package Aspose.Slides
   ```

4. Verwijzing naar uw project toevoegen:
   Voeg na de installatie een verwijzing naar uw project toe:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

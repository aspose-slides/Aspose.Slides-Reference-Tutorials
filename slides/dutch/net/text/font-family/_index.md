## Installatie-instructies

1. Installeren via NuGet Package Manager:
   ```
   Install-Package Aspose.Slides
   ```

2. Installeren via .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Voeg handmatig een referentie naar uw project toe:
   - Download de nieuwste versie van [Aspose.Slides voor .NET](https://releases.aspose.com/slides/net/)
   - Voeg de DLL toe als referentie in uw project

4. Initialiseer een licentie:
   ```csharp
   // Initialiseer een licentie
   License license = new License();
   license.SetLicense("Aspose.Slides.lic");
   ```

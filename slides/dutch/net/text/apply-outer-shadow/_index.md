## Installatie-instructies

1. Installeer Aspose.Slides via NuGet Package Manager:
   ```
   Install-Package Aspose.Slides
   ```
   
   Als alternatief kunt u de .NET CLI gebruiken:
   ```
   dotnet add package Aspose.Slides
   ```

2. Voeg de nodige using-richtlijn toe:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using Aspose.Slides.Animation;
   ```

3. Raadpleeg de API-documentatie:
   - [Aspose.Slides voor .NET-documentatie](https://docs.aspose.com/slides/net/)
   - [Aspose.Slides voor .NET API-referentie](https://reference.aspose.com/slides/net/)

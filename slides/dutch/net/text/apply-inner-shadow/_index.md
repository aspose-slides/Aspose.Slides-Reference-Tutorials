# PowerPoint-presentaties maken met tekstvakken in Aspose.Slides voor .NET

Aspose.Slides voor .NET is een krachtige API waarmee ontwikkelaars PowerPoint-presentaties kunnen maken, aanpassen en converteren zonder Microsoft PowerPoint. Dit voorbeeld laat zien hoe u een presentatie met een aangepast tekstvak maakt, opmaak toepast en deze opslaat als een PPTX-bestand.

## Installatie-instructies

1. Installeren via NuGet Package Manager:
   ```
   Install-Package Aspose.Slides
   ```

2. Installeren via .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Handmatig installeren:
   - Download de nieuwste versie van de [Aspose.Slides voor .NET-releasespagina](https://releases.aspose.com/slides/net/)
   - Voeg de DLL-referentie toe aan uw project

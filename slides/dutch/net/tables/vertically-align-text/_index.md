## Installatie-instructies

1. Installeer Aspose.Slides voor .NET via NuGet Package Manager:
   ```
   Install-Package Aspose.Slides
   ```
   Of via .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

2. Voeg referenties toe aan uw project:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

3. Documentatiebronnen downloaden:
   - [API-referentie](https://reference.aspose.com/slides/net/)
   - [Ontwikkelaarsgids](https://docs.aspose.com/slides/net/)

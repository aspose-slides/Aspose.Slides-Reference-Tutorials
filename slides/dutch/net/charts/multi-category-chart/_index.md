## Installatie

```csharp
// Installeren via NuGet Package Manager
Install-Package Aspose.Slides
```

Of voeg de volgende referentie toe aan uw projectbestand:

```xml
<PackageReference Include="Aspose.Slides" Version="23.2.0" />
```

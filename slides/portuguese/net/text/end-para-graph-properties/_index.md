# Criando apresentações do PowerPoint com texto formatado

Aspose.Slides para .NET é uma API poderosa que permite aos desenvolvedores criar, manipular e converter apresentações do PowerPoint sem a necessidade do Microsoft PowerPoint. Este exemplo demonstra como criar uma apresentação simples contendo um formato retangular com parágrafos de texto formatados.

O código cria uma nova apresentação, adiciona um retângulo ao primeiro slide e, em seguida, adiciona vários parágrafos com diferentes opções de formatação de texto. Ele mostra como definir propriedades de fonte, como tamanho e família, para trechos específicos do texto.

## Instruções de instalação

1. Instalar via Gerenciador de Pacotes NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Instalar via .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Instalar via Console do Gerenciador de Pacotes:
   ```
   PM> Install-Package Aspose.Slides
   ```

4. Adicione referência ao seu projeto:
   Após a instalação, adicione uma referência ao seu projeto:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

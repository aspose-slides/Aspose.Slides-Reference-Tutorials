# Criando marcadores multinível no PowerPoint com Aspose.Slides

O Aspose.Slides para .NET fornece uma API poderosa para criar e manipular apresentações do PowerPoint programaticamente. Este exemplo demonstra como criar uma apresentação com marcadores de vários níveis, mostrando como adicionar uma forma automática com uma moldura de texto e formatar parágrafos com diferentes estilos de marcadores e níveis de recuo. Você pode criar facilmente conteúdo hierárquico com marcadores personalizados para cada nível de recuo, perfeito para criar apresentações estruturadas e profissionais sem a necessidade de instalar o Microsoft PowerPoint.

## Instruções de instalação

1. Instalar via Gerenciador de Pacotes NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Instalar via .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Instalar via Console do Gerenciador de Pacotes:
   ```
   PM> Install-Package Aspose.Slides
   ```

4. Instalação manual:
   - Baixe a versão mais recente em [Aspose.Slides para .NET](https://releases.aspose.com/slides/net/)
   - Adicione referência a Aspose.Slides.dll em seu projeto

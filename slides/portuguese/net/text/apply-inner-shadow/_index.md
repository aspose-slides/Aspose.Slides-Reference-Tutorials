# Crie apresentações do PowerPoint com TextBox no Aspose.Slides para .NET

Aspose.Slides para .NET é uma API poderosa que permite aos desenvolvedores criar, modificar e converter apresentações do PowerPoint sem a necessidade do Microsoft PowerPoint. Este exemplo demonstra como criar uma apresentação com uma caixa de texto personalizada, aplicar formatação e salvá-la como um arquivo PPTX.

## Instruções de instalação

1. Instalar via Gerenciador de Pacotes NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Instalar via .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Instalar manualmente:
   - Baixe a versão mais recente do [Página de lançamentos do Aspose.Slides para .NET](https://releases.aspose.com/slides/net/)
   - Adicione a referência DLL ao seu projeto

# Criando texto vertical no PowerPoint usando Aspose.Slides

O Aspose.Slides para .NET oferece recursos avançados para criar e manipular apresentações do PowerPoint programaticamente. Este exemplo demonstra como criar uma apresentação contendo um retângulo com texto rotacionado verticalmente. O texto é orientado em um ângulo de 270 graus, permitindo a leitura de baixo para cima. Essa técnica é útil para criar rótulos laterais, layouts de texto exclusivos ou elementos de design especializados em suas apresentações.

## Instruções de instalação

1. Instalar Aspose.Slides para .NET
   
   Instale o pacote Aspose.Slides usando o Gerenciador de Pacotes NuGet:
   
   ```csharp
   Install-Package Aspose.Slides
   ```
   
   Ou usando o .NET CLI:
   
   ```csharp
   dotnet add package Aspose.Slides
   ```

2. Adicionar referências
   
   Adicione a seguinte instrução using ao seu arquivo C#:
   
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

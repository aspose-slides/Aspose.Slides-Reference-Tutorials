# Criando marcadores personalizados em apresentações do PowerPoint

Aspose.Slides para .NET oferece recursos robustos para criar e manipular apresentações do PowerPoint programaticamente. Este exemplo se concentra especificamente na implementação de marcadores personalizados em quadros de texto, mostrando como controlar o tipo, a cor, o recuo e o tamanho dos marcadores. O código cria uma apresentação retangular contendo dois parágrafos: um com um marcador de símbolo e outro com um estilo de marcador numerado.

## Instruções de instalação

1. Instale o Aspose.Slides para .NET usando o Gerenciador de Pacotes NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Alternativamente, baixe o pacote diretamente:
   - Visite o [Página de download do Aspose.Slides](https://downloads.aspose.com/slides/net)
   - Baixe a versão mais recente
   - Extraia e referencie o Aspose.Slides.dll em seu projeto

3. Adicione a instrução using necessária:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

# Girando títulos e rótulos de gráficos no PowerPoint com Aspose.Slides

Aspose.Slides para .NET oferece recursos poderosos para criar e manipular apresentações do PowerPoint programaticamente. Este exemplo demonstra como criar uma apresentação com um gráfico de colunas agrupadas que apresenta elementos de texto rotacionados — especificamente, rotacionando o título do gráfico e os rótulos de dados para melhorar a legibilidade e o apelo visual.

## Instruções de instalação

1. Instalar via Gerenciador de Pacotes NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Ou instale via .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Adicione o namespace ao seu arquivo C#:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Charts;
   ```

# Crie tabelas padronizadas no PowerPoint com Aspose.Slides

Aspose.Slides para .NET é uma API poderosa que permite aos desenvolvedores criar, ler, escrever, modificar e converter apresentações do PowerPoint programaticamente, sem a necessidade do Microsoft PowerPoint. Este exemplo mostra como criar uma tabela padronizada com bordas personalizadas em uma apresentação do PowerPoint.

## Instruções de instalação

1. Instalar via Gerenciador de Pacotes NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Ou via .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Adicione o seguinte namespace ao seu arquivo C#:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Tables;
   using System.Drawing;
   ```

# Criando tabelas sem bordas em apresentações do PowerPoint

Aspose.Slides para .NET é uma biblioteca poderosa que permite aos desenvolvedores criar, manipular e converter apresentações do PowerPoint programaticamente. Este exemplo mostra como criar uma tabela em uma apresentação e personalizar sua aparência removendo todas as bordas das células.

O código cria uma nova apresentação com uma tabela no primeiro slide e, em seguida, itera por cada célula para definir que todas as bordas (superior, inferior, esquerda e direita) não tenham preenchimento, criando efetivamente uma tabela sem bordas.

## Instruções de instalação

1. Instalar via Gerenciador de Pacotes NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Ou instale via .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Adicione namespace ao seu código:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

# Adicionar imagens às células da tabela no PowerPoint com Aspose.Slides

Este guia demonstra como adicionar imagens programaticamente a células de tabela em apresentações do PowerPoint usando o Aspose.Slides para .NET. O exemplo mostra como criar uma apresentação, adicionar uma tabela com dimensões personalizadas, carregar uma imagem e inseri-la dentro de uma célula de tabela com a formatação adequada. Essa técnica é útil para criar apresentações profissionais com conteúdo visual rico e dados tabulares organizados.

## Instruções de instalação

1. Instale o Aspose.Slides para .NET usando um destes métodos:
   - Gerenciador de pacotes NuGet: `Install-Package Aspose.Slides.NET`
   - CLI .NET: `dotnet add package Aspose.Slides.NET`
   - Adicione referência a Aspose.Slides.dll diretamente no seu projeto

2. Adicione referência de namespace no seu arquivo C#:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

3. Baixe a documentação e a referência da API:
   - [Documentação do Aspose.Slides para .NET](https://docs.aspose.com/slides/net/)
   - [Referência da API do Aspose.Slides para .NET](https://reference.aspose.com/slides/net/)

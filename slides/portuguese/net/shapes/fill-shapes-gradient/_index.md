# Criando formas de elipse com preenchimento de gradiente no PowerPoint com Aspose.Slides

Este exemplo demonstra como criar programaticamente uma apresentação do PowerPoint contendo uma forma de elipse com um efeito de preenchimento de gradiente personalizado. O código cria uma nova apresentação, adiciona uma elipse ao primeiro slide e aplica um preenchimento de gradiente linear com interrupções de cor vermelha e roxa. O Aspose.Slides para .NET fornece uma API abrangente para criar e manipular apresentações do PowerPoint com controle preciso sobre formas, formatação e efeitos.

## Instruções de instalação

1. Instale o Aspose.Slides para .NET usando um destes métodos:
   - Gerenciador de pacotes NuGet: `Install-Package Aspose.Slides.NET`
   - CLI .NET: `dotnet add package Aspose.Slides.NET`
   - Console do gerenciador de pacotes: `Install-Package Aspose.Slides.NET`

2. Adicione a seguinte instrução using ao seu arquivo C#:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

# Adicionando imagens aos slides do PowerPoint com Aspose.Slides

O Aspose.Slides para .NET oferece recursos avançados para criar e modificar programaticamente apresentações do PowerPoint. Este exemplo demonstra como carregar uma imagem externa e adicioná-la a um slide de apresentação como uma moldura — um requisito comum ao gerar relatórios, criar modelos ou construir apresentações dinamicamente a partir de fontes de dados.

O código abaixo cuida de tudo, desde a criação de uma nova apresentação até a adição da imagem e o salvamento do arquivo PPTX final, tudo com apenas algumas linhas de código C#.

## Instruções de instalação

1. Instale o Aspose.Slides para .NET usando um destes métodos:
   - Gerenciador de pacotes NuGet: `Install-Package Aspose.Slides.NET`
   - CLI .NET: `dotnet add package Aspose.Slides.NET`
   - Ou baixe diretamente de [Site Aspose](https://releases.aspose.com/slides/net/)

2. Adicione o namespace ao seu arquivo C#:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

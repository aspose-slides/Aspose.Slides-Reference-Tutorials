# Criando fundos de imagens em mosaico no PowerPoint com Aspose.Slides

O Aspose.Slides para .NET oferece recursos avançados para criar e manipular apresentações do PowerPoint programaticamente. Este exemplo demonstra como criar uma apresentação com um retângulo preenchido com um padrão de imagem em mosaico. O código mostra como carregar uma imagem, adicioná-la a um slide como plano de fundo de um retângulo e aplicar diversas propriedades de mosaico, como deslocamento, escala, alinhamento e opções de inversão. Essa técnica é útil para criar planos de fundo personalizados, marcas d'água ou elementos decorativos em suas apresentações, com controle preciso sobre o comportamento do mosaico.

## Instruções de instalação

1. Instale o Aspose.Slides para .NET usando o Gerenciador de Pacotes NuGet:
   ```
   Install-Package Aspose.Slides.NET
   ```

2. Ou instale via .NET CLI:
   ```
   dotnet add package Aspose.Slides.NET
   ```

3. Adicione as seguintes diretivas using ao seu arquivo C#:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.IO;
   ```

# Alinhando e agrupando formas em apresentações do PowerPoint

Aspose.Slides para .NET oferece recursos avançados para criar e manipular apresentações do PowerPoint programaticamente. Este exemplo demonstra como alinhar formas dentro de slides e trabalhar com formas agrupadas em apresentações do PowerPoint. Você aprenderá a posicionar retângulos em coordenadas específicas, alinhá-los em relação ao slide ou entre si e alinhar formas seletivamente dentro de grupos.

## Instruções de instalação

1. Instale o Aspose.Slides para .NET usando um destes métodos:
   
   - Gerenciador de pacotes NuGet:
     ```
     Install-Package Aspose.Slides.NET
     ```
   
   - Console do gerenciador de pacotes:
     ```
     PM> Install-Package Aspose.Slides.NET
     ```
   
   - CLI .NET:
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. Adicione referências ao seu projeto:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.IO;
   ```

3. Baixe o código de exemplo abaixo e integre-o ao seu projeto.

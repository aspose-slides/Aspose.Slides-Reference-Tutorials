# Crie formas de grupo no PowerPoint com Aspose.Slides

O Aspose.Slides para .NET fornece APIs abrangentes para criar e manipular apresentações do PowerPoint programaticamente. Este exemplo demonstra como criar uma forma de grupo, adicionar várias formas retangulares dentro dela, posicionar as formas corretamente, adicionar moldura e salvar a apresentação. Formas de grupo são essenciais para organizar formas relacionadas que precisam ser manipuladas como uma única unidade, proporcionando mais estrutura e controle sobre os elementos da sua apresentação.

## Instruções de instalação

1. Instale o Aspose.Slides para .NET usando um destes métodos:

   - Gerenciador de pacotes NuGet:
     ```
     Install-Package Aspose.Slides.NET
     ```

   - Console do gerenciador de pacotes:
     ```
     PM> Install-Package Aspose.Slides.NET
     ```

   - CLI .NET:
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. Adicione referências ao seu projeto:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

3. Configurar uma licença (opcional para uso em produção):
   ```csharp
   // Definir licença para remover restrições de avaliação
   License license = new License();
   license.SetLicense("Aspose.Slides.lic");
   ```

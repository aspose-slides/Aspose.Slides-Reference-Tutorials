# Criando e girando formas retangulares no PowerPoint

Aspose.Slides para .NET é uma API poderosa que permite aos desenvolvedores criar, modificar e converter apresentações do PowerPoint sem a necessidade de instalação do Microsoft PowerPoint. Este exemplo demonstra como criar uma apresentação com um retângulo rotacionado, demonstrando os recursos de manipulação de formas da biblioteca.

Com o Aspose.Slides, você pode adicionar programaticamente diversas formas aos seus slides e aplicar transformações como rotação. Essa funcionalidade é particularmente útil para criar apresentações dinâmicas com controle preciso sobre os elementos visuais.

## Instruções de instalação

1. Instalar via Gerenciador de Pacotes NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Instalar via .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Adicione a diretiva using no seu código:
   ```csharp
   using Aspose.Slides;
   ```

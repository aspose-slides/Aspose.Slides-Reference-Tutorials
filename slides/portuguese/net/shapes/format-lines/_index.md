# Criando um retângulo com formatação de linha personalizada no PowerPoint

O Aspose.Slides para .NET oferece recursos avançados para criar e modificar programaticamente apresentações do PowerPoint. Este exemplo demonstra como criar um retângulo com formatação de linha personalizada, incluindo a configuração do estilo, largura, padrão de tracejado e cor da linha. O código de exemplo cria uma nova apresentação, adiciona um retângulo ao primeiro slide e aplica formatação específica ao preenchimento e ao contorno da forma, resultando em um elemento visualmente atraente com preenchimento branco e contorno tracejado azul.

## Instruções de instalação

1. Instale o Aspose.Slides para .NET usando um dos seguintes métodos:

   - Gerenciador de pacotes NuGet:
     ```
     Install-Package Aspose.Slides
     ```

   - Console do gerenciador de pacotes:
     ```
     PM> Install-Package Aspose.Slides
     ```

   - CLI .NET:
     ```
     dotnet add package Aspose.Slides
     ```

2. Adicione o namespace necessário ao seu arquivo C#:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

# Criando formas com preenchimento de imagem em mosaico no PowerPoint usando Aspose.Slides

O Aspose.Slides para .NET oferece recursos avançados para criar e manipular apresentações do PowerPoint programaticamente. Este exemplo demonstra como criar um retângulo e preenchê-lo com um padrão de imagem em mosaico. O preenchimento com imagem em mosaico é particularmente útil para criar fundos texturizados, marcas d'água ou elementos decorativos em suas apresentações.

## Instruções de instalação

1. Instalar via Gerenciador de Pacotes NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Instalar via .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Instalação manual:
   - Baixe a versão mais recente em [Aspose.Slides para .NET](https://releases.aspose.com/slides/net/)
   - Adicione referência a Aspose.Slides.dll em seu projeto

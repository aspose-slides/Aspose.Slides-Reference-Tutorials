# Criando formas elípticas no PowerPoint com Aspose.Slides

Aspose.Slides para .NET é uma API poderosa que permite aos desenvolvedores criar, ler, escrever, modificar e converter apresentações do PowerPoint sem a necessidade do Microsoft PowerPoint. Este exemplo demonstra como criar uma forma elíptica em uma apresentação, aplicar formatação, incluindo cor de preenchimento e propriedades de linha, e salvar o resultado como um arquivo PPTX. O código demonstra a capacidade do Aspose.Slides de manipular formas com controle preciso sobre sua aparência e posicionamento.

## Instruções de instalação

1. Instalar via Gerenciador de Pacotes NuGet:
   ```
   Install-Package Aspose.Slides
   ```

   Ou usando o .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

2. Adicionar referências:
   Adicione o seguinte namespace ao seu arquivo C#:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

3. Configuração de licença (opcional):
   Se você tiver uma licença, aplique-a usando:
   ```csharp
   License license = new License();
   license.SetLicense("Aspose.Slides.lic");
   ```

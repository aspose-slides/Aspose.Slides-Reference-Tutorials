# Adicionando segmentos ao GeometryShape em apresentações do PowerPoint

Este exemplo demonstra como acessar e modificar o caminho geométrico de formas em apresentações do PowerPoint usando o Aspose.Slides para .NET. Ao adicionar segmentos de linha personalizados a uma forma retangular, você pode criar figuras geométricas mais complexas que não estão disponíveis nos tipos de forma padrão. Essa técnica é útil para criar diagramas, fluxogramas e outros elementos visuais personalizados em suas apresentações.

## Instruções de instalação

1. Instale o Aspose.Slides para .NET usando um dos seguintes métodos:

   - Gerenciador de pacotes NuGet:
     ```
     Install-Package Aspose.Slides.NET
     ```

   - Console do gerenciador de pacotes:
     ```
     PM> Install-Package Aspose.Slides.NET
     ```

   - CLI .NET:
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. Adicione o namespace ao seu arquivo C#:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

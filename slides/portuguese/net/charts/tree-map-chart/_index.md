## Instruções de instalação:  
Para começar a usar o Aspose.Slides para .NET, siga estas etapas:  

1. Baixe Aspose.Slides para .NET do site oficial [Página de lançamentos do Aspose](https://releases.aspose.com/slides/net/).  
2. Instale o pacote usando o Gerenciador de Pacotes NuGet no Visual Studio:  
   ```shell
   Install-Package Aspose.Slides
   ```  
3. Adicione os namespaces necessários no seu código:  
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Charts;
   ```

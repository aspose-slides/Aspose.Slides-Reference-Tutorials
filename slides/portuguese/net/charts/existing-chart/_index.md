# Modificando gráficos no PowerPoint com Aspose.Slides

O Aspose.Slides para .NET fornece APIs poderosas para criar e modificar apresentações do PowerPoint programaticamente. Este exemplo demonstra como trabalhar com gráficos em apresentações existentes, atualizando os dados do gráfico, adicionando novas séries e alterando os tipos de gráfico — tudo isso sem a necessidade de instalar o Microsoft PowerPoint.

## Instruções de instalação

1. Instalar via Gerenciador de Pacotes NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Ou usando o .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Adicione as seguintes diretivas using:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Charts;
   ```

# Criando gráficos de colunas agrupadas com Aspose.Slides para .NET

O Aspose.Slides para .NET oferece recursos avançados para criar e manipular apresentações do PowerPoint programaticamente. Este exemplo demonstra como criar um gráfico de colunas agrupadas com categorias e séries de dados personalizadas.

## Instalação

```csharp
// Instalar via Gerenciador de Pacotes NuGet
Install-Package Aspose.Slides
```

Ou adicione a seguinte referência ao seu arquivo de projeto:

```xml
<PackageReference Include="Aspose.Slides" Version="23.2.0" />
```

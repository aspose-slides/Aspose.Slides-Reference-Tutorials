## Instruções de instalação
Para começar a usar o Aspose.Slides para .NET, siga estas etapas:

1. Baixe Aspose.Slides para .NET  
   - Baixe a versão mais recente do [Página de download do Aspose.Slides para .NET](https://releases.aspose.com/slides/net/).

2. Instalar via Gerenciador de Pacotes NuGet  
   Execute o seguinte comando no Console do Gerenciador de Pacotes NuGet:
   ```shell
   Install-Package Aspose.Slides
   ```

3. Adicionar referências necessárias  
   No seu projeto do Visual Studio, adicione uma referência ao `Aspose.Slides.dll`.

4. Importar namespaces  
   Adicione os namespaces necessários no início do seu código:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Charts;
   using System.Drawing;
   ```

5. Aplicar uma licença (opcional, mas recomendado)  
   Para evitar limitações de avaliação, você pode [aplicar uma licença](https://purchase.aspose.com/temporary-license/).

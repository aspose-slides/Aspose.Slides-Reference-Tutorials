# Membuat Latar Belakang Gambar Berubin di PowerPoint dengan Aspose.Slides

Aspose.Slides untuk .NET menyediakan kemampuan hebat untuk membuat dan memanipulasi presentasi PowerPoint secara terprogram. Contoh ini menunjukkan cara membuat presentasi dengan bentuk persegi panjang yang diisi dengan pola gambar yang disusun. Kode tersebut menunjukkan cara memuat gambar, menambahkannya ke slide sebagai latar belakang persegi panjang, dan menerapkan berbagai properti penyusunan seperti opsi offset, skala, penyelarasan, dan pembalikan. Teknik ini berguna untuk membuat latar belakang kustom, tanda air, atau elemen dekoratif dalam presentasi Anda dengan kontrol yang tepat atas perilaku penyusunan.

## Petunjuk Instalasi

1. Instal Aspose.Slides untuk .NET menggunakan NuGet Package Manager:
   ```
   Install-Package Aspose.Slides.NET
   ```

2. Atau instal melalui .NET CLI:
   ```
   dotnet add package Aspose.Slides.NET
   ```

3. Tambahkan perintah penggunaan berikut ke file C# Anda:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.IO;
   ```

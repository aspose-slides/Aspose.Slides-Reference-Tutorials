# Membuat Bentuk Elips Isian Gradien di PowerPoint dengan Aspose.Slides

Contoh ini menunjukkan cara membuat presentasi PowerPoint yang berisi bentuk elips dengan efek isian gradien kustom secara terprogram. Kode tersebut membuat presentasi baru, menambahkan elips ke slide pertama, dan menerapkan isian gradien linier dengan hentian warna merah dan ungu. Aspose.Slides untuk .NET menyediakan API komprehensif untuk membuat dan memanipulasi presentasi PowerPoint dengan kontrol yang tepat atas bentuk, pemformatan, dan efek.

## Petunjuk Instalasi

1. Instal Aspose.Slides untuk .NET menggunakan salah satu metode berikut:
   - Manajer Paket NuGet: `Install-Package Aspose.Slides.NET`
   - .NET CLI: `dotnet add package Aspose.Slides.NET`
   - Konsol Manajer Paket: `Install-Package Aspose.Slides.NET`

2. Tambahkan pernyataan using berikut ke file C# Anda:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

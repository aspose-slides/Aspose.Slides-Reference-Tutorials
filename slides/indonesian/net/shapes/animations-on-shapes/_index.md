# Membuat Efek Teks Animasi di PowerPoint dengan Aspose.Slides

Aspose.Slides untuk .NET menyediakan kemampuan hebat untuk membuat efek teks animasi dalam presentasi PowerPoint secara terprogram. Contoh ini menunjukkan cara membuat kotak teks dengan efek animasi, termasuk jalur gerakan kustom yang dipicu oleh klik tombol. Kode tersebut membuat presentasi dengan bentuk persegi panjang yang berisi teks, menerapkan animasi PathFootball, dan menambahkan jalur gerakan kustom yang ditentukan pengguna yang dapat dipicu secara interaktif.

## Petunjuk Instalasi

1. Instal Aspose.Slides untuk .NET menggunakan salah satu metode berikut:

   Manajer Paket NuGet:
   ```csharp
   Install-Package Aspose.Slides
   ```

   Konsol Manajer Paket:
   ```csharp
   PM> Install-Package Aspose.Slides
   ```

   .NET CLI:
   ```csharp
   dotnet add package Aspose.Slides
   ```

2. Tambahkan namespace yang diperlukan ke kode C# Anda:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Animation;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

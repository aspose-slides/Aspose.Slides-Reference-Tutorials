# Menambahkan Gambar ke Slide PowerPoint dengan Aspose.Slides

Aspose.Slides untuk .NET menyediakan kemampuan canggih untuk membuat dan memodifikasi presentasi PowerPoint secara terprogram. Contoh ini menunjukkan cara memuat gambar eksternal dan menambahkannya ke slide presentasi sebagai bingkai gambar - persyaratan umum saat membuat laporan, membuat templat, atau membangun presentasi secara dinamis dari sumber data.

Kode di bawah ini menangani semuanya mulai dari membuat presentasi baru hingga menambahkan gambar dan menyimpan file PPTX final, semuanya hanya dengan beberapa baris kode C#.

## Petunjuk Instalasi

1. Instal Aspose.Slides untuk .NET menggunakan salah satu metode berikut:
   - Manajer Paket NuGet: `Install-Package Aspose.Slides.NET`
   - .NET CLI: `dotnet add package Aspose.Slides.NET`
   - Atau unduh langsung dari [Situs web Aspose](https://releases.aspose.com/slides/net/)

2. Tambahkan namespace ke file C# Anda:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

# Menambahkan Segmen ke GeometryShape dalam Presentasi PowerPoint

Contoh ini menunjukkan cara mengakses dan mengubah jalur geometri bentuk dalam presentasi PowerPoint menggunakan Aspose.Slides for .NET. Dengan menambahkan segmen garis kustom ke bentuk persegi panjang, Anda dapat membuat bentuk geometris yang lebih kompleks yang tidak tersedia melalui jenis bentuk standar. Teknik ini berguna untuk membuat diagram kustom, diagram alur, dan elemen visual lainnya dalam presentasi Anda.

## Petunjuk Instalasi

1. Instal Aspose.Slides untuk .NET menggunakan salah satu metode berikut:

   - Manajer Paket NuGet:
     ```
     Install-Package Aspose.Slides.NET
     ```

   - Konsol Manajer Paket:
     ```
     PM> Install-Package Aspose.Slides.NET
     ```

   - .NET CLI:
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. Tambahkan namespace ke file C# Anda:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

# Membuat Bentuk Elips di PowerPoint dengan Aspose.Slides

Aspose.Slides untuk .NET adalah API canggih yang memungkinkan pengembang membuat, membaca, menulis, memodifikasi, dan mengonversi presentasi PowerPoint tanpa memerlukan Microsoft PowerPoint. Contoh ini menunjukkan cara membuat bentuk elips dalam presentasi, menerapkan pemformatan termasuk warna isian dan properti garis, serta menyimpan hasilnya sebagai file PPTX. Kode tersebut menunjukkan kemampuan Aspose.Slides untuk memanipulasi bentuk dengan kontrol yang tepat atas tampilan dan posisinya.

## Petunjuk Instalasi

1. Instal melalui Manajer Paket NuGet:
   ```
   Install-Package Aspose.Slides
   ```

   Atau menggunakan .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

2. Tambahkan Referensi:
   Tambahkan namespace berikut ke file C# Anda:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

3. Pengaturan Lisensi (Opsional):
   Jika Anda memiliki lisensi, terapkan menggunakan:
   ```csharp
   License license = new License();
   license.SetLicense("Aspose.Slides.lic");
   ```

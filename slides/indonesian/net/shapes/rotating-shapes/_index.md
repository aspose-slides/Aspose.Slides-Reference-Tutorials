# Membuat dan Memutar Bentuk Persegi Panjang di PowerPoint

Aspose.Slides untuk .NET adalah API canggih yang memungkinkan pengembang membuat, memodifikasi, dan mengonversi presentasi PowerPoint tanpa memerlukan instalasi Microsoft PowerPoint. Contoh ini menunjukkan cara membuat presentasi dengan bentuk persegi panjang yang diputar, yang memperlihatkan kemampuan manipulasi bentuk dari pustaka tersebut.

Dengan Aspose.Slides, Anda dapat menambahkan berbagai bentuk ke slide secara terprogram dan menerapkan transformasi seperti rotasi. Fungsionalitas ini sangat berguna untuk membuat presentasi dinamis dengan kontrol yang tepat atas elemen visual.

## Petunjuk Instalasi

1. Instal melalui Manajer Paket NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Instal melalui .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Tambahkan perintah using pada kode Anda:
   ```csharp
   using Aspose.Slides;
   ```

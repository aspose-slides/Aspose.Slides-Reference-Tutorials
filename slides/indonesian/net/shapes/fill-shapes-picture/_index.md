# Membuat Bentuk dengan Isian Gambar Berubin di PowerPoint menggunakan Aspose.Slides

Aspose.Slides untuk .NET menyediakan kemampuan hebat untuk membuat dan memanipulasi presentasi PowerPoint secara terprogram. Contoh ini menunjukkan cara membuat bentuk persegi panjang dan mengisinya dengan pola gambar ubin. Isian gambar ubin sangat berguna untuk membuat latar belakang bertekstur, tanda air, atau elemen dekoratif dalam presentasi Anda.

## Petunjuk Instalasi

1. Instal melalui Manajer Paket NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Instal melalui .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Instalasi Manual:
   - Unduh versi terbaru dari [Aspose.Slides untuk .NET](https://releases.aspose.com/slides/net/)
   - Tambahkan referensi ke Aspose.Slides.dll di proyek Anda

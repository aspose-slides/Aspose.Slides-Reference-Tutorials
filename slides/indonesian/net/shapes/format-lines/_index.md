# Membuat Bentuk Persegi Panjang dengan Pemformatan Garis Kustom di PowerPoint

Aspose.Slides untuk .NET menyediakan kemampuan canggih untuk membuat dan memodifikasi presentasi PowerPoint secara terprogram. Contoh ini menunjukkan cara membuat bentuk persegi panjang dengan format garis yang disesuaikan, termasuk pengaturan gaya garis, lebar, pola garis putus-putus, dan warna. Kode contoh membuat presentasi baru, menambahkan persegi panjang ke slide pertama, dan menerapkan format tertentu pada isian dan garis bentuk, sehingga menghasilkan elemen yang menarik secara visual dengan isian putih dan garis putus-putus biru.

## Petunjuk Instalasi

1. Instal Aspose.Slides untuk .NET menggunakan salah satu metode berikut:

   - Manajer Paket NuGet:
     ```
     Install-Package Aspose.Slides
     ```

   - Konsol Manajer Paket:
     ```
     PM> Install-Package Aspose.Slides
     ```

   - .NET CLI:
     ```
     dotnet add package Aspose.Slides
     ```

2. Tambahkan namespace yang diperlukan ke file C# Anda:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

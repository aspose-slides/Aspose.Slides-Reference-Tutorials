# Menyelaraskan dan Mengelompokkan Bentuk dalam Presentasi PowerPoint

Aspose.Slides untuk .NET menyediakan kemampuan hebat untuk membuat dan memanipulasi presentasi PowerPoint secara terprogram. Contoh ini menunjukkan cara menyelaraskan bentuk dalam slide dan bekerja dengan bentuk yang dikelompokkan dalam presentasi PowerPoint. Anda akan mempelajari cara memosisikan persegi panjang pada koordinat tertentu, menyelaraskannya relatif terhadap slide atau satu sama lain, dan menyelaraskan bentuk secara selektif dalam kelompok.

## Petunjuk Instalasi

1. Instal Aspose.Slides untuk .NET menggunakan salah satu metode berikut:
   
   - Manajer Paket NuGet:
     ```
     Install-Package Aspose.Slides.NET
     ```
   
   - Konsol Manajer Paket:
     ```
     PM> Install-Package Aspose.Slides.NET
     ```
   
   - .NET CLI:
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. Tambahkan Referensi ke proyek Anda:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.IO;
   ```

3. Unduh Contoh Kode di bawah ini dan integrasikan ke dalam proyek Anda.

# Membuat Bentuk Grup di PowerPoint dengan Aspose.Slides

Aspose.Slides untuk .NET menyediakan API yang komprehensif untuk membuat dan memanipulasi presentasi PowerPoint secara terprogram. Contoh ini menunjukkan cara membuat bentuk grup, menambahkan beberapa bentuk persegi panjang di dalamnya, memposisikan bentuk dengan benar, menambahkan bingkai, dan menyimpan presentasi. Bentuk grup sangat penting untuk mengatur bentuk terkait yang perlu dimanipulasi sebagai satu unit, menyediakan lebih banyak struktur dan kontrol atas elemen presentasi Anda.

## Petunjuk Instalasi

1. Instal Aspose.Slides untuk .NET menggunakan salah satu metode berikut:

   - Manajer Paket NuGet:
     ```
     Install-Package Aspose.Slides.NET
     ```

   - Konsol Manajer Paket:
     ```
     PM> Install-Package Aspose.Slides.NET
     ```

   - .NET CLI:
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. Tambahkan referensi ke proyek Anda:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

3. Siapkan lisensi (opsional untuk penggunaan produksi):
   ```csharp
   // Tetapkan lisensi untuk menghapus batasan evaluasi
   License license = new License();
   license.SetLicense("Aspose.Slides.lic");
   ```

# Membuat Poin-Poin Bertingkat di PowerPoint dengan Aspose.Slides

Aspose.Slides untuk .NET menyediakan API yang canggih untuk membuat dan memanipulasi presentasi PowerPoint secara terprogram. Contoh ini menunjukkan cara membuat presentasi dengan poin-poin bertingkat, menunjukkan cara menambahkan bentuk otomatis dengan bingkai teks dan memformat paragraf dengan gaya poin dan tingkat indentasi yang berbeda. Anda dapat dengan mudah membuat konten hierarkis dengan simbol poin yang disesuaikan untuk setiap tingkat indentasi, sempurna untuk membuat presentasi terstruktur dan profesional tanpa memerlukan Microsoft PowerPoint untuk diinstal.

## Petunjuk Instalasi

1. Instal melalui Manajer Paket NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Instal melalui .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Instal melalui Konsol Manajer Paket:
   ```
   PM> Install-Package Aspose.Slides
   ```

4. Instalasi Manual:
   - Unduh versi terbaru dari [Aspose.Slides untuk .NET](https://releases.aspose.com/slides/net/)
   - Tambahkan referensi ke Aspose.Slides.dll di proyek Anda

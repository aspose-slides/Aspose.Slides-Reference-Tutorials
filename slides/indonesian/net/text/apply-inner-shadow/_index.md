# Membuat Presentasi PowerPoint dengan TextBox di Aspose.Slides untuk .NET

Aspose.Slides untuk .NET adalah API canggih yang memungkinkan pengembang membuat, memodifikasi, dan mengonversi presentasi PowerPoint tanpa memerlukan Microsoft PowerPoint. Contoh ini menunjukkan cara membuat presentasi dengan kotak teks yang disesuaikan, menerapkan pemformatan, dan menyimpannya sebagai file PPTX.

## Petunjuk Instalasi

1. Instal melalui Manajer Paket NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Instal melalui .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Instal secara manual:
   - Unduh versi terbaru dari [Halaman rilis Aspose.Slides untuk .NET](https://releases.aspose.com/slides/net/)
   - Tambahkan referensi DLL ke proyek Anda

# Membuat Presentasi PowerPoint dengan Teks Berformat

Aspose.Slides untuk .NET adalah API canggih yang memungkinkan pengembang membuat, memanipulasi, dan mengonversi presentasi PowerPoint tanpa memerlukan Microsoft PowerPoint. Contoh ini menunjukkan cara membuat presentasi sederhana yang berisi bentuk persegi panjang dengan paragraf teks yang diformat.

Kode tersebut membuat presentasi baru, menambahkan bentuk persegi panjang ke slide pertama, lalu menambahkan beberapa paragraf dengan opsi pemformatan teks yang berbeda. Kode tersebut menunjukkan cara mengatur properti font seperti ukuran dan jenis untuk bagian teks tertentu.

## Petunjuk Instalasi

1. Instal melalui Manajer Paket NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Instal melalui .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Instal melalui Konsol Manajer Paket:
   ```
   PM> Install-Package Aspose.Slides
   ```

4. Tambahkan Referensi ke Proyek Anda:
   Setelah instalasi, tambahkan referensi ke proyek Anda:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

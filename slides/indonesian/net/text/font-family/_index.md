# Membuat Kotak Teks Berformat di PowerPoint dengan Aspose.Slides

Aspose.Slides untuk .NET adalah API canggih yang memungkinkan pengembang membuat, memodifikasi, dan mengonversi presentasi Microsoft PowerPoint tanpa memerlukan instalasi Microsoft Office. Contoh ini menunjukkan cara membuat presentasi dengan kotak teks berformat, menerapkan gaya, ukuran, dan warna font khusus untuk meningkatkan daya tarik visual.

## Petunjuk Instalasi

1. Instal melalui Manajer Paket NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Instal melalui .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Tambahkan referensi ke proyek Anda secara manual:
   - Unduh versi terbaru dari [Aspose.Slides untuk .NET](https://releases.aspose.com/slides/net/)
   - Tambahkan DLL sebagai referensi dalam proyek Anda

4. Inisialisasi lisensi:
   ```csharp
   // Inisialisasi lisensi
   License license = new License();
   license.SetLicense("Aspose.Slides.lic");
   ```

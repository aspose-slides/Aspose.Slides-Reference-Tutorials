# Memutar Judul dan Label Bagan di PowerPoint dengan Aspose.Slides

Aspose.Slides untuk .NET menyediakan kemampuan hebat untuk membuat dan memanipulasi presentasi PowerPoint secara terprogram. Contoh ini menunjukkan cara membuat presentasi dengan bagan kolom berkelompok yang menampilkan elemen teks yang diputar - khususnya, memutar judul bagan dan label data untuk meningkatkan keterbacaan dan daya tarik visual.

## Petunjuk Instalasi

1. Instal melalui Manajer Paket NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Atau instal melalui .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Tambahkan namespace ke file C# Anda:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Charts;
   ```

# Membuat Teks Vertikal di PowerPoint Menggunakan Aspose.Slides

Aspose.Slides untuk .NET menyediakan kemampuan hebat untuk membuat dan memanipulasi presentasi PowerPoint secara terprogram. Contoh ini menunjukkan cara membuat presentasi yang berisi bentuk persegi panjang dengan teks yang diputar secara vertikal. Teks diorientasikan pada sudut 270 derajat, sehingga dapat dibaca dari bawah ke atas. Teknik ini berguna untuk membuat label samping, tata letak teks unik, atau elemen desain khusus dalam presentasi Anda.

## Petunjuk Instalasi

1. Instal Aspose.Slides untuk .NET
   
   Instal paket Aspose.Slides menggunakan NuGet Package Manager:
   
   ```csharp
   Install-Package Aspose.Slides
   ```
   
   Atau menggunakan .NET CLI:
   
   ```csharp
   dotnet add package Aspose.Slides
   ```

2. Tambahkan Referensi
   
   Tambahkan pernyataan using berikut ke file C# Anda:
   
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

# Membuat Poin-Poin Khusus dalam Presentasi PowerPoint

Aspose.Slides untuk .NET menyediakan kemampuan yang tangguh untuk membuat dan memanipulasi presentasi PowerPoint secara terprogram. Contoh ini secara khusus berfokus pada penerapan poin-poin khusus dalam bingkai teks, yang menunjukkan cara mengontrol jenis poin, warna, indentasi, dan ukuran. Kode tersebut membuat presentasi dengan bentuk persegi panjang yang berisi dua paragraf - satu dengan poin simbol dan satu lagi dengan gaya poin bernomor.

## Petunjuk Instalasi

1. Instal Aspose.Slides untuk .NET menggunakan NuGet Package Manager:
   ```
   Install-Package Aspose.Slides
   ```

2. Atau, unduh paketnya secara langsung:
   - Kunjungi [Halaman Unduhan Aspose.Slides](https://downloads.aspose.com/slides/net)
   - Unduh versi terbaru
   - Ekstrak dan referensi Aspose.Slides.dll di proyek Anda

3. Tambahkan pernyataan penggunaan yang diperlukan:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

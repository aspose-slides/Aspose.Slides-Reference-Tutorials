# Membuat Tabel Standar di PowerPoint dengan Aspose.Slides

Aspose.Slides untuk .NET adalah API canggih yang memungkinkan pengembang membuat, membaca, menulis, memodifikasi, dan mengonversi presentasi PowerPoint secara terprogram tanpa memerlukan Microsoft PowerPoint. Contoh ini menunjukkan cara membuat tabel standar dengan batas berformat khusus dalam presentasi PowerPoint.

## Petunjuk Instalasi

1. Instal melalui Manajer Paket NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Atau melalui .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Tambahkan namespace berikut ke file C# Anda:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Tables;
   using System.Drawing;
   ```

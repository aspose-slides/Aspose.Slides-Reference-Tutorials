## Petunjuk Instalasi

1. Instal Aspose.Slides untuk .NET menggunakan salah satu metode berikut:
   - Manajer Paket NuGet: `Install-Package Aspose.Slides.NET`
   - .NET CLI: `dotnet add package Aspose.Slides.NET`
   - Tambahkan referensi ke Aspose.Slides.dll langsung di proyek Anda

2. Tambahkan referensi namespace di file C# Anda:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

3. Unduh dokumentasi dan referensi API:
   - [Dokumentasi Aspose.Slides untuk .NET](https://docs.aspose.com/slides/net/)
   - [Referensi API Aspose.Slides untuk .NET](https://reference.aspose.com/slides/net/)

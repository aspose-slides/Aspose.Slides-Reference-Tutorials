# Membuat Tabel dengan Teks Vertikal dalam Presentasi PowerPoint

Aspose.Slides untuk .NET menyediakan kemampuan canggih untuk membuat dan memanipulasi presentasi PowerPoint secara terprogram. Contoh ini menunjukkan cara membuat tabel dengan format yang disesuaikan, termasuk perataan dan rotasi teks vertikal - persyaratan umum untuk presentasi bisnis khusus dan visualisasi data.

## Petunjuk Instalasi

1. Instal Aspose.Slides untuk .NET melalui Manajer Paket NuGet:
   ```
   Install-Package Aspose.Slides
   ```
   Atau melalui .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

2. Tambahkan referensi ke proyek Anda:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

3. Unduh sumber dokumentasi:
   - [Referensi API](https://reference.aspose.com/slides/net/)
   - [Panduan Pengembang](https://docs.aspose.com/slides/net/)

# Membuat Bagan Kolom Berkelompok dengan Aspose.Slides untuk .NET

Aspose.Slides untuk .NET menyediakan kemampuan canggih untuk membuat dan memanipulasi presentasi PowerPoint secara terprogram. Contoh ini menunjukkan cara membuat bagan kolom berkelompok dengan kategori dan seri data khusus.

## Instalasi

```csharp
// Instal melalui Pengelola Paket NuGet
Install-Package Aspose.Slides
```

Atau tambahkan referensi berikut ke berkas proyek Anda:

```xml
<PackageReference Include="Aspose.Slides" Version="23.2.0" />
```

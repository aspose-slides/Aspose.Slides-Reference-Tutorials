# Memodifikasi Bagan di PowerPoint dengan Aspose.Slides

Aspose.Slides untuk .NET menyediakan API yang canggih untuk membuat dan memodifikasi presentasi PowerPoint secara terprogram. Contoh ini menunjukkan cara bekerja dengan bagan dalam presentasi yang ada dengan memperbarui data bagan, menambahkan seri baru, dan mengubah jenis bagan - semuanya tanpa mengharuskan Microsoft PowerPoint diinstal.

## Petunjuk Instalasi

1. Instal melalui Manajer Paket NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Atau menggunakan .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Tambahkan perintah menggunakan berikut ini:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Charts;
   ```

## Οδηγός Εφαρμογής

Αυτή η ενότητα περιγράφει λεπτομερώς τον τρόπο εφαρμογής εφέ περιστροφής 3D.

### Εφαρμογή τρισδιάστατης περιστροφής σε ορθογώνιο σχήμα

#### Επισκόπηση

Προσθέστε βάθος και προοπτική σε ορθογώνια σχήματα χρησιμοποιώντας τρισδιάστατες περιστροφές.

#### Βήμα προς βήμα εφαρμογή

**1. Προσθέστε ένα ορθογώνιο σχήμα:**

```python
auto_shape = pres.slides[0].shapes.add_auto_shape(
    slides.ShapeType.RECTANGLE, 30, 30, 200, 200)
```

*Εξήγηση*Αυτός ο κώδικας προσθέτει ένα ορθογώνιο στη θέση (30, 30) με διαστάσεις 200x200.

**2. Εφαρμογή περιστροφής 3D:**

```python
auto_shape.three_d_format.depth = 6
auto_shape.three_d_format.camera.set_rotation(40, 35, 20)
auto_shape.three_d_format.camera.camera_type = slides.CameraPresetType.ISOMETRIC_LEFT_UP
auto_shape.three_d_format.light_rig.light_type = slides.LightRigPresetType.BALANCED
```

*Εξήγηση*: 
- `depth`: Ορίζει το βάθος του εφέ 3D.
- `camera.set_rotation()`: Ρυθμίζει τις γωνίες περιστροφής για τους άξονες X, Y και Z.
- `camera_type`: Ορίζει την προοπτική της κάμερας.
- `light_rig.light_type`: Προσαρμόζει τον φωτισμό για να βελτιώσει την τρισδιάστατη εμφάνιση.

**3. Αποθηκεύστε την παρουσίασή σας:**

```python
pres.save("shapes_apply_3d_rotation_to_rectangle_out.pptx", slides.export.SaveFormat.PPTX)
```

### Εφαρμογή περιστροφής 3D σε σχήμα γραμμής

#### Επισκόπηση

Δημιουργήστε ενδιαφέροντα οπτικά στοιχεία προσθέτοντας τρισδιάστατα εφέ σε σχήματα γραμμών.

#### Βήμα προς βήμα εφαρμογή

**1. Προσθέστε ένα σχήμα γραμμής:**

```python
auto_shape = pres.slides[0].shapes.add_auto_shape(
    slides.ShapeType.LINE, 30, 300, 200, 200)
```

*Εξήγηση*Αυτός ο κώδικας προσθέτει μια γραμμή στη θέση (30, 300) με διαστάσεις 200x200.

**2. Εφαρμογή περιστροφής 3D:**

```python
auto_shape.three_d_format.depth = 6
auto_shape.three_d_format.camera.set_rotation(0, 35, 20)
auto_shape.three_d_format.camera.camera_type = slides.CameraPresetType.ISOMETRIC_LEFT_UP
auto_shape.three_d_format.light_rig.light_type = slides.LightRigPresetType.BALANCED
```

*Εξήγηση*Παρόμοιο με το ορθογώνιο σχήμα, αλλά με διαφορετικές γωνίες περιστροφής για μοναδικά εφέ.

**3. Αποθηκεύστε την παρουσίασή σας:**

```python
pres.save("shapes_apply_3d_rotation_to_line_out.pptx", slides.export.SaveFormat.PPTX)
```

#### Ρύθμιση εφέ μετάβασης
Θα ορίσουμε ένα εφέ μετάβασης στην πρώτη διαφάνεια της παρουσίασής σας:

```python
# Πρόσβαση στην πρώτη διαφάνεια
slide = presentation.slides[0]

# Ορίστε τον τύπο του εφέ μετάβασης
slide.slide_show_transition.type = slides.slideshow.TransitionType.CUT

# Πρόσθετες ιδιότητες μετάβασης (π.χ., από μαύρο)
slide.slide_show_transition.value.from_black = True
```

#### Εξήγηση:
- **Τύπος μετάβασης**: Αυτό ορίζει τον συγκεκριμένο τύπο κινούμενης εικόνας κατά τη μετακίνηση μεταξύ διαφανειών. `CUT` σημαίνει άμεση αλλαγή.
- **Από το Μαύρο**: Μια ειδική ιδιότητα για την έναρξη της διαφάνειας με μαύρη οθόνη.

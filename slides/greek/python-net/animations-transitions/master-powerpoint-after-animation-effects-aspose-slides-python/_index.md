## Οδηγός Εφαρμογής
Θα εξερευνήσουμε τρία βασικά χαρακτηριστικά: την απόκρυψη στοιχείων στο επόμενο κλικ του ποντικιού, τη ρύθμιση χρωμάτων και την απόκρυψη κινούμενων εικόνων μετά την κίνηση.

### Αλλαγή του τύπου εφέ κίνησης μετά την επιλογή "Απόκρυψη στο επόμενο κλικ του ποντικιού"

#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να αποκρύψετε στοιχεία μετά από μια συγκεκριμένη αλληλεπίδραση χρήστη, βελτιώνοντας την διαδραστικότητα των διαφανειών.

#### Βήματα Υλοποίησης

##### Φόρτωση παρουσίασης και προσθήκη διαφάνειας
Αρχικά, ανοίξτε το αρχείο παρουσίασής σας και κλωνοποιήστε μια υπάρχουσα διαφάνεια:
```python
import aspose.slides as slides

with slides.Presentation("YOUR_DOCUMENT_DIRECTORY/AnimationAfterEffect.pptx") as pres:
    # Κλωνοποιήστε την πρώτη διαφάνεια για να δημιουργήσετε μια νέα με παρόμοιο περιεχόμενο
    slide1 = pres.slides.add_clone(pres.slides[0])
```

##### Τροποποίηση τύπου εφέ κίνησης μετά
Αλλάξτε το εφέ κίνησης μετά την ακολουθία για κάθε στοιχείο στην ακολουθία σας:
```python
# Λήψη της κύριας ακολουθίας κινήσεων για τη νέα διαφάνεια που προστέθηκε
seq = slide1.timeline.main_sequence

# Ορίστε τον τύπο εφέ σε "Απόκρυψη στο επόμενο κλικ του ποντικιού"
for effect in seq:
    effect.after_animation_type = slides.animation.AfterAnimationType.HIDE_ON_NEXT_MOUSE_CLICK

pres.save("YOUR_OUTPUT_DIRECTORY/AnimationAfterEffect-out.pptx", slides.export.SaveFormat.PPTX)
```

**Εξήγηση:** Αυτός ο κώδικας επαναλαμβάνει όλα τα εφέ κίνησης και τα ορίζει να αποκρύπτονται στο επόμενο κλικ του ποντικιού, δημιουργώντας μια διαδραστική εμπειρία για τους χρήστες.

### Αλλαγή τύπου εφέ κίνησης μετά σε χρώμα

#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να τροποποιείτε τα εφέ των κινούμενων εικόνων αλλάζοντας τα χρώματά τους, προσθέτοντας οπτική πινελιά στην παρουσίασή σας.

#### Βήματα Υλοποίησης

##### Τροποποίηση τύπου εφέ κίνησης μετά το χρώμα
Όπως και με την απόκρυψη εφέ, ορίστε τον τύπο εφέ και καθορίστε ένα χρώμα:
```python
import aspose.slides as slides
import aspose.pydrawing as drawing

with slides.Presentation("YOUR_DOCUMENT_DIRECTORY/AnimationAfterEffect.pptx") as pres:
    # Κλωνοποίηση υπάρχουσας διαφάνειας για τροποποίηση
    slide2 = pres.slides.add_clone(pres.slides[0])
    
    # Πρόσβαση στην κύρια ακολουθία κινουμένων σχεδίων
    seq = slide2.timeline.main_sequence
    
    # Αλλάξτε τον τύπο εφέ σε "Χρώμα" και ορίστε τον σε πράσινο
    for effect in seq:
        effect.after_animation_type = slides.animation.AfterAnimationType.COLOR
        effect.after_animation_color.color = drawing.Color.green

pres.save("YOUR_OUTPUT_DIRECTORY/AnimationAfterEffect-out.pptx", slides.export.SaveFormat.PPTX)
```

**Εξήγηση:** Αυτό το απόσπασμα προσαρμόζει τον τύπο κίνησης μετά την εμφάνιση σε "Χρώμα" και τον ορίζει σε πράσινο, βελτιώνοντας την οπτική ελκυστικότητα.

### Αλλαγή τύπου εφέ κίνησης μετά την επιλογή "Απόκρυψη κίνησης μετά την επιλογή"

#### Επισκόπηση
Αυτόματη απόκρυψη στοιχείων μετά την κίνηση για πιο καθαρή εμφάνιση όταν ολοκληρωθούν οι μεταβάσεις.

#### Βήματα Υλοποίησης

##### Τροποποίηση τύπου εφέ κίνησης μετά
Ρυθμίστε τις κινούμενες εικόνες ώστε να αποκρύπτονται αυτόματα μετά την αναπαραγωγή τους:
```python
import aspose.slides as slides

with slides.Presentation("YOUR_DOCUMENT_DIRECTORY/AnimationAfterEffect.pptx") as pres:
    # Κλωνοποίηση της πρώτης διαφάνειας για εργασία σε μια νέα
    slide3 = pres.slides.add_clone(pres.slides[0])
    
    # Πρόσβαση στην ακολουθία κινουμένων σχεδίων
    seq = slide3.timeline.main_sequence
    
    # Ορίστε τον τύπο εφέ σε "Απόκρυψη μετά την κίνηση"
    for effect in seq:
        effect.after_animation_type = slides.animation.AfterAnimationType.HIDE_AFTER_ANIMATION

pres.save("YOUR_OUTPUT_DIRECTORY/AnimationAfterEffect-out.pptx", slides.export.SaveFormat.PPTX)
```

**Εξήγηση:** Αυτός ο κώδικας διασφαλίζει ότι τα στοιχεία κρύβονται αυτόματα μετά τις κινούμενες εικόνες τους, παρέχοντας μια απρόσκοπτη μετάβαση μεταξύ των διαφανειών.

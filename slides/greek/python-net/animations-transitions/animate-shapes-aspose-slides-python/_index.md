## Οδηγός Εφαρμογής

### Χαρακτηριστικό 1: Δημιουργία σχημάτων σε παρουσίαση

#### Επισκόπηση
Αυτή η ενότητα δείχνει πώς να προσθέσετε σχήματα, συγκεκριμένα ορθογώνια, σε μια διαφάνεια χρησιμοποιώντας το Aspose.Slides για Python. Αυτό το βήμα είναι θεμελιώδες για την προσαρμογή των διαφανειών με συγκεκριμένα στοιχεία σχεδίασης.

##### Βήμα προς βήμα εφαρμογή
**Προσθήκη ορθογωνίων σχημάτων**
Ξεκινήστε δημιουργώντας μια συνάρτηση για να προσθέσετε ορθογώνια σχήματα:
```python
def create_shapes():
    with slides.Presentation() as pres:
        # Προσθήκη δύο ορθογωνίων σχημάτων στην πρώτη διαφάνεια
        shp1 = pres.slides[0].shapes.add_auto_shape(
            slides.ShapeType.RECTANGLE, 0, 0, 50, 50)
        
        shp2 = pres.slides[0].shapes.add_auto_shape(
            slides.ShapeType.RECTANGLE, 100, 0, 50, 50)
```
**Επεξήγηση παραμέτρων:**
- `slides.ShapeType.RECTANGLE`: Καθορίζει τον τύπο σχήματος.
- Συντεταγμένες `(x, y)` και διαστάσεις `(width, height)`: Ορισμός θέσης και μεγέθους.

### Χαρακτηριστικό 2: Προσθήκη εφέ ξεθωριασμένου ζουμ σε σχήματα

#### Επισκόπηση
Εφαρμόστε ένα δυναμικό εφέ Faded Zoom στα σχήματα στις διαφάνειές σας. Αυτό βελτιώνει την οπτική ελκυστικότητα και την αλληλεπίδραση κατά τη διάρκεια των παρουσιάσεων.

##### Βήμα προς βήμα εφαρμογή
**Εφαρμογή εφέ ξεθωριασμένου ζουμ**
Δημιουργήστε μια συνάρτηση για να εφαρμόσετε αυτά τα εφέ:
```python
def apply_faded_zoom_effect():
    with slides.Presentation() as pres:
        # Δημιουργήστε δύο ορθογώνια σχήματα για την εφαρμογή εφέ
        shp1 = pres.slides[0].shapes.add_auto_shape(
            slides.ShapeType.RECTANGLE, 0, 0, 50, 50)
        
        shp2 = pres.slides[0].shapes.add_auto_shape(
            slides.ShapeType.RECTANGLE, 100, 0, 50, 50)

        # Εφαρμογή εφέ Faded Zoom στο πρώτο σχήμα με υποτύπο κέντρου αντικειμένου
        ef1 = pres.slides[0].timeline.main_sequence.add_effect(
            shp1, slides.animation.EffectType.FADED_ZOOM,
            slides.animation.EffectSubtype.OBJECT_CENTER,
            slides.animation.EffectTriggerType.ON_CLICK)

        # Εφαρμογή εφέ ξεθωριασμένου ζουμ στο δεύτερο σχήμα με υποτύπο κέντρου διαφάνειας
        ef2 = pres.slides[0].timeline.main_sequence.add_effect(
            shp2, slides.animation.EffectType.FADED_ZOOM,
            slides.animation.EffectSubtype.SLIDE_CENTER,
            slides.animation.EffectTriggerType.ON_CLICK)
```
**Βασικές επιλογές διαμόρφωσης:**
- `EffectSubtype`: Επιλέξτε μεταξύ OBJECT_CENTER και SLIDE_CENTER.
- `EffectTriggerType`: Ορίστε σε ON_CLICK για διαδραστικές παρουσιάσεις.

### Λειτουργία 3: Αποθήκευση παρουσίασης στον κατάλογο εξόδου

#### Επισκόπηση
Βεβαιωθείτε ότι η παρουσίασή σας με όλα τα πρόσθετα εφέ έχει αποθηκευτεί σωστά. Αυτό το βήμα ολοκληρώνει την εργασία σας, επιτρέποντάς σας να την κοινοποιήσετε ή να την παρουσιάσετε αλλού.

##### Βήμα προς βήμα εφαρμογή
**Αποθήκευση της εργασίας σας**
Υλοποιήστε μια συνάρτηση για να αποθηκεύσετε την παρουσίασή σας:
```python
def save_presentation():
    with slides.Presentation() as pres:
        # Δημιουργήστε δύο ορθογώνια σχήματα για επίδειξη
        shp1 = pres.slides[0].shapes.add_auto_shape(
            slides.ShapeType.RECTANGLE, 0, 0, 50, 50)
        
        shp2 = pres.slides[0].shapes.add_auto_shape(
            slides.ShapeType.RECTANGLE, 100, 0, 50, 50)

        # Προσθήκη εφέ ξεθωριασμένου ζουμ σε σχήματα
        ef1 = pres.slides[0].timeline.main_sequence.add_effect(
            shp1, slides.animation.EffectType.FADED_ZOOM,
            slides.animation.EffectSubtype.OBJECT_CENTER,
            slides.animation.EffectTriggerType.ON_CLICK)
        
        ef2 = pres.slides[0].timeline.main_sequence.add_effect(
            shp2, slides.animation.EffectType.FADED_ZOOM,
            slides.animation.EffectSubtype.SLIDE_CENTER,
            slides.animation.EffectTriggerType.ON_CLICK)

        # Αποθηκεύστε την παρουσίαση στον κατάλογο 'YOUR_OUTPUT_DIRECTORY/'
        pres.save('YOUR_OUTPUT_DIRECTORY/AnimatedPresentation.pptx',
                  slides.export.SaveFormat.PPTX)
```
**Συμβουλές αντιμετώπισης προβλημάτων:**
- Εξασφαλίζω `YOUR_OUTPUT_DIRECTORY` υπάρχει και είναι εγγράψιμο.
- Ελέγξτε τα δικαιώματα αρχείων εάν αντιμετωπίσετε σφάλματα κατά την αποθήκευση.

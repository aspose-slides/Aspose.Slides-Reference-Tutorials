## Οδηγός Εφαρμογής

Ακολουθήστε αυτά τα βήματα για να δημιουργήσετε κίνηση σε σειρές γραφημάτων.

### Φόρτωση της παρουσίασης

Φόρτωση μιας υπάρχουσας παρουσίασης PowerPoint που περιέχει ένα γράφημα.

#### Βήμα 1: Φόρτωση παρουσίασης

```python
def animate_chart_series():
    with slides.Presentation(DOCUMENT_DIRECTORY + "charts_existing_chart.pptx") as presentation:
        slide = presentation.slides[0]
```

Αποκτήστε πρόσβαση στην πρώτη διαφάνεια και αντικαταστήστε `"YOUR_DOCUMENT_DIRECTORY/"` με την πραγματική σου πορεία.

### Πρόσβαση στο Διάγραμμα

#### Βήμα 2: Προσδιορίστε το σχήμα του γραφήματος

```python
shapes = slide.shapes
chart = shapes[0]  # Υποθέτοντας ότι το πρώτο σχήμα είναι ένα διάγραμμα
```

Αποκτήστε πρόσβαση σε όλα τα σχήματα στη διαφάνεια και υποθέστε ότι το πρώτο είναι το γράφημά μας. Προσαρμόστε τα εάν είναι απαραίτητο.

### Προσθήκη εφέ κίνησης

#### Βήμα 3: Εφαρμογή κινούμενης εικόνας

```python
main_sequence = slide.timeline.main_sequence
main_sequence.add_effect(
    chart, slides.animation.EffectType.FADE,
    slides.animation.EffectSubtype.NONE,
    slides.animation.EffectTriggerType.AFTER_PREVIOUS
)

for i in range(4):
    main_sequence.add_effect(
        chart, 
        slides.animation.EffectChartMajorGroupingType.BY_SERIES,
        i,  # Ευρετήριο σειράς
        slides.animation.EffectType.APPEAR,
        slides.animation.EffectSubtype.NONE,
        slides.animation.EffectTriggerType.AFTER_PREVIOUS
    )
```

Εφαρμόστε ένα εφέ fade στο γράφημα και δώστε κίνηση σε κάθε σειρά ξεχωριστά με `EffectChartMajorGroupingType.BY_SERIES`.

### Αποθήκευση της παρουσίασης

#### Βήμα 4: Αποθήκευση αλλαγών

```python
with slides.Presentation(DOCUMENT_DIRECTORY + "charts_existing_chart.pptx") as presentation:
    presentation.save(OUTPUT_DIRECTORY + "charts_animating_series_out.pptx", slides.export.SaveFormat.PPTX)
```

Αποθηκεύστε τις αλλαγές σας σε ένα νέο αρχείο. Αντικατάσταση `"YOUR_OUTPUT_DIRECTORY/"` με την επιθυμητή θέση εξόδου.

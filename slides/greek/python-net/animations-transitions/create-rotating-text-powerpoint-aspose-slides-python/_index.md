## Οδηγός Εφαρμογής
Σε αυτήν την ενότητα, θα αναλύσουμε κάθε χαρακτηριστικό της περιστροφής κειμένου σε μια διαφάνεια του PowerPoint.

### Προσθήκη σχημάτων σε διαφάνειες
Αρχικά, ας προσθέσουμε ένα ορθογώνιο σχήμα που θα περιέχει το περιστρεφόμενο κείμενό μας. Αυτό το σχήμα λειτουργεί ως δοχείο για κείμενο και μπορεί να προσαρμοστεί εκτενώς.

#### Οδηγός βήμα προς βήμα:
1. **Δημιουργήστε μια παρουσία παρουσίασης:**

   ```python
   with slides.Presentation() as presentation:
       slide = presentation.slides[0]
   ```
2. **Προσθήκη ορθογωνίου σχήματος:**

   Εδώ, προσθέτουμε ένα ορθογώνιο στην πρώτη διαφάνεια. Οι παράμετροι καθορίζουν τη θέση και το μέγεθός του.

   ```python
   auto_shape = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 150, 75, 350, 350)
   ```
### Περιστροφή κειμένου στο σχήμα
Τώρα που το σχήμα μας είναι έτοιμο, ας επικεντρωθούμε στην περιστροφή του κειμένου κάθετα μέσα σε αυτό.
1. **Δημιουργία και ρύθμιση παραμέτρων ενός TextFrame:**

   ```python
   text_frame = auto_shape.add_text_frame(" ")
   auto_shape.fill_format.fill_type = slides.FillType.NO_FILL
   ```
2. **Ορισμός κατακόρυφου προσανατολισμού:**

   Αυτό το βήμα περιλαμβάνει τη ρύθμιση του κατακόρυφου προσανατολισμού του πλαισίου κειμένου σε 270 μοίρες, η οποία το περιστρέφει κατακόρυφα.

   ```python
   text_frame.text_frame_format.text_vertical_type = slides.TextVerticalType.VERTICAL270
   ```
3. **Προσθήκη περιεχομένου κειμένου:**

   Αντιστοιχίστε κείμενο στην παράγραφο σας και προσαρμόστε την εμφάνισή του.

   ```python
   para = text_frame.paragraphs[0]
   portion = para.portions[0]
   portion.text = "A quick brown fox jumps over the lazy dog. A quick brown fox jumps over the lazy dog."
   
   # Ορίστε τον τύπο γεμίσματος για κείμενο σε συμπαγές και χρωματίστε το μαύρο
   portion.portion_format.fill_format.fill_type = slides.FillType.SOLID
   portion.portion_format.fill_format.solid_fill_color.color = drawing.Color.black
   ```
4. **Αποθήκευση της παρουσίασής σας:**

   Τέλος, αποθηκεύστε την παρουσίαση με τις τροποποιήσεις σας.

   ```python
   presentation.save("YOUR_OUTPUT_DIRECTORY/text_rotate_out.pptx", slides.export.SaveFormat.PPTX)
   ```

## Ενότητα Συχνών Ερωτήσεων
**Ε: Πώς μπορώ να περιστρέψω κείμενο οριζόντια;**
Α: Σετ `text_vertical_type` να `TEXT_VERTICAL_TYPE.HORIZONTAL`.

**Ε: Μπορώ να αλλάξω το μέγεθος και το στυλ της γραμματοσειράς;**
Α: Ναι, τροποποίηση `portion.portion_format` για τις ιδιότητες γραμματοσειράς.

**Ε: Τι γίνεται αν η παρουσίασή μου δεν αποθηκεύεται σωστά;**
Α: Βεβαιωθείτε ότι έχετε δικαιώματα εγγραφής στον κατάλογο εξόδου σας.

**Ε: Πώς μπορώ να προσθέσω πολλές παραγράφους περιστρεφόμενου κειμένου;**
Α: Δημιουργήστε επιπλέον παραγράφους χρησιμοποιώντας `text_frame.paragraphs.add_empty_paragraph()`.

**Ε: Υπάρχουν περιορισμοί στο μέγεθος του πλαισίου κειμένου;**
Α: Τα μεγάλα σχήματα ενδέχεται να επηρεάσουν την απόδοση, επομένως βελτιστοποιήστε το μέγεθος ανάλογα με τις ανάγκες.

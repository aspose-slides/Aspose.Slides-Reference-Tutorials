## Οδηγός Εφαρμογής

Θα αναλύσουμε τη διαδικασία σε ενότητες με βάση κάθε χαρακτηριστικό.

### Χαρακτηριστικό 1: Φόρτωση και Επανάληψη Παρουσίασης

#### Επισκόπηση:
Αυτή η λειτουργία σάς επιτρέπει να φορτώσετε ένα αρχείο παρουσίασης PowerPoint και να επαναλάβετε τις διαφάνειές του, κάτι που είναι χρήσιμο για την αυτοματοποίηση της επεξεργασίας διαφανειών ή την εξαγωγή συγκεκριμένων δεδομένων.

#### Βήμα προς βήμα εφαρμογή:
**Βήμα 1: Ορισμός της συνάρτησης**
Ορισμός συνάρτησης `load_presentation` που λαμβάνει τη διαδρομή προς το αρχείο παρουσίασής σας ως όρισμα.

```python
def load_presentation(presentation_path):
    with slides.Presentation(presentation_path) as pres:
        for slide in pres.slides:
            print(f"Slide #Το {slide.slide_number} έχει φορτωθεί.")
```
**Εξήγηση:**
- `slides.Presentation(presentation_path)` ανοίγει το αρχείο PowerPoint σας.
- Ο διαχειριστής περιβάλλοντος διασφαλίζει ότι η παρουσίαση κλείνει σωστά μετά την επεξεργασία.

**Βήμα 2: Παράδειγμα Χρήσης**
Αντικαθιστώ `'YOUR_DOCUMENT_DIRECTORY/'` με την πραγματική διαδρομή καταλόγου όπου είναι αποθηκευμένο το έγγραφό σας:

```python
load_presentation('YOUR_DOCUMENT_DIRECTORY/shapes_animation_example.pptx')
```

### Χαρακτηριστικό 2: Εξαγωγή εφέ κίνησης από διαφάνειες

#### Επισκόπηση:
Εξαγάγετε και εκτυπώστε λεπτομέρειες σχετικά με τα εφέ κίνησης που εφαρμόζονται σε σχήματα σε κάθε διαφάνεια. Αυτό βοηθά στην ανάλυση των ρυθμίσεων κίνησης στις παρουσιάσεις σας.

#### Βήμα προς βήμα εφαρμογή:
**Βήμα 1: Ορισμός της συνάρτησης**
Δημιουργήστε μια συνάρτηση `extract_animation_effects` που φορτώνει την παρουσίαση και επαναλαμβάνει τις κινούμενες εικόνες της.

```python
def extract_animation_effects(presentation_path):
    with slides.Presentation(presentation_path) as pres:
        for slide in pres.slides:
            for effect in slide.timeline.main_sequence:
                print(f"{effect.type} animation effect is set to shape#{effect.target_shape.unique_id} στη διαφάνεια #{slide.slide_number}")
```
**Εξήγηση:**
- `slide.timeline.main_sequence` παρέχει πρόσβαση σε όλες τις κινούμενες εικόνες που εφαρμόζονται σε μια διαφάνεια.
- Κάθε `effect` Το αντικείμενο περιέχει λεπτομέρειες σχετικά με τον τύπο της κινούμενης εικόνας και το σχήμα-στόχο της.

**Βήμα 2: Παράδειγμα Χρήσης**
Χρησιμοποιήστε τη συνάρτηση με τη διαδρομή παρουσίασής σας:

```python
extract_animation_effects('YOUR_DOCUMENT_DIRECTORY/shapes_animation_example.pptx')
```

## Οδηγός Εφαρμογής
### Μετατροπή παρουσίασης σε TIFF με σημειώσεις
Αυτή η λειτουργία σάς επιτρέπει να εξάγετε παρουσιάσεις PowerPoint σε μορφή TIFF, διασφαλίζοντας ότι οι σημειώσεις περιλαμβάνονται στο κάτω μέρος κάθε διαφάνειας.

#### Επισκόπηση
Η διαδικασία περιλαμβάνει τη ρύθμιση συγκεκριμένων επιλογών για την απόδοση των διαφανειών ως αρχεία TIFF και τη διαμόρφωση του τρόπου εμφάνισης των σημειώσεων.

#### Βήμα προς βήμα εφαρμογή
**1. Εισαγωγή Aspose.Slides**
Ξεκινήστε εισάγοντας την απαραίτητη ενότητα:

```python
import aspose.slides as slides
```

**2. Ρύθμιση επιλογών εξαγωγής**
Διαμορφώστε το `TiffOptions` για να συμπεριλάβετε ρυθμίσεις διάταξης για σημειώσεις διαφανειών:

```python
# Δημιουργία αντικειμένου TiffOptions
 tiff_options = slides.export.TiffOptions()

# Ρύθμιση παραμέτρων επιλογών διάταξης σημειώσεων
slides_layout_options = slides.export.NotesCommentsLayoutingOptions()
slides_layout_options.notes_position = slides.export.NotesPositions.BOTTOM_FULL

# Αντιστοιχίστε αυτές τις επιλογές διάταξης σε επιλογές TIFF
tiff_options.slides_layout_options = slides_layout_options
```

**3. Φόρτωση και μετατροπή της παρουσίασης**
Φορτώστε το αρχείο PowerPoint και μετατρέψτε το σε εικόνα TIFF χρησιμοποιώντας τις διαμορφωμένες επιλογές:

```python
with slides.Presentation('YOUR_DOCUMENT_DIRECTORY/presentation_with_notes.pptx') as pres:
    # Αποθηκεύστε την παρουσίαση σε μορφή TIFF με σημειώσεις στο κάτω μέρος
    pres.save('YOUR_OUTPUT_DIRECTORY/convert_to_tiff_with_notes_out.tiff',
              slides.export.SaveFormat.TIFF, tiff_options)
```

**Εξήγηση**
- `tiff_options`: Ρυθμίζει τον τρόπο με τον οποίο κάθε διαφάνεια αποδίδεται σε εικόνα TIFF.
- `slides_layout_options.notes_position`: Εξασφαλίζει ότι οι σημειώσεις τοποθετούνται πλήρως στο κάτω μέρος κάθε διαφάνειας.

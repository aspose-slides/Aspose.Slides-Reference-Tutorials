## Οδηγός Εφαρμογής

### Μετατροπή παρουσίασης σε φυλλάδια

Ακολουθήστε αυτά τα βήματα για να μετατρέψετε παρουσιάσεις PowerPoint σε PDF με φυλλάδια.

#### Φόρτωση της παρουσίασής σας

Αρχικά, φορτώστε την παρουσίαση που επιθυμείτε χρησιμοποιώντας το `Presentation` τάξη:
```python
import aspose.slides as slides

DOCUMENT_PATH = "YOUR_DOCUMENT_DIRECTORY/HandoutExample.pptx"
OUTPUT_PATH = "YOUR_OUTPUT_DIRECTORY/HandoutExample.pdf"

def convert_to_handout():
    # Φόρτωση παρουσίασης από καθορισμένη διαδρομή
    with slides.Presentation(DOCUMENT_PATH) as pres:
        pass  # Θα ακολουθήσουν επιπλέον βήματα εδώ
```

#### Ρύθμιση παραμέτρων επιλογών εξαγωγής PDF

Ρυθμίστε τις επιλογές για τον έλεγχο της εξαγωγής των σημειώσεων ακροατηρίου σας, συμπεριλαμβανομένης της εμφάνισης κρυφών διαφανειών και της επιλογής διάταξης:
```python
        # Ρύθμιση παραμέτρων επιλογών εξαγωγής PDF
        pdf_options = slides.export.PdfOptions()
        
        # Επιλογή εμφάνισης κρυφών διαφανειών στην έξοδο
        pdf_options.show_hidden_slides = True
        
        # Ρύθμιση επιλογών διάταξης φυλλαδίου
        slides_layout_options = slides.export.HandoutLayoutingOptions()
        
        # Επιλέξτε έναν συγκεκριμένο τύπο διάταξης φυλλαδίου (4 διαφάνειες ανά σελίδα, οριζόντια)
        slides_layout_options.handout = slides.export.HandoutType.HANDOUTS_4_HORIZONTAL
        pdf_options.slides_layout_options = slides_layout_options
```

#### Αποθήκευση της παρουσίασης ως PDF

Τέλος, αποθηκεύστε την παρουσίασή σας με τις διαμορφωμένες επιλογές:
```python
        # Αποθήκευση της παρουσίασης ως PDF με συγκεκριμένες επιλογές
        pres.save(OUTPUT_PATH, slides.export.SaveFormat.PDF, pdf_options)
```

### Συμβουλές αντιμετώπισης προβλημάτων

- **Προβλήματα διαδρομής αρχείου**: Βεβαιωθείτε `DOCUMENT_PATH` και `OUTPUT_PATH` είναι έγκυροι κατάλογοι.
- **Σφάλματα άδειας χρήσης**Επιβεβαιώστε ότι η άδειά σας έχει εφαρμοστεί σωστά εάν αντιμετωπίζετε περιορισμούς λειτουργιών.

## Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, θα δούμε πώς να μετατρέπετε ένα αρχείο PowerPoint σε μορφή XPS χρησιμοποιώντας το Aspose.Slides για Python.

### Επισκόπηση: Μετατροπή παρουσίασης σε XPS

Η κύρια λειτουργικότητα αυτού του σεμιναρίου είναι να δείξει πώς μπορείτε να μετατρέψετε αρχεία PPT σε πιο φορητή και ευέλικτη μορφή XPS.

#### Βήμα 1: Ορισμός καταλόγων
Ξεκινήστε ορίζοντας τους καταλόγους εισόδου και εξόδου όπου βρίσκεται το αρχείο PowerPoint και πού θέλετε να αποθηκεύσετε το αρχείο XPS που έχει μετατραπεί:

```python
input_directory = "YOUR_DOCUMENT_DIRECTORY/"
output_directory = "YOUR_OUTPUT_DIRECTORY/"
```

Αυτές οι διαδρομές θα χρησιμοποιηθούν αργότερα στη συνάρτηση μετατροπής μας.

#### Βήμα 2: Φόρτωση της παρουσίασης
Δημιουργήστε ένα `Presentation` αντικείμενο που αντιπροσωπεύει το αρχείο PowerPoint. Ορίστε τη διαδρομή προς το `.pptx` αρχείο:

```python
demo_presentation_path = input_directory + "welcome-to-powerpoint.pptx"
```

Χρησιμοποιώντας έναν διαχειριστή περιβάλλοντος (`with slides.Presentation(demo_presentation_path) as pres:`), διασφαλίζουμε ότι οι πόροι διαχειρίζονται σωστά.

#### Βήμα 3: Αποθήκευση σε μορφή XPS
Αφού φορτώσετε την παρουσίαση, καθορίστε πού θέλετε να αποθηκεύσετε την έξοδο και χρησιμοποιήστε το `save` μέθοδος μετατροπής:

```python
dxps_output_path = output_directory + "converted_to_xps_out.xps"
pres.save(dxps_output_path, slides.export.SaveFormat.XPS)
```

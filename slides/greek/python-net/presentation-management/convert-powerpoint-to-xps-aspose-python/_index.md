### Οδηγός Εφαρμογής
Θα αναλύσουμε τη διαδικασία σε διαχειρίσιμα βήματα για να διασφαλίσουμε τη σαφήνεια και την ευκολία εφαρμογής.

#### Βήμα 1: Εισαγωγή βιβλιοθηκών
Ξεκινήστε εισάγοντας την απαραίτητη ενότητα:

```python
import aspose.slides as slides
```

Αυτή η εντολή εισαγωγής μας επιτρέπει την πρόσβαση σε όλες τις λειτουργίες που παρέχονται από το Aspose.Slides για Python.

#### Βήμα 2: Ορισμός Συνάρτησης Μετατροπής
Δημιουργήστε μια συνάρτηση που ενσωματώνει τη λογική μετατροπής μας:

```python
def convert_to_xps_with_options():
    # Καθορίστε τη διαδρομή του αρχείου εισόδου χρησιμοποιώντας τον κατάλογο placeholder
    input_file = "YOUR_DOCUMENT_DIRECTORY/welcome-to-powerpoint.pptx"

    # Ανοίξτε το αρχείο παρουσίασης με έναν διαχειριστή περιβάλλοντος για τη διαχείριση πόρων
    with slides.Presentation(input_file) as pres:
        # Δημιουργήστε μια παρουσία του XpsOptions για να διαμορφώσετε τις ρυθμίσεις εξαγωγής
        xps_options = slides.export.XpsOptions()

        # Ορίστε την επιλογή για αποθήκευση μετααρχείων ως εικόνες PNG μέσα στο έγγραφο XPS
        xps_options.save_metafiles_as_png = True

        # Ορίστε τη διαδρομή του αρχείου εξόδου χρησιμοποιώντας τον κατάλογο placeholder
        output_file = "YOUR_OUTPUT_DIRECTORY/convert_to_xps_with_options_out.xps"

        # Αποθήκευση της παρουσίασης σε μορφή XPS με καθορισμένες επιλογές
        pres.save(output_file, slides.export.SaveFormat.XPS, xps_options)
```

#### Επεξήγηση βασικών στοιχείων
- **`XpsOptions`**Αυτή η κλάση σάς επιτρέπει να διαμορφώσετε διάφορες ρυθμίσεις εξαγωγής. Στο παράδειγμά μας, ορίσαμε `save_metafiles_as_png` σε True για να διασφαλίσετε ότι τα μετααρχεία αποθηκεύονται ως εικόνες PNG στο έγγραφο XPS.
  
- **Διαχείριση Πόρων**: Χρήση διαχειριστή περιβάλλοντος (`with slides.Presentation(input_file) as pres:`) διασφαλίζει ότι οι πόροι διαχειρίζονται σωστά και αποδεσμεύονται μετά τη χρήση τους.

#### Βήμα 3: Εκτέλεση μετατροπής
Τέλος, καλέστε τη συνάρτηση για να εκτελέσετε τη μετατροπή:

```python
convert_to_xps_with_options()
```

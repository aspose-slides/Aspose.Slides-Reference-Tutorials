## Οδηγός Εφαρμογής

Αυτή η ενότητα θα σας καθοδηγήσει στη μετατροπή PPTX σε TIFF χρησιμοποιώντας το Aspose.Slides.

### Επισκόπηση της διαδικασίας μετατροπής

Θα μετατρέψουμε ένα αρχείο PowerPoint σε εικόνα TIFF, εφαρμόζοντας προσαρμοσμένες ρυθμίσεις μορφής pixel και συμπεριλαμβάνοντας σημειώσεις διαφανειών στο κάτω μέρος. Αυτή η διαδικασία είναι ιδανική για τη δημιουργία εικόνων αρχειακής ποιότητας ή την ενσωμάτωση παρουσιάσεων σε ροές εργασίας εγγράφων.

#### Βήμα 1: Εισαγωγή βιβλιοθηκών

Ξεκινήστε εισάγοντας τις απαραίτητες ενότητες:

```python
import aspose.slides as slides
```

#### Βήμα 2: Αρχικοποίηση αντικειμένου παρουσίασης

Φορτώστε το αρχείο παρουσίασής σας χρησιμοποιώντας έναν διαχειριστή περιβάλλοντος για να χειριστείτε αποτελεσματικά τη διαχείριση πόρων:

```python\with slides.Presentation('YOUR_DOCUMENT_DIRECTORY/welcome-to-powerpoint.pptx') as presentation:
    # Further processing goes here
```

#### Βήμα 3: Ρύθμιση παραμέτρων του TiffOptions

Δημιουργήστε μια παρουσία του `TiffOptions` για να καθορίσετε ρυθμίσεις εξαγωγής, συμπεριλαμβανομένης της μορφής pixel και των επιλογών διάταξης για σημειώσεις:

```python
tiff_options = slides.export.TiffOptions()
# Ορίστε τη μορφή pixel σε FORMAT_8BPP_INDEXED (8 bit ανά pixel, με ευρετήριο)
tiff_options.pixel_format = slides.export.ImagePixelFormat.FORMAT_8BPP_INDEXED

# Ρύθμιση παραμέτρων εμφάνισης σημειώσεων στην έξοδο TIFF
slides_layout_options = slides.export.NotesCommentsLayoutingOptions()
slides_layout_options.notes_position = slides.export.NotesPositions.BOTTOM_FULL
tiff_options.slides_layout_options = slides_layout_options
```

#### Βήμα 4: Αποθήκευση ως TIFF

Τέλος, αποθηκεύστε την παρουσίαση σε ένα αρχείο TIFF με τις καθορισμένες επιλογές σας:

```python
output_file = 'YOUR_OUTPUT_DIRECTORY/convert_to_tiff_image_pixel_format_out.tiff'
presentation.save(output_file, slides.export.SaveFormat.TIFF, tiff_options)
```

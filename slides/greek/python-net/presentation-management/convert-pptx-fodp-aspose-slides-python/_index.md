## Οδηγός Εφαρμογής

### Μετατροπή PowerPoint (PPTX) σε FODP

#### Επισκόπηση:
Μετατρέψτε μια παρουσίαση PowerPoint σε μορφή FODP για συμβατότητα με συστήματα που υποστηρίζουν αυτό το πρότυπο ανοιχτού εγγράφου.

#### Βήματα Υλοποίησης:

##### Φόρτωση του αρχείου εισόδου PPTX
Φορτώστε το αρχείο PowerPoint χρησιμοποιώντας το Aspose.Slides, διασφαλίζοντας τις σωστές διαδρομές καταλόγων.

```python
def convert_to_fodp():
    # Φορτώστε το αρχείο PowerPoint εισόδου από έναν καθορισμένο κατάλογο.
    with slides.Presentation("YOUR_DOCUMENT_DIRECTORY/welcome-to-powerpoint.pptx") as pres:
        # Αποθηκεύστε το σε μορφή FODP σε έναν κατάλογο εξόδου.
        pres.save("YOUR_OUTPUT_DIRECTORY/convert_to_fodp_out.fodp", slides.export.SaveFormat.FODP)
```

- **Εξήγηση**: Το `Presentation` η κλάση φορτώνει το αρχείο PPTX και `pres.save()` το γράφει σε μορφή FODP.

##### Αποθήκευση ως FODP
Χρήση `SaveFormat.FODP` για να καθορίσετε τη μορφή εξόδου, διασφαλίζοντας την ακεραιότητα των δεδομένων κατά τη μετατροπή.

### Μετατροπή FODP πίσω σε PowerPoint (PPTX)

#### Επισκόπηση:
Αντιστρέψτε τη διαδικασία μετατροπής από FODP σε PPTX για ευρύτερη χρήση παρουσίασης σε όλες τις πλατφόρμες.

#### Βήματα Υλοποίησης:

##### Φόρτωση του αρχείου FODP
Ξεκινήστε φορτώνοντας το αρχείο FODP χρησιμοποιώντας το Aspose.Slides με παρόμοιο τρόπο όπως πριν.

```python
def convert_fodp_to_pptx():
    # Φορτώστε το αρχείο FODP από έναν κατάλογο εξόδου.
    with slides.Presentation("YOUR_OUTPUT_DIRECTORY/convert_to_fodp_out.fodp") as pres:
        # Μετατρέψτε το και αποθηκεύστε το ξανά σε μορφή PowerPoint στον καθορισμένο κατάλογο.
        pres.save("YOUR_OUTPUT_DIRECTORY/convert_to_fodp_out.pptx", slides.export.SaveFormat.PPTX)
```

- **Εξήγηση**: Το `SaveFormat.PPTX` Η παράμετρος διασφαλίζει ότι η παρουσίασή σας αποθηκεύεται ξανά ως αρχείο .pptx.

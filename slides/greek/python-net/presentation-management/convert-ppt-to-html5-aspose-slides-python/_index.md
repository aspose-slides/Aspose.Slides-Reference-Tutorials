## Οδηγός Εφαρμογής
Σε αυτήν την ενότητα, θα σας καθοδηγήσουμε στη μετατροπή μιας παρουσίασης PPT σε μορφή HTML5 με ενεργοποιημένες τις κινούμενες εικόνες.

### Βήμα 1: Ορισμός καταλόγων εισόδου και εξόδου
Ρυθμίστε τους καταλόγους εισόδου και εξόδου χρησιμοποιώντας την Python `pathlib` βιβλιοθήκη:
```python
from pathlib import Path

data_dir = Path("YOUR_DOCUMENT_DIRECTORY/") / "welcome-to-powerpoint.pptx"
out_dir = Path("YOUR_OUTPUT_DIRECTORY/")
output_file = out_dir / "convert_to_html5_out.html"
# Βεβαιωθείτε ότι υπάρχουν κατάλογοι
Path("YOUR_DOCUMENT_DIRECTORY/").mkdir(parents=True, exist_ok=True)
Path("YOUR_OUTPUT_DIRECTORY/").mkdir(parents=True, exist_ok=True)
```
### Βήμα 2: Ανοίξτε την παρουσίαση
Ανοίξτε το αρχείο παρουσίασής σας χρησιμοποιώντας το Aspose.Slides:
```python
with slides.Presentation(data_dir) as pres:
    # Συνεχίστε με τα βήματα μετατροπής εδώ
```
### Βήμα 3: Ρύθμιση παραμέτρων επιλογών εξαγωγής HTML5
Για να συμπεριλάβετε κινούμενες εικόνες στην έξοδο HTML5, διαμορφώστε τις επιλογές εξαγωγής:
```python
html5_options = slides.export.Html5Options()
html5_options.animate_shapes = True     # Ενεργοποίηση κινούμενων εικόνων σχήματος
click to enable transition animations
html5_options.animate_transitions = True
```
### Βήμα 4: Αποθήκευση της παρουσίασης ως HTML5
Τέλος, αποθηκεύστε την παρουσίασή σας με τις καθορισμένες επιλογές:
```python
pres.save(output_file, slides.export.SaveFormat.HTML5, html5_options)
```
Αυτό διασφαλίζει ότι όλες οι μεταβάσεις διαφανειών και οι κινούμενες εικόνες σχήματος διατηρούνται στην έξοδο HTML5.

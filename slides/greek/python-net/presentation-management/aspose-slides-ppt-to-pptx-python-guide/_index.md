## Οδηγός Υλοποίησης: Μετατροπή PPT σε PPTX

### Επισκόπηση της διαδικασίας μετατροπής
Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε παρουσιάσεις PowerPoint από μορφή .ppt σε .pptx, διασφαλίζοντας τη συμβατότητα με το σύγχρονο λογισμικό.

#### Βήμα 1: Φόρτωση του αρχείου PPT
Ξεκινήστε φορτώνοντας το υπάρχον αρχείο .ppt χρησιμοποιώντας το Aspose.Slides:
```python
# Φόρτωση του αρχείου PPT
current_presentation = slides.Presentation("YOUR_DOCUMENT_DIRECTORY/welcome-to-powerpoint.ppt")
```

#### Βήμα 2: Αποθήκευση ως PPTX
Μετά τη φόρτωση, μετατρέψτε και αποθηκεύστε την παρουσίασή σας σε μορφή .pptx:
```python
# Μετατρέψτε και αποθηκεύστε το αρχείο ως PPTX
current_presentation.save("YOUR_OUTPUT_DIRECTORY/convert_to_ppt_out.pptx", slides.export.SaveFormat.PPTX)
```

Αυτό το απόσπασμα κώδικα δείχνει πώς να φορτώσετε ένα αρχείο PowerPoint και να το μετατρέψετε σε άλλη μορφή, παρουσιάζοντας τις δυνατότητες μετατροπής του Aspose.Slides.

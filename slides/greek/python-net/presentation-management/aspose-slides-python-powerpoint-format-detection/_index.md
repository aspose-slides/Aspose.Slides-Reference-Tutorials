## Οδηγός Εφαρμογής

### Λειτουργία ανίχνευσης μορφής αρχείου

Ας εξερευνήσουμε πώς να προσδιορίσουμε τη μορφή ενός αρχείου PowerPoint με το Aspose.Slides.

#### Βήμα 1: Πρόσβαση στις πληροφορίες παρουσίασης

Αρχικά, αποκτήστε πρόσβαση στις λεπτομέρειες της παρουσίασης:

```python
def get_file_format(document_path):
    info = slides.PresentationFactory.instance.get_presentation_info(document_path)
```

Αυτό ανακτά μεταδεδομένα σχετικά με το αρχείο σας, τα οποία είναι κρίσιμα για την αναγνώριση της μορφής.

#### Βήμα 2: Προσδιορίστε τη μορφή αρχείου

Στη συνέχεια, ελέγξτε αν το αρχείο είναι PPTX ή άγνωστο:

```python
def get_file_format(document_path):
    info = slides.PresentationFactory.instance.get_presentation_info(document_path)
    if info.load_format == slides.LoadFormat.PPTX:
        return "pptx"
    elif info.load_format == slides.LoadFormat.UNKNOWN:
        return "unknown"

# Παράδειγμα Χρήσης:
document_path = "YOUR_DOCUMENT_DIRECTORY/welcome-to-powerpoint.pptx"
file_format = get_file_format(document_path)
print(file_format)
```

**Εξήγηση**: Το `get_presentation_info` Η μέθοδος ανακτά τη μορφή φόρτωσης του αρχείου. Το συγκρίνουμε με γνωστές σταθερές για να προσδιορίσουμε αν πρόκειται για PPTX ή για άγνωστη μορφή.

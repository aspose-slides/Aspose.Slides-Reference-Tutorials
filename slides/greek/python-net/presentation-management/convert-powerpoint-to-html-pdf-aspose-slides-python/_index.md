## Οδηγός Εφαρμογής

### Μετατροπή PowerPoint σε HTML με σημειώσεις που περιλαμβάνονται

Ιδανική για ενσωμάτωση στο web, αυτή η λειτουργία σάς επιτρέπει να εμφανίζετε σημειώσεις παρουσίασης κάτω από κάθε διαφάνεια.

#### Επισκόπηση

Η μετατροπή ενός αρχείου PowerPoint σε έγγραφο HTML διασφαλίζει τη συμβατότητα σε όλες τις συσκευές και τα προγράμματα περιήγησης. Αυτή η μέθοδος περιλαμβάνει τη δυνατότητα παρουσίασης σημειώσεων κάτω από τις διαφάνειες.

#### Βήματα

##### Φόρτωση της παρουσίασής σας

Ξεκινήστε φορτώνοντας το αρχείο PowerPoint:

```python
import aspose.slides as slides

def convert_to_html_with_notes():
    presentation_path = 'YOUR_DOCUMENT_DIRECTORY/welcome-to-powerpoint.pptx'
    with slides.Presentation(presentation_path) as pres:
        # Περαιτέρω βήματα θα ακολουθήσουν εδώ...
```

##### Ρύθμιση παραμέτρων επιλογών HTML

Ορίστε επιλογές για να συμπεριλάβετε σημειώσεις στο κάτω μέρος:

```python
        html_options = slides.export.HtmlOptions()
        layout_options = slides.export.NotesCommentsLayoutingOptions()
        layout_options.notes_position = slides.export.NotesPositions.BOTTOM_FULL
        html_options.slides_layout_options = layout_options
```

##### Αποθήκευση ως HTML

Αποθηκεύστε την παρουσίασή σας με αυτές τις διαμορφώσεις:

```python
        output_path = 'YOUR_OUTPUT_DIRECTORY/convert_to_html_with_notes_out.html'
        pres.save(output_path, slides.export.SaveFormat.HTML, html_options)
```

#### Συμβουλές αντιμετώπισης προβλημάτων

- Βεβαιωθείτε ότι η άδειά σας έχει ρυθμιστεί σωστά εάν αντιμετωπίζετε περιορισμούς χρήσης.

### Μετατροπή PowerPoint σε PDF

Η μετατροπή των παρουσιάσεων σε μορφή PDF διασφαλίζει ότι μπορούν να προβληθούν σε οποιαδήποτε συσκευή χωρίς πρόσθετο λογισμικό.

#### Επισκόπηση

Αυτή η διαδικασία διατηρεί την οπτική διάταξη και την ποιότητα των διαφανειών σας, καθιστώντας τες ιδανικές για κοινή χρήση σε διάφορες πλατφόρμες.

#### Βήματα

##### Φόρτωση της παρουσίασής σας

Ξεκινήστε φορτώνοντας το αρχείο PowerPoint όπως φαίνεται παραπάνω.

##### Αποθήκευση ως PDF

Χρησιμοποιήστε αυτό το απόσπασμα για να αποθηκεύσετε την παρουσίασή σας σε μορφή PDF:

```python
def convert_to_pdf():
    presentation_path = 'YOUR_DOCUMENT_DIRECTORY/welcome-to-powerpoint.pptx'
    with slides.Presentation(presentation_path) as pres:
        output_path = 'YOUR_OUTPUT_DIRECTORY/convert_to_pdf_out.pdf'
        pres.save(output_path, slides.export.SaveFormat.PDF)
```

#### Συμβουλές αντιμετώπισης προβλημάτων
- Ελέγξτε ξανά τις διαδρομές καταλόγων για να αποφύγετε σφάλματα.
- Εάν αντιμετωπίσετε τυχόν περιορισμούς κατά τη μετατροπή, βεβαιωθείτε ότι η άδειά σας είναι ενεργή.

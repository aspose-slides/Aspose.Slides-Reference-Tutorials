## Οδηγός υλοποίησης: Μετατροπή PPTX σε PDF με κρυφές διαφάνειες

### Επισκόπηση της λειτουργίας
Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε μια παρουσίαση PowerPoint σε αρχείο PDF, διασφαλίζοντας ότι όλες οι κρυφές διαφάνειες περιλαμβάνονται στο αποτέλεσμα. Αυτό είναι ιδιαίτερα χρήσιμο όταν κάθε κομμάτι περιεχομένου πρέπει να διατηρηθεί για σκοπούς αρχειοθέτησης ή κοινής χρήσης.

#### Βήμα 1: Φόρτωση της παρουσίασης
Ξεκινήστε φορτώνοντας το αρχείο PPTX χρησιμοποιώντας το `Presentation` τάξη.

```python
with slides.Presentation("YOUR_DOCUMENT_DIRECTORY/presentation_with_hidden_slides.pptx") as presentation:
    # Περαιτέρω επεξεργασία θα πραγματοποιηθεί εδώ
```

#### Βήμα 2: Ρύθμιση παραμέτρων επιλογών PDF
Δημιουργήστε ένα υπόδειγμα `PdfOptions` αντικείμενο για να καθορίσετε επιλογές για τη μετατροπή PDF. Εδώ, θα ορίσετε την επιλογή συμπερίληψης κρυφών διαφανειών.

```python
class PdfOptions:
    def __init__(self):
        self.εμφάνιση_κρυφών_διαφανειών = False

pdf_options = PdfOptions()
pdf_options.show_hidden_slides = True
```

- **show_hidden_slides**Αυτή η παράμετρος είναι κρίσιμη, καθώς καθορίζει εάν οι κρυφές διαφάνειες περιλαμβάνονται στο PDF εξόδου.

#### Βήμα 3: Αποθήκευση της παρουσίασης
Τέλος, αποθηκεύστε την παρουσίασή σας ως αρχείο PDF με τις καθορισμένες επιλογές.

```python
target_directory = "YOUR_OUTPUT_DIRECTORY"
presentation.save(f"{target_directory}/convert_to_pdf_hidden_slides_out.pdf", \
                 slides.export.SaveFormat.PDF, pdf_options)
```

## Οδηγός Εφαρμογής

Τώρα που έχετε τακτοποιήσει τις προϋποθέσεις, ας εμβαθύνουμε στον χειρισμό παρουσιάσεων PowerPoint.

### Άνοιγμα και κλωνοποίηση διαφανειών

#### Επισκόπηση

Σε αυτήν την ενότητα, θα ανοίξουμε ένα υπάρχον αρχείο PowerPoint και θα κλωνοποιήσουμε μια διαφάνεια που περιέχει ένα στοιχείο ελέγχου ActiveX σε μια νέα παρουσία παρουσίασης.

#### Βήματα

**Βήμα 1: Ανοίξτε ένα υπάρχον αρχείο PowerPoint**

Ξεκινήστε ανοίγοντας το αρχείο PowerPoint προορισμού χρησιμοποιώντας το `Presentation` τάξη:

```python
with slides.Presentation(YOUR_DOCUMENT_DIRECTORY + "activex_template.pptx") as pres:
    # Αποκτήστε πρόσβαση στην υπάρχουσα παρουσίασή σας εδώ
```

**Βήμα 2: Κατάργηση προεπιλεγμένης διαφάνειας**

Δημιουργήστε μια νέα παρουσίαση και καταργήστε την προεπιλεγμένη διαφάνειά της για να την προετοιμάσετε για κλωνοποίηση:

```python
new_pres = slides.Presentation()
new_pres.slides.remove_at(0)
```

**Βήμα 3: Κλωνοποιήστε τη διαφάνεια με το ActiveX Control**

Κλωνοποιήστε μια συγκεκριμένη διαφάνεια από την αρχική σας παρουσίαση στη νέα:

```python
new_pres.slides.insert_clone(0, pres.slides[0])
```

### Τροποποίηση στοιχείων ελέγχου ActiveX

#### Επισκόπηση

Τα στοιχεία ελέγχου ActiveX μπορούν να αποτελέσουν ισχυρά εργαλεία μέσα σε διαφάνειες. Εδώ, θα τροποποιήσουμε ένα υπάρχον στοιχείο ελέγχου του Media Player.

#### Βήματα

**Βήμα 4: Πρόσβαση και τροποποίηση ιδιοτήτων ελέγχου**

Αποκτήστε πρόσβαση στο πρώτο στοιχείο ελέγχου στην κλωνοποιημένη διαφάνεια και αλλάξτε τις ιδιότητές του:

```python
control = new_pres.slides[0].controls[0]
control.properties.remove("URL")
control.properties.add("URL", YOUR_DOCUMENT_DIRECTORY + "video.mp4")
```

### Αποθήκευση της παρουσίασής σας

#### Επισκόπηση

Αφού επεξεργαστείτε τις διαφάνειές σας, ήρθε η ώρα να αποθηκεύσετε την τροποποιημένη παρουσίαση.

**Βήμα 5: Αποθήκευση της παρουσίασης**

```python
new_pres.save(YOUR_OUTPUT_DIRECTORY + "activex_linking_video_activex_control_out.pptx", slides.export.SaveFormat.PPTX)
```

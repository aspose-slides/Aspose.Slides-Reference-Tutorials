### Βασική Αρχικοποίηση και Ρύθμιση
Μόλις εγκατασταθεί, αρχικοποιήστε το Aspose.Slides στο Python script σας:
```python
import aspose.slides as slides

# Αρχικοποίηση του αντικειμένου παρουσίασης
presentation = slides.Presentation("path_to_your_pptx_file")
```

## Οδηγός Εφαρμογής

Τώρα, ας επικεντρωθούμε στην εφαρμογή της δυνατότητας μετατροπής σημειώσεων PowerPoint σε αρχείο PDF.

### Φόρτωση της παρουσίασης με σημειώσεις
Ξεκινήστε φορτώνοντας την παρουσίασή σας που περιλαμβάνει λεπτομερείς σημειώσεις ομιλητή:
```python
# Βήμα 1: Φόρτωση σημειώσεων στην παρουσίαση
presentation_path = "YOUR_DOCUMENT_DIRECTORY/presentation_with_notes.pptx"
with slides.Presentation(presentation_path) as presentation:
    # Ο κώδικας για τη μετατροπή ακολουθεί...
```

### Ρύθμιση παραμέτρων επιλογών για εξαγωγή σε PDF
Στη συνέχεια, διαμορφώστε τις ρυθμίσεις εξαγωγής για να βεβαιωθείτε ότι όλες οι σημειώσεις καταγράφονται σωστά στο PDF που προκύπτει:
```python
# Βήμα 2: Διαμορφώστε τις επιλογές για εξαγωγή σε PDF
pdf_options = slides.export.PdfOptions()

# Ορισμός επιλογών διάταξης για σημειώσεις και σχόλια
default_layout = slides.export.NotesCommentsLayoutingOptions()
default_layout.notes_position = slides.export.NotesPositions.BOTTOM_FULL

# Αντιστοιχίστε τις επιλογές διάταξης σημειώσεων στις επιλογές εξαγωγής PDF
pdf_options.slides_layout_options = default_layout
```

### Αποθήκευση της παρουσίασης ως αρχείο PDF με σημειώσεις
Τέλος, αποθηκεύστε την παρουσίασή σας σε ένα νέο αρχείο PDF διατηρώντας παράλληλα όλες τις σημειώσεις:
```python
# Βήμα 3: Αποθηκεύστε την παρουσίαση ως αρχείο PDF με σημειώσεις
output_path = "YOUR_OUTPUT_DIRECTORY/convert_notes_to_pdf_out.pdf"
presentation.save(output_path, slides.export.SaveFormat.PDF, pdf_options)
```

### Επεξήγηση των επιλογών διαμόρφωσης κλειδιού
- **`NotesCommentsLayoutingOptions()`**Αυτή η κλάση σάς επιτρέπει να καθορίσετε τον τρόπο εμφάνισης των σημειώσεων στο PDF.
- **`notes_position = slides.export.NotesPositions.BOTTOM_FULL`**Τοποθετεί τις σημειώσεις στο κάτω μέρος κάθε σελίδας, διασφαλίζοντας την ορατότητα και την πληρότητα.

**Συμβουλές αντιμετώπισης προβλημάτων:**
- Βεβαιωθείτε ότι οι διαδρομές σας έχουν καθοριστεί σωστά. Οι σχετικές διαδρομές μπορεί μερικές φορές να προκαλέσουν προβλήματα εάν δεν οριστούν σωστά.
- Βεβαιωθείτε ότι το αρχείο PowerPoint περιέχει σημειώσεις. Διαφορετικά, δεν θα εμφανίζονται στο PDF.

## Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, θα αναλύσουμε τη διαδικασία μετατροπής σε διαχειρίσιμα βήματα.

### Μετατροπή PPTX σε HTML με διατήρηση των αρχικών γραμματοσειρών

#### Επισκόπηση:
Το κύριο χαρακτηριστικό αυτής της υλοποίησης είναι η μετατροπή μιας παρουσίασης PowerPoint διατηρώντας παράλληλα τις αρχικές γραμματοσειρές της και εξαιρώντας συγκεκριμένες προεπιλεγμένες από την ενσωμάτωση. Αυτό μπορεί να είναι ιδιαίτερα χρήσιμο για τη διατήρηση της συνέπειας της επωνυμίας σε όλες τις παρουσιάσεις ιστού.

#### Βήμα προς βήμα εφαρμογή:

**1. Ορισμός διαδρομών εισόδου και εξόδου**

Ρυθμίστε τους καταλόγους όπου βρίσκεται το αρχείο PPTX εισόδου και όπου θέλετε να αποθηκεύσετε το αρχείο HTML εξόδου.

```python
data_dir = "YOUR_DOCUMENT_DIRECTORY/"
out_dir = "YOUR_OUTPUT_DIRECTORY/"
```

**2. Ανοίξτε το Αρχείο Παρουσίασης**

Χρήση του Aspose.Slides `Presentation` κλάση για να φορτώσετε το αρχείο PPTX σας:

```python
with slides.Presentation(data_dir + "welcome-to-powerpoint.pptx") as pres:
    # Ο κωδικός μετατροπής σας θα τοποθετηθεί εδώ.
```

Αυτός ο διαχειριστής περιβάλλοντος διασφαλίζει ότι οι πόροι απελευθερώνονται σωστά μετά τη λειτουργία.

**3. Δημιουργήστε έναν ελεγκτή ενσωμάτωσης προσαρμοσμένων γραμματοσειρών**

Εξαίρεση ορισμένων γραμματοσειρών από την ενσωμάτωση χρησιμοποιώντας `EmbedAllFontsHtmlController`:

```python
font_name_exclude_list = ["Calibri", "Arial"]
embed_fonts_controller = slides.export.EmbedAllFontsHtmlController(font_name_exclude_list)
```

Εδώ, οι όροι "Calibri" και "Arial" εξαιρούνται από την ενσωμάτωση στην έξοδο HTML.

**4. Ρύθμιση παραμέτρων επιλογών εξαγωγής HTML**

Στήνω `HtmlOptions` για να χρησιμοποιήσετε ένα προσαρμοσμένο εργαλείο μορφοποίησης γραμματοσειρών με τον ελεγκτή σας:

```python
html_options_embed = slides.export.HtmlOptions()
html_options_embed.html_formatter = slides.export.HtmlFormatter.create_custom_formatter(embed_fonts_controller)
```

Αυτό το βήμα διασφαλίζει ότι μόνο οι απαραίτητες γραμματοσειρές ενσωματώνονται στο τελικό αποτέλεσμα.

**5. Αποθηκεύστε την παρουσίαση ως HTML**

Τέλος, αποθηκεύστε την παρουσίαση σε ένα αρχείο HTML με τις καθορισμένες επιλογές σας:

```python
pres.save(out_dir + "convert_to_html_with_preserving_original_fonts_out.html", 
          slides.export.SaveFormat.HTML, html_options_embed)
```

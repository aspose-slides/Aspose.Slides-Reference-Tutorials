## Οδηγός Εφαρμογής

Μάθετε πώς να κλειδώνετε και να ξεκλειδώνετε τις αναλογίες διαστάσεων πίνακα στο PowerPoint χρησιμοποιώντας το Aspose.Slides για Python.

### Κλείδωμα της αναλογίας διαστάσεων ενός πίνακα (Λειτουργία: Κλείδωμα αναλογίας διαστάσεων)

#### Επισκόπηση

Αυτή η λειτουργία διασφαλίζει ότι η αλλαγή μεγέθους των πινάκων δεν παραμορφώνει το σχήμα τους, διατηρώντας την οπτική ομοιομορφία σε όλες τις διαφάνειες.

#### Βήμα προς βήμα εφαρμογή

##### Πρόσβαση στην παρουσίαση και τον πίνακα

Φορτώστε την παρουσίασή σας και αποκτήστε πρόσβαση στον πίνακα που θέλετε να τροποποιήσετε:

```python
import aspose.slides as slides

def lock_aspect_ratio():
    with slides.Presentation('YOUR_DOCUMENT_DIRECTORY/tables.pptx') as pres:
        # Ας υποθέσουμε ότι το πρώτο σχήμα στην πρώτη διαφάνεια είναι ένας πίνακας.
        table = pres.slides[0].shapes[0]
```

##### Έλεγχος τρέχουσας κατάστασης κλειδώματος αναλογίας διαστάσεων

Ελέγξτε αν το κλείδωμα αναλογίας διαστάσεων είναι ήδη ενεργοποιημένο:

```python
print(f"Lock aspect ratio set: {table.shape_lock.aspect_ratio_locked}")
```

##### Ενεργοποίηση/απενεργοποίηση του κλειδώματος αναλογίας διαστάσεων

Αντιστρέψτε την τρέχουσα κατάσταση του κλειδώματος αναλογίας διαστάσεων:

```python
table.shape_lock.aspect_ratio_locked = not table.shape_lock.aspect_ratio_locked
```

##### Αποθήκευση αλλαγών στην παρουσίασή σας

Αποθηκεύστε την τροποποιημένη παρουσίασή σας:

```python
pres.save('YOUR_OUTPUT_DIRECTORY/tables_pres_lock_aspect_ratio_out.pptx', slides.export.SaveFormat.PPTX)
```

##### Βήμα 1: Δημιουργήστε μια παρουσία της κλάσης παρουσίασης

Ο `Presentation` Η κλάση αντιπροσωπεύει ένα αρχείο PPT. Εδώ, θα ανοίξουμε ή θα δημιουργήσουμε μια νέα παρουσίαση:

```python
with slides.Presentation() as pres:
    # Η παρουσία παρουσίασης χρησιμοποιείται εντός αυτού του μπλοκ διαχειριστή περιβάλλοντος.
```

##### Βήμα 2: Πρόσβαση στην πρώτη διαφάνεια

Η πρόσβαση στην πρώτη διαφάνεια μας επιτρέπει να προσθέσουμε τον πίνακά μας εκεί:

```python
slide = pres.slides[0]  # Αυτό ανακτά την πρώτη διαφάνεια από την παρουσίαση.
```

```python
dbl_cols = [50, 50, 50]  # Πλάτος στηλών
dbl_rows = [50, 30, 30, 30, 30]  # Ύψη σειρών

table = slide.shapes.add_table(50, 50, dbl_cols, dbl_rows)  # Προσθήκη πίνακα στη διαφάνεια.
```

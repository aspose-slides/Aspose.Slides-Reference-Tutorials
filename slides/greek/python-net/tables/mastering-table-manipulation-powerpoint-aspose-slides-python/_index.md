## Οδηγός Εφαρμογής

### Πρόσβαση και τροποποίηση ενός πίνακα στο PowerPoint

#### Επισκόπηση
Θα ανοίξουμε ένα υπάρχον αρχείο PPTX, θα εντοπίσουμε έναν συγκεκριμένο πίνακα, θα ενημερώσουμε τα περιεχόμενά του και θα αποθηκεύσουμε τις αλλαγές. Αυτή η διαδικασία είναι ιδανική για μαζικές ενημερώσεις δεδομένων παρουσίασης.

#### Βήματα
1. **Άνοιγμα της παρουσίασής σας**
   Φορτώστε το αρχείο PowerPoint:
   ```python
   with slides.Presentation("YOUR_DOCUMENT_DIRECTORY/tables_update.pptx") as presentation:
       slide = presentation.slides[0]
   ```
   Αυτός ο κώδικας ανοίγει το αρχείο και έχει πρόσβαση στην πρώτη διαφάνεια.

2. **Εύρεση και ενημέρωση του πίνακα**
   Προσδιορισμός και ενημέρωση κελιών πίνακα:
   ```python
   for shape in slide.shapes:
       if isinstance(shape, slides.Table):
           # Ενημέρωση κειμένου σε ένα συγκεκριμένο κελί
           shape.rows[0][1].text_frame.text = "New"
   ```
   Αυτό το τμήμα κώδικα ενημερώνει το επιθυμητό κελί μέσα στην πρώτη γραμμή.

3. **Αποθήκευση των αλλαγών σας**
   Αποθηκεύστε την ενημερωμένη παρουσίασή σας:
   ```python
   presentation.save("YOUR_OUTPUT_DIRECTORY/tables_update_table_out.pptx", slides.export.SaveFormat.PPTX)
   ```
   Η εντολή γράφει τις αλλαγές στο δίσκο σε μορφή PPTX.

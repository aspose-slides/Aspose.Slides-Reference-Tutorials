### Απόκτηση Άδειας

Το Aspose.Slides λειτουργεί με ένα μοντέλο αδειοδότησης. Ξεκινήστε με μια δωρεάν δοκιμή ή αποκτήστε μια προσωρινή άδεια χρήσης για να εξερευνήσετε όλες τις δυνατότητές του. Για χρήση σε παραγωγή, σκεφτείτε να αγοράσετε μια συνδρομή.

#### Βασική Αρχικοποίηση και Ρύθμιση

Μετά την εγκατάσταση, αρχικοποιήστε το περιβάλλον σας:

```python
from aspose.slides import Presentation

# Φόρτωση υπάρχουσας παρουσίασης
pres = Presentation("tables.pptx")
```

## Οδηγός Εφαρμογής

### Ορισμός της πρώτης γραμμής ως κεφαλίδας

Αυτοματοποιήστε τη μορφοποίηση πινάκων επισημαίνοντας την πρώτη γραμμή ως κεφαλίδα, κάτι που συχνά απαιτεί ειδικό στυλ.

#### Βήμα 1: Εισαγωγή απαιτούμενων ενοτήτων

Ξεκινήστε εισάγοντας τις απαραίτητες ενότητες:

```python
import os
from aspose.slides import Presentation, slides
```

#### Βήμα 2: Ορισμός διαδρομών εγγράφων

Ορίστε διαδρομές για τα αρχεία εισόδου και εξόδου:

```python
document_directory = "YOUR_DOCUMENT_DIRECTORY/"
output_directory = "YOUR_OUTPUT_DIRECTORY/"

tpptx_path = os.path.join(document_directory, 'tables.pptx')
```

#### Βήμα 3: Φόρτωση της παρουσίασης

Ανοίξτε το αρχείο PowerPoint και αποκτήστε πρόσβαση στην πρώτη του διαφάνεια:

```python
with Presentation(pptx_path) as pres:
    slide = pres.slides[0]
```

#### Βήμα 4: Επαναλάβετε τα σχήματα για να βρείτε πίνακες

Περιηγηθείτε σε κάθε σχήμα στη διαφάνεια για να εντοπίσετε πίνακες:

```python
for shape in slide.shapes:
    if isinstance(shape, slides.Table):
        # Σημειώστε την πρώτη γραμμή ως κεφαλίδα
        shape.header_rows = 1  # Διορθωμένη μέθοδος για τον ορισμό κεφαλίδων
```

#### Βήμα 5: Αποθήκευση της τροποποιημένης παρουσίασης

Αποθηκεύστε τις αλλαγές σας σε ένα νέο αρχείο:

```python
output_pptx_path = os.path.join(output_directory, 'tables_first_row_as_header_out.pptx')
pres.save(output_pptx_path, slides.export.SaveFormat.PPTX)
```

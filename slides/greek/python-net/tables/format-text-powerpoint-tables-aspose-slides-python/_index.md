## Οδηγός Εφαρμογής

Ας αναλύσουμε τα βήματα για τη μορφοποίηση κειμένου μέσα σε μια γραμμή πίνακα στο PowerPoint χρησιμοποιώντας το Aspose.Slides.

### Πρόσβαση και μορφοποίηση γραμμών πίνακα (H2)

#### Επισκόπηση
Θα ξεκινήσουμε φορτώνοντας μια υπάρχουσα παρουσίαση, αποκτώντας πρόσβαση σε έναν συγκεκριμένο πίνακα μέσα σε αυτήν και εφαρμόζοντας διαφορετικές επιλογές μορφοποίησης στις γραμμές της.

#### Βήμα 1: Φόρτωση της παρουσίασής σας

Αρχικά, δημιουργήστε ή ανοίξτε ένα αρχείο PowerPoint με έναν πίνακα:

```python
input_presentation = 'YOUR_DOCUMENT_DIRECTORY/tables.pptx'
output_presentation = 'YOUR_OUTPUT_DIRECTORY/tables_text_format_inside_row_out.pptx'

with slides.Presentation(input_presentation) as presentation:
    # Πρόσβαση στο πρώτο σχήμα στην πρώτη διαφάνεια, που υποτίθεται ότι είναι ένας πίνακας
    table = presentation.slides[0].shapes[0]
```

#### Βήμα 2: Ορισμός ύψους γραμματοσειράς για τα κελιά στην πρώτη γραμμή

Προσαρμόστε το μέγεθος της γραμματοσειράς χρησιμοποιώντας `PortionFormat`:

```python
# Ορισμός ύψους γραμματοσειράς για τα κελιά στην πρώτη γραμμή
portion_format = slides.PortionFormat()
portion_format.font_height = 25  # Αλλαγή στο επιθυμητό ύψος γραμματοσειράς
table.rows[0].set_text_format(portion_format)
```

**Εξήγηση:** Ο `font_height` Η παράμετρος ελέγχει το μέγεθος του κειμένου μέσα σε κάθε κελί, βελτιώνοντας την ορατότητα.

#### Βήμα 3: Στοίχιση κειμένου και ορισμός περιθωρίων

Για να στοιχίσετε δεξιά το κείμενο στα κελιά της πρώτης γραμμής:

```python
# Ορισμός στοίχισης κειμένου και δεξιού περιθωρίου για τα κελιά στην πρώτη γραμμή
paragraph_format = slides.ParagraphFormat()
paragraph_format.alignment = slides.TextAlignment.RIGHT
paragraph_format.margin_right = 20  # Διάστημα από το δεξί άκρο
table.rows[0].set_text_format(paragraph_format)
```

**Εξήγηση:** `ParagraphFormat` σας επιτρέπει να ευθυγραμμίσετε το κείμενο και να ορίσετε περιθώρια, παρέχοντας μια κομψή εμφάνιση.

#### Βήμα 4: Ορισμός κάθετου τύπου κειμένου για κελιά στη δεύτερη γραμμή

Για κατακόρυφο προσανατολισμό κειμένου:

```python
# Ορισμός κάθετου τύπου κειμένου για τα κελιά στη δεύτερη γραμμή
text_frame_format = slides.TextFrameFormat()
text_frame_format.text_vertical_type = slides.TextVerticalType.VERTICAL
table.rows[1].set_text_format(text_frame_format)
```

**Εξήγηση:** `TextFrameFormat` αλλάζει τον τρόπο εμφάνισης του κειμένου, κάτι που μπορεί να είναι χρήσιμο για γλώσσες όπως τα Ιαπωνικά ή τα Κινέζικα.

#### Βήμα 5: Αποθηκεύστε την παρουσίασή σας

Τέλος, αποθηκεύστε τις αλλαγές σε ένα νέο αρχείο:

```python
# Αποθήκευση της τροποποιημένης παρουσίασης σε ένα νέο αρχείο στον κατάλογο εξόδου
table.save(output_presentation, slides.export.SaveFormat.PPTX)
```

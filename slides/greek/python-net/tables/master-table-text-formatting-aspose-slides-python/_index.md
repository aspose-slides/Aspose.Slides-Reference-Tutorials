# Μορφοποίηση κύριων πινάκων και κειμένου στο PowerPoint με το Aspose.Slides για Python

## Εισαγωγή

Στον σημερινό κόσμο που βασίζεται στις παρουσιάσεις, η δημιουργία διαφανειών οπτικά ελκυστικών, ενώ παράλληλα μεταφέρει αποτελεσματικά πληροφορίες, είναι ζωτικής σημασίας. Αν δυσκολεύεστε να μορφοποιήσετε τέλεια πίνακες ή κείμενο στο PowerPoint χρησιμοποιώντας Python, αυτό το σεμινάριο είναι για εσάς. Θα σας καθοδηγήσουμε στη δημιουργία και μορφοποίηση πινάκων, στην προσθήκη στυλιζαρισμένου κειμένου σε σχήματα και στη σχεδίαση ορθογωνίων γύρω από συγκεκριμένα τμήματα κειμένου—όλα με το Aspose.Slides για Python. Στο τέλος, θα είστε σε θέση να βελτιώσετε τις παρουσιάσεις σας χωρίς κόπο.

**Τι θα μάθετε:**
- Δημιουργία και μορφοποίηση πινάκων χρησιμοποιώντας Aspose.Slides Python

Ας ξεκινήσουμε με τις προαπαιτούμενες προϋποθέσεις.

## Οδηγός Εφαρμογής

Αυτή η ενότητα αναλύει κάθε λειτουργία σε εφαρμόσιμα βήματα.

### Δημιουργία και μορφοποίηση πίνακα

**Επισκόπηση:**
Η δημιουργία δομημένων πινάκων βοηθά στην αποτελεσματική οργάνωση των δεδομένων. Θα προσθέσουμε έναν προσαρμοσμένο πίνακα με μορφοποιημένο κείμενο μέσα στα κελιά του χρησιμοποιώντας το Aspose.Slides Python.

#### Βήμα 1: Αρχικοποίηση παρουσίασης

Ξεκινήστε ρυθμίζοντας το αντικείμενο παρουσίασης:

```python
import aspose.slides as slides

def create_and_format_table():
    # Αρχικοποίηση αντικειμένου παρουσίασης
    with slides.Presentation() as pres:
        pass  # Περαιτέρω βήματα θα προστεθούν εδώ
```

#### Βήμα 2: Προσθήκη και μορφοποίηση πίνακα

Προσθέστε έναν πίνακα στη διαφάνειά σας, καθορίζοντας τη θέση και τις διαστάσεις του:

```python
# Προσθήκη πίνακα στην πρώτη διαφάνεια
table = pres.slides[0].shapes.add_table(50, 50, [50, 70], [50, 50, 50])
```

#### Βήμα 3: Εισαγωγή κειμένου σε κελιά πίνακα

Δημιουργήστε παραγράφους με τμήματα κειμένου και προσθέστε τις στο κελί σας:

```python
# Δημιουργία παραγράφων για τα κελιά του πίνακα
paragraph0 = slides.Paragraph()
paragraph0.portions.add(slides.Portion("Text "))
paragraph0.portions.add(slides.Portion("in0"))
paragraph0.portions.add(slides.Portion(" Cell"))

cell = table.rows[1][1]
cell.text_frame.paragraphs.clear()  # Διαγραφή υπαρχουσών παραγράφων
cell.text_frame.paragraphs.extend([paragraph0])
```

#### Βήμα 4: Αποθήκευση της παρουσίασης

Τέλος, αποθηκεύστε την παρουσίασή σας για να δείτε τις αλλαγές:

```python
# Αποθήκευση της παρουσίασης με μορφοποιημένους πίνακες
pres.save("YOUR_OUTPUT_DIRECTORY/text_create_table_out.pptx", slides.export.SaveFormat.PPTX)
```

### Προσθήκη και μορφοποίηση κειμένου σε σχήμα

**Επισκόπηση:**
Η προσθήκη κειμένου μέσα σε σχήματα όπως ορθογώνια δίνει έμφαση σε σημαντικά σημεία.

#### Βήμα 1: Προσθήκη αυτόματου σχήματος

Δημιουργήστε ένα ορθογώνιο σχήμα για να χωρέσετε το κείμενό σας:

```python
def add_and_format_text_in_shape():
    with slides.Presentation() as pres:
        # Προσθήκη αυτόματου σχήματος στην πρώτη διαφάνεια
        auto_shape = pres.slides[0].shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 400, 100, 60, 120)
```

#### Βήμα 2: Ορισμός κειμένου και στοίχισης

Αντιστοίχιση κειμένου και ορισμός στοίχισης:

```python
# Ορισμός κειμένου και στοίχισης για το σχήμα
auto_shape.text_frame.text = "Text in shape"
auto_shape.text_frame.paragraphs[0].paragraph_format.alignment = slides.TextAlignment.LEFT
```

#### Βήμα 3: Αποθήκευση των αλλαγών σας

Αποθηκεύστε την παρουσίασή σας για να δείτε μορφοποιημένο κείμενο μέσα σε σχήματα:

```python
pres.save("YOUR_OUTPUT_DIRECTORY/text_auto_shape_out.pptx", slides.export.SaveFormat.PPTX)
```

### Σχεδίαση ορθογωνίων γύρω από τμήματα κειμένου και παραγράφους

**Επισκόπηση:**
Επισημάνετε συγκεκριμένα τμήματα ή παραγράφους σχεδιάζοντας ορθογώνια γύρω τους.

#### Βήμα 1: Δημιουργήστε έναν πίνακα με κείμενο

Ξεκινήστε δημιουργώντας έναν πίνακα και εισάγοντας κείμενο:

```python
def draw_rectangles_around_text():
    with slides.Presentation() as pres:
        # Δημιουργήστε έναν πίνακα και προσθέστε κείμενο στο κελί του
        table = pres.slides[0].shapes.add_table(50, 50, [50, 70], [50, 50, 50])
        paragraph0 = slides.Paragraph()
        paragraph0.portions.add(slides.Portion("Text "))
        paragraph0.portions.add(slides.Portion("in0"))
        paragraph0.portions.add(slides.Portion(" Cell"))
```

#### Βήμα 2: Τοποθέτηση και σχεδίαση ορθογωνίων

Υπολογίστε θέσεις και σχεδιάστε ορθογώνια γύρω από συγκεκριμένα τμήματα κειμένου:

```python
# Υπολογισμός θέσης για σχεδίαση
x = table.x + cell.offset_x
y = table.y + cell.offset_y

for para in cell.text_frame.paragraphs:
    if "0" in para.text:
        rect = para.get_rect()
        shape = pres.slides[0].shapes.add_auto_shape(slides.ShapeType.RECTANGLE, rect.x + x, rect.y + y, rect.width, rect.height)
        shape.line_format.fill_format.solid_fill_color.color = drawing.Color.yellow
```

#### Βήμα 3: Αποθήκευση της παρουσίασης

Αποθηκεύστε την παρουσίασή σας για να δείτε τα επισημασμένα τμήματα κειμένου:

```python
pres.save("YOUR_OUTPUT_DIRECTORY/text_draw_rect_out.pptx", slides.export.SaveFormat.PPTX)
```

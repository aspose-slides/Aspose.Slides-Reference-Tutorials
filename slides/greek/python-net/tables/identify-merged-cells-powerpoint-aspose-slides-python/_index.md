## Οδηγός Εφαρμογής

### Αναγνώριση συγχωνευμένων κελιών σε πίνακες PowerPoint

#### Επισκόπηση

Αυτή η λειτουργία σαρώνει κάθε κελί σε έναν πίνακα μέσα σε μια διαφάνεια του PowerPoint για να ελέγξει εάν αποτελεί μέρος ενός συγχωνευμένου συνόλου, παρέχοντας λεπτομέρειες σχετικά με το εύρος και την αρχική του θέση.

#### Βήματα για την Ταυτοποίηση
1. **Φόρτωση της παρουσίασης**
   
   Φορτώστε το αρχείο παρουσίασής σας όπου υποψιάζεστε ότι ενδέχεται να υπάρχουν συγχωνευμένα κελιά:
   ```python
   with slides.Presentation("YOUR_DOCUMENT_DIRECTORY/tables.pptx") as pres:
       # Πρόσβαση στο πρώτο σχήμα στην πρώτη διαφάνεια (υποθέτοντας ότι είναι πίνακας)
       table = pres.slides[0].shapes[0]
   ```

2. **Επανάληψη μέσω κελιών**
   
   Πραγματοποιήστε επανάληψη σε κάθε κελί για να ελέγξετε την κατάσταση συγχώνευσης και να συγκεντρώσετε λεπτομέρειες:
   ```python
   def dump_merged_cell(i, j, current_cell):
       # Εκτύπωση πληροφοριών σχετικά με το συγχωνευμένο κελί
       print(f"Cell {i}{j} is part of a merged cell with row_span={current_cell.row_span}, col_span={current_cell.col_span}, starting from Cell {current_cell.first_row_index}{current_cell.first_column_index}.")
   
   for i, row in enumerate(table.rows):
       for j, cell in enumerate(row):
           if cell.is_merged_cell:
               dump_merged_cell(i, j, cell)
   ```

#### Εξήγηση
- **`is_merged_cell`:** Ελέγχει αν το κελί αποτελεί μέρος ενός συγχωνευμένου συνόλου.
- **`row_span` και `col_span`:** Υποδείξτε πόσες γραμμές ή στήλες εκτείνεται το συγχωνευμένο κελί.
- **`first_row_index` και `first_column_index`:** Παρέχετε την αρχική θέση της συγχώνευσης.

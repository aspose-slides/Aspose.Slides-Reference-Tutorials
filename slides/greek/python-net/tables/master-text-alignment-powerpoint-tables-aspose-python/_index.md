## Οδηγός Εφαρμογής

Θα αναλύσουμε τη διαδικασία κάθετης στοίχισης κειμένου μέσα σε κελιά πίνακα σε διαχειρίσιμα βήματα.

### Πρόσβαση στη διαφάνεια και προσθήκη πίνακα

Αρχικά, πρέπει να έχουμε πρόσβαση σε μια διαφάνεια και να ορίσουμε τις διαστάσεις του πίνακά μας:

```python
with slides.Presentation() as presentation:
    slide = presentation.slides[0]
    dbl_cols = [120, 120, 120, 120]
    dbl_rows = [100, 100, 100, 100]

    # Προσθέστε τον πίνακα στη διαφάνεια.
    tbl = slide.shapes.add_table(100, 50, dbl_cols, dbl_rows)
```

### Εισαγωγή και Στοίχιση Κειμένου

Στη συνέχεια, εισαγάγετε κείμενο σε κελιά και εφαρμόστε κάθετη στοίχιση:

```python
# Εισαγωγή κειμένου σε συγκεκριμένα κελιά.
tbl.rows[1][0].text_frame.text = "10"
tbl.rows[2][0].text_frame.text = "20"
tbl.rows[3][0].text_frame.text = "30"

# Αποκτήστε πρόσβαση στο πλαίσιο κειμένου του πρώτου κελιού για να τροποποιήσετε τις ιδιότητες.
text_frame = tbl.rows[0][0].text_frame
paragraph = text_frame.paragraphs[0]
portion = paragraph.portions[0]

# Ορίστε κείμενο και στυλ για αυτό το τμήμα.
portion.text = "Text here"
portion.portion_format.fill_format.fill_type = slides.FillType.SOLID
portion.portion_format.fill_format.solid_fill_color.color = drawing.Color.black

# Ευθυγραμμίστε το κείμενο κάθετα.
cell = tbl.rows[0][0]
cell.text_anchor_type = slides.TextAnchorType.CENTER
cell.text_vertical_type = slides.TextVerticalType.VERTICAL270
```

### Αποθήκευση της παρουσίασής σας

Τέλος, αποθηκεύστε την τροποποιημένη παρουσίασή σας:

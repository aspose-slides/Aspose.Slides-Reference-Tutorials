## Οδηγός Εφαρμογής (H2)
### Πρόσβαση σε τιμές φόντου διαφάνειας
Αυτή η λειτουργία σάς επιτρέπει να έχετε πρόσβαση και να εκτυπώνετε τις ενεργές τιμές φόντου μιας διαφάνειας στην παρουσίαση του PowerPoint. Δείτε πώς μπορείτε να την εφαρμόσετε βήμα προς βήμα:

#### Βήμα 1: Ανοίξτε το αρχείο παρουσίασης
Χρησιμοποιώντας το Aspose.Slides, ανοίξτε το αρχείο παρουσίασής σας με το `Presentation` τάξη.

```python
import aspose.slides as slides

def get_background_effective_values():
    # Διαδρομή προς τον κατάλογο εγγράφων σας
    document_directory = "YOUR_DOCUMENT_DIRECTORY/"
    
    # Άνοιγμα αρχείου παρουσίασης
    with slides.Presentation(document_directory + "background.pptx") as pres:
        # Συνέχεια επεξεργασίας...
```

#### Βήμα 2: Πρόσβαση στο ενεργό φόντο της πρώτης διαφάνειας
Ανακτήστε τις ισχύουσες ιδιότητες φόντου της πρώτης διαφάνειας.

```python
        # Πρόσβαση στο ενεργό φόντο της πρώτης διαφάνειας
        effective_background = pres.slides[0].background.get_effective()
```

#### Βήμα 3: Έλεγχος και εκτύπωση τύπου και χρώματος γεμίσματος
Προσδιορίστε εάν ο τύπος πλήρωσης είναι `SOLID` και εκτυπώστε τις σχετικές πληροφορίες ανάλογα.

```python
        # Ελέγξτε τον τύπο συμπλήρωσης και εκτυπώστε τις σχετικές πληροφορίες
        if effective_background.fill_format.fill_type == slides.FillType.SOLID:
            # Εκτύπωση συμπαγούς χρώματος γεμίσματος
            print("Fill color: " + str(effective_background.fill_format.solid_fill_color))
        else:
            # Εκτύπωση του τύπου γεμίσματος
            print("Fill type: " + str(effective_background.fill_format.fill_type))

# Κλήση συνάρτησης για εκτέλεση
get_background_effective_values()
```

### Παράμετροι και Σκοποί Μεθόδου
- `slides.Presentation`: Ανοίγει ένα αρχείο PowerPoint.
- `pres.slides[0].background.get_effective()`Ανακτά τις ισχύουσες ιδιότητες φόντου της πρώτης διαφάνειας.
- `fill_type` και `solid_fill_color`: Χρησιμοποιείται για τον προσδιορισμό και την εμφάνιση του τύπου και του χρώματος γεμίσματος της διαφάνειας.

## Οδηγός Εφαρμογής

Τώρα, ας εφαρμόσουμε τον ορισμό προεπιλεγμένων γραμματοσειρών για κανονικό και ασιατικό κείμενο.

### Ορισμός προεπιλεγμένων γραμματοσειρών

Αυτή η λειτουργία σάς επιτρέπει να ορίσετε ποιες γραμματοσειρές θα χρησιμοποιηθούν όταν δεν έχει καθοριστεί γραμματοσειρά στο ίδιο το περιεχόμενο της παρουσίασης.

#### Βήμα 1: Δημιουργία LoadOptions

Ξεκινήστε ορίζοντας `LoadOptions` για να καθορίσετε τις παραμέτρους φόρτωσής σας:

```python
load_options = slides.LoadOptions()
load_options.load_format = slides.LoadFormat.AUTO
```

Αυτό υποδεικνύει στο Aspose.Slides πώς να ερμηνεύσει αυτόματα τη μορφή αρχείου.

#### Βήμα 2: Καθορισμός προεπιλεγμένων γραμματοσειρών

Στη συνέχεια, ορίστε τόσο την κανονική όσο και την ασιατική γραμματοσειρά. Σε αυτό το παράδειγμα, χρησιμοποιούμε "Wingdings" για λόγους απλότητας:

```python
load_options.default_regular_font = "Wingdings"
load_options.default_asian_font = "Wingdings"
```

Αυτό διασφαλίζει τη συνέπεια σε όλο το κείμενο της παρουσίασής σας.

#### Βήμα 3: Φόρτωση της παρουσίασης

Αφού ορίσετε τις επιλογές σας, φορτώστε το αρχείο PowerPoint χρησιμοποιώντας αυτές τις παραμέτρους:

```python
with slides.Presentation("YOUR_DOCUMENT_DIRECTORY/text_default_fonts.pptx", load_options) as pptx:
    # Δημιουργήστε μια μικρογραφία διαφάνειας και αποθηκεύστε την ως PNG
    pptx.slides[0].get_image(1, 1).save("YOUR_OUTPUT_DIRECTORY/text_default_fonts_out.png", slides.ImageFormat.PNG)
    
    # Αποθήκευση της παρουσίασης σε μορφή PDF
    pptx.save("YOUR_OUTPUT_DIRECTORY/text_default_fonts_out.pdf", slides.export.SaveFormat.PDF)
    
    # Επιπλέον, αποθηκεύστε το ως αρχείο XPS
    pptx.save("YOUR_OUTPUT_DIRECTORY/text_default_fonts_out.xps", slides.export.SaveFormat.XPS)
```

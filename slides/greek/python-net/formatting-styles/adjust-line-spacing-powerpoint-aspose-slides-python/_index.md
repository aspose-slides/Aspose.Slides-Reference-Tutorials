#### Βήμα 3: Τροποποίηση απόστασης παραγράφων

Προσαρμόστε τις ιδιότητες απόστασης γραμμών για τις παραγράφους:

```python
def adjust_line_spacing():
    input_path = 'YOUR_DOCUMENT_DIRECTORY/text_fonts.pptx'
    output_path = 'YOUR_OUTPUT_DIRECTORY/text_line_spacing_out.pptx'

    with slides.Presentation(input_path) as presentation:
        slide = presentation.slides[0]
        tf1 = slide.shapes[0].text_frame

        # Πρόσβαση στην πρώτη παράγραφο στο πλαίσιο κειμένου
        para1 = tf1.paragraphs[0]

        # Προσαρμογή ιδιοτήτων διάστιχου της παραγράφου
        para1.paragraph_format.space_within = 80  # Χώρος εντός γραμμών
        para1.paragraph_format.space_before = 40   # Κενό πριν από την παράγραφο
        para1.paragraph_format.space_after = 40    # Κενό μετά την παράγραφο

        pass  # Αποθήκευση αλλαγών στη συνέχεια
```

#### Βήμα 4: Αποθήκευση της τροποποιημένης παρουσίασης

Αποθηκεύστε την παρουσίασή σας με ενημερωμένες ρυθμίσεις:

```python
def adjust_line_spacing():
    input_path = 'YOUR_DOCUMENT_DIRECTORY/text_fonts.pptx'
    output_path = 'YOUR_OUTPUT_DIRECTORY/text_line_spacing_out.pptx'

    with slides.Presentation(input_path) as presentation:
        slide = presentation.slides[0]
        tf1 = slide.shapes[0].text_frame
        para1 = tf1.paragraphs[0]

        para1.paragraph_format.space_within = 80  
        para1.paragraph_format.space_before = 40   
        para1.paragraph_format.space_after = 40    

        # Αποθήκευση της τροποποιημένης παρουσίασης σε νέο αρχείο
        presentation.save(output_path, slides.export.SaveFormat.PPTX)
```

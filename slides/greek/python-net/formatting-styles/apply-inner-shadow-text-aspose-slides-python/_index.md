## Οδηγός Εφαρμογής
Τώρα ας επικεντρωθούμε στην εφαρμογή ενός εφέ εσωτερικής σκιάς σε κείμενο σε μια διαφάνεια του PowerPoint.
### Προσθήκη κειμένου με εφέ εσωτερικής σκιάς
#### Επισκόπηση
Θα δημιουργήσουμε ένα ορθογώνιο σχήμα, θα προσθέσουμε κείμενο σε αυτό και, στη συνέχεια, θα εφαρμόσουμε το εφέ εσωτερικής σκιάς. Αυτή η μέθοδος βελτιώνει την αισθητική των διαφανειών σας προσθέτοντας βάθος στο κείμενο.
#### Οδηγός βήμα προς βήμα
**1. Πρόσβαση στη διαφάνεια**
Αρχικά, βρείτε μια αναφορά στην πρώτη διαφάνεια της παρουσίασής σας:

```python
slide = total_presentation.slides[0]
```
**2. Προσθήκη Αυτόματου Σχήματος**
Προσθέστε ένα ορθογώνιο σχήμα για να κρατήσετε το κείμενό μας:

```python
auto_shape = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 150, 75, 400, 300)
auto_shape.fill_format.fill_type = slides.FillType.NO_FILL
```
**3. Εισαγωγή κειμένου**
Εισαγάγετε ένα πλαίσιο κειμένου και ορίστε το περιεχόμενο για το ορθογώνιό σας:

```python
auto_shape.add_text_frame("Aspose TextBox")
port = auto_shape.text_frame.paragraphs[0].portions[0]
pf = port.portion_format
pf.font_height = 50  # Ορίστε το μέγεθος της γραμματοσειράς για να βελτιώσετε την ορατότητα
```
**4. Εφαρμογή εφέ εσωτερικής σκιάς**
Ενεργοποίηση και διαμόρφωση του εφέ εσωτερικής σκιάς στο κείμενο:

```python
ef = pf.effect_format
ef.enable_inner_shadow_effect()
# Διαμορφώστε τις παραμέτρους εσωτερικής σκιάς
ef.inner_shadow_effect.blur_radius = 8.0  # Ακτίνα θολώματος για πιο απαλή σκιά
ef.inner_shadow_effect.direction = 90.0  # Κατεύθυνση σκιάς σε μοίρες
ef.inner_shadow_effect.distance = 6.0    # Απόσταση σκιάς από κείμενο
ef.inner_shadow_effect.shadow_color.b = 189  # Μπλε στοιχείο του χρώματος σκιάς
# Ορίστε συνεπή θέματα χρησιμοποιώντας χρώματα συνδυασμού
ef.inner_shadow_effect.shadow_color.color_type = slides.ColorType.SCHEME
ef.inner_shadow_effect.shadow_color.scheme_color = slides.SchemeColor.ACCENT1
```
**5. Αποθήκευση της παρουσίασής σας**
Τέλος, αποθηκεύστε την παρουσίασή σας σε ένα αρχείο:

```python
presentation.save("YOUR_OUTPUT_DIRECTORY/text_apply_inner_shadow_out.pptx")
```

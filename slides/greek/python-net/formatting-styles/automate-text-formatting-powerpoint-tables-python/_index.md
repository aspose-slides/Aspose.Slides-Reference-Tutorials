## Οδηγός Εφαρμογής
Ας αναλύσουμε τη διαδικασία μορφοποίησης κειμένου μέσα σε στήλες πίνακα σε διαχειρίσιμα βήματα.
### Βήμα 1: Άνοιγμα και πρόσβαση σε έναν πίνακα στην παρουσίασή σας
Ξεκινήστε ανοίγοντας το αρχείο PowerPoint και αποκτώντας πρόσβαση στον πρώτο πίνακα στην πρώτη διαφάνεια:
```python
def apply_text_formatting_to_table_columns():
    input_path = "YOUR_DOCUMENT_DIRECTORY/tables.pptx"
    
    # Φόρτωση μιας υπάρχουσας παρουσίασης που περιέχει έναν πίνακα
    with slides.Presentation(input_path) as pres:
        # Πρόσβαση στο πρώτο σχήμα (υποτίθεται ότι είναι πίνακας) στην πρώτη διαφάνεια
        table = pres.slides[0].shapes[0]
```
**Εξήγηση:**
Εδώ, ανοίγουμε ένα αρχείο PowerPoint και υποθέτουμε ότι το πρώτο σχήμα στην πρώτη διαφάνεια είναι ο επιθυμητός πίνακας. Αυτή η ρύθμιση μας επιτρέπει να εφαρμόσουμε απευθείας αλλαγές μορφοποίησης.
### Βήμα 2: Ορισμός ύψους γραμματοσειράς για τα κελιά στην πρώτη στήλη
Για να τροποποιήσετε την εμφάνιση του κειμένου, όπως το ύψος της γραμματοσειράς, χρησιμοποιήστε `PortionFormat`:
```python
# Ορισμός ύψους γραμματοσειράς για τα κελιά στην πρώτη στήλη
portion_format = slides.PortionFormat()
portion_format.font_height = 25
table.columns[0].set_text_format(portion_format)
```
**Εξήγηση:**
Αυτό το απόσπασμα εφαρμόζει ένα ομοιόμορφο μέγεθος γραμματοσειράς 25 στιγμών σε όλο το κείμενο εντός της πρώτης στήλης, βελτιώνοντας την αναγνωσιμότητα.
### Βήμα 3: Στοίχιση κειμένου και ορισμός περιθωρίων
Η προσαρμογή της ευθυγράμμισης και των περιθωρίων είναι ζωτικής σημασίας για άψογες παρουσιάσεις:
```python
# Στοίχιση κειμένου προς τα δεξιά και ορισμός περιθωρίου για τα κελιά στην πρώτη στήλη
paragraph_format = slides.ParagraphFormat()
paragraph_format.alignment = slides.TextAlignment.RIGHT
paragraph_format.margin_right = 20
table.columns[0].set_text_format(paragraph_format)
```
**Εξήγηση:**
Η ευθυγράμμιση κειμένου προς τα δεξιά με περιθώριο 20 σημείων δημιουργεί μια καθαρή και επαγγελματική εμφάνιση, ιδιαίτερα χρήσιμη για στήλες με αριθμητικά δεδομένα ή βασικά σημεία.
### Βήμα 4: Ορίστε την κατακόρυφη στοίχιση κειμένου στη δεύτερη στήλη
Για δημιουργικές παρουσιάσεις, η κάθετη στοίχιση κειμένου μπορεί να είναι ένα εντυπωσιακό χαρακτηριστικό:
```python
# Ορισμός κατακόρυφης στοίχισης κειμένου για κελιά στη δεύτερη στήλη
text_frame_format = slides.TextFrameFormat()
text_frame_format.text_vertical_type = slides.TextVerticalType.VERTICAL
table.columns[1].set_text_format(text_frame_format)
```
**Εξήγηση:**
Αυτή η διαμόρφωση περιστρέφει το κείμενο σε κατακόρυφο προσανατολισμό, ιδανικό για κεφαλίδες ή ειδικές ενότητες μέσα στον πίνακά σας.
### Βήμα 5: Αποθήκευση της παρουσίασης
Τέλος, αποθηκεύστε όλες τις αλλαγές για να δημιουργήσετε μια νέα έκδοση της παρουσίασής σας:
```python
# Αποθήκευση της παρουσίασης με τις εφαρμοσμένες αλλαγές μορφοποίησης
output_path = "YOUR_OUTPUT_DIRECTORY/tables_text_format_inside_column_out.pptx"
pres.save(output_path, slides.export.SaveFormat.PPTX)
```
**Εξήγηση:**
Η αποθήκευση της εργασίας σας διασφαλίζει ότι όλες οι τροποποιήσεις διατηρούνται και μπορούν εύκολα να κοινοποιηθούν ή να παρουσιαστούν.

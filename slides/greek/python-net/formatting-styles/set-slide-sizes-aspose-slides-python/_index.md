## Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, θα δούμε τα βήματα για τον ορισμό μεγεθών διαφανειών χρησιμοποιώντας το Aspose.Slides.

### Ρύθμιση μεγέθους διαφάνειας με προσαρμογή περιεχομένου

Για να διασφαλίσετε ότι το περιεχόμενό σας ταιριάζει σε συγκεκριμένες διαστάσεις χωρίς να αλλοιωθεί η αναλογία διαστάσεων, χρησιμοποιήστε το `set_size` μέθοδος με `ENSURE_FIT`Αυτό εγγυάται ότι όλα τα στοιχεία στη διαφάνεια είναι ορατά στο προβλεπόμενο μέγεθός τους.

#### Βήμα προς βήμα εφαρμογή:
1. **Εισαγωγή Aspose.Slides**:
   ```python
   import aspose.slides as slides
   ```
2. **Φόρτωση της παρουσίασής σας**:
   Καθορίστε τη διαδρομή προς το έγγραφό σας και τα αρχεία εξόδου.
   
   ```python
document_path = 'Ο ΚΑΤΑΛΟΓΟΣ_ΕΓΓΡΑΦΩΝ_ΣΑΣ/Καλώς ορίσατε-στο-powerpoint.pptx'
διαδρομή_εξόδου = 'ΚΑΤΑΛΟΓΟΣ_ΕΞΟΔΟΥ_ΣΑΣ/διάταξη_διαφάνειας_μεγέθους_κλίμακας_εξόδου.pptx'
```
3. **Adjust Slide Size for Content Fit**:
   Access the first slide and set its size.

   ```python
   with slides.Presentation(document_path) as presentation:
       # Ensure content fits within 540x720 dimensions
       presentation.slide_size.set_size(540, 720, slides.SlideSizeScaleType.ENSURE_FIT)
   ```
### Ρύθμιση μεγέθους διαφάνειας σε A4 και μεγιστοποίηση περιεχομένου
Για παρουσιάσεις που απαιτούν τήρηση μορφών χαρτιού όπως A4, μεγιστοποιώντας παράλληλα την ορατότητα του περιεχομένου:

1. **Ορισμός μεγέθους διαφάνειας σε A4**:

   ```python
   with slides.Presentation(document_path) as presentation:
       # Ορίστε το μέγεθος της διαφάνειας σε μορφή A4 και μεγιστοποιήστε το περιεχόμενο μέσα σε αυτήν
       presentation.slide_size.set_size(slides.SlideSizeType.A4_PAPER, slides.SlideSizeScaleType.MAXIMIZE)
   ```
2. **Αποθήκευση της παρουσίασης**:

   ```python
   with slides.Presentation() as aux_presentation:
       # Αποθηκεύστε απευθείας τις τροποποιήσεις σε ένα νέο αρχείο
       aux_presentation.save(output_path, slides.export.SaveFormat.PPTX)
   ```
### Επεξήγηση παραμέτρων
- `set_size(width, height, scale_type)`: Προσαρμόζει τις διαστάσεις της διαφάνειας. `scale_type` καθορίζει τον τρόπο με τον οποίο προσαρμόζεται το περιεχόμενο.
  - `slides.SlideSizeScaleType.ENSURE_FIT`: Εξασφαλίζει ότι όλο το περιεχόμενο χωράει εντός του καθορισμένου πλάτους και ύψους χωρίς κλιμάκωση πέρα από το δεδομένο μέγεθος.
  - `slides.SlideSizeScaleType.MAXIMIZE`: Μεγιστοποιεί το περιεχόμενο για να γεμίσει την περιοχή της διαφάνειας όσο το δυνατόν περισσότερο.

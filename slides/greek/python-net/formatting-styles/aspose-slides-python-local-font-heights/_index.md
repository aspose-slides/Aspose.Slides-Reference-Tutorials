## Οδηγός Εφαρμογής
### Ορισμός τοπικών υψών γραμματοσειράς σε πλαίσια κειμένου
Αυτή η λειτουργία σάς επιτρέπει να προσαρμόσετε τμήματα κειμένου μέσα σε ένα μόνο πλαίσιο—ιδανικό για να τονίσετε συγκεκριμένα μέρη της παρουσίασής σας.
#### Επισκόπηση
Τροποποιώντας τοπικά τα ύψη των γραμματοσειρών, μπορείτε να επιστήσετε την προσοχή σε βασικές φράσεις ή ενότητες χωρίς να αλλάξετε τη συνολική διάταξη. Αυτό το σεμινάριο καλύπτει τον ορισμό διαφορετικών υψών για διάφορα τμήματα μιας παραγράφου.
#### Βήματα Υλοποίησης
##### Βήμα 1: Αρχικοποίηση παρουσίασης και προσθήκη σχήματος
Ξεκινήστε δημιουργώντας μια νέα παρουσίαση και προσθέτοντας ένα σχήμα όπου θα βρίσκεται το κείμενό σας:
```python
def set_local_font_height_values():
    with slides.Presentation() as pres:
        # Προσθήκη ορθογωνίου σχήματος στην πρώτη διαφάνεια
        new_shape = pres.slides[0].shapes.add_auto_shape(
            slides.ShapeType.RECTANGLE, 100, 100, 400, 75, False)
```
Εδώ, προσθέτουμε ένα ορθογώνιο σχήμα με καθορισμένες συντεταγμένες και διαστάσεις.
##### Βήμα 2: Δημιουργία πλαισίου κειμένου
Στη συνέχεια, δημιουργήστε ένα κενό πλαίσιο κειμένου μέσα στο σχήμα που μόλις προσθέσατε:
```python
        # Δημιουργία ενός κενού πλαισίου κειμένου
        new_shape.add_text_frame("")
        new_shape.text_frame.paragraphs[0].portions.clear()
```
Η εκκαθάριση των υπαρχόντων τμημάτων διασφαλίζει μια καθαρή βάση για την προσθήκη προσαρμοσμένου κειμένου.
##### Βήμα 3: Προσθήκη και προσαρμογή τμημάτων κειμένου
Προσθέστε δύο ξεχωριστά τμήματα κειμένου στην παράγραφο σας και, στη συνέχεια, προσαρμόστε τα ύψη γραμματοσειράς τους:
```python
        # Προσθήκη τμημάτων κειμένου με διαφορετικά ύψη
        portion0 = slides.Portion("Sample text with first portion")
        portion1 = slides.Portion(" and second portion.")
        
        new_shape.text_frame.paragraphs[0].portions.add(portion0)
        new_shape.text_frame.paragraphs[0].portions.add(portion1)

        # Ρύθμιση ύψους γραμματοσειράς
        pres.default_text_style.get_level(0).default_portion_format.font_height = 24
        new_shape.text_frame.paragraphs[0].paragraph_format.default_portion_format.font_height = 40
        
        new_shape.text_frame.paragraphs[0].portions[0].portion_format.font_height = 55
        new_shape.text_frame.paragraphs[0].portions[1].portion_format.font_height = 18
```
Ο `font_height` Η παράμετρος είναι κρίσιμη για τον ορισμό της οπτικής προβολής κάθε τμήματος.
##### Βήμα 4: Αποθήκευση της παρουσίασης
Τέλος, αποθηκεύστε την παρουσίασή σας:
```python
        # Αποθήκευση σε καθορισμένο κατάλογο
        pres.save("YOUR_OUTPUT_DIRECTORY/text_SetLocalFontHeightValues_out.pptx", slides.export.SaveFormat.PPTX)
```

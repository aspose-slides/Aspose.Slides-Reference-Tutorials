## Οδηγός Εφαρμογής
Ας δούμε τη διαδικασία προσθήκης προσαρμοσμένου κειμένου κράτησης θέσης σε μια παρουσίαση του PowerPoint.

### Προσθήκη προσαρμοσμένου κειμένου κράτησης θέσης
Τροποποιήστε placeholders όπως τίτλους και υπότιτλους με προσαρμοσμένες οδηγίες ή κείμενο χρησιμοποιώντας το Aspose.Slides για Python.

#### Οδηγός βήμα προς βήμα
**Βήμα 1: Ορίστε τις διαδρομές σας**
Ορίστε διαδρομές προς τα αρχεία εισόδου και εξόδου. Αντικαταστήστε `'YOUR_DOCUMENT_DIRECTORY'` και `'YOUR_OUTPUT_DIRECTORY'` με πραγματικούς καταλόγους στο σύστημά σας.

```python
document_path = 'YOUR_DOCUMENT_DIRECTORY/text_add_custom_placeholder_text.pptx'
output_path = 'YOUR_OUTPUT_DIRECTORY/text_add_custom_placeholder_text_out.pptx'
```

**Βήμα 2: Ανοίξτε την παρουσίαση**
Ανοίξτε το αρχείο PowerPoint χρησιμοποιώντας το Aspose.Slides, αρχικοποιώντας ένα `Presentation` αντικείμενο.

```python
def add_custom_prompt_text():
    with slides.Presentation(document_path) as pres:
        slide = pres.slides[0]
```

**Βήμα 3: Επανάληψη σχημάτων διαφάνειας**
Περιηγηθείτε στα σχήματα στην πρώτη σας διαφάνεια και ελέγξτε για δεσμευτικά θέσης.

```python
for shape in slide.shapes:
    if isinstance(shape, slides.AutoShape) and shape.placeholder is not None:
        text = ''
        # Ελέγξτε τον τύπο κράτησης θέσης και ορίστε προσαρμοσμένο κείμενο ανάλογα
```

**Βήμα 4: Ορισμός προσαρμοσμένου κειμένου κράτησης θέσης**
Προσδιορίστε τον τύπο placeholder και αντιστοιχίστε το κατάλληλο προσαρμοσμένο κείμενο.

```python
if shape.placeholder.type == slides.PlaceholderType.CENTERED_TITLE:
    text = 'Click to add a custom title'
elif shape.placeholder.type == slides.PlaceholderType.SUBTITLE:
    text = 'Click to add a custom subtitle'

shape.text_frame.text = text
```

**Βήμα 5: Αποθήκευση της τροποποιημένης παρουσίασης**
Αφού τροποποιήσετε τα placeholder, αποθηκεύστε την παρουσίασή σας.

```python
pres.save(output_path, slides.export.SaveFormat.PPTX)
```

## Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, θα αναλύσουμε τη διαδικασία δημιουργίας ενός σχήματος ομάδας μέσα σε μια διαφάνεια παρουσίασης.

### Δημιουργία σχημάτων ομάδας σε διαφάνειες παρουσίασης

Αυτή η λειτουργία βοηθά στην οργάνωση πολλαπλών σχημάτων σε μια συνεκτική μονάδα για καλύτερη δομή και οπτική ελκυστικότητα.

#### Βήμα 1: Δημιουργία ή άνοιγμα παρουσίασης

Ξεκινήστε ανοίγοντας μια υπάρχουσα παρουσίαση ή δημιουργώντας μια νέα:

```python
def create_group_shape():
    with slides.Presentation() as pres:
        slide = pres.slides[0]
```

*Γιατί:* Χρησιμοποιούμε το `with` Δήλωση για τη διαχείριση περιβάλλοντος, διασφαλίζοντας ότι οι πόροι καθαρίζονται σωστά μετά τις λειτουργίες.

#### Βήμα 2: Πρόσβαση στη συλλογή σχημάτων

Αποκτήστε πρόσβαση στα σχήματα στην τρέχουσα διαφάνειά σας:

```python
shapes = slide.shapes
```

Αυτή η συλλογή μας επιτρέπει να χειριζόμαστε και να προσθέτουμε νέα σχήματα.

#### Βήμα 3: Προσθήκη σχήματος ομάδας

Προσθέστε ένα σχήμα ομάδας για να φιλοξενήσετε μεμονωμένα σχήματα:

```python
group_shape = shapes.add_group_shape()
```

*Γιατί:* Η ομαδοποίηση σχημάτων απλοποιεί τον χειρισμό, επιτρέποντάς σας να τα μετακινήσετε ή να τα τροποποιήσετε ως μία μονάδα.

#### Βήμα 4: Εισαγωγή μεμονωμένων σχημάτων

Προσθέστε ορθογώνια μέσα στο σχήμα της ομάδας σε καθορισμένες θέσεις:

```python
group_shape.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 300, 100, 100, 100)
group_shape.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 500, 100, 100, 100)
group_shape.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 300, 300, 100, 100)
group_shape.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 500, 300, 100, 100)
```

*Γιατί:* Αυτό το βήμα περιλαμβάνει την προσθήκη σχημάτων για την επίδειξη δυνατοτήτων ομαδοποίησης.

#### Βήμα 5: Προσθήκη πλαισίου

Τοποθετήστε ένα πλαίσιο γύρω από το σχήμα της ομάδας για οπτική οριοθέτηση:

```python
group_shape.frame = slides.ShapeFrame(
    100, 300, 500, 40,
    slides.NullableBool.TRUE,
    slides.NullableBool.TRUE,
    0
)
```

#### Βήμα 6: Αποθήκευση της παρουσίασης

Τέλος, αποθηκεύστε την παρουσίασή σας σε έναν καθορισμένο κατάλογο:

```python
pres.save("YOUR_OUTPUT_DIRECTORY/shapes_create_group_shape_out.pptx", slides.export.SaveFormat.PPTX)
```

*Γιατί:* Η αποθήκευση διασφαλίζει ότι όλες οι αλλαγές αποθηκεύονται και ότι είναι δυνατή η πρόσβαση σε αυτές αργότερα.

# Πώς να εφαρμόσετε γέμισμα με διαβάθμιση σε σχήματα στο PowerPoint χρησιμοποιώντας το Aspose.Slides για Python

Βελτιώστε την οπτική ελκυστικότητα των παρουσιάσεών σας στο PowerPoint εφαρμόζοντας γεμίσματα με διαβάθμιση σε σχήματα χρησιμοποιώντας το Aspose.Slides για Python. Αυτό το σεμινάριο σας καθοδηγεί στη διαδικασία, καθιστώντας την προσβάσιμη τόσο για αρχάριους όσο και για έμπειρους προγραμματιστές.

Ακολουθώντας αυτόν τον οδηγό, θα μάθετε πώς να:
- Ρύθμιση και εγκατάσταση του Aspose.Slides για Python
- Δημιουργήστε μια διαφάνεια με ελλειπτικό σχήμα
- Εφαρμόστε εφέ γεμίσματος με διαβάθμιση χρησιμοποιώντας απλά αποσπάσματα κώδικα
- Βελτιστοποιήστε την απόδοση της παρουσίασής σας

Ας ξεκινήσουμε διασφαλίζοντας ότι έχετε τις απαραίτητες προϋποθέσεις.

```python
import aspose.slides as slides
```

## Οδηγός Εφαρμογής

Αυτή η ενότητα περιγράφει τα βήματα για την προσθήκη διαβάθμισης γεμίσματος σε ένα ελλειπτικό σχήμα.

### Βήμα 1: Δημιουργία αρχικού στιγμιότυπου παρουσίασης

Δημιουργήστε μια παρουσία του `Presentation` τάξη:

```python
with slides.Presentation() as pres:
    # Οι λειτουργίες διαφανειών πηγαίνουν εδώ
```

Αυτό διασφαλίζει την αποτελεσματική διαχείριση των πόρων.

### Βήμα 2: Πρόσβαση ή δημιουργία διαφάνειας

Αποκτήστε πρόσβαση στην πρώτη διαφάνεια, δημιουργώντας μία, εάν είναι απαραίτητο:

```python
slide = pres.slides[0]
```

### Βήμα 3: Προσθέστε ένα ελλειπτικό σχήμα

Προσθέστε ένα σχήμα έλλειψης στη διαφάνειά σας:

```python
shape = slide.shapes.add_auto_shape(slides.ShapeType.ELLIPSE, 50, 150, 75, 150)
```

- `ShapeType.ELLIPSE` καθορίζει τον τύπο του σχήματος.
- Οι παράμετροι (50, 150, 75, 150) καθορίζουν τη θέση και το μέγεθος της έλλειψης.

### Βήμα 4: Εφαρμογή γεμίσματος με διαβάθμιση στο σχήμα

Διαμορφώστε το γέμισμα με διαβάθμιση:

```python
shape.fill_format.fill_type = slides.FillType.GRADIENT
shape.fill_format.gradient_format.gradient_shape = slides.GradientShape.LINEAR
shape.fill_format.gradient_format.gradient_direction = slides.GradientDirection.FROM_CORNER2
```

- **Τύπος πλήρωσης**: Ορισμός σε `GRADIENT`.
- **Σχήμα και κατεύθυνση διαβάθμισης**Αυτά καθορίζουν το στυλ και την κατεύθυνση του γεμίσματος με διαβάθμιση.

### Βήμα 5: Προσθήκη διαβαθμίσεων

Ορίστε δύο στάσεις διαβάθμισης για τη μετάβαση χρώματος:

```python
shape.fill_format.gradient_format.gradient_stops.add(1.0, slides.PresetColor.PURPLE)
shape.fill_format.gradient_format.gradient_stops.add(0, slides.PresetColor.RED)
```

- `1.0` και `0` είναι οι θέσεις των στοπ κλίσης.
- `PresetColor.PURPLE` και `PresetColor.RED` ορίστε τα χρώματα.

### Βήμα 6: Αποθηκεύστε την παρουσίασή σας

Αποθηκεύστε την τροποποιημένη παρουσίασή σας:

```python
pres.save(global_opts.out_dir + "shapes_fill_gradient_out.pptx", slides.export.SaveFormat.PPTX)
```

Αυτό εγγράφει τις αλλαγές σας σε ένα νέο αρχείο με το όνομα `shapes_fill_gradient_out.pptx`.

### Προσθήκη και μορφοποίηση κειμένου στο PowerPoint

#### Επισκόπηση
Δημιουργήστε μια παρουσίαση που περιέχει μία διαφάνεια με σχήμα ορθογωνίου στην οποία θα εισάγουμε τρεις μορφοποιημένες παραγράφους.

#### Βήμα 1: Δημιουργήστε μια παρουσίαση
Ρυθμίστε την παρουσίαση και αποκτήστε πρόσβαση στην πρώτη της διαφάνεια:

```python
import aspose.slides as slides
import aspose.pydrawing as drawing

def add_multiple_paragraphs():
    # Δημιουργήστε μια κλάση παρουσίασης που αντιπροσωπεύει ένα αρχείο PPTX
    with slides.Presentation() as pres:
        # Πρόσβαση στην πρώτη διαφάνεια
        slide = pres.slides[0]
```

#### Βήμα 2: Προσθήκη Αυτόματου Σχήματος
Προσθέστε ένα ορθογώνιο σχήμα για να συγκρατήσετε το κείμενό σας:

```python
        # Προσθήκη Αυτόματου Σχήματος τύπου Ορθογώνιου
        auto_shape = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 50, 150, 300, 150)
        
        # Access TextFrame του AutoShape
        tf = auto_shape.text_frame
```

#### Βήμα 3: Δημιουργία παραγράφων και τμημάτων
Δημιουργήστε παραγράφους με διαφορετικές μορφές κειμένου:

```python
        # Δημιουργήστε την πρώτη παράγραφο με δύο μέρη
        para0 = tf.paragraphs[0]
        port01 = slides.Portion()
        port02 = slides.Portion()
        para0.portions.add(port01)
        para0.portions.add(port02)

        # Προσθήκη δεύτερης παραγράφου με τρία μέρη
        para1 = slides.Paragraph()
        tf.paragraphs.add(para1)
        port10 = slides.Portion()
        port11 = slides.Portion()
        port12 = slides.Portion()
        para1.portions.add(port10)
        para1.portions.add(port11)
        para1.portions.add(port12)

        # Προσθήκη τρίτης παραγράφου με τρία μέρη
        para2 = slides.Paragraph()
        tf.paragraphs.add(para2)
        port20 = slides.Portion()
        port21 = slides.Portion()
        port22 = slides.Portion()
        para2.portions.add(port20)
        para2.portions.add(port21)
        para2.portions.add(port22)
```

#### Βήμα 4: Εφαρμογή μορφοποίησης σε τμήματα
Επανάληψη παραγράφων και τμημάτων για μορφοποίηση κειμένου:

```python
        # Επανάληψη παραγράφων και τμημάτων για ορισμό κειμένου και μορφοποίησης
        for i in range(3):
            for j in range(3):
                tf.paragraphs[i].portions[j].text = 'Portion0' + str(j)
                
                # Εφαρμόστε κόκκινο χρώμα, έντονη γραμματοσειρά και ύψος 15 στο πρώτο μέρος κάθε παραγράφου
                if j == 0:
                    tf.paragraphs[i].portions[j].portion_format.fill_format.fill_type = slides.FillType.SOLID
                    tf.paragraphs[i].portions[j].portion_format.fill_format.solid_fill_color.color = drawing.Color.red
                    tf.paragraphs[i].portions[j].portion_format.font_bold = slides.NullableBool.TRUE
                    tf.paragraphs[i].portions[j].portion_format.font_height = 15
                
                # Εφαρμογή μπλε χρώματος, πλάγιας γραμματοσειράς και ύψους 18 στο δεύτερο μέρος κάθε παραγράφου
                elif j == 1:
                    tf.paragraphs[i].portions[j].portion_format.fill_format.fill_type = slides.FillType.SOLID
                    tf.paragraphs[i].portions[j].portion_format.fill_format.solid_fill_color.color = drawing.Color.blue
                    tf.paragraphs[i].portions[j].portion_format.font_italic = slides.NullableBool.TRUE
                    tf.paragraphs[i].portions[j].portion_format.font_height = 18
        
        # Αποθήκευση της παρουσίασης σε δίσκο σε μορφή PPTX
        pres.save('YOUR_OUTPUT_DIRECTORY/text_multiple_paragraphs_out.pptx', slides.export.SaveFormat.PPTX)
```

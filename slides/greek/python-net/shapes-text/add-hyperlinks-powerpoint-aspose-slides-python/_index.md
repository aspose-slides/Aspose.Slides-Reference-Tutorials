## Οδηγός Εφαρμογής
Θα αναλύσουμε την προσθήκη υπερσυνδέσμων σε διαφάνειες του PowerPoint σε βήματα.

### Προσθήκη αυτόματου σχήματος και πλαισίου κειμένου
Αρχικά, χρειαζόμαστε ένα σχήμα στη διαφάνειά μας για το κείμενο. Δείτε πώς μπορείτε να το προσθέσετε:

#### Βήμα 1: Δημιουργία αντικειμένου παρουσίασης
```python
with slides.Presentation() as presentation:
    # Ο κωδικός σας θα μπει εδώ
```
Αυτό αρχικοποιεί μια νέα παρουσίαση PowerPoint.

#### Βήμα 2: Προσθήκη αυτόματου σχήματος
Προσθέστε ένα ορθογώνιο σχήμα με κείμενο:
```python
shape1 = presentation.slides[0].shapes.add_auto_shape(
    slides.ShapeType.RECTANGLE, 100, 100, 600, 50, False)
```
Οι παράμετροι περιλαμβάνουν τη θέση και το μέγεθος του σχήματος.

#### Βήμα 3: Προσθήκη κειμένου στο σχήμα
Εισαγάγετε το κείμενο που θέλετε στο σχήμα:
```python
shape1.add_text_frame("Aspose: File Format APIs")
```

### Ορισμός υπερσυνδέσμου σε κείμενο
Τώρα, κάντε αυτό το κείμενο να είναι κλικάριστο προσθέτοντας έναν υπερσύνδεσμο.

#### Βήμα 4: Αντιστοίχιση υπερσυνδέσμου
Συνδέστε το κείμενο με μια διεύθυνση URL:
```python
shape1.text_frame.paragraphs[0].portions[0].portion_format.hyperlink_click = slides.Hyperlink(
    "https://www.aspose.com/")
```
Αυτό το απόσπασμα κώδικα μετατρέπει το πρώτο μέρος της πρώτης παραγράφου σε υπερσύνδεσμο.

#### Βήμα 5: Προσθήκη συμβουλής εργαλείου για υπερσύνδεσμο
Παρέχετε πρόσθετες πληροφορίες μέσω tooltip:
```python
shape1.text_frame.paragraphs[0].portions[0].portion_format.hyperlink_click.tooltip = \\
    "More than 70% Fortune 100 companies trust Aspose APIs"
```

### Προσαρμογή εμφάνισης κειμένου
Προσαρμόστε την εμφάνιση για να την κάνετε πιο εμφανή.

#### Βήμα 6: Ορισμός μεγέθους γραμματοσειράς
Αυξήστε το μέγεθος της γραμματοσειράς για καλύτερη ορατότητα:
```python
shape1.text_frame.paragraphs[0].portions[0].portion_format.font_height = 32
```

### Αποθήκευση της παρουσίασής σας
Τέλος, αποθηκεύστε την παρουσίασή σας με όλες τις αλλαγές που έχουν εφαρμοστεί.
```python\presentation.save("YOUR_OUTPUT_DIRECTORY/hyperlink_add_hyperlink_out.pptx")
```
Αντικαθιστώ `YOUR_OUTPUT_DIRECTORY` με την πραγματική διαδρομή όπου θέλετε να αποθηκεύσετε το αρχείο.

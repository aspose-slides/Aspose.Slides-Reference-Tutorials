```python
class PresentationDemo:

```

**Εξήγηση:**

### Προσθήκη αυτόματου σχήματος στη διαφάνεια

#### Επισκόπηση:
Προσθέστε ένα ορθογώνιο σχήμα στη διαφάνειά σας όπου μπορείτε να εφαρμόσετε προσαρμοσμένη μορφοποίηση γραμμής.

```python
# Λήψη της πρώτης διαφάνειας από την παρουσίαση
class PresentationDemo:
    def format_lines(self):
        with self.presentation as pres:
            slide = pres.slides[0]

            # Προσθήκη ενός αυτόματου σχήματος ορθογωνίου τύπου στη διαφάνεια
            shape = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 50, 150, 150, 75)
```

**Εξήγηση:**
- `add_auto_shape()` Η μέθοδος χρησιμοποιείται για την εισαγωγή ενός νέου σχήματος. Εδώ, το καθορίζουμε ως ορθογώνιο και παρέχουμε παραμέτρους θέσης και μεγέθους.

### Μορφοποίηση του στυλ γραμμής του σχήματος

#### Επισκόπηση:
Εφαρμόστε ένα στυλ παχιάς-λεπτής γραμμής με προσαρμοσμένο πλάτος και μοτίβο παύλας για να βελτιώσετε την εμφάνιση του σχήματός σας.

```python
class PresentationDemo:
    def format_lines(self):
        with self.presentation as pres:
            slide = pres.slides[0]
            shape = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 50, 150, 150, 75)

            # Ορίστε το χρώμα γεμίσματος του ορθογωνίου σε λευκό
            shape.fill_format.fill_type = slides.FillType.SOLID
            shape.fill_format.solid_fill_color.color = drawing.Color.white

            # Εφαρμογή στυλ γραμμής με συγκεκριμένο πλάτος και στυλ παύλας
            shape.line_format.style = slides.LineStyle.THICK_THIN
            shape.line_format.width = 7
            shape.line_format.dash_style = slides.LineDashStyle.DASH

            # Ορίστε το χρώμα του περιγράμματος του ορθογωνίου σε μπλε
            shape.line_format.fill_format.fill_type = slides.FillType.SOLID
            shape.line_format.fill_format.solid_fill_color.color = drawing.Color.blue
```

**Εξήγηση:**
- Ο `fill_format` και `line_format` Οι ιδιότητες σάς επιτρέπουν να προσαρμόσετε τόσο το στυλ γεμίσματος όσο και το στυλ περιγράμματος των σχημάτων.
- Ρύθμιση παραμέτρων `LineStyle`, `width`, και `dash_style` σας επιτρέπει να επιτύχετε συγκεκριμένα οπτικά εφέ.

### Αποθήκευση της παρουσίασής σας

#### Επισκόπηση:
Αποθηκεύστε τη μορφοποιημένη παρουσίασή σας σε ένα αρχείο για μελλοντική χρήση ή κοινή χρήση.

```python
class PresentationDemo:
    def save_presentation(self, output_path):
        # Αποθήκευση της παρουσίασης με μορφοποιημένα σχήματα στο δίσκο
        self.presentation.save(output_path, slides.export.SaveFormat.PPTX)
```

**Εξήγηση:**
- `save()` Η μέθοδος διατηρεί τις αλλαγές, διασφαλίζοντας ότι όλες οι τροποποιήσεις αποθηκεύονται σε ένα νέο αρχείο.

### Λειτουργία 1: Αυτόματη προσαρμογή πλαισίου κειμένου σε μια διαφάνεια

#### Επισκόπηση
Αυτή η λειτουργία δείχνει πώς να ορίσετε τον τύπο αυτόματης προσαρμογής για ένα πλαίσιο κειμένου μέσα σε ένα Αυτόματο Σχήμα σε μια διαφάνεια, διασφαλίζοντας ότι το κείμενό σας ταιριάζει τέλεια χωρίς χειροκίνητες προσαρμογές.

#### Βήμα προς βήμα εφαρμογή

##### Προσθήκη Αυτόματου Σχήματος και Ορισμός Τύπου Αυτόματης Προσαρμογής
```python
import aspose.slides as slides

def set_autofit_of_text_frame():
    with slides.Presentation() as presentation:
        # Πρόσβαση στην πρώτη διαφάνεια
        slide = presentation.slides[0]

        # Προσθήκη ενός αυτόματου σχήματος ορθογωνίου στη διαφάνεια
        auto_shape = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 150, 75, 350, 350)

        # Ορισμός τύπου αυτόματης προσαρμογής για πλαίσιο κειμένου
        text_frame = auto_shape.text_frame
        text_frame.text_frame_format.autofit_type = slides.TextAutofitType.SHAPE

        # Προσθήκη κειμένου στην παράγραφο εντός του πλαισίου κειμένου
        para = text_frame.paragraphs[0]
        portion = para.portions[0]
        portion.text = "A quick brown fox jumps over the lazy dog."

        # Ορισμός μορφής γεμίσματος κειμένου σε μαύρο συμπαγές χρώμα
        portion.portion_format.fill_format.fill_type = slides.FillType.SOLID
        portion.portion_format.fill_format.solid_fill_color.color = drawing.Color.black

        # Αποθήκευση της παρουσίασης
        presentation.save("text_format_text_out.pptx", slides.export.SaveFormat.PPTX)
```
- **Επεξήγηση παραμέτρων**:
  - `ShapeType.RECTANGLE`: Ορίζει τον τύπο σχήματος του Αυτόματου Σχήματος.
  - `150, 75, 350, 350`Συντεταγμένες X, Y και πλάτος, ύψος για την τοποθέτηση του σχήματος.
  - `slides.TextAutofitType.SHAPE`: Προσαρμόζει αυτόματα το κείμενο ώστε να χωράει μέσα στο σχήμα.

### Χαρακτηριστικό 2: Δημιουργία και Προσαρμογή Αυτόματου Σχήματος

#### Επισκόπηση
Αυτή η λειτουργία σάς καθοδηγεί στην προσθήκη ενός Αυτόματου Σχήματος σε μια διαφάνεια και στην προσαρμογή της εμφάνισής της ορίζοντας τύπους γεμίσματος ή χρώματα.

#### Βήμα προς βήμα εφαρμογή

##### Προσθήκη και προσαρμογή ενός αυτόματου σχήματος
```python
def create_and_customize_auto_shape():
    with slides.Presentation() as presentation:
        # Πρόσβαση στην πρώτη διαφάνεια
        slide = presentation.slides[0]

        # Προσθήκη ενός αυτόματου σχήματος ορθογωνίου στη διαφάνεια
        auto_shape = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 150, 75, 350, 350)

        # Ορισμός μη γεμίσματος για φόντο σχήματος
        auto_shape.fill_format.fill_type = slides.FillType.NO_FILL

        # Προσθήκη περιεχομένου κειμένου στο Αυτόματο Σχήμα
        text_frame = auto_shape.text_frame
        para = text_frame.paragraphs[0]
        portion = para.portions[0]
        portion.text = "A quick brown fox jumps over the lazy dog."

        # Αποθήκευση της παρουσίασης
        presentation.save("auto_shape_out.pptx", slides.export.SaveFormat.PPTX)
```
- **Εξήγηση**:
  - `FillType.NO_FILL`: Εξασφαλίζει ότι δεν εφαρμόζεται γέμισμα φόντου στο σχήμα.

# Σύνδεση σχημάτων με συνδέσμους σε Python χρησιμοποιώντας το Aspose.Slides

Αυτό το σεμινάριο θα σας καθοδηγήσει στη χρήση του Aspose.Slides για Python για τη σύνδεση δύο σχημάτων—έναν κύκλο (έλλειψη) και ένα ορθογώνιο—χρησιμοποιώντας έναν σύνδεσμο.

## Ρύθμιση του Aspose.Slides για Python

Για να ξεκινήσετε να χρησιμοποιείτε το Aspose.Slides για Python, πρέπει να το εγκαταστήσετε. Αυτή η διαδικασία είναι απλή:

**εγκατάσταση pip:**

```bash
pip install aspose.slides
```

### Βασική Αρχικοποίηση και Ρύθμιση

Δείτε πώς μπορείτε να αρχικοποιήσετε την πρώτη σας παρουσίαση:

```python
import aspose.slides as slides

# Δημιουργία αρχικού στιγμιότυπου της κλάσης παρουσίασης που αντιπροσωπεύει το αρχείο PPTX
class Presentation:
    def __enter__(self):
        self.pres = slides.Presentation()
        return self.pres

    def __exit__(self, exc_type, exc_val, exc_tb):
        del self.pres

with Presentation() as pres:
    # Ο κωδικός σας θα μπει εδώ
```

Αυτό δημιουργεί μια νέα παρουσία παρουσίασης όπου μπορείτε να προσθέσετε και να χειριστείτε σχήματα.

## Οδηγός Εφαρμογής

### Σύνδεση σχημάτων με Aspose.Slides σε Python

Ας αναλύσουμε τα βήματα για τη σύνδεση δύο σχημάτων χρησιμοποιώντας έναν σύνδεσμο.

**1. Προσθήκη σχημάτων**

Ξεκινήστε προσθέτοντας μια έλλειψη και ένα ορθογώνιο στη διαφάνειά σας:

```python
# Πρόσβαση στη συλλογή σχημάτων για την επιλεγμένη διαφάνεια
shapes = pres.slides[0].shapes

# Προσθήκη έλλειψης αυτόματου σχήματος στη θέση (0, 100) με πλάτος και ύψος 100
elipse = shapes.add_auto_shape(slides.ShapeType.ELLIPSE, 0, 100, 100, 100)

# Προσθήκη ορθογωνίου αυτόματης διαμόρφωσης στη θέση (100, 300) με πλάτος και ύψος 100
rectangle = shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 100, 300, 100, 100)
```

**2. Προσθήκη σύνδεσης**

Στη συνέχεια, δημιουργήστε μια γραμμή σύνδεσης για να συνδέσετε αυτά τα δύο σχήματα:

```python
# Προσθήκη σχήματος σύνδεσης σε συλλογή σχημάτων διαφανειών
contractor = shapes.add_connector(slides.ShapeType.BENT_CONNECTOR2, 0, 0, 10, 10)

# Σύνδεση σχημάτων σε συνδέσμους
contractor.start_shape_connected_to = elipse
contractor.end_shape_connected_to = rectangle

# Καλέστε την αναδρομολόγηση για να ορίσετε την αυτόματη συντομότερη διαδρομή μεταξύ σχημάτων
contractor.reroute()
```

Ο `add_connector` η μέθοδος δημιουργεί ένα σχήμα λυγισμένου συνδετήρα. Η `reroute()` Η συνάρτηση προσαρμόζει αυτόματα τη διαδρομή της σύνδεσης.

**3. Αποθήκευση της παρουσίασής σας**

Τέλος, αποθηκεύστε την παρουσίασή σας:

```python
pres.save("YOUR_OUTPUT_DIRECTORY/shapes_connect_shapes_using_connectors_out.pptx", slides.export.SaveFormat.PPTX)
```

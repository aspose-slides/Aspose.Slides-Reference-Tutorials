## Οδηγός Εφαρμογής

Αυτή η ενότητα σάς καθοδηγεί στην προσθήκη κειμένου εκθέτη και υποδείκτη σε μια διαφάνεια.

### Δημιουργία νέας παρουσίασης

Ξεκινήστε δημιουργώντας ένα νέο αντικείμενο παρουσίασης:

```python
def adding_superscript_and_subscript_text():
    with slides.Presentation() as presentation:
        slide = presentation.slides[0]
```

Εδώ, `presentation.slides[0]` αποκτά πρόσβαση στην πρώτη διαφάνεια στην παρουσίασή σας. Μπορείτε να προσθέσετε περισσότερες διαφάνειες ανάλογα με τις ανάγκες.

### Προσθήκη σχημάτων και πλαισίων κειμένου

Προσθέστε ένα αυτόματο σχήμα για να φιλοξενήσετε το κείμενό σας:

```python
shape = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 100, 100, 200, 100)
text_frame = shape.text_frame
text_frame.paragraphs.clear()
```

Αυτό το τμήμα κώδικα δημιουργεί ένα ορθογώνιο και διαγράφει τυχόν υπάρχουσες παραγράφους στο πλαίσιο κειμένου.

### Προσθήκη κειμένου εκθέτη

Για να προσθέσετε κείμενο εκθέτη:
1. **Δημιουργία παραγράφου**: 
   ```python
   super_para = slides.Paragraph()
   ```
2. **Προσθήκη συνηθισμένου κειμένου**: 
   ```python
   portion1 = slides.Portion()
   portion1.text = "SlideTitle"
   super_para.portions.add(portion1)
   ```
3. **Προσθήκη τμήματος εκθέτη**: 
   Προσαρμόστε το escapement για να μορφοποιήσετε κείμενο ως εκθέτη.
   ```python
   super_portion = slides.Portion()
   super_portion.portion_format.escapement = 30  # Τοποθέτηση εκθέτη
   super_portion.text = "TM"
   super_para.portions.add(super_portion)
   ```

### Προσθήκη κειμένου υπογραφής

Ομοίως, για το κείμενο με υπογραφή:
1. **Δημιουργία νέας παραγράφου**: 
   ```python
   paragraph2 = slides.Paragraph()
   ```
2. **Προσθήκη συνηθισμένου κειμένου**: 
   ```python
   portion2 = slides.Portion()
   portion2.text = "a"
   paragraph2.portions.add(portion2)
   ```
3. **Προσθήκη τμήματος δείκτη**: 
   Προσαρμόστε το escapement για να μορφοποιήσετε κείμενο ως δείκτη.
   ```python
   sub_portion = slides.Portion()
   sub_portion.portion_format.escapement = -25  # Τοποθέτηση υπογεγραμμένου δείκτη
   sub_portion.text = "i"
   paragraph2.portions.add(sub_portion)
   ```

### Αποθήκευση της παρουσίασης

Τέλος, προσθέστε τις παραγράφους στο πλαίσιο κειμένου και αποθηκεύστε την παρουσίασή σας:

```python
text_frame.paragraphs.add(super_para)
text_frame.paragraphs.add(paragraph2)

presentation.save("YOUR_OUTPUT_DIRECTORY/text_add_superscript_and_subscript_out.pptx", slides.export.SaveFormat.PPTX)
```

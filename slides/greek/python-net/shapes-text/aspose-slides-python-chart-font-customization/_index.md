### Βασική Αρχικοποίηση και Ρύθμιση

Ξεκινήστε εισάγοντας τις απαραίτητες ενότητες και αρχικοποιώντας ένα αντικείμενο παρουσίασης:

```python
import aspose.slides as slides
```

## Οδηγός Εφαρμογής

### Προσθήκη γραφήματος ομαδοποιημένων στηλών και ενεργοποίηση πίνακα δεδομένων

#### Επισκόπηση

Αρχικά, θα προσθέσουμε ένα γράφημα ομαδοποιημένων στηλών στην παρουσίασή μας και θα ενεργοποιήσουμε τη λειτουργία πίνακα δεδομένων.

#### Βήμα προς βήμα εφαρμογή

1. **Προσθήκη γραφήματος ομαδοποιημένων στηλών:**
   
   Προσθέστε το ακόλουθο απόσπασμα κώδικα για να δημιουργήσετε ένα βασικό γράφημα ομαδοποιημένων στηλών στην πρώτη σας διαφάνεια:

    ```python
    chart = pres.slides[0].shapes.add_chart(
        slides.charts.ChartType.CLUSTERED_COLUMN, 50, 50, 600, 400)
    ```
   
2. **Ενεργοποίηση εμφάνισης πίνακα δεδομένων:**
   
   Στη συνέχεια, ενεργοποιήστε τον πίνακα δεδομένων για το γράφημα για να επιτρέψετε την προσαρμογή γραμματοσειράς:

    ```python
    chart.has_data_table = True
    ```

### Προσαρμογή ιδιοτήτων γραμματοσειράς

#### Επισκόπηση

Με ενεργοποιημένο τον πίνακα δεδομένων, μπορούμε πλέον να προσαρμόσουμε τις ιδιότητες γραμματοσειράς του για να βελτιώσουμε την αναγνωσιμότητα και το στυλ.

#### Βήμα προς βήμα εφαρμογή

1. **Ορισμός έντονης γραμματοσειράς:**
   
   Χρησιμοποιήστε αυτό το απόσπασμα για να κάνετε το κείμενο του πίνακα δεδομένων σας έντονο:

    ```python
    chart.chart_data_table.text_format.portion_format.font_bold = slides.NullableBool.TRUE
    ```

2. **Προσαρμογή ύψους γραμματοσειράς:**
   
   Αλλάξτε το μέγεθος της γραμματοσειράς για καλύτερη ορατότητα:

    ```python
    chart.chart_data_table.text_format.portion_format.font_height = 20
    ```

## Οδηγός Εφαρμογής

Ας αναλύσουμε τη διαδικασία προσθήκης τμημάτων σε ένα γεωμετρικό σχήμα χρησιμοποιώντας το Aspose.Slides για Python.

### Δημιουργία και διαμόρφωση της παρουσίασης

#### Επισκόπηση

Αυτή η λειτουργία σάς επιτρέπει να προσθέσετε προσαρμοσμένα τμήματα γραμμής σε ένα υπάρχον ορθογώνιο σχήμα μέσα στην παρουσίασή σας, βελτιώνοντας την οπτική της ελκυστικότητα.

#### Βήμα 1: Προσθήκη νέου ορθογωνίου σχήματος

Ξεκινήστε δημιουργώντας μια νέα διαφάνεια με σχήμα ορθογωνίου:

```python
import aspose.slides as slides

def add_segment_to_geometry_shape():
    # Δημιουργήστε μια νέα παρουσία παρουσίασης
    with slides.Presentation() as pres:
        # Προσθήκη ορθογωνίου σχήματος στην πρώτη διαφάνεια στις καθορισμένες συντεταγμένες
        shape = pres.slides[0].shapes.add_auto_shape(
            slides.ShapeType.RECTANGLE, 100, 100, 200, 100
        )
```

#### Βήμα 2: Πρόσβαση στη διαδρομή γεωμετρίας

Ανακτήστε τη γεωμετρική διαδρομή από το νεοδημιουργημένο ορθογώνιο:

```python
# Λήψη της πρώτης γεωμετρικής διαδρομής του σχήματος
geometry_path = shape.get_geometry_paths()[0]
```

#### Βήμα 3: Προσθήκη τμημάτων γραμμής στη διαδρομή

Προσθέστε τμήματα γραμμής με διαφορετικά βάρη για να προσαρμόσετε τη διαδρομή:

```python
# Προσθέστε δύο τμήματα γραμμής στη γεωμετρική διαδρομή
# Πρώτο τμήμα με βάρος 1
geometry_path.line_to(100, 50, 1)
# Δεύτερο τμήμα με βάρος 4
geometry_path.line_to(100, 50, 4)
```

#### Βήμα 4: Ενημέρωση της γεωμετρικής διαδρομής του σχήματος

Βεβαιωθείτε ότι το σχήμα σας αντικατοπτρίζει αυτά τα νέα τμήματα:

```python
# Ενημέρωση του σχήματος με την τροποποιημένη γεωμετρική διαδρομή
dshape.set_geometry_path(geometry_path)
```

#### Βήμα 5: Αποθηκεύστε την παρουσίασή σας

Τέλος, αποθηκεύστε τις αλλαγές σε ένα αρχείο στον επιθυμητό κατάλογο:

```python
# Αποθήκευση της παρουσίασης σε έναν κατάλογο εξόδου
pres.save("YOUR_OUTPUT_DIRECTORY/shapes_add_segment_to_geometry_path_out.pptx", slides.export.SaveFormat.PPTX)
```

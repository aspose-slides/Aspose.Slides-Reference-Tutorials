## Οδηγός Εφαρμογής

Τώρα που έχουμε έτοιμο το περιβάλλον μας, ας εμβαθύνουμε στην υλοποίηση. Θα καλύψουμε κάθε βήμα που απαιτείται για την προσθήκη ενός πλαισίου κειμένου σε μια διαφάνεια.

### Δημιουργήστε μια νέα παρουσίαση και αποκτήστε πρόσβαση στην πρώτη διαφάνεια

Αρχικά, δημιουργήστε μια παρουσία μιας παρουσίασης και αποκτήστε πρόσβαση στην πρώτη της διαφάνεια:

```python
def add_text_box_to_slide():
    with slides.Presentation() as pres:
        # Πρόσβαση στην πρώτη διαφάνεια
        slide = pres.slides[0]
```

**Εξήγηση**: Το `Presentation()` Η κλάση αρχικοποιεί μια νέα παρουσίαση. Χρησιμοποιώντας `pres.slides[0]`, έχουμε πρόσβαση στην πρώτη διαφάνεια.

### Προσθήκη ορθογωνίου AutoShape

Προσθέστε ένα ορθογώνιο σχήμα στη διαφάνειά σας:

```python
# Προσθήκη αυτόματου σχήματος ορθογωνίου
auto_shape = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 150, 75, 150, 50)
```

**Παράμετροι**: Το `add_auto_shape` Η μέθοδος λαμβάνει τον τύπο σχήματος και τις συντεταγμένες για τη θέση (X, Y) μαζί με το πλάτος και το ύψος.

### Εισαγωγή πλαισίου κειμένου

Εισαγάγετε ένα πλαίσιο κειμένου σε αυτό το ορθογώνιο:

```python
# Προσθήκη πλαισίου κειμένου στο σχήμα
auto_shape.add_text_frame(" ")
```

**Σκοπός**: Αυτό δημιουργεί ένα κενό πλαίσιο κειμένου όπου μπορείτε να προσθέσετε το περιεχόμενό σας.

### Ορισμός κειμένου στο πλαίσιο κειμένου

Τροποποιήστε το κείμενο μέσα στο νέο πλαίσιο κειμένου:

```python
# Πρόσβαση και ορισμός κειμένου
text_frame = auto_shape.text_frame
para = text_frame.paragraphs[0]
portion = para.portions[0]
portion.text = "Aspose TextBox"
```

**Εξήγηση**Εδώ, έχουμε πρόσβαση στην πρώτη παράγραφο και το τμήμα του πλαισίου κειμένου για να ορίσουμε το επιθυμητό κείμενο.

### Αποθήκευση της παρουσίασης

Τέλος, αποθηκεύστε την παρουσίασή σας:

```python
# Αποθήκευση της παρουσίασης
pres.save("YOUR_OUTPUT_DIRECTORY/text_TextBox_out.pptx")
```

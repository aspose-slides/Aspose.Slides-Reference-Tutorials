Μετά την εγκατάσταση, ας εξερευνήσουμε τις βασικές ρυθμίσεις:

Δείτε πώς μπορείτε να αρχικοποιήσετε το Aspose.Slides σε Python:
```python
import aspose.pydrawing as drawing
import aspose.slides as slides

# Βασική αρχικοποίηση ενός αντικειμένου παρουσίασης
class PresentationManager:
    def __init__(self):
        self.pres = slides.Presentation()
        self.slide = self.pres.slides[0]
```

## Οδηγός Εφαρμογής

Ας εμβαθύνουμε στον τρόπο υλοποίησης κουκκίδων χρησιμοποιώντας το Aspose.Slides για Python.

### Χαρακτηριστικό: Κουκκίδες παραγράφων με σύμβολο

#### Επισκόπηση
Αυτή η ενότητα παρουσιάζει την προσθήκη μιας κουκκίδας που βασίζεται σε σύμβολα στην παρουσίασή σας. Προσαρμόστε την εμφάνιση της κουκκίδας, συμπεριλαμβανομένου του χρώματος και του μεγέθους, για καλύτερο οπτικό αποτέλεσμα.

##### Βήμα 1: Ρύθμιση της διαφάνειας και του σχήματός σας
Αποκτήστε πρόσβαση στη διαφάνεια όπου θέλετε να προσθέσετε την κουκκίδα και δημιουργήστε ένα Αυτόματο Σχήμα (ορθογώνιο).
```python
class BulletPointManager(PresentationManager):
    def __init__(self):
        super().__init__()
        # Προσθέστε ένα ορθογώνιο σχήμα και πάρτε το πλαίσιο κειμένου του
        self.auto_shape = self.slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 200, 200, 400, 200)
        self.text_frame = self.auto_shape.text_frame

    def remove_default_paragraphs(self):
        # Κατάργηση τυχόν προεπιλεγμένων παραγράφων
        self.text_frame.paragraphs.remove_at(0)
```

##### Βήμα 2: Ρύθμιση παραμέτρων του σημείου κουκκίδων
Δημιουργήστε μια νέα παράγραφο και ορίστε τις ιδιότητες κουκκίδων της.
```python
class SymbolBulletManager(BulletPointManager):
    def __init__(self):
        super().__init__()
        
    def create_symbol_bullet(self):
        # Δημιουργήστε μια νέα παράγραφο με ρυθμίσεις συμβόλων κουκκίδων
        para = slides.Paragraph()
        para.paragraph_format.bullet.type = slides.BulletType.SYMBOL
        para.paragraph_format.bullet.char = chr(8226)  # Unicode για τον χαρακτήρα κουκκίδας
        para.text = "Welcome to Aspose.Slides"
        para.paragraph_format.indent = 25

        # Προσαρμόστε το χρώμα και το μέγεθος των κουκκίδων
        para.paragraph_format.bullet.color.color_type = slides.ColorType.RGB
        para.paragraph_format.bullet.color.color = drawing.Color.black
        para.paragraph_format.bullet.is_bullet_hard_color = slides.NullableBool.TRUE
        para.paragraph_format.bullet.height = 100

        # Προσθήκη της παραγράφου στο πλαίσιο κειμένου
        self.text_frame.paragraphs.add(para)
```

##### Βήμα 3: Αποθηκεύστε την παρουσίασή σας
```python
class SymbolBulletManager(BulletPointManager):
    def __init__(self):
        super().__init__()
        
    # ... υπάρχοντας κώδικας ...

    def save_presentation(self, output_directory):
        self.pres.save(f"{output_directory}/text_paragraph_bullets_out.pptx", slides.export.SaveFormat.PPTX)
```

### Χαρακτηριστικό: Κουκκίδες παραγράφων με αριθμημένο στυλ

#### Επισκόπηση
Αυτή η ενότητα καλύπτει την εφαρμογή ενός στυλ αριθμημένης κουκκίδας και την προσαρμογή της εμφάνισής του.

##### Βήμα 1: Ρύθμιση της διαφάνειας και του σχήματός σας
Αποκτήστε πρόσβαση στην επιθυμητή διαφάνεια και προσθέστε ένα Αυτόματο Σχήμα όπως πριν.
```python
class NumberedBulletManager(BulletPointManager):
    def __init__(self):
        super().__init__()
```

##### Βήμα 2: Ρύθμιση παραμέτρων του αριθμημένου σημείου κουκκίδων
Δημιουργήστε μια νέα παράγραφο για την αριθμημένη κουκκίδα σας.
```python
class NumberedBulletManager(BulletPointManager):
    def create_numbered_bullet(self):
        # Δημιουργήστε μια νέα παράγραφο με ρυθμίσεις αριθμημένων κουκκίδων
        para2 = slides.Paragraph()
        para2.paragraph_format.bullet.type = slides.BulletType.NUMBERED
        para2.paragraph_format.bullet.numbered_bullet_style = slides.NumberedBulletStyle.BULLET_CIRCLE_NUM_WD_BLACK_PLAIN
        para2.text = "This is a numbered bullet"
        para2.paragraph_format.indent = 25

        # Προσαρμόστε το χρώμα και το μέγεθος της κουκκίδας
        para2.paragraph_format.bullet.color.color_type = slides.ColorType.RGB
        para2.paragraph_format.bullet.color.color = drawing.Color.black
        para2.paragraph_format.bullet.is_bullet_hard_color = slides.NullableBool.TRUE
        para2.paragraph_format.bullet.height = 100

        # Προσθήκη της παραγράφου στο πλαίσιο κειμένου
        self.text_frame.paragraphs.add(para2)
```

##### Βήμα 3: Αποθηκεύστε την παρουσίασή σας
```python
class NumberedBulletManager(BulletPointManager):
    def __init__(self):
        super().__init__()
        
    # ... υπάρχοντας κώδικας ...
```

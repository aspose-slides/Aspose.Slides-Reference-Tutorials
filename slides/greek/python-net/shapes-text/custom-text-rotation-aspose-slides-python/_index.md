## Οδηγός Εφαρμογής

### Προσθήκη γραφήματος ομαδοποιημένων στηλών και περιστροφή πλαισίων κειμένου

Αυτή η ενότητα σάς καθοδηγεί στην προσθήκη ενός γραφήματος ομαδοποιημένων στηλών στην παρουσίασή σας και στον ορισμό προσαρμοσμένων γωνιών περιστροφής για πλαίσια κειμένου μέσα σε αυτό το γράφημα.

#### Βήμα 1: Δημιουργήστε μια παρουσία της κλάσης παρουσίασης

Ξεκινήστε δημιουργώντας ένα `Presentation` αντικείμενο χρησιμοποιώντας τον διαχειριστή περιβάλλοντος, εξασφαλίζοντας αυτόματη διαχείριση πόρων:

```python
import aspose.slides as slides

def rotate_text_frame():
    # Χρησιμοποιήστε τον διαχειριστή περιβάλλοντος για την αυτόματη διαχείριση πόρων
    with slides.Presentation() as presentation:
        pass  # Πλαίσιο κράτησης θέσης για τα επόμενα βήματα
```

#### Βήμα 2: Προσθήκη γραφήματος ομαδοποιημένων στηλών

Προσθέστε ένα γράφημα ομαδοποιημένων στηλών στην πρώτη διαφάνεια στη θέση (50, 50) με καθορισμένες διαστάσεις:

```python
# Προσθήκη γραφήματος στην πρώτη διαφάνεια
class ChartType:
    CLUSTERED_COLUMN = 'ClusteredColumn'
chart = presentation.slides[0].shapes.add_chart(
    ChartType.CLUSTERED_COLUMN, 50, 50, 500, 300
)
```

#### Βήμα 3: Πρόσβαση σε σειρές γραφημάτων και ρύθμιση παραμέτρων ετικετών

Αποκτήστε πρόσβαση στην πρώτη σειρά στα δεδομένα του γραφήματός σας για να χειριστείτε τις ετικέτες της:

```python
# Αποκτήστε πρόσβαση στην πρώτη σειρά
class DataLabelFormatType:
    SHOW_VALUE = 'ShowValue'
series = chart.chart_data.series[0]

# Εμφάνιση τιμών σε ετικέτες
series.labels.default_data_label_format.show_value = True
```

#### Βήμα 4: Ορίστε προσαρμοσμένη γωνία περιστροφής για τη μορφή μπλοκ κειμένου

Ορίστε μια προσαρμοσμένη γωνία περιστροφής για τη μορφή μπλοκ κειμένου για να κάνετε τα δεδομένα σας πιο οπτικά ελκυστικά:

```python
# Ορισμός προσαρμοσμένης γωνίας περιστροφής
class TextBlockFormatType:
    ROTATION_ANGLE = 'RotationAngle'
series.labels.default_data_label_format.text_format.text_block_format.rotation_angle = 65
```

#### Βήμα 5: Προσθήκη και περιστροφή τίτλου γραφήματος

Προσθέστε έναν τίτλο στο γράφημά σας και εφαρμόστε μια προσαρμοσμένη γωνία περιστροφής για βελτιωμένη εμφάνιση:

```python
# Προσθήκη και περιστροφή τίτλου γραφήματος
class TextFrameFormatType:
    ROTATION_ANGLE = 'RotationAngle'
chart.has_title = True
chart.chart_title.add_text_frame_for_overriding("Custom Title").text_frame_format.rotation_angle = -30
```

#### Βήμα 6: Αποθήκευση της παρουσίασης

Τέλος, αποθηκεύστε την παρουσίασή σας σε έναν κατάλογο εξόδου:

```python
# Αποθήκευση της παρουσίασης
class SaveFormatType:
    PPTX = 'Pptx'
presentation.save(
    "YOUR_OUTPUT_DIRECTORY/text_textframe_rotation_out.pptx",
    SaveFormatType.PPTX
)
```

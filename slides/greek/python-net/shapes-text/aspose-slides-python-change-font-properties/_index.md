## Οδηγός Εφαρμογής

### Χαρακτηριστικό: Τροποποίηση ιδιοτήτων γραμματοσειράς

#### Επισκόπηση:
Αυτή η λειτουργία δείχνει πώς μπορείτε να αλλάξετε ιδιότητες γραμματοσειράς όπως η οικογένεια, η έντονη γραφή, η πλάγια γραφή και το χρώμα για κείμενο σε διαφάνειες του PowerPoint χρησιμοποιώντας το Aspose.Slides για Python.

#### Βήματα για την τροποποίηση γραμματοσειρών:

**1. Φορτώστε την παρουσίασή σας**

```python
import aspose.slides as slides

# Άνοιγμα μιας υπάρχουσας παρουσίασης
with slides.Presentation("YOUR_DOCUMENT_DIRECTORY/text_default_fonts.pptx") as pres:
    slide = pres.slides[0]
```

Αυτό το απόσπασμα κώδικα φορτώνει ένα αρχείο PowerPoint, επιτρέποντάς σας να έχετε πρόσβαση στις διαφάνειές του για τροποποίηση.

**2. Πρόσβαση σε πλαίσια κειμένου**

```python
# Ανάκτηση πλαισίων κειμένου από τα δύο πρώτα σχήματα στη διαφάνεια
shape1 = slide.shapes[0]  # Πρώτο σχήμα
tf1 = shape1.text_frame
shape2 = slide.shapes[1]  # Δεύτερο σχήμα
tf2 = shape2.text_frame

# Λάβετε την πρώτη παράγραφο από κάθε πλαίσιο κειμένου
para1 = tf1.paragraphs[0]
para2 = tf2.paragraphs[0]

# Πρόσβαση στο πρώτο μέρος του κειμένου σε κάθε παράγραφο
port1 = para1.portions[0]
port2 = para2.portions[0]
```

Η πρόσβαση σε πλαίσια κειμένου και παραγράφους είναι ζωτικής σημασίας για τον εντοπισμό των τμημάτων του κειμένου που θέλετε να τροποποιήσετε.

**3. Ορίστε νέες οικογένειες γραμματοσειρών**

```python
import aspose.slides as slides

# Ορισμός νέων οικογενειών γραμματοσειρών
fd1 = slides.FontData("Elephant")  # Έντονη γραμματοσειρά σε στυλ ελέφαντα
dfd2 = slides.FontData("Castellar")  # Γραμματοσειρά Castellar

port1.portion_format.latin_font = fd1
port2.portion_format.latin_font = fd2
```

Εδώ, καθορίζουμε τις επιθυμητές γραμματοσειρές για τα τμήματα κειμένου, ενισχύοντας την οπτική ελκυστικότητα.

**4. Εφαρμογή έντονης και πλάγιας γραφής**

```python
# Ορισμός στυλ γραμματοσειράς σε έντονη γραφή
port1.portion_format.font_bold = slides.NullableBool.TRUE
port2.portion_format.font_bold = slides.NullableBool.TRUE

# Εφαρμογή πλάγιου στυλ
port1.portion_format.font_italic = slides.NullableBool.TRUE
port2.portion_format.font_italic = slides.NullableBool.TRUE
```

Η προσθήκη έντονης και πλάγιας γραφής δίνει έμφαση σε συγκεκριμένο κείμενο, κάνοντάς το να ξεχωρίζει.

**5. Αλλαγή χρωμάτων γραμματοσειράς**

```python
import aspose.pydrawing as drawing

# Ορισμός χρωμάτων γραμματοσειράς
port1.portion_format.fill_format.fill_type = slides.FillType.SOLID
port1.portion_format.fill_format.solid_fill_color.color = drawing.Color.purple  # Μωβ χρώμα

port2.portion_format.fill_format.fill_type = slides.FillType.SOLID
port2.portion_format.fill_format.solid_fill_color.color = drawing.Color.peru  # Χρώμα Περού
```

Η προσαρμογή των χρωμάτων της γραμματοσειράς μπορεί να κάνει την παρουσίασή σας πιο ζωντανή και ελκυστική.

**6. Αποθήκευση της τροποποιημένης παρουσίασης**

```python
# Αποθήκευση αλλαγών σε νέο αρχείο
pres.save("YOUR_OUTPUT_DIRECTORY/text_font_properties_out.pptx", slides.export.SaveFormat.PPTX)
```

Η αποθήκευση της τροποποιημένης παρουσίασης διασφαλίζει ότι όλες οι αλλαγές διατηρούνται για μελλοντική χρήση.

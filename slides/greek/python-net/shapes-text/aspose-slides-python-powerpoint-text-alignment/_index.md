## Οδηγός Εφαρμογής

Ας αναλύσουμε πώς να αξιοποιήσουμε το Aspose.Slides για ευθυγράμμιση κειμένου και χειρισμό παρουσιάσεων.

### Χαρακτηριστικό: Στοίχιση παραγράφων στο PowerPoint

#### Επισκόπηση:
Η ευθυγράμμιση κειμένου στις παρουσιάσεις σας όχι μόνο βελτιώνει την αναγνωσιμότητα, αλλά δίνει και μια κομψή εμφάνιση. Αυτή η λειτουργία δείχνει την κεντρική ευθυγράμμιση παραγράφων σε όλες τις διαφάνειες χρησιμοποιώντας Python.

#### Βήματα:

**1. Ορισμός διαδρομών αρχείων**

Αρχικά, ορίστε τις διαδρομές προς τα αρχεία εισόδου και εξόδου:

```python
input_path = "YOUR_DOCUMENT_DIRECTORY/text_paragraphs_alignment.pptx"
output_path = "YOUR_OUTPUT_DIRECTORY/text_paragraphs_alignment_out.pptx"
```

**2. Ανοίξτε την παρουσίαση και αποκτήστε πρόσβαση στη διαφάνεια**

Ανοίξτε μια υπάρχουσα παρουσίαση και λάβετε την πρώτη διαφάνεια:

```python
with slides.Presentation(input_path) as pres:
    slide = pres.slides[0]
```

**3. Τροποποίηση πλαισίων κειμένου**

Αποκτήστε πρόσβαση σε πλαίσια κειμένου από συγκεκριμένα placeholders για να ενημερώσετε το περιεχόμενό τους:

```python
tf1 = slide.shapes[0].text_frame
# Βεβαιωθείτε ότι το σχήμα έχει ένα πλαίσιο κειμένου πριν από την πρόσβαση σε αυτό
if tf1 is not None:
    tf2 = slide.shapes[1].text_frame
    if tf2 is not None:
        tf1.text = "Center Align by Aspose"
        tf2.text = "Center Align by Aspose"
```

**4. Ορισμός στοίχισης παραγράφων**

Ευθυγραμμίστε το κείμενο στο κέντρο κάθε παραγράφου:

```python
para1 = tf1.paragraphs[0]
# Ελέγξτε αν υπάρχουν διαθέσιμες παράγραφοι
if para1 is not None:
    para2 = tf2.paragraphs[0]
    # Βεβαιωθείτε ότι υπάρχει η παράγραφος 2 πριν ορίσετε την ευθυγράμμιση
    if para2 is not None:
        para1.paragraph_format.alignment = slides.TextAlignment.CENTER
        para2.paragraph_format.alignment = slides.TextAlignment.CENTER
```

**5. Αποθήκευση αλλαγών**

Τέλος, αποθηκεύστε τις αλλαγές σας σε ένα νέο αρχείο:

```python
pres.save(output_path, slides.export.SaveFormat.PPTX)
```

### Δυνατότητα: Φόρτωση και αποθήκευση παρουσιάσεων PowerPoint

#### Επισκόπηση:
Αυτή η λειτουργία σάς βοηθά να φορτώνετε παρουσιάσεις, να τις τροποποιείτε προσθέτοντας κείμενο και, στη συνέχεια, να αποθηκεύετε αποτελεσματικά τα ενημερωμένα αρχεία.

#### Βήματα:

**1. Ορισμός διαδρομών αρχείων**

Ορίστε διαδρομές εισόδου και εξόδου παρόμοια με το προηγούμενο παράδειγμα:

```python
input_path = "YOUR_DOCUMENT_DIRECTORY/sample_input.pptx"
output_path = "YOUR_OUTPUT_DIRECTORY/sample_output.pptx"
```

**2. Φόρτωση παρουσίασης και πρόσβαση σε διαφάνεια**

Ανοίξτε το αρχείο παρουσίασής σας και αποκτήστε πρόσβαση στην πρώτη του διαφάνεια:

```python
with slides.Presentation(input_path) as pres:
    slide = pres.slides[0]
```

**3. Προσθήκη κειμένου σε σχήμα**

Ελέγξτε αν το πλαίσιο κειμένου είναι κενό πριν προσθέσετε νέο περιεχόμενο:

```python
tf = slide.shapes[0].text_frame
# Ελέγξτε για "Κανένα" πριν από την πρόσβαση σε ακίνητα
if tf and not tf.text:
    tf.text = "New Text Added"
```

**4. Αποθήκευση της παρουσίασης**

Αποθηκεύστε τις αλλαγές σας:

```python
pres.save(output_path, slides.export.SaveFormat.PPTX)
```

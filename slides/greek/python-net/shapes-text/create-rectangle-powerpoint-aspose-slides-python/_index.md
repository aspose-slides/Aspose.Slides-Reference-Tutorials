# Πώς να δημιουργήσετε και να αποθηκεύσετε ένα απλό ορθογώνιο στο PowerPoint χρησιμοποιώντας το Aspose.Slides Python

## Οδηγός Εφαρμογής
Ας χωρίσουμε τη διαδικασία σε σαφή βήματα για να δημιουργήσουμε ένα ορθογώνιο σχήμα και να αποθηκεύσουμε την παρουσίαση.
### Δημιουργήστε μια παρουσίαση
Αρχικά, δημιουργήστε ένα παράδειγμα του `Presentation` κλάση. Αυτό λειτουργεί σαν ένα κοντέινερ για όλες τις διαφάνειες στην παρουσίασή σας:
```python
with slides.Presentation() as pres:
```
Χρησιμοποιώντας `with`, διασφαλίζει ότι οι πόροι διαχειρίζονται σωστά, κλείνοντας αρχεία ακόμα και σε περίπτωση σφάλματος.
### Πρόσβαση στην πρώτη διαφάνεια
Για να προσθέσετε σχήματα, αποκτήστε πρόσβαση στην πρώτη διαφάνεια:
```python
slide = pres.slides[0]
```
Αυτός ο κώδικας ανακτά την πρώτη διαφάνεια από το αντικείμενο παρουσίασής σας.
### Προσθήκη ορθογωνίου σχήματος
Τώρα, ας προσθέσουμε ένα ορθογώνιο σχήμα σε μια συγκεκριμένη θέση με καθορισμένες διαστάσεις:
```python
# Προσθήκη αυτόματου σχήματος τύπου ορθογωνίου στη θέση (50, 150) με πλάτος 150 και ύψος 50
slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 50, 150, 150, 50)
```
Εδώ, `add_auto_shape` χρησιμοποιείται για την προσθήκη ενός σχήματος. Καθορίζουμε τον τύπο ως `RECTANGLE`, μαζί με τη θέση του `(x=50, y=150)` και μέγεθος `(width=150, height=50)`Αυτή η μέθοδος επιστρέφει ένα αντικείμενο σχήματος το οποίο μπορεί να προσαρμοστεί περαιτέρω εάν χρειάζεται.
### Αποθήκευση της παρουσίασης
Τέλος, αποθηκεύστε την παρουσίασή σας:
```python
# Εγγραφή του αρχείου PPTX στο δίσκο χρησιμοποιώντας έναν κατάλογο εξόδου με κράτηση θέσης
pres.save("YOUR_OUTPUT_DIRECTORY/shapes_add_rectangle_out.pptx", slides.export.SaveFormat.PPTX)
```
Αντικαθιστώ `YOUR_OUTPUT_DIRECTORY` με την επιθυμητή διαδρομή. Η μέθοδος `save` γράφει την τροποποιημένη παρουσίαση πίσω στον δίσκο σε μορφή PPTX.
#### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι οι διαδρομές είναι σωστές και ότι υπάρχουν κατάλογοι πριν από την αποθήκευση.
- Χειριστείτε εξαιρέσεις για λειτουργίες αρχείων χρησιμοποιώντας μπλοκ try-except, εάν χρειάζεται.


### Βασική Αρχικοποίηση

Μόλις εγκατασταθεί, αρχικοποιήστε το Aspose.Slides στο σκριπτ σας:

```python
import aspose.slides as slides

# Αρχικοποίηση παρουσίασης
presentation = slides.Presentation()
```

## Οδηγός Εφαρμογής

Ας αναλύσουμε τη διαδικασία αλλαγής της σειράς των σχημάτων σε διαχειρίσιμα βήματα.

### Βήμα 1: Φόρτωση της παρουσίασής σας

Ξεκινήστε φορτώνοντας ένα υπάρχον αρχείο PowerPoint. Ας υποθέσουμε ότι έχετε ένα αρχείο με το όνομα `welcome-to-powerpoint.pptx`:

```python
# Φόρτωση παρουσίασης
data_dir = 'YOUR_DOCUMENT_DIRECTORY/'
with slides.Presentation(data_dir + 'welcome-to-powerpoint.pptx') as presentation:
    # Πρόσβαση στην πρώτη διαφάνεια
    slide = presentation.slides[0]
```

### Βήμα 2: Προσθήκη και ρύθμιση παραμέτρων σχημάτων

#### Προσθήκη ορθογωνίου σχήματος

Προσθέστε ένα ορθογώνιο στη διαφάνειά σας και διαμορφώστε τις ιδιότητές του:

```python
# Προσθήκη ορθογωνίου σχήματος
rectangle = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 200, 365, 400, 150)
rectangle.fill_format.fill_type = slides.FillType.NO_FILL
rectangle.add_text_frame('')
```

#### Εισαγωγή κειμένου στο ορθογώνιο

Εισαγάγετε κείμενο για να εξατομικεύσετε το σχήμα σας:

```python
# Προσθήκη κειμένου σε ορθογώνιο
text_frame = rectangle.text_frame
para = text_frame.paragraphs[0]
portion = para.portions[0]
portion.text = 'Watermark Text Watermark Text Watermark Text'
```

### Βήμα 3: Προσθέστε ένα σχήμα τριγώνου

Στη συνέχεια, προσθέστε ένα άλλο σχήμα—ένα τρίγωνο:

```python
# Προσθέστε ένα τριγωνικό σχήμα
triangle = slide.shapes.add_auto_shape(slides.ShapeType.TRIANGLE, 200, 365, 400, 150)
```

### Βήμα 4: Αναδιάταξη σχημάτων

Αναδιάταξη σχημάτων μετακινώντας το τρίγωνο μπροστά από άλλα:

```python
# Μετακίνηση τριγώνου προς τα εμπρός
slide.shapes.reorder(2, triangle)
```

### Βήμα 5: Αποθήκευση της τροποποιημένης παρουσίασης

Τέλος, αποθηκεύστε τις αλλαγές σας σε ένα νέο αρχείο:

```python
# Αποθήκευση παρουσίασης
output_dir = 'YOUR_OUTPUT_DIRECTORY/'
presentation.save(output_dir + 'shapes_reorder_out.pptx', slides.export.SaveFormat.PPTX)
```

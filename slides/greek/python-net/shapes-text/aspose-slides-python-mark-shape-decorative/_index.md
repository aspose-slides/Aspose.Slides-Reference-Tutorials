## Οδηγός Εφαρμογής

Τώρα που έχετε ρυθμίσει τα πάντα, ας προχωρήσουμε στην επισήμανση ενός σχήματος ως διακοσμητικού.

### Δημιουργία παρουσίασης και προσθήκη σχήματος

#### Επισκόπηση

Θα ξεκινήσουμε ανοίγοντας (ή δημιουργώντας) μια παρουσίαση, προσθέτοντας ένα αυτόματο σχήμα (όπως ένα ορθογώνιο) και επισημαίνοντάς το ως διακοσμητικό.

#### Βήμα 1: Άνοιγμα ή δημιουργία νέας παρουσίασης
```python
with slides.Presentation() as pres:
    # Πρόσβαση στην πρώτη διαφάνεια της παρουσίασης
    first_slide = pres.slides[0]
```
**Εξήγηση**Αυτός ο κώδικας αρχικοποιεί ένα νέο αντικείμενο παρουσίασης, δημιουργώντας αυτόματα μια αρχική διαφάνεια με την οποία μπορούμε να εργαστούμε.

#### Βήμα 2: Προσθήκη αυτόματου σχήματος στη διαφάνεια
```python
rectangle_shape = first_slide.shapes.add_auto_shape(
    slides.ShapeType.RECTANGLE, 10, 10, 100, 100
)
```
**Παράμετροι**: Το `ShapeType` καθορίζει τον τύπο σχήματος και οι ακόλουθοι τέσσερις αριθμοί ορίζουν τη θέση του (x, y) και το μέγεθός του (πλάτος, ύψος).

#### Βήμα 3: Ορισμός σχήματος ως διακοσμητικού
```python
rectangle_shape.is_decorative = True
```
**Σκοπός**Αυτή η γραμμή επισημαίνει το ορθογώνιο ως διακοσμητικό, υποδεικνύοντας ότι πρέπει να διατηρηθεί αλλά όχι να αλλάξει μέγεθος ή θέση μέσω αυτοματοποιημένων προσαρμογών διάταξης.

### Αποθήκευση της παρουσίασής σας

Αφού επισημάνετε το σχήμα, αποθηκεύστε την παρουσίασή σας:
```python
pres.save('YOUR_OUTPUT_DIRECTORY/DecorativeDemo.pptx', slides.export.SaveFormat.PPTX)
```
**Εξήγηση**: Αυτό αποθηκεύει την τρέχουσα κατάσταση της παρουσίασής σας σε μια καθορισμένη διαδρομή με `.pptx` σχήμα και διάταξις βιβλίου.

## Οδηγός Εφαρμογής
### Προσθήκη σχημάτων και ρύθμιση παραμέτρων ιδιοτήτων
#### Επισκόπηση
Θα προσθέσουμε ένα σχήμα έλλειψης στη διαφάνειά μας, θα διαμορφώσουμε τις ιδιότητες γεμίσματος και γραμμής και θα εφαρμόσουμε ένα εφέ τρισδιάστατης λοξοτομής για μια γυαλισμένη εμφάνιση.

#### Προσθήκη σχήματος έλλειψης
Αρχικά, προσθέστε ένα βασικό σχήμα έλλειψης:
```python
# Πρόσβαση στην πρώτη διαφάνεια της παρουσίασης
slide = pres.slides[0]

# Προσθήκη σχήματος έλλειψης στη διαφάνεια
shape = slide.shapes.add_auto_shape(
    slides.ShapeType.ELLIPSE, 30, 30, 100, 100
)
```
Αυτός ο κώδικας δημιουργεί μια απλή έλλειψη τοποθετημένη στο (30,30) με διαστάσεις 100x100.

#### Ορισμός ιδιοτήτων γεμίσματος και γραμμής
Στη συνέχεια, ορίστε τις ιδιότητες χρώματος γεμίσματος και γραμμής για το σχήμα μας:
```python
# Ορίστε τον τύπο γεμίσματος σε συμπαγές και επιλέξτε ένα πράσινο χρώμα
drawing.Color.green
shape.fill_format.fill_type = slides.FillType.SOLID
shape.fill_format.solid_fill_color.color = Color.green

# Ορίστε τη μορφή γραμμής με πορτοκαλί συμπαγές γέμισμα και ορίστε το πλάτος της
type: solid
fill_format = shape.line_format.fill_format
fill_format.fill_type = slides.FillType.SOLID
fill_format.solid_fill_color.color = Color.orange
shape.line_format.width = 2.0
```
Αυτές οι ρυθμίσεις κάνουν την έλλειψη μας να ξεχωρίζει στη διαφάνεια.

#### Εφαρμογή εφέ 3D λοξοτομής
Το τελευταίο βήμα είναι η εφαρμογή του εφέ λοξοτομής για την προσθήκη βάθους:
```python
# Διαμορφώστε τη μορφή 3D του σχήματος και εφαρμόστε ένα κυκλικό εφέ λοξοτομής
type: circle
shape.three_d_format.depth = 4
shape.three_d_format.bevel_top.bevel_type = slides.BevelPresetType.CIRCLE
shape.three_d_format.bevel_top.height = 6
shape.three_d_format.bevel_top.width = 6

# Ρυθμίστε την κάμερα και τον φωτισμό για ένα ρεαλιστικό αποτέλεσμα
type: orthographic_front
camera = shape.three_d_format.camera
camera.camera_type = slides.CameraPresetType.ORTHOGRAPHIC_FRONT
light_rig = shape.three_d_format.light_rig
light_rig.light_type = slides.LightRigPresetType.THREE_PT
light_rig.direction = slides.LightingDirection.TOP
```
Αυτές οι διαμορφώσεις δημιουργούν ένα οπτικά ελκυστικό τρισδιάστατο εφέ, ενισχύοντας την αισθητική της παρουσίασης.

#### Αποθήκευση της παρουσίασής σας

## Οδηγός Εφαρμογής

Τώρα που είστε έτοιμοι, ας εφαρμόσουμε την εναλλαγή σχήματος βήμα προς βήμα:

### Προσθήκη και περιστροφή σχημάτων στο PowerPoint

#### Επισκόπηση
Αυτή η ενότητα εστιάζει στην προσθήκη ενός ορθογώνιου σχήματος σε μια διαφάνεια και στην περιστροφή της κατά 90 μοίρες.

#### Βήμα προς βήμα εφαρμογή

##### Αρχικοποίηση παρουσίασης

Ξεκινήστε δημιουργώντας μια παρουσία του `Presentation` κλάση, η οποία αντιπροσωπεύει το αρχείο PPTX σας:
```python
with slides.Presentation() as pres:
    # Θα εργαστούμε εντός αυτού του διαχειριστή περιβάλλοντος για την αποτελεσματική διαχείριση των πόρων.
```

##### Πρόσβαση στη διαφάνεια και προσθήκη σχήματος

Αποκτήστε πρόσβαση στην πρώτη διαφάνεια της παρουσίασης και προσθέστε ένα ορθογώνιο σχήμα:
```python
slide = pres.slides[0]

shape = slide.shapes.add_auto_shape(
    slides.ShapeType.RECTANGLE, 50, 150, 75, 150)
# Οι παράμετροι ορίζουν τη θέση (x, y) και το μέγεθος (πλάτος, ύψος).
```

##### Περιστροφή του σχήματος

Περιστρέψτε το νεοπροστιθέμενο σχήμα ορίζοντας την ιδιότητα περιστροφής του:
```python
shape.rotation = 90
# Η περιστροφή ορίζεται σε μοίρες.
```

##### Αποθήκευση παρουσίασης

Τέλος, αποθηκεύστε τις αλλαγές σας σε έναν καθορισμένο κατάλογο εξόδου:
```python
pres.save("YOUR_OUTPUT_DIRECTORY/shapes_rotate_out.pptx", slides.export.SaveFormat.PPTX)
# Βεβαιωθείτε ότι η διαδρομή υπάρχει ή προσαρμόστε την ανάλογα.
```

#### Συμβουλές αντιμετώπισης προβλημάτων
- **Το σχήμα δεν εμφανίζεται**Ελέγξτε τις παραμέτρους θέσης και μεγέθους. Εάν οι τιμές είναι εκτός οθόνης, προσαρμόστε τις.
- **Προβλήματα εναλλαγής**: Επαληθεύστε ότι `shape.rotation` έχει οριστεί σωστά· βεβαιωθείτε ότι δεν υπάρχουν αντικρουόμενοι μετασχηματισμοί.

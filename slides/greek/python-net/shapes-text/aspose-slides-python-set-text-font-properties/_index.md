## Οδηγός Εφαρμογής

### Ορισμός ιδιοτήτων γραμματοσειράς κειμένου (Επισκόπηση λειτουργιών)
Σε αυτήν την ενότητα, μάθετε πώς να ορίσετε διάφορες ιδιότητες γραμματοσειράς για κείμενο μέσα σε μια διαφάνεια στο PowerPoint χρησιμοποιώντας το Aspose.Slides για Python.

#### Βήμα 1: Δημιουργία Παρουσίασης
Ξεκινήστε δημιουργώντας μια παρουσία του `Presentation` τάξη:

```python
def set_text_font_properties():
    with slides.Presentation() as presentation:
        slide = presentation.slides[0]
```
**Εξήγηση:** Χρησιμοποιούμε έναν διαχειριστή περιβάλλοντος (`with`για να διασφαλιστεί η σωστή διαχείριση των πόρων, η οποία βοηθά στην αποτελεσματική χρήση της μνήμης.

#### Βήμα 2: Προσθήκη Αυτόματου Σχήματος
Προσθέστε ένα ορθογώνιο σχήμα για την τοποθέτηση κειμένου στη διαφάνειά σας:

```python
auto_shape = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 50, 50, 200, 50)
```
**Εξήγηση:** Ο `add_auto_shape` η μέθοδος προσθέτει ένα σχήμα συγκεκριμένου τύπου και διαστάσεων. Εδώ, χρησιμοποιούμε ένα ορθογώνιο στη θέση `(50, 50)` με πλάτος `200` και ύψος `50`.

#### Βήμα 3: Προσαρμόστε το TextFrame
Αποκτήστε πρόσβαση στο πλαίσιο κειμένου για να προσθέσετε και να προσαρμόσετε κείμενο:

```python
tf = auto_shape.text_frame
tf.text = "Aspose TextBox"
```
**Εξήγηση:** Ο `text_frame` Το χαρακτηριστικό σάς επιτρέπει να έχετε πρόσβαση ή να τροποποιείτε το περιεχόμενο ενός σχήματος.

#### Βήμα 4: Ορισμός ιδιοτήτων γραμματοσειράς
Εφαρμόστε διαφορετικές ιδιότητες γραμματοσειράς όπως έντονη γραφή, πλάγια γραφή, υπογράμμιση και χρώμα:

```python
port = tf.paragraphs[0].portions[0]
# Ορισμός ονόματος γραμματοσειράς σε 'Times New Roman'
port.portion_format.latin_font = slides.FontData("Times New Roman")
# Εφαρμόστε έντονο στυλ
port.portion_format.font_bold = slides.NullableBool.TRUE
# Εφαρμογή πλάγιας γραφής
port.portion_format.font_italic = slides.NullableBool.TRUE
# Υπογράμμισε το κείμενο
port.portion_format.font_underline = slides.TextUnderlineType.SINGLE
# Ορισμός ύψους γραμματοσειράς σε 25 σημεία
port.portion_format.font_height = 25
# Αλλαγή χρώματος κειμένου σε μπλε
color = drawing.Color.blue
port.portion_format.fill_format.fill_type = slides.FillType.SOLID
port.portion_format.fill_format.solid_fill_color.color = color
```
**Εξήγηση:** 
- **Όνομα γραμματοσειράς**: Ορίζει την οικογένεια γραμματοσειρών.
- **Έντονα και πλάγια στυλ**: Ενισχύστε την έμφαση εναλλάσσοντας αυτά τα στυλ.
- **Υπογραμμίζω**Προσθέτει μια υπογράμμιση μίας γραμμής για διάκριση.
- **Ύψος γραμματοσειράς**: Προσαρμόζει το μέγεθος του κειμένου για καλύτερη ορατότητα.
- **Χρώμα**: Αλλάζει το χρώμα του κειμένου για να το κάνει να ξεχωρίζει.

#### Βήμα 5: Αποθηκεύστε την παρουσίασή σας
Αποθηκεύστε την παρουσίασή σας με όλες τις τροποποιήσεις:

```python
def save_presentation(presentation, output_directory):
    presentation.save(f"{output_directory}/text_SetTextFontProperties_out.pptx", slides.export.SaveFormat.PPTX)
```
**Εξήγηση:** Ο `save` Η μέθοδος γράφει την τροποποιημένη παρουσίαση σε ένα αρχείο. Βεβαιωθείτε ότι η διαδρομή έχει καθοριστεί σωστά για την επιτυχή αποθήκευση.

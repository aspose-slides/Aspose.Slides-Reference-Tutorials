## Οδηγός Εφαρμογής

Αυτή η ενότητα περιγράφει λεπτομερώς τη διαδικασία μετατροπής μιας εικόνας SVG σε μια ομάδα σχημάτων μέσα σε μια παρουσίαση PowerPoint.

### Μετατροπή εικόνας SVG σε ομάδα σχημάτων

Δείτε πώς μπορείτε να μετατρέψετε μια ενσωματωμένη εικόνα SVG σε μια διαφάνεια σε μια ομάδα σχημάτων με δυνατότητα χειρισμού:

#### Επισκόπηση

Φορτώστε μια παρουσίαση, εντοπίστε μια εικόνα SVG μέσα σε αυτήν και μετατρέψτε αυτήν την εικόνα σε μια ομάδα σχημάτων για βελτιωμένες επιλογές επεξεργασίας.

#### Βήμα 1: Φόρτωση της παρουσίασης

Ανοίξτε το αρχείο PowerPoint χρησιμοποιώντας το Aspose.Slides:

```python
with slides.Presentation('YOUR_DOCUMENT_DIRECTORY/save_convert_svg_to_group_of_shapes.pptx') as pres:
    picture_frame = pres.slides[0].shapes[0]
```

#### Βήμα 2: Ελέγξτε για εικόνα SVG

Προσδιορίστε εάν το πρώτο σχήμα στη διαφάνειά σας περιέχει μια εικόνα SVG:

```python
svg_image = picture_frame.picture_format.picture.image.svg_image
if svg_image is not None:
    # Συνέχεια με τη μετατροπή
```

Ο `picture_format` Το αντικείμενο προσδιορίζει εάν ένα πλαίσιο περιέχει ένα SVG.

#### Βήμα 3: Μετατροπή σε ομάδα σχημάτων

Μετασχηματίστε το SVG σε μια ομάδα σχημάτων στην αρχική του θέση:

```python
group_shape = pres.slides[0].shapes.add_group_shape(
    svg_image,
    picture_frame.frame.x,
    picture_frame.frame.y,
    picture_frame.frame.width,
    picture_frame.frame.height
)
```

Ο `add_group_shape` Η μέθοδος είναι κρίσιμη για τη διατήρηση της συνέπειας της διάταξης.

#### Βήμα 4: Αφαίρεση του αρχικού πλαισίου

Μετά τη μετατροπή, αφαιρέστε την αρχική εικόνα SVG:

```python
pres.slides[0].shapes.remove(picture_frame)
```

Αυτό το βήμα διασφαλίζει ότι δεν θα υπάρξει διπλότυπη δημοσίευση περιεχομένου μέσα στη διαφάνειά σας.

#### Βήμα 5: Αποθήκευση της παρουσίασης

Τέλος, αποθηκεύστε την τροποποιημένη παρουσίασή σας σε ένα νέο αρχείο:

```python
pres.save('YOUR_OUTPUT_DIRECTORY/save_convert_svg_to_group_of_shapes_out.pptx', slides.export.SaveFormat.PPTX)
```

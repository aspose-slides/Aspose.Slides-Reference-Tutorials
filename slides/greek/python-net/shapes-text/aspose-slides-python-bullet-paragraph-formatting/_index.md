## Οδηγός Εφαρμογής

Ας αναλύσουμε τη διαδικασία προσθήκης εσοχής κουκκίδων και μορφοποίησης παραγράφων σε διαχειρίσιμες ενότητες.

### Προσθήκη σχημάτων σε διαφάνειες

#### Επισκόπηση

Αρχικά, πρέπει να προσθέσουμε ένα σχήμα στη διαφάνειά μας που θα περιέχει κείμενο. Αυτό βοηθά στην οργάνωση του περιεχομένου με τάξη.

#### Βήματα:

1. **Αποκτήστε την πρώτη διαφάνεια**: Πρόσβαση στην πρώτη διαφάνεια της παρουσίασής σας.
2. **Προσθήκη ορθογωνίου σχήματος**: Χρήση `add_auto_shape` για να δημιουργήσετε ένα ορθογώνιο για τη συγκράτηση κειμένου.

```python
# Λήψη πρώτης διαφάνειας
slide = pres.slides[0]

# Προσθήκη ορθογωνίου σχήματος στη διαφάνεια
rect = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 100, 100, 500, 150)
```

### Εισαγωγή και μορφοποίηση κειμένου

#### Επισκόπηση

Μόλις έχουμε το σχήμα μας, ήρθε η ώρα να εισαγάγουμε κείμενο και να το μορφοποιήσουμε για σαφήνεια και εντυπωσιακό αποτέλεσμα.

#### Βήματα:

1. **Προσθήκη πλαισίου κειμένου**: Δημιουργήστε ένα `TextFrame` για να κρατήσετε το κείμενό σας.
2. **Τύπος αυτόματης προσαρμογής**Βεβαιωθείτε ότι το κείμενο χωράει αυτόματα μέσα στο ορθογώνιο.
3. **Αφαίρεση περιγραμμάτων**Για οπτική ευκρίνεια, αφαιρέστε τις γραμμές περιγράμματος του σχήματος.

```python
# Προσθήκη TextFrame στο ορθογώνιο
tf = rect.add_text_frame("This is first line \r\nThis is second line \r\nThis is third line")

# Ορίστε το κείμενο ώστε να χωράει αυτόματα μέσα στο σχήμα
tf.text_frame_format.autofit_type = slides.TextAutofitType.SHAPE

# Αφαιρέστε τις γραμμές περιγράμματος του ορθογωνίου για οπτική ευκρίνεια
rect.line_format.fill_format.fill_type = slides.FillType.NONE
```

### Προσαρμογή στυλ κουκκίδων και εσοχών

#### Επισκόπηση

Η πραγματική δύναμη έγκειται στην προσαρμογή των στυλ κουκκίδων και στην προσαρμογή των εσοχών των παραγράφων για να κάνετε το περιεχόμενό σας οπτικά ελκυστικό.

#### Βήματα:

1. **Ορισμός στυλ κουκκίδας**: Ορίστε τον τύπο και τον χαρακτήρα των κουκκίδων για κάθε παράγραφο.
2. **Προσαρμογή ευθυγράμμισης και βάθους**: Στοίχιση κειμένου και ορισμός επιπέδων βάθους για την ιεραρχία.
3. **Ορισμός εσοχής**: Καθορίστε διαφορετικές τιμές εσοχής για ποικίλα διαστήματα.

```python
# Μορφοποίηση πρώτης παραγράφου: Ορισμός στυλ κουκκίδας, συμβόλου, στοίχισης και εσοχών
def format_paragraph(para, char, align, depth, indent):
    para.paragraph_format.bullet.type = slides.BulletType.SYMBOL
    para.paragraph_format.bullet.char = char
    para.paragraph_format.alignment = align
    para.paragraph_format.depth = depth
    para.paragraph_format.indent = indent

para1 = tf.paragraphs[0]
format_paragraph(para1, chr(8226), slides.TextAlignment.LEFT, 2, 30)

# Επαναλάβετε για τη δεύτερη και την τρίτη παράγραφο με διαφορετικές τιμές εσοχής
def format_multiple_paragraphs(paragraphs):
    for i, para in enumerate(paragraphs[1:], start=1):
        format_paragraph(para, chr(8226), slides.TextAlignment.LEFT, 4, 40 + i * 10)

format_multiple_paragraphs(tf.paragraphs)
```

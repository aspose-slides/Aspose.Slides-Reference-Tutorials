### Δημιουργία και διαμόρφωση ενός τρισδιάστατου σχήματος στο PowerPoint

#### Επισκόπηση

Αυτή η ενότητα σας καθοδηγεί στην προσθήκη ενός ορθογωνίου σχήματος, στον ορισμό του κειμένου του και στην εφαρμογή εφέ 3D χρησιμοποιώντας το Aspose.Slides.

#### Βήμα προς βήμα εφαρμογή

##### Προσθήκη αυτόματου σχήματος

Αρχικά, προσθέστε ένα ορθογώνιο στη διαφάνειά σας:

```python
def render_3d_shape():
    with slides.Presentation() as pres:
        # Προσθήκη αυτόματου σχήματος (ορθογώνιου) στην πρώτη διαφάνεια
        shape = pres.slides[0].shapes.add_auto_shape(
            slides.ShapeType.RECTANGLE, 200, 150, 200, 200)
```

##### Ρύθμιση κειμένου και μεγέθους γραμματοσειράς

Προσαρμόστε το κείμενο μέσα στο ορθογώνιό σας:

```python
        # Ορίστε κείμενο μέσα στο ορθογώνιο και προσαρμόστε το μέγεθος της γραμματοσειράς
        shape.text_frame.text = "3D"
        shape.text_frame.paragraphs[0].paragraph_format.default_portion_format.font_height = 64
```

##### Διαμόρφωση ρυθμίσεων 3D

Διαμορφώστε την κάμερα, τον φωτισμό και την εξώθηση για ένα ρεαλιστικό τρισδιάστατο εφέ:

```python
        # Διαμόρφωση ρυθμίσεων 3D για το σχήμα
        shape.three_d_format.camera.camera_type = slides.CameraPresetType.ORTHOGRAPHIC_FRONT
        shape.three_d_format.camera.set_rotation(20, 30, 40)
        shape.three_d_format.light_rig.light_type = slides.LightRigPresetType.FLAT
        shape.three_d_format.light_rig.direction = slides.LightingDirection.TOP
        shape.three_d_format.material = slides.MaterialPresetType.FLAT
        shape.three_d_format.extrusion_height = 100
        shape.three_d_format.extrusion_color.color = drawing.Color.blue
```

##### Αποθήκευση της παρουσίασης

Τέλος, αποθηκεύστε τη διαφάνειά σας ως εικόνα και παρουσίαση:

```python
        # Αποθήκευση της διαφάνειας ως εικόνας και της παρουσίασης στον καθορισμένο κατάλογο εξόδου
```

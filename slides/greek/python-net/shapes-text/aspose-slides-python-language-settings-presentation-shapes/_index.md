### Δημιουργία παρουσίασης
**Επισκόπηση:** Ξεκινήστε αρχικοποιώντας μια νέα παρουσίαση PowerPoint όπου θα προσθέσουμε τα σχήματα κειμένου μας με συγκεκριμένες ρυθμίσεις γλώσσας.

#### Βήμα 1: Αρχικοποίηση της παρουσίασης
Ξεκινήστε δημιουργώντας μια παρουσία μιας παρουσίασης χρησιμοποιώντας το `with` Δήλωση για τη διαχείριση πόρων. Αυτό διασφαλίζει ότι τα αρχεία κλείνουν σωστά μετά τη χρήση, αποτρέποντας διαρροές μνήμης.
```python
import aspose.slides as slides

# Δημιουργία νέας παρουσίασης
text_setting_language(pres):
    # Ο κώδικας για την τροποποίηση της παρουσίασης βρίσκεται εδώ
```

#### Βήμα 2: Προσθήκη Αυτόματου Σχήματος
Προσθέστε ένα ορθογώνιο σχήμα στη διαφάνειά σας. Αυτό θα χρησιμεύσει ως δοχείο κειμένου όπου μπορούμε να ορίσουμε ρυθμίσεις για συγκεκριμένες γλώσσες.
```python
# Προσθήκη Αυτόματου Σχήματος τύπου Ορθογώνιου
shape = pres.slides[0].shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 50, 50, 200, 50)
```
- **Παράμετροι:** `50, 50` είναι οι συντεταγμένες x και y για τον προσδιορισμό της θέσης. `200, 50` ορίστε το πλάτος και το ύψος του ορθογωνίου.

#### Βήμα 3: Εισαγωγή κειμένου και ορισμός γλώσσας
Εισαγάγετε κείμενο στο σχήμα σας και καθορίστε το αναγνωριστικό γλώσσας του για να ενεργοποιήσετε τον ορθογραφικό έλεγχο σε αυτήν τη γλώσσα.
```python
# Προσθήκη πλαισίου κειμένου και ορισμός περιεχομένου
text_setting_language(pres):
    shape.add_text_frame("Text to apply spellcheck language")

# Ορισμός του αναγνωριστικού γλώσσας για τα Αγγλικά - Ηνωμένο Βασίλειο
text_setting_language(pres):
    shape.text_frame.paragraphs[0].portions[0].portion_format.language_id = "en-GB"
```
- **Αναγνωριστικό γλώσσας:** Αλλαγή `"en-GB"` σε άλλους κωδικούς ISO 639-2, όπως απαιτείται (π.χ., `fr-FR` για τα γαλλικά).

#### Βήμα 4: Αποθήκευση της παρουσίασης
Τέλος, αποθηκεύστε την παρουσίασή σας σε μορφή PPTX σε έναν καθορισμένο κατάλογο εξόδου.
```python
# Αποθήκευση της παρουσίασης με συγκεκριμένο όνομα και μορφή
text_setting_language(pres):
    pres.save("YOUR_OUTPUT_DIRECTORY/text_SettingPresentationLanguageAndShapeText_out.pptx",
              slides.export.SaveFormat.PPTX)
```

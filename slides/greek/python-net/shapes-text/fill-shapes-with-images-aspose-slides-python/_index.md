### Γέμισμα σχημάτων με εικόνες
Αυτή η λειτουργία σάς επιτρέπει να βελτιώσετε τις διαφάνειές σας χρησιμοποιώντας εικόνες ως γέμισμα για διάφορα σχήματα, προσθέτοντας μια επαγγελματική πινελιά ή θεματική συνέπεια στις παρουσιάσεις σας.

#### Βήμα 1: Εισαγωγή Aspose.Slides
Ξεκινήστε εισάγοντας την απαραίτητη ενότητα:

```python
import aspose.slides as slides
```

#### Βήμα 2: Ορίστε τις διαδρομές εικόνας σας
Καθορίστε διαδρομές για τους καταλόγους εισόδου και εξόδου:

```python
data_dir = "YOUR_DOCUMENT_DIRECTORY/"
out_dir = "YOUR_OUTPUT_DIRECTORY/"
```

Αντικαθιστώ `"YOUR_DOCUMENT_DIRECTORY/"` με τη διαδρομή του καταλόγου προέλευσης εικόνας και `"YOUR_OUTPUT_DIRECTORY/"` με το πού θέλετε να αποθηκεύσετε την τελική παρουσίαση.

#### Βήμα 3: Δημιουργία μιας παρουσίας παρουσίασης
Δημιουργήστε ένα στιγμιότυπο του `Presentation` κλάση, η οποία αντιπροσωπεύει ένα αρχείο PowerPoint:

```python
with slides.Presentation() as pres:
    slide = pres.slides[0]
```

Εδώ, έχουμε πρόσβαση στην πρώτη διαφάνεια της παρουσίασης. Μπορείτε να τροποποιήσετε ή να προσθέσετε νέες διαφάνειες με βάση τις απαιτήσεις σας.

#### Βήμα 4: Προσθήκη και ρύθμιση παραμέτρων σχημάτων
Προσθέστε ένα αυτόματο σχήμα στη διαφάνεια και διαμορφώστε τον τύπο γέμισής του:

```python
shape = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 50, 150, 75, 150)
shape.fill_format.fill_type = slides.FillType.PICTURE
```

Αυτός ο κώδικας προσθέτει ένα ορθογώνιο σχήμα σε καθορισμένες συντεταγμένες με διαστάσεις πλάτους 75 και ύψους 150.

#### Βήμα 5: Ορισμός λειτουργίας γεμίσματος εικόνας
Ορίστε πώς η εικόνα θα γεμίσει το σχήμα:

```python
shape.fill_format.picture_fill_format.picture_fill_mode = slides.PictureFillMode.TILE
```

Χρησιμοποιώντας `TILE` Η λειτουργία σχηματίζει πλακίδια στην εικόνα σε ολόκληρη την περιοχή του σχήματος, δημιουργώντας ένα εφέ απρόσκοπτου μοτίβου.

img = slides.Images.from_file(data_dir + "image2.jpg")

shape.fill_format.picture_fill_format.picture.image = imgx

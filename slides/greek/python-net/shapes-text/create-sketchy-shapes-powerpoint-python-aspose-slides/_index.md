### Δημιουργία σχηματικών σχημάτων στο PowerPoint

#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να προσθέσετε ένα εφέ σχηματικής γραμμής στα σχήματα στην παρουσίασή σας, δίνοντάς τους μια καλλιτεχνική και χειρόγραφη εμφάνιση.

#### Προσθήκη ορθογωνίου με στυλ γραμμής σκαριφήματος

##### Βήμα 1: Αρχικοποίηση νέας παρουσίασης
Ξεκινήστε δημιουργώντας μια νέα παρουσία παρουσίασης:
```python
with slides.Presentation() as pres:
    # Συνεχίστε με την προσθήκη σχημάτων
```

##### Βήμα 2: Προσθήκη αυτόματου σχήματος (ορθογώνιο)
Εισαγάγετε ένα ορθογώνιο σχήμα στην πρώτη διαφάνεια χρησιμοποιώντας `add_auto_shape`:
```python
shape = pres.slides[0].shapes.add_auto_shape(
    slides.ShapeType.RECTANGLE, 20, 20, 300, 150
)
```
Οι παράμετροι καθορίζουν τον τύπο του σχήματος και τη θέση/μέγεθος του στη διαφάνεια.

##### Βήμα 3: Ορίστε τον Τύπο Γεμίσματος σε 'NO_FILL'
Για να εστιάσετε στο εφέ σκίτσου, αφαιρέστε οποιοδήποτε γέμισμα:
```python
shape.fill_format.fill_type = slides.FillType.NO_FILL
```

##### Βήμα 4: Εφαρμογή εφέ σκίτσου γραμμής σκαριφήματος
Βελτιώστε το σχήμα σας με ένα στυλ γραμμικής σκιαγράφησης:
```python
shape.line_format.sketch_format.sketch_type = slides.LineSketchType.SCRIBBLE
```
Αυτή η ρύθμιση εφαρμόζει την ακανόνιστη εμφάνιση στο περίγραμμα του σχήματος.

##### Βήμα 5: Αποθήκευση ως PNG και PPTX
Εξαγάγετε πρώτα τη διαφάνεια ως εικόνα και, στη συνέχεια, αποθηκεύστε την ως αρχείο PowerPoint:
```python
pres.slides[0].get_image(4 / 3, 4 / 3).save(
    "YOUR_OUTPUT_DIRECTORY/shapes_sketch_format_out.png",
    slides.ImageFormat.PNG
)
pres.save("YOUR_OUTPUT_DIRECTORY/shapes_sketch_format_out.pptx", 
          slides.export.SaveFormat.PPTX)
```
Αντικαθιστώ `"YOUR_OUTPUT_DIRECTORY"` με την επιθυμητή διαδρομή αποθήκευσης.

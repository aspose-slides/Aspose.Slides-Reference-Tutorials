# Πώς να δημιουργήσετε κουκκίδες πολλαπλών επιπέδων σε παρουσιάσεις χρησιμοποιώντας το Aspose.Slides για Python

Αυτό το σεμινάριο θα σας καθοδηγήσει στην εφαρμογή κουκκίδων πολλαπλών επιπέδων στις διαφάνειές σας χρησιμοποιώντας το Aspose.Slides για Python—ένα ισχυρό εργαλείο που απλοποιεί τον αυτοματισμό των παρουσιάσεων.

**Τι θα μάθετε:**
- Πώς να ρυθμίσετε το Aspose.Slides για Python
- Δημιουργία μιας βασικής διαφάνειας με πολλαπλά επίπεδα κουκκίδων
- Προσαρμογή χαρακτήρων και χρωμάτων κουκκίδων
- Αποτελεσματική αποθήκευση παρουσιάσεων

Ας εξερευνήσουμε τις απαραίτητες προϋποθέσεις πριν ξεκινήσουμε την εφαρμογή αυτής της δυνατότητας στα έργα σας.

## Ρύθμιση του Aspose.Slides για Python

### Εγκατάσταση

Για να ξεκινήσετε να χρησιμοποιείτε το Aspose.Slides, εγκαταστήστε το πακέτο μέσω του pip:

```bash
pip install aspose.slides
```

**Απόκτηση Άδειας:**
Το Aspose προσφέρει μια δωρεάν δοκιμαστική περίοδο για να εξερευνήσετε τις δυνατότητές του. Αποκτήστε μια προσωρινή άδεια χρήσης για να δοκιμάσετε όλες τις λειτουργίες χωρίς περιορισμούς. Εξετάστε το ενδεχόμενο αγοράς μιας συνδρομής για εκτεταμένη χρήση.

```python
import aspose.slides as slides

# Αρχικοποίηση κλάσης παρουσίασης
def create_presentation():
    with slides.Presentation() as pres:
        # Ο κώδικά σας εδώ για να χειριστείτε την παρουσίαση
```

## Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, θα καλύψουμε τη δημιουργία κουκκίδων πολλαπλών επιπέδων σε μια διαφάνεια. Θα την αναλύσουμε σε διαχειρίσιμα βήματα.

### Δημιουργία διαφάνειας με κουκκίδες πολλαπλών επιπέδων

**Επισκόπηση:**
Θα προσθέσουμε ένα Αυτόματο Σχήμα (ένα ορθογώνιο) στην πρώτη μας διαφάνεια και θα την συμπληρώσουμε με κείμενο που περιέχει πολλαπλά επίπεδα κουκκίδων.

1. **Πρόσβαση στην πρώτη διαφάνεια**
   ```python
   # Πρόσβαση στην πρώτη διαφάνεια από την παρουσίαση
   slide = pres.slides[0]
   ```

2. **Προσθήκη αυτόματου σχήματος**
   ```python
   # Προσθέστε ένα ορθογώνιο σχήμα για να κρατήσετε τα σημεία με τις κουκκίδες μας
   auto_shape = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 200, 200, 400, 200)
   ```

3. **Ρύθμιση παραμέτρων του πλαισίου κειμένου**
   Εδώ διαμορφώνουμε το πλαίσιο κειμένου που θα περιέχει τα σημεία κουκκίδων μας.
   
   ```python
   # Λήψη και διαγραφή τυχόν προεπιλεγμένων παραγράφων στο πλαίσιο κειμένου
   text = auto_shape.add_text_frame("")
   text.paragraphs.clear()
   ```

4. **Προσθήκη κουκκίδων**
   Δημιουργούμε και προσθέτουμε πολλαπλά επίπεδα κουκκίδων, το καθένα με ξεχωριστούς χαρακτήρες και βάθη εσοχής.
   
   - **Σφαίρα πρώτου επιπέδου:**
     ```python
     para1 = slides.Paragraph()
     para1.text = "Content"
     para1.paragraph_format.bullet.type = slides.BulletType.SYMBOL
     para1.paragraph_format.bullet.char = chr(8226)  # Χαρακτήρας κουκκίδας
     para1.paragraph_format.default_portion_format.fill_format.fill_type = slides.FillType.SOLID
     para1.paragraph_format.default_portion_format.fill_format.solid_fill_color.color = drawing.Color.black
     para1.paragraph_format.depth = 0  # Κουκκίδα Επιπέδου 0
     ```
   
   - **Κουκκίδα Δεύτερου Επιπέδου:**
     ```python
     para2 = slides.Paragraph()
     para2.text = "Second Level"
     para2.paragraph_format.bullet.type = slides.BulletType.SYMBOL
     para2.paragraph_format.bullet.char = '-'  # Χαρακτήρας κουκκίδας
     para2.paragraph_format.default_portion_format.fill_type = slides.FillType.SOLID
     para2.paragraph_format.default_portion_format.fill_format.solid_fill_color.color = drawing.Color.black
     para2.paragraph_format.depth = 1  # Κουκκίδα Επιπέδου 1
     ```
   
   - **Σφαίρα Τρίτου Επιπέδου:**
     ```python
     para3 = slides.Paragraph()
     para3.text = "Third Level"
     para3.paragraph_format.bullet.type = slides.BulletType.SYMBOL
     para3.paragraph_format.bullet.char = chr(8226)  # Χαρακτήρας κουκκίδας
     para3.paragraph_format.default_portion_format.fill_type = slides.FillType.SOLID
     para3.paragraph_format.default_portion_format.fill_format.solid_fill_color.color = drawing.Color.black
     para3.paragraph_format.depth = 2  # Κουκκίδα Επιπέδου 2
     ```
   
   - **Σφαίρα Τέταρτου Επιπέδου:**
     ```python
     para4 = slides.Paragraph()
     para4.text = "Fourth Level"
     para4.paragraph_format.bullet.type = slides.BulletType.SYMBOL
     para4.paragraph_format.bullet.char = '-'  # Χαρακτήρας κουκκίδας
     para4.paragraph_format.default_portion_format.fill_type = slides.FillType.SOLID
     para4.paragraph_format.default_portion_format.fill_format.solid_fill_color.color = drawing.Color.black
     para4.paragraph_format.depth = 3  # Κουκκίδα Επιπέδου 3
     ```
   
5. **Προσθήκη παραγράφων στο πλαίσιο κειμένου**
   Μόλις διαμορφωθούν όλες οι παράγραφοι, προσθέστε τις στο πλαίσιο κειμένου:
   
   ```python
   # Προσθήκη όλων των παραγράφων στη συλλογή του πλαισίου κειμένου
   text.paragraphs.add(para1)
   text.paragraphs.add(para2)
   text.paragraphs.add(para3)
   text.paragraphs.add(para4)
   ```

6. **Αποθήκευση της παρουσίασης**
   Τέλος, αποθηκεύστε την παρουσίασή σας ως αρχείο PPTX:
   
   ```python
   # Αποθήκευση της παρουσίασης
   pres.save("YOUR_OUTPUT_DIRECTORY/text_multilevel_bullet_out.pptx", slides.export.SaveFormat.PPTX)
   ```

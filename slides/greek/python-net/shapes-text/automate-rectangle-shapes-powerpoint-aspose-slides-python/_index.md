### Δημιουργία του ορθογωνίου σχήματος
#### Επισκόπηση
Σε αυτήν την ενότητα, θα επικεντρωθούμε στην προσθήκη ενός ορθογωνίου σχήματος σε μια διαφάνεια του PowerPoint χρησιμοποιώντας το Aspose.Slides για Python.
#### Βήματα για τη δημιουργία του σχήματος
1. **Άνοιγμα ή δημιουργία παρουσίασης:**
   ```python
   import aspose.slides as slides
   
   with slides.Presentation() as pres:
       # Θα προσθέσουμε το ορθογώνιό μας εδώ
   ```
2. **Πρόσβαση στη διαφάνεια:**
   Ανακτούμε την πρώτη διαφάνεια όπου θέλουμε να προσθέσουμε το σχήμα.
   ```python
   slide = pres.slides[0]
   ```
3. **Προσθήκη ορθογωνίου σχήματος:**
   Χρησιμοποιήστε το `add_auto_shape` μέθοδος για τη δημιουργία ενός ορθογωνίου στη διαφάνεια.
   ```python
   shape = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 50, 150, 150, 50)
   ```
   - Παράμετροι: `ShapeType.RECTANGLE`, θέση x (50), θέση y (150), πλάτος (150), ύψος (50).

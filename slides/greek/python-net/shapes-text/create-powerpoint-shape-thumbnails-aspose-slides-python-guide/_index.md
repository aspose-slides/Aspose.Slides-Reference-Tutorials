#### Βήμα 2: Πρόσβαση και δημιουργία μικρογραφίας
Αποκτήστε πρόσβαση στην πρώτη διαφάνεια και στο πρώτο της σχήμα και, στη συνέχεια, δημιουργήστε μια μικρογραφία:

```python
        # Ας υποθέσουμε ότι υπάρχει τουλάχιστον μία διαφάνεια και ένα σχήμα
        shape = presentation.slides[0].shapes[0]

        # Δημιουργήστε μια μικρογραφία της εμφάνισης του σχήματος
        with shape.get_image(slides.ShapeThumbnailBounds.APPEARANCE, 1, 1) as image:
            # Αποθήκευση της μικρογραφίας ως PNG
            image.save(output_directory, slides.ImageFormat.PNG)
```

**Εξήγηση:**
- `shape.get_image(...)`: Καταγράφει μια εικόνα της εμφάνισης του σχήματος. Οι παράμετροι `(slides.ShapeThumbnailBounds.APPEARANCE, 1, 1)` Καθορίστε τη στόχευση του σχήματος που συνδέεται με την εμφάνιση με συντελεστές κλίμακας για το πλάτος και το ύψος.
- `image.save()`Αποθηκεύει τη μικρογραφία που δημιουργήθηκε σε μορφή PNG στον καθορισμένο κατάλογο εξόδου.

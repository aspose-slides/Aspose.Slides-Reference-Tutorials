Αυτό το σεμινάριο θα σας καθοδηγήσει στη χρήση του Aspose.Slides για Python για να προσθέσετε στήλες σε πλαίσια κειμένου μέσα σε διαφάνειες του PowerPoint χωρίς κόπο.

- Βασικές επιλογές διαμόρφωσης για τη βελτίωση της διάταξης του κειμένου σας

Ας ξεκινήσουμε εξετάζοντας τις προϋποθέσεις.

## Οδηγός Εφαρμογής

Αυτή η ενότητα εστιάζει στην προσθήκη στηλών σε πλαίσια κειμένου μέσα σε διαφάνειες του PowerPoint.

### Επισκόπηση χαρακτηριστικών προσθήκης στήλης

Η λειτουργία οργανώνει μεγάλα ποσά κειμένου με τάξη, διαιρώντας τα σε πολλαπλές στήλες μέσα σε ένα μόνο πλαίσιο κειμένου, βελτιώνοντας την αναγνωσιμότητα και διατηρώντας καθαρό σχεδιασμό διαφανειών.

#### Βήμα προς βήμα εφαρμογή

**1. Δημιουργήστε μια νέα παρουσίαση**

Ξεκινήστε δημιουργώντας μια παρουσία μιας παρουσίασης PowerPoint:

```python
with slides.Presentation() as presentation:
    # Πρόσβαση στην πρώτη διαφάνεια της παρουσίασης
    slide = presentation.slides[0]
```

**2. Προσθήκη Αυτόματου Σχήματος στη διαφάνεια**

Προσθέστε ένα ορθογώνιο σχήμα που θα χρησιμεύσει ως δοχείο κειμένου:

```python
# Προσθέστε ένα ορθογώνιο σχήμα στη θέση (100, 100) με μέγεθος (300x300)
shape = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 100, 100, 300, 300)
```

**3. Εισαγωγή πλαισίου κειμένου στο σχήμα**

Εισαγάγετε περιεχόμενο κειμένου στο νεοδημιουργημένο ορθογώνιο σχήμα:

```python
# Προσθέστε ένα πλαίσιο κειμένου στο ορθογώνιο με το κείμενο που επιθυμείτε
text = ("All these columns are limited to be within a single text container -- " +
         "you can add or delete text and the new or remaining text automatically adjusts " +
         "itself to flow within the container. You cannot have text flow from one container " +
         "to other though -- we told you PowerPoint's column options for text are limited!")
shape.add_text_frame(text)
```

**4. Ρύθμιση παραμέτρων στηλών σε πλαίσιο κειμένου**

Ορίστε τον αριθμό των στηλών και την απόσταση:

```python
# Πρόσβαση και ρύθμιση παραμέτρων της μορφής πλαισίου κειμένου
text_frame_format = shape.text_frame.text_frame_format

# Ορίστε τον αριθμό στηλών σε 3 και ορίστε την απόσταση μεταξύ των στηλών σε 10 σημεία
text_frame_format.column_count = 3
text_frame_format.column_spacing = 10
```

**5. Αποθήκευση της παρουσίασης**

Τέλος, αποθηκεύστε την παρουσίασή σας με τις αλλαγές που εφαρμόστηκαν:

```python
presentation.save("YOUR_OUTPUT_DIRECTORY/text_add_text_frame_out.pptx", slides.export.SaveFormat.PPTX)
```

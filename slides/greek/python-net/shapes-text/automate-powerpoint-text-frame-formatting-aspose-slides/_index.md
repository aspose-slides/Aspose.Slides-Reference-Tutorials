## Οδηγός Εφαρμογής

### Εξαγωγή δεδομένων μορφής πλαισίου κειμένου
Αυτή η λειτουργία σάς βοηθά να αποκτάτε πρόσβαση και να εμφανίζετε μέσω προγραμματισμού λεπτομέρειες μορφοποίησης πλαισίου κειμένου από μια διαφάνεια του PowerPoint.

#### Επισκόπηση της λειτουργίας:
Αυτή η διαδικασία περιλαμβάνει την πρόσβαση στο πρώτο σχήμα στην πρώτη διαφάνεια της παρουσίασής σας, την ανάκτηση των ιδιοτήτων μορφής του πλαισίου κειμένου που είναι ενεργές και την εμφάνισή τους. 

##### Βήμα προς βήμα εφαρμογή:
**1. Πρόσβαση στη διαφάνεια:**
Ξεκινήστε φορτώνοντας το αρχείο παρουσίασης και αποκτώντας πρόσβαση στη διαφάνεια και το σχήμα που θέλετε.
```python
# Φόρτωση του αρχείου παρουσίασης
current_pres = "YOUR_DOCUMENT_DIRECTORY/text_add_animation_effect.pptx"
with slides.Presentation(current_pres) as pres:
    # Πρόσβαση στο πρώτο σχήμα στην πρώτη διαφάνεια
    shape = pres.slides[0].shapes[0]
```

**2. Ανάκτηση ιδιοτήτων μορφής πλαισίου κειμένου:**
Ανάκτηση και αποθήκευση ιδιοτήτων μορφής πλαισίου κειμένου που ισχύουν από το επιλεγμένο σχήμα.
```python
# Λήψη της μορφής πλαισίου κειμένου και των αποτελεσματικών ιδιοτήτων της
if shape.text_frame is not None:
    text_frame_format = shape.text_frame.text_frame_format
    effective_text_frame_format = text_frame_format.get_effective()
```

**3. Εμφάνιση αποτελεσματικών δεδομένων:**
Εμφανίστε τον τύπο αγκύρωσης, τις ρυθμίσεις αυτόματης προσαρμογής, την κατακόρυφη στοίχιση και τα περιθώρια του πλαισίου κειμένου.
```python
# Εμφάνιση των δεδομένων μορφής πλαισίου κειμένου που ισχύουν
if effective_text_frame_format:
    print("Anchoring type: " + str(effective_text_frame_format.anchoring_type))
    print("Autofit type: " + str(effective_text_frame_format.autofit_type))
    print("Text vertical type: " + str(effective_text_frame_format.text_vertical_type))
    print("Margins")
    print("   Left: " + str(effective_text_frame_format.margin_left))
    print("   Top: " + str(effective_text_frame_format.margin_top))
    print("   Right: " + str(effective_text_frame_format.margin_right))
    print("   Bottom: " + str(effective_text_frame_format.margin_bottom))
```

**Συμβουλές αντιμετώπισης προβλημάτων:**
- Βεβαιωθείτε ότι η διαδρομή του αρχείου PowerPoint είναι σωστή για να αποφύγετε `FileNotFoundError`.
- Ελέγξτε ξανά ότι οι δείκτες διαφανειών και σχημάτων βρίσκονται εντός του εύρους της παρουσίασής σας.

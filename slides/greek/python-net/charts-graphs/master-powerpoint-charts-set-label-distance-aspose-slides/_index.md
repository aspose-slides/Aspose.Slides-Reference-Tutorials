## Οδηγός Εφαρμογής

Τώρα, ας επικεντρωθούμε στον ορισμό της απόστασης της ετικέτας από τον άξονα στο γράφημά σας.

### Προσθήκη γραφήματος ομαδοποιημένων στηλών σε μια διαφάνεια

Αρχικά, θα προσθέσουμε ένα γράφημα ομαδοποιημένων στηλών:

```python
# Πρόσβαση στην πρώτη διαφάνεια της παρουσίασης
class SlideManager:
    def __init__(self, presentation):
        self.slide = presentation.slides[0]

    def add_chart(self):
        return self.slide.shapes.add_chart(
            slides.charts.ChartType.CLUSTERED_COLUMN, 20, 20, 500, 300)

with PresentationManager() as presentation:
    slide_manager = SlideManager(presentation)
    chart = slide_manager.add_chart()
```

**Εξήγηση**Αυτός ο κώδικας δημιουργεί ένα νέο γράφημα στην πρώτη διαφάνεια, τοποθετημένο στο (20, 20) με διαστάσεις 500x300.

### Ορισμός μετατόπισης ετικέτας από τον άξονα

Στη συνέχεια, ρυθμίστε την μετατόπιση της ετικέτας:

```python
# Ορισμός μετατόπισης ετικέτας από τον άξονα για τον οριζόντιο άξονα
class ChartManager:
    def __init__(self, chart):
        self.chart = chart

    def set_label_offset(self, offset):
        self.chart.axes.horizontal_axis.label_offset = offset

chart_manager = ChartManager(chart)
chart_manager.set_label_offset(500)
```

**Εξήγηση**: Με ρύθμιση `label_offset`, διασφαλίζουμε ότι οι ετικέτες έχουν την κατάλληλη απόσταση μεταξύ τους. Η τιμή μπορεί να προσαρμοστεί ανάλογα με τις συγκεκριμένες ανάγκες σας.

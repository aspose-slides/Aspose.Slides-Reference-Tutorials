## Οδηγός Εφαρμογής

### Δημιουργία βασικού γραφήματος γραμμών με δείκτες

#### Επισκόπηση

Ξεκινήστε προσθέτοντας ένα απλό γράφημα γραμμών στη διαφάνειά σας, το οποίο θα προσαρμοστεί αργότερα.

#### Βήματα
1. **Αρχικοποίηση παρουσίασης**

    ```python
    import aspose.slides as slides

    def create_line_chart_with_markers():
        with slides.Presentation() as pres:
            slide = pres.slides[0]
    ```

2. **Προσθήκη γραφήματος γραμμών**

   Προσθήκη του γραφήματος στη θέση `(0, 0)` και μέγεθος `400x400`.

    ```python
    chart = slide.shapes.add_chart(slides.charts.ChartType.LINE_WITH_MARKERS, 0, 0, 400, 400)
    ```

3. **Πρόσβαση σε δεδομένα γραφήματος**

   Διαγράψτε τις υπάρχουσες σειρές και προσθέστε νέα σημεία δεδομένων.

    ```python
    fact = chart.chart_data.chart_data_workbook
    chart.chart_data.series.clear()
    chart.chart_data.series.add(fact.get_cell(0, 1, 1, "Series 1"), chart.type)
    ```

4. **Αποθήκευση της παρουσίασης**

   Αποθηκεύστε την εργασία σας σε ένα αρχείο.

    ```python
    pres.save("YOUR_OUTPUT_DIRECTORY/charts_marker_options_out.pptx", slides.export.SaveFormat.PPTX)
    ```

### Προσθήκη εικόνων ως δείκτες

#### Επισκόπηση

Βελτιώστε το γράφημα γραμμών σας χρησιμοποιώντας εικόνες ως δείκτες, καθιστώντας τα σημεία δεδομένων πιο ευδιάκριτα.

#### Βήματα
1. **Αρχικοποίηση παρουσίασης**

    ```python
    import aspose.slides as slides

    def add_images_to_chart():
        with slides.Presentation() as pres:
            slide = pres.slides[0]
    ```

2. **Προσθήκη γραφήματος γραμμών**

   Όπως και στην προηγούμενη ενότητα, προσθέστε ένα γράφημα γραμμών.

    ```python
    chart = slide.shapes.add_chart(slides.charts.ChartType.LINE_WITH_MARKERS, 0, 0, 400, 400)
    fact = chart.chart_data.chart_data_workbook
    ```

3. **Φόρτωση και προσθήκη εικόνων**

   Ορίστε μια συνάρτηση για τη φόρτωση εικόνων.

    ```python
    def load_and_add_image(pres, image_path):
        img = slides.Images.from_file(image_path)
        return pres.images.add_image(img)

    imgx1 = load_and_add_image(pres, "YOUR_DOCUMENT_DIRECTORY/image1.jpg")
    imgx2 = load_and_add_image(pres, "YOUR_DOCUMENT_DIRECTORY/image2.jpg")
    ```

4. **Προσθήκη σημείων δεδομένων με δείκτες εικόνας**

   Προσαρμόστε τα σημεία δεδομένων για να χρησιμοποιήσετε εικόνες ως δείκτες.

    ```python
    series = chart.chart_data.series[0]

    point = series.data_points.add_data_point_for_line_series(fact.get_cell(0, 1, 1, 4.5))
    point.marker.format.fill.fill_type = slides.FillType.PICTURE
    point.marker.format.fill.picture_fill_format.picture.image = imgx1

    # Επαναλάβετε για άλλα σημεία δεδομένων με διαφορετικές εικόνες, όπως απαιτείται
    ```

5. **Ορισμός μεγέθους δείκτη**

   Προσαρμόστε το μέγεθος των δεικτών στη σειρά.

    ```python
    series.marker.size = 15
    ```

6. **Αποθήκευση της παρουσίασης**

   Αποθηκεύστε την παρουσίασή σας με προσθήκη δεικτών εικόνας.

    ```python
    pres.save("YOUR_OUTPUT_DIRECTORY/charts_with_image_markers_out.pptx", slides.export.SaveFormat.PPTX)
    ```

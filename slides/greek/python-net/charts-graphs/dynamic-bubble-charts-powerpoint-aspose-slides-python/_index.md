## Οδηγός Εφαρμογής

### Δημιουργία ενός δυναμικού γραφήματος φυσαλίδων

Ας εμβαθύνουμε στη δημιουργία ενός δυναμικού γραφήματος φυσαλίδων στο PowerPoint χρησιμοποιώντας το Aspose.Slides. Αυτή η λειτουργία σάς επιτρέπει να απεικονίζετε σημεία δεδομένων με ποικίλα μεγέθη, καθιστώντας την ιδανική για τη σύγκριση πολλαπλών διαστάσεων συνόλων δεδομένων.

#### Προσθήκη του γραφήματος

**Βήμα 1: Αρχικοποίηση παρουσίασης**

Ξεκινήστε δημιουργώντας ή ανοίγοντας μια παρουσίαση όπου θα προστεθεί το γράφημα:

```python
with slides.Presentation() as pres:
    slide = pres.slides[0]  # Πρόσβαση στην πρώτη διαφάνεια
```

**Βήμα 2: Προσθήκη δυναμικού γραφήματος φυσαλίδων**

Προσθέστε το δυναμικό γράφημα φυσαλίδων στην επιλεγμένη διαφάνεια σε συγκεκριμένες συντεταγμένες με καθορισμένες διαστάσεις:

```python
chart = slide.shapes.add_chart(
    slides.charts.ChartType.BUBBLE, 100, 100, 400, 300
)
```

Αυτό το απόσπασμα κώδικα δημιουργεί ένα δυναμικό γράφημα φυσαλίδων τοποθετημένο στο (100, 100) στη διαφάνεια με πλάτος 400 και ύψος 300.

#### Ρύθμιση κλίμακας μεγέθους φυσαλίδας

**Βήμα 3: Ορισμός μεγέθους φυσαλίδας**

Βελτιστοποιήστε την οπτικοποίηση των δεδομένων σας προσαρμόζοντας την κλίμακα μεγέθους για τις φυσαλίδες στην πρώτη ομάδα σειρών:

```python
chart.chart_data.series_groups[0].bubble_size_scale = 150
```

Αυτή η προσαρμογή κλιμακώνει τα μεγέθη των φυσαλίδων, βελτιώνοντας τη σαφήνεια και την οπτική επίδραση.

#### Αποθήκευση της παρουσίασής σας

**Βήμα 4: Αποθήκευση του αρχείου**

Αφού κάνετε τις προσαρμογές σας, αποθηκεύστε την παρουσίαση για να διατηρήσετε τις αλλαγές σας:

```python
pres.save('dynamic_bubble_chart_scaling_out.pptx', slides.export.SaveFormat.PPTX)
```

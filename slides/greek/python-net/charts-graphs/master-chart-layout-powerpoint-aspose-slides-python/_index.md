## Οδηγός Εφαρμογής: Ρύθμιση Λειτουργίας Διάταξης Γραφήματος

Ας αναλύσουμε πώς να ορίσουμε τη λειτουργία διάταξης ενός γραφήματος μέσα σε μια παρουσίαση PowerPoint.

### Δημιουργία και πρόσβαση σε μια διαφάνεια

Ξεκινήστε δημιουργώντας μια νέα παρουσίαση PowerPoint και αποκτώντας πρόσβαση στην πρώτη της διαφάνεια:

```python
with slides.Presentation() as presentation:
    slide = presentation.slides[0]
```

Αυτό ρυθμίζει το περιβάλλον σας για την προσθήκη γραφημάτων.

### Προσθήκη γραφήματος ομαδοποιημένων στηλών

Προσθήκη ενός γραφήματος ομαδοποιημένων στηλών στην καθορισμένη θέση στη διαφάνεια:

```python
chart = slide.shapes.add_chart(
    slides.charts.ChartType.CLUSTERED_COLUMN, 20, 100, 600, 400
)
```

Παράμετροι:
- `ChartType.CLUSTERED_COLUMN`: Ορίζει τον τύπο του γραφήματος.
- `(20, 100)`Οι συντεταγμένες x και y όπου τοποθετείται το γράφημα στη διαφάνεια.
- `(600, 400)`: Πλάτος και ύψος του γραφήματος σε σημεία.

### Προσαρμογή ιδιοτήτων διάταξης

Τώρα, προσαρμόστε τις ιδιότητες διάταξης της περιοχής σχεδίασης για να ορίσετε τη θέση και το μέγεθός της:

```python
chart.plot_area.as_i_layoutable.x = 0.2
chart.plot_area.as_i_layoutable.y = 0.2
chart.plot_area.as_i_layoutable.width = 0.7
chart.plot_area.as_i_layoutable.height = 0.7
```

Αυτές οι τιμές είναι σχετικές μονάδες, διασφαλίζοντας ότι το γράφημα προσαρμόζεται δυναμικά σε διαφορετικά μεγέθη διαφανειών.

### Καθορισμός τύπου στόχου διάταξης

Ορίστε τον τύπο στόχου διάταξης για ακριβή έλεγχο του τρόπου συμπεριφοράς της περιοχής σχεδίασης:

```python
chart.plot_area.layout_target_type = slides.charts.LayoutTargetType.INNER
```

Αυτή η διαμόρφωση διασφαλίζει ότι η περιοχή του γραφήματος είναι κεντραρισμένη μέσα στο κοντέινερ της, διατηρώντας μια καθαρή εμφάνιση.

### Αποθήκευση της παρουσίασής σας

Τέλος, αποθηκεύστε την παρουσίασή σας σε έναν καθορισμένο κατάλογο εξόδου:

```python
output_directory = 'YOUR_OUTPUT_DIRECTORY/'
presentation.save(output_directory + 'charts_set_layout_mode_out.pptx', slides.export.SaveFormat.PPTX)
```

## Οδηγός Εφαρμογής
Σε αυτήν την ενότητα, θα αναλύσουμε κάθε βήμα που απαιτείται για τη δημιουργία και την προσαρμογή ενός γραφήματος μετοχών.

### Προσθήκη γραφήματος μετοχών
Αρχικά, ας προσθέσουμε το διάγραμμα μετοχών στην παρουσίασή σας:

```python
with slides.Presentation() as pres:
    # Προσθήκη γραφήματος μετοχών στη θέση (50, 50) με μέγεθος (600, 400)
    chart = pres.slides[0].shapes.add_chart(slides.charts.ChartType.OPEN_HIGH_LOW_CLOSE, 50, 50, 600, 400, False)

    # Διαγραφή υπαρχόντων δεδομένων
    chart.chart_data.series.clear()
    chart.chart_data.categories.clear()

    # Πρόσβαση στο βιβλίο εργασίας για χειρισμό κελιών
    wb = chart.chart_data.chart_data_workbook
```

### Διαμόρφωση κατηγοριών και σειρών
Στη συνέχεια, θα διαμορφώσουμε κατηγορίες και σειρές για να διατηρήσουμε τα δεδομένα αποθέματός σας:

```python
# Προσθήκη κατηγοριών (A, B, C)
chart.chart_data.categories.add(wb.get_cell(0, 1, 0, "A"))
chart.chart_data.categories.add(wb.get_cell(0, 2, 0, "B"))
chart.chart_data.categories.add(wb.get_cell(0, 3, 0, "C"))

# Προσθήκη σειράς για δεδομένα ανοίγματος, υψηλού, χαμηλού και κλεισίματος
series_names = ["Open", "High", "Low", "Close"]
for i, name in enumerate(series_names):
    chart.chart_data.series.add(wb.get_cell(0, 0, i + 1, name), chart.type)
```

### Προσθήκη σημείων δεδομένων
Τώρα, ας συμπληρώσουμε τη σειρά με σημεία δεδομένων:

```python
# Δεδομένα για «Άνοιγμα», «Υψηλό», «Χαμηλό» και «Κλείσιμο»
data = [
    [72, 172, 12, 25],
    [25, 57, 12, 38],
    [38, 57, 13, 50]
]

# Αντιστοίχιση δεδομένων σε κάθε σειρά
for i in range(4):
    series = chart.chart_data.series[i]
    for j in range(3):
        series.data_points.add_data_point_for_stock_series(wb.get_cell(0, j + 1, i + 1, data[j][i]))
```

### Προσαρμογή εμφάνισης γραφήματος
Βελτιώστε την οπτική ελκυστικότητα του γραφήματος μετοχών σας:

```python
# Ενεργοποίηση γραμμών πάνω-κάτω και ορισμός μορφής γραμμής πάνω-κάτω
chart.chart_data.series_groups[0].up_down_bars.has_up_down_bars = True
chart.chart_data.series_groups[0].hi_low_lines_format.line.fill_format.fill_type = slides.FillType.SOLID

# Ορίστε τις γραμμές σειράς σε μηδενικό γέμισμα για πιο καθαρή εμφάνιση
for ser in chart.chart_data.series:
    ser.format.line.fill_format.fill_type = slides.FillType.NO_FILL
```

### Αποθήκευση της παρουσίασης
Τέλος, αποθηκεύστε την παρουσίασή σας με το νέο γράφημα μετοχών:

```python
# Αποθήκευση της παρουσίασης σε δίσκο
pres.save("charts_stock_chart_out.pptx", slides.export.SaveFormat.PPTX)
```

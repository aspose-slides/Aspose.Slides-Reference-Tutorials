Θα αναλύσουμε την υλοποίηση σε βασικά χαρακτηριστικά. Κάθε ενότητα παρέχει μια λεπτομερή εξήγηση για το τι μπορείτε να επιτύχετε με το Aspose.Slides.

### Δημιουργήστε ένα γράφημα Sunburst στο PowerPoint
#### Επισκόπηση
Η δημιουργία ενός γραφήματος στο PowerPoint είναι απλή με το Aspose.Slides, το οποίο επιτρέπει τον ακριβή έλεγχο της θέσης και του μεγέθους.

#### Βήματα Υλοποίησης
1. **Αρχικοποίηση παρουσίασης**Ξεκινήστε δημιουργώντας ένα νέο αντικείμενο παρουσίασης.
2. **Προσθήκη γραφήματος**Εισαγωγή ενός γραφήματος Sunburst στην πρώτη διαφάνεια στις καθορισμένες συντεταγμένες.

```python
import aspose.slides as slides

with slides.Presentation() as pres:
    chart = pres.slides[0].shapes.add_chart(slides.charts.ChartType.SUNBURST, 100, 100, 450, 400)
```

**Επεξήγηση παραμέτρων:**
- `ChartType.SUNBURST`: Καθορίζει τον τύπο του γραφήματος.
- Συντεταγμένες `(100, 100)`: Θέση στη διαφάνεια.
- Μέγεθος `(450, 400)`: Διαστάσεις του διαγράμματος.

### Προσαρμογή ετικετών σημείων δεδομένων σε γραφήματα
#### Επισκόπηση
Η προσαρμογή ετικετών σημείων δεδομένων μπορεί να βελτιώσει τη σαφήνεια και την εστίαση, εμφανίζοντας συγκεκριμένες πληροφορίες, όπως τιμές ή ονόματα σειρών.

#### Βήματα Υλοποίησης
1. **Σημεία δεδομένων πρόσβασης**Ανάκτηση των σημείων δεδομένων από την πρώτη σειρά.
2. **Εμφάνιση τιμών**Ενεργοποίηση εμφάνισης τιμών για ένα συγκεκριμένο σημείο δεδομένων.
3. **Τροποποίηση ιδιοτήτων ετικέτας**: Προσαρμόστε τις ρυθμίσεις ετικέτας για να εμφανίσετε το όνομα της κατηγορίας, το όνομα της σειράς και να αλλάξετε το χρώμα του κειμένου.

```python
import aspose.slides as slides
import aspose.pydrawing as drawing

def customize_data_point_labels():
    with slides.Presentation() as pres:
        chart = pres.slides[0].shapes.add_chart(slides.charts.ChartType.SUNBURST, 100, 100, 450, 400)
        data_points = chart.chart_data.series[0].data_points
        
        # Εμφάνιση τιμής για ένα συγκεκριμένο σημείο δεδομένων
        data_points[3].data_point_levels[0].label.data_label_format.show_value = True

        # Προσαρμόστε τις ιδιότητες ετικέτας για έναν άλλο κλάδο
        branch1_label = data_points[0].data_point_levels[2].label
        branch1_label.data_label_format.show_category_name = False
        branch1_label.data_label_format.show_series_name = True
        branch1_label.data_label_format.text_format.portion_format.fill_format.fill_type = slides.FillType.SOLID
        branch1_label.data_label_format.text_format.portion_format.fill_format.solid_fill_color.color = drawing.Color.yellow
```

**Βασικές διαμορφώσεις:**
- Χρήση `data_label_format` για εναλλαγή των επιλογών εμφάνισης.
- Εφαρμόστε χρώμα χρησιμοποιώντας το `FillType` και `Color` τάξεις.

### Αλλαγή χρώματος γεμίσματος ενός σημείου δεδομένων
#### Επισκόπηση
Η αλλαγή του χρώματος γεμίσματος μπορεί να επισημάνει συγκεκριμένα σημεία δεδομένων, κάνοντάς τα να ξεχωρίζουν στο γράφημά σας.

#### Βήματα Υλοποίησης
1. **Σημεία δεδομένων πρόσβασης**: Λάβετε το σημείο δεδομένων που θέλετε να προσαρμόσετε.
2. **Ορισμός τύπου και χρώματος γεμίσματος**: Τροποποιήστε τις ρυθμίσεις γεμίσματος για να εφαρμόσετε νέα χρώματα.

```python
def change_data_point_fill_color():
    with slides.Presentation() as pres:
        chart = pres.slides[0].shapes.add_chart(slides.charts.ChartType.SUNBURST, 100, 100, 450, 400)
        data_points = chart.chart_data.series[0].data_points
        
        # Αλλαγή χρώματος γεμίσματος για ένα συγκεκριμένο σημείο δεδομένων
        steam4_format = data_points[9].format
        steam4_format.fill.fill_type = slides.FillType.SOLID
        steam4_format.fill.solid_fill_color.color = drawing.Color.from_argb(0, 176, 240, 255)
```

**Επεξήγηση παραμέτρων:**
- `fill.fill_type`: Ορίζει τον τύπο γεμίσματος (π.χ., συμπαγές).
- `from_argb()`: Ορίζει το χρώμα χρησιμοποιώντας τιμές άλφα, κόκκινο, πράσινο και μπλε.

def save_presentation():
    with slides.Presentation() as pres:
        chart = pres.slides[0].shapes.add_chart(slides.charts.ChartType.SUNBURST, 100, 100, 450, 400)

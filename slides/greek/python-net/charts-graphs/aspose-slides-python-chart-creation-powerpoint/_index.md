### Δημιουργία γραφήματος ομαδοποιημένων στηλών

#### Επισκόπηση
Ένα γράφημα ομαδοποιημένων στηλών είναι εξαιρετικό για τη σύγκριση δεδομένων μεταξύ κατηγοριών. Ας προσθέσουμε ένα στην πρώτη σας διαφάνεια στη θέση (100, 100) με διαστάσεις 400x300.

```python
import aspose.slides as slides
from aspose.slides import Presentation, SaveFormat

with Presentation() as pres:
    # Προσθήκη γραφήματος ομαδοποιημένων στηλών
    chart = pres.slides[0].shapes.add_chart(
        slides.charts.ChartType.CLUSTERED_COLUMN,
        100, 100, 400, 300
    )
```

#### Εξήγηση
- **Τύπος γραφήματος.CLUSTERED_COLUMN**: Καθορίζει τον τύπο του γραφήματος.
- **Θέση και Διαστάσεις**: (100, 100) για θέση· 400x300 για μέγεθος.

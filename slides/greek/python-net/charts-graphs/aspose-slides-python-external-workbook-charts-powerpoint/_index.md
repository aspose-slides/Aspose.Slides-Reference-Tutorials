## Οδηγός Εφαρμογής

### Χαρακτηριστικό 1: Αντιγραφή εξωτερικού βιβλίου εργασίας

#### Επισκόπηση
Η αντιγραφή ενός εξωτερικού βιβλίου εργασίας είναι απαραίτητη για να διασφαλίσετε ότι η παρουσίασή σας αναφέρεται στο πιο πρόσφατο σύνολο δεδομένων. Αυτή η λειτουργία δείχνει πώς να αντιγράψετε ένα αρχείο από έναν κατάλογο προέλευσης σε έναν προορισμό χρησιμοποιώντας την Python. `shutil` μονάδα μέτρησης.

#### Βήματα για την εφαρμογή
**Βήμα 1**Εισαγωγή απαραίτητων ενοτήτων
```python
import shutil
```

**Βήμα 2**: Ορισμός συνάρτησης αντιγραφής βιβλίου εργασίας
Δημιουργήστε μια συνάρτηση για να χειριστείτε τη διαδικασία αντιγραφής:
```python
def copy_external_workbook():
    external_workbook_file_name = "charts_external_workbook.xlsx"
    # Χρησιμοποιήστε το shutil.copyfile για να μετακινήσετε το αρχείο από την πηγή στον προορισμό
    shutil.copyfile(
        "YOUR_DOCUMENT_DIRECTORY/" + external_workbook_file_name,
        "YOUR_OUTPUT_DIRECTORY/" + external_workbook_file_name
    )
```
- **Παράμετροι**: `shutil.copyfile(source, destination)` όπου `source` είναι η αρχική διαδρομή του αρχείου σας και `destination` είναι ο κατάλογος προορισμού.

### Χαρακτηριστικό 2: Δημιουργία παρουσίασης με εξωτερικό γράφημα βιβλίου εργασίας

#### Επισκόπηση
Αυτή η δυνατότητα περιλαμβάνει τη δημιουργία μιας παρουσίασης PowerPoint και την προσθήκη ενός γραφήματος που αναφέρεται σε ένα εξωτερικό βιβλίο εργασίας, επιτρέποντας δυναμικές ενημερώσεις κάθε φορά που αλλάζουν τα δεδομένα προέλευσης.

#### Βήματα για την εφαρμογή
**Βήμα 1**Εισαγωγή ενότητας Aspose.Slides
```python
import aspose.slides as slides
```

**Βήμα 2**: Ορισμός συνάρτησης δημιουργίας παρουσίασης
Κατασκευάστε μια συνάρτηση για να δημιουργήσετε την παρουσίασή σας με γραφήματα:
```python
def create_presentation_with_external_chart():
    # Άνοιγμα ή δημιουργία νέας παρουσίασης
    with slides.Presentation() as pres:
        # Προσθήκη κυκλικού γραφήματος σε καθορισμένες συντεταγμένες και μέγεθος
        chart = pres.slides[0].shapes.add_chart(slides.charts.ChartType.PIE, 50, 50, 500, 400)

        # Διαγραφή υπαρχόντων δεδομένων στο βιβλίο εργασίας
        chart.chart_data.chart_data_workbook.clear(0)

        # Ορισμός εξωτερικού βιβλίου εργασίας για το γράφημα
        chart.chart_data.set_external_workbook("YOUR_OUTPUT_DIRECTORY/charts_external_workbook.xlsx")

        # Ορισμός περιοχής κελιών από το "Φύλλο1" για χρήση ως πηγή δεδομένων
        chart.chart_data.set_range("Sheet1!$A$2:$B$5")

        # Ορισμός παραλλαγής χρώματος για την πρώτη σειρά στο γράφημα
        series = chart.chart_data.series[0]
        series.parent_series_group.is_color_varied = True

        # Αποθήκευση της παρουσίασης με καθορισμένο όνομα και μορφή
        pres.save("YOUR_OUTPUT_DIRECTORY/charts_create_external_workbook_out.pptx", slides.export.SaveFormat.PPTX)
```
- **Παράμετροι**:
  - `slides.charts.ChartType`: Ορίζει τον τύπο του γραφήματος.
  - `set_external_workbook(path)`: Ορίζει τη διαδρομή προς το εξωτερικό βιβλίο εργασίας σας.
  - `set_range(range_string)`: Καθορίζει ποια κελιά στο Excel θα χρησιμοποιηθούν για δεδομένα.

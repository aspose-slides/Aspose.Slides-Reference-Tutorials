## Οδηγός Εφαρμογής

Ας δούμε πώς μπορείτε να προσθέσετε ένα γράφημα στην παρουσίασή σας.

### Δημιουργία νέας παρουσίασης με γράφημα

#### Επισκόπηση

Θα δημιουργήσουμε μια νέα παρουσίαση και θα προσθέσουμε ένα γράφημα περιοχής. Αυτή η ενότητα καλύπτει τη ρύθμιση των δεδομένων του γραφήματος και τη διαμόρφωση της εμφάνισής τους.

#### Βήμα προς βήμα εφαρμογή

**1. Αρχικοποίηση της παρουσίασης**

Δημιουργήστε ένα `Presentation` αντικείμενο για εργασία σε διαφάνειες και σχήματα:

```python
def initialize_presentation():
    with slides.Presentation() as pres:
        # Ο κωδικός σας πηγαίνει εδώ
```

**2. Προσθέστε ένα γράφημα περιοχής στην πρώτη διαφάνεια**

Προσθέστε ένα γράφημα σε καθορισμένες συντεταγμένες και μέγεθος στην πρώτη διαφάνεια χρησιμοποιώντας `add_chart`:

```python
def add_area_chart(pres):
    chart = pres.slides[0].shapes.add_chart(
        slides.charts.ChartType.AREA, 50, 50, 450, 300
    )
```

**3. Βιβλίο εργασίας δεδομένων γραφήματος Access**

Αποκτήστε πρόσβαση στο βιβλίο εργασίας για να χειριστείτε δεδομένα γραφήματος:

```python
def get_workbook(chart):
    return chart.chart_data.chart_data_workbook
```

**4. Διαγραφή υπαρχουσών κατηγοριών και σειρών**

Διαγράψτε τυχόν υπάρχουσες κατηγορίες ή σειρές στο γράφημα:

```python
def clear_chart_data(chart):
    chart.chart_data.categories.clear()
    chart.chart_data.series.clear()
```

**5. Προσθέστε ημερομηνίες ως κατηγορίες**

Χρησιμοποιήστε την Python `datetime` ενότητα για τη συμπλήρωση κατηγοριών βάσει ημερομηνίας:

```python
def add_date_categories(wb, chart):
    from datetime import date
    
    chart.chart_data.categories.add(wb.get_cell(0, "A2", date(2015, 1, 1)))
    chart.chart_data.categories.add(wb.get_cell(0, "A3", date(2016, 1, 1)))
    chart.chart_data.categories.add(wb.get_cell(0, "A4", date(2017, 1, 1)))
    chart.chart_data.categories.add(wb.get_cell(0, "A5", date(2018, 1, 1)))
```

**6. Προσθήκη σειράς γραμμών**

Εισαγωγή και συμπλήρωση μιας νέας σειράς με σημεία δεδομένων:

```python
def add_line_series(wb, chart):
    series = chart.chart_data.series.add(slides.charts.ChartType.LINE)
    
    series.data_points.add_data_point_for_line_series(wb.get_cell(0, "B2", 1))
    series.data_points.add_data_point_for_line_series(wb.get_cell(0, "B3", 2))
    series.data_points.add_data_point_for_line_series(wb.get_cell(0, "B4", 3))
    series.data_points.add_data_point_for_line_series(wb.get_cell(0, "B5", 4))
```

**7. Διαμόρφωση του άξονα κατηγορίας**

Ορίστε τον άξονα κατηγορίας ώστε να εμφανίζει ημερομηνίες σε συγκεκριμένη μορφή:

```python
def configure_category_axis(chart):
    chart.axes.horizontal_axis.category_axis_type = slides.charts.CategoryAxisType.DATE
    chart.axes.horizontal_axis.is_number_format_linked_to_source = False
    chart.axes.horizontal_axis.number_format = "yyyy"
```

**8. Αποθήκευση της παρουσίασης**

Αποθηκεύστε την παρουσίασή σας σε έναν κατάλογο εξόδου:

```python
def save_presentation(pres, path):
    pres.save(path, slides.export.SaveFormat.PPTX)
```

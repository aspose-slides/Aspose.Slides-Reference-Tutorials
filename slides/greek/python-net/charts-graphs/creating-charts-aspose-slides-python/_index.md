# Δημιουργία γραφημάτων σε Python με το Aspose.Slides: Ένας ολοκληρωμένος οδηγός

## Εισαγωγή
Η δημιουργία οπτικά ελκυστικών γραφημάτων στις παρουσιάσεις σας μπορεί να κάνει τα δεδομένα πιο εύπεπτα, επιτρέποντάς σας να μεταφέρετε πολύπλοκες πληροφορίες χωρίς κόπο. Αυτό το σεμινάριο θα σας καθοδηγήσει στη δημιουργία και τη διαμόρφωση γραφημάτων χρησιμοποιώντας το Aspose.Slides για Python—μια ισχυρή βιβλιοθήκη που μεταμορφώνει τον τρόπο που σχεδιάζετε παρουσιάσεις προσφέροντας ισχυρές λειτουργίες για τον χειρισμό γραφημάτων.

**Τι θα μάθετε:**
- Πώς να δημιουργήσετε ένα γράφημα σωρευμένων στηλών σε μια παρουσίαση
- Προσθήκη και μορφοποίηση σειρών δεδομένων με προσαρμοσμένες ετικέτες
- Αποθήκευση της διαμορφωμένης παρουσίασής σας

Μέχρι το τέλος αυτού του σεμιναρίου, θα έχετε αποκτήσει πρακτική εμπειρία χρησιμοποιώντας το Aspose.Slides Python για να βελτιώσετε τις παρουσιάσεις σας. Ας εμβαθύνουμε στη ρύθμιση του περιβάλλοντός σας πριν ξεκινήσουμε να δημιουργούμε μερικά εκπληκτικά γραφήματα!

## Προαπαιτούμενα
Πριν ξεκινήσουμε, βεβαιωθείτε ότι πληροίτε τις ακόλουθες προϋποθέσεις:

1. **Περιβάλλον Python:** Θα πρέπει να έχετε εγκατεστημένη την Python στο σύστημά σας (συνιστάται η έκδοση 3.x).
2. **Aspose.Slides για Python:** Αυτό μπορεί να εγκατασταθεί μέσω pip.
3. **Απόκτηση Άδειας:** Ενώ είναι διαθέσιμη μια δωρεάν δοκιμαστική περίοδος, σκεφτείτε να αποκτήσετε μια προσωρινή ή πλήρη άδεια χρήσης για να ξεκλειδώσετε όλες τις λειτουργίες.

Για να ξεκινήσετε να χρησιμοποιείτε το Aspose.Slides στα έργα σας, πρέπει να εγκαταστήσετε τη βιβλιοθήκη και να κατανοήσετε πώς να ρυθμίσετε το περιβάλλον σας:

```bash
pip install aspose.slides
```

Μετά την εγκατάσταση, μπορείτε να αρχικοποιήσετε και να χρησιμοποιήσετε το Aspose.Slides εισάγοντάς το στο σκριπτ σας. Για να αξιοποιήσετε πλήρως τις δυνατότητές του, αποκτήστε μια άδεια χρήσης. Διατίθεται δωρεάν δοκιμαστική περίοδος ή, για πιο εκτεταμένη χρήση, εξετάστε το ενδεχόμενο αγοράς ή υποβολής αίτησης για μια προσωρινή άδεια χρήσης.

## Οδηγός Εφαρμογής

### Λειτουργία 1: Δημιουργία και διαμόρφωση παρουσίασης με γραφήματα
**Επισκόπηση:** Αυτή η ενότητα σας καθοδηγεί στη δημιουργία μιας διαφάνειας παρουσίασης και στην προσθήκη ενός γραφήματος σε αυτήν χρησιμοποιώντας το Aspose.Slides Python.

#### Βήμα 1: Αρχικοποίηση της παρουσίασης
Ξεκινήστε δημιουργώντας ένα νέο αντικείμενο παρουσίασης. Χρησιμοποιήστε το `with` Δήλωση για αυτόματη διαχείριση πόρων:
```python
import aspose.slides as slides

with slides.Presentation() as presentation:
    # Πρόσβαση στην πρώτη διαφάνεια της παρουσίασης
    slide = presentation.slides[0]
```

#### Βήμα 2: Προσθήκη γραφήματος στη διαφάνεια
Εδώ, προσθέτουμε ένα γράφημα σωρευμένων στηλών σε μια καθορισμένη θέση με καθορισμένες διαστάσεις:
```python
# Προσθήκη γραφήματος σωρευμένων στηλών στη διαφάνεια
chart = slide.shapes.add_chart(slides.charts.ChartType.PERCENTS_STACKED_COLUMN, 20, 20, 500, 400)
```

#### Βήμα 3: Ρύθμιση παραμέτρων αξόνων γραφήματος
Ρυθμίστε τη μορφή αρίθμησης του κατακόρυφου άξονα για καλύτερη αναπαράσταση δεδομένων:
```python
# Ρύθμιση παραμέτρων της μορφής αριθμού του κατακόρυφου άξονα
chart.axes.vertical_axis.is_number_format_linked_to_source = False
chart.axes.vertical_axis.number_format = "0.00%"
```

### Λειτουργία 2: Προσθήκη και μορφοποίηση σειρών δεδομένων σε γράφημα
**Επισκόπηση:** Αυτή η ενότητα εστιάζει στην προσθήκη μιας σειράς δεδομένων, στη συμπλήρωσή της με τιμές και στην προσαρμογή της εμφάνισής της.

#### Βήμα 1: Ορισμός του βιβλίου εργασίας δεδομένων
Αρχικοποιήστε το βιβλίο εργασίας δεδομένων του γραφήματός σας:
```python
default_worksheet_index = 0
workbook = chart.chart_data.chart_data_workbook
```

#### Βήμα 2: Προσθήκη και συμπλήρωση σειρών δεδομένων
Προσθέστε μια νέα σειρά με το όνομα "Κόκκινα" στο γράφημά σας και, στη συνέχεια, συμπληρώστε την με σημεία δεδομένων:
```python
# Προσθήκη νέας σειράς και συμπλήρωση με σημεία δεδομένων
series = chart.chart_data.series.add(workbook.get_cell(default_worksheet_index, 0, 1, "Reds"), chart.type)

for i in range(1, 5):
    series.data_points.add_data_point_for_bar_series(
        workbook.get_cell(default_worksheet_index, i, 1, [0.30, 0.50, 0.80, 0.65][i-1])
    )
```

#### Βήμα 3: Μορφοποίηση της εμφάνισης της σειράς
Προσαρμόστε το χρώμα γεμίσματος και τη μορφή της ετικέτας δεδομένων:
```python
# Ορισμός γεμίσματος σειράς σε κόκκινο
series.format.fill.fill_type = slides.FillType.SOLID
series.format.fill.solid_fill_color.color = drawing.Color.red

# Ρύθμιση παραμέτρων ετικετών δεδομένων για την εμφάνιση ποσοστού
series.labels.default_data_label_format.show_value = True
series.labels.default_data_label_format.number_format = "0.0%"
```

### Λειτουργία 3: Προσθήκη και μορφοποίηση δεύτερης σειράς δεδομένων σε γράφημα
**Επισκόπηση:** Αυτή η ενότητα επεκτείνει την προσθήκη μιας δεύτερης σειράς δεδομένων με το δικό της στυλ.

#### Βήμα 1: Προσθέστε τη δεύτερη σειρά
Προσθέστε μια άλλη σειρά με το όνομα "Blues":
```python
# Προσθήκη δεύτερης σειράς με τίτλο "Blues"
series2 = chart.chart_data.series.add(workbook.get_cell(default_worksheet_index, 0, 2, "Blues"), chart.type)
```

#### Βήμα 2: Συμπλήρωση και μορφοποίηση της σειράς
Συμπληρώστε το με σημεία δεδομένων και εφαρμόστε μορφοποίηση:
```python
# Συμπλήρωση δεύτερης σειράς
for i in range(1, 5):
    series2.data_points.add_data_point_for_bar_series(
        workbook.get_cell(default_worksheet_index, i, 2, [0.70, 0.50, 0.20, 0.35][i-1])
    )

# Ορίστε το γέμισμα σε μπλε και διαμορφώστε τις ετικέτες
series2.format.fill.fill_type = slides.FillType.SOLID
series2.format.fill.solid_fill_color.color = drawing.Color.blue

series2.labels.default_data_label_format.show_value = True
```

### Λειτουργία 4: Αποθήκευση παρουσίασης σε δίσκο
**Επισκόπηση:** Μόλις διαμορφωθεί το γράφημά σας, αποθηκεύστε την παρουσίαση.

#### Βήμα 1: Αποθηκεύστε την εργασία σας
Χρησιμοποιήστε το `save` μέθοδος αποθήκευσης του αρχείου σας:
```python
# Αποθήκευση της παρουσίασης σε δίσκο
directory = "YOUR_OUTPUT_DIRECTORY"
presentation.save(f"{directory}/charts_set_data_labels_percentage_sign_out.pptx", slides.export.SaveFormat.PPTX)
```

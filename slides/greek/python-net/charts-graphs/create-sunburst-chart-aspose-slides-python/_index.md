# Πώς να δημιουργήσετε γραφήματα Sunburst σε Python χρησιμοποιώντας το Aspose.Slides

Στον σημερινό κόσμο που βασίζεται στα δεδομένα, εργαλεία όπως το Aspose.Slides απλοποιούν την ενσωμάτωση προηγμένων δυνατοτήτων δημιουργίας γραφημάτων στις εφαρμογές σας. Ακολουθήστε αυτόν τον οδηγό από την εγκατάσταση έως την υλοποίηση, διασφαλίζοντας ότι ακόμη και οι αρχάριοι μπορούν να δημιουργήσουν ελκυστικά γραφήματα ηλιοφάνειας χωρίς κόπο.

## Οδηγός Εφαρμογής
### Δημιουργία του γραφήματος Sunburst
Ας αναλύσουμε τα βήματα που απαιτούνται για τη δημιουργία και τη διαμόρφωση του γραφήματος ηλιοφάνειας χρησιμοποιώντας το Aspose.Slides.

#### Βήμα 1: Αρχικοποίηση ενός αντικειμένου παρουσίασης
Ξεκινήστε δημιουργώντας ένα νέο αντικείμενο παρουσίασης, το οποίο λειτουργεί ως κοντέινερ για τις διαφάνειες και τα γραφήματά σας:

```python
def create_sunburst_chart():
    with slides.Presentation() as pres:
        # Αυτό δημιουργεί έναν διαχειριστή περιβάλλοντος για τον χειρισμό του κύκλου ζωής της παρουσίασης.
```

#### Βήμα 2: Προσθέστε το γράφημα Sunburst
Προσθέστε ένα διάγραμμα ηλιοφάνειας σε καθορισμένες συντεταγμένες μέσα στην πρώτη σας διαφάνεια. Προσαρμόστε τη θέση και το μέγεθός του όπως απαιτείται:

```python
        chart = pres.slides[0].shapes.add_chart(
            slides.charts.ChartType.SUNBURST, 50, 50, 500, 400)
        
        # Παράμετροι: Τύπος γραφήματος, θέση x, θέση y, πλάτος, ύψος
```

#### Βήμα 3: Εκκαθάριση υπαρχόντων δεδομένων
Πριν συμπληρώσετε το γράφημά σας με δεδομένα, διαγράψτε τυχόν προεπιλεγμένες κατηγορίες και σειρές για να ξεκινήσετε από την αρχή:

```python
        chart.chart_data.categories.clear()
        chart.chart_data.series.clear()
        
        # Πρόσβαση στο βιβλίο εργασίας για χειρισμό δεδομένων γραφήματος
        wb = chart.chart_data.chart_data_workbook
        wb.clear(0)  # Καθαρίζει όλα τα κελιά στο βιβλίο εργασίας
```

#### Βήμα 4: Ρύθμιση παραμέτρων κατηγοριών και επιπέδων ομαδοποίησης
Ορίστε ιεραρχικές κατηγορίες προσθέτοντας φύλλα, μίσχους και κλαδιά. Χρησιμοποιήστε επίπεδα ομαδοποίησης για να οργανώσετε οπτικά τα δεδομένα σας:

```python
        # Διαμόρφωση Υποκαταστήματος 1
        leaf = chart.chart_data.categories.add(wb.get_cell(0, "C1", "Leaf1"))
        leaf.grouping_levels.set_grouping_item(1, "Stem1")
        leaf.grouping_levels.set_grouping_item(2, "Branch1")

        # Προσθέστε επιπλέον φύλλα κάτω από το κλαδί 1
        chart.chart_data.categories.add(wb.get_cell(0, "C2", "Leaf2"))
```

Συνεχίστε αυτό το μοτίβο για άλλα κλαδιά και φύλλα, όπως απαιτείται.

#### Βήμα 5: Προσθήκη Σειράς Δεδομένων
Δημιουργήστε μια σειρά δεδομένων και συμπληρώστε την με τιμές. Αυτό το βήμα συνδέει τις κατηγορίες σας με τα αντίστοιχα σημεία δεδομένων:

```python
        series = chart.chart_data.series.add(slides.charts.ChartType.SUNBURST)
        series.labels.default_data_label_format.show_category_name = True
        
        # Προσθήκη σημείων δεδομένων στη σειρά
        series.data_points.add_data_point_for_sunburst_series(wb.get_cell(0, "D1", 4))
```

#### Βήμα 6: Αποθηκεύστε την παρουσίασή σας
Τέλος, αποθηκεύστε την παρουσίασή σας με το πρόσφατα δημιουργημένο διάγραμμα ηλιοφάνειας:

```python
        pres.save("YOUR_OUTPUT_DIRECTORY/charts_sunburst_chart_out.pptx", slides.export.SaveFormat.PPTX)

```

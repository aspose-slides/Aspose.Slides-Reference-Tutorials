# Πώς να δημιουργήσετε γραφήματα φυσαλίδων με ετικέτες δεδομένων σε Python χρησιμοποιώντας το Aspose.Slides
## Εισαγωγή

Αυτό το σεμινάριο δείχνει πώς να αυτοματοποιήσετε αυτήν τη διαδικασία χρησιμοποιώντας το Aspose.Slides για Python, επιτρέποντάς σας να δημιουργείτε γραφήματα φυσαλίδων με αυτόματη επισήμανση δεδομένων από τιμές κελιών στις παρουσιάσεις σας.

- Ρύθμιση του Aspose.Slides για Python.
- Δημιουργία γραφήματος φυσαλίδων με ετικέτες δεδομένων που προέρχονται απευθείας από κελιά.

## Οδηγός Εφαρμογής
Ακολουθήστε αυτά τα βήματα για να δημιουργήσετε ένα γράφημα φυσαλίδων με ετικέτες δεδομένων από τιμές κελιών.
### Δημιουργήστε ένα γράφημα φυσαλίδων
#### Επισκόπηση
Αυτή η ενότητα δείχνει πώς να προσθέσετε ένα γράφημα φυσαλίδων σε μια υπάρχουσα παρουσίαση PowerPoint και να τη ρυθμίσετε ώστε να περιλαμβάνει ετικέτες δεδομένων που προέρχονται απευθείας από συγκεκριμένα κελιά.
#### Οδηγίες βήμα προς βήμα
##### 1. Φορτώστε το αρχείο παρουσίασης
Ανοίξτε το αρχείο παρουσίασής σας όπου θέλετε να εισαγάγετε το γράφημα φυσαλίδων:
```python
import aspose.slides as slides

def create_bubble_chart_with_labels():
    # Ορίστε τα κείμενα των ετικετών για λόγους σαφήνειας
    lbl0 = "Label 0 cell value"
    lbl1 = "Label 1 cell value"
    lbl2 = "Label 2 cell value"
    
    # Ανοίξτε το αρχείο παρουσίασής σας από έναν συγκεκριμένο κατάλογο
    with slides.Presentation("YOUR_DOCUMENT_DIRECTORY/charts_workbook_as_datalabel.pptx") as pres:
        # Συνέχεια στο επόμενο βήμα...
```
*Εξήγηση*: Αυτό το απόσπασμα κώδικα ανοίγει ένα υπάρχον αρχείο PowerPoint. Αντικατάσταση `"YOUR_DOCUMENT_DIRECTORY"` με την πραγματική σου πορεία.
##### 2. Προσθέστε ένα γράφημα φυσαλίδων
Εισαγάγετε το διάγραμμα στις καθορισμένες συντεταγμένες και διαστάσεις:
```python
        # Εισαγάγετε ένα γράφημα φυσαλίδων στις συντεταγμένες (50, 50) με διαστάσεις 600x400 pixel
        chart = pres.slides[0].shapes.add_chart(slides.charts.ChartType.BUBBLE, 50, 50, 600, 400, True)
```
*Εξήγηση*: Το `add_chart` Η μέθοδος δημιουργεί ένα νέο γράφημα φυσαλίδων. Προσαρμόστε τη θέση και το μέγεθος όπως απαιτείται.
##### 3. Ρύθμιση παραμέτρων ετικετών δεδομένων
Ορίστε ετικέτες δεδομένων για την εμφάνιση τιμών από συγκεκριμένα κελιά:
```python
        # Πρόσβαση στη σειρά του γραφήματος
        series = chart.chart_data.series
        
        # Ενεργοποίηση εμφάνισης τιμής ετικέτας απευθείας από το κελί
        series[0].labels.default_data_label_format.show_label_value_from_cell = True
        
        # Ανάκτηση του βιβλίου εργασίας που σχετίζεται με τα δεδομένα του γραφήματος
        wb = chart.chart_data.chart_data_workbook
        
        # Αντιστοίχιση τιμών ετικέτας για κάθε σημείο της σειράς από συγκεκριμένα κελιά
        series[0].labels[0].value_from_cell = wb.get_cell(0, "A10", lbl0)
        series[0].labels[1].value_from_cell = wb.get_cell(0, "A11", lbl1)
        series[0].labels[2].value_from_cell = wb.get_cell(0, "A12", lbl2)
```
*Εξήγηση*: Αυτή η ενότητα διαμορφώνει τις ετικέτες δεδομένων για κάθε σημείο στο γράφημα ώστε να εμφανίζει τιμές από συγκεκριμένα κελιά. Προσαρμόστε τις αναφορές κελιών όπως απαιτείται.
##### 4. Αποθήκευση της παρουσίασης
Αποθηκεύστε την τροποποιημένη παρουσίασή σας:
```python

# Εκτελέστε τη συνάρτηση για να δημιουργήσετε το γράφημα
create_bubble_chart_with_labels()
```
*Εξήγηση*: Αυτό αποθηκεύει την παρουσίασή σας με το πρόσφατα προστιθέμενο και διαμορφωμένο γράφημα φυσαλίδων.

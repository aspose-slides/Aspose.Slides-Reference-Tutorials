## Οδηγός Εφαρμογής
Σε αυτήν την ενότητα, θα εξερευνήσουμε πώς να ορίσετε τις ιδιότητες γραμματοσειράς γραφήματος βήμα προς βήμα.

### Προσθήκη γραφήματος ομαδοποιημένων στηλών
Αρχικά, ας προσθέσουμε ένα γράφημα ομαδοποιημένων στηλών στην παρουσίασή μας:

```python
# Προσθέστε ένα γράφημα ομαδοποιημένων στηλών στην καθορισμένη θέση και μέγεθος.
chart = pres.slides[0].shapes.add_chart(
    slides.charts.ChartType.CLUSTERED_COLUMN, 100, 100, 500, 400
)
```
**Εξήγηση**: Αυτό το απόσπασμα προσθέτει ένα νέο γράφημα στην πρώτη διαφάνεια της παρουσίασής σας. Το `add_chart` Η μέθοδος απαιτεί να καθορίσετε τον τύπο γραφήματος, καθώς και τη θέση και το μέγεθός του στη διαφάνεια.

### Ορισμός ιδιοτήτων γραμματοσειράς
Στη συνέχεια, ας ορίσουμε το ύψος της γραμματοσειράς για το κείμενο μέσα στο γράφημά μας:

```python
# Ορίστε το ύψος γραμματοσειράς για το κείμενο στο γράφημα.
chart.text_format.portion_format.font_height = 20
```
**Εξήγηση**: Αυτή η γραμμή προσαρμόζει το μέγεθος γραμματοσειράς όλων των τμημάτων κειμένου μέσα στο γράφημά σας. `font_height` Η ιδιότητα καθορίζεται σε σημεία και μπορείτε να προσαρμόσετε αυτήν την τιμή ώστε να ταιριάζει στις ανάγκες σχεδιασμού σας.

## Οδηγός Εφαρμογής

Θα επικεντρωθούμε στην τροποποίηση του άξονα κατηγορίας του γραφήματος. Ας αναλύσουμε τη διαδικασία βήμα προς βήμα.

### Πρόσβαση στην παρουσίαση και το διάγραμμα

Ξεκινήστε φορτώνοντας το αρχείο παρουσίασής σας. Βεβαιωθείτε ότι γνωρίζετε τη διαδρομή προς το έγγραφό σας:

```python
def change_chart_category_axis():
    data_dir = "YOUR_DOCUMENT_DIRECTORY/"
    
    with slides.Presentation(data_dir + "charts_existing_chart.pptx") as presentation:
        chart = presentation.slides[0].shapes[0]
```

Αυτό το απόσπασμα ανοίγει ένα αρχείο PowerPoint και έχει πρόσβαση στο πρώτο σχήμα της πρώτης διαφάνειας, υποθέτοντας ότι περιέχει ένα γράφημα.

### Τροποποίηση του άξονα κατηγορίας

Στη συνέχεια, αλλάξτε τον τύπο άξονα κατηγορίας σε DATE:

```python
chart.axes.horizontal_axis.category_axis_type = slides.charts.CategoryAxisType.DATE
```

Ο ορισμός του τύπου άξονα σε DATE διασφαλίζει ότι τα δεδομένα σας ευθυγραμμίζονται με τις ημερομηνίες ημερολογίου, βελτιώνοντας την αναγνωσιμότητα των δεδομένων χρονοσειρών.

### Ρύθμιση παραμέτρων ιδιοτήτων άξονα

Προσαρμόστε τον οριζόντιο άξονα ορίζοντας κύριες μονάδες και κλίμακες:

```python
chart.axes.horizontal_axis.is_automatic_major_unit = False
chart.axes.horizontal_axis.major_unit = 1
chart.axes.horizontal_axis.major_unit_scale = slides.charts.TimeUnitType.MONTHS
```

Απενεργοποιώντας τον αυτόματο υπολογισμό των κύριων μονάδων, αποκτάτε τον έλεγχο του τρόπου με τον οποίο τα σημεία δεδομένων κατανέμονται στον άξονα. `major_unit` ορίζει διαστήματα (π.χ., κάθε μήνα), ενώ `major_unit_scale` καθορίζει ότι αυτές οι μονάδες αντιπροσωπεύουν μήνες.

### Αποθήκευση των αλλαγών σας

Τέλος, αποθηκεύστε την τροποποιημένη παρουσίασή σας:

```python
out_dir = "YOUR_OUTPUT_DIRECTORY/"
presentation.save(out_dir + "charts_change_chart_category_axis_out.pptx", slides.export.SaveFormat.PPTX)
```

Αυτό το βήμα εγγράφει τις αλλαγές σε ένα νέο αρχείο στον καθορισμένο κατάλογο εξόδου.

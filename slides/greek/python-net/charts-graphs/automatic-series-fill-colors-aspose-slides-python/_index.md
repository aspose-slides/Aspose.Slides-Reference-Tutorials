## Οδηγός Εφαρμογής

Ακολουθήστε αυτά τα βήματα για να εφαρμόσετε αυτόματα χρώματα γεμίσματος σειράς σε γραφήματα με το Aspose.Slides για Python.

### Προσθήκη γραφήματος και ρύθμιση χρωμάτων αυτόματης σειράς

#### Επισκόπηση
Θα αυτοματοποιήσουμε τη διαδικασία ορισμού χρωμάτων σειρών σε ένα γράφημα ομαδοποιημένων στηλών στην πρώτη διαφάνεια της παρουσίασής σας.

#### Βήμα προς βήμα εφαρμογή
**1. Αρχικοποιήστε την παρουσίασή σας:**
Ξεκινήστε δημιουργώντας ένα νέο αντικείμενο παρουσίασης:

```python
import aspose.slides as slides

def charts_set_automatic_series_fill_color():
    with slides.Presentation() as presentation:
        # Προσθήκη γραφήματος ομαδοποιημένων στηλών στην πρώτη διαφάνεια
```

**2. Προσθέστε ένα γράφημα ομαδοποιημένων στηλών:**
Προσθέστε ένα γράφημα χρησιμοποιώντας το Aspose.Slides, καθορίζοντας τον τύπο και τις διαστάσεις του:

```python
chart = presentation.slides[0].shapes.add_chart(
    slides.charts.ChartType.CLUSTERED_COLUMN, 100, 50, 600, 400
)
```

**3. Ορίστε χρώματα αυτόματης συμπλήρωσης σειρών:**
Περιηγηθείτε σε κάθε σειρά στο γράφημα για να εφαρμόσετε αυτόματα χρώματα:

```python
for i in range(len(chart.chart_data.series)):
    chart.chart_data.series[i].format.fill.set_fill_type(slides.FillType.SOLID)
    chart.chart_data.series[i].format.fill.solid_fill_color.color = slides.Color.from_argb(255, 0, 0) # Παράδειγμα για ένα συμπαγές κόκκινο χρώμα
```

**4. Αποθηκεύστε την παρουσίασή σας:**
Τέλος, αποθηκεύστε την παρουσίασή σας σε έναν καθορισμένο κατάλογο:

```python
presentation.save("YOUR_OUTPUT_DIRECTORY/charts_set_automatic_series_fill_color_out.pptx")
```

## Εισαγωγή
Η δημιουργία οπτικά ελκυστικών γραφημάτων στο PowerPoint μπορεί να κάνει τα δεδομένα σας πιο ελκυστικά και πιο εύκολα κατανοητά. Μια συνηθισμένη πρόκληση είναι η έλλειψη επιλογών προσαρμογής κατά τη δημιουργία αυτών των γραφημάτων μέσω προγραμματισμού. Αυτό το σεμινάριο λύνει αυτό το πρόβλημα, δείχνοντας πώς να δημιουργήσετε ένα γράφημα ντόνατ με προσαρμοσμένο μέγεθος τρύπας χρησιμοποιώντας το Aspose.Slides για Python.

**Λέξεις-κλειδιά:** Aspose.Slides Python, Γράφημα Ντόνατ, Προσαρμοσμένο Μέγεθος Τρύπας

### Τι θα μάθετε:
- Ρύθμιση και χρήση του Aspose.Slides για Python
- Δημιουργία γραφήματος ντόνατ στο PowerPoint
- Προσαρμογή του μεγέθους της τρύπας του γραφήματος ντόνατ σας
- Βέλτιστες πρακτικές για την αποθήκευση και την εξαγωγή παρουσιάσεων

## Οδηγός Εφαρμογής
Αυτή η ενότητα αναλύει τα βήματα που απαιτούνται για τη δημιουργία και την προσαρμογή ενός γραφήματος ντόνατ στο PowerPoint χρησιμοποιώντας το Aspose.Slides.

### Βήμα 1: Πρόσβαση και τροποποίηση μιας διαφάνειας
Για να ξεκινήσετε, αποκτήστε πρόσβαση στην πρώτη διαφάνεια από την παρουσίασή σας. Εδώ θα προσθέσετε το προσαρμοσμένο γράφημα ντόνατς σας.

```python
# Πρόσβαση στην πρώτη διαφάνεια
class SlideModifier:

        first_slide = presentation.slides[0]
```

### Βήμα 2: Προσθήκη γραφήματος ντόνατ
Μπορείτε να προσθέσετε ένα γράφημα ντόνατ σε οποιαδήποτε διαφάνεια καθορίζοντας τη θέση και το μέγεθός του. Εδώ, θα το τοποθετήσουμε σε συντεταγμένες (50, 50) με διαστάσεις 400x400.

```python
class ChartAdder:
    def add_doughnut_chart(self, first_slide):
        # Προσθήκη γραφήματος ντόνατ
        chart = first_slide.shapes.add_chart(
            slides.charts.ChartType.DOUGHNUT,
            50, 50, 400, 400
        )
```

### Βήμα 3: Προσαρμογή του μεγέθους της τρύπας
Η ρύθμιση του μεγέθους της τρύπας του γραφήματος ντόνατς είναι απλή. Ρυθμίστε το στο 90% για ένα έντονο αποτέλεσμα.

```python
class ChartCustomizer:
    def customize_hole_size(self, chart):
        # Ορισμός προσαρμοσμένου μεγέθους τρύπας
        chart.chart_data.series_groups[0].doughnut_hole_size = 90
```

### Βήμα 4: Αποθήκευση της παρουσίασής σας
Τέλος, αποθηκεύστε την παρουσίασή σας στην επιθυμητή θέση με το επιλεγμένο όνομα αρχείου.

```python
class PresentationSaver:
    def save_presentation(self, presentation):
        # Αποθήκευση της παρουσίασης
        presentation.save(
            "charts_doughnut_chart_hole_out.pptx",
            slides.export.SaveFormat.PPTX
        )
```

### Οδηγός Εφαρμογής

#### Χαρακτηριστικό 1: Δημιουργία παρουσίασης με γράφημα

Αυτή η λειτουργία θα δείξει πώς να δημιουργήσετε μια παρουσίαση PowerPoint και να προσθέσετε ένα γράφημα πίτας στην πρώτη διαφάνεια.

##### Προσθήκη του γραφήματος

Ξεκινήστε δημιουργώντας μια νέα παρουσίαση και προσθέτοντας ένα γράφημα πίτας στη θέση (50, 50) στην πρώτη διαφάνεια:

```python
import aspose.slides as slides

with slides.Presentation() as presentation:
    # Προσθήκη γραφήματος «Πίτας από πίτα» με καθορισμένες διαστάσεις
    chart = presentation.slides[0].shapes.add_chart(
        slides.charts.ChartType.PIE_OF_PIE, 50, 50, 500, 400)
```

##### Ρύθμιση παραμέτρων ετικετών δεδομένων

Για να βελτιώσετε την αναγνωσιμότητα, διαμορφώστε τις ετικέτες δεδομένων ώστε να εμφανίζουν τιμές:

```python
# Ενεργοποιήστε την εμφάνιση τιμών στις ετικέτες δεδομένων για μεγαλύτερη σαφήνεια
chart.chart_data.series[0].labels.default_data_label_format.show_value = True
```

##### Ρύθμιση επιλογών πίτας

Ρυθμίστε συγκεκριμένες ιδιότητες για το γράφημα πίτας πίτας, όπως το μέγεθος της δεύτερης πίτας και τη θέση διαίρεσης:

```python
# Ορισμός μεγέθους δεύτερης πίτας και ιδιοτήτων διαίρεσης
chart.chart_data.series[0].parent_series_group.second_pie_size = 149
chart.chart_data.series[0].parent_series_group.pie_split_by = slides.charts.PieSplitType.BY_PERCENTAGE
chart.chart_data.series[0].parent_series_group.pie_split_position = 53
```

##### Αποθήκευση της παρουσίασης

Τέλος, αποθηκεύστε την παρουσίασή σας στον επιθυμητό κατάλογο:

```python
# Αποθήκευση της παρουσίασης με το γράφημα
presentation.save("YOUR_OUTPUT_DIRECTORY/charts_second_plot_options_out.pptx", slides.export.SaveFormat.PPTX)
```

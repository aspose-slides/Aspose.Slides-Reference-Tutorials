# Πώς να ορίσετε την περιστροφή τίτλου κάθετου άξονα ενός γραφήματος στο Aspose.Slides για Python

## Εισαγωγή

Στις παρουσιάσεις δεδομένων, η βελτίωση της αναγνωσιμότητας των γραφημάτων είναι ζωτικής σημασίας. Η ρύθμιση της γωνίας περιστροφής του τίτλου του κατακόρυφου άξονα του γραφήματός σας χρησιμοποιώντας το Aspose.Slides για Python μπορεί να κάνει τους τίτλους να ταιριάζουν άψογα ή να ξεχωρίζουν στις διαφάνειές σας. Αυτό το σεμινάριο σας καθοδηγεί στη ρύθμιση αυτής της γωνίας περιστροφής για να βελτιώσετε τόσο τη λειτουργικότητα όσο και την οπτική ελκυστικότητα.

**Τι θα μάθετε:**
- Πώς να εγκαταστήσετε και να ρυθμίσετε το Aspose.Slides για Python.
- Βήματα για την προσθήκη και προσαρμογή γραφημάτων μέσα στις διαφάνειές σας.
- Τεχνικές για τον ορισμό της γωνίας περιστροφής των τίτλων γραφημάτων.
- Εφαρμογές στον πραγματικό κόσμο για αυτά τα χαρακτηριστικά στην οπτικοποίηση δεδομένων.

Ας ξεκινήσουμε καλύπτοντας τις προϋποθέσεις πριν προχωρήσουμε στην υλοποίηση.

### Προσθήκη και Προσαρμογή Γραφημάτων

#### Επισκόπηση

Σε αυτήν την ενότητα, θα προσθέσουμε ένα γράφημα ομαδοποιημένων στηλών στη διαφάνειά σας και θα το προσαρμόσουμε ορίζοντας τη γωνία περιστροφής του τίτλου του κατακόρυφου άξονα.

#### Βήματα:

##### Βήμα 1: Προσθήκη γραφήματος ομαδοποιημένων στηλών

Ξεκινήστε προσθέτοντας ένα γράφημα σε συγκεκριμένες συντεταγμένες με καθορισμένες διαστάσεις:

```python
def main():
    import aspose.slides as slides

    with slides.Presentation() as pres:
        # Προσθήκη γραφήματος ομαδοποιημένων στηλών στη διαφάνεια 1
        chart = pres.slides[0].shapes.add_chart(
            slides.charts.ChartType.CLUSTERED_COLUMN, 50, 50, 450, 300)
```

##### Βήμα 2: Ρύθμιση παραμέτρων του τίτλου του κατακόρυφου άξονα

Ενεργοποίηση και ορισμός της γωνίας περιστροφής για τον τίτλο του κατακόρυφου άξονα:

```python
def configure_chart(chart):
    # Ενεργοποίηση τίτλου κατακόρυφου άξονα
    chart.axes.vertical_axis.has_title = True
    
    # Ρυθμίστε τη γωνία περιστροφής στις 90 μοίρες
    chart.axes.vertical_axis.title.text_format.text_block_format.rotation_angle = 90
```

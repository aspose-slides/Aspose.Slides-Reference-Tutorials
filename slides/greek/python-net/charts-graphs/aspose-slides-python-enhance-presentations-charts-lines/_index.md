```python
import aspose.slides as slides

# Αρχικοποίηση αντικειμένου παρουσίασης
def setup_presentation():
```

## Οδηγός Εφαρμογής
Σε αυτήν την ενότητα, θα περιηγηθούμε στη διαδικασία προσθήκης γραφημάτων και προσαρμοσμένων γραμμών στην παρουσίασή σας χρησιμοποιώντας το Aspose.Slides για Python. Θα την χωρίσουμε σε δύο κύρια χαρακτηριστικά: την προσθήκη ενός γραφήματος και την βελτίωσή του με προσαρμοσμένες γραμμές.

### Λειτουργία 1: Προσθήκη γραφήματος σε παρουσίαση
#### Επισκόπηση
Η προσθήκη ενός γραφήματος ομαδοποιημένων στηλών παρέχει μια οπτική αναπαράσταση των δεδομένων, διευκολύνοντας το κοινό σας να κατανοήσει γρήγορα σύνθετες πληροφορίες.

#### Βήματα για την προσθήκη ενός γραφήματος ομαδοποιημένων στηλών
##### Βήμα 1: Δημιουργήστε το αντικείμενο παρουσίασης
Ξεκινήστε αρχικοποιώντας ένα νέο αντικείμενο παρουσίασης:

```python
def add_chart_to_presentation():
    with slides.Presentation() as pres:
        # Τα επόμενα βήματα θα προστεθούν εδώ
```

##### Βήμα 2: Προσθέστε το γράφημα ομαδοποιημένων στηλών
Προσθέστε το γράφημα στην πρώτη σας διαφάνεια σε μια καθορισμένη θέση και μέγεθος:

```python
# Προσθήκη ενός γραφήματος ομαδοποιημένων στηλών στην πρώτη διαφάνεια στο (100, 100) με διαστάσεις (500, 400)
chart = pres.slides[0].shapes.add_chart(
    slides.charts.ChartType.CLUSTERED_COLUMN,
    100, 100, 500, 400
)
```

##### Βήμα 3: Αποθήκευση της παρουσίασης
Τέλος, αποθηκεύστε την παρουσίασή σας σε έναν καθορισμένο κατάλογο:

```python
# Αποθήκευση της παρουσίασης
def save_presentation(pres):
    pres.save("YOUR_OUTPUT_DIRECTORY/charts_adding_custom_lines_out.pptx", slides.export.SaveFormat.PPTX)

add_chart_to_presentation()
```

### Λειτουργία 2: Προσθήκη προσαρμοσμένων γραμμών σε γράφημα
#### Επισκόπηση
Μπορούν να προστεθούν προσαρμοσμένες γραμμές (σχήματα) σε ένα γράφημα για να επισημανθούν συγκεκριμένα σημεία δεδομένων ή τάσεις, ενισχύοντας την οπτική ελκυστικότητα και τη σαφήνεια της παρουσίασής σας.

#### Βήματα για την προσθήκη προσαρμοσμένων γραμμών
##### Βήμα 1: Αρχικοποίηση αντικειμένου παρουσίασης
Ξεκινήστε με την αρχικοποίηση ενός νέου αντικειμένου παρουσίασης:

```python
def add_custom_lines_to_chart():
    with slides.Presentation() as pres:
        # Συνεχίστε με την προσθήκη του γραφήματος και των προσαρμοσμένων γραμμών
```

##### Βήμα 2: Προσθήκη του γραφήματος ομαδοποιημένων στηλών (Επαναλαμβανόμενη)
Χρησιμοποιήστε ξανά τα βήματα από την προηγούμενη ενότητα εάν ξεκινάτε από την αρχή:

```python
chart = pres.slides[0].shapes.add_chart(
    slides.charts.ChartType.CLUSTERED_COLUMN,
    100, 100, 500, 400
)
```

##### Βήμα 3: Προσθήκη σχήματος γραμμής στο γράφημα
Ενσωματώστε μια προσαρμοσμένη γραμμή στο γράφημά σας:

```python
# Προσθήκη σχήματος οριζόντιας γραμμής στο μέσο του γραφήματος
def add_line_to_chart(chart):
    shape = chart.user_shapes.shapes.add_auto_shape(
        slides.ShapeType.LINE,
        0, chart.height / 2, chart.width, 0
    )

    # Ορίστε τη μορφή γεμίσματος σε συμπαγή και χρωματίστε την με κόκκινο χρώμα για ορατότητα
    shape.line_format.fill_format.fill_type = slides.FillType.SOLID
    shape.line_format.fill_format.solid_fill_color.color = drawing.Color.red

add_custom_lines_to_chart()
```

##### Βήμα 4: Αποθήκευση της παρουσίασης
Αποθηκεύστε την βελτιωμένη παρουσίασή σας:

```python
def save_presentation(pres):
    pres.save("YOUR_OUTPUT_DIRECTORY/charts_adding_custom_lines_out.pptx", slides.export.SaveFormat.PPTX)

add_custom_lines_to_chart()
```

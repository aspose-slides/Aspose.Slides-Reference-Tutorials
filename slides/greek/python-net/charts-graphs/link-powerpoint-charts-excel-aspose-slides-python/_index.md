## Οδηγός Εφαρμογής

### Επισκόπηση

Η σύνδεση ενός γραφήματος PowerPoint με ένα αρχείο Excel επιτρέπει την αυτόματη ενημέρωση και την δυναμική οπτικοποίηση δεδομένων. Αυτή η ενότητα σας καθοδηγεί στη δημιουργία μιας παρουσίασης, στην προσθήκη ενός γραφήματος και στη διαμόρφωσή της για χρήση σε ένα εξωτερικό βιβλίο εργασίας.

### Δημιουργία νέας παρουσίασης

Αρχικά, αρχικοποιήστε το περιβάλλον της παρουσίασής σας χρησιμοποιώντας το `with` δήλωση:

```python
with slides.Presentation() as pres:
    # Ο κωδικός σας εδώ...
```

Αυτό διασφαλίζει την ορθή διαχείριση των πόρων, απελευθερώνοντας αυτόματα τους πόρους μόλις ολοκληρωθούν οι λειτουργίες.

### Προσθήκη γραφήματος στη διαφάνεια

Προσθέστε ένα γράφημα πίτας στη διαφάνειά σας με καθορισμένες διαστάσεις και θέση:

```python
chart = pres.slides[0].shapes.add_chart(slides.charts.ChartType.PIE, 50, 50, 400, 600, True)
```

Παράμετροι:
- `ChartType.PIE`: Καθορίζει ότι το γράφημα είναι κυκλικό γράφημα.
- `(50, 50)`: Συντεταγμένες X και Y στη διαφάνεια όπου θα τοποθετηθεί το διάγραμμα.
- `400, 600`Πλάτος και ύψος του γραφήματος σε pixel.

### Ρύθμιση εξωτερικού βιβλίου εργασίας για δεδομένα γραφήματος

Αποκτήστε πρόσβαση στα δεδομένα του γραφήματος και συνδέστε τα με ένα εξωτερικό βιβλίο εργασίας:

```python
chart_data = chart.chart_data
chart_data.set_external_workbook("YOUR_DOCUMENT_DIRECTORY/external_workbook.xlsx", False)
```

Εδώ:
- `"YOUR_DOCUMENT_DIRECTORY/external_workbook.xlsx"`: Διαδρομή προς το αρχείο Excel σας.
- `False`: Υποδεικνύει ότι τα δεδομένα δεν θα πρέπει να ενημερώνονται αυτόματα.

### Αποθήκευση της παρουσίασης

Τέλος, αποθηκεύστε την παρουσίασή σας με τις αλλαγές:

```python
class InvalidDataError(Exception):
    pass

def validate_data(data):
    if not isinstance(data, list) or any(not isinstance(item, (int, float)) for item in data):
        raise InvalidDataError("Invalid data format. Must be a list of numbers.")

validate_data(chart.chart_data.workbook.get_worksheet_by_name(0).cells["A1:C5").get_value())

pres.save("YOUR_OUTPUT_DIRECTORY/charts_set_external_workbook_with_update_chart_data_out.pptx", slides.export.SaveFormat.PPTX)
```

Αυτή η εντολή γράφει την τροποποιημένη παρουσίαση σε έναν καθορισμένο κατάλογο σε μορφή PPTX.

## Οδηγός Εφαρμογής

Ας αναλύσουμε τώρα τον τρόπο υλοποίησης ενός γραφήματος χάρτη σε μια παρουσίαση PowerPoint χρησιμοποιώντας το Aspose.Slides για Python.

### Δημιουργία και αποθήκευση παρουσίασης

#### Επισκόπηση

Θα δημιουργήσουμε ένα νέο αρχείο PowerPoint, θα προσθέσουμε μια διαφάνεια, θα εισαγάγουμε ένα γράφημα χάρτη, θα το συμπληρώσουμε με δεδομένα, θα προσαρμόσουμε την εμφάνισή του και θα αποθηκεύσουμε το τελικό αποτέλεσμα.

##### Αρχικοποίηση νέας παρουσίασης

Ξεκινήστε αρχικοποιώντας την παρουσίασή σας:

```python
def create_and_save_presentation():
    """Create and save a presentation with a map chart."""
    # Αρχικοποίηση ενός νέου αντικειμένου παρουσίασης
    with slides.Presentation() as presentation:
        pass  # Θα συμπληρώσουμε την υπόλοιπη λογική εδώ

create_and_save_presentation()
```

##### Προσθήκη χάρτη

Προσθέστε ένα γράφημα τύπου MAP στην πρώτη σας διαφάνεια:

```python
with slides.Presentation() as presentation:
    # Εισαγωγή ενός χάρτη στη θέση (50, 50) με μέγεθος (500x400)
    chart = presentation.slides[0].shapes.add_chart(
        slides.charts.ChartType.MAP, 50, 50, 500, 400, False
    )
```

- **Παράμετροι:** 
  - `ChartType.MAP`: Καθορίζει τον τύπο του γραφήματος.
  - `(50, 50)`: Η θέση στη διαφάνεια.
  - `(500x400)`: Διαστάσεις πλάτους και ύψους.

##### Προσθήκη Σειρών και Σημείων Δεδομένων

Συμπληρώστε το διάγραμμα του χάρτη σας με σημεία δεδομένων:

```python
wb = chart.chart_data.chart_data_workbook

# Προσθήκη σειρών και σημείων δεδομένων
to_series = chart.chart_data.series.add(slides.charts.ChartType.MAP)
to_series.data_points.add_data_point_for_map_series(wb.get_cell(0, "B2", 5))
to_series.data_points.add_data_point_for_map_series(wb.get_cell(0, "B3", 1))
to_series.data_points.add_data_point_for_map_series(wb.get_cell(0, "B4", 10))
```

- **Γιατί:** Αυτό το βήμα προσθέτει τα πραγματικά δεδομένα που θα εμφανίσει το διάγραμμα του χάρτη σας.

##### Ορισμός κατηγοριών για το διάγραμμα χάρτη

Αντιστοιχίστε γεωγραφικές κατηγορίες σε κάθε σημείο δεδομένων:

```python
# Προσθήκη κατηγοριών
to_chart.chart_data.categories.add(wb.get_cell(0, "A2", "United States"))
to_chart.chart_data.categories.add(wb.get_cell(0, "A3", "Mexico"))
to_chart.chart_data.categories.add(wb.get_cell(0, "A4", "Brazil"))
```

- **Γιατί:** Αυτό καθορίζει τις περιοχές που αντιπροσωπεύουν τα σημεία δεδομένων σας.

##### Προσαρμόστε την εμφάνιση του σημείου δεδομένων

Βελτιώστε την οπτική ελκυστικότητα προσαρμόζοντας ένα σημείο δεδομένων:

```python
# Προσαρμόστε την εμφάνιση ενός σημείου δεδομένων
data_point = to_series.data_points[1]
data_point.color_value.as_cell.value = "15"
data_point.format.fill.fill_type = slides.FillType.SOLID
data_point.format.fill.solid_fill_color.color = drawing.Color.green
```

- **Γιατί:** Η βελτίωση ενός συγκεκριμένου σημείου δεδομένων το βοηθά να ξεχωρίζει για έμφαση.

##### Αποθήκευση της παρουσίασης

Τέλος, αποθηκεύστε την παρουσίασή σας:

```python
# Αποθήκευση σε καθορισμένο κατάλογο
presentation.save("YOUR_OUTPUT_DIRECTORY/charts_map_chart_out.pptx", slides.export.SaveFormat.PPTX)
```

- **Γιατί:** Αυτό το βήμα γράφει την εργασία σας σε ένα αρχείο που μπορείτε να μοιραστείτε ή να παρουσιάσετε.

- Δημιουργία παρουσίασης με το Aspose.Slides.
- Προσθήκη και διαμόρφωση γραφημάτων στις διαφάνειές σας.
- Εξαγωγή τιμών κατακόρυφου άξονα (μέγιστες και ελάχιστες).
- Λήψη μοναδιαίων κλιμάκων οριζόντιου άξονα (μείζονες και ελάσσονες μονάδες).

Πριν ξεκινήσουμε με το σεμινάριο, ας εξετάσουμε τις απαραίτητες προϋποθέσεις για να ξεκινήσουμε.

## Οδηγός Εφαρμογής

### Εξαγωγή τιμών άξονα γραφήματος

Ακολουθήστε αυτά τα βήματα για να εξαγάγετε τιμές αξόνων από ένα γράφημα χρησιμοποιώντας το Aspose.Slides.

#### Βήμα 1: Δημιουργία και διαμόρφωση της παρουσίασής σας

Ξεκινήστε δημιουργώντας μια νέα παρουσία παρουσίασης και προσθέτοντας ένα γράφημα περιοχής στην πρώτη διαφάνεια:
```python
with slides.Presentation() as pres:
    # Προσθήκη γραφήματος περιοχής στην πρώτη διαφάνεια
    chart = pres.slides[0].shapes.add_chart(slides.charts.ChartType.AREA, 100, 100, 500, 350)
```

#### Βήμα 2: Επικύρωση διάταξης γραφήματος

Βεβαιωθείτε ότι η διάταξη του γραφήματος έχει ρυθμιστεί σωστά πριν από την εξαγωγή τιμών:
```python
chart.validate_chart_layout()
```
Αυτό το βήμα διασφαλίζει ότι τα δεδομένα και η διαμόρφωση του γραφήματος είναι έτοιμα για εξαγωγή τιμών.

#### Βήμα 3: Εξαγωγή τιμών άξονα

Ανακτήστε τις μέγιστες και ελάχιστες τιμές από τον κατακόρυφο άξονα και τις κλίμακες μονάδων από τον οριζόντιο άξονα:
```python
# Τιμές κατακόρυφου άξονα
max_value = chart.axes.vertical_axis.actual_max_value
min_value = chart.axes.vertical_axis.actual_min_value

# Κλίμακες μονάδων οριζόντιου άξονα
major_unit = chart.axes.horizontal_axis.actual_major_unit
minor_unit = chart.axes.horizontal_axis.actual_minor_unit
```

#### Βήμα 4: Εμφάνιση εξαγόμενων τιμών

Εκτυπώστε αυτές τις τιμές για να επαληθεύσετε τη διαδικασία εξαγωγής:
```python
print(f"Max Value: {max_value}, Min Value: {min_value}")
print(f"Major Unit: {major_unit}, Minor Unit: {minor_unit}")
```

## Οδηγός Εφαρμογής

### Επισκόπηση της προσαρμογής υπομνημάτων γραφημάτων
Η προσαρμογή των υπομνημάτων γραφήματος περιλαμβάνει τον ορισμό ιδιοτήτων όπως η θέση, το μέγεθος και η στοίχιση σε σχέση με τις διαστάσεις του γραφήματος. Αυτή η ενότητα σας καθοδηγεί στην προσθήκη ενός γραφήματος ομαδοποιημένων στηλών και στην τροποποίηση του υπομνήματός του.

#### Βήμα 1: Δημιουργία νέας παρουσίασης
```python
import aspose.slides as slides

def charts_set_legend_custom_options():
    with slides.Presentation() as presentation:
        slide = presentation.slides[0]
```
Αυτός ο κώδικας αρχικοποιεί μια νέα παρουσίαση και έχει πρόσβαση στην πρώτη διαφάνεια για τροποποιήσεις.

#### Βήμα 2: Προσθήκη γραφήματος ομαδοποιημένων στηλών
```python
chart = slide.shapes.add_chart(
    slides.charts.ChartType.CLUSTERED_COLUMN,
    50, 50, 500, 500
)
```
Προσθέστε ένα γράφημα ομαδοποιημένων στηλών στη διαφάνεια. Οι παράμετροι καθορίζουν τον τύπο γραφήματος, τη θέση και τις διαστάσεις του στη διαφάνεια.

#### Βήμα 3: Ορισμός ιδιοτήτων υπομνήματος
Η προσαρμογή των ιδιοτήτων του υπομνήματος περιλαμβάνει τον υπολογισμό των θέσεων ως κλάσματα του πλάτους και του ύψους του γραφήματος:
```python
chart.legend.x = 50 / chart.width
chart.legend.y = 50 / chart.height
chart.legend.width = 100 / chart.width
chart.legend.height = 100 / chart.height
```
Εδώ, `x`, `y`, `width`, και `height` προσαρμόζονται ως κλάσματα για να διατηρηθεί η ανταπόκριση.

#### Βήμα 4: Αποθήκευση της παρουσίασης
```python
presentation.save("YOUR_OUTPUT_DIRECTORY/charts_set_legend_custom_options_out.pptx")
```
Αντικαθιστώ `"YOUR_OUTPUT_DIRECTORY"` με την επιθυμητή τοποθεσία αποθήκευσης. Αυτό το βήμα αποθηκεύει την προσαρμοσμένη παρουσίασή σας.

### Λειτουργία 2: Προσθήκη τρισδιάστατου γραφήματος σωρευμένων στηλών σε διαφάνεια
**Επισκόπηση**Μάθετε πώς να προσθέσετε ένα οπτικά ελκυστικό τρισδιάστατο γράφημα σωρευμένων στηλών στη διαφάνειά σας.
#### Βήμα προς βήμα εφαρμογή
**1. Δημιουργία και διαμόρφωση του γραφήματος**

```python
def add_3d_stacked_column_chart(slide):
    chart = slide.shapes.add_chart(
        slides.charts.ChartType.STACKED_COLUMN_3D,
        0, 0, 500, 500
    )
    return chart
```
*Εξήγηση*: Εδώ, `add_chart` δημιουργεί ένα νέο τρισδιάστατο γράφημα σωρευμένων στηλών στην καθορισμένη θέση με προεπιλεγμένες διαστάσεις.

### Λειτουργία 3: Διαχείριση δεδομένων και σειρών γραφημάτων
**Επισκόπηση**: Αυτή η ενότητα καλύπτει την προσθήκη σειρών δεδομένων και κατηγοριών στο γράφημά σας.
#### Βήμα προς βήμα εφαρμογή
**1. Προσθήκη Σειρών και Κατηγοριών**

```python
def manage_chart_data(chart):
    fact = chart.chart_data.chart_data_workbook
    
    # Προσθήκη σειράς
    chart.chart_data.series.add(
        fact.get_cell(0, 0, 1, "Series 1"),
        chart.type
    )
    chart.chart_data.series.add(
        fact.get_cell(0, 0, 2, "Series 2"),
        chart.type
    )

    # Προσθήκη κατηγοριών
    chart.chart_data.categories.add(fact.get_cell(0, 1, 0, "Category 1"))
    chart.chart_data.categories.add(fact.get_cell(0, 2, 0, "Category 2"))
    chart.chart_data.categories.add(fact.get_cell(0, 3, 0, "Category 3"))

    return chart
```
*Εξήγηση*Χρησιμοποιούμε `chart_data_workbook` για να προσθέσετε σειρές και κατηγορίες, θέτοντας τις βάσεις για την απεικόνιση δεδομένων.

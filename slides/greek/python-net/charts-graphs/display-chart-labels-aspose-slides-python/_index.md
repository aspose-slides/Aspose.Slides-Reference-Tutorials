## Οδηγός Εφαρμογής
Ακολουθήστε αυτά τα βήματα για να εμφανίσετε ετικέτες γραφήματος σε μια παρουσίαση PowerPoint.

### Βήμα 1: Δημιουργήστε μια νέα παρουσίαση και διαφάνεια
Δημιουργήστε μια νέα παρουσίαση και προσθέστε μια διαφάνεια:

```python
def display_chart_labels():
    with slides.Presentation() as presentation:
        # Αποκτήστε πρόσβαση στην πρώτη διαφάνεια (από προεπιλογή, δημιουργείται μία).
        slide = presentation.slides[0]
```

### Βήμα 2: Προσθήκη κυκλικού γραφήματος στη διαφάνεια
Προσθήκη κυκλικού γραφήματος στη θέση `(50, 50)` με διαστάσεις `500x400`:

```python
        # Προσθήκη γραφήματος πίτας στην πρώτη διαφάνεια.
        chart = slide.shapes.add_chart(
            slides.charts.ChartType.PIE, 50, 50, 500, 400)
```

### Βήμα 3: Ρύθμιση παραμέτρων επιλογών εμφάνισης ετικετών
Ρυθμίστε τις ιδιότητες ετικέτας για καλύτερη οπτικοποίηση δεδομένων:
- **Εμφάνιση ετικετών τιμών**: Εμφάνιση αριθμητικών τιμών σε κάθε τομή.
- **Επεξηγήσεις δεδομένων**Χρησιμοποιήστε γραμμές επεξήγησης για να συνδέσετε ετικέτες με φέτες.

```python
        # Ρύθμιση παραμέτρων επιλογών εμφάνισης ετικετών σειράς γραφημάτων
        series_labels = chart.chart_data.series[0].labels.default_data_label_format
        series_labels.show_value = True  # Εμφάνιση ετικετών τιμών από προεπιλογή
        series_labels.show_label_as_data_callout = True  # Χρήση επεξηγήσεων δεδομένων
```

### Βήμα 4: Προσαρμογή συγκεκριμένων ετικετών
Απενεργοποιήστε την επεξήγηση δεδομένων για συγκεκριμένες ετικέτες, όπως η τρίτη ετικέτα:

```python
        # Παράκαμψη της ρύθμισης επεξήγησης δεδομένων για μια συγκεκριμένη ετικέτα
        chart.chart_data.series[0].labels[2].data_label_format.show_label_as_data_callout = False
```

### Βήμα 5: Αποθήκευση της παρουσίασης
Αποθηκεύστε την παρουσίασή σας σε έναν κατάλογο εξόδου με το επιθυμητό όνομα αρχείου:

```python
        # Αποθήκευση της βελτιωμένης παρουσίασης
        presentation.save("YOUR_OUTPUT_DIRECTORY/charts_display_chart_labels_out.pptx")
```

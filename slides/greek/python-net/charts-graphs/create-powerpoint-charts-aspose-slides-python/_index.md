## Οδηγός Εφαρμογής

Ας εμβαθύνουμε στις λεπτομέρειες του χειρισμού γραφημάτων χρησιμοποιώντας το Aspose.Slides.

### Δημιουργία γραφήματος ομαδοποιημένων στηλών

**Επισκόπηση:**  
Αυτή η ενότητα εστιάζει στην προσθήκη ενός γραφήματος ομαδοποιημένων στηλών στην παρουσίαση του PowerPoint και στην προσαρμογή της εμφάνισης και των δεδομένων του.

#### Προσθήκη γραφήματος ομαδοποιημένων στηλών

```python
# Προσθέστε ένα γράφημα ομαδοποιημένων στηλών σε καθορισμένες συντεταγμένες (x: 50, y: 50) με πλάτος 600 και ύψος 400.
chart = pres.slides[0].shapes.add_chart(
    slides.charts.ChartType.CLUSTERED_COLUMN, 50, 50, 600, 400, True
)
```

#### Πρόσβαση και Εκκαθάριση Συλλογής Σειρών

```python
# Αποκτήστε τη συλλογή σειρών από τα δεδομένα του γραφήματος.
series_collection = chart.chart_data.series
# Διαγράψτε τυχόν υπάρχουσες σειρές για να ξεκινήσετε από την αρχή.
series_collection.clear()
```

### Προσθήκη σημείων δεδομένων με επιλογές αντιστροφής

**Επισκόπηση:**  
Σε αυτήν την ενότητα, θα μάθετε πώς να προσθέτετε σημεία δεδομένων σε μια σειρά και να διαχειρίζεστε τις ιδιότητές τους, όπως την αντιστροφή γραμμών για αρνητικές τιμές.

#### Προσθήκη Σειρών και Σημείων Δεδομένων

```python
# Προσθήκη νέας σειράς στο γράφημα.
series = series_collection.add(
    chart.chart_data.chart_data_workbook.get_cell(0, "B1"), chart.type
)

# Προσθέστε σημεία δεδομένων στην πρώτη σειρά. Μερικά είναι αρνητικά.
series.data_points.add_data_point_for_bar_series(chart.chart_data.chart_data_workbook.get_cell(0, "B2", -5))
series.data_points.add_data_point_for_bar_series(chart.chart_data.chart_data_workbook.get_cell(0, "B3", 3))
series.data_points.add_data_point_for_bar_series(chart.chart_data.chart_data_workbook.get_cell(0, "B4", -2))
series.data_points.add_data_point_for_bar_series(chart.chart_data.chart_data_workbook.get_cell(0, "B5", 1))
```

#### Προσαρμογή `invert_if_negative` Ιδιοκτησία

```python
# Ορίστε την τιμή invert_if_negative για ολόκληρη τη σειρά σε False.
series.invert_if_negative = False

# Αντιστρέψτε συγκεκριμένα το τρίτο σημείο δεδομένων.
series.data_points[2].invert_if_negative = True
```

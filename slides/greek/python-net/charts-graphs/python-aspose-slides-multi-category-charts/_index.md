## Οδηγός Εφαρμογής
Ας αναλύσουμε τη διαδικασία δημιουργίας ενός γραφήματος πολλαπλών κατηγοριών σε διαχειρίσιμα βήματα.

### Δημιουργία της Δομής Γραφήματος (H2)
#### Επισκόπηση:
Θα ξεκινήσουμε ορίζοντας τη βασική δομή του γραφήματός μας, συμπεριλαμβανομένης της αρχικοποίησης μιας παρουσίασης και της προσθήκης ενός γραφήματος ομαδοποιημένων στηλών σε μια διαφάνεια.

**Βήμα 1: Αρχικοποίηση παρουσίασης**

```python
import aspose.slides as slides

def create_multi_category_chart():
    with slides.Presentation() as pres:
        slide = pres.slides[0]  # Πρόσβαση στην πρώτη διαφάνεια
```

- **Γιατί;**Αυτή η ρύθμιση μας επιτρέπει να ξεκινήσουμε να κατασκευάζουμε την παρουσίασή μας από την αρχή.

**Βήμα 2: Προσθήκη γραφήματος σε διαφάνεια**

```python
        ch = slide.shapes.add_chart(
            slides.charts.ChartType.CLUSTERED_COLUMN, 
            100, 100, 600, 450
        )
```

- **Παράμετροι**: 
  - `ChartType.CLUSTERED_COLUMN`: Ορίζει τον τύπο γραφήματος.
  - `(100, 100)`: Η θέση στη διαφάνεια.
  - `(600, 450)`: Πλάτος και ύψος του γραφήματος.

**Βήμα 3: Εκκαθάριση υπαρχόντων δεδομένων**

```python
        ch.chart_data.series.clear()
        ch.chart_data.categories.clear()
```

- **Γιατί;**Αυτό διασφαλίζει ότι δεν θα υπάρξουν δεδομένα που να επηρεάζουν τη νέα μας διαμόρφωση γραφήματος.

### Διαμόρφωση Κατηγοριών και Σειρών (H2)
#### Επισκόπηση:
Στη συνέχεια, θα ορίσουμε κατηγορίες με επίπεδα ομαδοποίησης και θα προσθέσουμε σειρές με σημεία δεδομένων στο γράφημα.

**Βήμα 4: Ορισμός κατηγοριών**

```python
        fact = ch.chart_data.chart_data_workbook 
        category_labels = ['A', 'B', 'C', 'D', 'E', 'F', 'G', 'H']
        grouping_levels = ['Group1', 'Group2', 'Group3', 'Group4']

        for i, label in enumerate(category_labels):
            category = ch.chart_data.categories.add(fact.get_cell(0, f"c{i+2}", label))
            if i < len(grouping_levels):
                category.grouping_levels.set_grouping_item(1, grouping_levels[i])
```

- **Γιατί;**Η ομαδοποίηση κατηγοριών βελτιώνει την αναγνωσιμότητα και επιτρέπει τη συγκριτική ανάλυση.

**Βήμα 5: Προσθήκη Σειρών με Σημεία Δεδομένων**

```python
        series = ch.chart_data.series.add(
            fact.get_cell(0, "D1", "Series 1"), slides.charts.ChartType.CLUSTERED_COLUMN)
        
        values = [10, 20, 30, 40, 50, 60, 70, 80]
        for i, value in enumerate(values):
            series.data_points.add_data_point_for_bar_series(
                fact.get_cell(0, f"D{i+2}", value))
```

- **Γιατί;**Τα σημεία δεδομένων είναι κρίσιμα για την εμφάνιση των πραγματικών τιμών σε κάθε κατηγορία.

### Αποθήκευση της παρουσίασης (H2)
**Βήμα 6: Αποθηκεύστε την εργασία σας**

```python
        pres.save("YOUR_OUTPUT_DIRECTORY/charts_multi_category_chart_out.pptx", slides.export.SaveFormat.PPTX)
```

# Δημιουργήστε και προσαρμόστε γραφήματα TreeMap με το Aspose.Slides για Python

Αυτό το σεμινάριο σας καθοδηγεί στη χρήση του Aspose.Slides για Python για τη δημιουργία και τη διαμόρφωση ενός γραφήματος TreeMap—ένα ισχυρό εργαλείο οπτικοποίησης για την αποτελεσματική εμφάνιση ενσωματωμένων κατηγοριών δεδομένων.

- Ρύθμιση του περιβάλλοντός σας με το Aspose.Slides για Python.

- Πρακτικές περιπτώσεις χρήσης όπου ένα γράφημα TreeMap αποδεικνύεται χρήσιμο.
- Συμβουλές βελτιστοποίησης απόδοσης κατά την εργασία με μεγάλα σύνολα δεδομένων.

## Οδηγός Εφαρμογής
Ας δούμε τα βήματα που απαιτούνται για τη δημιουργία και τη διαμόρφωση του γραφήματος TreeMap.

### Προσθήκη γραφήματος TreeMap σε μια διαφάνεια

#### Επισκόπηση
Ένα γράφημα TreeMap είναι ιδανικό για την οπτική αναπαράσταση ιεραρχικών δεδομένων. Ομαδοποιεί τα δεδομένα σε ορθογώνια που ποικίλλουν σε μέγεθος ανάλογα με τις τιμές τους, διευκολύνοντας τη σύγκριση διαφορετικών τμημάτων με μια ματιά.

#### Βήματα για την προσθήκη ενός γραφήματος TreeMap
1. **Αρχικοποίηση παρουσίασης:**
   Ξεκινήστε δημιουργώντας μια παρουσία του `Presentation` τάξη:
   ```python
   import aspose.slides as slides
   
   with slides.Presentation() as pres:
       # Ο κώδικας για την προσθήκη γραφημάτων θα τοποθετηθεί εδώ
   ```
2. **Προσθήκη γραφήματος TreeMap:**
   Χρησιμοποιήστε το `add_chart()` μέθοδος για να τοποθετήσετε το γράφημά σας στην πρώτη διαφάνεια σε καθορισμένες συντεταγμένες και διαστάσεις:
   ```python
   chart = pres.slides[0].shapes.add_chart(
       slides.charts.ChartType.TREEMAP, 50, 50, 500, 400)
   ```
   Αυτό θα δημιουργήσει ένα TreeMap με πλάτος 500 pixel και ύψος 400 pixel στις συντεταγμένες (50, 50).
3. **Διαγραφή υπαρχόντων δεδομένων:**
   Πριν από την προσθήκη νέων δεδομένων, βεβαιωθείτε ότι οι υπάρχουσες κατηγορίες και σειρές έχουν διαγραφεί:
   ```python
   chart.chart_data.categories.clear()
   chart.chart_data.series.clear()
   
   wb = chart.chart_data.chart_data_workbook
   wb.clear(0)
   ```
### Ρύθμιση παραμέτρων κατηγοριών γραφημάτων
#### Επισκόπηση
Η οργάνωση των δεδομένων σας σε ιεραρχικές ομάδες είναι ζωτικής σημασίας για μια ουσιαστική αναπαράσταση TreeMap.
#### Βήματα για τη διαμόρφωση κατηγοριών
1. **Προσθήκη και ομαδοποίηση κατηγοριών:**
   Ορίστε τις κατηγορίες και τα ιεραρχικά τους επίπεδα χρησιμοποιώντας το `grouping_levels` ιδιότης:
   ```python
   leaf = chart.chart_data.categories.add(wb.get_cell(0, "C1", "Leaf1"))
   leaf.grouping_levels.set_grouping_item(1, "Stem1")
   leaf.grouping_levels.set_grouping_item(2, "Branch1")
   
   # Επαναλάβετε για άλλες κατηγορίες, όπως απαιτείται
   ```
   Αυτός ο κώδικας αντιστοιχίζει το "Leaf1" σε μια ιεραρχία με "Stem1" και "Branch1".
### Προσθήκη σειρών και σημείων δεδομένων
#### Επισκόπηση
Τα σημεία δεδομένων αντιπροσωπεύουν μεμονωμένες τιμές στο TreeMap σας. Η σωστή συσχέτισή τους βελτιώνει την αναγνωσιμότητα του γραφήματος.
#### Βήματα για την προσθήκη σημείων δεδομένων
1. **Δημιουργία νέας σειράς:**
   Αρχικοποιήστε μια σειρά για τα δεδομένα σας:
   ```python
   series = chart.chart_data.series.add(slides.charts.ChartType.TREEMAP)
   ```
2. **Ρύθμιση παραμέτρων ετικετών:**
   Ορίστε επιλογές ετικέτας για βελτίωση της σαφήνειας:
   ```python
   series.labels.default_data_label_format.show_category_name = True
   ```
3. **Προσθήκη σημείων δεδομένων:**
   Συμπληρώστε τη σειρά σας με τιμές που αντιστοιχούν σε κάθε κατηγορία:
   ```python
   data_points = [4, 5, 3, 6, 9, 9, 4, 3]
   cells = [("D1", 4), ("D2", 5), ("D3", 3), ("D4", 6),
            ("D5", 9), ("D6", 9), ("D7", 4), ("D8", 3)]
   
   for cell, value in zip(cells, data_points):
       series.data_points.add_data_point_for_treemap_series(
           wb.get_cell(0, *cell))
   ```
### Οριστικοποίηση και Αποθήκευση
#### Επισκόπηση
Αφού ρυθμίσετε τις παραμέτρους του γραφήματός σας, αποθηκεύστε την παρουσίαση σε ένα αρχείο.
#### Βήματα για αποθήκευση
1. **Αποθήκευση παρουσίασης:**
   Χρησιμοποιήστε το `save()` μέθοδος αποθήκευσης της εργασίας σας:
   ```python
   pres.save("YOUR_OUTPUT_DIRECTORY/charts_tree_map_chart_out.pptx", 
             slides.export.SaveFormat.PPTX)
   ```
Αυτό το βήμα διασφαλίζει ότι το γράφημά σας αποθηκεύεται σε μορφή PPTX, έτοιμο για κοινή χρήση ή περαιτέρω επεξεργασία.

## Οδηγός Εφαρμογής

### Εμφάνιση μονάδας στον άξονα του γραφήματος
#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να επισημάνετε τους άξονες του γραφήματος με προσαρμοσμένες μονάδες όπως εκατομμύρια ή δισεκατομμύρια, βελτιώνοντας την αναγνωσιμότητα των δεδομένων στις παρουσιάσεις.

#### Βήμα προς βήμα εφαρμογή
1. **Αρχικοποίηση της παρουσίασης**
   Ξεκινήστε δημιουργώντας μια νέα παρουσία παρουσίασης όπου θα προστεθεί το γράφημά σας:

   ```python
   with slides.Presentation() as pres:
       # Ο κώδικά σας για τον χειρισμό διαφανειών και γραφημάτων βρίσκεται εδώ.
   ```

2. **Προσθήκη γραφήματος ομαδοποιημένων στηλών**
   Προσθέστε ένα γράφημα ομαδοποιημένων στηλών σε καθορισμένες συντεταγμένες στην πρώτη διαφάνεια:

   ```python
   chart = pres.slides[0].shapes.add_chart(
       slides.charts.ChartType.CLUSTERED_COLUMN, 50, 50, 450, 300
   )
   ```

3. **Ορισμός μονάδας εμφάνισης κατακόρυφου άξονα**
   Ρυθμίστε τον κατακόρυφο άξονα για να εμφανίζει τιμές σε εκατομμύρια:

   ```python
   chart.axes.vertical_axis.display_unit = slides.charts.DisplayUnitType.MILLIONS
   ```

4. **Αποθήκευση της παρουσίασης**
   Αποθηκεύστε την παρουσίασή σας με το διαμορφωμένο γράφημα:

   ```python
   pres.save("YOUR_OUTPUT_DIRECTORY/charts_showing_display_unit_label_out.pptx", slides.export.SaveFormat.PPTX)
   ```

#### Παράμετροι και Μέθοδοι
- `add_chart`: Προσθέτει ένα νέο αντικείμενο γραφήματος στη διαφάνεια.
- `display_unit`: Ορίζει τη μονάδα εμφάνισης για αριθμητικές τιμές στον κατακόρυφο άξονα.

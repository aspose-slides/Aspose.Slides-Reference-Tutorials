## Οδηγός Εφαρμογής

### Προσθήκη και επικύρωση διάταξης γραφήματος

Ας αναλύσουμε πώς να προσθέσετε ένα γράφημα ομαδοποιημένων στηλών και να επικυρώσετε τη διάταξή του.

#### Βήμα 1: Δημιουργία νέας παρουσίασης

Ξεκινήστε δημιουργώντας μια νέα παρουσία μιας παρουσίασης. Αυτή θα είναι η βάση εργασίας μας:

```python
class ChartManager(PresentationManager):
    def __init__(self):
        super().__init__()

    def add_clustered_column_chart(self, x, y, width, height):
        chart = self.pres.slides[0].shapes.add_chart(
            slides.charts.ChartType.CLUSTERED_COLUMN, 
            x, y, width, height
        )
        return chart
```

#### Βήμα 2: Προσθήκη γραφήματος ομαδοποιημένων στηλών

Προσθέστε το γράφημά σας στην πρώτη διαφάνεια στις καθορισμένες συντεταγμένες και διαστάσεις.

```python
# Παράδειγμα χρήσης:
class ChartExample(ChartManager):
    def create_chart(self):
        return self.add_clustered_column_chart(100, 100, 500, 350)
```

#### Βήμα 3: Επικύρωση της διάταξης του γραφήματος

Βεβαιωθείτε ότι το γράφημά σας πληροί τα απαιτούμενα πρότυπα διάταξης χρησιμοποιώντας τη μέθοδο επικύρωσης Aspose.Slides.

```python
class ChartValidator(ChartExample):
    def validate_layout(self, chart):
        try:
            chart.validate_chart_layout()
            print("Chart layout validated successfully.")
        except Exception as e:
            print(f"Error validating chart layout: {e}")
```

#### Βήμα 4: Ανάκτηση διαστάσεων επιφάνειας οικοπέδου

Για περαιτέρω προσαρμογή ή επαλήθευση, εξαγάγετε τις διαστάσεις της επιφάνειας του γραφήματος:

```python
class ChartDimensions(ChartValidator):
    def get_plot_area_dimensions(self, chart):
        x = chart.plot_area.actual_x
        y = chart.plot_area.actual_y
        w = chart.plot_area.actual_width
        h = chart.plot_area.actual_height
        return x, y, w, h
```

#### Βήμα 5: Αποθηκεύστε την παρουσίασή σας

Τέλος, αποθηκεύστε την παρουσίασή σας στην επιθυμητή τοποθεσία.

```python
class ChartSaver(ChartDimensions):
    def run_example(self, output_directory):
        chart = self.create_chart()
        self.validate_layout(chart)
        dimensions = self.get_plot_area_dimensions(chart)
        print(f"Plot Area Dimensions: {dimensions}")
        self.save_presentation(output_directory + "/charts_validate_chart_layout_out.pptx")
```

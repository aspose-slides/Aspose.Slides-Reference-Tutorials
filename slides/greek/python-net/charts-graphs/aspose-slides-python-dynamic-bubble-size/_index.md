## Οδηγός Εφαρμογής

Ας εμβαθύνουμε στον ορισμό δυναμικών μεγεθών φυσαλίδων σε γραφήματα.

### Δημιουργία και τροποποίηση γραφήματος φυσαλίδων

#### Επισκόπηση

Θα δημιουργήσουμε μια παρουσίαση PowerPoint, θα προσθέσουμε ένα γράφημα φυσαλίδων σε αυτήν και θα τροποποιήσουμε τα μεγέθη των φυσαλίδων με βάση συγκεκριμένες διαστάσεις δεδομένων χρησιμοποιώντας το Aspose.Slides.

#### Βήμα προς βήμα εφαρμογή

**1. Αρχικοποίηση παρουσίασης**

Ξεκινήστε δημιουργώντας μια παρουσία του `Presentation` μέσα σε έναν διαχειριστή περιβάλλοντος:

```python
import aspose.slides as slides

def charts_bubble_size_representation():
    with slides.Presentation() as pres:
        # Ο κώδικας συνεχίζεται...
```

**2. Προσθήκη γραφήματος φυσαλίδων**

Προσθήκη γραφήματος φυσαλίδων στη θέση `(50, 50)` με διαστάσεις `600x400` στην πρώτη διαφάνεια.

```python
chart = pres.slides[0].shapes.add_chart(
    slides.charts.ChartType.BUBBLE,
    50, 50, 600, 400, True
)
```

**3. Ορισμός αναπαράστασης μεγέθους φυσαλίδας**

Ρυθμίστε την αναπαράσταση μεγέθους φυσαλίδας σε `WIDTH` για την πρώτη ομάδα σειράς:

```python
chart.chart_data.series_groups[0].bubble_size_representation = \\
    slides.charts.BubbleSizeRepresentationType.WIDTH
```

**4. Αποθήκευση παρουσίασης**

Τέλος, αποθηκεύστε την παρουσίασή σας σε έναν καθορισμένο κατάλογο:

```python
pres.save(
    "YOUR_OUTPUT_DIRECTORY/charts_bubble_size_representation_out.pptx"
)
```

Ακολουθώντας αυτόν τον οδηγό, θα μάθετε πώς να δημιουργείτε γραφήματα γραμμών με ακριβή μορφοποίηση στο PowerPoint χρησιμοποιώντας το Aspose.Slides για Python. Μετατρέψτε τα ακατέργαστα δεδομένα σε κομψές παρουσιάσεις χωρίς κόπο.

**Τι θα μάθετε:**

```python
import aspose.slides as slides
```
## Οδηγός Εφαρμογής
Θα σας καθοδηγήσουμε στη δημιουργία ενός γραφήματος γραμμών και στον ορισμό της ακρίβειας των δεδομένων του. 
### Προσθήκη γραφήματος γραμμών στο PowerPoint
**Επισκόπηση**Θα προσθέσουμε ένα γράφημα γραμμών στην παρουσίασή σας, το οποίο θα εμφανίζει δεδομένα με μορφοποιημένες τιμές.
#### Βήμα 1: Αρχικοποίηση παρουσίασης
Δημιουργήστε μια παρουσία του `Presentation` τάξη χρησιμοποιώντας το `with` Δήλωση για αποτελεσματική διαχείριση πόρων:
```python
with slides.Presentation() as pres:
    # Ο κωδικός σας εδώ
```
#### Βήμα 2: Προσθήκη γραφήματος γραμμών
Προσθέστε ένα γράφημα στην πρώτη διαφάνεια, καθορίζοντας τη θέση και το μέγεθός του:
```python
chart = pres.slides[0].shapes.add_chart(
    slides.charts.ChartType.LINE, 50, 50, 450, 300
)
```
**Επεξήγηση παραμέτρων**: 
- `ChartType.LINE`: Καθορίζει ότι πρόκειται για γράφημα γραμμών.
- `(50, 50)`Θέσεις X και Y στη διαφάνεια.
- `(450, 300)`: Πλάτος και ύψος του γραφήματος.
#### Βήμα 3: Ενεργοποίηση πίνακα δεδομένων
Εμφάνιση τιμών δεδομένων απευθείας στο γράφημα:
```python
chart.has_data_table = True
```
#### Βήμα 4: Ορισμός μορφής αριθμών
Μορφοποιήστε τους αριθμούς σε δύο δεκαδικά ψηφία για ακρίβεια:
```python
chart.chart_data.series[0].number_format_of_values = "#,##0.00"
```
**Γιατί αυτό έχει σημασία**: Εξασφαλίζει σαφήνεια και συνέπεια στην αναπαράσταση δεδομένων.
### Αποθήκευση της παρουσίασής σας
Τέλος, αποθηκεύστε την παρουσίασή σας σε έναν καθορισμένο κατάλογο:
```python
pres.save("YOUR_OUTPUT_DIRECTORY/charts_precision_of_data_out.pptx", slides.export.SaveFormat.PPTX)
```

## Οδηγός Εφαρμογής

### Προσθήκη ψηφιακής υπογραφής στο PowerPoint

Η προσθήκη μιας ψηφιακής υπογραφής περιλαμβάνει διάφορα βήματα:

#### Βήμα 1: Φόρτωση ή δημιουργία παρουσίασης
Ξεκινήστε ανοίγοντας μια υπάρχουσα παρουσίαση ή δημιουργώντας μια νέα χρησιμοποιώντας το Aspose.Slides:

```python
import aspose.slides as slides

# Άνοιγμα ή δημιουργία παρουσίασης
class SecurePPTWithSignature:
    def __init__(self):
        self.pres = None

    def load_or_create_presentation(self, path=None):
        if path:
            self.pres = slides.Presentation(path)
        else:
            self.pres = slides.Presentation()
```

Αυτός ο κώδικας αρχικοποιεί το αρχείο PowerPoint στο οποίο θα εργαστείτε. Εάν δεν υπάρχει, δημιουργείται ένα νέο.

#### Βήμα 2: Δημιουργήστε το αντικείμενο DigitalSignature
Για να προσθέσετε μια ψηφιακή υπογραφή, δημιουργήστε πρώτα μια παρουσία του `DigitalSignature` χρησιμοποιώντας το αρχείο πιστοποιητικού και τον κωδικό πρόσβασής σας:

```python
class SecurePPTWithSignature(SecurePPTWithSignature):
    def __init__(self, cert_path, cert_password):
        super().__init__()
        self.signature = slides.DigitalSignature(cert_path, cert_password)
```

Εδώ, `"YOUR_DOCUMENT_DIRECTORY/cert.pfx"` είναι η διαδρομή προς το ψηφιακό σας πιστοποιητικό και `"testpass1"` είναι ο αντίστοιχος κωδικός πρόσβασης.

#### Βήμα 3: Προσθήκη σχολίων (Προαιρετικό)
Η προσθήκη σχολίων μπορεί να βοηθήσει στην αναγνώριση ή την τήρηση αρχείων:

```python
class SecurePPTWithSignature(SecurePPTWithSignature):
    def add_comments_to_signature(self, comment):
        self.signature.comments = comment
```

Αυτό το βήμα είναι προαιρετικό αλλά συνιστάται για καλύτερη τεκμηρίωση.

#### Βήμα 4: Προσθήκη της ψηφιακής υπογραφής στην παρουσίαση
Ενσωματώστε την ψηφιακή σας υπογραφή στο αντικείμενο παρουσίασης:

```python
class SecurePPTWithSignature(SecurePPTWithSignature):
    def add_signature_to_presentation(self):
        if self.pres:
            self.pres.digital_signatures.add(self.signature)
```

Καλώντας `add()`, ασφαλίζετε το PowerPoint με το παρεχόμενο πιστοποιητικό.

#### Βήμα 5: Αποθήκευση της υπογεγραμμένης παρουσίασης
Τέλος, αποθηκεύστε την παρουσίασή σας σε μορφή PPTX, συμπεριλαμβανομένης της ψηφιακής υπογραφής:

```python
class SecurePPTWithSignature(SecurePPTWithSignature):
    def save_signed_presentation(self, output_path):
        if self.pres:
            self.pres.save(output_path, slides.export.SaveFormat.PPTX)
```

Το αρχείο θα αποθηκευτεί στο `"YOUR_OUTPUT_DIRECTORY"`Βεβαιωθείτε ότι αυτός ο κατάλογος υπάρχει ή προσαρμόστε τη διαδρομή ανάλογα.

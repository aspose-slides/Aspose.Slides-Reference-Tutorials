## Οδηγός Εφαρμογής

### Πρόσβαση σε προσαρμοσμένες ιδιότητες

#### Επισκόπηση
Η πρόσβαση σε προσαρμοσμένες ιδιότητες σάς επιτρέπει να ανακτήσετε μεταδεδομένα που είναι αποθηκευμένα σε μια παρουσίαση PowerPoint. Αυτά μπορεί να περιλαμβάνουν σημειώσεις συντάκτη ή πληροφορίες έκδοσης.

#### Βήματα Υλοποίησης

##### Φόρτωση της παρουσίασης
Ξεκινήστε ανοίγοντας το αρχείο PowerPoint που επιθυμείτε:
```python
class PresentationManager:
    # ... προηγούμενος κώδικας ...

    def access_properties(self):
        with self.load_presentation() as presentation:
            document_properties = presentation.document_properties

            for i in range(document_properties.count_of_custom_properties):
                custom_property_name = document_properties.get_custom_property_name(i)
                custom_property_value = document_properties.get_custom_property_value(i)

                # Εκτυπώστε τις λεπτομέρειες της τρέχουσας προσαρμοσμένης ιδιότητας
                print(f"Custom Property Name: {custom_property_name}")
                print(f"Custom Property Value: {custom_property_value}")
```

### Τροποποίηση προσαρμοσμένων ιδιοτήτων

#### Επισκόπηση
Μόλις αποκτήσετε πρόσβαση στις ιδιότητές σας, η τροποποίησή τους μπορεί να σας βοηθήσει να διατηρείτε τις παρουσιάσεις σας ενημερωμένες με σχετικές πληροφορίες.

#### Βήματα Υλοποίησης

##### Ενημέρωση κάθε ιδιότητας
Αλλάξτε κάθε προσαρμοσμένη ιδιότητα σε μια νέα τιμή χρησιμοποιώντας τον δείκτη της:
```python
class PresentationManager:
    # ... προηγούμενος κώδικας ...

    def modify_properties(self):
        with self.load_presentation() as presentation:
            document_properties = presentation.document_properties

            for i in range(document_properties.count_of_custom_properties):
                new_value = f"New Value {i + 1}"
                document_properties.set_custom_property_value(i, new_value)

            # Αποθήκευση της τροποποιημένης παρουσίασης σε έναν κατάλογο εξόδου
            output_path = "YOUR_OUTPUT_DIRECTORY/modified_presentation.pptx"
            presentation.save(output_path, slides.export.SaveFormat.PPTX)
```

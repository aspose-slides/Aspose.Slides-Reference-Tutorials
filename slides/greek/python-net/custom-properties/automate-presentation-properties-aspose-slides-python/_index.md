## Οδηγός Εφαρμογής
### Δημιουργία ιδιοτήτων προτύπου
Αυτή η λειτουργία επιτρέπει τον ορισμό ιδιοτήτων εγγράφου που μπορούν να εφαρμοστούν ομοιόμορφα σε όλες τις παρουσιάσεις.
#### Επισκόπηση
Ο `create_template_properties` Η συνάρτηση ορίζει χαρακτηριστικά μεταδεδομένων όπως συγγραφέα, τίτλο και λέξεις-κλειδιά σε ένα πρότυπο.
#### Απόσπασμα κώδικα
```python
def create_template_properties():
    # Ρύθμιση παραμέτρων ενός νέου αντικειμένου DocumentProperties
    template = slides.DocumentProperties()
    template.author = 'Template Author'
    template.title = 'Template Title'
    template.category = 'Template Category'
    template.keywords = 'Keyword1, Keyword2, Keyword3'
    template.company = 'Our Company'
    template.comments = 'Created from template'
    template.content_type = 'Template Content'
    template.subject = 'Template Subject'

    return template
```
#### Εξήγηση
- **Ιδιότητες Εγγράφου**: Διατηρεί μεταδεδομένα για μια παρουσίαση.
- **Παράμετροι**Προσαρμόστε πεδία όπως `author`, `title` για να ταιριάζει στις ανάγκες σας.

### Αντιγραφή και ενημέρωση παρουσιάσεων με ιδιότητες προτύπου
Αυτοματοποιήστε την αντιγραφή παρουσιάσεων από έναν κατάλογο σε έναν άλλο, ενώ παράλληλα ενημερώνετε τις ιδιότητές τους χρησιμοποιώντας ένα πρότυπο.
#### Επισκόπηση
Ο `copy_and_update_presentations` Η συνάρτηση διαχειρίζεται τις λειτουργίες αρχείων και ενημερώνει τις ιδιότητες του εγγράφου για κάθε αντιγραμμένη παρουσίαση.
#### Βήματα που εμπλέκονται
1. **Αντιγραφή αρχείων**: Χρήση `shutil.copyfile()` για να αντιγράψετε αρχεία.
2. **Ενημέρωση ιδιοτήτων**: Εφαρμόστε το πρότυπο που δημιουργήθηκε νωρίτερα σε κάθε παρουσίαση.
#### Απόσπασμα κώδικα
```python
import shutil

def copy_and_update_presentations():
    # Λίστα παρουσιάσεων προς επεξεργασία
    presentation_files = ['doc1.pptx', 'doc2.odp', 'doc3.ppt']
    
    for file_name in presentation_files:
        # Αντιγραφή αρχείων από την πηγή στον προορισμό
        shutil.copyfile('YOUR_DOCUMENT_DIRECTORY/' + file_name,
                        'YOUR_OUTPUT_DIRECTORY/' + file_name)
    
    template = create_template_properties()
    
    for file_name in presentation_files:
        update_by_template('YOUR_OUTPUT_DIRECTORY/' + file_name, template)

def update_by_template(path, template):
    # Ανάκτηση και ενημέρωση ιδιοτήτων εγγράφου
    to_update = slides.PresentationFactory.instance.get_presentation_info(path)
    to_update.update_document_properties(template)
    to_update.write_binded_presentation(path)
```
#### Εξήγηση
- **shutil.copyfile()**: Αντιγράφει αρχεία διατηρώντας παράλληλα τα μεταδεδομένα.
- **ενημέρωση_από_πρότυπο()**Ενημερώνει τις ιδιότητες κάθε παρουσίασης χρησιμοποιώντας το καθορισμένο πρότυπο.

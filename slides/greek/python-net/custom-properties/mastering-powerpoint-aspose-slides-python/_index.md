## Οδηγός Εφαρμογής
Τώρα που έχουμε έτοιμη τη ρύθμισή μας, ας εξερευνήσουμε τις δυνατότητες προσθήκης προσαρμοσμένων ιδιοτήτων σε παρουσιάσεις PowerPoint.
### Προσθήκη προσαρμοσμένων ιδιοτήτων εγγράφου
#### Επισκόπηση
Η προσθήκη προσαρμοσμένων ιδιοτήτων εγγράφου σάς επιτρέπει να ενσωματώσετε μεταδεδομένα στα αρχεία PowerPoint σας. Αυτά μπορεί να είναι οτιδήποτε, από στοιχεία συντάκτη έως πληροφορίες έργου ή αριθμούς έκδοσης.
#### Βήματα για την Υλοποίηση
##### Βήμα 1: Δημιουργήστε την Κλάση Παρουσίασης
Ξεκινήστε δημιουργώντας ένα αντικείμενο παρουσίασης:
```python
with slides.Presentation() as presentation:
    # Πρόσβαση στις Ιδιότητες Εγγράφου
    document_properties = presentation.document_properties
```
##### Βήμα 2: Προσθήκη προσαρμοσμένων ιδιοτήτων
Μπορείτε να προσθέσετε προσαρμοσμένες ιδιότητες χρησιμοποιώντας `set_custom_property_value` μέθοδος. Δείτε πώς μπορείτε να προσθέσετε τρεις διαφορετικές προσαρμοσμένες ιδιότητες:
```python
document_properties.set_custom_property_value("New Custom", 12)
document_properties.set_custom_property_value("My Name", "Mudassir")
document_properties.set_custom_property_value("Custom", 124)
```
- **Παράμετροι**Η πρώτη παράμετρος είναι το όνομα της ιδιότητας (μια συμβολοσειρά) και η δεύτερη είναι η τιμή της, η οποία μπορεί να είναι οποιουδήποτε τύπου δεδομένων που υποστηρίζεται από τις ιδιότητες του PowerPoint.
##### Βήμα 3: Ανάκτηση ακινήτου
Για να ανακτήσετε το όνομα μιας προσαρμοσμένης ιδιότητας κατά ευρετήριο:
```python
property_name = document_properties.get_custom_property_name(2)
```
- **Εξήγηση**: Αυτό ανακτά το όνομα της τρίτης ιδιότητας (ο δείκτης βασίζεται στο μηδέν).
##### Βήμα 4: Κατάργηση προσαρμοσμένης ιδιότητας
Μπορείτε να καταργήσετε ιδιότητες χρησιμοποιώντας τα ονόματά τους:
```python
document_properties.remove_custom_property(property_name)
```
Αυτό το βήμα διασφαλίζει ότι η επιλεγμένη προσαρμοσμένη ιδιότητα θα καταργηθεί από το έγγραφό σας.
##### Αποθήκευση της παρουσίασής σας
Μην ξεχάσετε να αποθηκεύσετε την παρουσίασή σας αφού κάνετε αλλαγές:
```python
presentation.save("YOUR_OUTPUT_DIRECTORY/props_add_custom_document_properties_out.pptx", slides.export.SaveFormat.PPTX)
```

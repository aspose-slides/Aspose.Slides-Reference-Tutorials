## Οδηγός Εφαρμογής
Αυτή η ενότητα θα σας καθοδηγήσει στην ενσωμάτωση και την περικοπή ήχου χρησιμοποιώντας το Aspose.Slides.

### Προσθήκη ηχητικού πλαισίου σε παρουσίαση
**Επισκόπηση**Βελτιώστε την διαδραστικότητα της παρουσίασής σας προσθέτοντας ένα αρχείο ήχου ως ενσωματωμένο καρέ σε μια διαφάνεια του PowerPoint.

#### Βήμα 1: Ανοίξτε την παρουσίαση για τροποποίηση
```python
# Άνοιγμα ή δημιουργία νέας παρουσίασης
current_pres = slides.Presentation()
```

#### Βήμα 2: Ανάγνωση και προσθήκη αρχείου ήχου
```python
    # Ανοίξτε το αρχείο ήχου από τον κατάλογό σας σε δυαδική λειτουργία
    with open('YOUR_DOCUMENT_DIRECTORY/audio.m4a', 'rb') as audio_file:
        # Προσθήκη του ήχου στη συλλογή της παρουσίασης
        current_audio = current_pres.audios.add_audio(audio_file)
```

#### Βήμα 3: Ενσωμάτωση ηχητικού πλαισίου σε διαφάνεια
```python
    # Προσθήκη ενσωματωμένου ηχητικού καρέ σε καθορισμένες συντεταγμένες (50, 50) με μέγεθος (100, 100)
    audio_frame = current_pres.slides[0].shapes.add_audio_frame_embedded(50, 50, 100, 100, current_audio)
```

### Περικοπή πλαισίου ήχου σε παρουσίαση
**Επισκόπηση**Η περικοπή της αρχής και του τέλους ενός ηχητικού καρέ μπορεί να είναι ζωτικής σημασίας για τον ακριβή χρονισμό στην παρουσίασή σας.

#### Βήμα 1: Ορισμός έναρξης περικοπής
```python
    # Περικοπή της αρχής του ήχου κατά 500 χιλιοστά του δευτερολέπτου (0,5 δευτερόλεπτα)
    audio_frame.trim_from_start = 500
```

#### Βήμα 2: Ορισμός κοπής άκρων
```python
    # Περικοπή του τέλους του ήχου κατά 1000 χιλιοστά του δευτερολέπτου (1 δευτερόλεπτο)
    audio_frame.trim_from_end = 1000
```

### Αποθήκευση της παρουσίασης
Αποθηκεύστε την τροποποιημένη παρουσίασή σας σε έναν κατάλογο εξόδου:
```python
    current_pres.save('YOUR_OUTPUT_DIRECTORY/AudioFrameTrim_out.pptx', slides.export.SaveFormat.PPTX)
```

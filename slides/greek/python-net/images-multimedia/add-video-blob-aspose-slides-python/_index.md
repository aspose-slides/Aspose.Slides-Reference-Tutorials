## Οδηγός Εφαρμογής

Τώρα, ας αναλύσουμε τη διαδικασία προσθήκης ενός βίντεο blob στην παρουσίαση του PowerPoint σας.

### 1. Προετοιμασία του περιβάλλοντός σας

Ξεκινήστε ρυθμίζοντας καταλόγους για αρχεία εισόδου και εξόδου:

```python
import os

# Ορισμός διαδρομών για την αποθήκευση εγγράφων
data_directory = 'YOUR_DOCUMENT_DIRECTORY/'
output_directory = 'YOUR_OUTPUT_DIRECTORY/'

# Δημιουργήστε καταλόγους εάν δεν υπάρχουν
os.makedirs(data_directory, exist_ok=True)
os.makedirs(output_directory, exist_ok=True)
```

### 2. Δημιουργία αρχείου βίντεο

Για σκοπούς επίδειξης, δημιουργήστε ένα αρχείο βίντεο με placeholder:

```python
video_file_path = os.path.join(data_directory, "video.mp4")
with open(video_file_path, 'wb') as video_file:
    # Προσομοιωμένα δυαδικά δεδομένα για το παράδειγμα
    video_file.write(b'\x00\x01\x02')
```

### 3. Προσθήκη του βίντεο σε μια παρουσίαση

Τώρα, ας προσθέσουμε αυτό το βίντεο ως blob σε ένα νέο αρχείο PowerPoint:

```python
with slides.Presentation() as pres:
    with open(video_file_path, "rb") as file_stream:
        # Προσθέστε το βίντεο χρησιμοποιώντας τη συμπεριφορά KEEP_LOCKED για ασφάλεια
        video = pres.videos.add_video(file_stream, slides.LoadingStreamBehavior.KEEP_LOCKED)
        
        # Εισαγωγή ενός καρέ βίντεο στην πρώτη διαφάνεια
        pres.slides[0].shapes.add_video_frame(0, 0, 480, 270, video)

    # Αποθηκεύστε την παρουσίασή σας με το πρόσθετο βίντεο blod
    output_file_path = os.path.join(output_directory, "props_add_blob_to_presentation_out.pptx")
    pres.save(output_file_path, slides.export.SaveFormat.PPTX)
```

**Βασικές επιλογές διαμόρφωσης:**
- **Συμπεριφορά KEEP_LOCKED**Διασφαλίζει ότι μόλις ενσωματωθεί ένα βίντεο, δεν μπορεί να τροποποιηθεί ακούσια.

### Εξαγωγή ήχου από υπερσύνδεσμο

Αυτή η λειτουργία σάς επιτρέπει να έχετε πρόσβαση και να εξάγετε δεδομένα ήχου που συνδέονται μέσω υπερσυνδέσμου στο πρώτο σχήμα της πρώτης διαφάνειας σε μια παρουσίαση PowerPoint. Αυτό είναι ιδιαίτερα χρήσιμο για παρουσιάσεις όπου οι διαφάνειες συμπληρωμάτων ήχου δεν ενσωματώνονται απευθείας σε αυτές.


Χρησιμοποιήστε το Aspose.Slides για να ανοίξετε το αρχείο παρουσίασής σας, διασφαλίζοντας ότι διαθέτει υπερσυνδέσμους με δεδομένα ήχου.

```python
with slides.Presentation(input_directory + 'HyperlinkSound.pptx') as pres:
```


##### 4. Εξαγωγή και αποθήκευση δεδομένων ήχου

## Οδηγός Εφαρμογής

### Προσθήκη εικόνας Blob στο PowerPoint

Αυτή η λειτουργία δείχνει πώς να προσθέσετε μια μεγάλη εικόνα ως blob διατηρώντας παράλληλα την αποτελεσματικότητα της μνήμης χρησιμοποιώντας το Aspose.Slides.

#### Οδηγίες βήμα προς βήμα

1. **Άνοιγμα και ανάγνωση του αρχείου εικόνας**
   - Διαβάστε το μεγάλο αρχείο εικόνας σας σε δυαδική λειτουργία για αποτελεσματική επεξεργασία:
   ```python
   with open("YOUR_DOCUMENT_DIRECTORY/large_image.jpg", "br") as file_stream:
       # Αυτό διασφαλίζει την αποτελεσματική χρήση της μνήμης κατά τον χειρισμό μεγάλων αρχείων
   ```

2. **Δημιουργία νέας παρουσίας παρουσίασης**
   - Αρχικοποιήστε μια νέα παρουσίαση, η οποία θα χρησιμεύσει ως κοντέινερ για την εικόνα σας:
   ```python
   with slides.Presentation() as pres:
       # Αυτός ο διαχειριστής περιβάλλοντος χειρίζεται αυτόματα τη διαχείριση πόρων
   ```

3. **Προσθήκη εικόνας σε παρουσίαση χρησιμοποιώντας τη συμπεριφορά KEEP_LOCKED**
   - Προσθέστε την εικόνα χρησιμοποιώντας συγκεκριμένη συμπεριφορά φόρτωσης για αποτελεσματική διαχείριση μνήμης:
   ```python
   img = pres.images.add_image(file_stream, slides.LoadingStreamBehavior.KEEP_LOCKED)
       # Διατηρεί το αρχείο κλειδωμένο κατά την επεξεργασία για βέλτιστο χειρισμό πόρων
   ```

4. **Εισαγωγή κορνίζας εικόνας στην πρώτη διαφάνεια**
   - Τοποθετήστε την εικόνα μέσα σε μια διαφάνεια χρησιμοποιώντας καθορισμένες διαστάσεις και θέση:
   ```python
   pres.slides[0].shapes.add_picture_frame(
       slides.ShapeType.RECTANGLE, 0, 0, 300, 200, img
   )
       # Ορίζει τον τύπο σχήματος και το μέγεθος του πλαισίου στη διαφάνεια
   ```

5. **Αποθήκευση της παρουσίασης**
   - Αποθηκεύστε την παρουσίασή σας σε μορφή PPTX:
   ```python
   pres.save("YOUR_OUTPUT_DIRECTORY/save_add_blob_image_out.pptx", slides.export.SaveFormat.PPTX)
       # Αποθηκεύει όλες τις αλλαγές σε ένα νέο αρχείο στον καθορισμένο κατάλογο
   ```

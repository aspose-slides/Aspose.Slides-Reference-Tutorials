## Οδηγός Εφαρμογής

### Χαρακτηριστικό 1: Προσθήκη καρέ βίντεο από το YouTube

Αυτή η λειτουργία δείχνει πώς να προσθέσετε ένα καρέ βίντεο με ένα βίντεο YouTube και τη μικρογραφία του σε μια διαφάνεια του PowerPoint.

#### Οδηγός βήμα προς βήμα

##### Βήμα 1: Δημιουργήστε ένα καρέ βίντεο
Δημιουργήστε ένα καρέ βίντεο στην πρώτη διαφάνεια στη θέση (10, 10) με διαστάσεις 427x240 pixel:
```python
def add_video_from_youtube(pres, video_id):
    video_frame = pres.slides[0].shapes.add_video_frame(10, 10, 427, 240, "https://www.youtube.com/embed/" + video_id)
```
*Οι παράμετροι καθορίζουν τη θέση και το μέγεθος του καρέ βίντεο μέσα στη διαφάνεια.*

##### Βήμα 2: Ορισμός λειτουργίας αναπαραγωγής βίντεο
Ρυθμίστε τη λειτουργία αναπαραγωγής ώστε να ξεκινά αυτόματα όταν κάνετε κλικ:
```python
    video_frame.play_mode = slides.VideoPlayModePreset.AUTO
```

##### Βήμα 3: Φόρτωση εικόνας μικρογραφίας
Ανάκτηση και ορισμός μιας μικρογραφίας εικόνας από το YouTube για το καρέ βίντεο:
```python
    from urllib.request import urlopen
    
    thumbnail_uri = "http://img.youtube.com/vi/" + video_id + "/hqdefault.jpg"
    with urlopen(thumbnail_uri) as f:
        video_frame.picture_format.picture.image = pres.images.add_image(f.read())
```

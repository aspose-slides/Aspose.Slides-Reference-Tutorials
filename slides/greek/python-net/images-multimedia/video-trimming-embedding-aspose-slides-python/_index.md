## Οδηγός Εφαρμογής

### Περικοπή και ενσωμάτωση βίντεο σε διαφάνειες PowerPoint

Αυτή η λειτουργία μας επιτρέπει να περικόψουμε ένα βίντεο κλιπ και να το ενσωματώσουμε σε μια παρουσίαση PowerPoint χρησιμοποιώντας το Aspose.Slides για Python.

#### Προσθήκη καρέ βίντεο σε μια διαφάνεια

Αρχικά, καθορίστε διαδρομές για το βίντεο προέλευσης και τον κατάλογο εξόδου. Στη συνέχεια, δημιουργήστε μια νέα παρουσία παρουσίασης:

```python
import aspose.slides as slides
from pathlib import Path

video_file_name = Path("YOUR_DOCUMENT_DIRECTORY/") / "Wildlife.mp4"
output_file_path = Path("YOUR_OUTPUT_DIRECTORY/") / "VideoTrimming-out.pptx"

with slides.Presentation() as pres:
    slide = pres.slides[0]
```

#### Ανάγνωση και προσθήκη δεδομένων βίντεο

Στη συνέχεια, διαβάστε το αρχείο βίντεο και προσθέστε το στην παρουσίαση:

```python
    with open(video_file_name, "rb") as video_file:
        video_data = video_file.read()
        video = pres.videos.add_video(video_data)
        
    # Προσθήκη καρέ βίντεο στη διαφάνεια
    video_frame = slide.shapes.add_video_frame(0, 0, 200, 200, video)
```

#### Περικοπή του βίντεο

Ρυθμίστε την περικοπή καθορίζοντας τις ώρες έναρξης και λήξης σε χιλιοστά του δευτερολέπτου:

```python
    # Περικοπή από την αρχή (12 δευτερόλεπτα) έως το τέλος (16 δευτερόλεπτα)
    video_frame.trim_from_start = 12000
    video_frame.trim_from_end = 14000
    
    pres.save(str(output_file_path), slides.export.SaveFormat.PPTX)
```

### Εξήγηση

- **Παράμετροι**: `trim_from_start` και `trim_from_end` προσδιορίστε το κομμένο τμήμα του βίντεο.
- **Σκοπός**Η περικοπή βελτιστοποιεί το μήκος της παρουσίασης χωρίς περιττό περιεχόμενο.

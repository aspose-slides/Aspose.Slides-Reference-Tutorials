## Οδηγός Εφαρμογής

Ας αναλύσουμε πώς μπορείτε να ορίσετε μετατοπίσεις τεντώματος για πλαίσια εικόνων μέσα στα Αυτόματα Σχήματα σε διαφάνειες του PowerPoint.

### Ρύθμιση μετατοπίσεων τεντώματος σε κορνίζες

Ο στόχος εδώ είναι να προσαρμόσετε το γέμισμα της εικόνας μέσα σε ένα σχήμα, διασφαλίζοντας ότι ταιριάζει απόλυτα στις ανάγκες του σχεδιασμού σας. Ακολουθήστε τα παρακάτω βήματα:

#### 1. Δημιουργία αρχικών παρουσιάσεων

Ξεκινήστε δημιουργώντας μια παρουσία του `Presentation` τάξη:
```python
with slides.Presentation() as pres:
    slide = pres.slides[0]
```
Αυτό ανοίγει την πρώτη διαφάνεια για επεξεργασία.

#### 2. Φόρτωση και προσθήκη εικόνας

Φορτώστε την εικόνα που επιθυμείτε στη συλλογή εικόνων της παρουσίασης:
```python
img = slides.Images.from_file('YOUR_DOCUMENT_DIRECTORY/image1.jpg')
imgx = pres.images.add_image(img)
```
Αντικαθιστώ `'YOUR_DOCUMENT_DIRECTORY/image1.jpg'` με τη διαδρομή προς την εικόνα σας.

#### 3. Προσθήκη Αυτόματου Σχήματος και Ορισμός Τύπου Γεμίσματος

Προσθέστε ένα ορθογώνιο σχήμα στη διαφάνεια:
```python
auto_shape = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 100, 100, 300, 300)
auto_shape.fill_format.fill_type = slides.FillType.PICTURE
```
Αυτός ο κώδικας καθορίζει τη θέση και το μέγεθος του σχήματος στη διαφάνεια.

#### 4. Ρύθμιση παραμέτρων λειτουργίας γεμίσματος εικόνας

Ορίστε τη λειτουργία γεμίσματος εικόνας σε τέντωμα:
```python
auto_shape.fill_format.picture_fill_format.picture_fill_mode = slides.PictureFillMode.STRETCH
auto_shape.fill_format.picture_fill_format.picture.image = imgx
```
Αυτό διασφαλίζει ότι η εικόνα σας τεντώνεται για να χωρέσει μέσα στο σχήμα.

#### 5. Ορισμός μετατοπίσεων τεντώματος

Ρυθμίστε τις μετατοπίσεις για ακριβή τοποθέτηση:
```python
auto_shape.fill_format.picture_fill_format.stretch_offset_left = 25
auto_shape.fill_format.picture_fill_format.stretch_offset_right = 25
auto_shape.fill_format.picture_fill_format.stretch_offset_top = -20
auto_shape.fill_format.picture_fill_format.stretch_offset_bottom = -10
```
Αυτές οι τιμές τροποποιούν τον τρόπο με τον οποίο η εικόνα ευθυγραμμίζεται εντός των ορίων του σχήματος.

#### 6. Αποθήκευση παρουσίασης

Τέλος, αποθηκεύστε τις αλλαγές σας:
```python
pres.save('YOUR_OUTPUT_DIRECTORY/shapes_stretch_offset_out.pptx', slides.export.SaveFormat.PPTX)
```
Αντικαθιστώ `'YOUR_OUTPUT_DIRECTORY'` με την επιθυμητή διαδρομή εξόδου.

### Συμβουλές αντιμετώπισης προβλημάτων

- Βεβαιωθείτε ότι η διαδρομή της εικόνας είναι σωστή για να αποφύγετε σφάλματα "δεν βρέθηκε αρχείο".
- Ελέγξτε ότι οι μετατοπίσεις δεν υπερβαίνουν τα όρια του σχήματος, κάτι που μπορεί να προκαλέσει μη αναμενόμενα αποτελέσματα.

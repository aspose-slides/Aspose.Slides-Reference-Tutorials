## Οδηγός Εφαρμογής

### Προσθήκη κουκκίδων εικόνας σε παραγράφους στο PowerPoint

#### Επισκόπηση
Βελτιώστε την οπτική ελκυστικότητα και προσελκύστε το κοινό σας προσθέτοντας κουκκίδες εικόνας σε παραγράφους μέσα σε μια διαφάνεια.

#### Βήματα για την εφαρμογή

**Πρόσβαση στη διαφάνεια:**
```python
# Άνοιγμα ή δημιουργία παρουσίασης
with slides.Presentation() as presentation:
    # Πρόσβαση στην πρώτη διαφάνεια
    slide = presentation.slides[0]
```

**Προσθήκη εικόνας για κουκκίδες:**
```python
# Φόρτωση εικόνας από αρχείο και προσθήκη στη συλλογή εικόνων της παρουσίασης
image = slides.Images.from_file("YOUR_DOCUMENT_DIRECTORY/bullets.png")
ippx_image = presentation.images.add_image(image)
```
*Αυτό το βήμα περιλαμβάνει τη φόρτωση της επιθυμητής εικόνας κουκκίδας και την προσθήκη της στη διαφάνεια.*

**Δημιουργία πλαισίου κειμένου με κουκκίδες εικόνας:**
```python
# Προσθήκη ενός Αυτόματου Σχήματος (ορθογώνιου) και πρόσβαση στο πλαίσιο κειμένου του
auto_shape = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 200, 200, 400, 200)
text_frame = auto_shape.text_frame

# Αφαίρεση προεπιλεγμένης παραγράφου, εάν υπάρχει
if len(text_frame.paragraphs) > 0:
    text_frame.paragraphs.remove_at(0)

# Δημιουργήστε μια νέα παράγραφο και ορίστε τον τύπο κουκκίδας σε εικόνα
paragraph = slides.Paragraph()
paragraph.text = "Welcome to Aspose.Slides"
paragraph.paragraph_format.bullet.type = slides.BulletType.PICTURE
paragraph.paragraph_format.bullet.picture.image = ippx_image
paragraph.paragraph_format.bullet.height = 100

# Προσθήκη της παραγράφου στο πλαίσιο κειμένου
text_frame.paragraphs.add(paragraph)
```
*Αυτό το μπλοκ κώδικα δημιουργεί μια νέα παράγραφο, αντιστοιχίζει μια εικόνα ως κουκκίδα και προσαρμόζει τις ιδιότητές της.*

**Αποθήκευση της παρουσίασης:**
```python
# Αποθήκευση της παρουσίασής σας με τις αλλαγές
presentation.save("YOUR_OUTPUT_DIRECTORY/text_picture_bullets_out.pptx", slides.export.SaveFormat.PPTX)
```

### Πρόσβαση και χειρισμός στοιχείων διαφάνειας

#### Επισκόπηση
Μάθετε πώς να έχετε πρόσβαση σε στοιχεία διαφανειών, όπως σχήματα και πλαίσια κειμένου, για περαιτέρω προσαρμογή.

**Πρόσβαση στη διαφάνεια και το σχήμα:**
```python
# Άνοιγμα ή δημιουργία παρουσίασης
with slides.Presentation() as presentation:
    # Πρόσβαση στην πρώτη διαφάνεια
    slide = presentation.slides[0]

    # Προσθέστε ένα Αυτόματο Σχήμα (ορθογώνιο) για να δείξετε τον χειρισμό
    auto_shape = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 200, 200, 400, 200)
text_frame = auto_shape.text_frame

    # Αφαίρεση της πρώτης παραγράφου, εάν υπάρχει
    if len(text_frame.paragraphs) > 0:
        text_frame.paragraphs.remove_at(0)

    # Δημιουργία και προσθήκη νέας παραγράφου με προσαρμοσμένο κείμενο
    paragraph = slides.Paragraph()
    paragraph.text = "Manipulating Slide Elements"
text_frame.paragraphs.add(paragraph)
```

**Αποθήκευση της τροποποιημένης παρουσίασης:**
```python
# Αποθήκευση της παρουσίασης μετά τις τροποποιήσεις
presentation.save("YOUR_OUTPUT_DIRECTORY/modified_slide.pptx", slides.export.SaveFormat.PPTX)
```

##### **Φόρτωση της εικόνας**
Στη συνέχεια, φορτώστε την επιθυμητή εικόνα στη συλλογή εικόνων της παρουσίασης:

```python
        img = slides.Images.from_file('YOUR_DOCUMENT_DIRECTORY/image1.jpg')
        image = presentation.images.add_image(img)
```

**Εξήγηση**: Το `Images.from_file()` Η μέθοδος φορτώνει μια εικόνα από μια καθορισμένη διαδρομή και την προσθέτει στη συλλογή της παρουσίασης.

##### **Προσθήκη Κορνίζας**
Τώρα, προσθέστε το πλαίσιο εικόνας στην πρώτη διαφάνεια με συγκεκριμένες διαστάσεις:

```python

        )
```

**Εξήγηση**: Το `add_picture_frame()` Η μέθοδος τοποθετεί ένα ορθογώνιο πλαίσιο σε συντεταγμένες (50, 50) με πλάτος και ύψος 100 μονάδες. Οι παράμετροι καθορίζουν τον τύπο σχήματος, τη θέση, το μέγεθος και την εικόνα.

##### **Ορισμός σχετικής κλίμακας πλάτους και ύψους**

```python

        pf.relative_scale_width = 1.35
```

**Εξήγηση**Αυτές οι ιδιότητες σάς επιτρέπουν να προσαρμόσετε δυναμικά το ύψος και το πλάτος του πλαισίου σε σχέση με το αρχικό του μέγεθος.

##### **Αποθήκευση της παρουσίασης**


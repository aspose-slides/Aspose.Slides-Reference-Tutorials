## Οδηγός Εφαρμογής

Τώρα, ας εφαρμόσουμε την προσθήκη μιας εικόνας ως κορνίζας.

### Προσθήκη εικόνας ως κορνίζας (Επισκόπηση λειτουργιών)

Αυτή η λειτουργία περιλαμβάνει τη φόρτωση μιας εικόνας και την τοποθέτησή της μέσα σε μια διαφάνεια ως κορνίζα. Είναι χρήσιμη για την προσαρμογή παρουσιάσεων με οπτικά στοιχεία ενσωματωμένα άψογα στις διαφάνειες.

#### Βήμα 1: Δημιουργία αρχικού στιγμιότυπου παρουσίασης

Δημιουργήστε ένα αντικείμενο παρουσίασης που αντιπροσωπεύει το αρχείο PPTX σας:

```python
import aspose.slides as slides

# Αρχικοποίηση της παρουσίασης
total_presentation = slides.Presentation()
try:
    # Ο κώδικας για τον χειρισμό της διαφάνειας θα τοποθετηθεί εδώ
finally:
    total_presentation.dispose()
```

#### Βήμα 2: Αποκτήστε την πρώτη διαφάνεια

Αποκτήστε πρόσβαση στην πρώτη διαφάνεια της παρουσίασης:

```python
# Πρόσβαση στην πρώτη διαφάνεια
slide = total_presentation.slides[0]
```

#### Βήμα 3: Φόρτωση εικόνας από τον κατάλογο εγγράφων

Φορτώστε το αρχείο εικόνας που επιθυμείτε στην παρουσίαση. Αντικαταστήστε `'YOUR_DOCUMENT_DIRECTORY/'` με την πραγματική διαδρομή προς τις εικόνες σας.

```python
# Φόρτωση εικόνας
image_to_add = slides.Images.from_file('YOUR_DOCUMENT_DIRECTORY/image1.jpg')
```

#### Βήμα 4: Προσθήκη φορτωμένης εικόνας στη συλλογή εικόνων της παρουσίασης

Προσθέστε την εικόνα που φορτώθηκε στη συλλογή εικόνων που διαχειρίζεται η παρουσίαση:

```python
# Προσθήκη εικόνας στη συλλογή εικόνων της παρουσίασης
image_in_presentation = total_presentation.images.add_image(image_to_add)
```

#### Βήμα 5: Προσθήκη κορνίζας εικόνας στη διαφάνεια

Τώρα, προσθέστε μια κορνίζα με συγκεκριμένες διαστάσεις και τοποθετήστε την στην επιθυμητή θέση μέσα στη διαφάνεια:

```python
# Προσθήκη κορνίζας εικόνας στη διαφάνεια
drawable_shape = slide.shapes.add_picture_frame(
    slides.ShapeType.RECTANGLE,  # Τύπος σχήματος για ορθογώνιο
    50,                          # Συντεταγμένη Χ της επάνω αριστερής γωνίας
    150,                         # Συντεταγμένη Y της επάνω αριστερής γωνίας
    image_in_presentation.width, # Πλάτος της εικόνας
    image_in_presentation.height,# Ύψος της εικόνας
    image_in_presentation        # Αντικείμενο εικόνας που θα προστεθεί
)
```

#### Βήμα 6: Αποθήκευση της παρουσίασης

Τέλος, αποθηκεύστε την παρουσίασή σας με τη νέα κορνίζα:

```python
# Αποθήκευση της ενημερωμένης παρουσίασης
total_presentation.save('YOUR_OUTPUT_DIRECTORY/shapes_add_stretch_offset_out.pptx', slides.export.SaveFormat.PPTX)
```

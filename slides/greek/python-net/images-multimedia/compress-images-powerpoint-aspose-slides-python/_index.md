## Οδηγός Εφαρμογής
### Πρόσβαση και τροποποίηση διαφανειών
#### Επισκόπηση
Για να συμπιέσετε μια εικόνα μέσα σε μια παρουσίαση, πρέπει πρώτα να αποκτήσετε πρόσβαση στη συγκεκριμένη διαφάνεια και στο πλαίσιο της εικόνας. Δείτε πώς μπορείτε να το πετύχετε αυτό χρησιμοποιώντας το Aspose.Slides:
#### Βήμα προς βήμα εφαρμογή
**1. Φορτώστε την παρουσίαση:**
```python
import aspose.slides as slides
import os

document_path = "YOUR_DOCUMENT_DIRECTORY/CroppedImage.pptx"
output_path = "YOUR_OUTPUT_DIRECTORY/CroppedImage-Compress-out.pptx"

with slides.Presentation(document_path) as presentation:
```
*Εξήγηση*Χρησιμοποιήστε έναν διαχειριστή περιβάλλοντος για να ανοίξετε το αρχείο PowerPoint, διασφαλίζοντας ότι κλείνει σωστά μετά την επεξεργασία.
**2. Πρόσβαση στην Πρώτη Διαφάνεια:**
```python
    slide = presentation.slides[0]
```
*Εξήγηση*: Αυτό ανακτά την πρώτη διαφάνεια στην παρουσίασή σας.
**3. Αποκτήστε το Πλαίσιο Εικόνας:**
```python
    picture_frame = slide.shapes[0]  # Υποθέτει ότι το πρώτο σχήμα είναι ένα PictureFrame
```
*Εξήγηση*Υποθέτουμε ότι το πρώτο σχήμα στη διαφάνεια είναι ένα πλαίσιο εικόνας (PictureFrame). Προσαρμόστε το εάν χρειάζεται, ανάλογα με τη συγκεκριμένη περίπτωση χρήσης σας.
**4. Συμπίεση της εικόνας:**
```python
    compression_result = picture_frame.picture_format.compress_image(True, 150)
```
*Εξήγηση*: Το `compress_image` Η μέθοδος μειώνει την ανάλυση της εικόνας στα 150 DPI, κατάλληλη για χρήση στο διαδίκτυο, διατηρώντας παράλληλα διαχειρίσιμα τα μεγέθη των αρχείων.
**5. Αποθηκεύστε την παρουσίαση:**
```python
    presentation.save(output_path, slides.export.SaveFormat.PPTX)

# Εμφάνιση μεγεθών της πηγής και των παρουσιάσεων που προκύπτουν για σύγκριση
original_size = os.stat(document_path).st_size
compressed_size = os.stat(output_path).st_size
print("Source presentation size:", original_size)  # Σε bytes
print("Compressed presentation size:", compressed_size)  # Σε bytes
```
*Εξήγηση*Η παρουσίαση αποθηκεύεται με τη νέα, συμπιεσμένη εικόνα. Επίσης, εκτυπώνουμε τα μεγέθη αρχείων για να παρουσιάσουμε τη σμίκρυνση που επιτεύχθηκε.

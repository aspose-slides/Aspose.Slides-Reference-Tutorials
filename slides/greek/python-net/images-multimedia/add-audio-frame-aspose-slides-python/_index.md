## Οδηγός Εφαρμογής

Αυτή η ενότητα σάς καθοδηγεί στην προσθήκη ενός ηχητικού καρέ σε μια παρουσίαση PowerPoint.

### Προσθήκη ήχου σε μια παρουσίαση

**Επισκόπηση:**
Προσθέστε ένα αρχείο ήχου στην πρώτη διαφάνεια της παρουσίασής σας. Αυτό περιλαμβάνει τη φόρτωση του ήχου, την ενσωμάτωσή του ως ηχητικό πλαίσιο σε μια διαφάνεια και την αποθήκευση της ενημερωμένης παρουσίασης.

#### Βήμα 1: Ρύθμιση διαδρομών αρχείων
Ορίστε διαδρομές για το αρχείο ήχου εισόδου και την παρουσίαση εξόδου:
```python
input_audio_path = 'YOUR_DOCUMENT_DIRECTORY/audio.m4a'
output_presentation_path = 'YOUR_OUTPUT_DIRECTORY/AudioFrameValue_out.pptx'
```
Αντικαθιστώ `YOUR_DOCUMENT_DIRECTORY` με τον κατάλογο που περιέχει το αρχείο ήχου σας, και `YOUR_OUTPUT_DIRECTORY` με το σημείο όπου θέλετε να αποθηκεύσετε την παρουσίαση.

#### Βήμα 2: Δημιουργία μιας παρουσίας παρουσίασης
Χρησιμοποιήστε έναν διαχειριστή περιβάλλοντος για σωστή διαχείριση πόρων:
```python
with slides.Presentation() as pres:
    # Περαιτέρω βήματα θα εκτελεστούν εντός αυτού του μπλοκ.
```

#### Βήμα 3: Φόρτωση και προσθήκη ήχου
Ανοίξτε το αρχείο ήχου σας σε λειτουργία δυαδικής ανάγνωσης και, στη συνέχεια, προσθέστε το στη συλλογή ηχητικών αποσπασμάτων της παρουσίασης:
```python
with open(input_audio_path, "rb") as in_file:
    audio = pres.audios.add_audio(in_file)
```
Ο `add_audio` Η συνάρτηση προσθέτει το αρχείο ήχου σας στην εσωτερική συλλογή για ενσωμάτωση σε διαφάνειες.

#### Βήμα 4: Ενσωμάτωση ηχητικού πλαισίου σε διαφάνεια
Ενσωματώστε το ηχητικό πλαίσιο στην πρώτη διαφάνεια σε μια καθορισμένη θέση με καθορισμένες διαστάσεις:
```python
audio_frame = pres.slides[0].shapes.add_audio_frame_embedded(50, 50, 100, 100, audio)
```
Οι παράμετροι `(50, 50, 100, 100)` Καθορίστε τη θέση x, τη θέση y, το πλάτος και το ύψος του ηχητικού καρέ.

### Αποθήκευση της παρουσίασης
Η παρουσίαση αποθηκεύεται αυτόματα όταν κλείσετε την `with` μπλοκ. Βεβαιωθείτε ότι η διαδρομή εξόδου έχει καθοριστεί σωστά για να αποτρέψετε την αντικατάσταση ή την απώλεια αρχείων.

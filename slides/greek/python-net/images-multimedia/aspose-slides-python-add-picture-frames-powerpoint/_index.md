## Οδηγός Εφαρμογής

Ας αναλύσουμε τα βήματα για την προσθήκη και τη μορφοποίηση πλαισίων εικόνων.

### Βήμα 1: Δημιουργία νέας παρουσίασης (H3)

Ξεκινήστε αρχικοποιώντας ένα νέο αντικείμενο παρουσίασης PowerPoint. Αυτό λειτουργεί ως καμβάς για όλες τις τροποποιήσεις.

```python
with slides.Presentation() as pres:
    # Η μεταβλητή 'pres' αντιπροσωπεύει τώρα την παρουσίασή μας.
```

**Σκοπός**: Καθορίζει τη βάση για την προσθήκη διαφανειών και περιεχομένου.

### Βήμα 2: Πρόσβαση στην πρώτη διαφάνεια (H3)

Αποκτήστε πρόσβαση στην πρώτη διαφάνεια για να προσθέσετε το πλαίσιο εικόνας σας. Στο PowerPoint, κάθε παρουσίαση ξεκινά με μία μόνο διαφάνεια από προεπιλογή.

```python
slide = pres.slides[0]
# Ο όρος «διαφάνεια» αναφέρεται πλέον στην πρώτη διαφάνεια της παρουσίασής μας.
```

**Σκοπός**: Μας επιτρέπει να στοχεύουμε και να τροποποιούμε συγκεκριμένες διαφάνειες μέσα στην παρουσίαση.

### Βήμα 3: Φόρτωση εικόνας (H3)

Φορτώστε την εικόνα που επιλέξατε από τον κατάλογό της. Αυτή η εικόνα θα χρησιμοποιηθεί ως κορνίζα.

```python
img_path = "YOUR_DOCUMENT_DIRECTORY/image1.jpg"
with open(img_path, 'rb') as img_file:
    imgx = pres.images.add_image(drawing.Image.load(img_file))
# Το 'imgx' είναι πλέον το αντικείμενο εικόνας που έχει φορτωθεί και προστέθηκε στην παρουσίαση.
```

**Σκοπός**: Προετοιμάζει την εικόνα για εισαγωγή σε μια διαφάνεια.

### Βήμα 4: Προσθήκη κορνίζας εικόνας (H3)

Τοποθετήστε το πλαίσιο εικόνας χρησιμοποιώντας την εικόνα που έχετε φορτώσει στη διαφάνεια-στόχο σας. Καθορίστε τη θέση και το μέγεθός του εδώ.

```python
cf = slide.shapes.add_picture_frame(
    slides.ShapeType.RECTANGLE, 50, 150, imgx.width, imgx.height, imgx)
# Το 'cf' αντιπροσωπεύει την πρόσφατα προστιθέμενη κορνίζα.
```

**Επεξήγηση παραμέτρων**: 
- `ShapeType.RECTANGLE`: Ορίζει το σχήμα του πλαισίου.
- `(50, 150)`: Συντεταγμένες X και Y για τη θέση στη διαφάνεια.
- `imgx.width`, `imgx.height`: Διαστάσεις της εικόνας.

### Βήμα 5: Εφαρμογή μορφοποίησης (H3)

Προσαρμόστε την κορνίζα σας με χρώμα περιγράμματος, πάχος γραμμής και γωνία περιστροφής για να βελτιώσετε την εμφάνισή της.

```python
cf.line_format.fill_format.fill_type = slides.FillType.SOLID
cf.line_format.fill_format.solid_fill_color.color = drawing.Color.blue
cf.line_format.width = 20
cf.rotation = 45
# Αυτές οι ρυθμίσεις τροποποιούν το στυλ περιγράμματος του πλαισίου.
```

**Επιλογές διαμόρφωσης**: 
- **Τύπος πλήρωσης**: Συμπαγές χρώμα για το περίγραμμα του πλαισίου.
- **Χρώμα**: Προσαρμόσιμο σε οποιοδήποτε `drawing.Color` αξία.
- **Πλάτος**: Πάχος της γραμμής ορίου.
- **Περιστροφή**: Γωνία της κορνίζας.

### Βήμα 6: Αποθήκευση της παρουσίασής σας (H3)

Τέλος, αποθηκεύστε την παρουσίασή σας με όλες τις τροποποιήσεις που έχετε κάνει. Καθορίστε έναν κατάλογο και ένα όνομα αρχείου για εύκολη πρόσβαση αργότερα.

```python
output_path = "YOUR_OUTPUT_DIRECTORY/shapes_picture_frame_format_out.pptx"
pres.save(output_path, slides.export.SaveFormat.PPTX)
# Η τροποποιημένη παρουσίαση αποθηκεύεται στην καθορισμένη διαδρομή.
```

**Σκοπός**: Διασφαλίζει ότι όλη η εργασία σας διατηρείται σε νέα μορφή αρχείου.

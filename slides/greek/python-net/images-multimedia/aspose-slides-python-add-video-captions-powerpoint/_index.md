# Πώς να προσθέσετε και να αφαιρέσετε λεζάντες βίντεο στο PowerPoint με το Aspose.Slides για Python

## Εισαγωγή

Η προσθήκη λεζάντων στις παρουσιάσεις PowerPoint σας μπορεί να βελτιώσει σημαντικά την προσβασιμότητα, ειδικά για ποικίλα κοινά ή για όσους χρειάζονται υπότιτλους. Με το Aspose.Slides για Python, μπορείτε εύκολα να ενσωματώσετε λεζάντες στο περιεχόμενο βίντεο μέσα σε διαφάνειες PowerPoint. Αυτό το σεμινάριο θα σας καθοδηγήσει στην προσθήκη και αφαίρεση λεζάντων από βίντεο σε παρουσιάσεις PowerPoint χρησιμοποιώντας το Aspose.Slides.

**Τι θα μάθετε:**
- Πώς να προσθέσετε λεζάντες βίντεο από ένα αρχείο VTT.
- Τεχνικές για την εξαγωγή και την αφαίρεση υπαρχόντων λεζάντων.
- Βέλτιστες πρακτικές για τη βελτιστοποίηση της απόδοσης με το Aspose.Slides.

Ας στήσουμε το περιβάλλον σας και ας ξεκινήσουμε!

Πριν ξεκινήσετε, βεβαιωθείτε ότι έχετε τα εξής:
- **Περιβάλλον Python**Python 3.6 ή νεότερη έκδοση εγκατεστημένη στο σύστημά σας.
- **Aspose.Slides για Python**Εγκατάσταση μέσω pip όπως φαίνεται παρακάτω.
- **Αρχεία VTT**Προετοιμάστε ένα αρχείο VTT για υπότιτλους και αρχεία βίντεο για δοκιμή.

## Οδηγός Εφαρμογής
Θα εξερευνήσουμε δύο κύριες λειτουργίες: την προσθήκη λεζάντων και την αφαίρεσή τους από βίντεο που είναι ενσωματωμένα σε παρουσιάσεις PowerPoint.

### Προσθήκη λεζάντων σε ένα καρέ βίντεο
Αυτή η λειτουργία σάς επιτρέπει να βελτιώσετε την προσβασιμότητα του περιεχομένου βίντεο συμπεριλαμβάνοντας υπότιτλους απευθείας μέσα στην παρουσίασή σας.

#### Βήμα 1: Δημιουργία και φόρτωση μιας παρουσίασης
Ξεκινήστε δημιουργώντας ένα νέο αντικείμενο παρουσίασης:

```python
import aspose.slides as slides

def add_video_captions():
    # Δημιουργία νέας παρουσίασης
    with slides.Presentation() as pres:
        ...
```

#### Βήμα 2: Προσθήκη του αρχείου βίντεο
Φορτώστε το αρχείο βίντεο στην παρουσίαση. Βεβαιωθείτε ότι έχετε τη σωστή διαδρομή προς το βίντεό σας:

```python
        with open("YOUR_DOCUMENT_DIRECTORY/NewVideo.mp4", "rb") as f:
            video = pres.videos.add_video(f.read())
```

#### Βήμα 3: Εισαγωγή καρέ βίντεο και προσθήκη λεζάντων
Εισαγάγετε ένα `VideoFrame` στην επιθυμητή θέση και προσθέστε λεζάντες χρησιμοποιώντας το αρχείο VTT:

```python
        # Προσθήκη VideoFrame με καθορισμένες διαστάσεις
        video_frame = pres.slides[0].shapes.add_video_frame(0, 0, 100, 100, video)
        
        # Επισύναψη κομματιού υποτίτλων από αρχείο VTT
        video_frame.caption_tracks.add("New track", "YOUR_DOCUMENT_DIRECTORY/bunny.vtt")
```

#### Βήμα 4: Αποθήκευση της παρουσίασης
Τέλος, αποθηκεύστε την ενημερωμένη παρουσίασή σας με λεζάντες:

```python
        pres.save("YOUR_OUTPUT_DIRECTORY/VideoCaptionsAdd_out.pptx", slides.export.SaveFormat.PPTX)
```

### Εξαγωγή και αφαίρεση υποτίτλων από ένα καρέ βίντεο
Τώρα που προσθέσατε υπότιτλους, ας εξερευνήσουμε πώς να τους εξαγάγετε για έλεγχο ή να τους καταργήσετε εντελώς.

#### Βήμα 1: Άνοιγμα μιας υπάρχουσας παρουσίασης
Ξεκινήστε φορτώνοντας την παρουσίαση που περιέχει το βίντεό σας με λεζάντες:

```python
def extract_and_remove_captions():
    # Φόρτωση της υπάρχουσας παρουσίασης
    with slides.Presentation("YOUR_OUTPUT_DIRECTORY/VideoCaptionsAdd_out.pptx") as pres:
        ...
```

#### Βήμα 2: Εξαγωγή δεδομένων υποτίτλων
Επαναλάβετε κάθε κομμάτι υποτίτλων για να αποθηκεύσετε τα δεδομένα του σε αρχεία VTT:

```python
        video_frame = pres.slides[0].shapes[0]
        if video_frame is not None:
            for idx, caption_track in enumerate(video_frame.caption_tracks):
                with open(f"YOUR_OUTPUT_DIRECTORY/VideoCaption_out_{idx}.vtt", "wb") as f:
                    f.write(caption_track.binary_data)
```

#### Βήμα 3: Αφαίρεση λεζάντων
Διαγραφή όλων των υπότιτλων από το καρέ του βίντεο:

```python
            # Διαγραφή όλων των κομματιών υπότιτλων
            video_frame.caption_tracks.clear()
            
            # Αποθήκευση αλλαγών σε νέο αρχείο
            pres.save("YOUR_OUTPUT_DIRECTORY/VideoCaptionsRemove_out.pptx", slides.export.SaveFormat.PPTX)
```

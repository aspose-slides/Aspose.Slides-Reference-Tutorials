## Οδηγός Εφαρμογής
Τώρα που έχετε ρυθμίσει το Aspose.Slides για Python, ας εξερευνήσουμε πώς να προσθέσετε ένα καρέ βίντεο στη διαφάνεια του PowerPoint.

### Προσθήκη καρέ βίντεο

#### Επισκόπηση
Θα δείξουμε πώς μπορείτε να προσθέσετε ένα καρέ βίντεο στην πρώτη διαφάνεια μιας παρουσίασης. Αυτή η λειτουργία είναι χρήσιμη όταν θέλετε να συμπεριλάβετε περιεχόμενο πολυμέσων απευθείας στις διαφάνειές σας.

#### Βήμα προς βήμα εφαρμογή:
##### Πρόσβαση στην πρώτη διαφάνεια
```python
class PresentationManager:
    def __init__(self):
        self.presentation = slides.Presentation()

    def access_first_slide(self):
        # Πρόσβαση στην πρώτη διαφάνεια από τη συλλογή
        return self.presentation.slides[0]
```
*Γιατί;*Αυτό το βήμα διασφαλίζει ότι εργάζεστε με τη σωστή διαφάνεια στην οποία σκοπεύετε να προσθέσετε το βίντεό σας.

##### Προσθήκη καρέ βίντεο
```python
class PresentationManager:
    def __init__(self):
        self.presentation = slides.Presentation()

    def access_first_slide(self):
        return self.presentation.slides[0]

    def add_video_frame(self, slide, video_path):
        # Προσθήκη καρέ βίντεο στη διαφάνεια σε καθορισμένη θέση και μέγεθος
        vf = slide.shapes.add_video_frame(50, 150, 300, 150, video_path)
        return vf
```
*Εξήγηση*: Αυτή η γραμμή εισάγει ένα καρέ βίντεο στη διαφάνειά σας. Οι παράμετροι `50`, `150`, `300`, `150` ορίστε τις συντεταγμένες X, Y και το πλάτος και ύψος του καρέ βίντεο αντίστοιχα.

##### Ρύθμιση παραμέτρων αναπαραγωγής βίντεο
```python
class PresentationManager:
    def __init__(self):
        self.presentation = slides.Presentation()

    def access_first_slide(self):
        return self.presentation.slides[0]

    def add_video_frame(self, slide, video_path):
        vf = slide.shapes.add_video_frame(50, 150, 300, 150, video_path)
        # Ρυθμίστε τη λειτουργία αναπαραγωγής βίντεο ώστε να ξεκινά αυτόματα όταν εμφανίζεται η διαφάνεια
        vf.play_mode = slides.VideoPlayModePreset.AUTO
        # Ρύθμιση της έντασης του βίντεο
        vf.volume = slides.AudioVolumeMode.LOUD
        return vf
```
*Σκοπός*Αυτές οι διαμορφώσεις διασφαλίζουν ότι το κοινό σας θα ακούσει και θα δει το βίντεο αμέσως μόλις φτάσει στη διαφάνεια.

##### Αποθήκευση της παρουσίασης
```python
class PresentationManager:
    def __init__(self):
        self.presentation = slides.Presentation()

    def access_first_slide(self):
        return self.presentation.slides[0]

    def add_video_frame(self, slide, video_path):
        vf = slide.shapes.add_video_frame(50, 150, 300, 150, video_path)
        vf.play_mode = slides.VideoPlayModePreset.AUTO
        vf.volume = slides.AudioVolumeMode.LOUD
        return vf

    def save_presentation(self, output_directory):
        # Αποθήκευση της παρουσίασης με νέο όνομα στον καθορισμένο κατάλογο εξόδου
        self.presentation.save(f"{output_directory}/shapes_add_video_out.pptx")
```
*Γιατί;*Αυτό το βήμα ολοκληρώνει τις αλλαγές σας αποθηκεύοντάς τες σε ένα αρχείο, διασφαλίζοντας ότι η εργασία σας δεν θα χαθεί και θα μπορεί να κοινοποιηθεί ή να παρουσιαστεί.

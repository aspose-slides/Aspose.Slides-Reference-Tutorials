### Διαγραφή περικομμένων περιοχών
#### Επισκόπηση
Αφαιρέστε αποτελεσματικά τα ανεπιθύμητα κομμένα τμήματα μέσα σε ένα PictureFrame σε μια διαφάνεια με αυτήν τη λειτουργία.
##### Βήμα 1: Ορίστε τις διαδρομές αρχείων σας
Ορίστε διαδρομές για παρουσιάσεις πηγής και εξόδου:
```python
presentation_name = "YOUR_DOCUMENT_DIRECTORY/CroppedImage.pptx"
out_file_path = "YOUR_OUTPUT_DIRECTORY/CroppedImage-out.pptx"
```
##### Βήμα 2: Ανοίξτε την παρουσίαση
Φορτώστε την παρουσίασή σας χρησιμοποιώντας έναν διαχειριστή περιβάλλοντος για αποτελεσματικό χειρισμό πόρων:
```python
with slides.Presentation(presentation_name) as pres:
    # Πρόσβαση στην πρώτη διαφάνεια της παρουσίασης
    slide = pres.slides[0]
    
    # Υποθέστε ότι το πρώτο σχήμα είναι ένα PictureFrame
    pic_frame = slide.shapes[0]
```
##### Βήμα 3: Διαγραφή Περικοπμένων Περιοχών
Χρήση `delete_picture_cropped_areas` για να αφαιρέσετε κομμένα μέρη:
```python
# Αφαίρεση κομμένων τμημάτων από την εικόνα μέσα στο PictureFrame
cropped_image = pic_frame.picture_format.delete_picture_cropped_areas()
```
##### Βήμα 4: Αποθήκευση της παρουσίασης
Αποθηκεύστε την τροποποιημένη παρουσίασή σας:
```python
pres.save(out_file_path, slides.export.SaveFormat.PPTX)
```

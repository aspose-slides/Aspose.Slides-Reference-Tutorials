Αυτή η λειτουργία σάς επιτρέπει να διατηρείτε ενσωματωμένα αντικείμενα στα αρχεία PowerPoint σας κατά τη μετατροπή τους σε PDF, διασφαλίζοντας ότι δεν θα χάσετε πληροφορίες ή λειτουργίες.

#### Βήμα 1: Φόρτωση της παρουσίασής σας

Φορτώστε την παρουσίαση που περιέχει αντικείμενα OLE χρησιμοποιώντας το Aspose.Slides.

```python
document_directory = 'YOUR_DOCUMENT_DIRECTORY/'
output_directory = 'YOUR_OUTPUT_DIRECTORY/'

with slides.Presentation(document_directory + "PresOleExample.pptx") as pres:
    # Συνεχίστε με τη δημιουργία επιλογών εξαγωγής PDF
```

#### Βήμα 2: Δημιουργία επιλογών εξαγωγής PDF

Εδώ, ορίζουμε τις ρυθμίσεις για την εξαγωγή της παρουσίασής σας.

```python
options = slides.export.PdfOptions()
options.include_ole_data = True  # Αυτό διασφαλίζει ότι τα δεδομένα OLE διατηρούνται στο PDF
```

#### Βήμα 3: Αποθήκευση ως PDF

Αποθηκεύστε την παρουσίαση με τις καθορισμένες επιλογές για να δημιουργήσετε ένα αρχείο PDF που διατηρεί όλα τα ενσωματωμένα αντικείμενα.

```python

```

### Αναγνώριση πλαισίων αντικειμένων OLE

Προσδιορίστε αν ένα σχήμα είναι `OleObjectFrame`, υποδεικνύοντας ότι περιέχει ενσωματωμένα δεδομένα:

```python
count = 0
for slide in pres.slides:
    for shape in slide.shapes:
        if isinstance(shape, slides.OleObjectFrame):
            # Αυτό το σχήμα περιέχει ένα αντικείμενο OLE με ενσωματωμένα δεδομένα
```

### Εξαγωγή δεδομένων ενσωματωμένου αρχείου

Αφού εντοπίσετε τα αντικείμενα OLE, εξαγάγετε τα δεδομένα τους και αποθηκεύστε τα χρησιμοποιώντας ένα μοναδικό όνομα αρχείου:

```python
count = 0
for slide in pres.slides:
    for shape in slide.shapes:
        if isinstance(shape, slides.OleObjectFrame):
            count += 1
            
            # Εξαγωγή δεδομένων αρχείου και επέκτασης
            data = shape.embedded_data.embedded_file_data
            extension = shape.embedded_data.embedded_file_extension
            
            # Δημιουργήστε ένα όνομα αρχείου με βάση τον αριθμό αντικειμένου
            file_name = f"shapes_ole_objects{count}_out.{extension}"
            
            # Εγγραφή στον κατάλογο εξόδου
            with open(f"YOUR_OUTPUT_DIRECTORY/{file_name}", "wb") as file:
                file.write(data)
```

### Παράμετροι και τιμές επιστροφής

- **προεπισκοπήσεις**: Επαναλαμβάνει όλες τις διαφάνειες της παρουσίασης.
- **shape.embedded_data.embedded_file_data**: Περιέχει ακατέργαστα δεδομένα του ενσωματωμένου αρχείου.
- **shape.embedded_data.embedded_file_extension**: Χρησιμοποιείται για σκοπούς ονοματοδοσίας.

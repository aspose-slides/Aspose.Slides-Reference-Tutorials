## Οδηγός Εφαρμογής

### Προσθήκη και τροποποίηση γραφικών SmartArt

#### Επισκόπηση
Σε αυτήν την ενότητα, θα μάθουμε πώς να προσθέσετε ένα γραφικό SmartArt στη διαφάνειά σας και να τροποποιήσετε τις ιδιότητές του, όπως αντιστρέφοντας την κατάστασή του.

#### Βήμα προς βήμα εφαρμογή:

**1. Δημιουργήστε μια νέα παρουσίαση:**

```python
with slides.Presentation() as presentation:
    # Πρόσβαση στην πρώτη διαφάνεια (ευρετήριο 0)
slide = presentation.slides[0]
```

Αυτό το βήμα αρχικοποιεί ένα νέο αντικείμενο παρουσίασης και το ανοίγει για επεξεργασία χρησιμοποιώντας τεχνικές διαχείρισης πόρων.

**2. Προσθήκη γραφικού SmartArt:**

```python
# Προσθήκη γραφικού SmartArt με καθορισμένες διαστάσεις και τύπο διάταξης
smart = slide.shapes.add_smart_art(
    x=10, y=10, width=400, height=300,
    layout_type=slides.smartart.SmartArtLayoutType.BASIC_PROCESS
)
```

Εδώ, προσθέτουμε μια βασική διεργασία SmartArt στις δεδομένες συντεταγμένες. `add_smart_art` Η μέθοδος επιτρέπει την ακριβή τοποθέτηση και τη διαμόρφωση μεγέθους.

**3. Τροποποιήστε την κατάσταση αντιστροφής:**

```python
# Ορισμός αντιστροφής του γραφικού SmartArt
smart.is_reversed = True
```

Αυτή η γραμμή αλλάζει τον προσανατολισμό του SmartArt, προσθέτοντας ένα δυναμικό οπτικό εφέ.

**4. Αποθηκεύστε την παρουσίαση:**

```python
presentation.save("YOUR_OUTPUT_DIRECTORY/smart_art_change_state_out.pptx")
```

Τέλος, αποθηκεύστε την παρουσίασή σας σε έναν καθορισμένο κατάλογο. Βεβαιωθείτε ότι έχετε αντικαταστήσει `YOUR_OUTPUT_DIRECTORY` με μια πραγματική διαδρομή στο σύστημά σας.

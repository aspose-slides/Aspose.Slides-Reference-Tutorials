## Οδηγός Εφαρμογής

### Δημιουργία SmartArt με συμπλήρωση κουκκίδων εικόνας

#### Επισκόπηση

Σε αυτήν την ενότητα, θα μάθετε πώς να προσθέσετε ένα σχήμα SmartArt σε μια διαφάνεια και να ορίσετε μια εικόνα ως μορφή γεμίσματος κουκκίδων.

#### Βήμα 1: Δημιουργία αντικειμένου παρουσίασης

Ξεκινήστε δημιουργώντας ένα αντικείμενο παρουσίασης. Αυτός θα είναι ο καμβάς σας:

```python
with slides.Presentation() as document:
    # Ο κώδικας για την προσθήκη SmartArt βρίσκεται εδώ.
```

#### Βήμα 2: Προσθήκη σχήματος SmartArt

Προσθέστε ένα σχήμα SmartArt στην πρώτη σας διαφάνεια στην επιθυμητή θέση και μέγεθος:

```python
smart = document.slides[0].shapes.add_smart_art(
    10, 10, 500, 400,
    slides.smartart.SmartArtLayoutType.VERTICAL_PICTURE_LIST
)
```

#### Βήμα 3: Πρόσβαση στον πρώτο κόμβο

Αποκτήστε πρόσβαση στον πρώτο κόμβο για να εφαρμόσετε μορφοποίηση εικόνας κουκκίδων:

```python
node = smart.all_nodes[0]
```

#### Βήμα 4: Ορισμός μορφής γεμίσματος με κουκκίδες

Ελέγξτε αν υπάρχει μορφή συμπλήρωσης κουκκίδων και ορίστε μια εικόνα ως κουκκίδα:

```python
if node.bullet_fill_format is not None:
    img = slides.Images.from_file("YOUR_DOCUMENT_DIRECTORY/image1.jpg")
    image = document.images.add_image(img)

    node.bullet_fill_format.fill_type = slides.FillType.PICTURE
    node.bullet_fill_format.picture_fill_format.picture.image = image
    node.bullet_fill_format.picture_fill_format.picture_fill_mode = slides.PictureFillMode.STRETCH
```

#### Βήμα 5: Αποθήκευση της παρουσίασης

Τέλος, αποθηκεύστε την παρουσίασή σας με τις αλλαγές:

```python
document.save("YOUR_OUTPUT_DIRECTORY/smart_art_bullet_fill_format_out.pptx", slides.export.SaveFormat.PPTX)
```

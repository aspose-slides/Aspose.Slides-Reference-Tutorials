#### Βήμα 1: Δημιουργία στιγμιαίας παρουσίασης του αντικειμένου παρουσίασης
```python
import aspose.slides as slides

def create_smart_art_shape():
    # Δημιουργήστε ένα νέο αντικείμενο παρουσίασης
    with slides.Presentation() as pres:
        pass  # Θα προσθέσουμε περισσότερο κώδικα εδώ αργότερα
```

```python
    slide = pres.slides[0]  # Πρόσβαση στην πρώτη διαφάνεια
```
- **Εξήγηση**: Αποκτήστε πρόσβαση στην πρώτη διαφάνεια για να προσθέσετε SmartArt.
#### Βήμα 3: Προσθήκη σχήματος SmartArt
```python
        smart = slide.shapes.add_smart_art(
            0, 0, 400, 400, slides.SmartArtLayoutType.BASIC_BLOCK_LIST
```

## Οδηγός Εφαρμογής

Ας αναλύσουμε τώρα τη διαδικασία προσθήκης και προσαρμογής του SmartArt στο PowerPoint χρησιμοποιώντας το Aspose.Slides για Python.

### Προσθήκη γραφικού SmartArt

#### Επισκόπηση
Δημιουργήστε μια νέα διαφάνεια και προσθέστε ένα γραφικό SmartArt τύπου οργανογράμματος σε αυτήν:

```python
import aspose.slides as slides

# Δημιουργήστε μια παρουσία παρουσίασης\με slides.Presentation() ως παρουσίαση:
    # Προσθήκη SmartArt με καθορισμένες διαστάσεις στη θέση (10, 10)
    smart = presentation.slides[0].shapes.add_smart_art(
        x=10,
        y=10,
        width=400,
        height=300,
        layout_type=slides.smartart.SmartArtLayoutType.ORGANIZATION_CHART
    )
```

#### Παράμετροι και Σκοπός της Μεθόδου
- **x, y**: Θέση του γραφικού SmartArt στη διαφάνεια.
- **πλάτος, ύψος**Διαστάσεις για σωστή ορατότητα.
- **τύπος_διάταξης**Καθορίζει τον τύπο της διάταξης SmartArt, σε αυτήν την περίπτωση, ένα οργανόγραμμα.

### Προσαρμογή της διάταξης του οργανογράμματος

#### Επισκόπηση
Προσαρμόστε τον πρώτο κόμβο στο γραφικό SmartArt μας ορίζοντας τη διάταξή του σε LEFT_HANGING:

```python
# Ορισμός του πρώτου κόμβου σε διάταξη που κρέμεται αριστερά
smart.nodes[0].organization_chart_layout = slides.smartart.OrganizationChartLayoutType.LEFT_HANGING
```

# Πώς να αλλάξετε τις διατάξεις SmartArt στο PowerPoint χρησιμοποιώντας Python και Aspose.Slides

## Εισαγωγή

Βελτιώστε τις παρουσιάσεις PowerPoint τροποποιώντας τη διάταξη των γραφικών SmartArt με Python και Aspose.Slides. Αυτό το σεμινάριο θα σας καθοδηγήσει στην αλλαγή του σχεδιασμού ενός γραφικού SmartArt από «Βασική λίστα μπλοκ» σε «Βασική διαδικασία», βελτιώνοντας τόσο την οπτική εμφάνιση όσο και τη σαφήνεια.

**Τι θα μάθετε:**
- Εγκατάσταση και ρύθμιση του Aspose.Slides για Python
- Δημιουργία νέων παρουσιάσεων PowerPoint με Python
- Προσθήκη και τροποποίηση γραφικών SmartArt σε διαφάνειες
- Αποθήκευση της ενημερωμένης παρουσίασης

## Οδηγός Εφαρμογής

Ακολουθήστε αυτά τα βήματα για να αλλάξετε μια διάταξη SmartArt στο PowerPoint χρησιμοποιώντας Python.

### Δημιουργία και τροποποίηση διατάξεων SmartArt

#### Επισκόπηση:
Προσθέστε μέσω προγραμματισμού ένα γραφικό SmartArt στη διαφάνειά σας και αλλάξτε τον τύπο διάταξής του.

#### Βήμα 1: Αρχικοποίηση παρουσίασης
Δημιουργήστε ένα αντικείμενο παρουσίασης, εξασφαλίζοντας αποτελεσματικό χειρισμό πόρων με διαχείριση περιβάλλοντος:

```python
with slides.Presentation() as presentation:
    # Αποκτήστε πρόσβαση στην πρώτη διαφάνεια της παρουσίασης.
slide = presentation.slides[0]
```

#### Βήμα 2: Προσθήκη γραφικού SmartArt
Προσθέστε ένα γραφικό SmartArt 'BasicBlockList' σε μια καθορισμένη θέση και μέγεθος χρησιμοποιώντας:

```python
smart_art = slide.shapes.add_smart_art(
    10, 
    10, 
    400, 
    300,
    slides.smartart.SmartArtLayoutType.BASIC_BLOCK_LIST
)
```

Οι παράμετροι καθορίζουν τη θέση x και y, το πλάτος, το ύψος και τον αρχικό τύπο διάταξης.

#### Βήμα 3: Αλλαγή διάταξης SmartArt
Τροποποιήστε τη διάταξη σε 'BasicProcess':

```python
smart_art.layout = slides.smartart.SmartArtLayoutType.BASIC_PROCESS
```

Αυτό ενημερώνει τη σχεδίαση του γραφικού SmartArt σας για καλύτερη οπτική αναπαράσταση των διαδοχικών βημάτων.

#### Βήμα 4: Αποθήκευση παρουσίασης
Αποθήκευση της τροποποιημένης παρουσίασης:

```python
output_path = 'YOUR_OUTPUT_DIRECTORY/smart_art_change_layout_out.pptx'
presentation.save(output_path, slides.export.SaveFormat.PPTX)
```

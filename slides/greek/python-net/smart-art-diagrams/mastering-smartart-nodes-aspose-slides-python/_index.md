## Οδηγός Εφαρμογής

### Πρόσβαση σε θυγατρικούς κόμβους στο SmartArt

Μάθετε πώς να αποκτάτε πρόσβαση και να επαναλαμβάνετε μέσω θυγατρικών κόμβων μέσα σε ένα αντικείμενο SmartArt χρησιμοποιώντας το Aspose.Slides για Python.

#### Επισκόπηση
Η πρόσβαση σε κόμβους SmartArt επιτρέπει την άμεση εξαγωγή ή τροποποίηση δεδομένων, διευκολύνοντας την βαθύτερη προσαρμογή της παρουσίασης. Ακολουθήστε τα παρακάτω βήματα:

#### Βήμα προς βήμα εφαρμογή:
**1. Φορτώστε την παρουσίασή σας**
Ξεκινήστε φορτώνοντας το αρχείο PowerPoint που περιέχει SmartArt.
```python
def access_child_nodes():
    with slides.Presentation("YOUR_DOCUMENT_DIRECTORY/smart_art_access_child_nodes.pptx") as pres:
```

**2. Επανάληψη μέσω σχημάτων**
Περιηγηθείτε σε κάθε σχήμα στην πρώτη διαφάνεια για να αναγνωρίσετε αντικείμενα SmartArt.
```python
        for shape in pres.slides[0].shapes:
            if isinstance(shape, slides.SmartArt):
```

**3. Πρόσβαση σε θυγατρικούς κόμβους**
Για κάθε αντικείμενο SmartArt, επαναλάβετε τους κόμβους και τους θυγατρικούς κόμβους του, εκτυπώνοντας σχετικές πληροφορίες.
```python
                for node0 in shape.all_nodes:
                    for node in node0.child_nodes:
                        print(f"Text = {node.text_frame.text}, Level = {node.level}, Position = {node.position}")
```

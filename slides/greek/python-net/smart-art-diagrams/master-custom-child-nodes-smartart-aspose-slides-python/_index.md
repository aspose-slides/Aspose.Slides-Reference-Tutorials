## Οδηγός Εφαρμογής
Τώρα που είστε έτοιμοι, ας εξερευνήσουμε τις βασικές λειτουργίες χειρισμού θυγατρικών κόμβων SmartArt.

### Προσθήκη και τοποθέτηση ενός σχήματος SmartArt
**Επισκόπηση:**
Θα ξεκινήσουμε προσθέτοντας ένα οργανόγραμμα στην πρώτη σας διαφάνεια και τοποθετώντας το σωστά.
1. **Φόρτωση παρουσίασης**:
   Ξεκινήστε φορτώνοντας το υπάρχον αρχείο παρουσίασης ή δημιουργώντας ένα νέο, εάν είναι απαραίτητο.

```python
with slides.Presentation("YOUR_DOCUMENT_DIRECTORY/smart_art_access.pptx") as pres:
    # Ο κώδικας συνεχίζεται...
```
2. **Προσθήκη σχήματος SmartArt**:
   Προσθέστε ένα οργανόγραμμα στην πρώτη διαφάνεια σε καθορισμένες συντεταγμένες και μέγεθος:

```python
smart = pres.slides[0].shapes.add_smart_art(
    20, 20, 600, 500, slides.smartart.SmartArtLayoutType.ORGANIZATION_CHART)
```
### Χειρισμός θυγατρικών κόμβων
Στη συνέχεια, θα χειριστούμε διάφορα χαρακτηριστικά των θυγατρικών κόμβων SmartArt.
#### Μετακίνηση ενός σχήματος
**Επισκόπηση:**
Προσαρμόστε τη θέση ενός συγκεκριμένου σχήματος SmartArt τροποποιώντας το `x` και `y` συντεταγμένες.
3. **Μετακίνηση κόμβου**:
   Αποκτήστε πρόσβαση σε έναν κόμβο και προσαρμόστε τη θέση του:

```python
node = smart.all_nodes[1]
shape = node.shapes[1]
shape.x += (shape.width * 2)  # Μετακίνηση δεξιά κατά διπλάσιο πλάτος
shape.y -= (shape.height / 2)  # Μετακινηθείτε προς τα πάνω κατά το μισό ύψος
```
#### Αλλαγή μεγέθους σχήματος
**Επισκόπηση:**
Αυξήστε τόσο το πλάτος όσο και το ύψος συγκεκριμένων σχημάτων SmartArt.
4. **Αλλαγή πλάτους**:
   Προσαρμόστε το πλάτος:

```python
node = smart.all_nodes[2]
shape = node.shapes[1]
shape.width += (shape.width / 2)  # Αύξηση κατά 50%
```
5. **Αλλαγή ύψους**:
   Ομοίως, ρυθμίστε το ύψος:

```python
node = smart.all_nodes[3]
shape = node.shapes[1]
shape.height += (shape.height / 2)  # Αύξηση κατά 50%
```
#### Περιστροφή ενός σχήματος
**Επισκόπηση:**
Περιστρέψτε ένα συγκεκριμένο σχήμα SmartArt για καλύτερο οπτικό προσανατολισμό.
6. **Περιστροφή κόμβου**:
   Περιστρέψτε το σχήμα:

```python
node = smart.all_nodes[4]
shape = node.shapes[1]
shape.rotation = 90  # Περιστροφή κατά 90 μοίρες
```
### Αποθήκευση της παρουσίασης
Τέλος, αποθηκεύστε τις αλλαγές σας σε ένα νέο αρχείο στον κατάλογο εξόδου.
7. **Αποθήκευση αλλαγών**:
   Αποθήκευση της τροποποιημένης παρουσίασης:

```python
pres.save("YOUR_OUTPUT_DIRECTORY/smart_art_custom_child_nodes_out.pptx", slides.export.SaveFormat.PPTX)
```

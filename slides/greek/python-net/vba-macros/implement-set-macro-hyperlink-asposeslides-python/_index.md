### Προσθήκη σχημάτων με υπερσυνδέσμους μακροεντολών

#### Επισκόπηση
Αυτή η ενότητα σάς καθοδηγεί στην προσθήκη ενός σχήματος κουμπιού στη διαφάνεια του PowerPoint και στην αντιστοίχιση ενός συμβάντος κλικ σε υπερσύνδεσμο μακροεντολής, κάτι που είναι απαραίτητο για την αυτοματοποίηση εργασιών μέσα σε παρουσιάσεις.

#### Βήμα προς βήμα εφαρμογή

##### Προσθήκη σχήματος κουμπιού

Αρχικά, θα προσθέσουμε ένα κενό σχήμα κουμπιού στην πρώτη διαφάνεια σε συγκεκριμένες συντεταγμένες:

```python
import aspose.slides as slides

macro_name = "TestMacro"
with slides.Presentation() as presentation:
    # Προσθήκη ενός κενού σχήματος κουμπιού στην πρώτη διαφάνεια
    shape = presentation.slides[0].shapes.add_auto_shape(
        slides.ShapeType.BLANK_BUTTON, 20, 20, 80, 30
    )
```
- **Παράμετροι:**
  - `ShapeType.BLANK_BUTTON`: Καθορίζει ότι προσθέτουμε ένα κενό κουμπί.
  - `(20, 20, 80, 30)`: Οι συντεταγμένες x, y και το πλάτος, ύψος του σχήματος.

##### Ορισμός κλικ σε υπερσύνδεσμο μακροεντολής

Στη συνέχεια, ορίστε την υπερσύνδεση μακροεντολής κάνοντας κλικ στο προστιθέμενο σχήμα:

```python
    # Αντιστοίχιση υπερσυνδέσμου μακροεντολής στο σχήμα
    shape.hyperlink_manager.set_macro_hyperlink_click(macro_name)
```
- **Παράμετροι:**
  - `macro_name`: Το όνομα της μακροεντολής που θα ενεργοποιηθεί όταν κάνετε κλικ στο κουμπί.

## Οδηγός Εφαρμογής

Τώρα που έχετε εγκαταστήσει το Aspose.Slides για Python, ας δούμε πώς να δημιουργούμε μια διαφάνεια που παρουσιάζει το Πυθαγόρειο θεώρημα.

### Βήμα 1: Αρχικοποίηση της παρουσίασης

Ξεκινήστε ρυθμίζοντας το περιβάλλον της παρουσίασής σας χρησιμοποιώντας το `with` Δήλωση για την αποτελεσματική διαχείριση των πόρων:

```python
with slides.Presentation() as pres:
    # Ο κωδικός σας θα μπει εδώ
```

Αυτό διασφαλίζει ότι η παρουσίαση κλείνει σωστά μετά τις λειτουργίες σας, αποτρέποντας διαρροές πόρων.

### Βήμα 2: Προσθήκη ορθογωνίου σχήματος

Στη συνέχεια, προσθέστε ένα Αυτόματο Σχήμα για να διατηρήσετε τη μαθηματική σας έκφραση. Αυτό το σχήμα χρησιμεύει ως δοχείο για κείμενο και μαθηματικό περιεχόμενο:

```python
math_shape = pres.slides[0].shapes.add_auto_shape(
    slides.ShapeType.RECTANGLE, 10, 10, 100, 25
)
```

Εδώ, `slides.ShapeType.RECTANGLE` καθορίζει τον τύπο του σχήματος, ενώ οι αριθμοί καθορίζουν τη θέση και το μέγεθός του στη διαφάνεια.

### Βήμα 3: Εισαγωγή μαθηματικής έκφρασης

Αποκτήστε πρόσβαση στο πλαίσιο κειμένου μέσα στο σχήμα σας για να εισαγάγετε μαθηματικές παραστάσεις χρησιμοποιώντας τις μαθηματικές λειτουργίες του Aspose.Slides:

```python
math_paragraph = math_shape.text_frame.paragraphs[0].portions[0].math_paragraph
```

Κατασκευάστε την έκφραση του Πυθαγόρειου θεωρήματος:

```python
math_block = mathtext.MathematicalText("c").set_superscript("2") \
    .join("=") \
    .join(mathtext.MathematicalText("a").set_superscript("2")) \
    .join("") \
    .join(mathtext.MathematicalText("b").set_superscript("2"))
```

Αυτός ο κώδικας δημιουργεί την έκφραση (c^2 = a^2 + b^2) χρησιμοποιώντας `MathematicalText` αντικείμενα που αναπαριστούν κάθε στοιχείο.

### Βήμα 4: Αποθήκευση της παρουσίασης

Τέλος, αποθηκεύστε την παρουσίασή σας με το νεοδημιουργημένο μαθηματικό περιεχόμενο:

```python
pres.save("YOUR_OUTPUT_DIRECTORY/shapes_add_math_text_out.pptx", slides.export.SaveFormat.PPTX)
```

Αντικαθιστώ `"YOUR_OUTPUT_DIRECTORY"` με τη διαδρομή όπου θέλετε να αποθηκεύσετε το αρχείο σας.

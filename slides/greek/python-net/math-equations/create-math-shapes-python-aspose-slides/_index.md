## Οδηγός Εφαρμογής

### Δημιουργία και προσθήκη μαθηματικών σχημάτων

Το πρώτο βήμα είναι η δημιουργία μιας παρουσίασης και η προσθήκη ενός μαθηματικού σχήματος.

#### Βήμα 1: Αρχικοποίηση της παρουσίασης

Ξεκινήστε αρχικοποιώντας την παρουσίασή σας:

```python
import aspose.slides as slides
import aspose.slides.mathtext as mathtext

def create_and_manipulate_math_shape():
    with slides.Presentation() as presentation:
        slide = presentation.slides[0]
```

#### Βήμα 2: Προσθήκη μαθηματικού σχήματος

Προσθέστε ένα μαθηματικό σχήμα στη διαφάνειά σας:

```python
        # Προσθήκη ενός MathShape στη θέση (10, 10) με πλάτος και ύψος 500
        math_shape = slide.shapes.add_math_shape(10, 10, 500, 500)
```

#### Βήμα 3: Δημιουργία και προσθήκη μαθηματικού κειμένου

Τώρα, δημιουργήστε μαθηματικά μπλοκ κειμένου:

```python
        # Πρόσβαση στην μαθηματική παράγραφο του πρώτου μέρους της πρώτης παραγράφου
        math_paragraph = math_shape.text_frame.paragraphs[0].portions[0].math_paragraph

        # Δημιουργήστε ένα MathBlock με μια έκφραση "F + (1/y) underbar"
        math_block = mathtext.MathBlock(
            mathtext.MathematicalText("F").join(".add")
            .join(mathtext.MathematicalText("1").divide("y")).underbar())

        # Προσθήκη του MathBlock στο MathParagraph
        math_paragraph.add(math_block)
```

#### Βήμα 4: Εκτύπωση Μαθηματικών Στοιχείων

Για να δείτε τα στοιχεία σας, χρησιμοποιήστε μια αναδρομική συνάρτηση:

```python
def foreach_math_element(root):
    for child in root.get_children():
        element_info = f"{type(child)}"
        if isinstance(child, slides.mathtext.MathematicalText):
            element_info += ": " + str(child.value)
        print(element_info)
        foreach_math_element(child)

# Εκτύπωση όλων των στοιχείων στο μαθηματικό μπλοκ
foreach_math_element(math_block)
```

#### Βήμα 5: Αποθήκευση της παρουσίασης

Τέλος, αποθηκεύστε την παρουσίασή σας:

```python
        # Αποθήκευση σε καθορισμένο κατάλογο εξόδου
        presentation.save("YOUR_OUTPUT_DIRECTORY/shapes_mathtext_get_children_out.pptx", slides.export.SaveFormat.PPTX)

create_and_manipulate_math_shape()
```

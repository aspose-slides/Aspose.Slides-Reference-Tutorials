**1. Αρχικοποίηση παρουσίασης**

Ξεκινήστε δημιουργώντας ένα νέο αντικείμενο παρουσίασης:

```python

import aspose.slides.mathtext as mathtext

with slides.Presentation() as pres:
    # Το πλαίσιο για τις δραστηριότητές μας έχει καθοριστεί.
```

**2. Προσθήκη μαθηματικού σχήματος στη διαφάνεια**

Προσθέστε ένα μαθηματικό σχήμα στην επιθυμητή θέση στη διαφάνειά σας:

```python
# Προσθήκη μαθηματικού σχήματος με καθορισμένες διαστάσεις (x, y, πλάτος, ύψος)
auto_shape = pres.slides[0].shapes.add_math_shape(0, 0, 500, 50)
```

Ανακτήστε την μαθηματική παράγραφο για να την τροποποιήσετε:

```python
# Πρόσβαση στην μαθηματική παράγραφο στο πλαίσιο κειμένου του σχήματος
math_paragraph = auto_shape.text_frame.paragraphs[0].portions[0].math_paragraph
```

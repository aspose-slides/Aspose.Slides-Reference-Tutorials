## Οδηγός Εφαρμογής

Αυτός ο οδηγός θα σας καθοδηγήσει στη δημιουργία μιας παρουσίασης με σχήμα έλλειψης χρησιμοποιώντας το Aspose.Slides για Python.

### Δημιουργία νέας παρουσίασης

#### Επισκόπηση
Ξεκινήστε αρχικοποιώντας ένα νέο αντικείμενο παρουσίασης. Αυτό χρησιμεύει ως βάση όπου θα προστεθούν όλες οι διαφάνειες και το περιεχόμενό σας.

```python
import aspose.slides as slides
import aspose.pydrawing as drawing

# Δημιουργήστε μια νέα παρουσία παρουσίασης
total_pres = slides.Presentation()
```

#### Εξήγηση
- **`slides.Presentation()`**: Αυτό δημιουργεί μια κενή παρουσίαση. Το `with` Η δήλωση διασφαλίζει την αποτελεσματική διαχείριση των πόρων.

### Προσθήκη και μορφοποίηση σχημάτων σε διαφάνειες

#### Επισκόπηση
Στη συνέχεια, θα επικεντρωθούμε στην προσθήκη ενός σχήματος στην πρώτη διαφάνεια και στην εφαρμογή επιλογών μορφοποίησης όπως το χρώμα γεμίσματος και το στυλ περιγράμματος.

```python
# Λήψη της πρώτης διαφάνειας (ευρετήριο 0)
slide = total_pres.slides[0]

# Προσθήκη σχήματος έλλειψης στη διαφάνεια
shape = slide.shapes.add_auto_shape(slides.ShapeType.ELLIPSE, 50, 150, 150, 50)

# Εφαρμογή συμπαγούς χρώματος γεμίσματος στο εσωτερικό της έλλειψης
shape.fill_format.fill_type = slides.FillType.SOLID
shape.fill_format.solid_fill_color.color = drawing.Color.chocolate

# Ορισμός της μορφής γραμμής για το περίγραμμα της έλλειψης
shape.line_format.fill_format.fill_type = slides.FillType.SOLID
shape.line_format.fill_format.solid_fill_color.color = drawing.Color.black
shape.line_format.width = 5
```

#### Εξήγηση
- **`slide.shapes.add_auto_shape()`**: Προσθέτει ένα σχήμα στη διαφάνεια. Εδώ, χρησιμοποιούμε μια έλλειψη.
- **`fill_format` και `line_format`**Αυτές οι ιδιότητες καθορίζουν τον τρόπο με τον οποίο μορφοποιείται το εσωτερικό και το περίγραμμα του σχήματος.

### Αποθήκευση της παρουσίασης
Τέλος, αποθηκεύστε την παρουσίασή σας σε έναν καθορισμένο κατάλογο:

```python
# Αποθήκευση της παρουσίασης σε έναν καθορισμένο κατάλογο
total_pres.save("YOUR_OUTPUT_DIRECTORY/shapes_formatted_ellipse_out.pptx", slides.export.SaveFormat.PPTX)
```

#### Εξήγηση
- **`total_pres.save()`**Αυτή η μέθοδος γράφει τα δεδομένα της παρουσίασης σε ένα αρχείο, επιτρέποντάς σας να αποθηκεύσετε την εργασία σας μόνιμα.

1. **Πώς μπορώ να προσθέσω περισσότερες διαφάνειες;**

2. **Μπορώ να αλλάξω τον τύπο σχήματος;**
   - Ναι, αντικατάσταση `ShapeType.ELLIPSE` με άλλους τύπους όπως `RECTANGLE`.
3. **Τι γίνεται αν το αρχείο παρουσίασής μου δεν αποθηκεύεται;**
   - Βεβαιωθείτε ότι η διαδρομή του καταλόγου εξόδου είναι σωστή και έχει δικαιώματα εγγραφής.

## Οδηγός Εφαρμογής

### Αντικατάσταση κειμένου σε διαφάνειες PowerPoint

**Επισκόπηση:**
Αυτή η λειτουργία σάς επιτρέπει να αυτοματοποιήσετε τη διαδικασία εύρεσης και αντικατάστασης κειμένου μέσα σε δεσμευτικά θέσης σε μια διαφάνεια. Αυτό είναι ιδιαίτερα χρήσιμο για μαζική επεξεργασία ή τυποποίηση περιεχομένου σε πολλές διαφάνειες.

#### Βήμα 1: Φόρτωση της παρουσίασής σας
Ξεκινήστε φορτώνοντας το υπάρχον αρχείο PPTX:

```python
in_file_path = 'YOUR_DOCUMENT_DIRECTORY/text_default_fonts.pptx'

# Άνοιγμα της παρουσίασης από τον δίσκο
with slides.Presentation(in_file_path) as pres:
    # Πρόσβαση στην πρώτη διαφάνεια της παρουσίασης
    slide = pres.slides[0]
```

#### Βήμα 2: Επανάληψη σχημάτων και αντικατάσταση κειμένου
Επαναλάβετε κάθε σχήμα στη διαφάνεια για να εντοπίσετε τα σύμβολα κράτησης θέσης και να αντικαταστήσετε το περιεχόμενο κειμένου τους:

```python
for shape in slide.shapes:
    if shape.placeholder is not None:
        # Αντικατάσταση κειμένου κράτησης θέσης
        shape.text_frame.text = "This is Placeholder"
```

#### Βήμα 3: Αποθήκευση της τροποποιημένης παρουσίασης
Μόλις ολοκληρωθούν οι τροποποιήσεις, αποθηκεύστε την παρουσίασή σας ξανά στον δίσκο:

```python
out_file_path = 'YOUR_OUTPUT_DIRECTORY/text_replacing_out.pptx'
pres.save(out_file_path, slides.export.SaveFormat.PPTX)
```

### Πρόσβαση και τροποποίηση σχημάτων διαφανειών

**Επισκόπηση:**
Μάθετε πώς να αποκτάτε πρόσβαση σε διαφορετικά σχήματα σε μια διαφάνεια και να τροποποιείτε τις ιδιότητές τους, όπως το χρώμα ή το στυλ.

#### Βήμα 1: Ανοίξτε την παρουσίαση
Ανοίξτε το αρχείο PPTX και επιλέξτε τη διαφάνεια που θέλετε να επεξεργαστείτε:

```python
in_file_path = 'YOUR_DOCUMENT_DIRECTORY/example.pptx'

with slides.Presentation(in_file_path) as pres:
    slide = pres.slides[0]
```

#### Βήμα 2: Τροποποίηση ιδιοτήτων σχήματος
Περιηγηθείτε σε κάθε σχήμα, προσδιορίστε αν είναι ένα `AutoShape`και εφαρμόστε τροποποιήσεις όπως η αλλαγή του χρώματος γεμίσματος:

```python
for shape in slide.shapes:
    if isinstance(shape, slides.AutoShape):
        # Αλλαγή χρώματος γεμίσματος σε συμπαγές μπλε
        shape.fill_format.fill_type = slides.FillType.SOLID
        shape.fill_format.solid_fill_color.color = slides.Color.blue
```

#### Βήμα 3: Αποθήκευση της ενημερωμένης παρουσίασης
Αποθηκεύστε τις αλλαγές σας σε ένα νέο αρχείο:

```python
out_file_path = 'YOUR_OUTPUT_DIRECTORY/shapes_modified_out.pptx'
pres.save(out_file_path, slides.export.SaveFormat.PPTX)
```

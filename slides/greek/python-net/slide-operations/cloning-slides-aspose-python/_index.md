Μετά την εγκατάσταση, αρχικοποιήστε το αντικείμενο παρουσίασής σας:

```python

current_presentation = slides.Presentation()
```

## Οδηγός Εφαρμογής

Αυτή η ενότητα σάς καθοδηγεί στην κλωνοποίηση διαφανειών μεταξύ ενοτήτων σε μια παρουσίαση.

### Επισκόπηση: Κλωνοποίηση διαφανειών μεταξύ ενοτήτων

Στόχος μας είναι να κλωνοποιήσουμε μια διαφάνεια από μια ενότητα και να την τοποθετήσουμε σε μια άλλη. Αυτό μπορεί να είναι χρήσιμο για την αντιγραφή περιεχομένου που χρειάζεται επανάληψη σε διαφορετικά μέρη της παρουσίασής σας.

#### Βήμα 1: Δημιουργία αρχικής διαφάνειας με σχήμα

Αρχικά, προσθέστε ένα ορθογώνιο σχήμα στην πρώτη διαφάνεια ως πρότυπο:

```python
current_presentation.slides[0].shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 200, 50, 300, 100)
```

#### Βήμα 2: Δημιουργία και ανάθεση ενοτήτων

Δημιουργήστε μια νέα ενότητα με το όνομα «Ενότητα 1» και αντιστοιχίστε την αρχική διαφάνεια σε αυτήν:

```python
current_presentation.sections.add_section("Section 1", current_presentation.slides[0])
```

Στη συνέχεια, προσθέστε μια κενή ενότητα με το όνομα «Ενότητα 2»:

```python
section2 = current_presentation.sections.append_empty_section("Section 2")
```

#### Βήμα 3: Κλωνοποίηση διαφάνειας σε νέα ενότητα

Χρησιμοποιήστε το `add_clone` μέθοδος για την κλωνοποίηση της πρώτης διαφάνειας στη δεύτερη ενότητα:

```python
current_presentation.slides.add_clone(current_presentation.slides[0], section2)
```

#### Βήμα 4: Αποθήκευση παρουσίασης

Τέλος, αποθηκεύστε την παρουσίασή σας στον επιθυμητό κατάλογο:

```python
current_presentation.save("YOUR_OUTPUT_DIRECTORY/crud_append_empty_section_out.pptx", slides.export.SaveFormat.PPTX)
```

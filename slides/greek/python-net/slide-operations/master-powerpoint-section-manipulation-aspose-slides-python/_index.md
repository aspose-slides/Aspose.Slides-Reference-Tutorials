## Οδηγός Εφαρμογής
Αυτή η ενότητα παρέχει σαφή βήματα για τη φόρτωση και τον χειρισμό ενοτήτων του PowerPoint:

### Φόρτωση της παρουσίασης
Ξεκινήστε ορίζοντας διαδρομές για τους καταλόγους εισόδου και εξόδου και ελέγχοντας την ύπαρξη αρχείων:
```python
import os
from pathlib import Path
import aspose.slides as slides

data_directory = "YOUR_DOCUMENT_DIRECTORY/"
output_directory = "YOUR_OUTPUT_DIRECTORY/"
input_presentation_path = data_directory + 'welcome-to-powerpoint.pptx'
output_presentation_path = output_directory + 'crud_sections_out.pptx'

def load_and_manipulate_sections():
    if not Path(input_presentation_path).is_file():
        raise FileNotFoundError(f"The file {input_presentation_path} does not exist.")
```

### Αναδιάταξη Ενοτήτων
Για να αναδιατάξετε μια ενότητα, αποκτήστε πρόσβαση σε αυτήν μέσω ευρετηρίου και χρησιμοποιήστε το `reorder_section_with_slides` μέθοδος:
```python
with slides.Presentation(input_presentation_path) as pres:
    section_to_reorder = pres.sections[2]  # Πρόσβαση στην τρίτη ενότητα (ευρετήριο 2)
    pres.sections.reorder_section_with_slides(section_to_reorder, 0)  # Μετακίνηση στην πρώτη θέση
```

### Αφαίρεση τμημάτων
Αφαίρεση μιας ενότητας και όλων των διαφανειών της με `remove_section_with_slides`:
```python
pres.sections.remove_section_with_slides(pres.sections[0])  # Αφαίρεση πρώτου τμήματος
```

### Προσθήκη νέων ενοτήτων
Προσθήκη νέων ενοτήτων χρησιμοποιώντας `append_empty_section` ή `add_section` για περισσότερο έλεγχο:
```python
pres.sections.append_empty_section("Last empty section")  # Προσθήκη νέας κενής ενότητας
pres.sections.add_section("First empty", pres.slides[7])  # Προσθήκη με δείκτη διαφάνειας 7 ως πρώτη διαφάνεια
```

### Μετονομασία τμημάτων
Αλλάξτε το όνομα μιας υπάρχουσας ενότητας ενημερώνοντάς την `name` ιδιοκτησία:
```python
pres.sections[0].name = "New section name"  # Μετονομασία πρώτης ενότητας
```

### Αποθήκευση της παρουσίασης
Αποθηκεύστε τις αλλαγές σας με το `save` μέθοδος:
```python
pres.save(output_presentation_path, slides.export.SaveFormat.PPTX)
```

## Οδηγός Εφαρμογής: Ορισμός Προεπιλεγμένης Γλώσσας Κειμένου

### Επισκόπηση

Αυτή η λειτουργία σάς επιτρέπει να ορίσετε μια προεπιλεγμένη γλώσσα κειμένου για όλα τα στοιχεία κειμένου σε μια παρουσίαση PowerPoint, απλοποιώντας τις ροές εργασίας εξαλείφοντας τις επαναλαμβανόμενες εργασίες.

### Βήμα προς βήμα εφαρμογή

#### Δημιουργήστε LoadOptions για να καθορίσετε την προεπιλεγμένη γλώσσα κειμένου

1. **Αρχικοποίηση Επιλογών Φόρτωσης**
   Ξεκινήστε δημιουργώντας μια παρουσία του `LoadOptions` για να καθορίσετε την επιθυμητή προεπιλεγμένη γλώσσα κειμένου:

   ```python
   load_options = slides.LoadOptions()
   ```

2. **Ορισμός της προεπιλεγμένης γλώσσας**
   Ορίστε την προεπιλεγμένη γλώσσα κειμένου χρησιμοποιώντας μια ετικέτα γλώσσας BCP-47 (π.χ., "en-US" για Αγγλικά, Ηνωμένες Πολιτείες):

   ```python
   load_options.default_text_language = "en-US"
   ```

#### Άνοιγμα και τροποποίηση παρουσίασης
3. **Φόρτωση παρουσίασης με το LoadOptions**
   Χρήση `LoadOptions` όταν ανοίγετε την παρουσίασή σας για να εφαρμόσετε την προεπιλεγμένη γλώσσα κειμένου:

   ```python
   with slides.Presentation(load_options) as pres:
       # Προσθήκη νέου ορθογωνίου σχήματος με κείμενο στην πρώτη διαφάνεια
       shp = pres.slides[0].shapes.add_auto_shape(
           slides.ShapeType.RECTANGLE, 50, 50, 150, 50)
       shp.text_frame.text = "New Text"
   ```

4. **Πρόσβαση και επαλήθευση αναγνωριστικού γλώσσας**
   Μπορείτε να ελέγξετε το αναγνωριστικό γλώσσας των τμημάτων κειμένου για να βεβαιωθείτε ότι έχει οριστεί σωστά:

   ```python
   # Πρόσβαση στο αναγνωριστικό γλώσσας για επαλήθευση (προαιρετικό βήμα επίδειξης)
   language_id = shp.text_frame.paragraphs[0].portions[0].portion_format.language_id
   ```

### Συμβουλές αντιμετώπισης προβλημάτων
- **Συνηθισμένο πρόβλημα**: Το προεπιλεγμένο κείμενο δεν αντικατοπτρίζει τις αλλαγές.
  - **Διάλυμα**: Βεβαιωθείτε `LoadOptions` εφαρμόζεται σωστά κατά το άνοιγμα της παρουσίασης.

## Οδηγός Εφαρμογής

Τώρα, ας εφαρμόσουμε τη λειτουργία για να επισημάνουμε κείμενο χρησιμοποιώντας regex.

### Άνοιγμα αρχείου παρουσίασης
Για να εργαστείτε με ένα αρχείο PowerPoint, θα πρέπει πρώτα να το ανοίξετε. Χρησιμοποιούμε τη διαχείριση περιβάλλοντος στην Python για να διασφαλίσουμε ότι οι πόροι διαχειρίζονται αποτελεσματικά:

```python
with slides.Presentation("YOUR_DOCUMENT_DIRECTORY/text_default_fonts.pptx") as presentation:
    # Ο κώδικας για τον χειρισμό της παρουσίασης βρίσκεται εδώ
```

### Πρόσβαση σε πλαίσια κειμένου
Μόλις φορτωθεί η παρουσίασή σας, αποκτήστε πρόσβαση στα πλαίσια κειμένου μέσα σε συγκεκριμένα σχήματα σε μια διαφάνεια. Δείτε πώς μπορείτε να στοχεύσετε το πρώτο σχήμα στην πρώτη διαφάνεια:

```python
text_frame = presentation.slides[0].shapes[0].text_frame
```

### Επισήμανση κειμένου με Regex
Για να επισημάνετε όλες τις λέξεις που περιέχουν 10 ή περισσότερους χαρακτήρες χρησιμοποιώντας regex, θα χρησιμοποιήσετε ένα μοτίβο που ταιριάζει με αυτά τα κριτήρια και θα εφαρμόσετε επισήμανση:

```python
# Το μοτίβο regex \b[^\s]{10,}\b βρίσκει λέξεις μήκους 10 ή μεγαλύτερου
text_frame.highlight_regex(r"\b[^\s]{10,}\b", drawing.Color.blue)
```

**Εξήγηση**: 
- `\b` υποδηλώνει ένα όριο λέξης.
- `[^\s]{10,}` ταιριάζει με τουλάχιστον 10 χαρακτήρες που δεν είναι κενά διαστήματα.
- `drawing.Color.blue` καθορίζει το χρώμα επισήμανσης.

### Αποθήκευση της τροποποιημένης παρουσίασης
Αφού εφαρμόσετε τις αλλαγές, αποθηκεύστε την παρουσίαση σε έναν κατάλογο εξόδου:

```python
presentation.save("YOUR_OUTPUT_DIRECTORY/text_highlight_regex_out.pptx", slides.export.SaveFormat.PPTX)
```

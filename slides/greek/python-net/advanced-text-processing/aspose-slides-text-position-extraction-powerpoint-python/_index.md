## Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, θα αναλύσουμε τη διαδικασία ανάκτησης των συντεταγμένων θέσης τμημάτων κειμένου μέσα σε μια διαφάνεια.

### Εξαγωγή συντεταγμένων θέσης

Ο στόχος είναι να εξαχθούν και να εκτυπωθούν οι συντεταγμένες X και Y κάθε τμήματος κειμένου σε μια συγκεκριμένη διαφάνεια.

#### Φόρτωση της παρουσίασης

Αρχικά, φορτώστε το αρχείο παρουσίασής σας χρησιμοποιώντας το Aspose.Slides:

```python
with slides.Presentation('YOUR_DOCUMENT_DIRECTORY/open_shapes.pptx') as presentation:
    # Πρόσβαση στην πρώτη διαφάνεια
    shape = presentation.slides[0].shapes[0]
    text_frame = shape.text_frame
```

#### Επανάληψη παραγράφων και τμημάτων

Στη συνέχεια, κάντε επανάληψη σε κάθε παράγραφο και τμήμα μέσα στο πλαίσιο κειμένου για να ανακτήσετε συντεταγμένες:

```python
for paragraph in text_frame.paragraphs:
    for portion in paragraph.portions:
        # Ανάκτηση και εκτύπωση των συντεταγμένων X και Y
        point = portion.get_coordinates()
        if point is not None:
            print('Coordinates X = {0} Y = {1}'.format(point.x, point.y))
```

**Παράμετροι & Σκοπός μεθόδου:**

- **`presentation.slides[0].shapes[0]`:** Πρόσβαση στο πρώτο σχήμα της πρώτης διαφάνειας.
- **`get_coordinates()`:** Ανακτά τις συντεταγμένες θέσης ενός τμήματος κειμένου. Σημείωση: Ελέγξτε αν `point` δεν είναι None για την αποφυγή σφαλμάτων με σχήματα χωρίς τμήματα κειμένου.

#### Βασικές επιλογές διαμόρφωσης

### Επισήμανση κειμένου σε μια διαφάνεια
Εφαρμόστε αυτό βήμα προς βήμα:

#### Βήμα 1: Φόρτωση της παρουσίασής σας
Φορτώστε το αρχείο PowerPoint όπου απαιτούνται αλλαγές:
```python
def load_presentation(file_path):
    with slides.Presentation(file_path) as presentation:
        # Συνεχίστε με την επισήμανση κειμένου εδώ.
```

#### Βήμα 2: Ρύθμιση παραμέτρων επιλογών αναζήτησης κειμένου
Ορίστε πώς θα συμπεριφέρεται η αναζήτηση κειμένου:
```python
def configure_search_options():
    options = slides.TextSearchOptions()
    options.whole_words_only = True
    return options
```
Αυτή η ρύθμιση διασφαλίζει ότι επισημαίνονται μόνο ολόκληρες λέξεις που ταιριάζουν με τα κριτήριά σας.

#### Βήμα 3: Επισημάνετε συγκεκριμένες λέξεις
Χρήση `highlight_text` για να εφαρμόσετε χρωματική επισήμανση:
```python
def highlight_specific_words(presentation, shape_index=0):
    # Επισημάνετε τον τίτλο με ανοιχτό μπλε χρώμα
    presentation.slides[shape_index].shapes[0].text_frame.highlight_text("title", drawing.Color.light_blue)

    # Επισημάνετε το 'προς' χρησιμοποιώντας τις διαμορφωμένες επιλογές αναζήτησης, με βιολετί χρώμα
    options = configure_search_options()
    presentation.slides[shape_index].shapes[0].text_frame.highlight_text("to", drawing.Color.violet, options, None)
```

#### Βήμα 4: Αποθήκευση της τροποποιημένης παρουσίασης
Αποθήκευση αλλαγών σε ένα αρχείο:
```python
def save_presentation(presentation, output_path):
    # Αποθήκευση της ενημερωμένης παρουσίασης
    presentation.save(output_path, slides.export.SaveFormat.PPTX)
```
Αυτό το βήμα διασφαλίζει ότι όλες οι αλλαγές διατηρούνται σε ένα νέο ή υπάρχον αρχείο.

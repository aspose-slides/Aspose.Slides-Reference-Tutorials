## Οδηγός Εφαρμογής

### Δυνατότητα διαίρεσης κειμένου ανά στήλες

Αυτή η λειτουργία σάς επιτρέπει να διαιρέσετε το περιεχόμενο ενός πλαισίου κειμένου σε πολλές στήλες μέσα σε μια παρουσίαση. Δείτε πώς λειτουργεί:

#### Βήμα προς βήμα εφαρμογή
**1. Φόρτωση της παρουσίασης**
Ξεκινήστε φορτώνοντας το αρχείο PowerPoint που περιέχει τα πλαίσια κειμένου.

```python
import aspose.slides as slides

def split_text_by_columns():
    input_path = "YOUR_DOCUMENT_DIRECTORY/MultiColumnText.pptx"
    output_path = "YOUR_OUTPUT_DIRECTORY/output.txt"  # Προαιρετικά: Ορισμός για αποθήκευση εξόδου
    
    with slides.Presentation(input_path) as pres:
        slide = pres.slides[0]
```

**2. Πρόσβαση στο Πλαίσιο Κειμένου**
Προσδιορίστε και αποκτήστε πρόσβαση στο πρώτο πλαίσιο κειμένου στη διαφάνειά σας.

```python
shape = slide.shapes[0]  # Υποθέτοντας ότι είναι ένα σχήμα που περιέχει κείμενο
text_frame = shape.text_frame
```

**3. Διαχωρίστε το περιεχόμενο σε στήλες**
Χρησιμοποιήστε το `split_text_by_columns` μέθοδος για τη διαίρεση του περιεχομένου.

```python
columns_text = text_frame.split_text_by_columns()
```

**4. Εξαγωγή ή χρήση του αποτελέσματος**
Επαναλάβετε το κείμενο κάθε στήλης για να επαληθεύσετε την έξοδο:

```python
for column in columns_text:
    print(column)
```

### Εξήγηση
- **Παράμετροι & Επιστρεφόμενες Τιμές:** Ο `split_text_by_columns` Η μέθοδος δεν απαιτεί παραμέτρους και επιστρέφει μια λίστα συμβολοσειρών, καθεμία από τις οποίες αντιπροσωπεύει το περιεχόμενο μιας στήλης.
- **Συμβουλή αντιμετώπισης προβλημάτων:** Βεβαιωθείτε ότι το πλαίσιο κειμένου περιέχει πολλές γραμμές για να δείξετε αποτελεσματικά τον διαχωρισμό στηλών.

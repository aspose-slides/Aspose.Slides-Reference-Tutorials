## Οδηγός Εφαρμογής

Αυτή η ενότητα σάς καθοδηγεί στην εξαγωγή τρεχόντων και αρχικών υπερσυνδέσμων από διαφάνειες του PowerPoint.

### Εξαγωγή URL από Διαφάνειες

#### Επισκόπηση

Εξαγάγετε τόσο ψεύτικους (τρέχοντες) όσο και αρχικούς υπερσυνδέσμους για να παρέχετε διαφάνεια σχετικά με τυχόν τροποποιήσεις με την πάροδο του χρόνου στα στοιχεία της διαφάνειάς σας.

#### Βήμα προς βήμα εφαρμογή

**1. Εισαγωγή απαιτούμενων βιβλιοθηκών**
Ξεκινήστε εισάγοντας την απαραίτητη ενότητα Aspose.Slides:

```python
import aspose.slides as slides
```

**2. Ορίστε διαδρομές αρχείων**
Ορίστε διαδρομές για το έγγραφο παρουσίασης και τον κατάλογο εξόδου:

```python
document_path = "YOUR_DOCUMENT_DIRECTORY/ExternalUrlOriginal.pptx"
output_directory = "YOUR_OUTPUT_DIRECTORY/"
```

**3. Φόρτωση της παρουσίασης**
Ανοίξτε το αρχείο PowerPoint χρησιμοποιώντας το Aspose.Slides `Presentation` τάξη:

```python
with slides.Presentation(document_path) as presentation:
    # Ο κώδικας επεξεργασίας σας πηγαίνει εδώ
```

**4. Πρόσβαση σε στοιχεία διαφάνειας**
Μεταβείτε στο συγκεκριμένο στοιχείο σχήματος και κειμένου όπου θέλετε να εξαγάγετε υπερσυνδέσμους:

```python
portion = presentation.slides[0].shapes[1].text_frame.paragraphs[0].portions[0]
```
*Εδώ, `shapes[1]` αναφέρεται στο δεύτερο σχήμα στην πρώτη διαφάνεια. Τροποποιήστε αυτόν τον δείκτη με βάση τις συγκεκριμένες ανάγκες σας.*

**5. Εξαγωγή πληροφοριών υπερσυνδέσμου**
Ανακτήστε τόσο τους ψεύτικους όσο και τους αρχικούς υπερσυνδέσμους:

```python
external_url = portion.portion_format.hyperlink_click.external_url
external_url_original = portion.portion_format.hyperlink_click.external_url_original
```

**6. Διευθύνσεις URL εμφάνισης**
Εκτυπώστε ή καταγράψτε αυτές τις διευθύνσεις URL για επαλήθευση:

```python
print("Fake External Hyperlink:", external_url)
print("Real External Hyperlink:", external_url_original)
```

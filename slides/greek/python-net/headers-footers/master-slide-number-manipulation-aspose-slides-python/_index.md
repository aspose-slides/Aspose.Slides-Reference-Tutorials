## Οδηγός Εφαρμογής

### Απόδοση και Ρύθμιση Αριθμού Διαφάνειας

#### Επισκόπηση:
Αυτή η λειτουργία σάς επιτρέπει να φορτώσετε μια παρουσίαση PowerPoint, να ανακτήσετε και να τροποποιήσετε τον πρώτο αριθμό διαφάνειας και, στη συνέχεια, να αποθηκεύσετε αποτελεσματικά τις αλλαγές.

#### Βήματα:

##### Βήμα 1: Ορισμός διαδρομών αρχείων
Ξεκινήστε ορίζοντας διαδρομές για τα αρχεία εισόδου και εξόδου. Αντικαταστήστε τα placeholders με πραγματικά ονόματα καταλόγων.

```python
input_path = "YOUR_DOCUMENT_DIRECTORY/welcome-to-powerpoint.pptx"
output_path = "YOUR_OUTPUT_DIRECTORY/rendering_set_slide_number_out.pptx"
```

##### Βήμα 2: Φόρτωση της παρουσίασης

Χρήση `slides.Presentation` για να φορτώσετε το αρχείο PowerPoint. Αυτός ο διαχειριστής περιβάλλοντος διασφαλίζει ότι οι πόροι θα αποδεσμευτούν όταν ολοκληρωθεί η διαδικασία.

```python
with slides.Presentation(input_path) as presentation:
    # Συνέχεια με τον χειρισμό του αριθμού των διαφανειών
```

##### Βήμα 3: Ανάκτηση και τροποποίηση αριθμού διαφάνειας

Ανακτήστε τον τρέχοντα αριθμό της πρώτης διαφάνειας για επαλήθευση και, στη συνέχεια, ορίστε μια νέα τιμή:

```python
first_slide_number = presentation.first_slide_number
print(f"Original First Slide Number: {first_slide_number}")

presentation.first_slide_number = 10
print("First slide number set to 10.")
```

##### Βήμα 4: Αποθήκευση της τροποποιημένης παρουσίασης

Τέλος, αποθηκεύστε τις αλλαγές σας. Αυτό το βήμα διασφαλίζει ότι όλες οι τροποποιήσεις αποθηκεύονται.

```python
presentation.save(output_path, slides.export.SaveFormat.PPTX)
print(f"Presentation saved with new slide numbering at {output_path}")
```

#### Συμβουλές αντιμετώπισης προβλημάτων:
- Βεβαιωθείτε ότι οι διαδρομές έχουν καθοριστεί σωστά για να αποφύγετε σφάλματα "δεν βρέθηκε αρχείο".
- Βεβαιωθείτε ότι το αρχείο PowerPoint είναι προσβάσιμο και δεν είναι κατεστραμμένο.
- Ελέγξτε ότι έχετε δικαίωμα εγγραφής αρχείων στον κατάλογο εξόδου.

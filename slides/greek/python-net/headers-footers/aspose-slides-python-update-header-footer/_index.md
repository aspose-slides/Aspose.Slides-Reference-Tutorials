## Οδηγός Εφαρμογής

### Λειτουργία 1: Ενημέρωση κειμένου κεφαλίδας στις κύριες σημειώσεις

Αυτή η λειτουργία εστιάζει στην ενημέρωση του κειμένου κεφαλίδας των placeholders μέσα στις κύριες σημειώσεις μιας διαφάνειας. Δείτε πώς μπορείτε να το πετύχετε αυτό:

#### Επισκόπηση
Θα επαναλάβετε τα σχήματα στις κύριες σημειώσεις και θα ενημερώσετε τυχόν κεφαλίδες που βρέθηκαν.

#### Βήματα Υλοποίησης
**Βήμα 1: Ορισμός συνάρτησης για την ενημέρωση κεφαλίδων**
```python
import aspose.slides as slides

def update_header_footer_text(master):
    """
    Iterate through shapes in the master and update header text if applicable.
    
    Args:
        master (slides.MasterSlide): The master slide containing the shapes to be updated.
    """
    for shape in master.shapes:
        # Ελέγξτε αν το σχήμα είναι placeholder και συγκεκριμένα τύπου HEADER
        if shape.placeholder is not None and shape.placeholder.type == slides.PlaceholderType.HEADER:
            shape.text_frame.text = "HI there new header"
```
**Βήμα 2: Πρόσβαση στη διαφάνεια κύριων σημειώσεων**
Φορτώστε την παρουσίασή σας, αποκτήστε πρόσβαση στη διαφάνεια των κύριων σημειώσεων και εφαρμόστε την ενημέρωση κεφαλίδας.
```python
def manage_header_footer_text():
    data_dir = "/path/to/your/document/directory/"
    out_dir = "/path/to/your/output/directory/"

    with slides.Presentation(data_dir + "layout_presentation.ppt") as pres:
        # Πρόσβαση στη διαφάνεια των κύριων σημειώσεων για ενημέρωση του κειμένου της κεφαλίδας
        master_notes_slide = pres.master_notes_slide_manager.master_notes_slide
        if master_notes_slide is not None:
            update_header_footer_text(master_notes_slide)

        # Αποθήκευση της παρουσίασης με ενημερωμένες κεφαλίδες
        pres.save(out_dir + "layout_update_header_footer_text_out.pptx", slides.export.SaveFormat.PPTX)
```
### Λειτουργία 2: Διαχείριση κειμένου κεφαλίδας και υποσέλιδου

Εδώ, θα ορίσουμε κείμενο υποσέλιδου σε όλες τις διαφάνειες και θα αποθηκεύσουμε τις τροποποιήσεις.

#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να ορίσετε και να εμφανίσετε υποσέλιδα σε όλες τις διαφάνειες μιας παρουσίασης.

**Βήμα 1: Ορισμός κειμένου υποσέλιδου**
Χρησιμοποιήστε τον διαχειριστή κεφαλίδας-υποσέλιδου για να ενημερώσετε τα υποσέλιδα για όλες τις διαφάνειες:
```python
def manage_header_footer_text():
    data_dir = "/path/to/your/document/directory/"
    out_dir = "/path/to/your/output/directory/"

    with slides.Presentation(data_dir + "layout_presentation.ppt") as pres:
        # Ενημέρωση κειμένου υποσέλιδου και ορατότητά του σε όλες τις διαφάνειες
        pres.header_footer_manager.set_all_footers_text("My Footer Text")
        pres.header_footer_manager.set_all_footers_visibility(True)
        
        # Αποθήκευση της ενημερωμένης παρουσίασης
        pres.save(out_dir + "layout_update_header_footer_text_out.pptx", slides.export.SaveFormat.PPTX)
```

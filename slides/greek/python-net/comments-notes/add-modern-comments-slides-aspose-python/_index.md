## Οδηγός Εφαρμογής

### Προσθήκη σύγχρονων σχολίων σε διαφάνειες PowerPoint

#### Επισκόπηση

Αυτή η λειτουργία σάς επιτρέπει να προσθέτετε σύγχρονα σχόλια απευθείας στις διαφάνειες της παρουσίασής σας. Αυτά τα σχόλια συνδέονται με τους συντάκτες, επιτρέποντας τη συλλογική συμβολή και ανατροφοδότηση.

#### Βήμα προς βήμα εφαρμογή

**1. Αρχικοποίηση παρουσίασης**

Ξεκινήστε δημιουργώντας μια παρουσία του `Presentation` τάξη:

```python
with slides.Presentation() as pres:
    # Ο κώδικας θα προστεθεί εδώ
```

**2. Προσθήκη Συγγραφέα για Σχόλια**

Προσθέστε έναν συγγραφέα που θα είναι υπεύθυνος για τα σχόλια:

```python
new_author = pres.comment_authors.add_author("Some Author", "SA")
```
- **Παράμετροι:** Όνομα συγγραφέα και μοναδικό αναγνωριστικό.

**3. Προσθήκη σύγχρονου σχολίου**

Στη συνέχεια, προσθέστε ένα σύγχρονο σχόλιο στη διαφάνεια-στόχο σας:

```python
modern_comment = new_author.comments.add_modern_comment(
    "This is a modern comment",
    pres.slides[0],  # Στόχευση της πρώτης διαφάνειας
    None,            # Δεν υπάρχει συγκεκριμένο σχήμα για το σχόλιο
    drawing.PointF(100, 100),  # Θέση του σχολίου στη διαφάνεια
    date.today()     # Τρέχουσα ημερομηνία ως χρονική σήμανση
)
```
- **Παράμετροι:**
  - `text`: Το περιεχόμενο του σχολίου.
  - `slide_index`Ευρετήριο της διαφάνειας-στόχου.
  - `shape`: Αναφορά σχήματος (προαιρετικό, Καμία εάν δεν χρησιμοποιείται).
  - `point`: Θέση στη διαφάνεια όπου θα τοποθετηθεί το σχόλιο.
  - `date_time`: Χρονική σήμανση για το πότε προστέθηκε το σχόλιο.

**4. Αποθήκευση παρουσίασης**

Τέλος, αποθηκεύστε την παρουσίασή σας για να βεβαιωθείτε ότι όλες οι αλλαγές έχουν αποθηκευτεί:

```python
pres.save("YOUR_OUTPUT_DIRECTORY/comments_add_modern_comment_out.pptx", slides.export.SaveFormat.PPTX)
```
- **Παράμετροι:** 
  - Διαδρομή αρχείου με όνομα.
  - Μορφή εξαγωγής (PPTX σε αυτήν την περίπτωση).

## Οδηγός Εφαρμογής

### Προσθήκη σχολίων γονέα

#### Επισκόπηση

Αυτή η λειτουργία σάς επιτρέπει να προσθέτετε σχόλια και τις ιεραρχικές απαντήσεις τους σε παρουσιάσεις PowerPoint. Αυτό είναι ιδιαίτερα χρήσιμο για την οργάνωση σχολίων και συζητήσεων απευθείας μέσα στις διαφάνειές σας.

#### Βήμα προς βήμα εφαρμογή

**1. Δημιουργήστε μια παρουσία παρουσίασης**

Ξεκινήστε δημιουργώντας μια παρουσία της παρουσίασης:

```python
import aspose.slides as slides
from datetime import date
import aspose.pydrawing as drawing

def add_parent_comments():
    with slides.Presentation() as pres:
        # Προσθήκη κύριου σχολίου και απαντήσεων
```

**2. Προσθήκη κύριου σχολίου**

Προσθήκη κύριου σχολίου χρησιμοποιώντας έναν συντάκτη:

```python
author1 = pres.comment_authors.add_author("Author_1", "A.A.")
comment1 = author1.comments.add_comment("Main comment", pres.slides[0], drawing.PointF(10, 10), date.today())
```

**3. Προσθήκη απάντησης στο κύριο σχόλιο**

Δημιουργήστε μια απάντηση στο κύριο σχόλιο:

```python
author2 = pres.comment_authors.add_author("Author_2", "B.b.")
reply1 = author2.comments.add_comment("Reply 1 for main comment", pres.slides[0], drawing.PointF(10, 10), date.today())
reply1.parent_comment = comment1
```

**4. Προσθήκη δευτερεύουσας απάντησης σε μια απάντηση**

Προσθέστε περαιτέρω ιεραρχία προσθέτοντας δευτερεύουσες απαντήσεις:

```python
sub_reply = author1.comments.add_comment("Sub-reply for reply 1", pres.slides[0], drawing.PointF(10, 10), date.today())
sub_reply.parent_comment = reply1
```

**5. Εμφάνιση ιεραρχίας σχολίων**

Εκτυπώστε την ιεραρχία σχολίων για να επαληθεύσετε τη δομή:

```python
slide = pres.slides[0]
comments = slide.get_slide_comments(None)
for i in range(len(comments)):
    comment = comments[i]
    while comment.parent_comment is not None:
        print("\t")
        comment = comment.parent_comment
    # Εκτύπωση συγγραφέα και κειμένου
    print(f"{comments[i].author.name} : {comments[i].text}")
```

**6. Αποθήκευση της παρουσίασης**

Τέλος, αποθηκεύστε την παρουσίασή σας με όλα τα σχόλια:

```python
pres.save("output/comments_parent_comment_out.pptx", slides.export.SaveFormat.PPTX)
```

## Οδηγός Εφαρμογής

Αυτή η ενότητα σας καθοδηγεί στη δημιουργία μιας παρουσίασης και στη διαμόρφωσή της ώστε να περιλαμβάνει μια εικόνα σε μορφή πλακιδίων.

### Δημιουργία και διαμόρφωση παρουσίασης

#### Επισκόπηση
Θα δημιουργήσουμε μια νέα παρουσίαση, θα προσθέσουμε μια διαφάνεια, θα εισαγάγουμε μια εικόνα και θα διαμορφώσουμε ένα σχήμα με μορφή γεμίσματος εικόνας σε πλακίδια.

#### Πρόσβαση στην πρώτη διαφάνεια

Ξεκινήστε αποκτώντας πρόσβαση στην πρώτη διαφάνεια:

```python
# Αρχικοποίηση του αντικειμένου παρουσίασης\με slides.Presentation() ως pres:
    # Πρόσβαση στην πρώτη διαφάνεια της παρουσίασης
    first_slide = pres.slides[0]
```

#### Προσθήκη εικόνας στην παρουσίαση

Φορτώστε και προσθέστε την επιθυμητή εικόνα από έναν κατάλογο:

```python
# Φορτώστε μια εικόνα από έναν καθορισμένο κατάλογο και προσθέστε την στη συλλογή εικόνων της παρουσίασης\with slides.Images.from_file("YOUR_DOCUMENT_DIRECTORY/image.png") ως new_image:
    pp_image = pres.images.add_image(new_image)
```

#### Προσθήκη σχήματος με γέμισμα εικόνας με πλακίδια

Προσθέστε ένα ορθογώνιο σχήμα στη διαφάνειά σας:

```python
# Προσθήκη ορθογωνίου σχήματος στην πρώτη διαφάνεια
ew_shape = first_slide.shapes.add_auto_shape(
    slides.ShapeType.RECTANGLE, 0, 0, 350, 350
)

# Ορίστε τον τύπο γεμίσματος του σχήματος σε Εικόνα και διαμορφώστε τον για παράθεση
new_shape.fill_format.fill_type = slides.FillType.PICTURE
picture_fill_format = new_shape.fill_format.picture_fill_format

# Αντιστοιχίστε την εικόνα που φορτώθηκε στο σχήμα με τη μορφή γεμίσματος εικόνας\ppicture_fill_format.picture.image = pp_image

# Ρύθμιση παραμέτρων ιδιοτήτων γεμίσματος σε πλακίδια\ppicture_fill_format.picture_fill_mode = slides.PictureFillMode.TILE
picture_fill_format.tile_offset_x = -275
picture_fill_format.tile_offset_y = -247
picture_fill_format.tile_scale_x = 120
picture_fill_format.tile_scale_y = 120
picture_fill_format.tile_alignment = slides.RectangleAlignment.BOTTOM_RIGHT
picture_fill_format.tile_flip = slides.TileFlip.FLIP_BOTH
```

#### Αποθήκευση της παρουσίασης

## Οδηγός Εφαρμογής
### Λειτουργία 1: Φόρτωση παρουσίασης
#### Επισκόπηση
Η φόρτωση της παρουσίασής σας είναι ζωτικής σημασίας πριν από την εξαγωγή διαφανειών. Αυτή η ενότητα παρουσιάζει το άνοιγμα και την επαλήθευση του αρχείου παρουσίασής σας.
**Βήμα 1: Ρύθμιση του καταλόγου εγγράφων σας**
```python
import os
import aspose.slides as slides

document_directory = "YOUR_DOCUMENT_DIRECTORY/"
```
**Βήμα 2: Φόρτωση της παρουσίασης**
Βεβαιωθείτε ότι έχετε ένα `.pptx` Το αρχείο είναι έτοιμο στον κατάλογό σας:
```python
with slides.Presentation(os.path.join(document_directory, 'welcome-to-powerpoint.pptx')) as pres:
    # Αποκτήστε πρόσβαση στην πρώτη διαφάνεια για να επαληθεύσετε ότι έχει φορτωθεί σωστά
    all_slides = pres.slides[0]
```
### Χαρακτηριστικό 2: Εξαγωγή διαφάνειας σε SVG
#### Επισκόπηση
Αυτή η λειτουργία δείχνει πώς να εξάγετε μια διαφάνεια PowerPoint σε ένα αρχείο SVG, κατάλληλο για κλιμακούμενα γραφικά σε εφαρμογές ιστού.
**Βήμα 1: Ορίστε τη συνάρτηση που θα αποθηκευτεί ως SVG**
Δημιουργήστε μια συνάρτηση που χειρίζεται την εξαγωγή:
```python
def save_slide_as_svg(slide, output_directory):
    with open(os.path.join(output_directory, 'slide_out.svg'), "wb") as stream:
        slide.write_as_svg(stream)
```
**Βήμα 2: Χρησιμοποιήστε τη συνάρτηση για εξαγωγή**
Χρησιμοποιήστε αυτήν τη συνάρτηση στον διαχειριστή περιβάλλοντος:
```python
document_directory = "YOUR_DOCUMENT_DIRECTORY/"
output_directory = "YOUR_OUTPUT_DIRECTORY/"

with slides.Presentation(os.path.join(document_directory, 'welcome-to-powerpoint.pptx')) as pres:
    # Πρόσβαση στην πρώτη διαφάνεια
    all_slides = pres.slides[0]
    
    # Αποθήκευση της διαφάνειας στην οποία έγινε πρόσβαση σε ένα αρχείο SVG στον καθορισμένο κατάλογο εξόδου
    save_slide_as_svg(all_slides, output_directory)
```
**Επεξήγηση παραμέτρων:**
- `slide`: Το συγκεκριμένο αντικείμενο διαφάνειας που θέλετε να εξαγάγετε.
- `output_directory`: Κατάλογος όπου θα αποθηκευτεί το αρχείο SVG.

## Οδηγός Εφαρμογής

Τώρα που το περιβάλλον σας έχει ρυθμιστεί, ας δούμε πώς να δημιουργείτε μια παρουσίαση με έναν πίνακα.

### Δημιουργία της παρουσίασης

Αρχικά, δημιουργήστε μια παρουσία του `Presentation` τάξη για να ξεκινήσει η εργασία σε διαφάνειες:

```csharp
// Αρχικοποίηση νέας παρουσίασης
Presentation pres = new Presentation();
```

Αυτό το βήμα θέτει τις βάσεις για την προσθήκη περιεχομένου στο αρχείο PowerPoint. Στη συνέχεια, αποκτήστε πρόσβαση στην πρώτη διαφάνεια από τη συλλογή:

```csharp
// Πρόσβαση στην πρώτη διαφάνεια
ISlide slide = pres.Slides[0];
```

### Προσθήκη πίνακα

Τώρα, ας ορίσουμε τις διαστάσεις του πίνακα και ας τις προσθέσουμε στη διαφάνεια:

**Ορισμός διαστάσεων:**
Καθορίστε τα πλάτη των στηλών και τα ύψη των γραμμών για τον πίνακά σας. Αυτό το βήμα είναι κρίσιμο, καθώς καθορίζει τον τρόπο οργάνωσης του περιεχομένου μέσα σε κάθε κελί.

```csharp
// Ορισμός πλάτους στηλών και ύψους γραμμών
double[] colWidth = { 100, 50, 30 };
double[] rowHeight = { 30, 50, 30 };
```

**Προσθήκη του πίνακα:**
Προσθέστε ένα σχήμα πίνακα στη διαφάνειά σας χρησιμοποιώντας αυτές τις διαστάσεις. Θα καθορίσετε τη θέση στη διαφάνεια με συντεταγμένες x και y.

```csharp
// Προσθήκη πίνακα στην πρώτη διαφάνεια στο (x=100, y=100)
ITable table = slide.Shapes.AddTable(100, 100, colWidth, rowHeight);
```

### Αποθήκευση της παρουσίασης

Τέλος, αποθηκεύστε την παρουσίασή σας σε μορφή PPTX:

```csharp
// Αποθήκευση της παρουσίασης σε μια καθορισμένη διαδρομή καταλόγου
pres.Save("YOUR_DOCUMENT_DIRECTORY/TestTable_out.pptx");
```

Αυτό το βήμα διασφαλίζει ότι οι τροποποιήσεις σας διατηρούνται και ότι μπορείτε να τις αποκτήσετε πρόσβαση ή να τις κοινοποιήσετε αργότερα.

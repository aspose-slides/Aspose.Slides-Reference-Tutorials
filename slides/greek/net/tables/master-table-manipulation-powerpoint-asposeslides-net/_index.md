## Οδηγός Εφαρμογής

Θα αναλύσουμε το σεμινάριο σε ξεχωριστά χαρακτηριστικά για να το παρακολουθήσουμε πιο εύκολα.

### Δημιουργία και συμπλήρωση πίνακα

**Επισκόπηση:** Μάθετε πώς να δημιουργείτε έναν πίνακα σε μια διαφάνεια και να τον γεμίζετε με κείμενο χρησιμοποιώντας το Aspose.Slides για .NET.

#### Βήμα 1: Αρχικοποίηση αντικειμένου παρουσίασης

Ξεκινήστε φορτώνοντας το αρχείο PowerPoint:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
using (Presentation presentation = new Presentation(dataDir + "presentation.pptx"))
{
    // Πρόσβαση στην πρώτη διαφάνεια
    ISlide sld = presentation.Slides[0];
```

#### Βήμα 2: Ορισμός διαστάσεων πίνακα

Καθορίστε τα πλάτη των στηλών και τα ύψη των γραμμών:

```csharp
double[] dblCols = { 50, 50, 50 };
double[] dblRows = { 50, 30, 30, 30, 30 };

// Προσθήκη νέου πίνακα στη διαφάνεια στη θέση (100, 50)
ITable table = sld.Shapes.AddTable(100, 50, dblCols, dblRows);
```

#### Βήμα 3: Συμπλήρωση πίνακα με κείμενο

Συμπληρώστε τα κελιά με κείμενο και κλωνοποιήστε τις γραμμές:

```csharp
// Ορισμός αρχικών τιμών κελιών
table[0, 0].TextFrame.Text = "Row 1 Cell 1";
table[1, 0].TextFrame.Text = "Row 1 Cell 2";

// Κλωνοποιήστε την πρώτη γραμμή που θα προστεθεί στο τέλος του πίνακα
table.Rows.AddClone(table.Rows[0], false);

table[0, 1].TextFrame.Text = "Row 2 Cell 1";
table[1, 1].TextFrame.Text = "Row 2 Cell 2";
}
```

### Κλωνοποίηση γραμμών και στηλών σε έναν πίνακα

**Επισκόπηση:** Ανακαλύψτε πώς να κλωνοποιήσετε υπάρχουσες γραμμές και στήλες μέσα σε έναν πίνακα PowerPoint.

#### Βήμα 4: Αρχικοποίηση νέου πίνακα

Δημιουργήστε μια άλλη παρουσία ενός πίνακα για επίδειξη κλωνοποίησης:

```csharp
using (Presentation presentation = new Presentation(dataDir + "presentation.pptx"))
{
    ISlide sld = presentation.Slides[0];
    ITable table = sld.Shapes.AddTable(100, 50, new double[] { 50, 50, 50 }, new double[] { 50, 30, 30, 30, 30 });
```

#### Βήμα 5: Κλωνοποίηση γραμμών και στηλών

Κλωνοποιήστε τη δεύτερη γραμμή σε μια συγκεκριμένη θέση και στήλες με παρόμοιο τρόπο:

```csharp
// Εισαγωγή κλώνου της δεύτερης γραμμής ως τέταρτης γραμμής
table.Rows.InsertClone(3, table.Rows[1], false);

// Προσθήκη κλώνου της πρώτης στήλης στο τέλος
table.Columns.AddClone(table.Columns[0], false);

// Εισαγωγή κλώνου της δεύτερης στήλης στον τέταρτο δείκτη
table.Columns.InsertClone(3, table.Columns[1], false);
}
```

### Αποθήκευση παρουσίασης με τροποποιήσεις

**Επισκόπηση:** Μάθετε πώς να αποθηκεύσετε την τροποποιημένη παρουσίασή σας ξανά στον δίσκο.

#### Βήμα 6: Αποθήκευση αλλαγών στο δίσκο

Τέλος, αποθηκεύστε όλες τις αλλαγές που πραγματοποιήσατε κατά τη διάρκεια της συνεδρίας:

```csharp
using (Presentation presentation = new Presentation(dataDir + "presentation.pptx"))
{
    // Εκτελέστε τροποποιήσεις όπως προσθήκη πινάκων, κλωνοποίηση γραμμών/στηλών κ.λπ.
    
    string outputDir = "YOUR_OUTPUT_DIRECTORY";
    // Αποθήκευση τροποποιημένης παρουσίασης
    presentation.Save(outputDir + "table_out.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
}
```

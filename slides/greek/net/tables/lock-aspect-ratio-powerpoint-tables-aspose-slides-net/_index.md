Τώρα που όλα έχουν ρυθμιστεί, ας δούμε πώς να κλειδώσετε ή να ξεκλειδώσετε την αναλογία διαστάσεων ενός πίνακα στο PowerPoint χρησιμοποιώντας το Aspose.Slides.
### Κλείδωμα/Ξεκλείδωμα αναλογίας διαστάσεων
Αυτή η λειτουργία σάς επιτρέπει να διατηρήσετε τις διαστάσεις των πινάκων σας ακόμα και όταν αλλάζετε το μέγεθος άλλων στοιχείων στη διαφάνειά σας. Δείτε πώς λειτουργεί:
#### Βήμα 1: Φόρτωση της παρουσίασής σας
Αρχικά, φορτώστε το αρχείο παρουσίασης που περιέχει τον πίνακα:
```csharp
using (Presentation pres = new Presentation(dataDir + "/pres.pptx"))
{
    // Ο κώδικας για τον χειρισμό του πίνακα θα τοποθετηθεί εδώ
}
```
#### Βήμα 2: Πρόσβαση στο σχήμα πίνακα
Προσδιορίστε και αποκτήστε πρόσβαση στο πρώτο σχήμα στη διαφάνειά σας, διασφαλίζοντας ότι πρόκειται για πίνακα:
```csharp
ITable table = (ITable)pres.Slides[0].Shapes[0];
```
#### Βήμα 3: Ενεργοποίηση/απενεργοποίηση κλειδώματος αναλογίας διαστάσεων
Ελέγξτε αν η αναλογία διαστάσεων είναι κλειδωμένη αυτήν τη στιγμή. Στη συνέχεια, αλλάξτε την κατάστασή της σε κλείδωμα ή ξεκλείδωμα:
```csharp
bool originalLockState = table.ShapeLock.AspectRatioLocked;
table.ShapeLock.AspectRatioLocked = !originalLockState; // Αντιστροφή της τρέχουσας κατάστασης
```
#### Βήμα 4: Αποθήκευση των αλλαγών σας
Τέλος, αποθηκεύστε την τροποποιημένη παρουσίασή σας σε ένα νέο αρχείο:
```csharp
pres.Save(outputPath + "/pres-out.pptx", SaveFormat.Pptx);
```
### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι το σχήμα στο οποίο αποκτάτε πρόσβαση είναι πράγματι ένας πίνακας.
- Επαληθεύστε ότι οι διαδρομές για τα αρχεία εισόδου και εξόδου έχουν οριστεί σωστά.
- Εάν οι αλλαγές στην αναλογία διαστάσεων δεν αντικατοπτρίζονται, ελέγξτε εάν άλλα στοιχεία της διαφάνειας ενδέχεται να επηρεάζουν τις διαστάσεις.

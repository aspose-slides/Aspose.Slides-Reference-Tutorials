## Οδηγός Εφαρμογής

### Επισκόπηση λειτουργιών: Αναγνώριση συγχωνευμένων κελιών

Αυτή η λειτουργία σάς επιτρέπει να προσδιορίζετε μέσω προγραμματισμού ποια κελιά σε έναν πίνακα αποτελούν μέρος μιας ομάδας συγχώνευσης. Είναι ιδιαίτερα χρήσιμη κατά τον χειρισμό ή την ανάλυση δεδομένων από σύνθετες παρουσιάσεις.

#### Βήμα προς βήμα εφαρμογή

**1. Φόρτωση της παρουσίασης**
Ξεκινήστε φορτώνοντας την παρουσίαση PowerPoint που περιέχει τον πίνακα:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
using (Presentation pres = new Presentation(dataDir + "/SomePresentationWithTable.pptx"))
{
    // Πρόσβαση στην πρώτη διαφάνεια και υπόθεση ότι το πρώτο σχήμα είναι ένας πίνακας.
    ITable table = pres.Slides[0].Shapes[0] as ITable;

    // Περαιτέρω βήματα θα ακολουθήσουν εδώ...
}
```

**2. Επανάληψη μέσω κελιών πίνακα**
Πραγματοποιήστε επανάληψη σε κάθε κελί στον πίνακα για να προσδιορίσετε εάν αποτελεί μέρος ενός συγχωνευμένου κελιού:
```csharp
for (int i = 0; i < table.Rows.Count; i++)
{
    for (int j = 0; j < table.Columns.Count; j++)
    {
        ICell currentCell = table.Rows[i][j];

        // Ελέγξτε αν το τρέχον κελί αποτελεί μέρος ενός συγχωνευμένου κελιού.
        if (currentCell.IsMergedCell)
        {
            Console.WriteLine(string.Format(
                "Cell {0};{1} is part of a merged cell with RowSpan={2} and ColSpan={3}, starting from Cell {4};{5}.",
                i, j,
                currentCell.RowSpan,
                currentCell.ColSpan,
                currentCell.FirstRowIndex,
                currentCell.FirstColumnIndex));
        }
    }
}
```

**Εξήγηση:**
- **`IsMergedCell`:** Καθορίζει εάν ένα κελί αποτελεί μέρος μιας συγχωνευμένης ομάδας.
- **`RowSpan` και `ColSpan`:** Υποδεικνύει το εύρος του συγχωνευμένου κελιού σε γραμμές και στήλες, αντίστοιχα.
- **Αρχική θέση:** Προσδιορίζει από πού ξεκινά η συγχώνευση.

#### Συμβουλές αντιμετώπισης προβλημάτων

- Βεβαιωθείτε ότι η διαδρομή του αρχείου παρουσίασής σας είναι σωστή για να αποφύγετε σφάλματα "το αρχείο δεν βρέθηκε".
- Επαληθεύστε ότι η δομή του πίνακα στη διαφάνειά σας ταιριάζει με τις υποθέσεις σας (π.χ., είναι πράγματι το πρώτο σχήμα).

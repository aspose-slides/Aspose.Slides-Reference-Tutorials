## Οδηγός Εφαρμογής

### Δημιουργία πίνακα στο PowerPoint

#### Επισκόπηση
Αυτή η ενότητα καλύπτει τη δημιουργία ενός πίνακα μέσα σε μια διαφάνεια, επιτρέποντάς σας να ορίσετε προσαρμοσμένα πλάτη στηλών και ύψη γραμμών.

#### Βήμα 1: Ορισμός πλάτους στηλών και ύψους γραμμών
Καθορίστε τις διαστάσεις για στήλες και γραμμές:
```csharp
double[] dblCols = { 70, 70, 70, 70 }; // Πλάτος στηλών
double[] dblRows = { 70, 70, 70, 70 }; // Ύψη σειρών
```

#### Βήμα 2: Προσθήκη πίνακα στη διαφάνεια
Προσθέστε το σχήμα πίνακα στη διαφάνειά σας με καθορισμένες διαστάσεις:
```csharp
ISlide slide = presentation.Slides[0];
ITable table = slide.Shapes.AddTable(100, 50, dblCols, dblRows);
```
*Σημείωμα*: `100` και `50` είναι οι συντεταγμένες X και Y όπου τοποθετείται ο πίνακας.

#### Βήμα 3: Μορφοποίηση περιγραμμάτων πίνακα
Βελτιώστε την οπτική ελκυστικότητα μορφοποιώντας το περίγραμμα κάθε κελιού:
```csharp
foreach (IRow row in table.Rows) {
    foreach (ICell cell in row) {
        // Ορισμός ιδιοτήτων άνω περιγράμματος
        cell.CellFormat.BorderTop.FillFormat.FillType = FillType.Solid;
        cell.CellFormat.BorderTop.FillFormat.SolidFillColor.Color = Color.Red;
        cell.CellFormat.BorderTop.Width = 5;

        // Επαναλάβετε για κάτω, αριστερό και δεξί περίγραμμα
    }
}
```
*Γιατί*: Ρύθμιση `FillType` να `Solid` εξασφαλίζει ομοιόμορφη εμφάνιση περιγράμματος. Η ρύθμιση του χρώματος και του πλάτους επιτρέπει την προσαρμογή ανάλογα με την επωνυμία σας.

### Συμβουλές αντιμετώπισης προβλημάτων
- **Συνηθισμένο πρόβλημα**: Τα περιγράμματα δεν είναι ορατά.
  - *Διάλυμα*: Βεβαιωθείτε ότι έχετε ορίσει `BorderWidth` σε θετική τιμή μεγαλύτερη από το μηδέν.

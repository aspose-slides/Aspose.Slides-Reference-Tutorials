## Οδηγός Εφαρμογής

### Μορφοποίηση κειμένου σε γραμμές πίνακα

#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να βελτιώσετε την αναγνωσιμότητα και την ευθυγράμμιση του κειμένου μέσα στα κελιά του πίνακα. Θα επικεντρωθούμε στη ρύθμιση του ύψους της γραμματοσειράς, της ευθυγράμμισης του κειμένου, του δεξιού περιθωρίου και του κατακόρυφου προσανατολισμού του κειμένου.

#### Βήμα προς βήμα εφαρμογή

##### Ορισμός ύψους γραμματοσειράς για κελιά
1. **Αρχικοποίηση παρουσίασης**
   ```csharp
   using Aspose.Slides;
   
   Presentation presentation = new Presentation("YOUR_DOCUMENT_DIRECTORY\SomePresentationWithTable.pptx");
   ISlide slide = presentation.Slides[0];
   ITable someTable = slide.Shapes[0] as ITable; // Υποθέτοντας ότι το πρώτο σχήμα είναι ένας πίνακας
   ```

2. **Ρύθμιση ύψους γραμματοσειράς**
   ```csharp
   PortionFormat portionFormat = new PortionFormat();
   portionFormat.FontHeight = 25; // Ορίστε το επιθυμητό ύψος γραμματοσειράς
   someTable.Rows[0].SetTextFormat(portionFormat);
   ```
   - **Σκοπός**: Προσαρμόζει το μέγεθος της γραμματοσειράς μέσα στα κελιά του πίνακα για βελτιωμένη αναγνωσιμότητα.

##### Ρύθμιση στοίχισης κειμένου και δεξιού περιθωρίου
3. **Ρύθμιση παραμέτρων μορφής παραγράφου**
   ```csharp
   ParagraphFormat paragraphFormat = new ParagraphFormat();
   paragraphFormat.Alignment = TextAlignment.Right; // Στοίχιση κειμένου προς τα δεξιά
   paragraphFormat.MarginRight = 20; // Ορίστε ένα δεξί περιθώριο 20 μονάδων
   someTable.Rows[0].SetTextFormat(paragraphFormat);
   ```
   - **Σκοπός**: Παρέχει συνεπή ευθυγράμμιση και απόσταση μέσα στα κελιά.

##### Ορισμός κάθετου τύπου κειμένου
4. **Εφαρμογή κάθετης μορφοποίησης κειμένου**
   ```csharp
   TextFrameFormat textFrameFormat = new TextFrameFormat();
   textFrameFormat.TextVerticalType = TextVerticalType.Vertical; // Ορισμός κατακόρυφου προσανατολισμού κειμένου
   someTable.Rows[1].SetTextFormat(textFrameFormat);
   ```
   - **Σκοπός**: Χρήσιμο για τη δημιουργία μοναδικών σχεδίων και την εξοικονόμηση χώρου σε παρουσιάσεις.

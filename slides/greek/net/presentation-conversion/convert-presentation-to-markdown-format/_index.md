## 5. Διαμόρφωση επιλογών μετατροπής Markdown

Για να διαμορφώσουμε τις επιλογές μετατροπής Markdown, θα δημιουργήσουμε το MarkdownSaveOptions. Αυτό μας επιτρέπει να προσαρμόσουμε τον τρόπο δημιουργίας του εγγράφου Markdown. Για παράδειγμα, μπορούμε να καθορίσουμε εάν θα εξάγουμε γραφικά, να ορίσουμε τον φάκελο για την αποθήκευση εικόνων και να ορίσουμε τη βασική διαδρομή για τις εικόνες.

```csharp
string outPath = "Your Output Directory";

// Δημιουργήστε επιλογές δημιουργίας Markdown
MarkdownSaveOptions mdOptions = new MarkdownSaveOptions();

// Ορίστε την παράμετρο για την απόδοση όλων των στοιχείων
mdOptions.ExportType = MarkdownExportType.Visual;

// Ορισμός ονόματος φακέλου για αποθήκευση εικόνων
mdOptions.ImagesSaveFolderName = "md-images";

// Ορισμός διαδρομής για εικόνες φακέλων
mdOptions.BasePath = outPath;
```

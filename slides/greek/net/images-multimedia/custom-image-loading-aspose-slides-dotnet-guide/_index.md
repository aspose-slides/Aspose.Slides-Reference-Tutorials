## Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, θα σας καθοδηγήσουμε στην εφαρμογή της φόρτωσης προσαρμοσμένων εικόνων χρησιμοποιώντας επανακλήσεις. Θα αναλύσουμε τη διαδικασία σε διαχειρίσιμα βήματα.

### Προσαρμοσμένη φόρτωση πόρων για επανάκληση εικόνων

**Επισκόπηση:**
Αυτή η λειτουργία σάς επιτρέπει να αντικαταστήσετε εικόνες που λείπουν με προκαθορισμένα υποκατάστατα και να χειρίζεστε συγκεκριμένες μορφές εικόνας διαφορετικά κατά τη φόρτωση μιας παρουσίασης.

#### Βήμα 1: Δημιουργήστε μια κλάση ImageLoadingHandler

Ξεκινήστε ορίζοντας μια κλάση που υλοποιεί `IResourceLoadingCallback`Αυτό θα σας επιτρέψει να αναχαιτίσετε συμβάντα φόρτωσης πόρων:

```csharp
using Aspose.Slides;
using System.IO;

public class ImageLoadingHandler : IResourceLoadingCallback
{
    string dataDir = @"YOUR_DOCUMENT_DIRECTORY";

    public ResourceLoadingAction ResourceLoading(IResourceLoadingArgs args)
    {
        // Ελέγξτε αν η αρχική εικόνα είναι JPEG
        if (args.OriginalUri.EndsWith(".jpg"))
        {
            try // Προσπάθεια φόρτωσης μιας εικόνας αντικατάστασης
            {
                byte[] imageBytes = File.ReadAllBytes(Path.Combine(dataDir, "aspose-logo.jpg"));
                args.SetData(imageBytes); // Παρέχετε τα byte της υποκατάστατης εικόνας
                return ResourceLoadingAction.UserProvided; // Υποδεικνύει ότι η προσαρμοσμένη διαχείριση ήταν επιτυχής
            }
            catch (Exception)
            {
                return ResourceLoadingAction.Skip; // Παράλειψη εάν υπάρχει σφάλμα κατά τη φόρτωση της εικόνας
            }
        }
        else if (args.OriginalUri.EndsWith(".png"))
        {
            args.Uri = "http://www.google.com/images/logos/ps_logo2.png"; // Αντικατάσταση PNG με μια διεύθυνση URL
            return ResourceLoadingAction.Default; // Χρήση προεπιλεγμένου χειρισμού για το νέο URI
        }

        return ResourceLoadingAction.Skip; // Παράλειψη όλων των άλλων εικόνων
    }
}
```
**Εξήγηση:**
- **Λογική φόρτωσης πόρων**: Εάν λείπει μια εικόνα και είναι αρχείο JPEG, την αντικαθιστούμε με `aspose-logo.jpg`Για αρχεία PNG, ανακατευθύνουμε σε μια καθορισμένη διεύθυνση URL.
- **Χειρισμός σφαλμάτων**Σε περίπτωση προβλημάτων κατά τη φόρτωση της εικόνας υποκατάστασης, παρακάμπτουμε τον πόρο για να αποφύγουμε σφάλματα εφαρμογής.

#### Βήμα 2: Φόρτωση παρουσίασης με προσαρμοσμένες επιλογές

Στη συνέχεια, αρχικοποιήστε την παρουσίασή σας χρησιμοποιώντας τον προσαρμοσμένο χειριστή:

```csharp
using Aspose.Slides;
using System.IO;

string dataDir = @"YOUR_DOCUMENT_DIRECTORY";
LoadOptions opts = new LoadOptions();
opts.ResourceLoadingCallback = new ImageLoadingHandler();

Presentation presentation = new Presentation(Path.Combine(dataDir, "presentation.pptx"), opts);
```
**Εξήγηση:**
- **Επιλογές Φόρτωσης**: Ρυθμίζει τον τρόπο φόρτωσης της παρουσίασης. Ορίζοντας `ResourceLoadingCallback`, μπορείτε να προσαρμόσετε τη φόρτωση της εικόνας.
- **Αρχικοποίηση παρουσίασης**: Το `Presentation` Το αντικείμενο δημιουργείται με μια διαδρομή προς το αρχείο PPTX και προσαρμοσμένες επιλογές φόρτωσης.

### Συμβουλές αντιμετώπισης προβλημάτων

- Βεβαιωθείτε ότι οι εικόνες αντικατάστασης έχουν τοποθετηθεί σωστά `YOUR_DOCUMENT_DIRECTORY`.
- Επαληθεύστε την πρόσβαση στο δίκτυο εάν αντικαθιστάτε εικόνες με URL από τον ιστό.
- Ελέγξτε τα αρχεία καταγραφής εξαιρέσεων για λεπτομερή μηνύματα σφάλματος κατά την ανάπτυξη.

## Οδηγός Εφαρμογής
Τώρα, ας εξερευνήσουμε τις λεπτομέρειες υλοποίησης για την εξαγωγή βίντεο και ήχων από παρουσιάσεις PowerPoint.

### Εξαγωγή βίντεο από παρουσίαση
#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να εξαγάγετε αρχεία βίντεο που είναι ενσωματωμένα σε μια παρουσίαση PowerPoint χωρίς να φορτώσετε ολόκληρο το αρχείο στη μνήμη, βελτιστοποιώντας την απόδοση.

#### Οδηγός βήμα προς βήμα
**1. Ρύθμιση επιλογών φόρτωσης**
```csharp
LoadOptions loadOptions = new LoadOptions
{
    BlobManagementOptions =
    {
        PresentationLockingBehavior = PresentationLockingBehavior.KeepLocked,
    }
};
```
Ο `PresentationLockingBehavior.KeepLocked` Η επιλογή αποτρέπει τη φόρτωση ολόκληρου του αρχείου στη μνήμη, κάτι που είναι κρίσιμο για τον χειρισμό μεγάλων παρουσιάσεων.

**2. Πρόσβαση και εξαγωγή βίντεο**
```csharp
using (Presentation pres = new Presentation(hugePresentationWithAudiosAndVideosFile, loadOptions))
{
    byte[] buffer = new byte[8 * 1024]; // Μέγεθος buffer 8KB

    for (var index = 0; index < pres.Videos.Count; index++)
    {
        IVideo video = pres.Videos[index];

        using (Stream presVideoStream = video.GetStream())
        {
            using (FileStream outputFileStream = File.OpenWrite($"video{index}.avi"))
            {
                int bytesRead;
                while ((bytesRead = presVideoStream.Read(buffer, 0, buffer.Length)) > 0)
                {
                    outputFileStream.Write(buffer, 0, bytesRead);
                }
            }
        }
    }
}
```
**Εξήγηση:**
- **Μέγεθος buffer**Χρησιμοποιούμε ένα buffer 8KB για την ανάγνωση και την εγγραφή δεδομένων σε τμήματα, ελαχιστοποιώντας τη χρήση μνήμης.
- **Βρόχος εξαγωγής βίντεο**: Επαναλαμβάνει κάθε βίντεο που είναι ενσωματωμένο στην παρουσίαση, το εξάγει ως ροή και το εγγράφει σε ένα αρχείο.

#### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι έχετε τα κατάλληλα δικαιώματα ανάγνωσης/εγγραφής για τον κατάλογο προορισμού σας.
- Βεβαιωθείτε ότι η διαδρομή του αρχείου παρουσίασής σας είναι σωστή και προσβάσιμη.

### Εξαγωγή ήχου από παρουσίαση
#### Επισκόπηση
Όπως και με τα βίντεο, αυτή η λειτουργία επιτρέπει την αποτελεσματική εξαγωγή αρχείων ήχου που είναι ενσωματωμένα σε παρουσιάσεις PowerPoint.

#### Οδηγός βήμα προς βήμα
**1. Ρύθμιση επιλογών φόρτωσης**
Αυτό το βήμα παραμένει ίδιο με τη διαδικασία εξαγωγής βίντεο:
```csharp
LoadOptions loadOptions = new LoadOptions
{
    BlobManagementOptions =
    {
        PresentationLockingBehavior = PresentationLockingBehavior.KeepLocked,
    }
};
```
**2. Πρόσβαση και εξαγωγή ήχων**
```csharp
using (Presentation pres = new Presentation(hugePresentationWithAudiosAndVideosFile, loadOptions))
{
    byte[] buffer = new byte[8 * 1024]; // Μέγεθος buffer 8KB

    for (var index = 0; index < pres.Audios.Count; index++)
    {
        IAudio audio = pres.Audios[index];

        using (Stream presAudioStream = audio.GetStream())
        {
            using (FileStream outputFileStream = File.OpenWrite($"audio{index}.wav"))
            {
                int bytesRead;
                while ((bytesRead = presAudioStream.Read(buffer, 0, buffer.Length)) > 0)
                {
                    outputFileStream.Write(buffer, 0, bytesRead);
                }
            }
        }
    }
}
```
**Εξήγηση:**
Η λογική υλοποίησης αντικατοπτρίζει αυτήν της εξαγωγής βίντεο. Επαναλαμβάνει τα αρχεία ήχου και τα εγγράφει στο δίσκο χρησιμοποιώντας μια προσέγγιση buffered.

#### Συμβουλές αντιμετώπισης προβλημάτων
- Επιβεβαιώστε ότι οι διαδρομές των αρχείων ήχου σας έχουν οριστεί σωστά.
- Βεβαιωθείτε ότι υπάρχει επαρκής χώρος αποθήκευσης για τα εξαγόμενα αρχεία ήχου.

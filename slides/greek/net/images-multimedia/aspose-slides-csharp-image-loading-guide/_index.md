## Οδηγός Εφαρμογής

Τώρα που έχετε ρυθμίσει το περιβάλλον σας, ας υλοποιήσουμε τη λειτουργικότητα φόρτωσης και εμφάνισης εικόνων.

### Δυνατότητα: Φόρτωση και εμφάνιση εικόνων σε μια παρουσίαση

Αυτή η λειτουργία δείχνει πώς να φορτώσετε μια εικόνα από το σύστημα αρχείων και να την προσθέσετε ως πλαίσιο εικόνας στην πρώτη διαφάνεια μιας παρουσίασης χρησιμοποιώντας το Aspose.Slides για .NET.

#### Επισκόπηση:
Σε αυτήν την ενότητα, θα δούμε τα βήματα για τη φόρτωση μιας εικόνας, την εισαγωγή της σε μια διαφάνεια και την αποθήκευση της παρουσίασής σας.

**Βήμα 1: Δημιουργία καταλόγων**
Ορίστε διαδρομές για τον κατάλογο εγγράφων και τον κατάλογο εξόδου. Εάν δεν υπάρχουν, δημιουργήστε τες χρησιμοποιώντας:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Ορίστε εδώ τη διαδρομή του καταλόγου εγγράφων σας
string outputDir = "YOUR_OUTPUT_DIRECTORY"; // Ορίστε εδώ τη διαδρομή του καταλόγου εξόδου σας

// Δημιουργήστε τον κατάλογο δεδομένων εάν δεν υπάρχει.
if (!Directory.Exists(dataDir))
{
    Directory.CreateDirectory(dataDir);
}
```

**Βήμα 2: Φόρτωση και εισαγωγή εικόνας**
Δημιουργήστε μια νέα παρουσία παρουσίασης και αποκτήστε πρόσβαση στην πρώτη της διαφάνεια. Στη συνέχεια, φορτώστε μια εικόνα από το σύστημα αρχείων:
```csharp
using (Presentation pres = new Presentation())
{
    // Πρόσβαση στην πρώτη διαφάνεια της παρουσίασης
    ISlide sld = pres.Slides[0];

    // Φόρτωση μιας εικόνας από το σύστημα αρχείων και προσθήκη της στη συλλογή εικόνων της παρουσίασης
    IImage img = Images.FromFile(Path.Combine(dataDir, "aspose-logo.jpg"));
    IPPImage imgx = pres.Images.AddImage(img);

    // Προσθέστε μια κορνίζα με διαστάσεις που ταιριάζουν με αυτές της εικόνας που έχετε φορτώσει
    sld.Shapes.AddPictureFrame(ShapeType.Rectangle, 50, 150, imgx.Width, imgx.Height, imgx);
}
```

**Βήμα 3: Αποθήκευση της παρουσίασης**
Τέλος, αποθηκεύστε την τροποποιημένη παρουσίασή σας στο δίσκο σε μορφή PPTX:
```csharp
pres.Save(Path.Combine(outputDir, "AddStretchOffsetForImageFill_out.pptx"), SaveFormat.Pptx);
```

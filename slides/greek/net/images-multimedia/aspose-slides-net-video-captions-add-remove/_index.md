## Οδηγός Εφαρμογής
Αυτή η ενότητα θα σας καθοδηγήσει στην προσθήκη λεζάντων σε καρέ βίντεο και στην εξαγωγή ή αφαίρεσή τους. Κάθε λειτουργία περιγράφεται λεπτομερώς παρακάτω.

### Χαρακτηριστικό 1: Προσθήκη λεζάντων σε ένα VideoFrame

#### Επισκόπηση
Αυτή η λειτουργία βοηθά στην εισαγωγή λεζάντων από ένα εξωτερικό αρχείο (όπως το VTT) σε ένα καρέ βίντεο, βελτιώνοντας την προσβασιμότητα για το κοινό σας.

#### Βήματα Υλοποίησης
**Βήμα 1: Προετοιμασία των αρχείων σας**
Βεβαιωθείτε ότι έχετε το βίντεο (`sample_bunny.mp4`) και αρχεία υποτίτλων (`bunny.vtt`).

```csharp
string mediaFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample_bunny.mp4");
string trackFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "bunny.vtt");
```

**Βήμα 2: Προσθήκη βίντεο στην παρουσίαση**
Δημιουργήστε ένα `Presentation` αντιταχθείτε και προσθέστε το βίντεό σας.

```csharp
using (Presentation pres = new Presentation())
{
    IVideo video = pres.Videos.AddVideo(File.ReadAllBytes(mediaFile));
    var videoFrame = pres.Slides[0].Shapes.AddVideoFrame(0, 0, 100, 100, video);
```

**Βήμα 3: Προσθήκη κομματιού υποτίτλων**
Επισυνάψτε το αρχείο υποτίτλων στο καρέ του βίντεο.

```csharp
videoFrame.CaptionTracks.Add("New track", trackFile);
pres.Save(Path.Combine("YOUR_OUTPUT_DIRECTORY", "VideoCaptionAdd_out.pptx"), SaveFormat.Pptx);
}
```

#### Παράμετροι και Σκοποί Μεθόδου
- `Presentation`: Αντιπροσωπεύει μια παρουσίαση PowerPoint.
- `IVideo` και `IVideoFrame`: Αναπαριστούν το περιεχόμενο βίντεο και το πλαίσιό του μέσα σε διαφάνειες, αντίστοιχα.
- `captionTracks.Add()`: Προσθέτει υπότιτλους στο καθορισμένο κομμάτι.

### Λειτουργία 2: Εξαγωγή και αφαίρεση λεζάντων από ένα VideoFrame

#### Επισκόπηση
Αφού προσθέσετε λεζάντες, ενδέχεται να υπάρχουν περιπτώσεις όπου θα πρέπει να τις εξαγάγετε ή να τις αφαιρέσετε. Αυτή η λειτουργία εστιάζει στον τρόπο αποτελεσματικής εκτέλεσης και των δύο εργασιών.

#### Βήματα Υλοποίησης
**Βήμα 1: Φόρτωση παρουσίασης**
Ανοίξτε την παρουσίαση που περιέχει το βίντεό σας με λεζάντες.

```csharp
string outAddPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "VideoCaptionAdd_out.pptx");
using (Presentation pres = new Presentation(outAddPath))
{
    IVideoFrame videoFrame = pres.Slides[0].Shapes[0] as VideoFrame;
```

**Βήμα 2: Εξαγωγή λεζάντων**
Εξαγωγή δυαδικών δεδομένων από λεζάντες και αποθήκευσή τους σε ένα αρχείο.

```csharp
if (videoFrame != null)
{
    foreach (var captionTrack in videoFrame.CaptionTracks) 
    {
        File.WriteAllBytes(Path.Combine("YOUR_OUTPUT_DIRECTORY", "Caption_out.vtt"), captionTrack.BinaryData);
    }
```

**Βήμα 3: Αφαίρεση λεζάντων**
Διαγράψτε όλους τους υπότιτλους από το VideoFrame.

```csharp
videoFrame.CaptionTracks.Clear();
pres.Save(Path.Combine("YOUR_OUTPUT_DIRECTORY", "VideoCaptionRemove_out.pptx"), SaveFormat.Pptx);
}
```

#### Παράμετροι και Σκοποί Μεθόδου
- `BinaryData`: Αντιπροσωπεύει τα δεδομένα λεζάντας σε δυαδική μορφή.
- `CaptionTracks.Clear()`: Αφαιρεί όλους τους υπότιτλους από το καρέ του βίντεο.

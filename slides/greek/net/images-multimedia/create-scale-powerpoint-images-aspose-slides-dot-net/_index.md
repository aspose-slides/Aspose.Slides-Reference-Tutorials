Αυτή η λειτουργία σάς επιτρέπει να δημιουργείτε εικόνες διαφανειών παρουσίασης σε προσαρμοσμένα μεγέθη, κάτι που είναι απαραίτητο για σκοπούς προβολής ή ενσωμάτωσης εφαρμογών.

```csharp
    public class ThumbnailWithUserDefinedDimensions

            string dataDir = "YOUR_DOCUMENT_DIRECTORY";

            {
                // Περαιτέρω βήματα θα ακολουθήσουν εδώ...
```

#### Βήμα 2: Πρόσβαση στην επιθυμητή διαφάνεια
Αποκτήστε πρόσβαση στη διαφάνεια που θέλετε να μετατρέψετε:
```csharp
// Πρόσβαση στην πρώτη διαφάνεια
ISlide sld = pres.Slides[0];
```

#### Βήμα 3: Ορισμός διαστάσεων και υπολογισμός συντελεστών κλιμάκωσης
Ορίστε τις επιθυμητές διαστάσεις εικόνας και, στη συνέχεια, υπολογίστε τους συντελεστές κλιμάκωσης:
```csharp
int desiredX = 1200;
int desiredY = 800;

float ScaleX = (float)(1.0 / pres.SlideSize.Size.Width) * desiredX;
float ScaleY = (float)(1.0 / pres.SlideSize.Size.Height) * desiredY;
```

#### Βήμα 4: Δημιουργία και αποθήκευση της κλιμακωτής εικόνας
Δημιουργήστε την εικόνα από τη διαφάνειά σας χρησιμοποιώντας συντελεστές κλιμάκωσης:
```csharp
IImage img = sld.GetThumbnail(ScaleX, ScaleY);

string outputDir = "YOUR_OUTPUT_DIRECTORY";
Directory.CreateDirectory(outputDir); // Βεβαιωθείτε ότι υπάρχει κατάλογος
img.Save(Path.Combine(outputDir, "Thumbnail2_out.jpg"), System.Drawing.Imaging.ImageFormat.Jpeg);
```

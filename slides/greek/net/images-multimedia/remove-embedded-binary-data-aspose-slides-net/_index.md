## Οδηγός Εφαρμογής
### Αφαίρεση ενσωματωμένων δυαδικών δεδομένων
#### Επισκόπηση
Η αφαίρεση ενσωματωμένων δυαδικών δεδομένων από ένα αρχείο PPTX μειώνει το μέγεθος και την πολυπλοκότητα του αρχείου, κάτι που είναι απαραίτητο για παρουσιάσεις που περιέχουν περιττά ή παρωχημένα ενσωματωμένα αρχεία.

**Βήματα Υλοποίησης:**
1. **Ορισμός διαδρομών αρχείων:** Καθορίστε τους καταλόγους εισόδου και εξόδου.
   ```csharp
   string pptxFileName = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "OlePptx.pptx");
   string outPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "OlePptx-out.pptx");
   ```
2. **Ορισμός επιλογών φόρτωσης:** Ρυθμίστε τις παραμέτρους φόρτωσης για να διαγράψετε ενσωματωμένα δυαδικά αντικείμενα.
   ```csharp
   LoadOptions loadOption = new LoadOptions { DeleteEmbeddedBinaryObjects = true };
   ```
3. **Φόρτωση και αποθήκευση παρουσίασης:**
   ```csharp
   using (Presentation pres = new Presentation(pptxFileName, loadOption))
   {
       // Καταμέτρηση πλαισίων OLE πριν από την αποθήκευση
       int emptyOleFrames;
       int oleFramesCount = GetOleObjectFrameCount(pres.Slides, out emptyOleFrames);

       // Αποθήκευση της παρουσίασης χωρίς ενσωματωμένα δεδομένα
       pres.Save(outPath, SaveFormat.Pptx);
       
       using (Presentation outPres = new Presentation(outPath))
       {
           // Επαλήθευση πλαισίων OLE μετά την αποθήκευση
           oleFramesCount = GetOleObjectFrameCount(outPres.Slides, out emptyOleFrames);
       }
   }
   ```
4. **Βοηθητική μέθοδος:**
   ```csharp
   private static int GetOleObjectFrameCount(ISlideCollection slides, out int emptyOleFrames)
   {
       int oleFramesCount = 0;
       emptyOleFrames = 0;

       foreach (ISlide sld in slides)
       {
           foreach (IShape shape in sld.Shapes)
           {
               OleObjectFrame objectFrame = shape as OleObjectFrame;
               if (objectFrame == null) continue;

               oleFramesCount++;
               byte[] embeddedData = objectFrame.EmbeddedData?.EmbeddedFileData;
               if (embeddedData == null || embeddedData.Length == 0)
                   emptyOleFrames++;
           }
       }

       return oleFramesCount;
   }
   ```
**Εξήγηση:**
- **Επιλογές Φόρτωσης:** Ρυθμίζει τον τρόπο φόρτωσης της παρουσίασης, με `DeleteEmbeddedBinaryObjects` οριστεί σε true.
- **Τάξη Παρουσίασης:** Διαχειρίζεται τη φόρτωση και την αποθήκευση αρχείων PPTX.
- **Μέθοδος GetOleObjectFrameCount:** Μετράει τα πλαίσια OLE στις διαφάνειες, βοηθώντας στην επαλήθευση εάν έχουν αφαιρεθεί ενσωματωμένα δεδομένα.

**Συμβουλές αντιμετώπισης προβλημάτων:**
- Βεβαιωθείτε ότι έχουν καθοριστεί οι σωστές διαδρομές αρχείων.
- Επιβεβαιώστε ότι η παρουσίαση περιέχει αντικείμενα OLE πριν από την επεξεργασία.
- Χειριστείτε εξαιρέσεις κατά τη διάρκεια των λειτουργιών εισόδου/εξόδου αρχείων για να αποτρέψετε σφάλματα.

## Οδηγός Εφαρμογής

### Προσθήκη εικόνας SVG από εξωτερικό πόρο

#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να προσθέσετε μια κλιμακούμενη εικόνα διανυσματικού γραφικού (SVG) στην παρουσίαση του PowerPoint σας, εξασφαλίζοντας γραφικά υψηλής ποιότητας που παραμένουν ευκρινή σε οποιοδήποτε μέγεθος.

#### Βήμα προς βήμα εφαρμογή
**1. Διαβάστε το περιεχόμενο SVG:**
Ξεκινήστε διαβάζοντας το περιεχόμενο SVG από ένα εξωτερικό αρχείο:
```csharp
string svgContent = File.ReadAllText(Path.Combine(dataDir, "image1.svg"));
```
Αυτό το βήμα διασφαλίζει ότι έχετε τα ακατέργαστα διανυσματικά δεδομένα που χρειάζεστε για ενσωμάτωση στη διαφάνειά σας.

**2. Δημιουργήστε μια παρουσία SvgImage:**
Δημιουργήστε μια παρουσία του `SvgImage` χρησιμοποιώντας το περιεχόμενο SVG και έναν προσαρμοσμένο αναλυτή για τυχόν εξωτερικούς πόρους:
```csharp
ISvgImage svgImage = new SvgImage(svgContent, new ExternalResourceResolver(), dataDir);
```
Αυτό επιτρέπει τον χειρισμό εικόνων ή στυλ που αναφέρονται στο SVG σας.

**3. Αρχικοποίηση αντικειμένου παρουσίασης:**
Ανοίξτε ή δημιουργήστε μια παρουσίαση PowerPoint για να εργαστείτε με διαφάνειες:
```csharp
using (var p = new Presentation())
{
    // Ο κώδικας συνεχίζεται...
}
```

**4. Προσθήκη της εικόνας στη διαφάνεια:**
Προσθέστε την εικόνα SVG στη συλλογή εικόνων της παρουσίασής σας και εισαγάγετέ την ως πλαίσιο εικόνας στην πρώτη διαφάνεια:
```csharp
IPPImage ppImage = p.Images.AddImage(svgImage);
p.Slides[0].Shapes.AddPictureFrame(ShapeType.Rectangle, 0, 0, ppImage.Width, ppImage.Height, ppImage);
```
Αυτό το βήμα τοποθετεί την εικόνα SVG σας σε μια διαφάνεια στις αρχικές της διαστάσεις.

**5. Αποθηκεύστε την παρουσίαση:**
Τέλος, αποθηκεύστε την παρουσίασή σας με την εικόνα που μόλις προσθέσατε:
```csharp
p.Save(outPptxPath, SaveFormat.Pptx);
```

#### Επισκόπηση
Υλοποιώντας ένα `ExternalResourceResolver` σας επιτρέπει να χειρίζεστε δυναμικά τυχόν εξωτερικούς πόρους που απαιτούνται από το περιεχόμενο SVG.

**1. Ορίστε την Κλάση Επίλυσης:**
Δημιουργήστε μια κλάση που υλοποιεί `IExternalResourceResolver`:
```csharp
class ExternalResourceResolver : IExternalResourceResolver
{
    public Uri ResolveUri(Uri baseUri, string path)
    {
        // Υλοποιήστε λογική για να επιλύσετε και να επιστρέψετε το URI ενός εξωτερικού πόρου.
        throw new NotImplementedException();
    }
}
```
Αυτή η κλάση λειτουργεί ως placeholder όπου μπορείτε αργότερα να ορίσετε τον τρόπο με τον οποίο η εφαρμογή σας επιλύει εξωτερικούς πόρους.

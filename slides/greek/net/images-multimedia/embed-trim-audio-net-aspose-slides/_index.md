## Οδηγός Εφαρμογής

### Προσθήκη ηχητικού πλαισίου με ενσωματωμένο ήχο

#### Επισκόπηση
Ενσωματώστε αρχεία ήχου απευθείας στις διαφάνειες της παρουσίασής σας για μια απρόσκοπτη εμπειρία προβολής.

#### Βήματα:
1. **Αρχικοποίηση παρουσίασης**
   Δημιουργήστε ένα νέο `Presentation` έχετε αντίρρηση να κρατάτε διαφάνειες και πολυμέσα.
   ```csharp
   using Aspose.Slides;
   string mediaFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "audio.m4a");
   string outPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "AudioFrame_out.pptx");
   using (Presentation pres = new Presentation())
   ```
2. **Προσθήκη ήχου στη συλλογή**
   Χρήση `pres.Audios.AddAudio` για να προσθέσετε το αρχείο ήχου σας.
   ```csharp
   IAudio audio = pres.Audios.AddAudio(File.ReadAllBytes(mediaFile));
   ```
3. **Ενσωματώστε το ηχητικό πλαίσιο**
   Προσθέστε ένα ενσωματωμένο ηχητικό πλαίσιο στην πρώτη διαφάνεια.
   ```csharp
   IAudioFrame audioFrame = pres.Slides[0].Shapes.AddAudioFrameEmbedded(50, 50, 100, 100, audio);
   ```
4. **Αποθήκευση της παρουσίασης**
   Αποθηκεύστε την παρουσίασή σας με το ενσωματωμένο ηχητικό πλαίσιο.
   ```csharp
   pres.Save(outPath, SaveFormat.Pptx);
   ```

### Ρύθμιση χρόνων περικοπής ήχου

#### Επισκόπηση
Καθορίστε ποιο τμήμα ενός αρχείου ήχου θα πρέπει να αναπαραχθεί σε μια παρουσίαση.

#### Βήματα:
1. **Αρχικοποίηση παρουσίασης**
   Όπως και με την προσθήκη ενός ηχητικού καρέ, ξεκινήστε δημιουργώντας ένα νέο `Presentation` αντικείμενο.
   ```csharp
   using Aspose.Slides;
   string mediaFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "audio.m4a");
   string outPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "AudioFrameTrim_out.pptx");
   using (Presentation pres = new Presentation())
   ```
2. **Προσθήκη ήχου και ενσωμάτωση πλαισίου**
   Προσθέστε τον ήχο στη συλλογή και ενσωματώστε τον σε μια διαφάνεια όπως πριν.
   ```csharp
   IAudio audio = pres.Audios.AddAudio(File.ReadAllBytes(mediaFile));
   IAudioFrame audioFrame = pres.Slides[0].Shapes.AddAudioFrameEmbedded(50, 50, 100, 100, audio);
   ```
3. **Περικοπή έναρξης και τέλους ήχου**
   Ορίστε τις ώρες έναρξης και λήξης για το ηχητικό σας κλιπ.
   ```csharp
   // Περικοπή από την αρχή στα 500ms (0,5 δευτερόλεπτα)
   audioFrame.TrimFromStart = 500f;
   
   // Περικοπή στο τέλος στα 1000ms (1 δευτερόλεπτο)
   audioFrame.TrimFromEnd = 1000f;
   ```
4. **Αποθήκευση παρουσίασης**
   Αποθηκεύστε την παρουσίασή σας με τον περικομμένο ήχο.
   ```csharp
   pres.Save(outPath, SaveFormat.Pptx);
   ```

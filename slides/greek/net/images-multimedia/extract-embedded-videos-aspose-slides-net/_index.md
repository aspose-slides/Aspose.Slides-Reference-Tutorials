## Οδηγός Εφαρμογής
### Εξαγωγή ενσωματωμένων βίντεο από το PowerPoint
Αυτή η λειτουργία σάς επιτρέπει να εξαγάγετε βίντεο ενσωματωμένα στις διαφάνειες του PowerPoint σας. Ας αναλύσουμε τα βήματα:
#### Επισκόπηση της λειτουργίας
Θα επαναλάβουμε την επεξεργασία κάθε διαφάνειας και σχήματος, ελέγχοντας για καρέ βίντεο και, στη συνέχεια, θα εξαγάγουμε και θα αποθηκεύσουμε το βίντεο.
#### Βήμα προς βήμα εφαρμογή
##### 1. Φόρτωση της παρουσίασης
Ξεκινήστε φορτώνοντας το αρχείο παρουσίασης χρησιμοποιώντας το Aspose.Slides.
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
Presentation presentation = new Presentation(dataDir + "/Video.pptx");
```
##### 2. Επαναλάβετε τις διαφάνειες και τα σχήματα
Περιηγηθείτε σε κάθε διαφάνεια και, στη συνέχεια, σε κάθε σχήμα μέσα στις διαφάνειες για να βρείτε καρέ βίντεο.
```csharp
foreach (ISlide slide in presentation.Slides) {
    foreach (IShape shape in slide.Shapes) {
        if (shape is VideoFrame) {
            // Επεξεργασία καρέ βίντεο
        }
    }
}
```
##### 3. Αναγνώριση και εξαγωγή βίντεο
Ελέγξτε αν το σχήμα είναι `VideoFrame`, εξαγάγετε το περιεχόμενό του και αποθηκεύστε το.
```csharp
if (shape is VideoFrame vf) {
    String type = vf.EmbeddedVideo.ContentType;
    int ss = type.LastIndexOf('/');
    type = type.Remove(0, ss + 1);
    Byte[] buffer = vf.EmbeddedVideo.BinaryData;

    using (FileStream stream = new FileStream("YOUR_OUTPUT_DIRECTORY/NewVideo_out." + type, FileMode.Create, FileAccess.Write, FileShare.Read)) {
        stream.Write(buffer, 0, buffer.Length);
    }
}
```
**Εξήγηση:**
- **Τύπος περιεχομένου**: Καθορίζει την επέκταση αρχείου του βίντεο.
- **Δυαδικά Δεδομένα**: Περιέχει τα ακατέργαστα δεδομένα βίντεο για εξαγωγή.
##### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι οι διαδρομές του καταλόγου σας έχουν οριστεί σωστά για να αποφύγετε `FileNotFoundException`.
- Εάν δεν εξάγονται βίντεο, επαληθεύστε ότι τα σχήματα είναι όντως `VideoFrame` περιπτώσεις.

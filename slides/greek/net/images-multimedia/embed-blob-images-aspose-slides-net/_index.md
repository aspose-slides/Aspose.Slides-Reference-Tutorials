## Οδηγός Εφαρμογής

Αυτή η ενότητα περιγράφει τα βήματα που απαιτούνται για την αποτελεσματική προσθήκη μιας εικόνας blob στην παρουσίαση του PowerPoint.

### Προσθήκη εικόνας ως κηλίδα

#### Επισκόπηση
Η ενσωμάτωση μεγάλων εικόνων απευθείας από δυαδικά δεδομένα χωρίς την ανάγκη προσωρινών αρχείων είναι ιδιαίτερα χρήσιμη για εφαρμογές που χειρίζονται ευαίσθητα ή μεγάλης κλίμακας οπτικά δεδομένα.

#### Βήμα προς βήμα εφαρμογή

##### 1. Ορισμός καταλόγου εγγράφων και διαδρομής εικόνας
Ξεκινήστε καθορίζοντας πού θα αποθηκευτεί η εικόνα και η παρουσίασή σας:
```csharp
string dataDir = \@"YOUR_DOCUMENT_DIRECTORY";
string pathToLargeImage = Path.Combine(dataDir, "large_image.jpg");
```
**Εξήγηση**: `dataDir` είναι ο κατάλογος για την αποθήκευση εικόνων και παρουσιάσεων. `pathToLargeImage` συνδυάζει αυτόν τον κατάλογο με το όνομα του αρχείου εικόνας σας.

##### 2. Δημιουργήστε μια νέα παρουσία παρουσίασης
Δημιουργήστε ένα νέο αντικείμενο παρουσίασης για να αποθηκεύσετε τις διαφάνειές σας:
```csharp
using (Presentation pres = new Presentation())
{
    // Ο κώδικας θα μπει εδώ
}
```
**Εξήγηση**: Το `Presentation` Η κλάση αντιπροσωπεύει ολόκληρο το έγγραφο του PowerPoint, επιτρέποντάς σας να προσθέσετε ή να τροποποιήσετε διαφάνειες.

##### 3. Ανοίξτε το αρχείο εικόνας ως ροή και προσθέστε εικόνα
Χρησιμοποιήστε μια ροή αρχείων για να ανοίξετε την εικόνα σας και να την προσθέσετε ως εικόνα στην παρουσίαση:
```csharp
using (FileStream fileStream = new FileStream(pathToLargeImage, FileMode.Open))
{
    IPPImage img = pres.Images.AddImage(fileStream, LoadingStreamBehavior.KeepLocked);
}
```
**Εξήγηση**: `AddImage` προσθέτει την εικόνα στην εσωτερική συλλογή εικόνων της παρουσίασής σας. `LoadingStreamBehavior.KeepLocked` διασφαλίζει ότι το ρέμα δεν θα κλείσει ή δεν θα απορριφθεί αμέσως.

##### 4. Προσθήκη Πλαισίου Εικόνας σε Σλάιντ
Ενσωματώστε την εικόνα σε μια διαφάνεια προσθέτοντας ένα πλαίσιο εικόνας:
```csharp
pres.Slides[0].Shapes.AddPictureFrame(ShapeType.Rectangle, 0, 0, 300, 200, img);
```
**Εξήγηση**Αυτή η γραμμή προσθέτει ένα ορθογώνιο πλαίσιο στην πρώτη διαφάνεια (`Slides[0]`) σε καθορισμένες συντεταγμένες και διαστάσεις.

##### 5. Αποθήκευση παρουσίασης
Τέλος, αποθηκεύστε την παρουσίασή σας στο δίσκο:
```csharp
pres.Save(Path.Combine(dataDir, "presentationWithLargeImage.pptx"), SaveFormat.Pptx);
```
**Εξήγηση**: Το `Save` Η μέθοδος γράφει την τροποποιημένη παρουσίαση πίσω στον δίσκο σε μορφή PPTX.

#### Συμβουλές αντιμετώπισης προβλημάτων:
- **Εξαίρεση "Δεν βρέθηκε αρχείο"**Βεβαιωθείτε ότι η διαδρομή της εικόνας είναι σωστή και προσβάσιμη.
- **Προβλήματα μνήμης**Όταν εργάζεστε με μεγάλες εικόνες, σκεφτείτε να βελτιστοποιήσετε τη χρήση μνήμης του συστήματός σας ή να προσαρμόσετε τις ρυθμίσεις ροής για μεγαλύτερη αποτελεσματικότητα.

# Δημιουργία γραφημάτων ομαδοποιημένων στηλών με το Aspose.Slides για .NET

Το Aspose.Slides για .NET παρέχει ισχυρές δυνατότητες για τη δημιουργία και τον χειρισμό παρουσιάσεων PowerPoint μέσω προγραμματισμού. Αυτό το παράδειγμα δείχνει πώς να δημιουργήσετε ένα γράφημα στηλών σε ομαδοποίηση με προσαρμοσμένες κατηγορίες και σειρές δεδομένων.

## Εγκατάσταση

```csharp
// Εγκατάσταση μέσω του NuGet Package Manager
Install-Package Aspose.Slides
```

Ή προσθέστε την ακόλουθη αναφορά στο αρχείο του έργου σας:

```xml
<PackageReference Include="Aspose.Slides" Version="23.2.0" />
```

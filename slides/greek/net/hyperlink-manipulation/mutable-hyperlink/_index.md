## Δημιουργία μεταβλητής υπερσύνδεσης

### Βήμα 1: Ρύθμιση του έργου σας
Πρώτα, δημιουργήστε ένα νέο έργο ή ανοίξτε ένα υπάρχον στο IDE σας. Βεβαιωθείτε ότι έχετε αναφερθεί σωστά στο Aspose.Slides για .NET στο έργο σας.

### Βήμα 2: Εισαγωγή χώρων ονομάτων
Στο αρχείο κώδικα, εισαγάγετε τους απαραίτητους χώρους ονομάτων για εργασία με το Aspose.Slides:

```csharp
using Aspose.Slides;
using Aspose.Slides.Export;
using Aspose.Slides.Shape;
```

### Βήμα 3: Δημιουργήστε μια νέα παρουσίαση
Για να δημιουργήσετε μια νέα παρουσίαση PowerPoint, χρησιμοποιήστε τον ακόλουθο κώδικα:

```csharp
string dataDir = "Your Document Directory";

using (Presentation presentation = new Presentation())
{
    // Ο κώδικάς σας για τη δημιουργία και τον χειρισμό της παρουσίασης βρίσκεται εδώ
    presentation.Save(dataDir + "presentation-out.pptx", SaveFormat.Pptx);
}
```

### Βήμα 4: Προσθήκη σχήματος υπερσύνδεσης
Τώρα, ας προσθέσουμε ένα σχήμα στην παρουσίασή σας με έναν υπερσύνδεσμο. Σε αυτό το παράδειγμα, θα δημιουργήσουμε ένα ορθογώνιο σχήμα με έναν υπερσύνδεσμο στον ιστότοπο Aspose:

```csharp
IAutoShape shape1 = presentation.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 100, 100, 600, 50, false);
shape1.AddTextFrame("Aspose: File Format APIs");
shape1.TextFrame.Paragraphs[0].Portions[0].PortionFormat.HyperlinkClick = new Hyperlink("https://www.aspose.com/");
shape1.TextFrame.Paragraphs[0].Portions[0].PortionFormat.HyperlinkClick.Tooltip = "More than 70% Fortune 100 companies trust Aspose APIs";
shape1.TextFrame.Paragraphs[0].Portions[0].PortionFormat.FontHeight = 32;
```

Σε αυτό το βήμα, προσθέσαμε ένα ορθογώνιο σχήμα με το κείμενο "Aspose: File Format APIs" και έναν υπερσύνδεσμο με δυνατότητα κλικ. Μπορείτε να προσαρμόσετε το σχήμα, το κείμενο και την υπερσύνδεση σύμφωνα με τις ανάγκες σας.

### Βήμα 5: Αποθήκευση της παρουσίασης
Τέλος, αποθηκεύστε την παρουσίασή σας σε ένα αρχείο χρησιμοποιώντας τον ακόλουθο κώδικα:

```csharp
presentation.Save(dataDir + "presentation-out.pptx", SaveFormat.Pptx);
```

Η παρουσίαση της μεταβλητής υπερσύνδεσής σας είναι τώρα έτοιμη!

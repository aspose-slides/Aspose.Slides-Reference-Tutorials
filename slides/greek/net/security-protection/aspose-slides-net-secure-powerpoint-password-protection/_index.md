## Οδηγός Εφαρμογής
Τώρα που όλα έχουν ρυθμιστεί, ας εφαρμόσουμε προστασία με κωδικό πρόσβασης για τις παρουσιάσεις σας.

### Κρυπτογράφηση και αποθήκευση παρουσίασης με προστασία με κωδικό πρόσβασης
#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να κρυπτογραφήσετε ένα αρχείο PowerPoint ορίζοντας έναν κωδικό πρόσβασης, διασφαλίζοντας ότι μόνο εξουσιοδοτημένοι χρήστες έχουν πρόσβαση σε αυτό. 

#### Βήματα για την εφαρμογή
**1. Δημιουργήστε τον κατάλογό σας**
Βεβαιωθείτε ότι έχετε τη σωστή διαδρομή καταλόγου όπου θα αποθηκευτούν τα έγγραφά σας:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";

// Ελέγξτε αν ο κατάλογος υπάρχει και δημιουργήστε τον, εάν είναι απαραίτητο.
bool isExists = Directory.Exists(dataDir);
if (!isExists)
    Directory.CreateDirectory(dataDir);
```
Αυτό το βήμα διασφαλίζει ότι το αρχείο σας θα αποθηκευτεί σε μια συγκεκριμένη τοποθεσία στο σύστημά σας.

**2. Δημιουργήστε μια νέα παρουσίαση**
Δημιουργήστε ένα υπόδειγμα `Presentation` αντικείμενο για εργασία με:

```csharp
// Δημιουργήστε ένα αντικείμενο παρουσίασης.
Presentation pres = new Presentation();
```
Μπορείτε να εκτελέσετε διάφορες λειτουργίες σε αυτήν την παρουσίαση, όπως προσθήκη διαφανειών ή μορφοποίηση περιεχομένου.

**3. Κρυπτογράφηση της παρουσίασης**
Ορίστε έναν κωδικό πρόσβασης για την κρυπτογράφηση της παρουσίασής σας χρησιμοποιώντας την ακόλουθη μέθοδο:

```csharp
// Ορίστε τον κωδικό πρόσβασης για την κρυπτογράφηση.
pres.ProtectionManager.Encrypt("pass");
```
Ο `Encrypt` Η μέθοδος δέχεται μια παράμετρο συμβολοσειράς που λειτουργεί ως κωδικός πρόσβασης, προστατεύοντας το αρχείο από μη εξουσιοδοτημένη πρόσβαση.

**4. Αποθηκεύστε την κρυπτογραφημένη παρουσίαση**
Τέλος, αποθηκεύστε την κρυπτογραφημένη παρουσίαση σε μορφή PPTX:

```csharp
// Αποθηκεύστε την παρουσίαση με προστασία κωδικού πρόσβασης.
pres.Save(dataDir + "/SecurePresentation_out.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
```
Αποθηκεύοντας το αρχείο με αυτόν τον τρόπο, θα είναι ασφαλές και θα απαιτείται κωδικός πρόσβασης για να ανοίξει.

### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι η διαδρομή καταλόγου έχει οριστεί σωστά. Διαφορετικά, ενδέχεται να αντιμετωπίσετε `DirectoryNotFoundException`.
- Επαληθεύστε ότι το έργο σας αναφέρει τη σωστή έκδοση του Aspose.Slides για .NET.

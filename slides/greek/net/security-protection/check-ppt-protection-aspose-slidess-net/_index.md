## Οδηγός Εφαρμογής
Εξερευνήστε ξεχωριστές λειτουργίες που εστιάζουν στους ελέγχους προστασίας του PowerPoint χρησιμοποιώντας C#.

### Δυνατότητα 1: Έλεγχος προστασίας εγγραφής παρουσίασης μέσω της διεπαφής IPresentationInfo
**Επισκόπηση:**
Προσδιορίστε εάν μια παρουσίαση έχει προστασία εγγραφής αξιοποιώντας το `IPresentationInfo` διεπαφή, η οποία εστιάζει στην προστασία που βασίζεται σε κωδικό πρόσβασης.

#### Βήμα προς βήμα εφαρμογή
**Βήμα 1: Ορίστε τη διαδρομή αρχείου**
Προσδιορίστε και καθορίστε τον κατάλογο του αρχείου παρουσίασής σας:
```csharp
string pptxFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "modify_pass2.pptx");
```

**Βήμα 2: Λήψη πληροφοριών παρουσίασης**
Χρήση `PresentationFactory` για πρόσβαση στις λεπτομέρειες:
```csharp
IPresentationInfo presentationInfo = PresentationFactory.Instance.GetPresentationInfo(pptxFile);
```

**Βήμα 3: Έλεγχος κατάστασης προστασίας εγγραφής**
Επαληθεύστε εάν το αρχείο προστατεύεται με κωδικό πρόσβασης και επικυρώστε τον:
```csharp
bool isWriteProtectedByPassword = presentationInfo.IsWriteProtected == NullableBool.True &&
                                   presentationInfo.CheckWriteProtection("pass2");
```

### Λειτουργία 2: Έλεγχος προστασίας εγγραφής παρουσίασης μέσω της διεπαφής IProtectionManager
**Επισκόπηση:**
Αυτή η λειτουργία επιτρέπει τον έλεγχο της προστασίας εγγραφής μιας παρουσίασης χρησιμοποιώντας το `IProtectionManager` διεπαφή.

#### Βήμα προς βήμα εφαρμογή
**Βήμα 1: Ανοίξτε την παρουσίαση**
Φορτώστε το αρχείο παρουσίασης:
```csharp
using (var presentation = new Presentation(pptxFile))
{
    // Συνέχεια με τους ελέγχους
}
```

**Βήμα 2: Επαλήθευση προστασίας εγγραφής**
Ελέγξτε αν η προστασία εγγραφής είναι ενεργή και επικυρώστε τη χρησιμοποιώντας έναν κωδικό πρόσβασης:
```csharp
bool isWriteProtected = presentation.ProtectionManager.CheckWriteProtection("pass2");
```

### Δυνατότητα 3: Έλεγχος προστασίας ανοιχτής παρουσίασης μέσω της διεπαφής IPresentationInfo
**Επισκόπηση:**
Αυτή η μέθοδος ελέγχει εάν το αρχείο PowerPoint απαιτεί κωδικό πρόσβασης για να ανοίξει.

#### Βήμα προς βήμα εφαρμογή
**Βήμα 1: Ορίστε τη διαδρομή αρχείου**
Καθορίστε τη διαδρομή για την προστατευμένη παρουσίασή σας:
```csharp
string pptFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "open_pass1.ppt");
```

**Βήμα 2: Ανάκτηση πληροφοριών παρουσίασης**
Πρόσβαση σε πληροφορίες χρησιμοποιώντας `IPresentationInfo`:
```csharp
IPresentationInfo presentationInfo = PresentationFactory.Instance.GetPresentationInfo(pptFile);
```

**Βήμα 3: Προσδιορισμός κατάστασης ανοιχτής προστασίας**
Ελέγξτε αν το αρχείο είναι ανοιχτό και προστατεύεται με κωδικό πρόσβασης:
```csharp
if (presentationInfo.IsPasswordProtected)
{
    // Το αρχείο απαιτεί κωδικό πρόσβασης για να ανοίξει.
}
```

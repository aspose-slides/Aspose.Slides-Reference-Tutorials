## Οδηγός Εφαρμογής

### Ορισμός συνιστώμενης ιδιότητας μόνο για ανάγνωση

Αυτή η λειτουργία διασφαλίζει ότι οι παρουσιάσεις σας ανοίγουν σε λειτουργία μόνο για ανάγνωση, προστατεύοντάς τες από μη εξουσιοδοτημένες επεξεργασίες.

#### Βήμα 1: Δημιουργία νέου αντικειμένου παρουσίασης
Ξεκινήστε δημιουργώντας ένα `Presentation` αντικείμενο:
```csharp
using Aspose.Slides;

// Δημιουργήστε ένα νέο αντικείμενο παρουσίασης
var pres = new Presentation();
```

#### Βήμα 2: Ορίστε την ιδιότητα ReadOnlyRecommended σε True
Χρησιμοποιήστε το `ProtectionManager` τάξη:
```csharp
// Ορίστε την ιδιότητα ReadOnlyRecommended σε true
pres.ProtectionManager.ReadOnlyRecommended = true;
```

#### Βήμα 3: Ορισμός διαδρομής εξόδου και αποθήκευση
Καθορίστε τη διαδρομή εξόδου και αποθηκεύστε την παρουσίαση:
```csharp
using System.IO;

// Ορισμός διαδρομής εξόδου με τον πραγματικό κατάλογο
string outPptxPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "ReadOnlyRecommended.pptx");

// Αποθήκευση της παρουσίασης ως αρχείο PPTX
pres.Save(outPptxPath, SaveFormat.Pptx);
```

### Συμβουλές αντιμετώπισης προβλημάτων
- **Λανθασμένες διαδρομές αρχείων:** Βεβαιωθείτε ότι η διαδρομή του καταλόγου εξόδου είναι σωστή και προσβάσιμη.
- **Προβλήματα δικαιωμάτων:** Ελέγξτε αν έχετε δικαιώματα εγγραφής για τον κατάλογο αποθήκευσης.

## Οδηγός Εφαρμογής
Τώρα, ας δούμε πώς να ενημερώνετε τις ιδιότητες παρουσίασης.

### Ενημέρωση λειτουργίας ιδιοτήτων παρουσίασης
Αυτή η λειτουργία σάς επιτρέπει να αλλάξετε μέσω προγραμματισμού τον δημιουργό και τον τίτλο μιας παρουσίασης PowerPoint.

#### Βήμα 1: Επαλήθευση ύπαρξης αρχείου
Βεβαιωθείτε ότι το αρχείο υπάρχει στον καθορισμένο κατάλογο πριν από την πρόσβαση σε αυτό.
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";

if (File.Exists(dataDir + "/ModifyBuiltinProperties1.pptx")) {
    // Συνέχεια με την ενημέρωση των ιδιοτήτων
} else {
    Console.WriteLine("The specified presentation file does not exist.");
}
```

#### Βήμα 2: Λήψη πληροφοριών παρουσίασης
Ανάκτηση πληροφοριών σχετικά με την παρουσίαση χρησιμοποιώντας `PresentationFactory`.
```csharp
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo(dataDir + "/ModifyBuiltinProperties1.pptx");
```

#### Βήμα 3: Ανάγνωση και ενημέρωση ιδιοτήτων εγγράφου
Αποκτήστε πρόσβαση σε τρέχουσες ιδιότητες και ενημερώστε τες όπως απαιτείται.
```csharp
IDocumentProperties props = info.ReadDocumentProperties();
props.Author = "New Author";
props.Title = "New Title";
info.UpdateDocumentProperties(props);
```

#### Βήμα 4: Αποθήκευση αλλαγών
Διατηρήστε τις αλλαγές σας πίσω στο αρχείο.
```csharp
info.WriteBindedPresentation(dataDir + "/ModifyBuiltinProperties1.pptx");
```

### Συμβουλές αντιμετώπισης προβλημάτων:
- Βεβαιωθείτε ότι οι διαδρομές είναι σωστές και προσβάσιμες.
- Χειριστείτε τις εξαιρέσεις για τις λειτουργίες εισόδου/εξόδου αρχείων με ομαλό τρόπο.

## Οδηγός Εφαρμογής

Αφού ρυθμίσουμε το περιβάλλον μας, ας τροποποιήσουμε τις ιδιότητες της παρουσίασης του PowerPoint χρησιμοποιώντας το Aspose.Slides για .NET.

### Πρόσβαση στις Ιδιότητες Παρουσίασης

#### Επισκόπηση
Πρόσβαση και τροποποίηση ενσωματωμένων ιδιοτήτων ενός αρχείου PowerPoint:

```csharp
using System;
using Aspose.Slides;

// Ορίστε τους καταλόγους εγγράφων σας
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// Δημιουργήστε την κλάση παρουσίασης
Presentation presentation = new Presentation(dataDir + "/ModifyBuiltinProperties.pptx");

// Πρόσβαση σε ενσωματωμένες ιδιότητες
IDocumentProperties documentProperties = presentation.DocumentProperties;
```

#### Εξήγηση
- **`dataDir`**: Διαδρομή προς το αρχείο PowerPoint εισόδου σας.
- **`outputDir`**: Κατάλογος όπου θα αποθηκευτεί η τροποποιημένη παρουσίαση.

### Τροποποίηση ενσωματωμένων ιδιοτήτων
Ορίστε διάφορες ιδιότητες ως εξής:

**Συγγραφέας:**
```csharp
documentProperties.Author = "Aspose.Slides for .NET";
```
- Ορίζει τον συγγραφέα της παρουσίασης.

**Τίτλος:**
```csharp
documentProperties.Title = "Modifying Presentation Properties with Aspose.Slides";
```
- Ενημερώνει τον τίτλο της παρουσίασής σας.

**Θέμα, Σχόλια και Διευθυντής:**
```csharp
documentProperties.Subject = "Aspose Subject";
documentProperties.Comments = "Aspose Description";
documentProperties.Manager = "Aspose Manager";
```
- Αυτές οι ιδιότητες παρέχουν πρόσθετα μεταδεδομένα σχετικά με το έγγραφο.

### Αποθήκευση αλλαγών
Αποθηκεύστε τις τροποποιήσεις σας με:

```csharp
presentation.Save(outputDir + "/DocumentProperties_out.pptx", SaveFormat.Pptx);
```

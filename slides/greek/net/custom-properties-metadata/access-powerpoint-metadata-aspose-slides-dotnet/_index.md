## Οδηγός Εφαρμογής

Αυτή η ενότητα σας καθοδηγεί στην πρόσβαση στις ενσωματωμένες ιδιότητες παρουσίασης χρησιμοποιώντας το Aspose.Slides για .NET.

### Πρόσβαση σε ενσωματωμένες ιδιότητες
#### Επισκόπηση
Αποκτήστε πρόσβαση σε ενσωματωμένες ιδιότητες για να εξαγάγετε μεταδεδομένα όπως συγγραφέα, τίτλο και σχόλια από ένα αρχείο PowerPoint. Αυτό είναι κρίσιμο για την παρακολούθηση εκδόσεων εγγράφων ή την αυτοματοποίηση εργασιών διαχείρισης περιεχομένου.

#### Βήμα προς βήμα εφαρμογή
**1. Ορισμός Διαδρομής Εγγράφου**
Καθορίστε τη διαδρομή όπου είναι αποθηκευμένο το αρχείο PowerPoint:
```csharp
string dataDir = @"YOUR_DOCUMENT_DIRECTORY\AccessBuiltin Properties.pptx";
```

**2. Δημιουργία αντικειμένου παρουσίασης**
Δημιουργήστε ένα `Presentation` αντικείμενο για την αναπαράσταση του αρχείου PPTX σας:
```csharp
using (Presentation pres = new Presentation(dataDir))
{
    // Ο κωδικός σας εδώ
}
```

**3. Πρόσβαση στις Ιδιότητες Εγγράφου**
Ανάκτηση των ιδιοτήτων χρησιμοποιώντας `IDocumentProperties` που σχετίζονται με την παρουσίαση:
```csharp
IDocumentProperties documentProperties = pres.DocumentProperties;
```

**4. Εμφάνιση ενσωματωμένων ιδιοτήτων**
Εκτυπώστε διάφορα χαρακτηριστικά μεταδεδομένων για να κατανοήσετε καλύτερα την παρουσίασή σας:
```csharp
Console.WriteLine("Category : " + documentProperties.Category);
Console.WriteLine("Current Status : " + documentProperties.ContentStatus);
Console.WriteLine("Creation Date : " + documentProperties.CreatedTime);
Console.WriteLine("Author : " + documentProperties.Author);
Console.WriteLine("Description : " + documentProperties.Comments);
Console.WriteLine("KeyWords : " + documentProperties.Keywords);
Console.WriteLine("Last Modified By : " + documentProperties.LastSavedBy);
Console.WriteLine("Supervisor : " + documentProperties.Manager);
Console.WriteLine("Modified Date : " + documentProperties.LastSavedTime);
Console.WriteLine("Presentation Format : " + documentProperties.PresentationFormat);
Console.WriteLine("Last Print Date : " + documentProperties.LastPrinted);
Console.WriteLine("Is Shared between producers : " + documentProperties.SharedDoc);
Console.WriteLine("Subject : " + documentProperties.Subject);
Console.WriteLine("Title : " + documentProperties.Title);
```

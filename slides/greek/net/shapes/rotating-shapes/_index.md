# Δημιουργία και περιστροφή ορθογωνίων σχημάτων στο PowerPoint

Το Aspose.Slides για .NET είναι ένα ισχυρό API που επιτρέπει στους προγραμματιστές να δημιουργούν, να τροποποιούν και να μετατρέπουν παρουσιάσεις PowerPoint χωρίς να απαιτείται εγκατάσταση του Microsoft PowerPoint. Αυτό το παράδειγμα δείχνει πώς να δημιουργήσετε μια παρουσίαση με περιστρεφόμενο ορθογώνιο σχήμα, παρουσιάζοντας τις δυνατότητες χειρισμού σχημάτων της βιβλιοθήκης.

Με το Aspose.Slides, μπορείτε να προσθέσετε μέσω προγραμματισμού διάφορα σχήματα στις διαφάνειές σας και να εφαρμόσετε μετασχηματισμούς όπως περιστροφή. Αυτή η λειτουργικότητα είναι ιδιαίτερα χρήσιμη για τη δημιουργία δυναμικών παρουσιάσεων με ακριβή έλεγχο των οπτικών στοιχείων.

## Οδηγίες εγκατάστασης

1. Εγκατάσταση μέσω του NuGet Package Manager:
   ```
   Install-Package Aspose.Slides
   ```

2. Εγκατάσταση μέσω .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Προσθέστε την οδηγία using στον κώδικά σας:
   ```csharp
   using Aspose.Slides;
   ```

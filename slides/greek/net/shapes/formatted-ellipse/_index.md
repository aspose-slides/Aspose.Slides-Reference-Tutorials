# Δημιουργία ελλειπτικών σχημάτων στο PowerPoint με το Aspose.Slides

Το Aspose.Slides για .NET είναι ένα ισχυρό API που επιτρέπει στους προγραμματιστές να δημιουργούν, να διαβάζουν, να γράφουν, να τροποποιούν και να μετατρέπουν παρουσιάσεις PowerPoint χωρίς να απαιτούν το Microsoft PowerPoint. Αυτό το παράδειγμα δείχνει πώς να δημιουργήσετε ένα ελλειπτικό σχήμα σε μια παρουσίαση, να εφαρμόσετε μορφοποίηση που περιλαμβάνει χρώμα γεμίσματος και ιδιότητες γραμμής και να αποθηκεύσετε το αποτέλεσμα ως αρχείο PPTX. Ο κώδικας παρουσιάζει την ικανότητα του Aspose.Slides να χειρίζεται σχήματα με ακριβή έλεγχο της εμφάνισης και της τοποθέτησής τους.

## Οδηγίες εγκατάστασης

1. Εγκατάσταση μέσω του NuGet Package Manager:
   ```
   Install-Package Aspose.Slides
   ```

   Ή χρησιμοποιώντας το .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

2. Προσθήκη Αναφορών:
   Προσθέστε τον ακόλουθο χώρο ονομάτων στο αρχείο C#:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

3. Ρύθμιση Άδειας Χρήσης (Προαιρετικό):
   Εάν έχετε άδεια χρήσης, εφαρμόστε την χρησιμοποιώντας:
   ```csharp
   License license = new License();
   license.SetLicense("Aspose.Slides.lic");
   ```

# Δημιουργία σχημάτων έλλειψης με διαβάθμιση στο PowerPoint με το Aspose.Slides

Αυτό το παράδειγμα δείχνει πώς να δημιουργήσετε μέσω προγραμματισμού μια παρουσίαση PowerPoint που περιέχει ένα σχήμα έλλειψης με ένα προσαρμοσμένο εφέ γεμίσματος με διαβάθμιση. Ο κώδικας δημιουργεί μια νέα παρουσίαση, προσθέτει μια έλλειψη στην πρώτη διαφάνεια και εφαρμόζει ένα γραμμικό γέμισμα με διαβάθμιση με κόκκινα και μοβ χρωματικά στοπ. Το Aspose.Slides για .NET παρέχει ένα ολοκληρωμένο API για τη δημιουργία και τον χειρισμό παρουσιάσεων PowerPoint με ακριβή έλεγχο των σχημάτων, της μορφοποίησης και των εφέ.

## Οδηγίες εγκατάστασης

1. Εγκαταστήστε το Aspose.Slides για .NET χρησιμοποιώντας μία από τις ακόλουθες μεθόδους:
   - Διαχειριστής πακέτων NuGet: `Install-Package Aspose.Slides.NET`
   - .NET CLI: `dotnet add package Aspose.Slides.NET`
   - Κονσόλα Διαχείρισης Πακέτων: `Install-Package Aspose.Slides.NET`

2. Προσθέστε την ακόλουθη εντολή στο αρχείο C# χρησιμοποιώντας την εντολή:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

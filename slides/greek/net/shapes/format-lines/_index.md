# Δημιουργία ορθογωνίου σχήματος με προσαρμοσμένη μορφοποίηση γραμμών στο PowerPoint

Το Aspose.Slides για .NET παρέχει ισχυρές δυνατότητες για τη δημιουργία και τροποποίηση παρουσιάσεων PowerPoint μέσω προγραμματισμού. Αυτό το παράδειγμα δείχνει πώς να δημιουργήσετε ένα ορθογώνιο σχήμα με προσαρμοσμένη μορφοποίηση γραμμής, συμπεριλαμβανομένου του ορισμού του στυλ γραμμής, του πλάτους, του μοτίβου παύλας και του χρώματος. Το δείγμα κώδικα δημιουργεί μια νέα παρουσίαση, προσθέτει ένα ορθογώνιο στην πρώτη διαφάνεια και εφαρμόζει συγκεκριμένη μορφοποίηση τόσο στο γέμισμα όσο και στο περίγραμμα του σχήματος, με αποτέλεσμα ένα οπτικά ελκυστικό στοιχείο με λευκό γέμισμα και μπλε διακεκομμένο περίγραμμα.

## Οδηγίες εγκατάστασης

1. Εγκαταστήστε το Aspose.Slides για .NET χρησιμοποιώντας μία από τις ακόλουθες μεθόδους:

   - Διαχειριστής πακέτων NuGet:
     ```
     Install-Package Aspose.Slides
     ```

   - Κονσόλα Διαχείρισης Πακέτων:
     ```
     PM> Install-Package Aspose.Slides
     ```

   - .NET CLI:
     ```
     dotnet add package Aspose.Slides
     ```

2. Προσθέστε τον απαραίτητο χώρο ονομάτων στο αρχείο C#:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

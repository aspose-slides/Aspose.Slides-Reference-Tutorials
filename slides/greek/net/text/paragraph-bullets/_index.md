# Δημιουργία προσαρμοσμένων κουκκίδων σε παρουσιάσεις PowerPoint

Το Aspose.Slides για .NET παρέχει ισχυρές δυνατότητες για τη δημιουργία και τον χειρισμό παρουσιάσεων PowerPoint μέσω προγραμματισμού. Αυτό το παράδειγμα εστιάζει συγκεκριμένα στην εφαρμογή προσαρμοσμένων κουκκίδων σε πλαίσια κειμένου, δείχνοντας πώς να ελέγχετε τον τύπο, το χρώμα, την εσοχή και το μέγεθος των κουκκίδων. Ο κώδικας δημιουργεί μια παρουσίαση με σχήμα ορθογωνίου που περιέχει δύο παραγράφους - μία με κουκκίδα συμβόλου και μια άλλη με στυλ κουκκίδας με αρίθμηση.

## Οδηγίες εγκατάστασης

1. Εγκαταστήστε το Aspose.Slides για .NET χρησιμοποιώντας το NuGet Package Manager:
   ```
   Install-Package Aspose.Slides
   ```

2. Εναλλακτικά, κατεβάστε απευθείας το πακέτο:
   - Επισκεφθείτε το [Σελίδα λήψης Aspose.Slides](https://downloads.aspose.com/slides/net)
   - Κατεβάστε την τελευταία έκδοση
   - Εξαγάγετε και αναφέρετε το Aspose.Slides.dll στο έργο σας

3. Προσθέστε την απαραίτητη εντολή χρησιμοποιώντας:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

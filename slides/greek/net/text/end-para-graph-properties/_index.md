# Δημιουργία παρουσιάσεων PowerPoint με μορφοποιημένο κείμενο

Το Aspose.Slides για .NET είναι ένα ισχυρό API που επιτρέπει στους προγραμματιστές να δημιουργούν, να χειρίζονται και να μετατρέπουν παρουσιάσεις PowerPoint χωρίς να απαιτούν το Microsoft PowerPoint. Αυτό το παράδειγμα δείχνει πώς να δημιουργήσετε μια απλή παρουσίαση που περιέχει ένα ορθογώνιο σχήμα με μορφοποιημένες παραγράφους κειμένου.

Ο κώδικας δημιουργεί μια νέα παρουσίαση, προσθέτει ένα ορθογώνιο σχήμα στην πρώτη διαφάνεια και, στη συνέχεια, προσθέτει πολλές παραγράφους με διαφορετικές επιλογές μορφοποίησης κειμένου. Δείχνει πώς να ορίσετε ιδιότητες γραμματοσειράς όπως μέγεθος και οικογένεια για συγκεκριμένα τμήματα κειμένου.

## Οδηγίες εγκατάστασης

1. Εγκατάσταση μέσω του NuGet Package Manager:
   ```
   Install-Package Aspose.Slides
   ```

2. Εγκατάσταση μέσω .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Εγκατάσταση μέσω της Κονσόλας Διαχείρισης Πακέτων:
   ```
   PM> Install-Package Aspose.Slides
   ```

4. Προσθήκη αναφοράς στο έργο σας:
   Μετά την εγκατάσταση, προσθέστε μια αναφορά στο έργο σας:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

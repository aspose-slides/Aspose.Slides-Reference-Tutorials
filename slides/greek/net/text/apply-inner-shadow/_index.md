# Δημιουργήστε παρουσιάσεις PowerPoint με το TextBox στο Aspose.Slides για .NET

Το Aspose.Slides για .NET είναι ένα ισχυρό API που επιτρέπει στους προγραμματιστές να δημιουργούν, να τροποποιούν και να μετατρέπουν παρουσιάσεις PowerPoint χωρίς να απαιτούν το Microsoft PowerPoint. Αυτό το παράδειγμα δείχνει πώς να δημιουργήσετε μια παρουσίαση με ένα προσαρμοσμένο πλαίσιο κειμένου, να εφαρμόσετε μορφοποίηση και να την αποθηκεύσετε ως αρχείο PPTX.

## Οδηγίες εγκατάστασης

1. Εγκατάσταση μέσω του NuGet Package Manager:
   ```
   Install-Package Aspose.Slides
   ```

2. Εγκατάσταση μέσω .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Εγκατάσταση χειροκίνητα:
   - Κατεβάστε την τελευταία έκδοση από το [Σελίδα εκδόσεων Aspose.Slides για .NET](https://releases.aspose.com/slides/net/)
   - Προσθέστε την αναφορά DLL στο έργο σας

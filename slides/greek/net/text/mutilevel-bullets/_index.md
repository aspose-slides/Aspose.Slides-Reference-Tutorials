# Δημιουργία κουκκίδων πολλαπλών επιπέδων στο PowerPoint με το Aspose.Slides

Το Aspose.Slides για .NET παρέχει ένα ισχυρό API για τη δημιουργία και τον χειρισμό παρουσιάσεων PowerPoint μέσω προγραμματισμού. Αυτό το παράδειγμα δείχνει πώς να δημιουργήσετε μια παρουσίαση με κουκκίδες πολλαπλών επιπέδων, δείχνοντας πώς να προσθέσετε ένα αυτόματο σχήμα με ένα πλαίσιο κειμένου και να μορφοποιήσετε παραγράφους με διαφορετικά στυλ κουκκίδων και επίπεδα εσοχής. Μπορείτε εύκολα να δημιουργήσετε ιεραρχικό περιεχόμενο με προσαρμοσμένα σύμβολα κουκκίδων για κάθε επίπεδο εσοχής, ιδανικό για τη δημιουργία δομημένων και επαγγελματικών παρουσιάσεων χωρίς να απαιτείται η εγκατάσταση του Microsoft PowerPoint.

## Οδηγίες εγκατάστασης

1. Εγκατάσταση μέσω του NuGet Package Manager:
   ```
   Install-Package Aspose.Slides
   ```

2. Εγκατάσταση μέσω .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Εγκατάσταση μέσω της Κονσόλας Διαχείρισης Πακέτων:
   ```
   PM> Install-Package Aspose.Slides
   ```

4. Χειροκίνητη εγκατάσταση:
   - Κατεβάστε την τελευταία έκδοση από [Aspose.Slides για .NET](https://releases.aspose.com/slides/net/)
   - Προσθέστε αναφορά στο Aspose.Slides.dll στο έργο σας

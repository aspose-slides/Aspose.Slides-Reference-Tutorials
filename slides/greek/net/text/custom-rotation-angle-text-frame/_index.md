# Εναλλαγή τίτλων και ετικετών γραφημάτων στο PowerPoint με το Aspose.Slides

Το Aspose.Slides για .NET παρέχει ισχυρές δυνατότητες για τη δημιουργία και τον χειρισμό παρουσιάσεων PowerPoint μέσω προγραμματισμού. Αυτό το παράδειγμα δείχνει πώς να δημιουργήσετε μια παρουσίαση με ένα γράφημα ομαδοποιημένων στηλών που διαθέτει περιστρεφόμενα στοιχεία κειμένου - συγκεκριμένα, περιστρέφοντας τόσο τον τίτλο του γραφήματος όσο και τις ετικέτες δεδομένων για βελτιωμένη αναγνωσιμότητα και οπτική ελκυστικότητα.

## Οδηγίες εγκατάστασης

1. Εγκατάσταση μέσω του NuGet Package Manager:
   ```
   Install-Package Aspose.Slides
   ```

2. Ή εγκαταστήστε μέσω .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Προσθέστε τον χώρο ονομάτων στο αρχείο C# σας:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Charts;
   ```

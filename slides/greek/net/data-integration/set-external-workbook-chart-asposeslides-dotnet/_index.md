# Πώς να ορίσετε ένα εξωτερικό βιβλίο εργασίας για ένα γράφημα στο Aspose.Slides .NET: Οδηγός βήμα προς βήμα

## Εισαγωγή

Η ενσωμάτωση δεδομένων απευθείας από εξωτερικές πηγές στις παρουσιάσεις σας μπορεί να βελτιώσει σημαντικά την αξία τους. Με το Aspose.Slides για .NET, μπορείτε να ορίσετε απρόσκοπτα ένα εξωτερικό βιβλίο εργασίας για γραφήματα μέσα σε διαφάνειες, επιτρέποντας δυναμικές και ενημερωμένες απεικονίσεις. Αυτό το σεμινάριο θα σας καθοδηγήσει στη διαδικασία σύνδεσης ενός αρχείου Excel που βασίζεται σε δίκτυο με ένα γράφημα στην παρουσίασή σας.

**Τι θα μάθετε:**
- Ρύθμιση παραμέτρων ενός περιβάλλοντος Aspose.Slides .NET.
- Ρύθμιση εξωτερικού βιβλίου εργασίας από μια θέση δικτύου για γραφήματα.
- Υλοποίηση ενός προσαρμοσμένου χειριστή φόρτωσης πόρων σε C#.
- Πρακτικές εφαρμογές ενσωμάτωσης εξωτερικών πηγών δεδομένων με παρουσιάσεις.

Ας ξεκινήσουμε!

## Προαπαιτούμενα

Πριν ξεκινήσετε την κωδικοποίηση, βεβαιωθείτε ότι πληροίτε τις ακόλουθες προϋποθέσεις:

- **Απαιτούμενες βιβλιοθήκες και εξαρτήσεις**Εγκαταστήστε το Aspose.Slides για .NET στο έργο σας.
- **Απαιτήσεις Ρύθμισης Περιβάλλοντος**: Ρύθμιση ενός περιβάλλοντος ανάπτυξης C# (π.χ., Visual Studio).
- **Προαπαιτούμενα Γνώσεων**Να έχετε βασικές γνώσεις προγραμματισμού C# και εξοικείωση με το Aspose.Slides.

## Ρύθμιση του Aspose.Slides για .NET

Ξεκινήστε εγκαθιστώντας τη βιβλιοθήκη Aspose.Slides στο έργο σας. Μπορείτε να χρησιμοποιήσετε οποιαδήποτε από αυτές τις μεθόδους:

**.NET CLI**
```bash
dotnet add package Aspose.Slides
```

**Κονσόλα διαχείρισης πακέτων**
```bash

```

**Διεπαφή χρήστη του διαχειριστή πακέτων NuGet**Αναζητήστε το "Aspose.Slides" και εγκαταστήστε την πιο πρόσφατη έκδοση.

## Οδηγός Εφαρμογής

Ας αναλύσουμε την υλοποίηση σε βασικά χαρακτηριστικά.

### Ρύθμιση εξωτερικού βιβλίου εργασίας από δίκτυο

Αυτή η δυνατότητα σάς επιτρέπει να συνδέσετε ένα αρχείο Excel που βασίζεται σε δίκτυο ως εξωτερικό βιβλίο εργασίας για ένα γράφημα στην παρουσίασή σας.

#### Βήμα 1: Καθορίστε τη διαδρομή εξωτερικού βιβλίου εργασίας
Καθορίστε τη διαδρομή του εξωτερικού βιβλίου εργασίας σας που βρίσκεται σε μια μονάδα δίσκου δικτύου:
```csharp
string externalWbPath = "http://Ο ΚΑΤΑΛΟΓΟΣ_ΕΓΓΡΑΦΩΝ_ΣΑΣ/styles/2.xlsx";
```
Αντικαθιστώ `YOUR_DOCUMENT_DIRECTORY` με τον πραγματικό κατάλογο όπου φιλοξενείται το αρχείο Excel σας.

#### Βήμα 2: Ρύθμιση παραμέτρων επιλογών φόρτωσης
Ορίστε επιλογές φόρτωσης και καθορίστε μια προσαρμοσμένη επιστροφή κλήσης φόρτωσης πόρων:
```csharp
LoadOptions opts = new LoadOptions();
opts.ResourceLoadingCallback = new WorkbookLoadingHandler();
```

#### Βήμα 3: Δημιουργία παρουσίασης και προσθήκη γραφήματος
Δημιουργήστε μια παρουσία παρουσίασης και προσθέστε ένα γράφημα στην πρώτη διαφάνεια:
```csharp
using (Presentation pres = new Presentation(opts))
{
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.Pie, 50, 50, 400, 600, false);
    IChartData chartData = chart.ChartData;
    
    // Ορισμός της διαδρομής του εξωτερικού βιβλίου εργασίας για τα δεδομένα γραφήματος
    (chartData as ChartData).SetExternalWorkbook(externalWbPath);
}
```

### Χειριστής φόρτωσης βιβλίου εργασίας

Αυτή η λειτουργία περιλαμβάνει τη δημιουργία ενός προσαρμοσμένου χειριστή φόρτωσης πόρων για την ανάκτηση του αρχείου Excel από την καθορισμένη τοποθεσία δικτύου σας.

#### Βήμα 1: Υλοποίηση επανακλήσης φόρτωσης πόρων
Δημιουργήστε μια κλάση που υλοποιεί `IResourceLoadingCallback`:
```csharp
class WorkbookLoadingHandler : IResourceLoadingCallback
{
    public ResourceLoadingAction ResourceLoading(IResourceLoadingArgs args)
    {
        string workbookPath = args.OriginalUri;
        
        // Ελέγξτε αν η διαδρομή είναι μια τοποθεσία δικτύου (όχι μια τοπική διαδρομή αρχείου)
        if (workbookPath.IndexOf(':') > 1 && !workbookPath.StartsWith("file:///"))
        {
            try
            {
                WebRequest request = WebRequest.Create(workbookPath);
                request.Credentials = new NetworkCredential("testuser", "testuser");
                
                using (WebResponse response = request.GetResponse())
                using (Stream responseStream = response.GetResponseStream())
                {
                    // Παρέχετε τα δεδομένα που ανακτήθηκαν στο Aspose.Slides
                    return ResourceLoadingAction.UserProvided;
                }
            }
            catch (Exception ex)
            {
                throw new InvalidOperationException(ex.ToString());
            }
        }
        else
        {
            return ResourceLoadingAction.Default;
        }
    }
}
```

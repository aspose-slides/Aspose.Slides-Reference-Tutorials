## Οδηγός Εφαρμογής
Τώρα ας εξερευνήσουμε πώς να δημιουργήσουμε μια νέα παρουσίαση με μια γραμμή αυτόματης διαμόρφωσης.

### Δημιουργία νέας παρουσίασης και προσθήκη σχήματος γραμμής
#### Επισκόπηση
Αυτή η ενότητα παρουσιάζει την αρχικοποίηση μιας νέας παρουσίασης, την πρόσβαση στην προεπιλεγμένη διαφάνεια, την προσθήκη ενός σχήματος γραμμής και την αποθήκευση του αρχείου.

#### Βήμα προς βήμα εφαρμογή
**1. Δημιουργήστε ένα αντίγραφο του αντικειμένου παρουσίασης**
Δημιουργήστε μια νέα παρουσία του `Presentation` κλάση που αντιπροσωπεύει το αρχείο PowerPoint σας:
```csharp
using (Presentation presentation = new Presentation())
{
    // Ο κώδικας θα μπει εδώ
}
```
Αυτό αρχικοποιεί μια κενή παρουσίαση την οποία μπορούμε να τροποποιήσουμε.

**2. Πρόσβαση στην πρώτη διαφάνεια**
Οι διαφάνειες σε μια παρουσίαση είναι προσβάσιμες μέσω μιας συλλογής με ευρετήριο. Δείτε πώς μπορείτε να αποκτήσετε την πρώτη διαφάνεια:
```csharp
ISlide slide = presentation.Slides[0];
```

**3. Προσθήκη γραμμής αυτόματης διαμόρφωσης**
Για να προσθέσουμε μια γραμμή, χρησιμοποιούμε το `AddAutoShape` μέθοδος με συγκεκριμένες παραμέτρους για τον τύπο σχήματος και τις διαστάσεις:
```csharp
slide.Shapes.AddAutoShape(ΤύποςΣχήματος.Γραμμή, 50, 150, 300, 0);
```
- **ShapeType.Line**: Καθορίζει ότι το σχήμα είναι μια γραμμή.
- **Συντεταγμένες (50, 150)**: Ορίστε το σημείο έναρξης της γραμμής στη διαφάνεια.
- **Διαστάσεις (300, 0)**: Ορίστε το μήκος και το πλάτος. Το μηδενικό πλάτος διασφαλίζει ότι είναι απλώς μια γραμμή.

**4. Αποθήκευση της παρουσίασης**
Καθορίστε τον κατάλογο εξόδου και αποθηκεύστε την παρουσίαση στην επιθυμητή μορφή:
```csharp
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
string outputFile = outputDirectory + "/NewPresentation_out.pptx";

presentation.Save(outputFile, SaveFormat.Pptx);
```

Στη συνέχεια, πρέπει να διαμορφώσουμε τις επιλογές μετατροπής HTML. Συγκεκριμένα, θέλουμε να τοποθετήσουμε τις σημειώσεις στο κάτω μέρος των σελίδων HTML. Χρησιμοποιήστε το ακόλουθο απόσπασμα κώδικα για να ρυθμίσετε τις επιλογές:

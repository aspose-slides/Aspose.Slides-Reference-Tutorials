# Δημιουργία μικρογραφίας με όρια για σχήμα στο Aspose.Slides

Σε αυτό το σεμινάριο, θα σας παρουσιάσουμε τη διαδικασία δημιουργίας μιας μικρογραφίας με όρια για ένα σχήμα χρησιμοποιώντας το Aspose.Slides.

## Εισαγωγή χώρων ονομάτων
Ξεκινήστε εισάγοντας τους απαραίτητους χώρους ονομάτων για να αξιοποιήσετε τη λειτουργικότητα του Aspose.Slides. Προσθέστε τον ακόλουθο κώδικα στην αρχή του έργου σας:
```csharp
using System.Drawing;
using System.Drawing.Imaging;
using Aspose.Slides;
```
Τώρα, ας αναλύσουμε τον παρεχόμενο κώδικα σε πολλά βήματα για μια ολοκληρωμένη κατανόηση:
## Βήμα 1: Δημιουργία αρχικού στιγμιότυπου παρουσίασης
```csharp
string dataDir = "Your Documents Directory";
// Δημιουργήστε μια κλάση παρουσίασης που αντιπροσωπεύει το αρχείο παρουσίασης
using (Presentation presentation = new Presentation(dataDir + "HelloWorld.pptx"))
{
    // Το αντικείμενο παρουσίασης είναι πλέον έτοιμο για περαιτέρω χειρισμό.
}
```
Σε αυτό το βήμα, αρχικοποιούμε το Aspose.Slides `Presentation` κλάση, που αντιπροσωπεύει το αρχείο παρουσίασης PowerPoint. Η `using` Η δήλωση διασφαλίζει την ορθή διάθεση των πόρων μόλις ολοκληρωθεί η έξοδος από το μπλοκ.
## Βήμα 2: Δημιουργήστε μια εικόνα δεσμευμένου σχήματος
```csharp
// Δημιουργήστε μια εικόνα σχήματος με δεσμό εμφάνισης
using (Bitmap bitmap = presentation.Slides[0].Shapes[0].GetThumbnail(ShapeThumbnailBounds.Appearance, 1, 1))
{
    // Το αντικείμενο bitmap περιέχει τώρα την εικόνα μικρογραφίας με καθορισμένα όρια.
}
```
Αυτό το βήμα περιλαμβάνει τη δημιουργία μιας μικρογραφίας ενός σχήματος με καθορισμένα όρια. Εδώ, `ShapeThumbnailBounds.Appearance` χρησιμοποιείται για τον ορισμό των ορίων εμφάνισης. Προσαρμόστε τις παραμέτρους (1, 1) σύμφωνα με τις απαιτήσεις σας.
## Βήμα 3: Αποθήκευση της εικόνας στο δίσκο
```csharp
// Αποθήκευση της εικόνας στο δίσκο σε μορφή PNG
bitmap.Save(dataDir + "Shape_thumbnail_Bound_Shape_out.png", ImageFormat.Png);
```
Σε αυτό το τελικό βήμα, η δημιουργημένη μικρογραφία αποθηκεύεται στον δίσκο σε μορφή PNG. Μπορείτε να προσαρμόσετε το όνομα και τη μορφή του αρχείου με βάση τις προτιμήσεις σας.

# Δημιουργία μικρογραφίας για τη σημείωση SmartArt Child στο Aspose.Slides

Στον τομέα των δυναμικών παρουσιάσεων, το Aspose.Slides για .NET ξεχωρίζει ως ένα ισχυρό εργαλείο, παρέχοντας στους προγραμματιστές τη δυνατότητα να χειρίζονται και να βελτιώνουν τις παρουσιάσεις PowerPoint μέσω προγραμματισμού. Ένα ενδιαφέρον χαρακτηριστικό είναι η δυνατότητα δημιουργίας μικρογραφιών για τις σημειώσεις SmartArt Child, προσθέτοντας ένα επίπεδο οπτικής ελκυστικότητας στις παρουσιάσεις σας. Αυτός ο οδηγός βήμα προς βήμα θα σας καθοδηγήσει στη διαδικασία δημιουργίας μικρογραφιών για τις σημειώσεις SmartArt Child χρησιμοποιώντας το Aspose.Slides για .NET.
## Προαπαιτούμενα

- Δείγμα παρουσίασης: Δημιουργήστε ή αποκτήστε μια παρουσίαση PowerPoint που περιέχει SmartArt με σημειώσεις θυγατρικών για δοκιμή.
## Εισαγωγή χώρων ονομάτων
Ξεκινήστε εισάγοντας τους απαραίτητους χώρους ονομάτων στο έργο σας σε C#. Αυτοί οι χώροι ονομάτων παρέχουν πρόσβαση στις κλάσεις και τις μεθόδους που απαιτούνται για την εργασία με το Aspose.Slides.
```csharp
using System.Drawing;
using System.Drawing.Imaging;
using Aspose.Slides.SmartArt;
using Aspose.Slides;
```
## Βήμα 1: Δημιουργία αρχικού στιγμιότυπου παρουσίασης
Ξεκινήστε δημιουργώντας ένα στιγμιότυπο του `Presentation` κλάση, που αντιπροσωπεύει το αρχείο PPTX με το οποίο θα εργαστείτε.
```csharp
string dataDir = "Your Documents Directory";
Presentation pres = new Presentation();
```
## Βήμα 2: Προσθήκη SmartArt
Τώρα, προσθέστε SmartArt σε μια διαφάνεια μέσα στην παρουσίαση. Σε αυτό το παράδειγμα, χρησιμοποιούμε το `BasicCycle` σχέδιο.
```csharp
ISmartArt smart = pres.Slides[0].Shapes.AddSmartArt(10, 10, 400, 300, SmartArtLayoutType.BasicCycle);
```
## Βήμα 3: Λήψη αναφοράς κόμβου
Για να εργαστείτε με έναν συγκεκριμένο κόμβο στο SmartArt, λάβετε την αναφορά του χρησιμοποιώντας το ευρετήριό του.
```csharp
ISmartArtNode node = smart.Nodes[1];
```
## Βήμα 4: Λήψη μικρογραφίας
Ανακτήστε την εικόνα μικρογραφίας της θυγατρικής σημείωσης μέσα στον κόμβο SmartArt.
```csharp
Bitmap bmp = node.Shapes[0].GetThumbnail();
```
## Βήμα 5: Αποθήκευση μικρογραφίας
Αποθηκεύστε τη μικρογραφία που δημιουργήθηκε σε έναν καθορισμένο κατάλογο.
```csharp
bmp.Save(dataDir + "SmartArt_ChildNote_Thumbnail_out.jpeg", ImageFormat.Jpeg);
```
Επαναλάβετε αυτά τα βήματα για κάθε κόμβο SmartArt στην παρουσίασή σας, προσαρμόζοντας τη διάταξη και τα στυλ όπως απαιτείται.

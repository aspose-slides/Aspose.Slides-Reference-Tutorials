## Οδηγός Εφαρμογής
Αφού ολοκληρωθεί η εγκατάσταση, ας εξαγάγουμε κείμενο από τους κόμβους SmartArt.

### Φόρτωση της παρουσίασης
Ξεκινήστε φορτώνοντας ένα αρχείο παρουσίασης PowerPoint. Δημιουργήστε μια παρουσία του `Presentation` τάξη και να περάσετε το μονοπάτι προς το δικό σας `.pptx` αρχείο:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string presentationPath = Path.Combine(dataDir, "Presentation.pptx");

using (Presentation presentation = new Presentation(presentationPath))
{
    // Πρόσβαση στην πρώτη διαφάνεια της παρουσίασης
    ISlide slide = presentation.Slides[0];
}
```

### Πρόσβαση στο σχήμα SmartArt
Ανάκτηση του σχήματος SmartArt από τη συλλογή σχημάτων της διαφάνειας:
```csharp
ISmartArt smartArt = (ISmartArt)slide.Shapes[0];
```
Αυτός ο κώδικας υποθέτει ότι το πρώτο σχήμα στη διαφάνεια είναι ένα αντικείμενο SmartArt. Επαληθεύστε το αυτό στις πραγματικές σας παρουσιάσεις.

### Εξαγωγή κειμένου από κόμβους
Επαναλάβετε την εκτέλεση κάθε κόμβου μέσα στο SmartArt για να αποκτήσετε πρόσβαση στα σχήματά του και να εξαγάγετε κείμενο:
```csharp
ISmartArtNodeCollection smartArtNodes = smartArt.AllNodes;

foreach (ISmartArtNode smartArtNode in smartArtNodes)
{
    foreach (ISmartArtShape nodeShape in smartArtNode.Shapes)
    {
        if (nodeShape.TextFrame != null)
        {
            // Εξαγωγή του κειμένου από το πλαίσιο κειμένου κάθε σχήματος
            Console.WriteLine(nodeShape.TextFrame.Text);
        }
    }
}
```
**Εξήγηση:**
- **`smartArtNodes`:** Αντιπροσωπεύει όλους τους κόμβους μέσα στο αντικείμενο SmartArt.
- **`nodeShape.TextFrame`:** Ελέγχει εάν ένας κόμβος έχει ένα συσχετισμένο πλαίσιο κειμένου.
- **Εξαγωγή κειμένου:** Χρήσεις `Console.WriteLine` για να εμφανίσετε το εξαγόμενο κείμενο.

#### Φόρτωση της παρουσίασης

Δείτε πώς μπορείτε να ανοίξετε μια παρουσίαση από έναν καθορισμένο κατάλογο:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY/AccessSmartArtShape.pptx";

{
    // Περαιτέρω επεξεργασία θα γίνει εδώ

```

### Διέλευση από σχήματα διαφανειών

#### Επισκόπηση

Κάθε διαφάνεια στην παρουσίασή σας περιέχει διάφορα σχήματα. Πρέπει να προσδιορίσετε ποια είναι SmartArt.

Περιηγηθείτε σε κάθε σχήμα στην πρώτη διαφάνεια για να ελέγξετε για SmartArt:

```csharp

        // Αναγνώριση και επεξεργασία σχημάτων SmartArt εδώ

```

#### Έλεγχος του τύπου διάταξης


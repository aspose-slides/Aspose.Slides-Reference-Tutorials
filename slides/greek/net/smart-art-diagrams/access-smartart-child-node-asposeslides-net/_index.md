## Οδηγός Εφαρμογής

Αυτή η ενότητα θα σας καθοδηγήσει στην πρόσβαση σε έναν συγκεκριμένο θυγατρικό κόμβο μέσα σε ένα σχήμα SmartArt σε μια παρουσίαση. Θα αναλύσουμε κάθε βήμα για να σας διευκολύνουμε να το παρακολουθήσετε.

### Προσθήκη σχήματος SmartArt

Αρχικά, πρέπει να δημιουργήσουμε μια νέα παρουσίαση και να προσθέσουμε ένα σχήμα SmartArt στην πρώτη διαφάνεια:
```csharp
using System.IO;
using Aspose.Slides;
using Aspose.Slides.SmartArt;

// Ορισμός διαδρομών καταλόγων για έγγραφα και έξοδο
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// Δημιουργήστε καταλόγους εάν δεν υπάρχουν
if (!Directory.Exists(dataDir))
    Directory.CreateDirectory(dataDir);
if (!Directory.Exists(outputDir))
    Directory.CreateDirectory(outputDir);

// Δημιουργία νέας παρουσίασης
Presentation pres = new Presentation();

// Πρόσβαση στην πρώτη διαφάνεια της παρουσίασης
ISlide slide = pres.Slides[0];

// Προσθέστε ένα σχήμα SmartArt στην πρώτη διαφάνεια στη θέση (0, 0) με μέγεθος 400x400 χρησιμοποιώντας τον τύπο διάταξης StackedList
ISmartArt smart = slide.Shapes.AddSmartArt(0, 0, 400, 400, SmartArtLayoutType.StackedList);
```

### Πρόσβαση σε συγκεκριμένο θυγατρικό κόμβο

Στη συνέχεια, θα έχουμε πρόσβαση σε έναν συγκεκριμένο θυγατρικό κόμβο μέσα στο σχήμα SmartArt:
```csharp
// Πρόσβαση στον πρώτο κόμβο του σχήματος SmartArt
ISmartArtNode node = smart.AllNodes[0];

// Καθορίστε τον δείκτη θέσης για πρόσβαση σε έναν θυγατρικό κόμβο μέσα στον γονικό κόμβο
int position = 1;
SmartArtNode chNode = (SmartArtNode)node.ChildNodes[position];

// Ανάκτηση παραμέτρων του θυγατρικού κόμβου SmartArt στον οποίο έγινε πρόσβαση
string outString = string.Format("j = {0}, Text = {1}, Level = {2}, Position = {3}", 
    position, chNode.TextFrame.Text, chNode.Level, chNode.Position);
```

**Εξήγηση:**
- **`AllNodes[0]`:** Αποκτά πρόσβαση στον πρώτο κόμβο του σχήματος SmartArt.
- **`ChildNodes[position]`:** Ανακτά έναν συγκεκριμένο θυγατρικό κόμβο με βάση το παρεχόμενο ευρετήριο. Προσαρμόστε `position` για να στοχεύσετε διαφορετικούς κόμβους.
- **Παράμετροι:** Η συμβολοσειρά εξόδου περιέχει λεπτομέρειες όπως κείμενο, επίπεδο και θέση του κόμβου στον οποίο έγινε πρόσβαση.

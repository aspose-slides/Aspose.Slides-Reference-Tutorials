## Οδηγός Εφαρμογής
Θα καλύψουμε συγκεκριμένες λειτουργίες χρησιμοποιώντας το Aspose.Slides για .NET.

### Φόρτωση παρουσίασης
Ξεκινήστε φορτώνοντας ένα υπάρχον αρχείο παρουσίασης για να προσθέσετε SmartArt ή να κάνετε τροποποιήσεις.

**Απόσπασμα κώδικα:**
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
Presentation pres = new Presentation(dataDir + "/AccessChildNodes.pptx");
```
*Εξήγηση:* Ο παραπάνω κώδικας φορτώνει ένα αρχείο PowerPoint από τον καθορισμένο κατάλογο, προετοιμάζοντάς το για περαιτέρω χειρισμό.

### Προσθήκη και τοποθέτηση ενός σχήματος SmartArt
Βελτιώστε τη διαφάνειά σας προσθέτοντας ένα σχήμα SmartArt. Αυτή η ενότητα σας καθοδηγεί στην ακριβή τοποθέτηση του SmartArt στη διαφάνειά σας.

**Επισκόπηση:**
Προσθέστε μια διάταξη SmartArt στην πρώτη διαφάνεια σε συγκεκριμένες συντεταγμένες με καθορισμένες διαστάσεις.

**Απόσπασμα κώδικα:**
```csharp
ISmartArt smart = pres.Slides[0].Shapes.AddSmartArt(20, 20, 600, 500, SmartArtLayoutType.OrganizationChart);
```
*Εξήγηση:* Ο `AddSmartArt` Η μέθοδος τοποθετεί ένα νέο σχήμα SmartArt στη διαφάνεια. Οι παράμετροι καθορίζουν τη θέση και το μέγεθός του.

**Μετακίνηση σχήματος θυγατρικού κόμβου:**
```csharp
ISmartArtNode node = smart.AllNodes[1];
ISmartArtShape shape = node.Shapes[1];
shape.X += (shape.Width * 2); // Μετακίνηση δεξιά κατά το διπλάσιο πλάτος του
shape.Y -= (shape.Height / 2); // Μετακινηθείτε προς τα πάνω κατά το μισό ύψος του
```
*Εξήγηση:* Προσαρμόστε τη θέση του σχήματος ενός συγκεκριμένου θυγατρικού κόμβου μέσα στο SmartArt.

### Ρύθμιση πλάτους και ύψους σχήματος
Τροποποιήστε τις διαστάσεις των σχημάτων ώστε να ταιριάζουν καλύτερα στις ανάγκες σχεδίασης της παρουσίασής σας.

**Απόσπασμα κώδικα:**
```csharp
node = smart.AllNodes[2];
shape = node.Shapes[1];
shape.Width += (shape.Width / 2); // Αυξήστε το πλάτος κατά το ήμισυ του αρχικού του μεγέθους

node = smart.AllNodes[3];
shape = node.Shapes[1];
shape.Height += (shape.Height / 2); // Αύξηση ύψους κατά το ήμισυ
```
*Εξήγηση:* Αυτές οι γραμμές κώδικα προσαρμόζουν τις διαστάσεις του σχήματος, ενισχύοντας την οπτική ελκυστικότητα.

### Περιστροφή ενός σχήματος SmartArt
Περιστρέψτε σχήματα για να δημιουργήσετε δυναμικές και οπτικά ενδιαφέρουσες διατάξεις.

**Απόσπασμα κώδικα:**
```csharp
node = smart.AllNodes[4];
shape = node.Shapes[1];
shape.Rotation = 90; // Περιστροφή κατά 90 μοίρες
```
*Εξήγηση:* Αυτή η απλή γραμμή κώδικα περιστρέφει το επιλεγμένο σχήμα μέσα στο SmartArt, προσθέτοντας μια δημιουργική πινελιά στη διαφάνειά σας.

### Αποθήκευση της παρουσίασης
Αφού κάνετε όλες τις αλλαγές σας, αποθηκεύστε την παρουσίαση στον επιθυμητό κατάλογο εξόδου.

**Απόσπασμα κώδικα:**
```csharp
pres.Save("YOUR_OUTPUT_DIRECTORY/SmartArt.pptx");
```
*Εξήγηση:* Ο `Save` Η μέθοδος ολοκληρώνει όλες τις τροποποιήσεις που έγιναν κατά τη διάρκεια της συνεδρίας σε ένα νέο αρχείο.

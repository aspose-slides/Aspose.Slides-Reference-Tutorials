## Οδηγός Εφαρμογής
Ας αναλύσουμε την υλοποίηση σε συγκεκριμένα χαρακτηριστικά δημιουργίας και χειρισμού SmartArt.

### Δημιουργία SmartArt με διάταξη ακτινικού κύκλου
#### Επισκόπηση
Αυτή η λειτουργία δείχνει πώς να δημιουργήσετε ένα γραφικό SmartArt χρησιμοποιώντας τη διάταξη Ακτινικού Κύκλου, ιδανική για την απεικόνιση κυκλικών διεργασιών ή διαγραμμάτων ροής στις παρουσιάσεις σας.

#### Βήμα προς βήμα εφαρμογή
**1. Αρχικοποίηση παρουσίασης**
Ξεκινήστε δημιουργώντας μια παρουσία του `Presentation` τάξη:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Ορίστε τη διαδρομή προς τον κατάλογο εγγράφων σας.
using (Presentation presentation = new Presentation())
{
    ...
}
```

**2. Προσθήκη γραφικού SmartArt**
Προσθέστε ένα γραφικό SmartArt με συγκεκριμένες συντεταγμένες και διαστάσεις χρησιμοποιώντας τη διάταξη Radial Cycle.
```csharp
ISmartArt smart = presentation.Slides[0].Shapes.AddSmartArt(10, 10, 400, 300, SmartArtLayoutType.RadialCycle);
```
- **Παράμετροι**: Το `AddSmartArt` Η μέθοδος λαμβάνει συντεταγμένες x, y, καθώς και πλάτος και ύψος για την τοποθέτηση του γραφικού.

**3. Αποθήκευση παρουσίασης**
Τέλος, αποθηκεύστε την παρουσίασή σας σε ένα αρχείο:
```csharp
presentation.Save(dataDir + "CreateSmartArt_out.pptx", SaveFormat.Pptx);
```

### Προσθήκη κόμβων στο SmartArt
#### Επισκόπηση
Μάθετε πώς να προσθέτετε δυναμικά κόμβους σε ένα υπάρχον γραφικό SmartArt, βελτιώνοντας τις λεπτομέρειες και την πληροφοριακή του αξία.

#### Βήμα προς βήμα εφαρμογή
**1. Προσθήκη κόμβου**
Αφού δημιουργήσετε το αρχικό σας SmartArt:
```csharp
ISmartArtNode node = smart.AllNodes.AddNode();
```
- **Κατανόηση των κόμβων**Οι κόμβοι αντιπροσωπεύουν μεμονωμένα στοιχεία εντός της δομής SmartArt.

### Έλεγχος κρυφής ιδιότητας κόμβου στο SmartArt
#### Επισκόπηση
Ανακαλύψτε πώς να ελέγξετε αν ένας συγκεκριμένος κόμβος είναι κρυφός, επιτρέποντας τον δυναμικό έλεγχο ορατότητας στις παρουσιάσεις σας.

#### Βήμα προς βήμα εφαρμογή
**1. Ελέγξτε την ορατότητα**
Μετά την προσθήκη ενός κόμβου:
```csharp
bool hidden = node.IsHidden; // Επιστρέφει true ή false με βάση την ορατότητα
```

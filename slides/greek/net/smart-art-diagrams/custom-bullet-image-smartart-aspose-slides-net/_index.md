## Οδηγός Εφαρμογής

### Ρύθμιση του έργου σας

1. **Αρχικοποίηση αντικειμένου παρουσίασης:**
   Ξεκινήστε δημιουργώντας ένα νέο `Presentation` αντικείμενο. Αυτό αντιπροσωπεύει το αρχείο PowerPoint σας.
   ```csharp
   using Aspose.Slides;
   using System.Drawing; // Για τον χειρισμό εικόνων
   using System.IO; // Για λειτουργίες αρχείων

   string dataDir = "YOUR_DOCUMENT_DIRECTORY";
   string outputDir = "YOUR_OUTPUT_DIRECTORY";

   Directory.CreateDirectory(dataDir);
   Directory.CreateDirectory(outputDir);

   using (Presentation presentation = new Presentation())
   {
       // Ο κώδικας συνεχίζεται...
   }
   ```

### Προσθήκη σχήματος SmartArt

2. **Προσθήκη SmartArt στη διαφάνεια:**
   Δημιουργήστε και τοποθετήστε το αντικείμενο SmartArt σας στη διαφάνεια.
   ```csharp
   ISmartArt smart = presentation.Slides[0].Shapes.AddSmartArt(10, 10, 500, 400, SmartArtLayoutType.VerticalPictureList);
   ```

3. **Πρόσβαση σε έναν κόμβο:**
   Ανακτήστε τον πρώτο κόμβο για να εφαρμόσετε προσαρμοσμένες ρυθμίσεις κουκκίδων.
   ```csharp
   ISmartArtNode node = smart.AllNodes[0];
   ```

### Προσαρμογή εικόνας κουκκίδας

4. **Ορίστε μια προσαρμοσμένη εικόνα κουκκίδων:**
   Φορτώστε και αντιστοιχίστε μια εικόνα ως κουκκίδα για τον κόμβο SmartArt σας.
   ```csharp
   if (node.BulletFillFormat != null)
   {
       string imagePath = Path.Combine(dataDir, "aspose-logo.jpg");
       IImage img = Images.FromFile(imagePath);
       IPPImage image = presentation.Images.AddImage(img);

       // Εφαρμογή της προσαρμοσμένης εικόνας κουκκίδας
       node.BulletFillFormat.FillType = FillType.Picture;
       node.BulletFillFormat.PictureFillFormat.Picture.Image = image;
       node.BulletFillFormat.PictureFillFormat.PictureFillMode = PictureFillMode.Stretch;
   }
   ```

### Αποθήκευση της παρουσίασής σας

5. **Αποθήκευση της τροποποιημένης παρουσίασης:**
   Τέλος, αποθηκεύστε την παρουσίασή σας με προσαρμοσμένο SmartArt.
   ```csharp
   string outputPath = Path.Combine(outputDir, "out.pptx");
   presentation.Save(outputPath, SaveFormat.Pptx);
   ```

## Ενότητα Συχνών Ερωτήσεων

1. **Πώς μπορώ να αλλάξω το μέγεθος της εικόνας κουκκίδων;**
   - Προσαρμόστε το `Stretch` λειτουργία για να ταιριάζει σε διαφορετικά μεγέθη ή να αλλάξετε χειροκίνητα το μέγεθος των εικόνων πριν τις προσθέσετε.

2. **Ποιες μορφές αρχείων υποστηρίζονται για προσαρμοσμένες κουκκίδες;**
   - Υποστηρίζονται κοινές μορφές όπως JPEG, PNG και BMP. Διασφαλίστε τη συμβατότητα μετατρέποντας αρχεία όπως απαιτείται.

3. **Μπορώ να εφαρμόσω αυτήν την προσαρμογή σε όλους τους κόμβους σε ένα γραφικό SmartArt;**
   - Ναι, επανάληψη `smart.AllNodes` και εφαρμόστε παρόμοιες ρυθμίσεις σε κάθε κόμβο.

4. **Τι πρέπει να κάνω εάν η εικόνα μου δεν φορτώνει;**
   - Επαληθεύστε ότι η διαδρομή του αρχείου είναι σωστή και βεβαιωθείτε ότι η εικόνα υπάρχει σε αυτήν τη θέση.

5. **Πώς μπορώ να προσαρμόσω περαιτέρω τα γραφικά SmartArt μου;**
   - Εξερευνήστε άλλα ακίνητα του/της `ISmartArt` και `ISmartArtNode` για να προσαρμόσετε χρώματα, στυλ και άλλα.

## Οδηγός Εφαρμογής

### Αρχικοποίηση παρουσίασης
Ξεκινήστε δημιουργώντας μια νέα παρουσία του `Presentation` κλάση. Αυτό αντιπροσωπεύει ένα κενό αρχείο PowerPoint όπου θα προσθέσουμε το οργανόγραμμα SmartArt.

**Βήμα 1: Δημιουργία νέου αντικειμένου παρουσίασης**
```csharp
using Aspose.Slides;
using Aspose.Slides.Export;

// Αρχικοποίηση ενός νέου αντικειμένου παρουσίασης
using (Presentation presentation = new Presentation()) {
    // Ο κώδικας για την προσθήκη SmartArt θα τοποθετηθεί εδώ
}
```

### Προσθήκη SmartArt
Τώρα, προσθέστε το οργανόγραμμα στην πρώτη σας διαφάνεια χρησιμοποιώντας `AddSmartArt`.

**Βήμα 2: Προσθήκη SmartArt**
```csharp
// Προσθήκη SmartArt με καθορισμένες συντεταγμένες, μέγεθος και τύπο διάταξης
ISmartArt smart = presentation.Slides[0].Shapes.AddSmartArt(10, 10, 400, 300, SmartArtLayoutType.OrganizationChart);
```
Αυτό το βήμα περιλαμβάνει τον καθορισμό της θέσης (`x`, `y`), διαστάσεις (πλάτος, ύψος) και τύπος διάταξης για το SmartArt σας.

### Ρύθμιση διάταξης κόμβου
Κάθε κόμβος στο οργανόγραμμα μπορεί να διαμορφωθεί ξεχωριστά. Δείτε πώς μπορείτε να ορίσετε μια προσαρμοσμένη διάταξη για τον πρώτο κόμβο.

**Βήμα 3: Ορισμός διάταξης οργανογράμματος**
```csharp
// Ορίστε τη διάταξη οργανογράμματος για τον πρώτο κόμβο
smart.Nodes[0].OrganizationChartLayout = OrganizationChartLayoutType.LeftHanging;
```

### Αποθήκευση της παρουσίασής σας
Τέλος, αποθηκεύστε την παρουσίασή σας σε ένα αρχείο. Βεβαιωθείτε ότι έχετε καθορίσει σωστά τον κατάλογο εξόδου.

**Βήμα 4: Αποθήκευση της παρουσίασης**
```csharp
// Αποθήκευση της παρουσίασης στον καθορισμένο κατάλογο εξόδου
presentation.Save(outputDir + "OrganizeChartLayoutType_out.pptx", SaveFormat.Pptx);
```

## Οδηγός Εφαρμογής

### Προσθήκη SmartArt σε μια διαφάνεια

#### Επισκόπηση
Αυτή η ενότητα δείχνει πώς να προσθέσετε ένα δυναμικό γραφικό SmartArt για να βελτιώσετε την οπτική ελκυστικότητα της παρουσίασής σας.

**Βήματα:**

##### 1. Αρχικοποίηση αντικειμένου παρουσίασης
Ξεκινήστε δημιουργώντας ένα νέο `Presentation` αντικείμενο.

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
using (Presentation presentation = new Presentation())
{
    // Αποκτήστε πρόσβαση στην πρώτη διαφάνεια της παρουσίασης.
    ISlide slide = presentation.Slides[0];
```

##### 2. Προσθήκη σχήματος SmartArt
Προσθέστε ένα σχήμα SmartArt στη διαφάνεια που επιθυμείτε, καθορίζοντας τη διάταξη και τη θέση.

```csharp
    var chevron = slide.Shapes.AddSmartArt(10, 10, 800, 60, SmartArtLayoutType.ClosedChevronProcess);
```
- **Παράμετροι:** 
  - `10, 10`: Θέση στη διαφάνεια (συντεταγμένες X, Y)
  - `800x60`: Μέγεθος του σχήματος
  - `ClosedChevronProcess`Τύπος διάταξης για δομημένη ροή

##### 3. Προσαρμογή κόμβων
Προσθέστε και προσαρμόστε κόμβους για να εμφανίσετε συγκεκριμένες πληροφορίες.

```csharp
    var node = chevron.AllNodes.AddNode();
    node.TextFrame.Text = "Some text";
}
```

### Ρύθμιση χρώματος γεμίσματος κόμβου

#### Επισκόπηση
Προσαρμόστε την εμφάνιση των κόμβων SmartArt αλλάζοντας το χρώμα γεμίσματός τους.

**Βήματα:**

##### 1. Τροποποίηση τύπου και χρώματος γεμίσματος
Επαναλάβετε τη διαδικασία μέσω κόμβων για να προσαρμόσετε τις οπτικές ιδιότητες.

```csharp
using System.Drawing;

foreach (var item in chevron.AllNodes[0].Shapes)
{
    // Αλλάξτε τον τύπο γεμίσματος σε συμπαγές και ορίστε το χρώμα σε κόκκινο.
    item.FillFormat.Τύπος Γεμίσματος = FillType.Solid;
    item.FillFormat.SolidFillColor.Color = Color.Red;
}
```
- **FillType**: Ορίζει τον τρόπο πλήρωσης του σχήματος
- **Χρώμα**: Καθορίζει το χρώμα που χρησιμοποιείται

### Αποθήκευση παρουσίασης

#### Επισκόπηση
Αποθηκεύστε την προσαρμοσμένη παρουσίασή σας σε μια καθορισμένη τοποθεσία.

**Βήματα:**

##### 1. Ορισμός καταλόγου εξόδου και αποθήκευση αρχείου

```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
presentation.Save(outputDir + "/FillFormat_SmartArt_ShapeNode_out.pptx", ΑποθήκευσηΜορφής.Pptx);
```
- **SaveFormat.Pptx**: Εξασφαλίζει ότι το αρχείο αποθηκεύεται σε μορφή PowerPoint.

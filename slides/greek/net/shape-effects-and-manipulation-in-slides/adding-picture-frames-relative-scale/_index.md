# Προσθήκη Κορνιζών με το Aspose.Slides .NET

## Εισαγωγή
Το Aspose.Slides για .NET είναι μια ισχυρή βιβλιοθήκη που επιτρέπει στους προγραμματιστές να δημιουργούν, να χειρίζονται και να μετατρέπουν παρουσιάσεις PowerPoint στις εφαρμογές τους .NET χωρίς κόπο. Σε αυτό το σεμινάριο, θα εμβαθύνουμε στη διαδικασία προσθήκης πλαισίων εικόνων με σχετικό ύψος κλίμακας χρησιμοποιώντας το Aspose.Slides για .NET. Ακολουθήστε αυτόν τον οδηγό βήμα προς βήμα για να βελτιώσετε τις δεξιότητές σας στη δημιουργία παρουσιάσεων.
## Προαπαιτούμενα
Πριν ξεκινήσουμε, βεβαιωθείτε ότι έχετε τα εξής:
- Βασική γνώση της γλώσσας προγραμματισμού C#.
- Εγκατεστημένο το Visual Studio ή οποιοδήποτε άλλο προτιμώμενο περιβάλλον ανάπτυξης C#.
- Το Aspose.Slides για τη βιβλιοθήκη .NET προστέθηκε στο έργο σας.
## Εισαγωγή χώρων ονομάτων
Ξεκινήστε εισάγοντας τους απαραίτητους χώρους ονομάτων στον κώδικα C#. Αυτό το βήμα διασφαλίζει ότι έχετε πρόσβαση στις κλάσεις και τις λειτουργίες που παρέχονται από τη βιβλιοθήκη Aspose.Slides.
```csharp
using System.Drawing;
using Aspose.Slides.Export;
using Aspose.Slides;
```
## Βήμα 1: Ρύθμιση του έργου σας
Ξεκινήστε δημιουργώντας ένα νέο έργο C# στο περιβάλλον ανάπτυξης που προτιμάτε. Βεβαιωθείτε ότι έχετε προσθέσει τη βιβλιοθήκη Aspose.Slides for .NET στο έργο σας κάνοντας αναφορά σε αυτήν.
## Βήμα 2: Φόρτωση παρουσίασης και εικόνας
```csharp
string dataDir = "Your Document Directory";
using (Presentation presentation = new Presentation())
{
    // Φόρτωση εικόνας που θα προστεθεί στη συλλογή εικόνων παρουσίασης
    Image img = new Bitmap(dataDir + "aspose-logo.jpg");
    IPPImage image = presentation.Images.AddImage(img);
    // ...
}
```
Σε αυτό το βήμα, δημιουργούμε ένα νέο αντικείμενο παρουσίασης και φορτώνουμε την εικόνα που θέλουμε να προσθέσουμε στην παρουσίαση.
## Βήμα 3: Προσθήκη πλαισίου εικόνας σε διαφάνεια
```csharp
IPictureFrame pf = presentation.Slides[0].Shapes.AddPictureFrame(ShapeType.Rectangle, 50, 50, 100, 100, image);
```
Τώρα, προσθέστε ένα πλαίσιο εικόνας στην πρώτη διαφάνεια της παρουσίασης. Προσαρμόστε τις παραμέτρους όπως τον τύπο σχήματος, τη θέση και τις διαστάσεις σύμφωνα με τις απαιτήσεις σας.
## Βήμα 4: Ορισμός σχετικού πλάτους και ύψους κλίμακας
```csharp
pf.RelativeScaleHeight = 0.8f;
pf.RelativeScaleWidth = 1.35f;
```
Ορίστε το σχετικό ύψος και πλάτος κλίμακας για το πλαίσιο εικόνας για να επιτύχετε το επιθυμητό εφέ κλιμάκωσης.
## Βήμα 5: Αποθήκευση παρουσίασης
```csharp
presentation.Save(dataDir + "Adding Picture Frame with Relative Scale_out.pptx", SaveFormat.Pptx);
```
Τέλος, αποθηκεύστε την παρουσίαση με το προστιθέμενο πλαίσιο εικόνας στην καθορισμένη μορφή εξόδου.

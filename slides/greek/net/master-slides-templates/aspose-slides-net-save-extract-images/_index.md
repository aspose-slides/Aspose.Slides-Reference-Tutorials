## Οδηγός Εφαρμογής
Τώρα που έχουμε όλα έτοιμα, ας εφαρμόσουμε τα κύρια χαρακτηριστικά μας: αποθήκευση παρουσιάσεων και εξαγωγή εικόνων.

### Αποθήκευση αρχείου παρουσίασης
**Επισκόπηση**
Η αποθήκευση μιας παρουσίασης περιλαμβάνει την εγγραφή των τροποποιημένων ή νεοδημιουργημένων διαφανειών σας στον δίσκο. Αυτό είναι απαραίτητο για τη διατήρηση των αλλαγών που έγιναν μέσω προγραμματισμού.

#### Βήμα 1: Φόρτωση της παρουσίασης
Αρχικά, φορτώστε ένα υπάρχον αρχείο PowerPoint:
```csharp
Presentation presentation = new Presentation("input.pptx");
```
Αυτό φορτώνει την παρουσίασή σας στη μνήμη, έτοιμη για τροποποιήσεις ή αποθήκευση.

#### Βήμα 2: Αποθήκευση της παρουσίασης
Στη συνέχεια, αποθηκεύστε το σε μια καθορισμένη τοποθεσία:
```csharp
presentation.Save("output.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
```
Βεβαιωθείτε ότι `YOUR_OUTPUT_DIRECTORY` αντικαθίσταται με την επιθυμητή διαδρομή. Αυτό το βήμα εγγράφει όλες τις αλλαγές πίσω στον δίσκο.

### Εξαγωγή εικόνων από μια παρουσίαση
**Επισκόπηση**
Εξαγωγή εικόνων ενσωματωμένων σε διαφάνειες για χρήση αλλού σε εφαρμογές ή για ανάλυση.

#### Βήμα 1: Πρόσβαση στη διαφάνεια
Επαναλάβετε κάθε διαφάνεια:
```csharp
foreach (ISlide slide in presentation.Slides)
{
    // Επεξεργασία κάθε διαφάνειας
}
```
Αυτός ο βρόχος σάς παρέχει πρόσβαση σε μεμονωμένες διαφάνειες και στα στοιχεία τους.

#### Βήμα 2: Εξαγωγή εικόνων
Μέσα σε κάθε διαφάνεια, εξαγάγετε εικόνες:
```csharp
int imageIndex = 0;
foreach (IPPImage img in slide.Images)
{
    using (FileStream fileStream = new FileStream($"image{imageIndex++}.png", FileMode.Create))
    {
        img.SystemImage.Save(fileStream, ImageFormat.Png);
    }
}
```
Αυτός ο κώδικας αποθηκεύει κάθε εικόνα στον δίσκο. `imageIndex` διασφαλίζει μοναδικά ονόματα αρχείων για τις εξαγόμενες εικόνες.

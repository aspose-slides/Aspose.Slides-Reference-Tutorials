## Οδηγός Εφαρμογής

Θα αναλύσουμε την υλοποίηση σε βασικά χαρακτηριστικά για να σας βοηθήσουμε να κατανοήσετε και να εφαρμόσετε αποτελεσματικά κάθε πτυχή.

### Χαρακτηριστικό: Ρύθμιση και Αρχικοποίηση

Το πρώτο βήμα είναι η αρχικοποίηση του περιβάλλοντός σας. Αυτή η ρύθμιση προετοιμάζει το Aspose.Slides για τη διαχείριση γραμματοσειρών σε παρουσιάσεις.

```csharp
using Aspose.Slides;
using System.Collections.Generic;

string dataDir = @"YOUR_DOCUMENT_DIRECTORY";

IFontFallBackRulesCollection rulesList = new FontFallBackRulesCollection();
```

**Εξήγηση**: 
- `dataDir`: Καθορίζει τον κατάλογο για τα αρχεία της παρουσίασής σας.
- `rulesList`: Ένα αντικείμενο για τη διαχείριση κανόνων εφεδρικής γραμματοσειράς.

### Χαρακτηριστικό: Προσθήκη και τροποποίηση κανόνων εφεδρικής γραμματοσειράς

Η δημιουργία και η προσαρμογή κανόνων εφεδρικής γραμματοσειράς διασφαλίζει ότι οι μη υποστηριζόμενες γραμματοσειρές αντικαθίστανται από εναλλακτικές, διατηρώντας την οπτική συνέπεια.

#### Βήμα 1: Προσθήκη βασικού κανόνα
```csharp
rulesList.Add(new FontFallBackRule(0x400, 0x4FF, "Times New Roman"));
```

**Εξήγηση**: 
- Προσθέτει έναν κανόνα για τους χαρακτήρες στο εύρος `0x400` να `0x4FF` για να χρησιμοποιήσετε την γραφή "Times New Roman".

#### Βήμα 2: Τροποποίηση υφιστάμενων κανόνων
```csharp
foreach (IFontFallBackRule fallBackRule in rulesList)
{
    // Αφαίρεση του "Tahoma" από τις εναλλακτικές επιλογές
    fallBackRule.Remove("Tahoma");

    // Προσθέστε το "Verdana" για συγκεκριμένα εύρη χαρακτήρων
    if ((fallBackRule.RangeEndIndex >= 0x4000) && (fallBackRule.RangeStartIndex < 0x5000))
        fallBackRule.AddFallBackFonts("Verdana");
}
```

**Εξήγηση**: 
- Επαναλαμβάνει τους κανόνες για να προσαρμόσει τις εφεδρικές γραμματοσειρές, αφαιρώντας το "Tahoma" και προσθέτοντας το "Verdana" για συγκεκριμένα εύρη.

#### Βήμα 3: Κατάργηση κανόνα
```csharp
if (rulesList.Count > 0)
    rulesList.Remove(rulesList[0]);
```

**Εξήγηση**: 
- Αφαιρεί με ασφάλεια τον πρώτο κανόνα, εάν υπάρχει, δείχνοντας πώς να διαχειρίζεστε δυναμικά τη λίστα κανόνων σας.

### Χαρακτηριστικό: Επεξεργασία παρουσίασης με εφεδρικούς κανόνες γραμματοσειράς

Η εφαρμογή αυτών των κανόνων σε μια παρουσίαση διασφαλίζει ότι όλες οι διαφάνειες αποδίδονται με τις σωστές γραμματοσειρές.

```csharp
using (Presentation pres = new Presentation(dataDir + "input.pptx"))
{
    // Αντιστοίχιση κανόνων εφεδρικής γραμματοσειράς στον διαχειριστή γραμματοσειρών της παρουσίασης
    pres.FontsManager.FontFallBackRulesCollection = rulesList;
    
    // Απόδοση και αποθήκευση της πρώτης διαφάνειας ως εικόνα PNG
    pres.Slides[0].GetImage(1f, 1f).Save(dataDir + "Slide_0.png");
}
```

**Εξήγηση**: 
- Φορτώνει μια παρουσίαση και αντιστοιχίζει το `rulesList` στον διαχειριστή γραμματοσειρών του.
- Αποδίδει την πρώτη διαφάνεια χρησιμοποιώντας τους καθορισμένους κανόνες και την αποθηκεύει ως εικόνα.

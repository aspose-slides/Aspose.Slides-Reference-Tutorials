## Οδηγός Εφαρμογής

Τώρα που όλα έχουν ρυθμιστεί, ας αναλύσουμε τις λειτουργίες σε διαχειρίσιμα βήματα.

### Προσδιορισμός μορφής αρχείου PowerPoint

#### Επισκόπηση
Αυτή η λειτουργία βοηθά στον εντοπισμό του εάν ένα αρχείο PowerPoint χρησιμοποιεί μια παλαιότερη μορφή όπως το PPT95, επιτρέποντάς σας να το χειριστείτε κατάλληλα στην εφαρμογή σας.

#### Βήματα:

**1. Εισαγωγή Aspose.Slides**
```csharp
using Aspose.Slides;
```

**2. Φόρτωση πληροφοριών παρουσίασης**
```csharp
String dataDir = "YOUR_DOCUMENT_DIRECTORY/presentation.ppt"; // Ενημέρωση με τη διαδρομή αρχείου σας

// Ανάκτηση πληροφοριών παρουσίασης για προσδιορισμό της μορφής
PresentationInfo presentationInfo = PresentationFactory.Instance.getPresentationInfo(dataDir);
```

**3. Ελέγξτε τη μορφή**
```csharp
bool isOldFormat = presentationInfo.getLoadFormat() == LoadFormat.Ppt95;

if (isOldFormat) {
    Console.WriteLine("The file is in an older PPT format.");
} else {
    Console.WriteLine("The file is not in the old PPT format.");
}
```

**Εξήγηση:** Ο `PresentationFactory` Η κλάση παρέχει πληροφορίες σχετικά με την παρουσίαση, συμπεριλαμβανομένης της μορφής της. Έλεγχος σε σχέση με `LoadFormat.Ppt95` μας λέει αν είναι παλαιότερη έκδοση.

#### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι η διαδρομή του αρχείου σας είναι σωστή και προσβάσιμη.
- Χειριστείτε εξαιρέσεις που ενδέχεται να προκύψουν από μη υποστηριζόμενες μορφές, τυλίγοντας κώδικα σε μπλοκ try-catch.

### Προσθήκη νέας διαφάνειας σε μια παρουσίαση

#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να προσθέσετε εύκολα μια νέα διαφάνεια σε μια υπάρχουσα παρουσίαση PowerPoint, χρησιμοποιώντας την πρώτη διαθέσιμη διάταξη.

#### Βήματα:

**1. Εισαγωγή Aspose.Slides**
```csharp
using Aspose.Slides;
```

**2. Φόρτωση υπάρχουσας παρουσίασης**
```csharp
String dataDir = "YOUR_DOCUMENT_DIRECTORY/presentation.pptx"; // Ενημέρωση με τη διαδρομή αρχείου σας

// Άνοιγμα της υπάρχουσας παρουσίασης
Presentation pres = new Presentation(dataDir);
```

**3. Προσθήκη νέας διαφάνειας**
```csharp
ISlide slide = pres.getSlides().addEmptySlide(pres.getLayoutSlides().get_Item(0));

pres.save("YOUR_OUTPUT_DIRECTORY/ModifiedPresentation.pptx", SaveFormat.Pptx);

Console.WriteLine("New slide added successfully.");
```

**Εξήγηση:** Ο `Slides` συλλογή μέσα σε ένα `Presentation` Το αντικείμενο επιτρέπει την προσθήκη νέων διαφανειών. Εδώ, χρησιμοποιούμε την πρώτη διαφάνεια διάταξης ως πρότυπο.

#### Συμβουλές αντιμετώπισης προβλημάτων
- Επαληθεύστε ότι ο κατάλογος εξόδου υπάρχει και είναι εγγράψιμος.
- Βεβαιωθείτε ότι η παρουσίαση εισόδου σας δεν είναι κλειδωμένη ή κατεστραμμένη.

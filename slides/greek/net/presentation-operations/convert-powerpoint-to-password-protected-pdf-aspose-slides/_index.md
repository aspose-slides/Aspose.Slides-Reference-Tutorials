## Οδηγός Εφαρμογής

Ακολουθήστε αυτά τα βήματα για να μετατρέψετε μια παρουσίαση PowerPoint σε PDF που προστατεύεται με κωδικό πρόσβασης.

### Βήμα 1: Ορίστε τον κατάλογο εγγράφων σας

Ορίστε τη διαδρομή καταλόγου εγγράφων όπου αποθηκεύονται τα αρχεία PPTX:
```csharp
string dataDir = "/path/to/your/documents";
```

### Βήμα 2: Φόρτωση του αρχείου παρουσίασης

Δημιουργήστε μια παρουσία του `Presentation` κλάση που αναπαραστήσει το αρχείο PowerPoint που θέλετε να μετατρέψετε:
```csharp
using (Presentation presentation = new Presentation(dataDir + "/DemoFile.pptx"))
{
    // Περαιτέρω επεξεργασία εδώ
}
```
Ο `using` Η δήλωση διασφαλίζει ότι οι πόροι απορρίπτονται σωστά μετά τη χρήση.

### Βήμα 3: Ρύθμιση παραμέτρων επιλογών PDF

Δημιουργήστε μια παρουσία του `PdfOptions` κλάση για να καθορίσετε επιλογές για την έξοδο PDF, συμπεριλαμβανομένου του ορισμού κωδικού πρόσβασης:
```csharp
PdfOptions pdfOptions = new PdfOptions();
pdfOptions.Password = "your_password"; // Ορίστε τον επιθυμητό κωδικό πρόσβασης εδώ
```
Αυτό το βήμα διασφαλίζει ότι μόνο οι χρήστες με τον σωστό κωδικό πρόσβασης μπορούν να έχουν πρόσβαση στο PDF.

### Βήμα 4: Αποθήκευση ως PDF με προστασία κωδικού πρόσβασης

Αποθηκεύστε την παρουσίαση σε μορφή PDF χρησιμοποιώντας τις διαμορφωμένες επιλογές:
```csharp
string outputPath = "/path/to/output/PasswordProtectedPDF_out.pdf";
presentation.Save(outputPath, SaveFormat.Pdf, pdfOptions);
```
Ο `Save` Η μέθοδος γράφει το αρχείο εξόδου στον καθορισμένο κατάλογο.

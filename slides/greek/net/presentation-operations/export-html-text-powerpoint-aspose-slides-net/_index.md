## Οδηγός Εφαρμογής

### Εξαγωγή κειμένου HTML από μια διαφάνεια PowerPoint

Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε κείμενο από συγκεκριμένες διαφάνειες σε μορφή HTML. Δείτε πώς λειτουργεί:

#### Βήμα 1: Φόρτωση της παρουσίασής σας

Αρχικά, φορτώστε το αρχείο παρουσίασής σας χρησιμοποιώντας το `Presentation` τάξη.

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Ορίστε τη διαδρομή του καταλόγου εγγράφων σας

using (Presentation pres = new Presentation(dataDir + "/ExportingHTMLText.pptx"))
{
    // Συνεχίστε με την πρόσβαση σε διαφάνειες και σχήματα...
}
```

#### Βήμα 2: Πρόσβαση στην επιθυμητή διαφάνεια

Αποκτήστε πρόσβαση στη διαφάνεια από την οποία θέλετε να εξαγάγετε κείμενο. Σε αυτό το παράδειγμα, θα έχουμε πρόσβαση στην πρώτη διαφάνεια.

```csharp
ISlide slide = pres.Slides[0];
```

#### Βήμα 3: Ανάκτηση και εξαγωγή κειμένου ως HTML

Ανακτήστε το σχήμα που περιέχει το κείμενό σας και χρησιμοποιήστε το `ExportToHtml` μέθοδος για να το μετατρέψετε σε μορφή HTML.

```csharp
int index = 0;
IAutoShape ashape = (IAutoShape)slide.Shapes[index];

using (StreamWriter sw = new StreamWriter(dataDir + "/output_out.html", false, Encoding.UTF8))
{
    // Εξαγωγή παραγράφων ως HTML
    sw.Write(ashape.TextFrame.Paragraphs.ExportToHtml(0, ashape.TextFrame.Paragraphs.Count, null));
}
```

**Εξήγηση**: 
- **`IAutoShape`**: Αναπαριστά ένα σχήμα με κείμενο. Το ανακτούμε από τη συλλογή σχημάτων της διαφάνειας.
- **`ExportToHtml` Μέθοδος**Μετατρέπει παραγράφους σε HTML. Οι παράμετροι ορίζουν τον δείκτη έναρξης και τον αριθμό των παραγράφων.

### Συμβουλές αντιμετώπισης προβλημάτων

- Βεβαιωθείτε ότι το σχήμα στο οποίο αποκτάτε πρόσβαση περιέχει ένα πλαίσιο κειμένου με παραγράφους.

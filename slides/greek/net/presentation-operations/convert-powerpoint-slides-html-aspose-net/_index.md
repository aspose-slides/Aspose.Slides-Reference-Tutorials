## Οδηγός Εφαρμογής

Ας αναλύσουμε τη διαδικασία σε διαχειρίσιμα βήματα για τη μετατροπή μεμονωμένων διαφανειών του PowerPoint σε HTML.

### Μετατροπή μεμονωμένων διαφανειών
**Επισκόπηση:**
Αυτή η λειτουργία σάς επιτρέπει να εξαγάγετε κάθε διαφάνεια από μια παρουσίαση PowerPoint και να την αποθηκεύσετε ως ανεξάρτητο αρχείο HTML, προσφέροντας ευελιξία στην ενσωμάτωση στο web.

#### Βήμα 1: Ορίστε τις διαδρομές των εγγράφων σας
Ορίστε τις διαδρομές εισόδου και εξόδου για τα αρχεία παρουσίασης:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY/Individual-Slide.pptx";
```

#### Βήμα 2: Φόρτωση της παρουσίασης
Φορτώστε το αρχείο PowerPoint χρησιμοποιώντας το Aspose.Slides:

```csharp
using (Presentation presentation = new Presentation(dataDir))
{
    // Συνεχίστε με τα βήματα μετατροπής εδώ...
}
```

*Γιατί;*Αυτό το βήμα διασφαλίζει ότι η παρουσίασή σας είναι έτοιμη για επεξεργασία σε ένα περιβάλλον διαχειριζόμενων πόρων.

#### Βήμα 3: Ρύθμιση παραμέτρων επιλογών HTML
Ρυθμίστε τις επιλογές μορφοποίησης HTML για να προσαρμόσετε την έξοδο:

```csharp
HtmlOptions htmlOptions = new HtmlOptions();
htmlOptions.HtmlFormatter = HtmlFormatter.CreateCustomFormatter(new CustomFormattingController());
```

*Γιατί;*: Η προσαρμογή αυτών των ρυθμίσεων σάς επιτρέπει να διαχειριστείτε τον τρόπο απόδοσης των διαφανειών σε HTML, συμπεριλαμβανομένης της διάταξης και των σημειώσεων.

#### Βήμα 4: Ορισμός θέσης σημειώσεων
Προσαρμόστε τη θέση τυχόν σημειώσεων διαφανειών:

```csharp
INotesCommentsLayoutingOptions notesOptions = new NotesCommentsLayoutingOptions();
notesOptions.NotesPosition = NotesPositions.BottomFull;
htmlOptions.SlidesLayoutOptions = notesOptions;
```

*Γιατί;*Αυτό διασφαλίζει ότι οι σημειώσεις σας περιλαμβάνονται και μορφοποιούνται σωστά στην έξοδο HTML.

#### Βήμα 5: Αποθήκευση κάθε διαφάνειας ως HTML
Επαναλάβετε κάθε διαφάνεια για να την αποθηκεύσετε ξεχωριστά:

```csharp
for (int i = 0; i < presentation.Slides.Count; i++)
{
    string outputFilePath = "YOUR_OUTPUT_DIRECTORY/Individual_Slide" + (i + 1) + ".html";
    presentation.Save(outputFilePath, new[] { i + 1 }, SaveFormat.Html, htmlOptions);
}
```

*Γιατί;*: Αυτός ο βρόχος επεξεργάζεται κάθε διαφάνεια ξεχωριστά, επιτρέποντας την προσθήκη προσαρμοσμένων αρχείων HTML ανά διαφάνεια.

### Ελεγκτής προσαρμοσμένης μορφοποίησης για μετατροπή HTML
**Επισκόπηση:**
Υλοποιήστε έναν προσαρμοσμένο ελεγκτή για να τροποποιήσετε την έξοδο HTML, βελτιώνοντας τον έλεγχο της μορφής και της δομής των διαφανειών σας σε HTML.

#### Υλοποίηση του CustomController
Ορίστε πώς θέλετε να μορφοποιηθεί η αρχή και το τέλος κάθε διαφάνειας:

```csharp
class CustomFormattingController : IHtmlFormattingController
{
    void IHtmlFormattingController.WriteDocumentStart(IHtmlGenerator generator, IPresentation presentation) {}

    void IHtmlFormattingController.WriteDocumentEnd(IHtmlGenerator generator, IPresentation presentation) {}

    void IHtmlFormattingController.WriteSlideStart(IHtmlGenerator generator, ISlide slide)
    {
        generator.AddHtml(string.Format(SlideHeader, generator.SlideIndex + 1));
    }

    void IHtmlFormattingController.WriteSlideEnd(IHtmlGenerator generator, ISlide slide)
    {
        generator.AddHtml(SlideFooter);
    }

    private const string SlideHeader = "<div class=\"slide\" name=\"slide\" id=\"slide{0}\">";
    private const string SlideFooter = "</div>";
}
```

*Γιατί;*Αυτή η προσαρμογή σάς επιτρέπει να εισάγετε συγκεκριμένες ετικέτες HTML στην αρχή και στο τέλος κάθε διαφάνειας, διασφαλίζοντας ομοιόμορφο στυλ σε όλα τα αρχεία που έχετε μετατρέψει.

**Επισκόπηση:** Υπολογίστε την κατεύθυνση μιας γραμμής με βάση τις διαστάσεις της και τις ιδιότητες αναστροφής της.

Καθορίστε τιμές πλάτους, ύψους και λογικών τιμών που υποδεικνύουν οριζόντιες ή κάθετες αναστροφές:

```csharp
float width = /* η αξία σου */;

```

#### Βήμα 2: Υπολογισμός κατεύθυνσης

Χρησιμοποιήστε τη συνάρτηση τόξου εφαπτομένης για να προσδιορίσετε τη γωνία μεταξύ της γραμμής και του άξονα y και, στη συνέχεια, ομαλοποιήστε την:

```csharp
class LineDirectionCalculator
{
    public static double CalculateDirection(float width, float height, bool flipH, bool flipV)
    {
        float endLineX = width * (flipH ? -1 : 1);
        float endLineY = height * (flipV ? -1 : 1);

        float endYAxisX = 0;
        float endYAxisY = height;

        double angle = (Math.Atan2(endYAxisY, endYAxisX) - Math.Atan2(endLineY, endLineX));

        if (angle < 0) angle += 2 * Math.PI;

        return angle * 180.0 / Math.PI;
    }
}
```

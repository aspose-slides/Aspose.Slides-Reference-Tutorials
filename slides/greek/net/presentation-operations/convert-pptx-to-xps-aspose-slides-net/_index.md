## Οδηγός Εφαρμογής

### Μετατροπή PPTX σε XPS χωρίς πρόσθετες επιλογές

Αυτή η ενότητα σάς καθοδηγεί στη μετατροπή μιας παρουσίασης PowerPoint σε έγγραφο XPS χρησιμοποιώντας τις προεπιλεγμένες ρυθμίσεις.

#### Βήμα 1: Φόρτωση του αρχείου παρουσίασης

Ξεκινήστε φορτώνοντας το αρχείο .pptx. Αυτό το βήμα αρχικοποιεί το `Presentation` αντικείμενο με την καθορισμένη διαδρομή αρχείου:
```csharp
using Aspose.Slides;

string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Αντικατάσταση με τον πραγματικό κατάλογο
string outputDir = "YOUR_OUTPUT_DIRECTORY"; // Επιθυμητή διαδρομή εξόδου

// Φόρτωση της παρουσίασης
using (Presentation pres = new Presentation(dataDir + "/Convert_XPS.pptx"))
{
    // Τα ακόλουθα βήματα θα επεξεργαστούν αυτήν την φορτωμένη παρουσίαση.
}
```

#### Βήμα 2: Αποθήκευση ως έγγραφο XPS

Μόλις φορτωθεί η παρουσίασή σας, αποθηκεύστε την σε μορφή XPS χρησιμοποιώντας `SaveFormat.Xps`:
```csharp
using Aspose.Slides.Export;

// Αποθήκευση της παρουσίασης ως έγγραφο XPS χωρίς πρόσθετες επιλογές
pres.Save(outputDir + "/XPS_Output_Without_XPSOption_out.xps", SaveFormat.Xps);
```

### Βασικές Σκέψεις
- **Μεταβλητές διαδρομής:** Εξασφαλίζω `dataDir` και `outputDir` έχουν οριστεί σωστά στις διαδρομές των αρχείων σας.
- **Δικαιώματα αρχείων:** Επαληθεύστε ότι η εφαρμογή σας διαθέτει τα απαραίτητα δικαιώματα για ανάγνωση και εγγραφή σε αυτούς τους καταλόγους.

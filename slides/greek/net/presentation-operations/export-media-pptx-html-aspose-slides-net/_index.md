## Οδηγός Εφαρμογής

Θα αναλύσουμε τη διαδικασία εξαγωγής αρχείων πολυμέσων σε διαχειρίσιμες ενότητες.

### Βήμα 1: Ορισμός διαδρομών καταλόγου και αρχικοποίηση μεταβλητών

Ξεκινήστε ορίζοντας τις διαδρομές του εγγράφου και του καταλόγου εξόδου. Επίσης, καθορίστε το όνομα αρχείου για την έξοδο HTML:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Αντικαταστήστε με την πραγματική σας διαδρομή
string outputDir = "YOUR_OUTPUT_DIRECTORY"; // Αντικαταστήστε με την επιθυμητή διαδρομή εξόδου
const string fileName = "ExportMediaFiles_out.html";
const string baseUri = "http://www.example.com/";
```

### Βήμα 2: Φόρτωση της παρουσίασης PowerPoint

Δημιουργήστε μια παρουσία του `Presentation` κλάση για να φορτώσετε το αρχείο PPTX σας:

```csharp
using (Presentation pres = new Presentation(dataDir + "/Media File.pptx"))
{
    // Συνέχεια στην περαιτέρω εφαρμογή...
}
```
**Γιατί αυτό το βήμα;**Η φόρτωση της παρουσίασης είναι ζωτικής σημασίας, καθώς σας επιτρέπει να έχετε πρόσβαση και να χειρίζεστε το περιεχόμενο πολυμέσων της.

### Βήμα 3: Αρχικοποίηση ελεγκτή HTML

Χρήση `VideoPlayerHtmlController` για να διαχειριστείτε τον τρόπο ενσωμάτωσης των αρχείων πολυμέσων στον κώδικα HTML σας:

```csharp
VideoPlayerHtmlController controller = new VideoPlayerHtmlController(outputDir, fileName, baseUri);
```
**Γιατί αυτό το βήμα;**Ο ελεγκτής διευκολύνει τη διαδικασία μετατροπής χειριζόμενος διαμορφώσεις και ενσωμάτωση ειδικά για κάθε μέσο.

### Βήμα 4: Ρύθμιση παραμέτρων επιλογών HTML

Στήνω `HtmlOptions` για να προσαρμόσετε τον τρόπο εξαγωγής των διαφανειών:

```csharp
HtmlOptions htmlOptions = new HtmlOptions(controller);
SVGOptions svgOptions = new SVGOptions(controller);

// Ορισμός προσαρμοσμένου μορφοποιητή και μορφής εικόνας διαφάνειας
htmlOptions.HtmlFormatter = HtmlFormatter.CreateCustomFormatter(controller);
htmlOptions.SlideImageFormat = SlideImageFormat.Svg(svgOptions);
```
**Γιατί αυτό το βήμα;**Η σωστή διαμόρφωση διασφαλίζει ότι το HTML που προκύπτει διατηρεί την οπτική πιστότητα και λειτουργικότητα.

### Βήμα 5: Εξαγωγή σε HTML

Τέλος, αποθηκεύστε την παρουσίασή σας ως αρχείο HTML:

```csharp
pres.Save(Path.Combine(outputDir, fileName), SaveFormat.Html, htmlOptions);
```
**Γιατί αυτό το βήμα;**: Εδώ συνδυάζονται όλες οι διαμορφώσεις για να παραχθεί το τελικό αποτέλεσμα σε μια φιλική προς το διαδίκτυο μορφή.

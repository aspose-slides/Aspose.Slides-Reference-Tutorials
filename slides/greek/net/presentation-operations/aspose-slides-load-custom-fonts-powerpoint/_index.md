## Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, θα εξερευνήσουμε τον τρόπο φόρτωσης μιας παρουσίασης PowerPoint χρησιμοποιώντας προσαρμοσμένες ρυθμίσεις γραμματοσειράς.

### Φόρτωση παρουσίασης με προσαρμοσμένες γραμματοσειρές

#### Επισκόπηση

Η φόρτωση παρουσιάσεων με συγκεκριμένες γραμματοσειρές διασφαλίζει ότι οι διαφάνειές σας εμφανίζουν το κείμενο ακριβώς όπως προβλέπεται. Αυτό είναι κρίσιμο για τη διατήρηση της ακεραιότητας της επωνυμίας και της οπτικής συνέπειας σε όλα τα έγγραφα.

#### Βήματα

**1. Ορίστε τον Κατάλογο Εγγράφων**

Αρχικά, καθορίστε πού βρίσκονται τα αρχεία σας:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
```

**2. Φόρτωση γραμματοσειρών στη μνήμη**

Φορτώστε προσαρμοσμένες γραμματοσειρές από τον τοπικό χώρο αποθήκευσης στη μνήμη για να βεβαιωθείτε ότι είναι διαθέσιμες όταν χρειάζονται:

```csharp
byte[] memoryFont1 = File.ReadAllBytes("customfonts\\CustomFont1.ttf");
byte[] memoryFont2 = File.ReadAllBytes("customfonts\\CustomFont2.ttf");
```

**3. Ρύθμιση επιλογών φόρτωσης**

Ρυθμίστε τις επιλογές φόρτωσης για να καθορίσετε τις πηγές γραμματοσειρών:

```csharp
LoadOptions loadOptions = new LoadOptions();
loadOptions.DocumentLevelFontSources.FontFolders = new string[] { "assets\\fonts", "global\\fonts" };
loadOptions.DocumentLevelFontSources.MemoryFonts = new byte[][] { memoryFont1, memoryFont2 };
```

**4. Φόρτωση της παρουσίασης**

Αφού έχετε προετοιμάσει τις γραμματοσειρές σας και έχετε ρυθμίσει τις επιλογές φόρτωσης, μπορείτε τώρα να φορτώσετε την παρουσίασή σας:

```csharp
using (IPresentation presentation = new Presentation("MyPresentation.pptx", loadOptions))
{
    // Η παρουσίαση είναι φορτωμένη με συγκεκριμένες προσαρμοσμένες γραμματοσειρές.
}
```

#### Εξήγηση

- **`LoadOptions`:** Ορίζει τους καταλόγους προέλευσης γραμματοσειρών και τις γραμματοσειρές που φορτώνονται στη μνήμη.
- **`MemoryFonts`:** Πίνακας από πίνακες byte που αναπαριστούν γραμματοσειρές που έχουν φορτωθεί στη μνήμη.

### Συμβουλές αντιμετώπισης προβλημάτων

Εάν οι γραμματοσειρές σας δεν εμφανίζονται σωστά, βεβαιωθείτε ότι:
- Τα αρχεία γραμματοσειρών βρίσκονται σωστά σε καθορισμένους καταλόγους ή διαδρομές.
- Τα δεδομένα πίνακα byte αντιπροσωπεύουν με ακρίβεια τα περιεχόμενα του αρχείου γραμματοσειράς.

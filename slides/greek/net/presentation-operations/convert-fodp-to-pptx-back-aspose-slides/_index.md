## Οδηγός Εφαρμογής

### Φόρτωση και αποθήκευση αρχείου FODP ως PPTX

#### Επισκόπηση
Φορτώστε ένα υπάρχον αρχείο FODP στην εφαρμογή σας και αποθηκεύστε το ως αρχείο PPTX, ιδανικό για την κοινή χρήση παρουσιάσεων στην ευρέως υποστηριζόμενη μορφή PowerPoint.

#### Βήματα
**Βήμα 1: Φόρτωση του αρχείου FODP**
Δημιουργήστε ένα `Presentation` αντικείμενο φορτώνοντας το αρχείο FODP σας:
```csharp
using System.IO;
using Aspose.Slides;

string fodpFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "Example.fodp");

// Φορτώστε το αρχείο FODP σε ένα αντικείμενο παρουσίασης.
using (Presentation presentation = new Presentation(fodpFilePath))
{
    // Το αντικείμενο Presentation περιέχει πλέον το περιεχόμενο FODP σας.
}
```
**Βήμα 2: Αποθήκευση ως PPTX**
Αποθηκεύστε την παρουσίαση που φορτώθηκε σε μορφή PPTX:
```csharp
string pptxOutputPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "FodpToPptxConversion.pptx");

// Αποθηκεύστε την παρουσίαση που φορτώθηκε ως αρχείο PPTX.
presentation.Save(pptxOutputPath, SaveFormat.Pptx);
```
### Μετατροπή PPTX πίσω σε μορφή FODP

#### Επισκόπηση
Η μετατροπή ενός αρχείου PPTX σε μορφή FODP διατηρεί συγκεκριμένες δυνατότητες ή μεταδεδομένα μοναδικά για τη μορφή FODP.

#### Βήματα
**Βήμα 1: Φόρτωση του αρχείου PPTX**
Φορτώστε το αρχείο PPTX σας σε ένα `Presentation` αντικείμενο:
```csharp
string pptxFilePath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "FodpToPptxConversion.pptx");

// Φορτώστε το αρχείο PPTX σε ένα αντικείμενο παρουσίασης.
using (Presentation pres = new Presentation(pptxFilePath))
{
    // Το αντικείμενο Presentation περιέχει πλέον το περιεχόμενο PPTX σας
}
```
**Βήμα 2: Αποθήκευση ως FODP**
Αποθηκεύστε ξανά την παρουσίαση σε μορφή FODP:
```csharp
string fodpOutputPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "PptxToFodpConversion.fodp");

// Αποθηκεύστε την παρουσίαση που φορτώθηκε ως αρχείο FODP.
pres.Save(fodpOutputPath, SaveFormat.Fodp);
```

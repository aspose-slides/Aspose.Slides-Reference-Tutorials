## Οδηγός Εφαρμογής
### Ρύθμιση διάστιχου σε διαφάνειες PowerPoint
Η ρύθμιση της απόστασης μεταξύ των γραμμών είναι ζωτικής σημασίας για την τελειοποίηση των διαφανειών και την καλύτερη αναγνωσιμότητα του κειμένου. Ακολουθήστε αυτά τα βήματα χρησιμοποιώντας το Aspose.Slides .NET.
#### Βήμα 1: Ρύθμιση διαδρομών εγγράφων
Ορίστε πού βρίσκεται το έγγραφο εισόδου σας και πού θα αποθηκευτεί το αρχείο εξόδου:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";
```
Αυτό το βήμα ορίζει διαδρομές για τη φόρτωση μιας υπάρχουσας παρουσίασης και την αποθήκευση τροποποιήσεων.
#### Βήμα 2: Φόρτωση παρουσίασης
Φόρτωση ενός αρχείου PowerPoint που περιέχει κείμενο για μορφοποίηση:
```csharp
// Φόρτωση παρουσίασης με συγκεκριμένες γραμματοσειρές
document.Presentation presentation = new Presentation(dataDir + "/Fonts.pptx");
```
Αυτή η μέθοδος φορτώνει την παρουσίασή σας για προγραμματιστικό χειρισμό.
#### Βήμα 3: Πρόσβαση στη διαφάνεια
Αποκτήστε πρόσβαση στη διαφάνεια όπου θέλετε να προσαρμόσετε την απόσταση κειμένου. Θα επικεντρωθούμε στην πρώτη διαφάνεια:
```csharp
ISlide sld = presentation.Slides[0];
```
#### Βήμα 4: Ανάκτηση του TextFrame
Ανάκτηση ενός `TextFrame` για να αποκτήσετε πρόσβαση και να τροποποιήσετε κείμενο μέσα σε σχήματα:
```csharp
ITextFrame tf1 = ((IAutoShape)sld.Shapes[0]).TextFrame;
```
Υποθέτοντας ότι το πρώτο σχήμα στη διαφάνεια είναι ένα Αυτόματο Σχήμα που περιέχει κείμενο.
#### Βήμα 5: Πρόσβαση στην παράγραφο
Αποκτήστε πρόσβαση στην παράγραφο για τροποποίηση, επιτρέποντας μεμονωμένες προσαρμογές απόστασης:
```csharp
IParagraph para1 = tf1.Paragraphs[0];
```
#### Βήμα 6: Ρύθμιση παραμέτρων ιδιοτήτων απόστασης
Ορίστε τις ιδιότητες διάστιχου μεταξύ γραμμών για να βελτιώσετε την αναγνωσιμότητα:
```csharp
para1.ParagraphFormat.SpaceWithin = 80; // Διάστημα γραμμής μέσα στην ίδια παράγραφο
para1.ParagraphFormat.SpaceBefore = 40; // Κενό πριν από την έναρξη της παραγράφου
para1.ParagraphFormat.SpaceAfter = 40;  // Κενό μετά το τέλος της παραγράφου
```
Ο `SpaceWithin` η παράμετρος ελέγχει την απόσταση μεταξύ των γραμμών σε μια παράγραφο, ενώ `SpaceBefore` και `SpaceAfter` έλεγχο του περιβάλλοντος χώρου.
#### Βήμα 7: Αποθήκευση τροποποιημένης παρουσίασης
Αποθηκεύστε την παρουσίασή σας με τις αλλαγές που εφαρμόστηκαν:
```csharp
document.Presentation.Save(outputDir + "/LineSpacing_out.pptx", SaveFormat.Pptx);
```
Αυτό εγγράφει την τροποποιημένη παρουσίαση σε ένα νέο αρχείο στον καθορισμένο κατάλογο εξόδου.

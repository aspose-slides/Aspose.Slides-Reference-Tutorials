## Οδηγός Εφαρμογής
Αυτή η ενότητα χωρίζεται σε δύο κύριες λειτουργίες: Δημιουργία καταλόγου και Μορφοποίηση κειμένου σε παρουσίαση PowerPoint. Κάθε λειτουργία περιλαμβάνει έναν λεπτομερή οδηγό υλοποίησης.

### Χαρακτηριστικό 1: Δημιουργία καταλόγου
#### Επισκόπηση
Αυτή η λειτουργικότητα διασφαλίζει ότι η εφαρμογή σας μπορεί να ελέγξει μέσω προγραμματισμού εάν υπάρχει ένας κατάλογος και να τον δημιουργήσει εάν όχι, διασφαλίζοντας ότι οι απαραίτητες διαδρομές αρχείων είναι διαθέσιμες για την αποθήκευση παρουσιάσεων ή άλλων αρχείων.

#### Βήματα Υλοποίησης
##### Βήμα 1: Ορίστε τη διαδρομή καταλόγου
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
```

##### Βήμα 2: Έλεγχος ύπαρξης καταλόγου
```csharp
bool isExists = Directory.Exists(dataDir);
if (!isExists)
{
    // Δημιουργήστε έναν κατάλογο εάν δεν υπάρχει
    Directory.CreateDirectory(dataDir);
}
```
**Εξήγηση**: Το `Directory.Exists` Η μέθοδος ελέγχει την ύπαρξη ενός καταλόγου στην καθορισμένη διαδρομή. Εάν επιστρέψει `false`, `Directory.CreateDirectory` δημιουργεί τον κατάλογο, διασφαλίζοντας ότι η εφαρμογή σας διαθέτει μια έγκυρη θέση αποθήκευσης.

### Χαρακτηριστικό 2: Μορφοποίηση κειμένου σε παρουσίαση PowerPoint
#### Επισκόπηση
Αυτή η λειτουργία δείχνει πώς να δημιουργήσετε μια νέα παρουσίαση, να προσθέσετε ένα Αυτόματο Σχήμα με κείμενο και να εφαρμόσετε διάφορα στυλ μορφοποίησης, όπως αλλαγές γραμματοσειράς, έντονη γραφή, πλάγια γραφή, υπογράμμιση, μέγεθος γραμματοσειράς και χρώμα.

#### Βήματα Υλοποίησης
##### Βήμα 1: Δημιουργήστε την Κλάση Παρουσίασης
```csharp
using (Presentation pres = new Presentation())
{
    // Συνεχίστε για να προσθέσετε μια διαφάνεια και ένα σχήμα...
}
```
**Εξήγηση**: Το `Presentation` η κλάση αρχικοποιεί μια νέα παρουσίαση PowerPoint. Χρησιμοποιώντας το `using` Η δήλωση διασφαλίζει ότι οι πόροι διατίθενται σωστά μόλις εξέλθει το πεδίο εφαρμογής.

##### Βήμα 2: Προσθήκη Αυτόματου Σχήματος με Κείμενο
```csharp
ISlide sld = pres.Slides[0];
IAutoShape ashp = sld.Shapes.AddAutoShape(ShapeType.Rectangle, 50, 50, 200, 50);
ashp.FillFormat.FillType = FillType.NoFill;
ITextFrame tf = ashp.TextFrame;
tf.Text = "Aspose TextBox";
```
**Εξήγηση**: Αυτός ο κώδικας προσθέτει ένα ορθογώνιο Αυτόματο Σχήμα στην πρώτη διαφάνεια και της αντιστοιχίζει κείμενο. Το γέμισμα του σχήματος ορίζεται σε `NoFill` να εστιάσει στο περιεχόμενο του κειμένου.

```csharp
IPortion port = tf.Paragraphs[0].Portions[0];
port.PortionFormat.LatinFont = new FontData("Times New Roman");

port.PortionFormat.FontItalic = NullableBool.True;

port.PortionFormat.FontHeight = 25;
port.PortionFormat.FillFormat.FillType = FillType.Solid;
port.PortionFormat.FillFormat.SolidFillColor.Color = Color.Blue;
```

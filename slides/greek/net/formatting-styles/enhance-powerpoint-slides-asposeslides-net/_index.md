## Οδηγός Εφαρμογής
Τώρα, ας εφαρμόσουμε τη λειτουργία για την προσθήκη και τη μορφοποίηση ενός πλαισίου εικόνας στο PowerPoint χρησιμοποιώντας C#.

### Προσθήκη εικόνας ως κορνίζας

**Επισκόπηση:**
Αυτή η ενότητα καλύπτει τον τρόπο με τον οποίο μπορείτε να εισαγάγετε μέσω προγραμματισμού μια εικόνα στη διαφάνεια της παρουσίασής σας ως πλαίσιο εικόνας, ορίζοντας με ακρίβεια τις διαστάσεις και τη θέση της.

#### Βήμα 1: Ρύθμιση του καταλόγου εγγράφων σας
Αρχικά, ορίστε τον κατάλογο στον οποίο βρίσκονται τα έγγραφά σας. Βεβαιωθείτε ότι αυτός ο κατάλογος υπάρχει ή δημιουργήστε τον, εάν είναι απαραίτητο:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
bool IsExists = Directory.Exists(dataDir);
if (!IsExists)
    Directory.CreateDirectory(dataDir);
```

#### Βήμα 2: Δημιουργήστε μια νέα παρουσίαση και αποκτήστε πρόσβαση στην πρώτη διαφάνεια
Στη συνέχεια, αρχικοποιήστε ένα νέο αντικείμενο παρουσίασης και αποκτήστε πρόσβαση στην πρώτη του διαφάνεια:

```csharp
using (Presentation pres = new Presentation())
{
    ISlide sld = pres.Slides[0];
```

#### Βήμα 3: Φόρτωση εικόνας στην παρουσίαση
Φορτώστε το αρχείο εικόνας που επιθυμείτε στην παρουσίαση. Αυτό το παράδειγμα χρησιμοποιεί μια εικόνα με το όνομα "aspose-logo.jpg":

```csharp
IImage img = Images.FromFile(dataDir + "aspose-logo.jpg");
IPPImage imgx = pres.Images.AddImage(img);
```

#### Βήμα 4: Προσθήκη πλαισίου εικόνας στη διαφάνεια
Προσθέστε την κορνίζα εικόνας με καθορισμένες διαστάσεις και θέση στη διαφάνεια:

```csharp
IPictureFrame pf = sld.Shapes.AddPictureFrame(
    ShapeType.Rectangle, 50, 150, imgx.Width, imgx.Height, imgx);
```

#### Βήμα 5: Μορφοποίηση του πλαισίου εικόνας
Προσαρμόστε την εμφάνιση της κορνίζας σας ορίζοντας το χρώμα, το πλάτος και την περιστροφή της γραμμής:

```csharp
pf.LineFormat.FillFormat.FillType = FillType.Solid;
pf.LineFormat.FillFormat.SolidFillColor.Color = Color.Blue;
pf.LineFormat.Width = 20;
pf.Rotation = 45;
```

#### Βήμα 6: Αποθήκευση της παρουσίασης
Τέλος, αποθηκεύστε την παρουσίασή σας με το νέο μορφοποιημένο πλαίσιο εικόνας:

```csharp
pres.Save(dataDir + "RectPicFrameFormat_out.pptx", SaveFormat.Pptx);
}
```

**Συμβουλή αντιμετώπισης προβλημάτων:** Εάν αντιμετωπίσετε σφάλματα στη διαδρομή αρχείου, ελέγξτε ξανά το `dataDir` και βεβαιωθείτε ότι όλα τα απαραίτητα αρχεία βρίσκονται σωστά.

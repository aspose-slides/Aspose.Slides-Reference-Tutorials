### Προσαρμογή της κεφαλίδας HTML
Η κεφαλίδα HTML είναι ζωτικής σημασίας για τον καθορισμό της εμφάνισης του εγγράφου σας κατά τη μετατροπή. Δείτε πώς μπορείτε να την προσαρμόσετε:

**1. Ορίστε το πρότυπο κεφαλίδας**
Δημιουργήστε μια σταθερή συμβολοσειρά που ορίζει τη δομή HTML σας, συμπεριλαμβανομένων των απαραίτητων μετα-ετικετών και συνδέσμων προς εξωτερικά φύλλα στυλ.
```csharp
const string Header = "<!DOCTYPE html>
" +
                      "<html>
" +
                      "<head>
" +
                      "<meta http-equiv="Content-Type" content="text/html; charset=UTF-8">
" +
                      "<meta http-equiv="X-UA-Compatible" content="IE=9">
" +
                      "<link rel="stylesheet" type="text/css" href="{0}">
"; // Δυναμικός σύνδεσμος CSS
```

**2. Καθορίστε τη διαδρομή προς το αρχείο CSS σας**
Βεβαιωθείτε ότι θα αντικαταστήσετε `"YOUR_DOCUMENT_DIRECTORY"` με την πραγματική σου πορεία.
```csharp
string cssFileName = @"YOUR_DOCUMENT_DIRECTORY/css/styles.css";
```

### Ενσωμάτωση γραμματοσειρών σε HTML
Για να ενσωματώσετε όλες τις γραμματοσειρές, επεκτείνετε το `EmbedAllFontsHtmlController` τάξη και προσαρμόστε την στις ανάγκες σας.

**1. Δημιουργήστε έναν προσαρμοσμένο ελεγκτή**
Ορίστε μια νέα κλάση που κληρονομεί από `EmbedAllFontsHtmlController`.
```csharp
public class CustomHeaderAndFontsController : EmbedAllFontsHtmlController
{
    private readonly string m_cssFileName;

    public CustomHeaderAndFontsController(string cssFileName)
    {
        // Αποθηκεύστε τη διαδρομή του αρχείου CSS.
        m_cssFileName = cssFileName;
    }

    protected override void WriteDocumentStart(IHtmlGenerator generator, IPresentation pptxPresentation)
    {
        // Εισαγωγή προσαρμοσμένης κεφαλίδας με ενσωματωμένες γραμματοσειρές
        generator.AddHtmlContent(Header.Replace("{0}", m_cssFileName));
    }
}
```

**2. Επεξήγηση Βασικών Στοιχείων**
- `m_cssFileName`: Αποθηκεύει τη διαδρομή προς το αρχείο CSS σας.
- `WriteDocumentStart`: Μέθοδος όπου εισάγετε το προσαρμοσμένο περιεχόμενο HTML.

## Οδηγός Εφαρμογής

### Χαρακτηριστικό: Ανάκτηση δεδομένων με συμπλήρωση κουκκίδων

Αυτή η λειτουργία ανακτά και εμφανίζει δεδομένα αποτελεσματικής συμπλήρωσης για κουκκίδες σε μια διαφάνεια παρουσίασης, επιτρέποντάς σας να προσαρμόσετε την εμφάνισή τους μέσω προγραμματισμού.

#### Βήμα 1: Ορισμός διαδρομών καταλόγου

Ξεκινήστε ορίζοντας διαδρομές προς τον κατάλογο εγγράφων σας και το αρχείο παρουσίασης:

```csharp
string dataDir = @"YOUR_DOCUMENT_DIRECTORY";
string pptxFile = Path.Combine(dataDir, "BulletData.pptx");
```

*Εξήγηση:* Ο `dataDir` η μεταβλητή αποθηκεύει τη διαδρομή προς τα έγγραφά σας, ενώ `pptxFile` συνδυάζει αυτό με το συγκεκριμένο όνομα αρχείου της παρουσίασής σας.

#### Βήμα 2: Φόρτωση του αρχείου παρουσίασης

Φορτώστε το αρχείο PowerPoint χρησιμοποιώντας το Aspose.Slides:

```csharp
using (Presentation pres = new Presentation(pptxFile))
{
    // Αποκτήστε πρόσβαση στο πρώτο σχήμα της πρώτης διαφάνειας, το οποίο αναμένεται να είναι ένα Αυτόματο Σχήμα
    AutoShape autoShape = (AutoShape)pres.Slides[0].Shapes[0];
}
```

*Εξήγηση:* Ο `Presentation` Το αντικείμενο αρχικοποιείται με το αρχείο σας και έχετε πρόσβαση στο σχήμα-στόχο χρησιμοποιώντας το ευρετήριό του.

#### Βήμα 3: Επανάληψη παραγράφων

Επαναλάβετε κάθε παράγραφο στο πλαίσιο κειμένου:

```csharp
foreach (Paragraph para in autoShape.TextFrame.Paragraphs)
{
    // Ανάκτηση δεδομένων αποτελεσματικής μορφής κουκκίδων για κάθε παράγραφο
    IBulletFormatEffectiveData bulletFormatEffective = para.ParagraphFormat.Bullet.GetEffective();
}
```

*Εξήγηση:* Αυτός ο βρόχος επεξεργάζεται κάθε παράγραφο, ανακτώντας την αποτελεσματική μορφή κουκκίδων.

#### Βήμα 4: Εμφάνιση τύπου γεμίσματος με κουκκίδες

Ελέγξτε αν υπάρχει κουκκίδα και εμφανίστε τον τύπο συμπλήρωσής της:

```csharp
if (bulletFormatEffective.Type != BulletType.None)
{
    switch (bulletFormatEffective.FillFormat.FillType)
    {
        case FillType.Solid:
            Console.WriteLine("Solid fill color: " + bulletFormatEffective.FillFormat.SolidFillColor);
            break;
        case FillType.Gradient:
            Console.WriteLine("Gradient stops count: " +
                              bulletFormatEffective.FillFormat.GradientFormat.GradientStops.Count);
            foreach (IGradientStopEffectiveData gradStop in bulletFormatEffective.FillFormat.GradientFormat.GradientStops)
                Console.WriteLine(gradStop.Position + ": " + gradStop.Color);
            break;
        case FillType.Pattern:
            Console.WriteLine("Pattern style: " +
                              bulletFormatEffective.FillFormat.PatternFormat.PatternStyle);
            Console.WriteLine("Fore color: " +
                              bulletFormatEffective.FillFormat.PatternFormat.ForeColor);
            Console.WriteLine("Back color: " +
                              bulletFormatEffective.FillFormat.PatternFormat.BackColor);
            break;
    }
}
```

*Εξήγηση:* Ανάλογα με τον τύπο γεμίσματος (Στερεό, Διαβάθμιση, Μοτίβο), εμφανίζονται διαφορετικές ιδιότητες.

### Συμβουλές αντιμετώπισης προβλημάτων

- **Συνηθισμένο πρόβλημα:** Βεβαιωθείτε ότι το αρχείο παρουσίασής σας έχει τουλάχιστον μία διαφάνεια με ένα πλαίσιο κειμένου που περιέχει κουκκίδες.
- **Αποσφαλμάτωση:** Χρησιμοποιήστε σημεία διακοπής για να διασχίσετε κάθε παράγραφο και να επαληθεύσετε το περιεχόμενό της πριν αποκτήσετε πρόσβαση σε δεδομένα κουκκίδων.

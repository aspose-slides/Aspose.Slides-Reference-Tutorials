## Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, θα αναλύσουμε τον τρόπο διαχείρισης του κειμένου του υποσέλιδου και ενημέρωσης του κειμένου της κεφαλίδας χρησιμοποιώντας το Aspose.Slides για .NET.

### Διαχείριση κειμένου υποσέλιδου σε παρουσιάσεις PowerPoint

#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να ορίσετε ομοιόμορφο κείμενο υποσέλιδου σε όλες τις διαφάνειες μιας παρουσίασης, διασφαλίζοντας συνέπεια και εξοικονομώντας χρόνο.

#### Βήμα προς βήμα εφαρμογή

**1. Φόρτωση της παρουσίασης**

Φορτώστε το υπάρχον αρχείο PowerPoint από τον καθορισμένο κατάλογο:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY/headerTest.pptx";
Presentation pres = new Presentation(dataDir);
```

**2. Ορισμός κειμένου υποσέλιδου σε όλες τις διαφάνειες**

Για να εφαρμόσετε ένα συγκεκριμένο κείμενο υποσέλιδου και να το κάνετε ορατό σε όλες τις διαφάνειες, χρησιμοποιήστε τις ακόλουθες μεθόδους:
```csharp
pres.HeaderFooterManager.SetAllFootersText("My Footer text");
pres.HeaderFooterManager.SetAllFootersVisibility(true);
```
- `SetAllFootersText(string footerText)`: Ορίζει το ίδιο κείμενο υποσέλιδου για κάθε διαφάνεια.
- `SetAllFootersVisibility(bool isVisible)`: Ελέγχει την ορατότητα των υποσέλιδων σε όλες τις διαφάνειες.

**3. Αποθήκευση αλλαγών**

Αποθηκεύστε την ενημερωμένη παρουσίασή σας σε μια νέα τοποθεσία:
```csharp
pres.Save("YOUR_OUTPUT_DIRECTORY/HeaderFooterJava.pptx", SaveFormat.Pptx);
```

### Ενημέρωση κειμένου κεφαλίδας σε κύριες διαφάνειες

#### Επισκόπηση
Αυτή η λειτουργία δείχνει πώς να αποκτήσετε πρόσβαση και να ενημερώσετε το κείμενο της κεφαλίδας μέσα σε κύριες διαφάνειες του PowerPoint, παρέχοντας έλεγχο στα πρότυπα διαφανειών.

#### Βήμα προς βήμα εφαρμογή

**1. Πρόσβαση στη διαφάνεια κύριων σημειώσεων**

Φορτώστε την παρουσίασή σας και ελέγξτε αν υπάρχει διαθέσιμη μια κύρια διαφάνεια σημειώσεων:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY/headerTest.pptx";
Presentation pres = new Presentation(dataDir);
IMasterNotesSlide masterNotesSlide = pres.MasterNotesSlideManager.MasterNotesSlide;
```

**2. Ενημέρωση κειμένου κεφαλίδας**

Εάν η κύρια διαφάνεια σημειώσεων υπάρχει, ενημερώστε το κείμενο της κεφαλίδας της χρησιμοποιώντας μια βοηθητική μέθοδο:
```csharp
if (masterNotesSlide != null) {
    UpdateHeaderFooterText(masterNotesSlide);
}
```

**3. Ορίστε τη Βοηθητική Μέθοδο**

Δημιουργήστε μια μέθοδο για την επανάληψη σχημάτων και την ενημέρωση κεφαλίδων όπου είναι εφικτό:
```csharp
public static void UpdateHeaderFooterText(IBaseSlide master) {
    foreach (IShape shape in master.Shapes) {
        if (shape.Placeholder != null && 
            shape.Placeholder.Type == PlaceholderType.Header) {
            ((IAutoShape)shape).TextFrame.Text = "HI there new header";
        }
    }
}
```
- Επαναλαμβάνει κάθε σχήμα μέσα στην κύρια διαφάνεια.
- Έλεγχοι για placeholders τύπου `Header` και ενημερώνει το κείμενο αναλόγως.

## Οδηγός Εφαρμογής
Τώρα που είστε έτοιμοι, ας δούμε πώς να αποδίδετε μικρογραφίες διαφανειών με προσαρμοσμένες γραμματοσειρές.

### Επισκόπηση λειτουργιών: Απόδοση μικρογραφιών με προσαρμοσμένες γραμματοσειρές
Αυτή η λειτουργία σάς επιτρέπει να αποδώσετε την πρώτη διαφάνεια μιας παρουσίασης ως εικόνα χρησιμοποιώντας συγκεκριμένες ρυθμίσεις γραμματοσειράς. Είναι ιδιαίτερα χρήσιμη για σκοπούς εμπορικής προβολής και για τη διασφάλιση της συνέπειας μεταξύ των παρουσιάσεων.

#### Βήμα 1: Φόρτωση της παρουσίασής σας
Ξεκινήστε φορτώνοντας το αρχείο PowerPoint στο `Presentation` αντικείμενο:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string presPath = Path.Combine(dataDir, "RenderingOptions.pptx");
using (Presentation pres = new Presentation(presPath))
{
    // Συνέχεια με τις ρυθμίσεις απόδοσης
}
```

#### Βήμα 2: Ρύθμιση παραμέτρων επιλογών απόδοσης
Ορίστε την επιθυμητή γραμματοσειρά ως προεπιλογή για την απόδοση:
```csharp
IRenderingOptions renderingOpts = new RenderingOptions();
renderingOpts.DefaultRegularFont = "Arial Black";
```
Αυτό το βήμα διασφαλίζει ότι το κείμενο στην εικόνα που αποδίδεται ταιριάζει με την επωνυμία ή τον οδηγό στυλ σας.

#### Βήμα 3: Απόδοση και αποθήκευση της διαφάνειας
Χρησιμοποιήστε το `GetImage` μέθοδος για την απόδοση της διαφάνειας και την αποθήκευσή της ως εικόνα:
```csharp
double aspectRatio = 4 / 3.0;
pres.Slides[0].GetImage(renderingOpts, aspectRatio, aspectRatio)
    .Save(Path.Combine("YOUR_OUTPUT_DIRECTORY", "output.png"), ImageFormat.Png);
```
Εδώ, `aspectRatio` αντιπροσωπεύει τις διαστάσεις της εικόνας. Προσαρμόστε όπως απαιτείται για να ταιριάζει στις απαιτήσεις σας.

### Συμβουλές αντιμετώπισης προβλημάτων
- **Λείπουν γραμματοσειρές:** Βεβαιωθείτε ότι η καθορισμένη γραμματοσειρά είναι εγκατεστημένη στο σύστημά σας.
- **Προβλήματα διαδρομής αρχείου:** Ελέγξτε ξανά τις διαδρομές καταλόγων για τυπογραφικά λάθη ή δικαιώματα πρόσβασης.
- **Σφάλματα μορφής εικόνας:** Βεβαιωθείτε ότι χρησιμοποιείτε μια υποστηριζόμενη μορφή εικόνας στο `Save()`.

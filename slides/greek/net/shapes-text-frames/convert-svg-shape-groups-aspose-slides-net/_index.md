## Οδηγός Εφαρμογής

### Μετατροπή SVG σε ομάδα σχημάτων (H2)
Σε αυτήν την ενότητα, θα δούμε τα βήματα που απαιτούνται για να μετατρέψουμε μια εικόνα SVG σε μια ομάδα σχημάτων.

#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε ενσωματωμένες εικόνες SVG μέσα σε μια διαφάνεια του PowerPoint σε διαχειρίσιμα στοιχεία σχήματος. Αυτή η μετατροπή διευκολύνει την τροποποίηση και προσαρμογή γραφικών στην παρουσίασή σας.

#### Βήμα προς βήμα εφαρμογή (H3)
1. **Φόρτωση της παρουσίασής σας**
   Ξεκινήστε φορτώνοντας την παρουσίαση που περιέχει την εικόνα SVG:
   ```csharp
   string dataDir = @"YOUR_DOCUMENT_DIRECTORY";
   using (Presentation pres = new Presentation(dataDir + "image.pptx")) {
       // Ο κώδικας συνεχίζεται...
   }
   ```
2. **Πρόσβαση στην εικόνα SVG**
   Προσδιορίστε και αποκτήστε πρόσβαση στο PictureFrame που περιέχει την εικόνα SVG σας:
   ```csharp
   PictureFrame pFrame = pres.Slides[0].Shapes[0] as PictureFrame;
   ISvgImage svgImage = pFrame.PictureFormat.Picture.Image.SvgImage;

   if (svgImage != null) {
       // Συνέχεια με τη μετατροπή...
   }
   ```
3. **Μετατροπή και τοποθέτηση του SVG**
   Μετατρέψτε το SVG σε μια ομάδα σχημάτων, τοποθετώντας το στην αρχική θέση του πλαισίου:
   ```csharp
   IGroupShape groupShape = pres.Slides[0].Shapes.AddGroupShape(
       svgImage,
       pFrame.Frame.X,
       pFrame.Frame.Y,
       pFrame.Frame.Width,
       pFrame.Frame.Height);
   ```
4. **Αφαίρεση αρχικής εικόνας SVG**
   Αφαιρέστε το αρχικό PictureFrame για να καθαρίσετε τη διαφάνειά σας:
   ```csharp
   pres.Slides[0].Shapes.Remove(pFrame);
   ```
5. **Αποθήκευση της παρουσίασής σας**
   Τέλος, αποθηκεύστε την τροποποιημένη παρουσίαση με την ομάδα σχημάτων που μόλις δημιουργήσατε:
   ```csharp
   pres.Save(dataDir + "image_group.pptx");
   ```

#### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι η εικόνα SVG σας είναι σωστά ενσωματωμένη σε ένα PictureFrame.
- Επαληθεύστε τις διαδρομές αρχείων και βεβαιωθείτε ότι οδηγούν στους σωστούς καταλόγους.

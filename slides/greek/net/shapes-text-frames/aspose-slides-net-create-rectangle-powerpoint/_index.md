## Οδηγός Εφαρμογής

### Χαρακτηριστικό 1: Απλή δημιουργία ορθογωνίου στο PowerPoint (H2)

Αυτοματοποιήστε την προσθήκη ορθογωνίων σχημάτων για να εξοικονομήσετε χρόνο και να διασφαλίσετε τη συνέπεια σε όλες τις παρουσιάσεις. Δείτε πώς μπορείτε να προσθέσετε ένα ορθογώνιο χρησιμοποιώντας το Aspose.Slides για .NET.

#### Βήμα προς βήμα εφαρμογή (H3)

1. **Αρχικοποίηση Κλάσης Παρουσίασης**
   
   Δημιουργήστε μια παρουσία του `Presentation` κλάση για την αναπαράσταση του αρχείου PowerPoint σας:

   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;

   string YOUR_DOCUMENT_DIRECTORY = "YOUR_DOCUMENT_DIRECTORY";

   using (Presentation pres = new Presentation())
   {
       // Ο κώδικας συνεχίζεται εδώ...
   }
   ```

2. **Πρόσβαση στην πρώτη διαφάνεια**

   Ανακτήστε την πρώτη διαφάνεια από την παρουσίασή σας:

   ```csharp
   ISlide sld = pres.Slides[0];
   ```

3. **Προσθήκη ορθογωνίου σχήματος**

   Χρήση `AddAutoShape` για να προσθέσετε ένα ορθογώνιο σε καθορισμένες θέσεις και μεγέθη:

   ```csharp
   sld.Shapes.AddAutoShape(ShapeType.Rectangle, 50, 150, 150, 50);
   ```
   
   - **Παράμετροι**: Η μέθοδος δέχεται `ShapeType`, θέση x, θέση y, πλάτος και ύψος για να ορίσετε την τοποθέτηση και το μέγεθος του σχήματος.

4. **Αποθήκευση παρουσίασης**

   Αποθηκεύστε την παρουσίασή σας για να αποθηκεύσετε όλες τις αλλαγές:

   ```csharp
   pres.Save(YOUR_DOCUMENT_DIRECTORY + "/RectShp1_out.pptx", SaveFormat.Pptx);
   ```

#### Συμβουλές αντιμετώπισης προβλημάτων

- Εξασφαλίζω `YOUR_DOCUMENT_DIRECTORY` οι διαδρομές έχουν οριστεί σωστά.
- Επαληθεύστε ότι το Aspose.Slides αναφέρεται σωστά στο έργο σας.

### Χαρακτηριστικό 2: Δημιουργία και Επαλήθευση Καταλόγου (H2)

Η αποτελεσματική διαχείριση καταλόγων αποτρέπει σφάλματα κατά την αποθήκευση αρχείων. Εφαρμόστε αυτόν τον έλεγχο για να βεβαιωθείτε ότι υπάρχουν κατάλογοι πριν επιχειρήσετε να αποθηκεύσετε ένα αρχείο.

#### Βήμα προς βήμα εφαρμογή (H3)

1. **Ορισμός διαδρομής καταλόγου**

   Καθορίστε πού θα αποθηκευτούν τα έγγραφά σας:

   ```csharp
   string dataDir = YOUR_DOCUMENT_DIRECTORY;
   ```

2. **Έλεγχος και δημιουργία καταλόγου εάν είναι απαραίτητο**

   Χρήση `Directory.Exists` για να επαληθεύσετε την ύπαρξη του καταλόγου, δημιουργώντας τον εάν χρειάζεται:

   ```csharp
   bool isExists = Directory.Exists(dataDir);
   if (!isExists)
   {
       Directory.CreateDirectory(dataDir);
   }
   ```

#### Συμβουλές αντιμετώπισης προβλημάτων

- Επιβεβαιώστε ότι η εφαρμογή σας έχει άδεια δημιουργίας καταλόγων στην καθορισμένη διαδρομή.
- Χειρισμός εξαιρέσεων από μη έγκυρες διαδρομές ή ανεπαρκή δικαιώματα.

# Εξοικείωση με την αντικατάσταση κειμένου με επανάκληση χρησιμοποιώντας το Aspose.Slides .NET

Αυτό το σεμινάριο δείχνει πώς να αντικαθιστάτε αποτελεσματικά συγκεκριμένο κείμενο και να παρακολουθείτε τις λεπτομέρειες κάθε αντικατάστασης χρησιμοποιώντας το Aspose.Slides για .NET, εστιάζοντας στη λειτουργικότητα επανάκλησης.

Σε αυτόν τον οδηγό, θα ανακαλύψετε:
- Πώς να εκτελέσετε αντικατάσταση κειμένου στο PowerPoint με το Aspose.Slides για .NET
- Εφαρμογή επανακλήσεων για την παρακολούθηση αντικαταστάσεων
- Εφαρμογές αυτών των χαρακτηριστικών στον πραγματικό κόσμο

Πριν προχωρήσουμε στην υλοποίηση, ας εξετάσουμε τις προϋποθέσεις.

Βεβαιωθείτε ότι έχετε τα ακόλουθα πριν ξεκινήσετε:

## Οδηγός Εφαρμογής

### Χαρακτηριστικό 1: Αντικατάσταση κειμένου με επανάκληση

Αυτή η λειτουργία επιτρέπει την αντικατάσταση κειμένου μέσα σε μια παρουσίαση, ενώ παράλληλα χρησιμοποιείται ένας μηχανισμός επανάκλησης για τη συλλογή λεπτομερειών σχετικά με κάθε αντικατάσταση.

#### Βήμα προς βήμα εφαρμογή

**1. Ορισμός διαδρομών και αρχικοποίηση παρουσίασης**
Ορίστε τις διαδρομές των αρχείων εισόδου και εξόδου και, στη συνέχεια, φορτώστε την παρουσίαση:
```csharp
string presentationName = "YOUR_DOCUMENT_DIRECTORY/TextReplaceExample.pptx";
string outPath = "YOUR_OUTPUT_DIRECTORY/TextReplaceExampleReplace-out.pptx";

using (Presentation pres = new Presentation(presentationName))
{
    // Συνεχίστε με τις εργασίες αντικατάστασης εδώ
}
```

**2. Υλοποιήστε την Επανάκληση**
Δημιουργήστε μια κλάση επανάκλησης για να καταγράψετε πληροφορίες σχετικά με κάθε αντικατάσταση:
```csharp
class FindResultCallback : IFindResultCallback
{
    public readonly List<WordInfo> Words = new List<WordInfo>();

    public int Count => Words.Count;

    public void FoundResult(ITextFrame textFrame, string oldText, string foundText, int textPosition)
    {
        Words.Add(new WordInfo(textFrame, oldText, foundText, textPosition));
    }
}
```

**3. Εκτέλεση αντικατάστασης κειμένου**
Αντικαταστήστε το καθορισμένο κείμενο και καλέστε την επανάκληση:
```csharp
FindResultCallback callback = new FindResultCallback();
pres.ReplaceText("[this block] ", "my text", new TextSearchOptions(), callback);
```

### Χαρακτηριστικό 2: Υλοποίηση επανάκλησης για αντικατάσταση κειμένου
Ο μηχανισμός επανάκλησης είναι κρίσιμος για την παρακολούθηση κάθε αντικατάστασης, παρέχοντας πληροφορίες σχετικά με τις αλλαγές που έγιναν.

**4. Ορισμός Κλάσης Πληροφοριών**
Δημιουργήστε μια κλάση για να αποθηκεύσετε λεπτομερείς πληροφορίες σχετικά με το κείμενο που βρέθηκε:
```csharp
class WordInfo
{
    internal WordInfo(ITextFrame textFrame, string sourceText, string foundText, int textPosition)
    {
        TextFrame = textFrame;
        SourceText = sourceText;
        FoundText = foundText;
        TextPosition = textPosition;
    }

    public string FoundText { get; }
    public string SourceText { get; }
    public int TextPosition { get; }
    public ITextFrame TextFrame { get; }
}
```

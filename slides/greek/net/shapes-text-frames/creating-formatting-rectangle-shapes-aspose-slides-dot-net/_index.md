## Οδηγός Εφαρμογής
### Δημιουργία ορθογωνίου σχήματος
Στόχος μας είναι να δημιουργήσουμε ένα ορθογώνιο σχήμα στην πρώτη διαφάνεια. Ας αναλύσουμε τα βήματα:
#### Βήμα 1: Αρχικοποίηση παρουσίασης
Ξεκινήστε ρυθμίζοντας το περιβάλλον σας με το Aspose.Slides και δημιουργώντας ένα νέο αντικείμενο παρουσίασης.
```csharp
using System;
using Aspose.Slides;
using System.Drawing;

string dataDir = "YOUR_DOCUMENT_DIRECTORY";
bool isExists = System.IO.Directory.Exists(dataDir);
if (!isExists)
    System.IO.Directory.CreateDirectory(dataDir);

using (Presentation pres = new Presentation())
{
    // Ο κώδικας συνεχίζεται...
}
```
*Εξήγηση:* Αυτός ο κώδικας αρχικοποιεί μια νέα παρουσίαση PowerPoint και διασφαλίζει ότι υπάρχει ο κατάλογος για την αποθήκευση αρχείων.
#### Βήμα 2: Πρόσβαση στην πρώτη διαφάνεια
Αποκτήστε πρόσβαση στην πρώτη διαφάνεια όπου θα προσθέσουμε το ορθογώνιό μας.
```csharp
ISlide sld = pres.Slides[0];
```
*Εξήγηση:* Ανακτούμε την πρώτη διαφάνεια από την παρουσίαση για να εργαστούμε.
#### Βήμα 3: Προσθήκη ορθογωνίου σχήματος
Προσθέστε ένα ορθογώνιο αυτόματου σχήματος τύπου στη διαφάνεια.
```csharp
IShape shp = sld.Shapes.AddAutoShape(ShapeType.Rectangle, 50, 150, 150, 50);
```
*Εξήγηση:* Αυτό δημιουργεί ένα ορθογώνιο στη θέση (50, 150) με διαστάσεις 150x50. Οι παράμετροι καθορίζουν τον τύπο σχήματος και τη θέση/μέγεθος του.
### Μορφοποίηση του ορθογωνίου
Τώρα που έχουμε το ορθογώνιο μας, ας εφαρμόσουμε κάποια στυλ σε αυτό.
#### Βήμα 4: Εφαρμογή συμπαγούς χρώματος γεμίσματος
Ορίστε ένα συμπαγές χρώμα γεμίσματος για το σώμα του ορθογωνίου.
```csharp
shp.FillFormat.FillType = FillType.Solid;
shp.FillFormat.SolidFillColor.Color = Color.Chocolate;
```
*Εξήγηση:* Εδώ, αλλάζουμε το εσωτερικό του ορθογωνίου σε χρώμα καφέ σοκολάτας.
#### Βήμα 5: Εφαρμογή μορφοποίησης γραμμής περιγράμματος
Προσαρμόστε το περίγραμμα με συμπαγές γέμισμα και προσαρμόστε το πλάτος του.
```csharp
shp.LineFormat.FillFormat.FillType = FillType.Solid;
shp.LineFormat.FillFormat.SolidFillColor.Color = Color.Black;
shp.LineFormat.Width = 5;
```
*Εξήγηση:* Το περίγραμμα του ορθογωνίου έχει οριστεί σε μαύρο, με πλάτος γραμμής 5 pixel.
### Αποθήκευση της παρουσίασης
Τέλος, αποθηκεύστε τις αλλαγές σας σε ένα αρχείο.
```csharp
pres.Save(dataDir + "/RectShp2_out.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
```
*Εξήγηση:* Αυτό αποθηκεύει την παρουσίαση με το νέο μορφοποιημένο ορθογώνιο σχήμα στον καθορισμένο κατάλογο.

#### Βήμα 2: Δημιουργία αντικειμένου παρουσίασης

    static void Main()

#### Βήμα 3: Προσθήκη Αυτόματου Σχήματος με Εσωτερική Σκιά
```csharp
// Προσθήκη ορθογωνίου σχήματος με θέση (150, 75) και μέγεθος (150x50)
IAutoShape ashp = sld.Shapes.AddAutoShape(ShapeType.Rectangle, 150, 75, 150, 50);

// Προσθήκη κειμένου στο σχήμα
txtFrame = ashp.TextFrame;
para = txtFrame.Paragraphs[0];
portion = para.Portions[0];

// Ορισμός του κειμένου του τμήματος
portion.Text = "Aspose TextBox";
```
Αυτή η ενότητα προσθέτει ένα ορθογώνιο σχήμα στη διαφάνειά σας και την ορίζει με ένα κενό πλαίσιο κειμένου. Μπορείτε αργότερα να εφαρμόσετε εφέ όπως εσωτερική σκιά σε αυτό το σχήμα.

Για να προσθέσετε μια εσωτερική σκιά, συνήθως θα τροποποιήσετε το `ashp` ιδιότητες στυλ αντικειμένου. Ωστόσο, το Aspose.Slides για .NET δεν υποστηρίζει άμεσα την εσωτερική σκιά μέσω ενσωματωμένων μεθόδων κατά τη στιγμή της σύνταξης, επομένως ίσως χρειαστεί να χρησιμοποιήσετε τεχνικές παράκαμψης ή πρόσθετες βιβλιοθήκες που προσφέρουν πιο προηγμένους χειρισμούς γραφικών.

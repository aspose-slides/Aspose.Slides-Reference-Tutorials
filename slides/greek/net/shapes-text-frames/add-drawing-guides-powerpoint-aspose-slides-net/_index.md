## Οδηγός Εφαρμογής
Τώρα που έχουμε ρυθμίσει το περιβάλλον μας, ας προσθέσουμε αυτούς τους οδηγούς σχεδίασης.

### Προσθήκη οδηγών σχεδίασης σε μια διαφάνεια του PowerPoint
#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να βελτιώσετε την ακρίβεια των ολισθήσεων προσθέτοντας κάθετους και οριζόντιους οδηγούς ανάλογα με τις απαιτήσεις σας.

##### Βήμα 1: Δημιουργία νέας παρουσίασης
Δημιουργήστε μια παρουσία του `Presentation` τάξη. Αυτός θα είναι ο καμβάς μας όπου θα προσθέσουμε οδηγούς σχεδίασης.

```csharp
using Aspose.Slides;
using System.IO;

string outFilePath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "GuidesProperties-out.pptx");

using (Presentation pres = new Presentation())
{
    // Ο κώδικας για την προσθήκη οδηγών θα τοποθετηθεί εδώ
}
```

##### Βήμα 2: Πρόσβαση στο μέγεθος διαφάνειας
Ανακτήστε τις διαστάσεις της διαφάνειας για να τοποθετήσετε τους οδηγούς με ακρίβεια.

```csharp
var slideSize = pres.SlideSize.Size;
```

##### Βήμα 3: Προσθήκη κάθετων και οριζόντιων οδηγών
Πρόσβαση στο `DrawingGuidesCollection` από `SlideViewProperties` για να προσθέσετε νέους οδηγούς. Εδώ, προσθέτουμε έναν κατακόρυφο οδηγό στα δεξιά του κέντρου και έναν οριζόντιο οδηγό από κάτω του.

```csharp
IDrawingGuidesCollection guides = pres.ViewProperties.SlideViewProperties.DrawingGuides;

// Προσθήκη ενός κατακόρυφου οδηγού σε μια θέση μετατόπισης
guides.Add(Orientation.Vertical, slideSize.Width / 2 + 12.5f);

// Προσθήκη οριζόντιου οδηγού σε θέση μετατόπισης
guides.Add(Orientation.Horizontal, slideSize.Height / 2 + 12.5f);
```

##### Βήμα 4: Αποθήκευση της παρουσίασης
Τέλος, αποθηκεύστε την παρουσίασή σας με τους πρόσθετους οδηγούς.

```csharp
pres.Save(outFilePath, SaveFormat.Pptx);
```

#### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι η διαδρομή του καταλόγου εξόδου σας είναι σωστή για να αποφύγετε `DirectoryNotFoundException`.
- Εάν οι οδηγοί δεν εμφανίζονται όπως αναμένεται, επαληθεύστε τους υπολογισμούς για τις θέσεις των οδηγών σε σχέση με το μέγεθος της διαφάνειας.

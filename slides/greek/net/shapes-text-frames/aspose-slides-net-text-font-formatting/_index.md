## Οδηγός Εφαρμογής

Θα χωρίσουμε την υλοποίηση σε ξεχωριστές ενότητες με βάση τη λειτουργικότητά της.

### Προσθήκη κειμένου σε σχήμα

#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να προσθέτετε προσαρμοσμένο κείμενο μέσα στα Αυτόματα Σχήματα, όπως ορθογώνια στις διαφάνειές σας. Είναι ζωτικής σημασίας για την παροχή προσαρμοσμένου περιεχομένου απευθείας σε σχήματα διαφανειών.

#### Βήματα για την εφαρμογή

**1. Δημιουργία και προσθήκη ενός αυτόματου σχήματος**

```csharp
using (Presentation pres = new Presentation())
{
    IAutoShape newShape = pres.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 100, 100, 400, 75, false);
```
- **Παράμετροι**: 
  - `ShapeType.Rectangle`: Ορίζει τον τύπο σχήματος.
  - Συντεταγμένες (x=100, y=100) και διαστάσεις (πλάτος=400, ύψος=75): Θέση και μέγεθος του σχήματος.

**2. Προσθήκη πλαισίου κειμένου**

```csharp
    newShape.AddTextFrame("");
```
- **Σκοπός**: Αρχικοποιεί ένα κενό πλαίσιο κειμένου για να περιέχει το προσαρμοσμένο κείμενο.

**3. Εισαγωγή τμημάτων κειμένου**

```csharp
    newShape.TextFrame.Paragraphs[0].Portions.Clear();
    
    IPortion portion0 = new Portion("Sample text with first portion");
    IPortion portion1 = new Portion(" and second portion.");
    
    newShape.TextFrame.Paragraphs[0].Portions.Add(portion0);
    newShape.TextFrame.Paragraphs[0].Portions.Add(portion1);
}
```
- **Εξήγηση**: Διαγράψτε τα υπάρχοντα τμήματα και, στη συνέχεια, δημιουργήστε και προσθέστε νέα τμήματα κειμένου. Αυτό επιτρέπει την τμηματοποίηση περιεχομένου μέσα σε μία μόνο παράγραφο.

### Ορισμός προεπιλεγμένου ύψους γραμματοσειράς για παρουσίαση

#### Επισκόπηση
Ο ορισμός ενός ομοιόμορφου ύψους γραμματοσειράς σε ολόκληρη την παρουσίασή σας διασφαλίζει τη συνέπεια στο σχεδιασμό και την αναγνωσιμότητα.

#### Βήματα για την εφαρμογή

**1. Προσθήκη τμημάτων κειμένου**
Χρησιμοποιήστε ξανά τον κώδικα για την προσθήκη τμημάτων κειμένου όπως φαίνεται παραπάνω.

**2. Ορισμός προεπιλεγμένου ύψους γραμματοσειράς**

```csharp
    pres.DefaultTextStyle.GetLevel(0).DefaultPortionFormat.FontHeight = 24;
```
- **Σκοπός**Εφαρμόζει ένα σταθερό ύψος γραμματοσειράς 24 στιγμών σε όλα τα τμήματα κειμένου της παρουσίασης.

### Ορισμός προεπιλεγμένου ύψους γραμματοσειράς για μια παράγραφο

#### Επισκόπηση
Μπορείτε να προσαρμόσετε μεμονωμένες παραγράφους μέσα στις διαφάνειές σας, κάνοντας συγκεκριμένο περιεχόμενο να ξεχωρίζει.

#### Βήματα για την εφαρμογή

**1. Προσθήκη τμημάτων κειμένου**
Όπως περιγράφηκε προηγουμένως.

**2. Προσαρμόστε το ύψος γραμματοσειράς για μια συγκεκριμένη παράγραφο**

```csharp
    newShape.TextFrame.Paragraphs[0].ParagraphFormat.DefaultPortionFormat.FontHeight = 40;
```
- **Εξήγηση**Ορίζει το ύψος της γραμματοσειράς όλων των τμημάτων αυτής της παραγράφου σε 40 σημεία, ενισχύοντας την οπτική της επίδραση.

### Ορισμός ύψους γραμματοσειράς για ένα μεμονωμένο τμήμα

#### Επισκόπηση
Για ακριβή έλεγχο της τυπογραφίας της παρουσίασής σας, προσαρμόστε το μέγεθος γραμματοσειράς συγκεκριμένων τμημάτων κειμένου ξεχωριστά.

#### Βήματα για την εφαρμογή

**1. Προσθήκη τμημάτων κειμένου**
Ανατρέξτε στα αρχικά βήματα για την προσθήκη τμημάτων κειμένου.

**2. Ορίστε συγκεκριμένα ύψη γραμματοσειράς**

```csharp
    newShape.TextFrame.Paragraphs[0].Portions[0].PortionFormat.FontHeight = 55;
    newShape.TextFrame.Paragraphs[0].Portions[1].PortionFormat.FontHeight = 18;
```
- **Εξήγηση**Αυτή η προσαρμογή δίνει σε κάθε τμήμα μοναδικά ύψη γραμματοσειράς, επιτρέποντας λεπτομερή έμφαση όπου χρειάζεται.

### Αποθήκευση της παρουσίασης

#### Επισκόπηση
Μόλις η παρουσίασή σας βελτιστοποιηθεί στο έπακρο, αποθηκεύστε την σε μορφή αρχείου της επιλογής σας.

```csharp
using (Presentation pres = new Presentation())
{
    // Προσθέστε σχήματα και κείμενο όπως περιγράφεται παραπάνω...

    // Αποθήκευση της παρουσίασης
    pres.Save("YOUR_OUTPUT_DIRECTORY\SetLocalFontHeightValues.pptx", SaveFormat.Pptx);
}
```
- **Καθέκαστα**Αυτό αποθηκεύει τις μορφοποιημένες διαφάνειές σας σε ένα αρχείο PPTX, έτοιμο για διανομή ή περαιτέρω επεξεργασία.

### Εύρεση σχημάτων σε διαφάνειες με εναλλακτικό κείμενο

#### Επισκόπηση:
Η αυτοματοποίηση της αναζήτησης για συγκεκριμένα σχήματα χρησιμοποιώντας το εναλλακτικό τους κείμενο μπορεί να βελτιώσει σημαντικά την παραγωγικότητά σας κατά την επεξεργασία αρχείων PowerPoint. Ας εξερευνήσουμε πώς λειτουργεί αυτή η λειτουργία.

##### Βήμα 3: Αναζήτηση για σχήμα με εναλλακτικό κείμενο
Υλοποιήστε μια μέθοδο για να βρείτε και να επιστρέψετε το σχήμα με βάση το εναλλακτικό του κείμενο.

```csharp
public static IShape FindShape(ISlide slide, string altText) {
    foreach (var shape in slide.Shapes) {
        if (shape.AlternativeText == altText) {
            return shape;
        }
    }
    return null; // Επιστρέφει null εάν το σχήμα δεν βρεθεί
}
```

**Εξήγηση:** Αυτή η συνάρτηση επαναλαμβάνει όλα τα σχήματα σε μια διαφάνεια, ελέγχοντας το εναλλακτικό κείμενο κάθε σχήματος σε σχέση με την παρεχόμενη είσοδο. Επιστρέφει το αντίστοιχο σχήμα ή `null` εάν δεν βρεθεί αντιστοιχία.

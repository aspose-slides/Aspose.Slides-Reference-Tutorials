## Οδηγός Εφαρμογής

Ας αναλύσουμε τη διαδικασία ρύθμισης και χρήσης κανόνων εφεδρικής γραμματοσειράς στο Aspose.Slides για .NET.

### Δημιουργία γραμματοσειράς FallBackRulesCollection

Το βασικό χαρακτηριστικό είναι η δημιουργία μιας συλλογής που καθορίζει τον τρόπο με τον οποίο η εφαρμογή σας θα πρέπει να χειρίζεται γραμματοσειρές που δεν είναι διαθέσιμες στο σύστημα. 

#### Επισκόπηση

Οι εναλλακτικοί κανόνες γραμματοσειράς είναι απαραίτητοι όταν θέλετε να διασφαλίσετε ότι συγκεκριμένες γραμματοσειρές αποδίδονται σωστά, ειδικά για μη τυπικούς χαρακτήρες ή σενάρια.

##### Βήμα 1: Αρχικοποίηση FontFallBackRulesCollection

Ξεκινήστε αρχικοποιώντας ένα νέο `IFontFallBackRulesCollection` αντικείμενο:

```csharp
using (Presentation presentation = new Presentation())
{
    IFontFallBackRulesCollection userRulesList = new FontFallBackRulesCollection();
}
```

#### Προσθήκη κανόνων εφεδρικής αναπαραγωγής

Για να προσθέσετε εφεδρικούς κανόνες γραμματοσειράς, χρησιμοποιήστε το `Add()` μέθοδος. Αυτό σας επιτρέπει να καθορίσετε εύρη Unicode και τις αντίστοιχες γραμματοσειρές.

##### Βήμα 2: Ορισμός προσαρμοσμένων κανόνων εφεδρικής λειτουργίας

1. **Αντιστοίχιση εύρους Unicode U+0B80-U+0BFF με τη γραμματοσειρά "Vijaya"**
   
   Αυτός ο κανόνας διασφαλίζει ότι οι χαρακτήρες σε αυτό το εύρος Unicode έχουν προεπιλεγμένη γραμματοσειρά "Vijaya", εάν είναι διαθέσιμη:
   
   ```csharp
   userRulesList.Add(new FontFallBackRule(0x0B80, 0x0BFF, "Vijaya"));
   ```

2. **Αντιστοίχιση εύρους Unicode U+3040-U+309F σε "MS Mincho, MS Gothic"**
   
   Αυτός ο κανόνας καλύπτει χαρακτήρες στο καθορισμένο εύρος και τους αντιστοιχίζει είτε σε "MS Mincho" είτε σε "MS Gothic":
   
   ```csharp
   userRulesList.Add(new FontFallBackRule(0x3040, 0x309F, "MS Mincho, MS Gothic"));
   ```

#### Αντιστοίχιση κανόνων εφεδρικής λειτουργίας σε παρουσίαση

Μόλις ορίσετε τους κανόνες σας, αντιστοιχίστε τους στον διαχειριστή γραμματοσειρών της παρουσίασης:

```csharp
presentation.FontsManager.FontFallBackRulesCollection = userRulesList;
```

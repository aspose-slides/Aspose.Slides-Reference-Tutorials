## Οδηγός Εφαρμογής

### Λειτουργία 1: Δημιουργία και προσθήκη αυτόματου σχήματος σε διαφάνεια

#### Επισκόπηση

Αυτή η ενότητα δείχνει πώς να δημιουργήσετε μια παρουσίαση, να αποκτήσετε πρόσβαση σε μια διαφάνεια και να προσθέσετε ένα Αυτόματο Σχήμα τύπου Ορθογώνιο.

#### Βήματα:

**Βήμα 1**Αρχικοποίηση της παρουσίασης
```csharp
// Δημιουργήστε μια παρουσία της κλάσης Presentation
tPresentation presentation = new tPresentation();
```

**Βήμα 2**: Πρόσβαση στην πρώτη διαφάνεια
```csharp
// Πρόσβαση στην πρώτη διαφάνεια
tISlide slide = presentation.Slides[0];
```

**Βήμα 3**: Προσθήκη αυτόματου σχήματος ορθογωνίου
```csharp
// Προσθήκη ενός AutoShape τύπου Rectangle στη θέση (150, 75) με μέγεθος (350, 350)
tIAutoShape ashp = slide.Shapes.AddAutoShape(tShapeType.Rectangle, 150, 75, 350, 350);
```

**Βήμα 4**: Αποθήκευση της παρουσίασης
```csharp
// Αποθηκεύστε την παρουσίαση σε έναν καθορισμένο κατάλογο presentation.Save("YOUR_OUTPUT_DIRECTORY/formatText_out.pptx", tSaveFormat.Pptx);
```

### Χαρακτηριστικό 2: Προσθήκη και μορφοποίηση TextFrame στο AutoShape

#### Επισκόπηση

Αυτή η λειτουργία εξηγεί πώς να προσθέσετε ένα TextFrame σε ένα υπάρχον AutoShape, να ρυθμίσετε τις επιλογές αυτόματης προσαρμογής και να ορίσετε ιδιότητες κειμένου.

#### Βήματα:

**Βήμα 1**: Προσθήκη TextFrame
```csharp
// Υποθέτοντας ότι το 'ashp' είναι μια παρουσία IAutoShape από την προηγούμενη λειτουργία
// Προσθήκη TextFrame στο ορθογώνιο
tashp.AddTextFrame(" ");
```

**Βήμα 2**: Ρύθμιση παραμέτρων τύπου αυτόματης προσαρμογής
```csharp
// Ορισμός τύπου αυτόματης προσαρμογής για καλύτερη ευθυγράμμιση κειμένου μέσα στο σχήμα
tITextFrame txtFrame = ashp.TextFrame;
txtFrame.TextFrameFormat.AutofitType = tTextAutofitType.Shape;
```

**Βήμα 3**: Μορφοποίηση και Εισαγωγή Κειμένου
```csharp
// Δημιουργήστε ένα αντικείμενο παραγράφου και ορίστε το περιεχόμενο
tIParagraph para = txtFrame.Paragraphs[0];
tIPortion portion = para.Portions[0];

portion.Text = "A quick brown fox jumps over the lazy dog. A quick brown fox jumps over the lazy dog.";
portion.PortionFormat.FillFormat.FillType = tFillType.Solid;
portion.PortionFormat.FillFormat.SolidFillColor.Color = tColor.Black;
```

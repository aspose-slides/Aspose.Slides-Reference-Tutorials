## Οδηγός Εφαρμογής

Ας εξερευνήσουμε πώς να εφαρμόσουμε λειτουργίες ευθυγράμμισης σχήματος χρησιμοποιώντας το Aspose.Slides για .NET.

### Στοίχιση σχημάτων σε διαφάνεια (H2)
Αυτή η λειτουργία δείχνει πώς να ευθυγραμμίσετε τα σχήματα μέσα σε μια ολόκληρη διαφάνεια. Δείτε πώς μπορείτε να το πετύχετε:

#### Βήμα 1: Δημιουργία και προσθήκη σχημάτων
Προσθέστε μερικά ορθογώνια στη διαφάνειά σας ως placeholder:

```csharp
ISlide slide = pres.Slides[0];
slide.Shapes.AddAutoShape(ShapeType.Rectangle, 100, 100, 100, 100);
slide.Shapes.AddAutoShape(ShapeType.Rectangle, 200, 200, 100, 100);
slide.Shapes.AddAutoShape(ShapeType.Rectangle, 300, 300, 100, 100);
```

#### Βήμα 2: Ευθυγράμμιση σχημάτων
Χρησιμοποιήστε το `AlignShapes` μέθοδος για την ευθυγράμμιση αυτών των σχημάτων στο κάτω μέρος:

```csharp
SlideUtil.AlignShapes(ShapesAlignmentType.AlignBottom, true, pres.Slides[0]);
```
**Εξήγηση:** Οι παράμετροι ορίζουν τον τύπο ευθυγράμμισης (`AlignBottom`), εάν θα συμπεριληφθεί κείμενο (`true`), και διαφάνεια στόχου.

#### Βήμα 3: Αποθήκευση της παρουσίασης
Αποθηκεύστε τις αλλαγές σας σε ένα νέο αρχείο:

```csharp
pres.Save("ShapesAlignment_out.pptx", SaveFormat.Pptx);
```

### Στοίχιση σχημάτων σε σχήμα ομάδας (H2)
Αυτή η ενότητα δείχνει πώς να ευθυγραμμίσετε τα σχήματα μέσα σε ένα σχήμα ομάδας, διασφαλίζοντας συνεκτική ευθυγράμμιση.

#### Βήμα 1: Δημιουργία σχήματος ομάδας και προσθήκη σχημάτων
Προσθέστε τα σχήματά σας σε μια νέα ομάδα:

```csharp
ISlide slide = pres.Slides.AddEmptySlide(pres.Slides[0].LayoutSlide);
IGroupShape groupShape = slide.Shapes.AddGroupShape();
groupShape.Shapes.AddAutoShape(ShapeType.Rectangle, 350, 50, 50, 50);
// Προσθέστε περισσότερα σχήματα όπως απαιτείται
```

#### Βήμα 2: Στοίχιση σχημάτων εντός ομάδας
Ευθυγραμμίστε όλα αυτά τα σχήματα προς τα αριστερά μέσα στην ομάδα τους:

```csharp
SlideUtil.AlignShapes(ShapesAlignmentType.AlignLeft, false, groupShape);
```

### Στοίχιση συγκεκριμένων σχημάτων σε σχήμα ομάδας (H2)
Μπορείτε επίσης να στοχεύσετε συγκεκριμένα σχήματα για στοίχιση χρησιμοποιώντας ευρετήρια.

#### Βήμα 1: Ορίστε το σχήμα της ομάδας σας
Όπως και στην προηγούμενη ενότητα, δημιουργήστε την ομάδα σας και προσθέστε σχήματα:

```csharp
IGroupShape groupShape = slide.Shapes.AddGroupShape();
groupShape.Shapes.AddAutoShape(ShapeType.Rectangle, 350, 50, 50, 50);
// Επιπλέον σχήματα...
```

#### Βήμα 2: Ευθυγράμμιση συγκεκριμένων σχημάτων
Χρησιμοποιήστε ευρετήρια για να καθορίσετε ποια σχήματα θα ευθυγραμμιστούν:

```csharp
SlideUtil.AlignShapes(ShapesAlignmentType.AlignLeft, false, groupShape, new int[] { 0, 2 });
```
**Εξήγηση:** Αυτό ευθυγραμμίζει μόνο το πρώτο και το τρίτο σχήμα μέσα στην ομάδα.

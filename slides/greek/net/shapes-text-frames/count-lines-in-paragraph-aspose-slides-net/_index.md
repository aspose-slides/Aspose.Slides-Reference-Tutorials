## Οδηγός Εφαρμογής

Θα αναλύσουμε τη διαδικασία σε διαχειρίσιμα βήματα για την καταμέτρηση γραμμών σε μια παράγραφο χρησιμοποιώντας το Aspose.Slides.

### Βήμα 1: Δημιουργία νέας παρουσίασης

Ξεκινήστε δημιουργώντας μια παρουσία μιας παρουσίασης. Αυτός θα είναι ο χώρος εργασίας μας για την προσθήκη διαφανειών και σχημάτων.

```csharp
using (Presentation presentation = new Presentation())
{
    // Αποκτήστε πρόσβαση στη διαφάνειά σας εδώ...
}
```

### Βήμα 2: Προσθήκη διαφάνειας και σχήματος

Αποκτήστε πρόσβαση στην πρώτη διαφάνεια και, στη συνέχεια, προσθέστε ένα σχήμα όπου θα τοποθετήσετε κείμενο για ανάλυση.

```csharp
ISlide sld = presentation.Slides[0];
IAutoShape ashp = sld.Shapes.AddAutoShape(ShapeType.Rectangle, 150, 75, 150, 50);
```

### Βήμα 3: Εισαγωγή κειμένου και μέτρηση γραμμών

Εισαγάγετε κείμενο στην πρώτη παράγραφο του σχήματος και χρησιμοποιήστε `GetLinesCount()` να μετρήσω γραμμές.

```csharp
IParagraph para = ashp.TextFrame.Paragraphs[0];
IPortion portion = para.Portions[0];
portion.Text = "Aspose Paragraph GetLinesCount() Example";

int lineCount = para.GetLinesCount();
Console.WriteLine("Lines Count = {0}", lineCount);
```

### Βήμα 4: Προσαρμογή διαστάσεων σχήματος

Δείξτε πώς η αλλαγή των διαστάσεων του σχήματος μπορεί να επηρεάσει τον αριθμό των γραμμών.

```csharp
ashp.Width = 250;
int newLineCount = para.GetLinesCount();
Console.WriteLine("Lines Count after changing shape width = {0}", newLineCount);
```

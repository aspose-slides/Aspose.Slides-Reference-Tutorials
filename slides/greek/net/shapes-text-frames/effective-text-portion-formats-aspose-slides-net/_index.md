## Οδηγός Εφαρμογής
Αυτή η ενότητα καλύπτει την ανάκτηση αποτελεσματικών μορφών πλαισίων και τμημάτων κειμένου χρησιμοποιώντας το Aspose.Slides για .NET.

### Αποκτήστε αποτελεσματική μορφή TextFrame
#### Επισκόπηση
Ανακτήστε όλες τις ενεργές ιδιότητες ενός πλαισίου κειμένου σε μια διαφάνεια του PowerPoint για να κατανοήσετε τόσο την τοπική μορφοποίηση όσο και τα κληρονομημένα στυλ από γονικές διαφάνειες ή κύριες διατάξεις.
##### Βήμα 1: Φόρτωση της παρουσίασης
Φορτώστε το αρχείο παρουσίασής σας χρησιμοποιώντας το Aspose.Slides `Presentation` τάξη:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
using (Presentation pres = new Presentation(dataDir + "Presentation1.pptx"))
{
    // Η πρόσβαση στη λογική των διαφανειών και των σχημάτων ακολουθεί εδώ...
}
```
##### Βήμα 2: Πρόσβαση στο Αυτόματο Σχήμα
Ανάκτηση του `AutoShape` που περιέχει το κείμενο-στόχο σας από την πρώτη διαφάνεια:
```csharp
IAutoShape shape = pres.Slides[0].Shapes[0] as IAutoShape;
```
##### Βήμα 3: Ανάκτηση TextFrameFormat και Ενεργών Ιδιοτήτων
Αποκτήστε το τοπικό `TextFrameFormat` για το σχήμα, στη συνέχεια χρησιμοποιήστε `GetEffective()` για να ανακτήσετε όλες τις αποτελεσματικές ιδιότητες:
```csharp
ITextFrameFormat localTextFrameFormat = shape.TextFrame.TextFrameFormat;
ITextFrameFormatEffectiveData effectiveTextFrameFormat = localTextFrameFormat.GetEffective();
```
### Αποκτήστε Αποτελεσματική Μορφή Μερίδας
#### Επισκόπηση
Αποκτήστε πρόσβαση στις ενεργές ιδιότητες ενός τμήματος κειμένου μέσα σε ένα σχήμα για λεπτομερείς ανάγκες στυλ.
##### Βήμα 1: Φόρτωση της παρουσίασης
Φορτώστε το αρχείο PowerPoint σας με παρόμοιο τρόπο:
```csharp
using (Presentation pres = new Presentation(dataDir + "Presentation1.pptx"))
{
    // Η πρόσβαση στη λογική των διαφανειών και των σχημάτων ακολουθεί εδώ...
}
```
##### Βήμα 2: Πρόσβαση στη μορφή μερίδας
Μεταβείτε στην πρώτη παράγραφο και το τμήμα μέσα σε ένα `AutoShape` στη διαφάνειά σας:
```csharp
IAutoShape shape = pres.Slides[0].Shapes[0] as IAutoShape;
IPortionFormat localPortionFormat = shape.TextFrame.Paragraphs[0].Portions[0].PortionFormat;
```
##### Βήμα 3: Ανάκτηση Ισχύοντος Ιδιοτήτων
Χρήση `GetEffective()` για να ανακτήσετε όλες τις αποτελεσματικές ιδιότητες:
```csharp
IPortionFormatEffectiveData effectivePortionFormat = localPortionFormat.GetEffective();
```

Ας αναλύσουμε τη διαδικασία σε σαφή βήματα για την επανάληψη σχημάτων και τον εντοπισμό πλαισίων κειμένου.

### Χαρακτηριστικό: Επανάληψη σχημάτων παρουσίασης

Αυτή η λειτουργία εστιάζει στην επανάληψη όλων των σχημάτων που υπάρχουν σε μια διαφάνεια, ελέγχοντας αν το καθένα είναι πλαίσιο κειμένου. Δείτε πώς μπορείτε να την εφαρμόσετε:

#### Βήμα 1: Φόρτωση της παρουσίασής σας

Αρχικά, βεβαιωθείτε ότι η διαδρομή του αρχείου παρουσίασής σας έχει οριστεί σωστά:

```csharp
string presentationPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "CheckTextShapes.pptx");
```

Ανοίξτε την παρουσίαση χρησιμοποιώντας το Aspose.Slides:

```csharp
using (Presentation presentation = new Presentation(presentationPath))
{
    // Ο κώδικας για την επανάληψη σχημάτων θα τοποθετηθεί εδώ
}
```

#### Βήμα 2: Επαναλάβετε τα σχήματα

Πλοηγηθείτε σε κάθε σχήμα σε μια συγκεκριμένη διαφάνεια. Σε αυτό το παράδειγμα, εξετάζουμε την πρώτη διαφάνεια:

```csharp
foreach (IShape shape in presentation.Slides[0].Shapes)
{
    // Ελέγξτε εάν το σχήμα είναι αυτόματο σχήμα και προσδιορίστε εάν είναι πλαίσιο κειμένου
}
```

#### Βήμα 3: Προσδιορίστε τα πλαίσια κειμένου

Ελέγξτε αν κάθε σχήμα είναι `AutoShape` και στη συνέχεια επαληθεύστε αν περιέχει κείμενο:

```csharp
if (shape is AutoShape autoShape)
{
    bool isTextBox = autoShape.IsTextBox;
    // Χρησιμοποιήστε την συνάρτηση 'isTextBox' για να προσδιορίσετε εάν το σχήμα είναι πλαίσιο κειμένου.
}
```

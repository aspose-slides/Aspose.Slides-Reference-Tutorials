### Οδηγός Εφαρμογής

#### Εφαρμογή γεμίσματος με διαβάθμιση σε σχήματα

Αυτή η λειτουργία σάς επιτρέπει να βελτιώσετε την οπτική εμφάνιση των σχημάτων στις διαφάνειες του PowerPoint προσθέτοντας ένα γέμισμα με διαβάθμιση. Ας εξερευνήσουμε πώς να το εφαρμόσετε:

##### Βήμα 1: Δημιουργήστε ένα σχήμα έλλειψης

```csharp
// Φόρτωση ή δημιουργία μιας παρουσίασης\χρησιμοποιώντας (Presentation pres = new Presentation())
{
    // Πρόσβαση στην πρώτη διαφάνεια
    ISlide sld = pres.Slides[0];
    
    // Προσθήκη αυτόματου σχήματος τύπου έλλειψης
    IAutoShape ashp = sld.Shapes.AddAutoShape(ShapeType.Ellipse, 50, 150, 150, 50);
}
```

Σε αυτό το βήμα, δημιουργούμε μια έλλειψη στην πρώτη διαφάνεια. Οι παράμετροι καθορίζουν τη θέση και το μέγεθός της.

##### Βήμα 2: Εφαρμογή γεμίσματος με διαβάθμιση

```csharp
// Ορισμός τύπου γεμίσματος σε διαβάθμιση
ashp.FillFormat.FillType = FillType.Gradient;

// Ορίστε τα χρώματα και το στυλ διαβάθμισης
ashp.FillFormat.GradientFormat.StartColor = Color.Red;
ashp.FillFormat.GradientFormat.EndColor = Color.Blue;
ashp.FillFormat.GradientFormat.TileFlip = TileFlip.FlipBoth;
```

Εδώ, ρυθμίζουμε την έλλειψη ώστε να έχει γέμισμα με διαβάθμιση, μεταβαίνοντας από κόκκινο σε μπλε.

##### Βήμα 3: Αποθήκευση της παρουσίασης

```csharp
// Ορισμός διαδρομής εξόδου
string dataDir = "YOUR_DOCUMENT_DIRECTORY";

// Βεβαιωθείτε ότι υπάρχει κατάλογος
if (!Directory.Exists(dataDir))
{
    Directory.CreateDirectory(dataDir);
}

// Αποθήκευση της παρουσίασης
pres.Save(Path.Combine(dataDir, "GradientEllipse.pptx"), SaveFormat.Pptx);
```

Αυτό το απόσπασμα διασφαλίζει ότι η παρουσίαση αποθηκεύεται στον καθορισμένο κατάλογο.

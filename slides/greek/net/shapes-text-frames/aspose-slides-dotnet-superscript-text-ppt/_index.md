## Οδηγός Εφαρμογής
Μάθετε πώς να προσθέτετε κείμενο εκθέτη χρησιμοποιώντας το Aspose.Slides για .NET. Αυτή η λειτουργία σάς επιτρέπει να δημιουργείτε κομψές και λεπτομερείς διαφάνειες χωρίς κόπο.

### Προσθήκη κειμένου εκθέτη
#### Επισκόπηση
Βελτιώστε την αναγνωσιμότητα με κείμενο εκθέτη για τύπους, σχολιασμούς ή παραπομπές:

1. **Πρόσβαση στη διαφάνεια**: Τοποθετήστε μια διαφάνεια στο σημείο όπου θέλετε να προσθέσετε κείμενο.
2. **Δημιουργία σχήματος**Προσθέστε ένα σχήμα (όπως ένα ορθογώνιο) για να συγκρατήσετε το κείμενό σας.
3. **Ρύθμιση παραμέτρων πλαισίου κειμένου**: Ρυθμίστε το πλαίσιο κειμένου σας και διαγράψτε τις υπάρχουσες παραγράφους.
4. **Προσθήκη τμήματος εκθέτη**: Εισαγάγετε το τμήμα του κειμένου που πρέπει να είναι εκθέτης.

#### Βήμα προς βήμα εφαρμογή
**1. Πρόσβαση στη διαφάνεια**
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
using (Presentation presentation = new Presentation())
{
    ISlide slide = presentation.Slides[0];
```
Φορτώστε μια υπάρχουσα παρουσίαση και αποκτήστε πρόσβαση στην πρώτη της διαφάνεια.

**2. Δημιουργία σχήματος**
```csharp
IAutoShape shape = slide.Shapes.AddAutoShape(ShapeType.Rectangle, 100, 100, 200, 100);
ITextFrame textFrame = shape.TextFrame;
```
Προσθέστε ένα ορθογώνιο σχήμα στη διαφάνεια και προετοιμάστε την για εισαγωγή κειμένου.

**3. Ρύθμιση παραμέτρων πλαισίου κειμένου**
```csharp
textFrame.Paragraphs.Clear();
IParagraph superPar = new Paragraph();
```
Διαγράψτε τις υπάρχουσες παραγράφους για να ξεκινήσετε από την αρχή και, στη συνέχεια, δημιουργήστε μια νέα παράγραφο για το κείμενο του εκθέτη.

**4. Προσθήκη τμήματος εκθέτη**
Για να προσθέσετε εκθέτη:
- Δημιουργήστε κανονικά και εκθέτα τμήματα.
- Ορίστε το `PortionFormat.FontHeight` και άλλες ιδιότητες όπως απαιτείται.

```csharp
IPortion portion1 = new Portion { Text = "Slide Title" };
portion1.PortionFormat.FontHeight = 20;

// Κείμενο εκθέτη
IPortion portion2 = new Portion { Text = "Superscript Example" };
portion2.PortionFormat.FontHeight = 10;
portion2.ParagraphFormat.DefaultPortionFormat.FontBold = NullableBool.True;
portion2.TextFrame.Paragraphs[0].Portions[1].PortionFormat.Superscript = new Superscript() 
{ 
    FontSize = 50 %, 
    Position = SuperscriptPosition.VerticallyAboveBaseline
};

superPar.Portions.Add(portion1);
superPar.Portions.Add(portion2);
textFrame.Paragraphs.Add(superPar);
```
**Συμβουλές αντιμετώπισης προβλημάτων**:
- Εξασφαλίζω `PortionFormat.Superscript` έχει ρυθμιστεί σωστά με το κατάλληλο μέγεθος και θέση γραμματοσειράς.
- Βεβαιωθείτε ότι τα τμήματα προστίθενται στις παραγράφους με τη σωστή σειρά.

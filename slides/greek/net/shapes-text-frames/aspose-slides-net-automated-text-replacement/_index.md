## Οδηγός Εφαρμογής

Τώρα που έχετε ρυθμίσει τα πάντα, ας προχωρήσουμε στην εφαρμογή της λειτουργίας Αντικατάσταση κειμένου.

### Επισκόπηση λειτουργιών: Αντικατάσταση κειμένου σε διαφάνειες PowerPoint

Αυτή η λειτουργία αναζητά συγκεκριμένο κείμενο κράτησης θέσης (π.χ., "[αυτό το μπλοκ]") και το αντικαθιστά με το επιθυμητό περιεχόμενο σε όλες τις διαφάνειες. Είναι ιδιαίτερα χρήσιμη κατά την ενημέρωση κοινών φράσεων ή ονομάτων προϊόντων σε όλη την παρουσίαση.

#### Βήμα 1: Φόρτωση της παρουσίασής σας
Ξεκινήστε φορτώνοντας την παρουσίαση στο σημείο που θέλετε να αντικαταστήσετε το κείμενο:

```csharp
Presentation pres = new Presentation("example.pptx");
```

#### Βήμα 2: Ορισμός παραμέτρων αντικατάστασης κειμένου

Προσδιορίστε το σύμβολο κράτησης θέσης και το κείμενο αντικατάστασης. Για παράδειγμα, αντικαταστήστε το "[αυτό το μπλοκ]" με το "το κείμενό μου":

```csharp
string strToFind = "[this block]";
string strToReplaceWith = "my text";
```

#### Βήμα 3: Επανάληψη σε διαφάνειες και αντικατάσταση κειμένου

Πραγματοποιήστε επανάληψη σε κάθε διαφάνεια στην παρουσίασή σας για να βρείτε και να αντικαταστήσετε το κείμενο κράτησης θέσης:

```csharp
foreach (ISlide slide in pres.Slides)
{
    foreach (IAutoShape shape in slide.Shapes.OfType<IAutoShape>())
    {
        if (shape.TextFrame != null)
        {
            ITextFrame textFrame = shape.TextFrame;
            foreach (IParagraph para in textFrame.Paragraphs)
            {
                foreach (Portion portion in para.Portions)
                {
                    if (portion.Text.Contains(strToFind))
                    {
                        // Αντικατάσταση του κειμένου
                        portion.Text = portion.Text.Replace(strToFind, strToReplaceWith);
                    }
                }
            }
        }
    }
}
```

#### Εξήγηση:
- **Παράμετροι**: `strToFind` είναι το κείμενο κράτησης θέσης στο οποίο στοχεύετε. `strToReplaceWith` είναι αυτό που θέλετε να αντικαταστήσετε.
- **Σκοπός της μεθόδου**Η μέθοδος επαναλαμβάνει τα σχήματα κάθε διαφάνειας, αναζητώντας πλαίσια κειμένου με το καθορισμένο σύμβολο κράτησης θέσης και αντικαθιστώντας το.

### Συμβουλές αντιμετώπισης προβλημάτων

- Βεβαιωθείτε ότι οι μεταβλητές συμβολοσειράς κειμένου σας (`strToFind` και `strToReplaceWith`) έχουν οριστεί σωστά.
- Ελέγξτε αν οι διαφάνειες περιέχουν την αναμενόμενη μορφή (π.χ., να έχουν Αυτόματα Σχήματα) για να αποφύγετε εξαιρέσεις μηδενικής αναφοράς.

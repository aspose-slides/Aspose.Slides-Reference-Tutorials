### Επισήμανση κειμένου χρησιμοποιώντας Regex

Αυτή η λειτουργία σάς επιτρέπει να επισημαίνετε αυτόματα συγκεκριμένο κείμενο στις διαφάνειές σας με βάση ένα μοτίβο regex. Δείτε πώς λειτουργεί:

#### Επισκόπηση

Θα χρησιμοποιήσουμε μια κανονική έκφραση για να βρούμε όλες τις λέξεις με πέντε ή περισσότερους χαρακτήρες και να τις επισημάνουμε μέσα σε ένα Αυτόματο Σχήμα.

#### Βήμα προς βήμα εφαρμογή

1. **Πρόσβαση στη διαφάνεια και το σχήμα**
   Αποκτήστε πρόσβαση στην πρώτη διαφάνεια και στο πρώτο της σχήμα, υποθέτοντας ότι είναι ένα Αυτόματο Σχήμα:
   ```csharp
   using Aspose.Slides;
   
   string dataDir = "YOUR_DOCUMENT_DIRECTORY";
   Presentation presentation = new Presentation(dataDir + "SomePresentation.pptx");
   AutoShape shape = (AutoShape)presentation.Slides[0].Shapes[0];
   ```

2. **Ορισμός και εφαρμογή μοτίβου Regex**
   Χρησιμοποιήστε ένα μοτίβο regex για να προσδιορίσετε το κείμενο που θέλετε να επισημάνετε:
   ```csharp
   using System.Text.RegularExpressions;
   using System.Drawing;

   // Ορίστε το μοτίβο regex για λέξεις με 5 ή περισσότερους χαρακτήρες
   string pattern = @"\b[^\s]{5,}\b";

   // Επισήμανση κειμένου που ταιριάζει στο σχήμα
   shape.TextFrame.HighlightRegex(pattern);
   ```

3. **Αποθήκευση της παρουσίασης**
   Μόλις επισημάνετε το κείμενο που θέλετε, αποθηκεύστε την παρουσίαση:
   ```csharp
   presentation.Save(dataDir + "HighlightedPresentation.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
   ```

#### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι το σχήμα είναι πράγματι ένα Αυτόματο Σχήμα για να αποφύγετε σφάλματα χύτευσης.
- Επαληθεύστε ότι το μοτίβο regex ταιριάζει σωστά με τα κριτήριά σας.

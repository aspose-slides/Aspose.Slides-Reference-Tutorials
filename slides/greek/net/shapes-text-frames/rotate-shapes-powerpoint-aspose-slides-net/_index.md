## Οδηγός Εφαρμογής

### Περιστροφή σχημάτων στο PowerPoint

Αυτή η ενότητα σάς καθοδηγεί στην περιστροφή ενός αυτόματου σχήματος μέσα σε μια διαφάνεια για να προσθέσετε οπτικό ενδιαφέρον και να τονίσετε συγκεκριμένα μέρη του περιεχομένου.

#### Βήμα 1: Προετοιμάστε το περιβάλλον σας

Ορίστε τον κατάλογο για την αποθήκευση εγγράφων:
```csharp
string dataDir = @"YOUR_DOCUMENT_DIRECTORY";
bool isExists = System.IO.Directory.Exists(dataDir);
if (!isExists)
    System.IO.Directory.CreateDirectory(dataDir);
```
Αυτό διασφαλίζει ότι ο κατάλογος εξόδου σας υπάρχει, αποτρέποντας σφάλματα κατά την αποθήκευση αρχείων.

#### Βήμα 2: Δημιουργία νέας παρουσίασης

Αρχικοποίηση και πρόσβαση στην πρώτη διαφάνεια:
```csharp
using (Presentation pres = new Presentation())
{
    // Πρόσβαση στην πρώτη διαφάνεια
    ISlide sld = pres.Slides[0];
```
Δημιουργήστε μια παρουσία παρουσίασης και αποκτήστε πρόσβαση στην πρώτη της διαφάνεια για να προσθέσετε το σχήμα σας.

#### Βήμα 3: Προσθήκη και περιστροφή ενός αυτόματου σχήματος

Προσθέστε ένα ορθογώνιο σχήμα και περιστρέψτε το κατά 90 μοίρες:
```csharp
// Προσθήκη αυτόματου σχήματος ορθογωνίου
IShape shp = sld.Shapes.AddAutoShape(ShapeType.Rectangle, 50, 150, 75, 150);

// Περιστρέψτε το ορθογώνιο κατά 90 μοίρες
shp.Rotation = 90;
```
Ο `AddAutoShape` Η μέθοδος τοποθετεί το σχήμα σε καθορισμένες συντεταγμένες και διαστάσεις. Η `Rotation` Η ιδιότητα προσαρμόζει τη γωνία της.

#### Βήμα 4: Αποθηκεύστε την παρουσίασή σας

Αποθηκεύστε την παρουσίασή σας:
```csharp
// Αποθήκευση της τροποποιημένης παρουσίασης
pres.Save(dataDir + "RectShpRot_out.pptx");
}
```
Αυτό εγγράφει τις αλλαγές σας σε ένα αρχείο στον καθορισμένο κατάλογο.

## Οδηγός Εφαρμογής

Αυτή η ενότητα καλύπτει την υλοποίηση δύο κύριων λειτουργιών: τη δημιουργία καταλόγων και την προσθήκη σχημάτων έλλειψης σε διαφάνειες του PowerPoint χρησιμοποιώντας C#.

### Λειτουργία 1: Δημιουργία καταλόγου εάν δεν υπάρχει

**Επισκόπηση:** Αυτή η λειτουργία διασφαλίζει ότι υπάρχει ένας κατάλογος πριν από την εκτέλεση λειτουργιών αρχείων, αποτρέποντας σφάλματα που σχετίζονται με διαδρομές που λείπουν.

#### Βήμα προς βήμα εφαρμογή:

**Έλεγχος και δημιουργία καταλόγου**
```csharp
using System.IO;

string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Αντικαταστήστε με την πραγματική σας διαδρομή
bool isExists = Directory.Exists(dataDir);

if (!isExists)
{
    Directory.CreateDirectory(dataDir); // Δημιουργεί τον κατάλογο εάν δεν υπάρχει
}
```

- **Εξήγηση**: `Directory.Exists()` ελέγχει αν υπάρχει ένας κατάλογος και `Directory.CreateDirectory()` το δημιουργεί εάν απουσιάζει. Αυτό διασφαλίζει ότι όλες οι λειτουργίες αρχείου έχουν μια έγκυρη διαδρομή.

### Χαρακτηριστικό 2: Προσθήκη σχήματος έλλειψης στη διαφάνεια

**Επισκόπηση:** Αυτοματοποιήστε την προσθήκη σχημάτων σε διαφάνειες του PowerPoint, ξεκινώντας με ένα σχήμα έλλειψης στην πρώτη διαφάνεια.

#### Βήμα προς βήμα εφαρμογή:

**Προσθήκη σχήματος έλλειψης**
```csharp
using Aspose.Slides;
using Aspose.Slides.Export;

string outputDir = "YOUR_DOCUMENT_DIRECTORY"; // Αντικατάσταση με τη διαδρομή σας
string outputFile = Path.Combine(outputDir, "EllipseShape_out.pptx");

using (Presentation pres = new Presentation())
{
    ISlide sld = pres.Slides[0]; // Αποκτήστε την πρώτη διαφάνεια

    // Προσθέστε ένα σχήμα έλλειψης στη διαφάνεια στη θέση (50, 150) με πλάτος 150 και ύψος 50
    sld.Shapes.AddAutoShape(ShapeType.Ellipse, 50, 150, 150, 50);

    pres.Save(outputFile, SaveFormat.Pptx); // Αποθήκευση της παρουσίασης σε μορφή PPTX
}
```

- **Εξήγηση**: Το `AddAutoShape` Η μέθοδος σάς επιτρέπει να καθορίσετε τον τύπο σχήματος και τις διαστάσεις. Αυτό το τμήμα κειμένου προσθέτει μια έλλειψη στην πρώτη διαφάνεια μιας νέας παρουσίασης.

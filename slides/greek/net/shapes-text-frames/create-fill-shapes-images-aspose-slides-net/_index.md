## Οδηγός Εφαρμογής

Αυτή η ενότητα αναλύει τη διαδικασία σε διαχειρίσιμα χαρακτηριστικά.

### Δημιουργία καταλόγων

Για να διασφαλίσουμε ότι τα αρχεία της παρουσίασής μας αποθηκεύονται σωστά, ελέγχουμε πρώτα αν ο κατάλογος προορισμού υπάρχει. Εάν όχι, τον δημιουργούμε:

### Εργασία με παρουσιάσεις

Ξεκινάμε δημιουργώντας μια παρουσία μιας παρουσίασης και στη συνέχεια χειριζόμαστε τις διαφάνειές της:
```csharp
using Aspose.Slides;

// Δημιουργία αρχικού στιγμιότυπου της κλάσης παρουσίασης που αντιπροσωπεύει το αρχείο PPTX
using (Presentation pres = new Presentation())
{
    // Λήψη της πρώτης διαφάνειας από την παρουσίαση
    ISlide sld = pres.Slides[0];

    // Προσθήκη αυτόματου σχήματος τύπου ορθογωνίου στη διαφάνεια
    IShape shp = sld.Shapes.AddAutoShape(ShapeType.Rectangle, 50, 150, 75, 150);
}
```

### Ρύθμιση πλήρωσης σχήματος με εικόνα

Στη συνέχεια, γεμίζουμε ένα σχήμα με μια εικόνα ορίζοντας τον τύπο γεμίσματός του:
```csharp
using Aspose.Slides;
using System.Drawing;

// Ορίστε τον τύπο γεμίσματος του σχήματος σε Εικόνα
shp.FillFormat.FillType = FillType.Picture;
// Ρύθμιση παραμέτρων της λειτουργίας γεμίσματος εικόνας ως Παράθεση
shp.FillFormat.PictureFillFormat.PictureFillMode = PictureFillMode.Tile;

// Φόρτωση μιας εικόνας από έναν καθορισμένο κατάλογο και ορισμός της στη μορφή γεμίσματος του σχήματος
IImage img = Images.FromFile("YOUR_DOCUMENT_DIRECTORY/Tulips.jpg");
IPPImage imgx = pres.Images.AddImage(img);
shp.FillFormat.PictureFillFormat.Picture.Image = imgx;
```

### Αποθήκευση παρουσιάσεων

Τέλος, αποθηκεύστε την παρουσίασή σας με όλες τις αλλαγές:
```csharp
using Aspose.Slides.Export;

// Αποθήκευση της τροποποιημένης παρουσίασης ξανά στο δίσκο
pres.Save("YOUR_OUTPUT_DIRECTORY/RectShpPic_out.pptx", SaveFormat.Pptx);
```

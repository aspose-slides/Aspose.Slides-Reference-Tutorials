## Οδηγός Εφαρμογής

Τώρα, ας προχωρήσουμε στην κλωνοποίηση αυτών των σχημάτων! Θα αναλύσουμε κάθε μέρος της διαδικασίας για λόγους σαφήνειας.

### Κλωνοποίηση σχημάτων μεταξύ διαφανειών

#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να αντιγράφετε συγκεκριμένα σχήματα από μια διαφάνεια και να τα τοποθετείτε σε μια άλλη, είτε σε καθορισμένες συντεταγμένες είτε από προεπιλεγμένη τοποθέτηση.

#### Βήμα προς βήμα εφαρμογή

**Ρύθμιση της παρουσίασής σας**

Ξεκινήστε ορίζοντας τη διαδρομή του εγγράφου σας και φορτώνοντας την παρουσίασή σας:

```csharp
string dataDir = @"YOUR_DOCUMENT_DIRECTORY";
using (Presentation srcPres = new Presentation(dataDir + "Source Frame.pptx"))
{
    // Συνέχιση των εργασιών κλωνοποίησης
}
```

**Πρόσβαση σε συλλογές σχημάτων**

Ανάκτηση των συλλογών σχημάτων από τις διαφάνειες προέλευσης και προορισμού:

```csharp
// Λήψη της συλλογής σχημάτων από την πρώτη διαφάνεια
IShapeCollection sourceShapes = srcPres.Slides[0].Shapes;

// Αποκτήστε μια κενή διαφάνεια διάταξης για να δημιουργήσετε μια νέα διαφάνεια χωρίς περιεχόμενο
ILayoutSlide blankLayout = srcPres.Masters[0].LayoutSlides.GetByType(SlideLayoutType.Blank);

// Προσθήκη κενής διαφάνειας χρησιμοποιώντας την κενή διάταξη
ISlide destSlide = srcPres.Slides.AddEmptySlide(blankLayout);
IShapeCollection destShapes = destSlide.Shapes;
```

**Κλωνοποίηση σχημάτων με καθορισμένες συντεταγμένες**

Κλωνοποιήστε ένα συγκεκριμένο σχήμα και τοποθετήστε το στις επιθυμητές συντεταγμένες στη διαφάνεια προορισμού:

```csharp
// Κλωνοποίηση σχήματος σε καθορισμένες συντεταγμένες στη διαφάνεια προορισμού
destShapes.AddClone(sourceShapes[1], 50, 150 + sourceShapes[0].Height);
```

**Κλωνοποίηση σχήματος χωρίς νέα θέση**

Μπορείτε επίσης να κλωνοποιήσετε σχήματα χωρίς να καθορίσετε νέες συντεταγμένες. Θα προστεθούν διαδοχικά:

```csharp
// Κλωνοποίηση ενός άλλου σχήματος στην προεπιλεγμένη θέση στη διαφάνεια προορισμού
destShapes.AddClone(sourceShapes[2]);
```

**Εισαγωγή κλωνοποιημένου σχήματος σε συγκεκριμένο δείκτη**

Εισαγάγετε ένα κλωνοποιημένο σχήμα στην αρχή της συλλογής σχημάτων της διαφάνειας προορισμού:

```csharp
// Εισαγωγή κλωνοποιημένου σχήματος στο ευρετήριο 0 με καθορισμένες συντεταγμένες
destShapes.InsertClone(0, sourceShapes[0], 50, 150);
```

### Αποθήκευση της παρουσίασής σας

Τέλος, αποθηκεύστε την τροποποιημένη παρουσίασή σας στο δίσκο:

```csharp
srcPres.Save(dataDir + "CloneShape_out.pptx", SaveFormat.Pptx);
```

## Οδηγός Εφαρμογής

Αυτή η ενότητα χωρίζεται σε δύο κύριες λειτουργίες: δημιουργία καταλόγων εάν δεν υπάρχουν και εργασία με σχήματα παρουσίασης για την προσθήκη εικόνων.

### Δημιουργία καταλόγων

#### Επισκόπηση
Η διασφάλιση της ύπαρξης ενός καταλόγου πριν από την εκτέλεση εργασιών σε αρχεία είναι ζωτικής σημασίας. Αυτή η λειτουργία βοηθά στον έλεγχο της ύπαρξης ενός συγκεκριμένου καταλόγου και τον δημιουργεί εάν απουσιάζει, αποτρέποντας πιθανά σφάλματα κατά τη διάρκεια των χειρισμών αρχείων.

#### Βήματα Υλοποίησης

**Βήμα 1: Ορισμός διαδρομής καταλόγου**
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
```
*Αντικαθιστώ `YOUR_DOCUMENT_DIRECTORY` με την επιθυμητή σας διαδρομή.*

**Βήμα 2: Έλεγχος και δημιουργία καταλόγου**
```csharp
bool isExists = Directory.Exists(dataDir);
if (!isExists) {
    Directory.CreateDirectory(dataDir);
}
```
Αυτός ο κώδικας ελέγχει αν ο κατάλογος υπάρχει χρησιμοποιώντας `Directory.Exists`Αν επιστρέψει false, `Directory.CreateDirectory` καλείται για τη δημιουργία του καταλόγου.

### Εργασία με παρουσιάσεις και σχήματα

#### Επισκόπηση
Η ενσωμάτωση εικόνων στις παρουσιάσεις σας μπορεί να τις κάνει πιο ελκυστικές. Αυτή η λειτουργία δείχνει πώς να φορτώσετε μια παρουσίαση, να προσθέσετε μια εικόνα ως γέμισμα σχήματος και να διαμορφώσετε μετατοπίσεις για καλύτερη τοποθέτηση.

#### Βήματα Υλοποίησης

**Βήμα 1: Φόρτωση εικόνας**
```csharp
IImage img = Images.FromFile(dataDir + "aspose-logo.jpg");
```
*Βεβαιωθείτε ότι η διαδρομή της εικόνας είναι σωστή.*

**Βήμα 2: Αρχικοποίηση παρουσίασης και προσθήκη σχήματος**
```csharp
using (Presentation pres = new Presentation()) {
    ISlide slide = pres.Slides[0];
    IAutoShape aShape = slide.Shapes.AddAutoShape(ShapeType.Rectangle, 100, 100, 300, 300);
    
    aShape.FillFormat.FillType = FillType.Picture;
    aShape.FillFormat.PictureFillFormat.PictureFillMode = PictureFillMode.Stretch;
    IPPImage imgEx = pres.Images.AddImage(img);
    aShape.FillFormat.PictureFillFormat.Picture.Image = imgEx;

    // Ορισμός μετατοπίσεων
    aShape.FillFormat.PictureFillFormat.StretchOffsetLeft = 25;
    aShape.FillFormat.PictureFillFormat.StretchOffsetRight = 25;
    aShape.FillFormat.PictureFillFormat.StretchOffsetTop = -20;
    aShape.FillFormat.PictureFillFormat.StretchOffsetBottom = -10;

    pres.Save(dataDir + "StretchOffsetLeftForPictureFrame_out.pptx", SaveFormat.Pptx);
}
```
Αυτό το απόσπασμα φορτώνει μια εικόνα, την προσθέτει στην πρώτη διαφάνεια ως γέμισμα σχήματος ορθογωνίου και ορίζει μετατοπίσεις για βελτιωμένη ευθυγράμμιση.

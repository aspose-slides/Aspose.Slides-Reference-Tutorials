### Επισκόπηση λειτουργιών: Ορισμός γλώσσας κειμένου σχήματος

Αυτή η λειτουργία σάς επιτρέπει να καθορίσετε τη γλώσσα του κειμένου μέσα σε ένα σχήμα PowerPoint. Ορίζοντας το αναγνωριστικό γλώσσας, διασφαλίζετε ότι ο ορθογραφικός έλεγχος και άλλες λειτουργίες που αφορούν συγκεκριμένες γλώσσες εφαρμόζονται σωστά.

#### Βήμα 1: Αρχικοποίηση παρουσίασης

Ξεκινήστε δημιουργώντας μια παρουσία του `Presentation` τάξη.

```csharp
using (Presentation pres = new Presentation())
{
    // Ο κωδικός σας εδώ
}
```

Αυτό αρχικοποιεί ένα νέο αντικείμενο παρουσίασης PowerPoint το οποίο θα χειριστούμε.

#### Βήμα 2: Προσθήκη αυτόματου σχήματος και πλαισίου κειμένου

Προσθέστε ένα ορθογώνιο σχήμα στη διαφάνειά σας και εισαγάγετε κείμενο σε αυτό:

```csharp
IAutoShape shape = pres.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 50, 50, 200, 50);
shape.AddTextFrame("Text to apply spellcheck language");
```

Εδώ, `AddAutoShape` προσθέτει ένα ορθογώνιο στην πρώτη διαφάνεια. Οι παράμετροι καθορίζουν τη θέση και το μέγεθός του.

#### Βήμα 3: Ορισμός αναγνωριστικού γλώσσας

```csharp

```

Αυτό ορίζει τα Αγγλικά (ΗΒ) ως τη γλώσσα για τον ορθογραφικό έλεγχο.

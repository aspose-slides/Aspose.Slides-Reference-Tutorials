### Επισκόπηση της λειτουργίας
Αυτή η λειτουργία δείχνει πώς να δημιουργήσετε ένα ορθογώνιο σχήμα σε μια διαφάνεια και να το γεμίσετε με μια εικόνα χρησιμοποιώντας το Aspose.Slides. Αυτή η τεχνική μπορεί να χρησιμοποιηθεί για να βελτιώσετε τις διαφάνειές σας προσθέτοντας λογότυπα, φόντα ή οποιαδήποτε γραφικά στοιχεία που κάνουν την παρουσίασή σας πιο ελκυστική.

### Βήμα προς βήμα εφαρμογή
#### 1. Αρχικοποίηση του αντικειμένου παρουσίασης
Ξεκινήστε δημιουργώντας ένα νέο αντικείμενο παρουσίασης. Αυτό θα χρησιμεύσει ως έγγραφο εργασίας μας όπου θα προσθέσουμε σχήματα και άλλα στοιχεία.
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Ορίστε τη διαδρομή του καταλόγου εγγράφων σας
total slides count: pres.Slides.Count;
using (Presentation pres = new Presentation())
{
    ISlide firstSlide = pres.Slides[0]; // Πρόσβαση στην πρώτη διαφάνεια

    // Φόρτωση εικόνας για χρήση ως γέμισμα
    IPPImage ppImage;
    using (IImage newImage = Aspose.Slides.Images.FromFile(Path.Combine(dataDir, "image.png")))
        ppImage = pres.Images.AddImage(newImage); // Προσθήκη εικόνας στη συλλογή εικόνων της παρουσίασης

    // Προσθέτει ένα ορθογώνιο σχήμα με καθορισμένες διαστάσεις
    var newShape = firstSlide.Shapes.AddAutoShape(ShapeType.Rectangle, 0, 0, 350, 350);

    // Ορισμός τύπου γεμίσματος του σχήματος σε Εικόνα
    newShape.FillFormat.FillType = FillType.Picture;
    IPictureFillFormat pictureFillFormat = newShape.FillFormat.PictureFillFormat;
    pictureFillFormat.Picture.Image = ppImage; // Ορισμός φορτωμένης εικόνας ως γέμισμα για το ορθογώνιο

    // Αποθήκευση της παρουσίασης
    pres.Save(Path.Combine("YOUR_OUTPUT_DIRECTORY", "RectangleWithPictureFill.pptx"), SaveFormat.Pptx);
}
```
#### Επεξήγηση Βασικών Βημάτων:
- **Φόρτωση εικόνας**: Το `FromFile` Η μέθοδος φορτώνει μια εικόνα από τον καθορισμένο κατάλογο, η οποία στη συνέχεια προστίθεται στη συλλογή εικόνων της παρουσίασης.
  
- **Προσθήκη ορθογωνίου σχήματος**Χρησιμοποιούμε `AddAutoShape` με `ShapeType.Rectangle` και ορίστε τις διαστάσεις του. Αυτό δημιουργεί ένα ορθογώνιο στη διαφάνεια.

- **Ρύθμιση γεμίσματος εικόνας**: Με την ανάθεση `FillType.Picture` στη μορφή γεμίσματος του σχήματος, μετατρέπουμε το ορθογώνιο σε ένα δοχείο εικόνας. Η εικόνα που φορτώνεται στη συνέχεια ορίζεται ως αυτό το γέμισμα χρησιμοποιώντας το `Picture.Image` ιδιοκτησία.


```csharp
using Aspose.Slides;
```

## Οδηγός Εφαρμογής

Ακολουθήστε αυτά τα βήματα για να δημιουργήσετε μια διαφάνεια PowerPoint με κατακόρυφα περιστρεφόμενο κείμενο:

### Βήμα 1: Ρύθμιση καταλόγου αποθήκευσης εγγράφων
Ορίστε πού θα αποθηκεύονται οι παρουσιάσεις σας:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
```

Αυτή η διαδρομή είναι κρίσιμη για την αποθήκευση και την πρόσβαση στα αρχεία της παρουσίασής σας.

### Βήμα 2: Δημιουργία νέας παρουσίασης
Αρχικοποίηση του `Presentation` κλάση για να ξεκινήσετε ένα νέο αρχείο PowerPoint:

```csharp
Presentation presentation = new Presentation();
```

Ο `Presentation` Το αντικείμενο λειτουργεί ως ο περιέκτης για όλες τις διαφάνειες και το περιεχόμενο.

### Βήμα 3: Πρόσβαση στην πρώτη διαφάνεια
Ανακτήστε την πρώτη διαφάνεια από την παρουσίασή σας:

```csharp
ISlide slide = presentation.Slides[0];
```

Αυτό το βήμα διασφαλίζει ότι έχουμε μια διαφάνεια για να προσθέσουμε το περιστρεφόμενο κείμενό μας.

### Βήμα 4: Προσθήκη Αυτόματου Σχήματος για Κείμενο
Προσθέστε ένα ορθογώνιο σχήμα για να περιέχει το κείμενο:

```csharp
IAutoShape ashp = slide.Shapes.AddAutoShape(ShapeType.Rectangle, 150, 75, 350, 350);
```

Εδώ, `ShapeType.Rectangle` επιλέγεται για την ευελιξία του στην περιεκτικότητα κειμένου.

### Βήμα 5: Ρύθμιση παραμέτρων TextFrame και περιστροφής
Προσθέστε ένα πλαίσιο κειμένου στο σχήμα και ορίστε την περιστροφή:

```csharp
ashp.AddTextFrame(" ");
ashp.FillFormat.FillType = FillType.NoFill;
ITextFrame txtFrame = ashp.TextFrame;
txtFrame.TextFrameFormat.TextVerticalType = TextVerticalType.Vertical270;
```

Ο `TextVerticalType` Η ιδιότητα καθορίζει τον προσανατολισμό του κειμένου μέσα στο πλαίσιο.

### Βήμα 6: Προσθήκη και μορφοποίηση κειμένου
Εισαγωγή παραγράφου με μορφοποιημένο κείμενο στο πλαίσιο κειμένου:

```csharp
IParagraph para = txtFrame.Paragraphs[0];
IPortion portion = para.Portions[0];
portion.Text = "A quick brown fox jumps over the lazy dog. A quick brown fox jumps over the lazy dog.";
portion.PortionFormat.FillFormat.FillType = FillType.Solid;
portion.PortionFormat.FillFormat.SolidFillColor.Color = Color.Black;
```

Αυτό το απόσπασμα προσθέτει περιεχόμενο κειμένου και ορίζει το χρώμα του σε μαύρο για καλύτερη ορατότητα.

### Βήμα 7: Αποθηκεύστε την παρουσίασή σας
Τέλος, αποθηκεύστε την παρουσίασή σας με το περιστρεφόμενο κείμενο:

```csharp
presentation.Save(dataDir + "RotateText_out.pptx", SaveFormat.Pptx);
```

Το αρχείο θα αποθηκευτεί στον καθορισμένο κατάλογο ως αρχείο PowerPoint.

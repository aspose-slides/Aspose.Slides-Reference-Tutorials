# Προσθήκη κινούμενων σχημάτων σε παρουσιάσεις χρησιμοποιώντας το Aspose.Slides για .NET

## Εισαγωγή

Η προσθήκη διαδραστικών στοιχείων, όπως κινούμενα σχήματα, μπορεί να βελτιώσει σημαντικά την παρουσίασή σας. Αυτό το σεμινάριο θα σας καθοδηγήσει στη χρήση του Aspose.Slides για .NET για να προσθέσετε ένα κινούμενο σχήμα κουμπιού στις διαφάνειές σας, κάνοντάς τες πιο ελκυστικές και αξέχαστες.

**Τι θα μάθετε:**
- Πώς να δημιουργήσετε καταλόγους σε C# με το Aspose.Slides
- Προσθήκη βασικών σχημάτων με εφέ κίνησης
- Υλοποίηση διαδραστικών κουμπιών με προσαρμοσμένες διαδρομές κίνησης

Είστε έτοιμοι να αναβαθμίσετε τις παρουσιάσεις σας; Ας δούμε βήμα προς βήμα τη ρύθμιση του περιβάλλοντός σας και τον προγραμματισμό αυτών των λειτουργιών.

### Χαρακτηριστικό 2: Προσθήκη σχήματος με κινούμενα σχέδια

Στη συνέχεια, ας προσθέσουμε ένα σχήμα σε μια διαφάνεια και ας εφαρμόσουμε ένα εφέ κίνησης χρησιμοποιώντας το Aspose.Slides:

#### Προσθήκη κινούμενων σχημάτων
```csharp
using Aspose.Slides;
using Aspose.Slides.Animation;

string outputDir = "YOUR_OUTPUT_DIRECTORY";

// Δημιουργήστε μια νέα παρουσίαση.
using (Presentation pres = new Presentation())
{
    ISlide sld = pres.Slides[0];

    // Προσθέστε ένα ορθογώνιο σχήμα με κείμενο στη διαφάνεια.
    IAutoShape ashp = sld.Shapes.AddAutoShape(ShapeType.Rectangle, 150, 150, 250, 25);
    ashp.AddTextFrame("Animated TextBox");

    // Εφαρμόστε το εφέ κίνησης PathFootball στο σχήμα.
    sld.Timeline.MainSequence.AddEffect(
        ashp,
        EffectType.PathFootball,
        EffectSubtype.None,
        EffectTriggerType.AfterPrevious
    );

    // Αποθηκεύστε την παρουσίαση με κινούμενα σχέδια.
    pres.Save(outputDir + "AnimExample_out.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
}
```

Αυτός ο κώδικας προσθέτει ένα ορθογώνιο σχήμα στη διαφάνειά σας και εφαρμόζει ένα κινούμενο εφέ, καθιστώντας την πιο ελκυστική.

### Χαρακτηριστικό 3: Προσθήκη διαδραστικού σχήματος κουμπιού με προσαρμοσμένη διαδρομή κίνησης

Για διαδραστικές παρουσιάσεις, δημιουργήστε σχήματα κουμπιών που ενεργοποιούν προσαρμοσμένες κινούμενες εικόνες:

#### Δημιουργία διαδραστικών κουμπιών
```csharp
using Aspose.Slides;
using Aspose.Slides.Animation;

string outputDir = "YOUR_OUTPUT_DIRECTORY";

// Δημιουργήστε μια νέα παρουσίαση.
using (Presentation pres = new Presentation())
{
    ISlide sld = pres.Slides[0];

    // Δημιουργήστε ένα σχήμα κουμπιού στη διαφάνεια.
    IShape shapeTrigger = sld.Shapes.AddAutoShape(ShapeType.Bevel, 10, 10, 20, 20);

    // Προσθήκη διαδραστικής ακολουθίας στο κουμπί.
    ISequence seqInter = sld.Timeline.InteractiveSequences.Add(shapeTrigger);

    // Υποθέστε ότι το δεύτερο σχήμα είναι ο στόχος μας για την κινούμενη εικόνα.
    IAutoShape ashp = sld.Shapes[1] as IAutoShape;

    // Προσθέστε ένα προσαρμοσμένο εφέ PathUser που ενεργοποιείται με το κλικ.
    IEffect fxUserPath = seqInter.AddEffect(
        ashp,
        EffectType.PathUser,
        EffectSubtype.None,
        EffectTriggerType.OnClick
    );

    // Ορίστε τη διαδρομή κίνησης για την κινούμενη εικόνα.
    IMotionEffect motionBhv = ((IMotionEffect)fxUserPath.Behaviors[0]);
    PointF[] pts = new PointF[1];

    // Εντολή για μετακίνηση κατά μήκος μιας γραμμής.
    pts[0] = new PointF(0.076f, 0.59f);
    motionBhv.Path.Add(
        MotionCommandPathType.LineTo,
        pts,
        MotionPathPointsType.Auto,
        true
    );

    // Μετακινηθείτε σε άλλο σημείο και προσθέστε εντολή.
    pts[0] = new PointF(-0.076f, -0.59f);
    motionBhv.Path.Add(
        MotionCommandPathType.LineTo,
        pts,
        MotionPathPointsType.Auto,
        false
    );

    // Τερματίστε τη διαδρομή.
    motionBhv.Path.Add(MotionCommandPathType.End, null, MotionPathPointsType.Auto, false);

    // Αποθηκεύστε την παρουσίαση με διαδραστικά κινούμενα σχέδια.
    pres.Save(outputDir + "ButtonAnimExample_out.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
}
```

Αυτός ο κώδικας δημιουργεί ένα διαδραστικό κουμπί που ενεργοποιεί μια προσαρμοσμένη διαδρομή κινούμενης εικόνας όταν κάνετε κλικ σε αυτό.

## Οδηγός Εφαρμογής
Δείτε πώς μπορείτε να ορίσετε προσαρμοσμένες αριθμημένες κουκκίδες σε διαφάνειες PowerPoint χρησιμοποιώντας το Aspose.Slides .NET.

### Προσθήκη προσαρμοσμένων αριθμημένων κουκκίδων σε μια διαφάνεια
#### Βήμα 1: Δημιουργήστε μια νέα παρουσίαση και προσθέστε ένα αυτόματο σχήμα
Δημιουργήστε μια παρουσία παρουσίασης και προσθέστε ένα ορθογώνιο σχήμα στην πρώτη διαφάνεια ως κοντέινερ κειμένου:
```csharp
var shape = presentation.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 200, 200, 400, 200);
```
#### Βήμα 2: Πρόσβαση στο Πλαίσιο Κειμένου
Πρόσβαση στο `ITextFrame` του δημιουργημένου σχήματος για τον χειρισμό περιεχομένου κειμένου:
```csharp
ITextFrame textFrame = shape.TextFrame;
```
#### Βήμα 3: Προσαρμόστε τις αριθμημένες κουκκίδες
Προσαρμόστε τα σημεία κουκκίδων ορίζοντας τους αρχικούς αριθμούς τους. Δείτε πώς για τρία διαφορετικά στοιχεία λίστας:
1. **Πρώτο στοιχείο λίστας** με έναν προσαρμοσμένο αριθμό εκκίνησης:
   ```csharp
   var paragraph1 = new Paragraph { Text = "bullet 2" };
   paragraph1.ParagraphFormat.Depth = 4; 
   paragraph1.ParagraphFormat.Bullet.NumberedBulletStartWith = 2;
   paragraph1.ParagraphFormat.Bullet.Type = BulletType.Numbered;
   textFrame.Paragraphs.Add(paragraph1);
   ```
2. **Δεύτερο στοιχείο λίστας** με διαφορετικό αριθμό εκκίνησης:
   ```csharp
   var paragraph2 = new Paragraph { Text = "bullet 3" };
   paragraph2.ParagraphFormat.Depth = 4;
   paragraph2.ParagraphFormat.Bullet.NumberedBulletStartWith = 3; 
   paragraph2.ParagraphFormat.Bullet.Type = BulletType.Numbered;
   textFrame.Paragraphs.Add(paragraph2);
   ```
3. **Τρίτο στοιχείο λίστας** με έναν άλλο προσαρμοσμένο αριθμό:
   ```csharp
   var paragraph5 = new Paragraph { Text = "bullet 7" };
   paragraph5.ParagraphFormat.Depth = 4;
   paragraph5.ParagraphFormat.Bullet.NumberedBulletStartWith = 7;
   paragraph5.ParagraphFormat.Bullet.Type = BulletType.Numbered;
   textFrame.Paragraphs.Add(paragraph5);
   ```
#### Βήμα 4: Αποθήκευση της παρουσίασης
Αποθηκεύστε την παρουσίασή σας σε έναν καθορισμένο κατάλογο:
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY"; // Αντικαταστήστε με την πραγματική σας διαδρομή
presentation.Save(Path.Combine(outputDir, "SetCustomBulletsNumber-slides.pptx"), SaveFormat.Pptx);
```

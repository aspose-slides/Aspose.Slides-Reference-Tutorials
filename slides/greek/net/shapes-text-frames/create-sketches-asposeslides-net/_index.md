## Οδηγός Εφαρμογής

Αφού όλα έχουν ρυθμιστεί, ας εφαρμόσουμε τη λειτουργία σκιτσαρισμένου σχήματος.

### Προσθήκη και τροποποίηση σχημάτων

#### Επισκόπηση

Σε αυτήν την ενότητα, θα προσθέσουμε ένα Αυτόματο Σχήμα τύπου ορθογωνίου σε μια διαφάνεια και θα διαμορφώσουμε τις ιδιότητές του για να δημιουργήσουμε ένα σκιτσαρισμένο εφέ.

**Προσθήκη ορθογωνίου σχήματος**

Ξεκινήστε δημιουργώντας μια νέα παρουσία παρουσίασης και προσθέτοντας ένα ορθογώνιο σχήμα:
```csharp
using Aspose.Slides;
using Aspose.Slides.Export;

string outPptxFile = Path.Combine("YOUR_OUTPUT_DIRECTORY", "SketchedShapes_out.pptx");
string outPngFile = Path.Combine("YOUR_OUTPUT_DIRECTORY", "SketchedShapes_out.png");

using (Presentation pres = new Presentation())
{
    // Προσθήκη ενός Αυτόματου Σχήματος τύπου Ορθογώνιου στην πρώτη διαφάνεια
    IAutoShape shape = pres.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 20, 20, 300, 150);
}
```

#### Ρύθμιση μορφής γεμίσματος

Για να του δώσετε μια σκιαγραφημένη εμφάνιση, αφαιρέστε τυχόν γέμισμα από το σχήμα:
```csharp
shape.FillFormat.FillType = FillType.NoFill;
```

### Εφαρμογή εφέ σκίτσων σε σχήματα

#### Επισκόπηση

Στη συνέχεια, μετατρέψτε το ορθογώνιο σε ένα σκίτσο ελεύθερου χεριού.

**Μετασχηματισμός σχήματος σε σκίτσο**

Χρησιμοποιήστε το `SketchFormat` ιδιότητα για την εφαρμογή ενός εφέ σκαριφήματος:
```csharp
// Μετασχηματισμός του σχήματος σε σκίτσο ελεύθερου χεριού (Scribble)
shape.LineFormat.SketchFormat.SketchType = LineSketchType.Scribble;
```

### Αποθήκευση παρουσιάσεων και εικόνων

Τέλος, αποθηκεύστε την εργασία σας ως αρχείο παρουσίασης και ως εικόνα.

**Αποθήκευση ως PPTX**
```csharp
// Αποθήκευση της παρουσίασης σε αρχείο PPTX
pres.Save(outPptxFile, SaveFormat.Pptx);
```

**Αποθήκευση ως εικόνα PNG**
```csharp
// Αποθήκευση της διαφάνειας ως αρχείο εικόνας σε μορφή PNG
pres.Slides[0].GetThumbnail(4/3f, 4/3f).Save(outPngFile, System.Drawing.Imaging.ImageFormat.Png);
```

## Οδηγός Εφαρμογής
### Χαρακτηριστικό 1: Σύνδεση σχημάτων χρησιμοποιώντας την τοποθεσία σύνδεσης
Αυτή η λειτουργία δείχνει τη σύνδεση μιας έλλειψης και ενός ορθογωνίου χρησιμοποιώντας μια σύνδεση σε ένα συγκεκριμένο δείκτη θέσης σύνδεσης.

#### Βήμα προς βήμα εφαρμογή:
**1. Ορίστε τη διαδρομή καταλόγου εγγράφου εξόδου**
Καθορίστε πού θα αποθηκευτεί η παρουσίαση εξόδου σας.
```csharp
string dataDir = @"YOUR_DOCUMENT_DIRECTORY/ShapeConnectionOutput.pptx";
```

**2. Δημιουργήστε ένα αντικείμενο παρουσίασης**
Δημιουργήστε ένα νέο `Presentation` αντικείμενο, που αντιπροσωπεύει το αρχείο PowerPoint σας:
```csharp
using (Presentation presentation = new Presentation())
{
    // Περισσότερος κώδικας εδώ...
}
```

**3. Αποκτήστε πρόσβαση στη συλλογή σχημάτων της πρώτης διαφάνειας**
Αποκτήστε πρόσβαση σε όλα τα σχήματα στην πρώτη διαφάνεια.
```csharp
IShapeCollection shapes = presentation.Slides[0].Shapes;
```

**4. Προσθέστε ένα σχήμα σύνδεσης**
Προσθέστε μια γραμμή σύνδεσης που θα συνδέει άλλα σχήματα μεταξύ τους:
```csharp
IConnector connector = shapes.AddConnector(ShapeType.BentConnector3, 0, 0, 10, 10);
```

**5. Προσθήκη σχημάτων (έλλειψη και ορθογώνιο)**
Εισαγάγετε μια έλλειψη και ένα ορθογώνιο στη συλλογή.
```csharp
IAutoShape ellipse = shapes.AddAutoShape(ShapeType.Ellipse, 0, 100, 100, 100);
IAutoShape rectangle = shapes.AddAutoShape(ShapeType.Rectangle, 100, 200, 100, 100);
```

**6. Συνδέστε τα σχήματα χρησιμοποιώντας τον σύνδεσμο**
Συνδέστε την έλλειψη και το ορθογώνιο χρησιμοποιώντας τον σύνδεσμο.
```csharp
connector.StartShapeConnectedTo = ellipse;
connector.EndShapeConnectedTo = rectangle;
```

**7. Καθορίστε ένα Ευρετήριο Θέσης Σύνδεσης στο Ellipse**
Επιλέξτε ένα συγκεκριμένο ευρετήριο τοποθεσίας σύνδεσης για ακριβείς συνδέσεις:
```csharp
uint wantedIndex = 6;

if (ellipse.ConnectionSiteCount > wantedIndex)
{
    connector.StartShapeConnectionSiteIndex = wantedIndex;
}
```

**8. Αποθήκευση της παρουσίασης**
Αποθηκεύστε την παρουσίασή σας για να διατηρηθούν οι αλλαγές.
```csharp
presentation.Save(dataDir, SaveFormat.Pptx);
```

### Λειτουργία 2: Προσθήκη σχημάτων σε διαφάνεια
Αυτή η λειτουργία δείχνει πώς να προσθέσετε διάφορα σχήματα όπως ελλείψεις και ορθογώνια απευθείας σε μια διαφάνεια.

#### Βήμα προς βήμα εφαρμογή:
**1. Ορίστε τη διαδρομή καταλόγου εγγράφου εξόδου**
Καθορίστε πού θα αποθηκευτεί το αρχείο εξόδου σας.
```csharp
string dataDir = @"YOUR_DOCUMENT_DIRECTORY/ShapeAdditionOutput.pptx";
```

**2. Δημιουργήστε ένα αντικείμενο παρουσίασης**
Ξεκινήστε δημιουργώντας ένα νέο `Presentation` αντικείμενο:
```csharp
using (Presentation presentation = new Presentation())
{
    // Περισσότερος κώδικας εδώ...
}
```

**3. Αποκτήστε πρόσβαση στη συλλογή σχημάτων της πρώτης διαφάνειας**
Αποκτήστε πρόσβαση σε όλα τα σχήματα στην πρώτη διαφάνεια.
```csharp
IShapeCollection shapes = presentation.Slides[0].Shapes;
```

**4. Προσθέστε ένα σχήμα έλλειψης**
Προσθήκη έλλειψης στη συλλογή:
```csharp
IAutoShape ellipse = shapes.AddAutoShape(ShapeType.Ellipse, 50, 150, 150, 100);
```

**5. Προσθέστε ένα ορθογώνιο σχήμα**
Ομοίως, προσθέστε ένα ορθογώνιο σχήμα.
```csharp
IAutoShape rectangle = shapes.AddAutoShape(ShapeType.Rectangle, 250, 350, 200, 150);
```

**6. Αποθήκευση της παρουσίασης**
Αποθηκεύστε την παρουσίασή σας για να οριστικοποιήσετε τις αλλαγές.
```csharp
presentation.Save(dataDir, SaveFormat.Pptx);
```

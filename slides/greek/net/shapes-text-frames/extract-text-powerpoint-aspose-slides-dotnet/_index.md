### Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, θα αναλύσουμε τον τρόπο εξαγωγής ακατέργαστου κειμένου από παρουσιάσεις PowerPoint.

#### Εξαγωγή ακατέργαστου κειμένου

**Επισκόπηση**Αυτή η λειτουργία σάς επιτρέπει να ανακτήσετε όλα τα μη τακτοποιημένα δεδομένα κειμένου—όπως κείμενα διαφανειών και σημειώσεις—από ένα αρχείο παρουσίασης.

1. **Ορίστε τον κατάλογο εγγράφων σας**
   ```csharp
   string documentDirectory = @"YOUR_DOCUMENT_DIRECTORY\";
   ```

2. **Δημιουργήστε την πλήρη διαδρομή προς το αρχείο παρουσίασής σας**
   ```csharp
   string presentationName = Path.Combine(documentDirectory, "PresentationText.pptx");
   ```

3. **Λήψη ακατέργαστου κειμένου χρησιμοποιώντας `PresentationFactory`**
   ```csharp
   IPresentationText presentationText = 
       PresentationFactory.Instance.GetPresentationText(presentationName, 
                                                       TextExtractionArrangingMode.Unarranged);
   ```

4. **Πρόσβαση και αποθήκευση συγκεκριμένων δεδομένων διαφανειών**
   - Ανάκτηση σχολίων από την πρώτη διαφάνεια:
     ```csharp
     string commentsSlide1 = presentationText.SlidesText[0].CommentsText;
     ```
   
   - Λήψη κειμένου από την πρώτη διαφάνεια:
     ```csharp
     string textSlide1 = presentationText.SlidesText[0].Text;
     ```

   - Πρόσβαση στις σημειώσεις από τη δεύτερη διαφάνεια:
     ```csharp
     string notesSlide2 = presentationText.SlidesText[1].NotesText;
     ```

**Συμβουλές αντιμετώπισης προβλημάτων**Βεβαιωθείτε ότι οι διαδρομές των αρχείων σας έχουν οριστεί σωστά και ελέγξτε για τυχόν προβλήματα δικαιωμάτων πρόσβασης σε αρχεία.

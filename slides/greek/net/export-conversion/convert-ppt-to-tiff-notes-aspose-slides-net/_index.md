## Οδηγός Εφαρμογής
Ας αναλύσουμε τη διαδικασία μετατροπής μιας παρουσίασης PowerPoint σε TIFF με σημειώσεις τοποθετημένες στο κάτω μέρος.

### Βήμα 1: Ορισμός καταλόγων
Ξεκινήστε ρυθμίζοντας καταλόγους για τα αρχεία εισόδου και εξόδου. Αυτό βοηθά στην αποτελεσματική οργάνωση των πόρων.
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Κατάλογος που περιέχει την παρουσίαση πηγής
string outputDir = "YOUR_OUTPUT_DIRECTORY"; // Κατάλογος όπου θα αποθηκευτεί το TIFF
```

### Βήμα 2: Φόρτωση της παρουσίασής σας
Δημιουργήστε μια παρουσία του `Presentation` αντικείμενο, που αντιπροσωπεύει το αρχείο PowerPoint σας.
```csharp
using (Presentation pres = new Presentation(dataDir + "/ConvertWithNote.pptx"))
{
    // Συνεχίστε με τα βήματα μετατροπής εδώ
}
```
Αυτό το βήμα αρχικοποιεί τα δεδομένα παρουσίασης για χειρισμό.

### Βήμα 3: Ρύθμιση παραμέτρων του TiffOptions
Για εξαγωγή σε μορφή TIFF, ρυθμίστε τις παραμέτρους `TiffOptions`Καθορίστε πώς θα πρέπει να τοποθετούνται οι νότες.
```csharp
// Δημιουργήστε μια παρουσία του TiffOptions για εξαγωγή σε μορφή TIFF
TiffOptions opts = new TiffOptions();

// Ορίστε τις επιλογές διάταξης για να τοποθετήσετε τις σημειώσεις στην κάτω πλήρη προβολή
INotesCommentsLayoutingOptions notesOptions = new NotesCommentsLayoutingOptions();
notesOptions.NotesPosition = NotesPositions.BottomFull;
opts.SlidesLayoutOptions = notesOptions;
```
Εδώ, `NotesPositions.BottomFull` διασφαλίζει ότι οι σημειώσεις σας είναι πλήρως ορατές κάτω από κάθε διαφάνεια.

### Βήμα 4: Αποθήκευση της παρουσίασης
Τέλος, αποθηκεύστε την παρουσίαση ως αρχείο TIFF χρησιμοποιώντας τις διαμορφωμένες επιλογές.
```csharp
// Αποθηκεύστε την παρουσίαση σε αρχείο TIFF με σημειώσεις
pres.Save(outputDir + "/TestNotes_out.tiff", SaveFormat.Tiff, opts);
```
Αυτή η μέθοδος μετατρέπει και αποθηκεύει την παρουσίασή σας στην επιθυμητή μορφή, διατηρώντας παράλληλα τις σχολιασμοί.

- Επαληθεύστε ότι το Aspose.Slides έχει εγκατασταθεί σωστά και αναφέρεται στο έργο σας.

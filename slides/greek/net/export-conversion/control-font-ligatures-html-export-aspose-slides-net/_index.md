Με το Aspose.Slides για .NET, αποκτάτε ακριβή έλεγχο σχετικά με την ενεργοποίηση ή απενεργοποίηση αυτών των συμπλεγμάτων κατά την εξαγωγή. Αυτός ο οδηγός θα σας καθοδηγήσει στα απαραίτητα βήματα για την αποτελεσματική διαχείριση αυτής της λειτουργίας.

**Τι θα μάθετε:**

## Οδηγός Εφαρμογής

### Δυνατότητα: Απενεργοποίηση συμπλεγμάτων γραμματοσειρών κατά την εξαγωγή

#### Επισκόπηση

Αυτή η ενότητα θα σας καθοδηγήσει στην απενεργοποίηση των συμπλεγμάτων γραμματοσειρών κατά την εξαγωγή μιας παρουσίασης ως HTML χρησιμοποιώντας το Aspose.Slides για .NET.

#### Βήμα προς βήμα εφαρμογή

**Βήμα 1: Ρύθμιση του έργου σας**
Δημιουργήστε ένα νέο έργο C# και βεβαιωθείτε ότι έχετε κάνει αναφορά στη βιβλιοθήκη Aspose.Slides. 

```csharp
using Aspose.Slides;
using Aspose.Slides.Export;
using System.IO;
```

**Βήμα 2: Ορισμός διαδρομών για την προέλευση και την έξοδο**
Προσδιορίστε πού βρίσκεται η παρουσίαση πηγής σας και ορίστε διαδρομές για τα αρχεία HTML εξόδου.

```csharp
string presentationName = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "TextLigatures.pptx");
string outPathEnabled = Path.Combine("YOUR_OUTPUT_DIRECTORY", "EnableLigatures-out.html");
string outPathDisabled = Path.Combine("YOUR_OUTPUT_DIRECTORY", "DisableLigatures-out.html");
```

**Βήμα 3: Φόρτωση της παρουσίασης**
Φορτώστε το αρχείο παρουσίασής σας χρησιμοποιώντας το Aspose.Slides.

```csharp
using (Presentation pres = new Presentation(presentationName))
{
    // Συνέχεια με τη διαμόρφωση επιλογών εξαγωγής
}
```

**Βήμα 4: Εξαγωγή με ενεργοποιημένα τα Ligatures**
Αποθηκεύστε την παρουσίαση σε μορφή HTML για να δείξετε την προεπιλεγμένη συμπεριφορά με ενεργοποιημένες τις δίγλωσσες συνδέσεις.

```csharp
pres.Save(outPathEnabled, SaveFormat.Html);
```

**Βήμα 5: Ρύθμιση παραμέτρων επιλογών για απενεργοποίηση των συμπλεγμάτων γραμματοσειρών**
Στήνω `HtmlOptions` και απενεργοποιήστε τους συνδέσμους γραμματοσειρών.

```csharp
HtmlOptions options = new HtmlOptions { DisableFontLigatures = true };
```

**Βήμα 6: Εξαγωγή με απενεργοποιημένα τα συμπλέγματα**
Εξαγάγετε ξανά την παρουσίαση, αυτή τη φορά χρησιμοποιώντας τις διαμορφωμένες επιλογές.

```csharp
pres.Save(outPathDisabled, SaveFormat.Html, options);
```

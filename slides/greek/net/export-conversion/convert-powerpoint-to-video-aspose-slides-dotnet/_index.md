## Οδηγός Εφαρμογής

### Χαρακτηριστικό: Μετατροπή παρουσίασης σε βίντεο

Αυτή η λειτουργία έχει σχεδιαστεί για την εξαγωγή καρέ από μια παρουσίαση PowerPoint και τη μεταγλώττισή τους σε μορφή βίντεο.

#### Βήμα 1: Ορίστε τη διαδρομή

Ξεκινήστε καθορίζοντας τη διαδρομή προς το αρχείο παρουσίασής σας:
```csharp
string presentationName = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "SimpleAnimations.pptx");
```

#### Βήμα 2: Ορισμός ρυθμού καρέ

Αποφασίστε για τον ρυθμό καρέ για το βίντεο εξόδου σας, ο οποίος επηρεάζει την ποιότητα και το μέγεθος του αρχείου:
```csharp
const int FPS = 30;
```

#### Βήμα 3: Αρχικοποίηση αντικειμένου παρουσίασης

Φορτώστε την παρουσίασή σας χρησιμοποιώντας το Aspose.Slides:
```csharp
using (Presentation presentation = new Presentation(presentationName))
{
    // Περαιτέρω επεξεργασία εδώ...
}
```

#### Βήμα 4: Χειρισμός κινούμενων εικόνων

Χρησιμοποιήστε το `PresentationAnimationsGenerator` για τη διαχείριση κινήσεων διαφανειών.
```csharp
using (var animationsGenerator = new PresentationAnimationsGenerator(presentation))
{
    // Λογική χειρισμού κινούμενων εικόνων...
}
```

#### Βήμα 5: Δημιουργία πλαισίων

Δημιουργήστε μια παρουσία προγράμματος αναπαραγωγής για να εξάγετε καρέ με τον καθορισμένο ρυθμό καρέ και να χειρίζεστε κάθε καρέ χρησιμοποιώντας έναν χειριστή συμβάντων.
```csharp
using (var player = new PresentationPlayer(animationsGenerator, FPS))
{
    player.FrameTick += (sender, args) =>
    {
        // Αποθηκεύστε κάθε καρέ ως PNG για τη δημιουργία βίντεο.
        args.GetFrame().Save(Path.Combine("YOUR_OUTPUT_DIRECTORY", $"frame_{sender.FrameIndex}.png"));
    };

    animationsGenerator.Run(presentation.Slides);
}
```

### Δυνατότητα 1: Δημιουργία και αποθήκευση παρουσίασης PowerPoint σε ροή

#### Επισκόπηση
Αυτή η λειτουργία εστιάζει στη δημιουργία μιας απλής παρουσίασης PowerPoint, στην εισαγωγή κειμένου και στην απευθείας αποθήκευσή της ως ροή για περαιτέρω χειρισμό ή αποθήκευση.

##### Οδηγός βήμα προς βήμα

**Δημιουργία νέας παρουσίασης**
Ξεκινήστε δημιουργώντας μια παρουσία του `Presentation` κλάση, που αντιπροσωπεύει το αρχείο PowerPoint σας:

```csharp
using Aspose.Slides;

namespace PresentationToStreamExample
{
    public class SavePresentationToStream
    {
        public static void Main()
        {
            string dataDir = @"YOUR_DOCUMENT_DIRECTORY"; // Καθορίστε εδώ τη διαδρομή του καταλόγου σας

            using (Presentation presentation = new Presentation())
            {
                // Συνέχεια με τον χειρισμό των διαφανειών...
```

**Προσθήκη σχήματος κειμένου στην πρώτη διαφάνεια**
Προσθέστε ένα αυτόματο σχήμα ορθογωνίου τύπου και εισαγάγετε κείμενο σε αυτό:

```csharp
                IAutoShape shape = presentation.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 200, 200, 200, 200);
                shape.TextFrame.Text = "This demo shows how to Create PowerPoint file and save it to Stream.";
```

**Αποθήκευση της παρουσίασης ως ροής**
Ορίστε μια ροή όπου θα αποθηκευτεί η παρουσίασή σας:

```csharp
                using (FileStream toStream = new FileStream(dataDir + "Save_As_Stream_out.pptx", FileMode.Create))
                {
                    // Αποθηκεύστε την παρουσίαση στη ροή.
                    presentation.Save(toStream, Aspose.Slides.Export.SaveFormat.Pptx);
                }
            }
        }
    }
}
```

**Εξήγηση:**
- `Presentation` χειρίζεται αρχεία PowerPoint στη μνήμη.
- Το ορθογώνιο σχήμα προστίθεται στην πρώτη διαφάνεια με καθορισμένες διαστάσεις και συντεταγμένες.
- Ένα FileStream χρησιμοποιείται για την αποθήκευση της παρουσίασης σε μορφή PPTX, επιτρέποντας ευέλικτο χειρισμό δεδομένων.

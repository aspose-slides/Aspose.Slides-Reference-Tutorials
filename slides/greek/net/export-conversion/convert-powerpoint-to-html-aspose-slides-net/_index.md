## Οδηγός Εφαρμογής

Ας αναλύσουμε πώς να μετατρέψετε PPT σε HTML με ενσωματωμένες γραμματοσειρές χρησιμοποιώντας το Aspose.Slides .NET.

### Μετατροπή παρουσίασης σε HTML με ενσωματωμένες γραμματοσειρές

#### Επισκόπηση
Αυτή η λειτουργία εστιάζει στη μετατροπή μιας παρουσίασης PowerPoint σε έγγραφο HTML, ενσωματώνοντας όλες τις γραμματοσειρές που χρησιμοποιούνται στις διαφάνειες για να διατηρηθεί η ακεραιότητα του σχεδιασμού σε διαφορετικές πλατφόρμες.

#### Οδηγός βήμα προς βήμα

1. **Φόρτωση της παρουσίασης:**
   Ξεκινήστε φορτώνοντας το υπάρχον αρχείο PPT χρησιμοποιώντας το Aspose.Slides. Βεβαιωθείτε ότι έχετε καθορίσει τη σωστή διαδρομή προς το αρχείο παρουσίασής σας.
   
   ```csharp
   string dataDir = "YOUR_DOCUMENT_DIRECTORY";
   using (Presentation pres = new Presentation(dataDir + "/presentation.pptx"))
   {
       // Περαιτέρω βήματα θα εκτελεστούν εντός αυτού του μπλοκ
   }
   ```

2. **Ρύθμιση παραμέτρων ενσωμάτωσης γραμματοσειρών:**
   Χρησιμοποιήστε το `EmbedAllFontsHtmlController` για να διαχειριστείτε τις επιλογές ενσωμάτωσης γραμματοσειρών. Στο παράδειγμά μας, δεν εξαιρούμε καμία γραμματοσειρά.
   
   ```csharp
   string[] fontNameExcludeList = { };
   EmbedAllFontsHtmlController embedFontsController = new EmbedAllFontsHtmlController(fontNameExcludeList);
   ```

3. **Ορισμός επιλογών HTML:**
   Δημιουργήστε προσαρμοσμένες επιλογές HTML για να χρησιμοποιήσετε τον ελεγκτή ενσωμάτωσης γραμματοσειρών, διασφαλίζοντας ότι όλες οι γραμματοσειρές είναι ενσωματωμένες στην έξοδο.
   
   ```csharp
   HtmlOptions htmlOptionsEmbed = new HtmlOptions
   {
       HtmlFormatter = HtmlFormatter.CreateCustomFormatter(embedFontsController)
   };
   ```

4. **Αποθήκευση ως HTML:**
   Τέλος, αποθηκεύστε την παρουσίασή σας ως αρχείο HTML χρησιμοποιώντας τις καθορισμένες επιλογές.
   
   ```csharp
   string outputDir = "YOUR_OUTPUT_DIRECTORY";
   pres.Save(outputDir + "/pres.html", SaveFormat.Html, htmlOptionsEmbed);
   ```

#### Βασικές επιλογές διαμόρφωσης
- **Λίστα εξαιρέσεων ονόματος γραμματοσειράς:** Καθορίστε τις γραμματοσειρές που δεν θέλετε να ενσωματώσετε. Αφήστε το κενό για να ενσωματώσετε όλες τις γραμματοσειρές.
- **HtmlFormatter:** Προσαρμόζει τον τρόπο μορφοποίησης της HTML κατά τη μετατροπή.

### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι οι διαδρομές και για τους καταλόγους εισόδου και εξόδου έχουν οριστεί σωστά για να αποφύγετε σφάλματα "δεν βρέθηκε αρχείο".
- Επαληθεύστε ότι η εφαρμογή σας διαθέτει τα απαραίτητα δικαιώματα για ανάγνωση και εγγραφή σε αυτούς τους καταλόγους.

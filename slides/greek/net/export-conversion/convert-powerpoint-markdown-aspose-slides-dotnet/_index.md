### Μετατροπή παρουσίασης σε Markdown

Αυτή η λειτουργία απεικονίζει τον μετασχηματισμό διαφανειών PowerPoint σε αρχεία Markdown διατηρώντας παράλληλα τη δομή τους και τυχόν περιλαμβανόμενα μέσα.

#### Αρχικοποίηση του αντικειμένου παρουσίασης

Ξεκινήστε φορτώνοντας το αρχείο παρουσίασής σας:

```csharp
using System.IO;
using Aspose.Slides;

string presentationName = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "PresentationDemo.pptx");
// Φόρτωση της παρουσίασης με το Aspose.Slides
using (Presentation pres = new Presentation(presentationName))
{
    // Ο κώδικας συνεχίζεται...
}
```

#### Ρύθμιση παραμέτρων επιλογών μετατροπής Markdown

Ρυθμίστε τις προτιμήσεις μετατροπής χρησιμοποιώντας `MarkdownSaveOptions`:

```csharp
string outPath = "YOUR_OUTPUT_DIRECTORY";  // Ορισμός καταλόγου εξόδου για αρχεία Markdown

// Δημιουργία και ρύθμιση παραμέτρων του MarkdownSaveOptions
MarkdownSaveOptions mdOptions = new MarkdownSaveOptions();
mdOptions.ExportType = MarkdownExportType.Visual; // Επιλέξτε τύπο εξαγωγής οπτικών στοιχείων
mdOptions.ImagesSaveFolderName = "md-images";    // Καθορισμός φακέλου για εικόνες
mdOptions.BasePath = outPath;                     // Ορίστε τη βασική διαδρομή

// Αποθήκευση της παρουσίασης ως αρχείο Markdown
pres.Save(Path.Combine(outPath, "pres.md"), SaveFormat.Md, mdOptions);
```

**Επεξήγηση βασικών επιλογών:**
- `ExportType`: Καθορίζει τον τρόπο εξαγωγής του περιεχομένου. Ο τύπος απεικόνισης περιλαμβάνει όλα τα στοιχεία στην αρχική τους διάταξη.
- `ImagesSaveFolderName` και `BasePath`: Ορίστε πού θα αποθηκεύονται οι εικόνες που εξάγονται από την παρουσίαση.

### Συμβουλές αντιμετώπισης προβλημάτων

- Βεβαιωθείτε ότι ο κατάλογος εξόδου υπάρχει πριν από την αποθήκευση αρχείων για να αποφύγετε εξαιρέσεις.
- Βεβαιωθείτε ότι η διαδρομή φακέλου για τις εικόνες είναι σωστή και προσβάσιμη, εάν δεν αποδίδονται σωστά.

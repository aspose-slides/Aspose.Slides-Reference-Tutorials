## Οδηγός Εφαρμογής

Ακολουθήστε αυτά τα βήματα για να αποθηκεύσετε μια παρουσίαση PPTX ως έγγραφο HTML με εξωτερικές εικόνες.

### Ρύθμιση παραμέτρων επιλογών Html5 για εξωτερικές εικόνες

**Επισκόπηση:**
Ρυθμίζοντας `EmbedImages` να κάνω ψευδή εισαγωγή `Html5Options`, δίνετε εντολή στο Aspose.Slides να μην ενσωματώνει εικόνες μέσα στο αρχείο HTML, χρησιμοποιώντας έτσι εξωτερικές διαδρομές εικόνας.

**Βήματα Υλοποίησης:**

#### Βήμα 1: Ορισμός διαδρομών για την πηγή και την έξοδο
Ορίστε διαδρομές για την παρουσίαση πηγής και τον κατάλογο εξόδου:
```csharp
string presentationName = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "PresentationDemo.pptx");
string outFilePath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "HTMLConversion");
```

#### Βήμα 2: Φόρτωση της παρουσίασης
Χρησιμοποιήστε το `Presentation` κλάση για να φορτώσετε το αρχείο PPTX σας:
```csharp
using (Presentation pres = new Presentation(presentationName))
{
    // Ο κώδικας συνεχίζεται εδώ...
}
```

#### Βήμα 3: Ρύθμιση παραμέτρων Html5Options
Δημιουργήστε μια παρουσία του `Html5Options`, ρύθμιση `EmbedImages` σε false και καθορίζοντας τον κατάλογο εξόδου για τις εικόνες:
```csharp
Html5Options options = new Html5Options()
{
    EmbedImages = false,
    OutputPath = "YOUR_OUTPUT_DIRECTORY"
};
```

#### Βήμα 4: Βεβαιωθείτε ότι υπάρχει κατάλογος εξόδου
Ελέγξτε αν υπάρχει ο κατάλογος εξόδου και δημιουργήστε τον, εάν είναι απαραίτητο:
```csharp
if (!Directory.Exists(outFilePath))
{
    Directory.CreateDirectory(outFilePath);
}
```

#### Βήμα 5: Αποθήκευση ως HTML με εξωτερικές εικόνες
Αποθηκεύστε την παρουσίαση χρησιμοποιώντας `SaveFormat.Html5` μαζί με τις διαμορφωμένες επιλογές σας. Αυτό έχει ως αποτέλεσμα ένα έγγραφο HTML και ξεχωριστά αρχεία εικόνας στον καθορισμένο κατάλογο εξόδου:
```csharp
pres.Save(Path.Combine(outFilePath, "pres.html"), SaveFormat.Html5, options);
```

### Συμβουλές αντιμετώπισης προβλημάτων

- **Λείπουν εικόνες:** Εξασφαλίζω `EmbedImages` έχει οριστεί σε false.
- **Προβλήματα πρόσβασης στον κατάλογο:** Ελέγξτε τα δικαιώματα αρχείων για τον κατάλογο εξόδου.

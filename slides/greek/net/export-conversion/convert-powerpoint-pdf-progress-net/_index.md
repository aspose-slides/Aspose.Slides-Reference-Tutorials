```csharp
            {
                // Τα βήματα διαμόρφωσης ακολουθούν...
```

#### Βήμα 2: Ρύθμιση παραμέτρων επιλογών PDF

Ρυθμίστε το `PdfOptions` για να συμπεριλάβει μια επανάκληση προόδου, παρέχοντας ενημερώσεις κατά τη μετατροπή.

```csharp
                ISaveOptions saveOptions = new PdfOptions();
                saveOptions.ProgressCallback = new ExportProgressHandler();

                presentation.Save(dataDir + "ConvertToPDF.pdf", SaveFormat.Pdf, saveOptions);
            }
        }
    }
}
```

#### Επεξήγηση παραμέτρων
- **`PdfOptions`**Σας επιτρέπει να καθορίσετε διάφορες ρυθμίσεις για τη μετατροπή PDF.
- **`ExportProgressHandler`**: Μια προσαρμοσμένη κλάση που υλοποιεί `IProgressCallback`, παρέχοντας ενημερώσεις σε πραγματικό χρόνο.

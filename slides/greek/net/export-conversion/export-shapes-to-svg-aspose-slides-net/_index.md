## Οδηγός Εφαρμογής

Εστιάστε στην εξαγωγή σχημάτων σε μορφή SVG με αυτόν τον λεπτομερή οδηγό:

### Εξαγωγή σχήματος σε SVG

#### Επισκόπηση
Εξαγωγή σχημάτων από οποιαδήποτε διαφάνεια PowerPoint σε αρχείο SVG, χρήσιμο για την ενσωμάτωση διανυσματικών γραφικών σε εφαρμογές web ή συστήματα λογισμικού που απαιτούν κλιμακούμενες μορφές.

#### Οδηγός βήμα προς βήμα
**1. Ορίστε διαδρομές για αρχεία εισόδου και εξόδου**
Ορίστε καταλόγους για αρχεία εισόδου και εξόδου:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Κατάλογος που περιέχει το αρχείο PowerPoint
string outSvgFileName = "YOUR_OUTPUT_DIRECTORY/SingleShape.svg"; // Διαδρομή αρχείου SVG εξόδου
```

**2. Φορτώστε την παρουσίασή σας**
Φόρτωση παρουσίασης χρησιμοποιώντας το Aspose.Slides:

```csharp
using (Presentation pres = new Presentation(dataDir + "/TestExportShapeToSvg.pptx"))
{
    // Πρόσβαση στην πρώτη διαφάνεια και το πρώτο της σχήμα
    var slide = pres.Slides[0];
    var shape = slide.Shapes[0];

    // Δημιουργήστε ένα FileStream για το αρχείο SVG εξόδου
    using (Stream stream = new FileStream(outSvgFileName, FileMode.Create, FileAccess.Write))
    {
        // Εξαγωγή του σχήματος σε μορφή SVG
        shape.WriteAsSvg(stream);
    }
}
```

**Εξήγηση:**
- `dataDir`: Κατάλογος που περιέχει το αρχείο PowerPoint σας.
- `outSvgFileName`: Διαδρομή όπου θα αποθηκευτεί το εξαγόμενο SVG.
- **`Presentation` Αντικείμενο**: Αντιπροσωπεύει το έγγραφο του PowerPoint.
- **`Slide.Shapes[0]`**: Πρόσβαση στο πρώτο σχήμα της πρώτης διαφάνειας για εξαγωγή.

### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι η διαδρομή του αρχείου εισαγωγής είναι σωστή και προσβάσιμη.
- Ελέγξτε τα δικαιώματα αρχείου για να επιβεβαιώσετε την πρόσβαση εγγραφής στον κατάλογο εξόδου.
- Βεβαιωθείτε ότι το αρχείο PowerPoint δεν είναι κατεστραμμένο ανοίγοντάς το στο Microsoft PowerPoint.

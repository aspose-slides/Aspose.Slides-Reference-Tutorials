## Οδηγός Εφαρμογής

### Αποθήκευση παρουσίασης ως HTML με προσαρμοσμένες γραμματοσειρές

#### Επισκόπηση
Αυτή η λειτουργία δείχνει πώς να μετατρέψετε μια παρουσίαση PowerPoint σε HTML, καθορίζοντας διαφορετικές προεπιλεγμένες κανονικές γραμματοσειρές. Αυτό διασφαλίζει συνεπή τυπογραφία σε διάφορες πλατφόρμες.

#### Βήμα προς βήμα εφαρμογή

**1. Ορίστε διαδρομές εγγράφων:**
Ξεκινήστε ορίζοντας τις διαδρομές καταλόγου για το αρχείο PPT πηγής και την HTML εξόδου.
```csharp
string dataDir = "/path/to/your/documents";
string outPath = "/output/directory";
```

**2. Φορτώστε την παρουσίαση:**
Χρήση `Presentation` τάξη για να φορτώσετε το αρχείο PowerPoint σας.
```csharp
using (Presentation pres = new Presentation(dataDir + "/DefaultFonts.pptx"))
{
    // Τα επόμενα βήματα θα ακολουθήσουν εδώ...
}
```
*Γιατί;* Η φόρτωση της παρουσίασης είναι απαραίτητη, καθώς προετοιμάζει το έγγραφό σας για περαιτέρω χειρισμό.

**3. Δημιουργία επιλογών HTML:**
Αρχικοποίηση `HtmlOptions` για να καθορίσετε πώς θέλετε να μετατραπεί το PPT σας.
```csharp
HtmlOptions htmlOpts = new HtmlOptions();
```

**4. Ορισμός προεπιλεγμένης κανονικής γραμματοσειράς:**
Προσαρμόστε την προεπιλεγμένη γραμματοσειρά που χρησιμοποιείται στη διαδικασία μετατροπής.
```csharp
htmlOpts.DefaultRegularFont = "Arial Black";
pres.Save(outPath + "/Presentation-out-ArialBlack.html", SaveFormat.Html, htmlOpts);
```
*Γιατί;* Ο ορισμός μιας προσαρμοσμένης γραμματοσειράς διασφαλίζει ότι η παρουσίασή σας διατηρεί την οπτική της συνέπεια όταν προβάλλεται ως HTML.

#### Συμβουλές αντιμετώπισης προβλημάτων:
- **Σφάλματα διαδρομής αρχείου:** Ελέγξτε ξανά τις διαδρομές καταλόγου σας για τυπογραφικά λάθη.
- **Λείπουν γραμματοσειρές:** Βεβαιωθείτε ότι οι καθορισμένες γραμματοσειρές είναι διαθέσιμες στο σύστημά σας.

## Οδηγός Εφαρμογής
Ας εμβαθύνουμε στη μετατροπή παρουσιάσεων PowerPoint διατηρώντας παράλληλα τις γραμματοσειρές χρησιμοποιώντας το Aspose.Slides για .NET. Θα το αναλύσουμε βήμα προς βήμα:

### Επισκόπηση χαρακτηριστικών
Αυτή η λειτουργία επιτρέπει τη μετατροπή αρχείων PPTX σε έγγραφα HTML, διατηρώντας τα αρχικά στυλ γραμματοσειράς όπως εμφανίζονται στην παρουσίαση.

#### Βήμα 1: Φόρτωση της παρουσίασής σας
Ξεκινήστε φορτώνοντας το αρχείο PowerPoint σε ένα `Presentation` αντικείμενο. Αυτό είναι κρίσιμο για την πρόσβαση και τον χειρισμό των διαφανειών.

```csharp
using Aspose.Slides;
using Aspose.Slides.Export;

string dataDir = @"YOUR_DOCUMENT_DIRECTORY";
using (Presentation pres = new Presentation(dataDir + "input.pptx"))
{
    // Περαιτέρω επεξεργασία εδώ
}
```

**Εξήγηση:** Ξεκινάμε δημιουργώντας ένα `Presentation` αντικείμενο, το οποίο μας επιτρέπει να αλληλεπιδρούμε με τις διαφάνειες στο αρχείο PowerPoint σας.

#### Βήμα 2: Ρύθμιση παραμέτρων γραμματοσειράς
Προαιρετικά, καθορίστε τυχόν γραμματοσειρές που θέλετε να εξαιρέσετε από την ενσωμάτωση εντός του HTML. Αυτό μπορεί να βελτιστοποιήσει τους χρόνους φόρτωσης και να μειώσει το μέγεθος του αρχείου.

```csharp
string[] fontNameExcludeList = { "Calibri" };
```

**Εξήγηση:** Ο `fontNameExcludeList` Ο πίνακας ορίζει ποιες γραμματοσειρές δεν πρέπει να ενσωματωθούν στο τελικό έγγραφο HTML, βοηθώντας στην αποτελεσματική διαχείριση της χρήσης πόρων.

#### Βήμα 3: Μετατροπή σε HTML
Στη συνέχεια, μετατρέψτε τις διαφάνειες της παρουσίασής σας σε μορφή HTML. Μπορείτε να προσαρμόσετε περαιτέρω αυτήν τη διαδικασία καθορίζοντας πρόσθετες ρυθμίσεις, εάν χρειάζεται.

```csharp
pres.Save(outputDir + "output.html", SaveFormat.Html5);
```

**Εξήγηση:** Ο `Save` η μέθοδος εξάγει την παρουσίαση ως έγγραφο HTML, με `Html5` διασφαλίζοντας τη συμβατότητα με όλα τα σύγχρονα προγράμματα περιήγησης ιστού.

### Συμβουλές αντιμετώπισης προβλημάτων:
- Εξασφαλίστε διαδρομές σε `dataDir` και `outputDir` είναι σωστά.
- Ελέγξτε αν υπάρχουν διαθέσιμες εξαιρούμενες γραμματοσειρές στις συσκευές-στόχους για να αποφύγετε την απώλεια στυλ.

## Οδηγός Εφαρμογής

Θα εξερευνήσουμε δύο κύριες λειτουργίες: τη ρύθμιση μεγέθους διαφάνειας και τη διαμόρφωση επιλογών εξαγωγής PDF.

### Ορισμός μεγέθους διαφάνειας παρουσίασης σε A4

#### Επισκόπηση

Αυτή η λειτουργία διασφαλίζει ότι οι διαφάνειές σας ταιριάζουν τέλεια σε ένα φύλλο A4, διατηρώντας την αναλογία διαστάσεων χωρίς περικοπή ή παραμόρφωση.

**Βήματα Υλοποίησης:**
1. **Δημιουργία αντικειμένου παρουσίασης:** Δημιουργήστε ένα νέο αντικείμενο παρουσίασης.
    ```csharp
    Presentation presentation = new Presentation();
    ```
2. **Ορισμός τύπου μεγέθους διαφάνειας και κλίμακας:** Χρησιμοποιήστε το `SetSize` μέθοδος για να προσαρμόσετε το μέγεθος της διαφάνειάς σας σε μορφή A4, διασφαλίζοντας ότι εφαρμόζει σωστά.
    ```csharp
    // Ορίστε το SlideSize.Type σε μέγεθος χαρτιού A4 με τον τύπο κλίμακας EnsureFit
    presentation.SlideSize.SetSize(SlideSizeType.A4Paper, SlideSizeScaleType.EnsureFit);
    ```
3. **Αποθήκευση της παρουσίασης:** Αποθηκεύστε το αρχείο παρουσίασής σας σε μορφή PPTX.
    ```csharp
    // Αποθήκευση της παρουσίασης σε δίσκο
    presentation.Save("YOUR_OUTPUT_DIRECTORY/SetSlideSize_out.pptx", SaveFormat.Pptx);
    ```

**Βασικές επιλογές διαμόρφωσης:**
- `SlideSizeType.A4Paper`: Καθορίζει το μέγεθος χαρτιού A4.
- `SlideSizeScaleType.EnsureFit`Εξασφαλίζει ότι το περιεχόμενο ταιριάζει εντός των ορίων της διαφάνειας.

### Ρύθμιση παραμέτρων επιλογών εξαγωγής PDF

#### Επισκόπηση
Προσαρμόστε τις ρυθμίσεις εξαγωγής PDF για να επιτύχετε αποτελέσματα υψηλής ανάλυσης, καθιστώντας τα ιδανικά για εκτύπωση ή κοινή χρήση.

**Βήματα Υλοποίησης:**
1. **Φόρτωση υπάρχουσας παρουσίασης:** Αρχικοποίηση ενός αντικειμένου παρουσίασης από ένα υπάρχον αρχείο.
    ```csharp
    Presentation presentation = new Presentation("YOUR_INPUT_FILE.pptx");
    ```
2. **Δημιουργία και ρύθμιση παραμέτρων PdfOptions:** Δημιουργήστε ένα στιγμιότυπο του `PdfOptions` κλάση για να ορίσετε τις ρυθμίσεις PDF.
    ```csharp
    // Ρύθμιση επιλογών PDF για υψηλή ανάλυση
    PdfOptions opts = new PdfOptions();
    opts.SufficientResolution = 600;
    ```
3. **Εξαγωγή ως PDF με επιλογές:** Αποθηκεύστε την παρουσίαση ως PDF, εφαρμόζοντας τις καθορισμένες επιλογές εξαγωγής.
    ```csharp
    // Εξαγωγή σε PDF με τις καθορισμένες ρυθμίσεις
    presentation.Save("YOUR_OUTPUT_DIRECTORY/SetPDFPageSize_out.pdf", SaveFormat.Pdf, opts);
    ```

**Βασικές επιλογές διαμόρφωσης:**
- `SufficientResolution`Ελέγχει την ανάλυση του εξαγόμενου PDF. Μια υψηλότερη τιμή έχει ως αποτέλεσμα καλύτερη ποιότητα.

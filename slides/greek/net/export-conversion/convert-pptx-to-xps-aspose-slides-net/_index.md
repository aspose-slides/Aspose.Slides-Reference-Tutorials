## Οδηγός Εφαρμογής
Σε αυτήν την ενότητα, θα αναλύσουμε τα βήματα για τη μετατροπή μιας παρουσίασης PowerPoint σε έγγραφο XPS χρησιμοποιώντας συγκεκριμένες επιλογές.

### 1. Δημιουργία του Περιβάλλοντος
Ξεκινήστε καθορίζοντας τους καταλόγους εισόδου και εξόδου για τα αρχεία:
```csharp
string dataDir = @"YOUR_DOCUMENT_DIRECTORY";
string outputDir = @"YOUR_OUTPUT_DIRECTORY";
```

### 2. Φόρτωση του αρχείου παρουσίασης
Φορτώστε το αρχείο παρουσίασής σας σε ένα `Presentation` αντικείμενο:
```csharp
using (Presentation pres = new Presentation(dataDir + "Convert_XPS_Options.pptx"))
{
    // Περαιτέρω επεξεργασία θα γίνει εδώ.
}
```

### 3. Ρύθμιση παραμέτρων επιλογών μετατροπής
Δημιουργήστε μια παρουσία του `XpsOptions` για να καθορίσετε τις ρυθμίσεις μετατροπής:
```csharp
// Δημιουργήστε ένα αντικείμενο XpsOptions για τον ορισμό παραμέτρων μετατροπής
XpsOptions opts = new XpsOptions();
opts.SaveMetafilesAsPng = true; // Διασφαλίζει ότι τα μετααρχεία αποθηκεύονται ως PNG, διατηρώντας την ποιότητα.
```

### 4. Μετατροπή και αποθήκευση της παρουσίασης
Χρησιμοποιήστε το `Save` μέθοδος μετατροπής της παρουσίασης:
```csharp
pres.Save(outputDir + "XPS_With_Options_out.xps", SaveFormat.Xps, opts);
```
**Εξήγηση**Αυτός ο κώδικας μετατρέπει το αρχείο PPTX σε έγγραφο XPS, ενώ αποθηκεύει τα μετααρχεία ως εικόνες PNG για καλύτερη ποιότητα.

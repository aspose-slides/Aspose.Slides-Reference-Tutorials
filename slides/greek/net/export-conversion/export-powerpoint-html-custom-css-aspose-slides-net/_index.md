## Οδηγός Εφαρμογής
### Εξαγωγή PowerPoint σε HTML με προσαρμοσμένο CSS
Μετατρέψτε παρουσιάσεις σε αρχεία HTML με στυλ χρησιμοποιώντας προσαρμοσμένο CSS.

#### Βήμα 1: Ορισμός καταλόγων και φόρτωση παρουσίασης
Ρυθμίστε τους καταλόγους εγγράφων και εξόδου και, στη συνέχεια, φορτώστε την παρουσίαση:
```csharp
using Aspose.Slides;
using Aspose.Slides.Export;

string dataDir = "YOUR_DOCUMENT_DIRECTORY";  // Θέση αρχείου προέλευσης.
string outputDir = "YOUR_OUTPUT_DIRECTORY";    // Αποθήκευση τοποθεσίας HTML.

// Φόρτωση του αρχείου PowerPoint
using (Presentation pres = new Presentation(dataDir + "/pres.pptx"))
{
    // Η υλοποίηση συνεχίζεται εδώ...
}
```

#### Βήμα 2: Εφαρμογή προσαρμοσμένου CSS με ελεγκτή
Δημιουργήστε έναν προσαρμοσμένο ελεγκτή κεφαλίδας και γραμματοσειρών για τη διαχείριση στυλ:
```csharp
CustomHeaderAndFontsController htmlController = new CustomHeaderAndFontsController(outputDir + "/styles.css");
```
Αυτό το βήμα ρυθμίζει την εισαγωγή προσαρμοσμένου CSS στην εξαγόμενη HTML.

#### Βήμα 3: Ρύθμιση παραμέτρων επιλογών εξαγωγής
Ορίστε επιλογές για εξαγωγή ως HTML χρησιμοποιώντας το Aspose.Slides:
```csharp
HtmlOptions options = new HtmlOptions
{
    HtmlFormatter = HtmlFormatter.CreateCustomFormatter(htmlController),  // Εφαρμόστε τον προσαρμοσμένο μορφοποιητή σας εδώ.
};
```
Ο `HtmlFormatter` επιτρέπει την προσαρμογή της απόδοσης των διαφανειών σε μορφή HTML.

#### Βήμα 4: Αποθήκευση ως HTML
Αποθηκεύστε την παρουσίαση με τις καθορισμένες επιλογές:
```csharp
pres.Save(outputDir + "/pres.html", SaveFormat.Html, options);
```
Αυτό αποθηκεύει την παρουσίαση σε ένα αρχείο HTML στην επιθυμητή θέση, εφαρμόζοντας όλα τα προσαρμοσμένα στυλ που έχουν οριστεί.

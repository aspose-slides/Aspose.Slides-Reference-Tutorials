## Οδηγός Εφαρμογής

Τώρα που έχουμε ρυθμίσει τα πάντα, ας προχωρήσουμε στην υλοποίηση των λειτουργιών.

### Αντικατάσταση γραμματοσειράς

**Επισκόπηση**
Η αντικατάσταση γραμματοσειράς είναι απαραίτητη όταν μια γραμματοσειρά πηγής δεν είναι διαθέσιμη στο σύστημα-στόχο. Με το Aspose.Slides, μπορείτε να ορίσετε κανόνες για την απρόσκοπτη αντικατάσταση γραμματοσειρών κατά την απόδοση της παρουσίασης.

#### Οδηγός βήμα προς βήμα
1. **Φόρτωση της παρουσίασής σας**
   Ξεκινήστε φορτώνοντας το αρχείο παρουσίασής σας σε ένα `Presentation` αντικείμενο:
   
   ```csharp
   string dataDir = "YOUR_DOCUMENT_DIRECTORY";
   Presentation presentation = new Presentation(dataDir + "/Fonts.pptx");
   ```

2. **Ορισμός γραμματοσειρών για αντικατάσταση**
   Καθορίστε τη γραμματοσειρά προέλευσης που θα αντικατασταθεί και τη γραμματοσειρά προορισμού:
   
   ```csharp
   IFontData sourceFont = new FontData("SomeRareFont");
   IFontData destFont = new FontData("Arial");
   ```

3. **Δημιουργία κανόνα αντικατάστασης γραμματοσειράς**
   Ορίστε έναν κανόνα αντικατάστασης για να αντικαταστήσετε τη γραμματοσειρά προέλευσης με τη γραμματοσειρά προορισμού όταν δεν είναι προσβάσιμη:
   
   ```csharp
   IFontSubstRule fontSubstRule = new FontSubstRule(sourceFont, destFont, FontSubstCondition.WhenInaccessible);
   ```

4. **Προσθήκη του κανόνα στη συλλογή**
   Αρχικοποιήστε και προσθέστε τον κανόνα αντικατάστασης στη συλλογή στο `FontsManager`:
   
   ```csharp
   IFontSubstRuleCollection fontSubstRuleCollection = new FontSubstRuleCollection();
   fontSubstRuleCollection.Add(fontSubstRule);
   presentation.FontsManager.FontSubstRuleList = fontSubstRuleCollection;
   ```

5. **Συμβουλές αντιμετώπισης προβλημάτων**
   - Βεβαιωθείτε ότι η γραμματοσειρά προορισμού είναι εγκατεστημένη στο σύστημά σας.
   - Επαληθεύστε τις διαδρομές αρχείων και βεβαιωθείτε ότι είναι προσβάσιμες.

### Εξαγωγή εικόνας διαφάνειας

**Επισκόπηση**
Η εξαγωγή εικόνων διαφανειών μπορεί να είναι χρήσιμη για τη δημιουργία μικρογραφιών ή την ενσωμάτωση διαφανειών σε άλλες μορφές πολυμέσων.

#### Οδηγός βήμα προς βήμα
1. **Φόρτωση της παρουσίασής σας**
   Όπως και πριν, φορτώστε την παρουσίαση:
   
   ```csharp
   string dataDir = "YOUR_DOCUMENT_DIRECTORY";
   Presentation presentation = new Presentation(dataDir + "/Fonts.pptx");
   ```

2. **Εξαγωγή και αποθήκευση μιας διαφάνειας ως εικόνας**
   Χρήση `GetThumbnail` για να δημιουργήσετε μια εικόνα της διαφάνειας και να την αποθηκεύσετε σε μορφή JPEG:
   
   ```csharp
   IImage img = presentation.Slides[0].GetThumbnail(1f, 1f);
   img.Save(dataDir + "/Slide_Image_out.jpg", ImageFormat.Jpeg);
   ```

3. **Συμβουλές αντιμετώπισης προβλημάτων**
   - Ελέγξτε τα δικαιώματα του καταλόγου εξόδου.
   - Βεβαιωθείτε ότι το `ImageFormat` είναι σωστά καθορισμένο.

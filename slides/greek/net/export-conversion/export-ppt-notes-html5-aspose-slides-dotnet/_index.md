### Εξαγωγή παρουσίασης με σημειώσεις σε HTML5

#### Επισκόπηση

Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε μια παρουσίαση PowerPoint μαζί με τις σημειώσεις του ομιλητή σε ένα εύκολα διανεμητέο αρχείο HTML5. Αυτή η δυνατότητα είναι ανεκτίμητη κατά την κοινή χρήση παρουσιάσεων σε περιβάλλοντα όπου το PowerPoint δεν είναι διαθέσιμο ή δεν προτιμάται.

#### Οδηγός βήμα προς βήμα

##### Ορισμός διαδρομών για αρχεία εισόδου και εξόδου

Καθορίστε τις διαδρομές καταλόγου για την παρουσίαση εισόδου και το αρχείο HTML εξόδου:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Κατάλογος που περιέχει το αρχείο παρουσίασης πηγής
string resultPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "Html5NotesResult.html"); // Διαδρομή εξόδου
```

Εδώ, `dataDir` είναι εκεί που είσαι εσύ `.pptx` το αρχείο βρίσκεται, και `resultPath` Καθορίζει πού θα πρέπει να αποθηκευτεί η έξοδος HTML.

##### Φόρτωση της παρουσίασης

Δημιουργήστε ένα `Presentation` αντικείμενο για να φορτώσετε το αρχείο PowerPoint σας:
```csharp
using (Presentation pres = new Presentation(dataDir + "/ConvertWithNote.pptx"))
{
    // Ο κώδικας επεξεργασίας θα μεταφερθεί εδώ
}
```

Αυτό το μπλοκ αρχικοποιεί την παρουσίαση, επιτρέποντάς σας να την χειριστείτε και να την εξαγάγετε.

##### Ρύθμιση παραμέτρων επιλογών εξαγωγής HTML5

Ρυθμίστε τις επιλογές για εξαγωγή σε HTML5, εστιάζοντας στη διάταξη των σημειώσεων:
```csharp
Html5Options options = new Html5Options
{
    OutputPath = "YOUR_OUTPUT_DIRECTORY",
    NotesCommentsLayouting = new NotesCommentsLayoutingOptions
    {
        NotesPosition = NotesPositions.BottomTruncated // Τοποθέτηση σημειώσεων στο κάτω μέρος των διαφανειών
    }
};
```

Εδώ, `NotesPosition` Καθορίζει πού θα εμφανίζονται οι σημειώσεις του ομιλητή σε σχέση με το περιεχόμενο της διαφάνειας.

##### Αποθήκευση ως HTML5

Τέλος, αποθηκεύστε την παρουσίαση χρησιμοποιώντας τις διαμορφωμένες επιλογές:
```csharp
pres.Save(resultPath, SaveFormat.Html5, options);
```

Αυτό το βήμα μετατρέπει το αρχείο PowerPoint σε έγγραφο HTML5, με σημειώσεις τοποθετημένες σύμφωνα με τις ρυθμίσεις σας.

### Συμβουλές αντιμετώπισης προβλημάτων

- **Το αρχείο δεν βρέθηκε**: Βεβαιωθείτε `dataDir` δείχνει σωστά την πηγή σου `.pptx`.
- **Προβλήματα δικαιωμάτων**: Επαλήθευση πρόσβασης εγγραφής για τον κατάλογο που καθορίζεται στο `resultPath`.

## Οδηγός Εφαρμογής

Τώρα, ας δούμε τη διαδικασία εξαγωγής μιας παρουσίασης PowerPoint σε αρχείο PDF με ρυθμίσεις συμμόρφωσης χρησιμοποιώντας το Aspose.Slides.

### Εξαγωγή PowerPoint σε συμβατό PDF

Αυτή η λειτουργία σάς επιτρέπει να αποθηκεύετε τις παρουσιάσεις σας ως αρχεία PDF που συμμορφώνονται με τα πρότυπα του κλάδου, όπως το PDF/A2a. Δείτε πώς μπορείτε να το πετύχετε αυτό:

#### Βήμα 1: Φόρτωση της παρουσίασης

Ξεκινήστε φορτώνοντας το αρχείο παρουσίασής σας σε ένα `Aspose.Slides.Presentation` αντικείμενο.
```csharp
using System;
using Aspose.Slides;

// Ορισμός διαδρομών εισόδου και εξόδου
double x = Convert.ToDouble(Console.ReadLine());
string presentationName = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "ConvertToPDF.pptx");

// Φόρτωση της παρουσίασης
Presentation presentation = new Presentation(presentationName);
```

#### Βήμα 2: Ρύθμιση παραμέτρων επιλογών PDF

Στη συνέχεια, διαμορφώστε το `PdfOptions` για να καθορίσετε τα επίπεδα συμμόρφωσης. Για αυτό το παράδειγμα, θα χρησιμοποιήσουμε `PdfA2a`μια παραλλαγή του PDF/A που υποστηρίζει την προσθήκη ετικετών για προσβασιμότητα.
```csharp
using Aspose.Slides.Export;

// Ρύθμιση επιλογών PDF με επίπεδο συμμόρφωσης
textbox = new TextBox(x, y, width, height);
txtbx.TextFrame.Text = "Example Text";
txtbx.TextFrame.Paragraphs[0].ParagraphFormat.DefaultPortionFormat.FillFormat.FillType = FillType.Solid;

PdfOptions pdfOptions = new PdfOptions() { Compliance = PdfCompliance.PdfA2a };
```

#### Βήμα 3: Αποθηκεύστε την παρουσίαση ως PDF

Τέλος, αποθηκεύστε την παρουσίασή σας σε μορφή PDF με τις καθορισμένες ρυθμίσεις συμμόρφωσης.
```csharp
string outPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "ConvertToPDF-Comp.pdf");

// Εξαγωγή σε PDF με ρυθμίσεις συμμόρφωσης
txtbx.TextFrame.Paragraphs[0].ParagraphFormat.DefaultPortionFormat.FillFormat.SolidFillColor.Color = Color.Black;
presentation.Save(outPath, SaveFormat.Pdf, pdfOptions);
```

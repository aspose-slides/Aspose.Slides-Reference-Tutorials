## Οδηγός Εφαρμογής

### Προσθήκη σχολίων για διαφάνειες

#### Επισκόπηση
Σε αυτήν την ενότητα, θα επικεντρωθούμε στον τρόπο προσθήκης σχολίων σε μια συγκεκριμένη διαφάνεια. Αυτό μπορεί να είναι χρήσιμο για την προσθήκη σχολίων σε διαφάνειες κατά τη διάρκεια παρουσιάσεων ή την παροχή σχολίων.

#### Βήματα για την προσθήκη σχολίων:
**1. Δημιουργήστε μια παρουσία παρουσίασης**
   - Ξεκινήστε δημιουργώντας μια παρουσία του `Presentation` κλάση, η οποία αντιπροσωπεύει το αρχείο PowerPoint σας.
   
```csharp
using (Presentation presentation = new Presentation())
{
    // Ο κώδικας θα μπει εδώ
}
```

**2. Προσθήκη διάταξης διαφάνειας**
   - Χρησιμοποιήστε την πρώτη διαφάνεια διάταξης ως πρότυπο για να προσθέσετε μια νέα κενή διαφάνεια.

```csharp
ISlideLayoutSlide layoutSlide = presentation.LayoutSlides[0];
presentation.Slides.AddEmptySlide(layoutSlide);
```

**3. Προσθέστε έναν συγγραφέα για σχόλια**
Δημιουργήστε έναν συγγραφέα που θα σχετίζεται με τα σχόλια. Αυτό είναι κρίσιμο επειδή κάθε σχόλιο στο Aspose.Slides συνδέεται με έναν συγγραφέα.

```csharp
ICommentAuthor author = presentation.CommentAuthors.AddAuthor("Jawad", "");
```

**4. Προσθήκη του σχολίου**
   - Προσθέστε ένα σχόλιο στη διαφάνεια. Καθορίστε τη θέση της και το περιεχόμενο κειμένου.

```csharp
ISlide slide = presentation.Slides[0];
float xPosition = 100;
float yPosition = 100;

// Δημιουργία αντικειμένου σχολίου για τον πρώτο συγγραφέα στην πρώτη διαφάνεια
IAutoShape shape = slide.Shapes.AddAutoShape(ShapeType.Rectangle, xPosition, yPosition, 200, 50);
shape.FillFormat.FillType = FillType.NoFill;

IParagraph para = new Paragraph();
para.Portions.Add(new Portion("This is a comment."));
IComment comment = author.Comments.AddComment(para, slide, DateTime.Now);
```

#### Επεξήγηση παραμέτρων:
- **Συγγραφέας**Αντιπροσωπεύει το άτομο που προσθέτει το σχόλιο. Αυτό βοηθά στην παρακολούθηση του ποιος έκανε κάθε σχολιασμό.
- **Θέση (xΘέση, yΘέση)**: Συντεταγμένες όπου θα τοποθετηθεί το σχόλιο στη διαφάνεια.
- **Ημερομηνία/Ώρα.Τώρα**: Ορίζει τη χρονική σήμανση για την προσθήκη του σχολίου.

#### Βασικές επιλογές διαμόρφωσης
- Προσαρμόζω `ShapeType` για να αλλάξετε τον τρόπο οπτικής αναπαράστασης των σχολίων.
- Προσαρμόστε το χρώμα και τη γραμματοσειρά κειμένου τροποποιώντας το `Portion` ιδιότητες αντικειμένου.

**Συμβουλές αντιμετώπισης προβλημάτων:**
- Βεβαιωθείτε ότι έχετε πρόσβαση εγγραφής στον κατάλογο εξόδου όπου αποθηκεύετε την παρουσίασή σας.
- Ελέγξτε ξανά την ορθογραφία στα ονόματα των συγγραφέων, καθώς αυτό θα επηρεάσει τον τρόπο με τον οποίο αποδίδονται τα σχόλια.

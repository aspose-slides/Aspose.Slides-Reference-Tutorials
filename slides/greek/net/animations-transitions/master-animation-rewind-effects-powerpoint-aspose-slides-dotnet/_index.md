## Οδηγός Εφαρμογής

### Διαχείριση εφέ επαναφοράς κίνησης

Αυτή η λειτουργία δείχνει πώς να καθορίσετε εάν ένα εφέ κίνησης θα επαναφέρεται μετά την αναπαραγωγή.

**Επισκόπηση:**
Ρυθμίζοντας το `Rewind` Με αυτήν την ιδιότητα, μπορείτε να ελέγξετε εάν μια κινούμενη εικόνα θα αναπαράγεται προς τα πίσω μόλις ολοκληρωθεί. Αυτό είναι ιδιαίτερα χρήσιμο για την ενίσχυση βασικών σημείων κατά τη διάρκεια μιας παρουσίασης ή για να κάνετε τις διαφάνειές σας πιο διαδραστικές.

#### Βήμα προς βήμα εφαρμογή

**1. Φορτώστε την παρουσίασή σας**

Ξεκινήστε φορτώνοντας το αρχείο PowerPoint όπου θέλετε να διαχειριστείτε τις κινούμενες εικόνες.
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
using (Presentation presentation = new Presentation(dataDir + "/AnimationRewind.pptx"))
{
    // Συνεχίστε με τα βήματα διαχείρισης κινούμενων εικόνων...
}
```

**2. Πρόσβαση στην ακολουθία κινουμένων σχεδίων**

Ανακτήστε την κύρια ακολουθία εφέ για μια συγκεκριμένη διαφάνεια, συνήθως την πρώτη.
```csharp
ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;
```

**3. Ρύθμιση παραμέτρων ιδιότητας Rewind**

Επιλέξτε ένα εφέ από την ακολουθία και ορίστε το `Rewind` η ιδιότητα σε true. Αυτό ενεργοποιεί τη λειτουργία επαναφοράς.
```csharp
IEffect effect = effectsSequence[0];
effect.Timing.Rewind = true;
```

**4. Αποθηκεύστε την παρουσίασή σας**

Μετά τη διαμόρφωση, αποθηκεύστε την τροποποιημένη παρουσίαση σε ένα νέο αρχείο.
```csharp
string outPath = "YOUR_OUTPUT_DIRECTORY";
presentation.Save(outPath + "/AnimationRewind-out.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
```

Έλεγχος του `Rewind` Η κατάσταση ιδιοτήτων βοηθά στη διασφάλιση της αναμενόμενης συμπεριφοράς των κινούμενων εικόνων σας μετά τις τροποποιήσεις.

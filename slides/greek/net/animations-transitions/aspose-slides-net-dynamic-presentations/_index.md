## Οδηγός Εφαρμογής

### Λειτουργία 1: Δημιουργία νέας διαφάνειας

**Επισκόπηση:**
Η προσθήκη διαφανειών με συγκεκριμένες διατάξεις ή φόντα είναι θεμελιώδης για τη δημιουργία επαγγελματικών παρουσιάσεων. Αυτή η λειτουργία σάς επιτρέπει να εισάγετε μια κενή διαφάνεια και να προσαρμόσετε το χρώμα φόντου της.

#### Βήμα 1: Δημιουργία νέας παρουσίασης
```csharp
Presentation pres = new Presentation();
```

#### Βήμα 2: Προσθήκη κενής διαφάνειας
```csharp
ISlide slide = pres.Slides.AddEmptySlide(pres.Slides[0].LayoutSlide);
```
*Εξήγηση:* Αυτό το βήμα προσθέτει μια νέα διαφάνεια με βάση τη διάταξη της πρώτης διαφάνειας.

#### Βήμα 3: Ορισμός χρώματος φόντου
```csharp
slide.Background.FillFormat.FillType = FillType.Solid;
slide.Background.FillFormat.SolidFillColor.Color = Color.YellowGreen;
slide.Background.Type = BackgroundType.OwnBackground;
```
*Εξήγηση:* Εδώ, ορίζουμε ένα συμπαγές χρώμα φόντου και καθορίζουμε ότι αυτή η διαφάνεια έχει το δικό της μοναδικό φόντο.

### Λειτουργία 2: Προσθήκη νέας ενότητας στην παρουσίαση

**Επισκόπηση:**
Οι ενότητες βοηθούν στην οργάνωση των διαφανειών σε ομάδες με νόημα. Αυτή η λειτουργία δείχνει πώς να δημιουργήσετε μια νέα ενότητα που σχετίζεται με μια συγκεκριμένη διαφάνεια.

#### Βήμα 1: Προσθήκη νέας ενότητας
```csharp
pres.Sections.AddSection("Section 1", slide);
```
*Εξήγηση:* Αυτή η εντολή δημιουργεί μια νέα ενότητα με το όνομα "Ενότητα 1" και τη συσχετίζει με τη διαφάνεια που δημιουργήθηκε προηγουμένως.

### Λειτουργία 3: Προσθήκη SectionZoomFrame στη διαφάνεια

**Επισκόπηση:**
Η λειτουργία SectionZoomFrame επιτρέπει στους χρήστες να εστιάζουν σε συγκεκριμένα μέρη της παρουσίασής σας, βελτιώνοντας την πλοήγηση και την εμπειρία χρήστη.

#### Βήμα 1: Προσθήκη ενός SectionZoomFrame
```csharp
ISectionZoomFrame sectionZoomFrame = pres.Slides[0].Shapes.AddSectionZoomFrame(20, 20, 300, 200, pres.Sections[1]);
```
*Εξήγηση:* Αυτό το βήμα τοποθετεί ένα πλαίσιο ζουμ στη διαφάνεια στις συντεταγμένες (20, 20) με μέγεθος 300x200 pixel και το συνδέει με τη δεύτερη ενότητα.

### Λειτουργία 4: Αποθήκευση της παρουσίασης

**Επισκόπηση:**
Αφού τροποποιήσετε την παρουσίασή σας, πρέπει να αποθηκεύσετε αυτές τις αλλαγές. Η τελευταία λειτουργία δείχνει πώς να το κάνετε αυτό αποτελεσματικά.

#### Βήμα 1: Αποθηκεύστε την παρουσίασή σας
```csharp
string resultPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "SectionZoomPresentation.pptx");
pres.Save(resultPath, SaveFormat.Pptx);
```
*Εξήγηση:* Αυτό αποθηκεύει την παρουσίασή σας σε μορφή PPTX στην καθορισμένη διαδρομή καταλόγου. Αντικατάσταση `"YOUR_OUTPUT_DIRECTORY"` με την επιθυμητή τοποθεσία αποθήκευσης.

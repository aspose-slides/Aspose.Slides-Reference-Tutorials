## Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, θα εξερευνήσουμε τον τρόπο εναλλαγής γραμμών και στηλών σε ένα γράφημα χρησιμοποιώντας το Aspose.Slides για .NET.

### Προσθήκη και πρόσβαση σε γραφήματα

#### Επισκόπηση:
Για να χειριστείτε γραφήματα, πρέπει πρώτα να προσθέσετε ένα στη διαφάνεια της παρουσίασής σας και να αποκτήσετε πρόσβαση στις σειρές δεδομένων και στις κατηγορίες του.

**1. Φόρτωση μιας υπάρχουσας παρουσίασης:**

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
using (Presentation pres = new Presentation(Path.Combine(dataDir, "Test.pptx")))
{
    // Πρόσβαση στην πρώτη διαφάνεια της παρουσίασης
    ISlide slide = pres.Slides[0];
```

**2. Προσθέστε ένα γράφημα ομαδοποιημένων στηλών:**

```csharp
// Προσθήκη γραφήματος ομαδοποιημένων στηλών στη διαφάνεια
IChart chart = slide.Shapes.AddChart(ChartType.ClusteredColumn, 100, 100, 400, 300);
```

#### Εξήγηση:
- **`AddChart`:** Αυτή η μέθοδος προσθέτει ένα νέο διάγραμμα συγκεκριμένου τύπου και διαστάσεων.
- **Παράμετροι:** `ChartType`, θέση (`x`, `y`), πλάτος, ύψος.

### Εναλλαγή γραμμών και στηλών

#### Επισκόπηση:
Για να αλλάξετε γραμμές με στήλες στα δεδομένα του γραφήματος, πρέπει να αποκτήσετε πρόσβαση στις σειρές και τις κατηγορίες γραφημάτων.

**1. Πρόσβαση σε σειρές γραφημάτων:**

```csharp
// Αποθήκευση αναφορών σε όλες τις σειρές στο γράφημα
IChartSeries[] series = new IChartSeries[chart.ChartData.Series.Count];
chart.ChartData.Series.CopyTo(series, 0);
```

**2. Μετατροπή κατηγοριών σε αναφορές κελιών:**

```csharp
// Αποθήκευση αναφορών σε όλα τα κελιά κατηγορίας στα δεδομένα του γραφήματος
IChartDataCell[] categoriesCells = new IChartDataCell[chart.ChartData.Categories.Count];

for (int i = 0; i < chart.ChartData.Categories.Count; i++)
{
    // Μετατροπή κάθε κατηγορίας σε αναφορά κελιού
    categoriesCells[i] = chart.ChartData.Categories[i].AsCell;
}
```

#### Εξήγηση:
- **`IChartSeries`:** Αντιπροσωπεύει μεμονωμένες σειρές δεδομένων στο γράφημα.
- **`IChartDataCell`:** Επιτρέπει τον χειρισμό κελιών κατηγορίας για την εναλλαγή λογικής.

### Συμβουλές αντιμετώπισης προβλημάτων

- Βεβαιωθείτε ότι όλες οι αναφορές σε σειρές και κατηγορίες έχουν αρχικοποιηθεί σωστά πριν επιχειρήσετε τροποποιήσεις.
- Επικυρώστε τη διαδρομή του καταλόγου σας κατά τη φόρτωση παρουσιάσεων για να αποφύγετε σφάλματα "δεν βρέθηκε αρχείο".

# Προσαρμογή κύριας γραμματοσειράς σε γραφήματα PowerPoint χρησιμοποιώντας το Aspose.Slides για .NET

## Πώς να ορίσετε ιδιότητες γραμματοσειράς για κείμενα γραφημάτων χρησιμοποιώντας το Aspose.Slides .NET

### Εισαγωγή

Η βελτίωση της αναγνωσιμότητας και της οπτικής ελκυστικότητας του κειμένου γραφήματος στα γραφήματα PowerPoint είναι ζωτικής σημασίας, είτε προετοιμάζετε επιχειρηματικές αναφορές είτε ακαδημαϊκές παρουσιάσεις. Αυτός ο οδηγός θα σας δείξει πώς να ορίσετε ιδιότητες γραμματοσειράς όπως η έντονη γραφή και το ύψος χρησιμοποιώντας το Aspose.Slides για .NET.

**Τι θα μάθετε:**
- Πώς να ενσωματώσετε το Aspose.Slides στο έργο σας
- Βήματα για την προσθήκη και προσαρμογή ενός γραφήματος ομαδοποιημένων στηλών στο PowerPoint
- Τεχνικές για την τροποποίηση ιδιοτήτων γραμματοσειράς μέσα σε κείμενα γραφημάτων
- Βέλτιστες πρακτικές για την αποθήκευση και τη διαχείριση παρουσιάσεων

Ετοιμαστείτε να ενισχύσετε την οπτική επίδραση των γραφημάτων σας!

## Ρύθμιση του Aspose.Slides για .NET

Ακολουθήστε αυτά τα βήματα για να ρυθμίσετε το έργο σας χρησιμοποιώντας το Aspose.Slides για .NET:

### Εγκατάσταση μέσω .NET CLI

Εκτελέστε την ακόλουθη εντολή στο τερματικό σας:
```bash
dotnet add package Aspose.Slides
```

### Εγκατάσταση μέσω της Κονσόλας Διαχείρισης Πακέτων

Εκτελέστε αυτήν την εντολή στην κονσόλα NuGet Package Manager:
```powershell
Install-Package Aspose.Slides
```

### Εγκατάσταση μέσω του περιβάλλοντος εργασίας χρήστη του NuGet Package Manager

- Ανοίξτε το έργο σας στο Visual Studio.
- Πλοήγηση σε **Εργαλεία > Διαχειριστής πακέτων NuGet > Διαχείριση πακέτων NuGet για λύση**.
- Αναζητήστε το "Aspose.Slides" και κάντε κλικ στην επιλογή Εγκατάσταση.

### Βήματα απόκτησης άδειας χρήσης

1. **Δωρεάν δοκιμή**: Κατεβάστε μια δοκιμαστική έκδοση από το [Ιστότοπος Aspose](https://releases.aspose.com/slides/net/).
2. **Προσωρινή Άδεια**Αποκτήστε μια προσωρινή άδεια χρήσης για να εξερευνήσετε όλες τις λειτουργίες χωρίς περιορισμούς.
3. **Αγορά**Σκεφτείτε να το αγοράσετε εάν το θεωρείτε ωφέλιμο για μακροχρόνια χρήση.

Μόλις εγκατασταθεί, αρχικοποιήστε το Aspose.Slides στο έργο σας συμπεριλαμβάνοντας τον χώρο ονομάτων:
```csharp
using Aspose.Slides;
```

## Οδηγός Εφαρμογής

Αφού ρυθμίσετε το περιβάλλον σας, ακολουθήστε τα παρακάτω βήματα για να αλλάξετε τις ιδιότητες γραμματοσειράς στα κείμενα των γραφημάτων:

### Βήμα 1: Φόρτωση ενός υπάρχοντος αρχείου παρουσίασης

Φορτώστε ένα αρχείο παρουσίασης από τον κατάλογο όπου θέλετε να εφαρμόσετε τις αλλαγές:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Αντικαταστήστε με τη διαδρομή του εγγράφου σας
string filePath = Path.Combine(dataDir, "test.pptx");
```
**Εξήγηση**Αυτός ο κώδικας ορίζει τη διαδρομή αρχείου για τη φόρτωση της υπάρχουσας παρουσίασης PowerPoint.

### Βήμα 2: Ανοίξτε την παρουσίαση

Ανοίξτε την παρουσίαση χρησιμοποιώντας το Aspose.Slides:
```csharp
using (Presentation pres = new Presentation(filePath))
{
    // Τα επόμενα βήματα θα ενσωματωθούν σε αυτό το μπλοκ
}
```
**Εξήγηση**: Το `Presentation` Η κλάση χειρίζεται το άνοιγμα και τον χειρισμό του αρχείου PowerPoint σας. Χρησιμοποιώντας ένα `using` Η δήλωση διασφαλίζει ότι οι πόροι διατίθενται σωστά.

### Βήμα 3: Προσθήκη γραφήματος ομαδοποιημένων στηλών

Προσθήκη ενός γραφήματος ομαδοποιημένων στηλών στην πρώτη διαφάνεια:
```csharp
IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 50, 50, 600, 400);
```
**Εξήγηση**: Αυτό το βήμα δημιουργεί ένα νέο γράφημα ομαδοποιημένων στηλών σε καθορισμένες συντεταγμένες και διαστάσεις.

### Βήμα 4: Ενεργοποίηση της εμφάνισης πίνακα δεδομένων

Βεβαιωθείτε ότι ο πίνακας δεδομένων είναι ορατός μέσα στο γράφημα:
```csharp
chart.HasDataTable = true;
```
**Εξήγηση**: Ρύθμιση `HasDataTable` Η τιμή true διασφαλίζει ότι εμφανίζονται οι ετικέτες δεδομένων, τις οποίες θα προσαρμόσουμε στη συνέχεια.

### Βήμα 5: Ορισμός ιδιοτήτων γραμματοσειράς για κείμενο γραφήματος

Προσαρμόστε τις ιδιότητες γραμματοσειράς, όπως η έντονη γραφή και το ύψος, για το κείμενο του πίνακα δεδομένων του γραφήματός σας:
```csharp
chart.ChartDataTable.TextFormat.PortionFormat.FontBold = NullableBool.True; // Κάντε το κείμενο έντονο
chart.ChartDataTable.TextFormat.PortionFormat.FontHeight = 20; // Ορισμός ύψους γραμματοσειράς σε 20 σημεία
```
**Εξήγηση**Αυτές οι γραμμές προσαρμόζουν το οπτικό στυλ των ετικετών δεδομένων του γραφήματός σας, καθιστώντας τες πιο εμφανείς και ευανάγνωστες.

### Βήμα 6: Αποθήκευση της τροποποιημένης παρουσίασης

Τέλος, αποθηκεύστε την παρουσίαση με τις αλλαγές:
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY"; // Αντικαταστήστε με τη διαδρομή εξόδου σας
string outputPath = Path.Combine(outputDir, "output.pptx");
pres.Save(outputPath, SaveFormat.Pptx);
```
**Εξήγηση**: Αυτό το βήμα εγγράφει την ενημερωμένη παρουσίαση σε ένα νέο αρχείο στον καθορισμένο κατάλογο.

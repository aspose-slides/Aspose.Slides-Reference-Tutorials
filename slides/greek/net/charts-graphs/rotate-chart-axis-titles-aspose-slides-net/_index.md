## Οδηγός Εφαρμογής
Αυτός ο οδηγός θα σας καθοδηγήσει στη ρύθμιση της γωνίας περιστροφής ενός τίτλου άξονα γραφήματος χρησιμοποιώντας το Aspose.Slides για .NET.
### Επισκόπηση λειτουργιών: Ρύθμιση γωνίας περιστροφής του τίτλου του άξονα του γραφήματος
Η ρύθμιση της γωνίας περιστροφής μπορεί να βελτιώσει την αναγνωσιμότητα και την αισθητική, ειδικά σε διαφάνειες με περιορισμένο χώρο. Δείτε πώς μπορείτε να εφαρμόσετε αυτήν τη λειτουργία:
#### Βήμα 1: Δημιουργήστε μια παρουσίαση και προσθέστε ένα γράφημα
Ξεκινήστε δημιουργώντας μια νέα παρουσίαση και προσθέτοντας ένα γράφημα ομαδοποιημένων στηλών.
```csharp
using Aspose.Slides;

string dataDir = "YOUR_DOCUMENT_DIRECTORY";

// Αρχικοποιήστε μια νέα παρουσία παρουσίασης.
using (Presentation pres = new Presentation())
{
    // Προσθέστε ένα γράφημα ομαδοποιημένων στηλών στην πρώτη διαφάνεια στη θέση (50, 50) με πλάτος 450 και ύψος 300.
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 50, 50, 450, 300);
```
#### Βήμα 2: Ενεργοποίηση τίτλου κάθετου άξονα
Ενεργοποιήστε τον τίτλο του κατακόρυφου άξονα για να προσαρμόσετε την εμφάνισή του.
```csharp
    // Ενεργοποιήστε τον τίτλο του κατακόρυφου άξονα για το γράφημα.
    chart.Axes.VerticalAxis.HasTitle = true;
```
#### Βήμα 3: Ορισμός γωνίας περιστροφής
Ορίστε τη γωνία περιστροφής της μορφής μπλοκ κειμένου για τον τίτλο του κατακόρυφου άξονα.
```csharp
    // Ρυθμίστε τη γωνία περιστροφής στις 90 μοίρες.
    chart.Axes.VerticalAxis.Title.TextFormat.TextBlockFormat.RotationAngle = 90;

    // Αποθηκεύστε την παρουσίαση με το τροποποιημένο γράφημα σε ένα αρχείο .pptx στον καθορισμένο κατάλογο.
    pres.Save(dataDir + "test.pptx", SaveFormat.Pptx);
}
```
### Βασικές επιλογές διαμόρφωσης
- **Γωνία περιστροφής**Προσαρμόστε μεταξύ -180 και 180 μοιρών ανάλογα με τις ανάγκες σχεδιασμού σας.
- **Μορφή τίτλου άξονα**: Τροποποιήστε το μέγεθος, το στυλ και το χρώμα της γραμματοσειράς για καλύτερη ορατότητα.

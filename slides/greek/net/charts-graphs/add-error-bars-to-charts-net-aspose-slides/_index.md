## Οδηγός Εφαρμογής
Τώρα, ας αναλύσουμε τα βήματα για την προσθήκη γραμμών σφάλματος σε ένα γράφημα.

### Προσθήκη γραμμών σφάλματος σε ένα γράφημα
#### Επισκόπηση
Η προσθήκη γραμμών σφάλματος σάς βοηθά να αναπαραστήσετε οπτικά τη μεταβλητότητα ή την αβεβαιότητα των δεδομένων στα γραφήματά σας. Αυτή η λειτουργία είναι ιδιαίτερα χρήσιμη σε επιστημονικές και οικονομικές παρουσιάσεις όπου η ακρίβεια έχει σημασία.

#### Βήμα προς βήμα εφαρμογή
**1. Δημιουργήστε μια κενή παρουσίαση**
Ξεκινήστε δημιουργώντας ένα νέο αντικείμενο παρουσίασης:
```csharp
using (Presentation presentation = new Presentation())
{
    // Περαιτέρω κώδικας θα μπει εδώ.
}
```

**2. Προσθέστε ένα γράφημα φυσαλίδων στη διαφάνεια**
Προσθέστε ένα γράφημα στη διαφάνειά σας σε καθορισμένες συντεταγμένες με τις επιθυμητές διαστάσεις:
```csharp
IChart chart = presentation.Slides[0].Shapes.AddChart(
    ChartType.Bubble, 50, 50, 400, 300, true);
```

**3. Ρύθμιση παραμέτρων γραμμών σφάλματος για τους άξονες X και Y**
Αποκτήστε πρόσβαση στις μορφές της γραμμής σφαλμάτων για να τις προσαρμόσετε:
```csharp
IErrorBarsFormat errBarX = chart.ChartData.Series[0].ErrorBarsXFormat;
IErrorBarsFormat errBarY = chart.ChartData.Series[0].ErrorBarsYFormat;

errBarX.IsVisible = true;  // Ενεργοποίηση ορατότητας για γραμμές σφάλματος X
erBarY.IsVisible = true;  // Ενεργοποίηση ορατότητας για γραμμές σφάλματος Y

// Ορισμός τύπων και τιμών για τις γραμμές σφάλματος
errBarX.ValueType = ErrorBarValueType.Fixed;
errBarX.Value = 0.1f;  // Σταθερή τιμή για τη γραμμή σφάλματος X

errBarY.ValueType = ErrorBarValueType.Percentage;
erBarY.Value = 5;  // Ποσοστιαία τιμή για τη γραμμή σφάλματος Y

// Ρύθμιση παραμέτρων πρόσθετων ιδιοτήτων
erBarX.Type = ErrorBarType.Plus;
errBarY.Format.Line.Width = 2;  // Ορισμός πλάτους γραμμής για γραμμές σφάλματος Y
erBarX.HasEndCap = true;  // Ενεργοποίηση τελικού ορίου για γραμμές σφάλματος X
```

**4. Αποθήκευση της παρουσίασης**
Τέλος, αποθηκεύστε την παρουσίασή σας σε έναν καθορισμένο κατάλογο:
```csharp
presentation.Save(dataDir + "ErrorBars_out.pptx");
```

- **Ελέγξτε τη διαδρομή καταλόγου δεδομένων:** Βεβαιωθείτε ότι το `dataDir` Η μεταβλητή δείχνει σε μια έγκυρη διαδρομή καταλόγου.

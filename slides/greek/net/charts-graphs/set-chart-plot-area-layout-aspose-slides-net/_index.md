# Ορισμός διάταξης περιοχής γραφήματος στο PowerPoint χρησιμοποιώντας το Aspose.Slides .NET

Αυτό το σεμινάριο σας καθοδηγεί στη διαμόρφωση της περιοχής σχεδίασης ενός γραφήματος χρησιμοποιώντας το Aspose.Slides.

Ας ξεκινήσουμε κατανοώντας τις προϋποθέσεις.

## Οδηγός Εφαρμογής
### Διάταξη περιοχής σχεδίασης γραφήματος
Η διαμόρφωση της διάταξης της περιοχής γραφήματος σάς επιτρέπει να προσαρμόσετε τον τρόπο με τον οποίο η οπτικοποίηση δεδομένων ταιριάζει μέσα στο κοντέινερ της.

#### Βήμα 1: Δημιουργία και πρόσβαση σε μια διαφάνεια
Βεβαιωθείτε ότι η παρουσίασή σας έχει τουλάχιστον μία διαφάνεια:
```csharp
using Aspose.Slides;
// Αρχικοποίηση μιας νέας παρουσίας παρουσίασης
Presentation presentation = new Presentation();
// Πρόσβαση στην πρώτη διαφάνεια της παρουσίασης
ISlide slide = presentation.Slides[0];
```

#### Βήμα 2: Προσθήκη γραφήματος στη διαφάνεια
Προσθέστε ένα γράφημα ομαδοποιημένων στηλών σε καθορισμένες συντεταγμένες με δεδομένες διαστάσεις:
```csharp
// Προσθήκη ενός γραφήματος ομαδοποιημένων στηλών στη θέση (20, 100) με μέγεθος (600x400)
IChart chart = slide.Shapes.AddChart(ChartType.ClusteredColumn, 20, 100, 600, 400);
```

#### Βήμα 3: Διαμόρφωση διάταξης περιοχής γραφήματος
Ορίστε τις ιδιότητες διάταξης για την περιοχή σχεδίασης:
```csharp
// Ορισμός διάταξης ως κλάσμα του διαθέσιμου χώρου
chart.PlotArea.AsILayoutable.X = 0.2f;
chart.PlotArea.AsILayoutable.Y = 0.2f;
chart.PlotArea.AsILayoutable.Width = 0.7f;
chart.PlotArea.AsILayoutable.Height = 0.7f;
// Καθορίστε τη διάταξη σε σχέση με την εσωτερική περιοχή
chart.PlotArea.LayoutTargetType = LayoutTargetType.Inner;
```

#### Βήμα 4: Αποθήκευση της παρουσίασης
Αποθηκεύστε την παρουσίασή σας:
```csharp
// Ορισμός καταλόγου εγγράφου και ονόματος αρχείου
string dataDir = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "SetLayoutMode_outer.pptx");
presentation.Save(dataDir, Aspose.Slides.Export.SaveFormat.Pptx);
```
Αυτή η διαμόρφωση διασφαλίζει ότι η περιοχή του οικοπέδου προσαρμόζεται δυναμικά ώστε να ταιριάζει αποτελεσματικά στον καθορισμένο χώρο της.

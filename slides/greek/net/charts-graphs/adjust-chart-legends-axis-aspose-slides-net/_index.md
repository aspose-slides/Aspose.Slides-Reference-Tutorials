### Προσαρμογή μεγέθους γραμματοσειράς υπομνήματος
Η προσαρμογή του μεγέθους της γραμματοσειράς του υπομνήματος βελτιώνει την αναγνωσιμότητα. Δείτε πώς μπορείτε να το κάνετε:

#### Επισκόπηση
Θα τροποποιήσουμε το μέγεθος της γραμματοσειράς του κειμένου του υπομνήματος ενός γραφήματος χρησιμοποιώντας το Aspose.Slides για .NET.

#### Βήματα
**1. Φορτώστε την παρουσίασή σας:**
Ξεκινήστε φορτώνοντας το αρχείο PowerPoint όπου θέλετε να προσαρμόσετε τους θρύλους του γραφήματος.
```csharp
using (Presentation pres = new Presentation(dataDir))
{
    // Αποκτήστε πρόσβαση στην πρώτη διαφάνεια και προσθέστε ένα γράφημα ομαδοποιημένων στηλών.
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 50, 50, 600, 400);
```

**2. Ορισμός μεγέθους γραμματοσειράς υπομνήματος:**
Καθορίστε το επιθυμητό ύψος γραμματοσειράς για καλύτερη ορατότητα.
```csharp
    // Προσαρμόστε το μέγεθος γραμματοσειράς του κειμένου του υπομνήματος σε 20.
    chart.Legend.TextFormat.PortionFormat.FontHeight = 20;
}
```
- **Εξήγηση:** `FontHeight` ορίζει το μέγεθος σε στιγμές, βελτιώνοντας την αναγνωσιμότητα.

**3. Αποθηκεύστε την παρουσίασή σας:**
Αφού κάνετε αλλαγές, αποθηκεύστε την παρουσίασή σας για να τις διατηρήσετε.
```csharp
pres.Save(outputDir, Aspose.Slides.Export.SaveFormat.Pptx);
```

### Ρύθμιση παραμέτρων ελάχιστων και μέγιστων τιμών κατακόρυφου άξονα
Η προσαρμογή των τιμών των αξόνων επιτρέπει την ακριβή αναπαράσταση δεδομένων.

#### Επισκόπηση
Μάθετε πώς να ορίζετε συγκεκριμένες ελάχιστες και μέγιστες τιμές για τον κατακόρυφο άξονα του γραφήματός σας.

#### Βήματα
**1. Φορτώστε την παρουσίασή σας:**
Όπως και πριν, ανοίξτε την παρουσίαση που περιέχει το γράφημά σας.
```csharp
using (Presentation pres = new Presentation(dataDir))
{
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 50, 50, 600, 400);
```

**2. Ορίστε τιμές προσαρμοσμένων αξόνων:**
Απενεργοποιήστε τις αυτόματες ρυθμίσεις τιμών άξονα και ορίστε τις δικές σας.
```csharp
    // Απενεργοποίηση αυτόματης ελαχιστοποίησης για τον κατακόρυφο άξονα.
    chart.Axes.VerticalAxis.IsAutomaticMinValue = false;
    // Ορίστε μια προσαρμοσμένη ελάχιστη τιμή -5.
    chart.Axes.VerticalAxis.MinValue = -5;

    // Ομοίως, απενεργοποιήστε την αυτόματη μέγιστη τιμή και ορίστε την σε 10.
    chart.Axes.VerticalAxis.IsAutomaticMaxValue = false;
    chart.Axes.VerticalAxis.MaxValue = 10;
}
```
- **Εξήγηση:** Η προσαρμογή αυτών των τιμών επιτρέπει την προσαρμοσμένη κλιμάκωση δεδομένων.

**3. Αποθηκεύστε την παρουσίασή σας:**
Βεβαιωθείτε ότι οι αλλαγές σας αποθηκεύονται γράφοντας ξανά στο αρχείο.
```csharp
pres.Save(outputDir, Aspose.Slides.Export.SaveFormat.Pptx);
```

## Ρύθμιση του Aspose.Slides για .NET

Για να ξεκινήσετε την προσαρμογή των υπομνημάτων των γραφημάτων σας, θα πρέπει πρώτα να ρυθμίσετε το Aspose.Slides στο έργο σας. Δείτε πώς:

### Εγκατάσταση

**Χρησιμοποιώντας το .NET CLI:**
```bash
dotnet add package Aspose.Slides
```

**Μέσω της Κονσόλας Διαχείρισης Πακέτων:**
```powershell
Install-Package Aspose.Slides
```

**Μέσω του περιβάλλοντος εργασίας χρήστη του NuGet Package Manager:**
- Ανοίξτε το έργο σας στο Visual Studio.
- Μετάβαση σε `Tools` > `NuGet Package Manager` > `Manage NuGet Packages for Solution`.
- Αναζητήστε το "Aspose.Slides" και εγκαταστήστε την πιο πρόσφατη έκδοση.

### Απόκτηση Άδειας

Για να εξερευνήσετε πλήρως τις δυνατότητες του Aspose.Slides χωρίς περιορισμούς, σκεφτείτε να αποκτήσετε μια άδεια χρήσης:

1. **Δωρεάν δοκιμή**: Ξεκινήστε με μια δοκιμή για να αξιολογήσετε τις λειτουργίες.
2. **Προσωρινή Άδεια**Αίτημα προσωρινής άδειας για εκτεταμένες δοκιμές.
3. **Αγορά**Για μακροχρόνια χρήση, αγοράστε μια άδεια χρήσης μέσω της επίσημης ιστοσελίδας.

### Βασική Αρχικοποίηση και Ρύθμιση

Μόλις εγκατασταθεί, αρχικοποιήστε το Aspose.Slides στο έργο σας ως εξής:

Δημιουργήστε μια παρουσία του `Presentation` για να φορτώσετε ή να δημιουργήσετε αρχεία PowerPoint μέσω προγραμματισμού.

## Οδηγός Εφαρμογής

Ας εμβαθύνουμε στην προσαρμογή των ιδιοτήτων της γραμματοσειράς υπομνήματος βήμα προς βήμα.

### Πρόσβαση και τροποποίηση καταχωρίσεων υπομνήματος

Αρχικά, ας προσθέσουμε ένα γράφημα στη διαφάνειά σας και ας αποκτήσουμε πρόσβαση στους υπόμνημές του:

#### Προσθήκη γραφήματος
```csharp
// Φόρτωση υπάρχουσας παρουσίασης
using (Presentation pres = new Presentation("YOUR_DOCUMENT_DIRECTORY/test.pptx"))
{
    // Προσθέστε ένα γράφημα ομαδοποιημένων στηλών στη θέση x=50, y=50 με πλάτος=600 και ύψος=400
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 50, 50, 600, 400);
}
```

#### Πρόσβαση στο Υπόμνημα
```csharp
// Πρόσβαση στο αντικείμενο μορφοποίησης κειμένου της δεύτερης καταχώρησης υπομνήματος
IChartTextFormat tf = chart.Legend.Entries[1].TextFormat;
```

### Προσαρμογή ιδιοτήτων γραμματοσειράς

Τώρα, προσαρμόστε τις ιδιότητες της γραμματοσειράς όπως η έντονη γραφή, το ύψος και το χρώμα:

#### Ρύθμιση γραμματοσειράς σε έντονη και πλάγια γραφή
```csharp
tf.PortionFormat.FontBold = NullableBool.True; // Κάντε το κείμενο έντονο
tf.PortionFormat.FontItalic = NullableBool.True; // Εφαρμογή πλάγιου στυλ
```

#### Ρύθμιση ύψους γραμματοσειράς
```csharp
tf.PortionFormat.FontHeight = 20; // Ορισμός μεγέθους γραμματοσειράς σε 20 στιγμές
```

#### Αλλαγή χρώματος γραμματοσειράς
```csharp
// Ορίστε τον τύπο γεμίσματος και το χρώμα του κειμένου
tf.PortionFormat.FillFormat.FillType = FillType.Solid;
tf.PortionFormat.FillFormat.SolidFillColor.Color = Color.Blue; // Εφαρμογή μπλε χρώματος
```

### Αποθήκευση της παρουσίασής σας

Τέλος, αποθηκεύστε την τροποποιημένη παρουσίασή σας:

```csharp
pres.Save("YOUR_DOCUMENT_DIRECTORY/output.pptx", SaveFormat.Pptx);
```

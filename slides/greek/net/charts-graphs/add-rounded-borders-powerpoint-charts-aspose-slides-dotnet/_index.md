## Οδηγός Εφαρμογής: Προσθήκη Στρογγυλεμένων Περιγραμμάτων σε Γραφήματα

### Επισκόπηση

Θα ξεκινήσουμε δημιουργώντας ένα γράφημα ομαδοποιημένων στηλών και στη συνέχεια θα εφαρμόσουμε στρογγυλεμένες γωνίες στο όριο του. Αυτή η διαδικασία βελτιώνει την οπτική αισθητική, καθιστώντας την παρουσίαση των δεδομένων σας πιο ελκυστική.

#### Βήμα 1: Δημιουργία νέας παρουσίασης

```csharp
using Aspose.Slides;
using Aspose.Slides.Charts;

// Ορίστε τον κατάλογο για την αποθήκευση της εξόδου
string dataDir = "YOUR_DOCUMENT_DIRECTORY";

// Δημιουργία αντικειμένου παρουσίασης
using (Presentation presentation = new Presentation())
{
    // Συνεχίστε με την προσθήκη ενός γραφήματος...
```

#### Βήμα 2: Προσθήκη γραφήματος στη διαφάνειά σας

Αποκτήστε πρόσβαση στην πρώτη σας διαφάνεια και προσθέστε ένα γράφημα ομαδοποιημένων στηλών:

```csharp
    ISlide slide = presentation.Slides[0];
    
    // Προσθέστε το γράφημα στη θέση (20, 100) με μέγεθος (600, 400)
    IChart chart = slide.Shapes.AddChart(ChartType.ClusteredColumn, 20, 100, 600, 400);
```

#### Βήμα 3: Ρύθμιση παραμέτρων μορφής γραμμής γραφήματος

Ορίστε τη μορφή γραμμής για να διασφαλίσετε συμπαγή περιγράμματα:

```csharp
    // Τύπος συμπαγούς γεμίσματος για γραμμές με ένα μόνο στυλ
    chart.LineFormat.FillFormat.FillType = FillType.Solid;
    chart.LineFormat.Style = LineStyle.Single;
```

#### Βήμα 4: Ενεργοποίηση στρογγυλεμένων γωνιών

Ενεργοποιήστε τη λειτουργία στρογγυλεμένων γωνιών:

```csharp
    // Εφαρμογή στρογγυλεμένων περιγραμμάτων στην περιοχή του γραφήματος
    chart.HasRoundedCorners = true;
    
    // Αποθήκευση της παρουσίασής σας
    presentation.Save(dataDir + "out.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
}
```

### Βασικές επιλογές διαμόρφωσης
- **Τύπος Γεμίσματος**: Καθορίζει εάν το περίγραμμα είναι συμπαγές ή κάποιο άλλο στυλ.
- **Στυλ γραμμής**: Ορίζει το πάχος του περιγράμματος.
- **ΈχειΣτρογγυλεμένεςΓωνίες**: Επιτρέπει στρογγυλεμένες γωνίες για αισθητική βελτίωση.

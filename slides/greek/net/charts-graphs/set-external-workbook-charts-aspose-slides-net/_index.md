# Πώς να χρησιμοποιήσετε το Aspose.Slides .NET για να ορίσετε ένα εξωτερικό βιβλίο εργασίας ως προέλευση δεδομένων γραφήματος
## Εισαγωγή
Η δημιουργία οπτικά ελκυστικών γραφημάτων σε παρουσιάσεις είναι ζωτικής σημασίας για την αποτελεσματική επικοινωνία πληροφοριών που βασίζονται σε δεδομένα. Η διαχείριση δεδομένων γραφημάτων ξεχωριστά από αρχεία παρουσίασης μπορεί να είναι περίπλοκη. Με το Aspose.Slides για .NET, μπορείτε να συνδέσετε ένα εξωτερικό βιβλίο εργασίας ως πηγή δεδομένων για τα γραφήματά σας, βελτιστοποιώντας τη ροή εργασίας σας και διατηρώντας τα δεδομένα σας οργανωμένα. Αυτό το σεμινάριο θα σας καθοδηγήσει στην εφαρμογή της λειτουργίας "Ορισμός δεδομένων γραφήματος από εξωτερικό βιβλίο εργασίας" χρησιμοποιώντας το Aspose.Slides .NET.

**Τι θα μάθετε:**
- Πώς να χρησιμοποιήσετε το Aspose.Slides για .NET για να ορίσετε ένα εξωτερικό βιβλίο εργασίας ως προέλευση δεδομένων για γραφήματα.
- Βήματα για την προσθήκη και τη διαμόρφωση ενός γραφήματος στην παρουσίασή σας με εξωτερικά δεδομένα.
- Ενσωμάτωση των λειτουργιών Aspose.Slides στα έργα .NET σας.

Ας ξεκινήσουμε ορίζοντας τις απαραίτητες προϋποθέσεις.

## Οδηγός Εφαρμογής
### Ορισμός εξωτερικού βιβλίου εργασίας για ένα γράφημα
Αυτή η λειτουργία σάς επιτρέπει να συνδέσετε τα δεδομένα του γραφήματός σας με ένα εξωτερικό βιβλίο εργασίας του Excel, διασφαλίζοντας ότι τυχόν ενημερώσεις στο βιβλίο εργασίας αντικατοπτρίζονται αυτόματα στην παρουσίασή σας.
#### Βήμα 1: Αρχικοποίηση παρουσίασης και προσθήκη γραφήματος
Δημιουργήστε μια νέα παρουσία παρουσίασης και προσθέστε ένα γράφημα πίτας στην πρώτη διαφάνεια.
```csharp
using Aspose.Slides;
using Aspose.Slides.Charts;

public class Feature_SetExternalWorkbook {
    public static void Run() {
        string dataDir = "YOUR_DOCUMENT_DIRECTORY";
        
        using (Presentation pres = new Presentation()) {
            // Προσθήκη γραφήματος πίτας στην πρώτη διαφάνεια στη θέση 50,50 με μέγεθος 400x600
            IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.Pie, 50, 50, 400, 600, false);
```
#### Βήμα 2: Πρόσβαση σε δεδομένα γραφήματος και ορισμός εξωτερικού βιβλίου εργασίας
Αποκτήστε πρόσβαση στη συλλογή δεδομένων γραφήματος για να καθορίσετε το εξωτερικό βιβλίο εργασίας σας ως προέλευση δεδομένων.
```csharp
            // Πρόσβαση στα δεδομένα του γραφήματος για χειρισμό.
            IChartData chartData = chart.ChartData;
            
            // Ορίστε το εξωτερικό βιβλίο εργασίας που περιέχει τα δεδομένα γραφήματος.
            chartData.SetExternalWorkbook(dataDir + "externalWorkbook.xlsx");
```
#### Βήμα 3: Προσθήκη Σειρών και Σημείων Δεδομένων από Εξωτερικό Βιβλίο Εργασίας
Προσθέστε μια νέα σειρά στο γράφημά σας, συνδέοντάς την με συγκεκριμένα κελιά στο εξωτερικό βιβλίο εργασίας τόσο για κατηγορίες όσο και για τιμές.
```csharp
            // Προσθήκη νέας σειράς χρησιμοποιώντας δεδομένα από το κελί B1 στο εξωτερικό βιβλίο εργασίας
            chartData.Series.Add(chartData.ChartDataWorkbook.GetCell(0, "B1"), ChartType.Pie);

            // Προσθήκη σημείων δεδομένων για τη σειρά από τα κελιά B2, B3 και B4
            chartData.Series[0].DataPoints.AddDataPointForPieSeries(
                chartData.ChartDataWorkbook.GetCell(0, "B2"));
            chartData.Series[0].DataPoints.AddDataPointForPieSeries(
                chartData.ChartDataWorkbook.GetCell(0, "B3"));
            chartData.Series[0].DataPoints.AddDataPointForPieSeries(
                chartData.ChartDataWorkbook.GetCell(0, "B4"));

            // Ορίστε κατηγορίες για τη σειρά χρησιμοποιώντας δεδομένα από τα κελιά A2, A3 και A4
            chartData.Categories.Add(chartData.ChartDataWorkbook.GetCell(0, "A2"));
            chartData.Categories.Add(chartData.ChartDataWorkbook.GetCell(0, "A3"));
            chartData.Categories.Add(chartData.ChartDataWorkbook.GetCell(0, "A4"));

            // Αποθήκευση της παρουσίασης με το καθορισμένο όνομα αρχείου
            pres.Save(dataDir + "Presentation_with_externalWorkbook.pptx");
        }
    }
}
```

## Οδηγός Εφαρμογής

### Χαρακτηριστικό: Εναλλαγή γραμμών και στηλών γραφήματος

#### Επισκόπηση
Η εναλλαγή γραμμών και στηλών σε γραφήματα είναι απαραίτητη κατά την προετοιμασία παρουσιάσεων που βασίζονται σε δεδομένα. Αυτή η λειτουργία επιτρέπει απρόσκοπτες προσαρμογές με το Aspose.Slides, διασφαλίζοντας ότι τα δεδομένα σας παρουσιάζονται με σαφήνεια.

#### Βήματα για την εφαρμογή

##### Βήμα 1: Δημιουργία νέας παρουσίασης
Ξεκινήστε αρχικοποιώντας μια νέα παρουσίαση όπου θα προσθέσετε το γράφημα:

```csharp
using (Presentation pres = new Presentation())
{
    // Ο κώδικας για την προσθήκη και τροποποίηση γραφημάτων βρίσκεται εδώ
}
```

##### Βήμα 2: Προσθήκη γραφήματος ομαδοποιημένων στηλών
Προσθέστε ένα γράφημα ομαδοποιημένων στηλών στην πρώτη σας διαφάνεια σε μια καθορισμένη θέση και μέγεθος:

```csharp
IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 100, 100, 400, 300);
```

##### Βήμα 3: Πρόσβαση σε δεδομένα γραφήματος
Ανακτήστε τα δεδομένα σειρών και κατηγοριών από το γράφημά σας για να τα διαχειριστείτε:

```csharp
IChartSeries[] series = new IChartSeries[chart.ChartData.Series.Count];
chart.ChartData.Series.CopyTo(series, 0);

IChartDataCell[] categoriesCells = new IChartDataCell[chart.ChartData.Categories.Count];
for (int i = 0; i < chart.ChartData.Categories.Count; i++)
{
    categoriesCells[i] = chart.ChartData.Categories[i].AsCell;
}

IChartDataCell[] seriesCells = new IChartDataCell[chart.ChartData.Series.Count];
for (int i = 0; i < chart.ChartData.Series.Count; i++)
{
    seriesCells[i] = chart.ChartData.Series[i].Name.AsCells[0];
}
```

##### Βήμα 4: Εναλλαγή γραμμών και στηλών
Καλέστε τη μέθοδο για να αλλάξετε γραμμές και στήλες, προσαρμόζοντας τον προσανατολισμό των δεδομένων σας:

```csharp
chart.ChartData.SwitchRowColumn();
```

##### Βήμα 5: Αποθηκεύστε την παρουσίασή σας
Τέλος, αποθηκεύστε την παρουσίασή σας με το τροποποιημένο γράφημα:

```csharp
pres.Save("YOUR_OUTPUT_DIRECTORY" + "SwitchChartRowColumns_out.pptx", SaveFormat.Pptx);
```

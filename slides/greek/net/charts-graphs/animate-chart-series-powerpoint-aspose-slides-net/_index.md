## Οδηγός Εφαρμογής

Η δημιουργία κίνησης σε μια σειρά γραφημάτων περιλαμβάνει την προσθήκη εφέ όπως fade-in ή κινούμενα σχέδια εμφάνισης. Ας αναλύσουμε τη διαδικασία σε διαχειρίσιμα βήματα:

### Βήμα 1: Φόρτωση της παρουσίασής σας

Αρχικά, φορτώστε την υπάρχουσα παρουσίαση PowerPoint που περιέχει το γράφημα στο οποίο θέλετε να προσθέσετε κίνηση.

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Ορίστε αυτό στη διαδρομή του καταλόγου σας
using (Presentation presentation = new Presentation(dataDir + "/ExistingChart.pptx"))
{
    // Αποκτήστε πρόσβαση σε συλλογές διαφανειών και σχημάτων εδώ.
}
```

### Βήμα 2: Πρόσβαση σε συλλογές διαφανειών και σχημάτων

Για να χειριστείτε το γράφημα, αποκτήστε πρόσβαση στην επιθυμητή διαφάνεια και τα σχήματά της.

```csharp
var slide = presentation.Slides[0] as Slide;
var shapes = slide.Shapes as ShapeCollection;
```

### Βήμα 3: Ανάκτηση του αντικειμένου γραφήματος

Προσδιορίστε και ανακτήστε το αντικείμενο του γραφήματος από τη συλλογή σχημάτων. Τα γραφήματα συνήθως αποθηκεύονται σε `IChart` αντικείμενα.

```csharp
var chart = shapes[0] as IChart; // Υποθέτοντας ότι είναι το πρώτο σχήμα
```

### Βήμα 4: Προσθήκη εφέ εξασθένισης στο γράφημα

Για να δημιουργήσετε μια διακριτική είσοδο, προσθέστε ένα εφέ fade που ενεργοποιείται μετά από τυχόν προηγούμενες κινούμενες εικόνες.

```csharp
slide.Timeline.MainSequence.AddEffect(chart, EffectType.Fade, EffectSubtype.None, EffectTriggerType.AfterPrevious);
```

### Βήμα 5: Κίνηση σειράς με εφέ εμφάνισης

Επαναλάβετε κάθε σειρά και εφαρμόστε μια κινούμενη εικόνα εμφάνισης για ένα δυναμικό εφέ αποκάλυψης.

```csharp
Sequence mainSequence = (Sequence)slide.Timeline.MainSequence;
for (int i = 0; i < 4; i++)
{
    mainSequence.AddEffect(chart, EffectChartMajorGroupingType.BySeries, i,
        EffectType.Appear, EffectSubtype.None, EffectTriggerType.AfterPrevious);
}
```

### Βήμα 6: Αποθήκευση της παρουσίασης

Τέλος, αποθηκεύστε την παρουσίασή σας με τις νέες κινούμενες εικόνες που προστέθηκαν.

```csharp
presentation.Save(dataDir + "/AnimatingSeries_out.pptx", SaveFormat.Pptx);
```

## Οδηγός Εφαρμογής
Θα αναλύσουμε τη διαδικασία δημιουργίας ενός γραφήματος box-and-whisker χρησιμοποιώντας το Aspose.Slides σε διαχειρίσιμες ενότητες.

### Δημιουργία ενός γραφήματος Box-and-Whisker
#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να δημιουργήσετε μέσω προγραμματισμού ένα λεπτομερές γράφημα box-and-whisker στο PowerPoint, πλήρες με προσαρμοσμένα δεδομένα και διαμορφώσεις.

#### Βήμα προς βήμα εφαρμογή
##### 1. Ορισμός καταλόγου εγγράφων
Ξεκινήστε καθορίζοντας τον κατάλογο όπου βρίσκεται ή θα αποθηκευτεί το αρχείο της παρουσίασής σας:
```csharp
string dataDir = @"YOUR_DOCUMENT_DIRECTORY";
```
Αυτή η διαδρομή διασφαλίζει ότι το σενάριό σας γνωρίζει από πού να διαβάσει ή να γράψει σε αρχεία.

##### 2. Φόρτωση ή δημιουργία παρουσίασης
Ανοίξτε μια υπάρχουσα παρουσίαση PowerPoint ή δημιουργήστε μια νέα, εάν είναι απαραίτητο:
```csharp
using (Presentation pres = new Presentation(dataDir + "test.pptx"))
{
    // Ο κώδικας για την προσθήκη και τη διαμόρφωση του γραφήματος βρίσκεται εδώ.
}
```
##### 3. Προσθήκη γραφήματος Box-and-Whisker στη διαφάνεια
Εισαγωγή ενός γραφήματος box-and-whisker στην πρώτη διαφάνεια στη θέση `(50, 50)` με διαστάσεις `500 x 400`:
```csharp
IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.BoxAndWhisker, 50, 50, 500, 400);
```
Αυτό το βήμα περιλαμβάνει την επιλογή της επιθυμητής διαφάνειας και τη διαμόρφωση της αρχικής τοποθέτησης του γραφήματός σας.
##### 4. Εκκαθάριση υπαρχόντων δεδομένων
Καταργήστε τυχόν υπάρχουσες κατηγορίες ή σειρές για να ξεκινήσετε από την αρχή:
```csharp
chart.ChartData.Categories.Clear();
chart.ChartData.Series.Clear();
```
Η εκκαθάριση διασφαλίζει ότι δεν θα αντιγράψετε κατά λάθος δεδομένα κατά την προσθήκη νέων καταχωρίσεων.
##### 5. Βιβλίο εργασίας γραφημάτων Access
Χρησιμοποιήστε το βιβλίο εργασίας που σχετίζεται με τα δεδομένα του γραφήματός σας για περαιτέρω χειρισμό:
```csharp
IChartDataWorkbook wb = chart.ChartData.ChartDataWorkbook;
```
Το βιβλίο εργασίας λειτουργεί ως κοντέινερ όπου μπορείτε να προσθέσετε ή να τροποποιήσετε δεδομένα γραφήματος μέσω προγραμματισμού.
##### 6. Εκκαθάριση δεδομένων βιβλίου εργασίας
Βεβαιωθείτε ότι δεν υπάρχουν περισσεύματα κελιών, διαγράφοντας τα από το αρχικό ευρετήριο:
```csharp
wb.Clear(0);
```
##### 7. Προσθήκη κατηγοριών στο γράφημα
Επαναλάβετε και συμπληρώστε τις κατηγορίες για το γράφημά σας, προσθέτοντας την καθεμία ως νέα γραμμή στη στήλη Α:
```csharp
for (int i = 1; i <= 6; i++)
{
    chart.ChartData.Categories.Add(wb.GetCell(0, "A" + i, "Category 1"));
}
```
Αυτό το βήμα σάς επιτρέπει να οργανώσετε συστηματικά τις κατηγορίες δεδομένων σας μέσα στο γράφημα.

#### Βασικές επιλογές διαμόρφωσης
- **Τύπος γραφήματος:** Επιλέγω `ChartType.BoxAndWhisker` για τη δημιουργία διαγραμμάτων κουτιού και μουστακιού.
- **Τοποθέτηση και Διαστασιολόγηση:** Προσαρμογή θέσης `(50, 50)` και μέγεθος `(500, 400)` με βάση τις απαιτήσεις διάταξης διαφανειών.
- **Διαχείριση Δεδομένων:** Χρησιμοποιήστε το βιβλίο εργασίας για να διαχειριστείτε δεδομένα αποτελεσματικά.

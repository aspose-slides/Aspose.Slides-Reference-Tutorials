## Οδηγός Εφαρμογής

### Προσθήκη ενός γραφήματος PieOfPie στην παρουσίασή σας

#### Επισκόπηση

Αυτή η ενότητα δείχνει πώς να δημιουργήσετε και να προσθέσετε ένα γράφημα PieOfPie στη διαφάνεια του PowerPoint χρησιμοποιώντας το Aspose.Slides.

#### Οδηγίες βήμα προς βήμα

**1. Αρχικοποίηση της παρουσίασης**

Δημιουργήστε μια παρουσία του `Presentation` τάξη:

```csharp
using Aspose.Slides;

Presentation presentation = new Presentation();
```

**2. Προσθέστε ένα γράφημα PieOfPie**

Εισαγάγετε το γράφημα στην επιθυμητή θέση και στις επιθυμητές διαστάσεις στην πρώτη διαφάνεια:

```csharp
using Aspose.Slides.Charts;

IChart chart = presentation.Slides[0].Shapes.AddChart(ChartType.PieOfPie, 50, 50, 500, 400);
```

**3. Αποθηκεύστε την παρουσίασή σας**

Αποθηκεύστε το αρχείο σας σε μορφή PPTX αφού προσθέσετε το γράφημα:

```csharp
using Aspose.Slides.Export;

presentation.Save("YOUR_OUTPUT_DIRECTORY/SecondPlotOptionsforCharts_out.pptx", SaveFormat.Pptx);
```

### Ρύθμιση παραμέτρων ετικετών δεδομένων γραφήματος και ιδιοτήτων ομάδας σειρών

#### Επισκόπηση

Βελτιώστε το γράφημά σας διαμορφώνοντας ετικέτες δεδομένων και ιδιότητες ομάδας σειρών για καλύτερη οπτικοποίηση.

**1. Ορισμός μορφής ετικέτας δεδομένων**

Εμφάνιση τιμών στην πρώτη σειρά:

```csharp
chart.ChartData.Series[0].Labels.DefaultDataLabelFormat.ShowValue = true;
```

**2. Προσαρμόστε το μέγεθος της δεύτερης πίτας**

Ορίστε ένα κατάλληλο μέγεθος για λόγους σαφήνειας:

```csharp
chart.ChartData.Series[0].ParentSeriesGroup.SecondPieSize = 149;
```

**3. Προσαρμόστε τη διαίρεση κατά ποσοστό και θέση**

Βελτιστοποίηση διαχωρισμού δεδομένων εντός του γραφήματος:

```csharp
chart.ChartData.Series[0].ParentSeriesGroup.PieSplitBy = PieSplitType.ByPercentage;
chart.ChartData.Series[0].ParentSeriesGroup.PieSplitPosition = 53;
```

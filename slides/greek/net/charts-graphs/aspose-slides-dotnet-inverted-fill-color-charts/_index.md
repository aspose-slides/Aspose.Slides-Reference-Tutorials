## Οδηγός Εφαρμογής
Αυτή η ενότητα σας καθοδηγεί στη δημιουργία ενός γραφήματος με ανεστραμμένα χρώματα γεμίσματος για αρνητικές τιμές χρησιμοποιώντας το Aspose.Slides. Κάθε χαρακτηριστικό αναλύεται βήμα προς βήμα για να διασφαλιστεί η σαφήνεια και η ευκολία κατανόησης.
### Δημιουργία νέας παρουσίασης
Ξεκινήστε αρχικοποιώντας ένα νέο `Presentation` παράδειγμα:
```csharp
using (Presentation pres = new Presentation())
{
    // Τα επόμενα βήματα θα εκτελεστούν εντός αυτού του μπλοκ.
}
```
### Προσθήκη γραφήματος ομαδοποιημένων στηλών
Προσθέστε ένα γράφημα ομαδοποιημένων στηλών στην πρώτη διαφάνεια και διαμορφώστε τις διαστάσεις του:
```csharp
IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 100, 100, 400, 300);
// Αυτή η γραμμή προσθέτει ένα νέο διάγραμμα στη θέση (100, 100) με πλάτος 400 και ύψος 300.
```
### Πρόσβαση στο βιβλίο εργασίας δεδομένων γραφήματος
Για να χειριστείτε τα δεδομένα μέσα στο γράφημά σας, αποκτήστε πρόσβαση στο βιβλίο εργασίας του:
```csharp
IChartDataWorkbook workBook = chart.ChartData.ChartDataWorkbook;
```
Αυτό το βήμα είναι κρίσιμο για την προσθήκη και τροποποίηση σειρών και κατηγοριών.
### Διαγραφή υπαρχουσών σειρών και κατηγοριών
Εξασφαλίστε μια καθαρή σελίδα διαγράφοντας τα υπάρχοντα δεδομένα γραφήματος:
```csharp
chart.ChartData.Series.Clear();
chart.ChartData.Categories.Clear();
// Αυτό διασφαλίζει ότι τυχόν προηγούμενα δεδομένα δεν επηρεάζουν τη νέα ρύθμιση.
```
### Προσθήκη νέων σειρών και κατηγοριών
Ορίστε τη δομή των δεδομένων σας προσθέτοντας σειρές και κατηγορίες:
```csharp
chart.ChartData.Series.Add(workBook.GetCell(0, 0, 1, "Series 1"), chart.Type);
chart.ChartData.Categories.Add(workBook.GetCell(0, 1, 0, "Category 1"));
chart.ChartData.Categories.Add(workBook.GetCell(0, 2, 0, "Category 2"));
chart.ChartData.Categories.Add(workBook.GetCell(0, 3, 0, "Category 3"));
// Αυτή η ρύθμιση παρέχει ένα πλαίσιο για την εισαγωγή σημείων δεδομένων.
```
### Συμπλήρωση σημείων δεδομένων σειράς
Εισαγάγετε δεδομένα στη σειρά του γραφήματός σας:
```csharp
IChartSeries series = chart.ChartData.Series[0];
series.DataPoints.AddDataPointForBarSeries(workBook.GetCell(0, 1, 1, -20));
series.DataPoints.AddDataPointForBarSeries(workBook.GetCell(0, 2, 1, 50));
series.DataPoints.AddDataPointForBarSeries(workBook.GetCell(0, 3, 1, -30));
// Αυτά τα σημεία δεδομένων απεικονίζουν αρνητικές και θετικές τιμές.
```
### Ρύθμιση παραμέτρων χρώματος ανεστραμμένου γεμίσματος για αρνητικές τιμές
Προσαρμόστε την εμφάνιση των αρνητικών τιμών στο γράφημά σας:
```csharp
var seriesColor = series.GetAutomaticSeriesColor();
series.InvertIfNegative = true;
series.Format.Fill.FillType = FillType.Solid;
series.Format.Fill.SolidFillColor.Color = seriesColor;
series.InvertedSolidFillColor.Color = Color.Red; // Ορίστε αυτό σε οποιοδήποτε χρώμα προτιμάτε για αρνητικές τιμές.
```
Αυτό το βήμα βελτιώνει την ορατότητα των δεδομένων διαφοροποιώντας τις αρνητικές τιμές με ένα ξεχωριστό χρώμα γεμίσματος.
### Αποθήκευση της παρουσίασης
Τέλος, αποθηκεύστε το αρχείο παρουσίασής σας:
```csharp
pres.Save("YOUR_DOCUMENT_DIRECTORY/SetInvertFillColorChart_out.pptx", SaveFormat.Pptx);
// Αντικαταστήστε το YOUR_DOCUMENT_DIRECTORY με την πραγματική διαδρομή καταλόγου σας.
```

## Ενότητα Συχνών Ερωτήσεων
1. **Πώς μπορώ να χειριστώ πολλαπλές σειρές σε ένα γράφημα;**
   - Προσθέστε κάθε σειρά χρησιμοποιώντας `chart.ChartData.Series.Add` και συμπληρώστε με μεμονωμένα σημεία δεδομένων όπως φαίνεται παραπάνω.
2. **Μπορώ να προσαρμόσω το χρώμα και για θετικές τιμές;**
   - Ναι, τροποποίηση `series.Format.Fill.SolidFillColor.Color` για να ορίσετε ένα συγκεκριμένο χρώμα για όλες τις μη αρνητικές τιμές.
3. **Τι γίνεται αν το γράφημά μου δεν εμφανίζει σωστά τις αρνητικές τιμές;**
   - Εξασφαλίζω `InvertIfNegative` έχει οριστεί σε true και ελέγξτε ότι στα σημεία δεδομένων σας έχουν αντιστοιχιστεί σωστά αρνητικές τιμές.
4. **Πώς μπορώ να αποθηκεύσω παρουσιάσεις σε διαφορετικές μορφές;**
   - Χρησιμοποιήστε την κατάλληλη τιμή από το `SaveFormat` απαρίθμηση κατά την κλήση `Save`.
5. **Υπάρχει τρόπος να αυτοματοποιήσω τις ενημερώσεις γραφημάτων με δεδομένα σε πραγματικό χρόνο;**
   - Ενώ το Aspose.Slides δεν υποστηρίζει τη σύνδεση δεδομένων σε πραγματικό χρόνο, μπορείτε να ενημερώσετε τα γραφήματα μέσω προγραμματισμού τροποποιώντας σημεία δεδομένων και αποθηκεύοντας τις αλλαγές.

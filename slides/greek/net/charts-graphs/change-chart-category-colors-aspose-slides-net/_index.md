## Οδηγός Εφαρμογής

### Αλλαγή χρωμάτων κατηγορίας γραφήματος

Ας επικεντρωθούμε στην αλλαγή του χρώματος συγκεκριμένων κατηγοριών γραφημάτων. Αυτή η λειτουργία βελτιώνει την οπτικοποίηση των δεδομένων σας, επιτρέποντάς σας να επισημάνετε βασικά σημεία δεδομένων με διαφορετικά χρώματα.

#### Προσθήκη γραφήματος στη διαφάνειά σας

Αρχικά, προσθέστε ένα γράφημα στη διαφάνεια της παρουσίασής σας:

```csharp
// Προσθήκη γραφήματος ομαδοποιημένων στηλών στην πρώτη διαφάνεια
IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 50, 50, 600, 400);
```

#### Πρόσβαση σε σημεία δεδομένων

Στη συνέχεια, αποκτήστε πρόσβαση και τροποποιήστε μεμονωμένα σημεία δεδομένων:

```csharp
// Πρόσβαση στο πρώτο σημείο δεδομένων στην πρώτη σειρά του γραφήματος
IChartDataPoint point = chart.ChartData.Series[0].DataPoints[0];

// Ορίστε τον τύπο γεμίσματος σε συμπαγές για καλύτερη ορατότητα χρώματος
point.Format.Fill.FillType = FillType.Solid;

// Αλλάξτε το χρώμα σε μπλε για οπτική έμφαση
point.Format.Fill.SolidFillColor.Color = Color.Blue;
```

#### Αποθήκευση της παρουσίασής σας

Τέλος, αποθηκεύστε την τροποποιημένη παρουσίασή σας:

```csharp
// Αποθήκευση της παρουσίασης με τις αλλαγές
pres.Save("YOUR_DOCUMENT_DIRECTORY/output.pptx", SaveFormat.Pptx);
```

## Οδηγός Εφαρμογής

Τώρα που είστε έτοιμοι, ας εξερευνήσουμε τη διαδικασία βήμα προς βήμα για να δημιουργήσετε ένα γράφημα μετοχών χρησιμοποιώντας το Aspose.Slides .NET.

### Δημιουργία γραφήματος μετοχών

#### Επισκόπηση

Η δημιουργία ενός γραφήματος μετοχών περιλαμβάνει την αρχικοποίηση ενός αντικειμένου παρουσίασης, την προσθήκη ενός νέου γραφήματος σε μια διαφάνεια και τη διαμόρφωσή του με τα απαραίτητα σημεία δεδομένων για τιμές ανοίγματος, υψηλής, χαμηλής και κλεισίματος.

#### Βήμα 1: Αρχικοποίηση παρουσίασης και προσθήκη γραφήματος

Ξεκινήστε δημιουργώντας ένα `Presentation` αντικείμενο και προσθέστε ένα γράφημα μετοχών στην πρώτη διαφάνεια:

```csharp
string dataDir = @"YOUR_DOCUMENT_DIRECTORY";

using (Presentation pres = new Presentation())
{
    IChart chart = pres.Slides[0].Shapes.AddChart(
        ChartType.OpenHighLowClose, 50, 50, 600, 400, false);
}
```

#### Βήμα 2: Διαγραφή υπαρχουσών σειρών και κατηγοριών

Βεβαιωθείτε ότι το γράφημα είναι έτοιμο για νέα δεδομένα, διαγράφοντας τις υπάρχουσες σειρές και κατηγορίες:

```csharp
chart.ChartData.Series.Clear();
chart.ChartData.Categories.Clear();

IChartDataWorkbook wb = chart.ChartData.ChartDataWorkbook;
```

#### Βήμα 3: Προσθήκη κατηγοριών και σειρών

Προσθέστε τις απαραίτητες κατηγορίες (A, B, C) και σειρές για τιμές ανοίγματος, υψηλής, χαμηλής και κλεισίματος:

```csharp
// Προσθήκη κατηγοριών
chart.ChartData.Categories.Add(wb.GetCell(0, 1, 0, "A"));
chart.ChartData.Categories.Add(wb.GetCell(0, 2, 0, "B"));
chart.ChartData.Categories.Add(wb.GetCell(0, 3, 0, "C"));

// Προσθήκη σειράς
chart.ChartData.Series.Add(wb.GetCell(0, 0, 1, "Open"), chart.Type);
chart.ChartData.Series.Add(wb.GetCell(0, 0, 2, "High"), chart.Type);
chart.ChartData.Series.Add(wb.GetCell(0, 0, 3, "Low"), chart.Type);
chart.ChartData.Series.Add(wb.GetCell(0, 0, 4, "Close"), chart.Type);
```

#### Βήμα 4: Προσθήκη σημείων δεδομένων για κάθε σειρά

Εισαγάγετε σημεία δεδομένων σε κάθε σειρά με την ακόλουθη προσέγγιση:

```csharp
// Ανοιχτά σημεία δεδομένων σειράς
IChartSeries openSeries = chart.ChartData.Series[0];
openSeries.DataPoints.AddDataPointForStockSeries(wb.GetCell(0, 1, 1, 72));
openSeries.DataPoints.AddDataPointForStockSeries(wb.GetCell(0, 2, 1, 25));
openSeries.DataPoints.AddDataPointForStockSeries(wb.GetCell(0, 3, 1, 38));

// Επαναλάβετε για σειρές Υψηλής, Χαμηλής και Κλειστής
IChartSeries highSeries = chart.ChartData.Series[1];
highSeries.DataPoints.AddDataPointForStockSeries(wb.GetCell(0, 1, 2, 172));
highSeries.DataPoints.AddDataPointForStockSeries(wb.GetCell(0, 2, 2, 57));
highSeries.DataPoints.AddDataPointForStockSeries(wb.GetCell(0, 3, 2, 57));

IChartSeries lowSeries = chart.ChartData.Series[2];
lowSeries.DataPoints.AddDataPointForStockSeries(wb.GetCell(0, 1, 3, 12));
lowSeries.DataPoints.AddDataPointForStockSeries(wb.GetCell(0, 2, 3, 12));
lowSeries.DataPoints.AddDataPointForStockSeries(wb.GetCell(0, 3, 3, 13));

IChartSeries closeSeries = chart.ChartData.Series[3];
closeSeries.DataPoints.AddDataPointForStockSeries(wb.GetCell(0, 1, 4, 25));
closeSeries.DataPoints.AddDataPointForStockSeries(wb.GetCell(0, 2, 4, 38));
closeSeries.DataPoints.AddDataPointForStockSeries(wb.GetCell(0, 3, 4, 50));
```

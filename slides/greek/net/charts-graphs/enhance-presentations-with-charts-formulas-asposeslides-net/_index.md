## Οδηγός Εφαρμογής

### Δημιουργία παρουσίασης και προσθήκη γραφήματος
**Επισκόπηση:**
Αυτή η ενότητα εστιάζει στη δημιουργία μιας παρουσίασης PowerPoint και στην ενσωμάτωση ενός γραφήματος ομαδοποιημένων στηλών σε αυτήν. Τα γραφήματα είναι ένας αποτελεσματικός τρόπος για την οπτικοποίηση δεδομένων, καθιστώντας τις παρουσιάσεις σας πιο αποτελεσματικές.

#### Βήμα 1: Ορίστε τη διαδρομή εξόδου
Αρχικά, καθορίστε πού θέλετε να αποθηκεύσετε το αρχείο παρουσίασής σας:
```csharp
string resultPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "CreateChart_out.pptx");
```

#### Βήμα 2: Δημιουργήστε μια παρουσίαση και προσθέστε ένα γράφημα
Στη συνέχεια, δημιουργήστε ένα υπόδειγμα `Presentation` αντικείμενο και προσθέστε ένα γράφημα ομαδοποιημένων στηλών στην πρώτη διαφάνεια.
```csharp
using (Presentation presentation = new Presentation())
{
    IChart s_chart = presentation.Slides[0].Shapes.AddChart(
        ChartType.ClusteredColumn, 10, 10, 600, 300);
}
```
Εδώ, το `AddChart` Οι παράμετροι της μεθόδου καθορίζουν τον τύπο γραφήματος, τη θέση και το μέγεθός του μέσα στη διαφάνεια.

### Ορισμός και υπολογισμός τύπων σε βιβλίο εργασίας δεδομένων γραφήματος
**Επισκόπηση:**
Σε αυτήν την ενότητα, θα δούμε πώς να ορίζουμε τύπους για κελιά μέσα στο βιβλίο εργασίας δεδομένων ενός γραφήματος, να εκτελούμε υπολογισμούς και να ενημερώνουμε τιμές δυναμικά.

#### Βήμα 1: Δημιουργήστε μια παρουσίαση με ένα διάγραμμα
Ξεκινήστε δημιουργώντας μια παρουσία παρουσίασης και προσθέτοντας το αρχικό γράφημα:
```csharp
using (Presentation presentation = new Presentation())
{
    IChart s_chart = presentation.Slides[0].Shapes.AddChart(
        ChartType.ClusteredColumn, 10, 10, 600, 300);
    var workbook = s_chart.ChartData.ChartDataWorkbook;
}
```

#### Βήμα 2: Ορισμός και Υπολογισμός Τύπων
Ορισμός τύπων για συγκεκριμένα κελιά στο βιβλίο εργασίας δεδομένων γραφήματος:
```csharp
// Ορισμός τύπου για το κελί A1
IChartDataCell cellA1 = workbook.GetCell(0, "A1");
cellA1.Formula = "ABS(A2) + MAX(B2:C2)";

// Αντιστοίχιση τιμής στο κελί A2 και υπολογισμός τύπων
workbook.GetCell(0, "A2").Value = -1;
workbook.CalculateFormulas();

// Ορισμός τύπου για το B2 και επανυπολογισμός
workbook.GetCell(0, "B2").Formula = "2";
workbook.CalculateFormulas();

// Ενημέρωση τύπου κελιού A1
cellA1.Formula = "MAX(2:2)";
workbook.CalculateFormulas();
```

### Αποθήκευση της παρουσίασης
**Επισκόπηση:**
Αφού δημιουργήσετε την παρουσίασή σας και ρυθμίσετε τους τύπους γραφήματος, αποθηκεύστε την σε μια καθορισμένη διαδρομή.

#### Βήμα 1: Ορισμός διαδρομής αποθήκευσης
Ορίστε πού θέλετε να αποθηκεύσετε την τελική παρουσίαση:
```csharp
string resultPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "SavePresentation_out.pptx");
```

#### Βήμα 2: Αποθήκευση της παρουσίασης
Τέλος, χρησιμοποιήστε το `Save` μέθοδος για να αποθηκεύσετε την παρουσίασή σας σε μορφή PPTX.
```csharp
using (Presentation presentation = new Presentation())
{
    // Εκτελέστε τη δημιουργία γραφήματος και τον ορισμό τύπου εδώ...
    presentation.Save(resultPath, Aspose.Slides.Export.SaveFormat.Pptx);
}
```

## Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, θα δούμε πώς να δημιουργούμε μια παρουσίαση και να προσθέτουμε ένα γράφημα ομαδοποιημένων στηλών.

### Δημιουργία κενής παρουσίασης

Ξεκινήστε ορίζοντας τη διαδρομή του καταλόγου εγγράφων σας. Εδώ θα αποθηκευτεί η δημιουργημένη παρουσίαση:
```csharp

Presentation pres = new Presentation();
```

### Προσθήκη γραφήματος ομαδοποιημένων στηλών στη διαφάνεια

Στη συνέχεια, προσθέστε ένα γράφημα ομαδοποιημένων στηλών στην πρώτη διαφάνεια στην καθορισμένη θέση και μέγεθος:
```csharp
// Προσθέστε ένα γράφημα ομαδοποιημένων στηλών στο (20, 20) με διαστάσεις (500x400)
IChart chart = pres.Slides[0].Shapes.AddChart(
    ChartType.ClusteredColumn,
    20, 20, 500, 400);
```
**Εξήγηση:** Αυτό το τμήμα κώδικα δημιουργεί μια κενή παρουσίαση και προσθέτει ένα γράφημα ομαδοποιημένων στηλών. `AddChart` Η μέθοδος καθορίζει τον τύπο του γραφήματος (`ClusteredColumn`) και η θέση/τα μεγέθη του (x: 20, y: 20, πλάτος: 500, ύψος: 400).

### Αποθήκευση της παρουσίασης

Τέλος, αποθηκεύστε την παρουσίασή σας για να βεβαιωθείτε ότι όλες οι αλλαγές έχουν αποθηκευτεί:
```csharp
// Αποθηκεύστε την παρουσίαση στον καθορισμένο κατάλογο.
pres.Save(dataDir + "CreateAndAddChart_out.pptx");
```
**Εξήγηση:** Ο `Save` Η μέθοδος γράφει τα δεδομένα της παρουσίασης σε ένα αρχείο. Προσαρμόστε τη διαδρομή όπως απαιτείται για το περιβάλλον σας.

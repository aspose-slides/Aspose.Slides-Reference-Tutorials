## Οδηγός Εφαρμογής
### Προσθήκη γραφήματος φυσαλίδων στην παρουσίασή σας
Για να ξεκινήσετε, θα χρειαστεί να προσθέσετε ένα γράφημα φυσαλίδων στη διαφάνειά σας:

#### Βήμα 1: Δημιουργία ή άνοιγμα παρουσίασης
```csharp
using Aspose.Slides;
using Aspose.Slides.Charts;
// Ορίστε τη διαδρομή καταλόγου για την αποθήκευση εγγράφων
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
// Δημιουργήστε μια νέα παρουσία παρουσίασης
using (Presentation pres = new Presentation())
{
    // Προσθέστε ένα γράφημα φυσαλίδων στην πρώτη διαφάνεια στη θέση (50, 50) με πλάτος και ύψος 600x400 pixel
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.Bubble, 50, 50, 600, 400, true);
```
#### Βήμα 2: Ρύθμιση παραμέτρων αναπαράστασης μεγέθους φυσαλίδας
Ορίστε το μέγεθος του φυσαλίδας ώστε να αντιπροσωπεύει μια συγκεκριμένη διάσταση δεδομένων. Αυτό το παράδειγμα χρησιμοποιεί το `Width` ιδιοκτησία:
```csharp
    // Ορισμός αναπαράστασης μεγέθους φυσαλίδας με βάση το 'Πλάτος'
    chart.ChartData.SeriesGroups[0].BubbleSizeRepresentation = BubbleSizeRepresentationType.Width;
```
#### Βήμα 3: Αποθηκεύστε την παρουσίασή σας
Τέλος, αποθηκεύστε την παρουσίασή σας για να δείτε τις αλλαγές που αντικατοπτρίζονται στα γραφήματά σας.
```csharp
    // Αποθήκευση της τροποποιημένης παρουσίασης
    pres.Save(dataDir + "Presentation_BubbleSizeRepresentation.pptx");
}
```
### Βασικές επιλογές διαμόρφωσης
- **Τύπος αναπαράστασης μεγέθους φυσαλίδας**: Επιλέξτε ανάμεσα `Width`, `Height`, ή `Volume` με βάση τα χαρακτηριστικά των δεδομένων σας.
- **ChartType.Bubble**: Απαραίτητο για τη δημιουργία γραφημάτων φυσαλίδων που μπορούν να αναπαραστήσουν πολλαπλές διαστάσεις δεδομένων.

#### Επισκόπηση

Θα ξεκινήσουμε προσθέτοντας ένα γράφημα στη διαφάνεια της παρουσίασής σας και διαμορφώνοντάς το ώστε να εμφανίζει ημερομηνίες στην επιθυμητή μορφή.

// Ορίστε τον κατάλογο για την αποθήκευση εγγράφων
class Program

### 2. Πρόσβαση και τροποποίηση δεδομένων γραφήματος

#### Επισκόπηση

Θα τροποποιήσουμε το βιβλίο εργασίας δεδομένων γραφήματος για να εισαγάγουμε τιμές ημερομηνίας ως κατηγορίες.

#### Διαγραφή υπαρχουσών κατηγοριών και σειρών

```csharp
// Πρόσβαση στο βιβλίο εργασίας δεδομένων γραφήματος για χειρισμό
class Program
{
    static void Main()
    {
        using (Presentation pres = new Presentation())
        {
            IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.Area, 50, 50, 450, 300);
            IChartDataWorkbook wb = chart.ChartData.ChartDataWorkbook;

            // Διαγραφή υπαρχουσών κατηγοριών και σειρών στα δεδομένα του γραφήματος
            chart.ChartData.Categories.Clear();
            chart.ChartData.Series.Clear();
        }
    }
}
```

#### Προσθήκη τιμών ημερομηνίας ως νέων κατηγοριών

Χρησιμοποιήστε αυτό το απόσπασμα για να εισαγάγετε ημερομηνίες:

```csharp
// Πρόσβαση στο βιβλίο εργασίας δεδομένων γραφήματος για χειρισμό
class Program
{
    static void Main()
    {
        using (Presentation pres = new Presentation())
        {
            IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.Area, 50, 50, 450, 300);
            IChartDataWorkbook wb = chart.ChartData.ChartDataWorkbook;

            // Προσθήκη τιμών ημερομηνίας ως νέες κατηγορίες στο γράφημα
            chart.ChartData.Categories.Add(wb.GetCell(0, "A2", DateTime.Now.AddDays(-30)));
            chart.ChartData.Categories.Add(wb.GetCell(0, "A3", DateTime.Now));
            chart.ChartData.Categories.Add(wb.GetCell(0, "A4", DateTime.Now.AddDays(30)));

            // Προσθήκη σειράς και συμπλήρωση με δεδομένα
            IChartSeries series = chart.ChartData.Series.Add(wb.GetCell(0, "B1", "Sample Series"), chart.Type);
        }
    }
}
```

### 3. Ορίστε προσαρμοσμένη μορφή ημερομηνίας

#### Επισκόπηση

Τώρα, διαμορφώστε τον άξονα κατηγοριών ώστε να εμφανίζει τις ημερομηνίες στην προτιμώμενη μορφή σας.

#### Ρύθμιση παραμέτρων άξονα κατηγορίας

```csharp
// Πρόσβαση στον άξονα κατηγορίας και ορισμός προσαρμοσμένης μορφής ημερομηνίας
class Program
{
    static void Main()
    {
        using (Presentation pres = new Presentation())
        {
            IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.Area, 50, 50, 450, 300);
            IChartDataWorkbook wb = chart.ChartData.ChartDataWorkbook;

            // Προσθήκη τιμών ημερομηνίας ως νέες κατηγορίες στο γράφημα
            chart.ChartData.Categories.Add(wb.GetCell(0, "A2", DateTime.Now.AddDays(-30)));
            chart.ChartData.Categories.Add(wb.GetCell(0, "A3", DateTime.Now));
            chart.ChartData.Categories.Add(wb.GetCell(0, "A4", DateTime.Now.AddDays(30)));

            // Προσθήκη σειράς και συμπλήρωση με δεδομένα
            IChartSeries series = chart.ChartData.Series.Add(wb.GetCell(0, "B1", "Sample Series"), chart.Type);

            // Πρόσβαση στον άξονα κατηγορίας και ορισμός προσαρμοσμένης μορφής ημερομηνίας
            IAxis categoryAxis = chart.Axes.HorizontalAxis;
            categoryAxis.MajorUnit = 1; // Ορισμός κύριας μονάδας ως ημέρες
            categoryAxis.NumberFormat.FormatCode = "dd-MMM"; // Προσαρμοσμένη μορφή: συντομογραφία ημέρας-μήνα

            // Αποθήκευση της παρουσίασης με τις αλλαγές
            pres.Save(@"YOUR_DOCUMENT_DIRECTORY\FormattedChart.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
        }
    }
}
```

#### Παράμετροι και μέθοδοι Επεξήγηση
- **Κύρια Μονάδα**: Ορίζει το διάστημα για τα κύρια τικ στον άξονα.
- **NumberFormat.FormatCode**: Ορίζει τον τρόπο εμφάνισης των ημερομηνιών. Η μορφή `"dd-MMM"` εμφανίζει τη συντομογραφία ημέρας και μήνα.

### Συμβουλές αντιμετώπισης προβλημάτων

1. Βεβαιωθείτε ότι η άδεια χρήσης Aspose.Slides έχει ρυθμιστεί σωστά για να αποφύγετε περιορισμούς στη λειτουργικότητα.
2. Επαληθεύστε τις τιμές και τις μορφές ημερομηνίας, ειδικά όταν πρόκειται για διαφορετικές τοπικές ρυθμίσεις ή περιφερειακές ρυθμίσεις.

## Οδηγός Εφαρμογής

Τώρα που έχουμε ρυθμίσει τα πάντα, ας εμβαθύνουμε στην εφαρμογή της προσαρμογής ετικετών γραφημάτων χρησιμοποιώντας το Aspose.Slides για .NET.

### Χαρακτηριστικό: Εμφάνιση ετικετών γραφήματος
#### Επισκόπηση:
Αυτή η λειτουργία δείχνει πώς να προσαρμόσετε και να εμφανίσετε διαφορετικούς τύπους ετικετών στα γραφήματά σας μέσα σε παρουσιάσεις PowerPoint. Σας επιτρέπει να εμφανίζετε τιμές απευθείας στις ετικέτες ή να τις μορφοποιείτε ως επεξηγήσεις δεδομένων, βελτιώνοντας τη σαφήνεια και τον επαγγελματισμό των διαφανειών της παρουσίασής σας.

#### Προσθήκη κυκλικού διαγράμματος:
1. **Δημιουργία αντικειμένου παρουσίασης**: 
   Ξεκινήστε δημιουργώντας ένα νέο `Presentation` αντικείμενο όπου θα προσθέσουμε το γράφημά μας.
   ```csharp
   using (Presentation presentation = new Presentation())
   {
       // Ο κωδικός σας πηγαίνει εδώ
   }
   ```
2. **Προσθήκη κυκλικού γραφήματος**: 
   Εισαγωγή κυκλικού γραφήματος στη θέση `(50, 50)` με διαστάσεις `500x400`.
   ```csharp
   IChart chart = presentation.Slides[0].Shapes.AddChart(ChartType.Pie, 50, 50, 500, 400);
   ```

#### Προσαρμογή ετικετών γραφημάτων:
3. **Δεδομένα σειράς Access**: 
   Αποκτήστε πρόσβαση στην πρώτη σειρά δεδομένων στο γράφημα πίτας σας.
   ```csharp
   var series = chart.ChartData.Series[0];
   ```
4. **Ορισμός προεπιλεγμένων μορφών ετικετών**: 
   Προσαρμόστε τις προεπιλεγμένες ρυθμίσεις ετικετών για να εμφανίσετε τιμές και να τις μορφοποιήσετε ως επεξηγήσεις.
   ```csharp
   // Εμφάνιση τιμής σε όλες τις ετικέτες
   series.Labels.DefaultDataLabelFormat.ShowValue = true;

   // Χρήση επεξηγήσεων δεδομένων από προεπιλογή
   series.Labels.DefaultDataLabelFormat.ShowLabelAsDataCallout = true;
   ```
5. **Παράκαμψη συγκεκριμένης μορφής ετικέτας**: 
   Για παράδειγμα, αν θέλετε να προσαρμόσετε την τρίτη ετικέτα διαφορετικά:
   ```csharp
   // Να μην εμφανίζεται αυτό ως επεξήγηση δεδομένων
   series.Labels[2].DataLabelFormat.ShowLabelAsDataCallout = false;
   ```
6. **Αποθήκευση της παρουσίασής σας**: 
   Τέλος, αποθηκεύστε την παρουσίασή σας με όλες τις προσαρμογές.
   ```csharp
   string outputDir = "YOUR_OUTPUT_DIRECTORY";
   presentation.Save(outputDir + "DisplayChartLabels_out.pptx", SaveFormat.Pptx);
   ```

### Συμβουλές αντιμετώπισης προβλημάτων:
- Εξασφαλίστε διαδρομές για `dataDir` και `outputDir` έχουν ρυθμιστεί σωστά για την αποφυγή σφαλμάτων "δεν βρέθηκε αρχείο".
- Εάν οι ετικέτες δεν εμφανίζονται, επαληθεύστε ότι η σειρά έχει συμπληρωμένα σημεία δεδομένων.

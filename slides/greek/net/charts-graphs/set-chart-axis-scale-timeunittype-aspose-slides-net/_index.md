## Οδηγός Εφαρμογής
### Χρήση της απαρίθμησης TimeUnitType για την κλιμάκωση των αξόνων του γραφήματος
Αυτή η ενότητα παρουσιάζει τον τρόπο χρήσης του `TimeUnitType` απαρίθμηση για τον ορισμό της κλίμακας άξονα του γραφήματός σας.

#### Βήμα 1: Δημιουργία αντικειμένου παρουσίασης
Ξεκινήστε δημιουργώντας μια παρουσία του `Presentation` τάξη:
```csharp
// Αρχικοποίηση αντικειμένου παρουσίασης
var presentation = new Presentation();
```
*Γιατί αυτό το βήμα; Ρυθμίζει το βασικό περιβάλλον για τον χειρισμό διαφανειών και γραφημάτων.*

#### Βήμα 2: Προσθήκη διαφάνειας γραφήματος
Προσθέστε μια διαφάνεια με ένα γράφημα χρησιμοποιώντας το ακόλουθο απόσπασμα κώδικα:
```csharp
// Πρόσβαση στην πρώτη διαφάνεια
ISlide slide = presentation.Slides[0];

// Προσθήκη γραφήματος με προεπιλεγμένα δεδομένα
IChart chart = slide.Shapes.AddChart(ChartType.ClusteredColumn, 50, 50, 450, 300);
```
*Γιατί αυτό το βήμα; Χρειάζεστε ένα γράφημα για να εφαρμόσετε τις ρυθμίσεις TimeUnitType.*

#### Βήμα 3: Ρύθμιση κλίμακας άξονα χρησιμοποιώντας τον τύπο μονάδας χρόνου
Ορίστε το `MajorUnitScale` του άξονά σας χρησιμοποιώντας την απαρίθμηση TimeUnitType:
```csharp
// Λήψη άξονα Χ (Κατηγορία) από την πρώτη σειρά του γραφήματος
IAxis xAxis = chart.Axes.HorizontalAxis;

// Ορισμός κλίμακας κύριας μονάδας σε ημέρες
xAxis.MajorUnitScale = TimeUnitType.Days;
```
*Γιατί αυτό το βήμα; Προσαρμογή του `MajorUnitScale` σας επιτρέπει να αναπαραστήσετε τον χρόνο με ακρίβεια στον άξονα Χ.*

#### Συμβουλές αντιμετώπισης προβλημάτων
- **Μη έγκυρη Μονάδα Χρόνου:** Βεβαιωθείτε ότι χρησιμοποιείται μια έγκυρη τιμή TimeUnitType. Η απαρίθμηση υποστηρίζει διάφορες κλίμακες, όπως Ημέρες ή Εβδομάδες.

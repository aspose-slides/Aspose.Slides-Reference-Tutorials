### Προσθήκη γραφήματος ομαδοποιημένων στηλών με αυτόματο χρώμα γεμίσματος σειράς
#### Επισκόπηση
Δημιουργήστε ένα γράφημα ομαδοποιημένων στηλών στην παρουσίασή σας, ρυθμίζοντάς το ώστε να προσδιορίζει αυτόματα τα χρώματα σειράς για βελτιωμένη αισθητική και αποτελεσματικότητα.

#### Βήμα 1: Δημιουργία νέας παρουσίασης
Αρχικοποίηση νέου `Presentation` αντικείμενο:
```csharp
using Aspose.Slides;
using Aspose.Slides.Charts;
// Καθορίστε τη διαδρομή του καταλόγου εγγράφων σας
cstring dataDir = "YOUR_DOCUMENT_DIRECTORY";
using (Presentation presentation = new Presentation()) {
    // Προχωρήστε στην προσθήκη ενός γραφήματος στα επόμενα βήματα...
}
```

#### Βήμα 2: Προσθήκη γραφήματος ομαδοποιημένων στηλών
Προσθέστε ένα γράφημα ομαδοποιημένων στηλών στη θέση (100, 50) με διαστάσεις (600x400):
```csharp
// Προσθήκη ενός γραφήματος ομαδοποιημένων στηλών\IChart chart = presentation.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 100, 50, 600, 400);
```

#### Βήμα 3: Ρύθμιση παραμέτρων αυτόματου χρώματος σειράς

```csharp
// Βρόχος πάνω από κάθε σειρά για αυτόματη ρύθμιση χρώματος
type IChartSeries series;
for (int i = 0; i < chart.ChartData.Series.Count; i++) {
    series = chart.ChartData.Series[i];
    // Ορίστε αυτόματα το χρώμα της σειράς
    series.Format.Fill.FillType = FillType.Solid;
    series.Format.Fill.SolidFillColor.Color = Color.FromArgb(255, GetRandomColor());
}
```

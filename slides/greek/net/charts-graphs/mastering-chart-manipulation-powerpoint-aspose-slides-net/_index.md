## Οδηγός Εφαρμογής

### Εξαγωγή εικόνων γραφημάτων από παρουσιάσεις PowerPoint

#### Επισκόπηση
Η εξαγωγή μιας εικόνας γραφήματος είναι πολύτιμη κατά την κοινή χρήση ή την αρχειοθέτηση συγκεκριμένων απεικονίσεων δεδομένων ανεξάρτητα από την παρουσίαση πηγής τους. 

**Βήμα 1: Φόρτωση της παρουσίασής σας**
Ξεκινήστε φορτώνοντας το υπάρχον αρχείο PowerPoint:
```csharp
using (Presentation pres = new Presentation("YOUR_DOCUMENT_DIRECTORY/test.pptx"))
{
    // Συνέχεια επεξεργασίας...
}
```
Αντικαθιστώ `"YOUR_DOCUMENT_DIRECTORY"` με τη διαδρομή όπου είναι αποθηκευμένο το έγγραφό σας.

**Βήμα 2: Πρόσβαση στην επιθυμητή διαφάνεια και διάγραμμα**
Αποκτήστε πρόσβαση σε μια συγκεκριμένη διαφάνεια και γράφημα χρησιμοποιώντας ευρετήρια:
```csharp
ISlide slide = pres.Slides[0]; // Πρώτη διαφάνεια
IChart chart = (IChart)slide.Shapes[1]; // Υποθέτει ότι το διάγραμμα είναι δεύτερο σχήμα
```

**Βήμα 3: Ανάκτηση της εικόνας του γραφήματος**
Χρησιμοποιήστε το `GetImage` μέθοδος για την εξαγωγή μιας αναπαράστασης εικόνας:
```csharp
IImage img = chart.GetImage();
img.Save("YOUR_OUTPUT_DIRECTORY/image.png", Aspose.Slides.Export.ImageFormat.Png);
```
Αυτό αποθηκεύει το εξαγόμενο γράφημα ως αρχείο PNG. Προσαρμόστε τη διαδρομή και τη μορφή εξόδου όπως απαιτείται.

### Προσθήκη διαφορετικών τύπων γραφημάτων στο PowerPoint

#### Επισκόπηση
Η προσθήκη ποικίλων γραφημάτων εμπλουτίζει την παρουσίασή σας, προσφέροντας πολλαπλές προοπτικές στα δεδομένα.

**Βήμα 1: Δημιουργία νέας παρουσίασης**
Ξεκινήστε με μια κενή ή υπάρχουσα παρουσίαση:
```csharp
using (Presentation pres = new Presentation())
{
    ISlide slide = pres.Slides[0]; // Πρόσβαση στην πρώτη διαφάνεια
```

**Βήμα 2: Προσθήκη διαφόρων τύπων γραφημάτων**
Προσθέστε διαφορετικούς τύπους γραφημάτων, όπως ομαδοποιημένες στήλες και γραφήματα πίτας:
```csharp
IChart chart1 = slide.Shapes.AddChart(ChartType.ClusteredColumn, 50, 50, 300, 200);
IChart chart2 = slide.Shapes.AddChart(ChartType.Pie, 400, 50, 300, 200);
```

**Βήμα 3: Αποθήκευση της ενημερωμένης παρουσίασης**
Αποθηκεύστε την παρουσίαση αφού προσθέσετε τα γραφήματά σας:
```csharp
pres.Save("YOUR_DOCUMENT_DIRECTORY/ChartsPresentation.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
```

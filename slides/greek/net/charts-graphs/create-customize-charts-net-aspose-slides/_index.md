# Πώς να δημιουργήσετε και να προσαρμόσετε γραφήματα σε παρουσιάσεις .NET χρησιμοποιώντας το Aspose.Slides για .NET

## Εισαγωγή

Τα γραφήματα είναι ζωτικής σημασίας εργαλεία για τη σαφή και συνοπτική μεταφορά σύνθετων δεδομένων. Αυτό το σεμινάριο σας καθοδηγεί στη δημιουργία δυναμικών γραφημάτων σε παρουσιάσεις .NET χρησιμοποιώντας το Aspose.Slides για .NET—μια ισχυρή βιβλιοθήκη που απλοποιεί τις εργασίες αυτοματοποίησης εγγράφων.

**Τι θα μάθετε:**
- Ρύθμιση του Aspose.Slides για .NET
- Δημιουργία παρουσίασης με γράφημα ομαδοποιημένων στηλών
- Μορφοποίηση σημείων δεδομένων μέσα στα γραφήματά σας

Μέχρι το τέλος αυτού του σεμιναρίου, θα έχετε αποκτήσει πρακτική εμπειρία στη δημιουργία και προσαρμογή γραφημάτων σε παρουσιάσεις .NET χρησιμοποιώντας το Aspose.Slides.

## Ρύθμιση του Aspose.Slides για .NET

### Βήματα εγκατάστασης:

#### Χρησιμοποιώντας το .NET CLI:
```bash
dotnet add package Aspose.Slides
```

#### Χρήση της Κονσόλας Διαχείρισης Πακέτων:
```powershell
Install-Package Aspose.Slides
```

#### Διεπαφή χρήστη του διαχειριστή πακέτων NuGet:
Αναζητήστε το "Aspose.Slides" και εγκαταστήστε την πιο πρόσφατη έκδοση.

### Απόκτηση Άδειας
Για να χρησιμοποιήσετε όλες τις λειτουργίες του Aspose.Slides, χρειάζεστε μια άδεια χρήσης. Μπορείτε να την αποκτήσετε μέσω:
- **Δωρεάν δοκιμή:** Ξεκινήστε με μια προσωρινή δωρεάν δοκιμή για να εξερευνήσετε βασικές λειτουργίες.
- **Προσωρινή Άδεια:** Αποκτήστε μια προσωρινή άδεια για πλήρη πρόσβαση χωρίς περιορισμούς κατά την αξιολόγηση.
- **Αγορά:** Για τρέχοντα έργα, σκεφτείτε να αγοράσετε μια συνδρομή.

### Βασική Αρχικοποίηση
Για να αρχικοποιήσετε το Aspose.Slides στο έργο σας, συμπεριλάβετε τον χώρο ονομάτων και δημιουργήστε ένα αντίγραφο ενός `Presentation` αντικείμενο:

## Οδηγός Εφαρμογής
Θα δούμε πώς να δημιουργούμε παρουσιάσεις και να προσθέτουμε γραφήματα με το Aspose.Slides για .NET.

### Χαρακτηριστικό 1: Δημιουργία παρουσίασης και προσθήκη γραφήματος

#### Επισκόπηση:
Αυτή η λειτουργία δείχνει πώς να δημιουργήσετε μια παρουσίαση και να προσθέσετε ένα γράφημα ομαδοποιημένων στηλών στην πρώτη διαφάνεια. Τα γραφήματα είναι απαραίτητα για την αποτελεσματική οπτικοποίηση των τάσεων των δεδομένων.

#### Βήμα προς βήμα εφαρμογή:

##### 1. Ορίστε τη διαδρομή για την αποθήκευση εγγράφων
Ξεκινήστε καθορίζοντας πού θέλετε να αποθηκευτούν τα αρχεία σας.

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
```

##### 2. Δημιουργήστε ένα νέο αντικείμενο παρουσίασης
Δημιουργήστε μια παρουσία του `Presentation` τάξη για να ξεκινήσετε τη δημιουργία της παρουσίασής σας.

```csharp
Presentation pres = new Presentation();
```

##### 3. Πρόσβαση στην Πρώτη Διαφάνεια
Αποκτήστε πρόσβαση στην πρώτη διαφάνεια της παρουσίασής σας χρησιμοποιώντας:

```csharp
ISlide slide = pres.Slides[0];
```

##### 4. Προσθέστε ένα γράφημα ομαδοποιημένων στηλών
Προσθέστε ένα γράφημα στην επιθυμητή θέση στη διαφάνεια.

```csharp
IChart chart = slide.Shapes.AddChart(ChartType.ClusteredColumn, 50, 50, 500, 400);
```
Αυτό προσθέτει ένα γράφημα ομαδοποιημένων στηλών στις συντεταγμένες (50, 50) με διαστάσεις 500x400 pixel.

##### 5. Αποθήκευση της παρουσίασης
Τέλος, αποθηκεύστε την παρουσίασή σας στον καθορισμένο κατάλογο.

```csharp
pres.Save(dataDir + "CreatePresentationWithChart_out.pptx", SaveFormat.Pptx);
```

### Λειτουργία 2: Ρύθμιση προκαθορισμένης μορφής αριθμών για σημεία δεδομένων γραφήματος

#### Επισκόπηση:
Μάθετε πώς να ορίσετε μια προκαθορισμένη μορφή αριθμού (π.χ., ποσοστό) για σημεία δεδομένων σε σειρές γραφημάτων, βελτιώνοντας την αναγνωσιμότητα των γραφημάτων σας.

#### Βήμα προς βήμα εφαρμογή:

##### 1. Πρόσβαση και διέλευση σειρών
Αφού προσθέσετε το γράφημά σας, αποκτήστε πρόσβαση στη συλλογή σειρών του.

```csharp
IChartSeriesCollection series = chart.ChartData.Series;
```

##### 2. Μορφοποίηση κάθε σημείου δεδομένων
Ορίστε μια μορφή αριθμού για κάθε σημείο δεδομένων στη σειρά σε '0,00%'.

```csharp
foreach (ChartSeries ser in series)
{
    foreach (IChartDataPoint cell in ser.DataPoints)
    {
        // Ορίστε τη μορφή αριθμών για καλύτερη αναγνωσιμότητα
        cell.Value.AsCell.PresetNumberFormat = 10; // Μορφή ως 0,00%
    }
}
```

##### 3. Αποθηκεύστε την παρουσίαση με μορφοποιημένους αριθμούς

```csharp
pres.Save(dataDir + "SetPresetNumberFormat_out.pptx", SaveFormat.Pptx);
```

## Οδηγός Εφαρμογής
### Δημιουργία λειτουργίας γραφήματος διοχέτευσης
Αυτή η λειτουργία σάς επιτρέπει να προσθέσετε ένα γράφημα χοάνης στην παρουσίασή σας στο PowerPoint χωρίς κόπο. Ας το αναλύσουμε σε βήματα:

#### Βήμα 1: Ρύθμιση των καταλόγων εγγράφων σας
Αρχικά, ορίστε τις διαδρομές για τους καταλόγους εγγράφων και εξόδου.
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";
```

#### Βήμα 2: Φόρτωση ή δημιουργία παρουσίασης
Φορτώστε μια υπάρχουσα παρουσίαση ή δημιουργήστε μια νέα εάν δεν υπάρχει.
```csharp
using (Presentation pres = new Presentation(dataDir + "/test.pptx"))
{
    // Περαιτέρω βήματα θα γίνουν εδώ
}
```
Αυτό το βήμα διασφαλίζει ότι έχετε ένα βασικό αρχείο PowerPoint για να εργαστείτε.

#### Βήμα 3: Προσθέστε το γράφημα διοχέτευσης
Προσθέστε ένα γράφημα χοάνης στην πρώτη διαφάνεια.
```csharp
IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.Funnel, 50, 50, 500, 400);
```
Αυτή η γραμμή προσθέτει ένα νέο γράφημα διοχέτευσης με καθορισμένες διαστάσεις.

#### Βήμα 4: Διαγραφή υπαρχόντων δεδομένων
Βεβαιωθείτε ότι δεν υπάρχουν προϋπάρχουσες κατηγορίες ή σειρές που θα μπορούσαν να επηρεάσουν.
```csharp
chart.ChartData.Categories.Clear();
chart.ChartData.Series.Clear();
```

#### Βήμα 5: Ρύθμιση παραμέτρων δεδομένων γραφήματος
Αποκτήστε πρόσβαση στο βιβλίο εργασίας για αποθήκευση δεδομένων γραφήματος και διαγράψτε τα υπάρχοντα κελιά.
```csharp
IChartDataWorkbook wb = chart.ChartData.ChartDataWorkbook;
wb.Clear(0);
```
Στη συνέχεια, προσθέστε κατηγορίες στο γράφημα διοχέτευσης.
```csharp
chart.ChartData.Categories.Add(wb.GetCell(0, "A1", "Category 1"));
// Επαναλάβετε για επιπλέον κατηγορίες
```

#### Βήμα 6: Προσθήκη και συμπλήρωση σειρών
Δημιουργήστε μια νέα σειρά τύπου Funnel και συμπληρώστε την με σημεία δεδομένων.
```csharp
IChartSeries series = chart.ChartData.Series.Add(ChartType.Funnel);
series.DataPoints.AddDataPointForFunnelSeries(wb.GetCell(0, "B1", 50));
// Επαναλάβετε για επιπλέον σημεία δεδομένων
```
Κάθε σημείο δεδομένων αντιστοιχεί σε μια κατηγορία στη διοχέτευση.

#### Βήμα 7: Αποθηκεύστε την παρουσίασή σας
Τέλος, αποθηκεύστε την τροποποιημένη παρουσίασή σας.
```csharp
pres.Save(outputDir + "/Funnel.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
```

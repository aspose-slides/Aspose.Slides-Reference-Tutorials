## Οδηγός Εφαρμογής

### Εξαγωγή ψεύτικων και πραγματικών υπερσυνδέσμων από διαφάνειες PowerPoint

Σε αυτήν την ενότητα, θα αναλύσουμε τα βήματα για την εξαγωγή τόσο ψεύτικων (εμφάνισης) URL όσο και πραγματικών υπερσυνδέσμων από ένα συγκεκριμένο τμήμα κειμένου μέσα σε μια διαφάνεια.

#### Βήμα 1: Φόρτωση της παρουσίασης

Ξεκινήστε φορτώνοντας το αρχείο παρουσίασής σας:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
using (var presentation = new Presentation(dataDir + "/ExternalUrlOriginal.pptx"))
{
    // Συνεχίστε με την πρόσβαση σε διαφάνειες και σχήματα.
}
```

#### Βήμα 2: Πρόσβαση στο τμήμα κειμένου στη διαφάνεια

Προσδιορίστε το τμήμα κειμένου από το οποίο θέλετε να εξαγάγετε υπερσυνδέσμους:
```csharp
var portion = ((AutoShape)presentation.Slides[0].Shapes[1]).TextFrame.Paragraphs[0].Portions[0];
```
Αυτός ο κώδικας έχει πρόσβαση στο πρώτο σχήμα στην πρώτη διαφάνεια, στοχεύοντας συγκεκριμένα στο περιεχόμενο κειμένου του.

#### Βήμα 3: Εξαγωγή ψεύτικου υπερσυνδέσμου

Ανάκτηση του ψεύτικου (εμφάνιση) υπερσυνδέσμου:
```csharp
string fakeHyperlink = portion.PortionFormat.AsIHyperlinkContainer.HyperlinkClick.ExternalUrl;
```

#### Βήμα 4: Εξαγωγή πραγματικού υπερσυνδέσμου

Ομοίως, εξαγάγετε την πραγματική διεύθυνση URL υπερσύνδεσης:
```csharp
string realHyperlink = portion.PortionFormat.AsIHyperlinkContainer.HyperlinkClick.ExternalUrlOriginal;
```

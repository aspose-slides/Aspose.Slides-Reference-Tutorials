## Οδηγός Εφαρμογής

### Χαρακτηριστικό 1: Ενσωμάτωση αντικειμένου OLE σε παρουσίαση

#### Επισκόπηση
Αυτή η λειτουργία σάς καθοδηγεί στην ενσωμάτωση ενός ενσωματωμένου αρχείου ως αντικειμένου OLE μέσα σε μια διαφάνεια του PowerPoint χρησιμοποιώντας το Aspose.Slides για .NET.

#### Βήματα:

**Βήμα 1: Αρχικοποίηση της παρουσίασης**
```csharp
using (Presentation pres = new Presentation())
{
    // Ο κωδικός σας εδώ...
}
```
- **Εξήγηση:** Ξεκινάμε δημιουργώντας μια παρουσία του `Presentation` για να χειριστείτε διαφάνειες.

**Βήμα 2: Ορισμός καταλόγου εγγράφων και ανάγνωση byte αρχείου**
```csharp
string dataDir = \@"YOUR_DOCUMENT_DIRECTORY";
byte[] fileBytes = File.ReadAllBytes(dataDir + "test.zip");
```
- **Παράμετροι:** `dataDir` είναι η διαδρομή όπου αποθηκεύονται τα αρχεία σας.
- **Επιστρεφόμενη τιμή:** `fileBytes` περιέχει το δυαδικό περιεχόμενο του αρχείου σας, το οποίο είναι απαραίτητο για την ενσωμάτωση.

**Βήμα 3: Δημιουργία αντικειμένου OleEmbeddedDataInfo**
```csharp
IOleEmbeddedDataInfo dataInfo = new OleEmbeddedDataInfo(fileBytes, "zip");
```
- **Σκοπός:** Αυτό το αντικείμενο ενσωματώνει τα ενσωματωμένα δεδομένα και καθορίζει τον τύπο αρχείου (π.χ., zip).

**Βήμα 4: Προσθήκη πλαισίου αντικειμένου OLE σε διαφάνεια**
```csharp
IOleObjectFrame oleFrame = pres.Slides[0].Shapes.AddOleObjectFrame(150, 20, 50, 50, dataInfo);
oleFrame.IsObjectIcon = true;
```
- **Εξήγηση:** Το αντικείμενο OLE προστίθεται στην πρώτη διαφάνεια. Εδώ, `IsObjectIcon` έχει οριστεί σε true για να εμφανίζει ένα εικονίδιο αντί για ολόκληρο το αντικείμενο.

**Συμβουλές αντιμετώπισης προβλημάτων:**
- Βεβαιωθείτε ότι οι διαδρομές αρχείων είναι σωστές και προσβάσιμες.
- Επαληθεύστε ότι ο τύπος αρχείου που καθορίζεται στο `OleEmbeddedDataInfo` ταιριάζει με την πραγματική μορφή αρχείου σας.

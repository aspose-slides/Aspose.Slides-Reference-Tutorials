### Προσθήκη και διαμόρφωση ενός πλαισίου αντικειμένου OLE

Αυτή η ενότητα παρουσιάζει τον τρόπο ενσωμάτωσης ενός εγγράφου Excel ως εικονίδιο μέσα σε μια διαφάνεια του PowerPoint.

#### Επισκόπηση
Η ενσωμάτωση ενός αντικειμένου OLE σάς επιτρέπει να εισάγετε σύνθετα έγγραφα όπως υπολογιστικά φύλλα ή άλλα αρχεία απευθείας στις παρουσιάσεις σας, διατηρώντας τη λειτουργικότητά τους.

#### Βήματα Υλοποίησης

**1. Προετοιμασία του αρχείου προέλευσης**
Βεβαιωθείτε ότι έχετε έτοιμο ένα αρχείο Excel στη διεύθυνση `YOUR_DOCUMENT_DIRECTORY/ExcelObject.xlsx`.

**2. Διαβάστε και ενσωματώστε το αρχείο**

```csharp
using Aspose.Slides;
using System.IO;

string oleSourceFile = "YOUR_DOCUMENT_DIRECTORY/ExcelObject.xlsx";
byte[] allbytes = File.ReadAllBytes(oleSourceFile);
IOleEmbeddedDataInfo dataInfo = new OleEmbeddedDataInfo(allbytes, "xlsx");

using (Presentation pres = new Presentation()) {
    ISlide slide = pres.Slides[0];
    IOleObjectFrame oof = slide.Shapes.AddOleObjectFrame(20, 20, 50, 50, dataInfo);
    
    // Ορισμός του αντικειμένου OLE ώστε να εμφανίζεται ως εικονίδιο
    oof.IsObjectIcon = true;
}
```
- **Παράμετροι:** `AddOleObjectFrame` Λαμβάνει τη θέση και το μέγεθος του πλαισίου (x, y, πλάτος, ύψος) μαζί με τις πληροφορίες δεδομένων.
- **Σκοπός:** Σύνθεση `IsObjectIcon` να `true` διασφαλίζει ότι εμφανίζεται μόνο ένα εικονίδιο, εξοικονομώντας χώρο ενώ παράλληλα διατηρεί το περιεχόμενο προσβάσιμο.

### Προσθήκη και διαμόρφωση εικόνας αντικατάστασης για ένα πλαίσιο αντικειμένου OLE

Στη συνέχεια, θα αντικαταστήσουμε το προεπιλεγμένο εικονίδιο του Excel με μια προσαρμοσμένη εικόνα.

#### Επισκόπηση
Η προσαρμογή των εικονιδίων μπορεί να κάνει τις παρουσιάσεις σας πιο ελκυστικές οπτικά και να ευθυγραμμιστούν με τις οδηγίες επωνυμίας.

#### Βήματα Υλοποίησης

**1. Προετοιμασία του αρχείου εικονιδίων**
Βεβαιωθείτε ότι έχετε ένα αρχείο εικόνας στη διεύθυνση `YOUR_DOCUMENT_DIRECTORY/Image.png`.

**2. Ενσωμάτωση και αντικατάσταση του προεπιλεγμένου εικονιδίου**

```csharp
using Aspose.Slides;
using System.IO;

string oleIconFile = "YOUR_DOCUMENT_DIRECTORY/Image.png";
byte[] imgBuf = File.ReadAllBytes(oleIconFile);

using (Presentation pres = new Presentation()) {
    using (MemoryStream ms = new MemoryStream(imgBuf)) {
        IPPImage image = pres.Images.AddImage(System.Drawing.Image.FromStream(ms));
        ISlide slide = pres.Slides[0];
        IOleObjectFrame oof = slide.Shapes.AddOleObjectFrame(20, 20, 50, 50, new OleEmbeddedDataInfo(imgBuf, "png"));
        
        // Αντικατάσταση του εικονιδίου του αντικειμένου OLE με μια προσαρμοσμένη εικόνα
        oof.SubstitutePictureFormat.Picture.Image = image;
    }
}
```
- **Παράμετροι:** `AddImage` Η μέθοδος προσθέτει μια εικόνα στη συλλογή εικόνων παρουσίασης.
- **Σκοπός:** Η αντικατάσταση ενισχύει την οπτική ελκυστικότητα και παρέχει καλύτερο πλαίσιο με μια ματιά.

### Ορισμός λεζάντας για ένα εικονίδιο αντικειμένου OLE

Η προσθήκη λεζάντων μπορεί να διευκρινίσει τι αντιπροσωπεύει κάθε εικονίδιο στις διαφάνειές σας.

#### Επισκόπηση
Οι λεζάντες είναι ζωτικής σημασίας όταν πρόκειται για πολλά εικονίδια, εξασφαλίζοντας σαφήνεια χωρίς να γεμίζει η διαφάνεια με κείμενο.

#### Βήματα Υλοποίησης

**1. Επαναχρησιμοποιήστε το βήμα προετοιμασίας εικόνας**

```csharp
using Aspose.Slides;
using System.IO;

string oleIconFile = "YOUR_DOCUMENT_DIRECTORY/Image.png";
byte[] imgBuf = File.ReadAllBytes(oleIconFile);

using (Presentation pres = new Presentation()) {
    using (MemoryStream ms = new MemoryStream(imgBuf)) {
        IPPImage image = pres.Images.AddImage(System.Drawing.Image.FromStream(ms));
        ISlide slide = pres.Slides[0];
        IOleObjectFrame oof = slide.Shapes.AddOleObjectFrame(20, 20, 50, 50, new OleEmbeddedDataInfo(imgBuf, "png"));
        
        // Ορισμός του κειμένου λεζάντας για το εικονίδιο OLE
        oof.SubstitutePictureTitle = "Caption example";
    }
}
```
- **Σκοπός:** Ο `SubstitutePictureTitle` Η ιδιότητα σάς επιτρέπει να παρέχετε μια περιγραφική λεζάντα απευθείας στο εικονίδιο.

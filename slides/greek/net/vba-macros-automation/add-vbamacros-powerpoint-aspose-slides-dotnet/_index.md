## Οδηγός Εφαρμογής

Ακολουθήστε αυτά τα βήματα για να προσθέσετε μακροεντολές VBA σε μια παρουσίαση του PowerPoint.

### Προσθήκη έργου VBA στο PowerPoint

#### Επισκόπηση
Δημιουργήστε ένα έργο VBA μέσα στην παρουσίασή σας που θα περιέχει όλες τις μακροεντολές:
```csharp
// Δημιουργία παρουσίασης
using (Presentation presentation = new Presentation())
{
    // Δημιουργία νέου έργου VBA
    presentation.VbaProject = new VbaProject();
}
```

#### Προσθήκη κενής ενότητας
Προσθέστε μια ενότητα για τον κώδικα μακροεντολών σας χρησιμοποιώντας `AddEmptyModule`:
```csharp
// Προσθήκη κενής ενότητας στο έργο VBA
IVbaModule module = presentation.VbaProject.Modules.AddEmptyModule("Module");
```

### Ρύθμιση πηγαίου κώδικα ενότητας
Εισαγάγετε τον κώδικα μακροεντολής σας. Αυτό το παράδειγμα δείχνει ένα απλό πλαίσιο μηνύματος:
```csharp
// Ορισμός πηγαίου κώδικα ενότητας
module.SourceCode = "Sub Test(oShape As Shape) MsgBox \"Test\" End Sub";
```
#### Επεξήγηση παραμέτρων
- **Πηγαίος Κώδικας**: Ο κώδικας VBA που ορίζει τη λειτουργικότητα της μακροεντολής.

### Δημιουργία αναφορών
Προσθήκη αναφορών σε `stdole` και `Office` βιβλιοθήκες για συμβατότητα:
```csharp
// Δημιουργία αναφοράς στο stdole
VbaReferenceOleTypeLib stdoleReference = new VbaReferenceOleTypeLib(
    "stdole", 
    "*\\G{00020430-0000-0000-C000-000000000046}#2.0#0#C:\\Windows\\system32\\stdole2.tlb#OLE Automation");

// Δημιουργία αναφοράς στο Office
VbaReferenceOleTypeLib officeReference = new VbaReferenceOleTypeLib(
    "Office", 
    "*\\G{2DF8D04C-5BFA-101B-BDE5-00AA0044DE52}#2.0#0#C:\\Program Files\\Common Files\\Microsoft Shared\\OFFICE14\\MSO.DLL#Microsoft Office 14.0 Object Library");

// Προσθήκη αναφορών στο έργο VBA
presentation.VbaProject.References.Add(stdoleReference);
presentation.VbaProject.References.Add(officeReference);
```

### Αποθήκευση της παρουσίασής σας
Αποθηκεύστε την παρουσίασή σας με ενσωματωμένες μακροεντολές:
```csharp
// Αποθήκευση παρουσίασης
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
presentation.Save(dataDir + "AddVBAMacros_out.pptm", SaveFormat.Pptm);
```

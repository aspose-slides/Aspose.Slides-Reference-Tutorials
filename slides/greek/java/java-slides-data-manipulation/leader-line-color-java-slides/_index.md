# Χρώμα γραμμής Leader σε διαφάνειες Java


## Εισαγωγή στο χρώμα γραμμής οδηγού στο Aspose.Slides για Java

Σε αυτό το σεμινάριο, θα εξερευνήσουμε πώς να αλλάξουμε το χρώμα της γραμμής οδηγού ενός γραφήματος σε μια παρουσίαση PowerPoint χρησιμοποιώντας το Aspose.Slides για Java. Οι γραμμές οδηγού χρησιμοποιούνται στα γραφήματα για τη σύνδεση ετικετών δεδομένων με τα αντίστοιχα σημεία δεδομένων τους. Θα χρησιμοποιήσουμε κώδικα Java για να ολοκληρώσουμε αυτήν την εργασία.

## Προαπαιτούμενα

Πριν ξεκινήσετε, βεβαιωθείτε ότι έχετε τα εξής:

- Εγκατεστημένο το Aspose.Slides για το Java API. Μπορείτε να το κατεβάσετε από [εδώ](https://releases.aspose.com/slides/java/).

## Βήμα 1: Φόρτωση της παρουσίασης

Αρχικά, πρέπει να φορτώσετε την παρουσίαση PowerPoint που περιέχει το γράφημα που θέλετε να τροποποιήσετε. Αντικατάσταση `presentationName` με τη διαδρομή προς το αρχείο PowerPoint σας.

```java
String presentationName = "path/to/your/presentation.pptx";
String outPath = "output/path/output.pptx";
Presentation pres = new Presentation(presentationName);
```

## Βήμα 2: Πρόσβαση στις ετικέτες γραφήματος και δεδομένων

Στη συνέχεια, θα έχουμε πρόσβαση στο γράφημα και στις ετικέτες δεδομένων μέσα στην παρουσίαση. Σε αυτό το παράδειγμα, υποθέτουμε ότι το γράφημα βρίσκεται στην πρώτη διαφάνεια.

```java
// Λήψη του γραφήματος από την πρώτη διαφάνεια
IChart chart = (IChart)pres.getSlides().get_Item(0).getShapes().get_Item(0);

// Λήψη σειράς του γραφήματος
IChartSeriesCollection series = chart.getChartData().getSeries();

// Αποκτήστε ετικέτες της πρώτης σειράς
IDataLabelCollection labels = series.get_Item(0).getLabels();
```

## Βήμα 3: Αλλαγή χρώματος γραμμής οδηγού

Τώρα, θα αλλάξουμε το χρώμα όλων των γραμμών-οδηγών στη συλλογή σε κόκκινο. Μπορείτε να προσαρμόσετε το χρώμα σύμφωνα με τις απαιτήσεις σας.

```java
// Αλλαγή χρώματος όλων των γραμμών οδηγού στη συλλογή σε κόκκινο
labels.getLeaderLinesFormat().getLine().getFillFormat().getSolidFillColor().setColor(Color.RED);
```

## Βήμα 4: Αποθήκευση της τροποποιημένης παρουσίασης

Τέλος, αποθηκεύστε την παρουσίαση με τα τροποποιημένα χρώματα της γραμμής οδηγού σε ένα νέο αρχείο.

```java
// Αποθήκευση της τροποποιημένης παρουσίασης
pres.save(outPath, SaveFormat.Pptx);
```

## Πλήρης πηγαίος κώδικας για το χρώμα της γραμμής Leader σε διαφάνειες Java

```java
        String presentationName = "Your Document Directory";
        String outPath = "Your Output Directory" + "LeaderLinesColor-out.pptx";
        Presentation pres = new Presentation(presentationName);
        try {
            // Λήψη του γραφήματος από την πρώτη διαφάνεια
            IChart chart = (IChart)pres.getSlides().get_Item(0).getShapes().get_Item(0);
            // Λήψη σειράς του γραφήματος
            IChartSeriesCollection series = chart.getChartData().getSeries();
            // Αποκτήστε τα επίπεδα της πρώτης σειράς
            IDataLabelCollection labels = series.get_Item(0).getLabels();
            // Αλλαγή χρώματος όλων των γραμμών οδηγού στη συλλογή
            labels.getLeaderLinesFormat().getLine().getFillFormat().getSolidFillColor().setColor(Color.RED);
            // Αποθήκευση αποτελέσματος
            pres.save(outPath, SaveFormat.Pptx);
        } finally {
            if (pres != null) pres.dispose();
        }
```

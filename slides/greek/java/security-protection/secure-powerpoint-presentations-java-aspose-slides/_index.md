## Οδηγός Εφαρμογής

Θα εφαρμόσουμε δύο κύριες λειτουργίες: τη δημιουργία καταλόγων και την κρυπτογράφηση παρουσιάσεων.

### Δημιουργία καταλόγου

#### Επισκόπηση

Αυτή η λειτουργία ελέγχει για έναν υπάρχοντα κατάλογο και τον δημιουργεί εάν είναι απαραίτητο, διασφαλίζοντας ότι τα αρχεία της παρουσίασής σας είναι καλά οργανωμένα.

#### Βήματα για την εφαρμογή

##### Βήμα 1: Εισαγωγή απαιτούμενων πακέτων

```java
import java.io.File;
```

##### Βήμα 2: Ορισμός διαδρομής καταλόγου

Ορίστε τη διαδρομή κράτησης θέσης όπου θέλετε να αποθηκεύσετε τις παρουσιάσεις σας:

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
```

##### Βήμα 3: Έλεγχος και δημιουργία καταλόγου

Ελέγξτε αν ο κατάλογος υπάρχει και δημιουργήστε τον αν όχι:

```java
boolean isExists = new File(dataDir).exists();
if (!isExists) {
    new File(dataDir).mkdirs(); // Δημιουργεί τον κατάλογο με όλους τους απαραίτητους γονικούς καταλόγους.
}
```

#### Εξήγηση

- **`File.exists()`**: Επιστροφές `true` αν το αρχείο ή ο κατάλογος υπάρχει.
- **`File.mkdirs()`**: Δημιουργεί τον κατάλογο, συμπεριλαμβανομένων τυχόν απαραίτητων αλλά ανύπαρκτων γονικών καταλόγων.

### Φόρτωση και αποθήκευση παρουσίασης με κρυπτογράφηση κωδικού πρόσβασης

#### Επισκόπηση

Κρυπτογραφήστε την παρουσίασή σας για να προστατεύσετε το περιεχόμενό της από μη εξουσιοδοτημένη πρόσβαση χρησιμοποιώντας έναν κωδικό πρόσβασης.

#### Βήματα για την εφαρμογή

##### Βήμα 1: Εισαγωγή πακέτων Aspose.Slides

```java
import com.aspose.slides.Presentation;
import com.aspose.slides.SaveFormat;
```

##### Βήμα 2: Δημιουργία αντικειμένου παρουσίασης

Δημιουργήστε μια παρουσία του `Presentation` τάξη:

```java
Presentation pres = new Presentation();
```

##### Βήμα 3: Κρυπτογράφηση της παρουσίασης

Εφαρμόστε έναν κωδικό πρόσβασης για την κρυπτογράφηση της παρουσίασης:

```java
pres.getProtectionManager().encrypt("pass"); // Το «pass» είναι ο κωδικός πρόσβασης που έχετε επιλέξει.
```

##### Βήμα 4: Αποθήκευση του κρυπτογραφημένου αρχείου

Αποθηκεύστε την παρουσίαση στον καθορισμένο κατάλογο με εφαρμοσμένη κρυπτογράφηση:

```java
String outputPath = "YOUR_DOCUMENT_DIRECTORY/SaveWithPassword_out.pptx";
pres.save(outputPath, SaveFormat.Pptx); // Αποθηκεύεται ως αρχείο PPTX.
```

#### Εξήγηση

- **`encrypt()`**: Ασφαλίζει την παρουσίασή σας χρησιμοποιώντας τον παρεχόμενο κωδικό πρόσβασης.
- **`save()`**Εγγράφει την κρυπτογραφημένη παρουσίαση στον δίσκο στην καθορισμένη μορφή.

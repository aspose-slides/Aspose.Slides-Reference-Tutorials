# Φόρτωση απαρίθμησης μορφής σε διαφάνειες Java


## Εισαγωγή στη φόρτωση μορφής παρουσίασης σε διαφάνειες Java

Σε αυτό το σεμινάριο, θα εξερευνήσουμε πώς να προσδιορίσουμε τη μορφή μιας παρουσίασης PowerPoint χρησιμοποιώντας το Aspose.Slides για Java API. Θα επικεντρωθούμε συγκεκριμένα στη φόρτωση μιας παρουσίασης και στον έλεγχο της μορφής της χρησιμοποιώντας το `LoadFormat` απαρίθμηση. Αυτό θα σας βοηθήσει να προσδιορίσετε εάν η παρουσίαση είναι σε παλαιότερη μορφή, όπως το PowerPoint 95, ή σε πιο πρόσφατη μορφή.

## Προαπαιτούμενα

Πριν ξεκινήσουμε, βεβαιωθείτε ότι έχετε εγκαταστήσει και ρυθμίσει τη βιβλιοθήκη Aspose.Slides για Java στο έργο σας Java. Μπορείτε να την κατεβάσετε από το [Ιστότοπος Aspose](https://products.aspose.com/slides/java/) και ακολουθήστε τις οδηγίες εγκατάστασης.

## Βήμα 1: Εισαγωγή απαιτούμενων κλάσεων

Για να ξεκινήσετε, πρέπει να εισαγάγετε τις απαραίτητες κλάσεις από τη βιβλιοθήκη Aspose.Slides. Αυτές οι κλάσεις θα μας επιτρέψουν να εργαστούμε με παρουσιάσεις και να ελέγξουμε τις μορφές τους.

```java
import com.aspose.slides.LoadFormat;
import com.aspose.slides.PresentationFactory;
```

## Βήμα 2: Φόρτωση της παρουσίασης

Σε αυτό το βήμα, θα φορτώσουμε το αρχείο παρουσίασης PowerPoint του οποίου τη μορφή θέλετε να ελέγξετε. Αντικαταστήστε `"Your Document Directory"` με την πραγματική διαδρομή προς το αρχείο παρουσίασής σας.

```java
String dataDir = "Your Document Directory";
boolean isOldFormat = PresentationFactory.getInstance().getPresentationInfo(dataDir + "presentation.ppt").getLoadFormat() == LoadFormat.Ppt95;
```

Στον παραπάνω κώδικα, χρησιμοποιούμε `PresentationFactory.getInstance().getPresentationInfo()` για να λάβουμε πληροφορίες σχετικά με την παρουσίαση, συμπεριλαμβανομένης της μορφής της. Στη συνέχεια, συγκρίνουμε τη μορφή με `LoadFormat.Ppt95` για να ελέγξετε αν πρόκειται για παλαιότερη μορφή PowerPoint 95.

## Πλήρης πηγαίος κώδικας για απαρίθμηση μορφής φόρτωσης σε διαφάνειες Java

```java
        // Η διαδρομή προς τον κατάλογο εγγράφων.
        String dataDir = "Your Document Directory";
        boolean isOldFormat = PresentationFactory.getInstance().getPresentationInfo(dataDir + "presentation.ppt").getLoadFormat() == LoadFormat.Ppt95;
```
## Σύναψη

Σε αυτό το σεμινάριο, μάθαμε πώς να φορτώνουμε μια παρουσίαση PowerPoint σε Java χρησιμοποιώντας το Aspose.Slides και να ελέγχουμε τη μορφή της χρησιμοποιώντας το `LoadFormat` απαρίθμηση. Αυτό μπορεί να είναι χρήσιμο όταν χρειάζεται να χειρίζεστε παρουσιάσεις διαφορετικών μορφών διαφορετικά στην εφαρμογή Java σας.

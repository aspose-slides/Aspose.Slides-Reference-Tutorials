# Ενσωματωμένη συμπίεση γραμματοσειρών σε Java PowerPoint

Το Aspose.Slides για Java προσφέρει μια ισχυρή λύση για την ενίσχυση της αποτελεσματικότητας των παρουσιάσεων PowerPoint ενεργοποιώντας την ενσωματωμένη συμπίεση γραμματοσειρών. Αυτό το σεμινάριο θα σας καθοδηγήσει στη διαδικασία αξιοποίησης αυτής της λειτουργίας για την αποτελεσματική μείωση του μεγέθους των αρχείων, εξασφαλίζοντας ομαλότερη διανομή και βελτιωμένη απόδοση των παρουσιάσεών σας.

## Εισαγωγή πακέτων
Για να ξεκινήσετε, εισαγάγετε τα απαραίτητα πακέτα στο έργο Java σας για να αποκτήσετε πρόσβαση στις λειτουργίες του Aspose.Slides για Java:
```java
import com.aspose.slides.*;
import java.nio.file.Files;
import java.nio.file.Paths;
```
## 1. Φόρτωση της παρουσίασης
Αρχικά, πρέπει να φορτώσετε την παρουσίαση PowerPoint στην εφαρμογή Java χρησιμοποιώντας το Aspose.Slides:
```java
String presentationName = "Your Document Directory";
Presentation pres = new Presentation(presentationName);
```
## 2. Συμπίεση ενσωματωμένων γραμματοσειρών
Στη συνέχεια, καλέστε το `Compress.compressEmbeddedFonts()` μέθοδος για τη συμπίεση των ενσωματωμένων γραμματοσειρών στην παρουσίαση:
```java
Compress.compressEmbeddedFonts(pres);
```
## 3. Αποθηκεύστε το αποτέλεσμα
Αποθηκεύστε την συμπιεσμένη παρουσίαση σε έναν καθορισμένο κατάλογο εξόδου:
```java
String outPath = "Your Output Directory" + "presWithEmbeddedFonts-out.pptx";
pres.save(outPath, SaveFormat.Pptx);
```
## 4. Ανάκτηση πληροφοριών αρχείου
Προαιρετικά, μπορείτε να ανακτήσετε πληροφορίες σχετικά με τα μεγέθη των αρχείων προέλευσης και αποτελέσματος:
```java
// Λήψη πληροφοριών αρχείου προέλευσης
byte[] sourceFile = Files.readAllBytes(Paths.get(presentationName));
System.out.println(String.format("Source file size = %d bytes", sourceFile.length));
// Λήψη πληροφοριών αρχείου αποτελεσμάτων
byte[] outputFile = Files.readAllBytes(Paths.get(outPath));
System.out.println(String.format("Result file size = %d bytes", outputFile.length));
```

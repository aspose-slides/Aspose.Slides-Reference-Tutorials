## Πώς να αντικαταστήσετε κείμενο σε παρουσιάσεις PowerPoint

Θα περάσουμε από τα βασικά βήματα: φόρτωση αρχείου, ορισμός μορφής αντικατάστασης, εκτέλεση εύρεσης‑και‑αντικατάστασης και αποθήκευση του αποτελέσματος.

### Φόρτωση και αποθήκευση παρουσίασης

#### Φόρτωση της παρουσίασης
```java
String presentationName = "YOUR_DOCUMENT_DIRECTORY/TextReplaceExample.pptx";
Presentation pres = new Presentation(presentationName);
```

#### Αποθήκευση της τροποποιημένης παρουσίασης
```java
String outPath = "YOUR_OUTPUT_DIRECTORY/TextReplaceExample-out.pptx";
pres.save(outPath, SaveFormat.Pptx);
```

> **Συμβουλή:** Πάντα καλέστε `pres.dispose();` μετά το τέλος για να ελευθερώσετε τους εγγενείς πόρους.

### Μορφοποίηση κειμένου για αντικατάσταση

Αν θέλετε το νέο κείμενο να ξεχωρίζει, διαμορφώστε ένα `PortionFormat` πριν την αντικατάσταση.

```java
PortionFormat format = new PortionFormat();
format.setFontHeight(24f); // Set font height to 24 points
format.setFontItalic(NullableBool.True); // Make the font italic
format.getFillFormat().setFillType(FillType.Solid);
format.getFillFormat().getSolidFillColor().setColor(Color.RED); // Set text color to red
```

### Εύρεση και αντικατάσταση κειμένου στην παρουσίαση

Τώρα χρησιμοποιήστε την βοηθητική κλάση για να αντικαταστήσετε κάθε εμφάνιση ενός placeholder.

```java
String searchText = "[this block] ";
String replacementText = "my text";
SlideUtil.findAndReplaceText(pres, true, searchText, replacementText, format);
```

Η μέθοδος `findAndReplaceText` σαρώει όλες τις διαφάνειες, αντικαθιστά τη στοχευόμενη συμβολοσειρά και εφαρμόζει το `PortionFormat` που ορίσατε, παρέχοντάς σας **μορφοποιημένο κείμενο στις διαφάνειες** αυτόματα.

## Σκέψεις απόδοσης

- **Απόρριψη αντικειμένων:** Καλέστε `dispose()` σε στιγμιότυπα `Presentation` για να αποφύγετε διαρροές μνήμης.

- **Λειτουργία παρτίδας:** Επεξεργαστείτε αρχεία σε ομάδες αντί για ένα‑προς‑ένα για μείωση του φόρτου JVM.

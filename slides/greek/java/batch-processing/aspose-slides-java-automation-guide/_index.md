## Πώς να φορτώσετε και να αποθηκεύσετε μια παρουσίαση;

Φορτώστε το αρχείο PPTX με μία μόνο γραμμή κώδικα και διατηρήστε τις αλλαγές χρησιμοποιώντας τη μέθοδο `save` – αυτή είναι η πλήρης ροή εργασίας σε δύο βήματα. Η κλάση `Presentation` αντιπροσωπεύει ολόκληρη την παρουσίαση στη μνήμη, επιτρέποντάς σας να περιηγηθείτε στις διαφάνειες, να επεξεργαστείτε το περιεχόμενο και τελικά να γράψετε το αποτέλεσμα στο δίσκο.

### Φόρτωση της Παρουσίασης
Δημιουργήστε ένα αντικείμενο `Presentation` περνώντας τη διαδρομή του αρχείου προέλευσης:

```java
String presentationName = "YOUR_DOCUMENT_DIRECTORY/TextReplaceExample.pptx";
Presentation pres = new Presentation(presentationName);
```

### Αποθήκευση της Τροποποιημένης Παρουσίασης
Όταν ολοκληρώσετε την επεξεργασία, καλέστε `save` με τη μορφή εξόδου και τη διαδρομή που επιθυμείτε:

```java
String outPath = "YOUR_OUTPUT_DIRECTORY/TextReplaceExample-out.pptx";
pres.save(outPath, SaveFormat.Pptx);
```

Θυμηθείτε να καλέσετε `dispose()` στο αντικείμενο `Presentation` μετά την αποθήκευση για να ελευθερώσετε τους εγγενείς πόρους.

## Πώς να αντικαταστήσετε κείμενο σε όλες τις διαφάνειες;

Αντικαταστήστε κάθε εμφάνιση μιας φράσης με μία κλήση, στη συνέχεια εφαρμόστε συνεπή μορφοποίηση—όλα χωρίς να γράψετε βρόχο για κάθε διαφάνεια. Η μέθοδος `SlideUtil.findAndReplaceText` σαρώει ολόκληρη την παρουσίαση και αντικαθιστά τη στοχευόμενη συμβολοσειρά με την αντικατάσταση που παρέχετε.

Η `SlideUtil` είναι μια βοηθητική κλάση που προσφέρει μαζικές λειτουργίες σε διαφάνειες, συμπεριλαμβανομένης της γρήγορης εύρεσης‑και‑αντικατάστασης σε όλο το deck.

### Εκτέλεση Εύρεσης και Αντικατάστασης
Ορίστε τον όρο αναζήτησης, την αντικατάσταση και την προαιρετική μορφοποίηση, στη συνέχεια εκτελέστε:

```java
String searchText = "[this block] ";
String replacementText = "my text";
SlideUtil.findAndReplaceText(pres, true, searchText, replacementText, format);
```

Η μέθοδος ενημερώνει όλες τις αντίστοιχες εμφανίσεις κειμένου, διατηρώντας τη διάταξη και εφαρμόζοντας το `PortionFormat` που έχετε διαμορφώσει.

## Μορφοποίηση Κειμένου για Αντικατάσταση

Προσαρμόστε την εμφάνιση του νέου κειμένου χρησιμοποιώντας το `PortionFormat`. Αυτή η κλάση ελέγχει το μέγεθος γραμματοσειράς, το στυλ, το χρώμα και άλλα τυπογραφικά χαρακτηριστικά.

Το `PortionFormat` είναι το αντικείμενο του Aspose.Slides που περιγράφει το οπτικό στυλ για μια ακολουθία κειμένου. Ρυθμίστε τις ιδιότητές του πριν το περάσετε στη μέθοδο αντικατάστασης.

### Ορισμός Ιδιοτήτων Κειμένου
Δημιουργήστε μια παρουσίαση `PortionFormat` και ορίστε τις επιθυμητές ιδιότητες:

```java
PortionFormat format = new PortionFormat();
format.setFontHeight(24f); // Set font height to 24 points
format.setFontItalic(NullableBool.True); // Make the font italic
format.getFillFormat().setFillType(FillType.Solid);
format.getFillFormat().getSolidFillColor().setColor(Color.RED); // Set text color to red
```

Το απόσπασμα ρυθμίζει το μέγεθος γραμματοσειράς, το έντονο στυλ και το χρώμα, διασφαλίζοντας ότι το αντικατεστημένο κείμενο ταιριάζει με τις οδηγίες branding σας.

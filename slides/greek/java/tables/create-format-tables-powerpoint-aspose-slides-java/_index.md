## Οδηγός Εφαρμογής

### Δημιουργία και Μορφοποίηση Πινάκων
Ας ξεκινήσουμε προσθέτοντας έναν πίνακα στην πρώτη διαφάνεια της παρουσίασής σας στο PowerPoint.

#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να δημιουργήσετε έναν πίνακα με συγκεκριμένες διαστάσεις και να μορφοποιήσετε το περίγραμμα κάθε κελιού για καλύτερη οπτική ελκυστικότητα.

#### Βήμα προς βήμα εφαρμογή
**1. Πρόσβαση στην πρώτη διαφάνεια**
```java
ISlide sld = presentation.getSlides().get_Item(0);
```
Εδώ, `sld` αντιπροσωπεύει την πρώτη σας διαφάνεια, όπου θα προσθέσετε τον πίνακα.

**2. Ορισμός διαστάσεων πίνακα**
Ορίστε τα πλάτη των στηλών και τα ύψη των γραμμών όπως απαιτείται:
```java
double[] dblCols = {70, 70, 70, 70};
double[] dblRows = {70, 70, 70, 70};
```

**3. Προσθήκη πίνακα στη διαφάνεια**
Τοποθετήστε τον πίνακά σας στις συντεταγμένες (100, 50) στη διαφάνεια:
```java
ITable tbl = sld.getShapes().addTable(100, 50, dblCols, dblRows);
```

**4. Ορισμός ιδιοτήτων περιγράμματος για κάθε κελί**
Για να βελτιώσετε την αναγνωσιμότητα και το στυλ, μορφοποιήστε το περίγραμμα κάθε κελιού:
```java
for (IRow row : tbl.getRows()) {
    for (ICell cell : row) {
        setCellBorder(cell, Color.RED, 5);
    }
}
```
Ο `setCellBorder` Η μέθοδος εφαρμόζει ένα κόκκινο περίγραμμα πλάτους 5 σε κάθε κελί.

#### Εξήγηση Βοηθητικής Μεθόδου
Δείτε πώς λειτουργεί η μέθοδος βοηθού:
```java
private static void setCellBorder(ICell cell, Color color, double width) {
    BorderFormat borderFormat = cell.getCellFormat().getBorderTop();
    borderFormat.getFillFormat().setFillType(FillType.Solid);
    borderFormat.getFillFormat().getSolidFillColor().setColor(color);
    borderFormat.setWidth(width);

    // Επαναλάβετε για κάτω, αριστερά και δεξιά περιγράμματα
}
```
Αυτή η μέθοδος ορίζει τον τύπο γεμίσματος σε συμπαγές και εφαρμόζει το καθορισμένο χρώμα και πλάτος και στις τέσσερις πλευρές ενός κελιού.

### Συγχώνευση κελιών σε πίνακες
#### Επισκόπηση
Μερικές φορές χρειάζεται να συνδυάσετε πολλά κελιά σε ένα. Αυτή η λειτουργία δείχνει πώς να συγχωνεύσετε κελιά μέσω προγραμματισμού.

#### Βήμα προς βήμα εφαρμογή
**1. Πρόσβαση στον πίνακα**
Υποθέτω `tbl` είναι το αντικείμενο του πίνακά σας όπως δημιουργήθηκε νωρίτερα.

**2. Καθορισμός κελιών για συγχώνευση**
Συγχώνευση κελιών σε μια συγκεκριμένη περιοχή:
```java
// Συγχώνευση κελιών (1, 1) x (2, 1)
tbl.mergeCells(tbl.getRows().get_Item(1).get_Item(1), tbl.getRows().get_Item(2).get_Item(1), false);

// Συγχώνευση κελιών (1, 2) x (2, 2)
tbl.mergeCells(tbl.getRows().get_Item(1).get_Item(2), tbl.getRows().get_Item(2).get_Item(2), false);
```
Ο `mergeCells` Η μέθοδος συνδυάζει το καθορισμένο εύρος σε ένα μόνο κελί.

**3. Αποθήκευση της παρουσίασής σας**
Μην ξεχάσετε να αποθηκεύσετε τις αλλαγές σας:
```java
presentation.save("YOUR_DOCUMENT_DIRECTORY/MergeCells_out.pptx", SaveFormat.Pptx);
```

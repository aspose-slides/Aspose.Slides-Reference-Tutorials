## Ρύθμιση του Aspose.Slides για Java

Για να ξεκινήσετε να χρησιμοποιείτε το Aspose.Slides στο έργο σας, ενσωματώστε τη βιβλιοθήκη μέσω του Maven ή του Gradle:

**Maven**
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

**Γκράντλ**
```gradle
implementation group: 'com.aspose', name: 'aspose-slides', version: '25.4', classifier: 'jdk16'
```

Εναλλακτικά, κατεβάστε τη βιβλιοθήκη απευθείας από [Aspose.Slides για εκδόσεις Java](https://releases.aspose.com/slides/java/).

## Οδηγός Εφαρμογής

### Ανάκτηση Ισχύουσων Τιμών Μορφών Πίνακα

Αυτή η ενότητα περιγράφει λεπτομερώς την πρόσβαση σε διάφορες ιδιότητες μορφοποίησης μέσα σε έναν πίνακα PowerPoint χρησιμοποιώντας το Aspose.Slides για Java.

#### Πρόσβαση στον πρώτο πίνακα στην παρουσίαση
Φορτώστε την παρουσίασή σας και αποκτήστε πρόσβαση στον πρώτο πίνακα:
```java
Presentation pres = new Presentation("YOUR_DOCUMENT_DIRECTORY/pres.pptx");
ITable tbl = (ITable) pres.getSlides().get_Item(0).getShapes().get_Item(0);
```

#### Ανάκτηση Ισχύοντος Δεδομένου για Μορφή Πίνακα
Εξαγωγή λεπτομερειών μορφοποίησης που εφαρμόζονται απευθείας στον πίνακα:
```java
ITableFormatEffectiveData tableFormatEffective = tbl.getTableFormat().getEffective();
IFillFormatEffectiveData tableFillFormatEffective = tableFormatEffective.getFillFormat();
```

#### Ανάκτηση Ισχύοντος Δεδομένου για Μορφή Γραμμής
Πρόσβαση σε ιδιότητες μορφοποίησης ενεργής γραμμής:
```java
IRowFormatEffectiveData rowFormatEffective = tbl.getRows().get_Item(0).getRowFormat().getEffective();
IFillFormatEffectiveData rowFillFormatEffective = rowFormatEffective.getFillFormat();
```

#### Ανάκτηση Ισχύοντος Δεδομένου για Μορφή Στήλης
Μορφοποίηση συγκεκριμένης στήλης της Access:
```java
IColumnFormatEffectiveData columnFormatEffective = tbl.getColumns().get_Item(0).getColumnFormat().getEffective();
IFillFormatEffectiveData columnFillFormatEffective = columnFormatEffective.getFillFormat();
```

#### Ανάκτηση αποτελεσματικών δεδομένων για μορφή κελιού
Μορφές που αφορούν συγκεκριμένα κελιά, όπως το επάνω αριστερό κελί:
```java
ICellFormatEffectiveData cellFormatEffective = tbl.get_Item(0, 0).getCellFormat().getEffective();
IFillFormatEffectiveData cellFillFormatEffective = cellFormatEffective.getFillFormat();
```

**Ακρο**Να απορρίπτετε πάντα τους πόρους σωστά για να αποτρέψετε διαρροές μνήμης:
```java
finally {
    if (pres != null) pres.dispose(); // Απορρίψτε σωστά την παρουσίαση
}
```

# Δημιουργία μικρογραφιών διαφανειών με σημειώσεις χρησιμοποιώντας το Aspose.Slides Java
## Εκτύπωση & Απόδοση
### Ένας οδηγός βήμα προς βήμα
Στον σημερινό ταχύτατα εξελισσόμενο ψηφιακό κόσμο, η αποτελεσματική διαχείριση και κοινοποίηση περιεχομένου παρουσιάσεων είναι ζωτικής σημασίας. Είτε είστε προγραμματιστής που ενσωματώνει παρουσιάσεις PowerPoint είτε αυτοματοποιεί τη διαδικασία εξαγωγής μικρογραφιών διαφανειών με σημειώσεις, **Aspose.Slides για Java** παρέχει ισχυρές δυνατότητες για την απλοποίηση αυτών των εργασιών. Αυτό το ολοκληρωμένο σεμινάριο θα σας καθοδηγήσει στη δημιουργία μικρογραφιών διαφανειών ενώ παράλληλα θα εμφανίζετε σημειώσεις στο κάτω μέρος χρησιμοποιώντας το Aspose.Slides, παράλληλα με την αλλαγή των προεπιλεγμένων ρυθμίσεων γραμματοσειράς για τις διαφάνειές σας.

- Πώς να ανακτήσετε μια μικρογραφία διαφάνειας με ορατές σημειώσεις
- Αλλαγή της προεπιλεγμένης κανονικής γραμματοσειράς στην απόδοση διαφανειών
- Ρύθμιση και διαμόρφωση του Aspose.Slides για Java
- Πρακτικές εφαρμογές αυτών των χαρακτηριστικών

Ας δούμε τις προϋποθέσεις πριν ξεκινήσουμε.

### Προαπαιτούμενα
Πριν ξεκινήσετε, βεβαιωθείτε ότι έχετε τα εξής:
- **Aspose.Slides για Java** βιβλιοθήκη: Θα χρειαστείτε την έκδοση 25.4 ή νεότερη.
- Ένα κιτ ανάπτυξης Java (JDK) εγκατεστημένο στο σύστημά σας
- Βασική γνώση προγραμματισμού Java και εξοικείωση με τα εργαλεία δημιουργίας Maven ή Gradle

## Ρύθμιση του Aspose.Slides για Java
Για να χρησιμοποιήσετε το Aspose.Slides, πρέπει πρώτα να συμπεριλάβετε τη βιβλιοθήκη στο έργο σας.

### Εξάρτηση Maven
Προσθέστε αυτό στο δικό σας `pom.xml`:
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```
### Εξάρτηση Gradle
Συμπεριλάβετε αυτό στο δικό σας `build.gradle`:
```gradle
implementation group: 'com.aspose', name: 'aspose-slides', version: '25.4', classifier: 'jdk16'
```
### Άμεση Λήψη
Εναλλακτικά, κατεβάστε την πιο πρόσφατη βιβλιοθήκη από [Aspose.Slides για εκδόσεις Java](https://releases.aspose.com/slides/java/).

#### Απόκτηση Άδειας
Μπορείτε να ξεκινήσετε με μια δωρεάν δοκιμαστική περίοδο ή να ζητήσετε μια προσωρινή άδεια χρήσης για να εξερευνήσετε όλες τις λειτουργίες. Για συνεχή χρήση, σκεφτείτε να αγοράσετε μια άδεια χρήσης.

#### Βασική Αρχικοποίηση και Ρύθμιση
```java
import com.aspose.slides.Presentation;
// Φορτώστε το αρχείο παρουσίασής σας
Presentation pres = new Presentation("YOUR_DOCUMENT_DIRECTORY/RenderingOptions.pptx");
```
## Οδηγός Εφαρμογής
### Λήψη μικρογραφιών διαφανειών με διάταξη σημειώσεων
Αυτή η λειτουργία σάς επιτρέπει να δημιουργείτε μικρογραφίες διαφανειών, διασφαλίζοντας παράλληλα ότι οι σημειώσεις εμφανίζονται στο κάτω μέρος, παρέχοντας περιεχόμενο και πρόσθετες πληροφορίες.
#### Βήμα 1: Φόρτωση της παρουσίασής σας
Αρχικά, φορτώστε το αρχείο παρουσίασής σας χρησιμοποιώντας το Aspose.Slides:
```java
import com.aspose.slides.Presentation;
import com.aspose.slides.INotesCommentsLayoutingOptions;
import com.aspose.slides.NotesPositions;
String presPath = "YOUR_DOCUMENT_DIRECTORY/RenderingOptions.pptx";
Presentation pres = new Presentation(presPath);
```
#### Βήμα 2: Ρύθμιση παραμέτρων επιλογών απόδοσης
Στη συνέχεια, ρυθμίστε τις επιλογές απόδοσης ώστε να συμπεριλαμβάνουν σημειώσεις στο κάτω μέρος:
```java
import com.aspose.slides.IRenderingOptions;
import com.aspose.slides.RenderingOptions;
IRenderingOptions renderingOpts = new RenderingOptions();
INotesCommentsLayoutingOptions notesOptions = new NotesCommentsLayoutingOptions();
// Ορισμός θέσης των σημειώσεων που θα περικοπούν στο κάτω μέρος
notesOptions.setNotesPosition(NotesPositions.BottomTruncated);
renderingOpts.setSlidesLayoutOptions(notesOptions);
```
#### Βήμα 3: Ανάκτηση και αποθήκευση της μικρογραφίας
Τέλος, ανακτήστε και αποθηκεύστε την εικόνα διαφάνειας με τις επιθυμητές διαστάσεις:
```java
import com.aspose.slides.IImage;
import java.io.IOException;
// Καθορίστε τη διαδρομή και τη μορφή εξόδου
String outputPath = "YOUR_OUTPUT_DIRECTORY/RenderingOptions-Slide1-Original.png";
try {
    IImage image = pres.getSlides().get_Item(0).getImage(renderingOpts, 4 / 3f, 4 / 3f);
    image.save(outputPath, com.aspose.slides.export.ImageFormat.getPng());
} catch (IOException e) {
    e.printStackTrace();
} finally {
    if (pres != null) pres.dispose();
}
```
### Αλλαγή προεπιλεγμένης κανονικής γραμματοσειράς
Αυτή η λειτουργία δείχνει πώς να αλλάξετε την προεπιλεγμένη κανονική γραμματοσειρά που χρησιμοποιείται στην απόδοση μικρογραφιών διαφανειών.
#### Βήμα 1: Φόρτωση της παρουσίασής σας
Ξεκινήστε φορτώνοντας το αρχείο παρουσίασής σας, όπως και στην προηγούμενη ενότητα:
```java
String presPath = "YOUR_DOCUMENT_DIRECTORY/RenderingOptions.pptx";
Presentation pres = new Presentation(presPath);
```
#### Βήμα 2: Ορισμός προεπιλεγμένης κανονικής γραμματοσειράς
Ρυθμίστε τις επιλογές απόδοσης για να χρησιμοποιήσετε μια συγκεκριμένη γραμματοσειρά, όπως Arial Black ή Arial Narrow:
```java
IRenderingOptions renderingOpts = new RenderingOptions();
renderingOpts.setDefaultRegularFont("Arial Black");
```
#### Βήμα 3: Ανάκτηση και αποθήκευση της μικρογραφίας με νέες ρυθμίσεις γραμματοσειράς
Αποθηκεύστε την εικόνα διαφάνειας χρησιμοποιώντας τις ενημερωμένες ρυθμίσεις γραμματοσειράς:
```java
String outputPath = "YOUR_OUTPUT_DIRECTORY/RenderingOptions-Slide1-ArialBlackDefault.png";
try {
    IImage image = pres.getSlides().get_Item(0).getImage(renderingOpts, 4 / 3f, 4 / 3f);
    image.save(outputPath, com.aspose.slides.export.ImageFormat.getPng());
} catch (IOException e) {
    e.printStackTrace();
} finally {
    if (pres != null) pres.dispose();
}
```

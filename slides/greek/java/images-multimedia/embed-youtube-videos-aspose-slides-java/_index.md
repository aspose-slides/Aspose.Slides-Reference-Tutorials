## Οδηγός Εφαρμογής

### Προσθήκη καρέ βίντεο από μια πηγή ιστού

Αυτή η λειτουργία σάς επιτρέπει να ενσωματώσετε ένα καρέ βίντεο απευθείας σε μια διαφάνεια του PowerPoint χρησιμοποιώντας μια διεύθυνση URL, όπως μία από το YouTube.

#### Βήμα 1: Αρχικοποίηση παρουσίασης
```java
Presentation pres = new Presentation();
try {
    // Συνεχίστε με την προσθήκη του βίντεο
} finally {
    if (pres != null) pres.dispose();
}
```
**Εξήγηση**Δημιουργούμε μια παρουσία του `Presentation`, το οποίο αντιπροσωπεύει το αρχείο PowerPoint σας. Το `finally` Το μπλοκ διασφαλίζει ότι οι πόροι απελευθερώνονται καλώντας `dispose()`.

#### Βήμα 2: Προσθήκη βίντεο YouTube σε διαφάνεια

Δείτε πώς μπορείτε να προσθέσετε ένα καρέ βίντεο στην πρώτη διαφάνεια:
```java
private static void addVideoFromYouTube(Presentation pres, String videoId) throws IOException {
    IVideoFrame videoFrame = pres.getSlides().get_Item(0).getShapes()
                              .addVideoFrame(10, 10, 427, 240,
                                             "https://www.youtube.com/embed/" + αναγνωριστικό βίντεο);
    
    videoFrame.setPlayMode(VideoPlayModePreset.Auto);

    String thumbnailUri = "https://www.youtube.com/watch?v=" + αναγνωριστικό βίντεο;
    URL url = new URL(thumbnailUri);
    URLConnection connection = url.openConnection();
    connection.setConnectTimeout(5000);  // Ορίστε το χρονικό όριο σύνδεσης σε 5 δευτερόλεπτα.
    connection.setReadTimeout(10000);   // Ορίστε το χρονικό όριο ανάγνωσης στα 10 δευτερόλεπτα.

    try (InputStream input = connection.getInputStream();
         ByteArrayOutputStream output = new ByteArrayOutputStream()) {
        byte[] buffer = new byte[8192];
        int count;
        while ((count = input.read(buffer)) > 0) {
            output.write(buffer, 0, count);
        }
        
        videoFrame.getPictureFormat().getPicture()
                   .setImage(pres.getImages().addImage(output.toByteArray()));
    }
}
```

**Εξήγηση**: 
- **Δημιουργία καρέ βίντεο**: Προσθέτουμε ένα νέο `IVideoFrame` στην πρώτη διαφάνεια σε καθορισμένες συντεταγμένες (10x10) και διαστάσεις (427x240).
- **Ορισμός λειτουργίας αναπαραγωγής**: Το βίντεο έχει ρυθμιστεί να αναπαράγεται αυτόματα όταν εμφανίζεται η διαφάνεια.
- **Ανάκτηση μικρογραφίας**Μια μικρογραφία YouTube ανακτάται χρησιμοποιώντας μια σύνδεση HTTP, με καθορισμένα χρονικά όρια για σταθερότητα.

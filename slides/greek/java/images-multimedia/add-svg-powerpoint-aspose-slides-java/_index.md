## Ρύθμιση του Aspose.Slides για Java

Για να ξεκινήσετε να χρησιμοποιείτε το Aspose.Slides στα έργα Java σας, θα πρέπει να το προσθέσετε ως εξάρτηση. Δείτε πώς μπορείτε να το κάνετε αυτό:

### Εγκατάσταση Maven

Προσθέστε την ακόλουθη εξάρτηση στο `pom.xml` αρχείο:

```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

### Εγκατάσταση Gradle

Συμπεριλάβετε τα ακόλουθα στο `build.gradle` αρχείο:

```gradle
implementation group: 'com.aspose', name: 'aspose-slides', version: '25.4', classifier: 'jdk16'
```

### Προσθήκη εικόνας SVG από εξωτερικό πόρο

#### Επισκόπηση

Αυτή η λειτουργία σάς επιτρέπει να διαβάσετε ένα αρχείο SVG και να το ενσωματώσετε απευθείας σε μια διαφάνεια PowerPoint, βελτιώνοντας την παρουσίασή σας με κλιμακούμενα γραφικά.

#### Βήματα για την εφαρμογή

##### Βήμα 1: Ορισμός διαδρομών αρχείων

Ξεκινήστε καθορίζοντας τις διαδρομές τόσο για την εικόνα SVG πηγής όσο και για το αρχείο PPTX εξόδου:

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
String outPptxPath = dataDir + "presentation_external.pptx";
```

##### Βήμα 2: Δημιουργήστε ένα αντικείμενο παρουσίασης

Αρχικοποίηση νέου `Presentation` αντικείμενο, το οποίο λειτουργεί ως κοντέινερ της πλατφόρμας διαφανειών σας:

```java
Presentation p = new Presentation();
```

##### Βήμα 3: Ανάγνωση περιεχομένου SVG

Χρησιμοποιήστε το πακέτο NIO της Java για να διαβάσετε το περιεχόμενο του αρχείου SVG σε μια συμβολοσειρά:

```java
String svgContent = new String(Files.readAllBytes(Paths.get(dataDir + "image1.svg")));
```

##### Βήμα 4: Προσθήκη της εικόνας SVG

Δημιουργήστε ένα `ISvgImage` αντικείμενο χρησιμοποιώντας το περιεχόμενο SVG και, στη συνέχεια, προσθέστε το στη συλλογή εικόνων της παρουσίασής σας:

```java
ISvgImage svgImage = new SvgImage(svgContent, new ExternalResourceResolver(), dataDir);
IPPImage ppImage = p.getImages().addImage(svgImage);
```

##### Βήμα 5: Προσθήκη κορνίζας

Ενσωματώστε το SVG σε ένα πλαίσιο εικόνας στην πρώτη διαφάνεια. Αυτό το βήμα τοποθετεί την εικόνα σας και ορίζει τις διαστάσεις της:

```java
p.getSlides().get_Item(0).getShapes().addPictureFrame(
    ShapeType.Rectangle,
    0, // Συντεταγμένη Χ
    0, // Συντεταγμένη Υ
    ppImage.getWidth(),
    ppImage.getHeight(),
    ppImage
);
```

##### Βήμα 6: Αποθήκευση της παρουσίασης

Τέλος, αποθηκεύστε την παρουσίασή σας σε μορφή PPTX:

```java
p.save(outPptxPath, SaveFormat.Pptx);
```

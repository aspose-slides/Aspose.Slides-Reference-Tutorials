# Προτεινόμενες ιδιότητες μόνο για ανάγνωση σε διαφάνειες Java


## Εισαγωγή στην ενεργοποίηση των προτεινόμενων ιδιοτήτων μόνο για ανάγνωση σε διαφάνειες Java

Σε αυτό το σεμινάριο, θα εξερευνήσουμε πώς να ενεργοποιήσετε τις ιδιότητες "Προτεινόμενες μόνο για ανάγνωση" για παρουσιάσεις PowerPoint χρησιμοποιώντας το Aspose.Slides για Java. Οι ιδιότητες "Προτεινόμενες μόνο για ανάγνωση" μπορούν να είναι χρήσιμες όταν θέλετε να ενθαρρύνετε τους χρήστες να προβάλουν μια παρουσίαση χωρίς να κάνουν αλλαγές. Αυτές οι ιδιότητες υποδεικνύουν ότι η παρουσίαση πρέπει να ανοίγει σε λειτουργία μόνο για ανάγνωση. Θα σας παρέχουμε έναν οδηγό βήμα προς βήμα μαζί με τον πηγαίο κώδικα Java για να το πετύχετε αυτό.

## Προαπαιτούμενα

Πριν ξεκινήσουμε, βεβαιωθείτε ότι έχετε εγκαταστήσει τη βιβλιοθήκη Aspose.Slides για Java στο έργο σας. Μπορείτε να την κατεβάσετε από το [Aspose.Slides για ιστότοπο Java](https://products.aspose.com/slides/java/).

## Βήμα 1: Δημιουργήστε μια νέα παρουσίαση PowerPoint

Θα ξεκινήσουμε δημιουργώντας μια νέα παρουσίαση PowerPoint χρησιμοποιώντας το Aspose.Slides για Java. Εάν έχετε ήδη μια παρουσίαση, μπορείτε να παραλείψετε αυτό το βήμα.

```java
String outPptxPath = "Your Output Directory" + "ReadOnlyRecommended.pptx";
Presentation pres = new Presentation();
```

Στον παραπάνω κώδικα, ορίσαμε τη διαδρομή για το αρχείο PowerPoint εξόδου και δημιουργήσαμε ένα νέο αντικείμενο παρουσίασης.

## Βήμα 2: Ενεργοποίηση συνιστώμενης ιδιότητας μόνο για ανάγνωση

Τώρα, ας ενεργοποιήσουμε την ιδιότητα "Προτεινόμενο μόνο για ανάγνωση" για την παρουσίαση.

```java
try
{
    pres.getProtectionManager().setReadOnlyRecommended(true);
    pres.save(outPptxPath, SaveFormat.Pptx);
}
finally
{
    if (pres != null) pres.dispose();
}
```

Σε αυτό το απόσπασμα κώδικα, χρησιμοποιούμε το `getProtectionManager().setReadOnlyRecommended(true)` μέθοδος για να ορίσετε την ιδιότητα "Συνιστάται μόνο για ανάγνωση" σε `true`Αυτό διασφαλίζει ότι όταν κάποιος ανοίξει την παρουσίαση, θα του ζητηθεί να την ανοίξει σε λειτουργία μόνο για ανάγνωση.

## Βήμα 3: Αποθήκευση της παρουσίασης

Τέλος, αποθηκεύουμε την παρουσίαση με ενεργοποιημένη την ιδιότητα "Συνιστάται μόνο για ανάγνωση".

## Πλήρης πηγαίος κώδικας για προτεινόμενες ιδιότητες μόνο για ανάγνωση σε διαφάνειες Java

```java
String outPptxPath = "Your Output Directory" + "ReadOnlyRecommended.pptx";
Presentation pres = new Presentation();
try
{
	pres.getProtectionManager().setReadOnlyRecommended(true);
	pres.save(outPptxPath, SaveFormat.Pptx);
}
finally
{
	if (pres != null) pres.dispose();
}
```

## Σύναψη

Σε αυτό το σεμινάριο, μάθατε πώς να ενεργοποιήσετε την ιδιότητα "Read-Only Recommended" (Συνιστάται μόνο για ανάγνωση) για μια παρουσίαση PowerPoint χρησιμοποιώντας το Aspose.Slides για Java. Αυτή η λειτουργία μπορεί να είναι χρήσιμη όταν θέλετε να περιορίσετε την επεξεργασία και να ενθαρρύνετε τους θεατές να χρησιμοποιούν την παρουσίαση σε λειτουργία μόνο για ανάγνωση. Μπορείτε να βελτιώσετε περαιτέρω την ασφάλεια ορίζοντας έναν κωδικό πρόσβασης για την παρουσίαση.

## Συχνές ερωτήσεις

### Πώς μπορώ να απενεργοποιήσω την ιδιότητα "Προτεινόμενο μόνο για ανάγνωση";

Για να απενεργοποιήσετε την ιδιότητα "Προτεινόμενο μόνο για ανάγνωση", απλώς χρησιμοποιήστε τον ακόλουθο κώδικα:

```java
pres.getProtectionManager().setReadOnlyRecommended(false);
```

### Μπορώ να ορίσω κωδικό πρόσβασης για μια παρουσίαση που προτείνεται μόνο για ανάγνωση;

Ναι, μπορείτε να ορίσετε έναν κωδικό πρόσβασης για μια παρουσίαση που συνιστάται μόνο για ανάγνωση χρησιμοποιώντας το Aspose.Slides για Java. Μπορείτε να χρησιμοποιήσετε το `setPassword` μέθοδος για να ορίσετε έναν κωδικό πρόσβασης για την παρουσίαση. Εάν έχει οριστεί κωδικός πρόσβασης, οι χρήστες θα πρέπει να τον εισαγάγουν για να ανοίξουν την παρουσίαση, ακόμα και σε λειτουργία μόνο για ανάγνωση.

```java
pres.getProtectionManager().setPassword("YourPassword");
```

Θυμηθείτε να αντικαταστήσετε `"YourPassword"` με τον επιθυμητό κωδικό πρόσβασης.

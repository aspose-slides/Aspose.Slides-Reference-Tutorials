# Δημιουργήστε και προσαρμόστε γραφήματα πίτας στο PowerPoint με το Aspose.Slides για Java

## Εισαγωγή

Αυτό το σεμινάριο σας καθοδηγεί στη δημιουργία και τη διαμόρφωση βασικών κυκλικών γραφημάτων, στην τροποποίηση δεδομένων γραφημάτων και στη συμπλήρωση δεδομένων σειρών χρησιμοποιώντας το Aspose.Slides για Java. Θα μάθετε επίσης πώς να βελτιστοποιείτε την απόδοση των παρουσιάσεων και να εφαρμόζετε αυτές τις τεχνικές σε σενάρια πραγματικού κόσμου.

- Συμπλήρωση σημείων δεδομένων σειράς και προσαρμογή παραλλαγών χρωμάτων
- Βελτιστοποίηση του Aspose.Slides για απόδοση Java

## Ρύθμιση του Aspose.Slides για Java
Για να χρησιμοποιήσετε το Aspose.Slides για Java, προσθέστε τη βιβλιοθήκη στο έργο σας:

**Εγκατάσταση Maven:**
Προσθέστε αυτήν την εξάρτηση στο δικό σας `pom.xml` αρχείο:
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

**Εγκατάσταση Gradle:**
Συμπεριλάβετε αυτό στο δικό σας `build.gradle` αρχείο:
```gradle
implementation group: 'com.aspose', name: 'aspose-slides', version: '25.4', classifier: 'jdk16'
```
Εναλλακτικά, [κατεβάστε την τελευταία έκδοση](https://releases.aspose.com/slides/java/) από το Aspose.Slides για εκδόσεις Java.

**Βήματα απόκτησης άδειας:**
- **Δωρεάν δοκιμή:** Ξεκινήστε με μια δωρεάν δοκιμή για να εξερευνήσετε τις λειτουργίες.
- **Προσωρινή Άδεια:** Για εκτεταμένη αξιολόγηση χωρίς περιορισμούς, ζητήστε προσωρινή άδεια [εδώ](https://purchase.aspose.com/temporary-license/).

## Οδηγός Εφαρμογής

### Δημιουργία και διαμόρφωση γραφήματος πίτας
Ακολουθήστε αυτά τα βήματα για να δημιουργήσετε ένα βασικό γράφημα πίτας στο PowerPoint χρησιμοποιώντας το Aspose.Slides για Java.

**1. Δημιουργήστε την Κλάση Παρουσίασης**
Δημιουργήστε ένα `Presentation` αντικείμενο που αντιπροσωπεύει το αρχείο PPTX σας:
```java
import com.aspose.slides.Presentation;
import com.aspose.slides.ISlide;
import com.aspose.slides.IChart;
// Δημιουργήστε μια παρουσία της κλάσης Presentation
Presentation presentation = new Presentation();
```

**2. Πρόσβαση στην πρώτη διαφάνεια**
Αποκτήστε πρόσβαση στην πρώτη διαφάνεια από το `presentation` αντικείμενο:
```java
ISlide slides = presentation.getSlides().get_Item(0);
```

**3. Προσθέστε ένα γράφημα πίτας στη διαφάνεια**
Προσθήκη και διαμόρφωση ενός γραφήματος πίτας με προεπιλεγμένα δεδομένα σε καθορισμένες συντεταγμένες (x, y) και μέγεθος (πλάτος, ύψος):
```java
IChart chart = slides.getShapes().addChart(com.aspose.slides.ChartType.Pie, 100, 100, 400, 400);
```

**4. Ορίστε τον τίτλο του γραφήματος**
Προσαρμόστε το γράφημα πίτας σας με έναν τίτλο:
```java
chart.getChartTitle().addTextFrameForOverriding("Sample Title");
chart.getChartTitle().getTextFrameForOverriding().getTextFrameFormat().setCenterText(true);
chart.getChartTitle().setHeight(20);
chart.setTitle(true);
```

**5. Απόρριψη Πόρων**
Βεβαιωθείτε ότι οι πόροι απελευθερώνονται μετά τη χρήση:
```java
try {
    // Οι λειτουργίες του γραφήματος σας εδώ
} finally {
    if (presentation != null) presentation.dispose();
}
```

### Τροποποίηση Δεδομένων και Σειρών Γραφήματος
Τροποποιήστε τα υπάρχοντα δεδομένα γραφήματος διαγράφοντας τις προεπιλεγμένες σειρές και κατηγορίες και, στη συνέχεια, προσθέτοντας νέες.

**1. Διαγραφή προεπιλεγμένων σειρών και κατηγοριών**
Αποκτήστε πρόσβαση στην πρώτη διαφάνεια και αρχικοποιήστε το γράφημα πίτας:
```java
ISlide slides = presentation.getSlides().get_Item(0);
IChart chart = slides.getShapes().addChart(com.aspose.slides.ChartType.Pie, 100, 100, 400, 400);
// Διαγραφή προεπιλεγμένων σειρών και κατηγοριών
chart.getChartData().getSeries().clear();
chart.getChartData().getCategories().clear();
```

**2. Προσθήκη νέων κατηγοριών**
Ορίστε νέες κατηγορίες για τα δεδομένα σας:
```java
IChartDataWorkbook fact = chart.getChartData().getChartDataWorkbook();
chart.getChartData().getCategories().add(fact.getCell(0, 1, 0, "First Qtr"));
chart.getChartData().getCategories().add(fact.getCell(0, 2, 0, "2nd Qtr"));
chart.getChartData().getCategories().add(fact.getCell(0, 3, 0, "3rd Qtr"));
```

**3. Προσθήκη νέας σειράς**
Εισαγωγή μιας νέας σειράς στο chart:
```java
IChartSeries series = chart.getChartData().getSeries().add(fact.getCell(0, 0, 1, "Series 1"), chart.getType());
```

### Συμπλήρωση δεδομένων σειράς και αποθήκευση της παρουσίασης
Συμπληρώστε σημεία δεδομένων σειράς για ένα γράφημα πίτας, προσαρμόστε τις παραλλαγές χρωμάτων και αποθηκεύστε την παρουσίασή σας.

**1. Συμπλήρωση δεδομένων σειράς**
Συμπληρώστε το γράφημα με συγκεκριμένα σημεία δεδομένων:
```java
series.getDataPoints().addDataPointForPieSeries(fact.getCell(0, 1, 1, 20));
series.getDataPoints().addDataPointForPieSeries(fact.getCell(0, 2, 1, 50));
series.getDataPoints().addDataPointForPieSeries(fact.getCell(0, 3, 1, 30));
// Ενεργοποίηση ποικίλου χρώματος για κάθε φέτα
series.getParentSeriesGroup().setColorVaried(true);
```

**2. Αποθήκευση της παρουσίασης**
Αποθηκεύστε τις αλλαγές σας σε έναν καθορισμένο κατάλογο:
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
presentation.save(dataDir + "Pie.pptx", com.aspose.slides.SaveFormat.Pptx);
```

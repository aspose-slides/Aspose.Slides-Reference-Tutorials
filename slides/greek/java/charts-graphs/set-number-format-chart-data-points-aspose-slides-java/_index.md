# Πώς να ορίσετε μορφές αριθμών σε σημεία δεδομένων γραφήματος χρησιμοποιώντας το Aspose.Slides για Java

Δυσκολεύεστε με τη μορφοποίηση σημείων δεδομένων σε γραφήματα χρησιμοποιώντας Java; Είτε προετοιμάζετε παρουσιάσεις είτε χρειάζεστε ακριβείς μορφές αριθμών, αυτό το σεμινάριο θα σας καθοδηγήσει στη ρύθμιση και την προσαρμογή των σημείων δεδομένων του γραφήματος σας με το Aspose.Slides. 

**Τι θα μάθετε:**
- Πώς να ορίσετε μια προκαθορισμένη μορφή αριθμού για σημεία δεδομένων γραφήματος
- Δημιουργία καταλόγων για αποθήκευση εγγράφων σε Java
- Ρύθμιση του Aspose.Slides για Java χρησιμοποιώντας Maven ή Gradle

Ας δούμε τις προϋποθέσεις πριν ξεκινήσουμε!

## Προαπαιτούμενα

Πριν ξεκινήσουμε, βεβαιωθείτε ότι έχετε τα εξής:

1. **Απαιτούμενες βιβλιοθήκες και εκδόσεις:**
   - Aspose.Slides για Java έκδοση 25.4.

2. **Απαιτήσεις Ρύθμισης Περιβάλλοντος:**
   - JDK 16 ή νεότερη έκδοση εγκατεστημένη στον υπολογιστή σας.
   - Ολοκληρωμένο Περιβάλλον Ανάπτυξης (IDE) όπως το IntelliJ IDEA ή το Eclipse.

3. **Προαπαιτούμενα Γνώσεων:**
   - Βασική κατανόηση του προγραμματισμού Java.
   - Εξοικείωση με το Maven ή το Gradle για διαχείριση εξαρτήσεων.

## Ρύθμιση του Aspose.Slides για Java

Για να χρησιμοποιήσετε το Aspose.Slides στο έργο σας, μπορείτε να το προσθέσετε μέσω του Maven ή του Gradle:

**Εξάρτηση Maven:**

```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

**Υλοποίηση Gradle:**

```gradle
implementation group: 'com.aspose', name: 'aspose-slides', version: '25.4', classifier: 'jdk16'
```

Εναλλακτικά, μπορείτε να κατεβάσετε απευθείας την τελευταία έκδοση από [Aspose.Slides για εκδόσεις Java](https://releases.aspose.com/slides/java/).

### Απόκτηση Άδειας

- **Δωρεάν δοκιμή:** Ξεκινήστε με μια δωρεάν δοκιμή για να εξερευνήσετε τις δυνατότητες.
- **Προσωρινή Άδεια:** Υποβάλετε αίτηση για προσωρινή άδεια χρήσης εάν χρειάζεστε εκτεταμένη πρόσβαση χωρίς αγορά.
- **Αγορά:** Αγοράστε μια άδεια χρήσης για συνεχή χρήση.

Αρχικοποιήστε το έργο σας ρυθμίζοντας το περιβάλλον όπως περιγράφεται παραπάνω και διασφαλίζοντας ότι όλες οι απαραίτητες εξαρτήσεις περιλαμβάνονται στη διαμόρφωση κατασκευής σας.

## Οδηγός Εφαρμογής

### Ορισμός μορφών αριθμών σε σημεία δεδομένων γραφήματος

#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να προσαρμόσετε τον τρόπο εμφάνισης των αριθμητικών δεδομένων σε σειρές γραφημάτων, βελτιώνοντας την αναγνωσιμότητα με προκαθορισμένες μορφές όπως ποσοστά ή νομίσματα.

**Βήμα 1: Αρχικοποίηση παρουσίασης**

```java
// Εισαγάγετε τις απαραίτητες κλάσεις Aspose.Slides
import com.aspose.slides.*;

public class SetNumberFormatInChartDataPoints {
    public static void main(String[] args) {
        // Δημιουργήστε ένα νέο αντικείμενο παρουσίασης
        Presentation pres = new Presentation();
```

- **Γιατί:** Αρχικοποίηση του `Presentation` Το αντικείμενο είναι κρίσιμο καθώς χρησιμεύει ως δοχείο για τις διαφάνειες και τα γραφήματά σας.

**Βήμα 2: Προσθήκη γραφήματος σε διαφάνεια**

```java
        try {
            // Πρόσβαση στην πρώτη διαφάνεια της παρουσίασης
            ISlide slide = pres.getSlides().get_Item(0);

            // Προσθήκη γραφήματος ομαδοποιημένων στηλών στη διαφάνεια
            IChart chart = slide.getShapes().addChart(ChartType.ClusteredColumn, 50, 50, 500, 400);
```

- **Γιατί:** Η προσθήκη ενός γραφήματος σε συγκεκριμένες συντεταγμένες βοηθά στην ακριβή τοποθέτησή του στο σημείο που χρειάζεται στην παρουσίασή σας.

**Βήμα 3: Πρόσβαση και μορφοποίηση δεδομένων σειράς**

```java
            // Λήψη της συλλογής σειρών των δεδομένων γραφήματος
            IChartSeriesCollection series = chart.getChartData().getSeries();

            // Επαναλάβετε κάθε σειρά για να μορφοποιήσετε σημεία δεδομένων
            for (IChartSeries ser : series) {
                for (IChartDataPoint cell : ser.getDataPoints()) {
                    // Ορισμός προκαθορισμένης μορφής αριθμού για την τιμή δεδομένων
                    cell.getValue().getAsCell().setPresetNumberFormat((byte) 10); // Μορφή: 0,00%
                }
            }
```

- **Γιατί:** Η επανάληψη σειρών και η μορφοποίηση κάθε σημείου δεδομένων διασφαλίζει τη συνεπή παρουσίαση των αριθμητικών τιμών.

**Βήμα 4: Αποθηκεύστε την παρουσίασή σας**

```java
            // Αποθήκευση της ενημερωμένης παρουσίασης με μορφοποιημένα γραφήματα
            pres.save("YOUR_OUTPUT_DIRECTORY/PresetNumberFormat_out.pptx", SaveFormat.Pptx);
        } finally {
            if (pres != null) pres.dispose();
        }
    }
}
```

- **Γιατί:** Η σωστή αποθήκευση και διάθεση πόρων αποτρέπει τις διαρροές μνήμης και διασφαλίζει ότι η εργασία σας αποθηκεύεται.

### Δημιουργία και διαχείριση καταλόγων για αποθήκευση εγγράφων

#### Επισκόπηση
Αυτή η λειτουργικότητα ελέγχει την ύπαρξη ενός καταλόγου και τον δημιουργεί εάν είναι απαραίτητο, διασφαλίζοντας ότι τα έγγραφα έχουν μια καθορισμένη θέση αποθήκευσης.

**Βήμα 1: Ορισμός διαδρομής καταλόγου**

```java
import java.io.File;

public class DirectoryManagement {
    public static void main(String[] args) {
        // Ορίστε τη διαδρομή του καταλόγου εγγράφων σας
        String dataDir = "YOUR_DOCUMENT_DIRECTORY";
```

- **Γιατί:** Ο ορισμός της διαδρομής καταλόγου είναι απαραίτητος για τη διαχείριση και την οργάνωση αρχείων.

**Βήμα 2: Έλεγχος και δημιουργία καταλόγου**

```java
        // Επαληθεύστε αν ο κατάλογος υπάρχει. Δημιουργήστε τον αν όχι.
        boolean IsExists = new File(dataDir).exists();
        if (!IsExists) {
            new File(dataDir).mkdirs(); // Δημιουργεί καταλόγους αναδρομικά
        }
    }
}
```

- **Γιατί:** Η διασφάλιση της ύπαρξης του καταλόγου σας πριν από την προσπάθεια αποθήκευσης αρχείων αποτρέπει σφάλματα χρόνου εκτέλεσης.

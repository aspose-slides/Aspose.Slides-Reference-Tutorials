Για να ενσωματώσετε το Aspose.Slides στο έργο σας, μπορείτε να χρησιμοποιήσετε είτε το Maven είτε το Gradle ως εργαλείο δημιουργίας. Δείτε πώς μπορείτε να το προσθέσετε:
**Maven**
```xml

    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```
**Γκράντλ**

## Οδηγός Εφαρμογής
Σε αυτήν την ενότητα, θα σας καθοδηγήσουμε βήμα προς βήμα στη δημιουργία ενός γραφήματος με τίτλο περιστρεφόμενου άξονα.
### Προσθήκη γραφήματος ομαδοποιημένων στηλών
**Επισκόπηση**Ας ξεκινήσουμε προσθέτοντας ένα γράφημα ομαδοποιημένων στηλών στη διαφάνειά σας.
#### Βήμα 1: Δημιουργήστε την παρουσίαση
Αρχικοποίηση μιας νέας παρουσίας παρουσίασης:
```java
Presentation pres = new Presentation();
```
Αυτή η γραμμή κώδικα δημιουργεί ένα κενό αρχείο PowerPoint για χειρισμό.
#### Βήμα 2: Προσθήκη γραφήματος ομαδοποιημένων στηλών
Προσθέστε ένα γράφημα στην πρώτη διαφάνεια στη θέση (50, 50) με διαστάσεις (450, 300):
```java
IChart chart = pres.getSlides().get_Item(0).getShapes().addChart(ChartType.ClusteredColumn, 50, 50, 450, 300);
```
Εδώ, `ChartType.ClusteredColumn` καθορίζει τον τύπο του γραφήματος. Μπορείτε να το αλλάξετε σε άλλους τύπους, όπως `Pie`, `Bar`, κ.λπ., ανάλογα με τις ανάγκες σας.
#### Βήμα 3: Ενεργοποίηση και περιστροφή τίτλου κάθετου άξονα
Στη συνέχεια, ενεργοποιήστε τον τίτλο για τον κατακόρυφο άξονα και ορίστε τη γωνία περιστροφής του:
```java
// Ενεργοποίηση του τίτλου του κάθετου άξονα.
chart.getAxes().getVerticalAxis().setTitle(true);

// Ρυθμίστε τη γωνία περιστροφής στις 90 μοίρες.
chart.getAxes().getVerticalAxis().getTitle().getTextFormat().getTextBlockFormat().setRotationAngle(90);
```
Ο `setRotationAngle` Η μέθοδος σάς επιτρέπει να προσαρμόσετε τον προσανατολισμό του κειμένου, βελτιώνοντας την αναγνωσιμότητα όταν ο χώρος είναι περιορισμένος.
#### Βήμα 4: Αποθήκευση της παρουσίασης
Τέλος, αποθηκεύστε τις αλλαγές σας:
```java
pres.save("YOUR_DOCUMENT_DIRECTORY/test.pptx", SaveFormat.Pptx);
```
Αντικαταστήστε το "YOUR_DOCUMENT_DIRECTORY" με την πραγματική διαδρομή όπου θέλετε να αποθηκεύσετε την παρουσίασή σας.

# Πώς να προσθέσετε συγκεντρωτικό διάγραμμα στήλης στο PowerPoint χρησιμοποιώντας το Aspose.Slides for Java

## Εισαγωγή

Όταν χρειάζεται να **προσθέσετε συγκεντρωτικό διάγραμμα στήλης** σε μια παρουσίαση PowerPoint, μια σαφής οπτική απεικόνιση μπορεί να μετατρέψει ακατέργαστους αριθμούς σε μια άμεσα κατανοητή ιστορία. Η χειροκίνητη δημιουργία αυτού στο PowerPoint μπορεί να είναι χρονοβόρα, ειδικά όταν πρέπει να δημιουργήσετε πολλές διαφάνειες προγραμματιστικά. Το **Aspose.Slides for Java** αφαιρεί αυτό το εμπόδιο – σας επιτρέπει να δημιουργήσετε, να προσαρμόσετε διαγράμματα PowerPoint και να εισάγετε δεδομένα σε σειρά διαγράμματος με μερικές μόνο γραμμές κώδικα.

Σε αυτό το tutorial θα μάθετε πώς να:
- Αρχικοποιήσετε μια νέα παρουσίαση PowerPoint με το Aspose.Slides for Java.
- **Προσθέσετε διάγραμμα στη διαφάνεια** και να το ρυθμίσετε ως συγκεντρωτικό διάγραμμα στήλης.
- **Δημιουργήσετε ομαδοποιημένο διάγραμμα στήλης** ορίζοντας επίπεδα ομαδοποίησης για τις κατηγορίες.
- **Εισάγετε σειρά δεδομένων στο διάγραμμα** ώστε τα δεδομένα σας να εμφανίζονται σωστά.
- Αποθηκεύσετε την τελική παρουσίαση ως αρχείο PPTX.

Ας βεβαιωθούμε ότι έχετε όλα όσα χρειάζεστε πριν βουτήξουμε στον κώδικα.

## Γρήγορες Απαντήσεις
- **Ποια είναι η κύρια κλάση;** `Presentation` από το `com.aspose.slides`.
- **Ποιος τύπος διαγράμματος χρησιμοποιείται;** `ChartType.ClusteredColumn`.
- **Χρειάζεται άδεια για δοκιμή;** Μια δωρεάν δοκιμή λειτουργεί, αλλά μια άδεια αφαιρεί τους περιορισμούς αξιολόγησης.
- **Ποια έκδοση Java υποστηρίζεται;** JDK 16 ή νεότερη (το παράδειγμα χρησιμοποιεί JDK 16).
- **Πώς εκτελείται το δείγμα;** Προσθέστε την εξάρτηση Maven/Gradle, κάντε compile και τρέξτε τη μέθοδο `main`.

## Τι είναι το «προσθήκη συγκεντρωτικού διαγράμματος στήλης»;

Ένα *συγκεντρωτικό διάγραμμα στήλης* (επίσης γνωστό ως ομαδοποιημένο διάγραμμα στήλης) εμφανίζει πολλαπλές σειρές δεδομένων πλάι‑πλάι για κάθε κατηγορία, καθιστώντας εύκολη τη σύγκριση τιμών μεταξύ ομάδων. Στο PowerPoint αυτός ο τύπος διαγράμματος είναι ιδανικός για τριμηνιαίες πωλήσεις, αποτελέσματα ερευνών ή οποιοδήποτε σενάριο όπου χρειάζεται να συγκρίνετε πολλά σύνολα δεδομένων στην ίδια κατηγορία.

## Οδηγός Υλοποίησης

Θα περάσουμε βήμα‑βήμα, εξηγώντας **πώς να προσθέσετε διάγραμμα** και **πώς να προσαρμόσετε το διάγραμμα PowerPoint** κατά τη διάρκεια.

### Αρχικοποίηση Παρουσίασης

Αρχικά, δημιουργήστε ένα νέο αντικείμενο `Presentation` και πάρτε τη προεπιλεγμένη διαφάνεια.

```java
import com.aspose.slides.*;

// Feature: Initialize Presentation
Presentation pres = new Presentation();
ISlide slide = pres.getSlides().get_Item(0);
```

### Προσθήκη Διαγράμματος στη Διαφάνεια

Τώρα **προσθέτουμε διάγραμμα στη διαφάνεια** χρησιμοποιώντας τον τύπο `ClusteredColumn` και καθαρίζουμε τυχόν προεπιλεγμένα δεδομένα.

```java
// Feature: Add Chart to Slide
IChart ch = pres.getSlides().get_Item(0).getShapes().addChart(
    ChartType.ClusteredColumn, 100, 100, 600, 450);
ch.getChartData().getSeries().clear();
ch.getChartData().getCategories().clear();
```

### Προετοιμασία Εγγραφής Δεδομένων Διαγράμματος

Το διάγραμμα αποθηκεύει τα δεδομένα του σε ένα εσωτερικό workbook. Το καθαρίζουμε για να ξεκινήσουμε από την αρχή.

```java
// Feature: Prepare Chart Data Workbook
IChartDataWorkbook fact = ch.getChartData().getChartDataWorkbook();
fact.clear(0);
int defaultWorksheetIndex = 0;
```

### Προσθήκη Κατηγοριών με Επίπεδα Ομαδοποίησης

Η ομαδοποίηση των κατηγοριών δημιουργεί το εφέ **ομαδοποιημένου διαγράμματος στήλης**. Κάθε κατηγορία μπορεί να ανήκει σε λογική ομάδα.

```java
// Feature: Add Categories with Grouping Levels
IChartCategory category = ch.getChartData().getCategories().add(
    fact.getCell(0, "c2", "A"));
category.getGroupingLevels().setGroupingItem(1, "Group1");

category = ch.getChartData().getCategories().add(fact.getCell(0, "c3", "B"));
// Repeat for other categories
```

### Προσθήκη Σειρών Δεδομένων στο Διάγραμμα

Εδώ **εισάγουμε σειρά δεδομένων στο διάγραμμα** που θα εμφανιστούν ως ξεχωριστές στήλες.

```java
// Feature: Add Data Series to Chart
IChartSeries series = ch.getChartData().getSeries().add(
    fact.getCell(0, "D1", "Series 1"), ChartType.ClusteredColumn);
series.getDataPoints().addDataPointForBarSeries(fact.getCell(defaultWorksheetIndex, "D2", 10));
// Continue adding data points
```

### Αποθήκευση Παρουσίασης με Διάγραμμα

Τέλος, γράψτε το αρχείο PPTX στο δίσκο.

```java
// Feature: Save Presentation with Chart
pres.save("YOUR_OUTPUT_DIRECTORY/AsposeChart_out.pptx", SaveFormat.Pptx);
```

## Συνηθισμένα Πίπ και Συμβουλές

- **Πίπ:** Ξεχάσατε να καθαρίσετε τις υπάρχουσες σειρές/κατηγορίες, με αποτέλεσμα διπλότυπα δεδομένα.  
  **Συμβουλή:** Πάντα καλέστε `clear()` πριν γεμίσετε νέα δεδομένα.  
- **Πίπ:** Χρήση λανθασμένης διεύθυνσης κελιού (π.χ. `"c2"` αντί για `"C2"`).  
  **Συμβουλή:** Οι αναφορές κελιών είναι case‑insensitive, αλλά κρατήστε τη συνέπεια για ευανάγνωστη κωδικοποίηση.  
- **Συμβουλή:** Χρησιμοποιήστε `setGroupingItem` για να δημιουργήσετε ουσιαστικές ετικέτες ομάδας· εμφανίζονται αυτόματα στη λεζάντα του διαγράμματος.

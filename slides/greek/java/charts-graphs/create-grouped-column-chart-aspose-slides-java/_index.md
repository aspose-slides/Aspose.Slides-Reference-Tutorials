# Δημιουργία ομαδοποιημένων γραφημάτων στηλών στο PowerPoint χρησιμοποιώντας το Aspose.Slides για Java

## Εισαγωγή

Κατά την παρουσίαση δεδομένων, οι οπτικές αναπαραστάσεις συχνά μεταφέρουν πληροφορίες πιο αποτελεσματικά από τους ακατέργαστους αριθμούς μόνο. Ωστόσο, η δημιουργία οπτικά ελκυστικών και ενημερωτικών γραφημάτων μπορεί να είναι δυσκίνητη χωρίς τα κατάλληλα εργαλεία. **Aspose.Slides για Java** απλοποιεί αυτήν τη διαδικασία, επιτρέποντάς σας να προσθέσετε ένα ομαδοποιημένο γράφημα στηλών σε μια παρουσίαση PowerPoint χωρίς κόπο.

Σε αυτό το σεμινάριο, θα μάθετε πώς να:
- Αρχικοποιήστε μια νέα παρουσίαση PowerPoint με το Aspose.Slides για Java.
- Προσθέστε και προσαρμόστε γραφήματα ομαδοποιημένων στηλών σε διαφάνειες.
- Ομαδοποιήστε κατηγορίες μέσα στο γράφημα για βελτιωμένη οπτικοποίηση.
- Εισαγάγετε αποτελεσματικά σειρές δεδομένων στο γράφημά σας.
- Αποθηκεύστε την παρουσίασή σας σε μορφή PPTX.

Ας ξεκινήσουμε εξετάζοντας τις απαραίτητες προϋποθέσεις πριν ξεκινήσουμε τον προγραμματισμό!

## Ρύθμιση του Aspose.Slides για Java

Για να ενσωματώσετε τη βιβλιοθήκη Aspose.Slides στο έργο Java σας, ακολουθήστε τα παρακάτω βήματα χρησιμοποιώντας το Maven ή το Gradle:

**Maven**

```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

**Γκράντλ**

```gradle
implementation group: 'com.aspose', name: 'aspose-slides', version: '25.4', classifier: 'jdk16'
```

Εναλλακτικά, μπορείτε να κατεβάσετε απευθείας την τελευταία έκδοση από [Aspose.Slides για εκδόσεις Java](https://releases.aspose.com/slides/java/).

## Οδηγός Εφαρμογής

Θα αναλύσουμε τη διαδικασία σε λογικά βήματα, εστιάζοντας σε συγκεκριμένα χαρακτηριστικά του Aspose.Slides.

### Αρχικοποίηση παρουσίασης

Ξεκινήστε δημιουργώντας μια παρουσία του `Presentation` τάξη:

```java
import com.aspose.slides.*;

// Χαρακτηριστικό: Αρχικοποίηση παρουσίασης
Presentation pres = new Presentation();
ISlide slide = pres.getSlides().get_Item(0);
```

Εδώ, ξεκινάμε μια νέα παρουσίαση και επιλέγουμε την πρώτη διαφάνεια. Αυτή χρησιμεύει ως καμβάς για την προσθήκη γραφημάτων.

### Προσθήκη γραφήματος σε διαφάνεια

Στη συνέχεια, προσθέστε ένα γράφημα ομαδοποιημένων στηλών στην επιλεγμένη διαφάνεια:

```java
// Χαρακτηριστικό: Προσθήκη γραφήματος σε διαφάνεια
IChart ch = pres.getSlides().get_Item(0).getShapes().addChart(
    ChartType.ClusteredColumn, 100, 100, 600, 450);
ch.getChartData().getSeries().clear();
ch.getChartData().getCategories().clear();
```

Αυτό το απόσπασμα δημιουργεί ένα γράφημα τύπου `ClusteredColumn` με καθορισμένες διαστάσεις και το τοποθετεί στη διαφάνεια. Επίσης, διαγράφει τυχόν υπάρχουσες σειρές ή κατηγορίες για να ξεκινήσει από την αρχή.

### Προετοιμασία βιβλίου εργασίας δεδομένων γραφήματος

Για να διαχειριστείτε τα δεδομένα του γραφήματός σας, ετοιμάστε ένα βιβλίο εργασίας:

```java
// Χαρακτηριστικό: Προετοιμασία βιβλίου εργασίας δεδομένων γραφήματος
IChartDataWorkbook fact = ch.getChartData().getChartDataWorkbook();
fact.clear(0);
int defaultWorksheetIndex = 0;
```

Ο `IChartDataWorkbook` Το αντικείμενο λειτουργεί ως το δοχείο δεδομένων για το γράφημά σας, επιτρέποντάς σας να χειρίζεστε αποτελεσματικά τα σημεία δεδομένων.

### Προσθήκη κατηγοριών με επίπεδα ομαδοποίησης

Η ομαδοποίηση κατηγοριών βοηθά στην οργάνωση των δεδομένων με ουσιαστικό τρόπο. Δείτε πώς:

```java
// Χαρακτηριστικό: Προσθήκη κατηγοριών με επίπεδα ομαδοποίησης
IChartCategory category = ch.getChartData().getCategories().add(
    fact.getCell(0, "c2", "A"));
category.getGroupingLevels().setGroupingItem(1, "Group1");

category = ch.getChartData().getCategories().add(fact.getCell(0, "c3", "B"));
// Επαναλάβετε για άλλες κατηγορίες
```

Κάθε κατηγορία προστίθεται με ένα συγκεκριμένο επίπεδο ομαδοποίησης. Αυτό σας επιτρέπει να ορίσετε λογικές ομαδοποιήσεις μέσα στο γράφημά σας.

### Προσθήκη σειράς δεδομένων σε γράφημα

Για να οπτικοποιήσετε δεδομένα, προσθέστε σειρές στο γράφημα:

```java
// Χαρακτηριστικό: Προσθήκη σειράς δεδομένων σε γράφημα
IChartSeries series = ch.getChartData().getSeries().add(
    fact.getCell(0, "D1", "Series 1"), ChartType.ClusteredColumn);
series.getDataPoints().addDataPointForBarSeries(fact.getCell(defaultWorksheetIndex, "D2", 10));
// Συνέχεια προσθήκης σημείων δεδομένων
```

Ο `IChartSeries` Το αντικείμενο χρησιμοποιείται για την προσθήκη μιας σειράς σημείων δεδομένων, τα οποία αντιπροσωπεύουν τα πραγματικά δεδομένα στο γράφημά σας.

### Αποθήκευση παρουσίασης με γράφημα

Τέλος, αποθηκεύστε την παρουσίασή σας:

```java
// Χαρακτηριστικό: Αποθήκευση παρουσίασης με γράφημα
pres.save("YOUR_OUTPUT_DIRECTORY/AsposeChart_out.pptx", SaveFormat.Pptx);
```

Αυτό το βήμα εγγράφει όλες τις αλλαγές σε ένα αρχείο PPTX στον καθορισμένο κατάλογο.

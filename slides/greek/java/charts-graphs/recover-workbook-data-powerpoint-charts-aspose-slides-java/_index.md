### Ανάκτηση βιβλίου εργασίας από την προσωρινή μνήμη γραφημάτων
#### Επισκόπηση
Αυτή η λειτουργία επιτρέπει την πρόσβαση και την ανάκτηση δεδομένων βιβλίου εργασίας που είναι ενσωματωμένα σε γραφήματα μέσα σε παρουσιάσεις PowerPoint, διασφαλίζοντας ότι δεν θα υπάρξει απώλεια δεδομένων κατά τη διάρκεια των διαδικασιών μετατροπής ή επεξεργασίας.

#### Βήμα προς βήμα εφαρμογή
##### Ορισμός επιλογών φόρτωσης για ανάκτηση
Διαμορφώστε το `LoadOptions` για να ενεργοποιήσετε την ανάκτηση βιβλίου εργασίας:
```java
import com.aspose.slides.*;

String pptxFile = "YOUR_DOCUMENT_DIRECTORY/ExternalWB.pptx";
String outPptxFile = "YOUR_OUTPUT_DIRECTORY/ExternalWB_out.pptx";

// Βήμα 1: Ορίστε την επιλογή LoadOptions για να ανακτήσετε το βιβλίο εργασίας από την προσωρινή μνήμη γραφημάτων.
LoadOptions lo = new LoadOptions();
lo.getSpreadsheetOptions().setRecoverWorkbookFromChartCache(true);
```
Εδώ, `setRecoverWorkbookFromChartCache(true)` είναι κρίσιμο καθώς δίνει εντολή στο Aspose.Slides να ανακτήσει τυχόν ενσωματωμένα βιβλία εργασίας σε γραφήματα.

##### Φόρτωση παρουσίασης με επιλογές
Φορτώστε το αρχείο PowerPoint χρησιμοποιώντας αυτές τις επιλογές:
```java
// Βήμα 2: Φορτώστε την παρουσίαση με τις καθορισμένες επιλογές LoadOptions.
Presentation pres = new Presentation(pptxFile, lo);
```
Αυτό το βήμα διασφαλίζει ότι όλα τα απαραίτητα δεδομένα είναι έτοιμα για ανάκτηση.

##### Πρόσβαση και ανάκτηση δεδομένων
Στη συνέχεια, αποκτήστε πρόσβαση στο γράφημα και ανακτήστε τα σχετικά δεδομένα του βιβλίου εργασίας:
```java
try {
    // Βήμα 3: Αποκτήστε πρόσβαση στο πρώτο γράφημα στην πρώτη διαφάνεια.
    IChart chart = (IChart) pres.getSlides().get_Item(0).getShapes().get_Item(0);

    // Βήμα 4: Ανακτήστε το βιβλίο εργασίας δεδομένων που σχετίζεται με το γράφημα.
    IChartDataWorkbook wb = chart.getChartData().getChartDataWorkbook();
    
    // Βήμα 5: Αποθηκεύστε την παρουσίαση σε ένα νέο αρχείο.
    pres.save(outPptxFile, SaveFormat.Pptx);
} finally {
    if (pres != null) pres.dispose();
}
```
Σε αυτό το απόσπασμα:
- Αποκτούμε πρόσβαση στο πρώτο διάγραμμα και στο βιβλίο εργασίας δεδομένων του.
- Τέλος, αποθηκεύουμε την τροποποιημένη παρουσίαση.

### Ρύθμιση παραμέτρων LoadOptions
#### Επισκόπηση
Ρύθμιση παραμέτρων `LoadOptions` σας επιτρέπει αποτελεσματικά να ελέγχετε τον τρόπο διαχείρισης των ενσωματωμένων βιβλίων εργασίας κατά τη διάρκεια των εργασιών φόρτωσης.

#### Λεπτομερής εξήγηση
```java
// ΧΑΡΑΚΤΗΡΙΣΤΙΚΟ: Διαμόρφωση LoadOptions
import com.aspose.slides.*;

Επιλογές Φόρτωσης lo = new LoadOptions();
lo.getSpreadsheetOptions().setRecoverWorkbookFromChartCache(true);
```
- **LoadOptions**: Ρυθμίζει τη διαμόρφωση για τη φόρτωση της παρουσίασης.
- **getSpreadsheetOptions()**: Παρέχει πρόσβαση σε ρυθμίσεις που σχετίζονται με ενσωματωμένα υπολογιστικά φύλλα.
- **setRecoverWorkbookFromChartCache(true)**: Επιτρέπει την ανάκτηση δεδομένων βιβλίου εργασίας από προσωρινές μνήμες γραφημάτων.

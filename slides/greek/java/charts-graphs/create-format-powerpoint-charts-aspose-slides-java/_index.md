# Προσθήκη Στοιβαδίου Στήλης Γραφήματος σε PPT με Aspose.Slides Java

## Εισαγωγή
Σε αυτόν τον οδηγό θα **προσθέσετε ένα στοιβαδικό γράφημα στήλης** σε μια παρουσίαση PowerPoint προγραμματιστικά με το Aspose.Slides for Java. Είτε δημιουργείτε επιχειρηματικές αναφορές, εκπαιδευτικές παρουσιάσεις ή παρουσιάσεις μάρκετινγκ, η αυτοματοποίηση της δημιουργίας γραφημάτων εξοικονομεί χρόνο και εγγυάται συνέπεια. Θα περάσουμε από τη ρύθμιση της βιβλιοθήκης, τη δημιουργία μιας διαφάνειας, την προσθήκη του γραφήματος, την εφαρμογή στυλ γραμμής και στρογγυλεμένων γωνιών, και τέλος την αποθήκευση του αρχείου. Στο τέλος θα είστε άνετοι με ολόκληρη τη ροή εργασίας για **προσθήκη γραφήματος στη διαφάνεια** και ακόμη **δημιουργία διαφάνειας PowerPoint με Java**‑βασιζόμενες λύσεις.

### Γρήγορες Απαντήσεις
- **Ποια είναι η κύρια κλάση για εκκίνηση;** `Presentation`
- **Ποιος τύπος γραφήματος χρησιμοποιείται;** `ChartType.ClusteredColumn`
- **Πώς ενεργοποιείτε τις στρογγυλεμένες γωνίες;** `chart.setRoundedCorners(true);`
- **Ποια μορφή συνιστάται για αποθήκευση;** `SaveFormat.Pptx`
- **Χρειάζομαι άδεια για ανάπτυξη;** Μια δωρεάν δοκιμή λειτουργεί για δοκιμές· απαιτείται αγορασμένη άδεια για παραγωγή.

## Τι είναι ένα στοιβαδικό γράφημα στήλης;
Ένα στοιβαδικό γράφημα στήλης ομαδοποιεί πολλαπλές σειρές δεδομένων πλάι‑πλάι για κάθε κατηγορία, καθιστώντας το ιδανικό για σύγκριση τιμών μεταξύ διαφορετικών ομάδων. Το Aspose.Slides σας επιτρέπει να δημιουργήσετε αυτόν τον τύπο γραφήματος εξ ολοκλήρου μέσω κώδικα, χωρίς να ανοίξετε το PowerPoint.

## Γιατί να χρησιμοποιήσετε το Aspose.Slides for Java για την προσθήκη στοιβαδικού γραφήματος στήλης;
- **Πλήρης αυτοματοποίηση** – Δεν απαιτείται χειροκίνητη αλληλεπίδραση με το UI.  
- **Διαπλατφορμική** – Λειτουργεί σε οποιοδήποτε OS που υποστηρίζει Java.  
- **Πλούσια μορφοποίηση** – Έλεγχος στυλ γραμμών, γεμισμάτων, στρογγυλεμένων γωνιών κ.ά.  
- **Χωρίς εξαρτήσεις COM** – Σε αντίθεση με το Office Interop, εκτελείται με ασφάλεια σε διακομιστές.

## Οδηγός Υλοποίησης

### Δημιουργία Παρουσίασης και Προσθήκη Διαφάνειας
#### Επισκόπηση
Πρώτα, δημιουργούμε ένα νέο αντικείμενο `Presentation` και παίρνουμε τη προεπιλεγμένη διαφάνεια που περιλαμβάνεται σε ένα νέο αρχείο.

#### Βήμα‑βήμα
**1. Αρχικοποίηση του αντικειμένου Presentation**  
```java
Presentation presentation = new Presentation();
```

**2. Πρόσβαση στην Πρώτη Διαφάνεια**  
```java
ISlide slide = presentation.getSlides().get_Item(0);
```

**3. Αποδέσμευση Πόρων**  
```java
if (presentation != null) presentation.dispose();
```

### Προσθήκη Γραφήματος σε Διαφάνεια
#### Επισκόπηση
Τώρα ενσωματώνουμε ένα **στοιβαδικό γράφημα στήλης** στη διαφάνεια που μόλις προετοιμάσαμε.

#### Βήμα‑βήμα
**1. Αρχικοποίηση του αντικειμένου Presentation**  
```java
Presentation presentation = new Presentation();
```

**2. Πρόσβαση στην Πρώτη Διαφάνεια**  
```java
ISlide slide = presentation.getSlides().get_Item(0);
```

**3. Προσθήκη Στοιβαδικού Γραφήματος Στήλης**  
```java
IChart chart = slide.getShapes().addChart(ChartType.ClusteredColumn, 20, 100, 600, 400);
```

**4. Αποδέσμευση Πόρων**  
```java
if (presentation != null) presentation.dispose();
```

### Μορφοποίηση Στυλ Γραμμής Γραφήματος και Ορισμός Στρογγυλεμένων Γωνιών
#### Επισκόπηση
Βελτιώστε την οπτική εμφάνιση εφαρμόζοντας γεμιστό τύπο γραμμής, ένα ενιαίο στυλ γραμμής και στρογγυλεμένες γωνίες.

#### Βήμα‑βήμα
**1. Αρχικοποίηση του αντικειμένου Presentation**  
```java
Presentation presentation = new Presentation();
```

**2. Πρόσβαση στην Πρώτη Διαφάνεια**  
```java
ISlide slide = presentation.getSlides().get_Item(0);
```

**3. Προσθήκη Στοιβαδικού Γραφήματος Στήλης**  
```java
IChart chart = slide.getShapes().addChart(ChartType.ClusteredColumn, 20, 100, 600, 400);
```

**4. Ορισμός Μορφής Γραμμής σε Σταθερό Γέμισμα**  
```java
chart.getLineFormat().getFillFormat().setFillType(FillType.Solid);
```

**5. Εφαρμογή Ενιαίου Στυλ Γραμμής**  
```java
chart.getLineFormat().setStyle(LineStyle.Single);
```

**6. Ενεργοποίηση Στρογγυλεμένων Γωνιών για την Περιοχή του Γραφήματος**  
```java
chart.setRoundedCorners(true);
```

**7. Αποδέσμευση Πόρων**  
```java
if (presentation != null) presentation.dispose();
```

### Αποθήκευση Παρουσίασης
#### Επισκόπηση
Τέλος, γράφουμε την παρουσίαση στο δίσκο σε μορφή PPTX.

#### Βήμα‑βήμα
**1. Αρχικοποίηση του αντικειμένου Presentation**  
```java
Presentation presentation = new Presentation();
```

**2. Ορισμός Καταλόγου Εξόδου και Ονόματος Αρχείου**  
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY/";
String outputFile = dataDir + "out.pptx";
```

**3. Αποθήκευση της Παρουσίασης σε Μορφή PPTX**  
```java
presentation.save(outputFile, SaveFormat.Pptx);
```

**4. Αποδέσμευση Πόρων**  
```java
if (presentation != null) presentation.dispose();
```

## Συχνά Προβλήματα και Λύσεις
| Πρόβλημα | Λύση |
|----------|------|
| **`NullPointerException` on `getSlides()`** | Βεβαιωθείτε ότι το αντικείμενο `Presentation` έχει δημιουργηθεί επιτυχώς πριν την πρόσβαση στις διαφάνειες. |

## Συχνές Ερωτήσεις

**Q: Πώς μπορώ να προσθέσω διαφορετικούς τύπους γραφημάτων χρησιμοποιώντας το Aspose.Slides;**  
A: Αντικαταστήστε `ChartType.ClusteredColumn` με οποιαδήποτε άλλη τιμή enum όπως `ChartType.Pie`, `ChartType.Line` ή `ChartType.Bar`.

**Q: Τι πρέπει να κάνω αν αντιμετωπίσω σφάλματα μεταγλώττισης;**  
A: Ελέγξτε ξανά ότι χρησιμοποιείτε JDK 16 ή νεότερη έκδοση και ότι η εξάρτηση Maven/Gradle ταιριάζει με την έκδοση που εμφανίζεται παραπάνω.

**Q: Μπορώ να γεμίσω το γράφημα με δεδομένα από βάση δεδομένων;**  
A: Ναι. Πρόσβαση στη συλλογή `getChartData()` του γραφήματος, δημιουργήστε σειρές και κατηγορίες, και γεμίστε τις με τιμές που λαμβάνονται κατά την εκτέλεση.

**Q: Πώς μπορώ να βελτιώσω την απόδοση για πολύ μεγάλες παρουσιάσεις;**  
A: Διαχωρίστε τη δουλειά σε πολλαπλά αντικείμενα `Presentation`, επαναχρησιμοποιήστε πρότυπα γραφημάτων, και πάντα αποδεσμεύετε τα αντικείμενα άμεσα.

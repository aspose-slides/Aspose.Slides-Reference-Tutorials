## Οδηγός Εφαρμογής

### Χαρακτηριστικό: Αναπαράσταση μεγέθους φυσαλίδας σε γραφήματα

Αυτή η λειτουργία επιτρέπει την προσαρμογή των μεγεθών των φυσαλίδων στα γραφήματα, βελτιώνοντας την ερμηνευσιμότητα των δεδομένων.

#### Βήμα προς βήμα εφαρμογή

##### Αρχικοποίηση παρουσίασης και διαφάνειας
Αρχικά, δημιουργήστε ένα αντικείμενο παρουσίασης και αποκτήστε πρόσβαση στην πρώτη του διαφάνεια:
```java
Presentation pres = new Presentation();
try {
    ISlide slide = pres.getSlides().get_Item(0);
```

##### Προσθήκη γραφήματος φυσαλίδων σε διαφάνεια
Προσθέστε ένα γράφημα φυσαλίδων στην καθορισμένη θέση με τις επιθυμητές διαστάσεις:
```java
IChart chart = slide.getShapes().addChart(
    ChartType.Bubble, 50, 50, 600, 400, true
);
```
**Επεξήγηση παραμέτρων:**
- `ChartType.Bubble`: Καθορίζει τον τύπο του γραφήματος.
- `(50, 50)`: Συντεταγμένες X και Y για τη θέση του χάρτη στη διαφάνεια.
- `(600, 400)`: Πλάτος και ύψος του γραφήματος.

##### Ορισμός τύπου αναπαράστασης μεγέθους φυσαλίδας
Ορίστε το μέγεθος της φυσαλίδας ώστε να αντιπροσωπεύει τα δεδομένα με βάση το 'Πλάτος':
```java
chart.getChartData().getSeriesGroups().get_Item(0)
    .setBubbleSizeRepresentation(BubbleSizeRepresentationType.Width);
```
Αυτή η διαμόρφωση αλλάζει τον τρόπο με τον οποίο οι τιμές δεδομένων αντιστοιχίζονται σε μεγέθη φυσαλίδων, εστιάζοντας στο πλάτος για πιο καθαρή οπτικοποίηση.

##### Αποθήκευση και απόρριψη
Τέλος, αποθηκεύστε την παρουσίαση και απελευθερώστε τους πόρους:
```java
pres.save("YOUR_DOCUMENT_DIRECTORY/Presentation_BubbleSizeRepresentation.pptx", SaveFormat.Pptx);
} finally {
    if (pres != null) pres.dispose();
}
```
**Συμβουλή αντιμετώπισης προβλημάτων:** Βεβαιωθείτε ότι οι διαδρομές αρχείων έχουν καθοριστεί σωστά για να αποφύγετε σφάλματα αποθήκευσης.

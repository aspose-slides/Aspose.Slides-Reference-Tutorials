## Ρύθμιση του Aspose.Slides for Java
Πρώτα, συμπεριλάβετε το Aspose.Slides στο έργο σας μέσω Maven ή Gradle.

**Maven:**
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

**Gradle:**
```gradle
implementation group: 'com.aspose', name: 'aspose-slides', version: '25.4', classifier: 'jdk16'
```

Εναλλακτικά, μπορείτε να [κατεβάσετε την πιο πρόσφατη έκδοση](https://releases.aspose.com/slides/java/) απευθείας από την ιστοσελίδα της Aspose.

## Οδηγός Υλοποίησης
Παρακάτω χωρίζουμε τη λύση σε δύο λειτουργίες: προσθήκη διαγράμματος πίτας και πρόσβαση στο βιβλίο εργασίας δεδομένων του διαγράμματος.

### Λειτουργία 1: Δημιουργία Παρουσίασης και Προσθήκη Διαγράμματος
#### Επισκόπηση
Αυτή η ενότητα δείχνει πώς να δημιουργήσετε μια νέα παρουσίαση και **add a pie chart** στην πρώτη διαφάνεια.

#### Βήμα‑βήμα

**Βήμα 1: Αρχικοποίηση Νέου Αντικειμένου Παρουσίασης**  
```java
Presentation pres = new Presentation();
```
*Δημιουργεί το στιγμιότυπο `Presentation` που θα περιέχει όλες τις διαφάνειες.*

**Βήμα 2: Προσθήκη Διάγραμμα Πίτας**  
```java
IChart chart = pres.getSlides().get_Item(0).getShapes().addChart(
    ChartType.Pie,
    50,
    50,
    400,
    500
);
```
*Τοποθετεί ένα διάγραμμα πίτας στις συντεταγμένες (50, 50) με πλάτος 400 και ύψος 500. Η παράμετρος `ChartType.Pie` υποδεικνύει στο Aspose να αποδώσει διάγραμμα πίτας.*

**Βήμα 3: Αποδέσμευση Πόρων**  
```java
if (pres != null) pres.dispose();
```
*Απελευθερώνει τους εγγενείς πόρους· καλείτε πάντα το `dispose()` όταν τελειώσετε.*

### Λειτουργία 2: Πρόσβαση στο Βιβλίο Εργασίας Δεδομένων Διαγράμματος και στα Φύλλα Εργασίας
#### Επισκόπηση
Μάθετε πώς να φτάσετε στο υποκείμενο βιβλίο εργασίας που αποθηκεύει τα δεδομένα του διαγράμματος και να διασχίσετε τα φύλλα εργασίας του.

#### Βήμα‑βήμα

**Βήμα 1: (Επανάχρηση) Αρχικοποίηση Νέου Αντικειμένου Παρουσίασης**  
*Ίδιο με το Βήμα 1 της Λειτουργίας 1.*

**Βήμα 2: (Επανάχρηση) Προσθήκη Διάγραμμα Πίτας**  
*Ίδιο με το Βήμα 2 της Λειτουργίας 1.*

**Βήμα 3: Λήψη του Βιβλίου Εργασίας Δεδομένων Διαγράμματος**  
```java
IChartDataWorkbook workbook = chart.getChartData().getChartDataWorkbook();
```
*Ανακτά το `IChartDataWorkbook` που συνδέεται με το διάγραμμα.*

**Βήμα 4: Διασχίζοντας τα Φύλλα Εργασίας**  
```java
for (int i = 0; i < workbook.getWorksheets().size(); i++) {
    System.out.println(workbook.getWorksheets().get_Item(i).getName());
}
```
*Εκτυπώνει το όνομα κάθε φύλλου εργασίας, επιτρέποντάς σας να επαληθεύσετε τη δομή των δεδομένων.*

**Βήμα 5: Αποδέσμευση Πόρων**  
*Ίδιο με το Βήμα 3 της Λειτουργίας 1.*

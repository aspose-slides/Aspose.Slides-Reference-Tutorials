## Ρύθμιση του Aspose.Slides για Java

Για να χρησιμοποιήσετε το Aspose.Slides για Java, συμπεριλάβετέ το στο έργο σας. Δείτε πώς μπορείτε να ρυθμίσετε τη βιβλιοθήκη χρησιμοποιώντας διαφορετικά συστήματα διαχείρισης εξαρτήσεων:

**Maven:**
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

**Βαθμός:**
```gradle
implementation group: 'com.aspose', name: 'aspose-slides', version: '25.4', classifier: 'jdk16'
```

**Άμεση λήψη:** Μπορείτε επίσης να κατεβάσετε την τελευταία έκδοση από το [Aspose.Slides για εκδόσεις Java](https://releases.aspose.com/slides/java/).

## Οδηγός Εφαρμογής

### Δημιουργία και προσθήκη κυκλικού γραφήματος σε παρουσίαση

#### Επισκόπηση
Αυτή η ενότητα καλύπτει τα βήματα για τη δημιουργία ενός κυκλικού γραφήματος σε μια διαφάνεια παρουσίασης. Θα σας καθοδηγήσουμε στην αρχικοποίηση της παρουσίασης, στη δημιουργία ενός γραφήματος και στην προσαρμογή της εμφάνισής του.

#### Βήμα 1: Αρχικοποίηση παρουσίασης
Ξεκινήστε δημιουργώντας μια παρουσία του `Presentation` τάξη:
```java
demo.Presentation pres = new demo.Presentation();
```
Αυτό αρχικοποιεί την παρουσίασή σας όπου θα γίνουν όλες οι αλλαγές.

#### Βήμα 2: Προσθήκη γραφήματος πίτας σε διαφάνεια
Στη συνέχεια, προσθέστε ένα γράφημα πίτας στην πρώτη διαφάνεια σε καθορισμένες συντεταγμένες με δεδομένες διαστάσεις:
```java
// Ορισμός θέσης και μεγέθους για το γράφημα πίτας
int xPosition = 50;
int yPosition = 50;
int width = 400;
int height = 600;

demo.IChart chart = pres.getSlides().get_Item(0).getShapes().addChart(
    demo.ChartType.Pie, xPosition, yPosition, width, height, false);
```
Εδώ:
- `xPosition` και `yPosition` ορίστε τις συντεταγμένες επάνω αριστερά.
- `width` και `height` ορίστε τις διαστάσεις του γραφήματος.

#### Βήμα 3: Προσαρμογή γραφήματος πίτας
Προσαρμόστε το γράφημα πίτας τροποποιώντας τα σημεία δεδομένων, τα χρώματα ή τις ετικέτες του. Ακολουθεί ένα απλό παράδειγμα προσθήκης δεδομένων στο γράφημά σας:
```java
// Πρόσβαση στην προεπιλεγμένη σειρά δεδομένων για επίδειξη
demo.IChartDataWorkbook wb = chart.getChartData().getChartDataWorkbook();
chart.getChartData().getSeries().clear();

// Προσθήκη νέας σειράς και συμπλήρωση με δεδομένα
demo.IChartSeries series = chart.getChartData().getSeries().add(wb.getCell(0, "B1", "Category 1"), demo.ChartType.Pie);
series.getDataPoints().addDataPointForPieSeries(wb.getCell(0, "B2", 30));
series.getDataPoints().addDataPointForPieSeries(wb.getCell(0, "B3", 70));

// Προσαρμόστε τις ετικέτες σειράς
for (demo.IDataPoint point : series.getDataPoints()) {
    demo.IChartDataLabel label = point.getLabel();
    label.getDataLabelFormat().setShowCategoryName(true);
}
```
Αυτό το τμήμα κώδικα προσθέτει μια σειρά δεδομένων με δύο κατηγορίες και ρυθμίζει τα ονόματα των κατηγοριών ώστε να εμφανίζονται ως ετικέτες.

#### Συμβουλές αντιμετώπισης προβλημάτων
- **Συνηθισμένο πρόβλημα:** Εάν αντιμετωπίσετε σφάλμα σχετικά με ελλείπουσες εξαρτήσεις, βεβαιωθείτε ότι το `pom.xml` ή `build.gradle` τα αρχεία έχουν ρυθμιστεί σωστά.
- **Το γράφημα δεν εμφανίζεται:** Επαληθεύστε ότι όλες οι σειρές δεδομένων και τα σημεία έχουν προστεθεί σωστά. Τα γραφήματα μπορεί να εμφανίζονται κενά εάν δεν υπάρχουν συνδεδεμένα δεδομένα.

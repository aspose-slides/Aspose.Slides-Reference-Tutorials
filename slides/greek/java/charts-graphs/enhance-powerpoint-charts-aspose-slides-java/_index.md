# Βελτίωση γραφημάτων PowerPoint: Προσαρμογή γραμματοσειράς και άξονα με το Aspose.Slides για Java

Αυτό το σεμινάριο θα σας καθοδηγήσει στη βελτίωση της αισθητικής των γραφημάτων χρησιμοποιώντας αυτές τις λειτουργίες.

- Πώς να ορίσετε το μέγεθος γραμματοσειράς του υπομνήματος για να βελτιώσετε την αναγνωσιμότητα.

### Οδηγός Εφαρμογής

#### Χαρακτηριστικό 1: Υπόμνημα μεγέθους γραμματοσειράς στο γράφημα

**Επισκόπηση:**
Η προσαρμογή του μεγέθους γραμματοσειράς ενός υπομνήματος μπορεί να βελτιώσει σημαντικά την ορατότητα και την αναγνωσιμότητά του, καθιστώντας τα γραφήματά σας πιο φιλικά προς το χρήστη.

**Βήματα για την προσαρμογή του μεγέθους γραμματοσειράς υπομνήματος:**

**H3. Προσθήκη γραφήματος ομαδοποιημένων στηλών**
Ξεκινήστε δημιουργώντας ένα γράφημα ομαδοποιημένων στηλών στην πρώτη διαφάνεια στη θέση (50, 50) με διαστάσεις 600x400:
```java
import com.aspose.slides.ChartType;
import com.aspose.slides.IChart;
import com.aspose.slides.Presentation;

Presentation pres = new Presentation("YOUR_DOCUMENT_DIRECTORY/test.pptx");
try {
    IChart chart = pres.getSlides().get_Item(0).getShapes().addChart(
        ChartType.ClusteredColumn, 50, 50, 600, 400);

    // Ορισμός μεγέθους γραμματοσειράς υπομνήματος
    chart.getLegend().getTextFormat().getPortionFormat().setFontHeight(20);
    
pres.save("YOUR_OUTPUT_DIRECTORY/output.pptx", SaveFormat.Pptx);
} finally {
    if (pres != null) pres.dispose();
}
```
- **Εξήγηση:** Ο `setFontHeight` Η μέθοδος ορίζει το μέγεθος του κειμένου του υπομνήματος σε 20 στιγμές, βελτιώνοντας την αναγνωσιμότητά του.

**H3. Αποθήκευση των αλλαγών σας**
Βεβαιωθείτε ότι έχετε αποθηκεύσει την παρουσίασή σας για να εφαρμόσετε τις αλλαγές:
```java
pres.save("YOUR_OUTPUT_DIRECTORY/output.pptx", SaveFormat.Pptx);
```

#### Χαρακτηριστικό 2: Διαμόρφωση τιμών άξονα σε γράφημα

**Επισκόπηση:**
Η προσαρμογή των τιμών των αξόνων επιτρέπει τον ακριβή έλεγχο της αναπαράστασης δεδομένων, διευκολύνοντας το κοινό να κατανοήσει τις τάσεις.

**Βήματα για τη διαμόρφωση τιμών κατακόρυφου άξονα:**

**H3. Προσθήκη γραφήματος ομαδοποιημένων στηλών**
Όπως και πριν, προσθέστε ένα γράφημα ομαδοποιημένων στηλών:
```java
import com.aspose.slides.Presentation;
import com.aspose.slides.IChart;
import com.aspose.slides.ChartType;

Presentation pres = new Presentation("YOUR_DOCUMENT_DIRECTORY/test.pptx");
try {
    IChart chart = pres.getSlides().get_Item(0).getShapes().addChart(
        ChartType.ClusteredColumn, 50, 50, 600, 400);

    // Ρύθμιση παραμέτρων κάθετου άξονα
    chart.getAxes().getVerticalAxis().setAutomaticMinValue(false);
    chart.getAxes().getVerticalAxis().setMinValue(-5);
    
    chart.getAxes().getVerticalAxis().setAutomaticMaxValue(false);
    chart.getAxes().getVerticalAxis().setMaxValue(10);

    pres.save("YOUR_OUTPUT_DIRECTORY/output.pptx", SaveFormat.Pptx);
} finally {
    if (pres != null) pres.dispose();
}
```
- **Εξήγηση:** Η απενεργοποίηση των αυτόματων ρυθμίσεων ελάχιστης και μέγιστης τιμής σάς επιτρέπει να ορίσετε τις δικές σας, όπως -5 για ελάχιστη και 10 για μέγιστη, παρέχοντας ακριβή έλεγχο της κλιμάκωσης δεδομένων.

2. **Εκπαιδευτικές Παρουσιάσεις:** Η προσαρμογή των εύρων των αξόνων μπορεί να βοηθήσει στην απεικόνιση συγκεκριμένων τάσεων ή συγκρίσεων.
3. **Οικονομική Ανάλυση:** Η προσαρμογή των υπομνημάτων και των αξόνων μπορεί να κάνει τα σύνθετα οικονομικά δεδομένα πιο προσβάσιμα.

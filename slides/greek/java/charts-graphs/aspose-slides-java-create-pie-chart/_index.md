## Πώς να προσθέσετε pie chart PowerPoint χρησιμοποιώντας το Aspose.Slides για Java

Παρακάτω υπάρχει ένας οδηγός βήμα‑βήμα που δείχνει ακριβώς πώς να δημιουργήσετε και να προσαρμόσετε ένα διάγραμμα πίτας.

### Βήμα 1: Αρχικοποίηση της Παρουσίασης
Πρώτα, δημιουργήστε ένα νέο αντικείμενο `Presentation`. Αυτό αντιπροσωπεύει ένα κενό αρχείο PowerPoint.
```java
Presentation pres = new Presentation();
```

### Βήμα 2: Προσθήκη του Διαγράμματος Πίτας
Εισάγετε ένα διάγραμμα πίτας στην πρώτη διαφάνεια. Οι συντεταγμένες (50, 50) και το μέγεθος (600 × 400) λειτουργούν καλά για μια τυπική διαφάνεια 16:9.
```java
pres.getSlides().get_Item(0).getShapes().addChart(ChartType.Pie, 50, 50, 600, 400);
```

### Βήμα 3: Αποθήκευση της Παρουσίασης
Γράψτε την παρουσίαση στο δίσκο. Αντικαταστήστε το `YOUR_OUTPUT_DIRECTORY` με το φάκελο όπου θέλετε να αποθηκευτεί το αρχείο.
```java
pres.save("YOUR_OUTPUT_DIRECTORY/output.pptx", SaveFormat.Pptx);
```

### Βήμα 4: Καθαρισμός Πόρων
Αποδεσμεύστε το αντικείμενο `Presentation` για να ελευθερώσετε τους εγγενείς πόρους.
```java
if (pres != null) pres.dispose();
```

## Προσαρμογή Έκρηξης Σημείου Δεδομένων και Χρώματος

Η προσαρμογή μεμονωμένων τμημάτων κάνει το γράφημά σας πιο ευανάγνωστο, ειδικά όταν θέλετε να τονίσετε μια συγκεκριμένη τιμή.

### Βήμα 1: Φόρτωση Υπάρχουσας Παρουσίασης (ή επαναχρησιμοποίηση της πρόσφατα δημιουργημένης)
```java
Presentation pres = new Presentation("YOUR_DOCUMENT_DIRECTORY/test.pptx");
```

### Βήμα 2: Πρόσβαση στο Διάγραμμα και στο Στόχο Σημείου Δεδομένων
Εδώ ανακτούμε το δεύτερο σημείο δεδομένων (δείκτης 1) από την πρώτη σειρά.
```java
IChart chart = pres.getSlides().get_Item(0).getShapes().addChart(ChartType.Pie, 50, 50, 600, 400);
IChartDataPoint point = chart.getChartData().getSeries().get_Item(0).getDataPoints().get_Item(1);
```

### Βήμα 3: Εφαρμογή Έκρηξης και Χρώματος
Διαχωρίστε το τμήμα οπτικά και αλλάξτε το χρώμα γεμίσματος σε μπλε.
```java
point.setExplosion(30); // Set explosion distance
point.getFormat().getFill().setFillType(FillType.Solid);
point.getFormat().getFill().getSolidFillColor().setColor(Color.BLUE); // Change fill color
```

### Βήμα 4: Αποθήκευση και Αποδέσμευση
```java
pres.save("YOUR_OUTPUT_DIRECTORY/customized.pptx", SaveFormat.Pptx);
if (pres != null) pres.dispose();
```

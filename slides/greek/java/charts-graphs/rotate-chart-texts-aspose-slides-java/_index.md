## Ρύθμιση του Aspose.Slides για Java

### Εγκατάσταση
Προσθέστε την εξάρτηση Aspose.Slides στο έργο σας χρησιμοποιώντας το Maven ή το Gradle.

#### Maven
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

#### Γκράντλ
```gradle
implementation group: 'com.aspose', name: 'aspose-slides', version: '25.4', classifier: 'jdk16'
```

Εναλλακτικά, κατεβάστε την τελευταία έκδοση από το [Aspose.Slides για εκδόσεις Java](https://releases.aspose.com/slides/java/).

## Οδηγός Εφαρμογής
Θα αναλύσουμε τη δημιουργία και την προσαρμογή γραφημάτων με εναλλάξ κείμενα σε απλά βήματα.

### Βήμα 1: Αρχικοποίηση παρουσίασης
Δημιουργήστε μια παρουσία του `Presentation` τάξη για να εργαστείτε με το αρχείο παρουσίασής σας.

```java
import com.aspose.slides.Presentation;
import com.aspose.slides.IChart;
import com.aspose.slides.ChartType;
import com.aspose.slides.ISlide;
import com.aspose.slides.SaveFormat;

// Ορίστε τη διαδρομή του καταλόγου εγγράφων σας
String dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Αντικαταστήστε με τον πραγματικό σας κατάλογο

Presentation presentation = new Presentation();
```

### Βήμα 2: Προσθήκη γραφήματος στη διαφάνεια
Αποκτήστε πρόσβαση στην πρώτη διαφάνεια και προσθέστε ένα `ClusteredColumn` διάγραμμα.

```java
ISlide slide = presentation.getSlides().get_Item(0);
IChart chart = slide.getShapes().addChart(ChartType.ClusteredColumn, 50, 50, 500, 300);
```

### Βήμα 3: Προσαρμόστε την περιστροφή ετικέτας
Αποκτήστε πρόσβαση στην πρώτη σειρά για να τροποποιήσετε τις ετικέτες δεδομένων της.

```java
import com.aspose.slides.IChartData;

IChartSeries series = chart.getChartData().getSeries().get_Item(0);
series.getLabels().getDefaultDataLabelFormat().setShowValue(true);

// Ορισμός γωνίας περιστροφής κειμένου ετικέτας σε 65 μοίρες
series.getLabels().getDefaultDataLabelFormat().getTextFormat()
        .getTextBlockFormat().setRotationAngle(65);
```

### Βήμα 4: Προσαρμόστε την εναλλαγή τίτλου γραφήματος
Προσθέστε έναν τίτλο και ορίστε τη γωνία περιστροφής του.

```java
chart.setTitle(true);
chart.getChartTitle().addTextFrameForOverriding("Custom title")
        .getTextFrameFormat().setRotationAngle(-30); // Περιστροφή του κειμένου τίτλου κατά -30 μοίρες
```

### Βήμα 5: Αποθηκεύστε την παρουσίασή σας
Αποθηκεύστε την παρουσίασή σας με όλες τις προσαρμογές.

```java
presentation.save(dataDir + "/textframe-rotation_out.pptx", SaveFormat.Pptx);

// Βεβαιωθείτε ότι έχετε κλείσει το αντικείμενο παρουσίασης για να απελευθερώσετε πόρους
disposePresentation(presentation);
```

#### Βοηθητική Μέθοδος για Απόρριψη
Για την αποτελεσματική διαχείριση των πόρων, απορρίψτε τα `Presentation` αντιταχθείτε σωστά.

```java
private static void disposePresentation(Presentation pres) {
    if (pres != null) {
        pres.dispose();
    }
}
```

### Πώς να δημιουργήσετε Διάγραμμα Φυσαλίδων στο PowerPoint
#### Επισκόπηση
Αυτή η λειτουργία δείχνει πώς να προσθέσετε ένα προσαρμόσιμο διάγραμμα φυσαλίδων στις διαφάνειες PowerPoint. Θα διαμορφώσουμε το μέγεθος και την κλίμακα του για καλύτερη αναπαράσταση των δεδομένων.

#### Υλοποίηση βήμα‑βήμα
**1. Αρχικοποίηση Παρουσίασης**  
Ξεκινήστε δημιουργώντας ένα στιγμιότυπο του `Presentation`:
```java
Presentation pres = new Presentation();
```

**2. Προσθήκη Διαγράμματος Φυσαλίδων**  
Προσθέστε το διάγραμμα φυσαλίδων σε καθορισμένη θέση με ορισμένες διαστάσεις:
```java
IChart chart = pres.getSlides().get_Item(0).getShapes().addChart(
        ChartType.Bubble, 100, 100, 400, 300
);
```
- **Παράμετροι**: Το `ChartType.Bubble` καθορίζει τον τύπο του διαγράμματος. Οι αριθμοί αντιπροσωπεύουν τη θέση (x, y) και το μέγεθος (πλάτος, ύψος).

**3. Διαμόρφωση Κλίμακας Μεγέθους Φυσαλίδας**  
Ρυθμίστε το μέγεθος της φυσαλίδας για βελτιωμένη σαφήνεια:
```java
chart.getChartData().getSeriesGroups().get_Item(0).setBubbleSizeScale(150);
```
- **Σκοπός**: Ορίζοντας το `BubbleSizeScale` στο 150% μεγαλώνει τις φυσαλίδες, κάνοντάς τες πιο διακριτές.

**4. Αποθήκευση Παρουσίασης**  
Αποθηκεύστε τις αλλαγές σας με το νέο διάγραμμα:
```java
pres.save("YOUR_OUTPUT_DIRECTORY/Result.pptx", SaveFormat.Pptx);
```

#### Συμβουλές Επίλυσης Προβλημάτων
- Βεβαιωθείτε ότι έχετε δικαιώματα εγγραφής στον φάκελο εξόδου.
- Επιβεβαιώστε ότι το Aspose.Slides περιλαμβάνεται σωστά στις εξαρτήσεις του έργου σας.

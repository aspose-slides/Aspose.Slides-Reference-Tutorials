## Οδηγός Εφαρμογής

### Ανάκτηση και εμφάνιση εφέ κίνησης

#### Επισκόπηση
Αυτή η ενότητα δείχνει πώς να κάνετε επανάληψη στις διαφάνειες μιας παρουσίασης PowerPoint, αποκτώντας πρόσβαση σε κάθε εφέ κίνησης. Θα μάθετε πώς να εξάγετε βασικές λεπτομέρειες σχετικά με αυτά τα εφέ μέσω προγραμματισμού.

**1. Εισαγωγή απαραίτητων κλάσεων**
Βεβαιωθείτε ότι έχετε τις απαραίτητες εισαγωγές στην αρχή του αρχείου Java:
```java
import com.aspose.slides.IEffect;
import com.aspose.slides.ISlide;
import com.aspose.slides.Presentation;
```

**2. Αρχικοποίηση αντικειμένου παρουσίασης**
Ξεκινήστε φορτώνοντας την παρουσίασή σας σε ένα αντικείμενο Aspose.Slides:
```java
String presentationFileName = "YOUR_DOCUMENT_DIRECTORY/AnimationShapesExample.pptx";
Presentation pres = new Presentation(presentationFileName);
```

**3. Επαναλάβετε τις διαφάνειες και τα εφέ**
Στη συνέχεια, κάντε επανάληψη σε κάθε διαφάνεια στην παρουσίαση για να αποκτήσετε πρόσβαση στην ακολουθία κίνησης:
```java
try {
    for (ISlide slide : pres.getSlides()) {
        IEffect[] effects = slide.getTimeline().getMainSequence();

        for (IEffect effect : effects) {
            String effectType = effect.getType();
            int targetShapeId = effect.getTargetShape().getUniqueId();
            int slideNumber = slide.getSlideNumber();
            
            System.out.println(effectType + " animation effect is set to shape#" +
                    targetShapeId + " on slide#" + slideNumber);
        }
    }
} finally {
    pres.dispose(); // Να απορρίπτετε πάντα το αντικείμενο Presentation για να ελευθερώνετε πόρους
}
```

#### Εξήγηση:
- `getSlides()`: Ανακτά μια συλλογή διαφανειών από την παρουσίαση.
- `getTimeline().getMainSequence()`Πρόσβαση στην κύρια ακολουθία κίνησης για κάθε διαφάνεια.
- `getType()`, `getTargetShape()`: Εξάγει λεπτομέρειες τύπου και σχήματος στόχου για κάθε εφέ.

**Συμβουλές αντιμετώπισης προβλημάτων:**
- Βεβαιωθείτε ότι η διαδρομή του αρχείου παρουσίασής σας είναι σωστή.
- Επαληθεύστε ότι η έκδοση της βιβλιοθήκης Aspose.Slides ταιριάζει με τη συμβατότητα με το JDK.

## Οδηγός Υλοποίησης

Θα περάσουμε από τρία βασικά χαρακτηριστικά: φόρτωση & αποθήκευση μιας παρουσίασης, πρόσβαση στη σειρά εφέ διαφανειών, και ρύθμιση του χρονισμού των εφέ και των επιλογών επανάληψης.

### Χαρακτηριστικό 1: Φόρτωση και Αποθήκευση Παρουσίασης

#### Επισκόπηση
Η φόρτωση ενός αρχείου PPTX σας παρέχει ένα μεταβλητό αντικείμενο `Presentation` που μπορείτε να επεξεργαστείτε πριν αποθηκεύσετε τις αλλαγές.

#### Υλοποίηση Βήμα‑Βήμα

**Βήμα 1 – Φόρτωση της Παρουσίασης**

```java
import com.aspose.slides.Presentation;
import com.aspose.slides.SaveFormat;

String dataDir = "YOUR_DOCUMENT_DIRECTORY/AnimationOnSlide.pptx";
Presentation pres = new Presentation(dataDir);
```

**Βήμα 2 – Αποθήκευση της Τροποποιημένης Παρουσίασης**

```java
try {
    String outDir = "YOUR_OUTPUT_DIRECTORY/AnimationOnSlide-out.pptx";
    pres.save(outDir, SaveFormat.Pptx);
} finally {
    if (pres != null) pres.dispose();
}
```

Το μπλοκ `try‑finally` εγγυάται ότι οι πόροι απελευθερώνονται, αποτρέποντας διαρροές μνήμης.

### Χαρακτηριστικό 2: Πρόσβαση στη Σειρά Εφέ Διαφανειών

#### Επισκόπηση
Κάθε διαφάνεια περιέχει μια χρονογραμμή με κύρια σειρά εφέ. Η ανάκτηση αυτής της σειράς σας επιτρέπει να διαβάσετε ή να τροποποιήσετε μεμονωμένες μεταβάσεις.

#### Υλοποίηση Βήμα‑Βήμα

**Βήμα 1 – Φόρτωση της Παρουσίασης (επαναχρησιμοποίηση του ίδιου αρχείου)**

```java
Presentation pres = new Presentation("YOUR_DOCUMENT_DIRECTORY/AnimationOnSlide.pptx");
```

**Βήμα 2 – Ανάκτηση της Σειράς Εφέ**

```java
import com.aspose.slides.IEffect;
import com.aspose.slides.ISequence;

try {
    ISequence effectsSequence = pres.getSlides().get_Item(0).getTimeline().getMainSequence();
    IEffect effect = effectsSequence.get_Item(0);
} finally {
    if (pres != null) pres.dispose();
}
```

Εδώ ανακτούμε το πρώτο εφέ από την κύρια σειρά της πρώτης διαφάνειας.

### Χαρακτηριστικό 3: Τροποποίηση Χρονισμού Εφέ και Επιλογών Επανάληψης

#### Επισκόπηση
Η αλλαγή του χρονισμού και της συμπεριφοράς επανάληψης σας δίνει λεπτομερή έλεγχο του χρόνου εκτέλεσης μιας κίνησης και του πότε επανεκκινείται.

#### Υλοποίηση Βήμα‑Βήμα

```java
// Assume 'effect' is the IEffect instance obtained earlier

effect.getTiming().setRepeatUntilEndSlide(true);
effect.getTiming().setRepeatUntilNextClick(true);
```

Αυτές οι κλήσεις ρυθμίζουν το εφέ να επαναλαμβάνεται είτε μέχρι το τέλος της διαφάνειας είτε μέχρι να κάνει κλικ ο παρουσιαστής.

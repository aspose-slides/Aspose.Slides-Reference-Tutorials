## Οδηγός Εφαρμογής
### Εξαγωγή αντικειμένων OLE από PPTX σε PDF
Αυτή η λειτουργία εστιάζει στην εξαγωγή παρουσιάσεων PowerPoint με ενσωματωμένα αντικείμενα OLE σε μορφή PDF, διατηρώντας παράλληλα τα ενσωματωμένα δεδομένα. Δείτε πώς μπορείτε να το πετύχετε αυτό:

#### Βήμα 1: Φόρτωση της παρουσίασής σας
Φορτώστε το αρχείο παρουσίασής σας χρησιμοποιώντας το `Presentation` τάξη.

```java
import com.aspose.slides.Presentation;

String dataDir = "YOUR_DOCUMENT_DIRECTORY";
Presentation pres = new Presentation(dataDir + "/PresOleExample.pptx");
```

**Γιατί;**: Η φόρτωση της παρουσίασης αρχικοποιεί το μοντέλο αντικειμένου που θα χειριστεί το Aspose.Slides.

#### Βήμα 2: Ρύθμιση παραμέτρων επιλογών εξαγωγής PDF
Στήνω `PdfOptions` για να συμπεριλάβετε δεδομένα OLE στην εξαγωγή σας.

```java
import com.aspose.slides.PdfOptions;

PdfOptions options = new PdfOptions();
options.setIncludeOleData(true);
```

**Γιατί;**: Το `setIncludeOleData(true)` διασφαλίζει ότι τα ενσωματωμένα αντικείμενα OLE διατηρούνται κατά τη μετατροπή, διατηρώντας την ακεραιότητα των δεδομένων.

#### Βήμα 3: Εξαγωγή σε PDF
Αποθηκεύστε την παρουσίασή σας ως αρχείο PDF με τις καθορισμένες επιλογές.

```java
import com.aspose.slides.SaveFormat;

String outFilePath = "YOUR_OUTPUT_DIRECTORY/PresOleExample.pdf";
pres.save(outFilePath, SaveFormat.Pdf, options);
```

**Γιατί;**: Αυτό το βήμα εκτελεί τη μετατροπή και αποθηκεύει το PDF εξόδου στον επιλεγμένο κατάλογο. Καθορίζοντας `SaveFormat.Pdf`, κατευθύνετε το Aspose.Slides για να δημιουργήσει ένα αρχείο PDF.

### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι η διαδρομή της παρουσίασής σας είναι σωστή για να αποφύγετε `FileNotFoundException`.
- Βεβαιωθείτε ότι έχετε ρυθμίσει σωστά την άδεια χρήσης για να αποτρέψετε την εμφάνιση υδατογραφήματος στην έξοδο.
- Εάν τα δεδομένα OLE δεν εμφανίζονται, ελέγξτε ξανά `options.setIncludeOleData(true);` διαμόρφωση.

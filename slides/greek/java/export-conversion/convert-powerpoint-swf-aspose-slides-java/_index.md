## Οδηγός Εφαρμογής

### Μετατροπή παρουσίασης σε SWF χωρίς προβολή

Για να μετατρέψετε παρουσιάσεις σε αρχεία SWF χωρίς να ενσωματώσετε τη διεπαφή προβολής (ιδανικό για προσαρμοσμένες εφαρμογές):

#### Βήμα 1: Αρχικοποίηση αντικειμένου παρουσίασης
Δημιουργήστε ένα `Presentation` αντικείμενο για το αρχείο PPTX σας:
```java
import com.aspose.slides.Presentation;

// Ορισμός καταλόγου εισόδου
define String dataDir = "YOUR_DOCUMENT_DIRECTORY";

// Φόρτωση αρχείου παρουσίασης
Presentation presentation = new Presentation(dataDir + "/HelloWorld.pptx");
```

#### Βήμα 2: Ρύθμιση παραμέτρων επιλογών SWF
Στήνω `SwfOptions` για να αποκλείσετε τον θεατή:
```java
import com.aspose.slides.SwfOptions;
import com.aspose.slides.SaveFormat;
import com.aspose.slides.NotesPositions;

SwfOptions swfOptions = new SwfOptions();
swfOptions.setViewerIncluded(false);  // Εξαίρεση προγράμματος προβολής από την έξοδο

// Προαιρετικά, ορίστε τη θέση των σημειώσεων για καλύτερη διάταξη
swfOptions.getNotesCommentsLayouting().setNotesPosition(NotesPositions.BottomFull);
```

#### Βήμα 3: Αποθήκευση παρουσίασης ως SWF
Αποθηκεύστε την παρουσίαση σε μορφή SWF:
```java
define String outputDir = "YOUR_OUTPUT_DIRECTORY";

presentation.save(outputDir + "/SaveAsSwf_out.swf", SaveFormat.Swf, swfOptions);
```
Βεβαιωθείτε ότι οι πόροι απελευθερώνονται σωστά χρησιμοποιώντας ένα μπλοκ try-final.

### Μετατροπή παρουσίασης σε SWF με το Viewer
Για σενάρια όπου χρειάζεται ο θεατής:

#### Βήμα 1: Αρχικοποίηση και διαμόρφωση
Δημιουργήστε το `Presentation` αντικείμενο:
```java
Presentation presentation = new Presentation(dataDir + "/HelloWorld.pptx");
```

#### Βήμα 2: Ορισμός συμπερίληψης θεατή
Ρυθμίστε τις παραμέτρους συμπερίληψης του προγράμματος προβολής στις επιλογές SWF:
```java
SwfOptions swfOptions = new SwfOptions();
swfOptions.setViewerIncluded(true);  // Συμπερίληψη προγράμματος προβολής στην έξοδο
```

#### Βήμα 3: Αποθήκευση με το πρόγραμμα προβολής
Αποθηκεύστε την παρουσίασή σας για να συμπεριλάβετε τη διεπαφή προβολής:
```java
presentation.save(outputDir + "/SaveNotes_out.swf", SaveFormat.Swf, swfOptions);
```
Βεβαιωθείτε ότι οι πόροι διαχειρίζονται σωστά.

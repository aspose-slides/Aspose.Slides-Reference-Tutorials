## Οδηγός Εφαρμογής

### Εξαγωγή παρουσίασης PowerPoint σε HTML με Responsive Layout

Αυτή η λειτουργία σάς επιτρέπει να εξάγετε τις διαφάνειες του PowerPoint σε μορφή HTML που προσαρμόζεται αυτόματα σε διάφορα μεγέθη οθόνης, διασφαλίζοντας ότι το περιεχόμενό σας παρουσιάζεται πάντα όμορφα.

#### Βήμα 1: Δημιουργία `HtmlOptions` Αντικείμενο

Ξεκινήστε δημιουργώντας ένα `HtmlOptions` αντικείμενο. Αυτό σας επιτρέπει να καθορίσετε τον τρόπο εξαγωγής της παρουσίασης ως HTML.
```java
import com.aspose.slides.HtmlOptions;

// Αρχικοποίηση HtmlOptions
HtmlOptions saveOptions = new HtmlOptions();
```

#### Βήμα 2: Ενεργοποίηση διάταξης με δυνατότητα προσαρμογής

Για να διασφαλίσετε ότι τα στοιχεία SVG σας ανταποκρίνονται, ορίστε το `SvgResponsiveLayout` ορίστε την επιλογή σε true. Αυτό θα διασφαλίσει ότι η εξαγόμενη HTML προσαρμόζεται απρόσκοπτα σε διαφορετικές συσκευές.
```java
saveOptions.setSvgResponsiveLayout(true);
```

#### Βήμα 3: Αποθήκευση της παρουσίασης ως HTML

Τέλος, χρησιμοποιήστε το `save` μέθοδος του `Presentation` αντικείμενο, μεταβιβάζοντας τη διαδρομή για το αρχείο εξόδου και τις διαμορφωμένες επιλογές σας.
```java
import com.aspose.slides.SaveFormat;

public class ExportToHTMLWithResponsiveLayout {
    public static void main(String[] args) {
        String dataDir = "YOUR_DOCUMENT_DIRECTORY";
        
        Presentation presentation = new Presentation(dataDir + "/SomePresentation.pptx");
        try {
            HtmlOptions saveOptions = new HtmlOptions();
            saveOptions.setSvgResponsiveLayout(true);

            // Αποθήκευση της παρουσίασης ως HTML
            presentation.save(dataDir + "/SomePresentation-out.html", SaveFormat.Html, saveOptions);
        } finally {
            if (presentation != null) presentation.dispose();
        }
    }
}
```

### Συμβουλές αντιμετώπισης προβλημάτων

- **Συνήθη προβλήματα:** Εάν η έξοδος HTML δεν ανταποκρίνεται, ελέγξτε ξανά ότι `SvgResponsiveLayout` έχει οριστεί σε true.
- **Διαχείριση Πόρων:** Να βεβαιώνεστε πάντα ότι απορρίπτετε το `Presentation` αντίρρηση για την απελευθέρωση πόρων.

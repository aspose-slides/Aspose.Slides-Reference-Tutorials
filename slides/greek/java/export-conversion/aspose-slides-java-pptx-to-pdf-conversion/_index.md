## Οδηγός Εφαρμογής

### Μετατροπή PowerPoint σε PDF
Αυτή η λειτουργία απλοποιεί τη μετατροπή παρουσιάσεων PowerPoint σε PDF, διατηρώντας παράλληλα την αρχική μορφοποίηση. Δείτε πώς επιτυγχάνεται αυτό βήμα προς βήμα:

#### Βήμα 1: Φόρτωση της παρουσίασης πηγής
Αρχικά, φορτώστε το αρχείο της παρουσίασής σας χρησιμοποιώντας το `Presentation` τάξη.
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Διαδρομή προς τον κατάλογο εγγράφων σας

// Δημιουργήστε ένα αντικείμενο παρουσίασης που αντιπροσωπεύει ένα αρχείο PowerPoint
Presentation presentation = new Presentation(dataDir + "/ConvertToPDF.pptx");
```
Ο `Presentation` Ο κατασκευαστής διαβάζει το καθορισμένο `.pptx` αρχείο, προετοιμάζοντάς το για μετατροπή.

#### Βήμα 2: Αποθήκευση ως PDF
Στη συνέχεια, αποθηκεύστε την παρουσίαση που φορτώθηκε ως PDF χρησιμοποιώντας το `save()` μέθοδος.
```java
String outputDir = "YOUR_OUTPUT_DIRECTORY"; // Διαδρομή προς τον κατάλογο εξόδου σας

try {
    // Αποθηκεύστε την παρουσίαση ως PDF στον καθορισμένο κατάλογο εξόδου
    presentation.save(outputDir + "/output_out.pdf", SaveFormat.Pdf);
} finally {
    if (presentation != null) presentation.dispose(); // Καθαρίστε τους πόρους
}
```
Αυτό το βήμα διασφαλίζει ότι το αρχείο που έχει μετατραπεί αποθηκεύεται στην επιθυμητή τοποθεσία.

#### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι οι διαδρομές προς τους καταλόγους σας έχουν καθοριστεί σωστά.
- Ελέγξτε για δικαιώματα εγγραφής στον κατάλογο εξόδου.

### Φόρτωση αρχείου παρουσίασης
Η φόρτωση μιας υπάρχουσας παρουσίασης είναι απαραίτητη πριν από οποιαδήποτε επεξεργασία ή μετατροπή. Δείτε πώς μπορείτε να το κάνετε:

#### Βήμα 1: Φόρτωση της παρουσίασης
Δημιουργήστε μια νέα παρουσία του `Presentation` κλάση, η οποία θα φορτώσει το καθορισμένο αρχείο PPTX.
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Διαδρομή προς τον κατάλογο εγγράφων σας

// Φόρτωση μιας υπάρχουσας παρουσίασης από την καθορισμένη διαδρομή
Presentation loadedPresentation = new Presentation(dataDir + "/LoadPresentation.pptx");
```
Αυτό το βήμα αρχικοποιεί το `loadedPresentation` αντικείμενο, καθιστώντας το έτοιμο για περαιτέρω επεξεργασία.

#### Βήμα 2: Διαχείριση Πόρων
Να διασφαλίζετε πάντα ότι οι πόροι αποδεσμεύονται σωστά μετά τις λειτουργίες.
```java
try {
    // Το αντικείμενο Presentation είναι πλέον έτοιμο για χειρισμό ή μετατροπή.
} finally {
    if (loadedPresentation != null) loadedPresentation.dispose(); // Βεβαιωθείτε ότι οι πόροι απελευθερώνονται
}
```
Η σωστή διαχείριση πόρων αποτρέπει τις διαρροές μνήμης και διασφαλίζει την αποτελεσματική απόδοση των εφαρμογών.

Κατά τη μετατροπή παρουσιάσεων σε HTML, η διατήρηση των προσαρμοσμένων γραμματοσειρών σας είναι ζωτικής σημασίας λόγω των προεπιλεγμένων ρυθμίσεων γραμματοσειράς. Αυτός ο οδηγός δείχνει πώς το Aspose.Slides για Java μπορεί να σας βοηθήσει να εξαιρέσετε αυτές τις προεπιλογές και να διασφαλίσετε συνεπή τυπογραφία σε διάφορες πλατφόρμες.
**Τι θα μάθετε:**
- Ρύθμιση του περιβάλλοντος με το Aspose.Slides για Java
- Τεχνικές για την εξαίρεση προεπιλεγμένων γραμματοσειρών κατά τη μετατροπή HTML
- Βασικές επιλογές διαμόρφωσης και οι επιπτώσεις τους στην έξοδο
- Πρακτικές εφαρμογές σε σενάρια πραγματικού κόσμου
Ας ξεκινήσουμε συζητώντας τις προϋποθέσεις πριν εμβαθύνουμε στον οδηγό υλοποίησης.
## Προαπαιτούμενα
Για να ακολουθήσετε αποτελεσματικά αυτό το σεμινάριο, βεβαιωθείτε ότι έχετε:
- **Aspose.Slides για τη βιβλιοθήκη Java**Εγκαταστήστε την έκδοση 25.4 ή νεότερη.

- **Βασικές γνώσεις προγραμματισμού Java**: Προϋποτίθεται εξοικείωση με τη σύνταξη της Java και βασικές έννοιες προγραμματισμού.

**Maven:**
```xml

    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
```

## Οδηγός Εφαρμογής
### Επισκόπηση λειτουργιών: Εξαίρεση προεπιλεγμένων γραμματοσειρών από τη μετατροπή HTML
Αυτή η λειτουργία βοηθά στην προσαρμογή του χειρισμού των γραμματοσειρών κατά τη μετατροπή αρχείων PowerPoint σε HTML, βελτιώνοντας την επωνυμία και τη συνέπεια.
#### Βήμα 1: Προετοιμάστε το περιβάλλον σας
Βεβαιωθείτε ότι το Aspose.Slides έχει ρυθμιστεί σωστά σύμφωνα με τις παραπάνω οδηγίες. Αυτό περιλαμβάνει την προσθήκη εξαρτήσεων ή τη λήψη του JAR απευθείας στο έργο σας.
#### Βήμα 2: Φόρτωση της παρουσίασης
Φορτώστε την παρουσίασή σας χρησιμοποιώντας το `Presentation` τάξη:
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY/presentation.pptx";
try {
    Presentation pres = new Presentation(dataDir);
```
#### Βήμα 3: Ορισμός εξαιρέσεων γραμματοσειρών
Δημιουργήστε έναν πίνακα για να καθορίσετε τις γραμματοσειρές που θέλετε να εξαιρέσετε. Σε αυτό το παράδειγμα, ξεκινάμε με μια κενή λίστα ως σύμβολο κράτησης θέσης:
```java
String[] fontNameExcludeList = {};
```
#### Βήμα 4: Αρχικοποίηση προσαρμοσμένου ελεγκτή HTML
Ο `LinkAllFontsHtmlController` Η κλάση χρησιμοποιείται για τον χειρισμό προσαρμοσμένων γραμματοσειρών κατά τη διάρκεια της διαδικασίας μετατροπής.
```java
LinkAllFontsHtmlController linkcont = new LinkAllFontsHtmlController(fontNameExcludeList, "YOUR_DOCUMENT_DIRECTORY");
```
#### Βήμα 5: Ρύθμιση παραμέτρων επιλογών HTML
Ρυθμίστε το δικό σας `HtmlOptions` για να χρησιμοποιήσετε τον προσαρμοσμένο μορφοποιητή:
```java
HtmlOptions htmlOptionsEmbed = new HtmlOptions();
htmlOptionsEmbed.setHtmlFormatter(HtmlFormatter.createCustomFormatter(linkcont));
```
#### Βήμα 6: Αποθήκευση ως HTML
Τέλος, αποθηκεύστε την παρουσίαση που έχει μετατραπεί σε μορφή HTML:
```java
pres.save("YOUR_OUTPUT_DIRECTORY/pres.html", SaveFormat.Html, htmlOptionsEmbed);
} catch (Exception e) {
    e.printStackTrace();
}
```

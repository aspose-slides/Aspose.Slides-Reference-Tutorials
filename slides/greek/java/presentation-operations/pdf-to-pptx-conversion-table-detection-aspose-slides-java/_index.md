### Διαχείριση Εξαρτήσεων

**Maven**

Προσθέστε τα παρακάτω στο δικό σας `pom.xml`:

```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

Συμπεριλάβετε αυτό στο δικό σας `build.gradle`:

## Οδηγός Εφαρμογής

### Ρύθμιση παραμέτρων επιλογών εισαγωγής PDF με ανίχνευση πίνακα

Για να μετατρέψουμε ένα PDF σε PPTX κατά την ανίχνευση πινάκων, θα χρησιμοποιήσουμε συγκεκριμένες λειτουργίες της βιβλιοθήκης Aspose.Slides.

#### Επισκόπηση

Θα διαμορφώσουμε `PdfImportOptions` για την ανίχνευση και τη διατήρηση δομών πινάκων μέσα στο PDF σας κατά τη μετατροπή. Αυτή η λειτουργία διασφαλίζει ότι διατηρείται η ακεραιότητα των δεδομένων στις διαφάνειες του PowerPoint.

#### Βήμα προς βήμα εφαρμογή

**Βήμα 1: Ρύθμιση παραμέτρων επιλογών εισαγωγής PDF**

Δημιουργήστε μια παρουσία του `PdfImportOptions` και ενεργοποιήστε την ανίχνευση πίνακα:

```java
PdfImportOptions pdfImportOptions = new PdfImportOptions();
pdfImportOptions.setDetectTables(true);
```

Αυτή η ρύθμιση παραμέτρων υποδεικνύει στο Aspose.Slides να αναζητήσει πίνακες στο περιεχόμενο PDF.

**Βήμα 2: Άνοιγμα ροής εισόδου αρχείου**

Προετοιμάστε τη ροή εισόδου του εγγράφου PDF σας. Βεβαιωθείτε ότι έχετε αντικαταστήσει `"YOUR_DOCUMENT_DIRECTORY/SimpleTableExample.pdf"` με τη διαδρομή προς το πραγματικό σας αρχείο:

```java
FileInputStream stream = new FileInputStream("YOUR_DOCUMENT_DIRECTORY/SimpleTableExample.pdf");
```

**Βήμα 3: Προσθήκη διαφανειών από PDF**

Χρησιμοποιήστε το `addFromPdf` μέθοδος του `Presentation` κλάση, περνώντας τόσο τις επιλογές ροής εισόδου όσο και τις επιλογές εισαγωγής:

```java
pres.getSlides().addFromPdf(stream, pdfImportOptions);
```

Αυτό το βήμα μετατρέπει κάθε σελίδα του PDF σας σε μια διαφάνεια στην παρουσίαση του PowerPoint.

**Βήμα 4: Αποθήκευση της παρουσίασης**

Τέλος, αποθηκεύστε το νεοδημιουργημένο αρχείο PPTX σε έναν κατάλογο εξόδου:

```java
pres.save("YOUR_OUTPUT_DIRECTORY/SimpleTableExample.pptx", SaveFormat.Pptx);
```

Φροντίστε να αντικαταστήσετε `"YOUR_OUTPUT_DIRECTORY/SimpleTableExample.pptx"` με την επιθυμητή σας διαδρομή.

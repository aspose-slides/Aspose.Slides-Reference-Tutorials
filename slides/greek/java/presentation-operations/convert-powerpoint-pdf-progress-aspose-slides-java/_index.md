## Οδηγός Εφαρμογής

### Μετατροπή PowerPoint σε PDF με ενημερώσεις προόδου

Αυτή η λειτουργία σάς επιτρέπει να μετατρέψετε ένα αρχείο PPT σε μορφή PDF, ενώ παράλληλα λαμβάνετε ενημερώσεις προόδου κατά τη διάρκεια της διαδικασίας μετατροπής.

#### Βήμα 1: Φόρτωση της παρουσίασης
Αρχικά, φορτώστε την παρουσίασή σας χρησιμοποιώντας το Aspose.Slides. Αντικαταστήστε `"YOUR_DOCUMENT_DIRECTORY/ConvertToPDF.pptx"` με τη διαδρομή προς το αρχείο PowerPoint:
```java
Presentation presentation = new Presentation("YOUR_DOCUMENT_DIRECTORY/ConvertToPDF.pptx");
```

#### Βήμα 2: Ρύθμιση παραμέτρων επιλογών αποθήκευσης PDF
Δημιουργήστε μια παρουσία του `PdfOptions` για να καθορίσετε πώς θέλετε να συμπεριφέρεται η μετατροπή PDF. Αυτό επιτρέπει την προσαρμογή και πρόσθετες ρυθμίσεις κατά τη διαδικασία αποθήκευσης:
```java
ISaveOptions saveOptions = new PdfOptions();
```

#### Βήμα 3: Ρύθμιση αναφοράς προόδου
Ορίστε έναν χειριστή επανάκλησης προόδου χρησιμοποιώντας την προσαρμοσμένη κλάση μας `ExportProgressHandler` για να παρακολουθείτε την πρόοδο της μετατροπής.
```java
saveOptions.setProgressCallback(new ExportProgressHandler());
```

#### Βήμα 4: Αποθηκεύστε την παρουσίαση ως PDF
Αποθηκεύστε την παρουσίασή σας σε μορφή PDF. Βεβαιωθείτε ότι έχετε αντικαταστήσει `"YOUR_OUTPUT_DIRECTORY/ConvertToPDF.pdf"` με την επιθυμητή διαδρομή εξόδου:
```java
presentation.save("YOUR_OUTPUT_DIRECTORY/ConvertToPDF.pdf", SaveFormat.Pdf, saveOptions);
```
Να θυμάστε πάντα να απορρίπτετε το `Presentation` αντίρρηση για την απελευθέρωση πόρων:
```java
finally {
    if (presentation != null) presentation.dispose();
}
```

#### Υλοποίηση του ExportProgressHandler
Ο `ExportProgressHandler` η κλάση υλοποιεί το `IProgressCallback` διεπαφή για την αναφορά ενημερώσεων προόδου.
```java
class ExportProgressHandler implements IProgressCallback {
    public void reporting(double progressValue) {
        long progress = Math.round(progressValue);
        System.out.println(progress + "% file converted");
    }
}
```
Αυτός ο χειριστής παρέχει απλή έξοδο κονσόλας που υποδεικνύει το ποσοστό μετατροπής, κάτι που είναι χρήσιμο για την παρακολούθηση μεγάλων μετατροπών.

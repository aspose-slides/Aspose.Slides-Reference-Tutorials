Για να ξεκινήσετε να χρησιμοποιείτε το Aspose.Slides για Java, ενσωματώστε το στο έργο σας:

**Maven:**
```xml

    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

**Βαθμός:**

## Οδηγός Εφαρμογής

### Μετατροπή παρουσίασης σε HTML με ενσωματωμένα αρχεία πολυμέσων
Αυτή η ενότητα καλύπτει τη μετατροπή μιας παρουσίασης PowerPoint σε αρχείο HTML, συμπεριλαμβανομένων τυχόν ενσωματωμένων μέσων.

#### 1. Καθορίστε τον κατάλογο εγγράφου προέλευσης και τις διαδρομές εξόδου
Ξεκινήστε ορίζοντας πού βρίσκονται τα αρχεία προέλευσης και πού θέλετε να αποθηκευτεί η HTML εξόδου:
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
String htmlDocumentFileName = "presentationWithVideo.html"; 
String presentationPath = dataDir + "presentationWith.pptx";
```

#### 2. Φορτώστε την παρουσίαση PowerPoint
Φορτώστε το αρχείο παρουσίασής σας χρησιμοποιώντας το Aspose.Slides:
```java
Presentation pres = new Presentation(presentationPath);
try {
    // Τα βήματα επεξεργασίας βρίσκονται εδώ
} finally {
    if (pres != null) pres.dispose();
}
```

#### 3. Χειρισμός αρχείων πολυμέσων με το VideoPlayerHtmlController
Για να διαχειριστείτε αρχεία πολυμέσων, δημιουργήστε μια παρουσία του `VideoPlayerHtmlController`:
```java
VideoPlayerHtmlController controller = new VideoPlayerHtmlController("", htmlDocumentFileName, "http://www.example.com/");
```
Αυτό διασφαλίζει ότι τα βίντεο και άλλα μέσα υποβάλλονται σε σωστή επεξεργασία κατά τη μετατροπή.

#### 4. Διαμόρφωση επιλογών HTML για μετατροπή
Ορίστε τις απαραίτητες επιλογές για να ορίσετε τον τρόπο μετατροπής των διαφανειών:
```java
HtmlOptions htmlOptions = new HtmlOptions(controller);
SVGOptions svgOptions = new SVGOptions(controller); // Προαιρετικά: Χειρίζεται εικόνες SVG μέσα σε διαφάνειες

htmlOptions.setHtmlFormatter(HtmlFormatter.createCustomFormatter(controller));
htmlOptions.setSlideImageFormat(SlideImageFormat.svg(svgOptions));
```
Αυτές οι επιλογές επιτρέπουν την προσαρμοσμένη μορφοποίηση και τον χειρισμό εικόνων.

#### 5. Αποθηκεύστε την παρουσίαση ως HTML
Τέλος, αποθηκεύστε την παρουσίασή σας σε μορφή HTML χρησιμοποιώντας τις διαμορφωμένες επιλογές:
```java
pres.save(htmlDocumentFileName, SaveFormat.Html, htmlOptions);
```

### Βασική Αρχικοποίηση
Μόλις εγκατασταθεί, αρχικοποιήστε το Aspose.Slides στο έργο Java σας. Δείτε πώς μπορείτε να το ρυθμίσετε:
```java
import com.aspose.slides.Presentation;
import com.aspose.slides.SaveFormat;

public class ConvertToTiff {
    public static void main(String[] args) {
        Presentation pres = new Presentation("YOUR_DOCUMENT_DIRECTORY/Convert_Tiff_Default.pptx");
        try {
            // Μετατροπή και αποθήκευση της παρουσίασης σε αρχείο TIFF
            pres.save("YOUR_OUTPUT_DIRECTORY/Tiff_out.tiff", SaveFormat.Tiff);
        } finally {
            if (pres != null) pres.dispose();
        }
    }
}
```
Αυτή η ρύθμιση δείχνει την αρχικοποίηση ενός `Presentation` αντικείμενο, μετατρέποντάς το και διασφαλίζοντας ότι οι πόροι απελευθερώνονται σωστά.

#### Αποθήκευση ως TIFF (H3)
Χρησιμοποιήστε το `save` μέθοδος για τη μετατροπή της φορτωμένης παρουσίασης σε εικόνα TIFF:

- **Γιατί;**: Το `SaveFormat.Tiff` Η επιλογή καθορίζει ότι η έξοδος θα πρέπει να είναι σε μορφή TIFF, εξασφαλίζοντας αποτελέσματα υψηλής ποιότητας.

- **Γιατί;**Αυτό το βήμα είναι κρίσιμο για την αποτελεσματική διαχείριση πόρων και την αποτροπή διαρροών μνήμης στην εφαρμογή σας.

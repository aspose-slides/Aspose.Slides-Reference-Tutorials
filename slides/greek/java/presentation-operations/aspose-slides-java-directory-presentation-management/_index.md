## Οδηγός Εφαρμογής

### Λειτουργία 1: Δημιουργία και διαχείριση καταλόγων

#### Επισκόπηση
Η αποτελεσματική διαχείριση καταλόγων είναι ζωτικής σημασίας για κάθε εφαρμογή που χειρίζεται αρχεία. Αυτή η λειτουργία δείχνει πώς να ελέγξετε εάν υπάρχει ένας κατάλογος και να τον δημιουργήσετε εάν είναι απαραίτητο, διασφαλίζοντας ότι οι εφαρμογές σας χειρίζονται τις διαδρομές αποθήκευσης με ομαλό τρόπο.

##### Έλεγχος και δημιουργία καταλόγου

```java
import java.io.File;

public class DirectoryManager {
    public static void main(String[] args) {
        // Ορίστε τη διαδρομή όπου θα αποθηκευτούν τα έγγραφα.
        String documentDirectory = "YOUR_DOCUMENT_DIRECTORY";

        // Ελέγξτε αν ο κατάλογος υπάρχει. Εάν όχι, δημιουργήστε τον.
        boolean isExists = new File(documentDirectory).exists();
        if (!isExists) {
            new File(documentDirectory).mkdirs();  // Δημιουργεί καταλόγους αναδρομικά
        }
    }
}
```

- **Εξήγηση:** Ο `File` Η κλάση ελέγχει την ύπαρξη ενός καταλόγου και τον δημιουργεί χρησιμοποιώντας `mkdirs()` αν δεν υπάρχει. Αυτό διασφαλίζει ότι δημιουργούνται επίσης όλοι οι απαραίτητοι γονικοί κατάλογοι, αποφεύγοντας πιθανά σφάλματα.

### Λειτουργία 2: Δημιουργία παρουσίασης και αποθήκευση σε δίσκο

#### Επισκόπηση
Η δημιουργία δυναμικών παρουσιάσεων μέσω προγραμματισμού μπορεί να εξοικονομήσει χρόνο και να βελτιώσει τη συνέπεια. Αυτή η λειτουργία καλύπτει τη δημιουργία μιας νέας παρουσίασης, την προσθήκη σχημάτων με γεμίσματα μοτίβων και την αποθήκευση του αρχείου χρησιμοποιώντας το Aspose.Slides για Java.

##### Δημιουργία και αποθήκευση παρουσίασης

```java
import com.aspose.slides.*;

public class PresentationManager {
    public static void main(String[] args) {
        // Δημιουργήστε μια κλάση παρουσίασης που αντιπροσωπεύει ένα αρχείο PPTX.
        Presentation pres = new Presentation();
        try {
            // Αποκτήστε την πρώτη διαφάνεια από την παρουσίαση.
            ISlide sld = pres.getSlides().get_Item(0);

            // Προσθέστε ένα αυτόματο σχήμα τύπου ορθογωνίου στη διαφάνεια σε καθορισμένη θέση και μέγεθος.
            IShape shp = sld.getShapes().addAutoShape(ShapeType.Rectangle, 50, 150, 75, 150);

            // Ορίστε τον τύπο γεμίσματος του σχήματος σε Μοτίβο.
            shp.getFillFormat().setFillType(FillType.Pattern);

            // Ορίστε το στυλ μοτίβου ως Trellis.
            shp.getFillFormat().getPatternFormat().setPatternStyle(PatternStyle.Trellis);

            // Ορίστε το χρώμα πίσω και το χρώμα μπροστινού μέρους του μοτίβου.
            shp.getFillFormat().getPatternFormat().getBackColor().setColor(Color.LIGHT_GRAY);
            shp.getFillFormat().getPatternFormat().getForeColor().setColor(Color.YELLOW);

            // Ορίστε τη διαδρομή του καταλόγου εξόδου για την αποθήκευση του αρχείου παρουσίασης.
            String outputDirectory = "YOUR_OUTPUT_DIRECTORY";

            // Αποθηκεύστε την παρουσίαση στο δίσκο σε μορφή PPTX.
            pres.save(outputDirectory + "/RectShpPatt_out.pptx", SaveFormat.Pptx);
        } finally {
            if (pres != null) pres.dispose();  // Καθαρίστε τους πόρους
        }
    }
}
```

- **Εξήγηση:** Αυτό το τμήμα κώδικα αρχικοποιεί μια νέα παρουσίαση, προσθέτει ένα ορθογώνιο σχήμα με γέμισμα μοτίβου πλέγματος στην πρώτη διαφάνεια και την αποθηκεύει. `try-finally` Το μπλοκ διασφαλίζει ότι οι πόροι απελευθερώνονται σωστά.

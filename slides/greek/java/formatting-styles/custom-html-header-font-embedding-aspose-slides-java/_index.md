## Οδηγός Εφαρμογής

### Ελεγκτής προσαρμοσμένης κεφαλίδας και γραμματοσειρών

#### Επισκόπηση

Ο `CustomHeaderAndFontsController` Η κλάση σάς επιτρέπει να προσαρμόσετε την κεφαλίδα HTML των παρουσιάσεών σας που έχουν μετατραπεί, αναφέροντας ένα αρχείο CSS. Επιπλέον, διασφαλίζει ότι όλες οι γραμματοσειρές που χρησιμοποιούνται στην παρουσίασή σας είναι ενσωματωμένες, διατηρώντας την ακεραιότητα του σχεδιασμού σε διαφορετικές πλατφόρμες.

#### Βήμα προς βήμα εφαρμογή

##### 1. Δημιουργήστε την κλάση Custom Header και Fonts Controller

Ξεκινήστε δημιουργώντας μια νέα κλάση Java με το όνομα `CustomHeaderAndFontsController` που εκτείνεται `EmbedAllFontsHtmlController`:

```java
import com.aspose.slides.EmbedAllFontsHtmlController;
import com.aspose.slides.IHtmlGenerator;
import com.aspose.slides.IPresentation;

public class CustomHeaderAndFontsController extends EmbedAllFontsHtmlController {
    // Προσαρμοσμένο πρότυπο κεφαλίδας με ενσωματωμένη αναφορά αρχείου CSS
    private static String Header = "<!DOCTYPE html>
" +
            "<html>
" +
            "<head>
" +
            "<meta http-equiv="Content-Type" content="text/html; charset=UTF-8">
" +
            "<meta http-equiv="X-UA-Compatible" content="IE=9">
" +
            "<link rel="stylesheet" type="text/css" href="{0}">
" +
            "</head>";

    private String m_cssFileName;

    // Κατασκευαστής για να ορίσετε το όνομα αρχείου CSS για την προσαρμοσμένη κεφαλίδα
    public CustomHeaderAndFontsController(String cssFileName) {
        this.m_cssFileName = cssFileName;
    }

    // Μέθοδος παράκαμψης για την εγγραφή της αρχής του εγγράφου με μια προσαρμοσμένη κεφαλίδα HTML
    @Override
    public void writeDocumentStart(IHtmlGenerator generator, IPresentation presentation) {
        // Προσθήκη προσαρμοσμένης κεφαλίδας HTML χρησιμοποιώντας μορφοποιημένη συμβολοσειρά με όνομα αρχείου CSS
        generator.addHtml(String.format(Header, m_cssFileName));
        // Κλήση μεθόδου για ενσωμάτωση όλων των γραμματοσειρών στην παρουσίαση
        writeAllFonts(generator, presentation);
    }

    // Μέθοδος παράκαμψης για την προσθήκη σχολίου ενσωματωμένων γραμματοσειρών και την κλήση της γονικής μεθόδου για την ενσωμάτωση γραμματοσειρών
    @Override
    public void writeAllFonts(IHtmlGenerator generator, IPresentation presentation) {
        // Προσθήκη σχολίου που υποδεικνύει ότι όλες οι γραμματοσειρές ενσωματώνονται
        generator.addHtml("<!-- Embedded fonts -->");
        // Καλέστε τη μέθοδο superclass για να εκτελέσετε την πραγματική ενσωμάτωση γραμματοσειράς
        super.writeAllFonts(generator, presentation);
    }
}
```

##### 2. Επεξήγηση Βασικών Στοιχείων

- **Πρότυπο κεφαλίδας:** Ο `Header` Το string είναι ένα πρότυπο για την κεφαλίδα HTML που περιλαμβάνει μετα-ετικέτες και έναν σύνδεσμο προς το αρχείο CSS σας.
- **Κατασκευαστής:** Λαμβάνει τη διαδρομή του αρχείου CSS ως όρισμα που θα χρησιμοποιηθεί στην κεφαλίδα.
- **Μέθοδος writeDocumentStart:** Αυτή η μέθοδος παρακάμπτει τη λειτουργικότητα της βασικής κλάσης, προσθέτοντας μια προσαρμοσμένη κεφαλίδα στην αρχή του εγγράφου. Χρησιμοποιεί `String.format` για να εισαγάγετε το όνομα του αρχείου CSS στο πρότυπο HTML.
- **Μέθοδος writeAllFonts:** Προσθέτει ένα σχόλιο που υποδεικνύει την ενσωμάτωση γραμματοσειράς και καλεί τη μέθοδο της υπερκλάσης για να χειριστεί την πραγματική διαδικασία ενσωμάτωσης.

#### Βασικές επιλογές διαμόρφωσης

- **Διαδρομή αρχείου CSS:** Βεβαιωθείτε ότι η διαδρομή CSS έχει καθοριστεί σωστά στον κατασκευαστή, καθώς θα ενσωματωθεί στην κεφαλίδα HTML.
  
#### Συμβουλές αντιμετώπισης προβλημάτων

- Εάν οι γραμματοσειρές δεν εμφανίζονται όπως αναμένεται, επαληθεύστε ότι τα αρχεία γραμματοσειρών είναι προσβάσιμα και ότι έχουν σωστή αναφορά.
- Ελέγξτε για τυχόν σφάλματα ή προειδοποιήσεις κατά τη διάρκεια της διαδικασίας δημιουργίας, τα οποία μπορεί να υποδεικνύουν προβλήματα με εξαρτήσεις ή άδειες χρήσης.

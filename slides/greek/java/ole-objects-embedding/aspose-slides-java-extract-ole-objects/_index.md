## Οδηγός Εφαρμογής
Θα αναλύσουμε την εφαρμογή μας σε τρία κύρια χαρακτηριστικά:

### 1. Φόρτωση και πρόσβαση σε μια διαφάνεια παρουσίασης

#### Επισκόπηση
Η φόρτωση ενός αρχείου παρουσίασης είναι το πρώτο βήμα για την πρόσβαση στο περιεχόμενό του, συμπεριλαμβανομένων των διαφανειών και των ενσωματωμένων αντικειμένων.

#### Βήματα για την εφαρμογή

##### Αρχικοποίηση του αντικειμένου παρουσίασης

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY/";
Presentation pres = new Presentation(dataDir + "AccessingOLEObjectFrame.pptx");
```

Εδώ, `dataDir` θα πρέπει να αντικατασταθεί με τη διαδρομή όπου βρίσκεται το αρχείο παρουσίασής σας.

##### Πρόσβαση στην πρώτη διαφάνεια

```java
ISlide sld = pres.getSlides().get_Item(0);
```

Αυτός ο κώδικας έχει πρόσβαση στην πρώτη διαφάνεια της παρουσίασης. Μπορείτε να κάνετε επανάληψη στις διαφάνειες επαναλαμβάνοντας `pres.getSlides()` αν χρειαστεί.

### 2. Μετατροπή και πρόσβαση σε πλαίσιο αντικειμένου OLE

#### Επισκόπηση
Για να αλληλεπιδράσουμε με ενσωματωμένα αντικείμενα, πρέπει να μετατρέψουμε τα σχήματα διαφανειών σε `OleObjectFrame`.

#### Βήματα για την εφαρμογή

##### Πρόσβαση στο πρώτο σχήμα σε μια διαφάνεια

```java
OleObjectFrame oleObjectFrame = (OleObjectFrame) sld.getShapes().get_Item(0);
```

Βεβαιωθείτε ότι το σχήμα είναι πράγματι ένα αντικείμενο OLE πριν από τη μετατροπή, καθώς η λανθασμένη μετατροπή μπορεί να οδηγήσει σε σφάλματα χρόνου εκτέλεσης.

### 3. Εξαγωγή και αποθήκευση ενσωματωμένων δεδομένων αντικειμένου OLE

#### Επισκόπηση
Η εξαγωγή ενσωματωμένων δεδομένων από αντικείμενα OLE σάς επιτρέπει να τα χειρίζεστε ή να τα αποθηκεύετε ξεχωριστά.

#### Βήματα για την εφαρμογή

##### Εξαγωγή δεδομένων ενσωματωμένου αρχείου

```java
byte[] data = oleObjectFrame.getEmbeddedData().getEmbeddedFileData();
String fileExtension = oleObjectFrame.getEmbeddedData().getEmbeddedFileExtension();
```

Εδώ, `data` περιέχει το δυαδικό περιεχόμενο του ενσωματωμένου αντικειμένου και `fileExtension` βοηθά στην αποθήκευσή του με τη σωστή μορφή.

##### Αποθήκευση εξαγόμενων δεδομένων σε αρχείο

```java
String outputDir = "YOUR_OUTPUT_DIRECTORY/";
String extractedPath = outputDir + "excelFromOLE_out" + fileExtension;

try (FileOutputStream fstr = new FileOutputStream(extractedPath)) {
    fstr.write(data, 0, data.length);
}
```

Αυτός ο κώδικας γράφει τα δεδομένα του ενσωματωμένου αντικειμένου σε μια καθορισμένη διαδρομή.

## Οδηγός Εφαρμογής

### Φόρτωση παρουσίασης με επιλογή διαγραφής ενσωματωμένων δυαδικών αντικειμένων
#### Επισκόπηση:
Μάθετε πώς να φορτώνετε μια παρουσίαση και να αφαιρείτε περιττά ενσωματωμένα δυαδικά αντικείμενα, βελτιστοποιώντας το μέγεθος του αρχείου ή εξαλείφοντας ευαίσθητα δεδομένα.
##### Βήμα 1: Εισαγωγή απαραίτητων πακέτων
Βεβαιωθείτε ότι έχετε τις ακόλουθες εισαγωγές:
```java
import com.aspose.slides.Presentation;
import com.aspose.slides.LoadOptions;
import com.aspose.slides.SaveFormat;
```
##### Βήμα 2: Φόρτωση παρουσίασης με επιλογές
Στήνω `LoadOptions` για να διαγράψετε ενσωματωμένα δυαδικά αντικείμενα.
```java
String pptxFileName = "YOUR_DOCUMENT_DIRECTORY/OlePptx.pptx";
LoadOptions loadOption = new LoadOptions();
loadOption.setDeleteEmbeddedBinaryObjects(true);
Presentation pres = new Presentation(pptxFileName, loadOption);
try {
    // Εκτελέστε λειτουργίες στην παρουσίαση εδώ.
    pres.save("YOUR_OUTPUT_DIRECTORY/OlePptx-out.pptx", SaveFormat.Pptx);
} finally {
    if (pres != null) pres.dispose();
}
```
**Εξήγηση:**
- `setDeleteEmbeddedBinaryObjects(true)`Αυτή η επιλογή διασφαλίζει ότι τυχόν ενσωματωμένα δυαδικά αντικείμενα αφαιρούνται κατά τη φόρτωση της παρουσίασης, ενισχύοντας την αποτελεσματικότητα και την ασφάλεια.
### Καταμέτρηση πλαισίων αντικειμένων OLE σε μια παρουσίαση
#### Επισκόπηση:
Μάθετε πώς να μετράτε τόσο τα υπάρχοντα όσο και τα κενά πλαίσια αντικειμένων OLE μέσα στις διαφάνειές σας.
##### Βήμα 1: Εισαγωγή απαιτούμενων πακέτων
```java
import com.aspose.slides.ISlide;
import com.aspose.slides.ISlideCollection;
import com.aspose.slides.IList;
import com.aspose.slides.IShape;
import com.aspose.slides.OleObjectFrame;
```
##### Βήμα 2: Καταμέτρηση πλαισίων αντικειμένων OLE
Χρησιμοποιήστε μια μέθοδο για να επαναλάβετε διαφάνειες και σχήματα για να μετρήσετε πλαίσια OLE.
```java
public static int GetOleObjectFrameCount(ISlideCollection slides) {
    int oleFramesCount = 0;
    int emptyOleFrames = 0;

    for (ISlide sld : slides) {
        for (IShape shape : sld.getShapes()) {
            if (shape instanceof OleObjectFrame) {
                OleObjectFrame objectFrame = (OleObjectFrame) shape;
                oleFramesCount++;

                byte[] embeddedData = objectFrame.getEmbeddedData().getEmbeddedFileData();
                if (embeddedData == null || embeddedData.length == 0) {
                    emptyOleFrames++;
                }
            }
        }
    }

    return oleFramesCount; // Επιστρέψτε τον αριθμό των πλαισίων αντικειμένων OLE
}
```
**Εξήγηση:**
- Αυτή η μέθοδος διασχίζει κάθε διαφάνεια και σχήμα για να αναγνωρίσει `OleObjectFrame` περιπτώσεις.
- Ελέγχει εάν υπάρχουν ενσωματωμένα δεδομένα, μετρώντας ξεχωριστά τόσο το συνολικό όσο και τα κενά πλαίσια.

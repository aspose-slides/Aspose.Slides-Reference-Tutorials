## Ρύθμιση του Aspose.Slides για Java
Για να ξεκινήσετε την ενσωμάτωση αρχείων ZIP σε παρουσιάσεις PowerPoint, θα πρέπει πρώτα να ρυθμίσετε το Aspose.Slides για Java. Δείτε πώς:

### Maven
Προσθέστε την ακόλουθη εξάρτηση στο `pom.xml` αρχείο:
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

### Γκράντλ
Συμπεριλάβετε την εξάρτηση στο `build.gradle` αρχείο:
```gradle
implementation group: 'com.aspose', name: 'aspose-slides', version: '25.4', classifier: 'jdk16'
```

## Οδηγός Εφαρμογής

### Ενσωμάτωση αρχείου ZIP ως αντικειμένου OLE στο PowerPoint
Ακολουθήστε τα παρακάτω βήματα:

#### Βήμα 1: Αρχικοποίηση παρουσίασης
Δημιουργήστε μια νέα παρουσία του `Presentation` τάξη.
```java
class Main {
    public static void main(String[] args) {
        Presentation pres = new Presentation();
        // Περαιτέρω κώδικας...
    }
}
```

#### Βήμα 2: Ορισμός καταλόγου και ανάγνωση αρχείου
Καθορίστε τον κατάλογο του εγγράφου σας και διαβάστε τα byte του αρχείου ZIP:
```java
import java.nio.file.Files;
import java.nio.file.Paths;

String dataDir = "YOUR_DOCUMENT_DIRECTORY";
byte[] fileBytes = Files.readAllBytes(Paths.get(dataDir + "/test.zip"));
```

#### Βήμα 3: Δημιουργία πληροφοριών ενσωματωμένων δεδομένων OLE
Δημιουργήστε ένα `OleEmbeddedDataInfo` αντικείμενο με τα bytes του αρχείου ZIP:
```java
import com.aspose.slides.IOleEmbeddedDataInfo;

IOleEmbeddedDataInfo dataInfo = new OleEmbeddedDataInfo(fileBytes, "zip");
```

#### Βήμα 4: Προσθήκη πλαισίου αντικειμένου OLE σε διαφάνεια
Προσθήκη ενός πλαισίου αντικειμένου OLE στην πρώτη διαφάνεια:
```java
import com.aspose.slides.IOleObjectFrame;

IOleObjectFrame oleFrame = pres.getSlides().get_Item(0).getShapes()
    .addOleObjectFrame(150, 20, 50, 50, dataInfo);
```

#### Βήμα 5: Ορίστε ένα εικονίδιο για ορατότητα
Ορίστε ένα ορατό εικονίδιο για το ενσωματωμένο αντικείμενο:
```java
oleFrame.setObjectIcon(true);
```

#### Βήμα 6: Αποθήκευση παρουσίασης
Αποθηκεύστε την παρουσίασή σας με το ενσωματωμένο αντικείμενο OLE:
```java
pres.save(dataDir + "/EmbeddedZIPInPPT.pptx", SaveFormat.Pptx);
if (pres != null) pres.dispose();
```

### Φόρτωση και αποθήκευση παρουσίασης με ενσωματωμένα αντικείμενα OLE
Φορτώστε μια υπάρχουσα παρουσίαση για να την ενημερώσετε ή να την αποθηκεύσετε ξανά:

#### Φόρτωση υπάρχουσας παρουσίασης
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
class Main {
    public static void main(String[] args) {
        Presentation pres = new Presentation(dataDir + "/EmbeddedZIPInPPT.pptx");
        // Περαιτέρω κώδικας...
    }
}
```

#### Επανάληψη σε διαφάνειες και σχήματα
Πρόσβαση σε αντικείμενα OLE μέσα στις διαφάνειες:
```java
for (ISlide slide : pres.getSlides()) {
    for (IShape shape : slide.getShapes()) {
        if (shape instanceof IOleObjectFrame) {
            IOleObjectFrame oleFrame = (IOleObjectFrame) shape;
            // Εκτέλεση λειτουργιών στο πλαίσιο αντικειμένου OLE
        }
    }
}
```

#### Αποθήκευση ενημερωμένης παρουσίασης
```java
String outputDir = "YOUR_OUTPUT_DIRECTORY";
pres.save(outputDir + "/UpdatedPresentation.pptx", SaveFormat.Pptx);
if (pres != null) pres.dispose();
```

## Οδηγός Εφαρμογής
### Επισκόπηση του χειρισμού διακριτικών διακοπής
Τα διακριτικά διακοπής επιτρέπουν στην εφαρμογή σας να θέτει σε παύση ή να διακόπτει συγκεκριμένες εργασίες με ομαλό τρόπο. Αυτό είναι ιδιαίτερα χρήσιμο κατά την επεξεργασία μεγάλων παρουσιάσεων όπου ένας χρήστης μπορεί να χρειαστεί να ακυρώσει τη λειτουργία πριν από την ολοκλήρωσή της.

### Βήμα προς βήμα εφαρμογή
#### 1. Αρχικοποίηση της πηγής διακριτικού διακοπής
Αρχικά, δημιουργήστε ένα `InterruptionTokenSource` για την παρακολούθηση και τη διαχείριση διακοπών:
```java
import com.aspose.slides.InterruptionTokenSource;

final InterruptionTokenSource tokenSource = new InterruptionTokenSource();
```
#### 2. Δημιουργία μιας εκτελέσιμης εργασίας
Ορίστε την εργασία που φορτώνει και επεξεργάζεται την παρουσίαση:
```java
Runnable task = () -> {
    // Δημιουργήστε επιλογές φόρτωσης με ένα διακριτικό διακοπής.
    LoadOptions options = new LoadOptions();
    options.setInterruptionToken(tokenSource.getToken());

    // Φορτώστε την παρουσίαση χρησιμοποιώντας την καθορισμένη διαδρομή και τις επιλογές.
    Presentation presentation = new Presentation("YOUR_DOCUMENT_DIRECTORY/pres.pptx", options);
    try {
        // Αποθηκεύστε την παρουσίαση σε διαφορετική μορφή.
        presentation.save("YOUR_OUTPUT_DIRECTORY/pres.ppt", SaveFormat.Ppt);
    } finally {
        if (presentation != null) presentation.dispose();
    }
};
```
#### 3. Εκτέλεση και διακοπή της εργασίας
Εκτελέστε την εργασία σε ξεχωριστό νήμα και προσομοιώστε μια διακοπή μετά από κάποια καθυστέρηση:
```java
Thread thread = new Thread(task); // Εκτελέστε την εργασία σε ξεχωριστό νήμα.
thread.start();

Thread.sleep(10000); // Προσομοιώστε κάποια εργασία που εκτελείται πριν από τη διακοπή.

// Ενεργοποιήστε τη διακοπή, επηρεάζοντας την τρέχουσα επεξεργασία.
tokenSource.interrupt();
```
### Επεξήγηση βασικών στοιχείων
- **Πηγή Διακριτικού Διακοπής:** Διαχειρίζεται την κατάσταση των διακοπών και επικοινωνεί με την εκτελούμενη εργασία.
- **LoadOptions.setInterruptionToken():** Συσχετίζει ένα διακριτικό διακοπής με λειτουργίες φόρτωσης παρουσίασης.
- **Presentation.dispose():** Διασφαλίζει ότι οι πόροι απελευθερώνονται σωστά, ακόμη και αν διακοπούν.

## Ρύθμιση του Aspose.Slides για Java
Η έναρξη με το Aspose.Slides για Java είναι απλή, χάρη στην ποικιλία επιλογών εγκατάστασης που είναι διαθέσιμες:

Προσθέστε την ακόλουθη εξάρτηση στο `pom.xml`:
```xml

    <artifactId>aspose-slides</artifactId>

Συμπεριλάβετε αυτό στο δικό σας `build.gradle` αρχείο:

```

### Αλλαγή στυλ σχήματος SmartArt
#### Επισκόπηση
Μάθετε πώς να τροποποιείτε το QuickStyle των σχημάτων SmartArt σε μια διαφάνεια του PowerPoint, ενισχύοντας την οπτική επίδραση της παρουσίασής σας.

**Βήμα 1: Φόρτωση της παρουσίασης**
Ξεκινήστε φορτώνοντας το αρχείο PowerPoint:
```java
Presentation presentation = new Presentation(dataDir + "/AccessSmartArtShape.pptx");
```

**Βήμα 2: Διασχίστε και τροποποιήστε σχήματα**
Επαναλάβετε κάθε σχήμα στην πρώτη διαφάνεια για να εντοπίσετε αντικείμενα SmartArt. Χρησιμοποιήστε τη τυποποίηση για να τροποποιήσετε τα στυλ τους:
```java
for (IShape shape : presentation.getSlides().get_Item(0).getShapes()) {
    if (shape instanceof ISmartArt) {
        ISmartArt smart = (ISmartArt) shape;
        
        // Έλεγχος και αλλαγή του QuickStyle
        if (smart.getQuickStyle() == SmartArtQuickStyleType.SimpleFill) {
            smart.setQuickStyle(SmartArtQuickStyleType.Cartoon);
        }
    }
}
```

**Βήμα 3: Αποθήκευση των αλλαγών**
Αφού κάνετε αλλαγές, αποθηκεύστε την ενημερωμένη παρουσίαση:
```java
presentation.save(dataDir + "/ChangeSmartArtStyle_out.pptx", SaveFormat.Pptx);
```

### Τοποθέτηση και απόρριψη παρουσίασης
#### Επισκόπηση
Διασφαλίστε την ορθή διαχείριση των πόρων φορτώνοντας ένα αρχείο PowerPoint και απορρίπτοντάς το σωστά.

**Βήμα 1: Φόρτωση της παρουσίασης**
Όπως και στην προηγούμενη λειτουργία, φορτώστε την παρουσίασή σας:
```java
Presentation presentation = new Presentation(dataDir + "/AccessSmartArtShape.pptx");
```

**Βήμα 2: Εκτέλεση λειτουργιών**
Για επίδειξη, επαναλάβετε τις διαφάνειες και τα σχήματα, εκτυπώνοντας τους τύπους τους:
```java
for (ISlide slide : presentation.getSlides()) {
    for (IShape shape : slide.getShapes()) {
        System.out.println(shape.getClass().getSimpleName());
    }
}
```

**Βήμα 3: Απόρριψη πόρων**
Πάντα να απορρίπτετε το `Presentation` αντίρρηση για την απελευθέρωση πόρων:
```java
if (presentation != null) presentation.dispose();
```

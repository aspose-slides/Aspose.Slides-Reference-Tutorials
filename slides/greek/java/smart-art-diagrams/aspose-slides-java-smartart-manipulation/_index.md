## Ρύθμιση του Aspose.Slides για Java
Για να ξεκινήσετε, θα χρειαστεί να ενσωματώσετε τη βιβλιοθήκη Aspose.Slides στο έργο Java σας. Μπορείτε να το κάνετε αυτό μέσω του Maven ή του Gradle ή κατεβάζοντας απευθείας το αρχείο JAR από τον ιστότοπο του Aspose.

### Maven
Προσθέστε την ακόλουθη εξάρτηση στο `pom.xml`:

```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

### Γκράντλ
Συμπεριλάβετε αυτό στο δικό σας `build.gradle`:

```gradle
implementation group: 'com.aspose', name: 'aspose-slides', version: '25.4', classifier: 'jdk16'
```

## Οδηγός Εφαρμογής
Τώρα που έχουμε ρυθμίσει το περιβάλλον μας, ας προχωρήσουμε στην εφαρμογή των λειτουργιών χειρισμού SmartArt στην εφαρμογή Java σας. Κάθε λειτουργία θα εξηγηθεί βήμα προς βήμα.

### Προσθήκη SmartArt σε παρουσίαση
#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να προσθέσετε ένα οπτικά ελκυστικό γραφικό SmartArt στις διαφάνειες της παρουσίασής σας.

**Βήμα 1**: Δημιουργία διαφάνειας και προσθήκη SmartArt
- **Σκοπός**: Προσθήκη ενός τύπου Ακτινικού Κύκλου SmartArt σε καθορισμένες συντεταγμένες με καθορισμένες διαστάσεις.

```java
import com.aspose.slides.Presentation;
import com.aspose.slides.ISmartArt;
import com.aspose.slides.SmartArtLayoutType;

Presentation presentation = new Presentation();
try {
    // Δημιουργήστε και προσθέστε το γραφικό SmartArt στην πρώτη διαφάνεια.
    ISmartArt smart = presentation.getSlides().get_Item(0).getShapes().addSmartArt(
        10, 10, 400, 300, SmartArtLayoutType.RadialCycle
    );
} finally {
    if (presentation != null) presentation.dispose();
}
```

**Εξήγηση**: 
- `addSmartArt(int x, int y, int width, int height, SmartArtLayoutType layoutType)` προσθέτει ένα γραφικό SmartArt στη θέση `(x, y)` με καθορισμένες διαστάσεις και τύπο.

### Προσθήκη κόμβου στο SmartArt
#### Επισκόπηση
Μάθετε πώς να προσθέτετε δυναμικά κόμβους σε ένα υπάρχον γραφικό SmartArt για πιο σύνθετη αναπαράσταση πληροφοριών.

**Βήμα 2**: Ανάκτηση κόμβων και προσθήκη νέου κόμβου
- **Σκοπός**Βελτιώστε το SmartArt σας προσθέτοντας επιπλέον στοιχεία (κόμβους).

```java
import com.aspose.slides.ISmartArtNode;

try {
    // Ας υποθέσουμε ότι ο όρος «έξυπνος» έχει ήδη οριστεί από την προηγούμενη ενότητα.
    ISmartArtNode node = smart.getAllNodes().addNode();
} finally {
    if (presentation != null) presentation.dispose();
}
```

**Εξήγηση**: 
- `getAllNodes()` ανακτά όλους τους κόμβους σε ένα SmartArt και `addNode()` προσθέτει ένα νέο.

### Έλεγχος κρυφής ιδιότητας του κόμβου SmartArt
#### Επισκόπηση
Αυτή η λειτουργία σάς βοηθά να διαχειριστείτε την ορατότητα μεμονωμένων κόμβων μέσα στο γραφικό SmartArt σας.

**Βήμα 3**: Επαλήθευση εάν ο κόμβος είναι κρυφός
- **Σκοπός**: Προσδιορίστε εάν συγκεκριμένοι κόμβοι είναι κρυμμένοι από την προβολή.

```java
import com.aspose.slides.ISmartArtNode;

try {
    // Ας υποθέσουμε ότι ο όρος «κόμβος» έχει ήδη οριστεί.
    boolean hidden = node.isHidden();

    if (hidden) {
        System.out.println("The node is currently hidden.");
    }
} finally {
    if (presentation != null) presentation.dispose();
}
```

**Εξήγηση**: 
- `isHidden()` Επιστρέφει μια λογική τιμή που υποδεικνύει την κατάσταση ορατότητας ενός κόμβου SmartArt.

### Αποθήκευση παρουσίασης σε αρχείο
#### Επισκόπηση
Αποθηκεύστε την βελτιωμένη παρουσίασή σας σε μορφή PPTX για κοινή χρήση ή περαιτέρω επεξεργασία.

**Βήμα 4**: Ορισμός διαδρομής εξόδου και αποθήκευση
- **Σκοπός**: Διατήρηση των αλλαγών αποθηκεύοντας το τροποποιημένο αρχείο παρουσίασης.

```java
import com.aspose.slides.Presentation;
import com.aspose.slides.SaveFormat;

try {
    String dataDir = "YOUR_DOCUMENT_DIRECTORY"; 
    // Αντικαταστήστε την με την πραγματική διαδρομή καταλόγου σας.
    
    presentation.save(dataDir + "CheckSmartArtHiddenProperty_out.pptx", SaveFormat.Pptx);
} finally {
    if (presentation != null) presentation.dispose();
}
```

**Εξήγηση**: 
- `save(String path, int format)` γράφει την παρουσίαση σε ένα καθορισμένο αρχείο στην επιθυμητή μορφή.

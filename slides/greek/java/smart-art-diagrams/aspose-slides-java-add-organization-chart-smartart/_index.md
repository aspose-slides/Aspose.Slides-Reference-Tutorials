## Οδηγός Εφαρμογής
### Προσθήκη SmartArt σε μια παρουσίαση
**Επισκόπηση**
Αυτή η ενότητα δείχνει πώς να εισαγάγετε ένα SmartArt τύπου OrganizationChart στην πρώτη διαφάνεια της παρουσίασής σας.

**Βήμα 1: Δημιουργία νέας παρουσίας παρουσίασης**
```java
Presentation presentation = new Presentation();
```
- **Γιατί:** Αυτό αρχικοποιεί ένα νέο αντικείμενο παρουσίασης το οποίο θα τροποποιήσουμε προσθέτοντας σχήματα και περιεχόμενο.

**Βήμα 2: Πρόσβαση στην πρώτη διαφάνεια**
```java
ISlide slide = presentation.getSlides().get_Item(0);
```
- **Γιατί:** Η πρώτη διαφάνεια είναι συνήθως το σημείο από το οποίο ξεκινάτε με το κύριο περιεχόμενό σας, συμπεριλαμβανομένων των γραφικών SmartArt.

**Βήμα 3: Προσθήκη γραφικού SmartArt σε οργανόγραμμα**
```java
ISmartArt smart = slide.getShapes().addSmartArt(10, 10, 400, 300, SmartArtLayoutType.OrganizationChart);
```
- **Γιατί:** Αυτή η κλήση μεθόδου προσθέτει ένα νέο γραφικό SmartArt στη διαφάνεια με καθορισμένες διαστάσεις και τύπο διάταξης. Οι παράμετροι (x, y, πλάτος, ύψος) καθορίζουν τη θέση και το μέγεθός της.

### Ορισμός τύπου διάταξης οργανογράμματος
**Επισκόπηση**
Εδώ, θα μάθετε πώς να τροποποιήσετε τη διάταξη ενός υπάρχοντος οργανογράμματος στο γραφικό SmartArt σας.

**Βήμα 4: Τροποποίηση της διάταξης του πρώτου κόμβου**
```java
smart.getNodes().get_Item(0).setOrganizationChartLayout(OrganizationChartLayoutType.LeftHanging);
```
- **Γιατί:** Αυτό το βήμα προσαρμόζει τη διάταξη, προσφέροντας μια πιο προσαρμοσμένη οπτική αναπαράσταση για τα ιεραρχικά δεδομένα. 

### Αποθήκευση παρουσίασης σε αρχείο
**Επισκόπηση**
Σε αυτήν την τελευταία λειτουργία, θα αποθηκεύσετε την παρουσίασή σας με το πρόσθετο γραφικό SmartArt.

**Βήμα 5: Αποθηκεύστε την εργασία σας**
```java
String outputDir = "YOUR_OUTPUT_DIRECTORY";
presentation.save(outputDir + "OrganizeChartLayoutType_out.pptx", SaveFormat.Pptx);
```
- **Γιατί:** Αυτό διασφαλίζει ότι όλες οι αλλαγές αποθηκεύονται σε ένα αρχείο, το οποίο μπορεί να κοινοποιηθεί ή να παρουσιαστεί.

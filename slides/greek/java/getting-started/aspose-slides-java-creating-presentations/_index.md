## Ρύθμιση του Aspose.Slides για Java
Για να ξεκινήσετε να χρησιμοποιείτε το Aspose.Slides, πρέπει πρώτα να το συμπεριλάβετε στο έργο σας:

**Maven**
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

**Γκράντλ**
```gradle
implementation group: 'com.aspose', name: 'aspose-slides', version: '25.4', classifier: 'jdk16'
```

## Οδηγός Εφαρμογής

### Λειτουργία 1: Δημιουργία νέας παρουσίασης

**Επισκόπηση:** Αυτή η ενότητα σας καθοδηγεί στη δημιουργία μιας νέας παρουσίασης PowerPoint χρησιμοποιώντας το Aspose.Slides. Θα μάθετε πώς να προσθέσετε μια διαφάνεια και μια αυτόματη διαμόρφωση γραμμής τύπου.

#### Οδηγίες βήμα προς βήμα

**1. Δημιουργήστε ένα αντικείμενο παρουσίασης**
Ξεκινήστε δημιουργώντας μια παρουσία του `Presentation` κλάση, η οποία αντιπροσωπεύει το αρχείο PowerPoint σας.

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Πλαίσιο κράτησης θέσης για τη διαδρομή καταλόγου εγγράφου
Presentation presentation = new Presentation();
```

**2. Πρόσβαση και τροποποίηση διαφανειών**
Ανακτήστε την προεπιλεγμένη διαφάνεια που δημιουργήθηκε κατά τη δημιουργία και προσθέστε ένα σχήμα γραμμής.

```java
try {
    ISlide slide = presentation.getSlides().get_Item(0); // Πρόσβαση στην πρώτη διαφάνεια
    slide.getShapes().addAutoShape(ShapeType.Line, 50, 150, 300, 0); // Προσθήκη σχήματος γραμμής στη διαφάνεια
```

**3. Αποθήκευση της παρουσίασης**
Τέλος, αποθηκεύστε την παρουσίασή σας σε μορφή PPTX.

```java
presentation.save(dataDir + "NewPresentation_out.pptx", SaveFormat.Pptx); // Αποθήκευση της παρουσίασης
} finally {
    if (presentation != null) presentation.dispose();
}
```

### Χαρακτηριστικό 2: Αυτόματη Χειραγώγηση Σχήματος

**Επισκόπηση:** Αυτή η ενότητα καλύπτει την προσθήκη διαφόρων αυτόματων σχημάτων σε μια διαφάνεια, επιδεικνύοντας την ευελιξία του Aspose.Slides στην προσαρμογή των παρουσιάσεών σας.

#### Οδηγίες βήμα προς βήμα

**1. Δημιουργία και πρόσβαση σε παρουσίαση**
Όπως και στην πρώτη λειτουργία, ξεκινήστε ρυθμίζοντας το αντικείμενο παρουσίασής σας.

```java
Presentation presentation = new Presentation();
```

**2. Προσθέστε διαφορετικά αυτόματα σχήματα**
Προσθέστε ένα ορθογώνιο και μια έλλειψη για να δείξετε την ευελιξία του σχήματος.

```java
try {
    ISlide slide = presentation.getSlides().get_Item(0); // Πρόσβαση στην πρώτη διαφάνεια

    // Προσθήκη ορθογωνίου
    slide.getShapes().addAutoShape(ShapeType.Rectangle, 100, 100, 200, 100);

    // Προσθήκη έλλειψης
    slide.getShapes().addAutoShape(ShapeType.Ellipse, 350, 150, 150, 75);
```

**3. Αποθήκευση της παρουσίασης**
Βεβαιωθείτε ότι έχετε αποθηκεύσει τις αλλαγές σας σε ένα αρχείο.

```java
presentation.save(dataDir + "AutoshapesExample_out.pptx", SaveFormat.Pptx); // Αποθήκευση της τροποποιημένης παρουσίασης
} finally {
    if (presentation != null) presentation.dispose();
}
```

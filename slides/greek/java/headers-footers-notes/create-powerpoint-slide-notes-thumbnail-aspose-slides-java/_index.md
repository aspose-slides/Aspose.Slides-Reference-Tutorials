## Ρύθμιση του Aspose.Slides για Java
Για να ξεκινήσετε, πρέπει να ενσωματώσετε το Aspose.Slides στο έργο Java σας. Δείτε πώς μπορείτε να το κάνετε χρησιμοποιώντας το Maven ή το Gradle:

### Ρύθμιση Maven
Προσθέστε την ακόλουθη εξάρτηση στο `pom.xml` αρχείο:

```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

### Ρύθμιση Gradle
Συμπεριλάβετε αυτό στο δικό σας `build.gradle` αρχείο:

```gradle
implementation group: 'com.aspose', name: 'aspose-slides', version: '25.4', classifier: 'jdk16'
```

## Οδηγός Εφαρμογής
Τώρα που έχετε ρυθμίσει το Aspose.Slides για Java, ας δούμε πώς να δημιουργείτε μια μικρογραφία από σημειώσεις διαφανειών.

### Δημιουργία μικρογραφιών από σημειώσεις διαφανειών
Αυτή η λειτουργία δείχνει πώς να δημιουργήσετε μια εικόνα των σημειώσεων που σχετίζονται με μια διαφάνεια σε μια παρουσίαση PowerPoint.

#### Βήμα 1: Ορισμός διαδρομών και φόρτωση παρουσίασης
Ξεκινήστε ορίζοντας τους καταλόγους εγγράφων και εξόδου. Στη συνέχεια, φορτώστε το αρχείο παρουσίασής σας:

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY/ThumbnailFromSlideInNotes.pptx";
String outputDir = "YOUR_OUTPUT_DIRECTORY/";

// Δημιουργήστε μια κλάση Presentation που αντιπροσωπεύει το αρχείο παρουσίασης.
Presentation pres = new Presentation(dataDir);
```

#### Βήμα 2: Πρόσβαση στη διαφάνεια και ορισμός διαστάσεων μικρογραφίας
Αποκτήστε πρόσβαση στην επιθυμητή διαφάνεια και καθορίστε τις διαστάσεις για τη μικρογραφία σας:

```java
ISlide sld = pres.getSlides().get_Item(0);

int desiredX = 1200;
int desiredY = 800;

// Υπολογίστε κλιμακωτές τιμές με βάση το μέγεθος της διαφάνειας.
float ScaleX = (float) (1.0 / pres.getSlideSize().getSize().getWidth()) * desiredX;
float ScaleY = (float) (1.0 / pres.getSlideSize().getSize().getHeight()) * desiredY;
```

#### Βήμα 3: Δημιουργία και αποθήκευση μικρογραφίας
Δημιουργήστε μια μικρογραφία των σημειώσεων της διαφάνειας χρησιμοποιώντας την καθορισμένη κλίμακα και, στη συνέχεια, αποθηκεύστε την:

```java
IImage img = sld.getImage(ScaleX, ScaleY);
img.save(outputDir + "Notes_tnail_out.jpg");
```

#### Βήμα 4: Καθαρισμός πόρων
Τέλος, βεβαιωθείτε ότι έχετε διαθέσει πόρους για να αποτρέψετε διαρροές μνήμης:

```java
if (pres != null) pres.dispose();
```

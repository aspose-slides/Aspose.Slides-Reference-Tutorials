# Εξοικείωση με το Aspose.Slides Java: Σύνδεση σχημάτων στο PowerPoint

**Εισαγωγή**

Στον κόσμο των επαγγελματικών παρουσιάσεων, η αποτελεσματική σύνδεση σχημάτων μπορεί να μετατρέψει τις διαφάνειές σας από καλές σε εξαιρετικές. ## Οδηγός Εφαρμογής

Σε αυτήν την ενότητα, θα αναλύσουμε τον τρόπο σύνδεσης σχημάτων χρησιμοποιώντας συνδέσμους στο PowerPoint με το Aspose.Slides Java.

### Προσθήκη σχημάτων
Αρχικά, ας προσθέσουμε δύο βασικά σχήματα: μια έλλειψη και ένα ορθογώνιο. Θα τα τοποθετήσουμε στην πρώτη διαφάνεια της παρουσίασής μας.
```java
// Δημιουργία αρχικού στιγμιότυπου της κλάσης παρουσίασης που αντιπροσωπεύει το αρχείο PPTX
Presentation input = new Presentation();
try {
    // Πρόσβαση στη συλλογή σχημάτων για την επιλεγμένη διαφάνεια (πρώτη διαφάνεια)
    IShapeCollection shapes = input.getSlides().get_Item(0).getShapes();

    // Προσθήκη έλλειψης αυτόματου σχήματος στη θέση (0, 100) με μέγεθος (100x100)
    IAutoShape ellipse = shapes.addAutoShape(ShapeType.Ellipse, 0, 100, 100, 100);

    // Προσθήκη ορθογωνίου αυτόματης διαμόρφωσης στη θέση (100, 300) με μέγεθος (100x100)
    IAutoShape rectangle = shapes.addAutoShape(ShapeType.Rectangle, 100, 300, 100, 100);
```

### Σύνδεση σχημάτων
Τώρα που τα σχήματά μας είναι στη θέση τους, ας τα συνδέσουμε χρησιμοποιώντας έναν σύνδεσμο. Θα χρησιμοποιήσουμε έναν λυγισμένο σύνδεσμο για να συνδέσουμε την έλλειψη και το ορθογώνιο.
```java
    // Προσθήκη σχήματος σύνδεσης στη συλλογή σχημάτων διαφανειών ξεκινώντας από (0, 0) με μέγεθος (10x10)
    IConnector connector = shapes.addConnector(ShapeType.BentConnector2, 0, 0, 10, 10);

    // Σύνδεση της έλλειψης στην αρχή της γραμμής σύνδεσης
    connector.setStartShapeConnectedTo(ellipse);

    // Σύνδεση ορθογωνίου στο άκρο της γραμμής σύνδεσης
    connector.setEndShapeConnectedTo(rectangle);
```

### Αναδρομολόγηση της σύνδεσης
Μόλις συνδεθεί, αναδρομολογήστε τη γραμμή σύνδεσης για να βεβαιωθείτε ότι βρίσκει τη συντομότερη διαδρομή μεταξύ των σχημάτων.
```java
    // Αναδρομολόγηση σύνδεσης για αυτόματη εύρεση της συντομότερης διαδρομής μεταξύ σχημάτων
    connector.reroute();
```

### Αποθήκευση της παρουσίασης
Τέλος, αποθηκεύστε την παρουσίασή σας σε μορφή PPTX με ένα συγκεκριμένο όνομα.
```java
    // Αποθηκεύστε την παρουσίαση σε μορφή PPTX με ένα συγκεκριμένο όνομα
    input.save("Connecting_shapes_using_connectors_out.pptx", SaveFormat.Pptx);
} finally {
    if (input != null) input.dispose();
}
```

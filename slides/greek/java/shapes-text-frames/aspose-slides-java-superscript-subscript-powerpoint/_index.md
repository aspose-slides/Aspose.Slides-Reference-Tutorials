## Ρύθμιση του Aspose.Slides για Java

### Πληροφορίες εγκατάστασης

Για να χρησιμοποιήσετε το Aspose.Slides για Java στο έργο σας, προσθέστε το μέσω του Maven ή του Gradle. Εναλλακτικά, κατεβάστε το αρχείο JAR απευθείας από τον ιστότοπο του Aspose.

**Maven:**
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

**Βαθμός:**
```gradle
implementation group: 'com.aspose', name: 'aspose-slides', version: '25.4', classifier: 'jdk16'
```

**Άμεση λήψη:**
Κατεβάστε την τελευταία έκδοση από [Aspose.Slides για εκδόσεις Java](https://releases.aspose.com/slides/java/).

## Οδηγός Εφαρμογής

Ας αναλύσουμε την υλοποίηση σε δύο βασικά χαρακτηριστικά: την προσθήκη κειμένου εκθέτη και υπογεγραμμένου δείκτη.

### Προσθήκη κειμένου εκθέτη

Το κείμενο εκθέτη χρησιμοποιείται συχνά για επιστημονικούς τύπους ή σημειογραφίες. Αυτή η ενότητα σάς δείχνει πώς να το δημιουργήσετε στο PowerPoint χρησιμοποιώντας το Aspose.Slides για Java.

#### Επισκόπηση
Θα προσθέσουμε μια σημειογραφία εκθέτη "TM" δίπλα στον τίτλο μιας διαφάνειας, προσομοιώνοντας ένα σύμβολο εμπορικού σήματος.

#### Βήματα Υλοποίησης

1. **Αρχικοποίηση παρουσίασης:**
   ```java
   Presentation presentation = new Presentation();
   ```

2. **Πρόσβαση στην πρώτη διαφάνεια:**
   ```java
   ISlide slide = presentation.getSlides().get_Item(0);
   ```

3. **Προσθήκη Αυτόματου Σχήματος για Πλαίσιο Κειμένου:**
   ```java
   IAutoShape shape = slide.getShapes().addAutoShape(ShapeType.Rectangle, 100, 100, 200, 100);
   ITextFrame textFrame = shape.getTextFrame();
   textFrame.getParagraphs().clear(); // Διαγραφή υπάρχοντος κειμένου
   ```

4. **Δημιουργία παραγράφου εκθέτη:**
   ```java
   IParagraph superPar = new Paragraph();

   // Τμήμα κανονικού κειμένου
   IPortion portion1 = new Portion();
   portion1.setText("SlideTitle");
   superPar.getPortions().add(portion1);

   // Τμήμα κειμένου εκθέτη
   IPortion superPortion = new Portion();
   superPortion.getPortionFormat().setEscapement(30); // Θετική τιμή για εκθέτη
   superPortion.setText("TM");
   superPar.getPortions().add(superPortion);
   ```

5. **Προσθήκη παραγράφου σε πλαίσιο κειμένου:**
   ```java
   textFrame.getParagraphs().add(superPar);
   ```

6. **Αποθήκευση παρουσίασης:**
   ```java
   presentation.save("YOUR_OUTPUT_DIRECTORY/TestOut_Super.pptx", SaveFormat.Pptx);
   ```

#### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι η τιμή διαφυγής είναι θετική για τον εκθέτη.
- Επαληθεύστε την ευθυγράμμιση και τη θέση του κειμένου, εάν εμφανίζεται λανθασμένη.

### Προσθήκη κειμένου υπογραφής

Οι δείκτες χρησιμοποιούνται συνήθως σε χημικούς τύπους ή μαθηματικές εκφράσεις. Δείτε πώς μπορείτε να τους προσθέσετε:

#### Επισκόπηση
Θα δημιουργήσουμε ένα δείκτη "i" δίπλα σε ένα "a", προσομοιώνοντας το πεζό i του λατινικού αλφαβήτου.

#### Βήματα Υλοποίησης

1. **Αρχικοποίηση παρουσίασης:**
   ```java
   Presentation presentation = new Presentation();
   ```

2. **Πρόσβαση στην πρώτη διαφάνεια:**
   ```java
   ISlide slide = presentation.getSlides().get_Item(0);
   ```

3. **Προσθήκη Αυτόματου Σχήματος για Πλαίσιο Κειμένου:**
   ```java
   IAutoShape shape = slide.getShapes().addAutoShape(ShapeType.Rectangle, 100, 250, 200, 100); // Ρυθμίστε τη θέση Y για να αποφύγετε την επικάλυψη
   ITextFrame textFrame = shape.getTextFrame();
   textFrame.getParagraphs().clear(); // Διαγραφή υπάρχοντος κειμένου
   ```

4. **Δημιουργία παραγράφου με δείκτη:**
   ```java
   IParagraph subPar = new Paragraph();

   // Τμήμα κανονικού κειμένου
   IPortion portion2 = new Portion();
   portion2.setText("a");
   subPar.getPortions().add(portion2);

   // Τμήμα κειμένου υπογραφής
   IPortion subPortion = new Portion();
   subPortion.getPortionFormat().setEscapement(-25); // Αρνητική τιμή για δείκτη
   subPortion.setText("i");
   subPar.getPortions().add(subPortion);
   ```

5. **Προσθήκη παραγράφου σε πλαίσιο κειμένου:**
   ```java
   textFrame.getParagraphs().add(subPar);
   ```

6. **Αποθήκευση παρουσίασης:**
   ```java
   presentation.save("YOUR_OUTPUT_DIRECTORY/TestOut_Sub.pptx", SaveFormat.Pptx);
   ```

#### Συμβουλές αντιμετώπισης προβλημάτων
- Χρησιμοποιήστε αρνητικές τιμές διαφυγής για τον δείκτη.
- Προσαρμόστε το μέγεθος του πλαισίου κειμένου εάν το περιεχόμενο δεν ταιριάζει καλά.

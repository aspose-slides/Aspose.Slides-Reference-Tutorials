## Οδηγός Εφαρμογής
Τώρα, ας εφαρμόσουμε τη λειτουργία για την αφαίρεση ενός τμήματος από ένα γεωμετρικό σχήμα.
### Δημιουργήστε και τροποποιήστε ένα σχήμα καρδιάς
Θα ξεκινήσουμε δημιουργώντας ένα αντικείμενο σε σχήμα καρδιάς στο PowerPoint χρησιμοποιώντας το Aspose.Slides για Java. Αυτή η ενότητα εξηγεί πώς να αποκτήσετε πρόσβαση και να τροποποιήσετε τη γεωμετρική του διαδρομή.
#### Προσθήκη γεωμετρικού σχήματος
Αρχικά, προσθέστε ένα νέο γεωμετρικό σχήμα στην παρουσίασή σας:
```java
// Αρχικοποίηση κλάσης παρουσίασης
Presentation pres = new Presentation();
try {
    // Δημιουργήστε ένα σχήμα καρδιάς στην πρώτη διαφάνεια στη θέση (100, 100) με μέγεθος (300, 300)
    com.aspose.slides.ShapeType shapeType = com.aspose.slides.ShapeType.Heart;
    GeometryShape shape = (GeometryShape) pres.getSlides().get_Item(0).getShapes()
            .addAutoShape(shapeType, 100, 100, 300, 300);
```
#### Πρόσβαση στη διαδρομή γεωμετρίας
Στη συνέχεια, αποκτήστε πρόσβαση στη γεωμετρική διαδρομή του νεοδημιουργημένου σχήματος:
```java
// Αποκτήστε πρόσβαση στην πρώτη γεωμετρική διαδρομή του σχήματος καρδιάς
IGeometryPath path = shape.getGeometryPaths()[0];
```
#### Αφαίρεση τμήματος από τη διαδρομή
Για να αφαιρέσετε ένα τμήμα (για παράδειγμα, το τρίτο):
```java
// Αφαίρεση του τρίτου τμήματος (δείκτης 2) από τη γεωμετρική διαδρομή
path.removeAt(2);
```
#### Ενημέρωση και αποθήκευση της παρουσίασής σας
Τέλος, ενημερώστε το σχήμα σας με την τροποποιημένη διαδρομή και αποθηκεύστε την παρουσίαση:
```java
// Ενημέρωση του σχήματος με την τροποποιημένη γεωμετρική διαδρομή
shape.setGeometryPath(path);

// Ορίστε τη διαδρομή του αρχείου εξόδου και αποθηκεύστε την παρουσίαση σε μορφή PPTX
String resultPath = "YOUR_OUTPUT_DIRECTORY" +  "/GeometryShapeRemoveSegment.pptx";
pres.save(resultPath, SaveFormat.Pptx);
```

## Οδηγός Εφαρμογής
Θα δούμε τώρα τα βήματα που απαιτούνται για την εφαρμογή γεμίσματος με διαβάθμιση σε ένα σχήμα έλλειψης σε μια παρουσίαση PowerPoint χρησιμοποιώντας το Aspose.Slides για Java.

### Προσθήκη σχήματος έλλειψης με γέμισμα με διαβάθμιση
#### Βήμα 1: Δημιουργία και διαμόρφωση παρουσίασης
Αρχικά, αρχικοποιήστε ένα νέο `Presentation` αντικείμενο που αντιπροσωπεύει το αρχείο PowerPoint σας. Εδώ θα προσθέσετε τα σχήματά σας και θα εφαρμόσετε μορφοποίηση.

```java
import com.aspose.slides.*;

public class FillShapesGradient {
    public static void main(String[] args) {
        String dataDir = "YOUR_DOCUMENT_DIRECTORY";
        
        Presentation pres = new Presentation();
        try {
            ISlide sld = pres.getSlides().get_Item(0);
```

#### Βήμα 2: Προσθήκη σχήματος έλλειψης
Προσθέστε ένα σχήμα έλλειψης στη διαφάνεια. Μπορείτε να καθορίσετε τις διαστάσεις και τη θέση της στη διαφάνεια.

```java
            // Προσθέστε ένα σχήμα έλλειψης με καθορισμένες διαστάσεις και θέση.
            IShape shp = sld.getShapes().addAutoShape(ShapeType.Ellipse, 50, 150, 75, 150);
```

#### Βήμα 3: Εφαρμογή γεμίσματος με διαβάθμιση
Διαμορφώστε τις ρυθμίσεις γεμίσματος με διαβάθμιση για το σχήμα σας. Μπορείτε να επιλέξετε ανάμεσα σε διαφορετικά σχήματα και κατευθύνσεις διαβάθμισης.

```java
            // Ορισμός τύπου γεμίσματος με διαβάθμιση.
            shp.getFillFormat().setFillType(FillType.Gradient);

            // Επιλέξτε ένα γραμμικό σχήμα διαβάθμισης.
            shp.getFillFormat().getGradientFormat().setGradientShape(GradientShape.Linear);

            // Ορίστε την κατεύθυνση της κλίσης.
            shp.getFillFormat().getGradientFormat().setGradientDirection(GradientDirection.FromCorner2);
```

#### Βήμα 4: Προσαρμόστε τα χρώματα διαβάθμισης
Ορίστε τα χρώματα και τις θέσεις για τα stops διαβάθμισης. Αυτό ελέγχει τον τρόπο με τον οποίο η διαβάθμιση μεταβαίνει μεταξύ των χρωμάτων.

```java
            // Προσθέστε χρωματικά stop για να ορίσετε τη μετάβαση με διαβάθμιση.
            shp.getFillFormat().getGradientFormat().getGradientStops().add((float) 1.0, new Color(PresetColor.Purple));
            shp.getFillFormat().getGradientFormat().getGradientStops().add((float) 0, Color.RED);
```

#### Βήμα 5: Αποθήκευση της παρουσίασης
Τέλος, αποθηκεύστε την παρουσίασή σας σε ένα αρχείο με εφαρμοσμένο το σχήμα με διαβάθμιση.

```java
            // Αποθηκεύστε την παρουσίαση με την ενημερωμένη διαφάνεια.
            pres.save(dataDir + "EllipseShpGrad_out.pptx", SaveFormat.Pptx);
        } finally {
            if (pres != null) pres.dispose();
        }
    }
}
```

**Συμβουλές αντιμετώπισης προβλημάτων:**

- Επαληθεύστε ότι η έκδοση JDK ταιριάζει με τον ταξινομητή Aspose.Slides που χρησιμοποιείται στις εξαρτήσεις.

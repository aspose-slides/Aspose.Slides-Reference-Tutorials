## Ρύθμιση του Aspose.Slides για Java

Για να ξεκινήσετε, θα χρειαστεί να ενσωματώσετε τη βιβλιοθήκη Aspose.Slides στο έργο σας. Δείτε πώς:

### Ρύθμιση Maven
Προσθέστε την ακόλουθη εξάρτηση στο `pom.xml` αρχείο:
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

### Ρύθμιση Gradle
Συμπεριλάβετε αυτήν τη γραμμή στο δικό σας `build.gradle` αρχείο:
```gradle
implementation group: 'com.aspose', name: 'aspose-slides', version: '25.4', classifier: 'jdk16'
```

### Άμεση Λήψη
Εναλλακτικά, μπορείτε να κατεβάσετε την τελευταία έκδοση απευθείας από το [Aspose.Slides για εκδόσεις Java](https://releases.aspose.com/slides/java/).

#### Απόκτηση Άδειας
- **Δωρεάν δοκιμή**Αποκτήστε μια δωρεάν δοκιμαστική έκδοση για να δοκιμάσετε τις δυνατότητες της βιβλιοθήκης.
- **Προσωρινή Άδεια**Ζητήστε προσωρινή άδεια εάν χρειάζεται για εκτεταμένες δοκιμές.
- **Αγορά**Για πλήρη λειτουργικότητα, σκεφτείτε να αγοράσετε μια άδεια χρήσης.

### Βασική Αρχικοποίηση και Ρύθμιση
Μόλις ρυθμίσετε τη βιβλιοθήκη, αρχικοποιήστε την στην εφαρμογή Java ως εξής:

import com.aspose.slides.Presentation;

## Οδηγός Εφαρμογής

Ας εμβαθύνουμε στην εφαρμογή τρισδιάστατης περιστροφής σε ένα ορθογώνιο σχήμα στο PowerPoint χρησιμοποιώντας το Aspose.Slides για Java. Θα το αναλύσουμε σε διαχειρίσιμα βήματα.

### Δημιουργία παρουσίασης και προσθήκη σχήματος

#### Επισκόπηση
Αρχικά, δημιουργούμε μια νέα παρουσίαση και προσθέτουμε ένα ορθογώνιο σχήμα στην πρώτη διαφάνεια.
```java
// Δημιουργήστε μια παρουσία της κλάσης Presentation
Presentation pres = new Presentation();

// Προσθήκη ενός αυτόματου σχήματος ορθογωνίου στην πρώτη διαφάνεια
IShape autoShape = pres.getSlides().get_Item(0).getShapes().addAutoShape(
    ShapeType.Rectangle, 30, 30, 200, 200);
```
**Εξήγηση**: 
- `Presentation` αρχικοποιείται για τη δημιουργία μιας νέας παρουσίασης.
- Προσθέτουμε ένα AutoShape τύπου Rectangle στη θέση (30, 30) με διαστάσεις 200x200.

### Εφαρμογή περιστροφής 3D

#### Επισκόπηση
Στη συνέχεια, ρυθμίζουμε τα τρισδιάστατα εφέ στο ορθογώνιο σχήμα μας.
```java
// Ορίστε το βάθος του εφέ 3D
autoShape.getThreeDFormat().setDepth((short) 6);

// Ρύθμιση παραμέτρων περιστροφής και πληκτρολόγησης κάμερας για τρισδιάστατη προοπτική
autoShape.getThreeDFormat().getCamera().setRotation(40, 35, 20);
autoShape.getThreeDFormat().getCamera().setCameraType(CameraPresetType.IsometricLeftUp);

// Ορίστε τον τύπο φωτιστικού για ισορροπημένο φωτισμό
autoShape.getThreeDFormat().getLightRig().setLightType(LightRigPresetType.Balanced);
```
**Εξήγηση**: 
- `setDepth` Ρυθμίζει το βάθος εμφάνισης του εφέ 3D.
- Η περιστροφή και ο τύπος της κάμερας έχουν ρυθμιστεί ώστε να δημιουργούν μια συγκεκριμένη προοπτική.
- Για ομοιόμορφο φωτισμό, εφαρμόζεται ισορροπημένο σύστημα φωτισμού.

### Αποθήκευση της παρουσίασης

Τέλος, αποθηκεύστε την παρουσίασή σας με αυτά τα εφέ εφαρμοσμένα:
```java
// Αποθήκευση της παρουσίασης με εφαρμοσμένα εφέ 3D σε ένα αρχείο
pres.save("YOUR_OUTPUT_DIRECTORY\\Rotation_out.pptx", com.aspose.slides.SaveFormat.Pptx);
```
**Εξήγηση**: 
- Ο `save` Η μέθοδος εξάγει την τροποποιημένη παρουσίαση στην καθορισμένη διαδρομή.

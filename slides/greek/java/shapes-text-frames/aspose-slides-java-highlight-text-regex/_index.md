## Ρύθμιση του Aspose.Slides για Java

Για να χρησιμοποιήσετε το Aspose.Slides στο έργο Java σας, θα πρέπει να το συμπεριλάβετε ως εξάρτηση. Ακολουθούν οι τρόποι με τους οποίους μπορείτε να ρυθμίσετε το Aspose.Slides χρησιμοποιώντας διαφορετικά εργαλεία δημιουργίας:

### Maven
Προσθέστε αυτήν την εξάρτηση στο δικό σας `pom.xml` αρχείο:
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

### Γκράντλ
Συμπεριλάβετέ το στο δικό σας `build.gradle`:
```gradle

```

## Οδηγός Εφαρμογής

Αυτή η ενότητα αναλύει κάθε λειτουργία σε διαχειρίσιμα βήματα. Θα καλύψουμε τη φόρτωση παρουσιάσεων, τη μεταγλώττιση μοτίβων regex, την επισήμανση κειμένου και την αποθήκευση αρχείων.

### Φόρτωση παρουσίασης

**Επισκόπηση**Αυτή η λειτουργία σάς επιτρέπει να φορτώσετε ένα αρχείο PowerPoint από τον καθορισμένο κατάλογο χρησιμοποιώντας το Aspose.Slides Java.

1. **Εισαγωγή της Κλάσης Παρουσίασης**
   ```java
   import com.aspose.slides.Presentation;
   ```

2. **Καθορισμός καταλόγου εγγράφων και φόρτωση αρχείου**
   Αντικαθιστώ `"YOUR_DOCUMENT_DIRECTORY"` με την πραγματική διαδρομή όπου είναι αποθηκευμένη η παρουσίασή σας.
   ```java
   String dataDir = "YOUR_DOCUMENT_DIRECTORY";
   Presentation presentation = new Presentation(dataDir + "/SomePresentation.pptx");
   ```
   *Αυτό αρχικοποιεί ένα `Presentation` αντικείμενο, φορτώνοντας το αρχείο PowerPoint από την καθορισμένη θέση.*

### Μεταγλώττιση κανονικής έκφρασης

**Επισκόπηση**: Αυτή η λειτουργία δείχνει πώς να μεταγλωττίσετε ένα μοτίβο regex σε Java για να ταιριάζει με συγκεκριμένα μοτίβα κειμένου.

1. **Εισαγωγή κλάσης μοτίβου**
   ```java
   import java.util.regex.Pattern;
   ```

2. **Μεταγλώττιση Regex για λέξεις που ταιριάζουν με 10 ή περισσότερους χαρακτήρες**
   ```java
   Pattern regex = Pattern.compile("\\b[^\\s]{5,}\\b");
   ```
   *Το regex `\\b[^\\s]{5,}\\b` αντιστοιχίζει λέξεις που έχουν μήκος τουλάχιστον πέντε χαρακτήρες.*

### Επισήμανση κειμένου χρησιμοποιώντας Regex

**Επισκόπηση**Εδώ, θα μάθετε πώς να επισημαίνετε κείμενο σε μια παρουσίαση με βάση το μεταγλωττισμένο regex.

1. **Πρόσβαση και προετοιμασία σχήματος για τροποποίηση**
   ```java
   import com.aspose.slides.AutoShape;
   import java.awt.Color;

   AutoShape shape = (AutoShape) presentation.getSlides().get_Item(0).getShapes().get_Item(0);
   ```

2. **Επισήμανση κειμένου που ταιριάζει με μπλε χρώμα**
   ```java
   shape.getTextFrame().highlightRegex(regex, Color.BLUE, null);
   ```
   *Αυτή η μέθοδος αναζητά αντιστοιχίσεις regex μέσα στο πλαίσιο κειμένου και τις επισημαίνει με μπλε χρώμα.*

### Αποθήκευση παρουσίασης

**Επισκόπηση**: Αυτή η λειτουργία σάς επιτρέπει να αποθηκεύσετε την τροποποιημένη παρουσίασή σας σε έναν καθορισμένο κατάλογο.

1. **Καθορισμός καταλόγου εξόδου**
   ```java
   String outputDir = "YOUR_OUTPUT_DIRECTORY";
   ```

2. **Αποθήκευση της τροποποιημένης παρουσίασης**
   ```java
   import com.aspose.slides.SaveFormat;

   presentation.save(outputDir + "/SomePresentation-out.pptx", SaveFormat.Pptx);
   ```
   *Αυτό αποθηκεύει τις αλλαγές σας σε ένα νέο αρχείο, διατηρώντας όλες τις τροποποιήσεις.*

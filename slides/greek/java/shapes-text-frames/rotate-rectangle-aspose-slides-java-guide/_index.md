# Περιστροφή ορθογωνίου σε μια παρουσίαση χρησιμοποιώντας το Aspose.Slides Java

## Εισαγωγή

Η περιστροφή σχημάτων σε παρουσιάσεις μπορεί να είναι δύσκολη χωρίς τα κατάλληλα εργαλεία. Με το Aspose.Slides για Java, η περιστροφή ορθογωνίων και άλλων σχημάτων γίνεται απλή και αποτελεσματική. Αυτό το σεμινάριο θα σας καθοδηγήσει στη χρήση του Aspose.Slides για την απρόσκοπτη περιστροφή σχημάτων.

### Τι θα μάθετε
- Πώς να ρυθμίσετε το Aspose.Slides για Java
- Προσθήκη ορθογωνίου σχήματος σε μια διαφάνεια
- Περιστροφή του ορθογωνίου κατά συγκεκριμένες γωνίες
- Αποθήκευση αλλαγών στην παρουσίασή σας

Μέχρι το τέλος αυτού του οδηγού, θα έχετε εξοικειωθεί με την εναλλαγή σχημάτων σε παρουσιάσεις χρησιμοποιώντας το Aspose.Slides.

## Ρύθμιση του Aspose.Slides για Java

Εγκαταστήστε τη βιβλιοθήκη Aspose.Slides χρησιμοποιώντας μία από τις ακόλουθες μεθόδους:

**Maven**
Προσθέστε αυτήν την εξάρτηση στο δικό σας `pom.xml` αρχείο:

```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

**Γκράντλ**
Συμπεριλάβετε τα ακόλουθα στο `build.gradle` αρχείο:

```gradle
implementation group: 'com.aspose', name: 'aspose-slides', version: '25.4', classifier: 'jdk16'
```

**Άμεση Λήψη**
Κατεβάστε τη βιβλιοθήκη απευθείας από [Aspose.Slides για εκδόσεις Java](https://releases.aspose.com/slides/java/).

## Οδηγός Εφαρμογής

Αυτή η ενότητα σας καθοδηγεί στη δημιουργία και την περιστροφή ενός ορθογωνίου σχήματος μέσα σε μια παρουσίαση.

### Δημιουργία και περιστροφή ορθογωνίου σχήματος

#### Επισκόπηση
Θα προσθέσουμε ένα AutoShape τύπου ορθογωνίου σε μια διαφάνεια και θα την περιστρέψουμε κατά 90 μοίρες χρησιμοποιώντας το Aspose.Slides για Java, ιδανικό για δυναμικές παρουσιάσεις.

#### Βήμα προς βήμα εφαρμογή
**1. Ρύθμιση αντικειμένου παρουσίασης**
Δημιουργήστε ένα `Presentation` αντικείμενο που αντιπροσωπεύει το αρχείο PPTX σας:

```java
Presentation pres = new Presentation();
```

**2. Πρόσβαση στην πρώτη διαφάνεια**
Αποκτήστε πρόσβαση στην πρώτη διαφάνεια για να προσθέσετε σχήματα:

```java
ISlide sld = pres.getSlides().get_Item(0);
```

**3. Προσθήκη ορθογωνίου σχήματος**
Προσθήκη ενός Αυτόματου Σχήματος τύπου ορθογωνίου με συγκεκριμένες διαστάσεις και θέση:

```java
IShape shp = sld.getShapes().addAutoShape(ShapeType.Rectangle, 50, 150, 75, 150);
```
- `ShapeType.Rectangle`: Καθορίζει τον τύπο σχήματος.
- Συντεταγμένες `(50, 150)`Θέσεις X και Y στη διαφάνεια.
- Διαστάσεις `(75, 150)`: Πλάτος και ύψος του ορθογωνίου.

**4. Περιστρέψτε το σχήμα**
Περιστρέψτε το ορθογώνιό σας ορίζοντας την ιδιότητα περιστροφής του:

```java
shp.setRotation(90);
```
Αυτό περιστρέφει το σχήμα κατά 90 μοίρες δεξιόστροφα.

**5. Αποθήκευση της παρουσίασης**
Αποθηκεύστε την παρουσίαση με το περιστρεφόμενο ορθογώνιο:

```java
pres.save(dataDir + "/RectShpRot_out.pptx", SaveFormat.Pptx);
```

### Συμβουλές αντιμετώπισης προβλημάτων
- **Εξασφαλίστε τη σωστή διαδρομή**: Επαλήθευση `dataDir` δείχνει σε έναν υπάρχοντα κατάλογο.
- **Ελέγξτε τον τύπο σχήματος**: Επιβεβαιώστε ότι χρησιμοποιείτε `ShapeType.Rectangle`.

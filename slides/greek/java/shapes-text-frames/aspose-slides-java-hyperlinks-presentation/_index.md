Πριν ξεκινήσουμε, βεβαιωθείτε ότι έχετε τα εξής:

- Βασική κατανόηση προγραμματισμού Java και εργαλείων δημιουργίας Maven/Gradle.
- Ένα ολοκληρωμένο περιβάλλον ανάπτυξης (IDE) όπως το IntelliJ IDEA ή το Eclipse.


**Γκράντλ**


Εναλλακτικά, μπορείτε να κατεβάσετε την τελευταία έκδοση απευθείας από το [Aspose.Slides για εκδόσεις Java](https://releases.aspose.com/slides/java/).


### Προσθήκη και μορφοποίηση αυτόματου σχήματος με χρώμα υπερσύνδεσης

Στη συνέχεια, θα προσθέσουμε ένα αυτόματο σχήμα και θα το μορφοποιήσουμε με έναν έγχρωμο υπερσύνδεσμο:
```java
import com.aspose.slides.*;

// Δημιουργεί ένα νέο αντικείμενο παρουσίασης.
Presentation presentation = new Presentation();
try {
    // Προσθέτει ένα αυτόματο σχήμα τύπου ορθογωνίου στην πρώτη διαφάνεια.
    IAutoShape shape1 = presentation.getSlides().get_Item(0).getShapes()
        .addAutoShape(ShapeType.Rectangle, 100, 100, 450, 50, false);

    // Προσθέτει ένα πλαίσιο κειμένου με δείγμα κειμένου υπερσυνδέσμου.
    shape1.addTextFrame("This is a sample of colored hyperlink.");

    // Ορίζει τον υπερσύνδεσμο του πρώτου τμήματος σε μια καθορισμένη διεύθυνση URL.
    shape1.getTextFrame().getParagraphs().get_Item(0).getPortions()
        .get_Item(0)
        .getPortionFormat()
        .setHyperlinkClick(new Hyperlink("https://www.aspose.com/")).

    // Καθορίζει την πηγή του χρώματος υπερσυνδέσμου που θα προέρχεται από το PortionFormat.
    shape1.getTextFrame().getParagraphs().get_Item(0).getPortions()
        .get_Item(0)
        .getPortionFormat().getHyperlinkClick()
        .setColorSource(HyperlinkColorSource.PortionFormat);

    // Ορίζει τον τύπο γεμίσματος της υπερσύνδεσης σε συμπαγές και αλλάζει το χρώμα της σε κόκκινο.
    shape1.getTextFrame().getParagraphs().get_Item(0).getPortions()
        .get_Item(0)
        .getPortionFormat().getFillFormat()
        .setFillType(FillType.Solid);
    shape1.getTextFrame().getParagraphs().get_Item(0).getPortions()
        .get_Item(0)
        .getPortionFormat().getFillFormat().getSolidFillColor()
        .setColor(Color.RED);
} finally {
    if (presentation != null) presentation.dispose();
}
```

### Προσθήκη κανονικού υπερσυνδέσμου σε αυτόματο σχήμα

Για την προσθήκη ενός τυπικού υπερσυνδέσμου χωρίς ειδική μορφοποίηση:
```java
import com.aspose.slides.*;

// Δημιουργεί ένα νέο αντικείμενο παρουσίασης.
Presentation presentation = new Presentation();
try {
    // Προσθέτει ένα άλλο αυτόματο σχήμα τύπου ορθογωνίου στην πρώτη διαφάνεια.
    IAutoShape shape2 = presentation.getSlides().get_Item(0).getShapes()
        .addAutoShape(ShapeType.Rectangle, 100, 200, 450, 50, false);

    // Προσθέτει ένα πλαίσιο κειμένου με δείγμα κειμένου υπερσυνδέσμου χωρίς ειδική χρωματική μορφοποίηση.
    shape2.addTextFrame("This is a sample of usual hyperlink.");

    // Ορίζει τον υπερσύνδεσμο του πρώτου τμήματος σε μια καθορισμένη διεύθυνση URL.
    shape2.getTextFrame().getParagraphs().get_Item(0).getPortions()
        .get_Item(0)
        .getPortionFormat()
        .setHyperlinkClick(new Hyperlink("https://www.aspose.com/")).
} finally {
    if (presentation != null) presentation.dispose();
}
```

### Αποθήκευση της παρουσίασης σε αρχείο

Τέλος, ας αποθηκεύσουμε την εργασία μας:
```java

// Δημιουργεί ένα νέο αντικείμενο παρουσίασης.
Presentation presentation = new Presentation();
try {
    // Όλες οι προηγούμενες λειτουργίες προσθήκης σχημάτων και υπερσυνδέσμων θα βρίσκονταν εδώ.

    // Αποθηκεύει την παρουσίαση σε έναν καθορισμένο κατάλογο με ένα δεδομένο όνομα αρχείου.

    if (presentation != null) presentation.dispose();
}
```

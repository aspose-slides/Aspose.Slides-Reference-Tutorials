## Ρύθμιση του Aspose.Slides για Java
Μόλις προστεθεί η βιβλιοθήκη στο έργο σας, αρχικοποιήστε την μέσα στον κώδικά σας. Δείτε πώς μπορείτε να ξεκινήσετε:
1. **Εισαγάγετε τις απαραίτητες κλάσεις:**
   ```java
   import com.aspose.slides.Presentation;
   ```
2. **Αρχικοποίηση ενός αντικειμένου παρουσίασης:** Αυτό αντιπροσωπεύει ολόκληρη την παρουσίασή σας.
   ```java
   Presentation pres = new Presentation();
   try {
       // Ο κωδικός σας εδώ
   } finally {
       if (pres != null) pres.dispose();
   }
   ```
Αυτό το μοτίβο αρχικοποίησης διασφαλίζει ότι οι πόροι απελευθερώνονται όταν ολοκληρώσετε την παρουσίαση.
## Οδηγός Εφαρμογής
### Χαρακτηριστικό 1: Δημιουργία καταλόγου
**Επισκόπηση:** Βεβαιωθείτε ότι ο κατάλογος εγγράφων σας υπάρχει πριν προχωρήσετε σε λειτουργίες αρχείων.
#### Βήμα-βήμα
1. **Ορίστε τη διαδρομή του εγγράφου σας:**
   ```java
   String dataDir = "YOUR_DOCUMENT_DIRECTORY";
   ```
2. **Ελέγξτε και δημιουργήστε τον κατάλογο εάν είναι απαραίτητο:**
   ```java
   boolean isExists = new File(dataDir).exists();
   if (!isExists) {
       new File(dataDir).mkdirs(); // Δημιουργεί καταλόγους αναδρομικά
   }
   ```
### Χαρακτηριστικό 2: Δημιουργία παρουσίασης
**Επισκόπηση:** Δημιουργήστε μια νέα παρουσία παρουσίασης PowerPoint.
#### Βήμα-βήμα
1. **Δημιουργήστε ένα αντίγραφο του αντικειμένου παρουσίασης:**
   ```java
   Presentation pres = new Presentation();
   ```
### Λειτουργία 3: Προσθήκη Αυτόματου Σχήματος σε Διαφάνεια
**Επισκόπηση:** Προσθέστε σχήματα, όπως ορθογώνια, στις διαφάνειές σας για τη δομή του περιεχομένου.
#### Βήμα-βήμα
1. **Αποκτήστε πρόσβαση στην πρώτη διαφάνεια και προσθέστε ένα ορθογώνιο σχήμα:**
   ```java
   ISlide slide = pres.getSlides().get_Item(0);
   IAutoShape aShp = slide.getShapes().addAutoShape(
       ShapeType.Rectangle, 200, 200, 400, 200);
   ```
### Λειτουργία 4: Προσθήκη και μορφοποίηση κειμένου στο AutoShape
**Επισκόπηση:** Εισαγάγετε κείμενο σε σχήματα και εφαρμόστε μορφοποίηση κουκκίδων για μεγαλύτερη σαφήνεια.
#### Βήμα-βήμα
1. **Πρόσβαση στο Πλαίσιο Κειμένου του Σχήματος:**
   ```java
   ITextFrame text = aShp.addTextFrame("");
   ```
2. **Προσθήκη και μορφοποίηση παραγράφων με κουκκίδες:**
   ```java
   Paragraph para1 = new Paragraph();
   para1.setText("Content");
   para1.getParagraphFormat().getBullet().setType(BulletType.Symbol);
   para1.getParagraphFormat().setDepth((short) 0); // Κουκκίδα Επιπέδου 1

   text.getParagraphs().add(para1);
   ```
### Λειτουργία 5: Αποθήκευση παρουσίασης
**Επισκόπηση:** Αποθηκεύστε την παρουσίασή σας σε μια καθορισμένη διαδρομή σε μορφή PPTX.
#### Βήμα-βήμα
1. **Καθορίστε τη διαδρομή εξόδου και αποθηκεύστε το αρχείο:**
   ```java
   String outputPath = "YOUR_OUTPUT_DIRECTORY/MultilevelBullet.pptx";
   pres.save(outputPath, SaveFormat.Pptx);
   ```

## Οδηγός Εφαρμογής

### Μετατροπή σχημάτων PowerPoint σε SVG σε Java

Αυτή η ενότητα παρέχει έναν αναλυτικό οδηγό για το πώς να μετατρέψετε σχήματα PowerPoint σε αρχεία SVG χρησιμοποιώντας το Aspose.Slides για Java.

#### Βήμα 1: Αρχικοποίηση SVGOptions

Ο `SVGOptions` Η κλάση σάς επιτρέπει να διαμορφώσετε διάφορες ρυθμίσεις για τη διαδικασία μετατροπής:

```java
// Δημιουργία αντικειμένου SVGOptions
SVGOptions svgOptions = new SVGOptions();
```

**Εξήγηση:** Αυτό αρχικοποιεί τις επιλογές για τη μετατροπή σχημάτων σε SVG, δίνοντάς σας τον έλεγχο της εξόδου.

#### Βήμα 2: Ορισμός ρυθμίσεων μετατροπής

Προσαρμόστε τον τρόπο απόδοσης της παρουσίασής σας σε SVG:

- **Χρήση μεγέθους πλαισίου**: Συμπερίληψη του πλαισίου στην απόδοση.

  ```java
  // Ορίστε το UseFrameSize σε true
  svgOptions.setUseFrameSize(true);
  ```

- **Εξαίρεση περιστροφής**Μην περιστρέφετε τα σχήματα κατά τη μετατροπή.

  ```java
  // Ορισμός του UseFrameRotation σε false
  svgOptions.setUseFrameRotation(false);
  ```

**Εξήγηση:** Αυτές οι ρυθμίσεις σάς επιτρέπουν να ελέγχετε την περιοχή απόδοσης και τον προσανατολισμό της εξόδου SVG, διασφαλίζοντας ότι πληροί τις συγκεκριμένες απαιτήσεις σας.

#### Βήμα 3: Αποθήκευση ως SVG

Τέλος, αποθηκεύστε ένα σχήμα PowerPoint ως αρχείο SVG:

```java
import java.io.FileOutputStream;
import java.io.IOException;

String presentationName = "YOUR_DOCUMENT_DIRECTORY/SvgShapesConversion.pptx";
String outPath = "YOUR_OUTPUT_DIRECTORY/SvgShapesConversion.svg";

// Φόρτωση της παρουσίασης
Presentation presentation = new Presentation(presentationName);
try {
    // Αποθήκευση του πρώτου σχήματος από την πρώτη διαφάνεια ως SVG
    try (FileOutputStream stream = new FileOutputStream(outPath)) {
        presentation.getSlides().get_Item(0).getShapes().get_Item(0).writeAsSvg(stream, svgOptions);
    }
} catch(IOException e) {
    System.out.println("Error writing file: " + e.getMessage());
} finally {
    if (presentation != null) presentation.dispose();
}
```

**Εξήγηση:** Αυτό το απόσπασμα κώδικα δείχνει τη φόρτωση ενός αρχείου PowerPoint και την εξαγωγή του πρώτου σχήματος στην πρώτη διαφάνεια ως SVG χρησιμοποιώντας τις καθορισμένες επιλογές. Περιλαμβάνεται κατάλληλος χειρισμός σφαλμάτων για τη διαχείριση των λειτουργιών αρχείων.

# Biçimlendirilmiş Metinle PowerPoint Sunumları Oluşturma

Aspose.Slides for .NET, geliştiricilerin Microsoft PowerPoint gerektirmeden PowerPoint sunumları oluşturmasına, düzenlemesine ve dönüştürmesine olanak tanıyan güçlü bir API'dir. Bu örnek, biçimlendirilmiş metin paragrafları içeren dikdörtgen bir şekil içeren basit bir sunumun nasıl oluşturulacağını gösterir.

Kod yeni bir sunum oluşturur, ilk slayda dikdörtgen bir şekil ekler ve ardından farklı metin biçimlendirme seçeneklerine sahip birden fazla paragraf ekler. Metnin belirli bölümleri için boyut ve aile gibi yazı tipi özelliklerinin nasıl ayarlanacağını gösterir.

## Kurulum Talimatları

1. NuGet Paket Yöneticisi aracılığıyla kurulum:
   ```
   Install-Package Aspose.Slides
   ```

2. .NET CLI üzerinden kurulum:
   ```
   dotnet add package Aspose.Slides
   ```

3. Paket Yöneticisi Konsolu aracılığıyla kurulum:
   ```
   PM> Install-Package Aspose.Slides
   ```

4. Projenize Referans Ekleyin:
   Kurulumdan sonra projenize bir referans ekleyin:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

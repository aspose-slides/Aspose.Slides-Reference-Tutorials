# Aspose.Slides Kullanarak PowerPoint'te Dikey Metin Oluşturma

Aspose.Slides for .NET, PowerPoint sunumlarını programatik olarak oluşturmak ve düzenlemek için güçlü yetenekler sunar. Bu örnek, dikey olarak döndürülmüş metin içeren bir dikdörtgen şekli içeren bir sunumun nasıl oluşturulacağını gösterir. Metin 270 derecelik bir açıyla yönlendirilir ve bu da alttan üste doğru okunmasını sağlar. Bu teknik, sunumlarınızda yan etiketler, benzersiz metin düzenleri veya özel tasarım öğeleri oluşturmak için kullanışlıdır.

## Kurulum Talimatları

1. .NET için Aspose.Slides'ı yükleyin
   
   NuGet Paket Yöneticisi'ni kullanarak Aspose.Slides paketini yükleyin:
   
   ```csharp
   Install-Package Aspose.Slides
   ```
   
   Veya .NET CLI'yi kullanarak:
   
   ```csharp
   dotnet add package Aspose.Slides
   ```

2. Referans Ekle
   
   Aşağıdaki using ifadesini C# dosyanıza ekleyin:
   
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

# PowerPoint Sunumlarında Özel Madde İşaretleri Oluşturma

.NET için Aspose.Slides, PowerPoint sunumlarını programatik olarak oluşturmak ve düzenlemek için sağlam yetenekler sunar. Bu örnek özellikle metin çerçevelerinde özel madde işaretlerinin uygulanmasına odaklanarak madde işareti türü, rengi, girintisi ve boyutunun nasıl kontrol edileceğini gösterir. Kod, iki paragraf içeren dikdörtgen şekilli bir sunum oluşturur - biri sembol madde işareti ve diğeri numaralandırılmış madde işareti stiline sahiptir.

## Kurulum Talimatları

1. NuGet Paket Yöneticisi'ni kullanarak .NET için Aspose.Slides'ı yükleyin:
   ```
   Install-Package Aspose.Slides
   ```

2. Alternatif olarak paketi doğrudan indirebilirsiniz:
   - Ziyaret edin [Aspose.Slides İndirme Sayfası](https://downloads.aspose.com/slides/net)
   - En son sürümü indirin
   - Projenizde Aspose.Slides.dll dosyasını ayıklayın ve başvurun

3. Gerekli using ifadesini ekleyin:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

# Aspose.Slides ile PowerPoint'te Çok Düzeyli Madde İşaretleri Oluşturma

Aspose.Slides for .NET, PowerPoint sunumlarını programatik olarak oluşturmak ve düzenlemek için güçlü bir API sağlar. Bu örnek, çok seviyeli madde işaretleriyle bir sunumun nasıl oluşturulacağını, metin çerçevesiyle otomatik bir şeklin nasıl ekleneceğini ve paragrafların farklı madde işareti stilleri ve girinti seviyeleriyle nasıl biçimlendirileceğini gösterir. Her girinti seviyesi için özelleştirilmiş madde işareti sembolleriyle hiyerarşik içerikleri kolayca oluşturabilirsiniz; bu, Microsoft PowerPoint'in yüklenmesini gerektirmeden yapılandırılmış ve profesyonel sunumlar oluşturmak için mükemmeldir.

## Kurulum Talimatları

1. NuGet Paket Yöneticisi aracılığıyla kurulum:
   ```
   Install-Package Aspose.Slides
   ```

2. .NET CLI üzerinden kurulum:
   ```
   dotnet add package Aspose.Slides
   ```

3. Paket Yöneticisi Konsolu aracılığıyla kurulum:
   ```
   PM> Install-Package Aspose.Slides
   ```

4. Manuel Kurulum:
   - En son sürümü şu adresten indirin: [.NET için Aspose.Slides](https://releases.aspose.com/slides/net/)
   - Projenize Aspose.Slides.dll'ye referans ekleyin

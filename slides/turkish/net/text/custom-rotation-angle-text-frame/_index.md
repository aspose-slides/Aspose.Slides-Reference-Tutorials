# PowerPoint'te Aspose.Slides ile Dönen Grafik Başlıkları ve Etiketleri

Aspose.Slides for .NET, PowerPoint sunumlarını programatik olarak oluşturmak ve düzenlemek için güçlü yetenekler sunar. Bu örnek, döndürülmüş metin öğeleri içeren kümelenmiş bir sütun grafiğiyle bir sunumun nasıl oluşturulacağını gösterir - özellikle, daha iyi okunabilirlik ve görsel çekicilik için hem grafik başlığını hem de veri etiketlerini döndürme.

## Kurulum Talimatları

1. NuGet Paket Yöneticisi aracılığıyla kurulum:
   ```
   Install-Package Aspose.Slides
   ```

2. Veya .NET CLI üzerinden yükleyin:
   ```
   dotnet add package Aspose.Slides
   ```

3. Ad alanını C# dosyanıza ekleyin:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Charts;
   ```

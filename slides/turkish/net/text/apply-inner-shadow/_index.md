# Aspose.Slides for .NET'te TextBox ile PowerPoint Sunumları Oluşturun

Aspose.Slides for .NET, geliştiricilerin Microsoft PowerPoint gerektirmeden PowerPoint sunumları oluşturmasına, değiştirmesine ve dönüştürmesine olanak tanıyan güçlü bir API'dir. Bu örnek, özelleştirilmiş bir metin kutusuyla bir sunumun nasıl oluşturulacağını, biçimlendirmenin nasıl uygulanacağını ve PPTX dosyası olarak nasıl kaydedileceğini gösterir.

## Kurulum Talimatları

1. NuGet Paket Yöneticisi aracılığıyla kurulum:
   ```
   Install-Package Aspose.Slides
   ```

2. .NET CLI üzerinden kurulum:
   ```
   dotnet add package Aspose.Slides
   ```

3. Manuel olarak yükleyin:
   - En son sürümü şu adresten indirin: [Aspose.Slides for .NET sürümleri sayfası](https://releases.aspose.com/slides/net/)
   - DLL referansını projenize ekleyin

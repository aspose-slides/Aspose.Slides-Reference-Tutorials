# Aspose.Slides ile PowerPoint'te Grafikleri Değiştirme

Aspose.Slides for .NET, PowerPoint sunumlarını programatik olarak oluşturmak ve değiştirmek için güçlü API'ler sağlar. Bu örnek, grafik verilerini güncelleyerek, yeni seriler ekleyerek ve grafik türlerini değiştirerek mevcut sunumlardaki grafiklerle nasıl çalışılacağını gösterir - hepsi Microsoft PowerPoint'in yüklenmesini gerektirmeden.

## Kurulum Talimatları

1. NuGet Paket Yöneticisi aracılığıyla kurulum:
   ```
   Install-Package Aspose.Slides
   ```

2. Veya .NET CLI kullanarak:
   ```
   dotnet add package Aspose.Slides
   ```

3. Aşağıdaki using yönergelerini ekleyin:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Charts;
   ```

# PowerPoint Sunumlarında GeometryShape'e Segmentler Ekleme

Bu örnek, Aspose.Slides for .NET kullanarak PowerPoint sunumlarındaki şekillerin geometri yoluna nasıl erişileceğini ve bu yolun nasıl değiştirileceğini gösterir. Dikdörtgen bir şekle özel çizgi parçaları ekleyerek, standart şekil türleri aracılığıyla kullanılamayan daha karmaşık geometrik şekiller oluşturabilirsiniz. Bu teknik, sunumlarınızda özel diyagramlar, akış şemaları ve diğer görsel öğeler oluşturmak için kullanışlıdır.

## Kurulum Talimatları

1. Aşağıdaki yöntemlerden birini kullanarak Aspose.Slides for .NET'i yükleyin:

   - NuGet Paket Yöneticisi:
     ```
     Install-Package Aspose.Slides.NET
     ```

   - Paket Yöneticisi Konsolu:
     ```
     PM> Install-Package Aspose.Slides.NET
     ```

   - .NET Komut Satırı Arayüzü:
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. Ad alanını C# dosyanıza ekleyin:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

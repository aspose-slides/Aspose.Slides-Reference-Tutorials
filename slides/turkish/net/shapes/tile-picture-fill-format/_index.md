# Aspose.Slides ile PowerPoint'te Döşeli Görüntü Arka Planları Oluşturma

Aspose.Slides for .NET, PowerPoint sunumlarını programatik olarak oluşturmak ve düzenlemek için güçlü yetenekler sunar. Bu örnek, döşenmiş bir resim deseniyle doldurulmuş dikdörtgen bir şekille bir sunumun nasıl oluşturulacağını gösterir. Kod, bir resmin nasıl yükleneceğini, bir dikdörtgenin arka planı olarak bir slayda nasıl ekleneceğini ve ofset, ölçek, hizalama ve çevirme seçenekleri gibi çeşitli döşeme özelliklerinin nasıl uygulanacağını gösterir. Bu teknik, döşeme davranışı üzerinde hassas kontrolle sunumlarınızda özel arka planlar, filigranlar veya dekoratif öğeler oluşturmak için kullanışlıdır.

## Kurulum Talimatları

1. NuGet Paket Yöneticisi'ni kullanarak .NET için Aspose.Slides'ı yükleyin:
   ```
   Install-Package Aspose.Slides.NET
   ```

2. Veya .NET CLI üzerinden yükleyin:
   ```
   dotnet add package Aspose.Slides.NET
   ```

3. Aşağıdaki using yönergelerini C# dosyanıza ekleyin:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.IO;
   ```

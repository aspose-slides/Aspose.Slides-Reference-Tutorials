# PowerPoint'te Özel Satır Biçimlendirmesiyle Dikdörtgen Şekli Oluşturma

.NET için Aspose.Slides, PowerPoint sunumlarını programatik olarak oluşturmak ve değiştirmek için güçlü yetenekler sunar. Bu örnek, çizgi stili, genişlik, çizgi deseni ve rengi ayarlama dahil olmak üzere özelleştirilmiş çizgi biçimlendirmesiyle dikdörtgen şeklinin nasıl oluşturulacağını gösterir. Örnek kod yeni bir sunum oluşturur, ilk slayda bir dikdörtgen ekler ve şeklin hem dolgusuna hem de dış hatlarına belirli biçimlendirme uygular, beyaz dolgu ve mavi çizgi dış hatlarına sahip görsel olarak çekici bir öğeyle sonuçlanır.

## Kurulum Talimatları

1. Aşağıdaki yöntemlerden birini kullanarak Aspose.Slides for .NET'i yükleyin:

   - NuGet Paket Yöneticisi:
     ```
     Install-Package Aspose.Slides
     ```

   - Paket Yöneticisi Konsolu:
     ```
     PM> Install-Package Aspose.Slides
     ```

   - .NET Komut Satırı Arayüzü:
     ```
     dotnet add package Aspose.Slides
     ```

2. C# dosyanıza gerekli ad alanını ekleyin:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

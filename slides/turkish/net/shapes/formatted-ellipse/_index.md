# Aspose.Slides ile PowerPoint'te Eliptik Şekiller Oluşturma

.NET için Aspose.Slides, geliştiricilerin Microsoft PowerPoint gerektirmeden PowerPoint sunumları oluşturmasını, okumasını, yazmasını, değiştirmesini ve dönüştürmesini sağlayan güçlü bir API'dir. Bu örnek, bir sunumda eliptik bir şekil oluşturmayı, dolgu rengi ve çizgi özellikleri dahil biçimlendirmeyi uygulamayı ve sonucu bir PPTX dosyası olarak kaydetmeyi gösterir. Kod, Aspose.Slides'ın şekilleri görünümleri ve konumları üzerinde hassas kontrolle düzenleme yeteneğini sergiler.

## Kurulum Talimatları

1. NuGet Paket Yöneticisi aracılığıyla kurulum:
   ```
   Install-Package Aspose.Slides
   ```

   Veya .NET CLI'yi kullanarak:
   ```
   dotnet add package Aspose.Slides
   ```

2. Referans Ekle:
   Aşağıdaki ad alanını C# dosyanıza ekleyin:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

3. Lisans Kurulumu (İsteğe bağlı):
   Eğer lisansınız varsa, bunu kullanarak uygulayın:
   ```csharp
   License license = new License();
   license.SetLicense("Aspose.Slides.lic");
   ```

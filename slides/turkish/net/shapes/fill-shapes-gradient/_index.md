# Aspose.Slides ile PowerPoint'te Gradient Dolu Elips Şekilleri Oluşturma

Bu örnek, özel bir degrade dolgu efektine sahip bir elips şekli içeren bir PowerPoint sunumunun programatik olarak nasıl oluşturulacağını gösterir. Kod yeni bir sunum oluşturur, ilk slayda bir elips ekler ve kırmızı ve mor renk duraklarıyla doğrusal bir degrade dolgu uygular. .NET için Aspose.Slides, şekiller, biçimlendirme ve efektler üzerinde hassas kontrolle PowerPoint sunumları oluşturmak ve düzenlemek için kapsamlı bir API sağlar.

## Kurulum Talimatları

1. Aşağıdaki yöntemlerden birini kullanarak Aspose.Slides for .NET'i yükleyin:
   - NuGet Paket Yöneticisi: `Install-Package Aspose.Slides.NET`
   - .NET Komut Satırı Arayüzü: `dotnet add package Aspose.Slides.NET`
   - Paket Yöneticisi Konsolu: `Install-Package Aspose.Slides.NET`

2. Aşağıdaki using ifadesini C# dosyanıza ekleyin:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

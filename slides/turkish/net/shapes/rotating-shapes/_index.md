# PowerPoint'te Dikdörtgen Şekilleri Oluşturma ve Döndürme

Aspose.Slides for .NET, geliştiricilerin Microsoft PowerPoint kurulumu gerektirmeden PowerPoint sunumları oluşturmasına, değiştirmesine ve dönüştürmesine olanak tanıyan güçlü bir API'dir. Bu örnek, döndürülmüş dikdörtgen şekilli bir sunumun nasıl oluşturulacağını gösterir ve kitaplığın şekil düzenleme yeteneklerini sergiler.

Aspose.Slides ile slaytlarınıza programatik olarak çeşitli şekiller ekleyebilir ve döndürme gibi dönüşümler uygulayabilirsiniz. Bu işlevsellik, görsel öğeler üzerinde hassas kontrole sahip dinamik sunumlar oluşturmak için özellikle yararlıdır.

## Kurulum Talimatları

1. NuGet Paket Yöneticisi aracılığıyla kurulum:
   ```
   Install-Package Aspose.Slides
   ```

2. .NET CLI üzerinden kurulum:
   ```
   dotnet add package Aspose.Slides
   ```

3. Kodunuza using yönergesini ekleyin:
   ```csharp
   using Aspose.Slides;
   ```

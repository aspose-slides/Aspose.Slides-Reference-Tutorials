# PowerPoint Sunumlarında Şekilleri Hizalama ve Gruplama

Aspose.Slides for .NET, PowerPoint sunumlarını programatik olarak oluşturmak ve düzenlemek için güçlü yetenekler sunar. Bu örnek, slaytlar içindeki şekillerin nasıl hizalanacağını ve PowerPoint sunumlarında gruplanmış şekillerle nasıl çalışılacağını gösterir. Dikdörtgenleri belirli koordinatlara nasıl yerleştireceğinizi, bunları slayta veya birbirlerine göre nasıl hizalayacağınızı ve gruplar içindeki şekilleri seçici olarak nasıl hizalayacağınızı öğreneceksiniz.

## Kurulum Talimatları

1. Aşağıdaki yöntemlerden birini kullanarak Aspose.Slides for .NET'i yükleyin:
   
   - NuGet Paket Yöneticisi:
     ```
     Install-Package Aspose.Slides.NET
     ```
   
   - Paket Yöneticisi Konsolu:
     ```
     PM> Install-Package Aspose.Slides.NET
     ```
   
   - .NET Komut Satırı Arayüzü:
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. Projenize Referanslar Ekleyin:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.IO;
   ```

3. Aşağıdaki Örnek Kodu indirin ve projenize entegre edin.

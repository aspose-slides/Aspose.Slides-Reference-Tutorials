# Aspose.Slides kullanarak PowerPoint'te Döşenmiş Görüntü Doldurma ile Şekiller Oluşturma

Aspose.Slides for .NET, PowerPoint sunumlarını programatik olarak oluşturmak ve düzenlemek için güçlü yetenekler sunar. Bu örnek, dikdörtgen bir şekil oluşturmayı ve onu döşenmiş bir resim deseniyle doldurmayı gösterir. Döşenmiş resim dolgusu, sunumlarınızda dokulu arka planlar, filigranlar veya dekoratif öğeler oluşturmak için özellikle yararlıdır.

## Kurulum Talimatları

1. NuGet Paket Yöneticisi aracılığıyla kurulum:
   ```
   Install-Package Aspose.Slides
   ```

2. .NET CLI üzerinden kurulum:
   ```
   dotnet add package Aspose.Slides
   ```

3. Manuel Kurulum:
   - En son sürümü şu adresten indirin: [.NET için Aspose.Slides](https://releases.aspose.com/slides/net/)

# Aspose.Slides ile PowerPoint'te Grup Şekilleri Oluşturun

.NET için Aspose.Slides, PowerPoint sunumlarını programatik olarak oluşturmak ve düzenlemek için kapsamlı API'ler sağlar. Bu örnek, bir grup şeklinin nasıl oluşturulacağını, içine birden fazla dikdörtgen şeklinin nasıl ekleneceğini, şekillerin nasıl düzgün şekilde konumlandırılacağını, çerçevelemenin nasıl ekleneceğini ve sunumun nasıl kaydedileceğini gösterir. Grup şekilleri, tek bir birim olarak düzenlenmesi gereken ilgili şekilleri organize etmek, sunum öğeleriniz üzerinde daha fazla yapı ve kontrol sağlamak için önemlidir.

## Kurulum Talimatları

1. Aşağıdaki yöntemlerden birini kullanarak Aspose.Slides for .NET'i yükleyin:

   - NuGet Paket Yöneticisi:
     ```
     Install-Package Aspose.Slides.NET
     ```

   - Paket Yöneticisi Konsolu:
     ```
     PM> Install-Package Aspose.Slides.NET
     ```

   - .NET Komut Satırı Arayüzü:
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. Projenize referanslar ekleyin:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

3. Lisans kurulumu (üretim kullanımı için isteğe bağlı):
   ```csharp
   // Değerlendirme kısıtlamalarını kaldırmak için lisansı ayarlayın
   License license = new License();
   license.SetLicense("Aspose.Slides.lic");
   ```

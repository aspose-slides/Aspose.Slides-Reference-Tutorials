# Téglalap alakú alakzat létrehozása egyéni vonalformázással PowerPointban

Az Aspose.Slides for .NET hatékony funkciókat kínál PowerPoint-bemutatók programozott létrehozásához és módosításához. Ez a példa bemutatja, hogyan hozhat létre téglalap alakú alakzatot testreszabott vonalformázással, beleértve a vonalstílus, a szélesség, a szaggatott vonal mintázata és a szín beállítását. A mintakód létrehoz egy új bemutatót, hozzáad egy téglalapot az első diához, és speciális formázást alkalmaz az alakzat kitöltésére és körvonalára, így egy vizuálisan vonzó elemet hoz létre fehér kitöltéssel és kék szaggatott körvonallal.

## Telepítési utasítások

1. Telepítse az Aspose.Slides for .NET programot az alábbi módszerek egyikével:

   - NuGet csomagkezelő:
     ```
     Install-Package Aspose.Slides
     ```

   - Csomagkezelő konzol:
     ```
     PM> Install-Package Aspose.Slides
     ```

   - .NET parancssori felület:
     ```
     dotnet add package Aspose.Slides
     ```

2. Adja hozzá a szükséges névteret a C# fájljához:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

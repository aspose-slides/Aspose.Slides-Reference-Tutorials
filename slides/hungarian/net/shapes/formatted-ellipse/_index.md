# Ellipszis alakzatok létrehozása PowerPointban az Aspose.Slides segítségével

Az Aspose.Slides for .NET egy hatékony API, amely lehetővé teszi a fejlesztők számára PowerPoint-bemutatók létrehozását, olvasását, írását, módosítását és konvertálását Microsoft PowerPoint nélkül. Ez a példa bemutatja, hogyan hozhat létre ellipszis alakzatot egy bemutatóban, hogyan alkalmazhat formázást, beleértve a kitöltőszínt és a vonaltulajdonságokat, és hogyan mentheti az eredményt PPTX fájlként. A kód bemutatja az Aspose.Slides azon képességét, hogy pontosan szabályozza az alakzatok megjelenését és elhelyezkedését.

## Telepítési utasítások

1. Telepítés a NuGet csomagkezelőn keresztül:
   ```
   Install-Package Aspose.Slides
   ```

   Vagy a .NET parancssori felület használatával:
   ```
   dotnet add package Aspose.Slides
   ```

2. Referenciák hozzáadása:
   Adja hozzá a következő névteret a C# fájljához:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

3. Licenc beállítása (opcionális):
   Ha van engedélyed, igényeld a következő módon:
   ```csharp
   License license = new License();
   license.SetLicense("Aspose.Slides.lic");
   ```

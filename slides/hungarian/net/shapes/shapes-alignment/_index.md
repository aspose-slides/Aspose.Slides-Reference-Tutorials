# Alakzatok igazítása és csoportosítása PowerPoint-bemutatókban

Az Aspose.Slides for .NET hatékony funkciókat kínál PowerPoint-bemutatók programozott létrehozásához és kezeléséhez. Ez a példa bemutatja, hogyan igazíthatók alakzatok a diákon belül, és hogyan dolgozható csoportosított alakzatokkal PowerPoint-bemutatókban. Megtanulod, hogyan helyezheted el a téglalapokat adott koordináták szerint, hogyan igazíthatod őket a diához vagy egymáshoz képest, és hogyan igazíthatod szelektíven az alakzatokat a csoportokon belül.

## Telepítési utasítások

1. Telepítse az Aspose.Slides for .NET programot az alábbi módszerek egyikével:
   
   - NuGet csomagkezelő:
     ```
     Install-Package Aspose.Slides.NET
     ```
   
   - Csomagkezelő konzol:
     ```
     PM> Install-Package Aspose.Slides.NET
     ```
   
   - .NET parancssori felület:
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. Referenciák hozzáadása a projekthez:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.IO;
   ```

3. Töltsd le az alábbi példakódot, és integráld a projektedbe.

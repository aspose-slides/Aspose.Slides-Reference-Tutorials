# Színátmenettel kitöltött ellipszis alakzatok létrehozása PowerPointban az Aspose.Slides segítségével

Ez a példa bemutatja, hogyan hozható létre programozottan egy ellipszis alakzatot tartalmazó PowerPoint-bemutató egyéni színátmenetes kitöltési effektussal. A kód létrehoz egy új bemutatót, ellipszist ad az első diához, és egy lineáris színátmenetes kitöltést alkalmaz piros és lila színpontokkal. Az Aspose.Slides for .NET átfogó API-t biztosít PowerPoint-bemutatók létrehozásához és kezeléséhez, precíz vezérléssel az alakzatok, a formázás és az effektusok felett.

## Telepítési utasítások

1. Telepítse az Aspose.Slides for .NET programot az alábbi módszerek egyikével:
   - NuGet csomagkezelő: `Install-Package Aspose.Slides.NET`
   - .NET parancssori felület: `dotnet add package Aspose.Slides.NET`
   - Csomagkezelő konzol: `Install-Package Aspose.Slides.NET`

2. Add hozzá a következő using utasítást a C# fájlodhoz:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

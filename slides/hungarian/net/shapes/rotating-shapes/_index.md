# Téglalap alakú alakzatok létrehozása és forgatása PowerPointban

Az Aspose.Slides for .NET egy hatékony API, amely lehetővé teszi a fejlesztők számára PowerPoint-bemutatók létrehozását, módosítását és konvertálását a Microsoft PowerPoint telepítése nélkül. Ez a példa bemutatja, hogyan hozhat létre egy elforgatott téglalap alakú bemutatót, bemutatva a könyvtár alakzatmanipulációs képességeit.

Az Aspose.Slides segítségével programozottan adhatsz hozzá különféle alakzatokat a diáidhoz, és alkalmazhatsz transzformációkat, például forgatást. Ez a funkció különösen hasznos dinamikus prezentációk készítéséhez, amelyekkel pontosan szabályozhatod a vizuális elemeket.

## Telepítési utasítások

1. Telepítés a NuGet csomagkezelőn keresztül:
   ```
   Install-Package Aspose.Slides
   ```

2. Telepítés .NET CLI-n keresztül:
   ```
   dotnet add package Aspose.Slides
   ```

3. Adja hozzá a using direktívát a kódjához:
   ```csharp
   using Aspose.Slides;
   ```

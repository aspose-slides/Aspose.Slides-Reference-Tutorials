# Fürtözött oszlopdiagramok létrehozása az Aspose.Slides for .NET segítségével

Az Aspose.Slides for .NET hatékony funkciókat kínál PowerPoint-bemutatók programozott létrehozásához és kezeléséhez. Ez a példa bemutatja, hogyan hozhat létre egyéni kategóriákkal és adatsorokkal rendelkező fürtözött oszlopdiagramot.

## Telepítés

```csharp
// Telepítés a NuGet csomagkezelőn keresztül
Install-Package Aspose.Slides
```

Vagy adja hozzá a következő hivatkozást a projektfájljához:


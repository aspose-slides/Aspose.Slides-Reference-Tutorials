# PowerPoint-bemutatók létrehozása formázott szöveggel

Az Aspose.Slides for .NET egy hatékony API, amely lehetővé teszi a fejlesztők számára PowerPoint-bemutatók létrehozását, kezelését és konvertálását Microsoft PowerPoint nélkül. Ez a példa bemutatja, hogyan hozhat létre egy egyszerű, téglalap alakú, formázott szöveges bekezdéseket tartalmazó bemutatót.

A kód létrehoz egy új prezentációt, hozzáad egy téglalap alakú alakzatot az első diához, majd több bekezdést ad hozzá különböző szövegformázási beállításokkal. Bemutatja, hogyan állíthatja be a betűtípus tulajdonságait, például a méretet és a családot a szöveg egyes részeihez.

## Telepítési utasítások

1. Telepítés a NuGet csomagkezelőn keresztül:
   ```
   Install-Package Aspose.Slides
   ```

2. Telepítés .NET CLI-n keresztül:
   ```
   dotnet add package Aspose.Slides
   ```

3. Telepítés a Package Manager Console-on keresztül:
   ```
   PM> Install-Package Aspose.Slides
   ```

4. Referencia hozzáadása a projekthez:
   A telepítés után adj hozzá egy hivatkozást a projektedhez:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

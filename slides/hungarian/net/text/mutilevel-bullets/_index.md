# Többszintű felsorolásjelek létrehozása PowerPointban az Aspose.Slides segítségével

Az Aspose.Slides for .NET egy hatékony API-t biztosít PowerPoint-bemutatók programozott létrehozásához és kezeléséhez. Ez a példa bemutatja, hogyan hozhat létre bemutatót többszintű felsorolásjelekkel, hogyan adhat hozzá automatikus alakzatot szövegkerettel, és hogyan formázhatja a bekezdéseket különböző felsorolásjelstílusokkal és behúzási szintekkel. Könnyedén létrehozhat hierarchikus tartalmat testreszabott felsorolásjelekkel minden behúzási szinthez, ami tökéletes strukturált és professzionális bemutatók készítéséhez anélkül, hogy telepíteni kellene a Microsoft PowerPointot.

## Telepítési utasítások

1. Telepítés a NuGet csomagkezelőn keresztül:
   ```
   Install-Package Aspose.Slides
   ```

2. Telepítés .NET CLI-n keresztül:
   ```
   dotnet add package Aspose.Slides
   ```

3. Telepítés a Package Manager Console-on keresztül:
   ```
   PM> Install-Package Aspose.Slides
   ```

4. Manuális telepítés:
   - Töltsd le a legújabb verziót innen: [Aspose.Slides .NET-hez](https://releases.aspose.com/slides/net/)
   - Adjon hozzá hivatkozást az Aspose.Slides.dll fájlra a projektben

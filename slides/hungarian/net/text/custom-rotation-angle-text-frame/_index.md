# Diagramcímek és címkék forgatása PowerPointban az Aspose.Slides segítségével

Az Aspose.Slides for .NET hatékony funkciókat kínál PowerPoint-bemutatók programozott létrehozásához és kezeléséhez. Ez a példa bemutatja, hogyan hozhat létre prezentációt egy elforgatott szövegelemeket tartalmazó fürtözött oszlopdiagrammal – konkrétan a diagram címének és az adatcímkéknek az elforgatásával a jobb olvashatóság és a vizuális megjelenés érdekében.

## Telepítési utasítások

1. Telepítés a NuGet csomagkezelőn keresztül:
   ```
   Install-Package Aspose.Slides
   ```

2. Vagy telepítse a .NET CLI-n keresztül:
   ```
   dotnet add package Aspose.Slides
   ```

3. Add hozzá a névteret a C# fájlodhoz:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Charts;
   ```

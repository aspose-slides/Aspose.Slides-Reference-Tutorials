# Egyéni felsoroláspontok létrehozása PowerPoint-bemutatókban

Az Aspose.Slides for .NET robusztus képességeket biztosít PowerPoint-bemutatók programozott létrehozásához és kezeléséhez. Ez a példa kifejezetten az egyéni felsorolásjelek szövegkeretekben való megvalósítására összpontosít, bemutatva, hogyan lehet szabályozni a felsorolásjelek típusát, színét, behúzását és méretét. A kód egy téglalap alakú bemutatót hoz létre, amely két bekezdést tartalmaz - az egyiket szimbólumjellel, a másikat pedig számozott felsorolásjel stílussal.

## Telepítési utasítások

1. Telepítse az Aspose.Slides .NET-hez készült verzióját a NuGet csomagkezelővel:
   ```
   Install-Package Aspose.Slides
   ```

2. Vagy töltse le közvetlenül a csomagot:
   - Látogassa meg a [Aspose.Slides letöltési oldal](https://downloads.aspose.com/slides/net)
   - Töltse le a legújabb verziót
   - Bontsa ki és hivatkozzon az Aspose.Slides.dll fájlra a projektjében

3. Adja hozzá a szükséges using utasítást:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

# पायथन के लिए Aspose.Slides के साथ PowerPoint में 3D रोटेशन को लागू करना

## कार्यान्वयन मार्गदर्शिका

इस अनुभाग में 3D रोटेशन प्रभाव लागू करने का विवरण दिया गया है।

### आयताकार आकार पर 3D रोटेशन लागू करना

#### अवलोकन

3D रोटेशन का उपयोग करके आयताकार आकृतियों में गहराई और परिप्रेक्ष्य जोड़ें।

#### चरण-दर-चरण कार्यान्वयन

**1. एक आयताकार आकार जोड़ें:**

```python
auto_shape = pres.slides[0].shapes.add_auto_shape(
    slides.ShapeType.RECTANGLE, 30, 30, 200, 200)
```

*स्पष्टीकरण*यह कोड स्थिति (30, 30) पर 200x200 आयाम वाला एक आयत जोड़ता है।

**2. 3D रोटेशन लागू करें:**

```python
auto_shape.three_d_format.depth = 6
auto_shape.three_d_format.camera.set_rotation(40, 35, 20)
auto_shape.three_d_format.camera.camera_type = slides.CameraPresetType.ISOMETRIC_LEFT_UP
auto_shape.three_d_format.light_rig.light_type = slides.LightRigPresetType.BALANCED
```

*स्पष्टीकरण*: 
- `depth`: 3D प्रभाव की गहराई निर्धारित करता है.
- `camera.set_rotation()`: X, Y, और Z अक्षों के लिए घूर्णन कोण कॉन्फ़िगर करता है।
- `camera_type`: कैमरा परिप्रेक्ष्य को परिभाषित करता है.
- `light_rig.light_type`: 3D स्वरूप को बढ़ाने के लिए प्रकाश व्यवस्था को समायोजित करता है।

**3. अपनी प्रस्तुति सहेजें:**

```python
pres.save("shapes_apply_3d_rotation_to_rectangle_out.pptx", slides.export.SaveFormat.PPTX)
```

### रेखा आकार पर 3D रोटेशन लागू करना

#### अवलोकन

रेखा आकृतियों में 3D प्रभाव जोड़कर दिलचस्प दृश्य तत्व बनाएं।

#### चरण-दर-चरण कार्यान्वयन

**1. एक रेखा आकार जोड़ें:**

```python
auto_shape = pres.slides[0].shapes.add_auto_shape(
    slides.ShapeType.LINE, 30, 300, 200, 200)
```

*स्पष्टीकरण*यह कोड स्थिति (30, 300) पर 200x200 आयाम वाली एक पंक्ति जोड़ता है।

**2. 3D रोटेशन लागू करें:**

```python
auto_shape.three_d_format.depth = 6
auto_shape.three_d_format.camera.set_rotation(0, 35, 20)
auto_shape.three_d_format.camera.camera_type = slides.CameraPresetType.ISOMETRIC_LEFT_UP
auto_shape.three_d_format.light_rig.light_type = slides.LightRigPresetType.BALANCED
```

*स्पष्टीकरण*: आयताकार आकार के समान, लेकिन अद्वितीय प्रभावों के लिए अलग-अलग घूर्णन कोण के साथ।

**3. अपनी प्रस्तुति सहेजें:**

```python
pres.save("shapes_apply_3d_rotation_to_line_out.pptx", slides.export.SaveFormat.PPTX)
```

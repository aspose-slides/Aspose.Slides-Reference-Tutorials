# पायथन के लिए Aspose.Slides के साथ PowerPoint कक्षों में छवियाँ जोड़ें और क्रॉप करें

## परिचय
आकर्षक प्रस्तुतिकरण बनाना चुनौतीपूर्ण हो सकता है, खासकर जब पावरपॉइंट स्लाइड में टेबल सेल के भीतर छवियों जैसे विस्तृत ग्राफ़िक्स को शामिल किया जाता है। पायथन के लिए Aspose.Slides के साथ, टेबल सेल के अंदर छवियों को जोड़ना और क्रॉप करना सीधा है, जो आपकी स्लाइड की व्यावसायिकता को बढ़ाता है।

इस ट्यूटोरियल में, आप सीखेंगे कि पायथन में Aspose.Slides लाइब्रेरी का उपयोग करके PowerPoint टेबल सेल के अंदर छवियों को कैसे सहजता से एकीकृत और क्रॉप किया जाए। इन चरणों का पालन करके, आप उन्नत PowerPoint हेरफेर के लिए शक्तिशाली लाइब्रेरी का लाभ उठाएंगे।

**आप क्या सीखेंगे:**
- पायथन के लिए Aspose.Slides सेट अप करना
- तालिका सेल में छवि जोड़ना
- स्लाइडों के भीतर छवियों पर क्रॉपिंग लागू करना
- अपनी अनुकूलित प्रस्तुति को सहेजना

आइये शुरू करने से पहले आवश्यक पूर्वापेक्षाओं पर गौर करें!

## पायथन के लिए Aspose.Slides सेट अप करना
Aspose.Slides का उपयोग शुरू करने के लिए, इसे pip के माध्यम से इंस्टॉल करें:

```bash
pip install aspose.slides
```

एक बार इंस्टॉल हो जाने पर, अपनी स्क्रिप्ट में लाइब्रेरी आयात करके अपने परिवेश को आरंभ करें। यदि आपके पास लाइसेंस है, तो मूल्यांकन प्रतिबंधों को हटाने के लिए इसे लागू करें:

```python
import aspose.slides as slides

# लाइसेंस लागू करें (यदि उपलब्ध हो)
license = slides.License()
license.set_license("path_to_your_license_file")
```

इससे Aspose.Slides सेट हो जाता है, और आप उन्नत छवि हेरफेर क्षमताओं के साथ प्रस्तुतियाँ तैयार करने के लिए तैयार हैं।

## कार्यान्वयन मार्गदर्शिका
### चरण 1: प्रेजेंटेशन क्लास ऑब्जेक्ट को इंस्टैंशिएट करें
इसका एक उदाहरण बनाएं `Presentation` आपकी PowerPoint फ़ाइल का प्रतिनिधित्व करने वाला क्लास:

```python
with slides.Presentation() as presentation:
```

### चरण 2: पहली स्लाइड तक पहुंचें
उस स्लाइड तक पहुंचें जहां आप तालिका जोड़ना चाहते हैं:

```python
slide = presentation.slides[0]
```

### चरण 3: तालिका संरचना परिभाषित करें
अपनी तालिका के लिए स्तंभ की चौड़ाई और पंक्ति की ऊँचाई निर्दिष्ट करें। यहाँ, हम सरलता के लिए एक समान आकार निर्धारित कर रहे हैं।

```python
dbl_cols = [150, 150, 150, 150]  # स्तंभ की चौड़ाई (बिंदुओं में)
dbl_rows = [100, 100, 100, 100, 90]  # पंक्ति की ऊंचाई (बिंदुओं में)
```

### चरण 4: स्लाइड में तालिका जोड़ें
अपनी स्लाइड पर तालिका को निर्दिष्ट निर्देशांक पर रखें:

```python
tbl = slide.shapes.add_table(50, 50, dbl_cols, dbl_rows)
```

### चरण 5: छवि लोड करें और जोड़ें
किसी निर्देशिका से एक छवि लोड करें और उसे प्रस्तुति के छवि संग्रह में जोड़ें।

```python
image_path = "YOUR_DOCUMENT_DIRECTORY/image1.jpg"
image = slides.Images.from_file(image_path)
imgx1 = presentation.images.add_image(image)
```

### चरण 6: छवि को क्रॉपिंग के साथ भरें के रूप में सेट करें
लोड की गई छवि को तालिका सेल पर लागू करें और क्रॉपिंग विकल्प सेट करें:

```python
tbl.rows[0][0].cell_format.fill_format.fill_type = slides.FillType.PICTURE
tbl.rows[0][0].cell_format.fill_format.picture_fill_format.picture_fill_mode = slides.PictureFillMode.STRETCH
tbl.rows[0][0].cell_format.fill_format.picture_fill_format.picture.image = imgx1

# अंकों में मान काटना
tbl.rows[0][0].cell_format.fill_format.picture_fill_format.crop_right = 20
tbl.rows[0][0].cell_format.fill_format.picture_fill_format.crop_left = 20
tbl.rows[0][0].cell_format.fill_format.picture_fill_format.crop_top = 20
tbl.rows[0][0].cell_format.fill_format.picture_fill_format.crop_bottom = 20
```

### चरण 7: प्रस्तुति सहेजें
अंत में, अपनी प्रस्तुति को एक फ़ाइल में सहेजें:

```python
output_path = "YOUR_OUTPUT_DIRECTORY/tables_add_crop_image_to_cell_out.pptx"
presentation.save(output_path, slides.export.SaveFormat.PPTX)
```

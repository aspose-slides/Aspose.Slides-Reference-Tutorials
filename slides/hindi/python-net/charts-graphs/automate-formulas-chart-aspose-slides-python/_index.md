## कार्यान्वयन मार्गदर्शिका

आइये कार्यान्वयन को प्रबंधनीय खंडों में विभाजित करें।

### चार्ट डेटा सेल में फ़ॉर्मूला सेट करना

#### अवलोकन
यह सुविधा आपको डेटा सेल में सीधे फ़ॉर्मूला सेट करके अपने चार्ट के भीतर डेटा की गतिशील रूप से गणना करने की अनुमति देती है। यह अपडेट को स्वचालित करने और प्रस्तुतियों में सटीकता सुनिश्चित करने के लिए विशेष रूप से उपयोगी है।

#### कार्यान्वयन के चरण

1. **प्रस्तुति ऑब्जेक्ट बनाएँ:**
   प्रेजेंटेशन ऑब्जेक्ट को आरंभ करने से आरंभ करें जहां हम अपना चार्ट जोड़ेंगे।
   
   ```python
   import aspose.slides as slides
   
   def set_formula_in_chart_cell():
       with slides.Presentation() as presentation:
           # आगे के चरण इस प्रकार हैं...
   ```

2. **क्लस्टर्ड कॉलम चार्ट जोड़ें:**
   अपनी प्रस्तुति की पहली स्लाइड में एक क्लस्टर कॉलम चार्ट डालें।
   
   ```python
   chart = presentation.slides[0].shapes.add_chart(
       slides.charts.ChartType.CLUSTERED_COLUMN, 150, 150, 500, 300)
   ```

3. **चार्ट डेटा कार्यपुस्तिका तक पहुँचें:**
   डेटा कक्षों में हेरफेर करने के लिए चार्ट से संबद्ध कार्यपुस्तिका ऑब्जेक्ट को पुनः प्राप्त करें।
   
   ```python
   workbook = chart.chart_data.chart_data_workbook
   ```

4. **सेल B2 में सूत्र सेट करें:**
   मानक स्प्रेडशीट संकेतन का उपयोग करके सेल B2 के लिए सूत्र परिभाषित करें।
   
   ```python
   cell1 = workbook.get_cell(0, "B2")
   cell1.formula = "1 + SUM(F2:H5)"
   ```

5. **सेल C2 में R1C1 संकेतन का उपयोग करें:**
   वैकल्पिक रूप से, अधिक जटिल सूत्रों के लिए R1C1 संकेतन का उपयोग करें।
   
   ```python
   cell2 = workbook.get_cell(0, "C2")
   cell2.r1c1_formula = "MAX(R2C6:R5C8) / 3"
   ```

6. **गणना सूत्र:**
   अपने चार्ट में इन सूत्रों के परिणामों की गणना करें।
   
   ```python
   workbook.calculate_formulas()
   ```

7. **अपना प्रस्तुतीकरण सहेजें:**
   अपनी प्रस्तुति को किसी विशिष्ट आउटपुट निर्देशिका में सहेजें.
   
   ```python
   presentation.save("YOUR_OUTPUT_DIRECTORY/charts_data_cell_formulas_out.pptx")
   ```

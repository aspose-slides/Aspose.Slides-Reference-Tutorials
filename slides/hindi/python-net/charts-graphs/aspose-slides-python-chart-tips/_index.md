- चार्ट शीर्षक सेट करने और प्रारूपित करने की तकनीकें
- अक्ष गुणधर्मों, ग्रिड रेखाओं और शीर्षकों को अनुकूलित करने की विधियाँ
- लेजेंड और चार्ट के पृष्ठभूमि तत्वों को प्रारूपित करना

# पायथन के लिए Aspose.Slides का उपयोग करके PowerPoint में पाई चार्ट का पाई कैसे बनाएं

पाई ऑफ़ पाई चार्ट जैसे दिखने में आकर्षक चार्ट बनाना जटिल जानकारी को अधिक सुपाच्य बनाकर आपके पावरपॉइंट प्रेजेंटेशन को महत्वपूर्ण रूप से बेहतर बना सकता है। यह ट्यूटोरियल आपको पायथन के लिए Aspose.Slides का उपयोग करके पाई ऑफ़ पाई चार्ट बनाने के बारे में मार्गदर्शन करता है।

## आप क्या सीखेंगे

- पायथन के लिए Aspose.Slides सेट अप करना
- पाई ऑफ पाई चार्ट के साथ पावरपॉइंट प्रेजेंटेशन बनाने के चरण
- बेहतर पठनीयता के लिए डेटा लेबल और श्रृंखला समूह विकल्पों को कॉन्फ़िगर करना
- प्रस्तुतियों में पाई ऑफ पाई चार्ट के व्यावहारिक अनुप्रयोग

आइये, अपने परिवेश को स्थापित करने और इन सुविधाओं को क्रियान्वित करने के बारे में विस्तार से जानें।

### कार्यान्वयन मार्गदर्शिका

#### फ़ीचर 1: चार्ट के साथ प्रेजेंटेशन बनाएँ

यह सुविधा दिखाएगी कि पावरपॉइंट प्रेजेंटेशन कैसे बनाएं और पहली स्लाइड में पाई ऑफ पाई चार्ट कैसे जोड़ें।

##### चार्ट जोड़ना

एक नई प्रस्तुति बनाकर और पहली स्लाइड पर स्थिति (50, 50) पर पाई ऑफ पाई चार्ट जोड़कर आरंभ करें:

```python
import aspose.slides as slides

with slides.Presentation() as presentation:
    # निर्दिष्ट आयामों के साथ 'पाई ऑफ पाई' चार्ट जोड़ें
    chart = presentation.slides[0].shapes.add_chart(
        slides.charts.ChartType.PIE_OF_PIE, 50, 50, 500, 400)
```

##### डेटा लेबल कॉन्फ़िगर करना

पठनीयता बढ़ाने के लिए, डेटा लेबल को मान प्रदर्शित करने के लिए कॉन्फ़िगर करें:

```python
# बेहतर स्पष्टता के लिए डेटा लेबल में मान प्रदर्शन सक्षम करें
chart.chart_data.series[0].labels.default_data_label_format.show_value = True
```

##### पाई ऑफ पाई विकल्प सेट करना

पाई ऑफ पाई चार्ट के लिए विशिष्ट गुण कॉन्फ़िगर करें, जैसे कि दूसरे पाई का आकार और विभाजित स्थिति:

```python
# दूसरे पाई का आकार और विभाजन गुण सेट करें
chart.chart_data.series[0].parent_series_group.second_pie_size = 149
chart.chart_data.series[0].parent_series_group.pie_split_by = slides.charts.PieSplitType.BY_PERCENTAGE
chart.chart_data.series[0].parent_series_group.pie_split_position = 53
```

##### प्रस्तुति को सहेजना

अंत में, अपनी प्रस्तुति को इच्छित निर्देशिका में सहेजें:

```python
# चार्ट के साथ प्रस्तुति सहेजें
presentation.save("YOUR_OUTPUT_DIRECTORY/charts_second_plot_options_out.pptx", slides.export.SaveFormat.PPTX)
```

2. **शैक्षणिक परियोजनाएं**सर्वेक्षण के परिणाम प्रस्तुत करें, जिनमें प्रमुख विषयों के साथ-साथ कम महत्वपूर्ण निष्कर्ष भी दर्शाए गए हों।

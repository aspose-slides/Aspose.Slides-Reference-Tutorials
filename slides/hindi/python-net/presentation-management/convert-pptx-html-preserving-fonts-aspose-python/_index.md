# पायथन के लिए Aspose.Slides का उपयोग करके फ़ॉन्ट को संरक्षित करते हुए PPTX को HTML में बदलें

## परिचय

मूल फ़ॉन्ट को बनाए रखते हुए PowerPoint प्रस्तुतियों (PPTX) को HTML फ़ॉर्मेट में बदलना चुनौतीपूर्ण हो सकता है, खासकर यदि आप कुछ डिफ़ॉल्ट फ़ॉन्ट को एम्बेड होने से बाहर रखना चाहते हैं। "Aspose.Slides for Python" के साथ, यह कार्य सरल हो जाता है। यह ट्यूटोरियल आपको Python में Aspose.Slides का उपयोग करके संरक्षित फ़ॉन्ट के साथ PPTX फ़ाइलों को HTML में बदलने के बारे में मार्गदर्शन करता है।

**आप क्या सीखेंगे:**
- पायथन के लिए Aspose.Slides को कैसे स्थापित और सेट अप करें
- फ़ॉन्ट को संरक्षित रखते हुए पावरपॉइंट प्रस्तुतियों (PPTX) को HTML में परिवर्तित करना
- एम्बेडिंग से विशिष्ट डिफ़ॉल्ट फ़ॉन्ट को बाहर रखना
- रूपांतरण प्रक्रिया के दौरान प्रदर्शन को अनुकूलित करना

आइये शुरू करने से पहले आवश्यक शर्तों की समीक्षा करें!

## कार्यान्वयन मार्गदर्शिका

इस अनुभाग में, हम रूपांतरण प्रक्रिया को प्रबंधनीय चरणों में विभाजित करेंगे।

### PPTX को HTML में परिवर्तित करना मूल फ़ॉन्ट को संरक्षित रखना

#### अवलोकन:
इस कार्यान्वयन की प्राथमिक विशेषता पावरपॉइंट प्रेजेंटेशन को उसके मूल फ़ॉन्ट को संरक्षित करते हुए परिवर्तित करना और विशिष्ट डिफ़ॉल्ट फ़ॉन्ट को एम्बेड करने से बाहर रखना है। यह वेब प्रेजेंटेशन में ब्रांड की एकरूपता बनाए रखने के लिए विशेष रूप से उपयोगी हो सकता है।

#### चरण-दर-चरण कार्यान्वयन:

**1. इनपुट और आउटपुट पथ परिभाषित करें**

वह निर्देशिका सेट करें जहां आपकी इनपुट PPTX फ़ाइल स्थित है और जहां आप आउटपुट HTML फ़ाइल को सहेजना चाहते हैं।

```python
data_dir = "YOUR_DOCUMENT_DIRECTORY/"
out_dir = "YOUR_OUTPUT_DIRECTORY/"
```

**2. प्रेजेंटेशन फ़ाइल खोलें**

Aspose.Slides का उपयोग करें `Presentation` अपनी PPTX फ़ाइल लोड करने के लिए क्लास का उपयोग करें:

```python
with slides.Presentation(data_dir + "welcome-to-powerpoint.pptx") as pres:
    # आपका रूपांतरण कोड यहां जाएगा.
```

यह संदर्भ प्रबंधक यह सुनिश्चित करता है कि ऑपरेशन के बाद संसाधन उचित रूप से जारी किये जाएं।

**3. एक कस्टम फ़ॉन्ट एम्बेडिंग नियंत्रक बनाएँ**

का उपयोग करके कुछ फ़ॉन्ट को एम्बेडिंग से बाहर रखें `EmbedAllFontsHtmlController`:

```python
font_name_exclude_list = ["Calibri", "Arial"]
embed_fonts_controller = slides.export.EmbedAllFontsHtmlController(font_name_exclude_list)
```

यहां, "कैलिब्री" और "एरियल" को HTML आउटपुट में एम्बेड होने से बाहर रखा गया है।

**4. HTML निर्यात विकल्प कॉन्फ़िगर करें**

स्थापित करना `HtmlOptions` अपने नियंत्रक के साथ कस्टम फ़ॉन्ट फ़ॉर्मेटर का उपयोग करने के लिए:

```python
html_options_embed = slides.export.HtmlOptions()
html_options_embed.html_formatter = slides.export.HtmlFormatter.create_custom_formatter(embed_fonts_controller)
```

यह चरण यह सुनिश्चित करता है कि अंतिम आउटपुट में केवल आवश्यक फ़ॉन्ट ही एम्बेड किए जाएं।

**5. प्रेजेंटेशन को HTML के रूप में सेव करें**

अंत में, अपने निर्दिष्ट विकल्पों के साथ प्रस्तुति को HTML फ़ाइल में सहेजें:

```python
pres.save(out_dir + "convert_to_html_with_preserving_original_fonts_out.html", 
          slides.export.SaveFormat.HTML, html_options_embed)
```

### समस्या निवारण युक्तियों:
- सुनिश्चित करें कि पथ सही ढंग से निर्धारित और सुलभ हों।
- सिस्टम पर किसी भी अनुपलब्ध फ़ॉन्ट फ़ाइल की जाँच करें जो रूपांतरण को प्रभावित कर सकती है।

## व्यावहारिक अनुप्रयोगों

यहां कुछ वास्तविक परिदृश्य दिए गए हैं जहां यह सुविधा अविश्वसनीय रूप से उपयोगी हो सकती है:

1. **वेब पोर्टल**: ब्रांडिंग फ़ॉन्ट्स को खोए बिना वेब अनुप्रयोगों में सहज एकीकरण के लिए प्रस्तुतियों को HTML में परिवर्तित करें।
2. **दस्तावेज़ प्रबंधन प्रणालियाँ**दस्तावेज़ की विश्वसनीयता को बनाए रखते हुए आंतरिक पोर्टल में प्रस्तुतियाँ एम्बेड करें।
3. **ई-लर्निंग प्लेटफॉर्म**: ऑनलाइन पाठ्यक्रमों के भाग के रूप में परिवर्तित HTML फ़ाइलों का उपयोग करें, तथा एकरूपता बनाए रखें।

## प्रदर्शन संबंधी विचार

रूपांतरण के दौरान इष्टतम प्रदर्शन सुनिश्चित करने के लिए:
- **मेमोरी उपयोग को अनुकूलित करें**: अप्रयुक्त संसाधनों को तुरंत बंद करके संसाधन आवंटन का प्रबंधन करें।
- **प्रचय संसाधन**: ओवरहेड को कम करने के लिए कई प्रस्तुतियों को बैचों में परिवर्तित करें।
- **नवीनतम लाइब्रेरी संस्करण का उपयोग करें**बेहतर सुविधाओं और बग फिक्स के लिए हमेशा Aspose.Slides के नवीनतम संस्करण का उपयोग करें।

## कार्यान्वयन मार्गदर्शिका

### सुविधा 1: मास्टर नोट्स में हेडर टेक्स्ट अपडेट करें

यह सुविधा स्लाइड के मास्टर नोट्स के भीतर प्लेसहोल्डर्स के हेडर टेक्स्ट को अपडेट करने पर केंद्रित है। आप इसे इस तरह से प्राप्त कर सकते हैं:

#### अवलोकन
आप मास्टर नोट्स में आकृतियों को दोहराएंगे और पाए गए किसी भी हेडर को अद्यतन करेंगे।

#### कार्यान्वयन चरण
**चरण 1: हेडर अपडेट करने के लिए फ़ंक्शन परिभाषित करें**
```python
import aspose.slides as slides

def update_header_footer_text(master):
    """
    Iterate through shapes in the master and update header text if applicable.
    
    Args:
        master (slides.MasterSlide): The master slide containing the shapes to be updated.
    """
    for shape in master.shapes:
        # जाँच करें कि क्या आकृति प्लेसहोल्डर है और विशेष रूप से HEADER प्रकार की है
        if shape.placeholder is not None and shape.placeholder.type == slides.PlaceholderType.HEADER:
            shape.text_frame.text = "HI there new header"
```
**चरण 2: मास्टर नोट्स स्लाइड तक पहुंचें**
अपनी प्रस्तुति लोड करें, मास्टर नोट्स स्लाइड तक पहुंचें, और हेडर अपडेट लागू करें।
```python
def manage_header_footer_text():
    data_dir = "/path/to/your/document/directory/"
    out_dir = "/path/to/your/output/directory/"

    with slides.Presentation(data_dir + "layout_presentation.ppt") as pres:
        # हेडर टेक्स्ट को अपडेट करने के लिए मास्टर नोट्स स्लाइड तक पहुंचना
        master_notes_slide = pres.master_notes_slide_manager.master_notes_slide
        if master_notes_slide is not None:
            update_header_footer_text(master_notes_slide)

        # अद्यतन हेडर के साथ प्रस्तुति सहेजें
        pres.save(out_dir + "layout_update_header_footer_text_out.pptx", slides.export.SaveFormat.PPTX)
```
### फ़ीचर 2: हेडर और फ़ुटर टेक्स्ट प्रबंधित करें

यहां, हम सभी स्लाइडों में फ़ुटर टेक्स्ट सेट करेंगे और संशोधनों को सहेजेंगे।

#### अवलोकन
यह सुविधा आपको किसी प्रस्तुतिकरण में सभी स्लाइडों पर फ़ुटर सेट करने और प्रदर्शित करने की अनुमति देती है।

**चरण 1: फ़ुटर टेक्स्ट सेट करें**
सभी स्लाइडों के फ़ुटर अपडेट करने के लिए हेडर-फ़ुटर प्रबंधक का उपयोग करें:
```python
def manage_header_footer_text():
    data_dir = "/path/to/your/document/directory/"
    out_dir = "/path/to/your/output/directory/"

    with slides.Presentation(data_dir + "layout_presentation.ppt") as pres:
        # फ़ुटर टेक्स्ट को अपडेट करें और उसे सभी स्लाइड पर दृश्यमान बनाएं
        pres.header_footer_manager.set_all_footers_text("My Footer Text")
        pres.header_footer_manager.set_all_footers_visibility(True)
        
        # अद्यतन प्रस्तुति सहेजें
        pres.save(out_dir + "layout_update_header_footer_text_out.pptx", slides.export.SaveFormat.PPTX)
```

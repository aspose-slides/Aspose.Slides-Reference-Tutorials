## कार्यान्वयन मार्गदर्शिका

अब, आइए स्लाइड में लाइनों को फ़ॉर्मेट करने के कार्यान्वयन पर गौर करें।

### प्रस्तुति खोलना और तैयार करना

#### अवलोकन:
लाइन फ़ॉर्मेटिंग लागू करने के लिए पहले कोई मौजूदा प्रस्तुति खोलें या नई प्रस्तुति बनाएं।

```python
import aspose.slides as slides
class PresentationDemo:
    def format_lines(self):
        # प्रस्तुति खोलें या बनाएँ
        with self.presentation as pres:
            ...
```

**स्पष्टीकरण:**
- The `slides.Presentation()` संदर्भ प्रबंधक यह सुनिश्चित करता है कि संसाधनों का प्रबंधन स्वचालित रूप से हो, जो प्रदर्शन और स्मृति प्रबंधन के लिए महत्वपूर्ण है।

### स्लाइड में ऑटो-आकार जोड़ना

#### अवलोकन:
अपनी स्लाइड में एक आयताकार आकार जोड़ें जहां आप कस्टम लाइन फ़ॉर्मेटिंग लागू कर सकते हैं।

```python
# प्रस्तुति से पहली स्लाइड प्राप्त करें
class PresentationDemo:
    def format_lines(self):
        with self.presentation as pres:
            slide = pres.slides[0]

            # स्लाइड में आयत प्रकार का एक स्वचालित आकार जोड़ें
            shape = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 50, 150, 150, 75)
```

**स्पष्टीकरण:**
- `add_auto_shape()` विधि का उपयोग एक नया आकार सम्मिलित करने के लिए किया जाता है। यहाँ, हम इसे एक आयत के रूप में निर्दिष्ट करते हैं और स्थिति और आकार पैरामीटर प्रदान करते हैं।

### आकृति की रेखा शैली को प्रारूपित करना

#### अवलोकन:
अपनी आकृति के स्वरूप को बढ़ाने के लिए कस्टम चौड़ाई और डैश पैटर्न के साथ मोटी-पतली रेखा शैली लागू करें।

```python
class PresentationDemo:
    def format_lines(self):
        with self.presentation as pres:
            slide = pres.slides[0]
            shape = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 50, 150, 150, 75)

            # आयत का भरण रंग सफ़ेद पर सेट करें
            shape.fill_format.fill_type = slides.FillType.SOLID
            shape.fill_format.solid_fill_color.color = drawing.Color.white

            # विशिष्ट चौड़ाई और डैश शैली के साथ मोटी-पतली रेखा शैली लागू करें
            shape.line_format.style = slides.LineStyle.THICK_THIN
            shape.line_format.width = 7
            shape.line_format.dash_style = slides.LineDashStyle.DASH

            # आयत की सीमा का रंग नीला सेट करें
            shape.line_format.fill_format.fill_type = slides.FillType.SOLID
            shape.line_format.fill_format.solid_fill_color.color = drawing.Color.blue
```

**स्पष्टीकरण:**
- The `fill_format` और `line_format` गुण आपको आकृतियों की भरण और रूपरेखा दोनों शैलियों को अनुकूलित करने की अनुमति देते हैं।
- का विन्यास `LineStyle`, `width`, और `dash_style` आपको विशिष्ट दृश्य प्रभाव प्राप्त करने की सुविधा देता है।

### अपनी प्रस्तुति को सहेजना

#### अवलोकन:
बाद में उपयोग या साझा करने के लिए अपनी स्वरूपित प्रस्तुति को फ़ाइल में सहेजें।

```python
class PresentationDemo:
    def save_presentation(self, output_path):
        # स्वरूपित आकृतियों के साथ प्रस्तुति को डिस्क पर सहेजें
        self.presentation.save(output_path, slides.export.SaveFormat.PPTX)
```

**स्पष्टीकरण:**
- `save()` विधि परिवर्तनों को बनाए रखती है, यह सुनिश्चित करते हुए कि सभी संशोधन एक नई फ़ाइल में संग्रहीत हैं।

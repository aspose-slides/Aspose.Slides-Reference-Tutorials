## कार्यान्वयन मार्गदर्शिका

### PowerPoint में 3D आकार बनाना और कॉन्फ़िगर करना

#### अवलोकन

यह अनुभाग आपको Aspose.Slides का उपयोग करके आयताकार आकार जोड़ने, उसका टेक्स्ट सेट करने और 3D प्रभाव लागू करने की जानकारी देता है।

#### चरण-दर-चरण कार्यान्वयन

##### ऑटोशेप जोड़ना

सबसे पहले, अपनी स्लाइड में एक आयत जोड़ें:

```python
def render_3d_shape():
    with slides.Presentation() as pres:
        # पहली स्लाइड में एक स्वचालित आकार (आयताकार) जोड़ें
        shape = pres.slides[0].shapes.add_auto_shape(
            slides.ShapeType.RECTANGLE, 200, 150, 200, 200)
```

##### पाठ और फ़ॉन्ट आकार सेट करना

अपने आयत के अंदर पाठ समायोजित करें:

```python
        # आयत के अंदर पाठ सेट करें और फ़ॉन्ट आकार समायोजित करें
        shape.text_frame.text = "3D"
        shape.text_frame.paragraphs[0].paragraph_format.default_portion_format.font_height = 64
```

##### 3D सेटिंग्स कॉन्फ़िगर करना

यथार्थवादी 3D प्रभाव के लिए कैमरा, प्रकाश व्यवस्था और एक्सट्रूज़न को कॉन्फ़िगर करें:

```python
        # आकृति के लिए 3D सेटिंग कॉन्फ़िगर करें
        shape.three_d_format.camera.camera_type = slides.CameraPresetType.ORTHOGRAPHIC_FRONT
        shape.three_d_format.camera.set_rotation(20, 30, 40)
        shape.three_d_format.light_rig.light_type = slides.LightRigPresetType.FLAT
        shape.three_d_format.light_rig.direction = slides.LightingDirection.TOP
        shape.three_d_format.material = slides.MaterialPresetType.FLAT
        shape.three_d_format.extrusion_height = 100
        shape.three_d_format.extrusion_color.color = drawing.Color.blue
```

##### प्रस्तुति को सहेजना

अंत में, अपनी स्लाइड को छवि और प्रस्तुति के रूप में सहेजें:

```python
        # स्लाइड को छवि के रूप में तथा प्रस्तुति को निर्दिष्ट आउटपुट निर्देशिका में सहेजें
        pres.slides[0].get_image(2, 2).save("YOUR_OUTPUT_DIRECTORY/sample_3d.png")
        pres.save("YOUR_OUTPUT_DIRECTORY/rendering_3d_out.pptx", slides.export.SaveFormat.PPTX)
```

### व्यावहारिक अनुप्रयोगों

पावरपॉइंट में 3D आकृतियों को प्रस्तुत करने के लिए कुछ वास्तविक उपयोग के मामले यहां दिए गए हैं:

1. **उत्पाद प्रदर्शन:** इंटरैक्टिव, 3D विज़ुअल्स के साथ उत्पाद डेमो को बढ़ाएं।
2. **शैक्षिक प्रस्तुतियाँ:** जटिल अवधारणाओं को स्पष्ट रूप से समझाने के लिए 3D मॉडल का उपयोग करें।
3. **विपणन की चीजे:** आकर्षक प्रस्तुतियाँ बनाएँ जो ध्यान आकर्षित करें और संदेश को प्रभावी ढंग से संप्रेषित करें।

Aspose.Slides को अन्य प्रणालियों के साथ एकीकृत करने से आपका कार्यप्रवाह सुव्यवस्थित हो सकता है, जिससे स्वचालित रूप से शानदार प्रस्तुतियाँ तैयार की जा सकती हैं।

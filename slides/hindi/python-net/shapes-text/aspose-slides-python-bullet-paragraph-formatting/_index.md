## कार्यान्वयन मार्गदर्शिका

आइए बुलेट इंडेंटेशन और पैराग्राफ फॉर्मेटिंग को प्रबंधनीय अनुभागों में जोड़ने की प्रक्रिया को विभाजित करें।

### स्लाइड में आकृतियाँ जोड़ना

#### अवलोकन

सबसे पहले, हमें अपनी स्लाइड में एक आकृति जोड़ने की ज़रूरत है जिसमें टेक्स्ट होगा। इससे कंटेंट को व्यवस्थित करने में मदद मिलती है।

#### चरण:

1. **पहली स्लाइड प्राप्त करें**: अपनी प्रस्तुति की पहली स्लाइड तक पहुँचें.
2. **आयताकार आकार जोड़ें**: उपयोग `add_auto_shape` पाठ रखने के लिए एक आयत बनाने के लिए.

```python
# पहली स्लाइड प्राप्त करें
slide = pres.slides[0]

# स्लाइड में एक आयताकार आकार जोड़ें
rect = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 100, 100, 500, 150)
```

### पाठ सम्मिलित करना और प्रारूपित करना

#### अवलोकन

एक बार जब हमारा आकार तैयार हो जाता है, तो स्पष्टता और प्रभाव के लिए पाठ सम्मिलित करने और उसे प्रारूपित करने का समय आ जाता है।

#### चरण:

1. **टेक्स्ट फ़्रेम जोड़ें**: एक बनाने के `TextFrame` अपना पाठ रखने के लिए.
2. **ऑटो-फिट प्रकार**: सुनिश्चित करें कि पाठ स्वचालित रूप से आयत के भीतर फिट हो जाए।
3. **बॉर्डर हटाएँ**दृश्य स्पष्टता के लिए, आकृति की सीमा रेखाएं हटा दें.

```python
# आयत में टेक्स्टफ़्रेम जोड़ें
tf = rect.add_text_frame("This is first line \r\nThis is second line \r\nThis is third line")

# पाठ को स्वचालित रूप से आकृति में फ़िट होने के लिए सेट करें
tf.text_frame_format.autofit_type = slides.TextAutofitType.SHAPE

# दृश्य स्पष्टता के लिए आयत की सीमा रेखाएं हटाएँ
rect.line_format.fill_format.fill_type = slides.FillType.NONE
```

### बुलेट स्टाइल और इंडेंटेशन को अनुकूलित करना

#### अवलोकन

वास्तविक शक्ति बुलेट शैलियों को अनुकूलित करने और पैराग्राफ इंडेंटेशन को समायोजित करने में निहित है, ताकि आपकी सामग्री दृश्य रूप से आकर्षक बन सके।

#### चरण:

1. **बुलेट स्टाइल सेट करें**: प्रत्येक पैराग्राफ के लिए बुलेट का प्रकार और चरित्र परिभाषित करें।
2. **संरेखण और गहराई समायोजित करें**: पाठ संरेखित करें और पदानुक्रम के लिए गहराई स्तर निर्धारित करें।
3. **इंडेंटेशन परिभाषित करें**: विभिन्न रिक्तियों के लिए अलग-अलग इंडेंटेशन मान निर्दिष्ट करें।

```python
# पहले पैराग्राफ को प्रारूपित करें: बुलेट शैली, प्रतीक, संरेखण और इंडेंटेशन सेट करें
def format_paragraph(para, char, align, depth, indent):
    para.paragraph_format.bullet.type = slides.BulletType.SYMBOL
    para.paragraph_format.bullet.char = char
    para.paragraph_format.alignment = align
    para.paragraph_format.depth = depth
    para.paragraph_format.indent = indent

para1 = tf.paragraphs[0]
format_paragraph(para1, chr(8226), slides.TextAlignment.LEFT, 2, 30)

# दूसरे और तीसरे पैराग्राफ के लिए अलग-अलग इंडेंटेशन मानों के साथ दोहराएं
def format_multiple_paragraphs(paragraphs):
    for i, para in enumerate(paragraphs[1:], start=1):
        format_paragraph(para, chr(8226), slides.TextAlignment.LEFT, 4, 40 + i * 10)

format_multiple_paragraphs(tf.paragraphs)
```

# पायथन के लिए Aspose.Slides का उपयोग करके PowerPoint में छवियों के साथ आकृतियाँ कैसे भरें

## परिचय
चाहे आप व्यावसायिक पेशेवर हों या शिक्षक जो अपने दर्शकों को आकर्षित करना चाहते हैं, विज़ुअली आकर्षक पावरपॉइंट प्रेजेंटेशन बनाना बहुत ज़रूरी है। Aspose.Slides for Python का उपयोग करके अपनी स्लाइड्स को बेहतर बनाने का एक तरीका आकृतियों को छवियों से भरना है। यह सुविधा आपको अद्वितीय और रचनात्मक डिज़ाइन जोड़ने की अनुमति देती है जो आपकी सामग्री को अलग बना सकती है।

चाहे आप प्रोग्रामिंग प्रस्तुतियों के लिए नए हैं या दोहराए जाने वाले कार्यों को स्वचालित करने के तरीकों की तलाश कर रहे हैं, यह गाइड आपको दिखाएगा कि पायथन के लिए Aspose.Slides का उपयोग करके छवियों के साथ आकृतियों को प्रभावी ढंग से कैसे भरें।

**आप क्या सीखेंगे:**
- Aspose.Slides के साथ काम करने के लिए अपना वातावरण कैसे सेट करें
- पावरपॉइंट प्रेजेंटेशन में आकृतियों को छवियों से भरने की प्रक्रिया

## कार्यान्वयन मार्गदर्शिका
हम कार्यान्वयन को दो प्रमुख भागों में विभाजित करेंगे: आकृतियों को चित्रों से भरना और पावरपॉइंट प्रस्तुति को सहेजना। 

### चित्रों से आकृतियाँ भरना
यह सुविधा आपको विभिन्न आकृतियों के लिए छवियों का उपयोग करके अपनी स्लाइडों को बेहतर बनाने की अनुमति देती है, जिससे आपकी प्रस्तुतियों में एक पेशेवर स्पर्श या विषयगत एकरूपता आती है।

#### चरण 1: Aspose.Slides आयात करें
आवश्यक मॉड्यूल आयात करके प्रारंभ करें:

```python
import aspose.slides as slides
```

#### चरण 2: अपनी छवि पथ निर्धारित करें
इनपुट और आउटपुट दोनों निर्देशिकाओं के लिए पथ निर्दिष्ट करें:

```python
data_dir = "YOUR_DOCUMENT_DIRECTORY/"
out_dir = "YOUR_OUTPUT_DIRECTORY/"
```

प्रतिस्थापित करें `"YOUR_DOCUMENT_DIRECTORY/"` अपनी छवि स्रोत निर्देशिका पथ के साथ और `"YOUR_OUTPUT_DIRECTORY/"` अंतिम प्रस्तुति को आप कहां सहेजना चाहते हैं।

#### चरण 3: एक प्रेजेंटेशन इंस्टेंस बनाएं
उदाहरण प्रस्तुत करें `Presentation` क्लास, जो एक पावरपॉइंट फ़ाइल का प्रतिनिधित्व करता है:

```python
with slides.Presentation() as pres:
    slide = pres.slides[0]
```

यहाँ, हम प्रस्तुति की पहली स्लाइड तक पहुँचते हैं। आप अपनी आवश्यकताओं के आधार पर नई स्लाइड को संशोधित या जोड़ सकते हैं।

#### चरण 4: आकृतियाँ जोड़ें और कॉन्फ़िगर करें
स्लाइड में ऑटोशेप जोड़ें और उसका भरण प्रकार कॉन्फ़िगर करें:

```python
shape = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 50, 150, 75, 150)
shape.fill_format.fill_type = slides.FillType.PICTURE
```

यह कोड निर्दिष्ट निर्देशांक पर 75 चौड़ाई और 150 ऊंचाई के आयामों के साथ एक आयताकार आकार जोड़ता है।

#### चरण 5: चित्र भरण मोड सेट करें
परिभाषित करें कि छवि आकृति को कैसे भरेगी:

```python
shape.fill_format.picture_fill_format.picture_fill_mode = slides.PictureFillMode.TILE
```

का उपयोग करते हुए `TILE` यह मोड छवि को आकृति के संपूर्ण क्षेत्र में टाइल कर देता है, जिससे एक निर्बाध पैटर्न प्रभाव उत्पन्न होता है।

#### चरण 6: छवि लोड करें और असाइन करें
एक छवि लोड करें और उसे प्रस्तुति में जोड़ें:

```python
img = slides.Images.from_file(data_dir + "image2.jpg")
imgx = pres.images.add_image(img)
shape.fill_format.picture_fill_format.picture.image = imgx
```

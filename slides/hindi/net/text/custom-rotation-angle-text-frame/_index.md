# Aspose.Slides के साथ PowerPoint में चार्ट शीर्षक और लेबल घुमाना

Aspose.Slides for .NET, PowerPoint प्रस्तुतियों को प्रोग्रामेटिक रूप से बनाने और उनमें हेरफेर करने के लिए शक्तिशाली क्षमताएँ प्रदान करता है। यह उदाहरण दर्शाता है कि क्लस्टर किए गए कॉलम चार्ट के साथ एक प्रस्तुति कैसे बनाई जाए जिसमें घुमाए गए टेक्स्ट तत्व हों - विशेष रूप से, बेहतर पठनीयता और दृश्य अपील के लिए चार्ट शीर्षक और डेटा लेबल दोनों को घुमाना।

## स्थापना निर्देश

1. NuGet पैकेज मैनेजर के माध्यम से इंस्टॉल करें:
   ```
   Install-Package Aspose.Slides
   ```

2. या .NET CLI के माध्यम से स्थापित करें:
   ```
   dotnet add package Aspose.Slides
   ```

3. अपनी C# फ़ाइल में नामस्थान जोड़ें:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Charts;
   ```

# पावरपॉइंट प्रस्तुतियों में कस्टम बुलेट पॉइंट बनाना

Aspose.Slides for .NET प्रोग्रामेटिक रूप से PowerPoint प्रस्तुतियों को बनाने और उनमें हेरफेर करने के लिए मजबूत क्षमताएँ प्रदान करता है। यह उदाहरण विशेष रूप से टेक्स्ट फ़्रेम में कस्टम बुलेट पॉइंट्स को लागू करने पर केंद्रित है, जो बुलेट प्रकार, रंग, इंडेंटेशन और आकार को नियंत्रित करने का तरीका दिखाता है। कोड एक आयताकार आकार के साथ एक प्रस्तुति बनाता है जिसमें दो पैराग्राफ होते हैं - एक प्रतीक बुलेट के साथ और दूसरा क्रमांकित बुलेट शैली के साथ।

## स्थापना निर्देश

1. NuGet पैकेज मैनेजर का उपयोग करके .NET के लिए Aspose.Slides स्थापित करें:
   ```
   Install-Package Aspose.Slides
   ```

2. वैकल्पिक रूप से, पैकेज को सीधे डाउनलोड करें:
   - दौरा करना [Aspose.Slides डाउनलोड पृष्ठ](https://downloads.aspose.com/slides/net)
   - नवीनतम संस्करण डाउनलोड करें
   - अपने प्रोजेक्ट में Aspose.Slides.dll को निकालें और संदर्भित करें

3. आवश्यक using कथन जोड़ें:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

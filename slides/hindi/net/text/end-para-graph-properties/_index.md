# स्वरूपित पाठ के साथ पावरपॉइंट प्रस्तुतियाँ बनाना

Aspose.Slides for .NET एक शक्तिशाली API है जो डेवलपर्स को Microsoft PowerPoint की आवश्यकता के बिना PowerPoint प्रस्तुतियों को बनाने, हेरफेर करने और परिवर्तित करने की अनुमति देता है। यह उदाहरण दर्शाता है कि स्वरूपित पाठ पैराग्राफ के साथ एक आयताकार आकार वाली एक सरल प्रस्तुति कैसे बनाई जाए।

कोड एक नई प्रस्तुति बनाता है, पहली स्लाइड में एक आयताकार आकार जोड़ता है, और फिर अलग-अलग टेक्स्ट फ़ॉर्मेटिंग विकल्पों के साथ कई पैराग्राफ़ जोड़ता है। यह दिखाता है कि टेक्स्ट के विशिष्ट भागों के लिए आकार और परिवार जैसे फ़ॉन्ट गुण कैसे सेट करें।

## स्थापना निर्देश

1. NuGet पैकेज मैनेजर के माध्यम से इंस्टॉल करें:
   ```
   Install-Package Aspose.Slides
   ```

2. .NET CLI के माध्यम से स्थापित करें:
   ```
   dotnet add package Aspose.Slides
   ```

3. पैकेज प्रबंधक कंसोल के माध्यम से स्थापित करें:
   ```
   PM> Install-Package Aspose.Slides
   ```

4. अपने प्रोजेक्ट में संदर्भ जोड़ें:
   स्थापना के बाद, अपने प्रोजेक्ट में संदर्भ जोड़ें:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

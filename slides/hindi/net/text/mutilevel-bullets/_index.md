# Aspose.Slides के साथ PowerPoint में बहु-स्तरीय बुलेट पॉइंट बनाना

Aspose.Slides for .NET, PowerPoint प्रस्तुतियों को प्रोग्रामेटिक रूप से बनाने और उनमें हेरफेर करने के लिए एक शक्तिशाली API प्रदान करता है। यह उदाहरण दर्शाता है कि बहु-स्तरीय बुलेट पॉइंट के साथ एक प्रस्तुति कैसे बनाई जाती है, यह दर्शाता है कि टेक्स्ट फ़्रेम के साथ एक ऑटो शेप कैसे जोड़ा जाता है और विभिन्न बुलेट शैलियों और इंडेंटेशन स्तरों के साथ पैराग्राफ़ को फ़ॉर्मेट कैसे किया जाता है। आप प्रत्येक इंडेंटेशन स्तर के लिए अनुकूलित बुलेट प्रतीकों के साथ आसानी से पदानुक्रमित सामग्री बना सकते हैं, जो Microsoft PowerPoint को इंस्टॉल किए बिना संरचित और पेशेवर प्रस्तुतियाँ बनाने के लिए एकदम सही है।

## स्थापना निर्देश

1. NuGet पैकेज मैनेजर के माध्यम से इंस्टॉल करें:
   ```
   Install-Package Aspose.Slides
   ```

2. .NET CLI के माध्यम से स्थापित करें:
   ```
   dotnet add package Aspose.Slides
   ```

3. पैकेज प्रबंधक कंसोल के माध्यम से स्थापित करें:
   ```
   PM> Install-Package Aspose.Slides
   ```

4. मैनुअल स्थापना:
   - नवीनतम संस्करण यहाँ से डाउनलोड करें [.NET के लिए Aspose.Slides](https://releases.aspose.com/slides/net/)
   - अपने प्रोजेक्ट में Aspose.Slides.dll का संदर्भ जोड़ें

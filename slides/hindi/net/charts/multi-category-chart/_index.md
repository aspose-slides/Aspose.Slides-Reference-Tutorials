# .NET के लिए Aspose.Slides के साथ क्लस्टर्ड कॉलम चार्ट बनाना

Aspose.Slides for .NET प्रोग्रामेटिक रूप से PowerPoint प्रस्तुतियों को बनाने और उनमें हेरफेर करने के लिए शक्तिशाली क्षमताएँ प्रदान करता है। यह उदाहरण दर्शाता है कि कस्टम श्रेणियों और डेटा श्रृंखला के साथ क्लस्टर किए गए कॉलम चार्ट को कैसे बनाया जाए।

## इंस्टालेशन

```csharp
// NuGet पैकेज मैनेजर के माध्यम से इंस्टॉल करें
Install-Package Aspose.Slides
```

या अपनी प्रोजेक्ट फ़ाइल में निम्नलिखित संदर्भ जोड़ें:

```xml
<PackageReference Include="Aspose.Slides" Version="23.2.0" />
```

# Aspose.Slides के साथ PowerPoint में समूह आकृतियाँ बनाएँ

Aspose.Slides for .NET, PowerPoint प्रस्तुतियों को प्रोग्रामेटिक रूप से बनाने और उनमें हेरफेर करने के लिए व्यापक API प्रदान करता है। यह उदाहरण दर्शाता है कि समूह आकार कैसे बनाएं, उसके भीतर कई आयताकार आकृतियाँ कैसे जोड़ें, आकृतियों को उचित रूप से कैसे रखें, फ़्रेमिंग कैसे जोड़ें और प्रस्तुति को कैसे सेव करें। समूह आकृतियाँ संबंधित आकृतियों को व्यवस्थित करने के लिए आवश्यक हैं जिन्हें एक इकाई के रूप में हेरफेर करने की आवश्यकता होती है, जो आपके प्रस्तुति तत्वों पर अधिक संरचना और नियंत्रण प्रदान करती हैं।

## स्थापना निर्देश

1. इनमें से किसी एक विधि का उपयोग करके .NET के लिए Aspose.Slides स्थापित करें:

   - नुगेट पैकेज मैनेजर:
     ```
     Install-Package Aspose.Slides.NET
     ```

   - पैकेज प्रबंधक कंसोल:
     ```
     PM> Install-Package Aspose.Slides.NET
     ```

   - .नेट सीएलआई:
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. अपनी परियोजना में संदर्भ जोड़ें:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

3. लाइसेंस सेट करें (उत्पादन उपयोग के लिए वैकल्पिक):
   ```csharp
   // मूल्यांकन प्रतिबंध हटाने के लिए लाइसेंस सेट करें
   License license = new License();
   license.SetLicense("Aspose.Slides.lic");
   ```

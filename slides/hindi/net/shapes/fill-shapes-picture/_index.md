# Aspose.Slides का उपयोग करके PowerPoint में टाइल्ड इमेज फ़िल के साथ आकृतियाँ बनाना

Aspose.Slides for .NET, PowerPoint प्रस्तुतियों को प्रोग्रामेटिक रूप से बनाने और उनमें हेरफेर करने के लिए शक्तिशाली क्षमताएँ प्रदान करता है। यह उदाहरण दर्शाता है कि आयताकार आकार कैसे बनाया जाए और इसे टाइल वाली छवि पैटर्न से कैसे भरा जाए। टाइल वाली छवि भरना आपके प्रस्तुतियों के भीतर बनावट वाली पृष्ठभूमि, वॉटरमार्क या सजावटी तत्व बनाने के लिए विशेष रूप से उपयोगी है।

## स्थापना निर्देश

1. NuGet पैकेज मैनेजर के माध्यम से इंस्टॉल करें:
   ```
   Install-Package Aspose.Slides
   ```

2. .NET CLI के माध्यम से स्थापित करें:
   ```
   dotnet add package Aspose.Slides
   ```

3. मैनुअल स्थापना:
   - नवीनतम संस्करण यहाँ से डाउनलोड करें [.NET के लिए Aspose.Slides](https://releases.aspose.com/slides/net/)
   - अपने प्रोजेक्ट में Aspose.Slides.dll का संदर्भ जोड़ें

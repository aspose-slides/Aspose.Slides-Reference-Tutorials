# पावरपॉइंट में आयताकार आकृतियाँ बनाना और घुमाना

Aspose.Slides for .NET एक शक्तिशाली API है जो डेवलपर्स को Microsoft PowerPoint इंस्टॉलेशन की आवश्यकता के बिना PowerPoint प्रस्तुतियाँ बनाने, संशोधित करने और परिवर्तित करने की अनुमति देता है। यह उदाहरण दर्शाता है कि लाइब्रेरी की आकृति हेरफेर क्षमताओं को प्रदर्शित करते हुए, एक घुमावदार आयताकार आकार के साथ एक प्रस्तुति कैसे बनाई जाए।

Aspose.Slides के साथ, आप प्रोग्रामेटिक रूप से अपनी स्लाइड्स में विभिन्न आकृतियाँ जोड़ सकते हैं और रोटेशन जैसे परिवर्तन लागू कर सकते हैं। यह कार्यक्षमता विशेष रूप से दृश्य तत्वों पर सटीक नियंत्रण के साथ गतिशील प्रस्तुतियाँ बनाने के लिए उपयोगी है।

## स्थापना निर्देश

1. NuGet पैकेज मैनेजर के माध्यम से इंस्टॉल करें:
   ```
   Install-Package Aspose.Slides
   ```

2. .NET CLI के माध्यम से स्थापित करें:
   ```
   dotnet add package Aspose.Slides
   ```

3. अपने कोड में using निर्देश जोड़ें:
   ```csharp
   using Aspose.Slides;
   ```

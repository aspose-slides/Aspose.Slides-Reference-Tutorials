# पावरपॉइंट में कस्टम लाइन फ़ॉर्मेटिंग के साथ आयताकार आकार बनाना

Aspose.Slides for .NET प्रोग्रामेटिक रूप से PowerPoint प्रस्तुतियों को बनाने और संशोधित करने के लिए शक्तिशाली क्षमताएँ प्रदान करता है। यह उदाहरण दर्शाता है कि लाइन स्टाइल, चौड़ाई, डैश पैटर्न और रंग सेट करने सहित अनुकूलित लाइन फ़ॉर्मेटिंग के साथ एक आयताकार आकार कैसे बनाया जाए। नमूना कोड एक नई प्रस्तुति बनाता है, पहली स्लाइड में एक आयत जोड़ता है, और आकृति के भरण और रूपरेखा दोनों पर विशिष्ट स्वरूपण लागू करता है, जिसके परिणामस्वरूप एक सफ़ेद भरण और एक नीली धराशायी रूपरेखा के साथ एक आकर्षक तत्व बनता है।

## स्थापना निर्देश

1. निम्न विधियों में से किसी एक का उपयोग करके .NET के लिए Aspose.Slides स्थापित करें:

   - नुगेट पैकेज मैनेजर:
     ```
     Install-Package Aspose.Slides
     ```

   - पैकेज प्रबंधक कंसोल:
     ```
     PM> Install-Package Aspose.Slides
     ```

   - .नेट सीएलआई:
     ```
     dotnet add package Aspose.Slides
     ```

2. अपनी C# फ़ाइल में आवश्यक नामस्थान जोड़ें:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

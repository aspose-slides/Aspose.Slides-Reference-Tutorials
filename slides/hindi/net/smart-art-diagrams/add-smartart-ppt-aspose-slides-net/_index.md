**स्पष्टीकरण:** The `Presentation` क्लास को इंस्टैंशिएट किया जाता है, और निर्दिष्ट पथ का उपयोग करके एक खाली फ़ाइल को सहेजा जाता है।

### फ़ीचर: स्मार्टआर्ट आकार जोड़ें (H3)
**अवलोकन:** जानें कि अपनी प्रस्तुति की पहली स्लाइड में बेहतर दृश्य अपील के लिए स्मार्टआर्ट ग्राफ़िक कैसे जोड़ें।
```csharp
using Aspose.Slides;
using Aspose.Slides.SmartArt;

class FeatureAddSmartArtShape {
    public static void Run() {
        // एक नया प्रेजेंटेशन ऑब्जेक्ट आरंभ करें
        Presentation pres = new Presentation();

        // प्रस्तुति में पहली स्लाइड तक पहुँचें
        ISlide slide = pres.Slides[0];

        // स्लाइड में निर्दिष्ट स्थान और आकार पर स्मार्टआर्ट आकृति जोड़ें
        ISmartArt smart = slide.Shapes.AddSmartArt(50, 150, 400, 400, SmartArtLayoutType.StackedList);

        // प्रस्तुतीकरण को स्मार्टआर्ट के साथ सहेजें
        string outputDir = "/YOUR_OUTPUT_DIRECTORY";  // अपने वास्तविक पथ के साथ अपडेट करें
        pres.Save(outputDir + "PresentationWithSmartArt_out.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
    }
}
```
**स्पष्टीकरण:** यह कोड पहली स्लाइड तक पहुंचता है, एक जोड़ता है `StackedList` निर्दिष्ट निर्देशांक पर स्मार्टआर्ट ग्राफ़िक टाइप करें, और इसे सेव करें। अपने लेआउट में फिट करने के लिए स्थिति और आकार समायोजित करें।

### फ़ीचर: स्मार्टआर्ट (H3) में विशिष्ट स्थान पर नोड जोड़ें
**अवलोकन:** अपने मौजूदा स्मार्टआर्ट को उसके पदानुक्रम में सटीक स्थानों पर नोड्स जोड़कर उन्नत करें।
```csharp
using Aspose.Slides;
using Aspose.Slides.SmartArt;

class FeatureAddNodeToSmartArt {
    public static void Run() {
        // एक नया प्रेजेंटेशन ऑब्जेक्ट आरंभ करें
        Presentation pres = new Presentation();

        // प्रस्तुति में पहली स्लाइड तक पहुँचें
        ISlide slide = pres.Slides[0];

        // स्लाइड में निर्दिष्ट स्थान और आकार पर स्मार्टआर्ट आकृति जोड़ें
        ISmartArt smart = slide.Shapes.AddSmartArt(50, 150, 400, 400, SmartArtLayoutType.StackedList);

        // स्मार्टआर्ट के पहले नोड तक पहुँचना
        ISmartArtNode node = smart.AllNodes[0];

        // पैरेंट नोड के संतान संग्रह में स्थिति सूचकांक 2 पर एक नया संतान नोड जोड़ना
        SmartArtNode chNode = (SmartArtNode)((SmartArtNodeCollection)node.ChildNodes).AddNodeByPosition(2);

        // नए जोड़े गए नोड के लिए पाठ सेट करें
        chNode.TextFrame.Text = "Sample Text Added";

        // संशोधित स्मार्टआर्ट के साथ प्रस्तुति को सहेजें
        string outputDir = "/YOUR_OUTPUT_DIRECTORY";  // अपने वास्तविक पथ के साथ अपडेट करें
        pres.Save(outputDir + "ModifiedSmartArt_out.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
    }
}
```
**स्पष्टीकरण:** यह स्निपेट स्मार्टआर्ट ग्राफ़िक के भीतर नोड्स तक पहुँचने और उन्हें संशोधित करने का प्रदर्शन करता है। `AddNodeByPosition` यह विधि सटीक प्लेसमेंट की अनुमति देती है, जो संरचित सामग्री के लिए आवश्यक है।

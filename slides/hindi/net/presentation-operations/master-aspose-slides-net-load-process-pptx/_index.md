### दिशा गणना

**अवलोकन:** किसी रेखा की दिशा की गणना उसके आयाम और फ्लिप गुणों के आधार पर करें।

#### चरण 1: पैरामीटर परिभाषित करें

क्षैतिज या ऊर्ध्वाधर फ़्लिप को इंगित करने वाली चौड़ाई, ऊँचाई और बूलियन मान निर्दिष्ट करें:

```csharp
float width = /* आपका मूल्य */;
float height = /* आपका मूल्य */;
bool flipH = /* आपका बूलियन मान */;
bool flipV = /* आपका बूलियन मान */;
```

#### चरण 2: दिशा की गणना करें

रेखा और y-अक्ष के बीच का कोण निर्धारित करने के लिए आर्कटेन्जेन्ट फ़ंक्शन का उपयोग करें, फिर इसे सामान्य करें:

```csharp
class LineDirectionCalculator
{
    public static double CalculateDirection(float width, float height, bool flipH, bool flipV)
    {
        float endLineX = width * (flipH ? -1 : 1);
        float endLineY = height * (flipV ? -1 : 1);

        float endYAxisX = 0;
        float endYAxisY = height;

        double angle = (Math.Atan2(endYAxisY, endYAxisX) - Math.Atan2(endLineY, endLineX));

        if (angle < 0) angle += 2 * Math.PI;

        return angle * 180.0 / Math.PI;
    }
}
```

## कार्यान्वयन मार्गदर्शिका

### एम्बेडेड ऑडियो के साथ ऑडियो फ़्रेम जोड़ना

#### अवलोकन
निर्बाध देखने के अनुभव के लिए ऑडियो फ़ाइलों को सीधे अपनी प्रस्तुति स्लाइडों में एम्बेड करें।

#### चरण:
1. **प्रस्तुति आरंभ करें**
   एक नया बनाएँ `Presentation` स्लाइड और मीडिया को रखने के लिए ऑब्जेक्ट.
   ```csharp
   using Aspose.Slides;
   string mediaFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "audio.m4a");
   string outPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "AudioFrame_out.pptx");
   using (Presentation pres = new Presentation())
   ```
2. **संग्रह में ऑडियो जोड़ें**
   उपयोग `pres.Audios.AddAudio` अपनी ऑडियो फ़ाइल जोड़ने के लिए.
   ```csharp
   IAudio audio = pres.Audios.AddAudio(File.ReadAllBytes(mediaFile));
   ```
3. **ऑडियो फ़्रेम एम्बेड करें**
   पहली स्लाइड पर एक एम्बेडेड ऑडियो फ़्रेम जोड़ें.
   ```csharp
   IAudioFrame audioFrame = pres.Slides[0].Shapes.AddAudioFrameEmbedded(50, 50, 100, 100, audio);
   ```
4. **प्रस्तुति सहेजें**
   अपनी प्रस्तुति को एम्बेडेड ऑडियो फ्रेम के साथ सहेजें.
   ```csharp
   pres.Save(outPath, SaveFormat.Pptx);
   ```

### ऑडियो ट्रिमिंग समय सेट करना

#### अवलोकन
निर्दिष्ट करें कि प्रस्तुति में ऑडियो फ़ाइल का कौन सा भाग चलाया जाना चाहिए।

#### चरण:
1. **प्रस्तुति आरंभ करें**
   ऑडियो फ़्रेम जोड़ने के समान, एक नया फ़्रेम बनाकर आरंभ करें `Presentation` वस्तु।
   ```csharp
   using Aspose.Slides;
   string mediaFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "audio.m4a");
   string outPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "AudioFrameTrim_out.pptx");
   using (Presentation pres = new Presentation())
   ```
2. **ऑडियो जोड़ें और फ़्रेम एम्बेड करें**
   ऑडियो को संग्रह में जोड़ें और पहले की तरह इसे स्लाइड में एम्बेड करें।
   ```csharp
   IAudio audio = pres.Audios.AddAudio(File.ReadAllBytes(mediaFile));
   IAudioFrame audioFrame = pres.Slides[0].Shapes.AddAudioFrameEmbedded(50, 50, 100, 100, audio);
   ```
3. **ऑडियो आरंभ और अंत ट्रिम करें**
   अपनी ऑडियो क्लिप के लिए आरंभ और समाप्ति समय निर्धारित करें.
   ```csharp
   // 500ms (0.5 सेकंड) पर शुरू से ट्रिम करें
   audioFrame.TrimFromStart = 500f;
   
   // 1000ms (1 सेकंड) पर समाप्त करने के लिए ट्रिम करें
   audioFrame.TrimFromEnd = 1000f;
   ```
4. **प्रस्तुति सहेजें**
   अपनी प्रस्तुति को ट्रिम किए गए ऑडियो के साथ सहेजें.
   ```csharp
   pres.Save(outPath, SaveFormat.Pptx);
   ```

## कार्यान्वयन मार्गदर्शिका

अब जब आपने अपना परिवेश सेट कर लिया है, तो आइए छवि लोडिंग और प्रदर्शन कार्यक्षमता को क्रियान्वित करें।

### विशेषता: प्रस्तुति में छवियाँ लोड करना और प्रदर्शित करना

यह सुविधा दर्शाती है कि फ़ाइल सिस्टम से एक छवि को कैसे लोड किया जाए और इसे Aspose.Slides for .NET का उपयोग करके प्रस्तुति की पहली स्लाइड में चित्र फ़्रेम के रूप में कैसे जोड़ा जाए।

#### अवलोकन:
इस अनुभाग में, हम छवि लोड करने, उसे स्लाइड में सम्मिलित करने, तथा आपकी प्रस्तुति को सहेजने के चरणों पर चर्चा करेंगे।

**चरण 1: निर्देशिकाएँ बनाएँ**
अपने दस्तावेज़ निर्देशिका और आउटपुट निर्देशिका के लिए पथ परिभाषित करें। यदि वे मौजूद नहीं हैं, तो उन्हें निम्न का उपयोग करके बनाएँ:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // अपना दस्तावेज़ निर्देशिका पथ यहाँ परिभाषित करें
string outputDir = "YOUR_OUTPUT_DIRECTORY"; // अपना आउटपुट निर्देशिका पथ यहां परिभाषित करें

// यदि डेटा निर्देशिका मौजूद नहीं है तो उसे बनाएं।
if (!Directory.Exists(dataDir))
{
    Directory.CreateDirectory(dataDir);
}
```

**चरण 2: छवि लोड करें और डालें**
एक नया प्रेजेंटेशन इंस्टेंस बनाएं और इसकी पहली स्लाइड तक पहुँचें। फिर, फ़ाइल सिस्टम से एक छवि लोड करें:
```csharp
using (Presentation pres = new Presentation())
{
    // प्रस्तुति में पहली स्लाइड तक पहुँचें
    ISlide sld = pres.Slides[0];

    // फ़ाइल सिस्टम से एक छवि लोड करें और इसे प्रस्तुति के छवि संग्रह में जोड़ें
    IImage img = Images.FromFile(Path.Combine(dataDir, "aspose-logo.jpg"));
    IPPImage imgx = pres.Images.AddImage(img);

    // लोड की गई छवि से मेल खाते आयामों के साथ एक चित्र फ़्रेम जोड़ें
    sld.Shapes.AddPictureFrame(ShapeType.Rectangle, 50, 150, imgx.Width, imgx.Height, imgx);
}
```

**चरण 3: प्रस्तुति सहेजें**
अंत में, अपनी संशोधित प्रस्तुति को PPTX प्रारूप में डिस्क पर सहेजें:
```csharp
pres.Save(Path.Combine(outputDir, "AddStretchOffsetForImageFill_out.pptx"), SaveFormat.Pptx);
```

# .NET के लिए Aspose.Slides का उपयोग करके PowerPoint आकृतियों में भाषा कैसे सेट करें

डिजिटल प्रस्तुतियों की दुनिया में, यह सुनिश्चित करना कि आपकी सामग्री विभिन्न भाषाओं में सुलभ और सही ढंग से स्वरूपित है, एक चुनौती हो सकती है। .NET के लिए Aspose.Slides के साथ, आप PowerPoint स्लाइड में आकृतियों के भीतर पाठ के लिए आसानी से भाषा विशेषताएँ सेट कर सकते हैं। बहुभाषी दस्तावेज़ तैयार करते समय या वैश्विक संचार में स्थिरता सुनिश्चित करते समय यह सुविधा विशेष रूप से लाभकारी होती है।

**आप क्या सीखेंगे:**
- स्वचालित आकृतियाँ जोड़ना और उनमें पाठ सम्मिलित करना।
- Aspose.Slides का उपयोग करके पाठ भागों के लिए भाषा आईडी सेट करना।
- कस्टम कॉन्फ़िगरेशन के साथ प्रस्तुतियाँ सहेजना.

आइये देखें कि आप इस सुविधा को कैसे सहजता से क्रियान्वित कर सकते हैं।

## आवश्यक शर्तें

शुरू करने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित हैं:

- **पुस्तकालय और निर्भरताएँ**: आपके पास Aspose.Slides for .NET इंस्टॉल होना चाहिए। यह लाइब्रेरी C# में PowerPoint प्रेजेंटेशन को मैनिपुलेट करने के लिए आवश्यक है।
  
- **पर्यावरण सेटअप**: .NET कोर या .NET फ्रेमवर्क के साथ एक विकास वातावरण की आवश्यकता है।

- **ज्ञान पूर्वापेक्षाएँ**बुनियादी C# प्रोग्रामिंग अवधारणाओं से परिचित होना और ऑब्जेक्ट-ओरिएंटेड प्रोग्रामिंग सिद्धांतों की समझ उपयोगी होगी।

## .NET के लिए Aspose.Slides सेट अप करना

आरंभ करने के लिए, आपको Aspose.Slides लाइब्रेरी स्थापित करनी होगी। आप निम्न में से किसी एक विधि का उपयोग करके ऐसा कर सकते हैं:

**.NET सीएलआई**
```shell
dotnet add package Aspose.Slides
```

**पैकेज प्रबंधक**

अब जब हमने सेटअप कर लिया है, तो आइए आकृति पाठ के लिए भाषा सेट करने की सुविधा को क्रियान्वित करें।

### फ़ीचर अवलोकन: आकार पाठ भाषा सेट करना

यह सुविधा आपको PowerPoint आकृति के भीतर पाठ की भाषा निर्दिष्ट करने की अनुमति देती है। भाषा आईडी सेट करके, आप सुनिश्चित करते हैं कि वर्तनी जाँच और अन्य भाषा-विशिष्ट सुविधाएँ सही तरीके से लागू की गई हैं।

#### चरण 1: प्रस्तुति आरंभ करें

इसका एक उदाहरण बनाकर शुरू करें `Presentation` कक्षा।

```csharp
using (Presentation pres = new Presentation())
{
    // आपका कोड यहाँ
}
```

यह एक नया पावरपॉइंट प्रेजेंटेशन ऑब्जेक्ट आरंभ करता है, जिसे हम संशोधित करेंगे।

#### चरण 2: ऑटो आकार और टेक्स्ट फ़्रेम जोड़ें

अपनी स्लाइड में एक आयताकार आकार जोड़ें और उसमें पाठ डालें:

```csharp
IAutoShape shape = pres.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 50, 50, 200, 50);
shape.AddTextFrame("Text to apply spellcheck language");
```

यहाँ, `AddAutoShape` पहली स्लाइड में एक आयत जोड़ता है। पैरामीटर इसकी स्थिति और आकार को परिभाषित करते हैं।

#### चरण 3: भाषा आईडी सेट करें

आकृति के भीतर पाठ भाग के लिए भाषा सेट करें:

```csharp
shape.TextFrame.Paragraphs[0].Portions[0].PortionFormat.LanguageId = "en-EN";
```

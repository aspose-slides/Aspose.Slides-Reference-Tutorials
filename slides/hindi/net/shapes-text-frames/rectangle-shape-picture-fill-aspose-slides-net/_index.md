## कार्यान्वयन गाइड: चित्र भरण के साथ आयताकार आकार जोड़ें
अब जबकि हमारा वातावरण तैयार है, आइए एक छवि से भरा आयताकार आकार जोड़ने के लिए एक सुविधा को क्रियान्वित करें।

### फ़ीचर का अवलोकन
यह सुविधा दर्शाती है कि Aspose.Slides का उपयोग करके स्लाइड पर आयताकार आकार कैसे बनाया जाए और उसे छवि से कैसे भरा जाए। इस तकनीक का उपयोग लोगो, पृष्ठभूमि या किसी भी ग्राफ़िक तत्व को जोड़कर आपकी स्लाइड को बेहतर बनाने के लिए किया जा सकता है जो आपकी प्रस्तुति को और अधिक आकर्षक बनाता है।

### चरण-दर-चरण कार्यान्वयन
#### 1. प्रेजेंटेशन ऑब्जेक्ट को प्रारंभ करें
एक नया प्रेजेंटेशन ऑब्जेक्ट बनाकर शुरू करें। यह हमारे वर्किंग डॉक्यूमेंट के रूप में काम करेगा, जहाँ हम आकृतियाँ और अन्य तत्व जोड़ेंगे।
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // अपने दस्तावेज़ निर्देशिका पथ सेट करें
total slides count: pres.Slides.Count;
using (Presentation pres = new Presentation())
{
    ISlide firstSlide = pres.Slides[0]; // पहली स्लाइड पर पहुँचें

    // भरण के रूप में उपयोग करने के लिए एक छवि लोड करें
    IPPImage ppImage;
    using (IImage newImage = Aspose.Slides.Images.FromFile(Path.Combine(dataDir, "image.png")))
        ppImage = pres.Images.AddImage(newImage); // प्रस्तुति के चित्र संग्रह में चित्र जोड़ें

    // निर्दिष्ट आयामों के साथ एक आयताकार आकार जोड़ता है
    var newShape = firstSlide.Shapes.AddAutoShape(ShapeType.Rectangle, 0, 0, 350, 350);

    // आकृति का भरण प्रकार चित्र पर सेट करें
    newShape.FillFormat.FillType = FillType.Picture;
    IPictureFillFormat pictureFillFormat = newShape.FillFormat.PictureFillFormat;
    pictureFillFormat.Picture.Image = ppImage; // आयत के लिए लोड की गई छवि को भरण के रूप में निर्दिष्ट करें

    // प्रस्तुति सहेजें
    pres.Save(Path.Combine("YOUR_OUTPUT_DIRECTORY", "RectangleWithPictureFill.pptx"), SaveFormat.Pptx);
}
```
#### प्रमुख चरणों का स्पष्टीकरण:
- **छवि लोड हो रही है**: द `FromFile` विधि आपके निर्दिष्ट निर्देशिका से एक छवि लोड करती है, जिसे फिर प्रस्तुति के छवि संग्रह में जोड़ दिया जाता है।
  
- **आयताकार आकार जोड़ना**: हम उपयोग करते हैं `AddAutoShape` साथ `ShapeType.Rectangle` और इसके आयाम निर्धारित करें। इससे स्लाइड पर एक आयत बन जाती है।

- **चित्र भरण सेटिंग**: असाइन करके `FillType.Picture` आकृति के भरण प्रारूप में, हम आयत को एक छवि कंटेनर में बदल देते हैं। लोड की गई तस्वीर को फिर इस भरण के रूप में सेट किया जाता है `Picture.Image` संपत्ति।

## अक्सर पूछे जाने वाले प्रश्न अनुभाग
**प्रश्न 1: क्या मैं पुराने संस्करणों में बनाई गई PowerPoint फ़ाइलों के साथ इस सुविधा का उपयोग कर सकता हूँ?**
हां, Aspose.Slides PowerPoint प्रारूपों की एक विस्तृत श्रृंखला का समर्थन करता है और पश्चगामी संगतता सुनिश्चित करता है।

**प्रश्न 2: मैं रनटाइम के दौरान गतिशील रूप से छवि भरण कैसे बदल सकता हूँ?**
आप अपडेट कर सकते हैं `Picture.Image` आवश्यकतानुसार भरण छवि को बदलने के लिए रनटाइम पर संपत्ति का उपयोग करें।

**प्रश्न 3: क्या एक आकृति के भीतर टाइल पैटर्न में एकाधिक छवियों को लागू करना संभव है?**
हाँ, सेट करके `TileOffsetX`, `TileOffsetY`, और अन्य टाइलिंग गुण `IPictureFillFormat`.

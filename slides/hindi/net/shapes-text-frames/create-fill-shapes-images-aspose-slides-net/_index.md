## कार्यान्वयन मार्गदर्शिका

यह अनुभाग प्रक्रिया को प्रबंधनीय विशेषताओं में विभाजित करता है।

### निर्देशिकाएँ बनाना

यह सुनिश्चित करने के लिए कि हमारी प्रस्तुति फ़ाइलें सही ढंग से सहेजी गई हैं, हम पहले जाँचते हैं कि लक्ष्य निर्देशिका मौजूद है या नहीं। यदि नहीं, तो हम इसे बनाते हैं:
```csharp
using System.IO;

string dataDir = "YOUR_DOCUMENT_DIRECTORY";
bool isExists = Directory.Exists(dataDir);
if (!isExists)
{
    // यदि निर्देशिका मौजूद नहीं है तो उसे बनाएं
    Directory.CreateDirectory(dataDir);
}
```

### प्रस्तुतियों के साथ कार्य करना

हम एक प्रस्तुति का उदाहरण बनाकर शुरू करते हैं और फिर उसकी स्लाइडों में बदलाव करते हैं:
```csharp
using Aspose.Slides;

// PPTX फ़ाइल का प्रतिनिधित्व करने वाले प्रेजेंटेशन क्लास को इंस्टेंटिएट करें
using (Presentation pres = new Presentation())
{
    // प्रस्तुति से पहली स्लाइड प्राप्त करें
    ISlide sld = pres.Slides[0];

    // स्लाइड में आयत प्रकार का ऑटोशेप जोड़ें
    IShape shp = sld.Shapes.AddAutoShape(ShapeType.Rectangle, 50, 150, 75, 150);
}
```

### चित्र के साथ आकार भरें सेटिंग

इसके बाद, हम आकृति का भरण प्रकार निर्धारित करके उसे छवि से भरते हैं:
```csharp
using Aspose.Slides;
using System.Drawing;

// आकृति का भरण प्रकार चित्र पर सेट करें
shp.FillFormat.FillType = FillType.Picture;
// चित्र भरण मोड को टाइल के रूप में कॉन्फ़िगर करें
shp.FillFormat.PictureFillFormat.PictureFillMode = PictureFillMode.Tile;

// किसी निर्दिष्ट निर्देशिका से छवि लोड करें और उसे आकृति के भरण प्रारूप में सेट करें
IImage img = Images.FromFile("YOUR_DOCUMENT_DIRECTORY/Tulips.jpg");
IPPImage imgx = pres.Images.AddImage(img);
shp.FillFormat.PictureFillFormat.Picture.Image = imgx;
```

### प्रस्तुतियाँ सहेजना

अंत में, अपनी प्रस्तुति को सभी परिवर्तनों के साथ सहेजें:
```csharp
using Aspose.Slides.Export;

// संशोधित प्रस्तुति को डिस्क पर वापस सहेजें
pres.Save("YOUR_OUTPUT_DIRECTORY/RectShpPic_out.pptx", SaveFormat.Pptx);
```

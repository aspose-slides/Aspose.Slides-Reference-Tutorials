# .NET के लिए Aspose.Slides का उपयोग करके PowerPoint में आयत आकार कैसे बनाएं और प्रारूपित करें
## परिचय
दिखने में आकर्षक प्रस्तुतियाँ बनाना आपके संदेश के प्रभाव को काफी हद तक बढ़ा सकता है, चाहे आप कोई व्यावसायिक पिच दे रहे हों या जटिल डेटा प्रस्तुत कर रहे हों। अपनी स्लाइड्स को अलग दिखाने का एक तरीका सटीक फ़ॉर्मेटिंग के साथ कस्टम आकृतियों को शामिल करना है - जैसे आयताकार जो अपने रंग और बॉर्डर स्टाइलिंग से ध्यान आकर्षित करते हैं।

**आप क्या सीखेंगे:**
- .NET के लिए Aspose.Slides के साथ अपना वातावरण कैसे सेट करें।
- कोड का उपयोग करके PowerPoint में एक आयताकार आकार बनाने की प्रक्रिया।

## कार्यान्वयन मार्गदर्शिका
### एक आयताकार आकार बनाना
हमारा लक्ष्य पहली स्लाइड पर एक आयताकार आकार बनाना है। आइए चरणों को तोड़ें:
#### चरण 1: प्रस्तुति आरंभ करें
Aspose.Slides के साथ अपना परिवेश सेट अप करके और एक नया प्रस्तुति ऑब्जेक्ट बनाकर आरंभ करें।
```csharp
using System;
using Aspose.Slides;
using System.Drawing;

string dataDir = "YOUR_DOCUMENT_DIRECTORY";
bool isExists = System.IO.Directory.Exists(dataDir);
if (!isExists)
    System.IO.Directory.CreateDirectory(dataDir);

using (Presentation pres = new Presentation())
{
    // कोड जारी है...
}
```
*स्पष्टीकरण:* यह कोड एक नई पावरपॉइंट प्रस्तुति आरंभ करता है और यह सुनिश्चित करता है कि फ़ाइलों को सहेजने के लिए निर्देशिका मौजूद है।
#### चरण 2: पहली स्लाइड तक पहुंचें
पहली स्लाइड पर पहुँचें जहाँ हम अपना आयत जोड़ेंगे।
```csharp
ISlide sld = pres.Slides[0];
```
*स्पष्टीकरण:* हम कार्य करने के लिए प्रस्तुति से पहली स्लाइड प्राप्त करते हैं।
#### चरण 3: एक आयताकार आकार जोड़ें
स्लाइड में आयत प्रकार का एक स्वचालित आकार जोड़ें.
```csharp
IShape shp = sld.Shapes.AddAutoShape(ShapeType.Rectangle, 50, 150, 150, 50);
```
*स्पष्टीकरण:* यह स्थिति (50, 150) पर 150x50 आयामों के साथ एक आयत बनाता है। पैरामीटर आकार के प्रकार और उसके स्थान/आकार को परिभाषित करते हैं।
### आयत को प्रारूपित करना
अब जबकि हमारे पास आयत है, तो चलिए इसमें कुछ स्टाइलिंग लागू करते हैं।
#### चरण 4: ठोस भरण रंग लागू करें
आयत के मुख्य भाग के लिए एक ठोस भरण रंग सेट करें.
```csharp
shp.FillFormat.FillType = FillType.Solid;
shp.FillFormat.SolidFillColor.Color = Color.Chocolate;
```
*स्पष्टीकरण:* यहां, हम आयत के अंदरूनी भाग को चॉकलेटी भूरे रंग में बदल रहे हैं।
#### चरण 5: बॉर्डर लाइन फ़ॉर्मेटिंग लागू करें
ठोस भरण के साथ बॉर्डर को अनुकूलित करें और इसकी चौड़ाई समायोजित करें।
```csharp
shp.LineFormat.FillFormat.FillType = FillType.Solid;
shp.LineFormat.FillFormat.SolidFillColor.Color = Color.Black;
shp.LineFormat.Width = 5;
```
*स्पष्टीकरण:* आयत की सीमा काले रंग में सेट की गई है, तथा रेखा की चौड़ाई 5 पिक्सेल है।
### प्रस्तुति को सहेजना
अंत में, अपने परिवर्तनों को एक फ़ाइल में सहेजें।
```csharp
pres.Save(dataDir + "/RectShp2_out.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
```
*स्पष्टीकरण:* यह प्रस्तुति को नए स्वरूपित आयत आकार के साथ आपकी निर्दिष्ट निर्देशिका में सहेज देता है।

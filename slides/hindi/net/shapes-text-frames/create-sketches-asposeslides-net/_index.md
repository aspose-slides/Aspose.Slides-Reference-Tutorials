## कार्यान्वयन मार्गदर्शिका

सब कुछ सेट हो जाने के बाद, आइए स्केच्ड आकार सुविधा को क्रियान्वित करें।

### आकृतियाँ जोड़ना और संशोधित करना

#### अवलोकन

इस अनुभाग में, हम स्लाइड पर आयत प्रकार का एक ऑटोशेप जोड़ेंगे और एक स्केच प्रभाव बनाने के लिए इसके गुणों को कॉन्फ़िगर करेंगे।

**आयताकार आकार जोड़ना**

एक नया प्रस्तुतिकरण उदाहरण बनाकर और एक आयताकार आकार जोड़कर आरंभ करें:
```csharp
using Aspose.Slides;
using Aspose.Slides.Export;

string outPptxFile = Path.Combine("YOUR_OUTPUT_DIRECTORY", "SketchedShapes_out.pptx");
string outPngFile = Path.Combine("YOUR_OUTPUT_DIRECTORY", "SketchedShapes_out.png");

using (Presentation pres = new Presentation())
{
    // पहली स्लाइड पर Rectangle प्रकार का एक AutoShape जोड़ें
    IAutoShape shape = pres.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 20, 20, 300, 150);
}
```

#### भरण प्रारूप सेट करना

इसे रेखाचित्रित रूप देने के लिए, आकृति से किसी भी भराव को हटा दें:
```csharp
shape.FillFormat.FillType = FillType.NoFill;
```

### आकृतियों पर स्केच प्रभाव लागू करना

#### अवलोकन

इसके बाद, आयत को मुक्तहस्त शैली के रेखाचित्र में परिवर्तित करें।

**आकृति को रेखाचित्र में बदलना**

उपयोग `SketchFormat` स्क्रिबल प्रभाव लागू करने के लिए संपत्ति:
```csharp
// आकृति को मुक्तहस्त शैली (स्क्रिबल) के रेखाचित्र में रूपांतरित करें
shape.LineFormat.SketchFormat.SketchType = LineSketchType.Scribble;
```

### प्रस्तुतियाँ और छवियाँ सहेजना

अंत में, अपने कार्य को एक प्रस्तुति फ़ाइल और एक छवि दोनों के रूप में सहेजें।

**PPTX के रूप में सहेजा जा रहा है**
```csharp
// प्रस्तुति को PPTX फ़ाइल में सहेजें
pres.Save(outPptxFile, SaveFormat.Pptx);
```

**PNG छवि के रूप में सहेजना**
```csharp
// स्लाइड को PNG प्रारूप में छवि फ़ाइल के रूप में सहेजें
pres.Slides[0].GetThumbnail(4/3f, 4/3f).Save(outPngFile, System.Drawing.Imaging.ImageFormat.Png);
```

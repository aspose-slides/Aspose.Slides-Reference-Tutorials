# .NET के लिए Aspose.Slides के साथ PowerPoint में ऑटोशेप्स बनाना और फ़ॉर्मेट करना: एक चरण-दर-चरण मार्गदर्शिका

## परिचय

आकर्षक पावरपॉइंट प्रेजेंटेशन बनाना समय लेने वाला और जटिल दोनों हो सकता है, खासकर तब जब आपको प्रोग्रामेटिक रूप से आकृतियाँ जोड़ने और उनमें टेक्स्ट को फ़ॉर्मेट करने की ज़रूरत हो। .NET के लिए Aspose.Slides दर्ज करें—एक शक्तिशाली लाइब्रेरी जो आपके .NET अनुप्रयोगों में पावरपॉइंट फ़ाइलों में हेरफेर करने की प्रक्रिया को सरल बनाती है। इस ट्यूटोरियल में, हम Aspose.Slides का उपयोग करके ऑटोशेप बनाने और उसके टेक्स्टफ़्रेम को फ़ॉर्मेट करने का तरीका जानेंगे।

**आप क्या सीखेंगे:**
- स्लाइड में आयताकार आकार कैसे जोड़ें।
- ऑटोशेप के भीतर पाठ को प्रारूपित करना।
- आकृतियों और पाठों के लिए प्रमुख कॉन्फ़िगरेशन विकल्प.
- अपनी परियोजनाओं में इन सुविधाओं का व्यावहारिक अनुप्रयोग।

आइए कोड कार्यान्वयन में आगे बढ़ने से पहले आवश्यक पूर्वापेक्षाओं को कवर करके शुरुआत करें।

## कार्यान्वयन मार्गदर्शिका

### फ़ीचर 1: स्लाइड में ऑटोशेप बनाएँ और जोड़ें

#### अवलोकन

यह अनुभाग दर्शाता है कि प्रस्तुतिकरण कैसे बनाएं, स्लाइड तक कैसे पहुंचें, तथा आयत प्रकार का ऑटोशेप कैसे जोड़ें।

#### चरण:

**स्टेप 1**प्रस्तुति आरंभ करें
```csharp
// प्रेजेंटेशन क्लास का एक उदाहरण बनाएँ
tPresentation presentation = new tPresentation();
```

**चरण दो**: पहली स्लाइड तक पहुंचें
```csharp
// पहली स्लाइड पर पहुँचें
tISlide slide = presentation.Slides[0];
```

**चरण 3**: आयत ऑटोशेप जोड़ें
```csharp
// स्थिति (150, 75) पर आयत प्रकार का एक ऑटोशेप जोड़ें जिसका आकार (350, 350) हो
tIAutoShape ashp = slide.Shapes.AddAutoShape(tShapeType.Rectangle, 150, 75, 350, 350);
```

**चरण 4**: प्रस्तुति सहेजें
```csharp
// प्रस्तुति को निर्दिष्ट निर्देशिका में सहेजें presentation.Save("YOUR_OUTPUT_DIRECTORY/formatText_out.pptx", tSaveFormat.Pptx);
```

### फ़ीचर 2: ऑटोशेप में टेक्स्टफ़्रेम जोड़ें और फ़ॉर्मेट करें

#### अवलोकन

यह सुविधा बताती है कि किसी मौजूदा ऑटोशेप में टेक्स्टफ्रेम कैसे जोड़ें, ऑटोफिट विकल्प कॉन्फ़िगर कैसे करें, और टेक्स्ट गुण कैसे सेट करें।

#### चरण:

**स्टेप 1**: टेक्स्टफ्रेम जोड़ें
```csharp
// मान लें कि 'ashp' पिछले ऑपरेशन से एक IAutoShape उदाहरण है
// आयत में टेक्स्टफ़्रेम जोड़ें
tashp.AddTextFrame(" ");
```

**चरण दो**: ऑटोफिट प्रकार कॉन्फ़िगर करें
```csharp
// आकृति के भीतर बेहतर पाठ संरेखण के लिए ऑटोफिट प्रकार सेट करें
tITextFrame txtFrame = ashp.TextFrame;
txtFrame.TextFrameFormat.AutofitType = tTextAutofitType.Shape;
```

**चरण 3**: टेक्स्ट को फॉर्मेट करें और डालें
```csharp
// पैराग्राफ़ ऑब्जेक्ट बनाएँ और सामग्री सेट करें
tIParagraph para = txtFrame.Paragraphs[0];
tIPortion portion = para.Portions[0];

portion.Text = "A quick brown fox jumps over the lazy dog. A quick brown fox jumps over the lazy dog.";
portion.PortionFormat.FillFormat.FillType = tFillType.Solid;
portion.PortionFormat.FillFormat.SolidFillColor.Color = tColor.Black;
```

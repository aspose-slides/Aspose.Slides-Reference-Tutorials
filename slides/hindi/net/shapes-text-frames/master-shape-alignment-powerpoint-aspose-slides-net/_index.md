## कार्यान्वयन मार्गदर्शिका

आइए जानें कि .NET के लिए Aspose.Slides का उपयोग करके आकार संरेखण सुविधाओं को कैसे लागू किया जाए।

### स्लाइड में आकृतियाँ संरेखित करें (H2)
यह सुविधा संपूर्ण स्लाइड में आकृतियों को संरेखित करने का प्रदर्शन करती है। आप इसे कैसे प्राप्त कर सकते हैं, यहाँ बताया गया है:

#### चरण 1: आकृतियाँ बनाएँ और जोड़ें
अपनी स्लाइड में प्लेसहोल्डर के रूप में कुछ आयत जोड़ें:

```csharp
ISlide slide = pres.Slides[0];
slide.Shapes.AddAutoShape(ShapeType.Rectangle, 100, 100, 100, 100);
slide.Shapes.AddAutoShape(ShapeType.Rectangle, 200, 200, 100, 100);
slide.Shapes.AddAutoShape(ShapeType.Rectangle, 300, 300, 100, 100);
```

#### चरण 2: आकृतियाँ संरेखित करें
उपयोग `AlignShapes` इन आकृतियों को नीचे संरेखित करने की विधि:

```csharp
SlideUtil.AlignShapes(ShapesAlignmentType.AlignBottom, true, pres.Slides[0]);
```
**स्पष्टीकरण:** पैरामीटर संरेखण प्रकार को परिभाषित करते हैं (`AlignBottom`), क्या पाठ शामिल करना है (`true`), और लक्ष्य स्लाइड.

#### चरण 3: प्रस्तुति सहेजें
अपने परिवर्तनों को एक नई फ़ाइल में सहेजें:

```csharp
pres.Save("ShapesAlignment_out.pptx", SaveFormat.Pptx);
```

### ग्रुपशेप (H2) में आकृतियों को संरेखित करें
यह अनुभाग दिखाता है कि समूह आकृति के भीतर आकृतियों को कैसे संरेखित किया जाए, ताकि सुसंगत संरेखण सुनिश्चित हो सके।

#### चरण 1: समूह आकार बनाएं और आकार जोड़ें
अपनी आकृतियों को एक नए समूह में जोड़ें:

```csharp
ISlide slide = pres.Slides.AddEmptySlide(pres.Slides[0].LayoutSlide);
IGroupShape groupShape = slide.Shapes.AddGroupShape();
groupShape.Shapes.AddAutoShape(ShapeType.Rectangle, 350, 50, 50, 50);
// आवश्यकतानुसार और आकृतियाँ जोड़ें
```

#### चरण 2: समूह के भीतर आकृतियों को संरेखित करें
इन सभी आकृतियों को उनके समूह में बाईं ओर संरेखित करें:

```csharp
SlideUtil.AlignShapes(ShapesAlignmentType.AlignLeft, false, groupShape);
```

### ग्रुपशेप (H2) में विशिष्ट आकृतियों को संरेखित करें
आप इंडेक्स का उपयोग करके संरेखण के लिए विशिष्ट आकृतियों को भी लक्षित कर सकते हैं।

#### चरण 1: अपना समूह आकार निर्धारित करें
पिछले अनुभाग के समान, अपना समूह बनाएं और आकृतियाँ जोड़ें:

```csharp
IGroupShape groupShape = slide.Shapes.AddGroupShape();
groupShape.Shapes.AddAutoShape(ShapeType.Rectangle, 350, 50, 50, 50);
// अतिरिक्त आकृतियाँ...
```

#### चरण 2: विशिष्ट आकृतियों को संरेखित करें
कौन सी आकृतियों को संरेखित करना है यह निर्दिष्ट करने के लिए अनुक्रमणिकाओं का उपयोग करें:

```csharp
SlideUtil.AlignShapes(ShapesAlignmentType.AlignLeft, false, groupShape, new int[] { 0, 2 });
```
**स्पष्टीकरण:** यह समूह के भीतर केवल पहली और तीसरी आकृतियों को संरेखित करता है।

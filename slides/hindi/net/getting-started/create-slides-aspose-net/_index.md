## कार्यान्वयन मार्गदर्शिका

### स्लाइड बनाना और कॉन्फ़िगर करना

#### अवलोकन

यह अनुभाग आपको स्लाइड बनाने, आकृतियाँ जोड़ने और प्रस्तुति को सहेजने के बारे में बताएगा।

1. **प्रस्तुति आरंभ करें**
   अपनी कार्यशील निर्देशिका को सेट करके और प्रारंभ करके प्रारंभ करें `Presentation` कक्षा:
    
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";

if (!Directory.Exists(dataDir))
    Directory.CreateDirectory(dataDir);
    
Presentation pres = new Presentation();
```

2. **एक आयताकार आकार जोड़ें**
   अपनी स्लाइड में एक आकृति जोड़ें जहां आप बाद में पाठ रख सकें।
    
```csharp
ISlide sld = pres.Slides[0];
IAutoShape rect = sld.Shapes.AddAutoShape(ShapeType.Rectangle, 100, 100, 500, 150);
```

3. **प्रस्तुति सहेजें**
   अपना कार्य डिस्क पर सहेजें:
    
```csharp
pres.Save(dataDir + "/CreatedSlide.pptx", SaveFormat.Pptx);
```

### आकृति में पाठ जोड़ना और प्रारूपित करना

#### अवलोकन
यहां, हम अपनी आकृति में पाठ जोड़ेंगे और उसका स्वरूप कॉन्फ़िगर करेंगे।

1. **टेक्स्टफ़्रेम जोड़ें**
   एम्बेड करें `TextFrame` आपके द्वारा बनाए गए आयत के भीतर:
    
```csharp
ITextFrame tf = rect.AddTextFrame("This is first line \rThis is second line \rThis is third line");
```

2. **ऑटोफिट प्रकार सेट करें**
   सुनिश्चित करें कि पाठ आकृति की सीमाओं के भीतर फिट बैठता है:
    
```csharp
tf.TextFrameFormat.AutofitType = TextAutofitType.Shape;
```

3. **आकृति रेखाएं छिपाएं**
   वैकल्पिक रूप से, साफ़-सुथरे लुक के लिए आयताकार रेखाओं को छिपाएँ:
    
```csharp
rect.LineFormat.FillFormat.FillType = FillType.NoFill; // कोई रेखा न दिखने के कारण NoFill में परिवर्तित किया गया
```

4. **प्रस्तुति सहेजें**
   अपने परिवर्तन सहेजें:
    
```csharp
pres.Save(dataDir + "/TextFormattedSlide.pptx", SaveFormat.Pptx);
```

### पैराग्राफ इंडेंट और बुलेट स्टाइल कॉन्फ़िगर करना

#### अवलोकन
अब, आइए अपने पैराग्राफ को बुलेट पॉइंट्स और इंडेंटेशन के साथ फॉर्मेट करें।

1. **पैराग्राफ़ के लिए बुलेट और संरेखण सेट करें**
   प्रत्येक पैराग्राफ़ को बुलेट पॉइंट प्रदर्शित करने के लिए कॉन्फ़िगर करें:
    
```csharp
foreach (IParagraph para in tf.Paragraphs)
{
    para.ParagraphFormat.Bullet.Type = BulletType.Symbol;
    para.ParagraphFormat.Bullet.Char = Convert.ToChar(8226);
    para.ParagraphFormat.Alignment = TextAlignment.Left;

    // पैराग्राफ इंडेक्स के आधार पर गहराई और इंडेंट सेट करें
    para.ParagraphFormat.Depth = 2; 
    para.ParagraphFormat.Indent = 30 + (tf.Paragraphs.IndexOf(para) * 10);
}
```

2. **प्रस्तुति सहेजें**
   अपने परिवर्तनों को अंतिम रूप दें:
    
```csharp
pres.Save(dataDir + "/IndentedTextSlide.pptx", SaveFormat.Pptx);
```

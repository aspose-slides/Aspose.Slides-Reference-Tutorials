## कार्यान्वयन मार्गदर्शिका

आइए किसी आकृति पर मैक्रो हाइपरलिंक सेट करने की प्रक्रिया देखें।

### सुविधा अवलोकन: मैक्रो हाइपरलिंक सेट करना

यह सुविधा आपको .NET के लिए Aspose.Slides का उपयोग करके PowerPoint में आकृतियों में मैक्रो फ़ंक्शन संलग्न करने की अनुमति देती है, जो उपयोगकर्ता इनपुट पर प्रतिक्रिया देने वाली इंटरैक्टिव प्रस्तुतियाँ बनाने के लिए आदर्श है।

#### चरण 1: आकृति बनाएं

अपनी स्लाइड में स्वचालित आकार जोड़ें:

```csharp
using Aspose.Slides;

string macroName = "TestMacro";
using (Presentation presentation = new Presentation())
{
    // स्थिति (20, 20) पर (80x30) आयामों के साथ एक खाली बटन आकार जोड़ें
    IAutoShape shape = presentation.Slides[0].Shapes.AddAutoShape(ShapeType.BlankButton, 20, 20, 80, 30);
```

#### चरण 2: मैक्रो हाइपरलिंक सेट करें

इस आकृति में मैक्रो संलग्न करें:

```csharp
    // आकृति को मैक्रो हाइपरलिंक क्लिक इवेंट से संबद्ध करें
    shape.HyperlinkManager.SetMacroHyperlinkClick(macroName);

    // प्रस्तुति सहेजें
    presentation.Save("output.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
}
```
**स्पष्टीकरण:**
- `AddAutoShape(ShapeType.BlankButton, 20, 20, 80, 30)`: निर्दिष्ट निर्देशांक और आकार पर एक रिक्त बटन आकार जोड़ता है।
- `SetMacroHyperlinkClick(macroName)`: मैक्रो को आकृति के क्लिक इवेंट से लिंक करता है.

#### समस्या निवारण युक्तियों

- **मैक्रो नहीं चल रहा है:** सुनिश्चित करें कि मैक्रो आपके पावरपॉइंट टेम्पलेट में मौजूद है।
- **आकार स्थिति संबंधी मुद्दे:** स्लाइड पर सटीक स्थान के लिए निर्देशांक मानों की दोबारा जांच करें।

## व्यावहारिक अनुप्रयोगों

आकृतियों के साथ मैक्रोज़ को एकीकृत करने से विभिन्न उद्देश्य पूरे हो सकते हैं:
1. **स्वचालित डेटा प्रविष्टि**बटन क्लिक द्वारा ट्रिगर किए गए मैक्रोज़ डेटा प्रविष्टि या फ़ॉर्मेटिंग जैसे दोहराए जाने वाले कार्यों को स्वचालित कर सकते हैं।
2. **इंटरैक्टिव क्विज़**: प्रश्नोत्तरी प्रतिक्रियाओं के आधार पर स्लाइडों के बीच नेविगेट करने के लिए मैक्रोज़ का उपयोग करें, जिससे उपयोगकर्ता की सहभागिता बढ़े।
3. **कस्टम नेविगेशन**: कस्टम बटन बनाएं जो स्लाइड डेक के भीतर विशिष्ट प्रस्तुतियों या अनुभागों को ट्रिगर करते हैं।

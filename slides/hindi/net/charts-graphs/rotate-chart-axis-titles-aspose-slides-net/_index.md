## कार्यान्वयन मार्गदर्शिका
यह मार्गदर्शिका आपको .NET के लिए Aspose.Slides का उपयोग करके चार्ट अक्ष शीर्षक के घूर्णन कोण को सेट करने में मार्गदर्शन करेगी।
### फ़ीचर अवलोकन: चार्ट अक्ष शीर्षक का रोटेशन कोण सेट करना
रोटेशन एंगल को एडजस्ट करने से पठनीयता और सौंदर्यबोध में वृद्धि हो सकती है, खास तौर पर सीमित स्थान वाली स्लाइड्स में। इस सुविधा को लागू करने का तरीका इस प्रकार है:
#### चरण 1: एक प्रस्तुति बनाएं और एक चार्ट जोड़ें
एक नया प्रस्तुतीकरण बनाकर और एक क्लस्टर कॉलम चार्ट जोड़कर आरंभ करें।
```csharp
using Aspose.Slides;

string dataDir = "YOUR_DOCUMENT_DIRECTORY";

// एक नया प्रस्तुतिकरण उदाहरण आरंभ करें.
using (Presentation pres = new Presentation())
{
    // प्रथम स्लाइड में स्थिति (50, 50) पर 450 चौड़ाई और 300 ऊंचाई वाला एक क्लस्टर कॉलम चार्ट जोड़ें।
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 50, 50, 450, 300);
```
#### चरण 2: वर्टिकल अक्ष शीर्षक सक्षम करें
इसके स्वरूप को अनुकूलित करने के लिए ऊर्ध्वाधर अक्ष शीर्षक को सक्षम करें.
```csharp
    // चार्ट के लिए ऊर्ध्वाधर अक्ष शीर्षक सक्षम करें.
    chart.Axes.VerticalAxis.HasTitle = true;
```
#### चरण 3: रोटेशन कोण सेट करें
ऊर्ध्वाधर अक्ष शीर्षक के लिए पाठ ब्लॉक प्रारूप का घूर्णन कोण सेट करें।
```csharp
    // घूर्णन कोण को 90 डिग्री पर सेट करें।
    chart.Axes.VerticalAxis.Title.TextFormat.TextBlockFormat.RotationAngle = 90;

    // संशोधित चार्ट के साथ प्रस्तुति को निर्दिष्ट निर्देशिका में .pptx फ़ाइल में सहेजें।
    pres.Save(dataDir + "test.pptx", SaveFormat.Pptx);
}
```
### मुख्य कॉन्फ़िगरेशन विकल्प
- **वर्तन कोण**: अपनी डिज़ाइन आवश्यकताओं के आधार पर -180 और 180 डिग्री के बीच अनुकूलित करें।
- **अक्ष शीर्षक प्रारूप**: बेहतर दृश्यता के लिए फ़ॉन्ट का आकार, शैली और रंग संशोधित करें।

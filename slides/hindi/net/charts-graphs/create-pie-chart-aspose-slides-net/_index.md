# .NET के लिए Aspose.Slides का उपयोग करके किसी प्रेजेंटेशन में पाई चार्ट कैसे बनाएं और जोड़ें
## परिचय
आकर्षक प्रस्तुतियाँ बनाने में अक्सर सिर्फ़ टेक्स्ट से ज़्यादा कुछ शामिल होता है; चार्ट जैसे विज़ुअल तत्व आपके डेटा स्टोरीटेलिंग के प्रभाव को काफ़ी हद तक बढ़ा सकते हैं। अगर आप अपने पावरपॉइंट प्रस्तुतियों में प्रोग्रामेटिक रूप से डायनेमिक पाई चार्ट जोड़ना चाहते हैं, तो **.NET के लिए Aspose.Slides** एक शक्तिशाली उपकरण है जो इस कार्य को सहज और कुशल बनाता है। यह ट्यूटोरियल आपको प्रेजेंटेशन स्लाइड में पाई चार्ट जोड़ने और बाहरी डेटा स्रोतों के साथ इसे कॉन्फ़िगर करने के बारे में मार्गदर्शन करेगा।

### आप क्या सीखेंगे
- .NET के लिए Aspose.Slides का उपयोग करके एक नई प्रस्तुति कैसे बनाएं
- अपनी पहली स्लाइड में पाई चार्ट जोड़ना
- अपने चार्ट के लिए डेटा स्रोत के रूप में बाहरी कार्यपुस्तिका URL सेट करना
- अपनी प्रस्तुति को PPTX प्रारूप में सहेजना
आइए, हम इस बात पर गौर करें कि आप इसे आसानी से कैसे प्राप्त कर सकते हैं, सबसे पहले इसकी पूर्व-आवश्यकताओं से शुरुआत करते हैं।

## कार्यान्वयन मार्गदर्शिका
अब जब आपने सेटअप कर लिया है, तो आइए प्रत्येक सुविधा के बारे में चरण-दर-चरण जानकारी लें।
### प्रस्तुति में चार्ट बनाएं और जोड़ें
#### अवलोकन
हम एक प्रेजेंटेशन बनाकर और पहली स्लाइड में पाई चार्ट जोड़कर शुरुआत करेंगे।
#### चरण:
1. **प्रस्तुति आरंभ करें**
   इसका एक उदाहरण बनाकर शुरू करें `Presentation` क्लास, जो आपकी पावरपॉइंट फ़ाइल का प्रतिनिधित्व करता है.
   ```csharp
   using Aspose.Slides;
   
   string dataDir = "YOUR_DOCUMENT_DIRECTORY";
   
   using (Presentation pres = new Presentation())
   {
       // यहीं पर हम अपना चार्ट जोड़ेंगे।
   }
   ```
2. **पाई चार्ट जोड़ें**
   उपयोग `Shapes.AddChart` अपनी स्लाइड पर विशिष्ट निर्देशांक पर पाई चार्ट सम्मिलित करने की विधि।
   ```csharp
   IChart chart = pres.Slides[0].Shapes.AddChart(
       ChartType.Pie, 50, 50, 400, 600, true);
   ```
### चार्ट डेटा के लिए बाह्य कार्यपुस्तिका सेट करें
#### अवलोकन
अब आइए पाई चार्ट को बाहरी कार्यपुस्तिका से डेटा का उपयोग करने के लिए कॉन्फ़िगर करें।
#### चरण:
1. **चार्ट डेटा तक पहुंच**
   चार्ट डेटा इंटरफ़ेस प्राप्त करें जहाँ आप अपना बाह्य डेटा स्रोत URL निर्दिष्ट करेंगे.
   ```csharp
   IChartData chartData = chart.ChartData;
   ```
2. **बाह्य कार्यपुस्तिका URL सेट करें**
   अपने डेटा स्रोत के लिए URL सेट करें `SetExternalWorkbook`यह उदाहरण प्लेसहोल्डर URL का उपयोग करता है, जिसे आपके वास्तविक डेटा स्रोत पथ से प्रतिस्थापित किया जाना चाहिए।
   ```csharp
   (chartData as ChartData).SetExternalWorkbook("http://पथ/नहीं/मौजूद है", गलत);
   ```
### प्रस्तुति को फ़ाइल में सहेजें
#### अवलोकन
अंत में, प्रस्तुति को PPTX प्रारूप में अपने इच्छित स्थान पर सहेजें।
#### चरण:
1. **प्रस्तुति सहेजें**
   उपयोग `Save` की विधि `Presentation` फ़ाइल को डिस्क पर लिखने के लिए क्लास का उपयोग करें।
   ```csharp
   pres.Save(dataDir + "SetExternalWorkbookWithUpdateChartData.pptx", SaveFormat.Pptx);
   ```

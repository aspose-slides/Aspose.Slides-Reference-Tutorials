# .NET के लिए Aspose.Slides का उपयोग करके C# में मार्कर के साथ लाइन चार्ट कैसे बनाएं

## परिचय
C# में प्रभावी डेटा प्रस्तुति के लिए दृश्य रूप से आकर्षक और सूचनाप्रद लाइन चार्ट बनाना आवश्यक है। **.NET के लिए Aspose.Slides** मार्करों सहित पेशेवर दिखने वाले चार्ट जोड़ने की प्रक्रिया को सरल बनाता है। यह ट्यूटोरियल आपको .NET के लिए Aspose.Slides का उपयोग करके डिफ़ॉल्ट मार्करों के साथ एक लाइन चार्ट बनाने में मार्गदर्शन करेगा।

इस ट्यूटोरियल में आप सीखेंगे:
- .NET के लिए Aspose.Slides का उपयोग करने के लिए अपना वातावरण सेट अप करना।
- मार्करों को शामिल करने वाले लाइन चार्ट के साथ प्रस्तुति बनाना और उसे अनुकूलित करना।
- श्रेणियों, श्रृंखलाओं और डेटा बिंदुओं जैसे चार्ट गुणों को कॉन्फ़िगर करना।
- अंतिम प्रस्तुति फ़ाइल को सहेजना.

आइए अपने समाधान को क्रियान्वित करने से पहले आवश्यक पूर्वावश्यकताओं की समीक्षा करके शुरुआत करें।

## कार्यान्वयन मार्गदर्शिका
### मार्करों के साथ लाइन चार्ट बनाना
इस अनुभाग में, आप .NET के लिए Aspose.Slides का उपयोग करके अपनी प्रस्तुति में डिफ़ॉल्ट मार्करों के साथ एक लाइन चार्ट बनाने और कॉन्फ़िगर करने के लिए आवश्यक प्रत्येक चरण को सीखेंगे।

#### चरण 1: एक प्रेजेंटेशन ऑब्जेक्ट बनाएँ
इसका एक उदाहरण बनाकर शुरू करें `Presentation` कक्षा:
```csharp
using (Presentation pres = new Presentation())
{
    ISlide slide = pres.Slides[0];
```
यहां, हम नव निर्मित प्रस्तुति में पहली स्लाइड तक पहुंचते हैं।

#### चरण 2: मार्कर के साथ लाइन चार्ट जोड़ें
इसके बाद, अपनी स्लाइड में मार्कर के साथ एक लाइन चार्ट जोड़ें:
```csharp
IChart chart = slide.Shapes.AddChart(ChartType.LineWithMarkers, 10, 10, 400, 400);
```
यह कोड एक नया चार्ट जोड़ता है `LineWithMarkers` निर्देशांक पर `(10, 10)` आयामों के साथ `400x400`.

#### चरण 3: मौजूदा श्रृंखला और श्रेणियाँ साफ़ करें
डेटा जोड़ने से पहले, किसी भी मौजूदा श्रृंखला या श्रेणी को साफ़ करें:
```csharp
chart.ChartData.Series.Clear();
chart.ChartData.Categories.Clear();
```
इससे यह सुनिश्चित होता है कि हमारा चार्ट एक साफ़ स्लेट से शुरू हो।

#### चरण 4: चार्ट डेटा कार्यपुस्तिका कॉन्फ़िगर करें
तक पहुंच `ChartDataWorkbook` अपने चार्ट का डेटा प्रबंधित करने के लिए:
```csharp
IChartDataWorkbook fact = chart.ChartData.ChartDataWorkbook;
```
यह ऑब्जेक्ट श्रृंखला और श्रेणी डेटा वाले कक्षों के प्रबंधन के लिए महत्वपूर्ण है।

#### चरण 5: श्रृंखला और श्रेणियाँ जोड़ें
चार्ट में एक नई श्रृंखला जोड़ें और उसे डेटा बिंदुओं से भरें:
```csharp
chart.ChartData.Series.Add(fact.GetCell(0, 0, 1, "Series 1"), chart.Type);
IChartSeries series = chart.ChartData.Series[0];

// श्रेणियाँ और संबंधित डेटा बिंदु परिभाषित करें
chart.ChartData.Categories.Add(fact.GetCell(0, 1, 0, "C1"));
series.DataPoints.AddDataPointForLineSeries(fact.GetCell(0, 1, 1, 24));
chart.ChartData.Categories.Add(fact.GetCell(0, 2, 0, "C2"));
series.DataPoints.AddDataPointForLineSeries(fact.GetCell(0, 2, 1, 23));
chart.ChartData.Categories.Add(fact.GetCell(0, 3, 0, "C3"));
series.DataPoints.AddDataPointForLineSeries(fact.GetCell(0, 3, 1, -10));
chart.ChartData.Categories.Add(fact.GetCell(0, 4, 0, "C4"));

// लुप्त मानों के प्रबंधन को प्रदर्शित करने के लिए शून्य डेटा बिंदु जोड़ें
series.DataPoints.AddDataPointForLineSeries(fact.GetCell(0, 4, 1, (double?)null));
```
यहाँ, हम चार्ट को श्रेणियों और संबंधित श्रृंखला डेटा से भरते हैं। ध्यान दें कि कैसे `null` मूल्य को एक प्रदर्शन के रूप में संभाला जाता है।

#### चरण 6: एक और श्रृंखला जोड़ें
एक और श्रृंखला जोड़ने के लिए प्रक्रिया को दोहराएं:
```csharp
chart.ChartData.Series.Add(fact.GetCell(0, 0, 2, "Series 2"), chart.Type);
IChartSeries series2 = chart.ChartData.Series[1];

series2.DataPoints.AddDataPointForLineSeries(fact.GetCell(0, 1, 2, 30));
series2.DataPoints.AddDataPointForLineSeries(fact.GetCell(0, 2, 2, 10));
series2.DataPoints.AddDataPointForLineSeries(fact.GetCell(0, 3, 2, 60));
series2.DataPoints.AddDataPointForLineSeries(fact.GetCell(0, 4, 2, 40));
```

#### चरण 7: लेजेंड को सक्षम और कॉन्फ़िगर करें
पठनीयता में सुधार के लिए चार्ट लेजेंड सक्षम करें:
```csharp
chart.HasLegend = true;
chart.Legend.Overlay = false;
```
इससे यह सुनिश्चित होता है कि लेजेंड दृश्यमान है और चार्ट पर फैला हुआ नहीं है।

#### चरण 8: प्रस्तुति सहेजें
अंत में, अपने प्रेजेंटेशन को नए जोड़े गए चार्ट के साथ सेव करें:
```csharp
pres.Save("DefaultMarkersInChart.pptx");
}
```
### समस्या निवारण युक्तियों
- **डेटा बाइंडिंग त्रुटियाँ:** सुनिश्चित करें कि डेटा बिंदु श्रेणियों से सही ढंग से मेल खाते हों।
- **चार्ट प्रदर्शित नहीं हो रहा है:** सत्यापित करें कि `chart.HasLegend` और अन्य गुण उचित रूप से सेट किए गए हैं.

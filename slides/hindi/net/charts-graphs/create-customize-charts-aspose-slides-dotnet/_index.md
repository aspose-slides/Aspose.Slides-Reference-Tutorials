# Aspose.Slides .NET के साथ चार्ट कैसे बनाएं और अनुकूलित करें: लेबल के रूप में प्रतिशत प्रदर्शित करें

कई क्षेत्रों में डेटा को प्रभावी ढंग से प्रस्तुत करना महत्वपूर्ण है, और चार्ट जटिल जानकारी को स्पष्ट दृश्यों में बदलकर महत्वपूर्ण भूमिका निभाते हैं। सही चार्ट बनाने में लेबल पर प्रतिशत प्रदर्शित करने जैसे अनुकूलन कार्य शामिल हैं - .NET के लिए Aspose.Slides के साथ यह कार्य आसान हो गया है। यह लाइब्रेरी PowerPoint प्रस्तुतियों के भीतर चार्ट बनाने और संशोधित करने की प्रक्रिया को सरल बनाती है।

इस ट्यूटोरियल में, आप सीखेंगे कि .NET के लिए Aspose.Slides का उपयोग करके स्क्रैच से स्टैक्ड कॉलम चार्ट कैसे बनाएं और प्रतिशत मानों को डेटा लेबल के रूप में प्रदर्शित करके इसे कस्टमाइज़ करें। इन चरणों का पालन करके, आप अपनी स्लाइड्स को सटीक और आकर्षक डेटा अभ्यावेदन के साथ बढ़ाएँगे।

**आप क्या सीखेंगे:**
- .NET के लिए Aspose.Slides आरंभ करना
- स्टैक्ड कॉलम चार्ट बनाना
- डेटा लेबल पर प्रतिशत की गणना और प्रदर्शन
- चार्ट प्रदर्शन को अनुकूलित करने के सर्वोत्तम अभ्यास

इससे पहले कि हम कार्यान्वयन में उतरें, आइए सुनिश्चित करें कि आपके पास आरंभ करने के लिए सब कुछ तैयार है।

## .NET के लिए Aspose.Slides सेट अप करना

Aspose.Slides for .NET एक शक्तिशाली लाइब्रेरी है जो आपको PowerPoint प्रस्तुतियों के साथ प्रोग्रामेटिक रूप से काम करने की अनुमति देती है। इसे अपने प्रोजेक्ट में जोड़ने का तरीका यहां बताया गया है:

### इंस्टालेशन

**.नेट सीएलआई:**
```bash
dotnet add package Aspose.Slides
```

**पैकेज प्रबंधक कंसोल:**
```powershell
Install-Package Aspose.Slides
```

**NuGet पैकेज मैनेजर UI:** 
- NuGet पैकेज मैनेजर खोलें और "Aspose.Slides" खोजें। नवीनतम संस्करण स्थापित करें।

Aspose.Slides का पूरा उपयोग करने के लिए, निःशुल्क परीक्षण से शुरुआत करें। विस्तारित उपयोग के लिए, अस्थायी लाइसेंस प्राप्त करने या यहाँ से खरीदने पर विचार करें [असपोज](https://purchase.aspose.com/buy)अपने प्रोजेक्ट परिवेश में अपना लाइसेंस सेट अप करने के लिए उनके दिशानिर्देशों का पालन करें।

### मूल आरंभीकरण

एक बार इंस्टॉल हो जाने पर, प्रारंभ करें `Presentation` स्लाइड बनाना शुरू करने के लिए क्लास:

## कार्यान्वयन मार्गदर्शिका

### स्टैक्ड कॉलम चार्ट बनाएं

हमारा लक्ष्य एक स्टैक्ड कॉलम चार्ट बनाना और प्रतिशत को डेटा लेबल के रूप में दिखाकर इसे कस्टमाइज़ करना है। यहाँ बताया गया है कि कैसे:

#### प्रस्तुति आरंभ करें

इसका एक उदाहरण बनाकर शुरू करें `Presentation`:
```csharp
using Aspose.Slides;

// प्रेजेंटेशन क्लास इंस्टैंस आरंभ करें
tPresentation presentation = new Presentation();
ISlide slide = presentation.Slides[0];
```

#### स्लाइड में चार्ट जोड़ें

अपनी पहली स्लाइड में निर्दिष्ट निर्देशांक और आयामों पर एक स्टैक्ड कॉलम चार्ट जोड़ें:
```csharp
IChart chart = slide.Shapes.AddChart(ChartType.StackedColumn, 20, 20, 400, 400);
```
यह पंक्ति एक `StackedColumn` चार्ट स्थिति (20, 20) पर 400 की चौड़ाई और ऊंचाई के साथ।

#### प्रतिशत गणना के लिए कुल मान की गणना करें

प्रतिशत प्रदर्शित करने के लिए, सभी श्रृंखलाओं में प्रत्येक श्रेणी के लिए कुल मान की गणना करें:
```csharp
IChartSeries series;
double[] total_for_Cat = new double[chart.ChartData.Categories.Count];

for (int k = 0; k < chart.ChartData.Categories.Count; k++)
{
    IChartCategory cat = chart.ChartData.Categories[k];
    // प्रत्येक श्रेणी के लिए सभी श्रृंखलाओं के मानों का योग करें
    for (int i = 0; i < chart.ChartData.Series.Count; i++)
    {
        total_for_Cat[k] += Convert.ToDouble(chart.ChartData.Series[i].DataPoints[k].Value.Data);
    }
}
```

#### प्रतिशत मान दिखाने के लिए डेटा लेबल को अनुकूलित करें

इसके बाद, प्रत्येक श्रृंखला को दोहराएं और डेटा लेबल को अनुकूलित करें:
```csharp
for (int x = 0; x < chart.ChartData.Series.Count; x++)
{
    series = chart.ChartData.Series[x];
    series.Labels.DefaultDataLabelFormat.ShowLegendKey = false;

    for (int j = 0; j < series.DataPoints.Count; j++)
    {
        IDataLabel lbl = series.DataPoints[j].Label;
        
        // प्रतिशत की गणना करें
        double dataPontPercent = (Convert.ToDouble(series.DataPoints[j].Value.Data) / total_for_Cat[j]) * 100;
        IPortion port = new Portion();
        port.Text = String.Format("{0:F2} %", dataPontPercent);
        port.PortionFormat.FontHeight = 8f;

        lbl.TextFrameForOverriding.Text = ""; // ओवरलैप से बचने के लिए टेक्स्ट साफ़ करें
        IParagraph para = lbl.TextFrameForOverriding.Paragraphs[0];
        para.Portions.Add(port);

        // डिफ़ॉल्ट डेटा लेबल छिपाने के लिए लेबल प्रारूप कॉन्फ़िगर करें
        lbl.DataLabelFormat.ShowSeriesName = false;
        lbl.DataLabelFormat.ShowPercentage = false; 
        lbl.DataLabelFormat.ShowLegendKey = false;
        lbl.DataLabelFormat.ShowCategoryName = false;
        lbl.DataLabelFormat.ShowBubbleSize = false;
    }
}
```

यह अनुभाग प्रत्येक डेटा बिंदु के लिए प्रतिशत की गणना करता है और इसे एक कस्टम लेबल के रूप में सेट करता है, जिससे यह सुनिश्चित होता है कि डिफ़ॉल्ट लेबल के साथ कोई ओवरलैप न हो।

#### प्रस्तुति सहेजें

अंत में, परिणाम देखने के लिए अपनी प्रस्तुति सहेजें:
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
presentation.Save(outputDir + "/DisplayPercentageAsLabels_out.pptx", SaveFormat.Pptx);
```

## व्यावहारिक अनुप्रयोगों

चार्ट में प्रतिशत प्रदर्शित करना विशेष रूप से निम्नलिखित परिदृश्यों में उपयोगी हो सकता है:
1. **वित्तीय रिपोर्टिंग:** पोर्टफोलियो वितरण या निवेश रिटर्न को प्रतिशत के रूप में दिखाएं।
2. **बिक्री विश्लेषण:** विभिन्न क्षेत्रों में प्रदर्शन को उजागर करने के लिए बाजार हिस्सेदारी के आंकड़ों को प्रतिशत के आधार पर प्रस्तुत करें।
3. **सर्वेक्षण परिणाम:** बेहतर दृश्य तुलना के लिए सर्वेक्षण प्रतिक्रियाओं को प्रतिशत के रूप में प्रदर्शित करें।
4. **परियोजना प्रबंधन:** संसाधन आवंटन को दर्शाने के लिए प्रतिशत के साथ पाई चार्ट का उपयोग करें।
5. **शिक्षा:** स्पष्ट प्रतिशत-आधारित दृश्यों का उपयोग करके सांख्यिकीय अवधारणाओं को समझाएं।

इन अनुकूलित चार्टों को CRM या ERP जैसी प्रणालियों में एकीकृत करने से डैशबोर्ड और रिपोर्ट बेहतर हो सकती हैं, तथा निर्णय लेने की प्रक्रिया में सहायता मिल सकती है।

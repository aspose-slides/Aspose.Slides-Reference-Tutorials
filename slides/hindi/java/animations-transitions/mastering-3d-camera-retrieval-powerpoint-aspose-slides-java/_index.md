### कार्यान्वयन मार्गदर्शिका
अब जब आपका वातावरण तैयार है, तो आइए PowerPoint में 3D आकृतियों से कैमरा डेटा निकालें और उसमें बदलाव करें।

#### चरण-दर-चरण कैमरा डेटा पुनर्प्राप्ति
**1. प्रेजेंटेशन लोड करें**
अपनी लक्षित स्लाइड और आकृति वाली प्रस्तुति फ़ाइल लोड करके आरंभ करें:

```java
import com.aspose.slides.Presentation;
import com.aspose.slides.IThreeDFormatEffectiveData;

Presentation pres = new Presentation("YOUR_DOCUMENT_DIRECTORY/Presentation1.pptx");
```
यह कोड एक आरंभीकरण करता है `Presentation` ऑब्जेक्ट आपकी पावरपॉइंट फ़ाइल की ओर इशारा करता है.

**2. आकृति के प्रभावी डेटा तक पहुंचें**
3D प्रारूप प्रभावी डेटा तक पहुंचने के लिए पहली स्लाइड और उसके पहले आकार पर नेविगेट करें:

```java
IThreeDFormatEffectiveData threeDEffectiveData = pres.getSlides().get_Item(0)
    .getShapes().get_Item(0).getThreeDFormat().getEffective();
```
यह चरण आकृति पर प्रभावी रूप से लागू 3D गुणों को पुनः प्राप्त करता है।

**3. कैमरा गुण पुनः प्राप्त करें**
कैमरा प्रकार, दृश्य क्षेत्र कोण, और ज़ूम सेटिंग्स निकालें:

```java
String cameraType = threeDEffectiveData.getCamera().getCameraType();
float fieldOfViewAngle = threeDEffectiveData.getCamera().getFieldOfViewAngle();
double zoom = threeDEffectiveData.getCamera().getZoom();

// सत्यापित करने के लिए मान प्रिंट करें
System.out.println("Camera Type: " + cameraType);
System.out.println("Field of View Angle: " + fieldOfViewAngle);
System.out.println("Zoom Level: " + zoom);
```
ये गुण आपको लागू किए गए 3D परिप्रेक्ष्य को समझने में मदद करते हैं।

**4. संसाधनों की सफाई करें**
हमेशा संसाधन जारी करें:

```java
finally {
    if (pres != null) pres.dispose();
}
```

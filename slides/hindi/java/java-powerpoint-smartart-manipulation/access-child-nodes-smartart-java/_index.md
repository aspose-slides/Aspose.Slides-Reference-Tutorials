## परिचय

## पैकेज आयात करें
सबसे पहले, आपको Aspose.Slides से आवश्यक पैकेज आयात करने होंगे। ये आयात प्रस्तुति तत्वों तक पहुँचने और उनमें हेरफेर करने के लिए आवश्यक हैं।
```java
import com.aspose.slides.*;
```
आइए स्मार्टआर्ट में चाइल्ड नोड्स तक पहुंचने की प्रक्रिया को सरल, प्रबंधनीय चरणों में विभाजित करें।
## चरण 1: अपना वातावरण सेट करें
इससे पहले कि आप किसी प्रस्तुति में बदलाव कर सकें, आपको अपने प्रोजेक्ट में Aspose.Slides लाइब्रेरी को शामिल करके अपना विकास वातावरण स्थापित करना होगा।
1.  Aspose.Slides डाउनलोड करें: लाइब्रेरी प्राप्त करें[लिंक को डाउनलोड करें](https://releases.aspose.com/slides/java/).
2. लाइब्रेरी शामिल करें: डाउनलोड की गई JAR फ़ाइल को अपने प्रोजेक्ट के बिल्ड पथ में जोड़ें।
## चरण 2: प्रस्तुति लोड करें
उस पावरपॉइंट प्रस्तुति को लोड करें जिसमें वह स्मार्टआर्ट ग्राफ़िक है जिसे आप संशोधित करना चाहते हैं।
```java
String dataDir = "Your Document Directory";
Presentation pres = new Presentation(dataDir + "AccessChildNodes.pptx");
```
## चरण 3: स्मार्टआर्ट आकार तक पहुंचें
स्मार्टआर्ट आकृति ढूंढने के लिए पहली स्लाइड में आकृतियों को पार करें।
```java
for (IShape shape : pres.getSlides().get_Item(0).getShapes()) {
    if (shape instanceof SmartArt) {
        ISmartArt smart = (ISmartArt) shape;
        // आगे की कार्यवाही यहां होगी
    }
}
```
## चरण 4: स्मार्टआर्ट नोड्स को पार करें
एक बार जब आप स्मार्टआर्ट आकृति तक पहुंच प्राप्त कर लें, तो उसके सभी नोड्स पर जाएं।
```java
for (int i = 0; i < smart.getAllNodes().size(); i++) {
    ISmartArtNode node0 = (ISmartArtNode) smart.getAllNodes().get_Item(i);
    // आगे की कार्यवाही यहां होगी
}
```
## चरण 5: चाइल्ड नोड्स तक पहुंचें
प्रत्येक स्मार्टआर्ट नोड के भीतर, उसके चाइल्ड नोड्स तक पहुंचें।
```java
for (int j = 0; j < node0.getChildNodes().size(); j++) {
    ISmartArtNode node = (ISmartArtNode) node0.getChildNodes().get_Item(j);
    // आगे की कार्यवाही यहां होगी
}
```
## चरण 6: नोड विवरण प्रिंट करें
प्रत्येक चाइल्ड नोड का विवरण प्रिंट करें, जैसे पाठ, स्तर और स्थिति।
```java
String outString = String.format("j = %d, Text = %s, Level = %d, Position = %d", j, node.getTextFrame().getText(), node.getLevel(), node.getPosition());
System.out.println(outString);
```

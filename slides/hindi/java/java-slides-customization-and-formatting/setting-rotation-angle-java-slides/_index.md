## जावा स्लाइड्स में रोटेशन कोण सेट करने का परिचय

इस ट्यूटोरियल में, हम Aspose.Slides for Java लाइब्रेरी का उपयोग करके चार्ट अक्ष शीर्षक में टेक्स्ट के लिए रोटेशन कोण सेट करने का तरीका जानेंगे। रोटेशन कोण को समायोजित करके, आप अपनी प्रस्तुति आवश्यकताओं के अनुरूप अपने चार्ट के अक्ष शीर्षकों की उपस्थिति को बेहतर ढंग से अनुकूलित कर सकते हैं।

## आवश्यक शर्तें

शुरू करने से पहले, सुनिश्चित करें कि आपके पास Aspose.Slides for Java लाइब्रेरी स्थापित है और आपके Java प्रोजेक्ट में सेट अप है। आप Aspose वेबसाइट से लाइब्रेरी डाउनलोड कर सकते हैं और उनके दस्तावेज़ों में दिए गए इंस्टॉलेशन निर्देशों का पालन कर सकते हैं।

## चरण 1: एक प्रस्तुति बनाएं

सबसे पहले, आपको एक नया प्रेजेंटेशन बनाना होगा या मौजूदा प्रेजेंटेशन लोड करना होगा। इस उदाहरण में, हम एक नया प्रेजेंटेशन बनाएंगे:

```java
// दस्तावेज़ निर्देशिका का पथ.
String dataDir = "Your Document Directory";
Presentation pres = new Presentation();
```

## चरण 2: स्लाइड में चार्ट जोड़ें

इसके बाद, हम स्लाइड में एक चार्ट जोड़ेंगे। इस उदाहरण में, हम एक क्लस्टर्ड कॉलम चार्ट जोड़ रहे हैं:

```java
try
{
    IChart chart = pres.getSlides().get_Item(0).getShapes().addChart(ChartType.ClusteredColumn, 50, 50, 450, 300);
```

## चरण 3: अक्ष शीर्षक के लिए रोटेशन कोण सेट करें

अक्ष शीर्षक के लिए रोटेशन कोण सेट करने के लिए, आपको चार्ट के ऊर्ध्वाधर अक्ष शीर्षक तक पहुँचना होगा और उसके रोटेशन कोण को समायोजित करना होगा। यहाँ बताया गया है कि आप यह कैसे कर सकते हैं:

```java
    chart.getAxes().getVerticalAxis().setTitle(true);
    chart.getAxes().getVerticalAxis().getTitle().getTextFormat().getTextBlockFormat().setRotationAngle(90);
```

इस कोड स्निपेट में, हम रोटेशन एंगल को 90 डिग्री पर सेट कर रहे हैं, जो टेक्स्ट को लंबवत घुमाएगा। आप कोण को अपने इच्छित मान पर समायोजित कर सकते हैं।

## चरण 4: प्रस्तुति सहेजें

अंत में, प्रस्तुति को पावरपॉइंट फ़ाइल में सहेजें:

```java
    pres.save(dataDir + "test.pptx", SaveFormat.Pptx);
}
finally
{
    if (pres != null) pres.dispose();
}
```

## जावा स्लाइड्स में रोटेशन कोण सेट करने के लिए पूर्ण स्रोत कोड

```java
// दस्तावेज़ निर्देशिका का पथ.
String dataDir = "Your Document Directory";
Presentation pres = new Presentation();
try
{
	IChart chart = pres.getSlides().get_Item(0).getShapes().addChart(ChartType.ClusteredColumn, 50, 50, 450, 300);
	chart.getAxes().getVerticalAxis().setTitle(true);
	chart.getAxes().getVerticalAxis().getTitle().getTextFormat().getTextBlockFormat().setRotationAngle(90);
	pres.save(dataDir + "test.pptx", SaveFormat.Pptx);
}
finally
{
	if (pres != null) pres.dispose();
}
```

## निष्कर्ष

इस ट्यूटोरियल में, आपने सीखा है कि Aspose.Slides for Java का उपयोग करके चार्ट अक्ष शीर्षक में टेक्स्ट के लिए रोटेशन कोण कैसे सेट किया जाए। यह सुविधा आपको अपने चार्ट की उपस्थिति को अनुकूलित करने की अनुमति देती है ताकि आकर्षक प्रस्तुतिकरण बनाए जा सकें। अपने चार्ट के लिए वांछित रूप प्राप्त करने के लिए विभिन्न रोटेशन कोणों के साथ प्रयोग करें।

## अक्सर पूछे जाने वाले प्रश्न

### मैं स्लाइड में अन्य टेक्स्ट तत्वों के लिए रोटेशन कोण कैसे बदल सकता हूँ?

आप इसी तरह के तरीके का इस्तेमाल करके अन्य टेक्स्ट तत्वों, जैसे कि आकृतियों या टेक्स्ट बॉक्स के लिए रोटेशन एंगल बदल सकते हैं। तत्व के टेक्स्ट फ़ॉर्मेट तक पहुँचें और आवश्यकतानुसार रोटेशन एंगल सेट करें।

### क्या मैं क्षैतिज अक्ष शीर्षक में भी पाठ घुमा सकता हूँ?

हां, आप रोटेशन एंगल को एडजस्ट करके क्षैतिज अक्ष शीर्षक में टेक्स्ट को घुमा सकते हैं। बस रोटेशन एंगल को अपने मनचाहे मान पर सेट करें, जैसे कि वर्टिकल टेक्स्ट के लिए 90 डिग्री या हॉरिजॉन्टल टेक्स्ट के लिए 0 डिग्री।

### चार्ट शीर्षकों के लिए अन्य कौन से स्वरूपण विकल्प उपलब्ध हैं?

Aspose.Slides for Java चार्ट शीर्षकों के लिए विभिन्न स्वरूपण विकल्प प्रदान करता है, जिसमें फ़ॉन्ट शैलियाँ, रंग और संरेखण शामिल हैं। आप चार्ट शीर्षकों को अनुकूलित करने के बारे में अधिक जानकारी के लिए दस्तावेज़ देख सकते हैं।

### क्या चार्ट अक्ष शीर्षक में पाठ के घूर्णन को एनिमेट करना संभव है?

हां, आप Aspose.Slides for Java का उपयोग करके, चार्ट अक्ष शीर्षकों सहित टेक्स्ट तत्वों में एनिमेशन प्रभाव जोड़ सकते हैं। अपनी प्रस्तुतियों में एनिमेशन जोड़ने के बारे में जानकारी के लिए दस्तावेज़ देखें।

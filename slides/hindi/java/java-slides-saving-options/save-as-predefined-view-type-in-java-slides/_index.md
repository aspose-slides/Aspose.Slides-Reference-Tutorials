## जावा स्लाइड्स में पूर्वनिर्धारित दृश्य प्रकार के रूप में सहेजें का परिचय

इस चरण-दर-चरण मार्गदर्शिका में, हम यह पता लगाएंगे कि Aspose.Slides for Java का उपयोग करके किसी पूर्वनिर्धारित दृश्य प्रकार के साथ किसी प्रस्तुति को कैसे सहेजा जाए। हम आपको इस कार्य को सफलतापूर्वक पूरा करने के लिए आवश्यक कोड और स्पष्टीकरण प्रदान करेंगे।

## आवश्यक शर्तें

शुरू करने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित हैं:

- जावा प्रोग्रामिंग का बुनियादी ज्ञान.
- Aspose.Slides for Java लाइब्रेरी स्थापित की गई।
- अपनी पसंद का एकीकृत विकास वातावरण (आईडीई)।

## अपना वातावरण स्थापित करना

आरंभ करने के लिए, अपना विकास वातावरण स्थापित करने हेतु इन चरणों का पालन करें:

1. अपने IDE में एक नया जावा प्रोजेक्ट बनाएं।
2. Aspose.Slides for Java लाइब्रेरी को अपनी परियोजना में निर्भरता के रूप में जोड़ें।

अब जब आपका वातावरण तैयार हो गया है, तो चलिए कोड के साथ आगे बढ़ते हैं।

## चरण 1: प्रेजेंटेशन बनाना

किसी प्रेजेंटेशन को पूर्वनिर्धारित दृश्य प्रकार के साथ सहेजने का प्रदर्शन करने के लिए, हम पहले एक नया प्रेजेंटेशन बनाएंगे। प्रेजेंटेशन बनाने के लिए कोड इस प्रकार है:

```java
// दस्तावेज़ निर्देशिका का पथ.
String dataDir = "Your Document Directory";
// प्रस्तुति फ़ाइल खोलना
Presentation presentation = new Presentation();
```

 इस कोड में, हम एक नया कोड बनाते हैं`Presentation` ऑब्जेक्ट, जो हमारे पावरपॉइंट प्रेजेंटेशन का प्रतिनिधित्व करता है।

## चरण 2: दृश्य प्रकार सेट करना

इसके बाद, हम अपनी प्रस्तुति के लिए दृश्य प्रकार सेट करेंगे। दृश्य प्रकार परिभाषित करते हैं कि प्रस्तुति को खोलने पर वह किस तरह प्रदर्शित होगी। इस उदाहरण में, हम इसे "स्लाइड मास्टर दृश्य" पर सेट करेंगे। यहाँ कोड है:

```java
// दृश्य प्रकार सेट करना
presentation.getViewProperties().setLastView(ViewType.SlideMasterView);
```

 उपरोक्त कोड में, हम उपयोग करते हैं`setLastView` की विधि`ViewProperties` दृश्य प्रकार सेट करने के लिए क्लास`SlideMasterView`आप आवश्यकतानुसार अन्य दृश्य प्रकार चुन सकते हैं.

## चरण 3: प्रस्तुति को सहेजना

अब जबकि हमने अपनी प्रस्तुति बना ली है और दृश्य प्रकार सेट कर लिया है, अब प्रस्तुति को सहेजने का समय आ गया है। हम इसे PPTX प्रारूप में सहेजेंगे। यहाँ कोड है:

```java
// प्रस्तुति सहेजना
presentation.save(dataDir + "SetViewType_out.pptx", SaveFormat.Pptx);
```

 इस कोड में, हम उपयोग करते हैं`save` की विधि`Presentation` क्लास का उपयोग करके प्रस्तुति को निर्दिष्ट फ़ाइल नाम और प्रारूप के साथ सहेजा जा सकता है।

## जावा स्लाइड्स में पूर्वनिर्धारित दृश्य प्रकार के रूप में सहेजने के लिए पूर्ण स्रोत कोड

```java
// दस्तावेज़ निर्देशिका का पथ.
String dataDir = "Your Document Directory";
// प्रस्तुति फ़ाइल खोलना
Presentation presentation = new Presentation();
try
{
	// दृश्य प्रकार सेट करना
	presentation.getViewProperties().setLastView(ViewType.SlideMasterView);
	// प्रस्तुति सहेजना
	presentation.save(dataDir + "SetViewType_out.pptx", SaveFormat.Pptx);
}
finally
{
	if (presentation != null) presentation.dispose();
}
```

## निष्कर्ष

इस ट्यूटोरियल में, हमने सीखा है कि Aspose.Slides for Java का उपयोग करके जावा में पूर्वनिर्धारित दृश्य प्रकार के साथ प्रेजेंटेशन को कैसे सहेजा जाए। दिए गए कोड और चरणों का पालन करके, आप आसानी से अपनी प्रस्तुतियों का दृश्य प्रकार सेट कर सकते हैं और उन्हें वांछित प्रारूप में सहेज सकते हैं।

## अक्सर पूछे जाने वाले प्रश्न

### मैं दृश्य प्रकार को "स्लाइड मास्टर दृश्य" के अलावा किसी अन्य प्रकार में कैसे बदल सकता हूँ?

 दृश्य प्रकार को "स्लाइड मास्टर दृश्य" के अलावा किसी अन्य प्रकार में बदलने के लिए, बस प्रतिस्थापित करें`ViewType.SlideMasterView` वांछित दृश्य प्रकार के साथ, जैसे`ViewType.NormalView` या`ViewType.SlideSorterView`, कोड में जहां हम दृश्य प्रकार सेट करते हैं।

### क्या मैं प्रस्तुति में अलग-अलग स्लाइडों के लिए दृश्य गुण सेट कर सकता हूँ?

हां, आप Aspose.Slides for Java का उपयोग करके अलग-अलग स्लाइड के लिए दृश्य गुण सेट कर सकते हैं। आप प्रस्तुति में स्लाइड के माध्यम से पुनरावृत्ति करके प्रत्येक स्लाइड के लिए अलग-अलग गुणों तक पहुंच सकते हैं और उनमें हेरफेर कर सकते हैं।

### मैं अपनी प्रस्तुति को अन्य किस प्रारूप में सहेज सकता हूँ?

Aspose.Slides for Java विभिन्न आउटपुट फ़ॉर्मेट का समर्थन करता है, जिसमें PPTX, PDF, TIFF, HTML, और बहुत कुछ शामिल है। आप अपनी प्रस्तुति को सहेजते समय उपयुक्त का उपयोग करके वांछित फ़ॉर्मेट निर्दिष्ट कर सकते हैं`SaveFormat` enum मान.

### क्या Aspose.Slides for Java प्रस्तुतियों के बैच प्रसंस्करण के लिए उपयुक्त है?

हां, Aspose.Slides for Java बैच प्रोसेसिंग कार्यों के लिए उपयुक्त है। आप जावा कोड का उपयोग करके कई प्रस्तुतियों की प्रोसेसिंग को स्वचालित कर सकते हैं, परिवर्तन लागू कर सकते हैं और उन्हें बल्क में सहेज सकते हैं।

### मैं Aspose.Slides for Java के लिए अधिक जानकारी और दस्तावेज़ कहां पा सकता हूं?

 Aspose.Slides for Java से संबंधित विस्तृत दस्तावेज़ीकरण और संदर्भों के लिए, कृपया दस्तावेज़ीकरण वेबसाइट पर जाएँ:[Aspose.Slides for Java दस्तावेज़ीकरण](https://reference.aspose.com/slides/java/).

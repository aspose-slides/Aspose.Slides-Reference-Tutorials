अपने में निम्नलिखित निर्भरता जोड़ें `pom.xml` फ़ाइल:


## कार्यान्वयन मार्गदर्शिका

अब जब आपका वातावरण तैयार है, तो आइए Aspose.Slides Java का उपयोग करके गणितीय प्रस्तुति सुविधाओं को लागू करें।

### गणितीय आकृतियों के साथ प्रस्तुति बनाना और कॉन्फ़िगर करना

#### अवलोकन

यह सुविधा आपको आसानी से एक नई प्रस्तुति बनाने और गणितीय आकृतियाँ जोड़ने में मदद करती है। 

**चरण 1: एक नई प्रस्तुति बनाएँ**

```java
// एक नया प्रेजेंटेशन ऑब्जेक्ट आरंभ करें
tPresentation pres = new Presentation();
try {
    // पहली स्लाइड में स्थिति (0, 0) पर 500 चौड़ाई और 50 ऊंचाई वाली एक गणितीय आकृति जोड़ें।
    IAutoShape autoShape = pres.getSlides().get_Item(0).getShapes().addMathShape(0, 0, 500, 50);
} finally {
    if (pres != null) pres.dispose();
}
```

इस स्निपेट में, हम एक नया प्रेजेंटेशन ऑब्जेक्ट आरंभ करते हैं और पहली स्लाइड में एक गणितीय आकृति जोड़ते हैं। `IAutoShape` वर्ग बहुमुखी अनुकूलन के लिए अनुमति देता है।

### गणित पैराग्राफ तक पहुँचना और उन्हें संशोधित करना

#### अवलोकन

यह अनुभाग दर्शाता है कि आप अपनी आकृतियों में विद्यमान गणितीय अनुच्छेदों तक कैसे पहुंच सकते हैं तथा गणितीय पाठ जोड़कर उन्हें कैसे संशोधित कर सकते हैं।

**चरण 2: गणितीय पाठ संशोधित करें**

```java
// एक नया प्रस्तुतिकरण बनाएं
Presentation pres = new Presentation();
try {
    // स्थिति (0, 0) पर 500 चौड़ाई और 50 ऊंचाई वाली एक गणितीय आकृति जोड़ें।
    IAutoShape autoShape = pres.getSlides().get_Item(0).getShapes().addMathShape(0, 0, 500, 50);

    // पहले पैराग्राफ के पहले भाग को MathPortion के रूप में एक्सेस करें
    IMathParagraph mathParagraph = ((MathPortion)autoShape.getTextFrame().getParagraphs().get_Item(0).getPortions().get_Item(0)).getMathParagraph();

    // गणित पैराग्राफ में एक सूत्र जोड़ें: "a^2 + b^2 = c^2"
    mathParagraph.add(new MathematicalText("a").setSuperscript("2")
            .join("+")
            .join(new MathematicalText("b").setSuperscript("2"))
            .join(=)
            .join(new MathematicalText("c").setSuperscript("2")));
} finally {
    if (pres != null) pres.dispose();
}
```

यहाँ, हम अपने गणित आकार के पहले पैराग्राफ तक पहुँचते हैं और एक सूत्र जोड़कर इसे संशोधित करते हैं। `MathematicalText` क्लास सुपरस्क्रिप्ट सेट करने और समीकरणों के विभिन्न भागों को संयोजित करने के तरीके प्रदान करता है।

### गणितीय पैराग्राफ़ को LaTeX में निर्यात करना

#### अवलोकन

अपनी गणितीय सामग्री को LaTeX में परिवर्तित करना दस्तावेज़ीकरण या साझाकरण उद्देश्यों के लिए आवश्यक हो सकता है।

**चरण 3: LaTeX में कनवर्ट करें**

```java
// एक नया प्रस्तुतिकरण बनाएं
Presentation pres = new Presentation();
try {
    // स्थिति (0, 0) पर 500 चौड़ाई और 50 ऊंचाई वाली एक गणितीय आकृति जोड़ें।
    IAutoShape autoShape = pres.getSlides().get_Item(0).getShapes().addMathShape(0, 0, 500, 50);

    // पहले पैराग्राफ के पहले भाग को MathPortion के रूप में एक्सेस करें
    IMathParagraph mathParagraph = ((MathPortion)autoShape.getTextFrame().getParagraphs().get_Item(0).getPortions().get_Item(0)).getMathParagraph();

    // गणित पैराग्राफ में एक सूत्र जोड़ें: "a^2 + b^2 = c^2"
    mathParagraph.add(new MathematicalText("a").setSuperscript("2")
            .join("+")
            .join(new MathematicalText("b").setSuperscript("2"))
            .join(=)
            .join(new MathematicalText("c").setSuperscript("2"));

    // गणित पैराग्राफ़ को LaTeX स्ट्रिंग में बदलें
    String latexString = mathParagraph.toLatex();
} finally {
    if (pres != null) pres.dispose();
}
```

The `toLatex()` विधि आपके गणितीय पाठ को LaTeX स्वरूपित स्ट्रिंग में परिवर्तित करती है, जिससे इसे साझा करना या प्रकाशित करना आसान हो जाता है।

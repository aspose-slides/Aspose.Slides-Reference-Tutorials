# Aspose.Slides Java का उपयोग करके आकृतियों में ग्रेडिएंट भरण कैसे लागू करें
चाहे आप कोई विचार प्रस्तुत कर रहे हों या अपना काम प्रदर्शित कर रहे हों, दिखने में आकर्षक प्रस्तुतियाँ बनाना बहुत ज़रूरी है। अपनी PowerPoint स्लाइड्स को बेहतर बनाने का एक तरीका आकृतियों पर ग्रेडिएंट फिल लगाना है। इस ट्यूटोरियल में, हम इस बात पर चर्चा करेंगे कि आप PowerPoint प्रस्तुति में दीर्घवृत्त आकार पर शानदार ग्रेडिएंट फिल लगाने के लिए Aspose.Slides for Java लाइब्रेरी का उपयोग कैसे कर सकते हैं।

**आप क्या सीखेंगे:**

- वास्तविक दुनिया के परिदृश्यों में इस सुविधा के व्यावहारिक अनुप्रयोग।

आइये इस कार्यक्षमता को क्रियान्वित करने से पहले आवश्यक पूर्वापेक्षाओं पर नजर डालते हैं।

## कार्यान्वयन मार्गदर्शिका
अब हम Aspose.Slides for Java का उपयोग करके PowerPoint प्रस्तुति में दीर्घवृत्त आकार पर ग्रेडिएंट भरण लागू करने के लिए आवश्यक चरणों को देखेंगे।

### ग्रेडिएंट फिल के साथ दीर्घवृत्त आकार जोड़ना
#### चरण 1: प्रस्तुति बनाएं और कॉन्फ़िगर करें
सबसे पहले, एक नया आरंभ करें `Presentation` ऑब्जेक्ट जो आपकी पावरपॉइंट फ़ाइल का प्रतिनिधित्व करता है। यह वह जगह है जहाँ आप अपनी आकृतियाँ जोड़ेंगे और फ़ॉर्मेटिंग लागू करेंगे।

```java
import com.aspose.slides.*;

public class FillShapesGradient {
    public static void main(String[] args) {
        String dataDir = "YOUR_DOCUMENT_DIRECTORY";
        
        Presentation pres = new Presentation();
        try {
            ISlide sld = pres.getSlides().get_Item(0);
```

#### चरण 2: एक दीर्घवृत्त आकार जोड़ें
स्लाइड में दीर्घवृत्ताकार आकृति जोड़ें। आप स्लाइड पर इसके आयाम और स्थिति निर्दिष्ट कर सकते हैं।

```java
            // परिभाषित आयाम और स्थिति के साथ एक दीर्घवृत्त आकार जोड़ें।
            IShape shp = sld.getShapes().addAutoShape(ShapeType.Ellipse, 50, 150, 75, 150);
```

#### चरण 3: ग्रेडिएंट भरण लागू करें
अपने आकार के लिए ग्रेडिएंट भरण सेटिंग कॉन्फ़िगर करें। आप विभिन्न ग्रेडिएंट आकृतियों और दिशाओं के बीच चयन कर सकते हैं।

```java
            // ग्रेडिएंट भरण प्रकार सेट करें.
            shp.getFillFormat().setFillType(FillType.Gradient);

            // एक रेखीय ढाल आकार चुनें.
            shp.getFillFormat().getGradientFormat().setGradientShape(GradientShape.Linear);

            // ढाल दिशा को परिभाषित करें.
            shp.getFillFormat().getGradientFormat().setGradientDirection(GradientDirection.FromCorner2);
```

#### चरण 4: ग्रेडिएंट रंग अनुकूलित करें
अपने ग्रेडिएंट स्टॉप के लिए रंग और स्थिति निर्धारित करें। यह नियंत्रित करता है कि रंगों के बीच ग्रेडिएंट कैसे परिवर्तित होता है।

```java
            // ग्रेडिएंट संक्रमण को परिभाषित करने के लिए रंग स्टॉप जोड़ें।
            shp.getFillFormat().getGradientFormat().getGradientStops().add((float) 1.0, new Color(PresetColor.Purple));
            shp.getFillFormat().getGradientFormat().getGradientStops().add((float) 0, Color.RED);
```

#### चरण 5: प्रस्तुति सहेजें
अंत में, अपनी प्रस्तुति को ग्रेडिएंट से भरी आकृति के साथ एक फ़ाइल में सहेजें।

```java
            // अद्यतन स्लाइड के साथ प्रस्तुति को सहेजें.
            pres.save(dataDir + "EllipseShpGrad_out.pptx", SaveFormat.Pptx);
        } finally {
            if (pres != null) pres.dispose();
        }
    }
}
```

**समस्या निवारण युक्तियों:**
- सुनिश्चित करें कि आपकी फ़ाइल पथ सही ढंग से सेट की गई है ताकि कोई समस्या न हो `IOException`.
- सत्यापित करें कि JDK संस्करण निर्भरताओं में प्रयुक्त Aspose.Slides क्लासिफायर से मेल खाता है।

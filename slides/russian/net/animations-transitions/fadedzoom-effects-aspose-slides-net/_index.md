## Руководство по внедрению
Давайте рассмотрим, как реализовать эффект FadedZoom с подтипами ObjectCenter и SlideCenter.

### Применение эффекта затухания масштабирования с подтипом ObjectCenter
Эта функция позволяет создать анимацию, сосредоточенную вокруг самой формы, что делает ее идеальной для подчеркивания определенных элементов на слайде.

#### Шаг 1: Инициализация презентации и добавление формы
```csharp
using Aspose.Slides;
using Aspose.Slides.Animation;

public class ApplyFadedZoomObjectCenter
{
    public void CreateAnimation()
    {
        using (Presentation pres = new Presentation())
        {
            // Создайте прямоугольник на первом слайде.
            var shp1 = pres.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 0, 0, 50, 50);
```
#### Шаг 2: Добавьте эффект FadedZoom

```csharp
            // Применить эффект FadedZoom с подтипом ObjectCenter к фигуре
            pres.Slides[0].Timeline.MainSequence.AddEffect(
                shp1, EffectType.FadedZoom, EffectSubtype.ObjectCenter, EffectTriggerType.OnClick
            );

            // Сохраните презентацию в желаемом каталоге.
            pres.Save("YOUR_OUTPUT_DIRECTORY/AnimationFadedZoom_ObjectCenter.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
        }
    }
}
```
**Объяснение:** Здесь, `EffectSubtype.ObjectCenter` фокусирует анимацию вокруг самой формы. Эффект запускается щелчком.

### Применение эффекта постепенного увеличения с подтипом SlideCenter
Этот подтип сосредотачивает эффект масштабирования на самом слайде, что идеально подходит для перехода между слайдами или подчеркивания общего содержания слайда.

#### Шаг 1: Инициализация презентации и добавление формы
```csharp
using Aspose.Slides;
using Aspose.Slides.Animation;

public class ApplyFadedZoomSlideCenter
{
    public void CreateAnimation()
    {
        using (Presentation pres = new Presentation())
        {
            // Создайте прямоугольник на первом слайде в другом месте.
            var shp2 = pres.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 100, 0, 50, 50);
```
#### Шаг 2: Добавьте эффект FadedZoom

```csharp
            // Применить эффект FadedZoom с подтипом SlideCenter к фигуре
            pres.Slides[0].Timeline.MainSequence.AddEffect(
                shp2, EffectType.FadedZoom, EffectSubtype.SlideCenter, EffectTriggerType.OnClick
            );

            // Сохраните презентацию в желаемом каталоге.
            pres.Save("YOUR_OUTPUT_DIRECTORY/AnimationFadedZoom_SlideCenter.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
        }
    }
}
```
**Объяснение:** `EffectSubtype.SlideCenter` фокусирует анимацию на центре слайда, создавая более широкое воздействие по мере того, как эффект масштабирования распространяется наружу.

## Раздел часто задаваемых вопросов
**В1: Можно ли применить несколько анимаций к одной фигуре?**
- Да, вы можете добавить несколько эффектов в последовательность, вызвав `AddEffect` многократно для разных анимаций.

**В2: Как запустить анимацию автоматически, а не по щелчку?**
- Изменять `EffectTriggerType.OnClick` к другому типу триггера, например `AfterPrevious` или `WithPrevious`.

**В3: Что произойдет, если файл моей презентации окажется большим?**
- Большие файлы могут повлиять на производительность; рассмотрите возможность оптимизации использования контента и эффектов.

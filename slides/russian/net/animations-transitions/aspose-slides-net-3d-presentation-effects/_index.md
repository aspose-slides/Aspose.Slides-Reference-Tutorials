## Руководство по внедрению
В этом разделе мы сосредоточимся на реализации эффектов 3D-вращения с помощью Aspose.Slides для .NET.
### Добавление 3D-вращения к фигурам
#### Обзор
Мы добавим прямоугольник и линейную форму к слайду, применяя 3D-преобразования. Эти эффекты могут сделать ваши слайды выделяющимися в любой презентации.
#### Пошаговое руководство
**1. Подготовьте презентацию**
Начните с создания экземпляра `Presentation` сорт:
```csharp
using Aspose.Slides;
using Aspose.Slides.Export;

public void Apply3DRotation()
{
    // Определить пути к каталогам
    string dataDir = "YOUR_DOCUMENT_DIRECTORY";
    string outputDir = "YOUR_OUTPUT_DIRECTORY";
    
    // Инициализируйте новый объект Presentation
    Presentation pres = new Presentation();
```
**2. Добавьте прямоугольную форму и настройте 3D-эффекты.**
Добавьте к первому слайду прямоугольник и примените 3D-вращение:
```csharp
// Добавьте прямоугольную форму
IShape autoShape = pres.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 30, 30, 200, 200);

// Установите глубину 3D-объекта
autoShape.ThreeDFormat.Depth = 6;

// Поверните камеру для получения желаемого 3D-эффекта
autoShape.ThreeDFormat.Camera.SetRotation(40, 35, 20);

// Определите тип предустановки камеры
autoShape.ThreeDFormat.Camera.CameraType = CameraPresetType.IsometricLeftUp;

// Настройте освещение в сцене
autoShape.ThreeDFormat.LightRig.LightType = LightRigPresetType.Balanced;
```
**3. Добавьте линейную форму с различными 3D-настройками**
Добавьте еще одну фигуру, на этот раз линию, и примените особые 3D-настройки:
```csharp
// Добавить форму линии
autoShape = pres.Slides[0].Shapes.AddAutoShape(ShapeType.Line, 30, 300, 200, 200);

// Установите глубину 3D-объекта для формы линии.
autoShape.ThreeDFormat.Depth = 6;

// Настройте вращение камеры иначе, чем прямоугольник
autoShape.ThreeDFormat.Camera.SetRotation(0, 35, 20);

// Используйте те же настройки камеры, что и раньше.
autoShape.ThreeDFormat.Camera.CameraType = CameraPresetType.IsometricLeftUp;

// Применяйте согласованные настройки освещения
autoShape.ThreeDFormat.LightRig.LightType = LightRigPresetType.Balanced;
```
**4. Сохраните презентацию**
Наконец, сохраните презентацию со всеми примененными 3D-эффектами:
```csharp
// Сохранить в файл PPTX
pres.Save(outputDir + "/Rotation_out.pptx", SaveFormat.Pptx);
}
```
### Советы по устранению неполадок
- **Форма не отображается**: Убедитесь, что координаты и размеры фигуры заданы правильно.
- **Нет видимого 3D-эффекта**: Проверьте глубину, настройки камеры и конфигурацию осветительного оборудования.

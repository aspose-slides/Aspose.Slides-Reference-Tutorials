## Руководство по внедрению

### Создание каталога, если он не существует

**Обзор:**
Эта функция обеспечивает наличие каталога для хранения документов и его создание при необходимости.

#### Шаг 1: Определите каталог документов
Сначала укажите путь к каталогу документов в переменной.
```csharp
string dataDir = @"YOUR_DOCUMENT_DIRECTORY";
```

#### Шаг 2: Проверка и создание каталога
Использовать `Directory.Exists` для проверки существования каталога. Если он не существует, создайте его с помощью `Directory.CreateDirectory`.
```csharp
bool isExists = Directory.Exists(dataDir);
if (!isExists)
{
    // Это создаст новый каталог по указанному пути, если он еще не существует.
    Directory.CreateDirectory(dataDir);
}
```
**Параметры и назначение:**
- `dataDir`: Путь к целевому каталогу. 
- `Directory.Exists`: Возвращает true, если каталог существует.
- `Directory.CreateDirectory`: Создает каталог, указанный по пути.

### Создание презентации и добавление прямоугольной фигуры с текстом

**Обзор:**
Эта функция демонстрирует, как создать новую презентацию, добавить прямоугольную фигуру и включить в нее текст с помощью Aspose.Slides для .NET.

#### Шаг 1: Создание презентации
Создать экземпляр `Presentation` который представляет ваш файл PowerPoint.
```csharp
string outputDir = @"YOUR_OUTPUT_DIRECTORY";
using (Presentation pres = new Presentation())
{
    // Доступ к первому слайду презентации
    ISlide sld = pres.Slides[0];
```

#### Шаг 2: Добавьте прямоугольную форму.
Добавьте на слайд автофигуру прямоугольного типа.
```csharp
    IAutoShape ashp = sld.Shapes.AddAutoShape(ShapeType.Rectangle, 150, 75, 150, 50);
    // Это добавляет прямоугольник в указанную позицию с заданными размерами (шириной и высотой).
```

#### Шаг 3: Вставьте текст в фигуру
Создайте текстовую рамку и добавьте текст в свою фигуру.
```csharp
    ashp.AddTextFrame(" ");
    ITextFrame txtFrame = ashp.TextFrame;
    IParagraph para = txtFrame.Paragraphs[0];
    IPortion portion = para.Portions[0];
    portion.Text = "Aspose TextBox";
    // Поместите текст внутри прямоугольника.
```

#### Шаг 4: Сохраните презентацию
Наконец, сохраните презентацию в желаемом месте.
```csharp
    pres.Save(outputDir + "TextBox_out.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
}
// Это сохранит файл в формате PPTX с указанным именем.
```

1. **Каково основное применение Aspose.Slides для .NET?**

2. **Как проверить существование каталога в C#?**

3. **Могу ли я добавлять другие фигуры, кроме прямоугольников?**

### Экспорт PowerPoint в адаптивный HTML

#### Обзор
Эта функция позволяет преобразовывать слайды PowerPoint в удобный для веб-пространства формат, который динамически адаптируется к различным размерам экрана, обеспечивая оптимальный просмотр на любом устройстве.

#### Шаги по реализации
**Шаг 1: Определите каталоги**
Сначала укажите входной и выходной каталоги. Заменить `"YOUR_DOCUMENT_DIRECTORY"` и `"YOUR_OUTPUT_DIRECTORY"` с реальными путями.
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputPath = "YOUR_OUTPUT_DIRECTORY";
```

**Шаг 2: Загрузите презентацию**
Затем загрузите файл PowerPoint с помощью Aspose.Slides:
```csharp
Presentation presentation = new Presentation(dataDir + "SomePresentation.pptx");
```
The `Presentation` класс из Aspose.Slides представляет файл PowerPoint.

**Шаг 3: Настройте параметры экспорта HTML**
Настройте параметры экспорта, чтобы обеспечить адаптивность HTML-макета. Это включает включение адаптивного макета на основе SVG:
```csharp
HtmlOptions saveOptions = new HtmlOptions();
saveOptions.SvgResponsiveLayout = true; // Включает адаптивную компоновку SVG
```
The `SvgResponsiveLayout` свойство обеспечивает правильное масштабирование векторной графики, сохраняя качество на всех устройствах.

**Шаг 4: Сохранить как HTML**
Наконец, экспортируйте презентацию в HTML-файл, используя настроенные вами параметры:
```csharp
presentation.Save(outputPath + "SomePresentation-out.html", SaveFormat.Html, saveOptions);
```
The `Save` метод сохраняет презентацию в указанном формате с предоставленными параметрами.

#### Советы по устранению неполадок
- **Файл не найден**: Убедитесь, что пути указаны правильно и файлы существуют.
- **Проблемы с SVG**: Проверьте совместимость браузера с SVG, если на определенных устройствах возникают проблемы с отображением.

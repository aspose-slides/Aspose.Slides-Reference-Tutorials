## Руководство по внедрению

### Сохранение презентации в формате HTML с пользовательскими шрифтами

#### Обзор
Эта функция демонстрирует, как преобразовать презентацию PowerPoint в HTML, указав различные стандартные шрифты по умолчанию. Это обеспечивает единообразную типографику на различных платформах.

#### Пошаговая реализация

**1. Настройте пути к документам:**
Начните с определения путей к каталогам для исходного файла PPT и выходного HTML.
```csharp
string dataDir = "/path/to/your/documents";
string outPath = "/output/directory";
```

**2. Загрузите презентацию:**
Использовать `Presentation` класс для загрузки вашего файла PowerPoint.
```csharp
using (Presentation pres = new Presentation(dataDir + "/DefaultFonts.pptx"))
{
    // Дальнейшие шаги будут описаны здесь...
}
```
*Почему?* Загрузка презентации необходима, поскольку она подготавливает документ к дальнейшим манипуляциям.

**3. Создайте параметры HTML:**
Инициализировать `HtmlOptions` чтобы указать, как вы хотите преобразовать свой PPT.
```csharp
HtmlOptions htmlOpts = new HtmlOptions();
```

**4. Установить обычный шрифт по умолчанию:**
Настройте шрифт по умолчанию, используемый в процессе конвертации.
```csharp
htmlOpts.DefaultRegularFont = "Arial Black";
pres.Save(outPath + "/Presentation-out-ArialBlack.html", SaveFormat.Html, htmlOpts);
```
*Почему?* Установка пользовательского шрифта гарантирует, что ваша презентация сохранит визуальную согласованность при просмотре в формате HTML.

#### Советы по устранению неполадок:
- **Ошибки пути к файлу:** Еще раз проверьте пути к каталогам на предмет опечаток.
- **Отсутствующие шрифты:** Убедитесь, что указанные шрифты доступны в вашей системе.

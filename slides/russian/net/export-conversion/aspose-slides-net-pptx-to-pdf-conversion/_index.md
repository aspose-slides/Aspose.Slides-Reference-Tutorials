## Руководство по внедрению

### Конвертировать презентацию в PDF
Эта функция позволяет конвертировать презентации PowerPoint в высококачественные файлы PDF с помощью Aspose.Slides для .NET.

#### Шаг 1: Создание объекта презентации
Сначала загрузите ваш файл PPTX в экземпляр `Presentation` класс. Этот объект представляет вашу презентацию в памяти.

```csharp
using Aspose.Slides;
using Aspose.Slides.Export;

// Загрузить презентацию PowerPoint из указанного пути
Presentation presentation = new Presentation("YOUR_DOCUMENT_DIRECTORY/ConvertToPDF.pptx");
```

#### Шаг 2: Сохраните презентацию в формате PDF
Теперь используйте `Save` метод конвертации и сохранения вашей презентации в виде файла PDF.

```csharp
// Конвертируйте и сохраните презентацию как PDF-документ
presentation.Save("YOUR_OUTPUT_DIRECTORY/output_out.pdf", SaveFormat.Pdf);
```

### Загрузка и сохранение презентаций в разных форматах
Эта функция демонстрирует, как загрузить существующий файл PPTX и сохранить его в другом формате, например PDF.

#### Шаг 1: Загрузите существующую презентацию
Используйте `Presentation` класс, чтобы открыть нужный вам файл PowerPoint.

```csharp
// Открыть файл презентации
type loadedPresentation = new Presentation("YOUR_DOCUMENT_DIRECTORY/sample.pptx");
```

#### Шаг 2: Сохраните в другом формате
Выберите нужный вам формат и сохраните презентацию соответствующим образом.

```csharp
// Сохраните презентацию в формате PDF или любом другом поддерживаемом формате.
loadedPresentation.Save("YOUR_OUTPUT_DIRECTORY/saved_output.pdf", SaveFormat.Pdf);
```

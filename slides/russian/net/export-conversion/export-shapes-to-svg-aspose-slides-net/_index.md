## Руководство по внедрению

Изучите подробное руководство по экспорту фигур в формат SVG:

### Экспортировать форму в SVG

#### Обзор
Экспортируйте фигуры из любого слайда PowerPoint в файл SVG, что полезно для интеграции векторной графики в веб-приложения или программные системы, требующие масштабируемых форматов.

#### Пошаговое руководство
**1. Задайте пути для входных и выходных файлов**
Определите каталоги для входных и выходных файлов:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Каталог, содержащий файл PowerPoint
string outSvgFileName = "YOUR_OUTPUT_DIRECTORY/SingleShape.svg"; // Путь к выходному файлу SVG
```

**2. Загрузите презентацию**
Загрузите презентацию с помощью Aspose.Slides:

```csharp
using (Presentation pres = new Presentation(dataDir + "/TestExportShapeToSvg.pptx"))
{
    // Доступ к первому слайду и его первой форме
    var slide = pres.Slides[0];
    var shape = slide.Shapes[0];

    // Создать FileStream для выходного SVG-файла
    using (Stream stream = new FileStream(outSvgFileName, FileMode.Create, FileAccess.Write))
    {
        // Экспортировать форму в формат SVG
        shape.WriteAsSvg(stream);
    }
}
```

**Объяснение:**
- `dataDir`: Каталог, содержащий ваш файл PowerPoint.
- `outSvgFileName`: Путь, по которому будет сохранен экспортированный SVG.
- **`Presentation` Объект**: Представляет документ PowerPoint.
- **`Slide.Shapes[0]`**: Доступ к первой форме первого слайда для экспорта.

### Советы по устранению неполадок
- Убедитесь, что путь к входному файлу правильный и доступный.
- Проверьте права доступа к файлу, чтобы подтвердить право записи в выходной каталог.
- Убедитесь, что файл PowerPoint не поврежден, открыв его в Microsoft PowerPoint.

## Настройка Aspose.Slides для .NET

Чтобы начать использовать Aspose.Slides, установите библиотеку в свой проект.

**Установка через .NET CLI:**

```bash
dotnet add package Aspose.Slides
```

**Использование менеджера пакетов:**

```powershell
Install-Package Aspose.Slides
```

Или найдите «Aspose.Slides» с помощью пользовательского интерфейса диспетчера пакетов NuGet в Visual Studio и установите последнюю версию.

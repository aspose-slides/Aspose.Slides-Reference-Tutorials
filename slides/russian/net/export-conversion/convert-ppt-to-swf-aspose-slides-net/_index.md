## Руководство по внедрению

Мы рассмотрим преобразование презентаций в SWF с опциями для зрителей и настройкой положения заметок.

### Функция 1: Преобразование презентации в SWF

#### Обзор
Эта функция демонстрирует, как преобразовать презентацию PowerPoint в формат SWF. Вы можете включить или исключить встроенный просмотрщик в выходной файл.

**Пошаговая реализация:**

##### Шаг 1: Инициализация объекта презентации
Начните с загрузки файла PowerPoint с помощью Aspose.Slides:

```csharp
using Aspose.Slides;
using Aspose.Slides.Export;

string dataDir = "YOUR_DOCUMENT_DIRECTORY";

// Загрузить презентацию
using (Presentation presentation = new Presentation(dataDir + "/HelloWorld.pptx"))
{
    // Продолжить настройку параметров конвертации...
}
```

##### Шаг 2: Задайте параметры SWF
Настройте параметры преобразования SWF с помощью `SwfOptions`:

```csharp
SwfOptions swfOptions = new SwfOptions();
swfOptions.ViewerIncluded = false; // Начните без зрителя.
```

**Почему:** Эта опция позволяет вам решить, нужно ли вам интерактивное средство просмотра в вашем SWF-файле, что может быть решающим для презентаций, требующих взаимодействия с пользователем.

##### Шаг 3: Сохраните презентацию как SWF
Сохраните презентацию с указанными параметрами:

```csharp
// Сохранить без просмотра
presentation.Save(dataDir + "/SaveAsSwf_out.swf", SaveFormat.Swf, swfOptions);
```

**Шаг 4: Включить Viewer в вывод**
Чтобы включить интерактивное средство просмотра:

```csharp
swfOptions.ViewerIncluded = true;
presentation.Save(dataDir + "/SaveNotes_out.swf", SaveFormat.Swf, swfOptions);
```

### Функция 2: Конфигурация положения заметок

#### Обзор
Настройка положения нот позволяет контролировать отображение нот в выходном файле SWF, повышая ясность.

**Пошаговая реализация:**

##### Шаг 1: Доступ к параметрам макета заметки
Доступ и настройка макета заметок:

```csharp
INotesCommentsLayoutingOptions notesOptions = swfOptions.NotesCommentsLayouting;
notesOptions.NotesPosition = NotesPositions.BottomFull; // Установите полную ширину внизу.
```

**Почему:** Такая конфигурация гарантирует, что ваша аудитория сможет просматривать все заметки без прокрутки, что повышает удобство использования.

##### Шаг 2: Сохраните презентацию
Сохраните презентацию с настроенными позициями заметок:

```csharp
presentation.Save(dataDir + "/SaveWithNotes_out.swf", SaveFormat.Swf, swfOptions);
```

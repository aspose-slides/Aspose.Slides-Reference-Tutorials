Начните с указания каталогов для входных и выходных файлов:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Каталог, содержащий ваш файл PPTX
string outputDir = "YOUR_OUTPUT_DIRECTORY"; // Каталог для сохранения HTML-файла
const string htmlDocumentFileName = "presentationWithVideo.html";
```

#### Шаг 2: Загрузите презентацию

Загрузите презентацию PowerPoint с помощью Aspose.Slides:

```csharp
using (Presentation pres = new Presentation(dataDir + "\presentationWith.pptx"))
{
    // Продолжить шаги по интеграции медиа можно здесь...
}
```
*The `Presentation` класс загружает файл PPTX, позволяя нам манипулировать слайдами и встроенными медиафайлами.*

#### Шаг 3: Создайте пользовательский HTML-контроллер

Для обработки видеоплееров в HTML-выводе используйте пользовательский HTML-контроллер:


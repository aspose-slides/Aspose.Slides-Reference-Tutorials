## Руководство по внедрению

Давайте рассмотрим скрытие и отображение рукописных аннотаций во время экспорта PDF, а также настройку операций ROP для рендеринга.

### Скрыть рукописные примечания в экспортированных PDF-файлах

#### Обзор

При экспорте презентации в формате PDF вам может потребоваться удалить рукописные аннотации (например, рукописные заметки), чтобы документ выглядел чистым. Эта функция особенно полезна при подготовке презентаций для профессионального распространения.

#### Этапы внедрения
1. **Загрузите презентацию:**
   Начните с загрузки файла PowerPoint в `Presentation` объект.
   
   ```csharp
   string dataDir = "YOUR_DOCUMENT_DIRECTORY";
   using (Presentation pres = new Presentation(dataDir + "/InkOptions.pptx"))
   {
       // Код продолжается...
   }
   ```

2. **Настройте параметры экспорта PDF:**
   Настройте `PdfOptions` чтобы скрыть объекты чернил, установив `HideInk` к истине.
   
   ```csharp
   PdfOptions options = new PdfOptions();
   options.InkOptions.HideInk = true;
   ```

3. **Экспорт в PDF:**
   Сохраните презентацию с указанными параметрами, получив в результате чистый PDF-файл без рукописных пометок.
   
   ```csharp
   string outFilePath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "HideInkDemo.pdf");
   pres.Save(outFilePath, SaveFormat.Pdf, options);
   ```

### Показывать рукописные аннотации и настраивать операции ROP

#### Обзор
Для презентаций, где аннотации имеют решающее значение, вы можете выбрать отображение объектов чернил в экспортированном PDF. Кроме того, настройка параметров Raster Operation (ROP) позволяет настраивать рендеринг этих аннотаций.

#### Этапы внедрения
1. **Загрузите презентацию:**
   Как и прежде, загрузите свою презентацию в `Presentation` объект.
   
   ```csharp
   string dataDir = "YOUR_DOCUMENT_DIRECTORY";
   using (Presentation pres = new Presentation(dataDir + "/InkOptions.pptx"))
   {
       // Код продолжается...
   }
   ```

2. **Настройте параметры экспорта PDF:**
   На этот раз установите `HideInk` на false и настройте параметры ROP, установив `InterpretMaskOpAsOpacity`.
   
   ```csharp
   PdfOptions options = new PdfOptions();
   options.InkOptions.HideInk = false;
   options.InkOptions.InterpretMaskOpAsOpacity = false; // Стандартная интерпретация ROP
   ```

3. **Экспорт в PDF:**
   Сохраните презентацию, демонстрирующую объекты чернил с выбранными вами настройками рендеринга.
   
   ```csharp
   string outFilePath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "ROPInkDemo.pdf");
   pres.Save(outFilePath, SaveFormat.Pdf, options);
   ```

#### Советы по устранению неполадок
- Убедитесь, что пути к файлам указаны правильно, чтобы избежать `FileNotFoundException`.
- Если объекты рукописного ввода не отображаются должным образом, еще раз проверьте настройки ROP и убедитесь, что ваша презентация содержит видимые аннотации.

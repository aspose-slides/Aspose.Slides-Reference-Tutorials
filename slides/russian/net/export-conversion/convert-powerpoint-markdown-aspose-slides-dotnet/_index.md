Завершив настройку, давайте преобразуем презентацию PowerPoint в формат Markdown с помощью Aspose.Slides .NET.

### Преобразовать презентацию в Markdown

Эта функция иллюстрирует преобразование слайдов PowerPoint в файлы Markdown с сохранением их структуры и любых включенных медиафайлов.

#### Инициализация объекта презентации

Начните с загрузки файла презентации:

```csharp
using System.IO;
using Aspose.Slides;

string presentationName = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "PresentationDemo.pptx");
// Загрузите презентацию с помощью Aspose.Slides
using (Presentation pres = new Presentation(presentationName))
{
    // Код продолжается...
}
```

#### Настройка параметров преобразования Markdown

Настройте параметры конверсии, используя `MarkdownSaveOptions`:

```csharp
string outPath = "YOUR_OUTPUT_DIRECTORY";  // Определить выходной каталог для файлов Markdown

// Создать и настроить MarkdownSaveOptions
MarkdownSaveOptions mdOptions = new MarkdownSaveOptions();
mdOptions.ExportType = MarkdownExportType.Visual; // Выберите визуальный тип экспорта
mdOptions.ImagesSaveFolderName = "md-images";    // Укажите папку для изображений
mdOptions.BasePath = outPath;                     // Установить базовый путь

// Сохраните презентацию как файл Markdown
pres.Save(Path.Combine(outPath, "pres.md"), SaveFormat.Md, mdOptions);
```

**Объяснение основных параметров:**
- `ExportType`: Определяет, как экспортируется контент. Визуальный тип включает все элементы в их исходном макете.
- `ImagesSaveFolderName` и `BasePath`: Определите, где будут сохранены изображения, извлеченные из презентации.

### Советы по устранению неполадок

- Перед сохранением файлов убедитесь, что выходной каталог существует, чтобы избежать исключений.
- Проверьте правильность пути к папке с изображениями и доступность их, если они не отображаются должным образом.

### Замена шрифта

**Обзор**
Замена шрифта необходима, когда исходный шрифт недоступен в целевой системе. С помощью Aspose.Slides вы можете определить правила для бесшовной замены шрифтов во время рендеринга презентации.

#### Пошаговое руководство
1. **Загрузите вашу презентацию**
   Начните с загрузки файла презентации в `Presentation` объект:
   
   ```csharp
   string dataDir = "YOUR_DOCUMENT_DIRECTORY";
   Presentation presentation = new Presentation(dataDir + "/Fonts.pptx");
   ```

2. **Определить шрифты для замены**
   Укажите исходный шрифт, который необходимо заменить, и целевой шрифт:
   
   ```csharp
   IFontData sourceFont = new FontData("SomeRareFont");
   IFontData destFont = new FontData("Arial");
   ```

3. **Создать правило замены шрифта**
   Настройте правило замены для замены исходного шрифта на целевой шрифт, когда он недоступен:
   
   ```csharp
   IFontSubstRule fontSubstRule = new FontSubstRule(sourceFont, destFont, FontSubstCondition.WhenInaccessible);
   ```

4. **Добавить правило в коллекцию**
   Инициализируйте и добавьте свое правило подстановки в коллекцию в `FontsManager`:
   
   ```csharp
   IFontSubstRuleCollection fontSubstRuleCollection = new FontSubstRuleCollection();
   fontSubstRuleCollection.Add(fontSubstRule);
   presentation.FontsManager.FontSubstRuleList = fontSubstRuleCollection;
   ```

5. **Советы по устранению неполадок**
   - Убедитесь, что целевой шрифт установлен в вашей системе.
   - Проверьте пути к файлам и убедитесь, что они доступны.

### Экспорт слайд-изображения

**Обзор**
Экспорт изображений слайдов может быть полезен для создания миниатюр или интеграции слайдов в другие медиаформаты.

#### Пошаговое руководство
1. **Загрузите вашу презентацию**
   Как и прежде, загрузите презентацию:
   
   ```csharp
   string dataDir = "YOUR_DOCUMENT_DIRECTORY";
   Presentation presentation = new Presentation(dataDir + "/Fonts.pptx");
   ```

2. **Извлечь и сохранить слайд как изображение**
   Использовать `GetThumbnail` чтобы создать изображение слайда и сохранить его в формате JPEG:
   
   ```csharp
   IImage img = presentation.Slides[0].GetThumbnail(1f, 1f);
   img.Save(dataDir + "/Slide_Image_out.jpg", ImageFormat.Jpeg);
   ```

3. **Советы по устранению неполадок**
   - Проверьте права доступа к выходному каталогу.
   - Обеспечить `ImageFormat` указано верно.

## Раздел часто задаваемых вопросов

**1. Что такое замена шрифтов в Aspose.Slides?**
Замена шрифта заменяет отсутствующий исходный шрифт указанным целевым шрифтом во время рендеринга презентации.

**2. Как экспортировать слайды как изображения с помощью Aspose.Slides?**
Используйте `GetThumbnail` метод на объекте слайда и сохраните его в желаемом формате, например JPEG.

**3. Могу ли я использовать разные форматы изображений для экспорта слайдов?**
Да, вы можете указать различные форматы изображений, поддерживаемые .NET. `ImageFormat`.

**4. Что произойдет, если целевой шрифт не установлен в моей системе?**
Замена не удастся; убедитесь, что целевой шрифт доступен, чтобы избежать проблем.

**5. Как обрабатывать презентации с несколькими слайдами в Aspose.Slides?**
Повторите все `Slides` собирать и применять логику обработки, например экспорт изображений или замену шрифтов, к каждому слайду по отдельности.

### Руководство по внедрению
#### Встраивание шрифтов в HTML
Внедрение пользовательских шрифтов обеспечивает единообразие типографики. Вот как это сделать с помощью Aspose.Slides для .NET.

##### Шаг 1: Загрузите презентацию PowerPoint
Создать `Presentation` пример для загрузки вашего файла PPTX:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outPath = "YOUR_OUTPUT_DIRECTORY";

using (Presentation pres = new Presentation(dataDir + "Presentation.pptx"))
{
    // Дальнейшие шаги будут здесь
}
```
##### Шаг 2: Настройте шрифты для встраивания
Укажите, какие шрифты вы хотите внедрить, и исключите определенные системные шрифты:
```csharp
string[] fontNameExcludeList = { "Arial" };
pres.FontsManager.EmbedAllFontsExcept(fontNameExcludeList);
```
Это сообщает Aspose.Slides о необходимости встраивать все пользовательские шрифты, за исключением перечисленных в `fontNameExcludeList`.

##### Шаг 3: Сохраните презентацию как HTML
Сохраните презентацию со встроенными шрифтами:
```csharp
HtmlOptions htmlOpt = new HtmlOptions();
htmlOpt.HtmlFormatter = HtmlFormatter.CreateDocumentFormatter("", false);
pres.Save(outPath + "Presentation.html", SaveFormat.Html, htmlOpt);
```
Это преобразует вашу презентацию в HTML-файл со встроенными указанными шрифтами.

### Раздел часто задаваемых вопросов
**В1: Можно ли встроить несколько шрифтов в один HTML-файл?**
A1: Да, укажите несколько пользовательских шрифтов для встраивания. Убедитесь, что они включены в настройки встраивания шрифтов.

**В2: Что произойдет, если встроенный шрифт недоступен в системе пользователя?**
A2: Браузер будет использовать встроенную версию шрифта вместо системных шрифтов по умолчанию.

**В3: Как мне оформить лицензирование пользовательских шрифтов?**
A3: Убедитесь, что у вас есть право встраивать и распространять шрифты. Некоторые лицензии могут ограничивать встраивание в цифровые файлы.

**В4: Влияют ли встроенные шрифты на производительность?**
A4: Да, большие файлы шрифтов могут увеличить время загрузки. Оптимизируйте, внедряя только необходимые символы и подмножества.

**В5: Могу ли я исключить внедрение пользовательских шрифтов на определенные слайды?**
A5: Aspose.Slides в настоящее время встраивает шрифты для всей презентации. Пользовательский контроль слайдов может потребовать дополнительной логики или ручных настроек после экспорта.

### Функция 1: Создание и сохранение презентации PowerPoint для потоковой передачи

#### Обзор
Эта функция позволяет создать простую презентацию PowerPoint, вставить текстовый контент и сохранить его непосредственно в виде потока для дальнейшей обработки или хранения.

##### Пошаговое руководство

**Создать новую презентацию**
Начните с создания экземпляра `Presentation` класс, представляющий ваш файл PowerPoint:

```csharp
using Aspose.Slides;

namespace PresentationToStreamExample
{
    public class SavePresentationToStream
    {
        public static void Main()
        {
            string dataDir = @"YOUR_DOCUMENT_DIRECTORY"; // Укажите здесь путь к вашему каталогу

            using (Presentation presentation = new Presentation())
            {
                // Продолжайте манипулировать слайдами...
```

**Добавьте текстовую фигуру на первый слайд**
Добавьте автофигуру типа прямоугольник и вставьте в нее текст:

```csharp
                IAutoShape shape = presentation.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 200, 200, 200, 200);
                shape.TextFrame.Text = "This demo shows how to Create PowerPoint file and save it to Stream.";
```

**Сохранить презентацию как поток**
Определите поток, в котором будет сохранена ваша презентация:

```csharp
                using (FileStream toStream = new FileStream(dataDir + "Save_As_Stream_out.pptx", FileMode.Create))
                {
                    // Сохраните презентацию в потоке.
                    presentation.Save(toStream, Aspose.Slides.Export.SaveFormat.Pptx);
                }
            }
        }
    }
}
```

**Объяснение:**
- `Presentation` обрабатывает файлы PowerPoint в памяти.
- На первый слайд добавляется прямоугольная фигура с указанными размерами и координатами.
- FileStream используется для сохранения презентации в формате PPTX, что обеспечивает гибкую обработку данных.

## Руководство по внедрению
Давайте углубимся в преобразование презентаций PowerPoint с сохранением шрифтов с помощью Aspose.Slides for .NET. Мы разберем это пошагово:

### Обзор функций
Эта функция позволяет преобразовывать файлы PPTX в документы HTML, сохраняя исходные стили шрифтов в том виде, в котором они отображаются в презентации.

#### Шаг 1: Загрузите презентацию
Начните с загрузки файла PowerPoint в `Presentation` объект. Это имеет решающее значение для доступа к слайдам и управления ими.

```csharp
using Aspose.Slides;
using Aspose.Slides.Export;

string dataDir = @"YOUR_DOCUMENT_DIRECTORY";
using (Presentation pres = new Presentation(dataDir + "input.pptx"))
{
    // Дальнейшая обработка здесь
}
```

**Объяснение:** Начнем с создания `Presentation` объект, который позволяет нам взаимодействовать со слайдами в вашем файле PowerPoint.

#### Шаг 2: Настройте параметры шрифта
При желании укажите любые шрифты, которые вы хотите исключить из встраивания в HTML. Это может оптимизировать время загрузки и уменьшить размер файла.

```csharp
string[] fontNameExcludeList = { "Calibri" };
```

**Объяснение:** The `fontNameExcludeList` Массив определяет, какие шрифты не следует встраивать в конечный HTML-документ, помогая эффективно управлять использованием ресурсов.

#### Шаг 3: Преобразование в HTML
Далее преобразуйте слайды презентации в формат HTML. Вы можете дополнительно настроить этот процесс, указав дополнительные параметры, если это необходимо.

```csharp
pres.Save(outputDir + "output.html", SaveFormat.Html5);
```

**Объяснение:** The `Save` метод экспортирует презентацию как HTML-документ, с `Html5` обеспечение совместимости с современными веб-браузерами.

### Советы по устранению неполадок:
- Обеспечить пути в `dataDir` и `outputDir` верны.
- Проверьте, доступны ли исключенные шрифты на целевых устройствах, чтобы избежать потери стилей.

### Инициализация презентации и настройка параметров

Сначала загрузите презентацию PowerPoint. Затем настройте параметры PDF, чтобы включить обработчик обратного вызова прогресса.

#### Шаг 1: Загрузите презентацию

```csharp

using Aspose.Slides.Export;

namespace ConversionFeature
{
    public class ConvertToPDFWithProgressUpdate
    {
        public static void Run()
        {
            string dataDir = @"YOUR_DOCUMENT_DIRECTORY";

            using (Presentation presentation = new Presentation(dataDir + "ConvertToPDF.pptx"))
            {
                // Далее следуют шаги настройки...
```

#### Шаг 2: Настройте параметры PDF

Настройте `PdfOptions` включить обратный вызов хода выполнения, предоставляя обновления во время преобразования.

```csharp
                ISaveOptions saveOptions = new PdfOptions();
                saveOptions.ProgressCallback = new ExportProgressHandler();

                presentation.Save(dataDir + "ConvertToPDF.pdf", SaveFormat.Pdf, saveOptions);
            }
        }
    }
}
```

#### Объяснение параметров
- **`PdfOptions`**: Позволяет указать различные параметры преобразования PDF.
- **`ExportProgressHandler`**: Пользовательский класс, реализующий `IProgressCallback`, предоставляя обновления в режиме реального времени.

## Руководство по внедрению
### Экспорт PowerPoint в HTML с пользовательским CSS
Конвертируйте презентации в стилизованные HTML-файлы с помощью пользовательского CSS.

#### Шаг 1: Определите каталоги и загрузите презентацию
Настройте каталоги документов и выходных данных, затем загрузите презентацию:
```csharp
using Aspose.Slides;
using Aspose.Slides.Export;

string dataDir = "YOUR_DOCUMENT_DIRECTORY";  // Расположение исходного файла.
string outputDir = "YOUR_OUTPUT_DIRECTORY";    // Сохранить местоположение HTML.

// Загрузите файл PowerPoint
using (Presentation pres = new Presentation(dataDir + "/pres.pptx"))
{
    // Реализация продолжается здесь...
}
```

#### Шаг 2: применение пользовательского CSS с контроллером
Создайте пользовательский контроллер заголовка и шрифтов для управления стилями:
```csharp
CustomHeaderAndFontsController htmlController = new CustomHeaderAndFontsController(outputDir + "/styles.css");
```
На этом этапе настраивается внедрение пользовательского CSS в экспортированный HTML.

#### Шаг 3: Настройте параметры экспорта
Задайте параметры экспорта в формате HTML с помощью Aspose.Slides:
```csharp
HtmlOptions options = new HtmlOptions
{
    HtmlFormatter = HtmlFormatter.CreateCustomFormatter(htmlController),  // Примените здесь свой пользовательский форматировщик.
};
```
The `HtmlFormatter` позволяет настраивать отображение слайдов в формате HTML.

#### Шаг 4: Сохранить как HTML
Сохраните презентацию с указанными параметрами:
```csharp
pres.Save(outputDir + "/pres.html", SaveFormat.Html, options);
```
Это сохранит презентацию в HTML-файле в нужном вам месте, применив все заданные пользовательские стили.

### Советы по устранению неполадок
- **Пути к файлам**: Убедитесь, что пути к исходному и выходному каталогам указаны правильно.
- **CSS-стили**: Проверьте синтаксис CSS в `styles.css` чтобы избежать проблем с рендерингом.

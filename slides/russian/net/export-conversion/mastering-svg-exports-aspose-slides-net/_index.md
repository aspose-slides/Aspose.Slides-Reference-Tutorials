### Функция: Форматирование SVG-фигур и текста с помощью Aspose.Slides
Эта функция позволяет вам настраивать `tspan` Атрибут Id при экспорте слайдов в формат SVG гарантирует, что ваши текстовые элементы будут однозначно идентифицированы и стилизованы по мере необходимости.

#### Шаг 1: Настройка среды
Убедитесь, что ваш проект ссылается на Aspose.Slides. Определите каталоги для ввода и вывода:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string pptxFileName = Path.Combine(dataDir, "Convert_Svg_Custom.pptx");
string outSvgFileName = Path.Combine("YOUR_OUTPUT_DIRECTORY", "Convert_Svg_Custom.svg");

using (Presentation pres = new Presentation(pptxFileName))
{
    using (FileStream stream = new FileStream(outSvgFileName, FileMode.Create))
    {
        // Настройте параметры экспорта SVG
        SVGOptions svgOptions = new SVGOptions
        {
            ShapeFormattingController = new MySvgShapeFormattingController()
        };

        // Экспортировать слайд в файл SVG
        pres.Slides[0].WriteAsSvg(stream, svgOptions);
    }
}
```

#### Шаг 2: Создание пользовательской формы SVG и контроллера форматирования текста
Осуществлять `MySvgShapeFormattingController` для управления уникальными идентификаторами фигур и текстовых диапазонов:
```csharp
using Aspose.Slides.Export;

class MySvgShapeFormattingController : ISvgShapeAndTextFormattingController
{
    private int m_shapeIndex, m_portionIndex, m_tspanIndex;

    public MySvgShapeFormattingController(int shapeStartIndex = 0)
    {
        m_shapeIndex = shapeStartIndex;
        m_portionIndex = 0;
    }

    public void FormatShape(ISvgShape svgShape, IShape shape)
    {
        svgShape.Id = $"shape-{m_shapeIndex++}";
        m_portionIndex = m_tspanIndex = 0; // Сбросить индексы для форматирования текста
    }

    public void FormatText(ISvgTSpan svgTSpan, IPortion portion, ITextFrame textFrame)
    {
        int paragraphIndex = 0, portionIndex = 0;
        
        foreach (IParagraph para in textFrame.Paragraphs)
        {
            portionIndex = para.Portions.IndexOf(portion);
            if (portionIndex > -1) { paragraphIndex = Array.IndexOf(textFrame.Paragraphs.ToArray(), para); break; }
        }

        if (m_portionIndex != portionIndex)
        {
            m_tspanIndex = 0;
            m_portionIndex = portionIndex;
        }

        svgTSpan.Id = $"paragraph-{paragraphIndex}_portion-{m_portionIndex}_{m_tspanIndex++}";
    }

    public ISvgShapeFormattingController AsISvgShapeFormattingController => this;
}
```
**Основные параметры конфигурации:** Установив `svgOptions.ShapeFormattingController`, вы настраиваете способ экспорта фигур и текста, гарантируя, что каждому из них будет присвоен уникальный идентификатор.

## Раздел часто задаваемых вопросов
**В: Как обеспечить уникальные идентификаторы для всех элементов SVG?**
A: Реализуйте пользовательский контроллер форматирования, как показано выше, который назначает последовательные или вычисляемые идентификаторы на основе ваших критериев.

**В: Может ли Aspose.Slides экспортировать данные в форматы, отличные от SVG?**
A: Да, Aspose.Slides поддерживает различные форматы, включая PDF и изображения, такие как PNG и JPEG.

**В: Что делать, если мой выходной SVG-файл отличается от исходного слайда?**
A: Проверьте настройки форматирования и убедитесь, что все пользовательские контроллеры применены правильно. Различия также могут возникать из-за внутренних ограничений векторизации.

**В: Как управлять лицензиями на Aspose.Slides?**
A: Начните с бесплатной пробной версии, получите временную лицензию для оценки или приобретите полную лицензию на веб-сайте Aspose.

**В: Какие проблемы чаще всего возникают при экспорте SVG?**
A: Следите за отсутствием шрифтов и убедитесь, что все ресурсы (изображения и т. д.) встроены. Протестируйте на разных просмотрщиках, чтобы проверить совместимость.

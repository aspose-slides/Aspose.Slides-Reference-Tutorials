## Руководство по внедрению

Мы рассмотрим две основные функции: настройку размера слайда и настройку параметров экспорта в PDF.

### Установка размера слайда презентации на A4

#### Обзор

Эта функция гарантирует, что ваши слайды идеально уместятся на листе формата А4, сохраняя соотношение сторон без обрезки или искажения.

**Этапы реализации:**
1. **Создание объекта презентации:** Создайте новый объект презентации.
    ```csharp
    Presentation presentation = new Presentation();
    ```
2. **Задайте тип и масштаб размера слайда:** Используйте `SetSize` способ подогнать размер слайда под формат А4, обеспечив его правильное размещение.
    ```csharp
    // Установите SlideSize.Type на размер бумаги A4 с типом масштабирования EnsureFit
    presentation.SlideSize.SetSize(SlideSizeType.A4Paper, SlideSizeScaleType.EnsureFit);
    ```
3. **Сохранить презентацию:** Сохраните файл презентации в формате PPTX.
    ```csharp
    // Сохранить презентацию на диск
    presentation.Save("YOUR_OUTPUT_DIRECTORY/SetSlideSize_out.pptx", SaveFormat.Pptx);
    ```

**Основные параметры конфигурации:**
- `SlideSizeType.A4Paper`: Указывает размер бумаги A4.
- `SlideSizeScaleType.EnsureFit`Обеспечивает размещение контента в границах слайда.

### Настройка параметров экспорта PDF

#### Обзор
Настройте параметры экспорта PDF-файлов, чтобы добиться высокого разрешения на выходе, что сделает их идеальными для печати или распространения.

**Этапы реализации:**
1. **Загрузить существующую презентацию:** Инициализируйте объект презентации из существующего файла.
    ```csharp
    Presentation presentation = new Presentation("YOUR_INPUT_FILE.pptx");
    ```
2. **Создание и настройка параметров PDF:** Создайте экземпляр `PdfOptions` класс для определения настроек PDF.
    ```csharp
    // Настройте параметры PDF для высокого разрешения
    PdfOptions opts = new PdfOptions();
    opts.SufficientResolution = 600;
    ```
3. **Экспорт в формате PDF с параметрами:** Сохраните презентацию в формате PDF, применив указанные параметры экспорта.
    ```csharp
    // Экспорт в PDF с заданными настройками
    presentation.Save("YOUR_OUTPUT_DIRECTORY/SetPDFPageSize_out.pdf", SaveFormat.Pdf, opts);
    ```

**Основные параметры конфигурации:**
- `SufficientResolution`: Управляет разрешением экспортируемого PDF. Более высокое значение обеспечивает лучшее качество.

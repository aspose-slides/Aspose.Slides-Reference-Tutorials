# Создание миниатюр в слайдах с пользовательскими размерами


Создание пользовательских миниатюрных изображений ваших презентаций PowerPoint может быть ценным активом, независимо от того, создаете ли вы интерактивное приложение, улучшаете пользовательский опыт или оптимизируете контент для различных платформ. В этом руководстве мы проведем вас через процесс создания пользовательских миниатюрных изображений из презентаций PowerPoint с помощью библиотеки Aspose.Slides для .NET. Эта мощная библиотека позволяет вам программно манипулировать, конвертировать и улучшать файлы PowerPoint в приложениях .NET.

## Предпосылки

Прежде чем приступить к созданию пользовательских миниатюрных изображений, убедитесь, что выполнены следующие предварительные условия:

### 1. Aspose.Slides для .NET

Вам необходимо установить библиотеку Aspose.Slides for .NET в вашем проекте. Если вы еще этого не сделали, вы можете найти необходимую документацию и ссылки для скачивания [здесь](https://reference.aspose.com/slides/net/).

### 2. Презентация PowerPoint

Убедитесь, что у вас есть презентация PowerPoint, из которой вы хотите создать пользовательское изображение миниатюры. Эта презентация должна быть доступна в каталоге вашего проекта.

### 3. Среда разработки

Для прохождения этого урока вам необходимо иметь практические навыки программирования .NET с использованием C# и настроенную среду разработки, например Visual Studio.

Теперь, когда мы рассмотрели предварительные условия, давайте разберем процесс создания пользовательских миниатюр на пошаговые инструкции.

## Импорт пространств имен

Во-первых, вам нужно включить требуемые пространства имен в ваш код C#. Эти пространства имен позволяют вам работать с Aspose.Slides и манипулировать презентациями PowerPoint.

```csharp
using Aspose.Slides;
using System.Drawing;
```

## Шаг 1: Загрузите презентацию

Для начала загрузите презентацию PowerPoint, из которой вы хотите создать пользовательское изображение миниатюры. Это достигается с помощью библиотеки Aspose.Slides.

```csharp
string FilePath = @"..\..\..\Sample Files\";
string srcFileName = FilePath + "User Defined Thumbnail.pptx";

// Создайте экземпляр класса Presentation, представляющего файл презентации.
using (Presentation pres = new Presentation(srcFileName))
{
    // Ваш код для генерации миниатюр будет здесь
}
```

## Шаг 2: Получите доступ к слайду

В загруженной презентации вам необходимо получить доступ к определенному слайду, из которого вы хотите сгенерировать пользовательское изображение миниатюры. Вы можете выбрать слайд по его индексу.

```csharp
// Доступ к первому слайду (при необходимости можно изменить индекс)
ISlide sld = pres.Slides[0];
```

## Шаг 3: Определите пользовательские размеры миниатюры

Укажите желаемые размеры для вашего пользовательского миниатюрного изображения. Вы можете определить ширину и высоту в пикселях в соответствии с требованиями вашего приложения.

```csharp
int desiredX = 1200; // Ширина
int desiredY = 800;  // Высота
```

## Шаг 4: Рассчитайте коэффициенты масштабирования

Чтобы сохранить соотношение сторон слайда, рассчитайте коэффициенты масштабирования для размеров X и Y на основе размера слайда и желаемых размеров.

```csharp
float ScaleX = (float)(1.0 / pres.SlideSize.Size.Width) * desiredX;
float ScaleY = (float)(1.0 / pres.SlideSize.Size.Height) * desiredY;
```

## Шаг 5: Создание миниатюрного изображения

Создайте полномасштабное изображение слайда с указанными пользовательскими размерами и сохраните его на диске в формате JPEG.

```csharp
// Создайте полномасштабное изображение
Bitmap bmp = sld.GetThumbnail(ScaleX, ScaleY);

// Сохраните изображение на диск в формате JPEG
bmp.Save(destFileName, System.Drawing.Imaging.ImageFormat.Jpeg);
```

Теперь, выполнив эти шаги, вы должны были успешно создать пользовательское миниатюрное изображение из вашей презентации PowerPoint.

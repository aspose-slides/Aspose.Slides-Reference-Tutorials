# Изменение диаграмм в PowerPoint с помощью Aspose.Slides

Aspose.Slides for .NET предоставляет мощные API для создания и изменения презентаций PowerPoint программным способом. Этот пример демонстрирует, как работать с диаграммами в существующих презентациях, обновляя данные диаграмм, добавляя новые серии и изменяя типы диаграмм — и все это без необходимости установки Microsoft PowerPoint.

## Инструкция по установке

1. Установка через менеджер пакетов NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Или с помощью .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Добавьте следующие директивы using:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Charts;
   ```

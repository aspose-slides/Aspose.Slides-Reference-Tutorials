# Создание кластеризованных столбчатых диаграмм с помощью Aspose.Slides для .NET

Aspose.Slides for .NET предоставляет мощные возможности для создания и управления презентациями PowerPoint программным способом. Этот пример демонстрирует, как создать кластеризованную столбчатую диаграмму с пользовательскими категориями и рядами данных.

## Установка

```csharp
// Установить через менеджер пакетов NuGet
Install-Package Aspose.Slides
```

Или добавьте следующую ссылку в файл вашего проекта:

```xml
<PackageReference Include="Aspose.Slides" Version="23.2.0" />
```

# Отображать заметки при конвертации презентации в HTML


В сегодняшнюю цифровую эпоху преобразование презентаций в формат HTML стало обычным требованием. Это позволяет вам легко делиться своими презентациями в Интернете, делая их доступными для более широкой аудитории. Aspose.Slides для .NET — это мощный инструмент, который упрощает этот процесс. В этом пошаговом руководстве мы проведем вас через процесс преобразования презентации в HTML с помощью Aspose.Slides для .NET.

## 1. Введение

Aspose.Slides для .NET — это надежный .NET API, который позволяет вам работать с презентациями PowerPoint программно. Одной из его ключевых особенностей является возможность конвертировать презентации в различные форматы, включая HTML. В этом руководстве мы сосредоточимся на том, как выполнить это преобразование без проблем.

## 2. Предпосылки

Прежде чем начать, убедитесь, что у вас выполнены следующие предварительные условия:

- Visual Studio установлена в вашей системе.
- Библиотека Aspose.Slides для .NET добавлена в ваш проект.

## 3. Создание среды

Для начала создайте новый проект C# в Visual Studio. Убедитесь, что в вашем проекте правильно указана библиотека Aspose.Slides.

## 4. Загрузка презентации

В коде C# используйте следующий фрагмент кода для загрузки презентации:

```csharp
string dataDir = "Your Document Directory";
string outPath = "Your Output Directory";

using (Presentation pres = new Presentation(dataDir + "Presentation.pptx"))
{
    // Ваш код здесь
}
```

## 5. Настройка параметров HTML

Далее нам нужно настроить параметры преобразования HTML. В частности, мы хотим расположить заметки внизу страниц HTML. Используйте следующий фрагмент кода для настройки параметров:

```csharp
HtmlOptions opt = new HtmlOptions();
INotesCommentsLayoutingOptions options = opt.NotesCommentsLayouting;
options.NotesPosition = NotesPositions.BottomFull;
```

## 6. Сохранение HTML-вывода

Теперь, когда мы загрузили презентацию и настроили параметры HTML, пришло время сохранить вывод HTML. Используйте следующий код для этого:

```csharp
pres.Save(dataDir + "Output.html", SaveFormat.Html, opt);
```

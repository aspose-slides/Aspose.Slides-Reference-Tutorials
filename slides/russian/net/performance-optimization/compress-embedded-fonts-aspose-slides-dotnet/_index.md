## Руководство по внедрению: сжатие встроенных шрифтов в презентациях
### Обзор
Эта функция помогает уменьшить размер файлов за счет сжатия встроенных шрифтов, что упрощает обмен презентациями.

#### Пошаговая реализация
##### 1. Определите пути для входных и выходных документов
Настройте пути для ваших файлов:
```csharp
string presentationName = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "presWithEmbeddedFonts.pptx");
string outPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "presWithEmbeddedFonts-out.pptx");
```
##### 2. Загрузите презентацию
Загрузите файл PowerPoint с помощью Aspose.Slides:
```csharp
using (Presentation pres = new Presentation(presentationName))
{
    // На этом объекте будут проводиться дальнейшие работы.
}
```
##### 3. Сжатие встроенных шрифтов
Вызов `CompressEmbeddedFonts` для оптимизации хранения шрифтов в файле:
```csharp
pres.FontsManager.CompressEmbeddedFonts();
```
*Почему?*Этот метод уменьшает размер данных встроенных шрифтов без потери качества.
##### 4. Сохраните измененную презентацию.
Сохраните презентацию с новыми настройками:
```csharp
pres.Save(outPath, Aspose.Slides.Export.SaveFormat.Pptx);
```
##### Проверка результатов сжатия
Сравните размеры файлов до и после сжатия:
```csharp
FileInfo fi = new FileInfo(presentationName);
Console.WriteLine("Source file size = {0:N0} bytes", fi.Length);

fi = new FileInfo(outPath);
Console.WriteLine("Result file size = {0:N0} bytes", fi.Length);
```
### Советы по устранению неполадок
- Убедитесь, что путь к входному файлу правильный и доступный.
- Проверьте наличие обновлений Aspose.Slides, которые могут включать исправления ошибок или улучшения.

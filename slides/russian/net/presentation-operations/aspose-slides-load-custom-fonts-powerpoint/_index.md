### Загрузка презентации с пользовательскими шрифтами

#### Обзор

Загрузка презентаций с определенными шрифтами гарантирует, что ваши слайды будут отображать текст именно так, как задумано. Это имеет решающее значение для поддержания целостности бренда и визуальной согласованности во всех документах.

#### Шаги

**1. Определите каталог документов**

Сначала укажите, где находятся ваши файлы:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
```

**2. Загрузка шрифтов в память**

Загрузите пользовательские шрифты из локального хранилища в память, чтобы гарантировать их доступность при необходимости:

```csharp
byte[] memoryFont1 = File.ReadAllBytes("customfonts\\CustomFont1.ttf");
byte[] memoryFont2 = File.ReadAllBytes("customfonts\\CustomFont2.ttf");
```

**3. Настройте параметры загрузки**

Настройте параметры загрузки для указания источников шрифтов:

```csharp
LoadOptions loadOptions = new LoadOptions();
loadOptions.DocumentLevelFontSources.FontFolders = new string[] { "assets\\fonts", "global\\fonts" };
loadOptions.DocumentLevelFontSources.MemoryFonts = new byte[][] { memoryFont1, memoryFont2 };
```

**4. Загрузите презентацию**

Подготовив шрифты и настроив параметры загрузки, вы можете загрузить свою презентацию:

```csharp
using (IPresentation presentation = new Presentation("MyPresentation.pptx", loadOptions))
{
    // В презентацию загружены указанные пользовательские шрифты.
}
```

#### Объяснение

- **`LoadOptions`:** Устанавливает исходные каталоги шрифтов и загружаемые в память шрифты.
- **`MemoryFonts`:** Массив байтовых массивов, представляющих шрифты, загруженные в память.

### Советы по устранению неполадок

Если ваши шрифты отображаются неправильно, убедитесь, что:
- Файлы шрифтов правильно расположены в указанных каталогах или по указанным путям.
- Данные байтового массива точно представляют содержимое файла шрифта.

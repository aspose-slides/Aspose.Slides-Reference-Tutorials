## Руководство по внедрению

### Определить формат файла PowerPoint

#### Обзор
Эта функция помогает определить, использует ли файл PowerPoint устаревший формат, например PPT95, что позволяет вам соответствующим образом обрабатывать его в вашем приложении.

#### Шаги:

**1. Импорт Aspose.Slides**
```csharp
using Aspose.Slides;
```

**2. Загрузить информацию о презентации**
```csharp
String dataDir = "YOUR_DOCUMENT_DIRECTORY/presentation.ppt"; // Обновите, указав путь к файлу

// Получить информацию о презентации для определения формата
PresentationInfo presentationInfo = PresentationFactory.Instance.getPresentationInfo(dataDir);
```

**3. Проверьте формат**
```csharp
bool isOldFormat = presentationInfo.getLoadFormat() == LoadFormat.Ppt95;

if (isOldFormat) {
    Console.WriteLine("The file is in an older PPT format.");
} else {
    Console.WriteLine("The file is not in the old PPT format.");
}
```

**Объяснение:** The `PresentationFactory` класс предоставляет информацию о презентации, включая ее формат. Проверка по `LoadFormat.Ppt95` сообщает нам, является ли это более старой версией.

#### Советы по устранению неполадок
- Убедитесь, что путь к файлу правильный и доступный.
- Обрабатывайте исключения, которые могут возникать из-за неподдерживаемых форматов, заключая код в блоки try-catch.

### Добавить новый слайд в презентацию

#### Обзор
Эта функция позволяет легко добавлять новый слайд в существующую презентацию PowerPoint, используя первый доступный макет.

#### Шаги:

**1. Импорт Aspose.Slides**
```csharp
using Aspose.Slides;
```

**2. Загрузить существующую презентацию**
```csharp
String dataDir = "YOUR_DOCUMENT_DIRECTORY/presentation.pptx"; // Обновите, указав путь к файлу

// Открыть существующую презентацию
Presentation pres = new Presentation(dataDir);
```

**3. Добавить новый слайд**
```csharp
ISlide slide = pres.getSlides().addEmptySlide(pres.getLayoutSlides().get_Item(0));

pres.save("YOUR_OUTPUT_DIRECTORY/ModifiedPresentation.pptx", SaveFormat.Pptx);

Console.WriteLine("New slide added successfully.");
```

**Объяснение:** The `Slides` Коллекция в течение `Presentation` объект позволяет добавлять новые слайды. Здесь мы используем первый макет слайда в качестве шаблона.

#### Советы по устранению неполадок
- Убедитесь, что выходной каталог существует и доступен для записи.
- Убедитесь, что входное представление не заблокировано и не повреждено.

- Реализация практических вариантов использования и оптимизация производительности

## Руководство по внедрению

Выполните следующие действия, чтобы преобразовать презентацию PowerPoint в защищенный паролем PDF-файл.

### Шаг 1: Определите каталог документов

Укажите путь к каталогу документов, в котором хранятся ваши файлы PPTX:
```csharp
string dataDir = "/path/to/your/documents";
```

### Шаг 2: Загрузите файл презентации

Создайте экземпляр `Presentation` класс для представления файла PowerPoint, который вы хотите преобразовать:
```csharp
using (Presentation presentation = new Presentation(dataDir + "/DemoFile.pptx"))
{
    // Дальнейшая обработка здесь
}
```
The `using` заявление гарантирует правильную утилизацию ресурсов после использования.

### Шаг 3: Настройте параметры PDF-файла

Создайте экземпляр `PdfOptions` класс для указания параметров вывода PDF-файла, включая установку пароля:
```csharp
PdfOptions pdfOptions = new PdfOptions();
pdfOptions.Password = "your_password"; // Установите желаемый пароль здесь
```
Этот шаг гарантирует, что доступ к PDF-файлу смогут получить только пользователи с правильным паролем.

### Шаг 4: Сохранить как PDF-файл, защищенный паролем

Сохраните презентацию в формате PDF, используя настроенные параметры:
```csharp
string outputPath = "/path/to/output/PasswordProtectedPDF_out.pdf";
presentation.Save(outputPath, SaveFormat.Pdf, pdfOptions);
```
The `Save` Метод записывает выходной файл в указанный вами каталог.

### Советы по устранению неполадок
- **Файл не найден**: Убедитесь, что пути к файлам указаны правильно и доступны.
- **Проблемы с разрешением**: Проверьте, есть ли у вас соответствующие права на запись в выходной каталог.

### Расчет направления

**Обзор:** Рассчитайте направление линии на основе ее размеров и свойств отражения.

#### Шаг 1: Определите параметры

Укажите ширину, высоту и логические значения, указывающие горизонтальное или вертикальное отражение:

```csharp
float width = /* ваша ценность */;
float height = /* ваша ценность */;
bool flipH = /* ваше логическое значение */;
bool flipV = /* ваше логическое значение */;
```

#### Шаг 2: Рассчитайте направление

Используйте функцию арктангенса, чтобы определить угол между прямой и осью Y, затем нормализуйте его:

```csharp
class LineDirectionCalculator
{
    public static double CalculateDirection(float width, float height, bool flipH, bool flipV)
    {
        float endLineX = width * (flipH ? -1 : 1);
        float endLineY = height * (flipV ? -1 : 1);

        float endYAxisX = 0;
        float endYAxisY = height;

        double angle = (Math.Atan2(endYAxisY, endYAxisX) - Math.Atan2(endLineY, endLineX));

        if (angle < 0) angle += 2 * Math.PI;

        return angle * 180.0 / Math.PI;
    }
}
```

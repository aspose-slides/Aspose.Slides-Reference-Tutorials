## Руководство по внедрению

Чтобы сохранить презентации без создания миниатюр, выполните следующие действия.

### Сохранить презентацию без создания новой миниатюры

#### Шаг 1: Подготовьте окружающую среду

Убедитесь, что Aspose.Slides правильно установлен и настроен. Проверьте, нет ли ошибок компиляции, связанных с отсутствующими ссылками.

#### Шаг 2: Загрузите презентацию

Загрузите презентацию, которую вы хотите изменить:
```csharp
string pptxFile = "YOUR_DOCUMENT_DIRECTORY\Image.pptx";
Presentation pres = new Presentation(pptxFile);
```
The `Presentation` класс позволяет получать доступ к файлам PowerPoint и изменять их.

#### Шаг 3: Измените содержимое слайда (необязательно)

Внесите необходимые изменения. Для демонстрации удалите все фигуры с первого слайда:
```csharp
pres.Slides[0].Shapes.Clear();
```
Этот шаг гарантирует, что перед сохранением будет сохранено только необходимое содержимое.

#### Шаг 4: Сохранение без создания миниатюр

Используйте `Save` метод с определенными параметрами для предотвращения создания миниатюр:
```csharp
string resultPath = "YOUR_OUTPUT_DIRECTORY\result_with_old_thumbnail.pptx";
pres.Save(resultPath, SaveFormat.Pptx, new PptxOptions() {
    RefreshThumbnail = false // Предотвращает повторную генерацию миниатюр
});
```
The `RefreshThumbnail` свойство установлено в `false` указывает Aspose.Slides не создавать повторно миниатюры во время процесса сохранения.

#### Советы по устранению неполадок
- Убедитесь, что пути к файлам верны и доступны.
- Убедитесь, что ваша среда поддерживает функции .NET, используемые Aspose.Slides.
- Проверьте файлы журналов на наличие ошибок, если сохранение неожиданно завершилось неудачей.

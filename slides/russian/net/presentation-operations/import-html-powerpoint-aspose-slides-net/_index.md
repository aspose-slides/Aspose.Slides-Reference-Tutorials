## Руководство по внедрению

Давайте разобьем процесс внедрения на этапы.

### Функция: импорт HTML-текста в презентацию
Эта функция позволяет импортировать HTML-контент непосредственно в слайды PowerPoint.

#### Шаг 1: Настройка каталога документов
Определите, где находится ваш HTML-файл:
```csharp
string dataDir = @"YOUR_DOCUMENT_DIRECTORY";
```

#### Шаг 2: Создание новой презентации
Инициализируйте новый экземпляр презентации и получите доступ к его первому слайду:
```csharp
using (Presentation pres = new Presentation()) {
    ISlide slide = pres.Slides[0];
```

#### Шаг 3: Добавление автофигуры для HTML-контента
Добавьте AutoShape для размещения вашего HTML-контента. Настройте его так, чтобы он не имел фоновой заливки:
```csharp
IAutoShape ashape = slide.Shapes.AddAutoShape(ShapeType.Rectangle, 10, 10, pres.SlideSize.Size.Width - 20, pres.SlideSize.Size.Height - 10);
ashape.FillFormat.FillType = FillType.NoFill;
```

#### Шаг 4: Настройка текстовой рамки
Подготовьте текстовый фрейм для размещения вашего HTML-контента:
```csharp
ashape.AddTextFrame("");
ashape.TextFrame.Paragraphs.Clear();
```

#### Шаг 5: Импорт HTML-контента
Прочитайте и импортируйте содержимое HTML-файла в текстовый фрейм:
```csharp
using (TextReader tr = new StreamReader(dataDir + "file.html")) {
    ashape.TextFrame.Paragraphs.AddFromHtml(tr.ReadToEnd());
}
```

#### Шаг 6: Сохранение презентации
Сохраните презентацию в указанном каталоге:
```csharp
pres.Save(dataDir + "YOUR_OUTPUT_DIRECTORY\\output_out.pptx");
```

### Советы по устранению неполадок
- Убедитесь, что путь к HTML-файлу указан правильно.
- Убедитесь, что Aspose.Slides правильно лицензирован и инициализирован.

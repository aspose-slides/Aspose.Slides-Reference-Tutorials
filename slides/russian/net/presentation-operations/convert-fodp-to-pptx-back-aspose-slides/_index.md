## Руководство по внедрению

### Загрузка и сохранение файла FODP как PPTX

#### Обзор
Загрузите существующий файл FODP в свое приложение и сохраните его как файл PPTX, который идеально подходит для обмена презентациями в широко поддерживаемом формате PowerPoint.

#### Шаги
**Шаг 1: Загрузите файл FODP**
Создать `Presentation` объект, загрузив ваш файл FODP:
```csharp
using System.IO;
using Aspose.Slides;

string fodpFilePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "Example.fodp");

// Загрузите файл FODP в объект Presentation.
using (Presentation presentation = new Presentation(fodpFilePath))
{
    // Объект «Презентация» теперь содержит содержимое FODP.
}
```
**Шаг 2: Сохранить как PPTX**
Сохраните загруженную презентацию в формате PPTX:
```csharp
string pptxOutputPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "FodpToPptxConversion.pptx");

// Сохраните загруженную презентацию как файл PPTX.
presentation.Save(pptxOutputPath, SaveFormat.Pptx);
```
### Преобразование PPTX обратно в формат FODP

#### Обзор
Преобразование файла PPTX обратно в формат FODP сохраняет определенные функции или метаданные, уникальные для формата FODP.

#### Шаги
**Шаг 1: Загрузите файл PPTX**
Загрузите ваш файл PPTX в `Presentation` объект:
```csharp
string pptxFilePath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "FodpToPptxConversion.pptx");

// Загрузите файл PPTX в объект Presentation.
using (Presentation pres = new Presentation(pptxFilePath))
{
    // Объект Presentation теперь содержит ваш контент PPTX.
}
```
**Шаг 2: Сохранить как FODP**
Сохраните презентацию обратно в формате FODP:
```csharp
string fodpOutputPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "PptxToFodpConversion.fodp");

// Сохраните загруженную презентацию как файл FODP.
pres.Save(fodpOutputPath, SaveFormat.Fodp);
```

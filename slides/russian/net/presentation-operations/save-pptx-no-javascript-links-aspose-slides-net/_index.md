## Руководство по внедрению

### Сохранение презентаций без ссылок JavaScript

Эта функция фокусируется на сохранении файлов PPTX, исключая гиперссылки, содержащие JavaScript, что повышает безопасность.

#### Шаг 1: Настройте пути к файлам

Определите исходный и выходной пути для вашего файла PowerPoint.
```csharp
string pptxFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "JavaScriptLink.pptx");
string resultPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "JavaScriptLink-out.pptx");
```

#### Шаг 2: Загрузите презентацию

Загрузите презентацию с помощью Aspose.Slides, подготовив ее к обработке.
```csharp
using (Presentation pres = new Presentation(pptxFile))
{
    // Дальнейшая обработка будет происходить здесь
}
```

#### Шаг 3: Сохранение с пропущенными ссылками JavaScript

Сохраните презентацию, убедившись, что все ссылки JavaScript пропущены, чтобы повысить безопасность путем предотвращения потенциального выполнения скрипта.
```csharp
pres.Save(resultPath, SaveFormat.Pptx, new PptxOptions()
{
    SkipJavaScriptLinks = true
});
```
- **Объясняемые параметры:** 
  - `resultPath`: Путь назначения для сохраненного файла.
  - `SaveFormat.Pptx`: Указывает сохранение в формате PowerPoint.
  - `PptxOptions.SkipJavaScriptLinks`Критическая конфигурация для повышения безопасности.

**Советы по устранению неполадок:**
- Убедитесь, что пути заданы правильно и каталоги существуют.
- Убедитесь, что у вас есть права на запись в выходной каталог.

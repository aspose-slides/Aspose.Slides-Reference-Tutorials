## Руководство по внедрению
Теперь мы углубимся в реализацию каждой функции. Каждый раздел будет содержать краткий обзор, за которым последуют подробные шаги.

### Сохранение презентации как документа, совместимого с PDF/A-1a
**Обзор:**
Формат PDF/A-1a идеально подходит для долгосрочного архивирования, гарантируя неизменность и доступность документа с течением времени.

#### Пошаговая реализация:
##### 1. Определите пути к файлам
```csharp
string pptxFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "tagged-pdf-demo.pptx");
string outPdf1aFile = Path.Combine("YOUR_OUTPUT_DIRECTORY", "tagged-pdf-demo_1a.pdf");
```
##### 2. Загрузите презентацию
```csharp
using (Presentation presentation = new Presentation(pptxFile))
{
    // Продолжайте сохранять документ
}
```
##### 3. Сохранить как PDF/A-1a
```csharp
presentation.Save(outPdf1aFile, SaveFormat.Pdf,
    new PdfOptions { Compliance = PdfCompliance.PdfA1a });
```
**Объяснение:** The `PdfOptions` Объект указывает, что вывод должен соответствовать стандартам PDF/A-1a.

### Сохранение презентации как документа, совместимого с PDF/A-1b
**Обзор:**
Формат PDF/A-1b обеспечивает доступность за счет сохранения тегов, что облегчает работу программ чтения с экрана и других вспомогательных технологий.

#### Пошаговая реализация:
##### 1. Определите пути к файлам

##### 3. Сохранить как PDF/A-1b
```csharp

    new PdfOptions { Compliance = PdfCompliance.PdfA1b });
```
**Объяснение:** Это гарантирует доступность вашего документа за счет включения необходимых метаданных и тегов.

### Сохранение презентации как документа, совместимого с PDF/UA
**Обзор:**
PDF/UA ориентирован на всеобщую доступность, гарантируя, что документы могут использоваться всеми, независимо от каких-либо ограничений.

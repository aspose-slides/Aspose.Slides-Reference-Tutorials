## Руководство по внедрению

В этом разделе мы рассмотрим создание презентации и добавление кластеризованной столбчатой диаграммы.

### Создание пустой презентации

Начните с настройки пути к каталогу документов. Это место, где будет сохранена сгенерированная презентация:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
Presentation pres = new Presentation();
```

### Добавление кластеризованной столбчатой диаграммы на слайд

Затем добавьте кластеризованную столбчатую диаграмму на первый слайд в указанном месте и размере:
```csharp
// Добавьте кластеризованную столбчатую диаграмму в (20, 20) с размерами (500x400)
IChart chart = pres.Slides[0].Shapes.AddChart(
    ChartType.ClusteredColumn,
    20, 20, 500, 400);
```
**Объяснение:** Этот фрагмент создает пустую презентацию и добавляет кластеризованную столбчатую диаграмму. `AddChart` метод определяет тип диаграммы (`ClusteredColumn`) и его положение/размеры (x: 20, y: 20, ширина: 500, высота: 400).

### Сохранение презентации

Наконец, сохраните презентацию, чтобы гарантировать сохранение всех изменений:
```csharp
// Сохраните презентацию в указанном каталоге.
pres.Save(dataDir + "CreateAndAddChart_out.pptx");
```
**Объяснение:** The `Save` Метод записывает данные презентации в файл. Отрегулируйте путь в соответствии с вашей средой.

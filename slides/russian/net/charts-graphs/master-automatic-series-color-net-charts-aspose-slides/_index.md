#### Шаг 2: Добавьте кластеризованную столбчатую диаграмму
Добавьте кластеризованную столбчатую диаграмму в позицию (100, 50) с размерами (600x400):
```csharp
// Добавить кластеризованную столбчатую диаграмму\IChart chart = presentation.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 100, 50, 600, 400);
```

Повторите каждую серию, чтобы включить автоматическую заливку цветом:
```csharp
// Проведите цикл по каждой серии для автоматической настройки цвета
type IChartSeries series;
for (int i = 0; i < chart.ChartData.Series.Count; i++) {
    series = chart.ChartData.Series[i];
    // Автоматически устанавливать цвет серии
    series.Format.Fill.FillType = FillType.Solid;
    series.Format.Fill.SolidFillColor.Color = Color.FromArgb(255, GetRandomColor());
}
```

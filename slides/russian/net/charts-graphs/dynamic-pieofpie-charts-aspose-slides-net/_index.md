## Руководство по внедрению

### Добавление круговой диаграммы в презентацию

#### Обзор

В этом разделе показано, как создать и добавить диаграмму PieOfPie на слайд PowerPoint с помощью Aspose.Slides.

#### Пошаговые инструкции

**1. Инициализируйте презентацию**

Создайте экземпляр `Presentation` сорт:

```csharp
using Aspose.Slides;

Presentation presentation = new Presentation();
```

**2. Добавьте круговую диаграмму**

Вставьте диаграмму в нужное место и с нужными размерами на первом слайде:

```csharp
using Aspose.Slides.Charts;

IChart chart = presentation.Slides[0].Shapes.AddChart(ChartType.PieOfPie, 50, 50, 500, 400);
```

**3. Сохраните презентацию**

После добавления диаграммы сохраните файл в формате PPTX:

```csharp
using Aspose.Slides.Export;

presentation.Save("YOUR_OUTPUT_DIRECTORY/SecondPlotOptionsforCharts_out.pptx", SaveFormat.Pptx);
```

### Настройка меток данных диаграммы и свойств группы серий

#### Обзор

Улучшите свою диаграмму, настроив метки данных и свойства групп рядов для лучшей визуализации.

**1. Установите формат метки данных**

Отображение значений по первой серии:

```csharp
chart.ChartData.Series[0].Labels.DefaultDataLabelFormat.ShowValue = true;
```

**2. Отрегулируйте размер второго круга**

Установите подходящий размер для ясности:

```csharp
chart.ChartData.Series[0].ParentSeriesGroup.SecondPieSize = 149;
```

**3. Настройте разделение по проценту и положению**

Тонкая настройка разделения данных на диаграмме:

```csharp
chart.ChartData.Series[0].ParentSeriesGroup.PieSplitBy = PieSplitType.ByPercentage;
chart.ChartData.Series[0].ParentSeriesGroup.PieSplitPosition = 53;
```

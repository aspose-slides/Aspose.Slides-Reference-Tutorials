## Руководство по внедрению
Теперь, когда вы все настроили, давайте рассмотрим каждую функцию шаг за шагом.
### Создание и добавление диаграммы в презентацию
#### Обзор
Начнем с создания презентации и добавления круговой диаграммы на первый слайд.
#### Шаги:
1. **Инициализировать презентацию**
   Начните с создания экземпляра `Presentation` класс, представляющий ваш файл PowerPoint.
   ```csharp
   using Aspose.Slides;
   
   string dataDir = "YOUR_DOCUMENT_DIRECTORY";
   
   using (Presentation pres = new Presentation())
   {
       // Здесь мы добавим нашу диаграмму.
   }
   ```
2. **Добавить круговую диаграмму**
   Используйте `Shapes.AddChart` метод вставки круговой диаграммы в определенные координаты на слайде.
   ```csharp
   IChart chart = pres.Slides[0].Shapes.AddChart(
       ChartType.Pie, 50, 50, 400, 600, true);
   ```
### Установить внешнюю рабочую книгу для данных диаграммы
#### Обзор
Теперь давайте настроим круговую диаграмму для использования данных из внешней книги.
#### Шаги:
1. **Доступ к данным диаграммы**
   Получите интерфейс данных диаграммы, в котором вы укажете URL-адрес внешнего источника данных.
   ```csharp
   IChartData chartData = chart.ChartData;
   ```
2. **Установить внешний URL-адрес рабочей книги**
   Установите URL-адрес для вашего источника данных, используя `SetExternalWorkbook`. В этом примере используется URL-адрес-заполнитель, который следует заменить на фактический путь к источнику данных.
   ```csharp
   (chartData as ChartData).SetExternalWorkbook("http://путь/не/существует", false);
   ```
### Сохранить презентацию в файл
#### Обзор
Наконец, сохраните презентацию в формате PPTX в желаемом месте.
#### Шаги:
1. **Сохранить презентацию**
   Используйте `Save` Метод `Presentation` класс для записи файла на диск.
   ```csharp
   pres.Save(dataDir + "SetExternalWorkbookWithUpdateChartData.pptx", SaveFormat.Pptx);
   ```

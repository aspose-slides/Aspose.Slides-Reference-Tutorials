## Руководство по внедрению

### Обзор функций: настройка угла поворота заголовка оси диаграммы
Регулировка угла поворота может улучшить читаемость и эстетику, особенно в слайдах с ограниченным пространством. Вот как реализовать эту функцию:
#### Шаг 1: Создайте презентацию и добавьте диаграмму
Начните с создания новой презентации и добавления кластеризованной столбчатой диаграммы.
```csharp
using Aspose.Slides;

string dataDir = "YOUR_DOCUMENT_DIRECTORY";

// Инициализируйте новый экземпляр Presentation.
using (Presentation pres = new Presentation())
{
    // Добавьте кластеризованную столбчатую диаграмму на первый слайд в позицию (50, 50) шириной 450 и высотой 300.
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 50, 50, 450, 300);
```
#### Шаг 2: Включите заголовок вертикальной оси
Включите заголовок вертикальной оси, чтобы настроить ее внешний вид.
```csharp
    // Включите заголовок вертикальной оси для диаграммы.
    chart.Axes.VerticalAxis.HasTitle = true;
```
#### Шаг 3: Установите угол поворота
Установите угол поворота формата текстового блока для заголовка вертикальной оси.
```csharp
    // Установите угол поворота на 90 градусов.
    chart.Axes.VerticalAxis.Title.TextFormat.TextBlockFormat.RotationAngle = 90;

    // Сохраните презентацию с измененной диаграммой в файле .pptx в указанном каталоге.
    pres.Save(dataDir + "test.pptx", SaveFormat.Pptx);
}
```
### Основные параметры конфигурации
- **Угол поворота**: Настройте от -180 до 180 градусов в зависимости от потребностей вашего дизайна.
- **Формат названия оси**: Измените размер, стиль и цвет шрифта для лучшей видимости.

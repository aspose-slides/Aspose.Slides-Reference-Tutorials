## Руководство по внедрению

Давайте разобьем процесс создания кольцевой диаграммы с помощью Aspose.Slides для .NET на удобные для выполнения шаги.

### Создать кольцевую диаграмму

#### Обзор
Начнем с добавления кольцевой диаграммы на слайд PowerPoint, задав ее положение и размер.

**Добавление диаграммы:**
```csharp
using Aspose.Slides.Charts;

// Доступ к первому слайду презентации (по умолчанию он создан)
ISlide slide = presentation.Slides[0];

// Добавьте кольцевую диаграмму на слайд в позицию (50, 50) с шириной и высотой 400 единиц.
IChart chart = slide.Shapes.AddChart(ChartType.Doughnut, 50, 50, 400, 400);
```
- **Параметры:** `ChartType.Doughnut`, x-позиция: 50, y-позиция: 50, ширина: 400, высота: 400.

### Установите размер отверстия

#### Обзор
Далее мы настроим размер отверстий кольцевой диаграммы, чтобы сделать ее визуально привлекательной.

**Настройка размера отверстия:**
```csharp
// Установите размер отверстия для кольцевой диаграммы на 90%.
chart.ChartData.SeriesGroups[0].DoughnutHoleSize = 90;
```
- **Конфигурация ключа:** `DoughnutHoleSize` определяет, какая часть центра «вырезана». Значение от 0 до 100 представляет собой процент.

### Сохраните вашу презентацию

Наконец, сохраните изменения в новом файле PowerPoint:
```csharp
// Определите путь, по которому будет сохранена презентация.
string outputPath = \@"YOUR_OUTPUT_DIRECTORY\DoughnutHoleSize_out.pptx";

// Сохраните измененную презентацию в формате PPTX.
presentation.Save(outputPath, SaveFormat.Pptx);
```
- **Примечание:** Заменять `YOUR_OUTPUT_DIRECTORY` с желаемым расположением файла.

### Советы по устранению неполадок

- Убедитесь, что Aspose.Slides правильно установлен и импортирован.
- Перед сохранением презентации убедитесь, что путь к выходному каталогу существует.

## Руководство по внедрению

### Создание диаграммы площади
Начнем с добавления диаграммы с областями на слайд PowerPoint.

#### Добавление диаграммы
1. **Инициализировать презентацию:**
   Начните с создания нового экземпляра `Presentation`.
   ```csharp
   Presentation pres = new Presentation();
   ```
2. **Добавить диаграмму на слайд:**
   Добавьте карту областей с указанными координатами (100, 100) и размерами 500x350.
   ```csharp
   // Добавьте диаграмму областей на первый слайд.
   Chart chart = (Chart)pres.Slides[0].Shapes.AddChart(ChartType.Area, 100, 100, 500, 350);
   ```

#### Проверка макета
После создания проверьте макет диаграммы, используя:
```csharp
// Проверьте макет созданной диаграммы.
chart.ValidateChartLayout();
```
Этот шаг гарантирует, что все компоненты правильно выровнены и отображены.

### Получение значений осей и шкалы единиц
Понимание значений осей имеет решающее значение для представления данных. Вот как их можно получить:
1. **Получить значения вертикальной оси:**
   Извлеките максимальные и минимальные значения из вертикальной оси.
   ```csharp
double maxValue = диаграмма.Оси.ВертикальнаяОсь.ФактическоеМаксимальноеЗначение;
double minValue = chart.Axes.VerticalAxis.ActualMinValue;
```
2. **Get Horizontal Axis Scales:**
   Obtain major and minor unit scales for horizontal axis adjustment.
   ```csharp
double majorUnit = chart.Axes.HorizontalAxis.ActualMajorUnit;
double minorUnit = chart.Axes.HorizontalAxis.ActualMinorUnit;
```

### Сохранение презентации
Наконец, сохраните презентацию, чтобы убедиться, что все изменения сохранены:
```csharp
// Сохраните презентацию с изменениями.
pres.Save(dataDir + "ErrorBars_out.pptx", SaveFormat.Pptx);
```

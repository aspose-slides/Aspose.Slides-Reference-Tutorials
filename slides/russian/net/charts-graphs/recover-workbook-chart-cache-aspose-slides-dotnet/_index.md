### Восстановление данных рабочей книги из кэша диаграмм
Эта функция позволяет восстанавливать данные для диаграмм, связанных с внешними рабочими книгами, даже если исходный файл недоступен. Вот как это работает:

#### Шаг 1: Определите пути к файлам
Настройте пути к входным и выходным файлам с помощью заполнителей, чтобы обеспечить гибкость.

```csharp
string pptxFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "ExternalWB.pptx");
string outPptxFile = Path.Combine("YOUR_OUTPUT_DIRECTORY", "ExternalWB_out.pptx");
```

#### Шаг 2: Настройте параметры загрузки
Настройте параметры загрузки, чтобы включить восстановление рабочей книги из кэшей диаграмм.

```csharp
LoadOptions lo = new LoadOptions();
lo.SpreadsheetOptions.RecoverWorkbookFromChartCache = true;
```

#### Шаг 3: Откройте и обработайте презентацию
Используйте Aspose.Slides, чтобы открыть презентацию с указанными параметрами загрузки, получить доступ к данным диаграммы и восстановить информацию рабочей книги.

```csharp
using (Presentation pres = new Presentation(pptxFile, lo))
{
    IChart chart = pres.Slides[0].Shapes[0] as IChart;
    IChartDataWorkbook wb = chart.ChartData.ChartDataWorkbook;

    // Сохранить изменения в новом файле
    pres.Save(outPptxFile, SaveFormat.Pptx);
}
```

#### Основные параметры конфигурации
- **ВосстановитьРабочуюКнигуИзКэшаДиаграммы**: Этот параметр имеет решающее значение для восстановления данных рабочей книги из диаграмм с отсутствующими внешними ссылками.

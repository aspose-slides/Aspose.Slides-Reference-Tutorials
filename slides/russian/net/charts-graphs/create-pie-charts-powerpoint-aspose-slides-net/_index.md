## Руководство по внедрению
В этом разделе мы рассмотрим две функции: создание круговой диаграммы и доступ к рабочим листам данных диаграммы.

### Функция 1: Создание круговой диаграммы

#### Обзор
Добавление круговой диаграммы на слайд PowerPoint можно легко осуществить с помощью Aspose.Slides. Эта функция позволяет указать положение и размер диаграммы на слайде.

#### Этапы внедрения
**Шаг 1: Добавьте круговую диаграмму**
```csharp
using (Presentation pres = new Presentation())
{
    // Добавьте круговую диаграмму с указанными координатами ширины и высоты.
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.Pie, 50, 50, 400, 500);
}
```

**Шаг 2: Доступ к рабочей книге данных диаграммы**
```csharp
IChartDataWorkbook workbook = chart.ChartData.ChartDataWorkbook;
```

**Шаг 3: Просмотрите рабочие листы и выведите имена**
На этом этапе извлекаются имена каждого рабочего листа в рабочей книге данных диаграммы.
```csharp
for (int i = 0; i < workbook.Worksheets.Count; i++)
{
    Console.WriteLine(workbook.Worksheets[i].Name);
}
```

#### Основные параметры конфигурации
- **Позиционирование**: Регулировать `X` и `Y` параметры для точного размещения диаграммы.
- **Размер**: Изменить `width` и `height` для желаемых вами размеров.

### Функция 2: Доступ к коллекции рабочих листов данных диаграммы
Эта функция фокусируется на итерации по рабочим листам в рабочей книге данных диаграммы, что имеет решающее значение при работе со сложными наборами данных.

#### Обзор
Доступ к коллекциям рабочих листов позволяет эффективно управлять данными и манипулировать ими перед их отображением в диаграммах.

#### Этапы внедрения
Действия здесь аналогичны действиям в предыдущем разделе, поскольку обе функции используют схожие процессы для доступа к данным диаграммы:
**Шаг 1–3: Повторное использование кода из создания круговой диаграммы**
```csharp
using (Presentation pres = new Presentation())
{
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.Pie, 50, 50, 400, 500);
    IChartDataWorkbook workbook = chart.ChartData.ChartDataWorkbook;

    for (int i = 0; i < workbook.Worksheets.Count; i++)
    {
        Console.WriteLine(workbook.Worksheets[i].Name);
    }
}
```

#### Советы по устранению неполадок
- **Отсутствующие данные диаграммы**: Перед тем как открыть рабочий лист с данными диаграммы, убедитесь, что он не пуст.
- **Обработка исключений**: Оберните блоки кода в операторы try-catch для корректной обработки исключений.

## Руководство по внедрению

### Изменение цвета ряда на диаграмме

В этом разделе вы узнаете, как изменить цвет точки данных в серии диаграмм.

#### Шаг 1: Загрузите существующую презентацию

Загрузите файл PowerPoint, содержащий диаграмму:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
using (Presentation pres = new Presentation(dataDir + "/test.pptx"))
{
    // Продолжайте доступ к диаграмме и ее изменение.
}
```

#### Шаг 2: Доступ к диаграмме

Доступ к диаграмме на слайде. Здесь мы добавляем круговую диаграмму в качестве примера:

```csharp
IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.Pie, 50, 50, 600, 400);
```

#### Шаг 3: Измените цвет точки данных

Выберите точку данных, которую вы хотите изменить, и установите ее цвет. Мы нацелимся на вторую точку данных первой серии:

```csharp
IChartDataPoint point = chart.ChartData.Series[0].DataPoints[1];

// Примените взрыв для лучшего визуального разделения
point.Explosion = 30;

// Изменить тип и цвет заливки на синий
point.Format.Fill.FillType = FillType.Solid;
point.Format.Fill.SolidFillColor.Color = Color.Blue;
```

#### Шаг 4: Сохраните измененную презентацию.

Сохраните презентацию с обновленной диаграммой:

```csharp
pres.Save(dataDir + "/output.pptx");
```

### Советы по устранению неполадок

- **Проблема:** Точка данных не меняет цвет.
  - **Решение:** Убедитесь, что вы правильно получили доступ к точке данных и применили изменения `FillType` и `Color`.

#### Настройка свойств шрифта для диаграмм
Эта функция позволяет вам улучшить визуальную привлекательность ваших диаграмм путем настройки свойств шрифта. Вот как это реализовать:

**Шаг 1: Определите пути к каталогам**
Начните с указания того, где будут располагаться ваши входные и выходные файлы:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputPath = Path.Combine(dataDir, "FontPropertiesForChart.pptx");
```

**Шаг 2: Создание нового экземпляра презентации**
Инициализируйте новый объект презентации для размещения вашей диаграммы:
```csharp
using (Presentation pres = new Presentation()) {
    // Дальнейшие шаги будут реализованы здесь.
}
```

**Шаг 3: Добавьте кластеризованную столбчатую диаграмму**
Вставьте диаграмму на первый слайд в указанные координаты и размеры:
```csharp
IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 100, 100, 500, 400);
```

**Шаг 4: Установите высоту шрифта для текста в диаграмме**
Настройте размер шрифта для улучшения читабельности:
```csharp
chart.TextFormat.PortionFormat.FontHeight = 20;
```

**Шаг 5: Включите отображение значений на метках данных**
Убедитесь, что значения данных видны, добавив контекст к вашей диаграмме:
```csharp
chart.ChartData.Series[0].Labels.DefaultDataLabelFormat.ShowValue = true;
```

**Шаг 6: Сохраните презентацию**
Сохраните презентацию со всеми примененными настройками:
```csharp
pres.Save(outputPath, SaveFormat.Pptx);
```

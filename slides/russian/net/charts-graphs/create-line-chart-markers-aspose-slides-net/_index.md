# Как создать линейную диаграмму с маркерами в C# с помощью Aspose.Slides для .NET

## Введение
Создание визуально привлекательных и информативных линейных диаграмм имеет важное значение для эффективного представления данных в C#. **Aspose.Slides для .NET** упрощает процесс добавления профессионально выглядящих диаграмм, включая диаграммы с маркерами. Этот урок проведет вас через создание линейной диаграммы с маркерами по умолчанию с помощью Aspose.Slides для .NET.

В этом уроке вы узнаете:
- Настройка среды для использования Aspose.Slides для .NET.
- Создание и настройка презентации с линейной диаграммой, включающей маркеры.
- Настройка свойств диаграммы, таких как категории, серии и точки данных.
- Сохранение итогового файла презентации.

Давайте начнем с обзора предварительных условий, необходимых перед внедрением нашего решения.

## Предпосылки
Прежде чем начать, убедитесь, что у вас есть следующее:
- **Требуемые библиотеки:** Aspose.Slides для .NET устанавливается в вашей среде разработки через NuGet.
- **Требования к настройке среды:** Установленная на вашем компьютере рабочая среда разработки C#, например Visual Studio и .NET Framework.
- **Необходимые знания:** Базовые знания программирования на C# и навыки создания презентаций программным способом.

## Руководство по внедрению
### Создание линейной диаграммы с маркерами
В этом разделе вы изучите каждый шаг, необходимый для создания и настройки линейной диаграммы с маркерами по умолчанию в вашей презентации с использованием Aspose.Slides для .NET.

#### Шаг 1: Создание объекта презентации
Начните с создания экземпляра `Presentation` сорт:
```csharp
using (Presentation pres = new Presentation())
{
    ISlide slide = pres.Slides[0];
```
Здесь мы получаем доступ к первому слайду только что созданной презентации.

#### Шаг 2: Добавьте линейную диаграмму с маркерами
Затем добавьте на слайд линейную диаграмму с маркерами:
```csharp
IChart chart = slide.Shapes.AddChart(ChartType.LineWithMarkers, 10, 10, 400, 400);
```
Этот код добавляет новую диаграмму типа `LineWithMarkers` в координатах `(10, 10)` с размерами `400x400`.

#### Шаг 3: Очистите существующие серии и категории
Перед добавлением данных очистите все существующие серии или категории:
```csharp
chart.ChartData.Series.Clear();
chart.ChartData.Categories.Clear();
```
Это гарантирует, что наша диаграмма начнется с чистого листа.

#### Шаг 4: Настройка рабочей книги данных диаграммы
Доступ к `ChartDataWorkbook` для управления данными вашей диаграммы:
```csharp
IChartDataWorkbook fact = chart.ChartData.ChartDataWorkbook;
```
Этот объект имеет решающее значение для управления ячейками, содержащими данные серий и категорий.

#### Шаг 5: Добавьте серии и категории
Добавьте новую серию в диаграмму и заполните ее точками данных:
```csharp
chart.ChartData.Series.Add(fact.GetCell(0, 0, 1, "Series 1"), chart.Type);
IChartSeries series = chart.ChartData.Series[0];

// Определите категории и соответствующие точки данных
chart.ChartData.Categories.Add(fact.GetCell(0, 1, 0, "C1"));
series.DataPoints.AddDataPointForLineSeries(fact.GetCell(0, 1, 1, 24));
chart.ChartData.Categories.Add(fact.GetCell(0, 2, 0, "C2"));
series.DataPoints.AddDataPointForLineSeries(fact.GetCell(0, 2, 1, 23));
chart.ChartData.Categories.Add(fact.GetCell(0, 3, 0, "C3"));
series.DataPoints.AddDataPointForLineSeries(fact.GetCell(0, 3, 1, -10));
chart.ChartData.Categories.Add(fact.GetCell(0, 4, 0, "C4"));

// Добавьте нулевую точку данных, чтобы продемонстрировать обработку пропущенных значений.
series.DataPoints.AddDataPointForLineSeries(fact.GetCell(0, 4, 1, (double?)null));
```
Здесь мы заполняем диаграмму категориями и соответствующими данными серий. Обратите внимание, как `null` значение рассматривается как демонстрация.

#### Шаг 6: Добавьте еще одну серию
Повторите процесс, чтобы добавить еще одну серию:
```csharp
chart.ChartData.Series.Add(fact.GetCell(0, 0, 2, "Series 2"), chart.Type);
IChartSeries series2 = chart.ChartData.Series[1];

series2.DataPoints.AddDataPointForLineSeries(fact.GetCell(0, 1, 2, 30));
series2.DataPoints.AddDataPointForLineSeries(fact.GetCell(0, 2, 2, 10));
series2.DataPoints.AddDataPointForLineSeries(fact.GetCell(0, 3, 2, 60));
series2.DataPoints.AddDataPointForLineSeries(fact.GetCell(0, 4, 2, 40));
```

#### Шаг 7: Включение и настройка легенды
Включите легенду диаграммы для улучшения читабельности:
```csharp
chart.HasLegend = true;
chart.Legend.Overlay = false;
```
Это гарантирует, что легенда будет видна и не будет наложена на график.

#### Шаг 8: Сохраните презентацию
Наконец, сохраните презентацию с только что добавленной диаграммой:
```csharp
pres.Save("DefaultMarkersInChart.pptx");
}
```
### Советы по устранению неполадок
- **Ошибки привязки данных:** Убедитесь, что точки данных правильно соответствуют категориям.
- **Диаграмма не отображается:** Убедитесь, что `chart.HasLegend` и другие свойства установлены соответствующим образом.

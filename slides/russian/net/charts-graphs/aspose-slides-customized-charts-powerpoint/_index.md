# Настраиваемые диаграммы PowerPoint в .NET с использованием Aspose.Slides

- Эффективное управление сериями и размерами данных диаграмм
- Сохранение улучшенной презентации

Давайте рассмотрим, как можно улучшить диаграммы PowerPoint с помощью всего нескольких строк кода.

Для начала вам нужно установить Aspose.Slides. В зависимости от вашей среды разработки выберите один из следующих методов:

```bash
dotnet add package Aspose.Slides
```

**Через консоль диспетчера пакетов:**
```powershell
Install-Package Aspose.Slides
```

**Через пользовательский интерфейс диспетчера пакетов NuGet:**
- Найдите «Aspose.Slides» и установите последнюю версию.

## Руководство по внедрению

### Создать и настроить презентацию

#### Обзор
Начните с создания экземпляра презентации, который послужит основой для добавления диаграмм.

```csharp
using Aspose.Slides;

// Инициализировать новую презентацию
Presentation presentation = new Presentation();
```

Этот фрагмент создает пустой файл PowerPoint, готовый к заполнению визуальными материалами с большим объемом данных.

### Добавить диаграмму на слайд

#### Обзор
Добавьте линейную диаграмму с маркерами на первый слайд презентации.

```csharp
using Aspose.Slides.Charts;

// Доступ к первому слайду
ISlide slide = presentation.Slides[0];

// Добавить линейную диаграмму с маркерами
IChart chart = slide.Shapes.AddChart(ChartType.LineWithMarkers, 0, 0, 400, 400);
```

Этот фрагмент кода добавляет на слайд новую диаграмму, закладывая основу для визуализации данных.

### Настроить данные диаграммы

#### Обзор
Настройте данные для своей диаграммы, очистив существующие ряды и добавив новые.

```csharp
using Aspose.Slides.Charts;

// Получить рабочую книгу, используемую данными диаграммы
int defaultWorksheetIndex = 0;
IChartDataWorkbook fact = chart.ChartData.ChartDataWorkbook;

// Очистить все существующие серии
chart.ChartData.Series.Clear();

// Добавить новую серию в диаграмму
chart.ChartData.Series.Add(fact.GetCell(defaultWorksheetIndex, 1, 1, "Series 1"), chart.Type);
```

Эта конфигурация позволяет вам настраивать точки данных и названия серий.

### Добавить изображения в качестве маркеров

#### Обзор
Замените маркеры по умолчанию изображениями, чтобы создать визуально привлекательное представление точек данных.

```csharp
using Aspose.Slides;
using System.Drawing;

// Загрузить изображения из файлов
IImage image1 = Images.FromFile("YOUR_DOCUMENT_DIRECTORY/aspose-logo.jpg");
IPPImage imgx1 = presentation.Images.AddImage(image1);
IImage image2 = Images.FromFile("YOUR_DOCUMENT_DIRECTORY/Tulips.jpg");
IPPImage imgx2 = presentation.Images.AddImage(image2);

// Доступ к первой серии в диаграмме
IChartSeries series = chart.ChartData.Series[0];

// Добавьте точки данных с изображениями в качестве маркеров
IChartDataPoint point1 = series.DataPoints.AddDataPointForLineSeries(fact.GetCell(defaultWorksheetIndex, 1, 1, (double)4.5));
point1.Marker.Format.Fill.FillType = FillType.Picture;
point1.Marker.Format.Fill.PictureFillFormat.Picture.Image = imgx1;

IChartDataPoint point2 = series.DataPoints.AddDataPointForLineSeries(fact.GetCell(defaultWorksheetIndex, 2, 1, (double)2.5));
point2.Marker.Format.Fill.FillType = FillType.Picture;
point2.Marker.Format.Fill.PictureFillFormat.Picture.Image = imgx2;

IChartDataPoint point3 = series.DataPoints.AddDataPointForLineSeries(fact.GetCell(defaultWorksheetIndex, 3, 1, (double)3.5));
point3.Marker.Format.Fill.FillType = FillType.Picture;
point3.Marker.Format.Fill.PictureFillFormat.Picture.Image = imgx1;

IChartDataPoint point4 = series.DataPoints.AddDataPointForLineSeries(fact.GetCell(defaultWorksheetIndex, 4, 1, (double)4.5));
point4.Marker.Format.Fill.FillType = FillType.Picture;
point4.Marker.Format.Fill.PictureFillFormat.Picture.Image = imgx2;
```

В этом фрагменте показано, как визуально настраивать точки данных с помощью изображений.

### Настроить размер маркера серии

#### Обзор
Отрегулируйте размер маркера для лучшей видимости и воздействия.

```csharp
using Aspose.Slides.Charts;

// Установить размер маркера
series.Marker.Size = 15;
```

Эта настройка гарантирует, что ваши маркеры будут отчетливо видны и их будет легко обнаружить на графике.

### Сохранить презентацию

#### Обзор
Сохраните изменения в новом файле PowerPoint.

```csharp
using Aspose.Slides.Export;

// Сохраните презентацию со всеми изменениями
presentation.Save("YOUR_OUTPUT_DIRECTORY/MarkOptions_out.pptx", SaveFormat.Pptx);
```

Эта команда завершает вашу работу, записывая ее на диск в указанном формате.

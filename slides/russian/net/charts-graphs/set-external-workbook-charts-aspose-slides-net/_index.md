## Руководство по внедрению
### Настройка внешней рабочей книги для диаграммы
Эта функция позволяет связать данные диаграммы с внешней книгой Excel, гарантируя, что любые обновления в книге автоматически отразятся в презентации.
#### Шаг 1: Инициализация презентации и добавление диаграммы
Создайте новый экземпляр презентации и добавьте круговую диаграмму на первый слайд.
```csharp
using Aspose.Slides;
using Aspose.Slides.Charts;

public class Feature_SetExternalWorkbook {
    public static void Run() {
        string dataDir = "YOUR_DOCUMENT_DIRECTORY";
        
        using (Presentation pres = new Presentation()) {
            // Добавьте круговую диаграмму на первый слайд в позиции 50,50 размером 400x600.
            IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.Pie, 50, 50, 400, 600, false);
```
#### Шаг 2: Доступ к данным диаграммы и настройка внешней рабочей книги
Получите доступ к коллекции данных диаграммы, чтобы указать внешнюю книгу в качестве источника данных.
```csharp
            // Доступ к данным диаграммы для манипулирования.
            IChartData chartData = chart.ChartData;
            
            // Задайте внешнюю книгу, содержащую данные диаграммы.
            chartData.SetExternalWorkbook(dataDir + "externalWorkbook.xlsx");
```
#### Шаг 3: Добавьте ряды и точки данных из внешней рабочей книги
Добавьте новую серию в диаграмму, связав ее с определенными ячейками во внешней рабочей книге как для категорий, так и для значений.
```csharp
            // Добавить новый ряд, используя данные из ячейки B1 во внешней рабочей книге.
            chartData.Series.Add(chartData.ChartDataWorkbook.GetCell(0, "B1"), ChartType.Pie);

            // Добавьте точки данных для ряда из ячеек B2, B3 и B4.
            chartData.Series[0].DataPoints.AddDataPointForPieSeries(
                chartData.ChartDataWorkbook.GetCell(0, "B2"));
            chartData.Series[0].DataPoints.AddDataPointForPieSeries(
                chartData.ChartDataWorkbook.GetCell(0, "B3"));
            chartData.Series[0].DataPoints.AddDataPointForPieSeries(
                chartData.ChartDataWorkbook.GetCell(0, "B4"));

            // Определите категории для ряда, используя данные из ячеек A2, A3 и A4.
            chartData.Categories.Add(chartData.ChartDataWorkbook.GetCell(0, "A2"));
            chartData.Categories.Add(chartData.ChartDataWorkbook.GetCell(0, "A3"));
            chartData.Categories.Add(chartData.ChartDataWorkbook.GetCell(0, "A4"));

            // Сохраните презентацию с указанным именем файла
            pres.Save(dataDir + "Presentation_with_externalWorkbook.pptx");
        }
    }
}
```
### Советы по устранению неполадок
- Убедитесь, что путь к внешней рабочей книге правильный и доступный.
- Убедитесь, что ссылки на ячейки в вашем коде соответствуют ссылкам в файле Excel.

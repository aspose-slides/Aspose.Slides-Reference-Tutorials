## Руководство по внедрению

### Функция: переключение строк и столбцов диаграммы

#### Обзор
Переключение строк и столбцов в диаграммах имеет важное значение при подготовке презентаций, ориентированных на данные. Эта функция позволяет плавно корректировать Aspose.Slides, гарантируя четкое представление ваших данных.

#### Шаги по реализации

##### Шаг 1: Создайте новую презентацию
Начните с создания новой презентации, в которую вы добавите диаграмму:

```csharp
using (Presentation pres = new Presentation())
{
    // Код для добавления и изменения диаграмм находится здесь
}
```

##### Шаг 2: Добавьте кластеризованную столбчатую диаграмму
Добавьте кластеризованную столбчатую диаграмму на первый слайд в указанном месте и размере:

```csharp
IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 100, 100, 400, 300);
```

##### Шаг 3: Доступ к данным диаграммы
Извлеките данные серий и категорий из диаграммы, чтобы манипулировать ими:

```csharp
IChartSeries[] series = new IChartSeries[chart.ChartData.Series.Count];
chart.ChartData.Series.CopyTo(series, 0);

IChartDataCell[] categoriesCells = new IChartDataCell[chart.ChartData.Categories.Count];
for (int i = 0; i < chart.ChartData.Categories.Count; i++)
{
    categoriesCells[i] = chart.ChartData.Categories[i].AsCell;
}

IChartDataCell[] seriesCells = new IChartDataCell[chart.ChartData.Series.Count];
for (int i = 0; i < chart.ChartData.Series.Count; i++)
{
    seriesCells[i] = chart.ChartData.Series[i].Name.AsCells[0];
}
```

##### Шаг 4: Поменяйте местами строки и столбцы
Вызовите метод, чтобы поменять местами строки и столбцы, изменив ориентацию данных:

```csharp
chart.ChartData.SwitchRowColumn();
```

##### Шаг 5: Сохраните презентацию
Наконец, сохраните презентацию с измененной диаграммой:

```csharp
pres.Save("YOUR_OUTPUT_DIRECTORY" + "SwitchChartRowColumns_out.pptx", SaveFormat.Pptx);
```

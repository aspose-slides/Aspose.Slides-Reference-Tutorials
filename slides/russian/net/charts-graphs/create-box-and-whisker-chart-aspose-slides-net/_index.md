### Создание диаграммы «ящик с усами»
#### Обзор
Эта функция позволяет программно создавать в PowerPoint подробную диаграмму «ящик с усами», дополненную пользовательскими данными и конфигурациями.

#### Пошаговая реализация
##### 1. Определить каталог документов
Начните с указания каталога, в котором находится или будет сохранен файл вашей презентации:
```csharp
string dataDir = @"YOUR_DOCUMENT_DIRECTORY";
```
Этот путь гарантирует, что ваш скрипт знает, откуда считывать данные или куда записывать данные из файлов.

##### 2. Загрузить или создать презентацию
Откройте существующую презентацию PowerPoint или создайте новую, если необходимо:
```csharp
using (Presentation pres = new Presentation(dataDir + "test.pptx"))
{
    // Код для добавления и настройки диаграммы находится здесь.
}
```
##### 3. Добавьте диаграмму «ящик с усами» на слайд
Вставьте диаграмму «ящик с усами» в первый слайд в позицию `(50, 50)` с размерами `500 x 400`:
```csharp
IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.BoxAndWhisker, 50, 50, 500, 400);
```
На этом этапе необходимо выбрать нужный слайд и настроить начальное размещение диаграммы.
##### 4. Очистить существующие данные
Удалите все существующие категории или серии, чтобы начать с чистого листа:
```csharp
chart.ChartData.Categories.Clear();
chart.ChartData.Series.Clear();
```
Очистка гарантирует, что вы непреднамеренно не дублируете данные при добавлении новых записей.
##### 5. Рабочая тетрадь Access Chart
Используйте рабочую книгу, связанную с данными вашей диаграммы, для дальнейших манипуляций:
```csharp
IChartDataWorkbook wb = chart.ChartData.ChartDataWorkbook;
```
Рабочая книга действует как контейнер, в котором можно программно добавлять или изменять данные диаграммы.
##### 6. Очистить данные рабочей книги
Убедитесь, что не осталось ячеек, очистив начальный индекс:
```csharp
wb.Clear(0);
```
##### 7. Добавить категории в диаграмму
Пройдитесь по категориям и заполните их для вашей диаграммы, добавляя каждую в качестве новой строки в столбец A:
```csharp
for (int i = 1; i <= 6; i++)
{
    chart.ChartData.Categories.Add(wb.GetCell(0, "A" + i, "Category 1"));
}
```
Этот шаг позволяет вам систематизировать категории данных на диаграмме.

#### Основные параметры конфигурации
- **Тип диаграммы:** Выбирать `ChartType.BoxAndWhisker` для создания диаграмм типа «ящик с усами».
- **Позиционирование и размер:** Отрегулируйте положение `(50, 50)` и размер `(500, 400)` на основе требований к макету слайда.
- **Управление данными:** Используйте рабочую книгу для эффективного управления данными.

### Советы по устранению неполадок
Наиболее распространенные проблемы, с которыми вы можете столкнуться:
- **Ошибки пути к файлу:** Обеспечить `dataDir` настроен правильно, чтобы избежать исключений «файл не найден».
- **Проблемы с лицензией:** Если вы столкнулись с ограничениями функциональности, проверьте, правильно ли инициализирована ваша лицензия.

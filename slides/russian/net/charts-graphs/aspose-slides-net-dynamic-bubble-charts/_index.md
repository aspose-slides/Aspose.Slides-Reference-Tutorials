## Руководство по внедрению
### Добавление пузырьковой диаграммы в презентацию
Для начала вам нужно добавить на слайд пузырьковую диаграмму:

#### Шаг 1: Создайте или откройте презентацию
```csharp
using Aspose.Slides;
using Aspose.Slides.Charts;
// Укажите путь к каталогу для сохранения документов
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
// Создать новый экземпляр презентации
using (Presentation pres = new Presentation())
{
    // Добавьте пузырьковую диаграмму на первый слайд в позицию (50, 50) с шириной и высотой 600x400 пикселей.
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.Bubble, 50, 50, 600, 400, true);
```
#### Шаг 2: Настройте представление размера пузырька
Установите размер пузырька для представления определенного измерения данных. В этом примере используется `Width` свойство:
```csharp
    // Установить представление размера пузырька на основе «Ширины»
    chart.ChartData.SeriesGroups[0].BubbleSizeRepresentation = BubbleSizeRepresentationType.Width;
```
#### Шаг 3: Сохраните презентацию
Наконец, сохраните презентацию, чтобы увидеть изменения, отраженные в ваших диаграммах.
```csharp
    // Сохраните измененную презентацию
    pres.Save(dataDir + "Presentation_BubbleSizeRepresentation.pptx");
}
```
### Основные параметры конфигурации
- **BubbleSizeRepresentationType**: Выбирайте между `Width`, `Height`, или `Volume` на основе характеристик ваших данных.
- **ChartType.Bubble**: необходим для создания пузырьковых диаграмм, которые могут представлять несколько измерений данных.

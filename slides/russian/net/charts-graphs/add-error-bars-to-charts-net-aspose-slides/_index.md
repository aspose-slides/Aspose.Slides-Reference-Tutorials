### Добавление планок погрешностей на диаграмму
#### Обзор
Добавление планок погрешностей помогает визуально представить изменчивость или неопределенность данных в ваших диаграммах. Эта функция особенно полезна в научных и финансовых презентациях, где точность имеет значение.

#### Пошаговая реализация
**1. Создайте пустую презентацию**
Начните с создания нового объекта презентации:
```csharp
using (Presentation presentation = new Presentation())
{
    // Дальнейший код будет здесь.
}
```

**2. Добавьте пузырьковую диаграмму на слайд**
Добавьте диаграмму на слайд в указанных координатах с нужными размерами:
```csharp
IChart chart = presentation.Slides[0].Shapes.AddChart(
    ChartType.Bubble, 50, 50, 400, 300, true);
```

**3. Настройте планки погрешностей для осей X и Y.**
Доступ к форматам панели погрешностей для их настройки:
```csharp
IErrorBarsFormat errBarX = chart.ChartData.Series[0].ErrorBarsXFormat;
IErrorBarsFormat errBarY = chart.ChartData.Series[0].ErrorBarsYFormat;

errBarX.IsVisible = true;  // Включить видимость для X-полосок погрешностей
erBarY.IsVisible = true;  // Включить видимость для планок погрешностей Y

// Установите типы и значения для планок погрешностей
errBarX.ValueType = ErrorBarValueType.Fixed;
errBarX.Value = 0.1f;  // Фиксированное значение для планки погрешности X

errBarY.ValueType = ErrorBarValueType.Percentage;
erBarY.Value = 5;  // Процентное значение для полосы погрешности Y

// Настроить дополнительные свойства
erBarX.Type = ErrorBarType.Plus;
errBarY.Format.Line.Width = 2;  // Установить ширину линии для планок погрешностей Y
erBarX.HasEndCap = true;  // Включить конечный элемент для планок погрешностей X
```

**4. Сохраните презентацию**
Наконец, сохраните презентацию в указанном каталоге:
```csharp
presentation.Save(dataDir + "ErrorBars_out.pptx");
```

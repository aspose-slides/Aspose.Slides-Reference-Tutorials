### 2. Доступ к данным диаграммы и их изменение

#### Обзор

Мы изменим книгу данных диаграммы, чтобы вставить значения дат в качестве категорий.

#### Очистить существующие категории и серии

```csharp
// Доступ к рабочей книге данных диаграмм для манипуляций
class Program
{
    static void Main()
    {
        using (Presentation pres = new Presentation())
        {
            IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.Area, 50, 50, 450, 300);
            IChartDataWorkbook wb = chart.ChartData.ChartDataWorkbook;

            // Очистить существующие категории и серии в данных диаграммы
            chart.ChartData.Categories.Clear();
            chart.ChartData.Series.Clear();
        }
    }
}
```

#### Добавить значения дат как новые категории

Используйте этот фрагмент для вставки дат:

```csharp
// Доступ к рабочей книге данных диаграмм для манипуляций
class Program
{
    static void Main()
    {
        using (Presentation pres = new Presentation())
        {
            IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.Area, 50, 50, 450, 300);
            IChartDataWorkbook wb = chart.ChartData.ChartDataWorkbook;

            // Добавить значения дат в качестве новых категорий в диаграмму
            chart.ChartData.Categories.Add(wb.GetCell(0, "A2", DateTime.Now.AddDays(-30)));
            chart.ChartData.Categories.Add(wb.GetCell(0, "A3", DateTime.Now));
            chart.ChartData.Categories.Add(wb.GetCell(0, "A4", DateTime.Now.AddDays(30)));

            // Добавьте серию и заполните ее данными
            IChartSeries series = chart.ChartData.Series.Add(wb.GetCell(0, "B1", "Sample Series"), chart.Type);
        }
    }
}
```

### 3. Установите пользовательский формат даты

#### Обзор

Теперь настройте ось категорий для отображения дат в предпочитаемом вами формате.

#### Настроить ось категорий

```csharp
// Доступ к оси категорий и установка пользовательского формата даты.
class Program
{
    static void Main()
    {
        using (Presentation pres = new Presentation())
        {
            IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.Area, 50, 50, 450, 300);
            IChartDataWorkbook wb = chart.ChartData.ChartDataWorkbook;

            // Добавить значения дат в качестве новых категорий в диаграмму
            chart.ChartData.Categories.Add(wb.GetCell(0, "A2", DateTime.Now.AddDays(-30)));
            chart.ChartData.Categories.Add(wb.GetCell(0, "A3", DateTime.Now));
            chart.ChartData.Categories.Add(wb.GetCell(0, "A4", DateTime.Now.AddDays(30)));

            // Добавьте серию и заполните ее данными
            IChartSeries series = chart.ChartData.Series.Add(wb.GetCell(0, "B1", "Sample Series"), chart.Type);

            // Доступ к оси категорий и установка пользовательского формата даты.
            IAxis categoryAxis = chart.Axes.HorizontalAxis;
            categoryAxis.MajorUnit = 1; // Установить основную единицу измерения как дни
            categoryAxis.NumberFormat.FormatCode = "dd-MMM"; // Пользовательский формат: сокращение дня-месяца

            // Сохраните презентацию с изменениями
            pres.Save(@"YOUR_DOCUMENT_DIRECTORY\FormattedChart.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
        }
    }
}
```

#### Параметры и методы Пояснение
- **MajorUnit**: Устанавливает интервал для основных делений на оси.
- **ФорматЧисла.ФорматКод**: Определяет, как отображаются даты. Формат `"dd-MMM"` отображает сокращенное обозначение дня и месяца.

## Руководство по внедрению

Давайте шаг за шагом рассмотрим настройку свойств шрифта легенды.

### Доступ к записям легенд и их изменение

Сначала давайте добавим диаграмму на слайд и получим доступ к ее легендам:

#### Добавление диаграммы
```csharp
// Загрузить существующую презентацию
using (Presentation pres = new Presentation("YOUR_DOCUMENT_DIRECTORY/test.pptx"))
{
    // Добавьте кластеризованную столбчатую диаграмму в позиции x=50, y=50 с шириной=600 и высотой=400.
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 50, 50, 600, 400);
}
```

#### Доступ к легенде
```csharp
// Доступ к объекту текстового формата второй записи легенды
IChartTextFormat tf = chart.Legend.Entries[1].TextFormat;
```

### Настройка свойств шрифта

Теперь настройте свойства шрифта, такие как жирность, высоту и цвет:

#### Установка жирного шрифта и курсива
```csharp
tf.PortionFormat.FontBold = NullableBool.True; // Сделать текст жирным
tf.PortionFormat.FontItalic = NullableBool.True; // Применить курсив
```

#### Регулировка высоты шрифта
```csharp
tf.PortionFormat.FontHeight = 20; // Установить размер шрифта 20 пунктов.
```

#### Изменение цвета шрифта
```csharp
// Установите тип заливки и цвет текста
tf.PortionFormat.FillFormat.FillType = FillType.Solid;
tf.PortionFormat.FillFormat.SolidFillColor.Color = Color.Blue; // Применить синий цвет
```

### Сохранение презентации

Наконец, сохраните измененную презентацию:

```csharp
pres.Save("YOUR_DOCUMENT_DIRECTORY/output.pptx", SaveFormat.Pptx);
```

### Настройте размер шрифта легенды
Изменение размера шрифта легенды улучшает читаемость. Вот как это сделать:

#### Обзор
Мы изменим размер шрифта текста легенды диаграммы с помощью Aspose.Slides для .NET.

#### Шаги
**1. Загрузите презентацию:**
Начните с загрузки файла PowerPoint, в котором вы хотите настроить легенду диаграммы.
```csharp
using (Presentation pres = new Presentation(dataDir))
{
    // Откройте первый слайд и добавьте кластеризованную столбчатую диаграмму.
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 50, 50, 600, 400);
```

**2. Установите размер шрифта легенды:**
Укажите желаемую высоту шрифта для лучшей видимости.
```csharp
    // Измените размер шрифта текста легенды на 20.
    chart.Legend.TextFormat.PortionFormat.FontHeight = 20;
}
```
- **Объяснение:** `FontHeight` задает размер в пунктах, улучшая читаемость.

**3. Сохраните презентацию:**
После внесения изменений сохраните презентацию, чтобы сохранить их.
```csharp
pres.Save(outputDir, Aspose.Slides.Export.SaveFormat.Pptx);
```

### Настройте минимальные и максимальные значения вертикальной оси
Настройка значений осей позволяет обеспечить точное представление данных.

#### Обзор
Узнайте, как установить определенные минимальные и максимальные значения для вертикальной оси диаграммы.

#### Шаги
**1. Загрузите презентацию:**
Как и прежде, откройте презентацию, содержащую вашу диаграмму.
```csharp
using (Presentation pres = new Presentation(dataDir))
{
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 50, 50, 600, 400);
```

**2. Задайте пользовательские значения осей:**
Отключите автоматическую настройку значений осей и задайте свои собственные.
```csharp
    // Отключить авто-минимум для вертикальной оси.
    chart.Axes.VerticalAxis.IsAutomaticMinValue = false;
    // Установите минимальное значение -5.
    chart.Axes.VerticalAxis.MinValue = -5;

    // Аналогично отключите автомакс и установите значение 10.
    chart.Axes.VerticalAxis.IsAutomaticMaxValue = false;
    chart.Axes.VerticalAxis.MaxValue = 10;
}
```
- **Объяснение:** Настройка этих значений позволяет индивидуально масштабировать данные.

**3. Сохраните презентацию:**
Убедитесь, что ваши изменения сохранены, записав их обратно в файл.
```csharp
pres.Save(outputDir, Aspose.Slides.Export.SaveFormat.Pptx);
```

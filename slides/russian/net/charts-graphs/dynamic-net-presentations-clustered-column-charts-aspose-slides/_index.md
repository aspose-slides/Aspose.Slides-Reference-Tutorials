## Руководство по внедрению

### Функция 1: Создание презентации и добавление диаграммы

#### Обзор
Создание презентаций программным способом позволяет автоматизировать и настраивать. Эта функция демонстрирует, как инициализировать презентацию и добавить кластеризованную столбчатую диаграмму, идеально подходящую для сравнения данных по категориям.

#### Пошаговая реализация

**Инициализировать презентацию**
```csharp
Presentation pres = new Presentation();
```

**Доступ к первому слайду**
Начните с первого слайда:
```csharp
ISlide slide = pres.Slides[0];
```

**Добавить кластеризованную столбчатую диаграмму**
Вставьте диаграмму в позицию (100, 100) на слайде с размерами 600x450 пикселей.
```csharp
IChart ch = slide.Shapes.AddChart(ChartType.ClusteredColumn, 100, 100, 600, 450);
```
*Объяснение:* Этот метод создает новую кластеризованную столбчатую диаграмму. Параметры определяют ее положение и размер.

**Очистить существующие серии и категории**
Для начала свежие данные:
```csharp
ch.ChartData.Series.Clear();
ch.ChartData.Categories.Clear();
```

### Функция 2: Добавление категорий с уровнями группировки

#### Обзор
Организация данных по категориям с уровнями группировки улучшает читаемость и структуру, что крайне важно для эффективных презентаций.

**Создавайте категории и устанавливайте уровни группировки**
Выполните итерацию по диапазону для создания категорий:
```csharp
IChartDataWorkbook fact = ch.ChartData.ChartDataWorkbook;
fact.Clear(0);

int defaultWorksheetIndex = 0;

for (int i = 2; i <= 9; i++)
{
    IChartCategory category = ch.ChartData.Categories.Add(fact.GetCell(0, "c" + i, System.Convert.ToChar('A' + (i - 2))));
    
    string groupName = "Group" + ((i - 1) / 2 + 1);
    category.GroupingLevels.SetGroupingItem(1, groupName);
}
```
*Объяснение:* Этот цикл добавляет категории с уникальными уровнями группировки, улучшая иерархическую структуру диаграммы.

### Функция 3: Добавление рядов и точек данных на диаграмму

#### Обзор
Заполнение диаграммы точками данных имеет решающее значение для визуального представления. Этот шаг включает добавление серии данных, соответствующих каждой категории.

**Добавить серии и заполнить данные**
```csharp
IChartSeries series = ch.ChartData.Series.Add(fact.GetCell(0, "D1", "Series 1"), ChartType.ClusteredColumn);

for (int j = 2; j <= 9; j++)
{
    series.DataPoints.AddDataPointForBarSeries(fact.GetCell(defaultWorksheetIndex, "D" + j, j * 10));
}
```
*Объяснение:* Этот код добавляет новую серию данных и заполняет ее точками. Каждая точка представляет собой значение, полученное из местоположения ячейки.

### Функция 4: Сохранение презентации с диаграммой

#### Обзор
После того как диаграмма будет готова, сохранение презентации сохранит все изменения и позволит вам поделиться данными или представить их.

**Сохраните свою работу**
```csharp
pres.Save(outputPath + "/AsposeChart_out.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
```
*Объяснение:* The `Save` Метод фиксирует вашу работу в файле PPTX, делая его готовым к распространению или презентации.

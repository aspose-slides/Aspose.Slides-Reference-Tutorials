### Обзор функций: точность меток данных в диаграммах
Эта функция позволяет вам форматировать точность чисел в метках данных на диаграммах, гарантируя, что ваша числовая информация будет отображаться именно так, как нужно.

#### Шаг 1: Создайте презентацию
Начнем с создания нового экземпляра презентации, в котором будет находиться наша диаграмма:
```csharp
using Aspose.Slides;
using Aspose.Slides.Charts;

// Пути к каталогам
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// Инициализируйте объект презентации
global using (Presentation pres = new Presentation())
{
    // Добавьте линейную диаграмму на первый слайд в позицию (50, 50) с размером (450, 300)
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.Line, 50, 50, 450, 300);
    
    // Отобразить таблицу данных на диаграмме
    chart.HasDataTable = true;
```

#### Шаг 2: Форматирование меток данных
Установите числовой формат для значений ряда с двумя знаками после запятой:
```csharp
    // Установить числовой формат для значений серии с двумя знаками после запятой
    chart.ChartData.Series[0].NumberFormatOfValues = "#,##0.00";
    
    // Сохраните презентацию с отформатированными метками данных
    pres.Save(outputDir + "/PrecisionOfDatalabels_out.pptx");
}
```
- **Параметры и назначение метода:** `NumberFormatOfValues` — это свойство, позволяющее определить, как числа будут отображаться на диаграмме, обеспечивая точное форматирование.
  
### Советы по устранению неполадок
- Убедитесь, что указаны каталоги (`dataDir`, `outputDir`) существуют или обрабатывают исключения, если их нет.
- Если диаграмма отображается не так, как ожидалось, проверьте строку формата и проверьте ее на наличие опечаток.

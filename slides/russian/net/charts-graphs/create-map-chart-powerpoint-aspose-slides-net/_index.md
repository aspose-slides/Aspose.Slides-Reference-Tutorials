### Создание интерактивной карты-диаграммы в PowerPoint

#### Обзор
В этом разделе вы узнаете, как добавить картографическую диаграмму на свой первый слайд, настроить ее с помощью точек данных и сохранить презентацию. 

##### Добавление нового слайда с картой-диаграммой
1. **Добавить пустую карту**: Создайте новую карту-схему на первом слайде.

```csharp
using Aspose.Slides;
using Aspose.Slides.Charts;

string resultPath = @"YOUR_OUTPUT_DIRECTORY/MapChart_out.pptx";

using (Presentation presentation = new Presentation())
{
    // Добавить карту-схему в позицию (50, 50) размером (500, 400)
    IChart chart = presentation.Slides[0].Shapes.AddChart(ChartType.Map, 50, 50, 500, 400, false);
```

##### Настройка данных диаграммы
2. **Доступ к рабочей книге данных диаграмм**: Эта рабочая книга позволяет вам управлять данными для вашей серии карт.

```csharp
    IChartDataWorkbook wb = chart.ChartData.ChartDataWorkbook;
```

3. **Добавить серию с точками данных**: Заполните свою карту-схему, добавив ряд и связав его с определенными географическими точками данных.

```csharp
    // Добавить новую серию в диаграмму
    IChartSeries series = chart.ChartData.Series.Add(ChartType.Map);
    
    // Пример: добавление точки данных для страны во вторую строку, третий столбец рабочей книги.
    series.DataPoints.AddDataPointForMapSeries(wb.GetCell(0, "B2", "CountryName"));
```

##### Сохранение презентации
4. **Сохраните свой файл PowerPoint**: После настройки диаграммы сохраните презентацию, чтобы просмотреть карту.

```csharp
    // Сохраните презентацию с новой картой-схемой
    presentation.Save(resultPath, Aspose.Slides.Export.SaveFormat.Pptx);
}
```

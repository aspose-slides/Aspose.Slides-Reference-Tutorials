## Руководство по внедрению
### Настройка параметров пользовательской легенды для диаграммы
Настройка легенд диаграмм позволяет адаптировать презентации в соответствии с конкретными потребностями, повышая ясность и улучшая дизайн.

#### Обзор:
Эта функция позволяет настраивать положение и размеры легенды на диаграмме в PowerPoint с помощью Aspose.Slides для .NET.

#### Этапы реализации:
**Шаг 1: Создание экземпляра класса представления**
```csharp
// Определите каталог ваших документов
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
Presentation presentation = new Presentation();
```

**Шаг 2: Получите доступ к первому слайду**
```csharp
ISlide slide = presentation.Slides[0];
```

**Шаг 3: Добавьте на слайд кластеризованную столбчатую диаграмму**
```csharp
IChart chart = slide.Shapes.AddChart(ChartType.ClusteredColumn, 50, 50, 500, 500);
```
*Объяснение:* Этот фрагмент добавляет кластеризованную столбчатую диаграмму в указанных координатах на слайде.

**Шаг 4: Задайте свойства легенды**
```csharp
// Настройте положение легенды относительно размеров диаграммы
chart.Legend.X = 50 / chart.Width;
chart.Legend.Y = 50 / chart.Height;
// Определить ширину и высоту как процент от размера диаграммы.
chart.Legend.Width = 100 / chart.Width;
chart.Legend.Height = 100 / chart.Height;
```
*Почему это важно:* Изменение положения легенды гарантирует ее правильное вписывание в макет презентации.

**Шаг 5: Сохраните презентацию**
```csharp
presentation.Save(dataDir + "Legend_out.pptx", SaveFormat.Pptx);
```

### Создание презентации и добавление фигур
Добавление различных фигур, включая диаграммы, может повысить визуальную привлекательность ваших слайдов.

#### Обзор:
Эта функция демонстрирует, как создать презентацию PowerPoint и добавить различные фигуры, такие как прямоугольники или другие типы диаграмм.

#### Этапы реализации:
**Шаг 1: Инициализация нового экземпляра презентации**
```csharp
class Program
{
    static void Main()
    {
        // Инициализируйте новый экземпляр презентации
        Presentation presentation = new Presentation();
    }
}
```

**Шаг 2: Получите доступ к первому слайду**
```csharp
ISlide slide = presentation.Slides[0];
```

**Шаг 3: Добавьте фигуры на слайд**
```csharp
// Пример добавления прямоугольной формы
IShape rectangle = slide.Shapes.AddAutoShape(ShapeType.Rectangle, 150, 75, 150, 50);
```
*Объяснение:* Этот фрагмент кода добавляет прямоугольную фигуру в указанные координаты на первом слайде.

**Шаг 4: Сохраните презентацию**
```csharp
presentation.Save(dataDir + "Shapes_out.pptx", SaveFormat.Pptx);
```

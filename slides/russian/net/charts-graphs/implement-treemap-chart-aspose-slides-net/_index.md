## Руководство по внедрению
Давайте разобьем процесс добавления и настройки диаграммы TreeMap на удобные для выполнения шаги.

### Шаг 1: Загрузите существующую презентацию
Начните с загрузки существующего файла презентации, в который вы хотите добавить диаграмму TreeMap:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY/test.pptx";
using (Presentation pres = new Presentation(dataDir))
{
    // Продолжайте добавлять диаграмму TreeMap.
}
```

### Шаг 2: Добавьте диаграмму TreeMap
Добавьте диаграмму в желаемое место на первом слайде и укажите ее размеры:
```csharp
IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.Treemap, 50, 50, 500, 400);
```

### Шаг 3: Очистите существующие данные
Убедитесь, что все ранее существовавшие данные в вашей диаграмме удалены, чтобы начать все заново:
```csharp
chart.ChartData.Categories.Clear();
chart.ChartData.Series.Clear();

IChartDataWorkbook wb = chart.ChartData.ChartDataWorkbook;
wb.Clear(0); // Очищает рабочую книгу для чистого состояния
```

### Шаг 4: Определите и добавьте категории
Определите категории с иерархическими уровнями группировки. Эта структура помогает эффективно организовать данные:
```csharp
// Определить категории для ветви 1
IChartCategory leaf = chart.ChartData.Categories.Add(wb.GetCell(0, "C1", "Leaf1"));
leaf.GroupingLevels.SetGroupingItem(1, "Stem1");
leaf.GroupingLevels.SetGroupingItem(2, "Branch1");

chart.ChartData.Categories.Add(wb.GetCell(0, "C2", "Leaf2"));

// Повторите для дополнительных категорий.
```

### Шаг 5: Добавьте серию и настройте точки данных
Добавьте точки данных в ряд диаграммы, обеспечив представление каждой категории:
```csharp
IChartSeries series = chart.ChartData.Series.Add(ChartType.Treemap);
series.Labels.DefaultDataLabelFormat.ShowCategoryName = true;

// Добавление точек данных для категорий
series.DataPoints.AddDataPointForTreemapSeries(wb.GetCell(0, "D1", 4));
series.DataPoints.AddDataPointForTreemapSeries(wb.GetCell(0, "D2", 5));
// Продолжайте добавлять другие точки данных...
```

### Шаг 6: Настройте макет родительской метки
Измените макет, чтобы улучшить видимость и эстетику:
```csharp
series.ParentLabelLayout = ParentLabelLayoutType.Overlapping;
```

### Шаг 7: Сохраните презентацию
Наконец, сохраните презентацию с недавно добавленной диаграммой TreeMap:
```csharp
pres.Save("YOUR_OUTPUT_DIRECTORY/Treemap.pptx", SaveFormat.Pptx);
```

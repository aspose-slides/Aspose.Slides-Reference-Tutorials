### Функция: Создание презентации (H3)
**Обзор:** Эта функция демонстрирует, как инициализировать пустой файл PowerPoint с помощью Aspose.Slides.
```csharp

class FeatureCreatePresentation {
    public static void Run() {
        // Инициализируйте новый объект Presentation
        Presentation pres = new Presentation();

        // Сохраните презентацию в желаемом каталоге.

    }
}
```
**Объяснение:** The `Presentation` создается экземпляр класса, и пустой файл сохраняется по указанному пути.

### Функция: Добавить фигуру SmartArt (H3)
**Обзор:** Узнайте, как добавить графический элемент SmartArt на первый слайд презентации для повышения ее визуальной привлекательности.
```csharp
using Aspose.Slides;
using Aspose.Slides.SmartArt;

class FeatureAddSmartArtShape {
    public static void Run() {
        // Инициализируйте новый объект Presentation
        Presentation pres = new Presentation();

        // Доступ к первому слайду презентации
        ISlide slide = pres.Slides[0];

        // Добавить фигуру SmartArt на слайд в указанном месте и размере
        ISmartArt smart = slide.Shapes.AddSmartArt(50, 150, 400, 400, SmartArtLayoutType.StackedList);

        // Сохраните презентацию с добавленным SmartArt
        string outputDir = "/YOUR_OUTPUT_DIRECTORY";  // Обновите, указав ваш фактический путь
        pres.Save(outputDir + "PresentationWithSmartArt_out.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
    }
}
```
**Объяснение:** Этот код получает доступ к первому слайду, добавляет `StackedList` Введите графику SmartArt в указанные координаты и сохраните ее. Отрегулируйте положение и размеры в соответствии с вашим макетом.

### Функция: добавление узла в определенное положение в SmartArt (H3)
**Обзор:** Улучшите существующий SmartArt, добавив узлы в точные места в его иерархии.
```csharp
using Aspose.Slides;
using Aspose.Slides.SmartArt;

class FeatureAddNodeToSmartArt {
    public static void Run() {
        // Инициализируйте новый объект Presentation
        Presentation pres = new Presentation();

        // Доступ к первому слайду презентации
        ISlide slide = pres.Slides[0];

        // Добавить фигуру SmartArt на слайд в указанном месте и размере
        ISmartArt smart = slide.Shapes.AddSmartArt(50, 150, 400, 400, SmartArtLayoutType.StackedList);

        // Доступ к первому узлу SmartArt
        ISmartArtNode node = smart.AllNodes[0];

        // Добавление нового дочернего узла в позицию с индексом 2 в коллекции дочерних узлов родительского узла
        SmartArtNode chNode = (SmartArtNode)((SmartArtNodeCollection)node.ChildNodes).AddNodeByPosition(2);

        // Установить текст для нового добавленного узла
        chNode.TextFrame.Text = "Sample Text Added";

        // Сохраните презентацию с измененным SmartArt
        string outputDir = "/YOUR_OUTPUT_DIRECTORY";  // Обновите, указав ваш фактический путь
        pres.Save(outputDir + "ModifiedSmartArt_out.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
    }
}
```
**Объяснение:** В этом фрагменте демонстрируется доступ к узлам и их изменение в графике SmartArt. `AddNodeByPosition` Метод обеспечивает точное размещение, что необходимо для структурированного контента.

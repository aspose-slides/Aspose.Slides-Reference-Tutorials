### Создание графики SmartArt
#### Обзор
Начнем с добавления базовой графики SmartArt в нашу презентацию. Этот процесс включает в себя инициализацию `Presentation` класс, добавляющий фигуру SmartArt и устанавливающий ее начальный тип макета.

#### Пошаговая реализация
**1. Инициализация презентации**
Создайте экземпляр `Presentation` сорт:

```csharp
using (Presentation presentation = new Presentation())
{
    // Код для добавления SmartArt будет здесь
}
```

Эта строка инициализирует новую презентацию PowerPoint, в которую вы добавите свой SmartArt.

**2. Добавить фигуру SmartArt**
Добавьте графический элемент SmartArt на первый слайд с исходным макетом `BasicBlockList`:

```csharp
ISmartArt smart = presentation.Slides[0].Shapes.AddSmartArt(10, 10, 400, 300, SmartArtLayoutType.BasicBlockList);
```

Здесь, `AddSmartArt` размещает новый графический элемент SmartArt в позиции (10, 10) с размерами 400x300 пикселей. `BasicBlockList` макет представляет собой простой стиль маркированного списка.

**3. Изменить макет SmartArt**
Измените существующий SmartArt, чтобы использовать другой макет:

```csharp
smart.Layout = SmartArtLayoutType.BasicProcess;
```

Изменение макета обновляет визуальную структуру вашего SmartArt, преобразуя его в схему технологического процесса.

#### Пояснение кода
- **`AddSmartArt` Метод**: Этот метод имеет решающее значение для вставки новой графики SmartArt. Параметры включают координаты положения, размеры и начальный тип макета.
- **Изменение макета**: `smart.Layout` Свойство позволяет изменять существующий тип макета, обеспечивая универсальность в дизайне презентации.

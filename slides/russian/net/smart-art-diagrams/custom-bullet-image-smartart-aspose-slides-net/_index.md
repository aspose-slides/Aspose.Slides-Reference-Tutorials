## Руководство по внедрению

### Настройка вашего проекта

1. **Инициализация объекта презентации:**
   Начните с создания нового `Presentation` объект. Это представляет ваш файл PowerPoint.
   ```csharp
   using Aspose.Slides;
   using System.Drawing; // Для обработки изображений
   using System.IO; // Для файловых операций

   string dataDir = "YOUR_DOCUMENT_DIRECTORY";
   string outputDir = "YOUR_OUTPUT_DIRECTORY";

   Directory.CreateDirectory(dataDir);
   Directory.CreateDirectory(outputDir);

   using (Presentation presentation = new Presentation())
   {
       // Код продолжается...
   }
   ```

### Добавление фигуры SmartArt

2. **Добавьте SmartArt на слайд:**
   Создайте и разместите объект SmartArt на слайде.
   ```csharp
   ISmartArt smart = presentation.Slides[0].Shapes.AddSmartArt(10, 10, 500, 400, SmartArtLayoutType.VerticalPictureList);
   ```

3. **Доступ к узлу:**
   Извлеките первый узел для применения пользовательских настроек маркера.
   ```csharp
   ISmartArtNode node = smart.AllNodes[0];
   ```

### Настройка изображения маркера

4. **Установить пользовательское изображение маркера:**
   Загрузите и назначьте изображение в качестве маркера для узла SmartArt.
   ```csharp
   if (node.BulletFillFormat != null)
   {
       string imagePath = Path.Combine(dataDir, "aspose-logo.jpg");
       IImage img = Images.FromFile(imagePath);
       IPPImage image = presentation.Images.AddImage(img);

       // Применить пользовательское изображение маркера
       node.BulletFillFormat.FillType = FillType.Picture;
       node.BulletFillFormat.PictureFillFormat.Picture.Image = image;
       node.BulletFillFormat.PictureFillFormat.PictureFillMode = PictureFillMode.Stretch;
   }
   ```

### Сохранение презентации

5. **Сохраните измененную презентацию:**
   Наконец, сохраните свою презентацию с пользовательским рисунком SmartArt.
   ```csharp
   string outputPath = Path.Combine(outputDir, "out.pptx");
   presentation.Save(outputPath, SaveFormat.Pptx);
   ```

## Раздел часто задаваемых вопросов

1. **Как изменить размер изображения маркера?**
   - Отрегулируйте `Stretch` режим для подгонки под разные размеры или вручную изменить размер изображений перед их добавлением.

2. **Какие форматы файлов поддерживаются для пользовательских маркеров?**
   - Поддерживаются такие распространенные форматы, как JPEG, PNG и BMP; обеспечьте совместимость, конвертируя файлы по мере необходимости.

3. **Могу ли я применить эту настройку ко всем узлам в графике SmartArt?**
   - Да, повторить `smart.AllNodes` и применить аналогичные настройки к каждому узлу.

4. **Что делать, если изображение не загружается?**
   - Проверьте правильность пути к файлу и убедитесь, что изображение существует в этом месте.

5. **Как я могу дополнительно настроить графику SmartArt?**
   - Исследуйте другие свойства `ISmartArt` и `ISmartArtNode` для настройки цветов, стилей и многого другого.

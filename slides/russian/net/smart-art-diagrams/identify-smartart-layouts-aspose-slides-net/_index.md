#### Обзор

Доступ к презентации — это первый шаг. Вы загрузите файл в Aspose.Slides `Presentation` возразить, чтобы начать манипуляцию.

Вот как можно открыть презентацию из указанного каталога:

```csharp

{
    // Дальнейшая обработка будет проходить здесь

```

### Прохождение через слайд-формы

#### Обзор

Каждый слайд в вашей презентации содержит различные фигуры. Вам нужно определить, какие из них являются SmartArt.

#### Итерация по формам

Пройдитесь по каждой фигуре на первом слайде, чтобы проверить наличие SmartArt:

```csharp

        // Определите и обработайте фигуры SmartArt здесь
```

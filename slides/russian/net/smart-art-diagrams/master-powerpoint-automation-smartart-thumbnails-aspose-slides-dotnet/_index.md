## Руководство по внедрению

### Создание SmartArt и извлечение миниатюр

#### Обзор
В этом разделе мы добавим SmartArt в слайд PowerPoint и извлечем миниатюры из его узлов. Это автоматизирует создание графики и эффективно сохраняет визуальные элементы.

##### Шаг 1: Создание экземпляра класса представления
Создайте новый экземпляр `Presentation` сорт:

```csharp
using Aspose.Slides;

// Установите каталог документов
string dataDir = "YOUR_DOCUMENT_DIRECTORY";

// Создать новую презентацию
Presentation pres = new Presentation();
```

##### Шаг 2: Добавьте SmartArt на слайд
Добавьте фигуру SmartArt к первому слайду, используя базовый циклический макет:

```csharp
// Добавьте SmartArt в позицию (10, 10) шириной и высотой 400 пикселей каждый.
ISmartArt smart = pres.Slides[0].Shapes.AddSmartArt(10, 10, 400, 300, SmartArtLayoutType.BasicCycle);
```

##### Шаг 3: Доступ к узлу в SmartArt
Извлеките определенный узел, используя его индекс для работы с отдельными элементами:

```csharp
// Доступ ко второму узлу (индекс 1)
ISmartArtNode node = smart.Nodes[1];
```

##### Шаг 4: Извлечение и сохранение миниатюрного изображения
Получите миниатюру первой фигуры в этом узле и сохраните ее как файл изображения:

```csharp
// Получить миниатюру из первой фигуры в узле SmartArt
IImage img = node.Shapes[0].GetImage();

// Сохраните изображение по указанному пути.
img.Save(dataDir + "/SmartArt_ChildNote_Thumbnail_out.jpeg", ImageFormat.Jpeg);
```

### Основные параметры конфигурации и советы по устранению неполадок

- **Индексация формы**Доступ к допустимым индексам в узлах SmartArt. Индекс, выходящий за пределы диапазона, вызовет исключение.
- **Пути к файлам**: Обеспечить `dataDir` существует путь для предотвращения ошибок «файл не найден».

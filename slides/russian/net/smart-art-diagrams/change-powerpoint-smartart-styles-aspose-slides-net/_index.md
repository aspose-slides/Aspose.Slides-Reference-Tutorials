## Руководство по внедрению

Теперь давайте реализуем функцию изменения стилей SmartArt в презентации PowerPoint.

### Загрузка презентации

Начните с загрузки существующей презентации, в которой вы хотите изменить стили SmartArt:

```csharp
using Aspose.Slides;
using Aspose.Slides.Export;
using Aspose.Slides.SmartArt;

// Укажите каталог вашего документа
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
using (Presentation presentation = new Presentation(dataDir + "AccessSmartArtShape.pptx"))
{
    // Код реализации приведен ниже...
}
```

### Перемещение и изменение фигур SmartArt

Затем просмотрите фигуры в презентации, чтобы найти и изменить объекты SmartArt:

**Проверьте, является ли Shape объектом SmartArt:**

```csharp
foreach (IShape shape in presentation.Slides[0].Shapes)
{
    if (shape is ISmartArt)
    {
        // Продолжайте логику модификации...
```

**Изменить стиль SmartArt:**

Проверьте текущий стиль и обновите его при необходимости:

```csharp
        ISmartArt smart = (ISmartArt)shape;

        if (smart.QuickStyle == SmartArtQuickStyleType.SimpleFill)
        {
            smart.QuickStyle = SmartArtQuickStyleType.Cartoon;
        }
    }
}
```

### Сохранение измененной презентации

Наконец, сохраните изменения в новом файле:

```csharp
presentation.Save(dataDir + "ChangeSmartArtStyle_out.pptx", SaveFormat.Pptx);
```

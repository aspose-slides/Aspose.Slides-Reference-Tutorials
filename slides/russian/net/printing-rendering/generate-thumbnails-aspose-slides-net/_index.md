## Руководство по внедрению

В этом разделе вы узнаете, как создать миниатюру первой фигуры на слайде презентации.

### Создание миниатюры из формы слайда
Создание предварительного просмотра изображения (миниатюры) определенных фигур на слайдах полезно для веб-приложений, требующих быстрого предварительного просмотра, или при управлении большими презентациями.

#### Шаг 1: Настройка каталогов и файла презентации
Определите пути для входного документа и выходного каталога:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Замените на путь к каталогу ваших документов.
string outputDir = "YOUR_OUTPUT_DIRECTORY"; // Замените на путь к желаемому выходному каталогу.
```

#### Шаг 2: Загрузите презентацию
Создать экземпляр `Presentation` класс, представляющий ваш файл презентации:
```csharp
using (Presentation p = new Presentation(dataDir + "/HelloWorld.pptx"))
{
    // Доступ к первому слайду презентации
    ISlide slide = p.Slides[0];
```

#### Шаг 3: Доступ к фигуре и преобразование ее в изображение
Откройте первую фигуру на слайде и преобразуйте ее в изображение:
```csharp
    IShape shape = slide.Shapes[0];

    using (IImage img = shape.GetImage(ShapeThumbnailBounds.Shape, 1, 1))
    {
        // Сохраните полученную миниатюру на диск в формате PNG.
        img.Save(outputDir + "/Scaling Factor Thumbnail_out.png");
    }
}
```

**Объяснение:**
- `GetImage` захватывает полномасштабное изображение вашей формы. Параметры `(ShapeThumbnailBounds.Shape, 1, 1)` укажите захват всей фигуры без масштабирования.

#### Советы по устранению неполадок
- Убедитесь, что пути к файлам заданы правильно и доступны для вашего приложения.
- Проверьте наличие исключений, связанных с доступом к файлам или недопустимыми форматами представления.

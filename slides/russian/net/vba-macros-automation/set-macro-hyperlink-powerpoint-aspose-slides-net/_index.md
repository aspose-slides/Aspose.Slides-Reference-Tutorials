## Руководство по внедрению

Давайте рассмотрим установку макрос-гиперссылки на фигуру.

### Обзор функций: настройка макрос-гиперссылки

Эта функция позволяет прикреплять макрофункцию к фигурам в PowerPoint с помощью Aspose.Slides для .NET, что идеально подходит для создания интерактивных презентаций, реагирующих на действия пользователя.

#### Шаг 1: Создание формы

Добавьте автофигуру к своему слайду:

```csharp
using Aspose.Slides;

string macroName = "TestMacro";
using (Presentation presentation = new Presentation())
{
    // Добавьте форму пустой кнопки в позицию (20, 20) с размерами (80x30)
    IAutoShape shape = presentation.Slides[0].Shapes.AddAutoShape(ShapeType.BlankButton, 20, 20, 80, 30);
```

#### Шаг 2: Установите макрос-гиперссылку

Прикрепите макрос к этой форме:

```csharp
    // Свяжите фигуру с событием щелчка макрос-гиперссылки
    shape.HyperlinkManager.SetMacroHyperlinkClick(macroName);

    // Сохранить презентацию
    presentation.Save("output.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
}
```
**Объяснение:**
- `AddAutoShape(ShapeType.BlankButton, 20, 20, 80, 30)`: Добавляет пустую форму кнопки в указанные координаты и размер.
- `SetMacroHyperlinkClick(macroName)`: Связывает макрос с событием щелчка фигуры.

#### Советы по устранению неполадок

- **Макрос не запущен:** Убедитесь, что макрос присутствует в шаблоне PowerPoint.
- **Проблемы позиционирования формы:** Еще раз проверьте значения координат для точного размещения на слайде.

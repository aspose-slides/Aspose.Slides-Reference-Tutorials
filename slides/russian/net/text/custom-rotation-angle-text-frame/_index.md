# Поворот заголовков и меток диаграмм в PowerPoint с помощью Aspose.Slides

Aspose.Slides для .NET предоставляет мощные возможности для создания и управления презентациями PowerPoint программным способом. Этот пример демонстрирует, как создать презентацию с кластеризованной столбчатой диаграммой, которая содержит вращающиеся текстовые элементы — в частности, вращая как заголовок диаграммы, так и метки данных для улучшения читаемости и визуальной привлекательности.

## Инструкция по установке

1. Установка через менеджер пакетов NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Или установите через .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Добавьте пространство имен в ваш файл C#:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Charts;
   ```

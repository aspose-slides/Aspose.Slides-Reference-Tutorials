# Создание презентаций PowerPoint с помощью TextBox в Aspose.Slides для .NET

Aspose.Slides для .NET — это мощный API, позволяющий разработчикам создавать, изменять и конвертировать презентации PowerPoint без необходимости использования Microsoft PowerPoint. В этом примере показано, как создать презентацию с настраиваемым текстовым полем, применить форматирование и сохранить ее как файл PPTX.

## Инструкция по установке

1. Установка через менеджер пакетов NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Установка через .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Установить вручную:
   - Загрузите последнюю версию с сайта [Страница релизов Aspose.Slides для .NET](https://releases.aspose.com/slides/net/)
   - Добавьте ссылку на DLL в свой проект

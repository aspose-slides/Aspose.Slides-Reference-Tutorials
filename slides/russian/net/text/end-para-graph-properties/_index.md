# Создание презентаций PowerPoint с форматированным текстом

Aspose.Slides для .NET — это мощный API, который позволяет разработчикам создавать, изменять и конвертировать презентации PowerPoint без необходимости использования Microsoft PowerPoint. В этом примере показано, как создать простую презентацию, содержащую прямоугольную форму с отформатированными текстовыми абзацами.

Код создает новую презентацию, добавляет прямоугольную форму к первому слайду, а затем добавляет несколько абзацев с различными параметрами форматирования текста. Он показывает, как задать свойства шрифта, такие как размер и семейство, для определенных частей текста.

## Инструкция по установке

1. Установка через менеджер пакетов NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Установка через .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Установка через консоль диспетчера пакетов:
   ```
   PM> Install-Package Aspose.Slides
   ```

4. Добавьте ссылку на ваш проект:
   После установки добавьте ссылку на ваш проект:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

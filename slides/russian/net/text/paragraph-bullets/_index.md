# Создание пользовательских маркеров в презентациях PowerPoint

Aspose.Slides для .NET предоставляет надежные возможности для создания и управления презентациями PowerPoint программным способом. Этот пример специально фокусируется на реализации пользовательских маркеров в текстовых фреймах, показывая, как управлять типом маркера, цветом, отступом и размером. Код создает презентацию с прямоугольной формой, содержащую два абзаца — один с маркером-символом, а другой — с нумерованным стилем маркера.

## Инструкция по установке

1. Установите Aspose.Slides для .NET с помощью диспетчера пакетов NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Либо загрузите пакет напрямую:
   - Посетите [Страница загрузки Aspose.Slides](https://downloads.aspose.com/slides/net)
   - Загрузите последнюю версию
   - Извлеките и добавьте ссылку на Aspose.Slides.dll в свой проект.

3. Добавьте необходимый оператор using:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

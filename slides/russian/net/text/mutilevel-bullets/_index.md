# Создание многоуровневых маркированных списков в PowerPoint с помощью Aspose.Slides

Aspose.Slides для .NET предоставляет мощный API для создания и управления презентациями PowerPoint программным способом. Этот пример демонстрирует, как создать презентацию с многоуровневыми маркерами, показывая, как добавить автофигуру с текстовой рамкой и отформатировать абзацы с различными стилями маркеров и уровнями отступов. Вы можете легко создать иерархический контент с настраиваемыми символами маркеров для каждого уровня отступов, что идеально подходит для создания структурированных и профессиональных презентаций без необходимости установки Microsoft PowerPoint.

## Инструкция по установке

1. Установка через менеджер пакетов NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Установка через .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Установка через консоль диспетчера пакетов:
   ```
   PM> Install-Package Aspose.Slides
   ```

4. Ручная установка:
   - Загрузите последнюю версию с сайта [Aspose.Slides для .NET](https://releases.aspose.com/slides/net/)
   - Добавьте ссылку на Aspose.Slides.dll в свой проект

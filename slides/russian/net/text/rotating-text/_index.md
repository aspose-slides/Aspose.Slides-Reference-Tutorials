# Создание вертикального текста в PowerPoint с помощью Aspose.Slides

Aspose.Slides for .NET предоставляет мощные возможности для создания и управления презентациями PowerPoint программным способом. В этом примере показано, как создать презентацию, содержащую прямоугольную форму с вертикально повернутым текстом. Текст ориентирован под углом 270 градусов, что позволяет читать его снизу вверх. Этот метод полезен для создания боковых надписей, уникальных текстовых макетов или специализированных элементов дизайна в ваших презентациях.

## Инструкция по установке

1. Установить Aspose.Slides для .NET
   
   Установите пакет Aspose.Slides с помощью диспетчера пакетов NuGet:
   
   ```csharp
   Install-Package Aspose.Slides
   ```
   
   Или с помощью .NET CLI:
   
   ```csharp
   dotnet add package Aspose.Slides
   ```

2. Добавить ссылки
   
   Добавьте следующий оператор using в ваш файл C#:
   
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

Теперь, когда у нас все готово, давайте реализуем наши основные функции: сохранение презентаций и извлечение изображений.

### Сохранение файла презентации
**Обзор**
Сохранение презентации подразумевает запись измененных или вновь созданных слайдов на диск. Это необходимо для сохранения изменений, внесенных программным путем.

#### Шаг 1: Загрузите презентацию
Сначала загрузите существующий файл PowerPoint:
```csharp
Presentation presentation = new Presentation("input.pptx");
```
Ваша презентация загрузится в память и будет готова к изменению или сохранению.

#### Шаг 2: Сохраните презентацию
Далее сохраните его в указанном месте:
```csharp
presentation.Save("output.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
```
Убедитесь, что `YOUR_OUTPUT_DIRECTORY` заменяется на желаемый вами путь. Этот шаг записывает все изменения обратно на диск.

### Извлечение изображений из презентации
**Обзор**
Извлекайте изображения, встроенные в слайды, для использования в других приложениях или для анализа.

#### Шаг 1: Получите доступ к слайду
Повторите каждый слайд:
```csharp
foreach (ISlide slide in presentation.Slides)
{
    // Обработайте каждый слайд
}
```
Этот цикл обеспечивает вам доступ к отдельным слайдам и их компонентам.

#### Шаг 2: Извлечение изображений
Из каждого слайда извлеките изображения:
```csharp
int imageIndex = 0;
foreach (IPPImage img in slide.Images)
{
    using (FileStream fileStream = new FileStream($"image{imageIndex++}.png", FileMode.Create))
    {
        img.SystemImage.Save(fileStream, ImageFormat.Png);
    }
}
```
Этот код сохраняет каждое изображение на диск. `imageIndex` обеспечивает уникальные имена файлов для извлеченных изображений.

### Советы по устранению неполадок
- Убедитесь, что пути правильные и доступные.
- Обработка исключений при проблемах доступа к файлам.

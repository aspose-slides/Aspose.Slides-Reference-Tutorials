## Руководство по внедрению

Мы разобьем реализацию на две основные функции: извлечение слайдов макета и добавление слайдов с определенными макетами.

### Функция 1: Получить макет слайда по типу

#### Обзор

Эта функция позволяет вам получить макет слайда из коллекции мастер-слайдов на основе его типа. Это особенно полезно, когда вам нужно применить единообразное форматирование к разным слайдам в вашей презентации.

#### Пошаговая реализация

**Получить коллекцию макетов слайдов Мастер-слайда**

Начните с доступа к коллекции макетов слайдов мастер-слайда:
```csharp
IMasterLayoutSlideCollection layoutSlides = presentation.Masters[0].LayoutSlides;
```

**Попытка получить определенный тип макета слайда**

Использовать `GetByType` метод для извлечения определенных макетов, таких как `TitleAndObject` или `Title`.
```csharp
ILayoutSlide layoutSlide = layoutSlides.GetByType(SlideLayoutType.TitleAndObject) ?
                          layoutSlides.GetByType(SlideLayoutType.Title);
```

**Перебрать доступные макеты по имени**

Если нужный макет не найден, выполните перебор доступных макетов по имени:
```csharp
if (layoutSlide == null)
{
    foreach (ILayoutSlide titleAndObjectLayoutSlide in layoutSlides)
    {
        if (titleAndObjectLayoutSlide.Name == "Title and Object")
        {
            layoutSlide = titleAndObjectLayoutSlide;
            break;
        }
    }

    if (layoutSlide == null)
    {
        foreach (ILayoutSlide titleLayoutSlide in layoutSlides)
        {
            if (titleLayoutSlide.Name == "Title")
            {
                layoutSlide = titleLayoutSlide;
                break;
            }
        }

        // Возврат к пустому типу слайда или добавление нового макета слайда, если ничего не найдено
        if (layoutSlide == null)
        {
            layoutSlide = layoutSlides.GetByType(SlideLayoutType.Blank) ?
                          layoutSlides.Add(SlideLayoutType.TitleAndObject, "Title and Object");
        }
    }
}
```

**Советы по устранению неполадок:**
- Убедитесь, что файл презентации существует по указанному пути.
- Убедитесь, что ваш мастер-слайд содержит нужные макеты.

### Функция 2: Добавить слайд с макетом слайда

#### Обзор

Добавление нового слайда с использованием определенного макета может обеспечить согласованность всей презентации. Эта функция демонстрирует, как добиться этого эффективно.

#### Пошаговая реализация

**Получить или создать желаемый макет слайда**

Начните с получения или создания желаемого макета:
```csharp
ILayoutSlide layoutSlide = layoutSlides.GetByType(SlideLayoutType.TitleAndObject) ?
                           layoutSlides.GetByType(SlideLayoutType.Title);

if (layoutSlide == null)
{
    foreach (ILayoutSlide titleAndObjectLayoutSlide in layoutSlides)
    {
        if (titleAndObjectLayoutSlide.Name == "Title and Object")
        {
            layoutSlide = titleAndObjectLayoutSlide;
            break;
        }
    }

    if (layoutSlide == null)
    {
        foreach (ILayoutSlide titleLayoutSlide in layoutSlides)
        {
            if (titleLayoutSlide.Name == "Title")
            {
                layoutSlide = titleLayoutSlide;
                break;
            }
        }

        if (layoutSlide == null)
        {
            layoutSlide = layoutSlides.GetByType(SlideLayoutType.Blank) ?
                          layoutSlides.Add(SlideLayoutType.TitleAndObject, "Title and Object");
        }
    }
}
```

**Добавить новый слайд с выбранным макетом**

Вставьте пустой слайд в позицию 0, используя выбранный макет:
```csharp
presentation.Slides.InsertEmptySlide(0, layoutSlide);
```

**Советы по устранению неполадок:**
- Подтвердите, что `layoutSlide` не является нулевым перед вставкой.
- Проверьте, поддерживает ли ваша презентация предполагаемый тип макета.

**Что вы узнаете:**
- Настройка Aspose.Slides в среде .NET
- Настройка обычного состояния просмотра презентаций
- Регулировка горизонтальных и вертикальных разделительных планок
- Включение автоматической настройки для восстановленных видов
- Отображение значков контуров в презентации

## Руководство по внедрению
Вот как настроить нормальное состояние просмотра в несколько простых шагов:

### Настроить состояние горизонтальной полосы
Установите состояние горизонтальной полосы на восстановленное, свернутое или скрытое. Это определяет, как отображается панель слайдов при открытии.

#### Шаги:
1. **Создание объекта презентации:**
   ```csharp
   using Aspose.Slides;
   
   // Инициализировать новый экземпляр презентации
   Presentation pres = new Presentation();
   ```
2. **Установить состояние горизонтальной полосы:**
   ```csharp
   // Установить состояние горизонтальной полосы на восстановленное
   pres.ViewProperties.NormalViewProperties.HorizontalBarState = SplitterBarStateType.Restored;
   ```
   - **Почему?** Это гарантирует, что пользователи смогут видеть полный вид слайдов при открытии презентации.

### Настроить состояние вертикальной полосы
Вертикальная полоса облегчает навигацию по разделам или основным видам. Максимизация обеспечивает лучший контроль.

#### Шаги:
1. **Установить состояние вертикальной полосы:**
   ```csharp
   // Установить состояние вертикальной полосы в максимально развернутое состояние
   pres.ViewProperties.NormalViewProperties.VerticalBarState = SplitterBarStateType.Maximized;
   ```
   - **Почему?** Развернутая вертикальная панель обеспечивает обзор макетов слайдов, помогая лучше управлять презентацией.

### Включить автоматическую настройку для восстановленного вида сверху
Функция автоматической настройки обеспечивает адаптацию восстановленного вида к доступному пространству, повышая удобство чтения и удобство использования.

#### Шаги:
1. **Включить автоматическую настройку:**
   ```csharp
   // Включить автонастройку
   pres.ViewProperties.NormalViewProperties.RestoredTop.AutoAdjust = true;
   
   // Установите размер для лучшей видимости
   pres.ViewProperties.NormalViewProperties.RestoredTop.DimensionSize = 80;
   ```
   - **Почему?** Эта функция обеспечивает адаптивность презентации и эффективно адаптируется к экранам разных размеров.

### Отображение значков контура
Значки структуры помогают пользователям быстро определить структуру вашей презентации.

#### Шаги:
1. **Показать значки контура:**
   ```csharp
   // Включить отображение значков контуров
   pres.ViewProperties.NormalViewProperties.ShowOutlineIcons = true;
   ```
   - **Почему?** Эта визуальная подсказка помогает пользователям быстро понять иерархическую структуру содержимого вашей презентации.

### Сохранить настроенную презентацию
После настройки сохраните презентацию, чтобы сохранить эти настройки.

#### Шаги:
1. **Сохраните файл:**
   ```csharp
   string dataDir = "YOUR_DOCUMENT_DIRECTORY/";

   // Сохранить с указанным именем файла и форматом
   pres.Save(Path.Combine(dataDir, "presentation_normal_view_state.pptx"), SaveFormat.Pptx);
   ```

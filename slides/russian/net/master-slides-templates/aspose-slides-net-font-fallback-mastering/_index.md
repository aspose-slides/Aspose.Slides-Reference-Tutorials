## Руководство по внедрению

Мы разберем реализацию на ключевые функции, чтобы помочь вам понять и эффективно применить каждый аспект.

### Функция: Настройка и инициализация

Первый шаг — инициализация вашей среды. Эта настройка подготавливает Aspose.Slides к обработке шрифтов в презентациях.

```csharp
using Aspose.Slides;
using System.Collections.Generic;

string dataDir = @"YOUR_DOCUMENT_DIRECTORY";

IFontFallBackRulesCollection rulesList = new FontFallBackRulesCollection();
```

**Объяснение**: 
- `dataDir`: Указывает каталог для файлов презентации.
- `rulesList`: Объект для управления правилами резервного копирования шрифтов.

### Функция: Добавление и изменение правил резервного копирования шрифтов

Создание и настройка правил резервного копирования шрифтов гарантирует замену неподдерживаемых шрифтов альтернативными, сохраняя визуальную согласованность.

#### Шаг 1: Добавьте базовое правило
```csharp
rulesList.Add(new FontFallBackRule(0x400, 0x4FF, "Times New Roman"));
```

**Объяснение**: 
- Добавляет правило для символов в диапазоне `0x400` к `0x4FF` использовать шрифт «Times New Roman».

#### Шаг 2: Измените существующие правила
```csharp
foreach (IFontFallBackRule fallBackRule in rulesList)
{
    // Удалить «Tahoma» из резервных вариантов
    fallBackRule.Remove("Tahoma");

    // Добавьте «Verdana» для определенных диапазонов символов.
    if ((fallBackRule.RangeEndIndex >= 0x4000) && (fallBackRule.RangeStartIndex < 0x5000))
        fallBackRule.AddFallBackFonts("Verdana");
}
```

**Объяснение**: 
- Выполняет итерации по правилам для настройки резервных шрифтов, удаляя «Tahoma» и добавляя «Verdana» для определенных диапазонов.

#### Шаг 3: Удалить правило
```csharp
if (rulesList.Count > 0)
    rulesList.Remove(rulesList[0]);
```

**Объяснение**: 
- Безопасно удаляет первое правило, если оно существует, демонстрируя, как динамически управлять списком правил.

### Функция: Обработка презентаций с использованием правил резервного копирования шрифтов

Применение этих правил к презентации гарантирует, что все слайды будут отображаться с использованием правильных шрифтов.

```csharp
using (Presentation pres = new Presentation(dataDir + "input.pptx"))
{
    // Назначить правила резервного копирования шрифтов менеджеру шрифтов презентации
    pres.FontsManager.FontFallBackRulesCollection = rulesList;
    
    // Сделать рендеринг и сохранить первый слайд как изображение PNG
    pres.Slides[0].GetImage(1f, 1f).Save(dataDir + "Slide_0.png");
}
```

**Объяснение**: 
- Загружает презентацию и назначает `rulesList` в его менеджер шрифтов.
- Визуализирует первый слайд, используя указанные правила, и сохраняет его как изображение.

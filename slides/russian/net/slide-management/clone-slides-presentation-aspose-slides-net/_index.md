## Руководство по внедрению

Чтобы клонировать слайды между разделами одной презентации, выполните следующие действия.

### Создание и клонирование слайдов

**Обзор**Мы создадим слайд, поместим его в один раздел, а затем клонируем его в другой указанный раздел той же презентации.

#### Шаг 1: Инициализация презентации

Настройте свой экземпляр презентации с помощью:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Укажите путь к каталогу ваших документов здесь

using (IPresentation presentation = new Presentation()) {
    // Код для создания и клонирования слайда будет здесь
}
```

#### Шаг 2: Создайте начальный слайд

Добавьте форму к первому слайду:
```csharp
presentation.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 200, 50, 300, 100);
// Добавляет прямоугольную форму к первому слайду.
```

#### Шаг 3: Добавьте слайд в раздел

Свяжите начальный слайд с «Разделом 1»:
```csharp
presentation.Sections.AddSection("Section 1", presentation.Slides[0]);
// Связывает первый слайд с «Разделом 1»
```

#### Шаг 4: Добавьте пустой раздел

Создайте и добавьте новый раздел под названием «Раздел 2»:
```csharp
ISection section2 = presentation.Sections.AppendEmptySection("Section 2");
// Создает и добавляет пустой раздел с именем «Раздел 2».
```

#### Шаг 5: Клонировать слайд в определенный раздел

Клонируйте первый слайд в «Раздел 2»:
```csharp
presentation.Slides.AddClone(presentation.Slides[0], section2);
// Клонирует первый слайд и вставляет его в «Раздел 2».
```

### Сохранение презентации

Сохраните вашу презентацию в файл:
```csharp
presentation.Save(Path.Combine(dataDir, "CloneSlideIntoSpecifiedSection.pptx"), SaveFormat.Pptx);
// Сохраняет презентацию с внесенными изменениями
```

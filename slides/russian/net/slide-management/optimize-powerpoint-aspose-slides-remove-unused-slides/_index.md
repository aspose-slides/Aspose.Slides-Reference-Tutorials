### Удаление неиспользуемых мастер-слайдов

#### Обзор
Мастер-слайды помогают поддерживать единообразный вид на протяжении всей презентации, но могут стать излишними, если не используются. Эта функция автоматически удаляет все неиспользуемые мастер-слайды, оптимизируя размер файла и повышая производительность.

**Пошаговая реализация:**
1. **Загрузить файл презентации**
   - Убедитесь, что у вас есть путь к вашему файлу PPTX.
   
```csharp
using Aspose.Slides;
using System.IO;

string pptxFileName = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "MultipleMaster.pptx");
```

2. **Инициализация и загрузка презентации**

```csharp
// Создайте экземпляр класса Presentation для загрузки презентации.
using (Presentation pres = new Presentation(pptxFileName))
{
    // Далее мы удалим неиспользуемые мастер-слайды.
}
```

3. **Удалить неиспользуемые мастер-слайды**

```csharp
// Используйте функцию сжатия Aspose для оптимизации и удаления неиспользуемых мастеров.
Aspose.Slides.LowCode.Compress.RemoveUnusedMasterSlides(pres);
```

### Удаление неиспользуемых слайдов макета

#### Обзор
Подобно мастер-слайдам, макеты слайдов — это шаблоны, которые могут стать ненужными, если они не используются в презентации. Эффективное их удаление гарантирует, что ваш файл останется компактным.

**Пошаговая реализация:**
1. **Загрузить файл презентации**
   - Повторно используйте тот же путь к файлу и код инициализации из предыдущего раздела.

2. **Инициализация и загрузка презентации**

```csharp
// Повторная инициализация с использованием класса Aspose Presentation для повторного использования в различных операциях.
using (Presentation pres = new Presentation(pptxFileName))
{
    // Теперь мы сосредоточимся на удалении неиспользуемых слайдов макета.
}
```

3. **Удалить неиспользуемые слайды макета**

```csharp
// Используйте специальный метод для очистки и удаления неиспользуемых макетов.
Aspose.Slides.LowCode.Compress.RemoveUnusedLayoutSlides(pres);
```

**Советы по устранению неполадок:**
- Проверьте правильность путей к файлам.
- Перед выполнением операций убедитесь, что у вас имеется действующая лицензия.

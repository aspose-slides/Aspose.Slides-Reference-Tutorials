## Руководство по внедрению

### Добавление пользовательских свойств документа

В презентации можно добавлять пользовательские свойства для различных целей, например, для хранения пользовательских данных или метаданных проекта.

**1. Доступ к свойствам документа**

Начните с доступа к свойствам документа презентации:

```csharp
IDocumentProperties documentProperties = presentation.DocumentProperties;
```

**2. Добавление свойств**

Вот как можно добавить целочисленные и строковые свойства в документ:

```csharp
documentProperties["New Custom"] = 12; // Пример целочисленного свойства
documentProperties["My Name"] = "Mudassir"; // Пример свойства строки
documentProperties["Custom"] = 124; // Другое целочисленное свойство
```

**Объяснение**: `IDocumentProperties` Интерфейс позволяет управлять свойствами документа как парами ключ-значение, где ключи представляют собой строки.

### Получение пользовательских свойств документа

Извлечение пользовательских свойств подразумевает доступ к ним по индексу или имени:

```csharp
String getPropertyName = documentProperties.GetCustomPropertyName(2); // Получить имя третьего объекта недвижимости
```

**Объяснение**: `GetCustomPropertyName` Метод помогает получить имя свойства на основе его положения в коллекции.

### Удаление пользовательских свойств документа

Чтобы удалить пользовательское свойство, используйте его имя:

```csharp
documentProperties.RemoveCustomProperty(getPropertyName);
```

**Совет по устранению неполадок**: Убедитесь, что имя свойства получено правильно и существует, прежде чем пытаться удалить его.

### Сохранение изменений

Наконец, сохраните презентацию со всеми изменениями:

```csharp
presentation.Save("YOUR_OUTPUT_DIRECTORY/CustomDocumentProperties_out.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
```

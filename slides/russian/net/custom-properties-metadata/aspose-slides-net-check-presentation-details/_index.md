### Проверьте детали создания или изменения презентации

Эта функция позволяет извлекать метаданные о том, кто создал или последним изменил презентацию, включая название и версию приложения.

#### Обзор:
Вы извлечете информацию, хранящуюся в свойствах файла PPTX, с помощью Aspose.Slides. `PresentationFactory` класс. Это особенно полезно для целей аудита или поддержания согласованности документов в вашем рабочем процессе.

##### Шаг 1: Настройте каталог документов

Начните с определения пути к месту расположения вашего документа:
```csharp
// Определите путь к каталогу, убедившись, что он указывает на файл вашей презентации.
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
```

Заменять `"YOUR_DOCUMENT_DIRECTORY"` с фактическим путем к папке, содержащей ваш `props.pptx` файл.

##### Шаг 2: Загрузите презентацию

Объедините путь к каталогу и имя файла, чтобы найти вашу презентацию:
```csharp
// Объедините пути для доступа к «props.pptx» в вашем каталоге документов
string presentationPath = Path.Combine(dataDir, "props.pptx");
```

Гарантировать `props.pptx` существует в этом каталоге, прежде чем продолжить.

##### Шаг 3: Извлечение информации о презентации

Используйте `PresentationFactory` класс для сбора информации о презентации:
```csharp
// Доступ к сведениям о презентации с помощью Aspose.Slides
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo(presentationPath);
```

Этот шаг имеет решающее значение, поскольку он инициализирует процесс чтения свойств документа.

##### Шаг 4: Прочтите свойства документа

Извлеките необходимые свойства, такие как имя и версия приложения:
```csharp
// Извлечь свойства документа из презентации
documentProperties props = info.ReadDocumentProperties();

// Извлечь и сохранить имя приложения
string app = props.NameOfApplication;

// Извлеките и сохраните версию приложения, используемую для модификации.
string ver = props.AppVersion;
```

Эти шаги извлекают метаданные, которые можно регистрировать или отображать по мере необходимости.

#### Советы по устранению неполадок:
- Убедитесь, что пути к файлам указаны правильно, чтобы избежать `FileNotFoundException`.
- Проверьте разрешения на доступ к каталогу, если у вас возникли проблемы с доступом.
- Еще раз проверьте, что ваш пакет Aspose.Slides обновлен для совместимости с новыми версиями PPTX.

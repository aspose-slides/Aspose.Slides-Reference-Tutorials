### Функция обновления свойств презентации
Эта функция позволяет программно изменить автора и название презентации PowerPoint.

#### Шаг 1: Проверка существования файла
Прежде чем получить к нему доступ, убедитесь, что файл существует в указанном вами каталоге.
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";

if (File.Exists(dataDir + "/ModifyBuiltinProperties1.pptx")) {
    // Продолжить обновление свойств
} else {
    Console.WriteLine("The specified presentation file does not exist.");
}
```

#### Шаг 2: Получите информацию о презентации
Получить информацию о презентации с помощью `PresentationFactory`.
```csharp
IPresentationInfo info = PresentationFactory.Instance.GetPresentationInfo(dataDir + "/ModifyBuiltinProperties1.pptx");
```

#### Шаг 3: Чтение и обновление свойств документа
Получайте доступ к текущим свойствам и обновляйте их по мере необходимости.
```csharp
IDocumentProperties props = info.ReadDocumentProperties();
props.Author = "New Author";
props.Title = "New Title";
info.UpdateDocumentProperties(props);
```

#### Шаг 4: Сохраните изменения.
Сохраните изменения в файле.
```csharp
info.WriteBindedPresentation(dataDir + "/ModifyBuiltinProperties1.pptx");
```

### Советы по устранению неполадок:
- Убедитесь, что пути правильные и доступные.
- Корректная обработка исключений при операциях ввода-вывода файлов.

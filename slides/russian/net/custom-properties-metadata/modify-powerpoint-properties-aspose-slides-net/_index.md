## Руководство по внедрению

Настроив нашу среду, давайте изменим свойства презентации PowerPoint с помощью Aspose.Slides для .NET.

### Доступ к свойствам презентации

#### Обзор
Доступ к встроенным свойствам файла PowerPoint и их изменение:

```csharp
using System;
using Aspose.Slides;

// Определите каталоги ваших документов
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// Создайте экземпляр класса Presentation
Presentation presentation = new Presentation(dataDir + "/ModifyBuiltinProperties.pptx");

// Доступ к встроенным свойствам
IDocumentProperties documentProperties = presentation.DocumentProperties;
```

#### Объяснение
- **`dataDir`**: Путь к входному файлу PowerPoint.
- **`outputDir`**: Каталог, в котором будет сохранена измененная презентация.

### Изменение встроенных свойств
Установите различные свойства следующим образом:

**Автор:**
```csharp
documentProperties.Author = "Aspose.Slides for .NET";
```
- Устанавливает автора презентации.

**Заголовок:**
```csharp
documentProperties.Title = "Modifying Presentation Properties with Aspose.Slides";
```
- Обновляет название вашей презентации.

**Тема, комментарии и менеджер:**
```csharp
documentProperties.Subject = "Aspose Subject";
documentProperties.Comments = "Aspose Description";
documentProperties.Manager = "Aspose Manager";
```
- Эти свойства предоставляют дополнительные метаданные о документе.

### Сохранение изменений
Сохраните ваши изменения с помощью:

```csharp
presentation.Save(outputDir + "/DocumentProperties_out.pptx", SaveFormat.Pptx);
```

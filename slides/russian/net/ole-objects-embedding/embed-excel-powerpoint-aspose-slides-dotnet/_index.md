### Добавление и настройка фрейма объекта OLE

В этом разделе показано, как встроить документ Excel в виде значка в слайд PowerPoint.

#### Обзор
Внедрение объекта OLE позволяет вставлять сложные документы, такие как электронные таблицы или другие файлы, непосредственно в презентации, сохраняя их функциональность.

#### Этапы внедрения

**1. Подготовьте исходный файл**
Убедитесь, что у вас готов файл Excel `YOUR_DOCUMENT_DIRECTORY/ExcelObject.xlsx`.

**2. Прочитайте и вставьте файл**

```csharp
using Aspose.Slides;
using System.IO;

string oleSourceFile = "YOUR_DOCUMENT_DIRECTORY/ExcelObject.xlsx";
byte[] allbytes = File.ReadAllBytes(oleSourceFile);
IOleEmbeddedDataInfo dataInfo = new OleEmbeddedDataInfo(allbytes, "xlsx");

using (Presentation pres = new Presentation()) {
    ISlide slide = pres.Slides[0];
    IOleObjectFrame oof = slide.Shapes.AddOleObjectFrame(20, 20, 50, 50, dataInfo);
    
    // Установите OLE-объект для отображения в виде значка
    oof.IsObjectIcon = true;
}
```
- **Параметры:** `AddOleObjectFrame` принимает положение и размер кадра (x, y, ширина, высота) вместе с информацией о данных.
- **Цель:** Параметр `IsObjectIcon` к `true` обеспечивает отображение только значка, экономя место и сохраняя доступность контента.

### Добавление и настройка заменяющего изображения для рамки объекта OLE

Далее мы заменим значок Excel по умолчанию на пользовательское изображение.

#### Обзор
Настройка значков может сделать ваши презентации более визуально привлекательными и соответствующими принципам брендинга.

#### Этапы внедрения

**1. Подготовьте файл значка**
Убедитесь, что у вас есть файл изображения по адресу `YOUR_DOCUMENT_DIRECTORY/Image.png`.

**2. Встроить и заменить значок по умолчанию**

```csharp
using Aspose.Slides;
using System.IO;

string oleIconFile = "YOUR_DOCUMENT_DIRECTORY/Image.png";
byte[] imgBuf = File.ReadAllBytes(oleIconFile);

using (Presentation pres = new Presentation()) {
    using (MemoryStream ms = new MemoryStream(imgBuf)) {
        IPPImage image = pres.Images.AddImage(System.Drawing.Image.FromStream(ms));
        ISlide slide = pres.Slides[0];
        IOleObjectFrame oof = slide.Shapes.AddOleObjectFrame(20, 20, 50, 50, new OleEmbeddedDataInfo(imgBuf, "png"));
        
        // Заменить значок объекта OLE на пользовательское изображение
        oof.SubstitutePictureFormat.Picture.Image = image;
    }
}
```
- **Параметры:** `AddImage` метод добавляет изображение в коллекцию изображений презентации.
- **Цель:** Замена повышает визуальную привлекательность и обеспечивает лучший контекст с первого взгляда.

### Установка подписи для значка объекта OLE

Добавление подписей может пояснить, что представляет собой каждый значок на слайдах.

#### Обзор
Подписи имеют решающее значение при работе с несколькими значками, обеспечивая ясность, не загромождая слайд текстом.

#### Этапы внедрения

**1. Повторно используйте этап подготовки изображения**

```csharp
using Aspose.Slides;
using System.IO;

string oleIconFile = "YOUR_DOCUMENT_DIRECTORY/Image.png";
byte[] imgBuf = File.ReadAllBytes(oleIconFile);

using (Presentation pres = new Presentation()) {
    using (MemoryStream ms = new MemoryStream(imgBuf)) {
        IPPImage image = pres.Images.AddImage(System.Drawing.Image.FromStream(ms));
        ISlide slide = pres.Slides[0];
        IOleObjectFrame oof = slide.Shapes.AddOleObjectFrame(20, 20, 50, 50, new OleEmbeddedDataInfo(imgBuf, "png"));
        
        // Задайте текст подписи для значка OLE
        oof.SubstitutePictureTitle = "Caption example";
    }
}
```
- **Цель:** The `SubstitutePictureTitle` Свойство позволяет вам добавить описательную подпись непосредственно на значок.

## Руководство по внедрению

### Извлечение поддельных и настоящих гиперссылок из слайдов PowerPoint

В этом разделе мы рассмотрим шаги по извлечению как поддельных (отображаемых) URL-адресов, так и настоящих гиперссылок из определенной части текста на слайде.

#### Шаг 1: Загрузите презентацию

Начните с загрузки файла презентации:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
using (var presentation = new Presentation(dataDir + "/ExternalUrlOriginal.pptx"))
{
    // Продолжайте доступ к слайдам и фигурам.
}
```

#### Шаг 2: Доступ к текстовой части слайда

Определите часть текста, из которой вы хотите извлечь гиперссылки:
```csharp
var portion = ((AutoShape)presentation.Slides[0].Shapes[1]).TextFrame.Paragraphs[0].Portions[0];
```
Этот код обращается к первой фигуре на первом слайде, конкретно ориентируясь на ее текстовое содержимое.

#### Шаг 3: Извлечение поддельной гиперссылки

Получить поддельную (отображаемую) гиперссылку:
```csharp
string fakeHyperlink = portion.PortionFormat.AsIHyperlinkContainer.HyperlinkClick.ExternalUrl;
```

#### Шаг 4: Извлечение настоящей гиперссылки

Аналогичным образом извлеките реальный URL-адрес гиперссылки:
```csharp
string realHyperlink = portion.PortionFormat.AsIHyperlinkContainer.HyperlinkClick.ExternalUrlOriginal;
```

### Советы по устранению неполадок

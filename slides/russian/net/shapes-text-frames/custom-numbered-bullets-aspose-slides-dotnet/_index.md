## Руководство по внедрению
Вот как настроить пользовательские нумерованные маркеры на слайдах PowerPoint с помощью Aspose.Slides .NET.

### Добавление пользовательских пронумерованных маркеров на слайд
#### Шаг 1: Создайте новую презентацию и добавьте автофигуру
Создайте экземпляр презентации и добавьте к первому слайду прямоугольник в качестве текстового контейнера:
```csharp
var shape = presentation.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 200, 200, 400, 200);
```
#### Шаг 2: Доступ к текстовому фрейму
Доступ к `ITextFrame` созданной формы для управления текстовым содержимым:
```csharp
ITextFrame textFrame = shape.TextFrame;
```
#### Шаг 3: Настройте пронумерованные маркеры
Настройте маркеры, установив их начальные номера. Вот как это сделать для трех разных элементов списка:
1. **Первый пункт списка** с индивидуальным стартовым номером:
   ```csharp
   var paragraph1 = new Paragraph { Text = "bullet 2" };
   paragraph1.ParagraphFormat.Depth = 4; 
   paragraph1.ParagraphFormat.Bullet.NumberedBulletStartWith = 2;
   paragraph1.ParagraphFormat.Bullet.Type = BulletType.Numbered;
   textFrame.Paragraphs.Add(paragraph1);
   ```
2. **Второй пункт списка** с другим стартовым номером:
   ```csharp
   var paragraph2 = new Paragraph { Text = "bullet 3" };
   paragraph2.ParagraphFormat.Depth = 4;
   paragraph2.ParagraphFormat.Bullet.NumberedBulletStartWith = 3; 
   paragraph2.ParagraphFormat.Bullet.Type = BulletType.Numbered;
   textFrame.Paragraphs.Add(paragraph2);
   ```
3. **Третий пункт списка** с другим пользовательским номером:
   ```csharp
   var paragraph5 = new Paragraph { Text = "bullet 7" };
   paragraph5.ParagraphFormat.Depth = 4;
   paragraph5.ParagraphFormat.Bullet.NumberedBulletStartWith = 7;
   paragraph5.ParagraphFormat.Bullet.Type = BulletType.Numbered;
   textFrame.Paragraphs.Add(paragraph5);
   ```
#### Шаг 4: Сохраните презентацию
Сохраните презентацию в указанном каталоге:
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY"; // Замените на ваш реальный путь
presentation.Save(Path.Combine(outputDir, "SetCustomBulletsNumber-slides.pptx"), SaveFormat.Pptx);
```
### Советы по устранению неполадок
- Убедитесь, что библиотека Aspose.Slides указана правильно.
- Проверьте права на запись для сохранения файлов в указанном каталоге.

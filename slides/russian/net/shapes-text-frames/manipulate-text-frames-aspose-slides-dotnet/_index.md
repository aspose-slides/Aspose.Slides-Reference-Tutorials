### Загрузка и управление текстовыми фреймами презентации
#### Обзор
Мы покажем, как загрузить файл PowerPoint и настроить `KeepTextFlat` свойство в текстовых рамках. Это свойство влияет на то, остается ли текст плоским или сохраняет исходное форматирование при экспорте или печати.
#### Пошаговая реализация
**1. Настройка вашей среды**
Сначала определите каталог документов, в котором будут находиться файлы презентации:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string pptxFileName = Path.Combine(dataDir, "KeepTextFlat.pptx");
```
**2. Загрузка презентации**
Используйте Aspose.Slides для открытия файла PowerPoint:
```csharp
using (Presentation pres = new Presentation(pptxFileName))
{
    // Доступ к фигурам на первом слайде
    var shape1 = pres.Slides[0].Shapes[0] as AutoShape;
    var shape2 = pres.Slides[0].Shapes[1] as AutoShape;

    // Манипулировать свойствами текстового фрейма
}
```
**3. Настройка свойств текстового фрейма**
Отрегулируйте `KeepTextFlat` свойство для различных форм:
```csharp
// Установите для параметра «Сохранить текст плоским» значение «Ложь» для формы 1
shape1.TextFrame.TextFrameFormat.KeepTextFlat = false;

// Установите для параметра «Сохранить текст плоским» значение «true» для фигуры 2
shape2.TextFrame.TextFrameFormat.KeepTextFlat = true;
```
**Объяснение:**
- **Почему `KeepTextFlat`?** Это свойство определяет, следует ли выравнивать текст, что может помочь уменьшить размер файла и обеспечить единообразное форматирование на разных устройствах.

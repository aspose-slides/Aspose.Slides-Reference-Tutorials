### Создание и форматирование текстового поля
#### Шаг 1: Настройте среду вашего проекта
Во-первых, убедитесь, что вы настроили каталог своего проекта:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
bool isExists = Directory.Exists(dataDir);
if (!isExists)
    Directory.CreateDirectory(dataDir);
```

Этот фрагмент проверяет, существует ли указанный каталог, и создает его, если нет. Это гарантирует, что ваши файлы презентации будут сохранены в правильном месте.

#### Шаг 2: Создание объекта презентации
```csharp
class Program
{
    static void Main()
    {
        using (Presentation pres = new Presentation())
        {
            ISlide sld = pres.Slides[0]; // Доступ к первому слайду
```
Здесь мы создаем экземпляр `Presentation` объект и доступ к его первому слайду. Все манипуляции производятся на этом слайде.

#### Шаг 3: Добавьте автофигуру с внутренней тенью
```csharp
// Добавление прямоугольника с положением (150, 75) и размером (150x50)
IAutoShape ashp = sld.Shapes.AddAutoShape(ShapeType.Rectangle, 150, 75, 150, 50);

// Добавление текста в форму
txtFrame = ashp.TextFrame;
para = txtFrame.Paragraphs[0];
portion = para.Portions[0];

// Установка текста части
portion.Text = "Aspose TextBox";
```
Этот раздел добавляет прямоугольную форму к вашему слайду и настраивает ее с пустой текстовой рамкой. Позже вы можете применить к этой форме эффекты, такие как внутренняя тень.

#### Шаг 4: Примените эффект внутренней тени
Чтобы добавить внутреннюю тень, обычно нужно изменить `ashp` Свойства стиля объекта. Однако Aspose.Slides для .NET на момент написания статьи не поддерживает внутреннюю тень напрямую через встроенные методы, поэтому вам может потребоваться использовать обходные приемы или дополнительные библиотеки, которые предлагают более продвинутые графические манипуляции.

## Руководство по внедрению

Теперь, когда вы все настроили, давайте перейдем к реализации функции замены текста.

### Обзор функции: замена текста в слайдах PowerPoint

Эта функция ищет определенный текст-заполнитель (например, "[этот блок]") и заменяет его желаемым содержимым на всех слайдах. Это особенно полезно при обновлении общих фраз или названий продуктов в презентации.

#### Шаг 1: Загрузите презентацию
Начните с загрузки презентации, в которой вы хотите заменить текст:

```csharp
Presentation pres = new Presentation("example.pptx");
```

#### Шаг 2: Определите параметры замены текста

Определите заполнитель и текст замены. Например, замените "[этот блок]" на "мой текст":

```csharp
string strToFind = "[this block]";
string strToReplaceWith = "my text";
```

#### Шаг 3: Перебор слайдов и замена текста

Просмотрите каждый слайд презентации, чтобы найти и заменить текст-заполнитель:

```csharp
foreach (ISlide slide in pres.Slides)
{
    foreach (IAutoShape shape in slide.Shapes.OfType<IAutoShape>())
    {
        if (shape.TextFrame != null)
        {
            ITextFrame textFrame = shape.TextFrame;
            foreach (IParagraph para in textFrame.Paragraphs)
            {
                foreach (Portion portion in para.Portions)
                {
                    if (portion.Text.Contains(strToFind))
                    {
                        // Заменить текст
                        portion.Text = portion.Text.Replace(strToFind, strToReplaceWith);
                    }
                }
            }
        }
    }
}
```

#### Объяснение:
- **Параметры**: `strToFind` — это текст-заполнитель, на который вы ориентируетесь. `strToReplaceWith` это то, что вы хотите заменить.
- **Метод Цель**: Метод перебирает фигуры каждого слайда, ищет текстовые фреймы с указанным заполнителем и заменяет его.

### Советы по устранению неполадок

- Убедитесь, что ваши текстовые строковые переменные (`strToFind` и `strToReplaceWith`) определены правильно.
- Проверьте, содержат ли слайды ожидаемый формат (например, автофигуры), чтобы избежать исключений нулевых ссылок.

## Раздел часто задаваемых вопросов

### Распространенные вопросы:
1. **Как учитывать регистр при замене текста?**
   - Aspose.Slides по умолчанию выполняет поиск с учетом регистра, но вы можете изменить логику, чтобы игнорировать регистр.
2. **Можно ли заменить текст в нескольких презентациях одновременно?**
   - Да, пройдитесь по файлам презентации в цикле и примените ту же логику.
3. **Что делать, если мой заполнитель является частью другого слова?**
   - Измените критерии поиска или используйте регулярные выражения для более точного соответствия.
4. **Есть ли поддержка замены изображений вместо текста?**
   - Хотя в этом руководстве основное внимание уделяется тексту, Aspose.Slides также предлагает API для управления и замены изображений в презентациях.
5. **Как работать со слайдами без заполнителей?**
   - Убедитесь, что ваша логика включает проверку наличия заполнителей, прежде чем пытаться их заменить.

### Клонирование фигур между слайдами

#### Обзор
Эта функция позволяет дублировать определенные фигуры с одного слайда и размещать их на другом, либо в указанных координатах, либо в месте размещения по умолчанию.

#### Пошаговая реализация

**Настройте свою презентацию**

Начните с определения пути к документу и загрузки презентации:

```csharp
string dataDir = @"YOUR_DOCUMENT_DIRECTORY";
using (Presentation srcPres = new Presentation(dataDir + "Source Frame.pptx"))
{
    // Продолжить операции по клонированию
}
```

**Доступ к коллекциям фигур**

Извлеките коллекции фигур из исходного и целевого слайдов:

```csharp
// Получите коллекцию фигур с первого слайда
IShapeCollection sourceShapes = srcPres.Slides[0].Shapes;

// Получите пустой макет слайда, чтобы создать новый слайд без контента.
ILayoutSlide blankLayout = srcPres.Masters[0].LayoutSlides.GetByType(SlideLayoutType.Blank);

// Добавьте пустой слайд, используя пустой макет.
ISlide destSlide = srcPres.Slides.AddEmptySlide(blankLayout);
IShapeCollection destShapes = destSlide.Shapes;
```

**Клонировать фигуры с указанными координатами**

Клонируйте определенную фигуру и разместите ее в нужных координатах на целевом слайде:

```csharp
// Клонировать фигуру в указанные координаты на целевом слайде
destShapes.AddClone(sourceShapes[1], 50, 150 + sourceShapes[0].Height);
```

**Клонировать форму без новой позиции**

Вы также можете клонировать фигуры без указания новых координат. Они будут добавлены последовательно:

```csharp
// Клонировать другую фигуру в положение по умолчанию на целевом слайде
destShapes.AddClone(sourceShapes[2]);
```

**Вставить клонированную форму в указанном индексе**

Вставьте клонированную фигуру в начало коллекции фигур целевого слайда:

```csharp
// Вставить клонированную фигуру в индекс 0 с указанными координатами
destShapes.InsertClone(0, sourceShapes[0], 50, 150);
```

### Сохранение презентации

Наконец, сохраните измененную презентацию на диск:

```csharp
srcPres.Save(dataDir + "CloneShape_out.pptx", SaveFormat.Pptx);
```

#### Советы по устранению неполадок
- Убедитесь, что пути для загрузки и сохранения файлов указаны правильно.
- Убедитесь, что индексы, используемые в коллекциях фигур, существуют на исходном слайде.

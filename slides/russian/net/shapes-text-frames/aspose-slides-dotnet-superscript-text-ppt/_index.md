### Добавление надстрочного текста
#### Обзор
Улучшите читабельность с помощью надстрочного текста для формул, аннотаций или цитат:

1. **Доступ к слайду**: Загрузите слайд, на который вы хотите добавить текст.
2. **Создание формы**: Добавьте фигуру (например, прямоугольник) для размещения текста.
3. **Настройка текстового фрейма**: Настройте текстовую рамку и очистите существующие абзацы.
4. **Добавление надстрочной части**: Вставьте часть текста, которая должна быть надстрочным индексом.

#### Пошаговая реализация
**1. Доступ к слайду**
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
using (Presentation presentation = new Presentation())
{
    ISlide slide = presentation.Slides[0];
```
Загрузите существующую презентацию и откройте ее первый слайд.

**2. Создание фигуры**
```csharp
IAutoShape shape = slide.Shapes.AddAutoShape(ShapeType.Rectangle, 100, 100, 200, 100);
ITextFrame textFrame = shape.TextFrame;
```
Добавьте к слайду прямоугольную фигуру и подготовьте ее для ввода текста.

**3. Настройка текстового фрейма**
```csharp
textFrame.Paragraphs.Clear();
IParagraph superPar = new Paragraph();
```
Очистите существующие абзацы, чтобы начать все заново, а затем создайте новый абзац для надстрочного текста.

**4. Добавление надстрочной части**
Чтобы добавить верхний индекс:
- Создайте обычные и надстрочные части.
- Установите `PortionFormat.FontHeight` и другие свойства по мере необходимости.

```csharp
IPortion portion1 = new Portion { Text = "Slide Title" };
portion1.PortionFormat.FontHeight = 20;

// Надстрочный текст
IPortion portion2 = new Portion { Text = "Superscript Example" };
portion2.PortionFormat.FontHeight = 10;
portion2.ParagraphFormat.DefaultPortionFormat.FontBold = NullableBool.True;
portion2.TextFrame.Paragraphs[0].Portions[1].PortionFormat.Superscript = new Superscript() 
{ 
    FontSize = 50 %, 
    Position = SuperscriptPosition.VerticallyAboveBaseline
};

superPar.Portions.Add(portion1);
superPar.Portions.Add(portion2);
textFrame.Paragraphs.Add(superPar);
```
**Советы по устранению неполадок**:
- Гарантировать `PortionFormat.Superscript` установлен правильно с соответствующим размером шрифта и положением.
- Убедитесь, что части добавляются в абзацы в правильном порядке.

## Руководство по внедрению
### Создание прямоугольной формы
Наша цель — создать прямоугольник на первом слайде. Давайте разберем шаги:
#### Шаг 1: Инициализация презентации
Начните с настройки среды с помощью Aspose.Slides и создания нового объекта презентации.
```csharp
using System;
using Aspose.Slides;
using System.Drawing;

string dataDir = "YOUR_DOCUMENT_DIRECTORY";
bool isExists = System.IO.Directory.Exists(dataDir);
if (!isExists)
    System.IO.Directory.CreateDirectory(dataDir);

using (Presentation pres = new Presentation())
{
    // Код продолжается...
}
```
*Объяснение:* Этот код инициализирует новую презентацию PowerPoint и обеспечивает существование каталога для сохранения файлов.
#### Шаг 2: Получите доступ к первому слайду
Откройте первый слайд, куда мы добавим наш прямоугольник.
```csharp
ISlide sld = pres.Slides[0];
```
*Объяснение:* Извлекаем первый слайд из презентации для работы.
#### Шаг 3: Добавьте прямоугольную форму.
Добавьте на слайд автофигуру типа прямоугольник.
```csharp
IShape shp = sld.Shapes.AddAutoShape(ShapeType.Rectangle, 50, 150, 150, 50);
```
*Объяснение:* Это создает прямоугольник в позиции (50, 150) с размерами 150x50. Параметры определяют тип фигуры и ее местоположение/размер.
### Форматирование прямоугольника
Теперь, когда у нас есть прямоугольник, давайте применим к нему стили.
#### Шаг 4: Примените сплошную заливку цветом
Установите сплошной цвет заливки для тела прямоугольника.
```csharp
shp.FillFormat.FillType = FillType.Solid;
shp.FillFormat.SolidFillColor.Color = Color.Chocolate;
```
*Объяснение:* Здесь мы меняем внутреннюю часть прямоугольника на шоколадно-коричневый цвет.
#### Шаг 5: Примените форматирование границ
Настройте границу со сплошной заливкой и отрегулируйте ее ширину.
```csharp
shp.LineFormat.FillFormat.FillType = FillType.Solid;
shp.LineFormat.FillFormat.SolidFillColor.Color = Color.Black;
shp.LineFormat.Width = 5;
```
*Объяснение:* Граница прямоугольника имеет черный цвет, толщина линии — 5 пикселей.
### Сохранение презентации
Наконец, сохраните изменения в файле.
```csharp
pres.Save(dataDir + "/RectShp2_out.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
```
*Объяснение:* Это сохранит презентацию с новой отформатированной прямоугольной формой в указанном вами каталоге.

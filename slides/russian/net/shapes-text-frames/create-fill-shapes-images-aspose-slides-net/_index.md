## Руководство по внедрению

В этом разделе процесс разбивается на управляемые функции.

### Создание каталогов

Чтобы убедиться, что наши файлы презентации сохранены правильно, мы сначала проверяем, существует ли целевой каталог. Если нет, мы его создаем:
```csharp
using System.IO;

string dataDir = "YOUR_DOCUMENT_DIRECTORY";
bool isExists = Directory.Exists(dataDir);
if (!isExists)
{
    // Создайте каталог, если он не существует
    Directory.CreateDirectory(dataDir);
}
```

### Работа с презентациями

Начнем с создания экземпляра презентации, а затем будем манипулировать ее слайдами:
```csharp
using Aspose.Slides;

// Создать экземпляр класса Presentation, представляющего файл PPTX
using (Presentation pres = new Presentation())
{
    // Получить первый слайд из презентации
    ISlide sld = pres.Slides[0];

    // Добавить на слайд автофигуру прямоугольного типа
    IShape shp = sld.Shapes.AddAutoShape(ShapeType.Rectangle, 50, 150, 75, 150);
}
```

### Установка заливки фигуры картинкой

Далее мы заполняем фигуру изображением, задавая ее тип заливки:
```csharp
using Aspose.Slides;
using System.Drawing;

// Установите тип заливки фигуры на Изображение.
shp.FillFormat.FillType = FillType.Picture;
// Настройте режим заливки изображения как «Плитка»
shp.FillFormat.PictureFillFormat.PictureFillMode = PictureFillMode.Tile;

// Загрузите изображение из указанного каталога и установите его в формате заливки фигуры.
IImage img = Images.FromFile("YOUR_DOCUMENT_DIRECTORY/Tulips.jpg");
IPPImage imgx = pres.Images.AddImage(img);
shp.FillFormat.PictureFillFormat.Picture.Image = imgx;
```

### Сохранение презентаций

Наконец, сохраните презентацию со всеми изменениями:
```csharp
using Aspose.Slides.Export;

// Сохраните измененную презентацию обратно на диск.
pres.Save("YOUR_OUTPUT_DIRECTORY/RectShpPic_out.pptx", SaveFormat.Pptx);
```

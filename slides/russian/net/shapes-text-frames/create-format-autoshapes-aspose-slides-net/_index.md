## Руководство по внедрению

### Функция 1: Создание и добавление автофигуры на слайд

#### Обзор

В этом разделе показано, как создать презентацию, получить доступ к слайду и добавить автофигуру типа «Прямоугольник».

#### Шаги:

**Шаг 1**Инициализация презентации
```csharp
// Создать экземпляр класса Presentation
tPresentation presentation = new tPresentation();
```

**Шаг 2**: Доступ к первому слайду
```csharp
// Доступ к первому слайду
tISlide slide = presentation.Slides[0];
```

**Шаг 3**: Добавить прямоугольную автофигуру
```csharp
// Добавьте автофигуру типа «Прямоугольник» в положение (150, 75) с размером (350, 350)
tIAutoShape ashp = slide.Shapes.AddAutoShape(tShapeType.Rectangle, 150, 75, 350, 350);
```

**Шаг 4**: Сохранить презентацию
```csharp
// Сохраните презентацию в указанном каталоге presentation.Save("YOUR_OUTPUT_DIRECTORY/formatText_out.pptx", tSaveFormat.Pptx);
```

### Функция 2: Добавление и форматирование текстового фрейма в AutoShape

#### Обзор

В этой статье объясняется, как добавить TextFrame к существующей автофигуре, настроить параметры автоподбора и задать свойства текста.

#### Шаги:

**Шаг 1**: Добавить текстовый фрейм
```csharp
// Предположим, что «ashp» — это экземпляр IAutoShape из предыдущей операции.
// Добавить TextFrame к прямоугольнику
tashp.AddTextFrame(" ");
```

**Шаг 2**: Настроить тип автоподбора
```csharp
// Установите тип автоподбора для лучшего выравнивания текста внутри фигуры
tITextFrame txtFrame = ashp.TextFrame;
txtFrame.TextFrameFormat.AutofitType = tTextAutofitType.Shape;
```

**Шаг 3**: Форматирование и вставка текста
```csharp
// Создайте объект «Абзац» и задайте его содержимое.
tIParagraph para = txtFrame.Paragraphs[0];
tIPortion portion = para.Portions[0];

portion.Text = "A quick brown fox jumps over the lazy dog. A quick brown fox jumps over the lazy dog.";
portion.PortionFormat.FillFormat.FillType = tFillType.Solid;
portion.PortionFormat.FillFormat.SolidFillColor.Color = tColor.Black;
```

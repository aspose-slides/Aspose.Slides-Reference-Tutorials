## Руководство по внедрению

Чтобы создать слайд PowerPoint с вертикально повернутым текстом, выполните следующие действия:

### Шаг 1: Настройте каталог хранения документов
Определите, где будут храниться ваши презентации:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
```

Этот путь имеет решающее значение для сохранения и доступа к файлам презентации.

### Шаг 2: Создайте новую презентацию
Инициализируйте `Presentation` класс для создания нового файла PowerPoint:

```csharp
Presentation presentation = new Presentation();
```

The `Presentation` объект выступает в качестве контейнера для всех слайдов и контента.

### Шаг 3: Откройте первый слайд
Извлеките первый слайд из вашей презентации:

```csharp
ISlide slide = presentation.Slides[0];
```

Этот шаг гарантирует, что у нас будет слайд для добавления повернутого текста.

### Шаг 4: Добавьте автофигуру для текста
Добавьте прямоугольник, в который будет помещен текст:

```csharp
IAutoShape ashp = slide.Shapes.AddAutoShape(ShapeType.Rectangle, 150, 75, 350, 350);
```

Здесь, `ShapeType.Rectangle` выбран из-за его универсальности в плане размещения текста.

### Шаг 5: Настройка TextFrame и поворота
Добавьте текстовую рамку к фигуре и задайте поворот:

```csharp
ashp.AddTextFrame(" ");
ashp.FillFormat.FillType = FillType.NoFill;
ITextFrame txtFrame = ashp.TextFrame;
txtFrame.TextFrameFormat.TextVerticalType = TextVerticalType.Vertical270;
```

The `TextVerticalType` свойство определяет ориентацию текста внутри фрейма.

### Шаг 6: Добавьте и отформатируйте текст
Вставьте абзац с форматированным текстом в текстовую рамку:

```csharp
IParagraph para = txtFrame.Paragraphs[0];
IPortion portion = para.Portions[0];
portion.Text = "A quick brown fox jumps over the lazy dog. A quick brown fox jumps over the lazy dog.";
portion.PortionFormat.FillFormat.FillType = FillType.Solid;
portion.PortionFormat.FillFormat.SolidFillColor.Color = Color.Black;
```

Этот фрагмент добавляет текстовое содержимое и задает его черный цвет для лучшей видимости.

### Шаг 7: Сохраните презентацию
Наконец, сохраните презентацию с повернутым текстом:

```csharp
presentation.Save(dataDir + "RotateText_out.pptx", SaveFormat.Pptx);
```

Файл будет сохранен в указанном каталоге как файл PowerPoint.

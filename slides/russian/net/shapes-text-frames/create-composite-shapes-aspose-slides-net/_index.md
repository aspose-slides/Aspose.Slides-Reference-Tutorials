### Создание составных фигур из геометрических контуров
#### Обзор
В этом разделе показано, как создать пользовательскую форму, объединив два геометрических контура. Этот метод полезен для проектирования сложных элементов слайда или логотипов.
#### Шаг 1: Определите путь к выходному файлу
Сначала задайте путь к выходному файлу, используя структуру каталогов:
```csharp
string resultPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "CompositeShape.pptx");
```
#### Шаг 2: Инициализация объекта презентации
Начните с создания объекта презентации, в котором вы спроектируете свою составную форму:
```csharp
using (Presentation pres = new Presentation())
{
    // Реализация продолжается...
}
```
#### Шаг 3: Создание геометрических контуров
Определите два геометрических пути следующим образом:
```csharp
// Определите первый путь
IAutoShape shape1 = pres.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 50, 150, 200, 100);
shape1.FillFormat.FillType = FillType.NoFill;

// Определите второй путь (например, эллипс)
IAutoShape shape2 = pres.Slides[0].Shapes.AddAutoShape(ShapeType.Ellipse, 300, 150, 200, 100);
shape2.FillFormat.FillType = FillType.Solid;
shape2.FillFormat.SolidFillColor.Color = Color.Blue;
```
#### Шаг 4: Объедините контуры в составную форму
Используйте `Combine` метод объединения этих путей:
```csharp
// Путь доступа к коллекции shape1
IGeometryShape geoShape1 = (GeometryShape)shape1.Shape;
IPathCollection pathCollection1 = geoShape1.Path;

// Путь доступа к коллекции shape2
IGeometryShape geoShape2 = (GeometryShape)shape2.Shape;
IPathCollection pathCollection2 = geoShape2.Path;

// Объединить пути в один
pathCollection1.Add(pathCollection2[0]);
```
#### Шаг 5: Сохраните презентацию
Наконец, сохраните вашу презентацию в файл:
```csharp
pres.Save(resultPath, Aspose.Slides.Export.SaveFormat.Pptx);
```

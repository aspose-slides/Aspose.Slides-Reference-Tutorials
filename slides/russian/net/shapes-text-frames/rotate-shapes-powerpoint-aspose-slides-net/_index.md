## Руководство по внедрению

### Вращение фигур в PowerPoint

В этом разделе вы узнаете, как вращать автофигуру на слайде, чтобы добавить визуального интереса и подчеркнуть определенные части содержимого.

#### Шаг 1: Подготовьте окружающую среду

Определите каталог для сохранения документов:
```csharp
string dataDir = @"YOUR_DOCUMENT_DIRECTORY";
bool isExists = System.IO.Directory.Exists(dataDir);
if (!isExists)
    System.IO.Directory.CreateDirectory(dataDir);
```
Это гарантирует существование выходного каталога и предотвращает ошибки при сохранении файла.

#### Шаг 2: Создайте новую презентацию

Инициализируйте и получите доступ к первому слайду:
```csharp
using (Presentation pres = new Presentation())
{
    // Доступ к первому слайду
    ISlide sld = pres.Slides[0];
```
Создайте экземпляр презентации и откройте ее первый слайд, чтобы добавить свою форму.

#### Шаг 3: Добавьте и поверните автофигуру

Добавьте прямоугольную форму и поверните ее на 90 градусов:
```csharp
// Добавить прямоугольную автофигуру
IShape shp = sld.Shapes.AddAutoShape(ShapeType.Rectangle, 50, 150, 75, 150);

// Повернуть прямоугольник на 90 градусов
shp.Rotation = 90;
```
The `AddAutoShape` Метод помещает фигуру в указанные координаты и размеры. `Rotation` свойство регулирует свой угол.

#### Шаг 4: Сохраните презентацию

Сохраните вашу презентацию:
```csharp
// Сохраните измененную презентацию
pres.Save(dataDir + "RectShpRot_out.pptx");
}
```
Это запишет ваши изменения в файл в указанном каталоге.

### Советы по устранению неполадок
- **Отсутствующие библиотеки:** Убедитесь, что все зависимости установлены правильно.
- **Проблемы с путем к файлу:** Убедитесь, что `dataDir` установлен на доступный путь в вашей системе.
- **Ошибки поворота формы:** Проверьте значения параметров размеров формы и угла поворота.

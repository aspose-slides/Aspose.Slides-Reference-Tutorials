## Руководство по внедрению

### Функция 1: Простое создание прямоугольника в PowerPoint (H2)

Автоматизируйте добавление прямоугольных фигур, чтобы сэкономить время и обеспечить единообразие в презентациях. Вот как добавить прямоугольник с помощью Aspose.Slides для .NET.

#### Пошаговая реализация (H3)

1. **Инициализировать класс представления**
   
   Создайте экземпляр `Presentation` класс для представления вашего файла PowerPoint:

   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;

   string YOUR_DOCUMENT_DIRECTORY = "YOUR_DOCUMENT_DIRECTORY";

   using (Presentation pres = new Presentation())
   {
       // Продолжение кода здесь...
   }
   ```

2. **Доступ к первому слайду**

   Извлеките первый слайд из вашей презентации:

   ```csharp
   ISlide sld = pres.Slides[0];
   ```

3. **Добавить прямоугольную форму**

   Использовать `AddAutoShape` чтобы добавить прямоугольник в указанных позициях и размерах:

   ```csharp
   sld.Shapes.AddAutoShape(ShapeType.Rectangle, 50, 150, 150, 50);
   ```
   
   - **Параметры**: Метод принимает `ShapeType`, x-позиция, y-позиция, ширина и высота для определения размещения и размера фигуры.

4. **Сохранить презентацию**

   Сохраните презентацию, чтобы сохранить все изменения:

   ```csharp
   pres.Save(YOUR_DOCUMENT_DIRECTORY + "/RectShp1_out.pptx", SaveFormat.Pptx);
   ```

#### Советы по устранению неполадок

- Гарантировать `YOUR_DOCUMENT_DIRECTORY` пути установлены правильно.
- Убедитесь, что Aspose.Slides правильно указан в вашем проекте.

### Функция 2: Создание и проверка каталога (H2)

Эффективное управление каталогами предотвращает ошибки при сохранении файлов. Реализуйте эту проверку, чтобы убедиться, что каталоги существуют, прежде чем пытаться сохранить файл.

#### Пошаговая реализация (H3)

1. **Определить путь к каталогу**

   Укажите, где будут храниться ваши документы:

   ```csharp
   string dataDir = YOUR_DOCUMENT_DIRECTORY;
   ```

2. **Проверьте и создайте каталог, если необходимо**

   Использовать `Directory.Exists` для проверки существования каталога и создания его при необходимости:

   ```csharp
   bool isExists = Directory.Exists(dataDir);
   if (!isExists)
   {
       Directory.CreateDirectory(dataDir);
   }
   ```

#### Советы по устранению неполадок

- Убедитесь, что ваше приложение имеет разрешение на создание каталогов по указанному пути.
- Обрабатывайте исключения из-за недопустимых путей или недостаточных разрешений.

## Руководство по внедрению: добавление прямоугольной формы с заливкой изображением
Теперь, когда наша среда готова, давайте реализуем функцию добавления прямоугольной формы, заполненной изображением.

### Обзор функции
Эта функция демонстрирует, как создать прямоугольную форму на слайде и заполнить ее изображением с помощью Aspose.Slides. Этот метод можно использовать для улучшения слайдов путем добавления логотипов, фонов или любых графических элементов, которые сделают вашу презентацию более привлекательной.

### Пошаговая реализация
#### 1. Инициализируйте объект презентации
Начнем с создания нового объекта презентации. Он будет служить нам рабочим документом, в который мы будем добавлять фигуры и другие элементы.
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Укажите путь к каталогу ваших документов
total slides count: pres.Slides.Count;
using (Presentation pres = new Presentation())
{
    ISlide firstSlide = pres.Slides[0]; // Доступ к первому слайду

    // Загрузите изображение для использования в качестве заливки
    IPPImage ppImage;
    using (IImage newImage = Aspose.Slides.Images.FromFile(Path.Combine(dataDir, "image.png")))
        ppImage = pres.Images.AddImage(newImage); // Добавить изображение в коллекцию изображений презентации

    // Добавляет прямоугольную форму с указанными размерами.
    var newShape = firstSlide.Shapes.AddAutoShape(ShapeType.Rectangle, 0, 0, 350, 350);

    // Установите тип заливки фигуры на Изображение
    newShape.FillFormat.FillType = FillType.Picture;
    IPictureFillFormat pictureFillFormat = newShape.FillFormat.PictureFillFormat;
    pictureFillFormat.Picture.Image = ppImage; // Назначить загруженное изображение в качестве заливки для прямоугольника

    // Сохранить презентацию
    pres.Save(Path.Combine("YOUR_OUTPUT_DIRECTORY", "RectangleWithPictureFill.pptx"), SaveFormat.Pptx);
}
```
#### Объяснение ключевых шагов:
- **Загрузка изображения**: `FromFile` Метод загружает изображение из указанного вами каталога, которое затем добавляется в коллекцию изображений презентации.
  
- **Добавление прямоугольной формы**: Мы используем `AddAutoShape` с `ShapeType.Rectangle` и определить его размеры. Это создаст прямоугольник на слайде.

- **Настройка заливки изображения**: Назначая `FillType.Picture` в формате заполнения формы, мы преобразуем прямоугольник в контейнер изображения. Загруженное изображение затем устанавливается как эта заливка с помощью `Picture.Image` свойство.

## Раздел часто задаваемых вопросов
**В1: Могу ли я использовать эту функцию с файлами PowerPoint, созданными в старых версиях?**
Да, Aspose.Slides поддерживает широкий спектр форматов PowerPoint и обеспечивает обратную совместимость.

**В2: Как динамически изменять заливку изображения во время выполнения?**
Вы можете обновить `Picture.Image` свойство во время выполнения для изменения заливочного изображения по мере необходимости.

**В3: Можно ли применить несколько изображений в виде мозаичного узора внутри фигуры?**
Да, установив `TileOffsetX`, `TileOffsetY`, и другие свойства мозаичного изображения `IPictureFillFormat`.

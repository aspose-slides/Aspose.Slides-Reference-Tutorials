## Руководство по внедрению
В этом разделе рассматривается получение эффективных форматов текстовых фреймов и частей с помощью Aspose.Slides для .NET.

### Получите эффективный формат TextFrame
#### Обзор
Извлеките все эффективные свойства текстовой рамки на слайде PowerPoint, чтобы понять как локальное форматирование, так и унаследованные стили из родительских слайдов или основных макетов.
##### Шаг 1: Загрузите презентацию
Загрузите файл презентации с помощью Aspose.Slides `Presentation` сорт:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
using (Presentation pres = new Presentation(dataDir + "Presentation1.pptx"))
{
    // Логика доступа к слайдам и формам приведена ниже...
}
```
##### Шаг 2: Доступ к AutoShape
Получить `AutoShape` содержащий ваш целевой текст с первого слайда:
```csharp
IAutoShape shape = pres.Slides[0].Shapes[0] as IAutoShape;
```
##### Шаг 3: Извлечение TextFrameFormat и эффективных свойств
Получите местный `TextFrameFormat` для формы, то используйте `GetEffective()` для извлечения всех эффективных свойств:
```csharp
ITextFrameFormat localTextFrameFormat = shape.TextFrame.TextFrameFormat;
ITextFrameFormatEffectiveData effectiveTextFrameFormat = localTextFrameFormat.GetEffective();
```
### Получить эффективный формат порции
#### Обзор
Доступ к эффективным свойствам текстовой части внутри фигуры для детального стилизации.
##### Шаг 1: Загрузите презентацию
Загрузите файл PowerPoint аналогичным образом:
```csharp
using (Presentation pres = new Presentation(dataDir + "Presentation1.pptx"))
{
    // Логика доступа к слайдам и формам приведена ниже...
}
```
##### Шаг 2: Доступ к формату порции
Перейдите к первому абзацу и его части `AutoShape` на вашем слайде:
```csharp
IAutoShape shape = pres.Slides[0].Shapes[0] as IAutoShape;
IPortionFormat localPortionFormat = shape.TextFrame.Paragraphs[0].Portions[0].PortionFormat;
```
##### Шаг 3: Извлечение эффективных свойств
Использовать `GetEffective()` для извлечения всех эффективных свойств:
```csharp
IPortionFormatEffectiveData effectivePortionFormat = localPortionFormat.GetEffective();
```

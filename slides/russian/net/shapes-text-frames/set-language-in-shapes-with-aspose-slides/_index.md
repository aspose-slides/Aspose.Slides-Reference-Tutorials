# Как задать язык в фигурах PowerPoint с помощью Aspose.Slides для .NET

В мире цифровых презентаций обеспечение доступности и правильного форматирования вашего контента на разных языках может быть сложной задачей. С Aspose.Slides для .NET вы можете без усилий устанавливать языковые атрибуты для текста в фигурах на слайдах PowerPoint. Эта функция особенно полезна при подготовке многоязычных документов или обеспечении согласованности в глобальных коммуникациях.

**Что вы узнаете:**
- Добавление автофигур и вставка в них текста.
- Установка идентификатора языка для текстовых фрагментов с помощью Aspose.Slides.
- Сохранение презентаций с пользовательскими конфигурациями.

Давайте рассмотрим, как можно без проблем реализовать эту функцию.

## Предпосылки

Прежде чем начать, убедитесь, что у вас есть следующее:

- **Библиотеки и зависимости**: Вам необходимо установить Aspose.Slides for .NET. Эта библиотека необходима для управления презентациями PowerPoint в C#.
  
- **Настройка среды**: Требуется среда разработки с .NET Core или .NET Framework.

- **Необходимые знания**: Знакомство с базовыми концепциями программирования на языке C# и понимание принципов объектно-ориентированного программирования будет полезным.

### Обзор функций: настройка языка текста фигуры

Эта функция позволяет вам указать язык текста в форме PowerPoint. Установив идентификатор языка, вы гарантируете, что проверка орфографии и другие функции, зависящие от языка, будут применены правильно.

#### Шаг 1: Инициализация презентации

Начните с создания экземпляра `Presentation` сорт.

```csharp
using (Presentation pres = new Presentation())
{
    // Ваш код здесь
}
```

Это инициализирует новый объект презентации PowerPoint, которым мы будем управлять.

#### Шаг 2: Добавьте автофигуру и текстовую рамку

Добавьте к слайду прямоугольник и вставьте в него текст:

```csharp
IAutoShape shape = pres.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 50, 50, 200, 50);
shape.AddTextFrame("Text to apply spellcheck language");
```

Здесь, `AddAutoShape` добавляет прямоугольник к первому слайду. Параметры определяют его положение и размер.

#### Шаг 3: Установите идентификатор языка

```csharp
shape.TextFrame.Paragraphs[0].Portions[0].PortionFormat.LanguageId = "en-EN";
```

Это назначает английский (Великобритания) в качестве языка проверки орфографии.

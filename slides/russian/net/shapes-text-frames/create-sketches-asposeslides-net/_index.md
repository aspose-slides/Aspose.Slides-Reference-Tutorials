## Руководство по внедрению

Когда все готово, давайте реализуем функцию эскизной фигуры.

### Добавление и изменение фигур

#### Обзор

В этом разделе мы добавим на слайд автофигуру прямоугольного типа и настроим ее свойства для создания эффекта наброска.

**Добавление прямоугольной формы**

Начните с создания нового экземпляра презентации и добавления прямоугольной формы:
```csharp
using Aspose.Slides;
using Aspose.Slides.Export;

string outPptxFile = Path.Combine("YOUR_OUTPUT_DIRECTORY", "SketchedShapes_out.pptx");
string outPngFile = Path.Combine("YOUR_OUTPUT_DIRECTORY", "SketchedShapes_out.png");

using (Presentation pres = new Presentation())
{
    // Добавьте автофигуру типа «Прямоугольник» на первый слайд.
    IAutoShape shape = pres.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 20, 20, 300, 150);
}
```

#### Настройка формата заполнения

Чтобы придать фигуре вид эскиза, удалите всю заливку с фигуры:
```csharp
shape.FillFormat.FillType = FillType.NoFill;
```

### Применение эффектов эскиза к фигурам

#### Обзор

Затем преобразуйте прямоугольник в набросок от руки.

**Преобразование формы в эскиз**

Используйте `SketchFormat` свойство для применения эффекта каракулей:
```csharp
// Преобразуйте форму в набросок в стиле свободной руки (Scribble)
shape.LineFormat.SketchFormat.SketchType = LineSketchType.Scribble;
```

### Сохранение презентаций и изображений

Наконец, сохраните свою работу как файл презентации и как изображение.

**Сохранение как PPTX**
```csharp
// Сохраните презентацию в файл PPTX
pres.Save(outPptxFile, SaveFormat.Pptx);
```

**Сохранение как PNG-изображение**
```csharp
// Сохраните слайд как файл изображения в формате PNG.
pres.Slides[0].GetThumbnail(4/3f, 4/3f).Save(outPngFile, System.Drawing.Imaging.ImageFormat.Png);
```

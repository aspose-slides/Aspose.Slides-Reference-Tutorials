## Руководство по внедрению: маркировка фигур как декоративных

В этом разделе мы рассмотрим, как пометить фигуру как декоративную в PowerPoint с помощью C#.

### Добавление и настройка автофигуры

#### Обзор
Создание визуальных элементов в вашей презентации становится простым с помощью `AddAutoShape` метод. Мы отметим эти фигуры как декоративные, чтобы они улучшали дизайн, не влияя на инструменты доступности.

#### Шаг 1: Создание нового экземпляра презентации
Начните с создания нового экземпляра презентации PowerPoint:

```csharp
using (Presentation pres = new Presentation())
{
    // Дальнейшая настройка будет происходить здесь.
}
```

#### Шаг 2: Добавьте автофигуру на слайд
Добавьте прямоугольную форму к слайду в позицию `(10, 10)` с размерами `100x100`:

```csharp
IShape shape1 = pres.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 10, 10, 100, 100);
```

#### Шаг 3: Отметьте фигуру как декоративную
Чтобы пометить прямоугольник как декоративный, установите `IsDecorative` к истине:

```csharp
shape1.IsDecorative = true;
```

Этот шаг имеет решающее значение для обеспечения того, чтобы программы чтения с экрана пропускали эти элементы.

#### Шаг 4: Сохраните презентацию
Наконец, сохраните вашу презентацию в формате PPTX в указанном месте:

```csharp
string outFilePath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "DecorativeDemo.pptx");
pres.Save(outFilePath, SaveFormat.Pptx);
```

# Изменение порядка фигур в PowerPoint с помощью Aspose.Slides для .NET
## Введение
Улучшите свои презентации PowerPoint, динамически изменяя порядок фигур с помощью Aspose.Slides для .NET — мощной библиотеки для программного управления файлами презентаций.
**Aspose.Slides для .NET** предоставляет надежные функции для автоматизации и преобразования презентаций. Это пошаговое руководство покажет вам, как изменить порядок фигур, таких как прямоугольники и треугольники, в слайдах, гарантируя, что ваш контент будет отображаться в желаемом порядке.
### Что вы узнаете:
- Настройка Aspose.Slides для .NET
- Добавление и изменение текстовых рамок в фигурах
- Изменение порядка фигур на слайде PowerPoint
- Сохранение измененной презентации
Давайте рассмотрим предварительные условия перед реализацией переупорядочивания фигур.

Перед началом убедитесь, что у вас есть:

## Руководство по внедрению
Выполните следующие действия, чтобы изменить порядок фигур на слайде PowerPoint с помощью Aspose.Slides для .NET.
### Добавление и изменение порядка фигур
#### Обзор
Динамически изменяйте порядок фигур на слайде, что полезно для презентаций, требующих корректировки визуальной иерархии.
**Шаг 1: Загрузите существующую презентацию**
Загрузите файл PowerPoint в Aspose.Slides:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
// Загрузить существующую презентацию
Presentation presentation1 = new Presentation(dataDir + "HelloWorld.pptx");
```
**Шаг 2: Откройте слайд и добавьте фигуры**
Откройте нужный слайд и добавьте фигуру, например прямоугольник для текста:
```csharp
ISlide slide = presentation1.Slides[0];
// Добавьте прямоугольник без заливки.
IAutoShape shp3 = slide.Shapes.AddAutoShape(ShapeType.Rectangle, 200, 365, 400, 150);
shp3.FillFormat.FillType = FillType.NoFill;
```
**Шаг 3: Вставьте текст в фигуру**
Манипулирование текстом внутри фигур:
```csharp
// Добавьте текстовую рамку и установите текст водяного знака
ITextFrame txtFrame = shp3.TextFrame;
IParagraph para = txtFrame.Paragraphs[0];
IPortion portion = para.Portions[0];
portion.Text = "Watermark Text Watermark Text Watermark Text";
```
**Шаг 4: Добавьте еще одну форму**
Добавьте к слайду треугольную форму:
```csharp
shp3 = slide.Shapes.AddAutoShape(ShapeType.Triangle, 200, 365, 400, 150);
```
**Шаг 5: Измените порядок фигур**
Управляйте визуальным порядком наложения, изменяя порядок фигур:
```csharp
// Переместите треугольник на позицию 2 в коллекции фигур.
slide.Shapes.Reorder(2, shp3);
```
### Сохранение презентации
Сохраните измененную презентацию:
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
presentation1.Save(outputDir + "Reshape_out.pptx");
```

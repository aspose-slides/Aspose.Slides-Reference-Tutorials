## Руководство по внедрению

В этом разделе мы рассмотрим, как реализовать две ключевые функции с помощью Aspose.Slides для .NET.

### Установка языка текста по умолчанию с параметрами загрузки

**Обзор**: эта функция позволяет указать язык текста по умолчанию при загрузке презентаций, обеспечивая единообразие слайдов.

1. **Инициализировать LoadOptions**
   
   Начните с настройки параметров загрузки:
   ```csharp
   LoadOptions loadOptions = new LoadOptions();
   loadOptions.DefaultTextLanguage = "en-US"; // Установить английский (США) как язык по умолчанию
   ```

2. **Загрузить презентацию с указанными параметрами**
   
   Используйте эти параметры при создании нового экземпляра презентации:
   ```csharp
   using (Presentation pres = new Presentation(loadOptions))
   {
       // Добавляйте фигуры или управляйте слайдами здесь
   }
   ```

3. **Добавить и проверить язык текста**
   
   Вы можете добавлять текст к фигурам и проверять язык:
   ```csharp
   IAutoShape shp = pres.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 50, 50, 150, 50);
   shp.TextFrame.Text = "New Text";

   var languageId = shp.TextFrame.Paragraphs[0].Portions[0].PortionFormat.LanguageId;
   ```

### Добавление фигуры с текстом на слайд

**Обзор**: эта функция позволяет добавлять фигуры, содержащие текст, улучшая визуальную привлекательность и функциональность слайдов.

1. **Инициализировать презентацию**

   Начните с создания новой презентации:
   ```csharp
   using (Presentation pres = new Presentation())
   {
       // Доступ к первому слайду
       ISlide slide = pres.Slides[0];

       // Добавьте прямоугольник с текстом
       IAutoShape shp = slide.Shapes.AddAutoShape(ShapeType.Rectangle, 50, 50, 150, 50);
       shp.TextFrame.Text = "Hello World";
   }
   ```

2. **Настроить свойства формы**

   При необходимости отрегулируйте размер и положение в соответствии со стилем вашей презентации.

### Советы по устранению неполадок

- Убедитесь, что Aspose.Slides правильно установлен и лицензирован.

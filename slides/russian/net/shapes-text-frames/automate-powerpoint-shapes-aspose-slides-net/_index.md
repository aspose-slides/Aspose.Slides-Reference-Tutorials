### Функция 1: Загрузка и доступ к презентации
**Обзор:**
Загрузка презентации PowerPoint с помощью Aspose.Slides проста. Эта функция демонстрирует, как получить доступ к существующему файлу и подготовить его для манипуляций.

#### Пошаговая реализация:

##### **1. Определите каталог документов**
Определите, где хранятся ваши файлы PowerPoint. Используйте `Path.Combine` для создания полного пути к файлу презентации.

```csharp
using System.IO;
using Aspose.Slides;

string documentDirectory = @"YOUR_DOCUMENT_DIRECTORY";
string presentationName = Path.Combine(documentDirectory, "PresetGeometry.pptx");
```

##### **2. Загрузите презентацию**
Создать `Presentation` объект, передав путь к вашему файлу PPTX.

```csharp
// Загрузите презентацию по указанному пути.
Presentation pres = new Presentation(presentationName);
```

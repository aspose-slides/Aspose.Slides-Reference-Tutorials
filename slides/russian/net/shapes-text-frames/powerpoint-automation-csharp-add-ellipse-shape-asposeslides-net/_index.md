## Руководство по внедрению

В этом разделе рассматривается реализация двух основных функций: создание каталогов и добавление эллиптических фигур в слайды PowerPoint с использованием C#.

### Функция 1: Создать каталог, если он не существует

**Обзор:** Эта функция гарантирует наличие каталога перед выполнением файловых операций, предотвращая ошибки, связанные с отсутствием путей.

#### Пошаговая реализация:

**Проверить и создать каталог**
```csharp
using System.IO;

string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Замените на ваш реальный путь
bool isExists = Directory.Exists(dataDir);

if (!isExists)
{
    Directory.CreateDirectory(dataDir); // Создает каталог, если он не существует
}
```

- **Объяснение**: `Directory.Exists()` проверяет, существует ли каталог, и `Directory.CreateDirectory()` создает его, если отсутствует. Это гарантирует, что все файловые операции имеют допустимый путь.

### Функция 2: добавление эллипса к слайду

**Обзор:** Автоматизируйте добавление фигур на слайды PowerPoint, начав с эллипса на первом слайде.

#### Пошаговая реализация:

**Добавить форму эллипса**
```csharp
using Aspose.Slides;
using Aspose.Slides.Export;

string outputDir = "YOUR_DOCUMENT_DIRECTORY"; // Заменить на ваш путь
string outputFile = Path.Combine(outputDir, "EllipseShape_out.pptx");

using (Presentation pres = new Presentation())
{
    ISlide sld = pres.Slides[0]; // Получить первый слайд

    // Добавьте к слайду в позиции (50, 150) эллипс шириной 150 и высотой 50.
    sld.Shapes.AddAutoShape(ShapeType.Ellipse, 50, 150, 150, 50);

    pres.Save(outputFile, SaveFormat.Pptx); // Сохраните презентацию в формате PPTX
}
```

- **Объяснение**: `AddAutoShape` Метод позволяет указать тип и размеры фигуры. Этот фрагмент добавляет эллипс к первому слайду новой презентации.

## Руководство по внедрению

Давайте рассмотрим, как реализовать функции выравнивания фигур с помощью Aspose.Slides для .NET.

### Выровнять фигуры на слайде (H2)
Эта функция демонстрирует выравнивание фигур в пределах всего слайда. Вот как этого можно добиться:

#### Шаг 1: Создание и добавление фигур
Добавьте на слайд несколько прямоугольников в качестве заполнителей:

```csharp
ISlide slide = pres.Slides[0];
slide.Shapes.AddAutoShape(ShapeType.Rectangle, 100, 100, 100, 100);
slide.Shapes.AddAutoShape(ShapeType.Rectangle, 200, 200, 100, 100);
slide.Shapes.AddAutoShape(ShapeType.Rectangle, 300, 300, 100, 100);
```

#### Шаг 2: Выровняйте фигуры
Используйте `AlignShapes` метод выравнивания этих фигур внизу:

```csharp
SlideUtil.AlignShapes(ShapesAlignmentType.AlignBottom, true, pres.Slides[0]);
```
**Объяснение:** Параметры определяют тип выравнивания (`AlignBottom`), следует ли включать текст (`true`) и целевой слайд.

#### Шаг 3: Сохраните презентацию
Сохраните изменения в новом файле:

```csharp
pres.Save("ShapesAlignment_out.pptx", SaveFormat.Pptx);
```

### Выровнять фигуры в GroupShape (H2)
В этом разделе показано, как выравнивать фигуры в групповой фигуре, обеспечивая единое выравнивание.

#### Шаг 1: Создайте групповую фигуру и добавьте фигуры
Добавьте свои фигуры в новую группу:

```csharp
ISlide slide = pres.Slides.AddEmptySlide(pres.Slides[0].LayoutSlide);
IGroupShape groupShape = slide.Shapes.AddGroupShape();
groupShape.Shapes.AddAutoShape(ShapeType.Rectangle, 350, 50, 50, 50);
// При необходимости добавьте больше фигур.
```

#### Шаг 2: Выровняйте фигуры внутри группы
Выровняйте все эти фигуры по левому краю внутри их группы:

```csharp
SlideUtil.AlignShapes(ShapesAlignmentType.AlignLeft, false, groupShape);
```

### Выровнять определенные фигуры в GroupShape (H2)
Вы также можете выбрать определенные фигуры для выравнивания с помощью индексов.

#### Шаг 1: Настройте форму своей группы
Аналогично предыдущему разделу создайте свою группу и добавьте фигуры:

```csharp
IGroupShape groupShape = slide.Shapes.AddGroupShape();
groupShape.Shapes.AddAutoShape(ShapeType.Rectangle, 350, 50, 50, 50);
// Дополнительные формы...
```

#### Шаг 2: Выровняйте определенные фигуры
Используйте индексы, чтобы указать, какие фигуры следует выравнивать:

```csharp
SlideUtil.AlignShapes(ShapesAlignmentType.AlignLeft, false, groupShape, new int[] { 0, 2 });
```
**Объяснение:** Это выравнивает только первую и третью фигуры в группе.

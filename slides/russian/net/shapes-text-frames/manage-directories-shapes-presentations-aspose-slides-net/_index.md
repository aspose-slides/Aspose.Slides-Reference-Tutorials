## Руководство по внедрению

Этот раздел разделен на две основные функции: создание каталогов, если они не существуют, и работа с формами презентаций для добавления изображений.

### Создание каталогов

#### Обзор
Обеспечение существования каталога перед выполнением операций с файлами имеет решающее значение. Эта функция помогает проверить существование указанного каталога и создает его, если он отсутствует, предотвращая потенциальные ошибки во время манипуляций с файлами.

#### Этапы внедрения

**Шаг 1: Определите путь к каталогу**
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
```
*Заменять `YOUR_DOCUMENT_DIRECTORY` с желаемым вами путем.*

**Шаг 2: Проверка и создание каталога**
```csharp
bool isExists = Directory.Exists(dataDir);
if (!isExists) {
    Directory.CreateDirectory(dataDir);
}
```
Этот код проверяет, существует ли каталог, используя `Directory.Exists`. Если он возвращает false, `Directory.CreateDirectory` вызывается для создания каталога.

### Работа с презентациями и фигурами

#### Обзор
Включение изображений в презентации может сделать их более интересными. Эта функция демонстрирует, как загрузить презентацию, добавить изображение в качестве заливки фигуры и настроить смещения для лучшего позиционирования.

#### Этапы внедрения

**Шаг 1: Загрузите изображение**
```csharp
IImage img = Images.FromFile(dataDir + "aspose-logo.jpg");
```
*Убедитесь, что путь к изображению указан правильно.*

**Шаг 2: Инициализация презентации и добавление формы**
```csharp
using (Presentation pres = new Presentation()) {
    ISlide slide = pres.Slides[0];
    IAutoShape aShape = slide.Shapes.AddAutoShape(ShapeType.Rectangle, 100, 100, 300, 300);
    
    aShape.FillFormat.FillType = FillType.Picture;
    aShape.FillFormat.PictureFillFormat.PictureFillMode = PictureFillMode.Stretch;
    IPPImage imgEx = pres.Images.AddImage(img);
    aShape.FillFormat.PictureFillFormat.Picture.Image = imgEx;

    // Установить смещения
    aShape.FillFormat.PictureFillFormat.StretchOffsetLeft = 25;
    aShape.FillFormat.PictureFillFormat.StretchOffsetRight = 25;
    aShape.FillFormat.PictureFillFormat.StretchOffsetTop = -20;
    aShape.FillFormat.PictureFillFormat.StretchOffsetBottom = -10;

    pres.Save(dataDir + "StretchOffsetLeftForPictureFrame_out.pptx", SaveFormat.Pptx);
}
```
Этот фрагмент загружает изображение, добавляет его к первому слайду в качестве заливки прямоугольника и задает смещения для улучшенного выравнивания.

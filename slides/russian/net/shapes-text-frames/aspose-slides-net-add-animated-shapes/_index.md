# Добавление анимированных фигур в презентации с помощью Aspose.Slides для .NET

Добавление интерактивных элементов, таких как анимированные фигуры, может значительно улучшить вашу презентацию. Это руководство проведет вас через использование Aspose.Slides для .NET для добавления анимированной формы кнопки к вашим слайдам, что сделает их более увлекательными и запоминающимися.

**Что вы узнаете:**
- Как создать каталоги в C# с помощью Aspose.Slides

- Реализация интерактивных кнопок с пользовательскими путями анимации

Готовы вывести свои презентации на новый уровень? Давайте углубимся в настройку вашей среды и пошаговое кодирование этих функций.

### Предпосылки

## Руководство по внедрению

### Функция 1: Создание каталога

Перед добавлением любого контента убедитесь, что выходной каталог существует. Вот как это сделать с помощью C#:

#### Проверить и создать каталог
```csharp
using System.IO;

// Определите путь к каталогу ваших документов.
string dataDir = "YOUR_DOCUMENT_DIRECTORY";

// Проверьте, существует ли каталог; если нет, создайте его.
bool isExists = Directory.Exists(dataDir);
if (!isExists)
{
    Directory.CreateDirectory(dataDir);
}
```

Этот простой скрипт проверяет наличие указанного каталога и создает его, если он не существует, гарантируя правильное сохранение ваших файлов.

### Функция 2: Добавление формы с помощью анимации

Далее добавим фигуру к слайду и применим эффект анимации с помощью Aspose.Slides:

#### Добавление анимированных фигур
```csharp
using Aspose.Slides;
using Aspose.Slides.Animation;

string outputDir = "YOUR_OUTPUT_DIRECTORY";

// Создайте новую презентацию.
using (Presentation pres = new Presentation())
{
    ISlide sld = pres.Slides[0];

    // Добавьте на слайд прямоугольник с текстом.
    IAutoShape ashp = sld.Shapes.AddAutoShape(ShapeType.Rectangle, 150, 150, 250, 25);
    ashp.AddTextFrame("Animated TextBox");

    // Примените к фигуре эффект анимации PathFootball.
    sld.Timeline.MainSequence.AddEffect(
        ashp,
        EffectType.PathFootball,
        EffectSubtype.None,
        EffectTriggerType.AfterPrevious
    );

    // Сохраните презентацию с анимацией.
    pres.Save(outputDir + "AnimExample_out.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
}
```

Этот код добавляет к слайду прямоугольную форму и применяет эффект анимации, делая его более привлекательным.

### Функция 3: добавление интерактивной формы кнопки с пользовательским путем анимации

Для интерактивных презентаций создайте формы кнопок, которые запускают пользовательскую анимацию:

#### Создание интерактивных кнопок
```csharp
using Aspose.Slides;
using Aspose.Slides.Animation;

string outputDir = "YOUR_OUTPUT_DIRECTORY";

// Создайте новую презентацию.
using (Presentation pres = new Presentation())
{
    ISlide sld = pres.Slides[0];

    // Создайте на слайде форму кнопки.
    IShape shapeTrigger = sld.Shapes.AddAutoShape(ShapeType.Bevel, 10, 10, 20, 20);

    // Добавьте интерактивную последовательность к кнопке.
    ISequence seqInter = sld.Timeline.InteractiveSequences.Add(shapeTrigger);

    // Предположим, что вторая фигура является нашей целью для анимации.
    IAutoShape ashp = sld.Shapes[1] as IAutoShape;

    // Добавьте пользовательский эффект PathUser, срабатывающий при щелчке.
    IEffect fxUserPath = seqInter.AddEffect(
        ashp,
        EffectType.PathUser,
        EffectSubtype.None,
        EffectTriggerType.OnClick
    );

    // Определите траекторию движения для анимации.
    IMotionEffect motionBhv = ((IMotionEffect)fxUserPath.Behaviors[0]);
    PointF[] pts = new PointF[1];

    // Команда двигаться вдоль линии.
    pts[0] = new PointF(0.076f, 0.59f);
    motionBhv.Path.Add(
        MotionCommandPathType.LineTo,
        pts,
        MotionPathPointsType.Auto,
        true
    );

    // Перейдите в другую точку и добавьте команду.
    pts[0] = new PointF(-0.076f, -0.59f);
    motionBhv.Path.Add(
        MotionCommandPathType.LineTo,
        pts,
        MotionPathPointsType.Auto,
        false
    );

    // Завершите путь.
    motionBhv.Path.Add(MotionCommandPathType.End, null, MotionPathPointsType.Auto, false);

    // Сохраните презентацию с помощью интерактивной анимации.
    pres.Save(outputDir + "ButtonAnimExample_out.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
}
```

Этот код создает интерактивную кнопку, которая при нажатии запускает пользовательский путь анимации.

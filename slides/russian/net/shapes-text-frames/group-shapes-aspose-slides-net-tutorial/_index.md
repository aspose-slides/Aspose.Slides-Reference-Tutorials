## Руководство по внедрению
В этом разделе мы рассмотрим каждый шаг, необходимый для создания групповых фигур и добавления в них отдельных фигур.

### Создание групповой фигуры на слайде
Начните с доступа к слайду, на который вы хотите добавить фигуру группы:
```csharp
// Доступ к первому слайду презентации
ISlide sld = pres.Slides[0];
```
Затем возьмите коллекцию фигур на этом слайде и создайте новую групповую фигуру:
```csharp
// Получить коллекцию форм слайда
IShapeCollection slideShapes = sld.Shapes;

// Добавьте на слайд фигуру группы
IGroupShape groupShape = slideShapes.AddGroupShape();
```

### Добавление отдельных фигур внутри группы
Создав групповую фигуру, вы теперь можете добавлять в нее различные фигуры. Вот как добавлять прямоугольники:
```csharp
// Добавьте фигуры внутрь созданной групповой фигуры.
groupShape.Shapes.AddAutoShape(ShapeType.Rectangle, 300, 100, 100, 100);
groupShape.Shapes.AddAutoShape(ShapeType.Rectangle, 500, 100, 100, 100);
groupShape.Shapes.AddAutoShape(ShapeType.Rectangle, 300, 300, 100, 100);
groupShape.Shapes.AddAutoShape(ShapeType.Rectangle, 500, 300, 100, 100);
```
**Объясняемые параметры:**
- `ShapeType.Rectangle`: Тип добавляемой вами формы.
- `x`, `y` (например, 300, 100): Расположите координаты на слайде.
- Ширина и высота (например, 100, 100): Размеры фигуры.

### Сохранение презентации

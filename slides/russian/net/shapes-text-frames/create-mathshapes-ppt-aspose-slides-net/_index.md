## Руководство по внедрению
Давайте разобьем процесс на управляемые шаги. Этот раздел проведет вас через создание MathShape и добавление MathBlock.

### Создание MathShape на слайде PowerPoint
#### Обзор
Начнем с настройки новой презентации, откроем первый слайд, а затем добавим к нему MathShape.

#### Шаги:
**Шаг 1: Инициализация презентации**
Начните с создания нового экземпляра `Presentation` класс. Это представляет весь ваш файл PowerPoint.

```csharp
using (var presentation = new Presentation())
{
    // Код для создания фигур будет здесь
}
```

**Почему**: Это создает среду, в которой вы можете программно управлять слайдами.

#### Шаг 2: Добавьте MathShape на слайд
Теперь давайте добавим MathShape в определенное место на слайде.

```csharp
ISlide slide = presentation.Slides[0];
IAutoShape mathShape = slide.Shapes.AddMathShape(10, 10, 500, 500);
```

**Почему**На этом этапе на слайде размещается математический контейнер, куда впоследствии можно добавлять уравнения или выражения.

### Добавление MathBlock
#### Обзор
Далее мы сосредоточимся на заполнении MathShape реальным математическим содержимым с помощью MathBlock.

#### Шаги:
**Шаг 3: Доступ к MathParagraph**
Получить `IMathParagraph` объект из MathShape для вставки математического текста.

```csharp
IMathParagraph mathParagraph = (mathShape.TextFrame.Paragraphs[0].Portions[0] as MathPortion).MathParagraph;
```

**Почему**: Это позволяет вам управлять абзацем, в котором будут находиться ваши уравнения.

**Шаг 4: Создайте и добавьте MathBlock**
Создать новый `MathBlock` с примером математического выражения и добавьте его в MathParagraph.

```csharp
IMathBlock mathBlock = new MathBlock(new MathematicalText("F").Join(".")
    .Join(new MathematicalText("1").Divide("y")).Underbar());
mathParagraph.Add(mathBlock);
```

**Почему**: На этом этапе создается сложное математическое выражение и встраивается в слайд.

### Сохранение презентации
Наконец, сохраните вашу презентацию в файл:

```csharp
string outPptxFile = Path.Combine(YOUR_DOCUMENT_DIRECTORY, "MathShape_GetChildren_out.pptx");
presentation.Save(outPptxFile, SaveFormat.Pptx);
```

**Почему**: Это гарантирует сохранение всех изменений в новом файле PowerPoint.

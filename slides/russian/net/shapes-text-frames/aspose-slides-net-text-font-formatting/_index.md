### Добавление текста в фигуру

#### Обзор
Эта функция позволяет добавлять пользовательский текст в AutoShapes, например, прямоугольники в слайдах. Это важно для доставки индивидуального контента непосредственно на слайдах.

#### Шаги по реализации

**1. Создание и добавление автофигуры**

```csharp
using (Presentation pres = new Presentation())
{
    IAutoShape newShape = pres.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 100, 100, 400, 75, false);
```
- **Параметры**: 
  - `ShapeType.Rectangle`: Определяет тип фигуры.
  - Координаты (x=100, y=100) и размеры (ширина=400, высота=75): Положение и размер фигуры.

**2. Добавьте текстовую рамку**

```csharp
    newShape.AddTextFrame("");
```
- **Цель**: Инициализирует пустой текстовый фрейм для хранения вашего пользовательского текста.

**3. Вставка фрагментов текста**

```csharp
    newShape.TextFrame.Paragraphs[0].Portions.Clear();
    
    IPortion portion0 = new Portion("Sample text with first portion");
    IPortion portion1 = new Portion(" and second portion.");
    
    newShape.TextFrame.Paragraphs[0].Portions.Add(portion0);
    newShape.TextFrame.Paragraphs[0].Portions.Add(portion1);
}
```
- **Объяснение**: Очистите существующие части, затем создайте и добавьте новые текстовые сегменты. Это позволяет сегментировать контент в пределах одного абзаца.

### Установка высоты шрифта по умолчанию для презентации

#### Обзор
Установка единой высоты шрифта во всей презентации обеспечивает единообразие дизайна и удобочитаемость.

#### Шаги по реализации

**1. Добавьте текстовые части**
Повторно используйте код для добавления текстовых фрагментов, как показано выше.

**2. Установите высоту шрифта по умолчанию**

```csharp
    pres.DefaultTextStyle.GetLevel(0).DefaultPortionFormat.FontHeight = 24;
```
- **Цель**: Применяет одинаковую высоту шрифта 24 пункта ко всем текстовым частям презентации.

### Установка высоты шрифта по умолчанию для абзаца

#### Обзор
Вы можете настраивать отдельные абзацы на слайдах, выделяя определенный контент.

#### Шаги по реализации

**1. Добавьте текстовые части**
Как уже было сказано ранее.

**2. Настройте высоту шрифта для определенного абзаца**

```csharp
    newShape.TextFrame.Paragraphs[0].ParagraphFormat.DefaultPortionFormat.FontHeight = 40;
```
- **Объяснение**: Устанавливает высоту шрифта всех частей этого абзаца на 40 пунктов, что усиливает его визуальное воздействие.

### Установка высоты шрифта для отдельной части

#### Обзор
Для точного управления типографикой презентации настройте размер шрифта отдельных фрагментов текста по отдельности.

#### Шаги по реализации

**1. Добавьте текстовые части**
Вернитесь к начальным шагам по добавлению текстовых фрагментов.

**2. Установите определенную высоту шрифта**

```csharp
    newShape.TextFrame.Paragraphs[0].Portions[0].PortionFormat.FontHeight = 55;
    newShape.TextFrame.Paragraphs[0].Portions[1].PortionFormat.FontHeight = 18;
```
- **Объяснение**: Эта настройка позволяет задать для каждой части уникальную высоту шрифта, что позволяет выделить детали там, где это необходимо.

### Сохранение презентации

#### Обзор
Как только ваша презентация будет доведена до совершенства, сохраните ее в формате файла по вашему выбору.

```csharp
using (Presentation pres = new Presentation())
{
    // Добавьте фигуры и текст, как описано выше...

    // Сохранить презентацию
    pres.Save("YOUR_OUTPUT_DIRECTORY\SetLocalFontHeightValues.pptx", SaveFormat.Pptx);
}
```
- **Подробности**: Это сохранит отформатированные слайды в файле PPTX, готовом к распространению или дальнейшему редактированию.

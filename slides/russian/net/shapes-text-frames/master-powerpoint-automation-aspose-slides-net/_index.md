### Функция 1: Создание каталогов
**Обзор:** Эта функция гарантирует, что ваше приложение имеет необходимую структуру каталогов перед попыткой выполнения файловых операций.

#### Пошаговое руководство:
1. **Проверить существование каталога**
   ```csharp
   using System.IO;

   string dataDir = "YOUR_DOCUMENT_DIRECTORY";
   bool isExists = Directory.Exists(dataDir);
   ```
2. **Создать каталог, если он не существует**
   ```csharp
   if (!isExists)
   {
       Directory.CreateDirectory(dataDir); // Создает каталог по указанному пути.
   }
   ```
   
#### Объяснение:
- `Directory.Exists`: Проверяет, существует ли каталог по указанному пути.
- `Directory.CreateDirectory`: Создает новый каталог.

### Функция 3: Добавление автофигуры с эффектами TextFrame и Shadow
**Обзор:** Узнайте, как добавить прямоугольную фигуру с текстом и применить эффекты тени для визуального улучшения.

#### Пошаговое руководство:
1. **Добавить автофигуру**
   ```csharp
   ISlide slide = pres.Slides[0]; // Получите ссылку на первый слайд.
   IAutoShape autoShape = slide.Shapes.AddAutoShape(ShapeType.Rectangle, 150, 75, 150, 50); // Добавьте прямоугольную форму.
   ```
2. **Добавить текстовый фрейм**
   ```csharp
   autoShape.AddTextFrame("Aspose TextBox"); // Вставьте текст в форму.
   autoShape.FillFormat.FillType = FillType.NoFill; // Отключите заливку для видимости эффекта тени.
   ```
3. **Применить эффекты тени**
   ```csharp
   autoShape.EffectFormat.EnableOuterShadowEffect(); 
   IOuterShadow shadow = autoShape.EffectFormat.OuterShadowEffect;

   // Настройте свойства тени:
   shadow.BlurRadius = 4.0; // Установить радиус размытия.
   shadow.Direction = 45; // Определите угол направления.
   shadow.Distance = 3; // Укажите расстояние от текста.
   shadow.RectangleAlign = RectangleAlignment.TopLeft; // Выровняйте прямоугольник тени.
   shadow.ShadowColor.PresetColor = PresetColor.Black; // Выберите черный цвет для тени.
   ```

#### Объяснение:
- **Автоформа**: Универсальная форма, которую можно настраивать с помощью различных свойств, включая текст и эффекты.
- **OuterShadowEffect**: Применяет реалистичную тень для усиления визуальной глубины.

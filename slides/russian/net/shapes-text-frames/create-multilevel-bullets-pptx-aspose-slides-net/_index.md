## Руководство по внедрению

### Создание и управление каталогами

Во-первых, нам нужно убедиться, что каталог, в котором будет сохранена наша презентация, существует. Вот как это можно сделать:

**Шаг 1: Проверьте существование каталога**

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Укажите путь к документу здесь
bool isExists = Directory.Exists(dataDir);
if (!isExists)
{
    Directory.CreateDirectory(dataDir); // Создайте каталог, если он не существует
}
```

**Объяснение:** Этот фрагмент проверяет, существует ли указанный каталог. Если нет, он создает его для хранения файлов презентации.

### Создание презентации с помощью Aspose.Slides

Теперь давайте создадим новую презентацию PowerPoint и откроем ее первый слайд:

```csharp
using (Presentation pres = new Presentation())
{
    ISlide slide = pres.Slides[0]; // Доступ к первому слайду
}
```

**Объяснение:** Мы инициализируем `Presentation` объект, представляющий наш файл PPTX. По умолчанию он включает один слайд.

### Добавление автофигуры к слайду

Чтобы добавить контент, вставим автофигуру (прямоугольник) и настроим ее текстовую рамку:

```csharp
IAutoShape aShp = slide.Shapes.AddAutoShape(ShapeType.Rectangle, 200, 200, 400, 200); // Положение и размер прямоугольника
ITextFrame text = aShp.AddTextFrame(""); // Создать пустой текстовый фрейм
text.Paragraphs.Clear(); // Удалить любой абзац по умолчанию
```

**Объяснение:** Этот фрагмент добавляет прямоугольную форму к слайду. Затем мы инициализируем его текстовую рамку для добавления маркированного контента.

### Управление форматированием абзацев с помощью маркеров

Далее мы форматируем абзацы с помощью маркеров разного уровня:

```csharp
// Добавление первого абзаца
IParagraph para1 = new Paragraph();
para1.Text = "Content";
para1.ParagraphFormat.Bullet.Type = BulletType.Symbol;
para1.ParagraphFormat.Bullet.Char = Convert.ToChar(8226);
para1.ParagraphFormat.DefaultPortionFormat.FillFormat.FillType = FillType.Solid;
para1.ParagraphFormat.DefaultPortionFormat.FillFormat.SolidFillColor.Color = Color.Black;
para1.ParagraphFormat.Depth = 0;

// Добавление последующих абзацев с различными типами и уровнями маркеров
IParagraph para2 = new Paragraph();
para2.Text = "Second Level";
para2.ParagraphFormat.Bullet.Type = BulletType.Symbol;
para2.ParagraphFormat.Bullet.Char = '-';
para2.ParagraphFormat.DefaultPortionFormat.FillFormat.FillType = FillType.Solid;
para2.ParagraphFormat.DefaultPortionFormat.FillFormat.SolidFillColor.Color = Color.Black;
para2.ParagraphFormat.Depth = 1;

// Повторите то же самое для para3 и para4 с соответствующими символами маркеров и уровнями.
```

**Объяснение:** Каждый абзац настраивается с использованием определенных стилей маркеров, цветов и уровней отступов для создания иерархии.

Наконец, мы добавляем эти абзацы в текстовый фрейм:

```csharp
text.Paragraphs.Add(para1);
text.Paragraphs.Add(para2);
// Повторите для пункта 3 и пункта 4.
```

### Сохранение презентации

Теперь, когда наша презентация готова, давайте сохраним ее как файл PPTX:

```csharp
pres.Save("YOUR_OUTPUT_DIRECTORY/MultilevelBullet.pptx", SaveFormat.Pptx); // Укажите выходной каталог
```

**Объяснение:** The `Save` метод записывает презентацию на диск в указанном формате.

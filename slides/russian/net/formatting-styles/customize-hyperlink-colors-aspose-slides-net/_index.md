## Руководство по внедрению

Мы рассмотрим две основные функции: установку пользовательского цвета для гиперссылок и добавление стандартных гиперссылок без настройки.

### Функция 1: Установка цвета гиперссылки в слайдах PowerPoint

Эта функция позволяет вам изменять цвет текста гиперссылки, улучшая ее видимость или подгоняя под вашу тему дизайна.

#### Пошаговая реализация:

**1. Загрузка презентации**
Начните с загрузки существующей презентации или создания новой с помощью Aspose.Slides.

```csharp
using (Presentation presentation = new Presentation())
{
    // Продолжайте выполнять дальнейшие шаги...
}
```

**2. Добавьте автофигуру и текстовую рамку**
Создайте фигуру и добавьте текст, включающий вашу гиперссылку.

```csharp
IAutoShape shape1 = presentation.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 100, 100, 450, 50, false);
shape1.AddTextFrame("This is a sample of colored hyperlink.");
```

**3. Задайте URL-адрес гиперссылки и источник цвета**
Назначьте URL-адрес гиперссылки и укажите, что цвет должен быть получен из PortionFormat.

```csharp
shape1.TextFrame.Paragraphs[0].Portions[0].PortionFormat.HyperlinkClick = new Hyperlink("https://www.aspose.com/");
shape1.TextFrame.Paragraphs[0].Portions[0].PortionFormat.HyperlinkClick.ColorSource = HyperlinkColorSource.PortionFormat;
```

**4. Настройте цвет заливки**
Измените цвет текста гиперссылки, установив сплошную заливку.

```csharp
shape1.TextFrame.Paragraphs[0].Portions[0].PortionFormat.FillFormat.FillType = FillType.Solid;
shape1.TextFrame.Paragraphs[0].Portions[0].PortionFormat.FillFormat.SolidFillColor.Color = Color.Red;
```

### Функция 2: Установить обычную гиперссылку

Для стандартной реализации гиперссылки без настройки цвета выполните следующие действия:

**1. Загрузка презентации**
Как и в предыдущей статье, начните с презентации.

```csharp
using (Presentation presentation = new Presentation())
{
    // Продолжайте добавлять гиперссылки...
}
```

**2. Добавьте автофигуру и текстовую рамку**
Создайте форму для текстовой гиперссылки.

```csharp
IAutoShape shape2 = presentation.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 100, 200, 450, 50, false);
shape2.AddTextFrame("This is a sample of usual hyperlink.");
```

**3. Назначить URL-адрес гиперссылки**
Задайте URL-адрес для гиперссылки.

```csharp
shape2.TextFrame.Paragraphs[0].Portions[0].PortionFormat.HyperlinkClick = new Hyperlink("https://www.aspose.com/");
```

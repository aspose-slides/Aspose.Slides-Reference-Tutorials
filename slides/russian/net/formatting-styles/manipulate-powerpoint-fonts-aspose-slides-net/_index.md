## Руководство по внедрению

### Функция: Манипулирование свойствами шрифта

Эта функция позволяет изменять стили шрифтов, цвета и другие свойства слайдов PowerPoint с помощью C#.

#### Шаг 1: Определите каталог документов
Укажите путь, где будут храниться ваши файлы PowerPoint:
```csharp
csharp
string dataDir = @"YOUR_DOCUMENT_DIRECTORY";
```

#### Шаг 2: Загрузка презентации
Создать `Presentation` объект для работы с вашим файлом PPTX:
```csharp
using (Presentation pres = new Presentation(dataDir + "FontProperties.pptx"))
{
    // Ваш код здесь
}
```

#### Шаг 3: Доступ к слайдам и текстовым фреймам
Получите доступ к слайду и его текстовым фреймам, используя их позиции в коллекции фигур:
```csharp
ISlide slide = pres.Slides[0];
ITextFrame tf1 = ((IAutoShape)slide.Shapes[0]).TextFrame;
ITextFrame tf2 = ((IAutoShape)slide.Shapes[1]).TextFrame;
```

#### Шаг 4: Манипулирование свойствами шрифта
Измените данные шрифта, стили и цвета следующим образом:
```csharp
IParagraph para1 = tf1.Paragraphs[0];
IPortion port1 = para1.Portions[0];

// Определение новых шрифтов с помощью FontData
FontData fd1 = new FontData("Elephant");
port1.PortionFormat.LatinFont = fd1;

// Задайте свойства шрифта, такие как «Жирный» и «Курсив»
port1.PortionFormat.FontBold = NullableBool.True;
port1.PortionFormat.FontItalic = NullableBool.True;

// Изменить цвет шрифта на сплошную заливку
port1.PortionFormat.FillFormat.FillType = FillType.Solid;
port1.PortionFormat.FillFormat.SolidFillColor.Color = Color.Purple;
```

#### Шаг 5: Сохраните презентацию
Сохраните изменения обратно в файл:
```csharp
pres.Save(dataDir + "WelcomeFont_out.pptx", SaveFormat.Pptx);
```

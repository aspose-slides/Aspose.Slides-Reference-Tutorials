## Руководство по внедрению

### Функция: Извлечение эффективных данных заполнения пуль

Эта функция извлекает и отображает эффективные данные заполнения для маркеров на слайде презентации, позволяя программно настраивать их внешний вид.

#### Шаг 1: Определите пути к каталогам

Начните с определения путей к каталогу документов и файлу презентации:

```csharp
string dataDir = @"YOUR_DOCUMENT_DIRECTORY";
string pptxFile = Path.Combine(dataDir, "BulletData.pptx");
```

*Объяснение:* The `dataDir` переменная хранит путь к вашим документам, в то время как `pptxFile` объединяет это с конкретным именем файла презентации.

#### Шаг 2: Загрузите файл презентации

Загрузите файл PowerPoint с помощью Aspose.Slides:

```csharp
using (Presentation pres = new Presentation(pptxFile))
{
    // Доступ к первой фигуре первого слайда, которая, как ожидается, будет автофигурой
    AutoShape autoShape = (AutoShape)pres.Slides[0].Shapes[0];
}
```

*Объяснение:* The `Presentation` объект инициализируется с вашим файлом, и вы получаете доступ к целевой форме, используя ее индекс.

#### Шаг 3: Повторите абзацы

Пройдитесь по каждому абзацу в текстовом фрейме:

```csharp
foreach (Paragraph para in autoShape.TextFrame.Paragraphs)
{
    // Получить эффективные данные формата маркера для каждого абзаца
    IBulletFormatEffectiveData bulletFormatEffective = para.ParagraphFormat.Bullet.GetEffective();
}
```

*Объяснение:* Этот цикл обрабатывает каждый абзац, выбирая эффективный формат маркера.

#### Шаг 4: Отображение типа заполнения маркера

Проверьте, существует ли маркер, и отобразите его тип заполнения:

```csharp
if (bulletFormatEffective.Type != BulletType.None)
{
    switch (bulletFormatEffective.FillFormat.FillType)
    {
        case FillType.Solid:
            Console.WriteLine("Solid fill color: " + bulletFormatEffective.FillFormat.SolidFillColor);
            break;
        case FillType.Gradient:
            Console.WriteLine("Gradient stops count: " +
                              bulletFormatEffective.FillFormat.GradientFormat.GradientStops.Count);
            foreach (IGradientStopEffectiveData gradStop in bulletFormatEffective.FillFormat.GradientFormat.GradientStops)
                Console.WriteLine(gradStop.Position + ": " + gradStop.Color);
            break;
        case FillType.Pattern:
            Console.WriteLine("Pattern style: " +
                              bulletFormatEffective.FillFormat.PatternFormat.PatternStyle);
            Console.WriteLine("Fore color: " +
                              bulletFormatEffective.FillFormat.PatternFormat.ForeColor);
            Console.WriteLine("Back color: " +
                              bulletFormatEffective.FillFormat.PatternFormat.BackColor);
            break;
    }
}
```

*Объяснение:* В зависимости от типа заливки (сплошная, градиентная, узорчатая) отображаются различные свойства.

### Советы по устранению неполадок

- **Распространенная проблема:** Убедитесь, что в вашем файле презентации есть хотя бы один слайд с текстовой рамкой, содержащей маркеры.
- **Отладка:** Используйте контрольные точки для пошагового прохождения каждого абзаца и проверки его содержимого перед доступом к данным маркера.

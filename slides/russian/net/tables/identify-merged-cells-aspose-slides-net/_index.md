### Обзор функций: определение объединенных ячеек

Эта функция позволяет программно определить, какие ячейки в таблице являются частью группы слияния. Это особенно полезно при манипулировании или анализе данных из сложных презентаций.

#### Пошаговая реализация

**1. Загрузите презентацию**
Начните с загрузки презентации PowerPoint, содержащей таблицу:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
using (Presentation pres = new Presentation(dataDir + "/SomePresentationWithTable.pptx"))
{
    // Открываем первый слайд и предполагаем, что первая фигура — это таблица.
    ITable table = pres.Slides[0].Shapes[0] as ITable;

    // Дальнейшие шаги будут описаны здесь...
}
```

**2. Итерация по ячейкам таблицы**
Пройдитесь по каждой ячейке таблицы, чтобы определить, является ли она частью объединенной ячейки:
```csharp
for (int i = 0; i < table.Rows.Count; i++)
{
    for (int j = 0; j < table.Columns.Count; j++)
    {
        ICell currentCell = table.Rows[i][j];

        // Проверьте, является ли текущая ячейка частью объединенной ячейки.
        if (currentCell.IsMergedCell)
        {
            Console.WriteLine(string.Format(
                "Cell {0};{1} is part of a merged cell with RowSpan={2} and ColSpan={3}, starting from Cell {4};{5}.",
                i, j,
                currentCell.RowSpan,
                currentCell.ColSpan,
                currentCell.FirstRowIndex,
                currentCell.FirstColumnIndex));
        }
    }
}
```

**Объяснение:**
- **`IsMergedCell`:** Определяет, является ли ячейка частью объединенной группы.
- **`RowSpan` и `ColSpan`:** Указывает охват объединенной ячейки по строкам и столбцам соответственно.
- **Исходная позиция:** Определяет, где начинается слияние.

#### Советы по устранению неполадок

- Убедитесь, что путь к файлу презентации указан правильно, чтобы избежать ошибок «файл не найден».
- Убедитесь, что структура таблицы на слайде соответствует вашим предположениям (например, это действительно первая форма).

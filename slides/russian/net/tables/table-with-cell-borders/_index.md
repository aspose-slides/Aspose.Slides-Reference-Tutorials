# Создание таблиц без границ в презентациях PowerPoint

Aspose.Slides для .NET — это мощная библиотека, которая позволяет разработчикам создавать, изменять и преобразовывать презентации PowerPoint программным способом. В этом примере показано, как создать таблицу в презентации и настроить ее внешний вид, удалив все границы ячеек.

Код создает новую презентацию с таблицей на первом слайде, затем проходит по каждой ячейке, чтобы установить все границы (верхнюю, нижнюю, левую и правую) без заливки, фактически создавая таблицу без границ.

## Инструкция по установке

1. Установка через менеджер пакетов NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Или установите через .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Добавьте пространство имен в свой код:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

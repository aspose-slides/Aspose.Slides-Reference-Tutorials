## Руководство по внедрению

### Форматирование текста в строках таблицы

#### Обзор
Эта функция позволяет улучшить читаемость и выравнивание текста в ячейках таблицы. Мы сосредоточимся на настройке высоты шрифта, выравнивания текста, правого поля и вертикальной ориентации текста.

#### Пошаговая реализация

##### Установка высоты шрифта для ячеек
1. **Инициализировать презентацию**
   ```csharp
   using Aspose.Slides;
   
   Presentation presentation = new Presentation("YOUR_DOCUMENT_DIRECTORY\SomePresentationWithTable.pptx");
   ISlide slide = presentation.Slides[0];
   ITable someTable = slide.Shapes[0] as ITable; // Предположим, что первая фигура — это стол.
   ```

2. **Настроить высоту шрифта**
   ```csharp
   PortionFormat portionFormat = new PortionFormat();
   portionFormat.FontHeight = 25; // Установите желаемую высоту шрифта
   someTable.Rows[0].SetTextFormat(portionFormat);
   ```
   - **Цель**: Изменяет размер шрифта в ячейках таблицы для улучшения читабельности.

##### Настройка выравнивания текста и правого поля
3. **Настроить формат абзаца**
   ```csharp
   ParagraphFormat paragraphFormat = new ParagraphFormat();
   paragraphFormat.Alignment = TextAlignment.Right; // Выровнять текст по правому краю
   paragraphFormat.MarginRight = 20; // Установить правое поле размером 20 единиц
   someTable.Rows[0].SetTextFormat(paragraphFormat);
   ```
   - **Цель**: Обеспечивает единообразное выравнивание и интервалы внутри ячеек.

##### Настройка вертикального типа текста
4. **Применить вертикальное форматирование текста**
   ```csharp
   TextFrameFormat textFrameFormat = new TextFrameFormat();
   textFrameFormat.TextVerticalType = TextVerticalType.Vertical; // Установить вертикальную ориентацию текста
   someTable.Rows[1].SetTextFormat(textFrameFormat);
   ```
   - **Цель**: Полезно для создания уникальных дизайнов и экономии места в презентациях.

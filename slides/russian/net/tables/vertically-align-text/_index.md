# Создание таблиц с вертикальным текстом в презентациях PowerPoint

Aspose.Slides для .NET предоставляет мощные возможности для программного создания и управления презентациями PowerPoint. Этот пример демонстрирует, как создать таблицу с настраиваемым форматированием, включая вертикальное выравнивание текста и поворот — общее требование для специализированных бизнес-презентаций и визуализации данных.

## Инструкция по установке

1. Установите Aspose.Slides для .NET через диспетчер пакетов NuGet:
   ```
   Install-Package Aspose.Slides
   ```
   Или через .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

2. Добавьте ссылки на ваш проект:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

3. Загрузите документационные ресурсы:
   - [Ссылка на API](https://reference.aspose.com/slides/net/)
   - [Руководство разработчика](https://docs.aspose.com/slides/net/)

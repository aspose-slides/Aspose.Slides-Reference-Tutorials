### Руководство по внедрению

#### Установка высоты шрифта в ячейках таблицы

Настройка высоты шрифта может сделать определенные данные заметными. Вот как это можно сделать:

**Обзор:**
Эта функция позволяет настраивать размер шрифта в ячейках таблицы, улучшая читабельность и визуальную привлекательность.

1. **Инициализировать объект презентации**
   
   ```csharp
   string dataDir = "YOUR_DOCUMENT_DIRECTORY";
   Presentation presentation = new Presentation(dataDir + "pres.pptx");
   ```

2. **Доступ к слайду и таблице**
   
   ```csharp
   ISlide slide = presentation.Slides[0];
   ITable someTable = (ITable)slide.Shapes[0];
   ```

3. **Установить высоту шрифта**
   
   Создать `PortionFormat` объект для определения свойств шрифта:
   
   ```csharp
   PortionFormat portionFormat = new PortionFormat { FontHeight = 25 };
   someTable.SetTextFormat(portionFormat);
   ```

4. **Сохранить презентацию**
   
   ```csharp
   presentation.Save(dataDir + "result_font_height.pptx", SaveFormat.Pptx);
   ```

#### Выравнивание текста и установка правого поля в ячейках таблицы

Выравнивание текста и определение полей имеют важное значение для структурированных презентаций.

**Обзор:**
Эта функция позволяет выравнивать текст по правому краю и устанавливать определенное правое поле в ячейках таблицы.

1. **Инициализировать объект презентации**
   
   ```csharp
   Presentation presentation = new Presentation(dataDir + "pres.pptx");
   ```

2. **Доступ к слайду и таблице**
   
   ```csharp
   ISlide slide = presentation.Slides[0];
   ITable someTable = (ITable)slide.Shapes[0];
   ```

3. **Установить выравнивание текста и поля**
   
   Используйте `ParagraphFormat` объект:
   
   ```csharp
   ParagraphFormat paragraphFormat = new ParagraphFormat { 
       Alignment = TextAlignment.Right, 
       MarginRight = 20 
   };
   someTable.SetTextFormat(paragraphFormat);
   ```

4. **Сохранить презентацию**
   
   ```csharp
   presentation.Save(dataDir + "result_text_alignment.pptx", SaveFormat.Pptx);
   ```

#### Настройка вертикального типа текста в ячейках таблицы

Вертикальная ориентация текста может придать вашим презентациям уникальный вид.

**Обзор:**
Эта функция позволяет устанавливать вертикальную ориентацию текста в ячейках таблицы, что полезно для креативных или языковых макетов.

1. **Инициализировать объект презентации**
   
   ```csharp
   Presentation presentation = new Presentation(dataDir + "pres.pptx");
   ```

2. **Доступ к слайду и таблице**
   
   ```csharp
   ISlide slide = presentation.Slides[0];
   ITable someTable = (ITable)slide.Shapes[0];
   ```

3. **Установить вертикальную ориентацию текста**
   
   Создать `TextFrameFormat` объект:
   
   ```csharp
   TextFrameFormat textFrameFormat = new TextFrameFormat { 
       TextVerticalType = TextVerticalType.Vertical 
   };
   someTable.SetTextFormat(textFrameFormat);
   ```

4. **Сохранить презентацию**
   
   ```csharp
   presentation.Save(dataDir + "result_vertical_text.pptx", SaveFormat.Pptx);
   ```

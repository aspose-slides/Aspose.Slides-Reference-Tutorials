### Форматирование текста в столбцах таблицы
В этом разделе мы рассмотрим, как форматировать текст внутри столбцов таблицы с помощью Aspose.Slides для .NET.

#### Регулировка высоты шрифта
Сначала давайте установим высоту шрифта для ячеек в первом столбце:
```csharp
using Aspose.Slides;
using Aspose.Slides.Export;

// Предположим, что ваша презентация уже загружена как «pres»
ISlide slide = pres.Slides[0];
ITable someTable = slide.Shapes[0] as ITable; // Предположим, что стол — это первая фигура.

PortionFormat portionFormat = new PortionFormat();
portionFormat.FontHeight = 25;
someTable.Columns[0].SetTextFormat(portionFormat);
```

**Объяснение**: Здесь мы создаем `PortionFormat` объект для указания высоты шрифта текста в первом столбце.

#### Настройка выравнивания текста и полей
Далее выровняем текст по правому краю и зададим поля для ячеек первого столбца:
```csharp
ParagraphFormat paragraphFormat = new ParagraphFormat();
paragraphFormat.Alignment = TextAlignment.Right;
paragraphFormat.MarginRight = 20; // Установите отступ справа в 20 пунктов.
someTable.Columns[0].SetTextFormat(paragraphFormat);
```

**Объяснение**: `ParagraphFormat` позволяет нам определять выравнивание и поля, гарантируя аккуратное расположение текста в ячейках таблицы.

#### Применение вертикального текста
Для таблиц, требующих вертикальной ориентации текста во втором столбце:
```csharp
TextFrameFormat textFrameFormat = new TextFrameFormat();
textFrameFormat.TextVerticalType = TextVerticalType.Vertical;
someTable.Columns[1].SetTextFormat(textFrameFormat);
```

**Объяснение**: `TextFrameFormat` класс позволяет нам изменять вертикальное выравнивание текста, что имеет решающее значение для определенных эстетических требований дизайна или языка.

### Сохранение презентации
После внесения изменений сохраните презентацию:
```csharp
pres.Save("YOUR_OUTPUT_DIRECTORY\\result.pptx", SaveFormat.Pptx);
```

**Объяснение**: На этом этапе все изменения форматирования сохраняются в файловой системе в формате PPTX.

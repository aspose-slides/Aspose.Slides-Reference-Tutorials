## Руководство по внедрению

### Блокировка/разблокировка соотношения сторон
Эта функция позволяет вам сохранять размеры ваших таблиц даже при изменении размеров других элементов на слайде. Вот как это работает:
#### Шаг 1: Загрузите презентацию
Сначала загрузите файл презентации, содержащий таблицу:
```csharp
using (Presentation pres = new Presentation(dataDir + "/pres.pptx"))
{
    // Код для управления таблицей будет здесь
}
```
#### Шаг 2: Доступ к форме таблицы
Определите и выберите первую фигуру на слайде, убедившись, что это таблица:
```csharp
ITable table = (ITable)pres.Slides[0].Shapes[0];
```
#### Шаг 3: Включение/выключение блокировки соотношения сторон
Проверьте, заблокировано ли соотношение сторон в данный момент. Затем переключите его состояние на заблокированное или разблокированное:
```csharp
bool originalLockState = table.ShapeLock.AspectRatioLocked;
table.ShapeLock.AspectRatioLocked = !originalLockState; // Инвертировать текущее состояние
```
#### Шаг 4: Сохраните изменения.
Наконец, сохраните измененную презентацию в новом файле:
```csharp
pres.Save(outputPath + "/pres-out.pptx", SaveFormat.Pptx);
```
### Советы по устранению неполадок
- Убедитесь, что форма, к которой вы обращаетесь, действительно является таблицей.
- Проверьте правильность указания путей для входных и выходных файлов.
- Если изменения соотношения сторон не отражаются, проверьте, не влияют ли на размеры другие элементы слайда.
## Практические применения
Блокировка или разблокировка соотношения сторон таблиц может быть полезна в различных сценариях:

2. **Адаптивные макеты**: Отрегулируйте размеры таблиц, не искажая представление данных при изменении размера презентаций для разных размеров экрана.
3. **Автоматизированные отчеты**: Создание отчетов, в которых размеры таблиц должны оставаться неизменными независимо от изменений содержимого.

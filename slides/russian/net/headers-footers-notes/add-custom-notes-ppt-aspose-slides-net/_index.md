### Добавить пользовательские заметки к слайдам (H2)
#### Обзор:
Добавление пользовательских заметок обеспечивает дополнительный контекст или аннотации на слайдах, повышая вовлеченность и понимание.
#### Этапы реализации:
**1. Определите пути к каталогам (H3)**
Сначала укажите местоположение файлов презентации и место, куда вы хотите сохранить вывод.
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Обновите, указав путь к вашему каталогу.
string outputDir = "YOUR_OUTPUT_DIRECTORY";  // Обновите, указав желаемый путь вывода.

// Убедитесь, что каталоги существуют
bool isExists = System.IO.Directory.Exists(dataDir);
if (!isExists)
{
    System.IO.Directory.CreateDirectory(dataDir);
}
```
**2. Загрузите презентацию (H3)**
Загрузите файл PowerPoint, который вы хотите изменить, с помощью Aspose.Slides:
```csharp
Presentation presentation = new Presentation(System.IO.Path.Combine(dataDir, "YourPresentation.pptx"));
```
**3. Добавить заметки к слайду (H3)**
Добавьте пользовательские заметки к определенному слайду, открыв его `NotesSlideManager` и создание новой заметки.
```csharp
ISlide slide = presentation.Slides[0]; // Откройте первый слайд.
INotesSlide notesSlide = slide.NotesSlideManager.AddNotesSlide();

// Настройте содержимое вашей заметки здесь
notesSlide.NotesTextFrame.Text = "This is a custom note.";
```
**4. Сохраните презентацию (H3)**
После добавления заметок сохраните измененную презентацию:
```csharp
presentation.Save(System.IO.Path.Combine(outputDir, "ModifiedPresentation.pptx"), SaveFormat.Pptx);
```
### Советы по устранению неполадок:
- Убедитесь, что пути к каталогам указаны правильно, чтобы избежать ошибок «файл не найден».
- Проверьте, есть ли у вас права на запись в выходной каталог.

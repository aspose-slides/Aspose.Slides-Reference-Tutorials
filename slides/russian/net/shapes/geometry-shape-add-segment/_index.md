# Добавление сегментов в GeometryShape в презентациях PowerPoint

В этом примере показано, как получить доступ к геометрическому пути фигур в презентациях PowerPoint и изменить его с помощью Aspose.Slides for .NET. Добавляя пользовательские сегменты линий к прямоугольной фигуре, можно создавать более сложные геометрические фигуры, которые недоступны через стандартные типы фигур. Этот метод полезен для создания пользовательских диаграмм, блок-схем и других визуальных элементов в презентациях.

## Инструкция по установке

1. Установите Aspose.Slides для .NET одним из следующих способов:

   - Менеджер пакетов NuGet:
     ```
     Install-Package Aspose.Slides.NET
     ```

   - Консоль менеджера пакетов:
     ```
     PM> Install-Package Aspose.Slides.NET
     ```

   - .NET CLI:
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. Добавьте пространство имен в ваш файл C#:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

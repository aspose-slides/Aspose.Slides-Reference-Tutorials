# Создание и вращение прямоугольных фигур в PowerPoint

Aspose.Slides для .NET — это мощный API, позволяющий разработчикам создавать, изменять и конвертировать презентации PowerPoint без необходимости установки Microsoft PowerPoint. В этом примере показано, как создать презентацию с повернутой прямоугольной формой, демонстрируя возможности библиотеки по манипулированию формой.

С Aspose.Slides вы можете программно добавлять различные формы к слайдам и применять преобразования, такие как вращение. Эта функциональность особенно полезна для создания динамических презентаций с точным контролем визуальных элементов.

## Инструкция по установке

1. Установка через менеджер пакетов NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Установка через .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Добавьте директиву using в свой код:
   ```csharp
   using Aspose.Slides;
   ```

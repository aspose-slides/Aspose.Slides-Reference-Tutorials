# Создание эллиптических фигур с градиентной заливкой в PowerPoint с помощью Aspose.Slides

В этом примере показано, как программно создать презентацию PowerPoint, содержащую эллипс с пользовательским эффектом градиентной заливки. Код создает новую презентацию, добавляет эллипс к первому слайду и применяет линейную градиентную заливку с красными и фиолетовыми цветовыми точками. Aspose.Slides для .NET предоставляет комплексный API для создания и управления презентациями PowerPoint с точным контролем над формами, форматированием и эффектами.

## Инструкция по установке

1. Установите Aspose.Slides для .NET одним из следующих способов:
   - Менеджер пакетов NuGet: `Install-Package Aspose.Slides.NET`
   - .NET CLI: `dotnet add package Aspose.Slides.NET`
   - Консоль менеджера пакетов: `Install-Package Aspose.Slides.NET`

2. Добавьте следующий оператор using в ваш файл C#:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

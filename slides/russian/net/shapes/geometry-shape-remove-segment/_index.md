# Изменение геометрических контуров в фигурах PowerPoint с помощью Aspose.Slides

Aspose.Slides для .NET предоставляет мощные возможности для программного изменения презентаций PowerPoint, включая расширенные манипуляции с фигурами. Этот пример демонстрирует, как создавать пользовательские фигуры, изменяя геометрический контур существующей фигуры — в частности, удаляя сегмент из фигуры сердца, чтобы создать уникальный вариант.

Геометрический контур определяет контур фигуры через ряд точек и сегментов. Получая доступ к этим контурам и изменяя их, разработчики могут создавать высоконастраиваемые фигуры, которые недоступны в стандартной библиотеке фигур PowerPoint.

## Инструкция по установке

1. Установка через менеджер пакетов NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Установка через .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Добавьте директиву using в свой код:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

4. Добавьте инициализацию лицензии при запуске приложения:
   ```csharp
   // Если у вас есть лицензия
   License license = new License();
   license.SetLicense("path/to/Aspose.Slides.lic");
   ```

# Создание фигур с мозаичной заливкой изображения в PowerPoint с помощью Aspose.Slides

Aspose.Slides for .NET предоставляет мощные возможности для создания и управления презентациями PowerPoint программным способом. В этом примере показано, как создать прямоугольную форму и заполнить ее мозаичным узором изображения. Мозаичная заливка изображения особенно полезна для создания текстурированных фонов, водяных знаков или декоративных элементов в ваших презентациях.

## Инструкция по установке

1. Установка через менеджер пакетов NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Установка через .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Ручная установка:
   - Загрузите последнюю версию с сайта [Aspose.Slides для .NET](https://releases.aspose.com/slides/net/)
   - Добавьте ссылку на Aspose.Slides.dll в свой проект

## Ресурсы

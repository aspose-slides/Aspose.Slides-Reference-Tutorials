# Создание эллиптических фигур в PowerPoint с помощью Aspose.Slides

Aspose.Slides для .NET — это мощный API, который позволяет разработчикам создавать, читать, писать, изменять и конвертировать презентации PowerPoint без необходимости использования Microsoft PowerPoint. Этот пример демонстрирует, как создать эллиптическую форму в презентации, применить форматирование, включая цвет заливки и свойства линий, и сохранить результат в виде файла PPTX. Код демонстрирует способность Aspose.Slides манипулировать фигурами с точным контролем над их внешним видом и позиционированием.

## Инструкция по установке

1. Установка через менеджер пакетов NuGet:
   ```
   Install-Package Aspose.Slides
   ```

   Или с помощью .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

2. Добавить ссылки:
   Добавьте следующее пространство имен в ваш файл C#:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

3. Настройка лицензии (необязательно):
   Если у вас есть лицензия, подайте ее с помощью:
   ```csharp
   License license = new License();
   license.SetLicense("Aspose.Slides.lic");
   ```

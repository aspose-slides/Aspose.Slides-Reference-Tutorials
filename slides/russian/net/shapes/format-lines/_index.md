# Создание прямоугольной фигуры с пользовательским форматированием линий в PowerPoint

Aspose.Slides для .NET предоставляет мощные возможности для программного создания и изменения презентаций PowerPoint. В этом примере показано, как создать прямоугольную фигуру с настраиваемым форматированием линий, включая настройку стиля линии, ширины, шаблона штрихов и цвета. Пример кода создает новую презентацию, добавляет прямоугольник к первому слайду и применяет определенное форматирование как к заливке, так и к контуру фигуры, в результате чего получается визуально привлекательный элемент с белой заливкой и синим пунктирным контуром.

## Инструкция по установке

1. Установите Aspose.Slides для .NET одним из следующих способов:

   - Менеджер пакетов NuGet:
     ```
     Install-Package Aspose.Slides
     ```

   - Консоль менеджера пакетов:
     ```
     PM> Install-Package Aspose.Slides
     ```

   - .NET CLI:
     ```
     dotnet add package Aspose.Slides
     ```

2. Добавьте необходимое пространство имен в ваш файл C#:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

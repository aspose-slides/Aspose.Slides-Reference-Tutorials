# Выравнивание и группировка фигур в презентациях PowerPoint

Aspose.Slides for .NET предоставляет мощные возможности для создания и управления презентациями PowerPoint программным способом. Этот пример демонстрирует, как выравнивать фигуры на слайдах и работать с сгруппированными фигурами в презентациях PowerPoint. Вы узнаете, как размещать прямоугольники в определенных координатах, выравнивать их относительно слайда или друг друга и выборочно выравнивать фигуры в группах.

## Инструкция по установке

1. Установите Aspose.Slides для .NET одним из следующих способов:
   
   - Менеджер пакетов NuGet:
     ```
     Install-Package Aspose.Slides.NET
     ```
   
   - Консоль менеджера пакетов:
     ```
     PM> Install-Package Aspose.Slides.NET
     ```
   
   - .NET CLI:
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. Добавьте ссылки в свой проект:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.IO;
   ```

3. Загрузите пример кода ниже и интегрируйте его в свой проект.

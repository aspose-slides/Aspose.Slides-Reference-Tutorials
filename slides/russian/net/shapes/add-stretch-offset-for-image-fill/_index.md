# Добавление изображений в слайды PowerPoint с помощью Aspose.Slides

Aspose.Slides for .NET предоставляет мощные возможности для программного создания и изменения презентаций PowerPoint. Этот пример демонстрирует, как загрузить внешнее изображение и добавить его в слайд презентации в качестве рамки изображения — обычное требование при создании отчетов, создании шаблонов или динамическом построении презентаций из источников данных.

Приведенный ниже код обрабатывает все, от создания новой презентации до добавления изображения и сохранения финального файла PPTX, и все это с помощью всего лишь нескольких строк кода C#.

## Инструкция по установке

1. Установите Aspose.Slides для .NET одним из следующих способов:
   - Менеджер пакетов NuGet: `Install-Package Aspose.Slides.NET`
   - .NET CLI: `dotnet add package Aspose.Slides.NET`
   - Или загрузите напрямую с [Сайт Aspose](https://releases.aspose.com/slides/net/)

2. Добавьте пространство имен в ваш файл C#:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

# Создание групповых фигур в PowerPoint с помощью Aspose.Slides

Aspose.Slides для .NET предоставляет комплексные API для создания и управления презентациями PowerPoint программным способом. В этом примере показано, как создать групповую фигуру, добавить в нее несколько прямоугольных фигур, правильно расположить фигуры, добавить рамку и сохранить презентацию. Групповые фигуры необходимы для организации связанных фигур, которые необходимо обрабатывать как единое целое, обеспечивая большую структуру и контроль над элементами презентации.

## Инструкция по установке

1. Установите Aspose.Slides для .NET одним из следующих способов:

   - Менеджер пакетов NuGet:
     ```
     Install-Package Aspose.Slides.NET
     ```

   - Консоль менеджера пакетов:
     ```
     PM> Install-Package Aspose.Slides.NET
     ```

   - .NET CLI:
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. Добавьте ссылки на ваш проект:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

3. Настройте лицензию (необязательно для производственного использования):
   ```csharp
   // Установить лицензию для снятия ограничений на оценку
   License license = new License();
   license.SetLicense("Aspose.Slides.lic");
   ```

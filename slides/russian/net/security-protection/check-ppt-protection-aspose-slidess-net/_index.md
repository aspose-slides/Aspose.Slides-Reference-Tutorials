## Руководство по внедрению
Изучите различные функции, уделяя особое внимание проверкам защиты PowerPoint с использованием C#.

### Функция 1: Проверка защиты презентации от записи через интерфейс IPresentationInfo
**Обзор:**
Определите, защищена ли презентация от записи, используя `IPresentationInfo` интерфейс, ориентированный на защиту на основе пароля.

#### Пошаговая реализация
**Шаг 1: Определите путь к файлу**
Определите и укажите каталог файла вашей презентации:
```csharp
string pptxFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "modify_pass2.pptx");
```

**Шаг 2: Получите информацию о презентации**
Использовать `PresentationFactory` для доступа к деталям:
```csharp
IPresentationInfo presentationInfo = PresentationFactory.Instance.GetPresentationInfo(pptxFile);
```

**Шаг 3: Проверьте статус защиты от записи**
Проверьте, защищен ли файл паролем, и подтвердите его:
```csharp
bool isWriteProtectedByPassword = presentationInfo.IsWriteProtected == NullableBool.True &&
                                   presentationInfo.CheckWriteProtection("pass2");
```

### Функция 2: Проверка защиты презентации от записи через интерфейс IProtectionManager
**Обзор:**
Эта функция позволяет проверить, защищена ли презентация от записи с помощью `IProtectionManager` интерфейс.

#### Пошаговая реализация
**Шаг 1: Откройте презентацию**
Загрузите файл презентации:
```csharp
using (var presentation = new Presentation(pptxFile))
{
    // Продолжить проверку
}
```

**Шаг 2: Проверка защиты от записи**
Проверьте, активна ли защита от записи, и подтвердите ее с помощью пароля:
```csharp
bool isWriteProtected = presentation.ProtectionManager.CheckWriteProtection("pass2");
```

### Функция 3: Проверка защиты открытия презентации через интерфейс IPresentationInfo
**Обзор:**
Этот метод проверяет, требуется ли пароль для открытия файла PowerPoint.

#### Пошаговая реализация
**Шаг 1: Определите путь к файлу**
Укажите путь к вашей защищенной презентации:
```csharp
string pptFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "open_pass1.ppt");
```

**Шаг 2: Извлечение информации о презентации**
Доступ к информации с помощью `IPresentationInfo`:
```csharp
IPresentationInfo presentationInfo = PresentationFactory.Instance.GetPresentationInfo(pptFile);
```

**Шаг 3: Определите статус открытой защиты**
Проверьте, защищен ли файл паролем от открытия:
```csharp
if (presentationInfo.IsPasswordProtected)
{
    // Для открытия файла требуется пароль.
}
```

# Сохранение оригинальных шрифтов — преобразование презентации в HTML


В этом подробном руководстве мы проведем вас через процесс сохранения исходных шрифтов при конвертации презентации в HTML с помощью Aspose.Slides для .NET. Мы предоставим вам необходимый исходный код C# и подробно объясним каждый шаг. К концу этого руководства вы сможете гарантировать, что шрифты в вашем преобразованном HTML-документе останутся верными исходной презентации.

## 1. Введение

При конвертации презентаций PowerPoint в HTML крайне важно сохранить исходные шрифты, чтобы обеспечить визуальную согласованность вашего контента. Aspose.Slides for .NET предоставляет мощное решение для достижения этой цели. В этом руководстве мы проведем вас через шаги, необходимые для сохранения исходных шрифтов в процессе конвертации.

## 2. Предпосылки

Прежде чем начать, убедитесь, что у вас выполнены следующие предварительные условия:

- Visual Studio установлена на вашем компьютере.
- Библиотека Aspose.Slides для .NET добавлена в ваш проект.

## 3. Настройка вашего проекта

Для начала создайте новый проект в Visual Studio и добавьте библиотеку Aspose.Slides для .NET в качестве ссылки.

## 4. Загрузка презентации

Используйте следующий код для загрузки презентации PowerPoint:

```csharp
string dataDir = "Your Document Directory";

using (Presentation pres = new Presentation("input.pptx"))
{
    // Ваш код здесь
}
```

Заменять `"Your Document Directory"` с путем к файлу вашей презентации.

## 5. Исключение шрифтов по умолчанию

Чтобы исключить шрифты по умолчанию, такие как Calibri и Arial, используйте следующий код:

```csharp
string[] fontNameExcludeList = { "Calibri", "Arial" };
```

Вы можете настроить этот список по мере необходимости.

## 6. Внедрение всех шрифтов

Далее мы встроим все шрифты в HTML-документ. Это гарантирует сохранение исходных шрифтов. Используйте следующий код:

```csharp
EmbedAllFontsHtmlController embedFontsController = new EmbedAllFontsHtmlController(fontNameExcludeList);

HtmlOptions htmlOptionsEmbed = new HtmlOptions
{
    HtmlFormatter = HtmlFormatter.CreateCustomFormatter(embedFontsController)
};
```

## 7. Сохранение в формате HTML

Теперь сохраните презентацию как HTML-документ со встроенными шрифтами:

```csharp
pres.Save("output.html", SaveFormat.Html, htmlOptionsEmbed);
```

Заменять `"output.html"` с желаемым именем выходного файла.

## 8. Заключение

В этом уроке мы продемонстрировали, как сохранить оригинальные шрифты при конвертации презентации PowerPoint в HTML с помощью Aspose.Slides for .NET. Выполнив эти шаги, вы можете быть уверены, что ваш конвертированный HTML-документ сохранит визуальную целостность исходной презентации.

## 9. Часто задаваемые вопросы

### В1: Могу ли я настроить список исключенных шрифтов?

Да, можно. Изменить `fontNameExcludeList` массив для включения или исключения определенных шрифтов в соответствии с вашими требованиями.

### В2: Что делать, если я не хочу встраивать все шрифты?

Если вы хотите встроить только определенные шрифты, вы можете изменить код соответствующим образом. Ознакомьтесь с документацией Aspose.Slides for .NET для получения более подробной информации.

### В3: Существуют ли какие-либо лицензионные требования для использования Aspose.Slides для .NET?

Да, вам может понадобиться действующая лицензия для использования Aspose.Slides for .NET в ваших проектах. Обратитесь на сайт Aspose для получения информации о лицензировании.

### В4: Могу ли я конвертировать другие форматы файлов в HTML с помощью Aspose.Slides для .NET?

Aspose.Slides for .NET в первую очередь фокусируется на презентациях PowerPoint. Для преобразования других форматов файлов в HTML вам может потребоваться изучить другие продукты Aspose, предназначенные для этих форматов.

### В5: Где я могу получить доступ к дополнительным ресурсам и поддержке?

Дополнительную документацию, руководства и поддержку можно найти на веб-сайте Aspose. Посетить [Документация Aspose.Slides для .NET](https://reference.aspose.com/slides/net/) для получения подробной информации.

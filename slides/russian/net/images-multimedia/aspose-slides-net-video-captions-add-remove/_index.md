## Руководство по внедрению
Этот раздел проведет вас через добавление субтитров к видеокадрам и их извлечение или удаление. Каждая функция подробно описана ниже.

### Функция 1: Добавление субтитров к видеокадру

#### Обзор
Эта функция позволяет вставлять субтитры из внешнего файла (например, VTT) в видеокадр, повышая доступность для вашей аудитории.

#### Этапы внедрения
**Шаг 1: Подготовьте файлы**
Убедитесь, что у вас есть видео (`sample_bunny.mp4`) и файлы дорожек субтитров (`bunny.vtt`).

```csharp
string mediaFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample_bunny.mp4");
string trackFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "bunny.vtt");
```

**Шаг 2: Добавьте видео в презентацию**
Создать `Presentation` объект и добавьте свое видео.

```csharp
using (Presentation pres = new Presentation())
{
    IVideo video = pres.Videos.AddVideo(File.ReadAllBytes(mediaFile));
    var videoFrame = pres.Slides[0].Shapes.AddVideoFrame(0, 0, 100, 100, video);
```

**Шаг 3: Добавьте дорожку субтитров**
Прикрепите файл с субтитрами к видеокадру.

```csharp
videoFrame.CaptionTracks.Add("New track", trackFile);
pres.Save(Path.Combine("YOUR_OUTPUT_DIRECTORY", "VideoCaptionAdd_out.pptx"), SaveFormat.Pptx);
}
```

#### Параметры и цели метода
- `Presentation`: Представляет собой презентацию PowerPoint.
- `IVideo` и `IVideoFrame`: представляет видеоконтент и его кадр на слайдах соответственно.
- `captionTracks.Add()`: Добавляет субтитры к указанной дорожке.

### Функция 2: Извлечение и удаление титров из видеокадра

#### Обзор
После добавления субтитров могут возникнуть ситуации, когда их нужно извлечь или удалить. Эта функция фокусируется на том, как эффективно выполнить обе задачи.

#### Этапы внедрения
**Шаг 1: Загрузка презентации**
Откройте презентацию, содержащую ваше видео с субтитрами.

```csharp
string outAddPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "VideoCaptionAdd_out.pptx");
using (Presentation pres = new Presentation(outAddPath))
{
    IVideoFrame videoFrame = pres.Slides[0].Shapes[0] as VideoFrame;
```

**Шаг 2: Извлечение субтитров**
Извлечь двоичные данные субтитров и сохранить их в файл.

```csharp
if (videoFrame != null)
{
    foreach (var captionTrack in videoFrame.CaptionTracks) 
    {
        File.WriteAllBytes(Path.Combine("YOUR_OUTPUT_DIRECTORY", "Caption_out.vtt"), captionTrack.BinaryData);
    }
```

**Шаг 3: Удалить субтитры**
Удалить все субтитры из видеокадра.

```csharp
videoFrame.CaptionTracks.Clear();
pres.Save(Path.Combine("YOUR_OUTPUT_DIRECTORY", "VideoCaptionRemove_out.pptx"), SaveFormat.Pptx);
}
```

#### Параметры и цели метода
- `BinaryData`: Представляет данные субтитров в двоичном формате.
- `CaptionTracks.Clear()`: Удаляет все субтитры из видеокадра.

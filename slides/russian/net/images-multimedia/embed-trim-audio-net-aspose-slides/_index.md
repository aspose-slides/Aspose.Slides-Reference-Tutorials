## Руководство по внедрению

### Добавление аудиофрейма со встроенным звуком

#### Обзор
Встраивайте аудиофайлы непосредственно в слайды презентации для удобства просмотра.

#### Шаги:
1. **Инициализировать презентацию**
   Создать новый `Presentation` объект для хранения слайдов и мультимедиа.
   ```csharp
   using Aspose.Slides;
   string mediaFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "audio.m4a");
   string outPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "AudioFrame_out.pptx");
   using (Presentation pres = new Presentation())
   ```
2. **Добавить аудио в коллекцию**
   Использовать `pres.Audios.AddAudio` чтобы добавить свой аудиофайл.
   ```csharp
   IAudio audio = pres.Audios.AddAudio(File.ReadAllBytes(mediaFile));
   ```
3. **Встроить аудиокадр**
   Добавьте встроенный аудиокадр на первый слайд.
   ```csharp
   IAudioFrame audioFrame = pres.Slides[0].Shapes.AddAudioFrameEmbedded(50, 50, 100, 100, audio);
   ```
4. **Сохранить презентацию**
   Сохраните свою презентацию со встроенным аудиокадром.
   ```csharp
   pres.Save(outPath, SaveFormat.Pptx);
   ```

### Установка времени обрезки звука

#### Обзор
Укажите, какая часть аудиофайла должна воспроизводиться в презентации.

#### Шаги:
1. **Инициализировать презентацию**
   Подобно добавлению аудиофрейма, начните с создания нового `Presentation` объект.
   ```csharp
   using Aspose.Slides;
   string mediaFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "audio.m4a");
   string outPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "AudioFrameTrim_out.pptx");
   using (Presentation pres = new Presentation())
   ```
2. **Добавить аудио и встроить фрейм**
   Добавьте аудио в коллекцию и вставьте его в слайд, как и раньше.
   ```csharp
   IAudio audio = pres.Audios.AddAudio(File.ReadAllBytes(mediaFile));
   IAudioFrame audioFrame = pres.Slides[0].Shapes.AddAudioFrameEmbedded(50, 50, 100, 100, audio);
   ```
3. **Обрезать начало и конец аудио**
   Установите время начала и окончания аудиоклипа.
   ```csharp
   // Обрезка с начала на 500 мс (0,5 секунды)
   audioFrame.TrimFromStart = 500f;
   
   // Обрезать до конца на 1000 мс (1 секунда)
   audioFrame.TrimFromEnd = 1000f;
   ```
4. **Сохранить презентацию**
   Сохраните презентацию с обрезанным звуком.
   ```csharp
   pres.Save(outPath, SaveFormat.Pptx);
   ```

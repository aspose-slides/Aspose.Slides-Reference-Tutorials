### Обзор
Эта функция позволяет создавать изображения слайдов презентаций в нестандартных размерах, что необходимо для целей отображения или интеграции приложений.

#### Шаг 2: Получите доступ к желаемому слайду
Откройте слайд, который вы хотите преобразовать:
```csharp
// Доступ к первому слайду
ISlide sld = pres.Slides[0];
```

#### Шаг 3: Определите размеры и рассчитайте коэффициенты масштабирования
Задайте желаемые размеры изображения, затем рассчитайте коэффициенты масштабирования:
```csharp
int desiredX = 1200;
int desiredY = 800;

float ScaleX = (float)(1.0 / pres.SlideSize.Size.Width) * desiredX;
float ScaleY = (float)(1.0 / pres.SlideSize.Size.Height) * desiredY;
```

#### Шаг 4: Создайте и сохраните масштабированное изображение.
Создайте изображение на основе слайда, используя коэффициенты масштабирования:
```csharp
IImage img = sld.GetThumbnail(ScaleX, ScaleY);

string outputDir = "YOUR_OUTPUT_DIRECTORY";
Directory.CreateDirectory(outputDir); // Убедитесь, что каталог существует
img.Save(Path.Combine(outputDir, "Thumbnail2_out.jpg"), System.Drawing.Imaging.ImageFormat.Jpeg);
```

### Основные параметры конфигурации
- **Формат изображения**: Сохраняйте изображения в различных форматах, таких как JPEG, PNG или BMP, изменяя `ImageFormat`.
- **Управление каталогами**: Убедитесь, что выходной каталог существует, чтобы избежать ошибок.

2. **Изображения, готовые к печати**: Создание изображений с индивидуальными размерами, подходящими для печатных материалов, таких как брошюры.

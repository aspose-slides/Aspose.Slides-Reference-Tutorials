## Руководство по внедрению

Теперь, когда вы настроены, давайте углубимся во встраивание аудиокадров в слайды PowerPoint. Мы пройдем каждый шаг, чтобы обеспечить ясность и понимание.

### Добавить аудиокадр со встроенным звуком

#### Обзор
Встраивание аудиокадра включает в себя несколько ключевых шагов: загрузку медиафайла, создание аудиокадра и настройку его свойств для оптимального отображения во время презентации.

#### Шаг 1: Загрузите медиафайл
Сначала определите путь к вашему аудиофайлу:

```csharp
string mediaFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "your_audio_file.mp3");
```

Убедитесь, что `mediaFile` указывает на допустимое местоположение, содержащее нужный вам аудиофайл.

#### Шаг 2: Создание аудиокадра
Далее мы добавим аудиокадр к слайду. Это включает в себя указание положения и размера кадра:

```csharp
// Добавить пустой слайд в презентацию
ISlide slide = presentation.Slides.AddEmptySlide(presentation.LayoutSlides[0]);

// Загрузите медиафайл в поток
using FileStream audioStream = new FileStream(mediaFile, FileMode.Open);

// Добавить аудиокадр к слайду в позицию (x: 50, y: 150) с шириной и высотой 100 пикселей.
IAudioFrame audioFrame = slide.Shapes.AddAudioFrameEmbedded(50, 150, 100, 100, audioStream);
```

#### Шаг 3: Настройте свойства аудиокадра
Настройте параметры воспроизведения в соответствии с вашими потребностями:

```csharp
// Установить режим воспроизведения звука и громкость
audioFrame.PlayMode = AudioPlayModePreset.Auto;
audioFrame.Volume = AudioVolumeMode.Low;

// При желании можно установить здесь изображение постера или другие свойства.
```

#### Советы по устранению неполадок
- **Распространенная проблема**: Убедитесь, что путь к медиафайлу указан правильно, чтобы избежать `FileNotFoundException`.
- **Звук не воспроизводится**Убедитесь, что настройки звука (например, громкость) настроены правильно.

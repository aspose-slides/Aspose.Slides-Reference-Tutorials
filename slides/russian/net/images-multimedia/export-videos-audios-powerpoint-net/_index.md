## Руководство по внедрению

### Экспорт видео из презентации
#### Обзор
Эта функция позволяет извлекать видеофайлы, встроенные в презентацию PowerPoint, не загружая весь файл в память, что оптимизирует производительность.

#### Пошаговое руководство
**1. Настройте параметры загрузки**
```csharp
LoadOptions loadOptions = new LoadOptions
{
    BlobManagementOptions =
    {
        PresentationLockingBehavior = PresentationLockingBehavior.KeepLocked,
    }
};
```
The `PresentationLockingBehavior.KeepLocked` опция предотвращает загрузку всего файла в память, что имеет решающее значение для обработки больших презентаций.

**2. Доступ к видео и их извлечение**
```csharp
using (Presentation pres = new Presentation(hugePresentationWithAudiosAndVideosFile, loadOptions))
{
    byte[] buffer = new byte[8 * 1024]; // Размер буфера 8КБ

    for (var index = 0; index < pres.Videos.Count; index++)
    {
        IVideo video = pres.Videos[index];

        using (Stream presVideoStream = video.GetStream())
        {
            using (FileStream outputFileStream = File.OpenWrite($"video{index}.avi"))
            {
                int bytesRead;
                while ((bytesRead = presVideoStream.Read(buffer, 0, buffer.Length)) > 0)
                {
                    outputFileStream.Write(buffer, 0, bytesRead);
                }
            }
        }
    }
}
```
**Объяснение:**
- **Размер буфера**: Мы используем буфер размером 8 КБ для чтения и записи данных порциями, что минимизирует использование памяти.
- **Цикл извлечения видео**: Просматривает каждое видео, встроенное в презентацию, извлекает его как поток и записывает в файл.

#### Советы по устранению неполадок
- Убедитесь, что у вас есть соответствующие права на чтение/запись для вашего целевого каталога.
- Убедитесь, что путь к файлу презентации правильный и доступный.

### Экспорт аудио из презентации
#### Обзор
Подобно видео, эта функция позволяет эффективно извлекать аудиофайлы, встроенные в презентации PowerPoint.

#### Пошаговое руководство
**1. Настройте параметры загрузки**
Этот шаг идентичен процессу извлечения видео:
```csharp
LoadOptions loadOptions = new LoadOptions
{
    BlobManagementOptions =
    {
        PresentationLockingBehavior = PresentationLockingBehavior.KeepLocked,
    }
};
```
**2. Доступ к аудиофайлам и их извлечение**
```csharp
using (Presentation pres = new Presentation(hugePresentationWithAudiosAndVideosFile, loadOptions))
{
    byte[] buffer = new byte[8 * 1024]; // Размер буфера 8КБ

    for (var index = 0; index < pres.Audios.Count; index++)
    {
        IAudio audio = pres.Audios[index];

        using (Stream presAudioStream = audio.GetStream())
        {
            using (FileStream outputFileStream = File.OpenWrite($"audio{index}.wav"))
            {
                int bytesRead;
                while ((bytesRead = presAudioStream.Read(buffer, 0, buffer.Length)) > 0)
                {
                    outputFileStream.Write(buffer, 0, bytesRead);
                }
            }
        }
    }
}
```
**Объяснение:**
Логика реализации зеркальна логике извлечения видео. Она перебирает аудиофайлы и записывает их на диск, используя буферизованный подход.

#### Советы по устранению неполадок
- Убедитесь, что пути к аудиофайлам указаны правильно.
- Убедитесь, что для извлеченных аудиофайлов достаточно места.

## Руководство по внедрению
### Добавить видеокадр из веб-источника
В этом разделе вы узнаете, как добавить видеокадр YouTube в презентацию PowerPoint.

#### Обзор
Встраивание видео может превратить статические презентации в интерактивные впечатления. С Aspose.Slides вы можете программно добавлять видеокадры и миниатюры из веб-источников, таких как YouTube.

#### Пошаговая реализация
##### 1. Определите каталог документов
Укажите, где будет сохранен ваш выходной файл:

```csharp
string dataDir = "/path/to/your/document/directory/";
```

Этот путь определяет, где `AddVideoFrameFromWebSource_out.pptx` будет находиться после сохранения.

##### 2. Создайте новый экземпляр презентации
Инициализируйте новую презентацию для работы:

```csharp
using (Presentation pres = new Presentation())
{
    // Добавьте видеокадр и сохраните презентацию
}
```
The `Presentation` Объект представляет ваш файл PowerPoint. `using` заявление гарантирует, что ресурсы будут очищены впоследствии.

##### 3. Добавьте видеокадр YouTube
Вставьте видеокадр в первый слайд презентации:

```csharp
IVideoFrame videoFrame = pres.Slides[0].Shapes.AddVideoFrame(10, 10, 427, 240,
    "https://www.youtube.com/embed/Tj75Arhq5ho");
```
Этот фрагмент кода размещает кадр в координатах (10, 10) с размерами 427x240 пикселей. Он использует URL-адрес вставки видео.

##### 4. Установите режим воспроизведения
Настройте параметры воспроизведения:

```csharp
videoFrame.PlayMode = VideoPlayModePreset.Auto;
```
Параметр `VideoPlayModePreset.Auto` автоматически воспроизводит видео при отображении слайда.

##### 5. Загрузите и установите миниатюру изображения
Получите миниатюру для вашего видеокадра с помощью веб-клиента:

```csharp
using (WebClient client = new WebClient())
{
    string thumbnailUri = "http://img.youtube.com/vi/Tj75Arhq5ho/hqdefault.jpg";
    videoFrame.PictureFormat.Picture.Image = pres.Images.AddImage(client.DownloadData(thumbnailUri));
}
```
URL-адрес миниатюры соответствует идентификатору видео YouTube. `DownloadData` Метод извлекает изображение и добавляет его в виде формата изображения к вашему видеокадру.

##### 6. Сохраните презентацию
Наконец, сохраните свою работу:

```csharp
pres.Save(dataDir + "AddVideoFrameFromWebSource_out.pptx", SaveFormat.Pptx);
```
Эта команда сохраняет вашу презентацию в формате PPTX в указанном месте.

#### Советы по устранению неполадок
- **Видео не воспроизводится:** Убедитесь, что URL-адрес видео правильный и общедоступный.
- **Проблемы с миниатюрами:** Убедитесь, что идентификатор видео YouTube соответствует URL-адресу миниатюры.
- **Ошибки пути к файлу:** Дважды проверьте `dataDir` path на предмет опечаток или проблем с разрешениями.

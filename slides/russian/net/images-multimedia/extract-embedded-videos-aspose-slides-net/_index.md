## Руководство по внедрению
### Извлечение встроенных видео из PowerPoint
Эта функция позволяет извлекать видео, встроенные в слайды PowerPoint. Давайте разберем шаги:
#### Обзор функций
Мы пройдемся по каждому слайду и форме, проверим наличие видеокадров, а затем извлечем и сохраним видео.
#### Пошаговая реализация
##### 1. Загрузите презентацию
Начните с загрузки файла презентации с помощью Aspose.Slides.
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
Presentation presentation = new Presentation(dataDir + "/Video.pptx");
```
##### 2. Повторяйте слайды и фигуры
Просмотрите каждый слайд, а затем каждую фигуру внутри слайдов, чтобы найти видеокадры.
```csharp
foreach (ISlide slide in presentation.Slides) {
    foreach (IShape shape in slide.Shapes) {
        if (shape is VideoFrame) {
            // Процесс видеокадра
        }
    }
}
```
##### 3. Определите и извлеките видео
Проверьте, является ли форма `VideoFrame`, извлеките его содержимое и сохраните.
```csharp
if (shape is VideoFrame vf) {
    String type = vf.EmbeddedVideo.ContentType;
    int ss = type.LastIndexOf('/');
    type = type.Remove(0, ss + 1);
    Byte[] buffer = vf.EmbeddedVideo.BinaryData;

    using (FileStream stream = new FileStream("YOUR_OUTPUT_DIRECTORY/NewVideo_out." + type, FileMode.Create, FileAccess.Write, FileShare.Read)) {
        stream.Write(buffer, 0, buffer.Length);
    }
}
```
**Объяснение:**
- **ТипСодержимого**: Определяет расширение файла видео.
- **ДвоичныеДанные**: Содержит необработанные видеоданные для извлечения.
##### Советы по устранению неполадок
- Убедитесь, что пути к каталогам установлены правильно, чтобы избежать `FileNotFoundException`.
- Если видео не извлекаются, проверьте, что формы действительно извлекаются. `VideoFrame` экземпляры.

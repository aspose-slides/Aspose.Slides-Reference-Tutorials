## Руководство по внедрению

### Включить элементы управления мультимедиа для показа слайдов
Эта функция позволяет вам контролировать видимость таких медиаэлементов, как видео и аудио, с помощью элементов управления во время презентации.

#### Обзор
Включение элементов управления мультимедиа в PowerPoint гарантирует, что ваша аудитория может останавливать, перематывать или пересылать медиаконтент прямо из своего поля зрения без необходимости использования отдельных приложений. Эта функция полезна для интерактивных сеансов, где участие пользователя имеет решающее значение.

#### Действия по включению управления мультимедиа
1. **Инициализировать класс представления**
   ```csharp
   using (Presentation pres = new Presentation())
   {
       // Код будет здесь
   }
   ```

2. **Установить свойство ShowMediaControls**
   ```csharp
   pres.SlideShowSettings.ShowMediaControls = true;
   ```
   - `pres.SlideShowSettings.ShowMediaControls`: Это свойство определяет, будут ли отображаться элементы управления мультимедиа в режиме слайд-шоу.

3. **Сохранить презентацию**
   ```csharp
   pres.Save("YOUR_DOCUMENT_DIRECTORY\\SlideShowMediaControl.pptx", SaveFormat.Pptx);
   ```

### Отключить элементы управления слайд-шоу
В сценариях, где предпочтителен бесперебойный просмотр без помех, отключение элементов управления мультимедиа может оказаться полезным.

#### Обзор
Отключение элементов управления мультимедиа помогает сохранять фокус, устраняя любые потенциальные отвлечения от экранных кнопок. Эта настройка идеально подходит для презентаций, которые должны просматриваться в непрерывном потоке без взаимодействия пользователя с элементами мультимедиа.

#### Действия по отключению управления мультимедиа
1. **Инициализировать класс представления**
   ```csharp
   using (Presentation pres = new Presentation())
   {
       // Код будет здесь
   }
   ```

2. **Установить свойство ShowMediaControls**
   ```csharp
   pres.SlideShowSettings.ShowMediaControls = false;
   ```
   - Это гарантирует, что элементы управления мультимедиа будут скрыты во время презентации, что позволит избежать отвлекающих факторов.

3. **Сохранить презентацию**
   ```csharp
   pres.Save("YOUR_DOCUMENT_DIRECTORY\\SlideShowMediaControl_Disabled.pptx", SaveFormat.Pptx);
   ```

### Советы по устранению неполадок
- Убедитесь, что ваша библиотека Aspose.Slides обновлена до последней версии.
- Убедитесь, что `outFilePath` path правильно указывает на доступный для записи каталог в вашей системе.
- Если элементы управления мультимедиа не появляются/не исчезают должным образом, еще раз проверьте совместимость фреймворка .NET вашего проекта с Aspose.Slides.

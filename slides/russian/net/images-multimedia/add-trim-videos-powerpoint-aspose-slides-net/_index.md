## Руководство по внедрению

### Добавление видео на слайд

#### Обзор
Эта функция позволяет встраивать видеофайлы непосредственно в слайды PowerPoint, повышая визуальную привлекательность и эффективность ваших презентаций.

#### Действия по добавлению видео
**Шаг 1: Подготовьте видеофайл**
Убедитесь, что ваш видеофайл (например, «Wildlife.mp4») доступен в каталоге ваших документов.

```csharp
string videoFileName = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "Wildlife.mp4");
```

**Шаг 2: Инициализация презентации и слайда**
Создайте новый объект презентации и получите доступ к первому слайду:

```csharp
using (Presentation pres = new Presentation())
{
    ISlide slide = pres.Slides[0];
```

**Шаг 3: Добавьте видео на слайд**
Добавьте ваш видеофайл в презентацию, затем вставьте его в рамку на слайде:

```csharp
IVideo video = pres.Videos.AddVideo(File.ReadAllBytes(videoFileName));
var videoFrame = slide.Shapes.AddVideoFrame(0, 0, 200, 200, video);
```

**Шаг 4: Сохраните презентацию**
Сохраните презентацию в выходном каталоге:

```csharp
pres.Save("YOUR_OUTPUT_DIRECTORY\\AddVideoOutput.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
}
```

### Установка времени начала и окончания обрезки видеокадра

#### Обзор
Эта функция позволяет вам определять время начала и окончания воспроизведения видео в вашей презентации, гарантируя показ только соответствующих разделов.

#### Действия по обрезке видео при воспроизведении
**Шаг 1: Инициализация презентации**
Инициализируйте объект презентации, как и прежде:

```csharp
using (Presentation pres = new Presentation())
{
    ISlide slide = pres.Slides[0];
```

**Шаг 2: Добавьте и настройте видеокадр**
Добавьте видеофайл в кадр и задайте параметры его обрезки:

```csharp
IVideo video = pres.Videos.AddVideo(File.ReadAllBytes(videoFileName));
var videoFrame = slide.Shapes.AddVideoFrame(0, 0, 200, 200, video);

// Установите время начала (в миллисекундах), с которого будет воспроизводиться видео.
videoFrame.TrimFromStart = 12000f; // Начало с 12 секунд

// Установите время окончания воспроизведения видео.
videoFrame.TrimFromEnd = 14000f;   // Конец на 16 секунде
```

**Шаг 3: Сохраните презентацию**
Сохраните вашу презентацию:

```csharp
pres.Save("YOUR_OUTPUT_DIRECTORY\\VideoTrimmingOutput.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
}
```

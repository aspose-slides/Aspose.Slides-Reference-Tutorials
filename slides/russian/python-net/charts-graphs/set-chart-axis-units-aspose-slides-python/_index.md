## Руководство по внедрению

### Отображение единицы измерения на оси диаграммы
#### Обзор
Эта функция позволяет маркировать оси диаграммы пользовательскими единицами измерения, например, миллионами или миллиардами, что улучшает читаемость данных в презентациях.

#### Пошаговая реализация
1. **Инициализировать презентацию**
   Начните с создания нового экземпляра презентации, куда будет добавлена ваша диаграмма:

   ```python
   with slides.Presentation() as pres:
       # Ваш код для управления слайдами и диаграммами находится здесь
   ```

2. **Добавить кластеризованную столбчатую диаграмму**
   Добавьте кластеризованную столбчатую диаграмму в указанных координатах на первом слайде:

   ```python
   chart = pres.slides[0].shapes.add_chart(
       slides.charts.ChartType.CLUSTERED_COLUMN, 50, 50, 450, 300
   )
   ```

3. **Установить единицу отображения вертикальной оси**
   Настройте вертикальную ось для отображения значений в миллионах:

   ```python
   chart.axes.vertical_axis.display_unit = slides.charts.DisplayUnitType.MILLIONS
   ```

4. **Сохранить презентацию**
   Сохраните презентацию с настроенной диаграммой:

   ```python
   pres.save("YOUR_OUTPUT_DIRECTORY/charts_showing_display_unit_label_out.pptx", slides.export.SaveFormat.PPTX)
   ```

#### Параметры и методы
- `add_chart`: Добавляет новый объект диаграммы на слайд.
- `display_unit`: Устанавливает единицу отображения числовых значений на вертикальной оси.

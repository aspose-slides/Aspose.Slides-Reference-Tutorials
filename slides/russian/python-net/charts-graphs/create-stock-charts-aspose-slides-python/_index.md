## Руководство по внедрению
В этом разделе мы подробно рассмотрим каждый шаг, необходимый для создания и настройки биржевой диаграммы.

### Добавление биржевой диаграммы
Для начала давайте добавим в вашу презентацию биржевую диаграмму:

```python
with slides.Presentation() as pres:
    # Добавьте биржевую диаграмму в позицию (50, 50) размером (600, 400)
    chart = pres.slides[0].shapes.add_chart(slides.charts.ChartType.OPEN_HIGH_LOW_CLOSE, 50, 50, 600, 400, False)

    # Очистить существующие данные
    chart.chart_data.series.clear()
    chart.chart_data.categories.clear()

    # Доступ к рабочей книге по манипуляциям с ячейками
    wb = chart.chart_data.chart_data_workbook
```

### Настройка категорий и серий
Далее мы настроим категории и серии для хранения ваших биржевых данных:

```python
# Добавить категории (A, B, C)
chart.chart_data.categories.add(wb.get_cell(0, 1, 0, "A"))
chart.chart_data.categories.add(wb.get_cell(0, 2, 0, "B"))
chart.chart_data.categories.add(wb.get_cell(0, 3, 0, "C"))

# Добавить ряды для данных открытия, максимума, минимума и закрытия
series_names = ["Open", "High", "Low", "Close"]
for i, name in enumerate(series_names):
    chart.chart_data.series.add(wb.get_cell(0, 0, i + 1, name), chart.type)
```

### Добавление точек данных
Теперь давайте заполним ряд точками данных:

```python
# Данные для «Открытия», «Максимума», «Низкого» и «Закрытия»
data = [
    [72, 172, 12, 25],
    [25, 57, 12, 38],
    [38, 57, 13, 50]
]

# Назначьте данные для каждой серии
for i in range(4):
    series = chart.chart_data.series[i]
    for j in range(3):
        series.data_points.add_data_point_for_stock_series(wb.get_cell(0, j + 1, i + 1, data[j][i]))
```

### Настройка внешнего вида диаграммы
Повысьте визуальную привлекательность вашего биржевого графика:

```python
# Включить полосы вверх-вниз и задать формат линии вверх-вниз
chart.chart_data.series_groups[0].up_down_bars.has_up_down_bars = True
chart.chart_data.series_groups[0].hi_low_lines_format.line.fill_format.fill_type = slides.FillType.SOLID

# Установите для серийных линий режим «без заливки» для более чистого вида
for ser in chart.chart_data.series:
    ser.format.line.fill_format.fill_type = slides.FillType.NO_FILL
```

### Сохранение презентации
Наконец, сохраните презентацию с только что созданной биржевой диаграммой:

```python
# Сохранить презентацию на диск
pres.save("charts_stock_chart_out.pptx", slides.export.SaveFormat.PPTX)
```

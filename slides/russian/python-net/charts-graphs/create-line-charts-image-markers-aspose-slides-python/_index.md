## Руководство по внедрению

### Создание базовой линейной диаграммы с маркерами

#### Обзор

Начните с добавления на слайд простой линейной диаграммы, которую вы позже настроите.

#### Шаги
1. **Инициализировать презентацию**

    ```python
    import aspose.slides as slides

    def create_line_chart_with_markers():
        with slides.Presentation() as pres:
            slide = pres.slides[0]
    ```

2. **Добавить линейную диаграмму**

   Добавить диаграмму в позицию `(0, 0)` и размер `400x400`.

    ```python
    chart = slide.shapes.add_chart(slides.charts.ChartType.LINE_WITH_MARKERS, 0, 0, 400, 400)
    ```

3. **Доступ к данным диаграммы**

   Очистите существующие ряды и добавьте новые точки данных.

    ```python
    fact = chart.chart_data.chart_data_workbook
    chart.chart_data.series.clear()
    chart.chart_data.series.add(fact.get_cell(0, 1, 1, "Series 1"), chart.type)
    ```

4. **Сохранить презентацию**

   Сохраните свою работу в файл.

    ```python
    pres.save("YOUR_OUTPUT_DIRECTORY/charts_marker_options_out.pptx", slides.export.SaveFormat.PPTX)
    ```

### Добавление изображений в качестве маркеров

#### Обзор

Улучшите свой линейный график, используя изображения в качестве маркеров, что сделает точки данных более различимыми.

#### Шаги
1. **Инициализировать презентацию**

    ```python
    import aspose.slides as slides

    def add_images_to_chart():
        with slides.Presentation() as pres:
            slide = pres.slides[0]
    ```

2. **Добавить линейную диаграмму**

   Аналогично предыдущему разделу добавьте линейную диаграмму.

    ```python
    chart = slide.shapes.add_chart(slides.charts.ChartType.LINE_WITH_MARKERS, 0, 0, 400, 400)
    fact = chart.chart_data.chart_data_workbook
    ```

3. **Загрузить и добавить изображения**

   Определите функцию для загрузки изображений.

    ```python
    def load_and_add_image(pres, image_path):
        img = slides.Images.from_file(image_path)
        return pres.images.add_image(img)

    imgx1 = load_and_add_image(pres, "YOUR_DOCUMENT_DIRECTORY/image1.jpg")
    imgx2 = load_and_add_image(pres, "YOUR_DOCUMENT_DIRECTORY/image2.jpg")
    ```

4. **Добавьте точки данных с помощью маркеров изображений**

   Настройте точки данных для использования изображений в качестве маркеров.

    ```python
    series = chart.chart_data.series[0]

    point = series.data_points.add_data_point_for_line_series(fact.get_cell(0, 1, 1, 4.5))
    point.marker.format.fill.fill_type = slides.FillType.PICTURE
    point.marker.format.fill.picture_fill_format.picture.image = imgx1

    # Повторите для других точек данных с другими изображениями по мере необходимости.
    ```

5. **Установить размер маркера**

   Отрегулируйте размер маркеров в серии.

    ```python
    series.marker.size = 15
    ```

6. **Сохранить презентацию**

   Сохраните презентацию с добавленными маркерами изображений.

    ```python
    pres.save("YOUR_OUTPUT_DIRECTORY/charts_with_image_markers_out.pptx", slides.export.SaveFormat.PPTX)
    ```

## Руководство по внедрению: настройка режима макета диаграммы

Давайте разберемся, как настроить режим макета диаграммы в презентации PowerPoint.

### Создать и получить доступ к слайду

Начните с создания новой презентации PowerPoint и откройте ее первый слайд:

```python
with slides.Presentation() as presentation:
    slide = presentation.slides[0]
```

Это настроит вашу среду для добавления диаграмм.

### Добавить кластеризованную столбчатую диаграмму

Добавьте кластеризованную столбчатую диаграмму в указанное место на слайде:

```python
chart = slide.shapes.add_chart(
    slides.charts.ChartType.CLUSTERED_COLUMN, 20, 100, 600, 400
)
```

Параметры:
- `ChartType.CLUSTERED_COLUMN`: Определяет тип диаграммы.
- `(20, 100)`Координаты x и y, по которым диаграмма размещается на слайде.
- `(600, 400)`: Ширина и высота диаграммы в пунктах.

### Настройте свойства макета

Теперь настройте свойства макета области построения, чтобы задать ее положение и размер:

```python
chart.plot_area.as_i_layoutable.x = 0.2
chart.plot_area.as_i_layoutable.y = 0.2
chart.plot_area.as_i_layoutable.width = 0.7
chart.plot_area.as_i_layoutable.height = 0.7
```

Эти значения являются относительными единицами, что обеспечивает динамическую подстройку диаграммы под различные размеры слайдов.

### Укажите тип целевого макета

Установите тип целевого макета для точного управления поведением области построения:

```python
chart.plot_area.layout_target_type = slides.charts.LayoutTargetType.INNER
```

Такая конфигурация гарантирует, что участок будет располагаться по центру контейнера, сохраняя при этом аккуратный вид.

### Сохраните вашу презентацию

Наконец, сохраните презентацию в указанном выходном каталоге:

```python
output_directory = 'YOUR_OUTPUT_DIRECTORY/'
presentation.save(output_directory + 'charts_set_layout_mode_out.pptx', slides.export.SaveFormat.PPTX)
```

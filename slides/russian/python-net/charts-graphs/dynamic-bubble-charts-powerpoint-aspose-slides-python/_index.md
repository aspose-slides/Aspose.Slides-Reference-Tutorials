## Руководство по внедрению

### Создание динамической пузырьковой диаграммы

Давайте погрузимся в создание динамической пузырьковой диаграммы в PowerPoint с помощью Aspose.Slides. Эта функция позволяет визуализировать точки данных с различными размерами, что делает ее идеальной для сравнения нескольких измерений наборов данных.

#### Добавление диаграммы

**Шаг 1: Инициализация презентации**

Начните с создания или открытия презентации, в которую будет добавлена диаграмма:

```python
with slides.Presentation() as pres:
    slide = pres.slides[0]  # Доступ к первому слайду
```

**Шаг 2: Добавьте динамическую пузырьковую диаграмму**

Добавьте динамическую пузырьковую диаграмму на выбранный слайд в определенных координатах с заданными размерами:

```python
chart = slide.shapes.add_chart(
    slides.charts.ChartType.BUBBLE, 100, 100, 400, 300
)
```

Этот фрагмент кода создает динамическую пузырьковую диаграмму, расположенную на слайде в точке (100, 100) шириной 400 и высотой 300.

#### Регулировка шкалы размера пузырьков

**Шаг 3: Установите размер пузырьков**

Настройте визуализацию данных, отрегулировав масштаб размера пузырьков в первой группе серий:

```python
chart.chart_data.series_groups[0].bubble_size_scale = 150
```

Эта настройка масштабирует размеры пузырьков, повышая четкость и визуальное воздействие.

#### Сохранение презентации

**Шаг 4: Сохраните файл.**

После внесения изменений сохраните презентацию, чтобы сохранить изменения:

```python
pres.save('dynamic_bubble_chart_scaling_out.pptx', slides.export.SaveFormat.PPTX)
```

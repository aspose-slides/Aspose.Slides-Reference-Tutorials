## Руководство по внедрению
В этом разделе подробно описаны шаги, необходимые для создания и настройки кольцевой диаграммы в PowerPoint с помощью Aspose.Slides.

### Шаг 1: Доступ к слайду и его изменение
Для начала откройте первый слайд вашей презентации. Здесь вы добавите свою собственную кольцевую диаграмму.

```python
# Доступ к первому слайду
class SlideModifier:
    def modify_slide(self, presentation):
        first_slide = presentation.slides[0]
```

### Шаг 2: Добавление кольцевой диаграммы
Вы можете добавить кольцевую диаграмму на любой слайд, указав ее положение и размер. Здесь мы разместим ее в координатах (50, 50) с размерами 400x400.

```python
class ChartAdder:
    def add_doughnut_chart(self, first_slide):
        # Добавить кольцевую диаграмму
        chart = first_slide.shapes.add_chart(
            slides.charts.ChartType.DOUGHNUT,
            50, 50, 400, 400
        )
```

### Шаг 3: Настройка размера отверстия
Настроить размер отверстия в вашей кольцевой диаграмме просто. Установите его на 90% для выраженного эффекта.

```python
class ChartCustomizer:
    def customize_hole_size(self, chart):
        # Установить индивидуальный размер отверстия
        chart.chart_data.series_groups[0].doughnut_hole_size = 90
```

### Шаг 4: Сохранение презентации
Наконец, сохраните презентацию в нужном месте под выбранным именем файла.

```python
class PresentationSaver:
    def save_presentation(self, presentation):
        # Сохранить презентацию
        presentation.save(
            "charts_doughnut_chart_hole_out.pptx",
            slides.export.SaveFormat.PPTX
        )
```

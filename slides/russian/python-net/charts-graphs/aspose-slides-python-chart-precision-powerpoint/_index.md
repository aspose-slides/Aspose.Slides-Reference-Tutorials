## Руководство по внедрению
Мы поможем вам создать линейный график и настроить точность данных. 
### Добавление линейной диаграммы в PowerPoint
**Обзор**: Мы добавим в вашу презентацию линейную диаграмму, отображающую данные с отформатированными значениями.
#### Шаг 1: Инициализация презентации
Создайте экземпляр `Presentation` класс с использованием `with` заявление об эффективном управлении ресурсами:
```python
with slides.Presentation() as pres:
    # Ваш код здесь
```
#### Шаг 2: Добавьте линейную диаграмму
Добавьте диаграмму на первый слайд, указав ее положение и размер:
```python
chart = pres.slides[0].shapes.add_chart(
    slides.charts.ChartType.LINE, 50, 50, 450, 300
)
```
**Объяснение параметров**: 
- `ChartType.LINE`: Указывает, что это линейная диаграмма.
- `(50, 50)`: Положения X и Y на слайде.
- `(450, 300)`: Ширина и высота диаграммы.
#### Шаг 3: Включить таблицу данных
Отображение значений данных непосредственно на диаграмме:
```python
chart.has_data_table = True
```
#### Шаг 4: Установка числового формата
Для точности отформатируйте числа до двух знаков после запятой:
```python
chart.chart_data.series[0].number_format_of_values = "#,##0.00"
```
**Почему это важно**: Обеспечивает ясность и последовательность представления данных.
### Сохранение презентации
Наконец, сохраните презентацию в указанном каталоге:
```python
pres.save("YOUR_OUTPUT_DIRECTORY/charts_precision_of_data_out.pptx", slides.export.SaveFormat.PPTX)
```

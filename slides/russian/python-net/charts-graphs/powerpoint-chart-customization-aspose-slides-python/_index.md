### Создайте диаграмму солнечных лучей в PowerPoint
#### Обзор
Создание диаграммы в PowerPoint упрощается с помощью Aspose.Slides, который позволяет точно контролировать положение и размер.

#### Этапы внедрения
1. **Инициализировать презентацию**: Начните с создания нового объекта презентации.
2. **Добавить диаграмму**: Вставьте диаграмму Sunburst на первый слайд в указанных координатах.

```python
import aspose.slides as slides

with slides.Presentation() as pres:
    chart = pres.slides[0].shapes.add_chart(slides.charts.ChartType.SUNBURST, 100, 100, 450, 400)
```

**Объясняемые параметры:**
- `ChartType.SUNBURST`: Указывает тип диаграммы.
- Координаты `(100, 100)`: Расположение на слайде.
- Размер `(450, 400)`: Размеры диаграммы.

### Настройте метки точек данных на диаграммах
#### Обзор
Настройка меток точек данных может повысить ясность и фокусировку, отображая конкретную информацию, например значения или названия рядов.

#### Этапы внедрения
1. **Точки доступа к данным**: Извлечь точки данных из первой серии.
2. **Показать значения**Включить отображение значения для определенной точки данных.
3. **Изменить свойства метки**: Настройте параметры метки, чтобы отобразить название категории, название серии и изменить цвет текста.

```python
import aspose.slides as slides
import aspose.pydrawing as drawing

def customize_data_point_labels():
    with slides.Presentation() as pres:
        chart = pres.slides[0].shapes.add_chart(slides.charts.ChartType.SUNBURST, 100, 100, 450, 400)
        data_points = chart.chart_data.series[0].data_points
        
        # Показать значение для определенной точки данных
        data_points[3].data_point_levels[0].label.data_label_format.show_value = True

        # Настройте свойства метки для другой ветки
        branch1_label = data_points[0].data_point_levels[2].label
        branch1_label.data_label_format.show_category_name = False
        branch1_label.data_label_format.show_series_name = True
        branch1_label.data_label_format.text_format.portion_format.fill_format.fill_type = slides.FillType.SOLID
        branch1_label.data_label_format.text_format.portion_format.fill_format.solid_fill_color.color = drawing.Color.yellow
```

**Ключевые конфигурации:**
- Использовать `data_label_format` для переключения параметров отображения.
- Нанесите цвет с помощью `FillType` и `Color` классы.

### Изменить цвет заливки точки данных
#### Обзор
Изменение цвета заливки может выделить определенные точки данных, сделав их более заметными на диаграмме.

#### Этапы внедрения
1. **Точки доступа к данным**: Получите точку данных, которую вы хотите настроить.
2. **Установить тип и цвет заливки**: Измените параметры заливки, чтобы применить новые цвета.

```python
def change_data_point_fill_color():
    with slides.Presentation() as pres:
        chart = pres.slides[0].shapes.add_chart(slides.charts.ChartType.SUNBURST, 100, 100, 450, 400)
        data_points = chart.chart_data.series[0].data_points
        
        # Изменить цвет заливки для определенной точки данных
        steam4_format = data_points[9].format
        steam4_format.fill.fill_type = slides.FillType.SOLID
        steam4_format.fill.solid_fill_color.color = drawing.Color.from_argb(0, 176, 240, 255)
```

**Объясняемые параметры:**
- `fill.fill_type`: Устанавливает тип заливки (например, сплошная).
- `from_argb()`: определяет цвет с использованием значений альфа, красного, зеленого и синего.

### Сохранить презентацию в выходной каталог
#### Обзор
После настройки диаграмм сохраните их в каталоге для совместного использования или дальнейшего редактирования.

#### Этапы внедрения
1. **Сохранить файл**: Используйте `save` метод с указанным путем и форматом.

```python
def save_presentation():
    with slides.Presentation() as pres:
        chart = pres.slides[0].shapes.add_chart(slides.charts.ChartType.SUNBURST, 100, 100, 450, 400)
        
        # Сохраните презентацию в YOUR_OUTPUT_DIRECTORY/
        pres.save("YOUR_OUTPUT_DIRECTORY/charts_add_color_to_data_points_out.pptx", slides.export.SaveFormat.PPTX)
```

**Ключевые моменты:**
- `SaveFormat.PPTX`: Гарантирует сохранение файла в формате PowerPoint.

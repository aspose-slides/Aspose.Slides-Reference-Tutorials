## Руководство по внедрению

Мы разобьем процесс на логические разделы на основе ключевых функций: создание презентаций, добавление диаграмм рассеяния, добавление рядов данных и настройка.

### Создание презентации с помощью точечной диаграммы

#### Обзор
Создание презентации и встраивание диаграммы рассеяния осуществляется просто с помощью Aspose.Slides. В этом разделе вы узнаете, как создать файл PowerPoint с начальной диаграммой рассеяния.

#### Этапы внедрения
**1. Инициализируйте презентацию:**

```python
import aspose.slides as slides

def create_and_add_scatter_chart():
    with slides.Presentation() as pres:
        slide = pres.slides[0]
```

**2. Добавьте на слайд точечную диаграмму:**
Здесь вы можете разместить и изменить размер диаграммы на слайде.

```python
        chart = slide.shapes.add_chart(
            slides.charts.ChartType.SCATTER_WITH_SMOOTH_LINES,
            0, 0, 400, 400
        )
```

**3. Сохраните презентацию:**
Обязательно сохраните презентацию после внесения изменений:

```python
    pres.save("YOUR_OUTPUT_DIRECTORY/charts_scattered_chart_out.pptx", slides.export.SaveFormat.PPTX)
```

### Добавление ряда данных на диаграмму

#### Обзор
Чтобы сделать диаграммы рассеяния осмысленными, вам нужны данные. В этом разделе объясняется, как добавлять ряд точек данных в вашу диаграмму.

**1. Очистить существующие серии:**

```python
        chart.chart_data.series.clear()
```

**2. Добавить новый ряд данных:**
Использовать `add` Метод вставки нового ряда данных в диаграмму:

```python
        series1 = chart.chart_data.series.add(
            fact.get_cell(default_worksheet_index, 1, 1, "Series 1"), chart.type
        )
        series2 = chart.chart_data.series.add(
            fact.get_cell(default_worksheet_index, 1, 3, "Series 2"), chart.type
        )
```

### Настройка рядов и добавление точек данных

#### Обзор
Настройка улучшает визуальную привлекательность и читаемость ваших диаграмм. В этом разделе рассматривается добавление точек данных и настройка маркеров серий.

**1. Добавьте точки данных:**

```python
        series1.data_points.add_data_point_for_scatter_series(
            fact.get_cell(default_worksheet_index, 2, 1, 1), 
            fact.get_cell(default_worksheet_index, 2, 2, 3)
        )
```

**2. Настройте маркеры серий:**

```python
        series1.marker.size = 10
        series1.marker.symbol = slides.charts.MarkerStyleType.STAR
```

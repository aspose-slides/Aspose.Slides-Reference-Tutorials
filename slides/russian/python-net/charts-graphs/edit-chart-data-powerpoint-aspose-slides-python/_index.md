### Редактирование данных диаграммы с помощью Aspose.Slides

#### Обзор

Эта функция позволяет вам программно корректировать точки данных диаграмм в ваших презентациях PowerPoint. Используя Aspose.Slides, вы можете автоматизировать задачи, которые в противном случае потребовали бы ручного редактирования.

#### Пошаговое руководство

**1. Настройте пути к файлам**

Сначала определите входные и выходные каталоги для файлов презентации:

```python
input_file = "YOUR_DOCUMENT_DIRECTORY/charts_with_external_workbook.pptx"
output_file = "YOUR_OUTPUT_DIRECTORY/charts_edit_chartdata_in_external_workbook_out.pptx"
```

**2. Загрузите презентацию**

Используйте Aspose.Slides, чтобы открыть файл PowerPoint и получить доступ к его содержимому:

```python
with slides.Presentation(input_file) as pres:
    # Доступ к первой фигуре, предполагая, что это диаграмма
    chart = pres.slides[0].shapes[0]
```
- **Почему**: Этот шаг гарантирует, что мы работаем с существующей презентацией и напрямую манипулируем ее элементами.

**3. Извлечение и изменение данных диаграммы**

Получите доступ к данным диаграммы для обновления определенных значений:

```python
chart_data = chart.chart_data

# Измените значение первой точки данных в первой серии.
chart_data.series[0].data_points[0].value.as_cell.value = 100
```
- **Почему**: Изменение `.as_cell.value` позволяет напрямую задавать новые значения, что эффективно для массовых обновлений.

**4. Сохраните изменения.**

Наконец, сохраните изменения в новом файле:

```python
pres.save(output_file, slides.export.SaveFormat.PPTX)
```
- **Почему**: Сохранение в виде другого файла гарантирует, что исходные данные останутся неизменными, если это не требуется.

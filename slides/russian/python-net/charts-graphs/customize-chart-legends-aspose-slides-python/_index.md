## Руководство по внедрению

### Обзор настройки легенд диаграмм
Настройка легенд диаграммы включает в себя настройку таких свойств, как положение, размер и выравнивание относительно размеров диаграммы. В этом разделе вы узнаете, как добавить кластеризованную столбчатую диаграмму и изменить ее легенду.

#### Шаг 1: Создайте новую презентацию
```python
import aspose.slides as slides

def charts_set_legend_custom_options():
    with slides.Presentation() as presentation:
        slide = presentation.slides[0]
```
Этот код инициализирует новую презентацию и получает доступ к первому слайду для внесения изменений.

#### Шаг 2: Добавьте кластеризованную столбчатую диаграмму
```python
chart = slide.shapes.add_chart(
    slides.charts.ChartType.CLUSTERED_COLUMN,
    50, 50, 500, 500
)
```
Добавьте на слайд кластеризованную столбчатую диаграмму. Параметры определяют тип диаграммы, ее положение и размеры на слайде.

#### Шаг 3: Задайте свойства легенды
Настройка свойств легенды подразумевает расчет позиций как долей ширины и высоты диаграммы:
```python
chart.legend.x = 50 / chart.width
chart.legend.y = 50 / chart.height
chart.legend.width = 100 / chart.width
chart.legend.height = 100 / chart.height
```
Здесь, `x`, `y`, `width`, и `height` корректируются как дроби для сохранения оперативности реагирования.

#### Шаг 4: Сохраните презентацию
```python
presentation.save("YOUR_OUTPUT_DIRECTORY/charts_set_legend_custom_options_out.pptx")
```
Заменять `"YOUR_OUTPUT_DIRECTORY"` с желаемым местом сохранения. Этот шаг сохраняет вашу настроенную презентацию.

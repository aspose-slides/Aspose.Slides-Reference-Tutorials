## Руководство по внедрению

### Функция: Копировать внешнюю рабочую книгу
**Обзор:**
Сначала мы покажем, как скопировать внешний файл рабочей книги из исходного каталога в целевой выходной каталог с помощью Python `shutil` модуль. Это гарантирует, что ваша презентация будет иметь доступ к необходимым данным.

#### Шаг 1: Импорт необходимых библиотек
```python
import shutil
```

#### Шаг 2: Определите пути к файлам и скопируйте книгу
```python
external_workbook_file_name = "charts_external_workbook.xlsx"
source_path = "YOUR_DOCUMENT_DIRECTORY/" + external_workbook_file_name
output_path = "YOUR_OUTPUT_DIRECTORY/" + external_workbook_file_name
shutil.copyfile(source_path, output_path)
```
Этот фрагмент копирует `charts_external_workbook.xlsx` из каталога ваших документов в выходной каталог.

### Функция: создание презентации и настройка внешней рабочей книги для данных диаграммы
**Обзор:**
Далее мы создадим презентацию и установим внешнюю книгу в качестве источника данных для диаграммы с помощью Aspose.Slides. Это позволяет визуализировать данные Excel непосредственно в слайдах PowerPoint.

#### Шаг 1: Импорт Aspose.Slides
```python
import aspose.slides as slides
```

#### Шаг 2: Определите функцию создания презентации
```python
def create_presentation_with_external_chart():
    with slides.Presentation() as pres:
        chart = pres.slides[0].shapes.add_chart(
            slides.charts.ChartType.PIE, 50, 50, 400, 600, False)
        
        chart_data = chart.chart_data
        chart_data.set_external_workbook("YOUR_OUTPUT_DIRECTORY/charts_external_workbook.xlsx")
        
        series = chart_data.series.add(chart_data.chart_data_workbook.get_cell(0, "B1"), slides.charts.ChartType.PIE)
        
        # Добавить точки данных для круговой диаграммы из внешних ячеек рабочей книги
        series.data_points.add_data_point_for_pie_series(chart_data.chart_data_workbook.get_cell(0, "B2"))
        series.data_points.add_data_point_for_pie_series(chart_data.chart_data_workbook.get_cell(0, "B3"))
        series.data_points.add_data_point_for_pie_series(chart_data.chart_data_workbook.get_cell(0, "B4"))

        chart_data.categories.add(chart_data.chart_data_workbook.get_cell(0, "A2"))
        chart_data.categories.add(chart_data.chart_data_workbook.get_cell(0, "A3"))
        chart_data.categories.add(chart_data.chart_data_workbook.get_cell(0, "A4"))
        
        pres.save("YOUR_OUTPUT_DIRECTORY/charts_set_external_workbook_out.pptx", slides.export.SaveFormat.PPTX)
```

#### Объяснение:
- **Создать презентацию**Начнем с открытия нового объекта презентации.
- **Добавить диаграмму**: Круговая диаграмма добавляется к первому слайду с указанными координатами и размерами.
- **Установить внешнюю рабочую книгу**: Путь к рабочей книге задается таким образом, чтобы Aspose.Slides знал, откуда извлекать данные.
- **Добавить ряды и точки данных**: Мы настраиваем ряды с определенными ячейками из внешней рабочей книги, обеспечивая динамические обновления.

#### Советы по устранению неполадок:
- Убедитесь, что пути к файлам указаны правильно. В противном случае вы столкнетесь с ошибками «Файл не найден».
- Убедитесь, что ссылки на ячейки в файле Excel соответствуют ссылкам в коде, чтобы избежать проблем с несоответствием данных.

## Руководство по внедрению

Давайте разберем процесс изменения цветов рядов в круговой диаграмме с помощью Aspose.Slides для Python.

### Создание круговой диаграммы

**Обзор:**
Добавление круговой диаграммы в вашу презентацию — наш первый шаг. Мы разместим ее в определенных координатах с определенными размерами.

#### Добавить круговую диаграмму

```python
# Создать экземпляр презентации
with slides.Presentation() as pres:
    # Добавьте круговую диаграмму с координатами (50, 50) и шириной 600 и высотой 400.
    chart = pres.slides[0].shapes.add_chart(slides.charts.ChartType.PIE, 50, 50, 600, 400)
```

**Объяснение:** 
Здесь, `add_chart` используется для вставки круговой диаграммы на первый слайд. Параметры определяют ее положение и размер.

### Доступ к точкам данных

**Обзор:**
Далее мы получаем доступ к определенным точкам данных в нашей серии для настройки.

#### Получите вторую точку данных первой серии

```python
# Доступ ко второй точке данных первой серии
point = chart.chart_data.series[0].data_points[1]
```

**Объяснение:** 
`chart.chart_data.series[0]` получает доступ к первой серии, и `.data_points[1]` выбирает вторую точку данных.

### Настройка цвета серии

**Обзор:**
Мы изменим цвет заливки выбранной точки данных, чтобы выделить ее.

#### Установить эффект взрыва и изменить тип заливки

```python
# Установить эффект взрыва для акцента
point.explosion = 30

# Измените тип заливки на сплошной и установите синий цвет.
point.format.fill.fill_type = slides.FillType.SOLID
point.format.fill.solid_fill_color.color = drawing.Color.blue
```

**Объяснение:** 
The `explosion` свойство разделяет точку данных, в то время как `fill_type` установлен на `SOLID`, что позволяет нам определить конкретный цвет с помощью `solid_fill_color`.

#### Сохраните вашу презентацию

Наконец, сохраните презентацию со всеми изменениями:

```python
# Сохраните презентацию с изменениями
pres.save("YOUR_OUTPUT_DIRECTORY/charts_changing_series_color_out.pptx", slides.export.SaveFormat.PPTX)
```

**Объяснение:** 
Это сохранит вашу работу в файле в указанном каталоге.

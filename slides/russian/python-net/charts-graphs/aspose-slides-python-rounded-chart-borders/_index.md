## Руководство по внедрению

### Обзор функций: закругленные границы области диаграммы

Эта функция направлена на улучшение эстетики диаграмм путем добавления закругленных углов в ваши презентации PowerPoint.

#### Шаг 1: Создайте новую презентацию
Начните с инициализации объекта презентации. Это служит основой для добавления ваших диаграмм и других элементов.

```python
def create_presentation_with_rounded_chart():
    with slides.Presentation() as presentation:
        # Доступ к первому слайду презентации
        slide = presentation.slides[0]
```

#### Шаг 2: Добавьте кластеризованную столбчатую диаграмму
Поместите кластеризованную столбчатую диаграмму на слайд. Укажите ее положение и размер для оптимального макета.

```python
# Добавьте кластеризованную столбчатую диаграмму в позицию (20, 100) шириной 600 и высотой 400.
chart = slide.shapes.add_chart(
    slides.charts.ChartType.CLUSTERED_COLUMN,
    20,
    100,
    600,
    400
)
```

#### Шаг 3: Настройте формат линии диаграммы
Примените сплошную заливку к границе диаграммы, чтобы она выделялась на фоне презентации.

```python
# Установить формат линии на сплошной тип заливки
cart.line_format.fill_format.fill_type = slides.FillType.SOLID
cart.line_format.style = slides.LineStyle.SINGLE
```

#### Шаг 4: Включите скругленные углы
Активируйте функцию закругленных углов, чтобы придать области диаграммы современный и изысканный вид.

```python
# Включить закругленные углы для области диаграммы
cart.has_rounded_corners = True
```

#### Шаг 5: Сохраните презентацию
Наконец, сохраните вашу презентацию в указанном каталоге с соответствующим именем файла.

```python
presentation.save(
    "YOUR_OUTPUT_DIRECTORY/charts_chart_area_rounded_borders_out.pptx",
    slides.export.SaveFormat.PPTX
)
```

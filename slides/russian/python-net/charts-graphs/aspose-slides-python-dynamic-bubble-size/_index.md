## Руководство по внедрению

### Создание и изменение пузырьковой диаграммы

#### Обзор

Мы создадим презентацию PowerPoint, добавим в нее пузырьковую диаграмму и изменим размеры пузырьков на основе конкретных измерений данных с помощью Aspose.Slides.

#### Пошаговая реализация

**1. Инициализация презентации**

Начните с создания экземпляра `Presentation` в менеджере контекста:

```python
import aspose.slides as slides

def charts_bubble_size_representation():
    with slides.Presentation() as pres:
        # Код продолжается...
```

**2. Добавить пузырьковую диаграмму**

Добавить пузырьковую диаграмму в позицию `(50, 50)` с размерами `600x400` на первом слайде.

```python
chart = pres.slides[0].shapes.add_chart(
    slides.charts.ChartType.BUBBLE,
    50, 50, 600, 400, True
)
```

**3. Установка представления размера пузырька**

Настройте представление размера пузырька так, чтобы `WIDTH` для первой группы серий:

```python
chart.chart_data.series_groups[0].bubble_size_representation = \\
    slides.charts.BubbleSizeRepresentationType.WIDTH
```

**4. Сохранить презентацию**

Наконец, сохраните презентацию в указанном каталоге:

```python
pres.save(
    "YOUR_OUTPUT_DIRECTORY/charts_bubble_size_representation_out.pptx"
)
```

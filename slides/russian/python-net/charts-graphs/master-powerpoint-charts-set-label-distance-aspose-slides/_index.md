## Руководство по внедрению

Теперь давайте сосредоточимся на настройке расстояния метки от оси на вашей диаграмме.

### Добавление кластеризованной столбчатой диаграммы на слайд

Сначала добавим кластеризованную столбчатую диаграмму:

```python
# Доступ к первому слайду презентации
class SlideManager:
    def __init__(self, presentation):
        self.slide = presentation.slides[0]

    def add_chart(self):
        return self.slide.shapes.add_chart(
            slides.charts.ChartType.CLUSTERED_COLUMN, 20, 20, 500, 300)

with PresentationManager() as presentation:
    slide_manager = SlideManager(presentation)
    chart = slide_manager.add_chart()
```

**Объяснение**: Этот код создает новую диаграмму на первом слайде, расположенную в точке (20, 20) с размерами 500x300.

### Установка смещения метки от оси

Далее настройте смещение метки:

```python
# Установить смещение метки от оси для горизонтальной оси
class ChartManager:
    def __init__(self, chart):
        self.chart = chart

    def set_label_offset(self, offset):
        self.chart.axes.horizontal_axis.label_offset = offset

chart_manager = ChartManager(chart)
chart_manager.set_label_offset(500)
```

**Объяснение**: Устанавливая `label_offset`, мы гарантируем, что этикетки будут размещены на соответствующем расстоянии. Значение может быть скорректировано в зависимости от ваших конкретных потребностей.

### Установка формулы в ячейке данных диаграммы

#### Обзор
Эта функция позволяет вам динамически рассчитывать данные в вашей диаграмме, устанавливая формулы непосредственно в ячейках данных. Это особенно полезно для автоматизации обновлений и обеспечения точности в презентациях.

#### Шаги по реализации

1. **Создать объект презентации:**
   Начнем с инициализации объекта презентации, куда мы добавим нашу диаграмму.
   
   ```python
   import aspose.slides as slides
   
   def set_formula_in_chart_cell():
       with slides.Presentation() as presentation:
           # Дальнейшие шаги следуют...
   ```

2. **Добавьте кластеризованную столбчатую диаграмму:**
   Вставьте кластеризованную столбчатую диаграмму в первый слайд презентации.
   
   ```python
   chart = presentation.slides[0].shapes.add_chart(
       slides.charts.ChartType.CLUSTERED_COLUMN, 150, 150, 500, 300)
   ```

3. **Рабочая книга данных диаграммы доступа:**
   Извлеките объект рабочей книги, связанный с диаграммой, для управления ячейками данных.
   
   ```python
   workbook = chart.chart_data.chart_data_workbook
   ```

4. **Задайте формулу в ячейке B2:**
   Определите формулу для ячейки B2, используя стандартную запись электронной таблицы.
   
   ```python
   cell1 = workbook.get_cell(0, "B2")
   cell1.formula = "1 + SUM(F2:H5)"
   ```

5. **Используйте обозначение R1C1 в ячейке C2:**
   В качестве альтернативы используйте обозначение R1C1 для более сложных формул.
   
   ```python
   cell2 = workbook.get_cell(0, "C2")
   cell2.r1c1_formula = "MAX(R2C6:R5C8) / 3"
   ```

6. **Формулы расчета:**
   Вычислите результаты этих формул в своей таблице.
   
   ```python
   workbook.calculate_formulas()
   ```

7. **Сохраните презентацию:**
   Сохраните презентацию в определенном выходном каталоге.
   
   ```python
   presentation.save("YOUR_OUTPUT_DIRECTORY/charts_data_cell_formulas_out.pptx")
   ```

### Создание кластеризованной столбчатой диаграммы

#### Обзор
Кластеризованная столбчатая диаграмма отлично подходит для сравнения данных по категориям. Давайте добавим ее на ваш первый слайд в позицию (100, 100) с размерами 400x300.

```python
import aspose.slides as slides
from aspose.slides import Presentation, SaveFormat

with Presentation() as pres:
    # Добавить кластеризованную столбчатую диаграмму
    chart = pres.slides[0].shapes.add_chart(
        slides.charts.ChartType.CLUSTERED_COLUMN,
        100, 100, 400, 300
    )
```

#### Объяснение
- **Тип диаграммы.CLUSTERED_COLUMN**: Указывает тип диаграммы.
- **Положение и размеры**: (100, 100) для положения; 400x300 для размера.

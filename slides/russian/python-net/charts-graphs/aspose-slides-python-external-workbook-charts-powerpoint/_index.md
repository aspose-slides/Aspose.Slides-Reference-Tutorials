## Руководство по внедрению

### Функция 1: Копирование внешней рабочей книги

#### Обзор
Копирование внешней рабочей книги необходимо для того, чтобы ваша презентация ссылалась на самый последний набор данных. Эта функция демонстрирует, как копировать файл из исходного каталога в целевой с помощью Python `shutil` модуль.

#### Шаги по реализации
**Шаг 1**: Импорт необходимых модулей
```python
import shutil
```

**Шаг 2**: Определить функцию копирования рабочей книги
Создайте функцию для управления процессом копирования:
```python
def copy_external_workbook():
    external_workbook_file_name = "charts_external_workbook.xlsx"
    # Используйте shutil.copyfile для перемещения файла из источника в место назначения.
    shutil.copyfile(
        "YOUR_DOCUMENT_DIRECTORY/" + external_workbook_file_name,
        "YOUR_OUTPUT_DIRECTORY/" + external_workbook_file_name
    )
```
- **Параметры**: `shutil.copyfile(source, destination)` где `source` ваш исходный путь к файлу и `destination` — целевой каталог.

### Функция 2: Создание презентации с внешней диаграммой рабочей книги

#### Обзор
Эта функция включает в себя создание презентации PowerPoint и добавление диаграммы, ссылающейся на внешнюю книгу, что позволяет выполнять динамические обновления при каждом изменении исходных данных.

#### Шаги по реализации
**Шаг 1**: Импорт модуля Aspose.Slides
```python
import aspose.slides as slides
```

**Шаг 2**: Определить функцию создания презентации
Создайте функцию для создания презентации с диаграммами:
```python
def create_presentation_with_external_chart():
    # Открыть или создать новую презентацию
    with slides.Presentation() as pres:
        # Добавить круговую диаграмму с указанными координатами и размером
        chart = pres.slides[0].shapes.add_chart(slides.charts.ChartType.PIE, 50, 50, 500, 400)

        # Очистить существующие данные в рабочей книге
        chart.chart_data.chart_data_workbook.clear(0)

        # Установить внешнюю рабочую книгу для диаграммы
        chart.chart_data.set_external_workbook("YOUR_OUTPUT_DIRECTORY/charts_external_workbook.xlsx")

        # Определите диапазон ячеек из «Листа1» для использования в качестве источника данных
        chart.chart_data.set_range("Sheet1!$A$2:$B$5")

        # Установить цветовую вариацию для первой серии в диаграмме
        series = chart.chart_data.series[0]
        series.parent_series_group.is_color_varied = True

        # Сохраните презентацию с указанным именем и форматом
        pres.save("YOUR_OUTPUT_DIRECTORY/charts_create_external_workbook_out.pptx", slides.export.SaveFormat.PPTX)
```
- **Параметры**:
  - `slides.charts.ChartType`: Определяет тип диаграммы.
  - `set_external_workbook(path)`: Устанавливает путь к внешней рабочей книге.
  - `set_range(range_string)`: Указывает, какие ячейки в Excel следует использовать для данных.

### Советы по устранению неполадок

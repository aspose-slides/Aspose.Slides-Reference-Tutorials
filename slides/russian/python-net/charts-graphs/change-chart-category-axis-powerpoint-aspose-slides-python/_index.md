## Руководство по внедрению

Мы сосредоточимся на изменении оси категорий диаграммы. Давайте разберем процесс пошагово.

### Доступ к презентации и диаграмме

Начните с загрузки файла презентации. Убедитесь, что вы знаете путь к документу:

```python
def change_chart_category_axis():
    data_dir = "YOUR_DOCUMENT_DIRECTORY/"
    
    with slides.Presentation(data_dir + "charts_existing_chart.pptx") as presentation:
        chart = presentation.slides[0].shapes[0]
```

Этот фрагмент открывает файл PowerPoint и получает доступ к первой фигуре первого слайда, предполагая, что он содержит диаграмму.

### Изменение оси категорий

Далее измените тип оси категорий на ДАТА:

```python
chart.axes.horizontal_axis.category_axis_type = slides.charts.CategoryAxisType.DATE
```

Установка типа оси на ДАТА гарантирует соответствие данных календарным датам, что повышает читаемость данных временных рядов.

### Настройка свойств оси

Настройте горизонтальную ось, задав основные единицы измерения и масштабы:

```python
chart.axes.horizontal_axis.is_automatic_major_unit = False
chart.axes.horizontal_axis.major_unit = 1
chart.axes.horizontal_axis.major_unit_scale = slides.charts.TimeUnitType.MONTHS
```

Отключив автоматический расчет основных единиц, вы получаете контроль над тем, как точки данных располагаются на оси. `major_unit` определяет интервалы (например, каждый месяц), в то время как `major_unit_scale` указывает, что эти единицы представляют месяцы.

### Сохранение изменений

Наконец, сохраните измененную презентацию:

```python
out_dir = "YOUR_OUTPUT_DIRECTORY/"
presentation.save(out_dir + "charts_change_chart_category_axis_out.pptx", slides.export.SaveFormat.PPTX)
```

На этом этапе изменения записываются обратно в новый файл в указанном вами выходном каталоге.

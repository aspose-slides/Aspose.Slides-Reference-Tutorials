### Руководство по внедрению

#### Функция 1: Создание презентации с диаграммой

Эта функция продемонстрирует, как создать презентацию PowerPoint и добавить круговую диаграмму на первый слайд.

##### Добавление диаграммы

Начните с создания новой презентации и добавления круговой диаграммы в позицию (50, 50) на первом слайде:

```python
import aspose.slides as slides

with slides.Presentation() as presentation:
    # Добавить круговую диаграмму с указанными размерами
    chart = presentation.slides[0].shapes.add_chart(
        slides.charts.ChartType.PIE_OF_PIE, 50, 50, 500, 400)
```

##### Настройка меток данных

Для повышения читабельности настройте метки данных для отображения значений:

```python
# Включите отображение значений в метках данных для большей ясности
chart.chart_data.series[0].labels.default_data_label_format.show_value = True
```

##### Настройка параметров Pie of Pie

Настройте определенные свойства для круговой диаграммы, такие как размер второго круга и положение разделения:

```python
# Установите размер второй круговой диаграммы и свойства разделения
chart.chart_data.series[0].parent_series_group.second_pie_size = 149
chart.chart_data.series[0].parent_series_group.pie_split_by = slides.charts.PieSplitType.BY_PERCENTAGE
chart.chart_data.series[0].parent_series_group.pie_split_position = 53
```

##### Сохранение презентации

Наконец, сохраните презентацию в желаемом каталоге:

```python
# Сохраните презентацию с диаграммой
presentation.save("YOUR_OUTPUT_DIRECTORY/charts_second_plot_options_out.pptx", slides.export.SaveFormat.PPTX)
```

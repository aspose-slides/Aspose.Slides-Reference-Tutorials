### Создать пузырьковую диаграмму
#### Обзор
В этом разделе показано, как добавить пузырьковую диаграмму в существующую презентацию PowerPoint и настроить ее для включения меток данных, полученных непосредственно из определенных ячеек.
#### Пошаговые инструкции
##### 1. Загрузите файл презентации
Откройте файл презентации, в который вы хотите вставить пузырьковую диаграмму:
```python
import aspose.slides as slides

def create_bubble_chart_with_labels():
    # Определите тексты этикеток для ясности
    lbl0 = "Label 0 cell value"
    lbl1 = "Label 1 cell value"
    lbl2 = "Label 2 cell value"
    
    # Откройте файл презентации из определенного каталога.
    with slides.Presentation("YOUR_DOCUMENT_DIRECTORY/charts_workbook_as_datalabel.pptx") as pres:
        # Перейдите к следующему шагу...
```
*Объяснение*: Этот фрагмент кода открывает существующий файл PowerPoint. Заменить `"YOUR_DOCUMENT_DIRECTORY"` с вашим реальным путем.
##### 2. Добавьте пузырьковую диаграмму
Вставьте диаграмму в указанные координаты и размеры:
```python
        # Вставьте пузырьковую диаграмму в точке с координатами (50, 50) с размерами 600x400 пикселей.
        chart = pres.slides[0].shapes.add_chart(slides.charts.ChartType.BUBBLE, 50, 50, 600, 400, True)
```
*Объяснение*: `add_chart` Метод создает новую пузырьковую диаграмму. Отрегулируйте положение и размер по мере необходимости.
##### 3. Настройте метки данных
Настройте метки данных для отображения значений из определенных ячеек:
```python
        # Доступ к серии диаграммы
        series = chart.chart_data.series
        
        # Включить отображение значения метки непосредственно из ячейки
        series[0].labels.default_data_label_format.show_label_value_from_cell = True
        
        # Получить рабочую книгу, связанную с данными диаграммы.
        wb = chart.chart_data.chart_data_workbook
        
        # Назначьте значения меток для каждой точки в серии из определенных ячеек
        series[0].labels[0].value_from_cell = wb.get_cell(0, "A10", lbl0)
        series[0].labels[1].value_from_cell = wb.get_cell(0, "A11", lbl1)
        series[0].labels[2].value_from_cell = wb.get_cell(0, "A12", lbl2)
```
*Объяснение*: Этот раздел настраивает метки данных для каждой точки в диаграмме для отображения значений из определенных ячеек. При необходимости настройте ссылки на ячейки.
##### 4. Сохраните презентацию
Сохраните измененную презентацию:
```python
        # Сохранить изменения в новом файле в указанном выходном каталоге.
        pres.save("YOUR_OUTPUT_DIRECTORY/charts_workbook_as_datalabel_out.pptx", slides.export.SaveFormat.PPTX)
# Выполнить функцию создания диаграммы
create_bubble_chart_with_labels()
```
*Объяснение*: Это сохранит вашу презентацию с недавно добавленной и настроенной пузырьковой диаграммой.

- **Ошибки меток данных**: Дважды проверьте точность ссылок на ячейки, чтобы избежать неправильной конфигурации этикеток.

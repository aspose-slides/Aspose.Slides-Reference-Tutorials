### Создать и сохранить презентацию

#### Обзор

Мы создадим новый файл PowerPoint, добавим слайд, вставим карту-диаграмму, заполним ее данными, настроим ее внешний вид и сохраним конечный результат.

##### Инициализировать новую презентацию

Начните с инициализации вашей презентации:

```python
def create_and_save_presentation():
    """Create and save a presentation with a map chart."""
    # Инициализировать новый объект презентации
    with slides.Presentation() as presentation:
        pass  # Мы заполним остальную часть логики здесь.

create_and_save_presentation()
```

##### Добавить карту

Добавьте диаграмму типа КАРТА на первый слайд:

```python
with slides.Presentation() as presentation:
    # Вставьте карту-схему в позицию (50, 50) размером (500x400)
    chart = presentation.slides[0].shapes.add_chart(
        slides.charts.ChartType.MAP, 50, 50, 500, 400, False
    )
```

- **Параметры:** 
  - `ChartType.MAP`: Указывает тип диаграммы.
  - `(50, 50)`: Положение на слайде.
  - `(500x400)`: Размеры ширины и высоты.

##### Добавить ряды и точки данных

Заполните свою карту точками данных:

```python
wb = chart.chart_data.chart_data_workbook

# Добавить ряды и точки данных
to_series = chart.chart_data.series.add(slides.charts.ChartType.MAP)
to_series.data_points.add_data_point_for_map_series(wb.get_cell(0, "B2", 5))
to_series.data_points.add_data_point_for_map_series(wb.get_cell(0, "B3", 1))
to_series.data_points.add_data_point_for_map_series(wb.get_cell(0, "B4", 10))
```

- **Почему:** На этом этапе добавляются фактические данные, которые будут отображаться на вашей карте.

##### Определить категории для карты-схемы

Назначьте географические категории каждой точке данных:

```python
# Добавить категории
to_chart.chart_data.categories.add(wb.get_cell(0, "A2", "United States"))
to_chart.chart_data.categories.add(wb.get_cell(0, "A3", "Mexico"))
to_chart.chart_data.categories.add(wb.get_cell(0, "A4", "Brazil"))
```

- **Почему:** Это определяет регионы, которые представляют ваши точки данных.

##### Настроить внешний вид точки данных

Повысьте визуальную привлекательность, настроив точку данных:

```python
# Настроить внешний вид одной точки данных
data_point = to_series.data_points[1]
data_point.color_value.as_cell.value = "15"
data_point.format.fill.fill_type = slides.FillType.SOLID
data_point.format.fill.solid_fill_color.color = drawing.Color.green
```

- **Почему:** Усиление определенной точки данных помогает выделить ее.

##### Сохранить презентацию

Наконец, сохраните вашу презентацию:

```python
# Сохранить в указанном каталоге
presentation.save("YOUR_OUTPUT_DIRECTORY/charts_map_chart_out.pptx", slides.export.SaveFormat.PPTX)
```

- **Почему:** На этом этапе ваша работа записывается в файл, которым вы можете поделиться или представить.

## Руководство по внедрению
В этом разделе мы шаг за шагом рассмотрим, как настроить свойства шрифта диаграммы.

### Добавление кластеризованной столбчатой диаграммы
Сначала добавим в нашу презентацию кластеризованную столбчатую диаграмму:

```python
# Добавьте кластеризованную столбчатую диаграмму в указанном месте и размере.
chart = pres.slides[0].shapes.add_chart(
    slides.charts.ChartType.CLUSTERED_COLUMN, 100, 100, 500, 400
)
```
**Объяснение**: Этот фрагмент добавляет новую диаграмму к первому слайду вашей презентации. `add_chart` Метод требует указания типа диаграммы, ее положения и размера на слайде.

### Настройка свойств шрифта
Далее давайте установим высоту шрифта для текста в нашей диаграмме:

```python
# Установите высоту шрифта для текста в диаграмме.
chart.text_format.portion_format.font_height = 20
```
**Объяснение**: Эта строка регулирует размер шрифта всех текстовых частей в вашей диаграмме. `font_height` свойство указывается в баллах, и вы можете настроить это значение в соответствии с потребностями вашего дизайна.

### Отображение меток данных
Для повышения читабельности мы будем отображать значения на метках данных:

```python
# Отобразите значения на метках данных первой серии.
chart.chart_data.series[0].labels.default_data_label_format.show_value = True
```
**Объяснение**: Эта настройка гарантирует, что каждая точка данных в первой серии показывает свое значение. Это особенно полезно для передачи точной информации с одного взгляда.

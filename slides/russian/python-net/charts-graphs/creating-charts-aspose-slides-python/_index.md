## Руководство по внедрению

### Функция 1: Создание и настройка презентации с диаграммами
**Обзор:** В этом разделе вы узнаете, как настроить слайд презентации и добавить к нему диаграмму с помощью Aspose.Slides Python.

#### Шаг 1: Инициализация презентации
Начните с создания нового объекта презентации. Используйте `with` заявление об автоматическом управлении ресурсами:
```python
import aspose.slides as slides

with slides.Presentation() as presentation:
    # Доступ к первому слайду презентации
    slide = presentation.slides[0]
```

#### Шаг 2: Добавьте диаграмму на слайд
Здесь мы добавляем столбчатую диаграмму с накоплением в указанной позиции с определенными размерами:
```python
# Добавьте на слайд столбчатую диаграмму с накоплением
chart = slide.shapes.add_chart(slides.charts.ChartType.PERCENTS_STACKED_COLUMN, 20, 20, 500, 400)
```

#### Шаг 3: Настройка осей диаграммы
Настройте числовой формат вертикальной оси для лучшего представления данных:
```python
# Настройте формат чисел вертикальной оси
chart.axes.vertical_axis.is_number_format_linked_to_source = False
chart.axes.vertical_axis.number_format = "0.00%"
```

### Функция 2: Добавление и форматирование рядов данных на диаграмму
**Обзор:** В этом разделе основное внимание уделяется добавлению ряда данных, заполнению его значениями и настройке его внешнего вида.

#### Шаг 1: Определите рабочую книгу данных
Инициализируйте рабочую книгу данных вашей диаграммы:
```python
default_worksheet_index = 0
workbook = chart.chart_data.chart_data_workbook
```

#### Шаг 2: Добавьте и заполните ряды данных
Добавьте в диаграмму новую серию под названием «Красные», затем заполните ее точками данных:
```python
# Добавьте новую серию и заполните ее точками данных.
series = chart.chart_data.series.add(workbook.get_cell(default_worksheet_index, 0, 1, "Reds"), chart.type)

for i in range(1, 5):
    series.data_points.add_data_point_for_bar_series(
        workbook.get_cell(default_worksheet_index, i, 1, [0.30, 0.50, 0.80, 0.65][i-1])
    )
```

#### Шаг 3: Отформатируйте внешний вид серии
Настройте цвет заливки и формат метки данных:
```python
# Установить заливку серии красным цветом
series.format.fill.fill_type = slides.FillType.SOLID
series.format.fill.solid_fill_color.color = drawing.Color.red

# Настройте метки данных для отображения процентов
series.labels.default_data_label_format.show_value = True
series.labels.default_data_label_format.number_format = "0.0%"
```

### Функция 3: Добавление и форматирование второго ряда данных на диаграмму
**Обзор:** В этом разделе подробно рассматривается добавление второго ряда данных с собственным стилем.

#### Шаг 1: Добавьте вторую серию
Добавьте еще одну серию под названием «Блюз»:
```python
# Добавить вторую серию под названием «Блюз»
series2 = chart.chart_data.series.add(workbook.get_cell(default_worksheet_index, 0, 2, "Blues"), chart.type)
```

#### Шаг 2: Заполнение и форматирование серии
Заполните его точками данных и примените форматирование:
```python
# Заполнить вторую серию
for i in range(1, 5):
    series2.data_points.add_data_point_for_bar_series(
        workbook.get_cell(default_worksheet_index, i, 2, [0.70, 0.50, 0.20, 0.35][i-1])
    )

# Установите заливку на синий цвет и настройте метки
series2.format.fill.fill_type = slides.FillType.SOLID
series2.format.fill.solid_fill_color.color = drawing.Color.blue

series2.labels.default_data_label_format.show_value = True
```

### Функция 4: Сохранение презентации на диске
**Обзор:** После настройки диаграммы сохраните презентацию.

#### Шаг 1: Сохраните свою работу
Используйте `save` метод сохранения вашего файла:
```python
# Сохранить презентацию на диск
directory = "YOUR_OUTPUT_DIRECTORY"
presentation.save(f"{directory}/charts_set_data_labels_percentage_sign_out.pptx", slides.export.SaveFormat.PPTX)
```

### Создание пузырьковой диаграммы с планками погрешностей

#### Шаг 1: Добавьте пузырьковую диаграмму в презентацию

Начните с создания пузырьковой диаграммы на первом слайде. Это служит основой для добавления планок погрешностей:

```python
# Доступ к первому слайду презентации
class SlideAccess:
    def __init__(self, presentation):
        self.first_slide = presentation.slides[0]

    def add_bubble_chart(self):
        # Добавьте пузырьковую диаграмму в позицию (50, 50) шириной 400 и высотой 300.
        self.chart = self.first_slide.shapes.add_chart(
            slides.charts.ChartType.BUBBLE, 50, 50, 400, 300, True)
```

#### Шаг 2: Доступ к планкам погрешностей

Вам необходимо получить доступ к планкам погрешностей как для оси X, так и для оси Y:

```python
class ErrorBarsAccess:
    def __init__(self, chart):
        self.err_bar_x = chart.chart_data.series[0].error_bars_x_format
        self.err_bar_y = chart.chart_data.series[0].error_bars_y_format
```

#### Шаг 3: Установите видимость планок погрешностей

Убедитесь, что полосы погрешностей видны:

```python
class ErrorBarsVisibility:
    def __init__(self, err_bar_x, err_bar_y):
        self.err_bar_x.is_visible = True
        self.err_bar_y.is_visible = True
```

#### Шаг 4: Настройте планки погрешностей оси X с фиксированными значениями

Установите фиксированный тип значения для планок погрешностей оси X, которые будут отображать постоянные значения погрешностей:

```python
class ConfigureXErrorBars:
    def __init__(self, err_bar_x):
        # Установите планку погрешности оси X, чтобы использовать фиксированные значения.
        self.err_bar_x.value_type = slides.charts.ErrorBarValueType.FIXED
        self.err_bar_x.value = 0.1  # Погрешность составляет 0,1 единицы.

        # Определите тип как PLUS и добавьте заглушки для визуальной ясности.
        self.err_bar_x.type = slides.charts.ErrorBarType.PLUS
        self.err_bar_x.has_end_cap = True
```

#### Шаг 5: Настройте планки погрешностей оси Y с процентными значениями

Для оси Y используйте процентные значения для представления изменчивости:

```python
class ConfigureYErrorBars:
    def __init__(self, err_bar_y):
        # Установите планку погрешности оси Y, чтобы использовать процентные значения.
        self.err_bar_y.value_type = slides.charts.ErrorBarValueType.PERCENTAGE
        self.err_bar_y.value = 5  # Погрешность 5%

        # Настройте ширину линии для лучшей видимости
        self.err_bar_y.format.line.width = 2
```

Наконец, сохраните презентацию в указанном каталоге:

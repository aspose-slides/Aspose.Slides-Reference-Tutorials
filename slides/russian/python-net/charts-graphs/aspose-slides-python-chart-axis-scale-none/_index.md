# Как установить масштаб оси диаграммы на NONE с помощью Aspose.Slides Python
## Введение
Создание визуально привлекательных диаграмм часто требует тонкой настройки масштабов их осей. В этом руководстве демонстрируется настройка масштаба основных единиц горизонтальной оси на `NONE` для диаграммы с использованием Aspose.Slides в Python, идеально подходящей для настройки визуализации данных в ваших презентациях.
**Что вы узнаете:**
- Настройте Aspose.Slides для Python.
- Создавайте и настраивайте диаграммы с определенными конфигурациями осей.
- Сохраняйте презентации программным способом.
- Устранение распространенных проблем при работе с осями диаграммы.

## Руководство по внедрению
### Создание диаграммы с пользовательской шкалой осей
#### Обзор
Мы создадим диаграмму типа AREA и установим ее масштаб основных единиц горизонтальной оси на `NONE`.
**Шаг 1: Инициализация презентации**
Начните с создания нового экземпляра презентации:
```python
with slides.Presentation() as pres:
    # Дальнейшие операции будут проводиться здесь.
```
Этот менеджер контекста обеспечивает эффективное управление ресурсами.
#### Шаг 2: Добавьте диаграмму
Добавьте на слайд диаграмму типа ОБЛАСТЬ с определенными координатами и размерами:
```python
chart = pres.slides[0].shapes.add_chart(slides.charts.ChartType.AREA, 10, 10, 400, 300, True)
```
Это добавит диаграмму размером 400x300 пикселей в позицию (10, 10) на первом слайде.
#### Шаг 3: Установите для параметра Масштаб оси значение ОТСУТСТВУЕТ
Измените масштаб основных единиц горизонтальной оси:
```python
chart.axes.horizontal_axis.major_unit_scale = slides.charts.TimeUnitType.NONE
```
Установка этого свойства удаляет предопределенные интервалы масштабирования по оси x.
#### Шаг 4: Сохраните презентацию
Сохраните изменения в файле в формате PPTX:
```python
pres.save("YOUR_OUTPUT_DIRECTORY/charts_time_unit_type_enum_out.pptx", slides.export.SaveFormat.PPTX)
```
Это позволит сохранить настроенную вами диаграмму в новом файле презентации.

## Раздел часто задаваемых вопросов
1. **Как обновить Aspose.Slides?**
   - Использовать `pip install --upgrade aspose.slides`.
2. **Можно ли установить значение NONE для масштабов как горизонтальной, так и вертикальной оси?**
   - Да, используйте `chart.axes.vertical_axis.major_unit_scale = slides.charts.TimeUnitType.NONE`.
3. **Что делать, если моя диаграмма не сохраняется должным образом?**
   - Проверьте пути к файлам и убедитесь, что выходной каталог доступен для записи.
4. **Есть ли возможность просмотреть изменения перед сохранением?**
   - Aspose.Slides не обеспечивает прямого предварительного просмотра, но выполняет итерацию с меньшими скриптами до тех пор, пока не будет достигнут результат.
5. **Как работать с различными типами диаграмм?**
   - Заменять `ChartType.AREA` с другими типами, такими как `Bar`, `Line`и т.д. по мере необходимости.

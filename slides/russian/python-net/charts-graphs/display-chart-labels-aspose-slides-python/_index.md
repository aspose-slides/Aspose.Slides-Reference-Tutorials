## Руководство по внедрению
Чтобы отобразить подписи диаграмм в презентации PowerPoint, выполните следующие действия.

### Шаг 1: Создайте новую презентацию и слайд
Создайте новую презентацию и добавьте слайд:

```python
def display_chart_labels():
    with slides.Presentation() as presentation:
        # Откройте первый слайд (по умолчанию он создан).
        slide = presentation.slides[0]
```

### Шаг 2: Добавьте круговую диаграмму на слайд
Добавить круговую диаграмму в позицию `(50, 50)` с размерами `500x400`:

```python
        # Добавляем круговую диаграмму на первый слайд.
        chart = slide.shapes.add_chart(
            slides.charts.ChartType.PIE, 50, 50, 500, 400)
```

### Шаг 3: Настройте параметры отображения меток
Настройте свойства метки для лучшей визуализации данных:
- **Показать метки значений**: Отображение числовых значений на каждом срезе.
- **Вызовы данных**: Используйте выносные линии для соединения меток с фрагментами.

```python
        # Настройте параметры отображения меток серий диаграмм
        series_labels = chart.chart_data.series[0].labels.default_data_label_format
        series_labels.show_value = True  # Показывать метки значений по умолчанию
        series_labels.show_label_as_data_callout = True  # Используйте выноски данных
```

### Шаг 4: Настройте определенные метки
Отключите выноску данных для определенных меток, например, для третьей метки:

```python
        # Переопределить настройку выноски данных для определенной метки
        chart.chart_data.series[0].labels[2].data_label_format.show_label_as_data_callout = False
```

### Шаг 5: Сохраните презентацию
Сохраните презентацию в выходном каталоге с желаемым именем файла:

```python
        # Сохраните улучшенную презентацию
        presentation.save("YOUR_OUTPUT_DIRECTORY/charts_display_chart_labels_out.pptx")
```

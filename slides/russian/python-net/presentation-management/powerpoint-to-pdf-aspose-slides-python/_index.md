## Руководство по внедрению

### Конвертировать PowerPoint в PDF с соблюдением A1A

PDF/A-1a идеально подходит для архивирования и долгосрочного хранения. Выполните следующие шаги:

#### Шаг 1: Загрузите презентацию

Загрузите файл PowerPoint:

```python
import aspose.slides as slides
presentation_path = 'YOUR_DOCUMENT_DIRECTORY/welcome-to-powerpoint.pptx'
with slides.Presentation(presentation_path) as presentation:
    # Последующие шаги будут...
```

#### Шаг 2: Настройте параметры PDF

Установите соответствие PDF/A-1a:

```python
class_pdf_options = slides.export.PdfOptions()
class_pdf_options.compliance = slides.export.PdfCompliance.PDF_A1A
```

#### Шаг 3: Сохранить как совместимый PDF-файл

Сохраните презентацию с указанными параметрами:

```python
output_path_a1a = 'YOUR_OUTPUT_DIRECTORY/convert_to_pdf_a1a_out.pdf'
presentation.save(output_path_a1a, slides.export.SaveFormat.PDF, class_pdf_options)
```

### Конвертировать PowerPoint в PDF с соблюдением A1B

PDF/A-1b ориентирован на визуальное воспроизведение без внедрения метаданных.

#### Шаг 1: Загрузите презентацию

Этот шаг остается таким же, как и для PDF/A-1a.

#### Шаг 2: Настройте параметры PDF

Установить соответствие PDF/A-1b:

```python
class_pdf_options = slides.export.PdfOptions()
class_pdf_options.compliance = slides.export.PdfCompliance.PDF_A1B
```

#### Шаг 3: Сохранить как совместимый PDF-файл

Сохраните файл по указанному пути:

```python
output_path_a1b = 'YOUR_OUTPUT_DIRECTORY/convert_to_pdf_a1b_out.pdf'
presentation.save(output_path_a1b, slides.export.SaveFormat.PDF, class_pdf_options)
```

### Конвертируйте PowerPoint в PDF с помощью Compliance UA

PDF/UA обеспечивает доступность для всех пользователей, включая людей с ограниченными возможностями.

#### Шаг 1: Загрузите презентацию

Повторите начальный шаг, как и прежде.

#### Шаг 2: Настройте параметры PDF

Установить соответствие PDF/UA:

```python
class_pdf_options = slides.export.PdfOptions()
class_pdf_options.compliance = slides.export.PdfCompliance.PDF_UA
```

#### Шаг 3: Сохранить как совместимый PDF-файл

Сохраните презентацию с новыми настройками соответствия:

```python
output_path_ua = 'YOUR_OUTPUT_DIRECTORY/convert_to_pdf_ua_out.pdf'
presentation.save(output_path_ua, slides.export.SaveFormat.PDF, class_pdf_options)
```

## Руководство по внедрению

В этом разделе мы разобьем процесс конвертации на управляемые этапы.

### Конвертация PPTX в HTML с сохранением оригинальных шрифтов

#### Обзор:
Основная функция этой реализации — конвертация презентации PowerPoint с сохранением ее оригинальных шрифтов и исключением определенных шрифтов по умолчанию из встраивания. Это может быть особенно полезно для поддержания единообразия бренда в веб-презентациях.

#### Пошаговая реализация:

**1. Определите входные и выходные пути**

Настройте каталоги, в которых будет находиться входной файл PPTX и куда вы хотите сохранить выходной файл HTML.

```python
data_dir = "YOUR_DOCUMENT_DIRECTORY/"
out_dir = "YOUR_OUTPUT_DIRECTORY/"
```

**2. Откройте файл презентации.**

Используйте Aspose.Slides `Presentation` класс для загрузки вашего файла PPTX:

```python
with slides.Presentation(data_dir + "welcome-to-powerpoint.pptx") as pres:
    # Ваш код конвертации будет здесь.
```

Этот менеджер контекста обеспечивает правильное высвобождение ресурсов после операции.

**3. Создайте пользовательский контроллер внедрения шрифтов**

Исключите определенные шрифты из встраивания с помощью `EmbedAllFontsHtmlController`:

```python
font_name_exclude_list = ["Calibri", "Arial"]
embed_fonts_controller = slides.export.EmbedAllFontsHtmlController(font_name_exclude_list)
```

Здесь «Calibri» и «Arial» исключены из встраивания в вывод HTML.

**4. Настройте параметры экспорта HTML**

Настраивать `HtmlOptions` чтобы использовать пользовательский форматировщик шрифтов с вашим контроллером:

```python
html_options_embed = slides.export.HtmlOptions()
html_options_embed.html_formatter = slides.export.HtmlFormatter.create_custom_formatter(embed_fonts_controller)
```

Этот шаг гарантирует, что в конечный результат будут встроены только необходимые шрифты.

**5. Сохраните презентацию как HTML**

Наконец, сохраните презентацию в HTML-файле с указанными вами параметрами:

```python
pres.save(out_dir + "convert_to_html_with_preserving_original_fonts_out.html", 
          slides.export.SaveFormat.HTML, html_options_embed)
```

### Советы по устранению неполадок:

- Проверьте наличие отсутствующих файлов шрифтов в системе, которые могут повлиять на преобразование.

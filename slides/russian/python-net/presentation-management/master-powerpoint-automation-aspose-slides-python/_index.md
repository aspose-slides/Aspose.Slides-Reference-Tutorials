## Руководство по внедрению

Теперь, когда вы разобрались с предварительными условиями, давайте углубимся в работу с презентациями PowerPoint.

### Открытие и клонирование слайдов

#### Обзор

В этом разделе мы откроем существующий файл PowerPoint и клонируем слайд, содержащий элемент управления ActiveX, в новый экземпляр презентации.

#### Шаги

**Шаг 1: Откройте существующий файл PowerPoint.**

Начните с открытия целевого файла PowerPoint с помощью `Presentation` сорт:

```python
with slides.Presentation(YOUR_DOCUMENT_DIRECTORY + "activex_template.pptx") as pres:
    # Доступ к вашей существующей презентации здесь
```

**Шаг 2: Удалить слайд по умолчанию**

Создайте новую презентацию и удалите ее слайд по умолчанию, чтобы подготовить ее к клонированию:

```python
new_pres = slides.Presentation()
new_pres.slides.remove_at(0)
```

**Шаг 3: Клонируйте слайд с помощью элемента управления ActiveX**

Клонируйте определенный слайд из исходной презентации в новую:

```python
new_pres.slides.insert_clone(0, pres.slides[0])
```

### Изменение элементов управления ActiveX

#### Обзор

Элементы управления ActiveX могут быть мощными инструментами в слайдах. Здесь мы изменим существующий элемент управления Media Player.

#### Шаги

**Шаг 4: Доступ и изменение свойств элемента управления**

Получите доступ к первому элементу управления на клонированном слайде и измените его свойства:

```python
control = new_pres.slides[0].controls[0]
control.properties.remove("URL")
control.properties.add("URL", YOUR_DOCUMENT_DIRECTORY + "video.mp4")
```

### Сохранение презентации

#### Обзор

После того, как вы отредактировали слайды, пришло время сохранить измененную презентацию.

**Шаг 5: Сохраните презентацию**

```python
new_pres.save(YOUR_OUTPUT_DIRECTORY + "activex_linking_video_activex_control_out.pptx", slides.export.SaveFormat.PPTX)
```

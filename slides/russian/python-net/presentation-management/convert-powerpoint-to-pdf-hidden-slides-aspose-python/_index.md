## Руководство по внедрению: преобразование PPTX в PDF со скрытыми слайдами

### Обзор функции
Эта функция позволяет вам преобразовать презентацию PowerPoint в файл PDF, гарантируя, что все скрытые слайды будут включены в вывод. Это особенно полезно, когда каждый фрагмент контента необходимо сохранить для архивирования или совместного использования.

#### Шаг 1: Загрузите презентацию
Начните с загрузки файла PPTX с помощью `Presentation` сорт.

```python
with slides.Presentation("YOUR_DOCUMENT_DIRECTORY/presentation_with_hidden_slides.pptx") as presentation:
    # Дальнейшая обработка будет происходить здесь
```

#### Шаг 2: Настройте параметры PDF
Создать экземпляр `PdfOptions` объект для указания параметров преобразования PDF. Здесь вы зададите опцию включения скрытых слайдов.

```python
class PdfOptions:
    def __init__(self):
        self.показывать_скрытые_слайды = False

pdf_options = PdfOptions()
pdf_options.show_hidden_slides = True
```

- **show_hidden_slides**: Этот параметр имеет решающее значение, поскольку он определяет, будут ли включены скрытые слайды в выходной PDF-файл.

#### Шаг 3: Сохраните презентацию
Наконец, сохраните презентацию в формате PDF-файла с указанными параметрами.

```python
target_directory = "YOUR_OUTPUT_DIRECTORY"
presentation.save(f"{target_directory}/convert_to_pdf_hidden_slides_out.pdf", \
                 slides.export.SaveFormat.PDF, pdf_options)
```

### Советы по устранению неполадок
- **Ошибки пути к файлу**Убедитесь, что пути для входных и выходных файлов верны. Используйте абсолютные пути, если относительные вызывают проблемы.

### Преобразование презентации в PDF

#### Обзор

Конвертируйте файлы .pptx в PDF-файлы без особых усилий, обеспечивая совместимость с различными платформами.

#### Пошаговая реализация

**1. Загрузите презентацию**

Загрузите файл PowerPoint из определенного каталога:
```python
def load_presentation(input_file_path):
    presentation = slides.Presentation(input_file_path)
    return presentation
```

**2. Сохранить как PDF**

Сохраните загруженную презентацию как PDF-файл:
```python
def save_as_pdf(presentation, output_file_path):
    presentation.save(output_file_path, slides.export.SaveFormat.PDF)
```

#### Пример полного кода

Объедините эти шаги в законченную функцию:
```python
import aspose.slides as slides

def convert_to_pdf(input_file_path, output_file_path):
    with slides.Presentation(input_file_path) as presentation:
        presentation.save(output_file_path, slides.export.SaveFormat.PDF)

# Пример использования
convert_to_pdf("path/to/presentation.pptx", "output/path/output.pdf")
```

**Объясняемые параметры:**
- `input_file_path`: Путь к исходному файлу PowerPoint.
- `output_file_path`: Желаемый путь для конечного PDF-файла.

**Советы по устранению неполадок:**
- Проверьте правильность и доступность путей к входным файлам.
- Проверьте наличие проблем с разрешениями при записи в выходной каталог.

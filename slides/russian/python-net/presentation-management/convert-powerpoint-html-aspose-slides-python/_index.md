## Руководство по внедрению

### Конвертировать презентацию в HTML со встроенными изображениями
Эта функция помогает вам интегрировать содержимое презентации непосредственно в ваши веб-страницы путем встраивания изображений в HTML-файл.

#### Обзор
Встраивание изображений гарантирует, что все визуальные элементы будут содержаться в одном HTML-документе, устраняя необходимость во внешних файлах изображений. Этот метод особенно полезен для автономных документов или при обеспечении автономной доступности презентаций.

#### Шаги
1. **Настроить выходной каталог**
   Определите, где будут храниться преобразованный HTML-код и ресурсы:
   ```python
   content_dir = "YOUR_OUTPUT_DIRECTORY/HTMLConversion/"
   ```

2. **Открыть презентацию PowerPoint**
   Загрузите файл презентации с помощью Aspose.Slides:
   ```python
   with slides.Presentation("YOUR_DOCUMENT_DIRECTORY/PresentationDemo.pptx") as pres:
       # Настройка для преобразования HTML выполняется следующим образом.
   ```

3. **Настроить параметры HTML**
   Установите параметры для встраивания изображений в результирующий HTML-документ:
   ```python
   html5_options = slides.export.Html5Options()
   html5_options.embed_images = True
   html5_options.output_path = "YOUR_OUTPUT_DIRECTORY/"
   ```

4. **Убедитесь, что каталог существует**
   Создайте выходной каталог, если он не существует, корректно обрабатывая любые исключения:
   ```python
   import os

   try:
       os.rmdir(content_dir)
   except OSError:
       pass  # Каталог может не существовать или не быть пустым

   os.makedirs(content_dir, exist_ok=True)
   ```

5. **Сохранить как HTML**
   Конвертируйте и сохраните вашу презентацию:
   ```python
   pres.save(content_dir + "pres.html", slides.export.SaveFormat.HTML5, html5_options)
   ```

#### Ключевые соображения
- Убедитесь, что пути заданы правильно, чтобы предотвратить ошибки «файл не найден».
- Грамотно обрабатывайте исключения при управлении каталогами.

### Конвертировать презентацию в HTML без встроенных изображений
Этот метод связывает изображения с внешними объектами, что может быть полезно для уменьшения размера HTML-документа или при работе с большими презентациями.

#### Обзор
Связывая изображения вместо их встраивания, вы сохраняете HTML-файл легким и разделяете файлы изображений в назначенном каталоге. Это идеально подходит для веб-сред, где использование полосы пропускания является проблемой.

#### Шаги
1. **Настроить выходной каталог**
   Аналогично предыдущей функции:
   ```python
   content_dir = "YOUR_OUTPUT_DIRECTORY/HTMLConversion/"
   ```

2. **Открыть презентацию PowerPoint**
   Загрузите файл презентации с помощью Aspose.Slides:
   ```python
   with slides.Presentation("YOUR_DOCUMENT_DIRECTORY/PresentationDemo.pptx") as pres:
       # Настройка для преобразования HTML выполняется следующим образом.
   ```

3. **Настроить параметры HTML**
   Установите параметры для внешней ссылки изображений в результирующем HTML-документе:
   ```python
   html5_options = slides.export.Html5Options()
   html5_options.embed_images = False
   html5_options.output_path = "YOUR_OUTPUT_DIRECTORY/"
   ```

4. **Убедитесь, что каталог существует**
   Создайте выходной каталог, если он не существует, корректно обрабатывая любые исключения:
   ```python
   try:
       os.rmdir(content_dir)
   except OSError:
       pass  # Каталог может не существовать или не быть пустым

   os.makedirs(content_dir, exist_ok=True)
   ```

5. **Сохранить как HTML**
   Конвертируйте и сохраните вашу презентацию:
   ```python
   pres.save(content_dir + "pres.html", slides.export.SaveFormat.HTML5, html5_options)
   ```

#### Ключевые соображения
- Проверьте пути к внешним ресурсам, чтобы убедиться, что они правильно связаны.
- Эффективно управляйте большим количеством изображений, организуя их в каталоги.

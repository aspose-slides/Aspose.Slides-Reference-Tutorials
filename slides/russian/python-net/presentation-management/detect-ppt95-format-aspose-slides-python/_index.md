## Руководство по внедрению

### Определение формата PPT95

Чтобы определить, находится ли презентация в старом формате PPT (PPT95), выполните следующие действия:

#### Пошаговая реализация

**1. Получить информацию о презентации**

Загрузите информацию о презентации с помощью Aspose.Slides:

```python
import aspose.slides as slides

def check_presentation_format():
    # Замените «YOUR_DOCUMENT_DIRECTORY/» на путь к вашему каталогу.
    load_info = slides.PresentationFactory.instance.get_presentation_info(
        "YOUR_DOCUMENT_DIRECTORY/open_presentation.ppt")
```

*Объяснение:* Мы используем `PresentationFactory` для получения деталей презентации. Метод `get_presentation_info` считывает метаданные файла, включая его формат.

**2. Определите формат**

Проверьте, является ли загруженный формат PPT95:

```python
    # Проверьте, имеет ли презентация формат PPT95.
is_old_format = load_info.load_format == slides.LoadFormat.PPT95

return is_old_format
```

*Объяснение:* Сравнивая `load_info.load_format` с `slides.LoadFormat.PPT95`, мы определяем, находится ли файл в старом формате PPT.

### Советы по устранению неполадок

- **Ошибки пути к файлу:** Убедитесь, что путь к каталогу и имя файла указаны правильно.
- **Проблемы с установкой:** Проверьте версии pip и Python. Используйте `pip --version` чтобы проверить правильность установки pip.
- **Проблемы с лицензией:** Дважды проверьте путь к лицензии и убедитесь, что он применен, прежде чем запускать скрипт.

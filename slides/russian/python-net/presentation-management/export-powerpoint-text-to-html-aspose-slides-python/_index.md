### Обзор функции

Цель — извлечь текст из определенного слайда презентации PowerPoint и сохранить его как HTML-файл с помощью Aspose.Slides для Python.

### Пошаговые инструкции

#### 1. Загрузите презентацию (H3)

Загрузите файл PowerPoint:

```python
import aspose.slides as slides

def exporting_html_text():
    # Загрузить презентацию
    with slides.Presentation("YOUR_DOCUMENT_DIRECTORY/text_export_text_frame_to_html.pptx") as pres:
        pass  # Дальнейшая обработка здесь
```

#### 2. Получите доступ к желаемому слайду (H3)

Откройте слайд, из которого вы хотите экспортировать текст:

```python
        # Доступ к первому слайду
        slide = pres.slides[0]
```

#### 3. Определите и получите доступ к фигуре, содержащей текст (H3)

Определите, какая фигура содержит текст на целевом слайде:

```python
        # Индекс для доступа к определенной фигуре на слайде
        index = 0

        # Доступ к форме по указанному индексу
        auto_shape = slide.shapes[index]
```

#### 4. Экспорт текста в HTML (H3)

Экспортируйте текст из идентифицированной фигуры и сохраните его как HTML-файл:

```python
        # Открыть HTML-файл в режиме записи
        with open("YOUR_OUTPUT_DIRECTORY/text_export_text_frame_to_html_out.html", "wt") as sw:
            # Экспорт текстового фрейма из абзацев в формат HTML
            data = auto_shape.text_frame.paragraphs.export_to_html(0, auto_shape.text_frame.paragraphs.count, None)
            
            # Запишите экспортированный HTML-контент в файл.
            sw.write(data)
```

### Объяснение

- **Загрузка презентации:** The `Presentation` класс загружает ваш файл PPTX.
- **Доступ к фигурам и текстовым фреймам:** Получайте доступ к определенным фигурам с помощью их индекса, чтобы точно определять текстовые фреймы для экспорта.
- **Функциональность экспорта:** `export_to_html()` извлекает текст в формате HTML, который затем записывается в выходной файл.

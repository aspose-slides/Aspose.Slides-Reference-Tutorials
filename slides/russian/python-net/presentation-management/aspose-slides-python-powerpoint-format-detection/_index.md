## Руководство по внедрению

### Функция определения формата файла

Давайте рассмотрим, как определить формат файла PowerPoint с помощью Aspose.Slides.

#### Шаг 1: Получите доступ к информации о презентации

Сначала ознакомьтесь с подробностями презентации:

```python
def get_file_format(document_path):
    info = slides.PresentationFactory.instance.get_presentation_info(document_path)
```

Это позволит получить метаданные о вашем файле, которые имеют решающее значение для идентификации формата.

#### Шаг 2: Определите формат файла

Далее проверьте, является ли файл PPTX или неизвестным:

```python
def get_file_format(document_path):
    info = slides.PresentationFactory.instance.get_presentation_info(document_path)
    if info.load_format == slides.LoadFormat.PPTX:
        return "pptx"
    elif info.load_format == slides.LoadFormat.UNKNOWN:
        return "unknown"

# Пример использования:
document_path = "YOUR_DOCUMENT_DIRECTORY/welcome-to-powerpoint.pptx"
file_format = get_file_format(document_path)
print(file_format)
```

**Объяснение**: `get_presentation_info` Метод извлекает формат загрузки файла. Мы сравниваем его с известными константами, чтобы определить, является ли он PPTX или неизвестным форматом.

## Руководство по внедрению

### Конвертируйте PowerPoint в HTML со встроенными шрифтами

В этом разделе вы узнаете, как встроить шрифты при экспорте презентации PowerPoint в виде HTML-файла.

#### Обзор

Цель состоит в том, чтобы преобразовать ваш `.pptx` файлы в `.html`, гарантируя, что все шрифты, используемые в исходном документе, будут внедрены в вывод. Это обеспечивает согласованность в различных средах и устройствах.

#### Пошаговая реализация

##### Открыть файл презентации

Начните с открытия презентации PowerPoint, которую вы хотите преобразовать:

```python
document_path = "YOUR_DOCUMENT_DIRECTORY/welcome-to-powerpoint.pptx"
with slides.Presentation(document_path) as pres:
    # Дальнейшая обработка будет происходить здесь.
```

Этот фрагмент кода загружает ваш файл PowerPoint в память, готовый к преобразованию.

##### Настроить встраивание шрифтов

Чтобы встроить все шрифты, используемые в презентации:

```python
# Создайте список шрифтов для исключения (оставьте пустым, если хотите включить все)
font_name_exclude_list = []

# Инициализируйте объект EmbedAllFontsHtmlController со списком исключений
embed_fonts_controller = slides.export.EmbedAllFontsHtmlController(font_name_exclude_list)
```

Такая настройка гарантирует, что каждый шрифт, используемый в презентации, будет включен в вывод HTML.

##### Настройте параметры экспорта HTML

Далее настройте параметры экспорта для использования пользовательского форматировщика:

```python
html_options_embed = slides.export.HtmlOptions()
html_options_embed.html_formatter = slides.export.HtmlFormatter.create_custom_formatter(embed_fonts_controller)
```

Здесь мы настраиваем способ преобразования файла PowerPoint в HTML путем внедрения шрифтов.

##### Сохранить как HTML со встроенными шрифтами

Наконец, сохраните вашу презентацию в формате HTML со всеми встроенными шрифтами:

```python
output_path = "YOUR_OUTPUT_DIRECTORY/convert_to_html_with_embed_all_fonts_out.html"
pres.save(output_path, slides.export.SaveFormat.HTML, html_options_embed)
```

На этом этапе преобразованный файл сохраняется в указанном вами каталоге.

### Советы по устранению неполадок

- **Отсутствующие шрифты**: Убедитесь, что все шрифты, используемые в презентации, установлены в вашей системе.

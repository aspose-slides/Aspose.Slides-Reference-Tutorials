Конвертация презентаций в формат PDF гарантирует, что их можно будет просматривать на любом устройстве без дополнительного программного обеспечения.

Этот процесс сохраняет визуальную компоновку и качество ваших слайдов, что делает их идеальными для распространения на разных платформах.

#### Шаги

##### Загрузите вашу презентацию

Начните с загрузки файла PowerPoint, как показано выше.

##### Сохранить как PDF

Используйте этот фрагмент, чтобы сохранить вашу презентацию в формате PDF:

```python
def convert_to_pdf():
    presentation_path = 'YOUR_DOCUMENT_DIRECTORY/welcome-to-powerpoint.pptx'
    with slides.Presentation(presentation_path) as pres:
        output_path = 'YOUR_OUTPUT_DIRECTORY/convert_to_pdf_out.pdf'
        pres.save(output_path, slides.export.SaveFormat.PDF)
```

#### Советы по устранению неполадок
- Дважды проверьте пути к каталогам, чтобы избежать ошибок.
- Если во время конвертации вы столкнетесь с какими-либо ограничениями, убедитесь, что ваша лицензия активна.

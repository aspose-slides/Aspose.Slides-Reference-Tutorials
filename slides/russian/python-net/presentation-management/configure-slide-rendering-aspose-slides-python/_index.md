## Руководство по внедрению

Давайте рассмотрим реализацию, сосредоточившись на настройке параметров рендеринга.

### Настройка параметров визуализации слайдов

#### Обзор
В этом разделе демонстрируется настройка различных параметров рендеринга для слайда презентации. Он включает в себя настройку параметров макета для заметок и комментариев и сохранение слайдов в виде изображений.

#### Пошаговая реализация
**Шаг 1**: Загрузить файл презентации

```python
with slides.Presentation("YOUR_DOCUMENT_DIRECTORY/rendering_options.pptx") as pres:
    # Инициализируйте параметры рендеринга.
```
Загрузите файл PowerPoint для работы с ним с помощью `Presentation` сорт.

**Шаг 2**: Настройка параметров макета

```python
rendering_opts = slides.export.RenderingOptions()
slides_layout_options = slides.export.NotesCommentsLayoutingOptions()
slides_layout_options.notes_position = slides.export.NotesPositions.BOTTOM_TRUNCATED
rendering_opts.slides_layout_options = slides_layout_options
```
The `RenderingOptions` класс позволяет устанавливать различные конфигурации, включая макет заметок и комментариев. Здесь мы устанавливаем позицию заметок на `BOTTOM_TRUNCATED`.

**Шаг 3**: Сохранить слайд как изображение

```python
pres.slides[0].get_image(rendering_opts, 4 / 3, 4 / 3).save(
    "YOUR_OUTPUT_DIRECTORY/rendering_options-Original.png", slides.ImageFormat.PNG)
```
Сохраните первый слайд как изображение, используя настроенные параметры рендеринга.

### Настройка позиции заметок на «Нет»

#### Обзор
Изменение макета заметок может изменить восприятие вашей презентации. В этом разделе основное внимание уделяется изменению настроек макета заметок.

**Шаг 1**: Изменить положение заметок

```python
slides_layout_options.notes_position = slides.export.NotesPositions.NONE
rendering_opts.slides_layout_options = slides_layout_options
```
Набор `notes_position` к `NONE` для исключения заметок из вывода слайдов.

**Шаг 2**: Установить обычный шрифт по умолчанию и сохранить изображение

```python
rendering_opts.default_regular_font = "Arial Black"
pres.slides[0].get_image(rendering_opts, 4 / 3, 4 / 3).save(
    "YOUR_OUTPUT_DIRECTORY/rendering_options-ArialBlackDefault.png", slides.ImageFormat.PNG)
```
Измените шрифт по умолчанию, используемый при рендеринге, и сохраните слайд как изображение.

### Изменение обычного шрифта по умолчанию на Arial Narrow

#### Обзор
Настройка шрифтов — ключ к согласованности бренда. В этом разделе показано изменение обычного шрифта по умолчанию.

**Шаг 1**: Установить новый обычный шрифт по умолчанию

```python
rendering_opts.default_regular_font = "Arial Narrow"
pres.slides[0].get_image(rendering_opts, 4 / 3, 4 / 3).save(
    "YOUR_OUTPUT_DIRECTORY/rendering_options-ArialNarrowDefault.png", slides.ImageFormat.PNG)
```
Обновите параметры рендеринга, чтобы использовать «Arial Narrow» в качестве шрифта по умолчанию, и сохраните слайд.

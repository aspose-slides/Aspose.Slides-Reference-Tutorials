## Руководство по внедрению

В этом разделе мы рассмотрим, как получить доступ к встроенным свойствам презентации с помощью Aspose.Slides.

### Доступ к встроенным свойствам презентации
#### Обзор
Доступ к встроенным свойствам и их отображение позволяют извлекать важные метаданные, связанные с файлом PowerPoint. Это может быть полезно для автоматизации отчетов или поддержания стандартов документации.

#### Этапы внедрения
##### Шаг 1: Загрузите презентацию
Начните с указания пути к файлу презентации:
```python
presentation_path = "YOUR_DOCUMENT_DIRECTORY/props_builtin.pptx"
```
##### Шаг 2: Открытие и доступ к свойствам документа
Используйте менеджер контекста для эффективного управления ресурсами:
```python
with slides.Presentation(presentation_path) as pres:
    document_properties = pres.document_properties
```
##### Шаг 3: Отображение каждого встроенного свойства
Извлеките и распечатайте каждое свойство с помощью простых операторов печати. Это поможет понять структуру вашей презентации:
```python
print("Category : " + document_properties.category)
print("Current Status : " + document_properties.content_status)
print("Creation Date : " + str(document_properties.created_time))
print("Author : " + document_properties.author)
print("Description : " + document_properties.comments)
print("KeyWords : " + document_properties.keywords)
print("Last Modified By : " + str(document_properties.last_saved_by))
print("Supervisor : " + document_properties.manager)
print("Modified Date : " + str(document_properties.last_saved_time))
print("Presentation Format : " + document_properties.presentation_format)
print("Last Print Date : " + str(document_properties.last_printed))
print("Is Shared between producers : " + str(document_properties.shared_doc))
print("Subject : " + document_properties.subject)
print("Title : " + document_properties.title)
```
#### Параметры и возвращаемые значения
- `presentation_path`: Строковый путь к файлу PowerPoint.
- `document_properties`: Объект, содержащий все встроенные свойства.

### Советы по устранению неполадок
Убедитесь, что путь к файлу презентации указан правильно, чтобы избежать `FileNotFoundError`. Убедитесь, что Aspose.Slides правильно установлен в вашей среде.

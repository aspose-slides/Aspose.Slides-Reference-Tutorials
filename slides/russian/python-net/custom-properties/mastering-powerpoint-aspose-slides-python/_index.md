## Руководство по внедрению
Теперь, когда наша настройка готова, давайте рассмотрим возможности добавления пользовательских свойств в презентации PowerPoint.
### Добавление пользовательских свойств документа
#### Обзор
Добавление пользовательских свойств документа позволяет вам встраивать метаданные в ваши файлы PowerPoint. Это может быть что угодно: от сведений об авторе до информации о проекте или номеров версий.
#### Шаги по реализации
##### Шаг 1: Создание экземпляра класса представления
Начнем с создания объекта презентации:
```python
with slides.Presentation() as presentation:
    # Доступ к свойствам документа
    document_properties = presentation.document_properties
```
##### Шаг 2: Добавьте пользовательские свойства
Вы можете добавлять пользовательские свойства, используя `set_custom_property_value` метод. Вот как добавить три различных пользовательских свойства:
```python
document_properties.set_custom_property_value("New Custom", 12)
document_properties.set_custom_property_value("My Name", "Mudassir")
document_properties.set_custom_property_value("Custom", 124)
```
- **Параметры**: Первый параметр — это имя свойства (строка), а второй — его значение, которое может иметь любой тип данных, поддерживаемый свойствами PowerPoint.
##### Шаг 3: Извлечение свойства
Чтобы получить имя пользовательского свойства по индексу:
```python
property_name = document_properties.get_custom_property_name(2)
```
- **Объяснение**: извлекает имя третьего свойства (индекс начинается с нуля).
##### Шаг 4: Удаление пользовательского свойства
Вы можете удалить свойства, используя их имена:
```python
document_properties.remove_custom_property(property_name)
```
Этот шаг гарантирует, что выбранное пользовательское свойство будет удалено из вашего документа.
##### Сохранение презентации
Не забудьте сохранить презентацию после внесения изменений:
```python
presentation.save("YOUR_OUTPUT_DIRECTORY/props_add_custom_document_properties_out.pptx", slides.export.SaveFormat.PPTX)
```

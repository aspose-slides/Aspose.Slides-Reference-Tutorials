## Руководство по внедрению
### Функция 1: Загрузка презентации
#### Обзор
Загрузка презентации имеет решающее значение перед экспортом слайдов. В этом разделе демонстрируется открытие и проверка файла презентации.
**Шаг 1: Настройте каталог документов**
```python
import os
import aspose.slides as slides

document_directory = "YOUR_DOCUMENT_DIRECTORY/"
```
**Шаг 2: Загрузите презентацию**
Убедитесь, что у вас есть `.pptx` готовый файл в вашем каталоге:
```python
with slides.Presentation(os.path.join(document_directory, 'welcome-to-powerpoint.pptx')) as pres:
    # Откройте первый слайд, чтобы убедиться, что он загружен правильно.
    all_slides = pres.slides[0]
```
### Функция 2: Экспорт слайда в SVG
#### Обзор
Эта функция показывает, как экспортировать слайд PowerPoint в файл SVG, подходящий для масштабируемой графики в веб-приложениях.
**Шаг 1: Определите функцию для сохранения в формате SVG**
Создайте функцию, которая обрабатывает экспорт:
```python
def save_slide_as_svg(slide, output_directory):
    with open(os.path.join(output_directory, 'slide_out.svg'), "wb") as stream:
        slide.write_as_svg(stream)
```
**Шаг 2: Использование функции для экспорта**
Используйте эту функцию в вашем менеджере контекста:
```python
document_directory = "YOUR_DOCUMENT_DIRECTORY/"
output_directory = "YOUR_OUTPUT_DIRECTORY/"

with slides.Presentation(os.path.join(document_directory, 'welcome-to-powerpoint.pptx')) as pres:
    # Доступ к первому слайду
    all_slides = pres.slides[0]
    
    # Сохраните полученный слайд в файл SVG в указанном выходном каталоге.
    save_slide_as_svg(all_slides, output_directory)
```
**Пояснение параметров:**
- `slide`: Конкретный объект слайда, который вы хотите экспортировать.
- `output_directory`: Каталог, в котором будет сохранен файл SVG.

## Руководство по внедрению

### Создание и добавление математических фигур

Первый шаг — создание презентации и добавление математической фигуры.

#### Шаг 1: Инициализация презентации

Начните с инициализации вашей презентации:

```python
import aspose.slides as slides
import aspose.slides.mathtext as mathtext

def create_and_manipulate_math_shape():
    with slides.Presentation() as presentation:
        slide = presentation.slides[0]
```

#### Шаг 2: Добавление математической фигуры

Добавьте на слайд математическую фигуру:

```python
        # Добавьте MathShape в позицию (10, 10) с шириной и высотой 500.
        math_shape = slide.shapes.add_math_shape(10, 10, 500, 500)
```

#### Шаг 3: Создание и добавление математического текста

Теперь создайте математические текстовые блоки:

```python
        # Доступ к математическому абзацу первой части первого абзаца
        math_paragraph = math_shape.text_frame.paragraphs[0].portions[0].math_paragraph

        # Создайте MathBlock с выражением «F + (1/y) подчеркивание»
        math_block = mathtext.MathBlock(
            mathtext.MathematicalText("F").join(".add")
            .join(mathtext.MathematicalText("1").divide("y")).underbar())

        # Добавьте MathBlock в MathParagraph
        math_paragraph.add(math_block)
```

#### Шаг 4: Печать математических элементов

Чтобы увидеть ваши элементы, используйте рекурсивную функцию:

```python
def foreach_math_element(root):
    for child in root.get_children():
        element_info = f"{type(child)}"
        if isinstance(child, slides.mathtext.MathematicalText):
            element_info += ": " + str(child.value)
        print(element_info)
        foreach_math_element(child)

# Распечатать все элементы в математическом блоке
foreach_math_element(math_block)
```

#### Шаг 5: Сохранение презентации

Наконец, сохраните вашу презентацию:

```python
        # Сохранить в указанном выходном каталоге
        presentation.save("YOUR_OUTPUT_DIRECTORY/shapes_mathtext_get_children_out.pptx", slides.export.SaveFormat.PPTX)

create_and_manipulate_math_shape()
```

### Добавление современных комментариев к слайдам PowerPoint

#### Обзор

Эта функция позволяет добавлять современные комментарии непосредственно на слайды презентации. Эти комментарии связаны с авторами, что позволяет осуществлять совместный ввод и обратную связь.

#### Пошаговая реализация

**1. Инициализация презентации**

Начните с создания экземпляра `Presentation` сорт:

```python
with slides.Presentation() as pres:
    # Код будет добавлен здесь
```

**2. Добавить автора для комментариев**

Добавьте автора, который будет отвечать за комментарии:

```python
new_author = pres.comment_authors.add_author("Some Author", "SA")
```
- **Параметры:** Имя автора и уникальный идентификатор.

**3. Добавить современный комментарий**

Затем добавьте современный комментарий к целевому слайду:

```python
modern_comment = new_author.comments.add_modern_comment(
    "This is a modern comment",
    pres.slides[0],  # Нацеливание на первый слайд
    None,            # Нет определенной формы для комментария
    drawing.PointF(100, 100),  # Положение комментария на слайде
    date.today()     # Текущая дата как временная метка
)
```
- **Параметры:**
  - `text`: Содержание комментария.
  - `slide_index`Индекс целевого слайда.
  - `shape`: Ссылка на форму (необязательно, нет, если не используется).
  - `point`: Положение на слайде, где будет размещен комментарий.
  - `date_time`: Временная метка добавления комментария.

**4. Сохранить презентацию**

Наконец, сохраните презентацию, чтобы гарантировать сохранение всех изменений:

```python
pres.save("YOUR_OUTPUT_DIRECTORY/comments_add_modern_comment_out.pptx", slides.export.SaveFormat.PPTX)
```
- **Параметры:** 
  - Путь к файлу с именем.
  - Формат экспорта (в данном случае PPTX).

#### Советы по устранению неполадок

- Убедитесь, что у вас есть права на запись в каталог, в котором вы сохраняете файл.
- Убедитесь, что индекс слайдов правильный и присутствует в вашей презентации.

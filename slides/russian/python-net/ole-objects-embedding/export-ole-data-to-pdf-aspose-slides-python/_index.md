## Руководство по внедрению

Теперь, когда все настроено, давайте перейдем к реализации экспорта данных OLE в PDF.

### Экспорт данных OLE в PDF

Эта функция позволяет сохранять встроенные объекты в файлах PowerPoint при конвертации в PDF-файлы, гарантируя отсутствие потери информации или функциональности.

#### Шаг 1: Загрузите презентацию

Загрузите презентацию, содержащую объекты OLE, с помощью Aspose.Slides.

```python
document_directory = 'YOUR_DOCUMENT_DIRECTORY/'
output_directory = 'YOUR_OUTPUT_DIRECTORY/'

with slides.Presentation(document_directory + "PresOleExample.pptx") as pres:
    # Перейти к созданию параметров экспорта PDF
```

#### Шаг 2: Создайте параметры экспорта PDF

Здесь мы определяем настройки экспорта вашей презентации.

```python
options = slides.export.PdfOptions()
options.include_ole_data = True  # Это гарантирует сохранение данных OLE в PDF.
```

#### Шаг 3: Сохранить как PDF

Сохраните презентацию с указанными параметрами, чтобы получить PDF-файл, в котором сохранены все внедренные объекты.

```python
pres.save(output_directory + "PresOleExample.pdf", slides.export.SaveFormat.PDF, options)
```

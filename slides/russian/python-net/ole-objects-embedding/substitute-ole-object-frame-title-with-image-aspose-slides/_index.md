## Руководство по внедрению

### Заменить заголовок изображения рамки объекта OLE

В этом разделе вы узнаете, как заменить заголовок по умолчанию рамки объекта OLE на изображение. Это может быть особенно полезно для визуального представления данных или документов на слайдах.

#### Шаг 1: Загрузите презентацию и откройте ее первый слайд

Начните с загрузки презентации и перехода к слайду, на который вы хотите добавить рамку объекта OLE.

```python
import aspose.slides as slides

def replace_picture_title_of_ole_object_frame():
    with slides.Presentation() as pres:
        # Доступ к первому слайду
        slide = pres.slides[0]
```

#### Шаг 2: Добавьте рамку объекта OLE с помощью файла Excel

Добавьте рамку объекта OLE на слайд. Здесь мы используем файл Excel в качестве встроенного документа.

```python
        excel_file_path = 'YOUR_DOCUMENT_DIRECTORY/book.xlsx'
        with open(excel_file_path, "rb") as file:
            all_bytes = file.read()
            data_info = slides.dom.ole.OleEmbeddedDataInfo(all_bytes, "xlsx")
        
        oof = slide.shapes.add_ole_object_frame(20, 20, 50, 50, data_info)
        oof.is_object_icon = True
```

#### Шаг 3: Добавьте изображение и замените его на OLE-значок.

Загрузите изображение из своего каталога и установите его в качестве замещающего значка для рамки объекта OLE.

```python
        img_path = 'YOUR_DOCUMENT_DIRECTORY/image1.jpg'
        with slides.Images.from_file(img_path) as images_collection:
            imgx = pres.images.add_image(images_collection[0])
            oof.substitute_picture_format.picture.image = imgx
```

#### Шаг 4: Задайте заголовок для заменяющего заголовка изображения

Наконец, задайте заголовок для рамки вашего объекта OLE, чтобы предоставить контекст или информацию.

```python
        oof.substitute_picture_title = "Caption example"
```

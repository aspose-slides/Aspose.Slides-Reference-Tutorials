## Руководство по внедрению
### Добавление рамки объекта OLE
В этом разделе мы покажем, как встроить файл Excel в слайд PowerPoint в качестве рамки объекта OLE.

#### Шаг 1: Загрузите файл Excel
Сначала создайте функцию для чтения вашего файла Excel и преобразования его в массив байтов. Это необходимо для встраивания:

```python
def load_excel_file(file_path):
    # Откройте файл Excel в режиме двоичного чтения.
    with open(file_path, "rb") as fs:
        return fs.read()
```

#### Шаг 2: Добавьте рамку объекта OLE к слайду
Далее давайте создадим функцию, которая добавит фрейм объекта OLE, содержащий данные Excel, к первому слайду:

```python
def add_ole_object_frame():
    # Создать экземпляр класса Presentation, представляющего файл PPTX
    with slides.Presentation() as pres:
        # Доступ к первому слайду
        slide = pres.slides[0]
        
        # Загрузить данные файла Excel в массив байтов
        excel_data = load_excel_file(DATA_DIR + "book.xlsx")
        
        # Создать объект данных для внедрения содержимого Excel
        data_info = slides.dom.ole.OleEmbeddedDataInfo(excel_data, "xlsx")
        
        # Добавьте форму рамки объекта OLE, чтобы покрыть весь слайд.
        ole_object_frame = slide.shapes.add_ole_object_frame(
            0, 0,                    # Положение (x, y)
            pres.slide_size.size.width, pres.slide_size.size.height, # Размер (ширина, высота)
            data_info                # Объект данных, содержащий содержимое Excel
        )
        
        # Сохраните презентацию на диске со встроенным OLE-объектом
        pres.save(OUTPUT_DIR + "shapes_add_ole_object_frame_out.pptx", slides.export.SaveFormat.PPTX)
```

### Параметры и методы
- **`add_ole_object_frame()`**: Эта функция создает рамку объекта OLE на слайде PowerPoint.
  - `0, 0`: Верхнее левое положение кадра на слайде.
  - `pres.slide_size.size.width`, `pres.slide_size.size.height`: Гарантирует, что рамка покрывает весь слайд.
  - `data_info`: Содержит данные Excel для внедрения.

### Советы по устранению неполадок
- **Проблемы с путями к файлам**: Убедитесь, что путь к файлу Excel указан правильно и он доступен из каталога выполнения скрипта.

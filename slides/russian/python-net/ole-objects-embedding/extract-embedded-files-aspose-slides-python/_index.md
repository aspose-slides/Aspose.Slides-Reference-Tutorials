## Руководство по внедрению

В этом разделе подробно описывается, как извлекать встроенные файловые данные из объектов OLE в презентациях PowerPoint.

### Загрузка и итерация слайдов

Загрузите презентацию и пройдитесь по формам каждого слайда:

```python
with slides.Presentation(document_path) as pres:
    for slide in pres.slides:
        # Обработайте каждую фигуру на слайде
```

### Определение фреймов объектов OLE

Определите, является ли фигура `OleObjectFrame`, что указывает на то, что он содержит встроенные данные:

```python
count = 0
for slide in pres.slides:
    for shape in slide.shapes:
        if isinstance(shape, slides.OleObjectFrame):
            # Эта форма содержит объект OLE со встроенными данными.
```

### Извлечение данных встроенного файла

После идентификации объектов OLE извлеките их данные и сохраните их, используя уникальное имя файла:

```python
count = 0
for slide in pres.slides:
    for shape in slide.shapes:
        if isinstance(shape, slides.OleObjectFrame):
            count += 1
            
            # Извлечь данные файла и расширение
            data = shape.embedded_data.embedded_file_data
            extension = shape.embedded_data.embedded_file_extension
            
            # Создайте имя файла на основе номера объекта.
            file_name = f"shapes_ole_objects{count}_out.{extension}"
            
            # Запись в выходной каталог
            with open(f"YOUR_OUTPUT_DIRECTORY/{file_name}", "wb") as file:
                file.write(data)
```

### Параметры и возвращаемые значения

- **прес.слайды**: Повторяет все слайды презентации.
- **форма.встроенные_данные.встроенные_данные_файла**: Содержит необработанные данные встроенного файла.
- **форма.встроенные_данные.встроенное_расширение_файла**: Используется для наименования.

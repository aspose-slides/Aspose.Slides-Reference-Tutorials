## Руководство по внедрению

В этом разделе мы рассмотрим включение режима ZIP64 для сохранения больших файлов PowerPoint.

### Включение сжатия ZIP64

Эта функция гарантирует, что презентации можно сохранять без ограничений по размеру, всегда используя сжатие ZIP64, когда это необходимо. Вот как это можно реализовать:

#### Шаг 1: Настройте параметры экспорта

Сначала настройте параметры экспорта, чтобы включить режим ZIP64.

```python
# Настройте PptxOptions для экспорта
class PresentationExporter:
    def __init__(self):
        self.pptx_options = slides.export.PptxOptions()
        self.pptx_options.zip_64_mode = slides.export.Zip64Mode.ALWAYS
```

- **Объяснение:** The `PptxOptions` класс позволяет устанавливать различные параметры для сохранения презентаций. Устанавливая `zip_64_mode` к `ALWAYS`мы гарантируем, что библиотека использует сжатие ZIP64, необходимое для обработки больших файлов.

#### Шаг 2: Создайте и сохраните презентацию

Далее создайте новую презентацию и сохраните ее с настроенными параметрами.

```python
class LargePresentationHandler:
    def __init__(self):
        exporter = PresentationExporter()
        with slides.Presentation() as presentation:
            # Определите содержание вашей презентации здесь (необязательно)

            # Сохраните презентацию в указанном выходном каталоге с включенным режимом ZIP64.
            presentation.save("YOUR_OUTPUT_DIRECTORY/PresentationZip64.pptx", 
                             slides.export.SaveFormat.PPTX, exporter.pptx_options)
```

- **Объяснение:** The `save` Метод записывает презентацию на диск. Предоставление нашего пользовательского `pptx_options`, мы гарантируем, что файл будет сохранен с включенным сжатием ZIP64.

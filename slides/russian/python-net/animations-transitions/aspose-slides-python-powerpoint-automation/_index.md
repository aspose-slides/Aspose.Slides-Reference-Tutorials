## Руководство по внедрению

### Функция 1: загрузка и итерация по презентации

#### Обзор:
Эта функция позволяет загружать файл презентации PowerPoint и перебирать его слайды, что полезно для автоматизации обработки слайдов или извлечения определенных данных.

#### Пошаговая реализация:
**Шаг 1: Определите функцию**
Определить функцию `load_presentation` который принимает в качестве аргумента путь к файлу презентации.

```python
def load_presentation(presentation_path):
    with slides.Presentation(presentation_path) as pres:
        for slide in pres.slides:
            print(f"Slide #{slide.slide_number} загружен.")
```
**Объяснение:**
- `slides.Presentation(presentation_path)` открывает ваш файл PowerPoint.
- Менеджер контекста обеспечивает правильное закрытие презентации после обработки.

**Шаг 2: Пример использования**
Заменять `'YOUR_DOCUMENT_DIRECTORY/'` с фактическим путем к каталогу, где хранится ваш документ:

```python
load_presentation('YOUR_DOCUMENT_DIRECTORY/shapes_animation_example.pptx')
```

### Функция 2: Извлечение эффектов анимации из слайдов

#### Обзор:
Извлекайте и печатайте сведения об эффектах анимации, примененных к фигурам на каждом слайде. Это помогает анализировать настройки анимации в ваших презентациях.

#### Пошаговая реализация:
**Шаг 1: Определите функцию**
Создать функцию `extract_animation_effects` который загружает презентацию и выполняет итерацию ее анимаций.

```python
def extract_animation_effects(presentation_path):
    with slides.Presentation(presentation_path) as pres:
        for slide in pres.slides:
            for effect in slide.timeline.main_sequence:
                print(f"{effect.type} animation effect is set to shape#{effect.target_shape.unique_id} на слайде № {slide.slide_number}")
```
**Объяснение:**
- `slide.timeline.main_sequence` обеспечивает доступ ко всем анимациям, примененным на слайде.
- Каждый `effect` объект содержит сведения о типе анимации и ее целевой форме.

**Шаг 2: Пример использования**
Используйте функцию с путем презентации:

```python
extract_animation_effects('YOUR_DOCUMENT_DIRECTORY/shapes_animation_example.pptx')
```

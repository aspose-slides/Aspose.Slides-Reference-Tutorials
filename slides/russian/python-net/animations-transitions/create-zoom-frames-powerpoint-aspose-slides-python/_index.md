### Добавление рамок масштабирования с предварительным просмотром слайдов

#### Обзор:
Рамки масштабирования позволяют вам сосредоточиться на определенных слайдах в пределах основного слайда презентации. Этот раздел поможет вам добавить рамку масштабирования, которая предварительно отображает другой слайд в вашей презентации.

#### Пошаговая реализация:

**1. Инициализируйте презентацию:**
Начните с создания или загрузки существующей презентации, в которую вы добавите кадры масштабирования.

```python
import aspose.slides as slides

def create_zoom_frames():
    with slides.Presentation() as pres:
        # Добавьте пустые слайды для демонстрации
```

**2. Подготовьте слайды для Zoom-кадров:**
Добавьте и настройте слайды, которые будут использоваться в предпросмотре масштабируемых кадров.

```python
        slide2 = pres.slides.add_empty_slide(pres.slides[0].layout_slide)
        slide3 = pres.slides.add_empty_slide(pres.slides[0].layout_slide)

        # Настроить слайд 2
        slide2.background.type = slides.BackgroundType.OWN_BACKGROUND
        slide2.background.fill_format.fill_type = slides.FillType.SOLID
        slide2.background.fill_format.solid_fill_color.color = drawing.Color.cyan
        auto_shape = slide2.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 100, 200, 500, 200)
        auto_shape.text_frame.text = "Second Slide"
```

**3. Добавьте рамку масштабирования с предварительным просмотром слайдов:**
Используйте `add_zoom_frame` метод создания рамки на основном слайде, которая представляет собой предварительный просмотр другого слайда.

```python
        zoom_frame1 = pres.slides[0].shapes.add_zoom_frame(20, 20, 250, 200, slide2)
        zoom_frame1.show_background = False
```

#### Основные параметры конфигурации:
- **Положение и размер**: Параметры `(x, y, width, height)` определите, где на слайде будет располагаться рамка, а также ее размеры.
- **`show_background`**: Установить на `False` если вы предпочитаете не показывать фон увеличенного слайда.

### Настройка рамок Zoom с изображениями

#### Обзор:
Улучшите свою презентацию, добавив пользовательские изображения в рамки масштабирования для более динамичного вида.

#### Пошаговая реализация:

**1. Загрузите и добавьте изображение:**
Сначала загрузите файл изображения, который вы хотите включить в рамку масштабирования.

```python
        image = pres.images.add_image(drawing.Image.from_file("YOUR_DOCUMENT_DIRECTORY/image1.jpg"))
```

**2. Создайте рамку масштабирования с пользовательским изображением:**
Добавьте новый кадр масштабирования, используя как предварительный просмотр слайда, так и наложение изображения.

```python
        zoom_frame2 = pres.slides[0].shapes.add_zoom_frame(200, 250, 250, 100, slide3, image)
        
        # Настроить внешний вид
        zoom_frame2.line_format.width = 5
        zoom_frame2.line_format.fill_format.fill_type = slides.FillType.SOLID
        zoom_frame2.line_format.fill_format.solid_fill_color.color = drawing.Color.hot_pink
        zoom_frame2.line_format.dash_style = slides.LineDashStyle.DASH_DOT
```

#### Советы по устранению неполадок:
- Убедитесь, что путь к образу указан правильно, чтобы избежать ошибок «файл не найден».
- Если у вас возникли проблемы с цветами или стилями, дважды проверьте `fill_type` и настройки цвета.

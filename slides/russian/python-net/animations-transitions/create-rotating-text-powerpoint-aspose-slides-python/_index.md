## Руководство по внедрению
В этом разделе мы рассмотрим каждую функцию поворота текста на слайде PowerPoint.

### Добавление фигур на слайды
Сначала добавим прямоугольную форму, которая будет содержать наш повернутый текст. Эта форма действует как контейнер для текста и может быть широко настроена.

#### Пошаговое руководство:
1. **Создать экземпляр презентации:**

   ```python
   with slides.Presentation() as presentation:
       slide = presentation.slides[0]
   ```
2. **Добавьте прямоугольную форму:**

   Здесь мы добавляем прямоугольник к первому слайду. Параметры задают его положение и размер.

   ```python
   auto_shape = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 150, 75, 350, 350)
   ```
### Вращение текста в форме
Теперь, когда наша фигура готова, давайте сосредоточимся на вращении текста внутри нее по вертикали.
1. **Создание и настройка TextFrame:**

   ```python
   text_frame = auto_shape.add_text_frame(" ")
   auto_shape.fill_format.fill_type = slides.FillType.NO_FILL
   ```
2. **Установить вертикальную ориентацию:**

   На этом этапе необходимо установить вертикальную ориентацию текстовой рамки на 270 градусов, что повернет ее вертикально.

   ```python
   text_frame.text_frame_format.text_vertical_type = slides.TextVerticalType.VERTICAL270
   ```
3. **Добавить текстовое содержимое:**

   Добавьте текст в абзац и настройте его внешний вид.

   ```python
   para = text_frame.paragraphs[0]
   portion = para.portions[0]
   portion.text = "A quick brown fox jumps over the lazy dog. A quick brown fox jumps over the lazy dog."
   
   # Установите тип заливки для текста на сплошной и закрасьте его в черный цвет.
   portion.portion_format.fill_format.fill_type = slides.FillType.SOLID
   portion.portion_format.fill_format.solid_fill_color.color = drawing.Color.black
   ```
4. **Сохраните презентацию:**

   Наконец, сохраните презентацию с внесенными вами изменениями.

   ```python
   presentation.save("YOUR_OUTPUT_DIRECTORY/text_rotate_out.pptx", slides.export.SaveFormat.PPTX)
   ```

## Раздел часто задаваемых вопросов
**В: Как повернуть текст по горизонтали?**
А: Установить `text_vertical_type` к `TEXT_VERTICAL_TYPE.HORIZONTAL`.

**В: Могу ли я изменить размер и стиль шрифта?**
A: Да, изменить `portion.portion_format` для свойств шрифта.

**В: Что делать, если моя презентация сохраняется неправильно?**
A: Убедитесь, что у вас есть права на запись в выходной каталог.

**В: Как добавить несколько абзацев повернутого текста?**
A: Создайте дополнительные абзацы, используя `text_frame.paragraphs.add_empty_paragraph()`.

**В: Существуют ли ограничения на размер текстового поля?**

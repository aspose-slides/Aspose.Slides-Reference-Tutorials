## Руководство по внедрению

В этом разделе подробно описано, как применять эффекты 3D-вращения.

### Применение 3D-вращения к прямоугольной форме

#### Обзор

Добавьте глубины и перспективы прямоугольным формам с помощью 3D-вращений.

#### Пошаговая реализация

**1. Добавьте прямоугольную форму:**

```python
auto_shape = pres.slides[0].shapes.add_auto_shape(
    slides.ShapeType.RECTANGLE, 30, 30, 200, 200)
```

*Объяснение*: Этот код добавляет прямоугольник в позицию (30, 30) с размерами 200x200.

**2. Применить 3D-вращение:**

```python
auto_shape.three_d_format.depth = 6
auto_shape.three_d_format.camera.set_rotation(40, 35, 20)
auto_shape.three_d_format.camera.camera_type = slides.CameraPresetType.ISOMETRIC_LEFT_UP
auto_shape.three_d_format.light_rig.light_type = slides.LightRigPresetType.BALANCED
```

*Объяснение*: 
- `depth`: Устанавливает глубину 3D-эффекта.
- `camera.set_rotation()`: Настраивает углы поворота для осей X, Y и Z.
- `camera_type`: Определяет перспективу камеры.
- `light_rig.light_type`: Регулирует освещение для улучшения трехмерного вида.

**3. Сохраните презентацию:**

```python
pres.save("shapes_apply_3d_rotation_to_rectangle_out.pptx", slides.export.SaveFormat.PPTX)
```

### Применение 3D-вращения к линейной форме

#### Обзор

Создавайте интересные визуальные элементы, добавляя 3D-эффекты к формам линий.

#### Пошаговая реализация

**1. Добавьте форму линии:**

```python
auto_shape = pres.slides[0].shapes.add_auto_shape(
    slides.ShapeType.LINE, 30, 300, 200, 200)
```

*Объяснение*: Этот код добавляет строку в позицию (30, 300) с размерами 200x200.

**2. Применить 3D-вращение:**

```python
auto_shape.three_d_format.depth = 6
auto_shape.three_d_format.camera.set_rotation(0, 35, 20)
auto_shape.three_d_format.camera.camera_type = slides.CameraPresetType.ISOMETRIC_LEFT_UP
auto_shape.three_d_format.light_rig.light_type = slides.LightRigPresetType.BALANCED
```

*Объяснение*: Подобна форме прямоугольника, но с другими углами поворота для создания уникальных эффектов.

**3. Сохраните презентацию:**

```python
pres.save("shapes_apply_3d_rotation_to_line_out.pptx", slides.export.SaveFormat.PPTX)
```

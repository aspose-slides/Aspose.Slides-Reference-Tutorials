# Как управлять встроенными шрифтами в PowerPoint с помощью Aspose.Slides для Python

Это руководство проведет вас через управление встроенными шрифтами с помощью мощной библиотеки Aspose.Slides в Python, помогая вам оптимизировать обработку шрифтов и ваши презентации.

- Лучшие практики сохранения измененной презентации в оптимизированном формате.

## Руководство по внедрению

В этом разделе процесс управления встроенными шрифтами разбит на удобные для выполнения этапы.

### Шаг 1: Откройте файл презентации.

Сначала загрузите файл PowerPoint с помощью Aspose.Slides. Этот шаг настраивает объект презентации для дальнейших операций.

```python
with slides.Presentation("YOUR_DOCUMENT_DIRECTORY/text_embedded_fonts.pptx") as presentation:
    # Презентация теперь открыта и готова к манипуляциям.
```

### Шаг 2: Визуализация и сохранение изображения слайда

Перед внесением любых изменений полезно сохранить текущее состояние слайда. Этот шаг фиксирует исходный вид.

```python
slide_image = presentation.slides[0].get_image(drawing.Size(960, 720))
slide_image.save("YOUR_OUTPUT_DIRECTORY/text_embedded_fonts_1_out.png", slides.ImageFormat.PNG)
```

### Шаг 3: Откройте диспетчер шрифтов.

Доступ к менеджеру шрифтов для выполнения операций со встроенными шрифтами. Этот объект позволяет вам извлекать и управлять настройками шрифтов в вашей презентации.

```python
fonts_manager = presentation.fonts_manager
```

### Шаг 4: Извлеките все встроенные шрифты

Получить список всех встроенных шрифтов в презентации. Затем вы можете перебрать этот список, чтобы найти определенные шрифты, например «Calibri».

```python
embedded_fonts = fonts_manager.get_embedded_fonts()
```

### Шаг 5: Удалите определенный шрифт (например, Calibri)

Проверьте и удалите из презентации нежелательные встроенные шрифты, такие как «Calibri».

```python
calibri_font = next((font for font in embedded_fonts if font.font_name == "Calibri"), None)
if calibri_font:
    fonts_manager.remove_embedded_font(calibri_font)
```

### Шаг 6: Сохраните измененное изображение слайда.

После внесения изменений сохраните еще одну версию слайда, чтобы наглядно увидеть эффект удаления шрифта.

```python
slide_image.save("YOUR_OUTPUT_DIRECTORY/text_embedded_fonts_2_out.png", slides.ImageFormat.PNG)
```

### Шаг 7: Сохраните измененную презентацию.

Наконец, сохраните презентацию с обновленными шрифтами. Этот шаг гарантирует, что все изменения будут сохранены в вашем файле.

```python
presentation.save(
    "YOUR_OUTPUT_DIRECTORY/text_embedded_fonts_out.ppt", slides.export.SaveFormat.PPT)
```

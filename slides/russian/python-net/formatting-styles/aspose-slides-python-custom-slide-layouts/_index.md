## Руководство по внедрению

Теперь давайте перейдем к созданию пользовательских макетов слайдов.

### Создание пустого макета слайда

#### Обзор
Пустой макет слайда служит базовой структурой для новых презентаций или дополнительных слайдов.

#### Шаги по созданию и настройке пустого макета

##### Получить пустой макет

```python
def get_blank_layout(pres):
    return pres.layout_slides.get_by_type(slides.SlideLayoutType.BLANK)
```

На этом этапе создается пустой шаблон для настройки.

##### Доступ к менеджеру заполнителей

```python
def access_placeholder_manager(layout):
    return layout.placeholder_manager
```

Менеджер заполнителей позволяет добавлять различные типы заполнителей, такие как текст или диаграммы.

### Добавление заполнителей

#### Обзор
Добавление различных заполнителей повышает функциональность и визуальную привлекательность.

##### Добавить заполнитель контента

```python
def add_content_placeholder(placeholder_manager):
    placeholder_manager.add_content_placeholder(10, 10, 300, 200)
```

Этот метод добавляет заполнитель контента в позицию `(x=10, y=10)` с размерами `width=300` и `height=200`.

##### Добавить вертикальный текстовый заполнитель

```python
def add_vertical_text_placeholder(placeholder_manager):
    placeholder_manager.add_vertical_text_placeholder(350, 10, 200, 300)
```

Используйте этот вариант для вертикального текста, идеально подходит для заметок или этикеток.

##### Добавить заполнитель диаграммы

```python
def add_chart_placeholder(placeholder_manager):
    placeholder_manager.add_chart_placeholder(10, 350, 300, 300)
```

Визуализация данных с помощью заполнителей диаграмм.

##### Добавить заполнитель таблицы

```python
def add_table_placeholder(placeholder_manager):
    placeholder_manager.add_table_placeholder(350, 350, 300, 200)
```

Идеально подходит для представления структурированной информации, такой как расписания или статистика.

### Завершение слайда

#### Добавление нового слайда с использованием пользовательского макета

```python
def add_custom_slide(pres, layout):
    pres.slides.add_empty_slide(layout)
```

Это обеспечивает единообразие слайдов вашей презентации.

#### Сохранение презентации

```python
def save_presentation(pres, output_path):
    pres.save(output_path, slides.export.SaveFormat.PPTX)
```

Сохраните свою работу для дальнейшей доработки или публикации.

## Руководство по внедрению

Теперь давайте реализуем настройку шрифтов по умолчанию для обычного и азиатского текста.

### Установка шрифтов по умолчанию

Эта функция позволяет вам определить, какие шрифты будут использоваться, если шрифт не указан в самом содержимом презентации.

#### Шаг 1: Создание LoadOptions

Начните с определения `LoadOptions` чтобы указать параметры загрузки:

```python
load_options = slides.LoadOptions()
load_options.load_format = slides.LoadFormat.AUTO
```

Это сообщает Aspose.Slides, как автоматически интерпретировать формат файла.

#### Шаг 2: Укажите шрифты по умолчанию

Далее, установите как обычные, так и азиатские шрифты. В этом примере мы используем "Wingdings" для простоты:

```python
load_options.default_regular_font = "Wingdings"
load_options.default_asian_font = "Wingdings"
```

Это обеспечивает единообразие всего текста в вашей презентации.

#### Шаг 3: Загрузите презентацию

Установив нужные параметры, загрузите файл PowerPoint, используя следующие параметры:

```python
with slides.Presentation("YOUR_DOCUMENT_DIRECTORY/text_default_fonts.pptx", load_options) as pptx:
    # Создайте миниатюру слайда и сохраните ее в формате PNG
    pptx.slides[0].get_image(1, 1).save("YOUR_OUTPUT_DIRECTORY/text_default_fonts_out.png", slides.ImageFormat.PNG)
    
    # Сохранить презентацию в формате PDF
    pptx.save("YOUR_OUTPUT_DIRECTORY/text_default_fonts_out.pdf", slides.export.SaveFormat.PDF)
    
    # Дополнительно сохраните его как файл XPS
    pptx.save("YOUR_OUTPUT_DIRECTORY/text_default_fonts_out.xps", slides.export.SaveFormat.XPS)
```

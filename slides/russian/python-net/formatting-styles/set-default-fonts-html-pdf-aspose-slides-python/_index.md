## Руководство по внедрению

В этом разделе вы узнаете, как настроить шрифты по умолчанию для экспорта в HTML и PDF.

### Функция 1: Установка обычного шрифта по умолчанию (экспорт HTML)

#### Обзор

Настроив определенный обычный шрифт, вы обеспечиваете единообразие типографики при экспорте презентации в виде HTML-файла.

#### Пошаговая реализация

##### Загрузить презентацию

Загрузите файл презентации, используя:

```python
def load_presentation(path):
    # Замените «YOUR_DOCUMENT_DIRECTORY/» на фактический путь к документу.
    return slides.Presentation(path)
```

##### Настройте параметры экспорта HTML

Настраивать `HtmlOptions` и определите желаемый шрифт:

```python
def configure_html_options():
    html_options = slides.export.HtmlOptions()
    html_options.default_regular_font = "Arial Black"  # Установите здесь предпочитаемый вами шрифт
    return html_options
```

##### Сохранить презентацию как HTML

Используйте настроенные параметры для сохранения презентации:

```python
def save_html(presentation, output_path, html_options):
    presentation.save(output_path, slides.export.SaveFormat.HTML, html_options)
```

### Функция 2: Установка обычного шрифта по умолчанию (экспорт PDF)

#### Обзор

Установите шрифт по умолчанию для экспорта PDF-файлов, чтобы сохранить единообразие текста в распечатанных или общих документах.

#### Пошаговая реализация

##### Настройте параметры экспорта PDF

Подготовьте `PdfOptions` пример:

```python
def configure_pdf_options():
    pdf_options = slides.export.PdfOptions()
    pdf_options.default_regular_font = "Arial Black"  # Установите здесь предпочитаемый вами шрифт
    return pdf_options
```

##### Сохранить презентацию как PDF

Экспортируйте файл в формат PDF, используя следующие параметры:

```python
def save_pdf(presentation, output_path, pdf_options):
    presentation.save(output_path, slides.export.SaveFormat.PDF, pdf_options)
```

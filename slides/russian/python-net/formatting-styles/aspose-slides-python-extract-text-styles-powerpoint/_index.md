### Извлечь информацию о стиле текста
Эта функция предназначена для извлечения и отображения эффективных стилей текста из определенной формы в презентации.

#### Шаг 1: Загрузите презентацию
Сначала загрузите файл PowerPoint с помощью Aspose.Slides. Замените `'YOUR_DOCUMENT_DIRECTORY/'` с фактическим путем к вашему документу.

```python
import aspose.slides as slides

# Определите путь к вашей презентации\presentation_path = 'YOUR_DOCUMENT_DIRECTORY/text_add_animation_effect.pptx'

# Откройте презентацию PowerPoint.
with slides.Presentation(presentation_path) as pres:
    # Доступ к первой фигуре с первого слайда
    shape = pres.slides[0].shapes[0]
```

#### Шаг 2: Получите информацию об эффективном стиле текста
Доступ и извлечение информации о стиле для текстового фрейма.

```python
# Получите эффективную информацию о стиле текста
effective_text_style = shape.text_frame.text_frame_format.text_style.get_effective()
```

#### Шаг 3: Перебор уровней стилей
Извлекайте и печатайте свойства стиля текста на каждом уровне, включая глубину, отступ, выравнивание и выравнивание шрифта.

```python
for i in range(9):
    effective_style_level = effective_text_style.get_level(i)
    
    # Распечатать сведения для каждого уровня стиля
    print(f'= Effective paragraph formatting for style level #{я} =')
    print('Depth:', effective_style_level.depth)
    print('Indent:', effective_style_level.indent)
    print('Alignment:', effective_style_level.alignment)
    print('Font alignment:', effective_style_level.font_alignment)
```

#### Советы по устранению неполадок
- Убедитесь, что путь к файлу PowerPoint указан правильно.
- Убедитесь, что ваша презентация содержит хотя бы одну фигуру с текстом на первом слайде.

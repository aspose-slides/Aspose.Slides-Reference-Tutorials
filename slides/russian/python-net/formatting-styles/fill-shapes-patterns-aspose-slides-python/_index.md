## Руководство по внедрению
В этом разделе мы рассмотрим, как заполнять фигуры узорами в ваших презентациях.

### Обзор
Заполнение фигур узором добавляет дополнительный уровень настройки и визуальной привлекательности. Вы можете использовать различные стили, такие как решетчатые или шахматные узоры, чтобы сделать ваши слайды более интересными.

#### Шаг 1: Создание экземпляра класса представления
Начнем с создания объекта презентации:

```python
with slides.Presentation() as pres:
    # Ваш код будет здесь
```
Этот менеджер контекста обеспечивает эффективное управление ресурсами.

#### Шаг 2: Доступ к фигурам и их изменение
Откройте первый слайд, затем добавьте прямоугольную фигуру, чтобы продемонстрировать заполнение узором:

```python
slide = pres.slides[0]
shape = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 50, 150, 75, 150)
```
Мы указываем положение (x, y) и размер (ширину, высоту) прямоугольника.

#### Шаг 3: Установите тип заливки на Узор
Измените тип заливки фигуры на узор:

```python
shape.fill_format.fill_type = slides.FillType.PATTERN
```
Это придаст нашей форме узорчатый вид.

#### Шаг 4: Настройте стиль и цвета узора
Определите стиль и цвета узора:

```python
shape.fill_format.pattern_format.pattern_style = slides.PatternStyle.TRELLIS
shape.fill_format.pattern_format.back_color.color = drawing.Color.light_gray
shape.fill_format.pattern_format.fore_color.color = drawing.Color.yellow
```
Здесь, `TRELLIS` выбран из-за его сетчатого вида. Экспериментируйте с другими стилями в соответствии с вашими дизайнерскими потребностями.

#### Шаг 5: Сохраните презентацию
Наконец, сохраните изменения в файле:

```python
pres.save("YOUR_OUTPUT_DIRECTORY/shapes_filltype_pattern_out.pptx", slides.export.SaveFormat.PPTX)
```
Обязательно укажите правильный выходной каталог для сохранения презентации.

## Руководство по внедрению
В этом разделе вы узнаете, как реализовать и понять код для извлечения и отображения эффективных двухцветных цветов на слайде презентации.

### Доступ к слайдам презентации
Сначала откройте или создайте презентацию, чтобы управлять ее содержимым:

```python
# Создать или открыть существующий экземпляр презентации
with slides.Presentation() as presentation:
    # Доступ к первому слайду
    slide = presentation.slides[0]
```

### Получение деталей эффекта Duotone
Доступ к формату фоновой заливки и получение сведений об эффекте дуплекса:

```python
# Получите формат заливки изображения для доступа к эффектам Duotone
duotone_effect = slide.background.fill_format.picture_fill_format.
                 picture.image_transform.get_duotone_effect()
```

### Отображение эффективных цветов
Извлеките и распечатайте эффективные цвета из эффекта дуплекса:

```python
# Получить эффективные цвета эффекта Duotone
duotone_effective = duotone_effect.get_effective()

# Отображение эффективных используемых цветов Duotone
print("Duotone effective color1: " + str(duotone_effective.color1))
print("Duotone effective color2: " + str(duotone_effective.color2))
```

### Основные параметры конфигурации
- **Формат заполнения изображения**: определяет, как изображения заполняются на слайде, что имеет решающее значение для доступа к настройкам дуплекса.
- **Трансформация изображения**: Класс, предоставляющий доступ к преобразованиям, связанным с изображениями, таким как дуотонирование.

## Руководство по внедрению
Теперь давайте сосредоточимся на применении эффекта внутренней тени к тексту на слайде PowerPoint.
### Добавление текста с эффектом внутренней тени
#### Обзор
Мы создадим прямоугольную форму, добавим к ней текст, а затем применим эффект внутренней тени. Этот метод улучшает эстетику ваших слайдов, добавляя глубину тексту.
#### Пошаговое руководство
**1. Доступ к слайду**
Сначала получите ссылку на первый слайд вашей презентации:

```python
slide = total_presentation.slides[0]
```
**2. Добавление автофигуры**
Добавьте прямоугольник для размещения текста:

```python
auto_shape = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 150, 75, 400, 300)
auto_shape.fill_format.fill_type = slides.FillType.NO_FILL
```
**3. Вставка текста**
Вставьте текстовую рамку и задайте содержимое прямоугольника:

```python
auto_shape.add_text_frame("Aspose TextBox")
port = auto_shape.text_frame.paragraphs[0].portions[0]
pf = port.portion_format
pf.font_height = 50  # Установите размер шрифта для улучшения видимости
```
**4. Применение эффекта внутренней тени**
Включите и настройте эффект внутренней тени для текста:

```python
ef = pf.effect_format
ef.enable_inner_shadow_effect()
# Настройте параметры внутренней тени
ef.inner_shadow_effect.blur_radius = 8.0  # Радиус размытия для более мягкой тени
ef.inner_shadow_effect.direction = 90.0  # Направление тени в градусах
ef.inner_shadow_effect.distance = 6.0    # Расстояние тени от текста
ef.inner_shadow_effect.shadow_color.b = 189  # Синяя составляющая цвета тени
# Установите согласованную тематику, используя цвета схемы
ef.inner_shadow_effect.shadow_color.color_type = slides.ColorType.SCHEME
ef.inner_shadow_effect.shadow_color.scheme_color = slides.SchemeColor.ACCENT1
```
**5. Сохранение презентации**
Наконец, сохраните вашу презентацию в файл:

```python
presentation.save("YOUR_OUTPUT_DIRECTORY/text_apply_inner_shadow_out.pptx")
```

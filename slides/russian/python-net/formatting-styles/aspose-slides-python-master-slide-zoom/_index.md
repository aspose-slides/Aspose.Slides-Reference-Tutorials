## Руководство по внедрению

В этом разделе подробно описывается, как настроить параметры масштабирования для слайдов и заметок.

### Настройка свойств масштабирования слайд-шоу

**Обзор**Определите масштаб основных слайдов презентации. Более высокий процент увеличивает размер контента на экране.

#### Шаг 1: Откройте или создайте презентацию
Начните с открытия существующего файла PowerPoint или создания нового:
```python
with slides.Presentation() as presentation:
    # Конфигурация масштабирования слайд-шоу будет здесь
```

#### Шаг 2: Настройте уровень масштабирования для просмотра слайдов
Задайте свойство масштаба, чтобы определить желаемый процент масштабирования:
```python
# Установите уровень масштабирования слайд-шоу на 100%
presentation.view_properties.slide_view_properties.scale = 100
```
**Объяснение**: `scale` Параметр принимает процентное значение, которое определяет видимость контента. Значение по умолчанию 100% означает стандартный размер.

### Настройка заметок Просмотр свойств масштабирования

**Обзор**: Отрегулируйте масштаб просмотра заметок, чтобы обеспечить правильный масштаб заметок докладчика во время презентаций.

#### Шаг 3: Настройте уровень масштабирования для просмотра заметок
Аналогично слайдам, установите процент масштабирования для заметок:
```python
# Установить масштаб просмотра заметок на 100%
presentation.view_properties.notes_view_properties.scale = 100
```
**Объяснение**: `scale` параметр обеспечивает отображение заметок в предпочитаемом вами размере.

### Сохранение презентации
Наконец, сохраните презентацию с примененными новыми настройками:
```python
# Сохраните измененную презентацию\presentation.save('YOUR_OUTPUT_DIRECTORY/rendering_set_zoom_out.pptx', slides.export.SaveFormat.PPTX)
```
**Объяснение**: На этом этапе изменения записываются в файл в указанном вами каталоге.

После установки вы можете инициализировать Aspose.Slides в своем скрипте Python:

## Руководство по внедрению: добавление слайдов макета
Теперь давайте разобьем процесс добавления слайдов макета на выполнимые шаги.
### Шаг 1: Откройте существующую презентацию
Начните с открытия файла PowerPoint, который вы хотите изменить:
```python
data_dir = "YOUR_DOCUMENT_DIRECTORY/welcome-to-powerpoint.pptx"
with slides.Presentation(data_dir) as presentation:
    # Дальнейшие операции по представлению
```
Этот код открывает указанную вами презентацию в режиме чтения и записи.
### Шаг 2: Получите доступ к слайдам макета и оцените их
Далее откройте коллекцию макетов слайдов из главного слайда:
```python
layout_slides = presentation.masters[0].layout_slides
```
Здесь мы получаем доступ к макетам первого мастер-слайда. 
#### Попробуйте получить определенный тип макета слайда
Попробуйте найти определенные типы макетов, такие как TITLE_AND_OBJECT или TITLE:
```python
layout_slide = (layout_slides.get_by_type(slides.SlideLayoutType.TITLE_AND_OBJECT) or
                layout_slides.get_by_type(slides.SlideLayoutType.TITLE))
```
Эта строка пытается получить желаемый тип слайда и возвращается к альтернативам, если они не найдены.
### Шаг 3: Обработка отсутствующих слайдов макета
Если предпочитаемый вами макет недоступен, реализуйте запасной вариант:
```python
if not layout_slide:
    for title_and_object_layout_slide in layout_slides:
        if title_and_object_layout_slide.name == "Title and Object":
            layout_slide = title_and_object_layout_slide
            break
    
    if not layout_slide:
        for titleLayoutSlide in layout_slides:
            if titleLayoutSlide.name == "Title":
                layout_slide = titleLayoutSlide
                break
        
        # Возврат к ПУСТОМУ или добавление нового типа слайда
        if not layout_slide:
            layout_slide = (layout_slides.get_by_type(slides.SlideLayoutType.BLANK) or
                            layout_slides.add(slides.SlideLayoutType.TITLE_AND_OBJECT, "Title and Object"))
```
Этот раздел обеспечивает надежность вашего кода, проверяя имена или добавляя новый тип слайда при необходимости.
### Шаг 4: Добавьте слайд
Вставьте пустой слайд, используя готовый макет:
```python
presentation.slides.insert_empty_slide(0, layout_slide)
```
Указав `0` в качестве индекса мы вставляем его в начало презентации.
### Шаг 5: Сохраните презентацию
Наконец, сохраните изменения в новом файле:
```python
out_dir = "YOUR_OUTPUT_DIRECTORY/layout_add_layout_slides_out.pptx"
presentation.save(out_dir, slides.export.SaveFormat.PPTX)
```
Это гарантирует сохранение всех изменений в выходном файле.

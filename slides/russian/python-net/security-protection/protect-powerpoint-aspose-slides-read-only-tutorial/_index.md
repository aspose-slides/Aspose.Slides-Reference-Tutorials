## Руководство по внедрению

### Рекомендуется установить презентацию только для чтения

**Обзор:**
В этом разделе описывается, как сделать презентацию PowerPoint доступной только для чтения с помощью `Aspose.Slides` библиотека. Эта настройка предполагает, что документ не следует редактировать, но не навязывает это строго.

#### Шаг 1: Импорт библиотеки
Начните с импорта необходимого модуля:

```python
import aspose.slides as slides
```

#### Шаг 2: Откройте или создайте презентацию
Вы можете открыть существующую презентацию или создать новую:

```python
with slides.Presentation() as pres:
    # Код для изменения презентации находится здесь
```

#### Шаг 3: Установите рекомендуемое свойство «Только для чтения»
Установите `read_only_recommended` свойство, предлагающее статус «только для чтения»:

```python
pres.protection_manager.read_only_recommended = True
```

*Почему это важно?*
На этом этапе ваша презентация помечается как рекомендованная для режима «только для чтения», что помогает предотвратить непреднамеренное редактирование.

#### Шаг 4: Сохраните презентацию
Сохраните изменения в указанном каталоге:

```python
pres.save("YOUR_OUTPUT_DIRECTORY/props_read_only_recommended_out.pptx", slides.export.SaveFormat.PPTX)
```

### Советы по устранению неполадок
- Убедитесь, что путь к выходному каталогу указан правильно.
- Убедитесь, что у вас есть права на запись в каталог.

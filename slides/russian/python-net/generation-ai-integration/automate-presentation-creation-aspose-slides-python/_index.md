## Руководство по внедрению

В этом разделе вы узнаете, как создать презентацию и настроить ее для включения изображения в мозаичном формате.

### Создание и настройка презентации

#### Обзор
Мы создадим новую презентацию, добавим слайд, вставим изображение и настроим фигуру с помощью формата заливки мозаичным изображением.

#### Доступ к первому слайду

Начните с доступа к первому слайду:

```python
# Инициализируйте объект Presentation_with slides.Presentation() как pres:
    # Доступ к первому слайду презентации
    first_slide = pres.slides[0]
```

#### Добавление изображения в презентацию

Загрузите и добавьте желаемое изображение из каталога:

```python
# Загрузите изображение из указанного каталога и добавьте его в коллекцию изображений презентации\with slides.Images.from_file("YOUR_DOCUMENT_DIRECTORY/image.png") как new_image:
    pp_image = pres.images.add_image(new_image)
```

#### Добавление фигуры с мозаичной заливкой изображения

Добавьте к слайду прямоугольную форму:

```python
# Добавьте прямоугольник к первому слайду.
ew_shape = first_slide.shapes.add_auto_shape(
    slides.ShapeType.RECTANGLE, 0, 0, 350, 350
)

# Установите тип заливки фигуры на «Изображение» и настройте ее для мозаичного размещения.
new_shape.fill_format.fill_type = slides.FillType.PICTURE
picture_fill_format = new_shape.fill_format.picture_fill_format

# Назначьте загруженное изображение для заполнения формы изображением format\ppicture_fill_format.picture.image = pp_image

# Настроить свойства мозаичной заливки\ppicture_fill_format.picture_fill_mode = slides.PictureFillMode.TILE
picture_fill_format.tile_offset_x = -275
picture_fill_format.tile_offset_y = -247
picture_fill_format.tile_scale_x = 120
picture_fill_format.tile_scale_y = 120
picture_fill_format.tile_alignment = slides.RectangleAlignment.BOTTOM_RIGHT
picture_fill_format.tile_flip = slides.TileFlip.FLIP_BOTH
```

#### Сохранение презентации

Наконец, сохраните вашу презентацию:

```python
# Сохраните презентацию в формате плитки изображения в выходной каталог\ppres.save("YOUR_OUTPUT_DIRECTORY/ImageTileExample.pptx")
```

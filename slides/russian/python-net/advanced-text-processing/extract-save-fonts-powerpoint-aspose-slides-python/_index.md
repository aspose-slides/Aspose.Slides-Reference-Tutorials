## Руководство по внедрению

Теперь давайте разберемся, как можно извлекать и сохранять шрифты из презентаций PowerPoint.

### Извлечение информации о шрифте

**Обзор:**
Эта функция позволяет получить доступ ко всем шрифтам, используемым в презентации, обеспечивая гибкость для дальнейшей обработки или анализа.

**Шаг 1: Загрузите презентацию**
Начните с загрузки файла PowerPoint. Это послужит основой для извлечения данных шрифтов.
```python
import aspose.slides as slides

# Откройте файл PowerPoint.
with slides.Presentation("YOUR_DOCUMENT_DIRECTORY/Presentation.pptx") as pres:
    # Извлечь менеджер шрифтов из презентации
```

**Шаг 2: Доступ к данным шрифтов**
Используйте `FontsManager` чтобы получить список всех шрифтов в вашем документе.
```python
# Получить все шрифты, используемые в презентации
fonts = pres.fonts_manager.get_fonts()
print("Fonts found:", [font.font_name for font in fonts])
```

### Сохранение шрифтов в виде файлов TTF

**Обзор:**
На этом этапе основное внимание уделяется преобразованию и сохранению определенного стиля шрифта в файл шрифта TrueType (TTF).

**Шаг 3: Извлечение байтов шрифта**
Извлечь байтовые данные выбранного шрифта. Затем эти данные можно сохранить в виде файла .ttf.
```python
# Извлечь массив байтов для обычного стиля первого шрифта
font_bytes = pres.fonts_manager.get_font_bytes(fonts[0], slides.drawing.FontStyle.REGULAR)
```

**Шаг 4: Сохраните данные шрифта**
Запишите извлеченные данные шрифта в файл TTF в нужном вам каталоге.
```python
# Сохраните байты шрифта как файл .ttf
with open("YOUR_OUTPUT_DIRECTORY/" + fonts[0].font_name + ".ttf", "wb") as f:
    f.write(font_bytes)
```

**Советы по устранению неполадок:**
- Убедитесь, что у вас есть права на запись в выходной каталог.
- Убедитесь, что путь презентации правильный и доступный.

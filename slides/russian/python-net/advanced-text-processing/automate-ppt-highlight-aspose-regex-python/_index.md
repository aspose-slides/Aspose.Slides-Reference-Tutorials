## Руководство по внедрению

Теперь давайте реализуем функцию выделения текста с помощью регулярных выражений.

### Открытие файла презентации
Чтобы работать с файлом PowerPoint, вам сначала нужно его открыть. Мы используем управление контекстом в Python, чтобы гарантировать эффективную обработку ресурсов:

```python
with slides.Presentation("YOUR_DOCUMENT_DIRECTORY/text_default_fonts.pptx") as presentation:
    # Код для управления презентацией находится здесь
```

### Доступ к текстовым фреймам
После загрузки презентации откройте текстовые рамки в определенных фигурах на слайде. Вот как выбрать первую фигуру на первом слайде:

```python
text_frame = presentation.slides[0].shapes[0].text_frame
```

### Выделение текста с помощью регулярных выражений
Чтобы выделить все слова, содержащие 10 или более символов, с помощью регулярного выражения, вам нужно использовать шаблон, который соответствует следующим критериям, и применить выделение:

```python
# Шаблон регулярного выражения \b[^\s]{10,}\b находит слова длиной 10 или более.
text_frame.highlight_regex(r"\b[^\s]{10,}\b", drawing.Color.blue)
```

**Объяснение**: 
- `\b` обозначает границу слова.
- `[^\s]{10,}` соответствует не менее 10 символам, не являющимся пробелами.
- `drawing.Color.blue` определяет цвет подсветки.

### Сохранение измененной презентации
После применения изменений сохраните презентацию в выходном каталоге:

```python
presentation.save("YOUR_OUTPUT_DIRECTORY/text_highlight_regex_out.pptx", slides.export.SaveFormat.PPTX)
```

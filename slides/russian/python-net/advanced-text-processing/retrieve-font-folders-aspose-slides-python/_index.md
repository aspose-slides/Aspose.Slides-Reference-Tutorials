## Руководство по внедрению

### Функция извлечения папок со шрифтами

Мы рассмотрим, как составить список каталогов, в которых хранятся файлы шрифтов, включая пользовательские каталоги, добавленные с помощью `LoadExternalFonts` метод.

#### Шаги по реализации

**Шаг 1: Импорт Aspose.Slides**

Начните с импорта необходимого модуля:

```python
import aspose.slides as slides
```

**Шаг 2: Определите функцию для получения папок шрифтов**

Создайте функцию, использующую API Aspose.Slides, для извлечения каталогов шрифтов.

```python
def get_fonts_folder():
    # Получить список папок шрифтов с помощью Aspose.Slides
    font_folders = slides.FontsLoader.get_font_folders()
    
    # Повторить и распечатать каждый путь к папке
    for font_folder in font_folders:
        print(font_folder)
```

**Объяснение**: 
- `get_font_folders()` извлекает все каталоги, где доступны шрифты, включая системные шрифты и шрифты, добавленные вручную.
- Функция выполняет итерацию по списку для отображения каждого каталога.

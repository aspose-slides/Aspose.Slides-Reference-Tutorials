## Руководство по внедрению: настройка языка текста по умолчанию

### Обзор

Эта функция позволяет вам установить язык текста по умолчанию для всех текстовых элементов в презентации PowerPoint, упрощая рабочие процессы за счет устранения повторяющихся задач.

### Пошаговая реализация

#### Создайте LoadOptions для указания языка текста по умолчанию

1. **Инициализировать LoadOptions**
   Начните с создания экземпляра `LoadOptions` чтобы указать желаемый язык текста по умолчанию:

   ```python
   load_options = slides.LoadOptions()
   ```

2. **Установить язык по умолчанию**
   Назначьте язык текста по умолчанию с помощью тега языка BCP-47 (например, «en-US» для английского языка, США):

   ```python
   load_options.default_text_language = "en-US"
   ```

#### Открыть и изменить презентацию
3. **Загрузить презентацию с помощью LoadOptions**
   Использовать `LoadOptions` при открытии презентации для применения языка текста по умолчанию:

   ```python
   with slides.Presentation(load_options) as pres:
       # Добавьте новый прямоугольник с текстом на первый слайд.
       shp = pres.slides[0].shapes.add_auto_shape(
           slides.ShapeType.RECTANGLE, 50, 50, 150, 50)
       shp.text_frame.text = "New Text"
   ```

4. **Доступ и проверка идентификатора языка**
   Вы можете проверить идентификатор языка фрагментов текста, чтобы убедиться, что он установлен правильно:

   ```python
   # Доступ к идентификатору языка для проверки (необязательный этап демонстрации)
   language_id = shp.text_frame.paragraphs[0].portions[0].portion_format.language_id
   ```

### Советы по устранению неполадок
- **Распространенная проблема**: Текст по умолчанию не отражает изменений.
  - **Решение**: Гарантировать `LoadOptions` правильно применяется при открытии презентации.

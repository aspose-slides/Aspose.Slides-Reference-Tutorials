## Руководство по внедрению

Теперь, когда все готово, давайте шаг за шагом реализуем функцию поиска и замены текста.

### Загрузите презентацию и настройте формат части

#### Обзор
Основная функциональность — загрузка презентации PowerPoint, поиск определенного текста, замена его новым текстом и применение пользовательских свойств шрифта.

#### Шаги

1. **Загрузите файл презентации**
   
   ```python
   DOCUMENT_DIR = 'YOUR_DOCUMENT_DIRECTORY/'
   OUTPUT_DIR = 'YOUR_OUTPUT_DIRECTORY/'

   def find_and_replace_text():
       # Откройте файл презентации из вашего каталога документов.
       with slides.Presentation(DOCUMENT_DIR + 'TextReplaceExample.pptx') as pres:
           pass  # Заполнитель для дополнительного кода
   ```

2. **Настроить формат порции**

   Создать `PortionFormat` экземпляр для определения того, как должен выглядеть замененный текст.

   ```python
   portion_format = slides.PortionFormat()
   portion_format.font_height = 24  # Установить высоту шрифта 24 пункта
   portion_format.font_italic = slides.NullableBool.TRUE  # Применить курсив
   portion_format.fill_format.fill_type = slides.FillType.SOLID  # Используйте сплошную заливку
   portion_format.fill_format.solid_fill_color.color = drawing.Color.red  # Установить красный цвет текста
   ```

3. **Найти и заменить текст**

   Используйте `SlideUtil.find_and_replace_text` метод автоматизации поиска и замены текста.

   ```python
   slides.util.SlideUtil.find_and_replace_text(
       pres, True, '[this block] ', 'my text', portion_format)
   ```

4. **Сохраните измененную презентацию**

   Сохраните изменения под новым именем файла в выходном каталоге.

   ```python
   pres.save(OUTPUT_DIR + 'TextReplaceExample-out.pptx', slides.export.SaveFormat.PPTX)
   ```

### Советы по устранению неполадок

- Обеспечить пути к `DOCUMENT_DIR` и `OUTPUT_DIR` верны.
- Убедитесь, что имя входного файла совпадает с именем в вашем каталоге.
- Проверьте наличие орфографических ошибок в текстовых шаблонах.

## Руководство по внедрению

В этом разделе описывается процесс извлечения эффективных значений из текстовых фреймов и фрагментов.

### Понимание эффективных ценностей

Эффективные значения в презентациях определяют, как применяются стили, когда есть иерархия или наследование форматирования. Извлечение этих значений позволяет вам понять, какие свойства фактически влияют на содержимое вашего слайда.

#### Шаг 1: Загрузите презентацию

```python
def get_effective_values():
    data_dir = 'YOUR_DOCUMENT_DIRECTORY/'
    file_name = 'text_add_animation_effect.pptx'
    
    with slides.Presentation(data_dir + file_name) as pres:
        # Доступ к первой фигуре на первом слайде
        shape = pres.slides[0].shapes[0]
```
- **Почему этот шаг:** Мы загружаем презентацию, чтобы получить доступ к ее структуре, уделяя особое внимание текстовым рамкам внутри фигур.

#### Шаг 2: Извлечение значений формата текстового фрейма

```python
        local_text_frame_format = shape.text_frame.text_frame_format
        effective_text_frame_format = local_text_frame_format.get_effective()
```
- **Объяснение:** `local_text_frame_format` содержит настройки формата, применяемые непосредственно к текстовому фрейму. Метод `get_effective()` извлекает окончательные значения после рассмотрения всех унаследованных свойств.

#### Шаг 3: Извлечение значений формата порции

```python
        local_portion_format = shape.text_frame.paragraphs[0].portions[0].portion_format
        effective_portion_format = local_portion_format.get_effective()
```
- **Почему этот шаг:** Доступ к формату части позволяет увидеть, как стилизованы текстовые части, учитывая как прямые, так и унаследованные свойства.

#### Шаг 4: Отображение эффективных значений

```python
        print('Effective Text Frame Format:', effective_text_frame_format)
        print('Effective Portion Format:', effective_portion_format)
```
- **Цель:** Печать этих значений позволяет нам проверить правильность применения стилей в содержимом нашей презентации.

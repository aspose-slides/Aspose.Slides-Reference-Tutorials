# Извлечение и управление гиперссылками в PowerPoint с помощью Aspose.Slides для Python: подробное руководство

В этом руководстве показано, как извлекать как текущие (поддельные), так и оригинальные гиперссылки из элементов слайда с помощью библиотеки Aspose.Slides для Python. Освоив эти методы, вы обеспечите точность информации о ссылках в своих презентациях.

**Что вы узнаете:**
- Настройка Aspose.Slides для Python.

- Вопросы производительности и стратегии оптимизации.

## Руководство по внедрению

В этом разделе вы узнаете, как извлечь текущие и исходные гиперссылки из слайдов PowerPoint.

### Извлечение URL-адресов из слайдов

#### Обзор

Извлекайте как поддельные (текущие), так и оригинальные гиперссылки, чтобы обеспечить прозрачность любых изменений в элементах слайда с течением времени.

#### Пошаговая реализация

**1. Импортируйте необходимые библиотеки**
Начните с импорта необходимого модуля Aspose.Slides:

```python
import aspose.slides as slides
```

**2. Настройте пути к файлам**
Определите пути для документа презентации и выходного каталога:

```python
document_path = "YOUR_DOCUMENT_DIRECTORY/ExternalUrlOriginal.pptx"
output_directory = "YOUR_OUTPUT_DIRECTORY/"
```

**3. Загрузите презентацию**
Откройте файл PowerPoint с помощью Aspose.Slides. `Presentation` сорт:

```python
with slides.Presentation(document_path) as presentation:
    # Ваш код обработки будет здесь
```

**4. Доступ к элементам слайда**
Перейдите к определенной форме и текстовому элементу, из которого вы хотите извлечь гиперссылки:

```python
portion = presentation.slides[0].shapes[1].text_frame.paragraphs[0].portions[0]
```
*Здесь, `shapes[1]` относится ко второй форме на первом слайде. Измените этот индекс в соответствии с вашими конкретными потребностями.*

**5. Извлечение информации из гиперссылки**
Получите как поддельные, так и оригинальные гиперссылки:

```python
external_url = portion.portion_format.hyperlink_click.external_url
external_url_original = portion.portion_format.hyperlink_click.external_url_original
```

**6. Отображение URL-адресов**
Распечатайте или запишите эти URL-адреса для проверки:

```python
print("Fake External Hyperlink:", external_url)
print("Real External Hyperlink:", external_url_original)
```

### Советы по устранению неполадок
- **Файл не найден:** Убедитесь, что пути к файлам указаны правильно и файлы существуют в указанных местах.
- **Ошибки индекса формы:** Проверьте индексы, используемые для доступа к фигурам и текстовым элементам, поскольку они должны соответствовать существующим элементам.

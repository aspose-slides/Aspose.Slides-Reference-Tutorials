def load_presentation(file_path):

#### Шаг 2: Настройте параметры текстового поиска
Определите, как будет вести себя текстовый поиск:
```python
def configure_search_options():
    options = slides.TextSearchOptions()
    options.whole_words_only = True
    return options
```
Эта настройка гарантирует, что будут выделены только целые слова, соответствующие вашим критериям.

#### Шаг 3: Выделите определенные слова
Использовать `highlight_text` для применения цветного выделения:
```python
def highlight_specific_words(presentation, shape_index=0):
    # Выделите «заголовок» светло-голубым цветом.
    presentation.slides[shape_index].shapes[0].text_frame.highlight_text("title", drawing.Color.light_blue)

    # Выделите «кому», используя настроенные параметры поиска, фиолетовым цветом
    options = configure_search_options()
    presentation.slides[shape_index].shapes[0].text_frame.highlight_text("to", drawing.Color.violet, options, None)
```

#### Шаг 4: Сохраните измененную презентацию.
Сохранить изменения обратно в файл:
```python
def save_presentation(presentation, output_path):
    # Сохраните обновленную презентацию
    presentation.save(output_path, slides.export.SaveFormat.PPTX)
```
Этот шаг гарантирует сохранение всех изменений в новом или существующем файле.

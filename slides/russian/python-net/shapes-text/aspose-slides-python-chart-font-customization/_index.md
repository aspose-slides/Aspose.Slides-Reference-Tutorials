## Руководство по внедрению

### Добавление кластеризованной столбчатой диаграммы и включение таблицы данных

#### Обзор

Во-первых, мы добавим в нашу презентацию кластеризованную столбчатую диаграмму и включим ее функцию таблицы данных.

#### Пошаговая реализация

1. **Добавьте кластеризованную столбчатую диаграмму:**
   
   Добавьте следующий фрагмент кода для создания простой кластеризованной столбчатой диаграммы на первом слайде:

    ```python
    chart = pres.slides[0].shapes.add_chart(
        slides.charts.ChartType.CLUSTERED_COLUMN, 50, 50, 600, 400)
    ```
   
2. **Включить отображение таблицы данных:**
   
   Далее включите таблицу данных для диаграммы, чтобы разрешить настройку шрифта:

    ```python
    chart.has_data_table = True
    ```

### Настройка свойств шрифта

#### Обзор

Включив таблицу данных, мы теперь можем настроить свойства ее шрифта, чтобы улучшить читаемость и стиль.

#### Пошаговая реализация

1. **Установить жирный шрифт:**
   
   Используйте этот фрагмент, чтобы сделать текст таблицы данных жирным:

    ```python
    chart.chart_data_table.text_format.portion_format.font_bold = slides.NullableBool.TRUE
    ```

2. **Настройте высоту шрифта:**
   
   Измените размер шрифта для лучшей видимости:

    ```python
    chart.chart_data_table.text_format.portion_format.font_height = 20
    ```

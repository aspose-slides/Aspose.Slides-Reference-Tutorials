# Как создать многоуровневые маркированные списки в презентациях с помощью Aspose.Slides для Python

Это руководство проведет вас через реализацию многоуровневых маркеров в ваших слайдах с помощью Aspose.Slides для Python — мощного инструмента, который упрощает автоматизацию презентаций.

- Как настроить Aspose.Slides для Python

- Настройка символов и цветов маркеров
- Эффективное сохранение презентаций

## Руководство по внедрению

В этом разделе мы рассмотрим создание многоуровневых маркеров на слайде. Мы разобьем это на управляемые шаги.

### Создание слайда с многоуровневыми маркерами

**Обзор:**
Мы добавим автофигуру (прямоугольник) к нашему первому слайду и заполним ее текстом, содержащим несколько уровней маркеров.

1. **Доступ к первому слайду**
   ```python
   # Доступ к первому слайду презентации
   slide = pres.slides[0]
   ```

2. **Добавление автофигуры**
   ```python
   # Добавьте прямоугольник, в который будут помещены пункты списка.
   auto_shape = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 200, 200, 400, 200)
   ```

3. **Настройка текстового фрейма**
   Здесь мы настраиваем текстовую рамку, которая будет содержать наши маркеры.
   
   ```python
   # Получить и очистить все абзацы по умолчанию в текстовом фрейме
   text = auto_shape.add_text_frame("")
   text.paragraphs.clear()
   ```

4. **Добавление маркеров**
   Мы создаем и добавляем несколько уровней маркированных списков, каждый из которых имеет различные символы и глубину отступов.
   
   - **Маркер первого уровня:**
     ```python
     para1 = slides.Paragraph()
     para1.text = "Content"
     para1.paragraph_format.bullet.type = slides.BulletType.SYMBOL
     para1.paragraph_format.bullet.char = chr(8226)  # Характер пули
     para1.paragraph_format.default_portion_format.fill_format.fill_type = slides.FillType.SOLID
     para1.paragraph_format.default_portion_format.fill_format.solid_fill_color.color = drawing.Color.black
     para1.paragraph_format.depth = 0  # Уровень 0 пуля
     ```
   
   - **Маркер второго уровня:**
     ```python
     para2 = slides.Paragraph()
     para2.text = "Second Level"
     para2.paragraph_format.bullet.type = slides.BulletType.SYMBOL
     para2.paragraph_format.bullet.char = '-'  # Характер пули
     para2.paragraph_format.default_portion_format.fill_type = slides.FillType.SOLID
     para2.paragraph_format.default_portion_format.fill_format.solid_fill_color.color = drawing.Color.black
     para2.paragraph_format.depth = 1  # Уровень 1 пуля
     ```
   
   - **Маркер третьего уровня:**
     ```python
     para3 = slides.Paragraph()
     para3.text = "Third Level"
     para3.paragraph_format.bullet.type = slides.BulletType.SYMBOL
     para3.paragraph_format.bullet.char = chr(8226)  # Характер пули
     para3.paragraph_format.default_portion_format.fill_type = slides.FillType.SOLID
     para3.paragraph_format.default_portion_format.fill_format.solid_fill_color.color = drawing.Color.black
     para3.paragraph_format.depth = 2  # Уровень 2 пуля
     ```
   
   - **Маркер четвертого уровня:**
     ```python
     para4 = slides.Paragraph()
     para4.text = "Fourth Level"
     para4.paragraph_format.bullet.type = slides.BulletType.SYMBOL
     para4.paragraph_format.bullet.char = '-'  # Характер пули
     para4.paragraph_format.default_portion_format.fill_type = slides.FillType.SOLID
     para4.paragraph_format.default_portion_format.fill_format.solid_fill_color.color = drawing.Color.black
     para4.paragraph_format.depth = 3  # Уровень 3 пуля
     ```
   
5. **Добавление абзацев в текстовый фрейм**
   После настройки всех абзацев добавьте их в текстовый фрейм:
   
   ```python
   # Добавить все абзацы в коллекцию текстового фрейма
   text.paragraphs.add(para1)
   text.paragraphs.add(para2)
   text.paragraphs.add(para3)
   text.paragraphs.add(para4)
   ```

6. **Сохранение презентации**
   Наконец, сохраните вашу презентацию как файл PPTX:
   
   ```python
   # Сохранить презентацию
   pres.save("YOUR_OUTPUT_DIRECTORY/text_multilevel_bullet_out.pptx", slides.export.SaveFormat.PPTX)
   ```

### Базовая инициализация и настройка

Вот как вы начинаете свою первую презентацию:

```python
import aspose.slides as slides

# Создать экземпляр класса Presentation, представляющего файл PPTX
class Presentation:
    def __enter__(self):
        self.pres = slides.Presentation()
        return self.pres

    def __exit__(self, exc_type, exc_val, exc_tb):
        del self.pres

with Presentation() as pres:
    # Ваш код будет здесь
```

Это создаст новый экземпляр презентации, в котором вы сможете добавлять фигуры и управлять ими.

## Руководство по внедрению

### Соедините фигуры с Aspose.Slides в Python

Давайте разберем пошаговую инструкцию по соединению двух фигур с помощью соединителя.

**1. Добавление фигур**

Начните с добавления на слайд эллипса и прямоугольника:

```python
# Доступ к коллекции фигур для выбранного слайда
shapes = pres.slides[0].shapes

# Добавить автофигуру Эллипс в позицию (0, 100) с шириной и высотой 100
elipse = shapes.add_auto_shape(slides.ShapeType.ELLIPSE, 0, 100, 100, 100)

# Добавить автофигуру «Прямоугольник» в позицию (100, 300) с шириной и высотой 100
rectangle = shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 100, 300, 100, 100)
```

**2. Добавление соединителя**

Далее создайте соединитель, чтобы связать эти две фигуры:

```python
# Добавление формы соединителя в коллекцию форм слайдов
contractor = shapes.add_connector(slides.ShapeType.BENT_CONNECTOR2, 0, 0, 10, 10)

# Присоединение фигур к соединителям
contractor.start_shape_connected_to = elipse
contractor.end_shape_connected_to = rectangle

# Вызовите функцию перемаршрутизации, чтобы задать автоматический кратчайший путь между фигурами.
contractor.reroute()
```

The `add_connector` Метод создает изогнутую форму соединителя. `reroute()` функция автоматически регулирует путь соединителя.

**3. Сохранение презентации**

Наконец, сохраните вашу презентацию:

```python
pres.save("YOUR_OUTPUT_DIRECTORY/shapes_connect_shapes_using_connectors_out.pptx", slides.export.SaveFormat.PPTX)
```

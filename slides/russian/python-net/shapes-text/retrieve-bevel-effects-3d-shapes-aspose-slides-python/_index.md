## Руководство по внедрению

Извлеките свойства скоса из верхней грани трехмерной фигуры с помощью Aspose.Slides для Python.

### Обзор функции

Получайте доступ и распечатывайте подробные свойства скоса, такие как тип, ширина и высота, для точного управления визуальными эффектами презентации.

#### Пошаговая реализация

1. **Откройте файл PowerPoint.**
   Откройте файл с 3D-фигурами:

   ```python
   input_file_path = 'YOUR_DOCUMENT_DIRECTORY/shapes_3d_effective.pptx'
   
   with slides.Presentation(input_file_path) as pres:
       # Доступ к первому слайду и его первой форме
       shape = pres.slides[0].shapes[0]
   ```

2. **Получить свойства формата 3D**
   Извлечение эффективных свойств 3D-формата формы:

   ```python
   three_d_effective_data = shape.three_d_format.get_effective()
   ```

3. **Свойства верхней грани выходного скоса**
   Тип, ширина и высота скоса печати для анализа:

   ```python
   print("= Effective shape's top face relief properties =")
   print("Type: " + str(three_d_effective_data.bevel_top.bevel_type))
   print("Width: " + str(three_d_effective_data.bevel_top.width))
   print("Height: " + str(three_d_effective_data.bevel_top.height))
   ```

**Советы по устранению неполадок:** 
- Убедитесь, что путь к документу указан правильно.
- Убедитесь, что у доступных фигур есть свойства 3D-форматирования.

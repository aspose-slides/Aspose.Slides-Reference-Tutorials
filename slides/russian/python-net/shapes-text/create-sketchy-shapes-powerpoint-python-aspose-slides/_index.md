### Создание эскизных фигур в PowerPoint

#### Обзор
Эта функция позволяет добавлять эффект эскизных линий к фигурам в презентации, придавая им художественный и нарисованный от руки вид.

#### Добавление прямоугольника со стилем линии Scribble

##### Шаг 1: Инициализация новой презентации
Начните с создания нового экземпляра презентации:
```python
with slides.Presentation() as pres:
    # Продолжайте добавлять формы
```

##### Шаг 2: Добавьте автофигуру (прямоугольник)
Вставьте прямоугольную форму на первый слайд, используя `add_auto_shape`:
```python
shape = pres.slides[0].shapes.add_auto_shape(
    slides.ShapeType.RECTANGLE, 20, 20, 300, 150
)
```
Параметры определяют тип фигуры и ее положение/размер на слайде.

##### Шаг 3: Установите тип заполнения на «NO_FILL»
Чтобы сосредоточиться на эффекте эскиза, удалите всю заливку:
```python
shape.fill_format.fill_type = slides.FillType.NO_FILL
```

##### Шаг 4: Примените эффект наброска каракулей
Улучшите свою форму с помощью каракулей:
```python
shape.line_format.sketch_format.sketch_type = slides.LineSketchType.SCRIBBLE
```
Этот параметр придает контуру фигуры схематичный вид.

##### Шаг 5: Сохраните в формате PNG и PPTX
Сначала экспортируйте слайд как изображение, а затем сохраните его как файл PowerPoint:
```python
pres.slides[0].get_image(4 / 3, 4 / 3).save(
    "YOUR_OUTPUT_DIRECTORY/shapes_sketch_format_out.png",
    slides.ImageFormat.PNG
)
pres.save("YOUR_OUTPUT_DIRECTORY/shapes_sketch_format_out.pptx", 
          slides.export.SaveFormat.PPTX)
```
Заменять `"YOUR_OUTPUT_DIRECTORY"` с желаемым путем сохранения.

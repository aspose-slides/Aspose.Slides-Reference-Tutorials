#### Шаг 2: Доступ и создание миниатюры
Откройте первый слайд и его первую фигуру, затем создайте миниатюру:

```python
        # Предположим, что есть по крайней мере один слайд и одна форма.
        shape = presentation.slides[0].shapes[0]

        # Создайте миниатюру внешнего вида фигуры
        with shape.get_image(slides.ShapeThumbnailBounds.APPEARANCE, 1, 1) as image:
            # Сохраните миниатюру в формате PNG
            image.save(output_directory, slides.ImageFormat.PNG)
```

**Объяснение:**
- `shape.get_image(...)`: Захватывает изображение внешнего вида формы. Параметры `(slides.ShapeThumbnailBounds.APPEARANCE, 1, 1)` укажите целевую фигуру, привязанную к внешнему виду, с коэффициентами масштабирования для ширины и высоты.
- `image.save()`: Сохраняет созданную миниатюру в формате PNG в указанном вами выходном каталоге.

# Как создавать составные пользовательские фигуры в PowerPoint с помощью Aspose.Slides для Python

В этом уроке мы рассмотрим, как создавать составные фигуры с использованием двух `GeometryPath` объекты с Aspose.Slides для Python. К концу этого руководства вы поймете:
- Настройка Aspose.Slides в вашей среде Python
- Создание пользовательских геометрических траекторий
- Объединение нескольких путей в одну форму

## Руководство по внедрению
Настроив среду, давайте создадим составную пользовательскую фигуру в PowerPoint.

### Шаг 1: Инициализация презентации
Начните с создания нового объекта презентации, который послужит холстом для форм и дизайнов.

```python
with slides.Presentation() as pres:
    # Код для управления слайдами находится здесь.
```
The `with` заявление обеспечивает эффективное управление ресурсами, автоматически закрывая презентацию по завершении.

### Шаг 2: Добавьте прямоугольную форму.
Добавьте авто-фигуру типа прямоугольник к первому слайду. Это служит нашей базовой фигурой для композитной настройки.

```python
shape = pres.slides[0].shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 100, 100, 200, 100)
```
Здесь, `add_auto_shape` создает прямоугольник с указанными параметрами положения и размера (x, y, ширина, высота).

### Шаг 3: Создание первого геометрического контура
Определите верхнюю часть вашей составной фигуры, используя `GeometryPath`. Это подразумевает перемещение к определенным координатам и рисование линий.

```python
g = slides.GeometryPath()
g.move_to(0, 0)  # Начните с исходной точки (верхний левый угол).
g.line_to(shape.width, 0)  # Проведите линию сверху.
g.line_to(shape.width, shape.height / 3)  # Опустите на одну треть высоты.
g.line_to(0, shape.height / 3)  # Вернитесь к левому краю на треть высоты.
g.close_figure()  # Замкните контур, чтобы получилась замкнутая фигура.
```

### Шаг 4: Создание второго геометрического контура
Аналогичным образом определите нижнюю часть вашей составной фигуры, используя другой `GeometryPath`.

```python
g1 = slides.GeometryPath()
g1.move_to(0, shape.height / 3 * 2)  # Начните с высоты двух третей.
g1.line_to(shape.width, shape.height / 3 * 2)  # Проведите линию по нижнему краю.
g1.line_to(shape.width, shape.height)  # Переместитесь в нижний правый угол.
g1.line_to(0, shape.height)  # Вернитесь в левый нижний угол.
g1.close_figure()  # Замкните контур, чтобы получилась замкнутая фигура.
```

### Шаг 5: Объедините геометрические контуры
Объедините оба геометрических контура в одну составную пользовательскую форму, используя `set_geometry_paths`.

```python
shape.set_geometry_paths([g, g1])
```
На этом этапе два отдельных контура объединяются в одну целостную форму на слайде.

### Шаг 6: Сохраните презентацию
Наконец, сохраните презентацию в указанном каталоге.

```python
pres.save("YOUR_OUTPUT_DIRECTORY/shapes_create_geometry_path_out.pptx", slides.export.SaveFormat.PPTX)
```
Заменять `YOUR_OUTPUT_DIRECTORY` на фактический путь, по которому вы хотите сохранить свой файл.

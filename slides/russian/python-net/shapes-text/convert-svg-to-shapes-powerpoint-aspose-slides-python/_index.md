## Руководство по внедрению

В этом разделе подробно описывается процесс преобразования изображения SVG в группу фигур в презентации PowerPoint.

### Преобразование изображения SVG в группу фигур

Вот как можно преобразовать встроенное в слайд изображение SVG в управляемую группу фигур:

#### Обзор

Загрузите презентацию, найдите в ней изображение SVG и преобразуйте это изображение в группу фигур для расширенных возможностей редактирования.

#### Шаг 1: Загрузите презентацию

Откройте файл PowerPoint с помощью Aspose.Slides:

```python
with slides.Presentation('YOUR_DOCUMENT_DIRECTORY/save_convert_svg_to_group_of_shapes.pptx') as pres:
    picture_frame = pres.slides[0].shapes[0]
```

#### Шаг 2: Проверьте наличие SVG-изображения

Определите, содержит ли первая фигура на слайде изображение SVG:

```python
svg_image = picture_frame.picture_format.picture.image.svg_image
if svg_image is not None:
    # Продолжить преобразование
```

The `picture_format` объект определяет, содержит ли фрейм SVG.

#### Шаг 3: Преобразование в группу фигур

Преобразуем SVG в группу фигур в исходном положении:

```python
group_shape = pres.slides[0].shapes.add_group_shape(
    svg_image,
    picture_frame.frame.x,
    picture_frame.frame.y,
    picture_frame.frame.width,
    picture_frame.frame.height
)
```

The `add_group_shape` Метод имеет решающее значение для поддержания согласованности макета.

#### Шаг 4: Удалите исходную рамку

После конвертации удалите исходное изображение SVG:

```python
pres.slides[0].shapes.remove(picture_frame)
```

Этот шаг гарантирует отсутствие дублирования контента на слайде.

#### Шаг 5: Сохраните презентацию

Наконец, сохраните измененную презентацию в новом файле:

```python
pres.save('YOUR_OUTPUT_DIRECTORY/save_convert_svg_to_group_of_shapes_out.pptx', slides.export.SaveFormat.PPTX)
```

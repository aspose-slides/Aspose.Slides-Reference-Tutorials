#### Шаг 2: Откройте первый слайд (H4)
Измените первый слайд, чтобы вставить эллипс.
```python
        # Откройте первый слайд.
        slide = pres.slides[0]
```

#### Шаг 3: Добавьте форму эллипса (H4)
Вставьте эллипс в указанное место с заданными размерами, используя `add_auto_shape` метод.
```python
        # Вставьте в слайд фигуру эллипса.
        slide.shapes.add_auto_shape(slides.ShapeType.ELLIPSE, 50, 150, 150, 50)
```
Здесь:
- **ShapeType.ELLIPSE**: Определяет форму эллипса.
- **50, 150**: Координаты x и y для позиционирования на слайде.
- **150, 50**: Ширина и высота эллипса.

#### Шаг 4: Сохраните презентацию (H4)
Сохраните презентацию в желаемом месте в формате PPTX:

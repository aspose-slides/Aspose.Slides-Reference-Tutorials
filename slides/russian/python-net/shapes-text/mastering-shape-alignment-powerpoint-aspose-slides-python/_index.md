## Руководство по внедрению

### Выровнять фигуры на слайде (H2)

Эта функция предназначена для выравнивания фигур в нижней части слайда.

#### Обзор

Мы добавим три прямоугольника на слайд и выровняем их по нижнему краю с помощью утилит выравнивания Aspose.Slides.

#### Шаги по реализации

##### Шаг 1: Создание и загрузка презентации

Начните с загрузки презентации с пустым макетом по умолчанию:

```python
with slides.Presentation() as pres:
    slide = pres.slides[0]
```

##### Шаг 2: Добавьте фигуры на слайд

Добавьте три прямоугольника в разные места на слайде.

```python
slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 100, 100, 100, 100)
slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 200, 200, 100, 100)
slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 300, 300, 100, 100)
```

##### Шаг 3: Выровняйте фигуры

Выровняйте все фигуры по низу слайда с помощью `align_shapes` метод.

```python
slides.util.SlideUtil.align_shapes(
    slides.ShapesAlignmentType.ALIGN_BOTTOM, True, pres.slides[0]
)
```

##### Шаг 4: Сохраните презентацию

Наконец, сохраните презентацию в указанном выходном каталоге.

```python
pres.save("YOUR_OUTPUT_DIRECTORY/shapes_align_out.pptx", slides.export.SaveFormat.PPTX)
```

### Выровнять фигуры в групповой форме на новом слайде (H2)

Теперь давайте рассмотрим выравнивание фигур внутри групповой фигуры на новом слайде.

#### Обзор

Эта функция позволяет создать набор прямоугольников внутри группы и выровнять их по левому краю.

#### Шаги по реализации

##### Шаг 1: Добавьте новый слайд с помощью групповой формы

Добавьте пустой слайд, а затем создайте внутри него групповую фигуру.

```python
with slides.Presentation() as pres:
    slide = pres.slides.add_empty_slide(pres.layout_slides[0])
group_shape = slide.shapes.add_group_shape()
```

##### Шаг 2: Добавьте прямоугольники к групповой фигуре.

Вставьте четыре прямоугольника в только что созданную групповую фигуру.

```python
group_shape.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 350, 50, 50, 50)
group_shape.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 450, 150, 50, 50)
group_shape.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 550, 250, 50, 50)
group_shape.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 650, 350, 50, 50)
```

##### Шаг 3: Выровняйте фигуры внутри группы

Выровняйте все фигуры по левому краю, используя:

```python
slides.util.SlideUtil.align_shapes(
    slides.ShapesAlignmentType.ALIGN_LEFT, False, group_shape
)
```

##### Шаг 4: Сохраните презентацию

Сохраните изменения, как и прежде.

```python
pres.save("YOUR_OUTPUT_DIRECTORY/shapes_align_out.pptx", slides.export.SaveFormat.PPTX)
```

### Выровнять определенные фигуры в групповой форме на новом слайде (H2)

Для большего контроля вы можете выровнять определенные фигуры внутри групповой фигуры по их индексам.

#### Обзор

Эта функция демонстрирует, как выборочно выравнивать определенные фигуры в группе.

#### Шаги по реализации

##### Шаг 1: Подготовьте слайд и групповую форму

Как и прежде, добавьте новый слайд с групповой формой:

```python
with slides.Presentation() as pres:
    slide = pres.slides.add_empty_slide(pres.layout_slides[0])
group_shape = slide.shapes.add_group_shape()
```

##### Шаг 2: Добавьте прямоугольники к групповой фигуре.

Вставьте в эту группу четыре прямоугольника.

```python
group_shape.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 350, 50, 50, 50)
group_shape.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 450, 150, 50, 50)
group_shape.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 550, 250, 50, 50)
group_shape.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 650, 350, 50, 50)
```

##### Шаг 3: Выровняйте определенные фигуры

Выровняйте только первый и третий прямоугольники по левому краю, указав их индексы:

```python
slides.util.SlideUtil.align_shapes(
    slides.ShapesAlignmentType.ALIGN_LEFT,
    False,
    group_shape,
    [0, 2]  # Индексы фигур для выравнивания
)
```

##### Шаг 4: Сохраните презентацию

Сохраните презентацию, как и прежде.

```python
pres.save("YOUR_OUTPUT_DIRECTORY/shapes_align_out.pptx", slides.export.SaveFormat.PPTX)
```

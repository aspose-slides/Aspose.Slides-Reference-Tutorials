## Руководство по внедрению

### Функция: Изменение свойств шрифта

#### Обзор:
Эта функция демонстрирует, как можно изменять свойства шрифта, такие как семейство, жирность, курсив и цвет текста на слайдах PowerPoint, используя Aspose.Slides для Python.

#### Действия по изменению шрифтов:

**1. Загрузите презентацию**

```python
import aspose.slides as slides

# Открыть существующую презентацию
with slides.Presentation("YOUR_DOCUMENT_DIRECTORY/text_default_fonts.pptx") as pres:
    slide = pres.slides[0]
```

Этот фрагмент кода загружает файл PowerPoint, позволяя вам получить доступ к его слайдам для редактирования.

**2. Доступ к текстовым фреймам**

```python
# Извлечь текстовые рамки из первых двух фигур на слайде
shape1 = slide.shapes[0]  # Первая форма
tf1 = shape1.text_frame
shape2 = slide.shapes[1]  # Вторая форма
tf2 = shape2.text_frame

# Получить первый абзац из каждого текстового фрейма
para1 = tf1.paragraphs[0]
para2 = tf2.paragraphs[0]

# Доступ к первой части текста в каждом абзаце
port1 = para1.portions[0]
port2 = para2.portions[0]
```

Доступ к текстовым фреймам и абзацам имеет решающее значение для определения того, какие части текста вы хотите изменить.

**3. Определите новые семейства шрифтов**

```python
import aspose.slides as slides

# Установить новые семейства шрифтов
fd1 = slides.FontData("Elephant")  # Жирный шрифт в стиле слона
dfd2 = slides.FontData("Castellar")  # Шрифт Кастеллар

port1.portion_format.latin_font = fd1
port2.portion_format.latin_font = fd2
```

Здесь мы указываем желаемые шрифты для текстовых частей, повышая визуальную привлекательность.

**4. Применение жирного и курсивного начертаний**

```python
# Установить стиль шрифта на жирный
port1.portion_format.font_bold = slides.NullableBool.TRUE
port2.portion_format.font_bold = slides.NullableBool.TRUE

# Применить курсив
port1.portion_format.font_italic = slides.NullableBool.TRUE
port2.portion_format.font_italic = slides.NullableBool.TRUE
```

Добавление жирного и курсивного начертаний выделяет определенный текст, делая его заметным.

**5. Изменить цвет шрифта**

```python
import aspose.pydrawing as drawing

# Установить цвета шрифта
port1.portion_format.fill_format.fill_type = slides.FillType.SOLID
port1.portion_format.fill_format.solid_fill_color.color = drawing.Color.purple  # Фиолетовый цвет

port2.portion_format.fill_format.fill_type = slides.FillType.SOLID
port2.portion_format.fill_format.solid_fill_color.color = drawing.Color.peru  # Перуанский цвет
```

Настройка цвета шрифта может сделать вашу презентацию более яркой и интересной.

**6. Сохраните измененную презентацию.**

```python
# Сохранить изменения в новом файле
pres.save("YOUR_OUTPUT_DIRECTORY/text_font_properties_out.pptx", slides.export.SaveFormat.PPTX)
```

Сохранение измененной презентации гарантирует сохранение всех изменений для будущего использования.

### Советы по устранению неполадок:

- Убедитесь, что индексы слайдов и количество форм соответствуют таковым в вашем конкретном файле презентации, чтобы избежать ошибок индексации.

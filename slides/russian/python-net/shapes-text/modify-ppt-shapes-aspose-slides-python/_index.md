## Руководство по внедрению
В этом разделе мы рассмотрим процесс изменения настроек формы.

### Доступ и изменение настроек формы
#### Обзор
Эта функция позволяет вам получить доступ к определенным точкам настройки фигур PowerPoint и изменять их свойства программным способом. Мы покажем, как работать с фигурами RoundRectangle и Arrow в презентации.

#### Шаг 1: Загрузите презентацию
Сначала загрузите существующий файл PowerPoint с помощью Aspose.Slides:

```python
with slides.Presentation('YOUR_DOCUMENT_DIRECTORY/PresetGeometry.pptx') as pres:
    # Доступ к первой форме первого слайда
    shape = pres.slides[0].shapes[0]
```

#### Шаг 2: Отображение типов корректировки для фигуры
Чтобы понять, какие корректировки доступны, выполните их итерацию:

```python
print("Adjustment types for a Rectangle:")
for i in range(len(shape.adjustments)):
    print(f"\tType for point {i} is", shape.adjustments[i].type.name)
```

#### Шаг 3: Измените точки корректировки
Если тип корректировки соответствует вашим критериям, измените его значение:

```python
# Пример: удвоение размера угла RoundRectangle
corner_adjustment_index = next((i for i, adj in enumerate(shape.adjustments) if adj.type == slides.ShapeAdjustmentType.CORNER_SIZE), None)
if corner_adjustment_index is not None:
    shape.adjustments[corner_adjustment_index].angle_value *= 2
```

#### Шаг 4: Сохраните изменения.
После внесения изменений сохраните презентацию, чтобы отразить изменения:

```python
pres.save('YOUR_OUTPUT_DIRECTORY/PresetGeometry_out.pptx', slides.export.SaveFormat.PPTX)
```

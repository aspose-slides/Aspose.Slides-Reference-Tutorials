### Обзор: экспорт фигур в SVG

Эта функция позволяет извлекать и сохранять определенные фигуры из презентаций PowerPoint в виде файлов SVG. Это особенно полезно для веб-разработчиков, которым нужна высококачественная графика, или дизайнеров, желающих повторно использовать элементы слайдов в разных форматах.

#### Пошаговая реализация

##### Доступ к презентации
Начните с открытия файла презентации, в котором находится ваша целевая форма:

```python
document_directory = "YOUR_DOCUMENT_DIRECTORY/"
pres = slides.Presentation(document_directory + "welcome-to-powerpoint.pptx")
```

##### Извлечение фигур
Откройте первый слайд и извлеките нужные фигуры:

```python
slide = pres.slides[0]
shape = slide.shapes[0]  # При необходимости отрегулируйте индекс для конкретной формы.
```
The `pres.slides` объект содержит все слайды вашей презентации, и `slide.shapes` содержит все фигуры в пределах определенного слайда.

##### Запись в формат SVG
Откройте файловый поток для записи вывода SVG:

```python
output_directory = "YOUR_OUTPUT_DIRECTORY/"
with open(output_directory + "export_shape_to_svg_out.svg", "wb") as stream:
    shape.write_as_svg(stream)
```
The `write_as_svg` Метод эффективно преобразует фигуру в формат SVG, записывая ее непосредственно в указанный вами путь к файлу.

#### Советы по устранению неполадок
- **Ошибки пути к файлу**: Убедитесь, что пути к каталогам документов и выходных данных определены правильно.
- **Проблемы с доступом к форме**: Дважды проверьте индексы слайдов и позиции фигур, если доступ невозможен.

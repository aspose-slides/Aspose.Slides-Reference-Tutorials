## Руководство по внедрению: добавление направляющих чертежей
### Понимание руководств по рисованию
Направляющие для рисования помогают точно выровнять объекты на слайде. Они могут быть вертикальными или горизонтальными, обеспечивая единообразный дизайн на нескольких слайдах.
#### Шаг 1: Создайте новую презентацию
Инициализируйте объект представления в менеджере контекста:
```python
def add_drawing_guides():
    with slides.Presentation() as pres:
        # Здесь обрабатывается получение размера слайда.
```
#### Шаг 2: Доступ к коллекции руководств по размерам слайдов и рисованию
Определите размеры текущего слайда для точного размещения направляющих:
```python
slide_size = pres.slide_size.size
guides = pres.view_properties.slide_view_properties.drawing_guides
```
#### Шаг 3: Добавьте вертикальные и горизонтальные направляющие
Добавьте вертикальную направляющую справа от центра и горизонтальную направляющую под центром с указанными смещениями:
```python
# Добавление вертикальной направляющей
guides.add(slides.Orientation.VERTICAL, slide_size.width / 2 + 12.5)

# Добавление горизонтальной направляющей
guides.add(slides.Orientation.HORIZONTAL, slide_size.height / 2 + 12.5)
```
- **Объяснение параметров**: 
  - `Orientation` определяет направление направляющей.
  - Второй параметр — это позиция со смещением для точности.
#### Шаг 4: Сохраните презентацию
Сохраните презентацию, чтобы сохранить все изменения:
```python
pres.save("YOUR_OUTPUT_DIRECTORY/GuidesProperties-out.pptx", slides.export.SaveFormat.PPTX)
```

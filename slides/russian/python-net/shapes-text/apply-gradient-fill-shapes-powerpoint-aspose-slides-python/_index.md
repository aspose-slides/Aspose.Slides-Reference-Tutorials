## Руководство по внедрению

В этом разделе описываются шаги по добавлению градиентной заливки к эллиптической форме.

### Шаг 1: Создание экземпляра класса представления

Создайте экземпляр `Presentation` сорт:

```python
with slides.Presentation() as pres:
    # Операции со слайдами идут здесь
```

Это обеспечивает эффективное управление ресурсами.

### Шаг 2: Откройте или создайте слайд

Откройте первый слайд и создайте его при необходимости:

```python
slide = pres.slides[0]
```

### Шаг 3: Добавьте эллиптическую форму

Добавьте к слайду форму эллипса:

```python
shape = slide.shapes.add_auto_shape(slides.ShapeType.ELLIPSE, 50, 150, 75, 150)
```

- `ShapeType.ELLIPSE` определяет тип формы.
- Параметры (50, 150, 75, 150) определяют положение и размер эллипса.

### Шаг 4: Примените градиентную заливку к фигуре

Настройте градиентную заливку:

```python
shape.fill_format.fill_type = slides.FillType.GRADIENT
shape.fill_format.gradient_format.gradient_shape = slides.GradientShape.LINEAR
shape.fill_format.gradient_format.gradient_direction = slides.GradientDirection.FROM_CORNER2
```

- **Тип заполнения**: Установить на `GRADIENT`.
- **Форма и направление градиента**: Они определяют стиль и направление градиентной заливки.

### Шаг 5: Добавьте остановки градиента

Определите две точки градиента для перехода цвета:

```python
shape.fill_format.gradient_format.gradient_stops.add(1.0, slides.PresetColor.PURPLE)
shape.fill_format.gradient_format.gradient_stops.add(0, slides.PresetColor.RED)
```

- `1.0` и `0` — это положения точек остановки градиента.
- `PresetColor.PURPLE` и `PresetColor.RED` определите цвета.

### Шаг 6: Сохраните презентацию

Сохраните измененную презентацию:

```python
pres.save(global_opts.out_dir + "shapes_fill_gradient_out.pptx", slides.export.SaveFormat.PPTX)
```

Это запишет ваши изменения в новый файл с именем `shapes_fill_gradient_out.pptx`.

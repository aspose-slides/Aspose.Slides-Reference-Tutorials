### Обзор

Извлечение идентификаторов фигур имеет важное значение, когда вам нужно автоматизировать изменения PowerPoint или выполнить определенные действия на основе данных фигур. Библиотека Aspose.Slides обеспечивает бесперебойный доступ к этим свойствам.

### Пошаговая реализация

#### Доступ к презентации

Сначала откроем ваш файл PowerPoint:

```python
input_document_path = 'YOUR_DOCUMENT_DIRECTORY/welcome-to-powerpoint.pptx'

with slides.Presentation(input_document_path) as presentation:
    # Здесь будет располагаться ваш код для доступа к фигурам.
```

Этот фрагмент открывает файл PowerPoint и подготавливает его к обработке.

#### Доступ к формам слайдов

Теперь получите доступ к слайду и его формам:

```python
slide = presentation.slides[0]  # Получить первый слайд
shape = slide.shapes[0]          # Получите первую форму с этого слайда
```

Получив доступ `presentation.slides`, вы можете перебирать слайды в презентации. Аналогично, `slide.shapes` позволяет взаимодействовать с каждой фигурой на слайде.

#### Извлечение идентификатора формы

Наконец, извлеките и распечатайте идентификатор формы взаимодействия Office:

```python
shape_id = shape.office_interop_shape_id  # Извлечь идентификатор формы
print(str(shape_id))                      # Распечатайте это
```

### Объяснение параметров и методов

- **`presentation.slides[0]`:** Доступ к первому слайду.
- **`slide.shapes[0]`:** Извлекает первую фигуру из текущего слайда.
- **`shape.office_interop_shape_id`:** Свойство, которое предоставляет вам идентификатор взаимодействия с Office для фигуры.

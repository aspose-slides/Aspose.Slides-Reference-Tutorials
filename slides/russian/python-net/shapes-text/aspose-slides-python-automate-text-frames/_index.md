### Функция 1: Автоматический подбор размера текстовой рамки на слайде

#### Обзор
Эта функция демонстрирует, как задать тип автоподбора для текстовой рамки внутри автофигуры на слайде, гарантируя идеальную подгонку текста без ручной корректировки.

#### Пошаговая реализация

##### Добавить автофигуру и задать тип автоподбора
```python
import aspose.slides as slides

def set_autofit_of_text_frame():
    with slides.Presentation() as presentation:
        # Доступ к первому слайду
        slide = presentation.slides[0]

        # Добавьте на слайд автофигуру в форме прямоугольника
        auto_shape = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 150, 75, 350, 350)

        # Установить тип автоподбора для текстовой рамки
        text_frame = auto_shape.text_frame
        text_frame.text_frame_format.autofit_type = slides.TextAutofitType.SHAPE

        # Добавить текст в абзац внутри текстовой рамки
        para = text_frame.paragraphs[0]
        portion = para.portions[0]
        portion.text = "A quick brown fox jumps over the lazy dog."

        # Установить формат заливки текста на сплошной черный цвет
        portion.portion_format.fill_format.fill_type = slides.FillType.SOLID
        portion.portion_format.fill_format.solid_fill_color.color = drawing.Color.black

        # Сохранить презентацию
        presentation.save("text_format_text_out.pptx", slides.export.SaveFormat.PPTX)
```
- **Объяснение параметров**:
  - `ShapeType.RECTANGLE`: Определяет тип формы автофигуры.
  - `150, 75, 350, 350`Координаты X, Y, а также ширина и высота для позиционирования фигуры.
  - `slides.TextAutofitType.SHAPE`: Автоматически подгоняет текст под форму.

### Функция 2: Создание и настройка AutoShape

#### Обзор
Эта функция поможет вам добавить автофигуру на слайд и настроить ее внешний вид, задав типы заливки или цвета.

#### Пошаговая реализация

##### Добавить и настроить автофигуру
```python
def create_and_customize_auto_shape():
    with slides.Presentation() as presentation:
        # Доступ к первому слайду
        slide = presentation.slides[0]

        # Добавьте на слайд автофигуру в форме прямоугольника
        auto_shape = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 150, 75, 350, 350)

        # Не устанавливать заливку для фона фигуры
        auto_shape.fill_format.fill_type = slides.FillType.NO_FILL

        # Добавить текстовое содержимое в автофигуру
        text_frame = auto_shape.text_frame
        para = text_frame.paragraphs[0]
        portion = para.portions[0]
        portion.text = "A quick brown fox jumps over the lazy dog."

        # Сохранить презентацию
        presentation.save("auto_shape_out.pptx", slides.export.SaveFormat.PPTX)
```
- **Объяснение**:
  - `FillType.NO_FILL`: Гарантирует, что к фигуре не будет применена фоновая заливка.

## Руководство по внедрению

### Функция: Подсчет строк в абзацах

Эта функция позволяет вам определить, сколько строк занимает ваш текст в автофигуре, предоставляя информацию для динамической корректировки содержимого.

#### Шаг 1: Создание нового экземпляра презентации

Начните с создания нового экземпляра презентации:
```python
import aspose.slides as slides

with slides.Presentation() as presentation:
    slide = presentation.slides[0]
```

#### Шаг 2: Добавьте автофигуру на слайд

Добавьте к слайду прямоугольную форму и задайте начальные размеры:
```python
auto_shape = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 150, 75, 150, 50)
```

#### Шаг 3: Доступ к тексту в абзаце и его настройка

Откройте первый абзац и задайте его текстовое содержимое:
```python
para = auto_shape.text_frame.paragraphs[0]
portion = para.portions[0]
portion.text = "Aspose Paragraph GetLinesCount() Example"
```

#### Шаг 4: Вывод количества строк

Определите, сколько строк занимает ваш текст, используя `get_lines_count()`:
```python
print("Lines Count =", para.get_lines_count())
```

#### Шаг 5: Отрегулируйте ширину фигуры и проверьте количество линий еще раз.

Изменение ширины фигуры влияет на количество линий. Вот как это настроить и проверить еще раз:
```python
auto_shape.width = 250
print("Lines Count after changing shape width =", para.get_lines_count())
```

**Совет по устранению неполадок**: Если текст не помещается, убедитесь, что размеры AutoShape соответствуют содержимому.

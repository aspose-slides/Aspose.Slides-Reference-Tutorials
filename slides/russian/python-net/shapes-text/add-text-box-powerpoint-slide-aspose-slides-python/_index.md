## Руководство по внедрению

Теперь, когда наша среда готова, давайте погрузимся в реализацию. Мы рассмотрим каждый шаг, необходимый для добавления текстового поля на слайд.

### Создайте новую презентацию и получите доступ к первому слайду

Сначала создайте экземпляр презентации и откройте ее первый слайд:

```python
def add_text_box_to_slide():
    with slides.Presentation() as pres:
        # Доступ к первому слайду
        slide = pres.slides[0]
```

**Объяснение**: `Presentation()` класс инициализирует новую презентацию. Использование `pres.slides[0]`, мы получаем доступ к первому слайду.

### Добавить прямоугольник автофигуры

Добавьте к слайду прямоугольную форму:

```python
# Добавление прямоугольной автофигуры
auto_shape = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 150, 75, 150, 50)
```

**Параметры**: `add_auto_shape` метод принимает тип фигуры и координаты положения (X, Y), а также ширину и высоту.

### Вставить текстовую рамку

Вставьте текстовую рамку в этот прямоугольник:

```python
# Добавление текстовой рамки к фигуре
auto_shape.add_text_frame(" ")
```

**Цель**: Это создаст пустой текстовый фрейм, в который вы можете добавить свой контент.

### Введите текст в текстовое поле.

Измените текст в только что созданном текстовом поле:

```python
# Доступ к тексту и его настройка
text_frame = auto_shape.text_frame
para = text_frame.paragraphs[0]
portion = para.portions[0]
portion.text = "Aspose TextBox"
```

**Объяснение**: Здесь мы получаем доступ к первому абзацу и части текстового фрейма, чтобы задать нужный нам текст.

### Сохранить презентацию

Наконец, сохраните вашу презентацию:

```python
# Сохранение презентации
pres.save("YOUR_OUTPUT_DIRECTORY/text_TextBox_out.pptx")
```

**Примечание**: Заменять `YOUR_OUTPUT_DIRECTORY` с желаемым путем к файлу.

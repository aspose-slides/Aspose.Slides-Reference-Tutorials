### Добавление фигур на слайд
Далее, давайте добавим несколько фигур на наш слайд. Этот пример включает добавление прямоугольника и объекта в форме луны.

```python
# Получить первый слайд из презентации
slide = pres.slides[0]

# Добавьте прямоугольную форму
```

### Настройка альтернативного текста для фигур
Наконец, пройдитесь по каждой фигуре на слайде и назначьте альтернативный текст. Этот шаг имеет решающее значение для доступности.

```python
# Пройдитесь по каждой фигуре на слайде и задайте альтернативный текст для автофигур
define for shape in slide.shapes:
    if isinstance(shape, slides.AutoShape):
        shape.alternative_text = "User Defined"
```

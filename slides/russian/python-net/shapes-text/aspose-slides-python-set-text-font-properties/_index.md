## Руководство по внедрению

### Настройка свойств шрифта текста (обзор функций)
В этом разделе вы узнаете, как задать различные свойства шрифта для текста на слайде в PowerPoint с помощью Aspose.Slides для Python.

#### Шаг 1: Создание презентации
Начните с создания экземпляра `Presentation` сорт:

```python
def set_text_font_properties():
    with slides.Presentation() as presentation:
        slide = presentation.slides[0]
```
**Объяснение:** Мы используем менеджер контекста (`with`для обеспечения правильного управления ресурсами, что способствует эффективному использованию памяти.

#### Шаг 2: Добавьте автофигуру
Добавьте прямоугольную форму для размещения текста на слайде:

```python
auto_shape = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 50, 50, 200, 50)
```
**Объяснение:** The `add_auto_shape` метод добавляет фигуру указанного типа и размеров. Здесь мы используем прямоугольник в позиции `(50, 50)` с шириной `200` и высота `50`.

#### Шаг 3: Настройте TextFrame
Откройте текстовый фрейм, чтобы добавить и настроить текст:

```python
tf = auto_shape.text_frame
tf.text = "Aspose TextBox"
```
**Объяснение:** The `text_frame` Атрибут позволяет вам получить доступ к содержимому фигуры или изменить его.

#### Шаг 4: Установка свойств шрифта
Применяйте различные свойства шрифта, такие как полужирный, курсив, подчеркивание и цвет:

```python
port = tf.paragraphs[0].portions[0]
# Установить имя шрифта на «Times New Roman»
port.portion_format.latin_font = slides.FontData("Times New Roman")
# Применить смелый стиль
port.portion_format.font_bold = slides.NullableBool.TRUE
# Применить курсивное начертание
port.portion_format.font_italic = slides.NullableBool.TRUE
# Подчеркните текст.
port.portion_format.font_underline = slides.TextUnderlineType.SINGLE
# Установить высоту шрифта 25 пунктов.
port.portion_format.font_height = 25
# Изменить цвет текста на синий
color = drawing.Color.blue
port.portion_format.fill_format.fill_type = slides.FillType.SOLID
port.portion_format.fill_format.solid_fill_color.color = color
```
**Объяснение:** 
- **Название шрифта**: Устанавливает семейство шрифтов.
- **Жирный и курсивный стили**: Усильте акцент, переключая эти стили.
- **Подчеркнуть**Добавляет подчеркивание одной линией для отличия.
- **Высота шрифта**: Изменяет размер текста для лучшей видимости.
- **Цвет**: Изменяет цвет текста, чтобы сделать его более заметным.

#### Шаг 5: Сохраните презентацию
Сохраните презентацию со всеми изменениями:

```python
def save_presentation(presentation, output_directory):
    presentation.save(f"{output_directory}/text_SetTextFontProperties_out.pptx", slides.export.SaveFormat.PPTX)
```
**Объяснение:** The `save` Метод записывает измененную презентацию в файл. Убедитесь, что путь указан правильно для успешного сохранения.

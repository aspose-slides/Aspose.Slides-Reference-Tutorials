## Руководство по внедрению

В этом разделе рассматриваются основные особенности реализации замены шрифтов на основе правил.

### Загрузить презентацию

**Обзор:** Начните с загрузки целевой презентации, чтобы применить замену шрифтов.

```python
import aspose.slides as slides

# Откройте презентацию из указанного вами каталога.
with slides.Presentation("YOUR_DOCUMENT_DIRECTORY/text_fonts.pptx") as presentation:
    # Продолжить определение правил замены шрифтов можно здесь.
```

### Определите исходные и конечные шрифты

**Обзор:** Укажите, какие шрифты вы хотите заменить в случае проблем с доступностью.

```python
# Определите исходный шрифт, который необходимо заменить.
source_font = slides.FontData("SomeRareFont")

# Укажите целевой шрифт для замены.
dest_font = slides.FontData("Arial")
```

### Создать правило замены шрифта

**Обзор:** Настройте правило замены шрифтов, когда источник недоступен.

```python
# Создайте правило подстановки, используя условие WHEN_INACCESSIBLE.
font_subst_rule = slides.FontSubstRule(source_font, dest_font, slides.FontSubstCondition.WHEN_INACCESSIBLE)
```

### Добавить правила в диспетчер шрифтов

**Обзор:** Управляйте и применяйте свои правила с помощью менеджера шрифтов презентации.

```python
# Инициализируйте коллекцию правил подстановки.
font_subst_rule_collection = slides.FontSubstRuleCollection()

# Добавьте свое правило в коллекцию.
font_subst_rule_collection.add(font_subst_rule)

# Назначьте список правил менеджеру шрифтов в презентации.
presentation.fonts_manager.font_subst_rule_list = font_subst_rule_collection
```

### Извлечь и сохранить изображение со слайда

**Обзор:** Продемонстрируйте функциональность, извлекая изображение из слайда.

```python
# Извлеките изображение из первого слайда для демонстрационных целей.
img = presentation.slides[0].get_image(1, 1)

# Сохраните извлеченное изображение в указанном вами выходном каталоге в формате JPEG.
img.save("YOUR_OUTPUT_DIRECTORY/text_rule_based_font_replacement_out.jpg", slides.ImageFormat.JPEG)
```

**Советы по устранению неполадок:** При настройке исходных и целевых шрифтов убедитесь, что пути указаны правильно и шрифты существуют в вашей системе.

## Раздел часто задаваемых вопросов

1. **Что такое замена шрифтов на основе правил?**
   - Он позволяет указать резервные шрифты на случай, если исходные шрифты недоступны, обеспечивая единообразное форматирование.
2. **Как установить Aspose.Slides для Python?**
   - Используйте пип: `pip install aspose.slides`.
3. **Можно ли заменить несколько шрифтов за один раз?**
   - Да, создать и добавить несколько `FontSubstRule` объекты для вашей коллекции правил.
4. **Что произойдет, если целевой шрифт также недоступен?**
   - Если ни исходный, ни целевой шрифты недоступны, Aspose.Slides будет использовать системный шрифт по умолчанию.
5. **Существует ли ограничение на количество правил замены, которые я могу создать?**
   - Явного ограничения нет, но чрезмерное количество сложных правил может повлиять на производительность.

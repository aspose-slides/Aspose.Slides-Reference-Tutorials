## Руководство по внедрению

В этом разделе мы настроим свойства текста с помощью Aspose.Slides для Python.

### Доступ к текстовым фреймам и их изменение

#### Обзор:
Мы продемонстрируем изменение свойства 'keep_text_flat' в текстовых фреймах на слайдах PowerPoint. Эта функция контролирует, сохраняет ли текст свое исходное форматирование или становится плоским для более простого отображения.

#### Пошаговая реализация:

**1. Загрузите презентацию:**
Начните с загрузки файла презентации с помощью Aspose.Slides.

```python
pres = slides.Presentation('YOUR_DOCUMENT_DIRECTORY/text_keep_text_flat.pptx')
```
Заменять `'YOUR_DOCUMENT_DIRECTORY'` на фактический путь к вашему файлу PowerPoint.

**2. Доступ к текстовым рамкам в фигурах:**
Доступ к определенным фигурам на слайде и их текстовым рамкам:

```python
shape1 = pres.slides[0].shapes[0]
shape2 = pres.slides[0].shapes[1]
```
В демонстрационных целях мы используем первые две фигуры на первом слайде.

**3. Измените свойство «Сохранять текст плоским»:**
Настройте это свойство для управления форматированием текста:

```python
# Отключить формат простого текста для фигуры 1
disabled_flat_text = False
shape1.text_frame.text_frame_format.keep_text_flat = disabled_flat_text

# Включить формат простого текста для формы 2
enabled_flat_text = True
shape2.text_frame.text_frame_format.keep_text_flat = enabled_flat_text
```
- `keep_text_flat=False` позволяет выполнять сложное форматирование текста.
- `keep_text_flat=True` упрощает текст до базового стиля.

**4. Сохраните и экспортируйте слайд:**
Наконец, сохраните изменения, экспортировав слайд:

```python
pres.slides[0].get_image(4 / 3, 4 / 3).save('YOUR_OUTPUT_DIRECTORY/text_keep_text_flat_out.png', slides.ImageFormat.PNG)
```
Гарантировать `'YOUR_OUTPUT_DIRECTORY'` указывается место, где вы хотите сохранить выходное изображение.

### Советы по устранению неполадок:
- Проверьте пути к входным и выходным файлам.
- Убедитесь, что библиотека Aspose.Slides установлена правильно.

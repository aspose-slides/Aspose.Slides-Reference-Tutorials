## Руководство по внедрению

Чтобы скрыть фигуры в PowerPoint с помощью альтернативного текста, выполните следующие действия:

### Шаг 1: Загрузите или создайте презентацию

Начните с загрузки существующей презентации или создания новой:

```python
import aspose.slides as slides

# Создать новый экземпляр презентации
total_shapes = []
with slides.Presentation() as pres:
    # Перейти к следующему шагу
```

### Шаг 2: Откройте первый слайд и добавьте фигуры.

Откройте первый слайд и добавьте фигуры для демонстрации:

```python
# Получить первый слайд
slide = pres.slides[0]

# Добавьте прямоугольную форму
total_shapes.append(shape1 = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 50, 40, 150, 50))

# Добавьте форму луны.
total_shapes.append(shape2 = slide.shapes.add_auto_shape(slides.ShapeType.MOON, 160, 40, 150, 50))
```

### Шаг 3: Установите альтернативный текст

Назначьте альтернативный текст фигурам для идентификации:

```python
# Назначить альтернативный текст
total_shapes[0].alternative_text = "User Defined"
total_shapes[1].alternative_text = "Do Not Hide"
```

### Шаг 4: Повторите и скройте фигуры

Пройдитесь по каждой фигуре, скрывая те, у которых есть соответствующий альтернативный текст:

```python
# Определите целевой альтернативный текст
target_alt_text = "User Defined"

# Перебрать все фигуры, чтобы найти соответствующий альтернативный текст.
total_shapes_to_hide = []
for shape in slide.shapes:
    if hasattr(shape, 'alternative_text') and shape.alternative_text == target_alt_text:
        # Скрыть форму
        shape.hidden = True
        total_shapes_to_hide.append(shape)
```

### Шаг 5: Сохраните презентацию

Сохраните измененную презентацию по допустимому пути вывода:

```python
# Сохранить презентацию
total_hidden_count = len(total_shapes_to_hide)
pres.save("YOUR_OUTPUT_DIRECTORY/shapes_hide_shape_out.pptx", slides.export.SaveFormat.PPTX)
```

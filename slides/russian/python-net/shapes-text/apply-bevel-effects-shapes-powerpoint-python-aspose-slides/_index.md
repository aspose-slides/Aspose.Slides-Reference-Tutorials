## Руководство по внедрению
### Добавление фигур и настройка свойств
#### Обзор
Мы добавим к нашему слайду форму эллипса, настроим его заливку и свойства линий, а также применим эффект 3D-скоса для придания ему изысканного вида.

#### Добавить форму эллипса
Сначала добавьте базовую форму эллипса:
```python
# Доступ к первому слайду презентации
slide = pres.slides[0]

# Добавьте к слайду форму эллипса.
shape = slide.shapes.add_auto_shape(
    slides.ShapeType.ELLIPSE, 30, 30, 100, 100
)
```
Этот код создает простой эллипс с координатами (30,30) и размерами 100x100.

#### Установить свойства заливки и линии
Далее определим цвет заливки и свойства линии для нашей фигуры:
```python
# Установите тип заливки «сплошной» и выберите зеленый цвет.
drawing.Color.green
shape.fill_format.fill_type = slides.FillType.SOLID
shape.fill_format.solid_fill_color.color = Color.green

# Определите формат линии с помощью сплошной заливки оранжевого цвета и задайте ее ширину.
type: solid
fill_format = shape.line_format.fill_format
fill_format.fill_type = slides.FillType.SOLID
fill_format.solid_fill_color.color = Color.orange
shape.line_format.width = 2.0
```
Эти настройки выделяют наш эллипс на слайде.

#### Применить 3D-эффекты скоса
Последний шаг — применение эффекта скоса для добавления глубины:
```python
# Настройте 3D-формат фигуры и примените эффект круговой фаски.
type: circle
shape.three_d_format.depth = 4
shape.three_d_format.bevel_top.bevel_type = slides.BevelPresetType.CIRCLE
shape.three_d_format.bevel_top.height = 6
shape.three_d_format.bevel_top.width = 6

# Настройте камеру и освещение для реалистичного эффекта
type: orthographic_front
camera = shape.three_d_format.camera
camera.camera_type = slides.CameraPresetType.ORTHOGRAPHIC_FRONT
light_rig = shape.three_d_format.light_rig
light_rig.light_type = slides.LightRigPresetType.THREE_PT
light_rig.direction = slides.LightingDirection.TOP
```
Эти конфигурации создают визуально привлекательный 3D-эффект, улучшая эстетику презентации.

#### Сохраните вашу презентацию
Наконец, сохраните изменения:

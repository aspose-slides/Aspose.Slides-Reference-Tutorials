После установки и лицензирования инициализируйте среду Aspose.Slides, чтобы начать создавать или изменять презентации. Вот базовая настройка:

```python
import aspose.slides as slides

# Создайте экземпляр класса Presentation, представляющий файл PowerPoint.
class FontCustomizationTutorial:
    def __init__(self):
        self.pres = slides.Presentation()
    
    def add_rectangle_shape(self):
        slide = self.pres.slides[0]
        auto_shape = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 50, 50, 200, 50)
        return auto_shape
```

## Руководство по внедрению

### Добавление фигур и настройка свойств шрифта в слайдах PowerPoint

#### Обзор
В этом разделе вы узнаете, как добавить прямоугольную форму на слайд и настроить свойства ее шрифта с помощью Aspose.Slides для Python.

**1. Создание экземпляра класса представления**
Начните с создания экземпляра `Presentation` класс, который служит отправной точкой для работы с файлами PowerPoint.

```python
class FontCustomizationTutorial:
    def __init__(self):
        self.pres = slides.Presentation()

# Добавьте прямоугольную форму и задайте свойства шрифта.
def customize_font(self):
    auto_shape = self.add_rectangle_shape()
    tf = auto_shape.text_frame
    tf.text = "Aspose TextBox"
    port = tf.paragraphs[0].portions[0]
```

**2. Настройте свойства шрифта**
Настройте различные свойства шрифта, такие как гарнитура, жирность, курсив, подчеркивание, размер и цвет текста внутри фигуры.
- **Установить семейство шрифтов:**
  
  ```python
  port.portion_format.latin_font = slides.FontData("Times New Roman")
  ```

- **Свойства, выделенные жирным шрифтом и курсивом:**

  ```python
  port.portion_format.font_bold = slides.NullableBool.TRUE
  port.portion_format.font_italic = slides.NullableBool.TRUE
  ```

- **Подчеркнуть текст:**

  ```python
  port.portion_format.font_underline = slides.TextUnderlineType.SINGLE
  ```

- **Установить размер и цвет шрифта:**

  ```python
  port.portion_format.font_height = 25
  port.portion_format.fill_format.fill_type = slides.FillType.SOLID
  port.portion_format.fill_format.solid_fill_color.color = drawing.Color.blue
  ```

**3. Сохраните презентацию**
Наконец, сохраните измененную презентацию в нужном каталоге.

```python
self.pres.save("YOUR_OUTPUT_DIRECTORY/text_font_family_out.pptx", slides.export.SaveFormat.PPTX)
```

## Руководство по внедрению

Теперь, когда вы все настроили, давайте приступим к обозначению фигуры как декоративной.

### Создание презентации и добавление фигуры

#### Обзор

Начнем с открытия (или создания) презентации, добавления автоматической фигуры (например, прямоугольника) и отметки ее как декоративной.

#### Шаг 1: Откройте или создайте новую презентацию
```python
with slides.Presentation() as pres:
    # Доступ к первому слайду презентации
    first_slide = pres.slides[0]
```
**Объяснение**: Этот код инициализирует новый объект презентации, автоматически создавая начальный слайд, с которым мы можем работать.

#### Шаг 2: Добавьте автофигуру на слайд
```python
rectangle_shape = first_slide.shapes.add_auto_shape(
    slides.ShapeType.RECTANGLE, 10, 10, 100, 100
)
```
**Параметры**: `ShapeType` указывает тип фигуры, а следующие четыре числа определяют ее положение (x, y) и размер (ширина, высота).

#### Шаг 3: Установите форму как декоративную
```python
rectangle_shape.is_decorative = True
```
**Цель**: Эта линия отмечает прямоугольник как декоративный, указывая, что его следует сохранить, но не изменять его размер или положение путем автоматической корректировки макета.

### Сохранение презентации

После выделения формы сохраните презентацию:
```python
pres.save('YOUR_OUTPUT_DIRECTORY/DecorativeDemo.pptx', slides.export.SaveFormat.PPTX)
```
**Объяснение**: Это сохраняет текущее состояние вашей презентации по указанному пути с помощью `.pptx` формат.

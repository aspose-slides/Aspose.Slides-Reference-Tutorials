## Руководство по внедрению
Мы разберем добавление гиперссылок на слайды PowerPoint по шагам.

### Добавление автофигуры и текстовой рамки
Во-первых, нам нужна фигура на нашем слайде для текста. Вот как ее добавить:

#### Шаг 1: Создание объекта презентации
```python
with slides.Presentation() as presentation:
    # Ваш код будет здесь
```
Это инициализирует новую презентацию PowerPoint.

#### Шаг 2: Добавьте автофигуру
Добавьте прямоугольник с текстом:
```python
shape1 = presentation.slides[0].shapes.add_auto_shape(
    slides.ShapeType.RECTANGLE, 100, 100, 600, 50, False)
```
Параметры включают положение и размер фигуры.

#### Шаг 3: Добавьте текст к фигуре
Вставьте нужный текст в форму:
```python
shape1.add_text_frame("Aspose: File Format APIs")
```

### Установка гиперссылки на текст
Теперь сделайте этот текст кликабельным, добавив гиперссылку.

#### Шаг 4: Назначьте гиперссылку
Свяжите текст с URL-адресом:
```python
shape1.text_frame.paragraphs[0].portions[0].portion_format.hyperlink_click = slides.Hyperlink(
    "https://www.aspose.com/")
```
Этот фрагмент кода превращает первую часть первого абзаца в гиперссылку.

#### Шаг 5: Добавьте подсказку для гиперссылки
Предоставьте дополнительную информацию с помощью подсказки:
```python
shape1.text_frame.paragraphs[0].portions[0].portion_format.hyperlink_click.tooltip = \\
    "More than 70% Fortune 100 companies trust Aspose APIs"
```

### Настройка внешнего вида текста
Измените внешний вид, чтобы сделать его более заметным.

#### Шаг 6: Установите размер шрифта
Увеличьте размер шрифта для лучшей видимости:
```python
shape1.text_frame.paragraphs[0].portions[0].portion_format.font_height = 32
```

### Сохранение презентации
Наконец, сохраните презентацию со всеми примененными изменениями.
```python\presentation.save("YOUR_OUTPUT_DIRECTORY/hyperlink_add_hyperlink_out.pptx")
```
Заменять `YOUR_OUTPUT_DIRECTORY` на фактический путь, по которому вы хотите сохранить файл.

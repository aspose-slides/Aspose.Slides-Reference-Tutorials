## Руководство по внедрению

### Добавление кластеризованной столбчатой диаграммы и поворот текстовых фреймов

В этом разделе вы узнаете, как добавить кластеризованную столбчатую диаграмму в презентацию и настроить пользовательские углы поворота для текстовых фреймов в этой диаграмме.

#### Шаг 1: Создание экземпляра класса представления

Начните с создания `Presentation` объект с использованием менеджера контекста, обеспечивающего автоматическое управление ресурсами:

```python
import aspose.slides as slides

def rotate_text_frame():
    # Используйте контекстный менеджер для автоматической обработки ресурсов
    with slides.Presentation() as presentation:
        pass  # Заполнитель для последующих шагов
```

#### Шаг 2: Добавьте кластеризованную столбчатую диаграмму

Добавьте кластеризованную столбчатую диаграмму на первый слайд в позицию (50, 50) с указанными размерами:

```python
# Добавить диаграмму на первый слайд
class ChartType:
    CLUSTERED_COLUMN = 'ClusteredColumn'
chart = presentation.slides[0].shapes.add_chart(
    ChartType.CLUSTERED_COLUMN, 50, 50, 500, 300
)
```

#### Шаг 3: Доступ к серии диаграмм и настройка меток

Получите доступ к первой серии данных диаграммы, чтобы управлять ее метками:

```python
# Доступ к первой серии
class DataLabelFormatType:
    SHOW_VALUE = 'ShowValue'
series = chart.chart_data.series[0]

# Отображение значений на этикетках
series.labels.default_data_label_format.show_value = True
```

#### Шаг 4: Установите пользовательский угол поворота для формата текстового блока

Задайте пользовательский угол поворота для формата текстового блока, чтобы сделать ваши данные более визуально привлекательными:

```python
# Установить пользовательский угол поворота
class TextBlockFormatType:
    ROTATION_ANGLE = 'RotationAngle'
series.labels.default_data_label_format.text_format.text_block_format.rotation_angle = 65
```

#### Шаг 5: Добавьте и поверните заголовок диаграммы

Добавьте заголовок к диаграмме и примените пользовательский угол поворота для улучшения внешнего вида:

```python
# Добавить и повернуть заголовок диаграммы
class TextFrameFormatType:
    ROTATION_ANGLE = 'RotationAngle'
chart.has_title = True
chart.chart_title.add_text_frame_for_overriding("Custom Title").text_frame_format.rotation_angle = -30
```

#### Шаг 6: Сохраните презентацию

Наконец, сохраните презентацию в выходном каталоге:

```python
# Сохранить презентацию
class SaveFormatType:
    PPTX = 'Pptx'
presentation.save(
    "YOUR_OUTPUT_DIRECTORY/text_textframe_rotation_out.pptx",
    SaveFormatType.PPTX
)
```

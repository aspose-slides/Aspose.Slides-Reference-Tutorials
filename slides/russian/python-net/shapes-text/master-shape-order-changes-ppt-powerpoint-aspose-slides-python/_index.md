После установки инициализируйте Aspose.Slides в своем скрипте:

import aspose.slides as slides

# Инициализировать презентацию
presentation = slides.Presentation()

## Руководство по внедрению

Давайте разобьем процесс изменения порядка фигур на управляемые шаги.

### Шаг 1: Загрузите презентацию

Начните с загрузки существующего файла PowerPoint. Предположим, у вас есть файл с именем `welcome-to-powerpoint.pptx`:

```python
# Загрузить презентацию
data_dir = 'YOUR_DOCUMENT_DIRECTORY/'
with slides.Presentation(data_dir + 'welcome-to-powerpoint.pptx') as presentation:
    # Доступ к первому слайду
    slide = presentation.slides[0]
```

### Шаг 2: Добавьте и настройте фигуры

#### Добавление прямоугольной формы

Добавьте прямоугольник на слайд и настройте его свойства:

```python
# Добавьте прямоугольную форму
rectangle = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 200, 365, 400, 150)
rectangle.fill_format.fill_type = slides.FillType.NO_FILL
rectangle.add_text_frame('')
```

#### Вставьте текст в прямоугольник

Вставьте текст, чтобы персонализировать вашу форму:

```python
# Добавить текст в прямоугольник
text_frame = rectangle.text_frame
para = text_frame.paragraphs[0]
portion = para.portions[0]
portion.text = 'Watermark Text Watermark Text Watermark Text'
```

### Шаг 3: Добавьте треугольную форму

Далее добавьте еще одну фигуру — треугольник:

```python
# Добавьте треугольную форму
triangle = slide.shapes.add_auto_shape(slides.ShapeType.TRIANGLE, 200, 365, 400, 150)
```

### Шаг 4: Измените порядок фигур

Измените порядок фигур, переместив треугольник перед другими:

```python
# Переместить треугольник вперед
slide.shapes.reorder(2, triangle)
```

### Шаг 5: Сохраните измененную презентацию.

Наконец, сохраните изменения в новом файле:

```python
# Сохранить презентацию
output_dir = 'YOUR_OUTPUT_DIRECTORY/'
presentation.save(output_dir + 'shapes_reorder_out.pptx', slides.export.SaveFormat.PPTX)
```

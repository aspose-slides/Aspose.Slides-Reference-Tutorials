## Руководство по внедрению

Теперь, когда все готово, давайте шаг за шагом реализуем вращение фигуры:

### Добавление и поворот фигур в PowerPoint

#### Обзор
В этом разделе рассматривается добавление прямоугольной фигуры на слайд и поворот ее на 90 градусов.

#### Пошаговая реализация

##### Инициализировать презентацию

Начните с создания экземпляра `Presentation` класс, представляющий ваш файл PPTX:
```python
with slides.Presentation() as pres:
    # Мы будем работать с этим менеджером контекста для эффективного управления ресурсами.
```

##### Доступ к слайду и добавление фигуры

Откройте первый слайд презентации и добавьте прямоугольную форму:
```python
slide = pres.slides[0]

shape = slide.shapes.add_auto_shape(
    slides.ShapeType.RECTANGLE, 50, 150, 75, 150)
# Параметры определяют положение (x, y) и размер (ширина, высота).
```

##### Повернуть фигуру

Поверните вновь добавленную фигуру, задав ее свойство вращения:
```python
shape.rotation = 90
# Поворот задается в градусах.
```

##### Сохранить презентацию

Наконец, сохраните изменения в указанном выходном каталоге:
```python
pres.save("YOUR_OUTPUT_DIRECTORY/shapes_rotate_out.pptx", slides.export.SaveFormat.PPTX)
# Убедитесь, что путь существует, или измените его соответствующим образом.
```

#### Советы по устранению неполадок
- **Форма не отображается**: Проверьте параметры положения и размера. Если значения выходят за пределы экрана, отрегулируйте их.
- **Проблемы ротации**: Убедитесь, что `shape.rotation` настроен правильно; убедитесь в отсутствии конфликтующих преобразований.

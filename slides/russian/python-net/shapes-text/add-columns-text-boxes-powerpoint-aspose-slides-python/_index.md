## Руководство по внедрению

В этом разделе рассматривается добавление столбцов в текстовые поля на слайдах PowerPoint.

### Обзор функции «Добавить столбец»

Эта функция аккуратно организует большие объемы текста, разделяя его на несколько столбцов в одном текстовом поле, что повышает читабельность и сохраняет четкий дизайн слайда.

#### Пошаговая реализация

**1. Создайте новую презентацию**

Начните с создания экземпляра презентации PowerPoint:

```python
with slides.Presentation() as presentation:
    # Доступ к первому слайду презентации
    slide = presentation.slides[0]
```

**2. Добавить автофигуру на слайд**

Добавьте прямоугольник, который будет служить текстовым контейнером:

```python
# Добавьте фигуру прямоугольника в положение (100, 100) размером (300x300)
shape = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 100, 100, 300, 300)
```

**3. Вставьте текстовую рамку в форму**

Вставьте текстовое содержимое в только что созданную прямоугольную форму:

```python
# Добавьте текстовую рамку в прямоугольник с нужным вам текстом.
text = ("All these columns are limited to be within a single text container -- " +
         "you can add or delete text and the new or remaining text automatically adjusts " +
         "itself to flow within the container. You cannot have text flow from one container " +
         "to other though -- we told you PowerPoint's column options for text are limited!")
shape.add_text_frame(text)
```

**4. Настройте столбцы в текстовом фрейме**

Определите количество столбцов и интервал:

```python
# Доступ и настройка формата текстового фрейма
text_frame_format = shape.text_frame.text_frame_format

# Установите количество столбцов равным 3 и определите интервал между столбцами равным 10 пунктам.
text_frame_format.column_count = 3
text_frame_format.column_spacing = 10
```

**5. Сохраните презентацию**

Наконец, сохраните презентацию с примененными изменениями:

```python
presentation.save("YOUR_OUTPUT_DIRECTORY/text_add_text_frame_out.pptx", slides.export.SaveFormat.PPTX)
```

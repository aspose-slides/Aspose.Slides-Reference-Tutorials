## Руководство по внедрению

В этом разделе мы подробно рассмотрим процесс создания групповой фигуры на слайде презентации.

### Создание групповых фигур на слайдах презентации

Эта функция помогает организовать несколько фигур в единое целое для лучшей структуры и визуальной привлекательности.

#### Шаг 1: Создайте или откройте презентацию

Начните с открытия существующей презентации или создания новой:

```python
def create_group_shape():
    with slides.Presentation() as pres:
        slide = pres.slides[0]
```

*Почему:* Мы используем `with` заявление об управлении контекстом, обеспечивающее надлежащую очистку ресурсов после операций.

#### Шаг 2: Доступ к коллекции фигур

Получите доступ к фигурам на текущем слайде:

```python
shapes = slide.shapes
```

Эта коллекция позволяет нам манипулировать и добавлять новые формы.

#### Шаг 3: Добавьте групповую форму

Добавьте групповую фигуру для размещения отдельных фигур:

```python
group_shape = shapes.add_group_shape()
```

*Почему:* Группировка фигур упрощает манипуляции, позволяя перемещать или изменять их как единое целое.

#### Шаг 4: Вставьте отдельные фигуры

Добавьте прямоугольники в форму группы в указанных местах:

```python
group_shape.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 300, 100, 100, 100)
group_shape.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 500, 100, 100, 100)
group_shape.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 300, 300, 100, 100)
group_shape.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 500, 300, 100, 100)
```

*Почему:* Этот шаг включает добавление фигур для демонстрации возможностей группировки.

#### Шаг 5: Добавьте рамку

Установите рамку вокруг фигуры группы для визуального обозначения:

```python
group_shape.frame = slides.ShapeFrame(
    100, 300, 500, 40,
    slides.NullableBool.TRUE,
    slides.NullableBool.TRUE,
    0
)
```

#### Шаг 6: Сохраните презентацию

Наконец, сохраните презентацию в указанном каталоге:

```python
pres.save("YOUR_OUTPUT_DIRECTORY/shapes_create_group_shape_out.pptx", slides.export.SaveFormat.PPTX)
```

*Почему:* Сохранение гарантирует, что все изменения будут сохранены и к ним можно будет получить доступ в дальнейшем.

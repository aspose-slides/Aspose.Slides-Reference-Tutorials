## Руководство по внедрению
Давайте погрузимся в создание пользовательских нумерованных списков. Каждый шаг разбит на части для ясности и простоты реализации.

### Добавление прямоугольной формы с текстовыми рамками
#### Обзор:
Сначала добавьте фигуру, которая будет содержать текстовые рамки для пунктов списка.

```python
# Добавьте прямоугольник к первому слайду.
shape = presentation.slides[0].shapes.add_auto_shape(
    slides.ShapeType.RECTANGLE, 200, 200, 400, 200)
```
- **Объяснение параметров**: `add_auto_shape` Метод принимает параметры для типа фигуры (прямоугольник), положения (координаты x и y) и размеров (ширина и высота).

### Настройка текстовых фреймов
#### Обзор:
Для добавления маркеров используйте текстовую рамку прямоугольника.

```python
# Доступ к текстовому фрейму созданной автофигуры
text_frame = shape.text_frame

# Удалить любой существующий абзац по умолчанию, если он есть
text_frame.paragraphs.clear()
```
- **Цель**: Обеспечивает «чистый лист» перед добавлением пользовательских маркеров.

### Добавление пользовательских пронумерованных маркеров
#### Обзор:
Добавьте абзацы с определенными настройками маркеров:

```python
# Добавьте абзацы с пользовательскими нумерованными маркерами.
for start_number, bullet_text in [(2, "bullet 2"), (3, "bullet 3"), (7, "bullet 7")]:
    paragraph = slides.Paragraph()
    paragraph.text = bullet_text
    paragraph.paragraph_format.depth = 4
    paragraph.paragraph_format.bullet.numbered_bullet_start_with = start_number
    paragraph.paragraph_format.bullet.type = slides.BulletType.NUMBERED
    text_frame.paragraphs.add(paragraph)
```
- **Конфигурация**: Каждый абзац начинается с определенного числа, что обеспечивает гибкость и контроль над форматированием презентации.

## Руководство по внедрению
Давайте разберем процесс на четкие шаги, чтобы создать прямоугольную форму и сохранить презентацию.
### Создать презентацию
Сначала создайте экземпляр `Presentation` класс. Это действует как контейнер для всех слайдов в вашей презентации:
```python
with slides.Presentation() as pres:
```
С использованием `with`, обеспечивает правильное управление ресурсами, закрывая файлы даже в случае возникновения ошибки.
### Доступ к первому слайду
Чтобы добавить фигуры, получите доступ к первому слайду:
```python
slide = pres.slides[0]
```
Этот код извлекает первый слайд из объекта презентации.
### Добавление прямоугольной формы
Теперь давайте добавим прямоугольник в определенное место с определенными размерами:
```python
# Добавить автофигуру прямоугольного типа в позицию (50, 150) с шириной 150 и высотой 50
slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 50, 150, 150, 50)
```
Здесь, `add_auto_shape` используется для добавления формы. Мы указываем тип как `RECTANGLE`, вместе с его позицией `(x=50, y=150)` и размер `(width=150, height=50)`Этот метод возвращает объект формы, который при необходимости можно настроить дополнительно.
### Сохранение презентации
Наконец, сохраните вашу презентацию:
```python
# Запишите файл PPTX на диск, используя выходной каталог-заполнитель.
pres.save("YOUR_OUTPUT_DIRECTORY/shapes_add_rectangle_out.pptx", slides.export.SaveFormat.PPTX)
```
Заменять `YOUR_OUTPUT_DIRECTORY` с желаемым путем. Метод `save` записывает измененную презентацию обратно на диск в формате PPTX.

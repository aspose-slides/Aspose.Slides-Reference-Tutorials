Это гарантирует высвобождение ресурсов после обработки.

#### Шаг 3: Проверьте наличие текстовой рамки в форме (H3)

Прежде чем получить доступ к тексту, убедитесь, что фигура содержит текстовую рамку, чтобы избежать ошибок:

```python
def get_paragraph_coordinates(shape):
    if shape.text_frame is not None:
        # Доступ к тексту здесь.
        text_frame = shape.text_frame
        paragraph = text_frame.paragraphs[0]
        rect = paragraph.get_rect()
        return rect
    else:
        raise ValueError('The selected shape does not contain a text frame.')
```

#### Шаг 4: Извлечение и возврат прямоугольных координат (H3)

Получите доступ к прямоугольным координатам первого абзаца, как показано в шаге 3.

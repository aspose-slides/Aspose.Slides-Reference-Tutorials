## Руководство по внедрению

Давайте разберем процесс редактирования геометрии фигуры с помощью `ShapeUtil`.

### Добавление и изменение фигур (шаг за шагом)

#### Шаг 1: Добавьте новую форму

Начните с добавления на слайд прямоугольной формы:

```python
import aspose.slides as slides

def edit_shape_geometry():
    with slides.Presentation() as pres:
        # Добавьте новую прямоугольную фигуру к первому слайду.
        shape = pres.slides[0].shapes.add_auto_shape(
            slides.ShapeType.RECTANGLE, 100, 100, 300, 100
        )
```

**Объяснение**: Этот фрагмент кода инициализирует презентацию и добавляет прямоугольник с указанными размерами.

#### Шаг 2: Доступ и изменение исходного геометрического контура

Измените путь новой добавленной фигуры:

```python
        # Доступ к исходным геометрическим траекториям формы
        original_path = shape.get_geometry_paths()[0]
        original_path.fill_mode = slides.PathFillModeType.NONE
```

**Объяснение**: `get_geometry_paths()` извлекает текущие пути, которые мы затем изменяем, удаляя заливку для настройки.

#### Шаг 3: Создайте новый графический контур с текстом

Создайте и настройте новый графический путь, содержащий текст:

```python
import aspose.pydrawing as drawing

        # Определите новый графический путь со встроенным текстом
        graphics_path = drawing.drawing2d.GraphicsPath()
        graphics_path.add_string(
            "Text in shape",
            drawing.FontFamily("Arial"),
            1,
            40.0,
            drawing.PointF(10, 10),
            drawing.StringFormat.generic_default
        )
```

**Объяснение**: Этот шаг создает `GraphicsPath` объект и добавляет к нему текст, используя указанный шрифт и размер.

#### Шаг 4: Преобразование графического пути в геометрический путь

Преобразуйте ваш графический путь в геометрический путь:

```python
        # Преобразовать графический контур для использования формы
        text_path = slides.util.ShapeUtil.graphics_path_to_geometry_path(graphics_path)
        text_path.fill_mode = slides.PathFillModeType.NORMAL
```

**Объяснение**: `ShapeUtil` используется здесь для преобразования `GraphicsPath` в формат, совместимый с формами слайдов.

#### Шаг 5: Объединение и установка геометрических контуров

Объедините исходные и новые пути, вернув их к форме:

```python
        # Объедините оба геометрических контура для получения окончательной конфигурации формы.
        shape.set_geometry_paths([original_path, text_path])
```

**Объяснение**: Это объединяет измененный контур с вновь созданным, чтобы обновить внешний вид фигуры.

#### Шаг 6: Сохраните презентацию

Наконец, сохраните вашу презентацию на диск:

```python
        # Вывести измененную презентацию
        pres.save("YOUR_OUTPUT_DIRECTORY/shapes_set_geometry_path_with_util_out.pptx", slides.export.SaveFormat.PPTX)
```

**Объяснение**: `save` Метод записывает изменения в указанный путь к файлу.

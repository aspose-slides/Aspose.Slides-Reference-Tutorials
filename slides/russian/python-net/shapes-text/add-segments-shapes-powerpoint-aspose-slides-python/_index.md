### Создание и настройка презентации

#### Обзор

Эта функция позволяет добавлять пользовательские сегменты линий к существующей прямоугольной форме в презентации, повышая ее визуальную привлекательность.

#### Шаг 1: Добавьте новую прямоугольную фигуру

Начните с создания нового слайда прямоугольной формы:

```python
import aspose.slides as slides

def add_segment_to_geometry_shape():
    # Создать новый экземпляр презентации
    with slides.Presentation() as pres:
        # Добавьте прямоугольник к первому слайду в указанных координатах.
        shape = pres.slides[0].shapes.add_auto_shape(
            slides.ShapeType.RECTANGLE, 100, 100, 200, 100
        )
```

#### Шаг 2: Доступ к геометрическому пути

Извлеките геометрический путь из только что созданного прямоугольника:

```python
# Получить первый геометрический путь фигуры
geometry_path = shape.get_geometry_paths()[0]
```

#### Шаг 3: Добавление сегментов линии к контуру

Добавьте сегменты линий с различной толщиной, чтобы настроить путь:

```python
# Добавьте два линейных сегмента к геометрическому контуру.
# Первый сегмент с весом 1
geometry_path.line_to(100, 50, 1)
# Второй сегмент с весом 4
geometry_path.line_to(100, 50, 4)
```

#### Шаг 4: Обновление геометрического контура фигуры

Убедитесь, что ваша форма отражает эти новые сегменты:

```python
# Обновите форму с измененным геометрическим контуром.
dshape.set_geometry_path(geometry_path)
```

#### Шаг 5: Сохраните презентацию

Наконец, сохраните изменения в файле в нужном вам каталоге:

```python
# Сохраните презентацию в выходной каталог.
pres.save("YOUR_OUTPUT_DIRECTORY/shapes_add_segment_to_geometry_path_out.pptx", slides.export.SaveFormat.PPTX)
```

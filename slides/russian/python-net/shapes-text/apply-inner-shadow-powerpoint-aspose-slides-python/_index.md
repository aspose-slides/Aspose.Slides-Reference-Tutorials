### Добавление эффекта внутренней тени
#### Обзор функции
Цель — создать визуально привлекательный текстовый блок с эффектом внутренней тени. Это улучшает читаемость и добавляет глубины содержимому слайда.

#### Пошаговая реализация
##### Шаг 1: Создание презентации
Начните с создания объекта презентации, обеспечив надлежащее управление ресурсами с помощью `with` заявление.
```python
def apply_inner_shadow():
    with slides.Presentation() as pres:
        # Перейти к следующим шагам
```

##### Шаг 2: Получите доступ к первому слайду
Найдите первый слайд, к которому вы хотите применить эффект.
```python
slide = pres.slides[0]
```

##### Шаг 3: Добавьте прямоугольную автофигуру
Добавьте автофигуру типа «Прямоугольник» для размещения текста.
```python
auto_shape = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 150, 75, 150, 50)
```
*Параметры Пояснение*: Координаты (150, 75) определяют положение; 150 и 50 определяют ширину и высоту соответственно.

##### Шаг 4: Добавьте текстовую рамку к фигуре
Создайте текстовую рамку внутри фигуры для добавления текста.
```python
auto_shape.add_text_frame(" ")
```

##### Шаг 5: Доступ к текстовому фрейму
Получить объект текстовой рамки из AutoShape.
```python
text_frame = auto_shape.text_frame
```

##### Шаг 6: Создание объекта «Абзац»
Добавьте абзац, чтобы закрепить текст в текстовой рамке.
```python
para = text_frame.paragraphs[0]
```

##### Шаг 7: Установка текстового содержимого
Используйте объект Portion, чтобы указать, какой текст должен быть в абзаце.
```python
portion = para.portions[0]
portion.text = "Aspose TextBox"
```

##### Шаг 8: Применение эффекта внутренней тени (пользовательская реализация)
Чтобы применить эффект внутренней тени, измените свойства фигуры. Вот как это можно сделать:
```python
# Предполагая, что Aspose.Slides поддерживает это напрямую или через пользовательское управление стилями.
def add_inner_shadow_effect(auto_shape):
    inner_shadow_effect = auto_shape.fill_format.effect_format
    # Установить свойства внутренней тени (это заполнитель для фактической реализации)
    inner_shadow_effect.inner_shadow.blur_radius = 4
    inner_shadow_effect.inner_shadow.distance = 3
    inner_shadow_effect.inner_shadow.color = slides.Color.black
```
*Примечание*: Что касается последних известных функций, вам может потребоваться расширить их функциональность с помощью пользовательских стилей или внешних библиотек.

##### Шаг 9: Сохраните презентацию
Наконец, сохраните презентацию со всеми изменениями.
```python
pres.save("YOUR_OUTPUT_DIRECTORY/text_add_textbox_out.pptx", slides.export.SaveFormat.PPTX)
```

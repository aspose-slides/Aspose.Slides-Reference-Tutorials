### Функция: выравнивание абзацев в PowerPoint

#### Обзор:
Выравнивание текста в презентациях не только улучшает читаемость, но и придает изысканный вид. Эта функция демонстрирует выравнивание абзацев по центру на слайдах с использованием Python.

#### Шаги:

**1. Определите пути к файлам**

Сначала задайте пути к входным и выходным файлам:

```python
input_path = "YOUR_DOCUMENT_DIRECTORY/text_paragraphs_alignment.pptx"
output_path = "YOUR_OUTPUT_DIRECTORY/text_paragraphs_alignment_out.pptx"
```

**2. Откройте презентацию и получите доступ к слайду**

Откройте существующую презентацию и получите первый слайд:

```python
with slides.Presentation(input_path) as pres:
    slide = pres.slides[0]
```

**3. Измените текстовые рамки**

Доступ к текстовым фреймам из определенных заполнителей для обновления их содержимого:

```python
tf1 = slide.shapes[0].text_frame
# Перед доступом к форме убедитесь, что у нее есть текстовая рамка.
if tf1 is not None:
    tf2 = slide.shapes[1].text_frame
    if tf2 is not None:
        tf1.text = "Center Align by Aspose"
        tf2.text = "Center Align by Aspose"
```

**4. Установите выравнивание абзаца**

Выровняйте текст по центру внутри каждого абзаца:

```python
para1 = tf1.paragraphs[0]
# Проверьте, есть ли доступные абзацы
if para1 is not None:
    para2 = tf2.paragraphs[0]
    # Перед настройкой выравнивания убедитесь, что para2 существует
    if para2 is not None:
        para1.paragraph_format.alignment = slides.TextAlignment.CENTER
        para2.paragraph_format.alignment = slides.TextAlignment.CENTER
```

**5. Сохраните изменения.**

Наконец, сохраните изменения в новом файле:

```python
pres.save(output_path, slides.export.SaveFormat.PPTX)
```

### Функция: загрузка и сохранение презентаций PowerPoint

#### Обзор:
Эта функция помогает вам загружать презентации, изменять их, добавляя текст, а затем эффективно сохранять обновленные файлы.

#### Шаги:

**1. Определите пути к файлам**

Настройте входные и выходные пути, как в предыдущем примере:

```python
input_path = "YOUR_DOCUMENT_DIRECTORY/sample_input.pptx"
output_path = "YOUR_OUTPUT_DIRECTORY/sample_output.pptx"
```

**2. Загрузите презентацию и откройте слайд**

Откройте файл презентации и откройте его первый слайд:

```python
with slides.Presentation(input_path) as pres:
    slide = pres.slides[0]
```

**3. Добавьте текст в фигуру**

Перед добавлением нового содержимого проверьте, пуста ли текстовая рамка:

```python
tf = slide.shapes[0].text_frame
# Перед доступом к свойствам проверьте значение None
if tf and not tf.text:
    tf.text = "New Text Added"
```

**4. Сохраните презентацию**

Сохраните изменения:

```python
pres.save(output_path, slides.export.SaveFormat.PPTX)
```

## Руководство по внедрению

В этом разделе вы узнаете, как добавить надстрочный и подстрочный текст на слайд.

### Создание новой презентации

Начнем с создания нового объекта презентации:

```python
def adding_superscript_and_subscript_text():
    with slides.Presentation() as presentation:
        slide = presentation.slides[0]
```

Здесь, `presentation.slides[0]` открывает первый слайд в презентации. Вы можете добавить больше слайдов по мере необходимости.

### Добавление фигур и текстовых рамок

Добавьте автофигуру для размещения вашего текста:

```python
shape = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 100, 100, 200, 100)
text_frame = shape.text_frame
text_frame.paragraphs.clear()
```

Этот фрагмент кода создает прямоугольник и очищает все существующие абзацы в текстовом фрейме.

### Добавление надстрочного текста

Чтобы добавить надстрочный текст:
1. **Создать абзац**: 
   ```python
   super_para = slides.Paragraph()
   ```
2. **Добавить обычный текст**: 
   ```python
   portion1 = slides.Portion()
   portion1.text = "SlideTitle"
   super_para.portions.add(portion1)
   ```
3. **Добавить часть надстрочного индекса**: 
   Отрегулируйте спуск, чтобы отформатировать текст как верхний индекс.
   ```python
   super_portion = slides.Portion()
   super_portion.portion_format.escapement = 30  # Позиционирование надстрочного индекса
   super_portion.text = "TM"
   super_para.portions.add(super_portion)
   ```

### Добавление нижнего текста

Аналогично для нижнего текста:
1. **Создать новый абзац**: 
   ```python
   paragraph2 = slides.Paragraph()
   ```
2. **Добавить обычный текст**: 
   ```python
   portion2 = slides.Portion()
   portion2.text = "a"
   paragraph2.portions.add(portion2)
   ```
3. **Добавить часть нижнего индекса**: 
   Отрегулируйте спуск, чтобы отформатировать текст как нижний индекс.
   ```python
   sub_portion = slides.Portion()
   sub_portion.portion_format.escapement = -25  # Позиционирование нижнего индекса
   sub_portion.text = "i"
   paragraph2.portions.add(sub_portion)
   ```

### Сохранение презентации

Наконец, добавьте абзацы в текстовый фрейм и сохраните презентацию:

```python
text_frame.paragraphs.add(super_para)
text_frame.paragraphs.add(paragraph2)

presentation.save("YOUR_OUTPUT_DIRECTORY/text_add_superscript_and_subscript_out.pptx", slides.export.SaveFormat.PPTX)
```

### Советы по устранению неполадок
- Убедитесь, что значения спуска установлены правильно для верхнего индекса (положительного) и нижнего индекса (отрицательного).
- Убедитесь, что в вашей среде установлена библиотека Aspose.Slides.

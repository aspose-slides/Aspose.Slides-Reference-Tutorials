В этом уроке мы рассмотрим, как можно использовать Aspose.Slides для Python для изменения стиля фигуры SmartArt в презентациях PowerPoint. Выполнив эти шаги, вы узнаете:

## Руководство по внедрению
Теперь давайте шаг за шагом рассмотрим процесс изменения стилей SmartArt.

### Загрузить презентацию PowerPoint
Чтобы начать изменять презентацию, загрузите существующий файл. Это достигается с помощью Aspose.Slides' `Presentation` сорт:
```python
# Загрузить существующий файл PowerPoint из указанного каталога
with slides.Presentation('YOUR_DOCUMENT_DIRECTORY/smart_art_access.pptx') as presentation:
    # Дальнейшие операции будут выполняться в этом менеджере контекста.
```

### Определите и измените фигуры SmartArt
После загрузки презентации просмотрите ее фигуры, чтобы определить те из них, которые имеют тип SmartArt:
```python
# Пройдитесь по каждой фигуре внутри первого слайда
for shape in presentation.slides[0].shapes:
    # Проверьте, относится ли фигура к типу SmartArt.
    if isinstance(shape, slides.smartart.SmartArt):
        # Доступ и проверка текущего стиля SmartArt
        if shape.quick_style == slides.smartart.SmartArtQuickStyleType.SIMPLE_FILL:
            # Измените быстрый стиль SmartArt на CARTOON
            shape.quick_style = slides.smartart.SmartArtQuickStyleType.CARTOON
```
- **Объяснение**: Мы проходим по каждой фигуре на первом слайде и проверяем, является ли она объектом SmartArt. Если ее текущий стиль — `SIMPLE_FILL`, мы меняем его на `CARTOON`.

### Сохраните измененную презентацию
Наконец, сохраните изменения в новом файле:
```python
# Сохраните измененную презентацию в указанном выходном каталоге.
presentation.save('YOUR_OUTPUT_DIRECTORY/smart_art_change_quick_style_out.pptx', slides.export.SaveFormat.PPTX)
```

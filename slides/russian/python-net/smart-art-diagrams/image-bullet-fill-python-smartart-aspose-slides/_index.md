### Создание SmartArt с заливкой маркерами изображений

#### Обзор

В этом разделе вы узнаете, как добавить фигуру SmartArt на слайд и задать изображение в качестве формата заполнения маркера.

#### Шаг 1: Создание объекта презентации

Начните с создания объекта презентации. Это будет ваш холст:

```python
with slides.Presentation() as document:
    # Код для добавления SmartArt находится здесь
```

#### Шаг 2: Добавьте фигуру SmartArt

Добавьте фигуру SmartArt на первый слайд в нужном месте и нужного размера:

```python
smart = document.slides[0].shapes.add_smart_art(
    10, 10, 500, 400,
    slides.smartart.SmartArtLayoutType.VERTICAL_PICTURE_LIST
)
```

#### Шаг 3: Получите доступ к первому узлу

Получите доступ к первому узлу, чтобы применить форматирование изображения маркера:

```python
node = smart.all_nodes[0]
```

#### Шаг 4: Установка формата заполнения маркера

Проверьте, существует ли формат заполнения маркера, и установите изображение в качестве маркера:

```python
if node.bullet_fill_format is not None:
    img = slides.Images.from_file("YOUR_DOCUMENT_DIRECTORY/image1.jpg")
    image = document.images.add_image(img)

    node.bullet_fill_format.fill_type = slides.FillType.PICTURE
    node.bullet_fill_format.picture_fill_format.picture.image = image
    node.bullet_fill_format.picture_fill_format.picture_fill_mode = slides.PictureFillMode.STRETCH
```

#### Шаг 5: Сохраните презентацию

Наконец, сохраните презентацию с изменениями:

```python
document.save("YOUR_OUTPUT_DIRECTORY/smart_art_bullet_fill_format_out.pptx", slides.export.SaveFormat.PPTX)
```

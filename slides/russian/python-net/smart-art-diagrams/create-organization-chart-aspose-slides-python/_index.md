### Обзор функций: создание организационной структуры

Эта функция позволяет создать презентацию с организационной диаграммой, используя макет «Организационная диаграмма с картинками» в Aspose.Slides.

#### Шаг 1: Инициализация объекта презентации

Создать новый `Presentation` объект, который будет служить холстом для добавления форм и контента:

```python
import aspose.slides as slides

def create_organization_chart():
    with slides.Presentation() as pres:
        # Дальнейшие шаги будут добавлены здесь
```

#### Шаг 2: Добавьте фигуру SmartArt на слайд

Используйте `PICTURE_ORGANIZATION_CHART` макет вашей организационной структуры:

```python
smart_art = pres.slides[0].shapes.add_smart_art(
    0,   # позиция х
    0,   # позиция Y
    400, # ширина
    400, # высота
    slides.smartart.SmartArtLayoutType.PICTURE_ORGANIZATION_CHART
)
```

**Объяснение**: Этот код добавляет фигуру SmartArt на первый слайд в указанных координатах с предопределенным размером. `SmartArtLayoutType` настроен на иерархическую визуализацию данных.

#### Шаг 3: Сохраните презентацию

Сохраните вашу организационную схему в формате PPTX:

```python
pres.save("YOUR_OUTPUT_DIRECTORY/charts_organization_chart_out.pptx", slides.export.SaveFormat.PPTX)
```

**Объяснение**: `save` Метод записывает презентацию в файл. Заменить `"YOUR_OUTPUT_DIRECTORY"` с желаемым вами путем.

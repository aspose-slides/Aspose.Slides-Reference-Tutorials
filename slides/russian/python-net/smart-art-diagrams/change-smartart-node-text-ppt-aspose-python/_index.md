### Изменить текст на узле SmartArt
В этом разделе будет показано, как изменить текст определенного узла в графическом элементе SmartArt в PowerPoint.

#### Обзор
Изменение текста в узлах SmartArt может сделать ваши презентации более динамичными и адаптивными. Это руководство покажет вам, как эффективно выбирать и обновлять текст узла.

#### Шаг 1: Загрузите или создайте презентацию
Сначала создайте новый экземпляр презентации:
```python
with slides.Presentation() as presentation:
    # Продолжайте добавлять графику SmartArt.
```

#### Шаг 2: Добавьте графику SmartArt
Здесь мы добавляем графику SmartArt к первому слайду, используя макет BasicCycle:
```python
smart = presentation.slides[0].shapes.add_smart_art(
    10, 10, 400, 300, slides.smartart.SmartArtLayoutType.BASIC_CYCLE)
```

#### Шаг 3: Выберите и измените текст узла
Выберите нужный узел и измените его текст:
```python
# Выберите второй корневой узел (индекс 1) из SmartArt.
define the node = smart.nodes[1]

# Установить новый текст для TextFrame выбранного узла
define the node.text_frame.text = "Second root node"
```

#### Шаг 4: Сохраните презентацию
Наконец, сохраните изменения в файле:
```python
presentation.save("YOUR_OUTPUT_DIRECTORY/smart_art_change_frame_text_out.pptx", slides.export.SaveFormat.PPTX)
```

### Советы по устранению неполадок
- Убедитесь, что индекс, используемый в `smart.nodes[1]` правильно соответствует узлу, который вы собираетесь изменить.
- Проверяйте пути при сохранении файлов, чтобы избежать проблем с разрешениями.

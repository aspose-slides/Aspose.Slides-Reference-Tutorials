### Изменить стиль цвета фигуры SmartArt

#### Обзор
Автоматизируйте процесс изменения цветов фигур SmartArt в презентациях PowerPoint с помощью Aspose.Slides для Python. Это обеспечивает согласованность и экономит время при подготовке.

#### Этапы внедрения

##### Шаг 1: Определите входные и выходные каталоги
Настройте каталоги документов и выходных данных:

```python
document_directory = "YOUR_DOCUMENT_DIRECTORY/"
output_directory = "YOUR_OUTPUT_DIRECTORY/"
```

Замените эти заполнители фактическими путями, по которым находятся ваши файлы PowerPoint и куда вы хотите сохранить измененные версии.

##### Шаг 2: Загрузите презентацию
Откройте файл PowerPoint с помощью Aspose.Slides:

```python
with slides.Presentation(document_directory + "smart_art_access.pptx") as presentation:
    # Код продолжается...
```

Этот фрагмент позволяет получать доступ к содержимому презентации и изменять его.

##### Шаг 3: Повторите фигуры на первом слайде
Пройдитесь по каждой фигуре на первом слайде:

```python
for shape in presentation.slides[0].shapes:
    if isinstance(shape, slides.smartart.SmartArt):
        # Продолжайте изменять цветовой стиль...
```

Мы проверяем, относится ли фигура к типу SmartArt, чтобы применить определенные изменения.

##### Шаг 4: Измените цветовой стиль
Если текущий цветовой стиль `COLORED_FILL_ACCENT1`, измените его на `COLORFUL_ACCENT_COLORS`:

```python
if shape.color_style == slides.smartart.SmartArtColorType.COLORED_FILL_ACCENT1:
    shape.color_style = slides.smartart.SmartArtColorType.COLORFUL_ACCENT_COLORS
```

Это условие гарантирует, что будут изменены только целевые фигуры SmartArt.

##### Шаг 5: Сохраните измененную презентацию.
Сохраните изменения в новом файле:

```python
presentation.save(output_directory + "smart_art_change_color_style_out.pptx", slides.export.SaveFormat.PPTX)
```

На этом этапе все изменения записываются обратно на диск, создавая обновленный файл презентации.

### Советы по устранению неполадок
- **Файл не найден:** Обеспечить пути в `document_directory` и `output_directory` верны.
- **Ошибки типа фигуры:** Прежде чем применять изменения, убедитесь, что вы получаете доступ к фигуре SmartArt.
- **Проблемы со стилем цвета:** Убедитесь, что первоначальный цветовой стиль соответствует ожидаемому в вашем сценарии.

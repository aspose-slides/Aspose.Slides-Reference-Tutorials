### Руководство по внедрению
Давайте разберемся, как удалить определенный узел из графики SmartArt.

#### Нагрузочные и траверсные направляющие
Сначала загрузите презентацию и пройдитесь по ее фигурам, чтобы определить SmartArt:

```python
import aspose.slides as slides

with slides.Presentation("YOUR_DOCUMENT_DIRECTORY/smart_art_access.pptx") as pres:
    # Повторите каждую фигуру на первом слайде.
    for shape in pres.slides[0].shapes:
        # Проверьте, является ли это объектом SmartArt.
        if isinstance(shape, slides.SmartArt):
            # Приступить к обработке узлов, если они существуют
            if len(shape.all_nodes) > 0:
                node = shape.all_nodes[0]
```

#### Доступ и удаление узла
Чтобы изменить графику SmartArt, перейдите к нужному узлу и удалите его:

```python
# Убедитесь, что имеется достаточно дочерних узлов для удаления.
count = len(node.child_nodes)
if count >= 2:
    # Удалить дочерний узел в позиции 1
    node.child_nodes.remove_node(1)
```

#### Сохраните изменения
Наконец, сохраните вашу презентацию с изменениями:

```python
pres.save("YOUR_OUTPUT_DIRECTORY/smart_art_remove_node_pos_out.pptx", slides.export.SaveFormat.PPTX)
```

**Объяснение параметров и методов:**
- **`all_nodes`**: Список узлов в графике SmartArt.
- **`remove_node(index)`**: Удаляет узел по указанному индексу. Убедитесь, что индекс действителен, чтобы избежать ошибок.

## Руководство по внедрению

В этом разделе мы расскажем вам, как получить доступ к узлам SmartArt и как их изменить на слайде PowerPoint.

### Доступ к узлам SmartArt и их изменение

**Обзор:** Эта функция позволяет программно получать доступ к определенным узлам в графике SmartArt и изменять их по мере необходимости. 

#### Шаг 1: Откройте первый слайд
```python
# Доступ к первому слайду презентации
slide = pres.slides[0]
```

#### Шаг 2: Добавьте фигуру SmartArt
```python
# Добавление фигуры SmartArt на первый слайд в указанном месте и размере
smart = slide.shapes.add_smart_art(0, 0, 400, 400, slides.smartart.SmartArtLayoutType.STACKED_LIST)
```
*Объяснение:* The `add_smart_art` Метод размещает графический элемент SmartArt на слайде и задает тип его макета.

#### Шаг 3: Доступ к определенному узлу
```python
# Доступ к первому узлу в графике SmartArt
node = smart.all_nodes[0]
```

#### Шаг 4: Доступ к дочернему узлу по индексу
```python
# Доступ к определенному дочернему узлу внутри родительского узла с использованием его индекса позиции
position = 1
child_node = node.child_nodes[position]

# Отображение параметров дочернего узла SmartArt, к которому получен доступ
print("j = {0}, Text = {1}, Level = {2}, Position = {3}".format(position, child_node.text_frame.text,
                                                                child_node.level, child_node.position))
```
*Объяснение:* На этом шаге показано, как перемещаться по узлам и извлекать информацию, такую как текст и положение.

**Совет по устранению неполадок:** Прежде чем обращаться к дочерним узлам, убедитесь, что структура SmartArt определена правильно, чтобы избежать ошибок индекса.

## Руководство по внедрению
В этом разделе вы узнаете, как создать объект SmartArt и добавить к нему узлы.
### Создание новой презентации и добавление SmartArt
**Обзор:** Начнем с создания новой презентации PowerPoint и вставки графического элемента SmartArt в первый слайд. 
#### Шаг 1: Создание нового экземпляра презентации
Создайте экземпляр класса Presentation, представляющий ваш файл PowerPoint:
```python
with slides.Presentation() as pres:
    # Ваш код будет здесь
```
#### Шаг 2: Получите доступ к первому слайду
Доступ к первому слайду презентации осуществляется с помощью его индекса:
```python
slide = pres.slides[0]
```
#### Шаг 3: Добавьте SmartArt на слайд
Добавьте графический элемент SmartArt в определенные координаты с определенными размерами:
```python
smart_art = slide.shapes.add_smart_art(0, 0, 400, 400, slides.smartart.SmartArtLayoutType.STACKED_LIST)
```
### Добавление и изменение узлов в SmartArt
**Обзор:** После добавления SmartArt вы можете изменить его, добавив узлы в определенных позициях.
#### Шаг 4: Доступ к первому узлу
Извлеките первый узел из объекта SmartArt:
```python
node = smart_art.all_nodes[0]
```
#### Шаг 5: Добавьте новый дочерний узел
Добавьте новый дочерний узел к существующему родительскому узлу в указанной позиции индекса:
```python
class NodeNotFoundException(Exception):
    pass

try:
    child_node = node.child_nodes.add_node_by_position(2)
except IndexError:
    raise NodeNotFoundException("Position does not exist in the current SmartArt layout.")
```
*Почему?* Это позволяет вам динамически структурировать SmartArt на основе конкретных требований.
#### Шаг 6: Задайте текст для нового узла
Определите текст для вновь добавленного дочернего узла:
```python
class InvalidTextException(Exception):
    pass

text = "Sample Text Added"
if not isinstance(text, str) or not text.strip():
    raise InvalidTextException("The text must be a non-empty string.")
child_node.text_frame.text = text
```
### Сохранение измененной презентации
**Обзор:** Наконец, сохраните изменения в новом файле PowerPoint.
#### Шаг 7: Сохраните презентацию
Сохраните презентацию в выходном каталоге с указанным именем файла:
```python
output_path = "./output/smart_art_add_node_by_position_out.pptx"
pres.save(output_path, slides.export.SaveFormat.PPTX)
```

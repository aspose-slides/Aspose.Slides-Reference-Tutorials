## Руководство по внедрению

Чтобы изменить макет SmartArt в PowerPoint с помощью Python, выполните следующие действия.

### Создание и изменение макетов SmartArt

#### Обзор:
Программно добавьте графический элемент SmartArt на слайд и измените тип его макета.

#### Шаг 1: Инициализация презентации
Создайте объект представления, обеспечивающий эффективную обработку ресурсов с помощью управления контекстом:

```python
with slides.Presentation() as presentation:
    # Откройте первый слайд презентации.
slide = presentation.slides[0]
```

#### Шаг 2: Добавьте графику SmartArt
Добавьте графический элемент SmartArt «BasicBlockList» в указанном месте и размере, используя:

```python
smart_art = slide.shapes.add_smart_art(
    10, 
    10, 
    400, 
    300,
    slides.smartart.SmartArtLayoutType.BASIC_BLOCK_LIST
)
```

Параметры определяют положение x и y, ширину, высоту и начальный тип макета.

#### Шаг 3: Измените макет SmartArt
Измените макет на «BasicProcess»:

```python
smart_art.layout = slides.smartart.SmartArtLayoutType.BASIC_PROCESS
```

Это обновит дизайн вашего графического элемента SmartArt для лучшего визуального представления последовательных шагов.

#### Шаг 4: Сохраните презентацию
Сохраните измененную презентацию:

```python
output_path = 'YOUR_OUTPUT_DIRECTORY/smart_art_change_layout_out.pptx'
presentation.save(output_path, slides.export.SaveFormat.PPTX)
```

### Советы по устранению неполадок

- Убедитесь, что пути к файлам для сохранения корректны в вашей системе.

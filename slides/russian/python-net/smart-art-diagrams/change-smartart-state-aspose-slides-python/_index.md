## Руководство по внедрению

### Добавление и изменение графики SmartArt

#### Обзор
В этом разделе мы узнаем, как добавить графический элемент SmartArt на слайд и изменить его свойства, например, изменить его состояние.

#### Пошаговая реализация:

**1. Создайте новую презентацию:**

```python
with slides.Presentation() as presentation:
    # Доступ к первому слайду (индекс 0)
slide = presentation.slides[0]
```

На этом этапе инициализируется новый объект презентации и открывается его для редактирования с использованием методов управления ресурсами.

**2. Добавьте графику SmartArt:**

```python
# Добавить графику SmartArt с указанными размерами и типом макета
smart = slide.shapes.add_smart_art(
    x=10, y=10, width=400, height=300,
    layout_type=slides.smartart.SmartArtLayoutType.BASIC_PROCESS
)
```

Здесь мы добавляем базовый процесс SmartArt в заданные координаты. `add_smart_art` Метод позволяет точно разместить и настроить размер.

**3. Измените состояние реверса:**

```python
# Настройте графику SmartArt так, чтобы она была обращена
smart.is_reversed = True
```

Эта линия изменяет ориентацию SmartArt, добавляя динамический визуальный эффект.

**4. Сохраните презентацию:**

```python
presentation.save("YOUR_OUTPUT_DIRECTORY/smart_art_change_state_out.pptx")
```

Наконец, сохраните вашу презентацию в указанном каталоге. Убедитесь, что вы заменили `YOUR_OUTPUT_DIRECTORY` с реальным путем в вашей системе.

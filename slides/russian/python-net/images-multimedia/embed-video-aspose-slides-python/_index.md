# Как встроить видеокадры в слайды PowerPoint с помощью Aspose.Slides для Python

- Как встроить видеокадр в слайд PowerPoint с помощью Aspose.Slides.
- Действия по загрузке и управлению видео в презентации.
- Основные параметры конфигурации параметров воспроизведения видео в PowerPoint.

Давайте убедимся, что все настроено правильно, прежде чем мы начнем встраивать эти видео!

## Руководство по внедрению

Мы разберем реализацию на две основные функции: встраивание видеокадра и загрузка видео.

### Функция 1: Встраивание видеокадра

Эта функция позволяет встраивать видео непосредственно в первый слайд презентации PowerPoint.

#### Пошаговая реализация
**Шаг 1:** Создайте новый объект «Презентация».

```python
with slides.Presentation() as pres:
    # Дальнейшие шаги здесь...
```

**Шаг 2:** Доступ к первому слайду.

```python
slide = pres.slides[0]
```

**Шаг 3:** Загрузите видео и добавьте его в презентацию.

Убедитесь, что ваш видеофайл готов. Мы будем использовать пример пути `video.mp4` для этого примера.

```python
video_path = "YOUR_DOCUMENT_DIRECTORY/video.mp4"
with open(video_path, "rb") as in_file:
    video = pres.videos.add_video(in_file, slides.LoadingStreamBehavior.READ_STREAM_AND_RELEASE)
```

**Шаг 4:** Добавьте видеокадр к слайду.

Разместите и измените размер видеокадра в соответствии с макетом слайда.

```python
vf = slide.shapes.add_video_frame(50, 150, 300, 350, video)
```

**Шаг 5:** Назначьте встроенное видео рамке.

Свяжите загруженное видео с назначенным ему кадром.

```python
vf.embedded_video = video
```

**Шаг 6:** Установите режим воспроизведения и громкость видео.

Настройте воспроизведение видео в режиме презентации.

```python
vf.play_mode = slides.VideoPlayModePreset.AUTO
vf.volume = slides.AudioVolumeMode.LOUD
```

**Шаг 7:** Сохраните презентацию со встроенным видео.

Выберите выходной каталог для сохранения файла PowerPoint.

```python
output_path = "YOUR_OUTPUT_DIRECTORY/shapes_embed_video_frame_out.pptx"
pres.save(output_path, slides.export.SaveFormat.PPTX)
```

### Функция 2: Загрузка видео в презентацию

Эта функция демонстрирует загрузку видео в коллекцию презентации без встраивания его в какой-либо конкретный кадр.

#### Пошаговая реализация
**Шаг 1:** Создать новый объект презентации.

```python
with slides.Presentation() as pres:
    # Дальнейшие шаги здесь...
```

**Шаг 2:** Загрузить видео из каталога.

```python
video_path = "YOUR_DOCUMENT_DIRECTORY/video.mp4"
with open(video_path, "rb") as in_file:
    video = pres.videos.add_video(in_file, slides.LoadingStreamBehavior.READ_STREAM_AND_RELEASE)
```

Если вы просто загружаете видео для последующего использования или справки, никаких дополнительных действий не требуется.

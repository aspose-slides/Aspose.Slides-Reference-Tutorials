## Руководство по внедрению
В этом разделе вы узнаете, как встраивать и обрезать аудио с помощью Aspose.Slides.

### Добавить аудиокадр в презентацию
**Обзор**: Повысьте интерактивность своей презентации, добавив аудиофайл в виде встроенного кадра в слайд PowerPoint.

#### Шаг 1: Откройте презентацию для внесения изменений
```python
# Открыть или создать новую презентацию
current_pres = slides.Presentation()
```

#### Шаг 2: Прочтите и добавьте аудиофайл
```python
    # Откройте аудиофайл из вашего каталога в двоичном режиме.
    with open('YOUR_DOCUMENT_DIRECTORY/audio.m4a', 'rb') as audio_file:
        # Добавить аудио в коллекцию презентации
        current_audio = current_pres.audios.add_audio(audio_file)
```

#### Шаг 3: Вставьте аудиокадр в слайд
```python
    # Добавить встроенный аудиокадр в указанные координаты (50, 50) размером (100, 100)
    audio_frame = current_pres.slides[0].shapes.add_audio_frame_embedded(50, 50, 100, 100, current_audio)
```

### Обрезать аудиокадр в презентации
**Обзор**: Обрезка начала и конца аудиокадра может иметь решающее значение для точного тайминга презентации.

#### Шаг 1: Установите начало обрезки
```python
    # Обрезать начало аудио на 500 миллисекунд (0,5 секунды)
    audio_frame.trim_from_start = 500
```

#### Шаг 2: Установка конечной обрезки
```python
    # Обрезать конец аудио на 1000 миллисекунд (1 секунду)
    audio_frame.trim_from_end = 1000
```

### Сохранение презентации
Сохраните измененную презентацию в выходном каталоге:
```python
    current_pres.save('YOUR_OUTPUT_DIRECTORY/AudioFrameTrim_out.pptx', slides.export.SaveFormat.PPTX)
```

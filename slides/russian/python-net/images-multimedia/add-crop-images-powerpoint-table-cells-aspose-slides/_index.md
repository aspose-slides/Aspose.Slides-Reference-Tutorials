## Руководство по внедрению
### Шаг 1: Создание объекта класса представления
Создайте экземпляр `Presentation` класс, представляющий ваш файл PowerPoint:

```python
with slides.Presentation() as presentation:
```

### Шаг 2: Доступ к первому слайду
Откройте слайд, на который вы хотите добавить таблицу:

```python
slide = presentation.slides[0]
```

### Шаг 3: Определите структуру таблицы
Укажите ширину столбцов и высоту строк для вашей таблицы. Здесь мы устанавливаем единые размеры для простоты.

```python
dbl_cols = [150, 150, 150, 150]  # Ширина столбцов в пунктах
dbl_rows = [100, 100, 100, 100, 90]  # Высота строк в пунктах
```

### Шаг 4: Добавьте таблицу на слайд
Разместите таблицу на слайде в указанных координатах:

```python
tbl = slide.shapes.add_table(50, 50, dbl_cols, dbl_rows)
```

### Шаг 5: Загрузите и добавьте изображение
Загрузите изображение из каталога и добавьте его в коллекцию изображений презентации.

```python
image_path = "YOUR_DOCUMENT_DIRECTORY/image1.jpg"
image = slides.Images.from_file(image_path)
imgx1 = presentation.images.add_image(image)
```

### Шаг 6: Установите изображение в качестве заливки с обрезкой
Примените загруженное изображение к ячейке таблицы и задайте параметры обрезки:

```python
tbl.rows[0][0].cell_format.fill_format.fill_type = slides.FillType.PICTURE
tbl.rows[0][0].cell_format.fill_format.picture_fill_format.picture_fill_mode = slides.PictureFillMode.STRETCH
tbl.rows[0][0].cell_format.fill_format.picture_fill_format.picture.image = imgx1

# Значения обрезки в точках
tbl.rows[0][0].cell_format.fill_format.picture_fill_format.crop_right = 20
tbl.rows[0][0].cell_format.fill_format.picture_fill_format.crop_left = 20
tbl.rows[0][0].cell_format.fill_format.picture_fill_format.crop_top = 20
tbl.rows[0][0].cell_format.fill_format.picture_fill_format.crop_bottom = 20
```

### Шаг 7: Сохраните презентацию
Наконец, сохраните вашу презентацию в файл:

```python
output_path = "YOUR_OUTPUT_DIRECTORY/tables_add_crop_image_to_cell_out.pptx"
presentation.save(output_path, slides.export.SaveFormat.PPTX)
```

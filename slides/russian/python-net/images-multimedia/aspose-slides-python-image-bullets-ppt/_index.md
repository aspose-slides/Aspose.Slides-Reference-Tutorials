## Руководство по внедрению

### Добавление маркеров изображений в абзацы в PowerPoint

#### Обзор
Повысьте визуальную привлекательность и вовлеките свою аудиторию, добавив маркированные изображения к абзацам на слайде.

#### Шаги по реализации

**Доступ к слайду:**
```python
# Открыть или создать презентацию
with slides.Presentation() as presentation:
    # Доступ к первому слайду
    slide = presentation.slides[0]
```

**Добавление изображения для маркеров:**
```python
# Загрузить изображение из файла и добавить в коллекцию изображений презентации
image = slides.Images.from_file("YOUR_DOCUMENT_DIRECTORY/bullets.png")
ippx_image = presentation.images.add_image(image)
```
*На этом этапе необходимо загрузить желаемое изображение маркера и добавить его на слайд.*

**Создание текстовой рамки с маркерами изображений:**
```python
# Добавьте автофигуру (прямоугольник) и получите доступ к ее текстовой рамке.
auto_shape = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 200, 200, 400, 200)
text_frame = auto_shape.text_frame

# Удалить абзац по умолчанию, если он существует
if len(text_frame.paragraphs) > 0:
    text_frame.paragraphs.remove_at(0)

# Создайте новый абзац и установите для него тип маркера «картинка».
paragraph = slides.Paragraph()
paragraph.text = "Welcome to Aspose.Slides"
paragraph.paragraph_format.bullet.type = slides.BulletType.PICTURE
paragraph.paragraph_format.bullet.picture.image = ippx_image
paragraph.paragraph_format.bullet.height = 100

# Добавьте абзац в текстовый фрейм
text_frame.paragraphs.add(paragraph)
```
*Этот блок кода создает новый абзац, назначает изображение в качестве его маркера и настраивает его свойства.*

**Сохранение презентации:**
```python
# Сохраните презентацию с изменениями
presentation.save("YOUR_OUTPUT_DIRECTORY/text_picture_bullets_out.pptx", slides.export.SaveFormat.PPTX)
```

### Доступ к элементам слайда и управление ими

#### Обзор
Узнайте, как получить доступ к элементам слайда, таким как фигуры и текстовые рамки, для дальнейшей настройки.

**Доступ к слайду и форме:**
```python
# Открыть или создать презентацию
with slides.Presentation() as presentation:
    # Доступ к первому слайду
    slide = presentation.slides[0]

    # Добавьте автофигуру (прямоугольник) для демонстрации манипуляций.
    auto_shape = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 200, 200, 400, 200)
text_frame = auto_shape.text_frame

    # Удалить первый абзац, если он есть.
    if len(text_frame.paragraphs) > 0:
        text_frame.paragraphs.remove_at(0)

    # Создайте и добавьте новый абзац с пользовательским текстом
    paragraph = slides.Paragraph()
    paragraph.text = "Manipulating Slide Elements"
text_frame.paragraphs.add(paragraph)
```

**Сохранение измененной презентации:**
```python
# Сохраните презентацию после внесения изменений
presentation.save("YOUR_OUTPUT_DIRECTORY/modified_slide.pptx", slides.export.SaveFormat.PPTX)
```

## Руководство по внедрению

В этом разделе вы узнаете, как добавить звуковой кадр в презентацию PowerPoint.

### Добавление аудио в презентацию

**Обзор:**
Добавьте аудиофайл к первому слайду презентации. Это включает загрузку аудио, встраивание его в качестве аудиофрейма в слайд и сохранение обновленной презентации.

#### Шаг 1: Настройте пути к файлам
Определите пути для входного аудиофайла и выходной презентации:
```python
input_audio_path = 'YOUR_DOCUMENT_DIRECTORY/audio.m4a'
output_presentation_path = 'YOUR_OUTPUT_DIRECTORY/AudioFrameValue_out.pptx'
```
Заменять `YOUR_DOCUMENT_DIRECTORY` с каталогом, содержащим ваш аудиофайл, и `YOUR_OUTPUT_DIRECTORY` укажите, где вы хотите сохранить презентацию.

#### Шаг 2: Создание экземпляра презентации
Используйте контекстный менеджер для правильного управления ресурсами:
```python
with slides.Presentation() as pres:
    # Дальнейшие шаги будут выполнены в рамках этого блока.
```

#### Шаг 3: Загрузите и добавьте аудио
Откройте ваш аудиофайл в режиме двоичного чтения, затем добавьте его в коллекцию аудиофайлов презентации:
```python
with open(input_audio_path, "rb") as in_file:
    audio = pres.audios.add_audio(in_file)
```
The `add_audio` функция добавляет ваш аудиофайл во внутреннюю коллекцию для встраивания в слайды.

#### Шаг 4: Вставьте аудиокадр в слайд
Вставьте аудиокадр в первый слайд в указанном месте с определенными размерами:
```python
audio_frame = pres.slides[0].shapes.add_audio_frame_embedded(50, 50, 100, 100, audio)
```
Параметры `(50, 50, 100, 100)` укажите координаты x, y, ширину и высоту аудиокадра.

### Сохранение презентации
Презентация автоматически сохраняется при выходе из `with` блок. Убедитесь, что путь вывода указан правильно, чтобы предотвратить перезапись или потерю файлов.

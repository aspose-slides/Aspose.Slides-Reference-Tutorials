## Руководство по внедрению
Мы рассмотрим две основные функции: добавление и удаление субтитров из видео, встроенных в презентации PowerPoint.

### Добавление субтитров к видеокадру
Эта функция позволяет вам повысить доступность вашего видеоконтента, включив субтитры или подписи непосредственно в вашу презентацию.

#### Шаг 1: Создание и загрузка презентации
Начните с создания нового объекта презентации:

```python
import aspose.slides as slides

def add_video_captions():
    # Создать новую презентацию
    with slides.Presentation() as pres:
        ...
```

#### Шаг 2: Добавьте видеофайл
Загрузите ваш видеофайл в презентацию. Убедитесь, что у вас правильный путь к вашему видео:

```python
        with open("YOUR_DOCUMENT_DIRECTORY/NewVideo.mp4", "rb") as f:
            video = pres.videos.add_video(f.read())
```

#### Шаг 3: Вставьте видеокадр и добавьте субтитры
Вставьте `VideoFrame` в нужном месте и добавьте подписи, используя ваш файл VTT:

```python
        # Добавить VideoFrame с указанными размерами
        video_frame = pres.slides[0].shapes.add_video_frame(0, 0, 100, 100, video)
        
        # Прикрепить дорожку субтитров из файла VTT
        video_frame.caption_tracks.add("New track", "YOUR_DOCUMENT_DIRECTORY/bunny.vtt")
```

#### Шаг 4: Сохраните презентацию
Наконец, сохраните обновленную презентацию с подписями:

```python
        pres.save("YOUR_OUTPUT_DIRECTORY/VideoCaptionsAdd_out.pptx", slides.export.SaveFormat.PPTX)
```

### Извлечение и удаление титров из видеокадра
Теперь, когда вы добавили субтитры, давайте рассмотрим, как извлечь их для просмотра или полностью удалить.

#### Шаг 1: Откройте существующую презентацию
Начните с загрузки презентации, содержащей ваше видео с субтитрами:

```python
def extract_and_remove_captions():
    # Загрузить существующую презентацию
    with slides.Presentation("YOUR_OUTPUT_DIRECTORY/VideoCaptionsAdd_out.pptx") as pres:
        ...
```

#### Шаг 2: Извлечение данных субтитров
Пройдитесь по каждой дорожке субтитров, чтобы сохранить ее данные в файлах VTT:

```python
        video_frame = pres.slides[0].shapes[0]
        if video_frame is not None:
            for idx, caption_track in enumerate(video_frame.caption_tracks):
                with open(f"YOUR_OUTPUT_DIRECTORY/VideoCaption_out_{idx}.vtt", "wb") as f:
                    f.write(caption_track.binary_data)
```

#### Шаг 3: Удалить субтитры
Удалить все субтитры из видеокадра:

```python
            # Очистить все дорожки субтитров
            video_frame.caption_tracks.clear()
            
            # Сохранить изменения в новом файле
            pres.save("YOUR_OUTPUT_DIRECTORY/VideoCaptionsRemove_out.pptx", slides.export.SaveFormat.PPTX)
```

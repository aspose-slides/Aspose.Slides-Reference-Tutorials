## Руководство по внедрению
### Доступ к слайдам и их изменение
#### Обзор
Чтобы сжать изображение в презентации, сначала нужно получить доступ к определенному слайду и кадру изображения. Вот как это сделать с помощью Aspose.Slides:
#### Пошаговая реализация
**1. Загрузите презентацию:**
```python
import aspose.slides as slides
import os

document_path = "YOUR_DOCUMENT_DIRECTORY/CroppedImage.pptx"
output_path = "YOUR_OUTPUT_DIRECTORY/CroppedImage-Compress-out.pptx"

with slides.Presentation(document_path) as presentation:
```
*Объяснение*: Используйте менеджер контекста, чтобы открыть файл PowerPoint и убедиться, что он правильно закрывается после обработки.
**2. Доступ к первому слайду:**
```python
    slide = presentation.slides[0]
```
*Объяснение*: Это позволит получить первый слайд вашей презентации.
**3. Получите рамку изображения:**
```python
    picture_frame = slide.shapes[0]  # Предположим, что первая фигура — PictureFrame.
```
*Объяснение*: Мы предполагаем, что первая фигура на слайде — это рамка изображения (PictureFrame). При необходимости отрегулируйте это в зависимости от вашего конкретного варианта использования.
**4. Сжатие изображения:**
```python
    compression_result = picture_frame.picture_format.compress_image(True, 150)
```
*Объяснение*: `compress_image` Метод снижает разрешение изображения до 150 точек на дюйм, что подходит для использования в Интернете, при этом размеры файлов остаются контролируемыми.
**5. Сохраните презентацию:**
```python
    presentation.save(output_path, slides.export.SaveFormat.PPTX)

# Отображение размеров исходного и полученного представления для сравнения
original_size = os.stat(document_path).st_size
compressed_size = os.stat(output_path).st_size
print("Source presentation size:", original_size)  # В байтах
print("Compressed presentation size:", compressed_size)  # В байтах
```
*Объяснение*: Презентация сохраняется с новым, сжатым изображением. Мы также распечатываем размеры файлов, чтобы продемонстрировать достигнутое сокращение.

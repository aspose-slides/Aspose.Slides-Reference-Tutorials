### Добавление видеокадра

#### Обзор
Мы продемонстрируем добавление видеокадра к первому слайду презентации. Эта функция полезна, когда вы хотите включить мультимедийный контент непосредственно в слайды.

#### Пошаговая реализация:
##### Доступ к первому слайду
```python
class PresentationManager:
    def __init__(self):
        self.presentation = slides.Presentation()

    def access_first_slide(self):
        # Доступ к первому слайду из коллекции
        return self.presentation.slides[0]
```
*Почему?*: Этот шаг гарантирует, что вы работаете с правильным слайдом, на который собираетесь добавить видео.

##### Добавление видеокадра
```python
class PresentationManager:
    def __init__(self):
        self.presentation = slides.Presentation()

    def access_first_slide(self):
        return self.presentation.slides[0]

    def add_video_frame(self, slide, video_path):
        # Добавить видеокадр на слайд в указанном месте и размере
        vf = slide.shapes.add_video_frame(50, 150, 300, 150, video_path)
        return vf
```
*Объяснение*: Эта строка вставляет видеокадр в ваш слайд. Параметры `50`, `150`, `300`, `150` определяют координаты X, Y и ширину, высоту видеокадра соответственно.

##### Настройка воспроизведения видео
```python
class PresentationManager:
    def __init__(self):
        self.presentation = slides.Presentation()

    def access_first_slide(self):
        return self.presentation.slides[0]

    def add_video_frame(self, slide, video_path):
        vf = slide.shapes.add_video_frame(50, 150, 300, 150, video_path)
        # Установите режим воспроизведения видео для автоматического запуска при показе слайда.
        vf.play_mode = slides.VideoPlayModePreset.AUTO
        # Установите громкость видео
        vf.volume = slides.AudioVolumeMode.LOUD
        return vf
```
*Цель*: Эти конфигурации гарантируют, что ваша аудитория услышит и увидит видео сразу же после того, как доберется до слайда.

##### Сохранение презентации
```python
class PresentationManager:
    def __init__(self):
        self.presentation = slides.Presentation()

    def access_first_slide(self):
        return self.presentation.slides[0]

    def add_video_frame(self, slide, video_path):
        vf = slide.shapes.add_video_frame(50, 150, 300, 150, video_path)
        vf.play_mode = slides.VideoPlayModePreset.AUTO
        vf.volume = slides.AudioVolumeMode.LOUD
        return vf

    def save_presentation(self, output_directory):
        # Сохраните презентацию под новым именем в указанном выходном каталоге.
        self.presentation.save(f"{output_directory}/shapes_add_video_out.pptx")
```
*Почему?*: На этом этапе вы завершаете свои изменения, сохраняя их в файле, гарантируя, что ваша работа не будет потеряна и ее можно будет опубликовать или представить.

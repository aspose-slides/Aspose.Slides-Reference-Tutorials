### Включение функции управления мультимедиа

#### Обзор

Включение управления мультимедиа позволяет пользователям воспроизводить, приостанавливать и перемещаться по встроенным медиафайлам во время презентации. Эта функция улучшает взаимодействие, предоставляя контроль над элементами мультимедиа без выхода из режима слайдов.

#### Этапы внедрения

##### Шаг 1: Создание экземпляра презентации

Начните с создания экземпляра `Presentation` класс, использующий менеджер контекста для эффективного управления ресурсами:

```python
def enable_media_controls_in_slideshow():
    with slides.Presentation() as pres:
        # Код для изменения представления находится здесь
```

##### Шаг 2: Включите элементы управления мультимедиа

Используйте `show_media_controls` атрибут, позволяющий отображать элементы управления мультимедиа в режиме слайд-шоу. Это гарантирует пользователям возможность прямого взаимодействия с медиафайлами во время презентаций:

```python
def enable_media_controls_in_slideshow():
    with slides.Presentation() as pres:
        # Включить отображение управления мультимедиа в режиме слайд-шоу
        pres.slide_show_settings.show_media_controls = True
        
        output_path = "YOUR_OUTPUT_DIRECTORY/SlideShowMediaControl.pptx"
        pres.save(output_path, slides.export.SaveFormat.PPTX)
```

##### Шаг 3: Сохраните презентацию

Наконец, сохраните измененную презентацию. `save` метод записывает изменения в указанный путь к файлу:

```python
output_path = "YOUR_OUTPUT_DIRECTORY/SlideShowMediaControl.pptx"
pres.save(output_path, slides.export.SaveFormat.PPTX)
```

#### Советы по устранению неполадок
- Перед сохранением убедитесь, что выходной каталог существует.
- Убедитесь, что медиа-файлы правильно встроены в слайды PowerPoint.

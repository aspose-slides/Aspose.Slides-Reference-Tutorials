## Руководство по внедрению

Теперь давайте разберем процесс добавления видеоблока в презентацию PowerPoint.

### 1. Подготовка вашей среды

Начните с настройки каталогов для входных и выходных файлов:

```python
import os

# Установить пути для хранения документов
data_directory = 'YOUR_DOCUMENT_DIRECTORY/'
output_directory = 'YOUR_OUTPUT_DIRECTORY/'

# Создайте каталоги, если они не существуют
os.makedirs(data_directory, exist_ok=True)
os.makedirs(output_directory, exist_ok=True)
```

### 2. Создание видеофайла

Для демонстрационных целей создайте видеофайл-заполнитель:

```python
video_file_path = os.path.join(data_directory, "video.mp4")
with open(video_file_path, 'wb') as video_file:
    # Имитированные двоичные данные для примера
    video_file.write(b'\x00\x01\x02')
```

### 3. Добавление видео в презентацию

Теперь давайте добавим это видео как фрагмент в новый файл PowerPoint:

```python
with slides.Presentation() as pres:
    with open(video_file_path, "rb") as file_stream:
        # Добавьте видео, используя поведение KEEP_LOCKED в целях безопасности.
        video = pres.videos.add_video(file_stream, slides.LoadingStreamBehavior.KEEP_LOCKED)
        
        # Вставьте видеокадр в первый слайд
        pres.slides[0].shapes.add_video_frame(0, 0, 480, 270, video)

    # Сохраните вашу презентацию с добавленным видеоблогом
    output_file_path = os.path.join(output_directory, "props_add_blob_to_presentation_out.pptx")
    pres.save(output_file_path, slides.export.SaveFormat.PPTX)
```

**Основные параметры конфигурации:**
- **Поведение KEEP_LOCKED**: Гарантирует, что после встраивания видео его невозможно будет непреднамеренно изменить.

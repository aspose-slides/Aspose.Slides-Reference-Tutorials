## Руководство по внедрению

### Настройка смещения растяжения в рамах для картин

Цель здесь — настроить заливку изображения внутри формы, гарантируя, что она идеально подходит под ваши потребности дизайна. Выполните следующие шаги:

#### 1. Создание экземпляра класса представления

Начните с создания экземпляра `Presentation` сорт:
```python
with slides.Presentation() as pres:
    slide = pres.slides[0]
```
Откроется первый слайд для редактирования.

#### 2. Загрузите и добавьте изображение

Загрузите нужное вам изображение в коллекцию изображений презентации:
```python
img = slides.Images.from_file('YOUR_DOCUMENT_DIRECTORY/image1.jpg')
imgx = pres.images.add_image(img)
```
Заменять `'YOUR_DOCUMENT_DIRECTORY/image1.jpg'` с путем к вашему изображению.

#### 3. Добавить автофигуру и задать тип заливки

Добавьте к слайду прямоугольную форму:
```python
auto_shape = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 100, 100, 300, 300)
auto_shape.fill_format.fill_type = slides.FillType.PICTURE
```
Этот код определяет положение и размер фигуры на слайде.

#### 4. Настройте режим заливки изображения

Установите режим заливки изображения на растяжение:
```python
auto_shape.fill_format.picture_fill_format.picture_fill_mode = slides.PictureFillMode.STRETCH
auto_shape.fill_format.picture_fill_format.picture.image = imgx
```
Это гарантирует, что изображение растянется и впишется в форму.

#### 5. Установите смещения растяжения

Отрегулируйте смещения для точного позиционирования:
```python
auto_shape.fill_format.picture_fill_format.stretch_offset_left = 25
auto_shape.fill_format.picture_fill_format.stretch_offset_right = 25
auto_shape.fill_format.picture_fill_format.stretch_offset_top = -20
auto_shape.fill_format.picture_fill_format.stretch_offset_bottom = -10
```
Эти значения изменяют способ выравнивания изображения в границах фигуры.

#### 6. Сохранить презентацию

Наконец, сохраните изменения:
```python
pres.save('YOUR_OUTPUT_DIRECTORY/shapes_stretch_offset_out.pptx', slides.export.SaveFormat.PPTX)
```
Заменять `'YOUR_OUTPUT_DIRECTORY'` с желаемым выходным путем.

### Советы по устранению неполадок

- Убедитесь, что путь к образу указан правильно, чтобы избежать ошибок «файл не найден».
- Убедитесь, что смещения не выходят за границы фигуры, так как это может привести к неожиданным результатам.

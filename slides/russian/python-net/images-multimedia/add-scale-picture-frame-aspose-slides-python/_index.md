### Функция 1: добавление рамки изображения с относительным масштабом

Эта функция демонстрирует, как добавить рамку изображения к первому слайду презентации PowerPoint и настроить ее масштаб по ширине и высоте.

#### Пошаговая реализация
##### **Настроить объект презентации**
Начните с создания объекта презентации с помощью Aspose.Slides. Это гарантирует правильное управление ресурсами:

```python
def add_relative_scale_picture_frame():
    with slides.Presentation() as presentation:
```

##### **Загрузить изображение**
Далее загрузите нужное изображение в коллекцию изображений презентации:

```python
        img = slides.Images.from_file('YOUR_DOCUMENT_DIRECTORY/image1.jpg')
        image = presentation.images.add_image(img)
```

**Объяснение**: `Images.from_file()` Метод загружает изображение по указанному пути и добавляет его в коллекцию презентации.

##### **Добавить рамку для фото**
Теперь добавьте рамку изображения к первому слайду с определенными размерами:

```python
        pf = presentation.slides[0].shapes.add_picture_frame(
            slides.ShapeType.RECTANGLE, 50, 50, 100, 100, image
        )
```

**Объяснение**: `add_picture_frame()` Метод размещает прямоугольную рамку в координатах (50, 50) с шириной и высотой 100 единиц. Параметры определяют тип фигуры, положение, размер и изображение.

##### **Установить относительную шкалу ширины и высоты**
Отрегулируйте масштаб для визуальной привлекательности:

```python
        pf.relative_scale_height = 0.8
        pf.relative_scale_width = 1.35
```

**Объяснение**: Эти свойства позволяют динамически регулировать высоту и ширину фрейма относительно его исходного размера.

##### **Сохранить презентацию**
Наконец, сохраните презентацию в желаемом каталоге:

```python

```

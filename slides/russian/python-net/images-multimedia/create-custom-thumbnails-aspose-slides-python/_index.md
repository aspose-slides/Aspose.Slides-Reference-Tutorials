## Руководство по внедрению
В этом разделе вы узнаете, как создавать эскизы слайдов PowerPoint нестандартного размера.

### Обзор функции
Эта функция позволяет вам определять желаемые размеры миниатюр слайдов и генерировать их программно.

#### Шаг 1: Определите входные и выходные пути
Укажите, где находится входной файл PowerPoint и куда вы хотите сохранить выходное миниатюрное изображение:
```python
input_file = "YOUR_DOCUMENT_DIRECTORY/welcome-to-powerpoint.pptx"
output_file = "YOUR_OUTPUT_DIRECTORY/thumbnail_user_defined_dimensions_out.jpg"
```

#### Шаг 2: Откройте презентацию
Используйте Aspose.Slides для открытия файла презентации. Этот шаг необходим для доступа к слайдам:
```python
import aspose.slides as slides

with slides.Presentation(input_file) as pres:
    slide = pres.slides[0]
```

#### Шаг 3: Установите желаемые размеры
Определите размеры, которые вы хотите для своего эскиза. В этом примере мы устанавливаем его на 1200x800 пикселей:
```python
desired_x, desired_y = 1200, 800
scale_x = (1.0 / pres.slide_size.size.width) * desired_x
scale_y = (1.0 / pres.slide_size.size.height) * desired_y
```

#### Шаг 4: Создайте и сохраните миниатюру
Создайте миниатюру, используя рассчитанные масштабы, и сохраните ее как файл JPEG:
```python
img = slide.get_image(scale_x, scale_y)
img.save(output_file, slides.ImageFormat.JPEG)
```

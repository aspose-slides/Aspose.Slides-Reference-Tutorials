## Руководство по внедрению

### Функция 1: Добавление видеокадра с YouTube

Эта функция демонстрирует, как добавить видеокадр с видео YouTube и его миниатюру в слайд PowerPoint.

#### Пошаговое руководство

##### Шаг 1: Создание видеокадра
Создайте видеокадр на первом слайде в позиции (10, 10) с размерами 427x240 пикселей:
```python
def add_video_from_youtube(pres, video_id):
    video_frame = pres.slides[0].shapes.add_video_frame(10, 10, 427, 240, "https://www.youtube.com/embed/" + video_id)
```
*Параметры определяют положение и размер видеокадра на слайде.*

##### Шаг 2: Установите режим воспроизведения видео
Настройте режим воспроизведения для автоматического запуска при нажатии:
```python
    video_frame.play_mode = slides.VideoPlayModePreset.AUTO
```

##### Шаг 3: Загрузите миниатюрное изображение
Загрузите и установите миниатюру изображения с YouTube для видеокадра:
```python
    from urllib.request import urlopen
    
    thumbnail_uri = "http://img.youtube.com/vi/" + video_id + "/hqdefault.jpg"
    with urlopen(thumbnail_uri) as f:
        video_frame.picture_format.picture.image = pres.images.add_image(f.read())
```

## Руководство по внедрению

### Рендеринг и установка номера слайда

#### Обзор:
Эта функция позволяет вам загрузить презентацию PowerPoint, извлечь и изменить номер первого слайда, а затем эффективно сохранить изменения.

#### Шаги:

##### Шаг 1: Определите пути к файлам
Начните с определения путей для ваших входных и выходных файлов. Замените заполнители реальными именами каталогов.

```python
input_path = "YOUR_DOCUMENT_DIRECTORY/welcome-to-powerpoint.pptx"
output_path = "YOUR_OUTPUT_DIRECTORY/rendering_set_slide_number_out.pptx"
```

##### Шаг 2: Загрузите презентацию

Использовать `slides.Presentation` для загрузки файла PowerPoint. Этот менеджер контекста гарантирует, что ресурсы будут освобождены после завершения.

```python
with slides.Presentation(input_path) as presentation:
    # Продолжайте манипулировать номерами слайдов.
```

##### Шаг 3: Извлечение и изменение номера слайда

Получите текущий номер первого слайда для проверки, затем установите новое значение:

```python
first_slide_number = presentation.first_slide_number
print(f"Original First Slide Number: {first_slide_number}")

presentation.first_slide_number = 10
print("First slide number set to 10.")
```

##### Шаг 4: Сохраните измененную презентацию.

Наконец, сохраните изменения. Этот шаг гарантирует, что все изменения будут сохранены.

```python
presentation.save(output_path, slides.export.SaveFormat.PPTX)
print(f"Presentation saved with new slide numbering at {output_path}")
```

#### Советы по устранению неполадок:
- Убедитесь, что пути указаны правильно, чтобы избежать ошибок «файл не найден».
- Убедитесь, что файл PowerPoint доступен и не поврежден.
- Проверьте, есть ли у вас разрешение на запись файлов в выходной каталог.

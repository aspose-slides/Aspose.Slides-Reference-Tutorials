### Удалить слайд с помощью индекса

#### Обзор:
Удаление слайда по его индексу позволяет быстро редактировать презентации без ручного перемещения по ним. Это особенно полезно для автоматизированных сценариев или задач массовой обработки.

#### Шаги:
**1. Доступ к коллекции слайдов:**
```python
import aspose.slides as slides

# Определить каталоги
data_directory = 'YOUR_DOCUMENT_DIRECTORY/'
output_directory = 'YOUR_OUTPUT_DIRECTORY/'

with slides.Presentation(data_directory + "welcome-to-powerpoint.pptx") as current_presentation:
    # Доступ к коллекции слайдов
```
*Объяснение:* Загрузка презентации позволяет нам программно манипулировать ее содержимым.

**2. Удалить слайд по индексу:**
```python
    # Удалить первый слайд, используя индекс 0
current_presentation.slides.remove_at(0)
```
*Объяснение:* `remove_at(index)` удаляет указанный слайд, начиная с нуля для первого слайда.

**3. Сохраните измененную презентацию:**
```python
    # Сохранить измененную презентацию в новый файл
current_presentation.save(output_directory + "modified-presentation.pptx", slides.export.SaveFormat.PPTX)
```
*Объяснение:* На этом этапе ваши изменения сохраняются, гарантируя, что изменения будут сохранены в новом файле.

### Советы по устранению неполадок:
- Во избежание ошибок убедитесь, что индекс находится в пределах существующих слайдов.
- Проверьте пути к каталогам для чтения и записи файлов, чтобы предотвратить исключения «файл не найден».

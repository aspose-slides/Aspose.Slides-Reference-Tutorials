### Руководство по внедрению

#### Клонирование слайда в той же презентации

**Обзор:**
Эта функция позволяет дублировать слайд в презентации, помещая его в определенный индекс. Это особенно полезно для повторения контента или поддержания единообразных макетов.

##### Пошаговый процесс:

1. **Загрузите вашу презентацию**
   Загрузите файл PowerPoint, из которого вы хотите клонировать слайды.
   
   ```python
   with slides.Presentation(YOUR_DOCUMENT_DIRECTORY + 'welcome-to-powerpoint.pptx') as pres:
       all_slides = pres.slides
   ```

2. **Клонировать и вставить по указанному индексу**
   Использовать `insert_clone` метод дублирования слайда и размещения его в желаемом месте.
   
   ```python
   def clone_slide_at_index():
       with slides.Presentation(YOUR_DOCUMENT_DIRECTORY + 'welcome-to-powerpoint.pptx') as pres:
           all_slides = pres.slides
            
           # Клонируйте первый слайд (индекс 1) и вставьте его в индекс 2.
           all_slides.insert_clone(2, pres.slides[1])
            
           # Сохраните измененную презентацию
           pres.save(YOUR_OUTPUT_DIRECTORY + 'crud_add_clone2_out.pptx', slides.export.SaveFormat.PPTX)
   ```

   **Объясняемые параметры:**
   - `index`: Положение, куда будет вставлен клонированный слайд.
   - `slide_to_clone`: Слайд для копирования.

3. **Сохраните изменения**
   Сохраните презентацию с изменениями, используя `save` метод, указав желаемый формат (PPTX).

#### Клонирование слайда в конце презентации

**Обзор:**
Эта функция добавляет клонированный слайд в конец существующей презентации, что идеально подходит для добавления резюме или дополнительного контента.

##### Пошаговый процесс:

1. **Загрузите вашу презентацию**
   Начните с открытия файла PowerPoint, который вы собираетесь изменить.
   
   ```python
   def clone_slide_at_end():
       with slides.Presentation(YOUR_DOCUMENT_DIRECTORY + 'welcome-to-powerpoint.pptx') as pres:
           all_slides = pres.slides
   ```

2. **Клонировать и добавить в конец**
   Использовать `add_clone` метод дублирования слайда и его добавления.
   
   ```python
   def clone_slide_at_end():
       with slides.Presentation(YOUR_DOCUMENT_DIRECTORY + 'welcome-to-powerpoint.pptx') as pres:
           all_slides = pres.slides
            
           # Клонируйте слайд и добавьте его в конец презентации.
           cloned_slide = all_slides.add_clone(pres.slides[0])
            
           # Сохраните измененную презентацию
           pres.save(YOUR_OUTPUT_DIRECTORY + 'crud_add_clone_end_out.pptx', slides.export.SaveFormat.PPTX)
   ```

3. **Сохраните изменения**
   Использовать `save` для сохранения обновленного файла.

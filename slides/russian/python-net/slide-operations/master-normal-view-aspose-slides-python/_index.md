## Руководство по внедрению
Давайте разобьем реализацию на управляемые разделы, сосредоточившись на настройке обычных свойств представления.
### Настройка состояний горизонтальной и вертикальной полосы
#### Обзор
Настройка состояний разделительной полосы позволяет контролировать, как визуально структурирована ваша презентация в ее стандартном виде. Это включает в себя установку горизонтальных полос в восстановленное или свернутое состояние и соответствующую настройку вертикальных полос.
#### Этапы внедрения
1. **Установить состояние горизонтальной полосы**
   Восстановите состояние горизонтальной полосы для лучшей видимости нескольких слайдов:
   ```python
   pres.view_properties.normal_view_properties.horizontal_bar_state = slides.SplitterBarStateType.RESTORED
   ```
2. **Увеличить состояние вертикальной полосы**
   Чтобы просматривать больше контента по вертикали, разверните вертикальную панель на весь экран:
   ```python
   pres.view_properties.normal_view_properties.vertical_bar_state = slides.SplitterBarStateType.MAXIMIZED
   ```
### Настройка свойств верхней реставрации
#### Обзор
Отрегулируйте свойства верхнего восстановления, чтобы гарантировать, что определенные области слайда видны по умолчанию. Это полезно для немедленного представления определенного раздела.
#### Этапы внедрения
1. **Автоматическая настройка и установка размера**
   Включите автонастройку и укажите размер для восстановления:
   ```python
   pres.view_properties.normal_view_properties.restored_top.auto_adjust = True
   pres.view_properties.normal_view_properties.restored_top.dimension_size = 80
   ```
### Показать значки контура
#### Обзор
Отображение значков контуров облегчает навигацию, обеспечивая быстрый обзор структуры презентации.
#### Этапы внедрения
1. **Включить значки контура**
   Включите или выключите этот параметр, чтобы показать или скрыть значки контура:
   ```python
   pres.view_properties.normal_view_properties.show_outline_icons = True
   ```
### Сохранение презентации
Убедитесь, что все изменения сохранены правильно:
```python
pres.save("YOUR_OUTPUT_DIRECTORY/presentation_normal_view_state.pptx", slides.export.SaveFormat.PPTX)
```

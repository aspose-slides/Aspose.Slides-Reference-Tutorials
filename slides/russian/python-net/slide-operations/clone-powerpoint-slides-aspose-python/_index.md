### Клонирование слайда из одной презентации в другую

#### Обзор
Клонирование подразумевает копирование слайда из одной презентации и добавление его в другую. Это может быть особенно полезно, когда вам нужно повторно использовать контент без ручного дублирования слайдов.

#### Пошаговая реализация

##### 1. Загрузите исходную презентацию
Сначала откройте исходный файл презентации:
```python
with slides.Presentation("YOUR_DOCUMENT_DIRECTORY/welcome-to-powerpoint.pptx") as source_pres:
    # Дополнительные операции будут выполнены над `source_pres`
```

##### 2. Создайте новую презентацию назначения
Далее инициализируйте пустую целевую презентацию, в которую будет клонирован слайд:
```python
with slides.Presentation() as dest_pres:
    all_slides = dest_pres.slides
```

##### 3. Клонировать и добавить слайд
Откройте первый слайд исходной презентации и добавьте его в конец целевой презентации:
```python
all_slides.add_clone(source_pres.slides[0])
```

##### 4. Сохраните измененную презентацию.
Наконец, сохраните изменения в новом файле в желаемом выходном каталоге:
```python
dest_pres.save("YOUR_OUTPUT_DIRECTORY/crud_add_clone_out.pptx", slides.export.SaveFormat.PPTX)
```
**Примечание:** The `SaveFormat.PPTX` обеспечивает сохранение презентации в формате PowerPoint.

#### Советы по устранению неполадок
- Во избежание ошибок убедитесь, что пути к файлам указаны правильно.
- Проверьте, есть ли у вас права на запись в выходной каталог.

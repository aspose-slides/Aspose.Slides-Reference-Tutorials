## Руководство по внедрению

### Замена текста в слайдах PowerPoint

**Обзор:**
Эта функция позволяет автоматизировать процесс поиска и замены текста в заполнителях на слайде. Это особенно полезно для массового редактирования или стандартизации контента на нескольких слайдах.

#### Шаг 1: Загрузите презентацию
Начните с загрузки существующего файла PPTX:

```python
in_file_path = 'YOUR_DOCUMENT_DIRECTORY/text_default_fonts.pptx'

# Открыть презентацию с диска
with slides.Presentation(in_file_path) as pres:
    # Доступ к первому слайду презентации
    slide = pres.slides[0]
```

#### Шаг 2: Перебор фигур и замена текста
Пройдитесь по каждой фигуре на слайде, чтобы найти заполнители и заменить их текстовое содержимое:

```python
for shape in slide.shapes:
    if shape.placeholder is not None:
        # Заменить текст-заполнитель
        shape.text_frame.text = "This is Placeholder"
```

#### Шаг 3: Сохраните измененную презентацию.
После завершения изменений сохраните презентацию на диск:

```python
out_file_path = 'YOUR_OUTPUT_DIRECTORY/text_replacing_out.pptx'
pres.save(out_file_path, slides.export.SaveFormat.PPTX)
```

### Доступ к формам слайдов и их изменение

**Обзор:**
Узнайте, как получить доступ к различным фигурам на слайде и изменить их свойства, такие как цвет или стиль.

#### Шаг 1: Откройте презентацию
Откройте файл PPTX и выберите слайд, который вы хотите отредактировать:

```python
in_file_path = 'YOUR_DOCUMENT_DIRECTORY/example.pptx'

with slides.Presentation(in_file_path) as pres:
    slide = pres.slides[0]
```

#### Шаг 2: Измените свойства фигуры
Пройдитесь по каждой фигуре и определите, является ли она `AutoShape`и примените изменения, например, измените цвет заливки:

```python
for shape in slide.shapes:
    if isinstance(shape, slides.AutoShape):
        # Изменить цвет заливки на сплошной синий
        shape.fill_format.fill_type = slides.FillType.SOLID
        shape.fill_format.solid_fill_color.color = slides.Color.blue
```

#### Шаг 3: Сохраните обновленную презентацию.
Сохраните изменения в новом файле:

```python
out_file_path = 'YOUR_OUTPUT_DIRECTORY/shapes_modified_out.pptx'
pres.save(out_file_path, slides.export.SaveFormat.PPTX)
```

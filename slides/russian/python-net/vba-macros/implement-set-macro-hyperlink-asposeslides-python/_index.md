### Добавление фигур с помощью макрос-гиперссылок

#### Обзор
В этом разделе вы узнаете, как добавить форму кнопки на слайд PowerPoint и назначить событие щелчка макрос-гиперссылки, что имеет решающее значение для автоматизации задач в презентациях.

#### Пошаговая реализация

##### Добавить форму кнопки

Сначала мы добавим пустую форму кнопки на первый слайд в определенных координатах:

```python
import aspose.slides as slides

macro_name = "TestMacro"
with slides.Presentation() as presentation:
    # Добавление пустой формы кнопки на первый слайд
    shape = presentation.slides[0].shapes.add_auto_shape(
        slides.ShapeType.BLANK_BUTTON, 20, 20, 80, 30
    )
```
- **Параметры:**
  - `ShapeType.BLANK_BUTTON`: Указывает, что мы добавляем пустую кнопку.
  - `(20, 20, 80, 30)`: Координаты x, y, а также ширина и высота фигуры.

##### Установить макрос гиперссылки щелчок

Далее устанавливаем макрос-гиперссылку, нажимаем на добавленную фигуру:

```python
    # Назначение макрос-гиперссылки фигуре
    shape.hyperlink_manager.set_macro_hyperlink_click(macro_name)
```
- **Параметры:**
  - `macro_name`: Имя макроса, который будет запущен при нажатии кнопки.

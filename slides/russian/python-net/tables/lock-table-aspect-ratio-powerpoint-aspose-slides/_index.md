## Руководство по внедрению

Узнайте, как блокировать и разблокировать пропорции таблицы в PowerPoint с помощью Aspose.Slides для Python.

### Блокировка соотношения сторон таблицы (Функция: Блокировка соотношения сторон)

#### Обзор

Эта функция гарантирует, что изменение размера таблиц не искажает их форму, сохраняя визуальную согласованность на всех слайдах.

#### Пошаговая реализация

##### Доступ к презентации и таблице

Загрузите презентацию и получите доступ к таблице, которую вы хотите изменить:

```python
import aspose.slides as slides

def lock_aspect_ratio():
    with slides.Presentation('YOUR_DOCUMENT_DIRECTORY/tables.pptx') as pres:
        # Предположим, что первая фигура на первом слайде — это таблица.
        table = pres.slides[0].shapes[0]
```

##### Проверка текущего состояния блокировки соотношения сторон

Проверьте, включена ли уже блокировка соотношения сторон:

```python
print(f"Lock aspect ratio set: {table.shape_lock.aspect_ratio_locked}")
```

##### Включение/выключение блокировки соотношения сторон

Инвертировать текущее состояние блокировки соотношения сторон:

```python
table.shape_lock.aspect_ratio_locked = not table.shape_lock.aspect_ratio_locked
```

##### Сохранение изменений в презентации

Сохраните измененную презентацию:

```python
pres.save('YOUR_OUTPUT_DIRECTORY/tables_pres_lock_aspect_ratio_out.pptx', slides.export.SaveFormat.PPTX)
```

#### Советы по устранению неполадок
- Обеспечьте права доступа для чтения и записи файлов.
- Перед изменением убедитесь, что форма представляет собой таблицу.

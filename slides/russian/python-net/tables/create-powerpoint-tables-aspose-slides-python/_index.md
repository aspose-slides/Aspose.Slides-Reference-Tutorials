#### Создание таблицы на слайде

**Обзор**: Мы создадим простую таблицу и добавим ее на первый слайд презентации PowerPoint. 

##### Шаг 1: Создание экземпляра класса представления

The `Presentation` класс представляет собой файл PPT. Здесь мы откроем или создадим новую презентацию:

```python
with slides.Presentation() as pres:
    # Экземпляр представления используется в этом блоке менеджера контекста.
```

##### Шаг 2: Получите доступ к первому слайду

Доступ к первому слайду позволяет нам добавить туда нашу таблицу:

```python
slide = pres.slides[0]  # Это извлечет первый слайд из презентации.
```

##### Шаг 3: Определите размеры таблицы и добавьте ее на слайд

Определите ширину столбцов и высоту строк, затем добавьте таблицу в указанные координаты (x=50, y=50):

```python
dbl_cols = [50, 50, 50]  # Ширина столбцов
dbl_rows = [50, 30, 30, 30, 30]  # Высота рядов

table = slide.shapes.add_table(50, 50, dbl_cols, dbl_rows)  # Добавление таблицы на слайд.
```

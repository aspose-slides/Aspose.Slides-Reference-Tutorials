## Руководство по внедрению
Чтобы добавить изображение в ячейку таблицы на слайде PowerPoint, выполните следующие действия.
### Добавление изображений в ячейки таблицы
#### Обзор
Встраивайте изображения в определенные ячейки таблицы на слайдах PowerPoint, повышая визуальную привлекательность и ясность информации.
#### Пошаговая реализация
**1. Создайте экземпляр класса представления**
Создайте экземпляр `Presentation` сорт:
```python
with slides.Presentation() as presentation:
    slide = presentation.slides[0]
```
Откроется новый файл PowerPoint с одним слайдом по умолчанию.
**2. Определите размеры таблицы**
Настройте ширину столбцов и высоту строк для вашей таблицы с помощью списков:
```python
dbl_cols = [150, 150, 150, 150]  # Ширина столбцов
dbl_rows = [100, 100, 100, 100, 90]  # Высота рядов
```
**3. Добавьте новую таблицу на слайд**
Создайте и разместите таблицу на слайде:
```python	bl = slide.shapes.add_table(50, 50, dbl_cols, dbl_rows)
```
Это добавит таблицу в позицию (50, 50) с указанными размерами.
**4. Загрузите и вставьте изображение в презентацию**
Загрузите файл изображения, чтобы вставить его в ячейку таблицы:
```python
image = slides.Images.from_file('YOUR_DOCUMENT_DIRECTORY/image1.jpg')
imx1 = presentation.images.add_image(image)
```
Заменять `YOUR_DOCUMENT_DIRECTORY` с фактическим путем, где хранится ваше изображение.
**5. Установить изображение в ячейке таблицы**
Настройте первую ячейку таблицы для отображения изображения:
```python	bl.rows[0][0].cell_format.fill_format.fill_type = slides.FillType.PICTURE
	tbl.rows[0][0].cell_format.fill_format.picture_fill_format.picture_fill_mode = slides.PictureFillMode.STRETCH
	tbl.rows[0][0].cell_format.fill_format.picture_fill_format.picture.image = imgx1
```
Это растягивает изображение, чтобы оно поместилось в ячейку.
**6. Сохраните презентацию**
Наконец, сохраните презентацию с только что добавленной таблицей и изображением:
```python
presentation.save('YOUR_OUTPUT_DIRECTORY/tables_add_image_to_cell_out.pptx', slides.export.SaveFormat.PPTX)
```
Заменять `YOUR_OUTPUT_DIRECTORY` с желаемым выходным путем для вашего файла.

## Руководство по внедрению

### Определение объединенных ячеек в таблицах PowerPoint

#### Обзор

Эта функция сканирует каждую ячейку таблицы на слайде PowerPoint, чтобы проверить, является ли она частью объединенного набора, предоставляя сведения о ее диапазоне и начальной позиции.

#### Шаги для идентификации
1. **Загрузить презентацию**
   
   Загрузите файл презентации, в котором, по вашему мнению, могут находиться объединенные ячейки:
   ```python
   with slides.Presentation("YOUR_DOCUMENT_DIRECTORY/tables.pptx") as pres:
       # Доступ к первой фигуре на первом слайде (предполагается, что это таблица)
       table = pres.slides[0].shapes[0]
   ```

2. **Итерация по ячейкам**
   
   Пройдитесь по каждой ячейке, чтобы проверить статус объединения и собрать подробную информацию:
   ```python
   def dump_merged_cell(i, j, current_cell):
       # Распечатать информацию об объединенной ячейке
       print(f"Cell {i}{j} is part of a merged cell with row_span={current_cell.row_span}, col_span={current_cell.col_span}, starting from Cell {current_cell.first_row_index}{current_cell.first_column_index}.")
   
   for i, row in enumerate(table.rows):
       for j, cell in enumerate(row):
           if cell.is_merged_cell:
               dump_merged_cell(i, j, cell)
   ```

#### Объяснение
- **`is_merged_cell`:** Проверяет, является ли ячейка частью объединенного набора.
- **`row_span` и `col_span`:** Укажите, сколько строк или столбцов охватывает объединенная ячейка.
- **`first_row_index` и `first_column_index`:** Укажите начальную позицию слияния.

## Руководство по внедрению

### Установка первой строки в качестве заголовка

Автоматизируйте форматирование таблиц, отметив первую строку как заголовок, что часто требует специального оформления.

#### Шаг 1: Импорт необходимых модулей

Начните с импорта необходимых модулей:

```python
import os
from aspose.slides import Presentation, slides
```

#### Шаг 2: Определите пути к документам

Настройте пути для входных и выходных файлов:

```python
document_directory = "YOUR_DOCUMENT_DIRECTORY/"
output_directory = "YOUR_OUTPUT_DIRECTORY/"

tpptx_path = os.path.join(document_directory, 'tables.pptx')
```

#### Шаг 3: Загрузите презентацию

Откройте файл PowerPoint и откройте его первый слайд:

```python
with Presentation(pptx_path) as pres:
    slide = pres.slides[0]
```

#### Шаг 4: Перебор фигур для поиска таблиц

Просмотрите каждую фигуру на слайде, чтобы идентифицировать таблицы:

```python
for shape in slide.shapes:
    if isinstance(shape, slides.Table):
        # Отметить первую строку как заголовок
        shape.header_rows = 1  # Исправлен метод установки заголовков
```

#### Шаг 5: Сохраните измененную презентацию.

Сохраните изменения в новом файле:

```python
output_pptx_path = os.path.join(output_directory, 'tables_first_row_as_header_out.pptx')
pres.save(output_pptx_path, slides.export.SaveFormat.PPTX)
```

### Советы по устранению неполадок

- **Обеспечьте правильные пути**: Убедитесь, что каталоги документов и выходных данных указаны правильно.

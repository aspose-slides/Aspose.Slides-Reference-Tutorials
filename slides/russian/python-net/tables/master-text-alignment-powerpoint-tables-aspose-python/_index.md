## Руководство по внедрению

Мы разобьем процесс вертикального выравнивания текста в ячейках таблицы на удобные для выполнения этапы.

### Доступ к слайду и добавление таблицы

Сначала нам нужно получить доступ к слайду и определить размеры нашей таблицы:

```python
with slides.Presentation() as presentation:
    slide = presentation.slides[0]
    dbl_cols = [120, 120, 120, 120]
    dbl_rows = [100, 100, 100, 100]

    # Добавьте таблицу на слайд.
    tbl = slide.shapes.add_table(100, 50, dbl_cols, dbl_rows)
```

### Вставка и выравнивание текста

Далее вставьте текст в ячейки и примените вертикальное выравнивание:

```python
# Вставьте текст в определенные ячейки.
tbl.rows[1][0].text_frame.text = "10"
tbl.rows[2][0].text_frame.text = "20"
tbl.rows[3][0].text_frame.text = "30"

# Для изменения свойств перейдите к текстовому фрейму первой ячейки.
text_frame = tbl.rows[0][0].text_frame
paragraph = text_frame.paragraphs[0]
portion = paragraph.portions[0]

# Задайте текст и стиль для этой части.
portion.text = "Text here"
portion.portion_format.fill_format.fill_type = slides.FillType.SOLID
portion.portion_format.fill_format.solid_fill_color.color = drawing.Color.black

# Выровняйте текст по вертикали.
cell = tbl.rows[0][0]
cell.text_anchor_type = slides.TextAnchorType.CENTER
cell.text_vertical_type = slides.TextVerticalType.VERTICAL270
```

### Сохранение презентации

Наконец, сохраните измененную презентацию:

```python
presentation.save("YOUR_OUTPUT_DIRECTORY/tables_vertical_align_text_out.pptx", slides.export.SaveFormat.PPTX)
```

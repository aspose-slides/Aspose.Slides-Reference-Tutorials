## Руководство по внедрению

### Функция: Встроенное сжатие шрифтов

Эта функция помогает уменьшить размер файла презентации PowerPoint путем сжатия встроенных шрифтов. Давайте рассмотрим, как реализовать это шаг за шагом.

#### Загрузить презентацию

Начните с загрузки существующего файла PowerPoint, содержащего встроенные шрифты:
```java
// Путь к исходной презентации со встроенными шрифтами
String presentationName = "YOUR_DOCUMENT_DIRECTORY/presWithEmbeddedFonts.pptx";

// Загрузить презентацию
Presentation pres = new Presentation(presentationName);
```

#### Сжать встроенные шрифты

Используйте `Compress.compressEmbeddedFonts` Метод сжатия шрифтов в презентации:
```java
try {
    // Сжимайте встроенные шрифты, чтобы уменьшить размер файла
    Compress.compressEmbeddedFonts(pres);
} finally {
    if (pres != null) pres.dispose();
}
```

#### Сохраните измененную презентацию

После сжатия сохраните измененную презентацию в новый файл:
```java
// Путь, по которому будет сохранена сжатая презентация
String outPath = "YOUR_OUTPUT_DIRECTORY/presWithEmbeddedFonts-out.pptx";

// Сохраните измененную презентацию
pres.save(outPath, SaveFormat.Pptx);
```

### Советы по устранению неполадок

- Убедитесь, что путь к входному файлу PowerPoint указан правильно.
- Убедитесь, что у вас есть права на запись в выходной каталог.
- Проверьте наличие исключений, возникающих во время сжатия, и обработайте их соответствующим образом.

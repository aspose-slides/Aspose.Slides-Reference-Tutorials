## Руководство по внедрению

Теперь, когда мы настроили Aspose.Slides, давайте перейдем к реализации наших двух основных функций: удалению неиспользуемых мастер-слайдов и макетов слайдов.

### Удаление неиспользуемых мастер-слайдов

#### Обзор
Эта функция поможет вам навести порядок в презентациях PowerPoint, удалив все неиспользуемые мастер-слайды, что позволит существенно сократить размер файла и повысить производительность.

**Пошаговая реализация**

##### Загрузить презентацию
Начните с загрузки презентации по указанному пути:
```java
String pptxFileName = "YOUR_DOCUMENT_DIRECTORY/MultipleMaster.pptx";
Presentation pres = new Presentation(pptxFileName);
```

##### Удалить неиспользуемые мастер-слайды
Используйте `Compress` класс для удаления неиспользуемых мастер-слайдов:
```java
try {
    Compress.removeUnusedMasterSlides(pres);
} finally {
    if (pres != null) pres.dispose();
}
```
*Объяснение:* Метод `removeUnusedMasterSlides()` просматривает все мастер-слайды и оставляет только те, которые используются.

### Удаление неиспользуемых слайдов макета

#### Обзор
Подобно мастер-слайдам, слайды макета также могут раздуть вашу презентацию. Эта функция гарантирует, что останутся только необходимые слайды макета.

**Пошаговая реализация**

##### Загрузить презентацию
Убедитесь, что вы загрузили презентацию так же, как и раньше:
```java
String pptxFileName = "YOUR_DOCUMENT_DIRECTORY/MultipleMaster.pptx";
Presentation pres = new Presentation(pptxFileName);
```

##### Удалить неиспользуемые слайды макета
Используйте `Compress` класс для устранения неиспользуемых слайдов макета:
```java
try {
    Compress.removeUnusedLayoutSlides(pres);
} finally {
    if (pres != null) pres.dispose();
}
```
*Объяснение:* Метод `removeUnusedLayoutSlides()` работает аналогичным образом, удаляя макеты, не связанные ни с одним слайдом.

### Советы по устранению неполадок
- Убедитесь, что путь к файлу PowerPoint указан правильно.
- При возникновении ошибок убедитесь, что вы используете совместимую версию JDK, как указано в зависимости Aspose.Slides.

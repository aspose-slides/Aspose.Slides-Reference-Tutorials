## Руководство по внедрению

### Функция 1: Создание новой презентации

**Обзор:** В этом разделе вы узнаете, как создать новую презентацию PowerPoint с помощью Aspose.Slides. Вы узнаете, как добавить слайд и автофигуру текстовой линии.

#### Пошаговые инструкции

**1. Создание объекта презентации**
Начните с создания экземпляра `Presentation` класс, представляющий ваш файл PowerPoint.

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Заполнитель для пути к каталогу документа
Presentation presentation = new Presentation();
```

**2. Доступ к слайдам и их изменение**
Извлеките слайд по умолчанию, созданный при создании экземпляра, и добавьте форму линии.

```java
try {
    ISlide slide = presentation.getSlides().get_Item(0); // Доступ к первому слайду
    slide.getShapes().addAutoShape(ShapeType.Line, 50, 150, 300, 0); // Добавьте на слайд форму линии
```

**3. Сохраните презентацию**
Наконец, сохраните презентацию в формате PPTX.

```java
presentation.save(dataDir + "NewPresentation_out.pptx", SaveFormat.Pptx); // Сохранить презентацию
} finally {
    if (presentation != null) presentation.dispose();
}
```

### Функция 2: Манипуляция автофигурами

**Обзор:** В этом разделе рассматривается добавление различных автофигур на слайд, демонстрируя гибкость Aspose.Slides в настройке ваших презентаций.

#### Пошаговые инструкции

**1. Создание и доступ к презентации**
Как и в первой функции, начните с настройки объекта презентации.

```java
Presentation presentation = new Presentation();
```

**2. Добавьте различные автофигуры**
Добавьте прямоугольник и эллипс, чтобы проиллюстрировать универсальность форм.

```java
try {
    ISlide slide = presentation.getSlides().get_Item(0); // Доступ к первому слайду

    // Добавить прямоугольник
    slide.getShapes().addAutoShape(ShapeType.Rectangle, 100, 100, 200, 100);

    // Добавить эллипс
    slide.getShapes().addAutoShape(ShapeType.Ellipse, 350, 150, 150, 75);
```

**3. Сохраните презентацию**
Обязательно сохраните изменения в файле.

```java
presentation.save(dataDir + "AutoshapesExample_out.pptx", SaveFormat.Pptx); // Сохраните измененную презентацию
} finally {
    if (presentation != null) presentation.dispose();
}
```

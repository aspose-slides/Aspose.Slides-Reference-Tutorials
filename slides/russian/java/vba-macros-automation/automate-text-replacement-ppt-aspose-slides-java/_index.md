## Руководство по внедрению

Теперь давайте разберем реализацию на удобные для выполнения разделы, чтобы подробно рассмотреть каждую функцию.

### Функция 1: Поиск и замена текста

Эта основная функция позволяет автоматизировать замену текста на всех слайдах презентации.

#### Шаг 1: Загрузка презентации
Начните с загрузки файла PPTX с помощью Aspose.Slides.
```java
Presentation pres = new Presentation("YOUR_DOCUMENT_DIRECTORY/TextReplaceExample.pptx");
```

#### Шаг 2: Реализация логики поиска и замены
Используйте `replaceText` метод поиска определенных текстовых шаблонов и их замены. Здесь мы заменяем вхождения "[этот блок]" на "мой текст".
```java
pres.replaceText("\\[this block\\]", "my text", new TextSearchOptions(), callback);
```

#### Шаг 3: Сохраните изменения.
После выполнения замены сохраните обновленную презентацию.
```java
pres.save("YOUR_OUTPUT_DIRECTORY/TextReplaceExampleReplace-out.pptx", SaveFormat.Pptx);
```

### Функция 2: Реализация FindResultCallback

Эта функция предназначена для отслеживания и обработки результатов текстового поиска во время замен.

#### Обзор
Создайте класс обратного вызова, реализующий `IFindResultCallback` для сбора подробностей о каждом вхождении искомого текста.

#### Шаг 1: Определите класс обратного вызова
Реализуйте методы управления найденными результатами, например, сохранение информации о словах в списке.
```java
class FindResultCallback implements IFindResultCallback {
    private List<WordInfo> Words = new ArrayList<>();

    @Override
    public void foundResult(ITextFrame textFrame, String oldText, String foundText, int textPosition) {
        Words.add(new WordInfo(textFrame, oldText, foundText, textPosition));
    }
}
```

#### Шаг 2: Извлечение результатов поиска
Реализуйте методы для доступа к количеству совпадений и их местоположению.
```java
public Integer[] getSlideNumbers() {
    List<Integer> slideNumbers = new ArrayList<>();
    for (WordInfo element : Words) {
        int slideNumber = ((ISlide)element.getTextFrame().getSlide()).getSlideNumber();
        if (!slideNumbers.contains(slideNumber))
            slideNumbers.add(slideNumber);
    }
    return slideNumbers.toArray(new Integer[0]);
}
```

### Функция 3: Класс WordInfo

Этот служебный класс хранит сведения о каждом вхождении текста, найденном во время поиска.

#### Обзор
Определите `WordInfo` класс для инкапсуляции данных, связанных с найденными текстами, таких как их источник и положение на слайдах.

#### Шаг 1: Создание класса WordInfo
Инициализируйте такие свойства, как `TextFrame`, `SourceText`, и `FoundText`.
```java
class WordInfo {
    private final ITextFrame TextFrame;
    private final String SourceText;
    private final String FoundText;
    private final int TextPosition;

    public WordInfo(ITextFrame textFrame, String sourceText, String foundText, int textPosition) {
        this.TextFrame = textFrame;
        this.SourceText = sourceText;
        this.FoundText = foundText;
        this.TextPosition = textPosition;
    }
}
```

## Настройка Aspose.Slides для Java

Настройка Aspose.Slides проста и может быть выполнена различными способами. Вот как вы можете добавить его в свой проект:

### Настройка Maven
Добавьте следующую зависимость к вашему `pom.xml` файл:
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

### Настройка Gradle
Включите в свой план следующее: `build.gradle` файл:
```gradle
implementation group: 'com.aspose', name: 'aspose-slides', version: '25.4', classifier: 'jdk16'
```

### Прямая загрузка
Либо загрузите последнюю версию с сайта [Aspose.Slides для релизов Java](https://releases.aspose.com/slides/java/).

#### Этапы получения лицензии

- **Временная лицензия**: Получите временную лицензию, если вам нужны более широкие возможности тестирования.
- **Покупка**: Для доступа ко всем функциям рассмотрите возможность приобретения лицензии.

Presentation pres = new Presentation();

## Руководство по внедрению

В этом разделе мы рассмотрим, как реализовать и извлечь макроссылки в приложениях Java с помощью Aspose.Slides.

### Добавление макрос-гиперссылки в автофигуру

**Обзор**: Эта функция позволяет добавлять интерактивные функции к фигурам в презентации. Когда пользователи нажимают на фигуру, она может запускать определенные действия или макросы, повышая вовлеченность пользователей.

#### Шаг 1: Откройте первый слайд
Начните с открытия первого слайда вашей презентации.
```java
var slide = pres.getSlides().get_Item(0);
```

#### Шаг 2: Добавьте фигуру на слайд
Создайте AutoShape на слайде. Здесь мы добавляем пустую форму кнопки в позицию (20, 20) с размерами 80x30.
```java
IAutoShape shape = slide.getShapes().addAutoShape(ShapeType.BlankButton, 20, 20, 80, 30);
```

#### Шаг 3: Установите макрос гиперссылки
Назначьте макрос-гиперссылку для вашей фигуры. Эта гиперссылка запускает указанный макрос (`macroName`) при щелчке по форме.
```java
shape.getHyperlinkManager().setMacroHyperlinkClick("TestMacro");
```

**Почему**: Установка макрос-гиперссылки позволяет выполнять определенный код при взаимодействии, делая презентации более интерактивными и автоматизированными.

### Извлечение информации о гиперссылке из формы

**Обзор**Понимание того, как извлекать информацию о гиперссылках, гарантирует, что вы сможете эффективно управлять ссылками и отлаживать их.

#### Шаг 1: Откройте первый слайд
Повторно используйте первую настройку слайда:
```java
var slide = pres.getSlides().get_Item(0);
```

#### Шаг 2: Добавьте и установите макрос-гиперссылку
Как и прежде, добавьте фигуру и установите ее макрос-гиперссылку.
```java
IAutoShape shape = slide.getShapes().addAutoShape(ShapeType.BlankButton, 20, 20, 80, 30);
shape.getHyperlinkManager().setMacroHyperlinkClick("TestMacro");
```

#### Шаг 3: Получите внешний URL-адрес
Извлекайте и отображайте любые внешние URL-адреса, связанные с гиперссылкой вашей фигуры.
```java
String externalUrl = shape.getHyperlinkClick().getExternalUrl();
System.out.println("External URL is " + externalUrl);
```
**Почему**: Этот шаг позволяет вам проверять или регистрировать URL-адреса, связанные с гиперссылками, для устранения неполадок или ведения учета.

#### Шаг 4: Определите тип действия
Определите и распечатайте тип действия гиперссылки вашей фигуры.
```java
String actionType = shape.getHyperlinkClick().getActionType();
System.out.println("Shape action type is " + actionType);
```
**Почему**: Знание типа действия помогает понять, как управляются взаимодействия с пользователем.

# Конвертируйте PPT в HTML/PDF с пользовательскими шрифтами с помощью Aspose.Slides для Java

Добро пожаловать в это всеобъемлющее руководство по использованию Aspose.Slides для Java для преобразования презентаций PowerPoint в форматы HTML и PDF с указанием обычного шрифта по умолчанию. Независимо от того, стремитесь ли вы к единообразной типографике на всех платформах или хотите оптимизировать рабочий процесс управления документами, это руководство поможет вам без усилий использовать мощь Aspose.Slides.

Преобразование файлов PowerPoint часто может привести к несоответствию шрифтов в выходных документах, что является проблемой при профессиональном представлении данных. С Aspose.Slides для Java мы решаем эту проблему, устанавливая обычный шрифт по умолчанию во время процессов преобразования. В этом руководстве вы узнаете, как сохранять презентации в формате HTML и PDF с указанными шрифтами с помощью Aspose.Slides.

**Что вы узнаете:**
- Как настроить Aspose.Slides для Java
- Действия по преобразованию файлов PowerPoint в HTML с указанием обычного шрифта по умолчанию
- Методы экспорта презентаций в формат PDF с сохранением единообразия типографики

Давайте начнем с обзора предварительных условий, прежде чем углубляться в руководство по внедрению.

Включите библиотеку Aspose.Slides в свой проект. Убедитесь, что Maven или Gradle настроены в вашей среде разработки.

### Необходимые знания

Для эффективного освоения материала рекомендуется иметь базовые знания программирования на Java и навыки работы с инструментами сборки Maven/Gradle.

## Настройка Aspose.Slides для Java

Чтобы начать использовать Aspose.Slides, включите его в зависимости вашего проекта. Вот как:

**Мейвен:**
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

**Градл:**
```gradle
implementation group: 'com.aspose', name: 'aspose-slides', version: '25.4', classifier: 'jdk16'
```

**Прямая загрузка:**
Для ручной настройки загрузите последнюю версию с сайта [Aspose.Slides для релизов Java](https://releases.aspose.com/slides/java/).

## Руководство по внедрению

В этом разделе вы узнаете, как преобразовать презентации PowerPoint, сохранив при этом единообразие шрифтов.

### Сохранение презентации в формате HTML с обычным шрифтом по умолчанию

Конвертация презентации в формат HTML позволяет просматривать ее в любом веб-браузере, обеспечивая более широкую доступность. Вот как установить обычный шрифт по умолчанию для этого преобразования:

#### Шаг 1: Инициализация объекта презентации
Загрузите файл PowerPoint с помощью `Presentation` сорт.
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY/DefaultFonts.pptx"; // Замените на путь к каталогу вашего документа.
Presentation pres = new Presentation(dataDir);
```

#### Шаг 2: Настройка параметров HTML
Настраивать `HtmlOptions`, указав обычный шрифт по умолчанию, который вы хотите использовать в экспортируемом HTML-файле.
```java
HtmlOptions htmlOpts = new HtmlOptions();
htmlOpts.setDefaultRegularFont("Arial Black"); // Установите желаемый шрифт
```

#### Шаг 3: Сохранить как HTML
Наконец, сохраните презентацию, используя настроенные параметры:
```java
String outPath = "YOUR_OUTPUT_DIRECTORY/";
pres.save(outPath + "Presentation-out-ArialBlack.html", SaveFormat.Html, htmlOpts);
```
При необходимости повторите эти шаги с другим шрифтом.

### Сохранение презентации в формате PDF с обычным шрифтом по умолчанию
Экспорт в PDF гарантирует, что ваши презентации могут быть распространены в универсально совместимом формате. Вот как вы можете указать стандартный шрифт по умолчанию для преобразования в PDF:

#### Шаг 1: Инициализация PdfOptions
Подобно HTML, начните с настройки `PdfOptions`.
```java
PdfOptions pdfOpts = new PdfOptions();
pdfOpts.setDefaultRegularFont("Arial Black"); // Здесь также установите желаемый шрифт.
```

#### Шаг 2: Сохранить как PDF
Экспортируйте презентацию с помощью следующих параметров:
```java
pres.save(outPath + "Presentation-out-ArialBlack.pdf", SaveFormat.Pdf, pdfOpts);
```

### Создание и уничтожение объекта презентации
**Обзор:** Научитесь эффективно создавать и управлять жизненным циклом объектов презентации для экономии ресурсов.

#### Создание экземпляра
```java
import com.aspose.slides.Presentation;

class Feature1 {
    public static void main(String[] args) {
        // Создайте экземпляр класса Presentation, представляющий файл PPT.
        Presentation presentation = new Presentation();
        try {
            // Выполняйте операции над презентацией здесь...
        } finally {
            // Удалить объект Presentation, чтобы освободить ресурсы.
            if (presentation != null) presentation.dispose();
        }
    }
}
```
**Объяснение:** Создание экземпляра `Presentation` инициализирует представление файла PPT в памяти. `try-finally` блок обеспечивает высвобождение ресурсов, предотвращая утечки памяти.

### Установить защиту свойств документа
**Обзор:** Защитите свойства вашего документа с помощью паролей или без них.

#### Включение/отключение шифрования
```java
import com.aspose.slides.Presentation;

class Feature2 {
    public static void main(String[] args) {
        Presentation presentation = new Presentation();
        try {
            // Включить или отключить шифрование свойств документа
            presentation.getProtectionManager().setEncryptDocumentProperties(false);
        } finally {
            if (presentation != null) presentation.dispose();
        }
    }
}
```
**Объяснение:** The `ProtectionManager` класс позволяет включать и выключать шифрование свойств документа, обеспечивая уровень безопасности.

### Зашифровать презентацию паролем
**Обзор:** Защитите всю свою презентацию, зашифровав ее паролем.

#### Установка шифрования
```java
import com.aspose.slides.Presentation;

class Feature3 {
    public static void main(String[] args) {
        Presentation presentation = new Presentation();
        try {
            // Установите пароль для шифрования свойств документа
            presentation.getProtectionManager().encrypt("pass");
        } finally {
            if (presentation != null) presentation.dispose();
        }
    }
}
```
**Объяснение:** Шифрование с помощью пароля гарантирует, что только авторизованные пользователи смогут получить доступ к вашей презентации или изменить ее.

### Сохранить презентацию в файл
**Обзор:** Узнайте, как сохранять презентации в разных форматах, обеспечивая гибкость и совместимость.

#### Сохранение презентации
```java
import com.aspose.slides.Presentation;
import com.aspose.slides.SaveFormat;

class Feature4 {
    public static void main(String[] args) {
        Presentation presentation = new Presentation();
        try {
            // Определите путь к выходному каталогу для сохранения файлов.
            String dataDir = "YOUR_DOCUMENT_DIRECTORY";

            // Сохраните презентацию в файл в формате Pptx
            presentation.save(dataDir + "/Password Protected Presentation_out.pptx", SaveFormat.Pptx);
        } finally {
            if (presentation != null) presentation.dispose();
        }
    }
}
```
**Объяснение:** The `save` Метод записывает вашу презентацию на диск. `SaveFormat` enum определяет желаемый формат, предлагая универсальность в параметрах вывода.

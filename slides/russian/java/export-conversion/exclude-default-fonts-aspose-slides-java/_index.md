## Предпосылки
Чтобы эффективно следовать этому руководству, убедитесь, что у вас есть:
- **Библиотека Aspose.Slides для Java**: Установите версию 25.4 или более позднюю.

## Руководство по внедрению
### Обзор функций: исключение шрифтов по умолчанию из преобразования HTML
Эта функция помогает настраивать обработку шрифтов во время преобразования файла PowerPoint в HTML, улучшая фирменный стиль и единообразие.
#### Шаг 1: Подготовьте окружающую среду
Убедитесь, что Aspose.Slides правильно настроен в соответствии с инструкциями выше. Это включает добавление зависимостей или загрузку JAR непосредственно в ваш проект.
#### Шаг 2: Загрузите презентацию
Загрузите вашу презентацию с помощью `Presentation` сорт:
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY/presentation.pptx";
try {
    Presentation pres = new Presentation(dataDir);
```
#### Шаг 3: Определите исключения шрифтов
Создайте массив для указания шрифтов, которые вы хотите исключить. В этом примере мы начинаем с пустого списка в качестве заполнителя:
```java
String[] fontNameExcludeList = {};
```
#### Шаг 4: Инициализация пользовательского HTML-контроллера
The `LinkAllFontsHtmlController` класс используется для пользовательской обработки шрифтов в процессе конвертации.
```java
LinkAllFontsHtmlController linkcont = new LinkAllFontsHtmlController(fontNameExcludeList, "YOUR_DOCUMENT_DIRECTORY");
```
#### Шаг 5: Настройка параметров HTML
Настройте свой `HtmlOptions` для использования пользовательского форматировщика:
```java
HtmlOptions htmlOptionsEmbed = new HtmlOptions();
htmlOptionsEmbed.setHtmlFormatter(HtmlFormatter.createCustomFormatter(linkcont));
```
#### Шаг 6: Сохранить как HTML
Наконец, сохраните преобразованную презентацию в формате HTML:
```java
pres.save("YOUR_OUTPUT_DIRECTORY/pres.html", SaveFormat.Html, htmlOptionsEmbed);
} catch (Exception e) {
    e.printStackTrace();
}
```

Чтобы включить библиотеку Aspose.Slides в свой проект, вы можете использовать Maven, Gradle или прямую загрузку:

### **Знаток**
Добавьте следующую зависимость к вашему `pom.xml` файл:
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

Включите это в свой `build.gradle` файл:
```gradle

```

## Руководство по внедрению

Теперь давайте рассмотрим процесс преобразования презентации PowerPoint в адаптивный формат HTML с помощью Aspose.Slides для Java.

### Создать объект презентации

Начните с создания экземпляра `Presentation` класс. Этот объект представляет ваш файл PowerPoint.

```java
// Создать новый объект презентации из указанного пути к файлу PowerPoint
title = "YOUR_DOCUMENT_DIRECTORY/Convert_HTML.pptx";
Presentation presentation = new Presentation(title);
```

Заменять `"YOUR_DOCUMENT_DIRECTORY/Convert_HTML.pptx"` с фактическим путем к вашему файлу PowerPoint. `Presentation` класс служит контейнером для всех слайдов и их элементов.

### Создать адаптивный HTML-контроллер

Далее, настройте `ResponsiveHtmlController`Этот контроллер будет определять, как ваша презентация будет адаптироваться к разным размерам экрана.

```java
// Инициализируйте экземпляр ResponsiveHtmlController
ResponsiveHtmlController controller = new ResponsiveHtmlController();
```
The `ResponsiveHtmlController` использование медиазапросов CSS гарантирует, что преобразованный HTML-код будет гибким и визуально единообразным на различных устройствах.

### Настройка параметров HTML

Настройте `HtmlOptions` чтобы указать, как должно обрабатываться преобразование. Вот где вы определяете использование пользовательского форматировщика:

```java
// Определите HtmlOptions с помощью пользовательского форматировщика на основе ResponsiveHtmlController
HtmlOptions htmlOptions = new HtmlOptions();
htmlOptions.setHtmlFormatter(HtmlFormatter.createCustomFormatter(controller));
```

Этот шаг настраивает `htmlOptions` использовать адаптивный контроллер, гарантируя, что выходной HTML-код будет адаптивным.

### Сохранить презентацию как адаптивный HTML

Наконец, сохраните вашу презентацию в адаптивном формате HTML:

```java
try {
    // Конвертируйте и сохраните презентацию в HTML-файл с адаптивными настройками.
title = "YOUR_OUTPUT_DIRECTORY/ConvertPresentationToResponsiveHTML_out.html";
presentation.save(title, SaveFormat.Html, htmlOptions);
} finally {
    if (presentation != null) presentation.dispose();
}
```

Этот фрагмент кода сохраняет ваш файл PowerPoint как HTML-документ в указанном каталоге. `dispose()` Метод необходим для высвобождения ресурсов после завершения преобразования.

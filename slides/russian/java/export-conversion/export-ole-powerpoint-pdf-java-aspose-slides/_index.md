## Руководство по внедрению
### Экспорт объектов OLE из PPTX в PDF
Эта функция фокусируется на экспорте презентаций PowerPoint со встроенными объектами OLE в формат PDF с сохранением встроенных данных. Вот как этого можно добиться:

#### Шаг 1: Загрузите презентацию
Загрузите файл презентации с помощью `Presentation` сорт.

```java
import com.aspose.slides.Presentation;

String dataDir = "YOUR_DOCUMENT_DIRECTORY";
Presentation pres = new Presentation(dataDir + "/PresOleExample.pptx");
```

**Почему?**: Загрузка презентации инициализирует объектную модель, которой будет управлять Aspose.Slides.

#### Шаг 2: Настройте параметры экспорта PDF
Настраивать `PdfOptions` для включения данных OLE в ваш экспорт.

```java
import com.aspose.slides.PdfOptions;

PdfOptions options = new PdfOptions();
options.setIncludeOleData(true);
```

**Почему?**: `setIncludeOleData(true)` обеспечивает сохранение встроенных объектов OLE во время преобразования, поддерживая целостность данных.

#### Шаг 3: Экспорт в PDF
Сохраните презентацию в формате PDF-файла с указанными параметрами.

```java
import com.aspose.slides.SaveFormat;

String outFilePath = "YOUR_OUTPUT_DIRECTORY/PresOleExample.pdf";
pres.save(outFilePath, SaveFormat.Pdf, options);
```

**Почему?**: Этот шаг выполняет преобразование и сохраняет выходной PDF-файл в выбранном вами каталоге. Указав `SaveFormat.Pdf`, вы даете указание Aspose.Slides создать PDF-файл.

### Советы по устранению неполадок
- Убедитесь, что ваш путь презентации правильный, чтобы избежать `FileNotFoundException`.
- Убедитесь, что вы правильно настроили лицензию, чтобы предотвратить появление водяных знаков в выходных данных.
- Если данные OLE не отображаются, проверьте еще раз. `options.setIncludeOleData(true);` конфигурация.

## Настройка Aspose.Slides для Java
Интегрируйте Aspose.Slides в свой проект с помощью Maven или Gradle:

### Установка Maven
Добавьте следующую зависимость к вашему `pom.xml` файл:
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

### Установка Gradle
Включите это в свой `build.gradle` файл:
```gradle
implementation group: 'com.aspose', name: 'aspose-slides', version: '25.4', classifier: 'jdk16'
```

### Прямая загрузка
Либо загрузите библиотеку Aspose.Slides для Java напрямую с [Aspose выпускает](https://releases.aspose.com/slides/java/).

### Базовая инициализация и настройка
Инициализируйте Aspose.Slides в вашем приложении Java следующим образом:

## Руководство по внедрению
Теперь давайте преобразуем презентацию PowerPoint в HTML со встроенными шрифтами с помощью Aspose.Slides.

### Обзор
Эта функция помогает сохранять стили шрифтов во время конвертации для единообразного отображения контента на разных платформах и в разных браузерах.

#### Шаг 1: Настройка среды
Убедитесь, что ваш проект настроен правильно, как описано ранее.

#### Шаг 2: Загрузка презентации
Загрузите ваш файл PowerPoint в `Presentation` объект:
```java
import com.aspose.slides.Presentation;

// Определить входной путь
String dataDir = "YOUR_DOCUMENT_DIRECTORY/input.pptx";

// Загрузить презентацию
Presentation pres = new Presentation(dataDir);
```

#### Шаг 3: Настройка внедрения шрифтов
Используйте встроенный контроллер шрифтов, чтобы указать шрифты для встраивания:
```java
import com.aspose.slides.EmbedAllFontsHtmlController;
import com.aspose.slides.HtmlOptions;
import com.aspose.slides.HtmlFormatter;

// Определите шрифты, которые необходимо исключить из встраивания
String[] fontNameExcludeList = {"Calibri", "Arial"};

// Создать контроллер для встраивания пользовательских шрифтов
EmbedAllFontsHtmlController embedFontsController = new EmbedAllFontsHtmlController(fontNameExcludeList);

// Настройка параметров HTML с помощью контроллера шрифтов
HtmlOptions htmlOptionsEmbed = new HtmlOptions();
htmlOptionsEmbed.setHtmlFormatter(HtmlFormatter.createCustomFormatter(embedFontsController));
```

#### Шаг 4: Сохранение как HTML
Сохраните презентацию как HTML-файл:
```java
import com.aspose.slides.SaveFormat;

// Определить выходной путь
String outputDir = "YOUR_OUTPUT_DIRECTORY/output.html";

// Сохраните презентацию со встроенными шрифтами
pres.save(outputDir, SaveFormat.Html, htmlOptionsEmbed);
```

#### Советы по устранению неполадок
- Убедитесь, что все пути указаны правильно.
- Убедитесь, что ваша версия JDK соответствует классификатору Aspose.Slides.
- Если шрифты не отображаются должным образом, проверьте список исключений и доступность шрифтов в вашей системе.

Теперь, когда у вас все настроено, давайте реализуем функцию заливки изображения со смещением и растяжением в PowerPoint с помощью Aspose.Slides для Java.

### Обзор: Добавление изображения с помощью смещения растяжения
Эта функция позволяет динамически добавлять изображения на слайды с эффектом растяжения, повышая визуальную привлекательность и делая презентации более интересными.

#### Шаг 1: Инициализация презентации и загрузка изображения
Начните с создания нового экземпляра презентации и загрузки изображения:

```java
// Экземпляр класса презентации
Presentation pres = new Presentation();
try {
    // Получить первый слайд
    ISlide sld = pres.getSlides().get_Item(0);

    // Определите пути к каталогам для документа и вывода
    String dataDir = "YOUR_DOCUMENT_DIRECTORY";  // Путь к вашим файлам изображений

    // Загрузить изображение в объект IImage
    IImage img = Images.fromFile(dataDir + "/aspose-logo.jpg");
```

#### Шаг 2: Добавьте изображение на слайд
Далее добавьте изображение в виде рамки с определенными размерами:

```java
    // Добавить изображение в коллекцию изображений презентации
    IPPImage imgx = pres.getImages().addImage(img);

    // Добавить фоторамку с указанными размерами
    sld.getShapes().addPictureFrame(ShapeType.Rectangle, 50, 150, imgx.getWidth(), imgx.getHeight(), imgx);
```

#### Шаг 3: Сохраните презентацию
Наконец, сохраните презентацию, чтобы применить изменения:

```java
    // Определите выходной каталог и сохраните презентацию
    String outputDir = "YOUR_OUTPUT_DIRECTORY";
    pres.save(outputDir + "/AddStretchOffsetForImageFill_out.pptx", SaveFormat.Pptx);
} finally {
    if (pres != null) pres.dispose();
}
```

### Советы по устранению неполадок
- **Отсутствует изображение**: Убедитесь, что путь к файлу изображения указан правильно.
- **Проблемы с памятью**: Утилизировать `Presentation` экземпляры правильно с блоком try-finally.

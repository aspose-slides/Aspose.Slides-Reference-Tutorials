## Руководство по внедрению

Мы реализуем две основные функции: создание каталогов и шифрование презентаций.

### Создать каталог

#### Обзор

Эта функция проверяет наличие существующего каталога и при необходимости создает его, гарантируя, что файлы презентации будут хорошо организованы.

#### Шаги по реализации

##### Шаг 1: Импорт необходимых пакетов

```java
import java.io.File;
```

##### Шаг 2: Определите путь к каталогу

Укажите путь к месту хранения презентаций:

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
```

##### Шаг 3: Проверка и создание каталога

Проверьте, существует ли каталог, и создайте его, если нет:

```java
boolean isExists = new File(dataDir).exists();
if (!isExists) {
    new File(dataDir).mkdirs(); // Создает каталог со всеми необходимыми родительскими каталогами.
}
```

#### Объяснение

- **`File.exists()`**: Возврат `true` если файл или каталог существует.
- **`File.mkdirs()`**: Создает каталог, включая все необходимые, но несуществующие родительские каталоги.

### Загрузите и сохраните презентацию с шифрованием пароля

#### Обзор

Зашифруйте свою презентацию, чтобы защитить ее содержимое от несанкционированного доступа с помощью пароля.

#### Шаги по реализации

##### Шаг 1: Импорт пакетов Aspose.Slides

```java
import com.aspose.slides.Presentation;
import com.aspose.slides.SaveFormat;
```

##### Шаг 2: Создание объекта презентации

Создайте экземпляр `Presentation` сорт:

```java
Presentation pres = new Presentation();
```

##### Шаг 3: Зашифруйте презентацию

Примените пароль для шифрования презентации:

```java
pres.getProtectionManager().encrypt("pass"); // «pass» — выбранный вами пароль.
```

##### Шаг 4: Сохраните зашифрованный файл.

Сохраните презентацию в указанном каталоге с применением шифрования:

```java
String outputPath = "YOUR_DOCUMENT_DIRECTORY/SaveWithPassword_out.pptx";
pres.save(outputPath, SaveFormat.Pptx); // Сохраняет как файл PPTX.
```

#### Объяснение

- **`encrypt()`**: защищает вашу презентацию с помощью предоставленного пароля.
- **`save()`**Записывает зашифрованную презентацию на диск в указанном формате.

### Установка уровня масштабирования для просмотра слайдов
Убедитесь, что весь слайд виден, установив уровень масштабирования 100%.
#### Пошаговая реализация
**1. Моментальное представление**
Создайте новый экземпляр `Presentation`:

```java
import com.aspose.slides.Presentation;
import com.aspose.slides.SaveFormat;

public class SetZoomFeature {
    public static void main(String[] args) {
        String dataDir = "YOUR_DOCUMENT_DIRECTORY";
        Presentation presentation = new Presentation();
```
**2. Отрегулируйте уровень масштабирования слайда.**
Используйте `setScale()` Метод установки уровня масштабирования:

```java
// Установить масштаб просмотра слайдов на 100%
presentation.getViewProperties().getSlideViewProperties().setScale(100);
```
*Почему этот шаг?* Настройка масштаба гарантирует, что весь контент уместится в видимой области, что повышает четкость и фокусировку.
**3. Сохраните презентацию**
Записать изменения обратно в файл:

```java
// Сохранить в формате PPTX
try {
    presentation.save(dataDir + "Zoom_out.pptx", SaveFormat.Pptx);
} finally {
    if (presentation != null) presentation.dispose();
}
```
*Зачем сохранять в PPTX?* Этот формат сохраняет все улучшения и широко поддерживается.
### Настройка уровня масштабирования для просмотра заметок
Аналогичным образом настройте вид заметок, чтобы обеспечить полную видимость:
**1. Отрегулируйте уровень масштабирования заметок**

```java
// Установить масштаб просмотра заметок на 100%
presentation.getViewProperties().getNotesViewProperties().setScale(100);
```
*Почему этот шаг?* Единый уровень масштабирования слайдов и заметок обеспечивает бесперебойную презентацию.

# Как извлекать и управлять свойствами 3D-камеры в PowerPoint с помощью Aspose.Slides Java

**Зависимость Gradle:**

### Руководство по внедрению
Теперь, когда ваша среда готова, давайте извлечем и обработаем данные камеры из 3D-фигур в PowerPoint.

#### Пошаговое извлечение данных с камеры
**1. Загрузите презентацию**
Начните с загрузки файла презентации, содержащего целевой слайд и форму:

```java
import com.aspose.slides.Presentation;
import com.aspose.slides.IThreeDFormatEffectiveData;

Presentation pres = new Presentation("YOUR_DOCUMENT_DIRECTORY/Presentation1.pptx");
```
Этот код инициализирует `Presentation` объект, указывающий на ваш файл PowerPoint.

**2. Получите доступ к эффективным данным формы**
Перейдите к первому слайду и его первой фигуре, чтобы получить доступ к эффективным данным в формате 3D:

```java
IThreeDFormatEffectiveData threeDEffectiveData = pres.getSlides().get_Item(0)
    .getShapes().get_Item(0).getThreeDFormat().getEffective();
```
На этом этапе извлекаются эффективно примененные к форме трехмерные свойства.

**3. Получить свойства камеры**
Извлеките тип камеры, угол поля зрения и настройки масштабирования:

```java
String cameraType = threeDEffectiveData.getCamera().getCameraType();
float fieldOfViewAngle = threeDEffectiveData.getCamera().getFieldOfViewAngle();
double zoom = threeDEffectiveData.getCamera().getZoom();

// Распечатать значения для проверки
System.out.println("Camera Type: " + cameraType);
System.out.println("Field of View Angle: " + fieldOfViewAngle);
System.out.println("Zoom Level: " + zoom);
```
Эти свойства помогут вам понять применяемую трехмерную перспективу.

**4. Очистите ресурсы**
Всегда освобождайте ресурсы:

```java
finally {
    if (pres != null) pres.dispose();
}
```

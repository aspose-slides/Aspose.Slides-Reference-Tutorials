## Настройка Aspose.Slides для Java
Чтобы начать использовать Aspose.Slides для Java, включите библиотеку в свой проект следующим образом:

**Настройка Maven**
Добавьте эту зависимость к вашему `pom.xml` файл:
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

**Настройка Gradle**
Включите это в свой `build.gradle` файл:
```gradle
implementation group: 'com.aspose', name: 'aspose-slides', version: '25.4', classifier: 'jdk16'
```

**Прямая загрузка**
Либо загрузите последнюю версию непосредственно с сайта [Aspose.Slides для релизов Java](https://releases.aspose.com/slides/java/).

## Руководство по внедрению
В этом разделе мы рассмотрим, как читать гиперссылки со слайдов презентации с помощью Aspose.Slides для Java.

### Чтение информации по гиперссылке
Эта функция позволяет извлекать как поддельную, так и настоящую внешнюю информацию гиперссылок. Давайте разберем ее:

#### Шаг 1: Инициализация объекта презентации
Создайте экземпляр `Presentation` class, указывающий на ваш каталог документов.
```java
Presentation presentation = new Presentation("YOUR_DOCUMENT_DIRECTORY/ExternalUrlOriginal.pptx");
```

#### Шаг 2: Доступ к текстовой части слайда
Получите доступ к определенной части текста, из которой вы хотите прочитать гиперссылки. В этом примере мы нацеливаемся на часть в первой форме первого слайда.
```java
IPortion portion = ((AutoShape)presentation.getSlides().get_Item(0)
        .getShapes().get_Item(1)).getTextFrame().getParagraphs().get_Item(0).getPortions().get_Item(0);
```

#### Шаг 3: Извлечение информации о гиперссылке
Извлеките как поддельные, так и реальные внешние гиперссылки, связанные с этой частью текста.
```java
String fakeExternalHyperlink = portion.getPortionFormat().getHyperlinkClick().getExternalUrl();
System.out.println("Fake External Hyperlink : " + fakeExternalHyperlink);

String realExternalHyperlink = portion.getPortionFormat().getHyperlinkClick().getExternalUrlOriginal();
System.out.println("Real External Hyperlink : " + realExternalHyperlink);
```

### Советы по устранению неполадок
- **Исключение нулевого указателя:** Убедитесь, что путь к файлу презентации правильный и доступный.
- **Проблемы с доступом к форме:** Убедитесь, что индекс формы присутствует на вашем слайде, чтобы предотвратить `IndexOutOfBoundsException`.

## Руководство по внедрению
Давайте разобьем реализацию на управляемые функции. Каждая функция проведет вас через создание и настройку слайдов презентации.

### Создать презентацию и форму
#### Обзор
Начните с инициализации новой презентации и добавления автоматической фигуры к первому слайду.

**Шаг 1:** Инициализируйте новый `Presentation` объект.
```java
Presentation pres = new Presentation();
```

**Шаг 2:** Откройте первый слайд.
```java
ISlide slide = pres.getSlides().get_Item(0);
```

**Шаг 3:** Добавьте на слайд автофигуру типа «Прямоугольник».
```java
IAutoShape shape = slide.getShapes().addAutoShape(ShapeType.Rectangle, 10, 10, 200, 250);
```

**Совет по устранению неполадок:**
Убедитесь, что библиотека Aspose.Slides добавлена правильно, чтобы избежать проблем с classpath.

### Добавить абзацы в текстовую рамку фигуры
#### Обзор
Узнайте, как добавлять текст в фигуру, используя абзацы и части для более детального управления форматированием.

**Шаг 1:** Очистить существующие абзацы.
```java
shape.getTextFrame().getParagraphs().clear();
```

**Шаг 2:** Создайте абзац с частью текста.
```java
Paragraph para1 = new Paragraph();
para1.getPortions().add(new Portion("Sample text"));
```

**Шаг 3:** Добавьте абзац в текстовую рамку фигуры.
```java
shape.getTextFrame().getParagraphs().add(para1);
```

### Настроить формат конечной части абзаца
#### Обзор
Настройте внешний вид отдельных частей абзацев.

**Шаг 1:** Создайте второй абзац с пользовательскими параметрами форматирования.
```java
Paragraph para2 = new Paragraph();
para2.getPortions().add(new Portion("Sample text 2"));
```

**Шаг 2:** Установите и примените форматирование к конечной части.
```java
PortionFormat format = new PortionFormat();
format.setFontHeight(48); // Высота шрифта в пунктах
format.setLatinFont(new FontData("Times New Roman")); // Семейство шрифтов

para2.setEndParagraphPortionFormat(format);
```

**Шаг 3:** Добавьте отформатированный абзац к вашей фигуре.
```java
shape.getTextFrame().getParagraphs().add(para2);
```

### Сохранить презентацию
#### Обзор
Когда ваша презентация будет готова, сохраните ее в определенном каталоге.

**Шаг 1:** Определите выходной путь.
```java
String outputPath = "YOUR_OUTPUT_DIRECTORY/pres.pptx";
```

**Шаг 2:** Сохраните презентацию в указанном формате.
```java
pres.save(outputPath, SaveFormat.Pptx);
```

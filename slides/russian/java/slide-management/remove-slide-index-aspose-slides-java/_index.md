### Настройка Maven
Добавьте следующую зависимость к вашему `pom.xml` файл:
```xml

    <artifactId>aspose-slides</artifactId>
```

## Руководство по внедрению

### Удалить функцию слайда по индексу

Эта функция позволяет удалить определенный слайд из презентации на основе его индекса.

#### Шаг 1: Загрузите презентацию
Создать экземпляр `Presentation` и загрузите ваш файл PowerPoint:
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY/";
Presentation pres = new Presentation(dataDir + "RemoveSlideUsingIndex.pptx");
```

#### Шаг 2: Удалить слайд с определенным индексом
Используйте `removeAt()` метод удаления слайда. Здесь мы удаляем первый слайд (индекс 0):
```java
pres.getSlides().removeAt(0);
```
**Зачем использовать `removeAt()`:** Этот метод эффективно удаляет слайды, не изменяя другие элементы презентации.

#### Шаг 3: Сохраните презентацию
После изменения презентации сохраните ее в новом файле:
```java
String outputDir = "YOUR_OUTPUT_DIRECTORY/";
pres.save(outputDir + "modified_out.pptx", SaveFormat.Pptx);
```

### Советы по устранению неполадок
- **Исключение нулевого указателя:** Убедитесь, что путь к вашим файлам правильный и доступный.
- **Ошибка «Файл не найден»:** Убедитесь, что `RemoveSlideUsingIndex.pptx` существует в вашем каталоге документов.

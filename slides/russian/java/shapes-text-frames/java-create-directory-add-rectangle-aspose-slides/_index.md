### Функция 1: Создание каталога для вывода

Эта функция гарантирует, что ваше приложение может сохранять выходные файлы, такие как презентации, не сталкиваясь с ошибками, связанными с каталогами. Вот как вы проверяете, существует ли каталог, и создаете его при необходимости:

**Проверить и создать каталог:**

```java
import java.io.File;

String outputDir = "YOUR_OUTPUT_DIRECTORY";

boolean isExists = new File(outputDir).exists();
if (!isExists) {
    boolean wasCreated = new File(outputDir).mkdirs();
    // При необходимости обработайте случай, когда каталог не был создан.
}
```

**Почему это важно:** Проверяя существование каталога перед попыткой сохранения файлов, ваше приложение становится более надежным и менее подверженным ошибкам во время выполнения.

### Функция 2: Создание новой презентации и добавление прямоугольной формы

#### Обзор

Добавление таких фигур, как прямоугольники, может помочь визуально организовать содержимое на слайдах. Вот как можно создать презентацию и добавить прямоугольную форму с помощью Aspose.Slides:

#### Пошаговая реализация

**Создать презентацию и добавить форму:**

```java
import com.aspose.slides.ISlide;
import com.aspose.slides.Presentation;
import com.aspose.slides.SaveFormat;
import com.aspose.slides.ShapeType;

String documentDir = "YOUR_DOCUMENT_DIRECTORY";
String outputDir = "YOUR_OUTPUT_DIRECTORY";

Presentation pres = new Presentation();
try {
    ISlide sld = pres.getSlides().get_Item(0);
    
    // Добавьте к слайду прямоугольную форму.
    sld.getShapes().addAutoShape(ShapeType.Rectangle, 50, 150, 150, 50);

    String outputPath = outputDir + "/RectShp1_out.pptx";
    pres.save(outputPath, SaveFormat.Pptx);
} finally {
    if (pres != null) pres.dispose();
}
```

**Почему это важно:** Программное добавление фигур позволяет динамически и автоматически создавать контент в презентациях, что может быть особенно полезно для создания отчетов или панелей мониторинга.

### Советы по устранению неполадок

- Убедитесь, что пути к выходным каталогам указаны правильно.
- Убедитесь, что у вас есть права на запись в указанные каталоги.
- Проверьте совместимость версии библиотеки Aspose.Slides с вашей настройкой JDK.

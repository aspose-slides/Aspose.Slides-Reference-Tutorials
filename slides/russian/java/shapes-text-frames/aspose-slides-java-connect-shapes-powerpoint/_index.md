## Руководство по внедрению

В этом разделе мы рассмотрим, как соединять фигуры с помощью соединителей в PowerPoint с помощью Aspose.Slides Java.

### Добавление фигур
Сначала добавим две базовые фигуры: эллипс и прямоугольник. Поместим их на первый слайд нашей презентации.
```java
// Создать экземпляр класса Presentation, представляющего файл PPTX
Presentation input = new Presentation();
try {
    // Доступ к коллекции фигур для выбранного слайда (первый слайд)
    IShapeCollection shapes = input.getSlides().get_Item(0).getShapes();

    // Добавить автофигуру Эллипс в позицию (0, 100) размером (100x100)
    IAutoShape ellipse = shapes.addAutoShape(ShapeType.Ellipse, 0, 100, 100, 100);

    // Добавить автофигуру «Прямоугольник» в положение (100, 300) с размером (100x100)
    IAutoShape rectangle = shapes.addAutoShape(ShapeType.Rectangle, 100, 300, 100, 100);
```

### Соединение фигур
Теперь, когда наши фигуры на месте, давайте соединим их с помощью соединителя. Мы будем использовать изогнутый соединитель, чтобы связать эллипс и прямоугольник.
```java
    // Добавление соединительной фигуры к коллекции слайдовых фигур, начиная с (0, 0) с размером (10x10)
    IConnector connector = shapes.addConnector(ShapeType.BentConnector2, 0, 0, 10, 10);

    // Присоединение эллипса к началу соединителя
    connector.setStartShapeConnectedTo(ellipse);

    // Присоединение прямоугольника к концу соединителя
    connector.setEndShapeConnectedTo(rectangle);
```

### Изменение маршрута соединителя
После подключения измените маршрут соединителя, чтобы убедиться, что он находит кратчайший путь между фигурами.
```java
    // Перенаправьте соединитель, чтобы автоматически найти кратчайший путь между фигурами
    connector.reroute();
```

### Сохранение презентации
Наконец, сохраните презентацию в формате PPTX под указанным именем.
```java
    // Сохраните презентацию в формате PPTX с указанным именем.
    input.save("Connecting_shapes_using_connectors_out.pptx", SaveFormat.Pptx);
} finally {
    if (input != null) input.dispose();
}
```

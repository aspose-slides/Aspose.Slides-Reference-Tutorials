## Руководство по внедрению

### Маркеры абзацев с символом
#### Обзор
Эта функция позволяет вам настраивать слайды, добавляя маркеры на основе символов. Она идеально подходит для подчеркивания ключевых моментов визуально отчетливым образом.

#### Шаги по реализации
**1. Создать экземпляр презентации**
```java
Presentation pres = new Presentation();
```

**2. Откройте слайд и добавьте фигуру.**
Откройте первый слайд и добавьте автофигуру:
```java
ISlide slide = pres.getSlides().get_Item(0);
IAutoShape aShp = slide.getShapes().addAutoShape(ShapeType.Rectangle, 200, 200, 400, 200);
```

**3. Настройте текстовую рамку**
Удалите все абзацы по умолчанию и создайте новый:
```java
ITextFrame txtFrm = aShp.getTextFrame();
txtFrm.getParagraphs().removeAt(0);

Paragraph para = new Paragraph();
para.getParagraphFormat().getBullet().setType(BulletType.Symbol);
para.getParagraphFormat().getBullet().setChar((char) 8226); // Характер пули
```

**4. Настройте внешний вид маркера**
Определите отступ, цвет и размер маркера:
```java
para.setText("Welcome to Aspose.Slides");
para.getParagraphFormat().setIndent(25);
para.getParagraphFormat().getBullet().setColor(Color.BLACK);
para.getParagraphFormat().getBullet().setHeight(100);

txtFrm.getParagraphs().add(para);
```

**5. Сохраните презентацию**
Всегда сохраняйте изменения:
```java
pres.save("YOUR_OUTPUT_DIRECTORY/Bullet_out.pptx", SaveFormat.Pptx);
```
Не забывайте правильно утилизировать ресурсы.

### Маркеры абзацев с нумерованным стилем
#### Обзор
Пронумерованные пункты помогают создавать упорядоченные списки, облегчая аудитории понимание последовательной информации.

#### Шаги по реализации
**1. Создать экземпляр презентации**
Повторно используйте шаги из маркеров символов для инициализации презентации.

**2. Настройте текстовую рамку и тип маркера**
Настройте текстовую рамку и определите стиль нумерованного маркера:
```java
Paragraph para2 = new Paragraph();
para2.getParagraphFormat().getBullet().setType(BulletType.Numbered);
para2.getParagraphFormat().getBullet().setNumberedBulletStyle(NumberedBulletStyle.BulletCircleNumWDBlackPlain);

para2.setText("This is numbered bullet");
```

**3. Настройте внешний вид**
Аналогично символьным маркерам, настройте параметры отступа и цвета:
```java
para2.getParagraphFormat().setIndent(25);
para2.getParagraphFormat().getBullet().setColor(Color.BLACK);
para2.getParagraphFormat().getBullet().setHeight(100);

txtFrm.getParagraphs().add(para2);
```

**4. Сохраните презентацию**
Выполните ту же процедуру сохранения, что и раньше.

## Руководство по внедрению

### Функция 1: Проверка текстовых фигур

#### Обзор
Эта функция позволяет определить, являются ли фигуры на первом слайде презентации PowerPoint текстовыми полями, с помощью Aspose.Slides для Java.

#### Пошаговая реализация

**1. Загрузите презентацию**
Начните с загрузки файла презентации в `Aspose.Slides.Presentation` объект.
```java
import com.aspose.slides.Presentation;

String documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
String presentationPath = documentDirectory + "/CheckTextShapes.pptx";

Presentation pres = new Presentation(presentationPath);
try {
    // Дальнейшие операции будут проводиться здесь.
} finally {
    if (pres != null) pres.dispose();
}
```
*Почему этот шаг?*: Он инициализирует `Presentation` объект, позволяющий манипулировать слайдами и анализировать их.

**2. Итерация по формам**
Просмотрите каждую фигуру на первом слайде, чтобы определить ее тип.
```java
import com.aspose.slides.IShape;
import com.aspose.slides.AutoShape;

// Итерация фигур на первом слайде
for (IShape shape : pres.getSlides().get_Item(0).getShapes()) {
    if (shape instanceof AutoShape) {
        AutoShape autoShape = (AutoShape) shape;
        
        // Проверьте и распечатайте, является ли это текстовым полем
        boolean isTextBox = autoShape.isTextBox();
        System.out.println(isTextBox ? "Shape is a text box" : "Shape is not a text box");
    }
}
```
*Почему этот шаг?*Проверяя тип каждой фигуры, вы можете программно проверять и обрабатывать только те из них, которые являются текстовыми полями.

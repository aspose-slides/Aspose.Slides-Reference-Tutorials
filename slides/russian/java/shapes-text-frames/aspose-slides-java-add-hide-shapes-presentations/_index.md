## Руководство по внедрению

### Функция 1: Создание презентации и добавление фигур

#### Обзор
Узнайте, как создать презентацию с нуля и добавить на слайды различные фигуры, такие как прямоугольники и луны.

##### Шаг 1: Создайте новую презентацию
Начните с создания экземпляра `Presentation` класс, который будет представлять ваш файл PowerPoint:

```java
// Создайте экземпляр класса Presentation, представляющего файл PPTX.
Presentation pres = new Presentation();
```

##### Шаг 2: Получите доступ к первому слайду
Для добавления фигур вам понадобится первый слайд из вашей презентации:

```java
// Получить первый слайд из презентации
ISlide sld = pres.getSlides().get_Item(0);
```

##### Шаг 3: Добавьте фигуры на слайд
Добавьте различные типы фигур, такие как прямоугольники и луны, используя соответствующие им `ShapeType` перечисления:

```java
// Добавить на слайд автофигуру прямоугольного типа
IShape shp1 = sld.getShapes().addAutoShape(ShapeType.Rectangle, 50, 40, 150, 50);

// Добавьте еще одну фигуру, автофигуру типа луны, на тот же слайд.
IShape shp2 = sld.getShapes().addAutoShape(ShapeType.Moon, 160, 40, 150, 50);
```

##### Шаг 4: Сохраните презентацию
После добавления фигур сохраните презентацию:

```java
// Сохраните презентацию на диск в формате PPTX в указанном выходном каталоге.
pres.save("YOUR_OUTPUT_DIRECTORY/Hiding_Shapes_out.pptx", SaveFormat.Pptx);
```

### Функция 2: Скрытие фигур с помощью определяемого пользователем альтернативного текста

#### Обзор
Эта функция позволяет скрывать определенные фигуры на основе их альтернативного текста, предоставляя эффективный способ управления видимостью контента.

##### Шаг 1: Получите доступ к слайду
Предполагая, `sld` уже определено из существующей презентации:

```java
// Предположим, что «sld» — это слайд, полученный из существующей презентации.
ISlide sld = new Presentation().getSlides().get_Item(0);
```

##### Шаг 2: Определите альтернативный текст, определяемый пользователем
Задайте альтернативный текст, который вы хотите использовать для скрытия фигур:

```java
String alttext = "User Defined";
```

##### Шаг 3: Перебор фигур и скрытие совпадающих
Пройдитесь по каждой фигуре на слайде, проверяя, соответствует ли она определенному альтернативному тексту. Если да, скройте ее:

```java
// Получить количество фигур, присутствующих на слайде
int iCount = sld.getShapes().size();

// Пройдитесь по каждой фигуре на слайде
for (int i = 0; i < iCount; i++) {
    // Привести форму к типу AutoShape
    AutoShape ashp = (AutoShape) sld.getShapes().get_Item(i);
    
    // Проверьте, соответствует ли альтернативный текст текущей фигуры тексту, определенному пользователем.
    if (ashp.getAlternativeText().equals(alttext)) {
        // Если фигура соответствует, установите для нее скрытый вид.
        ashp.setHidden(true);
    }
}
```

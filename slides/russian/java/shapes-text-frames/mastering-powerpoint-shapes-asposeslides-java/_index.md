## Настройка Aspose.Slides для Java

Начало работы с Aspose.Slides простое. Вы можете интегрировать библиотеку в свой проект с помощью Maven, Gradle или напрямую загрузить ее.

**Знаток**
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

**Градл**
```gradle
implementation group: 'com.aspose', name: 'aspose-slides', version: '25.4', classifier: 'jdk16'
```

**Прямая загрузка**: Для тех, кто предпочитает не использовать менеджер пакетов, вы можете загрузить последнюю версию с сайта [Aspose.Slides для релизов Java](https://releases.aspose.com/slides/java/).

## Руководство по внедрению

Теперь, когда вы готовы начать, давайте рассмотрим каждую функцию создания и соединения фигур с помощью Aspose.Slides для Java.

### Создавайте и соединяйте фигуры

В этом разделе основное внимание уделяется добавлению на слайды таких фигур, как эллипсы и прямоугольники, и их связыванию с помощью соединителей.

#### Шаг 1: Доступ к формам слайдов
```java
// Доступ к коллекции фигур первого слайда
IShapeCollection shapes = presentation.getSlides().get_Item(0).getShapes();
```
Здесь мы получаем доступ к коллекции, где будут находиться все наши новые формы. 

#### Шаг 2: Добавление соединительной формы
```java
// Добавьте изогнутый соединитель для соединения фигур.
IConnector connector = shapes.addConnector(ShapeType.BentConnector3, 0, 0, 10, 10);
```
Соединитель служит мостом между нашими фигурами.

#### Шаг 3: Создание эллипса
```java
// Добавьте к слайду форму эллипса.
IAutoShape ellipse = shapes.addAutoShape(ShapeType.Ellipse, 0, 100, 100, 100);
```

#### Шаг 4: Добавление прямоугольника
```java
// Добавьте на слайд прямоугольную форму.
IAutoShape rectangle = shapes.addAutoShape(ShapeType.Rectangle, 100, 200, 100, 100);
```
Теперь эти формы готовы к соединению.

#### Шаг 5: Соединение фигур с помощью соединителей
```java
// Соедините эллипс и прямоугольник с помощью соединителя.
connector.setStartShapeConnectedTo(ellipse);
connector.setEndShapeConnectedTo(rectangle);
```
Устанавливая эти соединения, вы создаете визуальную связь между двумя фигурами.

### Соедините форму на желаемом месте соединения

Если требуются определенные точки подключения, Aspose.Slides позволяет выполнить детальную настройку.

#### Шаг 1: Настройка соединителя и фигур
Как и прежде, настройте соединитель и фигуры, как описано в предыдущих шагах.

#### Шаг 2: Указание места подключения
```java
long wantedIndex = 6;
// Убедитесь, что желаемый индекс находится в пределах допустимого
if (ellipse.getConnectionSiteCount() > (wantedIndex & 0xFFFFFFFFL)) {
    // Подключитесь к определенному месту на эллипсе
    connector.setStartShapeConnectionSiteIndex(wantedIndex);
}
```
Это позволяет точно контролировать место возникновения соединений.

### Сохранить презентацию

Наконец, убедитесь, что ваша работа сохранена, сохранив файл презентации.
```java
// Определите выходной путь и сохраните презентацию в формате PPTX.
String outputPath = "YOUR_OUTPUT_DIRECTORY" + "/Connecting_Shape_on_desired_connection_site_out.pptx";
presentation.save(outputPath, SaveFormat.Pptx);
```
После этого шага ваш настроенный PowerPoint готов к использованию или распространению.

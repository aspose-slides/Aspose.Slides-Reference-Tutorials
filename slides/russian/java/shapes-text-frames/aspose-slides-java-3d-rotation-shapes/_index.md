## Настройка Aspose.Slides для Java

Для начала вам нужно будет интегрировать библиотеку Aspose.Slides в ваш проект. Вот как это сделать:

### Настройка Maven
Добавьте следующую зависимость к вашему `pom.xml` файл:
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

### Настройка Gradle
Включите эту строку в свой `build.gradle` файл:
```gradle
implementation group: 'com.aspose', name: 'aspose-slides', version: '25.4', classifier: 'jdk16'
```

## Руководство по внедрению

Давайте углубимся в применение 3D-вращения к прямоугольной форме в PowerPoint с помощью Aspose.Slides для Java. Мы разобьем это на управляемые шаги.

### Создание презентации и добавление фигуры

#### Обзор
Сначала мы создаем новую презентацию и добавляем прямоугольник к первому слайду.
```java
// Создать экземпляр класса Presentation
Presentation pres = new Presentation();

// Добавьте прямоугольную автофигуру к первому слайду
IShape autoShape = pres.getSlides().get_Item(0).getShapes().addAutoShape(
    ShapeType.Rectangle, 30, 30, 200, 200);
```
**Объяснение**: 
- `Presentation` инициализируется для создания новой презентации.
- Добавляем автофигуру типа «Прямоугольник» в позицию (30, 30) с размерами 200x200.

### Применение 3D-вращения

#### Обзор
Далее мы настраиваем 3D-эффекты на нашем прямоугольнике.
```java
// Установите глубину 3D-эффекта
autoShape.getThreeDFormat().setDepth((short) 6);

// Настройте поворот и тип камеры для трехмерной перспективы
autoShape.getThreeDFormat().getCamera().setRotation(40, 35, 20);
autoShape.getThreeDFormat().getCamera().setCameraType(CameraPresetType.IsometricLeftUp);

// Установите тип осветительного оборудования для сбалансированного освещения.
autoShape.getThreeDFormat().getLightRig().setLightType(LightRigPresetType.Balanced);
```
**Объяснение**: 
- `setDepth` регулирует глубину 3D-эффекта.
- Поворот и тип камеры настраиваются для создания определенной перспективы.
- Для равномерного освещения применяется сбалансированная осветительная установка.

### Сохранение презентации

Наконец, сохраните свою презентацию, применив следующие эффекты:
```java
// Сохраните презентацию с 3D-эффектами, примененными к файлу
pres.save("YOUR_OUTPUT_DIRECTORY\\Rotation_out.pptx", com.aspose.slides.SaveFormat.Pptx);
```
**Объяснение**: 
- The `save` метод выводит измененную презентацию по указанному пути.

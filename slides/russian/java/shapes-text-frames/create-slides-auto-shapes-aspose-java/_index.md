## Руководство по внедрению
### Функция 1: Создание каталога
**Обзор:** Прежде чем приступать к файловым операциям, убедитесь, что каталог с документами существует.
#### Шаг за шагом
1. **Определите путь к документу:**
   ```java
   String dataDir = "YOUR_DOCUMENT_DIRECTORY";
   ```
2. **Проверьте и создайте каталог, если необходимо:**
   ```java
   boolean isExists = new File(dataDir).exists();
   if (!isExists) {
       new File(dataDir).mkdirs(); // Создает каталоги рекурсивно
   }
   ```
### Функция 2: Создание презентаций
**Обзор:** Создайте новый экземпляр презентации PowerPoint.
#### Шаг за шагом
1. **Создайте экземпляр объекта презентации:**
   ```java
   Presentation pres = new Presentation();
   ```
### Функция 3: Добавление автофигуры к слайду
**Обзор:** Добавляйте к слайдам фигуры, например прямоугольники, для структурирования контента.
#### Шаг за шагом
1. **Откройте первый слайд и добавьте прямоугольную форму:**
   ```java
   ISlide slide = pres.getSlides().get_Item(0);
   IAutoShape aShp = slide.getShapes().addAutoShape(
       ShapeType.Rectangle, 200, 200, 400, 200);
   ```
### Функция 4: Добавление и форматирование текста в AutoShape
**Обзор:** Вставьте текст в фигуры и примените форматирование маркеров для ясности.
#### Шаг за шагом
1. **Доступ к текстовому фрейму фигуры:**
   ```java
   ITextFrame text = aShp.addTextFrame("");
   ```
2. **Добавьте и отформатируйте абзацы с помощью маркеров:**
   ```java
   Paragraph para1 = new Paragraph();
   para1.setText("Content");
   para1.getParagraphFormat().getBullet().setType(BulletType.Symbol);
   para1.getParagraphFormat().setDepth((short) 0); // Уровень 1 пуля

   text.getParagraphs().add(para1);
   ```
### Функция 5: Сохранение презентации
**Обзор:** Сохраните презентацию по указанному пути в формате PPTX.
#### Шаг за шагом
1. **Укажите выходной путь и сохраните файл:**
   ```java
   String outputPath = "YOUR_OUTPUT_DIRECTORY/MultilevelBullet.pptx";
   pres.save(outputPath, SaveFormat.Pptx);
   ```

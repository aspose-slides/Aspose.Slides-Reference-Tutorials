## Руководство по внедрению
Теперь давайте реализуем функцию удаления сегмента из геометрической фигуры.
### Создание и изменение формы сердца
Начнем с создания объекта в форме сердца в PowerPoint с помощью Aspose.Slides для Java. В этом разделе объясняется, как получить доступ к его геометрическому пути и изменить его.
#### Добавить геометрическую фигуру
Сначала добавьте в презентацию новую геометрическую фигуру:
```java
// Инициализация класса презентации
Presentation pres = new Presentation();
try {
    // Создайте форму сердца на первом слайде в позиции (100, 100) с размером (300, 300)
    com.aspose.slides.ShapeType shapeType = com.aspose.slides.ShapeType.Heart;
    GeometryShape shape = (GeometryShape) pres.getSlides().get_Item(0).getShapes()
            .addAutoShape(shapeType, 100, 100, 300, 300);
```
#### Доступ к геометрическому пути
Далее, перейдите к геометрическому пути вашей новой созданной фигуры:
```java
// Доступ к первому геометрическому контуру формы сердца
IGeometryPath path = shape.getGeometryPaths()[0];
```
#### Удалить сегмент из пути
Чтобы удалить сегмент (например, третий):
```java
// Удалить третий сегмент (индекс 2) из геометрического пути.
path.removeAt(2);
```
#### Обновите и сохраните презентацию
Наконец, обновите фигуру с измененным контуром и сохраните презентацию:
```java
// Обновите форму с измененным геометрическим контуром.
shape.setGeometryPath(path);

// Определите путь к выходному файлу и сохраните презентацию в формате PPTX.
String resultPath = "YOUR_OUTPUT_DIRECTORY" +  "/GeometryShapeRemoveSegment.pptx";
pres.save(resultPath, SaveFormat.Pptx);
```

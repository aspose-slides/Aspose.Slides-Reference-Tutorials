Здесь мы рассмотрим, как использовать Aspose.Slides for Java для бесшовного **добавления диаграммы на слайд**, её настройки и, наконец, **сохранения презентации PowerPoint PPTX**, эффективно **управляя ресурсами PowerPoint**.

### Как создать пузырчатую диаграмму в PowerPoint
#### Обзор
Эта функция демонстрирует добавление настраиваемой пузырчатой диаграммы в ваши слайды PowerPoint. Мы настроим её размер и масштаб для лучшего представления данных.

#### Пошаговая реализация
**1. Инициализация презентации**  
Создайте экземпляр `Presentation`:
```java
Presentation pres = new Presentation();
```

**2. Добавление пузырчатой диаграммы**  
Добавьте пузырчатую диаграмму в указанную позицию с заданными размерами:
```java
IChart chart = pres.getSlides().get_Item(0).getShapes().addChart(
        ChartType.Bubble, 100, 100, 400, 300
);
```
- **Параметры**: `ChartType.Bubble` указывает тип диаграммы. Числа представляют позицию (x, y) и размер (ширина, высота).

**3. Настройка масштаба размера пузырьков**  
Отрегулируйте размер пузырьков для лучшей читаемости:
```java
chart.getChartData().getSeriesGroups().get_Item(0).setBubbleSizeScale(150);
```
- **Назначение**: Установка `BubbleSizeScale` в 150 % увеличивает пузырьки, делая их более различимыми.

**4. Сохранение презентации**  
Сохраните изменения с добавленной диаграммой:
```java
pres.save("YOUR_OUTPUT_DIRECTORY/Result.pptx", SaveFormat.Pptx);
```

#### Советы по устранению неполадок
- Убедитесь, что у вас есть права записи в каталог вывода.
- Проверьте, что Aspose.Slides корректно включена в зависимости вашего проекта.

### Как эффективно управлять ресурсами PowerPoint
Эффективное управление ресурсами обеспечивает оптимальную производительность. Вот как правильно работать с жизненным циклом презентации:

**1. Создание и изменение**  
Начните с создания экземпляра `Presentation`:
```java
Presentation pres = new Presentation();
```
Выполните необходимые операции, такие как добавление диаграмм или слайдов.

**2. Освобождение ресурсов**  
Всегда освобождайте презентацию, чтобы освободить ресурсы:
```java
if (pres != null) pres.dispose();
```
Этот шаг критически важен для предотвращения утечек памяти.

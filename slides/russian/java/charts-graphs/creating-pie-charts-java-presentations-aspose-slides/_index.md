## Руководство по внедрению

### Создать и добавить круговую диаграмму в презентацию

#### Обзор
В этом разделе рассматриваются шаги по созданию круговой диаграммы на слайде презентации. Мы проведем вас через инициализацию презентации, создание диаграммы и настройку ее внешнего вида.

#### Шаг 1: Инициализация презентации
Начните с создания экземпляра `Presentation` сорт:
```java
demo.Presentation pres = new demo.Presentation();
```
Это инициализирует вашу презентацию, в которую будут внесены все изменения.

#### Шаг 2: Добавьте круговую диаграмму на слайд
Далее добавьте круговую диаграмму на первый слайд в указанных координатах с заданными размерами:
```java
// Определите положение и размер круговой диаграммы.
int xPosition = 50;
int yPosition = 50;
int width = 400;
int height = 600;

demo.IChart chart = pres.getSlides().get_Item(0).getShapes().addChart(
    demo.ChartType.Pie, xPosition, yPosition, width, height, false);
```
Здесь:
- `xPosition` и `yPosition` задайте верхние левые координаты.
- `width` и `height` определить размеры диаграммы.

#### Шаг 3: Настройте круговую диаграмму
Настройте круговую диаграмму, изменив ее точки данных, цвета или метки. Вот простой пример добавления данных в диаграмму:
```java
// Доступ к серии данных по умолчанию для демонстрации
demo.IChartDataWorkbook wb = chart.getChartData().getChartDataWorkbook();
chart.getChartData().getSeries().clear();

// Добавить новые серии и заполнить данными
demo.IChartSeries series = chart.getChartData().getSeries().add(wb.getCell(0, "B1", "Category 1"), demo.ChartType.Pie);
series.getDataPoints().addDataPointForPieSeries(wb.getCell(0, "B2", 30));
series.getDataPoints().addDataPointForPieSeries(wb.getCell(0, "B3", 70));

// Настройте серийные этикетки
for (demo.IDataPoint point : series.getDataPoints()) {
    demo.IChartDataLabel label = point.getLabel();
    label.getDataLabelFormat().setShowCategoryName(true);
}
```
Этот сегмент кода добавляет ряд данных с двумя категориями и настраивает названия категорий для отображения в виде меток.

#### Советы по устранению неполадок
- **Распространенная проблема:** Если вы столкнулись с ошибкой об отсутствии зависимостей, убедитесь, что ваш `pom.xml` или `build.gradle` файлы настроены правильно.
- **Диаграмма не отображается:** Проверьте, что все ряды данных и точки добавлены правильно. Диаграммы могут выглядеть пустыми, если данные не связаны.

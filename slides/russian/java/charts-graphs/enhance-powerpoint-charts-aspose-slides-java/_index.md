### Руководство по внедрению

#### Функция 1: Условные обозначения размера шрифта в таблице

**Обзор:**
Изменение размера шрифта легенды может значительно улучшить ее видимость и читаемость, сделав ваши диаграммы более удобными для пользователя.

**Шаги по настройке размера шрифта легенды:**

**H3. Добавить кластеризованную столбчатую диаграмму**
Начните с создания кластеризованной столбчатой диаграммы на первом слайде в позиции (50, 50) с размерами 600x400:
```java
import com.aspose.slides.ChartType;
import com.aspose.slides.IChart;
import com.aspose.slides.Presentation;

Presentation pres = new Presentation("YOUR_DOCUMENT_DIRECTORY/test.pptx");
try {
    IChart chart = pres.getSlides().get_Item(0).getShapes().addChart(
        ChartType.ClusteredColumn, 50, 50, 600, 400);

    // Установить размер шрифта легенды
    chart.getLegend().getTextFormat().getPortionFormat().setFontHeight(20);
    
pres.save("YOUR_OUTPUT_DIRECTORY/output.pptx", SaveFormat.Pptx);
} finally {
    if (pres != null) pres.dispose();
}
```
- **Объяснение:** The `setFontHeight` метод устанавливает размер текста легенды равным 20 пунктам, что повышает его читабельность.

**H3. Сохраните изменения**
Обязательно сохраните презентацию, чтобы изменения вступили в силу:
```java
pres.save("YOUR_OUTPUT_DIRECTORY/output.pptx", SaveFormat.Pptx);
```

#### Функция 2: Конфигурация значений осей на диаграмме

**Обзор:**
Настройка значений осей позволяет точно контролировать представление данных, что упрощает понимание тенденций для аудитории.

**Шаги по настройке значений вертикальной оси:**

**H3. Добавить кластеризованную столбчатую диаграмму**
Аналогично предыдущему примеру добавьте кластеризованную столбчатую диаграмму:
```java
import com.aspose.slides.Presentation;
import com.aspose.slides.IChart;
import com.aspose.slides.ChartType;

Presentation pres = new Presentation("YOUR_DOCUMENT_DIRECTORY/test.pptx");
try {
    IChart chart = pres.getSlides().get_Item(0).getShapes().addChart(
        ChartType.ClusteredColumn, 50, 50, 600, 400);

    // Настроить вертикальную ось
    chart.getAxes().getVerticalAxis().setAutomaticMinValue(false);
    chart.getAxes().getVerticalAxis().setMinValue(-5);
    
    chart.getAxes().getVerticalAxis().setAutomaticMaxValue(false);
    chart.getAxes().getVerticalAxis().setMaxValue(10);

    pres.save("YOUR_OUTPUT_DIRECTORY/output.pptx", SaveFormat.Pptx);
} finally {
    if (pres != null) pres.dispose();
}
```
- **Объяснение:** Отключение автоматических настроек минимального и максимального значений позволяет вам указать свои собственные, например, -5 для минимума и 10 для максимума, обеспечивая точный контроль над масштабированием данных.

## Настройка Aspose.Slides для Java

Чтобы использовать Aspose.Slides, добавьте его как зависимость. Вы можете управлять этим через Maven, Gradle или прямой загрузкой.

### Знаток

Добавьте следующую зависимость в ваш `pom.xml`:

```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

### Градл

В вашем `build.gradle`, включать:

```gradle
implementation group: 'com.aspose', name: 'aspose-slides', version: '25.4', classifier: 'jdk16'
```

Либо загрузите последнюю версию с сайта [Aspose.Slides для релизов Java](https://releases.aspose.com/slides/java/).

## Руководство по внедрению

Теперь давайте создадим диаграмму «ящик с усами» с помощью Aspose.Slides для Java. Этот раздел проведет вас через каждый шаг процесса.

### Создать презентацию

Сначала инициализируйте новую презентацию или откройте существующую:

```java
Presentation pres = new Presentation("YOUR_DOCUMENT_DIRECTORY/test.pptx");
```

### Добавить диаграмму «ящик с усами»

Добавьте диаграмму на первый слайд в нужном вам месте и размере:

```java
IChart chart = pres.getSlides().get_Item(0).getShapes().addChart(
    ChartType.BoxAndWhisker, 50, 50, 500, 400);
```

### Очистить существующие данные

Перед заполнением новых данных очистите все существующие категории и серии:

```java
chart.getChartData().getCategories().clear();
chart.getChartData().getSeries().clear();

IChartDataWorkbook wb = chart.getChartData().getChartDataWorkbook();
wb.clear(0); // Очищает содержимое, начиная с ячейки «A1»
```

### Настроить категории

Добавьте категории к данным вашей диаграммы:

```java
for (int i = 1; i <= 6; i++) {
    chart.getChartData().getCategories()
        .add(wb.getCell(0, "A" + i, "Category 1"));
}
```

### Создавайте и настраивайте серии

Создайте новую серию и настройте ее свойства:

```java
IChartSeries series = chart.getChartData().getSeries().add(ChartType.BoxAndWhisker);
series.setQuartileMethod(QuartileMethodType.Exclusive); // Установить метод квартиля на «Эксклюзивный»
series.setShowMeanLine(true); // Отображение средней линии
series.setShowMeanMarkers(true); // Показать маркеры для средних значений
series.setShowInnerPoints(true); // Отображение внутренних точек на графике
series.setShowOutlierPoints(true); // Показать точки выброса на графике

int[] data = {15, 41, 16, 10, 23, 16}; // Образцы точек данных
for (int i = 0; i < data.length; i++) {
    series.getDataPoints().addDataPointForBoxAndWhiskerSeries(
        wb.getCell(0, "B" + (i + 1), data[i]));
}
```

### Сохранить презентацию

Наконец, сохраните вашу презентацию:

```java
pres.save("YOUR_OUTPUT_DIRECTORY/BoxAndWhisker.pptx", SaveFormat.Pptx);
```

Всегда проверяйте, чтобы утилизировать `Presentation` объект для освобождения ресурсов:

```java
finally {
    if (pres != null) pres.dispose();
}
```

## Раздел часто задаваемых вопросов

**В1: Что такое диаграмма «ящик с усами»?**

Диаграмма «ящик с усами», также известная как диаграмма ящиков, отображает распределение данных на основе пяти сводных статистик. Она полезна для отображения медианы, квартилей и выбросов в наборе данных.

**В2: Могу ли я настроить внешний вид диаграммы «ящик с усами»?**

Да, Aspose.Slides предоставляет широкие возможности настройки, включая цвета, шрифты и стили точек данных.

**В3: Можно ли обрабатывать несколько рядов на одной диаграмме?**

Конечно. Вы можете добавить несколько серий в свою диаграмму, повторив процесс создания и настройки каждой серии.

**В4: Как решить проблемы с некорректным отображением данных?**

Убедитесь, что данные правильно заполнены в ячейках и что вы установили соответствующие свойства видимости, такие как `setShowMeanLine`.

**В5: Где я могу получить поддержку, если у меня возникнут проблемы?**

Посетите [Форум Aspose.Slides](https://forum.aspose.com/c/slides/11) для поддержки сообщества или обратитесь к официальной документации.

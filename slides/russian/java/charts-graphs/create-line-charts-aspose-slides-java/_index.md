# Создание линейных диаграмм с маркерами по умолчанию с помощью Aspose.Slides для Java
## Введение
Создание визуально привлекательных и информативных диаграмм необходимо для презентаций, отчетов и информационных панелей. Автоматизация этого процесса в разработке программного обеспечения экономит время и обеспечивает согласованность между документами. В этом руководстве показано, как создавать линейные диаграммы с маркерами с помощью Aspose.Slides для Java.
**Aspose.Slides для Java** — мощная библиотека, которая позволяет разработчикам программно манипулировать презентациями PowerPoint без необходимости установки Microsoft Office. Она упрощает такие задачи, как создание, редактирование и экспорт слайдов, что делает ее важным инструментом для автоматизированного создания документов.
**Что вы узнаете:**
- Как инициализировать Aspose.Slides для Java
- Шаги по созданию линейной диаграммы с маркерами
- Добавление серий и категорий в диаграммы
- Настройка легенд диаграммы
- Сохранение презентации
Готовы приступить к работе? Давайте сначала убедимся, что у вас все готово!
## Предпосылки
Прежде чем начать, убедитесь, что ваша среда разработки готова:
1. **Библиотеки и зависимости:**
   - Библиотека Aspose.Slides для Java (рекомендуется версия 25.4)
   - Java Development Kit (JDK) версии 16 или выше
2. **Настройка среды:**
   - Ваша IDE должна поддерживать инструменты сборки Maven или Gradle.
   - При необходимости убедитесь, что у вас есть действительный файл лицензии.
3. **Необходимые знания:**
   - Базовые знания программирования на Java
   - Знакомство с созданием проектов с использованием Maven или Gradle
Теперь, когда все готово, давайте настроим Aspose.Slides для вашего проекта!
## Настройка Aspose.Slides для Java
Чтобы использовать Aspose.Slides для Java, вам нужно включить его как зависимость в ваш проект. В зависимости от того, используете ли вы Maven или Gradle, настройка будет немного отличаться.
### Знаток
Добавьте следующую зависимость к вашему `pom.xml` файл:
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```
### Градл
Включите это в свой `build.gradle` файл:
```gradle
implementation group: 'com.aspose', name: 'aspose-slides', version: '25.4', classifier: 'jdk16'
```

## Руководство по внедрению
### Функция 1: Создание диаграмм с маркерами по умолчанию
В этом разделе показано, как создать линейную диаграмму с маркерами. Эта функция необходима для эффективной визуализации тенденций данных.
#### Добавление линейной диаграммы
Чтобы добавить линейный график с маркерами:
```java
import com.aspose.slides.*;
// Доступ к первому слайду
ISlide slide = pres.getSlides().get_Item(0);
// Добавьте линейную диаграмму с маркерами на слайд в позиции (10, 10) размером (400, 400)
IChart chart = slide.getShapes().addChart(
    ChartType.LineWithMarkers, 10, 10, 400, 400);
```
#### Очистка серий и категорий
Чтобы начать все сначала:
```java
// Очистите существующие серии и категории, чтобы обеспечить чистый лист
chart.getChartData().getSeries().clear();
chart.getChartData().getCategories().clear();
// Получите рабочую книгу данных диаграммы для дальнейшей обработки.
IChartDataWorkbook fact = chart.getChartData().getChartDataWorkbook();
```
### Функция 2: Добавление серий и категорий
Добавление серий и категорий имеет решающее значение для наполнения ваших диаграмм значимыми данными.
#### Создание новой серии
Чтобы добавить новую серию под названием «Серия 1»:
```java
// Добавить новую серию в диаграмму
chart.getChartData().getSeries().add(fact.getCell(0, 0, 1, "Series 1"), chart.getType());
// Доступ к первой серии для сбора данных
IChartSeries series = chart.getChartData().getSeries().get_Item(0);
```
#### Заполнение категорий и точек данных
Чтобы добавить категории и соответствующие точки данных:
```java
// Добавьте названия категорий и соответствующие им точки данных.
chart.getChartData().getCategories().add(fact.getCell(0, 1, 0, "C1"));
series.getDataPoints().addDataPointForLineSeries(fact.getCell(0, 1, 1, 24));

chart.getChartData().getCategories().add(fact.getCell(0, 2, 0, "C2"));
series.getDataPoints().addDataPointForLineSeries(fact.getCell(0, 2, 1, 23));

chart.getChartData().getCategories().add(fact.getCell(0, 3, 0, "C3"));
series.getDataPoints().addDataPointForLineSeries(fact.getCell(0, 3, 1, -10));

// Изящная обработка нулевых точек данных
chart.getChartData().getCategories().add(fact.getCell(0, 4, 0, "C4"));
series.getDataPoints().addDataPointForLineSeries(fact.getCell(0, 4, 1, null));
```
### Функция 3: Добавление второй серии и заполнение точек данных
Добавление дополнительных рядов придает вашим диаграммам большую глубину.
#### Создание и заполнение второй серии
Чтобы добавить «Серию 2»:
```java
// Добавить еще одну серию под названием «Серия 2»
chart.getChartData().getSeries().add(fact.getCell(0, 0, 2, "Series 2"), chart.getType());

// Доступ ко второй серии для сбора данных
IChartSeries series2 = chart.getChartData().getSeries().get_Item(1);

// Добавить точки данных для «Серии 2»
series2.getDataPoints().addDataPointForLineSeries(fact.getCell(0, 1, 2, 30));
series2.getDataPoints().addDataPointForLineSeries(fact.getCell(0, 2, 2, 10));
series2.getDataPoints().addDataPointForLineSeries(fact.getCell(0, 3, 2, 60));
series2.getDataPoints().addDataPointForLineSeries(fact.getCell(0, 4, 2, 40));
```
### Функция 4: Настройка легенды диаграммы
Настройка легенды повышает читаемость диаграммы.
#### Настройка параметров легенды
Для настройки:
```java
// Включите легенду и отключите ее наложение на точки данных.
chart.setLegend(true);
chart.getLegend().setOverlay(false);
```
### Функция 5: Сохранение презентации
Когда диаграмма будет готова, сохраните презентацию в файл.
```java
try {
    // Сохраните измененную презентацию в указанном каталоге.
    pres.save("YOUR_DOCUMENT_DIRECTORY/DefaultMarkersInChart.pptx");
} finally {
    if (pres != null) pres.dispose();
}
```

## Настройка Aspose.Slides для Java

### Установка
Добавьте зависимость Aspose.Slides в свой проект с помощью Maven или Gradle.

#### Знаток
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```


## Руководство по внедрению
Мы разберем создание и настройку диаграмм с повернутыми текстами на простые шаги.

### Шаг 1: Инициализация презентации
Создайте экземпляр `Presentation` класс для работы с вашим файлом презентации.

```java
import com.aspose.slides.Presentation;
import com.aspose.slides.IChart;
import com.aspose.slides.ChartType;
import com.aspose.slides.ISlide;
import com.aspose.slides.SaveFormat;

// Определите путь к каталогу ваших документов
String dataDir = "YOUR_DOCUMENT_DIRECTORY"; // Замените на ваш реальный каталог

Presentation presentation = new Presentation();
```

### Шаг 2: Добавьте диаграмму на слайд
Откройте первый слайд и добавьте `ClusteredColumn` диаграмма.

```java
ISlide slide = presentation.getSlides().get_Item(0);
IChart chart = slide.getShapes().addChart(ChartType.ClusteredColumn, 50, 50, 500, 300);
```

### Шаг 3: Настройте вращение этикеток
Получите доступ к первой серии, чтобы изменить ее метки данных.

```java
import com.aspose.slides.IChartData;

IChartSeries series = chart.getChartData().getSeries().get_Item(0);
series.getLabels().getDefaultDataLabelFormat().setShowValue(true);

// Установить угол поворота текста метки на 65 градусов.
series.getLabels().getDefaultDataLabelFormat().getTextFormat()
        .getTextBlockFormat().setRotationAngle(65);
```

### Шаг 4: Настройте поворот заголовка диаграммы
Добавьте заголовок и задайте угол его поворота.

```java
chart.setTitle(true);
chart.getChartTitle().addTextFrameForOverriding("Custom title")
        .getTextFrameFormat().setRotationAngle(-30); // Повернуть текст заголовка на -30 градусов
```

### Шаг 5: Сохраните презентацию
Сохраните презентацию со всеми настройками.

```java
presentation.save(dataDir + "/textframe-rotation_out.pptx", SaveFormat.Pptx);

// Убедитесь, что вы закрыли объект презентации, чтобы освободить ресурсы.
disposePresentation(presentation);
```

#### Вспомогательный метод утилизации
Чтобы эффективно управлять ресурсами, избавляйтесь от `Presentation` возражайте правильно.

```java
private static void disposePresentation(Presentation pres) {
    if (pres != null) {
        pres.dispose();
    }
}
```

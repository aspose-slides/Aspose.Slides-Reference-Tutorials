# Создание круговой диаграммы на Java с помощью Aspose.Slides: подробное руководство

## Диаграммы и графики

### Введение

В визуализации данных круговые диаграммы являются интуитивно понятным способом представления пропорций в наборе данных. Однако при работе со сложными наборами данных, где некоторые сегменты значительно меньше других, традиционные круговые диаграммы могут стать загроможденными и трудными для интерпретации. Круговые диаграммы круговых диаграмм решают эту проблему, разделяя небольшие сегменты на вторичную диаграмму, что повышает читаемость.

В этом уроке вы узнаете, как создать и управлять круговой диаграммой Pie of Pie Chart с помощью Aspose.Slides для Java. Вы рассмотрите настройку среды, создание диаграммы, настройку свойств, таких как метки данных и позиции разделения, а также сохранение презентации в формате PPTX. К концу вы освоите эти функции с практическими приложениями и советами по производительности.

**Что вы узнаете:**
- Настройка Aspose.Slides для Java
- Создание круговой диаграммы
- Настройка свойств диаграммы, таких как метки данных и конфигурации разделения
- Сохранение презентации на диск

Готовы начать? Давайте сначала рассмотрим предварительные условия!

## Предпосылки

Перед созданием круговой диаграммы убедитесь, что у вас есть:

### Требуемые библиотеки, версии и зависимости:
- **Aspose.Slides для Java**: Необходим для программного управления презентациями PowerPoint.

### Требования к настройке среды:
- Java Development Kit (JDK), установленный на вашем компьютере. Мы рекомендуем использовать JDK 16 или более позднюю версию.
- Интегрированная среда разработки (IDE), например IntelliJ IDEA, Eclipse или NetBeans.

### Необходимые знания:
- Базовые знания программирования на Java
- Знакомство с Maven или Gradle для управления зависимостями

## Настройка Aspose.Slides для Java

### Информация об установке:

**Мейвен:**
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

**Градл:**
```gradle
implementation group: 'com.aspose', name: 'aspose-slides', version: '25.4', classifier: 'jdk16'
```

**Прямая загрузка**: Вы можете загрузить последнюю версию с сайта [Aspose.Slides для релизов Java](https://releases.aspose.com/slides/java/).

### Этапы получения лицензии:

- **Временная лицензия**Запросите временную лицензию для расширенной оценки.
- **Покупка**: Рассмотрите возможность приобретения лицензии, если Aspose.Slides соответствует вашим потребностям.

## Руководство по внедрению

### Создание круговой диаграммы

#### Обзор
Начнем с создания экземпляра `Presentation` и добавьте круговую диаграмму на первый слайд. Эта диаграмма эффективно визуализирует данные, разделяя меньшие сегменты на вторичную круговую диаграмму, что повышает читаемость.

#### Шаг 1: Создание экземпляра класса представления
```java
// Создать новую презентацию
ePresentation presentation = new Presentation();
```
Этот код инициализирует вашу презентацию, в которую мы добавим наши диаграммы.

#### Шаг 2: Добавьте круговую диаграмму на первый слайд
```java
// Добавьте круговую диаграмму на первый слайд в позицию (50, 50) размером (500x400)
eIChart chart = presentation.getSlides().get_Item(0).getShapes().addChart(
    ChartType.PieOfPie, 50, 50, 500, 400);
```
Здесь мы указываем тип диаграммы (`PieOfPie`) и его положение и размеры на слайде.

#### Шаг 3: Установите метки данных для отображения значений ряда
```java
// Настройте метки данных для отображения значений
echart.getChartData().getSeries().get_Item(0)
    .getLabels()
    .getDefaultDataLabelFormat()
    .setShowValue(true);
```
Этот шаг гарантирует, что каждый сегмент нашей круговой диаграммы отображает соответствующее ему значение, что способствует быстрой интерпретации данных.

#### Шаг 4: Настройте размер второй круговой диаграммы и разделите ее по процентам
```java
// Установите размер вторичной круговой диаграммы
echart.getChartData().getSeries().get_Item(0)
    .getParentSeriesGroup()
    .setSecondPieSize(149);

// Разделить пирог по процентному соотношению
echart.getChartData().getSeries().get_Item(0)
    .getParentSeriesGroup()
    .setPieSplitBy(PieSplitType.ByPercentage);

// Установить позицию разделения
echart.getChartData().getSeries().get_Item(0)
    .getParentSeriesGroup()
    .setPieSplitPosition(53);
```
Эти конфигурации позволяют вам настраивать способ разделения диаграммы и отображения более мелких сегментов, повышая наглядность для зрителей.

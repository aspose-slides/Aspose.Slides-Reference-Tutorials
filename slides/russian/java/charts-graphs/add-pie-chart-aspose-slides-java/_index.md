## Что такое Aspose Slides Maven?
Aspose.Slides for Java — мощный API, позволяющий разработчикам программно создавать, изменять и рендерить файлы PowerPoint. Пакет Maven (`aspose-slides`) упрощает управление зависимостями, позволяя сосредоточиться на построении и настройке слайдов — например, добавлении круговой диаграммы — без необходимости работать с низкоуровневой обработкой файлов.

## Почему стоит использовать Aspose.Slides Maven для добавления диаграммы на слайд?
- **Автоматизация:** Автоматически генерировать отчёты и дашборды.  
- **Точность:** Полный контроль над типами диаграмм, данными и стилем.  
- **Кроссплатформенность:** Работает в любой среде, совместимой с Java.  

## Предварительные требования
- **Aspose.Slides for Java** версии 25.4 или новее (Maven/Gradle).  
- Установленный JDK 16+.  
- IDE (IntelliJ IDEA, Eclipse и т.д.).  
- Базовые знания Java и знакомство с Maven или Gradle.

## Настройка Aspose.Slides for Java
Сначала добавьте Aspose.Slides в ваш проект через Maven или Gradle.

**Maven:**  
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

**Gradle:**  
```gradle
implementation group: 'com.aspose', name: 'aspose-slides', version: '25.4', classifier: 'jdk16'
```

Или вы можете [скачать последнюю версию](https://releases.aspose.com/slides/java/) напрямую с сайта Aspose.

## Руководство по реализации
Ниже решение разбито на две функции: добавление круговой диаграммы и доступ к её рабочей книге данных.

### Функция 1: Создание презентации и добавление диаграммы
#### Обзор
Эта часть показывает, как создать новую презентацию и **add a pie chart** на первый слайд.

#### Пошагово

**Шаг 1: Инициализировать новый объект Presentation**  
```java
Presentation pres = new Presentation();
```
*Создаёт экземпляр `Presentation`, который будет содержать все слайды.*

**Шаг 2: Добавить круговую диаграмму**  
```java
IChart chart = pres.getSlides().get_Item(0).getShapes().addChart(
    ChartType.Pie,
    50,
    50,
    400,
    500
);
```
*Размещает круговую диаграмму в координатах (50, 50) с шириной 400 и высотой 500. Перечисление `ChartType.Pie` указывает Aspose отрисовать круговую диаграмму.*

**Шаг 3: Освободить ресурсы**  
```java
if (pres != null) pres.dispose();
```
*Освобождает нативные ресурсы; всегда вызывайте `dispose()` после завершения работы.*

### Функция 2: Доступ к рабочей книге данных диаграммы и листам
#### Обзор
Узнайте, как получить доступ к базовой рабочей книге, хранящей данные диаграммы, и пройтись по её листам.

#### Пошагово

**Шаг 1: (Повторно) Инициализировать новый объект Presentation**  
*То же, что в Функции 1, Шаг 1.*

**Шаг 2: (Повторно) Добавить круговую диаграмму**  
*То же, что в Функции 1, Шаг 2.*

**Шаг 3: Получить рабочую книгу данных диаграммы**  
```java
IChartDataWorkbook workbook = chart.getChartData().getChartDataWorkbook();
```
*Извлекает `IChartDataWorkbook`, связанный с диаграммой.*

**Шаг 4: Перебрать листы**  
```java
for (int i = 0; i < workbook.getWorksheets().size(); i++) {
    System.out.println(workbook.getWorksheets().get_Item(i).getName());
}
```
*Выводит имя каждого листа, позволяя проверить структуру данных.*

**Шаг 5: Освободить ресурсы**  
*То же, что в Функции 1, Шаг 3.*

# Как добавить круговую диаграмму в презентацию с помощью Aspose.Slides Java

### Что вы узнаете:

- Действия по добавлению и настройке круговой диаграммы на первом слайде презентации.
- Доступ к рабочим книгам с данными диаграмм и перечисление рабочих листов в них.

Давайте углубимся в то, как можно использовать Aspose.Slides Java для улучшения ваших презентаций с помощью динамических диаграмм!

## Предпосылки
Прежде чем начать, убедитесь, что у вас есть следующее:

### Требуемые библиотеки:
- **Aspose.Slides для Java** версия 25.4 или более поздняя.
  
### Настройка среды:
- В вашей системе установлена JDK 16 или более поздняя версия.
- IDE, например IntelliJ IDEA, Eclipse или любая другая предпочитаемая среда разработки.

### Необходимые знания:
- Базовые знания программирования на Java.
- Знакомство с системами сборки Maven или Gradle для управления зависимостями.

## Настройка Aspose.Slides для Java
Во-первых, вам нужно включить Aspose.Slides в ваш проект. Вы можете сделать это через Maven или Gradle:

**Мейвен:**
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

**Градл:**
```gradle
implementation group: 'com.aspose', name: 'aspose-slides', version: '25.4', classifier: 'jdk16'
```

В качестве альтернативы вы можете [загрузить последнюю версию](https://releases.aspose.com/slides/java/) непосредственно с сайта Aspose.

## Руководство по внедрению
Давайте разберем реализацию на две основные функции: добавление круговой диаграммы в презентацию и доступ к данным диаграммы.

### Функция 1: Создание презентации и добавление диаграммы
#### Обзор
В этом разделе показано, как инициализировать новый объект презентации и добавить круговую диаграмму на первый слайд.

#### Пошаговое руководство:
**Шаг 1: Инициализация нового объекта презентации**
```java
Presentation pres = new Presentation();
```
*Здесь мы создаем экземпляр `Presentation`, который служит нашим основным контейнером документов.*

**Шаг 2: Добавьте круговую диаграмму**
```java
IChart chart = pres.getSlides().get_Item(0).getShapes().addChart(
    ChartType.Pie,
    50,
    50,
    400,
    500
);
```
*Добавляем круговую диаграмму на первый слайд с указанными координатами (50, 50) с размерами ширины 400 и высоты 500. `ChartType.Pie` определяет тип диаграммы.*

**Шаг 3: Избавление от ресурсов**
```java
if (pres != null) pres.dispose();
```
*Важно освободить ресурсы, утилизировав объект презентации после завершения операций.*

### Функция 2: Доступ к рабочей книге и рабочим листам данных диаграмм
#### Обзор
Узнайте, как получить доступ к базовой рабочей книге данных, связанной с вашей диаграммой, и выполнить итерацию по ее рабочим листам.

#### Пошаговое руководство:
**Шаг 1: Инициализация нового объекта презентации**
*Повторно используйте шаг инициализации из предыдущей функции.*

**Шаг 2: Добавьте круговую диаграмму**
*Как и прежде, добавьте круговую диаграмму, чтобы начать работу с рабочими книгами данных.*

**Шаг 3: Получите рабочую книгу Chart Data**
```java
IChartDataWorkbook workbook = chart.getChartData().getChartDataWorkbook();
```
*Это извлекает `IChartDataWorkbook` объект, связанный с нашей диаграммой, позволяющий получить доступ к ее данным.*

**Шаг 4: Просмотрите рабочие листы**
```java
for (int i = 0; i < workbook.getWorksheets().size(); i++) {
    System.out.println(workbook.getWorksheets().get_Item(i).getName());
}
```
*Здесь мы просматриваем каждый рабочий лист в рабочей книге и выводим его название.*

**Шаг 5: Избавление от ресурсов**
*Утилизируйте объект презентации, как описано ранее, чтобы освободить ресурсы.*

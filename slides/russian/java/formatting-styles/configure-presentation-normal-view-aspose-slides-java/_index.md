# Как настроить нормальное состояние представления презентации с помощью Aspose.Slides для Java

## Введение

Настройка начального вида презентации может значительно повысить ее эффективность, будь то для собраний или образовательных модулей. Это руководство проведет вас через использование Aspose.Slides для Java для настройки обычного состояния просмотра ваших презентаций, что повысит удобство использования и профессионализм.

**Что вы узнаете:**
- Установка состояний горизонтальной и вертикальной разделительной полосы.
- Настройка восстановленных основных свойств, таких как автоматическая настройка и размер измерения.
- Включение контурных значков в обычном состоянии просмотра.
- Эффективное сохранение этих конфигураций.

Прежде чем начать, давайте рассмотрим предварительные условия для этого урока.

## Предпосылки

Убедитесь, что у вас есть:

### Необходимые библиотеки и зависимости
- **Aspose.Slides для Java**: Необходим для программного управления презентациями PowerPoint.
- **Комплект разработчика Java (JDK)**: Требуется JDK 16 или выше.

### Требования к настройке среды
- Интегрированная среда разработки (IDE), например IntelliJ IDEA, Eclipse или NetBeans, настроенная для разработки на Java.

### Необходимые знания
- Базовое понимание концепций программирования на Java.
- Знакомство с инструментами сборки Maven или Gradle для управления зависимостями.

## Настройка Aspose.Slides для Java

Прежде чем погрузиться в реализацию кода, вам нужно настроить библиотеку Aspose.Slides в вашем проекте. Вот как:

### Настройка Maven
Добавьте эту зависимость к вашему `pom.xml` файл:
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

### Настройка Gradle
Включите это в свой `build.gradle`:
```gradle
implementation group: 'com.aspose', name: 'aspose-slides', version: '25.4', classifier: 'jdk16'
```

### Прямая загрузка
В качестве альтернативы загрузите последнюю версию библиотеки Aspose.Slides для Java с их сайта [официальная страница релизов](https://releases.aspose.com/slides/java/).

#### Приобретение лицензии
- **Бесплатная пробная версия**: Начните с бесплатной пробной версии, чтобы изучить все возможности.
- **Временная лицензия**: Получите временную лицензию для расширенной оценки.
- **Покупка**: Рассмотрите возможность приобретения лицензии для долгосрочного использования.

После загрузки и настройки в вашем проекте инициализируйте Aspose.Slides, как показано ниже:
```java
import com.aspose.slides.Presentation;

// Инициализация класса презентации
Presentation pres = new Presentation();
```

## Руководство по внедрению

Теперь, когда все готово, давайте настроим нормальное состояние просмотра презентации.

### Настройка состояний панели разделителя

#### Обзор
Разделители помогают перемещаться по слайдам и заметкам. Вот как установить их состояния:

- **Горизонтальная разделительная полоса**: Управляет навигацией по слайдам.
- **Вертикальный разделитель**: Управляет видимостью панели заметок.

##### Установить состояние горизонтальной разделительной полосы
```java
pres.getViewProperties().getNormalViewProperties()
    .setHorizontalBarState(SplitterBarStateType.Restored);
```
**Объяснение:** Установка этого значения `Restored` обеспечивает полную видимость навигации по слайдам при открытии презентации.

##### Установить состояние вертикальной разделительной полосы
```java
pres.getViewProperties().getNormalViewProperties()
    .setVerticalBarState(SplitterBarStateType.Maximized);
```
**Объяснение:** В развернутом состоянии отображаются все заметки, что облегчает доступ к подробной информации о слайде.

### Настройка восстановленных основных свойств

#### Обзор
Настройка восстановленных верхних свойств улучшает пользовательский опыт, устанавливая начальный вид слайдов и заметок.

##### Автоматическая настройка и размер
```java
pres.getViewProperties().getNormalViewProperties()
    .getRestoredTop().setAutoAdjust(true);
pres.getViewProperties().getNormalViewProperties()
    .getRestoredTop().setDimensionSize(80);
```
**Объяснение:** Включение `auto-adjust` обеспечивает гибкую компоновку, адаптируемую к различным размерам экрана, а настройка размера управляет видимостью области заметок.

### Включение значков контура

#### Обзор
Значки контура помогают быстро перемещаться по структуре слайдов.

##### Включить значки контура
```java
pres.getViewProperties().getNormalViewProperties()
    .setShowOutlineIcons(true);
```
**Объяснение:** Этот параметр добавляет видимость значкам контура, способствуя быстрому доступу к контенту и его организации.

### Сохранение презентации
Наконец, сохраните презентацию с обновленными конфигурациями:
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY/presentation_normal_view_state.pptx";
pres.save(dataDir, SaveFormat.Pptx);
```
**Объяснение:** Это сохраняет изменения в указанном месте в формате PPTX.

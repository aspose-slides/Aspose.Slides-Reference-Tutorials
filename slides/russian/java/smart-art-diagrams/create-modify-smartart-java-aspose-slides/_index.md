## Руководство по внедрению
В этом разделе мы разберем каждую функциональность на логические шаги, которые помогут вам легко интегрировать SmartArt в ваши приложения Java.

### Создание и добавление SmartArt в презентацию
**Обзор:** Эта функция демонстрирует, как инициализировать новую презентацию и добавить фигуру SmartArt с указанными размерами и типом макета.
#### Пошаговая реализация
1. **Инициализировать презентацию**
   Начните с создания экземпляра `Presentation`:
   ```java
   Presentation presentation = new Presentation();
   ```
2. **Доступ к первому слайду**
   Найдите первый слайд, на который вы добавите свой SmartArt:
   ```java
   ISlide slide = presentation.getSlides().get_Item(0);
   ```
3. **Добавить фигуру SmartArt**
   Добавьте фигуру SmartArt с определенными размерами и типом макета:
   ```java
   ISmartArt smart = slide.getShapes().addSmartArt(
       10, // x-позиция
       10, // y-позиция
       400, // ширина
       300, // высота
       SmartArtLayoutType.BasicBlockList // начальный тип макета
   );
   ```
4. **Утилизация презентационного объекта**
   Всегда проверяйте, что вы избавляетесь от ресурсов:
   ```java
   if (presentation != null) presentation.dispose();
   ```
### Изменить тип макета SmartArt
**Обзор:** Узнайте, как изменить тип макета существующей фигуры SmartArt на слайде.
#### Пошаговая реализация
1. **Извлеките форму SmartArt**
   Получите доступ к первой фигуре на слайде, предполагая, что это SmartArt:
   ```java
   ISmartArt smart = (ISmartArt)slide.getShapes().get_Item(0);
   ```
2. **Изменить тип макета**
   Измените макет на `BasicProcess` или любой другой доступный тип:
   ```java
   smart.setLayout(SmartArtLayoutType.BasicProcess);
   ```
### Сохранить презентацию с измененным SmartArt
**Обзор:** Эта функция демонстрирует, как сохранить изменения в файле.
#### Пошаговая реализация
1. **Определить выходной путь**
   Укажите, где вы хотите сохранить презентацию:
   ```java
   String outputPath = "YOUR_OUTPUT_DIRECTORY/ChangeSmartArtLayout_out.pptx";
   ```
2. **Сохранить презентацию**
   Зафиксируйте изменения, сохранив их по указанному пути:
   ```java
   presentation.save(outputPath, SaveFormat.Pptx);
   ```

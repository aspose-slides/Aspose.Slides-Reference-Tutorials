## Предпосылки

Чтобы следовать этому руководству, включите библиотеку Aspose.Slides в свой проект Java. Используйте Maven или Gradle для управления зависимостями:

- **Знаток**
  Добавьте следующее к вашему `pom.xml` файл:
  ```xml
  <dependency>
      <groupId>com.aspose</groupId>
      <artifactId>aspose-slides</artifactId>
      <version>25.4</version>
      <classifier>jdk16</classifier>
  </dependency>
  ```

- **Градл**
  Включите это в свой `build.gradle`:
  ```gradle
  implementation group: 'com.aspose', name: 'aspose-slides', version: '25.4', classifier: 'jdk16'
  ```

Загрузите последнюю версию с сайта [Aspose.Slides для релизов Java](https://releases.aspose.com/slides/java/) если необходимо.

## Руководство по внедрению

### Доступ к SmartArt в презентациях

В этом разделе показано, как перебирать фигуры SmartArt с помощью Aspose.Slides для Java. Мы рассмотрим каждый шаг:

#### Обзор функций

Наша цель — получить доступ к объектам SmartArt на первом слайде и получить сведения о каждом узле в этих графиках.

#### Шаги по внедрению Access SmartArt

1. **Загрузить файл презентации:**
   Начните с загрузки файла презентации:
   ```java
   String dataDir = "YOUR_DOCUMENT_DIRECTORY";
   com.aspose.slides.Presentation pres = new com.aspose.slides.Presentation(dataDir + "/AccessSmartArt.pptx");
   ```

2. **Перебрать формы слайдов:**
   Получите доступ ко всем фигурам на первом слайде и проверьте наличие экземпляров SmartArt:
   ```java
   for (com.aspose.slides.IShape shape : pres.getSlides().get_Item(0).getShapes()) {
       if (shape instanceof com.aspose.slides.ISmartArt) {
           com.aspose.slides.ISmartArt smart = (com.aspose.slides.ISmartArt) shape;
           // Продолжайте итерацию по узлам
       }
   }
   ```

3. **Доступ к узлам SmartArt:**
   Для каждого объекта SmartArt выполните цикл по его узлам и извлеките данные:
   ```java
   for (int i = 0; i < smart.getAllNodes().size(); i++) {
       com.aspose.slides.ISmartArtNode node = (com.aspose.slides.ISmartArtNode) smart.getAllNodes().get_Item(i);
       String outString = String.format("i = {0}, Text: {1}, Level = {2}, Position = {3}", 
           i, node.getTextFrame().getText(), node.getLevel(), node.getPosition());
   }
   ```

4. **Распоряжение ресурсами:**
   Обязательно утилизируйте `Presentation` возражают против бесплатных ресурсов:
   ```java
   if (pres != null) pres.dispose();
   ```

### Добавить SmartArt в презентацию
#### Обзор
Эта функция позволяет добавлять визуально привлекательную графику SmartArt к слайдам презентации.

**Шаг 1**: Создайте слайд и добавьте SmartArt
- **Цель**: Добавить SmartArt типа «Радиальный цикл» в указанных координатах с определенными размерами.

```java
import com.aspose.slides.Presentation;
import com.aspose.slides.ISmartArt;
import com.aspose.slides.SmartArtLayoutType;

Presentation presentation = new Presentation();
try {
    // Создайте и добавьте графический элемент SmartArt на первый слайд.
    ISmartArt smart = presentation.getSlides().get_Item(0).getShapes().addSmartArt(
        10, 10, 400, 300, SmartArtLayoutType.RadialCycle
    );
} finally {
    if (presentation != null) presentation.dispose();
}
```

**Объяснение**: 
- `addSmartArt(int x, int y, int width, int height, SmartArtLayoutType layoutType)` добавляет графический элемент SmartArt в позицию `(x, y)` с указанными размерами и типом.

### Добавить узел в SmartArt
#### Обзор
Узнайте, как динамически добавлять узлы к существующей графике SmartArt для более сложного представления информации.

**Шаг 2**: Извлечение узлов и добавление нового узла
- **Цель**: Улучшите свой SmartArt, добавив дополнительные элементы (узлы).

```java
import com.aspose.slides.ISmartArtNode;

try {
    // Предположим, что понятие «умный» уже определено в предыдущем разделе.
    ISmartArtNode node = smart.getAllNodes().addNode();
} finally {
    if (presentation != null) presentation.dispose();
}
```

**Объяснение**: 
- `getAllNodes()` извлекает все узлы в SmartArt и `addNode()` добавляет новый.

### Проверьте скрытое свойство узла SmartArt
#### Обзор
Эта функция помогает вам управлять видимостью отдельных узлов в графике SmartArt.

**Шаг 3**: Проверьте, скрыт ли узел
- **Цель**: Определите, скрыты ли определенные узлы от просмотра.

```java
import com.aspose.slides.ISmartArtNode;

try {
    // Предположим, что «узел» уже определен.
    boolean hidden = node.isHidden();

    if (hidden) {
        System.out.println("The node is currently hidden.");
    }
} finally {
    if (presentation != null) presentation.dispose();
}
```

**Объяснение**: 
- `isHidden()` возвращает логическое значение, указывающее статус видимости узла SmartArt.

### Сохранить презентацию в файл
#### Обзор
Сохраните улучшенную презентацию в формате PPTX для распространения или дальнейшего редактирования.

**Шаг 4**: Определить выходной путь и сохранить
- **Цель**: Сохраните изменения, сохранив измененный файл презентации.

```java
import com.aspose.slides.Presentation;
import com.aspose.slides.SaveFormat;

try {
    String dataDir = "YOUR_DOCUMENT_DIRECTORY"; 
    // Замените на фактический путь к каталогу.
    
    presentation.save(dataDir + "CheckSmartArtHiddenProperty_out.pptx", SaveFormat.Pptx);
} finally {
    if (presentation != null) presentation.dispose();
}
```

**Объяснение**: 
- `save(String path, int format)` записывает презентацию в указанный файл в желаемом формате.

## Руководство по внедрению
### Добавление SmartArt в презентацию
**Обзор**
В этом разделе показано, как вставить элемент SmartArt типа OrganizationChart в первый слайд презентации.

**Шаг 1: Создание нового экземпляра презентации**
```java
Presentation presentation = new Presentation();
```
- **Почему:** Это инициализирует новый объект презентации, который мы изменим, добавив формы и содержимое.

**Шаг 2: Получите доступ к первому слайду**
```java
ISlide slide = presentation.getSlides().get_Item(0);
```
- **Почему:** На первом слайде обычно размещается основное содержимое, включая графику SmartArt.

**Шаг 3: Добавьте графику SmartArt организационной диаграммы**
```java
ISmartArt smart = slide.getShapes().addSmartArt(10, 10, 400, 300, SmartArtLayoutType.OrganizationChart);
```
- **Почему:** Этот вызов метода добавляет новый графический элемент SmartArt на слайд с указанными размерами и типом макета. Параметры (x, y, ширина, высота) определяют его положение и размер.

### Настройка типа макета организационной диаграммы
**Обзор**
Здесь вы узнаете, как изменить макет существующей организационной диаграммы в графике SmartArt.

**Шаг 4: Измените макет первого узла**
```java
smart.getNodes().get_Item(0).setOrganizationChartLayout(OrganizationChartLayoutType.LeftHanging);
```
- **Почему:** На этом этапе настраивается макет, предлагая более индивидуальное визуальное представление иерархических данных. 

### Сохранение презентации в файл
**Обзор**
В этой последней функции вы сохраните свою презентацию с добавленной графикой SmartArt.

**Шаг 5: Сохраните свою работу**
```java
String outputDir = "YOUR_OUTPUT_DIRECTORY";
presentation.save(outputDir + "OrganizeChartLayoutType_out.pptx", SaveFormat.Pptx);
```
- **Почему:** Это гарантирует, что все изменения будут сохранены в файле, которым можно поделиться или представить.

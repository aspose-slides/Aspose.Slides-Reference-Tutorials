# 在 PowerPoint 演示文稿中创建包含垂直文本的表格

Aspose.Slides for .NET 提供了强大的功能，可用于以编程方式创建和操作 PowerPoint 演示文稿。此示例演示了如何创建具有自定义格式的表格，包括垂直文本对齐和旋转——这是专业商务演示和数据可视化的常见需求。

## 安装说明

1. 通过 NuGet 包管理器安装 Aspose.Slides for .NET：
   ```
   Install-Package Aspose.Slides
   ```
   或者通过 .NET CLI：
   ```
   dotnet add package Aspose.Slides
   ```

2. 添加对您的项目的引用：
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

3. 下载文档资源：
   - [API 参考](https://reference.aspose.com/slides/net/)
   - [开发者指南](https://docs.aspose.com/slides/net/)

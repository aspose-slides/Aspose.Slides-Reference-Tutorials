Aspose.Slides for .NET 是一个功能强大的 API，允许开发人员以编程方式创建、读取、编写、修改和转换 PowerPoint 演示文稿，而无需 Microsoft PowerPoint。此示例演示如何在 PowerPoint 演示文稿中创建具有自定义格式边框的标准化表格。

## 安装说明

1. 通过 NuGet 包管理器安装：
   ```
   Install-Package Aspose.Slides
   ```

2. 或者通过 .NET CLI：
   ```
   dotnet add package Aspose.Slides
   ```

3. 将以下命名空间添加到您的 C# 文件：
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Tables;
   using System.Drawing;
   ```

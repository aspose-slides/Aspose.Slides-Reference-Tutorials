# 使用 Aspose.Slides 在 PowerPoint 中创建格式化文本框

Aspose.Slides for .NET 是一个功能强大的 API，允许开发人员创建、修改和转换 Microsoft PowerPoint 演示文稿，而无需安装 Microsoft Office。此示例演示了如何创建带有格式化文本框的演示文稿，并应用自定义字体样式、大小和颜色以增强视觉吸引力。

## 安装说明

1. 通过 NuGet 包管理器安装：
   ```
   Install-Package Aspose.Slides
   ```

2. 通过 .NET CLI 安装：
   ```
   dotnet add package Aspose.Slides
   ```

3. 手动添加对项目的引用：
   - 从下载最新版本 [Aspose.Slides for .NET](https://releases.aspose.com/slides/net/)
   - 在您的项目中添加 DLL 作为引用

4. 初始化许可证：
   ```csharp
   // 初始化许可证
   License license = new License();
   license.SetLicense("Aspose.Slides.lic");
   ```

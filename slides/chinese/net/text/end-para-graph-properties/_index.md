# 使用格式化文本创建 PowerPoint 演示文稿

Aspose.Slides for .NET 是一个功能强大的 API，允许开发人员创建、操作和转换 PowerPoint 演示文稿，而无需 Microsoft PowerPoint。此示例演示了如何创建一个包含矩形和格式化文本段落的简单演示文稿。

该代码会创建一个新的演示文稿，在第一张幻灯片中添加一个矩形，然后添加多个具有不同文本格式选项的段落。它演示了如何为文本的特定部分设置字体属性，例如大小和字体类型。

## 安装说明

1. 通过 NuGet 包管理器安装：
   ```
   Install-Package Aspose.Slides
   ```

2. 通过 .NET CLI 安装：
   ```
   dotnet add package Aspose.Slides
   ```

3. 通过程序包管理器控制台安装：
   ```
   PM> Install-Package Aspose.Slides
   ```

4. 添加对项目的引用：
   安装后，添加对项目的引用：
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

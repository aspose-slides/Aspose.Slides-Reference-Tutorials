# 使用 Aspose.Slides 在 PowerPoint 中旋转图表标题和标签

Aspose.Slides for .NET 提供了强大的功能，可以通过编程方式创建和操作 PowerPoint 演示文稿。本示例演示了如何创建包含簇状柱形图的演示文稿，该图具有旋转文本元素的功能 - 具体来说，旋转图表标题和数据标签以提高可读性和视觉吸引力。

## 安装说明

1. 通过 NuGet 包管理器安装：
   ```
   Install-Package Aspose.Slides
   ```

2. 或者通过 .NET CLI 安装：
   ```
   dotnet add package Aspose.Slides
   ```

3. 将命名空间添加到您的 C# 文件：
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Charts;
   ```

# 在 PowerPoint 演示文稿中创建自定义项目符号

Aspose.Slides for .NET 提供了强大的功能，可用于以编程方式创建和操作 PowerPoint 演示文稿。本示例重点介绍如何在文本框中实现自定义项目符号，并展示如何控制项目符号的类型、颜色、缩进和大小。该代码创建一个矩形演示文稿，其中包含两个段落 - 一个段落带有符号项目符号样式，另一个段落带有编号项目符号样式。

## 安装说明

1. 使用 NuGet 包管理器安装 Aspose.Slides for .NET：
   ```
   Install-Package Aspose.Slides
   ```

2. 或者，直接下载软件包：
   - 访问 [Aspose.Slides下载页面](https://downloads.aspose.com/slides/net)
   - 下载最新版本
   - 提取并引用项目中的 Aspose.Slides.dll

3. 添加必要的using语句：
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

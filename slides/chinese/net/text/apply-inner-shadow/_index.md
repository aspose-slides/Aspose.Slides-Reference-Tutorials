# 使用 Aspose.Slides for .NET 中的 TextBox 创建 PowerPoint 演示文稿

Aspose.Slides for .NET 是一个功能强大的 API，允许开发人员创建、修改和转换 PowerPoint 演示文稿，而无需 Microsoft PowerPoint。此示例演示了如何使用自定义文本框创建演示文稿、应用格式并将其保存为 PPTX 文件。

## 安装说明

1. 通过 NuGet 包管理器安装：
   ```
   Install-Package Aspose.Slides
   ```

2. 通过 .NET CLI 安装：
   ```
   dotnet add package Aspose.Slides
   ```

3. 手动安装：
   - 从下载最新版本 [Aspose.Slides for .NET 发布页面](https://releases.aspose.com/slides/net/)
   - 将 DLL 引用添加到您的项目

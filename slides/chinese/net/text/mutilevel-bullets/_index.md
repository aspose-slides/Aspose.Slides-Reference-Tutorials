# 使用 Aspose.Slides 在 PowerPoint 中创建多级项目符号

Aspose.Slides for .NET 提供了强大的 API，可通过编程方式创建和操作 PowerPoint 演示文稿。本示例演示了如何创建包含多级项目符号的演示文稿，包括如何添加带有文本框的自动形状，以及如何使用不同的项目符号样式和缩进级别来格式化段落。您可以轻松创建具有自定义项目符号的分层内容，并为每个缩进级别添加自定义项目符号，这非常适合创建结构化且专业的演示文稿，而无需安装 Microsoft PowerPoint。

## 安装说明

1. 通过 NuGet 包管理器安装：
   ```
   Install-Package Aspose.Slides
   ```

2. 通过 .NET CLI 安装：
   ```
   dotnet add package Aspose.Slides
   ```

3. 通过程序包管理器控制台安装：
   ```
   PM> Install-Package Aspose.Slides
   ```

4. 手动安装：
   - 从下载最新版本 [Aspose.Slides for .NET](https://releases.aspose.com/slides/net/)
   - 在您的项目中添加对 Aspose.Slides.dll 的引用

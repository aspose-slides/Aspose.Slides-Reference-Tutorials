# 使用 Aspose.Slides 在 PowerPoint 中创建垂直文本

Aspose.Slides for .NET 提供了强大的功能，可让您以编程方式创建和操作 PowerPoint 演示文稿。此示例演示了如何创建一个包含垂直旋转文本的矩形演示文稿。文本以 270 度角排列，使其从下向上阅读。此技术非常适合在演示文稿中创建侧边标签、独特的文本布局或专门的设计元素。

## 安装说明

1. 安装 Aspose.Slides for .NET
   
   使用 NuGet 包管理器安装 Aspose.Slides 包：
   
   ```csharp
   Install-Package Aspose.Slides
   ```
   
   或者使用 .NET CLI：
   
   ```csharp
   dotnet add package Aspose.Slides
   ```

2. 添加引用
   
   将以下 using 语句添加到您的 C# 文件中：
   
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

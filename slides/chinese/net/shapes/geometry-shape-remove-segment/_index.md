# 使用 Aspose.Slides 修改 PowerPoint 形状中的几何路径

Aspose.Slides for .NET 提供了强大的功能，可以通过编程修改 PowerPoint 演示文稿，包括高级形状操作。此示例演示了如何通过修改现有形状的几何路径来创建自定义形状 - 具体来说，通过从心形中删除一段来创建独特的变体。

几何路径通过一系列点和线段定义形状的轮廓。通过访问和修改这些路径，开发人员可以创建标准 PowerPoint 形状库中没有的高度自定义形状。

## 安装说明

1. 通过 NuGet 包管理器安装：
   ```
   Install-Package Aspose.Slides
   ```

2. 通过 .NET CLI 安装：
   ```
   dotnet add package Aspose.Slides
   ```

3. 在您的代码中添加 using 指令：
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

4. 在应用程序启动时添加许可证初始化：
   ```csharp
   // 如果你有驾照
   License license = new License();
   license.SetLicense("path/to/Aspose.Slides.lic");
   ```

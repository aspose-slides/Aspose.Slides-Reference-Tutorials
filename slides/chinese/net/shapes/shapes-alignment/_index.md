# 在 PowerPoint 演示文稿中对齐和分组形状

Aspose.Slides for .NET 提供了强大的功能，可用于以编程方式创建和操作 PowerPoint 演示文稿。本示例演示了如何在幻灯片中对齐形状以及如何在 PowerPoint 演示文稿中处理分组形状。您将学习如何将矩形定位到特定坐标，如何将它们相对于幻灯片或彼此对齐，以及如何选择性地对齐组中的形状。

## 安装说明

1. 使用以下方法之一安装 Aspose.Slides for .NET：
   
   - NuGet 包管理器：
     ```
     Install-Package Aspose.Slides.NET
     ```
   
   - 程序包管理器控制台：
     ```
     PM> Install-Package Aspose.Slides.NET
     ```
   
   - .NET CLI：
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. 向您的项目添加引用：
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.IO;
   ```

3. 下载下面的示例代码并将其集成到您的项目中。

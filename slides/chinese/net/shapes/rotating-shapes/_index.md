# 在 PowerPoint 中创建和旋转矩形

Aspose.Slides for .NET 是一个功能强大的 API，允许开发人员创建、修改和转换 PowerPoint 演示文稿，而无需安装 Microsoft PowerPoint。此示例演示了如何创建带有旋转矩形的演示文稿，展示了该库的形状操作功能。

使用 Aspose.Slides，您可以以编程方式向幻灯片添加各种形状，并应用旋转等变换。此功能对于创建能够精确控制视觉元素的动态演示文稿尤其有用。

## 安装说明

1. 通过 NuGet 包管理器安装：
   ```
   Install-Package Aspose.Slides
   ```

2. 通过 .NET CLI 安装：
   ```
   dotnet add package Aspose.Slides
   ```

3. 在您的代码中添加 using 指令：
   ```csharp
   using Aspose.Slides;
   ```

# 在 PowerPoint 演示文稿中向 GeometryShape 添加线段

此示例演示如何使用 Aspose.Slides for .NET 访问和修改 PowerPoint 演示文稿中形状的几何路径。通过向矩形添加自定义线段，您可以创建标准形状类型无法实现的更复杂的几何图形。此技术对于在演示文稿中创建自定义图表、流程图和其他视觉元素非常有用。

## 安装说明

1. 使用以下方法之一安装 Aspose.Slides for .NET：

   - NuGet 包管理器：
     ```
     Install-Package Aspose.Slides.NET
     ```

   - 程序包管理器控制台：
     ```
     PM> Install-Package Aspose.Slides.NET
     ```

   - .NET CLI：
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. 将命名空间添加到您的 C# 文件：
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

# 使用 Aspose.Slides 在 PowerPoint 中创建椭圆形

Aspose.Slides for .NET 是一个强大的 API，它使开发人员无需 Microsoft PowerPoint 即可创建、读取、编写、修改和转换 PowerPoint 演示文稿。此示例演示了如何在演示文稿中创建椭圆形、应用包括填充颜色和线条属性在内的格式，并将结果保存为 PPTX 文件。该代码展示了 Aspose.Slides 能够精确控制形状的外观和位置。

## 安装说明

1. 通过 NuGet 包管理器安装：
   ```
   Install-Package Aspose.Slides
   ```

   或者使用 .NET CLI：
   ```
   dotnet add package Aspose.Slides
   ```

2. 添加参考文献：
   将以下命名空间添加到您的 C# 文件：
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

3. 许可证设置（可选）：
   如果您有许可证，请使用以下命令申请：
   ```csharp
   License license = new License();
   license.SetLicense("Aspose.Slides.lic");
   ```

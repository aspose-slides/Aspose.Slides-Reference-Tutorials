# 使用 Aspose.Slides 在 PowerPoint 中创建平铺图像背景

Aspose.Slides for .NET 提供了强大的功能，可让您以编程方式创建和操作 PowerPoint 演示文稿。此示例演示了如何创建一个矩形形状的演示文稿，该矩形形状填充了平铺图像图案。代码展示了如何加载图像、将其作为矩形背景添加到幻灯片中，以及如何应用各种平铺属性，例如偏移、缩放、对齐和翻转选项。此技术可用于在演示文稿中创建自定义背景、水印或装饰元素，并可精确控制平铺行为。

## 安装说明

1. 使用 NuGet 包管理器安装 Aspose.Slides for .NET：
   ```
   Install-Package Aspose.Slides.NET
   ```

2. 或者通过 .NET CLI 安装：
   ```
   dotnet add package Aspose.Slides.NET
   ```

3. 将以下使用指令添加到您的 C# 文件：
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.IO;
   ```

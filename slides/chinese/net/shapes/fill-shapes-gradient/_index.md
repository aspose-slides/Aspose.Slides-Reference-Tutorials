# 使用 Aspose.Slides 在 PowerPoint 中创建渐变填充椭圆形

此示例演示如何以编程方式创建包含自定义渐变填充效果的椭圆形 PowerPoint 演示文稿。代码创建一个新的演示文稿，在第一张幻灯片中添加一个椭圆形，并应用带有红色和紫色色标的线性渐变填充。Aspose.Slides for .NET 提供了全面的 API，用于创建和操作 PowerPoint 演示文稿，并可精确控制形状、格式和效果。

## 安装说明

1. 使用以下方法之一安装 Aspose.Slides for .NET：
   - NuGet 包管理器： `Install-Package Aspose.Slides.NET`
   - .NET CLI： `dotnet add package Aspose.Slides.NET`
   - 程序包管理器控制台： `Install-Package Aspose.Slides.NET`

2. 将以下 using 语句添加到您的 C# 文件中：
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

# 使用 Aspose.Slides 在 PowerPoint 中创建带有平铺图像填充的形状

Aspose.Slides for .NET 提供了强大的功能，可用于以编程方式创建和操作 PowerPoint 演示文稿。此示例演示了如何创建一个矩形并使用平铺图像图案填充它。平铺图像填充对于在演示文稿中创建纹理背景、水印或装饰元素特别有用。

## 安装说明

1. 通过 NuGet 包管理器安装：
   ```
   Install-Package Aspose.Slides
   ```

2. 通过 .NET CLI 安装：
   ```
   dotnet add package Aspose.Slides
   ```

3. 手动安装：
   - 从下载最新版本 [Aspose.Slides for .NET](https://releases.aspose.com/slides/net/)
   - 在您的项目中添加对 Aspose.Slides.dll 的引用

# 使用 Aspose.Slides 在 PowerPoint 中创建组形状

Aspose.Slides for .NET 提供了全面的 API，用于以编程方式创建和操作 PowerPoint 演示文稿。此示例演示了如何创建组形状、在其中添加多个矩形、正确定位形状、添加边框以及保存演示文稿。组形状对于将需要作为单个单元进行操作的相关形状组织起来至关重要，从而提供更好的结构性和对演示文稿元素的控制。

## 安装说明

1. 使用以下方法之一安装 Aspose.Slides for .NET：

   - NuGet 包管理器：
     ```
     Install-Package Aspose.Slides.NET
     ```

   - 程序包管理器控制台：
     ```
     PM> Install-Package Aspose.Slides.NET
     ```

   - .NET CLI：
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. 添加对您的项目的引用：
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

3. 设置许可证（生产用途可选）：
   ```csharp
   // 设置许可证以删除评估限制
   License license = new License();
   license.SetLicense("Aspose.Slides.lic");
   ```

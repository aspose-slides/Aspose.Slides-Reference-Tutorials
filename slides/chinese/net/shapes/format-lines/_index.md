# 在 PowerPoint 中创建具有自定义线条格式的矩形

Aspose.Slides for .NET 提供了强大的功能，可用于以编程方式创建和修改 PowerPoint 演示文稿。此示例演示了如何创建具有自定义线条格式的矩形，包括设置线条样式、宽度、虚线图案和颜色。示例代码会创建一个新的演示文稿，在第一张幻灯片中添加一个矩形，并对形状的填充和轮廓应用特定的格式，最终生成一个具有白色填充和蓝色虚线轮廓的美观元素。

## 安装说明

1. 使用以下方法之一安装 Aspose.Slides for .NET：

   - NuGet 包管理器：
     ```
     Install-Package Aspose.Slides
     ```

   - 程序包管理器控制台：
     ```
     PM> Install-Package Aspose.Slides
     ```

   - .NET CLI：
     ```
     dotnet add package Aspose.Slides
     ```

2. 将必要的命名空间添加到您的 C# 文件：
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

## 安装说明

1. 使用以下方法之一安装 Aspose.Slides for .NET：
   - NuGet 包管理器： `Install-Package Aspose.Slides.NET`
   - .NET CLI： `dotnet add package Aspose.Slides.NET`
   - 或者直接从下载 [Aspose 网站](https://releases.aspose.com/slides/net/)

2. 将命名空间添加到您的 C# 文件：
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

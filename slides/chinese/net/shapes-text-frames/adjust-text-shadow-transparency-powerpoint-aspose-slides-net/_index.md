在开始编码之前，请确保以下事项：

### 所需库
- **Aspose.Slides for .NET**：此库允许您以编程方式操作 PowerPoint 演示文稿。请确保它已安装在您的项目中。

### 环境设置要求
- 支持 C# 的开发环境，例如 Visual Studio。
- .NET Core 或 .NET Framework，取决于您的项目需求。
  

### 知识前提

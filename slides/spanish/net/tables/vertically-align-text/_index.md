# Creación de tablas con texto vertical en presentaciones de PowerPoint

Aspose.Slides para .NET ofrece potentes funciones para crear y manipular presentaciones de PowerPoint mediante programación. Este ejemplo muestra cómo crear una tabla con formato personalizado, incluyendo alineación vertical y rotación de texto, un requisito común para presentaciones empresariales especializadas y visualización de datos.

## Instrucciones de instalación

1. Instalar Aspose.Slides para .NET a través del Administrador de paquetes NuGet:
   ```
   Install-Package Aspose.Slides
   ```
   O a través de .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

2. Añade referencias a tu proyecto:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

3. Descargar recursos de documentación:
   - [Referencia de API](https://reference.aspose.com/slides/net/)
   - [Guía para desarrolladores](https://docs.aspose.com/slides/net/)

# Cómo crear tablas sin bordes en presentaciones de PowerPoint

Aspose.Slides para .NET es una potente biblioteca que permite a los desarrolladores crear, manipular y convertir presentaciones de PowerPoint mediante programación. Este ejemplo muestra cómo crear una tabla en una presentación y personalizar su apariencia eliminando todos los bordes de las celdas.

El código crea una nueva presentación con una tabla en la primera diapositiva, luego itera a través de cada celda para configurar todos los bordes (superior, inferior, izquierdo y derecho) para que no tengan relleno, creando efectivamente una tabla sin bordes.

## Instrucciones de instalación

1. Instalar a través del Administrador de paquetes NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. O instalar a través de .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Añade un espacio de nombres a tu código:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

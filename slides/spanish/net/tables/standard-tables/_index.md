# Cree tablas estandarizadas en PowerPoint con Aspose.Slides

Aspose.Slides para .NET es una potente API que permite a los desarrolladores crear, leer, escribir, modificar y convertir presentaciones de PowerPoint mediante programación sin necesidad de Microsoft PowerPoint. Este ejemplo muestra cómo crear una tabla estandarizada con bordes con formato personalizado en una presentación de PowerPoint.

## Instrucciones de instalación

1. Instalar a través del Administrador de paquetes NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. O a través de .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Agregue el siguiente espacio de nombres a su archivo C#:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Tables;
   using System.Drawing;
   ```

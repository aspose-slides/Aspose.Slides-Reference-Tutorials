# Creación de formas elípticas en PowerPoint con Aspose.Slides

Aspose.Slides para .NET es una potente API que permite a los desarrolladores crear, leer, escribir, modificar y convertir presentaciones de PowerPoint sin necesidad de Microsoft PowerPoint. Este ejemplo muestra cómo crear una forma elíptica en una presentación, aplicar formato, incluyendo color de relleno y propiedades de línea, y guardar el resultado como un archivo PPTX. El código muestra la capacidad de Aspose.Slides para manipular formas con un control preciso sobre su apariencia y posicionamiento.

## Instrucciones de instalación

1. Instalar a través del Administrador de paquetes NuGet:
   ```
   Install-Package Aspose.Slides
   ```

   utilizando la CLI .NET:
   ```
   dotnet add package Aspose.Slides
   ```

2. Añadir referencias:
   Agregue el siguiente espacio de nombres a su archivo C#:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

3. Configuración de la licencia (opcional):
   Si tienes licencia, aplícala mediante:
   ```csharp
   License license = new License();
   license.SetLicense("Aspose.Slides.lic");
   ```

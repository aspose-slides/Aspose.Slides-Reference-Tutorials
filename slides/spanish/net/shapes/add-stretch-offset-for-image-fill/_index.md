# Cómo agregar imágenes a diapositivas de PowerPoint con Aspose.Slides

Aspose.Slides para .NET ofrece potentes funciones para crear y modificar presentaciones de PowerPoint mediante programación. Este ejemplo muestra cómo cargar una imagen externa y añadirla a una diapositiva de presentación como marco, un requisito común al generar informes, crear plantillas o crear presentaciones dinámicamente a partir de fuentes de datos.

El código a continuación maneja todo, desde la creación de una nueva presentación hasta la adición de la imagen y el guardado del archivo PPTX final, todo con solo unas pocas líneas de código C#.

## Instrucciones de instalación

1. Instale Aspose.Slides para .NET utilizando uno de estos métodos:
   - Administrador de paquetes NuGet: `Install-Package Aspose.Slides.NET`
   - CLI de .NET: `dotnet add package Aspose.Slides.NET`
   - O descargar directamente desde [Sitio web de Aspose](https://releases.aspose.com/slides/net/)

2. Agregue el espacio de nombres a su archivo C#:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

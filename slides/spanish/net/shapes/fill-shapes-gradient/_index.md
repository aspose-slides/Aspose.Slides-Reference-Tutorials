# Cómo crear elipses con relleno de degradado en PowerPoint con Aspose.Slides

Este ejemplo muestra cómo crear programáticamente una presentación de PowerPoint con una elipse y un efecto de relleno degradado personalizado. El código crea una nueva presentación, añade una elipse a la primera diapositiva y aplica un relleno degradado lineal con trazos de color rojo y morado. Aspose.Slides para .NET ofrece una API completa para crear y manipular presentaciones de PowerPoint con un control preciso sobre formas, formato y efectos.

## Instrucciones de instalación

1. Instale Aspose.Slides para .NET utilizando uno de estos métodos:
   - Administrador de paquetes NuGet: `Install-Package Aspose.Slides.NET`
   - CLI de .NET: `dotnet add package Aspose.Slides.NET`
   - Consola del administrador de paquetes: `Install-Package Aspose.Slides.NET`

2. Agregue la siguiente declaración using a su archivo C#:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

# Creación de efectos de texto animados en PowerPoint con Aspose.Slides

Aspose.Slides para .NET ofrece potentes funciones para crear efectos de texto animados en presentaciones de PowerPoint mediante programación. Este ejemplo muestra cómo crear un cuadro de texto con efectos de animación, incluyendo una trayectoria de movimiento personalizada que se activa al hacer clic en un botón. El código crea una presentación con forma de rectángulo que contiene texto, aplica una animación PathFootball y añade una trayectoria de movimiento personalizada, definida por el usuario, que se puede activar interactivamente.

## Instrucciones de instalación

1. Instale Aspose.Slides para .NET utilizando uno de los siguientes métodos:

   Administrador de paquetes NuGet:
   ```csharp
   Install-Package Aspose.Slides
   ```

   Consola del administrador de paquetes:
   ```csharp
   PM> Install-Package Aspose.Slides
   ```

   CLI de .NET:
   ```csharp
   dotnet add package Aspose.Slides
   ```

2. Agregue los espacios de nombres necesarios a su código C#:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Animation;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

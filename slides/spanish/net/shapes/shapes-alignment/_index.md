# Alinear y agrupar formas en presentaciones de PowerPoint

Aspose.Slides para .NET ofrece potentes funciones para crear y manipular presentaciones de PowerPoint mediante programación. Este ejemplo muestra cómo alinear formas dentro de diapositivas y trabajar con formas agrupadas en presentaciones de PowerPoint. Aprenderá a colocar rectángulos en coordenadas específicas, alinearlos con respecto a la diapositiva o entre sí, y alinear formas selectivamente dentro de grupos.

## Instrucciones de instalación

1. Instale Aspose.Slides para .NET utilizando uno de estos métodos:
   
   - Administrador de paquetes NuGet:
     ```
     Install-Package Aspose.Slides.NET
     ```
   
   - Consola del administrador de paquetes:
     ```
     PM> Install-Package Aspose.Slides.NET
     ```
   
   - CLI de .NET:
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. Añade referencias a tu proyecto:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.IO;
   ```

3. Descargue el código de ejemplo a continuación e intégrelo en su proyecto.

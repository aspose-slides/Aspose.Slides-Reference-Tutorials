# Cómo agregar segmentos a GeometryShape en presentaciones de PowerPoint

Este ejemplo muestra cómo acceder y modificar la ruta geométrica de formas en presentaciones de PowerPoint con Aspose.Slides para .NET. Al agregar segmentos de línea personalizados a un rectángulo, puede crear figuras geométricas más complejas que no están disponibles con los tipos de forma estándar. Esta técnica es útil para crear diagramas, diagramas de flujo y otros elementos visuales personalizados en sus presentaciones.

## Instrucciones de instalación

1. Instale Aspose.Slides para .NET utilizando uno de los siguientes métodos:

   - Administrador de paquetes NuGet:
     ```
     Install-Package Aspose.Slides.NET
     ```

   - Consola del administrador de paquetes:
     ```
     PM> Install-Package Aspose.Slides.NET
     ```

   - CLI de .NET:
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. Agregue el espacio de nombres a su archivo C#:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

# Cómo crear fondos de imagen en mosaico en PowerPoint con Aspose.Slides

Aspose.Slides para .NET ofrece potentes funciones para crear y manipular presentaciones de PowerPoint mediante programación. Este ejemplo muestra cómo crear una presentación con un rectángulo relleno con un patrón de imagen en mosaico. El código muestra cómo cargar una imagen, añadirla a una diapositiva como fondo del rectángulo y aplicar diversas propiedades de mosaico, como desplazamiento, escala, alineación y opciones de volteo. Esta técnica es útil para crear fondos personalizados, marcas de agua o elementos decorativos en las presentaciones con un control preciso del comportamiento del mosaico.

## Instrucciones de instalación

1. Instale Aspose.Slides para .NET mediante el Administrador de paquetes NuGet:
   ```
   Install-Package Aspose.Slides.NET
   ```

2. O instalar a través de .NET CLI:
   ```
   dotnet add package Aspose.Slides.NET
   ```

3. Agregue las siguientes directivas using a su archivo C#:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.IO;
   ```

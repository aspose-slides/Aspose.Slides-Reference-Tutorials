# Crear formas de grupo en PowerPoint con Aspose.Slides

Aspose.Slides para .NET ofrece API completas para crear y manipular presentaciones de PowerPoint mediante programación. Este ejemplo muestra cómo crear una forma de grupo, añadir varias formas rectangulares, colocarlas correctamente, enmarcarlas y guardar la presentación. Las formas de grupo son esenciales para organizar formas relacionadas que deben manipularse como una sola unidad, lo que proporciona mayor estructura y control sobre los elementos de la presentación.

## Instrucciones de instalación

1. Instale Aspose.Slides para .NET utilizando uno de estos métodos:

   - Administrador de paquetes NuGet:
     ```
     Install-Package Aspose.Slides.NET
     ```

   - Consola del administrador de paquetes:
     ```
     PM> Install-Package Aspose.Slides.NET
     ```

   - CLI de .NET:
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. Añade referencias a tu proyecto:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

3. Configurar una licencia (opcional para uso en producción):
   ```csharp
   // Establecer licencia para eliminar restricciones de evaluación
   License license = new License();
   license.SetLicense("Aspose.Slides.lic");
   ```

# Cómo crear y rotar formas rectangulares en PowerPoint

Aspose.Slides para .NET es una potente API que permite a los desarrolladores crear, modificar y convertir presentaciones de PowerPoint sin necesidad de instalar Microsoft PowerPoint. Este ejemplo muestra cómo crear una presentación con un rectángulo girado, mostrando las capacidades de manipulación de formas de la biblioteca.

Con Aspose.Slides, puedes añadir diversas formas a tus diapositivas mediante programación y aplicar transformaciones como la rotación. Esta función es especialmente útil para crear presentaciones dinámicas con un control preciso de los elementos visuales.

## Instrucciones de instalación

1. Instalar a través del Administrador de paquetes NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Instalar mediante la CLI de .NET:
   ```
   dotnet add package Aspose.Slides
   ```

3. Agregue la directiva using en su código:
   ```csharp
   using Aspose.Slides;
   ```

# Cómo crear un rectángulo con formato de línea personalizado en PowerPoint

Aspose.Slides para .NET ofrece potentes funciones para crear y modificar presentaciones de PowerPoint mediante programación. Este ejemplo muestra cómo crear un rectángulo con formato de línea personalizado, incluyendo la configuración del estilo, el ancho, el patrón de trazos y el color de línea. El código de ejemplo crea una nueva presentación, añade un rectángulo a la primera diapositiva y aplica un formato específico tanto al relleno como al contorno de la forma, lo que da como resultado un elemento visualmente atractivo con relleno blanco y contorno discontinuo azul.

## Instrucciones de instalación

1. Instale Aspose.Slides para .NET utilizando uno de los siguientes métodos:

   - Administrador de paquetes NuGet:
     ```
     Install-Package Aspose.Slides
     ```

   - Consola del administrador de paquetes:
     ```
     PM> Install-Package Aspose.Slides
     ```

   - CLI de .NET:
     ```
     dotnet add package Aspose.Slides
     ```

2. Agregue el espacio de nombres necesario a su archivo C#:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

# Creación de formas con relleno de imágenes en mosaico en PowerPoint con Aspose.Slides

Aspose.Slides para .NET ofrece potentes funciones para crear y manipular presentaciones de PowerPoint mediante programación. Este ejemplo muestra cómo crear un rectángulo y rellenarlo con un patrón de imagen en mosaico. El relleno de imagen en mosaico es especialmente útil para crear fondos con textura, marcas de agua o elementos decorativos en las presentaciones.

## Instrucciones de instalación

1. Instalar a través del Administrador de paquetes NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Instalar mediante la CLI de .NET:
   ```
   dotnet add package Aspose.Slides
   ```

3. Instalación manual:
   - Descargue la última versión desde [Aspose.Slides para .NET](https://releases.aspose.com/slides/net/)
   - Agregue una referencia a Aspose.Slides.dll en su proyecto

## Instalación

```csharp
// Instalar a través del Administrador de paquetes NuGet
Install-Package Aspose.Slides
```

O agregue la siguiente referencia a su archivo de proyecto:

```xml
<PackageReference Include="Aspose.Slides" Version="23.2.0" />
```

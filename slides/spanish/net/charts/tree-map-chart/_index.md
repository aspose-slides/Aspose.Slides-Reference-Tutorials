## Instrucciones de instalación:  
Para comenzar a utilizar Aspose.Slides para .NET, siga estos pasos:  

1. Descargue Aspose.Slides para .NET desde el sitio oficial [Página de lanzamiento de Aspose](https://releases.aspose.com/slides/net/).  
2. Instale el paquete mediante el Administrador de paquetes NuGet en Visual Studio:  
   ```shell
   Install-Package Aspose.Slides
   ```  
3. Agregue los espacios de nombres requeridos en su código:  
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Charts;
   ```

# Modificar gráficos en PowerPoint con Aspose.Slides

Aspose.Slides para .NET ofrece potentes API para crear y modificar presentaciones de PowerPoint mediante programación. Este ejemplo muestra cómo trabajar con gráficos en presentaciones existentes: actualizando los datos, añadiendo nuevas series y cambiando los tipos de gráfico, todo ello sin necesidad de instalar Microsoft PowerPoint.

## Instrucciones de instalación

1. Instalar a través del Administrador de paquetes NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. O usando .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Agregue las siguientes directivas using:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Charts;
   ```

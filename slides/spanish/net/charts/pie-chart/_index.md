## Instrucciones de instalación
Para comenzar a utilizar Aspose.Slides para .NET, siga estos pasos:

1. Descargar Aspose.Slides para .NET  
   - Descargue la última versión desde [Página de descarga de Aspose.Slides para .NET](https://releases.aspose.com/slides/net/).

2. Instalar a través del Administrador de paquetes NuGet  
   Ejecute el siguiente comando en la consola del Administrador de paquetes NuGet:
   ```shell
   Install-Package Aspose.Slides
   ```

3. Agregar referencias requeridas  
   En su proyecto de Visual Studio, agregue una referencia a `Aspose.Slides.dll`.

4. Importar espacios de nombres  
   Agregue los espacios de nombres necesarios al comienzo de su código:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Charts;
   using System.Drawing;
   ```

5. Solicitar una licencia (opcional pero recomendado)  
   Para evitar limitaciones de evaluación, puede: [solicitar una licencia](https://purchase.aspose.com/temporary-license/).

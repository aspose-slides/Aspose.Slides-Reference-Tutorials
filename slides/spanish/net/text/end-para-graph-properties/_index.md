# Creación de presentaciones de PowerPoint con texto formateado

Aspose.Slides para .NET es una potente API que permite a los desarrolladores crear, manipular y convertir presentaciones de PowerPoint sin necesidad de Microsoft PowerPoint. Este ejemplo muestra cómo crear una presentación sencilla con una forma rectangular y párrafos de texto formateados.

El código crea una nueva presentación, añade una forma rectangular a la primera diapositiva y luego añade varios párrafos con diferentes opciones de formato de texto. Muestra cómo configurar propiedades de fuente, como el tamaño y la familia, para secciones específicas del texto.

## Instrucciones de instalación

1. Instalar a través del Administrador de paquetes NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Instalar mediante la CLI de .NET:
   ```
   dotnet add package Aspose.Slides
   ```

3. Instalar a través de la consola del administrador de paquetes:
   ```
   PM> Install-Package Aspose.Slides
   ```

4. Añadir referencia a su proyecto:
   Después de la instalación, agregue una referencia a su proyecto:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

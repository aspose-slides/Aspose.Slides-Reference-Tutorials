# Cómo crear viñetas de varios niveles en PowerPoint con Aspose.Slides

Aspose.Slides para .NET ofrece una potente API para crear y manipular presentaciones de PowerPoint mediante programación. Este ejemplo muestra cómo crear una presentación con viñetas de varios niveles, cómo añadir una forma automática con un marco de texto y cómo dar formato a párrafos con diferentes estilos de viñeta y niveles de sangría. Puede crear fácilmente contenido jerárquico con símbolos de viñeta personalizados para cada nivel de sangría, lo que resulta perfecto para crear presentaciones estructuradas y profesionales sin necesidad de instalar Microsoft PowerPoint.

## Instrucciones de instalación

1. Instalar a través del Administrador de paquetes NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Instalar mediante la CLI de .NET:
   ```
   dotnet add package Aspose.Slides
   ```

3. Instalar a través de la consola del administrador de paquetes:
   ```
   PM> Install-Package Aspose.Slides
   ```

4. Instalación manual:
   - Descargue la última versión desde [Aspose.Slides para .NET](https://releases.aspose.com/slides/net/)
   - Agregue una referencia a Aspose.Slides.dll en su proyecto

# Creación de texto vertical en PowerPoint con Aspose.Slides

Aspose.Slides para .NET ofrece potentes funciones para crear y manipular presentaciones de PowerPoint mediante programación. Este ejemplo muestra cómo crear una presentación con un rectángulo y texto girado verticalmente. El texto está orientado a 270 grados, lo que permite leerlo de abajo a arriba. Esta técnica es útil para crear etiquetas laterales, diseños de texto únicos o elementos de diseño especializados en las presentaciones.

## Instrucciones de instalación

1. Instalar Aspose.Slides para .NET
   
   Instale el paquete Aspose.Slides usando el Administrador de paquetes NuGet:
   
   ```csharp
   Install-Package Aspose.Slides
   ```
   
   utilizando la CLI .NET:
   
   ```csharp
   dotnet add package Aspose.Slides
   ```

2. Agregar referencias
   
   Agregue la siguiente declaración using a su archivo C#:
   
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

# Creación de cuadros de texto formateados en PowerPoint con Aspose.Slides

Aspose.Slides para .NET es una potente API que permite a los desarrolladores crear, modificar y convertir presentaciones de Microsoft PowerPoint sin necesidad de instalar Microsoft Office. Este ejemplo muestra cómo crear una presentación con un cuadro de texto formateado, aplicando estilos, tamaños y colores de fuente personalizados para mejorar su atractivo visual.

## Instrucciones de instalación

1. Instalar a través del Administrador de paquetes NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Instalar mediante la CLI de .NET:
   ```
   dotnet add package Aspose.Slides
   ```

3. Añade una referencia a tu proyecto manualmente:
   - Descargue la última versión desde [Aspose.Slides para .NET](https://releases.aspose.com/slides/net/)
   - Añade la DLL como referencia en tu proyecto

4. Inicializar una licencia:
   ```csharp
   // Inicializar una licencia
   License license = new License();
   license.SetLicense("Aspose.Slides.lic");
   ```

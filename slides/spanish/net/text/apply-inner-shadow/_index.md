# Cree presentaciones de PowerPoint con TextBox en Aspose.Slides para .NET

Aspose.Slides para .NET es una potente API que permite a los desarrolladores crear, modificar y convertir presentaciones de PowerPoint sin necesidad de Microsoft PowerPoint. Este ejemplo muestra cómo crear una presentación con un cuadro de texto personalizado, aplicar formato y guardarla como archivo PPTX.

## Instrucciones de instalación

1. Instalar a través del Administrador de paquetes NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Instalar mediante la CLI de .NET:
   ```
   dotnet add package Aspose.Slides
   ```

3. Instalar manualmente:
   - Descargue la última versión desde [Página de lanzamientos de Aspose.Slides para .NET](https://releases.aspose.com/slides/net/)
   - Agregue la referencia DLL a su proyecto

## Recursos adicionales

- [Documentación de Aspose.Slides para .NET](https://docs.aspose.com/slides/net/)
- [Referencia de la API de Aspose.Slides para .NET](https://reference.aspose.com/slides/net/)

# Cómo rotar títulos y etiquetas de gráficos en PowerPoint con Aspose.Slides

Aspose.Slides para .NET ofrece potentes funciones para crear y manipular presentaciones de PowerPoint mediante programación. Este ejemplo muestra cómo crear una presentación con un gráfico de columnas agrupadas que incluye elementos de texto rotados; concretamente, rotando tanto el título del gráfico como las etiquetas de datos para mejorar la legibilidad y el atractivo visual.

## Instrucciones de instalación

1. Instalar a través del Administrador de paquetes NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. O instalar a través de .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Agregue el espacio de nombres a su archivo C#:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Charts;
   ```

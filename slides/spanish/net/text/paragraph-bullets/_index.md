# Cómo crear viñetas personalizadas en presentaciones de PowerPoint

Aspose.Slides para .NET ofrece potentes funciones para crear y manipular presentaciones de PowerPoint mediante programación. Este ejemplo se centra específicamente en la implementación de viñetas personalizadas en marcos de texto, mostrando cómo controlar el tipo, el color, la sangría y el tamaño de las viñetas. El código crea una presentación rectangular con dos párrafos: uno con viñetas de símbolos y otro con viñetas numeradas.

## Instrucciones de instalación

1. Instale Aspose.Slides para .NET mediante el Administrador de paquetes NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Alternativamente, descargue el paquete directamente:
   - Visita el [Página de descarga de Aspose.Slides](https://downloads.aspose.com/slides/net)
   - Descargue la última versión
   - Extraiga y haga referencia a Aspose.Slides.dll en su proyecto

3. Agregue la declaración using necesaria:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

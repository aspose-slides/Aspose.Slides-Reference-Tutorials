# 使用 Aspose.Slides 在 PowerPoint 中建立橢圓形

Aspose.Slides for .NET 是一個強大的 API，它使開發人員無需 Microsoft PowerPoint 即可建立、讀取、編寫、修改和轉換 PowerPoint 簡報。此範例示範如何在簡報中建立橢圓形，套用包括填滿色彩和線條屬性在內的格式，以及如何將結果儲存為 PPTX 檔案。該程式碼展示了 Aspose.Slides 透過精確控制形狀的外觀和位置來操縱形狀的能力。

## 安裝說明

1. 透過 NuGet 套件管理器安裝：
   ```
   Install-Package Aspose.Slides
   ```

   或使用 .NET CLI：
   ```
   dotnet add package Aspose.Slides
   ```

2. 新增參考文獻：
   將以下命名空間新增至您的 C# 檔案：
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

3. 許可證設定（可選）：
   如果您有許可證，請使用以下命令申請：
   ```csharp
   License license = new License();
   license.SetLicense("Aspose.Slides.lic");
   ```

# 使用 Aspose.Slides 修改 PowerPoint 形狀中的幾何路徑

Aspose.Slides for .NET 提供了強大的功能，可以透過程式設計修改 PowerPoint 簡報，包括進階形狀操作。此範例示範如何透過修改現有形狀的幾何路徑來建立自訂形狀 - 具體來說，透過從心形中刪除一個部分來建立獨特的變體。

幾何路徑透過一系列點和線段定義形狀的輪廓。透過存取和修改這些路徑，開發人員可以建立標準 PowerPoint 形狀庫中沒有的高度客製化的形狀。

## 安裝說明

1. 透過 NuGet 套件管理器安裝：
   ```
   Install-Package Aspose.Slides
   ```

2. 透過 .NET CLI 安裝：
   ```
   dotnet add package Aspose.Slides
   ```

3. 在您的程式碼中加入 using 指令：
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

4. 在應用程式啟動時新增許可證初始化：
   ```csharp
   // 如果你有駕照
   License license = new License();
   license.SetLicense("path/to/Aspose.Slides.lic");
   ```

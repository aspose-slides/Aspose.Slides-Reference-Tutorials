# 在 PowerPoint 中建立和旋轉矩形

Aspose.Slides for .NET 是一個強大的 API，可讓開發人員建立、修改和轉換 PowerPoint 簡報，而無需安裝 Microsoft PowerPoint。此範例示範如何建立具有旋轉矩形形狀的演示文稿，展示了該庫的形狀操作功能。

使用 Aspose.Slides，您可以以程式設計方式為投影片添加各種形狀並套用旋轉等變換。此功能對於建立可精確控制視覺元素的動態簡報特別有用。

## 安裝說明

1. 透過 NuGet 套件管理器安裝：
   ```
   Install-Package Aspose.Slides
   ```

2. 透過 .NET CLI 安裝：
   ```
   dotnet add package Aspose.Slides
   ```

3. 在您的程式碼中加入 using 指令：
   ```csharp
   using Aspose.Slides;
   ```

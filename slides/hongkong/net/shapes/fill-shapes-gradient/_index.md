# 使用 Aspose.Slides 在 PowerPoint 中建立漸層填滿橢圓形

此範例示範如何以程式設計方式建立包含具有自訂漸層填滿效果的橢圓形的 PowerPoint 簡報。程式碼會建立一個新的簡報，在第一張投影片中新增一個橢圓，並套用帶有紅色和紫色色標的線性漸層填滿。 Aspose.Slides for .NET 提供了全面的 API，可建立和操作 PowerPoint 簡報，並可精確控制形狀、格式和效果。

## 安裝說明

1. 使用下列方法之一安裝 Aspose.Slides for .NET：
   - NuGet 套件管理器： `Install-Package Aspose.Slides.NET`
   - .NET CLI： `dotnet add package Aspose.Slides.NET`
   - 套件管理器控制台： `Install-Package Aspose.Slides.NET`

2. 將以下 using 語句加入您的 C# 檔案：
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

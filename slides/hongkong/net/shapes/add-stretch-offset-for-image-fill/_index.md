# 使用 Aspose.Slides 將圖片新增至 PowerPoint 投影片

Aspose.Slides for .NET 提供了以程式設計方式建立和修改 PowerPoint 簡報的強大功能。此範例示範如何載入外部影像並將其作為相框新增至簡報投影片 - 這是產生報表、建立範本或從資料來源動態建立簡報時的常見要求。

下面的程式碼處理從建立新簡報到新增圖像和保存最終 PPTX 檔案的所有操作，只需幾行 C# 程式碼即可。

## 安裝說明

1. 使用下列方法之一安裝 Aspose.Slides for .NET：
   - NuGet 套件管理器： `Install-Package Aspose.Slides.NET`
   - .NET CLI： `dotnet add package Aspose.Slides.NET`
   - 或直接從下載 [Aspose 網站](https://releases.aspose.com/slides/net/)

2. 將命名空間新增到您的 C# 檔案：
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

# 在 PowerPoint 簡報中向 GeometryShape 新增線段

此範例示範如何使用 Aspose.Slides for .NET 存取和修改 PowerPoint 簡報中形狀的幾何路徑。透過為矩形新增自訂線段，您可以建立標準形狀類型無法提供的更複雜的幾何圖形。此技術對於在簡報中建立自訂圖表、流程圖和其他視覺元素很有用。

## 安裝說明

1. 使用下列方法之一安裝 Aspose.Slides for .NET：

   - NuGet 套件管理器：
     ```
     Install-Package Aspose.Slides.NET
     ```

   - 套件管理器控制台：
     ```
     PM> Install-Package Aspose.Slides.NET
     ```

   - .NET CLI：
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. 將命名空間新增到您的 C# 檔案：
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

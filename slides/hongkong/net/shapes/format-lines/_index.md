# 在 PowerPoint 中建立具有自訂線條格式的矩形

Aspose.Slides for .NET 提供了以程式設計方式建立和修改 PowerPoint 簡報的強大功能。此範例示範如何建立具有自訂線條格式的矩形，包括設定線條樣式、寬度、虛線圖案和顏色。範例程式碼創建了一個新的演示文稿，在第一張幻燈片中添加了一個矩形，並對形狀的填充和輪廓應用了特定的格式，從而產生了具有白色填充和藍色虛線輪廓的視覺吸引力元素。

## 安裝說明

1. 使用下列方法之一安裝 Aspose.Slides for .NET：

   - NuGet 套件管理器：
     ```
     Install-Package Aspose.Slides
     ```

   - 套件管理器控制台：
     ```
     PM> Install-Package Aspose.Slides
     ```

   - .NET CLI：
     ```
     dotnet add package Aspose.Slides
     ```

2. 將必要的命名空間新增到您的 C# 檔案：
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

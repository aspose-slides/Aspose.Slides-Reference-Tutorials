# 使用 Aspose.Slides 在 PowerPoint 中建立群組形狀

Aspose.Slides for .NET 提供了一個全面的 API，以程式設計方式建立和操作 PowerPoint 簡報。此範例示範如何建立群組形狀、在其中新增多個矩形形狀、正確定位形狀、新增框架以及儲存簡報。群組形狀對於組織需要作為單一單元進行操作的相關形狀至關重要，從而為演示元素提供更多結構和控制。

## 安裝說明

1. 使用下列方法之一安裝 Aspose.Slides for .NET：

   - NuGet 套件管理器：
     ```
     Install-Package Aspose.Slides.NET
     ```

   - 套件管理器控制台：
     ```
     PM> Install-Package Aspose.Slides.NET
     ```

   - .NET CLI：
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. 新增對您的項目的引用：
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

3. 設定許可證（生產用途可選）：
   ```csharp
   // 設定許可證以刪除評估限制
   License license = new License();
   license.SetLicense("Aspose.Slides.lic");
   ```

# 使用 Aspose.Slides 在 PowerPoint 中建立平鋪影像背景

Aspose.Slides for .NET 提供了以程式設計方式建立和操作 PowerPoint 簡報的強大功能。此範例示範如何建立一個以平鋪影像圖案填滿的矩形簡報。程式碼展示如何載入影像、將其新增至投影片作為矩形的背景，以及套用各種平鋪屬性，如偏移、縮放、對齊和翻轉選項。此技術可用於在簡報中建立自訂背景、浮水印或裝飾元素，並可精確控制平鋪行為。

## 安裝說明

1. 使用 NuGet 套件管理器安裝 Aspose.Slides for .NET：
   ```
   Install-Package Aspose.Slides.NET
   ```

2. 或透過 .NET CLI 安裝：
   ```
   dotnet add package Aspose.Slides.NET
   ```

3. 將以下使用指令新增至您的 C# 檔案：
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.IO;
   ```

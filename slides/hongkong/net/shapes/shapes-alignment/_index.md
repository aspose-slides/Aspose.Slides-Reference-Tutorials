# 在 PowerPoint 簡報中對齊和分組形狀

Aspose.Slides for .NET 提供了以程式設計方式建立和操作 PowerPoint 簡報的強大功能。此範例示範如何在投影片中對齊形狀以及如何在 PowerPoint 簡報中使用分組形狀。您將學習如何將矩形定位在特定座標處，如何將它們相對於投影片或彼此對齊，以及如何選擇性地對齊群組內的形狀。

## 安裝說明

1. 使用下列方法之一安裝 Aspose.Slides for .NET：
   
   - NuGet 套件管理器：
     ```
     Install-Package Aspose.Slides.NET
     ```
   
   - 套件管理器控制台：
     ```
     PM> Install-Package Aspose.Slides.NET
     ```
   
   - .NET CLI：
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. 為您的項目新增引用：
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.IO;
   ```

3. 下載下面的範例程式碼並將其整合到您的專案中。

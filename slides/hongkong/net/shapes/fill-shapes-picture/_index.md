# 使用 Aspose.Slides 在 PowerPoint 中建立具有平鋪影像填滿的形狀

Aspose.Slides for .NET 提供了以程式設計方式建立和操作 PowerPoint 簡報的強大功能。此範例示範如何建立矩形並用平鋪影像圖案填滿它。平鋪圖像填充對於在簡報中創建紋理背景、浮水印或裝飾元素特別有用。

## 安裝說明

1. 透過 NuGet 套件管理器安裝：
   ```
   Install-Package Aspose.Slides
   ```

2. 透過 .NET CLI 安裝：
   ```
   dotnet add package Aspose.Slides
   ```

3. 手動安裝：
   - 從下載最新版本 [Aspose.Slides for .NET](https://releases.aspose.com/slides/net/)
   - 在您的專案中新增對 Aspose.Slides.dll 的引用

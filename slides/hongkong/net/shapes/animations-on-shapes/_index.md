# 使用 Aspose.Slides 在 PowerPoint 中建立動畫文字效果

Aspose.Slides for .NET 提供了強大的功能，可以透過程式設計在 PowerPoint 簡報中建立動畫文字效果。此範例示範如何建立具有動畫效果的文字框，包括透過按鈕點擊觸發的自訂運動路徑。該程式碼創建了一個包含文字的矩形形狀的演示文稿，應用了 PathFootball 動畫，並添加了可以互動觸發的自訂用戶定義運動路徑。

## 安裝說明

1. 使用下列方法之一安裝 Aspose.Slides for .NET：

   NuGet 套件管理器：
   ```csharp
   Install-Package Aspose.Slides
   ```

   套件管理器控制台：
   ```csharp
   PM> Install-Package Aspose.Slides
   ```

   .NET CLI：
   ```csharp
   dotnet add package Aspose.Slides
   ```

2. 將所需的命名空間加入您的 C# 程式碼：
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Animation;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

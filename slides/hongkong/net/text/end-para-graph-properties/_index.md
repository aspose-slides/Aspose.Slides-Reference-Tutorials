# 使用格式化文字建立 PowerPoint 簡報

Aspose.Slides for .NET 是一個強大的 API，可讓開發人員建立、操作和轉換 PowerPoint 簡報，而無需 Microsoft PowerPoint。此範例示範如何建立一個包含具有格式化文字段落的矩形的簡單簡報。

該程式碼會建立一個新的演示文稿，在第一張幻燈片中添加一個矩形，然後添加具有不同文字格式選項的多個段落。它展示瞭如何為文字的特定部分設定字體屬性，例如大小和字體類型。

## 安裝說明

1. 透過 NuGet 套件管理器安裝：
   ```
   Install-Package Aspose.Slides
   ```

2. 透過 .NET CLI 安裝：
   ```
   dotnet add package Aspose.Slides
   ```

3. 透過套件管理器控制台安裝：
   ```
   PM> Install-Package Aspose.Slides
   ```

4. 新增對項目的引用：
   安裝後，新增對項目的引用：
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

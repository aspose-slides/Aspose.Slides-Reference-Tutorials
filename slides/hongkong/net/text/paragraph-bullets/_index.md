# 在 PowerPoint 簡報中建立自訂項目符號

Aspose.Slides for .NET 提供了以程式設計方式建立和操作 PowerPoint 簡報的強大功能。此範例特別著重於在文字方塊中實作自訂項目符號，展示如何控制項目符號的類型、顏色、縮排和大小。該程式碼會建立一個矩形形狀的演示文稿，其中包含兩個段落 - 一個帶有符號項目符號，另一個帶有編號項目符號樣式。

## 安裝說明

1. 使用 NuGet 套件管理器安裝 Aspose.Slides for .NET：
   ```
   Install-Package Aspose.Slides
   ```

2. 或者，直接下載軟體包：
   - 訪問 [Aspose.Slides下載頁面](https://downloads.aspose.com/slides/net)
   - 下載最新版本
   - 提取並引用項目中的 Aspose.Slides.dll

3. 加入必要的using語句：
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

# 使用 Aspose.Slides 在 PowerPoint 中旋轉圖表標題和標籤

Aspose.Slides for .NET 提供了以程式設計方式建立和操作 PowerPoint 簡報的強大功能。此範例示範如何建立具有旋轉文字元素的簇狀長條圖的簡報 - 具體來說，旋轉圖表標題和資料標籤以提高可讀性和視覺吸引力。

## 安裝說明

1. 透過 NuGet 套件管理器安裝：
   ```
   Install-Package Aspose.Slides
   ```

2. 或透過 .NET CLI 安裝：
   ```
   dotnet add package Aspose.Slides
   ```

3. 將命名空間新增到您的 C# 檔案：
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Charts;
   ```

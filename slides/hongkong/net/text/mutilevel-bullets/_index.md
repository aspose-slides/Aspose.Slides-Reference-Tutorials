# 使用 Aspose.Slides 在 PowerPoint 中建立多層項目符號

Aspose.Slides for .NET 提供了強大的 API，可以透過程式設計方式建立和操作 PowerPoint 簡報。此範例示範如何建立具有多層項目符號的簡報，展示如何新增帶有文字方塊的自動形狀以及如何使用不同的項目符號樣式和縮排等級設定段落格式。您可以輕鬆建立具有每個縮排層級的自訂項目符號的分層內容，非常適合建立結構化和專業的簡報，而無需安裝 Microsoft PowerPoint。

## 安裝說明

1. 透過 NuGet 套件管理器安裝：
   ```
   Install-Package Aspose.Slides
   ```

2. 透過 .NET CLI 安裝：
   ```
   dotnet add package Aspose.Slides
   ```

3. 透過套件管理器控制台安裝：
   ```
   PM> Install-Package Aspose.Slides
   ```

4. 手動安裝：
   - 從下載最新版本 [Aspose.Slides for .NET](https://releases.aspose.com/slides/net/)
   - 在您的專案中新增對 Aspose.Slides.dll 的引用

# 使用 Aspose.Slides for .NET 中的 TextBox 建立 PowerPoint 簡報

Aspose.Slides for .NET 是一個強大的 API，可讓開發人員建立、修改和轉換 PowerPoint 簡報，而無需 Microsoft PowerPoint。此範例示範如何建立具有自訂文字方塊的簡報、套用格式並將其儲存為 PPTX 檔案。

## 安裝說明

1. 透過 NuGet 套件管理器安裝：
   ```
   Install-Package Aspose.Slides
   ```

2. 透過 .NET CLI 安裝：
   ```
   dotnet add package Aspose.Slides
   ```

3. 手動安裝：
   - 從下載最新版本 [Aspose.Slides for .NET 發佈頁面](https://releases.aspose.com/slides/net/)
   - 將 DLL 引用新增至您的項目

# 使用 Aspose.Slides 在 PowerPoint 中建立垂直文字

Aspose.Slides for .NET 提供了以程式設計方式建立和操作 PowerPoint 簡報的強大功能。此範例示範如何建立包含具有垂直旋轉文字的矩形的簡報。文本以 270 度角排列，因此可以從下到上閱讀。此技術對於在簡報中建立側標籤、獨特的文字佈局或專門的設計元素很有用。

## 安裝說明

1. 安裝 Aspose.Slides for .NET
   
   使用 NuGet 套件管理器安裝 Aspose.Slides 套件：
   
   ```csharp
   Install-Package Aspose.Slides
   ```
   
   或使用 .NET CLI：
   
   ```csharp
   dotnet add package Aspose.Slides
   ```

2. 新增引用
   
   將以下 using 語句加入您的 C# 檔案：
   
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

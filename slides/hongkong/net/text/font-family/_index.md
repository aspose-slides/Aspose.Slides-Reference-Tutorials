# 使用 Aspose.Slides 在 PowerPoint 中建立格式化文字方塊

Aspose.Slides for .NET 是一個強大的 API，可讓開發人員建立、修改和轉換 Microsoft PowerPoint 簡報，而無需安裝 Microsoft Office。此範例示範如何建立具有格式化文字方塊的簡報，套用自訂字體樣式、大小和顏色以增強視覺吸引力。

## 安裝說明

1. 透過 NuGet 套件管理器安裝：
   ```
   Install-Package Aspose.Slides
   ```

2. 透過 .NET CLI 安裝：
   ```
   dotnet add package Aspose.Slides
   ```

3. 手動新增對項目的引用：
   - 從下載最新版本 [Aspose.Slides for .NET](https://releases.aspose.com/slides/net/)
   - 在您的專案中新增 DLL 作為引用

4. 初始化許可證：
   ```csharp
   // 初始化許可證
   License license = new License();
   license.SetLicense("Aspose.Slides.lic");
   ```

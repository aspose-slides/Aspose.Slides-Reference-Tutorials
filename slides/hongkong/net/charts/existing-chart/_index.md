# 使用 Aspose.Slides 修改 PowerPoint 中的圖表

Aspose.Slides for .NET 提供了強大的 API，可以透過程式設計方式建立和修改 PowerPoint 簡報。此範例示範如何透過更新圖表資料、新增系列和變更圖表類型來處理現有簡報中的圖表 - 所有這些都不需要安裝 Microsoft PowerPoint。

## 安裝說明

1. 透過 NuGet 套件管理器安裝：
   ```
   Install-Package Aspose.Slides
   ```

2. 或使用 .NET CLI：
   ```
   dotnet add package Aspose.Slides
   ```

3. 新增以下使用指令：
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Charts;
   ```

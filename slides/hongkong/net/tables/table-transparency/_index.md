## 安裝說明

1. 安裝 Aspose.Slides for .NET：
   
   使用 NuGet 套件管理器控制台：
   ```
   Install-Package Aspose.Slides.NET
   ```

   使用 .NET CLI：
   ```
   dotnet add package Aspose.Slides.NET
   ```

2. 為您的程式碼新增必要的使用指令：
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Tables;
   using System.IO;
   ```

# 在 PowerPoint 簡報中建立包含垂直文字的表格

Aspose.Slides for .NET 提供了以程式設計方式建立和操作 PowerPoint 簡報的強大功能。此範例示範如何建立具有自訂格式的表格，包括垂直文字對齊和旋轉 - 這是專業商業簡報和資料視覺化的常見要求。

## 安裝說明

1. 透過 NuGet 套件管理器安裝 Aspose.Slides for .NET：
   ```
   Install-Package Aspose.Slides
   ```
   或透過 .NET CLI：
   ```
   dotnet add package Aspose.Slides
   ```

2. 新增對您的項目的引用：
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

3. 下載文檔資源：
   - [API 參考](https://reference.aspose.com/slides/net/)
   - [開發者指南](https://docs.aspose.com/slides/net/)

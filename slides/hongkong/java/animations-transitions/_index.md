加入投影片轉場的 Java 程式碼讓您完整掌控每張投影片的流程與視覺衝擊。透過 Aspose.Slides，您可以從數十種內建轉場效果中挑選，設定時間，甚至根據使用者操作觸發轉場，全部不需手動開啟 PowerPoint。

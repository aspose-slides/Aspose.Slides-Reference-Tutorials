# Tạo biểu đồ cột nhóm với Aspose.Slides cho .NET

Aspose.Slides for .NET cung cấp các khả năng mạnh mẽ để tạo và thao tác các bài thuyết trình PowerPoint theo chương trình. Ví dụ này trình bày cách tạo biểu đồ cột cụm với các danh mục và chuỗi dữ liệu tùy chỉnh.

## Cài đặt

```csharp
// Cài đặt thông qua NuGet Package Manager
Install-Package Aspose.Slides
```

Hoặc thêm tham chiếu sau vào tệp dự án của bạn:

```xml
<PackageReference Include="Aspose.Slides" Version="23.2.0" />
```

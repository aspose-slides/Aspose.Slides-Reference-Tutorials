# Tạo hình nhóm trong PowerPoint với Aspose.Slides

Aspose.Slides for .NET cung cấp các API toàn diện để tạo và thao tác các bài thuyết trình PowerPoint theo chương trình. Ví dụ này minh họa cách tạo một hình nhóm, thêm nhiều hình chữ nhật vào đó, định vị các hình đúng cách, thêm khung và lưu bài thuyết trình. Các hình nhóm rất cần thiết để sắp xếp các hình liên quan cần được thao tác như một đơn vị duy nhất, cung cấp nhiều cấu trúc và khả năng kiểm soát hơn đối với các thành phần bài thuyết trình của bạn.

## Hướng dẫn cài đặt

1. Cài đặt Aspose.Slides cho .NET bằng một trong các phương pháp sau:

   - Trình quản lý gói NuGet:
     ```
     Install-Package Aspose.Slides.NET
     ```

   - Bảng điều khiển quản lý gói:
     ```
     PM> Install-Package Aspose.Slides.NET
     ```

   - .NETCLI:
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. Thêm tài liệu tham khảo vào dự án của bạn:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

3. Thiết lập giấy phép (tùy chọn cho mục đích sử dụng sản xuất):
   ```csharp
   // Đặt giấy phép để xóa các hạn chế đánh giá
   License license = new License();
   license.SetLicense("Aspose.Slides.lic");
   ```

# Căn chỉnh và nhóm các hình dạng trong bài thuyết trình PowerPoint

Aspose.Slides for .NET cung cấp các khả năng mạnh mẽ để tạo và thao tác các bài thuyết trình PowerPoint theo chương trình. Ví dụ này minh họa cách căn chỉnh các hình dạng trong các slide và làm việc với các hình dạng được nhóm trong các bài thuyết trình PowerPoint. Bạn sẽ học cách định vị các hình chữ nhật ở các tọa độ cụ thể, căn chỉnh chúng theo slide hoặc với nhau và căn chỉnh các hình dạng có chọn lọc trong các nhóm.

## Hướng dẫn cài đặt

1. Cài đặt Aspose.Slides cho .NET bằng một trong các phương pháp sau:
   
   - Trình quản lý gói NuGet:
     ```
     Install-Package Aspose.Slides.NET
     ```
   
   - Bảng điều khiển quản lý gói:
     ```
     PM> Install-Package Aspose.Slides.NET
     ```
   
   - .NETCLI:
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. Thêm tài liệu tham khảo vào dự án của bạn:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.IO;
   ```

3. Tải xuống Mã ví dụ bên dưới và tích hợp vào dự án của bạn.

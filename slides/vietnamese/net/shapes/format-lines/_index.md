# Tạo hình chữ nhật với định dạng đường tùy chỉnh trong PowerPoint

Aspose.Slides for .NET cung cấp các khả năng mạnh mẽ để tạo và sửa đổi các bài thuyết trình PowerPoint theo chương trình. Ví dụ này minh họa cách tạo hình chữ nhật với định dạng đường tùy chỉnh, bao gồm thiết lập kiểu đường, chiều rộng, mẫu gạch ngang và màu. Mã mẫu tạo một bài thuyết trình mới, thêm hình chữ nhật vào slide đầu tiên và áp dụng định dạng cụ thể cho cả phần tô và phần viền của hình, tạo thành một thành phần hấp dẫn về mặt thị giác với phần tô màu trắng và phần viền gạch ngang màu xanh lam.

## Hướng dẫn cài đặt

1. Cài đặt Aspose.Slides cho .NET bằng một trong các phương pháp sau:

   - Trình quản lý gói NuGet:
     ```
     Install-Package Aspose.Slides
     ```

   - Bảng điều khiển quản lý gói:
     ```
     PM> Install-Package Aspose.Slides
     ```

   - .NETCLI:
     ```
     dotnet add package Aspose.Slides
     ```

2. Thêm không gian tên cần thiết vào tệp C# của bạn:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

# Tạo hình elip tô màu gradient trong PowerPoint với Aspose.Slides

Ví dụ này minh họa cách lập trình tạo bản trình bày PowerPoint có hình elip với hiệu ứng tô màu gradient tùy chỉnh. Mã tạo bản trình bày mới, thêm hình elip vào trang chiếu đầu tiên và áp dụng tô màu gradient tuyến tính với các điểm dừng màu đỏ và tím. Aspose.Slides for .NET cung cấp API toàn diện để tạo và thao tác các bản trình bày PowerPoint với khả năng kiểm soát chính xác các hình dạng, định dạng và hiệu ứng.

## Hướng dẫn cài đặt

1. Cài đặt Aspose.Slides cho .NET bằng một trong các phương pháp sau:
   - Trình quản lý gói NuGet: `Install-Package Aspose.Slides.NET`
   - .NETCLI: `dotnet add package Aspose.Slides.NET`
   - Bảng điều khiển quản lý gói: `Install-Package Aspose.Slides.NET`

2. Thêm câu lệnh using sau vào tệp C# của bạn:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

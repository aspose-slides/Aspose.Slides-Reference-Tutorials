# Thêm hình ảnh vào slide PowerPoint bằng Aspose.Slides

Aspose.Slides for .NET cung cấp các khả năng mạnh mẽ để tạo và sửa đổi các bài thuyết trình PowerPoint theo chương trình. Ví dụ này minh họa cách tải hình ảnh bên ngoài và thêm vào slide thuyết trình dưới dạng khung hình - một yêu cầu phổ biến khi tạo báo cáo, tạo mẫu hoặc xây dựng bài thuyết trình động từ các nguồn dữ liệu.

Mã bên dưới xử lý mọi thứ từ việc tạo bản trình bày mới đến thêm hình ảnh và lưu tệp PPTX cuối cùng, tất cả chỉ bằng một vài dòng mã C#.

## Hướng dẫn cài đặt

1. Cài đặt Aspose.Slides cho .NET bằng một trong các phương pháp sau:
   - Trình quản lý gói NuGet: `Install-Package Aspose.Slides.NET`
   - .NETCLI: `dotnet add package Aspose.Slides.NET`
   - Hoặc tải trực tiếp từ [Trang web Aspose](https://releases.aspose.com/slides/net/)

2. Thêm không gian tên vào tệp C# của bạn:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

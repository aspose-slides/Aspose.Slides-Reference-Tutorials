# Tạo và xoay hình chữ nhật trong PowerPoint

Aspose.Slides for .NET là một API mạnh mẽ cho phép các nhà phát triển tạo, chỉnh sửa và chuyển đổi các bài thuyết trình PowerPoint mà không cần cài đặt Microsoft PowerPoint. Ví dụ này trình bày cách tạo bài thuyết trình có hình chữ nhật xoay, thể hiện khả năng thao tác hình dạng của thư viện.

Với Aspose.Slides, bạn có thể lập trình thêm nhiều hình dạng khác nhau vào slide của mình và áp dụng các phép biến đổi như xoay. Chức năng này đặc biệt hữu ích để tạo các bài thuyết trình động với khả năng kiểm soát chính xác các thành phần trực quan.

## Hướng dẫn cài đặt

1. Cài đặt thông qua Trình quản lý gói NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Cài đặt qua .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Thêm lệnh using vào mã của bạn:
   ```csharp
   using Aspose.Slides;
   ```

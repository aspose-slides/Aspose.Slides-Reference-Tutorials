# Tạo hình dạng với hình ảnh được tô theo ô trong PowerPoint bằng Aspose.Slides

Aspose.Slides for .NET cung cấp các khả năng mạnh mẽ để tạo và thao tác các bài thuyết trình PowerPoint theo chương trình. Ví dụ này minh họa cách tạo hình chữ nhật và tô hình chữ nhật đó bằng mẫu hình ảnh xếp gạch. Tô hình ảnh xếp gạch đặc biệt hữu ích để tạo nền có họa tiết, hình mờ hoặc các thành phần trang trí trong bài thuyết trình của bạn.

## Hướng dẫn cài đặt

1. Cài đặt thông qua Trình quản lý gói NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Cài đặt qua .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Cài đặt thủ công:
   - Tải xuống phiên bản mới nhất từ [Aspose.Slides cho .NET](https://releases.aspose.com/slides/net/)
   - Thêm tham chiếu đến Aspose.Slides.dll trong dự án của bạn

## Tài nguyên

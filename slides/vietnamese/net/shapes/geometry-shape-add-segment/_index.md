# Thêm các phân đoạn vào GeometryShape trong bài thuyết trình PowerPoint

Ví dụ này trình bày cách truy cập và sửa đổi đường dẫn hình học của các hình dạng trong bản trình bày PowerPoint bằng Aspose.Slides cho .NET. Bằng cách thêm các đoạn thẳng tùy chỉnh vào hình chữ nhật, bạn có thể tạo ra các hình dạng hình học phức tạp hơn mà không có sẵn thông qua các loại hình dạng chuẩn. Kỹ thuật này hữu ích để tạo sơ đồ tùy chỉnh, sơ đồ luồng và các thành phần trực quan khác trong bản trình bày của bạn.

## Hướng dẫn cài đặt

1. Cài đặt Aspose.Slides cho .NET bằng một trong các phương pháp sau:

   - Trình quản lý gói NuGet:
     ```
     Install-Package Aspose.Slides.NET
     ```

   - Bảng điều khiển quản lý gói:
     ```
     PM> Install-Package Aspose.Slides.NET
     ```

   - .NETCLI:
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. Thêm không gian tên vào tệp C# của bạn:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

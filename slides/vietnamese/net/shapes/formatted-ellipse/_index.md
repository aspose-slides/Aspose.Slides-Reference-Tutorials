# Tạo hình elip trong PowerPoint với Aspose.Slides

Aspose.Slides for .NET là một API mạnh mẽ cho phép các nhà phát triển tạo, đọc, viết, sửa đổi và chuyển đổi các bài thuyết trình PowerPoint mà không cần Microsoft PowerPoint. Ví dụ này trình bày cách tạo hình elip trong bài thuyết trình, áp dụng định dạng bao gồm màu tô và các thuộc tính đường kẻ, và lưu kết quả dưới dạng tệp PPTX. Mã này giới thiệu khả năng thao tác các hình dạng của Aspose.Slides với khả năng kiểm soát chính xác về giao diện và vị trí của chúng.

## Hướng dẫn cài đặt

1. Cài đặt thông qua Trình quản lý gói NuGet:
   ```
   Install-Package Aspose.Slides
   ```

   Hoặc sử dụng .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

2. Thêm tài liệu tham khảo:
   Thêm không gian tên sau vào tệp C# của bạn:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

3. Thiết lập giấy phép (Tùy chọn):
   Nếu bạn có giấy phép, hãy áp dụng bằng cách:
   ```csharp
   License license = new License();
   license.SetLicense("Aspose.Slides.lic");
   ```

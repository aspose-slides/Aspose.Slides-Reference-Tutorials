# Tạo văn bản dọc trong PowerPoint bằng Aspose.Slides

Aspose.Slides for .NET cung cấp các khả năng mạnh mẽ để tạo và thao tác các bài thuyết trình PowerPoint theo chương trình. Ví dụ này minh họa cách tạo bài thuyết trình có hình chữ nhật với văn bản xoay theo chiều dọc. Văn bản được định hướng theo góc 270 độ, khiến nó được đọc từ dưới lên trên. Kỹ thuật này hữu ích để tạo nhãn bên, bố cục văn bản độc đáo hoặc các thành phần thiết kế chuyên biệt trong bài thuyết trình của bạn.

## Hướng dẫn cài đặt

1. Cài đặt Aspose.Slides cho .NET
   
   Cài đặt gói Aspose.Slides bằng Trình quản lý gói NuGet:
   
   ```csharp
   Install-Package Aspose.Slides
   ```
   
   Hoặc sử dụng .NET CLI:
   
   ```csharp
   dotnet add package Aspose.Slides
   ```

2. Thêm tài liệu tham khảo
   
   Thêm câu lệnh using sau vào tệp C# của bạn:
   
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

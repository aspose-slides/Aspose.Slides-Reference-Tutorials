# Tạo bài thuyết trình PowerPoint với văn bản được định dạng

Aspose.Slides for .NET là một API mạnh mẽ cho phép các nhà phát triển tạo, thao tác và chuyển đổi các bài thuyết trình PowerPoint mà không cần Microsoft PowerPoint. Ví dụ này minh họa cách tạo một bài thuyết trình đơn giản có hình chữ nhật với các đoạn văn bản được định dạng.

Mã này tạo một bản trình bày mới, thêm hình chữ nhật vào trang chiếu đầu tiên, sau đó thêm nhiều đoạn văn với các tùy chọn định dạng văn bản khác nhau. Mã này cho biết cách thiết lập các thuộc tính phông chữ như kích thước và họ cho các phần văn bản cụ thể.

## Hướng dẫn cài đặt

1. Cài đặt thông qua Trình quản lý gói NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Cài đặt qua .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Cài đặt thông qua Package Manager Console:
   ```
   PM> Install-Package Aspose.Slides
   ```

4. Thêm tham chiếu vào dự án của bạn:
   Sau khi cài đặt, hãy thêm tham chiếu đến dự án của bạn:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

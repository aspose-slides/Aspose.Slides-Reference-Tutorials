# Tạo bài thuyết trình PowerPoint với TextBox trong Aspose.Slides cho .NET

Aspose.Slides for .NET là một API mạnh mẽ cho phép các nhà phát triển tạo, chỉnh sửa và chuyển đổi các bài thuyết trình PowerPoint mà không cần Microsoft PowerPoint. Ví dụ này minh họa cách tạo bài thuyết trình với hộp văn bản tùy chỉnh, áp dụng định dạng và lưu dưới dạng tệp PPTX.

## Hướng dẫn cài đặt

1. Cài đặt thông qua Trình quản lý gói NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Cài đặt qua .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Cài đặt thủ công:
   - Tải xuống phiên bản mới nhất từ [Trang phát hành Aspose.Slides cho .NET](https://releases.aspose.com/slides/net/)
   - Thêm tham chiếu DLL vào dự án của bạn

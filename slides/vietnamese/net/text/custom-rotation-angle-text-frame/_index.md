# Xoay tiêu đề và nhãn biểu đồ trong PowerPoint với Aspose.Slides

Aspose.Slides for .NET cung cấp các khả năng mạnh mẽ để tạo và thao tác các bài thuyết trình PowerPoint theo chương trình. Ví dụ này minh họa cách tạo bài thuyết trình với biểu đồ cột nhóm có các thành phần văn bản xoay - cụ thể là xoay cả tiêu đề biểu đồ và nhãn dữ liệu để cải thiện khả năng đọc và tính hấp dẫn trực quan.

## Hướng dẫn cài đặt

1. Cài đặt thông qua Trình quản lý gói NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Hoặc cài đặt thông qua .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Thêm không gian tên vào tệp C# của bạn:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Charts;
   ```

# Tạo các điểm đầu dòng nhiều cấp trong PowerPoint với Aspose.Slides

Aspose.Slides for .NET cung cấp một API mạnh mẽ để tạo và thao tác các bài thuyết trình PowerPoint theo chương trình. Ví dụ này minh họa cách tạo bài thuyết trình với các điểm bullet nhiều cấp, cho thấy cách thêm hình dạng tự động với khung văn bản và định dạng đoạn văn với các kiểu bullet và mức thụt lề khác nhau. Bạn có thể dễ dàng tạo nội dung phân cấp với các ký hiệu bullet tùy chỉnh cho từng mức thụt lề, hoàn hảo để tạo các bài thuyết trình có cấu trúc và chuyên nghiệp mà không cần cài đặt Microsoft PowerPoint.

## Hướng dẫn cài đặt

1. Cài đặt thông qua Trình quản lý gói NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. Cài đặt qua .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Cài đặt thông qua Package Manager Console:
   ```
   PM> Install-Package Aspose.Slides
   ```

4. Cài đặt thủ công:
   - Tải xuống phiên bản mới nhất từ [Aspose.Slides cho .NET](https://releases.aspose.com/slides/net/)
   - Thêm tham chiếu đến Aspose.Slides.dll trong dự án của bạn

## Tài nguyên hữu ích

- [Tài liệu tham khảo API Aspose.Slides cho .NET](https://reference.aspose.com/slides/net/)

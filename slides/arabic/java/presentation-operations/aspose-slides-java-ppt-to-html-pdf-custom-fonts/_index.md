# تحويل PPT إلى HTML/PDF باستخدام الخطوط المخصصة باستخدام Aspose.Slides لـ Java

مرحبًا بكم في هذا الدليل الشامل حول استخدام Aspose.Slides لجافا لتحويل عروض PowerPoint التقديمية إلى صيغ HTML وPDF مع تحديد خط عادي افتراضي. سواء كنت تسعى إلى تنسيق طباعة متناسق عبر مختلف المنصات أو تتطلع إلى تبسيط سير عمل إدارة مستنداتك، سيساعدك هذا البرنامج التعليمي على الاستفادة من قوة Aspose.Slides بسهولة.

## مقدمة

قد يؤدي تحويل ملفات PowerPoint غالبًا إلى ظهور خطوط غير متناسقة في مستندات الإخراج، مما يُمثل مشكلة عند عرض البيانات بشكل احترافي. باستخدام Aspose.Slides لـ Java، نعالج هذه المشكلة بتعيين خط عادي افتراضي أثناء عمليات التحويل. في هذا البرنامج التعليمي، ستتعلم كيفية حفظ عروضك التقديمية بتنسيق HTML وPDF باستخدام خطوط مُحددة باستخدام Aspose.Slides.

**ما سوف تتعلمه:**
- كيفية إعداد Aspose.Slides لـ Java
- خطوات تحويل ملفات PowerPoint إلى HTML مع تحديد الخط العادي الافتراضي
- طرق تصدير العروض التقديمية إلى صيغة PDF، مع الحفاظ على تناسق الطباعة

دعونا نبدأ بمراجعة المتطلبات الأساسية قبل الغوص في دليل التنفيذ.

## المتطلبات الأساسية

قبل تحويل العروض التقديمية الخاصة بك باستخدام Aspose.Slides لـ Java، تأكد من توفر الأساسيات التالية:

### المكتبات والإصدارات المطلوبة

أدرج مكتبة Aspose.Slides في مشروعك. تأكد من تثبيت Maven أو Gradle في بيئة التطوير لديك.

**متطلبات إعداد البيئة:**
- **مجموعة تطوير Java (JDK):** يعد JDK 16 مطلوبًا للتوافق مع Aspose.Slides الإصدار 25.4.
- **بيئة التطوير المتكاملة (IDE):** أي IDE مثل IntelliJ IDEA أو Eclipse سوف يعمل بشكل جيد.

### متطلبات المعرفة

يوصى بالفهم الأساسي لبرمجة Java والتعرف على أدوات بناء Maven/Gradle للمتابعة بفعالية.

## إعداد Aspose.Slides لـ Java

لبدء استخدام Aspose.Slides، أدرجه في تبعيات مشروعك. إليك الطريقة:

**مافن:**
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

**جرادل:**
```gradle
implementation group: 'com.aspose', name: 'aspose-slides', version: '25.4', classifier: 'jdk16'
```

**التحميل المباشر:**
للإعداد اليدوي، قم بتنزيل الإصدار الأحدث من [إصدارات Aspose.Slides لـ Java](https://releases.aspose.com/slides/java/).

### الحصول على الترخيص
يمكنك البدء بفترة تجريبية مجانية من Aspose.Slides لاستكشاف ميزاته. لاستخدام متواصل، فكّر في شراء ترخيص أو التقدم بطلب ترخيص مؤقت إذا كنت بحاجة إلى مزيد من الوقت للتقييم.

## دليل التنفيذ

يرشدك هذا القسم خلال الخطوات اللازمة لتحويل عروض PowerPoint التقديمية مع الحفاظ على اتساق الخط.

### حفظ العرض التقديمي بتنسيق HTML باستخدام الخط العادي الافتراضي

يتيح تحويل العرض التقديمي إلى تنسيق HTML عرضه في أي متصفح ويب، مما يضمن وصولاً أوسع. إليك كيفية تعيين خط عادي افتراضي لهذا التحويل:

#### الخطوة 1: تهيئة كائن العرض التقديمي
قم بتحميل ملف PowerPoint الخاص بك باستخدام `Presentation` فصل.
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY/DefaultFonts.pptx"; // استبدل بمسار دليل المستند الخاص بك
Presentation pres = new Presentation(dataDir);
```

#### الخطوة 2: تكوين خيارات HTML
يثبت `HtmlOptions`، تحديد الخط العادي الافتراضي الذي تريد استخدامه في ملف HTML المُصدَّر.
```java
HtmlOptions htmlOpts = new HtmlOptions();
htmlOpts.setDefaultRegularFont("Arial Black"); // قم بتعيين الخط المطلوب
```

#### الخطوة 3: الحفظ بصيغة HTML
وأخيرًا، احفظ العرض التقديمي الخاص بك باستخدام الخيارات التي تم تكوينها:
```java
String outPath = "YOUR_OUTPUT_DIRECTORY/";
pres.save(outPath + "Presentation-out-ArialBlack.html", SaveFormat.Html, htmlOpts);
```
كرر هذه الخطوات باستخدام خط مختلف إذا لزم الأمر.

### حفظ العرض التقديمي بصيغة PDF باستخدام الخط العادي الافتراضي
يضمن التصدير إلى PDF إمكانية مشاركة عروضك التقديمية بتنسيق متوافق عالميًا. إليك كيفية تحديد الخط العادي الافتراضي لتحويل PDF:

#### الخطوة 1: تهيئة PdfOptions
على غرار HTML، ابدأ بالتكوين `PdfOptions`.
```java
PdfOptions pdfOpts = new PdfOptions();
pdfOpts.setDefaultRegularFont("Arial Black"); // قم بتعيين الخط المطلوب هنا أيضًا
```

#### الخطوة 2: الحفظ بتنسيق PDF
تصدير العرض التقديمي باستخدام الخيارات التالية:
```java
pres.save(outPath + "Presentation-out-ArialBlack.pdf", SaveFormat.Pdf, pdfOpts);
```

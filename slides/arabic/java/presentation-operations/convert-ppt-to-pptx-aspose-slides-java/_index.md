**مافن:**

أضف هذه التبعية إلى `pom.xml` ملف:

```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>

    <classifier>jdk16</classifier>
</dependency>
```

## دليل التنفيذ

الآن بعد أن قمت بإعداد كل شيء، دعنا نقوم بتحويل ملف PPT إلى PPTX باستخدام Aspose.Slides لـ Java.

### ميزة تحويل PPT إلى PPTX

تُركز هذه الميزة على ترقية تنسيق ملفات العروض التقديمية من .ppt إلى .pptx. إليك كيفية تنفيذها:

#### الخطوة 1: تحديد أدلة المصدر والوجهة

قبل البدء، قم بتحديد مكان وجود ملف PPT المصدر والمكان الذي تريد حفظ ملف PPTX المحول فيه.

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
String outputDir = "YOUR_OUTPUT_DIRECTORY";
```

يستبدل `YOUR_DOCUMENT_DIRECTORY` و `YOUR_OUTPUT_DIRECTORY` مع المسارات الفعلية على نظامك.

#### الخطوة 2: إنشاء كائن عرض تقديمي

إنشاء مثيل لـ `Presentation` الفئة التي تُمثل ملف PPT الخاص بك. يُعد هذا الكائن نقطة دخول للعمليات على مستند العرض التقديمي.

```java
import com.aspose.slides.Presentation;

// قم بتحميل ملف PPT الموجود
Presentation pres = new Presentation(dataDir + "PPTtoPPTX.ppt");
```

#### الخطوة 3: احفظ العرض التقديمي بتنسيق PPTX

استخدم `save` طريقة لتحويل وحفظ العرض التقديمي كملف .pptx. تتطلب هذه الطريقة معلمتين: مسار الوجهة والتنسيق.

```java
import com.aspose.slides.SaveFormat;

try {
    // تحويل وحفظ PPT بصيغة PPTX
    pres.save(outputDir + "PPTtoPPTX_out.pptx", SaveFormat.Pptx);
} finally {
    if (pres != null) pres.dispose();
}
```

**توضيح:**
- ال `save` تحوّل هذه الطريقة عرضك التقديمي إلى التنسيق المحدد. هنا، نستخدم `SaveFormat.Pptx`.
- تخلص دائمًا من كائن العرض التقديمي في `finally` منع تحرير الموارد.

#### نصائح استكشاف الأخطاء وإصلاحها

- **استثناء عدم العثور على الملف:** تأكد من أن مسار ملف المصدر صحيح ويمكن الوصول إليه.
- **مشاكل الأذونات:** تحقق مما إذا كان لديك أذونات الكتابة لدليل الإخراج.

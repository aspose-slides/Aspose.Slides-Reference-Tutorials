## دليل التنفيذ
الآن، دعنا ننتقل إلى تنفيذ ميزتنا: إنشاء رسم SmartArt واستخراج الصورة المصغرة الخاصة به.

### إنشاء شكل SmartArt
1. **تهيئة العرض التقديمي**
   ابدأ بإنشاء مثيل `Presentation` الفئة التي تمثل ملف PPTX الخاص بك.

2. **إضافة رسم SmartArt**
   ```java
   // أضف شكل SmartArt في الموضع (10، 10) بعرض = 400 وارتفاع = 300 باستخدام تخطيط BasicCycle
   ISmartArt smart = pres.getSlides().get_Item(0).getShapes().addSmartArt(10, 10, 400, 300, SmartArtLayoutType.BasicCycle);
   ```
   - **المعلمات موضحة:**
     - `10, 10`:إحداثيات X وY لتحديد المواقع.
     - `400, 300`:العرض والارتفاع لشكل SmartArt.
     - `SmartArtLayoutType.BasicCycle`:نوع التخطيط الذي يحدد النمط.

### استخراج الصورة المصغرة من ملاحظة فرعية
1. **الوصول إلى عقدة محددة**
   ```java
   // الحصول على مرجع لعقدة باستخدام فهرسها (الفهرس 1)
   ISmartArtNode node = smart.getNodes().get_Item(1);
   ```
   - تمثل العقد في SmartArt عناصر فردية، ويمكنك الوصول إليها من خلال فهرسها.

2. **استخراج الصورة المصغرة**
   ```java
   // احصل على صورة مصغرة من الشكل الأول في ملاحظة الطفل
   IImage img = node.getShapes().get_Item(0).getImage();
   
   // احفظ الصورة المصغرة في دليل بتنسيق JPEG
   img.save("YOUR_OUTPUT_DIRECTORY/SmartArt_ChildNote_Thumbnail_out.jpeg", ImageFormat.Jpeg);
   ```
   - **لماذا هذه الخطوة؟** يتيح لك استخراج الصور المصغرة استخدام هذه الصور في أماكن أخرى، مثل التقارير أو العروض التقديمية.

### نصائح استكشاف الأخطاء وإصلاحها
- تأكد من أن دليل الإخراج الخاص بك مضبوط بشكل صحيح وقابل للكتابة.
- إذا واجهت مشكلات مع تنسيق الصورة، فتأكد من `ImageFormat` المعلمة تطابق متطلباتك.

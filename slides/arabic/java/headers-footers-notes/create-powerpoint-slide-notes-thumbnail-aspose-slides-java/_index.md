# إنشاء صور مصغرة لشرائح PowerPoint باستخدام Aspose.Slides لـ Java

- **متطلبات المعرفة**:فهم أساسيات برمجة Java والمعرفة بعروض PowerPoint.

## إعداد Aspose.Slides لـ Java
للبدء، عليك دمج Aspose.Slides في مشروع Java الخاص بك. إليك كيفية القيام بذلك باستخدام Maven أو Gradle:

### إعداد Maven
أضف التبعية التالية إلى ملفك `pom.xml` ملف:

```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

### إعداد Gradle
قم بتضمين هذا في `build.gradle` ملف:

```gradle
implementation group: 'com.aspose', name: 'aspose-slides', version: '25.4', classifier: 'jdk16'
```

## دليل التنفيذ
الآن بعد أن قمت بإعداد Aspose.Slides لـ Java، دعنا ننتقل إلى كيفية إنشاء صورة مصغرة من ملاحظات الشريحة.

### إنشاء صورة مصغرة من ملاحظات الشريحة
توضح هذه الميزة كيفية إنشاء صورة للملاحظات المرتبطة بشريحة في عرض تقديمي في PowerPoint.

#### الخطوة 1: تحديد المسارات وتحميل العرض التقديمي
ابدأ بتحديد مجلدات المستندات والإخراج. ثم حمّل ملف العرض التقديمي:

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY/ThumbnailFromSlideInNotes.pptx";
String outputDir = "YOUR_OUTPUT_DIRECTORY/";

// إنشاء فئة عرض تقديمي تمثل ملف العرض التقديمي.
Presentation pres = new Presentation(dataDir);
```

#### الخطوة 2: الوصول إلى الشريحة وتعيين أبعاد الصورة المصغرة
قم بالوصول إلى الشريحة المطلوبة وحدد أبعاد الصورة المصغرة الخاصة بك:

```java
ISlide sld = pres.getSlides().get_Item(0);

int desiredX = 1200;
int desiredY = 800;

// حساب القيم المقاسة بناءً على حجم الشريحة.
float ScaleX = (float) (1.0 / pres.getSlideSize().getSize().getWidth()) * desiredX;
float ScaleY = (float) (1.0 / pres.getSlideSize().getSize().getHeight()) * desiredY;
```

#### الخطوة 3: إنشاء الصورة المصغرة وحفظها
قم بإنشاء صورة مصغرة لملاحظات الشريحة باستخدام المقياس المحدد، ثم احفظها:

```java
IImage img = sld.getImage(ScaleX, ScaleY);
img.save(outputDir + "Notes_tnail_out.jpg");
```

#### الخطوة 4: تنظيف الموارد
وأخيرًا، تأكد من التخلص من الموارد لمنع تسرب الذاكرة:

```java
if (pres != null) pres.dispose();
```

### نصائح استكشاف الأخطاء وإصلاحها

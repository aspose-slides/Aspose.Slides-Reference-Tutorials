## إعداد Aspose.Slides لـ Java

لبدء استخدام Aspose.Slides في مشروع Java الخاص بك، عليك إضافته كتبعية. إليك الطريقة:

### إعداد Maven
أضف المقطع التالي إلى ملفك `pom.xml` ملف:
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

### إعداد Gradle
قم بتضمين هذا في `build.gradle` ملف:
```gradle
implementation group: 'com.aspose', name: 'aspose-slides', version: '25.4', classifier: 'jdk16'
```

## دليل التنفيذ

الآن بعد أن قمنا بإعداد Aspose.Slides، دعنا نتعمق في تنفيذ الميزتين الرئيسيتين لدينا: إزالة الشرائح الرئيسية وشرائح التخطيط غير المستخدمة.

### إزالة الشرائح الرئيسية غير المستخدمة

#### ملخص
تساعدك هذه الميزة على تنظيم عروض PowerPoint الخاصة بك عن طريق حذف أي شرائح رئيسية غير مستخدمة، مما يقلل حجم الملف بشكل فعال ويحسن الأداء.

**التنفيذ خطوة بخطوة**

##### تحميل العرض التقديمي
ابدأ بتحميل العرض التقديمي الخاص بك من المسار المحدد:
```java
String pptxFileName = "YOUR_DOCUMENT_DIRECTORY/MultipleMaster.pptx";
Presentation pres = new Presentation(pptxFileName);
```

##### إزالة الشرائح الرئيسية غير المستخدمة
استخدم `Compress` فئة لإزالة الشرائح الرئيسية غير المستخدمة:
```java
try {
    Compress.removeUnusedMasterSlides(pres);
} finally {
    if (pres != null) pres.dispose();
}
```
*توضيح:* الطريقة `removeUnusedMasterSlides()` يقوم بمسح كافة الشرائح الرئيسية ويحتفظ فقط بالشرائح المستخدمة.

### إزالة شرائح التخطيط غير المستخدمة

#### ملخص
كما هو الحال مع الشرائح الرئيسية، قد تُضخّم شرائح التخطيط عرضك التقديمي. تضمن هذه الميزة بقاء شرائح التخطيط الضرورية فقط.

**التنفيذ خطوة بخطوة**

##### تحميل العرض التقديمي
تأكد من تحميل العرض التقديمي الخاص بك تمامًا كما كان من قبل:
```java
String pptxFileName = "YOUR_DOCUMENT_DIRECTORY/MultipleMaster.pptx";
Presentation pres = new Presentation(pptxFileName);
```

##### إزالة شرائح التخطيط غير المستخدمة
استخدم `Compress` فئة لإزالة شرائح التخطيط غير المستخدمة:
```java
try {
    Compress.removeUnusedLayoutSlides(pres);
} finally {
    if (pres != null) pres.dispose();
}
```
*توضيح:* الطريقة `removeUnusedLayoutSlides()` تعمل بطريقة مماثلة عن طريق إزالة التخطيطات غير المرتبطة بأي شرائح.

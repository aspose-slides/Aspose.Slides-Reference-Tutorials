# كيفية تكوين حالة العرض التقديمي العادية باستخدام Aspose.Slides لـ Java

## مقدمة

يُمكن لتخصيص العرض التقديمي الأولي أن يُحسّن فعاليته بشكل ملحوظ، سواءً للاجتماعات أو الوحدات التعليمية. يُرشدك هذا البرنامج التعليمي إلى كيفية استخدام Aspose.Slides لجافا لضبط العرض العادي لعروضك التقديمية، مما يُحسّن سهولة الاستخدام والاحترافية.

**ما سوف تتعلمه:**
- ضبط حالة شريط التقسيم الأفقي والرأسي.
- ضبط خصائص الجزء العلوي المستعادة مثل الضبط التلقائي وحجم الأبعاد.
- تمكين أيقونات المخطط التفصيلي في حالة العرض العادي.
- حفظ هذه التكوينات بشكل فعال.

قبل أن نبدأ، دعونا نراجع المتطلبات الأساسية لهذا البرنامج التعليمي.

## المتطلبات الأساسية

تأكد من أن لديك:

### المكتبات والتبعيات المطلوبة
- **Aspose.Slides لـ Java**:ضروري للتعامل مع عروض PowerPoint برمجيًا.
- **مجموعة تطوير جافا (JDK)**:يُطلب JDK 16 أو أعلى.

### متطلبات إعداد البيئة
- بيئة تطوير متكاملة (IDE) مثل IntelliJ IDEA أو Eclipse أو NetBeans مصممة لتطوير Java.

### متطلبات المعرفة
- فهم أساسي لمفاهيم برمجة جافا.
- المعرفة بأدوات بناء Maven أو Gradle لإدارة التبعيات.

## إعداد Aspose.Slides لـ Java

قبل البدء بتنفيذ الكود، عليك إعداد مكتبة Aspose.Slides في مشروعك. إليك الطريقة:

### إعداد Maven
أضف هذه التبعية إلى `pom.xml` ملف:
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

### إعداد Gradle
قم بتضمين هذا في `build.gradle`:
```gradle
implementation group: 'com.aspose', name: 'aspose-slides', version: '25.4', classifier: 'jdk16'
```

### التحميل المباشر
بدلاً من ذلك، قم بتنزيل أحدث مكتبة Aspose.Slides لـ Java من [صفحة الإصدارات الرسمية](https://releases.aspose.com/slides/java/).

#### الحصول على الترخيص
- **نسخة تجريبية مجانية**:ابدأ بالتجربة المجانية لاستكشاف الإمكانيات الكاملة.
- **رخصة مؤقتة**:الحصول على ترخيص مؤقت للتقييم الموسع.
- **شراء**:فكر في شراء ترخيص للاستخدام على المدى الطويل.

بمجرد تنزيله وإعداده في مشروعك، قم بتشغيل Aspose.Slides كما هو موضح أدناه:
```java
import com.aspose.slides.Presentation;

// تهيئة فئة العرض التقديمي
Presentation pres = new Presentation();
```

## دليل التنفيذ

الآن بعد أن أصبح الإعداد جاهزًا، دعنا نقوم بتكوين حالة العرض العادي للعرض التقديمي.

### تكوين حالات شريط التقسيم

#### ملخص
تُساعدك أشرطة التقسيم على التنقل بين الشرائح والملاحظات. إليك كيفية ضبط حالتها:

- **شريط التقسيم الأفقي**:التحكم في التنقل بين الشرائح.
- **شريط الفاصل العمودي**:إدارة رؤية جزء الملاحظات.

##### تعيين حالة شريط التقسيم الأفقي
```java
pres.getViewProperties().getNormalViewProperties()
    .setHorizontalBarState(SplitterBarStateType.Restored);
```
**توضيح:** ضبط هذا على `Restored` يضمن أن يكون التنقل بين الشرائح مرئيًا بالكامل عند فتح العرض التقديمي.

##### تعيين حالة شريط التقسيم الرأسي
```java
pres.getViewProperties().getNormalViewProperties()
    .setVerticalBarState(SplitterBarStateType.Maximized);
```
**توضيح:** تعرض الحالة المكبرة جميع الملاحظات، مما يسهل الوصول إلى معلومات الشريحة التفصيلية.

### تكوين خصائص الجزء العلوي المستعادة

#### ملخص
يؤدي ضبط خصائص الجزء العلوي المستعادة إلى تحسين تجربة المستخدم من خلال تعيين مظهر الشريحة والملاحظات الأولية.

##### الضبط التلقائي وحجم الأبعاد
```java
pres.getViewProperties().getNormalViewProperties()
    .getRestoredTop().setAutoAdjust(true);
pres.getViewProperties().getNormalViewProperties()
    .getRestoredTop().setDimensionSize(80);
```
**توضيح:** تمكين `auto-adjust` يضمن تخطيطًا سلسًا يتكيف مع أحجام الشاشات المختلفة، في حين أن ضبط حجم الأبعاد يتحكم في رؤية لوحة الملاحظات.

### تمكين أيقونات المخطط التفصيلي

#### ملخص
تساعد أيقونات المخطط التفصيلي في التنقل السريع عبر هياكل الشرائح.

##### تمكين أيقونات المخطط التفصيلي
```java
pres.getViewProperties().getNormalViewProperties()
    .setShowOutlineIcons(true);
```
**توضيح:** يضيف هذا الإعداد رؤية واضحة لأيقونات المخطط التفصيلي، مما يساعد على الوصول السريع إلى المحتوى وتنظيمه.

### حفظ العرض التقديمي
وأخيرًا، احفظ العرض التقديمي الخاص بك باستخدام التكوينات المحدثة:
```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY/presentation_normal_view_state.pptx";
pres.save(dataDir, SaveFormat.Pptx);
```
**توضيح:** يؤدي هذا إلى حفظ التغييرات في موقع محدد بتنسيق PPTX.

# Aspose.Slides Java: تنسيق خلايا الجدول الرئيسي في PowerPoint

## كيفية ضبط ارتفاع الخط ومحاذاة النص والنوع الرأسي لخلايا الجدول باستخدام Aspose.Slides لـ Java

مرحبًا بكم في هذا البرنامج التعليمي الشامل حول استخدام Aspose.Slides لجافا لتحسين تنسيق خلايا الجدول في عروض PowerPoint التقديمية! سواء كنت مطورًا ترغب في أتمتة تعديلات الشرائح أو ترغب ببساطة في تحسين عرض بياناتك، فإن إتقان هذه الميزات سيزيد من احترافية شرائحك وسهولة قراءتها.

## مقدمة

إنشاء جداول جذابة بصريًا ومنسقة جيدًا في PowerPoint قد يكون أمرًا صعبًا. باستخدام Aspose.Slides لـ Java، يمكنك برمجيًا تعديل خطوط خلايا الجدول ومحاذاتها، وحتى تحديد أنواع النصوص العمودية داخل الخلايا. سيرشدك هذا الدليل خلال عملية ضبط ارتفاع الخط، ومحاذاة النص إلى اليمين مع هامش، وضبط اتجاه النص - كل ذلك بسهولة باستخدام لغة Java البرمجية.

**ما سوف تتعلمه:**

- كيفية تكوين ارتفاعات خطوط خلايا الجدول في شرائح PowerPoint
- تقنيات محاذاة النص داخل خلايا الجدول وتعيين الهوامش
- طرق تعيين أنواع النصوص العمودية في الجداول

دعونا نلقي نظرة على المتطلبات الأساسية التي ستحتاجها قبل البدء!

## دليل التنفيذ

سنستكشف ثلاث ميزات رئيسية: ضبط ارتفاعات الخطوط، ومحاذاة النص مع الهوامش، وتكوين أنواع النصوص الرأسية.

### ضبط ارتفاع خط خلايا الجدول

**ملخص:**

يمكن أن يؤدي تعديل ارتفاع الخط في خلايا الجدول إلى تحسين إمكانية القراءة وضمان الاتساق عبر شرائح العرض التقديمي الخاصة بك.

**خطوات:**

#### 1. قم بتحميل العرض التقديمي الخاص بك
ابدأ بتحميل ملف PowerPoint الخاص بك باستخدام Aspose.Slides `Presentation` فصل.
```java
Presentation presentation = new Presentation("YOUR_DOCUMENT_DIRECTORY/pres.pptx");
```

#### 2. الوصول إلى الجدول المطلوب
حدد الجدول الذي ترغب بتعديله، ثم افتحه. هنا، نفترض أنه الشكل الأول في الشريحة.
```java
ISlide slide = presentation.getSlides().get_Item(0);
ITable someTable = (ITable) slide.getShapes().get_Item(0); // يفترض أن الشكل الأول هو جدول
```

#### 3. تكوين PortionFormat لارتفاع الخط
إنشاء وإعداد `PortionFormat` لتحديد ارتفاع الخط المطلوب.
```java
PortionFormat portionFormat = new PortionFormat();
portionFormat.setFontHeight(25);
someTable.setTextFormat(portionFormat); // تطبيق هذا التنسيق على كل النص داخل خلايا الجدول
```

**نصيحة لاستكشاف الأخطاء وإصلاحها:** تأكد من تحديد الجدول بشكل صحيح من خلال فهرسه على الشريحة. استخدم أدوات التسجيل أو التصحيح إذا لزم الأمر.

### ضبط محاذاة النص والهامش الأيمن لخلايا الجدول

**ملخص:**

يمكن أن تعمل إعدادات المحاذاة والهامش الصحيحة على تحسين المظهر المرئي لجداولك بشكل كبير، مما يجعل تفسير البيانات أسهل.

**خطوات:**

#### 1. قم بتحميل العرض التقديمي الخاص بك
كرر الخطوة الأولية لتحميل ملف العرض التقديمي الخاص بك.
```java
Presentation presentation = new Presentation("YOUR_DOCUMENT_DIRECTORY/pres.pptx");
```

#### 2. الوصول إلى الجدول وتحديده
قم بتحديد الجدول كما فعلنا سابقًا.
```java
ISlide slide = presentation.getSlides().get_Item(0);
ITable someTable = (ITable) slide.getShapes().get_Item(0); // يفترض أن الشكل الأول هو جدول
```

#### 3. تكوين ParagraphFormat للمحاذاة والهامش
يثبت `ParagraphFormat` لمحاذاة النص إلى اليمين بهامش محدد.
```java
ParagraphFormat paragraphFormat = new ParagraphFormat();
paragraphFormat.setAlignment(TextAlignment.Right);
paragraphFormat.setMarginRight(20); // تعيين الهامش الأيمن بالنقاط
someTable.setTextFormat(paragraphFormat); // تطبيق هذه الإعدادات على جميع خلايا الجدول
```

**نصيحة لاستكشاف الأخطاء وإصلاحها:** إذا لم يظهر محاذاة النص كما هو متوقع، فتأكد من التحقق من تطبيق تحديد الخلية والتنسيق.

### إعداد نوع النص العمودي لخلايا الجدول

**ملخص:**

بالنسبة للعروض التقديمية الإبداعية أو أنواع معينة من البيانات، يمكن أن يكون تعيين اتجاه النص الرأسي طريقة فريدة لعرض المعلومات.

**خطوات:**

#### 1. قم بتحميل العرض التقديمي الخاص بك
قم بتحميل ملف PowerPoint الخاص بك مرة أخرى.
```java
Presentation presentation = new Presentation("YOUR_DOCUMENT_DIRECTORY/pres.pptx");
```

#### 2. الوصول إلى الجدول
يمكنك الوصول إلى الجدول باستخدام نفس النهج كما في السابق.
```java
ISlide slide = presentation.getSlides().get_Item(0);
ITable someTable = (ITable) slide.getShapes().get_Item(0); // يفترض أن الشكل الأول هو جدول
```

#### 3. تكوين TextFrameFormat لنوع النص العمودي
إنشاء وتكوين `TextFrameFormat` لتعيين اتجاه النص الرأسي.
```java
TextFrameFormat textFrameFormat = new TextFrameFormat();
textFrameFormat.setTextVerticalType(TextVerticalType.Vertical);
someTable.setTextFormat(textFrameFormat); // تطبيق هذا التنسيق داخل جميع خلايا الجدول
```

**نصيحة لاستكشاف الأخطاء وإصلاحها:** تأكد من أن تخطيط الشريحة لديك يدعم النص الرأسي لتجنب النتائج غير المتوقعة.

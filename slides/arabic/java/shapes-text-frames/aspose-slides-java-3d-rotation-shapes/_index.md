# إتقان التأثيرات ثلاثية الأبعاد: تطبيق التدوير ثلاثي الأبعاد على الأشكال باستخدام Aspose.Slides لـ Java

في عالم العروض التقديمية المتغير اليوم، يُمكن لإضافة العمق والأبعاد أن تُبرز شرائحك. سواءً كنت مطورًا محترفًا أو جديدًا في البرمجة، فإن تطبيق تأثيرات الدوران ثلاثية الأبعاد على الأشكال في عروض PowerPoint التقديمية باستخدام Aspose.Slides لـ Java يُحسّن المظهر بشكل ملحوظ. سيرشدك هذا البرنامج التعليمي خلال عملية إنشاء تأثيرات ثلاثية الأبعاد آسرة على أشكال المستطيلات.

## ما سوف تتعلمه

بالانتقال من هذه المقدمة، دعنا نستكشف المتطلبات الأساسية المطلوبة قبل الغوص في التنفيذ.

## إعداد Aspose.Slides لـ Java

للبدء، ستحتاج إلى دمج مكتبة Aspose.Slides في مشروعك. إليك الطريقة:

### إعداد Maven
أضف التبعية التالية إلى ملفك `pom.xml` ملف:
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

### إعداد Gradle
قم بتضمين هذا السطر في `build.gradle` ملف:
```gradle
implementation group: 'com.aspose', name: 'aspose-slides', version: '25.4', classifier: 'jdk16'
```

### التحميل المباشر
بدلاً من ذلك، يمكنك تنزيل الإصدار الأحدث مباشرةً من [إصدارات Aspose.Slides لـ Java](https://releases.aspose.com/slides/java/).

#### الحصول على الترخيص
- **نسخة تجريبية مجانية**:احصل على نسخة تجريبية مجانية لاختبار ميزات المكتبة.
- **رخصة مؤقتة**:اطلب ترخيصًا مؤقتًا إذا لزم الأمر لإجراء اختبار موسع.

### التهيئة والإعداد الأساسي
بمجرد إعداد المكتبة، قم بتهيئتها في تطبيق Java الخاص بك على النحو التالي:
```java
import com.aspose.slides.Presentation;
```

## دليل التنفيذ

لنتعمق في تطبيق التدوير ثلاثي الأبعاد على شكل مستطيل في PowerPoint باستخدام Aspose.Slides لجافا. سنُقسّم هذه العملية إلى خطوات سهلة.

### إنشاء عرض تقديمي وإضافة شكل

#### ملخص
أولاً، نقوم بإنشاء عرض تقديمي جديد ونضيف شكل مستطيل إلى الشريحة الأولى.
```java
// إنشاء مثيل لفئة العرض التقديمي
Presentation pres = new Presentation();

// إضافة شكل مستطيل تلقائي إلى الشريحة الأولى
IShape autoShape = pres.getSlides().get_Item(0).getShapes().addAutoShape(
    ShapeType.Rectangle, 30, 30, 200, 200);
```
**توضيح**: 
- `Presentation` يتم تهيئته لإنشاء عرض تقديمي جديد.
- نضيف شكلًا تلقائيًا من نوع مستطيل في الموضع (30، 30) بأبعاد 200 × 200.

### تطبيق الدوران ثلاثي الأبعاد

#### ملخص
بعد ذلك، نقوم بتكوين التأثيرات ثلاثية الأبعاد على شكل المستطيل الخاص بنا.
```java
// ضبط عمق التأثير ثلاثي الأبعاد
autoShape.getThreeDFormat().setDepth((short) 6);

// تكوين دوران الكاميرا والكتابة للحصول على منظور ثلاثي الأبعاد
autoShape.getThreeDFormat().getCamera().setRotation(40, 35, 20);
autoShape.getThreeDFormat().getCamera().setCameraType(CameraPresetType.IsometricLeftUp);

// اضبط نوع جهاز الإضاءة للحصول على إضاءة متوازنة
autoShape.getThreeDFormat().getLightRig().setLightType(LightRigPresetType.Balanced);
```
**توضيح**: 
- `setDepth` ضبط مدى عمق ظهور التأثير ثلاثي الأبعاد.
- يتم ضبط دوران الكاميرا ونوعها لإنشاء منظور محدد.
- يتم استخدام نظام إضاءة متوازن للحصول على إضاءة متساوية.

### حفظ العرض التقديمي

وأخيرًا، احفظ العرض التقديمي الخاص بك مع تطبيق التأثيرات التالية:
```java
// حفظ العرض التقديمي مع التأثيرات ثلاثية الأبعاد المطبقة على الملف
pres.save("YOUR_OUTPUT_DIRECTORY\\Rotation_out.pptx", com.aspose.slides.SaveFormat.Pptx);
```
**توضيح**: 
- ال `save` تقوم الطريقة بإخراج العرض التقديمي المعدل إلى المسار المحدد.

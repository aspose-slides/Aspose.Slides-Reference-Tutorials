# تدوير المستطيل في عرض تقديمي باستخدام Aspose.Slides Java

## مقدمة

قد يكون تدوير الأشكال في العروض التقديمية أمرًا صعبًا بدون الأدوات المناسبة. مع Aspose.Slides لجافا، أصبح تدوير المستطيلات والأشكال الأخرى سهلًا وفعالًا. سيرشدك هذا البرنامج التعليمي إلى كيفية استخدام Aspose.Slides لتدوير الأشكال بسلاسة.

### ما سوف تتعلمه
- كيفية إعداد Aspose.Slides لـ Java
- إضافة شكل مستطيل إلى شريحة
- تدوير المستطيل بزوايا محددة
- حفظ التغييرات في العرض التقديمي الخاص بك

بحلول نهاية هذا الدليل، ستتمكن من إتقان تدوير الأشكال داخل العروض التقديمية باستخدام Aspose.Slides.

## المتطلبات الأساسية

قبل المتابعة، تأكد من أن لديك:

### المكتبات والإصدارات المطلوبة
1. **Aspose.Slides لـ Java** إصدار المكتبة 25.4 أو أحدث.
2. تم تثبيت JDK (مجموعة تطوير Java) على نظامك.

### متطلبات إعداد البيئة
- بيئة التطوير المتكاملة (IDE) مثل IntelliJ IDEA أو Eclipse.
- أداة بناء Maven أو Gradle تم تكوينها في مشروعك.

### متطلبات المعرفة
إن الفهم الأساسي لبرمجة Java والتعرف على تنسيقات العرض مثل PPTX أمر مفيد.

## إعداد Aspose.Slides لـ Java

قم بتثبيت مكتبة Aspose.Slides باستخدام إحدى الطرق التالية:

**مافن**
أضف هذه التبعية إلى `pom.xml` ملف:

```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

**جرادل**
قم بتضمين ما يلي في `build.gradle` ملف:

```gradle
implementation group: 'com.aspose', name: 'aspose-slides', version: '25.4', classifier: 'jdk16'
```

**التحميل المباشر**
قم بتنزيل المكتبة مباشرة من [إصدارات Aspose.Slides لـ Java](https://releases.aspose.com/slides/java/).

## دليل التنفيذ

يرشدك هذا القسم خلال عملية إنشاء شكل مستطيل وتدويره ضمن العرض التقديمي.

### إنشاء شكل مستطيل وتدويره

#### ملخص
سنضيف شكلًا تلقائيًا من نوع المستطيل إلى شريحة ونقوم بتدويره بمقدار 90 درجة باستخدام Aspose.Slides لـ Java، وهو مثالي للعروض التقديمية الديناميكية.

#### التنفيذ خطوة بخطوة
**1. إعداد كائن العرض التقديمي**
إنشاء `Presentation` الكائن الذي يمثل ملف PPTX الخاص بك:

```java
Presentation pres = new Presentation();
```

**2. الوصول إلى الشريحة الأولى**
انتقل إلى الشريحة الأولى لإضافة الأشكال:

```java
ISlide sld = pres.getSlides().get_Item(0);
```

**3. أضف شكل المستطيل**
أضف شكلًا تلقائيًا من نوع المستطيل بأبعاد وموضع محددين:

```java
IShape shp = sld.getShapes().addAutoShape(ShapeType.Rectangle, 50, 150, 75, 150);
```
- `ShapeType.Rectangle`:يحدد نوع الشكل.
- الإحداثيات `(50, 150)`:مواضع X وY على الشريحة.
- أبعاد `(75, 150)`:عرض وارتفاع المستطيل.

**4. تدوير الشكل**
قم بتدوير المستطيل الخاص بك عن طريق ضبط خاصية التدوير الخاصة به:

```java
shp.setRotation(90);
```
يؤدي هذا إلى تدوير الشكل بمقدار 90 درجة في اتجاه عقارب الساعة.

**5. احفظ العرض التقديمي**
احفظ العرض التقديمي باستخدام المستطيل المُدار:

```java
pres.save(dataDir + "/RectShpRot_out.pptx", SaveFormat.Pptx);
```

### نصائح استكشاف الأخطاء وإصلاحها
- **تأكد من المسار الصحيح**: يؤكد `dataDir` يشير إلى دليل موجود.
- **التحقق من نوع الشكل**:تأكيد استخدامك `ShapeType.Rectangle`.

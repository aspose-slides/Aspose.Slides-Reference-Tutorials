## إعداد Aspose.Slides لـ Java
دمج Aspose.Slides في مشاريع Java سهل للغاية. إليك كيفية إعداده باستخدام Maven أو Gradle:

**مافن:**
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

**جرادل:**
```gradle
implementation group: 'com.aspose', name: 'aspose-slides', version: '25.4', classifier: 'jdk16'
```

بدلا من ذلك، يمكنك [تنزيل أحدث إصدار](https://releases.aspose.com/slides/java/) مباشرة من Aspose.

## دليل التنفيذ
سنقوم بتقسيم التنفيذ إلى أقسام مميزة استنادًا إلى الوظيفة.

### تهيئة العرض التقديمي وإنشاء الشكل
#### ملخص
في هذا القسم، سنُنشئ عرضًا تقديميًا ونضيف إليه أشكالًا. هذه هي نقطة البداية لإنشاء شرائح جذابة بصريًا.

**1. إنشاء عرض تقديمي جديد**
ابدأ بتهيئة مثيل لـ `Presentation` فصل:
```java
Presentation pres = new Presentation();
```

**2. الوصول إلى الشريحة الأولى**
احصل على الشريحة الأولى من كائن العرض التقديمي:
```java
ISlide slide = pres.getSlides().get_Item(0);
```

**3. إضافة الأشكال إلى الشريحة**
إضافة أشكال المستطيلات في المواضع والأحجام المحددة:
```java
IAutoShape shp1 = slide.getShapes().addAutoShape(ShapeType.Rectangle, 0, 0, 50, 50);
IAutoShape shp2 = slide.getShapes().addAutoShape(ShapeType.Rectangle, 100, 0, 50, 50);
```
يوضح مقتطف التعليمات البرمجية هذا كيفية إنشاء تخطيط أساسي لشريحتك.

### إضافة تأثيرات الرسوم المتحركة إلى الأشكال
#### ملخص
تُضفي الرسوم المتحركة حيويةً على العروض التقديمية. هنا، سنضيف تأثيراتٍ للأشكال التي أُنشئت سابقًا.

**1. تحميل عرض تقديمي موجود**
قم بتحميل ملف العرض التقديمي الخاص بك:
```java
Presentation pres = new Presentation("YOUR_DOCUMENT_DIRECTORY/presentation.pptx");
```

**2. استرداد تسلسل الرسوم المتحركة**
الوصول إلى التسلسل الرئيسي للرسوم المتحركة للشريحة:
```java
ISequence sequence = slide.getTimeline().getMainSequence();
```

**3. إضافة تأثيرات إلى الأشكال**
تطبيق `FadedZoom` التأثير مع المحفزات المحددة:
```java
IEffect ef1 = sequence.addEffect(pres.getSlides().get_Item(0).getShapes().get_Item(0),
        EffectType.FadedZoom,
        EffectSubtype.ObjectCenter,
        EffectTriggerType.OnClick);

IEffect ef2 = sequence.addEffect(pres.getSlides().get_Item(0).getShapes().get_Item(1),
        EffectType.FadedZoom,
        EffectSubtype.SlideCenter,
        EffectTriggerType.OnClick);
```
يضيف هذا تفاعلات ديناميكية إلى الأشكال الخاصة بك.

### حفظ العرض التقديمي
#### ملخص
وأخيرًا، احفظ عرضك التقديمي مع جميع التعديلات سليمة.

**1. تحديد مسار ملف الإخراج**
حدد المكان الذي تريد حفظ العرض التقديمي فيه:
```java
String outPres = "YOUR_OUTPUT_DIRECTORY/AnimationFadedZoom-out.pptx";
```

**2. احفظ العرض التقديمي**
استخدم `save` الطريقة لتخزين ملفك:
```java
pres.save(outPres, SaveFormat.Pptx);
```
ويضمن هذا الحفاظ على كافة التغييرات في ملف PPTX جديد.

## إعداد Aspose.Slides لـ Java
لبدء استخدام Aspose.Slides لجافا، يجب عليك أولاً تضمين المكتبة في مشروعك. إليك كيفية إعدادها باستخدام مديري حزم مختلفين:

### استخدام Maven
أضف التبعية التالية إلى ملفك `pom.xml` ملف:
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```
### استخدام Gradle
قم بتضمين هذا في `build.gradle`:
```gradle
implementation group: 'com.aspose', name: 'aspose-slides', version: '25.4', classifier: 'jdk16'
```

## دليل التنفيذ
### ضبط تباعد الأسطر في إطارات النص
توضح هذه الميزة كيفية ضبط تباعد الأسطر ضمن فقرة من إطار النص. إليك دليل خطوة بخطوة:
#### تحميل عرض تقديمي موجود
ابدأ بتحميل ملف العرض التقديمي الخاص بك.
```java
Presentation presentation = new Presentation("YOUR_DOCUMENT_DIRECTORY/Foo.pptx");
```
**توضيح**:هذا يقوم بتهيئة `Presentation` الكائن الذي يسمح لك بالتعامل مع الشرائح داخل مستند PowerPoint.
#### الوصول إلى الشريحة
بعد ذلك، قم بالوصول إلى الشريحة المحددة التي تحتوي على إطار النص الذي تريد تعديله:
```java
ISlide sld = presentation.getSlides().get_Item(0);
```
**توضيح**يتم استرجاع الشريحة الأولى باستخدام الفهرسة الصفرية. عدّل الفهرس حسب الشريحة المطلوبة.
#### الحصول على إطار النص من الشكل
بافتراض أن الشكل الخاص بك هو شكل تلقائي، فاسترد إطار النص الخاص به:
```java
ITextFrame tf1 = ((IAutoShape) sld.getShapes().get_Item(0)).getTextFrame();
```
**توضيح**:هذا يؤدي إلى الوصول إلى `TextFrame` للشكل الأول في الشريحة. تأكد من أن هذا الشكل هو بالفعل شكل نص تلقائي.
#### الوصول إلى الفقرة وتعديلها
استرجاع الفقرة التي تريد تعديلها:
```java
IParagraph para1 = tf1.getParagraphs().get_Item(0);
```
**توضيح**:يؤدي هذا إلى جلب الفقرة الأولى داخل إطار النص، والذي سنقوم بتعديله من أجل تباعد الأسطر.
قم بتكوين معلمات تباعد الأسطر المطلوبة:
```java
para1.getParagraphFormat().setSpaceWithin(80);  // المسافة بين أسطر الفقرة نفسها
para1.getParagraphFormat().setSpaceBefore(40);  // مسافة قبل بداية الفقرة
para1.getParagraphFormat().setSpaceAfter(40);   // مسافة بعد نهاية الفقرة
```
**توضيح**:تعمل هذه الطرق على ضبط المسافات لتعزيز قابلية القراءة والجاذبية البصرية.
#### احفظ عرضك التقديمي
وأخيرًا، احفظ التغييرات:
```java
presentation.save("YOUR_OUTPUT_DIRECTORY/LineSpacing_out.pptx", SaveFormat.Pptx);
```
**توضيح**:يؤدي هذا إلى كتابة كافة التعديلات مرة أخرى في ملف عرض تقديمي جديد في الدليل المحدد.

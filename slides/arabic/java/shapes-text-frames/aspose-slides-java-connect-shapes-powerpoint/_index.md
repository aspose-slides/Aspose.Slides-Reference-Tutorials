# إتقان Aspose.Slides في Java: ربط الأشكال في PowerPoint

**مقدمة**

## دليل التنفيذ

في هذا القسم، سنقوم بتفصيل كيفية ربط الأشكال باستخدام الموصلات في PowerPoint مع Aspose.Slides Java.

### إضافة الأشكال
أولاً، لنُضِف شكلين أساسيين: شكل بيضاوي ومستطيل. سنضعهما في الشريحة الأولى من عرضنا التقديمي.
```java
// إنشاء فئة عرض تقديمي تمثل ملف PPTX
Presentation input = new Presentation();
try {
    // الوصول إلى مجموعة الأشكال للشريحة المحددة (الشريحة الأولى)
    IShapeCollection shapes = input.getSlides().get_Item(0).getShapes();

    // أضف الشكل البيضاوي التلقائي في الموضع (0، 100) بحجم (100 × 100)
    IAutoShape ellipse = shapes.addAutoShape(ShapeType.Ellipse, 0, 100, 100, 100);

    // أضف شكل مستطيل تلقائي في الموضع (100، 300) بحجم (100 × 100)
    IAutoShape rectangle = shapes.addAutoShape(ShapeType.Rectangle, 100, 300, 100, 100);
```

### ربط الأشكال
الآن وقد أصبحت أشكالنا في مكانها، لنربطها بموصل. سنستخدم موصلًا منحنيًا لربط القطع الناقص بالمستطيل.
```java
    // إضافة شكل الموصل إلى مجموعة أشكال الشريحة بدءًا من (0، 0) بحجم (10x10)
    IConnector connector = shapes.addConnector(ShapeType.BentConnector2, 0, 0, 10, 10);

    // ربط القطع الناقص ببداية الموصل
    connector.setStartShapeConnectedTo(ellipse);

    // ربط المستطيل بنهاية الموصل
    connector.setEndShapeConnectedTo(rectangle);
```

### إعادة توجيه الموصل
بمجرد الاتصال، قم بإعادة توجيه الموصل للتأكد من أنه يجد أقصر مسار بين الأشكال.
```java
    // إعادة توجيه الموصل للعثور على أقصر مسار تلقائيًا بين الأشكال
    connector.reroute();
```

### حفظ العرض التقديمي
وأخيرًا، احفظ العرض التقديمي الخاص بك بتنسيق PPTX باسم محدد.
```java
    // احفظ العرض التقديمي بتنسيق PPTX باسم محدد
    input.save("Connecting_shapes_using_connectors_out.pptx", SaveFormat.Pptx);
} finally {
    if (input != null) input.dispose();
}
```

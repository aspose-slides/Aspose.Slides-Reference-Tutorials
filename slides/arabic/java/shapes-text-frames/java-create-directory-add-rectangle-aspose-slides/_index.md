### الميزة 1: إنشاء دليل للإخراج

#### ملخص

تضمن هذه الميزة أن يتمكن تطبيقك من حفظ ملفات الإخراج، مثل العروض التقديمية، دون مواجهة أخطاء متعلقة بالدليل. إليك كيفية التحقق من وجود دليل وإنشائه عند الحاجة:

#### التنفيذ خطوة بخطوة

**التحقق وإنشاء الدليل:**

```java
import java.io.File;

String outputDir = "YOUR_OUTPUT_DIRECTORY";

boolean isExists = new File(outputDir).exists();
if (!isExists) {
    boolean wasCreated = new File(outputDir).mkdirs();
    // التعامل مع الحالة التي لم يتم فيها إنشاء الدليل، إذا لزم الأمر
}
```

**لماذا هذا مهم:** من خلال التحقق من وجود دليل قبل محاولة حفظ الملفات، يصبح تطبيقك أكثر قوة وأقل عرضة لأخطاء وقت التشغيل.

### الميزة 2: إنشاء عرض تقديمي جديد وإضافة شكل مستطيل

#### ملخص

إضافة أشكال مثل المستطيلات تُساعد على تنظيم محتوى الشرائح بصريًا. إليك كيفية إنشاء عرض تقديمي وإضافة شكل مستطيل باستخدام Aspose.Slides:

#### التنفيذ خطوة بخطوة

**إنشاء عرض تقديمي وإضافة الشكل:**

```java
import com.aspose.slides.ISlide;
import com.aspose.slides.Presentation;
import com.aspose.slides.SaveFormat;
import com.aspose.slides.ShapeType;

String documentDir = "YOUR_DOCUMENT_DIRECTORY";
String outputDir = "YOUR_OUTPUT_DIRECTORY";

Presentation pres = new Presentation();
try {
    ISlide sld = pres.getSlides().get_Item(0);
    
    // أضف شكل مستطيل إلى الشريحة.
    sld.getShapes().addAutoShape(ShapeType.Rectangle, 50, 150, 150, 50);

    String outputPath = outputDir + "/RectShp1_out.pptx";
    pres.save(outputPath, SaveFormat.Pptx);
} finally {
    if (pres != null) pres.dispose();
}
```

**لماذا هذا مهم:** يتيح إضافة الأشكال برمجيًا إنشاء محتوى ديناميكي وآلي في العروض التقديمية، وهو ما قد يكون مفيدًا بشكل خاص لإنشاء التقارير أو لوحات المعلومات.

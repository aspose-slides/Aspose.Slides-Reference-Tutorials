## دليل التنفيذ
### تصدير كائنات OLE من PPTX إلى PDF
تُركز هذه الميزة على تصدير عروض PowerPoint التقديمية المُضمنة بعناصر OLE إلى صيغة PDF مع الاحتفاظ بالبيانات المُضمنة. إليك كيفية تحقيق ذلك:

#### الخطوة 1: تحميل العرض التقديمي الخاص بك
قم بتحميل ملف العرض التقديمي الخاص بك باستخدام `Presentation` فصل.

```java
import com.aspose.slides.Presentation;

String dataDir = "YOUR_DOCUMENT_DIRECTORY";
Presentation pres = new Presentation(dataDir + "/PresOleExample.pptx");
```

**لماذا؟**:يؤدي تحميل العرض التقديمي إلى تهيئة نموذج الكائن الذي سيتعامل معه Aspose.Slides.

#### الخطوة 2: تكوين خيارات تصدير PDF
يثبت `PdfOptions` لتضمين بيانات OLE في التصدير الخاص بك.

```java
import com.aspose.slides.PdfOptions;

PdfOptions options = new PdfOptions();
options.setIncludeOleData(true);
```

**لماذا؟**: ال `setIncludeOleData(true)` يضمن الحفاظ على كائنات OLE المضمنة أثناء التحويل، مما يحافظ على سلامة البيانات.

#### الخطوة 3: التصدير إلى PDF
احفظ العرض التقديمي الخاص بك كملف PDF مع الخيارات المحددة.

```java
import com.aspose.slides.SaveFormat;

String outFilePath = "YOUR_OUTPUT_DIRECTORY/PresOleExample.pdf";
pres.save(outFilePath, SaveFormat.Pdf, options);
```

**لماذا؟**:في هذه الخطوة، يتم التحويل ويحفظ ملف PDF الناتج في المجلد الذي اخترته. بتحديد `SaveFormat.Pdf`، يمكنك توجيه Aspose.Slides لإنشاء ملف PDF.

### نصائح استكشاف الأخطاء وإصلاحها
- تأكد من أن مسار العرض التقديمي الخاص بك صحيح لتجنب `FileNotFoundException`.
- تأكد من إعداد الترخيص بشكل صحيح لمنع ظهور العلامة المائية في مخرجاتك.
- إذا لم تظهر بيانات OLE، فتأكد من ذلك مرة أخرى `options.setIncludeOleData(true);` إعدادات.

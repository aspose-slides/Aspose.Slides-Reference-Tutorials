## إعداد Aspose.Slides لـ Java

للبدء، تحتاج إلى التكامل **Aspose.Slides لـ Java** في مشروعك. إليك الطريقة:

### مافن

أضف التبعية التالية إلى ملفك `pom.xml`:

```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

### جرادل

قم بتضمين هذا في `build.gradle`:

```gradle
implementation group: 'com.aspose', name: 'aspose-slides', version: '25.4', classifier: 'jdk16'
```

## دليل التنفيذ

سنقوم بتنفيذ ميزتين رئيسيتين: إنشاء الدلائل وتشفير العروض التقديمية.

### إنشاء دليل

#### ملخص

تتحقق هذه الميزة من وجود دليل موجود وتقوم بإنشائه إذا لزم الأمر، مما يضمن تنظيم ملفات العرض التقديمي لديك بشكل جيد.

#### خطوات التنفيذ

##### الخطوة 1: استيراد الحزم المطلوبة

```java
import java.io.File;
```

##### الخطوة 2: تحديد مسار الدليل

قم بتعيين مسار العنصر النائب الذي تريد تخزين العروض التقديمية الخاصة بك فيه:

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY";
```

##### الخطوة 3: التحقق من الدليل وإنشائه

تحقق مما إذا كان الدليل موجودًا، ثم قم بإنشائه إذا لم يكن كذلك:

```java
boolean isExists = new File(dataDir).exists();
if (!isExists) {
    new File(dataDir).mkdirs(); // إنشاء الدليل مع جميع الدلائل الرئيسية الضرورية.
}
```

#### توضيح

- **`File.exists()`**:الإرجاع `true` إذا كان الملف أو الدليل موجودًا.
- **`File.mkdirs()`**:ينشئ الدليل، بما في ذلك أي أدلة رئيسية ضرورية ولكن غير موجودة.

### تحميل وحفظ العرض التقديمي باستخدام تشفير كلمة المرور

#### ملخص

قم بتشفير العرض التقديمي الخاص بك لحماية محتواه من الوصول غير المصرح به باستخدام كلمة مرور.

#### خطوات التنفيذ

##### الخطوة 1: استيراد حزم Aspose.Slides

```java
import com.aspose.slides.Presentation;
import com.aspose.slides.SaveFormat;
```

##### الخطوة 2: إنشاء كائن العرض التقديمي

إنشاء مثيل لـ `Presentation` فصل:

```java
Presentation pres = new Presentation();
```

##### الخطوة 3: تشفير العرض التقديمي

تطبيق كلمة مرور لتشفير العرض التقديمي:

```java
pres.getProtectionManager().encrypt("pass"); // "pass" هي كلمة المرور التي اخترتها.
```

##### الخطوة 4: حفظ الملف المشفر

احفظ العرض التقديمي في الدليل المحدد مع تطبيق التشفير:

```java
String outputPath = "YOUR_DOCUMENT_DIRECTORY/SaveWithPassword_out.pptx";
pres.save(outputPath, SaveFormat.Pptx); // يحفظ كملف PPTX.
```

#### توضيح

- **`encrypt()`**:يقوم بتأمين العرض التقديمي الخاص بك باستخدام كلمة المرور المقدمة.
- **`save()`**:يكتب العرض التقديمي المشفر على القرص بالتنسيق المحدد.

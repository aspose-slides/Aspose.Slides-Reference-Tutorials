## دليل التنفيذ
سنقوم بتقسيم تنفيذنا إلى ثلاث ميزات رئيسية:

### 1. تحميل شريحة العرض التقديمي والوصول إليها

#### ملخص
يعد تحميل ملف العرض التقديمي الخطوة الأولى للوصول إلى محتواه، بما في ذلك الشرائح والكائنات المضمنة.

#### خطوات التنفيذ

##### تهيئة كائن العرض التقديمي

```java
String dataDir = "YOUR_DOCUMENT_DIRECTORY/";
Presentation pres = new Presentation(dataDir + "AccessingOLEObjectFrame.pptx");
```

هنا، `dataDir` ينبغي استبداله بالمسار الذي يوجد به ملف العرض التقديمي الخاص بك.

##### الوصول إلى الشريحة الأولى

```java
ISlide sld = pres.getSlides().get_Item(0);
```

يصل هذا الكود إلى الشريحة الأولى في العرض التقديمي. يمكنك التنقل بين الشرائح بالتكرار. `pres.getSlides()` إذا لزم الأمر.

### 2. إرسال إطار كائن OLE والوصول إليه

#### ملخص
للتفاعل مع الكائنات المضمنة، نحتاج إلى تحويل أشكال الشرائح إلى `OleObjectFrame`.

#### خطوات التنفيذ

##### الوصول إلى الشكل الأول على الشريحة

```java
OleObjectFrame oleObjectFrame = (OleObjectFrame) sld.getShapes().get_Item(0);
```

تأكد من أن الشكل هو بالفعل كائن OLE قبل الصب، حيث أن الصب غير الصحيح قد يؤدي إلى أخطاء وقت التشغيل.

### 3. استخراج بيانات كائن OLE المضمن وحفظها

#### ملخص
يتيح لك استخراج البيانات المضمنة من كائنات OLE التعامل معها أو حفظها بشكل منفصل.

#### خطوات التنفيذ

##### استخراج بيانات الملف المضمن

```java
byte[] data = oleObjectFrame.getEmbeddedData().getEmbeddedFileData();
String fileExtension = oleObjectFrame.getEmbeddedData().getEmbeddedFileExtension();
```

هنا، `data` يحتوي على المحتوى الثنائي للكائن المضمن، و `fileExtension` يساعد في حفظه بالتنسيق الصحيح.

##### حفظ البيانات المستخرجة في ملف

```java
String outputDir = "YOUR_OUTPUT_DIRECTORY/";
String extractedPath = outputDir + "excelFromOLE_out" + fileExtension;

try (FileOutputStream fstr = new FileOutputStream(extractedPath)) {
    fstr.write(data, 0, data.length);
}
```

يكتب هذا الكود بيانات الكائن المضمن إلى مسار محدد.

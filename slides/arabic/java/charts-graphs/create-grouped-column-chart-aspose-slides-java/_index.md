# إنشاء مخططات أعمدة مجمعة في PowerPoint باستخدام Aspose.Slides لـ Java

- قم بإدراج سلسلة البيانات في الرسم البياني الخاص بك بشكل فعال.

## إعداد Aspose.Slides لـ Java

لدمج مكتبة Aspose.Slides في مشروع Java الخاص بك، اتبع الخطوات التالية باستخدام Maven أو Gradle:

**مافن**

```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

**جرادل**

```gradle
implementation group: 'com.aspose', name: 'aspose-slides', version: '25.4', classifier: 'jdk16'
```

## دليل التنفيذ

سنقوم بتقسيم العملية إلى خطوات منطقية، مع التركيز على الميزات المحددة لـ Aspose.Slides.

### تهيئة العرض التقديمي

ابدأ بإنشاء مثيل لـ `Presentation` فصل:

```java
import com.aspose.slides.*;

// الميزة: تهيئة العرض التقديمي
Presentation pres = new Presentation();
ISlide slide = pres.getSlides().get_Item(0);
```

هنا، نبدأ عرضًا تقديميًا جديدًا ونختار الشريحة الأولى. تُستخدم هذه الشريحة كلوحة لإضافة المخططات.

### إضافة مخطط إلى الشريحة

بعد ذلك، أضف مخططًا عموديًا مجمعًا إلى الشريحة المحددة:

```java
// الميزة: إضافة مخطط إلى الشريحة
IChart ch = pres.getSlides().get_Item(0).getShapes().addChart(
    ChartType.ClusteredColumn, 100, 100, 600, 450);
ch.getChartData().getSeries().clear();
ch.getChartData().getCategories().clear();
```

هذه القطعة تنشئ مخططًا من النوع `ClusteredColumn` بأبعاد محددة، ويضعها على الشريحة. كما يمسح أي سلسلة أو فئات موجودة للبدء من جديد.

### إعداد مصنف بيانات الرسم البياني

لإدارة بيانات الرسم البياني الخاص بك، قم بإعداد مصنف:

```java
// الميزة: إعداد مصنف بيانات الرسم البياني
IChartDataWorkbook fact = ch.getChartData().getChartDataWorkbook();
fact.clear(0);
int defaultWorksheetIndex = 0;
```

ال `IChartDataWorkbook` يعمل الكائن كحاوية بيانات لمخططك، مما يسمح لك بمعالجة نقاط البيانات بشكل فعال.

### إضافة فئات مع مستويات التجميع

يُساعد تجميع الفئات على تنظيم البيانات بشكل هادف. إليك الطريقة:

```java
// الميزة: إضافة فئات مع مستويات التجميع
IChartCategory category = ch.getChartData().getCategories().add(
    fact.getCell(0, "c2", "A"));
category.getGroupingLevels().setGroupingItem(1, "Group1");

category = ch.getChartData().getCategories().add(fact.getCell(0, "c3", "B"));
// كرر ذلك للفئات الأخرى
```

تتم إضافة كل فئة بمستوى تجميع محدد. يتيح لك هذا تحديد مجموعات منطقية داخل مخططك.

### إضافة سلسلة بيانات إلى الرسم البياني

لتصور البيانات، أضف سلسلة إلى الرسم البياني:

```java
// الميزة: إضافة سلسلة بيانات إلى الرسم البياني
IChartSeries series = ch.getChartData().getSeries().add(
    fact.getCell(0, "D1", "Series 1"), ChartType.ClusteredColumn);
series.getDataPoints().addDataPointForBarSeries(fact.getCell(defaultWorksheetIndex, "D2", 10));
// استمرار إضافة نقاط البيانات
```

ال `IChartSeries` يتم استخدام الكائن لإضافة سلسلة من نقاط البيانات، والتي تمثل البيانات الفعلية في الرسم البياني الخاص بك.

### حفظ العرض التقديمي مع الرسم البياني

وأخيرًا، احفظ العرض التقديمي الخاص بك:

```java
// الميزة: حفظ العرض التقديمي مع الرسم البياني
pres.save("YOUR_OUTPUT_DIRECTORY/AsposeChart_out.pptx", SaveFormat.Pptx);
```

تكتب هذه الخطوة جميع التغييرات إلى ملف PPTX في الدليل المحدد.

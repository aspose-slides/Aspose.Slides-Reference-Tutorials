**ما سوف تتعلمه:**
- كيفية تعيين تنسيق رقمي محدد مسبقًا لنقاط بيانات الرسم البياني
- إنشاء أدلة لتخزين المستندات في Java
- إعداد Aspose.Slides لـ Java باستخدام Maven أو Gradle

دعونا نلقي نظرة على المتطلبات الأساسية قبل البدء!

قبل أن نبدأ، تأكد من أن لديك ما يلي:

   - تم تثبيت JDK 16 أو إصدار أحدث على جهازك.

3. **المتطلبات المعرفية:**

## إعداد Aspose.Slides لـ Java

لاستخدام Aspose.Slides في مشروعك، يمكنك إضافته عبر Maven أو Gradle:

**اعتماد Maven:**

```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

**تنفيذ Gradle:**

```gradle
implementation group: 'com.aspose', name: 'aspose-slides', version: '25.4', classifier: 'jdk16'
```

بدلاً من ذلك، يمكنك تنزيل الإصدار الأحدث مباشرةً من [إصدارات Aspose.Slides لـ Java](https://releases.aspose.com/slides/java/).

## دليل التنفيذ

### تعيين تنسيقات الأرقام في نقاط بيانات الرسم البياني

#### ملخص
تتيح لك هذه الميزة تخصيص كيفية ظهور البيانات الرقمية في سلسلة المخططات، مما يؤدي إلى تحسين قابلية القراءة باستخدام التنسيقات المحددة مسبقًا مثل النسب المئوية أو العملة.

**الخطوة 1: تهيئة العرض التقديمي**

```java
// استيراد فئات Aspose.Slides الضرورية
import com.aspose.slides.*;

public class SetNumberFormatInChartDataPoints {
    public static void main(String[] args) {
        // إنشاء كائن عرض تقديمي جديد
        Presentation pres = new Presentation();
```

- **لماذا:** تهيئة `Presentation` يعد هذا الكائن أمرًا بالغ الأهمية لأنه بمثابة الحاوية للشرائح والمخططات البيانية الخاصة بك.

**الخطوة 2: إضافة مخطط إلى الشريحة**

```java
        try {
            // الوصول إلى الشريحة الأولى من العرض التقديمي
            ISlide slide = pres.getSlides().get_Item(0);

            // إضافة مخطط عمودي مجمع إلى الشريحة
            IChart chart = slide.getShapes().addChart(ChartType.ClusteredColumn, 50, 50, 500, 400);
```

- **لماذا:** تساعد إضافة مخطط عند إحداثيات محددة على وضعه بدقة حيث تحتاج إليه في العرض التقديمي الخاص بك.

**الخطوة 3: الوصول إلى بيانات السلسلة وتنسيقها**

```java
            // احصل على مجموعة تسلسلية من بيانات الرسم البياني
            IChartSeriesCollection series = chart.getChartData().getSeries();

            // قم بالتكرار خلال كل سلسلة لتنسيق نقاط البيانات
            for (IChartSeries ser : series) {
                for (IChartDataPoint cell : ser.getDataPoints()) {
                    // تعيين تنسيق رقمي محدد مسبقًا لقيمة البيانات
                    cell.getValue().getAsCell().setPresetNumberFormat((byte) 10); // التنسيق: 0.00%
                }
            }
```

- **لماذا:** يضمن التكرار عبر السلسلة وتنسيق كل نقطة بيانات عرضًا متسقًا للقيم الرقمية.

**الخطوة 4: احفظ العرض التقديمي الخاص بك**

```java
            // احفظ العرض التقديمي المحدث مع المخططات المنسقة
            pres.save("YOUR_OUTPUT_DIRECTORY/PresetNumberFormat_out.pptx", SaveFormat.Pptx);
        } finally {
            if (pres != null) pres.dispose();
        }
    }
}
```

- **لماذا:** يؤدي الحفظ والتخلص من الموارد بشكل صحيح إلى منع تسرب الذاكرة ويضمن حفظ عملك.

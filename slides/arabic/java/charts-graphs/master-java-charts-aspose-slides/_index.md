# إتقان إنشاء المخططات وإدارتها في عروض Java التقديمية باستخدام Aspose.Slides

**مقدمة**

يُعد إنشاء عروض تقديمية ديناميكية تُوصل البيانات بفعالية تحديًا شائعًا يواجهه العديد من المطورين. سواء كنت تُعدّ تقارير أعمال أو أوراقًا أكاديمية أو مواد تسويقية، فإن دمج المخططات البيانية في شرائحك يُمكن أن يُحوّل النص العادي إلى صور جذابة. في هذا البرنامج التعليمي، سنستكشف كيفية تسخير قوة Aspose.Slides لجافا لإنشاء وإدارة المخططات البيانية في العروض التقديمية بكفاءة. باستخدام Aspose.Slides، يُمكنك أتمتة إنشاء المخططات البيانية، وتخصيص مدخلات البيانات، وتحسين أداء العرض التقديمي بسلاسة.

**ما سوف تتعلمه:**
- كيفية إعداد Aspose.Slides لـ Java
- إنشاء عرض تقديمي فارغ وإضافة مخطط
- إضافة بيانات الفئات والسلاسل إلى المخططات البيانية
- تبديل الصفوف والأعمدة في بيانات الرسم البياني
- حفظ العروض التقديمية باستخدام تكوينات مخصصة

بهذه المهارات، ستتمكن من تحسين عروضك التقديمية بشكل ملحوظ. لنستعرض المتطلبات الأساسية قبل البدء.

## إعداد Aspose.Slides لـ Java

للبدء في استخدام Aspose.Slides، يتعين عليك تضمينه في تبعيات مشروعك.

**مافن**
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

**جرادل**
```gradle
implementation group: 'com.aspose', name: 'aspose-slides', version: '25.4', classifier: 'jdk16'
```

بالنسبة لأولئك الذين يفضلون التنزيلات اليدوية، يمكنك الحصول على الإصدار الأحدث من [إصدارات Aspose.Slides لـ Java](https://releases.aspose.com/slides/java/).

## دليل التنفيذ

الآن بعد أن أصبحت بيئتنا جاهزة، فلننتقل إلى التنفيذ.

### إنشاء وتكوين العرض التقديمي

#### ملخص
الخطوة الأولى في إدارة المخططات البيانية هي إنشاء عرض تقديمي فارغ. سيرشدك هذا القسم خلال إعداد إطار العرض التقديمي الأولي باستخدام Aspose.Slides لجافا.

**الخطوة 1: تهيئة عرض تقديمي جديد**
```java
Presentation pres = new Presentation();
```

**الخطوة 2: إضافة مخطط إلى الشريحة**
هنا، نضيف مخططًا عموديًا مجمعًا عند الإحداثيات (100، 100) بأبعاد 400 × 300 بكسل.
```java
try {
    IChart chart = pres.getSlides().get_Item(0).getShapes().addChart(
        ChartType.ClusteredColumn, 100, 100, 400, 300
    );
} finally {
    if (pres != null) pres.dispose();
}
```
*ال `IChart` تتيح لك الواجهة التحكم في خصائص الرسم البياني وبياناته.*

### إضافة البيانات إلى الرسم البياني

#### ملخص
بعد إنشاء هيكل مخطط أساسي، من الضروري تزويده ببيانات مفيدة. يتناول هذا القسم إضافة الفئات والسلاسل إلى مخططك.

**الخطوة 1: الوصول إلى الفئات والسلاسل**
```java
IChart chart = new Presentation().getSlides().get_Item(0).getShapes()
    .addChart(ChartType.ClusteredColumn, 100, 100, 400, 300);

try {
    IChartDataCell[] categoriesCells = new IChartDataCell[chart.getChartData().getCategories().size()];
    for (int i = 0; i < chart.getChartData().getCategories().size(); i++) {
        categoriesCells[i] = chart.getChartData().getCategories().get_Item(i).getAsCell();
    }

    IChartDataCell[] seriesCells = new IChartDataCell[chart.getChartData().getSeries().size()];
    for (int i = 0; i < chart.getChartData().getSeries().size(); i++) {
        seriesCells[i] = chart.getChartData().getSeries().get_Item(i).getName().getAsCells().get_Item(0);
    }
} finally {
    if (pres != null) pres.dispose();
}
```
*هنا، `IChartDataCell` يمثل كل نقطة بيانات في الرسم البياني.*

### تبديل الصفوف والأعمدة في بيانات الرسم البياني

#### ملخص
يُمكن أن يُساعد تبديل الصفوف والأعمدة في إعادة تنظيم عرض بياناتك بشكل أكثر وضوحًا. لنرَ كيفية تطبيق هذه الميزة.

**الخطوة 1: تنفيذ التبديل بين الصفوف والأعمدة**
```java
try {
    chart.getChartData().switchRowColumn();
} finally {
    if (pres != null) pres.dispose();
}
```
*ال `switchRowColumn` تغير الطريقة اتجاه بياناتك.*

### حفظ العرض التقديمي

#### ملخص
بمجرد تكوين العرض التقديمي الخاص بك، يعد حفظه بالتنسيق المطلوب أمرًا ضروريًا.

**الخطوة 1: احفظ العرض التقديمي الخاص بك**
```java
try {
    pres.save("YOUR_OUTPUT_DIRECTORY/SwitchChartRowColumns_out.pptx", SaveFormat.Pptx);
} finally {
    if (pres != null) pres.dispose();
}
```
*حدد دليل الإخراج وتنسيق الملف للحفظ.*

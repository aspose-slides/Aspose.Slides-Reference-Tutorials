## دليل التنفيذ
الآن بعد أن قمت بإعداد Aspose.Slides، دعنا ننتقل إلى تخصيص خصائص نص الأسطورة.

### الوصول إلى خصائص نص التسمية التوضيحية وتعديلها

#### ملخص
يركز هذا القسم على كيفية تخصيص خصائص الخط لإدخالات الأسطورة الفردية في المخططات البيانية الخاصة بك.

#### إضافة مخطط إلى العرض التقديمي الخاص بك
1. **تحميل العرض التقديمي:**
   ```java
   Presentation pres = new Presentation(dataDir + "/test.pptx");
   ```

2. **إضافة مخطط عمودي مجمع:**
   ```java
   IChart chart = pres.getSlides().get_Item(0).getShapes().addChart(
       ChartType.ClusteredColumn, 50, 50, 600, 400);
   ```

#### تخصيص خصائص الخط
3. **تنسيق نص إدخال أسطورة الوصول:**
   ```java
   IChartTextFormat tf = chart.getLegend().getEntries().get_Item(1).getTextFormat();
   ```

4. **تعيين الأنماط العريضة والمائلة مع ارتفاع محدد:**
   ```java
   tf.getPortionFormat().setFontBold(NullableBool.True);
   tf.getPortionFormat().setFontHeight(20);
   tf.getPortionFormat().setFontItalic(NullableBool.True);
   ```

5. **تغيير نوع التعبئة إلى لون ثابت لتحسين الرؤية:**
   ```java
   tf.getPortionFormat().getFillFormat().setFillType(FillType.Solid);
   tf.getPortionFormat().getFillFormat().getSolidFillColor().setColor(Color.BLUE);
   ```

#### حفظ العرض التقديمي
6. **حفظ التغييرات:**
   ```java
   pres.save(outputDir + "/output.pptx", SaveFormat.Pptx);
   ```

1. **كيف يمكنني تغيير لون نص إدخال الأسطورة؟**
   يستخدم `getFillFormat().setFillType(FillType.Solid)` وضبط اللون المطلوب باستخدام `setColor(Color.YOUR_COLOR)`.

2. **هل يمكنني تطبيق هذه التغييرات على كافة الأساطير في العرض التقديمي؟**
   نعم، قم بالتكرار خلال أساطير كل مخطط باستخدام الحلقات.

   يمكن برمجة تعديلات الخط عن طريق حساب أبعاد النص قبل الإعداد `setFontHeight()`.

4. **ماذا لو واجهت مشاكل مع فهرسة إدخال الأسطورة؟**
   تأكد من منطق الكود الخاص بك للوصول إلى إدخالات الأسطورة وتأكد من أن الفهرس يتطابق مع تكوين الرسم البياني الخاص بك.

# إتقان Aspose.Slides Java: إضافة المخططات والصيغ إلى عروض PowerPoint التقديمية

## مقدمة

يُعد إنشاء عروض PowerPoint جذابة أمرًا بالغ الأهمية لعرض البيانات المعقدة بفعالية. باستخدام Aspose.Slides لجافا، يمكنك أتمتة إنشاء المخططات والصيغ الديناميكية بسلاسة، مما يعزز تأثير عرضك التقديمي. يرشدك هذا البرنامج التعليمي خلال إنشاء عرض تقديمي جديد في PowerPoint، وإضافة مخطط عمودي مجمع، ومعالجة بيانات المخطط باستخدام الصيغ، وحفظ عملك باستخدام Aspose.Slides.

**ما سوف تتعلمه:**
- إعداد Aspose.Slides لـ Java
- إنشاء عرض تقديمي في PowerPoint وإدراج المخططات البيانية
- الوصول إلى بيانات الرسم البياني وتعديلها باستخدام الصيغ
- حساب الصيغ وحفظ العرض التقديمي الخاص بك

دعونا نبدأ بمراجعة المتطلبات الأساسية!

## إعداد Aspose.Slides لـ Java

### تبعية Maven
لتضمين Aspose.Slides في مشروعك باستخدام Maven، أضف التبعية التالية إلى مشروعك `pom.xml`:

```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

### اعتماد Gradle
إذا كنت تستخدم Gradle، قم بتضمين هذا في `build.gradle`:

```gradle
implementation group: 'com.aspose', name: 'aspose-slides', version: '25.4', classifier: 'jdk16'
```

### التحميل المباشر
بدلاً من ذلك، قم بتنزيل أحدث إصدار من Aspose.Slides لـ Java من [إصدارات Aspose](https://releases.aspose.com/slides/java/).

## دليل التنفيذ

تم تقسيم هذا القسم إلى خطوات لمساعدتك على فهم كل جزء بوضوح.

### إنشاء عرض تقديمي وإضافة مخطط بياني

#### ملخص
تعرف على كيفية إنشاء شريحة PowerPoint وإضافة مخطط عمودي مجمع باستخدام Aspose.Slides لـ Java.

##### الخطوة 1: تهيئة العرض التقديمي
ابدأ بإنشاء حساب جديد `Presentation` هدف:

```java
Presentation presentation = new Presentation();
```

##### الخطوة 2: الوصول إلى الشريحة الأولى
استرداد الشريحة الأولى حيث ستضع الرسم البياني الخاص بك:

```java
ISlide slide = presentation.getSlides().get_Item(0);
```

##### الخطوة 3: إضافة مخطط عمودي مجمع
أضف الرسم البياني إلى الشريحة عند الإحداثيات والأبعاد المحددة:

```java
IChart chart = slide.getShapes().addChart(
    ChartType.ClusteredColumn, 
    150, 150, 
    500, 300
);
```
**المعلمات موضحة:**
- `ChartType`:يحدد نوع الرسم البياني.
- الإحداثيات (x، y): الموضع على الشريحة.
- العرض والارتفاع: أبعاد الرسم البياني.

### العمل مع مصنف بيانات الرسم البياني

#### ملخص
يمكنك التعامل مع بيانات الرسم البياني بشكل مباشر عن طريق تعيين الصيغ للخلايا داخل مصنف الرسم البياني.

##### الخطوة 1: الوصول إلى مصنف بيانات الرسم البياني
استرداد المصنف المرتبط بالرسم البياني الخاص بك:

```java
IChartDataWorkbook workbook = chart.getChartData().getChartDataWorkbook();
```

##### الخطوة 2: إعداد الصيغ
قم بتعيين الصيغ لإجراء الحسابات بشكل ديناميكي في بيانات الرسم البياني الخاص بك:

**الصيغة في الخلية B2**: 
```java
IChartDataCell cell1 = workbook.getCell(0, "B2");
cell1.setFormula("1 + SUM(F2:H5)");
```

**صيغة نمط R1C1 في الخلية C2**: 
```java
IChartDataCell cell2 = workbook.getCell(0, "C2");
cell2.setR1C1Formula("MAX(R2C6:R5C8) / 3");
```
تسمح لك هذه الصيغ بالتحديثات والحسابات الديناميكية داخل الرسم البياني الخاص بك.

### حساب الصيغ وحفظ العرض التقديمي

#### ملخص
تأكد من حساب كافة الصيغ قبل حفظ العرض التقديمي الخاص بك لتعكس التغييرات بدقة.

##### الخطوة 1: حساب جميع الصيغ
استدعاء طريقة الحساب في المصنف الخاص بك:

```java
workbook.calculateFormulas();
```

##### الخطوة 2: احفظ العرض التقديمي الخاص بك
احفظ عملك باسم ملف وتنسيق محددين:

```java
String outpptxFile = "YOUR_OUTPUT_DIRECTORY" + File.separator + "ChartDataCell_Formulas_out.pptx";
presentation.save(outpptxFile, SaveFormat.Pptx);
```
تأكد من الاستبدال `YOUR_OUTPUT_DIRECTORY` مع المسار الفعلي الذي تريد حفظ ملفك فيه.

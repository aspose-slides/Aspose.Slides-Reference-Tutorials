# كيفية إضافة مخطط دائري إلى عرض تقديمي باستخدام Aspose.Slides Java

- كيفية تهيئة كائن العرض التقديمي في Java.

دعونا نتعمق في كيفية الاستفادة من Aspose.Slides Java لتحسين العروض التقديمية الخاصة بك باستخدام المخططات الديناميكية!

## إعداد Aspose.Slides لـ Java
أولاً، ستحتاج إلى تضمين Aspose.Slides في مشروعك. يمكنك القيام بذلك عبر Maven أو Gradle:

**مافن:**
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

**جرادل:**
```gradle
implementation group: 'com.aspose', name: 'aspose-slides', version: '25.4', classifier: 'jdk16'
```

بدلا من ذلك، يمكنك [تنزيل أحدث إصدار](https://releases.aspose.com/slides/java/) مباشرة من موقع Aspose.

## دليل التنفيذ
دعنا نقسم التنفيذ إلى ميزتين رئيسيتين: إضافة مخطط دائري إلى عرض تقديمي والوصول إلى بيانات المخطط.

### الميزة 1: إنشاء عرض تقديمي وإضافة مخطط
#### ملخص
يوضح هذا القسم كيفية تهيئة كائن عرض تقديمي جديد وإضافة مخطط دائري إلى الشريحة الأولى.

#### دليل خطوة بخطوة:
**الخطوة 1: تهيئة كائن عرض تقديمي جديد**
```java
Presentation pres = new Presentation();
```
*هنا، نقوم بإنشاء مثيل لـ `Presentation`، والذي يعمل بمثابة حاوية المستندات الرئيسية لدينا.*

**الخطوة 2: إضافة مخطط دائري**
```java
IChart chart = pres.getSlides().get_Item(0).getShapes().addChart(
    ChartType.Pie,
    50,
    50,
    400,
    500
);
```
*نضيف مخططًا دائريًا إلى الشريحة الأولى عند الإحداثيات المحددة (50، 50) بأبعاد عرض 400 وارتفاع 500. `ChartType.Pie` يحدد نوع الرسم البياني.*

**الخطوة 3: التخلص من الموارد**
```java
if (pres != null) pres.dispose();
```
*من المهم تحرير الموارد عن طريق التخلص من كائن العرض بمجرد اكتمال العمليات.*

### الميزة 2: الوصول إلى مصنف بيانات الرسم البياني وأوراق العمل
#### ملخص
تعرف على كيفية الوصول إلى مصنف البيانات الأساسي المرتبط بمخططك والتكرار خلال أوراق العمل الخاصة به.

#### دليل خطوة بخطوة:
**الخطوة 1: تهيئة كائن عرض تقديمي جديد**
*أعد استخدام خطوة التهيئة من الميزة السابقة.*

**الخطوة 2: إضافة مخطط دائري**
*كما في السابق، أضف مخططًا دائريًا لبدء العمل مع مصنفات البيانات.*

**الخطوة 3: الحصول على مصنف بيانات الرسم البياني**
```java
IChartDataWorkbook workbook = chart.getChartData().getChartDataWorkbook();
```
*هذا يسترجع `IChartDataWorkbook` الكائن المرتبط بمخططنا، مما يسمح بالوصول إلى بياناته.*

**الخطوة 4: التكرار خلال أوراق العمل**
```java
for (int i = 0; i < workbook.getWorksheets().size(); i++) {
    System.out.println(workbook.getWorksheets().get_Item(i).getName());
}
```
*هنا، نقوم بالمرور على كل ورقة عمل في المصنف وطباعة اسمها.*

**الخطوة 5: التخلص من الموارد**
*تخلص من كائن العرض التقديمي كما هو موضح سابقًا لتحرير الموارد.*

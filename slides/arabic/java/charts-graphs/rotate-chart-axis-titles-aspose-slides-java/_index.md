## دليل التنفيذ
في هذا القسم، سنرشدك خلال إعداد مخطط بعنوان محور مدور خطوة بخطوة.
### إضافة مخطط عمودي مجمع
**ملخص**:لنبدأ بإضافة مخطط عمودي مجمع إلى الشريحة الخاصة بك.
#### الخطوة 1: إنشاء العرض التقديمي
تهيئة مثيل عرض تقديمي جديد:
```java
Presentation pres = new Presentation();
```
يقوم هذا السطر من التعليمات البرمجية بإعداد ملف PowerPoint فارغًا للتعامل معه.
#### الخطوة 2: إضافة مخطط عمودي مجمع
أضف مخططًا إلى الشريحة الأولى في الموضع (50، 50) بأبعاد (450، 300):
```java
IChart chart = pres.getSlides().get_Item(0).getShapes().addChart(ChartType.ClusteredColumn, 50, 50, 450, 300);
```
هنا، `ChartType.ClusteredColumn` يُحدد نوع الرسم البياني. يمكنك تغيير هذا إلى أنواع أخرى مثل `Pie`، `Bar`، وما إلى ذلك، اعتمادًا على احتياجاتك.
#### الخطوة 3: تمكين وتدوير عنوان المحور الرأسي
بعد ذلك، قم بتمكين عنوان المحور الرأسي وتعيين زاوية دورانه:
```java
// تمكين عنوان المحور الرأسي.
chart.getAxes().getVerticalAxis().setTitle(true);

// ضبط زاوية الدوران إلى 90 درجة.
chart.getAxes().getVerticalAxis().getTitle().getTextFormat().getTextBlockFormat().setRotationAngle(90);
```
ال `setRotationAngle` تتيح لك هذه الطريقة ضبط اتجاه النص، مما يعزز قابلية القراءة عندما تكون المساحة محدودة.
#### الخطوة 4: حفظ العرض التقديمي
وأخيرًا، احفظ التغييرات:
```java
pres.save("YOUR_DOCUMENT_DIRECTORY/test.pptx", SaveFormat.Pptx);
```
استبدل "YOUR_DOCUMENT_DIRECTORY" بالمسار الفعلي الذي تريد تخزين العرض التقديمي الخاص بك فيه.

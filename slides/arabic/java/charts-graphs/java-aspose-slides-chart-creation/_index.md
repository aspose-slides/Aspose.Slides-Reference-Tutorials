# إتقان إنشاء المخططات في Java باستخدام Aspose.Slides

## المخططات والرسوم البيانية

تعلّم إنشاء مخططات بيانية في عروض جافا التقديمية باستخدام Aspose.Slides. سيرشدك هذا الدليل الشامل خلال خطوات تهيئة عرض تقديمي، وإضافة المخططات البيانية وتخصيصها، وحفظ عملك - كل ذلك باستخدام جافا.

### مقدمة

غالبًا ما يتطلب إنشاء عروض تقديمية جذابة أكثر من مجرد نص؛ فتصور البيانات أساسي لعرض المعلومات بفعالية. مع Aspose.Slides لجافا، يمكنك دمج المخططات الديناميكية بسهولة في شرائحك، مما يعزز من جاذبية عرض البيانات ووضوحه. سيزودك هذا البرنامج التعليمي بالمهارات اللازمة لـ:

- تهيئة عرض تقديمي جديد في Java
- إضافة وتخصيص مخططات الأعمدة المجمعة
- تعيين خصائص الخط لعناصر نص الرسم البياني
- حفظ العروض التقديمية بتنسيق PPTX

بنهاية هذا الدليل، ستتمكن من استخدام Aspose.Slides لجافا لإنشاء عروض تقديمية احترافية مصممة خصيصًا مع تصورات بيانات مخصصة. لنبدأ بشرح المتطلبات الأساسية.

### المتطلبات الأساسية

قبل أن نبدأ، تأكد من أن لديك ما يلي:

- **مجموعة تطوير Java (JDK):** الإصدار 8 أو أعلى.
- **Aspose.Slides لـ Java:** سوف نستخدم الإصدار 25.4 من هذه المكتبة.
- **Maven أو Gradle:** لإدارة التبعيات في إعداد مشروعك.

بالإضافة إلى ذلك، فإن الفهم الأساسي لبرمجة Java والتعرف على برامج العرض التقديمي مثل Microsoft PowerPoint سيكون مفيدًا.

### إعداد Aspose.Slides لـ Java

لاستخدام Aspose.Slides في Java، عليك أولًا تضمينه كاعتمادية في مشروعك. إليك كيفية إعداده باستخدام Maven أو Gradle:

#### مافن

أضف التبعية التالية إلى ملفك `pom.xml` ملف:

```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```

#### جرادل

قم بتضمين هذا السطر في `build.gradle` ملف:

```gradle
implementation group: 'com.aspose', name: 'aspose-slides', version: '25.4', classifier: 'jdk16'
```

وبدلاً من ذلك، يمكنك تنزيل المكتبة مباشرةً من [إصدارات Aspose.Slides لـ Java](https://releases.aspose.com/slides/java/).

### الترخيص

لاستكشاف جميع الميزات دون قيود، فكّر في الحصول على ترخيص مؤقت أو شراء ترخيص جديد. يمكنك البدء بفترة تجريبية مجانية لتقييم إمكانيات Aspose.Slides.

### دليل التنفيذ

سنقوم بتقسيم التنفيذ إلى أقسام رئيسية: تهيئة العرض التقديمي الخاص بك، وإنشاء المخططات البيانية، وتعيين خصائص الخط، وحفظ عملك.

#### الميزة 1: تهيئة العرض التقديمي وإنشاء المخطط

**ملخص:**
يوضح هذا القسم كيفية بدء عرض تقديمي جديد وإضافة مخطط عمودي مجمع.

##### الخطوة 1: تهيئة عرض تقديمي جديد

```java
import com.aspose.slides.Presentation;
import com.aspose.slides.SaveFormat;

// إنشاء كائن عرض تقديمي جديد
Presentation pres = new Presentation();
```

هنا، نقوم بإنشاء مثيل لـ `Presentation` هذا الفصل هو بمثابة لوحة لإضافة الشرائح والمخططات.

##### الخطوة 2: إضافة مخطط عمودي مجمع

```java
import com.aspose.slides.ChartType;
import com.aspose.slides.IChart;

try {
    // أضف مخططًا عموديًا مجمعًا إلى الشريحة الأولى في الموضع (100، 100) بعرض 500 وارتفاع 400.
    IChart chart = pres.getSlides().get_Item(0).getShapes().addChart(
        ChartType.ClusteredColumn, 100, 100, 500, 400
    );
} finally {
    // تأكد من تحرير الموارد عن طريق التخلص من كائن العرض التقديمي.
    if (pres != null) pres.dispose();
}
```

نضيف مخططًا عموديًا مجمعًا إلى الشريحة الأولى. `addChart` تحدد الطريقة نوع الرسم البياني والأبعاد.

#### الميزة 2: تعيين خصائص الخط لنص الرسم البياني

**ملخص:**
قم بتخصيص عناصر النص داخل الرسم البياني الخاص بك عن طريق ضبط خصائص الخط.

##### الخطوة 1: تعيين ارتفاع الخط

```java
try {
    // قم بتعيين ارتفاع الخط إلى 20 نقطة لجميع أجزاء تنسيق نص الرسم البياني.
    chart.getTextFormat().getPortionFormat().setFontHeight(20);
} finally {
    if (pres != null) pres.dispose();
}
```

تحدد هذه القطعة حجم خط موحدًا لجميع عناصر النص في الرسم البياني، مما يعزز قابلية القراءة.

##### الخطوة 2: تمكين تسميات البيانات

```java
try {
    // تمكين عرض القيم على تسميات البيانات للسلسلة الأولى في الرسم البياني.
    chart.getChartData().getSeries().get_Item(0).getLabels()
        .getDefaultDataLabelFormat().setShowValue(true);
} finally {
    if (pres != null) pres.dispose();
}
```

من خلال تمكين تسميات البيانات، يمكنك توفير سياق فوري لجمهورك، مما يجعل المعلومات أكثر سهولة في الوصول إليها.

#### الميزة 3: حفظ العرض التقديمي

**ملخص:**
قم بإنهاء عملك عن طريق حفظ العرض التقديمي بتنسيق PPTX.

```java
try {
    // قم بتحديد مسار ملف الإخراج باستخدام دليل العنصر النائب.
    String outputFile = "YOUR_OUTPUT_DIRECTORY/FontPropertiesForChart.pptx";

    // احفظ العرض التقديمي بتنسيق PPTX في الموقع المحدد.
    pres.save(outputFile, SaveFormat.Pptx);
} finally {
    if (pres != null) pres.dispose();
}
```

تأكد من استبدال `YOUR_OUTPUT_DIRECTORY` مع مسار الملف الفعلي الخاص بك لتخزين الإخراج.

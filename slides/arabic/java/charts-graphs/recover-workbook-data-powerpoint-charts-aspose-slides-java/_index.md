## إعداد Aspose.Slides لـ Java
لاستخدام الميزات القوية لـ Aspose.Slides، قم بدمجه في مشروعك على النحو التالي:

**إعداد Maven:**
```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-slides</artifactId>
    <version>25.4</version>
    <classifier>jdk16</classifier>
</dependency>
```
**إعداد Gradle:**

## دليل التنفيذ
سنقوم بتقسيم التنفيذ إلى قسمين رئيسيين: استرداد بيانات المصنف من ذاكرة التخزين المؤقت للرسم البياني وتكوين LoadOptions.

### استعادة المصنف من ذاكرة التخزين المؤقت للمخطط
#### ملخص
تتيح هذه الميزة إمكانية الوصول إلى بيانات المصنف المضمنة في المخططات داخل عروض PowerPoint واستعادتها، مما يضمن عدم فقدان البيانات أثناء عمليات التحويل أو التحرير.

#### التنفيذ خطوة بخطوة
##### تعيين LoadOptions للاسترداد
تكوين `LoadOptions` لتمكين استرداد المصنف:
```java
import com.aspose.slides.*;

String pptxFile = "YOUR_DOCUMENT_DIRECTORY/ExternalWB.pptx";
String outPptxFile = "YOUR_OUTPUT_DIRECTORY/ExternalWB_out.pptx";

// الخطوة 1: قم بتعيين LoadOptions لاسترداد المصنف من ذاكرة التخزين المؤقت للرسم البياني.
LoadOptions lo = new LoadOptions();
lo.getSpreadsheetOptions().setRecoverWorkbookFromChartCache(true);
```
هنا، `setRecoverWorkbookFromChartCache(true)` يعد هذا أمرًا بالغ الأهمية لأنه يوجه Aspose.Slides لاسترداد أي مصنفات مضمنة في المخططات البيانية.

##### تحميل العرض التقديمي مع الخيارات
قم بتحميل ملف PowerPoint الخاص بك باستخدام الخيارات التالية:
```java
// الخطوة 2: قم بتحميل العرض التقديمي باستخدام LoadOptions المحددة.
Presentation pres = new Presentation(pptxFile, lo);
```
تضمن هذه الخطوة إعداد جميع البيانات اللازمة للاسترداد.

##### الوصول إلى البيانات واسترجاعها
بعد ذلك، قم بالوصول إلى الرسم البياني واسترداد بيانات المصنف المرتبطة به:
```java
try {
    // الخطوة 3: قم بالوصول إلى الرسم البياني الأول في الشريحة الأولى.
    IChart chart = (IChart) pres.getSlides().get_Item(0).getShapes().get_Item(0);

    // الخطوة 4: استرداد مصنف البيانات المرتبط بالرسم البياني.
    IChartDataWorkbook wb = chart.getChartData().getChartDataWorkbook();
    
    // الخطوة 5: احفظ العرض التقديمي في ملف جديد.
    pres.save(outPptxFile, SaveFormat.Pptx);
} finally {
    if (pres != null) pres.dispose();
}
```
في هذه المقتطفة:
- نقوم بالوصول إلى الرسم البياني الأول ومصنف البيانات الخاص به.
- وأخيرًا، قمنا بحفظ العرض التقديمي المعدّل.

### تكوين LoadOptions
#### ملخص
تكوين `LoadOptions` يتيح لك التحكم بشكل فعال في كيفية إدارة المصنفات المضمنة أثناء عمليات التحميل.

#### شرح مفصل
```java
// الميزة: تكوين LoadOptions
import com.aspose.slides.*;

خيارات التحميل lo = new LoadOptions();
lo.getSpreadsheetOptions().setRecoverWorkbookFromChartCache(true);
```
- **LoadOptions**:إعداد التكوين لتحميل العرض التقديمي.
- **الحصول على خيارات جدول البيانات ()**:يوفر الوصول إلى الإعدادات المتعلقة بجداول البيانات المضمنة.
- **setRecoverWorkbookFromChartCache(true)**:يمكّن استرداد بيانات المصنف من ذاكرات التخزين المؤقت للمخططات.

## دليل التنفيذ

### إنشاء مخطط دائري
يتضمن إنشاء مخطط دائري تهيئة كائن عرض تقديمي وإضافته إلى الشريحة. لنشرح الخطوات بالتفصيل:

#### الخطوة 1: تهيئة العرض التقديمي
ابدأ بإنشاء مثيل لـ `Presentation` فصل.
```java
Presentation pres = new Presentation();
```

#### الخطوة 2: إضافة مخطط دائري
أضف مخططًا دائريًا إلى الشريحة الأولى بأبعاد وموضع محددين:
```java
pres.getSlides().get_Item(0).getShapes().addChart(ChartType.Pie, 50, 50, 600, 400);
```
*توضيح:* يضع هذا الكود مخطط الفطيرة عند الإحداثيات (50، 50) بعرض 600 وارتفاع 400.

#### الخطوة 3: احفظ العرض التقديمي الخاص بك
بمجرد إعداد الرسم البياني الخاص بك، احفظ العرض التقديمي:
```java
pres.save("YOUR_OUTPUT_DIRECTORY/output.pptx", SaveFormat.Pptx);
```
*توضيح:* ال `save` تكتب الطريقة تغييراتك إلى ملف في الدليل المحدد.

#### تنظيف الموارد
قم دائمًا بتحرير الموارد لمنع تسرب الذاكرة:
```java
if (pres != null) pres.dispose();
```

### إعداد انفجار نقطة البيانات واللون
يُحسّن تخصيص نقاط البيانات سهولة القراءة. لنُعدّل مظهر نقطة بيانات مُحددة:

#### الخطوة 1: فتح عرض تقديمي موجود
قم بتحميل ملف العرض التقديمي الخاص بك حيث تريد تخصيص الرسم البياني.
```java
Presentation pres = new Presentation("YOUR_DOCUMENT_DIRECTORY/test.pptx");
```

#### الخطوة 2: الوصول إلى نقاط البيانات وتعديلها
استرجاع نقطة بيانات من السلسلة الأولى:
```java
IChart chart = pres.getSlides().get_Item(0).getShapes().addChart(ChartType.Pie, 50, 50, 600, 400);
IChartDataPoint point = chart.getChartData().getSeries().get_Item(0).getDataPoints().get_Item(1);
```
*توضيح:* يصل هذا الكود إلى نقطة البيانات الثانية للتخصيص.

#### الخطوة 3: تطبيق الانفجار واللون
اضبط الانفجار لفصله بصريًا وتغيير لونه:
```java
point.setExplosion(30); // ضبط مسافة الانفجار
point.getFormat().getFill().setFillType(FillType.Solid);
point.getFormat().getFill().getSolidFillColor().setColor(Color.BLUE); // تغيير لون التعبئة
```
*توضيح:* ال `setExplosion` إن الطريقة تخلق إزاحة بصرية، في حين أن تعيين نوع التعبئة واللون يضفي طابعًا شخصيًا على المظهر.

#### تنظيف الموارد
التخلص من الموارد بعد التعديلات:
```java
if (pres != null) pres.dispose();
```

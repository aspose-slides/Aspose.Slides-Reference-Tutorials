## دليل التنفيذ
الآن بعد أن قمت بإعداد Aspose.Slides، دعنا ننفذ انتقالات التحويل في شريحة PowerPoint.

### نظرة عامة على انتقالات مورف
تتيح انتقالات مورف تحويلات سلسة بين العناصر على مختلف الشرائح. يمكن ضبطها للانتقال حسب العنصر أو الكلمة أو الحرف، مما يعزز سلاسة وجاذبية عرضك التقديمي.

#### الخطوة 1: تحميل العرض التقديمي الخاص بك
ابدأ بتحميل ملف PowerPoint الحالي لديك باستخدام مدير السياق لضمان إدارة الموارد بشكل صحيح:

```python
import aspose.slides as slides

# حدد مسار العرض التقديمي الخاص بك
presentation_path = "YOUR_DOCUMENT_DIRECTORY/welcome-to-powerpoint.pptx"

with slides.Presentation(presentation_path) as presentation:
    slide = presentation.slides[0]  # الوصول إلى الشريحة الأولى
```

#### الخطوة 2: تعيين نوع الانتقال إلى Morph
حدد أنك تريد انتقالًا شكليًا للشريحة المحددة:

```python
# تكوين نوع الانتقال
slide.slide_show_transition.type = slides.slideshow.TransitionType.MORPH
```

#### الخطوة 3: تحديد التحويل حسب الكلمة
لتكوين انتقال الشكل ليحدث حسب الكلمة، اضبط `morph_type` وفقاً لذلك:

```python
# تعيين انتقال الشكل حسب الكلمة
slide.slide_show_transition.value.morph_type = slides.slideshow.TransitionMorphType.BY_WORD
```

### حفظ العرض التقديمي الخاص بك
بعد تكوين انتقالاتك، احفظ العرض التقديمي في ملف جديد:

```python
output_path = "YOUR_OUTPUT_DIRECTORY/transition_MORPH_out.pptx"

with slides.Presentation(presentation_path) as presentation:
    slide = presentation.slides[0]
    slide.slide_show_transition.type = slides.slideshow.TransitionType.MORPH
    slide.slide_show_transition.value.morph_type = slides.slideshow.TransitionMorphType.BY_WORD

# حفظ التغييرات
presentation.save(output_path, slides.export.SaveFormat.PPTX)
```

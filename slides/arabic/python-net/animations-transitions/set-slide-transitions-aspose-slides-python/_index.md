```python
input_directory = 'YOUR_DOCUMENT_DIRECTORY/'
output_directory = 'YOUR_OUTPUT_DIRECTORY/'

with slides.Presentation(input_directory + "welcome-to-powerpoint.pptx") as presentation:
    # يمكنك الوصول إلى الشرائح وتعديلها هنا.
```

#### إعداد تأثيرات الانتقال
سنقوم بتعيين تأثير انتقالي على الشريحة الأولى من العرض التقديمي الخاص بك:

```python
# الوصول إلى الشريحة الأولى
slide = presentation.slides[0]

# تعيين نوع تأثير الانتقال
slide.slide_show_transition.type = slides.slideshow.TransitionType.CUT

# خصائص انتقالية إضافية (على سبيل المثال، من الأسود)
slide.slide_show_transition.value.from_black = True
```

#### توضيح:
- **نوع الانتقال**:يؤدي هذا إلى تعيين نوع معين من الرسوم المتحركة عند التنقل بين الشرائح. `CUT` يعني التبديل الفوري.
- **من الأسود**:خاصية خاصة لبدء الشريحة بشاشة سوداء.

## دليل التنفيذ

### تعيين العرض التقديمي على "للقراءة فقط" موصى به

**ملخص:**
يغطي هذا القسم كيفية جعل عرض تقديمي لبرنامج PowerPoint للقراءة فقط باستخدام الموصى به `Aspose.Slides` يشير هذا الإعداد إلى أنه لا ينبغي تحرير المستند، ولكنه لا يفرض ذلك بشكل صارم.

#### الخطوة 1: استيراد المكتبة
ابدأ باستيراد الوحدة اللازمة:

```python
import aspose.slides as slides
```

#### الخطوة 2: فتح عرض تقديمي أو إنشائه
يمكنك فتح عرض تقديمي موجود أو إنشاء عرض تقديمي جديد:

```python
with slides.Presentation() as pres:
    # الكود لتعديل العرض التقديمي يذهب هنا
```

#### الخطوة 3: تعيين خاصية موصى بها للقراءة فقط
اضبط `read_only_recommended` خاصية تشير إلى حالة القراءة فقط:

```python
pres.protection_manager.read_only_recommended = True
```

*لماذا هذا مهم؟*
تعمل هذه الخطوة على تحديد عرضك التقديمي باعتباره موصى به لوضع القراءة فقط، مما يساعد على منع التحرير غير المقصود.

#### الخطوة 4: حفظ العرض التقديمي
حفظ التغييرات في الدليل المحدد:

```python
pres.save("YOUR_OUTPUT_DIRECTORY/props_read_only_recommended_out.pptx", slides.export.SaveFormat.PPTX)
```

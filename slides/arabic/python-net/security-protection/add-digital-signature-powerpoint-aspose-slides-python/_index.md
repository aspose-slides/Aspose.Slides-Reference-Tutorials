## دليل التنفيذ

### إضافة توقيع رقمي إلى PowerPoint

تتضمن إضافة التوقيع الرقمي عدة خطوات:

#### الخطوة 1: تحميل أو إنشاء عرض تقديمي
ابدأ بفتح عرض تقديمي موجود أو إنشاء عرض تقديمي جديد باستخدام Aspose.Slides:

```python
import aspose.slides as slides

# فتح أو إنشاء عرض تقديمي
class SecurePPTWithSignature:
    def __init__(self):
        self.pres = None

    def load_or_create_presentation(self, path=None):
        if path:
            self.pres = slides.Presentation(path)
        else:
            self.pres = slides.Presentation()
```

هذا الكود يُهيئ ملف PowerPoint الذي ستعمل عليه. إذا لم يكن موجودًا، فسيتم إنشاء ملف جديد.

#### الخطوة 2: إنشاء كائن التوقيع الرقمي
لإضافة توقيع رقمي، قم أولاً بإنشاء مثيل لـ `DigitalSignature` باستخدام ملف الشهادة وكلمة المرور الخاصة بك:

```python
class SecurePPTWithSignature(SecurePPTWithSignature):
    def __init__(self, cert_path, cert_password):
        super().__init__()
        self.signature = slides.DigitalSignature(cert_path, cert_password)
```

هنا، `"YOUR_DOCUMENT_DIRECTORY/cert.pfx"` هو المسار إلى شهادتك الرقمية، و `"testpass1"` هي كلمة المرور المقابلة.

#### الخطوة 3: إضافة التعليقات (اختياري)
قد تساعد إضافة التعليقات في تحديد الهوية أو حفظ السجلات:

```python
class SecurePPTWithSignature(SecurePPTWithSignature):
    def add_comments_to_signature(self, comment):
        self.signature.comments = comment
```

هذه الخطوة اختيارية ولكن يوصى بها للحصول على توثيق أفضل.

#### الخطوة 4: إضافة التوقيع الرقمي إلى العرض التقديمي
دمج توقيعك الرقمي في كائن العرض التقديمي:

```python
class SecurePPTWithSignature(SecurePPTWithSignature):
    def add_signature_to_presentation(self):
        if self.pres:
            self.pres.digital_signatures.add(self.signature)
```

عن طريق الاتصال `add()`، فأنت تقوم بتأمين PowerPoint باستخدام الشهادة المقدمة.

#### الخطوة 5: حفظ العرض التقديمي الموقع
وأخيرًا، احفظ عرضك التقديمي بتنسيق PPTX، بما في ذلك التوقيع الرقمي:

```python
class SecurePPTWithSignature(SecurePPTWithSignature):
    def save_signed_presentation(self, output_path):
        if self.pres:
            self.pres.save(output_path, slides.export.SaveFormat.PPTX)
```

سيتم حفظ الملف في `"YOUR_OUTPUT_DIRECTORY"`تأكد من وجود هذا الدليل أو قم بتعديل المسار وفقًا لذلك.

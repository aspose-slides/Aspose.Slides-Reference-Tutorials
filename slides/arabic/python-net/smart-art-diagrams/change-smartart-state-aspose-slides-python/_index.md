## دليل التنفيذ

### إضافة وتعديل رسومات SmartArt

#### ملخص
في هذا القسم، سنتعلم كيفية إضافة رسم SmartArt إلى الشريحة الخاصة بك وتعديل خصائصه مثل عكس حالته.

#### التنفيذ خطوة بخطوة:

**1. إنشاء عرض تقديمي جديد:**

```python
with slides.Presentation() as presentation:
    # الوصول إلى الشريحة الأولى (الفهرس 0)
slide = presentation.slides[0]
```

تعمل هذه الخطوة على تهيئة كائن عرض تقديمي جديد وفتحه للتحرير باستخدام تقنيات إدارة الموارد.

**2. إضافة رسم SmartArt:**

```python
# أضف رسم SmartArt بأبعاد محددة ونوع تخطيط
smart = slide.shapes.add_smart_art(
    x=10, y=10, width=400, height=300,
    layout_type=slides.smartart.SmartArtLayoutType.BASIC_PROCESS
)
```

هنا، نضيف عملية SmartArt الأساسية عند الإحداثيات المحددة. `add_smart_art` تسمح الطريقة بالتحديد الدقيق للوضع وتكوين الحجم.

**3. تعديل حالة الانعكاس:**

```python
# تعيين رسم SmartArt ليكون معكوسًا
smart.is_reversed = True
```

يقوم هذا الخط بتغيير اتجاه SmartArt، مما يضيف تأثيرًا بصريًا ديناميكيًا.

**4. احفظ العرض التقديمي:**

```python
presentation.save("YOUR_OUTPUT_DIRECTORY/smart_art_change_state_out.pptx")
```

أخيرًا، احفظ عرضك التقديمي في المجلد المحدد. تأكد من استبدال `YOUR_OUTPUT_DIRECTORY` مع المسار الفعلي على نظامك.

## دليل التنفيذ

### نظرة عامة على الميزة: إنشاء مخطط تنظيمي

تتيح لك هذه الميزة إنشاء عرض تقديمي يحتوي على مخطط تنظيمي باستخدام تخطيط مخطط التنظيم المصور في Aspose.Slides.

#### الخطوة 1: تهيئة كائن العرض التقديمي

إنشاء جديد `Presentation` كائن ليكون بمثابة لوحة لإضافة الأشكال والمحتوى:

```python
import aspose.slides as slides

def create_organization_chart():
    with slides.Presentation() as pres:
        # سيتم إضافة خطوات أخرى هنا
```

#### الخطوة 2: إضافة شكل SmartArt إلى الشريحة

استخدم `PICTURE_ORGANIZATION_CHART` تخطيط لهيكلك التنظيمي:

```python
smart_art = pres.slides[0].shapes.add_smart_art(
    0,   # موضع x
    0,   # موضع y
    400, # عرض
    400, # ارتفاع
    slides.smartart.SmartArtLayoutType.PICTURE_ORGANIZATION_CHART
)
```

**توضيح**:يضيف هذا الكود شكل SmartArt إلى الشريحة الأولى عند إحداثيات محددة وبحجم محدد مسبقًا. `SmartArtLayoutType` تم إعداده لتصور البيانات الهرمية.

#### الخطوة 3: حفظ العرض التقديمي

احفظ مخطط التنظيم الخاص بك بتنسيق PPTX:

```python
pres.save("YOUR_OUTPUT_DIRECTORY/charts_organization_chart_out.pptx", slides.export.SaveFormat.PPTX)
```

**توضيح**: ال `save` تكتب الطريقة العرض التقديمي إلى ملف. استبدل `"YOUR_OUTPUT_DIRECTORY"` مع المسار الذي تريده.

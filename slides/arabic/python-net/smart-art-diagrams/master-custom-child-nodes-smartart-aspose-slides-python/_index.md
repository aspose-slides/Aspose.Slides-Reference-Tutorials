## إعداد Aspose.Slides لـ Python
قبل التعامل مع SmartArt، تأكد من تثبيت الأدوات اللازمة.

**تثبيت:**

```bash
pip install aspose.slides
```

### خطوات الحصول على الترخيص
يتطلب Aspose.Slides ترخيصًا لاستخدامه بكامل وظائفه. إليك كيفية البدء:
- **نسخة تجريبية مجانية**:ابدأ بإصدار تجريبي مجاني لاستكشاف الميزات.
- **رخصة مؤقتة**:تقدم بطلب للحصول على ترخيص مؤقت إذا لزم الأمر.

**التهيئة الأساسية:**
بمجرد التثبيت، قم بتهيئة Aspose.Slides في البرنامج النصي Python الخاص بك:

```python
import aspose.slides as slides
# تهيئة كائن العرض التقديمي
presentation = slides.Presentation()
```

## دليل التنفيذ
الآن بعد أن قمت بالإعداد، دعنا نستكشف الوظيفة الأساسية للتعامل مع عقد SmartArt الفرعية.

### إضافة شكل SmartArt وتحديد موضعه
**ملخص:**
سنبدأ بإضافة مخطط تنظيمي إلى الشريحة الأولى ووضعه في المكان الصحيح.
1. **تحميل العرض التقديمي**:
   ابدأ بتحميل ملف العرض التقديمي الحالي لديك أو إنشاء ملف جديد إذا لزم الأمر.

```python
with slides.Presentation("YOUR_DOCUMENT_DIRECTORY/smart_art_access.pptx") as pres:
    # يستمر الكود...
```
2. **إضافة شكل SmartArt**:
   أضف مخططًا تنظيميًا إلى الشريحة الأولى عند الإحداثيات والحجم المحددين:

```python
smart = pres.slides[0].shapes.add_smart_art(
    20, 20, 600, 500, slides.smartart.SmartArtLayoutType.ORGANIZATION_CHART)
```
### التلاعب بالعقد الفرعية
بعد ذلك، سنقوم بالتلاعب بالسمات المختلفة لعقد SmartArt الفرعية.
#### تحريك الشكل
**ملخص:**
ضبط موضع شكل SmartArt معين عن طريق تعديله `x` و `y` الإحداثيات.
3. **نقل العقدة**:
   الوصول إلى العقدة وتعديل موضعها:

```python
node = smart.all_nodes[1]
shape = node.shapes[1]
shape.x += (shape.width * 2)  # التحرك إلى اليمين بمضاعفة العرض
shape.y -= (shape.height / 2)  # التحرك لأعلى بمقدار نصف الارتفاع
```
#### تغيير حجم الشكل
**ملخص:**
قم بزيادة كل من العرض والارتفاع لأشكال SmartArt المحددة.
4. **تغيير العرض**:
   ضبط العرض:

```python
node = smart.all_nodes[2]
shape = node.shapes[1]
shape.width += (shape.width / 2)  # زيادة بنسبة 50%
```
5. **تغيير الارتفاع**:
   وبالمثل، اضبط الارتفاع:

```python
node = smart.all_nodes[3]
shape = node.shapes[1]
shape.height += (shape.height / 2)  # زيادة بنسبة 50%
```
#### تدوير الشكل
**ملخص:**
قم بتدوير شكل SmartArt معين للحصول على توجيه بصري أفضل.
6. **تدوير العقدة**:
   تدوير الشكل:

```python
node = smart.all_nodes[4]
shape = node.shapes[1]
shape.rotation = 90  # تدوير بمقدار 90 درجة
```
### حفظ العرض التقديمي
وأخيرًا، احفظ التغييرات في ملف جديد في دليل الإخراج.
7. **حفظ التغييرات**:
   حفظ العرض التقديمي المعدل:

```python
pres.save("YOUR_OUTPUT_DIRECTORY/smart_art_custom_child_nodes_out.pptx", slides.export.SaveFormat.PPTX)
```

## دليل التنفيذ

في هذا القسم، سنرشدك خلال عملية الوصول إلى عقد SmartArt وتعديلها داخل شريحة PowerPoint.

### الوصول إلى عقد SmartArt وتعديلها

**ملخص:** تتيح لك هذه الميزة الوصول برمجيًا إلى عقد محددة في رسم SmartArt وتعديلها حسب الحاجة. 

#### الخطوة 1: الوصول إلى الشريحة الأولى
```python
# الوصول إلى الشريحة الأولى من العرض التقديمي
slide = pres.slides[0]
```

#### الخطوة 2: إضافة شكل SmartArt
```python
# إضافة شكل SmartArt إلى الشريحة الأولى في الموضع والحجم المحددين
smart = slide.shapes.add_smart_art(0, 0, 400, 400, slides.smartart.SmartArtLayoutType.STACKED_LIST)
```
*توضيح:* ال `add_smart_art` تضع الطريقة رسم SmartArt على الشريحة وتحدد نوع تخطيطه.

#### الخطوة 3: الوصول إلى عقدة محددة
```python
# الوصول إلى العقدة الأولى في رسم SmartArt
node = smart.all_nodes[0]
```

#### الخطوة 4: الوصول إلى عقدة فرعية عن طريق الفهرس
```python
# الوصول إلى عقدة فرعية محددة داخل العقدة الأصلية باستخدام مؤشر موقعها
position = 1
child_node = node.child_nodes[position]

# عرض معلمات عقدة SmartArt الفرعية التي تم الوصول إليها
print("j = {0}, Text = {1}, Level = {2}, Position = {3}".format(position, child_node.text_frame.text,
                                                                child_node.level, child_node.position))
```
*توضيح:* توضح هذه الخطوة كيفية التنقل عبر العقد واسترجاع المعلومات مثل النص والموضع.

**نصيحة لاستكشاف الأخطاء وإصلاحها:** تأكد من تعريف بنية SmartArt بشكل صحيح قبل الوصول إلى العقد الفرعية لتجنب أخطاء الفهرس.

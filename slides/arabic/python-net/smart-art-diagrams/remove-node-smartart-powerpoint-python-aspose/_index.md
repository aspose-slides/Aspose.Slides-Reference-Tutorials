### دليل التنفيذ
دعونا نوضح كيفية إزالة عقدة معينة من رسومات SmartArt.

#### تحميل الشرائح والعبور
أولاً، قم بتحميل العرض التقديمي وتصفح أشكاله لتحديد SmartArt:

```python
import aspose.slides as slides

with slides.Presentation("YOUR_DOCUMENT_DIRECTORY/smart_art_access.pptx") as pres:
    # كرر كل شكل في الشريحة الأولى
    for shape in pres.slides[0].shapes:
        # تحقق مما إذا كان كائن SmartArt
        if isinstance(shape, slides.SmartArt):
            # انتقل إلى معالجة العقد إذا كانت موجودة
            if len(shape.all_nodes) > 0:
                node = shape.all_nodes[0]
```

#### الوصول إلى العقدة وإزالتها
لتعديل رسم SmartArt، قم بالوصول إلى العقدة المطلوبة وإزالتها:

```python
# تأكد من وجود عدد كافٍ من العقد الفرعية للإزالة
count = len(node.child_nodes)
if count >= 2:
    # إزالة العقدة الفرعية في الموضع 1
    node.child_nodes.remove_node(1)
```

#### احفظ التغييرات
وأخيرًا، احفظ عرضك التقديمي مع التعديلات:

```python
pres.save("YOUR_OUTPUT_DIRECTORY/smart_art_remove_node_pos_out.pptx", slides.export.SaveFormat.PPTX)
```

**شرح المعلمات والطرق:**
- **`all_nodes`**:قائمة العقد الموجودة داخل رسم SmartArt.
- **`remove_node(index)`**: يُزيل العقدة عند الفهرس المُحدد. تأكد من صحة الفهرس لتجنب الأخطاء.

## دليل التنفيذ

تعرف على كيفية قفل وفتح نسب أبعاد الجدول في PowerPoint باستخدام Aspose.Slides لـ Python.

### قفل نسبة العرض إلى الارتفاع للجدول (الميزة: قفل نسبة العرض إلى الارتفاع)

#### ملخص

تضمن هذه الميزة أن تغيير حجم الجداول لا يؤدي إلى تشويه شكلها، مما يحافظ على الاتساق البصري عبر الشرائح.

#### التنفيذ خطوة بخطوة

##### الوصول إلى العرض التقديمي والجدول

قم بتحميل العرض التقديمي الخاص بك والوصول إلى الجدول الذي ترغب في تعديله:

```python
import aspose.slides as slides

def lock_aspect_ratio():
    with slides.Presentation('YOUR_DOCUMENT_DIRECTORY/tables.pptx') as pres:
        # افترض أن الشكل الأول في الشريحة الأولى هو جدول.
        table = pres.slides[0].shapes[0]
```

##### التحقق من حالة قفل نسبة العرض إلى الارتفاع الحالية

تحقق مما إذا كان قفل نسبة العرض إلى الارتفاع مُمكّنًا بالفعل:

```python
print(f"Lock aspect ratio set: {table.shape_lock.aspect_ratio_locked}")
```

##### تبديل قفل نسبة العرض إلى الارتفاع

عكس الحالة الحالية لقفل نسبة العرض إلى الارتفاع:

```python
table.shape_lock.aspect_ratio_locked = not table.shape_lock.aspect_ratio_locked
```

##### حفظ التغييرات على العرض التقديمي الخاص بك

احفظ العرض التقديمي المعدّل:

```python
pres.save('YOUR_OUTPUT_DIRECTORY/tables_pres_lock_aspect_ratio_out.pptx', slides.export.SaveFormat.PPTX)
```

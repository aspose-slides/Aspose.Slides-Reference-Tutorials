## دليل التنفيذ

### تحديد الخلايا المدمجة في جداول PowerPoint

#### ملخص

تقوم هذه الميزة بمسح كل خلية في جدول ضمن شريحة PowerPoint للتحقق مما إذا كانت جزءًا من مجموعة مدمجة، مما يوفر تفاصيل حول نطاقها وموضع البداية.

#### خطوات التعريف
1. **تحميل العرض التقديمي**
   
   قم بتحميل ملف العرض التقديمي الخاص بك حيث تشك في وجود خلايا مدمجة:
   ```python
   with slides.Presentation("YOUR_DOCUMENT_DIRECTORY/tables.pptx") as pres:
       # الوصول إلى الشكل الأول في الشريحة الأولى (على افتراض أنه جدول)
       table = pres.slides[0].shapes[0]
   ```

2. **التكرار عبر الخلايا**
   
   قم بالمرور على كل خلية للتحقق من حالة الدمج وجمع التفاصيل:
   ```python
   def dump_merged_cell(i, j, current_cell):
       # طباعة المعلومات حول الخلية المدمجة
       print(f"Cell {i}{j} is part of a merged cell with row_span={current_cell.row_span}, col_span={current_cell.col_span}, starting from Cell {current_cell.first_row_index}{current_cell.first_column_index}.")
   
   for i, row in enumerate(table.rows):
       for j, cell in enumerate(row):
           if cell.is_merged_cell:
               dump_merged_cell(i, j, cell)
   ```

#### توضيح
- **`is_merged_cell`:** التحقق مما إذا كانت الخلية جزءًا من مجموعة مدمجة.
- **`row_span` و `col_span`:** قم بالإشارة إلى عدد الصفوف أو الأعمدة التي تمتد عبرها الخلية المدمجة.
- **`first_row_index` و `first_column_index`:** توفير موضع البداية للدمج.

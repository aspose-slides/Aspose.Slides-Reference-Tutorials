## دليل التنفيذ

دعونا نستعرض الخطوات اللازمة لتنسيق النص داخل صف جدول في PowerPoint باستخدام Aspose.Slides.

### الوصول إلى صفوف الجدول وتنسيقها (H2)

#### ملخص
سنبدأ بتحميل عرض تقديمي موجود، والوصول إلى جدول محدد داخله، وتطبيق خيارات التنسيق المختلفة على صفوفه.

#### الخطوة 1: تحميل العرض التقديمي الخاص بك

أولاً، قم بإنشاء أو فتح ملف PowerPoint يحتوي على جدول:

```python
input_presentation = 'YOUR_DOCUMENT_DIRECTORY/tables.pptx'
output_presentation = 'YOUR_OUTPUT_DIRECTORY/tables_text_format_inside_row_out.pptx'

with slides.Presentation(input_presentation) as presentation:
    # الوصول إلى الشكل الأول في الشريحة الأولى، والذي يُفترض أنه جدول
    table = presentation.slides[0].shapes[0]
```

#### الخطوة 2: تعيين ارتفاع الخط للخلايا في الصف الأول

ضبط حجم الخط باستخدام `PortionFormat`:

```python
# تعيين ارتفاع الخط للخلايا في الصف الأول
portion_format = slides.PortionFormat()
portion_format.font_height = 25  # تغيير ارتفاع الخط إلى الارتفاع المطلوب
table.rows[0].set_text_format(portion_format)
```

**توضيح:** ال `font_height` تتحكم المعلمة في حجم النص داخل كل خلية، مما يعزز الرؤية.

#### الخطوة 3: محاذاة النص وتعيين الهوامش

لمحاذاة النص إلى اليمين في خلايا الصف الأول:

```python
# تعيين محاذاة النص والهامش الأيمن للخلايا في الصف الأول
paragraph_format = slides.ParagraphFormat()
paragraph_format.alignment = slides.TextAlignment.RIGHT
paragraph_format.margin_right = 20  # مسافة من الحافة اليمنى
table.rows[0].set_text_format(paragraph_format)
```

**توضيح:** `ParagraphFormat` يتيح لك محاذاة النص وتعيين الهوامش، مما يوفر مظهرًا مصقولًا.

#### الخطوة 4: تعيين نوع النص الرأسي للخلايا في الصف الثاني

لاتجاه النص الرأسي:

```python
# تعيين نوع النص الرأسي للخلايا في الصف الثاني
text_frame_format = slides.TextFrameFormat()
text_frame_format.text_vertical_type = slides.TextVerticalType.VERTICAL
table.rows[1].set_text_format(text_frame_format)
```

**توضيح:** `TextFrameFormat` يغير كيفية عرض النص، وهو أمر قد يكون مفيدًا للغات مثل اليابانية أو الصينية.

#### الخطوة 5: احفظ العرض التقديمي الخاص بك

وأخيرًا، احفظ التغييرات في ملف جديد:

```python
# احفظ العرض التقديمي المعدّل في ملف جديد في دليل الإخراج
table.save(output_presentation, slides.export.SaveFormat.PPTX)
```

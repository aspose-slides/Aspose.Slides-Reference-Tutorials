## دليل التنفيذ
الآن بعد أن أصبحت بيئتنا جاهزة، فلننتقل إلى إضافة الجداول وتنسيقها في شرائح PowerPoint.

### إضافة جدول إلى الشريحة
#### ملخص
توضح هذه الميزة كيفية إضافة جدول إلى الشريحة الأولى من عرض تقديمي باستخدام Aspose.Slides لبايثون. تتيح لك تحديد أبعاد مثل عرض الأعمدة وارتفاع الصفوف.

#### خطوات التنفيذ
**الخطوة 1: إنشاء فئة العرض التقديمي**
إنشاء مثيل لـ `Presentation` الفئة التي تمثل ملف PowerPoint الخاص بك:
```python
def add_table_to_slide():
    with slides.Presentation() as pres:
        slide = pres.slides[0]
```

**الخطوة 2: تحديد أبعاد الجدول**
قم بتحديد أبعاد الجدول الخاص بك، مع تحديد عرض الأعمدة وارتفاع الصفوف:
```python
dbl_cols = [50, 50, 50, 50]  # عرض الأعمدة بالنقاط
dbl_rows = [50, 30, 30, 30, 30]  # ارتفاعات الصفوف بالنقاط
```

**الخطوة 3: إضافة جدول إلى الشريحة**
استخدم `add_table` طريقة إضافة جدول في الموضع المطلوب على الشريحة:
```python
table = slide.shapes.add_table(100, 50, dbl_cols, dbl_rows)
```

**الخطوة 4: حفظ العرض التقديمي**
احفظ العرض التقديمي بالجدول المضاف حديثًا:
```python
pres.save("YOUR_OUTPUT_DIRECTORY/table_added.pptx", slides.export.SaveFormat.PPTX)
```

### تعيين تنسيق حدود الخلية
#### ملخص
توضح هذه الميزة كيفية تعيين تنسيقات حدود لكل خلية في جدول ضمن شريحة. خصّص مظهر جداولك بفعالية.

#### خطوات التنفيذ
**الخطوة 1: إضافة جدول إلى الشريحة (راجع القسم السابق)**
تأكد من أنك قمت بإضافة جدول كما هو موضح أعلاه.

**الخطوة 2: تعيين تنسيق الحدود لكل خلية**
قم بالتكرار عبر كل خلية في الجدول وتعيين تنسيق الحدود:
```python
for row in table.rows:
    for cell in row:
        # تطبيق نوع 'NO_FILL' على جميع حدود الخلية
        cell.cell_format.border_top.fill_format.fill_type = slides.FillType.NO_FILL
        cell.cell_format.border_bottom.fill_format.fill_type = slides.FillType.NO_FILL
        cell.cell_format.border_left.fill_format.fill_type = slides.FillType.NO_FILL
        cell.cell_format.border_right.fill_format.fill_type = slides.FillType.NO_FILL
```

**الخطوة 3: حفظ العرض التقديمي**
حفظ العرض التقديمي مع حدود الجدول المحدثة:
```python
pres.save("YOUR_OUTPUT_DIRECTORY/table_border_no_fill_out.pptx", slides.export.SaveFormat.PPTX)
```

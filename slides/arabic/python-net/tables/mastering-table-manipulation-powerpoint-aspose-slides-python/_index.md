## دليل التنفيذ

### الوصول إلى جدول وتعديله في PowerPoint

#### ملخص
سنفتح ملف PPTX موجودًا، ونحدد جدولًا محددًا، ونُحدِّث محتوياته، ثم نحفظ التغييرات. هذه العملية مثالية لتحديثات دفعية لبيانات العرض التقديمي.

#### خطوات
1. **افتح العرض التقديمي الخاص بك**
   قم بتحميل ملف PowerPoint الخاص بك:
   ```python
   with slides.Presentation("YOUR_DOCUMENT_DIRECTORY/tables_update.pptx") as presentation:
       slide = presentation.slides[0]
   ```
   يفتح هذا الكود الملف ويسمح بالوصول إلى الشريحة الأولى.

2. **البحث عن الجدول وتحديثه**
   تحديد خلايا الجدول وتحديثها:
   ```python
   for shape in slide.shapes:
       if isinstance(shape, slides.Table):
           # تحديث النص في خلية معينة
           shape.rows[0][1].text_frame.text = "New"
   ```
   يقوم هذا المقطع بتحديث الخلية المطلوبة ضمن الصف الأول.

3. **احفظ التغييرات**
   احفظ العرض التقديمي المحدث الخاص بك:
   ```python
   presentation.save("YOUR_OUTPUT_DIRECTORY/tables_update_table_out.pptx", slides.export.SaveFormat.PPTX)
   ```
   يكتب الأمر التغييرات على القرص بتنسيق PPTX.

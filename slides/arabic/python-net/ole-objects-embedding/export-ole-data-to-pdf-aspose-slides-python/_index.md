### تصدير بيانات OLE إلى PDF

تتيح لك هذه الميزة الاحتفاظ بالكائنات المضمنة في ملفات PowerPoint عند تحويلها إلى ملفات PDF، مما يضمن عدم فقدان المعلومات أو الوظائف.

#### الخطوة 1: تحميل العرض التقديمي الخاص بك

قم بتحميل العرض التقديمي الذي يحتوي على كائنات OLE باستخدام Aspose.Slides.

```python
document_directory = 'YOUR_DOCUMENT_DIRECTORY/'
output_directory = 'YOUR_OUTPUT_DIRECTORY/'

with slides.Presentation(document_directory + "PresOleExample.pptx") as pres:
    # انتقل إلى إنشاء خيارات تصدير PDF
```

#### الخطوة 2: إنشاء خيارات تصدير PDF

هنا، نقوم بتحديد الإعدادات لتصدير العرض التقديمي الخاص بك.

```python
options = slides.export.PdfOptions()
options.include_ole_data = True  # يضمن هذا الحفاظ على بيانات OLE في ملف PDF
```

#### الخطوة 3: الحفظ بتنسيق PDF

احفظ العرض التقديمي باستخدام الخيارات المحددة لإخراج ملف PDF يحتفظ بجميع الكائنات المضمنة.

```python
pres.save(output_directory + "PresOleExample.pdf", slides.export.SaveFormat.PDF, options)
```

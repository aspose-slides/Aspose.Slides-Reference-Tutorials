## دليل التنفيذ

يتناول هذا القسم بالتفصيل كيفية استخراج بيانات الملف المضمنة من كائنات OLE داخل عروض PowerPoint التقديمية.

### تحميل الشرائح وتكرارها

قم بتحميل العرض التقديمي الخاص بك وتكراره عبر أشكال كل شريحة:

```python
with slides.Presentation(document_path) as pres:
    for slide in pres.slides:
        # معالجة كل شكل على الشريحة
```

### تحديد إطارات كائنات OLE

تحديد ما إذا كان الشكل هو `OleObjectFrame`، مما يشير إلى أنه يحتوي على بيانات مضمنة:

```python
count = 0
for slide in pres.slides:
    for shape in slide.shapes:
        if isinstance(shape, slides.OleObjectFrame):
            # يحتوي هذا الشكل على كائن OLE يحتوي على بيانات مضمنة
```

### استخراج بيانات الملف المضمن

بعد تحديد كائنات OLE، قم باستخراج بياناتها وحفظها باستخدام اسم ملف فريد:

```python
count = 0
for slide in pres.slides:
    for shape in slide.shapes:
        if isinstance(shape, slides.OleObjectFrame):
            count += 1
            
            # استخراج بيانات الملف والامتداد
            data = shape.embedded_data.embedded_file_data
            extension = shape.embedded_data.embedded_file_extension
            
            # إنشاء اسم ملف بناءً على رقم الكائن
            file_name = f"shapes_ole_objects{count}_out.{extension}"
            
            # الكتابة إلى دليل الإخراج
            with open(f"YOUR_OUTPUT_DIRECTORY/{file_name}", "wb") as file:
                file.write(data)
```

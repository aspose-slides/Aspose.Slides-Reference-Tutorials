## دليل التنفيذ
### إضافة إطار كائن OLE
في هذا القسم، سنوضح كيفية تضمين ملف Excel في شريحة PowerPoint كإطار كائن OLE.

#### الخطوة 1: تحميل ملف Excel
أولاً، أنشئ دالة لقراءة ملف Excel وتحويله إلى مصفوفة بايتات. هذا ضروري للتضمين:

```python
def load_excel_file(file_path):
    # فتح ملف Excel في وضع القراءة الثنائية
    with open(file_path, "rb") as fs:
        return fs.read()
```

#### الخطوة 2: إضافة إطار كائن OLE إلى الشريحة
بعد ذلك، دعنا ننشئ دالة تضيف إطار كائن OLE يحتوي على بيانات Excel الخاصة بك إلى الشريحة الأولى:

```python
def add_ole_object_frame():
    # إنشاء فئة عرض تقديمي تمثل ملف PPTX
    with slides.Presentation() as pres:
        # الوصول إلى الشريحة الأولى
        slide = pres.slides[0]
        
        # تحميل بيانات ملف Excel إلى مصفوفة بايتات
        excel_data = load_excel_file(DATA_DIR + "book.xlsx")
        
        # إنشاء كائن بيانات لتضمين محتوى Excel
        data_info = slides.dom.ole.OleEmbeddedDataInfo(excel_data, "xlsx")
        
        # أضف شكل إطار كائن OLE لتغطية الشريحة بأكملها
        ole_object_frame = slide.shapes.add_ole_object_frame(
            0, 0,                    # الموضع (س، ص)
            pres.slide_size.size.width, pres.slide_size.size.height, # الحجم (العرض، الارتفاع)
            data_info                # كائن معلومات البيانات الذي يحتوي على محتوى Excel
        )
        
        # حفظ العرض التقديمي على القرص باستخدام كائن OLE المضمن
        pres.save(OUTPUT_DIR + "shapes_add_ole_object_frame_out.pptx", slides.export.SaveFormat.PPTX)
```

### المعلمات والطرق
- **`add_ole_object_frame()`**:تعمل هذه الوظيفة على إنشاء إطار كائن OLE في شريحة PowerPoint الخاصة بك.
  - `0, 0`:الموضع العلوي الأيسر للإطار على الشريحة.
  - `pres.slide_size.size.width`، `pres.slide_size.size.height`:يضمن أن يغطي الإطار الشريحة بأكملها.
  - `data_info`:تحتوي على بيانات Excel المراد تضمينها.

### نصائح استكشاف الأخطاء وإصلاحها
- **مشاكل مسار الملف**:تأكد من أن مسار ملف Excel الخاص بك صحيح ويمكن الوصول إليه من دليل تشغيل البرنامج النصي.

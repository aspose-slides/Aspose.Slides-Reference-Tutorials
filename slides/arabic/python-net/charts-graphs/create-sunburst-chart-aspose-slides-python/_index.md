# كيفية إنشاء مخططات Sunburst في Python باستخدام Aspose.Slides

## مقدمة
يُعد إنشاء مخططات بيانية جذابة بصريًا أمرًا أساسيًا لتصور البيانات بفعالية، خاصةً عند عرض البيانات الهرمية. يرشدك هذا البرنامج التعليمي إلى كيفية استخدام مكتبة Aspose.Slides القوية مع بايثون لإنشاء مخططات بيانية ديناميكية مناسبة لتقارير الأعمال ومجموعات البيانات المعقدة.

في عالمنا اليوم الذي يركز على البيانات، تُسهّل أدوات مثل Aspose.Slides دمج إمكانيات الرسوم البيانية المتقدمة في تطبيقاتك. اتبع هذا الدليل من الإعداد إلى التنفيذ، لضمان قدرة حتى المبتدئين على إنشاء رسوم بيانية جذابة دون عناء.

**ما سوف تتعلمه:**
- كيفية إعداد Aspose.Slides لـ Python
- خطوات تهيئة العرض التقديمي وإضافة مخطط أشعة الشمس
- تكوين الفئات وسلاسل البيانات
- تحسين مخطط Sunburst الخاص بك لتحقيق الأداء

دعونا نبدأ بالمتطلبات الأساسية اللازمة قبل أن نبدأ!

## دليل التنفيذ
### إنشاء مخطط Sunburst
دعنا نستعرض الخطوات المطلوبة لإنشاء مخطط Sunburst وتكوينه باستخدام Aspose.Slides.

#### الخطوة 1: تهيئة كائن العرض التقديمي
ابدأ بإنشاء كائن عرض تقديمي جديد، والذي يعمل كحاوية للشرائح والمخططات البيانية الخاصة بك:

```python
def create_sunburst_chart():
    with slides.Presentation() as pres:
        # يؤدي هذا إلى إنشاء مدير سياق للتعامل مع دورة حياة العرض التقديمي.
```

#### الخطوة 2: إضافة مخطط Sunburst
أضف مخططًا لشروق الشمس عند إحداثيات محددة ضمن الشريحة الأولى. عدّل موضعه وحجمه حسب الحاجة:

```python
        chart = pres.slides[0].shapes.add_chart(
            slides.charts.ChartType.SUNBURST, 50, 50, 500, 400)
        
        # المعلمات: نوع الرسم البياني، موضع x، موضع y، العرض، الارتفاع
```

#### الخطوة 3: مسح البيانات الموجودة
قبل ملء الرسم البياني الخاص بك بالبيانات، قم بمسح أي فئات وسلاسل افتراضية للبدء من جديد:

```python
        chart.chart_data.categories.clear()
        chart.chart_data.series.clear()
        
        # الوصول إلى المصنف لمعالجة بيانات الرسم البياني
        wb = chart.chart_data.chart_data_workbook
        wb.clear(0)  # مسح جميع الخلايا في المصنف
```

#### الخطوة 4: تكوين الفئات ومستويات التجميع
عرّف الفئات الهرمية بإضافة الأوراق والجذوع والفروع. استخدم مستويات التجميع لتنظيم بياناتك بصريًا:

```python
        # تكوين الفرع 1
        leaf = chart.chart_data.categories.add(wb.get_cell(0, "C1", "Leaf1"))
        leaf.grouping_levels.set_grouping_item(1, "Stem1")
        leaf.grouping_levels.set_grouping_item(2, "Branch1")

        # أضف أوراقًا إضافية أسفل الفرع 1
        chart.chart_data.categories.add(wb.get_cell(0, "C2", "Leaf2"))
```

واصل هذا النمط للفروع والأوراق الأخرى حسب الحاجة.

#### الخطوة 5: إضافة سلسلة البيانات
أنشئ سلسلة بيانات وأضف إليها القيم. تربط هذه الخطوة فئاتك بنقاط البيانات المقابلة.

```python
        series = chart.chart_data.series.add(slides.charts.ChartType.SUNBURST)
        series.labels.default_data_label_format.show_category_name = True
        
        # إضافة نقاط البيانات إلى السلسلة
        series.data_points.add_data_point_for_sunburst_series(wb.get_cell(0, "D1", 4))
```

#### الخطوة 6: احفظ العرض التقديمي الخاص بك
أخيرًا، احفظ العرض التقديمي الخاص بك باستخدام مخطط أشعة الشمس الذي تم إنشاؤه حديثًا:

```python
        pres.save("YOUR_OUTPUT_DIRECTORY/charts_sunburst_chart_out.pptx", slides.export.SaveFormat.PPTX)
        
        # تأكد من تحديد مسار دليل الإخراج صالحًا
```

### نصائح استكشاف الأخطاء وإصلاحها
- **عدم تطابق البيانات:** إذا كانت نقاط البيانات الخاصة بك لا تتوافق مع الفئات، فتأكد من التحقق من تكوينات الفئة والسلسلة.
- **الرسم البياني غير ظاهر:** تأكد من أن موضع الرسم البياني وحجمه يقعان ضمن حدود الشريحة.

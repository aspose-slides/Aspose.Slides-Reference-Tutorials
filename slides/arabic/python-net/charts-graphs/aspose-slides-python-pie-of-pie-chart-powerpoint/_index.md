# كيفية إنشاء مخطط دائري في PowerPoint باستخدام Aspose.Slides لـ Python

إنشاء مخططات جذابة بصريًا، مثل مخطط "فطيرة الفطيرة"، يُحسّن عروض PowerPoint التقديمية بشكل ملحوظ، إذ يُسهّل استيعاب المعلومات المعقدة. يُرشدك هذا البرنامج التعليمي إلى كيفية إنشاء مخطط "فطيرة الفطيرة" باستخدام Aspose.Slides لـ Python.

### دليل التنفيذ

#### الميزة 1: إنشاء عرض تقديمي باستخدام الرسم البياني

ستوضح هذه الميزة كيفية إنشاء عرض تقديمي في PowerPoint وإضافة مخطط دائري إلى الشريحة الأولى.

##### إضافة الرسم البياني

ابدأ بإنشاء عرض تقديمي جديد وإضافة دائرة أو مخطط دائري في الموضع (50، 50) على الشريحة الأولى:

```python
import aspose.slides as slides

with slides.Presentation() as presentation:
    # أضف مخطط "فطيرة الفطيرة" بأبعاد محددة
    chart = presentation.slides[0].shapes.add_chart(
        slides.charts.ChartType.PIE_OF_PIE, 50, 50, 500, 400)
```

##### تكوين تسميات البيانات

لتحسين قابلية القراءة، قم بتكوين تسميات البيانات لعرض القيم:

```python
# تمكين عرض القيمة في تسميات البيانات لتحقيق وضوح أفضل
chart.chart_data.series[0].labels.default_data_label_format.show_value = True
```

##### إعداد خيارات فطيرة الفطيرة

قم بتكوين خصائص محددة لفطيرة الرسم البياني الدائري، مثل حجم الفطيرة الثانية وموضع التقسيم:

```python
# تعيين حجم الفطيرة الثانية وخصائص التقسيم
chart.chart_data.series[0].parent_series_group.second_pie_size = 149
chart.chart_data.series[0].parent_series_group.pie_split_by = slides.charts.PieSplitType.BY_PERCENTAGE
chart.chart_data.series[0].parent_series_group.pie_split_position = 53
```

##### حفظ العرض التقديمي

وأخيرًا، احفظ العرض التقديمي في الدليل المطلوب:

```python
# حفظ العرض التقديمي مع الرسم البياني
presentation.save("YOUR_OUTPUT_DIRECTORY/charts_second_plot_options_out.pptx", slides.export.SaveFormat.PPTX)
```

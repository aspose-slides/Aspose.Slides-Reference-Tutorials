#### ملخص

سنقوم بإنشاء ملف PowerPoint جديد، وإضافة شريحة، وإدراج مخطط خريطة، وملئه بالبيانات، وتخصيص مظهره، وحفظ النتيجة النهائية.

```python
def create_and_save_presentation():
```

##### إضافة مخطط خريطة

أضف مخططًا من نوع MAP إلى الشريحة الأولى الخاصة بك:

```python
with slides.Presentation() as presentation:
    # أدخل مخطط الخريطة في الموضع (50، 50) بحجم (500 × 400)
    chart = presentation.slides[0].shapes.add_chart(
        slides.charts.ChartType.MAP, 50, 50, 500, 400, False
    )
```

- **حدود:** 
  - `ChartType.MAP`:يحدد نوع الرسم البياني.
  - `(50, 50)`:الموضع على الشريحة.
  - `(500x400)`:أبعاد العرض والارتفاع.

##### إضافة سلسلة ونقاط بيانات

املأ مخطط الخريطة الخاص بك بنقاط البيانات:

```python
wb = chart.chart_data.chart_data_workbook

# إضافة سلسلة ونقاط بيانات
to_series = chart.chart_data.series.add(slides.charts.ChartType.MAP)
to_series.data_points.add_data_point_for_map_series(wb.get_cell(0, "B2", 5))
to_series.data_points.add_data_point_for_map_series(wb.get_cell(0, "B3", 1))
to_series.data_points.add_data_point_for_map_series(wb.get_cell(0, "B4", 10))
```

- **لماذا:** تضيف هذه الخطوة البيانات الفعلية التي سيعرضها مخطط الخريطة الخاص بك.

##### تحديد الفئات لمخطط الخريطة

تعيين فئات جغرافية لكل نقطة بيانات:

```python
# إضافة الفئات
to_chart.chart_data.categories.add(wb.get_cell(0, "A2", "United States"))
to_chart.chart_data.categories.add(wb.get_cell(0, "A3", "Mexico"))
to_chart.chart_data.categories.add(wb.get_cell(0, "A4", "Brazil"))
```

- **لماذا:** يحدد هذا المناطق التي تمثلها نقاط البيانات الخاصة بك.

##### تخصيص مظهر نقطة البيانات

قم بتعزيز المظهر البصري من خلال تخصيص نقطة بيانات:

```python
# تخصيص مظهر نقطة بيانات واحدة
```

- **لماذا:** يساعد تعزيز نقطة بيانات محددة على إبرازها للتأكيد عليها.

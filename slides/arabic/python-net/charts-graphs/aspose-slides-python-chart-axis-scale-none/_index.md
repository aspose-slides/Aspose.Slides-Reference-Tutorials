# كيفية تعيين مقياس محور الرسم البياني إلى NONE باستخدام Aspose.Slides Python
## مقدمة
يتطلب إنشاء مخططات جذابة بصريًا ضبطًا دقيقًا لمقاييس محاورها. يوضح هذا البرنامج التعليمي ضبط مقياس الوحدة الرئيسية للمحور الأفقي على `NONE` لإنشاء مخطط باستخدام Aspose.Slides في Python، وهو مثالي لتخصيص تصور البيانات في العروض التقديمية الخاصة بك.
**ما سوف تتعلمه:**
- إعداد Aspose.Slides لـ Python.
- إنشاء المخططات وتخصيصها باستخدام تكوينات المحور المحددة.
- حفظ العروض التقديمية برمجيًا.
- استكشاف الأخطاء وإصلاحها عند العمل مع محاور الرسم البياني.

### المكتبات المطلوبة

## إعداد Aspose.Slides لـ Python

**تثبيت:**
```bash
pip install aspose.slides
```

import aspose.slides as slides

## دليل التنفيذ
### إنشاء مخطط باستخدام مقياس المحور المخصص
#### ملخص
سنقوم بإنشاء مخطط من نوع AREA وتعيين مقياس الوحدة الرئيسية للمحور الأفقي الخاص به إلى `NONE`.
**الخطوة 1: تهيئة العرض التقديمي**
ابدأ بإنشاء مثيل عرض تقديمي جديد:
```python
with slides.Presentation() as pres:
    # سيتم إجراء عمليات أخرى هنا.
```
يضمن مدير السياق هذا إدارة الموارد بكفاءة.
#### الخطوة 2: إضافة مخطط
أضف مخططًا من نوع AREA إلى الشريحة الخاصة بك عند إحداثيات وأبعاد محددة:
```python
chart = pres.slides[0].shapes.add_chart(slides.charts.ChartType.AREA, 10, 10, 400, 300, True)
```
يؤدي هذا إلى إضافة مخطط بحجم 400 × 300 بكسل في الموضع (10، 10) على الشريحة الأولى.
#### الخطوة 3: تعيين مقياس المحور إلى NONE
تعديل مقياس الوحدة الرئيسية للمحور الأفقي:
```python
chart.axes.horizontal_axis.major_unit_scale = slides.charts.TimeUnitType.NONE
```
يؤدي تعيين هذه الخاصية إلى إزالة فترات القياس المحددة مسبقًا على طول المحور x.
#### الخطوة 4: حفظ العرض التقديمي
احفظ التغييرات في ملف بتنسيق PPTX:
```python
pres.save("YOUR_OUTPUT_DIRECTORY/charts_time_unit_type_enum_out.pptx", slides.export.SaveFormat.PPTX)
```
يؤدي هذا إلى حفظ الرسم البياني المخصص الخاص بك داخل ملف عرض تقديمي جديد.

## قسم الأسئلة الشائعة
1. **كيف أقوم بتحديث Aspose.Slides؟**
   - يستخدم `pip install --upgrade aspose.slides`.
2. **هل يمكنني ضبط مقياس المحور الأفقي والرأسي على "لا شيء"؟**
   - نعم استخدم `chart.axes.vertical_axis.major_unit_scale = slides.charts.TimeUnitType.NONE`.
3. **ماذا لو لم يتم حفظ الرسم البياني الخاص بي بشكل صحيح؟**
   - تحقق من مسارات الملفات وتأكد من إمكانية الكتابة إلى دليل الإخراج الخاص بك.
4. **هل هناك طريقة لمعاينة التغييرات قبل الحفظ؟**
   - لا يوفر Aspose.Slides معاينة مباشرة، لكنه يكرر ذلك باستخدام نصوص أصغر حجمًا حتى يتم الرضا.
5. **كيف أتعامل مع أنواع المخططات المختلفة؟**
   - يستبدل `ChartType.AREA` مع أنواع أخرى مثل `Bar`، `Line`، وما إلى ذلك، حسب الحاجة.

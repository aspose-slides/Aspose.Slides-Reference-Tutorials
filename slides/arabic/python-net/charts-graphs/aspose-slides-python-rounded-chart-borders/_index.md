# تحسين مخططات PowerPoint باستخدام حدود مستديرة في Aspose.Slides

## مقدمة

حسّن عروض PowerPoint التقديمية بإضافة عناصر جذابة بصريًا، مثل حدود المخطط الدائرية، باستخدام Aspose.Slides لـ Python. سيرشدك هذا الدليل إلى كيفية إنشاء مخطط عمودي مجمع بزوايا دائرية، مما يعزز جمالية العرض وجاذبيته المهنية.

**ما سوف تتعلمه:**
- إنشاء العروض التقديمية في Aspose.Slides لـ Python.
- إضافة مخطط عمودي مجمع إلى الشرائح الخاصة بك.
- تطبيق حدود مستديرة على منطقة الرسم البياني.
- حفظ وتصدير العرض التقديمي الخاص بك بشكل فعال.

بإتقان هذه المهارات، ستُحسّن بشكل ملحوظ من مهاراتك في عرض البيانات في PowerPoint. لنتأكد من جاهزيتك التامة لبدء هذا البرنامج التعليمي.

## المتطلبات الأساسية

لمتابعة هذا الدليل، تأكد من أن لديك:

- **Aspose.Slides لـ Python** تم تثبيته على نظامك.
- فهم أساسي لبرمجة بايثون.
- بيئة تم إعدادها لتشغيل نصوص Python (على سبيل المثال، IDE مثل PyCharm أو VS Code).

تأكد من تثبيت مكتبة Aspose.Slides. يفترض هذا البرنامج التعليمي أنك تستخدم إصدارًا متوافقًا من بايثون (يوصى بالإصدار 3.x).

## إعداد Aspose.Slides لـ Python

ثبّت مكتبة Aspose.Slides باستخدام pip. افتح الطرفية أو موجه الأوامر وشغّل:

```bash
pip install aspose.slides
```

### الحصول على الترخيص
- **نسخة تجريبية مجانية**:استخدم Aspose.Slides في الوضع التجريبي لاستكشاف ميزاته.
- **رخصة مؤقتة**:احصل على ترخيص مؤقت للوظائف الكاملة دون قيود التقييم.
- **شراء الترخيص**:للاستخدام المستمر، فكر في شراء ترخيص.

بعد التثبيت، قم بتهيئة بيئتك باستخدام مقتطف التعليمات البرمجية التالي:

```python
import aspose.slides as slides

# تهيئة مثيل العرض التقديمي
presentation = slides.Presentation()
```

## دليل التنفيذ

### نظرة عامة على الميزة: حدود مستديرة في منطقة الرسم البياني

ترتكز هذه الميزة على تحسين جماليات المخطط من خلال دمج الزوايا المستديرة في عروض PowerPoint الخاصة بك.

#### الخطوة 1: إنشاء عرض تقديمي جديد
ابدأ بتهيئة كائن العرض التقديمي. هذا يُشكّل أساسًا لإضافة مخططاتك وعناصر أخرى.

```python
def create_presentation_with_rounded_chart():
    with slides.Presentation() as presentation:
        # الوصول إلى الشريحة الأولى في العرض التقديمي
        slide = presentation.slides[0]
```

#### الخطوة 2: إضافة مخطط عمودي مجمع
ضع مخططًا عموديًا مجمعًا على الشريحة. حدد موقعه وحجمه للحصول على تخطيط مثالي.

```python
# أضف مخططًا عموديًا مجمعًا في الموضع (20، 100) بعرض 600 وارتفاع 400
chart = slide.shapes.add_chart(
    slides.charts.ChartType.CLUSTERED_COLUMN,
    20,
    100,
    600,
    400
)
```

#### الخطوة 3: تكوين تنسيق خط الرسم البياني
قم بتطبيق نوع تعبئة صلب على حدود الرسم البياني، مع التأكد من بروزه مقارنة بخلفية العرض التقديمي.

```python
# تعيين تنسيق الخط إلى نوع التعبئة الصلبة
cart.line_format.fill_format.fill_type = slides.FillType.SOLID
cart.line_format.style = slides.LineStyle.SINGLE
```

#### الخطوة 4: تمكين الزوايا المستديرة
قم بتنشيط ميزة الزوايا المستديرة للحصول على مظهر عصري وأنيق لمنطقة الرسم البياني الخاصة بك.

```python
# تمكين الزوايا المستديرة لمنطقة الرسم البياني
cart.has_rounded_corners = True
```

#### الخطوة 5: احفظ العرض التقديمي الخاص بك
وأخيرًا، احفظ العرض التقديمي الخاص بك في الدليل المحدد باسم الملف المناسب.

```python
presentation.save(
    "YOUR_OUTPUT_DIRECTORY/charts_chart_area_rounded_borders_out.pptx",
    slides.export.SaveFormat.PPTX
)
```

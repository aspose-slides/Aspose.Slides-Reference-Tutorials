### إضافة مخطط الهيستوجرام إلى الشريحة
أضف مخططًا بيانيًا جديدًا من نوع "هيستوغرام" إلى الشريحة الأولى. هذا يُهيئ مساحة عملك لرسم البيانات.


### مسح البيانات الموجودة
تأكد من أن الرسم البياني يبدأ بدون أي بيانات موجودة مسبقًا عن طريق مسح الفئات والسلاسل.


```python
        # الخطوة 4: إضافة البيانات إلى السلسلة

        series.data_points.add_data_point_for_histogram_series(wb.get_cell(0, "A1", 15))
```

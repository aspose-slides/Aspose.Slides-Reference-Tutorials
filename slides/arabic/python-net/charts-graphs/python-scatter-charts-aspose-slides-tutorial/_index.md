# كيفية إنشاء مخططات التشتت وتخصيصها في PowerPoint باستخدام Python و Aspose.Slides

**ما سوف تتعلمه:**
- إعداد Aspose.Slides لـ Python.
- إنشاء عرض تقديمي أساسي باستخدام مخطط التشتت.
- إضافة سلسلة البيانات إلى الرسم البياني الخاص بك.
- تخصيص مظهر مخطط التشتت الخاص بك.

دعونا نتعمق في كيفية الاستفادة من Aspose.Slides لتحسين العروض التقديمية الخاصة بك!

## دليل التنفيذ

سنقوم بتقسيم العملية إلى أقسام منطقية استنادًا إلى الميزات الرئيسية: إنشاء العروض التقديمية، وإضافة مخططات التشتت، وإضافة سلسلة البيانات، والتخصيص.

### إنشاء عرض تقديمي باستخدام مخطط التشتت

#### ملخص
إنشاء عرض تقديمي وتضمين مخطط تشتت سهل باستخدام Aspose.Slides. يرشدك هذا القسم إلى كيفية إنشاء ملف PowerPoint يحتوي على مخطط تشتت أولي.

#### خطوات التنفيذ
**1. تهيئة العرض التقديمي:**

```python
import aspose.slides as slides

def create_and_add_scatter_chart():
    with slides.Presentation() as pres:
        slide = pres.slides[0]
```

**2. أضف مخططًا تشتتًا إلى الشريحة:**
هنا، يمكنك وضع الرسم البياني الخاص بك وتحديد حجمه داخل الشريحة.

```python
        chart = slide.shapes.add_chart(
            slides.charts.ChartType.SCATTER_WITH_SMOOTH_LINES,
            0, 0, 400, 400
        )
```

**3. احفظ العرض التقديمي:**
تأكد من حفظ العرض التقديمي الخاص بك بعد إجراء التغييرات:

```python
    pres.save("YOUR_OUTPUT_DIRECTORY/charts_scattered_chart_out.pptx", slides.export.SaveFormat.PPTX)
```

### إضافة سلسلة بيانات إلى الرسم البياني

#### ملخص
لجعل مخططات التشتت ذات معنى، تحتاج إلى بيانات. يشرح هذا القسم كيفية إضافة سلسلة من نقاط البيانات إلى مخططك.

**1. مسح السلسلة الحالية:**

```python
        chart.chart_data.series.clear()
```

**2. إضافة سلسلة بيانات جديدة:**
يستخدم `add` طريقة إدراج سلسلة بيانات جديدة في الرسم البياني:

```python
        series1 = chart.chart_data.series.add(
            fact.get_cell(default_worksheet_index, 1, 1, "Series 1"), chart.type
        )
        series2 = chart.chart_data.series.add(
            fact.get_cell(default_worksheet_index, 1, 3, "Series 2"), chart.type
        )
```

### تخصيص السلسلة وإضافة نقاط البيانات

#### ملخص
يُحسّن التخصيص من مظهر مخططاتك البيانية وسهولة قراءتها. يتناول هذا القسم إضافة نقاط البيانات وتخصيص علامات السلسلة.

**1. إضافة نقاط البيانات:**

```python
        series1.data_points.add_data_point_for_scatter_series(
            fact.get_cell(default_worksheet_index, 2, 1, 1), 
            fact.get_cell(default_worksheet_index, 2, 2, 3)
        )
```

**2. تخصيص علامات السلسلة:**

```python
        series1.marker.size = 10
        series1.marker.symbol = slides.charts.MarkerStyleType.STAR
```

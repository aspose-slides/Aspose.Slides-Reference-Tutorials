## دليل التنفيذ
دعونا نقوم بتقسيم عملية إنشاء مخطط متعدد الفئات إلى خطوات قابلة للإدارة.

### إنشاء هيكل الرسم البياني (H2)
#### ملخص:
سنبدأ بإعداد الهيكل الأساسي لمخططنا، بما في ذلك تهيئة العرض التقديمي وإضافة مخطط عمودي مجمع إلى شريحة.

**الخطوة 1: تهيئة العرض التقديمي**

```python
import aspose.slides as slides

def create_multi_category_chart():
    with slides.Presentation() as pres:
        slide = pres.slides[0]  # الوصول إلى الشريحة الأولى
```

- **لماذا؟**:يسمح لنا هذا الإعداد ببدء إنشاء العرض التقديمي الخاص بنا من البداية.

**الخطوة 2: إضافة الرسم البياني إلى الشريحة**

```python
        ch = slide.shapes.add_chart(
            slides.charts.ChartType.CLUSTERED_COLUMN, 
            100, 100, 600, 450
        )
```

- **حدود**: 
  - `ChartType.CLUSTERED_COLUMN`:يحدد نوع الرسم البياني.
  - `(100, 100)`:الموضع على الشريحة.
  - `(600, 450)`:عرض وارتفاع الرسم البياني.

**الخطوة 3: مسح البيانات الموجودة**

```python
        ch.chart_data.series.clear()
        ch.chart_data.categories.clear()
```

- **لماذا؟**:يضمن هذا عدم وجود أي بيانات متبقية تؤثر على تكوين الرسم البياني الجديد.

### تكوين الفئات والسلاسل (H2)
#### ملخص:
بعد ذلك، سنقوم بإعداد الفئات بمستويات التجميع وإضافة سلسلة تحتوي على نقاط البيانات إلى الرسم البياني.

**الخطوة 4: تحديد الفئات**

```python
        fact = ch.chart_data.chart_data_workbook 
        category_labels = ['A', 'B', 'C', 'D', 'E', 'F', 'G', 'H']
        grouping_levels = ['Group1', 'Group2', 'Group3', 'Group4']

        for i, label in enumerate(category_labels):
            category = ch.chart_data.categories.add(fact.get_cell(0, f"c{i+2}", label))
            if i < len(grouping_levels):
                category.grouping_levels.set_grouping_item(1, grouping_levels[i])
```

- **لماذا؟**:تعمل مجموعات الفئات على تحسين قابلية القراءة وتسمح بإجراء تحليل مقارن.

**الخطوة 5: إضافة سلسلة تحتوي على نقاط البيانات**

```python
        series = ch.chart_data.series.add(
            fact.get_cell(0, "D1", "Series 1"), slides.charts.ChartType.CLUSTERED_COLUMN)
        
        values = [10, 20, 30, 40, 50, 60, 70, 80]
        for i, value in enumerate(values):
            series.data_points.add_data_point_for_bar_series(
                fact.get_cell(0, f"D{i+2}", value))
```

- **لماذا؟**:تعتبر نقاط البيانات ضرورية لعرض القيم الفعلية ضمن كل فئة.

### حفظ العرض التقديمي (H2)
**الخطوة 6: احفظ عملك**

        pres.save("YOUR_OUTPUT_DIRECTORY/charts_multi_category_chart_out.pptx", slides.export.SaveFormat.PPTX)

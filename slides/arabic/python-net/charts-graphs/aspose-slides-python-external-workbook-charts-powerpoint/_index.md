## دليل التنفيذ

### الميزة 1: نسخ المصنف الخارجي

#### ملخص
يُعد نسخ مصنف خارجي أمرًا ضروريًا لضمان أن يشير عرضك التقديمي إلى أحدث مجموعة بيانات. توضح هذه الميزة كيفية نسخ ملف من دليل المصدر إلى الوجهة باستخدام بايثون. `shutil` وحدة.

#### خطوات التنفيذ
**الخطوة 1**:استيراد الوحدات الضرورية
```python
import shutil
```

**الخطوة 2**:تحديد وظيفة نسخ المصنف
إنشاء وظيفة للتعامل مع عملية النسخ:
```python
def copy_external_workbook():
    external_workbook_file_name = "charts_external_workbook.xlsx"
    # استخدم shutil.copyfile لنقل الملف من المصدر إلى الوجهة
    shutil.copyfile(
        "YOUR_DOCUMENT_DIRECTORY/" + external_workbook_file_name,
        "YOUR_OUTPUT_DIRECTORY/" + external_workbook_file_name
    )
```
- **حدود**: `shutil.copyfile(source, destination)` أين `source` هو مسار الملف الأصلي الخاص بك و `destination` هو الدليل المستهدف.

### الميزة 2: إنشاء عرض تقديمي باستخدام مخطط مصنف خارجي

#### ملخص
تتضمن هذه الميزة إنشاء عرض تقديمي في PowerPoint وإضافة مخطط يشير إلى مصنف خارجي، مما يسمح بالتحديثات الديناميكية كلما تغيرت بيانات المصدر.

#### خطوات التنفيذ
**الخطوة 1**:استيراد وحدة Aspose.Slides
```python
import aspose.slides as slides
```

**الخطوة 2**:تحديد وظيفة إنشاء العرض التقديمي
إنشاء دالة لبناء العرض التقديمي الخاص بك باستخدام المخططات البيانية:
```python
def create_presentation_with_external_chart():
    # فتح عرض تقديمي جديد أو إنشاء عرض تقديمي جديد
    with slides.Presentation() as pres:
        # إضافة مخطط دائري عند الإحداثيات والحجم المحددين
        chart = pres.slides[0].shapes.add_chart(slides.charts.ChartType.PIE, 50, 50, 500, 400)

        # مسح البيانات الموجودة في المصنف
        chart.chart_data.chart_data_workbook.clear(0)

        # تعيين مصنف خارجي للرسم البياني
        chart.chart_data.set_external_workbook("YOUR_OUTPUT_DIRECTORY/charts_external_workbook.xlsx")

        # قم بتحديد نطاق الخلايا من "Sheet1" لاستخدامه كمصدر للبيانات
        chart.chart_data.set_range("Sheet1!$A$2:$B$5")

        # تعيين اختلاف اللون للسلسلة الأولى في الرسم البياني
        series = chart.chart_data.series[0]
        series.parent_series_group.is_color_varied = True

        # حفظ العرض التقديمي باسم وتنسيق محددين
        pres.save("YOUR_OUTPUT_DIRECTORY/charts_create_external_workbook_out.pptx", slides.export.SaveFormat.PPTX)
```
- **حدود**:
  - `slides.charts.ChartType`:يحدد نوع الرسم البياني.
  - `set_external_workbook(path)`:يحدد المسار إلى المصنف الخارجي الخاص بك.
  - `set_range(range_string)`: يحدد الخلايا التي سيتم استخدامها للبيانات في Excel.

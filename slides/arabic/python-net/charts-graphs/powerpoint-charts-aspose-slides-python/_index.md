# أتمتة إنشاء مخططات PowerPoint باستخدام Aspose.Slides لـ Python - دليل خطوة بخطوة

إن أتمتة إنشاء المخططات في PowerPoint تُحسّن بشكل ملحوظ التأثير البصري لعرضك التقديمي، مع توفير الوقت في مهام تصور البيانات يدويًا. يُركز هذا الدليل الشامل على استخدام Aspose.Slides لـ Python لإنشاء وتخصيص المخططات ضمن عروض PowerPoint التقديمية، وهو مثالي للمطورين الذين يتطلعون إلى تبسيط سير عملهم.

قد يكون عرض مجموعات البيانات المعقدة بصريًا دون إنشاء كل مخطط يدويًا في PowerPoint مهمة شاقة. مع Aspose.Slides لـ Python، يمكنك أتمتة هذه العملية بكفاءة. يغطي هذا البرنامج التعليمي بشكل أساسي إنشاء مخططات عمودية مجمعة - وهو خيار شائع لتصور البيانات المقارنة - باستخدام Aspose.Slides.

**ما سوف تتعلمه:**
- قم بإعداد العروض التقديمية باستخدام الرسوم البيانية باستخدام Aspose.Slides.
- تنسيق أرقام سلسلة المخططات بشكل فعال.
- احفظ عروض PowerPoint الخاصة بك وقم بتصديرها بسلاسة.

بنهاية هذا الدليل، ستتمكن من أتمتة إنشاء المخططات البيانية في PowerPoint، مما يجعل عروض بياناتك التقديمية أكثر كفاءةً واحترافية. لنبدأ بمناقشة المتطلبات الأساسية لهذا التنفيذ.

## دليل التنفيذ
يقوم هذا القسم بتقسيم كل ميزة إلى خطوات قابلة للتنفيذ، مما يرشدك خلال عملية إنشاء المخطط وتخصيصه.

### الميزة 1: تهيئة العرض التقديمي وإنشاء المخطط
#### ملخص
قم بإنشاء عرض تقديمي جديد في PowerPoint وأضف مخططًا عموديًا مجمعًا في موضع محدد.

#### خطوات:
##### **تهيئة العرض التقديمي**
ابدأ بإنشاء مثيل لـ `Presentation`:
```python
import aspose.slides as slides

def initialize_presentation_and_add_chart():
    with slides.Presentation() as pres:
        slide = pres.slides[0]
```

##### **إضافة مخطط عمودي مجمع**
استخدم `add_chart()` الطريقة. حدد نوعها وموقعها وأبعادها:
```python
chart = slide.shapes.add_chart(
    slides.charts.ChartType.CLUSTERED_COLUMN,
    50, 50, 500, 400
)
```
**توضيح**:يضع هذا الكود مخططًا عموديًا مجمعًا عند الإحداثيات (50، 50) بعرض 500 بكسل وارتفاع 400 بكسل.

##### **إرجاع العرض التقديمي**
أخيرًا، قم بإرجاع كائن العرض لمزيد من المعالجة:
```python
return pres
```

### الميزة 2: تنسيق أرقام سلسلة المخططات
#### ملخص
تنسيق الأرقام في سلسلة المخططات باستخدام التنسيقات المحددة مسبقًا.

#### خطوات:
##### **مخطط الوصول والسلسلة**
انتقل عبر أشكال الشريحة لتحديد مخططك وسلسلته:
```python
def format_chart_number(pres):
    slide = pres.slides[0]
    chart = slide.shapes[0] if len(slide.shapes) > 0 else None
    
    if chart is not None and isinstance(chart, slides.charts.Chart):
        series = chart.chart_data.series
```

##### **تعيين تنسيق الأرقام**
قم بالتكرار على كل نقطة بيانات في السلسلة لتطبيق تنسيق مثل "0.00%":
```python
for ser in series:
    for cell in ser.data_points:
        cell.value.as_cell.preset_number_format = 10  # 10 يتوافق مع 0.00%
```
**توضيح**:تعمل هذه الحلقة على تنسيق جميع نقاط البيانات ضمن كل سلسلة لعرضها كنسب مئوية مع منزلتين عشريتين.

### الميزة 3: حفظ العرض التقديمي
#### ملخص
بمجرد أن يصبح عرضك التقديمي جاهزًا، احفظه بتنسيق PPTX.

#### خطوات:
##### **تحديد مسار الإخراج**
حدد المكان الذي تريد حفظ الملف فيه:
```python
def save_presentation(pres):
    output_path = "YOUR_OUTPUT_DIRECTORY/charts_number_format_out.pptx"
```

##### **حفظ العرض التقديمي**
استخدم `save()` طريقة كتابة العرض التقديمي على القرص:
```python
pres.save(output_path, slides.export.SaveFormat.PPTX)
```
**توضيح**:يحفظ هذا الكود العرض التقديمي بتنسيق PowerPoint في المسار المحدد.

### الوصول إلى المخططات وتعديلها

#### ملخص
تتيح لك هذه الميزة تعيين نطاق البيانات للمخططات المضمنة في عروض PowerPoint التقديمية بشكل برمجي، وربطها بمصنفات Excel الخارجية إذا لزم الأمر.

#### الخطوة 1: تحميل العرض التقديمي الخاص بك
ابدأ بتحميل ملف العرض التقديمي الخاص بك:

```python
# إعدادات المسار
input_document_path = 'YOUR_DOCUMENT_DIRECTORY/charts_with_external_workbook.pptx'

# تحميل العرض التقديمي
class PresentationManager:
    def __init__(self, path):
        self.presentation = slides.Presentation(path)

    def get_first_chart(self):
        slide = self.presentation.slides[0]
        chart = slide.shapes[0] if isinstance(slide.shapes[0], slides.Chart) else None
        return chart

def main():
    manager = PresentationManager(input_document_path)
    chart = manager.get_first_chart()
    if chart:
        # متابعة إعداد نطاق البيانات
```

**توضيح**: 
- نقوم بتحميل ملف PPTX باستخدام `slides.Presentation()`.
- يتم الوصول إلى الشريحة الأولى باستخدام `presentation.slides[0]`، متبوعًا باسترجاع الشكل الأول المفترض أنه مخطط، والتأكد من أنه مخطط بالفعل `isinstance()` يفحص.

#### الخطوة 2: تعيين نطاق البيانات للرسم البياني
تحديد نطاق البيانات داخل مصنف خارجي:

```python
# تعيين نطاق البيانات من مصنف خارجي
def set_chart_data_range(chart, range_string):
    if isinstance(chart, slides.Chart):
        chart.chart_data.set_range(range_string)
    else:
        raise ValueError("Provided shape is not a chart.")

set_chart_data_range(chart, 'Sheet1!A1:B4')
```

**توضيح**: 
- `set_range()` يحدد الخلايا التي سيتم استخدامها كمصدر للبيانات في ملف Excel الخارجي.
- الحجة `'Sheet1!A1:B4'` يشير إلى أننا نستخدم نطاقًا من Sheet1 يبدأ من الخلية A1 وينتهي في B4.

#### الخطوة 3: حفظ العرض التقديمي المعدّل
وأخيرًا، احفظ التغييرات:

```python
# إعدادات الإخراج
def save_presentation(presentation_manager, output_directory_path='YOUR_OUTPUT_DIRECTORY/', output_file_name='charts_set_data_range_out.pptx'):
    presentation_manager.presentation.save(
        f"{output_directory_path}{output_file_name}", 
        slides.export.SaveFormat.PPTX
    )

save_presentation(manager)
```

**توضيح**: 
- ال `save()` تكتب الطريقة التغييرات إلى ملف جديد في الدليل المحدد.
- تأكد من تحديد التنسيق الصحيح للحفظ (`slides.export.SaveFormat.PPTX`).

### نصائح استكشاف الأخطاء وإصلاحها
- **خطأ في الشكل وليس الرسم البياني**:تأكد من أن الشكل الذي تحاول الوصول إليه هو في الواقع مخطط باستخدام `isinstance(chart, slides.Chart)`.
- **مشاكل مسار الملف**:تحقق جيدًا من المسارات وأسماء الملفات بحثًا عن الأخطاء المطبعية أو الدلائل غير الصحيحة.

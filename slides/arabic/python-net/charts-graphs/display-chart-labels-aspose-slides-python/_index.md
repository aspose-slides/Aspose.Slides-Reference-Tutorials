# كيفية عرض تسميات المخططات في عروض PowerPoint باستخدام Aspose.Slides لـ Python

## مقدمة

حسّن عروض PowerPoint التقديمية بإضافة تسميات بيانية ثرية بالمعلومات وقابلة للتخصيص باستخدام Aspose.Slides للغة بايثون. سيرشدك هذا البرنامج التعليمي خلال عملية دمج تسميات البيانات في شرائحك، مما يجعل البيانات أكثر سهولة في الوصول إليها وجاذبية بصريًا.

**ما سوف تتعلمه:**
- إعداد Aspose.Slides لـ Python في بيئتك
- إنشاء عرض تقديمي باستخدام مخطط دائري
- تكوين وتخصيص خصائص التسمية على سلسلة المخططات
- حفظ العرض التقديمي المعزز

## المتطلبات الأساسية
قبل البدء، تأكد من أن لديك:

## دليل التنفيذ
اتبع الخطوات التالية لعرض تسميات المخطط في عرض تقديمي في PowerPoint.

### الخطوة 1: إنشاء عرض تقديمي وشريحة جديدة
إنشاء عرض تقديمي جديد وإضافة شريحة:

```python
def display_chart_labels():
    with slides.Presentation() as presentation:
        # قم بالوصول إلى الشريحة الأولى (يتم إنشاء شريحة واحدة بشكل افتراضي).
        slide = presentation.slides[0]
```

### الخطوة 2: إضافة مخطط دائري إلى الشريحة
أضف مخططًا دائريًا في الموضع `(50, 50)` مع الأبعاد `500x400`:

```python
        # إضافة مخطط دائري إلى الشريحة الأولى.
        chart = slide.shapes.add_chart(
            slides.charts.ChartType.PIE, 50, 50, 500, 400)
```

### الخطوة 3: تكوين خيارات عرض الملصق
قم بتكوين خصائص الملصق لتحسين تصور البيانات:
- **إظهار ملصقات القيمة**:عرض القيم العددية على كل شريحة.
- **بيانات الاستدعاء**:استخدم خطوط التعليق لتوصيل العلامات بالشرائح.

```python
        # تكوين خيارات عرض تسمية سلسلة الرسم البياني
        series_labels = chart.chart_data.series[0].labels.default_data_label_format
        series_labels.show_value = True  # إظهار تسميات القيمة بشكل افتراضي
        series_labels.show_label_as_data_callout = True  # استخدام مكالمات البيانات
```

### الخطوة 4: تخصيص تسميات محددة
تعطيل استدعاء البيانات لملصقات محددة، مثل الملصق الثالث:

```python
        # تجاوز إعداد استدعاء البيانات لعلامة محددة
        chart.chart_data.series[0].labels[2].data_label_format.show_label_as_data_callout = False
```

### الخطوة 5: حفظ العرض التقديمي
احفظ العرض التقديمي الخاص بك في دليل الإخراج بالاسم المطلوب:

```python
        # حفظ العرض التقديمي المعزز
        presentation.save("YOUR_OUTPUT_DIRECTORY/charts_display_chart_labels_out.pptx")
```

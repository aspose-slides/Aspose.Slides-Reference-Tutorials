## دليل التنفيذ

### إنشاء مخطط فقاعي ديناميكي

لنبدأ في بناء مخطط فقاعي ديناميكي في PowerPoint باستخدام Aspose.Slides. تتيح لك هذه الميزة تصوّر نقاط البيانات بأحجام مختلفة، مما يجعلها مثالية لمقارنة أبعاد متعددة لمجموعات البيانات.

#### إضافة الرسم البياني

**الخطوة 1: تهيئة العرض التقديمي**

ابدأ بإنشاء أو فتح عرض تقديمي حيث سيتم إضافة الرسم البياني:

```python
with slides.Presentation() as pres:
    slide = pres.slides[0]  # الوصول إلى الشريحة الأولى
```

**الخطوة 2: إضافة مخطط فقاعي ديناميكي**

أضف مخطط الفقاعات الديناميكي إلى الشريحة المحددة عند إحداثيات محددة بأبعاد محددة:

```python
chart = slide.shapes.add_chart(
    slides.charts.ChartType.BUBBLE, 100, 100, 400, 300
)
```

يؤدي مقتطف التعليمات البرمجية هذا إلى إنشاء مخطط فقاعي ديناميكي يقع عند (100، 100) على الشريحة بعرض 400 وارتفاع 300.

#### ضبط مقياس حجم الفقاعة

**الخطوة 3: تعيين حجم الفقاعة**

قم بضبط تصور البيانات لديك عن طريق ضبط مقياس الحجم للفقاعات في المجموعة الأولى من السلسلة:

```python
chart.chart_data.series_groups[0].bubble_size_scale = 150
```

يقوم هذا التعديل بتغيير أحجام الفقاعات، مما يعزز الوضوح والتأثير البصري.

#### حفظ العرض التقديمي الخاص بك

**الخطوة 4: حفظ الملف**

بعد إجراء التعديلات الخاصة بك، احفظ العرض التقديمي للحفاظ على التغييرات التي أجريتها:

```python
pres.save('dynamic_bubble_chart_scaling_out.pptx', slides.export.SaveFormat.PPTX)
```

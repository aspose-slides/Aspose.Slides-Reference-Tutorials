# إنشاء مخططات أعمدة مجمعة في PowerPoint باستخدام Aspose.Slides لـ Python

## مقدمة

حسّن عروضك التقديمية بإضافة مخططات بيانية ثاقبة بسهولة. سيرشدك هذا البرنامج التعليمي إلى كيفية إنشاء مخطط عمودي مجمع في PowerPoint باستخدام Aspose.Slides لـ Python. تعلّم كيفية ضبط إعدادات المحور الأفقي بكفاءة، مما يوفر الوقت ويحسّن جودة العرض التقديمي.

**ما سوف تتعلمه:**
- إعداد Aspose.Slides لـ Python
- إنشاء مخطط عمودي مجمع في شريحة PowerPoint
- تكوين محاور الرسم البياني بدقة

دعونا نتعمق في المتطلبات الأساسية قبل أن نبدأ!

للبدء، ستحتاج إلى تثبيت مكتبة Aspose.Slides لـ Python باستخدام pip:

```bash

```

import aspose.slides as slides

## دليل التنفيذ

سيقوم هذا القسم بتقسيم العملية إلى خطوات قابلة للإدارة لإنشاء مخطط عمودي مجمع وتكوينه في PowerPoint.

### إضافة مخطط عمودي مجمع

**ملخص:** سنبدأ بإنشاء مخطط عمودي مجمع أساسي ضمن شريحة العرض التقديمي الخاصة بك.

#### الخطوة 1: تهيئة العرض التقديمي

أولاً، افتح أو أنشئ كائن عرض تقديمي جديد:

```python
with slides.Presentation() as pres:
    # الوصول إلى الشريحة الأولى
    slide = pres.slides[0]
```

#### الخطوة 2: إضافة الرسم البياني

أضف مخططًا عموديًا مجمعًا عند الإحداثيات والأبعاد المحددة (50، 50) بعرض 450 وارتفاع 300:

```python
chart = slide.shapes.add_chart(
    slides.charts.ChartType.CLUSTERED_COLUMN, 
    50, 50, 450, 300
)
```

#### الخطوة 3: تكوين المحور الأفقي

اضبط المحور الأفقي لعرض الفئات بين نقاط البيانات لتحقيق وضوح أفضل:

```python
chart.axes.horizontal_axis.axis_between_categories = True
```

### حفظ العرض التقديمي الخاص بك

وأخيرًا، احفظ عرضك التقديمي باستخدام الرسم البياني المضاف حديثًا:

```python
pres.save("YOUR_OUTPUT_DIRECTORY/charts_setting_position_axis_out.pptx", slides.export.SaveFormat.PPTX)
```

**نصائح استكشاف الأخطاء وإصلاحها:**
- تأكد من ذلك `YOUR_OUTPUT_DIRECTORY` موجود أو قم بتعديل المسار وفقًا لذلك.
- التحقق من تثبيت Aspose.Slides وتوافق الإصدار.

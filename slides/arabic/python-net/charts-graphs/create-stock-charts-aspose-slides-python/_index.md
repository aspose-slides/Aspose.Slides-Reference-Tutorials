## إعداد Aspose.Slides لـ Python
للبدء، تأكد من تثبيت مكتبة Aspose.Slides باستخدام الأمر pip المذكور أعلاه.

بمجرد التثبيت، قم بتهيئة مكتبة Aspose.Slides في البرنامج النصي Python الخاص بك:

```python
import aspose.slides as slides

# تهيئة Aspose.Slides
pres = slides.Presentation()
```

## دليل التنفيذ
في هذا القسم، سنقوم بتفصيل كل خطوة مطلوبة لإنشاء مخطط الأسهم وتخصيصه.

### إضافة مخطط الأسهم
أولاً، دعنا نضيف مخطط الأسهم إلى العرض التقديمي الخاص بك:

```python
with slides.Presentation() as pres:
    # أضف مخطط الأسهم في الموضع (50، 50) بالحجم (600، 400)
    chart = pres.slides[0].shapes.add_chart(slides.charts.ChartType.OPEN_HIGH_LOW_CLOSE, 50, 50, 600, 400, False)

    # مسح البيانات الموجودة
    chart.chart_data.series.clear()
    chart.chart_data.categories.clear()

    # الوصول إلى مصنف العمل الخاص بالتلاعب بالخلايا
    wb = chart.chart_data.chart_data_workbook
```

### تكوين الفئات والسلاسل
بعد ذلك، سنقوم بتكوين الفئات والسلاسل لحمل بيانات الأسهم الخاصة بك:

```python
# إضافة الفئات (أ، ب، ج)
chart.chart_data.categories.add(wb.get_cell(0, 1, 0, "A"))
chart.chart_data.categories.add(wb.get_cell(0, 2, 0, "B"))
chart.chart_data.categories.add(wb.get_cell(0, 3, 0, "C"))

# إضافة سلسلة لبيانات الافتتاح والارتفاع والانخفاض والإغلاق
series_names = ["Open", "High", "Low", "Close"]
for i, name in enumerate(series_names):
    chart.chart_data.series.add(wb.get_cell(0, 0, i + 1, name), chart.type)
```

### إضافة نقاط البيانات
الآن، دعونا نملأ السلسلة بنقاط البيانات:

```python
# بيانات "الافتتاح"، و"الارتفاع"، و"الانخفاض"، و"الإغلاق"
data = [
    [72, 172, 12, 25],
    [25, 57, 12, 38],
    [38, 57, 13, 50]
]

# تعيين البيانات لكل سلسلة
for i in range(4):
    series = chart.chart_data.series[i]
    for j in range(3):
        series.data_points.add_data_point_for_stock_series(wb.get_cell(0, j + 1, i + 1, data[j][i]))
```

### تخصيص مظهر الرسم البياني
قم بتعزيز المظهر البصري لمخطط أسهمك:

```python
# تمكين أشرطة الأعلى والأسفل وتعيين تنسيق الخط المرتفع والمنخفض
chart.chart_data.series_groups[0].up_down_bars.has_up_down_bars = True
chart.chart_data.series_groups[0].hi_low_lines_format.line.fill_format.fill_type = slides.FillType.SOLID

# اضبط خطوط السلسلة على عدم التعبئة للحصول على مظهر أنظف
for ser in chart.chart_data.series:
    ser.format.line.fill_format.fill_type = slides.FillType.NO_FILL
```

### حفظ العرض التقديمي
وأخيرًا، احفظ العرض التقديمي الخاص بك باستخدام مخطط الأسهم الذي تم إنشاؤه حديثًا:

```python
# حفظ العرض التقديمي على القرص
pres.save("charts_stock_chart_out.pptx", slides.export.SaveFormat.PPTX)
```

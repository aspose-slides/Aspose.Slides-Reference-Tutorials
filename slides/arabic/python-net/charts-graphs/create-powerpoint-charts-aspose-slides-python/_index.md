## دليل التنفيذ

دعونا نتعمق في تفاصيل معالجة المخططات باستخدام Aspose.Slides.

### إنشاء مخطط عمودي مجمع

**ملخص:**  
يركز هذا القسم على إضافة مخطط عمودي مجمع إلى عرض PowerPoint الخاص بك وتخصيص مظهره وبياناته.

#### إضافة مخطط عمودي مجمع

```python
# أضف مخططًا عموديًا مجمعًا عند إحداثيات محددة (x: 50، y: 50) بعرض 600 وارتفاع 400.
chart = pres.slides[0].shapes.add_chart(
    slides.charts.ChartType.CLUSTERED_COLUMN, 50, 50, 600, 400, True
)
```

#### الوصول إلى مجموعة السلسلة ومسحها

```python
# احصل على مجموعة السلسلة من بيانات الرسم البياني.
series_collection = chart.chart_data.series
# قم بمسح أي سلسلة موجودة للبدء من جديد.
series_collection.clear()
```

### إضافة نقاط البيانات باستخدام خيارات الانعكاس

**ملخص:**  
في هذا القسم، ستتعلم كيفية إضافة نقاط البيانات إلى سلسلة وإدارة خصائصها، مثل عكس الأشرطة للقيم السلبية.

#### إضافة سلسلة ونقاط بيانات

```python
# أضف سلسلة جديدة إلى الرسم البياني.
series = series_collection.add(
    chart.chart_data.chart_data_workbook.get_cell(0, "B1"), chart.type
)

# أضف نقاط البيانات إلى السلسلة الأولى. بعضها سلبي.
series.data_points.add_data_point_for_bar_series(chart.chart_data.chart_data_workbook.get_cell(0, "B2", -5))
series.data_points.add_data_point_for_bar_series(chart.chart_data.chart_data_workbook.get_cell(0, "B3", 3))
series.data_points.add_data_point_for_bar_series(chart.chart_data.chart_data_workbook.get_cell(0, "B4", -2))
series.data_points.add_data_point_for_bar_series(chart.chart_data.chart_data_workbook.get_cell(0, "B5", 1))
```

#### تخصيص `invert_if_negative` ملكية

```python
# تعيين invert_if_negative على مستوى السلسلة إلى False.
series.invert_if_negative = False

# عكس نقطة البيانات الثالثة على وجه التحديد.
series.data_points[2].invert_if_negative = True
```

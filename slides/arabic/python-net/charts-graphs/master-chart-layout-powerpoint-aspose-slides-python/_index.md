## دليل التنفيذ: إعداد وضع تخطيط الرسم البياني

دعونا نوضح كيفية تعيين وضع تخطيط الرسم البياني ضمن عرض تقديمي في PowerPoint.

### إنشاء شريحة والوصول إليها

ابدأ بإنشاء عرض تقديمي جديد على PowerPoint والوصول إلى الشريحة الأولى منه:

```python
with slides.Presentation() as presentation:
    slide = presentation.slides[0]
```

يؤدي هذا إلى إعداد البيئة الخاصة بك لإضافة المخططات البيانية.

### إضافة مخطط عمودي مجمع

أضف مخططًا عموديًا مجمعًا إلى الموضع المحدد على الشريحة:

```python
chart = slide.shapes.add_chart(
    slides.charts.ChartType.CLUSTERED_COLUMN, 20, 100, 600, 400
)
```

حدود:
- `ChartType.CLUSTERED_COLUMN`:يحدد نوع الرسم البياني.
- `(20, 100)`:إحداثيات x وy حيث يتم وضع الرسم البياني على الشريحة.
- `(600, 400)`:عرض وارتفاع الرسم البياني بالنقاط.

### ضبط خصائص التخطيط

الآن، قم بضبط خصائص تخطيط منطقة الرسم البياني لتعيين موضعها وحجمها:

```python
chart.plot_area.as_i_layoutable.x = 0.2
chart.plot_area.as_i_layoutable.y = 0.2
chart.plot_area.as_i_layoutable.width = 0.7
chart.plot_area.as_i_layoutable.height = 0.7
```

تعتبر هذه القيم وحدات نسبية، مما يضمن أن يتكيف الرسم البياني ديناميكيًا مع أحجام الشرائح المختلفة.

### تحديد نوع هدف التخطيط

قم بتعيين نوع هدف التخطيط للتحكم الدقيق في كيفية سلوك منطقة الرسم البياني:

```python
chart.plot_area.layout_target_type = slides.charts.LayoutTargetType.INNER
```

يضمن هذا التكوين أن تكون منطقة الرسم البياني متمركزة داخل الحاوية الخاصة بها، مما يحافظ على مظهر نظيف.

### احفظ عرضك التقديمي

وأخيرًا، احفظ العرض التقديمي الخاص بك في دليل الإخراج المحدد:

```python
output_directory = 'YOUR_OUTPUT_DIRECTORY/'
presentation.save(output_directory + 'charts_set_layout_mode_out.pptx', slides.export.SaveFormat.PPTX)
```

3. **العروض التقديمية التسويقية**:استخدم تخطيطات المخططات المخصصة لتسليط الضوء على مقاييس التسويق بشكل فعال أثناء العروض التقديمية للعملاء.

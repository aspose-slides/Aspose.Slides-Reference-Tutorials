## دليل التنفيذ

### وحدة العرض على محور الرسم البياني
#### ملخص
تتيح لك هذه الميزة وضع علامات على محاور الرسم البياني باستخدام وحدات مخصصة مثل الملايين أو المليارات، مما يؤدي إلى تحسين قابلية قراءة البيانات في العروض التقديمية.

#### التنفيذ خطوة بخطوة
1. **تهيئة العرض التقديمي**
   ابدأ بإنشاء مثيل عرض تقديمي جديد حيث سيتم إضافة الرسم البياني الخاص بك:

   ```python
   with slides.Presentation() as pres:
       # ستجد هنا الكود الخاص بك للتعامل مع الشرائح والرسوم البيانية
   ```

2. **إضافة مخطط عمودي مجمع**
   أضف مخططًا عموديًا مجمعًا عند الإحداثيات المحددة في الشريحة الأولى:

   ```python
   chart = pres.slides[0].shapes.add_chart(
       slides.charts.ChartType.CLUSTERED_COLUMN, 50, 50, 450, 300
   )
   ```

3. **تعيين وحدة عرض المحور الرأسي**
   قم بتكوين المحور الرأسي لعرض القيم بالملايين:

   ```python
   chart.axes.vertical_axis.display_unit = slides.charts.DisplayUnitType.MILLIONS
   ```

4. **حفظ العرض التقديمي**
   احفظ العرض التقديمي الخاص بك باستخدام الرسم البياني الذي تم تكوينه:

   ```python
   pres.save("YOUR_OUTPUT_DIRECTORY/charts_showing_display_unit_label_out.pptx", slides.export.SaveFormat.PPTX)
   ```

#### المعلمات والطرق
- `add_chart`:يضيف كائن مخطط جديد إلى الشريحة.
- `display_unit`:تعيين وحدة العرض للقيم الرقمية على المحور الرأسي.

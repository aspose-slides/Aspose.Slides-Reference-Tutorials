# كيفية استخراج قيم محاور الرسم البياني باستخدام Aspose.Slides في Python: دليل خطوة بخطوة

## مقدمة

يُمكن لاستخراج قيم محاور الرسم البياني من عروض PowerPoint التقديمية تبسيط تحليل البيانات وتحسين إمكانيات العرض التقديمي. يوضح هذا الدليل كيفية استخدام **Aspose.Slides لـ Python** لاستخراج هذه القيم بكفاءة.

### ما سوف تتعلمه:
- إنشاء عرض تقديمي باستخدام Aspose.Slides.
- إضافة المخططات وتكوينها في الشرائح الخاصة بك.
- استخراج قيم المحور الرأسي (الحد الأقصى والحد الأدنى).
- الحصول على مقاييس وحدات المحور الأفقي (وحدات رئيسية وثانوية).

قبل الغوص في البرنامج التعليمي، دعنا نراجع المتطلبات الأساسية اللازمة للبدء.

## المتطلبات الأساسية

لمتابعة هذا الدليل، تأكد من أن لديك:
- **بايثون 3.x** تم تثبيته على نظامك.
- فهم أساسي لبرمجة بايثون.
- مكتبة Aspose.Slides لبايثون. ثبّتها باستخدام pip كما هو موضح أدناه.

لبدء استخدام Aspose.Slides، قم بإعداد بيئتك باتباع الخطوات التالية:

1. **تثبيت:**
   استخدم الأمر أدناه في محطتك الطرفية أو موجه الأوامر:

## دليل التنفيذ

### استخراج قيم محور الرسم البياني

اتبع الخطوات التالية لاستخراج قيم المحور من مخطط باستخدام Aspose.Slides.

#### الخطوة 1: إنشاء العرض التقديمي وتكوينه

ابدأ بإنشاء مثيل عرض تقديمي جديد وإضافة مخطط منطقة إلى الشريحة الأولى:
```python
with slides.Presentation() as pres:
    # إضافة مخطط منطقة إلى الشريحة الأولى
    chart = pres.slides[0].shapes.add_chart(slides.charts.ChartType.AREA, 100, 100, 500, 350)
```

#### الخطوة 2: التحقق من صحة تخطيط الرسم البياني

تأكد من إعداد تخطيط الرسم البياني الخاص بك بشكل صحيح قبل استخراج القيم:
```python
chart.validate_chart_layout()
```
تضمن هذه الخطوة أن بيانات الرسم البياني وتكوينه جاهزة لاستخراج القيمة.

#### الخطوة 3: استخراج قيم المحور

استرداد القيم القصوى والدنيا من المحور الرأسي ومقاييس الوحدة من المحور الأفقي:
```python
# قيم المحور الرأسي
max_value = chart.axes.vertical_axis.actual_max_value
min_value = chart.axes.vertical_axis.actual_min_value

# مقاييس وحدة المحور الأفقي
major_unit = chart.axes.horizontal_axis.actual_major_unit
minor_unit = chart.axes.horizontal_axis.actual_minor_unit
```

#### الخطوة 4: عرض القيم المستخرجة

اطبع هذه القيم للتحقق من عملية الاستخراج:
```python
print(f"Max Value: {max_value}, Min Value: {min_value}")
print(f"Major Unit: {major_unit}, Minor Unit: {minor_unit}")
```

### حفظ العرض التقديمي الخاص بك

احفظ العرض التقديمي الخاص بك مع تطبيق كافة التكوينات:
```python
pres.save("YOUR_OUTPUT_DIRECTORY/charts_get_values_and_unit_scale_from_axis_out.pptx", slides.export.SaveFormat.PPTX)
```
يستبدل `"YOUR_OUTPUT_DIRECTORY"` مع المسار الذي تريد حفظ الملف فيه.

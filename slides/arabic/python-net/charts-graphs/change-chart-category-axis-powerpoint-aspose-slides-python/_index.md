## دليل التنفيذ

سنركز على تعديل محور فئة الرسم البياني. لنشرح العملية خطوة بخطوة.

### الوصول إلى العرض التقديمي والرسم البياني

ابدأ بتحميل ملف العرض التقديمي. تأكد من معرفة مسار المستند:

```python
def change_chart_category_axis():
    data_dir = "YOUR_DOCUMENT_DIRECTORY/"
    
    with slides.Presentation(data_dir + "charts_existing_chart.pptx") as presentation:
        chart = presentation.slides[0].shapes[0]
```

يفتح هذا المقطع ملف PowerPoint ويسمح بالوصول إلى الشكل الأول للشريحة الأولى، على افتراض أنه يحتوي على مخطط.

### تعديل محور الفئة

بعد ذلك، قم بتغيير نوع محور الفئة إلى DATE:

```python
chart.axes.horizontal_axis.category_axis_type = slides.charts.CategoryAxisType.DATE
```

يؤدي تعيين نوع المحور إلى DATE إلى ضمان توافق بياناتك مع تواريخ التقويم، مما يعزز قابلية قراءة بيانات السلسلة الزمنية.

### تكوين خصائص المحور

تخصيص المحور الأفقي عن طريق تعيين الوحدات والمقاييس الرئيسية:

```python
chart.axes.horizontal_axis.is_automatic_major_unit = False
chart.axes.horizontal_axis.major_unit = 1
chart.axes.horizontal_axis.major_unit_scale = slides.charts.TimeUnitType.MONTHS
```

من خلال تعطيل حساب الوحدة الرئيسية التلقائي، يمكنك التحكم في كيفية توزيع نقاط البيانات على المحور. `major_unit` يحدد فترات زمنية (على سبيل المثال، كل شهر)، بينما `major_unit_scale` وينص على أن هذه الوحدات تمثل الأشهر.

### حفظ التغييرات

وأخيرًا، احفظ العرض التقديمي المعدّل:

```python
out_dir = "YOUR_OUTPUT_DIRECTORY/"
presentation.save(out_dir + "charts_change_chart_category_axis_out.pptx", slides.export.SaveFormat.PPTX)
```

تؤدي هذه الخطوة إلى كتابة التغييرات مرة أخرى في ملف جديد في دليل الإخراج المحدد.

# إنشاء وتخصيص مخططات TreeMap باستخدام Aspose.Slides لـ Python

## مقدمة
يُعد إنشاء مخططات بيانية جذابة بصريًا أمرًا بالغ الأهمية عند عرض هياكل بيانات معقدة بأشكال هرمية مثل خرائط الشجرة. يرشدك هذا البرنامج التعليمي إلى كيفية استخدام Aspose.Slides لـ Python لإنشاء وتكوين مخطط TreeMap، وهو أداة تصور فعّالة لعرض فئات البيانات المتداخلة بكفاءة.

**ما سوف تتعلمه:**
- إعداد البيئة الخاصة بك باستخدام Aspose.Slides لـ Python.
- خطوات تهيئة مخطط TreeMap وإضافته إلى العرض التقديمي الخاص بك.
- طرق تخصيص مظهر الرسم البياني وبياناته.
- حالات استخدام عملية حيث يثبت مخطط TreeMap أنه مفيد.
- نصائح لتحسين الأداء عند العمل مع مجموعات بيانات كبيرة.

هل أنت مستعد للبدء؟ لنبدأ بتغطية المتطلبات الأساسية التي ستحتاجها قبل البدء.

## المتطلبات الأساسية
لمتابعة هذا البرنامج التعليمي، تأكد من أن لديك:
- **تم تثبيت Python:** يوصى باستخدام الإصدار 3.6 أو الإصدار الأحدث للتوافق مع Aspose.Slides.
- **تم تثبيت Pip:** سيتم استخدام Pip لتثبيت الحزم الضرورية.
- **المعرفة الأساسية بلغة بايثون:** المعرفة بالبرمجة الكائنية التوجه في بايثون ومفاهيم المخططات الأساسية.

بالإضافة إلى ذلك، ستحتاج إلى بيئة يمكنك من خلالها تشغيل نصوص Python - يمكن أن يكون هذا إعدادًا محليًا أو بيئة تطوير متكاملة (IDE) مثل PyCharm أو VS Code.

## دليل التنفيذ
دعنا نستعرض الخطوات المطلوبة لإنشاء مخطط TreeMap وتكوينه.

### إضافة مخطط TreeMap إلى شريحة

#### ملخص
يُعد مخطط TreeMap مثاليًا لعرض البيانات الهرمية بصريًا. فهو يُجمّع البيانات في مستطيلات تختلف أحجامها وفقًا لقيمها، مما يُسهّل مقارنة الأجزاء المختلفة في لمحة واحدة.

#### خطوات إضافة مخطط TreeMap
1. **تهيئة العرض التقديمي:**
   ابدأ بإنشاء مثيل لـ `Presentation` فصل:
   ```python
   import aspose.slides as slides
   
   with slides.Presentation() as pres:
       # سيتم وضع الكود الخاص بإضافة المخططات هنا
   ```
2. **إضافة مخطط TreeMap:**
   استخدم `add_chart()` طريقة وضع الرسم البياني الخاص بك على الشريحة الأولى عند الإحداثيات والأبعاد المحددة:
   ```python
   chart = pres.slides[0].shapes.add_chart(
       slides.charts.ChartType.TREEMAP, 50, 50, 500, 400)
   ```
   سيؤدي هذا إلى إنشاء خريطة شجرة بعرض 500 بكسل وارتفاع 400 بكسل عند الإحداثيات (50، 50).
3. **مسح البيانات الموجودة:**
   قبل إضافة بيانات جديدة، تأكد من مسح الفئات والسلاسل الموجودة:
   ```python
   chart.chart_data.categories.clear()
   chart.chart_data.series.clear()
   
   wb = chart.chart_data.chart_data_workbook
   wb.clear(0)
   ```
### تكوين فئات المخطط
#### ملخص
يعد تنظيم بياناتك في مجموعات هرمية أمرًا بالغ الأهمية للحصول على تمثيل TreeMap مفيد.
#### خطوات تكوين الفئات
1. **إضافة وتجميع الفئات:**
   قم بتحديد الفئات ومستوياتها الهرمية باستخدام `grouping_levels` يصف:
   ```python
   leaf = chart.chart_data.categories.add(wb.get_cell(0, "C1", "Leaf1"))
   leaf.grouping_levels.set_grouping_item(1, "Stem1")
   leaf.grouping_levels.set_grouping_item(2, "Branch1")
   
   # كرر ذلك للفئات الأخرى حسب الحاجة
   ```
   يقوم هذا الكود بتعيين "Leaf1" إلى التسلسل الهرمي مع "Stem1" و"Branch1".
### إضافة السلاسل ونقاط البيانات
#### ملخص
تُمثل نقاط البيانات قيمًا فردية في خريطة الشجرة. يُحسّن ربطها بشكل صحيح سهولة قراءة الرسم البياني.
#### خطوات إضافة نقاط البيانات
1. **إنشاء سلسلة جديدة:**
   قم بتهيئة سلسلة لبياناتك:
   ```python
   series = chart.chart_data.series.add(slides.charts.ChartType.TREEMAP)
   ```
2. **تكوين العلامات:**
   تعيين خيارات التسمية لتحسين الوضوح:
   ```python
   series.labels.default_data_label_format.show_category_name = True
   ```
3. **إضافة نقاط البيانات:**
   املأ سلسلتك بالقيم المقابلة لكل فئة:
   ```python
   data_points = [4, 5, 3, 6, 9, 9, 4, 3]
   cells = [("D1", 4), ("D2", 5), ("D3", 3), ("D4", 6),
            ("D5", 9), ("D6", 9), ("D7", 4), ("D8", 3)]
   
   for cell, value in zip(cells, data_points):
       series.data_points.add_data_point_for_treemap_series(
           wb.get_cell(0, *cell))
   ```
### الانتهاء والحفظ
#### ملخص
بعد تكوين الرسم البياني الخاص بك، احفظ العرض التقديمي في ملف.
#### خطوات التوفير
1. **حفظ العرض التقديمي:**
   استخدم `save()` طريقة تخزين عملك:
   ```python
   pres.save("YOUR_OUTPUT_DIRECTORY/charts_tree_map_chart_out.pptx", 
             slides.export.SaveFormat.PPTX)
   ```
تضمن هذه الخطوة حفظ الرسم البياني الخاص بك بتنسيق PPTX، وجاهزيته للمشاركة أو التحرير الإضافي.

#### ملخص
يضمن إنشاء المخططات برمجيًا الاتساق والكفاءة، خاصةً عند التعامل مع مصادر البيانات الديناميكية أو مجموعات البيانات الكبيرة.

#### خطوات التنفيذ

##### الخطوة 1: إنشاء عرض تقديمي جديد
ابدأ بإنشاء نموذج عرض تقديمي جديد. سيكون هذا النموذج بمثابة حاوية لشرائحك وأشكالك.

```python
import aspose.slides as slides

    # تهيئة عرض تقديمي جديد

        # وسوف تتبع الخطوات التالية هنا...
```

##### الخطوة 2: إضافة مخطط عمودي مجمع
أضف مخططًا عموديًا مجمعًا إلى الشريحة الأولى عند الإحداثيات والأبعاد المحددة.

```python
        # أضف مخططًا إلى الشريحة الأولى
        chart = pres.slides[0].shapes.add_chart(slides.charts.ChartType.CLUSTERED_COLUMN, 50, 50, 600, 400)
```

هنا، `ChartType.CLUSTERED_COLUMN` يُحدد نوع الرسم البياني. المعلمات `50, 50, 600, 400` تشير إلى موضع x، وموضع y، والعرض، والارتفاع على التوالي.

##### الخطوة 3: الحصول على صورة الرسم البياني وحفظها
بمجرد إنشاء الرسم البياني، يمكنك استخراجه كصورة وحفظه في دليل محدد.

```python
        # استرجاع صورة الرسم البياني
        img = chart.get_image()
        
        # حفظ ملف الصورة
        img.save('YOUR_OUTPUT_DIRECTORY/charts_get_chart_image_out.png', slides.ImageFormat.PNG)
```

يستبدل `'YOUR_OUTPUT_DIRECTORY'` مع مسار الإخراج المطلوب. `get_image()` تلتقط الطريقة التمثيل المرئي للرسم البياني.

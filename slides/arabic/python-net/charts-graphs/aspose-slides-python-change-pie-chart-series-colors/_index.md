# كيفية تغيير ألوان سلسلة المخططات الدائرية في بايثون باستخدام Aspose.Slides: دليل خطوة بخطوة

- التطبيقات العملية لهذه المهارات

## دليل التنفيذ

دعونا نقوم بتحليل عملية تغيير ألوان السلسلة في مخطط دائري باستخدام Aspose.Slides لـ Python.

### إنشاء مخطط دائري

**ملخص:**
إضافة مخطط دائري إلى عرضك التقديمي هي خطوتنا الأولى. سنضعه عند إحداثيات محددة وأبعاد محددة.

#### إضافة مخطط دائري

```python
# إنشاء مثيل للعرض التقديمي
with slides.Presentation() as pres:
    # أضف مخططًا دائريًا موضوعًا عند (50، 50) بعرض 600 وارتفاع 400
    chart = pres.slides[0].shapes.add_chart(slides.charts.ChartType.PIE, 50, 50, 600, 400)
```

**توضيح:** 
هنا، `add_chart` يُستخدم لإدراج مخطط دائري على الشريحة الأولى. تُحدد المعلمات موضعه وحجمه.

### الوصول إلى نقاط البيانات

**ملخص:**
بعد ذلك، نقوم بالوصول إلى نقاط بيانات محددة ضمن سلسلتنا للتخصيص.

#### احصل على نقطة البيانات الثانية من السلسلة الأولى

```python
# الوصول إلى نقطة البيانات الثانية من السلسلة الأولى
point = chart.chart_data.series[0].data_points[1]
```

**توضيح:** 
`chart.chart_data.series[0]` يصل إلى السلسلة الأولى، و `.data_points[1]` يختار نقطة البيانات الثانية الخاصة به.

### تخصيص لون السلسلة

**ملخص:**
سوف نقوم بتغيير لون التعبئة لنقطة البيانات المحددة لجعلها بارزة.

#### تعيين تأثير الانفجار وتغيير نوع التعبئة

```python
# تعيين تأثير الانفجار للتأكيد
point.explosion = 30

# تغيير نوع التعبئة إلى لون ثابت وتعيين اللون إلى اللون الأزرق
point.format.fill.fill_type = slides.FillType.SOLID
point.format.fill.solid_fill_color.color = drawing.Color.blue
```

**توضيح:** 
ال `explosion` تفصل الخاصية نقطة البيانات، بينما `fill_type` تم ضبطه على `SOLID`، مما يسمح لنا بتحديد لون معين باستخدام `solid_fill_color`.

#### احفظ عرضك التقديمي

وأخيرًا، احفظ عرضك التقديمي مع جميع التعديلات:

```python
# حفظ العرض التقديمي مع التغييرات
pres.save("YOUR_OUTPUT_DIRECTORY/charts_changing_series_color_out.pptx", slides.export.SaveFormat.PPTX)
```

**توضيح:** 
يؤدي هذا إلى حفظ عملك في ملف في الدليل المحدد.

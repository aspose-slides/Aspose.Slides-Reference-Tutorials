# ترقيم الشرائح بكفاءة في PowerPoint باستخدام Aspose.Slides لـ Python

## دليل التنفيذ

### عرض وتعيين رقم الشريحة

#### ملخص:
تتيح لك هذه الميزة تحميل عرض تقديمي في PowerPoint، واسترجاع رقم الشريحة الأولى وتعديله، ثم حفظ التغييرات بفعالية.

#### خطوات:

##### الخطوة 1: تحديد مسارات الملفات
ابدأ بتحديد مسارات ملفات الإدخال والإخراج. استبدل العناصر النائبة بأسماء المجلدات الفعلية.

```python
input_path = "YOUR_DOCUMENT_DIRECTORY/welcome-to-powerpoint.pptx"
output_path = "YOUR_OUTPUT_DIRECTORY/rendering_set_slide_number_out.pptx"
```

##### الخطوة 2: تحميل العرض التقديمي

يستخدم `slides.Presentation` لتحميل ملف PowerPoint الخاص بك. يضمن مدير السياق هذا تحرير الموارد عند الانتهاء.

```python
with slides.Presentation(input_path) as presentation:
    # متابعة التلاعب بأرقام الشريحة
```

##### الخطوة 3: استرداد رقم الشريحة وتعديله

استرداد رقم الشريحة الأولى الحالية للتحقق، ثم تعيين قيمة جديدة:

```python
first_slide_number = presentation.first_slide_number
print(f"Original First Slide Number: {first_slide_number}")

presentation.first_slide_number = 10
print("First slide number set to 10.")
```

##### الخطوة 4: حفظ العرض التقديمي المعدّل

أخيرًا، احفظ تغييراتك. تضمن هذه الخطوة حفظ جميع التعديلات.

```python
presentation.save(output_path, slides.export.SaveFormat.PPTX)
print(f"Presentation saved with new slide numbering at {output_path}")
```

#### نصائح استكشاف الأخطاء وإصلاحها:
- تأكد من تحديد المسارات بشكل صحيح لتجنب أخطاء عدم العثور على الملف.
- تأكد من إمكانية الوصول إلى ملف PowerPoint وأنه غير تالف.

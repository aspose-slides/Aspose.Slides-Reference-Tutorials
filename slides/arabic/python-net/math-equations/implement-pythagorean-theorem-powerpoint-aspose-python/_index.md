## دليل التنفيذ

الآن بعد أن قمت بإعداد Aspose.Slides لـ Python، دعنا ننتقل إلى إنشاء شريحة تحتوي على نظرية فيثاغورس.

### الخطوة 1: تهيئة العرض التقديمي

ابدأ بإعداد سياق العرض التقديمي الخاص بك باستخدام `with` بيان لإدارة الموارد بشكل فعال:

```python
with slides.Presentation() as pres:
    # سيتم وضع الكود الخاص بك هنا
```

يضمن هذا إغلاق العرض التقديمي بشكل صحيح بعد العمليات الخاصة بك، مما يمنع تسرب الموارد.

### الخطوة 2: إضافة شكل مستطيل

بعد ذلك، أضف شكلاً تلقائياً لاحتواء تعبيرك الرياضي. يعمل هذا الشكل كحاوية للنص والمحتوى الرياضي.

```python
math_shape = pres.slides[0].shapes.add_auto_shape(
    slides.ShapeType.RECTANGLE, 10, 10, 100, 25
)
```

هنا، `slides.ShapeType.RECTANGLE` يحدد نوع الشكل، بينما تحدد الأرقام موضعه وحجمه على الشريحة.

### الخطوة 3: إدراج التعبير الرياضي

يمكنك الوصول إلى إطار النص داخل الشكل الخاص بك لإدراج التعبيرات الرياضية باستخدام ميزات Aspose.Slides الرياضية:

```python
math_paragraph = math_shape.text_frame.paragraphs[0].portions[0].math_paragraph
```

إنشاء تعبير نظرية فيثاغورس:

```python
math_block = mathtext.MathematicalText("c").set_superscript("2") \
    .join("=") \
    .join(mathtext.MathematicalText("a").set_superscript("2")) \
    .join("") \
    .join(mathtext.MathematicalText("b").set_superscript("2"))
```

يقوم هذا الكود ببناء التعبير (c^2 = a^2 + b^2) باستخدام `MathematicalText` الأشياء لتمثيل كل مكون.

### الخطوة 4: حفظ العرض التقديمي

وأخيرًا، احفظ عرضك التقديمي باستخدام المحتوى الرياضي الذي تم إنشاؤه حديثًا:

```python
pres.save("YOUR_OUTPUT_DIRECTORY/shapes_add_math_text_out.pptx", slides.export.SaveFormat.PPTX)
```

يستبدل `"YOUR_OUTPUT_DIRECTORY"` مع المسار الذي تريد تخزين ملفك فيه.

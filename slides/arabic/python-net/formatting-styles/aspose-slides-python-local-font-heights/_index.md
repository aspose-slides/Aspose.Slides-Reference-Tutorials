## دليل التنفيذ
### ضبط ارتفاعات الخطوط المحلية في إطارات النص
تتيح لك هذه الميزة تخصيص أجزاء من النص ضمن إطار واحد، وهو أمر مثالي للتأكيد على أجزاء معينة من العرض التقديمي الخاص بك.
#### ملخص
بتعديل ارتفاعات الخطوط محليًا، يمكنك لفت الانتباه إلى العبارات أو الأقسام الرئيسية دون تغيير التصميم العام. يتناول هذا البرنامج التعليمي ضبط ارتفاعات مختلفة لأجزاء مختلفة من الفقرة.
#### خطوات التنفيذ
##### الخطوة 1: تهيئة العرض التقديمي وإضافة الشكل
ابدأ بإنشاء عرض تقديمي جديد وإضافة شكل حيث سيتم وضع النص الخاص بك:
```python
def set_local_font_height_values():
    with slides.Presentation() as pres:
        # إضافة شكل مستطيل إلى الشريحة الأولى
        new_shape = pres.slides[0].shapes.add_auto_shape(
            slides.ShapeType.RECTANGLE, 100, 100, 400, 75, False)
```
هنا نضيف شكل مستطيل بإحداثيات وأبعاد محددة.
##### الخطوة 2: إنشاء إطار نصي
بعد ذلك، قم بإنشاء إطار نص فارغ داخل الشكل المضاف حديثًا:
```python
        # إنشاء إطار نص فارغ
        new_shape.add_text_frame("")
        new_shape.text_frame.paragraphs[0].portions.clear()
```
إن مسح الأجزاء الموجودة يضمن وجود سجل نظيف لإضافة نص مخصص.
##### الخطوة 3: إضافة أجزاء النص وتخصيصها
أضف جزأين نصيين مميزين إلى فقرتك، ثم قم بتخصيص ارتفاعات الخطوط الخاصة بهما:
```python
        # إضافة أجزاء نصية ذات ارتفاعات مختلفة
        portion0 = slides.Portion("Sample text with first portion")
        portion1 = slides.Portion(" and second portion.")
        
        new_shape.text_frame.paragraphs[0].portions.add(portion0)
        new_shape.text_frame.paragraphs[0].portions.add(portion1)

        # ضبط ارتفاعات الخطوط
        pres.default_text_style.get_level(0).default_portion_format.font_height = 24
        new_shape.text_frame.paragraphs[0].paragraph_format.default_portion_format.font_height = 40
        
        new_shape.text_frame.paragraphs[0].portions[0].portion_format.font_height = 55
        new_shape.text_frame.paragraphs[0].portions[1].portion_format.font_height = 18
```
ال `font_height` تعتبر المعلمة ضرورية لتحديد الأهمية البصرية لكل جزء.
##### الخطوة 4: حفظ العرض التقديمي
وأخيرًا، احفظ العرض التقديمي الخاص بك:
```python
        # الحفظ في دليل محدد
        pres.save("YOUR_OUTPUT_DIRECTORY/text_SetLocalFontHeightValues_out.pptx", slides.export.SaveFormat.PPTX)
```

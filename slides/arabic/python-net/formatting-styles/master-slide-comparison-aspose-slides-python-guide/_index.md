## دليل التنفيذ

يقوم هذا القسم بتقسيم عملية مقارنة شرائح PowerPoint الرئيسية إلى خطوات واضحة.

### ميزة مقارنة الشرائح

تعمل هذه الميزة على أتمتة مقارنة الشرائح الرئيسية بين عرضين تقديميين، مما يعد مفيدًا لتحديد القوالب المكررة أو الحفاظ على الاتساق بين المستندات.

#### الخطوة 1: تحميل العروض التقديمية

ابدأ بتحميل العروض التقديمية التي ترغب في مقارنتها:

```python
import aspose.slides as slides

# تحميل العرض التقديمي الأول
def load_presentations():
    with slides.Presentation('YOUR_DOCUMENT_DIRECTORY/welcome-to-powerpoint.pptx') as presentation1, \
         slides.Presentation('YOUR_DOCUMENT_DIRECTORY/background.pptx') as presentation2:
        return presentation1, presentation2
```

#### الخطوة 2: تكرار ومقارنة الشرائح الرئيسية

بعد ذلك، قم بالتكرار خلال كل شريحة رئيسية في كلا العرضين التقديميين للعثور على المطابقات:

```python
def compare_master_slides(presentation1, presentation2):
    for i in range(len(presentation1.masters)):
        for j in range(len(presentation2.masters)):
            # مقارنة الشرائح الرئيسية لكل عرض تقديمي
            if presentation1.masters[i] == presentation2.masters[j]:
                print(f'SomePresentation1 MasterSlide#{i} يساوي SomePresentation2 MasterSlide#{j}')
```

**توضيح**: 
- `presentation1.masters[i]` و `presentation2.masters[j]` يتم استخدامها للوصول إلى الشرائح الرئيسية الفردية.
- فحص المساواة (`==`) يحدد ما إذا كانت الشريحتان الرئيسيتان متطابقتين.

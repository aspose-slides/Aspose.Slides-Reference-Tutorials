```python
import aspose.slides as slides
```

## دليل التنفيذ
سوف يرشدك هذا القسم خلال تنفيذ وفهم الكود لاسترداد وعرض ألوان ثنائية اللون فعالة من شريحة العرض التقديمي.

### الوصول إلى شرائح العرض التقديمي
أولاً، افتح أو أنشئ عرضًا تقديميًا للتحكم في محتوياته:

```python
# إنشاء أو فتح مثيل عرض تقديمي موجود
with slides.Presentation() as presentation:
    # الوصول إلى الشريحة الأولى
    slide = presentation.slides[0]
```

### استرجاع تفاصيل تأثير Duotone
قم بالوصول إلى تنسيق تعبئة الخلفية واسترجاع تفاصيل تأثير اللون الثنائي:

```python
# احصل على تنسيق تعبئة الصورة للوصول إلى تأثيرات Duotone
duotone_effect = slide.background.fill_format.picture_fill_format.
                 picture.image_transform.get_duotone_effect()
```

### عرض الألوان الفعالة
استخراج وطباعة الألوان الفعالة من تأثير اللون الثنائي:

```python
# استرجاع الألوان الفعالة لتأثير Duotone
duotone_effective = duotone_effect.get_effective()

# عرض ألوان Duotone الفعالة المستخدمة
print("Duotone effective color1: " + str(duotone_effective.color1))
print("Duotone effective color2: " + str(duotone_effective.color2))
```

### خيارات تكوين المفاتيح
- **تنسيق تعبئة الصورة**:يحدد كيفية ملء الصور على الشريحة، وهو أمر ضروري للوصول إلى إعدادات النغمة الثنائية.
- **تحويل الصورة**:فئة توفر الوصول إلى التحولات المتعلقة بالصور مثل التمييز الثنائي.

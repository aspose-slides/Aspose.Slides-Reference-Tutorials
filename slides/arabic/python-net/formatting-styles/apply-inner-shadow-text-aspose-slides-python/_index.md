## دليل التنفيذ
الآن دعونا نركز على تطبيق تأثير الظل الداخلي على النص في شريحة PowerPoint.
### إضافة نص باستخدام تأثير الظل الداخلي
#### ملخص
سننشئ شكل مستطيل، ونضيف إليه نصًا، ثم نطبق عليه تأثير الظل الداخلي. تُحسّن هذه الطريقة جمالية شرائحك بإضافة عمق إلى النص.
#### دليل خطوة بخطوة
**1. الوصول إلى الشريحة**
أولاً، احصل على مرجع للشريحة الأولى في عرضك التقديمي:

```python
slide = total_presentation.slides[0]
```
**2. إضافة شكل تلقائي**
أضف شكل مستطيل لحمل النص الخاص بنا:

```python
auto_shape = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 150, 75, 400, 300)
auto_shape.fill_format.fill_type = slides.FillType.NO_FILL
```
**3. إدراج النص**
أدخل إطار نص واضبط المحتوى للمستطيل الخاص بك:

```python
auto_shape.add_text_frame("Aspose TextBox")
port = auto_shape.text_frame.paragraphs[0].portions[0]
pf = port.portion_format
pf.font_height = 50  # تعيين حجم الخط لتحسين الرؤية
```
**4. تطبيق تأثير الظل الداخلي**
تمكين وتكوين تأثير الظل الداخلي على النص:

```python
ef = pf.effect_format
ef.enable_inner_shadow_effect()
# تكوين معلمات الظل الداخلي
ef.inner_shadow_effect.blur_radius = 8.0  # نصف قطر التمويه للحصول على ظل أكثر نعومة
ef.inner_shadow_effect.direction = 90.0  # اتجاه الظل بالدرجات
ef.inner_shadow_effect.distance = 6.0    # مسافة الظل من النص
ef.inner_shadow_effect.shadow_color.b = 189  # المكون الأزرق من لون الظل
# تعيين موضوعات متسقة باستخدام ألوان المخطط
ef.inner_shadow_effect.shadow_color.color_type = slides.ColorType.SCHEME
ef.inner_shadow_effect.shadow_color.scheme_color = slides.SchemeColor.ACCENT1
```
**5. حفظ العرض التقديمي الخاص بك**
وأخيرًا، احفظ عرضك التقديمي في ملف:

```python
presentation.save("YOUR_OUTPUT_DIRECTORY/text_apply_inner_shadow_out.pptx")
```

## دليل التنفيذ

سوف يرشدك هذا القسم خلال عملية تعيين الخطوط الافتراضية لكل من صادرات HTML وPDF.

### الميزة 1: تعيين الخط العادي الافتراضي (تصدير HTML)

#### ملخص

من خلال تكوين خط منتظم محدد، يمكنك ضمان تناسق الطباعة عند تصدير العرض التقديمي الخاص بك كملف HTML.

#### التنفيذ خطوة بخطوة

##### تحميل العرض التقديمي

قم بتحميل ملف العرض التقديمي الخاص بك باستخدام:

```python
def load_presentation(path):
    # استبدل 'YOUR_DOCUMENT_DIRECTORY/' بالمسار الفعلي للمستند.
    return slides.Presentation(path)
```

##### تكوين خيارات تصدير HTML

يثبت `HtmlOptions` وحدد الخط المطلوب:

```python
def configure_html_options():
    html_options = slides.export.HtmlOptions()
    html_options.default_regular_font = "Arial Black"  # قم بتعيين الخط المفضل لديك هنا
    return html_options
```

##### حفظ العرض التقديمي بصيغة HTML

استخدم الخيارات المخصصة لحفظ العرض التقديمي:

```python
def save_html(presentation, output_path, html_options):
    presentation.save(output_path, slides.export.SaveFormat.HTML, html_options)
```

### الميزة 2: تعيين الخط العادي الافتراضي (تصدير ملفات PDF)

#### ملخص

قم بتعيين خط افتراضي لتصدير ملفات PDF للحفاظ على اتساق النص في المستندات المطبوعة أو المشتركة.

#### التنفيذ خطوة بخطوة

##### تكوين خيارات تصدير PDF

جهز `PdfOptions` مثال:

```python
def configure_pdf_options():
    pdf_options = slides.export.PdfOptions()
    pdf_options.default_regular_font = "Arial Black"  # قم بتعيين الخط المفضل لديك هنا
    return pdf_options
```

##### حفظ العرض التقديمي بصيغة PDF

قم بتصدير ملفك بتنسيق PDF باستخدام الخيارات التالية:

```python
def save_pdf(presentation, output_path, pdf_options):
    presentation.save(output_path, slides.export.SaveFormat.PDF, pdf_options)
```

## دليل التنفيذ

### إنشاء عرض تقديمي في PowerPoint

#### ملخص

سنبدأ بإنشاء عرض تقديمي بسيط يتضمن شريحة واحدة ومستطيلًا يُشكّل تلقائيًا. ستوضح هذه المهمة الأساسية كيفية التعامل مع الشرائح باستخدام بايثون.

#### إضافة شريحة وشكل

فيما يلي مقتطف لمساعدتك على البدء:

```python
import aspose.slides as slides

def create_presentation():
    with slides.Presentation() as presentation:
        # أضف شكلًا من نوع RECTANGLE إلى الشريحة الأولى
        shape = presentation.slides[0].shapes.add_auto_shape(
            slides.ShapeType.RECTANGLE, 200, 200, 200, 200)
        
        # إدراج النص في إطار النص الخاص بالشكل
        shape.text_frame.text = "This demo shows how to create a PowerPoint file and save it to Stream."
    
    return presentation

demo_presentation = create_presentation()
```

### حفظ العرض التقديمي في تيار

#### ملخص

سنركز بعد ذلك على حفظ هذا العرض التقديمي في مسار. يُعد هذا مفيدًا بشكل خاص للتطبيقات التي تتطلب إرسال أو تخزين العروض التقديمية دون كتابتها مباشرةً على القرص.

#### خطوات التنفيذ

```python
import io

def save_to_stream(presentation):
    # افتح مجرى ثنائي في الذاكرة (استخدم 'io.BytesIO' بدلاً من مسار الملف)
    with io.BytesIO() as fs:
        presentation.save(fs, slides.export.SaveFormat.PPTX)
        
        # اختياريًا: استرداد محتوى البث إذا لزم الأمر
        fs.seek(0)  # إعادة تعيين موضع البث للبدء
        ppt_data = fs.read()
    
    return ppt_data

demo_ppt_stream = save_to_stream(demo_presentation)
```

### شرح المعلمات والطرق

- **`add_auto_shape()`**:تضيف هذه الطريقة شكلاً إلى الشريحة. نحدد النوع (`RECTANGLE`) والأبعاد.
- **`save()`**:يحفظ العرض التقديمي في المسار المحدد. `SaveFormat.PPTX` يشير إلى أننا نقوم بالحفظ بتنسيق PowerPoint.

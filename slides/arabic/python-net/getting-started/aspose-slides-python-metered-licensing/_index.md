## دليل التنفيذ

### تطبيق الترخيص المقنن

سوف يرشدك هذا القسم خلال عملية إعداد الترخيص المقنن لمراقبة استهلاك واجهة برمجة التطبيقات (API) لديك بشكل فعال.

#### ملخص

يساعد الترخيص المقنن في تتبع مقدار وظيفة واجهة برمجة تطبيقات Aspose.Slides التي يتم استخدامها، مما يضمن بقائك ضمن حدود ترخيصك.

#### خطوات التنفيذ

**1. إنشاء مثيل من Metered**
ال `Metered` تدير الفئة مفتاحك المقيس وتتبع الاستخدام:

```python
metered = slides.Metered()
```

**2. اضبط المفتاح المقياس**
قم بتوفير مفاتيحك العامة والخاصة لأغراض التتبع:

```python
metered.set_metered_key("YOUR_PUBLIC_KEY", "YOUR_PRIVATE_KEY")
```

**3. تتبع استهلاك واجهة برمجة التطبيقات**
قبل استخدام أي من طرق Aspose.Slides، تحقق من كمية الاستهلاك لفهم مقدار الترخيص الذي تم استخدامه:

```python
amount_before = slides.Metered.get_consumption_quantity()
```

قم بإجراء العمليات المطلوبة باستخدام واجهة برمجة التطبيقات هنا.

**4. التحقق من الاستهلاك بعد الاستخدام**
بعد تنفيذ طرق API، قم بتتبع مستوى الاستهلاك الجديد:

```python
amount_after = slides.Metered.get_consumption_quantity()
```

**5. تأكيد قبول الترخيص**
تأكد من أن الترخيص المقنن قد تم قبوله وتطبيقه بشكل صحيح:

```python
is_metered_licensed = metered.is_metered_licensed()
```

**نتائج الإرجاع للتحقق:**
إليك كيفية تجميع تقرير عن استخدامك:

```python
def apply_metered_licensing():
    metered = slides.Metered()
    metered.set_metered_key("YOUR_PUBLIC_KEY", "YOUR_PRIVATE_KEY")
    
    amount_before = slides.Metered.get_consumption_quantity()
    # قم بإجراء عمليات Aspose.Slides هنا
    
    amount_after = slides.Metered.get_consumption_quantity()
    is_metered_licensed = metered.is_metered_licensed()
    
    return {
        "Amount Consumed Before": amount_before,
        "Amount Consumed After": amount_after,
        "Is Metered License Accepted": is_metered_licensed
    }

# مثال على الاستخدام:
result = apply_metered_licensing()
print(result)
```

# كيفية إنشاء مستطيل بسيط وحفظه في PowerPoint باستخدام Aspose.Slides Python
## مقدمة
هل سبق لك أن احتجت إلى أتمتة إنشاء الأشكال في عروض PowerPoint التقديمية؟ سواءً كنت تُحضّر عروض شرائح لاجتماعات عمل أو لأغراض تعليمية، فإن إضافة عناصر تصميمية متناسقة، مثل المستطيلات، يُمكن أن تُحسّن بشكل كبير من جاذبية عرضك التقديمي. سيرشدك هذا البرنامج التعليمي إلى كيفية إنشاء شكل مستطيل بسيط وحفظه في الشريحة الأولى من عرض PowerPoint تقديمي جديد باستخدام Aspose.Slides لـ Python.

**ما سوف تتعلمه:**
- كيفية إعداد Aspose.Slides لـ Python.
- إنشاء شكل مستطيل في شريحة PowerPoint.
- حفظ ملف PowerPoint الخاص بك مع الأشكال المضافة حديثًا.

دعونا نتعمق في كيفية تحقيق ذلك، بدءًا بالمتطلبات الأساسية اللازمة للمتابعة.

## دليل التنفيذ
دعونا نقسم العملية إلى خطوات واضحة لإنشاء شكل مستطيل وحفظ العرض التقديمي.
### إنشاء عرض تقديمي
أولاً، قم بإنشاء مثيل `Presentation` يعمل هذا كحاوية لجميع الشرائح في عرضك التقديمي:
```python
with slides.Presentation() as pres:
```
استخدام `with`، يضمن إدارة الموارد بشكل صحيح، وإغلاق الملفات حتى في حالة حدوث خطأ.
### الوصول إلى الشريحة الأولى
لإضافة الأشكال، انتقل إلى الشريحة الأولى:
```python
slide = pres.slides[0]
```
يقوم هذا الكود باسترجاع الشريحة الأولى من كائن العرض التقديمي الخاص بك.
### إضافة شكل مستطيل
الآن، دعنا نضيف شكل مستطيل في موضع محدد بأبعاد محددة:
```python
# أضف شكلًا تلقائيًا من نوع المستطيل في الموضع (50، 150) بعرض 150 وارتفاع 50
slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 50, 150, 150, 50)
```
هنا، `add_auto_shape` يُستخدم لإضافة شكل. نحدد النوع على أنه `RECTANGLE`، إلى جانب موقعها `(x=50, y=150)` والحجم `(width=150, height=50)`تقوم هذه الطريقة بإرجاع كائن الشكل الذي يمكن تخصيصه بشكل أكبر إذا لزم الأمر.
### حفظ العرض التقديمي
وأخيرًا، احفظ العرض التقديمي الخاص بك:
```python
# اكتب ملف PPTX على القرص باستخدام دليل إخراج العنصر النائب
pres.save("YOUR_OUTPUT_DIRECTORY/shapes_add_rectangle_out.pptx", slides.export.SaveFormat.PPTX)
```
يستبدل `YOUR_OUTPUT_DIRECTORY` مع المسار الذي تريده. الطريقة `save` يكتب العرض التقديمي المعدل مرة أخرى على القرص بتنسيق PPTX.
#### نصائح استكشاف الأخطاء وإصلاحها
- تأكد من صحة المسارات ووجود الدلائل قبل الحفظ.
- قم بمعالجة الاستثناءات لعمليات الملفات باستخدام كتل try-except إذا لزم الأمر.

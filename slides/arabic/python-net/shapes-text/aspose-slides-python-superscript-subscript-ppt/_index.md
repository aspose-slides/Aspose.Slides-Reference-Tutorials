# كيفية إضافة حروف علوية وسفلية في PowerPoint باستخدام Aspose.Slides لـ Python

## مقدمة

يُعدّ تحسين قابلية القراءة ونقل المعلومات التفصيلية بفعالية أمرًا بالغ الأهمية عند إعداد عروض تقديمية احترافية. إضافة حروف علوية وسفلية تُحسّن وضوح شرائحك بشكل كبير، خاصةً عند عرض البيانات العلمية أو عند إبراز العلامات التجارية.

في هذا البرنامج التعليمي، ستتعلم كيفية استخدام Aspose.Slides في بايثون لإضافة نص علوي وسفلي في شرائح PowerPoint. توفر هذه المكتبة القوية تكاملاً سلسًا وميزات غنية تُبسّط إدارة العروض التقديمية.

**ما سوف تتعلمه:**
- كيفية إضافة نص علوي وسفلي في شرائح PowerPoint
- الاستخدام الفعال لمكتبة Aspose.Slides
- خطوات رئيسية لإنشاء عروض تقديمية محسنة

قبل الغوص في الكود، تأكد من أن إعدادك جاهز لاتباع هذا الدليل.

## المتطلبات الأساسية

لتنفيذ تنسيق العلوي والسفلي باستخدام Aspose.Slides لـ Python، تأكد من تلبية المتطلبات الأساسية التالية:

- **المكتبات والإصدارات**: ثبّت Aspose.Slides لبايثون عبر pip. يمكنك القيام بذلك بتشغيل `pip install aspose.slides` في سطر الأوامر الخاص بك.
- **إعداد البيئة**:بيئة متوافقة مثل Windows أو macOS أو Linux مع Python (يوصى بالإصدار 3.x).
- **متطلبات المعرفة**:فهم أساسي لبرمجة بايثون والتعرف على كيفية العمل في واجهة سطر الأوامر.

## إعداد Aspose.Slides لـ Python

لبدء استخدام Aspose.Slides، قم بتثبيت الحزمة عبر pip:

```bash
pip install aspose.slides
```

### خطوات الحصول على الترخيص

توفر Aspose عدة خيارات للحصول على الترخيص:
- **نسخة تجريبية مجانية**:يمكنك الوصول إلى ميزات محدودة دون الحاجة إلى الشراء.
- **رخصة مؤقتة**:احصل على ترخيص مؤقت للوصول إلى الميزات الكاملة أثناء التقييم.
- **شراء**:شراء ترخيص تجاري للاستخدام طويل الأمد.

لتهيئة Aspose.Slides وإعداده، قم باستيراد المكتبة في البرنامج النصي Python الخاص بك:

```python
import aspose.slides as slides

# التهيئة الأساسية
presentation = slides.Presentation()
```

## دليل التنفيذ

يرشدك هذا القسم إلى كيفية إضافة نص علوي وسفلي إلى الشريحة.

### إنشاء عرض تقديمي جديد

ابدأ بإنشاء كائن عرض تقديمي جديد:

```python
def adding_superscript_and_subscript_text():
    with slides.Presentation() as presentation:
        slide = presentation.slides[0]
```

هنا، `presentation.slides[0]` يصل إلى الشريحة الأولى في عرضك التقديمي. يمكنك إضافة المزيد من الشرائح حسب الحاجة.

### إضافة الأشكال وإطارات النص

أضف شكلًا تلقائيًا لاستضافة النص الخاص بك:

```python
shape = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 100, 100, 200, 100)
text_frame = shape.text_frame
text_frame.paragraphs.clear()
```

يؤدي مقتطف التعليمات البرمجية هذا إلى إنشاء مستطيل ومسح أي فقرات موجودة في إطار النص.

### إضافة نص علوي

لإضافة نص علوي:
1. **إنشاء فقرة**: 
   ```python
   super_para = slides.Paragraph()
   ```
2. **إضافة نص معتاد**: 
   ```python
   portion1 = slides.Portion()
   portion1.text = "SlideTitle"
   super_para.portions.add(portion1)
   ```
3. **إضافة جزء علوي**: 
   قم بضبط الإفلات لتنسيق النص على شكل نص علوي.
   ```python
   super_portion = slides.Portion()
   super_portion.portion_format.escapement = 30  # وضع النص العلوي
   super_portion.text = "TM"
   super_para.portions.add(super_portion)
   ```

### إضافة نص سفلي

وبالمثل، بالنسبة للنص السفلي:
1. **إنشاء فقرة جديدة**: 
   ```python
   paragraph2 = slides.Paragraph()
   ```
2. **إضافة نص معتاد**: 
   ```python
   portion2 = slides.Portion()
   portion2.text = "a"
   paragraph2.portions.add(portion2)
   ```
3. **إضافة جزء منخفض**: 
   قم بضبط الإفلات لتنسيق النص على شكل خط سفلي.
   ```python
   sub_portion = slides.Portion()
   sub_portion.portion_format.escapement = -25  # وضع المؤشر السفلي
   sub_portion.text = "i"
   paragraph2.portions.add(sub_portion)
   ```

### حفظ العرض التقديمي

وأخيرًا، أضف الفقرات إلى إطار النص واحفظ العرض التقديمي الخاص بك:

```python
text_frame.paragraphs.add(super_para)
text_frame.paragraphs.add(paragraph2)

presentation.save("YOUR_OUTPUT_DIRECTORY/text_add_superscript_and_subscript_out.pptx", slides.export.SaveFormat.PPTX)
```

### نصائح استكشاف الأخطاء وإصلاحها
- تأكد من ضبط قيم الإفلات بشكل صحيح للأعلى (الإيجابية) والأسفل (السالبة).
- تأكد من تثبيت مكتبة Aspose.Slides في بيئتك.

## دليل التنفيذ

دعونا نوضح كيفية الاستفادة من Aspose.Slides لمحاذاة النص والتلاعب بالعرض التقديمي.

### الميزة: محاذاة الفقرات في PowerPoint

#### ملخص:
إن محاذاة النصوص في عروضك التقديمية لا تُحسّن سهولة القراءة فحسب، بل تُضفي عليها مظهرًا أنيقًا أيضًا. توضح هذه الميزة محاذاة الفقرات في منتصف الشرائح باستخدام بايثون.

#### خطوات:

**1. تحديد مسارات الملفات**

أولاً، قم بتعيين المسارات لملفات الإدخال والإخراج الخاصة بك:

```python
input_path = "YOUR_DOCUMENT_DIRECTORY/text_paragraphs_alignment.pptx"
output_path = "YOUR_OUTPUT_DIRECTORY/text_paragraphs_alignment_out.pptx"
```

**2. افتح العرض التقديمي واحصل على الشريحة**

افتح عرضًا تقديميًا موجودًا واحصل على الشريحة الأولى:

```python
with slides.Presentation(input_path) as pres:
    slide = pres.slides[0]
```

**3. تعديل إطارات النص**

الوصول إلى إطارات النص من عناصر نائبة محددة لتحديث محتواها:

```python
tf1 = slide.shapes[0].text_frame
# تأكد من أن الشكل يحتوي على إطار نص قبل الوصول إليه
if tf1 is not None:
    tf2 = slide.shapes[1].text_frame
    if tf2 is not None:
        tf1.text = "Center Align by Aspose"
        tf2.text = "Center Align by Aspose"
```

**4. ضبط محاذاة الفقرة**

محاذاة النص في وسط كل فقرة:

```python
para1 = tf1.paragraphs[0]
# تحقق مما إذا كانت هناك أي فقرات متاحة
if para1 is not None:
    para2 = tf2.paragraphs[0]
    # تأكد من وجود الفقرة 2 قبل ضبط المحاذاة
    if para2 is not None:
        para1.paragraph_format.alignment = slides.TextAlignment.CENTER
        para2.paragraph_format.alignment = slides.TextAlignment.CENTER
```

**5. حفظ التغييرات**

وأخيرًا، احفظ التغييرات في ملف جديد:

```python
pres.save(output_path, slides.export.SaveFormat.PPTX)
```

### الميزة: تحميل وحفظ عروض PowerPoint

#### ملخص:
تساعدك هذه الميزة على تحميل العروض التقديمية وتعديلها عن طريق إضافة نص، ثم حفظ الملفات المحدثة بكفاءة.

#### خطوات:

**1. تحديد مسارات الملفات**

قم بإعداد مسارات الإدخال والإخراج على غرار المثال السابق:

```python
input_path = "YOUR_DOCUMENT_DIRECTORY/sample_input.pptx"
output_path = "YOUR_OUTPUT_DIRECTORY/sample_output.pptx"
```

**2. تحميل العرض التقديمي والوصول إلى الشريحة**

افتح ملف العرض التقديمي الخاص بك وانتقل إلى الشريحة الأولى منه:

```python
with slides.Presentation(input_path) as pres:
    slide = pres.slides[0]
```

**3. إضافة نص إلى شكل**

تحقق مما إذا كان إطار النص فارغًا قبل إضافة محتوى جديد:

```python
tf = slide.shapes[0].text_frame
# تحقق من عدم وجود أي شيء قبل الوصول إلى الخصائص
if tf and not tf.text:
    tf.text = "New Text Added"
```

**4. احفظ العرض التقديمي**

احفظ التغييرات الخاصة بك:

```python
pres.save(output_path, slides.export.SaveFormat.PPTX)
```

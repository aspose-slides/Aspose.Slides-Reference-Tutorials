## دليل التنفيذ

### إضافة مخطط عمودي مجمع وتمكين جدول البيانات

#### ملخص

أولاً، سنضيف مخططًا عموديًا مجمعًا إلى عرضنا التقديمي ونقوم بتمكين ميزة جدول البيانات الخاصة به.

#### التنفيذ خطوة بخطوة

1. **إضافة مخطط عمودي مجمع:**
   
   أضف مقتطف التعليمات البرمجية التالي لإنشاء مخطط عمودي مجمع أساسي على الشريحة الأولى لديك:

    ```python
    chart = pres.slides[0].shapes.add_chart(
        slides.charts.ChartType.CLUSTERED_COLUMN, 50, 50, 600, 400)
    ```
   
2. **تمكين عرض جدول البيانات:**
   
   بعد ذلك، قم بتمكين جدول البيانات للرسم البياني للسماح بتخصيص الخط:

    ```python
    chart.has_data_table = True
    ```

### تخصيص خصائص الخط

#### ملخص

مع تمكين جدول البيانات، يمكننا الآن تخصيص خصائص الخط الخاص به لتحسين قابلية القراءة والأسلوب.

#### التنفيذ خطوة بخطوة

1. **تعيين الخط العريض:**
   
   استخدم هذا المقطع لجعل نص جدول البيانات الخاص بك غامقًا:

    ```python
    chart.chart_data_table.text_format.portion_format.font_bold = slides.NullableBool.TRUE
    ```

2. **ضبط ارتفاع الخط:**
   
   تغيير حجم الخط لتحسين الرؤية:

    ```python
    chart.chart_data_table.text_format.portion_format.font_height = 20
    ```

### نصائح استكشاف الأخطاء وإصلاحها

- تأكد من تثبيت جميع المكتبات المطلوبة بشكل صحيح.

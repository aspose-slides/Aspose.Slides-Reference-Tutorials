## دليل التنفيذ

الآن بعد أن قمت بتثبيت Aspose.Slides، دعنا نركز على إنشاء الأشكال التخطيطية.

### إنشاء أشكال تخطيطية في PowerPoint

#### ملخص
تتيح لك هذه الميزة إضافة تأثير خطي تخطيطي إلى الأشكال في العرض التقديمي الخاص بك، مما يمنحها مظهرًا فنيًا ومرسومًا يدويًا.

#### إضافة مستطيل باستخدام نمط خط الخربشة

##### الخطوة 1: تهيئة عرض تقديمي جديد
ابدأ بإنشاء مثيل عرض تقديمي جديد:
```python
with slides.Presentation() as pres:
    # المضي قدمًا في إضافة الأشكال
```

##### الخطوة 2: إضافة شكل تلقائي (مستطيل)
قم بإدراج شكل مستطيل في الشريحة الأولى باستخدام `add_auto_shape`:
```python
shape = pres.slides[0].shapes.add_auto_shape(
    slides.ShapeType.RECTANGLE, 20, 20, 300, 150
)
```
تحدد المعلمات نوع الشكل وموضعه/حجمه على الشريحة.

##### الخطوة 3: تعيين نوع التعبئة إلى 'NO_FILL'
للتركيز على تأثير الرسم التخطيطي، قم بإزالة أي تعبئة:
```python
shape.fill_format.fill_type = slides.FillType.NO_FILL
```

##### الخطوة 4: تطبيق تأثير رسم الخطوط المتقطعة
قم بتعزيز شكل جسمك باستخدام نمط الخط الخربش:
```python
shape.line_format.sketch_format.sketch_type = slides.LineSketchType.SCRIBBLE
```
يطبق هذا الإعداد المظهر المخطط على مخطط الشكل.

##### الخطوة 5: الحفظ بتنسيق PNG و PPTX
قم بتصدير الشريحة أولاً كصورة، ثم احفظها كملف PowerPoint:
```python
pres.slides[0].get_image(4 / 3, 4 / 3).save(
    "YOUR_OUTPUT_DIRECTORY/shapes_sketch_format_out.png",
    slides.ImageFormat.PNG
)
pres.save("YOUR_OUTPUT_DIRECTORY/shapes_sketch_format_out.pptx", 
          slides.export.SaveFormat.PPTX)
```
يستبدل `"YOUR_OUTPUT_DIRECTORY"` مع مسار الحفظ المطلوب.

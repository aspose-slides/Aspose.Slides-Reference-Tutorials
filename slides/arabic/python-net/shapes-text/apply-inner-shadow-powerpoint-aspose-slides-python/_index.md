## دليل التنفيذ
الآن بعد أن قمت بإعداد كل شيء، دعنا نركز على تطبيق تأثير الظل الداخلي على مربع النص الخاص ببرنامج PowerPoint باستخدام Aspose.Slides لـ Python.

### إضافة تأثير الظل الداخلي
#### نظرة عامة على الميزة
الهدف هو إنشاء مربع نص جذاب بصريًا مع تأثير ظل داخلي. هذا يُحسّن سهولة القراءة ويُضيف عمقًا لمحتوى الشريحة.

#### التنفيذ خطوة بخطوة
##### الخطوة 1: إنشاء عرض تقديمي
ابدأ بإنشاء كائن عرض تقديمي، مع التأكد من إدارة الموارد بشكل صحيح باستخدام `with` إفادة.
```python
def apply_inner_shadow():
    with slides.Presentation() as pres:
        # انتقل إلى الخطوات التالية
```

##### الخطوة 2: الوصول إلى الشريحة الأولى
استرجع الشريحة الأولى التي تريد تطبيق التأثير عليها.
```python
slide = pres.slides[0]
```

##### الخطوة 3: إضافة شكل مستطيل تلقائي
أضف شكلًا تلقائيًا من نوع المستطيل لاستضافة النص الخاص بك.
```python
auto_shape = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 150, 75, 150, 50)
```
*شرح المعلمات*:الإحداثيات (150، 75) تحدد الموضع؛ 150 و 50 تحدد العرض والارتفاع على التوالي.

##### الخطوة 4: إضافة إطار نصي إلى الشكل
قم بإنشاء إطار نص داخل الشكل الخاص بك لإضافة نص.
```python
auto_shape.add_text_frame(" ")
```

##### الخطوة 5: الوصول إلى إطار النص
احصل على كائن إطار النص من الشكل التلقائي.
```python
text_frame = auto_shape.text_frame
```

##### الخطوة 6: إنشاء كائن فقرة
أضف فقرة لإبقاء النص داخل إطار النص.
```python
para = text_frame.paragraphs[0]
```

##### الخطوة 7: تعيين محتوى النص
استخدم كائن الجزء لتحديد النص الذي تريده في الفقرة.
```python
portion = para.portions[0]
portion.text = "Aspose TextBox"
```

##### الخطوة 8: تطبيق تأثير الظل الداخلي (تنفيذ مخصص)
لتطبيق تأثير الظل الداخلي، عدّل خصائص الشكل. إليك كيفية القيام بذلك:
```python
# بافتراض أن Aspose.Slides يدعم هذا بشكل مباشر أو من خلال إدارة الأنماط المخصصة
def add_inner_shadow_effect(auto_shape):
    inner_shadow_effect = auto_shape.fill_format.effect_format
    # تعيين خصائص الظل الداخلي (هذا هو عنصر نائب للتنفيذ الفعلي)
    inner_shadow_effect.inner_shadow.blur_radius = 4
    inner_shadow_effect.inner_shadow.distance = 3
    inner_shadow_effect.inner_shadow.color = slides.Color.black
```
*ملحوظة*:اعتبارًا من الميزات الأخيرة المعروفة، قد تحتاج إلى توسيع هذه الوظائف باستخدام أنماط مخصصة أو مكتبات خارجية.

##### الخطوة 9: حفظ العرض التقديمي
وأخيرًا، احفظ العرض التقديمي الخاص بك مع جميع التغييرات.
```python
pres.save("YOUR_OUTPUT_DIRECTORY/text_add_textbox_out.pptx", slides.export.SaveFormat.PPTX)
```

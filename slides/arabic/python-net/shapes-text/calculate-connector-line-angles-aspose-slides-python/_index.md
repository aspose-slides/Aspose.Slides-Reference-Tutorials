## دليل التنفيذ
سنقوم بتنفيذ الميزة في جزأين رئيسيين: حساب اتجاهات الخطوط وتطبيق ذلك على موصلات PowerPoint.
### الميزة 1: حساب الاتجاه
#### ملخص
تحسب هذه الوظيفة الزوايا استنادًا إلى أبعاد وخصائص انعكاس الخطوط، مما يتيح التحكم الدقيق في اتجاهها.
#### التنفيذ خطوة بخطوة
**استيراد المكتبات المطلوبة**
```python
import math
```
**تعريف `get_direction` وظيفة**
احسب الزاوية مع الأخذ في الاعتبار العرض (`w`)، ارتفاع (`h`), الانقلاب الأفقي (`flip_h`)، والانقلاب العمودي (`flip_v`):
```python
def get_direction(w, h, flip_h, flip_v):
    # حساب إحداثيات النهاية باستخدام التقلبات
    end_line_x = w * (-1 if flip_h else 1)
    end_line_y = h * (-1 if flip_v else 1)

    # إحداثيات الخط العمودي المرجعي (المحور الصادي)
    end_y_axis_x = 0
    end_y_axis_y = h

    # احسب الزاوية بين المحور y والخط المعطى
    angle = math.atan2(end_y_axis_y, end_y_axis_x) - math.atan2(end_line_y, end_line_x)

    if angle < 0:
        angle += 2 * math.pi
    
    # تحويل الراديان إلى درجات لسهولة القراءة
    return angle * 180.0 / math.pi
```
**توضيح**
- **حدود**: `w` و `h` تحديد أبعاد الخط؛ `flip_h` و `flip_v` تحديد ما إذا كان يتم تطبيق التقلبات.
- **قيمة الإرجاع**:ترجع الدالة الزاوية بالدرجات، مما يشير إلى اتجاه الخط.
#### نصائح استكشاف الأخطاء وإصلاحها
- تأكد من أن جميع المعلمات عبارة عن أعداد صحيحة غير سلبية لتجنب النتائج غير المتوقعة.
- تأكد من أن العمليات الرياضية تتعامل مع الحالات الحدية مثل الأبعاد الصفرية بشكل جيد.
### الميزة 2: حساب زاوية خط الموصل
#### ملخص
تقوم هذه الميزة بحساب زوايا الاتجاه لخطوط الموصل في عرض تقديمي على PowerPoint، وأتمتة تحديد الزاوية باستخدام Aspose.Slides.
**استيراد المكتبات**
```python
import aspose.slides as slides
```
**تعريف `connector_line_angle` وظيفة**
قم بتحميل ملف PowerPoint ومعالجته لحساب الزوايا:
```python
def connector_line_angle():
    # تحميل ملف العرض التقديمي
    with slides.Presentation("YOUR_DOCUMENT_DIRECTORY/shapes_connector_line_angle.pptx") as pres:
        # الوصول إلى الشريحة الأولى
        slide = pres.slides[0]

        for shape in slide.shapes:
            direction = 0.0

            if isinstance(shape, slides.AutoShape):
                # تحقق مما إذا كان هذا شكلًا تلقائيًا من نوع الخط
                if shape.shape_type == slides.ShapeType.LINE:
                    direction = get_direction(
                        shape.width,
                        shape.height,
                        shape.frame.flip_h,
                        shape.frame.flip_v
                    )
            elif isinstance(shape, slides.Connector):
                # حساب الاتجاه للموصلات
                direction = get_direction(
                    shape.width,
                    shape.height,
                    shape.frame.flip_h,
                    shape.frame.flip_v
                )

            # إخراج زاوية الاتجاه المحسوبة
            print(f"Shape Direction: {direction} degrees")
```
**توضيح**
- **الوصول إلى الأشكال**:قم بالتكرار عبر كل شكل لتحديد نوعه وخصائصه.
- **حساب الاتجاه**: يتقدم `get_direction` لكل من الأشكال التلقائية (الخطوط) والموصلات.
- **الناتج**:طباعة زوايا الاتجاه المحسوبة بالدرجات.

## دليل التنفيذ

يتناول هذا القسم بالتفصيل عملية تحويل صورة SVG إلى مجموعة من الأشكال داخل عرض تقديمي في PowerPoint.

### تحويل صورة SVG إلى مجموعة من الأشكال

إليك كيفية تحويل صورة SVG المضمنة في شريحة إلى مجموعة قابلة للتلاعب من الأشكال:

#### ملخص

قم بتحميل عرض تقديمي، وحدد صورة SVG بداخله، ثم قم بتحويل هذه الصورة إلى مجموعة من الأشكال لتحسين خيارات التحرير.

#### الخطوة 1: تحميل العرض التقديمي

افتح ملف PowerPoint الخاص بك باستخدام Aspose.Slides:

```python
with slides.Presentation('YOUR_DOCUMENT_DIRECTORY/save_convert_svg_to_group_of_shapes.pptx') as pres:
    picture_frame = pres.slides[0].shapes[0]
```

#### الخطوة 2: التحقق من صورة SVG

حدد ما إذا كان الشكل الأول في الشريحة الخاصة بك يحتوي على صورة SVG:

```python
svg_image = picture_frame.picture_format.picture.image.svg_image
if svg_image is not None:
    # المضي قدما في التحويل
```

ال `picture_format` يقوم الكائن بتحديد ما إذا كان الإطار يحتوي على SVG.

#### الخطوة 3: التحويل إلى مجموعة من الأشكال

قم بتحويل SVG إلى مجموعة من الأشكال في موضعها الأصلي:

```python
group_shape = pres.slides[0].shapes.add_group_shape(
    svg_image,
    picture_frame.frame.x,
    picture_frame.frame.y,
    picture_frame.frame.width,
    picture_frame.frame.height
)
```

ال `add_group_shape` تعتبر هذه الطريقة ضرورية للحفاظ على اتساق التخطيط.

#### الخطوة 4: إزالة الإطار الأصلي

بعد التحويل، قم بإزالة صورة SVG الأصلية:

```python
pres.slides[0].shapes.remove(picture_frame)
```

تضمن هذه الخطوة عدم تكرار المحتوى داخل الشريحة الخاصة بك.

#### الخطوة 5: حفظ العرض التقديمي

وأخيرًا، احفظ العرض التقديمي المعدّل في ملف جديد:

```python
pres.save('YOUR_OUTPUT_DIRECTORY/save_convert_svg_to_group_of_shapes_out.pptx', slides.export.SaveFormat.PPTX)
```

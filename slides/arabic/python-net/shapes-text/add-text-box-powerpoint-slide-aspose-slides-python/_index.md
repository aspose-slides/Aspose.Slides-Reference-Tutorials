## دليل التنفيذ

بعد أن أصبحت بيئتنا جاهزة، لنبدأ التنفيذ. سنغطي كل خطوة مطلوبة لإضافة مربع نص إلى الشريحة.

### إنشاء عرض تقديمي جديد والوصول إلى الشريحة الأولى

أولاً، قم بإنشاء مثيل للعرض التقديمي والوصول إلى الشريحة الأولى منه:

```python
def add_text_box_to_slide():
    with slides.Presentation() as pres:
        # الوصول إلى الشريحة الأولى
        slide = pres.slides[0]
```

**توضيح**: ال `Presentation()` يقوم الفصل بتهيئة عرض تقديمي جديد. باستخدام `pres.slides[0]`، نصل إلى الشريحة الأولى.

### إضافة مستطيل الشكل التلقائي

أضف شكل مستطيل إلى الشريحة الخاصة بك:

```python
# إضافة شكل مستطيل تلقائي
auto_shape = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 150, 75, 150, 50)
```

**حدود**: ال `add_auto_shape` تأخذ الطريقة نوع الشكل وإحداثيات الموضع (X، Y) بالإضافة إلى العرض والارتفاع.

### إدراج إطار نص

أدخل إطار نص في هذا المستطيل:

```python
# إضافة إطار نص إلى الشكل
auto_shape.add_text_frame(" ")
```

**غاية**:يؤدي هذا إلى إنشاء إطار نص فارغ حيث يمكنك إضافة المحتوى الخاص بك.

### تعيين النص في مربع النص

تعديل النص داخل مربع النص الذي تم إنشاؤه حديثًا:

```python
# الوصول إلى النص وضبطه
text_frame = auto_shape.text_frame
para = text_frame.paragraphs[0]
portion = para.portions[0]
portion.text = "Aspose TextBox"
```

**توضيح**:هنا، نصل إلى الفقرة الأولى والجزء الأول من إطار النص لتعيين النص المطلوب.

### حفظ العرض التقديمي

وأخيرًا، احفظ العرض التقديمي الخاص بك:

```python
# حفظ العرض التقديمي
pres.save("YOUR_OUTPUT_DIRECTORY/text_TextBox_out.pptx")
```

**ملحوظة**: يستبدل `YOUR_OUTPUT_DIRECTORY` مع مسار الملف المطلوب.

## دليل التنفيذ

### استخراج بيانات تنسيق إطار النص
تساعدك هذه الميزة على الوصول إلى تفاصيل تنسيق إطار النص وعرضها برمجيًا من شريحة PowerPoint.

#### نظرة عامة على الميزة:
تتضمن هذه العملية الوصول إلى الشكل الأول في الشريحة الأولى من العرض التقديمي الخاص بك، واسترداد خصائص تنسيق إطار النص الفعالة الخاصة به، وعرضها. 

##### التنفيذ خطوة بخطوة:
**1. الوصول إلى الشريحة:**
ابدأ بتحميل ملف العرض التقديمي والوصول إلى الشريحة والشكل المطلوبين.
```python
# تحميل ملف العرض التقديمي
current_pres = "YOUR_DOCUMENT_DIRECTORY/text_add_animation_effect.pptx"
with slides.Presentation(current_pres) as pres:
    # الوصول إلى الشكل الأول في الشريحة الأولى
    shape = pres.slides[0].shapes[0]
```

**2. استرداد خصائص تنسيق إطار النص:**
جلب وتخزين خصائص تنسيق إطار النص الفعالة من الشكل المحدد.
```python
# احصل على تنسيق إطار النص وخصائصه الفعالة
if shape.text_frame is not None:
    text_frame_format = shape.text_frame.text_frame_format
    effective_text_frame_format = text_frame_format.get_effective()
```

**3. عرض البيانات الفعالة:**
إخراج نوع التثبيت، وإعدادات الملاءمة التلقائية، والمحاذاة الرأسية، وحواف إطار النص.
```python
# عرض بيانات تنسيق إطار النص الفعال
if effective_text_frame_format:
    print("Anchoring type: " + str(effective_text_frame_format.anchoring_type))
    print("Autofit type: " + str(effective_text_frame_format.autofit_type))
    print("Text vertical type: " + str(effective_text_frame_format.text_vertical_type))
    print("Margins")
    print("   Left: " + str(effective_text_frame_format.margin_left))
    print("   Top: " + str(effective_text_frame_format.margin_top))
    print("   Right: " + str(effective_text_frame_format.margin_right))
    print("   Bottom: " + str(effective_text_frame_format.margin_bottom))
```

**نصائح استكشاف الأخطاء وإصلاحها:**
- تأكد من أن مسار ملف PowerPoint الخاص بك صحيح لتجنب `FileNotFoundError`.
- تأكد مرة أخرى من أن مؤشرات الشريحة والشكل تقع ضمن نطاق العرض التقديمي الخاص بك.

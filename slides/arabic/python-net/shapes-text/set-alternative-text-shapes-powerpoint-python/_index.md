## دليل التنفيذ
الآن، دعونا نتناول عملية إعداد نص بديل للأشكال في عروض PowerPoint التقديمية.

### إعداد بيئة العرض التقديمي الخاص بك
أولاً، علينا إعداد مسارات المستندات وإنشاء فئة عرض تقديمي. تتضمن هذه الخطوة إنشاء أو تحميل ملف PPTX موجود، حيث يمكنك التعامل مع الأشكال.

#### تهيئة المسارات وفئة العرض

```python
import aspose.slides as slides
import os

document_directory = "YOUR_DOCUMENT_DIRECTORY/"
output_directory = "YOUR_OUTPUT_DIRECTORY/"

# تأكد من وجود دليل الإخراج
if not os.path.exists(output_directory):
    os.makedirs(output_directory)

with slides.Presentation() as pres:
    # الكود الخاص بك يذهب هنا
```

### إضافة الأشكال إلى الشريحة
الآن، لنُضِف بعض الأشكال إلى شريحتنا. يتضمن هذا المثال إضافة مستطيل وجسم على شكل قمر.

#### إضافة شكل مستطيل

```python
# احصل على الشريحة الأولى من العرض التقديمي
slide = pres.slides[0]

# أضف شكل مستطيل
shape1 = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 50, 40, 150, 50)
```

#### إضافة كائن على شكل قمر باستخدام تعبئة الألوان

```python
# أضف كائنًا على شكل قمر واضبط لون تعبئته على اللون الرمادي
define shape2 = slide.shapes.add_auto_shape(slides.ShapeType.MOON, 160, 40, 150, 50)
shape2.fill_format.fill_type = slides.FillType.SOLID
shape2.fill_format.solid_fill_color.color = drawing.Color.gray
```

### تعيين نص بديل للأشكال
أخيرًا، كرّر كل شكل في الشريحة وعيّن نصًا بديلًا. هذه الخطوة أساسية لتسهيل الوصول.

```python
# كرر كل شكل في الشريحة وحدد نصًا بديلاً للأشكال التلقائية
define for shape in slide.shapes:
    if isinstance(shape, slides.AutoShape):
        shape.alternative_text = "User Defined"
```

### حفظ العرض التقديمي الخاص بك
تأكد من حفظ العرض التقديمي الخاص بك بعد إجراء التغييرات:

```python
pres.save(os.path.join(output_directory, "shapes_set_alternative_text_out.pptx"), slides.export.SaveFormat.PPTX)
```

# كيفية استرداد خصائص تأثير الشطب من الأشكال ثلاثية الأبعاد باستخدام Aspose.Slides لـ Python

**ما سوف تتعلمه:**
- إعداد Aspose.Slides واستخدامه لـ Python.
- الوصول إلى خصائص الشطب في الأشكال ثلاثية الأبعاد في PowerPoint.
- دمج هذه الوظيفة في سير عمل العرض التقديمي الخاص بك.

## دليل التنفيذ

استرداد خصائص الشطبة من الوجه العلوي للشكل ثلاثي الأبعاد باستخدام Aspose.Slides لـ Python.

### نظرة عامة على الميزة

يمكنك الوصول إلى خصائص الشطب التفصيلية وطباعتها مثل النوع والعرض والارتفاع للتحكم في التأثيرات المرئية للعرض التقديمي بدقة.

#### التنفيذ خطوة بخطوة

1. **افتح ملف PowerPoint**
   فتح ملف يحتوي على أشكال ثلاثية الأبعاد:

   ```python
   input_file_path = 'YOUR_DOCUMENT_DIRECTORY/shapes_3d_effective.pptx'
   
   with slides.Presentation(input_file_path) as pres:
       # الوصول إلى الشريحة الأولى وشكلها الأول
       shape = pres.slides[0].shapes[0]
   ```

2. **استرداد خصائص تنسيق ثلاثي الأبعاد**
   استخراج خصائص تنسيق ثلاثي الأبعاد الفعالة للشكل:

   ```python
   three_d_effective_data = shape.three_d_format.get_effective()
   ```

3. **خصائص الوجه العلوي لشطبة الإخراج**
   اطبع نوع الشطبة والعرض والارتفاع للتحليل:

   ```python
   print("= Effective shape's top face relief properties =")
   print("Type: " + str(three_d_effective_data.bevel_top.bevel_type))
   print("Width: " + str(three_d_effective_data.bevel_top.width))
   print("Height: " + str(three_d_effective_data.bevel_top.height))
   ```

**نصائح استكشاف الأخطاء وإصلاحها:** 
- تأكد من أن مسار المستند صحيح.
- تأكد من أن الأشكال التي تم الوصول إليها تحتوي على خصائص تنسيق ثلاثية الأبعاد.

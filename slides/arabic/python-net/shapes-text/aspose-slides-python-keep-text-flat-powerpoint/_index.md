# إتقان Aspose.Slides في بايثون: كيفية تعديل خاصية "الحفاظ على النص مسطحًا" لأشكال ونصوص PowerPoint

## مقدمة

يتطلب إنشاء عروض تقديمية احترافية الحفاظ على نص واضح وجذاب بصريًا ضمن الأشكال. ومن التحديات الشائعة التحكم في ما إذا كان النص يبقى مسطحًا أو يدعم التنسيقات المتقدمة مثل WordArt. يرشدك هذا البرنامج التعليمي إلى تعديل خاصية "keep_text_flat" في PowerPoint باستخدام Aspose.Slides لـ Python، لضمان أن تكون عروضك التقديمية مصقولة وفعالة.

**ما سوف تتعلمه:**
- إعداد Aspose.Slides لـ Python
- تقنيات لتعديل خصائص 'keep_text_flat' لإطارات النص
- التطبيقات الواقعية لهذه التعديلات

دعنا نتعمق في أتمتة PowerPoint مع Aspose.Slides!

## دليل التنفيذ

في هذا القسم، سنقوم بتعديل خصائص النص باستخدام Aspose.Slides لـ Python.

### الوصول إلى إطارات النص وتعديلها

#### ملخص:
سنوضح تعديل خاصية "keep_text_flat" في إطارات النصوص ضمن شرائح PowerPoint. تتحكم هذه الخاصية في بقاء النص بتنسيقه الأصلي أو تسطيحه لتسهيل العرض.

#### التنفيذ خطوة بخطوة:

**1. قم بتحميل العرض التقديمي الخاص بك:**
ابدأ بتحميل ملف العرض التقديمي الخاص بك باستخدام Aspose.Slides.

```python
pres = slides.Presentation('YOUR_DOCUMENT_DIRECTORY/text_keep_text_flat.pptx')
```
يستبدل `'YOUR_DOCUMENT_DIRECTORY'` مع المسار الفعلي لملف PowerPoint الخاص بك.

**2. الوصول إلى إطارات النص في الأشكال:**
الوصول إلى أشكال محددة داخل الشريحة وإطارات النص الخاصة بها:

```python
shape1 = pres.slides[0].shapes[0]
shape2 = pres.slides[0].shapes[1]
```
نحن نصل إلى الشكلين الأولين على الشريحة الأولى لأغراض العرض التوضيحي.

**3. تعديل خاصية "الحفاظ على النص مسطحًا":**
قم بتعديل هذه الخاصية للتحكم في سلوك تنسيق النص:

```python
# تعطيل تنسيق النص المسطح للشكل 1
disabled_flat_text = False
shape1.text_frame.text_frame_format.keep_text_flat = disabled_flat_text

# تمكين تنسيق النص المسطح للشكل 2
enabled_flat_text = True
shape2.text_frame.text_frame_format.keep_text_flat = enabled_flat_text
```
- `keep_text_flat=False` يسمح بتنسيق النص المعقد.
- `keep_text_flat=True` يُبسط النص إلى النمط الأساسي.

**4. حفظ الشريحة وتصديرها:**
وأخيرًا، احفظ التغييرات عن طريق تصدير الشريحة:

```python
pres.slides[0].get_image(4 / 3, 4 / 3).save('YOUR_OUTPUT_DIRECTORY/text_keep_text_flat_out.png', slides.ImageFormat.PNG)
```
يضمن `'YOUR_OUTPUT_DIRECTORY'` تم ضبطه على المكان الذي تريد حفظ الصورة الناتجة فيه.

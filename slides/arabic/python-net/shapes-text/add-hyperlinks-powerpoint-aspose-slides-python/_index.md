## دليل التنفيذ
سنقوم بتقسيم عملية إضافة الارتباطات التشعبية إلى شرائح PowerPoint إلى خطوات.

### إضافة شكل تلقائي وإطار نص
أولاً، نحتاج إلى شكل على الشريحة للنص. إليك كيفية إضافته:

#### الخطوة 1: إنشاء كائن عرض تقديمي
```python
with slides.Presentation() as presentation:
    # سيتم وضع الكود الخاص بك هنا
```
يؤدي هذا إلى تهيئة عرض تقديمي جديد في PowerPoint.

#### الخطوة 2: إضافة شكل تلقائي
أضف شكل مستطيل مع النص:
```python
shape1 = presentation.slides[0].shapes.add_auto_shape(
    slides.ShapeType.RECTANGLE, 100, 100, 600, 50, False)
```
تتضمن المعلمات موضع الشكل وحجمه.

#### الخطوة 3: إضافة نص إلى الشكل
أدخل النص المطلوب في الشكل:
```python
shape1.add_text_frame("Aspose: File Format APIs")
```

### إعداد ارتباط تشعبي على النص
الآن، اجعل هذا النص قابلاً للنقر عن طريق إضافة رابط تشعبي.

#### الخطوة 4: تعيين ارتباط تشعبي
ربط النص إلى عنوان URL:
```python
shape1.text_frame.paragraphs[0].portions[0].portion_format.hyperlink_click = slides.Hyperlink(
    "https://www.aspose.com/")
```
يقوم مقتطف التعليمات البرمجية هذا بتحويل الجزء الأول من الفقرة الأولى إلى ارتباط تشعبي.

#### الخطوة 5: إضافة تلميح للارتباط التشعبي
توفير معلومات إضافية عبر تلميح الأدوات:
```python
shape1.text_frame.paragraphs[0].portions[0].portion_format.hyperlink_click.tooltip = \\
    "More than 70% Fortune 100 companies trust Aspose APIs"
```

### تخصيص مظهر النص
ضبط المظهر لجعله أكثر وضوحا.

#### الخطوة 6: تعيين حجم الخط
قم بزيادة حجم الخط لتحسين الرؤية:
```python
shape1.text_frame.paragraphs[0].portions[0].portion_format.font_height = 32
```

### حفظ العرض التقديمي الخاص بك
وأخيرًا، احفظ العرض التقديمي الخاص بك مع تطبيق كافة التغييرات.
```python\presentation.save("YOUR_OUTPUT_DIRECTORY/hyperlink_add_hyperlink_out.pptx")
```
يستبدل `YOUR_OUTPUT_DIRECTORY` مع المسار الفعلي الذي تريد حفظ الملف فيه.

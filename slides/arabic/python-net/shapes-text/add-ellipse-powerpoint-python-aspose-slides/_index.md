### إضافة شكل بيضاوي إلى شريحة (H3)
يوضح هذا القسم إضافة شكل بيضاوي إلى الشريحة الأولى من عرض تقديمي جديد. إليك الطريقة:

#### الخطوة 1: إنشاء نسخة عرض تقديمي (H4)
إنشاء مثيل لـ `Presentation` الفئة التي تمثل ملف PowerPoint الخاص بك.
```python
import aspose.slides as slides

def add_ellipse_to_slide():
    # تهيئة كائن عرض تقديمي جديد.
    with slides.Presentation() as pres:
```

#### الخطوة 2: الوصول إلى الشريحة الأولى (H4)
قم بتعديل الشريحة الأولى لإدراج القطع الناقص الخاص بك.
```python
        # الوصول إلى الشريحة الأولى.
        slide = pres.slides[0]
```

#### الخطوة 3: إضافة شكل بيضاوي (H4)
أدخل قطعًا ناقصًا في موضع محدد بأبعاد معينة باستخدام `add_auto_shape` طريقة.
```python
        # إدراج شكل بيضاوي في الشريحة.
        slide.shapes.add_auto_shape(slides.ShapeType.ELLIPSE, 50, 150, 150, 50)
```
هنا:
- **نوع الشكل.القطع الناقص**:يحدد الشكل على شكل قطع ناقص.
- **50، 150**:إحداثيات x و y لتحديد الموضع على الشريحة.
- **150، 50**:عرض وارتفاع القطع الناقص.

#### الخطوة 4: حفظ العرض التقديمي (H4)
احفظ العرض التقديمي الخاص بك في الموقع المطلوب بتنسيق PPTX:
```python
        # احفظ العرض التقديمي المعدّل.
        pres.save("YOUR_OUTPUT_DIRECTORY/shapes_add_ellipse_out.pptx", slides.export.SaveFormat.PPTX)
```

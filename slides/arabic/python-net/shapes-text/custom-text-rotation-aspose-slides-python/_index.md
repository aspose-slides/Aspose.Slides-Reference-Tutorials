# كيفية تدوير إطارات النص في PowerPoint باستخدام Aspose.Slides لـ Python: دليل خطوة بخطوة

## مقدمة

قد يُشكّل عرض البيانات بفعالية تحديًا عند عدم اتساق اتجاهات النص القياسية. يُضفي تدوير إطارات النص وضوحًا وأسلوبًا مميزًا على عروضك التقديمية أو تقاريرك. سيُرشدك هذا الدليل إلى كيفية ضبط زوايا تدوير مُخصصة لإطارات النص باستخدام Aspose.Slides لـ Python، مما يُحسّن سهولة القراءة والجاذبية البصرية.

بحلول نهاية هذا البرنامج التعليمي، سوف تتعلم كيفية:
- إنشاء عروض تقديمية لبرنامج PowerPoint برمجيًا
- إضافة المخططات البيانية ومعالجتها في الشرائح
- تعيين زوايا دوران مخصصة لكتل النص
- احفظ عرضك التقديمي بكفاءة

## دليل التنفيذ

### إضافة مخطط عمودي مجمع وتدوير إطارات النص

يرشدك هذا القسم خلال عملية إضافة مخطط عمودي مجمع إلى العرض التقديمي الخاص بك وتعيين زوايا دوران مخصصة لإطارات النص داخل هذا المخطط.

#### الخطوة 1: إنشاء مثيل لفئة العرض التقديمي

ابدأ بإنشاء `Presentation` الكائن باستخدام مدير السياق، مما يضمن إدارة الموارد التلقائية:

```python
import aspose.slides as slides

def rotate_text_frame():
    # استخدم مدير السياق للتعامل مع الموارد تلقائيًا
    with slides.Presentation() as presentation:
        pass  # عنصر نائب للخطوات اللاحقة
```

#### الخطوة 2: إضافة مخطط عمودي مجمع

أضف مخططًا عموديًا مجمعًا إلى الشريحة الأولى في الموضع (50، 50) بأبعاد محددة:

```python
# إضافة الرسم البياني إلى الشريحة الأولى
class ChartType:
    CLUSTERED_COLUMN = 'ClusteredColumn'
chart = presentation.slides[0].shapes.add_chart(
    ChartType.CLUSTERED_COLUMN, 50, 50, 500, 300
)
```

#### الخطوة 3: الوصول إلى سلسلة المخططات وتكوين التسميات

قم بالوصول إلى السلسلة الأولى في بيانات الرسم البياني لديك للتحكم في تسمياتها:

```python
# الوصول إلى السلسلة الأولى
class DataLabelFormatType:
    SHOW_VALUE = 'ShowValue'
series = chart.chart_data.series[0]

# عرض القيم على الملصقات
series.labels.default_data_label_format.show_value = True
```

#### الخطوة 4: تعيين زاوية الدوران المخصصة لتنسيق كتلة النص

قم بتعيين زاوية دوران مخصصة لتنسيق كتلة النص لجعل بياناتك أكثر جاذبية من الناحية البصرية:

```python
# تعيين زاوية الدوران المخصصة
class TextBlockFormatType:
    ROTATION_ANGLE = 'RotationAngle'
series.labels.default_data_label_format.text_format.text_block_format.rotation_angle = 65
```

#### الخطوة 5: إضافة عنوان الرسم البياني وتدويره

أضف عنوانًا إلى الرسم البياني الخاص بك وقم بتطبيق زاوية دوران مخصصة لتحسين المظهر:

```python
# إضافة عنوان الرسم البياني وتدويره
class TextFrameFormatType:
    ROTATION_ANGLE = 'RotationAngle'
chart.has_title = True
chart.chart_title.add_text_frame_for_overriding("Custom Title").text_frame_format.rotation_angle = -30
```

#### الخطوة 6: حفظ العرض التقديمي

وأخيرًا، احفظ العرض التقديمي الخاص بك في دليل الإخراج:

```python
# حفظ العرض التقديمي
class SaveFormatType:
    PPTX = 'Pptx'
presentation.save(
    "YOUR_OUTPUT_DIRECTORY/text_textframe_rotation_out.pptx",
    SaveFormatType.PPTX
)
```

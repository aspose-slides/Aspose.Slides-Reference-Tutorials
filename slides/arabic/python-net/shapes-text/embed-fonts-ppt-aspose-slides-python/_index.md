# تضمين الخطوط في عروض PowerPoint التقديمية باستخدام Aspose.Slides لـ Python

## مقدمة
غالبًا ما يتطلب إنشاء عروض PowerPoint جذابة بصريًا خطوطًا محددة قد لا تكون متوفرة على كل جهاز، مما يؤدي إلى عدم الاتساق. **Aspose.Slides لـ Python**يمكنك تضمين الخطوط مباشرةً في عروضك التقديمية لضمان عرض متناسق على جميع المنصات. سيرشدك هذا البرنامج التعليمي إلى كيفية استخدام Aspose.Slides لتضمين الخطوط.

**ما سوف تتعلمه:**
- تضمين الخطوط في PowerPoint باستخدام Aspose.Slides
- إعداد وتثبيت Aspose.Slides لـ Python
- التنفيذ خطوة بخطوة مع أمثلة التعليمات البرمجية
- التطبيقات العملية لتضمين الخطوط

## المتطلبات الأساسية
قبل البدء، تأكد من أن لديك:

### المكتبات والتبعيات المطلوبة
- **Aspose.Slides لـ Python**:ضروري لإدارة عروض PowerPoint.
- **بيئة بايثون**:استخدم Python 3.6 أو أحدث.

## دليل التنفيذ
الآن، دعونا ننفذ تضمين الخطوط في عروض PowerPoint.

### نظرة عامة على ميزة تضمين الخطوط
تضمن هذه الميزة تضمين جميع الخطوط لمنع أي اختلافات على الأجهزة المختلفة. كما تتحقق تلقائيًا من الخطوط غير المضمنة وتُضمّنها.

#### الخطوة 1: تحديد أدلة المستندات والإخراج
حدد موقع العرض المصدر ودليل ملف الإخراج:

```python
document_dir = 'YOUR_DOCUMENT_DIRECTORY/'
output_dir = 'YOUR_OUTPUT_DIRECTORY/'
```

#### الخطوة 2: تحميل العرض التقديمي
افتح ملف PowerPoint الموجود باستخدام Aspose.Slides:

```python
with slides.Presentation(document_dir + 'text_fonts.pptx') as presentation:
    # متابعة العمليات على العرض التقديمي
```

#### الخطوة 3: استرداد الخطوط والتحقق منها
تحديد الخطوط غير المضمنة في العرض التقديمي:

```python
all_fonts = presentation.fonts_manager.get_fonts()
embedded_fonts = presentation.fonts_manager.get_embedded_fonts()

for font in all_fonts:
    if font not in embedded_fonts:
        # سيتم تضمين هذا الخط
```

#### الخطوة 4: تضمين الخطوط غير المضمنة
قم بتضمين كل خط غير مضمن باستخدام Aspose.Slides:

```python
presentation.fonts_manager.add_embedded_font(font, slides.export.EmbedFontCharacters.ALL)
```

ويضمن هذا عرض النص بشكل متسق عبر الأجهزة.

#### الخطوة 5: حفظ العرض التقديمي المحدث
احفظ العرض التقديمي الخاص بك مع الخطوط المضمنة في ملف جديد:

```python
presentation.save(output_dir + 'text_add_embedded_font_out.pptx', slides.export.SaveFormat.PPTX)
```

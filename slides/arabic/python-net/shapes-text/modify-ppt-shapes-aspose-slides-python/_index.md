## دليل التنفيذ
في هذا القسم، سنتناول عملية تعديل تعديلات الشكل.

### الوصول إلى تعديلات الشكل وتعديلها
#### ملخص
تتيح لك هذه الميزة الوصول إلى نقاط تعديل محددة على أشكال PowerPoint وتعديل خصائصها برمجيًا. سنشرح كيفية العمل مع شكلي المستطيل الدائري والسهم في عرض تقديمي.

#### الخطوة 1: تحميل العرض التقديمي الخاص بك
أولاً، قم بتحميل ملف PowerPoint الحالي لديك باستخدام Aspose.Slides:

```python
with slides.Presentation('YOUR_DOCUMENT_DIRECTORY/PresetGeometry.pptx') as pres:
    # الوصول إلى الشكل الأول للشريحة الأولى
    shape = pres.slides[0].shapes[0]
```

#### الخطوة 2: أنواع تعديل العرض لشكل ما
افهم التعديلات المتاحة من خلال تكرارها:

```python
print("Adjustment types for a Rectangle:")
for i in range(len(shape.adjustments)):
    print(f"\tType for point {i} is", shape.adjustments[i].type.name)
```

#### الخطوة 3: تعديل نقاط التعديل
إذا كان نوع التعديل يطابق معاييرك، فقم بتعديل قيمته:

```python
# مثال: مضاعفة حجم زاوية المستطيل المستدير
corner_adjustment_index = next((i for i, adj in enumerate(shape.adjustments) if adj.type == slides.ShapeAdjustmentType.CORNER_SIZE), None)
if corner_adjustment_index is not None:
    shape.adjustments[corner_adjustment_index].angle_value *= 2
```

#### الخطوة 4: حفظ التغييرات
بعد إجراء التعديلات الخاصة بك، احفظ العرض التقديمي ليعكس التغييرات:

```python
pres.save('YOUR_OUTPUT_DIRECTORY/PresetGeometry_out.pptx', slides.export.SaveFormat.PPTX)
```

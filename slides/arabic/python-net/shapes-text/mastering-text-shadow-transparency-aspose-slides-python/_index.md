## دليل التنفيذ

اتبع الخطوات التالية لضبط شفافية ظل النص.

### تحميل العرض التقديمي
**ملخص**:ابدأ بتحميل ملف PowerPoint الموجود.

#### الخطوة 1: افتح ملف العرض التقديمي الخاص بك
استخدم مدير السياق لإدارة الموارد:
```python
with slides.Presentation('YOUR_DOCUMENT_DIRECTORY/text_transparency.pptx') as pres:
    # سيتم تنفيذ الخطوات الأخرى ضمن هذه الكتلة.
```

### عناصر نص الوصول
**ملخص**:تنقل عبر أشكال الشريحة لتحديد عناصر النص.

#### الخطوة 2: استرداد الشكل الأول على الشريحة
الوصول إلى الشكل الأول الذي يحتوي على النص:
```python
shape = pres.slides[0].shapes[0]
```

### تعديل شفافية الظل
**ملخص**:ضبط مستوى شفافية تأثير الظل المطبق على النص الخاص بك.

#### الخطوة 3: الوصول إلى تنسيق تأثير النص
استرداد تنسيق التأثير للجزء الأولي من النص:
```python
effects = shape.text_frame.paragraphs[0].portions[0].portion_format.effect_format
```

#### الخطوة 4: طباعة شفافية الظل الحالية
التحقق من مستوى الشفافية الحالي وطباعته:
```python
outer_shadow_effect = effects.outer_shadow_effect
color = outer_shadow_effect.shadow_color.color
transparency_percentage = (color.a / 255) * 100
print(f"Current shadow transparency: {transparency_percentage}%")
```

#### الخطوة 5: اضبط الظل على التعتيم الكامل
ضبط لون الظل للحصول على التعتيم الكامل:
```python
outer_shadow_effect.shadow_color.color = drawing.Color.from_argb(255, *color)
```

### حفظ العرض التقديمي المعدّل
**ملخص**:قم بتخزين التغييرات مرة أخرى في ملف PowerPoint.

#### الخطوة 6: حفظ التغييرات
تأكد من حفظ جميع التعديلات بشكل صحيح:
```python
pres.save('YOUR_OUTPUT_DIRECTORY/text_transparency_out.pptx', slides.export.SaveFormat.PPTX)
```

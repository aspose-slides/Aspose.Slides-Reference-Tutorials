## دليل التنفيذ

في هذا القسم، سنقوم بتوضيح عملية إنشاء شكل المجموعة داخل شريحة العرض التقديمي.

### إنشاء أشكال المجموعة في شرائح العرض التقديمي

تساعد هذه الميزة على تنظيم الأشكال المتعددة في وحدة متماسكة لتحقيق هيكل أفضل وجاذبية بصرية.

#### الخطوة 1: إنشاء عرض تقديمي أو فتحه

ابدأ بفتح عرض تقديمي موجود أو إنشاء عرض تقديمي جديد:

```python
def create_group_shape():
    with slides.Presentation() as pres:
        slide = pres.slides[0]
```

*لماذا:* نحن نستخدم `with` بيان لإدارة السياق، والتأكد من تنظيف الموارد بشكل صحيح بعد العمليات.

#### الخطوة 2: الوصول إلى مجموعة الأشكال

احصل على إمكانية الوصول إلى الأشكال الموجودة في الشريحة الحالية:

```python
shapes = slide.shapes
```

تتيح لنا هذه المجموعة إمكانية التلاعب بالأشكال وإضافة أشكال جديدة.

#### الخطوة 3: إضافة شكل المجموعة

أضف شكلًا جماعيًا لاحتواء الأشكال الفردية:

```python
group_shape = shapes.add_group_shape()
```

*لماذا:* يؤدي تجميع الأشكال إلى تبسيط التعامل معها، مما يسمح لك بنقلها أو تعديلها كوحدة واحدة.

#### الخطوة 4: إدراج الأشكال الفردية

أضف مستطيلات داخل شكل المجموعة في مواضع محددة:

```python
group_shape.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 300, 100, 100, 100)
group_shape.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 500, 100, 100, 100)
group_shape.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 300, 300, 100, 100)
group_shape.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 500, 300, 100, 100)
```

*لماذا:* تتضمن هذه الخطوة إضافة أشكال لإظهار قدرات التجميع.

#### الخطوة 5: إضافة إطار

قم بإعداد إطار حول شكل المجموعة للتوضيح البصري:

```python
group_shape.frame = slides.ShapeFrame(
    100, 300, 500, 40,
    slides.NullableBool.TRUE,
    slides.NullableBool.TRUE,
    0
)
```

#### الخطوة 6: حفظ العرض التقديمي

وأخيرًا، احفظ العرض التقديمي الخاص بك في الدليل المحدد:

```python
pres.save("YOUR_OUTPUT_DIRECTORY/shapes_create_group_shape_out.pptx", slides.export.SaveFormat.PPTX)
```

*لماذا:* يضمن الحفظ تخزين كافة التغييرات وإمكانية الوصول إليها لاحقًا.

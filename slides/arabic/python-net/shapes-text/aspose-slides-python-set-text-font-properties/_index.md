## دليل التنفيذ

### ضبط خصائص خط النص (نظرة عامة على الميزة)
في هذا القسم، ستتعلم كيفية تعيين خصائص الخط المختلفة للنص داخل شريحة في PowerPoint باستخدام Aspose.Slides لـ Python.

#### الخطوة 1: إنشاء عرض تقديمي
ابدأ بإنشاء مثيل لـ `Presentation` فصل:

```python
def set_text_font_properties():
    with slides.Presentation() as presentation:
        slide = presentation.slides[0]
```
**توضيح:** نحن نستخدم مدير السياق (`with`لضمان إدارة الموارد بشكل صحيح، مما يساعد في استخدام الذاكرة بكفاءة.

#### الخطوة 2: إضافة شكل تلقائي
أضف شكل مستطيل لوضع النص على الشريحة الخاصة بك:

```python
auto_shape = slide.shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 50, 50, 200, 50)
```
**توضيح:** ال `add_auto_shape` تضيف الطريقة شكلاً بنوع وأبعاد محددة. هنا، نستخدم مستطيلاً في الموضع `(50, 50)` مع العرض `200` والارتفاع `50`.

#### الخطوة 3: تخصيص إطار النص
قم بالوصول إلى إطار النص لإضافة النص وتخصيصه:

```python
tf = auto_shape.text_frame
tf.text = "Aspose TextBox"
```
**توضيح:** ال `text_frame` تتيح لك الخاصية الوصول إلى محتوى الشكل أو تعديله.

#### الخطوة 4: تعيين خصائص الخط
تطبيق خصائص الخط المختلفة مثل الغامق، المائل، المسطر، واللون:

```python
port = tf.paragraphs[0].portions[0]
# تعيين اسم الخط إلى "Times New Roman"
port.portion_format.latin_font = slides.FontData("Times New Roman")
# تطبيق التصميم الجريء
port.portion_format.font_bold = slides.NullableBool.TRUE
# تطبيق التصميم المائل
port.portion_format.font_italic = slides.NullableBool.TRUE
# تسطير النص
port.portion_format.font_underline = slides.TextUnderlineType.SINGLE
# ضبط ارتفاع الخط إلى 25 نقطة
port.portion_format.font_height = 25
# تغيير لون النص إلى اللون الأزرق
color = drawing.Color.blue
port.portion_format.fill_format.fill_type = slides.FillType.SOLID
port.portion_format.fill_format.solid_fill_color.color = color
```
**توضيح:** 
- **اسم الخط**:تعيين عائلة الخط.
- **الأنماط الجريئة والمائلة**:قم بتعزيز التركيز عن طريق تبديل هذه الأنماط.
- **تسطير**:يضيف خطًا واحدًا أسفله للتمييز.
- **ارتفاع الخط**:ضبط حجم النص لتحسين الرؤية.
- **لون**:تغيير لون النص لجعله بارزًا.

#### الخطوة 5: احفظ العرض التقديمي الخاص بك
احفظ العرض التقديمي الخاص بك مع جميع التعديلات:

```python
def save_presentation(presentation, output_directory):
    presentation.save(f"{output_directory}/text_SetTextFontProperties_out.pptx", slides.export.SaveFormat.PPTX)
```
**توضيح:** ال `save` تكتب الطريقة العرض التقديمي المُعدَّل إلى ملف. تأكد من تحديد المسار بشكل صحيح لحفظه بنجاح.

- تحقق من توفر الخط إذا لم يتم تطبيقه بشكل صحيح.

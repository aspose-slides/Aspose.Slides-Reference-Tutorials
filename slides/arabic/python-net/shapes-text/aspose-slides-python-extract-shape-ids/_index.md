## دليل التنفيذ

في هذا القسم، سنقوم بتفصيل الخطوات اللازمة لاستخراج معرفات الأشكال من شريحة PowerPoint.

### ملخص

يُعد استخراج مُعرِّفات الأشكال أمرًا ضروريًا عند الحاجة إلى أتمتة تعديلات PowerPoint أو تنفيذ إجراءات محددة بناءً على بيانات الأشكال. توفر مكتبة Aspose.Slides وصولاً سلسًا إلى هذه الخصائص.

### التنفيذ خطوة بخطوة

#### الوصول إلى العرض التقديمي

أولاً، دعنا نفتح ملف PowerPoint الخاص بك:

```python
input_document_path = 'YOUR_DOCUMENT_DIRECTORY/welcome-to-powerpoint.pptx'

with slides.Presentation(input_document_path) as presentation:
    # سيتم وضع الكود الخاص بك للوصول إلى الأشكال هنا.
```

تفتح هذه القطعة ملف PowerPoint وتجهزه للتعامل معه.

#### الوصول إلى أشكال الشرائح

الآن قم بالوصول إلى الشريحة وأشكالها:

```python
slide = presentation.slides[0]  # احصل على الشريحة الأولى
shape = slide.shapes[0]          # احصل على الشكل الأول من هذه الشريحة
```

عن طريق الوصول `presentation.slides`يمكنك تكرار الشرائح في عرضك التقديمي. وبالمثل، `slide.shapes` يتيح لك التفاعل مع كل شكل على الشريحة.

#### استخراج معرف الشكل

أخيرًا، قم باستخراج وطباعة معرف شكل التفاعل بين Office:

```python
shape_id = shape.office_interop_shape_id  # استخراج معرف الشكل
print(str(shape_id))                      # اطبعها
```

### شرح المعلمات والطرق

- **`presentation.slides[0]`:** الوصول إلى الشريحة الأولى.
- **`slide.shapes[0]`:** استرجاع الشكل الأول من الشريحة الحالية.
- **`shape.office_interop_shape_id`:** خاصية تمنحك معرف التفاعل بين Office والشكل.

إذا واجهتك مشكلات، فتأكد من:

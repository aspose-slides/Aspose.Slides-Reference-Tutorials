# ربط الأشكال بالموصلات في بايثون باستخدام Aspose.Slides

## مقدمة

عند إنشاء العروض التقديمية، يُحسّن ربط العناصر المرئية وضوح رسالتك بشكل ملحوظ. سواءً كنت تُوضّح سير العمل أو تربط المفاهيم، تُسهّل الموصلات فهم العلاقات بين الأشكال المختلفة في العرض التقديمي. سيُرشدك هذا البرنامج التعليمي إلى كيفية استخدام Aspose.Slides لبايثون لربط شكلين - دائرة (قطع ناقص) ومستطيل - باستخدام موصل.

**ما سوف تتعلمه:**
- كيفية إعداد Aspose.Slides واستخدامه لـ Python.
- ربط الأشكال بالموصلات برمجيًا.
- تحسين عملية إنشاء العرض التقديمي الخاص بك.

دعونا نبدأ من خلال وضع الأساس أولاً.

قبل أن نبدأ، تأكد من أن لديك ما يلي:

لبدء استخدام Aspose.Slides لبايثون، عليك تثبيته. العملية بسيطة:

### التهيئة والإعداد الأساسي

فيما يلي كيفية تهيئة عرضك التقديمي الأول:

```python
import aspose.slides as slides

# إنشاء فئة عرض تقديمي تمثل ملف PPTX
class Presentation:
    def __enter__(self):
        self.pres = slides.Presentation()
        return self.pres

    def __exit__(self, exc_type, exc_val, exc_tb):
        del self.pres

with Presentation() as pres:
    # سيتم وضع الكود الخاص بك هنا
```

يؤدي هذا إلى إنشاء مثيل عرض تقديمي جديد حيث يمكنك إضافة الأشكال ومعالجتها.

## دليل التنفيذ

### ربط الأشكال باستخدام Aspose.Slides في Python

دعونا نقوم بتقسيم الخطوات لتوصيل شكلين باستخدام موصل.

**1. إضافة الأشكال**

ابدأ بإضافة شكل بيضاوي ومستطيل إلى الشريحة الخاصة بك:

```python
# الوصول إلى مجموعة الأشكال للشريحة المحددة
shapes = pres.slides[0].shapes

# أضف الشكل البيضاوي التلقائي في الموضع (0، 100) بعرض وارتفاع 100
elipse = shapes.add_auto_shape(slides.ShapeType.ELLIPSE, 0, 100, 100, 100)

# أضف شكل مستطيل تلقائي في الموضع (100، 300) بعرض وارتفاع 100
rectangle = shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 100, 300, 100, 100)
```

**2. إضافة موصل**

بعد ذلك، قم بإنشاء موصل لربط هذين الشكلين:

```python
# إضافة شكل الموصل إلى مجموعة أشكال الشريحة
contractor = shapes.add_connector(slides.ShapeType.BENT_CONNECTOR2, 0, 0, 10, 10)

# ربط الأشكال بالموصلات
contractor.start_shape_connected_to = elipse
contractor.end_shape_connected_to = rectangle

# إعادة توجيه المكالمة لتعيين أقصر مسار تلقائي بين الأشكال
contractor.reroute()
```

ال `add_connector` الطريقة تُنشئ شكل موصل منحني. `reroute()` تقوم هذه الوظيفة بضبط مسار الموصل تلقائيًا.

**3. حفظ العرض التقديمي الخاص بك**

وأخيرًا، احفظ العرض التقديمي الخاص بك:

```python
pres.save("YOUR_OUTPUT_DIRECTORY/shapes_connect_shapes_using_connectors_out.pptx", slides.export.SaveFormat.PPTX)
```

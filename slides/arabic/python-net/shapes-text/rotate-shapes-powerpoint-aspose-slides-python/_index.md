## دليل التنفيذ

الآن بعد أن قمت بالإعداد، دعنا ننفذ دوران الشكل خطوة بخطوة:

### إضافة الأشكال وتدويرها في PowerPoint

#### ملخص
يركز هذا القسم على إضافة شكل مستطيل إلى شريحة وتدويره بمقدار 90 درجة.

#### التنفيذ خطوة بخطوة

##### تهيئة العرض التقديمي

ابدأ بإنشاء مثيل لـ `Presentation` الفئة التي تمثل ملف PPTX الخاص بك:
```python
with slides.Presentation() as pres:
    # سنعمل ضمن مدير السياق هذا لإدارة الموارد بكفاءة.
```

##### الوصول إلى الشريحة وإضافة الشكل

انتقل إلى الشريحة الأولى في العرض التقديمي وأضف شكل مستطيل:
```python
slide = pres.slides[0]

shape = slide.shapes.add_auto_shape(
    slides.ShapeType.RECTANGLE, 50, 150, 75, 150)
# تعرف المعلمات الموضع (x، y) والحجم (العرض، الارتفاع).
```

##### تدوير الشكل

قم بتدوير الشكل المضاف حديثًا عن طريق ضبط خاصية التدوير الخاصة به:
```python
shape.rotation = 90
# يتم ضبط الدوران بالدرجات.
```

##### حفظ العرض التقديمي

وأخيرًا، احفظ التغييرات في دليل الإخراج المحدد:
```python
pres.save("YOUR_OUTPUT_DIRECTORY/shapes_rotate_out.pptx", slides.export.SaveFormat.PPTX)
# تأكد من وجود المسار أو قم بتعديله وفقًا لذلك.
```

#### نصائح استكشاف الأخطاء وإصلاحها
- **الشكل غير ظاهر**: تحقق من معلمات الموضع والحجم. إذا كانت القيم خارج الشاشة، فاضبطها.
- **قضايا الدوران**:تأكد من ذلك `shape.rotation` تم ضبطه بشكل صحيح؛ تأكد من عدم وجود تحويلات متضاربة.

# كيفية إنشاء أشكال مخصصة مركبة في PowerPoint باستخدام Aspose.Slides لـ Python

## مقدمة
غالبًا ما يتطلب إنشاء عروض تقديمية جذابة بصريًا أشكالًا مخصصة تتجاوز الخيارات الأساسية المتاحة في PowerPoint. يوفر Aspose.Slides for Python ميزات متقدمة، بما في ذلك إنشاء أشكال مركبة. سواء كنت تصمم عرضًا تقديميًا لشركة أو عرض شرائح تعليميًا، فإن إتقان هذه الميزة يرتقي بشرائحك إلى مستويات جديدة من الاحترافية والإبداع.

في هذا البرنامج التعليمي، سنستكشف كيفية إنشاء أشكال مركبة باستخدام اثنين `GeometryPath` الكائنات باستخدام Aspose.Slides في بايثون. بنهاية هذا الدليل، ستفهم:
- إعداد Aspose.Slides في بيئة Python الخاصة بك
- إنشاء مسارات هندسية مخصصة
- دمج مسارات متعددة في شكل واحد
- حفظ العرض التقديمي الخاص بك

دعونا نبدأ بالتأكد من أن لدينا كل ما نحتاجه للمتابعة.

## المتطلبات الأساسية
قبل الغوص في الكود، تأكد من أن لديك ما يلي:
- **بيئة بايثون**:تأكد من تثبيت Python (الإصدار 3.6 أو أعلى) على نظامك.
- **مكتبة Aspose.Slides لـ Python**يستخدم هذا البرنامج التعليمي Aspose.Slides لمعالجة عروض PowerPoint التقديمية. ثبّته عبر pip.
- **أدوات التطوير**:سيكون محرر الكود مثل VSCode أو PyCharm أو أي IDE من اختيارك مفيدًا.

استيراد Aspose.Slides إلى البرنامج النصي Python الخاص بك:

```python
import aspose.slides as slides
```

## دليل التنفيذ
بعد إعداد البيئة، دعنا نقوم بإنشاء شكل مخصص مركب في PowerPoint.

### الخطوة 1: تهيئة العرض التقديمي
ابدأ بإنشاء كائن عرض تقديمي جديد، ليكون بمثابة لوحة للأشكال والتصميمات.

```python
with slides.Presentation() as pres:
    # يوجد هنا الكود الخاص بالتلاعب بالشرائح.
```
ال `with` تضمن هذه العبارة إدارة فعالة للموارد، وتغلق العرض التقديمي تلقائيًا عند الانتهاء.

### الخطوة 2: إضافة شكل مستطيل
أضف شكلًا تلقائيًا من نوع مستطيل إلى الشريحة الأولى. هذا الشكل هو أساس تخصيص المُركّب.

```python
shape = pres.slides[0].shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 100, 100, 200, 100)
```
هنا، `add_auto_shape` ينشئ مستطيلاً بمعلمات موضع وحجم محددة (x، y، العرض، الارتفاع).

### الخطوة 3: إنشاء مسار الهندسة الأول
قم بتحديد الجزء العلوي من الشكل المركب الخاص بك باستخدام `GeometryPath`يتضمن ذلك التحرك إلى إحداثيات محددة ورسم الخطوط.

```python
g = slides.GeometryPath()
g.move_to(0, 0)  # ابدأ من الأصل (الزاوية العلوية اليسرى).
g.line_to(shape.width, 0)  # ارسم خطًا عبر الجزء العلوي.
g.line_to(shape.width, shape.height / 3)  # انتقل إلى الأسفل إلى ارتفاع الثلث.
g.line_to(0, shape.height / 3)  # العودة إلى الحافة اليسرى على ارتفاع الثلث.
g.close_figure()  # أغلق المسار لتشكيل شكل مغلق.
```

### الخطوة 4: إنشاء مسار الهندسة الثاني
وبالمثل، قم بتحديد الجزء السفلي من الشكل المركب الخاص بك باستخدام شكل آخر `GeometryPath`.

```python
g1 = slides.GeometryPath()
g1.move_to(0, shape.height / 3 * 2)  # ابدأ من ارتفاع ثلثي الارتفاع.
g1.line_to(shape.width, shape.height / 3 * 2)  # ارسم خطًا عبر الحافة السفلية.
g1.line_to(shape.width, shape.height)  # انتقل إلى الأسفل إلى الزاوية اليمنى السفلية.
g1.line_to(0, shape.height)  # العودة إلى الزاوية اليسرى السفلية.
g1.close_figure()  # أغلق المسار لتشكيل شكل مغلق.
```

### الخطوة 5: دمج مسارات الهندسة
دمج كلا مسارات الهندسة في شكل مخصص مركب واحد باستخدام `set_geometry_paths`.

```python
shape.set_geometry_paths([g, g1])
```
تقوم هذه الخطوة بدمج المسارين المنفصلين في شكل متماسك واحد ضمن الشريحة الخاصة بك.

### الخطوة 6: احفظ العرض التقديمي الخاص بك
وأخيرًا، احفظ العرض التقديمي الخاص بك في الدليل المحدد.

```python
pres.save("YOUR_OUTPUT_DIRECTORY/shapes_create_geometry_path_out.pptx", slides.export.SaveFormat.PPTX)
```
يستبدل `YOUR_OUTPUT_DIRECTORY` مع المسار الفعلي الذي تريد تخزين ملفك فيه.

## دليل التنفيذ

الآن، دعونا ننفذ إضافة صورة كإطار للصورة.

### إضافة صورة كإطار صورة (نظرة عامة على الميزات)

تتضمن هذه الميزة تحميل صورة ووضعها ضمن شريحة كإطار. وهي مفيدة لتخصيص العروض التقديمية بعناصر مرئية مدمجة بسلاسة في الشرائح.

#### الخطوة 1: إنشاء فئة العرض التقديمي

إنشاء كائن عرض تقديمي يمثل ملف PPTX الخاص بك:

```python
import aspose.slides as slides

# تهيئة العرض التقديمي
total_presentation = slides.Presentation()
try:
    # سيتم وضع الكود الخاص بالتلاعب بالشريحة هنا
finally:
    total_presentation.dispose()
```

#### الخطوة 2: الحصول على الشريحة الأولى

الوصول إلى الشريحة الأولى من العرض التقديمي:

```python
# الوصول إلى الشريحة الأولى
slide = total_presentation.slides[0]
```

#### الخطوة 3: تحميل صورة من دليل المستندات

حمّل ملف الصورة المطلوب إلى العرض التقديمي. استبدله `'YOUR_DOCUMENT_DIRECTORY/'` مع المسار الفعلي لصورك.

```python
# تحميل صورة
image_to_add = slides.Images.from_file('YOUR_DOCUMENT_DIRECTORY/image1.jpg')
```

#### الخطوة 4: إضافة الصورة المحملة إلى مجموعة صور العرض التقديمي

أضف الصورة المحملة إلى مجموعة الصور التي تتم إدارتها بواسطة العرض التقديمي:

```python
# إضافة صورة إلى مجموعة صور العرض التقديمي
image_in_presentation = total_presentation.images.add_image(image_to_add)
```

#### الخطوة 5: إضافة إطار صورة على الشريحة

الآن، أضف إطار الصورة بالأبعاد المحددة وضعه في المكان المطلوب داخل الشريحة:

```python
# إضافة إطار الصورة إلى الشريحة
drawable_shape = slide.shapes.add_picture_frame(
    slides.ShapeType.RECTANGLE,  # نوع الشكل للمستطيل
    50,                          # إحداثيات X للزاوية العلوية اليسرى
    150,                         # إحداثي Y للزاوية العلوية اليسرى
    image_in_presentation.width, # عرض الصورة
    image_in_presentation.height,# ارتفاع الصورة
    image_in_presentation        # كائن الصورة المراد إضافته
)
```

#### الخطوة 6: حفظ العرض التقديمي

وأخيرًا، احفظ العرض التقديمي الخاص بك باستخدام إطار الصورة الجديد:

```python
# حفظ العرض التقديمي المحدث
total_presentation.save('YOUR_OUTPUT_DIRECTORY/shapes_add_stretch_offset_out.pptx', slides.export.SaveFormat.PPTX)
```

## قسم الأسئلة الشائعة

### س1: هل يمكنني إضافة صور متعددة إلى شريحة واحدة؟

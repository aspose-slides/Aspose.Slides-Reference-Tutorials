# كيفية تضمين إطارات الفيديو في شرائح PowerPoint باستخدام Aspose.Slides لـ Python

## مقدمة

هل تواجه صعوبة في إضافة مقاطع فيديو مباشرةً إلى شرائح PowerPoint؟ مع Aspose.Slides لـ Python، أصبح تضمين إطارات الفيديو في عروض PowerPoint التقديمية سهلاً وفعالاً. سيرشدك هذا البرنامج التعليمي خلال عملية دمج محتوى الفيديو بسلاسة.

**ما سوف تتعلمه:**
- كيفية تضمين إطار فيديو في شريحة PowerPoint باستخدام Aspose.Slides.
- خطوات تحميل وإدارة مقاطع الفيديو ضمن العرض التقديمي.
- خيارات التكوين الرئيسية لإعدادات تشغيل الفيديو في PowerPoint.

دعونا نتأكد من إعداد كل شيء بشكل صحيح قبل أن نبدأ في تضمين هذه الفيديوهات!

## المتطلبات الأساسية

قبل أن نبدأ، تأكد من أن لديك ما يلي:
- **Aspose.Slides لـ Python**:مكتبة أساسية لإنشاء عروض PowerPoint والتلاعب بها.
- **بيئة بايثون**:تأكد من تثبيت إصدار متوافق من Python (يفضل Python 3.6 أو إصدار أحدث).
- **معرفة التثبيت**:فهم أساسي لتثبيت المكتبات باستخدام pip.

## إعداد Aspose.Slides لـ Python

أولاً، قم بتثبيت مكتبة Aspose.Slides عن طريق تشغيل:

```bash
pip install aspose.slides
```

بعد ذلك، احصل على ترخيص كامل الوظائف. يمكنك البدء بفترة تجريبية مجانية أو التقدم بطلب للحصول على ترخيص مؤقت على [موقع Aspose](https://purchase.aspose.com/temporary-license/).

فيما يلي كيفية تهيئة الإعداد الخاص بك باستخدام Aspose.Slides:

```python
import aspose.slides as slides
# تهيئة كائن العرض التقديمي
pres = slides.Presentation()
```

## دليل التنفيذ

سنقوم بتقسيم التنفيذ إلى ميزتين رئيسيتين: تضمين إطار فيديو وتحميل فيديو.

### الميزة 1: تضمين إطار فيديو

تتيح لك هذه الميزة تضمين مقطع فيديو مباشرةً على الشريحة الأولى من عرض PowerPoint التقديمي.

#### التنفيذ خطوة بخطوة
**الخطوة 1:** إنشاء كائن عرض تقديمي جديد.

```python
with slides.Presentation() as pres:
    # الخطوات التالية تذهب إلى هنا...
```

**الخطوة 2:** الوصول إلى الشريحة الأولى.

```python
slide = pres.slides[0]
```

**الخطوة 3:** قم بتحميل الفيديو وإضافته إلى العرض التقديمي.

تأكد من تجهيز ملف الفيديو. سنستخدم مسارًا نموذجيًا. `video.mp4` على سبيل المثال.

```python
video_path = "YOUR_DOCUMENT_DIRECTORY/video.mp4"
with open(video_path, "rb") as in_file:
    video = pres.videos.add_video(in_file, slides.LoadingStreamBehavior.READ_STREAM_AND_RELEASE)
```

**الخطوة 4:** إضافة إطار فيديو إلى الشريحة.

قم بتحديد موضع وحجم إطار الفيديو الخاص بك وفقًا لتخطيط الشريحة الخاصة بك.

```python
vf = slide.shapes.add_video_frame(50, 150, 300, 350, video)
```

**الخطوة 5:** تعيين الفيديو المضمن للإطار.

ربط الفيديو المحمّل بالإطار المخصص له.

```python
vf.embedded_video = video
```

**الخطوة 6:** ضبط وضع التشغيل ومستوى الصوت للفيديو.

قم بتخصيص كيفية تشغيل الفيديو الخاص بك في وضع العرض التقديمي.

```python
vf.play_mode = slides.VideoPlayModePreset.AUTO
vf.volume = slides.AudioVolumeMode.LOUD
```

**الخطوة 7:** احفظ العرض التقديمي باستخدام الفيديو المضمن.

اختر دليل الإخراج لحفظ ملف PowerPoint الخاص بك.

```python
output_path = "YOUR_OUTPUT_DIRECTORY/shapes_embed_video_frame_out.pptx"
pres.save(output_path, slides.export.SaveFormat.PPTX)
```

### الميزة 2: تحميل مقطع فيديو في عرض تقديمي

تُظهر هذه الميزة كيفية تحميل مقطع فيديو إلى مجموعة العرض التقديمي دون تضمينه في أي إطار محدد.

#### التنفيذ خطوة بخطوة
**الخطوة 1:** إنشاء كائن عرض تقديمي جديد.

```python
with slides.Presentation() as pres:
    # الخطوات التالية تذهب إلى هنا...
```

**الخطوة 2:** تحميل الفيديو من الدليل.

```python
video_path = "YOUR_DOCUMENT_DIRECTORY/video.mp4"
with open(video_path, "rb") as in_file:
    video = pres.videos.add_video(in_file, slides.LoadingStreamBehavior.READ_STREAM_AND_RELEASE)
```

لا يلزم اتخاذ أي خطوات أخرى إذا كنت تقوم فقط بتحميل مقاطع الفيديو لاستخدامها لاحقًا أو للرجوع إليها.

## دليل التنفيذ

دعونا نوضح الخطوات اللازمة لإضافة إطارات الصور وتنسيقها.

### الخطوة 1: إنشاء عرض تقديمي جديد (H3)

ابدأ بتهيئة كائن عرض تقديمي جديد في PowerPoint. سيُستخدم هذا الكائن كلوحة عرض لجميع التعديلات.

```python
with slides.Presentation() as pres:
    # يمثل المتغير 'pres' الآن عرضنا التقديمي.
```

**غاية**:يؤسس الأساس لإضافة الشرائح والمحتوى.

### الخطوة 2: الوصول إلى الشريحة الأولى (H3)

انتقل إلى الشريحة الأولى لإضافة إطار الصورة. في PowerPoint، يبدأ كل عرض تقديمي بشريحة واحدة افتراضيًا.

```python
slide = pres.slides[0]
# يشير "الشريحة" الآن إلى الشريحة الأولى في عرضنا التقديمي.
```

**غاية**:يسمح لنا باستهداف شرائح محددة وتعديلها ضمن العرض التقديمي.

### الخطوة 3: تحميل صورة (H3)

حمّل الصورة التي اخترتها من مجلدها. ستُستخدم هذه الصورة كإطار.

```python
img_path = "YOUR_DOCUMENT_DIRECTORY/image1.jpg"
with open(img_path, 'rb') as img_file:
    imgx = pres.images.add_image(drawing.Image.load(img_file))
# "imgx" هو الآن كائن الصورة المحملة المضافة إلى العرض التقديمي.
```

**غاية**:تحضير الصورة لإدراجها في الشريحة.

### الخطوة 4: إضافة إطار الصورة (H3)

أدخل إطار الصورة المُحمّلة على الشريحة المستهدفة. حدّد موضعها وحجمها هنا.

```python
cf = slide.shapes.add_picture_frame(
    slides.ShapeType.RECTANGLE, 50, 150, imgx.width, imgx.height, imgx)
# يمثل 'cf' إطار الصورة المضاف حديثًا.
```

**شرح المعلمات**: 
- `ShapeType.RECTANGLE`:يحدد شكل الإطار.
- `(50, 150)`:إحداثيات X وY للموضع على الشريحة.
- `imgx.width`، `imgx.height`:أبعاد الصورة.

### الخطوة 5: تطبيق التنسيق (H3)

قم بتخصيص إطار صورتك بلون الحدود وعرض الخط وزاوية الدوران لتحسين مظهره.

```python
cf.line_format.fill_format.fill_type = slides.FillType.SOLID
cf.line_format.fill_format.solid_fill_color.color = drawing.Color.blue
cf.line_format.width = 20
cf.rotation = 45
# تعمل هذه الإعدادات على تعديل نمط حدود الإطار.
```

**خيارات التكوين**: 
- **نوع التعبئة**:لون ثابت لحدود الإطار.
- **لون**:قابلة للتخصيص لأي `drawing.Color` قيمة.
- **عرض**:سمك خط الحدود.
- **تناوب**:زاوية إطار الصورة.

### الخطوة 6: حفظ العرض التقديمي (H3)

أخيرًا، احفظ عرضك التقديمي مع جميع التعديلات التي أجريتها. حدد مجلدًا واسم ملف لسهولة الوصول إليه لاحقًا.

```python
output_path = "YOUR_OUTPUT_DIRECTORY/shapes_picture_frame_format_out.pptx"
pres.save(output_path, slides.export.SaveFormat.PPTX)
# يتم حفظ العرض التقديمي المعدّل في المسار المحدد.
```

**غاية**:يضمن الحفاظ على كافة أعمالك بتنسيق ملف جديد.

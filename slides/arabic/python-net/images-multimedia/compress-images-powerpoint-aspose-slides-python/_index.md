## دليل التنفيذ
### الوصول إلى الشرائح وتعديلها
#### ملخص
لضغط صورة داخل عرض تقديمي، عليك أولاً الوصول إلى الشريحة المحددة وإطار الصورة. إليك كيفية تحقيق ذلك باستخدام Aspose.Slides:
#### التنفيذ خطوة بخطوة
**1. تحميل العرض التقديمي:**
```python
import aspose.slides as slides
import os

document_path = "YOUR_DOCUMENT_DIRECTORY/CroppedImage.pptx"
output_path = "YOUR_OUTPUT_DIRECTORY/CroppedImage-Compress-out.pptx"

with slides.Presentation(document_path) as presentation:
```
*توضيح*:استخدم مدير السياق لفتح ملف PowerPoint، وتأكد من إغلاقه بشكل صحيح بعد المعالجة.
**2. الوصول إلى الشريحة الأولى:**
```python
    slide = presentation.slides[0]
```
*توضيح*:يؤدي هذا إلى استرجاع الشريحة الأولى في العرض التقديمي الخاص بك.
**3. احصل على إطار الصورة:**
```python
    picture_frame = slide.shapes[0]  # يفترض أن الشكل الأول هو إطار الصورة
```
*توضيح*نفترض أن الشكل الأول في الشريحة هو إطار صورة (PictureFrame). عدّل هذا إذا لزم الأمر بناءً على حالة استخدامك الخاصة.
**4. ضغط الصورة:**
```python
    compression_result = picture_frame.picture_format.compress_image(True, 150)
```
*توضيح*: ال `compress_image` تقلل الطريقة دقة الصورة إلى 150 نقطة في البوصة، وهي مناسبة للاستخدام على الويب مع الحفاظ على أحجام الملفات قابلة للإدارة.
**5. احفظ العرض التقديمي:**
```python
    presentation.save(output_path, slides.export.SaveFormat.PPTX)

# عرض أحجام المصدر والعروض التقديمية الناتجة للمقارنة
original_size = os.stat(document_path).st_size
compressed_size = os.stat(output_path).st_size
print("Source presentation size:", original_size)  # بالبايتات
print("Compressed presentation size:", compressed_size)  # بالبايتات
```
*توضيح*يُحفظ العرض التقديمي مع الصورة الجديدة المضغوطة. كما نطبع أحجام الملفات لعرض حجم التخفيض المُحقق.
### نصائح استكشاف الأخطاء وإصلاحها
- **خطأ في تحديد الصورة**:تأكد من أن الصورة التي تريد ضغطها هي بالفعل الشكل الأول على الشريحة الخاصة بك.
- **أخطاء مسار الملف**:تحقق جيدًا من المسارات للتأكد من تحديدها بشكل صحيح وإمكانية الوصول إليها.

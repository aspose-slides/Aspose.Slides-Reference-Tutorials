# كيفية إنشاء صور مصغرة بأحجام مخصصة باستخدام Aspose.Slides لـ Python

- أساسيات إنشاء صور مصغرة ذات حجم مخصص من شرائح PowerPoint
- كيفية إعداد Aspose.Slides واستخدامه في بيئة Python
- تنفيذ التعليمات البرمجية خطوة بخطوة لإنشاء الصورة المصغرة
- التطبيقات العملية واعتبارات الأداء

لنستعرض كيفية تطبيق هذه الميزة بسلاسة في مشاريعك. أولًا، تأكد من توفر المتطلبات الأساسية اللازمة.

## إعداد Aspose.Slides لـ Python
### تثبيت


## دليل التنفيذ
يرشدك هذا القسم خلال عملية إنشاء صور مصغرة ذات حجم مخصص من شرائح PowerPoint.

### نظرة عامة على الميزة
تتيح لك هذه الميزة تحديد الأبعاد المطلوبة للصور المصغرة للشرائح وإنشاءها برمجيًا.

#### الخطوة 1: تحديد مسارات الإدخال والإخراج
حدد مكان وجود ملف PowerPoint المدخل والمكان الذي تريد حفظ صورة المصغرة الناتجة فيه:
```python
input_file = "YOUR_DOCUMENT_DIRECTORY/welcome-to-powerpoint.pptx"
output_file = "YOUR_OUTPUT_DIRECTORY/thumbnail_user_defined_dimensions_out.jpg"
```

#### الخطوة 2: افتح العرض التقديمي
استخدم Aspose.Slides لفتح ملف العرض التقديمي. هذه الخطوة أساسية للوصول إلى شرائحه:
```python
import aspose.slides as slides

with slides.Presentation(input_file) as pres:
    slide = pres.slides[0]
```

#### الخطوة 3: تحديد الأبعاد المطلوبة
حدّد الأبعاد التي تريدها للصورة المصغّرة. في هذا المثال، ضبطناها على ١٢٠٠ × ٨٠٠ بكسل:
```python
desired_x, desired_y = 1200, 800
scale_x = (1.0 / pres.slide_size.size.width) * desired_x
scale_y = (1.0 / pres.slide_size.size.height) * desired_y
```

#### الخطوة 4: إنشاء الصورة المصغرة وحفظها
إنشاء الصورة المصغرة باستخدام المقاييس المحسوبة وحفظها كملف JPEG:
```python
img = slide.get_image(scale_x, scale_y)
img.save(output_file, slides.ImageFormat.JPEG)
```

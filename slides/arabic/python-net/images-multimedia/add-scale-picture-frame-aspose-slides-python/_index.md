# كيفية إضافة إطار صورة وتغيير حجمه في PowerPoint باستخدام Aspose.Slides لـ Python

## مقدمة

في هذه المقالة، سنشرح كيفية إضافة إطارات الصور وتغيير حجمها في شرائح PowerPoint بسهولة. ستتعلم:
- كيفية إعداد Aspose.Slides لـ Python
- تقنيات إضافة الصور باستخدام مقياس نسبي
- التطبيقات العملية لهذه التقنيات في سيناريوهات العالم الحقيقي

## دليل التنفيذ
في هذا القسم، سنقوم بتنفيذ ميزتين أساسيتين: إضافة إطار صورة مع مقياس نسبي وتحميل صورة في العرض التقديمي.

### الميزة 1: إضافة إطار صورة بمقياس نسبي
#### ملخص
توضح هذه الميزة كيفية إضافة إطار صورة إلى الشريحة الأولى من عرض PowerPoint التقديمي وضبط مقياس العرض والارتفاع.

#### التنفيذ خطوة بخطوة
##### **إعداد كائن العرض التقديمي**
ابدأ بإنشاء كائن عرض تقديمي باستخدام Aspose.Slides. هذا يضمن إدارة سليمة للموارد:

```python
def add_relative_scale_picture_frame():
    with slides.Presentation() as presentation:
```

##### **تحميل الصورة**
بعد ذلك، قم بتحميل الصورة المطلوبة إلى مجموعة صور العرض التقديمي:

```python
        img = slides.Images.from_file('YOUR_DOCUMENT_DIRECTORY/image1.jpg')
        image = presentation.images.add_image(img)
```

**توضيح**: ال `Images.from_file()` تقوم الطريقة بتحميل صورة من مسار محدد وإضافتها إلى مجموعة العرض التقديمي.

##### **إضافة إطار الصورة**
الآن، قم بإضافة إطار الصورة إلى الشريحة الأولى بأبعاد محددة:

```python
        pf = presentation.slides[0].shapes.add_picture_frame(
            slides.ShapeType.RECTANGLE, 50, 50, 100, 100, image
        )
```

**توضيح**: ال `add_picture_frame()` تضع هذه الطريقة إطارًا مستطيلًا عند الإحداثيات (٥٠، ٥٠) بعرض وارتفاع ١٠٠ وحدة. تُحدد المعلمات نوع الشكل، وموقعه، وحجمه، وصورته.

##### **تعيين العرض والارتفاع النسبيين للمقياس**
ضبط المقياس للحصول على مظهر جذاب:

```python
        pf.relative_scale_height = 0.8
        pf.relative_scale_width = 1.35
```

**توضيح**:تتيح لك هذه الخصائص ضبط ارتفاع وعرض الإطار بشكل ديناميكي بالنسبة لحجمه الأصلي.

##### **حفظ العرض التقديمي**
وأخيرًا، احفظ العرض التقديمي في الدليل المطلوب:

```python
        presentation.save('YOUR_OUTPUT_DIRECTORY/shapes_add_relative_scale_picture_frame_out.pptx',
                          slides.export.SaveFormat.PPTX)
```

### الميزة 2: تحميل الصورة وإضافتها إلى العرض التقديمي
#### ملخص
ترتكز هذه الميزة على تحميل صورة من نظام الملفات وإضافتها إلى مجموعة العرض التقديمي الخاص بك.

#### التنفيذ خطوة بخطوة
##### **تحميل الصورة**
استخدم نفس الطريقة المذكورة أعلاه:

```python
def load_and_add_image():
    with slides.Presentation() as presentation:
        img = slides.Images.from_file('YOUR_DOCUMENT_DIRECTORY/image1.jpg')
        image = presentation.images.add_image(img)
```

**ملحوظة**:لا تقوم هذه الوظيفة بحفظ العرض التقديمي أو عرضه، ولكنها توضح كيفية التعامل مع الصور.

## دليل التنفيذ

يرشدك هذا القسم إلى كيفية إضافة الصوت باستخدام تأثيرات التلاشي التدريجي إلى شريحة PowerPoint.

### إضافة إطار صوتي

**ملخص:**
يُعزز تضمين ملف صوتي في عرضك التقديمي التفاعل. تتيح لك هذه الميزة وضع الصوت مباشرةً داخل الشريحة لتشغيله أثناء العرض التقديمي.

#### الخطوة 1: تحميل العرض التقديمي الخاص بك

ابدأ بإنشاء عرض تقديمي أو فتحه:

```python
import aspose.slides as slides

def set_audio_fade_in_out():
    with slides.Presentation() as document:
        # تحميل ملف الصوت في الوضع الثنائي
        with open("YOUR_DOCUMENT_DIRECTORY/audio.m4a", "rb") as in_file:
            # أضف الصوت إلى العرض التقديمي الخاص بك
            audio = document.audios.add_audio(in_file)
```

**توضيح:**
- ال `Presentation()` يضمن مدير السياق إدارة الموارد بشكل صحيح.
- فتح ملف صوتي (`audio.m4a`) في وضع القراءة الثنائية للتضمين.

#### الخطوة 2: تضمين إطار الصوت

بعد ذلك، قم بتضمين الصوت في شريحة:

```python
        # إضافة إطار صوتي مضمن إلى الشريحة الأولى
        audio_frame = document.slides[0].shapes.add_audio_frame_embedded(50, 50, 100, 100, audio)
```

**توضيح:**
- `add_audio_frame_embedded()` يضع الصوت في إحداثيات محددة (x=50، y=50) بحجم 100 × 100 بكسل.
- هذه الطريقة تعيد `AudioFrame` كائن لمزيد من التخصيص.

#### الخطوة 3: تعيين مدة التلاشي

تكوين مدة التلاشي التدريجي والتلاشي التدريجي:

```python
        # تكوين تأثيرات التلاشي التدريجي والتلاشي التدريجي
        audio_frame.fade_in_duration = 200  # 200 ميلي ثانية
        audio_frame.fade_out_duration = 500  # 500 ميلي ثانية
```

**توضيح:**
- `fade_in_duration` و `fade_out_duration` يتم ضبطها بالمللي ثانية، مما يوفر انتقالات سلسة في بداية ونهاية الصوت.

#### الخطوة 4: حفظ العرض التقديمي

وأخيرًا، احفظ العرض التقديمي المحدث:

```python
        # حفظ التغييرات في ملف جديد
        document.save("YOUR_OUTPUT_DIRECTORY/AudioFrameFade_out.pptx", slides.export.SaveFormat.PPTX)
```

**توضيح:**
- ال `save()` تكتب الطريقة العرض التقديمي الخاص بك مع جميع التعديلات على المسار المحدد.

### وظيفة كاملة

هكذا تبدو الوظيفة الكاملة:

```python
def set_audio_fade_in_out():
    with slides.Presentation() as document:
        with open("YOUR_DOCUMENT_DIRECTORY/audio.m4a", "rb") as in_file:
            audio = document.audios.add_audio(in_file)
        
        audio_frame = document.slides[0].shapes.add_audio_frame_embedded(50, 50, 100, 100, audio)
        
        audio_frame.fade_in_duration = 200
        audio_frame.fade_out_duration = 500
        
        document.save("YOUR_OUTPUT_DIRECTORY/AudioFrameFade_out.pptx", slides.export.SaveFormat.PPTX)

set_audio_fade_in_out()
```

يوفر هذا القسم تعليمات مفصلة حول تنفيذ إنشاء الصور المصغرة مع التدرج في PowerPoint باستخدام Aspose.Slides.

### الخطوة 1: تحميل ملف العرض التقديمي

ابدأ بتحميل ملف العرض التقديمي. هذه الخطوة أساسية للوصول إلى الشريحة والشكل الذي ترغب في إنشاء صورة مصغرة منه.

```python
# قم بتحميل العرض التقديمي مع slides.Presentation('YOUR_DOCUMENT_DIRECTORY/welcome-to-powerpoint.pptx') كعرض تقديمي:
    # الوصول إلى الشريحة الأولى
    shape = pres.slides[0].shapes[0]
```

**توضيح**:هنا، نفتح ملف PowerPoint ونصل إلى الشريحة الأولى. `shape` يشير المتغير إلى الشكل الأول في هذه الشريحة.

### الخطوة 2: إنشاء صورة مصغرة باستخدام عوامل القياس

بعد ذلك، قم بإنشاء الصورة المصغرة باستخدام عوامل القياس المحددة للعرض والارتفاع.

```python
# تحديد عوامل القياس (عامل العرض = 2، عامل الارتفاع = 2)
with shape.get_image(slides.ShapeThumbnailBounds.SHAPE, 2, 2) as image:
    # حفظ الصورة المُنشأة في ملف PNG
    image.save('YOUR_OUTPUT_DIRECTORY/shapes_create_scaling_thumbnail_out.png', slides.ImageFormat.PNG)
```

**توضيح**: ال `get_image` تُولّد هذه الطريقة صورةً للشكل بعوامل القياس المُحدّدة. نحفظ هذه الصورة بصيغة PNG، لضمان جودة عالية.

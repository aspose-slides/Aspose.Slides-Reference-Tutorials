**ما سوف تتعلمه:**
- كيفية إضافة ترجمات الفيديو من ملف VTT.
- تقنيات لاستخراج وإزالة التسميات التوضيحية الموجودة.
- أفضل الممارسات لتحسين الأداء باستخدام Aspose.Slides.

## دليل التنفيذ
سنستكشف ميزتين رئيسيتين: إضافة التعليقات التوضيحية وإزالتها من مقاطع الفيديو المضمنة في عروض PowerPoint التقديمية.

### إضافة تعليقات توضيحية إلى إطار الفيديو
تتيح لك هذه الميزة تعزيز إمكانية الوصول إلى محتوى الفيديو الخاص بك من خلال تضمين ترجمات أو تعليقات توضيحية مباشرة داخل العرض التقديمي الخاص بك.

#### الخطوة 1: إنشاء عرض تقديمي وتحميله
ابدأ بإنشاء كائن عرض تقديمي جديد:

```python
import aspose.slides as slides

def add_video_captions():
    # إنشاء عرض تقديمي جديد
    with slides.Presentation() as pres:
        ...
```

#### الخطوة 2: إضافة ملف الفيديو
حمّل ملف الفيديو إلى العرض التقديمي. تأكد من صحة مسار الفيديو:

```python
        with open("YOUR_DOCUMENT_DIRECTORY/NewVideo.mp4", "rb") as f:
            video = pres.videos.add_video(f.read())
```

#### الخطوة 3: إدراج إطار فيديو وإضافة التسميات التوضيحية
أدخل `VideoFrame` في الموضع المطلوب وأضف التعليقات التوضيحية باستخدام ملف VTT الخاص بك:

```python
        # أضف إطار فيديو بأبعاد محددة
        video_frame = pres.slides[0].shapes.add_video_frame(0, 0, 100, 100, video)
        
        # إرفاق مسار التسمية التوضيحية من ملف VTT
        video_frame.caption_tracks.add("New track", "YOUR_DOCUMENT_DIRECTORY/bunny.vtt")
```

#### الخطوة 4: حفظ العرض التقديمي
وأخيرًا، احفظ عرضك التقديمي المحدث مع التسميات التوضيحية:

```python
        pres.save("YOUR_OUTPUT_DIRECTORY/VideoCaptionsAdd_out.pptx", slides.export.SaveFormat.PPTX)
```

### استخراج وإزالة التسميات التوضيحية من إطار الفيديو
الآن بعد أن قمت بإضافة التسميات التوضيحية، دعنا نستكشف كيفية استخراجها للمراجعة أو إزالتها بالكامل.

#### الخطوة 1: فتح عرض تقديمي موجود
ابدأ بتحميل العرض التقديمي الذي يحتوي على الفيديو الخاص بك مع التعليقات التوضيحية:

```python
def extract_and_remove_captions():
    # تحميل العرض التقديمي الحالي
    with slides.Presentation("YOUR_OUTPUT_DIRECTORY/VideoCaptionsAdd_out.pptx") as pres:
        ...
```

#### الخطوة 2: استخراج بيانات التسمية التوضيحية
قم بالتكرار خلال كل مسار تسمية توضيحية لحفظ بياناته في ملفات VTT:

```python
        video_frame = pres.slides[0].shapes[0]
        if video_frame is not None:
            for idx, caption_track in enumerate(video_frame.caption_tracks):
                with open(f"YOUR_OUTPUT_DIRECTORY/VideoCaption_out_{idx}.vtt", "wb") as f:
                    f.write(caption_track.binary_data)
```

#### الخطوة 3: إزالة التسميات التوضيحية
مسح جميع التسميات التوضيحية من إطار الفيديو:

```python
            # مسح جميع مسارات الترجمة التوضيحية
            video_frame.caption_tracks.clear()
            
            # حفظ التغييرات في ملف جديد
            pres.save("YOUR_OUTPUT_DIRECTORY/VideoCaptionsRemove_out.pptx", slides.export.SaveFormat.PPTX)
```

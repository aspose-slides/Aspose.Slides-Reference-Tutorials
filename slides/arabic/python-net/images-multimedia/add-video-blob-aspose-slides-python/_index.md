## دليل التنفيذ

الآن، دعنا نستعرض عملية إضافة كتلة فيديو إلى عرض PowerPoint التقديمي الخاص بك.

### 1. إعداد بيئتك

ابدأ بإعداد الدلائل لملفات الإدخال والإخراج:

```python
import os

# تعيين مسارات لتخزين المستندات
data_directory = 'YOUR_DOCUMENT_DIRECTORY/'
output_directory = 'YOUR_OUTPUT_DIRECTORY/'

# إنشاء الدلائل إذا لم تكن موجودة
os.makedirs(data_directory, exist_ok=True)
os.makedirs(output_directory, exist_ok=True)
```

### 2. إنشاء ملف فيديو

لأغراض العرض التوضيحي، قم بإنشاء ملف فيديو مؤقت:

```python
video_file_path = os.path.join(data_directory, "video.mp4")
with open(video_file_path, 'wb') as video_file:
    # محاكاة البيانات الثنائية للمثال
    video_file.write(b'\x00\x01\x02')
```

### 3. إضافة الفيديو إلى العرض التقديمي

الآن، دعنا نضيف هذا الفيديو ككتلة إلى ملف PowerPoint جديد:

```python
with slides.Presentation() as pres:
    with open(video_file_path, "rb") as file_stream:
        # أضف الفيديو باستخدام سلوك KEEP_LOCKED للأمان
        video = pres.videos.add_video(file_stream, slides.LoadingStreamBehavior.KEEP_LOCKED)
        
        # إدراج إطار فيديو في الشريحة الأولى
        pres.slides[0].shapes.add_video_frame(0, 0, 480, 270, video)

    # احفظ عرضك التقديمي باستخدام مقطع الفيديو المضاف
    output_file_path = os.path.join(output_directory, "props_add_blob_to_presentation_out.pptx")
    pres.save(output_file_path, slides.export.SaveFormat.PPTX)
```

**خيارات تكوين المفتاح:**
- **سلوك KEEP_LOCKED**:يضمن أنه بمجرد تضمين مقطع فيديو، لا يمكن تغييره بشكل غير مقصود.

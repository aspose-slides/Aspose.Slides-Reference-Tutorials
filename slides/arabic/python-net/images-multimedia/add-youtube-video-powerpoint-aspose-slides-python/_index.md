## دليل التنفيذ

### الميزة 1: إضافة إطار فيديو من YouTube

توضح هذه الميزة كيفية إضافة إطار فيديو يحتوي على مقطع فيديو على YouTube وصورته المصغرة إلى شريحة PowerPoint.

#### دليل خطوة بخطوة

##### الخطوة 1: إنشاء إطار فيديو
قم بإنشاء إطار فيديو على الشريحة الأولى في الموضع (10، 10) بأبعاد 427 × 240 بكسل:
```python
def add_video_from_youtube(pres, video_id):
    video_frame = pres.slides[0].shapes.add_video_frame(10, 10, 427, 240, "https://www.youtube.com/embed/" + video_id)
```
*تعرف المعلمات على موضع وحجم إطار الفيديو داخل الشريحة.*

##### الخطوة 2: ضبط وضع تشغيل الفيديو
قم بتكوين وضع التشغيل ليبدأ تلقائيًا عند النقر فوقه:
```python
    video_frame.play_mode = slides.VideoPlayModePreset.AUTO
```

##### الخطوة 3: تحميل صورة مصغرة
جلب صورة مصغرة من YouTube وتعيينها لإطار الفيديو:
```python
    from urllib.request import urlopen
    
    thumbnail_uri = "http://img.youtube.com/vi/" + video_id + "/hqdefault.jpg"
    with urlopen(thumbnail_uri) as f:
        video_frame.picture_format.picture.image = pres.images.add_image(f.read())
```

## دليل التنفيذ

### قص الفيديو وتضمينه في شرائح PowerPoint

تتيح لنا هذه الميزة قص مقطع فيديو ودمجه في عرض تقديمي لبرنامج PowerPoint باستخدام Aspose.Slides لـ Python.

#### إضافة إطار فيديو إلى شريحة

أولاً، حدد مسارات الفيديو المصدر ودليل الإخراج. ثم أنشئ نسخة عرض تقديمي جديدة:

```python
import aspose.slides as slides
from pathlib import Path

video_file_name = Path("YOUR_DOCUMENT_DIRECTORY/") / "Wildlife.mp4"
output_file_path = Path("YOUR_OUTPUT_DIRECTORY/") / "VideoTrimming-out.pptx"

with slides.Presentation() as pres:
    slide = pres.slides[0]
```

#### قراءة بيانات الفيديو وإضافتها

بعد ذلك، اقرأ ملف الفيديو وأضفه إلى العرض التقديمي:

```python
    with open(video_file_name, "rb") as video_file:
        video_data = video_file.read()
        video = pres.videos.add_video(video_data)
        
    # إضافة إطار فيديو إلى الشريحة
    video_frame = slide.shapes.add_video_frame(0, 0, 200, 200, video)
```

#### قص الفيديو

قم بإعداد عملية التشذيب عن طريق تحديد أوقات البداية والنهاية بالمللي ثانية:

```python
    # قص من البداية (12 ثانية) إلى النهاية (16 ثانية)
    video_frame.trim_from_start = 12000
    video_frame.trim_from_end = 14000
    
    pres.save(str(output_file_path), slides.export.SaveFormat.PPTX)
```

### توضيح

- **حدود**: `trim_from_start` و `trim_from_end` تحديد الجزء المقطوع من الفيديو.
- **غاية**:تعمل ميزة التشذيب على تحسين طول العرض التقديمي دون إضافة محتوى غير ضروري.

#### نصائح استكشاف الأخطاء وإصلاحها

إذا واجهت مشاكل:
- تأكد من أن مسار ملف الفيديو الخاص بك صحيح.
- تأكد من تثبيت مكتبة Aspose.Slides بشكل صحيح.

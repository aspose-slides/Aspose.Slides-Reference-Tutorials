# كيفية إضافة إطار صوتي في PowerPoint باستخدام Aspose.Slides للغة Python

### ما سوف تتعلمه:
- إعداد Aspose.Slides في Python
- إضافة ملف صوتي إلى شريحة

## دليل التنفيذ

يرشدك هذا القسم خلال عملية إضافة إطار صوتي إلى عرض تقديمي في PowerPoint.

### إضافة الصوت إلى العرض التقديمي

**ملخص:**
أضف ملفًا صوتيًا إلى الشريحة الأولى من عرضك التقديمي. يتضمن ذلك تحميل الصوت، وتضمينه كإطار صوتي في الشريحة، ثم حفظ العرض التقديمي المُحدّث.

#### الخطوة 1: إعداد مسارات الملفات
قم بتحديد المسارات لملف الصوت المدخل وعرض الإخراج:
```python
input_audio_path = 'YOUR_DOCUMENT_DIRECTORY/audio.m4a'
output_presentation_path = 'YOUR_OUTPUT_DIRECTORY/AudioFrameValue_out.pptx'
```
يستبدل `YOUR_DOCUMENT_DIRECTORY` مع الدليل الذي يحتوي على ملف الصوت الخاص بك، و `YOUR_OUTPUT_DIRECTORY` حيث تريد حفظ العرض التقديمي.

#### الخطوة 2: إنشاء نسخة عرض تقديمي
استخدم مدير السياق لإدارة الموارد بشكل صحيح:
```python
with slides.Presentation() as pres:
    # سيتم تنفيذ الخطوات الأخرى ضمن هذه الكتلة.
```

#### الخطوة 3: تحميل وإضافة الصوت
افتح ملف الصوت الخاص بك في وضع القراءة الثنائي، ثم أضفه إلى مجموعة الملفات الصوتية الخاصة بالعرض التقديمي:
```python
with open(input_audio_path, "rb") as in_file:
    audio = pres.audios.add_audio(in_file)
```
ال `add_audio` تضيف الوظيفة ملف الصوت الخاص بك إلى المجموعة الداخلية لتضمينه في الشرائح.

#### الخطوة 4: تضمين إطار الصوت على الشريحة
قم بتضمين إطار الصوت على الشريحة الأولى في موضع محدد بأبعاد محددة:
```python
audio_frame = pres.slides[0].shapes.add_audio_frame_embedded(50, 50, 100, 100, audio)
```
المعلمات `(50, 50, 100, 100)` تحديد موضع x وموضع y والعرض والارتفاع لإطار الصوت.

### حفظ العرض التقديمي
يتم حفظ العرض التقديمي تلقائيًا عند الخروج `with` تأكد من تحديد مسار الإخراج بشكل صحيح لمنع الكتابة فوق الملف أو فقدانه.

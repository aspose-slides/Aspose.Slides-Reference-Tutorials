## دليل التنفيذ: الوصول إلى تعليقات الشريحة وعرضها

دعونا نستعرض عملية الوصول إلى تعليقات الشريحة وعرضها باستخدام `Aspose.Slides for Python`.

### نظرة عامة على الميزة

تتيح لك هذه الميزة استخراج التعليقات برمجيًا من كل شريحة في ملف PowerPoint. وهي مثالية للتطبيقات التي تحتاج إلى مراجعة أو تلخيص التعليقات مباشرةً داخل العروض التقديمية.

### الوصول إلى تعليقات الشريحة

إليك كيفية الوصول إلى تفاصيل تعليقات الشريحة وطباعتها:

#### الخطوة 1: استيراد Aspose.Slides

ابدأ باستيراد الوحدة اللازمة:

```python
import aspose.slides as slides
```

#### الخطوة 2: تحميل ملف العرض التقديمي الخاص بك

إعداد `with` بيان لضمان إدارة الموارد بشكل صحيح:

```python
class SlideCommentExtractor(PresentationContext):
    def extract_comments(self):
        with slides.Presentation(self.file_path) as presentation:
            self.process_comments(presentation)

    def process_comments(self, presentation):
        for author in presentation.comment_authors:
            for comment in author.comments:
                print(f"Slide {comment.slide.slide_number} has comment '{comment.text}' with author '{comment.author.name}' posted on time {comment.created_time}")
```

**توضيح:** 
- **`presentation.comment_authors`**:إرجاع مجموعة من جميع المؤلفين الذين تركوا تعليقات.
- **`author.comments`**:يوفر إمكانية الوصول إلى قائمة التعليقات التي قدمها كل مؤلف.
- **طباعة البيان**:تنسيق وطباعة رقم الشريحة ونص التعليق واسم المؤلف والطابع الزمني.

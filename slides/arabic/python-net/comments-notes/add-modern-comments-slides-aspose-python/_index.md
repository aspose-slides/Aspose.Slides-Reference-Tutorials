## دليل التنفيذ

### إضافة تعليقات حديثة إلى شرائح PowerPoint

#### ملخص

تتيح لك هذه الميزة إضافة تعليقات حديثة مباشرةً إلى شرائح العرض التقديمي. ترتبط هذه التعليقات بالمؤلفين، مما يتيح تبادل الآراء والتعليقات بشكل تعاوني.

#### التنفيذ خطوة بخطوة

**1. تهيئة العرض التقديمي**

ابدأ بإنشاء مثيل لـ `Presentation` فصل:

```python
with slides.Presentation() as pres:
    # سيتم إضافة الكود هنا
```

**2. إضافة المؤلف للتعليقات**

أضف مؤلفًا سيكون مسؤولاً عن التعليقات:

```python
new_author = pres.comment_authors.add_author("Some Author", "SA")
```
- **حدود:** اسم المؤلف ومعرف فريد.

**3. إضافة تعليق حديث**

بعد ذلك، أضف تعليقًا حديثًا إلى الشريحة المستهدفة:

```python
modern_comment = new_author.comments.add_modern_comment(
    "This is a modern comment",
    pres.slides[0],  # استهداف الشريحة الأولى
    None,            # لا يوجد شكل محدد للتعليق
    drawing.PointF(100, 100),  # موضع التعليق على الشريحة
    date.today()     # التاريخ الحالي كطابع زمني
)
```
- **حدود:**
  - `text`:محتوى التعليق.
  - `slide_index`:فهرس الشريحة المستهدفة.
  - `shape`:مرجع الشكل (اختياري، لا يوجد مرجع إذا لم يتم استخدامه).
  - `point`:الموضع على الشريحة حيث سيتم وضع التعليق.
  - `date_time`:العلامة الزمنية لوقت إضافة التعليق.

**4. حفظ العرض التقديمي**

وأخيرًا، احفظ العرض التقديمي الخاص بك للتأكد من تخزين كافة التغييرات:

```python
pres.save("YOUR_OUTPUT_DIRECTORY/comments_add_modern_comment_out.pptx", slides.export.SaveFormat.PPTX)
```
- **حدود:** 
  - مسار الملف مع الاسم.
  - تنسيق التصدير (PPTX في هذه الحالة).

#### نصائح استكشاف الأخطاء وإصلاحها

- تأكد من أن لديك أذونات الكتابة إلى الدليل الذي تريد حفظ الملف فيه.

## دليل التنفيذ

الآن بعد أن قمت بتحديد المتطلبات الأساسية، دعنا نتعمق في معالجة عروض PowerPoint.

### فتح واستنساخ الشرائح

#### ملخص

في هذا القسم، سنفتح ملف PowerPoint موجودًا ونستخلص شريحة تحتوي على عنصر تحكم ActiveX إلى نسخة عرض تقديمي جديدة.

#### خطوات

**الخطوة 1: فتح ملف PowerPoint موجود**

ابدأ بفتح ملف PowerPoint المستهدف باستخدام `Presentation` فصل:

```python
with slides.Presentation(YOUR_DOCUMENT_DIRECTORY + "activex_template.pptx") as pres:
    # يمكنك الوصول إلى العرض التقديمي الحالي الخاص بك هنا
```

**الخطوة 2: إزالة الشريحة الافتراضية**

قم بإنشاء عرض تقديمي جديد وإزالة الشريحة الافتراضية الخاصة به لتحضيره للاستنساخ:

```python
new_pres = slides.Presentation()
new_pres.slides.remove_at(0)
```

**الخطوة 3: استنساخ الشريحة باستخدام عنصر التحكم ActiveX**

استنساخ شريحة محددة من العرض التقديمي الأصلي إلى العرض الجديد:

```python
new_pres.slides.insert_clone(0, pres.slides[0])
```

### تعديل عناصر تحكم ActiveX

#### ملخص

يمكن أن تكون عناصر تحكم ActiveX أدوات فعّالة داخل الشرائح. هنا، سنُعدّل عنصر تحكم مشغل الوسائط الحالي.

#### خطوات

**الخطوة 4: الوصول إلى خصائص التحكم وتعديلها**

قم بالوصول إلى عنصر التحكم الأول في الشريحة المستنسخة لديك وقم بتغيير خصائصها:

```python
control = new_pres.slides[0].controls[0]
control.properties.remove("URL")
control.properties.add("URL", YOUR_DOCUMENT_DIRECTORY + "video.mp4")
```

### حفظ العرض التقديمي الخاص بك

#### ملخص

بمجرد الانتهاء من معالجة الشرائح الخاصة بك، حان الوقت لحفظ العرض التقديمي المعدل.

**الخطوة 5: حفظ العرض التقديمي**

```python
new_pres.save(YOUR_OUTPUT_DIRECTORY + "activex_linking_video_activex_control_out.pptx", slides.export.SaveFormat.PPTX)
```

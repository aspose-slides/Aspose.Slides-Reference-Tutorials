## دليل التنفيذ

### حفظ عرض تقديمي في ملف

تتيح لك هذه الميزة حفظ كائن عرض تقديمي من Aspose.Slides في ملف. وهي مفيدة بشكل خاص لإنشاء نسخ احتياطية أو تحضير العروض التقديمية للمشاركة.

#### ملخص
ستقوم بإنشاء عرض تقديمي فارغ وحفظه باستخدام `save` الطريقة، مع تحديد مسار الإخراج والتنسيق المطلوب.

#### خطوات التنفيذ
**1. استيراد المكتبات الضرورية**
ابدأ باستيراد الوحدات المطلوبة:
```python
import aspose.slides as slides
```

**2. قم بتعريف وظيفة الحفظ**
إنشاء وظيفة لتغليف عملية الحفظ:
```python
def save_to_file():
    with slides.Presentation() as presentation:
        output_path = 'YOUR_OUTPUT_DIRECTORY/save_to_file_out.pptx'
        presentation.save(output_path, slides.export.SaveFormat.PPTX)
```
- **`slides.Presentation()`**:يقوم بإنشاء كائن عرض تقديمي جديد.
- **`presentation.save()`**:يحفظ العرض التقديمي في المسار المحدد.

### إدراج الملفات في الدليل

توفر هذه الميزة نموذجًا أساسيًا لسرد الملفات داخل دليل. وهي مفيدة لإدارة وتنظيم مكتبات العروض التقديمية.

#### ملخص
إدراج جميع الملفات الموجودة في دليل معين، وتصفية الدلائل من قائمة المحتويات.

#### خطوات التنفيذ
**1. استيراد المكتبات الضرورية**
سوف تحتاج `os` للتفاعل مع نظام الملفات:
```python
import os
```

**2. قم بتعريف وظيفة قائمة الملفات**
إنشاء وظيفة لاسترجاع الملفات وتصفيتها:
```python
def list_files_in_directory():
    document_dir = 'YOUR_DOCUMENT_DIRECTORY/'
    try:
        file_list = os.listdir(document_dir)
        files_only = [f for f in file_list if os.path.isfile(os.path.join(document_dir, f))]
        return files_only
    except FileNotFoundError:
        print(f'Directory not found: {document_dir}')
        return []
```
- **`os.listdir()`**:استرجاع كافة الإدخالات في الدليل المحدد.
- **منطق التصفية**:يضمن تضمين الملفات فقط في القائمة.

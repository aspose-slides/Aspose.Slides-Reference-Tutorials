## دليل التنفيذ

سوف يرشدك هذا القسم خلال تحويل PPTX إلى TIFF باستخدام Aspose.Slides.

### نظرة عامة على عملية التحويل

سنحوّل ملف PowerPoint إلى صورة TIFF، مع تطبيق إعدادات تنسيق بكسل مخصصة، وإضافة ملاحظات الشرائح أسفلها. هذه العملية مثالية لإنشاء صور بجودة أرشيفية أو دمج العروض التقديمية في سير عمل المستندات.

#### الخطوة 1: استيراد المكتبات

ابدأ باستيراد الوحدات الضرورية:

```python
import aspose.slides as slides
```

#### الخطوة 2: تهيئة كائن العرض التقديمي

قم بتحميل ملف العرض التقديمي الخاص بك باستخدام مدير السياق للتعامل مع إدارة الموارد بكفاءة:

```python\with slides.Presentation('YOUR_DOCUMENT_DIRECTORY/welcome-to-powerpoint.pptx') as presentation:
    # Further processing goes here
```

#### الخطوة 3: تكوين TiffOptions

إنشاء مثيل لـ `TiffOptions` لتحديد إعدادات التصدير، بما في ذلك تنسيق البكسل وخيارات التخطيط للملاحظات:

```python
tiff_options = slides.export.TiffOptions()
# اضبط تنسيق البكسل على FORMAT_8BPP_INDEXED (8 بت لكل بكسل، مفهرس)
tiff_options.pixel_format = slides.export.ImagePixelFormat.FORMAT_8BPP_INDEXED

# تكوين كيفية ظهور الملاحظات في إخراج TIFF
slides_layout_options = slides.export.NotesCommentsLayoutingOptions()
slides_layout_options.notes_position = slides.export.NotesPositions.BOTTOM_FULL
tiff_options.slides_layout_options = slides_layout_options
```

#### الخطوة 4: الحفظ بصيغة TIFF

وأخيرًا، احفظ العرض التقديمي في ملف TIFF باستخدام الخيارات المحددة:

```python
output_file = 'YOUR_OUTPUT_DIRECTORY/convert_to_tiff_image_pixel_format_out.tiff'
presentation.save(output_file, slides.export.SaveFormat.TIFF, tiff_options)
```

- **مشاكل مسار الملف**:تأكد من تحديد مسارات ملفات الإدخال والإخراج بشكل صحيح.
- **توافق تنسيق البكسل**:تحقق مما إذا كان عارض TIFF المستهدف يدعم الألوان المفهرسة بدقة 8BPP للحصول على عرض مثالي.

## دليل التنفيذ

دعونا نوضح كيفية ضبط حجم صفحة PDF باستخدام Aspose.Slides لـ Python.

### الخطوة 1: إنشاء كائن العرض التقديمي وتكوينه

ابدأ بإنشاء حساب جديد `Presentation` الكائن الذي يسمح لك بالتعامل مع ملف العرض التقديمي الخاص بك:

```python
with slides.Presentation() as presentation:
    # اضبط حجم الشريحة على A4 وتأكد من أن المحتوى يتناسب مع حدود الصفحة
    presentation.slide_size.set_size(
        slides.SlideSizeType.A4_PAPER,
        slides.SlideSizeScaleType.ENSURE_FIT
    )
```

**توضيح:**
- `slides.SlideSizeType.A4_PAPER` تعيين حجم الشريحة إلى A4.
- `slides.SlideSizeScaleType.ENSURE_FIT` يقوم بتعديل المحتوى للتأكد من ملاءمته للصفحة.

### الخطوة 2: تكوين خيارات تصدير PDF

إعداد خيارات التصدير للحصول على مخرجات PDF عالية الجودة:

```python
pdf_options = slides.export.PdfOptions()
pdf_options.sufficient_resolution = 600  # تعيين دقة عالية لتحسين وضوح الصورة
```

**توضيح:**
- `sufficient_resolution` يضمن أن ملف PDF المُصدَّر يحتوي على صور ونصوص واضحة.

### الخطوة 3: حفظ العرض التقديمي بتنسيق PDF

وأخيرًا، احفظ العرض التقديمي الخاص بك في دليل الإخراج المحدد:

```python
output_path = "layout_set_pdf_page_size_out.pdf"
presentation.save(output_path, slides.export.SaveFormat.PDF, pdf_options)
```

**توضيح:**
- ال `save` تكتب الطريقة الملف بتنسيق PDF باستخدام الخيارات المحددة.

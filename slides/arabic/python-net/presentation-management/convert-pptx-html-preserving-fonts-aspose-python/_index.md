## دليل التنفيذ

في هذا القسم، سنقوم بتقسيم عملية التحويل إلى خطوات قابلة للإدارة.

### تحويل PPTX إلى HTML مع الحفاظ على الخطوط الأصلية

#### ملخص:
الميزة الأساسية لهذا التطبيق هي تحويل عرض PowerPoint التقديمي مع الحفاظ على خطوطه الأصلية واستبعاد بعض الخطوط الافتراضية من التضمين. يُعد هذا مفيدًا بشكل خاص للحفاظ على اتساق العلامة التجارية عبر عروض الويب التقديمية.

#### التنفيذ خطوة بخطوة:

**1. تحديد مسارات الإدخال والإخراج**

قم بإعداد الدلائل التي يوجد بها ملف PPTX المدخل والمكان الذي تريد حفظ ملف HTML الناتج فيه.

```python
data_dir = "YOUR_DOCUMENT_DIRECTORY/"
out_dir = "YOUR_OUTPUT_DIRECTORY/"
```

**2. افتح ملف العرض التقديمي**

استخدم Aspose.Slides `Presentation` الفئة لتحميل ملف PPTX الخاص بك:

```python
with slides.Presentation(data_dir + "welcome-to-powerpoint.pptx") as pres:
    # سيتم وضع رمز التحويل الخاص بك هنا.
```

يضمن مدير السياق هذا تحرير الموارد بشكل صحيح بعد العملية.

**3. إنشاء وحدة تحكم تضمين الخط المخصصة**

استبعاد خطوط معينة من التضمين باستخدام `EmbedAllFontsHtmlController`:

```python
font_name_exclude_list = ["Calibri", "Arial"]
embed_fonts_controller = slides.export.EmbedAllFontsHtmlController(font_name_exclude_list)
```

هنا، يتم استبعاد "Calibri" و"Arial" من التضمين في مخرجات HTML.

**4. تكوين خيارات تصدير HTML**

يثبت `HtmlOptions` لاستخدام منسق الخط المخصص مع وحدة التحكم الخاصة بك:

```python
html_options_embed = slides.export.HtmlOptions()
html_options_embed.html_formatter = slides.export.HtmlFormatter.create_custom_formatter(embed_fonts_controller)
```

تضمن هذه الخطوة تضمين الخطوط الضرورية فقط في الناتج النهائي.

**5. احفظ العرض التقديمي بصيغة HTML**

وأخيرًا، احفظ العرض التقديمي في ملف HTML باستخدام الخيارات المحددة:

```python
pres.save(out_dir + "convert_to_html_with_preserving_original_fonts_out.html", 
          slides.export.SaveFormat.HTML, html_options_embed)
```

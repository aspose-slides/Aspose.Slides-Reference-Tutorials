## دليل التنفيذ

دعونا نقسم عملية التحويل إلى أجزاء قابلة للإدارة.

### تحميل العرض التقديمي الخاص بك
#### ملخص
يعد تحميل العرض التقديمي الخاص بك هو الخطوة الأولى لتحويله إلى ملف GIF. 

##### الخطوة 1: افتح ملف PPTX
```python
# تحميل العرض التقديمي من دليل محدد
with slides.Presentation("YOUR_DOCUMENT_DIRECTORY/welcome-to-powerpoint.pptx") as presentation:
    # تضمن عبارة "with" إدارة الموارد بشكل صحيح
```

### تكوين إخراج GIF الخاص بك
#### ملخص
قم بتخصيص كيفية تحويل ملف PowerPoint الخاص بك إلى صورة GIF متحركة.

##### الخطوة 2: إعداد GifOptions
```python
# تكوين خيارات لإخراج GIF
gif_options = slides.export.GifOptions()

# تخصيص حجم إطار صورة GIF الناتجة
gif_options.frame_size = drawing.Size(540, 480)

# حدد المدة التي سيتم عرض كل شريحة فيها (بالمللي ثانية)
gif_options.default_delay = 1500

# تعيين إطارات في الثانية لرسوم متحركة انتقالية لتحسين الجودة
gif_options.transition_fps = 60
```

### حفظ العرض التقديمي بصيغة GIF
#### ملخص
قم بتحويل العرض التقديمي المخصص الخاص بك وحفظه.

##### الخطوة 3: الحفظ كملف GIF
```python
# احفظ العرض التقديمي بتنسيق GIF في الدليل المطلوب
presentation.save("YOUR_OUTPUT_DIRECTORY/convert_to_gif_out.gif", slides.export.SaveFormat.GIF, gif_options)
```

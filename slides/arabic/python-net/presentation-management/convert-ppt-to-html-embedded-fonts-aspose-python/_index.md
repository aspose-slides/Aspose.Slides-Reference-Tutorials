## دليل التنفيذ

### تحويل PowerPoint إلى HTML باستخدام الخطوط المضمنة

سوف يرشدك هذا القسم خلال عملية تضمين الخطوط عند تصدير عرض تقديمي في PowerPoint كملف HTML.

#### ملخص

الهدف هو تحويل `.pptx` الملفات في `.html`، مع ضمان تضمين جميع الخطوط المستخدمة في المستند الأصلي في المخرجات. هذا يضمن الاتساق عبر مختلف البيئات والأجهزة.

#### التنفيذ خطوة بخطوة

##### فتح ملف العرض التقديمي

ابدأ بفتح عرض PowerPoint الذي ترغب في تحويله:

```python
document_path = "YOUR_DOCUMENT_DIRECTORY/welcome-to-powerpoint.pptx"
with slides.Presentation(document_path) as pres:
    # سيتم إجراء المزيد من المعالجة هنا
```

يقوم مقتطف التعليمات البرمجية هذا بتحميل ملف PowerPoint الخاص بك في الذاكرة، جاهزًا للتحويل.

##### إعداد تضمين الخط

لتضمين كافة الخطوط المستخدمة في العرض التقديمي:

```python
# إنشاء قائمة بالخطوط التي تريد استبعادها (اتركها فارغة إذا كنت تريد تضمين جميع الخطوط)
font_name_exclude_list = []

# تهيئة كائن EmbedAllFontsHtmlController باستخدام قائمة الاستبعاد
embed_fonts_controller = slides.export.EmbedAllFontsHtmlController(font_name_exclude_list)
```

يضمن هذا الإعداد تضمين كل الخطوط المستخدمة في العرض التقديمي في إخراج HTML.

##### تكوين خيارات تصدير HTML

بعد ذلك، قم بتكوين خيارات التصدير لاستخدام مُنسيق مخصص:

```python
html_options_embed = slides.export.HtmlOptions()
html_options_embed.html_formatter = slides.export.HtmlFormatter.create_custom_formatter(embed_fonts_controller)
```

هنا، نقوم بتخصيص كيفية تحويل ملف PowerPoint إلى HTML عن طريق تضمين الخطوط.

##### حفظ بتنسيق HTML مع الخطوط المضمنة

وأخيرًا، احفظ العرض التقديمي بتنسيق HTML مع تضمين جميع الخطوط:

```python
output_path = "YOUR_OUTPUT_DIRECTORY/convert_to_html_with_embed_all_fonts_out.html"
pres.save(output_path, slides.export.SaveFormat.HTML, html_options_embed)
```

تؤدي هذه الخطوة إلى إخراج الملف المُحوّل إلى الدليل المحدد.

### نصائح استكشاف الأخطاء وإصلاحها

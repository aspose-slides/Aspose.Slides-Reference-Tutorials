# تحويل عروض PowerPoint إلى PDF بما في ذلك الشرائح المخفية باستخدام Aspose.Slides لـ Python

## مقدمة

هل تفقد معلومات مهمة عند تحويل عروض PowerPoint التقديمية إلى ملفات PDF؟ سيوضح لك هذا الدليل كيفية تحويل ملفات PPTX إلى صيغة PDF مع الحفاظ على جميع الشرائح، بما في ذلك الشرائح المخفية. سنستخدم مكتبة Aspose.Slides القوية في Python لضمان عدم إغفال أي تفاصيل.

في هذا البرنامج التعليمي، سوف تتعلم:
- كيفية إعداد Aspose.Slides واستخدامه لـ Python
- الخطوات اللازمة لتحويل العروض التقديمية ذات الشرائح المخفية إلى ملفات PDF
- التطبيقات العملية لهذه الميزة

### المتطلبات الأساسية
لمتابعة هذا البرنامج التعليمي، تأكد من أن لديك ما يلي:
- **تم تثبيت بايثون**:الإصدار 3.6 أو أعلى.
- **Aspose.Slides لـ Python**:تعتبر هذه المكتبة ضرورية للتعامل مع ملفات PowerPoint في مشاريع Python الخاصة بك.
- **إعداد البيئة**:محرر نصوص أو بيئة تطوير متكاملة يمكنك من خلالها كتابة وتنفيذ كود Python (على سبيل المثال، Visual Studio Code، PyCharm).
- **المعرفة الأساسية بلغة بايثون**:ستكون المعرفة بقواعد اللغة Python وعمليات الملفات مفيدة.

```bash

```

import aspose.slides as slides

presentation = slides.Presentation("path_to_your_pptx_file")

## دليل التنفيذ: تحويل PPTX إلى PDF باستخدام الشرائح المخفية

### نظرة عامة على الميزة
تتيح لك هذه الميزة تحويل عرض تقديمي من PowerPoint إلى ملف PDF، مما يضمن تضمين جميع الشرائح المخفية في الناتج. تُعد هذه الميزة مفيدة بشكل خاص عند الحاجة إلى حفظ كل جزء من المحتوى لأغراض الأرشفة أو المشاركة.

#### الخطوة 1: تحميل العرض التقديمي
ابدأ بتحميل ملف PPTX الخاص بك باستخدام `Presentation` فصل.

```python
with slides.Presentation("YOUR_DOCUMENT_DIRECTORY/presentation_with_hidden_slides.pptx") as presentation:
    # سوف تتم المعالجة الإضافية هنا
```

#### الخطوة 2: تكوين خيارات PDF
إنشاء مثيل `PdfOptions` لتحديد خيارات تحويل PDF. هنا، يمكنك تحديد خيار تضمين الشرائح المخفية.

```python
class PdfOptions:
    def __init__(self):
        self.إظهار الشرائح المخفية = False

pdf_options = PdfOptions()
pdf_options.show_hidden_slides = True
```

- **show_hidden_slides**:تعتبر هذه المعلمة بالغة الأهمية لأنها تحدد ما إذا كانت الشرائح المخفية سيتم تضمينها في ملف PDF الناتج.

#### الخطوة 3: حفظ العرض التقديمي
وأخيرًا، احفظ العرض التقديمي الخاص بك كملف PDF مع الخيارات المحددة.

```python
target_directory = "YOUR_OUTPUT_DIRECTORY"
presentation.save(f"{target_directory}/convert_to_pdf_hidden_slides_out.pdf", \
                 slides.export.SaveFormat.PDF, pdf_options)
```

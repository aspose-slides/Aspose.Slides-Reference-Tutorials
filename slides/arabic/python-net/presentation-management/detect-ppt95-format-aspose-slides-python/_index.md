## دليل التنفيذ

### اكتشاف تنسيق PPT95

لتحديد ما إذا كان العرض التقديمي بتنسيق PPT القديم (PPT95)، اتبع الخطوات التالية:

#### التنفيذ خطوة بخطوة

**1. الحصول على معلومات العرض التقديمي**

قم بتحميل معلومات العرض التقديمي باستخدام Aspose.Slides:

```python
import aspose.slides as slides

def check_presentation_format():
    # استبدل 'YOUR_DOCUMENT_DIRECTORY/' بمسار الدليل الخاص بك.
    load_info = slides.PresentationFactory.instance.get_presentation_info(
        "YOUR_DOCUMENT_DIRECTORY/open_presentation.ppt")
```

*توضيح:* نحن نستخدم `PresentationFactory` لجلب تفاصيل العرض التقديمي. الطريقة `get_presentation_info` يقوم بقراءة بيانات الملف التعريفية، بما في ذلك تنسيقه.

**2. تحديد التنسيق**

التحقق من أن التنسيق المحمل هو PPT95:

```python
    # تحقق مما إذا كان تنسيق العرض التقديمي هو PPT95.
is_old_format = load_info.load_format == slides.LoadFormat.PPT95

return is_old_format
```

*توضيح:* من خلال المقارنة `load_info.load_format` مع `slides.LoadFormat.PPT95`، نحدد ما إذا كان الملف بتنسيق PPT القديم.

## دليل التنفيذ (H2)

يرشدك هذا القسم خلال عملية تصدير النص من PowerPoint إلى HTML.

### نظرة عامة على الميزة

الهدف هو استخراج النص من شريحة معينة في عرض تقديمي في PowerPoint وحفظه كملف HTML باستخدام Aspose.Slides لـ Python.

### تعليمات خطوة بخطوة

#### 1. تحميل العرض التقديمي (H3)

قم بتحميل ملف PowerPoint الخاص بك:

```python
import aspose.slides as slides

def exporting_html_text():
    # تحميل العرض التقديمي
    with slides.Presentation("YOUR_DOCUMENT_DIRECTORY/text_export_text_frame_to_html.pptx") as pres:
        pass  # مزيد من المعالجة هنا
```

#### 2. الوصول إلى الشريحة المطلوبة (H3)

قم بالوصول إلى الشريحة التي تريد تصدير النص منها:

```python
        # الوصول إلى الشريحة الأولى
        slide = pres.slides[0]
```

#### 3. تحديد الشكل الذي يحتوي على نص والوصول إليه (H3)

حدد الشكل الذي يحتوي على النص الموجود في الشريحة المستهدفة:

```python
        # فهرس للوصول إلى شكل معين في الشريحة
        index = 0

        # الوصول إلى الشكل عند الفهرس المحدد
        auto_shape = slide.shapes[index]
```

#### 4. تصدير النص إلى HTML (H3)

تصدير النص من الشكل المحدد وحفظه كملف HTML:

```python
        # فتح ملف HTML في وضع الكتابة
        with open("YOUR_OUTPUT_DIRECTORY/text_export_text_frame_to_html_out.html", "wt") as sw:
            # تصدير إطار النص من الفقرات إلى تنسيق HTML
            data = auto_shape.text_frame.paragraphs.export_to_html(0, auto_shape.text_frame.paragraphs.count, None)
            
            # اكتب محتوى HTML المُصدَّر في الملف
            sw.write(data)
```

### توضيح

- **تحميل العرض التقديمي:** ال `Presentation` يقوم class بتحميل ملف PPTX الخاص بك.
- **الوصول إلى الأشكال وإطارات النص:** يمكنك الوصول إلى أشكال محددة باستخدام فهرسها لتحديد إطارات النص للتصدير.
- **وظيفة التصدير:** `export_to_html()` يستخرج النص بتنسيق HTML، والذي يتم كتابته بعد ذلك في ملف إخراج.

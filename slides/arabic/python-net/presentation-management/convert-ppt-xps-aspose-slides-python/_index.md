## دليل التنفيذ

في هذا القسم، سنشرح كيفية تحويل ملف PowerPoint إلى تنسيق XPS باستخدام Aspose.Slides لـ Python.

### نظرة عامة: تحويل العرض التقديمي إلى XPS

الوظيفة الرئيسية لهذا البرنامج التعليمي هي إظهار كيفية تحويل ملفات PPT إلى تنسيق XPS الأكثر قابلية للنقل وتنوعًا.

#### الخطوة 1: تحديد الدلائل
ابدأ بتحديد أدلة الإدخال والإخراج التي يوجد بها ملف PowerPoint والمكان الذي تريد حفظ ملف XPS المحول فيه:

```python
input_directory = "YOUR_DOCUMENT_DIRECTORY/"
output_directory = "YOUR_OUTPUT_DIRECTORY/"
```

سيتم استخدام هذه المسارات لاحقًا في وظيفة التحويل الخاصة بنا.

#### الخطوة 2: تحميل العرض التقديمي
إنشاء `Presentation` الكائن الذي يمثل ملف PowerPoint. حدد المسار إلى ملفك `.pptx` ملف:

```python
demo_presentation_path = input_directory + "welcome-to-powerpoint.pptx"
```

باستخدام مدير السياق (`with slides.Presentation(demo_presentation_path) as pres:`), نحن نضمن أن الموارد يتم إدارتها بشكل صحيح.

#### الخطوة 3: الحفظ بتنسيق XPS
مع تحميل العرض التقديمي، حدد المكان الذي تريد حفظ الإخراج فيه واستخدم `save` طريقة التحويل:

```python
dxps_output_path = output_directory + "converted_to_xps_out.xps"
pres.save(dxps_output_path, slides.export.SaveFormat.XPS)
```

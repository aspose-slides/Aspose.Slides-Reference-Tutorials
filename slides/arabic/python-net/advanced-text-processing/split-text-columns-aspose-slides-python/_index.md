## دليل التنفيذ

### ميزة تقسيم النص حسب الأعمدة

تتيح لك هذه الميزة تقسيم محتوى إطار النص إلى عدة أعمدة ضمن العرض التقديمي. إليك كيفية عملها:

#### التنفيذ خطوة بخطوة
**1. تحميل العرض التقديمي**
ابدأ بتحميل ملف PowerPoint الذي يحتوي على إطارات النص.

```python
import aspose.slides as slides

def split_text_by_columns():
    input_path = "YOUR_DOCUMENT_DIRECTORY/MultiColumnText.pptx"
    output_path = "YOUR_OUTPUT_DIRECTORY/output.txt"  # اختياري: حدد لحفظ الناتج
    
    with slides.Presentation(input_path) as pres:
        slide = pres.slides[0]
```

**2. الوصول إلى إطار النص**
قم بتحديد إطار النص الأول الموجود على الشريحة الخاصة بك والوصول إليه.

```python
shape = slide.shapes[0]  # على افتراض أنه شكل يحتوي على نص
text_frame = shape.text_frame
```

**3. تقسيم المحتوى إلى أعمدة**
استخدم `split_text_by_columns` طريقة تقسيم المحتوى.

```python
columns_text = text_frame.split_text_by_columns()
```

**4. إخراج النتيجة أو استخدامها**
قم بالتكرار على نص كل عمود للتحقق من الناتج:

```python
for column in columns_text:
    print(column)
```

### توضيح
- **المعلمات وقيم الإرجاع:** ال `split_text_by_columns` لا تتطلب الطريقة معلمات وتعيد قائمة من السلاسل، حيث يمثل كل منها محتوى عمود.
- **نصيحة لاستكشاف الأخطاء وإصلاحها:** تأكد من أن إطار النص يحتوي على أسطر متعددة لإظهار تقسيم العمود بشكل فعال.

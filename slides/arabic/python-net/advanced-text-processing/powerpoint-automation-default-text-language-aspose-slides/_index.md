## دليل التنفيذ: تعيين لغة النص الافتراضية

### ملخص

تتيح لك هذه الميزة تعيين لغة نص افتراضية لجميع عناصر النص ضمن عرض تقديمي في PowerPoint، مما يؤدي إلى تبسيط سير العمل من خلال التخلص من المهام المتكررة.

### التنفيذ خطوة بخطوة

#### إنشاء LoadOptions لتحديد لغة النص الافتراضية

1. **تهيئة LoadOptions**
   ابدأ بإنشاء مثيل لـ `LoadOptions` لتحديد لغة النص الافتراضية المطلوبة:

   ```python
   load_options = slides.LoadOptions()
   ```

2. **تعيين اللغة الافتراضية**
   قم بتعيين لغة النص الافتراضية باستخدام علامة لغة BCP-47 (على سبيل المثال، "en-US" للغة الإنجليزية، الولايات المتحدة):

   ```python
   load_options.default_text_language = "en-US"
   ```

#### فتح العرض التقديمي وتعديله
3. **تحميل العرض التقديمي باستخدام LoadOptions**
   يستخدم `LoadOptions` عند فتح العرض التقديمي لتطبيق لغة النص الافتراضية:

   ```python
   with slides.Presentation(load_options) as pres:
       # أضف شكل مستطيل جديد مع النص على الشريحة الأولى
       shp = pres.slides[0].shapes.add_auto_shape(
           slides.ShapeType.RECTANGLE, 50, 50, 150, 50)
       shp.text_frame.text = "New Text"
   ```

4. **الوصول والتحقق من معرف اللغة**
   يمكنك التحقق من معرف اللغة لأجزاء النص للتأكد من ضبطها بشكل صحيح:

   ```python
   # الوصول إلى معرف اللغة للتحقق (خطوة توضيحية اختيارية)
   language_id = shp.text_frame.paragraphs[0].portions[0].portion_format.language_id
   ```

### نصائح استكشاف الأخطاء وإصلاحها
- **مشكلة شائعة**:النص الافتراضي لا يعكس التغييرات.
  - **حل**: يضمن `LoadOptions` يتم تطبيقه بشكل صحيح عند فتح العرض التقديمي.

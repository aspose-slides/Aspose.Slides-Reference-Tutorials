## دليل التنفيذ

يقوم هذا القسم بتحليل عملية استخراج القيم الفعالة من إطارات وأجزاء النص.

### فهم القيم الفعالة

تُحدد القيم الفعّالة في العروض التقديمية كيفية تطبيق الأنماط عند وجود تسلسل هرمي أو وراثة للتنسيق. يتيح لك استخراج هذه القيم فهم الخصائص التي تؤثر فعليًا على محتوى الشريحة.

#### الخطوة 1: تحميل العرض التقديمي

```python
def get_effective_values():
    data_dir = 'YOUR_DOCUMENT_DIRECTORY/'
    file_name = 'text_add_animation_effect.pptx'
    
    with slides.Presentation(data_dir + file_name) as pres:
        # الوصول إلى الشكل الأول في الشريحة الأولى
        shape = pres.slides[0].shapes[0]
```
- **لماذا هذه الخطوة:** نقوم بتحميل العرض التقديمي للوصول إلى بنيته، مع التركيز على إطارات النص داخل الأشكال.

#### الخطوة 2: استخراج قيم تنسيق إطار النص

```python
        local_text_frame_format = shape.text_frame.text_frame_format
        effective_text_frame_format = local_text_frame_format.get_effective()
```
- **توضيح:** `local_text_frame_format` يحتفظ بإعدادات التنسيق المطبقة مباشرةً على إطار النص. الطريقة `get_effective()` يسترجع القيم النهائية بعد مراعاة جميع الخصائص الموروثة.

#### الخطوة 3: استخراج قيم تنسيق الجزء

```python
        local_portion_format = shape.text_frame.paragraphs[0].portions[0].portion_format
        effective_portion_format = local_portion_format.get_effective()
```
- **لماذا هذه الخطوة:** يتيح لك الوصول إلى تنسيق الجزء رؤية كيفية تصميم أجزاء النص، مع مراعاة الخصائص المباشرة والموروثة.

#### الخطوة 4: عرض القيم الفعالة

```python
        print('Effective Text Frame Format:', effective_text_frame_format)
        print('Effective Portion Format:', effective_portion_format)
```
- **غاية:** تتيح لنا طباعة هذه القيم التحقق من التطبيق الصحيح للأنماط في محتوى العرض التقديمي الخاص بنا.

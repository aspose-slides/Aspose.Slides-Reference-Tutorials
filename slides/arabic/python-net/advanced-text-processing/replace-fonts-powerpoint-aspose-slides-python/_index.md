## دليل التنفيذ
في هذا القسم، سنقوم بتفصيل الخطوات المطلوبة لاستبدال الخطوط في عرض تقديمي في PowerPoint باستخدام Aspose.Slides لـ Python. 
### استبدال الخطوط صراحةً
#### ملخص
سنوضح لك كيفية تحميل عرض تقديمي واستبدال خط محدد بآخر عبر الشرائح.
#### التنفيذ خطوة بخطوة
**1. تعريف الدلائل:**
أولاً، قم بتحديد مكان وجود مستند المصدر والمكان الذي تريد حفظ الملف المحدث فيه:
```python
YOUR_DOCUMENT_DIRECTORY = 'path/to/your/document/directory/'
YOUR_OUTPUT_DIRECTORY = 'path/to/your/output/directory/'
```
استبدل هذه العناصر النائبة بالمسارات الفعلية على نظامك.
**2. تحميل العرض التقديمي:**
بعد ذلك، قم بتحميل العرض التقديمي باستخدام مدير السياق لإدارة الموارد بكفاءة:
```python
with slides.Presentation(YOUR_DOCUMENT_DIRECTORY + "text_fonts.pptx") as presentation:
    # انتقل إلى خطوات استبدال الخط
```
هنا، `"text_fonts.pptx"` هو الملف الذي تريد تعديله.
**3. تحديد الخطوط المصدر والوجهة:**
حدد الخط الذي تريد استبداله (المصدر) والخط (الوجهة):
```python
source_font = slides.FontData("Arial")
dest_font = slides.FontData("Times New Roman")
```
في هذا المثال، نقوم باستبدال "Arial" بـ "Times New Roman".
**4. استبدال الخطوط:**
استخدم `fonts_manager` لاستبدال جميع مثيلات الخط المصدر:
```python
presentation.fonts_manager.replace_font(source_font, dest_font)
```
تعمل هذه الطريقة على البحث في العرض التقديمي الخاص بك واستبدال الخطوط المحددة.
**5. حفظ العرض التقديمي المحدث:**
وأخيرًا، احفظ العرض التقديمي المعدّل كملف جديد:
```python
presentation.save(YOUR_OUTPUT_DIRECTORY + "text_updated_font_out.pptx")
```

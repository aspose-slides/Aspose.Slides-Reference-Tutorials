## دليل التنفيذ (H2)

### استنساخ شريحة داخل نفس العرض التقديمي (نظرة عامة على الميزة)
تتيح لك هذه الميزة تكرار شريحة وإضافتها إلى نهاية نفس العرض التقديمي، مما يوفر الوقت عند إنشاء محتوى متكرر.

#### خطوات استنساخ الشريحة:

**3.1 تحميل العرض التقديمي الحالي**
أولاً، قم بتحميل ملف العرض التقديمي الخاص بك باستخدام مكتبة Aspose.Slides.

```python
with slides.Presentation('YOUR_DOCUMENT_DIRECTORY/welcome-to-powerpoint.pptx') as pres:
    all_slides = pres.slides  # الوصول إلى مجموعة الشرائح
```

**3.2 استنساخ الشريحة وإضافتها**
استنسخ شريحة محددة (في هذه الحالة، الشريحة الأولى) وأضفها إلى نهاية العرض التقديمي.

```python
# استنساخ الشريحة الأولى
cloned_slide = all_slides.add_clone(pres.slides[0])
```

**3.3 حفظ العرض التقديمي المعدّل**
وأخيرًا، احفظ التغييرات في ملف جديد في دليل الإخراج المطلوب.

```python
pres.save('YOUR_OUTPUT_DIRECTORY/crud_add_clone3_out.pptx', slides.export.SaveFormat.PPTX)
```

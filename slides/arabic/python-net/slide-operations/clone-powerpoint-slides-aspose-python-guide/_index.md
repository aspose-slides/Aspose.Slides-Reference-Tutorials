### استنساخ شريحة بين العروض التقديمية

#### ملخص

تتيح لك هذه الميزة نسخ شريحة من ملف PowerPoint وإدراجها في ملف آخر في موضع محدد. يُعد هذا مفيدًا لإعادة استخدام المحتوى في عروض تقديمية متعددة.

#### تعليمات خطوة بخطوة

1. **تحميل العرض التقديمي المصدر**
   
   ابدأ بفتح العرض التقديمي المصدر الذي يحتوي على الشريحة التي تريد استنساخها:
   
   ```python
   import aspose.slides as slides

   def load_source_presentation(file_path):
       with slides.Presentation(file_path) as source_presentation:
           return source_presentation
   ```

2. **فتح عرض تقديمي لوجهة جديدة**
   
   قم بإنشاء العرض التقديمي أو افتحه حيث تريد إدراج الشريحة المستنسخة:
   
   ```python
   def load_destination_presentation():
       with slides.Presentation() as destination_presentation:
           return destination_presentation
   ```

3. **أدخل الشريحة المستنسخة**
   
   استخدم `insert_clone` طريقة لتكرار شريحة معينة من العرض التقديمي المصدر إلى الموضع المطلوب في الوجهة:
   

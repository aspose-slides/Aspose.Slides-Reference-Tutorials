## دليل التنفيذ

### استنساخ الشرائح باستخدام الشريحة الرئيسية

#### ملخص
في هذا القسم، سنوضح كيفية استنساخ شريحة والشريحة الرئيسية المرتبطة بها من عرض تقديمي إلى آخر باستخدام Aspose.Slides.

##### الخطوة 1: تحميل العرض التقديمي المصدر
أولاً، قم بتحميل ملف PowerPoint المصدر الخاص بك:

```python
with slides.Presentation("YOUR_DOCUMENT_DIRECTORY/welcome-to-powerpoint.pptx") as source_pres:
    # الوصول إلى الشريحة الأولى والشريحة الرئيسية الخاصة بها
    source_slide = source_pres.slides[0]
    source_master = source_slide.layout_slide.master_slide
```
**توضيح**:نحن نحمل `welcome-to-powerpoint.pptx` للوصول إلى الشريحة الأولى والشريحة الرئيسية المرتبطة بها.

##### الخطوة 2: إنشاء عرض تقديمي جديد للوجهة
بعد ذلك، قم بإنشاء عرض تقديمي جديد حيث سيتم إضافة الشرائح المستنسخة:

```python
with slides.Presentation() as dest_pres:
    # الوصول إلى مجموعة الشرائح الرئيسية في العرض التقديمي الوجهة
    masters = dest_pres.masters
```
**توضيح**:يتم إنشاء عرض تقديمي فارغ لحمل المحتوى المستنسخ.

##### الخطوة 3: استنساخ الشريحة الرئيسية
الآن، قم باستنساخ الشريحة الرئيسية من المصدر إلى الوجهة:

```python
cloned_master = masters.add_clone(source_master)
```
**توضيح**: ال `add_clone` تقوم الطريقة بتكرار الشريحة الرئيسية في مجموعة الشرائح الرئيسية للعرض التقديمي الجديد.

##### الخطوة 4: استنساخ الشريحة مع تخطيطها
استنساخ الشريحة الأصلية باستخدام تخطيط الرئيسي المستنسخ:

```python
dest_slides = dest_pres.slides
dest_slides.add_clone(source_slide, cloned_master, True)
```
**توضيح**:تعمل هذه الخطوة على تكرار الشريحة أثناء ربطها بالشريحة الرئيسية المستنسخة حديثًا.

##### الخطوة 5: حفظ عرض الوجهة
أخيرًا، احفظ العرض التقديمي المعدّل في الموقع المطلوب:

```python
dest_pres.save("YOUR_OUTPUT_DIRECTORY/crud_clone_with_master_out.pptx")
```
**توضيح**:يتم حفظ ملف الإخراج في `crud_clone_with_master_out.pptx`، تعكس جميع التغييرات المستنسخة.

#### نصائح استكشاف الأخطاء وإصلاحها

### التهيئة الأساسية

بعد التثبيت، قم بتهيئة كائن العرض التقديمي الخاص بك:

```python
import aspose.slides as slides

current_presentation = slides.Presentation()
```

## دليل التنفيذ

يرشدك هذا القسم خلال عملية استنساخ الشرائح بين الأقسام في العرض التقديمي.

### نظرة عامة: استنساخ الشرائح بين الأقسام

هدفنا هو نسخ شريحة من قسم ووضعها في قسم آخر. قد يكون هذا مفيدًا لتكرار المحتوى الذي يحتاج إلى التكرار في أجزاء مختلفة من عرضك التقديمي.

#### الخطوة 1: إنشاء الشريحة الأولية بالشكل

أولاً، أضف شكل مستطيل إلى الشريحة الأولى كقالب:

```python
current_presentation.slides[0].shapes.add_auto_shape(slides.ShapeType.RECTANGLE, 200, 50, 300, 100)
```

#### الخطوة 2: إنشاء الأقسام وتعيينها

قم بإنشاء قسم جديد باسم "القسم 1" وقم بتعيين الشريحة الأولية له:

```python
current_presentation.sections.add_section("Section 1", current_presentation.slides[0])
```

بعد ذلك، قم بإضافة قسم فارغ يسمى "القسم 2":

```python
section2 = current_presentation.sections.append_empty_section("Section 2")
```

#### الخطوة 3: استنساخ الشريحة إلى قسم جديد

استخدم `add_clone` طريقة استنساخ الشريحة الأولى في القسم الثاني:

```python
current_presentation.slides.add_clone(current_presentation.slides[0], section2)
```

#### الخطوة 4: حفظ العرض التقديمي

وأخيرًا، احفظ العرض التقديمي في الدليل المطلوب:

```python
current_presentation.save("YOUR_OUTPUT_DIRECTORY/crud_append_empty_section_out.pptx", slides.export.SaveFormat.PPTX)
```

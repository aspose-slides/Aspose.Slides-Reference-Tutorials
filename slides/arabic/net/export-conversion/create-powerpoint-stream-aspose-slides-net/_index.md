## دليل التنفيذ

### الميزة 1: إنشاء عرض تقديمي لبرنامج PowerPoint وحفظه للبث

#### ملخص
ترتكز هذه الميزة على إنشاء عرض تقديمي بسيط في PowerPoint، وإدراج محتوى نصي، وحفظه مباشرة كتدفق لمزيد من المعالجة أو التخزين.

##### دليل خطوة بخطوة

**إنشاء عرض تقديمي جديد**
ابدأ بإنشاء مثيل لـ `Presentation` الفئة التي تمثل ملف PowerPoint الخاص بك:

```csharp
using Aspose.Slides;

namespace PresentationToStreamExample
{
    public class SavePresentationToStream
    {
        public static void Main()
        {
            string dataDir = @"YOUR_DOCUMENT_DIRECTORY"; // حدد مسار الدليل الخاص بك هنا

            using (Presentation presentation = new Presentation())
            {
                // استمر في التعامل مع الشريحة...
```

**إضافة شكل نص إلى الشريحة الأولى**
أضف شكلًا تلقائيًا من نوع المستطيل وأدرج نصًا فيه:

```csharp
                IAutoShape shape = presentation.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 200, 200, 200, 200);
                shape.TextFrame.Text = "This demo shows how to Create PowerPoint file and save it to Stream.";
```

**حفظ العرض التقديمي كتدفق**
قم بتحديد مجرى سيتم حفظ العرض التقديمي الخاص بك فيه:

```csharp
                using (FileStream toStream = new FileStream(dataDir + "Save_As_Stream_out.pptx", FileMode.Create))
                {
                    // احفظ العرض التقديمي في التدفق.
                    presentation.Save(toStream, Aspose.Slides.Export.SaveFormat.Pptx);
                }
            }
        }
    }
}
```

**توضيح:**
- `Presentation` يتعامل مع ملفات PowerPoint في الذاكرة.
- يتم إضافة شكل المستطيل إلى الشريحة الأولى بالأبعاد والإحداثيات المحددة.
- يتم استخدام FileStream لحفظ العرض التقديمي بتنسيق PPTX، مما يسمح بمعالجة البيانات بشكل مرن.

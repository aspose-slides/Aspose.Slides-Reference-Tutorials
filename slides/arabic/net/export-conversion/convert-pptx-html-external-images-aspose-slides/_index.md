# كيفية تحويل PPTX إلى HTML مع الصور الخارجية باستخدام Aspose.Slides لـ .NET

**الدروس المستفادة:**
- تكوين Aspose.Slides لـ .NET في مشروعك
- حفظ عرض تقديمي كمستند HTML مع صور خارجية باستخدام C#
- فهم تكوينات فئة Html5Options
- استكشاف التطبيقات العملية واعتبارات الأداء

## دليل التنفيذ

اتبع الخطوات التالية لحفظ عرض تقديمي PPTX كمستند HTML مع صور خارجية.

### تكوين خيارات HTML5 للصور الخارجية

**ملخص:**
عن طريق الإعداد `EmbedImages` إلى خطأ في `Html5Options`، فأنت تطلب من Aspose.Slides عدم تضمين الصور داخل ملف HTML، وبالتالي استخدام مسارات الصور الخارجية بدلاً من ذلك.

**خطوات التنفيذ:**

#### الخطوة 1: تعيين مسارات المصدر والإخراج
قم بتحديد المسارات لعرض المصدر ودليل الإخراج الخاص بك:
```csharp
string presentationName = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "PresentationDemo.pptx");
string outFilePath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "HTMLConversion");
```

#### الخطوة 2: تحميل العرض التقديمي
استخدم `Presentation` الفئة لتحميل ملف PPTX الخاص بك:
```csharp
using (Presentation pres = new Presentation(presentationName))
{
    // يستمر الكود هنا...
}
```

#### الخطوة 3: تكوين خيارات Html5
إنشاء مثيل لـ `Html5Options`، جلسة `EmbedImages` إلى false وتحديد دليل الإخراج للصور:
```csharp
Html5Options options = new Html5Options()
{
    EmbedImages = false,
    OutputPath = "YOUR_OUTPUT_DIRECTORY"
};
```

#### الخطوة 4: تأكد من وجود دليل الإخراج
تحقق مما إذا كان دليل الإخراج موجودًا وقم بإنشائه إذا لزم الأمر:
```csharp
if (!Directory.Exists(outFilePath))
{
    Directory.CreateDirectory(outFilePath);
}
```

#### الخطوة 5: الحفظ بتنسيق HTML مع الصور الخارجية
احفظ العرض التقديمي باستخدام `SaveFormat.Html5` مع خياراتك المُعدّة. ينتج عن ذلك مستند HTML وملفات صور منفصلة في دليل الإخراج المُحدّد:
```csharp
pres.Save(Path.Combine(outFilePath, "pres.html"), SaveFormat.Html5, options);
```

### نصائح استكشاف الأخطاء وإصلاحها

- **الصور المفقودة:** يضمن `EmbedImages` تم ضبطه على خطأ.
- **مشاكل الوصول إلى الدليل:** التحقق من أذونات الملف لدليل الإخراج.

# كيفية تضمين الخطوط المخصصة في HTML باستخدام Aspose.Slides لـ .NET

## مقدمة

### دليل التنفيذ
#### تضمين الخطوط في HTML
يضمن تضمين الخطوط المخصصة تناسق الطباعة. إليك كيفية القيام بذلك باستخدام Aspose.Slides لـ .NET.

##### الخطوة 1: تحميل عرض PowerPoint الخاص بك
إنشاء `Presentation` مثال لتحميل ملف PPTX الخاص بك:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outPath = "YOUR_OUTPUT_DIRECTORY";

using (Presentation pres = new Presentation(dataDir + "Presentation.pptx"))
{
    // سيتم اتخاذ الخطوات التالية هنا
}
```
##### الخطوة 2: تكوين الخطوط المراد تضمينها
حدد الخطوط التي تريد تضمينها واستبعاد خطوط النظام المحددة:
```csharp
string[] fontNameExcludeList = { "Arial" };
pres.FontsManager.EmbedAllFontsExcept(fontNameExcludeList);
```
يخبر هذا Aspose.Slides بتضمين جميع الخطوط المخصصة باستثناء تلك المدرجة في `fontNameExcludeList`.

##### الخطوة 3: حفظ العرض التقديمي بصيغة HTML
احفظ العرض التقديمي الخاص بك باستخدام الخطوط المضمنة:
```csharp
HtmlOptions htmlOpt = new HtmlOptions();
htmlOpt.HtmlFormatter = HtmlFormatter.CreateDocumentFormatter("", false);
pres.Save(outPath + "Presentation.html", SaveFormat.Html, htmlOpt);
```
يؤدي هذا إلى تحويل العرض التقديمي الخاص بك إلى ملف HTML أثناء تضمين الخطوط المحددة.

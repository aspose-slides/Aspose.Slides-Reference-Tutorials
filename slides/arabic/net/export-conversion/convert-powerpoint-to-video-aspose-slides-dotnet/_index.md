## دليل التنفيذ

### الميزة: تحويل العرض التقديمي إلى فيديو

تم تصميم هذه الميزة لاستخراج الإطارات من عرض تقديمي لبرنامج PowerPoint وتجميعها في تنسيق الفيديو.

#### الخطوة 1: تحديد المسار

ابدأ بتحديد المسار إلى ملف العرض التقديمي الخاص بك:
```csharp
string presentationName = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "SimpleAnimations.pptx");
```

#### الخطوة 2: ضبط معدل الإطارات

حدد معدل الإطارات لفيديو الإخراج الخاص بك، والذي يؤثر على الجودة وحجم الملف:
```csharp
const int FPS = 30;
```

#### الخطوة 3: تهيئة كائن العرض التقديمي

قم بتحميل العرض التقديمي الخاص بك باستخدام Aspose.Slides:
```csharp
using (Presentation presentation = new Presentation(presentationName))
{
    // مزيد من المعالجة هنا...
}
```

#### الخطوة 4: التعامل مع الرسوم المتحركة

استخدم `PresentationAnimationsGenerator` لإدارة رسوم متحركة للشرائح.
```csharp
using (var animationsGenerator = new PresentationAnimationsGenerator(presentation))
{
    // منطق التعامل مع الرسوم المتحركة...
}
```

#### الخطوة 5: إنشاء الإطارات

قم بإنشاء مثيل للاعب لاستخراج الإطارات بمعدل الإطارات المحدد لديك والتعامل مع كل إطار باستخدام معالج حدث.
```csharp
using (var player = new PresentationPlayer(animationsGenerator, FPS))
{
    player.FrameTick += (sender, args) =>
    {
        // احفظ كل إطار بصيغة PNG لإنشاء الفيديو.
        args.GetFrame().Save(Path.Combine("YOUR_OUTPUT_DIRECTORY", $"frame_{sender.FrameIndex}.png"));
    };

    animationsGenerator.Run(presentation.Slides);
}
```

### نصائح استكشاف الأخطاء وإصلاحها

- تأكد من وجود الدلائل المحددة قبل تشغيل الكود الخاص بك لتجنب أخطاء مسار الملف.
- تأكد من أن العرض التقديمي الخاص بك يحتوي على رسوم متحركة إذا كنت تستهدف الإطارات المتحركة.

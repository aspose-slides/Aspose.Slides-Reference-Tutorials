## دليل التنفيذ

### تصدير MathML باستخدام Aspose.Slides .NET

تتيح لك هذه الميزة تصدير الفقرات الرياضية إلى تنسيق MathML، مما يتيح التكامل مع الويب بسهولة.

#### الخطوة 1: إنشاء شكل رياضي

ابدأ بإنشاء شكل رياضي في عرضك التقديمي. سيحتوي هذا الشكل على التعبير الرياضي.

```csharp
var autoShape = pres.Slides[0].Shapes.AddMathShape(0, 0, 500, 50);
```

**توضيح:**
يضيف هذا الخط شكلًا رياضيًا جديدًا إلى الشريحة الأولى بأبعاد محددة (العرض: 500، الارتفاع: 50).

#### الخطوة 2: استرداد وإنشاء MathParagraph

بعد ذلك، استرد `MathParagraph` من شكل الرياضيات الخاص بك وإنشاء المعادلة الخاصة بك.

```csharp
var mathParagraph = ((MathPortion)autoShape.TextFrame.Paragraphs[0].Portions[0]).MathParagraph;

mathParagraph.Add(new Aspose.Slides.MathText.MathematicalText("a").SetSuperscript("2")
    .Join("")
    .Join(new Aspose.Slides.MathText.MathematicalText("b").SetSuperscript("2"))
    .Join("=")
    .Join(new Aspose.Slides.MathText.MathematicalText("c").SetSuperscript("2")));
```

**توضيح:**
يقوم هذا المقطع بإنشاء المعادلة (a^2 + b^2 = c^2) عن طريق إنشاء `MathematicalText` الكائنات وتعيين الحروف العلوية حيثما كان ذلك ضروريا.

#### الخطوة 3: التصدير إلى MathML

وأخيرًا، اكتب فقرتك الرياضية في ملف MathML.

```csharp
string outMathMlFileName = Path.Combine("YOUR_OUTPUT_DIRECTORY", "mathml.xml");

using (Stream stream = new FileStream(outMathMlFileName, FileMode.Create))
{
    mathParagraph.WriteAsMathMl(stream);
}
```

**توضيح:**
ال `WriteAsMathMl` تحفظ الطريقة تمثيل MathML للفقرة الخاصة بك في ملف محدد.

### نصائح استكشاف الأخطاء وإصلاحها
- تأكد من المسارات في `Path.Combine()` هي صحيحة.
- تأكد من صحة الإشارة إلى Aspose.Slides وترخيصه بشكل صحيح.

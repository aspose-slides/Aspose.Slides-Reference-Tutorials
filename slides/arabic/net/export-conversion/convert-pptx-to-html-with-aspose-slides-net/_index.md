## دليل التنفيذ
لنتعمق في تحويل عروض PowerPoint التقديمية مع الحفاظ على الخطوط باستخدام Aspose.Slides لـ .NET. سنشرحها خطوة بخطوة:

### نظرة عامة على الميزات
تتيح هذه الميزة تحويل ملفات PPTX إلى مستندات HTML، مع الحفاظ على أنماط الخطوط الأصلية كما تظهر في العرض التقديمي.

#### الخطوة 1: تحميل العرض التقديمي الخاص بك
ابدأ بتحميل ملف PowerPoint الخاص بك إلى `Presentation` هذا أمر بالغ الأهمية للوصول إلى الشرائح والتلاعب بها.

```csharp
using Aspose.Slides;
using Aspose.Slides.Export;

string dataDir = @"YOUR_DOCUMENT_DIRECTORY";
using (Presentation pres = new Presentation(dataDir + "input.pptx"))
{
    // مزيد من المعالجة هنا
}
```

**توضيح:** نبدأ بإنشاء `Presentation` الكائن الذي يسمح لنا بالتفاعل مع الشرائح في ملف PowerPoint الخاص بك.

#### الخطوة 2: تكوين إعدادات الخط
اختياريًا، حدد أي خطوط ترغب في استبعادها من التضمين في HTML. هذا يُحسّن أوقات التحميل ويُقلّل حجم الملف.

```csharp
string[] fontNameExcludeList = { "Calibri" };
```

**توضيح:** ال `fontNameExcludeList` تعرف المصفوفة الخطوط التي لا ينبغي تضمينها في مستند HTML النهائي، مما يساعد في إدارة استخدام الموارد بشكل فعال.

#### الخطوة 3: التحويل إلى HTML
بعد ذلك، حوّل شرائح العرض التقديمي إلى تنسيق HTML. يمكنك تخصيص هذه العملية بشكل أكبر بتحديد إعدادات إضافية إذا لزم الأمر.

```csharp
pres.Save(outputDir + "output.html", SaveFormat.Html5);
```

**توضيح:** ال `Save` تقوم الطريقة بتصدير العرض التقديمي كمستند HTML، مع `Html5` ضمان التوافق عبر متصفحات الويب الحديثة.

### نصائح استكشاف الأخطاء وإصلاحها:
- تأكد من المسارات في `dataDir` و `outputDir` هي صحيحة.
- تحقق مما إذا كانت الخطوط المستبعدة متاحة على الأجهزة المستهدفة لتجنب الأنماط المفقودة.

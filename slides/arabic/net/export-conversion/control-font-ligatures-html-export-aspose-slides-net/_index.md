## دليل التنفيذ

### الميزة: تعطيل ربط الخطوط أثناء التصدير

#### ملخص

سوف يرشدك هذا القسم خلال تعطيل ربط الخطوط عند تصدير عرض تقديمي بتنسيق HTML باستخدام Aspose.Slides لـ .NET.

#### التنفيذ خطوة بخطوة

**الخطوة 1: إعداد مشروعك**
قم بإنشاء مشروع C# جديد وتأكد من أنك قمت بالإشارة إلى مكتبة Aspose.Slides. 

```csharp
using Aspose.Slides;
using Aspose.Slides.Export;
using System.IO;
```

**الخطوة 2: تحديد مسارات المصدر والمخرجات**
حدد مكان عرض المصدر الخاص بك، ثم حدد المسارات لملفات HTML الناتجة.

```csharp
string presentationName = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "TextLigatures.pptx");
string outPathEnabled = Path.Combine("YOUR_OUTPUT_DIRECTORY", "EnableLigatures-out.html");
string outPathDisabled = Path.Combine("YOUR_OUTPUT_DIRECTORY", "DisableLigatures-out.html");
```

**الخطوة 3: تحميل العرض التقديمي**
قم بتحميل ملف العرض التقديمي الخاص بك باستخدام Aspose.Slides.

```csharp
using (Presentation pres = new Presentation(presentationName))
{
    // متابعة مع تكوين خيارات التصدير
}
```

**الخطوة 4: التصدير مع تمكين الربطات**
احفظ العرض التقديمي بتنسيق HTML لإظهار السلوك الافتراضي عند تمكين الربطات.

```csharp
pres.Save(outPathEnabled, SaveFormat.Html);
```

**الخطوة 5: تكوين الخيارات لتعطيل ربط الخطوط**
يثبت `HtmlOptions` وتعطيل ربط الخطوط.

```csharp
HtmlOptions options = new HtmlOptions { DisableFontLigatures = true };
```

**الخطوة 6: التصدير مع تعطيل الروابط**
قم بتصدير العرض التقديمي مرة أخرى، هذه المرة باستخدام الخيارات التي تم تكوينها.

```csharp
pres.Save(outPathDisabled, SaveFormat.Html, options);
```

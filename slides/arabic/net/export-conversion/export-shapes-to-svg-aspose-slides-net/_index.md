## دليل التنفيذ

ركز على تصدير الأشكال إلى تنسيق SVG باستخدام هذا الدليل التفصيلي:

### تصدير الشكل إلى SVG

#### ملخص
يمكنك تصدير الأشكال من أي شريحة PowerPoint إلى ملف SVG، وهو أمر مفيد لدمج الرسومات المتجهة في تطبيقات الويب أو أنظمة البرامج التي تتطلب تنسيقات قابلة للتطوير.

#### دليل خطوة بخطوة
**1. تعيين مسارات لملفات الإدخال والإخراج**
تحديد الدلائل لملفات الإدخال والإخراج:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // الدليل الذي يحتوي على ملف PowerPoint
string outSvgFileName = "YOUR_OUTPUT_DIRECTORY/SingleShape.svg"; // مسار ملف SVG الناتج
```

**2. قم بتحميل العرض التقديمي الخاص بك**
تحميل عرض تقديمي باستخدام Aspose.Slides:

```csharp
using (Presentation pres = new Presentation(dataDir + "/TestExportShapeToSvg.pptx"))
{
    // الوصول إلى الشريحة الأولى وشكلها الأول
    var slide = pres.Slides[0];
    var shape = slide.Shapes[0];

    // إنشاء FileStream لملف SVG الناتج
    using (Stream stream = new FileStream(outSvgFileName, FileMode.Create, FileAccess.Write))
    {
        // تصدير الشكل إلى تنسيق SVG
        shape.WriteAsSvg(stream);
    }
}
```

**توضيح:**
- `dataDir`:الدليل الذي يحتوي على ملف PowerPoint الخاص بك.
- `outSvgFileName`:المسار الذي سيتم حفظ SVG المُصدَّر فيه.
- **`Presentation` هدف**:يمثل مستند PowerPoint.
- **`Slide.Shapes[0]`**:الوصول إلى الشكل الأول للشريحة الأولى للتصدير.

### نصائح استكشاف الأخطاء وإصلاحها
- تأكد من أن مسار ملف الإدخال الخاص بك صحيح ويمكن الوصول إليه.
- تحقق من أذونات الملف لتأكيد إمكانية الوصول للكتابة إلى دليل الإخراج.
- تأكد من عدم تلف ملف PowerPoint عن طريق فتحه في Microsoft PowerPoint.

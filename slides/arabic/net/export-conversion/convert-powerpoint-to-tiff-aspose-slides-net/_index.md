## دليل التنفيذ

بعد إعداد بيئتنا، لنحوّل عرض PowerPoint التقديمي إلى ملف TIFF. اتبع الخطوات التالية:

### الميزة: تحويل العرض التقديمي إلى TIFF بالحجم الافتراضي

#### ملخص
قم بتحويل أي ملف PPT أو PPTX مباشرة إلى تنسيق TIFF، مع الحفاظ على إعدادات حجم الشريحة الافتراضية.

#### تحميل ملف العرض التقديمي

قم بتحميل العرض التقديمي الخاص بك إلى الذاكرة باستخدام Aspose.Slides `Presentation` فصل:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // قم بتحديث هذا المسار حسب الحاجة.
using (Presentation presentation = new Presentation(dataDir + "/DemoFile.pptx"))
{
    // تم الآن تحميل العرض التقديمي وهو جاهز للمعالجة.
}
```
**توضيح:**
- **دليل البيانات:** تعيين `dataDir` إلى المجلد الذي يوجد فيه ملف المصدر الخاص بك.
- **استخدام العبارة:** يضمن التخلص السليم من الموارد، مما يمنع تسرب الذاكرة.

#### حفظ بتنسيق TIFF

بمجرد تحميل العرض التقديمي الخاص بك، احفظه كملف TIFF:
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY"; // قم بتحديث هذا المسار حسب الحاجة.
presentation.Save(outputDir + "/Tiffoutput_out.tiff", Aspose.Slides.Export.SaveFormat.Tiff);
```
**توضيح:**
- **دليل الإخراج:** حدد المكان الذي يجب حفظ الملف المحول فيه.
- **طريقة الحفظ:** تقوم هذه الطريقة بتحويل العرض التقديمي وحفظه بالتنسيق المحدد.

- تطبيقات عملية لتحويل العروض التقديمية إلى صور GIF.

## دليل التنفيذ
الآن، دعنا ننفذ الميزة لتصدير العروض التقديمية بصيغة GIF.

### ميزة تصدير العرض التقديمي إلى GIF
تتيح لك هذه الوظيفة تحويل عرض تقديمي في PowerPoint إلى ملف GIF متحرك، وهو مثالي للمشاركة على المنصات التي تدعم تنسيقات الصور.

#### الخطوة 1: تحديد المسارات
ابدأ بتحديد المسارات لملفات الإدخال والإخراج الخاصة بك:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // مسار دليل المستند الخاص بك
string outPath = "YOUR_OUTPUT_DIRECTORY/ConvertToGif.gif"; // مسار ملف GIF الناتج
```

#### الخطوة 2: تحميل العرض التقديمي
إنشاء مثيل لـ `Presentation` الفئة لتحميل ملف PPTX الخاص بك:
```csharp
using Aspose.Slides;
using System.Drawing;

// تحميل عرض تقديمي من القرص
Presentation presentation = new Presentation(dataDir + "ConvertToGif.pptx");
```

#### الخطوة 3: تعيين خيارات GIF
قم بتكوين إعدادات التصدير عن طريق تحديد حجم الإطار والتأخير بين الشرائح ومعدل الإطارات في الثانية الانتقالي:
```csharp
using Aspose.Slides.Export;

var gifOptions = new GifOptions
{
    FrameSize = new Size(540, 480), // العرض × الارتفاع لملف GIF
    DefaultDelay = 1500,           // سيتم عرض كل شريحة في غضون ميلي ثانية
    TransitionFps = 60             // إطارات في الثانية للانتقالات السلسة
};
```

#### الخطوة 4: حفظ بصيغة GIF
وأخيرًا، احفظ العرض التقديمي في ملف GIF باستخدام الخيارات التالية:
```csharp
presentation.Save(outPath, SaveFormat.Gif, gifOptions);
```
**نصائح استكشاف الأخطاء وإصلاحها:**
- تأكد من أن مسار ملف PPTX المدخل لديك صحيح.
- تأكد من أن أذونات دليل الإخراج تسمح بكتابة الملفات.

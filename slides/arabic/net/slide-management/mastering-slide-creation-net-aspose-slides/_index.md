# إتقان إنشاء الشرائح في .NET باستخدام Aspose.Slides

## دليل التنفيذ
دعونا نقوم بتقسيم الميزات الرئيسية لكودنا خطوة بخطوة:

### إعداد الدليل
**ملخص:**  
تأكد من وجود دليل مُحدد لحفظ عرضك التقديمي. إذا لم يكن كذلك، فأنشئه تلقائيًا.

**خطوات التنفيذ:**

1. **التحقق من وجود الدليل:**  
   يستخدم `Directory.Exists` للتحقق مما إذا كان الدليل المستهدف موجودًا بالفعل.
   
2. **إنشاء الدليل:**  
   إذا لم يكن الدليل موجودًا، استخدم `Directory.CreateDirectory` لتأسيسه.

```csharp
using System.IO;

string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // استبدل بالمسار المطلوب

bool isExists = Directory.Exists(dataDir);
if (!isExists)
    Directory.CreateDirectory(dataDir);
```

### إنشاء العرض التقديمي
**ملخص:**  
قم بإعداد عرض تقديمي جديد والوصول إلى الشريحة الأولى منه، والتي تكون جاهزة للتخصيص.

**خطوات التنفيذ:**

1. **إنشاء مثيل العرض التقديمي:**  
   إنشاء مثيل `Presentation` هدف.
   
2. **استرجاع الشريحة الأولى:**  
   يمكنك الوصول إلى الشريحة الأولى باستخدام `Slides[0]` مُفهرس.

```csharp
using Aspose.Slides;

Presentation pres = new Presentation();
ISlide sld = pres.Slides[0];
```

### إضافة الشكل
**ملخص:**  
أضف شكل مستطيل إلى الشريحة الخاصة بك بأبعاد وموضع محددين.

**خطوات التنفيذ:**

1. **إضافة الشكل التلقائي:**  
   يستخدم `Shapes.AddAutoShape` لإضافة مستطيل إلى الشريحة.
   
2. **تعيين الأبعاد والموضع:**  
   قم بتحديد حجم الشكل وموقعه على الشريحة.

```csharp
using Aspose.Slides.Shapes;

IShape shp = sld.Shapes.AddAutoShape(ShapeType.Rectangle, 50, 150, 150, 75);
```

### تعبئة التنسيق
**ملخص:**  
قم بوضع طبقة من اللون الأبيض الصلب على شكل المستطيل الخاص بك للحصول على وضوح بصري.

**خطوات التنفيذ:**

1. **تعيين نوع التعبئة:**  
   تعيين `FillType.Solid` إلى تنسيق تعبئة الشكل.
   
2. **تعريف اللون:**  
   تعيين خاصية اللون إلى `Color.White`.

```csharp
shp.FillFormat.FillType = FillType.Solid;
shp.FillFormat.SolidFillColor.Color = System.Drawing.Color.White;
```

### تنسيق الخط
**ملخص:**  
قم بتخصيص نمط خط المستطيل الخاص بك بنمط سميك-رفيع، واضبط عرضه ونمط خطه.

**خطوات التنفيذ:**

1. **تطبيق نمط الخط:**  
   تعيين `LineStyle` ل `ThickThin`.
   
2. **ضبط العرض:**  
   تحديد سمك الخط.
   
3. **تعيين نمط الشرطة:**  
   اختر نمط الخط المتقطع باستخدام `LineDashStyle.Dash`.

```csharp
using Aspose.Slides.LineFormatting;

shp.LineFormat.Style = LineStyle.ThickThin;
shp.LineFormat.Width = 7;
shp.LineFormat.DashStyle = LineDashStyle.Dash;
```

### تنسيق ألوان الخط
**ملخص:**  
قم بتعزيز حدود المستطيل باستخدام اللون الأزرق الصلب.

**خطوات التنفيذ:**

1. **تعيين نوع التعبئة للحدود:**  
   يستخدم `FillType.Solid` لتنسيق تعبئة الخط.
   
2. **تحديد لون الحدود:**  
   تعيين `Color.Blue` إلى لون الخط.

```csharp
shp.LineFormat.FillFormat.FillType = FillType.Solid;
shp.LineFormat.FillFormat.SolidFillColor.Color = System.Drawing.Color.Blue;
```

### حفظ العرض التقديمي
**ملخص:**  
احفظ العرض التقديمي الخاص بك بتنسيق .pptx في الدليل المحدد.

**خطوات التنفيذ:**

1. **تحديد مسار الحفظ والتنسيق:**  
   يستخدم `pres.Save` مع مسار الملف المطلوب وتنسيق الحفظ.

```csharp
using Aspose.Slides.Export;

pres.Save(dataDir + "/RectShpLn_out.pptx", SaveFormat.Pptx);
```

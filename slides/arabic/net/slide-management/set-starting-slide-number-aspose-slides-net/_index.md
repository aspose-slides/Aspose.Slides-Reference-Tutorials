## دليل التنفيذ

الآن دعونا نتعمق في عملية تعيين رقم الشريحة الأولية في ملف العرض التقديمي.

### تعيين ميزة رقم الشريحة

يرشدك هذا القسم إلى كيفية تعديل رقم الشريحة الأولى باستخدام Aspose.Slides لـ .NET. تُعد هذه الميزة أساسية عند تنظيم الشرائح لجمهور أو أغراض مختلفة.

#### تهيئة كائن العرض التقديمي

ابدأ بإنشاء مثيل لـ `Presentation` الفئة التي تمثل ملف العرض التقديمي الخاص بك:

```csharp
using (Presentation presentation = new Presentation("HelloWorld.pptx"))
{
    // الكود سوف يذهب هنا
}
```

هنا، `"HelloWorld.pptx"` هذا هو ملف العرض التقديمي المصدر. استبدله بمسار الملف المحدد.

#### استرجاع وتعيين رقم الشريحة الأولى

بعد ذلك، قم بجلب رقم الشريحة الأولى الحالية وقم بتعيين رقم جديد:

```csharp
int firstSlideNumber = presentation.FirstSlideNumber; // احصل على رقم الشريحة الأولية الحالية

// تعيين رقم الشريحة الأولية إلى 10
presentation.FirstSlideNumber = 10;
```

يسترجع هذا المقطع شريحة البداية الحالية ويُحدِّثها. يضمن ضبط هذه القيمة أن يبدأ عرضك التقديمي من الشريحة رقم ١٠.

#### حفظ العرض التقديمي المعدل

وأخيرًا، احفظ التغييرات:

```csharp
presentation.Save("Set_Slide_Number_out.pptx");
```

من خلال حفظ الملف باسم أو مسار جديد، يمكنك الاحتفاظ بالإصدارين للرجوع إليهما واستخدامهما.

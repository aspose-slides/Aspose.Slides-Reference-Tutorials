## دليل التنفيذ

لنبدأ بضبط حجم الشريحة باستخدام Aspose.Slides لـ .NET. سنشرحها خطوة بخطوة لضمان الوضوح.

### الميزة: تعيين حجم الشريحة ونوعها

تتيح لك هذه الميزة مطابقة أبعاد الشريحة الخاصة بالعرض التقديمي الذي تم إنشاؤه مع أبعاد ملف المصدر الموجود، مما يضمن الاتساق في تخطيط المستند الخاص بك.

#### الخطوة 1: تحميل العرض التقديمي المصدر

ابدأ بإنشاء `Presentation` الكائن الذي يمثل ملف PowerPoint المصدر الخاص بك:
```csharp
// قم بتحميل العرض التقديمي المصدر من القرص.
Presentation presentation = new Presentation("YOUR_DOCUMENT_DIRECTORY/AccessSlides.pptx");
```

#### الخطوة 2: إنشاء عرض تقديمي مساعد

بعد ذلك، قم بإنشاء آخر `Presentation` مثال للتلاعب بأحجام الشرائح:
```csharp
// قم بإعداد عرض تقديمي مساعد جديد لإجراء التعديلات.
Presentation auxPresentation = new Presentation();
```

#### الخطوة 3: استرداد وتعيين حجم الشريحة

احصل على الشريحة الأولى من المصدر الخاص بك واضبط حجمها في العرض التقديمي المساعد:
```csharp
// قم بالوصول إلى الشريحة الأولى من العرض التقديمي الأصلي.
ISlide slide = presentation.Slides[0];

// قم بمطابقة حجم الشريحة مع حجم المصدر، مع التأكد من الملاءمة.
auxPresentation.SlideSize.SetSize(presentation.SlideSize.Type, SlideSizeScaleType.EnsureFit);
```

#### الخطوة 4: استنساخ الشرائح وتعديلها

قم بإدراج نسخة مستنسخة من الشريحة الأصلية في العرض التقديمي المساعد:
```csharp
// قم بإدراج الشريحة الأولى من المصدر كنسخة في العرض التقديمي المساعد.
auxPresentation.Slides.InsertClone(0, slide);

// قم بإزالة الشريحة الأولى الافتراضية للاحتفاظ بالشريحة المستنسخة فقط.
auxPresentation.Slides.RemoveAt(0);
```

#### الخطوة 5: حفظ العرض التقديمي المعدّل

وأخيرًا، احفظ التغييرات في ملف جديد:
```csharp
// إخراج العرض التقديمي المعدل بحجم الشريحة المعدل.
auxPresentation.Save("YOUR_DOCUMENT_DIRECTORY/Set_Size&Type_out.pptx", SaveFormat.Pptx);
```

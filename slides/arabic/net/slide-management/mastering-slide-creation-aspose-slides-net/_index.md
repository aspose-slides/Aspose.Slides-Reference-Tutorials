## دليل التنفيذ
دعونا نقسم عملية التنفيذ إلى ميزات مميزة. سيرشدك كل قسم إلى كيفية إضافة نص وتخصيصه في شرائحك.

### إضافة نص إلى شريحة (H2)
**ملخص:** تعرف على كيفية إدراج كتل النص في الشرائح الخاصة بك لتحقيق تواصل واضح.

#### الخطوة 1: إنشاء عرض تقديمي جديد (H3)
ابدأ بتهيئة كائن عرض تقديمي جديد:
```csharp
using (Presentation pres = new Presentation())
{
    // سيتم وضع الكود لإضافة النص هنا
}
```

#### الخطوة 2: إضافة شكل تلقائي ونص (H3)
أضف شكل مستطيل إلى الشريحة الخاصة بك، والذي سيكون بمثابة الحاوية للنص الخاص بك:
```csharp
IAutoShape shape = pres.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 10, 10, 200, 250);
```

#### الخطوة 3: إدراج الفقرة والجزء (H3)
قم بإنشاء فقرة تحتوي على نص لإضافته إلى إطار نص الشكل:
```csharp
Paragraph para1 = new Paragraph();
para1.Portions.Add(new Portion("Sample text"));
shape.TextFrame.Paragraphs.Add(para1);
```
**توضيح:** `IAutoShape` يسمح بالتلاعب الديناميكي بالأشكال. `Portion` تمثل الفئة كتلة من النص داخل فقرة.

### تخصيص خصائص الفقرة النهائية (H2)
**ملخص:** قم بتعديل مظهر فقراتك لتناسب احتياجات العرض التقديمي المحددة.

#### الخطوة 1: إضافة فقرة جديدة باستخدام خصائص مخصصة (H3)
بعد إضافة النص الأساسي، قم بتخصيص خصائصه للتأكيد:
```csharp
Paragraph para2 = new Paragraph();
para2.Portions.Add(new Portion("Sample text 2"));

PortionFormat endParaFormat = new PortionFormat()
{
    FontHeight = 48,
    LatinFont = new FontData("Times New Roman")
};
para2.EndParagraphPortionFormat = endParaFormat;
shape.TextFrame.Paragraphs.Add(para2);
```
**توضيح:** ال `PortionFormat` تتيح الفئة إمكانية التخصيص التفصيلي، مثل تغيير حجم الخط ونوعه.

### حفظ عرض تقديمي (H2)
**ملخص:** احفظ عملك للتأكد من الحفاظ على كافة التغييرات.

#### الخطوة 1: تصدير العرض التقديمي (H3)

## قسم الأسئلة الشائعة (H2)
1. **كيف يمكنني تخصيص لون النص في Aspose.Slides؟**
   - استخدم `PortionFormat.FillFormat` الخاصية لتعيين لون التعبئة المطلوب لأجزاء النص.

2. **هل يمكنني إضافة نقاط محددة باستخدام Aspose.Slides؟**

3. **هل من الممكن تنسيق فقرات متعددة في وقت واحد؟**
   - على الرغم من أن التخصيص الفردي أمر بسيط، فكر في التكرار عبر الفقرات لتطبيق تغييرات التنسيق المجمعة.

## دليل التنفيذ
الآن، دعنا ننفذ الميزة لإضافة إطار صورة وتنسيقه في PowerPoint باستخدام C#.

### إضافة صورة كإطار للصورة

**ملخص:**
يتناول هذا القسم كيفية إدراج صورة بطريقة برمجية في شريحة العرض التقديمي كإطار صورة، وتعيين أبعادها وموضعها بدقة.

#### الخطوة 1: إعداد دليل المستندات الخاص بك
أولاً، حدد الدليل الذي تتواجد فيه مستنداتك. تأكد من وجود هذا الدليل أو أنشئه إذا لزم الأمر:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
bool IsExists = Directory.Exists(dataDir);
if (!IsExists)
    Directory.CreateDirectory(dataDir);
```

#### الخطوة 2: إنشاء عرض تقديمي جديد والوصول إلى الشريحة الأولى
بعد ذلك، قم بإنشاء كائن عرض تقديمي جديد واحصل على إمكانية الوصول إلى الشريحة الأولى الخاصة به:

```csharp
using (Presentation pres = new Presentation())
{
    ISlide sld = pres.Slides[0];
```

#### الخطوة 3: تحميل صورة في العرض التقديمي
حمّل ملف الصورة المطلوب إلى العرض التقديمي. يستخدم هذا المثال صورةً باسم "aspose-logo.jpg":

```csharp
IImage img = Images.FromFile(dataDir + "aspose-logo.jpg");
IPPImage imgx = pres.Images.AddImage(img);
```

#### الخطوة 4: إضافة إطار صورة إلى الشريحة
أضف إطار الصورة بالأبعاد والموضع المحددين على الشريحة:

```csharp
IPictureFrame pf = sld.Shapes.AddPictureFrame(
    ShapeType.Rectangle, 50, 150, imgx.Width, imgx.Height, imgx);
```

#### الخطوة 5: تنسيق إطار الصورة
قم بتخصيص مظهر إطار صورتك عن طريق ضبط لون الخط والعرض والدوران:

```csharp
pf.LineFormat.FillFormat.FillType = FillType.Solid;
pf.LineFormat.FillFormat.SolidFillColor.Color = Color.Blue;
pf.LineFormat.Width = 20;
pf.Rotation = 45;
```

#### الخطوة 6: حفظ العرض التقديمي
وأخيرًا، احفظ العرض التقديمي الخاص بك باستخدام إطار الصورة المُنسق حديثًا:

```csharp
pres.Save(dataDir + "RectPicFrameFormat_out.pptx", SaveFormat.Pptx);
}
```

**نصيحة لاستكشاف الأخطاء وإصلاحها:** إذا واجهت أخطاء في مسار الملف، فتأكد من إعادة التحقق منه `dataDir` وتأكد من وضع جميع الملفات الضرورية بشكل صحيح.

# إتقان Aspose.Slides .NET: تخصيص ألوان الارتباطات التشعبية في PowerPoint

## مقدمة

قد يكون التنقل عبر عرض تقديمي في PowerPoint مملاً أحيانًا عندما تظهر الروابط التشعبية كنص عادي. تخيل إمكانية تخصيص ألوان هذه الروابط بسهولة! يوضح لك هذا الدليل كيفية تعيين ألوان الروابط التشعبية باستخدام Aspose.Slides لـ .NET، وهي مكتبة فعّالة لإدارة العروض التقديمية برمجيًا.

في هذا البرنامج التعليمي، سوف تتعلم:
- كيفية تخصيص ألوان الارتباط التشعبي في شرائح PowerPoint.
- خطوات إضافة الروابط التشعبية دون تخصيص الألوان.
- التطبيقات العملية وإمكانيات التكامل لـ Aspose.Slides لـ .NET.

دعونا نبدأ بمراجعة المتطلبات الأساسية اللازمة قبل أن نبدأ.

## دليل التنفيذ

سنستكشف ميزتين أساسيتين: تعيين لون مخصص للارتباطات التشعبية وإضافة ارتباطات تشعبية قياسية دون تخصيص.

### الميزة 1: تعيين لون الارتباط التشعبي في شرائح PowerPoint

تتيح لك هذه الميزة تغيير لون نص الارتباط التشعبي، مما يعزز الرؤية أو يطابق موضوع التصميم الخاص بك.

#### التنفيذ خطوة بخطوة:

**1. تحميل العرض التقديمي**
ابدأ بتحميل عرض تقديمي موجود أو إنشاء عرض تقديمي جديد باستخدام Aspose.Slides.

```csharp
using (Presentation presentation = new Presentation())
{
    // متابعة بالخطوات التالية...
}
```

**2. إضافة الشكل التلقائي وإطار النص**
قم بإنشاء شكل وأضف نصًا يتضمن ارتباطك التشعبي.

```csharp
IAutoShape shape1 = presentation.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 100, 100, 450, 50, false);
shape1.AddTextFrame("This is a sample of colored hyperlink.");
```

**3. تعيين عنوان URL للارتباط التشعبي ومصدر اللون**
قم بتعيين عنوان URL للرابط التشعبي وحدد أن اللون يجب أن يكون مشتقًا من PortionFormat.

```csharp
shape1.TextFrame.Paragraphs[0].Portions[0].PortionFormat.HyperlinkClick = new Hyperlink("https://www.aspose.com/");
shape1.TextFrame.Paragraphs[0].Portions[0].PortionFormat.HyperlinkClick.ColorSource = HyperlinkColorSource.PortionFormat;
```

**4. تخصيص لون التعبئة**
تغيير لون نص الارتباط التشعبي عن طريق تعيين تعبئة صلبة.

```csharp
shape1.TextFrame.Paragraphs[0].Portions[0].PortionFormat.FillFormat.FillType = FillType.Solid;
shape1.TextFrame.Paragraphs[0].Portions[0].PortionFormat.FillFormat.SolidFillColor.Color = Color.Red;
```

### الميزة 2: تعيين ارتباط تشعبي عادي

لتنفيذ الارتباط التشعبي القياسي دون تخصيص الألوان، اتبع الخطوات التالية:

**1. تحميل العرض التقديمي**
على غرار الميزة السابقة، ابدأ بالعرض التقديمي الخاص بك.

```csharp
using (Presentation presentation = new Presentation())
{
    // متابعة إضافة الروابط التشعبية...
}
```

**2. إضافة الشكل التلقائي وإطار النص**
إنشاء شكل لرابط النص التشعبي الخاص بك.

```csharp
IAutoShape shape2 = presentation.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 100, 200, 450, 50, false);
shape2.AddTextFrame("This is a sample of usual hyperlink.");
```

**3. تعيين عنوان URL للارتباط التشعبي**
تعيين عنوان URL للرابط التشعبي.

```csharp
shape2.TextFrame.Paragraphs[0].Portions[0].PortionFormat.HyperlinkClick = new Hyperlink("https://www.aspose.com/");
```

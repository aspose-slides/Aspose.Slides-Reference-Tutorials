# إتقان تباعد الأسطر في شرائح PowerPoint باستخدام Aspose.Slides لـ .NET
## مقدمة

في هذه المقالة، سنغطي:
- إعداد بيئتك باستخدام Aspose.Slides لـ .NET
- تنفيذ تعديلات تباعد الأسطر في نص الشريحة
- تطبيقات عملية ونصائح للأداء

## دليل التنفيذ
### ضبط تباعد الأسطر في شرائح PowerPoint
يُعد ضبط تباعد الأسطر أمرًا بالغ الأهمية لعروض تقديمية مُحسّنة وقراءة أفضل للنصوص. اتبع هذه الخطوات باستخدام Aspose.Slides .NET.
#### الخطوة 1: إعداد مسارات المستندات
قم بتحديد المكان الذي يوجد فيه مستند الإدخال الخاص بك وسيتم حفظ ملف الإخراج:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";
```
تحدد هذه الخطوة المسارات لتحميل عرض تقديمي موجود وحفظ التعديلات.
#### الخطوة 2: تحميل العرض التقديمي
قم بتحميل ملف PowerPoint يحتوي على نص للتنسيق:
```csharp
// تحميل العرض التقديمي بخطوط محددة
document.Presentation presentation = new Presentation(dataDir + "/Fonts.pptx");
```
تقوم هذه الطريقة بتحميل العرض التقديمي الخاص بك للتلاعب البرمجي.
#### الخطوة 3: الوصول إلى الشريحة
انتقل إلى الشريحة التي تريد ضبط تباعد النص فيها. سنركز على الشريحة الأولى:
```csharp
ISlide sld = presentation.Slides[0];
```
#### الخطوة 4: استرداد إطار النص
استرجاع `TextFrame` للوصول إلى النص وتعديله داخل الأشكال:
```csharp
ITextFrame tf1 = ((IAutoShape)sld.Shapes[0]).TextFrame;
```
بافتراض أن الشكل الأول على الشريحة هو شكل تلقائي يحتوي على نص.
#### الخطوة 5: الوصول إلى الفقرة
يمكنك الوصول إلى الفقرة للتعديل، مما يسمح بإجراء تعديلات التباعد الفردية:
```csharp
IParagraph para1 = tf1.Paragraphs[0];
```
#### الخطوة 6: تكوين خصائص التباعد
تعيين خصائص تباعد الأسطر لتحسين قابلية القراءة:
```csharp
para1.ParagraphFormat.SpaceWithin = 80; // مسافة السطر داخل الفقرة نفسها
para1.ParagraphFormat.SpaceBefore = 40; // مسافة قبل بداية الفقرة
para1.ParagraphFormat.SpaceAfter = 40;  // مسافة بعد نهاية الفقرة
```
ال `SpaceWithin` تتحكم المعلمة في المسافة بين الأسطر في الفقرة، بينما `SpaceBefore` و `SpaceAfter` السيطرة على المساحة المحيطة.
#### الخطوة 7: حفظ العرض التقديمي المعدّل
احفظ العرض التقديمي الخاص بك مع التغييرات المطبقة:
```csharp
document.Presentation.Save(outputDir + "/LineSpacing_out.pptx", SaveFormat.Pptx);
```
يؤدي هذا إلى كتابة العرض التقديمي المعدل إلى ملف جديد في دليل الإخراج المحدد.
### نصائح استكشاف الأخطاء وإصلاحها

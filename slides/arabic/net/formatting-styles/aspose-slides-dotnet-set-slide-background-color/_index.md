## دليل التنفيذ
الآن بعد إعداد بيئتنا، دعنا ننفذ الميزة لتخصيص ألوان خلفية الشريحة.

### تعيين خلفية الشريحة إلى لون ثابت

#### ملخص
يركز هذا القسم على تغيير خلفية شريحة PowerPoint إلى لون ثابت باستخدام Aspose.Slides لـ .NET. تساعد هذه التقنية في الحفاظ على تناسق العلامة التجارية أو إنشاء شرائح جذابة بصريًا.

##### الخطوة 1: إعداد مشروعك ومسارات الملفات
تأكد من تعريف المستندات ومجلدات الإخراج بشكل صحيح:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

bool isExists = System.IO.Directory.Exists(dataDir);
if (!isExists)
    System.IO.Directory.CreateDirectory(dataDir);
```

##### الخطوة 2: تهيئة العرض التقديمي
إنشاء مثيل لـ `Presentation` الفئة التي تمثل ملف PowerPoint الخاص بك:

```csharp
using (Presentation pres = new Presentation())
{
    // الوصول إلى الشريحة الأولى في العرض التقديمي
    ISlide slide = pres.Slides[0];
}
```

##### الخطوة 3: تعيين نوع الخلفية واللون
قم بتكوين نوع الخلفية وتنسيق التعبئة لتغييره إلى لون ثابت:

```csharp
slide.Background.Type = BackgroundType.OwnBackground;
slide.Background.FillFormat.FillType = FillType.Solid;

// تعيين لون الخلفية إلى اللون الأزرق
display.BackgroundColor.SolidFillColor.Color = System.Drawing.Color.Blue;
```

##### الخطوة 4: احفظ العرض التقديمي الخاص بك
وأخيرًا، احفظ التغييرات في ملف PowerPoint جديد:

```csharp
pres.Save(outputDir + "ContentBG_out.pptx", SaveFormat.Pptx);
```

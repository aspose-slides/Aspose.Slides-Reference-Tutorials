## دليل التنفيذ

### تدوير الأشكال في PowerPoint

يرشدك هذا القسم خلال عملية تدوير الشكل التلقائي داخل الشريحة لإضافة اهتمام بصري والتأكيد على أجزاء معينة من المحتوى.

#### الخطوة 1: جهّز بيئتك

تحديد الدليل لحفظ المستندات:
```csharp
string dataDir = @"YOUR_DOCUMENT_DIRECTORY";
bool isExists = System.IO.Directory.Exists(dataDir);
if (!isExists)
    System.IO.Directory.CreateDirectory(dataDir);
```
يضمن هذا وجود دليل الإخراج الخاص بك، مما يمنع حدوث أخطاء أثناء حفظ الملف.

#### الخطوة 2: إنشاء عرض تقديمي جديد

تهيئة الشريحة الأولى والوصول إليها:
```csharp
using (Presentation pres = new Presentation())
{
    // الوصول إلى الشريحة الأولى
    ISlide sld = pres.Slides[0];
```
قم بإنشاء نموذج عرض تقديمي والوصول إلى الشريحة الأولى لإضافة الشكل الخاص بك.

#### الخطوة 3: إضافة شكل تلقائي وتدويره

أضف شكل مستطيل وقم بتدويره بمقدار 90 درجة:
```csharp
// إضافة شكل مستطيل تلقائي
IShape shp = sld.Shapes.AddAutoShape(ShapeType.Rectangle, 50, 150, 75, 150);

// تدوير المستطيل بمقدار 90 درجة
shp.Rotation = 90;
```
ال `AddAutoShape` تضع الطريقة الشكل عند إحداثيات وأبعاد محددة. `Rotation` الخاصية تعدل زاويتها.

#### الخطوة 4: احفظ العرض التقديمي الخاص بك

احفظ العرض التقديمي الخاص بك:
```csharp
// حفظ العرض التقديمي المعدل
pres.Save(dataDir + "RectShpRot_out.pptx");
}
```
يؤدي هذا إلى كتابة تغييراتك في ملف في الدليل المحدد.

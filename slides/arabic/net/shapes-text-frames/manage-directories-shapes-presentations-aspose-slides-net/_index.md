## دليل التنفيذ

ينقسم هذا القسم إلى ميزتين رئيسيتين: إنشاء الدلائل إذا لم تكن موجودة والعمل مع أشكال العرض لإضافة الصور.

### إنشاء الدلائل

#### ملخص
من الضروري التأكد من وجود دليل قبل إجراء عمليات على الملفات. تساعد هذه الميزة في التحقق من وجود دليل محدد، وإن لم يكن موجودًا، يتم إنشاؤه، مما يمنع الأخطاء المحتملة أثناء معالجة الملفات.

#### خطوات التنفيذ

**الخطوة 1: تحديد مسار الدليل**
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
```
*يستبدل `YOUR_DOCUMENT_DIRECTORY` مع المسار الذي تريده.*

**الخطوة 2: التحقق من الدليل وإنشائه**
```csharp
bool isExists = Directory.Exists(dataDir);
if (!isExists) {
```

### العمل مع العروض التقديمية والأشكال

#### ملخص
دمج الصور في عروضك التقديمية يجعلها أكثر جاذبية. توضح هذه الميزة كيفية تحميل عرض تقديمي، وإضافة صورة لتعبئة الشكل، وضبط الإزاحات لتحسين الموضع.

#### خطوات التنفيذ

**الخطوة 1: تحميل الصورة**
```csharp
IImage img = Images.FromFile(dataDir + "aspose-logo.jpg");
```
*تأكد من أن مسار الصورة صحيح.*

**الخطوة 2: تهيئة العرض التقديمي وإضافة الشكل**
```csharp
using (Presentation pres = new Presentation()) {
    ISlide slide = pres.Slides[0];
    IAutoShape aShape = slide.Shapes.AddAutoShape(ShapeType.Rectangle, 100, 100, 300, 300);
    
    aShape.FillFormat.FillType = FillType.Picture;
    aShape.FillFormat.PictureFillFormat.PictureFillMode = PictureFillMode.Stretch;
    IPPImage imgEx = pres.Images.AddImage(img);
    aShape.FillFormat.PictureFillFormat.Picture.Image = imgEx;

    // إزاحات المجموعة
    aShape.FillFormat.PictureFillFormat.StretchOffsetLeft = 25;
    aShape.FillFormat.PictureFillFormat.StretchOffsetRight = 25;
    aShape.FillFormat.PictureFillFormat.StretchOffsetTop = -20;
    aShape.FillFormat.PictureFillFormat.StretchOffsetBottom = -10;

    pres.Save(dataDir + "StretchOffsetLeftForPictureFrame_out.pptx", SaveFormat.Pptx);
}
```
يقوم هذا المقطع بتحميل صورة، وإضافتها إلى الشريحة الأولى كتعبئة على شكل مستطيل، وتعيين الإزاحات لتحسين المحاذاة.

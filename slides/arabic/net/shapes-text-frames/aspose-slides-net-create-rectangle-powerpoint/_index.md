## دليل التنفيذ

### الميزة 1: إنشاء مستطيل بسيط في PowerPoint (H2)

أتمتة إضافة أشكال المستطيلات لتوفير الوقت وضمان الاتساق في العروض التقديمية. إليك كيفية إضافة مستطيل باستخدام Aspose.Slides لـ .NET.

#### التنفيذ خطوة بخطوة (H3)

1. **تهيئة فئة العرض التقديمي**
   
   إنشاء مثيل لـ `Presentation` الفئة التي تمثل ملف PowerPoint الخاص بك:

   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;

   string YOUR_DOCUMENT_DIRECTORY = "YOUR_DOCUMENT_DIRECTORY";

   using (Presentation pres = new Presentation())
   {
       // يستمر الكود هنا...
   }
   ```

2. **الوصول إلى الشريحة الأولى**

   استرجاع الشريحة الأولى من العرض التقديمي الخاص بك:

   ```csharp
   ISlide sld = pres.Slides[0];
   ```

3. **إضافة شكل مستطيل**

   يستخدم `AddAutoShape` لإضافة مستطيل في مواضع وأحجام محددة:

   ```csharp
   sld.Shapes.AddAutoShape(ShapeType.Rectangle, 50, 150, 150, 50);
   ```
   
   - **حدود**:الطريقة تقبل `ShapeType`، موضع x، وموضع y، والعرض، والارتفاع لتحديد موضع الشكل وحجمه.

4. **حفظ العرض التقديمي**

   احفظ العرض التقديمي الخاص بك لتخزين كافة التغييرات:

   ```csharp
   pres.Save(YOUR_DOCUMENT_DIRECTORY + "/RectShp1_out.pptx", SaveFormat.Pptx);
   ```

#### نصائح استكشاف الأخطاء وإصلاحها

- يضمن `YOUR_DOCUMENT_DIRECTORY` تم تعيين المسارات بشكل صحيح.
- تأكد من الإشارة إلى Aspose.Slides بشكل صحيح في مشروعك.

### الميزة 2: إنشاء الدليل والتحقق منه (H2)

إدارة المجلدات بكفاءة تمنع حدوث أخطاء عند حفظ الملفات. نفّذ هذا الفحص للتأكد من وجود المجلدات قبل محاولة حفظ الملف.

#### التنفيذ خطوة بخطوة (H3)

1. **تحديد مسار الدليل**

   حدد المكان الذي سيتم تخزين مستنداتك فيه:

   ```csharp
   string dataDir = YOUR_DOCUMENT_DIRECTORY;
   ```

2. **التحقق من الدليل وإنشائه إذا لزم الأمر**

   يستخدم `Directory.Exists` للتحقق من وجود الدليل، وإنشائه إذا لزم الأمر:

   ```csharp
   bool isExists = Directory.Exists(dataDir);
   if (!isExists)
   {
       Directory.CreateDirectory(dataDir);
   }
   ```

#### نصائح استكشاف الأخطاء وإصلاحها

- تأكد من أن تطبيقك لديه الإذن لإنشاء الدلائل في المسار المحدد.
- معالجة الاستثناءات من المسارات غير الصالحة أو الأذونات غير الكافية.

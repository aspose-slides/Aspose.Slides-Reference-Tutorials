### دليل التنفيذ

#### تطبيق التعبئة المتدرجة على الأشكال

تتيح لك هذه الميزة تحسين المظهر البصري للأشكال في شرائح PowerPoint بإضافة تعبئة متدرجة. لنستكشف كيفية تطبيق ذلك:

##### الخطوة 1: إنشاء شكل بيضاوي

```csharp
// تحميل أو إنشاء عرض تقديمي\باستخدام (Presentation pres = new Presentation())
{
    // الوصول إلى الشريحة الأولى
    ISlide sld = pres.Slides[0];
    
    // إضافة شكل تلقائي من نوع القطع الناقص
    IAutoShape ashp = sld.Shapes.AddAutoShape(ShapeType.Ellipse, 50, 150, 150, 50);
}
```

في هذه الخطوة، ننشئ شكلًا بيضاويًا على الشريحة الأولى. تُحدد المعلمات موقعه وحجمه.

##### الخطوة 2: تطبيق التعبئة المتدرجة

```csharp
// تعيين نوع التعبئة إلى التدرج
ashp.FillFormat.FillType = FillType.Gradient;

// تحديد ألوان التدرج والأسلوب
ashp.FillFormat.GradientFormat.StartColor = Color.Red;
ashp.FillFormat.GradientFormat.EndColor = Color.Blue;
ashp.FillFormat.GradientFormat.TileFlip = TileFlip.FlipBoth;
```

هنا، قمنا بتكوين القطع الناقص ليكون له تعبئة متدرجة، ينتقل من اللون الأحمر إلى اللون الأزرق.

##### الخطوة 3: حفظ العرض التقديمي

```csharp
// تحديد مسار الإخراج
string dataDir = "YOUR_DOCUMENT_DIRECTORY";

// تأكد من وجود الدليل
if (!Directory.Exists(dataDir))
{
    Directory.CreateDirectory(dataDir);
}

// حفظ العرض التقديمي
pres.Save(Path.Combine(dataDir, "GradientEllipse.pptx"), SaveFormat.Pptx);
```

تضمن هذه القطعة حفظ العرض التقديمي في الدليل المحدد.

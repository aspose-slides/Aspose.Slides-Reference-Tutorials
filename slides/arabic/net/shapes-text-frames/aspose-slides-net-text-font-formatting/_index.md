## دليل التنفيذ

سنقوم بتقسيم التنفيذ إلى أقسام مميزة استنادًا إلى الوظيفة.

### إضافة نص إلى شكل

#### ملخص
تتيح لك هذه الميزة إضافة نص مخصص داخل الأشكال التلقائية، مثل المستطيلات في شرائحك. وهي ضرورية لعرض محتوى مخصص مباشرةً على أشكال الشرائح.

#### خطوات التنفيذ

**1. إنشاء وإضافة شكل تلقائي**

```csharp
using (Presentation pres = new Presentation())
{
    IAutoShape newShape = pres.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 100, 100, 400, 75, false);
```
- **حدود**: 
  - `ShapeType.Rectangle`:يحدد نوع الشكل.
  - الإحداثيات (x=100، y=100) والأبعاد (العرض=400، الارتفاع=75): موضع وحجم الشكل.

**2. إضافة إطار نصي**

```csharp
    newShape.AddTextFrame("");
```
- **غاية**:يتم تهيئة إطار نص فارغ لحمل النص المخصص الخاص بك.

**3. إدراج أجزاء النص**

```csharp
    newShape.TextFrame.Paragraphs[0].Portions.Clear();
    
    IPortion portion0 = new Portion("Sample text with first portion");
    IPortion portion1 = new Portion(" and second portion.");
    
    newShape.TextFrame.Paragraphs[0].Portions.Add(portion0);
    newShape.TextFrame.Paragraphs[0].Portions.Add(portion1);
}
```
- **توضيح**:امسح الأجزاء الموجودة، ثم أنشئ وأضف مقاطع نصية جديدة. يتيح لك هذا تقسيم المحتوى إلى فقرة واحدة.

### تعيين ارتفاع الخط الافتراضي للعرض التقديمي

#### ملخص
يؤدي تعيين ارتفاع خط موحد عبر العرض التقديمي بأكمله إلى ضمان الاتساق في التصميم وسهولة القراءة.

#### خطوات التنفيذ

**1. إضافة أجزاء نصية**
أعد استخدام الكود لإضافة أجزاء النص كما هو موضح أعلاه.

**2. تعيين ارتفاع الخط الافتراضي**

```csharp
    pres.DefaultTextStyle.GetLevel(0).DefaultPortionFormat.FontHeight = 24;
```
- **غاية**:يتم تطبيق ارتفاع خط ثابت يبلغ 24 نقطة على جميع أجزاء النص في العرض التقديمي.

### تعيين ارتفاع الخط الافتراضي للفقرة

#### ملخص
بإمكانك تخصيص فقرات فردية ضمن شرائحك، مما يجعل محتوى معينًا بارزًا.

#### خطوات التنفيذ

**1. إضافة أجزاء نصية**
كما هو موضح سابقا.

**2. تخصيص ارتفاع الخط لفقرة محددة**

```csharp
    newShape.TextFrame.Paragraphs[0].ParagraphFormat.DefaultPortionFormat.FontHeight = 40;
```
- **توضيح**:يضبط ارتفاع الخط لجميع الأجزاء ضمن هذه الفقرة إلى 40 نقطة، مما يعزز تأثيرها البصري.

### ضبط ارتفاع الخط لجزء فردي

#### ملخص
للحصول على تحكم دقيق في طباعة العرض التقديمي الخاص بك، قم بتعديل حجم الخط لأجزاء النص المحددة بشكل فردي.

#### خطوات التنفيذ

**1. إضافة أجزاء نصية**
ارجع إلى الخطوات الأولية في إضافة أجزاء النص.

**2. تعيين ارتفاعات الخطوط المحددة**

```csharp
    newShape.TextFrame.Paragraphs[0].Portions[0].PortionFormat.FontHeight = 55;
    newShape.TextFrame.Paragraphs[0].Portions[1].PortionFormat.FontHeight = 18;
```
- **توضيح**:يمنح هذا التخصيص لكل جزء ارتفاعات خط فريدة، مما يسمح بالتأكيد التفصيلي حيثما لزم الأمر.

### حفظ العرض التقديمي

#### ملخص
بمجرد تصميم العرض التقديمي الخاص بك بشكل مثالي، احفظه بتنسيق الملف الذي تختاره.

```csharp
using (Presentation pres = new Presentation())
{
    // أضف الأشكال والنص كما هو موضح أعلاه...

    // حفظ العرض التقديمي
    pres.Save("YOUR_OUTPUT_DIRECTORY\SetLocalFontHeightValues.pptx", SaveFormat.Pptx);
}
```
- **تفاصيل**:يؤدي هذا إلى حفظ الشرائح المنسقة في ملف PPTX، جاهزًا للتوزيع أو التحرير الإضافي.

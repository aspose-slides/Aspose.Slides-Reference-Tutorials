## دليل التنفيذ

### إضافة نص علوي
#### ملخص
تحسين قابلية القراءة باستخدام نص علوي للصيغ أو التعليقات التوضيحية أو الاستشهادات:

1. **الوصول إلى الشريحة**:قم بتحميل الشريحة التي تريد إضافة النص إليها.
2. **إنشاء شكل**:أضف شكلاً (مثل المستطيل) لحمل النص الخاص بك.
3. **تكوين إطار النص**:قم بإعداد إطار النص الخاص بك ومسح الفقرات الموجودة.
4. **إضافة جزء علوي**:أدخل جزء النص الذي يجب أن يكون علويًا.

#### التنفيذ خطوة بخطوة
**1. الوصول إلى الشريحة**
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
using (Presentation presentation = new Presentation())
{
    ISlide slide = presentation.Slides[0];
```
قم بتحميل عرض تقديمي موجود والوصول إلى الشريحة الأولى منه.

**2. إنشاء شكل**
```csharp
IAutoShape shape = slide.Shapes.AddAutoShape(ShapeType.Rectangle, 100, 100, 200, 100);
ITextFrame textFrame = shape.TextFrame;
```
أضف شكلًا مستطيلًا إلى الشريحة وقم بإعدادها لإدخال النص.

**3. تكوين إطار النص**
```csharp
textFrame.Paragraphs.Clear();
IParagraph superPar = new Paragraph();
```
قم بمسح الفقرات الموجودة للبدء من جديد، ثم قم بإنشاء فقرة جديدة للنص العلوي.

**4. إضافة جزء علوي**
لإضافة نص علوي:
- إنشاء أجزاء عادية وعلوية.
- اضبط `PortionFormat.FontHeight` وخصائص أخرى حسب الحاجة.

```csharp
IPortion portion1 = new Portion { Text = "Slide Title" };
portion1.PortionFormat.FontHeight = 20;

// نص علوي
IPortion portion2 = new Portion { Text = "Superscript Example" };
portion2.PortionFormat.FontHeight = 10;
portion2.ParagraphFormat.DefaultPortionFormat.FontBold = NullableBool.True;
portion2.TextFrame.Paragraphs[0].Portions[1].PortionFormat.Superscript = new Superscript() 
{ 
    FontSize = 50 %, 
    Position = SuperscriptPosition.VerticallyAboveBaseline
};

superPar.Portions.Add(portion1);
superPar.Portions.Add(portion2);
textFrame.Paragraphs.Add(superPar);
```
**نصائح استكشاف الأخطاء وإصلاحها**:
- يضمن `PortionFormat.Superscript` تم ضبطه بشكل صحيح مع حجم الخط وموضعه المناسبين.
- تأكد من إضافة الأجزاء إلى الفقرات بالترتيب الصحيح.

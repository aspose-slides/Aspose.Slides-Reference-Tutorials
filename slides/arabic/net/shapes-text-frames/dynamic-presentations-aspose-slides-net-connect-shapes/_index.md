## دليل التنفيذ
### الميزة 1: ربط الأشكال باستخدام موقع الاتصال
توضح هذه الميزة كيفية توصيل قطع ناقص ومستطيل باستخدام موصل في مؤشر موقع اتصال محدد.

#### التنفيذ خطوة بخطوة:
**1. حدد مسار دليل مستند الإخراج**
حدد المكان الذي سيتم فيه حفظ العرض التقديمي الناتج.
```csharp
string dataDir = @"YOUR_DOCUMENT_DIRECTORY/ShapeConnectionOutput.pptx";
```

**2. إنشاء كائن عرض تقديمي**
إنشاء مثيل جديد `Presentation` الكائن الذي يمثل ملف PowerPoint الخاص بك:
```csharp
using (Presentation presentation = new Presentation())
{
    // مزيد من التعليمات البرمجية هنا...
}
```

**3. الوصول إلى مجموعة الأشكال الخاصة بالشريحة الأولى**
احصل على إمكانية الوصول إلى جميع الأشكال الموجودة في الشريحة الأولى.
```csharp
IShapeCollection shapes = presentation.Slides[0].Shapes;
```

**4. إضافة شكل موصل**
أضف موصلاً يربط الأشكال الأخرى معًا:
```csharp
IConnector connector = shapes.AddConnector(ShapeType.BentConnector3, 0, 0, 10, 10);
```

**5. إضافة الأشكال (القطع الناقص والمستطيل)**
إدراج القطع الناقص والمستطيل في المجموعة.
```csharp
IAutoShape ellipse = shapes.AddAutoShape(ShapeType.Ellipse, 0, 100, 100, 100);
IAutoShape rectangle = shapes.AddAutoShape(ShapeType.Rectangle, 100, 200, 100, 100);
```

**6. قم بتوصيل الأشكال باستخدام الموصل**
قم بربط القطع الناقص والمستطيل باستخدام الموصل.
```csharp
connector.StartShapeConnectedTo = ellipse;
connector.EndShapeConnectedTo = rectangle;
```

**7. حدد مؤشر موقع الاتصال على Ellipse**
اختر مؤشر موقع اتصال محددًا للاتصالات الدقيقة:
```csharp
uint wantedIndex = 6;

if (ellipse.ConnectionSiteCount > wantedIndex)
{
    connector.StartShapeConnectionSiteIndex = wantedIndex;
}
```

**8. احفظ العرض التقديمي**
احفظ العرض التقديمي الخاص بك للحفاظ على التغييرات.
```csharp
presentation.Save(dataDir, SaveFormat.Pptx);
```

### الميزة 2: إضافة الأشكال إلى الشريحة
تُظهر هذه الميزة كيفية إضافة أشكال مختلفة مثل القطع الناقص والمستطيلات مباشرة إلى شريحة.

#### التنفيذ خطوة بخطوة:
**1. حدد مسار دليل مستند الإخراج**
حدد المكان الذي سيتم حفظ ملف الإخراج الخاص بك فيه.
```csharp
string dataDir = @"YOUR_DOCUMENT_DIRECTORY/ShapeAdditionOutput.pptx";
```

**2. إنشاء كائن عرض تقديمي**
ابدأ بإنشاء حساب جديد `Presentation` هدف:
```csharp
using (Presentation presentation = new Presentation())
{
    // مزيد من التعليمات البرمجية هنا...
}
```

**3. الوصول إلى مجموعة الأشكال الخاصة بالشريحة الأولى**
الوصول إلى كافة الأشكال الموجودة في الشريحة الأولى.
```csharp
IShapeCollection shapes = presentation.Slides[0].Shapes;
```

**4. أضف شكلًا بيضاويًا**
إضافة قطع ناقص إلى المجموعة:
```csharp
IAutoShape ellipse = shapes.AddAutoShape(ShapeType.Ellipse, 50, 150, 150, 100);
```

**5. أضف شكل مستطيل**
وبالمثل، أضف شكل المستطيل.
```csharp
IAutoShape rectangle = shapes.AddAutoShape(ShapeType.Rectangle, 250, 350, 200, 150);
```

**6. احفظ العرض التقديمي**
احفظ العرض التقديمي الخاص بك لإكمال التغييرات.
```csharp
presentation.Save(dataDir, SaveFormat.Pptx);
```

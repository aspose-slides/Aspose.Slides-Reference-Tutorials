## دليل التنفيذ (H2)

سنقوم بتقسيم عملية إزالة شكل من شريحة إلى خطوات يمكن التحكم فيها.

### نظرة عامة على الميزة

يوضح هذا الدليل كيفية إزالة شكل برمجيًا من شريحة PowerPoint باستخدام Aspose.Slides لـ .NET. سنضيف شكلين إلى الشريحة، ثم نزيل أحدهما بناءً على نصه البديل، موضحين كيفية إدارة شرائحك ديناميكيًا.

### التنفيذ خطوة بخطوة (H3)

#### 1. إنشاء عرض تقديمي جديد

ابدأ بإنشاء جديد `Presentation` الكائن الذي يمثل ملف PowerPoint.

```csharp
Presentation pres = new Presentation();
```

يؤدي هذا إلى تهيئة عرض تقديمي فارغ للعمل عليه.

#### 2. الوصول إلى الشريحة الأولى

استرجاع الشريحة الأولى من العرض التقديمي لإضافة الأشكال وإجراء العمليات:

```csharp
ISlide sld = pres.Slides[0];
```

#### 3. إضافة الأشكال إلى الشريحة (H3)

أضف شكلين، مستطيل وشكل القمر، لأغراض التوضيح.

```csharp
IShape shp1 = sld.Shapes.AddAutoShape(ShapeType.Rectangle, 50, 40, 150, 50);
IShape shp2 = sld.Shapes.AddAutoShape(ShapeType.Moon, 160, 40, 150, 50);
```

#### 4. تعيين نص بديل (H3)

قم بتعيين نص بديل للشكل الأول لسهولة التعرف عليه لاحقًا.

```csharp
shp1.AlternativeText = "User Defined";
```

#### 5. تحديد الشكل وإزالته (H3)

قم بالتنقل بين الأشكال الموجودة على الشريحة ثم قم بإزالة الشكل الذي يحتوي على النص البديل المطابق:

```csharp
int iCount = sld.Shapes.Count;
for (int i = 0; i < iCount; i++)
{
    AutoShape ashp = (AutoShape)sld.Shapes[i]; // تم تصحيح الفهرسة لتكرار الحلقة.
    if (String.Compare(ashp.AlternativeText, "User Defined", StringComparison.Ordinal) == 0)
    {
        sld.Shapes.Remove(ashp);
    }
}
```

**لماذا يعمل هذا:** يعتبر النص البديل بمثابة معرف فريد لضمان استهداف الشكل الصحيح للإزالة.

#### 6. احفظ العرض التقديمي (H3)

وأخيرًا، احفظ العرض التقديمي المحدث على القرص:

```csharp
pres.Save("YOUR_OUTPUT_DIRECTORY/RemoveShape_out.pptx", SaveFormat.Pptx);
```

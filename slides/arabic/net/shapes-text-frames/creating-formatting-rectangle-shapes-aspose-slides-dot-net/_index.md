## دليل التنفيذ
### إنشاء شكل مستطيل
هدفنا هو إنشاء شكل مستطيل في الشريحة الأولى. لنشرح الخطوات بالتفصيل:
#### الخطوة 1: تهيئة العرض التقديمي
ابدأ بإعداد بيئتك باستخدام Aspose.Slides وإنشاء كائن عرض تقديمي جديد.
```csharp
using System;
using Aspose.Slides;
using System.Drawing;

string dataDir = "YOUR_DOCUMENT_DIRECTORY";
bool isExists = System.IO.Directory.Exists(dataDir);
if (!isExists)
    System.IO.Directory.CreateDirectory(dataDir);

using (Presentation pres = new Presentation())
{
    // يستمر الكود...
}
```
*توضيح:* يقوم هذا الكود بتهيئة عرض تقديمي جديد لبرنامج PowerPoint ويضمن وجود الدليل لحفظ الملفات.
#### الخطوة 2: الوصول إلى الشريحة الأولى
انتقل إلى الشريحة الأولى حيث سنضيف المستطيل الخاص بنا.
```csharp
ISlide sld = pres.Slides[0];
```
*توضيح:* نحن نسترد الشريحة الأولى من العرض التقديمي للعمل عليها.
#### الخطوة 3: إضافة شكل مستطيل
أضف شكلًا تلقائيًا من نوع المستطيل إلى الشريحة.
```csharp
IShape shp = sld.Shapes.AddAutoShape(ShapeType.Rectangle, 50, 150, 150, 50);
```
*توضيح:* يُنشئ هذا مستطيلاً في الموضع (٥٠، ١٥٠) بأبعاد ١٥٠×٥٠. تُحدد المعلمات نوع الشكل وموقعه/حجمه.
### تنسيق المستطيل
الآن بعد أن أصبح لدينا المستطيل، فلنطبق عليه بعض التصميمات.
#### الخطوة 4: تطبيق لون التعبئة الصلبة
قم بتعيين لون تعبئة ثابت لجسم المستطيل.
```csharp
shp.FillFormat.FillType = FillType.Solid;
shp.FillFormat.SolidFillColor.Color = Color.Chocolate;
```
*توضيح:* هنا، نقوم بتغيير الجزء الداخلي للمستطيل إلى اللون البني الشوكولاتي.
#### الخطوة 5: تطبيق تنسيق خط الحدود
قم بتخصيص الحدود باستخدام التعبئة الصلبة وضبط عرضها.
```csharp
shp.LineFormat.FillFormat.FillType = FillType.Solid;
shp.LineFormat.FillFormat.SolidFillColor.Color = Color.Black;
shp.LineFormat.Width = 5;
```
*توضيح:* تم تعيين حدود المستطيل إلى اللون الأسود، مع عرض خط يبلغ 5 بكسل.
### حفظ العرض التقديمي
وأخيرًا، احفظ التغييرات في الملف.
```csharp
pres.Save(dataDir + "/RectShp2_out.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
```
*توضيح:* يؤدي هذا إلى حفظ العرض التقديمي باستخدام شكل المستطيل المنسق حديثًا في الدليل المحدد.

## دليل التنفيذ

### إنشاء النص ومحاذاته في جداول PowerPoint

#### ملخص
سوف يرشدك هذا القسم خلال إنشاء جدول داخل شريحة PowerPoint ومحاذاة النص داخل خلاياه باستخدام Aspose.Slides لـ .NET.

#### الخطوة 1: تهيئة كائن العرض التقديمي
إنشاء مثيل لـ `Presentation` صف لتمثيل العرض التقديمي بأكمله.
```csharp
using Aspose.Slides;
// إنشاء عرض تقديمي جديد
Presentation presentation = new Presentation();
```

#### الخطوة 2: الوصول إلى الشريحة وتحديد أبعاد الجدول
انتقل إلى الشريحة الأولى من العرض التقديمي، حيث سنضيف جدولنا. حدّد عرض الأعمدة وارتفاع الصفوف حسب الحاجة.
```csharp
// احصل على الشريحة الأولى
ISlide slide = presentation.Slides[0];

// تحديد أبعاد الأعمدة والصفوف
double[] dblCols = { 120, 120, 120, 120 };
double[] dblRows = { 100, 100, 100, 100 };
```

#### الخطوة 3: إضافة جدول إلى الشريحة
أضف جدولاً في الموضع المحدد على الشريحة. هذا المثال يضعه عند الإحداثيات (100، 50).
```csharp
// إضافة شكل الجدول إلى الشريحة
ITable tbl = slide.Shapes.AddTable(100, 50, dblCols, dblRows);
```

#### الخطوة 4: ملء خلايا الجدول وتصميمها
املأ الخلايا بالنص. نوضح هنا كيفية ضبط لون خلفية جزء من النص ضمن فقرة.
```csharp
// تعيين النص في خلايا جدول محددة
tbl[1, 0].TextFrame.Text = "10";
tbl[2, 0].TextFrame.Text = "20";
tbl[3, 0].TextFrame.Text = "30";

// تخصيص مظهر نص الخلية الأولى
ITextFrame txtFrame = tbl[0, 0].TextFrame;
IParagraph paragraph = txtFrame.Paragraphs[0];
IPortion portion = paragraph.Portions[0];

portion.Text = "Text here";
portion.PortionFormat.FillFormat.FillType = FillType.Solid;
portion.PortionFormat.FillFormat.SolidFillColor.Color = Color.Black;
```

#### الخطوة 5: محاذاة النص في الخلايا
عيّن خصائص محاذاة النص للخلية المطلوبة. هنا، نقوم بتوسيط النص أفقيًا وتدويره رأسيًا.
```csharp
// ضبط محاذاة النص الأفقية والرأسية
ICell cell = tbl[0, 0];
cell.TextAnchorType = TextAnchorType.Center;
cell.TextVerticalType = TextVerticalType.Vertical270;
```

#### الخطوة 6: احفظ العرض التقديمي الخاص بك
بمجرد إعداد الجدول الخاص بك باستخدام نص محاذي، احفظ العرض التقديمي في الدليل المحدد.
```csharp
// حفظ العرض التقديمي المحدث
presentation.Save("YOUR_OUTPUT_DIRECTORY/Vertical_Align_Text_out.pptx", SaveFormat.Pptx);
```

### نصائح استكشاف الأخطاء وإصلاحها
- **ملف DLL مفقود Aspose.Slides**:تأكد من تثبيت الحزمة بشكل صحيح عبر NuGet وتضمينها `using Aspose.Slides;` في الكود الخاص بك.
- **النص لا يظهر محاذيًا**:تحقق مرة أخرى من إعدادات المحاذاة الخاصة بك (`TextAnchorType` و `TextVerticalType`) لكل خلية.

1. **كيف أقوم بمحاذاة النص أفقيًا وعموديًا؟**
   يستخدم `TextAnchorType.Center` للتمركز الأفقي و `TextVerticalType.Vertical270` للوضع الرأسي.

2. **هل يمكن لـ Aspose.Slides معالجة العروض التقديمية الموجودة؟**
   نعم، يمكنك تحميل عرض تقديمي موجود وتعديله حسب الحاجة.

3. **ما هي الفوائد الرئيسية لاستخدام Aspose.Slides مقارنة بمعالجة PowerPoint الأصلية؟**
   يوفر Aspose.Slides التحكم البرمجي، مما يجعل من الأسهل أتمتة المهام المتكررة والتكامل مع الأنظمة الأخرى.

4. **هل هناك فرق في الأداء بين طرق محاذاة النص في Aspose.Slides؟**
   تم تحسين محاذاة النص داخل المكتبة؛ ومع ذلك، يجب عليك دائمًا اختبار حالات الاستخدام المحددة لديك لضمان الكفاءة.

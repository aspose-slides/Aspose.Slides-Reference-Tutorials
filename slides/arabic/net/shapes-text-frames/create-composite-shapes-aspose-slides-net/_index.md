## دليل التنفيذ
سنقوم بتقسيم هذا البرنامج التعليمي إلى أقسام، يركز كل منها على ميزة محددة لإنشاء الأشكال المركبة.
### إنشاء أشكال مركبة من مسارات الهندسة
#### ملخص
يوضح هذا القسم كيفية إنشاء شكل مخصص بدمج مسارين هندسيين. هذه التقنية مفيدة لتصميم عناصر شرائح أو شعارات معقدة.
#### الخطوة 1: تحديد مسار ملف الإخراج
أولاً، قم بتعيين مسار ملف الإخراج باستخدام بنية الدليل الخاصة بك:
```csharp
string resultPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "CompositeShape.pptx");
```
#### الخطوة 2: تهيئة كائن العرض التقديمي
ابدأ بإنشاء كائن عرض حيث ستصمم الشكل المركب الخاص بك:
```csharp
using (Presentation pres = new Presentation())
{
    // يستمر التنفيذ...
}
```
#### الخطوة 3: إنشاء مسارات الهندسة
قم بتعريف مسارين هندسيين على النحو التالي:
```csharp
// حدد المسار الأول
IAutoShape shape1 = pres.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 50, 150, 200, 100);
shape1.FillFormat.FillType = FillType.NoFill;

// تحديد المسار الثاني (على سبيل المثال، القطع الناقص)
IAutoShape shape2 = pres.Slides[0].Shapes.AddAutoShape(ShapeType.Ellipse, 300, 150, 200, 100);
shape2.FillFormat.FillType = FillType.Solid;
shape2.FillFormat.SolidFillColor.Color = Color.Blue;
```
#### الخطوة 4: دمج المسارات في شكل مركب
استخدم `Combine` الطريقة لدمج هذه المسارات:
```csharp
// مجموعة مسارات الوصول للشكل 1
IGeometryShape geoShape1 = (GeometryShape)shape1.Shape;
IPathCollection pathCollection1 = geoShape1.Path;

// مجموعة مسارات الوصول للشكل 2
IGeometryShape geoShape2 = (GeometryShape)shape2.Shape;
IPathCollection pathCollection2 = geoShape2.Path;

// دمج المسارات في مسار واحد
pathCollection1.Add(pathCollection2[0]);
```
#### الخطوة 5: حفظ العرض التقديمي
وأخيرًا، احفظ عرضك التقديمي في ملف:
```csharp
pres.Save(resultPath, Aspose.Slides.Export.SaveFormat.Pptx);
```

## دليل التنفيذ

### تحويل SVG إلى مجموعة الأشكال (H2)
في هذا القسم، سنستعرض الخطوات اللازمة لتحويل صورة SVG إلى مجموعة من الأشكال.

#### ملخص
تتيح لك هذه الميزة تحويل صور SVG المضمنة في شريحة PowerPoint إلى عناصر شكلية سهلة الاستخدام. يُسهّل هذا التحويل تعديل وتخصيص الرسومات في عرضك التقديمي.

#### التنفيذ خطوة بخطوة (H3)
1. **تحميل العرض التقديمي الخاص بك**
   ابدأ بتحميل العرض التقديمي الذي يحتوي على صورة SVG:
   ```csharp
   string dataDir = @"YOUR_DOCUMENT_DIRECTORY";
   using (Presentation pres = new Presentation(dataDir + "image.pptx")) {
       // يستمر الكود...
   }
   ```
2. **الوصول إلى صورة SVG**
   قم بتحديد PictureFrame الذي يحتوي على صورة SVG الخاصة بك وقم بالوصول إليه:
   ```csharp
   PictureFrame pFrame = pres.Slides[0].Shapes[0] as PictureFrame;
   ISvgImage svgImage = pFrame.PictureFormat.Picture.Image.SvgImage;

   if (svgImage != null) {
       // متابعة التحويل...
   }
   ```
3. **تحويل SVG وتحديد موقعه**
   تحويل SVG إلى مجموعة من الأشكال، ووضعها في موقع الإطار الأصلي:
   ```csharp
   IGroupShape groupShape = pres.Slides[0].Shapes.AddGroupShape(
       svgImage,
       pFrame.Frame.X,
       pFrame.Frame.Y,
       pFrame.Frame.Width,
       pFrame.Frame.Height);
   ```
4. **إزالة صورة SVG الأصلية**
   قم بإزالة PictureFrame الأصلي لتنظيف الشريحة الخاصة بك:
   ```csharp
   pres.Slides[0].Shapes.Remove(pFrame);
   ```
5. **احفظ عرضك التقديمي**
   أخيرًا، احفظ العرض التقديمي المعدّل باستخدام مجموعة الأشكال التي تم إنشاؤها حديثًا:
   ```csharp
   pres.Save(dataDir + "image_group.pptx");
   ```

#### نصائح استكشاف الأخطاء وإصلاحها
- تأكد من تضمين صورة SVG الخاصة بك بشكل صحيح في PictureFrame.
- تحقق من مسارات الملفات وتأكد من أنها تشير إلى الدلائل الصحيحة.

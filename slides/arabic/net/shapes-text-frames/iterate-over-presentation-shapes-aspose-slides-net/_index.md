## دليل التنفيذ

دعنا نقسم العملية إلى خطوات واضحة للتكرار على الأشكال وتحديد مربعات النص.

### الميزة: التكرار على أشكال العرض التقديمي

تُركّز هذه الميزة على تكرار جميع الأشكال الموجودة في الشريحة، والتحقق مما إذا كان كلٌّ منها مربع نص. إليك كيفية تنفيذها:

#### الخطوة 1: تحميل العرض التقديمي الخاص بك

أولاً، تأكد من تعيين مسار ملف العرض التقديمي الخاص بك بشكل صحيح:

```csharp
string presentationPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "CheckTextShapes.pptx");
```

افتح العرض التقديمي باستخدام Aspose.Slides:

```csharp
using (Presentation presentation = new Presentation(presentationPath))
{
    // سيتم وضع الكود للتكرار على الأشكال هنا
}
```

#### الخطوة 2: التكرار على الأشكال

تنقّل بين كل شكل في شريحة محددة. في هذا المثال، ننظر إلى الشريحة الأولى:

```csharp
foreach (IShape shape in presentation.Slides[0].Shapes)
{
    // التحقق مما إذا كان الشكل عبارة عن شكل تلقائي وتحديد ما إذا كان مربع نص
}
```

#### الخطوة 3: تحديد مربعات النص

تحقق مما إذا كان كل شكل هو `AutoShape` ثم تحقق مما إذا كان يحتوي على نص:

```csharp
if (shape is AutoShape autoShape)
{
    bool isTextBox = autoShape.IsTextBox;
    // استخدم 'isTextBox' لتحديد ما إذا كان الشكل عبارة عن مربع نص.
}
```

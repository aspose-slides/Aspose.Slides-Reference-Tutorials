# إنشاء الدليل تلقائيًا وإضافة شكل القطع الناقص في PowerPoint باستخدام Aspose.Slides لـ .NET

## مقدمة

إن أتمتة عملية إنشاء الدليل وإضافة أشكال مثل النقاط البيضاوية إلى عروض PowerPoint التقديمية من شأنها أن تُبسّط سير عملك بشكل كبير. سيرشدك هذا البرنامج التعليمي إلى كيفية استخدام Aspose.Slides لـ .NET، وهي مكتبة فعّالة تُبسّط هذه المهام.

### ما سوف تتعلمه:
- التحقق من وجود الدليل ثم إنشائه إذا لزم الأمر.
- إضافة الأشكال وتنسيقها في عروض PowerPoint التقديمية.
- تكوين عناصر العرض التقديمي بشكل فعال.

## المتطلبات الأساسية

لمتابعة هذا البرنامج التعليمي، تحتاج إلى الإعداد التالي:

### المكتبات المطلوبة:
- **Aspose.Slides لـ .NET**:ضروري لإنشاء العروض التقديمية في PowerPoint والتلاعب بها.
- **مساحة اسم System.IO**:تستخدم لعمليات الدليل في C#.

### إعداد البيئة:
- Visual Studio أو IDE متوافق يدعم تطوير .NET.
- فهم أساسي لمفاهيم برمجة C#.

## إعداد Aspose.Slides لـ .NET

قم بتثبيت المكتبة باستخدام إحدى الطرق التالية:

**.NET CLI:**
```bash
dotnet add package Aspose.Slides
```

**مدير الحزمة:**

**واجهة مستخدم مدير حزمة NuGet:**
ابحث عن "Aspose.Slides" وقم بتثبيت الإصدار الأحدث عبر IDE الخاص بك.

#### التهيئة الأساسية:

## دليل التنفيذ

يغطي هذا الدليل ميزتين رئيسيتين: إنشاء دليل وإضافة شكل بيضاوي.

### الميزة 1: إنشاء الدليل إذا لم يكن موجودًا

#### ملخص:
تحقق من وجود دليل محدد، وأنشئه إن لم يكن موجودًا. هذا مفيد لتنظيم الملفات بشكل منهجي.

**الخطوة 1: التحقق من وجود الدليل**
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
bool isExists = Directory.Exists(dataDir);
```
- `dataDir`:المسار الذي تريد التحقق منه أو إنشاء الدليل فيه.
- `Directory.Exists()`:ترجع قيمة منطقية تشير إلى ما إذا كان الدليل المحدد موجودًا.

**الخطوة 2: إنشاء الدليل**
```csharp
if (!isExists)
    Directory.CreateDirectory(dataDir);
```
- يستخدم `Directory.CreateDirectory()` إذا لم يكن الدليل موجودًا لتجنب الأخطاء عند حفظ الملفات.

### الميزة 2: إضافة شكل تلقائي من نوع القطع الناقص

#### ملخص:
قم بتعزيز عروضك التقديمية عن طريق إضافة أشكال مثل القطع الناقص.

**الخطوة 1: تهيئة العرض التقديمي**
```csharp
using (Presentation pres = new Presentation())
{
    ISlide sld = pres.Slides[0];
```
- ابدأ عرضًا تقديميًا جديدًا وقم بالوصول إلى الشريحة الأولى لإضافة الأشكال.

**الخطوة 2: إضافة شكل القطع الناقص**
```csharp
IShape shp = sld.Shapes.AddAutoShape(ShapeType.Ellipse, 50, 150, 150, 50);
```
- `AddAutoShape()`:يضيف شكلًا بيضاويًا في الموضع المحدد بعرض وارتفاع محددين.

**الخطوة 3: تنسيق الشكل**
```csharp
// لون التعبئة
shp.FillFormat.FillType = FillType.Solid;
shp.FillFormat.SolidFillColor.Color = System.Drawing.Color.Chocolate;

// تنسيق الحدود
shp.LineFormat.FillFormat.FillType = FillType.Solid;
shp.LineFormat.FillFormat.SolidFillColor.Color = System.Drawing.Color.Black;
shp.LineFormat.Width = 5;
```
- تخصيص لون التعبئة لـ `Chocolate` وضبط حدود سوداء صلبة بعرض 5.

**الخطوة 4: حفظ العرض التقديمي**
```csharp
pres.Save(outputDir + "EllipseShp2_out.pptx", SaveFormat.Pptx);
```
- احفظ العرض التقديمي الخاص بك بتنسيق PPTX في دليل الإخراج المحدد. 

### نصائح استكشاف الأخطاء وإصلاحها:
- يضمن `dataDir` تم ضبطها بشكل صحيح ويمكن الوصول إليها.
- تحقق من تثبيت Aspose.Slides إذا واجهت أخطاء متعلقة بالمكتبة.

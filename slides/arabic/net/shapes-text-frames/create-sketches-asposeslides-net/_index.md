## دليل التنفيذ

بعد إعداد كل شيء، دعنا ننفذ ميزة الشكل المرسوم.

### إضافة الأشكال وتعديلها

#### ملخص

في هذا القسم، سنضيف شكلًا تلقائيًا من نوع المستطيل إلى شريحة ونقوم بتكوين خصائصه لإنشاء تأثير رسم تخطيطي.

**إضافة شكل مستطيل**

ابدأ بإنشاء مثيل عرض تقديمي جديد وإضافة شكل مستطيل:
```csharp
using Aspose.Slides;
using Aspose.Slides.Export;

string outPptxFile = Path.Combine("YOUR_OUTPUT_DIRECTORY", "SketchedShapes_out.pptx");
string outPngFile = Path.Combine("YOUR_OUTPUT_DIRECTORY", "SketchedShapes_out.png");

using (Presentation pres = new Presentation())
{
    // أضف شكلًا تلقائيًا من نوع المستطيل إلى الشريحة الأولى
    IAutoShape shape = pres.Slides[0].Shapes.AddAutoShape(ShapeType.Rectangle, 20, 20, 300, 150);
}
```

#### ضبط تنسيق التعبئة

لإضفاء مظهر تخطيطي عليه، قم بإزالة أي تعبئة من الشكل:
```csharp
shape.FillFormat.FillType = FillType.NoFill;
```

### تطبيق تأثيرات الرسم على الأشكال

#### ملخص

بعد ذلك، قم بتحويل المستطيل إلى رسم تخطيطي يدوي.

**تحويل الشكل إلى رسم تخطيطي**

استخدم `SketchFormat` الخاصية لتطبيق تأثير الخربشة:
```csharp
// تحويل الشكل إلى رسم تخطيطي بأسلوب الرسم الحر (Scribble)
shape.LineFormat.SketchFormat.SketchType = LineSketchType.Scribble;
```

### حفظ العروض التقديمية والصور

وأخيرًا، احفظ عملك كملف عرض تقديمي وصورة.

**الحفظ كـ PPTX**
```csharp
// حفظ العرض التقديمي في ملف PPTX
pres.Save(outPptxFile, SaveFormat.Pptx);
```

**حفظ كصورة PNG**
```csharp
// احفظ الشريحة كملف صورة بتنسيق PNG
pres.Slides[0].GetThumbnail(4/3f, 4/3f).Save(outPngFile, System.Drawing.Imaging.ImageFormat.Png);
```

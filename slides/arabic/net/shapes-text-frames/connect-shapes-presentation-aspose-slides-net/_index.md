# كيفية ربط الأشكال باستخدام الموصلات في PowerPoint باستخدام Aspose.Slides لـ .NET

## مقدمة

تحسين عروض PowerPoint التقديمية بربط أشكال مثل القطع الناقص والمستطيلات باستخدام موصلات أمر سهل مع Aspose.Slides لـ .NET. يرشدك هذا البرنامج التعليمي إلى كيفية ربط شكلين أساسيين بسلاسة.

**ما سوف تتعلمه:**
- إعداد Aspose.Slides لـ .NET
- إضافة الأشكال إلى الشريحة
- ربط الأشكال بالموصلات
- حفظ العرض التقديمي المحسّن

لنبدأ بالتأكد من أن لديك المتطلبات الأساسية اللازمة.

## المتطلبات الأساسية

قبل التنفيذ، تأكد من أن لديك:
- **المكتبات المطلوبة**:قم بتثبيت الإصدار الأحدث من Aspose.Slides لـ .NET.
- **إعداد البيئة**:استخدم بيئة تطوير تدعم C#، مثل Visual Studio.
- **متطلبات المعرفة**:سيكون من المفيد أن يكون لديك فهم أساسي للغة C# والمعرفة بعروض PowerPoint.

## إعداد Aspose.Slides لـ .NET

للبدء، قم بتثبيت مكتبة Aspose.Slides باستخدام أحد مديري الحزم التاليين:

**.NET CLI**
```shell
dotnet add package Aspose.Slides
```

**وحدة تحكم مدير الحزم**
```powershell
Install-Package Aspose.Slides
```

**واجهة مستخدم مدير الحزم NuGet**:ابحث عن "Aspose.Slides" وقم بتثبيت الإصدار الأحدث.

### الحصول على الترخيص
- **نسخة تجريبية مجانية**:ابدأ بإصدار تجريبي مجاني لاستكشاف الوظائف الأساسية.
- **رخصة مؤقتة**:تقدم بطلب للحصول على ترخيص مؤقت للوصول إلى الميزات الكاملة دون قيود.
- **شراء**:فكر في شراء ترخيص اشتراك للاستخدام المستمر.

بعد التثبيت، ابدأ مشروعك بإنشاء مثيل لفئة العرض التقديمي. هنا ستبدأ بإضافة الأشكال والموصلات.

## دليل التنفيذ

### إضافة الأشكال إلى الشريحة

**ملخص:**
أضف شكلين أساسيين - قطع ناقص ومستطيل - إلى الشريحة الخاصة بنا.

#### الخطوة 1: الوصول إلى مجموعة الأشكال
أولاً، قم بالوصول إلى مجموعة الأشكال الخاصة بالشريحة المطلوبة:
```csharp
IShapeCollection shapes = input.Slides[0].Shapes;
```

#### الخطوة 2: إضافة القطع الناقص
قم بإنشاء شكل بيضاوي في الموضع (x=0، y=100) بعرض وارتفاع 100.
```csharp
IAutoShape ellipse = shapes.AddAutoShape(ShapeType.Ellipse, 0, 100, 100, 100);
```

#### الخطوة 3: إضافة مستطيل
بعد ذلك، أضف مستطيلاً في الموضع (x=100، y=300) بنفس الأبعاد:
```csharp
IAutoShape rectangle = shapes.AddAutoShape(ShapeType.Rectangle, 100, 300, 100, 100);
```

### ربط الأشكال باستخدام الموصلات

**ملخص:**
الآن بعد أن أصبح لدينا الأشكال في مكانها، فلنقم بتوصيلها باستخدام موصل.

#### الخطوة 4: إضافة موصل
أضف موصلًا منحنيًا إلى الشريحة الخاصة بك:
```csharp
IConnector connector = shapes.AddConnector(ShapeType.BentConnector2, 0, 0, 10, 10);
```

#### الخطوة 5: ربط الأشكال
إنشاء اتصالات بين القطع الناقص والمستطيل باستخدام الموصل.
```csharp
connector.StartShapeConnectedTo = ellipse;
connector.EndShapeConnectedTo = rectangle;
```

#### الخطوة 6: تحسين مسار الموصل
يستخدم `Reroute` للعثور تلقائيًا على أقصر مسار للموصل:
```csharp
connector.Reroute();
```

### حفظ العرض التقديمي الخاص بك

وأخيرًا، احفظ العرض التقديمي الخاص بك بتنسيق PPTX.
```csharp
input.Save(dataDir + "Connecting shapes using connectors_out.pptx", SaveFormat.Pptx);
```

**نصائح استكشاف الأخطاء وإصلاحها**: 
- تأكد من `dataDir` يشير المتغير بشكل صحيح إلى الدليل المطلوب.
- تحقق من معرفات الشكل والمواضع الصحيحة إذا لم تظهر الاتصالات.

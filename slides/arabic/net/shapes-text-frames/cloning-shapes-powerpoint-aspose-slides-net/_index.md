## دليل التنفيذ

الآن، لنبدأ باستنساخ هذه الأشكال! سنشرح كل جزء من العملية للتوضيح.

### استنساخ الأشكال بين الشرائح

#### ملخص
تتيح لك هذه الميزة تكرار أشكال محددة من شريحة واحدة ووضعها على شريحة أخرى، إما في إحداثيات محددة أو في وضع افتراضي.

#### التنفيذ خطوة بخطوة

**إعداد العرض التقديمي الخاص بك**

ابدأ بتحديد مسار المستند الخاص بك وتحميل العرض التقديمي الخاص بك:

```csharp
string dataDir = @"YOUR_DOCUMENT_DIRECTORY";
using (Presentation srcPres = new Presentation(dataDir + "Source Frame.pptx"))
{
    // المضي قدمًا في عمليات الاستنساخ
}
```

**الوصول إلى مجموعات الأشكال**

استرداد مجموعات الأشكال من شرائح المصدر والوجهة:

```csharp
// احصل على مجموعة الأشكال من الشريحة الأولى
IShapeCollection sourceShapes = srcPres.Slides[0].Shapes;

// احصل على شريحة تخطيط فارغة لإنشاء شريحة جديدة بدون محتوى
ILayoutSlide blankLayout = srcPres.Masters[0].LayoutSlides.GetByType(SlideLayoutType.Blank);

// أضف شريحة فارغة باستخدام التخطيط الفارغ
ISlide destSlide = srcPres.Slides.AddEmptySlide(blankLayout);
IShapeCollection destShapes = destSlide.Shapes;
```

**استنساخ الأشكال باستخدام إحداثيات محددة**

استنساخ شكل محدد ووضعه في الإحداثيات المطلوبة على الشريحة الوجهة:

```csharp
// استنساخ شكل إلى إحداثيات محددة على الشريحة الوجهة
destShapes.AddClone(sourceShapes[1], 50, 150 + sourceShapes[0].Height);
```

**استنساخ الشكل بدون موضع جديد**

يمكنك أيضًا استنساخ الأشكال دون تحديد إحداثيات جديدة. ستتم إضافتها بالتتابع:

```csharp
// استنساخ شكل آخر إلى الموضع الافتراضي على الشريحة الوجهة
destShapes.AddClone(sourceShapes[2]);
```

**إدراج الشكل المستنسخ في مؤشر محدد**

إدراج شكل مستنسخ في بداية مجموعة الأشكال الخاصة بالشريحة الوجهة:

```csharp
// إدراج الشكل المستنسخ عند الفهرس 0 مع الإحداثيات المحددة
destShapes.InsertClone(0, sourceShapes[0], 50, 150);
```

### حفظ العرض التقديمي الخاص بك

وأخيرًا، احفظ العرض التقديمي المعدّل على القرص:

```csharp
srcPres.Save(dataDir + "CloneShape_out.pptx", SaveFormat.Pptx);
```

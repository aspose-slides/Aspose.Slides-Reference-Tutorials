### إضافة أجزاء إلى الأشكال الهندسية

#### ملخص
قم بتخصيص الأشكال الهندسية عن طريق إضافة أجزاء خطية إضافية، وهو أمر بالغ الأهمية لإنشاء تصميمات أو مخططات معقدة في العروض التقديمية.

#### التنفيذ خطوة بخطوة

**1. تحميل العرض التقديمي**
```csharp
using Aspose.Slides;
using System.IO;
// تحديد مسار الإخراج
string resultPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "modified_presentation.pptx");
// فتح عرض تقديمي موجود
Presentation pres = new Presentation("your_input_file.pptx");
```
**2. الوصول إلى الشريحة والشكل**
```csharp
// احصل على الشريحة الأولى
ISlide slide = pres.Slides[0];
// بافتراض وجود شكل واحد على الأقل، احصل على الشكل الأول
IAutoShape shape = (IAutoShape)slide.Shapes[0];
```
**3. تعديل الشكل الهندسي**
```csharp
if (shape.ShapeType == Aspose.Slides.ShapeType.Custom)
{
    // الوصول إلى بيانات الهندسة وتعديلها
    var customGeometry = (Aspose.Slides.Geometry.CustomShapeGeometry)shape.GeometryShape;
    
    // إضافة جزء جديد إلى الشكل
    int index = customGeometry.Path.AddLine(new float[] { 0f, 50f, 100f });
    
    // تكوين خصائص المقطع الجديد إذا لزم الأمر
}
```
**4. حفظ التغييرات**
```csharp
// حفظ العرض التقديمي المعدل
pres.Save(resultPath, Aspose.Slides.Export.SaveFormat.Pptx);
```
### نصائح استكشاف الأخطاء وإصلاحها
- **تأكد من نوع الشكل**:تأكد من أن الشكل الخاص بك من النوع `Custom` لتعديل هندسته.
- **الفهرس خارج النطاق**:تأكد من وصولك إلى المؤشرات الصالحة عند تعديل أجزاء المسار.

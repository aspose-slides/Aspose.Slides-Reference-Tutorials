## دليل التنفيذ
لنُقسّم العملية إلى خطوات سهلة. سيرشدك هذا القسم خلال إنشاء MathShape وإضافة MathBlock.

### إنشاء شكل رياضي على شريحة PowerPoint
#### ملخص
سنبدأ بإعداد عرض تقديمي جديد، والوصول إلى الشريحة الأولى، ثم إضافة MathShape إليها.

#### خطوات:
**الخطوة 1: تهيئة العرض التقديمي**
ابدأ بإنشاء مثيل جديد لـ `Presentation` يمثل هذا ملف PowerPoint بأكمله.

```csharp
using (var presentation = new Presentation())
{
    // سيتم وضع الكود لإنشاء الأشكال هنا
}
```

**لماذا**:يؤدي هذا إلى إنشاء بيئة يمكنك من خلالها التعامل مع الشرائح برمجيًا.

#### الخطوة 2: إضافة MathShape إلى الشريحة
الآن، دعنا نضيف MathShape في موضع محدد على الشريحة.

```csharp
ISlide slide = presentation.Slides[0];
IAutoShape mathShape = slide.Shapes.AddMathShape(10, 10, 500, 500);
```

**لماذا**:تضع هذه الخطوة حاوية رياضية على الشريحة حيث يمكنك لاحقًا إضافة المعادلات أو التعبيرات.

### إضافة MathBlock
#### ملخص
بعد ذلك، سنركز على ملء MathShape بمحتوى رياضي فعلي باستخدام MathBlock.

#### خطوات:
**الخطوة 3: الوصول إلى MathParagraph**
استرجاع `IMathParagraph` كائن من MathShape لإدراج نص رياضي.

```csharp
IMathParagraph mathParagraph = (mathShape.TextFrame.Paragraphs[0].Portions[0] as MathPortion).MathParagraph;
```

**لماذا**:يسمح لك هذا بالتلاعب بالفقرة التي ستحتوي على معادلاتك.

**الخطوة 4: إنشاء وإضافة MathBlock**
إنشاء جديد `MathBlock` مع مثال لتعبير رياضي وإضافته إلى MathParagraph.

```csharp
IMathBlock mathBlock = new MathBlock(new MathematicalText("F").Join(".")
    .Join(new MathematicalText("1").Divide("y")).Underbar());
mathParagraph.Add(mathBlock);
```

**لماذا**:تعمل هذه الخطوة على إنشاء تعبير رياضي معقد وتضمينه داخل الشريحة الخاصة بك.

### حفظ العرض التقديمي
وأخيرًا، احفظ عرضك التقديمي في ملف:

```csharp
string outPptxFile = Path.Combine(YOUR_DOCUMENT_DIRECTORY, "MathShape_GetChildren_out.pptx");
presentation.Save(outPptxFile, SaveFormat.Pptx);
```

**لماذا**:يضمن هذا الحفاظ على كافة التغييرات في ملف PowerPoint جديد.

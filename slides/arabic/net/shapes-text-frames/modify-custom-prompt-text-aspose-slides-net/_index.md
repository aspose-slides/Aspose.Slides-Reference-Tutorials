## دليل التنفيذ

### الميزة: تغيير نص العنصر النائب المخصص في شرائح PowerPoint
تتيح لك هذه الميزة تخصيص نص العنصر النائب للعناوين والعناوين الفرعية والعناصر الأخرى، مما يؤدي إلى تحسين مظهر العرض التقديمي الخاص بك.

#### ملخص
سنُعدِّل النص في شرائح PowerPoint مُحدَّدة باستخدام واجهة برمجة التطبيقات القوية لـ Aspose.Slides. يُعدّ هذا مفيدًا بشكل خاص لإنشاء هوية تجارية أو أدلة إرشادية مُتَّسقة ضمن العروض التقديمية.

#### خطوات التنفيذ

##### 1. إعداد كائن العرض التقديمي الخاص بك
ابدأ بتحميل العرض التقديمي الخاص بك إلى `Aspose.Slides.Presentation` هدف:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
using (Presentation pres = new Presentation(dataDir + "/Presentation2.pptx")) {
    ISlide slide = pres.Slides[0];
}
```

##### 2. تكرار أشكال الشرائح
قم بالمرور على كل شكل على الشريحة للعثور على العناصر النائبة:
```csharp
foreach (IShape shape in slide.Slide.Shapes) {
    if (shape.Placeholder != null && shape is AutoShape) {
        // معالجة الكود هنا
    }
}
```
*لماذا هذه الخطوة؟* نحن بحاجة إلى تحديد الأشكال التي تعتبر بمثابة عناصر نائبة حتى نتمكن من تعديل نصها.

##### 3. تعديل نص العنصر النائب
تحديد نوع العنصر النائب وتعيين النص المخصص الخاص بك:
```csharp
string text = "";
if (shape.Placeholder.Type == PlaceholderType.CenteredTitle) {
    text = "Click to add a custom title";
} else if (shape.Placeholder.Type == PlaceholderType.Subtitle) {
    text = "Click to add a custom subtitle";
}
((IAutoShape) shape).TextFrame.Text = text;
```
*لماذا يجب التحقق من نوع العنصر النائب؟* تخدم العناصر النائبة المختلفة أغراضًا مختلفة، لذا نقوم بتخصيص المطالبة وفقًا لذلك.

##### 4. احفظ عرضك التقديمي
بعد التعديلات، احفظ العرض التقديمي الخاص بك:
```csharp
pres.Save(dataDir + "/Placeholders_PromptText.pptx", SaveFormat.Pptx);
```

# إعادة ترتيب الأشكال في PowerPoint باستخدام Aspose.Slides لـ .NET
## مقدمة
قم بتعزيز عروض PowerPoint الخاصة بك عن طريق إعادة ترتيب الأشكال بشكل ديناميكي باستخدام Aspose.Slides for .NET، وهي مكتبة قوية لإدارة ملفات العرض التقديمي برمجيًا.
**Aspose.Slides لـ .NET** يوفر ميزات فعّالة لأتمتة العروض التقديمية وتحويلها. سيوضح لك هذا الدليل التفصيلي كيفية إعادة ترتيب الأشكال، مثل المستطيلات والمثلثات، داخل الشرائح، لضمان ظهور محتواك بالترتيب المطلوب.
### ما سوف تتعلمه:
- إعداد Aspose.Slides لـ .NET
- إضافة إطارات النص والتلاعب بها في الأشكال
- إعادة ترتيب الأشكال على شريحة PowerPoint
- حفظ العرض التقديمي المعدل
دعونا نستكشف المتطلبات الأساسية قبل تنفيذ إعادة ترتيب الشكل.
## المتطلبات الأساسية
قبل البدء، تأكد من أن لديك:
- **المكتبات المطلوبة:** قم بتثبيت الإصدار الأحدث من Aspose.Slides لـ .NET.
- **إعداد البيئة:** يفترض هذا البرنامج التعليمي معرفة أساسية بلغة C# وبيئة تطوير تدعم تطبيقات .NET (على سبيل المثال، Visual Studio).
- **المتطلبات المعرفية:** إن المعرفة بهياكل شرائح PowerPoint مفيدة ولكنها ليست ضرورية.
## إعداد Aspose.Slides لـ .NET
لاستخدام Aspose.Slides في مشروعك، قم بتثبيت المكتبة باستخدام أحد مديري الحزم التاليين:
**.NET CLI**
```bash
dotnet add package Aspose.Slides
```
**مدير الحزم**
```powershell
Install-Package Aspose.Slides
```
**واجهة مستخدم مدير حزمة NuGet:**
ابحث عن "Aspose.Slides" وقم بتثبيت الإصدار الأحدث.
### الحصول على الترخيص
ابدأ بفترة تجريبية مجانية لتقييم الميزات. للاستخدام المستمر، فكّر في شراء ترخيص أو طلب ترخيص مؤقت لتمديد فترة الوصول أثناء التطوير.
**التهيئة الأساسية:**
```csharp
using Aspose.Slides;
// تهيئة كائن العرض التقديمي
Presentation presentation = new Presentation();
```
## دليل التنفيذ
اتبع الخطوات التالية لإعادة ترتيب الأشكال على شريحة PowerPoint باستخدام Aspose.Slides لـ .NET.
### إضافة الأشكال وإعادة ترتيبها
#### ملخص
قم بضبط ترتيب الأشكال بشكل ديناميكي داخل الشريحة، وهو أمر مفيد للعروض التقديمية التي تتطلب تعديلات التسلسل الهرمي المرئي.
**الخطوة 1: تحميل عرض تقديمي موجود**
قم بتحميل ملف PowerPoint الخاص بك إلى Aspose.Slides:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
// تحميل عرض تقديمي موجود
Presentation presentation1 = new Presentation(dataDir + "HelloWorld.pptx");
```
**الخطوة 2: الوصول إلى الشريحة وإضافة الأشكال**
انتقل إلى الشريحة المطلوبة وأضف شكلًا، مثل مستطيل للنص:
```csharp
ISlide slide = presentation1.Slides[0];
// أضف مستطيلاً بدون تعبئة
IAutoShape shp3 = slide.Shapes.AddAutoShape(ShapeType.Rectangle, 200, 365, 400, 150);
shp3.FillFormat.FillType = FillType.NoFill;
```
**الخطوة 3: إدراج النص في الشكل**
التلاعب بالنص داخل الأشكال:
```csharp
// إضافة إطار نص وتعيين نص العلامة المائية
ITextFrame txtFrame = shp3.TextFrame;
IParagraph para = txtFrame.Paragraphs[0];
IPortion portion = para.Portions[0];
portion.Text = "Watermark Text Watermark Text Watermark Text";
```
**الخطوة 4: إضافة شكل آخر**
أضف شكل مثلث إلى الشريحة:
```csharp
shp3 = slide.Shapes.AddAutoShape(ShapeType.Triangle, 200, 365, 400, 150);
```
**الخطوة 5: إعادة ترتيب الأشكال**
التحكم في ترتيب التكديس المرئي عن طريق إعادة ترتيب الأشكال:
```csharp
// نقل المثلث إلى الفهرس 2 في مجموعة الأشكال
slide.Shapes.Reorder(2, shp3);
```
### حفظ العرض التقديمي
احفظ العرض التقديمي المعدّل:
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
presentation1.Save(outputDir + "Reshape_out.pptx");
```

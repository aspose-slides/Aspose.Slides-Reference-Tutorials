## دليل التنفيذ

دعونا نتعمق في الميزات الأساسية لتعديل الأشكال الهندسية في PowerPoint باستخدام Aspose.Slides.

### إزالة جزء من شكل هندسي

تُركز هذه الميزة على إزالة أجزاء مُحددة من شكل هندسي موجود. وتُعدّ هذه الميزة مفيدةً بشكل خاص عند الحاجة إلى تخصيص أو تبسيط الأشكال المُعقدة.

#### الخطوة 1: تهيئة العرض التقديمي
إنشاء كائن العرض التقديمي الخاص بك وتحميله:

```csharp
using (Presentation pres = new Presentation())
{
    // سيتم وضع الكود الخاص بك هنا
}
```

#### الخطوة 2: إضافة شكل قلب

أضف شكلًا هندسيًا على شكل قلب إلى الشريحة الأولى:

```csharp
GeometryShape shape = (GeometryShape)pres.Slides[0].Shapes.AddAutoShape(ShapeType.Heart, 100, 100, 300, 300);
```
- **حدود**: ال `ShapeType` يحدد نوع الشكل، والأرقام اللاحقة تحدد موضعه وحجمه.

#### الخطوة 3: الوصول إلى مسار الهندسة

استرداد مسار الهندسة للتلاعب به:

```csharp
IGeometryPath path = shape.GetGeometryPaths()[0];
```

#### الخطوة 4: إزالة جزء

قم بإزالة الجزء الثالث (المؤشر 2) من المسار:

```csharp
path.RemoveAt(2);
```
- **توضيح**: ال `RemoveAt` تقوم الطريقة بتعديل الهندسة عن طريق إزالة جزء محدد.

#### الخطوة 5: تحديث الشكل

تطبيق المسار المعدل مرة أخرى على الشكل:

```csharp
shape.SetGeometryPath(path);
```

#### الخطوة 6: احفظ العرض التقديمي الخاص بك

قم بتحديد دليل الإخراج الخاص بك وحفظ العرض التقديمي:

```csharp
string resultPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "GeometryShapeRemoveSegment.pptx");
pres.Save(resultPath, SaveFormat.Pptx);
```

### إضافة الأشكال التلقائية إلى العرض التقديمي

تتيح لك هذه الميزة إثراء شرائحك عن طريق إضافة أشكال تلقائية مختلفة.

#### الخطوة 1: تهيئة العرض التقديمي
ابدأ بعنصر عرض تقديمي جديد:

```csharp
using (Presentation pres = new Presentation())
{
    // سيتم وضع الكود الخاص بك هنا
}
```

#### الخطوة 2: إضافة شكل تلقائي

أضف شكل قلب إلى الشريحة الأولى، على غرار ما سبق:

```csharp
GeometryShape shape = (GeometryShape)pres.Slides[0].Shapes.AddAutoShape(ShapeType.Heart, 100, 100, 300, 300);
```

#### الخطوة 3: احفظ العرض التقديمي الخاص بك

احفظ العرض التقديمي بالأشكال الجديدة:

```csharp
string resultPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "AddAutoShape.pptx");
pres.Save(resultPath, SaveFormat.Pptx);
```

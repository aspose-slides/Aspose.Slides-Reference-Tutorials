### نظرة عامة على الميزة
توضح هذه الميزة كيفية إنشاء شكل مستطيل على شريحة وتعبئته بصورة باستخدام Aspose.Slides. يمكنك استخدام هذه التقنية لتحسين شرائحك بإضافة شعارات أو خلفيات أو أي عناصر رسومية تجعل عرضك التقديمي أكثر جاذبية.

### التنفيذ خطوة بخطوة
#### 1. تهيئة كائن العرض التقديمي
ابدأ بإنشاء كائن عرض تقديمي جديد. سيكون هذا الكائن بمثابة مستند عملنا الذي سنضيف إليه الأشكال والعناصر الأخرى.
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // تعيين مسار دليل المستندات الخاص بك
total slides count: pres.Slides.Count;
using (Presentation pres = new Presentation())
{
    ISlide firstSlide = pres.Slides[0]; // الوصول إلى الشريحة الأولى

    // قم بتحميل صورة لاستخدامها كملء
    IPPImage ppImage;
    using (IImage newImage = Aspose.Slides.Images.FromFile(Path.Combine(dataDir, "image.png")))
        ppImage = pres.Images.AddImage(newImage); // إضافة صورة إلى مجموعة صور العرض التقديمي

    // يضيف شكل مستطيل بأبعاد محددة
    var newShape = firstSlide.Shapes.AddAutoShape(ShapeType.Rectangle, 0, 0, 350, 350);

    // تعيين نوع التعبئة للشكل إلى صورة
    newShape.FillFormat.FillType = FillType.Picture;
    IPictureFillFormat pictureFillFormat = newShape.FillFormat.PictureFillFormat;
    pictureFillFormat.Picture.Image = ppImage; // تعيين الصورة المحملة كملء للمستطيل

    // حفظ العرض التقديمي
    pres.Save(Path.Combine("YOUR_OUTPUT_DIRECTORY", "RectangleWithPictureFill.pptx"), SaveFormat.Pptx);
}
```
#### شرح الخطوات الرئيسية:
- **جاري تحميل الصورة**: ال `FromFile` تقوم الطريقة بتحميل صورة من الدليل المحدد، والتي تتم إضافتها بعد ذلك إلى مجموعة صور العرض التقديمي.
  
- **إضافة شكل مستطيل**:نحن نستخدم `AddAutoShape` مع `ShapeType.Rectangle` وحدد أبعادها. سيؤدي هذا إلى إنشاء مستطيل على الشريحة.

- **إعداد تعبئة الصورة**:بالتعيين `FillType.Picture` إلى تنسيق تعبئة الشكل، نُحوّل المستطيل إلى حاوية صورة. ثم تُضبط الصورة المُحمّلة كملء باستخدام `Picture.Image` ملكية.

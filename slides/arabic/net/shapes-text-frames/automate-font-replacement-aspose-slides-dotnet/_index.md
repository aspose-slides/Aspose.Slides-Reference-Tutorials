## دليل التنفيذ

دعونا نستعرض كيفية تنفيذ استبدال الخط باستخدام Aspose.Slides.

### تحميل عرض PowerPoint

ابدأ بتحميل ملف العرض التقديمي الذي ترغب في تعديله. يتم ذلك باستخدام `Presentation` الفئة التي تمثل مستند PPTX.

```csharp
string sourceFilePath = "YOUR_DOCUMENT_DIRECTORY\\Fonts.pptx";
Presentation presentation = new Presentation(sourceFilePath);
```

### تحديد الخطوط واستبدالها

لاستبدال الخطوط، عليك تحديد الخط المصدر والخط الوجهة. إليك الطريقة:

#### الخطوة 1: تحديد الخط المصدر

حدد الخط الموجود في العرض التقديمي الذي تريد استبداله.

```csharp
IFontData sourceFont = new FontData("Arial");
```

#### الخطوة 2: تحديد خط الوجهة

قم بتحديد الخط الجديد الذي سيحل محل الخط الأصلي.

```csharp
IFontData destFont = new FontData("Times New Roman");
```

#### الخطوة 3: تنفيذ الاستبدال

يستخدم `FontsManager.ReplaceFont` لإجراء الاستبدال طوال العرض التقديمي الخاص بك:

```csharp
presentation.FontsManager.ReplaceFont(sourceFont, destFont);
```

### حفظ العرض التقديمي المحدث

وأخيرًا، احفظ العرض التقديمي المعدّل في ملف جديد.

```csharp
string outputFilePath = "YOUR_OUTPUT_DIRECTORY\\UpdatedFont_out.pptx";
presentation.Save(outputFilePath, SaveFormat.Pptx);
```

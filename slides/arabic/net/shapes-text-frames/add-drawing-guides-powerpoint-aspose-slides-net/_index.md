# دليل إضافة أدلة الرسم في PowerPoint باستخدام Aspose.Slides لـ .NET

## مقدمة
هل تواجه صعوبة في محاذاة العناصر بدقة داخل شريحة PowerPoint؟ تعلّم كيفية استخدام Aspose.Slides لـ .NET لإضافة أدلة رسم رأسية وأفقية بسهولة، مما يضمن دقة وضع الرسومات ومربعات النصوص والعناصر الأخرى.

**ما سوف تتعلمه:**
- إعداد Aspose.Slides لـ .NET في بيئة التطوير الخاصة بك.
- تعليمات خطوة بخطوة حول إضافة أدلة الرسم إلى الشريحة.
- فهم المعلمات والتكوينات المتوفرة مع هذه الميزة.

دعونا نتعمق في المتطلبات الأساسية أولاً!

قبل أن نبدأ، تأكد من أن لديك:

## إعداد Aspose.Slides لـ .NET
للبدء، ثبّت مكتبة Aspose.Slides. إليك الطريقة:

**استخدام .NET CLI:**
```bash
dotnet add package Aspose.Slides
```

**استخدام مدير الحزم:**
```powershell
Install-Package Aspose.Slides
```

**واجهة مستخدم مدير حزمة NuGet:**
- ابحث عن "Aspose.Slides" وانقر على "تثبيت" للحصول على الإصدار الأحدث.

License license = new License();

## دليل التنفيذ
الآن بعد أن قمنا بإعداد بيئتنا، فلنقم بإضافة أدلة الرسم تلك.

### إضافة أدلة الرسم إلى شريحة PowerPoint
#### ملخص
تتيح لك هذه الميزة تعزيز دقة الشريحة عن طريق إضافة أدلة رأسية وأفقية بناءً على متطلباتك.

##### الخطوة 1: إنشاء عرض تقديمي جديد
إنشاء مثيل لـ `Presentation` ستكون هذه هي لوحتنا التي سنضيف إليها أدلة الرسم.

```csharp
using Aspose.Slides;
using System.IO;

string outFilePath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "GuidesProperties-out.pptx");

using (Presentation pres = new Presentation())
{
    // سيتم وضع الكود الخاص بإضافة الأدلة هنا
}
```

##### الخطوة 2: الوصول إلى حجم الشريحة
استرجع أبعاد الشريحة الخاصة بك لتحديد موضع الأدلة بدقة.

```csharp
var slideSize = pres.SlideSize.Size;
```

##### الخطوة 3: إضافة أدلة عمودية وأفقية
الوصول إلى `DrawingGuidesCollection` من `SlideViewProperties` لإضافة أدلة جديدة. هنا، نضيف دليلاً رأسياً إلى يمين المركز ودليلاً أفقياً أسفله.

```csharp
IDrawingGuidesCollection guides = pres.ViewProperties.SlideViewProperties.DrawingGuides;

// أضف دليلاً رأسيًا في موضع الإزاحة
guides.Add(Orientation.Vertical, slideSize.Width / 2 + 12.5f);

// أضف دليلاً أفقيًا في موضع الإزاحة
guides.Add(Orientation.Horizontal, slideSize.Height / 2 + 12.5f);
```

##### الخطوة 4: حفظ العرض التقديمي
وأخيرًا، احفظ عرضك التقديمي باستخدام الأدلة المضافة.

```csharp
pres.Save(outFilePath, SaveFormat.Pptx);
```

#### نصائح استكشاف الأخطاء وإصلاحها
- تأكد من أن مسار دليل الإخراج الخاص بك صحيح لتجنب `DirectoryNotFoundException`.
- إذا لم تظهر الأدلة كما هو متوقع، فتأكد من صحة الحسابات الخاصة بمواضع الأدلة بالنسبة لحجم الشريحة.

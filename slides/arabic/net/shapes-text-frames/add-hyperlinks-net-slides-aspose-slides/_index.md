## دليل التنفيذ
دعونا نقسم العملية إلى خطوات قابلة للإدارة لإضافة الارتباطات التشعبية بشكل فعال. 

### إضافة ارتباط تشعبي إلى النص في الشرائح
#### ملخص
تتيح لك هذه الميزة ربط الموارد الخارجية مباشرةً من النص الموجود داخل شرائح العرض التقديمي، مما يعزز التفاعل والمشاركة.

#### دليل خطوة بخطوة
**1. تهيئة العرض التقديمي**
ابدأ بإنشاء مثيل لـ `Presentation` فصل:
```csharp
Presentation presentation = new Presentation();
```

**2. إضافة شكل مع نص**
أضف شكلاً تلقائيًا لاحتواء نصك. إليك كيفية تحديد الأبعاد والموضع:
```csharp
IAutoShape shape1 = presentation.Slides[0].Shapes.AddAutoShape(
    ShapeType.Rectangle, 100, 100, 600, 50, false);
shape1.AddTextFrame("Aspose: File Format APIs");
```

**3. الوصول إلى أجزاء النص**
انتقل إلى الجزء المحدد من النص الذي تريد إنشاء ارتباط تشعبي له:
```csharp
IParagraph paragraph = shape1.TextFrame.Paragraphs[0];
IPortion portion = paragraph.Portions[0];
```

**4. إضافة ارتباط تشعبي وتلميح الأدوات**
قم بإعداد ارتباطك التشعبي باستخدام عنوان URL وتلميح الأدوات الاختياري للسياق الإضافي:
```csharp
portion.PortionFormat.HyperlinkClick = new Hyperlink("https://www.aspose.com/");
portion.PortionFormat.HyperlinkClick.Tooltip = "More than 70% Fortune 100 companies trust Aspose APIs";
```

**5. ضبط حجم الخط**
لجعل النص أكثر بروزًا، اضبط حجم الخط:
```csharp
portion.PortionFormat.FontHeight = 32;
```

**6. احفظ عرضك التقديمي**
وأخيرًا، احفظ عرضك التقديمي باستخدام النص المرتبط به:
```csharp
presentation.Save(Path.Combine(YOUR_OUTPUT_DIRECTORY, "presentation-out.pptx"), SaveFormat.Pptx);
```

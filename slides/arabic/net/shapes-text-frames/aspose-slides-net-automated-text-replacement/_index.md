## دليل التنفيذ

الآن بعد أن قمت بإعداد كل شيء، دعنا ننتقل إلى تنفيذ ميزة استبدال النص.

### نظرة عامة على الميزة: استبدال النص في شرائح PowerPoint

تبحث هذه الميزة عن نص بديل محدد (مثل "[هذه الكتلة]") وتستبدله بالمحتوى المطلوب في جميع الشرائح. تُعد هذه الميزة مفيدة بشكل خاص عند تحديث العبارات الشائعة أو أسماء المنتجات في العرض التقديمي.

#### الخطوة 1: تحميل العرض التقديمي الخاص بك
ابدأ بتحميل العرض التقديمي حيث تريد استبدال النص:

```csharp
Presentation pres = new Presentation("example.pptx");
```

#### الخطوة 2: تحديد معلمات استبدال النص

حدد النص البديل والنص البديل. على سبيل المثال، استبدل "[هذه الكتلة]" بـ "نصي":

```csharp
string strToFind = "[this block]";
string strToReplaceWith = "my text";
```

#### الخطوة 3: تكرار الشرائح واستبدال النص

قم بالتنقل عبر كل شريحة في العرض التقديمي الخاص بك للعثور على النص النائب واستبداله:

```csharp
foreach (ISlide slide in pres.Slides)
{
    foreach (IAutoShape shape in slide.Shapes.OfType<IAutoShape>())
    {
        if (shape.TextFrame != null)
        {
            ITextFrame textFrame = shape.TextFrame;
            foreach (IParagraph para in textFrame.Paragraphs)
            {
                foreach (Portion portion in para.Portions)
                {
                    if (portion.Text.Contains(strToFind))
                    {
                        // استبدال النص
                        portion.Text = portion.Text.Replace(strToFind, strToReplaceWith);
                    }
                }
            }
        }
    }
}
```

#### توضيح:
- **حدود**: `strToFind` هو النص النائب الذي تستهدفه. `strToReplaceWith` هو ما تريد استبداله.
- **الطريقة والغرض**:تعمل الطريقة على تكرار أشكال كل شريحة، والبحث عن إطارات نصية تحتوي على العنصر النائب المحدد واستبداله.

### نصائح استكشاف الأخطاء وإصلاحها

- تأكد من متغيرات سلسلة النص الخاصة بك (`strToFind` و `strToReplaceWith`) يتم تعريفها بشكل صحيح.
- تحقق مما إذا كانت الشرائح تحتوي على التنسيق المتوقع (على سبيل المثال، وجود الأشكال التلقائية) لتجنب استثناءات المرجع الفارغ.

# إتقان أشكال المجموعة في Aspose.Slides .NET: برنامج تعليمي شامل

**ما سوف تتعلمه:**
- كيفية إعداد Aspose.Slides لـ .NET
- إنشاء شكل مجموعة على شريحة
- إضافة أشكال فردية داخل المجموعة
- حفظ العرض التقديمي الخاص بك مع الأشكال المجمعة

دعونا نلقي نظرة على المتطلبات الأساسية التي تحتاجها قبل البدء.

## إعداد Aspose.Slides لـ .NET
للبدء، عليك دمج Aspose.Slides في مشروعك. إليك الطريقة:

**استخدام .NET CLI:**
```bash

```

**استخدام مدير الحزم:**
```powershell

```

### التهيئة والإعداد الأساسي
بمجرد التثبيت، قم بتشغيل `Presentation` الصف الذي يعتبر بمثابة بوابة لإنشاء العروض التقديمية:
```csharp
using Aspose.Slides;
// إنشاء فئة عرض تقديمي
Presentation pres = new Presentation();
```

## دليل التنفيذ
في هذا القسم، سنتناول كل خطوة مطلوبة لإنشاء أشكال المجموعة وإضافة أشكال فردية داخلها.

### إنشاء شكل مجموعة على شريحة
ابدأ بالوصول إلى الشريحة التي تريد إضافة شكل المجموعة إليها:
```csharp
// الوصول إلى الشريحة الأولى من العرض التقديمي
ISlide sld = pres.Slides[0];
```
بعد ذلك، احصل على مجموعة الأشكال الموجودة على هذه الشريحة وقم بإنشاء شكل مجموعة جديد:
```csharp
// احصل على مجموعة أشكال الشريحة
IShapeCollection slideShapes = sld.Shapes;

// إضافة شكل مجموعة إلى الشريحة
IGroupShape groupShape = slideShapes.AddGroupShape();
```

### إضافة أشكال فردية داخل المجموعة
بعد إنشاء شكل مجموعتك، يمكنك الآن إضافة أشكال متنوعة إليها. إليك كيفية إضافة المستطيلات:
```csharp
// أضف الأشكال داخل شكل المجموعة الذي تم إنشاؤه
groupShape.Shapes.AddAutoShape(ShapeType.Rectangle, 300, 100, 100, 100);
groupShape.Shapes.AddAutoShape(ShapeType.Rectangle, 500, 100, 100, 100);
groupShape.Shapes.AddAutoShape(ShapeType.Rectangle, 300, 300, 100, 100);
groupShape.Shapes.AddAutoShape(ShapeType.Rectangle, 500, 300, 100, 100);
```
**المعلمات موضحة:**
- `ShapeType.Rectangle`:نوع الشكل الذي تضيفه.
- `x`، `y` (على سبيل المثال، 300، 100): إحداثيات الموضع على الشريحة.
- العرض والارتفاع (على سبيل المثال، 100، 100): أبعاد الشكل.

### حفظ العرض التقديمي الخاص بك
وأخيرًا، احفظ عرضك التقديمي في ملف:

```csharp
// حفظ العرض التقديمي على القرص
```

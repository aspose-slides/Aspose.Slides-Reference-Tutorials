## دليل التنفيذ
يرشدك هذا الدليل إلى كيفية إنشاء شريحة PowerPoint بتأثير ظل داخلي على النص باستخدام Aspose.Slides .NET. تنقسم العملية إلى خطوتين رئيسيتين: إنشاء شريحة وتطبيق التأثيرات.
### الميزة 1: إنشاء شريحة PowerPoint مع نص (H2)
#### ملخص
قم بإعداد عرض تقديمي جديد، وأضف شكل مستطيل، وأدرج نصًا، ثم احفظ النتيجة كملف PowerPoint.
#### التنفيذ خطوة بخطوة
**الخطوة 1**:تهيئة كائن العرض التقديمي
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
Presentation presentation = new Presentation();
```

**الخطوة 2**:الوصول إلى الشريحة الأولى
```csharp
ISlide slide = presentation.Slides[0];
```

**الخطوة 3**:أضف شكل مستطيل مع النص
- **إنشاء وتكوين الشكل**
```csharp
IAutoShape ashp = slide.Shapes.AddAutoShape(ShapeType.Rectangle, 150, 75, 400, 300);
ashp.FillFormat.FillType = FillType.NoFill;
```

- **إضافة إطار نص إلى المستطيل**
```csharp
ashp.AddTextFrame("Aspose TextBox");
IPortion port = ashp.TextFrame.Paragraphs[0].Portions[0];
IPortionFormat pf = port.PortionFormat;
pf.FontHeight = 50; // تعيين حجم الخط للرؤية
```

**الخطوة 4**:حفظ العرض التقديمي
```csharp
presentation.Save(dataDir + "WordArt_out.pptx", SaveFormat.Pptx);
```
### الميزة 2: إضافة تأثير الظل الداخلي إلى جزء النص (H2)
#### ملخص
قم بتعزيز النص الخاص بك باستخدام تأثير الظل الداخلي للحصول على مظهر ديناميكي.
#### التنفيذ خطوة بخطوة
**الخطوة 1**:تمكين تأثير الظل الداخلي
```csharp
IEffectFormat ef = pf.EffectFormat;
ef.EnableInnerShadowEffect();
```

**الخطوة 2**:تكوين خصائص الظل الداخلي
```csharp
// قم بتخصيص تأثير الظل الداخلي للحصول على مظهر متطور
ef.InnerShadowEffect.BlurRadius = 8.0; // التحكم في نصف قطر طمس الظل
ef.InnerShadowEffect.Direction = 90.0F; // ضبط الاتجاه بالدرجات
ef.InnerShadowEffect.Distance = 6.0; // حدد مدى بعد الظل عن النص

// ضبط إعدادات الألوان للحصول على مظهر أكثر تخصيصًا
ef.InnerShadowEffect.ShadowColor.B = 189;
ef.InnerShadowEffect.ShadowColor.ColorType = ColorType.Scheme;
ef.InnerShadowEffect.ShadowColor.SchemeColor = SchemeColor.Accent1;
```
**الخطوة 3**:احفظ عرضك التقديمي المحسّن
```csharp
presentation.Save(dataDir + "WordArt_out.pptx", SaveFormat.Pptx);
```
### نصائح استكشاف الأخطاء وإصلاحها
- تأكد من `dataDir` تم تعيين المسار بشكل صحيح لتجنب أخطاء حفظ الملف.
- تأكد من أبعاد الشكل والمواضع إذا لم تظهر كما هو متوقع.

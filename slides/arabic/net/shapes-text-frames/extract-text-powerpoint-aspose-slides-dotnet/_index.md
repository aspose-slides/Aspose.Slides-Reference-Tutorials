### دليل التنفيذ

في هذا القسم، سنشرح كيفية استخراج النص الخام من عروض PowerPoint.

#### استخراج النص الخام

**ملخص**:تتيح لك هذه الميزة استرداد جميع بيانات النص غير المرتبة—مثل نصوص الشرائح والملاحظات—من ملف العرض التقديمي.

1. **حدد دليل المستندات الخاص بك**
   ```csharp
   string documentDirectory = @"YOUR_DOCUMENT_DIRECTORY\";
   ```

2. **إنشاء المسار الكامل لملف العرض التقديمي الخاص بك**
   ```csharp
   string presentationName = Path.Combine(documentDirectory, "PresentationText.pptx");
   ```

3. **الحصول على النص الخام باستخدام `PresentationFactory`**
   ```csharp
   IPresentationText presentationText = 
       PresentationFactory.Instance.GetPresentationText(presentationName, 
                                                       TextExtractionArrangingMode.Unarranged);
   ```

4. **الوصول إلى بيانات الشريحة المحددة وتخزينها**
   - استرجاع التعليقات من الشريحة الأولى:
     ```csharp
     string commentsSlide1 = presentationText.SlidesText[0].CommentsText;
     ```
   
   - احصل على النص من الشريحة الأولى:
     ```csharp
     string textSlide1 = presentationText.SlidesText[0].Text;
     ```

   - ملاحظات الوصول من الشريحة الثانية:
     ```csharp
     string notesSlide2 = presentationText.SlidesText[1].NotesText;
     ```

**نصائح استكشاف الأخطاء وإصلاحها**:تأكد من تعيين مسارات الملفات بشكل صحيح وتحقق من وجود أي مشكلات تتعلق بأذونات الوصول إلى الملفات.

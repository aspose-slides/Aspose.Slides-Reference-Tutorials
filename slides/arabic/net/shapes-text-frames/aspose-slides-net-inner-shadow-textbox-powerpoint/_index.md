
### إنشاء مربع نص وتنسيقه
#### الخطوة 1: إعداد بيئة مشروعك
أولاً، تأكد من إعداد دليل المشروع الخاص بك:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
bool isExists = Directory.Exists(dataDir);
if (!isExists)
    Directory.CreateDirectory(dataDir);
```

يتحقق هذا المقطع من وجود دليل محدد، ويُنشئه إن لم يكن. هذا يضمن تخزين ملفات العرض التقديمي في المكان الصحيح.

#### الخطوة 2: إنشاء كائن العرض التقديمي
```csharp
class Program
{
    static void Main()
    {
        using (Presentation pres = new Presentation())
        {
            ISlide sld = pres.Slides[0]; // الوصول إلى الشريحة الأولى
```
هنا، نقوم بإنشاء مثال `Presentation` الكائن والوصول إلى الشريحة الأولى منه. تُجرى جميع العمليات على هذه الشريحة.

#### الخطوة 3: إضافة شكل تلقائي مع الظل الداخلي
```csharp
// إضافة شكل مستطيل بموضع (150، 75) وحجم (150 × 50)
IAutoShape ashp = sld.Shapes.AddAutoShape(ShapeType.Rectangle, 150, 75, 150, 50);

// إضافة نص إلى الشكل
txtFrame = ashp.TextFrame;
para = txtFrame.Paragraphs[0];
portion = para.Portions[0];

// ضبط نص الجزء
portion.Text = "Aspose TextBox";
```
يُضيف هذا القسم شكل مستطيل إلى الشريحة ويُجهّزه بإطار نص فارغ. يمكنك لاحقًا تطبيق تأثيرات مثل الظل الداخلي على هذا الشكل.

#### الخطوة 4: تطبيق تأثير الظل الداخلي
لإضافة ظل داخلي، عادةً ما تقوم بتعديل `ashp` خصائص نمط الكائن. مع ذلك، لا يدعم Aspose.Slides لـ .NET الظل الداخلي مباشرةً عبر الطرق المضمنة وقت كتابة هذا التقرير، لذا قد تحتاج إلى استخدام أساليب بديلة أو مكتبات إضافية توفر معالجة رسومية أكثر تطورًا.

في الوقت الحالي، دعونا نركز على حفظ العرض التقديمي الخاص بنا:

# إتقان معالجة SmartArt في عروض .NET التقديمية باستخدام Aspose.Slides

**ما سوف تتعلمه:**
- تحميل العروض التقديمية الموجودة.
- إضافة أشكال SmartArt وتحديد موقعها بشكل فعال.
- ضبط حجم وتدوير أشكال SmartArt.
- حفظ العرض التقديمي المحسن الخاص بك بسلاسة.

لنستكشف كيفية الاستفادة من Aspose.Slides لـ .NET لتصميم عروض تقديمية فعّالة. أولًا، تأكد من استيفاء هذه المتطلبات الأساسية.


## دليل التنفيذ
سنقوم بتغطية الميزات المحددة باستخدام Aspose.Slides لـ .NET.

### تحميل عرض تقديمي
ابدأ بتحميل ملف عرض تقديمي موجود لإضافة SmartArt أو إجراء تعديلات عليه.

**مقتطف من الكود:**
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
Presentation pres = new Presentation(dataDir + "/AccessChildNodes.pptx");
```
*توضيح:* يقوم الكود أعلاه بتحميل ملف PowerPoint من الدليل المحدد، وإعداده لمزيد من المعالجة.

### إضافة شكل SmartArt وتحديد موضعه
عزّز شريحتك بإضافة شكل SmartArt. يرشدك هذا القسم إلى كيفية وضع SmartArt بدقة على شريحتك.

**ملخص:**
أضف تخطيط SmartArt إلى الشريحة الأولى عند إحداثيات محددة بأبعاد محددة.

**مقتطف من الكود:**
```csharp
ISmartArt smart = pres.Slides[0].Shapes.AddSmartArt(20, 20, 600, 500, SmartArtLayoutType.OrganizationChart);
```
*توضيح:* ال `AddSmartArt` تضع هذه الطريقة شكل SmartArt جديدًا على الشريحة. تُحدد المعلمات موضعه وحجمه.

**نقل شكل العقدة الفرعية:**
```csharp
ISmartArtNode node = smart.AllNodes[1];
ISmartArtShape shape = node.Shapes[1];
shape.X += (shape.Width * 2); // التحرك إلى اليمين بمقدار ضعف عرضه
shape.Y -= (shape.Height / 2); // التحرك لأعلى بمقدار نصف ارتفاعه
```
*توضيح:* ضبط موضع شكل عقدة فرعية محددة داخل SmartArt.

### ضبط عرض الشكل وارتفاعه
قم بتعديل أبعاد الأشكال لتتناسب بشكل أفضل مع احتياجات تصميم العرض التقديمي الخاص بك.

**مقتطف من الكود:**
```csharp
node = smart.AllNodes[2];
shape = node.Shapes[1];
shape.Width += (shape.Width / 2); // زيادة العرض إلى نصف حجمه الأصلي

node = smart.AllNodes[3];
shape = node.Shapes[1];
shape.Height += (shape.Height / 2); // زيادة الارتفاع إلى النصف
```
*توضيح:* تعمل هذه الأسطر من التعليمات البرمجية على ضبط أبعاد الشكل، مما يعزز من جاذبيته البصرية.

### تدوير شكل SmartArt
قم بتدوير الأشكال لإنشاء تخطيطات ديناميكية ومثيرة للاهتمام بصريًا.

**مقتطف من الكود:**
```csharp
node = smart.AllNodes[4];
shape = node.Shapes[1];
shape.Rotation = 90; // تدوير بمقدار 90 درجة
```
*توضيح:* يقوم هذا السطر البسيط من التعليمات البرمجية بتدوير الشكل المحدد داخل SmartArt، مما يضيف لمسة إبداعية إلى الشريحة الخاصة بك.

### حفظ العرض التقديمي
بعد إجراء كافة التغييرات، احفظ العرض التقديمي في دليل الإخراج المطلوب.

**مقتطف من الكود:**
```csharp
pres.Save("YOUR_OUTPUT_DIRECTORY/SmartArt.pptx");
```
*توضيح:* ال `Save` تقوم الطريقة بتنفيذ جميع التعديلات التي تم إجراؤها أثناء الجلسة في ملف جديد.

## إعداد Aspose.Slides لـ .NET
إعداد مشروعك باستخدام Aspose.Slides سهل للغاية. اختر إحدى طرق التثبيت التالية:

```

```

**واجهة مستخدم مدير الحزم NuGet**:ابحث عن "Aspose.Slides" وقم بتثبيت الإصدار الأحدث.

## دليل التنفيذ
دعنا نقسم التنفيذ إلى ميزات محددة لإنشاء SmartArt والتلاعب به.

### إنشاء SmartArt باستخدام تخطيط الدورة الشعاعية
#### ملخص
توضح هذه الميزة كيفية إنشاء رسم SmartArt باستخدام تخطيط Radial Cycle، وهو مثالي لتوضيح العمليات الدورية أو المخططات الانسيابية في العروض التقديمية الخاصة بك.

#### التنفيذ خطوة بخطوة
**1. تهيئة العرض التقديمي**
ابدأ بإنشاء مثيل لـ `Presentation` فصل:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // قم بتعيين المسار إلى دليل المستند الخاص بك.
using (Presentation presentation = new Presentation())
{
    ...
}
```

**2. إضافة رسومات SmartArt**
قم بإضافة رسم SmartArt بإحداثيات وأبعاد محددة باستخدام تخطيط Radial Cycle.
```csharp
ISmartArt smart = presentation.Slides[0].Shapes.AddSmartArt(10, 10, 400, 300, SmartArtLayoutType.RadialCycle);
```
- **حدود**: ال `AddSmartArt` تأخذ الطريقة إحداثيات x وy والعرض والارتفاع لتحديد موضع الرسم البياني.

**3. حفظ العرض التقديمي**
وأخيرًا، احفظ عرضك التقديمي في ملف:
```csharp
presentation.Save(dataDir + "CreateSmartArt_out.pptx", SaveFormat.Pptx);
```

### إضافة العقد إلى SmartArt
#### ملخص
تعرف على كيفية إضافة العقد بشكل ديناميكي إلى رسم SmartArt الحالي، مما يعزز تفاصيله وقيمته المعلوماتية.

#### التنفيذ خطوة بخطوة
**1. إضافة عقدة**
بعد إنشاء SmartArt الأولي الخاص بك:
```csharp
ISmartArtNode node = smart.AllNodes.AddNode();
```
- **فهم العقد**:تمثل العقد عناصر فردية داخل بنية SmartArt.

### التحقق من خاصية العقدة المخفية في SmartArt
#### ملخص
اكتشف كيفية التحقق مما إذا كانت عقدة معينة مخفية، مما يسمح بالتحكم في الرؤية الديناميكية ضمن العروض التقديمية الخاصة بك.

#### التنفيذ خطوة بخطوة
**1. التحقق من الرؤية**
بعد إضافة عقدة:
```csharp
bool hidden = node.IsHidden; // إرجاع القيمة true أو false بناءً على الرؤية
```

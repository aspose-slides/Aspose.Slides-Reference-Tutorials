## دليل التنفيذ

الآن، لنبدأ بتطبيق معالجة المقاطعة خطوة بخطوة. تتيح لك هذه الميزة إيقاف المهام طويلة الأمد دون إنهائها فجأة.

### الخطوة 1: تكوين دعم المقاطعة

إنشاء إجراء يقوم بتحميل العرض التقديمي بإمكانيات المقاطعة:

```csharp
Action<IInterruptionToken> loadPresentationWithInterruptSupport = (IInterruptionToken token) =>
{
    // خيارات التحميل التي تم تكوينها باستخدام InterruptToken
    LoadOptions options = new LoadOptions { InterruptionToken = token };
    
    using (Presentation presentation = new Presentation(dataDir + "pres.pptx", options))
    {
        // احفظ بتنسيق مختلف، مما يوضح دعم المقاطعة
        presentation.Save(outputDir + "pres.ppt", SaveFormat.Ppt);
    }
};
```

**توضيح:** ال `LoadOptions` يستخدم الكائن `InterruptionToken`، مما يسمح بإيقاف المهمة مؤقتًا أو إيقافها بشكل سليم.

### الخطوة 2: تهيئة مصدر رمز المقاطعة

إنشاء مثيل لـ `InterruptionTokenSource`:

```csharp
// إنشاء رموز المقاطعة
InterruptionTokenSource tokenSource = new InterruptionTokenSource();
```

**توضيح:** ال `InterruptionTokenSource` يُنشئ رموزًا يمكن استخدامها للتحكم في تدفق التنفيذ.

### الخطوة 3: تشغيل ومقاطعة المهمة

قم بتنفيذ الإجراء الخاص بك على سلسلة منفصلة ومحاكاة الانقطاع:

```csharp
// تنفيذ في موضوع منفصل
Run(loadPresentationWithInterruptSupport, tokenSource.Token);

// محاكاة التأخير لانقطاع المهمة
Thread.Sleep(10000); // انتظر لمدة 10 ثواني

// إثارة المقاطعة
tokenSource.Interrupt();
```

**توضيح:** الطريقة `Run` يبدأ الإجراء على سلسلة جديدة، مما يسمح لك بالاتصال `Interrupt()` بعد مرور وقت محدد لإيقاف العملية.

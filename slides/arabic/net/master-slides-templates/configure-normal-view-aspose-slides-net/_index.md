# تكوين العرض العادي في Aspose.Slides .NET: دليل شامل للعروض التقديمية

## مقدمة

قد يكون من الصعب إدارة العرض العادي لعروض PowerPoint برمجيًا. سيساعدك هذا الدليل الشامل حول استخدام Aspose.Slides .NET، وهي مكتبة فعّالة لإدارة عروض PowerPoint، على تكوين ميزات أساسية مثل أوضاع شريط التقسيم وخيارات العرض.

**ما سوف تتعلمه:**
- إعداد Aspose.Slides في بيئة .NET
- تكوين حالة العرض العادية للعروض التقديمية
- ضبط أشرطة التقسيم الأفقية والرأسية
- تمكين التعديل التلقائي للمشاهد المستعادة
- عرض أيقونات المخطط التفصيلي داخل العرض التقديمي الخاص بك

## المتطلبات الأساسية
قبل البدء، تأكد من أن لديك:

### المكتبات المطلوبة:
- **Aspose.Slides لـ .NET**:المكتبة الأساسية لإدارة عروض PowerPoint.

### متطلبات إعداد البيئة:
- بيئة تطوير .NET عاملة (على سبيل المثال، Visual Studio).
- المعرفة الأساسية بمفاهيم البرمجة C# و.NET.

## إعداد Aspose.Slides لـ .NET
لبدء استخدام Aspose.Slides، ثبّته في مشروعك. إليك خطوات التثبيت:

### طرق التثبيت:
**.NET CLI:**
```bash
dotnet add package Aspose.Slides
```

**وحدة تحكم مدير الحزمة:**
```bash
Install-Package Aspose.Slides
```

**واجهة مستخدم مدير حزمة NuGet:** 
ابحث عن "Aspose.Slides" وقم بتثبيت الإصدار الأحدث.

### الحصول على الترخيص:
ابدأ بفترة تجريبية مجانية أو اطلب ترخيصًا مؤقتًا لاستكشاف جميع الميزات. للاستخدام طويل الأمد، فكّر في شراء اشتراك من خلال موقعهم الرسمي.

#### التهيئة الأساسية:
```csharp
using Aspose.Slides;

// تهيئة كائن عرض تقديمي جديد
Presentation pres = new Presentation();
```

## دليل التنفيذ
فيما يلي كيفية تكوين حالة العرض العادية في خطوات قابلة للإدارة:

### تكوين حالة الشريط الأفقي
اضبط حالة الشريط الأفقي على "مستعاد" أو "مُصغّر" أو "مخفي". يُحدد هذا كيفية عرض لوحة الشرائح عند فتحها.

#### خطوات:
1. **إنشاء كائن عرض تقديمي:**
   ```csharp
   using Aspose.Slides;
   
   // تهيئة مثيل عرض تقديمي جديد
   Presentation pres = new Presentation();
   ```
2. **تعيين حالة الشريط الأفقي:**
   ```csharp
   // تعيين حالة الشريط الأفقي إلى استعادة
   pres.ViewProperties.NormalViewProperties.HorizontalBarState = SplitterBarStateType.Restored;
   ```
   - **لماذا؟** يضمن هذا للمستخدمين إمكانية رؤية عرض كامل للشرائح عند فتح العرض التقديمي.

### تكوين حالة الشريط العمودي
يُسهّل الشريط العمودي التنقل بين الأقسام أو العروض الرئيسية. تكبيره يُحسّن التحكم.

#### خطوات:
1. **تعيين حالة الشريط العمودي:**
   ```csharp
   // تعيين حالة الشريط العمودي إلى الحد الأقصى
   pres.ViewProperties.NormalViewProperties.VerticalBarState = SplitterBarStateType.Maximized;
   ```
   - **لماذا؟** يوفر الشريط الرأسي المكبر نظرة عامة على تخطيطات الشرائح، مما يساعد في إدارة العرض التقديمي بشكل أفضل.

### تمكين الضبط التلقائي لاستعادة العرض العلوي
يضمن الضبط التلقائي أن يتكيف العرض المستعاد مع المساحة المتوفرة، مما يعزز إمكانية القراءة وتجربة المستخدم.

#### خطوات:
1. **تمكين الضبط التلقائي:**
   ```csharp
   // تمكين التعديل التلقائي
   pres.ViewProperties.NormalViewProperties.RestoredTop.AutoAdjust = true;
   
   // تعيين حجم الأبعاد لتحسين الرؤية
   pres.ViewProperties.NormalViewProperties.RestoredTop.DimensionSize = 80;
   ```
   - **لماذا؟** تساعد هذه الميزة على إبقاء العرض التقديمي الخاص بك متجاوبًا، ويتكيف مع أحجام الشاشات المختلفة بشكل فعال.

### أيقونات مخطط العرض
تساعد أيقونات المخطط التفصيلي المستخدمين على تحديد هيكل العرض التقديمي الخاص بك بسرعة.

#### خطوات:
1. **إظهار أيقونات المخطط التفصيلي:**
   ```csharp
   // تمكين عرض أيقونات المخطط التفصيلي
   pres.ViewProperties.NormalViewProperties.ShowOutlineIcons = true;
   ```
   - **لماذا؟** تساعد هذه الإشارة المرئية المستخدمين على فهم الهيكل الهرمي لمحتوى العرض التقديمي الخاص بك بسرعة.

### حفظ العرض التقديمي المُهيأ
بعد التكوين، احفظ العرض التقديمي للاحتفاظ بهذه الإعدادات.

#### خطوات:
1. **حفظ الملف:**
   ```csharp
   string dataDir = "YOUR_DOCUMENT_DIRECTORY/";

   // حفظ باسم الملف والتنسيق المحددين
   pres.Save(Path.Combine(dataDir, "presentation_normal_view_state.pptx"), SaveFormat.Pptx);
   ```

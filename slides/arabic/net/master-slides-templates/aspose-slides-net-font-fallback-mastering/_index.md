# إتقان استخدام الخطوط البديلة في العروض التقديمية باستخدام Aspose.Slides لـ .NET

هل تواجه مشكلة عدم تناسق الخطوط في عروضك التقديمية على مختلف الأجهزة والمنصات؟ يكمن الحل غالبًا في آليات فعّالة لاستبدال الخطوط. يستفيد هذا البرنامج التعليمي من **Aspose.Slides لـ .NET** لتنفيذ خط احتياطي قوي، وضمان الطباعة المتسقة في جميع الشرائح الخاصة بك.

- تطبيق هذه القواعد في معالجة العرض التقديمي
- تطبيقات عملية ونصائح لتحسين الأداء

تأكد من أن كل شيء جاهز قبل أن نبدأ.

## دليل التنفيذ

سنقوم بتقسيم التنفيذ إلى ميزات رئيسية لمساعدتك على فهم كل جانب وتطبيقه بشكل فعال.

### الميزة: الإعداد والتهيئة

الخطوة الأولى هي تهيئة بيئتك. يُهيئ هذا الإعداد Aspose.Slides للتعامل مع الخطوط في العروض التقديمية.

```csharp
using Aspose.Slides;
using System.Collections.Generic;

string dataDir = @"YOUR_DOCUMENT_DIRECTORY";

IFontFallBackRulesCollection rulesList = new FontFallBackRulesCollection();
```

**توضيح**: 
- `dataDir`:يحدد الدليل لملفات العرض التقديمي الخاص بك.
- `rulesList`:كائن لإدارة قواعد الرجوع إلى الخط.

### الميزة: إضافة وتعديل قواعد الرجوع إلى الخطوط

إن إنشاء قواعد الرجوع إلى الخطوط وتعديلها يضمن استبدال الخطوط غير المدعومة بخطوط بديلة، مما يحافظ على الاتساق البصري.

#### الخطوة 1: إضافة قاعدة أساسية
```csharp
rulesList.Add(new FontFallBackRule(0x400, 0x4FF, "Times New Roman"));
```

**توضيح**: 
- إضافة قاعدة للأحرف الموجودة في النطاق `0x400` ل `0x4FF` لاستخدام "Times New Roman".

#### الخطوة 2: تعديل القواعد الحالية
```csharp
foreach (IFontFallBackRule fallBackRule in rulesList)
{
    // إزالة "Tahoma" من خيارات العودة
    fallBackRule.Remove("Tahoma");

    // أضف "Verdana" لنطاقات الأحرف المحددة
    if ((fallBackRule.RangeEndIndex >= 0x4000) && (fallBackRule.RangeStartIndex < 0x5000))
        fallBackRule.AddFallBackFonts("Verdana");
}
```

**توضيح**: 
- يتكرر من خلال القواعد لضبط الخطوط البديلة، وإزالة "Tahoma" وإضافة "Verdana" لبعض النطاقات.

#### الخطوة 3: إزالة القاعدة
```csharp
if (rulesList.Count > 0)
    rulesList.Remove(rulesList[0]);
```

**توضيح**: 
- يقوم بإزالة القاعدة الأولى بشكل آمن إذا كانت موجودة، مما يوضح كيفية إدارة قائمة القواعد الخاصة بك بشكل ديناميكي.

### الميزة: معالجة العروض التقديمية باستخدام قواعد الرجوع إلى الخطوط

يؤدي تطبيق هذه القواعد على العرض التقديمي إلى ضمان عرض كافة الشرائح باستخدام الخطوط الصحيحة.

```csharp
using (Presentation pres = new Presentation(dataDir + "input.pptx"))
{
    // تعيين قواعد الخطوط الاحتياطية لمدير الخطوط في العرض التقديمي
    pres.FontsManager.FontFallBackRulesCollection = rulesList;
    
    // تقديم وحفظ الشريحة الأولى كصورة PNG
    pres.Slides[0].GetImage(1f, 1f).Save(dataDir + "Slide_0.png");
}
```

**توضيح**: 
- يقوم بتحميل العرض التقديمي وتعيينه `rulesList` إلى مدير الخطوط الخاص به.
- يقوم بعرض الشريحة الأولى باستخدام القواعد المحددة ويحفظها كصورة.

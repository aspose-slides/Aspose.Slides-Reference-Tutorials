## دليل التنفيذ
الآن بعد أن أصبح كل شيء جاهزًا، فلنبدأ في تنفيذ ميزاتنا الرئيسية: حفظ العروض التقديمية واستخراج الصور.

### حفظ ملف العرض التقديمي
**ملخص**
حفظ عرض تقديمي يتطلب كتابة الشرائح المُعدّلة أو المُنشأة حديثًا على القرص. هذا ضروري للحفاظ على التغييرات التي أُجريت برمجيًا.

#### الخطوة 1: تحميل العرض التقديمي
أولاً، قم بتحميل ملف PowerPoint الموجود:
```csharp
Presentation presentation = new Presentation("input.pptx");
```
يؤدي هذا إلى تحميل العرض التقديمي الخاص بك في الذاكرة، ليصبح جاهزًا للتعديل أو الحفظ.

#### الخطوة 2: حفظ العرض التقديمي
بعد ذلك، احفظه في الموقع المحدد:
```csharp
presentation.Save("output.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
```
تأكد من ذلك `YOUR_OUTPUT_DIRECTORY` يتم استبداله بالمسار المطلوب. هذه الخطوة تُعيد كتابة جميع التغييرات إلى القرص.

### استخراج الصور من العرض التقديمي
**ملخص**
استخراج الصور المضمنة داخل الشرائح لاستخدامها في أماكن أخرى في التطبيقات أو للتحليل.

#### الخطوة 1: الوصول إلى الشريحة
كرر كل شريحة:
```csharp
foreach (ISlide slide in presentation.Slides)
{
    // معالجة كل شريحة
}
```
تتيح لك هذه الحلقة الوصول إلى الشرائح الفردية ومكوناتها.

#### الخطوة 2: استخراج الصور
داخل كل شريحة، استخرج الصور:
```csharp
int imageIndex = 0;
foreach (IPPImage img in slide.Images)
{
    using (FileStream fileStream = new FileStream($"image{imageIndex++}.png", FileMode.Create))
    {
        img.SystemImage.Save(fileStream, ImageFormat.Png);
    }
}
```
يقوم هذا الكود بحفظ كل صورة على القرص. `imageIndex` يضمن أسماء ملفات فريدة للصور المستخرجة.

## دليل التنفيذ
دعونا نقوم بتقسيم الخطوات لملء الأشكال بالأنماط باستخدام Aspose.Slides لـ .NET.

### إضافة الأشكال وتطبيق الأنماط
#### ملخص:
تتيح لك هذه الميزة تحسين شرائحك عن طريق ملء الأشكال مثل المستطيلات أو الدوائر بأنماط مخصصة، مما يضيف عنصرًا مرئيًا فريدًا.

#### دليل خطوة بخطوة:
##### 1. إنشاء كائن عرض تقديمي
ابدأ بتهيئة العرض التقديمي:

```csharp
using Aspose.Slides;
// تحديد مسارات الدليل كعناصر نائبة
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

using (Presentation pres = new Presentation())
{
    // سيتم وضع الكود الخاص بك هنا
}
```
##### 2. الوصول إلى الشريحة الأولى
استرجاع الشريحة الأولى من العرض التقديمي الخاص بك:

```csharp
ISlide sld = pres.Slides[0];
```
*لماذا؟* يتيح لك هذا تطبيق التغييرات مباشرةً على شريحة موجودة أو إنشاء شريحة جديدة.

##### 3. إضافة شكل تلقائي
أضف شكل مستطيل حيث ستطبق تعبئة النمط:

```csharp
IShape shp = sld.Shapes.AddAutoShape(ShapeType.Rectangle, 50, 150, 75, 150);
```
*لماذا؟* يؤدي هذا إلى إعداد قماشك للتخصيص باستخدام الأنماط.

##### 4. اضبط نوع التعبئة على النمط
تغيير نوع تعبئة الشكل إلى نمط:

```csharp
shp.FillFormat.FillType = FillType.Pattern;
```

##### 5. تحديد نمط النمط
اختر نمطًا، مثل Trellis:

```csharp
shp.FillFormat.PatternFormat.PatternStyle = PatternStyle.Trellis;
```
*لماذا؟* تضيف الأنماط مثل Trellis الملمس والعمق إلى الشرائح الخاصة بك.

##### 6. تعيين ألوان الخلفية والأمامية
تخصيص الألوان لتحسين المظهر البصري:

```csharp
shp.FillFormat.PatternFormat.BackColor.Color = Color.LightGray;
shp.FillFormat.PatternFormat.ForeColor.Color = Color.Yellow;
```

##### 7. احفظ العرض التقديمي
وأخيرًا، احفظ التغييرات في ملف جديد:

```csharp
pres.Save(Path.Combine(dataDir, "RectShpPatt_out.pptx"), SaveFormat.Pptx);
```
*لماذا؟* تضمن هذه الخطوة تخزين كافة التعديلات وتجهيزها للعرض.

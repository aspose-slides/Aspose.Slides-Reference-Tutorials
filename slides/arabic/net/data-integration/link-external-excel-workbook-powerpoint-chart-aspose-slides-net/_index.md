## دليل التنفيذ

في هذا القسم، سنقوم بتفصيل الخطوات اللازمة لربط مصنف خارجي بمخطط في PowerPoint.

### إنشاء مصنف خارجي وربطه بالمخطط
#### ملخص
سنوضح كيفية ربط ملف Excel بمخطط دائري مُدمج في عرضك التقديمي. تتيح لك هذه الميزة إدارة البيانات خارجيًا مع الحفاظ على حيوية وتحديث شرائحك.

#### التنفيذ خطوة بخطوة
**1. إعداد العرض التقديمي**
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // استبدل بمسار دليل المستند الخاص بك
using (Presentation pres = new Presentation(dataDir + "/presentation.pptx"))
{
    string externalWbPath = dataDir + "/externalWorkbook1.xlsx";
```
*توضيح*نبدأ بتحميل ملف باوربوينت موجود. إذا لم يكن لديك ملف، أنشئ عرضًا تقديميًا فارغًا.

**2. إضافة الرسم البياني**
```csharp
// أضف مخططًا دائريًا إلى الشريحة الأولى في الموضع (50، 50) بالحجم (400، 600)
IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.Pie, 50, 50, 400, 600);
```
*توضيح*نضيف مخططًا دائريًا جديدًا إلى الشريحة الأولى. سيتم ربط هذا المخطط لاحقًا بمصنف خارجي.

**3. إدارة ملف المصنف الخارجي**
```csharp
// إذا كان ملف مصنف خارجي موجودًا بالفعل، فاحذفه لبدء تشغيل جديد
if (File.Exists(externalWbPath))
    File.Delete(externalWbPath);
```
*توضيح*:لتجنب التعارضات مع البيانات السابقة، نتحقق من وجود الملف ونقوم بحذفه.

**4. إنشاء البيانات وكتابتها في المصنف**
```csharp
using (FileStream fileStream = new FileStream(externalWbPath, FileMode.CreateNew))
{
    byte[] workbookData = chart.ChartData.ReadWorkbookStream().ToArray(); // قراءة تدفق بيانات مصنف الرسم البياني
    fileStream.Write(workbookData, 0, workbookData.Length); // اكتب هذه البيانات إلى ملف المصنف الخارجي الجديد
}
```
*توضيح*ننشئ ملف إكسل جديدًا ونكتب فيه بيانات الرسم البياني الأولية. هذه الخطوة أساسية لربط العرض التقديمي بدفتر العمل.

**5. تعيين المصنف الخارجي كمصدر للبيانات**
```csharp
// تعيين المصنف الخارجي الذي تم إنشاؤه حديثًا كمصدر بيانات للرسم البياني
chart.ChartData.SetExternalWorkbook(externalWbPath);
```
*توضيح*:من خلال تعيين مسار المصنف الخارجي، نقوم بربط ملف Excel بمخطط PowerPoint الخاص بنا.

**6. حفظ العرض التقديمي**
```csharp
pres.Save(dataDir + "/Presentation_with_externalWbPath.pptx", SaveFormat.Pptx);
}
```
*توضيح*:أخيرًا، احفظ العرض التقديمي مع تطبيق كافة التغييرات.

### نصائح استكشاف الأخطاء وإصلاحها
- تأكد من أن مسارات الملفات صحيحة ويمكن الوصول إليها.
- تأكد من ربط المصنف باستخدام `SetExternalWorkbook` إذا لم يتم عرض البيانات.
- راجع وثائق Aspose.Slides للتعرف على أنواع أو أحجام المخططات المدعومة في حالة ظهور مشكلات.

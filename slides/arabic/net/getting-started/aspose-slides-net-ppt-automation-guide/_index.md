## دليل التنفيذ
الآن دعونا نستكشف كيفية إنشاء عرض تقديمي جديد باستخدام خط تم تشكيله تلقائيًا.

### إنشاء عرض تقديمي جديد وإضافة شكل الخط
#### ملخص
يوضح هذا القسم كيفية تهيئة عرض تقديمي جديد، والوصول إلى الشريحة الافتراضية، وإضافة شكل خط، وحفظ الملف.

#### التنفيذ خطوة بخطوة
**1. إنشاء كائن العرض التقديمي**
إنشاء مثيل جديد من `Presentation` الفئة التي تمثل ملف PowerPoint الخاص بك:
```csharp
using (Presentation presentation = new Presentation())
{
    // الكود سوف يذهب هنا
}
```
يؤدي هذا إلى تهيئة عرض تقديمي فارغ يمكننا تعديله.

**2. الوصول إلى الشريحة الأولى**
يتم الوصول إلى شرائح العرض التقديمي من خلال مجموعة مُفهرسة. إليك كيفية الحصول على الشريحة الأولى:
```csharp
ISlide slide = presentation.Slides[0];
```

**3. إضافة خط مُشكَّل تلقائيًا**
لإضافة خط، نستخدم `AddAutoShape` طريقة مع معلمات محددة لنوع الشكل والأبعاد:
```csharp
slide.Shapes.AddAutoShape(نوع الشكل.الخط, 50, 150, 300, 0);
```
- **ShapeType.Line**:يشير إلى أن الشكل عبارة عن خط.
- **الإحداثيات (50، 150)**:قم بتحديد نقطة بداية الخط على الشريحة.
- **الأبعاد (300، 0)**: اضبط الطول والعرض. يضمن العرض الصفري أن يكون مجرد خط.

**4. احفظ العرض التقديمي**
حدد دليل الإخراج الخاص بك واحفظ العرض التقديمي بالتنسيق المطلوب:
```csharp
string outputDirectory = "YOUR_OUTPUT_DIRECTORY";
string outputFile = outputDirectory + "/NewPresentation_out.pptx";

presentation.Save(outputFile, SaveFormat.Pptx);
```

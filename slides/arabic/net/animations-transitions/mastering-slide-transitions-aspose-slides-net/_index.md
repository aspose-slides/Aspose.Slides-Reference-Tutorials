## دليل التنفيذ: تطبيق انتقالات الشرائح

### ملخص

حوّل العروض التقديمية الثابتة إلى تجارب سرد قصصية ديناميكية باستخدام انتقالات الشرائح. لنستكشف كيفية تطبيق تأثيرات انتقالية متنوعة باستخدام Aspose.Slides لـ .NET.

#### الخطوة 1: تحميل العرض التقديمي

إنشاء مثيل لـ `Presentation` الصف وتحميل ملف PowerPoint الخاص بك:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
using (Presentation pres = new Presentation(dataDir + "SimpleSlideTransitions.pptx"))
{
    // سيتم وضع الكود لتطبيق التحولات هنا.
}
```
يؤدي هذا إلى فتح العرض التقديمي للتلاعب به.

#### الخطوة 2: تطبيق انتقالات الشرائح

حدد نوع الانتقال الذي تريد تطبيقه. إليك كيفية ضبط التأثيرات المختلفة:

**انتقال الدائرة على الشريحة 1**
```csharp
// يطبق تأثير انتقال الشريحة الدائرية.
pres.Slides[0].SlideShowTransition.Type = TransitionType.Circle;
```
- `TransitionType.Circle` ينشئ رسوم متحركة دائرية سلسة.

**انتقال المشط على الشريحة 2**
```csharp
// يتم تطبيق انتقال يشبه المشط على الشريحة الثانية.
pres.Slides[1].SlideShowTransition.Type = TransitionType.Comb;
```
- `TransitionType.Comb` يقدم تأثير انتقال منظم خطوة بخطوة.

#### الخطوة 3: حفظ العرض التقديمي

بعد تطبيق التحولات، احفظ العرض التقديمي الخاص بك:
```csharp
// يحفظ العرض التقديمي المعدّل مرة أخرى على القرص.
pres.Save(dataDir + "SampleTransition_out.pptx", SaveFormat.Pptx);
```
- `SaveFormat.Pptx` ويضمن بقاء الملف بتنسيق PowerPoint.

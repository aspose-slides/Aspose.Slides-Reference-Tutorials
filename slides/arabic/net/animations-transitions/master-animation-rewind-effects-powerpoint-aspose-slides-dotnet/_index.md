## دليل التنفيذ

### إدارة تأثير إرجاع الرسوم المتحركة

توضح هذه الميزة كيفية تحديد ما إذا كان تأثير الرسوم المتحركة سيتم إرجاعه بعد التشغيل.

**ملخص:**
من خلال ضبط `Rewind` باستخدام هذه الخاصية، يمكنك التحكم في تشغيل الرسوم المتحركة بشكل عكسي بعد انتهائها. هذا مفيد بشكل خاص لتسليط الضوء على النقاط الرئيسية أثناء العرض التقديمي أو لجعل شرائحك أكثر تفاعلية.

#### التنفيذ خطوة بخطوة

**1. قم بتحميل العرض التقديمي الخاص بك**

ابدأ بتحميل ملف PowerPoint حيث تريد إدارة الرسوم المتحركة.
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
using (Presentation presentation = new Presentation(dataDir + "/AnimationRewind.pptx"))
{
    // متابعة خطوات إدارة الرسوم المتحركة...
}
```

**2. الوصول إلى تسلسل الرسوم المتحركة**

استرداد التسلسل الرئيسي للتأثيرات لشريحة معينة، عادةً الشريحة الأولى.
```csharp
ISequence effectsSequence = presentation.Slides[0].Timeline.MainSequence;
```

**3. تكوين خاصية الإرجاع**

حدد تأثيرًا من التسلسل واضبطه `Rewind` الخاصية إلى true. هذا يُفعّل وظيفة الإرجاع.
```csharp
IEffect effect = effectsSequence[0];
effect.Timing.Rewind = true;
```

**4. احفظ عرضك التقديمي**

بعد التكوين، احفظ العرض التقديمي المعدّل في ملف جديد.
```csharp
string outPath = "YOUR_OUTPUT_DIRECTORY";
presentation.Save(outPath + "/AnimationRewind-out.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
```

### قراءة حالة تأثير التراجع للرسوم المتحركة

تتيح لك هذه الميزة التحقق مما إذا كان تأثير الرسوم المتحركة مضبوطًا على التراجع.

**ملخص:**
التحقق من `Rewind` تساعد حالة الخاصية على ضمان سلوك الرسوم المتحركة الخاصة بك كما هو متوقع بعد التعديلات.

#### التنفيذ خطوة بخطوة

**1. قم بتحميل العرض التقديمي المعدّل**

افتح ملف العرض التقديمي الذي تم تعديل الرسوم المتحركة فيه.
```csharp
string outPath = "YOUR_OUTPUT_DIRECTORY";
using (Presentation pres = new Presentation(outPath + "/AnimationRewind-out.pptx"))
{
    // متابعة قراءة حالة الرسوم المتحركة...
}
```

**2. الوصول إلى حالة الإرجاع والتحقق منها**

الوصول إلى التسلسل الرئيسي للشريحة، واسترداد التأثير، والتحقق منه `Rewind` ملكية.
```csharp
ISequence effectsSequence = pres.Slides[0].Timeline.MainSequence;
IEffect effect = effectsSequence[0];
// تأكد إذا كان effect.Timing.Rewind صحيحًا
```

## دليل التنفيذ

### الميزة 1: إنشاء شريحة جديدة

**ملخص:**
إضافة شرائح ذات تخطيطات أو خلفيات محددة أمرٌ أساسي لإنشاء عروض تقديمية احترافية. تتيح لك هذه الميزة إدراج شريحة فارغة وتخصيص لون خلفيتها.

#### الخطوة 1: إنشاء عرض تقديمي جديد
```csharp
Presentation pres = new Presentation();
```

#### الخطوة 2: إضافة شريحة فارغة
```csharp
ISlide slide = pres.Slides.AddEmptySlide(pres.Slides[0].LayoutSlide);
```
*توضيح:* تضيف هذه الخطوة شريحة جديدة بناءً على تخطيط الشريحة الأولى.

#### الخطوة 3: تعيين لون الخلفية
```csharp
slide.Background.FillFormat.FillType = FillType.Solid;
slide.Background.FillFormat.SolidFillColor.Color = Color.YellowGreen;
slide.Background.Type = BackgroundType.OwnBackground;
```
*توضيح:* هنا، قمنا بتعيين لون خلفية ثابت وتحديد أن هذه الشريحة لها خلفية فريدة خاصة بها.

### الميزة 2: إضافة قسم جديد إلى العرض التقديمي

**ملخص:**
تساعد الأقسام على تنظيم الشرائح في مجموعات مفيدة. توضح هذه الميزة كيفية إنشاء قسم جديد مرتبط بشريحة محددة.

#### الخطوة 1: إضافة قسم جديد
```csharp
pres.Sections.AddSection("Section 1", slide);
```
*توضيح:* يؤدي هذا الأمر إلى إنشاء قسم جديد يسمى "القسم 1" وربطه بالشريحة التي تم إنشاؤها مسبقًا.

### الميزة 3: إضافة SectionZoomFrame إلى الشريحة

**ملخص:**
تتيح ميزة SectionZoomFrame للمستخدمين التركيز على أجزاء معينة من العرض التقديمي الخاص بك، مما يعزز التنقل وتجربة المستخدم.

#### الخطوة 1: إضافة SectionZoomFrame
```csharp
ISectionZoomFrame sectionZoomFrame = pres.Slides[0].Shapes.AddSectionZoomFrame(20, 20, 300, 200, pres.Sections[1]);
```
*توضيح:* تضع هذه الخطوة إطار تكبير على الشريحة عند الإحداثيات (20، 20) بحجم 300×200 بكسل وتربطه بالقسم الثاني.

### الميزة 4: حفظ العرض التقديمي

**ملخص:**
بعد تعديل عرضك التقديمي، عليك حفظ هذه التغييرات. توضح الميزة الأخيرة كيفية القيام بذلك بفعالية.

#### الخطوة 1: احفظ العرض التقديمي الخاص بك
```csharp
string resultPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "SectionZoomPresentation.pptx");
pres.Save(resultPath, SaveFormat.Pptx);
```
*توضيح:* سيؤدي هذا إلى حفظ عرضك التقديمي بتنسيق PPTX في مسار الدليل المحدد. استبدل `"YOUR_OUTPUT_DIRECTORY"` مع موقع الحفظ المطلوب.

## دليل التنفيذ: تحريك النص حسب الحرف
في هذا القسم، سنقوم بتفصيل الخطوات اللازمة لتحريك النص حرفًا بحرف باستخدام Aspose.Slides.

### نظرة عامة على ميزة الرسوم المتحركة
يُمكن لتحريك النصوص حرفًا حرفًا أن يُحسّن عروضك التقديمية ويجعلها أكثر جاذبية وتفاعلية. تتيح لك هذه الميزة التحكم في كيفية ظهور كل حرف على الشاشة، مما يُضفي لمسة ديناميكية على شرائحك.

#### الخطوة 1: إنشاء عرض تقديمي جديد
ابدأ بإنشاء مثيل لـ `Presentation`:
```csharp
using (Presentation presentation = new Presentation())
{
    // سيتم تنفيذ خطوات إضافية هنا.
}
```

#### الخطوة 2: إضافة شكل النص
أضف شكلًا، مثل شكل بيضاوي، وأدرج النص الخاص بك:
```csharp
IAutoShape oval = presentation.Slides[0].Shapes.AddAutoShape(ShapeType.Ellipse, 100, 100, 300, 150);
oval.TextFrame.Text = "The new animated text";
```

#### الخطوة 3: الوصول إلى الجدول الزمني للرسوم المتحركة
احصل على إمكانية الوصول إلى الجدول الزمني للشريحة لتطبيق الرسوم المتحركة:
```csharp
IAnimationTimeLine timeline = presentation.Slides[0].Timeline;
```

#### الخطوة 4: إضافة تأثير المظهر باستخدام المشغل
أضف تأثيرًا لجعل النص يظهر عند النقر عليه:
```csharp
IEffect effect = timeline.MainSequence.AddEffect(oval, EffectType.Appear, EffectSubtype.None, EffectTriggerType.OnClick);
```

#### الخطوة 5: تعيين نوع الرسوم المتحركة والتوقيت
قم بتكوين نوع الرسوم المتحركة والتأخير بين الحروف لتحقيق انتقالات سلسة:
```csharp
effect.AnimateTextType = AnimateTextType.ByLetter;
effect.DelayBetweenTextParts = -1.5f; // انتقال فوري
```

### شرح المعلمات
- **نوع النص المتحرك**: يحدد كيفية تحريك النص (`ByLetter` في هذه الحالة).
- **التأخير بين أجزاء النص**:يحدد التأخير بين كل رسوم متحركة للحرف (سلبية للحظة).

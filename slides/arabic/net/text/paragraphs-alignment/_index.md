## تعليمات التثبيت

1. التثبيت عبر مدير حزمة NuGet:
   ```
   Install-Package Aspose.Slides.NET
   ```

2. التثبيت عبر .NET CLI:
   ```
   dotnet add package Aspose.Slides.NET
   ```

3. أضف مرجعًا إلى مشروعك:
   بعد التثبيت، أضف مرجع المساحة الاسمية التالي إلى ملف C# الخاص بك:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

4. احصل على ترخيص مؤقت مجاني:
   لأغراض الاختبار، يمكنك [طلب ترخيص مؤقت](https://purchase.aspose.com/temporary-license/) من موقع Aspose.

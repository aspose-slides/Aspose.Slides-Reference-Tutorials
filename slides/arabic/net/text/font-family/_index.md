# إنشاء مربعات نص منسقة في PowerPoint باستخدام Aspose.Slides

Aspose.Slides for .NET هي واجهة برمجة تطبيقات فعّالة تُمكّن المطورين من إنشاء عروض تقديمية من Microsoft PowerPoint وتعديلها وتحويلها دون الحاجة إلى تثبيت Microsoft Office. يوضح هذا المثال كيفية إنشاء عرض تقديمي بمربع نص مُنسّق، مع استخدام أنماط وأحجام وألوان خطوط مخصصة لتحسين المظهر.

## تعليمات التثبيت

1. التثبيت عبر مدير حزمة NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. التثبيت عبر .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. أضف مرجعًا إلى مشروعك يدويًا:
   - قم بتنزيل أحدث إصدار من [Aspose.Slides لـ .NET](https://releases.aspose.com/slides/net/)
   - أضف DLL كمرجع في مشروعك

4. تهيئة الترخيص:
   ```csharp
   // تهيئة الترخيص
   License license = new License();
   license.SetLicense("Aspose.Slides.lic");
   ```

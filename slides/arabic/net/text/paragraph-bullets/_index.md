# إنشاء نقاط مخصصة في عروض PowerPoint التقديمية

يوفر Aspose.Slides لـ .NET إمكانيات فعّالة لإنشاء عروض PowerPoint التقديمية وتعديلها برمجيًا. يركز هذا المثال تحديدًا على تطبيق نقاط نقطية مخصصة في إطارات النصوص، موضحًا كيفية التحكم في نوع النقاط، ولونها، ومسافة بادئتها، وحجمها. يُنشئ الكود عرضًا تقديميًا بشكل مستطيل يحتوي على فقرتين - إحداهما بنمط نقطي رمزي والأخرى بنمط نقطي مرقم.

## تعليمات التثبيت

1. قم بتثبيت Aspose.Slides لـ .NET باستخدام NuGet Package Manager:
   ```
   Install-Package Aspose.Slides
   ```

2. بدلاً من ذلك، قم بتنزيل الحزمة مباشرة:
   - قم بزيارة [صفحة تنزيل Aspose.Slides](https://downloads.aspose.com/slides/net)
   - تنزيل أحدث إصدار
   - استخرج ملف Aspose.Slides.dll وأشر إليه في مشروعك

3. أضف عبارة الاستخدام الضرورية:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

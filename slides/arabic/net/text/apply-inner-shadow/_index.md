# إنشاء عروض تقديمية في PowerPoint باستخدام TextBox في Aspose.Slides لـ .NET

Aspose.Slides for .NET هي واجهة برمجة تطبيقات فعّالة تُمكّن المطورين من إنشاء عروض PowerPoint التقديمية وتعديلها وتحويلها دون الحاجة إلى Microsoft PowerPoint. يوضح هذا المثال كيفية إنشاء عرض تقديمي بمربع نص مُخصّص، وتطبيق التنسيقات اللازمة، وحفظه كملف PPTX.

## تعليمات التثبيت

1. التثبيت عبر مدير حزمة NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. التثبيت عبر .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. التثبيت يدويًا:
   - قم بتنزيل أحدث إصدار من [صفحة إصدارات Aspose.Slides لـ .NET](https://releases.aspose.com/slides/net/)
   - أضف مرجع DLL إلى مشروعك

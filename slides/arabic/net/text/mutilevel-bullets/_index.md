# إنشاء نقاط متعددة المستويات في PowerPoint باستخدام Aspose.Slides

يوفر Aspose.Slides for .NET واجهة برمجة تطبيقات فعّالة لإنشاء عروض PowerPoint التقديمية والتحكم بها برمجيًا. يوضح هذا المثال كيفية إنشاء عرض تقديمي بنقاط متعددة المستويات، مع توضيح كيفية إضافة شكل تلقائي مع إطار نصي، وتنسيق الفقرات بأنماط نقاط ومستويات مسافة بادئة مختلفة. يمكنك بسهولة إنشاء محتوى هرمي برموز نقاط مخصصة لكل مستوى مسافة بادئة، مما يجعله مثاليًا لإنشاء عروض تقديمية منظمة واحترافية دون الحاجة إلى تثبيت Microsoft PowerPoint.

## تعليمات التثبيت

1. التثبيت عبر مدير حزمة NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. التثبيت عبر .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. التثبيت عبر وحدة تحكم إدارة الحزم:
   ```
   PM> Install-Package Aspose.Slides
   ```

4. التثبيت اليدوي:
   - قم بتنزيل أحدث إصدار من [Aspose.Slides لـ .NET](https://releases.aspose.com/slides/net/)
   - أضف مرجعًا إلى Aspose.Slides.dll في مشروعك

# إنشاء نص عمودي في PowerPoint باستخدام Aspose.Slides

يوفر Aspose.Slides لـ .NET إمكانيات فعّالة لإنشاء عروض PowerPoint التقديمية وتعديلها برمجيًا. يوضح هذا المثال كيفية إنشاء عرض تقديمي يحتوي على شكل مستطيل مع نص مُدار رأسيًا. يتم توجيه النص بزاوية 270 درجة، مما يجعله يُقرأ من الأسفل إلى الأعلى. تُفيد هذه التقنية في إنشاء تسميات جانبية، أو تخطيطات نصية فريدة، أو عناصر تصميم متخصصة في عروضك التقديمية.

## تعليمات التثبيت

1. تثبيت Aspose.Slides لـ .NET
   
   قم بتثبيت حزمة Aspose.Slides باستخدام NuGet Package Manager:
   
   ```csharp
   Install-Package Aspose.Slides
   ```
   
   أو باستخدام .NET CLI:
   
   ```csharp
   dotnet add package Aspose.Slides
   ```

2. إضافة المراجع
   
   أضف العبارة التالية باستخدام إلى ملف C# الخاص بك:
   
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

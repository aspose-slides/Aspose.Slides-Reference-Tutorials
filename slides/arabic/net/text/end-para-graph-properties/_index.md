# إنشاء عروض PowerPoint بنص منسق

Aspose.Slides for .NET هي واجهة برمجة تطبيقات فعّالة تُمكّن المطورين من إنشاء عروض PowerPoint التقديمية وتعديلها وتحويلها دون الحاجة إلى Microsoft PowerPoint. يوضح هذا المثال كيفية إنشاء عرض تقديمي بسيط يحتوي على شكل مستطيل مع فقرات نصية منسقة.

يُنشئ الكود عرضًا تقديميًا جديدًا، ويضيف شكلًا مستطيلًا إلى الشريحة الأولى، ثم يُضيف فقرات متعددة بخيارات تنسيق نصية مختلفة. يُظهر كيفية ضبط خصائص الخط، مثل الحجم وفئة الخط، لأجزاء محددة من النص.

## تعليمات التثبيت

1. التثبيت عبر مدير حزمة NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. التثبيت عبر .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. التثبيت عبر وحدة تحكم إدارة الحزم:
   ```
   PM> Install-Package Aspose.Slides
   ```

4. أضف مرجعًا إلى مشروعك:
   بعد التثبيت، أضف مرجعًا إلى مشروعك:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

# تدوير عناوين المخططات والملصقات في PowerPoint باستخدام Aspose.Slides

## تعليمات التثبيت

1. التثبيت عبر مدير حزمة NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. أو قم بالتثبيت عبر .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. أضف مساحة الاسم إلى ملف C# الخاص بك:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Charts;
   ```

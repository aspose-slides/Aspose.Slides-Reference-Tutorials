## دليل التنفيذ

### إنشاء دليل إذا لم يكن موجودًا

**ملخص:**
تضمن هذه الميزة وجود دليل لتخزين المستندات، وإنشاء دليل إذا لزم الأمر.

#### الخطوة 1: تحديد دليل المستندات الخاص بك
أولاً، قم بتحديد مسار دليل المستند الخاص بك في متغير.
```csharp
string dataDir = @"YOUR_DOCUMENT_DIRECTORY";
```

#### الخطوة 2: التحقق من الدليل وإنشائه
يستخدم `Directory.Exists` للتحقق من وجود الدليل. إذا لم يكن موجودًا، فأنشئه باستخدام `Directory.CreateDirectory`.
```csharp
bool isExists = Directory.Exists(dataDir);
if (!isExists)
{
    // يؤدي هذا إلى إنشاء دليل جديد في المسار المحدد إذا لم يكن موجودًا بالفعل.
    Directory.CreateDirectory(dataDir);
}
```
**المعايير والغرض:**
- `dataDir`:مسار الدليل المستهدف. 
- `Directory.Exists`:ترجع القيمة true إذا كان الدليل موجودًا.
- `Directory.CreateDirectory`:إنشاء الدليل المحدد بواسطة المسار.

### إنشاء عرض تقديمي وإضافة شكل مستطيل مع النص

**ملخص:**
توضح هذه الميزة كيفية إنشاء عرض تقديمي جديد وإضافة شكل مستطيل وتضمين نص داخله باستخدام Aspose.Slides لـ .NET.

#### الخطوة 1: إنشاء عرض تقديمي
إنشاء مثيل لـ `Presentation` الذي يمثل ملف PowerPoint الخاص بك.
```csharp
string outputDir = @"YOUR_OUTPUT_DIRECTORY";
using (Presentation pres = new Presentation())
{
    // الوصول إلى الشريحة الأولى من العرض التقديمي
    ISlide sld = pres.Slides[0];
```

#### الخطوة 2: إضافة شكل مستطيل
أضف شكلًا تلقائيًا من نوع المستطيل إلى الشريحة الخاصة بك.
```csharp
    IAutoShape ashp = sld.Shapes.AddAutoShape(ShapeType.Rectangle, 150, 75, 150, 50);
    // يؤدي هذا إلى إضافة مستطيل في الموضع المحدد بالأبعاد المحددة (العرض والارتفاع).
```

#### الخطوة 3: إدراج النص في الشكل
قم بإنشاء إطار نص وأضف نصًا إلى الشكل الخاص بك.
```csharp
    ashp.AddTextFrame(" ");
    ITextFrame txtFrame = ashp.TextFrame;
    IParagraph para = txtFrame.Paragraphs[0];
    IPortion portion = para.Portions[0];
    portion.Text = "Aspose TextBox";
    // قم بوضع النص داخل شكل المستطيل.
```

#### الخطوة 4: حفظ العرض التقديمي
وأخيرًا، احفظ العرض التقديمي في الموقع المطلوب.
```csharp
    pres.Save(outputDir + "TextBox_out.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
}
// يؤدي هذا إلى حفظ الملف بتنسيق PPTX بالاسم المحدد.
```

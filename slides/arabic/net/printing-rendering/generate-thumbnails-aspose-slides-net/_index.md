## دليل التنفيذ

يرشدك هذا القسم خلال إنشاء صورة مصغرة للشكل الأول على شريحة العرض التقديمي الخاصة بك.

### إنشاء صورة مصغرة من شكل الشريحة
يعد إنشاء معاينة صورة (صورة مصغرة) لأشكال محددة داخل الشرائح مفيدًا لتطبيقات الويب التي تحتاج إلى معاينات سريعة أو عند إدارة العروض التقديمية الكبيرة.

#### الخطوة 1: إعداد الدلائل وملف العرض التقديمي
قم بتحديد المسارات لمستند الإدخال ودليل الإخراج:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // استبدل بالمسار إلى دليل المستندات الخاص بك
string outputDir = "YOUR_OUTPUT_DIRECTORY"; // استبدل بالمسار إلى دليل الإخراج المطلوب
```

#### الخطوة 2: تحميل العرض التقديمي
إنشاء مثيل `Presentation` الفئة التي تمثل ملف العرض التقديمي الخاص بك:
```csharp
using (Presentation p = new Presentation(dataDir + "/HelloWorld.pptx"))
{
    // الوصول إلى الشريحة الأولى في العرض التقديمي
    ISlide slide = p.Slides[0];
```

#### الخطوة 3: الوصول إلى الشكل وتحويله إلى صورة
قم بالوصول إلى الشكل الأول في الشريحة الخاصة بك وتحويله إلى صورة:
```csharp
    IShape shape = slide.Shapes[0];

    using (IImage img = shape.GetImage(ShapeThumbnailBounds.Shape, 1, 1))
    {
        // احفظ الصورة المصغرة الناتجة على القرص بتنسيق PNG
        img.Save(outputDir + "/Scaling Factor Thumbnail_out.png");
    }
}
```

**توضيح:**
- `GetImage` يلتقط صورة كاملة لشكل جسمك. المعلمات `(ShapeThumbnailBounds.Shape, 1, 1)` تحديد التقاط الشكل بأكمله دون تغيير الحجم.

#### نصائح استكشاف الأخطاء وإصلاحها
- تأكد من تعيين مسارات الملفات بشكل صحيح وإمكانية الوصول إليها بواسطة تطبيقك.
- التحقق من وجود استثناءات متعلقة بالوصول إلى الملفات أو تنسيقات العرض غير الصالحة.

## دليل التنفيذ
الآن بعد أن قمت بالإعداد، دعنا ننتقل إلى عرض الصور المصغرة للشرائح باستخدام الخطوط المخصصة.

### نظرة عامة على الميزة: عرض الصور المصغرة باستخدام الخطوط المخصصة
تتيح لك هذه الميزة عرض الشريحة الأولى من العرض التقديمي كصورة باستخدام إعدادات خط محددة. وهي مفيدة بشكل خاص لأغراض بناء العلامة التجارية وضمان الاتساق بين العروض التقديمية.

#### الخطوة 1: تحميل العرض التقديمي الخاص بك
ابدأ بتحميل ملف PowerPoint الخاص بك إلى `Presentation` هدف:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string presPath = Path.Combine(dataDir, "RenderingOptions.pptx");
using (Presentation pres = new Presentation(presPath))
{
    // متابعة إعدادات العرض
}
```

#### الخطوة 2: تكوين خيارات العرض
قم بتعيين الخط المطلوب كخط افتراضي للرسم:
```csharp
IRenderingOptions renderingOpts = new RenderingOptions();
renderingOpts.DefaultRegularFont = "Arial Black";
```
تضمن هذه الخطوة أن النص الموجود في الصورة المقدمة يتطابق مع علامتك التجارية أو دليل الأسلوب.

#### الخطوة 3: عرض الشريحة وحفظها
استخدم `GetImage` طريقة عرض الشريحة وحفظها كصورة:
```csharp
double aspectRatio = 4 / 3.0;
pres.Slides[0].GetImage(renderingOpts, aspectRatio, aspectRatio)
    .Save(Path.Combine("YOUR_OUTPUT_DIRECTORY", "output.png"), ImageFormat.Png);
```
هنا، `aspectRatio` يمثل أبعاد الصورة. عدّلها حسب الحاجة لتناسب احتياجاتك.

### نصائح استكشاف الأخطاء وإصلاحها
- **الخطوط المفقودة:** تأكد من تثبيت الخط المحدد على نظامك.
- **مشاكل مسار الملف:** تأكد من التحقق من مسارات الدليل بحثًا عن الأخطاء المطبعية أو أذونات الوصول.
- **أخطاء تنسيق الصورة:** تأكد من أنك تستخدم تنسيق الصورة المدعوم في `Save()`.

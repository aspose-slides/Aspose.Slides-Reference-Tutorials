## دليل التنفيذ

بعد إعداد Aspose.Slides، دعنا ننتقل إلى إنشاء الصور المصغرة للشرائح:

### إنشاء صورة مصغرة من الشريحة الأولى

#### ملخص
إنشاء صورة مصغرة للشريحة الأولى لأغراض المعاينة أو الفهرسة.

##### الخطوة 1: إعداد مسارات الدليل
تحديد المسارات لملفات الإدخال والإخراج.
```csharp
dirInput = "YOUR_DOCUMENT_DIRECTORY"; // مسار ملف الإدخال
dirOutput = "YOUR_OUTPUT_DIRECTORY"; // مسار الصورة الناتجة
```

##### الخطوة 2: تحميل العرض التقديمي
إنشاء `Presentation` كائن للعمل مع ملف PowerPoint الخاص بك.
```csharp
using (Presentation pres = new Presentation(dirInput + "/ThumbnailFromSlide.pptx"))
{
    ...
}
```
ال `using` يضمن البيان التخلص السليم من الموارد.

##### الخطوة 3: الوصول إلى الشريحة الأولى وإنشاء صورة
انتقل إلى الشريحة الأولى لإنشاء صورة بالحجم الكامل.
```csharp
ISlide sld = pres.Slides[0];
IImage img = sld.GetThumbnail(1f, 1f); // العرض والارتفاع بالحجم الكامل
```
المعلمات `(1f, 1f)` تمثل عوامل القياس للعرض والارتفاع.

##### الخطوة 4: حفظ الصورة المصغرة
احفظ الصورة الناتجة بتنسيق JPEG.
```csharp
img.Save(dirOutput + "/Thumbnail_out.jpg", System.Drawing.Imaging.ImageFormat.Jpeg);
```

#### نصائح استكشاف الأخطاء وإصلاحها
- تأكد من تعيين مسارات الملفات بشكل صحيح وإمكانية الوصول إليها.
- التحقق من وجود استثناءات متعلقة بالأذونات أو التنسيقات غير الصحيحة.

## دليل التنفيذ

دعونا نستعرض عملية استيراد ملف PDF إلى PowerPoint باستخدام Aspose.Slides.

### استيراد ملف PDF إلى PowerPoint

**ملخص:**
تتيح لك هذه الميزة تحويل كل صفحة من مستند PDF إلى شرائح منفصلة في عرض تقديمي على PowerPoint. كما تُسهّل إضافة مستندات معقدة إلى عروضك التقديمية دون الحاجة إلى إدخال يدوي.

#### التنفيذ خطوة بخطوة

##### إعداد المسارات

قم بتحديد المسارات لملف PDF المدخل وملف PPTX المخرج:

```csharp
using System.IO;

string pdfFileName = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "welcome-to-powerpoint.pdf");
string resultPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "fromPdfDocument.pptx");
```

**توضيح:** يستبدل `"YOUR_DOCUMENT_DIRECTORY"` و `"YOUR_OUTPUT_DIRECTORY"` مع المسارات الفعلية على نظامك.

##### تهيئة العرض التقديمي

إنشاء مثيل عرض تقديمي جديد لحمل الشرائح المستوردة:

```csharp
using (Presentation pres = new Presentation())
{
    // سيتم تنفيذ الخطوات التالية هنا.
}
```

**ملحوظة:** ال `using` يضمن البيان التخلص من الموارد بشكل صحيح بعد الاستخدام.

##### إضافة شرائح PDF

أضف الشرائح من مستند PDF إلى العرض التقديمي الخاص بك:

```csharp
pres.Slides.AddFromPdf(pdfFileName);
```

**الرؤية الرئيسية:** تقوم هذه الطريقة بتحويل كل صفحة في ملف PDF المحدد إلى شريحة وإضافتها إلى نهاية مجموعة الشرائح الحالية.

##### حفظ العرض التقديمي

وأخيرًا، احفظ العرض التقديمي المعدّل كملف PPTX:

```csharp	pres.Save(resultPath, Aspose.Slides.Export.SaveFormat.Pptx);
```

**لماذا هذا مهم:** التوفير في `SaveFormat.Pptx` يضمن أن يكون الناتج الخاص بك متوافقًا مع تطبيقات PowerPoint.

## دليل التنفيذ
سوف يرشدك هذا القسم خلال الميزات المختلفة التي يوفرها Aspose.Slides لـ .NET.
### تحميل عرض تقديمي باستخدام الخطوط الافتراضية (H2)
#### ملخص:
يضمن تحميل العروض التقديمية بخطوط مخصصة الاتساق، خاصةً عندما تختلف الخطوط الافتراضية بين الأنظمة. تتيح لك هذه الميزة تحديد الخطوط الافتراضية العادية والآسيوية.
**خطوات التنفيذ:**
##### 1. تحديد مسار المستند
قم بتعيين المسار الذي سيتم تخزين ملف العرض التقديمي الخاص بك فيه.
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
```
##### 2. إنشاء خيارات التحميل
يستخدم `LoadOptions` لتحديد الخطوط الافتراضية المطلوبة.
```csharp
LoadOptions loadOptions = new LoadOptions(LoadFormat.Auto);
loadOptions.DefaultRegularFont = "Wingdings"; // الخط العادي
loadOptions.DefaultAsianFont = "Wingdings";   // الخط الآسيوي
```
##### 3. تحميل العرض التقديمي
استخدم المحدد `LoadOptions` لفتح ملف العرض التقديمي الخاص بك.
```csharp
using (Presentation pptx = new Presentation(dataDir + "/DefaultFonts.pptx", loadOptions))
{
    // التعامل مع العرض التقديمي المحمّل حسب الحاجة
}
```
**توضيح**:من خلال تعيين الخطوط الافتراضية، يمكنك التأكد من أنه حتى لو كانت بعض الخطوط مفقودة على النظام، فسيتم استخدام Wingdings بدلاً منها.
### إنشاء صورة مصغرة للشريحة (H2)
#### ملخص:
يعد إنشاء صور مصغرة للشرائح مفيدًا لأغراض المعاينات أو الفهرسة في تطبيقاتك.
**خطوات التنفيذ:**
##### 1. تحديد مسار الإخراج
قم بتعيين الدليل الذي سيتم حفظ الصورة المصغرة فيه.
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
```
##### 2. إنشاء صورة مصغرة
قم بإنشاء كائن خريطة نقطية لالتقاط الصورة المصغرة للشريحة الأولى.
```csharp
int width = 1, height = 1; // أبعاد الصورة المصغرة
Bitmap bitmap = pptx.Slides[0].GetThumbnail(width, height);
bitmap.Save(outputDir + "/output_out.png", ImageFormat.Png); // حفظ بصيغة PNG
```
**توضيح**: ال `GetThumbnail` تلتقط الطريقة الشريحة عند أبعاد محددة.
### تصدير العرض التقديمي إلى PDF (H2)
#### ملخص:
يضمن تصدير العروض التقديمية إلى PDF إمكانية عرض الشرائح الخاصة بك على أي جهاز دون الحاجة إلى برنامج PowerPoint.
**خطوات التنفيذ:**
##### 1. تحديد مسار الإخراج
أشر إلى المكان الذي سيتم حفظ ملف PDF فيه.
```csharp
string pdfOutputDir = "YOUR_OUTPUT_DIRECTORY";
```
##### 2. تصدير إلى PDF
احفظ العرض التقديمي كمستند PDF.
```csharp
pptx.Save(pdfOutputDir + "/output_out.pdf", SaveFormat.Pdf);
```
**توضيح**: ال `Save` تقوم هذه الطريقة بتحويل العرض التقديمي الخاص بك إلى تنسيق PDF يمكن الوصول إليه عالميًا.
### تصدير العرض التقديمي إلى XPS (H2)
#### ملخص:
يعد تصدير العروض التقديمية إلى XPS مفيدًا للحفاظ على دقة المستندات وتوافقها مع أنظمة Windows.
**خطوات التنفيذ:**
##### 1. تحديد مسار الإخراج
قم بتعيين الدليل لحفظ ملف XPS.
```csharp
string xpsOutputDir = "YOUR_OUTPUT_DIRECTORY";
```
##### 2. التصدير إلى XPS
احفظ العرض التقديمي بتنسيق XPS.
```csharp
pptx.Save(xpsOutputDir + "/output_out.xps", SaveFormat.Xps);
```
**توضيح**:تضمن هذه الطريقة احتفاظ مستندك بتخطيطه وتنسيقه عبر منصات مختلفة.

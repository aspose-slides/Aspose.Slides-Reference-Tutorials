### تحميل العرض التقديمي باستخدام الخطوط المخصصة

#### ملخص

يضمن تحميل العروض التقديمية بخطوط محددة عرض النص بدقة كما هو مقصود. هذا أمر بالغ الأهمية للحفاظ على سلامة العلامة التجارية والاتساق البصري في جميع المستندات.

#### خطوات

**1. تحديد دليل المستندات**

أولاً، حدد مكان وجود ملفاتك:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
```

**2. تحميل الخطوط إلى الذاكرة**

قم بتحميل الخطوط المخصصة من التخزين المحلي إلى الذاكرة للتأكد من توفرها عند الحاجة إليها:

```csharp
byte[] memoryFont1 = File.ReadAllBytes("customfonts\\CustomFont1.ttf");
byte[] memoryFont2 = File.ReadAllBytes("customfonts\\CustomFont2.ttf");
```

**3. إعداد خيارات التحميل**

قم بتكوين خيارات التحميل لتحديد مصادر الخط:

```csharp
LoadOptions loadOptions = new LoadOptions();
loadOptions.DocumentLevelFontSources.FontFolders = new string[] { "assets\\fonts", "global\\fonts" };
loadOptions.DocumentLevelFontSources.MemoryFonts = new byte[][] { memoryFont1, memoryFont2 };
```

**4. تحميل العرض التقديمي**

بعد إعداد الخطوط وتكوين خيارات التحميل، يمكنك الآن تحميل العرض التقديمي الخاص بك:

```csharp
using (IPresentation presentation = new Presentation("MyPresentation.pptx", loadOptions))
{
    // يتم تحميل العرض التقديمي باستخدام الخطوط المخصصة المحددة.
}
```

#### توضيح

- **`LoadOptions`:** تعيين أدلة مصدر الخط والخطوط المحملة بالذاكرة.
- **`MemoryFonts`:** مجموعة من المصفوفات البايتية تمثل الخطوط المحملة في الذاكرة.

لاستخدام Aspose.Slides، قم بتثبيت المكتبة باستخدام إحدى الطرق التالية:

**استخدام .NET CLI:**
```bash
dotnet add package Aspose.Slides
```

**استخدام وحدة تحكم إدارة الحزم:**

## دليل التنفيذ

### تحويل PPTX إلى XPS بدون خيارات إضافية

يرشدك هذا القسم خلال عملية تحويل عرض تقديمي بتنسيق PowerPoint إلى مستند XPS باستخدام الإعدادات الافتراضية.

#### الخطوة 1: تحميل ملف العرض التقديمي

ابدأ بتحميل ملف .pptx. هذه الخطوة تُمهّد الطريق `Presentation` الكائن مع مسار الملف المحدد:
```csharp
using Aspose.Slides;

string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // استبدال بالدليل الفعلي
string outputDir = "YOUR_OUTPUT_DIRECTORY"; // مسار الإخراج المطلوب

// تحميل العرض التقديمي
using (Presentation pres = new Presentation(dataDir + "/Convert_XPS.pptx"))
{
    // سيتم معالجة هذا العرض التقديمي المحمّل بالخطوات التالية.
}
```

#### الخطوة 2: الحفظ كمستند XPS

بمجرد تحميل العرض التقديمي الخاص بك، احفظه بتنسيق XPS باستخدام `SaveFormat.Xps`:
```csharp
using Aspose.Slides.Export;

// حفظ العرض التقديمي كمستند XPS بدون خيارات إضافية
pres.Save(outputDir + "/XPS_Output_Without_XPSOption_out.xps", SaveFormat.Xps);
```

### الاعتبارات الرئيسية
- **متغيرات المسار:** يضمن `dataDir` و `outputDir` تم تعيينها بشكل صحيح إلى مسارات الملفات الخاصة بك.
- **أذونات الملف:** تأكد من أن تطبيقك لديه الأذونات اللازمة للقراءة من هذه الدلائل والكتابة إليها.

## دليل التنفيذ

### حفظ العروض التقديمية بدون روابط JavaScript

ترتكز هذه الميزة على حفظ ملفات PPTX مع استبعاد الروابط التشعبية التي تحتوي على JavaScript، مما يعزز الأمان.

#### الخطوة 1: إعداد مسارات الملفات

قم بتحديد مسارات المصدر والإخراج لملف PowerPoint الخاص بك.
```csharp
string pptxFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "JavaScriptLink.pptx");
string resultPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "JavaScriptLink-out.pptx");
```

#### الخطوة 2: تحميل العرض التقديمي

قم بتحميل العرض التقديمي باستخدام Aspose.Slides، وإعداده للتعامل معه.
```csharp
using (Presentation pres = new Presentation(pptxFile))
{
    // سوف تتم المعالجة الإضافية هنا
}
```

#### الخطوة 3: الحفظ مع تخطي روابط JavaScript

احفظ العرض التقديمي مع التأكد من تخطي جميع روابط JavaScript لتعزيز الأمان عن طريق منع تنفيذ البرنامج النصي المحتمل.
```csharp
pres.Save(resultPath, SaveFormat.Pptx, new PptxOptions()
{
    SkipJavaScriptLinks = true
});
```
- **المعلمات موضحة:** 
  - `resultPath`:مسار الوجهة للملف المحفوظ.
  - `SaveFormat.Pptx`:يحدد الحفظ بتنسيق PowerPoint.
  - `PptxOptions.SkipJavaScriptLinks`:تكوين مهم لتعزيز الأمان.

**نصائح استكشاف الأخطاء وإصلاحها:**
- تأكد من تعيين المسارات بشكل صحيح، ووجود الدلائل.
- تأكد من أن لديك أذونات الكتابة لدليل الإخراج.

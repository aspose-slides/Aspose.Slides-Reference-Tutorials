## دليل التنفيذ

يوضح هذا القسم كيفية إضافة إطار صوتي يحتوي على صوت مضمن في العرض التقديمي.

### إضافة إطار صوتي

#### ملخص

يُمكن أن يُعزز تضمين الصوت تفاعلية عروضك التقديمية، ويجعلها أكثر جاذبية. سنشرح كيفية إنشاء ملف صوتي وتضمينه في شريحة باستخدام Aspose.Slides لـ .NET.

#### التنفيذ خطوة بخطوة

##### 1. تحميل أو إنشاء عرض تقديمي

ابدأ بتحميل عرض تقديمي موجود أو إنشاء عرض تقديمي جديد:

```csharp
// إنشاء عرض تقديمي جديد أو تحميل عرض تقديمي موجود
Presentation pres = new Presentation();
```

##### 2. الوصول إلى الشريحة

حدد الشريحة التي تريد تضمين الصوت فيها:

```csharp
ISlide slide = pres.Slides[0]; // الوصول إلى الشريحة الأولى
```

##### 3. إضافة إطار صوتي

إليك كيفية إضافة إطار صوتي مع الصوت المضمن:

```csharp
// تحديد المسار لوسائط الإدخال وملف الإخراج
string mediaFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "audio.mp3");

// تحميل ملف الصوت إلى FileStream
using (FileStream fs = new FileStream(mediaFile, FileMode.Open))
{
    // إضافة إطار صوتي إلى الشريحة
    IAudioFrame audioFrame = slide.Shapes.AddAudioFrameEmbedded(50, 150, 100, 100, fs);
    
    // تكوين خصائص الصوت إذا لزم الأمر
    audioFrame.PlayMode = AudioPlayModePreset.OnClick;
}
```

**توضيح:**
- **إضافة إطار صوتي مضمن**تضيف هذه الطريقة إطارًا صوتيًا إلى الشريحة. تُحدد المعلمات موضع وحجم الإطار على الشريحة.
- **وضع اللعب**:يقوم بتكوين كيفية تشغيل الصوت، مثل البدء تلقائيًا أو عند النقر فوقه.

#### نصائح استكشاف الأخطاء وإصلاحها

- تأكد من أن مسار ملف الوسائط صحيح ويمكن الوصول إليه.
- التحقق من وجود أي استثناءات متعلقة بعمليات إدخال/إخراج الملفات ومعالجتها بشكل مناسب.

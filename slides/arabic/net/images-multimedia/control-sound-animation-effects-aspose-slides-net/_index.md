# كيفية التحكم في الصوت في الرسوم المتحركة في PowerPoint باستخدام Aspose.Slides .NET

## دليل التنفيذ
في هذا القسم، سنقوم بتفصيل كيفية التحكم في الصوت في تأثيرات الرسوم المتحركة باستخدام `StopPreviousSound` ملكية.

### فهم ميزة StopPreviousSound
ال `StopPreviousSound` تتيح لك خاصية التأثير إدارة الأصوات المتداخلة في عروضك التقديمية. عند ضبطها على "صحيح"، تُوقف أي صوت سابق عند تشغيل تأثير جديد، مما يضمن تشغيل صوت واحد فقط في كل مرة.

#### التنفيذ خطوة بخطوة:
**تحميل العرض التقديمي**
أولاً، قم بتحميل ملف العرض التقديمي الخاص بك حيث تريد التحكم في تأثيرات الرسوم المتحركة:

```csharp
string pptxFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "AnimationStopSound.pptx");

using (Presentation pres = new Presentation(pptxFile))
{
    // الكود سوف يذهب هنا
}
```

**الوصول إلى تأثيرات الرسوم المتحركة**
بعد ذلك، يمكنك الوصول إلى تأثيرات الرسوم المتحركة على شرائحك. هنا، نركز على الوصول إلى تأثيرات محددة وتعديلها:

```csharp
// يتم الوصول إلى التأثير الأول للتسلسل الرئيسي على الشريحة الأولى.
IEffect firstSlideEffect = pres.Slides[0].Timeline.MainSequence[0];

// يتم الوصول إلى التأثير الأول للتسلسل الرئيسي على الشريحة الثانية.
IEffect secondSlideEffect = pres.Slides[1].Timeline.MainSequence[0];
```

**ضبط إيقاف الصوت السابق**
تحقق مما إذا كان هناك صوت مرتبط بالرسوم المتحركة واضبطه `StopPreviousSound` وفقاً لذلك:

```csharp
// يتحقق مما إذا كان تأثير الشريحة الأولى يحتوي على صوت مرتبط به.
if (firstSlideEffect.Sound != null)
{
    // يوقف الأصوات السابقة عند تفعيل هذا التأثير.
    secondSlideEffect.StopPreviousSound = true;
}
```

**حفظ التغييرات**
وأخيرًا، احفظ العرض التقديمي المعدّل في مسار ملف جديد:

```csharp
string outPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "AnimationStopSound-out.pptx");
pres.Save(outPath, SaveFormat.Pptx);
```

### نصائح استكشاف الأخطاء وإصلاحها
- تأكد من أن المسارات لـ `pptxFile` و `outPath` هي صحيحة.
- تأكد من أن ملف العرض التقديمي الخاص بك يحتوي على شريحتين على الأقل مع التأثيرات لاختبار هذه الميزة.

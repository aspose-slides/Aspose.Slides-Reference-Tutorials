## دليل التنفيذ

الآن بعد أن قمت بإعداد بيئتك، دعنا ننفذ وظيفة تحميل الصور وعرضها.

### الميزة: تحميل الصور وعرضها في العرض التقديمي

توضح هذه الميزة كيفية تحميل صورة من نظام الملفات وإضافتها كإطار صورة إلى الشريحة الأولى من العرض التقديمي باستخدام Aspose.Slides لـ .NET.

#### ملخص:
في هذا القسم، سنتناول خطوات تحميل صورة، وإدراجها في شريحة، وحفظ العرض التقديمي الخاص بك.

**الخطوة 1: إنشاء الدلائل**
حدّد مساراتٍ لدليل المستندات ودليل المخرجات. إذا لم تكن موجودة، فأنشئها باستخدام:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // قم بتحديد مسار دليل المستند الخاص بك هنا
string outputDir = "YOUR_OUTPUT_DIRECTORY"; // قم بتحديد مسار دليل الإخراج الخاص بك هنا

// إنشاء دليل البيانات إذا لم يكن موجودًا.
if (!Directory.Exists(dataDir))
{
    Directory.CreateDirectory(dataDir);
}
```

**الخطوة 2: تحميل الصورة وإدراجها**
أنشئ عرضًا تقديميًا جديدًا، وافتح الشريحة الأولى منه. ثم حمّل صورة من نظام الملفات:
```csharp
using (Presentation pres = new Presentation())
{
    // الوصول إلى الشريحة الأولى في العرض التقديمي
    ISlide sld = pres.Slides[0];

    // قم بتحميل صورة من نظام الملفات وأضفها إلى مجموعة صور العرض التقديمي
    IImage img = Images.FromFile(Path.Combine(dataDir, "aspose-logo.jpg"));
    IPPImage imgx = pres.Images.AddImage(img);

    // أضف إطار صورة بأبعاد مطابقة لأبعاد الصورة المحملة
    sld.Shapes.AddPictureFrame(ShapeType.Rectangle, 50, 150, imgx.Width, imgx.Height, imgx);
}
```

**الخطوة 3: حفظ العرض التقديمي**
وأخيرًا، احفظ العرض التقديمي المعدّل على القرص بتنسيق PPTX:
```csharp
pres.Save(Path.Combine(outputDir, "AddStretchOffsetForImageFill_out.pptx"), SaveFormat.Pptx);
```

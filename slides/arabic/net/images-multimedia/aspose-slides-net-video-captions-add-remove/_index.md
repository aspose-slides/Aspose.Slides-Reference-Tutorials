## دليل التنفيذ
سيرشدك هذا القسم إلى كيفية إضافة التعليقات التوضيحية إلى إطارات الفيديو واستخراجها أو إزالتها. ستجد شرحًا مفصلاً لكل ميزة أدناه.

### الميزة 1: إضافة التعليقات التوضيحية إلى VideoFrame

#### ملخص
تساعد هذه الميزة على إدراج التعليقات التوضيحية من ملف خارجي (مثل VTT) في إطار الفيديو، مما يعزز إمكانية الوصول لجمهورك.

#### خطوات التنفيذ
**الخطوة 1: تحضير ملفاتك**
تأكد من أن لديك الفيديو (`sample_bunny.mp4`) وملفات مسار التسمية التوضيحية (`bunny.vtt`).

```csharp
string mediaFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample_bunny.mp4");
string trackFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "bunny.vtt");
```

**الخطوة 2: إضافة الفيديو إلى العرض التقديمي**
إنشاء `Presentation` الكائن وأضف الفيديو الخاص بك.

```csharp
using (Presentation pres = new Presentation())
{
    IVideo video = pres.Videos.AddVideo(File.ReadAllBytes(mediaFile));
    var videoFrame = pres.Slides[0].Shapes.AddVideoFrame(0, 0, 100, 100, video);
```

**الخطوة 3: إضافة مسار التسمية التوضيحية**
قم بإرفاق ملف مسار التسمية التوضيحية الخاص بك بإطار الفيديو.

```csharp
videoFrame.CaptionTracks.Add("New track", trackFile);
pres.Save(Path.Combine("YOUR_OUTPUT_DIRECTORY", "VideoCaptionAdd_out.pptx"), SaveFormat.Pptx);
}
```

#### المعلمات وأغراض الطريقة
- `Presentation`:يمثل عرض تقديمي في PowerPoint.
- `IVideo` و `IVideoFrame`:تمثيل محتوى الفيديو وإطاره ضمن الشرائح، على التوالي.
- `captionTracks.Add()`:يضيف التسميات التوضيحية إلى المسار المحدد.

### الميزة 2: استخراج وإزالة التسميات التوضيحية من VideoFrame

#### ملخص
بعد إضافة التسميات التوضيحية، قد تحتاج في بعض الحالات إلى استخراجها أو إزالتها. تُركز هذه الميزة على كيفية تحقيق كلتا المهمتين بفعالية.

#### خطوات التنفيذ
**الخطوة 1: تحميل العرض التقديمي**
افتح العرض التقديمي الذي يحتوي على الفيديو الخاص بك مع التعليقات التوضيحية.

```csharp
string outAddPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "VideoCaptionAdd_out.pptx");
using (Presentation pres = new Presentation(outAddPath))
{
    IVideoFrame videoFrame = pres.Slides[0].Shapes[0] as VideoFrame;
```

**الخطوة 2: استخراج التسميات التوضيحية**
استخرج البيانات الثنائية للتسميات التوضيحية واحفظها في ملف.

```csharp
if (videoFrame != null)
{
    foreach (var captionTrack in videoFrame.CaptionTracks) 
    {
        File.WriteAllBytes(Path.Combine("YOUR_OUTPUT_DIRECTORY", "Caption_out.vtt"), captionTrack.BinaryData);
    }
```

**الخطوة 3: إزالة التسميات التوضيحية**
مسح جميع التسميات التوضيحية من VideoFrame.

```csharp
videoFrame.CaptionTracks.Clear();
pres.Save(Path.Combine("YOUR_OUTPUT_DIRECTORY", "VideoCaptionRemove_out.pptx"), SaveFormat.Pptx);
}
```

#### المعلمات وأغراض الطريقة
- `BinaryData`:يمثل بيانات التسمية التوضيحية بتنسيق ثنائي.
- `CaptionTracks.Clear()`:يزيل جميع التسميات التوضيحية من إطار الفيديو.

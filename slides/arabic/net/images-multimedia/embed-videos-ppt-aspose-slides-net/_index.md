### تضمين إطار الفيديو في العرض التقديمي (الميزة 2)

#### ملخص
قم بتضمين مقاطع الفيديو في شرائح PowerPoint باستخدام Aspose.Slides لـ .NET، مما يجعل العروض التقديمية أكثر ديناميكية وتفاعلية.

**تهيئة العرض التقديمي**
```csharp
using Aspose.Slides;
using System.IO;

string videoDir = "YOUR_DOCUMENT_DIRECTORY"; // الدليل الذي يحتوي على ملف الفيديو الخاص بك
string resultPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "VideoFrame_out.pptx");

// إنشاء مثيل عرض تقديمي جديد
using (Presentation pres = new Presentation())
{
    // احصل على الشريحة الأولى في العرض التقديمي
    ISlide sld = pres.Slides[0];

    // افتح ملف الفيديو وأضفه إلى العرض التقديمي
    IVideo vid = pres.Videos.AddVideo(new FileStream(videoDir + "/Wildlife.mp4", FileMode.Open), LoadingStreamBehavior.ReadStreamAndRelease);
    
    // إضافة إطار فيديو جديد إلى الشريحة بالموضع والحجم المحددين
    IVideoFrame vf = sld.Shapes.AddVideoFrame(50, 150, 300, 350, vid);
    
    // تعيين الفيديو المضمن لإطار الفيديو
    vf.EmbeddedVideo = vid;
    
    // ضبط وضع تشغيل الفيديو ومستوى الصوت
    vf.PlayMode = VideoPlayModePreset.Auto;
    vf.Volume = AudioVolumeMode.Loud;
    
    // احفظ العرض التقديمي باستخدام إطار الفيديو المضمن
    pres.Save(resultPath, SaveFormat.Pptx);
}
```

**توضيح:**
- `Presentation`:يمثل ملف PowerPoint.
- `IVideo`:واجهة للتعامل مع ملفات الفيديو في العروض التقديمية.
- `AddVideo()`:إضافة ملف فيديو إلى العرض التقديمي.
- `AddVideoFrame()`:إدراج إطار في الشريحة لحمل الفيديو.
- `PlayMode` و `Volume`:تكوين إعدادات التشغيل.

**نصائح استكشاف الأخطاء وإصلاحها:**
- تأكد من أن مسار الفيديو الخاص بك صحيح؛ استخدم المسارات المطلقة لتحقيق الموثوقية.
- تعامل مع الاستثناءات، وخاصةً مع عمليات الملفات، باستخدام كتل try-catch.

## دليل التنفيذ

### تمكين عناصر التحكم في وسائط عرض الشرائح
تتيح لك هذه الميزة التحكم فيما إذا كانت عناصر الوسائط مثل مقاطع الفيديو وتشغيل الصوت مرئية باستخدام عناصر التحكم أثناء العرض التقديمي.

#### ملخص
يضمن تفعيل عناصر التحكم في الوسائط في PowerPoint تمكين جمهورك من إيقاف محتوى الوسائط مؤقتًا أو إرجاعه أو تقديمه مباشرةً من شاشتهم دون الحاجة إلى تطبيقات منفصلة. تُعد هذه الميزة مفيدة للجلسات التفاعلية التي يكون فيها تفاعل المستخدم أمرًا بالغ الأهمية.

#### خطوات لتمكين عناصر التحكم في الوسائط
1. **تهيئة فئة العرض التقديمي**
   ```csharp
   using (Presentation pres = new Presentation())
   {
       // الكود سوف يذهب هنا
   }
   ```

2. **تعيين خاصية ShowMediaControls**
   ```csharp
   pres.SlideShowSettings.ShowMediaControls = true;
   ```
   - `pres.SlideShowSettings.ShowMediaControls`:تحدد هذه الخاصية ما إذا كان سيتم عرض عناصر التحكم في الوسائط أثناء وضع عرض الشرائح.

3. **حفظ العرض التقديمي**
   ```csharp
   pres.Save("YOUR_DOCUMENT_DIRECTORY\\SlideShowMediaControl.pptx", SaveFormat.Pptx);
   ```

### تعطيل عناصر التحكم في وسائط عرض الشرائح
في السيناريوهات التي تفضل فيها تجربة مشاهدة سلسة بدون انقطاعات، قد يكون تعطيل عناصر التحكم في الوسائط مفيدًا.

#### ملخص
يُساعد تعطيل عناصر التحكم بالوسائط على الحفاظ على التركيز من خلال إزالة أي تشتيت محتمل من أزرار الشاشة. يُعد هذا الإعداد مثاليًا للعروض التقديمية المُصممة للعرض بشكل متواصل دون تفاعل المستخدم مع عناصر الوسائط.

#### خطوات تعطيل عناصر التحكم في الوسائط
1. **تهيئة فئة العرض التقديمي**
   ```csharp
   using (Presentation pres = new Presentation())
   {
       // الكود سوف يذهب هنا
   }
   ```

2. **تعيين خاصية ShowMediaControls**
   ```csharp
   pres.SlideShowSettings.ShowMediaControls = false;
   ```
   - ويضمن هذا إخفاء عناصر التحكم في الوسائط أثناء العرض التقديمي، مما يوفر تجربة خالية من التشتيت.

3. **حفظ العرض التقديمي**
   ```csharp
   pres.Save("YOUR_DOCUMENT_DIRECTORY\\SlideShowMediaControl_Disabled.pptx", SaveFormat.Pptx);
   ```

### نصائح استكشاف الأخطاء وإصلاحها
- تأكد من تحديث مكتبة Aspose.Slides الخاصة بك إلى الإصدار الأحدث.
- تأكد من أن `outFilePath` يشير المسار بشكل صحيح إلى دليل قابل للكتابة على نظامك.

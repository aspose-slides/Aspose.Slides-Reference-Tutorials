## دليل التنفيذ

بعد أن انتهيت من الإعداد، لنبدأ بتضمين إطارات الصوت في شرائح PowerPoint. سنشرح كل خطوة لضمان الوضوح والفهم.

### إضافة إطار صوتي باستخدام الصوت المضمن

#### ملخص
تتضمن عملية تضمين إطار صوتي عدة خطوات رئيسية: تحميل ملف الوسائط، وإنشاء إطار الصوت، وتعيين خصائصه للحصول على عرض مثالي أثناء العرض التقديمي.

#### الخطوة 1: تحميل ملف الوسائط
أولاً، قم بتحديد المسار إلى ملف الصوت الخاص بك:

```csharp
string mediaFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "your_audio_file.mp3");
```

تأكد من ذلك `mediaFile` يشير إلى موقع صالح يحتوي على ملف الصوت المطلوب.

#### الخطوة 2: إنشاء إطار صوتي
بعد ذلك، سنضيف إطارًا صوتيًا إلى الشريحة. يتطلب ذلك تحديد موضع وحجم الإطار:

```csharp
// إضافة شريحة فارغة إلى العرض التقديمي
ISlide slide = presentation.Slides.AddEmptySlide(presentation.LayoutSlides[0]);

// تحميل ملف الوسائط إلى تيار
using FileStream audioStream = new FileStream(mediaFile, FileMode.Open);

// أضف إطارًا صوتيًا إلى الشريحة في الموضع (x: 50، y: 150) بعرض وارتفاع 100 بكسل
IAudioFrame audioFrame = slide.Shapes.AddAudioFrameEmbedded(50, 150, 100, 100, audioStream);
```

#### الخطوة 3: تكوين خصائص إطار الصوت
تخصيص إعدادات التشغيل وفقًا لاحتياجاتك:

```csharp
// ضبط وضع تشغيل الصوت ومستوى الصوت
audioFrame.PlayMode = AudioPlayModePreset.Auto;
audioFrame.Volume = AudioVolumeMode.Low;

// اختياريًا، قم بتعيين صورة ملصق أو خصائص أخرى هنا
```

#### نصائح استكشاف الأخطاء وإصلاحها
- **مشكلة شائعة**:تأكد من أن مسار ملف الوسائط صحيح لتجنب `FileNotFoundException`.
- **الصوت لا يعمل**:تأكد من تكوين إعدادات الصوت (مثل مستوى الصوت) بشكل صحيح.

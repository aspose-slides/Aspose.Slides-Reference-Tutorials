## دليل التنفيذ

### استدعاء تحميل الموارد المخصصة للصور

**ملخص:**
تتيح لك هذه الميزة استبدال الصور المفقودة ببدائل محددة مسبقًا والتعامل مع تنسيقات الصور المحددة بشكل مختلف عند تحميل العرض التقديمي.

#### الخطوة 1: إنشاء فئة ImageLoadingHandler

ابدأ بتعريف فئة تنفذ `IResourceLoadingCallback`سيسمح لك هذا باعتراض أحداث تحميل الموارد:

```csharp
using Aspose.Slides;
using System.IO;

public class ImageLoadingHandler : IResourceLoadingCallback
{
    string dataDir = @"YOUR_DOCUMENT_DIRECTORY";

    public ResourceLoadingAction ResourceLoading(IResourceLoadingArgs args)
    {
        // تحقق مما إذا كانت الصورة الأصلية بتنسيق JPEG
        if (args.OriginalUri.EndsWith(".jpg"))
        {
            try // محاولة تحميل صورة بديلة
            {
                byte[] imageBytes = File.ReadAllBytes(Path.Combine(dataDir, "aspose-logo.jpg"));
                args.SetData(imageBytes); // توفير بايتات الصورة البديلة
                return ResourceLoadingAction.UserProvided; // يشير إلى نجاح المعالجة المخصصة
            }
            catch (Exception)
            {
                return ResourceLoadingAction.Skip; // تخطي إذا كان هناك خطأ في تحميل الصورة
            }
        }
        else if (args.OriginalUri.EndsWith(".png"))
        {
            args.Uri = "http://www.google.com/images/logos/ps_logo2.png"; // استبدال PNG بعنوان URL
            return ResourceLoadingAction.Default; // استخدم المعالجة الافتراضية لعنوان URI الجديد
        }

        return ResourceLoadingAction.Skip; // تخطي جميع الصور الأخرى
    }
}
```
**توضيح:**
- **منطق تحميل الموارد**:إذا كانت الصورة مفقودة، وكانت عبارة عن ملف JPEG، فإننا نستبدلها بـ `aspose-logo.jpg`بالنسبة لملفات PNG، نقوم بإعادة التوجيه إلى عنوان URL المحدد.
- **معالجة الأخطاء**:في حالة حدوث مشكلات أثناء تحميل الصورة البديلة، نقوم بتخطي المورد لتجنب تعطل التطبيق.

#### الخطوة 2: تحميل العرض التقديمي باستخدام الخيارات المخصصة

بعد ذلك، قم بتهيئة العرض التقديمي الخاص بك باستخدام المعالج المخصص:

```csharp
using Aspose.Slides;
using System.IO;

string dataDir = @"YOUR_DOCUMENT_DIRECTORY";
LoadOptions opts = new LoadOptions();
opts.ResourceLoadingCallback = new ImageLoadingHandler();

Presentation presentation = new Presentation(Path.Combine(dataDir, "presentation.pptx"), opts);
```
**توضيح:**
- **خيارات التحميل**: يُهيئ كيفية تحميل العرض التقديمي. عن طريق ضبط `ResourceLoadingCallback`يمكنك تخصيص تحميل الصورة.
- **تهيئة العرض التقديمي**: ال `Presentation` يتم إنشاء الكائن بمسار إلى ملف PPTX وخيارات تحميل مخصصة.

### نصائح استكشاف الأخطاء وإصلاحها

- تأكد من وضع الصور البديلة بشكل صحيح في `YOUR_DOCUMENT_DIRECTORY`.
- تحقق من الوصول إلى الشبكة إذا كنت تقوم باستبدال الصور بعناوين URL من الويب.
- تحقق من سجلات الاستثناءات للحصول على رسائل الخطأ التفصيلية أثناء التطوير.

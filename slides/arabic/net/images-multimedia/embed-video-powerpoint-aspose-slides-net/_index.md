## دليل التنفيذ

### إضافة إطار فيديو إلى شريحة PowerPoint

#### ملخص

يتيح لك تضمين إطار الفيديو دمج محتوى الفيديو بشكل مباشر في شرائح العرض التقديمي، مما يجعلها أكثر تفاعلية وجاذبية.

#### دليل خطوة بخطوة

**1. إعداد مشروعك**

أولاً، تأكد من تثبيت Aspose.Slides بشكل صحيح في مشروعك وتأكد من إعداد الترخيص إذا لزم الأمر.

```csharp
using Aspose.Slides;
using Aspose.Slides.Export;

// تحديد مسارات الدليل لتخزين المستندات
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputDir = "YOUR_OUTPUT_DIRECTORY";

// تأكد من وجود دليل الإخراج أو قم بإنشائه
bool IsExists = System.IO.Directory.Exists(outputDir);
if (!IsExists)
    System.IO.Directory.CreateDirectory(outputDir);

// إنشاء فئة عرض تقديمي لتمثيل ملف PPTX
using (Presentation pres = new Presentation())
{
```

**2. الوصول إلى الشرائح وتعديلها**

انتقل إلى الشريحة الأولى من العرض التقديمي الخاص بك لإضافة إطار الفيديو:

```csharp
    // الوصول إلى الشريحة الأولى في العرض التقديمي
    ISlide sld = pres.Slides[0];
    
    // أضف إطار فيديو بموضع وحجم ومسار محدد لملف الفيديو
    IVideoFrame vf = sld.Shapes.AddVideoFrame(50, 150, 300, 150, dataDir + "video1.avi");
```

- **المعلمات موضحة:**
  - `50, 150`:الإحداثيات (X، Y) حيث سيتم وضع إطار الفيديو.
  - `300, 150`:عرض وارتفاع إطار الفيديو.
  - `"video1.avi"`: مسار ملف الفيديو. تأكد من إمكانية الوصول إليه من دليل بياناتك.

**3. تكوين إعدادات التشغيل**

يمكنك التحكم في كيفية تصرف الفيديو أثناء العرض التقديمي:

```csharp
    // تكوين إعدادات التشغيل للفيديو
    vf.PlayMode = VideoPlayModePreset.Auto; // التشغيل التلقائي عند بدء عرض الشرائح
    vf.Volume = AudioVolumeMode.Loud;       // ضبط مستوى الصوت إلى مرتفع

    // حفظ العرض التقديمي المعدل على القرص
    pres.Save(outputDir + "VideoFrame_out.pptx", SaveFormat.Pptx);
}
```

- **خيارات التشغيل:**
  - `PlayMode`:يحدد كيفية تشغيل الفيديو. `Auto` يبدأ التشغيل تلقائيًا أثناء عرض الشرائح.
  - `Volume`:ضبط مستوى الصوت؛ تتضمن الخيارات `Loud`، `Soft`، إلخ.

## دليل التنفيذ

الآن بعد أن قمت بإعداد Aspose.Slides لـ .NET، فلنقم بإضافة إطار صوتي إلى الشريحة:

### إضافة إطار صوتي إلى شريحة

تتيح لك هذه الميزة تضمين الصوت مباشرةً في شرائح PowerPoint باستخدام لغة C#. اتبع الخطوات التالية:

#### الخطوة 1: تحضير ملف الدليل والعرض التقديمي

تأكد من تحديد مسار مجلد المستندات لحفظ ملف العرض التقديمي. هذا يُسهّل إدارة الملفات بفعالية.

```csharp
string dataDir = @"YOUR_DOCUMENT_DIRECTORY";

// تأكد من وجود الدليل؛ قم بإنشائه إذا لم يكن موجودًا.
bool isExists = Directory.Exists(dataDir);
if (!isExists)
    Directory.CreateDirectory(dataDir);

using (Presentation pres = new Presentation())
{
    // قم بالوصول إلى الشريحة الأولى في العرض التقديمي.
    ISlide sld = pres.Slides[0];
```

#### الخطوة 2: تضمين الصوت في الشريحة

افتح ملفًا صوتيًا وأدرجه كإطار داخل الشريحة. هنا، نفتح `sampleaudio.wav` وأضفها إلى الشريحة الخاصة بنا في الإحداثيات المحددة.

```csharp
    // افتح ملف صوتي كتدفق.
    using (FileStream fstr = new FileStream(dataDir + "sampleaudio.wav", FileMode.Open, FileAccess.Read))
    {
        // تضمين إطار الصوت في الشريحة.
        IAudioFrame audioFrame = sld.Shapes.AddAudioFrameEmbedded(50, 150, 100, 100, fstr);
```

#### الخطوة 3: تكوين تشغيل الصوت

حدّد خيارات تشغيل الصوت. يشمل ذلك التشغيل التلقائي للشرائح وإعدادات مستوى الصوت.

```csharp
        // قم بتكوين إطار الصوت ليتم تشغيله عبر الشرائح عند تنشيطه.
        audioFrame.PlayAcrossSlides = true;

        // ضبط الصوت ليتم إرجاعه تلقائيًا بعد التشغيل.
        audioFrame.RewindAudio = true;

        // قم بتحديد وضع التشغيل ومستوى الصوت للصوت.
        audioFrame.PlayMode = AudioPlayModePreset.Auto;
        audioFrame.Volume = AudioVolumeMode.Loud;
    }
```

#### الخطوة 4: حفظ العرض التقديمي

احفظ العرض التقديمي الخاص بك مع جميع التغييرات المطبقة، بما في ذلك إطار الصوت المضمن حديثًا.

```csharp
    // احفظ العرض التقديمي المعدّل.
    pres.Save(dataDir + "AudioFrameEmbed_out.pptx", SaveFormat.Pptx);
}
```

### نصائح استكشاف الأخطاء وإصلاحها
- **لم يتم العثور على الملف:** تأكد من أن مسار ملف الصوت الخاص بك صحيح ويمكن الوصول إليه.
- **مشاكل التشغيل:** تحقق مما إذا كانت إعدادات الصوت مثل `PlayMode` تم تكوينها بشكل صحيح.

**س2:** هل يمكنني استخدام تنسيقات صوتية مختلفة مع Aspose.Slides .NET؟

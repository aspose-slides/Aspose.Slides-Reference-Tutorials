## دليل التنفيذ

### التحقق من كلمات مرور العرض التقديمي

#### ملخص
تتحقق هذه الميزة مما إذا كانت كلمة مرور محددة قادرة على فتح عرض تقديمي محمي في PowerPoint، وهي مفيدة للتحقق من إمكانية الوصول دون فتح الملف يدويًا.

#### التنفيذ خطوة بخطوة
**1. تحديد مسار الملف**
تعيين المسار إلى العرض التقديمي المصدر الخاص بك:
```csharp
string pptFile = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "ProtectedPresentation.pptx");
```

**2. قم بتحميل العرض التقديمي بكلمة مرور**
استخدم Aspose.Slides `Presentation` الفئة التي يجب محاولة فتحها باستخدام كلمة المرور المقدمة.
```csharp
try
{
    // حاول فتح العرض التقديمي باستخدام كلمة المرور المحددة
    using (Presentation pres = new Presentation(pptFile, "YourPasswordHere"))
    {
        Console.WriteLine("The presentation is unlocked!");
    }
}
catch (Exception ex)
{
    if (ex is InvalidDataException)
    {
        Console.WriteLine("Incorrect password.");
    }
    else
    {
        // التعامل مع الاستثناءات الأخرى مثل عدم العثور على الملف
        Console.WriteLine(ex.Message);
    }
}
```
**توضيح:** 
- ال `Presentation` المُنشئ: يأخذ مسار الملف وكلمة المرور الاختيارية. إذا كان صحيحًا، فسيتم تحميل العرض التقديمي؛ وإلا، فسيتم طرح استثناء.
- معالجة الاستثناءات: اكتشاف استثناءات محددة لتحديد كلمات المرور غير الصحيحة.

## دليل التنفيذ
### الميزة 1: فتح PowerPoint المحمي بكلمة مرور (H2)
توضح هذه الميزة كيفية فتح ملف PowerPoint محمي بكلمة مرور باستخدام Aspose.Slides لـ .NET.

#### ملخص
يتطلب فتح ملف محمي بكلمة مرور تحديد خيارات التحميل الصحيحة. سيرشدك هذا القسم إلى كيفية إعداد هذه الخيارات والوصول إلى شرائحك.

##### الخطوة 1: تحديد دليل المستندات (H3)
قم بتحديد المسار إلى ملف PowerPoint المحمي بكلمة مرور:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY/OpenPasswordPresentation.pptx";
```
يستبدل `YOUR_DOCUMENT_DIRECTORY` مع الدليل الفعلي الذي يوجد به ملفك.

##### الخطوة 2: تعيين خيارات التحميل (H3)
إنشاء مثيل لـ `LoadOptions` لتحديد المعلمات المطلوبة لتحميل العرض التقديمي:
```csharp
LoadOptions loadOptions = new LoadOptions();
loadOptions.Password = "pass"; // استبدل كلمة "pass" بكلمة المرور الفعلية الخاصة بك
```
هنا، `Password` هي معلمة مهمة تسمح لـ Aspose.Slides بالمصادقة وفتح الملف.

##### الخطوة 3: فتح العرض التقديمي (H3)
استخدم `Presentation` منشئ الفئة مع خيارات التحميل المحددة:
```csharp
Presentation pres = new Presentation(dataDir, loadOptions);
```
تفتح هذه الخطوة العرض التقديمي الخاص بك للتفاعل البرمجي.

##### الخطوة 4: الوصول إلى عدد الشرائح (H3)
للتأكد من فتح الملف بشكل صحيح، قم بالوصول إلى العدد الإجمالي للشرائح:
```csharp
int slideCount = pres.Slides.Count;
Console.WriteLine($"The presentation contains {slideCount} slides.");
```
### الميزة 2: حفظ العرض التقديمي في دليل محدد (H2)
بعد الوصول إلى عرضك التقديمي أو تعديله، يُعد حفظه أمرًا ضروريًا. يشرح هذا القسم كيفية حفظ الملف في مجلد محدد.

#### ملخص
يتطلب حفظ العروض التقديمية تحديد مسار الإخراج وتنسيقه. إليك كيفية القيام بذلك بكفاءة باستخدام Aspose.Slides لـ .NET.

##### الخطوة 1: تعيين دليل الإخراج (H3)
حدد المكان الذي تريد حفظ العرض التقديمي الخاص بك فيه:
```csharp
string outputPath = "YOUR_OUTPUT_DIRECTORY/SavedPresentation.pptx";
```
تأكد `YOUR_OUTPUT_DIRECTORY` هو مسار دليل صالح على نظامك.

##### الخطوة 2: حفظ العرض التقديمي (H3)
على افتراض `pres` يحمل عرضك التقديمي المحمّل، استخدم `Save` طريقة كتابتها على القرص:
```csharp
pres.Save(outputPath, Aspose.Slides.Export.SaveFormat.Pptx);
```
هنا، `SaveFormat.Pptx` يُحدد الحفظ بتنسيق PowerPoint. تضمن هذه العملية حفظ تغييراتك.

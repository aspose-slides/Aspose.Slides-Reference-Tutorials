## كيفية إنشاء جدول وتنسيقه في PowerPoint باستخدام Aspose.Slides لـ .NET

إنشاء الجداول في عروض PowerPoint التقديمية يُحسّن وضوح واحترافية شرائحك بشكل ملحوظ. مع ذلك، قد يستغرق القيام بذلك يدويًا وقتًا طويلاً. مع Aspose.Slides لـ .NET، يمكنك تبسيط هذه العملية من خلال إنشاء الجداول وتنسيقها برمجيًا. سيرشدك هذا البرنامج التعليمي خلال إعداد عرض تقديمي جديد، وإضافة جدول إلى الشريحة الأولى، وتخصيص تخطيطه، وإضافة نصوص إلى الخلايا، وحفظ عملك بكفاءة.

- كيفية إعداد Aspose.Slides لـ .NET في مشروعك
- خطوات إنشاء الجداول وتنسيقها برمجيًا
- تقنيات لتخصيص خصائص الخلية مثل حجم النص والمحاذاة

دعنا نتعمق في إعداد بيئتك وإتقان إنشاء الجدول باستخدام هذه المكتبة القوية!

## دليل التنفيذ

### إنشاء جدول وإضافته إلى PowerPoint

دعونا نلقي نظرة على عملية إنشاء جدول في شريحة العرض التقديمي.

#### الخطوة 1: إنشاء عرض تقديمي جديد

ابدأ بإنشاء مثيل `Presentation` يمثل هذا الكائن ملف PowerPoint بأكمله.

```csharp
Presentation pres = new Presentation();
```

#### الخطوة 2: الوصول إلى الشريحة الأولى

استرجاع الشريحة الأولى من العرض التقديمي لإضافة عناصر إليها:

```csharp
ISlide sld = pres.Slides[0];
```

#### الخطوة 3: تحديد أبعاد الجدول وإضافتها

حدد عرض الأعمدة وارتفاع الصفوف لجدولك. تحدد هذه المصفوفات أبعاد كل عنصر.

```csharp
double[] dblCols = { 50, 50, 50 };
double[] dblRows = { 50, 30, 30, 30, 30 };

Aspose.Slides.ITable tbl = sld.Shapes.AddTable(50, 50, dblCols, dblRows);
```

#### الخطوة 4: ملء خلايا الجدول بالنص

كرّر كل خلية لإضافة نص. خصّص مظهر هذا النص حسب الحاجة.

```csharp
foreach (IRow row in tbl.Rows) {
    foreach (ICell cell in row) {
        ITextFrame tf = cell.TextFrame;
        tf.Text = "T" + cell.FirstRowIndex.ToString() + cell.FirstColumnIndex.ToString();
        tf.Paragraphs[0].Portions[0].PortionFormat.FontHeight = 10;
        tf.Paragraphs[0].ParagraphFormat.Bullet.Type = BulletType.None;
    }
}
```

#### الخطوة 5: احفظ العرض التقديمي الخاص بك

وأخيرًا، احفظ العرض التقديمي في الدليل المحدد.

```csharp
pres.Save("YOUR_OUTPUT_DIRECTORY\tblSLD.ppt", SaveFormat.Ppt);
```

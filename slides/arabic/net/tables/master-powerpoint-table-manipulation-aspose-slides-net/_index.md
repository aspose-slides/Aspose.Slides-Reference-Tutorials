### الوصول إلى الجداول وتعديلها في العروض التقديمية (H2)
#### ملخص:
سنركز على الوصول إلى جدول موجود في شريحة وتحديث محتواه برمجيًا. يُعد هذا مفيدًا بشكل خاص للعروض التقديمية التي تتطلب تحديثات متكررة للبيانات.
**الخطوة 1: تحميل العرض التقديمي**
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
using (Presentation presentation = new Presentation(dataDir + "/UpdateExistingTable.pptx"))
{
    // الكود الخاص بك هنا...
}
```
- **لماذا**:يعد تحميل العرض التقديمي ضروريًا للوصول إلى شرائحه وأشكاله.
**الخطوة 2: الوصول إلى الشريحة**
```csharp
ISlide sld = presentation.Slides[0];
```
- **لماذا**:نحن بحاجة إلى العمل مع شريحة محددة، وعادة ما نبدأ من الشريحة الأولى في هذا المثال.
**الخطوة 3: العثور على شكل الجدول**
```csharp
ITable table = null;
foreach (IShape shape in sld.Shapes)
{
    if (shape is ITable)
    {
        table = (ITable)shape; // وجدت طاولة.
        break; // الخروج من الحلقة بمجرد العثور عليها لتحسين الأداء.
    }
}
```
- **لماذا**:تحتوي عروض PowerPoint على أشكال مختلفة، لذا من المهم تحديد الشكل الذي يمثل `ITable`.
**الخطوة 4: تعديل محتوى الجدول**
```csharp
if (table != null)
{
    table[0, 1].TextFrame.Text = "New";
}
```
- **لماذا**:هذا يُحدِّث نص خلية مُحدَّدة في الجدول. عدِّل المؤشرات حسب احتياجاتك.
**الخطوة 5: حفظ العرض التقديمي**
```csharp
presentation.Save("YOUR_OUTPUT_DIRECTORY" + "/UpdateTable_out.pptx", SaveFormat.Pptx);
```
- **لماذا**:يضمن الحفظ أن يتم الاحتفاظ بجميع التغييرات على القرص لاستخدامها في المستقبل.

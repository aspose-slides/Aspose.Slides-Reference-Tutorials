## تعليمات التثبيت

1. تثبيت Aspose.Slides لـ .NET:
   
   استخدام وحدة تحكم إدارة الحزم NuGet:
   ```
   Install-Package Aspose.Slides.NET
   ```

   استخدام .NET CLI:
   ```
   dotnet add package Aspose.Slides.NET
   ```

2. أضف توجيهات الاستخدام الضرورية إلى الكود الخاص بك:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Tables;
   using System.IO;
   ```

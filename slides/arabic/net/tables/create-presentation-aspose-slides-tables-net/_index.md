## دليل التنفيذ

الآن بعد أن قمت بإعداد بيئتك، دعنا ننتقل إلى عملية إنشاء عرض تقديمي باستخدام جدول.

### إنشاء العرض التقديمي

أولاً، قم بإنشاء مثيل لـ `Presentation` الصف لبدء العمل على الشرائح:

```csharp
// تهيئة عرض تقديمي جديد
Presentation pres = new Presentation();
```

تُمهّد هذه الخطوة الطريق لإضافة محتوى إلى ملف PowerPoint. بعد ذلك، انتقل إلى الشريحة الأولى من المجموعة:

```csharp
// الوصول إلى الشريحة الأولى
ISlide slide = pres.Slides[0];
```

### إضافة جدول

الآن، دعنا نحدد أبعاد الجدول ونضيفها إلى الشريحة:

**تحديد الأبعاد:**
حدد عرض الأعمدة وارتفاع الصفوف لجدولك. هذه الخطوة بالغة الأهمية لأنها تحدد كيفية تنظيم المحتوى داخل كل خلية.

```csharp
// تحديد عرض الأعمدة وارتفاع الصفوف
double[] colWidth = { 100, 50, 30 };
double[] rowHeight = { 30, 50, 30 };
```

**إضافة الجدول:**
أضف شكل جدول إلى شريحتك باستخدام هذه الأبعاد. ستحدد موقعه على الشريحة باستخدام إحداثيات x وy.

```csharp
// أضف جدولاً إلى الشريحة الأولى عند (x=100، y=100)
ITable table = slide.Shapes.AddTable(100, 100, colWidth, rowHeight);
```

### حفظ العرض التقديمي

وأخيرًا، احفظ العرض التقديمي الخاص بك بتنسيق PPTX:

```csharp
// حفظ العرض التقديمي في مسار الدليل المحدد
pres.Save("YOUR_DOCUMENT_DIRECTORY/TestTable_out.pptx");
```

تضمن هذه الخطوة الحفاظ على تعديلاتك وإمكانية الوصول إليها أو مشاركتها لاحقًا.

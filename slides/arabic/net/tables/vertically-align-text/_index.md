# إنشاء جداول تحتوي على نص عمودي في عروض PowerPoint التقديمية

يوفر Aspose.Slides لـ .NET إمكانيات فعّالة لإنشاء عروض PowerPoint التقديمية ومعالجتها برمجيًا. يوضح هذا المثال كيفية إنشاء جدول بتنسيق مخصص، بما في ذلك محاذاة النص رأسيًا وتدويره - وهو متطلب شائع للعروض التقديمية المتخصصة للأعمال وتصور البيانات.

## تعليمات التثبيت

1. تثبيت Aspose.Slides لـ .NET عبر NuGet Package Manager:
   ```
   Install-Package Aspose.Slides
   ```
   أو عبر .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

2. أضف المراجع إلى مشروعك:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

3. تنزيل موارد التوثيق:
   - [مرجع واجهة برمجة التطبيقات](https://reference.aspose.com/slides/net/)
   - [دليل المطور](https://docs.aspose.com/slides/net/)

## دليل التنفيذ
الآن بعد إعداد كل شيء، دعنا نستعرض كيفية قفل نسبة العرض إلى الارتفاع لجدول في PowerPoint أو إلغاء قفلها باستخدام Aspose.Slides.
### نسبة العرض إلى الارتفاع للقفل/فتح القفل
تتيح لك هذه الميزة الحفاظ على أبعاد جداولك حتى عند تغيير حجم عناصر أخرى في الشريحة. إليك كيفية عملها:
#### الخطوة 1: تحميل العرض التقديمي الخاص بك
أولاً، قم بتحميل ملف العرض التقديمي الذي يحتوي على الجدول:
```csharp
using (Presentation pres = new Presentation(dataDir + "/pres.pptx"))
{
    // سيتم وضع الكود الخاص بالتلاعب بالجدول هنا
}
```
#### الخطوة 2: الوصول إلى شكل الجدول
قم بتحديد الشكل الأول على الشريحة الخاصة بك والوصول إليه، مع التأكد من أنه عبارة عن جدول:
```csharp
ITable table = (ITable)pres.Slides[0].Shapes[0];
```
#### الخطوة 3: تبديل قفل نسبة العرض إلى الارتفاع
تحقق مما إذا كانت نسبة العرض إلى الارتفاع مقفلة حاليًا. ثم بدّل حالتها إلى قفل أو إلغاء قفل:
```csharp
bool originalLockState = table.ShapeLock.AspectRatioLocked;
table.ShapeLock.AspectRatioLocked = !originalLockState; // عكس الحالة الحالية
```
#### الخطوة 4: حفظ التغييرات
وأخيرًا، احفظ العرض التقديمي المعدّل في ملف جديد:
```csharp
pres.Save(outputPath + "/pres-out.pptx", SaveFormat.Pptx);
```

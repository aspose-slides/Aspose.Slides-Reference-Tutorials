# إنشاء جداول موحدة في PowerPoint باستخدام Aspose.Slides

Aspose.Slides for .NET هي واجهة برمجة تطبيقات فعّالة تُمكّن المطورين من إنشاء عروض PowerPoint التقديمية وقراءتها وكتابتها وتعديلها وتحويلها برمجيًا دون الحاجة إلى Microsoft PowerPoint. يوضح هذا المثال كيفية إنشاء جدول قياسي بحدود مُنسقة خصيصًا في عرض PowerPoint التقديمي.

## تعليمات التثبيت

1. التثبيت عبر مدير حزمة NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. أو عبر .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. أضف مساحة الأسماء التالية إلى ملف C# الخاص بك:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Tables;
   using System.Drawing;
   ```

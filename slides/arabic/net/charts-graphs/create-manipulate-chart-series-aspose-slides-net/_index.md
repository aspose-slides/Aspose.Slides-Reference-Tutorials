### إضافة سلسلة المخططات وتكوينها
#### ملخص
تتضمن إضافة مخطط عمودي مجمع تهيئة المخطط، وتكوين خصائصه، وتزويده بالبيانات. اتبع الخطوات التالية:

##### الخطوة 1: تهيئة مستند العرض التقديمي الخاص بك
قم بإنشاء كائن عرض تقديمي لبدء إضافة المخططات البيانية الخاصة بك:
```csharp
string yourDocumentDirectory = "YOUR_DOCUMENT_DIRECTORY";
using (Presentation pres = new Presentation())
{
    // يظهر رمز إضافة الرسم البياني هنا
}
```
**لماذا**يقوم هذا الكود بإعداد بيئة العمل، مما يضمن تغليف كل شيء في كائن العرض.

##### الخطوة 2: إضافة مخطط عمودي مجمع
أضف مخططًا عموديًا مجمعًا إلى الشريحة الأولى:
```csharp
IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 50, 50, 600, 400, true);
```
**لماذا**:تضيف دعوة الطريقة هذه كائن مخطط جديد عند إحداثيات محددة بأبعاد محددة مسبقًا.

##### الخطوة 3: تكوين سلسلة المخططات
قم بمسح أي سلسلة موجودة وأضف سلسلتك الخاصة:
```csharp
IChartSeriesCollection series = chart.ChartData.Series;
series.Clear();
series.Add(chart.ChartData.ChartDataWorkbook.GetCell(0, "B1"), chart.Type);
```
**لماذا**يضمن المسح عدم تداخل البيانات المتبقية مع التكوينات الجديدة. إضافة سلسلة تُهيئها لإدراج نقاط البيانات.

##### الخطوة 4: إضافة نقاط البيانات
املأ الرسم البياني الخاص بك بالبيانات، بما في ذلك القيم السلبية:
```csharp
series[0].DataPoints.AddDataPointForBarSeries(chart.ChartData.ChartDataWorkbook.GetCell(0, "B2"), -50);
```
**لماذا**إضافة نقاط البيانات أمرٌ بالغ الأهمية لتصور مجموعة البيانات. تُدعم القيم السالبة لإظهار العجز أو الخسارة.

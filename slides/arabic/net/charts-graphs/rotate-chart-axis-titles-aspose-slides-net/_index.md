## دليل التنفيذ
سوف يرشدك هذا الدليل خلال عملية تعيين زاوية دوران عنوان محور الرسم البياني باستخدام Aspose.Slides لـ .NET.
### نظرة عامة على الميزة: ضبط زاوية دوران عنوان محور الرسم البياني
يُمكن لتعديل زاوية الدوران تحسين سهولة القراءة والجمال، خاصةً في الشرائح ذات المساحة المحدودة. إليك كيفية تطبيق هذه الميزة:
#### الخطوة 1: إنشاء عرض تقديمي وإضافة مخطط
ابدأ بإنشاء عرض تقديمي جديد وإضافة مخطط عمودي مجمع.
```csharp
using Aspose.Slides;

string dataDir = "YOUR_DOCUMENT_DIRECTORY";

// تهيئة مثيل عرض تقديمي جديد.
using (Presentation pres = new Presentation())
{
    // أضف مخططًا عموديًا مجمعًا إلى الشريحة الأولى في الموضع (50، 50) بعرض 450 وارتفاع 300.
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 50, 50, 450, 300);
```
#### الخطوة 2: تمكين عنوان المحور الرأسي
قم بتمكين عنوان المحور الرأسي لتخصيص مظهره.
```csharp
    // تمكين عنوان المحور الرأسي للرسم البياني.
    chart.Axes.VerticalAxis.HasTitle = true;
```
#### الخطوة 3: ضبط زاوية الدوران
تعيين زاوية دوران تنسيق كتلة النص لعنوان المحور الرأسي.
```csharp
    // ضبط زاوية الدوران إلى 90 درجة.
    chart.Axes.VerticalAxis.Title.TextFormat.TextBlockFormat.RotationAngle = 90;

    // احفظ العرض التقديمي مع الرسم البياني المعدل في ملف .pptx في الدليل المحدد.
    pres.Save(dataDir + "test.pptx", SaveFormat.Pptx);
}
```
### خيارات تكوين المفاتيح
- **زاوية الدوران**:تخصيص ما بين -180 و 180 درجة بناءً على احتياجات التصميم الخاصة بك.
- **تنسيق عنوان المحور**:تعديل حجم الخط والنمط واللون لتحسين الرؤية.

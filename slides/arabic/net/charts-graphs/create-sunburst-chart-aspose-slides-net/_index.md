# كيفية إنشاء مخطط Sunburst في .NET باستخدام Aspose.Slides

## مقدمة

يُعدّ تصوّر البيانات الهرمية بفعالية أمرًا بالغ الأهمية لعروض تقديمية جذابة. يُمكن لمخطط "الشمس المشرقة"، المعروف بجاذبيته البصرية ووضوحه، أن يُوضّح الهياكل المعقدة بسلاسة. سيُرشدك هذا البرنامج التعليمي إلى كيفية إنشاء مخطط "الشمس المشرقة" باستخدام Aspose.Slides بلغة C#، مما يُحسّن عروضك التقديمية بصور مرئية قوية ومُستندة إلى البيانات.

في هذا الدليل، سوف تتعلم:
- كيفية إعداد Aspose.Slides لـ .NET
- خطوات إنشاء مخطط انفجار الشمس من الصفر
- تقنيات لتكوين فئات المخططات والسلاسل
- أفضل الممارسات لتحسين الأداء

لنبدأ! أولاً، تأكد من جاهزية بيئتك.

قبل إنشاء مخطط انفجار الشمس، تأكد من استيفائك للمتطلبات التالية:

### المكتبات والإصدارات المطلوبة
- **Aspose.Slides لـ .NET**:المكتبة الأساسية لإنشاء العروض التقديمية على PowerPoint ومعالجتها.

### متطلبات إعداد البيئة

## إعداد Aspose.Slides لـ .NET

للبدء، قم بتثبيت مكتبة Aspose.Slides باستخدام إحدى الطرق التالية:

**استخدام .NET CLI**
```bash
dotnet add package Aspose.Slides
```

**استخدام مدير الحزم في Visual Studio**
```powershell
Install-Package Aspose.Slides
```

**واجهة مستخدم مدير الحزم NuGet**
- ابحث عن "Aspose.Slides" وقم بتثبيت الإصدار الأحدث.

### خطوات الحصول على الترخيص

1. **نسخة تجريبية مجانية**:ابدأ بإصدار تجريبي مجاني لاستكشاف ميزات المكتبة.
2. **رخصة مؤقتة**:الحصول على ترخيص مؤقت لإجراء اختبارات موسعة إذا لزم الأمر.
3. **شراء**:للاستخدام المستمر، قم بشراء اشتراك من الموقع الرسمي لـ Aspose.

لتهيئة مشروعك وإعداده:

```csharp
// تهيئة ترخيص Aspose.Slides (إذا كان لديك واحد)
Aspose.Slides.License license = new Aspose.Slides.License();
license.SetLicense("your-license-file.lic");
```

## دليل التنفيذ

اتبع الخطوات التالية لإنشاء مخطط انفجار الشمس:

### تحميل أو إنشاء عرض تقديمي

ابدأ بتحميل عرض تقديمي موجود أو إنشاء عرض تقديمي جديد:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
using (Presentation pres = new Presentation(dataDir + "test.pptx"))
{
    // الكود الخاص بك لإضافة الرسم البياني يذهب هنا
}
```

### إضافة مخطط Sunburst إلى الشريحة

أضف مخططًا لانفجار الشمس في الموضع المطلوب على الشريحة:

```csharp
IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.Sunburst, 50, 50, 500, 400);
```
- **حدود**:الموضع (x: 50، y: 50) والحجم (العرض: 500، الارتفاع: 400).

### مسح البيانات الموجودة

تأكد من أن الرسم البياني جاهز للبيانات الجديدة:

```csharp
chart.ChartData.Categories.Clear();
chart.ChartData.Series.Clear();
```

### مصنف بيانات مخطط الوصول

الوصول إلى المصنف للتعامل مع بيانات الرسم البياني:

```csharp
IChartDataWorkbook wb = chart.ChartData.ChartDataWorkbook;
wb.Clear(0);
```
- **لماذا واضح؟**:يؤدي هذا إلى إزالة أي بيانات متبقية قد تتداخل مع تكوينك.

### إضافة الفئات والسلاسل

قم بتحديد الفئات للمستويات الهرمية في مخطط أشعة الشمس الخاص بك:

```csharp
// مثال على إضافة فئة
IChartCategory leaf = chart.ChartData.Categories.Add(wb.GetCell(0, "C1", "CategoryName"));
```

# تخصيص الخط الرئيسي في مخططات PowerPoint باستخدام Aspose.Slides لـ .NET

## كيفية تعيين خصائص الخط لنصوص المخططات باستخدام Aspose.Slides .NET

### مقدمة

يُعدّ تحسين قابلية قراءة نص المخططات وجاذبيته البصرية في مخططات PowerPoint أمرًا بالغ الأهمية، سواءً كنت تُعدّ تقارير أعمال أو عروضًا تقديمية أكاديمية. سيوضح هذا الدليل كيفية ضبط خصائص الخط، مثل السماكة والارتفاع، باستخدام Aspose.Slides لـ .NET.

**ما سوف تتعلمه:**
- كيفية دمج Aspose.Slides في مشروعك
- خطوات إضافة مخطط عمودي مجمع وتخصيصه في PowerPoint
- تقنيات تعديل خصائص الخط داخل نصوص المخططات البيانية
- أفضل الممارسات لحفظ العروض التقديمية وإدارتها

استعد لتعزيز التأثير البصري لمخططاتك!

## إعداد Aspose.Slides لـ .NET

اتبع الخطوات التالية لإعداد مشروعك باستخدام Aspose.Slides لـ .NET:

### التثبيت عبر .NET CLI

قم بتشغيل الأمر التالي في محطتك الطرفية:
```bash
dotnet add package Aspose.Slides
```

### التثبيت عبر وحدة تحكم إدارة الحزم

قم بتنفيذ هذا الأمر في وحدة التحكم في إدارة الحزم NuGet:
```powershell
Install-Package Aspose.Slides
```

### التثبيت عبر واجهة مستخدم NuGet Package Manager

- افتح مشروعك في Visual Studio.
- انتقل إلى **الأدوات > مدير حزم NuGet > إدارة حزم NuGet للحلول**.
- ابحث عن "Aspose.Slides" وانقر فوق "تثبيت".

## دليل التنفيذ

بعد إعداد البيئة الخاصة بك، اتبع الخطوات التالية لتغيير خصائص الخط في نصوص الرسم البياني:

### الخطوة 1: تحميل ملف عرض تقديمي موجود

قم بتحميل ملف العرض التقديمي من الدليل الذي تريد تطبيق التغييرات عليه:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // استبدل بمسار المستند الخاص بك
string filePath = Path.Combine(dataDir, "test.pptx");
```
**توضيح**:يقوم هذا الكود بإعداد مسار الملف لتحميل عرض PowerPoint الحالي الخاص بك.

### الخطوة 2: افتح العرض التقديمي

افتح العرض التقديمي باستخدام Aspose.Slides:
```csharp
using (Presentation pres = new Presentation(filePath))
{
    // سيتم تضمين الخطوات اللاحقة داخل هذه الكتلة
}
```
**توضيح**: ال `Presentation` تتولى الفئة فتح ملف PowerPoint ومعالجته. باستخدام `using` يضمن البيان التخلص من الموارد بشكل صحيح.

### الخطوة 3: إضافة مخطط عمودي مجمع

أضف مخططًا عموديًا مجمعًا إلى الشريحة الأولى:
```csharp
IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 50, 50, 600, 400);
```
**توضيح**:تؤدي هذه الخطوة إلى إنشاء مخطط عمودي مجمع جديد عند الإحداثيات والأبعاد المحددة.

### الخطوة 4: تمكين عرض جدول البيانات

تأكد من أن جدول البيانات مرئي داخل الرسم البياني:
```csharp
chart.HasDataTable = true;
```
**توضيح**: جلسة `HasDataTable` يؤدي تعيين "true" إلى التأكد من عرض تسميات البيانات، والتي سنقوم بتخصيصها بعد ذلك.

### الخطوة 5: تعيين خصائص الخط لنص الرسم البياني

قم بتخصيص خصائص الخط مثل الغامق والارتفاع لنص جدول بيانات الرسم البياني الخاص بك:
```csharp
chart.ChartDataTable.TextFormat.PortionFormat.FontBold = NullableBool.True; // جعل النص غامقًا
chart.ChartDataTable.TextFormat.PortionFormat.FontHeight = 20; // ضبط ارتفاع الخط إلى 20 نقطة
```
**توضيح**:تعمل هذه الخطوط على ضبط النمط المرئي لعناوين بيانات الرسم البياني الخاص بك، مما يجعلها أكثر بروزًا وقابلية للقراءة.

### الخطوة 6: حفظ العرض التقديمي المعدّل

وأخيرًا، احفظ العرض التقديمي مع التغييرات:
```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY"; // استبدل بمسار الإخراج الخاص بك
string outputPath = Path.Combine(outputDir, "output.pptx");
pres.Save(outputPath, SaveFormat.Pptx);
```
**توضيح**:تكتب هذه الخطوة العرض التقديمي المحدث إلى ملف جديد في الدليل المحدد.

لبدء استخدام Aspose.Slides لتحريك المخططات، ستحتاج إلى دمج المكتبة في مشروع .NET الخاص بك. إليك كيفية القيام بذلك:

## دليل التنفيذ

يتضمن تحريك سلسلة من الرسوم البيانية إضافة تأثيرات مثل التلاشي التدريجي أو ظهور الرسوم المتحركة. لنُقسّم العملية إلى خطوات سهلة:

### الخطوة 1: تحميل العرض التقديمي الخاص بك

أولاً، قم بتحميل عرض PowerPoint الحالي الذي يحتوي على الرسم البياني الذي تريد تحريكه.

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // اضبط هذا على مسار الدليل الخاص بك
using (Presentation presentation = new Presentation(dataDir + "/ExistingChart.pptx"))
{
    // يمكنك الوصول إلى مجموعات الشرائح والأشكال هنا
}
```

### الخطوة 2: الوصول إلى مجموعات الشرائح والأشكال

للتحكم في الرسم البياني، قم بالوصول إلى الشريحة المطلوبة وأشكالها.

```csharp
var slide = presentation.Slides[0] as Slide;
var shapes = slide.Shapes as ShapeCollection;
```

### الخطوة 3: استرداد كائن الرسم البياني

حدد كائن الرسم البياني الخاص بك واسترجعه من مجموعة الأشكال. عادةً ما تُخزَّن الرسوم البيانية في `IChart` أشياء.

```csharp
var chart = shapes[0] as IChart; // على افتراض أنه الشكل الأول
```

### الخطوة 4: إضافة تأثير التلاشي إلى الرسم البياني

لإنشاء مدخل خفي، أضف تأثير التلاشي الذي يتم تشغيله بعد أي رسوم متحركة سابقة.

```csharp
slide.Timeline.MainSequence.AddEffect(chart, EffectType.Fade, EffectSubtype.None, EffectTriggerType.AfterPrevious);
```

### الخطوة 5: تحريك السلسلة باستخدام تأثير الظهور

قم بالتكرار خلال كل سلسلة وقم بتطبيق رسوم متحركة للمظهر للحصول على تأثير الكشف الديناميكي.

```csharp
Sequence mainSequence = (Sequence)slide.Timeline.MainSequence;
for (int i = 0; i < 4; i++)
{
    mainSequence.AddEffect(chart, EffectChartMajorGroupingType.BySeries, i,
        EffectType.Appear, EffectSubtype.None, EffectTriggerType.AfterPrevious);
}
```

### الخطوة 6: حفظ العرض التقديمي

وأخيرًا، احفظ عرضك التقديمي باستخدام الرسوم المتحركة المضافة حديثًا.

```csharp
presentation.Save(dataDir + "/AnimatingSeries_out.pptx", SaveFormat.Pptx);
```

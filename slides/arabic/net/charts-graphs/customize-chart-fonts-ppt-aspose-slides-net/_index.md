### دليل التنفيذ
دعونا نقسم التنفيذ إلى أقسام قابلة للإدارة.

#### تخصيص خصائص الخط للرسوم البيانية
تتيح لك هذه الميزة تحسين المظهر المرئي لرسوماتك البيانية من خلال تعديل خصائص الخط. إليك كيفية تنفيذها:

**الخطوة 1: تحديد مسارات الدليل**
ابدأ بتحديد المكان الذي سيتم فيه وضع ملفات الإدخال والإخراج الخاصة بك:
```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
string outputPath = Path.Combine(dataDir, "FontPropertiesForChart.pptx");
```

**الخطوة 2: إنشاء مثيل عرض تقديمي جديد**
قم بإعداد كائن عرض تقديمي جديد لاستضافة الرسم البياني الخاص بك:
```csharp
using (Presentation pres = new Presentation()) {
    // سيتم تنفيذ خطوات أخرى هنا.
}
```

**الخطوة 3: إضافة مخطط عمودي مجمع**
إدراج مخطط في الشريحة الأولى عند الإحداثيات والأبعاد المحددة:
```csharp
IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 100, 100, 500, 400);
```

**الخطوة 4: تعيين ارتفاع الخط للنص في الرسم البياني**
تخصيص حجم الخط لتحسين قابلية القراءة:
```csharp
chart.TextFormat.PortionFormat.FontHeight = 20;
```

**الخطوة 5: تمكين عرض القيم على تسميات البيانات**
تأكد من أن قيم البيانات مرئية، وأضف السياق إلى الرسم البياني الخاص بك:
```csharp
chart.ChartData.Series[0].Labels.DefaultDataLabelFormat.ShowValue = true;
```

**الخطوة 6: حفظ العرض التقديمي**
احفظ العرض التقديمي الخاص بك مع تطبيق جميع التخصيصات:
```csharp
pres.Save(outputPath, SaveFormat.Pptx);
```

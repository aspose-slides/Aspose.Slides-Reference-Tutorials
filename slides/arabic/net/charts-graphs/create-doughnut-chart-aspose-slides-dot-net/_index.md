للبدء، ستحتاج إلى تثبيت مكتبة Aspose.Slides. إليك كيفية القيام بذلك باستخدام طرق مختلفة:

**استخدام .NET CLI:**

**استخدام وحدة تحكم إدارة الحزم:**

## دليل التنفيذ

دعنا نقسم عملية إنشاء مخطط دائري باستخدام Aspose.Slides لـ .NET إلى خطوات قابلة للإدارة.

### إنشاء مخطط دائري

#### ملخص
سنبدأ بإضافة مخطط دائري إلى شريحة PowerPoint الخاصة بك، وتعيين موضعه وحجمه.

**إضافة الرسم البياني:**
```csharp
using Aspose.Slides.Charts;

// الوصول إلى الشريحة الأولى في العرض التقديمي (يتم إنشاء شريحة افتراضيًا)
ISlide slide = presentation.Slides[0];

// أضف مخططًا دائريًا إلى الشريحة في الموضع (50، 50) بعرض وارتفاع 400 وحدة
IChart chart = slide.Shapes.AddChart(ChartType.Doughnut, 50, 50, 400, 400);
```
- **حدود:** `ChartType.Doughnut`، موضع x: 50، موضع y: 50، العرض: 400، الارتفاع: 400.

### ضبط حجم الحفرة

#### ملخص
بعد ذلك، سنقوم بتكوين حجم ثقب مخطط الدونات لجعله جذابًا بصريًا.

**تكوين حجم الحفرة:**
```csharp
// اضبط حجم الفتحة في مخطط الدونات على 90%
chart.ChartData.SeriesGroups[0].DoughnutHoleSize = 90;
```
- **تكوين المفتاح:** `DoughnutHoleSize` يحدد مقدار الجزء المقطوع من المركز. القيمة بين 0 و100 تمثل النسبة المئوية.

### احفظ عرضك التقديمي

وأخيرًا، احفظ التغييرات في ملف PowerPoint جديد:
```csharp
// قم بتحديد المسار الذي سيتم حفظ العرض التقديمي فيه
string outputPath = \@"YOUR_OUTPUT_DIRECTORY\DoughnutHoleSize_out.pptx";

// احفظ العرض التقديمي المعدل بتنسيق PPTX
presentation.Save(outputPath, SaveFormat.Pptx);
```
- **ملحوظة:** يستبدل `YOUR_OUTPUT_DIRECTORY` مع موقع الملف المطلوب.

### نصائح استكشاف الأخطاء وإصلاحها

- تأكد من تثبيت Aspose.Slides واستيراده بشكل صحيح.
- تأكد من وجود مسار دليل الإخراج الخاص بك قبل حفظ العرض التقديمي.

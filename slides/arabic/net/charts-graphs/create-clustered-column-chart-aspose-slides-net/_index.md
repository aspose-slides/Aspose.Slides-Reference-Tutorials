في هذا القسم، سنشرح كيفية إنشاء عرض تقديمي وإضافة مخطط عمودي مجمع.

ابدأ بإعداد مسار مجلد مستندك. هذا هو المكان الذي سيتم فيه حفظ العرض التقديمي المُنشأ:
```csharp

```

### إضافة مخطط عمودي مجمع إلى الشريحة

بعد ذلك، أضف مخططًا عموديًا مجمعًا إلى الشريحة الأولى في الموضع والحجم المحددين:
```csharp
// أضف مخططًا عموديًا مجمعًا عند (20، 20) بأبعاد (500 × 400)
IChart chart = pres.Slides[0].Shapes.AddChart(
    ChartType.ClusteredColumn,
    20, 20, 500, 400);
```
**توضيح:** يُنشئ هذا المقطع عرضًا تقديميًا فارغًا ويضيف مخططًا عموديًا مجمعًا. `AddChart` تحدد الطريقة نوع الرسم البياني (`ClusteredColumn`) وموقعها/أحجامها (x: 20، y: 20، العرض: 500، الارتفاع: 400).

### حفظ العرض التقديمي

وأخيرًا، احفظ العرض التقديمي الخاص بك للتأكد من تخزين كافة التغييرات:
```csharp
// احفظ العرض التقديمي في الدليل المحدد.
pres.Save(dataDir + "CreateAndAddChart_out.pptx");
```
**توضيح:** ال `Save` تكتب هذه الطريقة بيانات العرض التقديمي إلى ملف. عدّل المسار بما يتناسب مع بيئتك.

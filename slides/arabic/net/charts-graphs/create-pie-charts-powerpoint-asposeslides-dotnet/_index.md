# كيفية إنشاء مخططات دائرية وتخصيصها في PowerPoint باستخدام Aspose.Slides لـ .NET

## دليل التنفيذ
الآن بعد أن قمنا بإعداد بيئتنا، فلنبدأ في تنفيذ عملية إنشاء المخطط الدائري.

### إنشاء عرض تقديمي جديد
ابدأ بإنشاء مثيل جديد لـ `Presentation` الفئة التي تمثل ملف PowerPoint الخاص بك:

```csharp
using (Presentation presentation = new Presentation())
{
    // سيتم وضع باقي الكود الخاص بك هنا.
}
```

تؤدي هذه الخطوة إلى تهيئة عرض تقديمي فارغ حيث يمكنك إضافة الشرائح والأشكال.

### الوصول إلى الشرائح
انتقل إلى الشريحة الأولى لإضافة مخطط دائري. عادةً ما تكون هذه هي الشريحة الافتراضية المُنشأة مع كل عرض تقديمي جديد.

```csharp
ISlide slide = presentation.Slides[0];
```

الآن، دعنا ننتقل إلى إضافة مخططنا الدائري.

### إضافة مخطط دائري
يستخدم `AddChart` الطريقة على كائن الشريحة الخاص بك لإدراج مخطط دائري عند إحداثيات محددة (x، y) وأبعاد (العرض، الارتفاع):

```csharp
IChart chart = slide.Shapes.AddChart(ChartType.Pie, 100, 100, 400, 400);
```

### تكوين عنوان الرسم البياني
قم بتعيين عنوان لمخططك لتوفير السياق. `TextFrameForOverriding` يسمح لك بتخصيص محتواه وتنسيقه:

```csharp
chart.ChartTitle.AddTextFrameForOverriding("Sample Title");
chart.ChartTitle.TextFrameForOverriding.TextFrameFormat.CenterText = NullableBool.True;
chart.ChartTitle.Height = 20;
chart.HasTitle = true;
```

تعمل هذه الإعدادات على تركيز نص العنوان وتعيين الارتفاع المناسب لسهولة القراءة.

### إعداد تسميات البيانات
قم بتكوين تسميات البيانات لإظهار القيم داخل مخططك الدائري، مما يجعل من الأسهل على المشاهدين فهم مساهمة كل جزء:

```csharp
chart.ChartData.Series[0].Labels.DefaultDataLabelFormat.ShowValue = true;
```

يقوم هذا الخط بتعديل السلسلة الأولى لعرض قيم نقاط البيانات الخاصة بها مباشرة على شرائح الرسم البياني.

### إضافة الفئات والسلاسل
قم بمسح أي سلسلة أو فئات موجودة، ثم قم بتحديد سلاسل أو فئات جديدة مع نقاط البيانات الخاصة بك:

```csharp
IChartDataWorkbook fact = chart.ChartData.ChartDataWorkbook;

// مسح البيانات الموجودة مسبقًا
chart.ChartData.Series.Clear();
chart.ChartData.Categories.Clear();

// إضافة فئات جديدة
chart.ChartData.Categories.Add(fact.GetCell(0, 1, 0, "First Qtr"));
chart.ChartData.Categories.Add(fact.GetCell(0, 2, 0, "2nd Qtr"));
chart.ChartData.Categories.Add(fact.GetCell(0, 3, 0, "3rd Qtr"));

// إضافة سلسلة جديدة بنقاط البيانات
IChartSeries series = chart.ChartData.Series.Add(fact.GetCell(0, 0, 1, "Series 1"), chart.Type);
series.DataPoints.AddDataPointForPieSeries(fact.GetCell(0, 1, 1, 20));
series.DataPoints.AddDataPointForPieSeries(fact.GetCell(0, 2, 1, 50));
series.DataPoints.AddDataPointForPieSeries(fact.GetCell(0, 3, 1, 30));

// تنويع الألوان لكل شريحة
series.ParentSeriesGroup.IsColorVaried = true;
```

يتيح لك هذا الإعداد تخصيص الفئات (على سبيل المثال، الأرباع) ونقاط بيانات السلسلة (على سبيل المثال، النسب المئوية).

### حفظ العرض التقديمي
وأخيرًا، احفظ العرض التقديمي الخاص بك في الدليل المحدد:

```csharp
string outputDir = "YOUR_OUTPUT_DIRECTORY";
presentation.Save(outputDir + "/Pie.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
```

## دليل التنفيذ

### تغيير ألوان فئة الرسم البياني

لنركز على تغيير لون فئات مُحددة من المخططات. تُحسّن هذه الميزة تصوّر بياناتك من خلال إبراز نقاط البيانات الرئيسية بألوان مُختلفة.

#### إضافة مخطط إلى الشريحة الخاصة بك

أولاً، أضف مخططًا إلى شريحة العرض التقديمي الخاصة بك:

```csharp
// إضافة مخطط عمودي مجمع إلى الشريحة الأولى
IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 50, 50, 600, 400);
```

#### الوصول إلى نقاط البيانات

بعد ذلك، قم بالوصول إلى نقاط البيانات الفردية وتعديلها:

```csharp
// الوصول إلى نقطة البيانات الأولى في السلسلة الأولى من الرسم البياني
IChartDataPoint point = chart.ChartData.Series[0].DataPoints[0];

// اضبط نوع التعبئة على ثابت لتحسين رؤية الألوان
point.Format.Fill.FillType = FillType.Solid;

// قم بتغيير اللون إلى الأزرق للتأكيد البصري
point.Format.Fill.SolidFillColor.Color = Color.Blue;
```

#### حفظ العرض التقديمي الخاص بك

وأخيرًا، احفظ العرض التقديمي المعدّل:

```csharp
// حفظ العرض التقديمي مع التغييرات
pres.Save("YOUR_DOCUMENT_DIRECTORY/output.pptx", SaveFormat.Pptx);
```

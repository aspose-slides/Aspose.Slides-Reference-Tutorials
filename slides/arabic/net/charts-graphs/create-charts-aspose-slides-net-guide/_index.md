## دليل التنفيذ
في هذا القسم، سنرشدك خلال عملية إنشاء مخطط وتكوينه داخل شريحة PowerPoint.

### إنشاء مخطط بياني

#### ملخص
أتمت عرض البيانات في عروضك التقديمية بإضافة مخططات بيانية برمجيًا. سنشرح كيفية إنشاء مخطط LineWithMarkers باستخدام Aspose.Slides لـ .NET.

#### خطوات التنفيذ
1. **إعداد مسار دليل المستندات الخاص بك**
   قم بتحديد الدليل الذي سيتم تخزين ملفات العرض التقديمي فيه:
   ```csharp
   string dataDir = "YOUR_DOCUMENT_DIRECTORY";
   ```
2. **إنشاء مثيل عرض تقديمي جديد**
   إنشاء كائن عرض تقديمي جديد للعمل معه:
   ```csharp
   Presentation pres = new Presentation(dataDir + "Test.pptx");
   ```
3. **الوصول إلى الشريحة الأولى من العرض التقديمي**
   استرجاع الشريحة الأولى من العرض التقديمي:
   ```csharp
   ISlide slide = pres.Slides[0];
   ```
4. **إضافة مخطط إلى الشريحة**
   أضف مخطط LineWithMarkers في الموضع (0، 0) بحجم (400، 400):
   ```csharp
   IChart chart = slide.Shapes.AddChart(ChartType.LineWithMarkers, 0, 0, 400, 400);
   ```
5. **مسح السلسلة الموجودة في الرسم البياني**
   تأكد من أن الرسم البياني يبدأ بدون أي بيانات:
   ```csharp
   chart.ChartData.Series.Clear();
   ```
6. **الوصول إلى مصنف بيانات الرسم البياني**
   استرداد المصنف المرتبط ببيانات الرسم البياني:
   ```csharp
   int defaultWorksheetIndex = 0;
   IChartDataWorkbook fact = chart.ChartData.ChartDataWorkbook;
   ```
7. **إضافة سلسلة جديدة إلى الرسم البياني**
   أضف سلسلة إلى الرسم البياني وحدد نوعها:
   ```csharp
   chart.ChartData.Series.Add(fact.GetCell(defaultWorksheetIndex, 1, 1, "Series 1"), chart.Type);
   ```

#### خيارات تكوين المفاتيح
- **نوع الرسم البياني**:اختر من بين أنواع مختلفة مثل الشريط، والفطيرة، والخط، وما إلى ذلك، استنادًا إلى احتياجاتك من البيانات.
- **الموقع والحجم**:قم بتخصيص موضع الرسم البياني وحجمه ليتناسب مع تخطيط الشريحة لديك.

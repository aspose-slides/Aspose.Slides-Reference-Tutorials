## دليل التنفيذ: إضافة حدود مستديرة إلى المخططات البيانية

### ملخص

سنبدأ بإنشاء مخطط عمودي مجمع، ثم نضيف زوايا مستديرة لحدوده. تُحسّن هذه العملية جمالية العرض، مما يجعل عرض بياناتك أكثر جاذبية.

#### الخطوة 1: إنشاء عرض تقديمي جديد

```csharp
using Aspose.Slides;
using Aspose.Slides.Charts;

// تحديد الدليل لحفظ المخرجات
string dataDir = "YOUR_DOCUMENT_DIRECTORY";

// إنشاء كائن عرض تقديمي
using (Presentation presentation = new Presentation())
{
    // انتقل إلى إضافة الرسم البياني...
```

#### الخطوة 2: إضافة مخطط إلى الشريحة الخاصة بك

انتقل إلى الشريحة الأولى وأضف مخططًا عموديًا مجمعًا:

```csharp
    ISlide slide = presentation.Slides[0];
    
    // أضف الرسم البياني في الموضع (20، 100) بالحجم (600، 400)
    IChart chart = slide.Shapes.AddChart(ChartType.ClusteredColumn, 20, 100, 600, 400);
```

#### الخطوة 3: تكوين تنسيق خط الرسم البياني

ضبط تنسيق الخط لضمان وجود حدود صلبة:

```csharp
    // نوع التعبئة الصلبة للخطوط ذات النمط الواحد
    chart.LineFormat.FillFormat.FillType = FillType.Solid;
    chart.LineFormat.Style = LineStyle.Single;
```

#### الخطوة 4: تمكين الزوايا المستديرة

تفعيل ميزة الزوايا المستديرة:

```csharp
    // تطبيق حدود مستديرة على منطقة الرسم البياني
    chart.HasRoundedCorners = true;
    
    // احفظ العرض التقديمي الخاص بك
    presentation.Save(dataDir + "out.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
}
```

### خيارات تكوين المفاتيح
- **نوع التعبئة**:يحدد ما إذا كانت الحدود صلبة أو ذات نمط آخر.
- **نمط الخط**:يحدد سمك الحدود.
- **زوايا مستديرة**:يسمح بزوايا مستديرة لتحسين المظهر الجمالي.

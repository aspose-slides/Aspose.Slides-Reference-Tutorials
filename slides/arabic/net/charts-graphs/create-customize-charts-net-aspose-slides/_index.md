## دليل التنفيذ
سنستعرض كيفية إنشاء العروض التقديمية وإضافة المخططات البيانية باستخدام Aspose.Slides لـ .NET.

### الميزة 1: إنشاء العرض التقديمي وإضافة المخططات

#### ملخص:
توضح هذه الميزة كيفية إنشاء عرض تقديمي وإضافة مخطط عمودي مجمع إلى الشريحة الأولى. تُعد المخططات البيانية أساسية لعرض اتجاهات البيانات بفعالية.

#### التنفيذ خطوة بخطوة:

##### 1. تحديد المسار لحفظ المستندات
ابدأ بتحديد المكان الذي تريد حفظ ملفاتك فيه.

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
```

##### 2. إنشاء كائن عرض تقديمي جديد
إنشاء مثيل لـ `Presentation` صف لبدء صياغة العرض التقديمي الخاص بك.

```csharp
Presentation pres = new Presentation();
```

##### 3. الوصول إلى الشريحة الأولى
احصل على إمكانية الوصول إلى الشريحة الأولى في العرض التقديمي الخاص بك باستخدام:

```csharp
ISlide slide = pres.Slides[0];
```

##### 4. إضافة مخطط عمودي مجمع
أضف مخططًا إلى الموضع المطلوب على الشريحة.

```csharp
IChart chart = slide.Shapes.AddChart(ChartType.ClusteredColumn, 50, 50, 500, 400);
```
يؤدي هذا إلى إضافة مخطط عمودي مجمع عند الإحداثيات (50، 50) بأبعاد 500 × 400 بكسل.

##### 5. احفظ العرض التقديمي
وأخيرًا، احفظ العرض التقديمي الخاص بك في الدليل المحدد.

```csharp
pres.Save(dataDir + "CreatePresentationWithChart_out.pptx", SaveFormat.Pptx);
```

### الميزة 2: إعداد تنسيق الأرقام المحدد مسبقًا لنقاط بيانات الرسم البياني

#### ملخص:
تعرف على كيفية تعيين تنسيق رقمي محدد مسبقًا (على سبيل المثال، النسبة المئوية) لنقاط البيانات في سلسلة المخططات، مما يعزز قابلية قراءة المخططات الخاصة بك.

#### التنفيذ خطوة بخطوة:

##### 1. الوصول إلى السلسلة وعبورها
بعد إضافة الرسم البياني الخاص بك، قم بالوصول إلى مجموعة السلسلة الخاصة به.

```csharp
IChartSeriesCollection series = chart.ChartData.Series;
```

##### 2. تنسيق كل نقطة بيانات
قم بتعيين تنسيق الرقم لكل نقطة بيانات في السلسلة إلى "0.00%".

```csharp
foreach (ChartSeries ser in series)
{
    foreach (IChartDataPoint cell in ser.DataPoints)
    {
        // تعيين تنسيق الأرقام لتحسين قابلية القراءة
        cell.Value.AsCell.PresetNumberFormat = 10; // تنسيق 0.00%
    }
}
```

##### 3. احفظ العرض التقديمي بأرقام منسقة

```csharp
pres.Save(dataDir + "SetPresetNumberFormat_out.pptx", SaveFormat.Pptx);
```

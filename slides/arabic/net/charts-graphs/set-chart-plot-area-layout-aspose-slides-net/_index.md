## دليل التنفيذ
### إعداد مخطط تخطيط مساحة الرسم البياني
يتيح لك تكوين تخطيط منطقة الرسم البياني ضبط كيفية ملاءمة تصور البيانات داخل الحاوية الخاصة بها.

#### الخطوة 1: إنشاء شريحة والوصول إليها
تأكد من أن العرض التقديمي الخاص بك يحتوي على شريحة واحدة على الأقل:
```csharp
using Aspose.Slides;
// تهيئة مثيل عرض تقديمي جديد
Presentation presentation = new Presentation();
// الوصول إلى الشريحة الأولى في العرض التقديمي
ISlide slide = presentation.Slides[0];
```

#### الخطوة 2: إضافة مخطط إلى الشريحة
أضف مخططًا عموديًا مجمعًا عند إحداثيات محددة مع أبعاد معينة:
```csharp
// أضف مخططًا عموديًا مجمعًا في الموضع (20، 100) بحجم (600 × 400)
IChart chart = slide.Shapes.AddChart(ChartType.ClusteredColumn, 20, 100, 600, 400);
```

#### الخطوة 3: تكوين تخطيط منطقة الرسم
تعيين خصائص التخطيط لمنطقة الرسم البياني:
```csharp
// تعيين التخطيط كجزء من المساحة المتوفرة
chart.PlotArea.AsILayoutable.X = 0.2f;
chart.PlotArea.AsILayoutable.Y = 0.2f;
chart.PlotArea.AsILayoutable.Width = 0.7f;
chart.PlotArea.AsILayoutable.Height = 0.7f;
// تحديد التخطيط بالنسبة للمنطقة الداخلية
chart.PlotArea.LayoutTargetType = LayoutTargetType.Inner;
```

#### الخطوة 4: حفظ العرض التقديمي
احفظ العرض التقديمي الخاص بك:
```csharp
// تحديد دليل المستند واسم الملف
string dataDir = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "SetLayoutMode_outer.pptx");
presentation.Save(dataDir, Aspose.Slides.Export.SaveFormat.Pptx);
```
يضمن هذا التكوين أن مساحة الرسم تتكيف ديناميكيًا لتناسب مساحتها المخصصة بكفاءة.

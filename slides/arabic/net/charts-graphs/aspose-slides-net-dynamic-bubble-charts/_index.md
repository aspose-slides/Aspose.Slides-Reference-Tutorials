## دليل التنفيذ
### إضافة مخطط فقاعي إلى العرض التقديمي الخاص بك
للبدء، ستحتاج إلى إضافة مخطط فقاعي إلى الشريحة الخاصة بك:

#### الخطوة 1: إنشاء عرض تقديمي أو فتحه
```csharp
using Aspose.Slides;
using Aspose.Slides.Charts;
// تعيين مسار الدليل لحفظ المستندات
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
// إنشاء مثيل عرض تقديمي جديد
using (Presentation pres = new Presentation())
{
    // أضف مخططًا فقاعيًا إلى الشريحة الأولى في الموضع (50، 50) بعرض وارتفاع 600 × 400 بكسل
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.Bubble, 50, 50, 600, 400, true);
```
#### الخطوة 2: تكوين تمثيل حجم الفقاعة
اضبط حجم الفقاعة لتمثيل بُعد بيانات محدد. يستخدم هذا المثال `Width` ملكية:
```csharp
    // تعيين حجم الفقاعة التمثيلية بناءً على "العرض"
    chart.ChartData.SeriesGroups[0].BubbleSizeRepresentation = BubbleSizeRepresentationType.Width;
```
#### الخطوة 3: احفظ العرض التقديمي الخاص بك
وأخيرًا، احفظ عرضك التقديمي لرؤية التغييرات التي تنعكس في مخططاتك.
```csharp
    // حفظ العرض التقديمي المعدل
    pres.Save(dataDir + "Presentation_BubbleSizeRepresentation.pptx");
}
```
### خيارات تكوين المفاتيح
- **نوع تمثيل حجم الفقاعة**:اختر بين `Width`، `Height`، أو `Volume` بناءً على خصائص بياناتك.
- **نوع المخطط.فقاعة**:ضروري لإنشاء مخططات الفقاعات التي يمكنها تمثيل أبعاد متعددة للبيانات.

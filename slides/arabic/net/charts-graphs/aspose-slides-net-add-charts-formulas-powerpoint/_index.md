## دليل التنفيذ

### إنشاء مخطط وإضافته إلى العرض التقديمي (H2)
#### ملخص:
سنبدأ بإنشاء عرض تقديمي جديد على PowerPoint وإضافة مخطط عمودي مجمع. سيشكل هذا أساسًا لمزيد من معالجة البيانات.

**الخطوة 1: إنشاء عرض تقديمي جديد**
```csharp
using System;
using Aspose.Slides;

// تهيئة عرض تقديمي جديد
Presentation presentation = new Presentation();
```
- **غاية**: يقوم بتهيئة مثيل لـ `Presentation` الفئة التي تمثل ملف PowerPoint.

**الخطوة 2: إضافة مخطط عمودي مجمع**
```csharp
using Aspose.Slides.Charts;

// أضف مخططًا إلى الشريحة الأولى عند الإحداثيات (150، 150) بحجم (500 × 300)
IChart chart = presentation.Slides[0].Shapes.AddChart(
    ChartType.ClusteredColumn, 150, 150, 500, 300);
```
- **شرح المعلمات**:
  - `ChartType.ClusteredColumn`:يحدد نوع الرسم البياني.
  - الإحداثيات والحجم: يحدد مكان ومدى حجم الرسم البياني الذي سيظهر على الشريحة.

### مصنف بيانات مخطط الوصول (H2)
#### ملخص:
يتيح لك الوصول إلى مصنف البيانات التعامل مع البيانات الأساسية للرسم البياني بشكل مباشر، وهو أمر بالغ الأهمية لتعيين الصيغ وتحديث القيم بشكل ديناميكي.

**الخطوة 1: استرداد مصنف بيانات الرسم البياني**
```csharp
using Aspose.Slides.Charts;

// الوصول إلى مخطط الشريحة الأولى
IChart chart = presentation.Slides[0].Shapes[0] as IChart;
IChartDataWorkbook workbook = chart.ChartData.ChartDataWorkbook;
```
- **لماذا**:يمنحك هذا التحكم في خلايا البيانات في الرسم البياني الخاص بك، مما يتيح لك المزيد من التخصيص وإعداد الصيغة.

### تعيين الصيغة في خلية بيانات الرسم البياني (H2)
#### ملخص:
يتيح لك إعداد الصيغ إجراء حسابات ديناميكية داخل مخططاتك. يمكنك استخدام صيغ قياسية مشابهة لصيغ Excel ومراجع بنمط R1C1.

**الخطوة 1: تعيين صيغة المجموع**
```csharp
using Aspose.Slides.Charts;

// تعيين الصيغة لحساب "1 + SUM(F2:H5)" في الخلية B2
IChartDataCell cell1 = workbook.GetCell(0, "B2");
cell1.Formula = "1 + SUM(F2:H5)";
```
- **غاية**:يوضح إعداد عملية حسابية أساسية مقترنة بمجموع النطاق.

**الخطوة 2: استخدام صيغة نمط R1C1**
```csharp
// تعيين صيغة لتقسيم القيمة القصوى في نطاق على 3 في الخلية C2
IChartDataCell cell2 = workbook.GetCell(0, "C2");
cell2.R1C1Formula = "MAX(R2C6:R5C8) / 3";
```
- **لماذا**:يوضح كيفية استخدام المراجع النسبية لإجراء حسابات أكثر تعقيدًا.

### حساب الصيغ في مصنف بيانات الرسم البياني (H2)
#### ملخص:
بعد تعيين الصيغ، تحتاج إلى حسابها لتحديث عرض بيانات الرسم البياني.

**الخطوة 1: حساب الصيغ**
```csharp
using Aspose.Slides.Charts;

// تحديث قيم خلايا الرسم البياني استنادًا إلى الصيغ المحسوبة
workbook.CalculateFormulas();
```
- **لماذا**:يضمن أن الرسم البياني الخاص بك يعكس أحدث الحسابات، مما يجعله دقيقًا وحديثًا.

### حفظ العرض التقديمي (H2)
#### ملخص:
أخيرًا، احفظ عرضك التقديمي في مكان محدد. هذه الخطوة أساسية لحفظ عملك.

**الخطوة 1: تحديد مسار الإخراج**
```csharp
using System.IO;
using Aspose.Slides;

// حدد المسار لحفظ العرض التقديمي
string outpptxFile = Path.Combine("YOUR_OUTPUT_DIRECTORY", "ChartDataCell_Formulas_out.pptx");
```

**الخطوة 2: حفظ العرض التقديمي**
```csharp
// حفظ بتنسيق PPTX
presentation.Save(outpptxFile, SaveFormat.Pptx);
```
- **لماذا**:يثبت تغييراتك عن طريق حفظها في ملف PowerPoint جديد.

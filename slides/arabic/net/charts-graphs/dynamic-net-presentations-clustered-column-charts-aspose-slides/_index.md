## دليل التنفيذ

### الميزة 1: إنشاء عرض تقديمي وإضافة مخطط

#### ملخص
يتيح إنشاء العروض التقديمية برمجيًا إمكانية الأتمتة والتخصيص. توضح هذه الميزة كيفية تهيئة عرض تقديمي وإضافة مخطط عمودي مجمع، وهو مثالي لمقارنة البيانات عبر الفئات.

#### التنفيذ خطوة بخطوة

**تهيئة العرض التقديمي**
```csharp
Presentation pres = new Presentation();
```

**الوصول إلى الشريحة الأولى**
ابدأ بالشريحة الأولى:
```csharp
ISlide slide = pres.Slides[0];
```

**إضافة مخطط عمودي مجمع**
قم بإدراج مخطط في الموضع (100، 100) على الشريحة بأبعاد 600 × 450 بكسل.
```csharp
IChart ch = slide.Shapes.AddChart(ChartType.ClusteredColumn, 100, 100, 600, 450);
```
*توضيح:* تُنشئ هذه الطريقة مخططًا عموديًا جديدًا مُجمّعًا. تُحدد المعلمات موقعه وحجمه.

**مسح السلاسل والفئات الموجودة**
للبدء ببيانات جديدة:
```csharp
ch.ChartData.Series.Clear();
ch.ChartData.Categories.Clear();
```

### الميزة 2: إضافة فئات مع مستويات التجميع

#### ملخص
يؤدي تنظيم بياناتك في فئات ذات مستويات تجميع إلى تحسين قابلية القراءة والبنية، وهو أمر حيوي للعروض التقديمية الفعالة.

**إنشاء الفئات وتعيين مستويات التجميع**
التكرار عبر نطاق لإنشاء الفئات:
```csharp
IChartDataWorkbook fact = ch.ChartData.ChartDataWorkbook;
fact.Clear(0);

int defaultWorksheetIndex = 0;

for (int i = 2; i <= 9; i++)
{
    IChartCategory category = ch.ChartData.Categories.Add(fact.GetCell(0, "c" + i, System.Convert.ToChar('A' + (i - 2))));
    
    string groupName = "Group" + ((i - 1) / 2 + 1);
    category.GroupingLevels.SetGroupingItem(1, groupName);
}
```
*توضيح:* تضيف هذه الحلقة فئات بمستويات تجميع فريدة، مما يعزز الهيكل الهرمي للرسم البياني.

### الميزة 3: إضافة سلسلة ونقاط بيانات إلى الرسم البياني

#### ملخص
يُعدّ ملء مخططك بنقاط البيانات أمرًا بالغ الأهمية للتمثيل المرئي. تتضمن هذه الخطوة إضافة سلسلة من البيانات التي تتوافق مع كل فئة.

**إضافة سلسلة وملء البيانات**
```csharp
IChartSeries series = ch.ChartData.Series.Add(fact.GetCell(0, "D1", "Series 1"), ChartType.ClusteredColumn);

for (int j = 2; j <= 9; j++)
{
    series.DataPoints.AddDataPointForBarSeries(fact.GetCell(defaultWorksheetIndex, "D" + j, j * 10));
}
```
*توضيح:* يضيف هذا الكود سلسلة بيانات جديدة ويملأها بالنقاط. تُمثل كل نقطة قيمةً مُشتقةً من موقع الخلية.

### الميزة 4: حفظ العرض التقديمي مع الرسم البياني

#### ملخص
بمجرد أن يصبح الرسم البياني جاهزًا، يؤدي حفظ العرض التقديمي إلى الحفاظ على جميع التغييرات ويسمح لك بمشاركة البيانات أو تقديمها.

**احفظ عملك**
```csharp
pres.Save(outputPath + "/AsposeChart_out.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
```
*توضيح:* ال `Save` تقوم الطريقة بإرسال عملك إلى ملف PPTX، مما يجعله جاهزًا للتوزيع أو العرض التقديمي.

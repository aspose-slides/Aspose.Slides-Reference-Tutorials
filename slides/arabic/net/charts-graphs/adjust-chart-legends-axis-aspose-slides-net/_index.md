## دليل التنفيذ
يرشدك هذا القسم خلال كل ميزة خطوة بخطوة.

### ضبط حجم خط التسمية التوضيحية
يُحسّن تعديل حجم خط التسمية التوضيحية سهولة القراءة. إليك الطريقة:

#### ملخص
سوف نقوم بتعديل حجم الخط الخاص بنص الأسطورة في الرسم البياني باستخدام Aspose.Slides لـ .NET.

#### خطوات
**1. قم بتحميل العرض التقديمي الخاص بك:**
ابدأ بتحميل ملف PowerPoint الخاص بك حيث تريد ضبط أساطير الرسم البياني.
```csharp
using (Presentation pres = new Presentation(dataDir))
{
    // انتقل إلى الشريحة الأولى وأضف مخططًا عموديًا مجمعًا.
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 50, 50, 600, 400);
```

**2. تعيين حجم خط الأسطورة:**
قم بتحديد ارتفاع الخط المطلوب لتحسين الرؤية.
```csharp
    // قم بضبط حجم الخط الخاص بنص الأسطورة إلى 20.
    chart.Legend.TextFormat.PortionFormat.FontHeight = 20;
}
```
- **توضيح:** `FontHeight` تعيين الحجم بالنقاط، مما يعزز قابلية القراءة.

**3. احفظ العرض التقديمي الخاص بك:**
بعد إجراء التغييرات، احفظ العرض التقديمي للحفاظ عليها.
```csharp
pres.Save(outputDir, Aspose.Slides.Export.SaveFormat.Pptx);
```

### تكوين الحد الأدنى والحد الأقصى لقيم المحور الرأسي
تتيح لك تخصيص قيم المحور الحصول على تمثيل دقيق للبيانات.

#### ملخص
تعرف على كيفية تعيين الحد الأدنى والأقصى لقيم المحور الرأسي للرسم البياني الخاص بك.

#### خطوات
**1. قم بتحميل العرض التقديمي الخاص بك:**
كما في السابق، افتح العرض التقديمي الذي يحتوي على الرسم البياني الخاص بك.
```csharp
using (Presentation pres = new Presentation(dataDir))
{
    IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.ClusteredColumn, 50, 50, 600, 400);
```

**2. تعيين قيم المحور المخصصة:**
قم بتعطيل إعدادات قيمة المحور التلقائية وقم بتحديد إعداداتك الخاصة.
```csharp
    // تعطيل الحد الأدنى التلقائي للمحور الرأسي.
    chart.Axes.VerticalAxis.IsAutomaticMinValue = false;
    // تعيين قيمة الحد الأدنى المخصصة لـ -5.
    chart.Axes.VerticalAxis.MinValue = -5;

    // وبالمثل، قم بتعطيل الحد الأقصى التلقائي وضبطه على 10.
    chart.Axes.VerticalAxis.IsAutomaticMaxValue = false;
    chart.Axes.VerticalAxis.MaxValue = 10;
}
```
- **توضيح:** يتيح تخصيص هذه القيم إمكانية تعديل حجم البيانات حسب الطلب.

**3. احفظ العرض التقديمي الخاص بك:**
تأكد من حفظ التغييرات عن طريق الكتابة مرة أخرى في الملف.
```csharp
pres.Save(outputDir, Aspose.Slides.Export.SaveFormat.Pptx);
```

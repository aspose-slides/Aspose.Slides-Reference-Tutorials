# كيفية إنشاء مخطط دائري وإضافته إلى عرض تقديمي باستخدام Aspose.Slides لـ .NET
## مقدمة
غالبًا ما يتطلب إنشاء عروض تقديمية جذابة أكثر من مجرد نص؛ فالعناصر المرئية، مثل المخططات البيانية، يمكن أن تُعزز بشكل كبير تأثير سرد بياناتك. إذا كنت ترغب في إضافة مخططات دائرية ديناميكية إلى عروض PowerPoint التقديمية برمجيًا، **Aspose.Slides لـ .NET** أداة فعّالة تُسهّل هذه المهمة وتُحسّنها. سيرشدك هذا البرنامج التعليمي إلى كيفية إضافة مخطط دائري إلى شريحة عرض تقديمي وتهيئته باستخدام مصادر بيانات خارجية.

### ما سوف تتعلمه
- كيفية إنشاء عرض تقديمي جديد باستخدام Aspose.Slides لـ .NET
- إضافة مخطط دائري إلى الشريحة الأولى
- تعيين عنوان URL لمصنف عمل خارجي كمصدر بيانات للرسم البياني الخاص بك
- حفظ العرض التقديمي الخاص بك بتنسيق PPTX
دعونا نتعمق في كيفية تحقيق ذلك بسهولة، بدءًا بالمتطلبات الأساسية.
## المتطلبات الأساسية
قبل أن نبدأ، تأكد من أن لديك ما يلي جاهزًا:
- **Aspose.Slides لـ .NET** تم تثبيت المكتبة. ستحتاج إلى إصدار متوافق مع .NET Framework أو .NET Core/.NET 5+.
- المعرفة الأساسية ببرمجة C# والتعرف على Visual Studio IDE.
- بيئة تطوير تم إعدادها على جهازك (Windows أو macOS أو Linux).
## إعداد Aspose.Slides لـ .NET
### تعليمات التثبيت
يمكن إضافة Aspose.Slides for .NET إلى مشروعك باستخدام طرق مختلفة:
**.NET CLI**
```shell
dotnet add package Aspose.Slides
```
**وحدة تحكم مدير الحزم**
```powershell

```
**واجهة مستخدم مدير الحزم NuGet**

2. ابحث عن "Aspose.Slides".
3. قم بتثبيت الإصدار الأحدث.

## دليل التنفيذ
الآن بعد أن قمت بالإعداد، دعنا نتناول كل ميزة خطوة بخطوة.
### إنشاء مخطط وإضافته إلى العرض التقديمي
#### ملخص
سنبدأ بإنشاء عرض تقديمي وإضافة مخطط دائري إلى الشريحة الأولى.
#### خطوات:
1. **تهيئة العرض التقديمي**
   ابدأ بإنشاء مثيل لـ `Presentation` الفئة التي تمثل ملف PowerPoint الخاص بك.
   ```csharp
   using Aspose.Slides;
   
   string dataDir = "YOUR_DOCUMENT_DIRECTORY";
   
   using (Presentation pres = new Presentation())
   {
       // هذا هو المكان الذي سنضيف فيه مخططنا.
   }
   ```
2. **إضافة مخطط دائري**
   استخدم `Shapes.AddChart` طريقة لإدراج مخطط دائري عند إحداثيات محددة على الشريحة الخاصة بك.
   ```csharp
   IChart chart = pres.Slides[0].Shapes.AddChart(
       ChartType.Pie, 50, 50, 400, 600, true);
   ```
### تعيين مصنف خارجي لبيانات الرسم البياني
#### ملخص
الآن دعنا نقوم بتكوين مخطط الفطيرة لاستخدام البيانات من مصنف خارجي.
#### خطوات:
1. **بيانات مخطط الوصول**
   استرداد واجهة بيانات الرسم البياني حيث ستحدد عنوان URL لمصدر البيانات الخارجي الخاص بك.
   ```csharp
   IChartData chartData = chart.ChartData;
   ```
2. **تعيين عنوان URL للمصنف الخارجي**
   قم بتعيين عنوان URL لمصدر البيانات الخاص بك باستخدام `SetExternalWorkbook`يستخدم هذا المثال عنوان URL مؤقتًا، والذي يجب استبداله بمسار مصدر البيانات الفعلي الخاص بك.
   ```csharp
   (chartData as ChartData).SetExternalWorkbook("http://المسار/غير/موجود"، خطأ)؛
   ```
### حفظ العرض التقديمي في ملف
#### ملخص
وأخيرًا، احفظ العرض التقديمي بتنسيق PPTX في الموقع المطلوب.
#### خطوات:
1. **حفظ العرض التقديمي**
   استخدم `Save` طريقة `Presentation` الفئة لكتابة الملف على القرص.
   ```csharp
   pres.Save(dataDir + "SetExternalWorkbookWithUpdateChartData.pptx", SaveFormat.Pptx);
   ```

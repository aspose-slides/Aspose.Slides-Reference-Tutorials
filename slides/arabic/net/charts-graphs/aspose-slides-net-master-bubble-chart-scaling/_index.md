## دليل التنفيذ

### الميزة: دعم مقياس مخطط الفقاعات

#### ملخص
في هذا القسم، سنتناول ضبط مقياس حجم الفقاعة في مخطط الفقاعات باستخدام **Aspose.Slides**تُعد هذه الميزة بالغة الأهمية عندما تحتاج إلى التحكم الدقيق في كيفية تمثيل نقاط البيانات بصريًا على الشرائح الخاصة بك.

##### الخطوة 1: إنشاء كائن عرض تقديمي
ابدأ بإنشاء مثيل جديد لـ `Presentation` فصل:

```csharp
using Aspose.Slides;

string dataDir = "YOUR_DOCUMENT_DIRECTORY";

// تهيئة كائن العرض التقديمي
using (Presentation pres = new Presentation())
{
    // سيتم تنفيذ خطوات أخرى داخل هذه الكتلة
}
```

تؤدي هذه الخطوة إلى إعداد البيئة الخاصة بك للعمل مع الشرائح.

##### الخطوة 2: إضافة مخطط فقاعي
أضف مخططًا فقاعيًا إلى الشريحة الأولى عند إحداثيات وأبعاد محددة:

```csharp
// أضف مخططًا فقاعيًا في الموضع (100، 100) بحجم (400 × 300)
IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.Bubble, 100, 100, 400, 300);
```

تضيف مقتطفات التعليمات البرمجية هذه مخطط الفقاعات الأولي إلى الشريحة الخاصة بك.

##### الخطوة 3: ضبط مقياس حجم الفقاعة
قم بتكوين مقياس حجم الفقاعة لمجموعة السلسلة الأولى:

```csharp
// اضبط مقياس حجم الفقاعة على 150
chart.ChartData.SeriesGroups[0].BubbleSizeScale = 150;
```

ضبط `BubbleSizeScale` يتيح لك التحكم في مقدار انعكاس حجم كل نقطة بيانات على قيمتها الأساسية.

##### الخطوة 4: حفظ العرض التقديمي
وأخيرًا، احفظ العرض التقديمي الخاص بك باستخدام هذه الإعدادات:

```csharp
// احفظ العرض التقديمي المعدل pres.Save(dataDir + "Result.pptx");
```

تؤدي هذه الخطوة إلى حفظ جميع التغييرات التي أجريتها على ملف العرض التقديمي في دليل محدد.

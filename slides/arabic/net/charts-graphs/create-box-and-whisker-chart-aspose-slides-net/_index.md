## دليل التنفيذ
سنقوم بتقسيم عملية إنشاء مخطط الصندوق والشارب باستخدام Aspose.Slides إلى أقسام قابلة للإدارة.

### إنشاء مخطط الصندوق والشارب
#### ملخص
تتيح لك هذه الميزة إنشاء مخطط تفصيلي للصندوق والشارب برمجيًا في PowerPoint، مع بيانات وتكوينات مخصصة.

#### التنفيذ خطوة بخطوة
##### 1. تحديد دليل المستندات
ابدأ بتحديد الدليل الذي يوجد فيه ملف العرض التقديمي الخاص بك أو الذي سيتم حفظه فيه:
```csharp
string dataDir = @"YOUR_DOCUMENT_DIRECTORY";
```
يضمن هذا المسار أن يعرف البرنامج النصي الخاص بك مكان القراءة من الملفات أو الكتابة إليها.

##### 2. تحميل أو إنشاء عرض تقديمي
افتح عرض تقديمي PowerPoint موجودًا، أو قم بإنشاء عرض تقديمي جديد إذا لزم الأمر:
```csharp
using (Presentation pres = new Presentation(dataDir + "test.pptx"))
{
    // يوجد هنا الكود الخاص بإضافة الرسم البياني وتكوينه.
}
```
##### 3. أضف مخطط الصندوق والشارب إلى الشريحة
أدخل مخطط الصندوق والشارب في الشريحة الأولى في الموضع `(50, 50)` مع الأبعاد `500 x 400`:
```csharp
IChart chart = pres.Slides[0].Shapes.AddChart(ChartType.BoxAndWhisker, 50, 50, 500, 400);
```
تتضمن هذه الخطوة تحديد الشريحة المطلوبة وتكوين الموضع الأولي للرسم البياني الخاص بك.
##### 4. مسح البيانات الموجودة
قم بإزالة أي فئات أو سلاسل موجودة للبدء بصفحة نظيفة:
```csharp
chart.ChartData.Categories.Clear();
chart.ChartData.Series.Clear();
```
تضمن عملية المسح عدم تكرار البيانات عن غير قصد عند إضافة إدخالات جديدة.
##### 5. مصنف مخطط الوصول
استخدم المصنف المرتبط ببيانات الرسم البياني الخاص بك لمزيد من المعالجة:
```csharp
IChartDataWorkbook wb = chart.ChartData.ChartDataWorkbook;
```
يعمل المصنف كحاوية حيث يمكنك إضافة بيانات الرسم البياني أو تعديلها برمجيًا.
##### 6. مسح بيانات المصنف
تأكد من عدم وجود خلايا متبقية عن طريق المسح من الفهرس الأولي:
```csharp
wb.Clear(0);
```
##### 7. إضافة فئات إلى الرسم البياني
قم بالتكرار وملء الفئات الخاصة بالرسم البياني الخاص بك، وأضف كل فئة كصف جديد في العمود A:
```csharp
for (int i = 1; i <= 6; i++)
{
    chart.ChartData.Categories.Add(wb.GetCell(0, "A" + i, "Category 1"));
}
```
تتيح لك هذه الخطوة تنظيم فئات البيانات الخاصة بك بشكل منهجي داخل الرسم البياني.

#### خيارات تكوين المفاتيح
- **نوع الرسم البياني:** يختار `ChartType.BoxAndWhisker` لإنشاء مخططات الصندوق والشارب.
- **التموضع والحجم:** ضبط الوضع `(50, 50)` والحجم `(500, 400)` بناءً على متطلبات تخطيط الشريحة.
- **إدارة البيانات:** استخدم المصنف لإدارة البيانات بكفاءة.

### نصائح استكشاف الأخطاء وإصلاحها
تتضمن المشكلات الشائعة التي قد تواجهها ما يلي:

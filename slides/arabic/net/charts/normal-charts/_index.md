## تعليمات التثبيت

1. التثبيت عبر مدير حزمة NuGet:
   ```
   Install-Package Aspose.Slides
   ```

   أو باستخدام .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

2. إضافة مساحات الأسماء:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Charts;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

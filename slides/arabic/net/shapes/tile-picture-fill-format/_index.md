# إنشاء خلفيات صور مبلطة في PowerPoint باستخدام Aspose.Slides

يوفر Aspose.Slides for .NET إمكانيات فعّالة لإنشاء عروض PowerPoint التقديمية وتعديلها برمجيًا. يوضح هذا المثال كيفية إنشاء عرض تقديمي بشكل مستطيل مملوء بنمط صورة مُبلّطة. يوضح الكود كيفية تحميل صورة، وإضافتها إلى شريحة كخلفية للمستطيل، وتطبيق خصائص تبليط متنوعة، مثل خيارات الإزاحة والقياس والمحاذاة والانعكاس. تُعد هذه التقنية مفيدة لإنشاء خلفيات مخصصة، أو علامات مائية، أو عناصر زخرفية في عروضك التقديمية مع تحكم دقيق في سلوك التبليط.

## تعليمات التثبيت

1. قم بتثبيت Aspose.Slides لـ .NET باستخدام NuGet Package Manager:
   ```
   Install-Package Aspose.Slides.NET
   ```

2. أو قم بالتثبيت عبر .NET CLI:
   ```
   dotnet add package Aspose.Slides.NET
   ```

3. أضف التوجيهات التالية إلى ملف C# الخاص بك:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.IO;
   ```

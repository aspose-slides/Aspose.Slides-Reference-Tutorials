# إنشاء أشكال باستخدام تعبئة الصور المبلطة في PowerPoint باستخدام Aspose.Slides

يوفر Aspose.Slides لـ .NET إمكانيات فعّالة لإنشاء عروض PowerPoint التقديمية وتعديلها برمجيًا. يوضح هذا المثال كيفية إنشاء شكل مستطيل وتعبئته بنمط صورة مُبلّطة. يُعدّ نمط الصورة المُبلّطة مفيدًا بشكل خاص لإنشاء خلفيات مُزخرفة، أو علامات مائية، أو عناصر زخرفية في عروضك التقديمية.

## تعليمات التثبيت

1. التثبيت عبر مدير حزمة NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. التثبيت عبر .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. التثبيت اليدوي:
   - قم بتنزيل أحدث إصدار من [Aspose.Slides لـ .NET](https://releases.aspose.com/slides/net/)
   - أضف مرجعًا إلى Aspose.Slides.dll في مشروعك

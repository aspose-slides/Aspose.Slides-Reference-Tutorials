# إنشاء إطارات تكبير/تصغير في PowerPoint باستخدام Aspose.Slides

يُمكّن Aspose.Slides for .NET المطورين من إنشاء عروض PowerPoint تفاعلية باستخدام إطارات التكبير/التصغير، وهي ميزة فعّالة تُمكّن المستخدمين من التنقل بين الشرائح باستخدام صور مُصغّرة. يوضح هذا المثال كيفية إنشاء عرض تقديمي بشرائح متعددة، وتخصيص خلفياتها، وإضافة محتوى نصي، وتطبيق إطارات التكبير/التصغير مع خيارات تنسيق مُختلفة لتحسين التنقل والجاذبية البصرية.

## تعليمات التثبيت

1. التثبيت عبر مدير حزمة NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. التثبيت عبر .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. التثبيت اليدوي:
   - قم بتنزيل أحدث إصدار من [Aspose.Slides لـ .NET](https://docs.aspose.com/slides/net/)
   - أضف مرجعًا إلى Aspose.Slides.dll في مشروعك

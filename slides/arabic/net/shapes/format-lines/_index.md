# إنشاء شكل مستطيل باستخدام تنسيق الخط المخصص في PowerPoint

يوفر Aspose.Slides for .NET إمكانيات فعّالة لإنشاء وتعديل عروض PowerPoint التقديمية برمجيًا. يوضح هذا المثال كيفية إنشاء شكل مستطيل بتنسيق خطوط مخصص، بما في ذلك ضبط نمط الخط وعرضه ونمط الشرطة ولونه. يُنشئ هذا المثال عرضًا تقديميًا جديدًا، ويضيف مستطيلًا إلى الشريحة الأولى، ويطبق تنسيقًا محددًا على كلٍّ من التعبئة والمخطط التفصيلي للشكل، مما ينتج عنه عنصر جذاب بصريًا بتعبئة بيضاء ومخطط تفصيلي أزرق متقطع.

## تعليمات التثبيت

1. قم بتثبيت Aspose.Slides لـ .NET باستخدام إحدى الطرق التالية:

   - مدير حزمة NuGet:
     ```
     Install-Package Aspose.Slides
     ```

   - وحدة تحكم مدير الحزمة:
     ```
     PM> Install-Package Aspose.Slides
     ```

   - .NET CLI:
     ```
     dotnet add package Aspose.Slides
     ```

2. أضف مساحة الأسماء الضرورية إلى ملف C# الخاص بك:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

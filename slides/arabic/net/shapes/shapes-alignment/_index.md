# محاذاة الأشكال وتجميعها في عروض PowerPoint التقديمية

يوفر Aspose.Slides for .NET إمكانيات فعّالة لإنشاء عروض PowerPoint التقديمية وتعديلها برمجيًا. يوضح هذا المثال كيفية محاذاة الأشكال داخل الشرائح والعمل مع الأشكال المجمعة في عروض PowerPoint التقديمية. ستتعلم كيفية وضع المستطيلات عند إحداثيات محددة، ومحاذاتها بالنسبة للشريحة أو بالنسبة لبعضها البعض، ومحاذاة الأشكال بشكل انتقائي داخل المجموعات.

## تعليمات التثبيت

1. قم بتثبيت Aspose.Slides لـ .NET باستخدام إحدى الطرق التالية:
   
   - مدير حزمة NuGet:
     ```
     Install-Package Aspose.Slides.NET
     ```
   
   - وحدة تحكم مدير الحزمة:
     ```
     PM> Install-Package Aspose.Slides.NET
     ```
   
   - .NET CLI:
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. أضف المراجع إلى مشروعك:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.IO;
   ```

3. قم بتنزيل الكود المثال أدناه ودمجه في مشروعك.

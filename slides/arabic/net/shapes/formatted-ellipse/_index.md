# إنشاء أشكال بيضاوية في PowerPoint باستخدام Aspose.Slides

Aspose.Slides for .NET هي واجهة برمجة تطبيقات فعّالة تُمكّن المطورين من إنشاء عروض PowerPoint التقديمية وقراءتها وكتابتها وتعديلها وتحويلها دون الحاجة إلى Microsoft PowerPoint. يوضح هذا المثال كيفية إنشاء شكل بيضاوي في عرض تقديمي، وتطبيق التنسيقات، بما في ذلك لون التعبئة وخصائص الخطوط، وحفظ النتيجة كملف PPTX. يُظهر الكود قدرة Aspose.Slides على التعامل مع الأشكال مع تحكم دقيق في مظهرها وموقعها.

## تعليمات التثبيت

1. التثبيت عبر مدير حزمة NuGet:
   ```
   Install-Package Aspose.Slides
   ```

   أو باستخدام .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

2. إضافة المراجع:
   أضف مساحة الأسماء التالية إلى ملف C# الخاص بك:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

3. إعداد الترخيص (اختياري):
   إذا كان لديك ترخيص، قم بتقديمه باستخدام:
   ```csharp
   License license = new License();
   license.SetLicense("Aspose.Slides.lic");
   ```

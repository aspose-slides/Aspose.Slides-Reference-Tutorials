# إضافة أجزاء إلى GeometryShape في عروض PowerPoint التقديمية

يوضح هذا المثال كيفية الوصول إلى مسارات الأشكال الهندسية في عروض PowerPoint التقديمية وتعديلها باستخدام Aspose.Slides لـ .NET. بإضافة مقاطع خطية مخصصة إلى شكل مستطيل، يمكنك إنشاء أشكال هندسية أكثر تعقيدًا لا تتوفر في أنواع الأشكال القياسية. تُعد هذه التقنية مفيدة لإنشاء مخططات ومخططات انسيابية مخصصة وعناصر مرئية أخرى في عروضك التقديمية.

## تعليمات التثبيت

1. قم بتثبيت Aspose.Slides لـ .NET باستخدام إحدى الطرق التالية:

   - مدير حزمة NuGet:
     ```
     Install-Package Aspose.Slides.NET
     ```

   - وحدة تحكم مدير الحزمة:
     ```
     PM> Install-Package Aspose.Slides.NET
     ```

   - .NET CLI:
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. أضف مساحة الاسم إلى ملف C# الخاص بك:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

# إنشاء تأثيرات نصية متحركة في PowerPoint باستخدام Aspose.Slides

يوفر Aspose.Slides for .NET إمكانيات فعّالة لإنشاء تأثيرات نصية متحركة في عروض PowerPoint التقديمية برمجيًا. يوضح هذا المثال كيفية إنشاء مربع نصي بتأثيرات متحركة، بما في ذلك مسار حركة مخصص يتم تشغيله بنقرة زر. يُنشئ الكود عرضًا تقديميًا بشكل مستطيل يحتوي على نص، ويُطبّق حركة PathFootball، ويضيف مسار حركة مخصصًا يُعرّفه المستخدم ويمكن تشغيله تفاعليًا.

## تعليمات التثبيت

1. قم بتثبيت Aspose.Slides لـ .NET باستخدام إحدى الطرق التالية:

   مدير حزمة NuGet:
   ```csharp
   Install-Package Aspose.Slides
   ```

   وحدة تحكم مدير الحزمة:
   ```csharp
   PM> Install-Package Aspose.Slides
   ```

   .NET CLI:
   ```csharp
   dotnet add package Aspose.Slides
   ```

2. أضف مساحات الأسماء المطلوبة إلى كود C# الخاص بك:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Animation;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

# إنشاء أشكال المجموعات في PowerPoint باستخدام Aspose.Slides

يوفر Aspose.Slides for .NET واجهات برمجة تطبيقات شاملة لإنشاء عروض PowerPoint التقديمية وتعديلها برمجيًا. يوضح هذا المثال كيفية إنشاء شكل مجموعة، وإضافة عدة أشكال مستطيلة داخلها، وتحديد موضع الأشكال بشكل صحيح، وإضافة إطارات، وحفظ العرض التقديمي. تُعد أشكال المجموعة أساسية لتنظيم الأشكال المترابطة التي تحتاج إلى تعديل كوحدة واحدة، مما يوفر هيكلًا وتحكمًا أكبر في عناصر العرض التقديمي.

## تعليمات التثبيت

1. قم بتثبيت Aspose.Slides لـ .NET باستخدام إحدى الطرق التالية:

   - مدير حزمة NuGet:
     ```
     Install-Package Aspose.Slides.NET
     ```

   - وحدة تحكم مدير الحزمة:
     ```
     PM> Install-Package Aspose.Slides.NET
     ```

   - .NET CLI:
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. أضف المراجع إلى مشروعك:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

3. إعداد الترخيص (اختياري للاستخدام الإنتاجي):
   ```csharp
   // تعيين الترخيص لإزالة قيود التقييم
   License license = new License();
   license.SetLicense("Aspose.Slides.lic");
   ```

# إضافة الصور إلى شرائح PowerPoint باستخدام Aspose.Slides

يوفر Aspose.Slides لـ .NET إمكانيات فعّالة لإنشاء عروض PowerPoint التقديمية وتعديلها برمجيًا. يوضح هذا المثال كيفية تحميل صورة خارجية وإضافتها إلى شريحة عرض تقديمي كإطار صورة - وهو مطلب شائع عند إنشاء التقارير، أو إنشاء القوالب، أو إنشاء عروض تقديمية ديناميكيًا من مصادر البيانات.

يتعامل الكود أدناه مع كل شيء بدءًا من إنشاء عرض تقديمي جديد إلى إضافة الصورة وحفظ ملف PPTX النهائي، كل ذلك باستخدام بضعة أسطر فقط من كود C#.

## تعليمات التثبيت

1. قم بتثبيت Aspose.Slides لـ .NET باستخدام إحدى الطرق التالية:
   - مدير حزمة NuGet: `Install-Package Aspose.Slides.NET`
   - .NET CLI: `dotnet add package Aspose.Slides.NET`
   - أو قم بالتحميل مباشرة من [موقع Aspose](https://releases.aspose.com/slides/net/)

2. أضف مساحة الاسم إلى ملف C# الخاص بك:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

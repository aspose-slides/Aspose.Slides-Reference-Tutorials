# إنشاء أشكال بيضاوية مملوءة بتدرج لوني في PowerPoint باستخدام Aspose.Slides

يوضح هذا المثال كيفية إنشاء عرض تقديمي في PowerPoint برمجيًا يحتوي على شكل بيضاوي مع تأثير تعبئة متدرجة مخصص. يُنشئ الكود عرضًا تقديميًا جديدًا، ويضيف شكلًا بيضاويًا إلى الشريحة الأولى، ويطبق تعبئة متدرجة خطية مع توقفات باللونين الأحمر والأرجواني. يوفر Aspose.Slides for .NET واجهة برمجة تطبيقات شاملة لإنشاء عروض PowerPoint التقديمية والتحكم فيها، مع تحكم دقيق في الأشكال والتنسيق والتأثيرات.

## تعليمات التثبيت

1. قم بتثبيت Aspose.Slides لـ .NET باستخدام إحدى الطرق التالية:
   - مدير حزمة NuGet: `Install-Package Aspose.Slides.NET`
   - .NET CLI: `dotnet add package Aspose.Slides.NET`
   - وحدة تحكم مدير الحزمة: `Install-Package Aspose.Slides.NET`

2. أضف العبارة التالية باستخدام إلى ملف C# الخاص بك:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

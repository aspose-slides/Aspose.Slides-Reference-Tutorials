# إنشاء أشكال المستطيل وتدويرها في PowerPoint

Aspose.Slides for .NET هي واجهة برمجة تطبيقات فعّالة تُمكّن المطورين من إنشاء عروض PowerPoint التقديمية وتعديلها وتحويلها دون الحاجة إلى تثبيت Microsoft PowerPoint. يوضح هذا المثال كيفية إنشاء عرض تقديمي بشكل مستطيل مُستدير، مُستعرضًا إمكانيات المكتبة في التعامل مع الأشكال.

مع Aspose.Slides، يمكنك برمجيًا إضافة أشكال متنوعة إلى شرائحك وتطبيق تحويلات مثل التدوير. تُعد هذه الميزة مفيدة بشكل خاص لإنشاء عروض تقديمية ديناميكية مع تحكم دقيق في العناصر المرئية.

## تعليمات التثبيت

1. التثبيت عبر مدير حزمة NuGet:
   ```
   Install-Package Aspose.Slides
   ```

2. التثبيت عبر .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. أضف التوجيه باستخدام في الكود الخاص بك:
   ```csharp
   using Aspose.Slides;
   ```

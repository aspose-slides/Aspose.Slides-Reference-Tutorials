## دليل التنفيذ: إزالة الملاحظات من جميع الشرائح

سوف يرشدك هذا القسم خلال عملية إزالة الملاحظات من كافة الشرائح في العرض التقديمي.

### ملخص

تتضمن العملية تكرار كل شريحة واستخدام `NotesSlideManager` لإزالة أي ملاحظات موجودة، مما يضمن إخراج عرض تقديمي نظيف.

### خطوات التنفيذ
#### الخطوة 1: تحديد مسارات الدليل
قم بإعداد المسارات لإدخال المستند والمكان الذي تريد حفظ الملف المعالج فيه.

```csharp
string documentDirectory = @"YOUR_DOCUMENT_DIRECTORY";
string outputDirectory = @"YOUR_OUTPUT_DIRECTORY";
```

#### الخطوة 2: تحميل العرض التقديمي
إنشاء `Presentation` كائن بمسار ملف العرض التقديمي. تأكد من وجود ملفك، مثلاً "AccessSlides.pptx"، في المجلد المحدد.

```csharp
Presentation presentation = new Presentation(documentDirectory + "AccessSlides.pptx");
```

#### الخطوة 3: التكرار على الشرائح
قم بالتنقل عبر كل شريحة والوصول إليها `NotesSlideManager`.

```csharp
INotesSlideManager mgr = null;
for (int i = 0; i < presentation.Slides.Count; i++)
{
    mgr = presentation.Slides[i].NotesSlideManager;

    // المتابعة إذا كانت الملاحظات موجودة
    if (mgr.NotesSlide != null)
    {
        mgr.RemoveNotesSlide();
    }
}
```

**توضيح:**
- **`INotesSlideManager`**:إدارة الملاحظات الخاصة بشريحة معينة.
- **`RemoveNotesSlide()`**:يزيل أي ملاحظات موجودة من الشريحة الحالية.

#### الخطوة 4: حفظ العرض التقديمي
بعد حذف الملاحظات، احفظ عرضك التقديمي على القرص. حدد اسم ملف الإخراج وصيغته.

```csharp
presentation.Save(outputDirectory + "RemoveNotesFromAllSlides_out.pptx", SaveFormat.Pptx);
```

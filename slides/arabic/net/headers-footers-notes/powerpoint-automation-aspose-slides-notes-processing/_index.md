## دليل التنفيذ
### الميزة 1: تحميل العرض التقديمي
#### ملخص
يُعد تحميل عرض تقديمي موجود على PowerPoint أمرًا أساسيًا قبل أي معالجة. هذه الخطوة تُهيئ ملفك لإجراء عمليات أخرى.

#### التنفيذ خطوة بخطوة
##### تحديد مسار الملف
أولاً، حدد مكانك `.pptx` الملف موجود في:

```csharp
string pptxFileName = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "ForEachPortion.pptx");
```

##### تهيئة فئة العرض التقديمي
إنشاء مثيل لـ `Presentation` فصل:

```csharp
using (Presentation pres = new Presentation(pptxFileName))
{
    // تم تحميل العرض التقديمي الآن وهو جاهز للعمليات الإضافية
}
```
**لماذا يعمل هذا؟**: ال `Presentation` تتضمن الفئة جميع الوظائف اللازمة لقراءة ملفات PowerPoint وتحريرها وحفظها. باستخدام `using` يضمن البيان التخلص السليم من الموارد بعد الاستخدام.

### الميزة 2: التكرار عبر الأجزاء في شرائح الملاحظات
#### ملخص
يُعدّ استخراج النص من شرائح الملاحظات أمرًا بالغ الأهمية للتوثيق أو إنشاء المحتوى تلقائيًا. سنستعرض كل جزء من النص داخل هذه الشرائح.

#### التنفيذ خطوة بخطوة
##### تحميل العرض التقديمي
تأكد من أنك قمت بتحميل العرض التقديمي الخاص بك كما هو موضح مسبقًا.

##### تكرار جزء من النص

```csharp
using (Presentation pres = new Presentation(pptxFileName))
{
    ForEach.Portion(pres, true, (portion, para, slide, index) =>
    {
        if (slide is NotesSlide && !string.IsNullOrEmpty(portion.Text))
        {
            // معالجة أو إخراج نص الجزء حسب الحاجة.
            Console.WriteLine($"Portion Text: {portion.Text}");
        }
    });
}
```
**النقاط الرئيسية**: 
- `ForEach.Portion` تكرر الطريقة جميع الأجزاء، مما يسمح بالمعالجة الشرطية استنادًا إلى نوع الشريحة ووجود المحتوى.
- تتحقق دالة lambda مما إذا كانت الشريحة من النوع `NotesSlide` وما إذا كان الجزء يحتوي على نص.

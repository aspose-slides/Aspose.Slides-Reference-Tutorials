## دليل التنفيذ
### إعداد الدليل
تضمن هذه الميزة وجود الدليل المحدد قبل حفظ أي ملفات عرض تقديمي.

#### ملخص
ستتعلم كيفية التحقق من وجود دليل وإنشائه عند الحاجة. هذا ضروري لتجنب الأخطاء عند محاولة حفظ الملفات في مسارات غير موجودة.

#### تنفيذ الكود
```csharp
using System.IO;

string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // قم بتعيين مسار دليل المستند الخاص بك هنا
bool isExists = Directory.Exists(dataDir);
if (!isExists)
{
    Directory.CreateDirectory(dataDir); // إنشاء الدليل إذا لم يكن موجودًا
}
```

**توضيح**: ال `Directory.Exists` تتحقق الطريقة من وجود دليل. إذا أرجعت القيمة "خطأ"، `Directory.CreateDirectory` يتم استدعاؤه لإنشاء المسار المحدد.

### تهيئة العرض التقديمي
يتناول هذا القسم كيفية البدء في العمل على عرض تقديمي جديد في PowerPoint والوصول إلى شرائحه.

#### ملخص
ستقوم بتهيئة كائن العرض التقديمي والحصول على مراجع لشرائحه لمزيد من التلاعب.

#### تنفيذ الكود
```csharp
using Aspose.Slides;

Presentation pptxPresentation = new Presentation(); // إنشاء مثيل عرض تقديمي جديد
ISlide slide = pptxPresentation.Slides[0]; // الوصول إلى الشريحة الأولى
```

**توضيح**: ال `Presentation` يتم إنشاء فئة من Aspose.Slides لإنشاء ملف PowerPoint جديد. يمكنك الوصول إلى شرائحه باستخدام `Slides` ملكية.

### إضافة شكل تلقائي مع النص
توضح هذه الميزة كيفية إضافة الأشكال وإدراج النص فيها، مما يعزز المظهر المرئي للعرض التقديمي الخاص بك.

#### ملخص
ستتعلم كيفية إضافة شكل تلقائي (مستطيل) وإدخال نص داخله على الشريحة.

#### تنفيذ الكود
```csharp
IAutoShape pptxAutoShape = (IAutoShape)slide.Shapes.AddAutoShape(ShapeType.Rectangle, 150, 150, 150, 50); // أضف شكل مستطيل
ITextFrame txtFrame = pptxAutoShape.TextFrame; // احصل على إطار النص المرتبط

// إدراج النص في الفقرة الأولى والجزء الأول من إطار النص
txtFrame.Paragraphs[0].Portions[0].Text = "Aspose.Slides";
```

**توضيح**: ال `AddAutoShape` تُستخدم هذه الطريقة لإضافة مستطيل. يُحدَّد موضعه وعرضه وارتفاعه كمعلمات. تتم معالجة إدراج النص في الشكل من خلال الوصول إلى إطار النص.

### إعداد الارتباط التشعبي
تتيح لك هذه الميزة إعداد ارتباطات تشعبية داخل عناصر النص في العرض التقديمي الخاص بك.

#### ملخص
ستقوم بتعيين إجراء النقر على ارتباط تشعبي خارجي للنص المدرج في الشكل التلقائي.

#### تنفيذ الكود
```csharp
IHyperlinkManager hyperlinkManager = txtFrame.Paragraphs[0].Portions[0].PortionFormat.HyperlinkManager; // مدير الارتباط التشعبي للوصول
hyperlinkManager.SetExternalHyperlinkClick("http://www.aspose.com"); // تعيين إجراء النقر على الرابط التشعبي الخارجي
```

**توضيح**:باستخدام `HyperlinkManager`يمكنك إدارة الروابط التشعبية داخل إطارات النص. هنا، نحدد عنوان URL الذي سيُفتح عند نقر المستخدم على النص المحدد.

### حفظ العرض التقديمي
وأخيرًا، تأكد من حفظ كافة التغييرات لإنشاء ملف العرض التقديمي النهائي.

#### ملخص
تعرف على كيفية حفظ العرض التقديمي الخاص بك في الدليل المخصص بتنسيق PPTX.

#### تنفيذ الكود
```csharp
cpptxPresentation.Save("YOUR_DOCUMENT_DIRECTORY/hLinkPPTX_out.pptx", Aspose.Slides.Export.SaveFormat.Pptx); // حفظ العرض التقديمي
```

**توضيح**: ال `Save` تكتب الطريقة الحالة الحالية لجهازك `Presentation` إضافة كائن إلى ملف. تأكد من تحديد مسار الدليل بشكل صحيح.

## دليل التنفيذ

دعونا نستعرض كيفية إعداد ارتباط تشعبي ماكرو على شكل ما.

### نظرة عامة على الميزة: إعداد ارتباط تشعبي للماكرو

تتيح لك هذه الميزة إرفاق وظيفة ماكرو بالأشكال في PowerPoint باستخدام Aspose.Slides لـ .NET، وهي مثالية لإنشاء عروض تقديمية تفاعلية تستجيب لمدخلات المستخدم.

#### الخطوة 1: إنشاء الشكل

أضف شكلًا تلقائيًا إلى الشريحة الخاصة بك:

```csharp
using Aspose.Slides;

string macroName = "TestMacro";
using (Presentation presentation = new Presentation())
{
    // أضف شكل زر فارغ في الموضع (20، 20) بأبعاد (80 × 30)
    IAutoShape shape = presentation.Slides[0].Shapes.AddAutoShape(ShapeType.BlankButton, 20, 20, 80, 30);
```

#### الخطوة 2: تعيين ارتباط الماكرو التشعبي

قم بإرفاق ماكرو بهذا الشكل:

```csharp
    // ربط الشكل بحدث النقر على رابط تشعبي للماكرو
    shape.HyperlinkManager.SetMacroHyperlinkClick(macroName);

    // حفظ العرض التقديمي
    presentation.Save("output.pptx", Aspose.Slides.Export.SaveFormat.Pptx);
}
```
**توضيح:**
- `AddAutoShape(ShapeType.BlankButton, 20, 20, 80, 30)`:يضيف شكل زر فارغ عند الإحداثيات والحجم المحددين.
- `SetMacroHyperlinkClick(macroName)`:ربط الماكرو بحدث النقر الخاص بالشكل.

#### نصائح استكشاف الأخطاء وإصلاحها

- **الماكرو لا يعمل:** تأكد من وجود الماكرو في قالب PowerPoint الخاص بك.
- **مشاكل تحديد موضع الشكل:** تأكد من صحة قيم الإحداثيات لتحديد موضعها الدقيق على الشريحة.

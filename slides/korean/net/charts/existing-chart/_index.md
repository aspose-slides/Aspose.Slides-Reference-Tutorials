# Aspose.Slides를 사용하여 PowerPoint에서 차트 수정하기

Aspose.Slides for .NET은 PowerPoint 프레젠테이션을 프로그래밍 방식으로 만들고 수정할 수 있는 강력한 API를 제공합니다. 이 예제는 Microsoft PowerPoint를 설치하지 않고도 차트 데이터를 업데이트하고, 새 시리즈를 추가하고, 차트 유형을 변경하여 기존 프레젠테이션의 차트를 사용하는 방법을 보여줍니다.

## 설치 지침

1. NuGet 패키지 관리자를 통해 설치:
   ```
   Install-Package Aspose.Slides
   ```

2. 또는 .NET CLI를 사용합니다.
   ```
   dotnet add package Aspose.Slides
   ```

3. 다음 지시문을 사용하여 추가합니다.
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Charts;
   ```

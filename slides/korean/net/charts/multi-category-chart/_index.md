# Aspose.Slides for .NET을 사용하여 클러스터형 막대형 차트 만들기

Aspose.Slides for .NET은 PowerPoint 프레젠테이션을 프로그래밍 방식으로 만들고 조작할 수 있는 강력한 기능을 제공합니다. 이 예제에서는 사용자 지정 범주와 데이터 계열을 사용하여 클러스터형 세로 막대형 차트를 만드는 방법을 보여줍니다.

## 설치

```csharp
// NuGet 패키지 관리자를 통해 설치
Install-Package Aspose.Slides
```

또는 프로젝트 파일에 다음 참조를 추가하세요.

```xml
<PackageReference Include="Aspose.Slides" Version="23.2.0" />
```

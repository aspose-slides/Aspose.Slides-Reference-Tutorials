# Aspose.Slides를 사용하여 PowerPoint에서 그라데이션으로 채워진 타원 모양 만들기

이 예제는 사용자 지정 그라데이션 채우기 효과가 적용된 타원 모양이 포함된 PowerPoint 프레젠테이션을 프로그래밍 방식으로 만드는 방법을 보여줍니다. 이 코드는 새 프레젠테이션을 만들고, 첫 번째 슬라이드에 타원을 추가하고, 빨간색과 보라색 색상 정지점을 사용하여 선형 그라데이션 채우기를 적용합니다. Aspose.Slides for .NET은 모양, 서식 및 효과를 정밀하게 제어하여 PowerPoint 프레젠테이션을 만들고 조작할 수 있는 포괄적인 API를 제공합니다.

## 설치 지침

1. 다음 방법 중 하나를 사용하여 .NET용 Aspose.Slides를 설치하세요.
   - NuGet 패키지 관리자: `Install-Package Aspose.Slides.NET`
   - .NET CLI: `dotnet add package Aspose.Slides.NET`
   - 패키지 관리자 콘솔: `Install-Package Aspose.Slides.NET`

2. C# 파일에 다음 using 문을 추가합니다.
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

# PowerPoint에서 사각형 모양 만들기 및 회전

Aspose.Slides for .NET은 개발자가 Microsoft PowerPoint를 설치하지 않고도 PowerPoint 프레젠테이션을 만들고, 수정하고, 변환할 수 있는 강력한 API입니다. 이 예제에서는 회전된 사각형 도형이 있는 프레젠테이션을 만드는 방법을 보여주며 라이브러리의 도형 조작 기능을 보여줍니다.

Aspose.Slides를 사용하면 슬라이드에 다양한 모양을 프로그래밍 방식으로 추가하고 회전과 같은 변형을 적용할 수 있습니다. 이 기능은 시각적 요소를 정밀하게 제어하여 역동적인 프레젠테이션을 만드는 데 특히 유용합니다.

## 설치 지침

1. NuGet 패키지 관리자를 통해 설치:
   ```
   Install-Package Aspose.Slides
   ```

2. .NET CLI를 통해 설치:
   ```
   dotnet add package Aspose.Slides
   ```

3. 코드에 using 지시문을 추가하세요.
   ```csharp
   using Aspose.Slides;
   ```

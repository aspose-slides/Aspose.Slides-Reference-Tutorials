# Aspose.Slides를 사용하여 PowerPoint 도형의 기하 경로 수정

Aspose.Slides for .NET은 고급 도형 조작을 포함하여 PowerPoint 프레젠테이션을 프로그래밍 방식으로 수정할 수 있는 강력한 기능을 제공합니다. 이 예제는 기존 도형의 기하 경로를 수정하여 사용자 지정 도형을 만드는 방법을 보여줍니다. 특히 하트 도형에서 세그먼트를 제거하여 고유한 변형을 만드는 방법을 보여줍니다.

기하 경로는 일련의 점과 선분을 통해 도형의 윤곽을 정의합니다. 개발자는 이러한 경로에 접근하고 수정함으로써 표준 PowerPoint 도형 라이브러리에서는 제공되지 않는 고도로 맞춤화된 도형을 만들 수 있습니다.

## 설치 지침

1. NuGet 패키지 관리자를 통해 설치:
   ```
   Install-Package Aspose.Slides
   ```

2. .NET CLI를 통해 설치:
   ```
   dotnet add package Aspose.Slides
   ```

3. 코드에 using 지시문을 추가하세요.
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

4. 애플리케이션 시작 시 라이선스 초기화를 추가합니다.
   ```csharp
   // 면허가 있으면
   License license = new License();
   license.SetLicense("path/to/Aspose.Slides.lic");
   ```

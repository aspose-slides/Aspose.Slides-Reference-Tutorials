# Aspose.Slides를 사용하여 PowerPoint에서 그룹 모양 만들기

Aspose.Slides for .NET은 PowerPoint 프레젠테이션을 프로그래밍 방식으로 만들고 조작할 수 있는 포괄적인 API를 제공합니다. 이 예제는 그룹 도형을 만들고, 그 안에 여러 개의 사각형 도형을 추가하고, 도형을 적절하게 배치하고, 프레임을 추가하고, 프레젠테이션을 저장하는 방법을 보여줍니다. 그룹 도형은 하나의 단위로 조작해야 하는 관련 도형을 구성하는 데 필수적이며, 프레젠테이션 요소에 대한 구조와 제어력을 향상시킵니다.

## 설치 지침

1. 다음 방법 중 하나를 사용하여 .NET용 Aspose.Slides를 설치하세요.

   - NuGet 패키지 관리자:
     ```
     Install-Package Aspose.Slides.NET
     ```

   - 패키지 관리자 콘솔:
     ```
     PM> Install-Package Aspose.Slides.NET
     ```

   - .NET CLI:
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. 프로젝트에 참조를 추가하세요:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

3. 라이선스 설정(생산용 선택 사항):
   ```csharp
   // 평가 제한을 제거하기 위한 라이선스 설정
   License license = new License();
   license.SetLicense("Aspose.Slides.lic");
   ```

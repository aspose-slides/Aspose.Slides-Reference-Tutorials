# PowerPoint 프레젠테이션에서 GeometryShape에 세그먼트 추가

이 예제는 Aspose.Slides for .NET을 사용하여 PowerPoint 프레젠테이션에서 도형의 기하 경로에 액세스하고 수정하는 방법을 보여줍니다. 직사각형 도형에 사용자 지정 선분을 추가하면 표준 도형 유형에서는 사용할 수 없는 더욱 복잡한 기하 도형을 만들 수 있습니다. 이 기술은 프레젠테이션에서 사용자 지정 다이어그램, 순서도 및 기타 시각적 요소를 만드는 데 유용합니다.

## 설치 지침

1. 다음 방법 중 하나를 사용하여 Aspose.Slides for .NET을 설치하세요.

   - NuGet 패키지 관리자:
     ```
     Install-Package Aspose.Slides.NET
     ```

   - 패키지 관리자 콘솔:
     ```
     PM> Install-Package Aspose.Slides.NET
     ```

   - .NET CLI:
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. C# 파일에 네임스페이스를 추가합니다.
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

# PowerPoint 프레젠테이션에서 도형 정렬 및 그룹화

Aspose.Slides for .NET은 PowerPoint 프레젠테이션을 프로그래밍 방식으로 만들고 조작할 수 있는 강력한 기능을 제공합니다. 이 예제는 슬라이드 내에서 도형을 정렬하고 PowerPoint 프레젠테이션에서 그룹화된 도형을 사용하는 방법을 보여줍니다. 사각형을 특정 좌표에 배치하고, 슬라이드 또는 다른 도형을 기준으로 정렬하고, 그룹 내에서 도형을 선택적으로 정렬하는 방법을 배웁니다.

## 설치 지침

1. 다음 방법 중 하나를 사용하여 .NET용 Aspose.Slides를 설치하세요.
   
   - NuGet 패키지 관리자:
     ```
     Install-Package Aspose.Slides.NET
     ```
   
   - 패키지 관리자 콘솔:
     ```
     PM> Install-Package Aspose.Slides.NET
     ```
   
   - .NET CLI:
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. 프로젝트에 참조를 추가하세요:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.IO;
   ```

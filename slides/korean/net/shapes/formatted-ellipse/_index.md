# Aspose.Slides를 사용하여 PowerPoint에서 타원형 모양 만들기

Aspose.Slides for .NET은 개발자가 Microsoft PowerPoint 없이도 PowerPoint 프레젠테이션을 만들고, 읽고, 쓰고, 수정하고, 변환할 수 있도록 지원하는 강력한 API입니다. 이 예제는 프레젠테이션에 타원형 도형을 만들고, 채우기 색 및 선 속성을 포함한 서식을 적용하고, 결과를 PPTX 파일로 저장하는 방법을 보여줍니다. 이 코드는 Aspose.Slides가 도형의 모양과 위치를 정밀하게 제어하여 조작할 수 있는 기능을 보여줍니다.

## 설치 지침

1. NuGet 패키지 관리자를 통해 설치:
   ```
   Install-Package Aspose.Slides
   ```

   또는 .NET CLI를 사용합니다.
   ```
   dotnet add package Aspose.Slides
   ```

2. 참조 추가:
   C# 파일에 다음 네임스페이스를 추가합니다.
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

3. 라이센스 설정(선택 사항):
   라이센스가 있는 경우 다음을 사용하여 적용하세요.
   ```csharp
   License license = new License();
   license.SetLicense("Aspose.Slides.lic");
   ```

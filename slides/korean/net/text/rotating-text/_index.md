# Aspose.Slides를 사용하여 PowerPoint에서 세로 텍스트 만들기

Aspose.Slides for .NET은 PowerPoint 프레젠테이션을 프로그래밍 방식으로 만들고 조작할 수 있는 강력한 기능을 제공합니다. 이 예제는 세로로 회전된 텍스트가 있는 사각형 모양의 프레젠테이션을 만드는 방법을 보여줍니다. 텍스트는 270도 각도로 배치되어 아래에서 위로 읽힙니다. 이 기법은 프레젠테이션에 측면 레이블, 고유한 텍스트 레이아웃 또는 특수 디자인 요소를 추가하는 데 유용합니다.

## 설치 지침

1. .NET용 Aspose.Slides 설치
   
   NuGet 패키지 관리자를 사용하여 Aspose.Slides 패키지를 설치하세요.
   
   ```csharp
   Install-Package Aspose.Slides
   ```
   
   또는 .NET CLI를 사용합니다.
   
   ```csharp
   dotnet add package Aspose.Slides
   ```

2. 참조 추가
   
   C# 파일에 다음 using 문을 추가합니다.
   
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

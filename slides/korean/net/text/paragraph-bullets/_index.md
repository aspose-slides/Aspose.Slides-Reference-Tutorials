# PowerPoint 프레젠테이션에서 사용자 지정 글머리 기호 만들기

Aspose.Slides for .NET은 PowerPoint 프레젠테이션을 프로그래밍 방식으로 만들고 조작할 수 있는 강력한 기능을 제공합니다. 이 예제는 특히 텍스트 프레임에 사용자 지정 글머리 기호를 구현하는 데 중점을 두고, 글머리 기호 유형, 색상, 들여쓰기 및 크기를 제어하는 방법을 보여줍니다. 이 코드는 두 개의 단락으로 구성된 직사각형 모양의 프레젠테이션을 생성합니다. 하나는 기호 글머리 기호를 사용하고 다른 하나는 번호가 매겨진 글머리 기호 스타일을 사용합니다.

## 설치 지침

1. NuGet 패키지 관리자를 사용하여 .NET용 Aspose.Slides를 설치하세요.
   ```
   Install-Package Aspose.Slides
   ```

2. 또는 패키지를 직접 다운로드하세요.
   - 방문하세요 [Aspose.Slides 다운로드 페이지](https://downloads.aspose.com/slides/net)
   - 최신 버전을 다운로드하세요
   - 프로젝트에서 Aspose.Slides.dll을 추출하여 참조하세요.

3. 필요한 using 문을 추가합니다.
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

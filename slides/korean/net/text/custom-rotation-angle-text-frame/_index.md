# Aspose.Slides를 사용하여 PowerPoint에서 차트 제목 및 레이블 회전

Aspose.Slides for .NET은 PowerPoint 프레젠테이션을 프로그래밍 방식으로 만들고 조작할 수 있는 강력한 기능을 제공합니다. 이 예제는 회전된 텍스트 요소를 포함하는 클러스터형 세로 막대형 차트를 사용하여 프레젠테이션을 만드는 방법을 보여줍니다. 특히, 차트 제목과 데이터 레이블을 모두 회전하여 가독성과 시각적 효과를 향상시킵니다.

## 설치 지침

1. NuGet 패키지 관리자를 통해 설치:
   ```
   Install-Package Aspose.Slides
   ```

2. 또는 .NET CLI를 통해 설치하세요.
   ```
   dotnet add package Aspose.Slides
   ```

3. C# 파일에 네임스페이스를 추가합니다.
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Charts;
   ```

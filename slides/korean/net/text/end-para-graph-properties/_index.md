# 서식 있는 텍스트로 PowerPoint 프레젠테이션 만들기

Aspose.Slides for .NET은 개발자가 Microsoft PowerPoint 없이도 PowerPoint 프레젠테이션을 만들고, 조작하고, 변환할 수 있도록 해주는 강력한 API입니다. 이 예제에서는 서식이 지정된 텍스트 단락이 있는 사각형 도형이 포함된 간단한 프레젠테이션을 만드는 방법을 보여줍니다.

이 코드는 새 프레젠테이션을 만들고, 첫 번째 슬라이드에 사각형 도형을 추가한 다음, 다양한 텍스트 서식 옵션이 적용된 여러 단락을 추가합니다. 텍스트의 특정 부분에 크기 및 글꼴 종류와 같은 글꼴 속성을 설정하는 방법을 보여줍니다.

## 설치 지침

1. NuGet 패키지 관리자를 통해 설치:
   ```
   Install-Package Aspose.Slides
   ```

2. .NET CLI를 통해 설치:
   ```
   dotnet add package Aspose.Slides
   ```

3. 패키지 관리자 콘솔을 통해 설치:
   ```
   PM> Install-Package Aspose.Slides
   ```

4. 프로젝트에 참조 추가:
   설치 후 프로젝트에 참조를 추가하세요.
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

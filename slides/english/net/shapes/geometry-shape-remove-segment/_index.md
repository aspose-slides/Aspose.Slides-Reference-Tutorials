# Modify Geometry Paths in PowerPoint Shapes with Aspose.Slides

Aspose.Slides for .NET provides powerful capabilities for programmatically modifying PowerPoint presentations, including advanced shape manipulation. This example demonstrates how to create custom shapes by modifying the geometry path of an existing shape - specifically by removing a segment from a heart shape to create a unique variant.

The geometry path defines the outline of a shape through a series of points and segments. By accessing and modifying these paths, developers can create highly customized shapes that aren't available in the standard PowerPoint shape library.

## Installation Instructions

1. Install via NuGet Package Manager:
   ```
   Install-Package Aspose.Slides
   ```

2. Install via .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Add using directive in your code:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

4. Add license initialization in your application startup:
   ```csharp
   // If you have a license
   License license = new License();
   license.SetLicense("path/to/Aspose.Slides.lic");
   ```

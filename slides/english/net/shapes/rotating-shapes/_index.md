# Creating and Rotating Rectangle Shapes in PowerPoint

Aspose.Slides for .NET is a powerful API that allows developers to create, modify, and convert PowerPoint presentations without requiring Microsoft PowerPoint installation. This example demonstrates how to create a presentation with a rotated rectangle shape, showcasing the shape manipulation capabilities of the library.

With Aspose.Slides, you can programmatically add various shapes to your slides and apply transformations like rotation. This functionality is particularly useful for creating dynamic presentations with precise control over visual elements.

## Installation Instructions

1. Install via NuGet Package Manager:
   ```
   Install-Package Aspose.Slides
   ```

2. Install via .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Add using directive in your code:
   ```csharp
   using Aspose.Slides;
   ```

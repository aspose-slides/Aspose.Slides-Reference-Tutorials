# Modifying Charts in PowerPoint with Aspose.Slides

Aspose.Slides for .NET provides powerful APIs to create and modify PowerPoint presentations programmatically. This example demonstrates how to work with charts in existing presentations by updating chart data, adding new series, and changing chart types - all without requiring Microsoft PowerPoint to be installed.

## Installation Instructions

1. Install via NuGet Package Manager:
   ```
   Install-Package Aspose.Slides
   ```

2. Or using .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Add the following using directives:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Charts;
   ```

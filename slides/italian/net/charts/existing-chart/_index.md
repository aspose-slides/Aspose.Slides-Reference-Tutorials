# Modifica dei grafici in PowerPoint con Aspose.Slides

Aspose.Slides per .NET fornisce potenti API per creare e modificare le presentazioni di PowerPoint a livello di codice. Questo esempio illustra come utilizzare i grafici nelle presentazioni esistenti aggiornando i dati dei grafici, aggiungendo nuove serie e modificando i tipi di grafico, il tutto senza richiedere l'installazione di Microsoft PowerPoint.

## Istruzioni per l'installazione

1. Installa tramite NuGet Package Manager:
   ```
   Install-Package Aspose.Slides
   ```

2. Oppure utilizzando .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Aggiungere le seguenti direttive using:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Charts;
   ```

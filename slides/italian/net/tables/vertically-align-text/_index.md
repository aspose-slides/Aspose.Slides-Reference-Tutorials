## Istruzioni per l'installazione

1. Installa Aspose.Slides per .NET tramite NuGet Package Manager:
   ```
   Install-Package Aspose.Slides
   ```
   Oppure tramite .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

2. Aggiungi riferimenti al tuo progetto:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

3. Scarica le risorse di documentazione:
   - [Riferimento API](https://reference.aspose.com/slides/net/)
   - [Guida per gli sviluppatori](https://docs.aspose.com/slides/net/)

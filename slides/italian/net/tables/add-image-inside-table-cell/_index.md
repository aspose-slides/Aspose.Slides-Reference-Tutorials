## Istruzioni per l'installazione

1. Installa Aspose.Slides per .NET utilizzando uno di questi metodi:
   - Gestore pacchetti NuGet: `Install-Package Aspose.Slides.NET`
   - Interfaccia della riga di comando .NET: `dotnet add package Aspose.Slides.NET`
   - Aggiungi un riferimento a Aspose.Slides.dll direttamente nel tuo progetto

2. Aggiungi il riferimento allo spazio dei nomi nel tuo file C#:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

3. Scarica la documentazione e il riferimento API:
   - [Documentazione di Aspose.Slides per .NET](https://docs.aspose.com/slides/net/)
   - [Riferimento API Aspose.Slides per .NET](https://reference.aspose.com/slides/net/)

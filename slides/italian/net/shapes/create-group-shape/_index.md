# Crea forme di gruppo in PowerPoint con Aspose.Slides

Aspose.Slides per .NET fornisce API complete per creare e manipolare presentazioni PowerPoint a livello di codice. Questo esempio illustra come creare una forma di gruppo, aggiungere più forme rettangolari al suo interno, posizionarle correttamente, aggiungere cornici e salvare la presentazione. Le forme di gruppo sono essenziali per organizzare forme correlate che devono essere manipolate come un'unica unità, offrendo maggiore struttura e controllo sugli elementi della presentazione.

## Istruzioni per l'installazione

1. Installa Aspose.Slides per .NET utilizzando uno di questi metodi:

   - Gestore pacchetti NuGet:
     ```
     Install-Package Aspose.Slides.NET
     ```

   - Console del gestore pacchetti:
     ```
     PM> Install-Package Aspose.Slides.NET
     ```

   - Interfaccia della riga di comando .NET:
     ```
     dotnet add package Aspose.Slides.NET
     ```

2. Aggiungi riferimenti al tuo progetto:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

3. Imposta una licenza (facoltativa per l'uso in produzione):
   ```csharp
   // Imposta la licenza per rimuovere le restrizioni di valutazione
   License license = new License();
   license.SetLicense("Aspose.Slides.lic");
   ```

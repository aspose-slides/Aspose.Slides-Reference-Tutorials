# Creazione e rotazione di forme rettangolari in PowerPoint

Aspose.Slides per .NET è una potente API che consente agli sviluppatori di creare, modificare e convertire presentazioni PowerPoint senza richiedere l'installazione di Microsoft PowerPoint. Questo esempio illustra come creare una presentazione con una forma rettangolare ruotata, evidenziando le funzionalità di manipolazione delle forme della libreria.

Con Aspose.Slides, puoi aggiungere programmaticamente diverse forme alle tue diapositive e applicare trasformazioni come la rotazione. Questa funzionalità è particolarmente utile per creare presentazioni dinamiche con un controllo preciso sugli elementi visivi.

## Istruzioni per l'installazione

1. Installa tramite NuGet Package Manager:
   ```
   Install-Package Aspose.Slides
   ```

2. Installa tramite .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Aggiungi la direttiva using nel tuo codice:
   ```csharp
   using Aspose.Slides;
   ```

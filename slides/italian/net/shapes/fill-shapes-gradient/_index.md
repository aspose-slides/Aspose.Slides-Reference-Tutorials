# Creazione di forme ellittiche con riempimento sfumato in PowerPoint con Aspose.Slides

Questo esempio illustra come creare a livello di codice una presentazione PowerPoint contenente una forma ellittica con un effetto di riempimento sfumato personalizzato. Il codice crea una nuova presentazione, aggiunge un'ellisse alla prima diapositiva e applica un riempimento sfumato lineare con interruzioni di colore rosso e viola. Aspose.Slides per .NET fornisce un'API completa per la creazione e la manipolazione di presentazioni PowerPoint con un controllo preciso su forme, formattazione ed effetti.

## Istruzioni per l'installazione

1. Installa Aspose.Slides per .NET utilizzando uno di questi metodi:
   - Gestore pacchetti NuGet: `Install-Package Aspose.Slides.NET`
   - Interfaccia della riga di comando .NET: `dotnet add package Aspose.Slides.NET`
   - Console del gestore pacchetti: `Install-Package Aspose.Slides.NET`

2. Aggiungere la seguente istruzione using al file C#:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

# Creazione di forme ellittiche in PowerPoint con Aspose.Slides

Aspose.Slides per .NET è una potente API che consente agli sviluppatori di creare, leggere, scrivere, modificare e convertire presentazioni PowerPoint senza dover utilizzare Microsoft PowerPoint. Questo esempio illustra come creare una forma ellittica in una presentazione, applicare la formattazione, inclusi il colore di riempimento e le proprietà della linea, e salvare il risultato come file PPTX. Il codice illustra la capacità di Aspose.Slides di manipolare le forme con un controllo preciso sul loro aspetto e posizionamento.

## Istruzioni per l'installazione

1. Installa tramite NuGet Package Manager:
   ```
   Install-Package Aspose.Slides
   ```

   Oppure utilizzando la CLI .NET:
   ```
   dotnet add package Aspose.Slides
   ```

2. Aggiungi riferimenti:
   Aggiungi il seguente namespace al tuo file C#:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

3. Impostazione della licenza (facoltativa):
   Se hai una licenza, applicala utilizzando:
   ```csharp
   License license = new License();
   license.SetLicense("Aspose.Slides.lic");
   ```

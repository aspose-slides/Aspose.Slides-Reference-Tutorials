# Creazione di forme con riempimento di immagini affiancate in PowerPoint utilizzando Aspose.Slides

Aspose.Slides per .NET offre potenti funzionalità per la creazione e la manipolazione di presentazioni PowerPoint a livello di codice. Questo esempio mostra come creare un rettangolo e riempirlo con un motivo a immagine affiancata. Il riempimento a immagine affiancata è particolarmente utile per creare sfondi con texture, filigrane o elementi decorativi all'interno delle presentazioni.

## Istruzioni per l'installazione

1. Installa tramite NuGet Package Manager:
   ```
   Install-Package Aspose.Slides
   ```

2. Installa tramite .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Installazione manuale:
   - Scarica l'ultima versione da [Aspose.Slides per .NET](https://releases.aspose.com/slides/net/)
   - Aggiungi un riferimento a Aspose.Slides.dll nel tuo progetto

## Istruzioni per l'installazione

1. Installa Aspose.Slides per .NET utilizzando NuGet Package Manager:
   ```
   Install-Package Aspose.Slides
   ```

2. In alternativa, scarica direttamente il pacchetto:
   - Visita il [Pagina di download di Aspose.Slides](https://downloads.aspose.com/slides/net)
   - Scarica l'ultima versione
   - Estrarre e fare riferimento ad Aspose.Slides.dll nel progetto

3. Aggiungere l'istruzione using necessaria:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

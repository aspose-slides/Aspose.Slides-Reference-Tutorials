# Creazione di presentazioni PowerPoint con testo formattato

Aspose.Slides per .NET è una potente API che consente agli sviluppatori di creare, modificare e convertire presentazioni PowerPoint senza dover utilizzare Microsoft PowerPoint. Questo esempio illustra come creare una semplice presentazione contenente una forma rettangolare con paragrafi di testo formattati.

Il codice crea una nuova presentazione, aggiunge una forma rettangolare alla prima diapositiva e quindi aggiunge diversi paragrafi con diverse opzioni di formattazione del testo. Mostra come impostare proprietà del carattere come dimensione e tipo di carattere per specifiche porzioni di testo.

## Istruzioni per l'installazione

1. Installa tramite NuGet Package Manager:
   ```
   Install-Package Aspose.Slides
   ```

2. Installa tramite .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Installa tramite la console del gestore pacchetti:
   ```
   PM> Install-Package Aspose.Slides
   ```

4. Aggiungi un riferimento al tuo progetto:
   Dopo l'installazione, aggiungi un riferimento al tuo progetto:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   ```

## Istruzioni per l'installazione

1. Installa tramite NuGet Package Manager:
   ```
   Install-Package Aspose.Slides
   ```

2. Installa tramite .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Installa manualmente:
   - Scarica l'ultima versione da [Pagina delle versioni di Aspose.Slides per .NET](https://releases.aspose.com/slides/net/)
   - Aggiungi il riferimento DLL al tuo progetto

# Creazione di testo verticale in PowerPoint tramite Aspose.Slides

Aspose.Slides per .NET offre potenti funzionalità per la creazione e la modifica di presentazioni PowerPoint a livello di codice. Questo esempio illustra come creare una presentazione contenente una forma rettangolare con testo ruotato verticalmente. Il testo è orientato a un angolo di 270 gradi, rendendolo leggibile dal basso verso l'alto. Questa tecnica è utile per creare etichette laterali, layout di testo unici o elementi di design specifici nelle presentazioni.

## Istruzioni per l'installazione

1. Installa Aspose.Slides per .NET
   
   Installa il pacchetto Aspose.Slides utilizzando NuGet Package Manager:
   
   ```csharp
   Install-Package Aspose.Slides
   ```
   
   Oppure utilizzando la CLI .NET:
   
   ```csharp
   dotnet add package Aspose.Slides
   ```

2. Aggiungi riferimenti
   
   Aggiungere la seguente istruzione using al file C#:
   
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Export;
   using System.Drawing;
   ```

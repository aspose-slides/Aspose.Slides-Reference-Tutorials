# Rotazione di titoli ed etichette di grafici in PowerPoint con Aspose.Slides

Aspose.Slides per .NET offre potenti funzionalità per la creazione e la modifica di presentazioni PowerPoint a livello di codice. Questo esempio illustra come creare una presentazione con un istogramma a colonne raggruppate che presenta elementi di testo ruotati, in particolare ruotando sia il titolo del grafico che le etichette dati per una migliore leggibilità e un impatto visivo migliore.

## Istruzioni per l'installazione

1. Installa tramite NuGet Package Manager:
   ```
   Install-Package Aspose.Slides
   ```

2. Oppure installa tramite .NET CLI:
   ```
   dotnet add package Aspose.Slides
   ```

3. Aggiungi lo spazio dei nomi al tuo file C#:
   ```csharp
   using Aspose.Slides;
   using Aspose.Slides.Charts;
   ```
